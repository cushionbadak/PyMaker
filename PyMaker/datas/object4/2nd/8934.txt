Dumping the traceback on a user signal¶


(signum, file=sys.stderr, all_threads=True, chain=False)¶
Register a user signal: install a handler for the signum signal to dump
the traceback of all threads, or of the current thread if all_threads is
, into file. Call the previous handler if chain is .
The file must be kept open until the signal is unregistered by
: see issue with file descriptors.
Not available on Windows.

Changed in version 3.5: Added support for passing file descriptor to this function.




(signum)¶
Unregister a user signal: uninstall the handler of the signum signal
installed by . Return  if the signal was registered,
 otherwise.
Not available on Windows.
