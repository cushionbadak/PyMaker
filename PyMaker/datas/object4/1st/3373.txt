<dt id="time.process_time">
<code class="descclassname">time.</code><code class="descname">process_time</code><span class="sig-paren">(</span><span class="sig-paren">)</span> → float<a class="headerlink" href="#time.process_time" title="Permalink to this definition">¶</a></dt>
<dd><p id="index-8">Return the value (in fractional seconds) of the sum of the system and user
CPU time of the current process.  It does not include time elapsed during
sleep.  It is process-wide by definition.  The reference point of the
returned value is undefined, so that only the difference between the results
of consecutive calls is valid.</p>

<p><span class="versionmodified">New in version 3.3.</span></p>

</dd>
