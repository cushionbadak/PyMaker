bytes.istitle()¶

bytearray.istitle()¶
Return true if the sequence is ASCII titlecase and the sequence is not
empty, false otherwise. See bytes.title() for more details on the
definition of “titlecase”.
For example:
>>> b'Hello World'.istitle()
True
>>> b'Hello world'.istitle()
False
