turtle.ontimer(fun, t=0)¶




Parameters:
fun – a function with no arguments
t – a number >= 0





Install a timer that calls fun after t milliseconds.
>>> running = True
>>> def f():
...     if running:
...         fd(50)
...         lt(60)
...         screen.ontimer(f, 250)
>>> f()   ### makes the turtle march around
>>> running = False
