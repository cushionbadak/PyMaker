<p>I need to know how you can count the number of times an item appears in a list WITHOUT using the <code>.count()</code> function. For example, I know that if I have a code that runs as</p>
<pre><code>&gt;&gt;&gt; [1,2,3,1,2,1].count(1)
</code></pre>
<p>then it will output <code>3</code>.</p>
<p>To clarify a little, I want to know specifically how I can get that output <code>3</code>, without using <code>.count()</code>.</p>

<div class="post-text" itemprop="text">
<p>One of the many ways of doing this:</p>
<pre><code>x = [1,2,3,1,2,1]
count = len([i for i in x if i == 1])
print count
</code></pre>
<p>Another way:</p>
<pre><code>&gt;&gt;&gt; from collections import Counter
&gt;&gt;&gt; counter = Counter(x)
Counter({1: 3, 2: 2, 3: 1})
&gt;&gt;&gt; counter.get(1)
3
&gt;&gt;&gt; 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>you can use a for loop like this </p>
<pre><code>x = [1,2,3,1,2,1]
count = 0
for i in x:
    if i == 3:
        count+= 1
print count
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum</code></a> and a <a href="https://docs.python.org/3/reference/expressions.html#grammar-token-generator_expression" rel="nofollow">generator expression</a>:</p>
<pre><code>&gt;&gt;&gt; lst = [1, 2, 3, 1, 2, 1]
&gt;&gt;&gt; sum(x == 1 for x in lst)
3
&gt;&gt;&gt; sum(1 for x in lst if x == 1) # Alternately
3
&gt;&gt;&gt;
</code></pre>
</div>
