<p>I need to remove \t from a string that is being written however when ever I do</p>
<pre><code>str(contents).replace('\t', ' ')
</code></pre>
<p>it just removes all of the tabs. I understand this is because \t is how you write tabs but I want to know how to just treat it like a regular string.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can prefix the string with <code>r</code> and create a <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow">raw-string</a>:</p>
<pre><code>str(contents).replace(r'\t', ' ')
</code></pre>
<p>Raw-strings do not process escape sequences.  Below is a demonstration:</p>
<pre><code>&gt;&gt;&gt; mystr = r'a\t\tb'  # Escape sequences are ignored
&gt;&gt;&gt; print(mystr)
a\t\tb
&gt;&gt;&gt; print(mystr.replace('\t', ' '))  # This replaces tab characters
a\t\tb
&gt;&gt;&gt; print(mystr.replace(r'\t', ' '))  # This replaces the string '\t'
a  b
&gt;&gt;&gt;
</code></pre>
