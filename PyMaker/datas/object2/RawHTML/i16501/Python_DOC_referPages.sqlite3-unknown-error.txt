<div class="post-text" itemprop="text">
<ol>
<li><p>Don't use Python <code>{num}</code> formatting, it won't quote correctly for SQL.  Use the <code>?</code> placeholder instead:</p>
<p>c.executemany('INSERT INTO stocks VALUES (?,?)', (12, 'whiskey'))</p></li>
<li><p>Do the sub selects separately, verify the values, then use them in the larger SQL statement -- it's clearer and simpler.</p></li>
</ol>
