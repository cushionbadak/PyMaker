<p>How can I do something like:</p>
<pre><code>&gt;&gt;&gt; s = u'hello'
&gt;&gt;&gt; isinstance(s,str)
False
</code></pre>
<p>But I would like <code>isinstance</code> to return <code>True</code> for this Unicode encoded string. Is there a Unicode string object type?</p>
</div>
<div class="post-text" itemprop="text">
<p>For <em>Python2</em>, you can use <a href="https://docs.python.org/2/library/functions.html#basestring" rel="nofollow noreferrer"><code>basestring</code></a> to test for <strong>both</strong>:</p>
<pre><code>isinstance(unicode_or_bytestring, basestring)
</code></pre>
<p><code>basestring</code> is only available in Python 2, and is the abstract base type of both <a href="https://docs.python.org/2/library/functions.html#str" rel="nofollow noreferrer"><code>str</code></a> and <a href="https://docs.python.org/2/library/functions.html#unicode" rel="nofollow noreferrer"><code>unicode</code></a>.</p>
<p>If you wanted to test for <em>just</em> <code>unicode</code>, then do so explicitly:</p>
<pre><code>isinstance(unicode_tring, unicode)
</code></pre>
<p>For Python 3, test for <code>str</code> <strong>only</strong>:</p>
<pre><code>isinstance(unicode_or_bytestring, str)
</code></pre>
<p>or, if you must handle bytestrings, test for <code>bytes</code> <em>separately</em>:</p>
<pre><code>isinstance(unicode_or_bytestring, bytes)
</code></pre>
<p>The two types are deliberately not exchangible; use explicit encoding (for <code>str</code> to <code>bytes</code>) and decoding (<code>bytes</code> to <code>str</code>) to convert between the types.</p>
</div>
<div class="post-text" itemprop="text">
<blockquote>
<p>Is there a Unicode string object type?</p>
</blockquote>
<p>Yes, it is called <a href="https://docs.python.org/2/library/functions.html#unicode" rel="noreferrer"><code>unicode</code></a>:</p>
<pre><code>&gt;&gt;&gt; s = u'hello'
&gt;&gt;&gt; isinstance(s, unicode)
True
&gt;&gt;&gt;
</code></pre>
<p>Note that in Python 3.x, this type was removed because <a href="https://docs.python.org/3/whatsnew/3.0.html#text-vs-data-instead-of-unicode-vs-8-bit" rel="noreferrer">all strings are now Unicode</a>.</p>
</div>
<div class="post-text" itemprop="text">
<blockquote>
<p>Is there a Unicode string object type?</p>
</blockquote>
<p>Yes this works:</p>
<pre><code>&gt;&gt;&gt; s = u'hello'
&gt;&gt;&gt; isinstance(s, unicode)
True
&gt;&gt;&gt;
</code></pre>
<p>This is however only useful if you know that it is unicode.
Another solution is to use the <a href="https://pythonhosted.org/six/" rel="nofollow noreferrer"><code>six</code></a> package, which saves you from python2.x and python3.x conversions and catches <code>unicode</code> and <code>str</code></p>
<pre><code>&gt;&gt;&gt; unicode_s = u'hello'
&gt;&gt;&gt; s = 'hello'
&gt;&gt;&gt; isinstance(unicode_s, str)
False
&gt;&gt;&gt; isinstance(unicode_s, unicode)
True
&gt;&gt;&gt; isinstance(s, str)
True
&gt;&gt;&gt; isinstance(unicode_s, str)
False
&gt;&gt;&gt; isinstance(s, six.string_types)
True
&gt;&gt;&gt; isinstance(unicode_s, six.string_types)
True
</code></pre>
