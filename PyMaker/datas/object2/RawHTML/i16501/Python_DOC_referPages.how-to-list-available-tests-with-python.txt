<p>How to just list all discovered tests?
I found this command:</p>
<pre><code>python3.4 -m unittest discover -s .
</code></pre>
<p>But it's not exactly what I want, because the above command executes tests. I mean let's have a project with a lot of tests. Execution time is a few minutes. This force me to wait until tests are finished.</p>
<p>What I want is something like this (above command's output)</p>
<pre><code>test_choice (test.TestSequenceFunctions) ... ok
test_sample (test.TestSequenceFunctions) ... ok
test_shuffle (test.TestSequenceFunctions) ... ok
</code></pre>
<p>or even better, something more like this (after editing above):</p>
<pre><code>test.TestSequenceFunctions.test_choice
test.TestSequenceFunctions.test_sample
test.TestSequenceFunctions.test_shuffle
</code></pre>
<p>but without execution, only printing tests "paths" for copy&amp;paste purpose.</p>

<div class="post-text" itemprop="text">
<p>Command line command <code>discover</code> is implemented using <a href="https://docs.python.org/3/library/unittest.html?highlight=discover#unittest.TestLoader"><code>unittest.TestLoader</code></a>. Here's the somewhat elegant solution</p>
<pre><code>import unittest

def print_suite(suite):
    if hasattr(suite, '__iter__'):
        for x in suite:
            print_suite(x)
    else:
        print(suite)

print_suite(unittest.defaultTestLoader.discover('.'))
</code></pre>
<p>Running example:</p>
<pre><code>In [5]: print_suite(unittest.defaultTestLoader.discover('.'))
test_accounts (tests.TestAccounts)
test_counters (tests.TestAccounts)
# More of this ...
test_full (tests.TestImages)
</code></pre>
<p>This works because <a href="https://docs.python.org/3/library/unittest.html?highlight=discover#unittest.TestLoader.discover"><code>TestLoader.discover</code></a>  returns <code>TestSuite</code> objects, that implement <code>__iter__</code> method and therefore are iterable.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could do something like:</p>
<pre><code>from your_tests import TestSequenceFunctions
print('\n'.join([f.__name__ for f in dir(TestSequenceFunctions) if f.__name__.startswith('test_')]))
</code></pre>
<p>I'm not sure if there is an exposed method for this via <code>unittest.main</code>.</p>
