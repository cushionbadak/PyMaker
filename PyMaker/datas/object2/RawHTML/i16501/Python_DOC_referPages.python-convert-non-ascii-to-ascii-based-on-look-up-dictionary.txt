<p>In Python, I need to convert special characters into ascii letters. I have a series of translations in a dictionary </p>
<pre><code>dict_trans = {"U+1E9A":"a", "U+1EA0":"a"} # + more

my_char = "ẚ"
</code></pre>
<p>How do I covert <code>my_char</code> into (in this case) <code>a</code>?</p>
<p>I can change the format of the characters in <code>dict_trans</code> (but to what)?</p>

<div class="post-text" itemprop="text">
<p>From the <a href="https://github.com/iki/unidecode" rel="nofollow"><code>unidecode</code></a> module, you could use the <code>unidecode</code> function.</p>
<pre><code>&gt;&gt;&gt; from unidecode import unidecode
&gt;&gt;&gt; unidecode('ẚ')
'a'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use thes names from <a href="https://docs.python.org/3/library/unicodedata.html" rel="nofollow"><code>unicodedata</code></a>:</p>
<pre><code>import unicodedata

unicodedata.name("a")
# 'LATIN SMALL LETTER A'    
unicodedata.name("ẚ")
# 'LATIN SMALL LETTER A WITH RIGHT HALF RING'    
unicodedata.lookup('LATIN SMALL LETTER A WITH RIGHT HALF RING')
# 'ẚ'

d = {'LATIN SMALL LETTER A WITH RIGHT HALF RING':'a'}
d['LATIN SMALL LETTER A WITH RIGHT HALF RING']
# 'a'
</code></pre>
</div>
