<p>The line breaks where you have defined your 3 lines don't do what you think ... Python will smash those strings together as if there was no return there at all (which is what you are seeing).  Instead, try putting this there:</p>
<pre><code>def show_info(self):
    lines = ['My Name', '123 Any Rd', 'Town Fl, 12345']
    tkinter.messagebox.showinfo('Text', "\n".join(lines))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It may seem simple, but for such few lines, the most elegant solution is to just end each one with a newline character (<code>\n</code>):</p>
<pre><code>def show_info(self):
    tkinter.messagebox.showinfo('Text',
                                'My Name\n'
                                '123 Any Rd\n'
                                'Town Fl, 12345\n')
</code></pre>
<p>If you have many lines though (such as a paragraph), you can use a <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow">multi-line string</a>:</p>
<pre><code>    def show_info(self):
        tkinter.messagebox.showinfo('Text', '''\
My Name
123 Any Rd
Town Fl, 12345''')
</code></pre>
</div>
