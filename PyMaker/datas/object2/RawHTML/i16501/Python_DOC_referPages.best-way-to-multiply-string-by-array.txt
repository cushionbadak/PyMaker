<div class="post-text" itemprop="text">
<p>I want to repeat each character in my string by a number I have in an array i.e. if </p>
<pre><code>rep = [1, 0, 1, 1, 3, 0, 0, 1, 0]
seq = 'AATCGGGAA'
</code></pre>
<p>I want something like </p>
<pre><code>seq*rep
</code></pre>
<p>to output </p>
<pre><code>ATCGGGA
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Fastest way to do this will be to use <code>map</code> with <code>operator.mul</code>:</p>
<pre><code>&gt;&gt;&gt; from operator import mul
&gt;&gt;&gt; ''.join(map(mul, seq, rep))
'ATCGGGA'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip"><code>zip</code></a>, a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions">list comprehension</a>, and <a href="https://docs.python.org/3/library/stdtypes.html#str.join"><code>str.join</code></a>:</p>
<pre><code>&gt;&gt;&gt; rep = [1, 0, 1, 1, 3, 0, 0, 1, 0]
&gt;&gt;&gt; seq = 'AATCGGGAA'
&gt;&gt;&gt;
&gt;&gt;&gt; list(zip(seq, rep)) # zip pairs up the items in the two lists
[('A', 1), ('A', 0), ('T', 1), ('C', 1), ('G', 3), ('G', 0), ('G', 0), ('A', 1), ('A', 0)]
&gt;&gt;&gt;
&gt;&gt;&gt; ''.join([x*y for x,y in zip(seq, rep)])
'ATCGGGA'
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This should do the trick:</p>
<pre><code>"".join([s*r for r, s in zip(rep, seq)])
</code></pre>
<p>Output:</p>
<pre><code>"ATCGGGA"
</code></pre>
</div>
