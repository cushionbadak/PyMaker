<div class="post-text" itemprop="text">
<p>I have a interger value like:</p>
<pre><code>99990
</code></pre>
<p>I want this value to be converted into like:</p>
<pre><code>999.90 
</code></pre>
<p>Regardless of the length of the number, I need to display the number with 2 decimal places at the end, and I'd like to do it in an efficient way. The purpose is to display money amount.</p>
</div>
<div class="post-text" itemprop="text">
<p>This will work quickly and easily.</p>
<pre><code>print "%.2f" % (99990 / 100.0)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>With an integer value, you can use <code>divmod</code> to produce separate dollar and cent components.</p>
<pre><code>dollars, cents = divmod(value, 100)
print("{0}.{1}".format(dollars, cents))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#format" rel="nofollow"><code>format</code></a> here:</p>
<pre><code>&gt;&gt;&gt; n = 99990
&gt;&gt;&gt; format(n / 100, '.02f')
'999.90'
&gt;&gt;&gt;
</code></pre>
<p>The result needs to be a string because Python automatically removes the trailing <code>0</code> with numbers:</p>
<pre><code>&gt;&gt;&gt; n = 99990
&gt;&gt;&gt; n / 100
999.9
&gt;&gt;&gt;
</code></pre>
<hr/>
<p>For those who are still using Python 2.x, you will need to divide the number by a float:</p>
<pre><code>&gt;&gt;&gt; n = 99990
&gt;&gt;&gt; format(n / 100.0, '.02f')
'999.90'
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Dividing with a float 100. value should covert it into a float. After that, it's only a matter of formatting to display it the way you like.</p>
<pre><code>$ python
Python 2.7.6 (default, Mar 22 2014, 22:59:56)
[GCC 4.8.2] on linux2
Type "help", "copyright", "credits" or "license" for more information.
&gt;&gt;&gt; number = 99990
&gt;&gt;&gt; print("{0:.2f}".format(number / 100.))
999.90
</code></pre>
</div>
