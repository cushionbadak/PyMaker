<div class="post-text" itemprop="text">
<p>Just use the <code>in</code> operator (which is accompanied by the <code>not in</code> operator):</p>
<pre><code>if (1, 'a') in edges:

if (1, 'a') not in edges:
</code></pre>
<p>Below is an excerpt from the <a href="https://docs.python.org/3/library/stdtypes.html#typesmapping">documentation</a> of Python's dictionary type, where <code>d</code> is a dictionary:</p>
<blockquote>
<p><code>key in d</code></p>
<p>Return <code>True</code> if <code>d</code> has a key <code>key</code>, else <code>False</code>.</p>
<p><code>key not in d</code></p>
<p>Equivalent to <code>not key in d</code>.</p>
</blockquote>
</div>
