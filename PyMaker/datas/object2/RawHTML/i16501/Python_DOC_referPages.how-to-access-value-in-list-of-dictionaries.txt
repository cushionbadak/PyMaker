<div class="post-text" itemprop="text">
<p>That is not a list of dictionaries; it is a list of two-item tuples:</p>
<pre><code>&gt;&gt;&gt; d =  [(u'Code', u'US'), (u'Country', u'United States'), (u'Name', u'Bill')]
&gt;&gt;&gt; type(d)
&lt;class 'list'&gt;
&gt;&gt;&gt; type(d[0])
&lt;class 'tuple'&gt;
&gt;&gt;&gt;
</code></pre>
<p>If you want <code>d</code> to be converted into a dictionary, place it in <a href="https://docs.python.org/3/library/functions.html#func-dict"><code>dict</code></a>:</p>
<pre><code>&gt;&gt;&gt; d =  [(u'Code', u'US'), (u'Country', u'United States'), (u'Name', u'Bill')]
&gt;&gt;&gt; d = dict(d)
&gt;&gt;&gt; d
{'Code': 'US', 'Name': 'Bill', 'Country': 'United States'}
&gt;&gt;&gt; d['Code']
'US'
&gt;&gt;&gt;
</code></pre>
