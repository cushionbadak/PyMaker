<p>Is there an easy way to delete a property of a dictionary in Python when it is possible that the property may not be present to begin with?</p>
<p>if the <code>del</code> statement is used, a KeyError is the result:</p>
<pre><code>a = {}
del a['foo']
&gt;&gt;&gt;KeyError: 'foo'
</code></pre>
<p>Its a little wordy to use an if statement and you have to type <code>'foo'</code> and <code>a</code> twice :</p>
<pre><code>a = {}
if 'foo' in a:
    del a['foo']
</code></pre>
<p>Looking for something similar to <code>dict.get()</code> that defaults to a <code>None</code> value if the property is not present:</p>
<pre><code>a = {}
a.get('foo')  # Returns None, does not raise error
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#dict.pop"><code>dict.pop</code></a>, which allows you to specify a default value to return if the key does not exist:</p>
<pre><code>&gt;&gt;&gt; d = {1:1, 2:2}
&gt;&gt;&gt; d.pop(1, None)
1
&gt;&gt;&gt; d
{2: 2}
&gt;&gt;&gt; d.pop(3, None)  # No error is thrown
&gt;&gt;&gt; d
{2: 2}
&gt;&gt;&gt;
</code></pre>
