<p>I'd like to call a parent's <strong>call</strong> method from inherited class</p>
<p>Code looks like this</p>
<pre><code>#!/usr/bin/env python

class Parent(object):

    def __call__(self, name):
        print "hello world, ", name


class Person(Parent):

    def __call__(self, someinfo):                                                                                                                                                            
        super(Parent, self).__call__(someinfo)

p = Person()
p("info")
</code></pre>
<p>And I get, </p>
<pre><code>File "./test.py", line 12, in __call__
super(Parent, self).__call__(someinfo)
AttributeError: 'super' object has no attribute '__call__'
</code></pre>
<p>And I can't figure out why, can somebody please help me with this?</p>
</div>
<div class="post-text" itemprop="text">
<p>The <code>super</code> function takes the <em>derived</em> class as its first parameter, not the base class.</p>
<pre><code>super(Person, self).__call__(someinfo)
</code></pre>
<p>If you need to use the base class, you can do it directly (but beware that this will break multiple inheritance, so you shouldn't do it unless you're sure that's what you want):</p>
<pre><code>Parent.__call__(self, someinfo)
</code></pre>
</div>
