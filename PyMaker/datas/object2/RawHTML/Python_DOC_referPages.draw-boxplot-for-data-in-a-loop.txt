<p>I have a 1000*8 dataset and each column represent the price of a stock in different time so there are 8 stocks. I want to draw 8 boxplots for all the stocks to examine the extreme values in a loop in python. Could you please tell me how I can do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>As a quick alternative to using matplotlib directly, Pandas has a reasonable boxplot function that could be used.</p>
<pre><code>df = pd.DataFrame(np.random.randn(1000, 8), columns=list('ABCDEFGH'))
df.boxplot(column = list(df.columns))
</code></pre>
<p><a href="https://i.stack.imgur.com/V4Ie2.png" rel="nofollow noreferrer"><img alt="output" src="https://i.stack.imgur.com/V4Ie2.png"/></a></p>
<p>edit: Just realise your question asked to do this in a loop.</p>
<pre><code>for c in df.columns:
    fig, ax = plt.subplots()
    ax = df.boxplot(column = c)
</code></pre>
