<div class="post-text" itemprop="text">
<p>create a new list with item inside the "{}", then join them with new line</p>
<pre><code>test = ["Jargon", "Hello", "This", "Is", "Great"]
group = '\n'.join([test[0]] + ['{'+item+'}' for item in test[1:]])
print(group)
</code></pre>
<p>output:</p>
<pre><code>Jargon
{Hello}
{This}
{Is}
{Great}
</code></pre>

<div class="post-text" itemprop="text">
<p>I am not absolutely sure what you are asking, as your post is not very concise, but this will print each item, except the first on its own line:</p>
<p><code>lst = ["Jargon", "Hello", "This", "Is", "Great"]
print '\n'.join([i for i in lst[1:]])</code></p>
<p>The <code>\n</code> is used to cause a line break when used within a string. Use list slicing to perform the operation on  all elements <em>after</em> the first item. Using the <code>for</code>loop allows you to iterate over all elements within the chosen indices. </p>
<p>output:
<code>
Hello
This
Is
Great
</code></p>
</div>
