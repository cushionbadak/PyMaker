<div class="post-text" itemprop="text">
<p>If the <code>instagramJSON</code> is a <code>str</code> then you should use <code>json.loads</code> instead of <code>json.load</code>. If <code>instagramJSON</code> is a string then it would have raised the same exception in Python 2 so there may be more going on.</p>
<p><a href="https://docs.python.org/3/library/json.html#json.loads" rel="nofollow noreferrer">Python JSON documentation</a></p>
<p>If it is a response generated by <code>urllib.request</code> then the top answer to <a href="https://stackoverflow.com/questions/6541767/python-urllib-error-attributeerror-bytes-object-has-no-attribute-read">this question</a> may be relevant.</p>
</div>
