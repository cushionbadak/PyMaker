<pre><code>a = [1, 23, 456, 6789]
for i in map(str, a):
  for j in range(len(i)):
    print(i[j:]+i[:j])
</code></pre>
<p>The built-in function <code>map</code> is documented on <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow">python.org</a>.</p>
<p>If the order of the numbers is relevant, then do:</p>
<pre><code>import itertools
a = [1, 23, 456, 6789]
for i in map(str, a):
  for j in itertools.chain(range(1),range(len(i)-1,0,-1)):
    print(i[j:]+i[:j])
</code></pre>
<p>The function <code>chain</code> of the built-in module <code>itertools</code> is documented on <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow">python.org</a>. You can also do it without <code>itertools.chain()</code>, but that's the first thing that popped into my mind.</p>

<div class="post-text" itemprop="text">
<pre><code>def rotationsHelper(s, n):
    return [] if n &lt;= 0 else [s] + rotationsHelper(s[-1] + s[:-1], n-1)

def rotations(s):
    return rotationsHelper(s, len(s))

def rotateIntList(int_list):
    if int_list == []:
        return []
    return map(int, rotations(str(int_list[0]))) + rotateIntList(int_list[1:])
</code></pre>
<p>then</p>
<pre><code>In [23]: rotateIntList(a)
Out[23]: [1, 23, 32, 456, 645, 564, 6789, 9678, 8967, 7896]
</code></pre>

<div class="post-text" itemprop="text">
<p>You can do this by - as you specify in your question - loop over the string <code>n</code> times, with <code>n</code> the length of the string. You however must first transform the number to an integer.</p>
<p>Furthermore you need to define a <em>rotate</em> operation that places the end of the string on the top, you can do this as:</p>
<pre><code>def rotate (s) :
    n = len(s)
    return s[n-1:n]+s[0:n-1]
</code></pre>
<p>Then you simply iterate over all numbers, convert them to string, and repeat <em>n</em> time - with <em>n</em> the length of the string - a print and a rotate function.</p>
<pre><code>for i in a:
  si = str(i)
  for j in range(0,len(si)):
    print si
    si = rotate(si)
</code></pre>
