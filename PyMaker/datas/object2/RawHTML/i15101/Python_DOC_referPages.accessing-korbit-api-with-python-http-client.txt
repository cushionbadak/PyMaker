<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/http.client.html#http.client.HTTPSConnection" rel="nofollow"><code>http.client.HTTPSConnection</code></a> accepts hostname, not a url.</p>
<pre><code>conn = http.client.HTTPSConnection("api.korbit.co.kr", 443)
</code></pre>
<p>BTW, I recommend to use library like <a href="http://docs.python-requests.org/en/latest/" rel="nofollow"><code>python-requests</code></a>:</p>
<pre><code>import requests

r = requests.get('https://api.korbit.co.kr/v1/ticker/detailed')
data = r.json()
</code></pre>
