<div class="post-text" itemprop="text">
<p>You are using Windows, and there is no <code>/tmp</code> temporary directory in Windows. The database connection fails because SQLite3 has no directory to create the database file in. Pick a different location for your database file.</p>
<p>In this case, you could use <a href="https://docs.python.org/3/library/tempfile.html#tempfile.gettempdir" rel="nofollow"><code>tempfile.gettempdir()</code></a> to build a path:</p>
<pre><code>import tempfile
import os.path

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///' + os.path.join(
    tempfile.gettempdir(), 'test.db')
</code></pre>
