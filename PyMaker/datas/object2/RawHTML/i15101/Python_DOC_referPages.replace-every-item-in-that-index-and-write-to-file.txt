<p>To match the text you can use: </p>
<pre><code>new_str = re.sub(r'\s+Title\s+=', 'Author                 =', old_str) 
</code></pre>
<p><code>\s</code> means whitespace, <code>+</code> means one or more. You can use <code>\s{4}</code> to match exactly 4 whitespaces, or as many as you need. More info <a href="https://docs.python.org/3/library/re.html" rel="nofollow">here</a>.</p>
<p>Alternatively, you can use <code>replace()</code>:</p>
<pre><code>new_str = old_str.replace('  Title                  =', 'Author                 =')
</code></pre>
<hr/>
<p>You can use <code>range()</code> to simplify the rest of your code a bit. range() has 3 arguments, 2 of which are optional; start, end, step.</p>
<pre><code>for i in range(2, 200, 25):
    print(i)
</code></pre>

<span class="comment-copy"><code>re.sub()</code> checks a string, and always returns a new string after replacing the match (if it exists). It doesn't modify the old string. Strings are immutable. For example for second element you can simply use <code>my_list[1] = re.sub(str1, str2 , my_list[1])</code>, where str1 is what you are looking for, and str2 what you replace it with. Check more about immutables here: <a href="https://docs.python.org/2/reference/datamodel.html" rel="nofollow noreferrer">docs.python.org/2/reference/datamodel.html</a></span>
