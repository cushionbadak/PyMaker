<p>I have two dictionaries with keys of a letter and the values are numbers (occurrences in the message I have)
For example ill have:</p>
<pre><code>secretWord = 'cat' 
secret = {'a':123, 't',: 42, 'c' 421}  # =&gt; cat
known = {'g':400, 'd': 40, 'o':90}     # =&gt; god
</code></pre>
<p>so what i want to do is find a way to replace the letters of the secretWord with the letters in known by the highest, second, and least numbers. So for example secretWord will become 'god'</p>
</div>
<div class="post-text" itemprop="text">
<p>Order by dictionary keys by values using <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow"><code>sorted</code></a>, then join the keys using <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join</code></a>:</p>
<pre><code>&gt;&gt;&gt; sorted(known, key=known.get, reverse=True)
['g', 'o', 'd']
&gt;&gt;&gt; ''.join(sorted(known, key=known.get, reverse=True))
'god'
</code></pre>
