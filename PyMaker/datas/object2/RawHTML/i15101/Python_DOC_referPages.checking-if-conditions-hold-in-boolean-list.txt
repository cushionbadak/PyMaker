<div class="post-text" itemprop="text">
<p>I'm trying to run some function if a list of booleans is true.
My list consists of boolean functions.</p>
<pre><code>list = [file.endswith('05',21,22), file.endswith('11',21,22),
       file.endswith('17',21,22), file.endswith('21',21,22),
       file.endswith('23',21,22)]

if any(True in list) == True:           
    # do something
</code></pre>
<p>Currently, the if clause gives me an error </p>
<pre><code>  if any(True in list) == True:
TypeError: 'bool' object is not iterable
</code></pre>
<p>Not really sure how to fix it.</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow"><code>any</code></a> is expecting an iterable argument, which it will then run through to see if any of its items evaluate to <code>True</code>.  Moreover, the expression <code>True in list</code> returns a non-iterable boolean value:</p>
<pre><code>&gt;&gt;&gt; lst = [False, True, False]
&gt;&gt;&gt; True in lst
True
&gt;&gt;&gt;
</code></pre>
<p>To fix the problem, you should simply pass the list to <code>any</code>:</p>
<pre><code>if any(list):
</code></pre>
<p>You should also refrain from making a user-defined name the same as one of the built-ins.  Doing so overshadows the built-in and makes it unusable in the current scope.</p>
</div>
<div class="post-text" itemprop="text">
<blockquote>
<p>any(iterable):</p>
<p>Return True if any element of the iterable is true. If the iterable is
  empty, return False.</p>
</blockquote>
<p>Your code should use:</p>
<pre><code> if any(list):
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Note that your code is much shorter if written as:</p>
<pre><code>if any(file.endswith(suffix, 21, 22) 
       for suffix in ['05', '11', '17', '21', '23']):
</code></pre>
</div>
<span class="comment-copy">Don't shadow the builtin <code>list</code>.</span>
