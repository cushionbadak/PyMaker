<div class="post-text" itemprop="text">
<p>Is there an <code>until</code> statement or loop in python? This does not work:</p>
<pre><code>x = 10
list = []
until x = 0:
    list.append(raw_input('Enter a word: '))
    x-=1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The equivalent is a <code>while x<sub>1</sub> != x<sub>2</sub></code> loop.</p>
<p>Thus, your code becomes:</p>
<pre><code>x = 10
lst = [] #Note: do not use list as a variable name, it shadows the built-in
while x != 0:
    lst.append(raw_input('Enter a word: '))
    x-=1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>while x != 0:
    #do stuff
</code></pre>
<p>This will run until x == 0</p>
</div>
<div class="post-text" itemprop="text">
<p>You don't really need to count how many times you're looping unless you're doing something with that variable.  Instead, you can use a <code>for</code> loop that will fire up to 10 times instead:</p>
<pre><code>li = []
for x in range(10):
    li.append(raw_input('Enter a word: '))
</code></pre>
<p>As an aside, don't use <code>list</code> as a variable name, as that masks the actual <code>list</code> method.</p>
</div>
<div class="post-text" itemprop="text">
<p>Python emulates <em>until</em> loops with the <a href="https://docs.python.org/3/library/functions.html" rel="nofollow"><code>iter(iterable, sentinel)</code></a> idiom:</p>
<pre><code>x = 10
list = []
for x in iter(lambda: x-1, 0):
    list.append(raw_input('Enter a word: '))
</code></pre>
<ul>
<li>I was forced to construct a trivial <em>lambda</em> for demonstration purposes; here a simple <code>range()</code> would suffice, as as @Makoto suggested.</li>
</ul>
