<div class="post-text" itemprop="text">
<p>You are trying to add up integers and a string:</p>
<pre><code>Random_num + Operator + Random_num2
</code></pre>
<p>You'd normally have to convert to a common type first, like a string:</p>
<pre><code>str(Random_num) + Operator + str(Random_num2)
</code></pre>
<p>but you can also use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow">string formatting</a>:</p>
<pre><code>User_input_string = int(input("What is the answer to {} {} {}".format(
    Random_num, Operator, Random_num2))
</code></pre>
<p>The latter gives you more flexibility in how the output string is formed.</p>
</div>
