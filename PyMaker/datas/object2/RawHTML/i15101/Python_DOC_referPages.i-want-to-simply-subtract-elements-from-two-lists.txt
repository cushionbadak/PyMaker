<div class="post-text" itemprop="text">
<p>The substraction operator is not defined for <code>list</code>, as it makes no sense in a general way. However, you can simply get the single items using the <code>[]</code> operator, and calculate the difference in a new <code>list</code>:</p>
<pre><code>newArray = list();
for i in xrange(students):
    newArray.append(firstWeight_Array[i] - secondWeight_Array[i]);
print newArray;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you need to subtract every item from one list to the corresponding item of a second list of same size, the easiest way (and probably the most idiomatic way) is to use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> and the <a href="https://docs.python.org/3.3/library/functions.html" rel="nofollow"><code>zip</code></a> function:</p>
<pre><code>diff = [first - second 
        for first, second in zip(firstWeight_Array, secondWeight_Array)]
</code></pre>
<p>Here is a simple example:</p>
<pre><code>&gt;&gt;&gt; firstWeight_Array = [10,20,30]
&gt;&gt;&gt; secondWeight_Array = [12,18,34]

&gt;&gt;&gt; diff = [first - second 
...         for first, second in zip(firstWeight_Array, secondWeight_Array)]

&gt;&gt;&gt; diff
[-2, 2, -4]
</code></pre>
<p>Please note that for space efficiency reason, in Python 2, you might prefer using <a href="https://docs.python.org/2/library/itertools.html#itertools.izip" rel="nofollow">itertools.izip</a> instead of a plain <code>zip</code>.</p>
</div>
