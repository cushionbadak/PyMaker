<div class="post-text" itemprop="text">
<p><code>dict.iteritems</code> was only around in Python 2.x, back when <code>dict.items</code> returned a list.  In Python 3.x however, <code>dict.items</code> was changed to return a <a href="https://docs.python.org/3/glossary.html#term-view" rel="nofollow">view object</a> and so <code>dict.iteritems</code> <a href="https://docs.python.org/3/whatsnew/3.0.html#views-and-iterators-instead-of-lists" rel="nofollow">was removed</a> since it was no longer necessary.</p>
<p>This means that the version of OWSLib that you are using was made for Python 2.x and is therefore incompatible with Python 3.x.  You need to either use Python 2.x or install a version of OWSLib that can use Python 3.x.</p>
