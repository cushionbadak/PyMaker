<div class="post-text" itemprop="text">
<p>I have a string like this below.</p>
<pre><code>aabbccddee
</code></pre>
<p>I want to replace the duplicate into one character in one line of RegEx to get the following result.</p>
<pre><code>abcde
</code></pre>
<p>I can easily do this by doing</p>
<pre><code>input_string = re.sub(ur"a+", u"a", input_string)
input_string = re.sub(ur"b+", u"b", input_string)
input_string = re.sub(ur"c+", u"c", input_string)
input_string = re.sub(ur"d+", u"d", input_string)
input_string = re.sub(ur"e+", u"e", input_string)
</code></pre>
<p>But this isn't as elegant as I want.</p>
</div>
<div class="post-text" itemprop="text">
<p>I have found the answer.</p>
<pre><code>input_string = re.sub(ur"([a-z])\1+", r"\1", input_string)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <code>groupby</code> from <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow">itertools</a> <em>(almost copy/pasted from the doc)</em>:</p>
<pre><code>&gt;&gt;&gt; from itertools import groupby
&gt;&gt;&gt; ''.join([k for k, g in groupby('aabbccddee')])
abcde
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could also do this:</p>
<pre><code>string = "aabbccddee"
print("".join(sorted(set(string), key = string.index)))
</code></pre>
<p>Output:</p>
<pre><code>abcde
</code></pre>
<p>You could even use <code>OrderedDict</code> here:</p>
<pre><code>from collections import OrderedDict
string = "aabbccddee"
print("".join(OrderedDict.fromkeys(string)))
</code></pre>
</div>
