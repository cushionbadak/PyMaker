<div class="post-text" itemprop="text">
<p>If I have a nested list like this:</p>
<pre><code>[['01'], ['02'], ['-1'], ['03'], ['04']]
</code></pre>
<p>Is there a way I split this nested list at every <code>['-1']</code>?</p>
<p>So that it looks like this:</p>
<pre><code>[[['01'], ['02']], [['03'], ['04']]]
</code></pre>
<p>Any sort of help would be appreciated :)</p>
</div>
<div class="post-text" itemprop="text">
<p>Try this,</p>
<pre><code>lists = [['01'], ['02'], ['-1'], ['03'], ['04'], ['-1'], ['05'], ['-1']]

results = list()
prev_idx = 0
for idx, l in enumerate(lists):
    if l == ['-1']:
        results.append(lists[prev_idx:idx])
        prev_idx = idx+1

if prev_idx &lt;= idx:     # the last group might be [] as shown in this case
    results.append(lists[prev_idx:])

print(results)
# Output
[[['01'], ['02']], [['03'], ['04']], [['05']]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3.4/library/itertools.html#itertools.groupby" rel="nofollow"><code>itertools.groupby</code></a> to group at every occurrence of your split value (here <code>['-1']</code>). <code>if not k</code> ensures that we leave out the split value itself. </p>
<pre><code>orig = [['01'], ['02'], ['-1'], ['03'], ['04']]
from itertools import groupby
n = [list(g) for k, g in groupby(orig, lambda x: x == ['-1']) if not k]
</code></pre>
</div>

<p>Seems like a usecase for <em>groupby</em></p>
<pre><code>&gt;&gt;&gt; from itertools import groupby
&gt;&gt;&gt; l = [['01'], ['02'], ['-1'], ['03'], ['04'], ['-1'], ['05'], ['06']]    
&gt;&gt;&gt; [list(g) for k,g in groupby(l, lambda x: x == ['-1']) if not k]
[[['01'], ['02']], [['03'], ['04']], [['05'], ['06']]]
</code></pre>

<div class="post-text" itemprop="text">
<p>A good old fashioned loop should do it:</p>
<pre><code>l = [['01'], ['02'], ['-1'], ['03'], ['04']]

new = []

current = []  # Build a new list here
for i, item in enumerate(l):
    if item != ['-1']:
        current.append(item)
        if i == len(l) - 1:  # If the item is the last in the list
            new.append(current)
    else:
        new.append(current)
        current = []

&gt;&gt;&gt;  [[['01'], ['02']], [['03'], ['04']]]
</code></pre>
