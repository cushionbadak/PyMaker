<div class="post-text" itemprop="text">
<p>Probably <a href="https://docs.python.org/3/library/itertools.html#itertools.starmap" rel="nofollow"><code>itertools.starmap</code></a> would be useful in your case:</p>
<pre><code>&gt;&gt;&gt; a = [1,2,3,4]
&gt;&gt;&gt; b = [1,0,1,5]
&gt;&gt;&gt; 
&gt;&gt;&gt; import itertools as it
&gt;&gt;&gt; 
&gt;&gt;&gt; import operator as op
&gt;&gt;&gt; 
&gt;&gt;&gt; list(it.starmap(op.sub, zip(a,b)))
[0, 2, 2, -1]
</code></pre>
<p>OR:</p>
<pre><code>&gt;&gt;&gt; [item for item in it.starmap(op.sub, zip(a,b))]
[0, 2, 2, -1]
</code></pre>
</div>

<div class="post-text" itemprop="text">
<p>Most people use <code>numpy</code> for numerical operations (although p.magalhaes has the answer for "pure" python).</p>
<pre><code>import numpy as np
a=np.array([1,2,3,4])
b=np.array([1,0,1,5])
c = a - b
c
</code></pre>
<p>returns</p>
<pre><code>array([ 0,  2,  2, -1])
</code></pre>
