<div class="post-text" itemprop="text">
<p>Simplest way to print a list:</p>
<pre><code>for i in a:
  print(i)
</code></pre>
<p><code>a</code> being the list.</p>
<hr/>
<p>To print a list element:</p>
<pre><code>print(a[index])
</code></pre>
<p><code>index</code> being the position within the list(<code>0</code> to <code>len-1</code>)</p>
<hr/>
<p>If you just want to check if a value exists in the list:</p>
<pre><code>value in a
</code></pre>
<p><code>value</code> being the value searched and <code>a</code> being the list.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can loop over the list three ways:</p>
<p><strong>1. If you want just each element in the list:</strong></p>
<pre><code>for i in a:
    print(i)
</code></pre>
<p><strong>2. if you want the index of the elements in a list:</strong></p>
<pre><code>for i in range(len(a)):
    print(i)
</code></pre>
<p><strong>3. If you want the index and the corresponding elements in the list:</strong></p>
<pre><code>for i, e in enumerate(a):
    print(i, e)
</code></pre>
<p>These are the three main methods of looping over lists. </p>
</div>
