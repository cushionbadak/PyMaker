<div class="post-text" itemprop="text">
<p>On python3.x, <code>range</code> returns a <code>range</code> object rather than a list.  Slicing the range object just returns another range object which is what is displaying in your terminal:</p>
<pre><code>&gt;&gt;&gt; range(1, 11)[::2]
range(1, 11, 2)
&gt;&gt;&gt; type(range(1, 11)[::2])
&lt;class 'range'&gt;
</code></pre>
<p>However, iterating over the result should produce the desired elements:</p>
<pre><code>&gt;&gt;&gt; list(range(1, 11)[::2])
[1, 3, 5, 7, 9]
</code></pre>
</div>
