<div class="post-text" itemprop="text">
<p>You can create a dictionary with a repeating key by wrapping the keys with a dummy class and then use dicttoxml on that dictionary. Use <a href="https://docs.python.org/3/library/collections.html#collections.OrderedDict" rel="nofollow noreferrer">collections.OrderedDict</a> if the order matters:</p>
<pre><code>from dicttoxml import dicttoxml
from collections import OrderedDict

class Node(object):
    def __init__(self, name):
        self._name = name

    def __str__(self):
        return self._name

quest_dict = OrderedDict([(Node('key1'), 'aaa'), (Node('key1'), 'bbb'), (Node('key1'), 'ccc')])
request_xml = dicttoxml(quest_dict, attr_type=False, root=False)
print(request_xml)
</code></pre>
<p>This gives your desired output:</p>
<pre><code>b'&lt;key1&gt;aaa&lt;/key1&gt;&lt;key1&gt;bbb&lt;/key1&gt;&lt;key1&gt;ccc&lt;/key1&gt;'
</code></pre>
