<p>Javascript code:</p>
<pre><code>var date = new Date(1466278504960);
</code></pre>
<p>return: Sat Jun 18 2016 20:35:04 GMT+0100 (WEST)</p>
<p>How can I convert the same number to date in python ?</p>
<p>When I use </p>
<pre><code>datetime.datetime.fromtimestamp(int("1466278504960")).strftime('%Y-%m-%d %H:%M:%S'))
</code></pre>
<p>I receive this error: ValueError: year is out of range</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.fromtimestamp" rel="nofollow"><code>datetime.datetime.fromtimestamp</code></a> will do this, but you need to divide the value by <code>1000</code> first (the numeric value you give and JavaScript's <code>Date</code> expects is in <em>milliseconds</em> since the epoch, where Python's API takes a floating point <em>seconds</em> since the epoch):</p>
<pre><code>from datetime import datetime

date = datetime.fromtimestamp(1466278504960 / 1000.)
</code></pre>
<p>That makes the raw <code>datetime</code> object; if you want it formatted the same, you should take a look at <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strftime" rel="nofollow"><code>datetime</code> object's <code>strftime</code> method</a>.</p>
</div>
