<div class="post-text" itemprop="text">
<p>The simple fix is to use <code>used.append(drawn_card)</code> as others have suggested. 
A better way is to randomly shuffle your deck and then slice the required number of cards from it:</p>
<pre><code>CARDS_PER_HAND = 2

random.shuffle(deck)
hand = deck[:CARDS_PER_HAND]
</code></pre>
<p>However, you are probably dealing multiple hands from the same deck. You can handle that by keeping track of the offset into the shuffled deck and then slice from there, but that's a hassle. Or you can make a copy of <code>deck</code>, shuffle it, and pop off the required number of cards for each hand:</p>
<pre><code>shuffled_deck = deck[:]    # make a copy of deck
random.shuffle(shuffled_deck)
hand1 = [shuffled_deck.pop() for i in range(CARDS_PER_HAND)]
hand2 = [shuffled_deck.pop() for i in range(CARDS_PER_HAND)]
... etc.
</code></pre>
<p>Possibly the cleanest way is to use <a href="https://docs.python.org/3/library/itertools.html#itertools.islice" rel="nofollow"><code>itertools.islice()</code></a> to take the first <em>n</em> items from an iterator:</p>
<pre><code>from itertools import islice

random.shuffle(deck)
shuffled_deck = iter(deck)    # returns an iterator
hand1 = list(islice(shuffled_deck, CARDS_PER_HAND))
hand2 = list(islice(shuffled_deck, CARDS_PER_HAND))
... etc.
</code></pre>
