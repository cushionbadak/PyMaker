<p>I'm trying to go through 2 arrays of same length in a <code>for</code> loop, but I'm getting a </p>
<blockquote>
<p>ValueError: too many values to unpack</p>
</blockquote>
<p>Here is the code generating the error :</p>
<pre><code>test = [1, 0, 5, 2, 6, 3, 4]
names = ['patient_map_id', 'birth_date', 'age_in_years_num', 
         'sex_cd', 'vital_status', 'event_map_id', 'start_date']

for col_num, name in (test, names) :
    #do stuff
</code></pre>
<p>If I'm not mistaken, there should be a way to do this but I don't see where the mistake comes from</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip</code></a> the lists before you iterate through them. A <code>for</code> loop cannot iterate through multiple iterables as such.<br/>
The <code>zip</code> function combines all the passed iterables into a single iterator.</p>
<p><strong>Note:</strong> The returned iterator is only as long as the shortest list passed to it.</p>
<pre><code># code

for col_num, name in zip(test, names):
    #do stuff
</code></pre>
