<div class="post-text" itemprop="text">
<p>Using <code>input</code> as a name will shadow the builtin <code>input</code> function, so that's a bad idea.</p>
<p>To keep <code>input_</code> intact, you can modify a copy of it, instead of modifying it directly:</p>
<pre><code>def fun1(input_):

      input_copy = input_[:] # A copy of input via slicing
      input_copy.reverse()
      return input_copy # return the mutated copy of your input
</code></pre>
<p>You can also use the explicit <code>copy</code> function to make a shallow copy:</p>
<pre><code>input_copy = input_.copy()
</code></pre>
<p>Your <code>input_</code> will not be modified and changes on the copy will not propagate to the original <code>input_</code></p>
</div>
<div class="post-text" itemprop="text">
<p>Assign new local variable for the reversed list and return it:</p>
<pre><code>def fun1(input):
    output = input[:]
return output.reverse()
</code></pre>
