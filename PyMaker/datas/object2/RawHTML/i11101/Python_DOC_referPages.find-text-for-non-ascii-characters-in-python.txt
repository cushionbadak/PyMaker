<div class="post-text" itemprop="text">
<p>You can filter out non-printable characters using <a href="https://docs.python.org/3/library/string.html#string.printable" rel="nofollow">string.printable</a>. Example:</p>
<pre><code>&gt;&gt;&gt;import string
&gt;&gt;&gt;something = "(abcd\xe2)"
&gt;&gt;&gt;printable = set(string.printable)
&gt;&gt;&gt;print (''.join(filter(lambda x: x in string.printable, something)))
</code></pre>
