<pre><code>from zipfile import *

<div class="post-text" itemprop="text">
<p>The first problem is that <code>open()</code> refers to the builtin function, not to any function in the <code>zipfile</code> - there is no <code>zipfile.open()</code> function.</p>
<p>To open a zip file use the <a href="https://docs.python.org/3/library/zipfile.html#zipfile.ZipFile" rel="nofollow"><code>zipfile.ZipFile</code></a> class:</p>
<pre><code>import zipfile

z = zipfile.ZipFile('shoppy.zip')
z.extract('cats.txt')
</code></pre>
<p>This will unzip the file into the current directory. If you would prefer to unzip into a string you can use <a href="https://docs.python.org/3/library/zipfile.html#zipfile.ZipFile.read" rel="nofollow"><code>zipfile.read()</code></a>:</p>
<pre><code>content = z.read('cats.txt')
</code></pre>
<p>Now <code>content</code> will contain the unzipped content of the file.</p>
