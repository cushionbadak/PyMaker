<div class="post-text" itemprop="text">
<p>You may want to take a look at the <a href="https://docs.python.org/3/library/array.html" rel="nofollow">documentation regarding arrays</a>. <code>A</code> should be defined this way :</p>
<pre><code>&gt;&gt;&gt; A=[[357,131],[319,171],[229,196],[398,135],[242,148]]
</code></pre>
<p>Your guess was right though, regarding how to access the index of the elements of the array :</p>
<pre><code>&gt;&gt;&gt; A.index([229,196])
2
&gt;&gt;&gt; A.index([242,148])
4
</code></pre>
<p>Be careful, those return 2 and 4, and not 3 and 5 ! Indeed, the index of an array starts at 0, and not 1 ! You can check this by doing <code>A[0]</code>.</p>
<p>Hope it'll be helpful.</p>
