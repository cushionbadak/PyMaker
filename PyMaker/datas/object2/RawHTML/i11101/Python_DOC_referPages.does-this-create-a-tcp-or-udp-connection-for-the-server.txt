<pre><code>from socket import *
host=gethostname()
port=7776
s=socket()
s.bind((host, port))
</code></pre>
<p>Is it UDP or TCP-IP? How do I write this as TCP-IP if it's UDP and vice  versa?</p>
</div>
<div class="post-text" itemprop="text">
<p>This will create a TCP socket because the type argument of <code>socket.socket(...)</code> defaults to <code>SOCK_STREAM</code>. For UDP you would need to use <code>SOCK_DGRAM</code> as type instead. See <a href="https://docs.python.org/3/library/socket.html" rel="nofollow">the extensive documentation</a> for more details.</p>
</div>
