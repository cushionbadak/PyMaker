<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/binascii.html#binascii.hexlify" rel="nofollow"><code>binascii.hexlify</code></a> to get hexa-decimal representation:</p>
<pre><code>&gt;&gt;&gt; import binascii
&gt;&gt;&gt; binascii.hexlify(b' \xc0\xff\xae\x91\x1c\tJ\xc5~\xbe\x8d\x93\xad\xf3\x0c')
b'20c0ffae911c094ac57ebe8d93adf30c'
</code></pre>
<p>and decode it if you want to get string instead of bytes object:</p>
<pre><code>&gt;&gt;&gt; binascii.hexlify(b' \xc0\xff\xae\x91\x1c\tJ\xc5~\xbe\x8d\x93\xad\xf3\x0c').decode()
'20c0ffae911c094ac57ebe8d93adf30c'
</code></pre>
