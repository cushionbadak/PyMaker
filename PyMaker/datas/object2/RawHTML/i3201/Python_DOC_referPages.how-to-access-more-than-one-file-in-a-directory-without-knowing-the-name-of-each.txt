<p>I want to access around 5000 files and work on them one by one. Is there any way to access each in succession without hard-coding the name of each file?</p>
</div>
<div class="post-text" itemprop="text">
<p>The following example is from this <a href="https://www.tutorialspoint.com/python/os_listdir.htm" rel="nofollow noreferrer">tutorial</a>.</p>
<pre><code>import os, sys

# Open a file
path = "/var/www/html/"
dirs = os.listdir( path )

# This would print all the files and directories
for file in dirs:
   print file
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>And if your directory contains parent directories and files, you could use <a href="https://docs.python.org/3/library/os.html#os.walk" rel="nofollow noreferrer">os.walk()</a> like - </p>
<pre><code># Example taken from os.walk documentation
import os
from os.path import join, getsize
for root, dirs, files in os.walk('python/Lib/email'):
    print(root, "consumes", end=" ")
    print(sum(getsize(join(root, name)) for name in files), end=" ")
    print("bytes in", len(files), "non-directory files")
    if 'CVS' in dirs:
        dirs.remove('CVS')
</code></pre>
<p>Or you could use <a href="https://pypi.org/project/scandir/" rel="nofollow noreferrer">scandir</a> -</p>
<pre><code>for entry in os.scandir(path):
    ...
</code></pre>
