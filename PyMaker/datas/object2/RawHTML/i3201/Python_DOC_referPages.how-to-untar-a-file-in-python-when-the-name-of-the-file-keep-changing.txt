<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/glob.html" rel="nofollow noreferrer">glob.glob()</a> to search for files using wildcards:</p>
<pre><code>files = glob.glob('/tmp/*.tar.gz')
for file in files:
    tarfile.open(file, r:gz)
</code></pre>
