<p>The following works fine to drop column names that contain the string <code>basket</code> anywhere in the column name of the <code>df</code>, how can I modify the below code to pass a list of strings to be filtered out instead of just a single string?</p>
<pre><code>banned_columns = ["basket","cricket","ball"]
condition = lambda col: "basket" in col
new_df = df.drop(*filter(condition, df.columns))
</code></pre>
<p>The above just filters <code>basket</code>. How can I filter out <code>basket</code>, <code>cricket</code> and <code>ball</code> from <code>df.columns</code> ?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can check exclude all the columns that contain any of the banned words using the built-in <code>any()</code> <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer">function</a>:</p>
<pre><code>banned_columns = ["basket","cricket","ball"]
condition = lambda col: any(word in col for word in banned_columns)
new_df = df.drop(*filter(condition, df.columns))
</code></pre>
</div>

<pre><code>condition = lambda col: any(item in col for item in banned_columns)
</code></pre>
