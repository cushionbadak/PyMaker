<p>So I have this inventory dict and an empty items list that I want to fill it with the keys in the inventory dict. Any clues on how do I fill the list?</p>
<pre><code># Inventory Dict
inventory = {}
inventory["item a"] = 12
inventory["item b"] = 10
inventory["item x"] = 25
print(inventory)

# Item List
items = []
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>items = list(inventory.keys())
</code></pre>
<p>or in a loop (as requested)</p>
<pre><code>for key in inventory:
    items.append(key)
</code></pre>
<p>though as is there's no need for the loop. Perhaps there would be in another situation</p>
