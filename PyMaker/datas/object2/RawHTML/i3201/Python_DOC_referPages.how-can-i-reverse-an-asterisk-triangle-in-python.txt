<p>Here is a function that can do it either way:</p>
<pre><code>def tristar(n,rev=False):
    for i in range(1,n+1):
        if rev:
            s = ' ' * (n-i) + '*' * i
            print(s)
        else:
            s = '*' * i
            print(s)
</code></pre>

<div class="post-text" itemprop="text">
<p>In case you are not allowed to use <code>rjust</code>:</p>
<pre><code>num = int(input("Enter the number of rows: "))
for i in range(1, num + 1):
    for j in range(num, 0, -1):
        if j &lt;= i:
            print('*', end=' ')
        else:
            print(' ', end=' ')
    print()
</code></pre>
<p>or alternatively:</p>
<pre><code>num = int(input("Enter the number of rows: "))
for i in range(num, 0, -1):
    for j in range(1, num + 1):
        if i &gt; j:
            print(' ', end=' ')
        else:
            print('*', end=' ')
    print()
</code></pre>
