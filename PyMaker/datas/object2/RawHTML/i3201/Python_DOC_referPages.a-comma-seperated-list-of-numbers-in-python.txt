<p>I want to print out from 1 to 1320 like d1,d2,d3,d4 and so on</p>
<p>The command <code>&gt;&gt;&gt; list(range(1320))</code> will print out like 1,2, 3 etc. so I tried with <code>d(list(range(1320)))</code> but it says <code>NameError: name 'd' is not defined</code></p>
<p>So my question id how to print the numbers like d1,d2,d3 ..... </p>
<p>Maybe I am missing very basic syntax here. Any help?</p>

<div class="post-text" itemprop="text">
<p><code>list(range(1320))</code> builds a list out of that range of numbers, and when that list is shown in your Python session, it lists the elements separated by commas. <code>[0, 1, 2, ...]</code></p>
<p>If you want to build a list of <code>d0, d1, d2</code> etc. you can use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>.</p>
<pre><code>&gt;&gt;&gt; ['d%s'%i for i in range(1320)]
['d0', 'd1', 'd2', 'd3', ....
</code></pre>
<p>But if you actually want to just print those strings separated by commas, you can use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>join</code></a>:</p>
<pre><code>&gt;&gt;&gt;&gt; print(', '.join(['d%s'%i for i in range(1320)]))
d0, d1, d2, d3, ...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use list comprehension like this:</p>
<pre><code>['d{}'.format(i) for i in range(1320)]
</code></pre>
<p>of for Python3.6 and higher:</p>
<pre><code>[f'd{i}' for i in range(1320)]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt;&gt;['d'+str(i) for i in range(1320)]
</code></pre>
<hr/>

Just loop till 1230 and concatenate with 'd'.</p>
