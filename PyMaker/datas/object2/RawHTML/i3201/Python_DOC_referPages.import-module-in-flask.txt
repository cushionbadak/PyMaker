<div class="post-text" itemprop="text">
<p>I am using python flask. I want to import a module from a different directory. </p>
<p>I have file called <code>hello.py</code></p>
<p>PATH = project/app/hello.py</p>
<pre><code>app = Flask(__name__)
</code></pre>
<p>I have another file called <code>tables.py</code></p>
<p>PATH = project/db/migration/tables.py</p>
<pre><code>app.config("Database")
</code></pre>
<p>So, I need to import app from hello.py</p>
<pre><code>from app.hello import app
app.config("Database")
</code></pre>
<p>I am executing the script <code>tables.py</code> like, </p>
<pre><code>cd project/db/migration
python3 tables.py
</code></pre>
<p>It is saying <code>no module found app.hello</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You need to create a <code>__init__.py</code> in the <code>app</code> folder to make it become a <a href="https://docs.python.org/3/tutorial/modules.html#packages" rel="nofollow noreferrer">Python package</a>, then you can import a module inside it with <code>from app.hello import app</code>. The file structure may like this:</p>
<pre><code>app / 
    | __init__.py
    | hello.py 
</code></pre>
<p>P.S. The <code>__init__.py</code>'s content can be empty.</p>
</div>
