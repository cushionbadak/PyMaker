<div class="post-text" itemprop="text">
<p>You need to create a <code>__init__.py</code> in a folder to make it become a <a href="https://docs.python.org/3/tutorial/modules.html#packages" rel="nofollow noreferrer">Python package</a>, then you can import modules inside it with <code>from platforms.hitta import Hitta</code>. The file structure may like this:</p>
<pre><code>Project / 
    | app.py
    | platforms / 
        | __init__.py
        | hitta.py
</code></pre>
<p>P.S. The <code>__init__.py</code>'s content can be empty.</p>
