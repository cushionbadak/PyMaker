<div class="post-text" itemprop="text">
<p>If you see here <a href="https://docs.python.org/3/tutorial/datastructures.html#tuples-and-sequences" rel="nofollow noreferrer">"Tuples and Sequences"</a> you can see that your data structure is a tuple. </p>
<p>You can see here <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer">"More on Lists"</a> that <code>sort()</code> is a function only for arrays.</p>
<p>You can use <a href="https://www.pythoncentral.io/how-to-sort-a-list-tuple-or-object-with-sorted-in-python/" rel="nofollow noreferrer">sorted()</a> to sort arrays or tuples</p>
<pre><code>new_list=("a","g","x","e","s","s")

</code></pre>
