<p>I am trying to convert now.month to decimal so it would look like 01 and not like 1. The code is:</p>
<pre><code>import datetime
now = datetime.datetime.now()
monthstr = ".%m" % now.month
</code></pre>
<p>it works in this line with %d</p>
<pre><code>daystr = "file_%d" % now.day
</code></pre>
<p>the error I get is </p>
<blockquote>
<p>ValueError: unsupported format character 'm' (0x6d) at index 2</p>
</blockquote>
<p>I also tried with the new type {:%m}. The error is the same.
Do i miss any libraries? Or is there another way except adding 0 to months from Jan to September</p>
<p>Python v.3.6.3</p>

<div class="post-text" itemprop="text">
<p>You are using string formatting like datetime formatting here.  String formatting has its own <a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="nofollow noreferrer">"mini-language"</a>, and <strong><code>m</code></strong> is not a valid symbol in this language.</p>
<p>For your purposes, it would be better to use <a href="https://docs.python.org/3/library/datetime.html#datetime.date.strftime" rel="nofollow noreferrer"><strong><code>strftime</code></strong></a></p>
<pre><code>&gt;&gt;&gt; now.strftime('.%m')
'.07'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is because <code>%d</code> do not stands for <code>days</code>. It stands for digits and is a placeholder for numbers when using string formatting.</p>
<p>You should use it for months also.</p>
<pre><code>monthstr = ".%d" % now.month
</code></pre>
<p>Since you want the format as <code>01 and not like 1</code>, use @user3483203 <a href="https://stackoverflow.com/a/51326939/7692463">answer</a>.</p>
</div>
