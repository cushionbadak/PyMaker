<div class="post-text" itemprop="text">
<p>If what you have tried is dealing with dates as strings I would recommend using <a href="https://docs.python.org/2/library/datetime.html" rel="nofollow">datetime</a>.</p>
<p>Convert your string dates to datetime objects and then just compare all dates in the list with start and end dates:</p>
<pre><code>from datetime import datetime

start_date = datetime.strptime('1/1/2014', '%m/%d/%Y')
end_date = datetime.strptime('12/31/2014', '%m/%d/%Y')
dates=['4/6/2013', '5/4/2013', '6/26/2013', '7/26/2013', '9/5/2013', '10/7/2013', '10/12/2013', '4/12/2014', '5/10/2014', '6/12/2014', '7/19/2014', '8/15/2014', '9/17/2014', '4/21/2015', '5/28/2015', '6/26/2015']

# this line creates a list of datetime objects from the given strings in list dates
dt_dates = [datetime.strptime(date, '%m/%d/%Y') for date in dates]
</code></pre>
<p>Now compare with start and end:</p>
<pre><code>in_between_dates = []
for d in dt_dates:
    if d &gt;= start_date and d &lt;= end_date:
        in_between_dates.append(d)
</code></pre>
<p>Here I print the output in the same string format as was given:</p>
<pre><code>print [d.strftime('%m/%d/%Y') for d in in_between_dates]
# prints: ['04/12/2014', '05/10/2014', '06/12/2014', '07/19/2014', '08/15/2014', '09/17/2014']
</code></pre>
<p>Two main methods in datetime used here are <a href="https://docs.python.org/2/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow"><code>strptime</code> and <code>strftime</code></a> used to convert a string to a datetime object and to convert datetime to a string, respectively.</p>

<div class="post-text" itemprop="text">
<pre><code>from datetime import datetime as dt  

st = "1/1/2014"
end = "12/31/2014"

# returns True if d1 is smaller than d2
def is_smaller(d1, d2):
  return dt.strptime(d1, "%m/%d/%Y") &lt; dt.strptime(d2, "%m/%d/%Y")

# returns True if date is in between st and end
def filter_func(date):
  return is_smaller(st, date) and is_smaller(date, end)

dates=['4/6/2013', '5/4/2013', '6/26/2013', '7/26/2013', '9/5/2013', '10/7/2013', '10/12/2013', '4/12/2014', '5/10/2014', '6/12/2014', '7/19/2014', '8/15/2014', '9/17/2014', '4/21/2015', '5/28/2015', '6/26/2015']

print(list(filter(filter_func, dates)))
</code></pre>
<p>I have just given you the idea. Now, you can modify this according to your needs. Find more information about <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow">filter</a>.</p>
