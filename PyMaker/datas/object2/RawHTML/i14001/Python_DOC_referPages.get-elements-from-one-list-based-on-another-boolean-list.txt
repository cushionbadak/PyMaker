<div class="post-text" itemprop="text">
<p>I have two lists of the same size, one is a list of strings, and the other a list of booleans (<code>True</code>, <code>False</code>) and I want to return a list of strings only if the index is True.</p>
<pre><code>b_list = [True, False, True]
s_list = ['abc', 'sfsfsfsf', 'def']
</code></pre>
<p>want </p>
<pre><code>s_list = ['abc','def'] 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/itertools.html#itertools.compress" rel="nofollow"><code>itertools.compress</code></a></p>
<blockquote>
<p><em>compress(data, selectors)</em>: Return <em>data</em> elements corresponding to true <em>selectors</em> elements</p>
</blockquote>
<p>So <code>s_list</code> is <em>data</em> and <code>b_list</code> is <em>selectors</em>:</p>
<pre><code>In [8]: import itertools

In [9]: list(itertools.compress(s_list, b_list))
Out[9]: ['abc', 'def']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Without <code>itertools</code> using list comprehension:</p>
<pre><code>[y for (x,y) in zip(b_list, s_list) if x]
</code></pre>
</div>
