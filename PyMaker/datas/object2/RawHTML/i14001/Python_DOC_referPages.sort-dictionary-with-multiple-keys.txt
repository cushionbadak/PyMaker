<p>Using Python 3.4, I am trying to sort a dictionary with two identical entries. 
When I run,</p>
<pre><code>my_dict = {
    '6/18/2015': [6, 'a'],
    '6/19/2015': [18, 't'],
    '6/17/2015': [3, 'r']
    }
for key, value in sorted(my_dict.iteritems(), key=lambda kvt: kvt[1][0]):
        print key, ':', value
</code></pre>
<p>I get a nicely printed result. </p>
<p>But when I use my actual data, which look more like: </p>
<pre><code>my_dict = {
    '6/18/2015': [6, 'a'],
    '6/19/2015': [18, 't'],
    '6/18/2015': [3, 'r'] #note the duplicate date
    }
for key, value in sorted(my_dict.iteritems(), key=lambda kvt: kvt[1][0]):
        print key, ':', value
</code></pre>
<p>the result just drops any repeated rows. </p>
<p>What am I doing wrong?</p>
<p>(Edited to add: thanks all for helping. Correct answer understood. In case anyone else has this problem, my dictionary's keys were actually tuples. Multiple tuples began with the same value and this confused me)</p>

<div class="post-text" itemprop="text">
<p>A dictionary cannot have duplicate keys, but you can create a list to store multiple values for the same key:</p>
<pre><code>my_dict = {
  '6/18/2015': [[6, 'a'], [3,'r']],
  '6/19/2015': [18, 't'],
}
</code></pre>
</div>
