<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html?highlight=items#dict.items" rel="nofollow"><code>dict.items</code></a> returns a tuple of the key value pairs. </p>
<blockquote>
<p>Return a new view of the dictionaryâ€™s items ((key, value) pairs)</p>
</blockquote>
<p>You will have to do this instead. </p>
<pre><code>for key,values in queue_dict.items():
     for v in values:
          print(key," : ",v)
</code></pre>
<p>Also you can not concatenate an <code>str</code> object and a <code>dict</code> object using <code>+</code>. You will get a <code>TypeError</code>. So you will have to cast it to a <code>str</code>. Instead you can use format as in <code>print("{} : {}".format(key,v))</code></p>
