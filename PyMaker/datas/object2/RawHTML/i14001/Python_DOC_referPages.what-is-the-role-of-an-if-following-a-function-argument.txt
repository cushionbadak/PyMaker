<div class="post-text" itemprop="text">
<p>In python, an empty string, or an empty list or set, etc are all <code>False</code> in boolean context, also <code>None</code> is <code>False</code> in boolean context (also number 0 is False) (that is when used in if statement), the check <code>if month:</code> , checks whether month is not <code>None</code> or <code>month</code> is not empty string (I am guessing month is string, as you call <code>capitalize()</code> on it).</p>
<p>Only if month is not empty and month is not None, the statements inside <code>if</code> are executed.</p>
<p>Examples to show this behavior -</p>
<pre><code>&gt;&gt;&gt; s = ''
&gt;&gt;&gt; if not s:
...     print("Blah")
...
Blah
&gt;&gt;&gt; s = 'abcd'
&gt;&gt;&gt; if s:
...     print("Blah1")
...
Blah1
&gt;&gt;&gt; s = ''
&gt;&gt;&gt; if s:
...     print("Will not print")
...
&gt;&gt;&gt; s = None
&gt;&gt;&gt; if s:
...     print("Will not print")
</code></pre>
<p>Reference for Python <a href="https://docs.python.org/2.4/lib/truth.html" rel="nofollow">Truth value testing</a> .</p>
</div>
<div class="post-text" itemprop="text">
<p>The line <code>if month:</code> is a <a href="https://docs.python.org/3/library/stdtypes.html#truth-value-testing" rel="nofollow">Truth Value Test</a>.</p>
<p>This means it checks that the variable <code>month</code> <strong>is not</strong> one of these values:</p>
<blockquote>
<ul>
<li><p><code>None</code></p></li>
<li><p><code>False</code></p></li>
<li><p>zero of any numeric type, for example, <code>0</code>, <code>0.0</code>, <code>0j</code> (and <code>0L</code> if Python 2.7).</p></li>
<li><p>any empty sequence, for example, <code>''</code>, <code>()</code>, <code>[]</code>.</p></li>
<li><p>any empty mapping, for example, <code>{}</code>.</p></li>
<li><p>instances of user-defined classes, if the class defines a <code>__nonzero__()</code> or <code>__len__()</code> method, when that method returns the integer zero or <code>bool</code> value <code>False</code>.</p></li>
</ul>
</blockquote>
</div>

<p>As per your sample program:</p>
<pre><code>def valid_month(month):
    if month:

        if cap_month in months:
            return cap_month

<p><strong>if month is list:</strong></p>
<ol>
<li>A list of months are passed to the function <code>valid_month</code></li>
<li>The first <code>if</code> checks if the month is not a empty list then you have used a function <code>capitalize()</code> on the list and getting a value </li>
<li>In second <code>if</code> you are checking if the value is present in the list and returning it</li>
</ol>
<p><strong>If month is a string:</strong></p>
<ol>
<li>A string month is  passed to the function <code>valid_month</code></li>
<li>The first <code>if</code> checks if the month is not a empty string then you have used a function <code>capitalize()</code> on the string to capitalize the month </li>
<li>In second <code>if</code> you are checking if the capitalized month  is present in the input month and returning it</li>
</ol>
