<p><code>input()</code> ins in build function,</p>
<p>In code you give <strong>variable</strong> name as <code>input</code>, give <strong>different variable name</strong>.</p>

<div class="post-text" itemprop="text">
<p>Your variable name <code>input</code> clashes with the built-in method <code>input</code>. The compiler is getting confused, thinking that the <code>input()</code> call is calling your locally-bound variable, and then complaining that you haven't defined it yet. Changing your variable name to <code>user_input</code> should fix this.</p>
