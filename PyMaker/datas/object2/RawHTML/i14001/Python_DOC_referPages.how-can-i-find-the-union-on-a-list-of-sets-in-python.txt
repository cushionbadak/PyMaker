<p>This is the input:  </p>
<pre><code>x = [{1, 2, 3}, {2, 3, 4}, {3, 4, 5}]
</code></pre>
<p>and the output should be:</p>
<pre><code>{1, 2, 3, 4, 5}
</code></pre>
<p>I tried to use <code>set().union(x)</code> but this is the error I'm getting:</p>
<pre class="lang-none prettyprint-override"><code>Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: unhashable type: 'set'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The signature of <a href="https://docs.python.org/3/library/stdtypes.html#set.union" rel="noreferrer"><code>set.union</code></a> is <code>union(other, ...)</code>. Unpack sets from your list:</p>
<pre><code>In [6]: set.union(*x)
Out[6]: {1, 2, 3, 4, 5}
</code></pre>
</div>
<span class="comment-copy">warning: do not work on empty list</span>
