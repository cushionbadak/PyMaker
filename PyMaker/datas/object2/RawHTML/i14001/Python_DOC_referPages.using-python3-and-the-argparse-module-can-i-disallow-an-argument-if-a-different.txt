<p>For example, say the user passes an argument such as</p>
<pre><code>python3 myscript.py -a
</code></pre>
<p>And I also want them to be able to pass</p>
<pre><code>python3 myscript.py -b
</code></pre>
<p>But I <strong>don't</strong> want them to be able to pass both at the same time such as</p>
<pre><code>python3 myscript.py -a -b
</code></pre>
<p>How would I go about doing this? I'm not too familiar with argparse as of yet and the documentation is kinda tough to comprehend.</p>
<p>Thanks a lot :)</p>
</div>
<div class="post-text" itemprop="text">
<p>You want a <a href="https://docs.python.org/3/library/argparse.html#mutual-exclusion" rel="nofollow">mutually exclusive group</a> of arguments. Here's a minimal example:</p>
<pre><code>import argparse
parser = argparse.ArgumentParser()
group = parser.add_mutually_exclusive_group()
group.add_argument('-a', help='Do A', action='store_true')
group.add_argument('-b', help='Do B', action='store_true')
args = parser.parse_args()
if args.a:
    print('A!')
if args.b:
    print('B!')
</code></pre>
<p>Test:</p>
<pre class="lang-none prettyprint-override"><code>$ python myscript.py -h
usage: myscript.py [-h] [-a | -b]

optional arguments:
  -h, --help  show this help message and exit
  -a          Do A
  -b          Do B
$ python myscript.py -a
A!
$ python myscript.py -b
B!
$ python myscript.py -a -b
usage: myscript.py [-h] [-a | -b]
myscript.py: error: argument -b: not allowed with argument -a
</code></pre>
