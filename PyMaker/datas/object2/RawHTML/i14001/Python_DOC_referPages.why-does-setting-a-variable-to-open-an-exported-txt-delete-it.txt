<div class="post-text" itemprop="text">
<p>If you check the <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow">official docummentation</a> <code>'w'</code> means </p>
<blockquote>
<p>open for writing, truncating the file first</p>
</blockquote>
<p>If you want to keep a current content you can open file in the append mode:</p>
<pre><code>open('/path/to/file', 'at')
</code></pre>
<p>it means </p>
<blockquote>
<p>open for writing, appending to the end of the file if it exists</p>
</blockquote>
</div>
