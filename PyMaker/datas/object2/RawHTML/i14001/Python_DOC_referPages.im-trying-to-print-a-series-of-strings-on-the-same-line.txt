<div class="post-text" itemprop="text">
<p><code>print(t),</code> should work unless <code>t</code> contains a newline. Use <code>print repr(t),</code> to find out. If you now see <code>\n</code>, then the <code>array</code> contains newline characters which you need to remove first.</p>
<p>If the newlines are at the end of the string, you can remove them with <code>t.strip()</code>. If they are in the middle, use <code>t.replace('\n', '')</code></p>
