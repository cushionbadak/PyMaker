<p>I am currently learning list comprehension in Python. How would I do the following:</p>
<pre><code>l1 = [2,4,6,8]
l2 = [2,3,4,5]
l = [*some list comprehension*]
</code></pre>
<p>so that</p>
<pre><code>l = [[2,2],[4,3],[6,4],[8,5]]
</code></pre>
<p>EDIT: Can I do this without zip?</p>
</div>
<div class="post-text" itemprop="text">
<p>You want the <a href="https://docs.python.org/2/library/functions.html#zip" rel="nofollow"><code>zip</code></a> function.</p>
<p>Example -</p>
<pre><code>&gt;&gt;&gt; l1 = [2,4,6,8]
&gt;&gt;&gt; l2 = [2,3,4,5]
&gt;&gt;&gt;
&gt;&gt;&gt; l = list(zip(l1,l2))
&gt;&gt;&gt; l
[(2, 2), (4, 3), (6, 4), (8, 5)]
</code></pre>
<p>If you want the inner lists to be of type <code>list</code> instead of tuple -</p>
<pre><code>&gt;&gt;&gt; l = [list(x) for x in zip(l1,l2)]
&gt;&gt;&gt; l
[[2, 2], [4, 3], [6, 4], [8, 5]]
</code></pre>
<p>In python 3.x, <code>zip</code> returns an iterator, so if you do not want a list, but just want to iterate over each combined (zipped) element, you can just directly use - <code>zip(l1,l2)</code> .</p>
<p>As it is asked in the question, to do it without <code>zip</code> function, you can use <a href="https://docs.python.org/2/library/functions.html#enumerate" rel="nofollow"><code>enumerate</code></a> function to get the index as well as the element from one list and then use the index to get the element from second list.</p>
<pre><code>&gt;&gt;&gt; l = [[x,l2[i]] for i,x in enumerate(l1)]
&gt;&gt;&gt; l
[[2, 2], [4, 3], [6, 4], [8, 5]]
</code></pre>
<p>But this would not work unless both lists have same size.Also not sure why you would want to do it without zip .</p>

<div class="post-text" itemprop="text">
<p>Using list comprehension and <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip</code></a>:</p>
<pre><code>&gt;&gt;&gt; l1 = [2, 4, 6, 8]
&gt;&gt;&gt; l2 = [2, 3, 4, 5]
&gt;&gt;&gt; [[x, y] for x, y in zip(l1, l2)]
[[2, 2], [4, 3], [6, 4], [8, 5]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>zip</code> as</p>
<pre><code>&gt;&gt;&gt; l1 = [2,4,6,8]
&gt;&gt;&gt; l2 = [2,3,4,5]
&gt;&gt;&gt; zip(l1,l2)
[(2, 2), (4, 3), (6, 4), (8, 5)]
&gt;&gt;&gt; [ list(x) for x in zip(l1,l2) ]
[[2, 2], [4, 3], [6, 4], [8, 5]]
</code></pre>
