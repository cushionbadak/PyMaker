<pre><code>test = theList[4:7, 12:18]

<pre><code>test = theList[4,5,6,12,13,14,15,16,17]

<div class="post-text" itemprop="text">
<p>You can add the two lists. </p>
<pre><code>&gt;&gt;&gt; theList = list(range(20))
&gt;&gt;&gt; theList[4:7] + theList[12:18]
[4, 5, 6, 12, 13, 14, 15, 16, 17]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can also use <a href="https://docs.python.org/3/library/itertools.html#module-itertools" rel="nofollow"><code>itertools</code></a> module :</p>
<pre><code>&gt;&gt;&gt; from itertools import islice,chain
&gt;&gt;&gt; theList=range(20)
&gt;&gt;&gt; list(chain.from_iterable(islice(theList,*t) for t in [(4,7),(12,18)]))
[4, 5, 6, 12, 13, 14, 15, 16, 17] 
</code></pre>
<p>Note that since <code>islice</code> returns a generator in each iteration it performs better than list slicing in terms of memory use.</p>
<p>Also you can use a function for more indices and a general way .</p>
<pre><code>&gt;&gt;&gt; def slicer(iterable,*args):
...    return chain.from_iterable(islice(iterable,*i) for i in args)
... 
&gt;&gt;&gt; list(slicer(range(40),(2,8),(10,16),(30,38)))
[2, 3, 4, 5, 6, 7, 10, 11, 12, 13, 14, 15, 30, 31, 32, 33, 34, 35, 36, 37]
</code></pre>
<p>Note : if you want to loop over the result you don't need convert the result to <code>list</code>!</p>
</div>

<p>You can add the two lists as @Bhargav_Rao stated. More generically, you can also use a list generator syntax:</p>
<pre><code>test = [theList[i] for i in range(len(theList)) if 4 &lt;= i &lt;= 7 or 12 &lt;= i &lt;= 18]
</code></pre>
