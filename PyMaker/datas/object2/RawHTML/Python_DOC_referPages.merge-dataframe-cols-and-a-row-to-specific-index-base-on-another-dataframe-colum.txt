<div class="post-text" itemprop="text">
<p>You can use <a href="http://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.merge.html" rel="nofollow noreferrer"><code>DataFrame.merge</code></a> with left join and <code>rename</code> columns:</p>
<pre><code>d = {'UniqueNum':'TestId'}
df = dataframe2.merge(dataframe1.rename(columns=d), how='left', on='TestId')
</code></pre>
<p>Or created index instead <code>rename</code> and change parameters for <code>left_on</code> and <code>right_index</code>:</p>
<pre><code>df = dataframe2.merge(dataframe1.set_index('UniqueNum'), 
                      how='left', 
                      left_on='TestId', 
                      right_index=True)
</code></pre>
<p>Or specified both columns and last remove <code>UniqueNum</code> column:</p>
<pre><code>df = dataframe2.merge(dataframe1, 
                      how='left', 
                      left_on='TestId', 
                      right_on='UniqueNum').drop('UniqueNum', axis=1)
</code></pre>
