<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow"><code>str.replace</code></a> with <code>count</code> argument, you can delete only <em>n</em> occurence of substring.</p>
<pre><code>&gt;&gt;&gt; sentence = 'Hello, This is string.'
&gt;&gt;&gt; sentence = sentence.replace('i', '', 1) # Replace the first occurence of `i`
&gt;&gt;&gt; sentence = sentence[::-1].replace('i', '', 1)[::-1] # reverse/replace/reverse
&gt;&gt;&gt; sentence.swapcase()
'hELLO, tHS IS STRNG.'
</code></pre>
<p>NOTE: <code>str.replace</code> works case-sensitively. Above code only remove lower-case <code>i</code>.</p>
<hr/>
<p>Alternative using <a href="http://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join</code></a>, <a href="http://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>str.split</code></a> and <a href="http://docs.python.org/3/library/stdtypes.html#str.rsplit" rel="nofollow"><code>str.rsplit</code></a>. <code>str.*split</code> accepts optional <code>maxsplit</code> argument similar to <code>str.replace</code>:</p>
<pre><code>&gt;&gt;&gt; sentence = 'Hello, This is string.'
&gt;&gt;&gt; sentence = ''.join(sentence.split('i', 1))
&gt;&gt;&gt; sentence = ''.join(sentence.rsplit('i', 1))
&gt;&gt;&gt; sentence.swapcase()
'hELLO, tHS IS STRNG.'
</code></pre>
