<p>As of this writing, Homebrew installs Python 3.3.3 (<code>$ brew info python3</code>).</p>
<p>And as of Python 3.3, Python's standard library now includes its own virtual environment implementation, and does not require the virtualenv package. See the <a href="http://docs.python.org/3/library/venv.html" rel="nofollow"><code>venv</code> module documentation</a>. With the Homebrew Python 3 package, the command-line tool is named <code>pyvenv-3.3</code>.</p>
<p>I believe using this implementation should resolve the issues you're encountering.</p>
