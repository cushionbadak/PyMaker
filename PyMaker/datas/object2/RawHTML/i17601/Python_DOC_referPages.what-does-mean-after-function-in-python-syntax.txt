<p>What mean in python when i'm after function set <code>[::-1]</code> or <code>[:64]</code> for example? Here is code example</p>
<pre><code>var = binascii.unhexlify(data)[:64]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>binascii.unhexlify</code> returns a string. The brackets are applied to that object, not the function. It is just a short version of</p>
<pre><code>x = binascii.unhexlify(data)
var = x[:64]
</code></pre>
