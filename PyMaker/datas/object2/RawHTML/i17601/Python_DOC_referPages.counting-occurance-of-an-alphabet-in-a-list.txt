<div class="post-text" itemprop="text">
<p>Python has better tools but if you want to keep your code, Just return the len of <code>y</code>:</p>
<pre><code>def countoccur(x,s):
    y=[e for e in s if e==x]
    return len(y)
</code></pre>
<p>Outout:</p>
<pre><code>output: 5
</code></pre>

<div class="post-text" itemprop="text">
<p>You can simply use the builtin <a href="http://docs.python.org/2/library/stdtypes.html#str.count" rel="nofollow"><code>str.count</code></a> function, like this</p>
<pre><code>def countoccur(char, input_string):
    return input_string.count(char)

print countoccur("s", "ssssdfrtsgjdt" )
</code></pre>
<p>I would suggest using a generator expression and <a href="http://docs.python.org/2/library/functions.html#sum" rel="nofollow"><code>sum</code></a> function like this</p>
<pre><code>return sum(1 for e in input_string if e == char)
</code></pre>
<p>This can be shortened to the following, as @volcano suggested in the comments</p>
<pre><code>return sum(e == char for e in input_string)
</code></pre>
<p>This works because, in Python, <code>True == 1</code> and <code>False == 0</code> :)</p>
<p>If you want to use only comprehension,</p>
<pre><code>return sum([e == char for e in input_string])
</code></pre>
</div>
