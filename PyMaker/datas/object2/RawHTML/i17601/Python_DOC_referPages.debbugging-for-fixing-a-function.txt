<div class="post-text" itemprop="text">
<p><a href="http://docs.python.org/3/library/stdtypes.html?highlight=sort#list.sort" rel="nofollow"><code>sort</code></a> returns <code>None</code> because it mutates the list in-place. Try:</p>
<pre><code>def buggy_dedup_sort_by_len(input):
    unique = list(set(input))
    unique.sort(key=len)
    return unique
</code></pre>
<p>Alternatively, use <a href="http://docs.python.org/3/library/functions.html?highlight=sorted#sorted" rel="nofollow"><code>sorted</code></a>, which <em>does</em> return a list.</p>
<pre><code>def buggy_dedup_sort_by_len(input):
    unique = list(set(input))
    return sorted(unique, key=len)
</code></pre>
