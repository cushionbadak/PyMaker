<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/stdtypes.html#str.ljust" rel="nofollow"><code>str.ljust</code></a>:</p>
<pre><code>&gt;&gt;&gt; '// {} {}'.format('USERID:'.ljust(16, '.'), 'ADET_ID')
'// USERID:......... ADET_ID'
</code></pre>
<p>More simply using <a href="http://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format</code></a> with <code>fill</code> character specified (See <a href="http://docs.python.org/3/library/string.html#formatstrings" rel="nofollow">Format string syntax</a>):</p>
<pre><code>&gt;&gt;&gt; '// {:.&lt;16} {}'.format('USERID:', 'ADET_ID')
'// USERID:......... ADET_ID'
</code></pre>
<ul>
<li><code>.</code>: fill character</li>
<li><code>&lt;</code>: left alignment</li>
<li><code>16</code>: width of the result string</li>
</ul>
<p><strong>Why 16?</strong></p>
<p><code>20 - 4</code></p>
<p>4 = 2 (for <code>//</code>) + 1 (space after <code>//</code>) + 1 (space after <code>.</code>)</p>

<p>While you should use @falsetru's answer for any production code, if you're new to programming it's useful to learn how to do it 'manually':</p>
<pre><code>DESIRED_SIZE= 10 #size of the final string we want
s= "USERID"      #our initial string
n= DESIRED_SIZE-len(s) # how many characters must we add
#create a string with n "." characters
#on a lower-level language (like C) you'll need to use a loop to do this
padding= "."*n   
result= s+padding
print result
</code></pre>
<p>this prints <code>USERID....</code></p>
