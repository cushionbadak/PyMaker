<p>I guess what you may be looking for is something like</p>
<pre><code>import webbrowser
url = "file://" + somepath + "/" + "index.html"
webbrowser.open(url)
</code></pre>
<p>See <a href="http://docs.python.org/3/library/webbrowser.html" rel="nofollow">webbrowser</a>.
This will send a command to display the file to your ordinary web browser (Firefox or so).</p>

<p>Please refer <a href="http://docs.python.org/2.7/library/simplehttpserver.html#module-SimpleHTTPServer" rel="nofollow" title="SimpleHTTPServer">SimpleHTTPServer</a> module.</p>
<p>From the python docs, the following should work. It'll automatically look for the index.html file.</p>
<pre><code>import SimpleHTTPServer
import SocketServer

PORT = 80

Handler = SimpleHTTPServer.SimpleHTTPRequestHandler

httpd = SocketServer.TCPServer(("", PORT), Handler)

print "serving at port", PORT
httpd.serve_forever()
</code></pre>
