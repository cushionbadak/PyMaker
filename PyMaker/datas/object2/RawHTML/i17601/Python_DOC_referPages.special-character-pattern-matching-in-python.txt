<p>I have to match this pattern [**] inside a given string but its not giving accurate result.</p>
<pre><code>import re
str1 = '[**] [1:123:43] hello madam [**]'
m1 = re.match('/[/*/*/] .*', str1)
</code></pre>
<p>Please Help!!</p>

<div class="post-text" itemprop="text">
<p>Use backslash(<code>\</code>) to escape meta characters, not forward-slash (<code>/</code>):</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; str1 = '[**] [1:123:43] hello madam [**]'
&gt;&gt;&gt; m1 = re.match(r'\[\*\*\] .*', str1)
&gt;&gt;&gt; m1
&lt;_sre.SRE_Match object at 0x0000000002898100&gt;
&gt;&gt;&gt; m1.group()
'[**] [1:123:43] hello madam [**]'
</code></pre>
<p>BTW, using <code>r'raw string literal'</code>, you don't need to escape backslash itself.</p>
<hr/>
<p>If you just want to check that the string start with <code>[**]</code>, use <a href="http://docs.python.org/3/library/stdtypes.html#str.startswith" rel="nofollow"><code>str.startswith</code></a>:</p>
<pre><code>&gt;&gt;&gt; str1.startswith('[**]')
True
</code></pre>
