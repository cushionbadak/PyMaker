<p>As a starting developer in Python I've seen this error message many times appearing in my console but I don't fully understand what does it means.</p>
<p>Could anyone tell me, in a general way, what kind of action produces this error?</p>
</div>
<div class="post-text" itemprop="text">
<p>That error occurs when you try to call, with <strong><code>()</code></strong>, an object that is not <a href="http://docs.python.org/2/reference/datamodel.html#object.__call__" rel="noreferrer">callable</a>. </p>
<p>A callable object can be a function or a class (that implements <code>__call__</code> method). According to <a href="http://docs.python.org/2/reference/datamodel.html#object.__call__" rel="noreferrer">Python Docs</a>:</p>
<blockquote>
<p><strong>object.__call__(self[, args...])</strong>: Called when the instance is “called” as a function</p>
</blockquote>
<p>For example:</p>
<pre><code>x = 1
print x()
</code></pre>
<p><code>x</code> is not a callable object, but you are trying to call it as if it were it. This example produces the error:</p>
<pre><code>TypeError: 'int' object is not callable
</code></pre>
<hr/>
<p>For better understaing of what is a <strong>callable</strong> object <a href="https://stackoverflow.com/a/111255/2670792">read this answer in another SO post.</a></p>
</div>
<div class="post-text" itemprop="text">
<p>The action occurs when you attempt to call an object which is not a function, as with <code>()</code>. For instance, this will produce the error:</p>
<pre><code>&gt;&gt;&gt; a = 5
&gt;&gt;&gt; a()
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: 'int' object is not callable
</code></pre>
<p>Class instances can also be called if they define a method <code>__call__</code></p>
<p>One common mistake that causes this error is trying to look up a list or dictionary element, but using parentheses instead of square brackets, i.e. <code>(0)</code> instead of <code>[0]</code></p>
</div>
<div class="post-text" itemprop="text">
<p>The other answers detail the reason for the error. A possible cause (to check) may be your class has a variable and method with the same name, which you then call. Python accesses the variable as a callable - with <code>()</code>.</p>
<p>e.g. Class A defines <code>self.a</code> and <code>self.a()</code>:</p>
<pre><code>&gt;&gt;&gt; class A:
...     def __init__(self, val):
...         self.a = val
...     def a(self):
...         return self.a
...
&gt;&gt;&gt; my_a = A(12)
&gt;&gt;&gt; val = my_a.a()
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: 'int' object is not callable
&gt;&gt;&gt;
</code></pre>
</div>

<pre><code>&gt;&gt;&gt; f = 1
&gt;&gt;&gt; callable(f)
False
&gt;&gt;&gt; f()
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: 'int' object is not callable
</code></pre>
