<div class="post-text" itemprop="text">
<p>In your example the for-loop basically loops through the rows of the CSV file. In each loop iteration the variable <code>d</code> gets filled with a <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">dictionary object</a> that contains all data from a single CSV-file row (possibly the name <code>d</code> was meant to be shorthand for 'dictionary'). The keys inside <code>d</code> are column names from the CSV-file, and the paired values are row data values from the CSV-file.</p>
<p>The expression <code>d['Code']</code> returns the data value from dictionary <code>d</code> corresponding to the key <code>'Code'</code>. In other words: it returns the CSV-file row data value found in the <code>'Code'</code> column.</p>
