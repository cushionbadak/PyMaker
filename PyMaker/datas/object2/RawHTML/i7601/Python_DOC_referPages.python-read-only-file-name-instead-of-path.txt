<p>How to get ONLY filename instead of full path?</p>
<p>For example:</p>
<pre><code>path = /folder/file.txt
</code></pre>
<p>and i need to get:</p>
<pre><code>filename = file.txt
</code></pre>
<p>How to do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You should use the <code>os</code> module:</p>
<pre><code>import os
filename = os.path.basename(path)
</code></pre>
<p>For other path manipulations look <a href="https://docs.python.org/2/library/os.path.html" rel="nofollow noreferrer">here</a> (for python 2.7) or <a href="https://docs.python.org/3/library/os.path.html" rel="nofollow noreferrer">here</a> (for python 3)</p>
