<p>Let's say I have a list of tuples currently sorted by the first element : </p>
<pre><code>tuple_list = [(1, "hello", "apple"), (1, "no", "orange"),
    (2, "bye", "grape"), (3, "okay", "banana"),
    (4, "how are you?", "raisin"), (4, "I'm doing well", "watermelon")]
</code></pre>
<p>I want to turn this list of tuples into a list of list of tuples based on the first element of each tuple. </p>
<p>For example:</p>
<pre><code>new_tuple_list = [[(1, "hello", "apple"), (1, "no", "orange")],
    [(2, "bye", "grape")], [(3, "okay", "banana")],
    [(4, "how are you?", "raisin"), (4, "I'm doing well", "watermelon")]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer"><code>itertools.groupby</code></a> to group items based on the integers:</p>
<pre><code>from itertools import groupby

lst = [list(g)for _, g in groupby(tuple_list, lambda x: x[0])]
print(lst)
</code></pre>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer"><code>itertools.groupby</code></a> for this:</p>
<pre><code>[list(x) for k, x in itertools.groupby(tuple_list, operator.itemgetter(0))]
</code></pre>
