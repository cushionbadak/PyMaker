<div class="post-text" itemprop="text">
<p>There's no <code>dict.append()</code>, but you are using <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer"><code>list.append()</code></a>. The problem is that you have</p>
<pre><code>        if type(list2[key]).__name__ == 'dict':
            anArray = getNum(list2[key])
</code></pre>
<p>instead of</p>
<pre><code>        if type(list2[key]).__name__ == 'dict':
            anArray.extend(getNum(list2[key]))
</code></pre>
<p>in your recursive function, so you overwrite the collected values with a new list every time you find a nested dict. Using <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer"><code>list.extend()</code></a> you append all the items found in the recursive call to your list.</p>
<p>Consider using <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer"><code>isinstance()</code></a>:</p>
<pre><code>        if isinstance(list2[key], dict):
</code></pre>
<p>etc. instead of checking the type name.</p>
</div>
<span class="comment-copy">Where do you use <code>dict.append()</code>? There is no <code>append()</code> method on python <code>dict</code>s. You are <code>append</code>-ing your numbers to a <code>list</code> that you call <code>anArray</code>. Make sure you have your concepts and naming right, otherwise it is difficult to discuss, or think about, what you are doing.</span>
