<p>I'm having problems to make some verifications and alterations with the return of a <code>get()</code> method:</p>
<pre class="lang-py prettyprint-override"><code>class Foo(object):
    def __init__(self, turnNum=1):
        self._turnNum = turnNum

    def set_turnNum(self, turnNum):
        self._turnNum = turnNum

    def get_turnNum(self):
        return self._turnNum
</code></pre>

<div class="post-text" itemprop="text">
<p><code>get_turnNum</code> is a method, so if you do <code>num.get_turnNum</code> it will return a bound method. You need to call it <code>num.get_turnNum()</code> - notice the ()</p>
<p>Alternatively, you can use a <a href="http://stackabuse.com/python-properties/" rel="nofollow noreferrer"><code>@property</code></a> decorator.</p>
<pre><code>@property
def turnNum(self):
    return self._turnNum

@turnNum.setter
def turnNum(self, value):
    self._turnNum = value


num = Foo()
num.turnNum = 13
if num.turnNum % 2 == 0:
    print('Working...')
</code></pre>
</div>
<span class="comment-copy">It's not an attribute, it's a method: <code>get_turnNum() </code>. You probably want to look into <a href="https://docs.python.org/3/library/functions.html#property" rel="nofollow noreferrer"><code>@property</code></a> decorator.</span>
