<p>Building on Akshay's answer, you could do:</p>
<pre><code>import re

str1 = '#lol #loalsd asd'
r = re.compile('#(.*?)\s+')
m = r.findall(str1)
if m:
   hashtags = m
   print(hashtags)
</code></pre>
<p>The hashtags is a list, so you can access them accordingly.</p>

<div class="post-text" itemprop="text">
<p>You may use a <em>list comprehension</em> with the <a href="https://docs.python.org/3/library/stdtypes.html#str.startswith" rel="nofollow noreferrer"><code>string.startswith</code></a> filter as:</p>
<pre><code>&gt;&gt;&gt; my_str = 'Gazipur #issue #Garbage #Garbagecollection'
&gt;&gt;&gt; prefix = '#'

#                 filter words starting with prefix v
&gt;&gt;&gt; [word for word in my_str.split() if word.startswith(prefix)]
['#issue', '#Garbage', '#Garbagecollection']
</code></pre>
<p>If you want to remove <em>prefix</em> from each word, you may slice the each string within your <em>list comprehension</em> expression as:</p>
<pre><code>#           v slice string to remove the `prefix` 
&gt;&gt;&gt; [word[len(prefix):] for word in my_str.split() if word.startswith(prefix)]
['issue', 'Garbage', 'Garbagecollection']
</code></pre>
</div>
