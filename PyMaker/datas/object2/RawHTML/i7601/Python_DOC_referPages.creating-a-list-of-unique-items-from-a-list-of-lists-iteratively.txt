<p>With the restrictions you mentioned this code should work - </p>
<pre><code>def unique(lists):
    uniquelst = []
    for lst in lists:
        for elem in lst:
            if elem not in uniquelst:
                uniquelst.append(elem)
                break

    return uniquelst
</code></pre>
<p>The second loop iterates over elements of the inner list and check if they are already present in <code>uniquelst</code>. In case they are not present, add the element and break from inner loop.</p>
<p>One thing I forgot to mention is that this code assumes the elements of inner list are in sorted order. If that is not the case, please sort inner list using <code>sorted()</code> or <code>sort()</code> before the second loop</p>
