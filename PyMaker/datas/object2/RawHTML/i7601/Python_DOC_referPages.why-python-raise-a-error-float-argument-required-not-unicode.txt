<div class="post-text" itemprop="text">
<p>You have <em>loads</em> of <code>%</code> formatters in your string. <code>%E</code> formats a float object. You have several of those in your string, including at the start:</p>
<pre><code>"http://www.zjjsggzy.gov.cn/%E6
#                           ^^
</code></pre>
<p>You'd need to double up <em>every single <code>%</code> used in a URL character escape</em>:</p>
<pre><code>"http://www.zjjsggzy.gov.cn/%%E6%%96%%B0%%E6%%B5%%81%%E7%%A8%%8B/..."
</code></pre>
<p>That'd be a lot of work, you'd be better off using a different string formatting style. Use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>str.format()</code></a>:</p>
<pre><code>url = (
    "http://www.zjjsggzy.gov.cn/"
    "%E6%96%B0%E6%B5%81%E7%A8%8B/%E6%8B%9B%E6%8A%95%E6%A0%87%E4%BF%A1%E6%81%AF"
    "/jyxx_1.html?iq=x&amp;type=%E6%8B%9B%E6%A0%87%E5%85%AC%E5%91%8A&amp;"
    "tpid={}&amp;tpTitle={}".format(
        each_ID, each_Title)
)
</code></pre>
<p>I broke the string up into multiple chunks to make it easier to read; the <code>{}</code> brackets delineate the placeholders.</p>
</div>
