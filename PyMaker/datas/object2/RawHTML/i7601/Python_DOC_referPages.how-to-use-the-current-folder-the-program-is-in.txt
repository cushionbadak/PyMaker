<div class="post-text" itemprop="text">
<p><code>'.'</code> stands for the current directory.</p>
</div>
<div class="post-text" itemprop="text">
<p>this is a <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow noreferrer"><code>pathlib</code></a> version:</p>
<pre><code>from pathlib import Path
HERE = Path(__file__).parent
source = list((HERE /  "../mydirectory").iterdir())
</code></pre>
<p>if you prefer <a href="https://docs.python.org/3/library/os.path.html#os.path.dirname" rel="nofollow noreferrer"><code>os.path</code></a>:</p>
<pre><code>import os.path
HERE = os.path.dirname(__file__)
source = os.listdir(os.path.join(HERE, "../mydirectory"))
</code></pre>
<p>note: this will often be different from the current working directory</p>
<pre><code>os.getcwd()  #  or '.'
</code></pre>
<p><a href="https://docs.python.org/3/reference/import.html#import-related-module-attributes" rel="nofollow noreferrer"><code>__file__</code></a> is the filename of your current python file. <code>HERE</code> is now the path of the directory where your python file lives.</p>
</div>
<div class="post-text" itemprop="text">
<p>try:</p>
<pre><code>os.listdir('./')
</code></pre>
<p>or:</p>
<pre><code>os.listdir(os.getcwd())
</code></pre>
