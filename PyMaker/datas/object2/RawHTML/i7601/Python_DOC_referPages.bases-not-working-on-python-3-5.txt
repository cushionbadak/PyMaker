<p>i've implemented a small code following a book for py33 in py35. I am trying to get the super classes instances from a given subclass as follows:</p>
<pre><code>class Super:
    def hello(self):
        self.data1 = 'spam'

class Sub(Super):
    def hola(self):
        self.data2 = 'eggs'

X = Sub()
X.__dict__
X.__class__
X.__bases__
</code></pre>
<p>However, when i excecute the <code>X.__bases__</code> command i get an error:</p>
<pre><code>AttributeError: 'Sub' object has no attribute '__bases__'
</code></pre>
<p>Could someone explain to me why is this happening and how can I (if not so) get the instances of the superclasses of a given class?</p>
<p>Thanks to you all :)</p>

<div class="post-text" itemprop="text">
<p>You can get it from the class, not the instance, see the <a href="https://docs.python.org/3/library/stdtypes.html#class.__bases__" rel="nofollow noreferrer">documentation of <code>__bases__</code></a>:</p>
<blockquote>
<p><code>class.__bases__</code></p>
<p>The tuple of base classes of a class object.</p>
</blockquote>
<p>so, in short, use <code>type(X).__bases__</code>.</p>
</div>
