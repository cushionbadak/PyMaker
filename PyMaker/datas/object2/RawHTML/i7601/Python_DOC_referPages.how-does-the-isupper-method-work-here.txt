<div class="post-text" itemprop="text">
<p>Why do I get <code>False</code> when I do:</p>
<pre><code>&gt;&gt;&gt; "123".isupper()
False
</code></pre>
<p>But <code>True</code> when I do:</p>
<pre><code>&gt;&gt;&gt; "ABC123".isupper()
True
</code></pre>
<p>Is this documented somewhere?</p>
</div>
<div class="post-text" itemprop="text">
<p>Per <a href="https://docs.python.org/3/library/stdtypes.html?highlight=isupper#str.isupper" rel="nofollow noreferrer">the Python 3.6 documentation</a>:</p>
<blockquote>
<p><strong><code>str.isupper()</code></strong></p>
<p>Return true if all cased characters <a href="https://docs.python.org/3/library/stdtypes.html?highlight=isupper#id15" rel="nofollow noreferrer">[4]</a> in the string are uppercase and there is at least one cased character, false otherwise.</p>
</blockquote>
<p>Since digits are not <em>cased characters</em>, there is not at least one cased character thus it returns false. In the second example, there is at least one cased character and they are all uppercase, thus it is true.</p>
</div>
