<div class="post-text" itemprop="text">
<p>This is what you were looking for <code>re.sub</code></p>
<pre><code>import re
mytext = "@ match everything upto"
myex = re.compile(r"(@|everything)")
final_str = re.sub(myex, '',mytext)
print(final_str)
</code></pre>
<p>output:</p>
<pre><code> match  upto
</code></pre>
<p>Explanations:</p>
<pre><code>re.compile(r"(@|everything)") 
</code></pre>
<p>The above <code>re</code> just means <code>@</code> or <code>everything</code>. And <code>re.sub()</code> takes three arguments <code>re.sub(pattern,replacement_string,string_to_be_searched)</code></p>
<p>So in your case : <code>re.sub(myex, '',mytext)</code> where replacement string is an empty string <code>''</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You can achieve  the same using <code>list comprehensions</code></p>
<pre><code>mytext = "@ match everything upto"
r_items=['@','everything']

mytext = [x for x in mytext.split() if x not in r_items]


print(' '.join(mytext))
</code></pre>
<p>output:</p>
<pre><code>match upto
</code></pre>
