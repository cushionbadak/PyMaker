<div class="post-text" itemprop="text">
<p>I import a class <code>Foo</code> that has a default parameter <code>dir</code> upon which if performs a function <code>doit</code>. How can I change the default <code>dir</code>? Do I need to inherit this class and then change it, how?</p>
<pre><code>class Foo(object):
  def __init__(self, dir='xxx'):
    self.bar = doit(dir) # fails because xxx is wrong
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>why don't you just provide a different argument when you construct an instance of the class:</p>
<pre><code>foo = Foo(dir='something else')
</code></pre>
<p>btw: <a href="https://docs.python.org/3/library/functions.html#dir" rel="nofollow noreferrer"><code>dir</code></a> is a python built-in and therefore not the best choice as variable name.</p>
<p>if you want the default changed, you can inherit indeed:</p>
<pre><code>class MyFoo(Foo):
    def __init__(self, d='somethig else'):
        super().__init__(d=d)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just create a factory function for <code>Foo</code> objects and be done with it:</p>
<pre><code>def create_foo():
    return Foo(dir='my correct value goes here')
</code></pre>
<p>Since you're importing <code>Foo</code>, you could go a step further and just shadow <code>Foo</code> like this:</p>
<pre><code>def Foo():
    import foo
    return foo.Foo(dir='my correct value goes here')
</code></pre>
<p>Of course you <em>can</em> inherit from <code>Foo</code>. Be sure to look up how to call the base
class constructors. I find that sooo hard to memorize that I just end up google it. Every. Single. Time.</p>
