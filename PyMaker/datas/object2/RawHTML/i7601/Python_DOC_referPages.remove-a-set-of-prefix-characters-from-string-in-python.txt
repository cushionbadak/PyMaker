<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#str.lstrip" rel="noreferrer"><code>lstrip()</code></a></p>
<blockquote>
<p>Return a copy of the string with leading characters removed. The <em>chars</em> argument is a string specifying the set of characters to be removed.</p>
</blockquote>
<pre><code>&gt;&gt;&gt; "@@@@b@@".lstrip("@")
'b@@'
</code></pre>
