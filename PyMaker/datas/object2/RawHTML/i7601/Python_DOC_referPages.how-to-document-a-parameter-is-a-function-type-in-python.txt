<p>You can use the "callable" type.</p>
<pre><code>def method(*, var_param, function_param):
    """Call 'function_param' with 'var_param'

    :param var_param: the value to pass to the function_param
    :type log: str
    :param function_param: the method to call with var_param
    :type function_param: callable[str]
    :return: Nothing
    :rtype: None
    """
    function_param(var_param)
</code></pre>
<p>Since your parameter is a <code>str</code>, you can set this in the function signature.</p>
