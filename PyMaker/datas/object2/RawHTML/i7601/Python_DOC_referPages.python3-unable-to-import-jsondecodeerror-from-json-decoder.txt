<div class="post-text" itemprop="text">
<p>According to <a href="https://docs.python.org/3.4/library/json.html" rel="noreferrer">3.4.x docs</a>, plain <code>ValueError</code> is raised when JSON decoding fails.</p>
<p><code>JSONDecodeError</code> class is available starting from <a href="https://docs.python.org/3.5/library/json.html#json.JSONDecodeError" rel="noreferrer">3.5.x</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>According to Docs from <a href="https://docs.python.org/3/library/json.html" rel="nofollow noreferrer">module json (Python version &gt;= 3.5.0)</a>, Python which version &lt; 3.5.0 does not support import statement like what you just did, but if you use Python(version&gt;=3.5.0), your import statement is definitely correct.</p>
</div>
<div class="post-text" itemprop="text">
<p>json is the version of simplejson that has been integrated into Python. They have since been developed separately and are not the same anymore. So they cannot necessarily be used interchangably.</p>
<p>See <a href="https://stackoverflow.com/questions/712791/what-are-the-differences-between-json-and-simplejson-python-modules">this answer</a> for more details about the differences.</p>
