<p>Here is my script:</p>
<pre><code>import re
_text = "sdiskpart(device='D:\\', mountpoint='D:\\', fstype='FAT32', opts='rw,fixed')"
print(re.findall("device=(\\'.*\\')", _text))
</code></pre>
<p>I'm trying to get the value of device, in this string it's "D:\"</p>

<blockquote>
<p>["'D:\', mountpoint='D:\', fstype='FAT32', opts='rw,fixed'"]</p>
</blockquote>

<div class="post-text" itemprop="text">
<p>You can use non-eager regexp</p>
<pre><code>import re

print( re.findall("device='(.*?)'", _text))
</code></pre>
<p>notice that the .*? means non-eager so it will take the least chars until the next ' ...</p>
</div>

<pre><code>&gt;&gt;&gt; print(re.findall("device=(\\'[A-Z]:\\\\')", _text))
["'D:\\'"]
</code></pre>
<p>You might have to replace * with [A-Z]. I think the drive letter are in caps always else use[A-Za-z]</p>
