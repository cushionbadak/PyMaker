<p>This code uses <a href="https://pandas.pydata.org/pandas-docs/version/0.22/generated/pandas.read_fwf.html" rel="nofollow noreferrer">Pandas.read_fwf()</a> to read the file by position, skips the first 6 rows, remove the blank lines and fill the Station Name for the empty rows with <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.fillna.html" rel="nofollow noreferrer">fill_na()</a>:</p>
<pre><code>cols = [(3,23),(24,30),(30,45),(46,62),(63,76),(77,90), (91,102), (103, 109), (110,121), (122,128)]
