<p>How can I replace <code>"True"</code> with the boolean <code>True</code> or <code>1</code>?</p>
<pre><code>mylist = ["Saturday", "True"]
</code></pre>
<p>I've tried replace, but get the error: </p>
<pre><code>TypeError: replace() argument 2 must be str, not bool
</code></pre>
<p>Thanks in advance!</p>
</div>
<div class="post-text" itemprop="text">
<p>Using list comprehension:</p>
<pre><code>result = [ True if x == "True" else x for x in mylist  ]
</code></pre>
<p><strong>Edited due OP comment</strong></p>
<p>You can use a dictionary to transform some values:</p>
<pre><code>&gt;&gt;&gt; changes = { "True": True,
...             "False": False,
...             "Lemon": "Gin Lemon"
...           }
&gt;&gt;&gt; 
&gt;&gt;&gt; mylist = ["Saturday", "True", "False", "Lemon", "Others" ]
&gt;&gt;&gt; 
&gt;&gt;&gt; [ changes.get( x, x ) for x in mylist ]

['Saturday', True, False, 'Gin Lemon', 'Others']
</code></pre>
<p>The tip is to use <a href="https://docs.python.org/3/library/stdtypes.html#dict.get" rel="nofollow noreferrer">python dictionary <code>get</code> method</a>  with a default value:</p>
<blockquote>
<p>Return the value for key if key is in the dictionary, else default. If default is not given, it defaults to None, so that this method never raises a KeyError.</p>
</blockquote>
