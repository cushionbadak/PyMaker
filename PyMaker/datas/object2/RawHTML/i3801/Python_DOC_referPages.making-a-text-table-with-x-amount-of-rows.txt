<p>as @NinjaWarrior said you need to use formats</p>
<pre><code>score_board = [["Charles",32,432,23],["Mathew",432,334,32]]
round_number = 3
print("|   Name   "  + ''.join(["| Round %02d " % round_iter for round_iter in range(1,round_number+1)]))
for score in score_board:
    print( ''.join(['|%-10s' % item for item in map(str, score)]))
</code></pre>
<p>which gives</p>
<pre><code>|   Name   | Round 01 | Round 02 | Round 03 
|Charles   |32        |432       |23        
|Mathew    |432       |334       |32   
</code></pre>
<hr/>
<p>the <code>printf</code> format is described <a href="https://docs.python.org/3/library/stdtypes.html#old-string-formatting" rel="nofollow noreferrer">here</a></p>
<p><code>"%s" % "robert"</code> outputs <code>"robert"</code></p>
<p><code>"|%s" % "robert"</code> outputs <code>"|robert"</code></p>
<p><code>"|%10s" % "robert"</code> pads s to the left to form a 10-length string: outputs <code>"|    robert"</code></p>
<p><code>"|%-10s" % "robert"</code> pads s to the right to form a 10-length string: outputs <code>"|robert    "</code></p>
<p>it's the same logic for numbers:</p>
<p><code>"%d" % 2</code> outputs 2</p>
<p><code>"%02d" % 2</code> outputs 02</p>
<p><code>"%03d" % 2</code> outputs 002</p>
</div>
