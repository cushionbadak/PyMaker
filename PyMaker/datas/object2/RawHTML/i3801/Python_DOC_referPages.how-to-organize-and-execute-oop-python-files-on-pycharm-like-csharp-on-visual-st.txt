<p>You can see below the standard call from file</p>
<p>1st step - create a new file for example: <code>funcs.py</code></p>
<p>with:</p>
<pre><code>def first():
  print('hey I'm the first one')
</code></pre>
<p>2nd step - create a new python file in the same folder like <code>menu.py</code></p>
<p>and you can call the <code>funcs.py</code> from <code>menu.py</code> like this:</p>
<pre><code>import funcs.py

funcs.first()
</code></pre>
