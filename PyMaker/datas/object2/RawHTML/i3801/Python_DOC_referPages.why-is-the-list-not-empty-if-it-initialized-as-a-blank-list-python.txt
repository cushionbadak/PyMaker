<p>You append each item to <code>counter_list</code> if it's longer than the previous one.</p>
<p>instead - only append the longest item of the row:</p>
<pre><code>def printTable(items):
    counter_list = []
    for i in  range (len(items[0])):
        print ()
        counter = 0
        for j in range(len(items)):
            if len(items[i][j]) &gt; counter:
                counter = len(items[i][j])
                itemName = items[i][j]
        counter_list.append(counter)
        print ('the longest string is: ' + itemName + '; and its length is ' + str(counter))
        print ('Counter list: ' + str(counter_list))
</code></pre>
<p>Also: instead of <code>range(len(items))</code> have a look at <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a>.</p>
<p>Actually there is no need to uses item indexes at all though.
Here is slightly refactored version:</p>
<pre><code>def printTable(items):
    counter_list = []
    for row in items:
        longest_item, longest_len = '', 0
        for item in row:
            if len(item) &gt; longest_len:
                longest_item, longest_len = item, len(item)
        counter_list.append(longest_len)
        print (f'the longest string is: {longest_item}; and its length is {longest_len}')
        print ('Counter list: ' + str(counter_list))
</code></pre>
<p>You might also want to have a look at the wonderful <a href="https://docs.python.org/2/library/collections.html#collections.Counter" rel="nofollow noreferrer">Counter</a> module:</p>
<pre><code>from collections import Counter
def printTable(tableData):
    counter_list = []
    for row in tableData:
        c = Counter({item:len(item) for item in row})
        item, item_len = next(iter(c.most_common()))
        counter_list.append(item_len)
        print(f'the longest string is: {item}; and its length is {item_len}')
        print(f'Counter list: {counter_list}')
</code></pre>
