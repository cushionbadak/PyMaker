<p>Add a variable that keeps track of the number of iterations currently passed:</p>
<pre><code>f= open('guru99.txt','r')
lines = f.readlines()
i =1
for line in lines:
    print([i],line)
    i = i+1
count = 0
action = int(input('Enter line number'))
if action == 1:
    for line in lines:
        count += 1
        print(count)
        if 'banana' in line:
            print(line)
            print('You already purchased banana! You can change the quantity')
            edit = line.split()
            qty = int(input('Enter new quanity'))
            edit[-1] = (int(edit[-1]) / float(edit[3]))
            bprice = (edit[-1])
            edit[3] = str(qty)
            edit[-1] = str(int(qty * bprice))
            bline = ' '.join(edit)
            print(bline)
        else:

<div class="post-text" itemprop="text">
<p>You can try replacing it with a while loop:</p>
<pre><code>i = 0
while i &lt; len(lines):
   # Your code here
   i = i + 1
</code></pre>
<p><code>i</code> is equal to the iteration and <code>lines[i]</code> is equal to the current line.</p>
