<div class="post-text" itemprop="text">
<p>I like to know how to use split function to count a total from list.</p>
<p>For example,</p>
<pre><code>workdays = {'work': '5,6,8,10,13,14,15,18,20,22,24,25,28,30'}
</code></pre>
<p>Output should be like this</p>
<p>I have worked 14 days.</p>
</div>
<div class="post-text" itemprop="text">
<p>That's not a list. You are using a dictionary with key and value. Get the value, <code>split</code> on comma and find length using <code>len</code>.</p>
<pre><code>workdays = {'work': '5,6,8,10,13,14,15,18,20,22,24,25,28,30'}

print('I have worked {} days'.format(len(workdays['work'].split(','))))
</code></pre>
<p>Also, you could count the number of commas and add <code>1</code> to it to get the same result like so:</p>
<pre><code>print('I have worked {} days'.format(workdays['work'].count(',')+1))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Hint: You can access a value in a dictionary by its key:</p>
<pre><code>&gt;&gt;&gt; workdays = {'work':'1,2,3,4'}
&gt;&gt;&gt; workdays['work']
'1,2,3,4'
</code></pre>
<p>Second hint: You can split a string using <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>str.split(delimiter)</code></a> like so:</p>
<pre><code>&gt;&gt;&gt; s = '1,2,3,4'
&gt;&gt;&gt; s.split(',')
['1', '2', '3', '4']
</code></pre>
<p>Third hint: <a href="https://docs.python.org/3.6/library/functions.html#len" rel="nofollow noreferrer"><code>len()</code></a></p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>str.split</code> with <code>len</code></p>
<p><strong>Ex:</strong></p>
<pre><code>workdays = {'work': '5,6,8,10,13,14,15,18,20,22,24,25,28,30'}
print(len(workdays["work"].split(",")))
</code></pre>
<p><strong>Output</strong>:</p>
<pre><code>14
</code></pre>
</div>
