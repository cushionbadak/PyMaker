import sys

<div class="post-text" itemprop="text">
<p>A few things are wrong with your program.</p>
<ol>
<li><p>It overwrites the input by using <code>N</code> as your loop variable.</p></li>
<li><p>The loop's <code>range</code> does not depend on the input.</p></li>
<li><p>The multiplied value is always <code>4</code> instead of being incremented from <code>1</code> to <code>N</code> in your loop.</p></li>
</ol>
<p>Here is the correct way to implement the factorial:</p>
<pre><code># Get N from the command line
import sys
n = int(sys.argv[1])

result = 1

for i in range(1, n + 1):
    result *= i

print(result)
</code></pre>
<p>Or alternatively, use the <code>math</code> standard library.</p>
<pre><code>import math

n = int(sys.argv[1])

result = math.factorial(n)

print(result)
</code></pre>

<div class="post-text" itemprop="text">
<p>You inquire <code>N</code> from sys.argv and then overwrite it within your loop. That's one problem.</p>
<p>Next: range(a,b) will create integers from a to b-1 (assuming stepsize is 1). So <code>range(1,4)</code> will create <code>[1,2,3]</code>.</p>
<p>Your code does the following:
It changes N to 1*4, then to 2*4, then to 3*4 (which is 12, not 16).</p>
<p>If you want to code a factorial the hard way, you need a seperated iterator (usually indicated as <code>i</code>) and multiply all the way up to <code>N</code> times.</p>
