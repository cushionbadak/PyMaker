<div class="post-text" itemprop="text">
<p>I have a list of tuples in Python and I would like to sort them based on the second value of the tuple (which is a string so I would like to sort them in alphabetical order) and swap the first values of the tuples. For example let's say that we have some list:</p>
<pre><code>lst = [(1, "z"), (2, "a"), (3, "b")]
</code></pre>
<p>I would like the final result to be like this:</p>
<pre><code>result = [(1, "a"), (2, "b"), (3, "z")]
</code></pre>
<p>Is this possible to do with a built in method or I have to write my own function?</p>
</div>
<div class="post-text" itemprop="text">
<p>Split the list into two with the old <code>zip(*lst)</code> transpose trick, then sort the 2nd list and <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> them back together:</p>
<pre><code>lst = [(1, "z"), (2, "a"), (3, "b")]

lst1, lst2 = zip(*lst)
result = list(zip(lst1, sorted(lst2)))

# lst1: (1, 2, 3)
# lst2: ('z', 'a', 'b')
# result: [(1, 'a'), (2, 'b'), (3, 'z')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>zip</code>:</p>
<pre><code>lst = [(1, "z"), (2, "a"), (3, "b")]
sort_l = [(a, c) for [a,_ ], c in zip(lst, sorted(b for _, b in lst))]
</code></pre>
<p>Output:</p>
<pre><code>[(1, 'a'), (2, 'b'), (3, 'z')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Here is a functional solution:</p>
<pre><code>from operator import itemgetter

lst = [(1, "z"), (2, "a"), (3, "b")]

res = list(zip(map(itemgetter(0), lst), sorted(map(itemgetter(1), lst))))

print(res)

[(1, 'a'), (2, 'b'), (3, 'z')]
</code></pre>
<p>Note you should not named variables after built-ins: use <code>lst</code> instead of <code>list</code>.</p>
</div>
