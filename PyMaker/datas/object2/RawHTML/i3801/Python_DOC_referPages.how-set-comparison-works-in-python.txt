<p><strong>Code:</strong></p>
<pre><code> x = {3, 7}    
 y = {1, 3, 7, 14}     
 a = x &lt;= y
</code></pre>
<p>The answer given to this question is a boolean <code>True</code>. Can somebody explain how they came to that conclusion?</p>
</div>
<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/stdtypes.html#frozenset.issubset" rel="nofollow noreferrer">documentation for sets</a> explains this:</p>
<blockquote>
<p><code>set &lt;= other</code></p>
<p>Test whether every element in the set is in other.</p>
</blockquote>
<p>So, since every element in x is also in y, the comparison is true.</p>
</div>
<span class="comment-copy"><code>&lt;=</code> is a conditional statement which is suppose to return boolean value. what other answer do you expect?</span>
<span class="comment-copy"><code>x</code> is a <b>subset</b> of <code>y</code>, so <code>x &lt;= y</code> returns <code>True</code> which is then assigned to <code>a</code></span>
