<p>The title pretty much explains what I am after.  However, I would like to note that the sub directories won't contain any directories only .JPG files. Essentially, just moving everything up one level in the file tree.</p>
<p>For example, <code>~/someDir/folder1</code> , <code>~/someDir/folder2</code> , ... , <code>~/someDir/folderN</code> and I want all of the contents of the sub directories brought up to <code>~/someDir/</code></p>

<div class="post-text" itemprop="text">
<p>shutil.move is a good option to move files.</p>
<pre><code>import shutil
import os

source = "/parent/subdir"
destination = "/parent/"
files_list = os.listdir(source)
for files in files_list:
    shutil.move(files, destination)
</code></pre>
<p>For Recursive move, you can try <code>shutil.copytree(SOURCE, DESTINATION)</code>. it just copies all files and if needed you can manually cleanup the source directory.</p>
</div>
