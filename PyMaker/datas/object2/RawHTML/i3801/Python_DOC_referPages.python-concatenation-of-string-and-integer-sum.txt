<p>I want to print sum of numbers and some string like:</p>
<pre><code>print("root: " + rootLeaf + " left:" + leftLeaf + "sum: " +(rootLeaf+leftLeaf) )
</code></pre>
<p>here "root", "left" and "sum" is string where as rootLeaf and leftleaf are integers and i want to find their sum.</p>
<p>I checked the <a href="https://stackoverflow.com/questions/2847386/python-string-and-integer-concatenation">post here</a> but i couldn't achieve the sum of the integers 
(math operation in string concatination)</p>

<div class="post-text" itemprop="text">
<pre><code>rootLeaf = 4
leftLeaf = 8
print("root: " + str(rootLeaf)+ " left: " + str(leftLeaf) + " sum: {}".format(rootLeaf + leftLeaf))
</code></pre>
<p>Here is your output:</p>
<pre><code>root: 4 left: 8 sum: 12
</code></pre>
<p>In Python, in order to print integers, they must first be converted to strings. The .format method allows you to convert the integer argument (the sum of the two leafs) to strings. You enter a place holder where you want the string {}, and then in your .format method you can specify what that integer will be. In this case, rootLeaf + leftLeaf.</p>
</div>
<div class="post-text" itemprop="text">
<p>Using format should handle all types easily.</p>
<pre><code>print("root: {} left: {} sum: {}".format(rootLeaf, rootRight, rootLeaf + rootRight))
</code></pre>
<p>And in general, look in this site for many more details: <a href="https://pyformat.info/" rel="nofollow noreferrer">https://pyformat.info/</a></p>

<div class="post-text" itemprop="text">
<p>You are getting a TypeError because you are concatenating a string with integers. You need first to convert the integers to strings. You can do this using the str function.</p>
<p>In this case</p>
<pre><code>print("root: " + str(rootLeaf) + " left:" + str(leftLeaf) + "sum: " +str(rootLeaf+leftLeaf))
</code></pre>
<p>will print what you want.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you are using Python 3, you can try using <code>.format</code>:</p>
<hr/>
<pre><code>print("root: {0} left: {1} sum: {2}".format(rootLeaf, leftLeaf, rootLeaf+leftLeaf))
</code></pre>
<p>Or:</p>
<hr/>
<pre><code>print("root: {}".format(rootLeaf), "left: {}".format(leftLeaf) , "sum: {}".format(rootLeaf+leftLeaf))
</code></pre>
<p>For older version of Python, you can try using <code>%</code>:</p>
<hr/>
<pre><code># '+' can be replaced by comma as well
print("root: %s" % rootLeaf + " left: %s" % leftLeaf + " sum: %s" %(rootLeaf+leftLeaf)) 
</code></pre>
<p>Or:</p>
<hr/>
<pre><code>print("root: %s left: %s sum: %s" % (rootLeaf, leftLeaf, rootLeaf+leftLeaf))
</code></pre>
