<div class="post-text" itemprop="text">
<p>I am trying to split a string, but keep all separators bundled together in a separate list.</p>
<pre><code>s = "This is a test     for \n a string"
</code></pre>
<p>should results in</p>
<pre><code>a = ["This", "is", "a", "test", "for", "a", "string"]
b = [" ", " ", " ", "     ", " \n ", " "]
</code></pre>
<p>Any idea on how to handle that?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow noreferrer"><code>re.split</code></a> is your friend:</p>
<pre><code>split = re.split(r'(\s+)', s)   
x = split[::2]
y = split[1::2]
</code></pre>
<p></p>
<pre><code>&gt;&gt;&gt; x
['This', 'is', 'a', 'test', 'for', 'a', 'string']
&gt;&gt;&gt; y
[' ', ' ', ' ', '     ', ' \n ', ' ']
</code></pre>
</div>
