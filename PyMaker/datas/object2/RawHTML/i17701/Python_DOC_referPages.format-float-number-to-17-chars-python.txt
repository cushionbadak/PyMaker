<p>I'm new to Python, and I'm struggling to format a number to 17 chars including the decimal point.</p>
<p>example:</p>
<pre><code>100.00
</code></pre>
<p>result:</p>
<pre><code> 0000000100.000000
</code></pre>
<p>I tried <code>.zfill(17)</code>, but it only displays one digit after the decimal point.</p>

<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format</code></a>:</p>
<pre><code>&gt;&gt;&gt; '{:017.6f}'.format(100.00)
'0000000100.000000'
</code></pre>
<p>or <a href="http://docs.python.org/3/library/functions.html#format" rel="nofollow"><code>format</code></a>:</p>
<pre><code>&gt;&gt;&gt; format(100.00, '017.6f')
'0000000100.000000'
</code></pre>
</div>
