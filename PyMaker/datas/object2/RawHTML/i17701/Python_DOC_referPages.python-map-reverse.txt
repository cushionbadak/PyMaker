<div class="post-text" itemprop="text">
<p>Is there a function that takes a list of functions and an input and outputs a list of the functions operation on the input?</p>
<p>So like <code>map</code>, but backwards:</p>
<pre><code>&gt;&gt;&gt;map(lambda x: 2*x,[1,2,3,4,5,6,7,8,9])
[2, 4, 6, 8, 10, 12, 14, 16, 18]
&gt;&gt;&gt;reverseMap([lambda x: x**2,lambda x: x**3],3)
[9,27]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can in fact use a <code>map</code> to do this:</p>
<pre><code>&gt;&gt;&gt; map(lambda f: f(3), [lambda x: x**2,lambda x: x**3])
[9, 27]
</code></pre>
<p>Use the list of functions you want to apply as the iterable, and then apply each of them to your input (in this case 3).</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a simple <a href="http://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="noreferrer">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; funcs = [lambda x: x**2, lambda x: x**3]
&gt;&gt;&gt; 
&gt;&gt;&gt; [f(3) for f in funcs]
[9, 27]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>How about simply:</p>
<pre><code>In [1]: [f(3) for f in lambda x: x**2,lambda x: x**3]
Out[1]: [9, 27]
</code></pre>
<p>You can wrap it in a function if you like.</p>
</div>
