<div class="post-text" itemprop="text">
<p>The funny symbols you're encountering are a UTF-8 <a href="http://en.wikipedia.org/wiki/Byte_order_mark" rel="noreferrer">BOM (Byte Order Mark)</a>. To get rid of them, open the file using the correct encoding (I'm assuming you're on Python 3):</p>
<pre><code>file = open(r"D:\zzzz\names2.txt", "r", encoding="utf-8-sig")
</code></pre>
<p>Furthermore, for counting, you can use <a href="http://docs.python.org/3/library/collections.html#collections.Counter" rel="noreferrer"><code>collections.Counter</code></a>:</p>
<pre><code>from collections import Counter
wordcount = Counter(file.read().split())
</code></pre>
<p>Displaying them is easy as well:</p>
<pre><code>&gt;&gt;&gt; for item in wordcount.items(): print("{}\t{}".format(*item))
...
snake   1
lion    2
goat    2
horse   3
</code></pre>

<pre><code>FILE_NAME = 'file.txt'

wordCounter = {}

with open(FILE_NAME,'r') as fh:
  for line in fh:
    # Replacing punctuation characters. Making the string to lower.
    # The split will spit the line into a list.
    word_list = line.replace(',','').replace('\'','').replace('.','').lower().split()
    for word in word_list:
      # Adding  the word into the wordCounter dictionary.
      if word not in wordCounter:
        wordCounter[word] = 1
      else:
        # if the word is already in the dictionary update its count.
        wordCounter[word] = wordCounter[word] + 1

print('{:15}{:3}'.format('Word','Count'))
print('-' * 18)

# printing the words and its occurrence.
for  (word,occurance)  in wordCounter.items(): 
  print('{:15}{:3}'.format(word,occurance))
</code></pre>
