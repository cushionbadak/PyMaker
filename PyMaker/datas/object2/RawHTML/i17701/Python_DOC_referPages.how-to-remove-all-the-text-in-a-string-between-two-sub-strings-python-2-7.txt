<div class="post-text" itemprop="text">
<pre><code>x = "About DoctorTardi bla bla Stats for Project Ares"
x = x[:17]+x[25:]
print x
</code></pre>
<p><strong>will print</strong> </p>
<blockquote>
<p>About DoctorTardi Stats for Project Ares</p>
</blockquote>
<ul>
<li><p><code>x[:17]</code> is the sub string of x, from the start to the 17th letter.</p></li>
<li><p><code>x[25:]</code> is the sub string of x, from the 25th letter to the end.</p></li>
<li><p><code>x[17:25]</code> is the sub string of x, from the 17th letter to the 25th
letter.</p></li>
</ul>
