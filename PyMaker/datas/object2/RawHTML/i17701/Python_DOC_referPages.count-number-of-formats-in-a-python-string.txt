<p>I'm reading a buffer from memory containing strings that may contain a number(up to 3) of formats within them(like <code>%d</code>, <code>%x</code> etc.) I need to know how many formats the string has so I can fetch them and print the string.. Is there any better way to do this but count the <code>%</code>'s in the string?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/stdtypes.html#str.count" rel="nofollow"><code>str.count</code></a>:</p>
<pre><code>&gt;&gt;&gt; fmt = '%d, %x, %%'
&gt;&gt;&gt; fmt.count('%')
4
</code></pre>
<p>To exclude <code>%%</code>:</p>
<pre><code>&gt;&gt;&gt; fmt.count('%') - fmt.count('%%') * 2
2
</code></pre>
</div>
