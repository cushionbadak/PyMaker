<p>Consider that Python will all one to type the name 'Jânis' into the Python CLI, if it is known that the 'â' character is hex "E2" in CP-1252:</p>
<pre><code>&gt;&gt;&gt; 'J\xe2nis'
'Jânis'
</code></pre>
<p><strong>How might one type that name into the Python CLI if the Unicode code point is known, but not the CP-1252 point?</strong> In fact, the code point in question is<code>U+00E2</code>. Also, the UTF-8 encoded character is <code>%C3 %A2</code>, is there any way to type that into the Python CLI if only that is known?</p>
<p>I am using Python 3.2 on Kubuntu Linux 12.10.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use unicode escape sequence (<code>\unnnn</code>):</p>
<pre><code>&gt;&gt;&gt; 'J\u00e2nis'
'Jânis'
</code></pre>
<p>If you know utf-8, use <a href="http://docs.python.org/3/library/stdtypes.html#bytes.decode" rel="nofollow"><code>bytes.decode</code></a> (<code>utf-8</code> is default encoding in Python 3.x, so it is optional):</p>
<pre><code>&gt;&gt;&gt; b'J\xc3\xa2nis'.decode('utf-8')
'Jânis'
</code></pre>
<p>If you have <code>%C3%A2</code>, use <a href="http://docs.python.org/3/library/urllib.parse.html#urllib.parse.unquote" rel="nofollow"><code>urllib.parse.unquote</code></a>:</p>
<pre><code>&gt;&gt;&gt; import urllib.parse
&gt;&gt;&gt; urllib.parse.unquote('J%c3%a2nis', encoding='utf-8')
'Jânis'
</code></pre>
</div>
