<div class="post-text" itemprop="text">
<p>You can't achieve this with ordinary values as parameters, referencing a local variable never does anything extra. You can, however, accept <em>functions</em> which supply a value. Then instead of passing some integer, you pass a function which returns an integer, and instead of changing <code>myfun</code> to do I/O, you just pass a function which does the I/O.</p>
<pre><code>myfun(lambda: 5, lambda: 6)
# I'm gonna assume Python 3
myfun(input, input)
</code></pre>
<p>You need to write the function a bit differently though, because you want the input to happen at a precise point in time. Something like this:</p>
<pre><code>def myfun(x_fun, y_fun):
    print("my x is", end=" ")
    x = x_fun()
    print("my y is", end=" ")
    y = y_fun()
    z=x+y
    print("my z is", z)
</code></pre>
