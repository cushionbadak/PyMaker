<div class="post-text" itemprop="text">
<p>I'm looking for the best way to represent a char in its bits representation <strong>by a given number of length(a variable)</strong>
For example:
in bytes representation, the number '5' is: 0101 if the given length is 4. </p>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; print '{:04b}'.format(5)
0101
</code></pre>
<p>In the string '{:04b}', the "4" means that we want the result to include 4 characters even if fewer are required and the "0" means that we want it to pad the result with zeros.</p>
<p>If you want the size under program control:</p>
<pre><code>&gt;&gt;&gt; n = 6 ; '{0:0{1}b}'.format(5, n)
'000101'
</code></pre>
<p>Or:</p>
<pre><code>&gt;&gt;&gt; n = 6 ; ('{:0%sb}' % n).format(5)
'000101'
</code></pre>
<p>Or:</p>
<pre><code>&gt;&gt;&gt; n = 6 ; ('{:0' + str(n) + 'b}').format(5)
'000101'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/3/library/functions.html#format" rel="nofollow"><code>format</code></a> or <a href="http://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format</code></a> with <code>04b</code> as <a href="http://docs.python.org/3/library/string.html#format-specification-mini-language" rel="nofollow">format specification</a>:</p>
<pre><code>&gt;&gt;&gt; format(5, '04b')
'0101'
&gt;&gt;&gt; '{:04b}'.format(5)
'0101'
</code></pre>
<p>You can also specify the length 4 as argument if you use <code>str.format</code>:</p>
<pre><code>&gt;&gt;&gt; '{:0{}b}'.format(5, 4)
'0101'
&gt;&gt;&gt; '{0:0{1}b}'.format(5, 4)
'0101'
</code></pre>
