<div class="post-text" itemprop="text">
<p>From the <a href="http://docs.python.org/3/library/stdtypes.html#typesseq-range" rel="nofollow">documentation</a>:</p>
<blockquote>
<p>The arguments to the range constructor must be integers</p>
</blockquote>
<p>So to use <code>range</code> in this case you need to change your code to this:</p>
<pre><code>domainmin=int(input("Min X Value: "))
domainmax=int(input("Max X Value: "))
</code></pre>
<p>Or, perhaps closer to what you intend, you can use instead <a href="http://docs.scipy.org/doc/numpy/reference/generated/numpy.arange.html#numpy-arange" rel="nofollow"><code>numpy.arange</code></a> or <a href="http://docs.scipy.org/doc/numpy/reference/generated/numpy.linspace.html?highlight=linspace#numpy.linspace" rel="nofollow"><code>numpy.linspace</code></a>, so you can use <code>float</code> ranges - you will have to additionally add granulation parameter (<code>step</code> for <code>arange</code>) or number of steps (<code>num</code> for <code>linspace</code>).</p>
</div>
