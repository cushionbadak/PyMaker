<p>Hi frens I have the following form of data</p>
<pre><code>fields = [{'name':'xxx', 'age':24, 'location':'city_name'},
          {'name':'yyy', 'age':24, 'location':'city_name'}]
</code></pre>
<p>Now I want to update the location in two dicts and the save the fields in the same format.How to do it?I am beginner.</p>

<div class="post-text" itemprop="text">
<p>Set same location for both fields.</p>
<pre><code>&gt;&gt;&gt; fields = [{'name':'xxx', 'age':24, 'location':'city_name'},
...           {'name':'yyy', 'age':24, 'location':'city_name'}]
&gt;&gt;&gt; for field in fields:
...     field['location'] = 'loc'
...
&gt;&gt;&gt; fields
[{'age': 24, 'name': 'xxx', 'location': 'loc'}, {'age': 24, 'name': 'yyy', 'location': 'loc'}]
</code></pre>
<p>To set different locations, use <a href="http://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip</code></a>:</p>
<pre><code>&gt;&gt;&gt; for field, loc in zip(fields, ['here', 'there']):
...     field['location'] = loc
...
&gt;&gt;&gt; fields
[{'age': 24, 'name': 'xxx', 'location': 'here'}, {'age': 24, 'name': 'yyy', 'location': 'there'}]
</code></pre>
