<p>I have a binary value that is stored in a variable. How do I convert it to decimal value or hex value??</p>
<pre><code> t = '0b'+bin(0o202)[::-1][:-2]
</code></pre>
<p>So the value of the t would be</p>
<pre><code> t = '0b01000001'
</code></pre>
<p>I need the value of the t converted to decimal or hex.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/3/library/functions.html#hex" rel="nofollow"><code>hex()</code></a> and <a href="http://docs.python.org/3/library/functions.html#int" rel="nofollow"><code>int()</code></a>:</p>
<pre><code>&gt;&gt;&gt; hex(int(t, 2))  # hex
'0x41'
&gt;&gt;&gt; 
&gt;&gt;&gt; str(int(t, 2))  # decimal
'65'
</code></pre>
<p>Note that integers are by default represented in decimal, which is why the last line works as it should.</p>
</div>
