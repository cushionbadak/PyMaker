<div class="post-text" itemprop="text">
<p>One of the reasons you get syntax errors is that <code>try</code> / <code>except</code> syntax changed from Python 2 to Python 3.</p>
<p>In Python 3 you should use</p>
<pre><code>except ImportError as exc:
</code></pre>
<p>instead of</p>
<pre><code>except ImportError, exc:
</code></pre>
<p>It's not about import syntax only, reading the error messages we see there are other problems due to incompatibilities between the two versions. I guess django-sphinx has not been updated to work with Python 3.</p>
<p>References: <a href="http://docs.python.org/3/whatsnew/3.0.html" rel="nofollow">http://docs.python.org/3/whatsnew/3.0.html</a></p>
