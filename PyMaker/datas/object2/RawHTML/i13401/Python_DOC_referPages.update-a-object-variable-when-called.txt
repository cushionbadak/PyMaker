<div class="post-text" itemprop="text">
<p>You could implement this using a <a href="https://docs.python.org/3/library/functions.html#property" rel="nofollow">property</a>:</p>
<pre><code>class Ant(object):  # note leading uppercase, per style guide (PEP-8)

    def __init__(self):  # you ignore the age parameter anyway
        self._age = 0

    def update_age(self):
        self._age += 1

    @property
    def age(self):
        self.update_age()
        return self._age
</code></pre>
<p>This makes <code>age</code> read-only, and increments correctly:</p>
<pre><code>&gt;&gt;&gt; an_ant = Ant()
&gt;&gt;&gt; an_ant.age
1
&gt;&gt;&gt; an_ant.age
2
&gt;&gt;&gt; an_ant.age = 10

Traceback (most recent call last):
  File "&lt;pyshell#5&gt;", line 1, in &lt;module&gt;
    an_ant.age = 10
AttributeError: can't set attribute
</code></pre>
