<p>You can use <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow"><code>map</code></a> function and <a href="https://docs.python.org/2/library/operator.html#operator.itemgetter" rel="nofollow"><code>operator.itemgetter</code></a> as it's function :</p>
<pre><code>&gt;&gt;&gt; from operator import itemgetter
&gt;&gt;&gt; map(itemgetter('HolderJob'), a['XMLSchemaPalletLoadTechData']['TechDataParams'])
&gt;&gt;&gt; [u'243340545', u'243340544']
</code></pre>
