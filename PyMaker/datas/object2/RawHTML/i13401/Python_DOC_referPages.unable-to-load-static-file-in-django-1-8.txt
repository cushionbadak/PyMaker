<div class="post-text" itemprop="text">
<p>Your <code>STATICFILES_DIR</code> contains a wrong value. You use <code>os.path.join(BASE_DIR, '/static')</code>. </p>
<p>Citing the <a href="https://docs.python.org/3/library/os.path.html#os.path.join" rel="nofollow">documentation for join</a>:</p>
<blockquote>
<p>If a component is an absolute path, all previous components are thrown
  away and joining continues from the absolute path component.</p>
</blockquote>
<p>Please have a look at the following code example:</p>
<pre><code>&gt;&gt;&gt; os.path.join('/a', 'b', '/c', 'd')
'/c/d'
</code></pre>
<p>Solution: remove the leading slash in <code>'/static'</code>: <code>os.path.join(BASE_DIR, '/static')</code></p>
</div>

<p>Let's see...
When you create a new project marca, you get a subfolder marca that contains your settings.py, wsgi.py etc. Usually, I create a static folder here for my non-app specific static files. I can then have a settings.py PROJ_DIR to target that e.g.</p>
<pre><code>PROJ_DIR = os.path.dirname(os.path.abspath(__file__)) 
</code></pre>
<p>Now I have a PROJ_DIR pointing at my subfolder marca in addition to BASE_DIR. Now to more settings.</p>
<pre><code>STATIC_URL = '/static/' #pretty much ok
STATICFILES_DIR = (
    os.path.join(PROJ_DIR, 'static'), #additional location of static files
)

STATIC_ROOT = os.path.join(BASE_DIR, 'static')
</code></pre>
<p>Files in STATIC_ROOT will be served at STATIC_URL (/static/ in your case), usually by a separate static file server like nginx. It should be empty initially. To get your files into STATIC_ROOT, run </p>
<pre><code>python manage.py collectstatic 
</code></pre>
<p>which will look in your app static folders and all paths in STATICFILES_DIR and create a directory structure ready to be served.</p>
<p>In your template, base.html, you'll need to add </p>
<pre><code>{% load staticfiles %}
</code></pre>
<p>I expect you'll get an error if you don't.
Basically, I'm proposing this new structure to you.</p>
<pre><code>manage.py
+-- static (your STATIC_ROOT is currently pointing here. empty initially but check after running collectstatic
+-- marca
    settings.py
    wsgi.py
    urls.py
    +-- static
    |   +-- css
    |   +-- img
    |   +-- gis
    |   +-- js
    +-- templates
+-- marcador
|   +-- migrations
|   +-- templates
|       +-- marcador
</code></pre>
