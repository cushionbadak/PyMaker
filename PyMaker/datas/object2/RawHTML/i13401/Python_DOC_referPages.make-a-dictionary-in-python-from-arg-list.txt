<div class="post-text" itemprop="text">
<p>You should use the standard <a href="https://docs.python.org/3/library/argparse.html" rel="nofollow"><strong><code>argparse</code></strong></a> library</p>
<p>However, if you don't want to, read on.</p>
<pre><code>&gt;&gt;&gt; arg_list = "--config=pools_issuance.config --current_mm=9 --current_yy=15 --agency_id='FH'"
</code></pre>
<p>Your main problem is that you iterate over every character in the string:</p>
<pre><code>&gt;&gt;&gt; for arg in arg_list:
...     print arg
-
-
c
o
n
f
i
...
</code></pre>
<p>We can split the string based on whitespace:</p>
<pre><code>&gt;&gt;&gt; arg_list.split()
['--config=pools_issuance.config',
 '--current_mm=9',
 '--current_yy=15',
 "--agency_id='FH'"]
</code></pre>
<p>Then for each argument strip the '--', and split on <code>'='</code> to get the key value pairs:</p>
<pre><code>&gt;&gt;&gt; [arg.strip('--').split('=') for arg in arg_list.split()]
[['config', 'pools_issuance.config'],
 ['current_mm', '9'],
 ['current_yy', '15'],
 ['agency_id', "'FH'"]]
</code></pre>
<p>Then pass them to a dictionary constructor:</p>
<pre><code>&gt;&gt;&gt; dict(arg.strip('--').split('=')
...      for arg in arg_list.split())
{'agency_id': "'FH'",
 'config': 'pools_issuance.config',
 'current_mm': '9',
 'current_yy': '15'}
</code></pre>
<p>You might want to clean up some of the values as the numbers will be strings, and the <code>agency_id</code> will be a string with the quotes included: <code>"'FH'"</code></p>
