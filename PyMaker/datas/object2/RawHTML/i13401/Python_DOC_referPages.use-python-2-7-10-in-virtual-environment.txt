<p><strong>UPDATE:</strong>
Thanks for both answers. I resolved the Question 1).</p>
<p>Essentially, you need to add --user when installing the setuptools and virtualenv. For example,</p>
<pre><code>~ $pip install virtualenv==1.5.2 --user
</code></pre>
