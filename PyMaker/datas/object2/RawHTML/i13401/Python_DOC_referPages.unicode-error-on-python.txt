<div class="post-text" itemprop="text">
<p>Assuming you're using Python2:</p>
<pre><code>with open('test1.csv', 'wb') as csvfile:
    writer = csv.writer(csvfile)
    for r in table:
        writer.writerow([x.encode('utf-8') for x in r])
</code></pre>
<p>Of course, when you open the csv file, you will also need to decode it using the same encoding:</p>
<pre><code>with open('test1.csv') as csvfile:
    reader = csv.reader(csvfile.decode('utf-8'))
</code></pre>
<p>(NB: if you used Python3, none of this would be necessary - your original example would work fine as it is).</p>
</div>
<div class="post-text" itemprop="text">
<p>First of all you don't need to use a list comprehension for write in csv file, secondly if you are using <code>python 2.X</code> you can use <code>codecs</code> module to open your file with a proper encoding and if you are using python 3.X you can use <code>encoding</code> argument in <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow"><code>open</code></a> function.</p>
<p>Also note that since the <code>write</code> method used your default encoding if still you got unicode error you can use <code>str.encode()</code> method in <code>write</code> method.</p>
<p>Python 2.X:</p>
<pre><code>import codecs
with codecs.open(filename, 'w', encoding='utf-8') as csvfile:
     writer = csv.writer(csvfile)
     for r in table:
         writer.writerow(r.encode('utf-8'))
</code></pre>
<p>Python 3.X :</p>
<pre><code>with open(filename, 'wb', encoding='utf-8') as csvfile:
     writer = csv.writer(csvfile)
     for r in table:
         writer.writerow(r.encode('utf-8'))
</code></pre>
