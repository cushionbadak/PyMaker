<div class="post-text" itemprop="text">
<p>Your function modifies the <code>backpack</code> object in place, and then returns <code>None</code> (since it doesn't have an explicit return statement). So instead of assigning the result:</p>
<pre><code>backpack = addToInventory(backpack, dragonLoot)
</code></pre>
<p>You should just call the function without assigning the result back. <code>backpack</code> will be changed in place:</p>
<pre><code>addToInventory(backpack, dragonLoot)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Don't write code which someone else has already written for you.  This performs the same operation:</p>
<pre><code>import collections

backpack = collections.Counter({'gold coin': 42, 'rope': 1})
dragonLoot = ['gold coin', 'dagger', 'gold coin', 'gold coin', 'ruby']
backpack.update(dragonLoot)
</code></pre>
<p>See <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow"><code>collections.Counter</code></a> for details.</p>
