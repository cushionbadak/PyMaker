<p>I am new to python and I read that there is no ++ operator in Python but I am not able to understand the below code.</p>
<pre><code>&gt;&gt;&gt;print (2++3)
&gt;&gt;&gt;5
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>+</code> and <code>-</code> act as unary as well as binary operators. So,</p>
<ul>
<li><code>a ++ b</code> is same as <code>a + (+b)</code></li>
<li><code>a -+ b</code> is same as <code>a - (+b)</code></li>
<li><code>a -- b</code> is same as <code>a - (-b)</code></li>
<li><code>a +- b</code> is same as <code>a + (-b)</code></li>
</ul>
<p>As can be seen below</p>
<pre><code>&gt;&gt;&gt; 2++3
5
&gt;&gt;&gt; 2-+3
-1
&gt;&gt;&gt; 2--3
5
&gt;&gt;&gt; 2+-3
-1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is not the ++ operator. Your code is interpreted as follows:</p>
<pre><code>2 + (+3)
</code></pre>
<p>Now since +3 is 3, the final result is 5. For fun, try the following:</p>
<ul>
<li>2++++++++++++++3</li>
<li>2+++++++++++++++++++++-8</li>
</ul>
<p>See <a href="https://docs.python.org/3/reference/expressions.html" rel="nofollow">Expressions</a> for details.</p>
</div>
