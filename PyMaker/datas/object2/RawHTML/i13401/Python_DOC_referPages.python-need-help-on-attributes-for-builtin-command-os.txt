<div class="post-text" itemprop="text">
<p>You can use help command which shows the docstring for any object or method</p>
<pre><code>help(os.getcwd)
</code></pre>
<p>It is little bit similar to man command in unix. For detail info, you can check out the documentation of <code>os</code> command - <a href="https://docs.python.org/2/library/os.html" rel="nofollow">os command documentation</a></p>
