<div class="post-text" itemprop="text">
<p><code>jsondata['queryResponse']['entity']['deviceGroupsDTO']</code> is a dictionary. </p>
<p>Iterate over <a href="https://docs.python.org/3/library/stdtypes.html#dict.items" rel="nofollow noreferrer">items()</a> to get key, value pairs:</p>
<pre><code>for key, value in jsondata['queryResponse']['entity']['deviceGroupsDTO'].items():
    print(key, value)
</code></pre>
<p>Note that, in case of python2, you would better use <a href="https://docs.python.org/2/library/stdtypes.html#dict.iteritems" rel="nofollow noreferrer">iteritems()</a> in place of <a href="https://docs.python.org/2/library/stdtypes.html#dict.items" rel="nofollow noreferrer">items()</a>.</p>
<p>See also: <a href="https://stackoverflow.com/questions/10458437/python-what-is-the-difference-between-dict-items-and-dict-iteritems">What is the difference between dict.items() and dict.iteritems()?</a></p>
