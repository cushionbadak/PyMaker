<div class="post-text" itemprop="text">
<p>I am trying to print out all the elements of a list one by one, like this</p>
<pre><code>a=[1,2,3]

print(i for i in a)
</code></pre>
<p>how do i print out</p>
<pre><code>1 
2
3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You want to use a <a href="https://docs.python.org/3/reference/compound_stmts.html#the-for-statement" rel="nofollow noreferrer">for-loop</a> here, not a <a href="http://legacy.python.org/dev/peps/pep-0289/" rel="nofollow noreferrer">generator expression</a>:</p>
<pre><code>&gt;&gt;&gt; a = [1, 2, 3]
&gt;&gt;&gt; for i in a:
...     print(i)
...
1
2
3
&gt;&gt;&gt; # As a one-liner
&gt;&gt;&gt; for i in a: print(i)
...
1
2
3
&gt;&gt;&gt;
</code></pre>
<hr/>
<p>Or, if you want to get fancy, you can use just <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>print</code></a> and <a href="https://stackoverflow.com/q/2322355/2555451">argument unpacking</a>:</p>
<pre><code>&gt;&gt;&gt; a = [1, 2, 3]
&gt;&gt;&gt; # This is the same as doing:  print(1, 2, 3, sep="\n")
&gt;&gt;&gt; print(*a, sep="\n")
1
2
3
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Here are two ways you can accomplish what you want.     </p>
<pre><code>a = [1, 2, 3]

for a_val in a:
    print a_val

for i in range(0, len(a)):
    print a[i]
</code></pre>
