<div class="post-text" itemprop="text">
<p>I notice you wants to play with 'carriage return' commands. First you need to learn both <code>'\b'</code> (moves the active position to the previous position) and <code>'\r'</code> (moves the active position to the start of line) <em>works for current active line</em>.<br/>
I think you are working on command line interpreter; at which you explicitly press enter for next line.<br/>
On command line interpreter use <code>;</code>  as follows:  </p>
<pre><code>&gt;&gt;&gt; print("abb", end=""); print("\b\bcc")
acc
&gt;&gt;&gt; print("a-&gt;", end=""); print("\b\b  ")
&gt;&gt;&gt; a
</code></pre>
<p>If you are using some script then your code should work, see:</p>
<pre><code>Upload$ cat s.py
print ("abcd-&gt;", end="")
print ("\b\b  ")
Upload$ python3.2 s.py
abcd    # notice -&gt; is removed  
</code></pre>
<p>But this is not much useful, correct approach is what <a href="https://stackoverflow.com/users/846892/awini-haudhary">Aशwini चhaudhary</a> has shown in his <a href="https://stackoverflow.com/a/23149489/1673391">answer</a>.</p>
<p>Edit: I found you mistake in your code</p>
<pre><code>   print('%d-&gt;' % addedlist[x], end="")    
print('\b\b\n')
</code></pre>
<p>You use <code>'\b'</code> to moves the active position to the previous position, but you don't overwrite <code>"-&gt;"</code>, you just outputs <code>'\n'</code> that shift cursors to next line, you should rectify your code as below.</p>
<pre><code>  print('%d-&gt;' % addedlist[x], end="")    
print('\b\b   \n')
 #         ^^^^  /b then overwrite with spaces 
</code></pre>
<p>Run at Linux shell as <code>$ python scriptname.py</code> (For command line Python's interpreter you can use something <a href="https://i.stack.imgur.com/tkC2v.png" rel="nofollow noreferrer">like code I written</a>, it is just to play, use str.join or use <code>sep</code> parameter in <code>print()</code>). </p>

<div class="post-text" itemprop="text">
<p>Use <code>str.join</code>:</p>
<pre><code>&gt;&gt;&gt; s = [9,5,1989,1,2,3,4,5,6,7,8,9]
&gt;&gt;&gt; print ('-&gt;'.join(map(str, s)))
9-&gt;5-&gt;1989-&gt;1-&gt;2-&gt;3-&gt;4-&gt;5-&gt;6-&gt;7-&gt;8-&gt;9
</code></pre>
<p>Or better <code>print()</code> with <code>sep</code> as <code>'-&gt;'</code>:</p>
<pre><code>&gt;&gt;&gt; print(*s, sep='-&gt;')
9-&gt;5-&gt;1989-&gt;1-&gt;2-&gt;3-&gt;4-&gt;5-&gt;6-&gt;7-&gt;8-&gt;9
</code></pre>
