<div class="post-text" itemprop="text">
<p>There's a built-in method of strings for this exact purpose called <a href="https://docs.python.org/2/library/stdtypes.html#str.splitlines" rel="nofollow"><code>splitlines</code></a>.</p>
<pre><code>&gt;&gt;&gt; tst = """dog
cat
monkey
cow"""
&gt;&gt;&gt; tst
'dog\ncat\nmonkey\ncow'   # for loop gives you each letter because it's 1 string
&gt;&gt;&gt; tst.splitlines()
['dog', 'cat', 'monkey', 'cow']
</code></pre>
<p>Now of course you can just append to it:</p>
<pre><code>&gt;&gt;&gt; lst = tst.splitlines()
&gt;&gt;&gt; lst.append("lemur")
&gt;&gt;&gt; lst
['dog', 'cat', 'monkey', 'cow', 'lemur']
</code></pre>
<p>Want it back as a multi-line string? Use <a href="https://docs.python.org/2/library/stdtypes.html#str.join" rel="nofollow"><code>join</code></a>.</p>
<pre><code>&gt;&gt;&gt; '\n'.join(lst)
'dog\ncat\nmonkey\ncow\nlemur'
&gt;&gt;&gt; print '\n'.join(lst)
dog
cat
monkey
cow
lemur
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I believe you are looking for <code>str.splitlines</code>:</p>
<pre><code>&gt;&gt;&gt; mystr = "dog\ncat\nmonkey\ncow"
&gt;&gt;&gt; print(mystr)
dog
cat
monkey
cow
&gt;&gt;&gt; mystr.splitlines()
['dog', 'cat', 'monkey', 'cow']
&gt;&gt;&gt;
</code></pre>
<p>From the <a href="https://docs.python.org/3/library/stdtypes.html#str.splitlines" rel="nofollow">docs</a>:</p>
<blockquote>
<p><code>str.splitlines([keepends])</code></p>
<p>Return a list of the lines in the string, breaking at line boundaries. This method uses the <a href="https://docs.python.org/3/glossary.html#term-universal-newlines" rel="nofollow">universal newlines</a> approach to
  splitting lines. Line breaks are not included in the resulting list
  unless <code>keepends</code> is given and true.</p>
</blockquote>
