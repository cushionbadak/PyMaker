<div class="post-text" itemprop="text">
<p>You did not assign the <code>columns</code> <code>names</code> , that is why when you call <code>names</code> the default output show as <code>[None]</code> </p>
<pre><code>df.columns.names
Out[934]: FrozenList([None])
list(df.columns.names)
Out[935]: [None]

df.columns.names=['SOMETHING']
df
Out[931]:
SOMETHING  id  A  B
0           1  1  2
1           2  2  3
df.columns.names
Out[932]: FrozenList(['SOMETHING'])
list(df.columns.names)
Out[937]: ['SOMETHING']
</code></pre>
<p>If you just want the dataframe columns name </p>
<pre><code>list(df)
</code></pre>
