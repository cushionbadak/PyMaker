<div class="post-text" itemprop="text">
<p>Try adding a <code>raise</code> to the end of your except to raise the exception again, like this:</p>
<pre><code>def func_A(int):
    try:
        ans = 1 + int
        print(ans)
    except:
        print('Oops your input was wrong')
        raise

func_A('Hello_World')
</code></pre>
<p>Output:</p>
<pre><code>Oops your input was wrong
Traceback (most recent call last):
  File "/home/ashish/s.py", line 15, in &lt;module&gt;
    func_A('Hello_World')
  File "/home/ashish/s.py", line 3, in func_A
    ans = 1 + int
TypeError: unsupported operand type(s) for +: 'int' and 'str'
</code></pre>
