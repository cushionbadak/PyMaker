<pre><code>listA = ["A", "B", "C"]

listB = ["1", "2", "3"]

listC = ["!", "@", "#"]
</code></pre>
<p>If I have these lists, how would I get a new list of </p>
<pre><code>[("A", "1", "!"), ("B", "2", "@"), ("!", "@", "#")]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#zip" rel="noreferrer"><code>zip</code></a>:</p>
<pre><code>list(zip(listA,listB,listC))

[('A', '1', '!'), ('B', '2', '@'), ('C', '3', '#')]
</code></pre>
