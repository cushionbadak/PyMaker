<p>I have an object that is a dict of dicts. I would like to tell my editor this object is so, how can I do this? Currently I have:</p>
<pre><code>def do_somthing(dict_of_dicts):
    """
    :type dict_of_dicts: dict of dict
    """

    for key, value in dict_of_dicts.items():
            for key2, value2 in value.items():
                print('Key')
</code></pre>
<p>However, my editor (Pycharm) does not seem to be registering this.</p>

<div class="post-text" itemprop="text">
<p>You can achieve this using type hinting using <a href="https://docs.python.org/3/library/typing.html#typing.Dict" rel="nofollow noreferrer"><code>Dict</code></a>:</p>
<pre><code>from typing import Dict, Any

def do_somthing(dict_of_dicts: Dict[Any, Dict]):
    """
    :type dict_of_dicts: Dict[Any, Dict]
    """

    for key, value in dict_of_dicts.items():
            for key2, value2 in value.items():
                print('Key')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I don't use PyCharm, butâ€¦ it understands type annotations, right? So, specify it as an annotation:</p>
<pre><code>from typing import Dict, Any

def do_something(dict_of_dicts: Dict[Any, Dict[Any, Any]]):
    # ...
</code></pre>
<p>Or you can even define a generic <code>DictOfDict</code> type (generic on the outer key, inner key, and inner value):</p>
<pre><code>from typing import Dict, Any, TypeVar, Generic:

K1, K2, V = TypeVar('K1'), TypeVar('K2'), TypeVar('V')
class DictOfDicts(Dict[K1, Dict[K2, V]]): ...

def do_something(dict_of_dicts: DictOfDicts[Any, Any, Any]):
    # ...
</code></pre>

<div class="post-text" itemprop="text">
<p>You can also add <code>param</code>:</p>
<pre><code>def do_somthing(dict_of_dicts):
    """
    :param dict_of_dicts: A dictionary of dictionaries.
    :type dict_of_dicts: dict of dict
    """

for key, value in dict_of_dicts.items():
        for key2, value2 in value.items():
            print('Key')
</code></pre>
