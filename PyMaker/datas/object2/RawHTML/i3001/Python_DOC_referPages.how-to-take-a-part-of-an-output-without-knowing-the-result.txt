<p>If you are receiving parsable output can simply do: </p>
<pre><code>ips= []

for data in output:
    ips.append(data["ip"])
</code></pre>
<p>Or as comment suggest:</p>
<pre><code>ips = [data["ip"] for data in output]
</code></pre>
