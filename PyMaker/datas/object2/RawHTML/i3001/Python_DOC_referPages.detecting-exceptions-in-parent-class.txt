<div class="post-text" itemprop="text">
<p>Your <code>PrivCustomer.withdraw</code> method will <em>replace</em> the <code>Customer.withdraw</code> method for <code>PrivCustomer</code> objects.  To call <code>Customer.withdraw</code>, you have to use <code>super().withdraw</code> to get the inherited method.  That method call will be where <code>LimitException</code> is raised</p>
<pre><code>class PrivCustomer(Customer):
    def withdraw(self, amount):
        try:
            super().withdraw(amount)
        except LimitException:
            ...
</code></pre>
