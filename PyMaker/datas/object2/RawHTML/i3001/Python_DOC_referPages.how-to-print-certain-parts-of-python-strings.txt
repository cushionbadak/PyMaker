<p>you can pass you list of strings to the <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer"><code>csv</code></a> module, then take only first and fourth entry of each row. Split first entry according to opening parenthesis:</p>
<pre><code>cr = csv.reader(a,delimiter="|")
for row in cr:
    print("{} | {}".format(row[0].split("(")[0].strip(),row[3]))
</code></pre>

<p>another method using csv writer for output only (but doesn't "space out" the fields, no space added before &amp; after the separator):</p>
<pre><code>cr = csv.reader(a,delimiter="|")
cw = csv.writer(sys.stdout,delimiter="|")
cw.writerows([row[0].split("(")[0].strip(),row[3]] for row in cr)
</code></pre>
<p><code>sys.stdout</code> can be an opened file handle too.</p>
<p>The most "complex" part is this:</p>
<pre><code>row[0].split("(")[0].strip()
</code></pre>
<p>this is just taking first column, splitting according to parenthese, taking the first part (the left part), and remove leading/trailing space around.</p>
