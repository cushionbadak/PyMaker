<p>Python's <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow noreferrer"><code>str.replace</code></a> is not an "in-place" method; instead, it returns a copy. Therefore, you need to assign back to your variable for the operation to store your result:</p>
<pre><code>x = 'http://urlshortener.io/s/1234abc/?s=https%3A%2F%2Fexample.com%2Fsome_contents%2F'    
x = x.replace('%2F', '/')

print(x)

http://urlshortener.io/s/1234abc/?s=https%3A//example.com/some_contents/
</code></pre>
<p>A better solution is to use the purpose-built <code>urllib</code>:</p>
<pre><code>from urllib import parse

x = 'http://urlshortener.io/s/1234abc/?s=https%3A%2F%2Fexample.com%2Fsome_contents%2F'

x = parse.unquote(x)

print(x)

http://urlshortener.io/s/1234abc/?s=https://example.com/some_contents/
</code></pre>
