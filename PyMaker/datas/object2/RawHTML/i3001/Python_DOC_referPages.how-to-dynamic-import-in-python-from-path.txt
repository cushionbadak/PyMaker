<p>I need to use dynamic import a module from a variable.</p>
<p>My problem is, that importing module is in directory.</p>
<p>Like this:</p>
<pre><code>from A.B.C import D
</code></pre>
<p>How can I import dynamic?</p>
<p>Like this...</p>
<pre><code>  module = __import__('A.B.{0}'.format('C'))
  my_class = getattr(module, 'D')
  instance = my_class()
</code></pre>
<p>I am getting a error: AttributeError: module 'A' has no attribute 'D'</p>
<p>Why?</p>
<p>Python 3.</p>

<div class="post-text" itemprop="text">
<p>You have to pass the <code>fromlist</code> argument in order to import a specific module under a package:</p>
<pre><code>module = __import__('A.B.{0}'.format('C'), fromlist=('D',))
instance = D()
</code></pre>
<p>Please refer to <a href="https://docs.python.org/3/library/functions.html#__import__" rel="nofollow noreferrer"><code>__import__</code>'s documentation</a> for more details.</p>
