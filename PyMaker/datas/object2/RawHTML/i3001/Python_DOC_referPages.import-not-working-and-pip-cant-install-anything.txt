Type "python" and press enter to use the console after you have installed Python.</p>

You don't <code>pip install random</code>, it's part of the <a href="https://docs.python.org/3/library/random.html" rel="nofollow noreferrer">standard library</a>. 2. <code>import random</code> is a <i>Python statement</i>, not a shell command.
