<p>I have in my code a list with multiple dictionaries in it (at the moment I have a specific number of dictionaries, but <strong>I need the program to work for any n numbers of dictionaries</strong>).</p>
<p>For example, let say I have 3 dictionaries to simplify:</p>
<p>What I have:</p>
<pre><code>my_list = [{'a': 1, 'b': 2, 'c': 3}, {'b':4, 'c': 5, 'd': 6}, {'c':7, 'd': 8, 'e': 9}]
</code></pre>
<p>what I need:</p>
<pre><code>my_list = [{'a': 1, 'b': 6, 'c': 15, 'd': 14, 'e': 9}]
</code></pre>
<p>or</p>
<pre><code>my_list = {'a': 1, 'b': 6, 'c': 15, 'd': 14, 'e': 9}
</code></pre>
<p>(It's essentially the same thing, isn't it?)</p>
<p>Thank you for your help!!!</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>sum()</code> and <code>collections.Counter</code>:</p>
<pre><code>In [90]: sum(map(Counter, my_list), Counter())
Out[90]: Counter({'a': 1, 'b': 6, 'c': 15, 'd': 14, 'e': 9})
</code></pre>
<p>Also read <a href="https://stackoverflow.com/a/42717000/2867928">https://stackoverflow.com/a/42717000/2867928</a> for more information.</p>
