<p>For avoiding exhausting checks, you need to restructure your code to smaller parts and introduce control structures, such as <code>while</code>.</p>
<p>In my example I created two functions <code>get_data_set()</code> and <code>check_months()</code>.</p>
<p>Function <code>get_data_set()</code> loops infinitely, until the input is correct. That means the number of words is divisible by 4 and every 4th word is a months name.</p>
<p>Function <code>check_months()</code> has one argument - <code>data_set</code> and checks it if every 4th word is month. If it is, returns <code>True</code>, in not <code>False</code>.</p>
<p>Example is bellow:</p>
<pre><code>import re

def get_data_set():
    while True:
        data_input = input("Please input data.\n")
        data_set = re.sub(r"[^\w]", " ", data_input).split()
        if len(data_set) % 4 != 0:
            print("That\'s not a valid input. For formatting help, type \'help\'.'")
            continue
        if check_months(data_set) == False:
            print("That\'s not a valid input.'")
            continue
        return data_set

def check_months(data_set):
    for word in data_set[::4]:
        if word not in ("JAN", "FEB", "MAR", "APR", "MAY", "JUN", "JUL", "AUG", "SEP", "OCT", "NOV", "DEC"):
            return False
    return True

data_set = get_data_set()
print('Valid data set!')
print(data_set)
</code></pre>
