<div class="post-text" itemprop="text">
<p>That command should be <a href="https://docs.python.org/3/using/cmdline.html" rel="nofollow noreferrer">run from your OS-level shell</a>, not from Python:</p>
<pre><code>$ python3 -m venv tutorial-env
</code></pre>
<p>or on Windows:</p>
<pre><code>C:\&gt; python3 -m venv tutorial-env
</code></pre>
<p>Depending on how Python was installed you might need to use <code>python</code> or even <code>python2</code> instead of <code>python3</code>.</p>
