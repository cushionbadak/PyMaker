<p>I am haviing trouble converting in python from this format:</p>
<pre><code>u'08:00:00' 
</code></pre>
<p>to a <code>datetime</code> object like:</p>
<pre><code>datetime.datetime(2017,02,22,8,0,0)
</code></pre>
<p>What is an easy way to do this?  </p>
<p>I can do it by just parsing the string but looking for more elegant solution</p>
</div>
<div class="post-text" itemprop="text">
<p>First you need to parse the time string. You can do that by hand, or use <code>datetime.strptime()</code>:</p>
<pre><code>from datetime import datetime

s = u'08:00:00'
t = datetime.strptime(s, '%H:%M:%S').time()
</code></pre>
<p><code>strptime()</code> doesn't care that you give it a unicode string. That will give you a <code>datetime.time</code> object:</p>
<pre><code>&gt;&gt;&gt; t
datetime.time(8, 0)
</code></pre>
<p>Then use <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.combine" rel="nofollow noreferrer"><code>datetime.combine()</code></a> to combine the time with today's date:</p>
<pre><code>from datetime import date

dt = datetime.combine(date.today(), t)
</code></pre>
<p>This will give you the required <code>datetime.datetime</code> object:</p>
<pre><code>&gt;&gt;&gt; dt
datetime.datetime(2017, 2, 24, 8, 0)
</code></pre>
