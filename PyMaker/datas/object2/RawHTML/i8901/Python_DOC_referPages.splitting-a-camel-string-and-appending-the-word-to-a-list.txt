<p>In case of lower-case letter, append to the last word (<code>array[-1]</code>):</p>
<pre><code>words = []
string = "ThisIsCamelCase"

for i in string:
    if i.isupper() or not words:  # upper case / very first letter
        words.append(i)
    else:                         # lower case
        words[-1] += i

print(words) # =&gt; ['This', 'Is', 'Camel', 'Case']
</code></pre>
