<div class="post-text" itemprop="text">
<p>Consider using <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><strong><em>re.findall()</em></strong></a> function:</p>
<pre><code>import re

s = '''
#encoding=utf8
__author__ = "naci"
__title__ = "test script"
__desc__ = "test description"
or __desc__ = """
    test description.
"""
'''

data = re.findall(r'__(?P&lt;attr&gt;\w+)_ = (?P&lt;val&gt;"[^"]+"|"""[^"]+""")', s)
print(data)
</code></pre>
<p>The output(pairs: <em>key/value</em>):</p>
<pre><code>[('author_', '"naci"'), ('title_', '"test script"'), ('desc_', '"test description"'), ('desc_', '"""\n    test description.\n"""')]
</code></pre>
