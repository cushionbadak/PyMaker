<div class="post-text" itemprop="text">
<p>This syntax was released with Python 3.5 as the part of <a href="https://www.python.org/dev/peps/pep-0448/" rel="nofollow noreferrer">PEP 448 - Additional Unpacking Generalizations</a>.</p>
<p>As per the <a href="https://docs.python.org/3/whatsnew/3.5.html#whatsnew-pep-448" rel="nofollow noreferrer">document</a>: since Python 3.5, tuple, list, set, and dictionary displays allow multiple unpackings like:</p>
<pre><code>&gt;&gt;&gt; *range(4), 4
(0, 1, 2, 3, 4)

&gt;&gt;&gt; [*range(4), 4]
[0, 1, 2, 3, 4]

&gt;&gt;&gt; {*range(4), 4, *(5, 6, 7)}
{0, 1, 2, 3, 4, 5, 6, 7}

&gt;&gt;&gt; {'x': 1, **{'y': 2}}
{'x': 1, 'y': 2}
</code></pre>
