<p>I am new to python. I have a very basic question. When we use the following command (I understand its not efficient to import everything using *)
from _ (<em>any module name</em>) import *</p>
<p>How can I check the things that get imported using the above command?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#dir" rel="nofollow noreferrer"><code>dir</code></a> to see what names are in the current module. By comparing the names before and after the import you can see what's imported:</p>
<pre><code>&gt;&gt;&gt; vars_before_import = set(dir())
&gt;&gt;&gt; from json import *
&gt;&gt;&gt; set(dir()) - vars_before_import
set(['load', 'JSONEncoder', 'dump', 'vars_before_import', 'JSONDecoder', 'dumps', 'loads'])
</code></pre>
<p>To exclude <code>vars_before_import</code>:</p>
<pre><code>&gt;&gt;&gt; set(dir()) - vars_before_import - {'vars_before_import'}
set(['load', 'JSONEncoder', 'dump', 'JSONDecoder', 'dumps', 'loads'])
</code></pre>
<p><strong>NOTE</strong></p>
<p>This won't catch objects that have been replaced (e.g. you defined <code>load</code> before importing everything in <code>json</code>).</p>
