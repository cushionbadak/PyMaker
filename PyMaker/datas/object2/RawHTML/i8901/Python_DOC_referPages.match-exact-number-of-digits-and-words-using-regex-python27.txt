<p>I have this string.</p>
<pre><code>P O BOX 32370, CA 92263
</code></pre>
<p>And this Regex <code>\w{2} \d{5}</code></p>
<p>But it matches both texts in bold. "P O B<strong>OX 32370</strong>, <strong>CA 92263</strong>"</p>
<p>Actually I want to extract State, and Zip code.</p>
<p>I want to grab <strong>text starting and a space, then exact 2 alphabets, then one space, then exact 5 digits.</strong></p>
</div>
<div class="post-text" itemprop="text">
<p>You can add word boundary <code>\b</code> to make sure the pattern doesn't have leading or trailing word characters (alphanumeric and underscore):</p>
<pre><code>import re

re.findall(r"\b\w{2} \d{5}\b", "P O BOX 32370, CA 92263")
#['CA 92263']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<blockquote>
<p>to grab text starting and space, then exact <strong>2</strong> <strong>alphabets</strong>, then one
  space, then exact 5 digits.</p>
</blockquote>
<p>Unfortunately, this pattern <code>\b\w{2} \d{5}\b</code> will also find a match in such strings as <code>"P O BOX 32370, 2A 92263"</code> giving the result which doesn't fit your requirement. <code>\w</code> - matches all alpha<strong>numeric</strong> characters.<br/>To extract <em>State</em>, and <em>Zip code</em> use the following approach with <a href="https://docs.python.org/3/library/re.html#re.search" rel="nofollow noreferrer">re.search()</a> and <a href="https://docs.python.org/3/library/re.html#re.match.groupdict" rel="nofollow noreferrer">match.groupdict()</a> (gets all the named subgroups of the match) methods:</p>
<pre><code>s = 'P O BOX 32370, CA 92263'
m = re.search(r'\b(?P&lt;state&gt;[a-zA-Z]{2}) (?P&lt;zip_code&gt;\d{5})\b', s)
result = m.groupdict() if m else ''

print(result)
</code></pre>
<p>The output:</p>
<pre><code>{'zip_code': '92263', 'state': 'CA'}
</code></pre>
</div>
