<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/reference/expressions.html#boolean-operations" rel="nofollow noreferrer"><code>or</code> operator</a> to assign a "default value" to a variable. This works because <code>a or b</code> is equivalent to <code>a if a else b</code>. Thus you could write</p>
<pre><code>lista = GETLISTA() or some_default
</code></pre>
<p>Notice, however, that this will also use <code>some_default</code> if <code>GETLISTA()</code> evaluates to an empty list <code>[]</code>, or any other "falsy" value, not only if it returns <code>None</code>. This may or may not be a problem in your case. Alternatively, use the ternary operator</p>
<pre><code>lista = GETLISTA()
lista = some_default if lista is None else lista
</code></pre>
<p>or a simple <code>if</code> check (slightly longer, but IMHO clearer than the ternary):</p>
<pre><code>lista = GETLISTA()
if lista is None:
    lista = some_default
</code></pre>
