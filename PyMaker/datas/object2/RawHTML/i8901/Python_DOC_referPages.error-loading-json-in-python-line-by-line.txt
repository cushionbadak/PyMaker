<div class="post-text" itemprop="text">
<p>You appear to be processing the entire file one line at a time. Why not simply use <a href="https://docs.python.org/3/library/io.html?highlight=read#io.TextIOBase.read" rel="nofollow noreferrer"><code>.read()</code></a> to get the entire contents at once, then feed that to json?</p>
<pre><code>with open('file') as f:
    contents = f.read()
    data = json.loads(contents)
</code></pre>
<p>Better yet, why not use <code>json.load()</code> to pass the readable directly and let it handle the slurping?</p>
<pre><code>with open('file') as f:
    data = json.load(f)
</code></pre>
</div>

<p><code>,}</code> is not allowed in JSON (I guess that's the problem according to the data given).</p>

<div class="post-text" itemprop="text">
<p>The problem is in your reading and decoding the file line by line. Any single line in your file (e.g., <code>"[{"</code>) is not a valid JSON expression. </p>
</div>
<div class="post-text" itemprop="text">
<p>Your individual lines are not valid JSON. For instance, the first line '[{' by itself is not a valid JSON. If your entire file is actually valid JSON and you want individual lines, first load the entire JSON and then browse through the python dictionary.</p>
<pre><code>import json

data = json.loads(open('file').read()) # this should be a list
for list_item in data:
    print(list_item['name'])
</code></pre>
