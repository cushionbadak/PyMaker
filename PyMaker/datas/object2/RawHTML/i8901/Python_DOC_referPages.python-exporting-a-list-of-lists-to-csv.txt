<div class="post-text" itemprop="text">
<p>With the structure:</p>
<pre><code>mylist = [ [a,b,c,d], [e,f,g,h], .......[w,x,y,z]]
</code></pre>
<p>How can I export to a .csv file using each column and row like?  Is there useful functions to quickly format lists to .csv files this way?  I want each value in list to be a new tab in excel.</p>
<pre><code>a   b   c   d
e   f   g   h
............
w   x   y   z
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use :</p>
<pre><code>sep = "\t"

mylist = [['a', 'b', 'c'], ['d', 'e', 'f']]

with open('my.csv', 'w') as csv:
    for row in mylist:
        csv.write(sep.join(row))
        csv.write("\n")
</code></pre>
<p>Or the official <a href="https://docs.python.org/2/library/csv.html" rel="nofollow noreferrer">csv lib</a>.</p>
<p><code>my.csv</code> is now :</p>
<pre><code>a   b   c
d   e   f
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I think this should do what you want.</p>
<pre><code>csv_lines = []
for line in mylist:
    csv_lines.append(",".join(line))

f = open("mylist.csv", "w")
f.write("\n".join(csv_lines))
f.close()
</code></pre>
<p>The file <code>mylist.csv</code> should then be read correctly by excel automatically.</p>
</div>
<div class="post-text" itemprop="text">
<p>This is an efficient way of doing it if you list is not VERY huge</p>
<pre><code>with open('filename.csv', 'w') as file_handle:
    file_handle.write(map(lambda x:x.join(", "), mylist).join("\n"))
</code></pre>
