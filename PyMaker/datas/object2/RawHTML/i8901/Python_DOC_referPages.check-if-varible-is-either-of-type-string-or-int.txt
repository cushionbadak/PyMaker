<div class="post-text" itemprop="text">
<p>As a matter of fact,you can use a tuple of classes as 2nd argument to <code>isinstance</code>.</p>
<pre><code>isinstance(var,(list,int))
</code></pre>
<p><strong>isinstance(object, classinfo)</strong></p>
<blockquote>
<p>If classinfo is a tuple of class or type objects (or recursively,
  other such tuples), return true if object is an instance of any of the
  classes or types. If classinfo is not a class, type, or tuple of
  classes, types, and such tuples, a TypeError exception is raised.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>You could use boolean <code>or</code> :</p>
<pre><code>&gt;&gt;&gt; isinstance(3,int) or isinstance(3,str)
True
&gt;&gt;&gt; isinstance("3",int) or isinstance("3",str)
True
&gt;&gt;&gt; isinstance([3],int) or isinstance([3],str)
False
</code></pre>
<p>or specify a tuple of possible types :</p>
<pre><code>&gt;&gt;&gt; isinstance(3, (int, str))
True
&gt;&gt;&gt; isinstance("3", (int, str))
True
&gt;&gt;&gt; isinstance([3], (int, str))
False
</code></pre>
<p>So in your case, it wouldn't be :</p>
<pre><code>isinstance(data(int, str))
</code></pre>
<p>but</p>
<pre><code>isinstance(data, (int, str))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <strong>isinstance(object, classinfo)</strong> built-in function.</p>
<p><a href="https://repl.it/Fw63/1" rel="nofollow noreferrer">This</a> should do the job</p>
<pre><code>def isStrOrInt(myVar):
  return isinstance(myVar,int) or isinstance(myVar,str)


print(isStrOrInt(5))   # True
print(isStrOrInt('5')) # True
print(isStrOrInt([]))  # False
print(isStrOrInt('Dinosaurs are Awesome!')) # True
print(isStrOrInt(3.141592653589793238462 )) # False (Because it's a decimal not an int)
</code></pre>
<p>Here's the running version: <a href="https://repl.it/Fw63/1" rel="nofollow noreferrer">https://repl.it/Fw63/1</a></p>
<hr/>
<p>This is what the <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer">documentation</a> says:</p>
<blockquote>
<p>Return true if the object argument is an instance of the classinfo
  argument, or of a (direct, indirect or virtual) subclass thereof. If
  object is not an object of the given type, the function always returns
  false. If classinfo is a tuple of type objects (or recursively, other
  such tuples), return true if object is an instance of any of the
  types. If classinfo is not a type or tuple of types and such tuples, a
  TypeError exception is raised.</p>
</blockquote>
<p><em>Basically, <code>isinstance</code> takes 2 parameters, the first is the variable you want to check, and the second is the data type/ class you want to check if it is an instance of (in your case <code>int</code> or <code>str</code>). <a href="http://effbot.org/pyfaq/how-do-i-check-if-an-object-is-an-instance-of-a-given-class-or-of-a-subclass-of-it.htm" rel="nofollow noreferrer">This tutorial</a> explains it well.</em></p>
