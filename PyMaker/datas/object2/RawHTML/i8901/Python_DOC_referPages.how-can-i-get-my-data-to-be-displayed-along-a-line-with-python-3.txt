<div class="post-text" itemprop="text">
<p>Since you tagged Python 3.6, use the new <a href="https://www.python.org/dev/peps/pep-0498/" rel="nofollow noreferrer">f-strings</a>:</p>
<pre><code>import collections

text = input()
symbols = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'
letters = collections.Counter(text.upper())
length = len(text)
print(f'Length of message: {length} characters')
for letter, times in sorted(letters.items()):
    if letter not in symbols:
        continue
    percent = times / length
    print(f'{letter} {times:2} {percent:5.1%}')
</code></pre>
<p>There is no need to compute the percent string manually.  Just compute the float value <code>percent = times / length</code> and use the proper formatting in the f-string.</p>
<p><code>{percent:5.1%}</code> means: insert "percent" variable here in a field of width 5, with one place after the decimal.  <code>%</code> is a format specifier that multiplies the number by 100 and adds a percent sign.  <code>{letter}</code> is inserted with no special formatting, and <code>{times:2}</code> makes a 2-wide field that is right-justified for numbers by default.</p>
<p>Output of input "abbbbbbbbbbccc":</p>
<blockquote>
<pre><code>Length of message: 14 characters
A  1  7.1%
B 10 71.4%
C  3 21.4%
</code></pre>
</blockquote>
