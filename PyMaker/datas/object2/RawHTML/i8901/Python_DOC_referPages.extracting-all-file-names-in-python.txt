<p>I have a application that converts from one photo format to another by inputting in <code>cmd.exe</code> following:  "AppConverter.exe" "file.tiff" "file.jpeg"</p>
<p>But since i don't want to input this every time i want a photo converted, i would like a script that converts all files in the folder. So far i have this:</p>
<pre><code>  def start(self):
    for root, dirs, files in os.walk("C:\\Users\\x\\Desktop\\converter"):
     for file in files:
        if file.endswith(".tiff"):
          subprocess.run(['AppConverter.exe', '.tiff', '.jpeg'])
</code></pre>
<p>So how do i get the names of all the files and put them in <code>subprocess</code>. I am thinking taking basename (no ext.) for every file and pasting it in <code>.tiff</code> and <code>.jpeg</code>, but im at lost on how to do it.</p>

<div class="post-text" itemprop="text">
<p>I think the fastest way would be to use the <a href="https://docs.python.org/3/library/glob.html" rel="nofollow noreferrer"><code>glob</code></a> module for expressions:</p>
<pre><code>import glob
import subprocess

for file in glob.glob("*.tiff"):
    subprocess.run(['AppConverter.exe', file, file[:-5] + '.jpeg'])
    # file will be like 'test.tiff'
    # file[:-5] will be 'test' (we remove the last 5 characters, so '.tiff'
    # we add '.jpeg' to our extension-less string
</code></pre>
<p>All those informations are on the post I've linked in the comments o your original question.</p>
