<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/ast.html" rel="nofollow noreferrer"><code>ast.literal_eval</code></a>:</p>
<pre><code>&gt;&gt;&gt; import ast
&gt;&gt;&gt; s = "['0101612-0', '0101611-2', '0101610-4', '0101608-8', '0101607-0', '0101606-2', '0100786-3', '0100784-8', '0100783-0', '0100782-2', '0100781-4', '0100780-6', '0100779-8', '0100778-0', '0100777-2']"
&gt;&gt;&gt; ast.literal_eval(s)
['0101612-0', '0101611-2', '0101610-4', '0101608-8', '0101607-0', '0101606-2', '0100786-3', '0100784-8', '0100783-0', '0100782-2', '0100781-4', '0100780-6', '0100779-8', '0100778-0', '0100777-2']
</code></pre>
<p>You can do it with <code>split</code> &amp; <code>strip</code> too but it's bit more complex:</p>
<pre><code>&gt;&gt;&gt; [x.strip('\'') for x in s.strip('[]').split(', ')]
['0101612-0', '0101611-2', '0101610-4', '0101608-8', '0101607-0', '0101606-2', '0100786-3', '0100784-8', '0100783-0', '0100782-2', '0100781-4', '0100780-6', '0100779-8', '0100778-0', '0100777-2']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The simplest (and probably most dangerous) way to do this is to use <code>eval</code>.</p>
<pre><code>my_list = eval(list_example)
</code></pre>
<p>This code should return a list object full of strings.  Warning:  make sure the user can't input any malicious code inside of <code>list_example</code>!</p>
