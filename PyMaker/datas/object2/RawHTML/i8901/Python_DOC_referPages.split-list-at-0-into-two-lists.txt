<p>I have a list:</p>
<pre><code>matrix = [1, 2, 3, 0, 4, 5, 0, 6]
</code></pre>
<p>which needs to be split into a random amount of lists at 0 (ignoring 0):</p>
<pre><code>matrix1 = [3, 2, 1]
matrix2 = [4, 5]
matrix3 = [6]
</code></pre>
<p>How would I go about doing this? The original list will be of random length and with random numbers.</p>
</div>
<div class="post-text" itemprop="text">
<p>One option is to use <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="noreferrer"><code>groupby</code></a>:</p>
<pre><code>from itertools import groupby

lst = [1, 2, 3, 0, 4, 5, 0, 6]
[list(g) for k, g in groupby(lst, lambda x: x != 0) if k]
# [[1, 2, 3], [4, 5], [6]]
</code></pre>
</div>

<span class="comment-copy">What do 'k' and 'g' mean in this context?</span>
<span class="comment-copy"><code>groupby</code> generates an iterable of tuples, the first element of the tuple is the key, the second element of the tuple is the grouped elements, so k, g are the key and grouped elements correspondingly, to see this better, you can check these two expressions <code>[(k, list(g)) for k, g in groupby(lst, lambda x: x!= 0)]</code> and <code>list(groupby(lst, lambda x: x!=0))</code></span>
