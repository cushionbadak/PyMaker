<p>In the following code:</p>
<pre><code>def b(i: int) -&gt; int:
    return i

def a(i: int, b: ?) -&gt; int:
    return i + b(i)

print(a(1, b))
</code></pre>
<p>How do we type hinting the function <code>b: ?</code> that is a parameter of <code>a</code>?  Thank you.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/3/library/typing.html#typing.Callable" rel="noreferrer"><code>typing.Callable</code> generic</a> to indicate you expect a function that takes a single integer argument, returning an integer:</p>
<pre><code>from typing import Callable

def a(i: int, b: Callable[[int], int]) -&gt; int:
    return i + b(i)
</code></pre>
<p><code>Callable[...]</code> takes two arguments, the first a list of argument types, the second the return type.</p>
