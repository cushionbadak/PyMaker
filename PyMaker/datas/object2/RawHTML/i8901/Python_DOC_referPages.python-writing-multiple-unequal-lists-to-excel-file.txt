<div class="post-text" itemprop="text">
<p>I think this works for you:</p>
<pre><code>import csv
from itertools import zip_longest


ANNA = (1, 2, 3, 4)
MAE = (1, 2)
CEL = (1, 2, 3, 4, 5, 6, 7)

with open('output.csv', 'w') as outcsv:
    writer = csv.writer(outcsv)
    writer.writerow(["ANNA", "MAE", "CEL"])
    for row in zip_longest(ANNA, MAE, CEL):
        writer.writerow(row)
</code></pre>
<p>Reference:</p>
<ul>
<li>zip_longest: <a href="https://pymotw.com/3/itertools/index.html#merging-and-splitting-iterators" rel="nofollow noreferrer">https://pymotw.com/3/itertools/index.html#merging-and-splitting-iterators</a></li>
</ul>
