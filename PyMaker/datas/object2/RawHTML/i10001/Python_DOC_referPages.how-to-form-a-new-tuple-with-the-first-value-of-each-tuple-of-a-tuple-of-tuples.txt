<p>I want to form a tuple with the first elements of each tuple. Im a beginner in python, maybe this is an easy one but i cant find a way to do it plz help.</p>
<p>ex:</p>
<p>input</p>
<pre><code> x = (('A','B','C'),('D','E','F'),('G','H','I'))
</code></pre>
<p>output:</p>
<pre><code>y= ('A', 'D', 'G')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use comprehension and convert to a tuple;</p>
<pre><code>&gt;&gt;&gt; x = (('A','B','C'),('D','E','F'),('G','H','I'))
&gt;&gt;&gt; y = tuple([l[0] for l in x])
&gt;&gt;&gt; y
('A', 'D', 'G')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>x = (('A','B','C'),('D','E','F'),('G','H','I'))

for each in zip(*x):
    print(each)
</code></pre>
<p>prints:</p>
<pre><code>('A', 'D', 'G')
('B', 'E', 'H')
('C', 'F', 'I')
</code></pre>
<p>Use the <code>zip()</code> function. <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow">Read more here.</a></p>
<p>Edited:</p>
<p>like CAB says if you just want the first one you can do:</p>
<pre><code>y = zip(*x)[0] 
</code></pre>
