<p>I created simple python script to do infinite loop to test my android app, however it only runs through succesfully once and stops.</p>
<pre><code># Imports the monkeyrunner modules used by this program
from com.android.monkeyrunner import MonkeyRunner, MonkeyDevice

# Connects to the current device, returning a MonkeyDevice object
device = MonkeyRunner.waitForConnection('66b6cc0e')

while True:
    device.touch (300, 1750, 'DOWN_AND_UP')
    MonkeyRunner.sleep(3)
    device.touch(742, 1213, 'DOWN_AND_UP')
    MonkeyRunner.sleep(10)
    device.touch(554, 1613, 'DOWN_AND_UP')
    MonkeyRunner.sleep(10)

# Push SEND MESSAGE
device.touch(300, 1750, 'DOWN_AND_UP')
MonkeyRunner.sleep(3)
device.touch(742, 1213, 'DOWN_AND_UP')
MonkeyRunner.sleep(10)
device.touch(554, 1613, 'DOWN_AND_UP')
MonkeyRunner.sleep(10)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You should have each of the commands within a <code>try</code> (<a href="https://docs.python.org/3/tutorial/errors.html" rel="nofollow">https://docs.python.org/3/tutorial/errors.html</a>). It's not preferable to have a bare except (i.e. a catch-all except), but you can try it for debugging. </p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://github.com/dtmilano/AndroidViewClient" rel="nofollow">AndroidViewClient/culebra</a> which is almost a drop in replacement for <code>monkeyruner</code> but it is pure python.</p>
<p>I made the changes to your script to show that</p>
<pre><code>#! /usr/bin/env python

# Imports the monkeyrunner modules used by this program
#from com.android.monkeyrunner import MonkeyRunner, MonkeyDevice

import re
import sys
import os

from com.dtmilano.android.viewclient import ViewClient


# Connects to the current device, returning a MonkeyDevice object
#device = MonkeyRunner.waitForConnection('66b6cc0e')

kwargs1 = {'ignoreversioncheck': False, 'verbose': False, 'ignoresecuredevice': False}
device, serialno = ViewClient.connectToDeviceOrExit(**kwargs1)


while True:
    print "loop"
    device.touch (300, 1750, 'DOWN_AND_UP')
    ViewClient.sleep(3)
    device.touch(742, 1213, 'DOWN_AND_UP')
    ViewClient.sleep(10)
    device.touch(554, 1613, 'DOWN_AND_UP')
    ViewClient.sleep(10)

# Push SEND MESSAGE
device.touch(300, 1750, 'DOWN_AND_UP')
ViewClient.sleep(3)
device.touch(742, 1213, 'DOWN_AND_UP')
ViewClient.sleep(10)
device.touch(554, 1613, 'DOWN_AND_UP')
ViewClient.sleep(10)
</code></pre>
