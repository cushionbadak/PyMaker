<div class="post-text" itemprop="text">
<p>The XML elements are in the default namespace <code>urn:ebay:apis:eBLBaseComponents</code> So all tags get prepended with the full URI of the namespace.</p>
<p>Changing your <code>find()</code> parameter to include the URI should work.</p>
<pre><code>import xml.etree.ElementTree as ET
XML = #Above Code in String
tree = ET.ElementTree(ET.fromstring(XML))
root = tree.getroot()
for Category in root[0]:
    one = Category.find("{urn:ebay:apis:eBLBaseComponents}WantedParm1").text
    two = Category.find("{urn:ebay:apis:eBLBaseComponents}WantedParm2").text
    print(one, two)
</code></pre>
<p>You can find more information on dealing with namespaces on the element tree docs <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#parsing-xml-with-namespaces" rel="nofollow noreferrer">here</a> or this stack overflow <a href="https://stackoverflow.com/questions/1181888/what-does-xmlns-in-xml-mean">post</a>.</p>
