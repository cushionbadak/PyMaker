<p>Is there a way in python to extract each substring thats inside a string?</p>
<p>For example if I have the string </p>
<pre><code>"Hello there my name is Python" 
</code></pre>
<p>I want to take out each sub-string (or individual word) from within this string so that I have <code>"Hello", "there" , "my" , "name" , "is"</code> and <code>"Python"</code> each taken out of this string?</p>
</div>
<div class="post-text" itemprop="text">
<p>I believe what you are looking for is the <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow">split</a> method.</p>
<p>It will break the string with specified delimeter. Default delimeter is a space.</p>
<pre><code>input_string = "Hello there my name is Python" 
for substring in input_string.split():
    print(substring)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use the <code>split()</code> string method.</p>
<pre><code>&gt;&gt;&gt; sentence = 'Hello there my name is Python'
&gt;&gt;&gt; words = sentence.split()
&gt;&gt;&gt; print words
['Hello', 'there', 'my', 'name', 'is', 'Python']
</code></pre>
