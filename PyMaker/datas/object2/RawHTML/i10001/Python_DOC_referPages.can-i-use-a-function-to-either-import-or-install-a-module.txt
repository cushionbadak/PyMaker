<div class="post-text" itemprop="text">
<p>The <code>import</code> statement takes the module name literally and not just as a reference to some other object. So <code>import mypack</code> does not translate to <code>import easygui</code>, but <em>import module mypack</em></p>
<p>Instead, you should use the builtin <a href="https://docs.python.org/3/library/functions.html#__import__" rel="nofollow"><code>__import__</code></a> which takes a name/string:</p>
<pre><code>__import__(mypack)
</code></pre>
