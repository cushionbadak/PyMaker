<div class="post-text" itemprop="text">
<p>You might want to consider using <a href="https://docs.python.org/3/library/csv.html#csv.DictReader" rel="nofollow"><code>DictReader</code></a>. It'll handle the header for you and allow you to access the columns by name rather than index (which can help with keeping code readable).</p>
<p>eg.</p>
<pre><code>import csv

with open('names.csv') as csvfile:
    reader = csv.DictReader(csvfile)
    row = next(reader)
    print(row['first_name'], row['last_name'])
</code></pre>
