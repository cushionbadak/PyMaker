<p>I would like to save a list of python dicts A into a JSON file B. I used</p>
<pre><code>json.dump(A, B)
</code></pre>
<p>to do that. But the saved JSON file's format is like </p>
<p>[{'a': 1, 'b': 1}, {'a':2, 'b':2}...] </p>
<p>What I want the display is to be something like:</p>
<pre><code>[ 
 {'a': 1, 'b': 1},
 {'a': 2, 'b': 2},
 ...
], 
</code></pre>
<p>so that others can easily read. 
Is there a way to do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/json.html#json.JSONEncoder" rel="nofollow"><code>indent</code></a> argument when using <code>json.dumps</code> (see end of section in link):</p>
<blockquote>
<p>If indent is a non-negative integer or string, then JSON array elements and object members will be pretty-printed with that indent level. An indent level of 0, negative, or "" will only insert newlines. None (the default) selects the most compact representation. Using a positive integer indent indents that many spaces per level. If indent is a string (such as "\t"), that string is used to indent each level.</p>
</blockquote>
<pre><code>&gt;&gt;&gt; print(json.dumps({1:'a', 2: 'b'}, indent=1))
{
 "1": "a",
 "2": "b"
}
</code></pre>
