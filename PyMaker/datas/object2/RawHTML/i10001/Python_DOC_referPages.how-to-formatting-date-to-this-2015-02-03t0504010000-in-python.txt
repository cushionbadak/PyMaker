<div class="post-text" itemprop="text">
<p>If you want to parse text to <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow"><code>datetime</code></a> object, use <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow"><code>strptime</code></a>:</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; datetime.datetime.strptime('2015-02-03T05:04:01+0000', '%Y-%m-%dT%H:%M:%S+0000')
datetime.datetime(2015, 2, 3, 5, 4, 1)
</code></pre>
<p>You can see the format I used (<code>'%Y-%m-%dT%H:%M:%S+0000'</code>) you can see the full details on how the format works on the link I provided, also keep in mind that you can use normal chars (like the <code>T</code> or <code>+0000</code>) to match the exact format you had.</p>

<p>Here Duan is right. But for the "+0000" you can use "%z".</p>
<pre><code>import datetime
print datetime.datetime.strftime(datetime.datetime.today(),'%Y-%m-%dT%H:%M:%S%z')
</code></pre>
<p>Other formatting char you can see here <a href="https://docs.python.org/2/library/datetime.html" rel="nofollow">https://docs.python.org/2/library/datetime.html</a></p>
