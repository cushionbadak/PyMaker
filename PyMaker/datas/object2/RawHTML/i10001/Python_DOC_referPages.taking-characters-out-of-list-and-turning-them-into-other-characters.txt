<div class="post-text" itemprop="text">
<p>You're inputting two letters, but your test conditions only contain one character each. You should iterate on the input string using a <code>for</code> and test each character in the string one at a time:</p>
<pre><code>before = input()

for i in before:
    if i=="A":
        print("Q")
    elif i=="B":
        print("W")
    elif i=="C":
        print("E")
    elif i=="D":
        print("R")
    else:
        print("--")
</code></pre>
<p>You can also improve your code by using a <em>mapping</em> instead of the <code>if/elif</code> as this will help you accommodate new translations more easily:</p>
<pre><code>before = input()
mapping = {'A': 'Q', 'B': 'W', 'C': 'E', 'D': 'R'}

after = ''
for i in before:
    after += mapping.get(i, '--') 

print(after)
</code></pre>
<p>Notice how the dictionary's <code>get</code> method was used to return the default <code>'--'</code> when the mapping does not contain the character.</p>

<div class="post-text" itemprop="text">
<p>You are reinventing the wheel, use <a href="https://docs.python.org/3/library/stdtypes.html#str.translate" rel="nofollow noreferrer"><code>str.translate</code></a>. To use <code>str.translate</code>, you need to first create a dictionary that can be used to map the original character to the new character. This transition table can be made using <a href="https://docs.python.org/3/library/stdtypes.html#str.maketrans" rel="nofollow noreferrer"><code>maketrans</code></a>. </p>
<pre><code>&gt;&gt;&gt; mapping = {'A': 'Q', 'B': 'W', 'C': 'E', 'D': 'R'}
&gt;&gt;&gt; str.maketrans(mapping)
{65: 'Q', 66: 'W', 67: 'E', 68: 'R'}
</code></pre>
<p>Now that we have the table ready, we just need to pass it to <code>translate</code>. </p>
<pre><code>&gt;&gt;&gt; table = str.maketrans(mapping)
&gt;&gt;&gt; "AB".translate(table)
'QW'
</code></pre>
<p>Now we get the desired result. </p>
