<div class="post-text" itemprop="text">
<p>This is because in python2 <a href="https://docs.python.org/2/library/functions.html#zip" rel="noreferrer"><code>zip()</code></a> returns a list of tuples, which <a href="http://matplotlib.org/api/path_api.html#matplotlib.path.Path" rel="noreferrer"><code>mpl.path.Path()</code></a> happily accepts. In python3, <a href="https://docs.python.org/3/library/functions.html#zip" rel="noreferrer"><code>zip()</code></a> returns an <a href="https://docs.python.org/3/tutorial/classes.html#iterators" rel="noreferrer">iterator</a>, which you must consume. You should be able to do something like:</p>
<pre><code>&gt;&gt;&gt; poly = mpl.path.Path(list(zip(listx, listy)))
</code></pre>
</div>
