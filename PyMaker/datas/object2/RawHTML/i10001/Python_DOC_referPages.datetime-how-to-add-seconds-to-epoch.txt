<div class="post-text" itemprop="text">
<p>Use a timedelta object, e.g:</p>
<pre><code>from datetime import datetime, timedelta

TIMESTRING_FORMAT = '%Y-%m-%dT%H:%M:%SZ'

dt = datetime.strptime('2016-11-04T03:02:00Z', TIMESTRING_FORMAT)
ndt = dt + timedelta(seconds=5)
print datetime.strftime(ndt, TIMESTRING_FORMAT)
</code></pre>
<p>See docs <a href="https://docs.python.org/2/library/datetime.html" rel="nofollow noreferrer">https://docs.python.org/2/library/datetime.html</a> and make sure that my TIMESTRING_FORMAT string is correct.</p>
