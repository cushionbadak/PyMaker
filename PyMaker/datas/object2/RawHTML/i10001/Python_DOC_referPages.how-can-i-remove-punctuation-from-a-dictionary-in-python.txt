<p>As PM 2Ring notes, a <code>Counter</code> object would be useful here or simply a <code>defaultdict</code> from the <code>collections</code> library.  We can use the regular expression package <code>re</code> for a more powerful <code>re.split()</code> or simply <code>re.findall()</code>:</p>
<pre><code>from re import findall, IGNORECASE
from operator import itemgetter
from collections import defaultdict

wordcount = defaultdict(int)

file = open("license.txt")

for vocab in findall(r"[A-Z]+", file.read(), flags=IGNORECASE):
    wordcount[vocab.lower()] += 1

for word, number in sorted(wordcount.items(), key=itemgetter(1), reverse=True):
    print(word, number)
</code></pre>

<p>There are always trade-offs: you might want to fine tune the pattern to allow hyphenated words or apostrophes, depending on your application.</p>
<p>Reading the entire file in and processing it is fine if the input file is relatively small.  If not, read it line by line in a loop with <code>readline()</code> instead and process each line in turn.</p>
