<p>Check this snippet. You can apply regexps to selectors:</p>
<pre><code>&gt;&gt;&gt; from scrapy import Selector
&gt;&gt;&gt; txt = """&lt;img src="//d30v2pzvrfyzpo.cloudfront.net/uk/images/restaurants/65115.gif" itemprop="image" alt="Gourmet Sushi" width="100" height="100" onerror="this.onerror = null;this.src = '//dy3erx8o0a6nh.cloudfront.net/images/serp/serp_default_logo.svg';"&gt;"""
&gt;&gt;&gt; sel = Selector(text=txt)
&gt;&gt;&gt; sel.re("src\s?=\s?['\"]([^'\"]+)['\"]")
[u'//d30v2pzvrfyzpo.cloudfront.net/uk/images/restaurants/65115.gif', u'//dy3erx8o0a6nh.cloudfront.net/images/serp/serp_default_logo.svg']
</code></pre>
<p>So, and then you can do <code>[response.urljoin(url) for url in sel.re("src\s?=\s?['\"]([^'\"]+)['\"]")]</code></p>
