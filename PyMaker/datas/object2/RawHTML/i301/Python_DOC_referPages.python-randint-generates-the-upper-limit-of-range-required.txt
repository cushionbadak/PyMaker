<div class="post-text" itemprop="text">
<p>You are wrong here: "According to the documentations randint should give me a number between 0 and len(order) -1."</p>
<p>The doc says: "random.randint(a, b). Return a random integer N such that a &lt;= N &lt;= b. Alias for randrange(a, b+1)."</p>
<p>See it here: <a href="https://docs.python.org/3/library/random.html#random.randint" rel="nofollow noreferrer">https://docs.python.org/3/library/random.html#random.randint</a></p>
</div>
<div class="post-text" itemprop="text">
<p>I recommend using <em>shuffle</em> from module <em>random</em>. I think this is what you need:</p>
<pre><code>import random

ordered = list(range(10000))
random.shuffle(ordered)
print(ordered)
</code></pre>
<p>Keep in mind that shuffle <strong>does not return the shuffled list</strong>. It is shuffeling the list you gave to it. </p>
