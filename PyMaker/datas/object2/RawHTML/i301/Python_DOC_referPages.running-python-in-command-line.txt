<div class="post-text" itemprop="text">
<p>The best approach to your problem would be to write out a <code>.py</code> file and run that.</p>
<p>However, I can imagine that for scripting purposes you may need something like this if you cannot write a file. In that case you could separate your commands by <code>;</code>, but you are going to be limited to programs without blocks, e.g.:</p>
<pre><code>python3 -c "import math; a = 10; print('Hello {} {}'.format(a, math.sin(a)))"

# prints:
# Hello 10 -0.5440211108893698
</code></pre>
<p>but:</p>
<pre><code>python3 -c "import math; a = 10; for i in range(a): print('Hello {} {}'.format(i, math.sin(i)))"
</code></pre>
<blockquote>
<p>SyntaxError: invalid syntax</p>
</blockquote>
<p>but again:</p>
<pre><code>python3 -c "import math; a = 10; [print('Hello {} {}'.format(i, math.sin(i))) for i in range(a)]"

# prints:
# Hello 0 0.0
# Hello 1 0.8414709848078965
# Hello 2 0.9092974268256817
# Hello 3 0.1411200080598672
# Hello 4 -0.7568024953079282
# Hello 5 -0.9589242746631385
# Hello 6 -0.27941549819892586
# Hello 7 0.6569865987187891
# Hello 8 0.9893582466233818
# Hello 9 0.4121184852417566
</code></pre>
<p>the last one is of course bad practice and all (a side-effect in a comprehension, etc.) but kind of works.</p>
</div>

</code></pre>
