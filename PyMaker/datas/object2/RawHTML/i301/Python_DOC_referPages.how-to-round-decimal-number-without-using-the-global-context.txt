<div class="post-text" itemprop="text">
<p>Use the method <a href="https://docs.python.org/3/library/decimal.html#decimal.Decimal.quantize" rel="nofollow noreferrer">Decimal.quantize</a>.</p>
<pre><code>from decimal import *

myNumber = Decimal(1.101901)
roundedNumber = myNumber.quantize(Decimal('1.00'), rounding=ROUND_DOWN) # 1.10
roundedNumber = myNumber.quantize(Decimal('1.00'), rounding=ROUND_UP)   # 1.11
</code></pre>
