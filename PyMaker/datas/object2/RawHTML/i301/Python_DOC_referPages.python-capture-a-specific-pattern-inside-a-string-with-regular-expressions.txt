<div class="post-text" itemprop="text">
<p>too many unrequired and harmful symbols in your expression.</p>
<p>Using surrounding <code>[]</code> made <code>findall</code> match individual characters, which explains the output you're getting.</p>
<p>This needs a full rethink: escape the parentheses, use <code>\d+</code> to match one or more digits, and explicit <code>cm</code> and spaces.</p>
<p>create a group to match only digits+unit, use <code>search</code> to find the group and display it.</p>
<pre><code>import re
string = '6\' 3" ( 190 cm )'
pattern = re.compile(r'\( (\d+ cm) \)')

&gt;&gt;&gt; pattern.search(string).group(1)
'190 cm'
</code></pre>
</div>

<div class="post-text" itemprop="text">
<p>With regular expressions:</p>
<pre><code>import re

s = '6\' 3" ( 190 cm )'
desired_output = re.search(r'\((.*?)\)',s).group(1).lstrip()

print(desired_output)
&gt;&gt;&gt; 190 cm
</code></pre>
<p>Without regular expressions: </p>
<pre><code>s = '6\' 3" ( 190 cm )'
desired_output = s[s.find("(")+1:s.find(")")].lstrip()

print(desired_output)
&gt;&gt;&gt; 190 cm
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could  use a capturing group which will be returned by <a href="https://docs.python.org/3/library/re.html?highlight=re%20findall#re.findall" rel="nofollow noreferrer">findall</a>:</p>
<pre><code>\(\s*([0-9]+\s*[a-z]+)\s*\)
</code></pre>
<p>That will match:</p>
<ul>
<li><code>\(\s*</code> match <code>(</code> and 0+ times a whitespace char</li>
<li><code>(</code> Capturing group

<ul>
<li><code>[0-9]+\s*[a-z]+</code> Match 1+ a digit, 0+ times a whitespace char and 1+ times a-z (or use <code>cm</code> instead of <code>[a-z]+</code> if you want to match that literally)</li>
</ul></li>
<li><code>)</code> Close capturing group</li>
<li><code>\s*\)</code> Match 0+ times a whitespace char</li>
</ul>
<p><a href="https://regex101.com/r/B40WhD/1/" rel="nofollow noreferrer">regex101 demo</a> | <a href="https://ideone.com/wPKSdu" rel="nofollow noreferrer">Python demo</a></p>
<p>For example:</p>
<pre><code>import re

string = '6\' 3" ( 190 cm )'
pattern = re.compile(r"\(\s*([0-9]+\s*[a-z]+)\s*\)")
print(pattern.findall(string))
</code></pre>
</div>
