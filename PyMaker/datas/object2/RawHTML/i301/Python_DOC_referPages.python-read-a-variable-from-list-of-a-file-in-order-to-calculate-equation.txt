<div class="post-text" itemprop="text">
<p>it's clear <a href="https://docs.python.org/3/tutorial/inputoutput.html" rel="nofollow noreferrer">here</a> that if you read a file with readlines(), it will return a string. So, when you want to do a math calculation, you need to convert it to integer first</p>
<pre><code>f=open('list.txt')
lines=f.readlines()

k1=int(lines[0]) # convert to integer

r1=k1*2  #this is first equation 
print (r1) 

k2=int(lines[1])  #second equation
r2=k2*2
print (r2) 

k3=int(lines[2])
r3=k3*3
print (r3)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>when you use the * operator on strings, you multiple them.
for example:  <code>"a"*3</code> will be <code>"aaa"</code>
and when you use it on numbers you perform the math operation.
for example: <code>3*6=18</code></p>
<p>each line is string, so you need to cast the line to int:  <code>int(line)</code>.</p>
<p>also, you should consider use the <a href="https://www.journaldev.com/22960/python-map-function" rel="nofollow noreferrer">map</a> function.</p>
<p>it will make your life easier.</p>
<p>map perform given function on every item in the list.</p>
<p>for example:</p>
<pre><code>result = list(map(lambda x: int(x) * 3, lines))
</code></pre>
<p>it takes every item in the list, cast it to int, and multiply it by 3.</p>
</div>
