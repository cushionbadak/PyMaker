<div class="post-text" itemprop="text">
<p>There are some things wrong. The main one is that you're not concatenating the lists, with the parenthesis you're creating a <code>tuple</code> of size 2 which first element will be the first list, and the second the second one. Everywhere where you are using parenthesis is a <code>tuple</code>, if you want <code>lists</code> use square brackets. To concatenate two lists, use the operator <code>+</code>.</p>
<pre class="lang-py prettyprint-override"><code>alist = list(range(1,100))
blist = list(range(600,700))
newListA = list(filter(lambda x: x%3==0 and x%5==0, alist))
newListB = list(filter(lambda x: x%3==0 and x%5==0, blist))
newListC = newListA + newListB
print(newListC)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can simply use the inbuilt <code>extend</code> functionality. <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer">Refer to Python docs.</a></p>
<pre><code>&gt;&gt;&gt; alist = list(range(1,100))

&gt;&gt;&gt; blist = list(range(600,700))

&gt;&gt;&gt; newListA = list(filter(lambda x: x%3==0 and x%5==0, alist))

&gt;&gt;&gt; newListB = list(filter(lambda x: x%3==0 and x%5==0, blist))

&gt;&gt;&gt; print(newListA.extend(newListB))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>To sum up and clarify, both <a href="https://stackoverflow.com/a/54875701/5049813">@Rushiraj Nenuji</a> and <a href="https://stackoverflow.com/a/54875594/5049813">@JosepJoestar</a> are correct. There are two ways to concatenate lists.</p>
<p>One is <code>old_list.extend(new_list)</code> which takes the <code>new_list</code> and concatenates <code>old_list</code> to it.</p>
<p>With this method, your code could be</p>
<pre><code>alist = list(range(1,100))
blist = list(range(600,700))

newListA = list(filter(lambda x: x%3==0 and x%5==0, alist))
newListB = list(filter(lambda x: x%3==0 and x%5==0, blist))
newListC = list()
newListC.extend(newListA)
newListC.extend(newListB)
# newListC is now the concatenation of newListA and newListB
</code></pre>
<p>Another way is to just use the <code>+</code> sign. So <code>list1 + list2</code> will return a value that concatenates both <code>list1</code> and <code>list2</code>.</p>
<p>With this method, your code could be</p>
<pre><code>alist = list(range(1,100))
blist = list(range(600,700))

newListA = list(filter(lambda x: x%3==0 and x%5==0, alist))
newListB = list(filter(lambda x: x%3==0 and x%5==0, blist))
newListC = newListA + newListB
# newListC is now the concatenation of newListA and newListB
</code></pre>
</div>
