<div class="post-text" itemprop="text">
<p>You need to write <code>[^a-zA-Z]</code> instead of <code>[^a-z]+[^A-Z]</code>. The <code>+</code> operator is for detecting repetitive characters and not to combine multiple conditions.</p>
<p>Try the below code for your requirement:    </p>
<pre><code>import re
string=re.sub('[^a-zA-Z]',' ',string)
print(string)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try using a list comprehension to check if each character is in <code>string.ascii_letters</code> or not, if it is, it will be stored:</p>
<pre><code>import string
String = '   529Wind3@.     '
print(''.join([i for i in String if i in string.ascii_letters]))
</code></pre>
<p>Output:</p>
<pre><code>Wind
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I agree with @U8-Forward's point but I think you may also want to know why your regular expression isn't working. This</p>
<pre><code>[^a-z]+[^A-Z]
</code></pre>
<p>doesn't do what you want because <code>W</code> matches <code>[^a-z]+</code> and so gets removed.</p>
<p>Put all of the characters you don't want in a single character class:</p>
<pre><code>[^a-zA-Z]+
</code></pre>
</div>
