<p><code>X</code> is a single number. Essentially I want to check in the list <code>listaa</code> to know if <code>x</code> is less than or equal to any element in <code>listaa</code>. I have,</p>
<pre><code>if x &lt;= listaa.any():
            continue
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just check if your value is less than or equal to the maximum of your list:</p>
<pre><code>if x &lt;= max(listaa):
    continue
</code></pre>
<p>If you wish to use <code>any</code>, you can use a generator expression. It's a <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer">built-in function</a> rather than a list method.</p>
<pre><code>if any(x &lt;= i for i in listaa):
    continue
</code></pre>
</div>
<span class="comment-copy">Lists by themselves don't have an <code>any()</code> method. But There is an built-in <code>any</code> function that you can use a generator expression to check your condition against all the list items and pass it to the <code>any()</code>. Like <code>any(x &lt;= i for i in listaa)</code>.</span>
