<p>I have multiple str-variables and I want to concatenate them to single line but with separator:</p>
<pre><code>var_a + 'SEPARATOR' + var_b + 'SEPARATOR' + var_c + 'SEPARATOR' + var_d
</code></pre>
<p>As you can see, it looks disgusting and unefficient.</p>
<p>So, is there any way to make it simpler by adding separator automatically between separate strings?</p>
<p>P.S.
I like the way it works in print function:</p>
<pre><code>print(var_a, var_b, var_c, var_d, sep="SEPARATOR")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>join()</code> function:   </p>
<pre><code>    words = [var_a,var_b,var_c,var_d]
    my_separator = "this is a separator"
    my_expected_sentence= my_separator.join(words)
</code></pre>
<p>Update:
Also you can use your custom method:</p>
<pre><code>def my_join(args, sep):
  sentence=""
  for i in args:
      sentence+=i+sep
  return sentence

words=["abc","def","ghi"]
data = my_join(words,'hello')
print(data)
</code></pre>
<p>In this way you can add separator before every words.Like:</p>
<pre><code>def my_preceding_join(args, sep):
      sentence=""
      for i in args:
          sentence+=sep+i
      return sentence

    data = my_preceding_join(words,'hello')
    print(data)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>str.join</code>:</p>
<pre><code>def custom_join(*args, sep=''):
  return sep.join(args)

print(custom_join('a', 'b', 'c', 'd', 'e', 'f', sep="SEPARATOR"))
</code></pre>
<p>Output:</p>
<pre><code>aSEPARATORbSEPARATORcSEPARATORdSEPARATOReSEPARATORf
</code></pre>
</div>

res = 'SEPARATOR'.join(vars)
</code></pre>

<div class="post-text" itemprop="text">
<p>Use <code>join</code>:</p>
<p>If your variables are strings, you can simply do:</p>
<pre><code>'SEPARATOR'.join([var_a, var_b, var_c, var_d])
</code></pre>
<p>If they aren't, you have to convert them to strings:</p>
<pre><code>'SEPARATOR'.join(map(str, [var_a, var_b, var_c, var_d]))
</code></pre>
</div>
