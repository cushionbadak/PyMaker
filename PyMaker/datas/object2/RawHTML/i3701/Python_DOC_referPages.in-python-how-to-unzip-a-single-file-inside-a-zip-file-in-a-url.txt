<p>In Python 3 I have a program to extract files from a zip in a link (zipurl):</p>
<pre><code>from io import BytesIO
from urllib.request import urlopen
from zipfile import ZipFile

zipurl = (f'http://agencia.tse.jus.br/estatistica/sead/odsele/prestacao_contas/prestacao_final_2014.zip')

with urlopen(zipurl) as zipresp:
    with ZipFile(BytesIO(zipresp.read())) as zfile:
        zfile.extractall('doacoes_2014')
</code></pre>
<p>Please, is there a way to unzip only one file? In case I need the file "receitas_candidatos_2014_brasil.txt"</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/zipfile.html#zipfile.ZipFile.extract" rel="nofollow noreferrer">ZipFile.extract()</a> to extract a single file instead of <a href="https://docs.python.org/3/library/zipfile.html#zipfile.ZipFile.extractall" rel="nofollow noreferrer">.extractall()</a>:</p>
<pre><code>zfile.extract('receitas_candidatos_2014_brasil.txt', 'doacoes_2014')
</code></pre>
