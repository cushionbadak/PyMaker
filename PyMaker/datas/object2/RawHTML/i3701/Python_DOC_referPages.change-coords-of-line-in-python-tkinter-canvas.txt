<p>I've drawn a line in <code>tkinter.Canvas</code> and now I want to move one end. Is this possible, e.g. with <code>itemconfig</code>?</p>
<pre><code>import tkinter

tk = tkinter.Tk()
canvas = tkinter.Canvas(tk)
canvas.pack()
line = canvas.create_line(0, 0, 100, 100)
tk.mainloop()
</code></pre>
<p>Now I want to change the end of the line to <code>75, 25</code>. Is there a better way as to delete the line and create a new one? </p>
</div>
<div class="post-text" itemprop="text">
<p>Once you've created some item on canvas you can use the following methods to edit the item:</p>
<ul>
<li><strong>coords( )</strong></li>
<li><strong>itemconfig( )</strong></li>
<li><strong>move( )</strong></li>
</ul>
<p><strong><em>For example:</em></strong></p>
<pre><code>from tkinter import *

root = Tk()
w = Canvas(root, width=200, height=200)
w.pack()
var = w.create_line(0, 0, 100, 100)
w.coords(var, 0, 0, 75, 25)
root.mainloop()
</code></pre>
<p>For learning more <strong>about Canvas</strong>, you may refer:</p>
<ul>
<li><p><a href="http://effbot.org/tkinterbook/canvas.htm" rel="nofollow noreferrer">Tkinter Canvas Widget</a></p></li>
<li><p><a href="https://docs.python.org/3/library/tk.html" rel="nofollow noreferrer">GUI with Tk(Python3.6.5)</a></p></li>
</ul>
</div>
<div class="post-text" itemprop="text">
<p><code>canvas.coords(line, 0, 0, 75, 25)</code> is the solution</p>
