<p>Your API returns a <code>list</code> of <code>dict</code>. If you want to get the <strong>3rd</strong> post, you'll have to get it with <code>my_post = data[2]</code>. </p>
<p><code>my_post</code> is a <code>dict</code> so you can get the body of your post: <code>my_post['body']</code>.</p>
