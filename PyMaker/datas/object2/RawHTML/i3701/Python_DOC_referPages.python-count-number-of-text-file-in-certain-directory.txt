<p>I am trying to get the number of the files in a certain directory, but I want it to count only text file because I have another directory in the accounts and DS. Store file. What should I modify to only get the number of text files?</p>
<pre><code>list = os.listdir("data/accounts/")
number_files = len(list)
print(number_files)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>From <a href="https://stackoverflow.com/questions/1320731/count-number-of-files-with-certain-extension-in-python?utm_medium=organic&amp;utm_source=google_rich_qa&amp;utm_campaign=google_rich_qa">Count number of files with certain extension in Python</a>,</p>
<p><strong>Solution 1.</strong></p>
<pre><code>fileCounter = 0
for root, dirs, files in os.walk("data/accounts/"):
    for file in files:    
        if file.endswith('.txt'):
            fileCounter += 1
</code></pre>
<p><strong>Solution 2.</strong></p>
<pre><code>fileCounter = len(glob.glob1("data/accounts/","*.txt"))
</code></pre>
<p>Read about glob here <a href="https://docs.python.org/2/library/glob.html" rel="nofollow noreferrer">here</a></p>
</div>
<div class="post-text" itemprop="text">
<p>Assuming your text files end in ".txt", you could use something like:</p>
<pre><code>files = [x for x in os.listdir("data/accounts/") if (os.isfile(x) and x.endswith('.txt'))]
number_files = len(list)
print(number_files)
</code></pre>
<p>Using <a href="https://docs.python.org/3.6/library/os.path.html#os.path.isfile" rel="nofollow noreferrer"><code>os.isfile()</code></a> to ignore directories and <code>string.endswith()</code> to determine text file-ness.</p>
</div>
<div class="post-text" itemprop="text">
<p>An alternative module that may work for you is <a href="https://docs.python.org/3/library/glob.html#module-glob" rel="nofollow noreferrer"><code>glob</code></a>.</p>
<p>It will allow you to use wildcards so you can capture just the files you are interested in.</p>
<pre><code>from glob import glob
filenames = glob("data/accounts/*.txt")
number_of_files = len(filenames)
print(number_of_files)
</code></pre>
</div>
