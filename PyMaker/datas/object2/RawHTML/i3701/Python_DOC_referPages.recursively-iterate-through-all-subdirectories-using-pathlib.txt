<div class="post-text" itemprop="text">
<p>How can I use <a href="https://docs.python.org/3/library/pathlib.html" rel="noreferrer">pathlib</a> to recursively iterate over all subdirectories of a given directory?</p>
<pre><code>p = Path('docs')
for child in p.iterdir(): child
</code></pre>
<p>only seems to iterate over the immediate children of a given directory.</p>
<p>I know this is possible with <code>os.walk()</code> or <code>glob</code>, but I want to use pathlib because I like working with the path objects.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>glob</code> method of a <code>Path</code> object:</p>
<pre><code>p = Path('docs')
for i in p.glob('**/*'):
     print(i.name)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>pathlib</code> has <code>glob</code> method where we can provide pattern as an argument. </p>
<p>For example : <code>Path('abc').glob('**/*.txt')</code> - It will look for current folder <code>abc</code> and all other subdirectories recursively to locate all <code>txt</code> files.</p>
</div>
<div class="post-text" itemprop="text">
<p>Alternatively, use <a href="https://docs.python.org/3/library/pathlib.html#pathlib.Path.rglob" rel="nofollow noreferrer"><code>Path.rglob</code></a> (substitutes the leading <code>**</code>):</p>
<pre><code>p = Path("docs")
for p in p.rglob("*"):
     print(p.name)
</code></pre>
</div>
