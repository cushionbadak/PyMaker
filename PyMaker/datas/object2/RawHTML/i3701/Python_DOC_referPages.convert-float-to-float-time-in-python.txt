<p>Try this:</p>
<p>time = 10.5</p>
<pre><code>print '{0:02.0f}:{1:02.0f}'.format(*divmod(float(time) * 60, 60))
</code></pre>
<blockquote>
<blockquote>
<blockquote>
<p>10:30</p>

<div class="post-text" itemprop="text">
<p>Split the number into its integer and decimal parts, then multiply the decimal part with 60 and put them back together:</p>
<pre><code>t = 10.5

hour, minute = divmod(t, 1)
minute *= 60
result = '{}:{}'.format(int(hour), int(minute))
# result: 10:30
</code></pre>
<p>See also the documentation for the <a href="https://docs.python.org/3/library/functions.html#divmod" rel="nofollow noreferrer"><code>divmod</code></a> function and <a href="https://stackoverflow.com/questions/6681743/splitting-a-number-into-the-integer-and-decimal-parts">this question</a> for other ways to split a float into two parts.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can pass the raw float value as <code>hours</code> to <code>datetime.timedelta()</code>, and then operate with it in what I think is the most comfortable way:</p>
<pre><code>from datetime import datetime, timedelta

td = timedelta(hours=10.5)
dt = datetime.min + td
print("{:%H:%M}".format(dt))
print(td.total_seconds())
</code></pre>
