<div class="post-text" itemprop="text">
<p>Becareful evaling a data file. Do not do this if you don't have complete control and confidence in the text file:</p>
<pre><code>with open('datafile.txt') as f:
    for line in f:
        d = eval(line)
        print d
</code></pre>
<p>outputs:</p>
<pre><code>{'name': 'sk', 'uname': 'sk', 'passwd': 'sk', 'confirm': 'sk', 'phone': 'sk', 'email': 'sk'}
{'name': 'skak', 'confirm': 'ak', 'passwd': 'ak', 'phone': 'ak', 'uname': 'ak', 'email': 'ak'}
</code></pre>
<p>It would be much safer to replace the single quotes to double and read as json.</p>
<pre><code>import json
d = json.loads(line.replace("'", '"'))
</code></pre>
