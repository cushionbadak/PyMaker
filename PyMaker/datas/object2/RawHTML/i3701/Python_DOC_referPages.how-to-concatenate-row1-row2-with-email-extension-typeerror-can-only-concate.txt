<p>Use <strong><code>zip</code></strong> and <strong><code>join</code></strong>:</p>
<pre><code>First_Name =['Name1', 'Name2']
Last_Name = ['Last1', 'Last2']

print('\n'.join(f'{x}.{y}@gmail.com' for x, y in zip(First_Name, Last_Name)))

# Name1.Last1@gmail.com
# Name2.Last2@gmail.com
</code></pre>
<p>Problem with your code is you are trying to concatenate <em>str</em> to a <em>list</em>. You could only concatenate <em>list</em> to a <em>list</em>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>zip()</code> to merge your two lists:</p>
<pre><code>first_name_list =['Name1', 'Name2']
last_name_list = ['Last1', 'Last2']

final_email_list = []
for first_name, last_name in zip(first_name_list, last_name_list):
    email = first_name + "." + last_name + "@gmail.com"
    final_email_list.append(email)

print(final_email_list[0]) # outputs: Name1.Last1@gmail.com
</code></pre>

<p>Try using <code>zip</code> and then use the string <code>join</code> function:</p>
<pre><code>First_Name =['Name1', 'Name2']
Last_Name = ['Last1', 'Last2']
print(' &amp; '.join([x+'.'+y+'@gmail.com' for x,y in list(zip(First_Name,Last_Name))]))
</code></pre>
<p>Output:</p>
<pre><code>Name1.Last1@gmail.com &amp; Name2.Last2@gmail.com
</code></pre>
<p>Or if you want a list, Try this:</p>
<pre><code>First_Name =['Name1', 'Name2']
Last_Name = ['Last1', 'Last2']
print([x+'.'+y+'@gmail.com' for x,y in list(zip(First_Name,Last_Name))])
</code></pre>
<p>Output:</p>
<pre><code>['Name1.Last1@gmail.com', 'Name2.Last2@gmail.com']
</code></pre>
<p>You can do a lot of stuff with lists, here is the documentation of it: <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer">link</a></p>
