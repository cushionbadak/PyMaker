<div class="post-text" itemprop="text">
<p>From your traceback: <code>TypeError: Required argument 'tz' (pos 1) not found</code>. Your script is using Python2, which is generally not compatible with Python3. In Python2 you have to provide a timezone. See the difference between the <a href="https://docs.python.org/2/library/datetime.html" rel="nofollow noreferrer">Python2</a> and <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer">Python3</a> documentation.</p>
<p>In Linux, the <code>python</code> word specifically means <code>python2</code>. Python3 is typed using <code>python3</code> unless you use an alias. To fix it, change all occurrences of <code>python</code> to <code>python3</code>, as in</p>
<pre><code>#! /usr/bin/env python3
python3 offset.py
</code></pre>
<p><strong>Note:</strong> 
From the Python3 Documentation, it does not work on all versions of Python3:</p>
<blockquote>
<p>Changed in version 3.3: tz now can be omitted.</p>
<p>Changed in version 3.6: The astimezone() method can now be called on naive instances that are presumed to represent system local time.`</p>
</blockquote>
