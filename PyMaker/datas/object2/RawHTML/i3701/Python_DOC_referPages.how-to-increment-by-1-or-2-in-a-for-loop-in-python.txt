<p>Why not simply test it using a list-slice?</p>
<pre><code>a = [16,1,2,3,4,5,6,7,8,9]
print ( a[0] &gt; max ( a[2::2] ) ) # prints True

a = [16,1,2,3,4,5,6,7,8,9,22]
print ( a[0] &gt; max ( a[2::2] ) ) # prints False
</code></pre>
<p><code>a[2::2]</code> starts with the 2nd element of <code>a</code> and takes every 2nd till its end. <code>max(..)</code> calculates the biggest one in it and <code>a[0] &gt; max(..)</code> checks if the fist element of a is bigger then all other even values in that list.</p>

<p>This will work for any length of <code>card_task_e</code>.</p>
<p>First, we loop through the odd integers, from 1 to the length. Inside that we loop between the even integers starting at the current odd index. The trailing 2 in each for loop tells the loop to increment by 2 in each iteration of the loop.</p>
<pre><code>list_size = len(card_task_e)
for odd_index in range(1, list_size, 2):
    for even_index in range(odd_index + 1, list_size, 2):
        ol_task_e.append(card_task_e[odd_index])
        counte += 1
</code></pre>

<div class="post-text" itemprop="text">
<p>You can reference a specific list element by using a counter that increments by 2 in each pass of the <code>for</code> loop.</p>
<pre><code>counter = 0
for ____ in ____:
    counter += 2
    &lt;do something to&gt; some_list[counter]
</code></pre>
<p>You can also add <code>if</code> statements inside the loop to dictate how the counter is incremented.</p>
