<p>SVG files are vectors and can be read as XML. for example, <a href="https://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow noreferrer">xml.etree.ElementTree</a> in the Python standard library can parse XML files.   </p>
<p>We have something line this : </p>
<pre><code>&lt;svg width="240" height="240" xmlns="http://www.w3.org/2000/svg"&gt;
</code></pre>
<p>If your file has width and height attributes, you can use them. Without width and height i don't think there is a way to get exactly size of SVG file because they can scale infinitely ( <strong>any resolution</strong> )</p>
<p>Width and height</p>
<pre><code>&lt;svg width="240" height="240" 
xmlns="http://www.w3.org/2000/svg"&gt;
</code></pre>
<p>Doubles the original scale.</p>
<pre><code>&lt;svg viewBox="0 0 120 120" width="240" height="240" 
xmlns="http://www.w3.org/2000/svg"&gt;
</code></pre>
<p>Infinite scaling</p>
<pre><code>&lt;svg viewBox="0 0 120 120" 
xmlns="http://www.w3.org/2000/svg"&gt;
</code></pre>
