<div class="post-text" itemprop="text">
<p>I have to write a function which return <code>True</code> even if one <strong>word</strong> between the two strings matches.</p>
<p><code>willis towers watson</code> and <code>willis singapore</code> -&gt; <code>True</code> as <code>willis</code> matches</p>
<p><code>willis towers watson</code> and <code>willis s pte ltd</code> -&gt; <code>True</code> as <code>willis</code> matches</p>
<p><code>willis towers watson</code> and <code>will tow wat</code> -&gt; <code>False</code> as no <code>Word</code> matches</p>
<p>I unsuccessfully tried a regex:</p>
<pre><code>bool(re.search('willis towers watson', 'willis singapore'))
</code></pre>
<p>One liner is preferable instead of a for loop.</p>
</div>
<div class="post-text" itemprop="text">
<blockquote>
<p>Just split the strings into sets of words and check if the two sets
  intersect.</p>
</blockquote>
<pre><code>&gt;&gt;&gt; a = set('willis towers watson'.split())
&gt;&gt;&gt; b = 'willis singapore'
&gt;&gt;&gt; any(v in a for v in b.split()) 
True
</code></pre>
<p>Alternatively, </p>
<pre><code>&gt;&gt;&gt; not a.isdisjoint(b.split())
True
</code></pre>
</div>
