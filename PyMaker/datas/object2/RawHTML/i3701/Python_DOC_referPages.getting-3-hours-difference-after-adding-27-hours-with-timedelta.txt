<div class="post-text" itemprop="text">
<p>Checking just the <code>seconds</code> attribute of a <code>timedelta</code> object is insufficient.</p>
<p>As per the <a href="https://docs.python.org/3/library/datetime.html#timedelta-objects" rel="nofollow noreferrer">timedelta documentation</a>:</p>
<blockquote>
<p>Only <em>days</em>, <em>seconds</em> and <em>microseconds</em> are stored internally.</p>
</blockquote>
<p>Consider these components in turn:</p>
<pre><code>days = baz.days
seconds = baz.seconds
microsecs = baz.microseconds

print(days, seconds, microsecs, sep=', ')
1, 10800, 0
</code></pre>
<p>You will need to add these together separately to get your total <code>timedelta</code>. If your aim is to extract the total number of seconds, you can use the <code>total_seconds</code> method:</p>
<pre><code>totalsecs = baz.total_seconds()  # 97200.0
assert totalsecs / 60 / 60 == 27  # check we have 27 hours
</code></pre>
