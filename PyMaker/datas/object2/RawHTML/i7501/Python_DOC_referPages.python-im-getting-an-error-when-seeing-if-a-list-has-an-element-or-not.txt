<div class="post-text" itemprop="text">
<p>The list index starts from zero.</p>
<p>So, if the length of the list is exactly one, than <code>clearargs[0]</code> is the right access to the first value in the list.</p>
<p>When asking to get the value from <code>clearargs[1]</code>, you assume that the length of the list is at least 2.</p>
</div>
<div class="post-text" itemprop="text">
<p>else block will be executed only when the<code>len(clearargs &lt; 1)</code> so the only valid index  is <code>clearargs[0]</code>.You are making a logical error.</p>
