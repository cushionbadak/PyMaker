<div class="post-text" itemprop="text">
<p>I would like to check the first few characters (the numbers are variable) in a string. 
E.g.</p>
<pre><code>a = '+6221-123-4567'
</code></pre>
<p>and I would like to check if the first few characters are in</p>
<pre><code>b = ['021', '+6221', '(021)', '(+62)']
</code></pre>
<p>I would like to do it programatically, without separating manually based on the number of characters:</p>
<pre><code>if a[:3] in ['021']: print('yes')
if a[:5] in ['+6221', '(021)', '(+62)']: print('yes')
</code></pre>
<p>Thank you!</p>
</div>
<div class="post-text" itemprop="text">
<blockquote>
<p>str.startswith(prefix[, start[, end]])</p>
<p>Return True if string starts with the prefix, otherwise return False.
  prefix can also be a tuple of prefixes to look for.</p>
</blockquote>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.startswith" rel="noreferrer">docs</a></p>
<p>Try this,
<code>a.startswith(tuple(b))</code>.</p>
<p>Full code,</p>
<pre><code>if a.startswith(tuple(b)):
    print("yes")
</code></pre>
</div>
