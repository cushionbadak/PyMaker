<p>There is a solution available for doing bitwise operations. But, it converts the integers in binary number and then do bitwise operations. I already have the numbers in binary format. How do I carry out the bitwise operations for these numbers?
Ex. x = 10101, y = 11001
I want z = x or y to be 11101.
Any help is highly appreciated.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can convert decimal <code>10101</code> to binary <code>10101</code> by converting it to string then <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow noreferrer"><code>int(.., 2)</code></a> (2 =&gt; base 2):</p>
<pre><code>&gt;&gt;&gt; str(x)
'10101'
&gt;&gt;&gt; int(str(x), 2)
21
</code></pre>
<p>Do operation you want (<code>|</code> =&gt; bitwise or):</p>
<pre><code>&gt;&gt;&gt; int(str(x), 2) | int(str(y), 2)
29
</code></pre>
<p>Then, convert it back to decimal, using <a href="https://docs.python.org/3/library/functions.html#format" rel="nofollow noreferrer"><code>format</code></a> (or <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>str.format</code></a>  with format string <code>b</code> -&gt; meaning binary representation) and <code>int</code>:</p>
<pre><code>&gt;&gt;&gt; format(int(str(x), 2) | int(str(y), 2), 'b')
'11101'
&gt;&gt;&gt; int(format(int(str(x), 2) | int(str(y), 2), 'b'))  # back to integer
11101
</code></pre>
