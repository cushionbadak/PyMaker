<div class="post-text" itemprop="text">
<p>Starting from Python 3.2, you can use <a href="https://docs.python.org/3/library/stdtypes.html#int.from_bytes" rel="nofollow noreferrer"><code>int.from_bytes</code></a>.</p>
<p>Second argument, <code>byteorder</code>, specifies <a href="https://en.wikipedia.org/wiki/Endianness" rel="nofollow noreferrer">endianness</a> of your bytestring. It can be either <code>'big'</code> or <code>'little'</code>. You can also use <code>sys.byteorder</code> to get your host machine's native byteorder.</p>
<pre class="lang-py prettyprint-override"><code>import sys
int.from_bytes(b'\x11', byteorder=sys.byteorder)  # =&gt; 17
bin(int.from_bytes(b'\x11', byteorder=sys.byteorder))  # =&gt; '0b10001'
</code></pre>
