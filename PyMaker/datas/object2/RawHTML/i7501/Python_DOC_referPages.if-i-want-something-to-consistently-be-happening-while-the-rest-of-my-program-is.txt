<p>If i want to make an interest class, where I use a time function to continuously add interest while the rest of the program runs, how would I carry that out?</p>

<div class="post-text" itemprop="text">
<p>Use threading:    </p>
<pre><code>import threading

def interest():
    while True:
        print('interest is running')

d = threading.Thread(target=interest)
d.setDaemon(True)
d.start()

print('Your main job is here')
</code></pre>
