<div class="post-text" itemprop="text">
<p>I am trying to generate the next code in Python:</p>
<pre><code>atribute_name = 'atr1'
exec("list_%s = []", atribute_name)
</code></pre>
<p>So the expected result should be: </p>
<pre><code>list_atr1 = []
</code></pre>
<p>But when I execute it I get the next error message:</p>
<pre><code>TypeError: exec: arg 2 must be a dictionary or None
</code></pre>
<p>Why is it happening?</p>
</div>
<div class="post-text" itemprop="text">
<p>Instead of comma, you should replace it with <code>%</code> for string concatenation.</p>
<pre><code>exec("list_%s = []" % atribute_name)
</code></pre>
</div>
<div class="post-text" itemprop="text">
