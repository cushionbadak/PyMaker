<p>You should switch to using <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow noreferrer">pathlib</a> when you're dealing with Windows paths. If you're on Python 2, it's available as a <a href="https://pypi.python.org/pypi/pathlib/" rel="nofollow noreferrer">package</a> too.</p>
<p>To give you an example how to use the pathlib library:</p>
<pre><code>def Test2(rootDir):
    f = []
    rootDir = pathlib.Path(rootDir)
    for iter_file in rootDir.iterdir(): 
        if iter_file.is_dir(): 
            Test2(iter_file)
        if iter_file.is_file(): 
            print(iter_file)
</code></pre>
