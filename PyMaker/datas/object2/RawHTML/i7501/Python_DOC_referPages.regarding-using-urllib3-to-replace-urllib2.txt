<p>In python 3 <code>urlopen</code> is part of <code>urllib.request</code>  so you have to modify your imports:  </p>
<pre><code>from urllib.request import urlopen  
</code></pre>
<p>If you want your script to run in python 2 and python 3 you can use:</p>
<pre><code>try:  
    from urllib2 import urlopen
except ImportError:  
    from urllib.request import urlopen
</code></pre>
