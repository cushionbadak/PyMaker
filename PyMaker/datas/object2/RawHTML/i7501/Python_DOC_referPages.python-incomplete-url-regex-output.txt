<p>I have created a "lite" URL regex. That means it may not detect all URLs. I created it with the aim of covering simple urls. </p>
<pre><code>#! python3
# urls.py - Detecting urls that begin with http:// or https://

import re

urlRegex = re.compile(r'''(
    (http://|https://)+          # the http(s) part of the url
    (w{3}\.)?              # the world-wide-web part
    ([a-z0-9-])+            # the domain name
    (\.[a-z]{2,4})?        # sub level domain
    (\.[a-z]{2,4})        # top level domain
    (/[-A-Za-z0-9+&amp;@#/%=~_|])* # extension i.e paths
)''', re.VERBOSE)
test = urlRegex.search('https://www.facebook.com/user_2033')
</code></pre>
<p>The output of <code>test.groups()</code> was this</p>
<pre><code>('https://www.facebook.com/user_2033', 'https://', 'www.', 'k', None, '.com', '/u')
[Finished in 0.058s]
</code></pre>
<p>After numerous attempts, I'm unable to display the complete website name and extension i.e <code>'facebook' not 'k'</code>. Any help without completely changing my own code would be most appreciated</p>

<p><code>(PATTERN)*</code> or <code>(PATTERN)+</code> will capture the last matched character only. It should be <code>(PATTERN*)</code> or <code>(PATTERN+)</code> to capture all characters.</p>
<hr/>
<pre><code>([a-z0-9-])+          # the domain name
</code></pre>
<p>should be replaced with:</p>
<pre><code>([a-z0-9-]+)          # the domain name
</code></pre>
<p>Same for the last part:</p>
<pre><code>(/[-A-Za-z0-9+&amp;@#/%=~_|])* # extension i.e paths

(/[-A-Za-z0-9+&amp;@#/%=~_|]*) # extension i.e paths
</code></pre>
<hr/>
<p>output:</p>
<pre><code>('https://www.facebook.com/user_2033', 'https://', 's', 'www.',
 'facebook', None, '.com', '/user_2033')
</code></pre>
<hr/>
<p>BTW, you can use <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.urlparse" rel="nofollow noreferrer"><code>urllib.parse.urlparse</code></a> (Python 3) / <a href="https://docs.python.org/2/library/urlparse.html#urlparse.urlparse" rel="nofollow noreferrer"><code>urlparse.urlparse</code></a> (Python 2) instead of regular expression:</p>
<pre><code>&gt;&gt;&gt; import urllib.parse
&gt;&gt;&gt; urllib.parse.urlparse('https://www.facebook.com/user_2033')
ParseResult(scheme='https', netloc='www.facebook.com',
            path='/user_2033', params='', query='', fragment='')
</code></pre>
