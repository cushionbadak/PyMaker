<div class="post-text" itemprop="text">
<p>Simply create a bat file with ping 8.8.8.8 written in it.
Then you can use the subprocess command to invoke it from within the python script.</p>
<pre><code>import subprocess
subprocess.Popen('ping.bat')
</code></pre>
</div>
