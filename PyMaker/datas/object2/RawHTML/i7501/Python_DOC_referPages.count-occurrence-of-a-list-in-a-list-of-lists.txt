<p>Python </p>
<p>I have a list of lists. like </p>
<pre><code>A = [[x,y],[a,b],[c,f],[e,f],[a,b],[x,y]]
</code></pre>
<p>I want to count how many times each list occurred in the main list. </p>
<p>My output should be like</p>
<pre><code>[x,y] = 2
[a,b] = 2
[c,f] = 1 
[e,f] = 1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just use <code>Counter</code> from <code>collections</code>:</p>
<pre><code>from collections import Counter
A = [[x,y],[a,b],[c,f],[e,f],[a,b],[x,y]]

new_A = map(tuple, A) #must convert to tuple because list is an unhaable type

final_count = Counter(new_A)


#final output:

for i in set(A):
   print i, "=", final_count(tuple(i))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>collections.Counter</code></a> - a dict subclass - to take the counts. First, convert the sublists to tuples to make them usable (i.e. hashable) as dictionary keys, then count:</p>
<pre><code>from collections import Counter

count = Counter(map(tuple, A))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Depending on your output, you can <code>loop</code> through a <code>set</code> of your <code>list</code> and print each item with its <code>count()</code> from the original <code>list</code>:</p>
<pre><code>for x in set(map(tuple, A)):
    print '{} = {}'.format(x, A.count(list(x)))
</code></pre>
</div>
