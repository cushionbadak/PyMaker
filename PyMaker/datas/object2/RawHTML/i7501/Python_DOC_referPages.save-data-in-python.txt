<p>Simple way to achieve what you want.</p>
<pre><code>import csv

with open('filename', 'w') as myfile:
    wr = csv.writer(myfile, quoting=csv.QUOTE_ALL)
    for tweet in tweepy.Cursor(api.search, q=search+ lang="en").items(num_tweets):
        data = tweet.text
        wr.writerow(tweet.text)
</code></pre>
<p>You can check what <a href="https://docs.python.org/3/library/csv.html#csv.QUOTE_ALL" rel="nofollow noreferrer">csv.QUOTE_ALL</a> does.</p>
