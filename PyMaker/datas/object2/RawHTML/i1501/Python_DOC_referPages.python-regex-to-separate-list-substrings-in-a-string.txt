<div class="post-text" itemprop="text">
<p>I have a string like this <code>'[[20, 20.4,aa], [c, 10.5, eee]]'</code>.
My aim is to enclose the characters inside single quotes and leave the numbers as it is.</p>
<p>For example:</p>
<pre><code>examples:
s1 = "[[20, 20.4, aa], [c, 10.5, eee]]"
s2 = "[[a , bg, 20], [cff, 20, edd]]"

required:
s1 = "[[20, 20.4,'aa'], ['c', 10.5, 'eee']]"
s2 = "[[a , 'bg', 20], ['cff', 20, 'edd']]"
</code></pre>
<p>I have done so far this:</p>
<pre><code>s = '[[20, 20.4,a], [c, 10.5, e]]'

s = ''.join(["'"+ i + "'" if i.isalpha() else i for i in s])
s # "[[20, 20.4,'a'], ['c', 10.5, 'e']]"
</code></pre>
<p>But it works only for single characters. If I have <code>aa</code> it will give <code>'a''a'</code> which is wrong.
How the problem can be fixed? </p>
</div>
<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/re.html#re.sub" rel="nofollow noreferrer">sub</a>:</p>
<pre><code>import re

s1 = '[[20, 20.4,aa], [c, 10.5, eee]]'
s2 = '[[a , bg,20], [cff, 20, edd]]'

rs1 = re.sub('([a-zA-Z]+)', r"'\1'", s1)
print(rs1)
rs2 = re.sub('([a-zA-Z]+)', r"'\1'", s2)
print(rs2)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[[20, 20.4,'aa'], ['c', 10.5, 'eee']]
[['a' , 'bg',20], ['cff', 20, 'edd']]
</code></pre>
<p>The pattern <code>([a-zA-Z]+)</code> means match one or more letters and put them inside a group, then refer to that group <code>r"'\1'"</code> and surrounded with single quotes.</p>
