<p>In the following representation of date time, what does <code>-0600</code> means? And how it can be used to compare dates?</p>
<pre><code>Tue, 27 Nov 2018 14:59:32 -0600
</code></pre>
<p>Tagging it with Python as looking for date comparison.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>-0600</code> is the timezone offset.  Use the <code>%z</code> directive of <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer"><code>datetime.datetime.strptime()</code></a> to use it when making a new datetime object.</p>
<pre><code>import datetime
s1 = 'Tue, 27 Nov 2018 14:59:32 -0600'
s2 = 'Tue, 27 Nov 2018 15:59:32 -0500'
s3 = 'Tue, 27 Nov 2018 14:59:32 -0500'

fmt = '%a, %d %b %Y %H:%M:%S %z'

dt1 = datetime.datetime.strptime(s1, fmt)        
dt2 = datetime.datetime.strptime(s2, fmt)        
dt3 = datetime.datetime.strptime(s3, fmt)    
</code></pre>
<hr/>
<pre><code>&gt;&gt;&gt; dt1
datetime.datetime(2018, 11, 27, 14, 59, 32, tzinfo=datetime.timezone(datetime.timedelta(-1, 64800)))
&gt;&gt;&gt; dt2
datetime.datetime(2018, 11, 27, 15, 59, 32, tzinfo=datetime.timezone(datetime.timedelta(-1, 68400)))
&gt;&gt;&gt; dt3
datetime.datetime(2018, 11, 27, 14, 59, 32, tzinfo=datetime.timezone(datetime.timedelta(-1, 68400)))
&gt;&gt;&gt; dt1 == dt2
True
&gt;&gt;&gt; dt1 == dt3
False
&gt;&gt;&gt; dt1 &gt; dt3
True
&gt;&gt;&gt;
</code></pre>
