<div class="post-text" itemprop="text">
<p>I prefer a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> with a self-documenting <a href="https://docs.python.org/3.3/tutorial/datastructures.html#tuples-and-sequences" rel="nofollow noreferrer">tuple unpacking</a>:</p>
<pre><code>[x for x,y in training_data]
</code></pre>
<p>Complete program:</p>
<pre><code>training_data=[(1.5, 11), (2.5, 22), (7.5, 77)]
training_data_x = [x for x,y in training_data]
print(training_data_x)
</code></pre>
</div>
