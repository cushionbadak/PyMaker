<p>did you mean this:</p>
<pre><code>from itertools import product

for basis_nums in product(range(0,10), range(0, 10), range(0, 10)):
    print(basis_nums)
</code></pre>
<p><a href="https://docs.python.org/3/library/itertools.html?highlight=itertools#itertools.product" rel="nofollow noreferrer"><code>itertools.product</code></a> saves you the indentation depth. (note: <code>basis_nums</code> are <code>tuples</code> now and not <code>lists</code>).</p>
</div>
<div class="post-text" itemprop="text">
<p>Or have an extra <code>repeat=3</code> at the end:</p>
<pre><code>for basis_nums in itertools.product(range(10),repeat=3):
    print(basis_nums)
</code></pre>
<p><a href="https://docs.python.org/3/library/itertools.html?highlight=itertools#itertools.product" rel="nofollow noreferrer"><code>itertools.product</code></a> is getting the mathematical product algorithm embedded into python, the <a href="https://docs.python.org/3/library/itertools.html?highlight=itertools#module-itertools" rel="nofollow noreferrer"><code>itertools</code></a> module also contains the combination algorithm, as <a href="https://docs.python.org/2/library/itertools.html#itertools.combinations" rel="nofollow noreferrer"><code>itertools.combination</code></a>, also the module contains the permutation algorithm, as <a href="https://docs.python.org/2/library/itertools.html#itertools.permutations" rel="nofollow noreferrer"><code>itertools.permutation</code></a>.</p>
</div>
