<div class="post-text" itemprop="text">
<p>The easiest way to pull that data in would be to download and parse the CSV file listed under the "Download" button. </p>
<p>You can use the <a href="http://docs.python-requests.org/en/master/" rel="nofollow noreferrer">Requests</a> library to download the file, then use the native <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer">CSV</a> library.</p>
</div>
<div class="post-text" itemprop="text">
<p>See <a href="https://stackoverflow.com/a/32400969/9214517">https://stackoverflow.com/a/32400969/9214517</a> for how to do it.</p>
<p>Let's say you allow to keep the data in a pandas DataFrame (as the link above do), this is the code:</p>
<pre><code>import pandas as pd
import requests
import io
url = "https://fred.stlouisfed.org/graph/fredgraph.csv?bgcolor=%23e1e9f0&amp;chart_type=line&amp;drp=0&amp;fo=open%20sans&amp;graph_bgcolor=%23ffffff&amp;height=450&amp;mode=fred&amp;recession_bars=on&amp;txtcolor=%23444444&amp;ts=12&amp;tts=12&amp;width=968&amp;nt=0&amp;thu=0&amp;trc=0&amp;show_legend=yes&amp;show_axis_titles=yes&amp;show_tooltip=yes&amp;id=FEDFUNDS&amp;scale=left&amp;cosd=1954-07-01&amp;coed=2018-10-01&amp;line_color=%234572a7&amp;link_values=false&amp;line_style=solid&amp;mark_type=none&amp;mw=3&amp;lw=2&amp;ost=-99999&amp;oet=99999&amp;mma=0&amp;fml=a&amp;fq=Monthly&amp;fam=avg&amp;fgst=lin&amp;fgsnd=2009-06-01&amp;line_index=1&amp;transformation=lin&amp;vintage_date=2018-11-28&amp;revision_date=2018-11-28&amp;nd=1954-07-01"
s = requests.get(url).content.decode("utf-8")
df = pd.read_csv(io.StringIO(s)
</code></pre>
<p>Then your <code>df</code> will be:</p>
<pre><code>           DATE  FEDFUNDS
0    1954-07-01      0.80
1    1954-08-01      1.22
2    1954-09-01      1.06
3    1954-10-01      0.85
4    1954-11-01      0.83
....
</code></pre>
<p>And if you insist on a <code>dict</code>, use this instead of the last line above to convert your CSV data <code>s</code>:</p>
<pre><code>mydict = dict([line.split(",") for line in s.splitlines()])
</code></pre>
<p>The key is how to get the URL: Hit the download button on the page you quoted, and copy the link to CSV.</p>
