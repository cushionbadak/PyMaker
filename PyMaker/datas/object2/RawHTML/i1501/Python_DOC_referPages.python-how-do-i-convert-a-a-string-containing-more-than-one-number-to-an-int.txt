<div class="post-text" itemprop="text">
<p>How can I convert a string containing more than one number to an int in python, for example: <code>"100, 200, 300"</code> and <code>"400, 500"</code>. I know how I can convert a string containing a single number such as <code>"100"</code> or <code>"56"</code> to an int but not a string which contains 2 numbers.</p>
<p>Does anyone know how this could be done?</p>
<p>Thank you so much for your help.</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer">map()</a> for mapping list element to <code>int</code></p>
<pre><code>x = "100, 200, 300"
list(map(int, x.split(",")))
</code></pre>
<p>Output:</p>
<pre><code>[100, 200, 300]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can also use <a href="https://www.python-course.eu/python3_list_comprehension.php" rel="nofollow noreferrer">list comprehensions</a>:</p>
<pre><code>In [635]: x = "100, 200, 300"
In [638]: results = [int(i) for i in x.split(',')]

In [639]: results
Out[639]: [100, 200, 300]
</code></pre>
</div>
<span class="comment-copy">But you need to be sure that substrings are integers, otherwise you'll get ValueError.</span>
