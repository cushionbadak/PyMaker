<div class="post-text" itemprop="text">
<p>You can also use <a href="https://docs.python.org/3/library/string.html#string.ascii_uppercase" rel="nofollow noreferrer"><code>string.ascii_uppercase</code></a> to get the letters and use a slice of length <code>n</code> for your coordinates:</p>
<pre><code>import string

def mapCells(n):
    cell_List = []
    for x in string.ascii_uppercase[:n]:
        for y in range(1, min(n + 1, 27)):
            cell = x + str(y)
            cell_List.append(cell)
    print(cell_List)

mapCells(3)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>['A1', 'A2', 'A3', 'B1', 'B2', 'B3', 'C1', 'C2', 'C3']
</code></pre>
