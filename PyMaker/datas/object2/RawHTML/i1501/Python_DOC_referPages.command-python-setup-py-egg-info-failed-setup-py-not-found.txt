<p>I found a solution to my case, appear to be

<a href="https://pypi.org/project/vcversioner/" rel="nofollow noreferrer">https://pypi.org/project/vcversioner/</a> 
without this package i could'nt download new packages to pycharm
i used this commend line to download vcversioner -&gt; pip install vcversioner 
and it solved the problem </p>
