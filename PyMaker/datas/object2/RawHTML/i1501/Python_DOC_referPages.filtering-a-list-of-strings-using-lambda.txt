<p>I have a list like,</p>
<pre><code>old_list=["a_apple","b_banana","c_cherry"]
</code></pre>
<p>and I want to get new list using <strong>lambda</strong>.</p>
<pre><code>new_list=["apple","banana","cherry"]
</code></pre>
<p>I tried but it doesn't work as I expected. here is what I wrote.</p>
<pre><code>new_list=filter(lambda x: x.split("_")[1], old_list)
</code></pre>
<p>what is the problem ?</p>

<div class="post-text" itemprop="text">
<p>Try this:</p>
<p><strong>Python-2</strong>:</p>
<pre><code>In [978]: old_list=["a_apple","b_banana","c_cherry"]

In [980]: new_list = map(lambda x: x.split("_")[1], old_list)

In [981]: new_list
Out[981]: ['apple', 'banana', 'cherry']
</code></pre>
<p><strong>Python-3</strong>:</p>
<pre><code>In [4]: new_list = list(map(lambda x: x.split("_")[1], old_list))

In [5]: new_list
Out[5]: ['apple', 'banana', 'cherry']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can <code>map</code> the list with a <code>lambda</code> function:</p>
<pre><code>list(map(lambda s: s.split('_')[1], old_list))
</code></pre>
<p>This returns:</p>
<pre><code>['apple', 'banana', 'cherry']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehensions</a></p>
<pre><code>lst=["a_apple","b_banana","c_cherry"]
[i.split("_")[1] for i in lst]
</code></pre>
<p>Output:</p>
<pre><code>['apple', 'banana', 'cherry']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>By using lambda and map:</p>
<pre><code>li=["a_apple","b_banana","c_cherry"]

new_li = map(lambda x: x.split('_')[1], li)
print (list(new_li))
# ['apple', 'banana', 'cherry']
</code></pre>
</div>
