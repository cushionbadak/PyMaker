<p>I believe this is a problem with encoding </p>
<p>try this :</p>
<pre><code>import pandas as pd
df = pd.read_csv("report.csv",encoding='cp1252')
df = df.drop("Agent Name", axis=1)
df.to_csv("agent_report_updated.csv")
</code></pre>
<p>There are other encoding options you can try utf-8 instead of cp1252.
<a href="https://docs.python.org/3/library/codecs.html#standard-encodings" rel="nofollow noreferrer">Here</a> is a list of encodings used.</p>
