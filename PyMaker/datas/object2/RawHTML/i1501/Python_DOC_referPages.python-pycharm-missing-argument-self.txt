<div class="post-text" itemprop="text">
<p>You have the error in <a href="https://www.tutorialspoint.com/python/python_classes_objects.htm" rel="nofollow noreferrer">Creating Instance Objects</a> </p>
<p>In this section <code>fish1 = Fish</code> you didn't create the instance, but try to use it <code>fish1.bubbles()</code>.</p>
<p>So try change <code>fish1 = Fish</code> to <code>fish1 = Fish(name, coords)</code>. </p>
<p><code>name</code> and <code>coords</code> required in your constructor.</p>
</div>
<div class="post-text" itemprop="text">
<p>The <code>Fish</code> is the class. You can't ask for an abstract fish to make bubbles. You have to ask a specific fish (that is, an <em>object</em> of that class). So, assuming your fish is "bob"</p>
<pre><code>bob = Fish()
bob.bubbles()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to <strong>call the constructor</strong> of the <a href="https://docs.python.org/3/tutorial/classes.html" rel="nofollow noreferrer">class</a> <strong>to create an instance</strong> of that <a href="https://docs.python.org/3/tutorial/classes.html" rel="nofollow noreferrer">class</a></p>
<pre><code>fish1 = Fish()
</code></pre>
<p>In your case it would be with parameters</p>
<pre><code>fish = Fish("fish_name", "coordinates")
</code></pre>
</div>
