<div class="post-text" itemprop="text">
<p>To get the values of a dict from a key, you can either do <code>main['dtt']</code> (see <a href="https://docs.python.org/3/library/stdtypes.html#mapping-types-dict" rel="nofollow noreferrer">doc</a>) or <code>main.get('dtt')</code> (see <a href="https://docs.python.org/3/library/stdtypes.html#dict.get" rel="nofollow noreferrer">this part</a> of the doc) but not <code>main('dtt')</code> as like the error says, a dict is not callable.</p>
<p>Option 1:</p>
<blockquote>
<p><code>d[key]</code>: Return the item of d with key key. Raises a <code>KeyError</code> if key is not in the map.</p>
</blockquote>
<p>Option 2:</p>
<blockquote>
<p><code>d.get(key[, default])</code>: Return the value for key if key is in the dictionary, else <code>default</code>. If <code>default</code> is not given, it defaults to <code>None</code>, so that this method never raises a <code>KeyError</code>.</p>
</blockquote>
</div>
