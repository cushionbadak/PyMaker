<p>Yes, you might have overcomplicated the solution as there are out of the box Python methods that help you easily solve problems like this. For iteration with indexes, use <code>enumerate</code>, in the example below we set the index to start at 1. We can also use some simple string formatting defined in <code>fmt</code> to ensure consistent spacings.</p>
<pre><code>li = ['Lets go outside','pizza time','show me the money']

print('line   number of words')
print('----   ---------------')
fmt = ('{}     {}')

for idx, sentence in enumerate(li,1):
    no_of_words = len(sentence.split())
    print(fmt.format(idx, no_of_words))
</code></pre>
<p>Then simple use <code>split</code> to split the whitespaces and get the total number of words and let <code>enumerate</code> manage the whole thing for you.</p>
<pre><code>&gt;&gt;
line   number of words
----   ---------------
1     3
2     2
3     4
</code></pre>

<pre><code>list1 = ['Lets go outside','pizza time','show me the money']
print('line   number of words')
print('----   ---------------')

for i in range(0, len(list1)):
    length = len(list1[i].split(" "))
    print(i + 1, "        ", length)
</code></pre>
