<div class="post-text" itemprop="text">
<p>Error is caused by statement in <code>randomFive</code> method</p>
<pre><code>self[random.randint(0,len(self)-1)]
</code></pre>
<p>because <code>self</code> is an instance of <code>Stack</code> which has no <a href="https://docs.python.org/3/reference/datamodel.html#object.__len__" rel="nofollow noreferrer"><code>__len__</code> method</a> available and for user-defined classes <a href="https://docs.python.org/library/functions.html#len" rel="nofollow noreferrer"><code>len</code> built-in function</a> tries to call it and raises an error if there is no any (more info at <a href="https://stackoverflow.com/questions/2481421/difference-between-len-and-len">this thread</a>).</p>
<p>Fast fix would be to change it to</p>
<pre><code>self[random.randint(0,len(self.stack)-1)]
</code></pre>
<p>but in longer perspective it will be better to define your own <code>__len__</code> method like</p>
<pre><code>class Stack:
    ...
    def __len__(self):
        return len(self.stack)
</code></pre>
<p>after that your original <code>randomFive</code> method should work.</p>
</div>
