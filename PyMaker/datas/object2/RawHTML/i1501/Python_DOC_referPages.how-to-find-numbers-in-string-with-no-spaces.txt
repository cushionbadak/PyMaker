<div class="post-text" itemprop="text">
<p>I would recommend you use a <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">regular expression</a>:</p>
<pre><code>import re

s = 'Is this the right number 560 or 670?'
nums = [int(n) for n in re.findall(r'\d+', s)]

print(nums)
# [560, 670]
</code></pre>
