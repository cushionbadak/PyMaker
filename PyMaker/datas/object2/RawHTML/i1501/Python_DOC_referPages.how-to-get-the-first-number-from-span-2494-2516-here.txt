<p>I want to cut a text from the point where my regex expression is found to the end of the text. The position may vary, so I need that number as a variable.
The position can already be seen in the result of <code>studentnrRegex.search(text)</code>:</p>
<pre><code>&gt;&gt;&gt; studentnrRegex = re.compile(r'(Studentnr = 18\d\d\d\d\d\d\d\d)')  
&gt;&gt;&gt; start = studentnrRegex.search(text)  
&gt;&gt;&gt; start  
&lt;_sre.SRE_Match object; span=(2494, 2516), match='Studentnr = 1825010243'&gt;
&gt;&gt;&gt; myText = text[2494:]  
&gt;&gt;&gt; myText 
'Studentnr = 1825010243\nTEXT = blablabla
</code></pre>
<p>Can I get the start position as a variable directly from my variable start, in this case 2494?</p>

<div class="post-text" itemprop="text">
<p>The <code>match</code> object returned by calling <code>.search()</code> has <a href="https://docs.python.org/3/library/re.html#re.Match.start" rel="nofollow noreferrer"><code>.start()</code></a> and <a href="https://docs.python.org/3/library/re.html#re.Match.end" rel="nofollow noreferrer"><code>.end()</code></a> methods that return the starting and ending positions of the match.</p>
<pre><code>studentnrRegex = re.compile(r'(Studentnr = 18\d\d\d\d\d\d\d\d)')
m = studentnrRegex.search(text)
start = m.start()
print(mytext[start:])
</code></pre>
<p>You can accomplish the same thing with a different regex that matches the student number and everything after it. This will save you the trouble of doing the slice:</p>
<pre><code>studentnrRegex = re.compile(r'(Studentnr = 18\d{8}).*', re.DOTALL)
m = studentnrRegex.search(text)
print(m.group())
</code></pre>
<p>The <code>{8}</code> matches 8 repeats of the <code>\d</code> and the <code>.*</code> matches all remaining characters until the end of the string (including newlines) as long as the <code>re.DOTALL</code> flag is specified. The full match is group 0, which is the default value for the <code>.group()</code> method of the match object. You can access the student number as <code>m.group(1)</code>.</p>
