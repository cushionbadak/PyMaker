<div class="post-text" itemprop="text">
<p>I read your question as wanting to validate that a string is a valid encoding name.  <a href="https://docs.python.org/3/library/codecs.html#codecs.lookup" rel="nofollow noreferrer">codecs.lookup(encoding)</a> can do that:</p>
<pre><code>&gt;&gt;&gt; import codecs
&gt;&gt;&gt; codecs.lookup('utf-8')
&lt;codecs.CodecInfo object for encoding utf-8 at 0x1e7c42905e8&gt;
&gt;&gt;&gt; codecs.lookup('abcd')
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
LookupError: unknown encoding: abcd
</code></pre>
<p>So you could use:</p>
<pre><code>def validate(name):
    try:
        codecs.lookup(name)
        return True
    except LookupError:
        return False
</code></pre>
