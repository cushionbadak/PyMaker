<p>There are many ways to do tests, often it depends on what framework you use. But, just assuming that you want some form of a formal test framework, you can use pytest. </p>
<p><strong>Step 1,</strong> install pytest</p>
<pre><code>$ pip install pytest
</code></pre>
<p><strong>Step 2,</strong> Format your code. You will test your output according to the return of the function. </p>
<pre><code>def host_is_pingable(x):
    #Do your verification logic here... putting the result in a string valid
    return valid
</code></pre>
<p><strong>Step 3,</strong> Now write the tests</p>
<pre><code>def test_answer():
    assert host_is_pingable(ip_address) != "some error"
</code></pre>
<p>Step 4, run the tests</p>
<pre><code>$ py.test
</code></pre>
<p>This is not a unittest, but a boilerplate test framework. </p>
<p>If you want to do unittests, there is a great guide to it here: <a href="http://www.onlamp.com/pub/a/python/2004/12/02/tdd_pyunit.html" rel="nofollow noreferrer">http://www.onlamp.com/pub/a/python/2004/12/02/tdd_pyunit.html</a></p>
