<div class="post-text" itemprop="text">
<p>Performance wise, you should consider using <em>list comprehensions</em> (as is mentioned in <em><a href="https://stackoverflow.com/a/48128945/2063361">Ursus's answer</a></em>)  with <code>str.split(' ')</code> as:</p>
<pre><code>&gt;&gt;&gt; a = ' girl\n    is'

&gt;&gt;&gt; [word for word in a.split(' ') if word]
['girl\n', 'is']
</code></pre>
<p>However if you are interested in <strong>functional approach</strong>, you may use <a href="https://docs.python.org/2/library/functions.html#filter" rel="nofollow noreferrer"><strong><code>filter</code></strong></a> as:</p>
<pre><code>&gt;&gt;&gt; list(filter(bool, a.split(' ')))
['girl\n', 'is']
</code></pre>
<p>Here <code>a.split(' ')</code> will split your string based on <em>space</em> as <em>separator</em> and <code>filter</code> <em>(with bool)</em> will filter out your empty strings from the list. </p>
<hr/>
<p><strong>Issue with your code</strong></p>
<p>As the Python's <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>str.split</code> document</a> says:</p>
<ul>
<li><p>if <em>separator</em> is not passed:</p>
<blockquote>
<p>or is <code>None</code>, a different splitting algorithm is applied: <strong>runs of consecutive whitespace are regarded as a single separator, and the result will contain no empty strings at the start or end if the string has leading or trailing whitespace</strong>. Consequently, splitting an empty string or a string consisting of just whitespace with a None separator returns [].</p>
</blockquote></li>
<li><p>if <em>separator</em> is passed:</p>
<blockquote>
<p><strong>consecutive delimiters are not grouped together and are deemed to delimit empty strings</strong>. Splitting an empty string with a specified separator returns [''].</p>
</blockquote></li>
</ul>

<div class="post-text" itemprop="text">
<p>You can use regular expressions in order to split on any <em>sequences</em> of <em>horizontal</em> whitespace (including e.g. tabs), but not vertical:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; re.split(r'[^\S\n\r]+', a.strip())
['girl\n', 'is']

# [^...]: not any of ...
# \S:     non whitespace
# \n, \r: line breaking white space 
# [^\S\n\r]+: one or more non-space or non-line-breaking space
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Easy to understand:</p>
<pre><code>b=a.split()
c=[i for i in b if i]
print(c)
</code></pre>
<p>Meaning:</p>
<pre><code>b=a.split()
c=[i for i in b if i!=' ']
print(c)
</code></pre>
