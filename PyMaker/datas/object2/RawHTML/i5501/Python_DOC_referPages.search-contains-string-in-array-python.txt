<div class="post-text" itemprop="text">
<p>Just go through each string in the list with a simple loop, and check if <code>'hello'</code> exists with the pythons membership <a href="https://docs.python.org/3/reference/expressions.html#in" rel="nofollow noreferrer"><code>in</code></a> operator:</p>
<pre><code>lst = ['123hello123', 'aasdasdasd123hello123', '123123hello']

for x in lst:
    if 'hello' in x:
        print('true')
</code></pre>
<p>Which outputs:</p>
<pre><code>true
true
true
</code></pre>
<p>Or if you want to check <a href="https://docs.python.org/3.3/library/functions.html#all" rel="nofollow noreferrer"><code>all()</code></a> the strings in <code>lst</code> at once:</p>
<pre><code>if all('hello' in x for x in lst):
    print('true')
</code></pre>
<p>Or if you want to check if <a href="https://docs.python.org/3.3/library/functions.html#any" rel="nofollow noreferrer"><code>any()</code></a> of the strings in <code>lst</code> at once:</p>
<pre><code>if any('hello' in x for x in lst):
    print('true')
</code></pre>
<p>Both of which will output:</p>
<pre><code>true
</code></pre>
<p><strong>Note:</strong> Using <code>list</code> as a variable name as shown in your question is not a good idea here, as it shadows the builtin function <a href="https://docs.python.org/3.3/library/functions.html#func-list" rel="nofollow noreferrer"><code>list()</code></a>. Also returning a boolean <code>True</code> or <code>False</code> here is fine, not need to return a string form of these. </p>

<p>Another solution is to have a own function as follows:</p>
<pre><code>def check_filename(filename_string, input_list):
    result = 'true'
    for _ in input_list:
        if filename_string not in _:
            result = 'false'
            break
    return result

filename = 'hello'
my_list = ['123hello123', 'aasdasdasd123hello123', '123123hello']

print(check_filename(filename_string=filename, input_list=my_list))

filename = 'hello'
