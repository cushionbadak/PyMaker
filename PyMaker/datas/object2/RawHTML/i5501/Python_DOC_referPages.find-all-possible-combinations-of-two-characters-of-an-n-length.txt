<div class="post-text" itemprop="text">
<p>I have two characters, for example:</p>
<pre><code>a = 'a'
b = 'b'
</code></pre>
<p>And I need to find all possible combinations of those two characters that will make a string of length N. For example, if <code>N = 3</code>:</p>
<pre><code>combinations = ['aab', 'aba', 'baa', 'abb', 'bab', 'bba', 'aaa', 'bbb']
</code></pre>
<p>Is there a function in some module for this?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow noreferrer"><code>itertools.product</code></a> is what you want here:</p>
<pre><code>&gt;&gt;&gt; from itertools import product
&gt;&gt;&gt; a = 'a'
&gt;&gt;&gt; b = 'b'
&gt;&gt;&gt; N = 3
&gt;&gt;&gt; lst = [a, b]
&gt;&gt;&gt; [''.join(x) for x in product(lst, repeat = N)]
['aaa', 'aab', 'aba', 'abb', 'baa', 'bab', 'bba', 'bbb']
</code></pre>
<p>Which can also be written with a triple nested list comprehension:</p>
<pre><code>&gt;&gt;&gt; [x + y + z for x in lst for y in lst for z in lst]
['aaa', 'aab', 'aba', 'abb', 'baa', 'bab', 'bba', 'bbb']
</code></pre>
</div>
<span class="comment-copy">The word you're looking for is cross product. Combinations are a different thing. And yes there is, in <a href="https://docs.python.org/2/library/itertools.html" rel="nofollow noreferrer">itertools</a>, but it might be beneficial to implement it yourself, especially if this is for a class assignment.</span>
