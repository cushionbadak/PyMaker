<p>Thank for @cᴏʟᴅsᴘᴇᴇᴅ and you already got the point.</p>
<p>try <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer">re.findall</a> : return a list of groups.</p>
<p>regex for capturing _0x263E9[digit]:</p>
<pre><code>re.findall(r'_0x263E9\[\d+\]', str)
</code></pre>
<p>or retrieving the digits only</p>
<pre><code>re.findall(r'_0x263E9\[\(d+)\]', str)
</code></pre>
