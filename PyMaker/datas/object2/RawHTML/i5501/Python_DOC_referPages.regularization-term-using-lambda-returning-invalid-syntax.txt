<div class="post-text" itemprop="text">
<p><code>lambda</code> is a <a href="https://docs.python.org/3/reference/lexical_analysis.html#keywords" rel="nofollow noreferrer">reserved keyword</a>, you can't use it as a general variable name.</p>
<p>Rename the variable; adding a <code>_</code> to the name would be an easy work-around:</p>
<pre><code>reg_term = (lambda_ / (2 * m)) * np.transpose(theta_revised).dot(theta_revised)
</code></pre>
<p>Note that you have not defined a value for <code>lambda_</code> anywhere, so no you will have a <code>NameError</code> exception. Also note that Python expressions do not need to end with <code>;</code>, you'd not normally use a semicolon.</p>
</div>
