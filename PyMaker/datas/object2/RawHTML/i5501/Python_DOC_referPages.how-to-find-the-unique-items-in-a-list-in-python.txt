<div class="post-text" itemprop="text">
<p>You may use <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><strong><code>collections.Counter</code></strong></a> to get the count of all the items present in the list, then filter it using <em>list comprehension</em> the items having count as <code>1</code>:</p>
<pre><code>&gt;&gt;&gt; from collections import Counter

&gt;&gt;&gt; mylist=[4, 4, 2, 1, 2]
&gt;&gt;&gt; [k for k, v in Counter(mylist).items() if v==1]
[1]
</code></pre>
