<p>To just print the line number:</p>
<pre><code>with open('victory.txt') as f_input:
    for index, row in enumerate(f_input):
        print('{} {}'.format(index, row), end='')
</code></pre>
<hr/>
<p>If you want to make changes to the existing file, you need to do it in two stages, i.e. read it in, close file, write out the updated values, for example:</p>
<pre><code>with open('victory.txt') as f_input:
    data = f_input.read().splitlines()

with open('victory.txt', 'w') as f_output:
    for index, row in enumerate(data):
        f_output.write('{} {}\n'.format(index, row))
</code></pre>
<p>Assuming that your input file is:</p>
<pre class="lang-none prettyprint-override"><code>Jenny
Carlos
Matty
Anna
</code></pre>
<p>You will get an updated output file:</p>
<pre class="lang-none prettyprint-override"><code>0 Jenny
1 Carlos
2 Matty
3 Anna
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html?highlight=enumerate#enumerate" rel="nofollow noreferrer"><code>enumerate()</code></a> is used to give you a counter for each line. If you wish to start the counter from <code>1</code> you could add <code>start=1</code>.</p>
<p><code>end=''</code> is used as each line read in from the file will already contain a trailing newline. Using <code>print()</code> also adds a newline, so with you need to remove the trailing newline or stop Python adding one when printing.</p>
