<p>Here is how you can write the turtle position on screen: the <code>write</code> method from the <code>turtle</code> module, must be called on a <code>Turtle</code> object.</p>
<pre><code>import turtle

playground = turtle.Screen()       # use nouns for objects, play is a verb

playground.bgcolor("black")
playground.screensize(250, 250)
playground.title("Turtle Keys")

tom = turtle.Turtle()              # use nouns for objects, run is a verb

tom.color("white")
tom.setposition(130, 100)
tom.speed("fastest")
tom.color("white")

p = tom.pos()               # here you get the coordinates of the turtle
tom.write(str(p), True)     # and you print a string representation on screen
tom.penup()

print(p)                    # this prints to the console

playground.exitonclick()
</code></pre>

<p>The way I approach this type of problem is to dedicate a turtle to each fixed label on the screen.  That way you can permanentely locate a turtle at that position and simply do <code>undo()</code> to clear the old text followed by <code>write()</code> to write the new.</p>
<p>Here's a dynamic example where you can drag the turtle around the screen with your mouse and the current position will be written to the center of the screen:</p>
<pre><code>from turtle import Turtle, Screen

FONT = ('Arial', 24, 'normal')

playground = Screen()
playground.setup(500, 500)
playground.bgcolor("black")

runner = Turtle("triangle")
runner.color("white")
runner.speed("fastest")
runner.penup()
runner.setposition(200, 200)

marker = Turtle(visible=False)  # our virtual magic marker
marker.penup()
marker.color("yellow")
marker.write(runner.position(), align='center', font=FONT)  # so we can undo it

def drag_handler(x, y):
    runner.ondrag(None)  # disable handler while in handler
    marker.undo()  # erase previous position
    marker.write((x, y), align='center', font=FONT)
    runner.setheading(runner.towards(x, y))  # head towards new location
    runner.setposition(x, y)
    runner.ondrag(drag_handler)

runner.ondrag(drag_handler)

playground.mainloop()
</code></pre>

<p>Ok. Here's the explanation.</p>
<p>So I defined a function writing the position of something (in this case, the turtle).</p>
<p>When I do:</p>
<pre><code>self.pu()
self.bk(len(str(position))-len(str(position))/2)
</code></pre>
<p>I:</p>
<ul>
<li>Lift the pen up and</li>
<li>Move backward some steps, so I can center the text.</li>
</ul>
<p>I then do:</p>
<pre><code>self.pd()
self.write(str(position))
</code></pre>
<p>Which:</p>
<ul>
<li>Puts the pen down and</li>
<li>Writes (or outputs) the position</li>
</ul>
<p>After that, when I do:</p>
<pre><code>self.pu()
self.fd(len(str(position))-len(str(position))/2)
self.pd()
</code></pre>
<p>I:</p>
<ul>
<li>Pick the pen up</li>
<li>Move it back to the original position and</li>
<li>Put the pen down again</li>
</ul>
