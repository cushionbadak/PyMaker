<p>I have a table called recentOrders and I want to return the row which has the highest value (basically the latest order number) from the column OrderID.</p>
<p>I can return a single seemingly random row with:</p>
<pre><code>s = recentOrders.select()
rs = s.execute()
row = rs.fetchone() 
print(row)
</code></pre>
<p>and I've tried various things along the lines of:</p>
<pre><code>s = recentOrders.select().order_by(OrderID.desc())
rs = s.execute()
row = rs.fetchone()
print(row)
</code></pre>
<p>But I get an error of</p>
<blockquote>
<p>NameError: name 'OrderID' is not defined</p>
</blockquote>
<p>What's the correct syntax for a query?</p>
</div>
<div class="post-text" itemprop="text">
<p>Thanks to Ilja Everil√§ for the answer.</p>
<pre><code>s = recentOrders.select().order_by(recentOrders.c.OrderID.desc())
rs = s.execute()
row = rs.fetchone()
</code></pre>
</div>
<span class="comment-copy">Try <code>recentOrders.c.OrderID.desc()</code>, if <code>recentOrders</code> is a <code>Table</code>. That NameError is the result of trying to use a non existent variable name, so perhaps <a href="https://docs.python.org/3/tutorial/" rel="nofollow noreferrer">refresh your Python basics</a>.</span>
<span class="comment-copy">Thank you, what does the c mean in the recentOrders.c.OrderID.desc() ?</span>
<span class="comment-copy"><a href="http://docs.sqlalchemy.org/en/latest/core/metadata.html#sqlalchemy.schema.Table.c" rel="nofollow noreferrer">It is an alias</a> of the <a href="http://docs.sqlalchemy.org/en/latest/core/metadata.html#sqlalchemy.schema.Table.columns" rel="nofollow noreferrer"><code>columns</code> attribute</a>, which is a name-based collection of columns of the <code>Table</code>. I'd suggest you also read the <a href="http://docs.sqlalchemy.org/en/latest/core/tutorial.html" rel="nofollow noreferrer">SQLAlchemy tutorials</a>.</span>
