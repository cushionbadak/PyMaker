<p>Suppose I have a dictionary:</p>
<pre><code>dict = {1: 'a', 2: 'b', 3: 'a'}
</code></pre>
<p>Is there an efficient way to invert the dictionary such that keys with identical values are grouped together?</p>
<p>The output should be:</p>
<pre><code>{'a': [1, 3], 'b': [3]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow noreferrer"><strong><code>defaultdict</code></strong></a> with a <code>list</code> as default value.</p>
<p>This creates a list for new keys and you can append to it immediately:</p>
<pre><code>from collections import defaultdict

original = {1: 'a', 2: 'b', 3: 'a'}

flipped = defaultdict(list)

for key, value in original:
    flipped[value].append(key)
</code></pre>
