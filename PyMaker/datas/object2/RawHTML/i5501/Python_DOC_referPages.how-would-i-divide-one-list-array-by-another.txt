<div class="post-text" itemprop="text">
<p>A nested loop will do:</p>
<pre><code>&gt;&gt;&gt; list1 = [1,2,3]
&gt;&gt;&gt; list2 = [3,2,1]
&gt;&gt;&gt; for x1 in list1:
...     for x2 in list2:
...         print(x1/x2)  # Python3
...         print(float(x1)/x2)  # Python2
</code></pre>
<p>Or <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow noreferrer"><code>itertools.product</code></a>:</p>
<pre><code>&gt;&gt;&gt; from itertools import product
&gt;&gt;&gt; for x1, x2 in product(list1, list2):
...     print(x1/x2)  # Python3
...     print(float(x1)/x2)  # Python2

0.3333333333333333
0.5
1.0
0.6666666666666666
1.0
2.0
1.0
1.5
3.0
</code></pre>

<div class="post-text" itemprop="text">
<pre><code>from itertools import product

div_list = [a / b for a, b in product(list1, list2)]

print('\n'.join(str(x) for x in div_list))
</code></pre>
