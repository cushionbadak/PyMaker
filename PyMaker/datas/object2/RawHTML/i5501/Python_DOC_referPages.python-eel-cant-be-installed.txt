<div class="post-text" itemprop="text">
<p>The <code>open</code> function in Python 2.7 has a different signature and especially no keyword argument called <code>encoding</code>, as you can read from the error message.</p>
<p>Eel seems to no support Python 2.7 (you should go with Python 3 anyways ;)</p>
<p><code>open</code> in Python 2.7: <a href="https://docs.python.org/2/library/functions.html#open" rel="nofollow noreferrer">https://docs.python.org/2/library/functions.html#open</a></p>
<p><code>open</code> in Python 3.6: <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer">https://docs.python.org/3/library/functions.html#open</a></p>
</div>
