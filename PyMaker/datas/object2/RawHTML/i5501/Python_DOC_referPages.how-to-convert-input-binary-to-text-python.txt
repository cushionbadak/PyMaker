<div class="post-text" itemprop="text">
<p>You should not use a lookup dict for this.  To convert a binary string to an ascii character, use built-in functions <a href="https://docs.python.org/3/library/functions.html#chr" rel="nofollow noreferrer"><code>chr</code></a> and <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow noreferrer"><code>int</code></a> respectively:</p>
<pre><code>&gt;&gt;&gt; chr(int('01100001', 2))
'a'
</code></pre>
<p>To go the other way, use built-in functions <a href="https://docs.python.org/3/library/functions.html#format" rel="nofollow noreferrer"><code>format</code></a> and <a href="https://docs.python.org/3/library/functions.html#ord" rel="nofollow noreferrer"><code>ord</code></a> respectively:</p>
<pre><code>&gt;&gt;&gt; format(ord('a'), '08b')
'01100001'
</code></pre>
