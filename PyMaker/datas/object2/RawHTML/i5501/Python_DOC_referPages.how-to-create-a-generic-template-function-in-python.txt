<div class="post-text" itemprop="text">
<p>If you are using unittest library, you can use the functions setUp() and tearDown() in you test class. setUp() is run before each test function and tearDown() after. For examples you can use this link:</p>
<p><a href="https://docs.python.org/3/library/unittest.html" rel="nofollow noreferrer">https://docs.python.org/3/library/unittest.html</a></p>
</div>
