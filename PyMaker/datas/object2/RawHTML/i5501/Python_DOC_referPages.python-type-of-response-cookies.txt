<p>I'm not sure why you think there's a contradiction here. The <a href="https://github.com/requests/requests/blob/master/requests/cookies.py#L172" rel="nofollow noreferrer">RequestsCookieJar</a> class implements the dictionary interface by subclassing <a href="https://docs.python.org/3/library/collections.abc.html" rel="nofollow noreferrer"><code>collections.MutableMapping</code></a>, so you can access items by key.</p>
</div>
