<p>The approach I would take to solve both issues is don't do the position comparisons yourself, use the <code>distance()</code> method that turtle provides:</p>
<pre><code>if player.distance(enemy) &lt; 10:  # if the centers are within 10 pixels
</code></pre>
<p>Here's a complete example that lets you click on the screen to move the player around but if the player gets too close to the heart of the enemy, it will teleport back to the start where you can once again move by clicking on the screen:</p>
<pre><code>from turtle import Turtle, Screen

wn = Screen()
wn.setup(500, 500)
wn.bgcolor('black')

player = Turtle(visible=False)
player.speed('fastest')
player.shapesize(2)
player.color('white')
player.penup()
player.setposition(200, -200)
player.showturtle()

enemy = Turtle('circle', visible=False)
enemy.shapesize(2)
enemy.color('red')
enemy.penup()
enemy.setposition(-200, 200)
enemy.showturtle()

def click_handler(x, y):
    wn.onclick(None)  # disable handler while in handler
    player.setheading(player.towards(x, y))  # head towards new location
    player.setposition(x, y)

    if player.distance(enemy) &lt; 10:
        player.hideturtle()  # teleport
        player.setposition(200, -200)
        player.showturtle()

    wn.onclick(click_handler)

wn.onclick(click_handler)

wn.mainloop()
</code></pre>

<p><code>Turtle.pos()</code> <em>returns</em> the turtle’s position; it doesn’t <em>test</em> being present at a given position (as attempted by your <code>if</code> line) or <em>set</em> it (as attempted by your last line).  The former job is performed by <a href="https://docs.python.org/3/library/turtle.html#turtle.Vec2D" rel="nofollow noreferrer"><code>Vec2D</code></a> comparison, while the latter is <code>Turtle.goto()</code>.</p>
<p>Separately, your code contains no provision for user input, so there is no way to “react” to the player arriving anywhere; perhaps you already know that.</p>

<p>In your if statement, you wrote <code>player.pos(100, -100)</code>.  <code>Turtle.pos()</code>returns the Turtle's position. To check if this position is equal to (100, -100), simple do: <code>if player.pos()==(100,-100)</code>. To make change the player's position, use <code>Turtle.goto(x,y)</code>. The following code should be what you are trying to do: </p>
<pre><code>if player.pos()==(100,-100):
    player.goto(100, -110)
