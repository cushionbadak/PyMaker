<p>The error is not in same but in previous line!</p>

<p>Python is expecting an indented <code>except:</code> statement. Hence, the error.</p>
<p>Other truely indent errors also exist in this code, which will become apparent once this error is corrected.</p>
<p>Maybe Python should give a <code>"missing except" error</code> rather than <code>"indent" error</code>.</p>
