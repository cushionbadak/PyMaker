<p>I have a code that checks if a string is in a list of strings like,</p>
<pre><code>    for a in array:
        if x in a:
            return True
    return False
</code></pre>
<p>if a string is found in the list, return <code>True</code> and <code>False</code> otherwise. I am wondering if it can be written in one line of code.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer"><code>any</code></a> to iterate through a sequence and see if any item matches some particular condition.</p>
<pre><code>return any(x in a for a in array)
</code></pre>
