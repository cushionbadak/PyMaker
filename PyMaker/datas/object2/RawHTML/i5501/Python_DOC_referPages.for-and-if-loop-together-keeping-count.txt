<p>How do I keep a count while executing a IF statement within a for loop?  I want to execute different code based on the counter.  Here is a simplified version of what I'm trying to do:</p>
<pre><code>li = [1,2,3,4,5]

for l in li:
    counter = 1 
    if counter == 1:
        counter +=1
        print(l)
else:
    print('this worked')
</code></pre>
<p>Results:</p>
<pre><code>1
2
3
4
5
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you want the number of times the <code>if</code> condition was hit. Declare the <code>counter</code> variable before the <code>for</code> loop</p>
</div>
<span class="comment-copy">Use the <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a> function</span>
<span class="comment-copy">Yes, you set <code>counter</code> to <code>1</code> <b>each iteration</b>, so it is <b>always</b> equal to 1.</span>
<span class="comment-copy"><code>for x, l in enumerate(li)</code>. <code>x</code> is your counter.</span>
<span class="comment-copy"><code>for counter, l in enumerate(li, 1): print(counter, l)</code>.</span>
