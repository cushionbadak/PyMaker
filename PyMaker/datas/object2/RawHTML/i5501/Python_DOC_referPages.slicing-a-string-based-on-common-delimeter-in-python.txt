<div class="post-text" itemprop="text">
<p>You can get each item by just splitting on <code>'|'</code> with <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>str.split()</code></a>, and using <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>str.strip()</code></a> to take away the leading and trailing whitespace of each item:</p>
<pre><code>address = '1234 Broadway Ave | Los Angeles | CA | 94530'

items = [x.strip() for x in address.split('|')]

print(items)
</code></pre>
<p>Which gives:</p>
<pre><code>['1234 Broadway Ave', 'Los Angeles', 'CA', '94530']
</code></pre>
<p>Additionally, you can also do this nicely with <a href="https://docs.python.org/3.3/library/functions.html#map" rel="nofollow noreferrer"><code>map()</code></a>:</p>
<pre><code>items = list(map(str.strip, address.split('|')))
</code></pre>
<p>You could also assign these items to a dictionary using <a href="https://docs.python.org/3.3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a>, like so:</p>
<pre><code>contents = ["street", "city", "state", "zip"]
d = dict(zip(contents, items))
print(d)
</code></pre>
<p>Which would give you this structure:</p>
<pre><code>{'street': '1234 Broadway Ave', 'city': 'Los Angeles', 'state': 'CA', 'zip': '94530'}
</code></pre>
<p>Then you could access each item in this dictionary by simply calling the keys:</p>
<pre><code>&gt;&gt;&gt; d['street']
1234 Broadway Ave
&gt;&gt;&gt; d['city']
Los Angeles
&gt;&gt;&gt; d['state']
CA
&gt;&gt;&gt; d['zip']
94530
</code></pre>
</div>
