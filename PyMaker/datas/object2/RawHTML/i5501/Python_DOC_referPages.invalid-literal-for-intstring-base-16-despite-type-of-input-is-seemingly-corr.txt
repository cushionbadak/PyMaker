<p>You have an extra embedded character in your text which has hex value <code>1D</code>.  To remove it try this:</p>
<h3>Code:</h3>
<pre><code>import string

def filter_out_junk(text):
    return ''.join(x for x in text if x in set(string.printable))
</code></pre>
<p>This will remove anything which is not considered <a href="https://docs.python.org/3/library/string.html#string.printable" rel="nofollow noreferrer">printable</a>.</p>
<h3>Test Code:</h3>
<pre><code>junk_string = '\x1dB'
print(int(filter_out_junk(junk_string), 16))
print(int(junk_string, 16))
</code></pre>

<pre><code>11
