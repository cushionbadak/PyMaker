<div class="post-text" itemprop="text">
<p>You're appending a list to a list. What you really want to do is <code>extend</code> the first list with each item of the second list:</p>
<pre><code>&gt;&gt;&gt; c1 = [1, 2, 3]
&gt;&gt;&gt; c2 = ['a', 'b']
&gt;&gt;&gt; c1.append(c2)
&gt;&gt;&gt; c1
[1, 2, 3, ['a', 'b']]
&gt;&gt;&gt; c1 = [1, 2, 3]
&gt;&gt;&gt; c1.extend(c2)
&gt;&gt;&gt; c1
[1, 2, 3, 'a', 'b']
</code></pre>
<p>See the <a href="https://docs.python.org/3/library/stdtypes.html#common-sequence-operations" rel="nofollow noreferrer">sequence operations</a></p>
</div>
<div class="post-text" itemprop="text">
<p>As @JoshSmeaton said, using <code>extend</code> rather than <code>append</code> can make it.</p>
<p>I just post another method in case situations where you already get a nested list, and can't change how the list adds new element.</p>
<pre><code>flatList = []

def unfold(X):
    for x in X:
        if isinstance(x, list):
            unfold(x)
        else:
            flatList.append(x)
</code></pre>
<p>Input:</p>
<pre><code>nestedList = [1, 2, [3, 4, 5]]
unfold(nestedList)
print(flatList)
</code></pre>
<p>Output:</p>
<pre><code>[1, 2, 3, 4, 5]
</code></pre>
