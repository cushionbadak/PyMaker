<div class="post-text" itemprop="text">
<p>Let's assume that we have this code:</p>
<pre><code>name = line.split('+')[-1]
</code></pre>
<p>What does the -1 do? I have seen it in various codes but not sure on what it does?
And what would the difference be if there was a <code>[0]</code> or a <code>[1]</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>The line of code you gave is basically doing three things:</p>
<ol>
<li><p>It takes the string <code>line</code> and splits it on <code>+</code>'s using <a href="http://docs.python.org/2.7/library/stdtypes.html#str.split" rel="noreferrer"><code>str.split</code></a>.  This will return a list of substrings:</p>
<pre><code>&gt;&gt;&gt; line = 'a+b+c+d'
&gt;&gt;&gt; line.split('+')
['a', 'b', 'c', 'd']
&gt;&gt;&gt;
</code></pre></li>
<li><p>The <code>[-1]</code> then indexes that list at position <code>-1</code>. Doing so will return the last item:</p>
<pre><code>&gt;&gt;&gt; ['a', 'b', 'c', 'd'][-1]
'd'
&gt;&gt;&gt;
</code></pre></li>
<li><p>It takes this item and assigns it as a value for the variable <code>name</code>.</p></li>
</ol>
<p>Below is a more complete demonstration of the concepts mentioned above:</p>
<pre><code>&gt;&gt;&gt; line = 'a+b+c+d'
&gt;&gt;&gt; line.split('+')
['a', 'b', 'c', 'd']
&gt;&gt;&gt; lst = line.split('+')
&gt;&gt;&gt; lst[-1]
'd'
&gt;&gt;&gt; lst[0]
'a'
&gt;&gt;&gt; lst[1]
'b'
&gt;&gt;&gt; lst[2]
'c'
&gt;&gt;&gt; lst[3]
'd'
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Negative indexes in Python are syntactic sugar for accessing the elements in reverse order, from right-to-left, starting in <code>-1</code>. So <code>-1</code> is the last item, <code>-2</code> is the second-to-last item, and so on - the first item would be <code>lst[-len(lst)]</code>. For example:</p>
<pre><code>lst = [1, 2, 3]
lst[-1]
=&gt; 3
lst[-2]
=&gt; 2
lst[-3]
=&gt; 1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="http://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>str.split</code></a> returns a list:</p>
<pre><code>&gt;&gt;&gt; '1+2+3'.split('+')
['1', '2', '3']
</code></pre>
<p><code>list[-1]</code> yields the last item (negative index starts from -1)</p>
<pre><code>&gt;&gt;&gt; '1+2+3'.split('+')[-1]
'3'
&gt;&gt;&gt; '1+2+3'.split('+')[0] # the first item (Python index starts from 0)
'1'
&gt;&gt;&gt; '1+2+3'.split('+')[1]
'2'
</code></pre>
<p>See <a href="http://docs.python.org/3/tutorial/introduction.html#lists" rel="nofollow"><code>Lists</code> - Python tutorial</a> (contains indexing, slicing).</p>
</div>
