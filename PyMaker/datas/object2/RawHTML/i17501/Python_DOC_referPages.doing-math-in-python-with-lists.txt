<p>Say I have a list...  How would I subtract the largest number with the smallest?</p>
<pre><code>lst = [23,41,54,65,73]
</code></pre>
<p>what I figure I do is </p>
<pre><code>mth = (lst[4]-lst[1])
</code></pre>
<p>but how would I make it find the largest number in the list and subtract it with the smallest?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>max</code> and <code>min</code>, which are explained <a href="http://docs.python.org/3/library/functions.html">here</a>:</p>
<pre><code>&gt;&gt;&gt; lst = [23,41,54,65,73]
&gt;&gt;&gt; max(lst) - min(lst)
50
&gt;&gt;&gt;
</code></pre>
<p><code>max</code> will get the largest number and <code>min</code> will get the smallest.</p>
</div>
