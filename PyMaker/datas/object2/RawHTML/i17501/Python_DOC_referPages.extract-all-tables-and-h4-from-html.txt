<div class="post-text" itemprop="text">
<p>I suggest using the module <a href="https://pypi.python.org/pypi/beautifulsoup4" rel="nofollow">BeautifulSoup</a>.</p>
<p>You could accomplish what you want by doing:</p>
<pre><code>    from bs4 import BeautifulSoup

    code = file("file.html")
    html = code.read()
    soup = BeautifulSoup(html)
    htag = soup.findall('h4')
    tabletag = soup.findall('table')
    for h in htag:
        print h.text
    for table in tabletag:
        print table.text
</code></pre>

<p>Since BeautifulSoup already has been mentioned i just want to hint at the tools of the standard library. </p>
<p>You may use the <a href="http://docs.python.org/3/library/html.parser.html#examples" rel="nofollow noreferrer">builtin html parser</a> or  an <a href="http://docs.python.org/3/library/re.html" rel="nofollow noreferrer">regular expression</a> (see <a href="http://docs.python.org/3/howto/regex.html" rel="nofollow noreferrer">tutorial</a>).</p>
<p>Sometimes these tools are sufficient. It depends on the task.</p>
<p>BTW: Notepad++ supports regular expressions.<code>&lt;h4.*?/h4&gt;</code> or <code>&lt;table.*?/table&gt;</code> allows you to select those blocks.<img alt="enter image description here" src="https://i.stack.imgur.com/wFkKV.png"/></p>
