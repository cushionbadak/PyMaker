<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; oldList = [1,23,4,3,5,345,4]
&gt;&gt;&gt; newList = [x for i, x in enumerate(oldList) if i != 3] # by index
&gt;&gt;&gt; newList
[1, 23, 4, 5, 345, 4]

&gt;&gt;&gt; newList = [x for x in oldList if x != 4] # by value
&gt;&gt;&gt; newList
[1, 23, 3, 5, 345]
</code></pre>

<div class="post-text" itemprop="text">
<p>Consider using the built-in function <a href="http://docs.python.org/2.7/library/functions.html#filter" rel="nofollow"><code>filter</code></a></p>
<pre><code>oldList = [1, 2, 3, 4, 5]
newList = filter(lambda number: number != 3, oldList)

# newList == [1,2,4,5]
</code></pre>
<p><code>filter</code>'s arguments are <code>function, iterable</code>.  It applies the function you give it to every element of the iterable and returns a list of the elements that the function returns True on.</p>
