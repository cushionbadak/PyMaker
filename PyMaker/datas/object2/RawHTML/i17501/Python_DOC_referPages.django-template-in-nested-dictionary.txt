<p>I am using Django template, and I met one problem with nested dictionary.</p>
<p>Dict:</p>
<pre><code>result_dict = {'type_0' : {'file_name' : 'abc', 'count' : 0},
               'type_1' : {'file_name' : 'xyz', 'count' : 50}}
</code></pre>
<p>and the template in my HTML file is:</p>
<pre><code>{% for type in result_dict %}
    {{ type }}, {{ type.file_name }}
{% endfor %}
</code></pre>
<p>How can I show the value of type_0 and type_1 only ?</p>
<p>I've tried:</p>
<pre><code>{% for key, value in result_dict %}
    {{ key }}, {{ value }}
{% endfor %}
</code></pre>
<p>but it doesn't work.</p>
<p>Thank you for your help.</p>

<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/stdtypes#dict.items" rel="nofollow"><code>dict.items</code></a> or <a href="http://docs.python.org/3/library/stdtypes#dict.values" rel="nofollow"><code>dict.values</code></a>:</p>
<pre><code>{% for key, value in result_dict.items %}
    {{ value }}
{% endfor %}
</code></pre>
<p>Example in interactive shell:</p>
<pre><code>&gt;&gt;&gt; result_dict = {'type_0' : {'file_name' : 'abc', 'count' : 0},
...                'type_1' : {'file_name' : 'xyz', 'count' : 50}}
&gt;&gt;&gt;
&gt;&gt;&gt; t = Template('''
... {% for key, value in result_dict.items %}
...     {{ value }}
... {% endfor %}
... ''')
&gt;&gt;&gt; print(t.render(Context({'result_dict': result_dict})))


    {&amp;#39;count&amp;#39;: 50, &amp;#39;file_name&amp;#39;: &amp;#39;xyz&amp;#39;}

    {&amp;#39;count&amp;#39;: 0, &amp;#39;file_name&amp;#39;: &amp;#39;abc&amp;#39;}
</code></pre>
<hr/>
<pre><code>&gt;&gt;&gt; t = Template('''
... {% for key, value in result_dict.items %}
...     {{ value|safe }}
... {% endfor %}
... ''')
&gt;&gt;&gt; print(t.render(Context({'result_dict': result_dict})))


    {'count': 50, 'file_name': 'xyz'}

    {'count': 0, 'file_name': 'abc'}
</code></pre>
