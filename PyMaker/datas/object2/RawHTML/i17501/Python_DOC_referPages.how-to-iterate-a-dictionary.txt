<div class="post-text" itemprop="text">
<p>You can use dict <code>items</code> function and list comprehension:</p>
<pre><code>&gt;&gt;&gt; [c for c,s in slides.items() if s == 'active']
['blue', 'orange', 'navy', 'red']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/reference/compound_stmts.html#the-if-statement" rel="nofollow"><code>if</code> statement</a>:</p>
<pre><code>&gt;&gt;&gt; slides = {
...     'blue': 'active',
...     'red': 'active',
...     'green': 'unactive',
...     'orange': 'active',
...     'navy': 'active'
... }
&gt;&gt;&gt;
&gt;&gt;&gt; li = []
&gt;&gt;&gt; for color in slides:
...     if slides[color] == 'active': # &lt;------
...         li.append(color)
...
&gt;&gt;&gt; li
['blue', 'orange', 'navy', 'red']
</code></pre>
<p>Call to <code>keys()</code> is redundant. Iterating a dictionary yields keys.</p>
</div>
<span class="comment-copy">See also <code>dict.iteritems</code> in python 2.x.</span>
