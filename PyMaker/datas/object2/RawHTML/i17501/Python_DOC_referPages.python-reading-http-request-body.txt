<div class="post-text" itemprop="text">
<p><code>\xc3\xb5</code> is UTF-8 representation of U+00F5</p>
<pre><code>&gt;&gt;&gt; u'\xf5'.encode('utf-8')
'\xc3\xb5'
</code></pre>
<p>You can covert it back using <a href="http://docs.python.org/2/library/stdtypes.html#str.decode" rel="nofollow"><code>str.decode</code></a> (<a href="http://docs.python.org/3/library/stdtypes.html#bytes.decode" rel="nofollow"><code>bytes.decode</code></a> in Python 3.x):</p>
<pre><code>&gt;&gt;&gt; b'\xc3\xb5'.decode('utf-8')
u'\xf5'
</code></pre>
