<p>I am brand new to python and I am trying to figure out how to validate user input. I want to ask the user to submit a DNA sequence and want to validate that it is a DNA sequence. Acceptable inputs can have upper or lowercase ATCGs and spaces and I'm not sure exactly how to do it.</p>
<p>So far I can ask for the input but not verify it.</p>
<pre><code>import sys
Var1 = raw_input("Enter Sequence 1:")
</code></pre>
<p>I then want to do something like:</p>
<pre><code>if Var1 != ATCG (somehow put 'characters that match ATCG or space)
    print "Please input valid DNA sequence"
    sys.exit() (to have it close the program)
</code></pre>
<p>Any help? I feel like this should be rather simple but I don't know how to specify that it can be any ATCG, atcg or space.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/3/library/functions.html#all" rel="nofollow"><code>all</code></a>, <a href="http://docs.python.org/3/library/stdtypes.html#str.lower" rel="nofollow"><code>str.lower</code></a>, and a <a href="http://www.python.org/dev/peps/pep-0289/" rel="nofollow">generator expression</a>:</p>
<pre><code>if not all(x in "agct " for x in Var1.lower()):
    print "Please input valid DNA sequence"
    sys.exit(1) 
</code></pre>
<p>In the above code, the last two lines will be run if any character in <code>Var1</code> is not one of the following:</p>
<pre><code>"A", "T", "C", "G", " ", "a", "t", "c", "g"
</code></pre>
</div>

<pre><code>&gt;&gt;&gt; import re
