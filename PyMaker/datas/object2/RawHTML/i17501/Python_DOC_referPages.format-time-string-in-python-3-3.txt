<p>I am trying to get current local time as a string in the format: year-month-day hour:mins:seconds. Which I will use for logging. By my reading of the documentation I can do this by:</p>
<pre><code>import time
'{0:%Y-%m-%d %H:%M:%S}'.format(time.localtime())
</code></pre>
<p>However I get the error:</p>
<pre>
Traceback (most recent call last):
File "", line 1, in 
ValueError: Invalid format specifier
</pre>
<p>What am I doing wrong? Is there a better way?</p>
</div>
<div class="post-text" itemprop="text">
<p><code>time.localtime</code> returns <code>time.struct_time</code> which does not support strftime-like formatting. </p>
<p>Pass <a href="http://docs.python.org/3/library/datetime.html#datetime.datetime"><code>datetime.datetime</code></a> object which support strftime formatting. (See <a href="http://docs.python.org/3/library/datetime.html#datetime.datetime.__format__"><code>datetime.datetime.__format__</code></a>)</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; '{0:%Y-%m-%d %H:%M:%S}'.format(datetime.datetime.now())
'2014-02-07 11:52:21'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can alternatively use <a href="https://docs.python.org/3.5/library/time.html#time.strftime" rel="noreferrer"><code>time.strftime</code></a>:</p>
<pre><code>time.strftime('{%Y-%m-%d %H:%M:%S}')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>And for newer versions of Python (3.6+, <a href="https://www.python.org/dev/peps/pep-0498/" rel="nofollow noreferrer">https://www.python.org/dev/peps/pep-0498/</a> purely for completeness), you can use the newer string formatting, ie.</p>
<pre><code>import datetime

today = datetime.date.today()

f'{today:%Y-%m-%d}'
&gt; '2018-11-01'
</code></pre>
</div>
