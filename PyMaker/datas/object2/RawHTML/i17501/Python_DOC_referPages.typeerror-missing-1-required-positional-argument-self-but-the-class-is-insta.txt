<div class="post-text" itemprop="text">
<p>You are calling the unbound method on the <em>parent</em> class:</p>
<pre><code>def vorstellen(self):
    Tier.vorstellen()
    print ("Ich fresse {0}".format(self.futter))
</code></pre>
<p>Add in <code>self</code> there:</p>
<pre><code>def vorstellen(self):
    Tier.vorstellen(self)
    print ("Ich fresse {0}".format(self.futter))
</code></pre>
<p>or use <a href="http://docs.python.org/3/library/functions.html#super" rel="nofollow"><code>super()</code></a> to load the next method in the hierarchy; <code>super()</code> binds the method for you:</p>
<pre><code>def vorstellen(self):
    super().vorstellen()
    print ("Ich fresse {0}".format(self.futter))
</code></pre>
<p>This assumes you are using Python 3; in Python 2, <code>Zoo</code> must inherit from <code>object</code> and you must pass in the current class (by name) and <code>self</code> to <code>super()</code>:</p>
<pre><code>def vorstellen(self):
    super(Affe, self).vorstellen()
    print ("Ich fresse {0}".format(self.futter))
</code></pre>
<p>Inheriting from <code>object</code> makes the <code>Zoo</code> class and all subclasses new-style classes. This is the default in Python 3.</p>
