<p>I need a code to choose between two characters within the same string.</p>
<p>Example: </p>
<p>I want to find <code>acf('h' or 'n')atd</code> in </p>
<pre><code>'avhavaacfhatdvatyacyactycatycasacfnatdtycstyscaacfnatdtycsatycatycaycsa'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is a problem which is solved using regular expressions. Try using <a href="http://docs.python.org/3.3/library/re.html#re.findall" rel="nofollow">findall</a> from the <a href="http://docs.python.org/3.3/library/re.html" rel="nofollow">re module</a>:</p>
<pre><code>import re

s = 'avhavaacfhatdvatyacyactycatycasacfnatdtycstyscaacfnatdtycsatycatycaycsa'
matches = re.findall('acf[hn]atd', s)
print matches
</code></pre>
<p>Output:</p>
<pre><code>['acfhatd', 'acfnatd', 'acfnatd']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="http://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><code>findall</code></a> only returns all non-overlapping matches of pattern in string, as a list of strings.
It doesn't contain inforamtion about <strong>position</strong>, which is probably what you want.</p>
<p>Try <a href="http://docs.python.org/3/library/re.html#re.finditer" rel="nofollow noreferrer"><code>finditer</code></a> instead, it returns an iterator yielding <a href="http://docs.python.org/3/library/re.html#match-objects" rel="nofollow noreferrer">match objects</a>, while match objects contain those info. </p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; s = 'avhavaacfhatdvatyacyactycatycasacfnatdtycstyscaacfnatdtycsatycatycaycsa'
&gt;&gt;&gt; match_iter = re.finditer('acf[hn]atd', s)
&gt;&gt;&gt; for match in match_iter:
        print("%s, start: %s, end: %s" % (match.group(), match.start(), match.end()))

acfhatd, start: 6, end: 13
acfnatd, start: 31, end: 38
acfnatd, start: 47, end: 54
</code></pre>
<p>The above code doesn't work if matched patterns overlap. To find all overlapping matches, see <a href="https://stackoverflow.com/questions/5616822/python-regex-find-all-overlapping-matches">Python regex find all overlapping matches?</a>.</p>
<p>For your question</p>
<pre><code>&gt;&gt;&gt; matcheobj = re.finditer(r'(?=(acf[hn]atd))', s)
</code></pre>
<p>Since your specific pattern definitly won't lead to overlapping, it makes no difference which form you use.</p>
