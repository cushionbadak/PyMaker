<div class="post-text" itemprop="text">
<p>How do you set/get the values of attributes of <code>t</code> given by <code>x</code>. </p>
<pre><code>class test():
   attr1 = int
   attr2 = int

t = test()
x = "attr1"
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>There is built-in functions called <a href="https://docs.python.org/2/library/functions.html#getattr" rel="noreferrer"><code>getattr</code></a> and <a href="https://docs.python.org/2/library/functions.html#setattr" rel="noreferrer"><code>setattr</code></a></p>
<pre><code>getattr(object, attrname)
setattr(object, attrname, value)
</code></pre>
<p>In this case </p>
<pre><code>x = getattr(t, "attr1")
setattr(t, 'attr1', 21)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>There is python built in functions setattr and getattr. Which can used to set and get the attribute of an class.</p>
<p>A brief example:</p>
<pre><code>&gt;&gt;&gt; from new import  classobj

&gt;&gt;&gt; obj = classobj('Test', (object,), {'attr1': int, 'attr2': int}) # Just created a class

&gt;&gt;&gt; setattr(obj, 'attr1', 10)

&gt;&gt;&gt; setattr(obj, 'attr2', 20)

&gt;&gt;&gt; getattr(obj, 'attr1')
10

&gt;&gt;&gt; getattr(obj, 'attr2')
20
</code></pre>
</div>
<span class="comment-copy">There is also delattr for deleting attributes, but this is rarely used.</span>
<span class="comment-copy">and hasattr for testing whether or not an object has a specific attr though in that case using the three argument form getattr(object, attrname, default) is often better.</span>
<span class="comment-copy">This is definitely the graceful way!</span>
