<div class="post-text" itemprop="text">
<p>You're using a font that doesn't support Hangul characters.  The <a href="https://pyfpdf.readthedocs.io/en/latest/Unicode/index.html" rel="nofollow noreferrer">fpdf docs</a> use the eunjin font for Hangul:</p>
<pre><code># Add a Alee Unicode font (uses UTF-8)
# General purpose Hangul truetype fonts that contain Korean syllable 
# and Latin9 (iso8859-15) characters.
pdf.add_font('eunjin', '', 'Eunjin.ttf', uni=True)
pdf.set_font('eunjin', '', 14)
pdf.write(8, u'Korean: 안녕하세요')
</code></pre>
