<div class="post-text" itemprop="text">
<p>Python 3 <a href="http://docs.python.org/2/library/urllib2.html" rel="noreferrer">refactored <code>urllib</code> and <code>urllib2</code></a> into a new package called <a href="http://docs.python.org/3/library/urllib.html" rel="noreferrer"><code>urllib</code> with submodules</a>.</p>
<p>Use <a href="http://docs.python.org/3/library/urllib.request.html" rel="noreferrer"><code>urllib.request</code></a> and <a href="http://docs.python.org/3/library/urllib.error.html#module-urllib.error" rel="noreferrer"><code>urllib.error</code></a> instead.</p>
</div>
<div class="post-text" itemprop="text">
<p>urllib2 now is called urllib, you can see it here <a href="http://www.python.org/dev/peps/pep-3108/#urllib-package" rel="nofollow">http://www.python.org/dev/peps/pep-3108/#urllib-package</a></p>
</div>
<div class="post-text" itemprop="text">
<p>For Python 3 use <code>urllib</code> like</p>
<pre><code>urllib.request
urllib.error
urllib.parse
urllib.robotparser
</code></pre>
