<div class="post-text" itemprop="text">
<p>By default, <code>print()</code> will end with a newline.  If you want to suppress this behavior specify <code>end</code>.</p>
<pre><code>print("\t", end="")
</code></pre>
<p>Documentation is here.  <a href="http://docs.python.org/3.3/library/functions.html#print">http://docs.python.org/3.3/library/functions.html#print</a></p>
