<p>At a high level, I would start with initializing a <code>datetime</code> object with the first timestamp in your range, and a 1-second <code>timedelta</code> object.</p>
<pre><code>import datetime
myDatetime = datetime.datetime(# TODO)
myTimedelta = datetime.timedelta(# TODO)
</code></pre>
<p>Then you could use a while loop and an <code>endDatetime</code> object with your last time in the range:</p>
<pre><code>endDateTime = datetime.datetime(# TODO)
while myDatetime &lt;= endDatetime:
    print(myDatetime) # need to format it to look like your example
    myDatetime = myDatetime + myTimedelta
</code></pre>
