<div class="post-text" itemprop="text">
<p>I have a folder of several images, and I would like to choose one randomly and add it to a subplot (using offsetbox for example).</p>
<p>How can I do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>random.choice(imgs)</code> where <code>imgs</code> is a list of images in the folder.</p>
<p>First, just specify the absolute path of the folder name:</p>
<pre><code>&gt;&gt;&gt; folder = '/Users/path/to/images/'
</code></pre>
<p>Then, you can get the images with <a href="https://docs.python.org/3/library/os.html#os.listdir" rel="nofollow noreferrer"><code>os.listdir()</code></a>:</p>
<pre><code>&gt;&gt;&gt; imgs = os.listdir(folder)
&gt;&gt;&gt; imgs
['img1.png', 'img2.jpg', 'otherimage.jpg']
</code></pre>
<p>Lastly, pick one of these at random and append it to the folder's path:</p>
<pre><code>&gt;&gt;&gt; img = os.path.join(folder, random.choice(imgs))
&gt;&gt;&gt; img
'/Users/path/to/images/img1.png'
</code></pre>
</div>
