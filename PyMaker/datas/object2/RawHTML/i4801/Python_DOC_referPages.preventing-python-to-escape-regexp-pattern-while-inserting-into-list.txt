<div class="post-text" itemprop="text">
<p>From the <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">docs</a>:</p>
<blockquote>
<p>re.match(pattern, string, flags=0)</p>
<p>If zero or more characters <strong>at the beginning of string</strong> match the regular expression pattern, return a corresponding match object. </p>
</blockquote>
<p>None of your patterns can be found at the beginning of <code>msg</code>, so it returns <code>None</code>.</p>
<p>If instead you use <code>re.search</code>, it will find the part of the string I assume you're looking for:</p>
<pre><code>&gt;&gt;&gt; re.search(combined, msg)
&lt;_sre.SRE_Match object; span=(54, 69), match='te3932-213 0 @4'&gt;
</code></pre>
