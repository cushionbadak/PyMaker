<p>CMD:</p>
<pre><code>    ln -s /users/me/link_file hello

<div class="post-text" itemprop="text">
<p><code>os.symlink()</code> is the provided function for creating symlinks. They are created as such:</p>
<pre><code>import os
os.symlink(src,dst)
</code></pre>
<p>Where <code>src</code> and <code>dst</code> are the paths.</p>
<p>See the documentation <a href="https://docs.python.org/3/library/os.html#os.symlink" rel="nofollow noreferrer">here</a> for more info.</p>
</div>
<span class="comment-copy">If you actually want to "perform the Linux command", see the <code>subprocess</code> module. If you want to create a symlink in the same way the Linux command does, see the <code>os</code> or <code>pathlib</code> module.</span>
