<p>Say I have the following regular expression and searches</p>
<pre><code>e = r"(?P&lt;int&gt;\d+)|(?P&lt;alpha_num&gt;\w)"
num = re.search(e, "123z")
letter = re.search(e, "z123")
</code></pre>
<p>I know that <code>num.group("int")</code> gives <code>123</code> and <code>num.group("alpha_num")</code> gives <code>None</code>.
Similarly <code>letter.group("int")</code> give <code>None</code> and <code>letter.group("alpha_num")</code> give <code>z</code></p>
<p>However, what I want is a way to get the "named category" for an arbitrary match.
So if I have an arbitrary match, say new_match, I could call <code>new_match.named_category()</code> and it would return "int" or "alpha_num" depending on how it matched.  </p>
<p>Does any such command exist or do I have to create my own?</p>
<p>Thanks!</p>

<div class="post-text" itemprop="text">
<p>For the specific example in your question, you can use <a href="https://docs.python.org/3/library/re.html#re.match.lastgroup" rel="nofollow noreferrer">lastgroup</a>:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; e = r"(?P&lt;int&gt;\d+)|(?P&lt;alpha_num&gt;\w)"
&gt;&gt;&gt; num = re.search(e, "123z")
&gt;&gt;&gt; letter = re.search(e, "z123")
&gt;&gt;&gt; num.lastgroup
'int'
&gt;&gt;&gt; letter.lastgroup
'alpha_num'
</code></pre>
