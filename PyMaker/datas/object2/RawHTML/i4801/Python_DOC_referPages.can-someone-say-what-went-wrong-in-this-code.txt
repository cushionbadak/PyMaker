<div class="post-text" itemprop="text">
<pre><code>m = [[0]*c]*r
</code></pre>
<p>This creates a list of <code>r</code> references to the same inner list. So, whenever you modify <code>m[0]</code>, you're also modifying <code>m[1]</code>, and so on, because they're the same list.</p>
<p>You probably wanted something like this:</p>
<pre><code>m = [[0 for _ in range(c)] for _ in range(r)]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>[0]*4</code> gives you four copies of the <em>same</em> object, not four independent lists.</p>
<p>Try </p>
<pre><code>m = [[0 for i in range(c)] for j in range(r)]
</code></pre>
