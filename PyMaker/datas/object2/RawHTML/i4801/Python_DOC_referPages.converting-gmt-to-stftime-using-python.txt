<p>I want to convert <code>Wed, 14 Mar 2018 07:30:00 GMT</code> to <strong>2018/03/14</strong> in python. I am getting <code>unicode object has no attribute strftime</code> error. Someone help me.</p>

<div class="post-text" itemprop="text">
<p>Use:</p>
<pre><code>from datetime import datetime
date_given = 'Wed, 14 Mar 2018 07:30:00 GMT'
datetime.strptime(date_given, '%a, %d %b %Y %H:%M:%S %Z').strftime('%Y/%m/%d')
</code></pre>
<p>You can read up below links for more details:</p>
<ul>
