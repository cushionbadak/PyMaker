<p>I have a columns string column like this:</p>
<pre><code>A
'(1+2)'
'1*(1/2)'
'5*2'
</code></pre>
<p>How is it possible to solve these calculation as a float or integer?</p>
<pre><code>calc_df['A'] = calc_df['A'].astype('Float64')
</code></pre>
<blockquote>
<p>ValueError: could not convert string to float: '(1+2)'</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>This will do it.</p>
<pre><code>import pandas as pd
A = ['(1+2)','1*(1/2)','5*2']
df = pd.DataFrame(A, columns=['A'])
pd.eval(df.A)

Result:  [3, 0.5, 10]
</code></pre>
</div>
<span class="comment-copy">You could remove the quotes and use <a href="https://docs.python.org/3/library/functions.html#eval" rel="nofollow noreferrer">eval()</a>.</span>
