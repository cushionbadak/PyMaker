<p>you could try this using <a href="https://docs.python.org/3/library/itertools.html?itertools.chain.from_iterable#itertools.chain.from_iterable" rel="nofollow noreferrer"><code>itertools.chain.from_iterable</code></a></p>
<pre><code>from itertools import chain

list1 = [2, 4, 6]
list2 = [3, 4, 2]

ret = list(chain.from_iterable(n * [item] for item, n in zip(list1, list2)))
print(ret)  # [2, 2, 2, 4, 4, 4, 4, 6, 6]
</code></pre>
<p><code>n * [item]</code> will be <code>3 * [2] = [2, 2, 2]</code> in the first iteration; all i do then is chain those elements to a list using <code>itertools.chain.from_iterable</code>.</p>
<p>or, using <a href="https://docs.python.org/3/library/itertools.html?highlight=itertools#itertools.repeat" rel="nofollow noreferrer"><code>itertools.repeat</code></a>:</p>
<pre><code>from itertools import chain, repeat

ret = list(chain.from_iterable(repeat(item, n) for item, n in zip(list1, list2)))
print(ret)  # [2, 2, 2, 4, 4, 4, 4, 6, 6]
</code></pre>
