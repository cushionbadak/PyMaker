<div class="post-text" itemprop="text">
<p><code>len(b)</code> produces an <code>int</code>. <a href="https://docs.python.org/3/reference/compound_stmts.html#for" rel="nofollow noreferrer">You can't iterate over an int.</a>. To generate a sequence of <code>int</code>s upto <code>len(b)</code> use <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer"><code>range</code>.</a></p>
<pre><code>def add (A,B):
    for i in range(len(B)):
        for j in range(len(A)):
            A[j][i] += B[i]
    return A

A = [[2, 8], [3, 7], [4, 5]]
B = [1, 2]
A = add(A,B)
print(C)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need an iterable object to iterate over in the <a href="https://docs.python.org/3/reference/compound_stmts.html#the-for-statement" rel="nofollow noreferrer"><code>for</code></a> loop. <code>len()</code> returns an <code>int</code> which is not iterable.</p>
<p>See <a href="https://stackoverflow.com/questions/9884132/what-exactly-are-pythons-iterator-iterable-and-iteration-protocols">this</a> post.</p>
<p>Try using <a href="https://docs.python.org/3/tutorial/controlflow.html#the-range-function" rel="nofollow noreferrer"><code>range()</code></a> instead like</p>
<pre><code>for i in range(len(B)):
    for j in range(len(A)):
        A[j][i] += B[i]
</code></pre>
</div>
