<p>With the new formatting syntax, Python can easily be made to print a thousands separator:</p>
<pre><code>print ("{0:,.2f} Euro".format(myVariable))
</code></pre>
<p>So the comma takes care for this. </p>
<p>How can this be done with the old syntax - without using locale? </p>
<pre><code>print ("%.2f Euro" % myVariable)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html?highlight=printf#printf-style-string-formatting" rel="nofollow noreferrer"><code>printf</code>-style formatting</a> does not support thousand separator.</p>
<p>You need to build a string with thousand separator:</p>
<pre><code>&gt;&gt;&gt; amount = 12345
&gt;&gt;&gt; '%s Euro' % format(amount, ',.2f')
'12,345.00 Euro'
</code></pre>
<p>Built-in function <a href="https://docs.python.org/3/library/functions.html#format" rel="nofollow noreferrer"><code>format</code></a> uses the same <a href="https://docs.python.org/3/library/string.html#formatspec" rel="nofollow noreferrer">Format Specification Mini-Language</a> that <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>str.format</code></a> uses.</p>
<p>If you can't use <code>format</code>, see <a href="https://stackoverflow.com/questions/1823058/how-to-print-number-with-commas-as-thousands-separators">this question</a> for alternatives.</p>
</div>
