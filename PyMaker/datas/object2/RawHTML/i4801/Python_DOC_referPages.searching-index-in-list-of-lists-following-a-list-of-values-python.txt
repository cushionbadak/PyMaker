<p>Please take time to understand <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer">enumerate</a> and list data structure in python.
for your case this code is working. Please understand it.</p>
<pre><code>list_result = []
for lst in xs:
    result_list = []
    for i, item in enumerate(lst):
        for s in ss:
            if s == item:
                result_list.append(i)
                break
    list_result.append(result_list)
print(list_result)
</code></pre>

<pre><code>list_result = []
for sub_list in xs:
    list_result.append([i for i, item in enumerate(sub_list) if item in ss])

print(list_result)
