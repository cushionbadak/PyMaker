<p>How about using <code>dateutil</code>? I think that there may be other methods. So please think of this as one of them.</p>
<h3>Modified script :</h3>
<pre><code>import dateutil.parser

for event in events:
        start = event['start'].get('dateTime', event['start'].get('date'))
        parsedDate = dateutil.parser.parse(start)
        print(parsedDate, event['summary'])
</code></pre>
