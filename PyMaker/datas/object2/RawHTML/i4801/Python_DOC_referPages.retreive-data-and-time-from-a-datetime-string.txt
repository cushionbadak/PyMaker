<div class="post-text" itemprop="text">
<p>You need to match your string exactly. Reference: <a href="https://docs.python.org/2/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer">strftime-and-strptime-behavior</a></p>
<pre><code>from datetime import datetime
dt = datetime.strptime('2018-03-07 06:46:02.737951', '%Y-%m-%d %H:%M:%S.%f')

print(dt.date())
print(dt.time())


d = dt.date() # extract date
t = dt.time() # extract time

print(type(d)) # printout the types
print(type(t))
</code></pre>
<p>Output:</p>
<pre><code>2018-03-07
06:46:02.737951

&lt;class 'datetime.date'&gt;
&lt;class 'datetime.time'&gt;
</code></pre>
<p>Your format string is something along the lines of:</p>
<pre><code>Month as locale’s abbreviated name.  
Day of the month as a zero-padded decimal number.
Year with century as a decimal number.
Hour (12-hour clock) as a zero-padded decimal number.
Minute as a zero-padded decimal number.
Locale’s equivalent of either AM or PM.
</code></pre>
<p>with some spaces and <code>:</code> in it - which does not match your format.</p>
</div>
