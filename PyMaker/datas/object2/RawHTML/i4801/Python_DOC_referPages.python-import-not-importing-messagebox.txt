<div class="post-text" itemprop="text">
<pre><code>from tkinter import *
from tkinter import messagebox

root = Tk()
messagebox.showinfo("Hello world", "you are the best")
root.mainloop()
</code></pre>
<p>Why do I have to have to import messagebox explicitly when I am import all using * </p>
</div>
<div class="post-text" itemprop="text">
<p><code>messagebox</code> is a submodule of the <code>tkinter</code> package.</p>
<p>The wildcard import syntax doesn't import submodules, only the names that are defined in the <code>tkinter</code> package itself.</p>
<p>Therefore you need to import the <code>messagebox</code> submodule explicitly.</p>
