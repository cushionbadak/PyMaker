<p>This is my question:
I have to write a function that accepts 2 integer values as parameters and returns the sum of all the integers between the two values, including the first and last values. The parameters may be in any order (i.e. the second parameter may be smaller than the first).</p>
<p>This is the example of the result:</p>
<p><img alt="screenshot" src="https://i.stack.imgur.com/vvxSN.png"/></p>
<p>This is what I've tried:</p>
<pre><code>def sum_range(int1,int2):
    count = 0
    for i in range(int1,int2+1):
         count = count + i
    return count
</code></pre>
<p>but for this example:</p>
<pre><code>result = sum_range(3, 2)

print(result)
</code></pre>
<p>I got the wrong result, can anyone help?</p>

<div class="post-text" itemprop="text">
<p>You'll need to swap your variables if <code>int2</code> is smaller than <code>int1</code>:</p>
<pre><code>def sum_range(int1, int2):
    if int1 &gt; int2:
        int2, int1 = int1, int2
    count = 0
    for i in range(int1, int2 + 1):
         count = count + i
    return count
</code></pre>
<p>You don't really need to use a loop here, if you passed your <code>range()</code> to the <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer"><code>sum()</code> function</a>, then you can leave the looping and summing to that function:</p>
<pre><code>def sum_range(int1, int2):
    if int1 &gt; int2:
        int2, int1 = int1, int2
    return sum(range(int1, int2 + 1))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using loops:</p>
<pre><code>def sum_range(num1, num2):
    sum = 0
    for i in range(min(num1, num2), max(num1, num2) + 1):
        sum += i
    return sum
</code></pre>

<pre><code>def sum_range(num1, num2):

## This works as you're essentially asking for the difference between two triangular numbers
</code></pre>

<div class="post-text" itemprop="text">
<p><code>range</code> counts from the first parameter <em>up to</em> (but not including) the second, which means the range is <em>empty</em> if the first is no smaller than the second.</p>
</div>
<div class="post-text" itemprop="text">
<p>You don't really need to use the loop:</p>
<pre><code>def sum_range(int1, int2):
    if int1 &gt; int2:
        int2, int1 = int1, int2
    return sum(list(range(int1,int2+1)))
</code></pre>

<pre><code>result = sum_range(4, 2)
print(result)
</code></pre>

<pre><code>9
