<div class="post-text" itemprop="text">
<p>You can try:</p>
<pre><code>for i, (name, num) in enumerate(teams, start=1):
    print("{}. {} {} Pts".format(i, name, num))  
</code></pre>
<p>enumerate returns the index (1-3 in this case) and each value of the iterable (teams in this case).  The values returned from the iterable will be tuples.  You can expand the tuple as shown above.</p>
</div>
<div class="post-text" itemprop="text">
<p>The <code>enumerate()</code> function adds a counter to an iterable so your <code>num</code> is still each item in your array (i.e. the tuple).</p>
<pre><code>teams = [
    ('Team1', '91'),
    ('Team2', '89'),
    ('Team3', '87')
]

for index, item in enumerate(teams, start=1):
    print("{}. {} {} Pts".format(index, item[0], item[1]))  
</code></pre>
