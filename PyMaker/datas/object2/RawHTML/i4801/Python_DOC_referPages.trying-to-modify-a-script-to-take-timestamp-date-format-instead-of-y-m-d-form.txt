<div class="post-text" itemprop="text">
<p>Just use <a href="https://docs.python.org/3/library/datetime.html?highlight=re#datetime.datetime.fromtimestamp" rel="nofollow noreferrer"><code>datetime.datetime.fromtimestamp</code></a>:</p>
<pre><code>from datetime import datetime

datetime.fromtimestamp(1504159200)  # datetime.datetime(2017, 8, 31, 2, 0)
datetime.fromtimestamp(1519974000)  # datetime.datetime(2018, 3, 2, 2, 0)
</code></pre>
