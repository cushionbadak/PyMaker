<div class="post-text" itemprop="text">
<p>Ok, I know that if you want to open a particular url using pyton you would run:</p>
<pre><code>import webbrowser
webbrowser.open("buinvent.com")
</code></pre>
<p>But what if you want to change it to a different url in the web browser without opening a new window or a new tab in the web browser. Is there a way to do that?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>import webbrowser
webbrowser.open('google.com', new = 0)
</code></pre>
<p>And the docs says:</p>
<blockquote>
<p>If new is 0, the url is opened in the same browser window if possible.
  If new is 1, a new browser window is opened if possible. If new is 2,
  a new browser page (“tab”) is opened if possible</p>
</blockquote>
<p><a href="https://docs.python.org/3/library/webbrowser.html" rel="nofollow noreferrer">https://docs.python.org/3/library/webbrowser.html</a></p>
</div>
