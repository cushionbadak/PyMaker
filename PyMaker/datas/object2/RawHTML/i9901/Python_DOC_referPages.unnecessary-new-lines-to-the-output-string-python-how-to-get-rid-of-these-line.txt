<div class="post-text" itemprop="text">
<p>In Python 3, you can use the <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>end</code></a> keyword argument to suppress newlines:</p>
<pre><code>print ('create:' + str(user) + ':' + str(passwd) + ':::' + str(user) 
       + ':/home/' + str(user) + ':/bin/bash:::::', end="")
</code></pre>
<p>By the way, you could simply use:</p>
<pre><code>user, passwd = line.split(separator)
</code></pre>
<p>instead of using an intermediate list variable (ie. <code>sout</code>).</p>
</div>
<div class="post-text" itemprop="text">
<p>You need to strip the line to get rid of the new line. The readline API reads the new line. This new line later ends up being part of the passwd variable.</p>
<pre><code>while line:
    line = line.strip()
</code></pre>
<p>From Python documentation: <a href="https://docs.python.org/2/tutorial/inputoutput.html" rel="nofollow noreferrer">https://docs.python.org/2/tutorial/inputoutput.html</a></p>
<blockquote>
<p>f.readline() reads a single line from the file; a newline character
  (\n) is left at the end of the string, and is only omitted on the last
  line of the file if the file doesnâ€™t end in a newline.</p>
</blockquote>
</div>
