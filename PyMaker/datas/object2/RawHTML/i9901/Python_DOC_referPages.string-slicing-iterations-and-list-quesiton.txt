<p>Say I have the string "BigJon".</p>
<p>Is there way to iterate through and slice it into two different words like 
<code>B igJon</code>, <code>Bi gJon</code>, <code>Big Jo</code>n, <code>Bigj on</code> and so on. And then all these separate pieces be a list?</p>
</div>
<div class="post-text" itemprop="text">
<p>One liner list comprehension</p>
<p><code>['%s %s' % (a[:i], a[i:]) for i in range(1, len(a))]</code></p>
</div>
<div class="post-text" itemprop="text">
<pre><code>s = 'Bigjohn'
for i in range(0,len(s)):
    b = s[:i]+" "+s[i:]
    print b
</code></pre>
<p>output:</p>
<pre><code> Bigjohn
B igjohn
Bi gjohn
Big john
Bigj ohn
Bigjo hn
Bigjoh n
</code></pre>
</div>
