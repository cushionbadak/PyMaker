<div class="post-text" itemprop="text">
<p>Qualify instance attributes with <code>self.</code>, not <code>self_</code>:</p>
<pre><code>class Circle(object):
    def __init__(self, radius = 1):
        self.radius = radius

    def __str__(self):
        return "Circle with radius {}".format(self.radius)
</code></pre>
<p>If you name the variable <code>self_radius</code>, it become local variable; not accessible from another method.</p>
