<div class="post-text" itemprop="text">
<p>If you're using Python 3.3+, you can use <a href="https://docs.python.org/3/library/types.html#types.SimpleNamespace" rel="nofollow noreferrer"><code>types.SimpleNamespace</code></a>:</p>
<pre><code>&gt;&gt;&gt; import types
&gt;&gt;&gt; types.SimpleNamespace(a=0, b=0, c=0)
namespace(a=0, b=0, c=0)

&gt;&gt;&gt; obj = types.SimpleNamespace(a=0, b=0, c=0)
&gt;&gt;&gt; obj.a
0
&gt;&gt;&gt; obj.b
0
&gt;&gt;&gt; obj.c
0
</code></pre>
<hr/>
<p>In lower version, use <a href="https://docs.python.org/2/library/collections.html#collections.namedtuple" rel="nofollow noreferrer"><code>collections.namedtuple</code></a> to make a custom type.</p>
<pre><code>&gt;&gt;&gt; from collections import namedtuple
&gt;&gt;&gt;
&gt;&gt;&gt; abc = namedtuple('abc', 'a b c')
&gt;&gt;&gt; abc(a=0, b=0, c=0)
abc(a=0, b=0, c=0)
</code></pre>
<p>But it does not allow attribute setting unlike <code>types.SimpleNamespace</code>:</p>
<pre><code>&gt;&gt;&gt; obj = abc(a=0, b=0, c=0)
&gt;&gt;&gt; obj.a
0
&gt;&gt;&gt; obj.a = 1
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
AttributeError: can't set attribute
</code></pre>
