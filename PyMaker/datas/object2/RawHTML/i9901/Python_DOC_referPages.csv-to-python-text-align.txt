<h1>Using <a href="https://www.tutorialspoint.com/python/string_ljust.htm" rel="nofollow noreferrer"><code>ljust()</code></a></h1>
<p>You can use <a href="https://www.tutorialspoint.com/python/string_ljust.htm" rel="nofollow noreferrer"><code>ljust()</code></a> function (for strings).</p>
<p><code>ljust</code> will pad the string from the right with spaces (or any other char if specified).
<br/>Example usage:</p>
<pre><code>&gt;&gt;&gt; 'hello'.ljust(10)
'hello     '
&gt;&gt;&gt; 'hello'.ljust(10, 'A')
'helloAAAAA'
</code></pre>
<p>So when you print the values, just pad them using <code>ljust</code></p>
<p><strong>Assume having <code>a</code> - list a of lists that contains your values:</strong></p>
<pre><code>for b in a:
    for c in b:
        print c.ljust(15),
    print ""
</code></pre>
<h1>Using <a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="nofollow noreferrer"><code>format</code></a></h1>
<p>You may use the string <code>format</code> function for that purpose:</p>
<pre><code>&gt;&gt;&gt; '{:10}'.format('hello')
'hello     '
</code></pre>
<p><strong>The general format of a standard format specifier is:</strong></p>
<pre><code>format_spec ::=  [[fill]align][sign][#][0][width][,][.precision][type]
fill        ::=  &lt;any character&gt;
align       ::=  "&lt;" | "&gt;" | "=" | "^"
sign        ::=  "+" | "-" | " "
width       ::=  integer
precision   ::=  integer
type        ::=  "b" | "c" | "d" | "e" | "E" | "f" | "F" | "g" | "G" | "n" | "o" | "s" | "x" | "X" | "%"
</code></pre>
<p>Another way to use the string <a href="https://docs.python.org/2/library/string.html#string-formatting" rel="nofollow noreferrer"><code>format</code></a>:</p>
<pre><code>&gt;&gt;&gt; '{text: &lt;10}'.format(text='hi')
'hi        '
</code></pre>
