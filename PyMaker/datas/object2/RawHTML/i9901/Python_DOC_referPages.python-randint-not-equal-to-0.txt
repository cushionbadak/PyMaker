<p>I am trying to use <code>random.randint</code> to randomly generate a single <code>int</code> between a range, e.g. <code>randint(-2,2)</code>, however, it could possibly generate <code>0</code>, which is not expected, so I am wondering is there a way to make it not equal to <code>0</code>. Or just use a loop until <code>randint(-2,2)</code> not generate <code>0</code>, and get the output.</p>

<div class="post-text" itemprop="text">
<p>Don't use <code>randint()</code> then; it picks an integer from a continuous range of values, not a discrete set.</p>
<p>Use the <a href="https://docs.python.org/3/library/random.html#random.choice" rel="nofollow noreferrer"><code>random.choice()</code> function</a> instead, passing in a list of <em>just</em> those integers you want to pick from:</p>
<pre><code>random.choice([-2, -1, 1, 2])
</code></pre>
<p>This'll return a random value from that list.</p>
