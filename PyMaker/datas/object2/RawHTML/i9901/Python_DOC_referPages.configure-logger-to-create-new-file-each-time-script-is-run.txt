<p>I'm not using dictConfig but I do find success using this setup.</p>
<pre><code>    logdatetime = time.strftime("%m%d")
    logging.basicConfig(format='%(asctime)s:%(levelname)s:%(message)s',
                        filename='../chown_parser_' + logdatetime + '.log',
                        datefmt='%m/%d/%Y %I:%M:%S %p',
                        level=logging.INFO)
</code></pre>
