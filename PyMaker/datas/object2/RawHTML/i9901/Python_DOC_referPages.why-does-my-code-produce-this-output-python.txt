<p>You're printing inside the <code>for</code> loop. <code>data_list</code> is a list of lists, just print it after the <code>for</code> loop finishes :) </p>
<pre><code>data_list = []

data_file = open('table.csv', 'r')

for line in data_file:
    data_list.append(line.strip().split(','))
print(data_list)
</code></pre>
