<p>I am trying to iterate through a list of tuples like this:</p>
<pre><code>[(0, 0.0083553666846612491), (1, 0.0054999681509501821)]
</code></pre>
<p>and find the smallest value on the right hand side of the tuple and then save the accompanying left hand side value to a variable.</p>
<p>What would the best way to go about this problem be?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>key</code> argument to the <a href="https://docs.python.org/3/library/functions.html?highlight=built%20functions#min" rel="nofollow noreferrer"><code>min()</code></a> function to find the tuple with the smallest second value:</p>
<pre><code>&gt;&gt;&gt; l = [(0, 0.0083553666846612491), (1, 0.0054999681509501821)]
&gt;&gt;&gt; smallest = min(l, key=lambda x: x[1])[0]
&gt;&gt;&gt; smallest
1
</code></pre>
<p>Here a lambda expression is used to select the second item of each tuple and <code>min</code> uses that value in its comparisons. <code>min()</code> returns the full tuple, so you can extract the first item using <code>[0]</code>.</p>
<p>You can also use <a href="https://docs.python.org/3/library/operator.html#operator.itemgetter" rel="nofollow noreferrer"><code>operator.itemgetter</code></a>:</p>
<pre><code>&gt;&gt;&gt; from operator import itemgetter
&gt;&gt;&gt; smallest = min(l, key=itemgetter(1))[0]
&gt;&gt;&gt; smallest
1
</code></pre>
</div>
