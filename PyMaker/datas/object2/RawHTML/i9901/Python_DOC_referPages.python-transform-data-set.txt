<div class="post-text" itemprop="text">
<p>Perform <code>groupby</code> w.r.t <em>id</em> column. Iterate over each group by converting the other two columns to a <code>list</code> and add the unique value corresponding to the <em>id</em> per group number alongside. Finally, convert them into a <code>tuple</code> and append these to a list.</p>
<pre><code>grouped = df.groupby('id')
L = []
for _, grp in grouped:
    L.append(tuple(grouped.get_group(_)['id'].unique().tolist() + grp[['type','value']].values.ravel().tolist()))
print(L)
#[(1, 'A', 10, 'C', 120), (2, 'B', 20, 'C', 40), (3, 'A', 10, 'B', 50)]
</code></pre>
