<p>I want to create a dictionary in Python where the keys are lists.</p>
<p>The issue I want to face is that I have a list of multiple integer values (from 1 to 300), and depending different value-ranges, I want to convert those into characters. For example:</p>
<ul>
<li>Values from 1 to 100 into character 'A'.</li>
<li>Values from 101 to 200 into character 'B'.</li>
<li>Values from 201 to 300 into character 'C'.</li>
</ul>
<p>I tried this way but it didn't work:</p>
<pre><code>dictionary = {[1, 100]:'A', [101, 200]:'B', [201, 300]:'C'}
</code></pre>
<p>But I get the error:</p>
<pre><code>TypeError: unhashable type: 'list'
</code></pre>
<p>How could I do it?</p>

<div class="post-text" itemprop="text">
<p>Convert the lists to tuples. Tuples are hashable. Lists are not.</p>
<pre><code>dictionary = {(1, 100):'A', (101, 200):'B', (201, 300):'C'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a tuple, a <a href="https://docs.python.org/2/library/json.html" rel="nofollow noreferrer">JSON</a> representation of the list, or a <a href="https://docs.python.org/3/library/functions.html#repr" rel="nofollow noreferrer">repr</a> of the list:</p>
<pre><code>&gt;&gt;&gt; d1={repr([1, 100]):'A', repr([101, 200]):'B', repr([201, 300]):'C'}
&gt;&gt;&gt; import json
&gt;&gt;&gt; d2={json.dumps([1, 100]):'A', json.dumps([101, 200]):'B', json.dumps([201, 300]):'C'}
&gt;&gt;&gt; d2
{'[101, 200]': 'B', '[1, 100]': 'A', '[201, 300]': 'C'}
&gt;&gt;&gt; d1
{'[101, 200]': 'B', '[1, 100]': 'A', '[201, 300]': 'C'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I don't know if I understood you correctly but you try to convert a list of ints in range [1, 300] to a list of characters with the characters A,B,C. Depending on the integer value. I don't see why you need a dict with keys of lists...</p>
<p>You can do it with a simple for loop and a few if statements. </p>
<pre><code>import random

ints = [random.randint(1,300) for i in xrange(10)]

chars = []

for el in ints:
    if el &gt; 0 and el &lt;= 300:
        if el &lt;= 100:
            chars.append("A")
        elif el &lt;= 200:
            chars.append("B")
        else:
            chars.append("C")
    else:
        raise RuntimeError("element {} not in valid range.".format(el))

print(ints)
print(chars)
</code></pre>
<p>which returns </p>
<pre><code>[23, 260, 276, 31, 170, 42, 16, 125, 102, 39]
['A', 'C', 'C', 'A', 'B', 'A', 'A', 'B', 'B', 'A']
</code></pre>
