<p>As described in <a href="https://stackoverflow.com/questions/5676646/how-can-i-fill-out-a-python-string-with-spaces">How can I fill out a Python string with spaces?</a> Python has some functions for this. Have a look at:</p>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.ljust" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html#str.ljust</a></p>
<blockquote>
<p>Return the string left justified in a string of length width. Padding is done using the specified fillchar (default is an ASCII space). The original string is returned if width is less than or equal to len(s).</p>
</blockquote>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.rjust" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html#str.rjust</a></p>
<blockquote>
<p>Return the string right justified in a string of length width. Padding is done using the specified fillchar (default is an ASCII space). The original string is returned if width is less than or equal to len(s).</p>
</blockquote>

<p>This is what I've got. It's working well.</p>
<pre><code>def formattage(text):

    text = text.rsplit("$")
    a = '{:{align}{width}}'.format(text[0], align='&lt;', width='35')
    b = '{:{align}{width}}'.format(text[1], align='&lt;', width='29')
    c = '{:{align}{width}}'.format(text[2], align='&lt;', width='83')
    d = '{:{align}{width}}'.format(text[3], align='&lt;', width='33')
    text = a+b+c+d
    print text
</code></pre>
