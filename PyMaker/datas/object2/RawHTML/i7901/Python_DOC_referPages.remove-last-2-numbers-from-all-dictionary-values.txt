<p>Something like:</p>
<pre><code>cv_name = ' '.join(cv_ver['name'].split(' ')[:-1])
</code></pre>
<p>should do it...</p>
<p>Explanation:</p>
<p>First we split our string by space character - <code>cv_ver['name'].split(' ')</code><br/>
return all elements except the last one - <code>[:-1]</code><br/>
and finally, join those list elements back with space character between them - <code>' '.join()</code></p>
<p>As <a href="https://stackoverflow.com/questions/44309446/remove-last-2-numbers-from-all-dictionary-values#comment75624914_44309610">MSeifert pointed out</a>, the alternative can be version with <a href="https://docs.python.org/3/library/stdtypes.html#str.rsplit" rel="nofollow noreferrer"><code>rsplit()</code></a>:</p>
<pre><code>cv_name = cv_ver['name'].rsplit(' ', 1)[0]
</code></pre>
</div>
