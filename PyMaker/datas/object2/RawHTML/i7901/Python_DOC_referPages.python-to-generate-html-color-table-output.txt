<div class="post-text" itemprop="text">
<p>The API documentation of <a href="https://www.decalage.info/python/html" rel="nofollow noreferrer">HTML.py</a> states that tables are created like this: <code>t = HTML.Table(header_row=['Test', 'Result'])</code>
replace HTML.table with <strong>HTML.Table</strong> in your code (like below)</p>
<pre><code> import HTML
 import sys
 import os
 table_data = [
        ['Last name',   'First name',   'Age'],
        ['Smith',       'John',         30],
        ['Carpenter',   'Jack',         47],
        ['Johnson',     'Paul',         62],
    ]
htmlcode = HTML.Table(table_data)

f = open("file123.html", 'w')
sys.stdout = f

f.close()
</code></pre>
<p>make sure to also install the module correctly as described <a href="https://github.com/decalage2/pyhtgen" rel="nofollow noreferrer">here</a>: </p>
<p>INSTALLATION:</p>
<ul>
<li>on Windows, double-click on install.bat, or type "setup.py install" in a CMD
window.</li>
<li>on other systems, type "python setup.py install" in a shell.</li>
</ul>
