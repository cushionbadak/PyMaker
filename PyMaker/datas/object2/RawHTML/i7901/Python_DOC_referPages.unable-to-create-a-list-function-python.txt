<p>Write a function, top_three, that takes a list as its argument, and returns a list of the three largest elements. For example, <code>top_three([2,3,5,6,8,4,2,1]) == [8, 6, 5]</code>. My attempt:</p>
<pre><code>def top_three(input_list):

sorted_list= sorted([input_list], reverse=True)

top_three = sorted_list[ :2]

return top_three
</code></pre>

<div class="post-text" itemprop="text">
<p>Don't wrap your input in another layer of <code>list</code>.</p>
<pre><code>sorted_list= sorted([input_list], reverse=True)
</code></pre>
<p>should be:</p>
<pre><code>sorted_list = sorted(input_list, reverse=True)
</code></pre>
<p>or it's just sorting a one-item <code>list</code> (containing the whole of the <code>list</code> provided).</p>
<p>Side-note: For real world code, just <a href="https://docs.python.org/3/library/heapq.html#heapq.nlargest" rel="nofollow noreferrer">use <code>heapq.nlargest</code></a>, which scales to large inputs and small numbers of items to select better than a global sort and slice:</p>
<pre><code>import heapq

def top_three(input_list):
    return heapq.nlargest(3, input_list)
</code></pre>
