<p><strong>Try This.. For Python: 3.6</strong></p>
<blockquote>
<p>From pip install Smtpd</p>
</blockquote>
<pre><code>pip install smtpd-tls
</code></pre>
<blockquote>
<p>install the latest version of a module and its dependencies from the
  Python Packaging</p>
</blockquote>
<pre><code>python -m pip install SomePackage
</code></pre>
<blockquote>
<p>For Specific Version and Minimum Version Install</p>
</blockquote>
<pre><code>python -m pip install SomePackage==3.6.1    # specific version
python -m pip install "SomePackage&gt;=1.0.4"  # minimum version
</code></pre>
