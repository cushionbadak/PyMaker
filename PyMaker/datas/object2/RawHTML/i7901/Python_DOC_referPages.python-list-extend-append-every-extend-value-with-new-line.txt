<div class="post-text" itemprop="text">
<p>If you wanted separate lists, make them separate. Don't use <code>list.extend()</code>, use <em>appending</em>:</p>
<pre><code>list1 = [['Para','op','qa', 'reason']]  # brackets, creating a list with a list
list1.append(['Power','pass','ok', 'NA'])
</code></pre>
<p>Now <code>list1</code> is a list with two objects, each itself a list:</p>
<pre><code>&gt;&gt;&gt; list1
[['Para', 'op', 'qa', 'reason'], ['Power', 'pass', 'ok', 'NA']]
</code></pre>
<p>If you are using the <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer"><code>csv</code> module</a> to write out your CSV file, use the <a href="https://docs.python.org/3/library/csv.html#csv.csvwriter.writerows" rel="nofollow noreferrer"><code>csvwriter.writerows()</code> method</a> to write each row into a separate line:</p>
<pre><code>&gt;&gt;&gt; import csv
&gt;&gt;&gt; import sys
&gt;&gt;&gt; writer = csv.writer(sys.stdout)
&gt;&gt;&gt; writer.writerows(list1)
Para,op,qa,reason
Power,pass,ok,NA
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your desired result, list1, should be a list of two elements, that each one of them is a list by itself.</p>
<pre><code>list1 = ['Para','op','qa', 'reason']
# wrapping list1 with [] crates a new list which its first element is the original list1.
# In your case, this action gives a list of lines with only one single line
# Only after that I can add a new list of lines that contains another single line
list1 = [list1] + [['Power','pass','ok', 'NA']]
print (list1)
</code></pre>
