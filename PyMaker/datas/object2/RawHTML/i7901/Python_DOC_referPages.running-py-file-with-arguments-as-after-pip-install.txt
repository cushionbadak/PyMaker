<div class="post-text" itemprop="text">
<p>You are probably running Python 2.</p>
<p>The <code>errors</code> parameter in <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer"><code>open()</code></a> is present in Python 3.</p>
<p>Try <code>python3 chapterize.py 10004.txt --nochapters</code></p>
