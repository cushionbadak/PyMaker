<div class="post-text" itemprop="text">
<p>You need <code>random.choice(CELLS)</code>, not <code>random.sample(CELLS,1)</code></p>
<p><a href="https://docs.python.org/3/library/random.html#random.choice" rel="nofollow noreferrer"><code>random.choice(seq)</code></a> returns one random element out <code>seq</code>:</p>
<pre><code>(1, 3)
</code></pre>
<p><a href="https://docs.python.org/3/library/random.html#random.sample" rel="nofollow noreferrer"><code>random.sample(seq, 1)</code></a> returns one sublist of <code>seq</code> with one random element:</p>
<pre><code>[(1, 3)]
</code></pre>
<p>With this small change, your program outputs :</p>
<pre><code> _ _ _ _ _
|_|_|_|_|_|
|_|_|_|_|_|
|_|_|_|_|_|
|_|X|_|_|_|
|_|_|_|_|_|
</code></pre>
