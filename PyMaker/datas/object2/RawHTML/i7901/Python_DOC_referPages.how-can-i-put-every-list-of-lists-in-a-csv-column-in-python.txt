<p>I'm developing a script in python that gets data from a log file and I need to save every type of data into a respective column. I'm using regex to obtain the data.</p>
<p>This is a part of my code where I get this result:</p>

<pre><code>#Getting data from log as list using regex
fecha = re.findall('\d{4}\-\d{2}\-\d{2}', str(listaValores))
hora = re.findall('\d{2}\:\d{2}\:\d{2}', str(listaValores))

#List of lists about data obtained
valoresFinales = [fecha, hora]

#Putting into .csv
with open("resultado.csv", "w") as f:
    wr = csv.writer(f, delimiter=';')
    wr.writerows(valoresFinales)
</code></pre>

<div class="post-text" itemprop="text">
<p>You give the <code>writerows</code> function a list of two elements, so you get two rows of data in the end.</p>
<p>Instead you want to give it something like <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip(fecha, hora)</code></a>:</p>
<pre><code>with open("resultado.csv", "w") as f:
    wr = csv.writer(f, delimiter=';')
    wr.writerows(zip(*valoresFinales))
</code></pre>
