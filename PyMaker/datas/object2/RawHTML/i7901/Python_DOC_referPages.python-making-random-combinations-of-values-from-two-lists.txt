<div class="post-text" itemprop="text">
<p>You can use the <code>random.choice()</code> function to select a random element from a list, so just use it on all 4 lists:</p>
<pre><code>from random import choice

combination = (choice(square), choice(circle), choice(line), choice(radiusshape))
</code></pre>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/random.html#random.sample" rel="nofollow noreferrer"><code>random.sample</code></a> to draw <code>k</code> random samples from your generated <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow noreferrer">cartesian product</a></p>
<pre><code># where k is number of samples to generate
samples = random.sample(itertools.product(square, circle, line, radiusshape), k)
</code></pre>
<p>For example</p>
<pre><code>&gt;&gt;&gt; a = [1, 2, 3, 4]
&gt;&gt;&gt; b = ['a', 'b', 'c', 'd']
&gt;&gt;&gt; c = ['foo', 'bar']
&gt;&gt;&gt; random.sample(set(itertools.product(a,b,c)), 5)
[(1, 'c', 'foo'),
 (4, 'c', 'bar'),
 (1, 'd', 'bar'),
 (2, 'a', 'foo'),
 (2, 'd', 'foo')]
</code></pre>
