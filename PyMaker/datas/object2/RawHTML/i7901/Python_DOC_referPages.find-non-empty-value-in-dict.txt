<div class="post-text" itemprop="text">
<p>I have a dict like this:</p>
<pre><code>d = {'first':'', 'second':'', 'third':'value', 'fourth':''}
</code></pre>
<p>and I want to find first non-empty value (and it's name, in this example <code>'third'</code>). There may be more than one non-empty value, but I only want the first one I find.</p>
<p>How can I do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You could use <code>next</code> (dictionaries are unordered - this somewhat changed in Python 3.6 but that's only an implementation detail currently) to get one "not-empty" key-value pair:</p>
<pre><code>&gt;&gt;&gt; next((k, v) for k, v in d.items() if v)
('third', 'value')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use an <code>OrderedDict</code> which preserves the order of elements. Then loop over them and find the first that isn't empty:</p>
<pre><code>from collections import OrderedDict
d = OrderedDict()
# fill d
for key, value in d.items():
    if value:
        print(key, " is not empty!")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Like this?</p>
<pre><code>def none_empty_finder(dict):
    for e in dict:
        if dict[e] != '':
            return [e,dict[e]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>d = {'first':'', 'second':'', 'third':'value', 'fourth':''}
for k, v in d.items():
    if v!='':
        return k, v
</code></pre>
<p><strong>Edit 1</strong></p>
<p><code>from the comment</code> if the value is <code>None</code> or <code>''</code> we better use <code>if v:</code> instead of <code>if v!=''</code>. <code>if v!=''</code> only check the <code>''</code> and skip others</p>
