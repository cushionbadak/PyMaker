<pre><code>df1 = {A: hello, B: bye, C:good morning, F: nice}
df2 = {A: hola, D: buenos dias, C: adi贸s, G: bad}
</code></pre>
<p>So the resulting dataframe I want is like:</p>
<pre><code>  A   |B     |C            |D           |F    |G  
----------------------------------------------------
hello |bye   |good morning |None        |nice |None
hola  |None  |adi贸s        |buenos dias |None |bad
</code></pre>
<p>I've been trying to merge the dataframes using the 'merge' pandas' function as follows:</p>
<pre><code>res = pandas.merge(df1, df2, on='A')
</code></pre>

<div class="post-text" itemprop="text">
<p>First, create your <code>pd.DataFrame</code> objects</p>
<pre><code>df1 = pd.DataFrame({'A': 'hello', 'B': 'bye', 'C':'good morning', 'F': 'nice'}, index=[0])
df2 = pd.DataFrame({'A': 'hola', "D":' buenos dias', 'C':' adi贸s', 'G': 'bad'}, index=[0])
</code></pre>
<p>Then, just use <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.merge.html" rel="nofollow noreferrer"><code>how='outer'</code></a></p>
<pre><code>pd.merge(df1, df2, how='outer')
</code></pre>
<p>which yields</p>
<pre><code>    A       B   C               F       D           G
0   hello   bye good morning    nice    NaN         NaN
1   hola    NaN adi贸s           NaN     buenos dias bad
</code></pre>
