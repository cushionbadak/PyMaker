<div class="post-text" itemprop="text">
<p>The <code>u</code> in front of the string makes a difference.The '<code>u</code>' in front of the string values means the string has been represented as unicode. It is a way to represent more characters than normal ascii can manage.</p>
<p>The default encoding for Python source code is <strong>UTF-8</strong>, so you can simply include a <strong>Unicode</strong> character in a string literal.</p>
