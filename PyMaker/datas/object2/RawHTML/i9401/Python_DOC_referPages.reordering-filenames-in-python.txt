<p>I have the following problem:</p>
<p>I need to load several data files. The files are named by my device like:</p>
<pre><code>meas98.dat
meas99.dat
meas100.dat
meas101.dat
</code></pre>
<p>With other words, there are no leading zeros. Therefore, if I get the filenames via</p>
<pre><code>os.listdir
</code></pre>
<p>they are ordered alphabetically, meaning "meas100.dat" will be the first one. This is obviously not what I want to achieve. The question is what is the most elegant way of doing this?</p>
<p>The (unelegant) way I came up with is: </p>
<ul>
<li>load the filenames</li>
<li>extract the filenumber</li>
<li>order the filenumber (get the indices)</li>
<li>order the filenames with those indices </li>
</ul>
<p>I am pretty sure python has something build in that can do this while loading the files...</p>

<div class="post-text" itemprop="text">
<p>Perhaps this will suit your problem:</p>
<pre><code>import re

l = ['meas100.dat',
     'meas101.dat',
     'meas98.dat',
     'meas99.dat']


sorted(l, key=lambda x: int(re.match('\D*(\d+)', x).group(1)))
</code></pre>
<p>Output:</p>
<pre><code>['meas98.dat', 'meas99.dat', 'meas100.dat', 'meas101.dat']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>l = ['meas98.dat',
    'meas99.dat',
    'meas100.dat',
    'meas101.dat']
l.sort(key=lambda i: int(i.strip('meas.dat')))
</code></pre>
<p>There is a pythonic way to do this by using <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow noreferrer">pathlib</a> module:</p>
<p>this is the files in my ternimal:</p>
<pre><code>~/so$ ls
meas100.dat  meas98.dat  meas99.dat
</code></pre>
<p>this is the files in python:</p>
<pre><code>from pathlib import Path
p = Path('/home/li/so/')
list(p.iterdir())
[PosixPath('/home/li/so/meas99.dat'),
 PosixPath('/home/li/so/meas98.dat'),
 PosixPath('/home/li/so/meas100.dat')]
</code></pre>
<p>looks like the <code>pathlib</code> has do this sort for you, you can take a try.</p>

<div class="post-text" itemprop="text">
<p>Using <code>slicing</code> <code>[4:-4]</code> to get only numbers from filename - and <code>sorted()</code> will use them to sort filenames.</p>
<pre><code># random order
l = [
    'meas98.dat',
    'meas100.dat',
    'meas99.dat',
    'meas101.dat',
    'meas1.dat',
]

sorted(l, key=lambda x: int(x[4:-4]))

print(l)
</code></pre>
<p>result</p>
<pre><code>['meas1.dat', 'meas98.dat', 'meas99.dat', 'meas100.dat', 'meas101.dat']
</code></pre>
