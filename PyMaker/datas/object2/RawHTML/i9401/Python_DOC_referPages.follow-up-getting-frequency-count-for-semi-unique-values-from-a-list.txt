<p>This is a follow up to this question that I asked: <a href="https://stackoverflow.com/questions/41497600/returning-semi-unique-values-from-a-list">Returning semi-unique values from a list</a>. In short I wanted to turn the list:</p>
<pre><code>[a,a,a,b,b,b,a,a,a]
</code></pre>
<p>into</p>
<pre><code>['a','b,'a']
</code></pre>
<p>User Psidom's answer worked a charm by doing the following:</p>
<pre><code>#Paraphrased answer from Psidom

from itertools import groupby  

lst = ['a','a','a','b','b','b','a','a','a']

[k for k, _ in groupby(lst)]
# ['a', 'b', 'a'] #call this TRANSFORM
</code></pre>
<p>My final goal is to make a pd.DataFrame which has rows=(number of elements in TRANSFORM) and two columns: label, and frequency. My trouble is getting the frequency count for each 'semi-unique' value from the original list, this should be:</p>
<pre><code>[3,3,3] 
</code></pre>

<p>Luke</p>

<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer"><code>groupby</code> documentation</a> has this example output:</p>
<pre><code># [k for k, g in groupby('AAAABBBCCDAABBB')] --&gt; A B C D A B
# [list(g) for k, g in groupby('AAAABBBCCD')] --&gt; AAAA BBB CC D
</code></pre>
<p>So, I think you need:</p>
<pre><code>[(k, len(list(v))) for k, v in groupby(lst)]
</code></pre>
<p>Which would give you the element and the frequency:</p>
<pre><code>[('a', 3), ('b', 3), ('a', 3)]
</code></pre>
