<div class="post-text" itemprop="text">
<p>Simply add a <a href="https://docs.python.org/3.6/library/datetime.html#timedelta-objects" rel="nofollow noreferrer"><code>timedelta</code></a> to the <code>timestamp</code>:</p>
<pre><code>timestamp = datetime.datetime(year=2016, month=12, day=02, hour=13, minute=26, second=49)
d = datetime.timedelta(seconds=1136)
new_timestamp = timestamp+d
</code></pre>
<p>Running this in the console:</p>
<pre><code>$ python
Python 2.7.12 (default, Nov 19 2016, 06:48:10) 
[GCC 5.4.0 20160609] on linux2
Type "help", "copyright", "credits" or "license" for more information.
&gt;&gt;&gt; import datetime
&gt;&gt;&gt; timestamp = datetime.datetime(year=2016, month=12, day=02, hour=13, minute=26, second=49)
&gt;&gt;&gt; d = datetime.timedelta(seconds=1136)
&gt;&gt;&gt; new_timestamp = timestamp+d
&gt;&gt;&gt; new_timestamp
datetime.datetime(2016, 12, 2, 13, 45, 45)
</code></pre>
<p>So the result is <em>December 12, 2016</em> at 13:45:45.</p>

<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; now = datetime.datetime.now()
&gt;&gt;&gt; now 
datetime.datetime(2017, 1, 9, 16, 16, 12, 257210)
&gt;&gt;&gt; now + datetime.timedelta(seconds=1136)
datetime.datetime(2017, 1, 9, 16, 22, 12, 257210)
&gt;&gt;&gt; 
</code></pre>

<div class="post-text" itemprop="text">
<p>I know, there are already <strong>four</strong> answers but consider using the <strong>arrow</strong> module. It makes many date and time manipulations easier.</p>
<pre><code>&gt;&gt;&gt; import arrow
&gt;&gt;&gt; arrow.get('2016-12-02 13:26:49').shift(seconds=+1136)
&lt;Arrow [2016-12-02T13:45:45+00:00]&gt;
&gt;&gt;&gt; newTime = arrow.get('2016-12-02 13:26:49').shift(seconds=+1136)
&gt;&gt;&gt; newTime.strftime('%d-%m-%y')
'02-12-16'
</code></pre>
<p>In the first statement after the import you can see that arrow can convert your timestamp to an internal time format and do the shift in one line of code.</p>
<p>In the next statement I save that result and show the it's easy to manipulate the internal format in the usual ways. (More are available with arrow.)</p>
