<div class="post-text" itemprop="text">
<p>It is loadable with <a href="https://docs.python.org/3/library/json.html#json.load" rel="nofollow noreferrer"><code>json.load()</code></a> as is:</p>
<pre><code>import json

with open("input.txt") as f:
    data = json.load(f)
    print(data)
</code></pre>
