<p>With <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><code>re.findall()</code></a>, you can achieve that like below:</p>
<pre><code>re.findall(r'(\d{4}\-\d{2}\-\d{2})', s)
</code></pre>
<p>If you want only the second date in each line, try:</p>
<pre><code>re.findall(r':(\d{4}\-\d{2}\-\d{2})', s)
</code></pre>
<p><strong>Output:</strong></p>
