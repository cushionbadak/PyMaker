<div class="post-text" itemprop="text">
<p>As per <a href="https://docs.python.org/3/library/random.html" rel="nofollow noreferrer">the Python docs</a>:</p>
<blockquote>
<p>Python uses the Mersenne Twister as the core generator.</p>
</blockquote>
</div>
<span class="comment-copy">You can view the <a href="https://hg.python.org/cpython/file/3.6/Lib/random.py" rel="nofollow noreferrer">code</a> and <a href="https://docs.python.org/3/library/random.html" rel="nofollow noreferrer">docs</a> online. These should answer your question.</span>
<span class="comment-copy">The default RNG is Mersenne Twister, but as the docs mention: "Class Random can also be subclassed if you want to use a different basic generator of your own devising". It's also easy to specify <code>SystemRandom</code> as the generator if you want a higher grade of randomness.</span>
