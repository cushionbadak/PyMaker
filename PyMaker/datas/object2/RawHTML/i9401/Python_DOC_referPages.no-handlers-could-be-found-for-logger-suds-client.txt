<p>You need to setup a looger first to get ride of this message. Take a look at <a href="https://fedorahosted.org/suds/wiki/Documentation#LOGGING" rel="nofollow noreferrer">https://fedorahosted.org/suds/wiki/Documentation#LOGGING</a> and 
<a href="https://docs.python.org/2/library/logging.html" rel="nofollow noreferrer">https://docs.python.org/2/library/logging.html</a> (or <a href="https://docs.python.org/3/library/logging.html" rel="nofollow noreferrer">https://docs.python.org/3/library/logging.html</a> depending on your version of python). However something like this should solve the problem:</p>
<pre><code>&gt;&gt;&gt; import logging
&gt;&gt;&gt; from suds.client import Client
&gt;&gt;&gt; logger = logging.getLogger('suds.client').setLevel(logging.DEBUG)
&gt;&gt;&gt; logger.setLevel(logging.DEBUG)
&gt;&gt;&gt; # ...
</code></pre>
