<p>Here's a simple function which returns the coordinates as a tuple (or <code>None</code> if no index is found). Note that this is for 2D matrices, and returns the first instance of the element in the matrix.</p>
<p><em>(Edit: see <a href="https://stackoverflow.com/a/41431027/7089349">hiro protagonist's answer</a> for an alternative Pythonic version)</em></p>
<pre><code>def find(element, matrix):
    for i in range(len(matrix)):
        for j in range(len(matrix[i])):
            if matrix[i][j] == element:
                return (i, j)
</code></pre>
<p>Or, if you want to find all indexes rather than just the first:</p>
<pre><code>def findall(element, matrix):
    result = []
    for i in range(len(matrix)):
        for j in range(len(matrix[i])):
            if matrix[i][j] == element:
                result.append((i, j))
    return result
</code></pre>
<p>You can use it like so:</p>
<pre><code>A = [[5, 10],
     [15, 20],
     [25, 5]]


find(25, A) # Will return (2, 0)
find(50, A) # Will return None

findall(5, A) # Will return [(0, 0), (2, 1)]
findall(4, A) # Will return []
</code></pre>

<p>a (in my opinion) more pythonic version of the <a href="https://stackoverflow.com/a/41430965/4954037">FlipTack's altorithm</a>:</p>
<pre><code>def find(element, matrix):
    for i, matrix_i in enumerate(matrix):
        for j, value in enumerate(matrix_i):
            if value == element:
                return (i, j)
</code></pre>
<p>in python it is often more natural to iterate over elements of lists instead of just the indices; if indices are needed as well, <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a> helps. this is also more efficient.</p>
<p>note: just as <code>list.index</code> (without a second argument) this will only find the first occurrence.</p>

<p>Since you say that you're a beginner, pardon me if you already know some of the below. Just in case I'll describe the basic logic you can use to write your own function or understand the other answers posted here better:
To access an element in a specific row of a list, for example, if you wanted to get the first element and save it in a variable:</p>
<pre><code>myList=[1,10,54,85]
myvar = myList[0] # note that you access the first element with index 0
</code></pre>
<p>myvar now stores 1. Why index 0? Think of the index as an indicator of "how far away from the beginning of the list an element is." In other words, the first element is a distance of 0 from the start.
What if you have a multi-dimensional list like so?</p>
<pre><code>multi = [[0, 1, 2],
         [3, 4, 5],
         [6, 7, 8]
        ]
</code></pre>
<p>Now you think in terms of row and column (and of course you could have n-dimensional lists and keep going).</p>
<p>How to retrieve the 5? That is a distance of 1 row from the start of the list of rows and 2 columns away from the start of the sub-list.
Then:</p>
<pre><code>myvar = multi[1][2]
</code></pre>
<p>retrieves the 5.</p>
<p>FlipTack's and hiro protagonist's functions wrap this logic in the nice compact procedures, which search the entire 2-dimensional list, comparing elements until the desired one is found, then returning a tuple of the indices or continuing to search for duplicate elements. Note that if your lists are guaranteed to sorted you can then use a binary search algorithm across rows and columns and get the answer faster, but no need to worry about that for now.
Hopefully this helps.</p>

<p>You can also add a tag to your function to search the occurrences of your input matrix/list.</p>
<p>For example: </p>
<p><strong>If you input is 1D vector:</strong></p>
<pre><code>def get_index_1d(a = [], val = 0, occurrence_pos = False):
    if not occurrence_pos:
        for k in range(len(a)):
            if a[k] == val:
                return k
    else:
        return [k for k in range(len(a)) if a[k] == val]
</code></pre>
<p>Output:</p>
<pre><code>a = [1,10,54,85, 10]
index = get_index_1d(a, 10, False)
print("Without occurrence: ", index)
index = get_index_1d(a, 10, True)
print("With occurrence: ", index)

&gt;&gt;&gt; Without occurrence:  1
&gt;&gt;&gt; With occurrence:  [1, 4]
</code></pre>
<p><strong>For 2D vector:</strong></p>
<pre><code>def get_index_2d(a = [], val = 0, occurrence_pos = False):
    if not occurrence_pos:
        for k in range(len(a)):
            for j in range(len(a[k])):
                if a[k][j] == val:
                    return (k, j)

    else:
        return [(k, j) for k in range(len(a)) for j in range(len(a[k])) if a[k][j] == val]
</code></pre>
<p>Output:</p>
<pre><code>b = [[1,2],[3,4],[5,6], [3,7]]
index = get_index_2d(b, 3, False)
print("Without occurrence: ", index)
index = get_index_2d(b, 3, True)
print("With occurrence: ", index)

&gt;&gt;&gt; Without occurrence:  (1, 0)
&gt;&gt;&gt; With occurrence:  [(1, 0), (3, 0)]
</code></pre>
