<div class="post-text" itemprop="text">
<p>Any expression can be used as the condition in <code>if</code> or <code>while</code>. See <a href="https://docs.python.org/2.4/lib/truth.html" rel="nofollow noreferrer">https://docs.python.org/2.4/lib/truth.html</a> for how the truth value of the expression is evaluated.</p>
</div>
<div class="post-text" itemprop="text">
<p>As far as I know, any expression of any type will be considered "True" unless it's an empty list/tuple, an empty string (which could technically also be considered a list of characters) or the number 0.</p>
<p>P.S. Just as a fun fact, the number 0.000001 returns True.</p>
</div>
<div class="post-text" itemprop="text">
<p>As stated in the <a href="https://docs.python.org/library/stdtypes.html#truth-value-testing" rel="nofollow noreferrer">python documentation on "Truth Value Testing"</a>:</p>
<blockquote>
<p>The following values are considered false: </p>
<p>[...] </p>
<ul>
<li>any empty sequence, for example, <code>''</code>, <code>()</code>, <code>[]</code>. </li>
</ul>
<p>[...] </p>
<p>All other values are considered true â€” so objects of many types are always true.</p>
</blockquote>
<p>So any not-empty list will be evaluated as <code>True</code> and any python object can be evaluated in a boolean context (if it doesn't throw an Exception in it's <code>__bool__</code> or <code>__len__</code> method, like for example <code>numpy.ndarray</code>).</p>
</div>
