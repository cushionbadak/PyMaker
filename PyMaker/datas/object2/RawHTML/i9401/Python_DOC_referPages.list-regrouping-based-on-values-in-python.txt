<p>I have been trying to convert some data, first value in each group is the group identifier:</p>
<pre><code>data = [(a, 1, 2), (a, 3, 4), (b, 5, 6)]
</code></pre>
<p>into</p>
<pre><code>data = [[(1, 2), (3, 4)], [(5, 6)]]
</code></pre>
<p>What is the simplest way to do this? Thanks in advance</p>
</div>
<div class="post-text" itemprop="text">
<p>If the <code>data</code> list is sorted by the grouping key, you can use <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer"><code>itertools.groupby</code></a>:</p>
<pre><code>In [1]: from itertools import groupby

In [2]: from operator import itemgetter

In [3]: data = [(a, 1, 2), (a, 3, 4), (b, 5, 6)]

In [4]: [[l[1:] for l in group] for _, group in groupby(data, key=itemgetter(0))]
Out[4]: [[(1, 2), (3, 4)], [(5, 6)]]
</code></pre>
</div>
