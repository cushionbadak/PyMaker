<div class="post-text" itemprop="text">
<p>You have lines with <code>"</code> for the tweet. CSV can <em>quote</em> columns by quoting them with <code>"</code> around the value, including newlines. Everything from the opening <code>"</code> to the next closing <code>"</code> is a single column value.</p>
<p>You can disable quote handling by setting the <a href="https://docs.python.org/3/library/csv.html#csv.Dialect.quoting" rel="nofollow noreferrer"><code>quoting</code> option</a> to <a href="https://docs.python.org/3/library/csv.html#csv.QUOTE_NONE" rel="nofollow noreferrer"><code>csv.QUOTE_NONE</code></a>:</p>
<pre><code>reader = csv.reader(f, delimiter='\t', quoting=csv.QUOTE_NONE)
</code></pre>
