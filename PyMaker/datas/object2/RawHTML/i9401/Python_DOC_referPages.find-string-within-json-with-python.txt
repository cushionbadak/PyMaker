<div class="post-text" itemprop="text">
<p><code>date['device']</code> contains a list of objects, so you should treat it as such and iterate over them:</p>
<pre><code>for element in data['device']:
    if element['serial'] == '00000000762c1d3c':
        print 'there'
        print element['registered_id']
        break
else:
    print 'not there'
</code></pre>
<p>This is using the somehow lesser-known <code>for-else</code> construct: <a href="https://docs.python.org/3/tutorial/controlflow.html#break-and-continue-statements-and-else-clauses-on-loops" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/controlflow.html#break-and-continue-statements-and-else-clauses-on-loops</a></p>

<div class="post-text" itemprop="text">
<p>first, your input isn't json. Json uses double quotes. But suppose you successfully <em>loaded</em> it with json, it's now a dictionary, called <code>d</code>.</p>
<p>Then you can scan all sub-dicts of <code>d</code> and test <code>serial</code> key against your value, stopping when found using <code>any</code> and a generator comprehension:</p>
<pre><code>print(any(sd['serial']=='00000000762c1d3c' for sd in d['device']))
</code></pre>
<p>returns <code>True</code> if serial found <code>False</code> otherwise.</p>
