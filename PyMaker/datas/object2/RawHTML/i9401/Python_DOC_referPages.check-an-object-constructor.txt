<div class="post-text" itemprop="text">
<p>I'm developing a Flask-Python app and I'd like to know how do I get an object class/constructor.</p>
<p>I want this because I had to merge two different arrays and now I have a <strong>result</strong> array with objects of both classes.</p>
<p>Imagine this:</p>
<pre><code>[Person, Person, Person, Person]
[Pet, Pet, Pet, Pet, Pet]
</code></pre>
<p>I merged both arrays in birthdate order and I have a final array:</p>
<pre><code>[Person, Pet, Pet, Person, Pet, Pet, Person]
</code></pre>
<p>But now I want to print them on my <strong>PetPerson.html</strong>. How do I differentiate both of them if the variable for <strong>birthdate</strong> is different?</p>
<pre><code>{% for a in personpets %}
if this object is a pet:
print a.petbirthdate
else:
print a.personbirthdate
{% endblock %}
</code></pre>
<p>Any idea?</p>
<p>Best regards</p>
</div>
<div class="post-text" itemprop="text">
<p>You basically want to know which class the object belongs to.</p>
<p>You can check that using <code>object_name.__class__.__name__</code>, this will give you either <code>Person</code> or <code>Pet</code> in this case. And hence you can replace this in your if condition as.</p>
<p><code>
{% for a in personpets %}
if object.__class__.__name__ == "Pet":
print a.petbirthdate
else:
print a.personbirthdate
{% endblock %}
</code></p>
<p>Cheers</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use inherit.</p>
<p>Let's say both person and pets are animals:</p>
<pre><code>class Animals(object):
    get_birthdate(self):pass
</code></pre>
<p>Then your person and pet inherit from animals:</p>
<pre><code>class Person(Animal):
get_birthdate(self):
    return self.personbirthdate

class Pet(Animal):
    get_birthdate(self):
        return self.petbirthdate
</code></pre>
<p>Then in your template:</p>
<pre><code>{% for a in personpets %}
{{ a.get_birthdate }}
{% endblock %}
</code></pre>
