<div class="post-text" itemprop="text">
<p>The statement </p>
<pre><code>for key,value in globalViewDict:
</code></pre>
<p>expects <em>two</em> values for each iteration, but iteration over a dictionary only gives you <em>single</em> objects, the keys. This is why you get your error.</p>
<p>You probably want to use <a href="https://docs.python.org/3/library/stdtypes.html#dict.items" rel="nofollow noreferrer"><code>dict.items()</code></a>:</p>
<pre><code>for key,value in globalViewDict.items():
</code></pre>
<p><code>dict.items()</code> lets you iterate over the <code>(key, value)</code> pairs in the dictionary. If you are using Python 2, you may want to use <a href="https://docs.python.org/2/library/stdtypes.html#dict.iteritems" rel="nofollow noreferrer"><code>dict.iteritems()</code></a> instead to avoid having to create a whole list first.</p>
