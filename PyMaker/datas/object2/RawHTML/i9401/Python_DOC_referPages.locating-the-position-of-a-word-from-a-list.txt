<div class="post-text" itemprop="text">
<h2>Awkwardness</h2>
<p>First, you should not use <code>list</code> as a variable because <a href="https://docs.python.org/3/library/functions.html" rel="nofollow noreferrer"><code>list</code></a> is one of the default Python functions. If you do so, the <code>list()</code> function (which is also a type) is replace by your variable in the local scope. So I renamed it <code>alist</code>.</p>
<pre><code>alist = ["apple", "banana", "carrot", "pear"]
</code></pre>
<h2>Explanation</h2>
<p>The following statement is a comprehension list:</p>
<pre><code>result = [i for i, x in enumerate(alist, start=1) if x == word]
</code></pre>
<p>This comprehension list can be written like this:</p>
<pre><code>result = []
for i, x in enumerate(alist, start=1):
    if x == word:
        result.append(i)
</code></pre>
<p>For instance:</p>
<pre><code>&gt;&gt;&gt; alist = ["apple", "banana", "carrot", "pear"]
&gt;&gt;&gt; word = "carrot"
&gt;&gt;&gt; [i for i, x in enumerate(alist, start=1) if x == word]
[3]
</code></pre>
<p>This comprehension list returns all the positions of the word "carrot" in ["apple", "banana", "carrot", "pear"].</p>
<p>The loop:</p>
<pre><code>for i in [i for i, x in enumerate(alist, start=1) if x == word]:
    print(i)
</code></pre>
<p>Is vainly complex because it iterates the comprehension list.</p>
<p>You can simplify like this:</p>
<pre><code>print([i for i, x in enumerate(alist, start=1) if x == word])
</code></pre>
<p>Or:</p>
<pre><code>for i, x in enumerate(alist, start=1):
    if x == word:
        print(i)
</code></pre>
<p><em>Note: <code>alist.index(word) + 1</code> can give the first word position in the list.</em></p>
