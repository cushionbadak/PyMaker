<p>All you need to do is use Python's standard file-writing behavior:</p>
<pre><code>img = GoogleMap(51.0, 0.0)
with open("GoogleMap.png", "wb") as png:
    png.write(img.image)
</code></pre>
<p>Here's a very simple way of accessing the three lat/long pairs you want:</p>
<pre><code>places = [GoogleMap(51.0, 0.0), GoogleMap(60.2, 5.2), GoogleMap(71.9, 8.9)]
for position, place in enumerate(places):
    with open("place_{}.png".format(position), "wb") as png:
        png.write(place.image)
</code></pre>
<p>I'll leave it up to you to write a function that takes arbitrary latitude/longitude pairs and saves images of them.</p>
