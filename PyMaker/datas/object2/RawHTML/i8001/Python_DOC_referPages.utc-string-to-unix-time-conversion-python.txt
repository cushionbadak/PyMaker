<div class="post-text" itemprop="text">
<p>You need to convert the string '20170124T1815' to a datetime instance:</p>
<pre><code>import datetime

dt = datetime.datetime.strptime('20170124T1815', '%Y%m%dT%H%M')
</code></pre>
<p>Then use <code>timestamp()</code> method to convert to UNIX time:</p>
<pre><code>ut = dt.timestamp()
# -&gt; 1485281700.0
</code></pre>
<p>The <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.timestamp" rel="nofollow noreferrer">documentation</a>:</p>
<blockquote>
<p>datetime.timestamp()</p>
<p>Return POSIX timestamp corresponding to the <em>datetime</em> instance. The return value is a float similar to that returned by <code>time.time()</code>.</p>
</blockquote>
<p><strong>EDIT</strong></p>
<p>For Python version &lt; 3.3, you can use:</p>
<pre><code>ut = (dt - datetime.datetime(1970, 1, 1)).total_seconds()
</code></pre>
<p>Or, you can use:</p>
<pre><code>import time

ut = time.mktime(dt.timetuple())
</code></pre>
