<div class="post-text" itemprop="text">
<p>You can use <code>in-built</code> functions as <a href="https://docs.python.org/3/library/stdtypes.html#str.isdigit" rel="nofollow noreferrer"><code>isdigit()</code></a> and <a href="https://docs.python.org/3/library/stdtypes.html#str.isalpha" rel="nofollow noreferrer"><code>isalpha()</code></a></p>
<pre><code>&gt;&gt;&gt; x = '111aa111aa'
&gt;&gt;&gt; number = ''.join([i for i in x if i.isdigit()])
'111111'
&gt;&gt;&gt; string = ''.join([i for i in x if i.isalpha()])
'aaaa'
</code></pre>
<p>Or You can use <code>regex</code> here :</p>
<pre><code>&gt;&gt;&gt; x = '111aa111aa'
&gt;&gt;&gt; import re
&gt;&gt;&gt; numbers = ''.join(re.findall(r'\d+', x))
'111111'
&gt;&gt;&gt; string = ''.join(re.findall(r'[a-zA-Z]', x))
'aaaa'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; my_string = '111aa111aa'
&gt;&gt;&gt; ''.join(filter(str.isdigit, my_string))
'111111'
&gt;&gt;&gt; ''.join(filter(str.isalpha, my_string))
'aaaa'
</code></pre>

Alternatively, I present some options using <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">dictionaries</a> that can help organize data, starting from basic to intermediate examples with arguably increasing intricacy.</p>

d = {"Number":"", "String":""}

d

# '111111'

import collections as ct

# Default Dictionary
dd = ct.defaultdict(str)
for char in s:
    if char.isdigit():
        dd["Number"] += char
    elif char.isalpha():
        dd["String"] += char
