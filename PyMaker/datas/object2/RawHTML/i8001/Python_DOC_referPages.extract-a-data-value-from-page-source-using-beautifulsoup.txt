<div class="post-text" itemprop="text">
<p>Using regular expression capturing group:</p>
<pre><code>var SYNCHRONIZER_TOKEN_VALUE = '(.+?)'
</code></pre>
<p>, you can get the captured group using <a href="https://docs.python.org/3/library/re.html#re.match.group" rel="nofollow noreferrer"><code>&lt;MatchObject&gt;.group(1)</code></a></p>
<hr/>
<pre><code>import re

html = '''
var SYNCHRONIZER_TOKEN_NAME = 'SynchronizerToken';
var SYNCHRONIZER_TOKEN_VALUE = 'dd3a0c31e365c458d2d3e68e3c98f772bd2103eccf38163e10ce039c2b70a61a';
'''

token = None
matched = re.search(r"var SYNCHRONIZER_TOKEN_VALUE = '(.+?)'", html)
if matched:
    token = matched.group(1)

# token =&gt; 'dd3a0c31e365c458d2d3e68e3c98f772bd2103eccf38163e10ce039c2b70a61a'
</code></pre>
</div>
