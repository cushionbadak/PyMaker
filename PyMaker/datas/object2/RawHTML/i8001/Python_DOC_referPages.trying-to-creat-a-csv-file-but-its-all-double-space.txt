<p>you might want to replace all \r\n with just \n.</p>
<p>One way to do that is: <code>theString.replace("\r\n","\n")</code></p>
<p>in your case:</p>
<pre><code>o.addJustString("ted \r\n test \r\n test \r\n test \r\n".replace("\r\n","\n")
</code></pre>

<div class="post-text" itemprop="text">
<p>To end lines in the correct way for any given operating system use os.linesep:</p>
<pre><code>import os
o.open()
o.addJustString("ted" + os.linesep + "test" + os.linesep + "test" + os.linesep + "test" + os.linesep)
</code></pre>
<p>Or better yet make your helper function handle that for you:</p>
<pre><code>import os
o.open()
o.addJustString("ted")
o.addJustString("test")
o.addJustString("test")
o.addJustString("test")

def addJustString(self, add):
    """add a string to a line of the csv file"""
    self.f.write(add + os.linesep)
</code></pre>
<p>os.linesep will be \n on linux and \r\n on windows</p>
</div>
<span class="comment-copy">Use <code>\n</code> instead of <code>\r\n</code> in the string. Python will convert them to <code>\r\n</code> if necessary on Windows (assuming you've opened the file in text mode).</span>
