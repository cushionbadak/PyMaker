<p>In order to convert your string to the list of <code>float</code> numbers, you may use a <em>list comprehension</em> expression with <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><strong><code>str.split()</code></strong></a> and type-cast each value to <a href="https://docs.python.org/3/library/functions.html#float" rel="nofollow noreferrer"><code>float</code></a> as:</p>
<pre><code>&gt;&gt;&gt; num_str = '-0.4152832031\t0.1403808593\t-0.1831054687\t-1.0670732259\t-0.0609756088\t0.0000000000\t185\t-1\t-683\r\n'

&gt;&gt;&gt; [float(n) for n in num_str.split()]
[-0.4152832031, 0.1403808593, -0.1831054687, -1.0670732259, -0.0609756088, 0.0, 185.0, -1.0, -683.0]
</code></pre>
<p>OR, you may use <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><strong><code>map()</code></strong></a> to get the same result as:</p>
<pre><code>&gt;&gt;&gt; map(float, num_str.split())
[-0.4152832031, 0.1403808593, -0.1831054687, -1.0670732259, -0.0609756088, 0.0, 185.0, -1.0, -683.0]
</code></pre>
<p><strong>Note:</strong> <code>map()</code> returns generator object in Python 3.x. In order to convert it to list, you have to explicitly type-cast the value to <code>list</code> using <code>list(map(..., ...))</code></p>
