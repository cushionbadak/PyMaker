<p>I want i to range from 65 to 91 and from 97 to 123 (ASCII for upper and lower case letters) omitting 92 to 96, but I can only input one start and one stop value. Is there a way to omit those or set diferent start and stop values?</p>

<div class="post-text" itemprop="text">
<p>You can <em>chain</em> the two range objects using <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow noreferrer"><code>itertools.chain</code></a>:</p>
<pre><code>&gt;&gt;&gt; from itertools import chain
&gt;&gt;&gt; chain(range(5), range(9, 14))
&lt;itertools.chain object at 0x101a8e588&gt;
&gt;&gt;&gt; list(_)
[0, 1, 2, 3, 4, 9, 10, 11, 12, 13]
</code></pre>
<hr/>
<pre><code> from itertools import chain

 for i in chain(range(65, 91), range(97, 123)):
     ...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/2/library/itertools.html#itertools.chain" rel="nofollow noreferrer"><code>itertools.chain</code></a>:</p>
<pre><code>import itertools
for i in itertools.chain(range(65, 92), range(97, 123)):
</code></pre>

<div class="post-text" itemprop="text">
<p>First, you should not use 'magic numbers' in your code. If 65 is the ASCII code for 'A', say so: <code>ord('A')</code>. Second, you can combine two lists into one list and iterate over it:</p>
<pre><code>for i in list(range(ord('A'),ord('Z')+1)) + list(range(ord('a'),ord('z')+1)):
</code></pre>
