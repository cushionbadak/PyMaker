<p>I tried to implement a logger into my python script according to the <a href="https://docs.python.org/3/howto/logging-cookbook.html" rel="nofollow noreferrer">python documentation</a>. This is the code:</p>
<pre><code>import logging

def generateLogger(loggername='SM-Logger', path="logs/log.log"):

    logger = logging.getLogger(loggername)
    logger.setLevel(logging.DEBUG)

    ch = logging.StreamHandler()
    ch.setLevel(logging.ERROR)

    formatter = logging.Formatter('%(asctime)s - %(name)s\
                              - %(levelname)s - %(message)s')

    ch.setFormatter(formatter)

    logger.addHandler(ch)

    return logger

logger = generateLogger("testlogger", "testlog.log")
logger.WARNING("testtest")
</code></pre>
<p>I get this error message:</p>
<pre><code>File "loggertest.py", line 39, in &lt;module&gt;
    logger.WARNING("testtest")
AttributeError: 'Logger' object has no attribute â€šWARNING'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The solution is to change the last line</p>
<pre><code>logger.WARNING("testtest")
</code></pre>
<p>into</p>
<pre><code>logger.warning("testtest")
</code></pre>
<p>Lower-case "warning" is the function, upper-case "WARNING" is the variable.</p>
</div>
