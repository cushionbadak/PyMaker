<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html?highlight=range#range" rel="nofollow noreferrer"><code>range()</code></a> function to iterate over the 700 integers:</p>
<pre><code>import os
import io

work_dir = "path/to/workdir"

for index in range(1, 701):
    name = "item{index}.txt".format(index=index)
    path = os.path.join(work_dir, name)
    with io.open(path, mode="r", encoding="utf-8") as fd:
        content = fd.read()
</code></pre>
<p>A different way is to use <a href="https://docs.python.org/3/library/glob.html" rel="nofollow noreferrer"><code>glob.glob</code></a> function to search for the text files:</p>
<pre><code>for path in glob.glob(os.path.join(work_dir, "item*.txt")):
    with io.open(path, mode="r", encoding="utf-8") as fd:
        content = fd.read()
</code></pre>
