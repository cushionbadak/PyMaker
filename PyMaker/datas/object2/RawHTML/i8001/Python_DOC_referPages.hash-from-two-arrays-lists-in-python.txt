<div class="post-text" itemprop="text">
<p>Using two Arrays of equal length, how can I create a Hash object where the elements from one array (the keys) are linked to the elements of the other (the values).</p>
<p>For example in Java I can use HashMap:</p>
<pre><code>String[] keys= {"apple", "banana", "cherry"};
int[] vals= {1, 2, 3};
HashMap&lt;String, Integer&gt; hash= new HashMap&lt;String, Integer&gt;();

for(int i= 0; i &lt; keys.length; i++){
   hash.put(keys[i], vals[i]);
}
</code></pre>
<p>How could do this Python?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> inside a dictionary comprehension to achieve the same result in python:</p>
<pre><code>&gt;&gt;&gt; keys = ['apple', 'banana', 'cherry']
&gt;&gt;&gt; values = [1, 2, 3]
&gt;&gt;&gt; hash = {key: value for key, value in zip(keys, values)}
&gt;&gt;&gt; hash
{'apple': 1, 'banana': 2, 'cherry': 3}
</code></pre>
</div>
