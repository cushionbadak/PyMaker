<p>I am working on a problem where I would like to print either "X:s" or "–" given certain conditions – like a coordinate system. </p>
<p>Given two list:</p>
<pre><code>my_list = [n for n in range(11)]
</code></pre>
<p>and</p>
<pre><code>coordinates = [3,5,7,8]
</code></pre>
<p>Let's say a given element is in <code>coordinates</code>and in <code>my_list</code> then an <code>X</code> would be printed at the index of that element's value; "–" would be printed in places where no such match existed.</p>
<p>Given <code>coordinates</code> and <code>my_list</code>the following would happen:
<code>&gt;&gt;&gt; ––X–X–XX</code></p>
<p>This what I have come up with so far</p>
<pre><code>for n in range(len(coordinates)):
    if coordinates[n] in my_list:
       print("–"*(coordinates[n]-(2**n)), end="")
       print("X", end="")
</code></pre>
<p>What is the best way to achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>I would suggest just going over the entire length of the string you need to print (<code>len(my_list)</code>), and on each index check whether you need to print an X or a -.
Something like that:</p>
<pre><code>for i in my_list:
    if i+1 in coordinates:
        print("X", end="")
    else:
        print("-", end="")
print("") # go down a line
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try this,</p>
<pre><code>In [42]: ''.join(['X' if i+1 in coordinates else '-' for i in my_list])
Out[42]: '--X-X-XX---'
</code></pre>
<p>And don't do <code>my_list = [n for n in range(11)]</code> , <code>range(11)</code> it's self give a list of elements, no need of list comprehension. </p>
</div>
