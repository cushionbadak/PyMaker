<p>Python list are classes with attributes. Here the append attribute is a method, and you are trying to re-assign its value. You should just use the function instead.</p>
<p>In python 3 you can append to a list like so :</p>
<pre><code>my_list = []
#Using member function append
my_list.append({"ID": 0})
# using [] operator
my_list[len(my_list)] = {"ID": 0 }
</code></pre>
<p>ref: <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/datastructures.html</a></p>
<p>Regarding the file writing, you might want to serialize your object usgin JSON before while it : </p>
<pre><code>fd = open("my_list.json","w")
json.dump(my_list, fd)
</code></pre>

<p>You should pick a formatting for saving your data. Here's your code modified to use json:</p>
<pre><code>#!/usr/bin/env python3

import json


def createFile(allColours):
    with open("colours.json", "w") as f:
        f.write(json.dumps(allColours, indent=4))


def data():
    allColours = []
    allColours.append({"ID": "1", "Shade": "Black", "Red": "0", "Green": "0", "Blue":"0"})
    allColours.append({"ID": "2", "Shade": "White", "Red": "255", "Green": "255", "Blue":"255"})
    allColours.append({"ID": "3", "Shade": "Red", "Red": "255", "Green": "0", "Blue":"0"})
    allColours.append({"ID": "4", "Shade": "Green", "Red": "0", "Green": "255", "Blue":"0"})
    allColours.append({"ID": "5", "Shade": "Blue", "Red": "0", "Green": "0", "Blue":"255"})
    createFile(allColours)


if __name__ == '__main__':
    data()
</code></pre>
<p>Here's another way to write your <code>data</code> function that it a bit easier to maintain:</p>
<pre><code>from collections import OrderedDict


def data2():
    keys = ("ID", "Shade", "Red", "Green", "Blue")
    values = (
        (1, "Black", 0, 0, 0),
        (2, "White", 255, 255, 255),
        (3, "Red", 255, 0, 0),
        (4, "Green", 0, 255, 0),
        (5, "Blue", 0, 0, 255),
    )
    # An OrderedDict will preserve the key order in the json output
    allColours = [OrderedDict(zip(keys, colour)) for colour in values]
    createFile(allColours)
</code></pre>
