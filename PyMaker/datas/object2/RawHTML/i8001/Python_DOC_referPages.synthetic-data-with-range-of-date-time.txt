<div class="post-text" itemprop="text">
<p>You can generate random dates between a date range using <a href="https://docs.python.org/3/library/datetime.html#timedelta-objects" rel="nofollow noreferrer">datetime.timedelta</a>, create a function that generates random start and end dates:</p>
<pre><code>import datetime
import random

def get_random_dates():
    start = datetime.date(2007, 1, 1)
    end = datetime.date(2017, 1, 1)
    d1 = start + datetime.timedelta(
        seconds=random.randint(0, (end - start).total_seconds()))
    d2 = start + datetime.timedelta(
        seconds=random.randint(0, (end - start).total_seconds()))

    # make sure start_date is before end_date
    return sorted([d1, d2])
</code></pre>
<p>Then you can get random start and end times by calling the function:</p>
<pre><code>startdatetime, enddatetime = get_random_dates()
</code></pre>
<p>And write these to your csv file along with other random generated data:</p>
<pre><code>output.write('%s,%.1f,%.6f,%.6f,%s,%s \n' % (hex1.lower(), 
                                       flt, 
                                       lon+dec_lon, 
                                       lat+dec_lat,
                                       str(startdatetime),
                                       str(enddatetime)))
</code></pre>
