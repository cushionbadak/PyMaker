<p>Your populating loop is:</p>
<pre><code>name_list = []
count = 0
name = raw_input("Enter the student's name: ")
while count &lt; 5:                        #CHANGE BACK TO 45 
    name == raw_input("Enter the student's name: ")
    name_list.append(name)
    count = count + 1
</code></pre>
<p>You first assign the value received through <code>raw_input</code> to <code>name</code>.</p>
<p>Then, for <code>count</code> from <code>0</code> to <code>4</code>, you check if <code>name</code> is equal to the input, and then, append it to <code>name_list</code>.</p>
<p>Instead of <strong>checking the equality</strong> by writing <code>name == raw_input(...)</code>, what you want is to <strong>assign</strong> the input value into <code>name</code>.</p>
<p>Therefore, you mustn't use <code>==</code>, but <code>=</code>.</p>
<p>Your loop should be:</p>
<pre><code>name_list = []
count = 0
name = raw_input("Enter the student's name: ")
while count &lt; 5:                        #CHANGE BACK TO 45 
    name = raw_input("Enter the student's name: ")
    name_list.append(name)
    count = count + 1
</code></pre>
<p>Now here is a more Pythonic way:</p>
<pre><code>names_list = []        # there are more than one name in the list
for _ in range(5):     # the loop index is not needed, so I use the anonymous underscore
    name = raw_input("Enter the student's name: ")
    names_list.append(name)
</code></pre>
