<div class="post-text" itemprop="text">
<p>To <strong>join list of multiple elements (strings) in the list</strong>, you may use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><strong><code>str.join</code></strong></a> as</p>
<pre><code>&gt;&gt;&gt; name = ['Aaron', 'Sheron']

#    v  to join the words in the list using space ' '
&gt;&gt;&gt; ' '.join(name)
'Aaron Sheron'
</code></pre>
<p>However, you are having a list of just one element. In order <strong>to access the element at <code>0</code>th index</strong>, you need to pass index as <em>(PS: <code>str.join</code> will work here too, but it is not required)</em>:</p>
<pre><code>&gt;&gt;&gt; name = ['Aaron']

#        v fetch `0`th element in the list
&gt;&gt;&gt; name[0]
'Aaron'
</code></pre>
<p>Please also refer:</p>
<ul>
<li><a href="https://stackoverflow.com/questions/10613131/how-to-access-list-elements">How to access List elements</a></li>
<li><a href="https://stackoverflow.com/questions/9008533/access-list-of-items-with-list-of-indices">Access list of items with list of indices</a></li>
</ul>
</div>

<p>You can also use:</p>
<pre><code>''.join(name)
</code></pre>
<p><code>join</code> joins all elements of the list into one string.</p>
