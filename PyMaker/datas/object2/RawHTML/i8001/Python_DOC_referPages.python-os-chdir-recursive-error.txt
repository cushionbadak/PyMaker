<div class="post-text" itemprop="text">
<p>Don't use <code>chdir()</code> which changes the working directory of your entire program.  Just pass paths directly:</p>
<pre><code>glob.glob(os.path.join("..", "data", "train", "Type_{}".format(t), "*.jpg"))
</code></pre>
</div>

<p>See @John's answer for a more suitable approach - don't <code>chdir()</code> unless you actually need to.</p>

<hr/>
<p>You're using relative paths, which is probably asking for trouble in this scenario.</p>
<p>Follow along below (it's not going to work):</p>
<ul>
<li><code>/home/joasa/my_project/</code></li>
<li><code>/home/joasa/data/train/Type_1/</code></li>
<li><code>/home/joasa/data/train/data/train/Type_2/</code></li>
<li><code>/home/joasa/data/train/data/train/data/train/Type_3/</code></li>
</ul>
<p>I would advise that you make this act on absolute paths, like below:</p>
<pre class="lang-py prettyprint-override"><code>import os

start_dir = os.getcwd()

for t in [1, 2, 3]:
    this_dir_rel = "../data/train/Type_%d" % ( t )
    this_dir_abs = os.path.join(start_dir, this_dir_rel)
    os.chdir(this_dir_abs)
</code></pre>
