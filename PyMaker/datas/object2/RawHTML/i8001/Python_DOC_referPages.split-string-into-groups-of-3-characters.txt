<p>I have a string</p>
<pre><code>ddffjh3gs
</code></pre>
<p>I want to convert it into a list</p>
<pre><code>["ddf", "fjh", "3gs"]
</code></pre>
<p>In groups of 3 characters as seen above.
What is the best way to do this in python 2.7?</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="noreferrer">list comprehension</a> with string slice:</p>
<pre><code>&gt;&gt;&gt; s = 'ddffjh3gs'
&gt;&gt;&gt; [s[i:i+3] for i in range(0, len(s), 3)]
['ddf', 'fjh', '3gs']
</code></pre>
