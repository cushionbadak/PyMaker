<p>I am trying to get the docstring of class methods using import module and getattr. I have done the following previously:</p>
<pre><code>getattr(import_module('string'),'capwords').__doc__
</code></pre>
<p>Which worked as expected?
Now I try to do the same with class methods and used <a href="https://docs.python.org/3/library/string.html#string.Formatter.format" rel="nofollow noreferrer">this</a> as an example </p>
<pre><code>getattr(import_module('string','Formatter'),'format').__doc__
</code></pre>
<p>I get the following error</p>
<pre><code>Traceback (most recent call last):
  File "&lt;input&gt;", line 1, in &lt;module&gt;

</code></pre>
<p>How do I acquire the doc string of a class method.</p>

<div class="post-text" itemprop="text">
<p>Feel like you are trying</p>
<pre><code>getattr(getattr(import_module('string'),'Formatter'), 'format').__doc__
</code></pre>
<p>OR</p>
<pre><code>from operator import attrgetter
attrgetter("Formatter.format")(import_module("string")).__doc__
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>To get the docstring of the <a href="https://docs.python.org/2/library/string.html#string.Formatter.format" rel="nofollow noreferrer"><code>string.Formatter.format</code></a> method, you can do:</p>
<pre><code>from string import Formatter

doc = Formatter.format.__doc__
</code></pre>
