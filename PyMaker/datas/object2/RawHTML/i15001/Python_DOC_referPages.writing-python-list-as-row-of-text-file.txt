<div class="post-text" itemprop="text">
<p>I have a python list as follows:</p>
<pre><code>data = [1,2,3,4,5]
</code></pre>
<p>I want to write it in text file as follows:</p>
<pre><code>1,2,3,4,5 ##as comma delineated.
</code></pre>
<p>I tried as:</p>
<pre><code>with open ('data.txt', 'w') as fo:
   for d in data:
     fo.write(str(d) + '\n')
</code></pre>
<p>But it wrote data as follows:</p>
<pre><code>1
2
3
4
5
</code></pre>
<p>How would you do it guys?</p>
<p>EDIT:</p>
<p>I will also have other lists obtained from for-loop, so all the one list i.e., 1,2,3,4,5 have to be adjusted in one line. In addition, I need faster performance.</p>
</div>
<div class="post-text" itemprop="text">
<p>It looks like you want a CSV (comma separated value) file as your output. You can use the <a href="https://docs.python.org/3/library/csv.html" rel="noreferrer">CSV</a> python module.</p>
<p>An example could look like:</p>
<pre><code>import csv

data = [1,2,3,4,5]

with open('test.csv', 'w') as csvfile:
    writer = csv.writer(csvfile, delimiter=",")
    writer.writerow(data)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>join</code> method</p>
<pre><code>with open ('data.txt', 'w') as fo:
     fo.write(','.join(str(i) for i in data))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The simplest way is to convert the list to a joined string, then write:</p>
<pre><code>str_data = ','.join(str(i) for i in data)
with open('data.txt', 'w') as fo:
    fo.write(str_data)
</code></pre>
<p>In case if the list is too long to convert it all to the string:</p>
<pre><code>with open('data.txt', 'w') as fo:
    for idx, item in enumerate(data):
        if idx:  # don't need a comma before the list
            fo.write(',')
        fo.write(item)
</code></pre>
<p>Update: that all said, aus_lacy's suggestion about CSV seems the best.</p>
</div>
