<p>I'm trying to code a program that will print the total price after a discount. If the price <code>paid</code> is over <code>100</code>, it will knock off <code>20%</code>. At the end of the program, I receive this error:</p>
<pre><code>if paid&gt;=100:
    TypeError: unorderable types: str() &gt;= int()
</code></pre>
<p>Here is my Code:</p>
<pre><code>paid=input('How much did you pay? ')
if paid&gt;=100:
    actualPay=100*1.25 #20% off total, 100/0.80 = 1.25
    print(actualPay)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Cast the <code>input</code> to an <code>int</code></p>
<pre><code>paid=int(input('How much did you pay? '))
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html#input"><code>input</code></a> always returns a string. You need to explicitly cast it.</p>
<p>Ex:</p>
<pre><code>paid=int(input('How much did you pay? '))
if paid&gt;=100:
    actualPay=100*1.25
    print(actualPay)
</code></pre>
<p>A better way would have a <code>try</code> and <code>except</code> block.</p>
<pre><code>try:
    paid=int(input('How much did you pay? '))
    if paid&gt;=100:
        actualPay=100*1.25
        print(actualPay)    
except TypeError:
        print "Please enter a valid input."
</code></pre>
