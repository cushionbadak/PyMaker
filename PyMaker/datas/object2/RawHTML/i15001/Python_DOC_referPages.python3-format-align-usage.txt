<div class="post-text" itemprop="text">
<p>Your format is easily translated to the <code>str.format()</code> <a href="https://docs.python.org/3/library/string.html#formatstrings" rel="nofollow">formatting syntax</a>:</p>
<pre><code>print("{:&gt;3s} {:20s} {:&gt;12s}".format("Id", "State", "Population"))
print("{id:3d} {name:20s} {population:12d}".format(**state))
</code></pre>
<p>Note that left-alignment is achieved by prefixing the width with <code>&lt;</code>, not <code>-</code>, and default alignment for strings is to left-align, so a <code>&gt;</code> is needed for the header strings and the <code>&lt;</code> can be omitted, but otherwise the formats are closely related.</p>
<p>This extracts the values directly from the <code>state</code> dictionary by using the keys in the format itself.</p>
<p>You may as well just use the actual output result of the first format directly:</p>
<pre><code>print(" Id State                  Population")
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; state = {'id': 15, 'name': 'New York', 'population': 19750000}
&gt;&gt;&gt; print("{:&gt;3s} {:20s} {:&gt;12s}".format("Id", "State", "Population"))
 Id State                  Population
&gt;&gt;&gt; print("{id:3d} {name:20s} {population:12d}".format(**state))
 15 New York                 19750000
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can write:</p>
<pre><code>print("{id:&gt;3s} {state:20s} {population:&gt;12s}".format(id='Id', state='State', population='Population'))
print("{id:&gt;3d} {state:20s} {population:&gt;12d}".format(id=state['id'], state=state['name'], population=state['population']))
</code></pre>
<p>Note that you have to use <code>&gt;</code> to right-align as the items are left-aligned by default. You can also name the items in the formatted string which makes it more readable to see what value goes where.</p>
