<p>Apologies if this has already been asked.  Is there a way in Python 3x to search for a whole word in a string and return its starting index?</p>
<p>Any help would be greatly appreciated.</p>

<div class="post-text" itemprop="text">
<p>Yes, with a regex and <a href="http://www.regular-expressions.info/wordboundaries.html" rel="nofollow">word boundary anchors</a>:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; s = "rebar bar barbed"
&gt;&gt;&gt; regex = re.compile(r"\bbar\b")
&gt;&gt;&gt; for match in regex.finditer(s):
...     print(match.group(), match.start(), match.end())
...
bar 6 9
</code></pre>
<p>The <code>\b</code> anchors make sure that only entire words can match. If you're dealing with non-ASCII words, use <code>re.UNICODE</code> to compile the regex, otherwise <code>\b</code> won't work as expected, at least not in Python 2.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you just want the first occurrence, you can use <a href="https://docs.python.org/2/library/re.html#re.RegexObject.finditer" rel="nofollow">re.finditer</a> and next.</p>
<pre><code>s =  "foo  bar foobar"
import re

m = next(re.finditer(r"\bfoobar\b",s),"")
if m:
   print(m.start())
</code></pre>
<p>Or as @Tim Pietzcker commented use <code>re.search</code>:</p>
<pre><code>import re
m = re.search(r"\bfoobar\b",s)
if m:
    print(m.start())
</code></pre>
