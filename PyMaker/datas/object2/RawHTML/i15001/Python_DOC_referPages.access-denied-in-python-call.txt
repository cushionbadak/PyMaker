<div class="post-text" itemprop="text">
<p>The problem is that you are not handling escape sequences properly.  <code>\b</code> denotes a backspace character and <code>\f</code> denotes a formfeed:</p>
<pre><code>&gt;&gt;&gt; '\b'
'\x08'
&gt;&gt;&gt; '\f'
'\x0c'
&gt;&gt;&gt;
</code></pre>
<p>This makes your paths invalid.  To fix the problem, simply use a <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow">raw-string</a>:</p>
<pre><code>call([r'"C:\Program Files\TortoiseSVN\bin\svn.exe" propset svn:ignore "C:\dir\file.dcu" "C:\dir"'])
</code></pre>
<p>By prefixing the string literal with <code>r</code>, we tell Python to not interpret escape sequences.</p>
