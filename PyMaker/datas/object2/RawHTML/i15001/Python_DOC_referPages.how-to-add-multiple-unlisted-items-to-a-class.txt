<p>This may be a very simple question and has probably been asked before, but it's probably an issue of me <a href="http://www.reddit.com/r/learnprogramming/comments/2w3byo/self_taught_professionals_what_were_your_areas_of/conabi0" rel="nofollow">lacking vocabulary</a>, so I apologize in advance.</p>
<p>From <a href="https://docs.python.org/2/tutorial/classes.html" rel="nofollow">the python documentation</a>:</p>
<pre><code>class Dog:
    def __init__(self, name):
        self.name = name
        self.tricks = []    # creates a new empty list for each dog
    def add_trick(self, trick):
        self.tricks.append(trick)

d = Dog('Fido')
d.add_trick('roll over')
d.add_trick('fetch')
d.tricks
</code></pre>
<p>This returns, as expected:</p>
<pre><code>['roll over', 'fetch']
</code></pre>
<p>But if I then run:</p>
<pre><code>d.add_trick('play dead', 'high five')
</code></pre>
<p>I get an error, so I tried:</p>
<pre><code>d.add_trick(['play dead', 'high five'])
</code></pre>
<p>But that gives me:</p>
<pre><code>d.tricks
['roll over', 'fetch', ['play dead', 'high five']]
</code></pre>
<p>When I want:</p>
<pre><code>['roll over', 'fetch', 'play dead', 'high five']
</code></pre>
<p>How can I unlist a list of items added to a class?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>extend</code>:</p>
<pre><code>d.tricks.extend(['play dead', 'high five'])

d.tricks
['roll over', 'fetch', 'play dead', 'high five']
</code></pre>
<p>Or you could change <code>add_tricks</code> to:</p>
<pre><code>def add_trick(self, *trick):
    self.tricks.extend(trick)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can redefine the <code>Dog.add_trick</code> method to take a variable number of arguments and then add them all to the list using <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow"><code>list.extend</code></a>:</p>
<pre><code>def add_trick(self, *tricks):
    self.tricks.extend(tricks)
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; class Dog:
...     def __init__(self, name):
...         self.name = name
...         self.tricks = []
...     def add_trick(self, *tricks):
...         self.tricks.extend(tricks)
...
&gt;&gt;&gt; d = Dog('Fido')
&gt;&gt;&gt; d.add_trick('roll over')
&gt;&gt;&gt; d.add_trick('fetch')
&gt;&gt;&gt; d.tricks
['roll over', 'fetch']
&gt;&gt;&gt; d.add_trick('play dead', 'high five')
&gt;&gt;&gt; d.tricks
['roll over', 'fetch', 'play dead', 'high five']
&gt;&gt;&gt;
</code></pre>
<p>More on the <code>*tricks</code> syntax can be found under <a href="https://docs.python.org/3/reference/compound_stmts.html#function-definitions" rel="nofollow">Function definitions</a> in the documentation.</p>
