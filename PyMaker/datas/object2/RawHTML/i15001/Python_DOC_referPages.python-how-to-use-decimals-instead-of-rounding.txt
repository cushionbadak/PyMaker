<div class="post-text" itemprop="text">
<p>Instead of <code>int(...)</code> use either <code>float(...)</code> or <code>decimal.Decimal(...)</code>.</p>
<p>Example:</p>
<pre><code>import decimal
d = decimal.Decimal(raw_input("\nEnter a number &gt;"))
print(d)
</code></pre>
<p>or:</p>
<pre><code>f = float(raw_input("\nEnter a number &gt;"))
print(f)
</code></pre>
