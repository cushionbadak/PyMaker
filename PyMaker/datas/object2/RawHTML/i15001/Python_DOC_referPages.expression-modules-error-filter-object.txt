<p>Quoting the <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow"><strong>documentation</strong></a></p>
<blockquote>
<p>Construct an iterator from those elements of iterable for which function returns true. iterable may be either a sequence, a container which supports iteration, or an iterator. If function is None, the identity function is assumed, that is, all elements of iterable that are false are removed.</p>
<p>Note that filter(function, iterable) is equivalent to the generator expression <code>(item for item in iterable if function(item))</code> if function is not None and <code>(item for item in iterable if item)</code> if function is None.</p>
</blockquote>
<p>To get thoses elements you need a loop or use <code>next</code></p>
<pre><code>&gt;&gt;&gt; next(filter(lambda x: re.match(r'A', x), s))
'ALLIGATOR'
</code></pre>
<p>or</p>
<pre><code> &gt;&gt;&gt; [word for word in filter(lambda x: re.match(r'A', x), s)]
 ['ALLIGATOR']
</code></pre>
