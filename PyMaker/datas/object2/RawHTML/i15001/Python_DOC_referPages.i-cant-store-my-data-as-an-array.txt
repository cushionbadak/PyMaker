<div class="post-text" itemprop="text">
<p>If you're using the <a href="https://docs.python.org/3/library/array.html" rel="nofollow">array class</a>, the constructor takes two arguments, the first being a character representing a type code.</p>
<p>So you want</p>
<pre><code>from array import array
out = &lt;...&gt;
self.data = array('b', out)
</code></pre>
<p>Or one of the other type codes listed in the docs instead of <code>'b'</code>.</p>
<p>Or you can use <a href="http://docs.scipy.org/doc/numpy/reference/generated/numpy.array.html" rel="nofollow">numpy.array</a>, in which case your code should work as-is:</p>
<pre><code>from numpy import array
out = &lt;...&gt;
self.data = array(out)
</code></pre>
