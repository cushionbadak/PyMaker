<p>The built-in function <a href="https://docs.python.org/3/library/functions.html#chr" rel="nofollow"><code>chr()</code></a> will convert an ascii code to the corresponding character.</p>
<p>Example:</p>
<pre><code>&gt;&gt;&gt; chr(65)
'A'
