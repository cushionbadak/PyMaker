<div class="post-text" itemprop="text">
<p>The magic of <code>~</code> is a shell feature that's not available here, so you have to write out the path in full.</p>
<p>In this case:</p>
<pre><code>sys.path.insert(0, '/root/Boost-Pretty-Printer')
</code></pre>
<p>Or, more flexibly, using <a href="https://docs.python.org/3/library/os.path.html#os.path.expanduser" rel="nofollow"><code>os.path.expanduser</code></a> to replace <code>~</code> with the user's home directory:</p>
<pre><code>sys.path.insert(0, os.path.expanduser('~/Boost-Pretty-Printer'))
</code></pre>
</div>
<span class="comment-copy">You could also use <a href="https://docs.python.org/3/library/os.path.html#os.path.expanduser" rel="nofollow noreferrer"><code>os.path.expanduser</code></a> to replace <code>~</code> with the user's home directory: <code>sys.path.insert(0, os.path.expanduser('~/Boost-Pretty-Printer'))</code>.</span>
