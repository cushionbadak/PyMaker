<div class="post-text" itemprop="text">
<p><code>.split()</code> returns a list, you can call the function <code>len()</code> on that list to get how many items `.split()' returns:</p>
<pre><code>&gt;&gt;&gt; s = 'one two three'
&gt;&gt;&gt; s.split()
['one', 'two', 'three']
&gt;&gt;&gt; lst = s.split()
&gt;&gt;&gt; len(lst)
3
</code></pre>
