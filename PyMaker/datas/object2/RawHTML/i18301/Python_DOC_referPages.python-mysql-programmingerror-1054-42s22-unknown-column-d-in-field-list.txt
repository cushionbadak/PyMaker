<div class="post-text" itemprop="text">
<p>There are two problems, the first is that you are never creating a <code>date</code> object. Given your instructions (Ano/mês/dia) this would be something like</p>
<pre><code>input_date = datetime.strptime(input_string, '%Y/%m/%d')
</code></pre>
<p>The second problem is that</p>
<pre><code>now = time.strftime('vb')
</code></pre>
<p>sets <code>now</code> to <code>'vb'</code> because it has no strftime directives in it. If you are looking for a standard time format, it is hard to go wrong with <a href="http://docs.python.org/3/library/datetime.html#datetime.date.isoformat" rel="nofollow">date.isoformat()</a> which would make</p>
<pre><code>now = input_date.isoformat()
</code></pre>
