<div class="post-text" itemprop="text">
<p>Dividing integer by integer (as you do with <code>n/H</code> inside <code>math.asin(n/H)</code>) returns an integer (which is the floor value of the division result) in Python before 3.0. You must either convert at least one operand to <code>float</code> or declare at the beginning of the Python source file</p>
<p><code>from __future__ import division</code></p>
<p>Read <a href="http://python.org/dev/peps/pep-0238" rel="nofollow">http://python.org/dev/peps/pep-0238</a> for details</p>
</div>

<h1>Code Corrections</h1>
<p>There are a few problems:</p>
<ol>
<li><p>There is no built-in <code>sin</code>, it needs to be used from <a href="https://docs.python.org/3/library/math.html" rel="nofollow noreferrer"><code>math</code></a> as <a href="https://docs.python.org/3/library/math.html#math.sin" rel="nofollow noreferrer"><code>math.sin</code></a></p>
<pre><code>---&gt; 23 Fy = -p*g*R*(((math.asin(n/H))/2.0)-sin((2*math.asin(n/H))/4.0))

NameError: name 'sin' is not defined
</code></pre></li>
<li><p>There is <a href="https://stackoverflow.com/a/10768737/2087463">zero division</a>. </p>
<pre><code>---&gt; 26 Direction = math.atan(Fy/Fx)

ZeroDivisionError: float division by zero
</code></pre>
<p>This originates from <code>n/H</code> integer division as @MichaelButscher <a href="https://stackoverflow.com/a/19053083/2087463">states</a>.</p></li>
<li><p>There is no <code>sqrt</code> built-in, also use <a href="https://docs.python.org/3/library/math.html#math.sqrt" rel="nofollow noreferrer"><code>math.sqrt</code></a></p>
<pre><code>---&gt; 30 Magnitude = sqrt(Fx*Fy + Fy*Fy)

NameError: name 'sqrt' is not defined
</code></pre></li>
<li><p>If the inputs are <code>float</code>s they are truncated to <code>int</code>s, this seem unnecessary.</p></li>
</ol>
<p>Final corrected code:</p>
<pre><code>import math
from decimal import *
from __future__ import division

H = 10.0 # raw_input("Please enter hight:")
H = float(H)
m = 0.2 # raw_input("Please enter crest thickness:")
m = float(m)
n = 2.0 # raw_input("Please enter base thikness:")
n = float(n)

Fx = 0.0
Fy = 0.0
Magnitude = 0.0
Direction = 0.0

p = 1000 #dencity
g = 9.81 #gravity

R = math.sqrt(H*H + n*n)
#Force in x direction
Fx = (-p*g*m*(H*H))/2.0
#Force in y direction
Fy = -p*g*R*(((math.asin(n/H))/2.0)-math.sin((2*math.asin(n/H))/4.0))

#Overall force
Direction = math.atan(Fy/Fx)
Magnitude = math.sqrt(Fx*Fy + Fy*Fy)

print ("The force magnitude is", Magnitude)
print ("The force direction is", Direction)
</code></pre>
