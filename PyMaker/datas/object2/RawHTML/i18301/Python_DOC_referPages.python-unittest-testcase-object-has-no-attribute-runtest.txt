<p>For the following code:</p>
<pre><code>import unittest

class Test(unittest.TestCase):
    def test1(self):
        assert(True == True)

if __name__ == "__main__":
    suite = unittest.TestSuite()
    suite.addTest(Test())
    unittest.TextTestRunner().run(suite)
</code></pre>
<p>Using Python 3 to execute it, the following error is raised:</p>
<pre><code>Traceback (most recent call last):

<div class="post-text" itemprop="text">
<p>You need to invoke a <a href="https://docs.python.org/3.7/library/unittest.html#unittest.TestLoader" rel="nofollow noreferrer"><code>TestLoader</code></a>:</p>
<pre><code>if __name__ == "__main__":
    suite = unittest.defaultTestLoader.loadTestsFromTestCase(Test)
    unittest.TextTestRunner().run(suite)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have to specify the test method name (<code>test1</code>):</p>
<pre><code>import unittest

class Test(unittest.TestCase):
    def test1(self):
        assert(True == True)

if __name__ == "__main__":
    suite = unittest.TestSuite()
    suite.addTest(Test('test1')) # &lt;----------------
    unittest.TextTestRunner().run(suite)
</code></pre>
<p>Or, if you want to run all tests in the file, Just calling <a href="http://docs.python.org/3/library/unittest.html#unittest.main"><code>unittest.main()</code></a> is enough:</p>
<pre><code>if __name__ == "__main__":
    unittest.main()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The actual test for any <code>TestCase</code> subclass is performed in the <code>runTest()</code> method.  Simply change your code to:</p>
<pre><code>class Test(unittest.TestCase):
    def runTest(self):
        assert(True == True)
</code></pre>
