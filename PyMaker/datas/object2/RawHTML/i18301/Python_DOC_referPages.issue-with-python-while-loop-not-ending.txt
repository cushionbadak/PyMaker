<div class="post-text" itemprop="text">
<p>If you are using Python 3.x, <a href="http://docs.python.org/3/library/functions.html#input" rel="nofollow"><code>input()</code></a> returns a string object.</p>
<p>Comparing string object with int always return False.</p>
<pre><code>&gt;&gt;&gt; '1' == 1
False
</code></pre>
<p>You should convert string object to int before compare them:</p>
<pre><code>b = int(b)
</code></pre>
<p>BTW, it's better to use following instead of <code>while</code> loop:</p>
<pre><code>for i in range(int(b)):
    ...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your problem is that <code>input()</code> on Python 3 returns a string, so <code>b</code> will be a string, and therefore <code>y == b</code> will never be <code>True</code>.</p>
<p>Use </p>
<pre><code>b = int(input('Enter number of beeps required'))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Change</p>
<pre><code>b=input ('Enter number of beeps required')
</code></pre>
<p>to </p>
<pre><code>b=int(input ('Enter number of beeps required'))
</code></pre>
<p>You are reading <code>b</code> as string and comparing it against an <code>int</code> in</p>
<pre><code>if y == b:
</code></pre>
<p>which will never be <code>True</code>. Thats why your code is infinitely looping.</p>
</div>
