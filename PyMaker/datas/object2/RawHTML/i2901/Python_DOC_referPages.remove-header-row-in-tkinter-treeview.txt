<p>Can anyone tell me how to remove the header row in a tkinter Treeview?</p>
<pre><code>from tkinter import *
from tkinter import ttk

root = Tk()

NewTree= ttk.Treeview(root)
NewTree.pack()
NewTree.heading("#0", text="How to remove this row?")
NewTree.insert("", "0", 'item1',text='Item number 1')

root.mainloop()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use the <code>show</code> option to only show the tree and not the heading:</p>
<pre><code>NewTree = ttk.Treeview(root, show="tree")
</code></pre>
<hr/>
<p><strong>Relevant documentation</strong></p>
<p>From <a href="https://docs.python.org/3/library/tkinter.ttk.html" rel="nofollow noreferrer">docs.python.org</a>:</p>
<blockquote>
<p>show   </p>
<p>A list containing zero or more of the following values, specifying which elements of the tree to display.</p>
<ul>
<li>tree: display tree labels in column #0.</li>
<li>headings: display the heading row.</li>
</ul>
<p>The default is “tree headings”, i.e., show all elements.</p>
<p><strong>Note</strong>: Column #0 always refers to the tree column, even if show=”tree” is not specified.</p>
</blockquote>
<p>From the <a href="http://infohost.nmt.edu/tcc/help/pubs/tkinter/web/ttk-Treeview.html" rel="nofollow noreferrer">New Mexico Tech Tkinter reference</a>:</p>
<blockquote>
<p>show</p>
<p>To suppress the labels at the top of each column, specify <code>show='tree'</code>. The default is to show the column labels.</p>
</blockquote>
<p>From <a href="https://tkdocs.com/tutorial/tree.html" rel="nofollow noreferrer">TkDocs</a>:</p>
<blockquote>
<p>You can optionally hide one or both of the column headings or the tree itself (leaving just the columns) using the <code>show</code> widget configuration option (default is <code>"tree headings"</code> to show both).</p>
</blockquote>
