<p>It seems Python automatically takes the value type as "String"</p>

<p>No. The value type <strong>is</strong> string, until you convert it to something else. There is no secret magic that guesses possible variable types from the user's input.</p>
<p>There is a <a href="https://docs.python.org/3/library/stdtypes.html#str.isdigit" rel="nofollow noreferrer">helper function <code>isdigit</code></a> you can use to check if a string only consists of digits.</p>
<pre><code>response = input("enter the string: ")

if str.isdigit(response):
    ...
</code></pre>
