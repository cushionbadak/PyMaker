<div class="post-text" itemprop="text">
<p>Look at the <a href="https://docs.python.org/3/library/random.html?#real-valued-distributions" rel="nofollow noreferrer">Real-valued distributions</a> in the <code>random</code> module, particularly <code>random.expovariate</code>.</p>
<p>You can make it into a generator using <code>yield</code> e.g.</p>
<pre><code>import random

def exponential(lambd):
    while True:
        yield random.expovariate(lambd)
</code></pre>
<p>Then you can take the first 1000 using <code>itertools.islice</code>.</p>
<pre><code>from itertools import islice

print(list(islice(exponential(1/0.3), 1000)))
</code></pre>
<p>If you want to truncate the distribution at a certain interval, you can filter it out first.</p>
<pre><code>from itertools import filterfalse

print(list(islice(filterfalse(lambda x: not 0.05 &lt;= x &lt;=0.30, exponential(1/0.3)), 1000)))
</code></pre>
<p>Put it all together,</p>
<pre><code>def FunctionExponentialPDF(minimum, maximum, scale, size):
    return islice(filterfalse(lambda x: not minimum &lt;= x &lt;= maximum,
                              exponential(1/scale)),
                  size)
</code></pre>
