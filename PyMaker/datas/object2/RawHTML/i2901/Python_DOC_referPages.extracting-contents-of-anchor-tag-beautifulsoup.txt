<div class="post-text" itemprop="text">
<p>You can use the <code>text</code> attribute:</p>
<pre><code>&gt;&gt;&gt; import requests
&gt;&gt;&gt; page = requests.get('https://fasttrack.grv.org.au/RaceField/ViewRaces/338181522')
&gt;&gt;&gt; from bs4 import BeautifulSoup
&gt;&gt;&gt; soup = BeautifulSoup(page.content, 'html.parser')
&gt;&gt;&gt; print([ i.text for i in soup.select("div#DogName_1_1 a")])
['MERCURY RISING [M]']
</code></pre>
<p>If it is just one element you are interested in, you may want to do something like:</p>
<pre><code>&gt;&gt;&gt; soup.find(id='DogName_1_1').text
'\nMERCURY RISING [M]\n'
</code></pre>
<p>If you want to get rid of the trailing/leading <code>\n</code>s you can use <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>strip</code></a>:</p>
<pre><code>&gt;&gt;&gt; soup.find(id='DogName_1_1').text.strip()
'MERCURY RISING [M]'
</code></pre>
