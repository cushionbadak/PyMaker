<p>Let's say you have a list, which contains a single string as:</p>
<pre><code>listExample = ['cat, dog, mouse, elephant']
</code></pre>
<p>So printing this list return the values as a single string:</p>
<pre><code>&gt;&gt;&gt;'cat, dog, mouse, elephant'
</code></pre>
<p>How do you get this string to the point where you can get the entries as multiple strings, such as:</p>
<pre><code>&gt;&gt;&gt;print(anotherList)
&gt;&gt;&gt;['cat', 'dog', 'mouse', 'elephant']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are describing the basic usage of <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>str.split</code></a>:</p>
<pre><code>&gt;&gt;&gt; listExample = ['cat, dog, mouse, elephant']
&gt;&gt;&gt; listExample[0].split(', ')
['cat', 'dog', 'mouse', 'elephant']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You should use the split method.
So do</p>
<pre><code>'cat, dog, mouse, elephant'.split(', ')
</code></pre>
<p>this will give you</p>
<pre><code>['cat', 'dog', 'mouse', 'elephant']
</code></pre>
<p>The <code>', '</code> means the string should be split every time there is a comma and then a space.</p>
<p>The list is not needed, but if you absolutely have to use the list then do</p>
<pre><code>listExample[0].split(', ')
</code></pre>
<p>to get the string out of the list and then split it.</p>
<p>For more information see <a href="https://www.tutorialspoint.com/python/string_split.htm" rel="nofollow noreferrer">this</a></p>
</div>
