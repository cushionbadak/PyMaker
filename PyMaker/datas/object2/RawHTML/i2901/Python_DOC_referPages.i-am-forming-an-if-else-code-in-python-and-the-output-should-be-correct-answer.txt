<div class="post-text" itemprop="text">
<p>The input function returns the entered text as a string, see <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer">the API documentation</a>:</p>
<blockquote>
<p>If the prompt argument is present, it is written to standard output without a trailing newline. The function then reads a line from input, converts it to a string (stripping a trailing newline), and returns that. </p>
</blockquote>
<p>Then your code is comparing the string "21" to an integer value 21, comparing a string with an int will evaluate to false.</p>
<p>You could convert the string to an integer value (you can use <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow noreferrer">int</a>):</p>
<pre><code>user_answer = input("What is 8+13")
answer=21
if int(user_answer) == answer:
    print("Correct answer!")
else:
    print("Try again!")
</code></pre>
<p>but this will raise an error if the input can't be converted to an int value. Alternatively convert the int to a string:</p>
<pre><code>user_answer = input("What is 8+13")
answer=21
if user_answer == str(answer):
    print("Correct answer!")
else:
    print("Try again!")
</code></pre>
</div>
