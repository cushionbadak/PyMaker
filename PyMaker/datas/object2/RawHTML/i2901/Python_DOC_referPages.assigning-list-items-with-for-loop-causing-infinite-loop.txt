<div class="post-text" itemprop="text">
<p>You need a little less loop:</p>
<pre><code>zip_code_file = open('zip_codes.txt')
zip_codes = []
latitude = []
longitude = []
for line in zip_code_file:
    zipcode, lat, lng = line.strip().split(',')
    zip_codes.append(zipcode)
    latitude.append(lat)
    longitude.append(lng)
print(zip_codes)
</code></pre>

<p>Files of comma separated values should be processed with the <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer"><code>csv</code></a> module</p>
<pre><code>import csv

with open('zip_codes.txt') as f:
    r = csv.reader(f)
    next(r)  # skip the header
    zip_codes, latitudes, longitudes = zip(*r)
</code></pre>
