<p>I've used <code>locals()</code> function and created some variables. Now I need to fetch all the variables created into a list, how should I do?</p>
<p>example:</p>
<pre><code>column_list=[x for x in range(100)]

for i,j in enumerate(column_list):
    locals()['v'+str(i)]=pd.DataFrame(['Y','N'],columns=[j])
</code></pre>
<p>Now, I have variables, v0 ~ v99, each of which is an independent data frame.</p>
<p>I need to fetch all the variables into a <code>variable_list</code></p>
<p>So I can use for loop on all the variables:</p>
<pre><code>variable_list -&gt; [v0,v1,v2,v3,v4.v5.......]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Level up the abstraction.  Writing to <a href="https://docs.python.org/3/library/functions.html#locals" rel="nofollow noreferrer"><code>locals()</code></a> is not reliable, and is advised against in the documentation.</p>
<p>You should change the code where it creates sequential local variables named <code>v0</code>, <code>v1</code>, ... <code>v99</code> so that appends into one list named <code>v</code> in the first place.  </p>
<p>Now, your "variable_list" is just <code>v</code> itself and the values in the list are accessible <code>v[0]</code>, <code>v[1]</code>, ... <code>v[99]</code>.  </p>
</div>
