<div class="post-text" itemprop="text">
<p><code>threading.Timer</code> takes <a href="https://docs.python.org/3/library/threading.html#timer-objects" rel="nofollow noreferrer">arguments</a> <code>(interval, function, args=None, kwargs=None)</code> so you're passing the arg <code>self.f_stop</code> to <code>self.stream_changes</code> which takes one argument <code>self</code> (which is automatically passed) but you're passing <code>self.f_stop</code> in addition.</p>
