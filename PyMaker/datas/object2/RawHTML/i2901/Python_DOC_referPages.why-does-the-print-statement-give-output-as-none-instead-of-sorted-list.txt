<pre><code>import random
L = []
for i in range(0, 10):
    L.append(random.randrange(1, 10))
print(L.sort())
</code></pre>
<hr/>
<pre><code>Output:None
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>list.sort</code> sorts the list in-place and returns <code>None</code>. You either want to do:</p>
<pre><code>L.sort()
print(L)
</code></pre>
<p>or use the <code>sorted</code> function.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>list.sort</code> mutates the list instead of returning it.</p>
<p>You can use the <code>sorted()</code> function instead:</p>
<pre><code>print(sorted(L))
</code></pre>
</div>
<span class="comment-copy">Because <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer"><code>list.sort()</code></a> doesn't return anything (i.e. <code>None</code>) - as per the official documentation, the call to the <code>sort()</code> method just sorts the list in-place.</span>
