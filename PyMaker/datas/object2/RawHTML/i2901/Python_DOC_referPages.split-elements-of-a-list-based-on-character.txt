<div class="post-text" itemprop="text">
<p>You can use a <a href="https://www.pythonforbeginners.com/lists/list-comprehensions-in-python/" rel="nofollow noreferrer">list-comprehension</a> with the standard scheme for <a href="https://stackoverflow.com/questions/952914/making-a-flat-list-out-of-list-of-lists-in-python">flattening arrays</a>:</p>
<pre><code>lst = ['Agrigento',
 ' Tal Andar, Regie Curie',
 ' Landar ta il Geuenie',
 ' Landar ta il Guedi,  Casali Bisibud',
 'Aragonia',
 'Athene']

lst = [x for y in lst for x in y.split(',')]
print(lst)  # -&gt; ['Agrigento', ' Tal Andar', ' Regie Curie', ' Landar ta il Geuenie', ' Landar ta il Guedi', '  Casali Bisibud', 'Aragonia', 'Athene']
</code></pre>
<hr/>
<p>Note: We are taking advantage of the fact that splitting a string that does not contain the character used as a delimiter returns the same string in a list:</p>
<pre><code>&gt;&gt; 'a'.split('b')
['a']
</code></pre>
<p>As a result, the flattening meets no obstacles. For more info, see <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>split</code></a>.</p>
<hr/>
<p>Finally, as @tobias says, if you want to get rid of the spaces in the front and back of the names, you can use the <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>strip()</code></a> method. Simply modify the above comprehension to the one given below.</p>
<pre><code>lst = [x.strip() for y in lst for x in y.split(',')]
print(lst)  # -&gt; ['Agrigento', 'Tal Andar', 'Regie Curie', 'Landar ta il Geuenie', 'Landar ta il Guedi', 'Casali Bisibud', 'Aragonia', 'Athene']
</code></pre>

<div class="post-text" itemprop="text">
<pre><code>elements = ['Agrigento', ' Tal Andar, Regie Curie', ' Landar ta il Geuenie', ' Landar ta il Guedi,  Casali Bisibud', 'Aragonia', 'Athene']
result = []

for elem in elements:
    if ',' in elem:
        result.extend([s for s in element.split(',')])
    else:
        result.append(elem)

print(result)
</code></pre>
