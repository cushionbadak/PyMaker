<p>I have this array of arrays in python.</p>
<pre><code>staff = [
            ['staff1', '2/18/1998', 21.63],
            ['staff2', '7/7/1999',  15.87],
            ['staff3', '8/26/2004', 123.46],
        ]
</code></pre>
<p>I have this array which specify the columns.</p>
<pre><code>out_columns = ['Name','Date','Profit']
</code></pre>
<p>Based on these 2 arrays, I would like to convert them into an array of dictionaries that look like this;</p>
<pre><code>dict_arr = [
                {"Name":"staff1",
                 "Date":"2/18/1998",
                 "Profit":21.63
                 },
                {"Name": "staff2",
                 "Date": "7/7/1999",
                 "Profit": 15.87
                 },
                {"Name": "staff3",
                 "Date": "8/26/2004",
                 "Profit": 123.46
                 },
]
</code></pre>
<p>I am using python v3.6</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip</a> function :</p>
<pre><code>nl = []
for zz in staff:
    aa = {}
    for key,val in zip(out_columns,zz):
        aa[key]= val
    nl.append( aa )
print(nl) # [{'Name': 'staff1', 'Date': '2/18/1998', 'Profit': 21.63} , ... ]
</code></pre>
