<div class="post-text" itemprop="text">
<p>Your regexp is mostly correct but some characters are excessively escaped in the character class (for example, <code>|</code> doesn't have to be escaped when used inside the class).</p>
<p>You need to use <code>search</code> method instead of <code>match</code>. There's a subtle difference between <code>search</code> and <code>match</code>: <a href="https://docs.python.org/3/library/re.html#re.match" rel="nofollow noreferrer">https://docs.python.org/3/library/re.html#re.match</a></p>
<pre><code>In [1]: import re

In [2]: r = re.compile('\.[\^$*+?{}\[\]|()]')

In [4]: bool(r.search("foo.*bar"))
Out[4]: True

In [5]: bool(r.search("foo.bar"))
Out[5]: False
</code></pre>
<p>Also, it is better to start learning with Python 3 â€” version 2 is obsolete.</p>
