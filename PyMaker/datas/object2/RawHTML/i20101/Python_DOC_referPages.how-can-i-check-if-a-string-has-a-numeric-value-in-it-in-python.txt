<div class="post-text" itemprop="text">
<p>Use the <a href="http://docs.python.org/library/stdtypes.html#str.isdigit" rel="noreferrer"><code>.isdigit()</code> method</a>:</p>
<pre><code>&gt;&gt;&gt; '123'.isdigit()
True
&gt;&gt;&gt; '1a23'.isdigit()
False
</code></pre>
<p>Quoting the documentation:</p>
<blockquote>
<p>Return true if all characters in the string are digits and there is at least one character, false otherwise.</p>
</blockquote>
<p>For <code>unicode</code> strings or Python 3 strings, you'll need to use a more precise definition and use the <a href="https://docs.python.org/2/library/stdtypes.html#unicode.isdecimal" rel="noreferrer"><code>unicode.isdecimal()</code></a> / <a href="https://docs.python.org/3/library/stdtypes.html#str.isdecimal" rel="noreferrer"><code>str.isdecimal()</code></a> instead; not all Unicode digits are interpretable as decimal numbers. <a href="http://codepoints.net/U+00B2" rel="noreferrer">U+00B2 SUPERSCRIPT 2</a> is a digit, but not a decimal, for example.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can always <code>try</code> it:</p>
<pre><code>try:
   a = int(yourstring)
except ValueError:
   print "can't convert"
</code></pre>
<p>Note that this method outshines <code>isdigit</code> if you want to know if you can convert a string to a floating point number using <code>float</code></p>
