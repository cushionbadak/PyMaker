<div class="post-text" itemprop="text">
<p>I have below String.</p>
<pre><code>a = "/root/LOAD/LOGS/2018-06-15_13:54/"
</code></pre>
<p>How can I get the output:</p>
<pre><code>"/root/LOAD/LOGS/"
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>os.path.dirname</code> to go up a directory as many times as you need to</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; a = "/root/LOAD/LOGS/2018-06-15_13:54/"
&gt;&gt;&gt; os.path.dirname(a)
'/root/LOAD/LOGS/2018-06-15_13:54'
&gt;&gt;&gt; os.path.dirname(os.path.dirname(a))
'/root/LOAD/LOGS'
</code></pre>
<p>This is easy to wrap in a helper function if you want to move up a known number of directories</p>
<pre><code>def parent_dir(path, n):
    if n &gt; 1:
        return parent_dir(os.path.dirname(path), n-1)
    return os.path.dirname(path)
</code></pre>
<p>Example</p>
<pre><code>&gt;&gt;&gt; parent_dir(a, 2)
'/root/LOAD/LOGS'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Personally, I prefer the <a href="https://docs.python.org/3/library/pathlib.html#concrete-paths" rel="nofollow noreferrer"><code>pathlib</code></a> module.</p>
<pre><code>from pathlib import Path
path = Path("/root/LOAD/LOGS/2018-06-15_13:54/")
parent = path.parent  # == Path("/root/LOAD/LOGS/")
parent_as_string = str(parent)  # == "/root/LOAD/LOGS/"
</code></pre>
<hr/>
<p>This is also repeatable</p>
<pre><code>parent_of_parent = path.parent.parent  # == Path('/root/LOAD')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>There are several methods, I would personally use the re (<code>import re</code>) module. But there is also the possibility to extract the directory without the additional module. It depends on whether you want to look at the slash (subdirectory) or the number of characters is enough (if the directory name does not change).</p>
<pre><code>&gt;&gt;&gt; a = "/root/LOAD/LOGS/2018-06-15_13:54/"    

&gt;&gt;&gt; "/".join(a.split("/")[:-2])    # from right side
'/root/LOAD/LOGS'

&gt;&gt;&gt; "/".join(a.split("/")[:4])     # from left side
'/root/LOAD/LOGS'

&gt;&gt;&gt; a[:15]
'/root/LOAD/LOGS'
</code></pre>
