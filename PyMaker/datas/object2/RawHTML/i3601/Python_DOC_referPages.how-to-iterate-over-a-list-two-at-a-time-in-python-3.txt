<p>I am trying iterate over a list, two at a time. This is my code:</p>
<pre><code>list_1 = [1,3,2,4,3,1,2,7]
</code></pre>
<p>The ouput should be like this(the iteration should start from the first element):</p>
<pre><code>1
2
3
2
</code></pre>
<p>Seven, is not there because the iteration in only 2.</p>
<p>This is my try:</p>
<pre><code>nums = [1,3,2,4,3,1,2,7]
for x, y in zip(*[iter(nums)]*2):
    print(x, y)
</code></pre>
<p>But my output is:</p>
<pre><code>1 3
2 4
3 1
2 7
</code></pre>
<p>How can I achieve the proper iteration using Python 3?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer">range</a> like this by using step (indexing):</p>
<pre><code>list_1 = [1,3,2,4,3,1,2,7]

for i in range(0,len(list_1),2):
    print(list_1[i])
</code></pre>
<p>or just using <a href="https://stackoverflow.com/questions/509211/understanding-pythons-slice-notation">python slice notation</a>:</p>
<pre><code>list_1 = [1,3,2,4,3,1,2,7]

for v in list_1[::2]:
    print(v)
</code></pre>
</div>
