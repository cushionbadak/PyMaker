<p>I have 2 dictionaries and I want to insert the values of dict1 into dict2 but in a specific place in dict2, ie:</p>
<p>dict1</p>
<pre><code>{
'apple': 'hard tasty', 'orange': 'soft tasty', 'banana': 'soft very-tasty'
}
</code></pre>
<p>dict2</p>
<pre><code>{
 'apple': '&lt;div class="apple"&gt;&lt;/div&gt;',
 'orange': '&lt;div class="orange"&gt;&lt;/div&gt;',
 'banana': '&lt;div class="banana"&gt;&lt;/div&gt;'
 }
</code></pre>
<p>I want to insert the values of dict1 into dict2 inside the 'class=' variable so it becomes</p>
<p>Desired</p>
<pre><code>{
 'apple': '&lt;div class="apple hard tasty"&gt;&lt;/div&gt;',
 'orange': '&lt;div class="orange soft tasty"&gt;&lt;/div&gt;',
 'banana': '&lt;div class="banana soft very-tasty"&gt;&lt;/div&gt;'
 }
</code></pre>
<p>Can you help me with this? I may have to import re or use regex. I've been using dict comprehensions for the iterations</p>

<div class="post-text" itemprop="text">
<p>There is no need of <code>regex</code> or any additional imports. Use <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow noreferrer"><code>str.replace</code></a> instead:</p>
<pre><code>dict1 = {
       'apple': 'hard tasty', 'orange': 'soft tasty',   
       'banana': 'soft very-tasty'
        }
dict2 = {
       'apple': '&lt;div class="apple"&gt;&lt;/div&gt;',
       'orange': '&lt;div class="orange"&gt;&lt;/div&gt;',
       'banana': '&lt;div class="banana"&gt;&lt;/div&gt;'
        }

d = {k: v.replace(k, k+' '+dict1[k]) for k, v in dict2.items()}
print(d)

# {'apple': '&lt;div class="apple hard tasty"&gt;&lt;/div&gt;', 
#  'orange': '&lt;div class="orange soft tasty"&gt;&lt;/div&gt;', 
#  'banana': '&lt;div class="banana soft very-tasty"&gt;&lt;/div&gt;'}
</code></pre>
