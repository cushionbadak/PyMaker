<p>I have this:</p>
<pre><code> a = [['huhu', 'dow'], ['haha', 'dow'], ['haha', 'pow'], ['haha', 'dow'],  ['haha', 'mat'], ['hihi', 'dow'], ['hihi', 'pow']]
</code></pre>
<p>and I want </p>
<blockquote>
<p>[['huhu', 'dow'], ['haha', 'dow', 'pow', 'dow', 'mat'], ['hihi',
  'dow'], ['hihi', 'pow']]</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>I would recommend using a dictionary here, it is much cleaner. In particular I would recommend a <a href="https://docs.python.org/3/library/collections.html" rel="nofollow noreferrer"><strong><code>collections.defaultdict</code></strong></a>:</p>
<pre><code>from collections import defaultdict
dct = defaultdict(list)

for key, *values in a:
    dct[key].extend(values)
</code></pre>
<p>Output:</p>
<pre><code>defaultdict(list,
            {'huhu': ['dow'],
             'haha': ['dow', 'pow', 'dow', 'mat'],
             'hihi': ['dow', 'pow']})
</code></pre>
<p>If you want this as a list, it is a fairly simple comprehension:</p>
<pre><code>[[k, *v] for k, v in dct.items()]

# [['huhu', 'dow'], ['haha', 'dow', 'pow', 'dow', 'mat'], ['hihi', 'dow', 'pow']]
</code></pre>
</div>
