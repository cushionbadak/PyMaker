<div class="post-text" itemprop="text">
<p>You can just use Python's <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer">getattr</a> method. The start would be the same as yours and then </p>
<pre><code>for opt, val in options.items():
    curl.setopt(getattr(curl,opt), val)
</code></pre>
