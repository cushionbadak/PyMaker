<div class="post-text" itemprop="text">
<p>You could use the <code>json</code> module:</p>
<pre><code>import json
json_s = json.loads('{"responseStatus":"ok","responseHeader":{"now":1528734419187,"status":"ok","requestId":"Wx6i0wquZS4AAFNeStwAAABg"},"responseData":{"id":38}}')

json_s['responseData']['id'] = '38' # Or str(json_s['responseData']['id'])


print(json_s)
</code></pre>
<p>Or in this case it is a valid dictionary so you could use <code>eval()</code>:</p>
<pre><code>json_s = '{"responseStatus":"ok","responseHeader":{"now":1528734419187,"status":"ok","requestId":"Wx6i0wquZS4AAFNeStwAAABg"},"responseData":{"id":38}}'

a = eval(json_s)

a['responseData']['id'] = '38' # Or str(a['responseData']['id'])

print(a)
</code></pre>
<p>If you want to convert the dictionary in <code>json_s</code> and <code>a</code> back to a string then just use <code>str(json_s)</code> and <code>str(a)</code> respectively.</p>
<p>See <a href="https://docs.python.org/3/library/functions.html#eval" rel="nofollow noreferrer">the documentation</a> for the use of <code>eval()</code>.</p>
<p>Otherwise to keep away from <code>eval()</code> use the <code>ast</code> module.  As described here:</p>
<p><a href="https://stackoverflow.com/questions/988228/convert-a-string-representation-of-a-dictionary-to-a-dictionary">Convert a String representation of a Dictionary to a dictionary?</a></p>
