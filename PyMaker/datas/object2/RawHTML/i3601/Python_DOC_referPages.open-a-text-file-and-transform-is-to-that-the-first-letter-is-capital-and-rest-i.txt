<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/stdtypes.html#str.title" rel="nofollow noreferrer"><code>title</code></a> method on the contents of the file</p>
<pre><code>with open("lyrics.txt") as f:
    s = f.read().title()

with open("lyrics.txt", "w") as f:
    f.write(s)
</code></pre>
<p>You could fit this all into one context manager, but I find the above more readable than</p>
<pre><code>with open("lyrics.txt", "r+") as f:
    s = f.read().title()
    f.seek(0)
    f.write(s)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Is this what you want?:</p>
<pre><code>with open('file.txt','r') as f:
   newl = []
   l = f.readlines()
   for i in l:
      newl.append(str(i[0]+i[1:].lower()).strip())
with open('new.txt','w') as f2:
   for i in newl:
      f2.write(i+'\n')
</code></pre>
