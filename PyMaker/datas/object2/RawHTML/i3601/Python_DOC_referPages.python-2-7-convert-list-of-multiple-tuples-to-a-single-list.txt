<div class="post-text" itemprop="text">
<p>I have the following list of multiple tuples:</p>
<pre><code>[('1', 'Mak'), ('2', 'Sam'), ('3', 'John')]
</code></pre>
<p>Type: <code>&lt;type 'list'&gt;</code></p>
<p>Convert to:</p>
<pre><code>['1,Mak\n', '2,Sam\n', '3,John\n']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/2/library/string.html#string.Formatter" rel="nofollow noreferrer"><strong><code>string.format</code></strong></a> with a list comprehension for Python 2.6+</p>
<pre><code>['{}\n'.format(','.join(i)) for i in arr]

# ['1,Mak\n', '2,Sam\n', '3,John\n']
</code></pre>
<p>Or with Python 3.6+ using <a href="https://docs.python.org/3/whatsnew/3.6.html#whatsnew36-pep498" rel="nofollow noreferrer">formatted string literals</a></p>
<pre><code>[f"{','.join(i)}\n" for i in arr]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use list comprehensions:</p>
<pre><code>l = [('1', 'Mak'), ('2', 'Sam'), ('3', 'John')]
l = [x + ',' + y + '\n' for x, y in l]
</code></pre>
<p>so, now <code>l</code> becomes:
<code>['1,Mak\n', '2,Sam\n', '3,John\n']</code></p>
</div>
<div class="post-text" itemprop="text">
<p>Using a list comprehension. </p>
<pre><code>l = [('1', 'Mak'), ('2', 'Sam'), ('3', 'John')]
print [",".join(i)+ "\n" for i in l]
</code></pre>
<p><em>or</em></p>
<p>Using a <code>map</code> with <code>lambda</code>.</p>
<p><strong>Ex:</strong> </p>
<pre><code>l = [('1', 'Mak'), ('2', 'Sam'), ('3', 'John')]
print map(lambda x: "{0}\n".format(",".join(x)), l)
</code></pre>
</div>
