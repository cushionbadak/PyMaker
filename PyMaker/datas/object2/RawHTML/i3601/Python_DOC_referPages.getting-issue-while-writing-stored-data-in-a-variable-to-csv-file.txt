<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/csv.html#csv.csvwriter.writerow" rel="nofollow noreferrer"><code>casv.writerow()</code></a> accepts an iterable as the row. In this case it interprets the string as an iterable and its characters as respective items for each field. If you want to write the string in one field you can put it in an iterable like tuple.</p>
<pre><code>writer.writerow((data,))
</code></pre>
<p>Or you can split the data with the intended delimiter:</p>
<pre><code>writer.writerow(data.split(','))
</code></pre>
