<div class="post-text" itemprop="text">
<p>Using <strong><code>enumerate</code></strong> with <code>format</code>:</p>
<pre><code>nameList = ['j', 'a', 'm', 'e', 's']
for index, i in enumerate(nameList):
    print('{}{}'.format(index, i), end = '')

# 0j1a2m3e4s
</code></pre>
<p>Or without <code>format</code>:</p>
<pre><code>nameList = ['j', 'a', 'm', 'e', 's']
for index, i in enumerate(nameList):
    print(str(index) + i, end = '')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>We can use <a href="https://docs.python.org/3/library/itertools.html#itertools.chain.from_iterable" rel="nofollow noreferrer"><code>itertools.chain.from_iterable</code></a> to flatten the <code>enumerate</code>, then use the empty string <code>''</code> to separate those characters.</p>
<pre><code>from itertools import chain

nameList = ['j', 'a', 'm', 'e', 's']
print(*chain.from_iterable(enumerate(nameList)), sep='')
# 0j1a2m3e4s
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <code>string.join</code> statement to achieve this:</p>
<pre><code>print "".join(str(index) + i for index, i in enumerate(nameList))

&gt;&gt;&gt; 0j1a2m3e4s
</code></pre>
</div>
