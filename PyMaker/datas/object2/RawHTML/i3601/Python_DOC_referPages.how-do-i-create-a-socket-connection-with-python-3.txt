<p>You can create a connection like this:</p>
<pre><code># create an INET, STREAMing socket
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
# now connect to the web server on port 80 - the normal http port
s.connect(("www.python.org", 80))
</code></pre>
<p>Also you can get more information in the <a href="https://docs.python.org/3/howto/sockets.html" rel="nofollow noreferrer">documentation</a>.</p>
