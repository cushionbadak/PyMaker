<div class="post-text" itemprop="text">
<p>Normally this would be done by holding all of your parameters in a dictionary and then using <a href="https://docs.python.org/3/library/urllib.parse.html?highlight=urlencode#urllib.parse.urlencode" rel="nofollow noreferrer"><code>urllib.parse.urlencode()</code></a> to convert it into a suitable URL. For example:</p>
<pre><code>import urllib

sender_name = '*AX*'
message = '*Avail Lmt'

parms = {'q' : f'sender_name:{sender_name} - message:{message}', 'rows' : 213, 'start' : 0}
url = 'http://localhost:8983/solr/data/select?indent=on&amp;' + urllib.parse.urlencode(parms, quote_via=urllib.parse.quote, safe='*:')
print(url)
</code></pre>
<p>This would give you:</p>
<pre class="lang-none prettyprint-override"><code>http://localhost:8983/solr/data/select?indent=on&amp;q=sender_name:*AX*%20-%20message:*Avail%20Lmt&amp;rows=213&amp;start=0
</code></pre>
<p><code>urllib.parse.quote()</code> is used to give <code>%20</code> for a space rather than <code>+</code>.</p>
