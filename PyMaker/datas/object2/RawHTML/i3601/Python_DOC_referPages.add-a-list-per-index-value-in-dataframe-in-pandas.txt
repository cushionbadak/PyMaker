<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow noreferrer"><code>itertools.product</code></a>:</p>
<pre><code>from itertools import product

df = pd.DataFrame(index=['i1', 'i2', 'i3'])
ls = ['asd', 'qwe', 'zxc']

res = pd.DataFrame(list(product(df.index, ls)))

print(res)

</code></pre>

<div class="post-text" itemprop="text">
<p>You are just looking after a <code>cartesian product</code> and a <code>list comprehension</code> can do the job for you (need not another package). </p>
<pre><code>import pandas as pd
index = ['i1', 'i2', 'i3']
ls = ['asd', 'qwe', 'zxc']
pd.DataFrame([ [i, j] for i in df.id for j in df.ls], columns = ["id", "ls"])
    id  ls
