<p>I'm going to assume that your array looks like the following:</p>
<pre><code>input = [(1,2), (3,4), ..., (x,y)]
</code></pre>
<p>You can use the <code>sorted</code> method (built-in) to accomplish list sorting:</p>
<pre><code>x_sorted = sorted(input, key=lambda tup: tup[0])
y_sorted = sorted(input, key=lambda tup: tup[1])
</code></pre>
<p>Where the lambda will sort your variable by the first and second indices, respectively</p>
</div>
<div class="post-text" itemprop="text">
<p>Alternatively, you can use the <strong><a href="https://docs.python.org/3/library/operator.html#operator.itemgetter" rel="nofollow noreferrer"><code>itemgetter()</code></a></strong>  function as the <strong><a href="https://docs.python.org/3/howto/sorting.html#key-functions" rel="nofollow noreferrer"><code>key</code></a></strong> parameter to the <strong><a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer"><code>sorted()</code></a></strong> function:</p>
<pre><code>from operator import itemgetter

points = [(7,3), (2,1), (4,5), (9,0)]
sorted_by_x = sorted(points) # you can add the key parameter itemgetter(0) if you want
sorted_by_y = sorted(points, key=itemgetter(1))

print("Points List: {}".format(points))
print("Sorted by X Points List: {}".format(sorted_by_x))
print("Sorted by Y Points List: {}".format(sorted_by_y))
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>Points List: [(7, 3), (2, 1), (4, 5), (9, 0)]
Sorted by X Points List: [(2, 1), (4, 5), (7, 3), (9, 0)]
Sorted by Y Points List: [(9, 0), (2, 1), (7, 3), (4, 5)]
</code></pre>
