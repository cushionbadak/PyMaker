<div class="post-text" itemprop="text">
<p>You are using wrong XPATH syntax!</p>
<blockquote>
<p>//div[class='inner-body']/div[class='serp-wrap-all']/table[class='table-valign-middle
  matrix-table']</p>
</blockquote>
<p>The correct format is to add "@" before "class"</p>
<blockquote>
<p>//div[@class='inner-body']/div[@class='serp-wrap-all']/..</p>
</blockquote>
<p>And there is no 'matrix-table' table in above link.</p>
<p>Try using something like:</p>
<blockquote>
<p>//div[@class='inner-body']/div[@class='serp-wrap-all']//*[contains(@class,'matrix-table')]</p>
</blockquote>
</div>
