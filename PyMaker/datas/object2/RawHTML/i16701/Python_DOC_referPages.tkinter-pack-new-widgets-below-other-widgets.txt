<div class="post-text" itemprop="text">
<p>I think you should consider replacing the text field with a <a href="https://docs.python.org/3/library/tkinter.scrolledtext.html" rel="nofollow">ScrolledText</a> field.
It's a lot easier to use and doesn't require manual scrollbar placement.
(Don't use <code>pack</code> to place it though. Use <code>grid</code>)</p>
<pre><code>import tkinter as tk
import tkinter.scrolledtext as tkst

self.chattext = tkst.ScrolledText(
    master = self,
    wrap   = tk.WORD,
    width  = 20,
    height = 10
)
</code></pre>
