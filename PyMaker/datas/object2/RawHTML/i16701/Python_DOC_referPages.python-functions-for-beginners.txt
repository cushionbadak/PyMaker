<p>You <em>need</em> to specify a <code>return</code> explicitly. </p>
<p>replace </p>
<pre><code>word.upper()
</code></pre>
<p>with</p>
<pre><code>return word.upper()
</code></pre>
<p>If no return statement is specified, <code>None</code> is returned by default, Hence the result.</p>
<pre><code>string = 'Bryan'

def caps(word):
    return word.upper()

returning_value = caps(string) #Note that you need to catch the returning the value. 
print returning_value
</code></pre>

<div class="post-text" itemprop="text">
<p>You need to do two things:</p>
<ol>
<li><p>Because <a href="https://docs.python.org/3/library/stdtypes.html#str.upper" rel="nofollow"><code>str.upper</code></a> returns an uppercase <em>copy</em> of the string on which the method was called, you need to return <code>word.upper()</code> from function <code>caps</code> and then</p></li>
<li><p>Reassign the variable <code>string</code> to that return value.</p></li>
</ol>
<p>Here is how your script should look:</p>
<pre><code>string = 'Bryan'

def caps(word):
    return word.upper()

string = caps(string)

print string  # Print the new uppercased string
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; string = 'Bryan'
&gt;&gt;&gt; def caps(word):
...     return word.upper()
...
&gt;&gt;&gt; string = caps(string)
&gt;&gt;&gt; print string
BRYAN
&gt;&gt;&gt;
</code></pre>
</div>

<p>I think you are missing a print command.</p>
<pre><code>def caps(word):
        print word.upper()
</code></pre>
<p>If you want value back in a variable then you will have to return the value in function and assign it to a variable. </p>
<pre><code>def caps(word):
        return word.upper()

string = caps(string)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have to remember to ask yourself, what do I want this function to do? 
In many cases, you might want it to give you the result. eg return</p>
<pre><code>string = 'Bryan'

def caps(word):
    return word.upper()

newstring = caps(string)
print newstring
</code></pre>
<p>In some cases you may just want the function to do what you were going to do anyway.</p>
<pre><code>string = 'Bryan'

def pcaps(word):
    print word.upper()

pcaps(string)
</code></pre>
