<div class="post-text" itemprop="text">
<p>Your question could be clearer, but if what you want is what I suggested in the comments, this will work:</p>
<pre><code>def keyfunc(item):
    key, value = item
    if isinstance(key, str):
        return 1
    else:
        return len(key)

max(d.items(), key=keyfunc)[0]
</code></pre>
<p>Here, we define a key function that takes a key/value 2-tuple as returned by <a href="https://docs.python.org/3/library/stdtypes.html#dict.items" rel="nofollow"><code>dict.items()</code></a>, and:</p>
<ul>
<li>if the value is a string, returns 1.</li>
<li>otherwise, returns the length of the list of strings.</li>
</ul>
<p>That key function is then used with the builtin <a href="https://docs.python.org/3/library/functions.html#max" rel="nofollow"><code>max()</code></a> function to return the item with the largest number of strings, and finally we use <code>[0]</code> to get the key from that item. </p>
</div>

<p>I understand that you want to return the key that has the maximum numbers of data.</p>
<p>A pretty functional solution:</p>
<pre><code>max_key = max(
    d.items(),
    key=lambda (_, values): len(values) if type(values) == list else 1
)[0]
</code></pre>
