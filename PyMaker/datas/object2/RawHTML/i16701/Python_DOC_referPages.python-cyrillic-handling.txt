<div class="post-text" itemprop="text">
<p>Chances are you have JSON data; JSON uses <code>\uhhhh</code> escape sequences to represent Unicode codepoints. Use the <a href="https://docs.python.org/3/library/json.html#json.loads" rel="nofollow"><code>json.loads()</code> function</a> on <em>unicode</em> (decoded) data to produce a Python string:</p>
<pre><code>import json

string = json.loads(data.decode('utf8'))
</code></pre>
<p>UTF-8 is the default JSON encoding; check your response headers (if you are using a HTTP-based API) to see if a different encoding was used.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; import json
&gt;&gt;&gt; json.loads(b'"\\u041a\\u0435\\u0439\\u0442\\u043b\\u0438\\u043d\\u043f\\u0440\\u043e"'.decode('utf8'))
'Кейтлинпро'
</code></pre>
