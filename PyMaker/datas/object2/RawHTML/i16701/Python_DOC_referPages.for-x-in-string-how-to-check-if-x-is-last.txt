<div class="post-text" itemprop="text">
<p>Is there a way to know if x is the last element of a string in the following basic example of a for loop</p>
<pre><code>for x in string_time:
     do something
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you want to know whether <code>x</code> is physically the last element in the string, you can use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow"><code>enumerate()</code></a>:</p>
<pre><code>for i,x in enumerate(string_time, start=1-len(string_time)):
    if not i:
        # last element
    ...
</code></pre>
<p>If, on the other hand, you want to know whether <code>x</code> is <em>equal</em> to the last element, you can simply use <code>==</code> (as was also mentioned in the comments):</p>
<pre><code>for x in string_time:
    if x == string_time[-1]:
        # last element
    ...
</code></pre>
<hr/>
<p>Just to describe what's going on in the first snippet: we're enumerating the string starting at <code>1-len(string)</code>, which does the following:</p>
<pre><code>&gt;&gt;&gt; s = 'abc'
&gt;&gt;&gt;
&gt;&gt;&gt; list(enumerate(s, start=1-len(s)))
[(-2, 'a'), (-1, 'b'), (0, 'c')]
</code></pre>
<p>So the last element is enumerated with <code>0</code>, meaning we can use the <code>not i</code> check to check if we're on the last element.</p>
</div>
<div class="post-text" itemprop="text">
<p>Another option is to process the last character in the string separately from the loop</p>
<pre><code>for x in string_time[:-1]:
    # do stuff
last_char = string_time[-1]
</code></pre>
<p>This assumes that the string is non-empty.</p>
