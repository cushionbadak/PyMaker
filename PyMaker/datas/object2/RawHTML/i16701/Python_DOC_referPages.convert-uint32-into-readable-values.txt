<pre><code>broadcast_add = dbus.UInt32(4278298816L, variant_level=1)
</code></pre>
<p>What I am doing so far is converting it to 32 bit binary by :</p>
<pre><code>In [1]: '{0:0&lt;32}'.format(bin(broadcast_add)[2:])
Out[1]: '11111111000000011010100011000000'
</code></pre>
<p>Is there any straightforward way to convert this uint32 to readable ip address like</p>
<pre><code>192.168.1.255
</code></pre>

<div class="post-text" itemprop="text">
<p>If you can access this value <code>4278298816</code>, you might try to use <a href="https://docs.python.org/3/library/struct.html" rel="nofollow"><code>struct</code></a>, for example:</p>
<pre><code>&gt;&gt;&gt; import struct
&gt;&gt;&gt;
&gt;&gt;&gt; struct.unpack('&lt;BBBB', struct.pack('&lt;I', 4278298816))
(192, 168, 1, 255)
&gt;&gt;&gt;
&gt;&gt;&gt; '{}.{}.{}.{}'.format(*_)
'192.168.1.255'
&gt;&gt;&gt;
</code></pre>
