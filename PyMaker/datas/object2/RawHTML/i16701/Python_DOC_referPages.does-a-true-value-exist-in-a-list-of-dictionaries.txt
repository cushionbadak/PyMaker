<div class="post-text" itemprop="text">
<p>I create a list of dictionaries like this:</p>
<pre><code>list = []

for i in xrange(4):
    list.append({})
    list[i]['a'] = False
</code></pre>
<p>Now after a while, I want to (using a single line of code) see if any of the 'a' values are True.</p>
<p>I have tried:</p>
<pre><code>anyTrue = True in list             # always returns false
anyTrue = True in list[:]['a']
</code></pre>
<p>Is there such a convenient way of doing this?</p>
<p>Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/2/library/functions.html#any" rel="nofollow"><code>any</code></a> with <a href="https://docs.python.org/2/tutorial/classes.html#generator-expressions" rel="nofollow">generator expression</a>:</p>
<pre><code>&gt;&gt;&gt; lst = []
&gt;&gt;&gt;
&gt;&gt;&gt; for i in xrange(4):
...     lst.append({})
...     lst[i]['a'] = False
...
&gt;&gt;&gt; any(d['a'] for d in lst)
False

&gt;&gt;&gt; lst[1]['a'] = True
&gt;&gt;&gt; any(d['a'] for d in lst)
True
</code></pre>
<p>BTW, don't use the <code>list</code> as a variable name. It shadows builtin function/type <a href="https://docs.python.org/2/library/functions.html#list" rel="nofollow"><code>list</code></a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow"><code>any</code></a> and a <a href="http://legacy.python.org/dev/peps/pep-0289/" rel="nofollow">generator expression</a>:</p>
<pre><code>if any(x['a'] for x in list):
    # Do stuff
</code></pre>
<p>See a demonstration below:</p>
<pre><code>&gt;&gt;&gt; lst = []
&gt;&gt;&gt; for i in xrange(4):
...     lst.append({})
...     lst[i]['a'] = False
...
&gt;&gt;&gt; any(x['a'] for x in lst)
False
&gt;&gt;&gt; lst[2]['a'] = True  # Set an 'a' value to True
&gt;&gt;&gt; any(x['a'] for x in lst)
True
&gt;&gt;&gt;
</code></pre>
<p>Also, you should refrain from naming a variable <code>list</code>.   Doing so overshadows the built-in.</p>
</div>
