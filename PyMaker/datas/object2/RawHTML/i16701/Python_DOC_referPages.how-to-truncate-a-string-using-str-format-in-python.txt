<div class="post-text" itemprop="text">
<p>How to truncate a string using <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="noreferrer"><code>str.format</code></a> in Python? Is it even possible?</p>
<p>There is a <code>width</code> parameter mentioned in the <a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="noreferrer">Format Specification Mini-Language</a>:</p>
<pre><code>format_spec ::=  [[fill]align][sign][#][0][width][,][.precision][type]
...
width       ::=  integer
...
</code></pre>
<p>But specifying it apparently only works for padding, not truncating:</p>
<pre><code>&gt;&gt;&gt; '{:5}'.format('aaa')
'aaa  '
&gt;&gt;&gt; '{:5}'.format('aaabbbccc')
'aaabbbccc'
</code></pre>
<p>So it's more a minimal width than width really.</p>
<p>I know I can slice strings, but the data I process here is completely dynamic, including the format string and the args that go in. I cannot just go and explicitly slice one.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>.precision</code> instead:</p>
<pre><code>&gt;&gt;&gt; '{:5.5}'.format('aaabbbccc')
'aaabb'
</code></pre>
<p>According to the documentation of the <a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="noreferrer">Format Specification Mini-Language</a>:</p>
<blockquote>
<p>The <em>precision</em> is a decimal number indicating how many digits should be displayed after the decimal point for a floating point value formatted with <code>'f'</code> and <code>'F'</code>, or before and after the decimal point for a floating point value formatted with <code>'g'</code> or <code>'G'</code>. <strong>For non-number types the field indicates the maximum field size - in other words, how many characters will be used from the field content.</strong> The <em>precision</em> is not allowed for integer values.</p>
</blockquote>
</div>
