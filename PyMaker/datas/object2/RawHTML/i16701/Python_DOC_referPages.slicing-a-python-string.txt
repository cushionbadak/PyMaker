<p>I have a list:<br/></p>
<pre><code>rows=['1\t0.00032822\n', '2\t0.00029311\n', '3\t0.0002048\n',...] 
</code></pre>
<p>and i want to have a list with the numbers only:<br/></p>
<pre><code>['0.00032822', '0.00029311', '0.0002048',...]. 
</code></pre>
<p>Something like:</p>
<pre><code>list = [sliceAppropiate(x) for x in rows]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; rows = ['1\t0.00032822\n', '2\t0.00029311\n', '3\t0.0002048\n'] 
&gt;&gt;&gt; [r.split()[1] for r in rows]
['0.00032822', '0.00029311', '0.0002048']
</code></pre>
<p>What this does is:</p>
<ul>
<li>Go through each item in <code>rows</code> in turn,</li>
<li>Give it the name <code>r</code>,</li>
<li>Split each <code>r</code> into a list with <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>split()</code></a>, by cutting at whitespace (which can be spaces, tabs or newlines),</li>
<li>Take item 1 from that list (where item 0 is the first one, because Python uses <a href="https://plus.google.com/115212051037621986145/posts/YTUxbXYZyfi" rel="nofollow">zero-based indexing</a>).</li>
</ul>
</div>
<div class="post-text" itemprop="text">
<p>Here is a very simple code that do it for you:</p>
<pre><code>list = []
for i in range(len(rows)): # from 0 to len(rows) - 1
    list[i] = rows[i].split('\t')[1]
</code></pre>
<p>If need float numbers use float function:</p>
<pre><code>float(rows[i].split('\t')[1])
</code></pre>
</div>
