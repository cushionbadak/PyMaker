<div class="post-text" itemprop="text">
<p>I developed a python package which I want to use in another project. 
The package structure looks like the following example:</p>
<pre><code>- foo
  - setup.py
  - README.md
  - foo
    - __init__.py
    - main.py
    - bar.py
    - sub_pkg1
       - __init__.py
       - example.py
    - sub_pkg2
       - __init__.py
       - example2.py
</code></pre>
<p>The <code>bar.py</code> File contains the <code>Bar</code>-Class which relies on the files in <code>sub_pkg1</code> and <code>sub_pkg2</code> which are imported like </p>
<pre><code>from sub_pkg1 import example.py
</code></pre>
<p>The <code>__init__.py</code> files just import each python file in it's directory.
For example the <code>__init__.py</code> of <code>sub_pkg1</code>:</p>
<pre><code>from example import ExampleClass
</code></pre>
<p>The <code>main.py</code> File just imports the Bar Class and executes some methods of it.</p>
<pre><code>from bar import Bar
bar = Bar()
bar.foo()
</code></pre>
<p>The <code>setup.py</code>file looks like the following:</p>
<pre><code>from setuptools import setup, find_packages

setup(
    name='Bar',
    version='0.0.7',
    author='me',
    packages=find_packages()
)
</code></pre>
<p>When I run the <code>main.py</code> file directly from the package it works like a charm.
However when I install it in my other project with pip using the command:
(I don't want to publish the package to PyPi yet and just want to include the development-version in my other project)</p>
<pre><code>pip install -e /path/to/package
</code></pre>
<p>and try to import it - I am getting the error: </p>
<pre><code>ModuleNotFoundError: No module named 'example'
</code></pre>
<p>Do I need to export the sub_packages somehow?</p>
<p>I use Python 3.6</p>
</div>
<div class="post-text" itemprop="text">
<p>instead of <code>from sub_pkg1 import example.py</code> use <code>from sub_pkg1 import ExampleClass</code></p>
<p>as in <code>__init__.py</code> of sub_pkg1 and sub_pkg2 you have already imported <code>ExampleClass</code> and you are now free to directly import <code>ExampleClass</code> from sub_pkg1 anywhere in your project.</p>
<p>Read more about <code>__init__.py</code> and modules here: <a href="https://docs.python.org/3/tutorial/modules.html" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/modules.html</a> </p>
</div>
