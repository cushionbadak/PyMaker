<p>I have a number of constants,variable in which i keep names.</p>
<pre><code>ATTR_ITEM_NAME = 'pro'
</code></pre>
<p>I check if the attribute is attached to an objects:</p>
<pre><code> if hasattr(obj1, ATTR_ITEM_NAME):
</code></pre>
<p>then if exist I want the attribute value to be passed to an attribute of an object, something like this:</p>
<pre><code>obj2.fm = obj1.ATTR_ITEM_NAME
</code></pre>
<p>ATTR_ITEM_NAME being a string and not an attribute is an error, I need something that works;</p>
</div>
<div class="post-text" itemprop="text">
<p>Python also has <a href="https://docs.python.org/3/library/functions.html?highlight=getattr#getattr" rel="nofollow noreferrer"><code>getattr</code></a> which works like <a href="https://docs.python.org/3/library/functions.html?highlight=getattr#hasattr" rel="nofollow noreferrer"><code>hasattr</code></a> but returns the value:</p>
<pre><code>obj2.fm = getattr(obj1, ATTR_ITEM_NAME)
</code></pre>
<p>If you are not sure the attribute exists you could:</p>
<ul>
<li><p>assign a default value (e.g. <code>None</code>)</p>
<pre><code>DEFAULT = None
obj2.fm = getattr(obj1, ATTR_ITEM_NAME, DEFAULT)
</code></pre></li>
<li><p>or catch the exception using</p>
<pre><code>try:
    obj2.fm = getattr(obj1, ATTR_ITEM_NAME)
except AttributeError:
    pass  # or do something else...
</code></pre></li>
</ul>
</div>
