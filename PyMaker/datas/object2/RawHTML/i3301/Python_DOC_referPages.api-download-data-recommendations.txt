<p>This is in JSON format. Use the python <a href="https://docs.python.org/3/library/json.html" rel="nofollow noreferrer">json</a> encoder/decoder to load this data. It will turn it into a dictionary, and something like</p>
<pre><code>my_json_dict['values'] 
</code></pre>
<p>will return you that list.</p>
