<p>How can I create a permutation mapping of two lists in python?</p>
<p>For example I have two lists <code>[1,2,3]</code> and <code>['A','B','C']</code></p>
<p>Then my code should generate a list of 6 dictionaries</p>
<pre><code>[ {1:'A',2:'B',3:'C'},
  {1:'A',2:'C',3:'B'},
  {1:'B',2:'A',3:'C'},
  {1:'B',2:'C',3:'A'},
  {1:'C',2:'A',3:'B'},
  {1:'C',2:'B',3:'A'} ]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/library/functions.html#zip" rel="noreferrer"><code>zip</code></a> and <a href="https://docs.python.org/3/library/itertools.html#itertools.permutations" rel="noreferrer"><code>itertools.permutations</code></a> in a list comprehension:</p>
<pre><code>&gt;&gt;&gt; from itertools import permutations
&gt;&gt;&gt; L1 = [1,2,3]
&gt;&gt;&gt; L2 = ['A','B','C']
&gt;&gt;&gt; [dict(zip(L1, p)) for p in permutations(L2)]
[{1: 'A', 2: 'B', 3: 'C'},
 {1: 'A', 2: 'C', 3: 'B'},
 {1: 'B', 2: 'A', 3: 'C'},
 {1: 'B', 2: 'C', 3: 'A'},
 {1: 'C', 2: 'A', 3: 'B'},
 {1: 'C', 2: 'B', 3: 'A'}]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You seem to permutate only the values of the dicts, so you could do something like</p>
<pre><code>from itertools import permutations

dicts = []


keys = [1, 2, 3]
for values in permutations(['A', 'B', 'C']):
    new_dict = dict(zip(keys, values))
    dicts.append(new_dict)
</code></pre>
