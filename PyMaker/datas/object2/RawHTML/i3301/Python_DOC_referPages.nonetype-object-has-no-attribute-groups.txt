&gt;&gt;&gt; sentence = "foo bar (foo) bar foo-bar foo_bar foo'bar bar-foo bar, foo"
&gt;&gt;&gt; word = 'foo'

<p>Possible misconceptions and the corresponding explanations:</p>
<ul>
<li>You need a raw string <code>r'\bfoo\b' to make python not interpret the special characters (in this example the backslash</code>`) and pass them unchanged to the regular expression library.</li>
<li>You probably want to <code>re.escape</code> the word in case it contains more regular expression special characters.</li>
<li><code>.search()</code> only finds the first match. To find all matches use <code>.findall()</code> or <code>.finditer()</code></li>
</ul>
