<p>I'm making some calculations where I get values like number*e-17 , but i would like to make all those small values become zero.</p>
<p>Is there a way to make something like a tolerance that will change low values to zero in the whole program?
I'm using sympy btw.</p>

<div class="post-text" itemprop="text">
<p>You could use <code>math.isclose</code> to test for closeness to zero, and set the values accordingly:</p>
<pre><code>import math

value = 0 if math.isclose(value, 0) else value
</code></pre>
<p><a href="https://docs.python.org/3/library/math.html" rel="nofollow noreferrer">more details in python docs</a> </p>
</div>
