<div class="post-text" itemprop="text">
<p>You can check if a number is inside a <a href="https://docs.python.org/3/library/stdtypes.html#range" rel="nofollow noreferrer">range()</a> if you have a continious range to cover. 
Do not convert to int multiple times, store your int value: <code>bAsInt = int(b)</code> and use that.</p>
<p>If you want to check against specific single values, use a <a href="https://docs.python.org/3/library/stdtypes.html#set" rel="nofollow noreferrer">set()</a> if you have 4 or more values - it is faster that a list-lookup:</p>
<pre><code>even = {1300,1302,1304,1306,1308}

for number in range(1299,1311):
    # print(number," is 130* :", number//10 == 130 ) # works too, integer division
    print(number," is 130* :", number in range(1300,1310), 
          " and ", "even" if number in even else "odd")
</code></pre>
<p>Output:</p>
<pre><code>1299  is 130* : False  and  odd
1300  is 130* : True  and  even
1301  is 130* : True  and  odd
1302  is 130* : True  and  even
1303  is 130* : True  and  odd
1304  is 130* : True  and  even
1305  is 130* : True  and  odd
1306  is 130* : True  and  even
1307  is 130* : True  and  odd
1308  is 130* : True  and  even
1309  is 130* : True  and  odd
1310  is 130* : False  and  odd
</code></pre>

<div class="post-text" itemprop="text">
<p>You can also use this</p>
<pre><code>def start_with(nub, start, stop):
    return [x for x in range(start,stop) if str(x).startswith(str(nub))]

print(start_with(130, 1, 5000))
</code></pre>
<p>use of this function you can find any list start with any number</p>
