<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">dict comprehension</a>:</p>
<pre><code>k=[1,3,5,7,9]
matrix = { 1:[0,0], 2:[0,1], 3:[0,2], 4: [1,0], 5:[1,1], 6:[1,2], 7:[2,0], 8:[2,1], 9:[2,2]};
print({i: v for i, v in matrix.items() if i in k})
</code></pre>
<p>This outputs:</p>
<pre><code>{1: [0, 0], 3: [0, 2], 5: [1, 1], 7: [2, 0], 9: [2, 2]}
</code></pre>
<p>Or, if you would like to preserve the order prior to Python 3.6, you can use <a href="https://docs.python.org/3/library/collections.html#collections.OrderedDict" rel="nofollow noreferrer">OrderedDict</a>:</p>
<pre><code>from collections import OrderedDict
print(OrderedDict(filter(lambda i: i[0] in k, matrix.items())))
</code></pre>
<p>This outputs:</p>
<pre><code>OrderedDict([(1, [0, 0]), (3, [0, 2]), (5, [1, 1]), (7, [2, 0]), (9, [2, 2])])
</code></pre>
