<div class="post-text" itemprop="text">
<p>The issue is that PV is a string and not a float. <code>input()</code> is <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer">Python3</a> does not evaluate the input unlike in <a href="https://docs.python.org/2/library/functions.html#input" rel="nofollow noreferrer">Python2</a>.</p>
<p>You need to convert it into a <code>int</code>/<code>float</code>:</p>
<pre><code>PV = int(input("investment amout:"))
</code></pre>
<p>If you multiply a string with an int, it performs concatenation. That is why multiplying by float does not make sense.</p>
<pre><code>&gt;&gt;&gt; PV = "123"
&gt;&gt;&gt; PV*2
'123123'
&gt;&gt;&gt; PV*2.3
Traceback (most recent call last):
 File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: can't multiply sequence by non-int of type 'float'
</code></pre>
