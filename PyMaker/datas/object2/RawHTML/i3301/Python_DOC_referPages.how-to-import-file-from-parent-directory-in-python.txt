<div class="post-text" itemprop="text">
<p>Way 1: absolute import:</p>
<pre><code>from asCrawlers.common import get_init_url, create_urls_to_fetch, MILEAGE_RANGES, notify_on_error
</code></pre>
<p>Way 2: relative import:</p>
<pre><code>from ..common import get_init_url, create_urls_to_fetch, MILEAGE_RANGES, notify_on_error
</code></pre>
<p>More on imports <a href="https://docs.python.org/3/reference/simple_stmts.html#the-import-statement" rel="nofollow noreferrer">here</a>:</p>
<blockquote>
<p>When specifying what module to import you do not have to specify the
  absolute name of the module. When a module or package is contained
  within another package it is possible to make a relative import within
  the same top package without having to mention the package name. By
  using leading dots in the specified module or package after from you
  can specify how high to traverse up the current package hierarchy
  without specifying exact names. One leading dot means the current
  package where the module making the import exists. Two dots means up
  one package level. Three dots is up two levels, etc.</p>
</blockquote>
