<div class="post-text" itemprop="text">
<p>Since you want to convert from a list, you'll need to use my linked duplicate with a list comprehension,</p>
<pre><code>from datetime import datetime

list_of_string_dates = ["2018-06-12","2018-06-13","2018-06-14","2018-06-15"]

list_of_dates= [datetime.strptime(date,"%Y-%m-%d") for date in list_of_string_dates]

print(max(list_of_dates)) # oldest
print(min(list_of_dates)) # earliest

2018-06-15 00:00:00
2018-06-12 00:00:00
</code></pre>
<p>Basically, you're converting the string representation of your dates to a <em>Python</em> date using <a href="https://docs.python.org/3.7/library/datetime.html#datetime.datetime.strptime" rel="nofollow noreferrer"><code>datetime.strptime</code></a> and then applying <code>max</code> and <code>min</code> which are implemented on this type of objects.</p>

<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer"><code>datetime</code></a> module. In particular, since your date is in the standard POSIX format, you'll be able to use the function <code>date.fromtimestamp(timestamp)</code> to return a datetime object from your string; otherwise, you can use the <code>strptime</code> function to read in a more complicated string - that has a few intricacies that you can figure out by looking at the documentation</p>
