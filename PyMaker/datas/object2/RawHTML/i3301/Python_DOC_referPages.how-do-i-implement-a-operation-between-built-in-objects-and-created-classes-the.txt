<p>Let's say I want to define a class with a <code>__mul__</code> method, for example:</p>
<pre><code>class A():
    def __init__(self,*arg,**kwarg):
        #some code
    def __mul__(self,other):
        #some code
</code></pre>
<p>If I do <code>A()*2</code> it will give me a result. But when I try the other way around (<code>2*A()</code>), it doesn't work.
Is there a way to implement this other way around operation?</p>
</div>
<div class="post-text" itemprop="text">
<p>You would use <code>object.__rmul__(self, other)</code>:</p>
<pre><code>class A():
    def __init__(self, *arg, **kwarg):
        #some code
    def __mul__(self, other):
        #some code
    def __rmul__(self, other):
        return self * other
</code></pre>
<p><a href="https://docs.python.org/3/reference/datamodel.html" rel="noreferrer">https://docs.python.org/3/reference/datamodel.html</a></p>
</div>
