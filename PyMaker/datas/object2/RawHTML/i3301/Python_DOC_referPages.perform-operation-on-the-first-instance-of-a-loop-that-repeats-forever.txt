<p>Right now I have a script that is meant to loop forever. However, I would like to do something different on the first 'lap' along the lines of the following:</p>
<pre><code>import math

for i in range(0,math.inf):
    if i == 0:
       print("I'm gonna start the first lap")
    print('this is one lap')

"I'm gonna start the first lap"
'this is a lap'
'this is a lap'
</code></pre>
<p>Note that this code does not work because math.inf is a float, NOT an integer. And <a href="https://stackoverflow.com/a/40519631/9525212">this post here</a> says that in Python there is no way to represent infinity as an integer.</p>
<p>Using <code>while True:</code> could make sense in this situation, but is there any way to get the function to print something different for first (or the xth) repetition for that matter?</p>

<div class="post-text" itemprop="text">
<p>If you wish to use an infinite <code>for</code> loop while keeping a counter, you can use <a href="https://docs.python.org/3/library/itertools.html#itertools.count" rel="nofollow noreferrer"><code>itertools.count</code></a>:</p>
<pre><code>from itertools import count

for i in count():
    if i == 0:
        print("I'm gonna start the first lap")
    print('This is one lap')
    # break
</code></pre>
