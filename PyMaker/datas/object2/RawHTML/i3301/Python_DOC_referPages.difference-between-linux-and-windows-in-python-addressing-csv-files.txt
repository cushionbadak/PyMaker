<p>here is some info for linux <a href="https://www.tecmint.com/linux-directory-structure-and-important-files-paths-explained/" rel="nofollow noreferrer">directory structure</a> </p>
<p>press <strong>Alt + Enter</strong> over your file then in that properties dialog copy the path to root its something like <code>/home/&lt;user_name&gt;/Documents</code> then append that path with <code>file name</code> of csv file.</p>
<p>At Final your csv file's path becomes something like </p>
<p><code>/home/&lt;user_name&gt;/Documents/filename.csv</code></p>

<p>You could use separator from <code>os</code> module:</p>
<pre><code>import os
path = os.path.join('your_folder_name', 'your_file_name')
</code></pre>
<p>That way it could be Windows/Linux independent.</p>
<p>Here is <a href="https://docs.python.org/3/library/os.path.html" rel="nofollow noreferrer">official documentation to os.path module</a>.</p>
