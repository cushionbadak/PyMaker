<p>There are many posts on Stack Overflow that explain how to list all the subdirectories in a directory. However, all of these answers allow one to get the <em>full path</em> of each subdirectory, instead of just the name of the subdirectory.</p>
<p>I have the following code. The problem is the variable <code>subDir[0]</code> outputs the full path of each subdirectory, instead of just the name of the subdirectory:</p>
<pre><code>import os 


#Get directory where this script is located 
currentDirectory = os.path.dirname(os.path.realpath(__file__))


#Traverse all sub-directories. 
for subDir in os.walk(currentDirectory):
    #I know none of my subdirectories will have their own subfolders 
    if len(subDir[1]) == 0:
        print("Subdirectory name: " + subDir[0])
        print("Files in subdirectory: " + str(subDir[2]))
</code></pre>
<p>How can I just get the name of each subdirectory?</p>
<p>For example, instead of getting this:</p>
<blockquote>
<p>C:\Users\myusername\Documents\Programming\Image-Database\Curated\Hype</p>
</blockquote>
<p>I would like this:</p>
<blockquote>
<p>Hype</p>
</blockquote>
<p>Lastly, I still need to know the list of files in each subdirectory. </p>

<div class="post-text" itemprop="text">
<p>Splitting your sub-directory string on <code>'\'</code> should suffice. Note that <code>'\'</code> is an escape character so we have to repeat it to use an actual slash.</p>
<pre><code>import os

#Get directory where this script is located
currentDirectory = os.path.dirname(os.path.realpath(__file__))

#Traverse all sub-directories.
for subDir in os.walk(currentDirectory):
    #I know none of my subdirectories will have their own subfolders
    if len(subDir[1]) == 0:
        print("Subdirectory name: " + subDir[0])
        print("Files in subdirectory: " + str(subDir[2]))
        print('Just the name of each subdirectory: {}'.format(subDir[0].split('\\')[-1]))
</code></pre>
</div>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/os.html#os.listdir" rel="nofollow noreferrer"><code>os.listdir</code></a> coupled with <a href="https://docs.python.org/3/library/os.path.html#os.path.isdir" rel="nofollow noreferrer"><code>os.path.isdir</code></a> for this.</p>
<p>Enumerate all items in the directory you want and iterate through them while printing out the ones that are directories.</p>
<pre><code>import os
current_directory = os.path.dirname(os.path.realpath(__file__))
for dir in os.listdir(current_directory):
    if os.path.isdir(os.path.join(current_directory, dir)):
        print(dir)
</code></pre>
</div>
