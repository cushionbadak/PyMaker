<div class="post-text" itemprop="text">
<p>You can use the <code>argparse</code> module to parse command-line arguments:</p>
<pre><code>import argparse
parser = argparse.ArgumentParser()
parser.add_argument('--num', type=int)
args = parser.parse_args()
</code></pre>
<p>Then you can refer to the <code>num</code> parameter in your code with:</p>
<pre><code>search_box.send_keys(args.num)
</code></pre>
<p>Please read argparse's documentation for details:
<a href="https://docs.python.org/3/library/argparse.html" rel="nofollow noreferrer">https://docs.python.org/3/library/argparse.html</a></p>
