<div class="post-text" itemprop="text">
<p>Let's say I have a root directory 'x' number of directories up from the current directory. How can I find the full path to this root directory in python assuming 'x' can change and thus can't be hard-coded in any way?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow noreferrer">pathlib</a>:</p>
<pre><code>from pathlib import Path

levels_up = 1  # or whatever you want

current_dir = Path.cwd()
root = current_dir.parents[levels_up - 1]

print(root)
# /home   &lt;-- one level up from my home directory
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>from os.path import abspath, join

def get_root(path, up_n):
    return abspath(join(path, join(['..'] * up_n)))
</code></pre>
<p>You might also consider using <code>realpath</code> instead of <code>abspath</code>, that will take care of resolving symbolic links too.</p>
</div>
