<p>I am trying to implement a hexbin plot inside a function, and have a rather complicated <code>reduce_C_function</code> that will need to receive an argument <code>a</code>. An (albeit trivial) example:
<pre><code>    def sum_i(z,a):
        return a*np.sum(z)
    def some_function(X,Y,Z,a):
        hexb = plt.hexbin(X,Y,C=Z,reduce_C_function=sum_i)</code></pre></p></div>
<div class="post-text" itemprop="text">
<p>You can make a <a href="https://docs.python.org/3/library/functools.html#functools.partial" rel="nofollow noreferrer"><code>partial</code></a> function from your <code>sum_i</code> and pass it to <code>plt.hexbin</code> as a function of a single argument:  </p>
<pre><code>from functools import partial

def sum_i(z,a):
    return a*np.sum(z)
def some_function(X,Y,Z,a):
    reduce_function = partial(sum_i, a=a)
    hexb = plt.hexbin(X,Y,C=Z,reduce_C_function=reduce_function)
</code></pre>
<p>From the looks of the <a href="https://matplotlib.org/_modules/matplotlib/axes/_axes.html#Axes.hexbin" rel="nofollow noreferrer">source code</a> there is no other way to pass additional arguments for <code>reduce_C_function</code>.  </p>
