<p>I have an array with as shape (0,5).
The content is <code>[[4.0, 3.0, 0.0, 1.0, 5.0]]</code></p>
<p>How can I convert it into <code>[[4 3 0 1 5]]</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>A simple list comprehension can do this.</p>
<pre><code>a = [[4.0, 3.0, 0.0, 1.0, 5.0]]
b = [[int(x) for x in a[0]]]
</code></pre>
<p>We are asking Python to convert each element of the list <code>a[0]</code> into an <code>int</code>. Look up <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehensions</a> for more info on how it works and how to do it yourself in the future.</p>
