<p>I'm trying to figure out how to get combinations of a list of lists based on one element.</p>
<p>So let's assume I have this list of lists </p>
<pre><code>    [["blue", 1], ["blue", 3], ["red", 6], ["green", 5]] 
</code></pre>
<p>How would I get these combinations in python:</p>
<pre><code>    [["blue", 1], ["red", 6], ["green", 5]], [["blue", 3], ["red", 6], ["green", 5]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This can be done with the help of <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer">groupby</a> and <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow noreferrer">product</a>:</p>
<pre><code>data = [["blue", 1], ["blue", 3], ["red", 6], ["green", 5]]
groups = []
for k, g in itertools.groupby(data, key=lambda item: item[0]):
    groups.append(list(g))
print(list(itertools.product(*groups)))
# [(['blue', 1], ['red', 6], ['green', 5]), (['blue', 3], ['red', 6], ['green', 5])]

# just for understanding how it works:
print(groups)
# [[['blue', 1], ['blue', 3]], [['red', 6]], [['green', 5]]]
</code></pre>
