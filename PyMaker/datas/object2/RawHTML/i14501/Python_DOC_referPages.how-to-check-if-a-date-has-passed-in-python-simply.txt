If the date is <code>01/05/2015</code>, and the date; <code>30/04/2015</code> was in-putted into Python, it would return True, to say the date has passed.</p>

<div class="post-text" itemprop="text">
<p>you may use datetime, first parse String to date, then you can compare</p>
<pre><code>import datetime
d1 = datetime.datetime.strptime('05/01/2015', "%d/%m/%Y").date()
d2 = datetime.datetime.strptime('30/04/2015', "%d/%m/%Y").date()
d2&gt;d1
</code></pre>

<div class="post-text" itemprop="text">
<pre><code>from datetime import datetime
present = datetime.now()
print datetime(2015,04,30) &lt; present #should return true
</code></pre>
