<div class="post-text" itemprop="text">
<p><code>^</code> is the <a href="http://en.wikipedia.org/wiki/Exclusive_or">xor</a> operator.</p>
<p><code>**</code> is exponentiation.</p>
<p><code>2**3 = 8</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You can also use the <code>math</code> library. For example:</p>
<pre><code>import math
x = math.pow(2,3) # x = 2 to the power of 3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>if you want to repeat it multiple times - you should consider using numpy:</p>
<pre><code>import numpy as np

def cube(number):
    "can be also called with a list"
    return np.exp(number,3)

print cube(2)
print cube([2,8])
</code></pre>
