<p>I have a text file that looks like this</p>
<pre><code>['M', '&gt;', 'F', 'P', ' ', 'N', 'I', '$', 'F', '\x19', 'J', 'B', 'P', 'T', '%', '&lt;', 'M', 'Q', '&gt;', '\x00', 'I', 'F', 'N', 'J', '\x16', 'X', '\\', 'H']
</code></pre>
<p>Its already in a list form. 
How do i load it into python as a list without it think that all the separators between the sections are actual characters?</p>
</div>
<div class="post-text" itemprop="text">
<p>Read the contents as a string, and use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow"><code>ast.literal_eval</code></a> to convert it into a Python data structure.</p>
<p>That is:</p>
<pre><code>import ast
with open('datafile.txt') as f:
    data = ast.literal_eval(f.read())
</code></pre>
