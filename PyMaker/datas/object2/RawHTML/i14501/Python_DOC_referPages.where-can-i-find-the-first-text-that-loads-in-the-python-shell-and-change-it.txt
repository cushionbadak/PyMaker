<div class="post-text" itemprop="text">
<p>I don't know of any way to <em>change</em> the default text without modifying/recompiling the python binary, but it seems you can use the environment variable <code>PYTHONSTARTUP</code> in order to add additional text via a python file with <code>print</code> commands. You can also change the prompt strings in this file. For example:</p>
<p>in my .bashrc:</p>
<pre><code>export PYTHONSTARTUP=/home/jake/.mypythonstartup
</code></pre>
<p>/home/jake/.mypythonstartup:</p>
<pre><code>import sys
print("Welcome, master!")
sys.ps1 = "How may I serve you? "
sys.ps2 = "                 ... "
</code></pre>
<p>Result:</p>
<pre><code>$ python
Python 2.7.5 (default, Mar  9 2014, 22:15:05) 
[GCC 4.2.1 Compatible Apple LLVM 5.0 (clang-500.0.68)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
Welcome, master!
How may I serve you? def test():
                 ...     print("test")
                 ...
How may I serve you? test()
test
How may I serve you? 
</code></pre>
<p>Documentation on <code>PYTHONSTARTUP</code> can be found here: <a href="https://docs.python.org/3/tutorial/appendix.html#the-interactive-startup-file" rel="nofollow">https://docs.python.org/3/tutorial/appendix.html#the-interactive-startup-file</a></p>

<div class="post-text" itemprop="text">
<p>Based on a quick snoop around the <a href="https://hg.python.org/cpython/file/293d9964cf6e/Lib/idlelib" rel="nofollow"><code>idlelib</code> source code</a>, you could do something like:</p>
<pre><code>from code import interact
interact("Welcome master.")
</code></pre>
<p>In use:</p>
<pre class="lang-none prettyprint-override"><code>$ python idle2.py
Welcome master.
&gt;&gt;&gt; print 'foo'
foo
</code></pre>
<hr/>
<p>You could also use the <a href="https://docs.python.org/2/using/cmdline.html" rel="nofollow">command line flags</a> to run a command then enter interactive mode:</p>
<pre class="lang-none prettyprint-override"><code>$ python -ic "print 'Welcome master.'"
Welcome master.
&gt;&gt;&gt; 
</code></pre>
