<p>I want to use <strong>Argument of an Exception</strong> to show additional information while an error occur in my code.
Here is my code.</p>
<p><strong>ArgumentOfAnException.py</strong> </p>
<pre><code>from pip._vendor.distlib.compat import raw_input

def to_celsius(f):
    try:
        c = (f-32)*5/9;
        return c;
    except ValueError, Argument:
        print("The argumet doesn't contain number\n", Argument);
        return;

f = raw_input("Enter temperature in *F : ");
print("Temperature in Celsius : ", to_celsius(float(f)));
print("Thank you...");
</code></pre>

<pre><code>File "F:\Document\Files\Coding\Java Eclipse\Python Project\First Project\src\Error\ArgumentOfAnException.py", line 7

<div class="post-text" itemprop="text">
<p>It seems like you're using python 3.x.</p>
<p>Use following syntax (using <code>as</code>):</p>
<pre><code>except ValueError as Argument:
</code></pre>
<p>Reference: <a href="https://docs.python.org/3/reference/compound_stmts.html#the-try-statement" rel="nofollow"><code>try</code> statement - Python 3 documentation</a></p>
