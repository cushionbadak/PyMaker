<p>Valid python <a href="https://docs.python.org/2/library/functions.html#super" rel="nofollow">syntax for calling super</a> is:</p>
<pre><code>def handle(self, *args, **options):
    super(CommandA, self).handle(*args, **options)
</code></pre>
<p>If you use python 3 then you can omit <a href="https://docs.python.org/3/library/functions.html#super" rel="nofollow"><code>super()</code></a> arguments:</p>
<pre><code>def handle(self, *args, **options):
    super().handle(*args, **options)
</code></pre>
