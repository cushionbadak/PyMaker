<div class="post-text" itemprop="text">
<p>Lines is a list of dicts.  If you iterate over a dict, you get its keys.  To get the values instead, use <code>dict.values()</code>.  So you need to change this line:</p>
<pre><code>outline = [k for k in lines[i] if k is not ''] + [cx,uf,rf,pc]
</code></pre>
<p>to</p>
<pre><code>outline = [k for k in lines[i].values() if k is not ''] + [cx,uf,rf,pc]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>keys</code> and <code>values</code> builtins:</p>
<pre><code>&gt;&gt;&gt; [k for k in lines[i].keys() if k is not '']
['fishType', 'timestamp', 'event', 'targetID', 'playerID', 'lights', 'adjusted timestamp']
</code></pre>
<p>and </p>
<pre><code>&gt;&gt;&gt; [k for k in lines[i].values() if k is not '']
['small green fish', '2014-08-23 16:21:05.430000-05:00', 'MakeSpawnFish', '34', '3', '1', '2014-08-23 17:20:05.43000']
</code></pre>
<p>These will iterate over the dictionary's keys and values. Check out the <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow">dictionary documentation here</a>.</p>
</div>
