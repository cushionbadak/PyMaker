<p>I am a beginner in python. This may seem very basic. I am using python 2.7</p>
<p>If I use input in python prompt, I get an error. I am typing this:</p>
<pre><code>my_reply= input ("Enter your reply")
</code></pre>
<p>If I enter any letter/character, I get an error saying that <code>my_reply</code> is not defined. However, If I enter a number, there is no error.</p>
<p>What am I doing wrong??</p>
</div>
<div class="post-text" itemprop="text">
<p>In Python 2.7, input is actually trying to evaluate the "string" you pass to it. Therefore, if you were to input something like</p>
<pre><code>Hello World
</code></pre>
<p>it would simply evaluate to the expression Hello World - not a string. This can be avoided by supplying correct python string:</p>
<pre><code>'Hello World'
</code></pre>
<p>The real solution here is using <code>raw_input</code> function. <code>raw_input</code> does not try to evaluate the value and therefore, the first example would work as expected. </p>
</div>
<div class="post-text" itemprop="text">
<p>In Python2, you should use <code>raw_input</code> rather than <code>input</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You should use <code>raw_input</code> so Python won't evaluate your input as a string. It is then your responsibility to validate the input.</p>
</div>
