<div class="post-text" itemprop="text">
<p>I'm using <code>subprocess</code> to call command line. For example,</p>
<pre><code>subprocess.check_output(["echo", "Hello World!"])
</code></pre>
<p>This is for one argument. I want to call an executable called "myFuntion" and pass <em>several</em> arguments to it, by their names. The command line I would have written is:</p>
<pre><code>./myFunction --arg1 a1 --arg2 a2 
</code></pre>
<p>How do I "translate" this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Simple: each argument is a separate element in the list.</p>
<p>So, if you understand how the shell would separate out this command line:</p>
<pre><code>./myFunction --arg1 a1 --arg2 a2
</code></pre>
<p>… you know what the arguments are. When there are no quotes or escapes or anything else fancy, the shell just separates on spaces. So this is just 5 arguments:</p>
<pre><code>subprocess.check_output(["./myFunction", "--arg1", "a1", "--arg2", "a2"])
</code></pre>
<p>If you have a command line that you're not sure how to translate, you can ask your shell to do it for you… or just use the <a href="https://docs.python.org/3/library/shlex.html" rel="nofollow"><code>shlex</code></a> module in Python:</p>
<pre><code>&gt;&gt;&gt; print(shlex.split('./myFunction --arg1 a1 --arg2 a2'))
['./myFunction', '--arg1', 'a1', '--arg2', 'a2']
</code></pre>
</div>
