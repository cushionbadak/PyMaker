<div class="post-text" itemprop="text">
<p>You have a few issues in your code above. I will address them all:</p>
<ul>
<li>Writing for loops and using iteration</li>
<li>Returning from a function</li>
<li>Best practice not to shadow built-ins</li>
<li>Parsing input</li>
</ul>
<p><strong>Writing for loops and iteration:</strong></p>
<blockquote>
<p>for(i = 1; i &lt; list.length-1; i += 2)</p>
</blockquote>
<p>is written as (<em>in Python</em>):</p>
<pre><code>for i in range(1, (len(list) - 1), 2)
</code></pre>
<p>See: <a href="https://docs.python.org/3/library/stdtypes.html?highlight=range#range" rel="nofollow"><code>range()</code></a></p>
<p><strong>Returning from a function:</strong></p>
<p>Your function <code>oddEvenSort()</code> also does not return anything.</p>
<p>So <code>y = oddEvenSort(...)</code> will return <code>None</code>. You need a <code>return list</code> at the end of the function.</p>
<p><strong>NOT Shadowing built-ins:</strong></p>
<p>DO NOT shadow built-in types and functions such as <code>list()</code> with variables named <code>list</code>. Use <code>xs</code> or something more appropriately named.</p>
<p><strong>Parsing input:</strong></p>
<p>Thanks to Joran Beasley; Yes your <code>input</code> is not in the correct form (<em>yet</em>). Try converting this to a list of integers by doing something like:</p>
<pre><code>s =  input("Enter some numbers (comma separated):")
xs = [x.strip() for x in s.split(",")]  # clean input
ys = list(map(int, xs))  # map input list into actual integers
</code></pre>
