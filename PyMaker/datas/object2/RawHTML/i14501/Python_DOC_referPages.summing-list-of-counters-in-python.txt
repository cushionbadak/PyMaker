<div class="post-text" itemprop="text">
<p>I am looking to sum a list of counters in python. For example to sum:</p>
<pre><code>counter_list = [Counter({"a":1, "b":2}), Counter({"b":3, "c":4})]
</code></pre>
<p>to give <code>Counter({'b': 5, 'c': 4, 'a': 1})</code></p>
<p>I can get the following code to do the summation:</p>
<pre><code>counter_master = Counter()
for element in counter_list:
    counter_master = counter_master + element
</code></pre>
<p>But I am confused as to why <code>counter_master = sum(counter_list)</code> results in the error <code>TypeError: unsupported operand type(s) for +: 'int' and 'Counter'</code> ? Given it is possible to add counters together, why is it not possible to sum them?</p>
</div>
<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html#sum"><code>sum</code></a> function has the optional <em>start</em> argument which defaults to 0. Quoting the linked page:</p>
<blockquote>
<p><code>sum(iterable[, start])</code></p>
<p>Sums <em>start</em> and the items of an <em>iterable</em> from left to right and returns
  the total</p>
</blockquote>
<p>Set <em>start</em> to (empty) <code>Counter</code> object to avoid the <code>TypeError</code>:</p>
<pre><code>In [5]: sum(counter_list, Counter())
Out[5]: Counter({'b': 5, 'c': 4, 'a': 1})
</code></pre>
</div>
