<div class="post-text" itemprop="text">
<p>Ubuntu 14.04 uses Python 2 by default, and the <code>pyenv</code> command does not exist in Python 2 out of the box. </p>
<p>You can, however, use <code>virtualenv</code> for the same purpose. You just need to install it!</p>
<p>You should:</p>
<ul>
<li>Install Python 3 and virtualenv <code>apt-get install -y python3 python-virtualenv</code></li>
<li>Create a Python 3 virtualenv: <code>virtualenv -p $(which python3) testDir</code></li>
<li>Activate the virtual environment with <code>source testDir/bin/activate</code></li>
</ul>
</div>
<div class="post-text" itemprop="text">
<p>It's also possible to create virtualenv by python itself.
python3 -m venv myenv</p>
<p>see documentation <a href="https://docs.python.org/3/library/venv.html" rel="nofollow">https://docs.python.org/3/library/venv.html</a></p>
</div>
<div class="post-text" itemprop="text">
<p>It's in the package <code>python3.4-venv</code> (Linux Mint) or <code>python3-venv</code> (Ubuntu - I guess).</p>
<p>The advantages of <code>venv</code> over <code>virtualenv</code> are that (1) it's in vanilla Python3, (2) interpreter <strong>does</strong> retain tab-completion.</p>
