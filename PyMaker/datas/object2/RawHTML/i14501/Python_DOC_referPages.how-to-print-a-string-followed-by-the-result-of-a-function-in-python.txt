<div class="post-text" itemprop="text">
<p>I have a function <code>trip_cost</code> which calculates the total cost of a vacation. If I want to print the result of the function I can do so without problem like so:</p>
<pre><code>print trip_cost(city, days, spending_money)
</code></pre>
<p>However if I try to code a more presentable, user-friendly version using a string I get a <code>Syntax Error: Invalid Syntax</code></p>
<pre><code>print "Your total trip cost is: " trip_cost(city, days, spending_money)
</code></pre>
<p>How can this problem be solved?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/2/library/string.html#string.Formatter.format" rel="nofollow noreferrer"><code>format()</code></a> string method:</p>
<pre><code>print "Your total trip cost is: {}".format(trip_cost(city, days, spending_money))
</code></pre>
<p>Update for Python 3.6+:</p>
<p>You can use <a href="https://docs.python.org/3/whatsnew/3.6.html#whatsnew36-pep498" rel="nofollow noreferrer">formatted string literals</a> in Python 3.6+</p>
<pre><code>print(f"Your total trip cost is: {trip_cost(city, days, spending_money)}")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>str.format()</code>:</p>
<pre><code>print "Your total trip cost is: {}".format(trip_cost(city, days, spending_money))
</code></pre>
<p>See <a href="https://docs.python.org/2/library/string.html#string-formatting" rel="nofollow">String Formatting</a></p>
<blockquote>
<p><code>format(format_string, *args, **kwargs)</code> format() is the primary API
  method. It takes a format string and an arbitrary set of positional
  and keyword arguments. format() is just a wrapper that calls
  vformat().</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>Use str</p>
<pre><code>print "Your total trip cost is: " + str(trip_cost(city, days, spending_money))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can Use <strong>format</strong></p>
<p>Or %s specifier</p>
<pre><code>print "Your total trip cost is: {}".format(trip_cost(city, days, spending_money))
</code></pre>
<p>OR</p>
<pre><code>print "Your total trip cost is: %s"%(trip_cost(city, days, spending_money))
</code></pre>
</div>
