<p>How can I create iterate group of three from a iterate object? For creating a pair of iteration function I can do something like
    from itertools import tee</p>
<pre><code>def func(iterate):
    i, j = tee(iterate)
    next(j, None)
    return zip(i, j)

l = [1,2,3,4,5]
for a, b in func(l):
    print(a, b)
&gt; 1, 2
&gt; 2, 3
&gt; 3, 4
&gt; 4, 5
</code></pre>

<div class="post-text" itemprop="text">
<p>You can expand on what you already did for groups of two, but with one more variable for the third item:</p>
<pre><code>def func(iterate):
    i, j, k = tee(iterate, 3)
    next(j, None)
    next(k, None)
    next(k, None)
    return zip(i, j, k)

l = [1,2,3,4,5]
for a, b, c in func(l):
    print(a, b, c)
</code></pre>
<p>This outputs:</p>
<pre><code>1 2 3
2 3 4
3 4 5
</code></pre>
<p>Note that your sample code in the question is incorrect as it is missing a call to <code>zip</code> in the returning value from <code>func</code>.</p>

<p>Use <a href="https://docs.python.org/3.3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a>:</p>
<pre><code>l = [1,2,3,4,5]

for a, b, c in zip(l, l[1:], l[2:]):
    print(a, b, c)

# 1 2 3
# 2 3 4
# 3 4 5
</code></pre>
<p>You can also create groups of two with this method:</p>
<pre><code>l = [1,2,3,4,5]

for a, b in zip(l, l[1:]):
    print(a, b)

# 1 2
# 2 3
# 3 4
# 4 5
</code></pre>
