<p>According to <a href="https://docs.python.org/3/library/socket.html" rel="nofollow noreferrer">https://docs.python.org/3/library/socket.html</a>, the AF_INET address family needs a tuple which contains a string ip address and a integer port. Your code passes a tuple which contains a string ip address and string port. Fixing that is very simple, just cast the port to int, like this:</p>
<pre><code>port = int(raw_input("Enter TCP/UDP port: "))
</code></pre>
<p>edit: when using python 2, you have to use raw_input() rather than input()</p>
