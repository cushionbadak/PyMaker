<div class="post-text" itemprop="text">
<p>You can define a function to take a variable number of parameters, and then use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> to work on them in one statement like:</p>
<h3>Code:</h3>
<pre><code>def increment_to_list(*args):
    return [a + 1 for a in args]
</code></pre>
<h3>Test Code:</h3>
<pre><code>print(increment_to_list(1,2,3))
print(increment_to_list(4,5,6,7,8))
</code></pre>
<h3>Results:</h3>
<pre><code>[2, 3, 4]
[5, 6, 7, 8, 9]
</code></pre>

<p>I think you want to do something like this:</p>
<pre><code>def f(*args):
    l = []
    for i in args:
        l.append(i+1)

    return l
</code></pre>
<p>Or, if you like list comprehensions like me, you could do:</p>
<pre><code>def f(*args):
    return [val + 1 for val in args]
</code></pre>

<div class="post-text" itemprop="text">
<p>I think you want variable number of arguments with list output:</p>
<pre><code>def myFunc(*args):
    return [i+1 for i in args]
</code></pre>
<p>Now you can get your <code>z</code> from this function like this:</p>
<pre><code>z = myFunc(1,2,3,4)
z
#[2, 3, 4, 5]
</code></pre>
