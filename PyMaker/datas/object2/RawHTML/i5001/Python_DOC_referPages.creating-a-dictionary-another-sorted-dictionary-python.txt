<p>I have a dictionary were each entry is formated like this: ("word":4, "word2":6 ...).  I want to create a new dictionary which consists of only the 10 entries in the old dictionary with the highest values.  Anyone know how i would do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can create a <code>collection.Counter</code> from the dictionary, get the <a href="https://docs.python.org/3/library/collections.html#collections.Counter.most_common" rel="nofollow noreferrer"><code>most_common</code></a> words and create a new <code>dict</code> from the returned list of tuples:</p>
<pre><code>&gt;&gt;&gt; d = {"word%d"%i : random.randint(0,100) for i in range(10)}
&gt;&gt;&gt; dict(collections.Counter(d).most_common(5))
{'word4': 92, 'word2': 83, 'word3': 89, 'word5': 72, 'word9': 47}
</code></pre>
<p>Or more "manually", using <code>sorted</code> with a special <code>key</code> function and slicing, same result:</p>
<pre><code>&gt;&gt;&gt; dict(sorted(d.items(), key=lambda t: -t[1])[:5])
</code></pre>
