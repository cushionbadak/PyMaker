<p>You have some indentation errors. An if-else statement should look like:</p>
<pre><code>if x == 1:
    print(1)
    print(2)
elif x == 2:
    print(3)
    print(4)
</code></pre>
<p>Notice how all of the lines between <code>if</code> and <code>elif</code> are indented!</p>
<p>Also, the last line of your code is <code>file.close</code>. That should be <code>file.close()</code>, it is a method. See the python documentation <a href="https://docs.python.org/3/tutorial/inputoutput.html" rel="nofollow noreferrer">here</a>.</p>
<p>They python documentation for the if/else statements can be found <a href="https://docs.python.org/3/tutorial/controlflow.html" rel="nofollow noreferrer">here</a>.</p>
