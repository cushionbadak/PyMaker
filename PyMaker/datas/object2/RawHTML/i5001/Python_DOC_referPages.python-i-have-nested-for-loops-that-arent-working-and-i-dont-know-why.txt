<div class="post-text" itemprop="text">
<p>I think that the problem is, you haven't reset your counter variables. Another problem is that your second loop starts from 1 instead of 0. (thanks Leonardo Gazdek for pointing out!)</p>
<p>Strings indexes start from 0:</p>
<pre><code>&gt;&gt;&gt; 'abc'[1]
'b'
</code></pre>
<p>Here's how you should do it:</p>
<pre><code>inVal=(input("Please enter a positive integer: "))
evencounter=0
oddcounter=0

for i in range(1,int(inVal)):
    # add those pieces of code:
    evencounter=0 
    oddcounter=0
    for j in range (0,len(str(i))): # start from 0
        if(int(str(i)[j])%2==0):
            evencounter+=1
        else:
            oddcounter+=1
    if(evencounter&gt;oddcounter):
        print(i)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#typesseq-range" rel="nofollow noreferrer"><code>range(start, stop[, step])</code></a> gives values from <code>start</code> till values <i>less than</i> <code>stop</code>.</p>
<p>In your inner loop, change</p>
<pre><code>for j in range (1,len(str(i))):
</code></pre>
<p>to </p>
<pre><code>for j in range (0,len(str(i))):
</code></pre>
<p>since the indexing starts from <code>0</code>.</p>
<p>And you need to reset the value of the counter after each iteration of the outer loop so that it can start over for the next number.</p>
<p>Also, if you want to include <code>inVal</code> as well, change the outer loop to</p>
<pre><code>for i in range(1,int(inVal)+1):
</code></pre>
