<div class="post-text" itemprop="text">
<p>If I have a string, lets just say, <code>b a hello b Hi</code>, how could I split the string by all <code>b</code>s <strong>AFTER</strong> the first occurrence of the letter <code>a</code>?  </p>
<p>As in, it would return <code>["b a hello", "Hi"]</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>This is documented here: <a href="https://docs.python.org/3/library/stdtypes.html#str.rsplit" rel="nofollow noreferrer">str.rsplit()</a></p>
<pre><code>sentence = 'b a hello b Hi'
sentence.rsplit('b', 1)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you note the location of the gate (first 'a') then you can split the string after that point like:</p>
<h2>Code:</h2>
<pre><code>a_string = 'b a hello b Hi'

first_a = a_string.index('a')
a_split = a_string[first_a:].split('b')
a_split[0] = a_string[:first_a] + a_split[0]
a_split = [x.strip() for x in a_split]

print(a_split)
</code></pre>
<h3>Result:</h3>
<pre><code>['b a hello', 'Hi']
</code></pre>
</div>

<div class="post-text" itemprop="text">
<p>In your example result you split the string by ' b ' so I'm going to use that.</p>
<pre><code>a = "b a hello b Hi"
index = a.index('a') + a[a.index('a'):].index(' b ') # That's the index of first ' b '.

# Since split will give 1 empty element at the beginning I exclude that.
result = [a[:index]] + a[index:].split(' b ')[1:] 
# ['b a hello', 'Hi']
</code></pre>
<p>If you want to split by 'b' then replace them.</p>
