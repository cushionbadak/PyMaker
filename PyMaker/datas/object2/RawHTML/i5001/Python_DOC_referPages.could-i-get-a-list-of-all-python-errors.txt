<div class="post-text" itemprop="text">
<p>As expected, the <a href="https://docs.python.org/3/library/exceptions.html" rel="nofollow noreferrer">official documentation</a> has the answer.</p>
<p>On this page, you'll find the base exceptions, and the concrete exceptions as well, sorted by category.
You'll find at section <a href="https://docs.python.org/3/library/exceptions.html#exception-hierarchy" rel="nofollow noreferrer">5.4. Exception hierarchy</a> an inheritance tree displaying the hierarchy between built-in exceptions.</p>
<p>The most interesting point certainly is that every exception extends <code>BaseException</code>, and that a custom exception should in most cases inherit from <code>Exception</code>.</p>
