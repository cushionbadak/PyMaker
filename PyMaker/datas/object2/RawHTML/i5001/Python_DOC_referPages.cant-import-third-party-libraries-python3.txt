<div class="post-text" itemprop="text">
<p>Quite simply you have named your file <code>string.py</code>.  <code>string</code> is a <a href="https://docs.python.org/3/library/string.html" rel="nofollow noreferrer">module</a> that is included in the default library.  There is nothing wrong with any of the modules you imported only that you importing your own script and not the module it actually needs. </p>
<p>Rename <code>string.py</code> to anything that is not a name that is not a module in the default library and it will work. </p>
