<div class="post-text" itemprop="text">
<p>In Python, lists can be iterated directly, and <code>enumerate</code> is used to generate indexes.  Its optional second parameter gives a starting number:</p>
<pre><code>&gt;&gt;&gt; elements = ['a','b','c','d']
&gt;&gt;&gt; for i,v in enumerate(elements,1):
...   print('{}) {}'.format(i,v))
...
1) a
2) b
3) c
4) d
</code></pre>
<p>If using Python 3.6+, formatting output is even more simple using f-strings:</p>
<pre><code>&gt;&gt;&gt; elements = ['a','b','c','d']
&gt;&gt;&gt; for i,v in enumerate(elements,1):
...   print(f'{i}) {v}')
...
1) a
2) b
3) c
4) d
</code></pre>

<p>One way would be to add a 1 in the range, then subtract a 1 from the index</p>
<pre><code>elements_list=['a','b','c','d']
lenght_list=len(elements_list)
for i in range(1, lenght_list+1):
     print (str(i) + str(')') + elements_list[i-1])
</code></pre>
<p>Edit: TheoretiCAL's approach is even more straight forward, simply adding 1 to the print statement achieves the same thing.</p>
</div>
