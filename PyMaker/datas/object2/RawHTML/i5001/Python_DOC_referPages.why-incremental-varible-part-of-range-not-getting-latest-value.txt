<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#range" rel="nofollow noreferrer">Range in pydocs</a> </p>
<blockquote>
<p>The range type represents an immutable sequence of numbers and is commonly used for looping a specific number of times in for loops.</p>
</blockquote>
<p>Which means that on every iteration your loop takes the next element from the list and the "+2" part gets dismissed. (overridden)</p>
</div>
<div class="post-text" itemprop="text">
<p>i is reset on each iteration, use a different variable to increment, for instance:</p>
<pre><code>j = 0
for i in range(i,count):
  j+=2
  print j
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>range</code> assigns the next value in the range to <code>i</code> in every iteration rendering  <code>i+=2</code> useless.</p>
<p>I would suggest you use </p>
<p><code>
for i in range(0,10,2):
  print(i)
</code></p>
