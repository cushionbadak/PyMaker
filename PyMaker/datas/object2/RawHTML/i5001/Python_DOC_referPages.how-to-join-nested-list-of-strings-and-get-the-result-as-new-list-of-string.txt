<p>I am having nested list of strings as:</p>
<pre><code>A = [["A","B","C"],["A","B"]]
</code></pre>
<p>I am trying to join the strings in sublist to get a single list of strings as:</p>
<pre><code>B = [['ABC'],['AB']]
</code></pre>
<p>OR,</p>
<pre><code>B = ['ABC','AB']
</code></pre>
<p>I tried using <code>''.join(A)</code> and <code>''.join(str(v) for v in A)</code> but they didn't work.</p>
<p>This is the error I got:</p>
<blockquote>
<p>TypeError: sequence item 0: expected str instance, list found</p>
</blockquote>

<div class="post-text" itemprop="text">
<p>You can <strong>use <a href="https://docs.python.org/3/library/string.html#string.join" rel="nofollow noreferrer"><code>''.join</code></a> with <code>map</code></strong> here to achieve this as:</p>
<pre><code>&gt;&gt;&gt; A = [["A","B","C"],["A","B"]]
&gt;&gt;&gt; list(map(''.join, A))
['ABC', 'AB']

# In Python 3.x, `map` returns a generator object.
# So explicitly type-casting it to `list`. You don't 
# need to type-cast it to `list` in Python 2.x
</code></pre>
<p>OR you may <strong>use it with <em>list comprehension</em></strong> to get the desired result as:</p>
<pre><code>&gt;&gt;&gt; [''.join(x) for x in A]
['ABC', 'AB']
</code></pre>
<p>For getting the results as <code>[['ABC'], ['AB']]</code>, you need to wrap the resultant string within another list as:</p>
<pre><code>&gt;&gt;&gt; [[''.join(x)] for x in A]
[['ABC'], ['AB']]

## Dirty one using map (only for illustration, please use *list comprehension*)
# &gt;&gt;&gt; list(map(lambda x: [''.join(x)], A))
# [['ABC'], ['AB']]
</code></pre>
<hr/>
<p><strong>Issue with your code:</strong> In your case <a href="https://docs.python.org/3/library/string.html#string.join" rel="nofollow noreferrer"><strong><code>''.join(A)</code></strong></a> didn't worked because <code>A</code> is a nested list, but <code>''.join(A)</code> tried to join together all the elements present in <code>A</code> treating them as <em>string</em>. And that's the cause for your error <em>"expected <code>str</code> instance, <code>list</code> found"</em>. Instead, you need to pass each sublist to your <code>''.join(..)</code> call. This can be achieved with <code>map</code> and <em>list comprehension</em> as illustrated above.</p>
</div>

<p>Simple</p>
<pre><code>map(''.join, A)

['ABC', 'AB']
</code></pre>
