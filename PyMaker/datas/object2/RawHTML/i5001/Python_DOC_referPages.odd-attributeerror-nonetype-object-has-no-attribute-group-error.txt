<p>call group only if there is a match.</p>
<pre><code>res = re.search(r"&lt;i&gt;(.*?)&lt;/i&gt;", str(englishWord))
# if there is a match
if res:
   englishWord = res.group(1)
</code></pre>
<p>As pointed out in the comments, <code>re.search</code> returns None when no match is found. Link : <a href="https://docs.python.org/3/library/re.html#re.search" rel="nofollow noreferrer">https://docs.python.org/3/library/re.html#re.search</a></p>
