<p>if i store objects in a list.
and i want this list to be stored in a file, in such a way that this list, when loaded from the file. It must contain same characteristics as before it saved, like element type.</p>

<div class="post-text" itemprop="text">
<p>You can use <code>pickle</code> for this, e.g : </p>
<pre><code>import pickle
my_list = [1,2,3]
pickle.dump( my_list, open( "save.p", "wb" ) )
my_list = pickle.load( open( "save.p", "rb" ) )
</code></pre>
<p>This will save the type and value of your object. More information about pickle usage can be found here : <a href="https://docs.python.org/3/library/pickle.html" rel="nofollow noreferrer">https://docs.python.org/3/library/pickle.html</a></p>
</div>
