<div class="post-text" itemprop="text">
<p>Here you can use <strong>list comprehension</strong> feature of python like </p>
<pre><code>list1 = [1,'f', 1.1, 1.01]
print([type(i) for i in list1])
</code></pre>
<p>result will be like <code>[&lt;class 'int'&gt;, &lt;class 'str'&gt;, &lt;class 'float'&gt;, &lt;class 'float'&gt;]</code></p>
</div>

<p>If you have python list say <code>l</code>;</p>
