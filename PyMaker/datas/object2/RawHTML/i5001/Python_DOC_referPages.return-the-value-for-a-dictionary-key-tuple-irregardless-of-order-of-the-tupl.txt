<div class="post-text" itemprop="text">
<p>Convert your keys to <a href="https://docs.python.org/3/library/stdtypes.html#frozenset" rel="nofollow noreferrer">frozensets</a>:</p>
<pre><code>submatrix = {('W', 'F'): 1, ('L', 'R'): -2, ('S', 'P'): -1}

d = {frozenset(k): v for k, v in submatrix.items()}

d[frozenset({'W', 'F'})]  # 1
d[frozenset({'F', 'W'})]  # 1
</code></pre>
<p>This works because frozensets are immutable and unordered.</p>
