<div class="post-text" itemprop="text">
<p>Use a <code>dict</code> (<a href="https://docs.python.org/3/library/stdtypes.html#mapping-types-dict" rel="nofollow noreferrer">[Python]: Mapping Types â€” dict</a>):</p>
<pre><code>combo_dict = {
    "DAP": 46,
    "MAP": 52,
    "SPS": 21,
    "SPT": 46,
}
</code></pre>
<p>and its usage (as current values are <code>int</code>s, convert them to <code>float</code>s):</p>
<pre><code>float(combo_dict[combo.get()])
</code></pre>
<p>or better (in case the string returned by the <em>combo</em> is not present in <code>combo_dict</code>: <a href="https://docs.python.org/3/library/stdtypes.html#dict.get" rel="nofollow noreferrer">[Python]: <strong>get</strong>(<em>key[, default]</em>)</a>):</p>
<pre><code>float(combo_dict.get(combo.get(), -1))
</code></pre>
