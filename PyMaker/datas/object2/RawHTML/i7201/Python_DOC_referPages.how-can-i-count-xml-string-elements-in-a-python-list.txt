<p>The problem:</p>
<blockquote>
<p>Write a function, <code>tag_count</code>, that takes as its argument a list of
  strings. It should return a count of how many of those strings are XML
  tags. You can tell if a string is an XML tag if it begins with a left
  angle bracket "&lt;" and end with a right angle bracket "&gt;".</p>
</blockquote>
<p>def tag_count(tokens):
    count = 0
    for token in tokens:
        if token[0] == '&lt;' and token[-1] == '&gt;':
            count += 1
    return count
    list1 = ['', 'Hello World!', '']
    count = tag_count(list1)
    print("Expected result: 2, Actual result: {}".format(count))</p>
<p>My results are always <code>0</code>. What am I doing wrong?</p>
</div>
<div class="post-text" itemprop="text">
<p>First of all, you are not counting anything since you are redefining <code>count</code> variable in the loop. Also, you are actually missing the XML string check (starts with <code>&lt;</code> and ends with <code>&gt;</code>).</p>
<p>Fixed version:</p>
<pre><code>def tag_count(list_strings):
    count = 0
    for item in list_strings:
        if item.startswith("&lt;") and item.endswith("&gt;"):
            count += 1
    return count
</code></pre>
<p>Which you may then improve by using the built-in <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer"><code>sum()</code></a> function:</p>
<pre><code>def tag_count(list_strings):
    return sum(1 for item in list_strings
               if item.startswith("&lt;") and item.endswith("&gt;"))
</code></pre>
