<pre><code>class Solution(object):
    def removeInvalidParentheses(self, s):
        """
        :type s: str
        :rtype: List[str]
        """
        def isvalid(s):
            ctr=0
            for string in s:
                if string=='(':
                    ctr+=1
                if string==')':
                    ctr-=1
                    if ctr&lt;0:
                        return False
            return ctr==0
        n=len(s)
        level={s}
        while level:
            valid = filter(isvalid, level)
            if valid:
                return valid
            level = {s[:i] + s[i+1:] for s in level for i in range(n)}

<div class="post-text" itemprop="text">
<p><code>{}</code> is a dictionary.</p>
<p><code>set()</code>, <code>{x}</code>, <code>{x, y}</code>, <code>{x, y, z, ...}</code> are sets.</p>
<p><code>{x:y, ...}</code> (i.e. if it has a colon <code>:</code>) are dictionaries.</p>
