<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.isdigit" rel="nofollow noreferrer"><code>str.isdigit</code></a> is a method not an attribute, so (as was already mentioned in the comments) you need to call it.</p>
<p>Also you check each character (starting from the left side) for <code>isdigit</code> but if it's not passing the <code>isdigit()</code> test you remove a character from the right side. That doesn't seem right. You probably wanted to iterate over the <code>for i in reversed(string)</code> (or using slicing: <code>for i in string[::-1]</code>).</p>
<p>But you also could simplify the logic by using a <code>while</code> loop:</p>
<pre><code>def string_conversion(string):
    while string and not string.isdigit():
        string = string[:-1]
    return string
</code></pre>
