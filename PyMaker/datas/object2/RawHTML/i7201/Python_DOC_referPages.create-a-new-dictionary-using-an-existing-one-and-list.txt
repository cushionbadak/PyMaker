<div class="post-text" itemprop="text">
<p>If I had:</p>
<pre><code>adict = {'a':3, 'b':6, 'c':9, 'd':12}
alist = ['a', 'z', 't', 's']
</code></pre>
<p>How would I create a new dict with the keys of the first dict and the items of the list, resulting in this?</p>
<pre><code>bdict = {'a': 'a', 'b': 'z', 'c': 't', 'd': 's'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>To bring the keys of <em>adict</em> together the values from <em>alist</em> use the <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip()</a> function.</p>
<pre><code>&gt;&gt;&gt; from collections import OrderedDict
&gt;&gt;&gt; adict = OrderedDict([('a', 3), ('b', 6), ('c', 9), ('d', 12)])
&gt;&gt;&gt; alist = ['a', 'z', 't', 's']
&gt;&gt;&gt; bdict = OrderedDict(zip(adict, alist))
&gt;&gt;&gt; bdict
OrderedDict([('a', 'a'), ('b', 'z'), ('c', 't'), ('d', 's')])
</code></pre>
<p>I've used ordered dictionaries here because the question only makes sense if the dictionaries are <a href="https://docs.python.org/3/library/collections.html#collections.OrderedDict" rel="nofollow noreferrer">OrderedDict</a>s; otherwise, you can't guarantee the pairwise one-to-one correspondence between <em>adict</em> and <em>alist</em>.</p>
