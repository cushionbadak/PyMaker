<div class="post-text" itemprop="text">
<p>Try using the flag <code>re.DOTALL</code>:</p>
<pre><code>import re
lines = """aaa
bbb
ccc
ddd"""

match = re.search(".*", lines, re.DOTALL)

print(match.group())
</code></pre>
<p>From the documentation:</p>
<blockquote>
<p>'.' (Dot.) In the default mode, this matches any character except a
  newline. If the DOTALL flag has been specified, this matches any
  character including a newline. </p>
<p>re.DOTALL Make the '.' special
  character match any character at all, including a newline; without
  this flag, '.' will match anything except a newline.</p>
</blockquote>
