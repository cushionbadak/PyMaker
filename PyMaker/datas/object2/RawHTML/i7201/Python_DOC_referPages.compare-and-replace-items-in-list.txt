<p>How do you compare the objects in two lists, and then if there are two of the same objects in both lists, replace it with something? For example:</p>
<pre><code>list1 = [a,b,c,d]
list2 = ['h','j','a','d']
</code></pre>
<p>I want to say, if some of the objects in <code>list1</code> are the same with some of the objects in <code>list2</code>, replace those objects with something else (For example, <code>'hello'</code>)</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>list1 = [x if x not in list2 else 'hello' for x in list1]
</code></pre>
</div>
