<div class="post-text" itemprop="text">
<p><code>\n</code> gives you a new line. You can put it anywhere in a string and when printing it you get a new line.</p>
<pre><code>In [1]: print('ab')
ab

In [2]: print('a\nb')
a
b
</code></pre>
<p>There are more of this kind, including tabs etc. 
<a href="https://docs.python.org/3/reference/lexical_analysis.html#literals" rel="nofollow noreferrer">https://docs.python.org/3/reference/lexical_analysis.html#literals</a></p>
