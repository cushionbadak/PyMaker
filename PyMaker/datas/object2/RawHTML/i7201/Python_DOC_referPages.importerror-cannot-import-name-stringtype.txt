<p>There is no <code>StringType</code> in Python3.</p>
<p>Try this instead:</p>
<pre><code> from types import *
 x=type('String')
</code></pre>
<p>To check the type of an object use:    </p>
<pre><code>type(x) is str
</code></pre>
