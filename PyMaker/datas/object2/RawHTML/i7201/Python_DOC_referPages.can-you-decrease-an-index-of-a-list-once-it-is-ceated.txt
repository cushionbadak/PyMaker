<div class="post-text" itemprop="text">
<p>This is a good use case for <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer"><code>list comprehension</code></a>:</p>
<pre><code>lives = [5, 5, 5, 5, 5]
outers = ['yes', 'no', 'yes', 'yes', 'no']
lives = [i-1 if j == 'yes' else i for i,j in zip(lives, outers)]
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; lives = [5, 5, 5, 5, 5]
&gt;&gt;&gt; outers = ['yes', 'no', 'yes', 'yes', 'no']
&gt;&gt;&gt; lives = [i-1 if j == 'yes' else i for i,j in zip(lives, outers)]
&gt;&gt;&gt;
&gt;&gt;&gt; lives
[4, 5, 4, 4, 5]
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your problem comes from the fact that y is an element of lives and not an index.</p>
<p>I believe</p>
<pre><code>for k in range(len(outers)):
    if outers[k]=="yes":
        lives[k]-=1
</code></pre>
<p>will give you what you want</p>
