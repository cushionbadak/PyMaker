<div class="post-text" itemprop="text">
<p>Use <code>list comprehension</code> with <code>sum</code>.</p>
<pre><code>In [4]: [sum(i) for i in zip(*_list)]
Out[4]: [0, 0, 3, 0, 3]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could <a href="https://docs.python.org/3.3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> the elements of the tuples and <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map</code></a> the result with <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer"><code>sum</code></a>:</p>
<pre><code>result = map(sum, zip(*lst))
</code></pre>
<p>Don't forget to tack on a <code>list(...)</code> if you're running python3.</p>
