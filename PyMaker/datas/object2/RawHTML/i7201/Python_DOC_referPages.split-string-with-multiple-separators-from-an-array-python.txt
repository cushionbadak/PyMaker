<p>Given an array of separators:</p>
<pre><code>columns = ["Name:", "ID:", "Date:", "Building:", "Room:", "Notes:"]
</code></pre>
<p>and a string where some columns were left blank (and there is random white space):</p>
<pre><code>input = "Name:      JohnID:123:45Date:  8/2/17Building:Room:Notes:  i love notes"
</code></pre>
<p>How can I get this:</p>
<pre><code>["John", "123:45", "8/2/17", "", "", "i love notes"]
</code></pre>
<p>I've tried simply removing the substrings to see where I can go from there but I'm still stuck</p>
<pre><code>import re
input = re.sub(r'|'.join(map(re.escape, columns)), "", input)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>use the list to generate a regular expression by inserting <code>(.*)</code> in between, then use <code>strip</code> to remove spaces:</p>
<pre><code>import re

columns = ["Name:", "ID:", "Date:", "Building:", "Room:", "Notes:"]
s = "Name:      JohnID:123:45Date:  8/2/17Building:Room:Notes:  i love notes"

result = [x.strip() for x in re.match("".join(map("{}(.*)".format,columns)),s).groups()]

print(result)
</code></pre>
<p>yields:</p>
<pre><code>['John', '123:45', '8/2/17', '', '', 'i love notes']
</code></pre>
<p>the <code>strip</code> part can be handled by the regular expression at the expense of a more complex regex, but simpler overall expression:</p>
<pre><code>result = re.match("".join(map("{}\s*(.*)\s*".format,columns)),s).groups()
</code></pre>
<p>more complex: if field data contains regex special chars, we have to escape them (not the case here):</p>
<pre><code>result = re.match("".join(["{}\s*(.*)\s*".format(re.escape(x)) for x in columns]),s).groups()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>How about using <a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow noreferrer"><code>re.split</code></a>?</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; columns = ["Name:", "ID:", "Date:", "Building:", "Room:", "Notes:"]
&gt;&gt;&gt; i = "Name:      JohnID:123:45Date:  8/2/17Building:Room:Notes:  i love notes"
&gt;&gt;&gt; re.split('|'.join(map(re.escape, columns)), i)
['', '      John', '123:45', '  8/2/17', '', '', '  i love notes']
</code></pre>
<p>To get rid of the whitespace, split on whitespace too:</p>
<pre><code>&gt;&gt;&gt; re.split(r'\s*' + (r'\s*|\s*'.join(map(re.escape, columns))) + r'\s*', i.strip())
['', 'John', '123:45', '8/2/17', '', '', '  i love notes']
</code></pre>
