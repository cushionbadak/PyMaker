<p>How can I setup the <code>PyArg_ParseTuple</code> function with more than one argument?</p>
</div>
<div class="post-text" itemprop="text">
<p>This is an example:</p>
<pre><code>const char *field = 0; PyObject *value; int typeField; size_t size = 0;
if (!PyArg_ParseTuple(args, "isO|n", &amp;typeField, &amp;field, &amp;value, &amp;size)) {
  return 0;
}
</code></pre>
<p>The characters in the 2<sup>nd</sup> argument have to correspond to the C types of the passed pointers, of course.</p>
<p>Btw. the <code>size</code> is an optional parameter.</p>
<p>A complete doc. about <code>PyArg_ParseTuple()</code> can be found in the Python doc.</p>
<p><a href="https://docs.python.org/3/c-api/arg.html" rel="nofollow noreferrer">Parsing arguments and building values</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use following code to get all arguments</p>
<pre><code>  vector&lt;PyObject*&gt; objects;
  int argc = PyTuple_GET_SIZE(args);
  objects.resize(argc);

  for (int i=0; i&lt;argc ; i++) {
    objects[i] = PyTuple_GET_ITEM(args, i);
  }
</code></pre>
