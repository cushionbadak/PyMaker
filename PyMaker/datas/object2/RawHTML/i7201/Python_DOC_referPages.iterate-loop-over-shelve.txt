<p>How would one iterate the contents of a Shelve?</p>
<pre><code>import Shelve
testShelve = Shelve.open("testShelve")
testShelve["key"] = "value"

for k in testShelve.keys():
    print(k)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As stated above, the Shelve object is dictionary-like and can be used the same.
To print the keys and values of the testShelve object:</p>
<pre><code>for key in testShelve:
    print(key, testShelve[key])
</code></pre>
</div>
<span class="comment-copy">The same way you would any other mapping (like a <code>dict</code>).</span>
<span class="comment-copy">Did you read <a href="https://docs.python.org/3/library/shelve.html" rel="nofollow noreferrer">the documentation</a>? “A “shelf” is a persistent, dictionary-like object.” – Dictionary-like means you can use it like a dictionary.</span>
