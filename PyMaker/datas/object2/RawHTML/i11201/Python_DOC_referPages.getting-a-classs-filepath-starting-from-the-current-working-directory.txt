<p>I am trying to print the path to a file. I want the full path from the current working directory to the file.</p>
<p>I am using the <code>inspect</code> module to find the path to my file. However this will give me the full path which is not what I'm looking for.</p>
<pre><code>import inspect

class Foo:
    def __init__(self):
        print(inspect.getfile(self.__class__))
</code></pre>
<p>Now for the filename itself I can use <code>os.path.basename</code>. But this will give me just the filename which is also not what I want.</p>
<pre><code>import inspect
import os

class Foo:
    def __init__(self):
        path = inspect.getfile(self.__class__)
        filename = os.path.basename(path)
        print(path)
        print(filename)
</code></pre>
<p>It would be relatively easy to use the <code>path</code> in combination with <code>os.getcwd()</code> to get the path I am looking for. This will give me the result I am looking for.</p>
<pre><code>import inspect
import os


class Foo:
    def __init__(self):
        path = inspect.getfile(self.__class__)
        cwd = os.getcwd()
        print(path[len(cwd):])
</code></pre>
<p>I am wondering if there is a better/easier way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>I think <a href="https://docs.python.org/2/library/os.path.html#os.path.relpath" rel="nofollow">os.path.relpath()</a> does what you want it to.</p>
<p>Example:</p>
<pre><code>import os, inspect

class Foo():
    def __init__(self):
        self.path = os.path.relpath(inspect.getfile(self.__class__))


foo = Foo()
print(foo.path)
</code></pre>
