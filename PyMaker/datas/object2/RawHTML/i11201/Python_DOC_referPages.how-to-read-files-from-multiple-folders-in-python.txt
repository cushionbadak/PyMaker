<div class="post-text" itemprop="text">
<p>IIUC you need <a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.read_csv.html" rel="nofollow"><code>read_csv</code></a> in folders <code>Type 2</code>, use <a href="https://docs.python.org/3/library/glob.html#glob.glob" rel="nofollow"><code>glob</code></a>:</p>
<pre><code>files = glob.glob('main/**/Type 2/*.csv')
dfs = [pd.read_csv(fp) for fp in files]
df = pd.concat(dfs)
</code></pre>
