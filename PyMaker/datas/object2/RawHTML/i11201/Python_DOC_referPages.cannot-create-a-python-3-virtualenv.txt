<div class="post-text" itemprop="text">
<p>You are using the virtualenv for Python 2.7 to create a virtual environment for Python 3.5, which I don't think will work.</p>
<p>Instead of <code>virtualenv</code>, On Python 3.3+ you could use the <a href="https://docs.python.org/3/library/venv.html" rel="nofollow">built-in <code>pyvenv</code> tool</a> to create a virtual environment instead.</p>
<pre><code>$ pyvenv-3.5 env
$ . env/bin/activate
(env)$ # etc...
</code></pre>
