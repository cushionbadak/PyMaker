<p>I need to sort by the first column in descending order an array. To be specific my code is:</p>
<pre><code>&gt;&gt;&gt; x = np.array([[2, 3], [1998,5], [1998,7]])
&gt;&gt;&gt; x = x[np.argsort(x[:,0])]
</code></pre>
<p>but the output is </p>
<pre><code>array([[   2,    3],
       [1998,    5],
       [1998,    7]])
</code></pre>
<p>but I need it in descending order. Can someone explain me how to do that?</p>
<p>EDIT:  @Babyburger suggested this solution :</p>
<pre><code>x = x[np.argsort(x[:,0])][::-1]
</code></pre>
<p>that give </p>
<pre><code>array([[1998,    7],
       [1998,    5],
       [   2,    3]])
</code></pre>
<p>it could be fine, but i would like that where the value on the first column is the same, the order is not changed. So the output would be</p>
<pre><code>array([[1998,    5],
       [1998,    7],
       [   2,    3]])
</code></pre>
<p>is there a simple way to do that?</p>

<div class="post-text" itemprop="text">
<p>in this particular requirement you can use python's <code>sorted</code> which is <a href="https://en.wikipedia.org/wiki/Sorting_algorithm#Stability" rel="nofollow">stable</a>:</p>
<pre><code>a =  np.array([[2, 3], [1998,5], [1998,7]])
res = np.array(sorted(a, key= lambda x: -x[0]))
</code></pre>
<p>it does: use the first element of each row for comparison (by <code>lambda</code> accessor) and negate it for decreasing order. By stability, the rows will preserve order if the first element is the same</p>
<p>ouput:</p>
<pre><code>[[1998    5]
 [1998    7]
 [   2    3]]
</code></pre>
<p>EDIT: btw if you wanted to sort by the following columns whenever the preceeding ones are identical (all of the same ordering):</p>
<pre><code>a =  np.array([[2, 3], [1998,5], [1998,7]])
res = np.array(sorted(a, key=lambda x:(-x).tolist()))
</code></pre>
<p>this converts the rows to lists and then uses <a href="https://docs.python.org/3/tutorial/datastructures.html#comparing-sequences-and-other-types" rel="nofollow">sequence comparison</a>. Note in this example it will be sorted decreasingly (hence <code>(-x)</code>)</p>
