<pre><code>list1= ['hello', 'say', 'morning', 'ironman', 'sayonara', 'king', 'yellow']  
list2= ['say', 'king', 'yellow']  
</code></pre>
<p>I want obtain a list3 like this:  </p>
<pre><code>list3= ['0', 'say', '0', '0', '0', 'king', 'yellow']  
</code></pre>
<p>So I want obtain the list1 size and where the words are not the same I want put a 0.  </p>
<p>I try with:  </p>
<pre><code>list1= ['hello', 'say', 'morning', 'ironman', 'sayonara', 'king', 'yellow']
list2= ['say', 'king', 'yellow']
list3 = list()
for i in list1:
     for j in list2:
          if(i=j):
              list3.append(i)
          else (i!=j):
              list3.append('0')
</code></pre>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehensions</a> to write your code in one-liner,</p>
<pre><code>list1= ['hello', 'say', 'morning', 'ironman', 'sayonara', 'king', 'yellow']
list2= ['say', 'king', 'yellow']

list3 = [s if s in list2 else '0' for s in list1]

print(list3)
['0', 'say', '0', '0', '0', 'king', 'yellow']
</code></pre>
<p>PS: convert <code>list2</code> into <code>set</code> to speed up.</p>
</div>
<div class="post-text" itemprop="text">
<p>Basically you just iterate the second list with each element from the first list. If they match you can break out of the second iteration and append the found element to the result, else append "0". Something like this:</p>
<pre><code>list1 = ['hello', 'say', 'morning', 'ironman', 'sayonara', 'king', 'yellow']  
list2 = ['say', 'king', 'yellow']  
result = [];
for word in list1:
    found = False;
    for testWord in list2:
        if(word == testWord):
            found = True;
            break;
    if found == True:
        result.append(word)
    else:
        result.append("0")


print(result) 
//prints ['0', 'say', '0', '0', '0', 'king', 'yellow']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>One for loop should do the trick:</p>
<pre><code>list1= ['hello', 'say', 'morning', 'ironman', 'sayonara', 'king', 'yellow']  
list2= ['say', 'king', 'yellow']

result = []
for word in list1:
    if word in list2:
        result.append(word)
    else:
        result.append("0")
print(result)

# Output
['0', 'say', '0', '0', '0', 'king', 'yellow']
</code></pre>
