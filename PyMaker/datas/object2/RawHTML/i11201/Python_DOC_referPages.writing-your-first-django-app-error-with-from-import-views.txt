<p>There are a couple of things that can cause this. I have two possible solutions that are not mentioned yet.</p>
<ol>
<li>Try to replace <code>url(r'^polls/', include('polls.urls')),</code> with <code>url(r'^polls/', include('mysite.polls.urls')),</code></li>
<li>Have you remembered to include an <code>__init__.py</code> file inside the <code>polls</code> folder? This is required for the import to work. See the <a href="https://docs.python.org/3/tutorial/modules.html#packages" rel="nofollow">docs</a> for reference. Simply create an empty file with the name <code>__init__.py</code>. Note the double underscores.</li>
</ol>
<p>Please let me know if this solves you problems.</p>
