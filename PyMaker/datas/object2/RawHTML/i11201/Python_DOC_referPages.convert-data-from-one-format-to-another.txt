<div class="post-text" itemprop="text">
<p>I have data in this format:</p>
<pre><code>{'a':['b','c','d'],'bla':['djjd','cop']}
</code></pre>
<p>And I want to convert the above into this format:</p>
<pre><code>('a',('b','c','d')),('bla',('djjd','cop'))
</code></pre>
<p>What can be done in Python to achieve the above?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use simple <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> that turns key value pairs from dictionary to tuples:</p>
<pre><code>&gt;&gt;&gt; d = {'a':['b','c','d'],'bla':['djjd','cop']}
&gt;&gt;&gt; [(k, tuple(v)) for k, v in d.items()]
[('a', ('b', 'c', 'd')), ('bla', ('djjd', 'cop'))]
</code></pre>
</div>
