<p>this gives a floating point number:</p>

5.666666666666667
</code></pre>
<p>However, when I try it I get:</p>
<pre><code>&gt;&gt;&gt; 17 / 3
5
</code></pre>

<div class="post-text" itemprop="text">
<p>That behaviour changed between Python 2.7 and 3.</p>
</div>
<div class="post-text" itemprop="text">
<p>The answer is in the URL:</p>
<pre><code>https://docs.python.org/3/tutorial/introduction.html
https://docs.python.org/2.7/tutorial/introduction.html
</code></pre>
<p>The URL you posted is for Python 3, but you are using Python 2.7.</p>
<p>There's a dropdown at the top of the page. You can select <code>2.7</code> from there to get the documentation for Python 2.7:</p>
<blockquote>
<pre><code>&gt;&gt;&gt; 17 / 3  # int / int -&gt; int
5
</code></pre>
</blockquote>
