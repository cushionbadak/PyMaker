<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/2/library/datetime.html#" rel="nofollow"><code>datetime</code></a> module.</p>
<pre><code>import datetime
date = datetime.datetime.strptime('10 jan 2016`,'%d %b %Y').date()
difference_in_dates = date - datetime.date.today() #this returns a timedelta object
</code></pre>
<p>Use <a href="https://docs.python.org/2/library/datetime.html#timedelta-objects" rel="nofollow"><code>datetime.timedelta</code></a> objects for comparison.</p>
<p>You can look up the documentation about the formats (%d, %m, etc.) <a href="https://docs.python.org/2/library/datetime.html#strftime-strptime-behavior" rel="nofollow">here</a></p>
