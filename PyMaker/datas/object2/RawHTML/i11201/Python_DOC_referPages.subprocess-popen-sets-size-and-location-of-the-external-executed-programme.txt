<p>I would like to execute an external programme from python via subprocess.Popen. I am wondering whether it is possible to set size and location of the window for the external programme that is executed via the command?</p>

<div class="post-text" itemprop="text">
<p>I assume Windows for such a question?</p>
<p>You may specify the window to be minimized or maximized, using the <a href="https://docs.python.org/3/library/subprocess.html#subprocess.STARTUPINFO" rel="nofollow"><code>startupinfo</code></a> parameter of popen.</p>
<pre><code>si = subprocess.STARTUPINFO()
si.dwFlags = subprocess.STARTF_USESHOWWINDOW
si.wShowWindow = 3   # SW_MAXIMIZE
</code></pre>
<p>See the <a href="https://msdn.microsoft.com/en-us/library/ms633548(v=vs.85).aspx" rel="nofollow"><code>ShowWindow</code></a> documentation for the list of values. AFAIK there is no named constant for them in the subprocess module.</p>
<p>The other window-related parameters of <code>STARTUPINFO</code> are not supported by popen at the moment. Most notably, none of <code>dwX</code>, <code>dwY</code>, <code>dwXSize</code> or <code>dwYSize</code> are supported.</p>
<p>As an alternative, you can check whether your implementation has a windows-specific extension, such as ActiveState's <a href="http://docs.activestate.com/activepython/3.4/pywin32/win32process__CreateProcess_meth.html" rel="nofollow"><code>win32process.CreateProcess</code></a></p>
