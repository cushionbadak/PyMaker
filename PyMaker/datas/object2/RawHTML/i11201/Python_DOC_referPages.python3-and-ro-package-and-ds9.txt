<div class="post-text" itemprop="text">
<p>In Python 3, <a href="https://docs.python.org/3/library/collections.html?highlight=userstring#collections.UserString" rel="nofollow"><code>UserString</code> is part of the <code>collections</code> module</a>.</p>
<p>As you can see on <a href="https://pypi.python.org/pypi/RO" rel="nofollow">the RO page</a>, this library does only support Python 2.6 and 2.7.</p>
