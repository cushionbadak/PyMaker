<div class="post-text" itemprop="text">
<p>Your code is not working because the variable <code>vow</code> is never reset to 0. You should reset it for each new word you encounter, else it will print the total of vowels in your string.</p>
<hr/>
<p>A better solution is to make use of <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>split()</code></a> instead of <a href="https://docs.python.org/3/library/stdtypes.html#str.isspace" rel="nofollow"><code>isspace()</code></a>, so words are "automatically" separated.</p>
<p>Then, you can iterate trough each word, and sum up letters which are vowels.</p>
<pre><code>words = s.split()
for word in words:
    vow = sum(letter in 'aeiou' for letter in word.lower())
    print("The number of vowels in", word, "are", vow)
</code></pre>
</div>

<p>Use the <code>re</code> module for regular expressions.</p>
<pre><code>import re

sentence = r'Hello World, how many vowels are in this sentence by word?'

for word in sentence.split():
    s = re.findall(r'[aeiou]', word, flags=re.IGNORECASE)
    print(word,'has',len(s),'vowels.')
</code></pre>

<pre><code>sentence = input("Enter a sentence: ")
vowels = "aeiouAEIOU"

for word in sentence.split():
    vowel_count = 0
    for letter in word:
        if letter in vowels:
            vowel_count += 1

    print("The number of vowels in %s is: %d" % (word, vowel_count))
</code></pre>
