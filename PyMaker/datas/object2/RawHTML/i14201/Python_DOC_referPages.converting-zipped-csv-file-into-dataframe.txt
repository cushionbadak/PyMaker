<p><code>zipfile.namelist</code> just returns a list of filenames -- it does not actually extract anything (<a href="https://docs.python.org/3/library/zipfile.html#zipfile.ZipFile.namelist" rel="nofollow noreferrer">https://docs.python.org/3/library/zipfile.html#zipfile.ZipFile.namelist</a>).</p>
<p>This should work:</p>
<pre><code>df = pd.read_csv(zipfile.open(zipfile.namelist()[0]))
</code></pre>
