<div class="post-text" itemprop="text">
<p>I have an array of zeros and ones like <code>[0,0,0,1,1,1,0,0,0,1,1]</code>. How to write a program to save neighboring 0's and 1's in different arrays.</p>
<p>E.g.
<code>[0,0,0,1,1,1,0,0,0,1,1]</code> giving <code>[0,0,0],[1,1,1],[0,0,0],[1,1]</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can group them with <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow"><code>itertools.groupby</code></a>, like this</p>
<pre><code>&gt;&gt;&gt; data = [0, 0, 0, 1, 1, 1, 0, 0, 0, 1, 1]
&gt;&gt;&gt; from itertools import groupby
&gt;&gt;&gt; [list(group) for item, group in groupby(data)]
[[0, 0, 0], [1, 1, 1], [0, 0, 0], [1, 1]]
</code></pre>
<p>The result of <code>groupby</code>, will be a tuple of actual item and an iterator which gives the grouped items. We just convert the grouped items to a list, with <code>list(group)</code>.</p>
<hr/>
<p>As per the comments,</p>
<pre><code>&gt;&gt;&gt; data = [1, 2, 1, 3, 4, 5, 6, 7, 1, 2, 3]
&gt;&gt;&gt; flag = [0, 0, 0, 1, 1, 1, 0, 0, 0, 1, 1]
</code></pre>
<p>Create a generator which will give the values multiplied,</p>
<pre><code>&gt;&gt;&gt; gen = (d * v for d, v in zip(data, flag))
</code></pre>
<p>Now, group based on the result of calling <code>bool</code> on each of the numbers. So, if the <code>bool</code> is called on <code>0</code>, it will give <code>False</code>, otherwise <code>True</code>.</p>
<pre><code>&gt;&gt;&gt; [list(g) for _, g in groupby(gen, key=bool)]
[[0, 0, 0], [3, 4, 5], [0, 0, 0], [2, 3]]
</code></pre>
