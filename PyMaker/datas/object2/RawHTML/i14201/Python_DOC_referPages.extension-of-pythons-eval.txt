<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#eval" rel="nofollow"><code>eval</code></a> is used only for expressions. From the documentation</p>
<blockquote>
<p>The <strong><em>expression</em></strong> argument is parsed and evaluated as a <strong><em>Python expression</em></strong></p>
</blockquote>
<p>Use <a href="https://docs.python.org/3/reference/simple_stmts.html#the-exec-statement" rel="nofollow"><code>exec</code></a> here</p>
<pre><code>&gt;&gt;&gt; string = "if X &gt; 3 :\n   print(\"X is Greater\")\nelse :\n   print(\"X is Lesser\")"
&gt;&gt;&gt; X = 6
&gt;&gt;&gt; exec(string)
X is Greater
</code></pre>
<p>However do note that both the statements are quite risky to use. (See <a href="http://nedbatchelder.com/blog/201206/eval_really_is_dangerous.html" rel="nofollow"><code>eval</code> really is dangerous</a>)</p>
