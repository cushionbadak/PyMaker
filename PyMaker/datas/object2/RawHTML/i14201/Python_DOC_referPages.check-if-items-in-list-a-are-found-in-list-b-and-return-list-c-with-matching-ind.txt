<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow"><code>index</code></a> from your second list, while iterating over elements of the first list in a list comprehension.</p>
<pre><code>&gt;&gt;&gt; a = ["string2" , "string4"]
&gt;&gt;&gt; b = ["string1" , "string2" , "string3" , "string4" , "string5"]
&gt;&gt;&gt; c = [b.index(i) for i in a]
&gt;&gt;&gt; c
[1, 3]
</code></pre>
<p>If there is a possibility that an element may be in <code>a</code> but not in <code>b</code> then you can modify this slightly</p>
<pre><code>&gt;&gt;&gt; [b.index(i) for i in a if i in b]
[1, 3]
</code></pre>

<p>Use <a href="https://docs.python.org/2/library/functions.html#enumerate" rel="nofollow">enumerate</a> combined with list comprehension to get the indexes directly in a list.</p>
<pre><code>&gt;&gt;&gt; [i for i,j in enumerate(b) if j in a]

</code></pre>

<div class="post-text" itemprop="text">
<p>You can make a dictionary of element-&gt;index by using <code>enumerate</code> on <code>b</code>. This has linear time complexity, but after you complete this step, all of your index lookups will be in constant time O(1), and you'll also have an easy way to see if the value from <code>a</code> could not be found in <code>b</code>, because <code>dict.get</code> will return <code>None</code>. You will also be able to do an O(1) filter operation on <code>a</code> by checking the existence of its elements in the dictionary first, which also makes your second loop have linear time complexity.</p>
<pre><code>&gt;&gt;&gt; a = [50, 150, 250]
&gt;&gt;&gt; b = list(range(200))
&gt;&gt;&gt; bindex = {x: i for i, x in enumerate(b)}
&gt;&gt;&gt; [bindex.get(x) for x in a]
[50, 150, None]
&gt;&gt;&gt; [bindex[x] for x in a if x in bindex]
[50, 150]
</code></pre>
</div>

set2 = set(b)
set3 = a &amp; b    #intersection
</code></pre>
<p>You can convert back 'set3' to a list and use a list comprehension.</p>
<pre><code>c = list(set3)
[c.index(i) for i in c]
