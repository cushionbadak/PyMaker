<div class="post-text" itemprop="text">
<p>I have two lists of string:</p>
<pre><code>a = ['a', 'b', 'c']
b = ['d', 'e', 'f']
</code></pre>
<p>I should result:</p>
<pre><code>['ad', 'be', 'cf']
</code></pre>
<p>What is the most pythonic way to do this ?</p>
</div>
<div class="post-text" itemprop="text">
<p>Probably with <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip</code></a>:</p>
<pre><code>c = [''.join(item) for item in zip(a,b)]
</code></pre>
<p>You can also put multiple sublists into one large iterable and use the <code>*</code> operator to unpack it, passing each sublist as a separate argument to <code>zip</code>:</p>
<pre><code>big_list = (a,b)
c = [''.join(item) for item in zip(*biglist)]
</code></pre>
<p>You can even use the <code>*</code> operator with <code>zip</code> to go in the other direction:</p>
<pre><code>&gt;&gt;&gt; list(zip(*c))
[('a', 'b', 'c'), ('d', 'e', 'f')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip</code></a></p>
<pre><code>&gt;&gt;&gt; a = ['a', 'b', 'c']
&gt;&gt;&gt; b = ['d', 'e', 'f']

&gt;&gt;&gt; [ai+bi for ai,bi in zip(a,b)]

['ad', 'be', 'cf']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>How about:</p>
<pre><code>[c + d for c,d in zip(a,b)]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow">zip</a>:</p>
<pre><code>zip(a,b)
[('a', 'd'), ('b', 'e'), ('c', 'f')]

[x+y for x,y in zip(a,b)]
['ad', 'be', 'cf']
</code></pre>
<p>You can also use enumerate with list comprehension:</p>
<pre><code>[j+b[i] for i,j in enumerate(a)]
['ad', 'be', 'cf']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>lambda</code> together with <code>map</code>:</p>
<pre><code>map(lambda x,y:x+y,a,b)
</code></pre>
<p>Or add <code>zip</code> to manage more than 2 lists:</p>
<pre><code>map(lambda x:''.join(x), zip(a,b,c))
</code></pre>
<p>For python &lt; 3 I would prefer the former option or replace <code>zip()</code> for <code>izip()</code>, for python 3 the change is not necessary (<code>zip</code> already return an iterator).</p>
</div>
<span class="comment-copy">This one also works with an arbitrary number of lists, by using <code>zip(*list_of_lists)</code>.</span>
<span class="comment-copy">Yes. Also keep in mind that you can use the <code>*</code> operator to un<code>zip</code> things: <code>list(zip(*c))</code> produces <code>[('a', 'b', 'c'), ('d', 'e', 'f')]</code>.</span>
