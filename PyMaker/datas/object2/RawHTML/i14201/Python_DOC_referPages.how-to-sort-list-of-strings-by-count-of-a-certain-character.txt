<div class="post-text" itemprop="text">
<p>I have a list of strings and I need to order it by the appearance of a certain character, let's say <code>"+"</code>.
So, for instance, if I have a list like this: </p>
<pre><code>["blah+blah", "blah+++blah", "blah+bl+blah", "blah"]
</code></pre>
<p>I need to get:</p>
<pre><code>["blah", "blah+blah", "blah+bl+blah", "blah+++blah"]
</code></pre>
<p>I've been studying the <code>sort()</code> method, but I don't fully understand how to use the key parameter for complex order criteria. Obviously <code>sort(key=count("+"))</code> doesn't work. Is it possible to order the list like I want with <code>sort()</code> or do I need to make a function for it?</p>
</div>
<div class="post-text" itemprop="text">
<p>Yes, <a href="https://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow"><code>list.sort</code></a> can do it, though you need to specify the <code>key</code> argument:</p>
<pre><code>In [4]: l.sort(key=lambda x: x.count('+'))

In [5]: l
Out[5]: ['blah', 'blah+blah', 'blah+bl+blah', 'blah+++blah']
</code></pre>
<p>In this code <code>key</code> function accepts a single argument and uses <code>str.count</code>  to count the occurrences of <code>'+'</code> in it. </p>
<p>As for <code>list.sort(key=count('+'))</code>, you <em>can</em> get it to work if you define the <code>count</code> function like this (with <a href="https://docs.python.org/3/library/operator.html#operator.methodcaller" rel="nofollow"><code>operator.methodcaller</code></a>):</p>
<pre><code>count = lambda x: methodcaller('count', x) # from operator import methodcaller
</code></pre>
</div>

<p>You can use -</p>
<pre><code>s = ["blah+blah","blah+++blah","blah+bl+blah","blah"]
print(sorted(s,key=lambda t: t.count('+')))
&gt;&gt;["blah","blah+blah","blah+bl+blah","blah+++blah"]
</code></pre>
