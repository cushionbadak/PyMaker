<div class="post-text" itemprop="text">
<p>Firstly don't name your variable <code>str</code> as it shadows the built-in. </p>
<p>To solve your problem you can use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow"><code>ast.literal_eval</code></a></p>
<pre><code>&gt;&gt;&gt; a = "[[1, 2], [2.0, 0.3], ['a', 'b']]"
&gt;&gt;&gt; import ast
&gt;&gt;&gt; ast.literal_eval(a)
[[1, 2], [2.0, 0.3], ['a', 'b']]
</code></pre>
<p>To address your latest edit</p>
<pre><code>&gt;&gt;&gt; str_sample = "[[1, 2], [2.0, 0.3], ['a', 'b', [None, (1, 3)], {'c': 'd'}]]"
&gt;&gt;&gt; ast.literal_eval(str_sample)
[[1, 2], [2.0, 0.3], ['a', 'b', [None, (1, 3)], {'c': 'd'}]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/2/library/functions.html#eval" rel="nofollow"><code>eval</code></a>, but this is not a <a href="http://nedbatchelder.com/blog/201206/eval_really_is_dangerous.html" rel="nofollow">good practice</a></p>
<pre><code>eval("[[1, 2], [2.0, 0.3], ['a', 'b']]")
[[1, 2], [2.0, 0.3], ['a', 'b']]
</code></pre>
