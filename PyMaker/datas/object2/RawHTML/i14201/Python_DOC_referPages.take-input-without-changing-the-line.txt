<div class="post-text" itemprop="text">
<p>Instead of <code>print()</code>, use <code>input()</code> to produce the prompt:</p>
<pre><code>x = input("Please Enter: ")
</code></pre>
<p>From the <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow"><code>input()</code> function documentation</a>:</p>
<blockquote>
<p><code>input([prompt])</code> </p>
<p>If the <em>prompt</em> argument is present, it is written to standard output without a trailing newline.</p>
</blockquote>
