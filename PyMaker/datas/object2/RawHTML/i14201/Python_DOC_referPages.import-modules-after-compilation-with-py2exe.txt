<p>I compiled a python executable with py2exe and want to import a python module by a user input.</p>
<pre><code>userinput = input("Enter module name")
module = __import__(userinput)
</code></pre>
<p>Does this work? Or is there another way to import a module after the file is compiled?</p>
</div>
<div class="post-text" itemprop="text">
<p>You could try <a href="https://docs.python.org/3/library/importlib.html#importlib.import_module" rel="nofollow"><code>importlib.import_module()</code></a>:</p>
<pre><code>import importlib
importlib.import_module(userinput)
</code></pre>
