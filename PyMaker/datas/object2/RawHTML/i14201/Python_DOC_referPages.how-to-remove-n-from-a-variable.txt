<div class="post-text" itemprop="text">
<p>Lines read from a file include the line separators. You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.rstrip" rel="nofollow"><code>str.rstrip()</code></a> to remove it:</p>
<pre><code>z, x, y, dist = line.rstrip().split(',')
</code></pre>
<p>This will remove <em>all</em> whitespace (spaces, tabs, line separators, etc.) from the end of the line before splitting the line by comma.</p>
<p>Your file format is perhaps more easily read with the <a href="https://docs.python.org/3/library/csv.html" rel="nofollow"><code>csv</code> module</a>:</p>
<pre><code>import csv

with open('data.txt', newline='') as csvfile:
    reader = csv.reader(csvfile)
    for row in reader:
        z, x, y, dist = map(int, row)
        print(z)
        print(x)
        print(y)
        print(dist)
        speed = dist / (y - x)
        print(speed)
</code></pre>
