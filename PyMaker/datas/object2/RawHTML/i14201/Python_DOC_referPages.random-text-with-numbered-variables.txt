<div class="post-text" itemprop="text">
<p>Do this instead:</p>
<pre><code>source1 = "First text"
source2 = "Second text"    
randomtext = random.choice([source1, source2])
liste_source = randomtext.rstrip('\n\r').split(" ")
</code></pre>
<p>Or even simpler:</p>
<pre><code>sources = ["first text", "second text"]
randomtext = random.choice(sources)
liste_source = randomtext.rstrip('\n\r').split(" ")
</code></pre>

<div class="post-text" itemprop="text">
<p>Instead of making multiple variables, use a list:</p>
<pre><code>source = [
    'First text',
    'Second text'
]

randomtext = source[randint(len(source))]
</code></pre>
<p>You can also use <a href="https://docs.python.org/3/library/random.html#random.choice" rel="nofollow"><code>random.choice</code></a> instead:</p>
<pre><code>randomtext = random.choice(source)
</code></pre>
