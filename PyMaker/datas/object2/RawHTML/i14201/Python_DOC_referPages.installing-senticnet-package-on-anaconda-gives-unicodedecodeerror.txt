<div class="post-text" itemprop="text">
<p>This is due to Python not opening the file as UTF-8 when reading.  It appears like Windows does not use Unicode by default so you have to pass in <code>encoding='utf-8'</code> to the call to open.  Looking at your error code, you need to change line 20 in <code>setup.py</code> to</p>
<pre><code>license=open('LICENSE', encoding='utf-8').read()
</code></pre>
<p>For more information, checkout <a href="https://docs.python.org/3/howto/unicode.html#reading-and-writing-unicode-data" rel="nofollow">Python's Unicode HOWTO</a> specifically the reading and writing of data.</p>
</div>
<div class="post-text" itemprop="text">
<p>Clone the github repository from:</p>
<pre><code>https://github.com/yurimalheiros/senticnetapi
</code></pre>
<p>After unzipping the file, open setup.py and replace line number 20 with:</p>
<p>license=open('LICENSE', encoding='utf-8').read()</p>
<p>after this, open command prompt here(make sure you have environment variable set for python)</p>
<p>run the command -&gt; 'python setup.py install' </p>
<p>This will install senticnet for python, now to get it running in Anaconda, copy the following folder:</p>
<p>C:\Users\HP\AppData\Local\Programs\Python\Python36-32\Lib\site-packages\senticnet-1.3-py3.6.egg\senticnet</p>
<p>to:</p>
<p>C:\Users\HP\Anaconda4\Lib\site-packages\senticnet
(the path will be different based upon your configurations)</p>
