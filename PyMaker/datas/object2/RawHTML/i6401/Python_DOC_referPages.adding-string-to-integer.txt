<div class="post-text" itemprop="text">
<p>In python (and a lot of other languages), the <code>+</code> operator serves a dual purpose. It can be used to get the sum of two numbers (number + number), or concatenate strings (string + string). In this case, python can't decide what the <code>+</code> should do, as one of your operands is a number, and the other is a string. </p>
<p>To fix this, you'll have to alter one operand to match the type of the other. In this case, your only option is to make the number into a string (easily done using the built-in <a href="https://docs.python.org/3/library/functions.html#func-str" rel="nofollow noreferrer"><code>str()</code></a> function:</p>
<pre><code>str(int(((score)/5)*100)) + "%"
</code></pre>
<p>Or, you can ditch <code>+</code> entirely and go with format syntax.</p>
<p><strong>Old syntax:</strong></p>
<pre><code>"%d%%" % int(((score)/5)*100)
</code></pre>
<p><strong>New syntax:</strong></p>
<pre><code>'{}%'.format(int(((score)/5)*100))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As the error says, you cannot apply the <code>+</code> operator between an int and a string. You could, however, convert the int to a string yourself:</p>
<pre><code>percentage = str(int(((score)/5)*100)) + ("%")
