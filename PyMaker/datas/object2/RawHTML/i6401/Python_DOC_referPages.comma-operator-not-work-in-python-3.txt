<p>In <strong>python 2</strong>, using a <strong>comma</strong> operator at the end of the statement to print the output in horizontal line. Like this:</p>
<pre><code>for i in range(5):
    print i,
</code></pre>
<p>And Above code would generate the following output:</p>
<pre><code>0 1 2 3 4
</code></pre>
<p>But, in <strong>Python 3</strong> comma operator not work to print the output in horizontal line. </p>
<pre><code>for i in range(5):
    print (i),
</code></pre>
<p>Generate the following output:</p>
<pre><code>0
1
2
3
4
</code></pre>
<p>So, How to print output in horizontal line in python 3?</p>
</div>
<div class="post-text" itemprop="text">
<p>In Python 3 you can pass the end argument to the <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer">print</a> function to change the end of line character</p>
<pre><code>for i in range(5):
    print(i, end=" ")

&gt;&gt; 0 1 2 3 4
</code></pre>
</div>
<span class="comment-copy">Did you look at the documentation for the print function?</span>
<span class="comment-copy">There is no such thing as a comma <i>operator</i>. A comma is part of a specific expressions, like literals (lists, dicts, sets, tuples) or statements. In Python 3, <code>print()</code> is a <i>function</i>, and putting a comma after a function doesn't affect what the function does. In Python 2, <code>print</code> was a <i>statement</i>, where the comma plays a specific part of the statement syntax.</span>
