<p>To detect whether a python string ends with a particular substring, say <code>".txt"</code>, there is a convenient built-in python string method;</p>
<pre><code> if file_string.endswith(".txt"):
</code></pre>
<p>I would like to to detect whether a python string begins with a particular substring, say <code>"begin_like_this"</code>. I am not able find a convenient method that I can use like this;</p>
<pre><code> if file_string.beginswith("begin_like_this"):
</code></pre>
