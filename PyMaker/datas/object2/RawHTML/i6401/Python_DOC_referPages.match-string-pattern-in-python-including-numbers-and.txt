<p>I'm here to ask about how to extract specific type of string in long string in python.</p>
<p>my string = 'start 1: 1.300454 end'</p>
<p>I don't no how to extract only "1: 1.300454" in my string sample.</p>
</div>
<div class="post-text" itemprop="text">
<p>Just use <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">regular expressions</a>.</p>
<pre><code>import re

pattern = re.compile(r'^start (.*) end$')

my_string = 'start 1: 1.300454 end'

extract = pattern.match(my_string).group(1)
print(extract)
</code></pre>
<p>Prints</p>
<pre><code>1: 1.300454
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can split on the substring " end" and take the first part, then split on "start: " and take the last part:</p>
<pre><code>my_string = 'start 1: 1.300454 end'
extract = my_string.split(' end', 1)[0].split("start ", 1)[-1]
print(extract)
</code></pre>
</div>
