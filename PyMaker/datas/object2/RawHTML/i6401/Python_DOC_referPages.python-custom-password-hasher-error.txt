<div class="post-text" itemprop="text">
<p>You have missed out the comma after your custom hasher. It should be:</p>
<pre><code>'honeywordHasher.hashers.MyHoneywordHasher',
</code></pre>
<p>Without the comma, Python <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-literal-concatenation" rel="nofollow noreferrer">concatenates</a> the string with the one on the next line to form <code>'honeywordHasher.hashers.MyHoneywordHasherdjango.contrib.auth.hashers.PBKDF2PasswordHasher'</code>, which causes the import error.</p>
