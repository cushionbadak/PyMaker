<div class="post-text" itemprop="text">
<p>The problem is the use of <code>==</code> where you need <code>=</code>.  This normally wouldn't cause a syntax error, but in your case, you have:</p>
<pre><code>a == not(b)
</code></pre>
<p>which is the same as:</p>
<pre><code>a == not b
</code></pre>
<p>This groups as:</p>
<pre><code>(a == not) b
</code></pre>
<p>and that causes the syntax error.</p>
<p>An assignment operator, on the other hand, has lower precedence, so:</p>
<pre><code>a = not b
</code></pre>
<p>groups as:</p>
<pre><code>a = (not b)
</code></pre>
<p>which is fine.</p>
