<p>I'm using anaconda to manage a virtual environment. I'm developing some Jupyter Notebooks and I'm using some functions in multiple Notebooks. So I want to extract these functions in a package, lets call it <code>mytools</code> and <code>import mytools</code> in my Notebooks. I installed the tools package from the parent dir with:</p>
<pre><code>source activate myenv
pip install -e ./mytools/    
</code></pre>
<p>And run the jupyter notebook server </p>
<pre><code>source activate myenv
jupyter notebook
</code></pre>
<p>When I execute <code>import mytools</code> I get the error:</p>
<pre><code>---------------------------------------------------------------------------
ImportError                               Traceback (most recent call last)
&lt;ipython-input-4-cbc927808b4d&gt; in &lt;module&gt;()
----&gt; 1 import mytools

ImportError: No module named 'mytools'
</code></pre>
<p>Why can't I import it in the juypter notebook?</p>
<p>When I run <code>pip list</code> in the conda environment I can see the installed package. </p>
<p>When I run:</p>
<pre><code>import sys
sys.executable
</code></pre>
<p>it points to the myenv environment <code>/Users/me/anaconda/envs/myenv/bin/python</code></p>
<p>My Folder structure:</p>
<pre><code>├── Notebook Project1
│    └── Notebook1.ipynb
├── Notebook Project2
│    └── Notebook2.ipynb
└── mytools
     ├── __init__.py
     ├── module1
     │    ├── modulecode.py
     │    └── __init__.py
     ├── module2
     │    ├── modulecode.py
     │    └── __init__.py
     └── setup.py
</code></pre>
