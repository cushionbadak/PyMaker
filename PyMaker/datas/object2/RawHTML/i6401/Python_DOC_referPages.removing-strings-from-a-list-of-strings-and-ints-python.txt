<div class="post-text" itemprop="text">
<p>I have a list such as <code>['A', 1, 'B', 2, 'C', 3]</code></p>
<p>I want a function so that I can remove the strings from my list and make it only have integers? There are always alternating strings and integers in the list if that helps.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can simply loop through your list and only keep the elements with type integer:</p>
<pre><code>original_list = ['A', 1, 'B', 2, 'C', 3]

[n for n in original_list if type(n) is int]
</code></pre>
<p>Which outputs</p>
<pre><code>[1, 2, 3]
</code></pre>
<p>If you want to wrap it to a function you can use this:</p>
<pre><code>def remove_all_but_int(my_list):
    my_list = [n for n in my_list if type(n) is int]
    return (my_list)
</code></pre>
<p>new_list = remove_all_but_int(original_list)</p>
<p>Which gives you the same result. </p>
</div>

<p>Just one more option - use of <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer">filter</a> function:</p>
<pre><code>orig = ['A', 1, 'B', 2, 'C', 3]

<p>One line solution :</p>
<pre><code>list_1=['A', 1, 'B', 2, 'C', 3]



print([i for i in list_1 if not isinstance(i,str)])
</code></pre>
<p>if you are having trouble with understanding above syntax here is detailed solution :</p>
<pre><code>new_list=[]

for i in list_1:
    if not isinstance(i,str):
        new_list.append(i)

print(new_list)
</code></pre>
<p>output:</p>
<pre><code>[1, 2, 3]
