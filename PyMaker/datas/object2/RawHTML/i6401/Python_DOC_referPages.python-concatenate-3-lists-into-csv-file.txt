<div class="post-text" itemprop="text">
<p>I have three lists:</p>
<p><code>name = ['rob', 'mike', 'bob']
age = ['19, '32', '88']
id = ['aaa', 'bbb', 'ccc']</code></p>
<p>Is there a way to concatenate these vertically and get a CSV as below?</p>
<p><code>rob, 19, aaa
mike, 32, bbb
bob, 88, ccc
</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You can utilize <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer">csv</a> module and <code>zip</code> to do this.</p>
<pre><code>name = ['rob', 'mike', 'bob']
age = ['19', '32', '88']
id = ['aaa', 'bbb', 'ccc']
import csv
with open('eggs.csv', 'w', newline='') as csvfile:
    spamwriter = csv.writer(csvfile, delimiter=',',
                            quotechar='|', quoting=csv.QUOTE_MINIMAL)

    data = list(zip(name, age, id))
    for row in data:
        row = list(row)
        spamwriter.writerow(row)
print("Program completed")
</code></pre>
<p>Output:</p>

</div>

<pre><code>for i,j,k in zip(name, age, id):
