<div class="post-text" itemprop="text">
<p>Each <code>input()</code> prompts for a new entry from the user.</p>
<p>What your code does is, for every set of 5 entries:</p>
<ul>
<li>discard the first entry</li>
<li>check if the second entry is <code>'w'</code> then move forward</li>
<li>check if the third entry is <code>'a'</code> then move left</li>
<li>check if the fourth entry is <code>'s'</code> then move right</li>
<li>check if the fifth entry is <code>'d'</code> then move back</li>
</ul>
<p>You should assign the first <code>input()</code> to a variable, then check with that.<br/>
Use <code>elif</code> to avoid checking other characters if it already satisfies one.</p>
<pre><code>while True:
    entry = input()
    if entry == 'w':
        movemattfd()

    elif entry == 'a':
        movemattlt()

    elif entry == 'd':
        movemattrt()

    elif entry == 's':
        movemattbk()
</code></pre>
