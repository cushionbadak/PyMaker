<p>Below is an interactive demonstration - the $UID is defined on the shell, but although Python is launched from the same shell, the 'UID' environment variable does not exist in <code>os.environ</code>.</p>

<div class="post-text" itemprop="text">
<p>that's expected because <code>UID</code> is a <a href="http://tldp.org/LDP/abs/html/internalvariables.html" rel="nofollow noreferrer">shell internal variable</a>. It is not propagated to subprocesses.</p>
<p>You have to get the values directly through the <code>os</code> module with <code>os.getuid()</code></p>
</div>
<div class="post-text" itemprop="text">
<p>I think you're looking for <a href="https://docs.python.org/3/library/os.html#os.geteuid" rel="nofollow noreferrer"><code>os.getuid()</code></a></p>
