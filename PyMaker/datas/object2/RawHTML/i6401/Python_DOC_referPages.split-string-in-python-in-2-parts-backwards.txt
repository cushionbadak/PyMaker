<div class="post-text" itemprop="text">
<p>I am trying to split my string  <code>str = 'I HAVE 6'</code> in two parts. The output should be<br/>
<code>['I HAVE','6']</code><br/>
There is no info about the length of the words given as input.
One way do this is to split it in three and add the first two. Is there any other way?  </p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.rsplit" rel="nofollow noreferrer"><code>rsplit</code></a>, which splits from the right, with a <code>maxsplit</code> of <code>1</code>:</p>
<pre><code>s = 'I HAVE 6'
s.rsplit(maxsplit=1)  # or (' ', 1)
['I HAVE', '6']
</code></pre>
<p>As an aside, don't use the name <code>str</code>. It masks the built-in class <code>str</code> and will lead to errors down the line.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>re.split()</code>:</p>
<pre><code>import re
s = 'I HAVE 6'
new_s = re.split("\s(?=\d)", s)
</code></pre>
<p>Output:</p>
<pre><code>['I HAVE', '6']
</code></pre>
</div>
