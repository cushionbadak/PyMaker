<div class="post-text" itemprop="text">
<p>Try composing the rotation with a shift:</p>
<ul>
<li>first apply a shift to move the rotation axis to the center point,</li>
<li>then apply the rotation with the parameter you want,</li>
<li>last apply the opposite of the first shift.</li>
</ul>
<p><a href="https://docs.scipy.org/doc/scipy-0.16.1/reference/generated/scipy.ndimage.interpolation.shift.html" rel="nofollow noreferrer">https://docs.scipy.org/doc/scipy-0.16.1/reference/generated/scipy.ndimage.interpolation.shift.html</a> should help.</p>
