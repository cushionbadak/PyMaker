<p>I want the name <code>Snowball</code> to update <code>yellow</code> to a value of <code>2</code> (and so on). I need to iterate through all of them, do some processing and update specific columns. What's the best way to do this?</p>

<div class="post-text" itemprop="text">
<p>You can do that by using </p>
<pre><code>df.loc[df['Name']=='Snowball','yellow'] = 2
</code></pre>
<p>The first parameter of loc is the row indexer, the second one is for selecting the column</p>
</div>
<div class="post-text" itemprop="text">
<p>To index a particular cell and change its value, use df.loc:</p>
<pre><code>df.loc[df.Name == 'Snowball', 'yellow'] = 2
</code></pre>
<p>To efficiently iterate through all the rows of the data frame, use df.iterrows:</p>
<pre><code>values_to_insert_into_yellow_by_name = {'Puppy': 1, 'London': 2, 'Snowball': 2, 'Malibu': 3}
for idx, row in df.iterrows():
    name = df.loc[idx, 'Name']
    insert = values_to_insert_into_yellow_by_name[name]
    df.loc[idx, 'yellow'] = insert
</code></pre>
<p>You can use a dictionary to store values that need to be inserted.</p>
