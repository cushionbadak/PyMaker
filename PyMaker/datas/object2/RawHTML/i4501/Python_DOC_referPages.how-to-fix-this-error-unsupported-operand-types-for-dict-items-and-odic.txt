<div class="post-text" itemprop="text">
<p>You can not use <code>+</code> with dictionaries when trying to add to one another (unlike lists which support <code>+</code> operator).</p>
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#dict.update" rel="nofollow noreferrer"><code>dict.update</code></a> instead:</p>
<pre><code>In [47]: from collections import OrderedDict
    ...: o = OrderedDict([(1,2), (3, 4)])
    ...: d = {5:6, 7:8}
    ...: 

In [48]: o+d
TypeError: unsupported operand type(s) for +: 'collections.OrderedDict' and 'dict'

In [49]: o.update(d)

In [50]: print(o)
OrderedDict([(1, 2), (3, 4), (5, 6), (7, 8)])
</code></pre>
<p>Alternatively:</p>
<pre><code>In [52]: from collections import OrderedDict
    ...: o = OrderedDict([(1,2), (3, 4)])
    ...: d = {5:6, 7:8}
    ...: 

In [53]: d.update(o); print(d)
{1: 2, 3: 4, 5: 6, 7: 8}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>OR operator</code>: <code>|</code> to <a href="http://treyhunner.com/2016/02/how-to-merge-dictionaries-in-python/" rel="nofollow noreferrer">merge the 2 dictionaries in python 3</a></p>
<pre><code>metadata = {
    "pk": smart_text(obj._get_pk_val(), strings_only=True),
    "model": smart_text(obj._meta),
    "title": smart_text(obj.training_type.name)
}
return dict(metadata.items() | self._current.items())
</code></pre>
