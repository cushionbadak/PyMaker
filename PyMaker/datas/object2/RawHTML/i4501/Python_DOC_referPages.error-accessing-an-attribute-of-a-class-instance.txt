<div class="post-text" itemprop="text">
<p>There are 2 problems with your code:</p>
<ol>
<li><code>init</code> needs to be surrounded by double underscore.</li>
<li>An instance variable needs to be accessed via the class instance, not the class itself.</li>
</ol>
<p>Try this instead:</p>
<pre><code>class chainlist:

    def __init__(self):
        self.chain = [6 ,7]

blockchain = chainlist()

print(blockchain.chain)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>print(blockchain.chain)</code> not <code>chainlist.chain</code>. Here is an example how you can access different variables in a class.</p>
<pre><code>class chainlist:
    myvar = [0, 1] # class variable shared by all instances
    def __init__(self):
        self.chain = [6 ,7]  # instance variable unique to each instance

blockchain = chainlist()
print(blockchain.chain)
print(chainlist.myvar)
</code></pre>
<p>Please read this <a href="https://docs.python.org/3/tutorial/classes.html" rel="nofollow noreferrer">'Class and Instance Variables'</a> to have clear understanding.</p>
