<p>I am scraping data from websites, and searching for a table with a certain id. I have something like:</p>
<pre><code>table = pd.read_html(page,attrs={'id': 'SpecificID'})[0]
</code></pre>
<p>The problem is that if the table with that id does not exist, my script stops with the following error message:</p>
<pre><code>ValueError: No tables found
</code></pre>
<p>Is there a way I can catch the error code for pd.read_html? Something like:</p>
<pre><code>if pd.read_html(page,attrs={'id': 'SpecificID'})[0]:
    # No error
    table = pd.read_html(page,attrs={'id': 'SpecificID'})[0]
else:
    # Error
    print("Error") 
</code></pre>
<p>Any help would be appreciated. Thanks.</p>

<div class="post-text" itemprop="text">
<p>Just use a try statement:</p>
<pre><code>try:
    # No error
    table = pd.read_html(page,attrs={'id': 'SpecificID'})[0]
except:
    # Error
    print("Error") 
</code></pre>
</div>
<span class="comment-copy">You should use a <a href="https://docs.python.org/3/tutorial/errors.html" rel="nofollow noreferrer"><code>try</code> statement</a></span>
