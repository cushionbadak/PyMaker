<pre><code>&lt;div xmlns="" style="box-sizing: border-box; width: 100%; margin: 0 0 10px 0; padding: 5px 10px; background: #fdc431; font-weight: bold; font-size: 14px; line-height: 20px; color: #fff;"&gt;42263 - Unencrypted Telnet Server&lt;/div&gt;

<p>I am trying to print <code>42263 - Unencrypted Telnet Server</code> using Beautiful Soup but the output is an empty element i.e, <code>[]</code></p>
<p>This is my Python code:</p>
<pre><code>from bs4 import BeautifulSoup
import csv

with open(r"C:\Users\sourabhk076\Documents\CBS_1.html") as fp:
    soup = BeautifulSoup(fp.read(), 'html.parser')

divs = soup.find_all('div', attrs={'background':'#fdc431'})

print(divs)
</code></pre>

<div class="post-text" itemprop="text">
<p><code>background</code> is not an attribute of the <code>div</code> tag. The attributes of the <code>div</code> tag are:</p>
<pre><code>{'xmlns': '', 'style': 'box-sizing: border-box; width: 100%; margin: 0 0 10px 0; padding: 5px 10px; background: #fdc431; font-weight: bold; font-size: 14px; line-height: 20px; color: #fff;'}
</code></pre>
<p>So, either you'll have to use</p>
<pre><code>soup.find_all('div', attrs={'style': 'box-sizing: border-box; width: 100%; margin: 0 0 10px 0; padding: 5px 10px; background: #fdc431; font-weight: bold; font-size: 14px; line-height: 20px; color: #fff;'}
</code></pre>
<p>or, you can use the <code>lambda</code> function to check if <code>background: #fdc431</code> is in the <code>style</code> attribute value, like this:</p>
<pre><code>soup = BeautifulSoup('&lt;div xmlns="" style="box-sizing: border-box; width: 100%; margin: 0 0 10px 0; padding: 5px 10px; background: #fdc431; font-weight: bold; font-size: 14px; line-height: 20px; color: #fff;"&gt;42263 - Unencrypted Telnet Server&lt;/div&gt;', 'html.parser')
print(soup.find(lambda t: t.name == 'div' and 'background: #fdc431' in t['style']).text)
# 42263 - Unencrypted Telnet Server
</code></pre>
<p>or, you can use RegEx, as shown by <a href="https://stackoverflow.com/a/49669665/7832176">Jatimir in his answer</a>.</p>

<div class="post-text" itemprop="text">
<p>Solution with <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">regexes</a>:</p>
<pre><code>from bs4 import BeautifulSoup
import re

with open(r"C:\Users\sourabhk076\Documents\CBS_1.html") as fp:
    soup = BeautifulSoup(fp.read(), 'html.parser')
</code></pre>
<p>Let's find the div that matches the following regular expression: <code>background:\s*#fdc431;</code>. <code>\s</code> matches a single Unicode whitespace character. I assumed that there can be 0 or more whitespaces so I added the <code>*</code> modifier to match 0 or more repetitions of the preceding RE. You can read more about regexes <a href="https://docs.python.org/3/library/re.html#regular-expression-syntax" rel="nofollow noreferrer">here</a> as they sometimes come in handy. I also recommend you this <a href="https://regex101.com/" rel="nofollow noreferrer">online regex tester</a>.</p>
<pre><code>div = soup.find('div', attrs={'style': re.compile(r'background:\s*#fdc431;')})
</code></pre>
<p>This however is equivalent to:</p>
<pre><code>div = soup.find('div', style=re.compile(r'background:\s*#fdc431;'))
</code></pre>
<p>You can read about that in the official documentation of <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/#the-keyword-arguments" rel="nofollow noreferrer">BeautifulSoup</a></p>
<p>Worth reading are also the sections about the <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/#kinds-of-filters" rel="nofollow noreferrer">kinds of filters</a> you can provide to the <code>find</code> and other similar methods.</p>
<p>You can supply either a string, regular expression, list, <code>True</code> or a function, as shown by <a href="https://stackoverflow.com/questions/49669516/beautiful-soup-returns-nothing/49669683#49669683">Keyur Potdar</a> in his anwser.</p>
<p>Assuming the div exists we can get its text by:</p>
<pre><code>&gt;&gt;&gt; div.text
'42263 - Unencrypted Telnet Server'
</code></pre>
