<p>I don't think a simple reference to <a href="https://docs.python.org/3/library/stdtypes.html#dict.items" rel="nofollow noreferrer"><code>dict.items()</code></a> should be counted a full fledged answer but since you insist:</p>
<pre><code>state_values = [value for key, value in kwargs.items() if key.startswith("state_")]
</code></pre>
<p>The issue was due to <code>kwargs</code> being a <code>dict</code>, and if you want to do a parameter expansion into <code>key</code> and <code>value</code>, you have to use <code>kwargs.items()</code>. This will return a list of key-value tuples which can then be readily expanded into, well, <code>key</code> and <code>value</code>.</p>
<p>Beware that in Python 2.x <a href="https://docs.python.org/2/library/stdtypes.html#dict.items" rel="nofollow noreferrer"><code>dict.items()</code></a> creates a whole list of key-value tuples instead of just giving you an iterator. To avoid needless resource usage, use <a href="https://docs.python.org/2/library/stdtypes.html#dict.iteritems" rel="nofollow noreferrer"><code>dict.iteritems()</code></a> instead.</p>

<p>Kwargs is a dict, so just use <code>items()</code>  in order to iterate over it.</p>
<p><code>state_values = [value for key, value in kwargs.items() if key.startswith("state_")]</code></p>
