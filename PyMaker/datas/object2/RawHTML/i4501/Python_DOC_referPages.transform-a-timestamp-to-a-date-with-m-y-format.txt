<div class="post-text" itemprop="text">
<p>What is the best way to transform a <code>timestamp</code> (for instance: <code>2015-07-01 00:00:00</code>) to a date with the following format <code>"03/2017"</code> in Python 3? </p>
</div>
<div class="post-text" itemprop="text">
<p>If I understand correctly, by <code>timestamp</code> you might mean a variable of <code>datetime</code> type. (considering your example)</p>
<p>The <code>datetime</code> class has a method <code>strftime</code>. strftime() is the method you are looking for.</p>
<p>For this specific example, it would look something like:</p>
<pre><code>your_datetime.strftime("%m / %Y")
</code></pre>
<p>for more information read <a href="https://docs.python.org/2/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">the docs</a>.</p>
<p>If by <code>timestamp</code> you really mean a  <a href="https://en.wikipedia.org/wiki/Unix_time" rel="nofollow noreferrer">posix epoch time</a> which you want to convert, your code would look like :</p>
<pre><code>datetime.datetime.utcfromtimestamp(your_datetime).strftime("%m / %Y")
</code></pre>
<p>And also your import statement should be :</p>
<pre><code>import datetime
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do this with <code>pd.Series.dt.strftime</code>:</p>
<pre><code>import pandas as pd

s = pd.Series(pd.date_range(pd.Timestamp('now'), periods=2))

# 0   2018-03-31 22:57:30.819192
# 1   2018-04-01 22:57:30.819192
# dtype: datetime64[ns]

res = s.dt.strftime('%m/%Y')

# 0    03/2018
# 1    04/2018
# dtype: object
</code></pre>
