<p>Pass a tuple as the argument after the <code>%</code> operator.</p>
<pre><code>print("Your name and password are: %s, %s" % (name, password))
</code></pre>
<p>This is called (old) <a href="https://docs.python.org/3/library/stdtypes.html#old-string-formatting" rel="nofollow noreferrer">printf-style String Formatting</a>, and it can get pretty fancy.</p>
