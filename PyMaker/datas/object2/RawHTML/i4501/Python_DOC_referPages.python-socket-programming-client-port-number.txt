<div class="post-text" itemprop="text">
<p>It shouldn't be hard to find considering <a href="https://docs.python.org/3/library/socket.html#socket.socket.accept" rel="nofollow noreferrer">socket.accept</a> returns it with every accepted connection. So does <a href="https://docs.python.org/3/library/socket.html#socket.socket.recvfrom" rel="nofollow noreferrer">socket.recvfrom</a>. A typical IPv4 socket address looks like:</p>
<pre><code>&gt;&gt;&gt; socket.getaddrinfo('www.google.com','www')
[(2, 1, 6, '', ('64.233.163.99', 80)),
 (2, 1, 6, '', ('64.233.163.104', 80)),
 (2, 1, 6, '', ('64.233.163.147', 80)),
 (2, 1, 6, '', ('64.233.163.105', 80)),
 (2, 1, 6, '', ('64.233.163.103', 80)),
 (2, 1, 6, '', ('64.233.163.106', 80)),
 (10, 1, 6, '', ('2a00:1450:4010:c06::6a', 80, 0, 0))]
&gt;&gt;&gt; _[0][4]
('64.233.163.99', 80)
</code></pre>
<p>That 80 is the port number. </p>
