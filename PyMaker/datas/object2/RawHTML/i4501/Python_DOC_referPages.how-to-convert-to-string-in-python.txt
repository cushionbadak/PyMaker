<div class="post-text" itemprop="text">
<p>The default string type in Python3 is Unicode.
You can verify this by writing <code>type(u'\x03a')</code> in the <code>Python3</code> interpreter.
It would give <code>&lt;class 'str'&gt;</code> which is the same result as would be if you write <code>type('\x03a')</code>.</p>
<p>To convert the <code>Python3 string</code> to <code>bytecode</code>, you may use <code>your_string.encode()</code>.</p>
<p>For information regarding the encode function, you may follow this link.
<a href="https://docs.python.org/3/library/stdtypes.html?highlight=encode#str.encode" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html?highlight=encode#str.encode</a></p>
<p>You may also convert the Unicode string to bytecode by writing <code>b'your_string'</code></p>
<p>Also, if you just want to print these characters without the Unicode format, you may just write <code>print('\\x03a')</code>, i.e. using the backslash escape character.</p>
