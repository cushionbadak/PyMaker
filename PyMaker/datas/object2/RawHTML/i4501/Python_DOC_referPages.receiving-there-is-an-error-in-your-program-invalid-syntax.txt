<div class="post-text" itemprop="text">
<p>You're probably running your code with Python 2 as opposed to Python 3, where <a href="https://docs.python.org/3/whatsnew/3.0.html#print-is-a-function" rel="nofollow noreferrer"><code>print</code> is a function</a> and not a statement.</p>
<p>Try running your code as such on your terminal:</p>
<pre><code>$ python3 file.py
</code></pre>
<p>Where <code>file.py</code> is obviously replaced as the name of your file.</p>
<p>You might need to install Python 3.</p>
<hr/>
<p>If Python 3 is not an option for whatever reason, you <em>can</em> use <code>print</code> as a function <a href="https://docs.python.org/2/library/__future__.html" rel="nofollow noreferrer">by importing it</a>:</p>
<pre><code>&gt;&gt;&gt; from __future__ import print_function
&gt;&gt;&gt; print("Hello!", end=" ")
Hello! &gt;&gt;&gt; 
</code></pre>
<p>Note that this will need to be done (the import statement) immediately at the top of your file.</p>
