<div class="post-text" itemprop="text">
<p>Instead of finding the biggest value of the list by sorting the list, use use the max function:</p>
<pre><code>largest_val = max(list)
</code></pre>
<p>To find the first occurrence of a value in a list, use the <code>.index</code> method.</p>
<pre><code>position = list.index(largest_val)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Another solution with <code>sorted()</code> (for more generic cases e.g. if you also want the 2nd, 3rd, etc. "biggest values") :</p>
<pre><code>sorted_idx_and_val = sorted(enumerate(liste), key=lambda x:x[1])
# e.g. if liste = [2, 10, 4, 6] then enumerate(liste)   = [(0, 2), (1, 10), (2, 4), (3, 6)]
#                                and sorted_idx_and_val = [(0, 2), (2, 4), (3, 6), (1, 10)]

largest_val     = sorted_ind_and_val[-1][1]
largest_val_idx = sorted_ind_and_val[-1][0]

second_largest_val     = sorted_ind_and_val[-2][1]
second_largest_val_idx = sorted_ind_and_val[-2][0]
# ...
</code></pre>
<p><code>enumerate()</code> gives you a list containing tuples of <code>(index, value)</code>.</p>
<p>The argument <code>key</code> of <code>sorted()</code> is a lambda to "preprocess" the key values to sort - here we tell <code>sorted()</code> to use only <code>value</code> (<code>x[1]</code>) from the <code>(index, value)</code> elements(<code>x</code>).</p>
