<div class="post-text" itemprop="text">
<p>The map object in 3.6 returns an iterator. You need to iterate all its values either by printing each single item, or simply wrapping it into a <code>list( map(...., ...))</code> statement:</p>
<pre><code>n = int(input())
mat = []
for i in range(n):
    row = list ( map(int, input().split()) ) # iterate all the values immediately
    mat.append(row)

print(mat)
</code></pre>
<p>For python 2.x the <code>map</code> command produces a list directly.</p>
<p>See <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer">https://docs.python.org/3/library/functions.html#map</a></p>
