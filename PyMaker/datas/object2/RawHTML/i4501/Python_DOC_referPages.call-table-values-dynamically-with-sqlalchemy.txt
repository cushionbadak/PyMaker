<p>Let's say I have a table named "info_test" mapped like this:</p>
<pre><code>class db_info_test(BASE):

__tablename__ = "info_test"

id              = Column(Integer, primary_key=True)
name            = Column(String)
project         = Column(String)
manufacturer    = Column(String)
</code></pre>
<p>If I want to get the names of all items from that table I would do something like this:</p>
<pre><code>import db_info_test     
for row in self.session.query(db_info_test):
    print(row.name)
</code></pre>
<p>which would print:</p>
<pre><code>name_of_item_A
name_of_item_B
...
</code></pre>
<p>Now I want to create a very generall function that prints the value of a key, where the key is given by an argument of that function. Something like this:</p>
<pre><code>def get_value(self, table, key):
    for row in self.session.query(table):
        print(row.key)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>In Python <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer"><code>getattr()</code></a> is used for getting attributes based on variables. As an addition, you don't need to query the whole model object and all its attributes, if all you want is just one:</p>
<pre><code>def get_value(self, table, key):
    # Note the comma after value. It unpacks the 1-tuples.
    for value, in self.session.query(getattr(table, key)):
        print(value)
</code></pre>
</div>

<pre><code>print(dict(row)[key])
</code></pre>
<p>do the job for you?</p>
