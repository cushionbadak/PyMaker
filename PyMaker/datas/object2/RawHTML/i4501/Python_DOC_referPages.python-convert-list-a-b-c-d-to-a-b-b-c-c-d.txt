<p>I have a thousand numbers in a list like this:</p>
<pre><code>A=[1,2,3,4]
</code></pre>
<p>I want to covert this in the following format:</p>
<pre><code>desired_A=[[1,2],[2,3],[3,4]]
</code></pre>
<p>How can I do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a> and a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> like:</p>
<h3>Code:</h3>
<pre><code>desired = [[a, b] for a, b in zip(A, A[1:])]
</code></pre>
<h3>Test Code:</h3>
<pre><code>A = [1, 2, 3, 4]
desired = [[a, b] for a, b in zip(A, A[1:])]
print(desired)
</code></pre>
<h3>Result:</h3>
<pre><code>[[1, 2], [2, 3], [3, 4]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do it in one neat list comprehension. You are creating a new list where for each element in A, the new list has a list that contains that element and the element after that element.</p>
<pre><code>desired_A = [[A[i], A[i+1]] for i in range(len(A)-1)]
</code></pre>
</div>
