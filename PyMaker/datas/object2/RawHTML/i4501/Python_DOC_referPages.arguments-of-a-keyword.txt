<p>Say I have a user-defined keyword with some arguments:</p>
<pre><code>*** Keywords ***
Add
  [Arguments]    ${foo}  ${bar}=${42}
  [return]       ${${foo} + ${bar}}
</code></pre>
<p>No in a python context, I have a dictionary of arguments and want to use it to call the keyword:</p>
<pre><code>from robot.libraries.BuiltIn import BuiltIn
def foo():
    args = {'foo' : 0}
    BuiltIn.run_keyword("Add", args) # does not work, of course
</code></pre>
<p>This approach fails because robot expects arguments as a list (either named or ordered correctly). I cannot simply pass all arguments as a list here, because robot will not ignore arguments that are not declared in the keyword.</p>
<p>What I need to do is to</p>
<ul>
<li>Lookup the keyword</li>
<li>Iterate over its arguments</li>
<li>Create the argument list from the dictionary</li>
</ul>
<p>Is that possible with the python API?</p>
</div>
<div class="post-text" itemprop="text">
<p>You want <a href="https://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists" rel="nofollow noreferrer">unpacking</a> <code>*</code> for lists and <code>**</code> for dicts:</p>
<pre><code>def my_func(a, b, c='c', d='d'):
    print(a, b, c, d)

my_func('a', 'b')
&gt;&gt; a b c d

my_list = ['A', 'B']
my_dict = {'c': 'C', 'd': 'D'}
my_func(*my_list, **my_dict)
&gt;&gt; A B C D
</code></pre>
</div>
