<p>I didn't see any specific single function to handle that format, but tweaking the text a little and using a custom format will get you what you need.</p>
<pre><code>from datetime import datetime 
txt = "2018-04-03T14:10:00-07:00"
txtfmt = txt[:10]+ " " + txt[11:19] + " " + txt[19:22] + txt[-2:]
dt = datetime.strptime(txtfmt,"%Y-%m-%d %H:%M:%S %z")
</code></pre>
<p>Hope it helps!</p>

<p>You might want to give <a href="http://arrow.readthedocs.io/en/latest/" rel="nofollow noreferrer">Arrow</a> a try.</p>
<pre><code>&gt;&gt;&gt; import arrow
&gt;&gt;&gt; d = arrow.get('2018-04-03T14:10:00-07:00')
&gt;&gt;&gt; d
&lt;Arrow [2018-04-03T14:10:00-07:00]&gt;
&gt;&gt;&gt; d.humanize()
'3 hours ago'
&gt;&gt;&gt; d.format()
'2018-04-03 14:10:00-07:00'
&gt;&gt;&gt;
</code></pre>
<p>For Django you can use it with <a href="https://github.com/gizmag/django-arrow-field" rel="nofollow noreferrer">django-arrow-field</a> for easier access.</p>
