<p>If I understand you correctly, I think you're trying to access one python module from another python module (each "module" is a file), and each module contains a class.</p>
<p>If class A is <code>Foo</code>, saved in <code>foo.py</code>, and class B is <code>Bar</code> saved in <code>bar.py</code>, and you want to use <code>Foo</code> inside of <code>Bar</code>, then you need to <code>import</code> <code>Foo</code> such that it is in scope when <code>Bar</code> is defined.</p>
<p>You might try, in <code>bar.py</code>, above where <code>Bar</code> is defined: <code>from foo import Foo</code></p>
<p>For more information, this might help: <a href="https://docs.python.org/3/tutorial/modules.html" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/modules.html</a></p>
