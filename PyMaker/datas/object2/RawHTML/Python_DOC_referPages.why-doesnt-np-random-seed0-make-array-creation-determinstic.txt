<div class="post-text" itemprop="text">
<p>You have to reseed the RNG every time.</p>
<pre><code>import numpy as np 
np.random.seed(0)
error = np.random.rand(2,5)
np.random.seed(0)
error2 = np.random.rand(2,5)
print('error: \n' ,error)
print('error2: \n' ,error2)
</code></pre>
<p>Otherwise you're just continuing the random sequence, not starting over.</p>

<div class="post-text" itemprop="text">
<p>If a (pseudo) random number generator is (re)initialized with the same seed, it will produce the <strong>same sequence of numbers</strong>.</p>
<pre><code>np.random.seed(0)
print (np.random.rand(),np.random.rand())
</code></pre>
<p><code>0.5488135039273248 0.7151893663724195</code></p>
<pre><code>np.random.seed(0)
print (np.random.rand(),np.random.rand())
</code></pre>
<p><code>0.5488135039273248 0.7151893663724195</code></p>
<p>It will not be a (pseudo) random number generator if it generates the same random number every time for a seed. It will be some deterministic function <code>f(seed)</code>. When you seed it, the sequence of number are deterministic but the numbers them self are random. </p>
</div>
<span class="comment-copy">Each created random number changes the internal state of the random number generator (otherwise it would return the same number again and again). You must reset it again (by setting seed) after the first array creation to get the same data again.</span>
