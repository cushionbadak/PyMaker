<div class="post-text" itemprop="text">
<p>In <strong>all versions of Python</strong>, you can use the function 
<a href="https://docs.python.org/2/library/binascii.html#binascii.a2b_hex" rel="nofollow noreferrer"><code>binascii.a2b_hex()</code></a> (also known as <a href="https://docs.python.org/2/library/binascii.html#binascii.unhexlify" rel="nofollow noreferrer"><code>binascii.unhexlify()</code></a>):</p>
<pre><code>&gt;&gt;&gt; import binascii
&gt;&gt;&gt; s = '356a192b7913b04c54574d18c28d46e6395428ab'
&gt;&gt;&gt; binascii.a2b_hex(s)
'5j\x19+y\x13\xb0LTWM\x18\xc2\x8dF\xe69T(\xab'
</code></pre>
<p>In <strong>Python 3.x</strong>, you can use <a href="https://docs.python.org/3/library/stdtypes.html#bytes.fromhex" rel="nofollow noreferrer"><code>bytes.fromhex(s)</code></a>.</p>
<p>In <strong>Python 2.x</strong>, you can use the <code>hex</code> str-to-str codec:</p>
<pre><code>&gt;&gt;&gt; s.decode("hex")
'5j\x19+y\x13\xb0LTWM\x18\xc2\x8dF\xe69T(\xab'
</code></pre>
<p>The codec internally calls <code>binascii.a2b_hex()</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>With <code>binascii</code> module:</p>
<pre><code>&gt;&gt;&gt; from binascii import unhexlify
&gt;&gt;&gt; s = '356a192b7913b04c54574d18c28d46e6395428ab'
&gt;&gt;&gt; unhexlify(s)
'5j\x19+y\x13\xb0LTWM\x18\xc2\x8dF\xe69T(\xab'
</code></pre>
