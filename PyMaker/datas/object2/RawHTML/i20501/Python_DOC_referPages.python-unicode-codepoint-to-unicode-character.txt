<div class="post-text" itemprop="text">
<p><strong>Python 2</strong>: Use <a href="http://docs.python.org/2/library/functions.html#unichr" rel="nofollow noreferrer"><code>unichr()</code></a>:</p>
<pre><code>&gt;&gt;&gt; print(unichr(1081))
Ð¹
</code></pre>
<p><strong>Python 3</strong>:  Use <a href="https://docs.python.org/3/library/functions.html#chr" rel="nofollow noreferrer"><code>chr()</code></a>:</p>
<pre><code>&gt;&gt;&gt; print(chr(1081))
Ð¹
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>So the answer to the question is:</p>
<ol>
<li>convert the hexadecimal value to decimal with <code>int(hex_value, 16)</code></li>
<li>then get the corresponding strin with <code>chr()</code>.</li>
</ol>
<p>To sum up:</p>
<pre><code>&gt;&gt;&gt; print(chr(int('0x897F', 16)))

</code></pre>
