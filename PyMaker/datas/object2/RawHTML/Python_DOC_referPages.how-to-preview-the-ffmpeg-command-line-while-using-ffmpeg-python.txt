<div class="post-text" itemprop="text">
<p>Let's say I implement the following python script that uses <a href="https://github.com/kkroening/ffmpeg-python" rel="nofollow noreferrer">ffmpeg-python</a>:</p>
<pre><code># facetime.py

import ffmpeg

ffmpeg \
    .input(
        'FaceTime',
        format='avfoundation',
        pix_fmt='uyvy422',
        framerate=30
    ) \
    .output(
        'out.mp4',
        pix_fmt='yuv420p',
        vframes=100
    ) \
    .run()
</code></pre>
<p>Is there a mechanism to print to the stdout the corresponding <code>ffmpeg</code> command that will be executed?</p>
</div>
<div class="post-text" itemprop="text">
<p>Found it:</p>
<pre><code>cmd = ffmpeg \
    .input(
        'FaceTime',
        format='avfoundation',
        pix_fmt='uyvy422',
        framerate=30
    ) \
    .output(
        'out.mp4',
        pix_fmt='yuv420p',
        vframes=100
    )

args = cmd.get_args()
print(f'Args: {args}')
</code></pre>
<p>Output:</p>
<pre><code>Args: ['-f', 'avfoundation', '-framerate', '30', '-pix_fmt', 'uyvy422', '-i', 'FaceTime', '-pix_fmt', 'yuv420p', '-vframes', '100', 'out.mp4']
</code></pre>
</div>
