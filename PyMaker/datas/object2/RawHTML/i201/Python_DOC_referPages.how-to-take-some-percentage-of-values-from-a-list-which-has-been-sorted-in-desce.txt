<p>You need to use list slicing:</p>
<pre><code>mylist = [717.1573237876664, 182.7371943521783, 159.83445384690813, 51.76429530389303, 45.6779696561034, 30.617835033786744, 28.400451713817905, 25.784283394118074, 19.37917065]

<div class="post-text" itemprop="text">
<p>First, calculate the number of elements you need -</p>
<pre><code>elements = int(len(x) * (percent/100)) # x here is your list and len(x) is the length of your list
</code></pre>
<p>Next, splice the old list -</p>
<pre><code>new_list = x[:elements+1]
</code></pre>
<p>Since your own list is sorter already, we don't need to sort again. In case you do need to sort the list, see <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer">sorted</a></p>
<p>Also, see how <a href="https://python-reference.readthedocs.io/en/latest/docs/brackets/slicing.html" rel="nofollow noreferrer">slicing</a> works.</p>
