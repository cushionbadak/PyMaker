<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow noreferrer"><code>itertools.chain</code></a> in order to flatten the nested list and <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>collections.Counter</code></a> to count how many times each element occurs:</p>
<pre><code>from collections import Counter
from itertools import chain

Counter(chain(*chain(*big_list)))
</code></pre>
<p><b> Output </b></p>
<pre><code>Counter({'four': 5, 'one': 4, 'two': 4, 'seven': 4, 'eight': 4, 'three': 3, 
         'five': 3, 'six': 3, 'nine': 2})
</code></pre>
<hr/>
<p>For a solution without imports you can do something like:</p>
<pre><code>d = {}
for i in big_list:
    for j in i:
        for k in j:
            if not d.get(k):
                d[k] = 1
            else:
                d[k] += 1

print(d)
# {'one': 4, 'two': 4, 'seven': 4, 'eight': 4, 'nine': 2, 
#  'four': 5, 'three': 3, 'five': 3, 'six': 3}
</code></pre>
