<p>Your <code>strptime()</code> function needs to be formatted exactly as the timestamp is formatted, including slashes and colons. You can find the details on <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">the python documentation</a>.</p>
<p>In this case, your timestamp is formatted as <code>1/1/2016 0:00</code> but your string format <code>"%Y-%m-%d %H:%M"</code> is expecting <code>2016-1-1 0:00</code>. If you use <code>'%d/%m/%Y %H:%M'</code> as your format string then the <code>strptime()</code> function works as expected. For example:</p>
<pre><code>import datetime as dt

with open('datac.csv','r') as file:
    for line in file:
        try:
            time = line.split(',')[0] #splits the line at the comma and takes the first bit
            time = dt.datetime.strptime(time, '%d/%m/%Y %H:%M')
            print(time)
        except:
            pass
</code></pre>
<p>You should be able to adapt your code with this in mind.</p>
