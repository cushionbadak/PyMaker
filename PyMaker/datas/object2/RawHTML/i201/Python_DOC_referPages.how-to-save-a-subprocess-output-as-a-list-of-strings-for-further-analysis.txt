<p>I think you can do what you want by adding the following line before printing the output:</p>
<pre><code>output = output.decode().splitlines()
</code></pre>
<p>Calling <code>decode()</code> will turn the <code>bytes</code> into a python string (<code>str</code>), and the <code>splitlines()</code> turns that into a list-of-strings.</p>
<p><a href="https://docs.python.org/3/library/stdtypes.html#bytes.decode" rel="nofollow noreferrer"><code>decode()</code></a> is <code>bytes</code> method that decodes them into a string assuming they've been encoded in utf-8 (by default) â€” it's not documented in the <code>subprocessing</code> documentation (that I know of).</p>
