<div class="post-text" itemprop="text">
<p>In Javascript, dictionary keys are implicitly converted to strings. This isn't the case in Python. So the equivalent Python code would be:</p>
<pre><code>MOVES_RBY = {
  '(No Move)': {
    'bp': 0,
    'type': 'Normal',
    'category': 'Physical'
  },
}
</code></pre>
