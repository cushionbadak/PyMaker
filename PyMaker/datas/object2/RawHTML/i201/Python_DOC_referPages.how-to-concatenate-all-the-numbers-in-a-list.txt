<p>You can achieve concatenation in a single for loop. <br/>Try Below Code:</p>
<pre><code>import random
key_num = []
for initial_num in range(7):
    key_num.append(random.randrange(1,26))
def value(n):
    return (chr(int(n) + 64))
final_string = ""
for i in range(7): # Range of Numbers you want
    final_string += value(key_num[i]) # Concatenate random character
print final_string
</code></pre>

<p>You've tagged your question Python2.7. If you can use Python3 instead, you can use <a href="https://docs.python.org/3/library/random.html#random.choices" rel="nofollow noreferrer">random.choices</a> to make things rather simple:</p>
<pre><code>from string import ascii_lowercase
from random import choices

print ("".join(choices(ascii_lowercase,k=7)))
</code></pre>
<p>If you're stuck in Python2, you can implement a (simplistic) <code>random.choices</code> thus:</p>
<pre><code>from random import choice

def choices(s,k=1):
  return [choice(s) for _ in xrange(k)]
</code></pre>

<p>this is a way:</p>
<pre><code>from random import randrange
from string import ascii_lowercase

def random_str(length=7):
    return ''.join(ascii_lowercase[randrange(len(ascii_lowercase))] for _ in range(length))
</code></pre>
<p><code>ascii_lowercase</code> is just <code>'abcdefghijklmnopqrstuvwxyz'</code>; i select a random entry from that and join the single letters with <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>str.join</code></a>.</p>
<p>...actually <a href="https://docs.python.org/3/library/random.html?highlight=randint#random.choice" rel="nofollow noreferrer"><code>random.choice</code></a> is better suited for that. see <a href="https://stackoverflow.com/a/54961462/4954037">this answer</a>.</p>
