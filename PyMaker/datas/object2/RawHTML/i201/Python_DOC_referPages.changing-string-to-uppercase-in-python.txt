<div class="post-text" itemprop="text">
<p>I' trying to change my variable to uppercase and I don't know what I'm getting wrong.  Sample code: </p>
<pre><code>Quacks = "time"
Quacks.upper()
print(Quacks)
</code></pre>
<p>When I run the above code, <code>time</code> is returned instead of <code>TIME</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Method <a href="https://www.w3schools.com/python/ref_string_upper.asp" rel="nofollow noreferrer">upper()</a> does not change the value of variable, it only returns a string where all characters are in upper case.</p>
<p>So, you may print it directly:</p>
<pre><code>print(Quacks.upper())
</code></pre>
<p>or change first and then print:</p>
<pre><code>Quacks = Quacks.upper()
print(Quacks)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>In Python <code>str.upper()</code> returns a copy of the string with upper case values, and does not mutate the original string. In fact, strings in Python are <a href="http://interactivepython.org/runestone/static/CS152f17/Strings/StringsareImmutable.html" rel="nofollow noreferrer">immutable</a> anyways.</p>
<p>Try: <code>Quacks = Quacks.upper()</code></p>
<p>Source: <a href="https://docs.python.org/3/library/stdtypes.html" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html</a></p>
<p>"<code>str.upper() - Return a copy of the string with all the cased characters [...] converted to uppercase.</code>"</p>
