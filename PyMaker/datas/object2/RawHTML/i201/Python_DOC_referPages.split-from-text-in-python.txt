<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow noreferrer"><code>re.split</code></a>, and you also do not need to call it 3 times, use unpacking:</p>
<pre><code>import re

for input_str in 'id-name-birth', 'id_name_birth':
    ID, name, birth = re.split('[-_]', input_str)
    print(ID, name, birth)
</code></pre>
<p>Outputs</p>
<pre><code>id name birth
id name birth
</code></pre>
