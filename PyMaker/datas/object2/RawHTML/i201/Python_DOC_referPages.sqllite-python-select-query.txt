<p>If for example I had this database:</p>
<pre><code>id  result
__________
1     55
2     67
</code></pre>
<p>And a function isPrime(number) which returns a boolean true or false if the number is prime,</p>
<p>Is there anyway for me to select numbers based on if isPrime = true;</p>
<p>something like this.. </p>
<pre><code>"SELECT * FROM DB WHERE result.isPrime()" 
</code></pre>

<div class="post-text" itemprop="text">
<p>Assuming <code>conn</code> is a sqlite connection object, something like:</p>
<pre><code>conn.create_function("isprime", 1, isPrime)
</code></pre>
<p>should let you call the function in SQL statements executed by that connection.</p>
<pre><code>conn.execute("SELECT * FROM db WHERE isprime(result)")
</code></pre>
</div>
<span class="comment-copy">if <code>isPrime()</code> is a Python function, then no, you cannot call that function inside your query because the query is executed inside the database, and there is not Python interpreter or python data types inside the database.</span>
<span class="comment-copy">IIRC, the python sqlite bindings let you write user defined functions in Python that can be called from SQL statements.</span>
