<div class="post-text" itemprop="text">
<p>You need to provide a translate map with <a href="https://docs.python.org/3/library/stdtypes.html#str.maketrans" rel="nofollow noreferrer"><code>str.maketrans</code></a> in Python 3 for <code>translate</code> to work. See the <a href="https://docs.python.org/3/library/stdtypes.html#str.translate" rel="nofollow noreferrer">docs</a> for more info. Here's something you can try:</p>
<pre><code>os.rename(file_name, file_name.translate(str.maketrans('', '', '0123456789 ')))
</code></pre>
<p>The first two arguments for <code>str.maketrans</code> are irrelevant for your use so just leave them as empty strings. The third argument defines characters to be removed.</p>
