<div class="post-text" itemprop="text">
<p>Is there any way to add a item to a list of list using python. </p>
<p>As an example there is a list as below:</p>
<pre><code>test_list = [['abc','2'],['cds','333'],['efg']]
</code></pre>
<p>I want to add a item <code>'444'</code> for the position <code>test_list[2][1]</code>.</p>
<p>At last list should be:</p>
<pre><code>test_list = [['abc','2'],['cds','333'],['efg','444']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Yes that is definitely something you can do.</p>
<p>You simply access the list object that you want to add another item to by its index, and then call <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer"><code>.append()</code></a> on that list object with the new value.</p>
<pre><code>test_list = [['abc','2'], ['cds','333'], ['efg']]
test_list[2].append('444')
# test_list is now: [['abc','2'], ['cds','333'], ['efg', '444']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>test_list = [['abc','2'],['cds','333'],['efg']]
test_list[2].insert(1,"444")
print(test_list)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try using <code>append</code>, <code>append</code> does your job:</p>
<pre><code>&gt;&gt;&gt; test_list = [['abc','2'],['cds','333'],['efg']]
&gt;&gt;&gt; test_list[2].append('444')
&gt;&gt;&gt; test_list
[['abc', '2'], ['cds', '333'], ['efg', '444']]
&gt;&gt;&gt; 
</code></pre>
<p>Or use <code>+=</code>, that adds stuff together, but two lists, so do:</p>
<pre><code>&gt;&gt;&gt; test_list = [['abc','2'],['cds','333'],['efg']]
&gt;&gt;&gt; test_list[2] += ['444']
&gt;&gt;&gt; test_list
[['abc', '2'], ['cds', '333'], ['efg', '444']]
&gt;&gt;&gt; 
</code></pre>
<p><code>append</code> is a builtin python list method, here is the <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer">documentation for it</a>, and for <code>+=</code>, that is a builtin addition operator, see the <a href="https://docs.python.org/3.1/library/operator.html#operator.iadd" rel="nofollow noreferrer">documentation for it</a>.</p>
</div>
