<p>Finally found what I was looking for, using just RE lib:</p>
<p><strong>search text and get value near searchkey</strong></p>
<pre><code>words = re.findall(r'\w+', contentwhereyouwanttosearchinside)
index = words.index('SEARCHKEY')
</code></pre>
<p><strong>print next 3 word e.g. near searchkey</strong></p>
<pre><code>value1 = words[index + 1:index + 3] 
print(value1)
</code></pre>
