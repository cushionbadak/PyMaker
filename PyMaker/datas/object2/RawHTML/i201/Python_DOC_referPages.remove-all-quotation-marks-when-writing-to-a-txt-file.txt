<p>Use <code>join</code> to join the different parts of each sublist with <code>', '</code>:</p>
<pre><code>deck = [['ad','ah','ac','as'],['2d','2h','2c','2s'],['3d','3h','3c','3s'],['4d','4h','4c','4s'],['5d','5h','5c','5s'],['6d','6h','6c','6s'],['7d','7h','7c','7s'],['8d','8h','8c','8s'],['9d','9h','9c','9s'],['td','th','tc','ts'],['jd','jh','jc','js'],['qd','qh','qc','qs'],['kd','kh','kc','ks']]

with open('list.txt','w') as f:
    for sublist in deck:
        f.write(', '.join(sublist)+'\n')
</code></pre>
<p>Output (in <code>list.txt</code>):</p>
<pre><code>ad, ah, ac, as
2d, 2h, 2c, 2s
3d, 3h, 3c, 3s
...
</code></pre>
<p>Also, note that it is better to use <code>with open...</code> to make sure that your file gets closed whatever happens.</p>

<p>Simply use <code>.replace("'", "")</code> instead of <code>.strip("'")</code>. Strip only removes the characters from the left and right side of the string and not in the middle, which is why you see the array being printed as <code>ad', 'ah', 'ac', 'as</code> on each line:</p>
<pre><code>deck = [['ad','ah','ac','as'],['2d','2h','2c','2s'],['3d','3h','3c','3s'],['4d','4h','4c','4s'],['5d','5h','5c','5s'],['6d','6h','6c','6s'],['7d','7h','7c','7s'],['8d','8h','8c','8s'],['9d','9h','9c','9s'],['td','th','tc','ts'],['jd','jh','jc','js'],['qd','qh','qc','qs'],['kd','kh','kc','ks']]

f = open('list.txt','w')
for i in deck:
    a=str(i)[1:-1].replace("'", "")+"\n"
    f.writelines(a)
f.close()
</code></pre>

<p>You can work along the lines of this:</p>
<pre><code>for each in deck:
  for INDEX, EACH in enumerate(each):
    if INDEX==0:
      print(EACH,end='')
    else:
      print(','+EACH,end='')
  print('\n')
</code></pre>

<pre><code>ad,ah,ac,as
