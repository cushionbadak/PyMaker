<p>Can someone kindly explain why the following throws an exception? And what should I do with variable <code>s</code> to find out whether it contains a number?</p>
<pre><code>s = str(10)
if s.isnumeric():
    print s
</code></pre>
<p>When I read Python documentation it seems to me the above should work. See:</p>
<p><a href="https://docs.python.org/3/library/stdtypes.html?highlight=isnumeric#str.isnumeric" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html?highlight=isnumeric#str.isnumeric</a></p>
<p>But what I get is:</p>
<blockquote>
<p>"AttributeError: 'str' object has no attribute 'isnumeric'"</p>
</blockquote>
<p>Any help is most appreciated. </p>
<p>Thank you!</p>

<div class="post-text" itemprop="text">
<p>replace <code>isnumeric()</code> with <code>isdigit()</code> as</p>
<pre class="lang-py prettyprint-override"><code>s = str(10)
if s.isdigit():
    print s
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>for python 2 <strong>isdigit()</strong> and for python 3 <strong>isnumeric()</strong></p>
<blockquote>
<p>python 2</p>
</blockquote>
<pre><code>s = str(10)
if s.isdigit():
    print s
</code></pre>
<blockquote>
<p>python 3</p>
</blockquote>
<pre><code> s = str(10)
 if s.isnumeric():
    print(s)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try using <code>replace</code> with <code>isdigit</code>:</p>
<pre><code>print(s.replace('.', '').isdigit())
</code></pre>

<div class="post-text" itemprop="text">
<p>You are obviously using python 2, so a such method does not exist. </p>
<p>That said, the most pythonic way would be to try to convert it</p>
<pre><code>try:
    print(float(s))  # or int(s)
except ValueError:
    print("s must be numeric")
</code></pre>
