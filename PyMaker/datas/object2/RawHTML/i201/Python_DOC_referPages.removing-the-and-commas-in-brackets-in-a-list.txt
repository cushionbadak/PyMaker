<p>You can do this with <code>print(*pop)</code>.</p>
<p>When given several parameters (e.g., <code>print(1, 2, 3)</code>, the <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>print</code></a> command will print them all out, converted to strings, separated by spaces.</p>
<p>With the <code>*</code>, your array <code>pop</code> is unpacked so that it forms the parameters for the <code>print</code> function.</p>
<p>By setting the <code>sep</code> parameter, you can even change the separator. For example, <code>print(*pop, sep=",")</code> would give <code>1,8,6,3,6,10,4,2,10,9</code>.</p>
