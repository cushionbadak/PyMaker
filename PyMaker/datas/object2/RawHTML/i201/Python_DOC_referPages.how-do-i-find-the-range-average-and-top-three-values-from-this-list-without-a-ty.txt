<div class="post-text" itemprop="text">
<p>You need to use list slicing like so:</p>
<pre><code>topthree = sorted_grades[:-4:-1]
</code></pre>
<p>I know it says <code>-4</code>, but it takes the top three.</p>
<p>If you want to use a list, it takes a bit more effort:</p>
<pre><code>indices = [-1, -2, -3]
topthree = [sorted_grades[i] for i in indices]
</code></pre>
<p>You can also sort in reverse:</p>
<pre><code>sorted_grades = sorted(numbers, reverse=True)
topthree = sorted_grades[:3]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Assuming that you have already collected the grades into a list called <code>grades</code>:</p>
<pre><code># New list of sorted entries
sorted_grades = sorted(grades)

# Sum of all list entries divided by the length
average = sum(grades)/len(grades)
# Last entry minus the first entry
range = grades[-1] - grades[0]
# Slice from the third-to-last entry to the end of the list
top_three = grades[-3:]
</code></pre>
<p>Syntax such as negative indexing and slicing is discussed further in the <a href="https://docs.python.org/3/tutorial/introduction.html#lists" rel="nofollow noreferrer">tutorial provided in the CPython documentation</a>:</p>
<blockquote>
<p>Like strings (and all other built-in sequence type), lists can be indexed and sliced:</p>
<pre><code>&gt;&gt;&gt; squares = [1, 4, 9, 16, 25]
&gt;&gt;&gt; squares
[1, 4, 9, 16, 25]
&gt;&gt;&gt; squares[0]  # indexing returns the item
1
&gt;&gt;&gt; squares[-1]
25
&gt;&gt;&gt; squares[-3:]  # slicing returns a new list
[9, 16, 25]
</code></pre>
<p>All slice operations return a new list containing the requested elements. This means that the following slice returns a new (shallow) copy of the list:</p>
<pre><code>&gt;&gt;&gt; squares[:]
[1, 4, 9, 16, 25]
</code></pre>
</blockquote>
<p>The general format for list indexing/slicing is <code>some_list[start:stop:step]</code>:</p>
<pre><code>&gt;&gt;&gt; numbers = [1,3,5,7,9]
&gt;&gt;&gt; numbers[0:3]  # slice from the first element to the third
[1, 3, 5]
&gt;&gt;&gt; numbers[:3]  # 0's can be omitted
[1, 3, 5]
&gt;&gt;&gt; numbers[1:3]  # slice from the second element to the third
[3, 5]
&gt;&gt;&gt; numbers[3:]  # slice from the third element to the end
[7, 9]
&gt;&gt;&gt; numbers[-3:]  # slice from the third-to-last to the end
[5, 7, 9]
&gt;&gt;&gt; numbers[::-1]  # slice of the whole list, stepping backward by 1 for each entry
[9, 7, 5, 3, 1]
&gt;&gt;&gt; numbers[1::2]  # slice of every other entry, starting with the second
[3, 7]
</code></pre>
<p>Note the list slices are end-exclusive, so <code>numbers[1:2]</code> only returns the second entry: <code>[3]</code>.</p>
</div>

<p>Python use <code>:</code> symbol for list slicing. So instead of use <code>topthree = sorted_grades[-1,-2,-3]</code>, use <code>topthree = sorted_grades[-1:-4:-1]</code>.
Format for list slicing is <code>[start:stop:step]</code>.</p>
