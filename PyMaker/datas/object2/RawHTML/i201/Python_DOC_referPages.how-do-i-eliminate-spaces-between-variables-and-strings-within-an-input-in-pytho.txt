<div class="post-text" itemprop="text">
<p>You can us old style % formatting, .format() or use f-strings (python 3.6+) to format your text:</p>
<pre><code>num = 42
tmp = int(input( "Please enter number %s:" % num))
tmp = int(input( "Please enter number {}:".format(num)))
tmp = int(input( f"Please enter number {num}:"))
</code></pre>
<p>Output (trice):</p>
<pre><code>Please enter number 42:
</code></pre>
<p>See </p>
<ul>
<li><a href="https://stackoverflow.com/questions/5082452/python-string-formatting-vs-format">old style % vs .format()</a></li>
<li><a href="https://docs.python.org/3/tutorial/inputoutput.html#fancier-output-formatting" rel="nofollow noreferrer">f-strings</a></li>
</ul>
</div>
<div class="post-text" itemprop="text">
<p>You should use string formatting to construct the string yourself.</p>
<p>Either using <code>.format</code></p>
<pre><code>num = 5
int(input("Please enter number {}:".format(num)))
</code></pre>
<p>Or using <code>f-strings</code> for python 3.6+</p>
<pre><code>num = 5
int(input(f"Please enter number {num}:"))

#Output:
Please enter number 5:
</code></pre>
</div>
<span class="comment-copy">Don't count on <code>input</code> for formatting, do it your self: <code>str(num)+":"</code> will leave no space.</span>
