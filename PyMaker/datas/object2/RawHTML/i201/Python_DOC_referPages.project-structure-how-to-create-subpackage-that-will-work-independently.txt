<div class="post-text" itemprop="text">
<p>One solution that should work :</p>
<ul>
<li>Add <code>__init__.py</code> in the subpackage,</li>
<li>Add the subpackage directory in PYTHONPATH, you can also do it with :
<code>sys.path.append(PATH_TO_THE_SUBPACKAGE)</code> .</li>
</ul>
<p>The package is now part of the PYTHONPATH, so you can import it like any other libraries with <code>import config</code>.</p>
