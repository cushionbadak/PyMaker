<div class="post-text" itemprop="text">
<p>Don't reset your list to an empty list <code>[]</code> on each loop.</p>
<p>Rather than controlling your index yourself, you can use a <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer">range</a>. </p>
<p>Also, use <code>append</code> to append an item to your list.</p>
<p>So, your code could become:</p>
<pre><code>li = []
for i in range(5):
    li.append(input('&gt;&gt;&gt;'))

print(li)
</code></pre>
</div>
