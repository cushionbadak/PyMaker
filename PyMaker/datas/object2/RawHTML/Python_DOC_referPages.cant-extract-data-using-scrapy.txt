<div class="post-text" itemprop="text">
<p>The site uses additional requests to return data you need.</p>
<p>If you open developer tools you can see request that returns data you need.</p>
<p>Url: <code>https://property.spatialest.com/nc/durham/data/propertycard</code></p>
<p>Method: <code>POST</code></p>
<p>Body: <code>parcelid=100016&amp;card=&amp;year=&amp;debug%5BcurrentURL%5D=https%3A%2F%2Fproperty.spatialest.com%2Fnc%2Fdurham%2F%23%2Fproperty%2F100016&amp;debug%5BpreviousURL%22%5D=</code></p>
<p>Response is json, and you can find all data here.</p>
<p>So you should to make request inside scrapy for take the data</p>
</div>
