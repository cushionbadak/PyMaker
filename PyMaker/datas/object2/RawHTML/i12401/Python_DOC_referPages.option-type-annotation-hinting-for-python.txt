<p>Is there a more or less standard type annotation or type hinting for option type à la SML, in Python?</p>
<p>So far, I'm writing something like this:</p>
<pre><code>def f(a: str) -&gt; str | None:
    …
</code></pre>
<p>Is this the recommended way to say the result may be of a given type or else <code>None</code>? If it is not, then what would be the recommended way?</p>
<p>I had a look at the <code>typing</code> module from Python 3.5, and could not find any, unless I overlooked it.</p>
</div>
<div class="post-text" itemprop="text">
<p>Starting from python3.5 the <a href="https://docs.python.org/3/library/typing.html#module-typing" rel="nofollow"><code>typing</code></a> module provides the <a href="https://docs.python.org/3/library/typing.html#typing.Optional" rel="nofollow"><code>Optional</code></a> type exactly for this purpose:</p>
<pre><code>def f(a: str) -&gt; Optional[str]:
    ...
</code></pre>
