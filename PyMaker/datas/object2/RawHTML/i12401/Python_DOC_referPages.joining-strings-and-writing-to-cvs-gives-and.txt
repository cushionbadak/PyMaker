<div class="post-text" itemprop="text">
<p>Try changing your code to this:</p>
<pre><code>for u in response_data:
    titolo = u['title'].encode('utf-8')
    anno = u['release_date']
    anno1 = str(anno[:4])
    output = titolo, anno1
    writertitolo.writerow(output)
</code></pre>
<p>When you do </p>
<pre><code>output = titolo, anno1
</code></pre>
<p><code>output</code> will contain the value <code>(titolo, anno1)</code>, </p>
<p>Now what you are writing to csv is </p>
<p><code>[output]</code> </p>
<p>This will translate into <code>[(titolo, anno1)]</code> </p>
<p>What you need to write to csv is <code>[titolo, anno1]</code> </p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/csv.html#csv.csvwriter.writerow" rel="nofollow"><code>csv.writerow</code></a> takes an iterable and writes the elements of that iterable.</p>
<p>In your case, <code>output</code> is already a tuple of <code>titolo</code> and <code>anno1</code>. So when you write <code>[output]</code> you actually get <code>[(titolo, anno1)]</code> which is a list with a single: a tuple. So when you write that one item, it gets converted to a string which causes the result you see.</p>
<p>Instead, you want to pass the tuple directly:</p>
<pre><code>writertitolo.writerow(output)
</code></pre>
</div>
