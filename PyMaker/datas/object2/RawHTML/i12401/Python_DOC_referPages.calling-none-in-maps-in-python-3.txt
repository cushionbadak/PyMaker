<p>I am doing the following in Python2.7:</p>
<pre><code>&gt;&gt;&gt; a = [1,2,3,4,5]
&gt;&gt;&gt; b = [2,1,3,4]
&gt;&gt;&gt; c = [3,4]
&gt;&gt;&gt; map(None, a, b, c)
[(1, 2, 3), (2, 1, 4), (3, 3, None), (4, 4, None), (5, None, None)]
</code></pre>
<p>I am trying to do something similar in Python3</p>
<pre><code>&gt;&gt;&gt; a = [1,2,3,4,5]
&gt;&gt;&gt; b = [2,1,3,4]
&gt;&gt;&gt; c = [3,4]
&gt;&gt;&gt; map(None, a, b, c)
&lt;map object at 0xb72289ec&gt;
&gt;&gt;&gt; for i,j,k in map(None, a, b, c):
...  print (i,j,k)
... 
Traceback (most recent call last):
File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: 'NoneType' object is not callable
</code></pre>
<p>How do I replicate the Python2 results in Python3?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="noreferrer"><code>itertools.zip_longest()</code> function</a> instead:</p>
<pre><code>from itertools import zip_longest

for i, j, k in zip_longest(a, b, c):
</code></pre>
<p>This zips together the three lists, padding out with the <code>fillvalue</code> keyword value (defaulting to <code>None</code>).</p>
