<p>How can I add up the elements of two lists in order to make a new list with the updated values. For example:</p>
<pre><code>a = [1,2,3]
b = [3,4,5]

answer = [4,6,8]
</code></pre>
<p>I know this is a simple question, but I am new to this thing and cant find an answer anywhere else...</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; a = [1,2,3]
&gt;&gt;&gt; b = [3,4,5]
&gt;&gt;&gt; map(sum, zip(a, b))
[4, 6, 8]
</code></pre>

<div class="post-text" itemprop="text">
<p>The zip() method would probably be the best way to add up columns, in that order.</p>
<pre><code>a = [1, 3, 5] #your two starting lists
b = [2, 4, 6]
c = [] #the list you would print to
for x,y in zip(a, b): #zip takes 2 iterables; x and y are placeholders for them.
    c.append(x + y) # adding on the the list

print c #final result
</code></pre>
<p>You may want to learn about list comprehensions, but for this task, it is not required.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; a = [1,2,3]
&gt;&gt;&gt; b = [3,4,5]
&gt;&gt;&gt; import operator
&gt;&gt;&gt; map(operator.add, a, b)
[4, 6, 8]
</code></pre>
<p>For Python3, you need to use <code>list</code> with the result of <code>map</code></p>
<pre><code>&gt;&gt;&gt; list(map(operator.add, a, b))
[4, 6, 8]
</code></pre>
<p>or just use the usual list comprehension</p>
<pre><code>&gt;&gt;&gt; [sum(x) for x in zip(a,b)]
[4, 6, 8]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow">itertools.izip_longest</a> method to help with this:</p>
<pre><code>def pairwise_sum(a, b):
  for x, y in itertools.izip_longest(a, b, fillvalue=0):
    yield x + y
</code></pre>
<p>You can then use it like this:</p>
<pre><code>a = [1, 2, 3]
b = [3, 4, 5]
answer = list(pairwise_sum(a, b))
</code></pre>
