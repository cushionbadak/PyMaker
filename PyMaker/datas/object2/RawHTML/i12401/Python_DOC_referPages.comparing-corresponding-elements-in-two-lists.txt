<p>I'm a bit of a Python beginner so I apologise if this is a very basic question.</p>
<p>I have two lists of data which are obtained from:</p>
<pre><code>with filein as f:
        reader=csv.reader(f)
        xs, ys = zip(*reader)
</code></pre>
<p>I would like to create a loop which would take the first item in "xs" and the first item in "ys" and print them out. I would then like to loop back and repeat for the second item in both lists and so forth.</p>
<p>I had thought something like:</p>
<pre><code>for x in xs and y in ys:
</code></pre>
<p>Or </p>
<pre><code>for x in xs:
    for y in ys:
</code></pre>
<p>But neither of these seems to give the desired result.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <code>zip</code> function, along with tuple unpacking:</p>
<pre><code>for x, y in zip(xs, ys):
    print x, y
</code></pre>
<p>In your case, depending on what you need the <code>xs</code> and <code>ys</code> for, you could have iterated through the <code>csv.reader</code> directly:</p>
<pre><code>with filein as f:
    reader=csv.reader(f)
    for x, y in reader:
        print x, y
</code></pre>
<p>The <code>zip(xs, ys)</code> line was effectively reversing your <code>xs, ys = zip(*reader)</code> line.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/functions.html?highlight=zip#zip" rel="nofollow">zip</a>:</p>
<pre><code>&gt;&gt;&gt; L=[1,2,3]
&gt;&gt;&gt; M=[4,5,6]
&gt;&gt;&gt; for a,b in zip(L,M):
...   print(a,b)
...
1 4
2 5
3 6
</code></pre>

<div class="post-text" itemprop="text">
<p>For a one line you can use combination of <code>map()</code> and <code>lambda()</code>. <a href="http://www.u.arizona.edu/~erdmann/mse350/topics/list_comprehensions.html" rel="nofollow">Look here if not familiar to this concepts.</a> </p>
<p>But be careful, you must be with python 3.x so that print is a function and can be used inside the lambda expression.</p>
<pre><code>&gt;&gt;&gt; from __future__ import print_function
&gt;&gt;&gt; l1 = [2,3,4,5]
&gt;&gt;&gt; l2 = [6,7,3,8]
&gt;&gt;&gt; list(map(lambda X: print(X[0],X[1]), list(zip(l1,l2))))
</code></pre>
<p>output</p>
<pre><code>2 6
3 7
4 3
5 8
</code></pre>
