<p>I have one file with so many values those are as below</p>
<pre><code>{'Name':'xyz','Date':'Thu Jun 26 13:14:17 +0000 2008'}
{'Name':'xyz','Date':'Thu Jun 19 13:14:17 +0000 2009'}
{'Name':'xyz','Date':'Thu Jun 26 13:15:17 +0000 2008'}
</code></pre>
<p>How to sort these values according to date and time?</p>
</div>
<div class="post-text" itemprop="text">
<p>If the file has these one per line, you'll need to convert to real <code>dict</code>s, e.g. with <code>json.loads</code> or <code>ast.literal_eval</code>, then sort:</p>
<pre><code>import json
import datetime

def parse_date(datestr):
    return datetime.datetime.strptime(datestr, '%a %b %m %H:%M:%S %z %Y')

with open(myfilename) as f:
    mydicts = map(json.loads, filter(str.strip, f))
    # This sorts on date value, but leaves them as original str
    # in dict, can convert ahead of time if you want, or not, as you please
    sorteddicts = sorted(mydicts, key=lambda d: parse_date(d['Date'])]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>dateutil.parser</code> for parsing dates and add the proper <code>key</code> in the <code>sort</code> function.
<br/>
This works - </p>
<pre><code>In [20]: from dateutil.parser import parse

In [21]: l = [{'Name':'xyz','Date':'Thu Jun 26 13:14:17 +0000 2008'},
{'Name':'xyz','Date':'Thu Jun 19 13:14:17 +0000 2009'},
{'Name':'xyz','Date':'Thu Jun 26 13:15:17 +0000 2008'}]

In [22]: sorted(l, key=lambda x: parse(x['Date']))
Out[22]:
[{'Date': 'Thu Jun 26 13:14:17 +0000 2008', 'Name': 'xyz'},
 {'Date': 'Thu Jun 26 13:15:17 +0000 2008', 'Name': 'xyz'},
 {'Date': 'Thu Jun 19 13:14:17 +0000 2009', 'Name': 'xyz'}]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use <code>email</code> package from stdlib, to parse the date format into "seconds since the Epoch":</p>
<pre><code>&gt;&gt;&gt; from email.utils import parsedate_tz, mktime_tz
&gt;&gt;&gt; mktime_tz(parsedate_tz('Thu Jun 26 13:14:17 +0000 2008'))
1214486057
</code></pre>
<p>It is easy to sort <code>dict</code>s by <code>Date</code> field:</p>
<pre><code>&gt;&gt;&gt; sorted(dicts, key=lambda d: mktime_tz(parsedate_tz(d['Date'])))
[{'Date': 'Thu Jun 26 13:14:17 +0000 2008', 'Name': 'xyz'},
 {'Date': 'Thu Jun 26 13:15:17 +0000 2008', 'Name': 'xyz'},
 {'Date': 'Thu Jun 19 13:14:17 +0000 2009', 'Name': 'xyz'}]
</code></pre>
<p>To load json objects from the file, you could use <a href="https://stackoverflow.com/a/35099083/4279">code from @ShadowRanger's answer</a>.</p>
