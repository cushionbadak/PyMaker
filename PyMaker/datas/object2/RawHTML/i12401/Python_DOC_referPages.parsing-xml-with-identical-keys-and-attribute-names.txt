<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#finding-interesting-elements" rel="nofollow">documentation</a> gives some examples along with explanation.  For example:</p>
<pre><code>import xml.etree.ElementTree as ET
tree = ET.parse('data.xml')
root = tree.getroot()

for fl in root.iter('FL'):
    val = fl.attrib['val']
    text = fl.text.strip()
    print("The %s is %s" % (val, text))
</code></pre>
