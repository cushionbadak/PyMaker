<p>I have a list like below...</p>
<pre><code>lst = ['dosa','idly','sambar']
</code></pre>
<p>i need to convert the above data to below format by using Python.</p>
<pre><code>[{'menuitem': 'idly'},
 {'menuitem': 'dosa'},
 {'menuitem': 'sambar'},
]
</code></pre>
<p>Thanks.</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="noreferrer">list comprehension</a>, make a list of dictionaries:</p>
<pre><code>&gt;&gt;&gt; lst = ['dosa','idly','sambar']
&gt;&gt;&gt; [{'menuitem': x} for x in lst]
[{'menuitem': 'dosa'}, {'menuitem': 'idly'}, {'menuitem': 'sambar'}]
</code></pre>
<p>Then, convert it to json using <a href="https://docs.python.org/3/library/json.html#json.dumps" rel="noreferrer"><code>json.dumps</code></a>:</p>
<pre><code>&gt;&gt;&gt; import json
&gt;&gt;&gt; json.dumps([{'menuitem': x} for x in lst])
'[{"menuitem": "dosa"}, {"menuitem": "idly"}, {"menuitem": "sambar"}]'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>lst = ['dosa','idly','sambar']
x="menuitem"
print map(lambda i:{x:i},lst)
</code></pre>
<p>Output:
<code>[{'menuitem': 'dosa'}, {'menuitem': 'idly'}, {'menuitem': 'sambar'}]</code></p>
