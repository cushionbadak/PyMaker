<p><a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow"><code>print</code></a> will add a single space between each argument by default.</p>
<blockquote>
<p>print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)</p>
</blockquote>
<p>If you change the separator, you can remove the space character you don't want.</p>
<pre><code>print("printing the second file object:", second_file_object.read(), sep="\n")  
</code></pre>
