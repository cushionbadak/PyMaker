<p>I've a list</p>
<pre><code>order = [8, 7, 5, 9, 10, 11]
</code></pre>
<p>and a list of dictionaries</p>
<pre><code>list_of_dct = [{'value':11}, {'value':8}, {'value':5}, {'value':7}, {'value':10}, {'value':9}]
</code></pre>
<p>I want to sort this <code>list_of_dct</code> by the order given in <code>order</code> list, i.e. the output should be the following:</p>
<pre><code>list_of_dct = [{'value':8}, {'value':7}, {'value':5}, {'value':9}, {'value':10}, {'value':11}]
</code></pre>
<p>I know how to sort by a given <code>key</code>, but not when an order is already given. How can I sort it?</p>
<p>PS: I already have an O(n^2) solution. Looking for a better solution.</p>
</div>

<p>Use index of the <code>order</code> list to sort-Just try if every dictionary has one value and you want sorting by that value-</p>
<pre><code>sorted(list_of_dct,key=lambda x:order.index(x.values()[0]))
</code></pre>
<p>But if you have multiple values for one key then change the index (i.e <code>[0]</code>) on which you will sort.</p>

<div class="post-text" itemprop="text">
<p>Make a mapping of 8 to 0, 7 to 1, ..., 11 to 5 using <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow"><code>enumerate</code></a>:</p>
<pre><code>&gt;&gt;&gt; order = [8,7,5,9,10,11]
&gt;&gt;&gt; list_of_dct = [{'value':11}, {'value':8}, {'value':5},
                   {'value':7}, {'value':10}, {'value':9}]
&gt;&gt;&gt; sort_keys = {item: i for i, item in enumerate(order)}
&gt;&gt;&gt; sort_keys
{5: 2, 7: 1, 8: 0, 9: 3, 10: 4, 11: 5}
</code></pre>
<p>And use it as a sorting key:</p>
<pre><code>&gt;&gt;&gt; list_of_dct.sort(key=lambda d: sort_keys.get(d['value'], len(sort_keys)))
&gt;&gt;&gt; list_of_dct
[{'value': 8}, {'value': 7}, {'value': 5}, {'value': 9},
 {'value': 10}, {'value': 11}]
</code></pre>
<p>use <code>sort_keys.get(..)</code> instead of <code>sort_keys[..]</code> to prevent <code>KeyError</code> in case of value is misisng in <code>order</code>.</p>
</div>
