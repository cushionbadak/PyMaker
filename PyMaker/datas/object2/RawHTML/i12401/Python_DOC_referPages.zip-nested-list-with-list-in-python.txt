a = [[1,2,3][4,5,6]]
b = [[0,1,2][3,4,5]]
c = [[2,3,4][5,6,7]]

[[False,False,False],[True,False,False],[False,False,False]]]

<div class="post-text" itemprop="text">
<p>The idea is to <code>zip()</code> the <code>main</code> list with the already zipped <code>a</code>, <code>b</code> and <code>c</code> lists and make a nested <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; [[[item == x for item in l] for l in lists] 
     for x, lists in zip(main, zip(a, b, c))]
[[[True, False, False], [False, True, False], [False, False, False]], 
 [[False, False, False], [True, False, False], [False, False, False]]]
</code></pre>
</div>
