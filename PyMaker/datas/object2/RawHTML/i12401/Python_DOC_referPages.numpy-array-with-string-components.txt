<p>I am trying to create a numpy array with string components. I have used the below command and it's output is shown. </p>
<pre><code>&gt;&gt;&gt; a = np.full((2,2), 'hello', dtype='S5')
&gt;&gt;&gt; a
array([[b'hello', b'hello'],
       [b'hello', b'hello']], 
      dtype='|S5')
</code></pre>
<p>Can you explain why the array components appear as b'hello' instead of just 'hello'? I was expecting the array to appear as:</p>
<pre><code>&gt;&gt;&gt; a
array([['hello', 'hello'],
      ['hello', 'hello']],
     dtype='|S5')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I think it's due to your <code>dtype</code> you could convert to usual string with <code>bytes.decode</code>. From <a href="http://docs.scipy.org/doc/numpy/reference/arrays.dtypes.html" rel="nofollow">docs</a>:</p>
<blockquote>
<p><code>'S'</code>, <code>'a'</code>  (byte-)string</p>
</blockquote>
<p>So there are bytestrings and bytestrings are showed with <code>b</code> prefix, you could also check type of your elemets:</p>
<pre><code>In [206]: type(a[0][0])
Out[206]: numpy.bytes_
</code></pre>
<p>For converting from <code>bytes</code> to <code>string</code>:</p>
<pre><code>In [189]: [list(map(bytes.decode, i)) for i in a]
Out[189]: [['hello', 'hello'], ['hello', 'hello']]
</code></pre>
<p>Or you could create your array with <code>unicode</code> strings with <code>U</code> dtype:</p>
<pre><code>a = np.full((2,2), 'hello', dtype='U5')

In [191]: a
Out[191]:
array([['hello', 'hello'],
       ['hello', 'hello']], dtype='&lt;U5')
</code></pre>
</div>
