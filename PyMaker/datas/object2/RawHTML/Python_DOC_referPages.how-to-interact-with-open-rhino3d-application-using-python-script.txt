<p>To check if Rhino is already open (Note: this is only a <em>partial</em> answer to my issue):</p>
<p><a href="https://stackoverflow.com/q/37678954/11225613">Checking if program is running programatically</a></p>
<pre><code>import psutil

def is_rhino_open():
    for pid in psutil.pids():
        p = psutil.Process(pid)
        if p.name() == "Rhino5.exe":
            print ("Rhino5 is running!")
</code></pre>
