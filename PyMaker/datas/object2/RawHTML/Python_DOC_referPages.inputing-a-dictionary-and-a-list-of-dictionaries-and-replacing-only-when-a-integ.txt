<p>This is one approach using a simple iteration </p>
<p><strong>Ex:</strong></p>
<pre><code>d1={"00f_125":["A","2","3"],"00f_126":["1","2","3"],"00f_127":["T","2","3"],"00f_128":["T","2","3"]}
d2=[{"marker":"00f_125","1":"T"},{"marker":"00f_126", "1":"G"},{"marker":"00f_127","1":"T"}]

for i in d2:
    if i["marker"] in d1:
        if d1[i["marker"]][0] in i:
            d1[i["marker"]][0] = i[d1[i["marker"]][0]]
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>{'00f_125': ['A', '2', '3'],
 '00f_126': ['G', '2', '3'],
 '00f_127': ['T', '2', '3'],
 '00f_128': ['T', '2', '3']}
</code></pre>

<div class="post-text" itemprop="text">
<p>It's pretty simple to do in-place on <code>d1</code>:</p>
<pre><code>for elem in d2:
    marker = elem["marker"] 
    if "1" in d1[marker]:          # is that specific int in d1?
        i = d1[marker].index("1")  # if so, then find its index
        d1[marker][i] = elem["1"]  # and replace it with your desired value
</code></pre>
<p>This is, of course, an oversimplification. If you have more than one key besides "marker", you might want to put all of those keys into a list within that dict (e.g.</p>
<pre><code>d2=[{"marker": "00f_125", "modifications": ["1": "G", "2": ...]}, ...]
</code></pre>
<p>), which would make it easier to iterate through them without hardcoding the value like I did above.</p>
<p>If you wanted to produce a separate dict <code>d3</code> without modifying the original, then you could make a shallow or deep copy of <code>d1</code> before doing this.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>d3 = d1.copy()
for i in d2:
    key = i['marker']
    if key in d3.keys():
        if (d3[key][0].isdigit()):
            d3[key][0] = i['1']
print(d3)
</code></pre>
<p>You can copy d1 to d3 and directly make changes in d3. For every list in d2, if the marker value is present in d3, then check if the first element is an integer. If it is, then replace that number with the character value.</p>
<p>This is the output of the above code.</p>
<pre><code>d3 = {'00f_126': ['G', '2', '3'], 
'00f_125': ['A', '2', '3'], 
'00f_127': ['T', '2', '3'], 
'00f_128': ['T', '2', '3']}
</code></pre>

<p>This is another generic approach without specifying the keys</p>
<pre><code>for i in d2:
    for key in i.keys():
        if i.get(key) in d1:
            lis_val = d1.get(i.get(key))
        if key in lis_val:
            lis_val[lis_val.index(key)] = i.get(key)
</code></pre>
