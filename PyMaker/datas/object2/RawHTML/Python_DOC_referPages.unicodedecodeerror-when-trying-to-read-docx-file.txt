<div class="post-text" itemprop="text">
<p><code>python-docx</code> can open a document from a so-called file-like <code>object</code>. It can also save to a file-like object:</p>
<pre><code>from docx import Document
f = open('jinuj.docx', 'rb')
document = Document(f)
f.close()
</code></pre>
<p><strong>OR</strong></p>
<pre><code>with open('jinuj.docx', 'rb') as f:
    source_stream = StringIO(f.read())
document = Document(source_stream)
source_stream.close()
</code></pre>
