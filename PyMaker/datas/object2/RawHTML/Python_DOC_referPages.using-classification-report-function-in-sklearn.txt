<div class="post-text" itemprop="text">
<p>y_true are the true labels of the samples and y_pred are the predictions made my the model. target_name lets you assign custom names to the class labels</p>
<p>classification_report repots the precision, recall, f1-score  and support of the model.</p>
<p>Example as shown in sklearn 
<a href="https://scikit-learn.org/stable/modules/generated/sklearn.metrics.classification_report.html" rel="nofollow noreferrer">https://scikit-learn.org/stable/modules/generated/sklearn.metrics.classification_report.html</a></p>
<pre><code>from sklearn.metrics import classification_report
y_true = [0, 1, 2, 2, 2]
y_pred = [0, 0, 2, 2, 1]
print(classification_report(y_true, y_pred, target_names=target_names))
</code></pre>
<p>For example the precision of class 0 is True Positive / True positive + False Positive which is 1/2 = 0.5 </p>
