<p>I have a list that contains dictionary's and I want to add every third entry in that list into a new one
The list looks like this:</p>
<pre><code>result = [{"link": "example.com", "text": "Some description"}, {"link": "example2.com", "text": "lorem ipsum"}] ...
</code></pre>
<p>Right now the loop I have  looks like this:</p>
<pre><code>for i in range(0, len(list), 3):
    cleanresults.extend(list[i])
</code></pre>
<p>but instead of copying the whole list it only adds the keys</p>
<pre><code>["link", "text", "link", "text"]
</code></pre>
<p>What did I do wrong?</p>
</div>
<div class="post-text" itemprop="text">
<p>You want to <strong>append</strong>, not extend:</p>
<pre><code>for i in range(0, len(list), 3):
    cleanresults.append(list[i])
</code></pre>
<p>Extending adds the <em>contained</em> elements from the object you passed in; when you iterate over a dictionary you get keys, so those are then added to the <code>cleanresults</code> list, not the dictionary itself.</p>
<p>More cleanly, you could just create a copy of the original list taking every 3rd element with a <a href="https://stackoverflow.com/questions/509211/explain-slice-notation">slice</a>:</p>
<pre><code>cleanresults = list[::3]
</code></pre>
<p>If you didn't want a copy, but only needed to access every third element while iterating, you could also use an <a href="https://docs.python.org/3/library/itertools.html#itertools.islice" rel="nofollow noreferrer"><code>itertools.islice()</code> object</a>:</p>
<pre><code>from itertools import islice

for every_third in islice(list, 0, None, 3):
    # do something with every third dictionary.
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can try this:</p>
<pre><code>result = [{"link": "example.com", "text": "Some description"}, {"link": "example2.com", "text": "lorem ipsum"}] ...

new_result = result[::3] #this list slicing will take every third element in the result list
</code></pre>
<p>Output:</p>
<pre><code>[{'text': 'Some description', 'link': 'example.com'}]
</code></pre>
</div>
<span class="comment-copy">to add an element into the list, use <code>append()</code> instead of <code>extend()</code></span>
<span class="comment-copy"><code>extend</code> takes an iterable, which <code>list[i]</code> (a <code>dict</code>) is; as an iterable, you just get the keys.</span>
