<p>datetime.now() will return the date down to seconds.</p>
<p>So to add to @thaavik 's answer you could do something like this to compare more accurately:</p>
<pre><code>datetime.strptime(my_input_time, '%Y%m%d %h:%m:%s')
</code></pre>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">datetime.strptime</a></p>
<pre><code>from datetime import datetime
t = datetime.strptime(my_input_time, '%Y%m%d') # or whatever your datetime format is
</code></pre>
<p>you can also make setting <code>d</code> more concise:
<code>d = X_value if t &lt; five_days_ago else Y_value</code></p>
