<div class="post-text" itemprop="text">
<p>According to the <a href="https://docs.python.org/3/library/tkinter.ttk.html#treeview" rel="nofollow noreferrer"><code>ttk.Treeview</code></a> documentation, the <strong>columns</strong> option is:</p>
<blockquote>
<p>A list of column identifiers, specifying the number of columns
  and their names</p>
</blockquote>
<p>So if you want your Treeview to have 3 columns named <code>1</code>, <code>2</code>, and <code>3</code>, you should write:</p>
<pre><code>ttk.Treeview.__init__(self, parent, columns=[1, 2, 3], ...)
</code></pre>
