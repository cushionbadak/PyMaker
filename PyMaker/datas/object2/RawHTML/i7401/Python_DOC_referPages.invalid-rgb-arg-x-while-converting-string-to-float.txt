<div class="post-text" itemprop="text">
<p>It appears that you are using an older version of matplotlib that does not have the 'xkcd' colors defined.  Using <code>ecolor='xkcd:salmon'</code> works in matplotlib 2.0.2. Either update matplotlib to a newer version, or use a different color (e.g. <code>ecolor='salmon'</code> should work).</p>
