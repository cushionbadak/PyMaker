<p>I want to run objdump -d [sample] command in my program and then I want to shows the output of it in a formated version. For example I want to show its output with one tab in each line of its output. How can I do that?</p>
<pre><code> cmd1 = "objdump -d "
 name = "sample"
 output = os.system(str(cmd1) + str(name))
 print(output)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/subprocess.html" rel="nofollow noreferrer"><code>subprocess.check_output</code></a>:</p>
<pre><code>In [864]: output = subprocess.check_output(['objdump -d a.out'], shell=True)
</code></pre>
<p>This returns a byte string of your output. You may then use <code>str.decode</code> to display your data. If you want to indent with a tab, you can just split on newline and then print line by line, like this:</p>
<pre><code>In [870]: for line in output.decode().split('\n'):

<span class="comment-copy"><code>os.system</code> doesn't return the "output". It returns the integer return code. So, for starters, bad choice of call. Check the documentation on it.</span>
