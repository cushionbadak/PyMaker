<div class="post-text" itemprop="text">
<p>You could read the row until you reach a not <code>'-'</code> value by calling <a href="https://docs.python.org/3/library/functions.html#next" rel="nofollow noreferrer"><code>next</code></a> on a <a href="https://docs.python.org/3/library/functions.html#next" rel="nofollow noreferrer">generator expression</a> of the filtered row:</p>
<pre><code>for row in reader:
    val = next(x for x in row if x!='-')
    print(val)
</code></pre>
<p>You may add a default in case all the columns in the row are <code>'-'</code>:</p>
<pre><code>...
val = next((x for x in row if x!='-'), '')
</code></pre>
