<div class="post-text" itemprop="text">
<p>Whenever you use the <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer"><code>input</code></a> function in python, it returns a string. To quote the docs (emphasis mine):</p>
<blockquote>
<p>The function then reads a line from input, <strong>converts it to a string</strong> (stripping a trailing newline), and returns that. </p>
</blockquote>
<p>As you see from the docs, the <code>input()</code> function converts the input to a string and returns it. Thus, you cannot perform operations like <code>+= 1</code> on it as it is a string.</p>
<p>To convert a string to an int, you need to use the <code>int()</code> function. Thus, the line of code will be:</p>
<pre><code>Exponet = int(input("Exponet: "))
</code></pre>
<p>This will fix your issue as the <code>int()</code> function will cast the string to an int and then you should be able to perform operations like <code>+=</code> on them.</p>
<hr/>
<p>Side Note: <code>Exponet</code> is a wrong spelling, it is <code>Exponent</code> (note the extra <code>n</code>).</p>
