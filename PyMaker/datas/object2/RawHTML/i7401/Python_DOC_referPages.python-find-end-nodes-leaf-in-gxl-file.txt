<p>Consider using <a href="https://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow noreferrer">xml.etree.ElementTree</a> from the Python Standard Library.</p>
<pre><code>import xml.etree.ElementTree as et

gxl_file_path = "C:\\some\\file\\path\\file.gxl"

tree = et.parse(gxl_file_path)
root = tree.getroot()  # At this point you can traverse the node structure as needed
</code></pre>
<p>Say you needed to find the name of a node:</p>
<pre><code>&gt;&gt;&gt; root.tag
'gxl'
</code></pre>
<p>Or if you wanted to iterate over all edge nodes:</p>
<pre><code>for edge in root.iter('edge'):
    # ... Logic ...
</code></pre>
<p>I can't tell exactly what you're trying to parse, but i believe you should iterate over the 'node' nodes, to the extent of something like this:</p>
<pre><code>for node in root.iter('node'):
    if node.find('attr'):  # If the attribute node is present
        name = node.find('attr').get('name')
</code></pre>
