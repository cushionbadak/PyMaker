<p>I have the following code:</p>
<pre><code>y = [sum(x) for x in ([0, 1, 2], [10, 11, 12], [20, 21, 22])]

print(y)
</code></pre>
<p>The output is:  [3, 33, 63]</p>
<p>What I am after is summing by position in each list, so the output I am wanting is:</p>
<pre><code>[30, 33, 36]

0 + 10 + 20 = 30
1 + 11 + 21 = 33
2 + 12 + 22 = 36
</code></pre>
<p>What am I doing wrong?</p>

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> the lists first:</p>
<pre><code>y = [sum(x) for x in zip([0, 1, 2], [10, 11, 12], [20, 21, 22])]

print(y)
# [30, 33, 36]
</code></pre>

<div class="post-text" itemprop="text">
<p>If you want single sums by index you could write a method that gets you that:</p>
<pre><code>def sum_by_index(array_2D,idx):
   s = 0
   for row in array_2D:
       s += row[idx]
   return s
</code></pre>
<p>If you want all the sums all at once you can do the same but all at once:</p>
<pre><code>def sums_by_index(array_2D):
    s = array_2D[0]
    for row in array_2D[1:]:
        for i,entry in enumerate(row):
            s[i] += entry
    return s
</code></pre>
