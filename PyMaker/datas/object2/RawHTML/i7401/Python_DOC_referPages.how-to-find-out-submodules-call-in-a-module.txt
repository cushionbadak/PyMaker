<p>You can check an object type using <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer">[Python]: isinstance(object, classinfo)</a>.<br/>
For modules, the <code>classinfo</code> argument should be <a href="https://docs.python.org/3/library/types.html#types.ModuleType" rel="nofollow noreferrer">[Python]: types.ModuleType</a>:</p>
<pre><code>isinstance(attr, types.ModuleType)
</code></pre>
<p>While we are on the subject, you could do the same for functions. So, your code would look like:</p>
<pre><code>from types import BuiltinFunctionType, FunctionType, ModuleType

# ...

os_list = list()
for name in dir(os):
    attr = getattr(os, name)
    if isinstance(attr, (BuiltinFunctionType, FunctionType, ModuleType)):
        os_list.append(name)
</code></pre>
<p><em>@EDIT0</em>: Included builtin functions as well.</p>
