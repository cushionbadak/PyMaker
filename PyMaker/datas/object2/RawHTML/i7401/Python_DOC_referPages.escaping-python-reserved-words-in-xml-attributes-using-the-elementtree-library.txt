<p>I'm using the ElementTree library and I have an xml tag that has an attribute with the key called "class". Now this is a reserved word in python obviously. Anyone know how to escape this or if it's even possible?</p>
<pre><code>ownerNode = et.SubElement(rootNode, "Owner")
referenceNode = et.SubElement(ownerNode, class="org.identity", name="john")
</code></pre>
<p>^^^ so, how do I escape the class keyword above?</p>
<p>Thanks!</p>

<div class="post-text" itemprop="text">
<p>Based on <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.SubElement" rel="nofollow noreferrer">the documentation</a>, it looks like you can pass attributes in a dictionary, with the keys as strings:</p>
<pre><code>referenceNode = et.SubElement(ownerNode, "refnode", {"class": "org.identity", "name": "john"})
</code></pre>
<p>David Lambert, in a <a href="http://bugs.python.org/issue5382" rel="nofollow noreferrer">python.org thread</a> discussing this issue, points out that for a function <code>f</code> that only takes keyword arguments, you can do this:</p>
<pre><code>def f(**kwargs):
    print(kwargs)

f(**{'class':'sidebar'})
</code></pre>
</div>
