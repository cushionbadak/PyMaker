<p>Consider adding <code>*</code> in order to unpack the tuple (python 3.x solution):</p>
<pre><code>print ("""crazy different style:
startpoint: %d\n
beans:\t\t %d\n
jars:\t\t %d\n
crates:\t\t %d\n
""" % (start_point, *(secret_formula(start_point))))
</code></pre>

<p>in python 3 you can <a href="https://docs.python.org/3/tutorial/controlflow.html#tut-unpacking-arguments" rel="nofollow noreferrer">unpack the tuple with a <code>*</code></a>:</p>
<pre><code>print("""crazy different style:
startpoint: %d\n
beans:\t\t %d\n
jars:\t\t %d\n
crates:\t\t %d\n
""" % (start_point, *secret_formula(start_point)))
