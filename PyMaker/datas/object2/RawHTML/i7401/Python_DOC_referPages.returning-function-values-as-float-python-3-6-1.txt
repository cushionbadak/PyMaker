<div class="post-text" itemprop="text">
<p><code>^</code> in Python is the <a href="https://docs.python.org/3/reference/expressions.html#grammar-token-xor_expr" rel="nofollow noreferrer"><em>bitwise XOR operator</em></a>, not the power operator:</p>
<blockquote>
<p>The ^ operator yields the bitwise XOR (exclusive OR) of its arguments, which must be intege</p>
</blockquote>
<p>You need to use <code>**</code> instead, which <em>is</em> the power operator:</p>
<pre><code>def calculateHypotenuse(a,b):
  return math.sqrt(a**2 + b**2)
</code></pre>
</div>
<span class="comment-copy">If you're trying to use the power operator, you need to use <code>**</code> instead. <code>^</code> has a completely different meaning in Python. It's the bitwise XOR operator.</span>
