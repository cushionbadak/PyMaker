<p>Use linux command <code>pdf2ps</code> to convert pdf to eps.</p>
<blockquote>
<pre><code>pdf2ps [ options ] input.pdf [output.ps]
</code></pre>
</blockquote>
<p>For example, </p>
<pre><code>pdf2ps input.pdf output.eps
</code></pre>
<p>If you really want to use python, you can call the above command with <code>subprocess.call</code>:</p>
<pre><code>from subprocess import call
call(["pdf2ps", "input.pdf", "output.eps"])
</code></pre>
