<p>I'm novice to Python and I am trying to extract a string from another string with specific format, for example:</p>
<p>I have original string: -</p>
<pre><code>--@$_ABC1234-XX12X
</code></pre>
<p>I need to extract exactly the string <code>ABC1234</code> (must include three first characters and followed by four digits).</p>

<div class="post-text" itemprop="text">
<p>You can use the curly brace <a href="https://docs.python.org/3/library/re.html#regular-expression-syntax" rel="nofollow noreferrer">repetition qualifiers</a> <code>{}</code> to match exactly three alphabetic characters and exactly four numeric characters:</p>
<pre><code>&gt;&gt;&gt; from re import search
&gt;&gt;&gt; 
&gt;&gt;&gt; string = '---@$_ABC1234-XX12X'
&gt;&gt;&gt; match = search('[a-zA-Z]{3}\d{4}', string)
&gt;&gt;&gt; match
&lt;_sre.SRE_Match object; span=(6, 13), match='ABC1234'&gt;
&gt;&gt;&gt; match.group(0) # Use this to get the string that was matched.
'ABC1234'
</code></pre>
<p>Explanation of regex:</p>
<ul>
<li><code>[a-zA-Z]</code>: Match any letter upper case of lower case...</li>
<li><code>{3}</code>: Exactly three times. And...</li>
<li><code>\d</code>: Any digit character...</li>
<li><code>{4}</code> Exactly four times.</li>
</ul>
</div>
<div class="post-text" itemprop="text">
<p>If you now the exact coordinates of the string you can use something like this:</p>
<pre><code>&gt;&gt;&gt; var = "--@$_ABC1234-XX12X"
&gt;&gt;&gt; newstring = var[5:12]
&gt;&gt;&gt; newstring
'ABC1234'
</code></pre>
<p>a python string has a slice method.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can make use of <code>re</code> module in Python</p>
<pre><code>matcher = re.search((?P&lt;matched_string&gt;[a-zA-Z]{3}\d{4}))
needed_string = matcher.groupdict()['matched_string']
</code></pre>
<p><code>needed_string</code> will be your desired output.</p>
<p>For the <code>re</code> module refer to: <a href="https://docs.python.org/3.4/library/re.html" rel="nofollow noreferrer">https://docs.python.org/3.4/library/re.html</a></p>
