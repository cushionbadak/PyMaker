<pre><code># In Python 3.x

hash={'a': 1, 'b': 2}

for k in hash:
    print (str(k) + "," + str(hash[k]))

# OR
for key, value in hash.items():
    print (str(key) + ',' + str(value))

# OR
for key, value in {'a': 1, 'b': 2}.items():
    print (str(key) + ',' + str(value))

# OR
for tup in hash.items():
    print (str(tup[0]) + ',' + str(tup[1]))

<p>Understanding dictionary, below is from official python documentation website ...</p>
<pre><code>## dictionary initialization
&gt;&gt;&gt; tel = {'jack': 4098, 'sape': 4139}

## set value for a key (either existing or new)
&gt;&gt;&gt; tel['guido'] = 4127

## print dictionary
&gt;&gt;&gt; tel
{'sape': 4139, 'guido': 4127, 'jack': 4098}

## get value for an existing key
&gt;&gt;&gt; tel['jack']
4098

## delete a key from dictionary
&gt;&gt;&gt; del tel['sape']

## add a new key with value
&gt;&gt;&gt; tel['irv'] = 4127

## print dictionary
&gt;&gt;&gt; tel
{'guido': 4127, 'irv': 4127, 'jack': 4098}

## get all keys in dictionary
&gt;&gt;&gt; list(tel.keys())
['irv', 'guido', 'jack']

## get all keys in dictionary (sorted)
&gt;&gt;&gt; sorted(tel.keys())
['guido', 'irv', 'jack']

## check if a key exists in dictionary
&gt;&gt;&gt; 'guido' in tel
True

## check if a key exists in dictionary
&gt;&gt;&gt; 'jack' not in tel
False

## Finally iterating thru dictionary
for key, value in tel.items():
    print(key, value)
</code></pre>
