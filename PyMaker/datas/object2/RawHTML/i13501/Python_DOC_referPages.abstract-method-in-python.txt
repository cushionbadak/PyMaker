<div class="post-text" itemprop="text">
<p>You're using Python 3.x. In Python 3.x, metaclass declaring syntax is changed.</p>
<pre><code>class SetterGetter(object, metaclass=abc.ABCMeta):  # &lt;---
    @abc.abstractmethod
    def set_val(self, inp):
        return

    @abc.abstractmethod
    def get_val(self):
        return
</code></pre>
<p>See <a href="https://docs.python.org/3/reference/datamodel.html#customizing-class-creation" rel="nofollow">Customizing class creation - Data model - Python 3.x documentation</a>, <a href="https://www.python.org/dev/peps/pep-3115/" rel="nofollow">PEP3115 - Metaclasses in Python 3</a></p>
