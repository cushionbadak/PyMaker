<p>What is the difference between python functions <code>datetime.now()</code> and <code>datetime.today()</code> ?</p>
<blockquote>
<p>In [1]: from datetime import datetime</p>
<p>In [2]: datetime.now()</p>
<p>Out[2]: datetime.datetime(2015, 9, 11, 12, 8,28, 909842)</p>
<p>In [3]: datetime.today()</p>
<p>Out[3]: datetime.datetime(2015, 9, 11, 12,8, 45, 175839)</p>
</blockquote>
<p>Thanks in advance.</p>
</div>
<div class="post-text" itemprop="text">
<p><strong>datetime.datetime.now()</strong> takes tzinfo as keyword argument but <strong>datetime.today()</strong> does not take any keyword arguments.</p>
<p>By default, <strong>now()</strong> executes with <strong>datetime.datetime.now(tz=None)</strong></p>
<p>As quoted in the docs: <a href="https://docs.python.org/3.6/library/datetime.html#datetime.datetime.now" rel="noreferrer">https://docs.python.org/3.6/library/datetime.html#datetime.datetime.now</a></p>
<blockquote>
<p><strong>datetime.now()</strong> return the current local date and time. If optional
  argument tz is None or not specified, this is like today(), but, if
  possible, supplies more precision than can be gotten from going
  through a time.time() timestamp (for example, this may be possible on
  platforms supplying the C gettimeofday() function).</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>See the docs for <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.today"><code>datetime.today</code></a> and <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.now"><code>datetime.now</code></a>, specifically this part from the latter link: </p>
<blockquote>
<p>If optional argument tz is None or not specified, this is like today(), but, if possible, supplies more precision than can be gotten from going through a time.time() timestamp (for example, this may be possible on platforms supplying the C gettimeofday() function).</p>
</blockquote>
<p>So in your example, both are the same, although specific platforms may provide more precision with <code>datetime.now</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>See <a href="https://docs.python.org/3.6/library/datetime.html#datetime.datetime.now" rel="nofollow">the documentation</a>: <code>now()</code> provides an optional timezone, and can give more precision.</p>
