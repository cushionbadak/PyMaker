<pre><code>list1 = [2,4,5,6,7,89]
lambdafunction = lambda x: x &gt; 20
print(any(lambdafunction(list1)))
</code></pre>
<p>I basically want it to print true or false on the condition x &gt; 20 if any of the numbers in the list are greater than 20.  How would I do that using an 'any' function and a lambda?  I know how to check for a specific position in the list, but not how to check the whole list.</p>

<div class="post-text" itemprop="text">
<p>To keep it functional you could use <code>map</code>:</p>
<pre><code>list1 = [2,4,5,6,7,89]
lambdafunction = lambda x: x &gt; 20
print(any(map(lambdafunction, list1)))
</code></pre>
<p><code>map</code> returns an <em>iterator</em> in python3 so the values will be lazily evaluated. If the first element in your lists is &gt; 20, not more values will be consumed from the iterator.</p>
<pre><code>In [1]: list1 = [25,4,5,6,7,89]

In [2]: lambdafunction = lambda x: x &gt; 20

In [3]: it = map(lambdafunction, list1)

In [4]: any(it) 
Out[4]: True

In [5]: list(it)
Out[5]: [False, False, False, False, True]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do:</p>
<pre><code>&gt;&gt;&gt; print(any(lambdafunction(x) for x in list1))
True
</code></pre>
<p>Or even:</p>
<pre><code>&gt;&gt;&gt; print(any(x &gt; 20 for x in list1))
True
</code></pre>
<p>This iterates over the elements of the list and it checks that each element is greater than 20. The <code>any()</code> function takes care of returning the correct answer. In your solution you're not iterating over each element.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could do something like this:</p>
<pre><code>list1 = [2,4,5,6,7,89]
lambdafunction = lambda x: [item &gt; 20 for item in x]
print(any(lambdafunction(list1)))
</code></pre>
<p><code>lambdafunction</code> would return an array of booleans which would be if each element in the array is greater than 20. Then, you could use <code>any</code> to see if any of them are true.</p>
<p>But I would recommend using a regular function (using <code>def</code>) if possible.</p>
</div>
