<pre><code>def find_smallest_circle_sqtime(list):
    inv_number_of_elements = 1.0 / len(list);
    smallest_circle = MyCircle(inv_number_of_elements * sum(p.get_x() for p in list),

<p>Where I'm getting the radius of the farthest point in <code>list</code> relative to the <code>smallest_circle.centre</code>. But actually I need to get the point itself. How can I do that in the most pythonic way?</p>

<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html#max"><code>max</code></a> function allows you to provide a custom ordering function (via the <code>key</code> argument):</p>
<pre><code>max(list, key=lambda p: smallest_circle.centre.sub(p).norm())
</code></pre>
<p>And please don't use semicolons. And you shouldn't use names of built-in types/functions as variable names.</p>
