<p>How do I serve a single JSON file over HTTP using only the Python standard library?</p>
</div>
<div class="post-text" itemprop="text">
<p>in Python 3,
to serve the current directory:</p>
<pre><code>$ python3 -m http.server
Serving HTTP on 0.0.0.0 port 8000 ...
</code></pre>
<p><a href="https://docs.python.org/3/library/http.server.html" rel="nofollow">https://docs.python.org/3/library/http.server.html</a></p>
<p>batteries included FTW!</p>
</div>
<div class="post-text" itemprop="text">
<p>Open a terminal. Navigate to the folder with the JSON file in. Run this:</p>
<pre><code>python -m SimpleHTTPServer
</code></pre>
<p>Browse to:</p>
<pre><code>http://localhost:8000/your.json
</code></pre>
<p>Enjoi!</p>
</div>
