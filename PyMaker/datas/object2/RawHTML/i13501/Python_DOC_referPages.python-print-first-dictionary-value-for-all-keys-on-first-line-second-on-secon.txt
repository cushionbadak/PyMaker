<div class="post-text" itemprop="text">
<p>I've got a dictionary like this:</p>
<pre><code>mapping = {
    'a':['_________','_________','_________'],
    'b':['---------','---------','---------'],
    'c':['=========','=========','========='],
}
</code></pre>
<p>And I need to print all the first items in the values lists printed on the first line, all the second values on the second line and all the third values on the third line. Like this:</p>
<pre><code>_________---------=========
_________---------=========
_________---------=========
</code></pre>
<p>All items are the same amount of characters long and all lists have the same number of items.</p>
<p>I have a list of letters which gets the keys in the right order, I'm just not sure how to print the values. I can print all the first items with:</p>
<pre><code>counter = 0
for l in letters:
      for value in mapping[l]:
        print(value[counter],end="")
</code></pre>
<p>How would I go about creating the output above?</p>
</div>
<div class="post-text" itemprop="text">
<p>You could 'transpose' your lists; the columns in the list have to become rows.</p>
<p>Once you have your lists in the right order, the <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code> function</a> can <a href="https://stackoverflow.com/questions/10507104/row-to-column-transposition-in-python">do the transposing for you</a>:</p>
<pre><code>right_order = (mapping[l] for l in letters)
transposed = zip(*right_order)
for column in transposed:
    print(*(column), sep='')
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; mapping = {
...     'a':['_________','_________','_________'],
...     'b':['---------','---------','---------'],
...     'c':['=========','=========','========='],
... }
&gt;&gt;&gt; letters = 'abc'
&gt;&gt;&gt; right_order = (mapping[l] for l in letters)
&gt;&gt;&gt; transposed = zip(*right_order)
&gt;&gt;&gt; for column in transposed:
...     print(*(column), sep='')
... 
_________---------=========
_________---------=========
_________---------=========
</code></pre>
<p>The alternative would be to use a row counter by using an outer loop over the number of elements in one of the lists:</p>
<pre><code>for rownum in range(len(mapping['a'])):
    for l in letters:
        print(mapping[l][rownum], end='')
    print()
</code></pre>

<p>You have to make sure you sort your dictionary, then iterate through it and print stuff. </p>
<pre><code>length = len(d['a'])

for i in range(length):
    for key in sorted(d.keys()):
        print(d[key][i], end = '')
    print()
</code></pre>
