<div class="post-text" itemprop="text">
<p>Command line arguments are in the <code>sys.argv</code> array.</p>
<pre><code>import sys
artist = sys.argv[1]
song = sys.argv[2]
</code></pre>
<p>You'll need to quote names that contain spaces:</p>
<pre><code>genius.py "the beatles" "a day in the life"
</code></pre>
<p>Otherwise there's no way to tell where the artist ends and the song begins.</p>
</div>
