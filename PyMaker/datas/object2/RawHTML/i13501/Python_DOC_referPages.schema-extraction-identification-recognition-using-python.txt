<p>Is there a module/package/strategy for identifying the schema/attribute list when we provide a list of json files.</p>
<p>Pseudo code:</p>
<ol>
<li>Provide source file/files</li>
<li>Each file can have one or more json objects</li>
<li>Identify the attributes in the json object</li>
</ol>
<p>Example:</p>
<pre><code>[
    {
        "id": "id1",
        "name": "John Doe"
    },
    {
        "id": "id2",
        "name": "Jane Doe",
        "age": 30
    }
]
</code></pre>
<p>Output:</p>
<pre><code>["id","name","age"]
</code></pre>
<p>Do we have a python module(preferably) that does this?</p>

<div class="post-text" itemprop="text">
<p>There's the python 3 json module from the standard library.</p>
<pre><code>import json

with open("my_file.json",'r') as file:
  objs = json.loads(file.read())
</code></pre>
<p>Objs would be an array if your json had an array. Objects are decoded to dictionaries.</p>
<p>See <a href="https://docs.python.org/3/library/json.html" rel="nofollow">https://docs.python.org/3/library/json.html</a> for details.</p>
