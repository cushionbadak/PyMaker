<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/2/library/configparser.html" rel="nofollow"><strong><code>ConfigParser</code></strong></a>:</p>
<pre><code>&gt;&gt;&gt; s = """
[A]
x:1
y:2
z:3
[B]
a:4
b:5
c:6"""

&gt;&gt;&gt; from ConfigParser import ConfigParser
&gt;&gt;&gt; from StringIO import StringIO

&gt;&gt;&gt; parser = ConfigParser()
&gt;&gt;&gt; parser.readfp(StringIO(s))
&gt;&gt;&gt; {section: {key: value
               for key, value in parser.items(section)}
     for section in parser.sections()}
{'A': {'x': '1', 'y': '2', 'z': '3'}, 'B': {'a': '4', 'b': '5', 'c': '6'}}
</code></pre>
