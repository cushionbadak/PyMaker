<div class="post-text" itemprop="text">
<p>I am importing an xml file of blog posts into Django. </p>
<p>Part of this process requires me to convert escaped html characters back to their html form by using replace:</p>
<pre><code>s = s.replace("&amp;lt;", "&lt;")
</code></pre>
<p>My problem is that the xml file, when viewed with vim, contains carriage returns in the form of "^M" -- I'd like to convert these carriage returns to break tags, but</p>
<pre><code>s = s.replace("^M", "&lt;br /&gt;")
</code></pre>
<p>does nothing. </p>
<p>I tried converting all of these tags in the xml file with vim, but then my minidom importer breaks. </p>
<p>Any idea on how I can accomplish this conversion using replace?</p>
</div>
<div class="post-text" itemprop="text">
<p>Using one or both of these should work:</p>
<pre><code>s = s.replace("\r\n", "&lt;br /&gt;")
s = s.replace("\n", "&lt;br /&gt;")
</code></pre>
</div>
<span class="comment-copy">That did it thanks.</span>
