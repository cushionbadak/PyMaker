<div class="post-text" itemprop="text">
<p>I am making a program that read 2 or more input at the same time. How can I read input that has the form <code>a b</code>? For example: <code>3 4</code>. I need to read 2 variables: <code>a = 3</code> and <code>b = 4</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do something like</p>
<pre><code>a,b = input().split()
</code></pre>
<p>For example:</p>
<pre><code>&gt;&gt;&gt; a,b = input().split()
3 4
&gt;&gt;&gt; a
'3'
&gt;&gt;&gt; b
'4'
</code></pre>
<p>For reference, see <a href="http://docs.python.org/3/library/functions.html#input" rel="nofollow"><code>input()</code></a> and <a href="http://docs.python.org/3.3/library/stdtypes.html#str.split" rel="nofollow"><code>str.split()</code></a>.</p>
<hr/>
<p>If you want <code>a</code> and <code>b</code> to be ints, you can call <a href="http://docs.python.org/3/library/functions.html#map" rel="nofollow"><code>map()</code></a> (as is described in the comments):</p>
<pre><code>a,b = map(int, input().split())
</code></pre>
</div>
