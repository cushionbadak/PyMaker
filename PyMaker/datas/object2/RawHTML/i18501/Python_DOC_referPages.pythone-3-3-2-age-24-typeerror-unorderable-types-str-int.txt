<div class="post-text" itemprop="text">
<p>On Python 3, <code>input()</code> <em>always</em> returns a string value. Use the <a href="http://docs.python.org/3/library/functions.html#int" rel="noreferrer"><code>int()</code> type</a> to convert it:</p>
<pre><code>if int(age) &gt;= 24:
</code></pre>
<p>A string value and an int are not orderable:</p>
<pre><code>&gt;&gt;&gt; '24' &gt; 23
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: unorderable types: str() &gt; int()
</code></pre>
<p>Note that <code>int()</code> can throw a <code>ValueError</code> exception, if the input cannot be converted:</p>
<pre><code>&gt;&gt;&gt; int('Why do you want to know my age?')
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
ValueError: invalid literal for int() with base 10: 'Why do you want to know my age?'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>age</code> is a string, not an int. To make it an int, use the <code>int()</code> function, so:</p>
<pre><code>print("how old are you")
age = input("&gt;")
if int(age) &gt;= 24:
    print("you are getting old")
    print (age)
...
</code></pre>
<p>Note the line:</p>
<pre><code>if int(age) &gt;= 24:
</code></pre>
</div>
