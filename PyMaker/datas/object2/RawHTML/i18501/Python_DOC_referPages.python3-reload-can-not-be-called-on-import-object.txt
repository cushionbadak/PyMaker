<div class="post-text" itemprop="text">
<p>The <code>reload</code> built-in function has been moved to <code>importlib</code> module in Python 3.4:</p>
<pre><code>In [18]: from importlib import reload

In [19]: reload?
Reload the module and return it.

The module must have been successfully imported before.
</code></pre>
<p>As pointed out by @JPaget in comments <a href="https://docs.python.org/3/library/importlib.html#importlib.reload" rel="noreferrer"><code>reload()</code></a> function has been moved from <a href="https://docs.python.org/3/library/imp.html#module-imp" rel="noreferrer"><code>imp</code></a> to <a href="https://docs.python.org/3/library/importlib.html#module-importlib" rel="noreferrer"><code>importlib</code></a> module in <strong>Python 3.4+</strong>. From <a href="https://docs.python.org/3/whatsnew/3.4.html#importlib" rel="noreferrer">what's new in Python 3.4</a>:</p>
<blockquote>
<p>The <code>reload()</code> function has been moved from <code>imp</code> to <code>importlib</code> as part of
  the <code>imp</code> module deprecation</p>
</blockquote>
</div>
