<p>What is <code>dict_keys()</code> and how to override this in a method (the way <code>__iter__</code>
is overridden)?</p>
</div>
<div class="post-text" itemprop="text">
<p><code>dict_keys()</code> is a <a href="http://docs.python.org/3/library/stdtypes.html#dictionary-view-objects" rel="nofollow noreferrer">dictionary view object</a>; such objects are iterators.</p>
<p>You can return your own implementation of such an object from a <code>.keys()</code> method. The specific type returned by <code>dict.keys()</code> is not available for re-use, you'll have to implement your own from scratch.</p>
<p>I covered creating your own in a <a href="https://stackoverflow.com/a/17750065">previous answer</a>; there are a fair number of methods to cover.</p>
</div>
