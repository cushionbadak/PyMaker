<div class="post-text" itemprop="text">
<p>The first line in your function is causing the problem...
<code>perc = float(goal/donation)</code></p>
<p>Reverse your ratio by replacing with this (Considering this note: You do not need to deal with the case of a goal of zero):</p>
<pre><code>def percent_of_goal(donation, goal): 
    """ percentofgoal
    """
    return (donation/goal) * 100
</code></pre>
<p>Otherwise, just add a simple if statement:</p>
<pre><code>def percent_of_goal(donation, goal): 
    """ percentofgoal
    """
    return (donation/goal) * 100 if goal != 0 else 0
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>To calculate the donation-to-goal ratio, you should divide donation by the goal, rather than the other way around:</p>
<pre><code>def percent_of_goal(donation, goal): 
    perc = donation / goal
    return  perc * 100
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Before using the division operator, you should always check that your denominator is valid, and provide an alternate output that is appropriate to your use case.</p>
<p>In this example, perhaps you want a function like this instead of using <code>/</code>:</p>
<pre><code>def protected_div(n, d):
    return float(n)/d if d != 0 else 0
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The problem is you can not divide by zero (just regular math) perhaps you could try using an <code>if</code> statement, where the perc returns zero if the <code>donation &lt;= 0</code>:</p>
<pre><code>if donation &lt;= 0:
  perc = 0
else:
  perc = goal/donation
</code></pre>
