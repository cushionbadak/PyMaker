<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer">docs</a> show exactly what <code>all()</code> does:</p>
<pre><code>def all(iterable):
    for element in iterable:
        if not element:
            return False
    return True
</code></pre>
<p>In other words, you could rewrite your statement as:</p>
<pre><code>result = True
for cId in cIds:
    if not cId in historyCId[rOBE]:
        result = False
        break
if not result:
    # do something
</code></pre>
