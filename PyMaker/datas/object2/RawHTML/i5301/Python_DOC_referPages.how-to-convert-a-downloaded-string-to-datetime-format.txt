<pre><code>import datetime

today = datetime.datetime.now().date()
Url_date = result.text
Url_date.strip()

if today &lt; Url_date :
    print "Today is less than future date"
    raw_input()
else:
    print "Today is greater than or = to future date"

</code></pre>
<p>The result that comes back is just this : <strong>2018,02,14</strong>. I use .strip() in case there might be blank spaces or extra lines. I've printed out result.text after strip() and it shows the correct details. Why is it that I can't check if <strong>today &lt; Url_date</strong>. It works fine if I enter manually a date into <strong>datetime.date(2018,02,14)</strong>, but when I'm downloading the string it won't work. Any suggestions?</p>

<div class="post-text" itemprop="text">
<p>You pass string to <code>datetime.date()</code> which should be each an integer.</p>
<pre><code>Url_list = []
Url_list = Url_date.split(",")
yr = int(Url_list[0])
mn = int(Url_list[1])
d = int(Url_list[2])
</code></pre>
<p>Now pass these integers to <code>datetime.date</code></p>
<pre><code>Url_date = datetime.date(yr, mn, d)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The arguments you pass to <code>datetime.date(arg1, arg2, arg3)</code> are not strings as a whole. When you pass it from url, what you are actually doing is</p>
<pre><code>datetime.date("2018,2,14")
</code></pre>
<p>Note that you are passing only one string argument and not 3 different integers. You should split the date string using comma and then convert each into integers and then pass them as arguments to datetime.date.</p>
</div>
<div class="post-text" itemprop="text">
<p>Here is what your code is trying to do :</p>
<pre><code>Url_date = datetime.date("2018,02,14")
</code></pre>
<p>But he wants to have:</p>
<pre><code>Url_date = datetime.date(2018,02,14)
</code></pre>
<p>Do</p>
<pre><code>Url_date.split(',') # Result: ['2018','02','14']  
</code></pre>
<p>And then convert all the <code>string</code> in the <code>array</code> in <code>integers</code></p>
<p>It should be ok  :)</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strptime" rel="nofollow noreferrer"><code>strptime</code></a>:</p>
<pre><code>import datetime

today = datetime.datetime.now().date()
parsed = datetime.datetime.strptime("2018,02,14", "%Y,%m,%d").date()
print(today &lt; parsed) # True
</code></pre>
