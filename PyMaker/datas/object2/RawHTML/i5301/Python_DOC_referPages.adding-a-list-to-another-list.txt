<div class="post-text" itemprop="text">
<p>let's say there's a list,</p>
<hr/>
<pre><code>list1= [[0, 4, 0, 0], [2, 0, 4, 0], [9, 8, 0, 0], [0, 2, 5, 0]]
</code></pre>
<p>and there's another list</p>
<hr/>
<pre><code>list2= [a,b,c,d]
</code></pre>
<p>I've seen multiple questions that asks similar things but I couldn't get the proper answer for this question.</p>
<p>I want to make a list </p>
<hr/>
<pre><code>list3= [[a,0,4,0,0],[b,2,0,4,0],[c,9,8,0,0],[d,0,2,5,0]]
</code></pre>
<p>Is there any easy way to achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can iterate the list and use the <strong><a href="https://docs.python.org/2/tutorial/datastructures.html" rel="nofollow noreferrer">list.insert</a></strong> method to do what you want</p>
<pre><code>list1= [[0, 4, 0, 0], [2, 0, 4, 0], [9, 8, 0, 0], [0, 2, 5, 0]]
list2= ["a","b","c","d"]

for i,v in enumerate(list2):
    list1[i].insert(0, v)

print list1
</code></pre>
<p>OR</p>
<pre><code>result = []
for i,v in zip(list1, list2):
    result.append([v] + i)

print result
</code></pre>
<p><strong>Output</strong>:</p>
<pre><code>[['a', 0, 4, 0, 0], ['b', 2, 0, 4, 0], ['c', 9, 8, 0, 0], ['d', 0, 2, 5, 0]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> with <a href="https://docs.python.org/3.3/library/functions.html#zip" rel="nofollow noreferrer">zip()</a> you can do:</p>
<pre><code>result = [[i]+j for i, j in zip(list2, list1)]
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; list1 = [[0, 4, 0, 0], [2, 0, 4, 0], [9, 8, 0, 0], [0, 2, 5, 0]]
&gt;&gt;&gt; list2 = ['a', 'b', 'c', 'd']
&gt;&gt;&gt; result = [[i]+j for i, j in zip(list2, list1)]
&gt;&gt;&gt; result
[['a', 0, 4, 0, 0], ['b', 2, 0, 4, 0], ['c', 9, 8, 0, 0], ['d', 0, 2, 5, 0]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>What about using an iterator + list comprehension:</p>
<pre><code>&gt;&gt;&gt; list1= [[0, 4, 0, 0], [2, 0, 4, 0], [9, 8, 0, 0], [0, 2, 5, 0]]
&gt;&gt;&gt; list2 = ['a', 'b', 'c', 'd']
&gt;&gt;&gt; it = iter(list2)
&gt;&gt;&gt; [[next(it)] + x for x in list1]
[['a', 0, 4, 0, 0], ['b', 2, 0, 4, 0], ['c', 9, 8, 0, 0], ['d', 0, 2, 5, 0]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using simple <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list-comprehensions</a> - </p>
<pre><code>In [1]: list1= [[0, 4, 0, 0], [2, 0, 4, 0], [9, 8, 0, 0], [0, 2, 5, 0]]

In [2]: list2= ["a","b","c","d"]

In [3]: [[list2[i]] + l1 for i, l1 in enumerate(list1)]  # Enumerate keeps track of the index, and also returns the current list element

Out[3]: [['a', 0, 4, 0, 0], ['b', 2, 0, 4, 0], ['c', 9, 8, 0, 0], ['d', 0, 2, 5, 0]]
</code></pre>
