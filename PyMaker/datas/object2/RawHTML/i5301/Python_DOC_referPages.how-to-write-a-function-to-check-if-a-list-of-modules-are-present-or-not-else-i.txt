<p>Here's how I'd do it:</p>
<pre><code>import os
def checkModules(module_list):
    missing = []
    for m in module_list:
        try:
            exec("import %s" % m)
        except ImportError:
            missing.append(m)
    if len(missing) &gt; 0:
        os.system("python -m pip install %s" % ' '.join(missing))
</code></pre>
<p>Hope that helps!</p>
