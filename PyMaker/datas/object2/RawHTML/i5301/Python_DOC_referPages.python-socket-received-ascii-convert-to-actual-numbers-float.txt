<div class="post-text" itemprop="text">
<p>To convert the floats from a socket, you can use <a href="https://docs.python.org/3/library/struct.html#struct.unpack" rel="nofollow noreferrer"><code>struct.unpack</code></a>.</p>
<h3>Code:</h3>
<pre><code>num_floats = int(len(raw_data) / 4)
format_str = '&gt;' + 'f' * num_floats
data = struct.unpack(format_str, raw_data[:num_floats * 4])
</code></pre>
<h3>Test Code:</h3>
<pre><code>import struct

raw_data = b'C\xbc\x7fZC\xbc\x7f5C\xbc\x7f&amp;C\xbc\x7f!C\xbc\x7f\x14C\xbc' \
           b'\x7f\x16C\xbc\x7f!C\xbc\x7f\x12C\xbc\x7f\x19C\xbc\x80\x03C' \
           b'\xbc\x81~C\xbc\x80FC\xbc\x7f\xd1C\xbc~\xcdC\xbc~\xd7C\xbc' \
           b'\x7f$;4\xe0\xb8;\x81\xf6\xd7;R\xf4T:\xc5v\xb4:\xc1&gt;\xc1:\xd9' \
           b'\x15\x9a:\xa8M7;K\xde{:\xfb\\xcd\xbc\tR\xd4\xba\xe1\xb6\xf3' \
           b'\xbb\xc9\xb8\xf4\xbby\x8a\x199\x9dH\x009\x0cp\x00:\xeb'
