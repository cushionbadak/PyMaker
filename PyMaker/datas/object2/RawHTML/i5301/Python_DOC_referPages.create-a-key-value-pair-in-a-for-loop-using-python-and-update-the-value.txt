<p>I am new to python and learning the stuff from scratch. One thing I want to do, but couldn't find out how is to create a key-value pair list using the for loop and update just the value. A sample code is:</p>
<pre><code>for i in range(0,6)
  #####some code that gives an output integer x.
  print(i,x)
</code></pre>
<p>The integer x is different for each i.  How do I create a list like {0:x0,1:x1,2:x2,3:x3,4:x4,5:x5}, using the for loop?</p>

<div class="post-text" itemprop="text">
<p>You're looking for <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">Python Dictionaries</a>.</p>
<pre><code>sample = {}
for i in range(6):
   # some code that gives you x
   sample[i] = x
print(sample)
</code></pre>
<p>Do note, however, that since your keys are just <code>0, 1, 2</code> etc., you may want to just use a list instead:</p>
<pre><code>sample = []
for i in range(6):
    # some code that gives you x
    sample.append(x)
print(sample)
</code></pre>
<p>Note that dictionary access using <code>sample[key]</code> will look identical to list access using <code>sample[index]</code> since you're just using consecutive numeric keys.</p>
