<div class="post-text" itemprop="text">
<p>It doesn't have anything directly to do with Unicode. <code>%04x</code> is a <em>format specifier</em>, which only makes sense with the <code>%</code> operator. It specifies that the number to be formatted (in this case, <code>ord(c)</code>) should be formatted as a hexadecimal with four digits and zero-padding on the left. So if <code>c</code> is <code>"A"</code>, whose Unicode code point is 65, it will be printed as <code>0041</code>.</p>
