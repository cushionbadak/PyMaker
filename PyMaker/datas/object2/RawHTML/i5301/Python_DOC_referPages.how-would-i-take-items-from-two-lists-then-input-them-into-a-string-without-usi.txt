<p>I'm quite new to python... so if</p>
<pre><code>string_1 = ["a", "b", "c", "d"]
string_2 = [1,2,3,4]
</code></pre>
<p>how can I make it so it would print out: </p>
<pre><code>a equals 1
b equals 2
c equals 3
d equals 4
</code></pre>
<p>I've tried:</p>
<pre><code>for i in string_1:
    for j in string_2:
    print(i, "equals", j)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As others have already said, python has <code>zip</code> for just this purpose.  However,  if you insist on doing this without zip:</p>
<pre><code>for i in range(len(string_1)):
    print(string_1[i] + ' equals ' + string_2[i])
</code></pre>
<p>will accomplish this.</p>
</div>
<div class="post-text" itemprop="text">
<p>It would be better to use a dictionary for something like this. </p>
<p>E.g.</p>
<pre><code>things = {'a': 1, 'b': 2, 'c': 3, 'd': 4}

for k, v in things.items():
  print(k, 'equals', v)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is a job for <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip</a> </p>
<pre><code>items_1 = ["a", "b", "c", "d"]
items_2 = [1,2,3,4]

for a, b in zip(items_1, items_2):
    print ("{0} equals {1}".format(a, b)
</code></pre>
</div>
