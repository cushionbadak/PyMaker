<p>The problem with your code is you are trying to match <code>time</code> object with a tuple of tuple. If you want to fix your solution you can use the following line to fix:</p>
<p>Change    </p>
<pre><code>&gt;&gt;&gt; print(cd_dt in result_set)
</code></pre>
<p>To</p>
<pre><code>&gt;&gt;&gt; any(d[1] == cd_dt for d in k)
&gt;&gt;&gt; True
</code></pre>
<p>You can use builtin <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter</code></a> to filter the result set. In this way you can also get the matched element(s) of result set.</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; k = ((0L, datetime.date(2018, 1, 29)), (0L, datetime.date(2018, 1, 30)), (1L, datetime.date(2018, 1, 31)))
&gt;&gt;&gt; list(filter(lambda x: x[1] == datetime.date.today(),k))
&gt;&gt;&gt; [(1L, datetime.date(2018, 1, 31))]
</code></pre>
<p>Or if you just want to check if the current date is in the result set you can use <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer"><code>any</code></a></p>
<pre><code>&gt;&gt;&gt; any(d[1] == datetime.date.today() for d in k)
&gt;&gt;&gt; True
</code></pre>
