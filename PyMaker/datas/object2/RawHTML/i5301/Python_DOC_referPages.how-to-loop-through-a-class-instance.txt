<p>I was asked this question.  Given the code:</p>
<pre><code>class MockList:
    ...code_here...

for i in MockList():
    print(i)
</code></pre>
<p>Expected result of for loop:</p>
<pre><code>1 2 3 4 5
</code></pre>
<p>How can I do this?  </p>
</div>
<div class="post-text" itemprop="text">
<p>If you implement an <a href="https://docs.python.org/3/reference/datamodel.html#object.__iter__" rel="noreferrer"><code>__iter__()</code></a> method, you can do this functionality:</p>
<h3>Code:</h3>
<pre><code>class MockList:

    def __iter__(self):
        return iter(range(1, 6))
</code></pre>
<h3>Test Code:</h3>
<pre><code>for i in MockList():
    print(i)
</code></pre>
<h3>Results:</h3>
<pre><code>1
2
3
4
5
</code></pre>
</div>
