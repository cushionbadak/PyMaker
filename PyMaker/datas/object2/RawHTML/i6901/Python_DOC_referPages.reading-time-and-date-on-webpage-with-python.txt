<div class="post-text" itemprop="text">
<p>You need to use <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer">datetime</a> module.</p>
<p>If you already parsed the string with date and time ("Aug-29-17 09:05AM" in your case) use <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">datetime.strptime()</a> to create datetime object. </p>
<pre><code>mydate = datetime.datetime.strptime('Aug 29 17 09:05AM', '%b %d %y %I:%M%p')
</code></pre>
<p>Than you can compare it with today's date - datetime.now().</p>

<div class="post-text" itemprop="text">
<p>If your question is how to scrap time and date with their responsive content from site's source code then you can do it using <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/" rel="nofollow noreferrer"><code>bs4</code></a> like so:</p>
<pre><code>#!/usr/bin/env python

import sys
reload(sys)
sys.setdefaultencoding('utf-8')

html = '''
&lt;tr&gt;
   &lt;td width="130" align="right" style="white-space:nowrap"&gt;Aug-29-17 09:05AM&amp;nbsp;&amp;nbsp;&lt;/td&gt;
   &lt;td align="left"&gt;
      &lt;a href="https://finance.yahoo.com/news/abeona-therapeutics-receives-fda-breakthrough-130500766.html" target="_blank"
         class="tab-link-news"&gt;Yahoo news&lt;/a&gt;&lt;span style="color:#aa6dc0;font-size:9px"&gt;GlobeNewswire&lt;/span&gt; &lt;span class="body-table-news-gain"&gt;
      +18.95%&lt;/span&gt;
   &lt;/td&gt;
&lt;/tr&gt;
&lt;tr&gt;
   &lt;td width="130" align="right" style="white-space:nowrap"&gt;Aug-25-17 07:30AM&amp;nbsp;&amp;nbsp;&lt;/td&gt;
   &lt;td align="left"&gt;
      &lt;a href="https://finance.yahoo.com/news/" target="_blank" class="tab-link-news"&gt;Corporate News Blog - &lt;/a&gt; 
      &lt;span style="color:#aa6dc0;font-size:9px"&gt;ACCESSWIRE&lt;/span&gt; &lt;span class="body-table-news-loss"&gt;-5.73%&lt;/span&gt;
   &lt;/td&gt;
&lt;/tr&gt;
'''

import re
from bs4 import BeautifulSoup

soup = BeautifulSoup(html, "html.parser")

for i,j in zip(soup.find_all('tr'), soup.find_all('td', attrs={'align':'left'})):
    print '\n[time and date] =&gt; {}'.format(i.td.text)
    print '[Content] =&gt; {}'.format(re.sub('&lt;[^&gt;]*&gt;', '', str(j)).replace('\n',' '))
    print '[Link] =&gt; {}'.format(j.find('a').get('href'))
</code></pre>
<p>This will give as a result:</p>
<pre><code>[time and date] =&gt; Aug-29-17 09:05AM
[Content] =&gt;  Yahoo newsGlobeNewswire     +18.95%
[Link] =&gt; https://finance.yahoo.com/news/abeona-therapeutics-receives-fda-breakthrough-130500766.html

[time and date] =&gt; Aug-25-17 07:30AM
[Content] =&gt;  Corporate News Blog -  ACCESSWIRE -5.73%
[Link] =&gt; https://finance.yahoo.com/news/
</code></pre>
