<p>My question is about how to make the compiler treats my string as an executable statement, consider my string mystr='1+2+3', now the question is what's the code to put in function <code>func</code> so when I execute the following command <code>func(mystr)</code> I get 6 on my screen.</p>

<div class="post-text" itemprop="text">
<p><code>eval(mystr)</code>.</p>
<p><a href="https://docs.python.org/3/library/functions.html#eval" rel="nofollow noreferrer">Documentation here.</a></p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/library/functions.html#exec" rel="nofollow noreferrer">exec()</a> for more than just expression strings:</p>
<pre><code>def func(mystr):
    exec(mystr)

func("mystr='1+2+3';print(mystr)")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use eval(mystr).</p>
<pre><code> print eval('1+2+3')
</code></pre>
