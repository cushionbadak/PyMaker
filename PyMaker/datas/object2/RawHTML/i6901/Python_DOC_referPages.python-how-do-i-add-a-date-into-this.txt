<p>Hint: To get the current month of the year, use the <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer"><code>datetime</code></a> module:</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; datetime.datetime.now().month
8
</code></pre>
<p>Where <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.now" rel="nofollow noreferrer"><code>datetime.datetime.now()</code></a> gets the current time as a <code>datetime</code> object, and the <code>.month</code> attribute prints the current month (as a number)</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>from datetime import *
monthNames=["January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"]
name1 = str(input("Please input your name: "))
thisMonth = int(datetime.now().month)
thisMonth_name = monthNames[thisMonth]

print("%s will play football on %s" %(name1, thisMonth_name))  
</code></pre>
<p>OR</p>
<pre><code>import datetime as dt
import calendar
name1 = str(input("Please input your name: "))
thisMonth = int(dt.datetime.now().month)+1
monthName = str(calendar.month_name[thisMonth])

print("%s will play football on %s" %(name1, monthName))
</code></pre>

<p>Often the easiest way of manipulating dates and times in ways like this is to use the <a href="http://arrow.readthedocs.io/en/latest/" rel="nofollow noreferrer">arrow</a> library. In this case, you can get what you want with this code:</p>
<pre><code>&gt;&gt;&gt; import arrow
&gt;&gt;&gt; arrow.now().shift(months=+1).format('MMMM')
'September'
</code></pre>

<p>If you want to print datetime one month later from current time,</p>
<p>you can use <code>datetime</code> and <code>dateutil</code> in combination.</p>
<pre><code>from datetime import date, datetime
from dateutil import relativedelta
today = datetime.today()
monthplus1 = today + relativedelta.relativedelta(months=1)
print today
print monthplus1
# output
&gt; datetime.datetime(2017, 8, 31, 8, 52, 3, 75585)
&gt; datetime.datetime(2017, 9, 30, 8, 52, 3, 75585)
</code></pre>
<p>then you can use the datetime attributes like <code>monthplus1.month</code> to get the next month.
and if you want names of the month, use a list like in below answer</p>
<p><code>monthNames=["", "January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"]</code></p>
<p>with little tweak, so the 0th will be empty and you can get the Month Names</p>
<pre><code>monthNames[monthplus1.month]
&gt; 'September'
</code></pre>
<p>Then use these values as you would normaly do using format</p>
<pre><code>print（'''

{} will play football on {}

'''.format(name1, monthNames[monthplus1.month]))
</code></pre>
