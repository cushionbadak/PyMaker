<div class="post-text" itemprop="text">
<ul>
<li><p>on python3.6.x you can use <a href="https://docs.python.org/3/library/html.html#html.unescape" rel="nofollow noreferrer">html.unescape</a>:</p>
<pre><code>import html
print(html.unescape('g&amp;#108bo&amp;#115w&amp;#111&amp;#114t&amp;#104'))
</code></pre></li>
<li><p>on python 2.x you can use <a href="https://docs.python.org/2/library/htmlparser.html" rel="nofollow noreferrer">HTMLParser</a>:</p>
<pre><code>from HTMLParser import HTMLParser
h = HTMLParser()
print(h.unescape('g&amp;#108bo&amp;#115w&amp;#111&amp;#114t&amp;#104'))
</code></pre></li>
</ul>

<div class="post-text" itemprop="text">
<p>you are trying to decode <a href="https://www.w3schools.com/html/html_entities.asp" rel="nofollow noreferrer">html escaped string</a>. you can use the <a href="https://docs.python.org/3/library/html.html#html.unescape" rel="nofollow noreferrer"><code>html.unescape(s)</code></a> function to do so (on python3):</p>
<pre><code>import html
print(html.unescape('g&amp;#108bo&amp;#115w&amp;#111&amp;#114t&amp;#104'))
</code></pre>
<p>outputs:</p>
<pre><code>'glbosworth'
</code></pre>
<p>take a look at <a href="https://stackoverflow.com/a/2087433/7438048">this</a> so answer for more info</p>
