<div class="post-text" itemprop="text">
<p><code>b</code> is a list, not an integer. If <code>b</code> was set to <code>1</code> you can just use that to index:</p>
<pre><code>b = 1
a[b]
</code></pre>
<p>If you were looking to store the <em>operation</em> as a variable (not just the index, but that you are <em>indexing</em> in the first place), use functions or helper objects from the <a href="https://docs.python.org/3/library/operator.html" rel="nofollow noreferrer"><code>operator</code> module</a>.</p>
<p>For example, you can store a <a href="https://docs.python.org/3/library/operator.html#operator.itemgetter" rel="nofollow noreferrer"><code>operator.itemgetter()</code> object</a>:</p>
<pre><code>from operator import itemgetter

b = itemgetter(1)  # ...[1]
b(a)               # apply to a
</code></pre>
