<p>I want to change time delta to integer value .</p>
<p>My code is as below.</p>
<pre><code>import datetime
now = datetime.date.today()
print(now.toordinal()) # 736570
cali_date = datetime.data(2017, 6, 14)
print(cali_date.toordinal()) # 736494
date1 = now - cali_date
print(date1) # 76 days, 0:00:00
</code></pre>
<p>But, I want to get just <code>76</code> with integer.
How can I solve this problem?
Thank you.</p>

<div class="post-text" itemprop="text">
<p>Just reference the <code>days</code> attribute of the <a href="https://docs.python.org/3/library/datetime.html#timedelta-objects" rel="nofollow noreferrer"><code>timedelta</code> object</a> you have there:</p>
<pre><code>print(date1.days)
</code></pre>
<p>There are also <code>timedelta.seconds</code> and <code>timedelta.microseconds</code> attributes, modeling the complete delta state. </p>
</div>
<div class="post-text" itemprop="text">
<p><code>date1</code> is a timedelta object  - use  <code>date1.days</code> to get the number of days as an integer, or <code>date1.total_seconds()</code> to see the number of seconds between the two datetime objects.</p>
