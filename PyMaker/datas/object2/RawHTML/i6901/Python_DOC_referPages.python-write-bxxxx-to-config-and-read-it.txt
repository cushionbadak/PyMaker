i want to save it to an config file and reload it
so i can decrypt it and i can use it again as password</p>

<div class="post-text" itemprop="text">
<p>You can do something like this:</p>
<pre><code># to write the file

cryptpw = "your encrypted password"

config = open("/path/to/config/file/pw.config","w")
config.write(cryptpw)
config.close()

# to reopen it

config = open("/path/to/config/file/pw.config","r")
print(config.read())
config.close()
</code></pre>
<p>It's up to you what you do with the contents of that file, i just chose to print it.</p>
