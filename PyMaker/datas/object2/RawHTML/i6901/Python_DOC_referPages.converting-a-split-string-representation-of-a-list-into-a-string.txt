<p>You should be able to do this quickly using <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow noreferrer"><code>ast.literal_eval</code></a>:</p>
<pre><code>import ast

s = ["[u'DUTY,", 'RHODE', 'ISLAND', 'STATE', 'ASSOCIATION', 'OF', 'FIRE', "OF']"]
out = ast.literal_eval(' '.join(s))[0]

print(str(out)) # the str bit is redundant for python3
'DUTY, RHODE ISLAND STATE ASSOCIATION OF FIRE OF'
</code></pre>
