<p>Use a negative step with <code>range()</code> :</p>
<pre><code>for i in range(10):
    for j in range(10, i, -1): # from 10 down to i+1 using range(start,stop,step)
        # Your code
</code></pre>

<div class="post-text" itemprop="text">
<p>From the <a href="https://docs.python.org/3/library/stdtypes.html#typesseq-range" rel="nofollow noreferrer"><code>range</code></a> docs:</p>
<blockquote>
<pre><code>class range(start, stop[, step]) 
</code></pre>
<p>For a <em>positive step</em>, the contents of a range r are determined by the
  formula <code>r[i] = start + step*i</code> where <code>i &gt;= 0</code> and <code>r[i] &lt; stop</code>.</p>
<p>For a <em>negative step</em>, the contents of the range are still determined by
  the formula <code>r[i] = start + step*i</code>, but the constraints are <code>i &gt;= 0</code> and
  <code>r[i] &gt; stop</code>.</p>
</blockquote>
<p>Translation inline.</p>
<pre><code>for (i = 0; i &lt; 10; i++)  //    for i in range(10):
  {
    for (j = 10; j &gt; i; j--)  //    for j in range(10, i, -1):
     {  
       a[i]*a[j];                       a[i] * a[j]; # a[i] *= a[j];  ?
     }
  }
</code></pre>
</div>
