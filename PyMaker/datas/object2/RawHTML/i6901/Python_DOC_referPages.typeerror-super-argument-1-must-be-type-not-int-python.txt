<div class="post-text" itemprop="text">
<p>You should do instead of <code>super(chat_id, user1).__init__()</code> you should do:</p>
<pre><code>super().__init__(chat_id, user1) # Work in Python 3.6
super(GroupMessage, self).__init__(chat_id, user1) # Work in Python 2.7
</code></pre>
<p>or</p>
<pre><code>Chats.__init__(self, chat_id, user1)
</code></pre>
<p>This last option is not recommended if exist change that your class hierarchy change in the future. I really don't like it for others motive but still worth it a mention.</p>
