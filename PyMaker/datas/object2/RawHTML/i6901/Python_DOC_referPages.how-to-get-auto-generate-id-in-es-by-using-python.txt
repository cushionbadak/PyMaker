<div class="post-text" itemprop="text">
<p>You don't need to do this in python -- if you index documents without an id, Elasticsearch will <a href="https://www.elastic.co/guide/en/elasticsearch/reference/current/docs-index_.html#_automatic_id_generation" rel="nofollow noreferrer">automatically create a unique id</a>.  However, if for some reason you want to generate the id in python, you could use <a href="https://docs.python.org/3/library/uuid.html" rel="nofollow noreferrer">uuid</a>.</p>
