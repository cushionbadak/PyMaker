<p>There are double quotes embedded your curl command:</p>
<pre><code>cmd = ​​​​"curl 'https://www.example.com/api/security/v2/security/authorize' -H 'cookie: check=true; [snip] RT="sl=1&amp;ss=1553025660875&amp;tt=3538&amp;obo=0&amp;bcn=%2F%2F36a3fec2.akstat.io%2F&amp;sh=1553025664433%3D1%3A0%3A3538&amp;dm=example.com&amp;si=a71a0de4-27c6-4c1a-806e-137e6c895955&amp;ld=1553025664433"; [snip] --compressed"
</code></pre>
<p>So you need to enclose the command in triple quotes rather than double:</p>
<pre><code>cmd = ​​​​"""curl 'https://www.example.com/api/security/v2/security/authorize' -H 'cookie: check=true; [snip] RT="sl=1&amp;ss=1553025660875&amp;tt=3538&amp;obo=0&amp;bcn=%2F%2F36a3fec2.akstat.io%2F&amp;sh=1553025664433%3D1%3A0%3A3538&amp;dm=example.com&amp;si=a71a0de4-27c6-4c1a-806e-137e6c895955&amp;ld=1553025664433"; [snip] --compressed"""
</code></pre>
<p>or escape the double-quotes using the "\" character:</p>
<pre><code>cmd = ​​​​"curl 'https://www.example.com/api/security/v2/security/authorize' -H 'cookie: check=true; [snip] RT=\"sl=1&amp;ss=1553025660875&amp;tt=3538&amp;obo=0&amp;bcn=%2F%2F36a3fec2.akstat.io%2F&amp;sh=1553025664433%3D1%3A0%3A3538&amp;dm=example.com&amp;si=a71a0de4-27c6-4c1a-806e-137e6c895955&amp;ld=1553025664433\"; [snip] --compressed"
</code></pre>
<p>Personally I'd use the triple-quotes, so you can cut-and-paste the command from your shell.</p>
