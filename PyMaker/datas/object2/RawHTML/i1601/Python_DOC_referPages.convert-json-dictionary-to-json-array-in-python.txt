<div class="post-text" itemprop="text">
<p>I have JSON dictionary something like this:</p>
<pre><code>{'foo': 3, 'bar': 1}
</code></pre>
<p>and i want it in JSON array form:</p>
<pre><code>[ { "key": "foo", "value": 3 }, { "key": "bar", "value": 1 }] 
</code></pre>
<p>What should I do?</p>
</div>
<div class="post-text" itemprop="text">
<p>You need to iterate over keys and values for this dictionary and then assign the necessary keys in the new dictionary.</p>
<pre><code>import json

input_dict = {'foo': 3, 'bar': 1}
result = []

for k, v in input_dict.items():
    result.append({'key': k, 'value': v})

print(json.dumps(result))
</code></pre>
<p>And the result:</p>
<pre><code>[{'value': 3, 'key': 'foo'}, {'value': 1, 'key': 'bar'}]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This could be handled with a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>import json
json_dict = {'foo': 3, 'bar': 1}
json_array = [ {'key' : k, 'value' : json_dict[k]} for k in json_dict]
print(json.dumps(json_array))
</code></pre>
<p>output:</p>
<blockquote>
<p>[{"key": "foo", "value": 3}, {"key": "bar", "value": 1}]</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>Try this (Python 2.7 and above):</p>
<pre><code>json_dict = {'foo': 3, 'bar': 1}  # your original dictionary;
json_array = []  # an array to store key-value pairs;

# Loop through the keys and values of the original dictionary:
for key, value in json_dict.items():
    # Append a new dictionaty separating keys and values from the original dictionary to the array:
    json_array.append({'key': key, 'value': value})
</code></pre>
<p>One-liner that does the same thing:</p>
<pre><code>json_array = [{'key': key, 'value': value} for key, value in {'foo': 3, 'bar': 1}.items()]
</code></pre>
