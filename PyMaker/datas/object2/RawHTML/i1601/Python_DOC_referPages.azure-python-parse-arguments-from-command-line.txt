<div class="post-text" itemprop="text">
<blockquote>
<p>parser.add_argument('--azure-client-id', nargs='?', const='ID',
  default='ID')</p>
</blockquote>
<p>nargs='?' = 0 or 1 arguments</p>
<p>const='ID' = sets default value when no arguments are passed</p>
<p>default='ID' = if '--azure-client-id' is not specified this will be the default value</p>
<p><a href="https://docs.python.org/3/library/argparse.html#nargs" rel="nofollow noreferrer">https://docs.python.org/3/library/argparse.html#nargs</a></p>
