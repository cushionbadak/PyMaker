<div class="post-text" itemprop="text">
<p>This is a good place to put a <a href="https://www.pythonforbeginners.com/error-handling/python-try-and-except" rel="nofollow noreferrer"><em><code>try-except</code> block</em></a> and <a href="https://docs.python.org/3/tutorial/errors.html" rel="nofollow noreferrer"><em>learn about exception handling</em></a>.</p>
<pre><code>try:
    overflow = int(input("Enter the number"))
    print(overflow)
except ValueError:
    print("Please enter numbers only.")
</code></pre>
<p><code>try-except</code> blocks are pretty cool, in the sense that you <em>force</em> an error, then <em>catch</em> it (that's what the <code>except</code> block does). </p>
<p><code>ValueError</code> exceptions will be...</p>
<blockquote>
<p>Raised when a built-in operation or function receives an argument that has the
  right type but an inappropriate value</p>
</blockquote>
<p>You can wrap this around a <code>while</code> block to prevent the code from automatically closing or carrying on.</p>
</div>
