<div class="post-text" itemprop="text">
<p>I'm working on a Python project in Visual Studio. I want to process a longer text file, this is a simplified version:</p>
<pre><code>David Tubb
Eduardo Cordero
Sumeeth Chandrashekar
</code></pre>
<p>So for reading this file I use this code:</p>
<pre><code>with open("data.txt", "r") as f:
    f_contents = f.read()
    print(f_contents)
</code></pre>
<p>I want to put these items into a new array that looks like that:</p>
<pre><code>['David Tubb','Eduardo Cordero','Sumeeth Chandrashekar']
</code></pre>
<p>Is that possible?</p>
</div>
<div class="post-text" itemprop="text">
<p>Yes, the following code will work for this:</p>
<pre><code>output = [] # the output list
nameFile = open('data.txt', 'r') 

for name in nameFile:
    # get rid of new line character and add it to your list
    output.append(name.rstrip('\n'))

print output

# don't forget to close the file!
nameFile.close()
</code></pre>
</div>

<p>The method stated by python for opening a file context is using "with open", this ensures the context will end during clean up.</p>
