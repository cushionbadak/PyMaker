<p>I have this code where given an integer n, I want to print out all integers in the interval [1â†’n] that divide n, separated with spaces. I wrote this code:</p>
<pre><code>n = int(input('Enter number:'))

for i in range(1, n+1):
    if (n%i==0):
        print (i)
</code></pre>
<p>I get this as the answer:</p>
<pre>
Enter number:8 

1

2

4

8
</pre>
<p>But I want my answer next to each other, separated using spaces (so: <code>1 2 4 8</code>). How do I do this?</p>

<div class="post-text" itemprop="text">
<p>Instead of:</p>
<pre><code>print(i)
</code></pre>
<p>You should put:</p>
<pre><code>print(i, end=" ")
</code></pre>
<p>This will change the end of line string from "\n" to " ". This will give you the desired output.</p>
<hr/>
<p>Another method would be to build a list of results and print it out at the end:</p>
<pre><code>n = int(input('Enter number:'))
final_results = list()

for i in range(1, n+1):
    if (n%i==0):
        final_results.append(str(i))

print(" ".join(final_results))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I would suggest accumulating all intermediate results, and only when the computation is done, print it.</p>
<pre><code>n = int(input('Enter number:'))
dividers = []
for i in range(1, n+1):
    if (n%i==0):
        dividers.append(i)
print(dividers)
</code></pre>
<p>If you want to print them with nice comma separation, you can do something like this:</p>
<pre><code>print(', '.join(str(divider) for divider in dividers))
</code></pre>
<p><strong>Benefits</strong></p>
<p>First, this reduces the number of calls to wherever you are printing to (by default, this is <a href="https://docs.python.org/3/library/sys.html#sys.stdout" rel="nofollow noreferrer">stdout</a>)</p>
<p>Second, the code becomes more readable and easier to adjust and expand later-on (for example, if you later decide that you want to pass those dividers onto another function) </p>
<p>Edit: adjusted the join operation per ritlew's comment</p>
</div>
