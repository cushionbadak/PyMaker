<div class="post-text" itemprop="text">
<p>Personally, I think <code>virtualenv</code> is deprecated now that <code>venv</code> comes with Python 3.7 (<a href="https://docs.python.org/3/library/venv.html" rel="nofollow noreferrer">https://docs.python.org/3/library/venv.html</a>).</p>
<p>Virtual environments are created like so with <code>venv</code> (according to the docs):</p>
<pre><code>    python3 -m venv /path/to/new/virtual/environment
</code></pre>
<p>I don't think <code>venv</code> is a <code>virtualenv</code> command (<a href="https://virtualenv.pypa.io/en/stable/userguide/#usage" rel="nofollow noreferrer">https://virtualenv.pypa.io/en/stable/userguide/#usage</a>; <a href="https://packaging.python.org/tutorials/installing-packages/#creating-virtual-environments" rel="nofollow noreferrer">https://packaging.python.org/tutorials/installing-packages/#creating-virtual-environments</a>), and at least not for what you are trying to do. The command for new virtual environments with <code>virtualenv</code> is <code>virtualenv ENV</code>.</p>
