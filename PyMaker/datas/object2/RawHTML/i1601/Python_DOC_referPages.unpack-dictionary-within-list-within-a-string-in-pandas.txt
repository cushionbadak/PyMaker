<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow noreferrer"><code>ast.literal_eval</code></a>, then the <code>str</code> accessor to retrieve the first (and only) list element. Here's an example:</p>
<pre><code>from ast import literal_eval

df = pd.DataFrame({'A': ['[{"a": 1, "b": 2}]', '[{"b": 3, "c": 4}]']})

df = df.join(pd.DataFrame(df.pop('A').apply(literal_eval).str[0].values.tolist()))

print(df)

     a  b    c
0  1.0  2  NaN
1  NaN  3  4.0
</code></pre>
