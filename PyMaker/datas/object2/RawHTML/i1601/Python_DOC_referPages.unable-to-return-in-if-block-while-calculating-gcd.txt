<p>I am trying to calculate a GCD of 2 numbers, the following code block works fine, I am using recursion, but when I am trying to return a value I am not able to do so, <code>return a</code> results in <code>None</code></p>
<pre><code>def gcd(a,b):
    if b == 0:
        print a
        return a   # This is not working 
    else:
        gcd(b,a%b)

XX = gcd(3, 5)
print (XX)
</code></pre>
<p>Output:</p>
<pre><code>1
None
</code></pre>

<div class="post-text" itemprop="text">
<p>Your code</p>
<pre><code>def gcd(a,b):
    if b == 0:
        print a
        return a   # This is not working 
    else:
        gcd(b,a%b)

XX=gcd(3,5)
print (XX)
</code></pre>
<p>will not work, because you are missing the return statement in the line <code>gcd(b,a%b)</code>. So it should be</p>
<pre><code>def gcd(a,b):
    if b == 0:
        print a
        return a
    else:
        return gcd(b,a%b)

print(gcd(12, 4))
</code></pre>
<p>By the way - if possible, don't code things yourself, use the predefined library:</p>
<pre><code>from fractions import gcd
print(gcd(4, 12))
</code></pre>

<div class="post-text" itemprop="text">
<p>Your recursion will not work... you are missing a <code>return</code> statement and also the algorithm does not work...</p>
<p>This is what a recursive gcd could look like:</p>
<pre><code>def gcd(a,b):
    r = a % b
    if r == 0:
        return a
    elif r == 1:
        return 1
    return gcd(b, r)
</code></pre>
<p>and here a non-recursive gcd:</p>
<pre><code>def gcd(a, b):
   while b:
       a, b = b, a % b
   return a
</code></pre>
<p>also note that you could just use <a href="https://docs.python.org/3/library/math.html?highlight=gcd#math.gcd" rel="nofollow noreferrer"><code>math.gcd</code></a>.</p>

<span class="comment-copy">A function returns <code>None</code> is no explicit <code>return</code> statement is executed.</span>
<span class="comment-copy">your recursive call to <code>gcd</code> needs to be returned, Like <code>return gcd(b,a%b)</code></span>
