<div class="post-text" itemprop="text">
<p>Your example is not valid JSON. Nor is it a valid Python dictionary.</p>
<p>When working with JSON, use the <a href="https://docs.python.org/3/library/json.html#module-json" rel="nofollow noreferrer">json</a> module, which will convert the <code>true</code> to Python boolean value <code>True</code>:</p>
<pre><code>&gt;&gt;&gt; jsontext="""{  
   "api_version":"3.1",
   "metadata":{  
      "total_matches":1,
      "kind":"vm",
      "length":1,
      "offset":true}
   }"""

&gt;&gt;&gt; json.loads(jsontext)

{'api_version': '3.1', 'metadata': {'total_matches': 1, 'kind': 'vm', 'length': 1, 'offset': True}}
</code></pre>
