<div class="post-text" itemprop="text">
<p>You have a string in the field  -- "False" as opposed to False. Which is cool for Django but not python.  Try <code>customer.is_vendor.to_python()</code> instead. That will wrangle it into a boolean for you. </p>
<pre><code>if "False": print 'True!' # is True
if False: print 'True!'   # Nope.

if customer.is_vendor.to_python():
   print "Is a vendor!"
</code></pre>
