<p>I have a list of tuples:</p>
<pre><code>my_list = 
 [(0.12497007846832275, 0.37186527252197266, 0.9681450128555298, 0.5542989373207092), 
 (0.18757864832878113, 0.6171307563781738, 0.8482183218002319, 0.8088157176971436), 
 (0.06923380494117737, 0.2164008915424347, 0.991775393486023, 0.41364166140556335)]
</code></pre>
<p>I want to multiply each value to 100 and then reassign them in the same order. But doing below, it throws error:</p>
<pre><code>for i in range(len(my_list)):
    for j in range(4):
        my_list[i][j] = 100 * my_list[i][j]
</code></pre>
<p><code>TypeError: 'tuple' object does not support item assignment</code></p>
<p>How can I modify these values and save them in their places? Thanks</p>

<div class="post-text" itemprop="text">
<p>If you read the <a href="https://docs.python.org/3/tutorial/datastructures.html#tuples-and-sequences" rel="nofollow noreferrer">docs</a>, you will learn that tuples are immutable (unlike lists), so you cannot change values of tuples.</p>
<p>Use a list-comprehension:</p>
<pre><code>my_list = [(0.12497007846832275, 0.37186527252197266, 0.9681450128555298, 0.5542989373207092), 
           (0.18757864832878113, 0.6171307563781738, 0.8482183218002319, 0.8088157176971436), 
           (0.06923380494117737, 0.2164008915424347, 0.991775393486023, 0.41364166140556335)]

my_list = [tuple(y * 100 for y in x ) for x in my_list]
# [(12.497007846832275, 37.186527252197266, 96.814501285552979, 55.429893732070923),   
#  (18.757864832878113, 61.713075637817383, 84.821832180023193, 80.881571769714355), 
#  (6.9233804941177368, 21.640089154243469, 99.177539348602295, 41.364166140556335)]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You cannot reassign values to tuples because they are immutable. Try creating a new list of tuples:</p>
<pre><code>my_list = [(0.12497007846832275, 0.37186527252197266, 0.9681450128555298, 0.5542989373207092), 
 (0.18757864832878113, 0.6171307563781738, 0.8482183218002319, 0.8088157176971436), 
 (0.06923380494117737, 0.2164008915424347, 0.991775393486023, 0.41364166140556335)]
my_list = [tuple(map(lambda x: x*100, item)) for item in my_list]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Tuples are immutable. A list of tuples is <em>not</em> the best structure for what you're looking to do. If you can use a 3rd party library, I recommend NumPy:</p>
<pre><code>import numpy as np

A = np.array(my_list)
A *= 100

print(A)

array([[ 12.49700785,  37.18652725,  96.81450129,  55.42989373],

       [  6.92338049,  21.64008915,  99.17753935,  41.36416614]])
</code></pre>
<p>The alternative is a list comprehension, which doesn't work in place.</p>
