<h2>Solution</h2>
<p><a href="https://docs.python.org/3/library/functions.html#exec" rel="nofollow noreferrer">exec</a> is exactly what I needed.</p>
<pre><code>if __name__ == '__main__':
    app = QApplication(sys.argv)
    ex = App()
    exec(base64.b64decode(ex.licenses.find_one({ "key": "derp"})['data']))
    sayHello()
</code></pre>
