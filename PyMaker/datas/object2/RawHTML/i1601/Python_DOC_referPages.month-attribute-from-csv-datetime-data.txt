<p>There is no easy answer on the request. You have data format that is not recognizable by <code>datetime</code> libarary for the time of writing this post.</p>
<p>You need to create your own utility function, huckish solution:</p>
<pre><code>def data_converter(date):
    subdates = date.split('/')
    if len(subdates[0]) == 1:
        subdates[0] = "0" + subdates[0]
    if len(subdates[1]) == 1:
        subdates[1] = "0" + subdates[1]
    return "/".join(subdates)
</code></pre>
<p>Input format would be <code>mm/dd/yy/ hh:mm:ss</code>:</p>
<pre><code>from datetime import datetime
dt = datetime.strptime('10/01/80/ 13:10:15', '%m/%d/%y/ %H:%M:%S')
</code></pre>
<p>Now, having ugly utility function and parser for the date you have all arsenal to create datetime in Python. You need to think and combine pieces together.</p>
<p>For more details take a look on documentation <a href="https://docs.python.org/2/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer">documentation</a> that should answer on some your questions.</p>
