<div class="post-text" itemprop="text">
<p>For example I have a list of words like so <code>['word1,word2', 'word2,word3' 'word4,word4' ...]</code></p>
<p>How can I return the list like this <code>[[word1,word2],[word3,word4],[word5,word6]...]</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You can use list comprehension with <code>split()</code></p>
<pre><code> l     = ['word1,word2', 'word2,word3' 'word4,word4']
 l_new = [x.split(',') for x in l]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>@Zeeshan Akhter's solution is nice, and very efficient. List comprehensions tend to be a bit unreadable, though, and this solution is basically the same thing, spread out for readability.</p>
<pre><code>l = ['word1,word2', 'word2,word3' 'word4,word4']
new = []
for item in l:
    new.append(item.split(','))
</code></pre>
