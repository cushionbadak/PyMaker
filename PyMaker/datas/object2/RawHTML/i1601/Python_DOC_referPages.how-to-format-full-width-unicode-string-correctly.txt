<p>It seems that python add only one half width space to format the string, but for full width characters, there should be a full width space or two half width space. Is there any solution?</p>
<p>the code is :</p>
<pre><code>template = "{:3s}测试"
print(template.format("测试"))
print(template.format("测试二"))
</code></pre>
<p>then the output:</p>
<pre><code>测试 测试
测试二测试
</code></pre>
<p>when I change <code>3</code> to <code>4</code> , the output is:</p>
<pre><code>测试  测试
测试二 测试
</code></pre>
<p>but the output I want is</p>
<pre><code>测试  测试
测试二测试
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>See <a href="https://docs.python.org/3/library/string.html#format-examples" rel="nofollow noreferrer">python docs format-examples</a> , the 6th example shows that we can use <code>{:*^30}</code> to make text center and filled with <code>*</code> ( <code>^</code> means center and the char in front of it is the fill char). Just use full width space <code>　</code> to replace <code>*</code> in this example.</p>
<p>So finally, my template string is:</p>
<p><code>{:　&gt;3s}测试</code></p>
<p><code>&gt;</code> means right align. See <a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="nofollow noreferrer">python docs format-specification</a> for all align chars</p>
<p>the output is:</p>
<pre><code>　测试测试
测试二测试
</code></pre>
<p>I also tried to use two half-width space as the fill char, but it raised <code>ValueError: Invalid format specifier</code> .</p>
<p>Hope the answer will help someone in the future</p>
</div>
