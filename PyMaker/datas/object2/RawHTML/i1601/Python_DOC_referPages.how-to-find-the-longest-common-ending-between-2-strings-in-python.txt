<p>I am trying to find code that will allow to find a common ending between two strings, for example:</p>
<pre><code>s1 = 'running'

s2 = 'ruminating' 
</code></pre>
<p>and the output would result in:</p>
<pre><code>'ing'
</code></pre>
<hr/>
<p>It must be specifically for the ending of the string but i am not sure how to do this </p>
</div>
<div class="post-text" itemprop="text">
<p>The stdlib already has <a href="https://docs.python.org/3/library/os.path.html#os.path.commonprefix" rel="nofollow noreferrer"><code>commonprefix</code></a>:</p>
<pre><code>&gt;&gt;&gt; strings = ['running', 'ruminating']
&gt;&gt;&gt; from os.path import commonprefix
&gt;&gt;&gt; commonprefix(strings)
'ru'
</code></pre>
<p>It's easily adapted for your needs:</p>
<pre><code>&gt;&gt;&gt; commonprefix([s[::-1] for s in strings])[::-1]
'ing'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can write a function and iterate:</p>
<pre><code>s1 = 'running'
s2 = 'ruminating'

def get_common(s1, s2):
    L = []
    for i, j in zip(s1[::-1], s2[::-1]):
        if i != j:
            break
        L.append(i)
    return ''.join(L[::-1])

common = get_common(s1, s2)  # ing
</code></pre>
</div>
