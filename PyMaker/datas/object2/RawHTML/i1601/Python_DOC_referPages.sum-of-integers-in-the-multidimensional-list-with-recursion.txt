<div class="post-text" itemprop="text">
<p>Well, that can easily be done with a simpler recursive function:</p>
<pre><code>def sum_it(l):
  if (isinstance(l, int)):
    return l
  elif (isinstance(l, list)):
    return sum(sum_it(elm) for elm in l)
</code></pre>
<p><strong>Highlights</strong>:<br/>
1. <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer">isinstance</a> is used for type checking.<br/>
2. <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer">sum</a> sums the elements of a given iterable</p>
<p><strong>Edit:</strong><br/>
If you are not allowed to use <code>sum</code>, a <code>for</code> loop can be used too:</p>
<pre><code>def sum_it(l):
  if (isinstance(l, int)):
    return l
  elif (isinstance(l, list)):
    res = 0
    for elm in l:
      res += sum_it(elm)
    return  res
</code></pre>
<p><strong>EDIT 2:</strong>
If for some ludicrous reason you are not even allowed to use <code>isinstance</code>, you may use exception handling, <strong>but I'd recommend against it</strong> since it makes the whole thing less readable and more messy than it is has to be:</p>
<pre><code>def sum_it(l):
  try:
    res = 0
    for elm in l:
      res += sum_it(elm)
    return  res
  except TypeError:
    return l
</code></pre>

<p>Other option using exceptions:</p>
<pre><code>mylist = [1, 8, [6, 4, 8, 9], 1, [4, 6, 9, 0], 5, 6]

def recusum(mylist, total = 0):
  for x in mylist:
    try: total = recusum(x, total)
    except: total += x
  return total

recusum(mylist) #=&gt; 67
</code></pre>
