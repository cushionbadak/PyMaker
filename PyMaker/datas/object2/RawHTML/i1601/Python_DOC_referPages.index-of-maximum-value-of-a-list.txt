<div class="post-text" itemprop="text">
<p>a better way is to just use max directly:</p>
<pre><code>max(enumerate(lst), key=lambda pair: pair[1], default=(None, None))[0]
</code></pre>
<p><code>enumerate</code> yields <code>(idx, value)</code> tuples for the list, the <code>key</code> checks for the max based on only the <code>value</code>, and then <code>[0]</code> grabs the <code>idx</code> from the tuple from <code>enumerate</code></p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#max" rel="nofollow noreferrer">max</a> built-in function to find the maximum value and <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer">index</a> method to find its index:</p>
<pre><code>def maximum_index(lst):
  if (not lst):
    return None
  return lst.index(max(lst))
</code></pre>
<p>Test it:</p>
<pre><code>print(maximum_index([1, 2, 3, 9, 8, 7, 6]))
print(maximum_index([]))
</code></pre>
<p>Output:</p>
<pre><code>3
None
</code></pre>

<div class="post-text" itemprop="text">
<p>If you aren't allowed to use <code>max</code> and <code>index</code> as suggested by @Ayxan, you should start the loop assuming that the first element <em>is</em> the max, and then updating accordingly.</p>
<pre><code>def maximum_index(lst):
    if not lst:  # check if list is empty
        return None

    # initialize maximum and index
    index = None
    maximum = lst[0]

    # loop and update
    for i, value in enumerate(lst):
        if value &gt; maximum:
            maximum = value
            index = i

    # return index
    return index


alist = [5, 6, 1, 199, -1, 0, 12]
print(maximum_index(alist))
print(maximum_index([]))
</code></pre>
