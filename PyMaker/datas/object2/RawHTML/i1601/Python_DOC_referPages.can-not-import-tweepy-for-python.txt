<div class="post-text" itemprop="text">
<p><code>async</code> may not be used as an argument name in Python 3.7.</p>
<p>See: <a href="https://docs.python.org/3/whatsnew/3.7.html#summary-release-highlights" rel="nofollow noreferrer">https://docs.python.org/3/whatsnew/3.7.html#summary-release-highlights</a></p>
<blockquote>
<p><strong>async</strong> and <strong>await</strong> are now reserved keywords.</p>
</blockquote>
<p>This issue is already known: <a href="https://github.com/tweepy/tweepy/issues/1017" rel="nofollow noreferrer">https://github.com/tweepy/tweepy/issues/1017</a></p>
<p>The current recommendation is to manually rename all occurrences of <code>async</code> to <code>async_</code> in /tweepy/streaming.py until a new release is available.</p>
</div>
