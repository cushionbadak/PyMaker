<p>I am kind of new to python and currently struggling with returning the list of tuples in a way I want to.</p>
<p>If I have a list of tuples that looks like</p>
<pre><code>  [('a',),('b',),('c',),('d',),('e',),('f',)]
</code></pre>
<p>How can I change it to </p>
<pre><code>  [('a','b'),('c','d'),('e','f')]
</code></pre>
<p>or </p>
<pre><code>  [('a','b','c'),('d','e'),('f',)] ?
</code></pre>
<p>Is there an easy way to regroup tuples?</p>
<p>Any help would be appreciated.</p>

<div class="post-text" itemprop="text">
<p>For a <em>consistent</em> length of inner tuples, you can flatten your list of tuples via <code>itertools.chain</code> and then define a chunking generator:</p>
<pre><code>from itertools import chain

L = [('a',),('b',),('c',),('d',),('e',),('f',)]

def chunker(L, n):
    T = tuple(chain.from_iterable(L))
    for i in range(0, len(L), n):
        yield T[i: i+n]

res_2 = list(chunker(L, 2))  # [('a', 'b'), ('c', 'd'), ('e', 'f')]
res_3 = list(chunker(L, 3))  # [('a', 'b', 'c'), ('d', 'e', 'f')]
res_4 = list(chunker(L, 4))  # [('a', 'b', 'c', 'd'), ('e', 'f')]
</code></pre>
<p>Otherwise, you need to first define logic to determine the size of each tuple.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could use <code>zip</code> with appropriate slices:</p>
<pre><code>l = [('a',),('b',),('c',),('d',),('e',),('f',)]

[x+y for x, y in zip(l[::2], l[1::2])]
# [('a', 'b'), ('c', 'd'), ('e', 'f')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <strong><code>grouper</code></strong> from  <a href="https://docs.python.org/3/library/itertools.html#itertools-recipes" rel="nofollow noreferrer"><code>itertools recipe</code></a>:</p>
<pre><code>from itertools import chain, zip_longest

lst = [('a',),('b',),('c',),('d',),('e',),('f',)]

def grouper(n, iterable, fillvalue=None):
    "grouper(3, 'ABCDEFG', 'x') --&gt; ABC DEF Gxx"
    args = [iter(iterable)] * n
    return zip_longest(*args, fillvalue=fillvalue)
</code></pre>
<p><strong>How to use?</strong> </p>
<pre><code>&gt;&gt;&gt; list(grouper(2, chain.from_iterable(lst)))
[('a', 'b'), ('c', 'd'), ('e', 'f')]
&gt;&gt;&gt; list(grouper(3, chain.from_iterable(lst)))
[('a', 'b', 'c'), ('d', 'e', 'f')]
</code></pre>
