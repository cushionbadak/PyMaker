<div class="post-text" itemprop="text">
<p>This is because of the line</p>
<pre><code>MT2=MT1.write("Rio de Janeiro,Brazil,30.0,18.0\n")
</code></pre>
<p>The <code>write</code> method returns <em>the number of bytes written</em>. So, after that, <code>MT2</code> is an integer equal to the length of the string you wrote.</p>
