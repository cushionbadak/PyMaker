<p>I searched a lot in the internet and I didn't find an appropriate code that searches for a string in a certain text file and replaces it with a different word?</p>
<p>Would appreciate help.. And thanks for the helpers!!</p>
<p>Let's say that the name of the text file is <code>fruits</code>and it has the following contents:</p>
<pre><code>watermelon
banana
apple orange
</code></pre>
<p>And I want to replace the string <code>"orange"</code> in the string <code>"grapes"</code></p>
</div>
<div class="post-text" itemprop="text">
<p>First open the file for reading and writing using the built-in <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer">open function</a>.</p>
<p>Then read all its contents and replace <code>orange</code> with <code>grapes</code> using <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow noreferrer">str.replace</a>.</p>
<p>Now set the offset to <code>0</code> with the <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer">seek-method</a> because you want to override the contents. 
After that write the result to the file.</p>
<pre><code>myfile = 'fruits.txt' # Replace this with the path to your file

with open(myfile, 'r+') as f:
    contents = f.read()
    result = contents.replace('orange', 'grapes')
    f.seek(0)
    f.write(result)
</code></pre>
<p>Example usage:</p>
<pre><code>$ echo -e "watermelon\nbanana\napple orange" &gt; fruits.txt
$ python test.py
$ cat fruits.txt
watermelon
banana
apple grapes
</code></pre>
