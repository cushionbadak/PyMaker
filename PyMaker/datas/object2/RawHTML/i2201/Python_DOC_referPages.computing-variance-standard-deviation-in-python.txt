<p>Python has a <a href="https://docs.python.org/3/library/statistics.html#module-statistics" rel="nofollow noreferrer">statistics</a> module. Maybe that'll be helpful.</p>
<pre><code>&gt;&gt;&gt; import statistics as stats
&gt;&gt;&gt; alist = [1,2,3,4,5,6,2,2,2]
&gt;&gt;&gt; stats.median(alist)
2
&gt;&gt;&gt; stats.mode(alist)
2
&gt;&gt;&gt; stats.mean(alist)
3
</code></pre>
