<p>If the name is too big(over 6 char) the tab will be moved one more down. You can either check if the name is too big and move it down own tab.By using something like
<code>numTabs = '\t' * (2-len(last_name)//6</code></p>
<p>Or a better approach would use something like <code>str.format</code> as mentioned by @Michael Butscher in the comments.</p>
</div>
