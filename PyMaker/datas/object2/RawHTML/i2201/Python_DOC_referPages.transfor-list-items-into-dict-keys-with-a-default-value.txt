<p>I have a list of strings of the type <code>cities=['Los Angeles', 'Cancun', 'Paris']</code>, and I want to generate a dict with a key for each item in the list, and an empty array as value, like this:</p>
<pre><code>cities_dict ={
    'Los Angeles': [],
    'Cancun': [],
    'Paris': []
}
</code></pre>
<p>I tried this <code>dict(([city.name], []) for city in cities))</code>, but I get a syntax error.</p>
<p>Is there any way I can get this dict without traversing list item by item? </p>
</div>
<div class="post-text" itemprop="text">
<p>Recent Python versions have <a href="https://docs.python.org/3/reference/expressions.html#dictionary-displays" rel="nofollow noreferrer"><code>dict</code> comprehensions</a>, so you can do this:</p>
<pre><code>{city: [] for city in cities}
</code></pre>
