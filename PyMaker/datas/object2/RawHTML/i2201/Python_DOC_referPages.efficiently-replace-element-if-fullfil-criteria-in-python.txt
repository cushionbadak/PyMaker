<p>is there an efficient way to replace the element in a list.
For example, I want to replace "banana" to 3</p>
<pre><code>thislist = ["apple", "banana", "cherry", "banana", "banana"]

thislist[thislist == "banana"] = 3
</code></pre>
<p>By doing the above code, instead of banana, the 1st row which correspond to apple is being change to int 3.</p>
<p>It will be interesting if we can skip the use of FOR-LOOP</p>
<p>Really appreciate if someone can point the right direction</p>

<div class="post-text" itemprop="text">
<p>you could use <a href="https://docs.python.org/3/library/functions.html?highlight=enumerate#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a> to get the index of the current item in your list:</p>
<pre><code>thislist = ["apple", "banana", "cherry", "banana", "banana"]

for i, fruit in enumerate(thislist):
    if fruit == 'banana':
        thislist[i] = 3
</code></pre>
<p>to explain what is happening in your code:</p>
<ul>
<li><code>thislist == "banana"</code> is always <code>False</code> (comparing a <code>str</code> to a <code>list</code>)</li>
<li><code>thislist[False] == thislist[0]</code> (as <code>False == 0</code>)</li>
<li><code>thislist[0] = 3</code>: you only set the first element.</li>
</ul>
<p>if you want to use <a href="http://www.numpy.org/" rel="nofollow noreferrer"><code>numpy</code></a> you could to the same without a (explicit) <code>for</code> loop (not in plain python though...)</p>
<pre><code>import numpy as np

a = np.array(["apple", "banana", "cherry", "banana", "banana"])
a[a == 'banana'] = 3
</code></pre>
<hr/>
<p>note that multiple replacements can be done with one <code>for</code> loop only:</p>
<pre><code>repl = {'banana': 3, 'cherry': 5}

for i, fruit in enumerate(thislist):
    if fruit in repl:
        thislist[i] = repl[fruit]

print(thislist)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>To avoid the for loop, you can try using the index() method of the list as follows:
    i = thislist.index('banana')
    thislist[i] = 3</p>
</div>
<div class="post-text" itemprop="text">
<p>When we have <code>numpy</code> solution, we nearly always have a <code>pandas</code> version solution, just FYI.</p>
<pre><code>import pandas as pd

thislist = ["apple", "banana", "cherry", "banana", "banana"]
newlist = pd.Series(thislist)
newlist[newlist=="banana"] = 3
print(newlist.tolist())
</code></pre>
<p>Output:</p>
<blockquote>
<p>['apple', 3, 'cherry', 3, 3]</p>
</blockquote>
