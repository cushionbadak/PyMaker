<div class="post-text" itemprop="text">
<p>You are passing function argument in wrong order:</p>
<p>Here,</p>
<pre><code>es = employer_search(d,ans)
</code></pre>
<p>should be</p>
<pre><code>es = employer_search(ans,d)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The issue you are getting is <a href="https://docs.python.org/3/library/exceptions.html#KeyError" rel="nofollow noreferrer"><code>KeyError</code></a> which <strong>is raised when  a mapping (dictionary) key is not found</strong> in the set of existing keys. In your case the <code>competencies</code> key is not in dict <code>d</code>.                           </p>
<p>So, you'll need to change your class call from:       </p>
<p><code>es = employer_search(d,ans)</code> </p>
<p>To:      </p>
<p><code>es = employer_search(ans,d)</code> </p>
<p>Side note:               </p>
<ul>
<li>You can make <code>__init__</code> call mandatory using named parameter by writing it as:<br/>
<code>def __init__(self,*,ans,d)</code> and then call it as <code>es = employer_search(ans=ans, d=d)</code> </li>
<li>Also, if you are using Python 3 you can simply declare it as <code>class EmployerSearch:</code>. (CamelCase)      </li>
</ul>
