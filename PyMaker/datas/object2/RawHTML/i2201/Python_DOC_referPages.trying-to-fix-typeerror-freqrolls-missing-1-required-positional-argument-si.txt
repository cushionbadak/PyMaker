<pre><code>def freqRolls(sides):
    for i in range(1, len(sides)) :
        print("%2d: %4d" % (i, sides[i]))
</code></pre>
<p>the "sides" is an argument, it means that the function expects a value and will call it "sides" only inside the function. To your function work, you need to pass this value in the moment you call it, like this:</p>
<pre><code> freqRolls(numSides)
