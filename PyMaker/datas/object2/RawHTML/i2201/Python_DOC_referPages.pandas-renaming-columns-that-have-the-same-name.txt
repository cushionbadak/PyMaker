<p>You can always just manually rename all the columns.  </p>
<pre><code>df.columns = ['a', 'b', 'c']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can simply do:</p>
<pre><code>df.columns = ['a','b','c']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If your columns are ordered and you want lettered columns, don't type names out manually. This is prone to error.</p>
<p>You can use <a href="https://docs.python.org/3/library/string.html#string.ascii_lowercase" rel="nofollow noreferrer"><code>string.ascii_lowercase</code></a>, assuming you have a maximum of 26 columns:</p>
<pre><code>from string import ascii_lowercase

df = pd.DataFrame(columns=['a', 'b', 'b1'])

df.columns = list(ascii_lowercase[:len(df.columns)])

print(df.columns)

Index(['a', 'b', 'c'], dtype='object')
</code></pre>
