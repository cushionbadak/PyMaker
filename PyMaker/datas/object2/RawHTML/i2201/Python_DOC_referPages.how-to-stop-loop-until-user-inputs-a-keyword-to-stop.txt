<p>There are a few problems here.</p>
<p>The first is that the logic to respond to a choice is outside your <code>while</code> loops. That can be fixed by indenting that whole block.</p>
<p>Second when the user inputs <code>5</code>, the condition in <code>while choice not in [1,2,3,4]:</code> evaluates to <code>True</code>, so the user is prompted to enter a valid choice again. This can be fixed by removing that inner <code>while</code> loop entirely.</p>
<p>Finally upon reaching the <code>elif choice == 5</code> block, the user will not see any of these receipt prints because <code>choice</code> is <code>5</code> and therefore not <code>1</code>, <code>2</code>, <code>3</code>, or <code>4</code>. I think what you mean here is for the count of <code>chocolate</code>, <code>strawberry</code>, <code>vanilla</code>, or <code>honey</code> to be nonzero. Also these should all be <code>if</code> rather than <code>elif</code> blocks since they are independent of each other (a user can get some chocolate and some vanilla).</p>
<p>With all that in mind here is a refactor:</p>

<h1>Final version</h1>
<p>You then have a simplified version of you code. Shorter and more easy to maintain: to add doughnuts you simply need to update the initial list <code>doughnuts</code>.</p>
<pre><code>doughnuts = [
    {'name': 'Chocolate-dipped Maple Puff', 'price': 3.50},
    {'name': 'Stawberry Twizzler', 'price': 2.25},
    {'name': 'Vanilla Chai Strudel', 'price': 4.05},
    {'name': 'Honey-drizzled Lemon Dutchie', 'price': 1.99}
]

def get_choice():
    allowed = map(str, range(1, len(doughnuts) + 2))
    while True:
        choice = input('&gt; ')
        if choice in allowed:
            return int(choice)
        else:
            print("Please enter a valid choice.")

if __name__ == '__main__':
    for i, doughnut in enumerate(doughnuts, start=1):
        print(f'{i}. {doughnut["name"]} (${doughnut["price"]} each)')
    print(f'{i + 1}. No more doughnuts.')

    receipt = [
        {
            **doughnuts[i],
            'qty': int(input(f'How many {doughnuts[i]["name"]}'))
        }
        for i in set(iter(get_choice, 5))
    ]

    print(f"Here is your receipt: ")
    for item in receipt:
        print("==========================================")
        print(f"{item['qty']} {item['name']}")
        print("==========================================")
        print(f"Total Cost: ${item['qty'] * item['price']:.2f}")
</code></pre>
