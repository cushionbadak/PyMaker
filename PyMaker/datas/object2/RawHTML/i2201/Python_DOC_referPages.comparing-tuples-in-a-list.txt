<p>So I have a list of tuples, for example:</p>
<pre><code>[('CX8888', '2017-01-01T00:10:59'), ('RJ5874', '2017-01-01T00:11:39'), ('RJ5874', '2017-01-01T00:12:08')]
</code></pre>
<p>And if the first index of one tuple is the same as another it will make a new list like this:</p>
<pre><code>[('CX8888', '2017-01-01T00:10:59'), ('RJ5874', ['2017-01-01T00:11:39', '2017-01-01T00:12:08'])]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use dictionary instead of list then convert dictionary to desired format:</p>
<pre><code>l = [('CX8888', '2017-01-01T00:10:59'), ('RJ5874', '2017-01-01T00:11:39'), ('RJ5874', '2017-01-01T00:12:08')]
d = {}
for tup in l:
    if tup[0] in d:
        d[tup[0]].append(tup[1])
    else:
        d[tup[0]] = [tup[1]]

list(zip(d.keys(),d.values()))
[('CX8888', ['2017-01-01T00:10:59']),
 ('RJ5874', ['2017-01-01T00:11:39', '2017-01-01T00:12:08'])]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>a simple way would be to use <a href="https://docs.python.org/3/library/collections.html?collections.defaultdict#collections.defaultdict" rel="nofollow noreferrer"><code>collections.defaultdict</code></a> and then convert back to a list:</p>
<pre><code>from collections import defaultdict

lst = [('CX8888', '2017-01-01T00:10:59'), 
       ('RJ5874', '2017-01-01T00:11:39'), ('RJ5874', '2017-01-01T00:12:08')]

d = defaultdict(list)
for key, value in lst:
       d[key].append(value)
d2 = list(d.items())
</code></pre>
<p>but considering that data structure you might be better off just using the dictionary <code>d</code> instead of the list <code>d2</code>.</p>
</div>
