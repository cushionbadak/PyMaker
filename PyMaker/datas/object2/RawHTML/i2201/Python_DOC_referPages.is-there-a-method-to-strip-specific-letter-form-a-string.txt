<div class="post-text" itemprop="text">
<p>Is there a method to strip specific letter form a string? I want to strip some letters from a string. For example, strip <code>a=</code> from <code>a=alpha</code> and strip <code>b=</code> from <code>b=bravo</code>. I have tried the normal strip method but the output would be <code>lph</code> and <code>ravo</code> while the targets are <code>alpha</code> and <code>bravo</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>.replace()</code> method.<br/>
Documentation - <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html#str.replace</a></p>
<p>Eg.</p>
<pre><code>&gt;&gt;&gt;a = "a=alpha"
&gt;&gt;&gt;a.replace("a=", "")
'alpha'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you want to make sure that 'a=' and 'b=' is stripped only from the beginning of the string, you have to use pattern matching, e.g.:</p>
<pre><code>import re
re.sub('^[^=]*=','','a=alpha')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can also use split like this:</p>
<pre><code>x = "a=alpha"

x.split("=")[1]
</code></pre>
</div>
