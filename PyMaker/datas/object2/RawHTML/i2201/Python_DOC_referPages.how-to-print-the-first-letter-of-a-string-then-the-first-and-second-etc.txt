<div class="post-text" itemprop="text">
<p>Here you index, so you print the character on the <code>i</code>-th index. You can use <a href="https://www.pythoncentral.io/cutting-and-slicing-strings-in-python/" rel="nofollow noreferrer"><strong><em>slicing</em></strong> [Pythoncentral]</a> to generate the list to (and perhaps including) <code>i</code>. For example:</p>
<pre><code>for i in range(len(string)):
    print(string<b>[:i+1]</b>)</code></pre>
<p>Notice the colon here, it means that we slice the string up to (but without) the <code>i</code>-th index. So <code>string[:4]</code>, will generate a list up to (but without) index <code>4</code>. So the string will have length <code>4</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can achieve that using <a href="https://developers.google.com/edu/python/strings#string-slices" rel="nofollow noreferrer">slicing</a> and the <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a> builtin:</p>
<pre><code>for index, c in enumerate(your_string, 1):
    print(your_string[:index])
</code></pre>
<p><code>enumerate</code> generates pairs of indexes and elements. Example:</p>
<pre><code>&gt;&gt;&gt; for i, c in enumerate('hello', 1):
        print(i, c)
1 h
2 e
3 l
4 l
5 o
</code></pre>
