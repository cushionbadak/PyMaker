<div class="post-text" itemprop="text">
<p>You have to make the variable <code>List</code> global to use or modify them in function:</p>
<pre><code>import random
global List
List = []

def Dice():
    global List
    List = List + [random.randint(1, 4)]
    return List

while multiplier &gt; 0:
    Dice()
    multiplier = multiplier - 1   #multiplier is how many times you rolled the dice
print(Dice())
</code></pre>
