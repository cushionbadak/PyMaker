<p>Did you try this one? See if it works.</p>
<pre><code>directory=os.path.dirname(os.path.expanduser("~/Desktop/Log.txt"))
</code></pre>
<p>if you want absolute path try</p>
<pre><code>fullpath=os.path.abspath(os.path.expanduser("~/Desktop/Log.txt"))
</code></pre>
