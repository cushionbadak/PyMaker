<div class="post-text" itemprop="text">
<p>There are a couple of ways. You can use <code>all</code> with a generator expression:</p>
<pre><code>def checker(x):
    acgt = set('acgt')
    return not all(i in acgt for i in x)
</code></pre>
<p>Alternatively, use <code>set.issubset</code> or its syntactic sugar <code>&lt;=</code>:</p>
<pre><code>def checker(x):
    return not set(x) &lt;= set('acgt')
</code></pre>
<p>Result:</p>
<pre><code>checker(AX557349)  # False
checker(AX557350)  # False
checker(Tester)    # True
</code></pre>
