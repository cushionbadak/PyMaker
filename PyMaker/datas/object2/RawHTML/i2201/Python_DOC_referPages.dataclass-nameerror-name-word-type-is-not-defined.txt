<p>The <a href="https://stackoverflow.com/questions/52627942/how-to-use-dataclasses-to-generate-a-field-value">question</a> you're referring to has this missing I believe, which is to define a custom type before the class definition.</p>
<pre><code>from typing import NewType
WORD_TYPE = NewType('WORD_TYPE', str)
</code></pre>
<p><a href="https://stackoverflow.com/questions/47955263/what-are-data-classes-and-how-are-they-different-from-common-classes">This</a> is a better example to work through for <code>dataclasses</code>.
