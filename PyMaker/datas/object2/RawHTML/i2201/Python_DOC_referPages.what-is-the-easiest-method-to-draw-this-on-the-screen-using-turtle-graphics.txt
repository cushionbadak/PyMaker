<div class="post-text" itemprop="text">
<p>It's kind of hard to answer this question without code, but if you know what the colored turtles are called, you can use:</p>
<pre><code>[turtle name goes here].forward([distance])
</code></pre>
<p>to move that given turtle forward</p>
<pre><code>[turtle name goes here].backward([distance])
</code></pre>
<p>to move it backward</p>
<pre><code>[turtle name goes here].right([degrees/radians])
[turtle name goes here].left([degrees/ radians])
</code></pre>
<p>to turn the turtle right and left</p>
<p>for example:</p>
<pre><code>#names the turtle Angie
Angie = Turtle()
#changes the color of Angie
Angie.color("blue")
#moves angie forward
Angie.forward(100)
</code></pre>
