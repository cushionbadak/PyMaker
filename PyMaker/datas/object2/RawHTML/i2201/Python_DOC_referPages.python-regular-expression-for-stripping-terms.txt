<p>try this:</p>
<blockquote>
<p>By Python definition '\W == [^a-zA-Z0-9_], which excludes all numbers,
  letters and _</p>
</blockquote>
<pre><code>&gt;&gt;&gt; import re, string
&gt;&gt;&gt; pattern = re.compile('[\W_]+')
&gt;&gt;&gt; stringe1="    hi bye hithisitstesing 8909 okay left "
&gt;&gt;&gt; pattern.sub('', stringe1.lstrip())
'hibyehithisitstesing8909okayleft'
</code></pre>
<p>another example:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; string = "Kl13@Â£$%[};'\""
&gt;&gt;&gt; pattern = re.compile('\W')
&gt;&gt;&gt; string = re.sub(pattern, '', string)
&gt;&gt;&gt; print string
Kl13
</code></pre>

<span class="comment-copy">You can use <code>str.lstrip()</code> and <code>str.rstrip()</code> in one line, too, just string them together: <code>your_string.lstrip(characters).rstrip(characters)</code>.</span>
