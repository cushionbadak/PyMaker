<div class="post-text" itemprop="text">
<p>You have already parsed a string into a datetime object. This is done by using <code>datetime.datetime.strptime()</code>. To format the object back into a string you can use the same syntax but using method <code>datetime.datetime.strftime()</code>, e.g.:</p>
<pre><code>date.strftime('%Y/%m/%d %H:%M')
</code></pre>
<p>See also <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">documentation</a>.</p>
<p>If you want <em>exactly</em> your input string (without leading 0), you can put a hyphen between percentage operator and directive character where necessary, e.g.:</p>
<pre><code>date.strftime('%Y/%-m/%-d %-H:%M')
</code></pre>
<p>This is well explained in: <a href="https://stackoverflow.com/q/904928/2648551">Python strftime - date without leading 0</a> but it is <a href="https://stackoverflow.com/a/28916983/2648551">platform dependent</a>.</p>

<p>What you are currently doing is creating a datetime object from a string and formatter as shown <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">here</a>. Likely somewhere in your code you put this object in a list and referenced it. Python doesn't know that you want to print the container(the list) with it in a certain string format.</p>
<p>If I'm understanding your question you want to print/return the element and not the container. Shown below:</p>
<pre><code>import datetime

l = []
today = datetime.date.today()
l.append(today)
#what you have
print(l)
#addressing just the first element
print(l[0])
</code></pre>
