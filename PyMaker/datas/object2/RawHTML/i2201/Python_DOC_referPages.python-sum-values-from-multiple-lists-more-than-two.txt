<div class="post-text" itemprop="text">
<p>Looking for a pythonic way to sum values from multiple lists:
I have got the following list of lists:</p>
<pre><code>a = [0,5,2]
b = [2,1,1]
c = [1,1,1]
d = [5,3,4]
my_list = [a,b,c,d]
</code></pre>
<p>I am looking for the output:</p>
<pre><code>[8,10,8]
</code></pre>
<p>I`ve used:</p>
<pre><code>print ([sum(x) for x in zip(*my_list )])
</code></pre>
<p>but zip only works when I have 2 elements in my_list.
Any idea?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> works for an arbitrary number of iterables:</p>
<pre><code>&gt;&gt;&gt; list(map(sum, zip(*my_list)))
[8, 10, 8]
</code></pre>
<p>which is, of course, roughly equivalent to <em>your</em> comprehension which also works:</p>
<pre><code>&gt;&gt;&gt; [sum(x) for x in zip(*my_list)]
[8, 10, 8]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Numpy has a nice way of doing this, it is also able to handle very large arrays. First we create the <code>my_list</code> as a numpy array as such:</p>
<pre><code>import numpy as np
a = [0,5,2]
b = [2,1,1]
c = [1,1,1]
d = [5,3,4]
my_list = np.array([a,b,c,d])
</code></pre>
<p>To get the sum over the columns, you can do the following</p>
<pre><code>np.sum(my_list, axis=0)
</code></pre>
<p>Alternatively, the sum over the rows can be retrieved by</p>
<pre><code>np.sum(my_list, axis=1)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I'd make it a <code>numpy</code> array and then sum along axis 0:</p>
<pre><code>my_list = numpy.array([a,b,c,d])    
my_list.sum(axis=0)
</code></pre>
<p>Output:</p>
<pre><code>[ 8 10  8]
</code></pre>
</div>
<span class="comment-copy">zip works for any number of elements.</span>
