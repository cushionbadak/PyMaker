<div class="post-text" itemprop="text">
<p><code>f"c: {c:_^5}"</code> is an <a href="https://docs.python.org/3/reference/lexical_analysis.html#f-strings" rel="nofollow noreferrer">f-string</a>, which can be used for formatting strings in Python 3.6+. It contains a replacement field <code>{c:_^5}</code>, which formats the value of <code>c</code> with center alignment, the width of 5 and <code>'_'</code> as the fill character.  </p>
<p>Check <a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="nofollow noreferrer">Format Specification Mini-Language</a> for more info.</p>
