<div class="post-text" itemprop="text">
<p>You need to change the method (<em>GET</em> to <em>POST</em>) in the html file and add the method in the decorator <code>@app.route</code> too. Do not forget that in this case a return (<em>GET</em>) is required to render the html file. </p>
<pre><code>@app.route("/login", methods = ['GET', 'POST'])
def login():
if request.method == "POST":
    # import pdb; pdb.set_trace()
    un = request.form.get('username', None)
    return un

return render_template('form.html')
</code></pre>

<div class="post-text" itemprop="text">
<p>no need to change your form action method. but when you use GET method for sending form data your data is transmitted as URL variables and you need to read data this way:</p>
<pre><code>if flask.request.method == 'GET':
    username = flask.request.args.get('username')
</code></pre>
<p>and if you change the method to POST, read content as below:</p>
<pre><code>if flask.request.method == 'POST':
    username = flask.request.values.get('username')
</code></pre>
