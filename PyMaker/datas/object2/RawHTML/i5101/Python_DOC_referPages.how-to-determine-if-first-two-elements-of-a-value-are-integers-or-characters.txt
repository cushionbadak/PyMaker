<p><strong>Pure Python</strong></p>
<p>You could use <code>next()</code> on a generator expression to get the first index of a character which <em>is not a digit</em> then <code>slice</code> up to this index:</p>
<pre><code>&gt;&gt;&gt; date_str = '10-2-2018'
&gt;&gt;&gt; date_str[:next(i for i, c in enumerate(date_str) if not c.isdigit())]
'10'
&gt;&gt;&gt; date_str = '3/5/2018'
&gt;&gt;&gt; date_str[:next(i for i, c in enumerate(date_str) if not c.isdigit())]
'3'
</code></pre>
<hr/>
<p><strong>Regex</strong></p>
<p>You can also use a <a href="https://docs.python.org/3/howto/regex.html" rel="nofollow noreferrer">regular expression</a>:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; date_str = '10-2-2018'
&gt;&gt;&gt; re.search('^[0-9]+', date_str).group(0)
'10'
&gt;&gt;&gt; date_str = '3/5/2018'
&gt;&gt;&gt; re.search('^[0-9]+', date_str).group(0)
'3'
</code></pre>
<hr/>
<p>I've also put this on <a href="https://regex101.com/r/CZADPb/1" rel="nofollow noreferrer"><code>regex101</code></a> where you can see an explanation of how this works.</p>

<div class="post-text" itemprop="text">
<p>You could use a regex:</p>
<pre><code>"^[0-3]?[0-9]"
</code></pre>
<ul>
<li><code>^</code> matches the beginning of the string</li>
<li><code>[0-3]?</code> matches a digit between <code>0</code> and <code>3</code>, zero or one time (because of <code>?</code>)</li>
<li><code>[0-9]</code> matches a digit between <code>0</code> and <code>9</code></li>
</ul>
<p>This regex will <strong>find your day</strong>, and also <strong>make sure that it is valid</strong>.</p>
<p>Using the regex with Python:</p>
<pre><code>import re

today_str = "10-2-2018"
date_numb = re.match(r"^[0-3]?[0-9]", today_str).group(0)

today_str = "3/5/2018"
date_numb = re.match(r"^[0-3]?[0-9]", today_str).group(0)
</code></pre>

<p><strong>This is a bad idea</strong>. You <em>can</em> use <code>re</code>, <code>isdigit</code>, <code>split</code> or some other generic functions.</p>
<p>But for your use case there are ready-made tools that <em>add</em> structure to your data, rather than turn a date into an alphanumeric variable. Here is an example with <code>pandas</code> library:</p>
<pre><code>import pandas as pd

today_str = '10-2-2018'

pd.to_datetime(today_str, dayfirst=True).day

# 10
</code></pre>
