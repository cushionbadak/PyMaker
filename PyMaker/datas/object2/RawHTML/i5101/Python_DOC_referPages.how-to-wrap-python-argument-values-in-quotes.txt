<p>Since your cmd string is created with single quotes, you can insert double quotes into it wherever you want. Just put them around the {}.</p>
<pre><code>var = 'variable with spaces'
cmd = r'python ./script.py -u username -rl "{}"'.format(var)

print(cmd)

python ./script.py -u username -rl "variable with spaces"
</code></pre>
