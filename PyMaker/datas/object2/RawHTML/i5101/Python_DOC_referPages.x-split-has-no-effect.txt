<div class="post-text" itemprop="text">
<p><code>data</code> in your code is a dictionary. So you can just access your the values of it like <code>data[state]</code> which evaluates to <code>1</code>.</p>
<p>If you get this data as a string like:
    data = "{'state':1, 'endTime':1518852709307, 'fileSize':000000}"</p>
<p>You could use <code>json.loads</code> to convert it into a dictionary and access the data like explained above.</p>
<pre><code>import json
data = '{"state":1, "endTime":1518852709307, "fileSize":0}'
data = json.loads(data)
for _,v in data.items():
   print(v)
</code></pre>
<p>If you want to parse the string yourself this should work:</p>
<pre><code>data = '{"state":1,"endTime":1518852709307,"fileSize":000000}'
