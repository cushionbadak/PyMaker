<div class="post-text" itemprop="text">
<p>You are running a command-line script <em>in your IDE</em>, with the <code>runfile()</code> command and no command-line arguments.</p>
<p><code>argparse</code> is working correctly. A required argument is missing (because <code>sys.argv</code> is not set when using <code>runfile()</code>), and so raises the <a href="https://docs.python.org/3/library/exceptions.html#SystemExit" rel="nofollow noreferrer"><code>SystemExit</code> exception</a> with the correct error code to exit the command-line script.</p>
<p>You appear to be using Spyder, in which case you'd have to set the command-line options with the <code>Run &gt; Configuration per file</code> option and add your command line options to the <code>Command line options</code> box. See <a href="https://stackoverflow.com/questions/26679272/not-sure-how-to-use-argv-with-spyder">Not sure how to use argv with Spyder</a>.</p>
