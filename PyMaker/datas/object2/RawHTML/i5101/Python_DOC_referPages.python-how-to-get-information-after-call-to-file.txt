<div class="post-text" itemprop="text">
<p><code>sys.argv</code> represents the command line arguments. Note the the first element (index 0) is the name of the script. So, in your case:\</p>
<pre><code>import sys
arg = sys.argv[1] # this will contain "getme"
</code></pre>
