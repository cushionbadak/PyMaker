<p>I am trying to convert an integer into its binary value and then print out the sum of its digits. 
my code is as follows:</p>
<pre><code>number = 30                    # declare int
binary_number = "{0:b}".format(number)        # convert into binary

print(sum(int(item) for item in binary_number ))     # print out the sum of 
digits

</code></pre>

<pre><code>for item in binary_number:
   print(sum(int(item)))

<div class="post-text" itemprop="text">
<p>This expression </p>
<pre><code>int(item) for item in binary_number
</code></pre>
<p>gives an <em>interable</em> which is why <code>sum(int(item) for item in binary_number)</code> works.</p>
<p>These lines</p>
<pre><code>for item in binary_number:
   print(sum(int(item)))
</code></pre>
<p>set <code>item</code> successively to <code>"1"</code> <code>"1"</code> <code>"1"</code> <code>"1"</code> <code>"0"</code> and inside the loop your code does <code>sum(int(item))</code> which resolves to either <code>sum(1)</code> or <code>sum(0)</code>. Now do you see why <code>sum()</code> doesn't do what you expect? As the error message says, an 'int' object is not iterable. Since you don't have an iterable, you need to do your own summing:</p>
<pre><code>total = 0
for item in binary_number:
   total += int(item)
print(total)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your rephrasing changed the semantics from</p>
<pre><code>print the sum of the elements of a list
</code></pre>
<p>to</p>
<pre><code>for the elements in this list print the sum of each single item.
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Mapping the binary number to integers and summing them will give the desired result.</p>
<pre><code>sum(map(int,"{0:b}".format(number)))
</code></pre>
