<div class="post-text" itemprop="text">
<p>The reason why you get this error:</p>
<pre><code>print(sum(self.numY,self.numX))
TypeError: 'int' object is not iterable
</code></pre>
<p>Is because you are using Pythons built-in <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer"><code>sum(iterable[, start])</code></a> method. And the first parameter it takes expects an iterable. Instead you will want to use your defined method <code>self.sum</code> that belongs to your <code>Calculator</code> class.</p>
<pre><code>print(self.sum(self.numY,self.numX))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Replace <code>sum()</code> with <code>self.sum()</code> where you get your error.
You want to do this because python's version is different from yours and needs an iterable as first argument to work. Everything in your calculator is using the self function instead of the python too so thats another reason.</p>
