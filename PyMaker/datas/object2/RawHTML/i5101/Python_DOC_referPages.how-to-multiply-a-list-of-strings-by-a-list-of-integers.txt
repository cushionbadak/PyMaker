<p>I want to multiply <code>a * b</code>:</p>
<pre><code>a = ['a', 'e', 'y']    
b = [3, 2, 1]
</code></pre>
<p>and get:</p>
<pre><code>c = ['a', 'a', 'a', 'e', 'e', 'y']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>That can be done with <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer"><code>sum()</code></a>, <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a> and a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> like:</p>
<h3>Code:</h3>
<pre><code>c = sum([[s] * n for s, n in zip(a, b)], [])
</code></pre>
<h3>Test Code:</h3>
<pre><code>a = ['a', 'e', 'y']
b = [3, 2, 1]

c = sum([[s] * n for s, n in zip(a, b)], [])
print(c)
</code></pre>
<h3>Result:</h3>
<pre><code>['a', 'a', 'a', 'e', 'e', 'y']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3.3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a> is your friend here:</p>
<pre><code>a = ['a', 'e', 'y']

b = [3, 2, 1]

c = []
for x, y in zip(a, b):
    c.extend([x] * y)

print(c)
# ['a', 'a', 'a', 'e', 'e', 'y']
</code></pre>
<p>Or with <a href="https://docs.python.org/3/library/itertools.html#itertools.chain.from_iterable" rel="nofollow noreferrer"><code>itertools.chain.from_iterable()</code></a>:</p>
<pre><code>from itertools import chain

c = list(chain.from_iterable([x] * y for x, y in zip(a, b)))

print(c)
# ['a', 'a', 'a', 'e', 'e', 'y']
</code></pre>
</div>

<p>The most basic loop approach would be this, in my opinion:</p>
<pre><code>a = ['a', 'e', 'y']
b = [3, 2, 1]
c = []

for i in range(len(a)):
    c.extend(list(a[i]*b[i]))
</code></pre>

<pre><code>a = ['a', 'e', 'y']
