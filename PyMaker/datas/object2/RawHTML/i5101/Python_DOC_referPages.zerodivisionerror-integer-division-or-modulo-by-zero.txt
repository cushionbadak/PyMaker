<p>Your loop starts with <code>0</code> so this <code>ZeroDivisionError</code> occurs.</p>
<blockquote>
<p>exception ZeroDivisionError
  Raised when the second argument of a division or modulo operation is zero. The associated value is a string indicating the type of the operands and the operation.<a href="https://docs.python.org/2/library/exceptions.html" rel="nofollow noreferrer">[source]</a></p>
</blockquote>
<p>You have to start your for loop from <code>1</code></p>
<p>like this: <code>for n in range(1,i+1):</code></p>
<p>and you don't have to do <code>while loop</code> this will go infinite.</p>
<p>your code will be:</p>
<pre><code>i = int(input("digite o 1o inteiro positivo: "))

        if i % n == 0 and j % n == 0: # 9 % (1,3,9) e 6 % (1,3,6)

<div class="post-text" itemprop="text">
<p>Function range() can have up to 3 arguments:</p>
<p>range(initial limit included, final limit not included, step)</p>
<ul>
<li><p>If you use for x in range(y) with only one argument, and such case is
equal to: for y in range(0, x, 1) # including: [0, 1, ..., x - 1]</p></li>
<li><p>If you need to start your range with 1, you need to define it: for y
in range(1, y)</p></li>
<li><p>If you need to change range step, you also need to define it: for y
in range(0, -50, -1)</p></li>
</ul>
