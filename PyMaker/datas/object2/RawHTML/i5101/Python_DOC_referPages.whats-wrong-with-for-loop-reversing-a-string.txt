<div class="post-text" itemprop="text">
<p>One way to do it in Python2 is:</p>
<pre><code>name = raw_input("Enter a name")

for i in range(0, len(name)):
    a = name[len(name)-1-i]
    print(a)
</code></pre>

<p>You probably want to go backwards like:</p>
<h3>Code:</h3>
<pre><code>name = "Enter a name"
for i in range(len(name), 0, -1):
    a = name[i-1]
    print(a)
</code></pre>
<p>But even better is:</p>
<pre><code>print(''.join(reversed(name)))
</code></pre>

<div class="post-text" itemprop="text">
<p>In the loop if you just assign to a, it will only have the first character of the original after the loop. If you need the full string after the loop, you need to keep appending to a. Also your range values are incorrect,Try the following,</p>
<pre><code>name = raw_input("Enter a name")
a=''
for i in range(len(name)-1,-1, -1):
    a += name[i]

print(a)
</code></pre>
<p>If you want the shorter approach, try the following,</p>
<pre><code>name = raw_input("Enter a name")
a = name[::-1]
print(a)
</code></pre>
