<div class="post-text" itemprop="text">
<p>I'm receiving a message that is a list of lists but is recognized as a string in Python. I can't figure out how to get Python to interpret the string as a list of lists, is there a simple way to do this? An example of the format I'm dealing with is shown below:</p>
<pre><code>&gt;&gt;soc = '[["hello","world"],["foo","bar"]]'
&gt;&gt;type(soc)
&lt;type 'str'&gt;
</code></pre>
<p>I want to convert this string into an unaltered list of lists:</p>
<pre><code>&gt;&gt;soc
[["hello","world"],["foo","bar"]]
&gt;&gt;type(soc)
&lt;type 'list'&gt;
</code></pre>
<p>I'd appreciate any help offered, thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/ast.html" rel="nofollow noreferrer"><code>ast.literal_eval</code></a>:</p>
<pre><code>import ast

soc = '[["hello","world"],["foo","bar"]]'

ast.literal_eval(soc)

# [['hello', 'world'], ['foo', 'bar']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/json.html#json.loads" rel="nofollow noreferrer"><code>json.loads</code></a></p>
<pre><code>&gt;&gt;&gt; import json
&gt;&gt;&gt; json.loads(soc)
[['hello', 'world'], ['foo', 'bar']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>why not the builtin <code>eval</code> function</p>
<pre><code>soc = eval('[["hello","world"],["foo","bar"]]')
type(soc)
Out[5]: list

soc
Out[6]: [['hello', 'world'], ['foo', 'bar']]

soc[1][0]
Out[7]: 'foo'
</code></pre>
