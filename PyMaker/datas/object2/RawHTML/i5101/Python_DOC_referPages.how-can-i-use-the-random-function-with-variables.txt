<pre><code>import random

MIN = int(input())
print ('max =')

print (random.randrange(MIN, MAX))
</code></pre>

<div class="post-text" itemprop="text">
<p>Whenever an Exception (an error) is raised in your code, the program stops runnning and displays an error message. </p>
<p>In this case it might have shown the following message on your screen:</p>
<pre><code>ValueError: non-integer arg 1 for randrange()
</code></pre>
<p>This is the way Python tells you that you did something wrong. In fact, you already got a hint of what to do to solve it: <strong><a href="https://docs.python.org/3/library/random.html#functions-for-integers" rel="nofollow noreferrer">the function randrange() only accepts integer arguments</a></strong>.</p>
<p>First you might take a look at Exceptions in The Python Standard Library, and understand what the exception <a href="https://docs.python.org/3/library/exceptions.html#ValueError" rel="nofollow noreferrer">ValueError</a> (the one raised in this case) actually means. Also, take a look at the definition of the <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer">input() function</a>, and what it really does to whatever the user sent through the prompt.</p>
<p>Then, try running your code again, but assigning your variables like this:</p>
<pre><code>MIN = int(input())
MAX = int(input())
</code></pre>
