<p>I have a string variable named dte that prints in the following format: </p>
<pre><code>Tue Feb 13 23:49:10 +0000 2018
</code></pre>
<p>I'm trying to convert it to a date type so that I can work out how long ago it was in days, hours, minutes etc but keep getting the following error:</p>
<pre><code>ValueError: time data 'Tue Feb 13 23:49:10 +0000 2018' does not match format '%a %b %d %I:%M:%S %z %Y'
</code></pre>
<p>I'm using this code:</p>
<pre><code>new_date = datetime.datetime.strptime(dte, "%a %b %d %I:%M:%S %z %Y")
print(new_date)
</code></pre>
<p>I've tried removing the colons but get the same error. I'm sure it's something very simple I'm overlooking but can't figure out what it is. </p>
<p>Any help appreciated.</p>

<div class="post-text" itemprop="text">
<p>You need <code>%H</code> for the hour component not <code>%I</code>:</p>
<pre><code>In[8]:
dte="Tue Feb 13 23:49:10 +0000 2018"
new_date = dt.datetime.strptime(dte, "%a %b %d %H:%M:%S %z %Y")
new_date

Out[8]: datetime.datetime(2018, 2, 13, 23, 49, 10, tzinfo=datetime.timezone.utc)
</code></pre>
<p>You have 24-hour hour component, not 12-hour which is what <code>%I</code> is for, see the <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer">docs</a></p>
</div>
