<p>You are trying all of your operations in the scope of <code>Student</code>.
You should try something like this:</p>
<pre><code>class Student:
    def __init__(self, fname, lname):  # self is a reference to the newly created object
        self.fname = fname
        self.lname = lname

Std1 = Student('Sushant', 'Shinde')  # Create an instance object, with specific fname and lname
print(Std1.fname)  # Sushant
Std1.fname = 'Sushant2'
print(Std1.fname)  # Sushant2

Std2 = Student('Another', 'Student')
print(Std2.fname)  # Another
print(Std2.lname)  # Student
</code></pre>

<p>Class is basically a blueprint for the objects and you have to define a class for student before creating objects for it.</p>
<pre><code>class Student:
    def __init__(self, fname, lname):
        self.fname = fname;
        self.lname = lname;


Std1 = Student('Sushant', 'Shinde')
print(Std1.fname)
</code></pre>
<p><code>__init__(self)</code> is function that acts as constructor and holds the attributes or member variable of the class. Above is right way to use class. For further reference you can read about class and objects <a href="https://docs.python.org/3/tutorial/classes.html" rel="nofollow noreferrer">here</a>.</p>

<p>This below code is a class definition, you define the attributes and behavior of the class <code>Student</code>. </p>
<pre><code>class Student:
    def __init__(self,first,last):
       self.first_name = first;
       self.last_name = last;

    def show_full_name(self):
       print(self.first_name+" "+self.last_name);
</code></pre>
<p>The function <code>__init__</code> is the function that will be called just after you create an object of <code>Student</code>, for example..</p>
<pre><code>student_A = Student("Ali", "Miles");
student_B = Student("Sarah", "Parker");
</code></pre>
<p>After this, you can use the <em>attributes</em>: <code>first_name</code> and <code>last_name</code>. You can also use the defined <em>method</em>: <code>show_full_name</code> to show the object's full name.</p>
<pre><code>student_A.show_full_name();
student_B.show_full_name();

print(student_A.first_name);
print(student_B.first_name);
</code></pre>
<p>Output of the above 4 lines code :</p>
<pre><code>Ali Miles
Sarah Parker
Ali
Sarah
</code></pre>
