<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer">open</a>() function.</p>
<p>You may define wrapper functions like:</p>
<pre><code>def save_to_file(filename, string):
    with open(filename,'w') as out_file:
        out_file.write(string)

def read_from_file(filename):
    _dir = ''
    with open(filename) as in_file:
        _dir = in_file.read()
    return _dir
</code></pre>
<p>To save(write) the string variable <code>opendir</code> to a file named <code>opendir.txt</code>:</p>
<pre><code>FILENAME = 'opendir.txt'
save_to_file(opendir, FILENAME)
</code></pre>
<p>To read from file later:</p>
<pre><code>out_dir = read_from_file(FILENAME)
</code></pre>
