<div class="post-text" itemprop="text">
<p>That's not how you capture exceptions.</p>
<pre><code>try:
    float(some value)
except ValueError as e:
    print("here's the message", e.args)
except OSError as e:
    print("here's a different message")
</code></pre>
<p>(Note, though, there's no instance when calling <code>float</code> would raise an OSError.)</p>
