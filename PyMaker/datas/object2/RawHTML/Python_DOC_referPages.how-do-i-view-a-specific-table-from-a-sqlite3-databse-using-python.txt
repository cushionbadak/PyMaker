<p>I use the below code to read a table called 'movies' from the database saved as 'ga2.db'</p>
<pre><code>conn = sqlite3.connect('ga2.db')
cur = conn.cursor()
pd.read_sql_table('movies', con=conn)
</code></pre>
<p>But I get this error <strong>"NotImplementedError: read_sql_table only supported for SQLAlchemy connectable."</strong> and I can't seem to find an example for how I can inspect a database table so that I can inspect its contents.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>pd.read_sql_query</code> instead:</p>
<pre><code>conn = sqlite3.connect('ga2.db')
qry = '''SELECT * FROM movies'''
df = pd.read_sql_query(qry, con=conn)
</code></pre>
