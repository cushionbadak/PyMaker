<div class="post-text" itemprop="text">
<p>To set which radio button is selected without using a Django form, you can apply some logic to each radio button which sets the <code>checked</code> attribute. Note that you should also ensure that the <code>value</code> attribute is unique for each element:</p>
<pre><code>&lt;th&gt;&lt;input type="radio" name="requirements_completed_{{req.pk}}" value="no"{% if not req.completed %} checked{% endif %}&gt; No
    &lt;input type="radio" name="requirements_completed_{{req.pk}}" value="yes"{% if req.completed %} checked{% endif %}&gt; Yes&lt;/th&gt;
</code></pre>
</div>
