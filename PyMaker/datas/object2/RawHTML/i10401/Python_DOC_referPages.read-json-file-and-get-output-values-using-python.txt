<div class="post-text" itemprop="text">
<p>You can do something simple like this</p>
<pre><code>import json

file = open('data.json')
json_data = json.load(file)

stud_list = json_data['Name']
y = {}
for var in stud_list:
    x = {var['name']: var['Avg']}
    y = dict(list(x.items()) + list(y.items()))

print(y)
</code></pre>
<p>It gives the output in dictionary format</p>
<p>{'Harry': '95.5', 'Lola': '78.93', 'Rose': '71.23', 'John': '55.7'}</p>
