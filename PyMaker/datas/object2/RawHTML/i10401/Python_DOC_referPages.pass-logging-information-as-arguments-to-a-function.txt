<p>I found out you can use something like a <a href="https://docs.python.org/3/library/logging.html#logging.Handler" rel="nofollow noreferrer"><code>Handler</code></a> object to do this. You'd need to define a subclass that implements <a href="https://docs.python.org/3/library/logging.html#logging.Handler.emit" rel="nofollow noreferrer"><code>emit</code></a>. See the example below for a basic setup:</p>
<pre><code>import logging

class CustomHandler(logging.Handler):
    def emit(record):
        log(record.levelname, record.msg)

logging.getLogger('some_logger').addHandler(CustomHandler())
</code></pre>
