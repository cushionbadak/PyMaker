<p>Passing arguments is simple. You can have a look at <a href="https://docs.python.org/3/library/argparse.html" rel="nofollow">https://docs.python.org/3/library/argparse.html</a></p>
<p>The most easiest way to pass argument to a python script is by adding these line to you python script and modifying them as per need:</p>
<pre><code>if __name__ == '__main__':

        sys.exit(1)
    commandlineValue = sys.argv[1] # sys.argv[0] contains the file name you are running
