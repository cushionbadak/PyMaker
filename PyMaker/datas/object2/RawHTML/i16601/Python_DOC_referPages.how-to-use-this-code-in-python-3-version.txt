<p>I want to run this code in the python 3 but i can't.Whenever i try to run the code,i get the invalid syntax error.</p>
<pre><code>age = 20
name = 'Swaroop'
print '{} was {} years old when he wrote this book'.format(name, age)
print 'Why is {} playing with that python?'.format(name)
</code></pre>
<p>Please help me.</p>
<p>Thank you.</p>

<div class="post-text" itemprop="text">
<p>Put parentheses around the <code>print</code> <em>function</em> calls.</p>
<pre><code>print('{} was {} years old when he wrote this book'.format(name, age))
print('Why is {} playing with that python?'.format(name))
</code></pre>
<hr/>
<p>In Python2, <code>print</code> is a statement, and does not require parenthesis.</p>
<p><a href="https://docs.python.org/3.0/whatsnew/3.0.html#print-is-a-function" rel="nofollow">In Python3, <code>print</code> is a function</a>, so it requires parentheses around its arguments.</p>
