<div class="post-text" itemprop="text">
<p>You are trying to use Python 2.x syntax in Python 3.x.</p>
<p>In Python 3.x, <code>print</code> is no longer a statement.  Instead, it was <a href="https://docs.python.org/3/whatsnew/3.0.html#print-is-a-function" rel="nofollow">converted into function</a> and therefore must now be called like one:</p>
<pre><code>&gt;&gt;&gt; print("hi")  # Note the parenthesis
hi
&gt;&gt;&gt;
</code></pre>
