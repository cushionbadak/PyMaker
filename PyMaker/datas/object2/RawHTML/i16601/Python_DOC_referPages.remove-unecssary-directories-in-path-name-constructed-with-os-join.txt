<div class="post-text" itemprop="text">
<p>In Python when I print directory path constructed with <code>os.join</code> I get something like this :</p>
<pre><code>rep/rep2/../rep1
</code></pre>
<p>Is there a way to get only this :</p>
<pre><code>rep/rep1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Yes, <code>os.path.normpath()</code> collapses redundant separators and up-references.</p>
<p><code>os.path.realpath()</code> converts the path to a canonical path, which includes eliminating '..' components, but it also eliminates symlinks.</p>
<p>See <a href="https://docs.python.org/2/library/os.path.html" rel="nofollow">https://docs.python.org/2/library/os.path.html</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/os.path.html#os.path.relpath" rel="nofollow"><code>os.path.relpath</code></a>:</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; os.path.relpath("rep/rep2/../rep1", start="")
'rep/rep1'
</code></pre>
<p>Or <a href="https://docs.python.org/3/library/os.path.html#os.path.normpath" rel="nofollow"><code>os.path.normpath</code></a>:</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; os.path.normpath("rep/rep2/../rep1")
'rep/rep1'
</code></pre>
</div>
