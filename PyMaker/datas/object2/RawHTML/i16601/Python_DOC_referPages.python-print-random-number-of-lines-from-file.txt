<pre><code>nameFile = ("/path/to/file.txt")

</code></pre>
<p>This will only print 1 line from file.txt</p>
<p>How would I print 1-2 lines (randomly)?</p>
<p><b>Example:</b></p>
<p>first output = apple <br/>
second output = cheesetree <br/>
third output = piecake <br/>
fourth output = cake</p>

<div class="post-text" itemprop="text">
<p>To produce more than one random number, use <a href="https://docs.python.org/2/library/random.html#random.sample" rel="nofollow"><code>random.sample()</code></a>. You can randomize the sample size:</p>
<pre><code>randomNames = random.sample(nameLines, random.randint(1, 2))
</code></pre>
<p>This gives you a <em>list</em> with either 1 or 2 items, picked as a random sample from the input.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; import random
&gt;&gt;&gt; nameLines = '''\
... apple
... cheese
... cake
... tree
... pie
... '''.splitlines()
&gt;&gt;&gt; random.sample(nameLines, random.randint(1, 2))
['apple', 'cake']
&gt;&gt;&gt; random.sample(nameLines, random.randint(1, 2))
['cheese']
</code></pre>
<p>Use <code>str.join()</code> to join the words together, if needed:</p>
<pre><code>&gt;&gt;&gt; ' '.join(random.sample(nameLines, random.randint(1, 2)))
'pie cake'
&gt;&gt;&gt; ' '.join(random.sample(nameLines, random.randint(1, 2)))
'cake'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have two basic options, depending on (let's say you're in a two-line case) whether you want to pick <em>two random lines</em>, or <em>any random line twice</em>. That is, whether or not duplicates are allowed.</p>
<p>If you want to allow duplicates, you need to pick a <a href="https://docs.python.org/3/library/random.html#random.randint" rel="nofollow"><code>randint</code></a> first, and then run the code you already have that many times. This is "pick a random line, a random number of times."</p>
<pre><code># print one or two random lines: possibly the same line twice!
for i in range(random.randint(1, 2)): # change the upper bound as desired
    print(random.choice(nameLines))
</code></pre>
<p>In the other case use <a href="https://docs.python.org/3/library/random.html#random.sample" rel="nofollow"><code>random.sample</code></a> and then print all the results. This is "pick a random number of discrete lines".</p>
<pre><code># print one or two distinct elements, chosen at random from nameLines
for line in random.sample(nameLines, random.randint(1, 2)):
    print(line)
</code></pre>
<hr/>
<p>Use the appropriate one for your use case!</p>
</div>
<div class="post-text" itemprop="text">
<p>Do you want an even probability among all of the outputs?</p>
<p>Assuming order doesn't matter and <code>n</code> lines in the text file, this means you want to choose from <code>n + n(n-1)/2 = n(n+1)/2</code> different outcomes.  That is <code>(n+1) choose 2</code>.  If you set an empty value as the additional outcome, then you will get the correct distribution.</p>
<p>Thus:</p>
<pre><code>nameFile = ("/path/to/file.txt")
nameLines = open(nameFile).read().splitlines()
nameLines.append("")
randomName = "".join(random.sample(nameLines, 2))
</code></pre>
<p>This is always choosing a <code>random.sample</code> of two, but one of the values could be the empty string that was added.  This will be as if you just choose one single value.</p>
<p>If you don't actually want an even distribution of all of the possible outcomes, then you would first want to choose if you want 1 or 2, then choose from the list of names accordingly.</p>
