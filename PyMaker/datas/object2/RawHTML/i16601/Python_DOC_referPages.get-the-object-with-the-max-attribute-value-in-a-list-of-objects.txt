<p><code>nodes_ancestors</code> is a list of objects with a <code>level</code> attribute.</p>
<p>I would like to get the object from the list with the highest level value, but I just get the highest value with:</p>
<pre><code>ancestor = max(node.level for node in nodes_ancestors)
</code></pre>
<p>What is right way to do it?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/operator.html#operator.attrgetter"><code>operator.attrgetter</code></a> and the <a href="https://docs.python.org/3/glossary.html#key-function"><code>key</code> parameter</a> of <code>max</code>:</p>
<pre><code>from operator import attrgetter
ancestor = max(nodes_ancestors, key=attrgetter('level'))
</code></pre>
<p>Below is a demonstration:</p>
<pre><code>&gt;&gt;&gt; from operator import attrgetter
&gt;&gt;&gt; class Obj:
...     def __init__(self, val):
...         self.attr = val
...
&gt;&gt;&gt; lst = [Obj(x) for x in range(10)]
&gt;&gt;&gt; obj = max(lst, key=attrgetter('attr'))
&gt;&gt;&gt; obj
&lt;__main__.Obj object at 0x021C2290&gt;
&gt;&gt;&gt; obj.attr
9
&gt;&gt;&gt;
</code></pre>
</div>
<span class="comment-copy">As always, don't forget to think about what you want to happen if there could be multiple objects with the same value of <code>level</code>.</span>
