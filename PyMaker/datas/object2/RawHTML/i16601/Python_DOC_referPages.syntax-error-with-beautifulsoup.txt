<div class="post-text" itemprop="text">
<p>In python3 you need to treat <code>print</code> as a function:</p>
<pre><code>print(tag['firstname'])
</code></pre>
<p>instead of</p>
<pre><code>print tag['firstname']
</code></pre>
<p>See this <a href="https://docs.python.org/3/whatsnew/3.0.html" rel="nofollow">Python3 release notes</a>:</p>
<blockquote>
<p>The print statement has been replaced with a print() function, with keyword arguments to replace most of the special syntax of the old print statement</p>
</blockquote>
