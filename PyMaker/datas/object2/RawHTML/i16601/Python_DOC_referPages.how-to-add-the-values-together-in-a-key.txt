<p>if I have a dictionary </p>
<pre><code>example = {'cat' : 1, 'dog' : 2, 'cow' : 3, 'horse' : 4}
</code></pre>
<p>How would I add the values together?</p>
<p>I've tried</p>
<pre><code>count = 0
for x in example:
    example[x] += count
    print example
</code></pre>
<p>But that just prints example three times. How do I do it so I get 10 as an answer?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/2/library/functions.html#sum" rel="nofollow"><code>sum</code></a> and <a href="https://docs.python.org/2/library/stdtypes.html#dict.itervalues" rel="nofollow"><code>dict.itervalues</code></a> (<a href="https://docs.python.org/3/library/stdtypes.html#dict.values" rel="nofollow"><code>dict.values</code></a> in Python 3.x):</p>
<pre><code>&gt;&gt;&gt; example = {'cat' : 1, 'dog' : 2, 'cow' : 3, 'horse' : 4}
&gt;&gt;&gt; sum(example.itervalues())
10
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>What you want is this:</p>
<pre><code>example = {'cat' : 1, 'dog' : 2, 'cow' : 3, 'horse' : 4}
for x in example:
    count += example[x]
print count
</code></pre>
<p>Or:</p>
<pre><code>example = {'cat' : 1, 'dog' : 2, 'cow' : 3, 'horse' : 4}
sum(example.values())
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You were close:</p>
<pre><code>example = {'cat' : 1, 'dog' : 2, 'cow' : 3, 'horse' : 4}
count = 0
for x in example:
    count += example[x]
print count
</code></pre>
