<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow"><code>input</code></a> returns a string object in Python 3.x.  You need to explicitly convert this into a number object:</p>
<pre><code>a = float(input("Enter 'a' value: "))
b = float(input("Enter 'b' value: "))
c = float(input("Enter 'c' value: "))
</code></pre>
<p>Otherwise, you will be giving a string to <code>math.pow</code>:</p>
<pre><code>&gt;&gt;&gt; import math
&gt;&gt;&gt;
&gt;&gt;&gt; a = input("Enter 'a' value: ")
Enter 'a' value: 2
&gt;&gt;&gt; math.pow(a, 2)
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: a float is required
&gt;&gt;&gt;
&gt;&gt;&gt; a = float(input("Enter 'a' value: "))
Enter 'a' value: 2
&gt;&gt;&gt; math.pow(a, 2)
4.0
&gt;&gt;&gt;
</code></pre>
</div>
<span class="comment-copy">A float is required, and you don't have one, because <code>input</code> returns a string in Python 3. Call <code>float</code> on the strings the user inputs to gets floats.</span>
