<p>You use numpy as a tag, so using specialized numpy methods is probably the right choice. Otherwise, you could avoid the comparatively slow loop by using a list comprehension (or generator expression):</p>
<pre><code>data = [1-1,1,1,1]
y = [8*x if x == -1 else x for x in data]
print(y)
