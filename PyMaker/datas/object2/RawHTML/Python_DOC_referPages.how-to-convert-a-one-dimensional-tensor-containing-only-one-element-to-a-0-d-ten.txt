<div class="post-text" itemprop="text">
<p>Assume that <code>a</code> is the one-dimensional tensor containing only one element. The operation which we are about to perform requires us to pass a 0-D tensor. For example, <a href="https://www.tensorflow.org/api_docs/python/tf/range" rel="nofollow noreferrer">tf.range()</a> where the start, limit and delta values have to be 0-D tensors and we have the values of start, limit and delta coming to us in the form of intermediate results of a computation and hence are in the shape as <code>[1]</code>. How do we convert them to a scalar?</p>
</div>
<div class="post-text" itemprop="text">
<p>When we declare a constant as follows:<br/>
<code>a = tf.constant(5, dtype = tf.int32)</code><br/>
then the value we get is a scalar constant.<br/>
On printing its shape we get:<br/>
<code>In[1]: se = tf.Session()</code><br/>
<code>In[2]: se.run(tf.shape(a))</code><br/>
<code>Out[2]: array([], dtype=int32)</code></p>
<p>Instead if <code>a</code> would have been of the form   </p>
<p><code>a = tf.constant([5], dtype = tf.int32)</code> </p>
<p>the shape would come as  </p>
<p><code>In[3]: se.run(tf.shape(a))</code><br/>
<code>Out[3]: array([1], dtype=int32)</code> </p>
<p><strong>Final Answer</strong></p>
<p>The above results clear it out that if <code>a</code> would have been a one dimensional tensor, in order to convert it to a scalar all we have to do is<br/>
<code>In[4]: tf.reshape(a, shape = [])</code></p>
</div>
