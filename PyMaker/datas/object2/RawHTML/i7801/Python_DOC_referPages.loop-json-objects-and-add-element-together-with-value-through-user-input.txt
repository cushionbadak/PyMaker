<p>Your code runs fine, so I am assuming the question is on how to add user input as a new key-value pair for each json you load from the file. For python 2.7 use raw_input, for python 3 raw_input was renamed to <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer">input</a>.</p>
<pre><code>import json
import pandas as pd
from pprint import pprint

tweets_data_path = 'data.json'

tweets_data = []
tweets_file = open(tweets_data_path, "r")
for line in tweets_file:
    try:
        tweet = json.loads(line)
        tweet["new_key"] = raw_input('provide input: ')
        tweets_data.append(tweet)
    except:
        continue

pprint(tweets_data)
</code></pre>
