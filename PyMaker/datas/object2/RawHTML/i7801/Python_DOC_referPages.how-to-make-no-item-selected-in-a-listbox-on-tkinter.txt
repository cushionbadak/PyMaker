<p>Is there a method to make it so that there are no items selected on a listbox in tkinter? The only idea I have right now is to delete whatever item is selected and then put it back.</p>
</div>
<div class="post-text" itemprop="text">
<p>There's a <a href="http://effbot.org/tkinterbook/listbox.htm#Tkinter.Listbox.selection_clear-method" rel="nofollow noreferrer"><code>selection_clear()</code></a> method for this purpose.</p>
<p>Try this snippet:</p>
<pre><code>try:
    import tkinter as tk
except ImportError:
    import Tkinter as tk

#   root
root = tk.Tk()

#   listbox
listbox = tk.Listbox(root, selectmode='multiple')
listbox.insert('end', *['A', 'B', 'C', 'D'])
listbox.pack()

#   clear selection button
clear_button = tk.Button(root, text='Clear Selection',
                         command=lambda: listbox.selection_clear(0, 'end'))
clear_button.pack()

#   mainloop
root.mainloop()
</code></pre>
