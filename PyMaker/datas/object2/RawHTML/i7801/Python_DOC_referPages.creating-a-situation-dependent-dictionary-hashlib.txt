<p>You can get the algorithm object using <a href="https://docs.python.org/3/library/hashlib.html#hashlib.new" rel="nofollow noreferrer">hashlib.new</a> and create a dict, iterating over <code>hashlib.algorithms_available</code>.</p>
<p>Something like this - </p>
