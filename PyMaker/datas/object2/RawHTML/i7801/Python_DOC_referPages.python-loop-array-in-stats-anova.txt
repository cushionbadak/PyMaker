<div class="post-text" itemprop="text">
<p>You can't. The closest to what you are wanting in python will be using <a href="https://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists" rel="nofollow noreferrer">argument unpacking</a>:</p>
<pre><code>some_args = ['one', 'two', 'three']
my_function(*some_args)

# equivalent to:
my_function(some_args[0], some_args[1], some_args[2])
</code></pre>
<p>Or if the function can take keyword arguments, you can unpack dictionaries:</p>
<pre><code>a = {'one':'1', 'two':'2', 'three':'3'}
my_function(**a)

# equivalent to:
my_function(one=1, two=2, three=3)
</code></pre>
<p>So, in your case, you could use a loop before the function call to build up a list of the arguments you want to use, and then use argument unpacking as above.</p>
