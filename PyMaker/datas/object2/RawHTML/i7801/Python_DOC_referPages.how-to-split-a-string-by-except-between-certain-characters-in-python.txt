<p>I'll take a stab at some code.
Splits a string by <code>needle</code> if it's outside <code>quotes</code>.
Assumes both <code>needle</code> and <code>quotes</code> are one character long.</p>
<pre><code>#!python3

def splitExceptBetween(istr, needle, quotes):
    inside = -1
    res = []
    oldt = 0
    for index, letter in enumerate(istr):
        if letter==quotes:
            inside = -inside
        elif letter==needle and inside == -1:
            res.append(istr[oldt:index])
            oldt = index+1
    if oldt&lt;len(istr):
        res.append(istr[oldt:])
    return res

istr = "as'das'd.asdas'd.a'sdas.drth..rt'h.r'th.'"
print(splitExceptBetween(istr, ".", "'"))
