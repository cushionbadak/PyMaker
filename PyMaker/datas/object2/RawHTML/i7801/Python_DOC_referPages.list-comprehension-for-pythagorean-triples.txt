<p>As for your actual question the following should be an equivalent list comprehension to print out out the <a href="http://%20https://en.wikipedia.org/wiki/Pythagorean_triple" rel="nofollow noreferrer"><strong>Pythagorean triples</strong></a> up til <code>n</code>:</p>

<p>You can use <code>itertools</code>. Try this - </p>
<pre><code>import itertools

n = int(input("Enter the value of n:"))

print([(a,b,c) for a, b, c in itertools.product(range(1,n+1), repeat=3) if a**2 + b**2 == c**2])
</code></pre>
<p>To know more about <code>itertools</code> and <code>product</code> follow this link -
<a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow noreferrer">https://docs.python.org/3/library/itertools.html#itertools.product</a></p>
<p>Using <code>product</code> we get a tuple of 3 elements which we unpack to three variables a, b, c and used comprehension to include only those tuple which satisfies the if condition.</p>
<p>This code however generates duplicates like (3,4,5) and (4,3,5)</p>
<p>To remove the duplicates you can easily modify the if statement like this - </p>
<pre><code>print([(a,b,c) for a, b, c in itertools.product(range(1,n+1), repeat=3) if a &lt; b &lt; c and a**2 + b**2 == c**2])
</code></pre>
<p>This modified code will get you following output for <code>n=17</code> -</p>
<pre><code>[(3, 4, 5), (5, 12, 13), (6, 8, 10), (8, 15, 17), (9, 12, 15)]
</code></pre>
