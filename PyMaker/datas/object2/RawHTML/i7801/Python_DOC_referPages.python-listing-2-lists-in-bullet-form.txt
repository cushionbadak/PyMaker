<div class="post-text" itemprop="text">
<p>In my python code, I have 2 lists</p>
<pre><code>myList = ["Example", "Example2", "Example3"]
mySecondList = ["0000", "1111", "2222"]
</code></pre>
<p>I need to print these so they look like this:</p>
<pre><code>- Example 0000
- Example2 1111
- Example3 2222
</code></pre>
<p>Is there any way to achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Yes it is, look for <a href="https://docs.python.org/2/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a>:</p>
<pre><code>myList = ["Example", "Example2", "Example3"]
mySecondList = ["0000", "1111", "2222"]

for a, b in zip(myList, mySecondList):
    print("- {} {}".format(a, b))
</code></pre>
<blockquote>
<pre><code>- Example 0000
- Example2 1111
- Example3 2222
</code></pre>
</blockquote>
<p>The above will work if the lists have the same size, otherwise you should look into <a href="https://docs.python.org/2/library/itertools.html#itertools.izip_longest" rel="nofollow noreferrer"><code>izip_longest</code></a> or <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow noreferrer"><code>zip_longest</code></a> from <code>itertools</code> module depending on the python version you're using</p>
</div>
<div class="post-text" itemprop="text">
<p>I would recommend using <code>zip()</code> or <code>zip_longest()</code> for your question.</p>
<p>But, without using any <code>built-in</code> modules/functions. You can create your own 'hacky' method by your own very similar to <code>zip()</code> function.</p>
<p>Here is an example:</p>
<pre><code>def custom_zip(a, b, fill=None):
    length = max(len(a), len(b))
    for k in range(length):
        if k &gt; len(a):
            yield fill, b[k]
        elif k &gt; len(b):
            yield a[k], fill
        else:
            yield a[k], b[k]

a = ["Example", "Example2", "Example3"]
b = ["0000", "1111", "2222"]

for k, v in custom_zip(a,b):
    print("- {} {}".format(k, v))
</code></pre>
<p>Output:</p>
<pre><code>- Example 0000
- Example2 1111
- Example3 2222
</code></pre>
<p>Also, you can take a look at the equivalent of <code>zip()</code> in the <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">official documentation</a>.</p>
