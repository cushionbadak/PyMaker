<p>There are two ways that I can think of, 1:<br/>
You can simply pass an ID argument to your threads, like so  </p>
<pre><code>t1 = threading.Thread(target=cart.add_to_cart, args=(['carabiner','palace','silver'],'one size', '1'))


class Cart:

  def add_to_cart(self,keywords,size, id):
      print(id + "- blah blah blah")  
</code></pre>
<p>Or 2, use the <a href="https://docs.python.org/3/library/threading.html#threading.get_ident" rel="nofollow noreferrer">thread.get_ident()</a> value if you plan on starting many threads and don't want to ID them all. </p>
<pre><code>class Cart:

  def add_to_cart(self,keywords,size):
      print(threading.get_ident() + "- blah blah blah")  
</code></pre>
