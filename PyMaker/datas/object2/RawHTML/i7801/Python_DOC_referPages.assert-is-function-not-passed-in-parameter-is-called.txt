<p>Let's say I have a function like that:</p>
<pre><code>def foo():
  bar()
</code></pre>
<p>Is there any way I could test if <code>bar</code> is called, when I'm not passing bar as parameter? My team use Python 3.6 and 3.5</p>
</div>
<div class="post-text" itemprop="text">
<p>You should use <a href="https://docs.python.org/3/library/unittest.mock.html#unittest.mock.patch" rel="nofollow noreferrer">patch</a> for that:</p>
<pre><code>@patch('path.to.bar')
def test_foo(self, mock_bar):
    foo() 
    self.assertTrue(mock_bar.called)
</code></pre>
<p>You can also test with which values the function was called like this:</p>
<pre><code>mock_bar.assert_called_with('some_param')
</code></pre>
<p>Hope it helps.</p>
</div>
<span class="comment-copy">You could probably modify <a href="https://wiki.python.org/moin/PythonDecoratorLibrary#Counting_function_calls" rel="nofollow noreferrer">these function call counting decorators</a> to suit your purpose.</span>
<span class="comment-copy">There are a number of alternatives, depending on your exact needs - search <code>python decorator function logging</code>.  Try one out and come back with questions about it if you get stuck.</span>
