<p>How can I find a string (<code>PartialWord</code>) inside a List (<code>WordList</code>) in Python 2.7?</p>
<pre><code>PartialWord = "ab"
WordList = ['absail', 'rehab', 'dolphin']
</code></pre>
<p>A search with a wildcard like: ab*</p>
<p>Where it would ONLY find the word, if it starts with those letters (i.e. the result should only give absail, but not rehab, despite both having 'ab').</p>
<p>The WordList would be a dictionary, which is over 700KB.</p>
<p>Please keep the answers fairly simple as I'm on day 4 of learning to code.</p>
</div>
<div class="post-text" itemprop="text">
<p>You may use <a href="https://docs.python.org/3/library/stdtypes.html#str.startswith" rel="nofollow noreferrer"><code>str.startswith(..)</code></a> along with a <em>list comprehension</em> to get the list of the words starting with some string as:</p>
<pre><code>&gt;&gt;&gt; PartialWord = "ab"
&gt;&gt;&gt; WordList = ['absail', 'rehab', 'dolphin']

&gt;&gt;&gt; [word for word in WordList if word.startswith(PartialWord)]
['absail']
</code></pre>
<hr/>
<p>As per the <a href="https://docs.python.org/3/library/stdtypes.html#str.startswith" rel="nofollow noreferrer"><code>str.startswith</code> document</a>: </p>
<blockquote>
<p><strong>str.startswith(prefix[, start[, end]]):</strong> </p>
<p>Return <code>True</code> if string starts
  with the <em>prefix</em>, otherwise return <code>False</code>. <em>prefix</em> can also be a <code>tuple</code> of
  prefixes to look for. With optional <em>start</em>, test string beginning at
  that position. With optional <em>end</em>, stop comparing string at that
  position.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">

    if word.startswith(PartialWord):

</code></pre>
