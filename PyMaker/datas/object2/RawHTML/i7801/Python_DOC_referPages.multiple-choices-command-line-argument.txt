<div class="post-text" itemprop="text">
<p>How could a multiple choices argument in the command line be implemented? There would be a predefined set of options and the user can choose multiple:</p>
<pre><code>python cli.py --alphabet upper,lower,digits,symbols
</code></pre>
<p>or</p>
<pre><code>python cli.py --alphabet upper lower digits symbols
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>See:</p>
<ul>
<li><a href="https://docs.python.org/3/library/argparse.html#choices" rel="nofollow noreferrer">https://docs.python.org/3/library/argparse.html#choices</a></li>
<li><a href="https://docs.python.org/3/library/argparse.html#nargs" rel="nofollow noreferrer">https://docs.python.org/3/library/argparse.html#nargs</a></li>
</ul>
<p>Example:</p>
<pre><code>&gt;&gt;&gt; import argparse
&gt;&gt;&gt; parser = argparse.ArgumentParser(prog='game.py')
&gt;&gt;&gt; parser.add_argument('--move', choices=['rock', 'paper', 'scissors'], nargs="+")
&gt;&gt;&gt; parser.parse_args(['--move', 'rock', 'paper'])
Namespace(move=['rock', 'paper'])
&gt;&gt;&gt; parser.parse_args(['--move','fire'])
usage: game.py [-h] [--move {rock,paper,scissors} [{rock,paper,scissors} ...]]
game.py: error: argument --move: invalid choice: 'fire' (choose from 'rock', 'paper', 'scissors')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>From <a href="https://pymotw.com/2/argparse/#variable-argument-lists" rel="nofollow noreferrer">Variable Argument Lists</a> of <a href="https://pymotw.com/3/argparse/" rel="nofollow noreferrer"><code>argparse</code></a> in <a href="https://pymotw.com/3/" rel="nofollow noreferrer">Python 3 Module of the Week</a>:</p>
<blockquote>
<p>You can configure a single argument definition to consume multiple arguments on the command line being parsed. Set <code>nargs</code> to one of these flag values, based on the number of required or expected arguments:</p>
</blockquote>
<p>So in your case you need to supply </p>
<pre><code>parser.add_argument('--alphabet', nargs='+')
</code></pre>
<p>Which would stand for <em>All, and at least one, argument</em></p>
<p>And then call it with:</p>
<pre><code>python cli.py --alphabet upper lower digits symbols
</code></pre>
</div>
