<div class="post-text" itemprop="text">
<p>In this case you can try getting it by split:</p>
<pre><code>expression.split("\"")[1]
</code></pre>

<div class="post-text" itemprop="text">
<pre class="lang-py prettyprint-override"><code>import re

re.match(
    '@RequestMapping\((value\s*=\s*)?"([^"]+)"\)',
    '@RequestMapping(value = "/list/base/info")'
).group(2)
</code></pre>
<p>That would output:</p>
<pre><code>'/list/base/info'
</code></pre>
