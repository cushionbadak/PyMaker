<div class="post-text" itemprop="text">
<p>I have a list of lists. How can I get a specific index members in each element as a list without using a loop and assignment. Below code snippet is an example. Actual list contains more than 3 sublists. Output should be like a list say mem_lst given below.</p>
<pre><code>    alst = [['abc', 1 , 2], ['cde', 2, 2], ['xyz', 2, 2]]
    o/p: mem_lst = [ 'abc', 'cde', 'xyz']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>In [1]: alst = [['abc', 1 , 2], ['cde', 2, 2], ['xyz', 2, 2]]

In [2]: import operator

In [3]: map(operator.itemgetter(0), alst)
Out[3]: &lt;map at 0x104b43160&gt;

In [4]: list(_)
Out[4]: ['abc', 'cde', 'xyz']
</code></pre>
<p>In this approach you don't use loop, and you can use <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map</code></a> as a iterator to avoiding storage of <code>list</code>, but if you strongly need a <code>list</code>, just need to wrap it in <a href="https://docs.python.org/3/library/functions.html#func-list" rel="nofollow noreferrer"><code>list()</code></a></p>
</div>
<div class="post-text" itemprop="text">
<p>Without any kind of a loop:</p>
<pre><code>alst = [['abc', 1 , 2], ['cde', 2, 2], ['xyz', 2, 2]]
lst = [alst[0][0],alst[1][0],alst[2][0]]
print(lst)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could do this, in python 2.x,</p>
<p><strong>zip(*alst)[0]</strong></p>
<pre><code>&gt;&gt;&gt; lst = [['abc', 1, 2], ['cde', 2, 2], ['xyz', 2, 2]]
&gt;&gt;&gt; zip(*lst)[0]
('abc', 'cde', 'xyz')
&gt;&gt;&gt; 
</code></pre>
<p>In python 3, you can do it just like this,</p>
<p><strong>list(zip(*alst))[0]</strong></p>
<pre><code>&gt;&gt;&gt; lst
[['abc', 1, 2], ['cde', 2, 2], ['xyz', 2, 2]]
&gt;&gt;&gt; list(zip(*lst))[0]
('abc', 'cde', 'xyz')
&gt;&gt;&gt; 
</code></pre>
</div>
