<div class="post-text" itemprop="text">
<p>This should produce exactly the same output as in the question:</p>
<pre><code>for chunk in chunks:
    print(''.join(str(x).rjust(5 if i else 2) for i, x in enumerate(chunk)))
</code></pre>
<p><strong>Update:</strong> For every chunk it will first convert numbers to strings with <code>str</code> and then right justify them with <a href="https://docs.python.org/3/library/stdtypes.html#str.rjust" rel="nofollow"><code>rjust</code></a> that uses space as default fill char. Since the first number on the row has width of <code>2</code> and rest of them have width of <code>5</code> <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow"><code>enumerate</code></a> is used to track the index so that correct argument can be passed to <code>rjust</code>. Enumerate returns tuples in form <code>(index, item)</code> and the index is then used in <code>5 if i else 2</code> to determine if number is first or not so width of <code>2</code> or <code>5</code> can be used respectively. Finally all the substrings are joined together for a row which is printed to screen.</p>
