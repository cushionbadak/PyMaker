<div class="post-text" itemprop="text">
<p>An empty string is always considered to be part of any string:</p>
<pre><code>&gt;&gt;&gt; "" in "abcd"
True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The empty string is a substring of <em>every</em> string in Python. This is because the substring of length 0 of any string <code>s[0:0]</code> is equal to the empty string.</p>
<p>A proper way to check if a string <code>s</code> is not empty is simply <code>if not s:</code>, as empty strings are "falsey".</p>
<p>To check if something is <em>equal</em> to the empty string, just use <code>==</code> as opposed to <code>in</code>. </p>
</div>
