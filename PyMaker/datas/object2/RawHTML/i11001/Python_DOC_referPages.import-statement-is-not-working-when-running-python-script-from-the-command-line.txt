<div class="post-text" itemprop="text">
<p>I found the answer for my question above, and the problem was much easier than I thought.</p>
<p><strong>Addressing the Problem</strong></p>
<ul>
<li>having many python packages in different directories</li>
<li>your script needs some/all packages, that are not in the standard lib-directory for your python installation (e.g.:<code>prefix/lib/pythonVersion</code>).</li>
</ul>
<p><strong>Solution</strong></p>
<p><strong>Short term solution</strong></p>
<p>As long you are using an IDE (e.g. PyCharm), it is sufficient within the code to add:</p>
<p><code>import sys
sys.path.append("path/to/package")</code></p>
<p>As soon as you have to run your script from the command line, you will get an <code>ImportError</code> as mentioned in the Question above.</p>
<p><strong>Better solution</strong></p>
<p>Add the directories of your packages and of your python installation to your shell-profile(e.g.: <code>.bashrc</code>) using the command:</p>
<p><code>export PYTHONPATH=prefix/lib/pythonVersion:/path/to/packages</code></p>
<p>To get more info about <code>PYTHONPATH</code>, check this <a href="https://docs.python.org/3/using/cmdline.html#envvar-PYTHONPATH" rel="nofollow">link</a></p>
<p>In this case you will not need to append the path of your packages within your code :)</p>
