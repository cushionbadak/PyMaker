<p>I only know how to enter strings and numbers, but is there a way to enter a list? Basically, I have a dictionary and each key is being put into the database. The value to each key is a list and I want this in the database as well. </p>

<div class="post-text" itemprop="text">
<p>SQLite3 does not have array data type. (See <a href="https://www.sqlite.org/datatype3.html" rel="nofollow">Datatypes in SQLite version 3</a>)</p>
<p>But, you can serialize to insert it to sqlite3 table.</p>
<p>For example, you can convert the list into json string, and use the string to insert into the table. (<a href="https://docs.python.org/3/library/json.html#json.dumps" rel="nofollow"><code>json.dumps</code></a>)</p>
<pre><code>cursor.execute('INSERT INTO a_table (..., a_list_field) values (..., ?)',
               [..., json.dumps(a_list_object)])
</code></pre>
<p>When you select it, convert it back to list by loading json string. (<a href="https://docs.python.org/3/library/json.html#json.loads" rel="nofollow"><code>json.loads</code></a>)</p>
</div>
<div class="post-text" itemprop="text">
<p>With sqlite3 I was able to execute this script that drops a database and rewrites it with new data using a list named <code>data</code>. This may or may not answer your specific question:</p>
<pre><code>from views import db
from config import DATABASE_PATH

import sqlite3
from datetime import datetime

with sqlite3.connect(DATABASE_PATH) as connection:
    c = connection.cursor()

    c.execute("""ALTER TABLE tasks RENAME TO old_tasks""")

    db.create_all()

    c.execute("""SELECT name, due_date, priority,
                status FROM old_tasks ORDER BY task_id ASC""")

    data = [(row[0], row[1], row[2], row[3],
        datetime.now(), 1) for row in c.fetchall()]

 c.executemany("""INSERT INTO tasks (name, due_date, priority, status,
                    posted_date, user_id) VALUES (?, ?, ?, ?, ?, ?)""", data)


    c.execute("DROP TABLE old_tasks")
</code></pre>
<p>When the above script is executed, <code>data</code> takes in <strong>4</strong> sub lists and uses the data from them to create a new database.</p>
<p>Hope I helped!</p>
