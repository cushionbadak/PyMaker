<div class="post-text" itemprop="text">
<p>You are calling a float value here:</p>
<pre><code>3.14159265359 (self.radius **2)
</code></pre>
<p>Note that there is nothing between the float and the <code>(...)</code>. You probably wanted to multiply, so you are missing a <code>*</code> there:</p>
<pre><code>return ((2 * 3.14159265359 * (self.radius **2)) + (2 * 3.14159265359 * self.radius * self.height))
</code></pre>
<p>You may want to use the <a href="https://docs.python.org/3/library/math.html#math.pi" rel="nofollow"><code>math.pi</code> constant</a> rather than typing it out each time:</p>
<pre><code>from math import pi

# ...

return 2 * pi * (self.radius ** 2) + 2 * pi * self.radius * self.height
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Replace <code>3.14159265359 (self.radius **2)</code> with <code>3.14159265359 * (self.radius **2)</code>. Or even better, import <code>math</code> and replace <code>3.14159265359</code> by <code>math.pi</code>. See <a href="https://docs.python.org/3.5/library/math.html" rel="nofollow">reference</a> for <code>math</code> module.</p>
