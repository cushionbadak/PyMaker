<p>Your solution uses 4 blocks of codes of 4 lines each, the last 3 lines are the same, and the first is almost the same too!</p>
<p>You can avoid this by using a loop, as you guessed.</p>
<p>Look at what changes between each of your blocks, and stores it in a tuple (for example). Then, loop through this tuple and apply the block with the variable element.</p>
<pre><code>texts = (
    'hello this is a test',
    'statement 1',
    'statement 2',
    'statement 3'
)

for text in texts:
    msg = visual.TextStim(win, text=text, pos=[0, +1], color='white')
    msg.draw()
    win.flip()
    core.wait(2)
</code></pre>
