<p>I agree that it's best not to use <code>regex</code> and better to use <code>urlparse</code> but here is my <code>regex</code>.
Classes like <code>urlparse</code> were developed specifically to handle all urls efficiently and are much more reliable than <code>regex</code> is so make use of them if you can.</p>
<pre><code>&gt;&gt;&gt; x = 'http://www.example.com:8080/abcd/dir/file1.html?query1=value1&amp;query2=value2'
&gt;&gt;&gt; query_pattern='(query\d+)=(\w+)'
&gt;&gt;&gt; # query_pattern='(\w+)=(\w+)'    a more general pattern
&gt;&gt;&gt; re.findall(query_pattern,x)
[('query1', 'value1'), ('query2', 'value2')]
</code></pre>
