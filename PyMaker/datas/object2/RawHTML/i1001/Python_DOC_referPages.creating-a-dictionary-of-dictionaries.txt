<div class="post-text" itemprop="text">
<p>Use nested-dictionary-comprehension:</p>
<pre><code>print({k:{int(k2):[] for k2 in list_two} for k in list_one})
</code></pre>
<p>Output:</p>
<pre><code>{'a': {1: [], 2: []}, 'b': {1: [], 2: []}}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could always make a nested <a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow noreferrer"><code>collections.defaultdict()</code></a> of lists:</p>
<pre><code>from collections import defaultdict
from pprint import pprint

list_one = ['a', 'b']
list_two = ['1', '2']

d = defaultdict(lambda : defaultdict(list))

for x in list_one:
    for y in list_two:
        d[x][int(y)]

pprint(d)
</code></pre>
<p>Which will automatically initialize a list inside for you:</p>
<pre><code>defaultdict(&lt;function &lt;lambda&gt; at 0x000002AEA8D4C1E0&gt;,
            {'a': defaultdict(&lt;class 'list'&gt;, {1: [], 2: []}),
             'b': defaultdict(&lt;class 'list'&gt;, {1: [], 2: []})})
</code></pre>
<p>You can then append values to these inner lists, since <code>defaultdict()</code> initialised empty lists for you. </p>
<p>Addtionally, you can also use <a href="https://docs.python.org/3/library/stdtypes.html#dict.setdefault" rel="nofollow noreferrer"><code>dict.setdefault()</code></a> here aswell:</p>
<pre><code>list_one = ['a', 'b']
list_two = ['1', '2']

d = {}
for x in list_one:
    d.setdefault(x, {})
    for y in list_two:
        d[x].setdefault(int(y), [])

print(d)
# {'a': {1: [], 2: []}, 'b': {1: [], 2: []}}
</code></pre>
</div>
