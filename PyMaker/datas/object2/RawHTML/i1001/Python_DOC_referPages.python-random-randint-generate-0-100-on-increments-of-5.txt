<div class="post-text" itemprop="text">
<p>Is it possible to generate a random number with <code>random.randint</code> from 0-100 on increments of 5?</p>
<pre><code>random.randint(0,100)
</code></pre>
<p>If its possible I need way to generate random integers from 0-100 but only with increments of 5.</p>
</div>
<div class="post-text" itemprop="text">
<p>Either use <a href="https://docs.python.org/3/library/random.html#random.randrange" rel="nofollow noreferrer"><code>random.randrange()</code></a> which lets you pick a step value:</p>
<pre><code>random.randrange(0, 101, 5)
</code></pre>
<p>or just generate a number between <code>0</code> and <code>20</code> and multiply that number by 5:</p>
<pre><code>random.randint(0, 20) * 5
</code></pre>
<p>I'm assuming here that <code>100</code> is a valid result. If <code>100</code> should not be generated, use <code>random.randrange(0, 100, 5)</code> or <code>random.randint(0, 19) * 5</code>.</p>
<p>If you wanted to generate a <code>numpy</code> array of such numbers using the <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.random.randint.html#numpy.random.randint" rel="nofollow noreferrer"><code>numpy.random.randint()</code> function</a> (instead of the <a href="https://docs.python.org/3/library/random.html#random.randint" rel="nofollow noreferrer">standard library <code>random</code> module version</a>), just use</p>
<pre><code>np.random.randint(0, 20, size=expected_size) * 5
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The following code will generate random integer between 0 to 100</p>
<pre><code>    import numpy as np
    random.choice(np.arange(0,101,5))
</code></pre>
</div>
