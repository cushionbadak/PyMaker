<p>I am given:</p>
<pre><code>list = [{'a','b'},{'c','d'}]
</code></pre>
<p>and I want to know if 'a' is in the list.</p>
<p>should I first unpack the list as <a href="https://stackoverflow.com/questions/51321915/python-unpacking-the-lists-and-sets">here</a> to new_list and the use 'a' in new_list</p>
<p>Or is there a shorter way (without importing modules)</p>

<div class="post-text" itemprop="text">
<p>use <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer">any</a></p>
<pre><code>spam = [{'a','b'},{'c','d'}]
eggs = [{'d','b'},{'c','d'}]

print(any('a' in my_set for my_set in spam))
print(any('a' in my_set for my_set in eggs))
</code></pre>
<p>Output</p>
<pre><code>True
False
&gt;&gt;&gt;
</code></pre>

<div class="post-text" itemprop="text">
<p>This is one approach using <code>any</code>.</p>
<p><strong>Ex:</strong></p>
<pre><code>l = [{'a','b'},{'c','d'}]
print( any(map(lambda x: "a" in x, l)) )
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>True
</code></pre>

    for list in lists:

            print("True")
            return True
    print("False")
    return False
