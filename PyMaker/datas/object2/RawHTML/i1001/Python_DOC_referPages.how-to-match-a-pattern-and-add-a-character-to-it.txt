<p>I have something like: </p>
<pre><code>GCF_002904975:2.6672e-05):2.6672e-05.
</code></pre>
and I would like to add the word '_S' right after any GCF(any number) entry before the next colon.

<p>In other words I would like my text becoming like:</p>
<pre><code>GCF_002904975_S:2.6672e-05):2.6672e-05.
</code></pre>
I have repeated pattern like that all along my text.
    </div>
<div class="post-text" itemprop="text">
<p>This can be easily done with <code>re.sub</code> function. A working example would look like this:</p>
<pre><code>import re

inp_string='(((GCF_001297375:2.6671e-05,GCF_002904975:2.6672e-05)0.924:0.060046136,(GCF_000144955:0.036474926,((GCF_001681075:0.017937143,...'

if __name__ == "__main__":
    outp_string = re.sub(r'GCF_(?P&lt;gfc_number&gt;\d+)\:', r'GCF_\g&lt;gfc_number&gt;_S:', inp_string)
    print(outp_string)
</code></pre>
<p>This code gives the following result, which is hopefully what you need:</p>
<p><code>(((GCF_001297375_S:2.6671e-05,GCF_002904975_S:2.6672e-05)0.924:0.060046136,(GCF_000144955_S:0.036474926,((GCF_001681075_S:0.017937143,...</code></p>
<p>For more info take a look at the docs:
<a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">https://docs.python.org/3/library/re.html</a></p>

<p>You can use regular expressions with a function substitution.  The solution below depends on the numbers always being 9 digits, but could be modified to work with other cases.</p>
<pre><code>test_str = '(((GCF_001297375:2.6671e-05,GCF_002904975:2.6672e-05)0.924:0.060046136,GCF_000144955:0.036474926,((GCF_001681075:0.017937143,...'
new_str = re.sub(r"GCF_\d{9}", lambda x: x.group(0) + "_S", test_str)

print(new_str)
#(((GCF_001297375_S:2.6671e-05,GCF_002904975_S:2.6672e-05)0.924:0.060046136,GCF_000144955_S:0.036474926,((GCF_001681075_S:0.017937143,...
</code></pre>
</div>
