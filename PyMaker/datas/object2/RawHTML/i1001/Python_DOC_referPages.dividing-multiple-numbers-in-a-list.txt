<p>You can use <a href="https://docs.python.org/3/library/functools.html#functools.reduce" rel="nofollow noreferrer"><code>reduce</code></a> for this.</p>
<pre><code>&gt;&gt;&gt; import functools, operator
&gt;&gt;&gt; functools.reduce(operator.truediv, [10, 5, 2]) 
1.0
