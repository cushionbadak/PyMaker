<div class="post-text" itemprop="text">
<p>In python I want to access the values of some element(s). Particular, I have the following structure extracted in a local variable <code>msg</code>: </p>
<pre><code>        &lt;timestamp&gt;2019-01-08T03:45:30-08:00&lt;/timestamp&gt;
        &lt;stage&gt;QUEUE&lt;/stage&gt;
        &lt;text&gt;Added to NSG run queue.&lt;/text&gt;
</code></pre>
<p>I just want to print the content. The following code is working</p>
<pre><code>print(msg.find('timestamp').text, msg.find('stage').text, msg.find('text').text)
</code></pre>
<p>but it looks like extreme overhead. Is there a simpler way to do this?</p>
<p>Expected output: </p>
<pre><code>('2019-01-08T03:45:30-08:00', 'QUEUE', 'Added to NSG run queue.')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try xmltodict. That gives you add values as the dict's <code>values()</code>.</p>
<pre><code>import xmltodict

s = """&lt;xml&gt;
          &lt;timestamp&gt;2019-01-08T03:45:30-08:00&lt;/timestamp&gt;
          &lt;stage&gt;QUEUE&lt;/stage&gt;
          &lt;text&gt;Added to NSG run queue.&lt;/text&gt;
          &lt;/xml&gt;
  """

print(", ".join(xmltodict.parse(s)['xml'].values()))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Seems what u want was contents of all labels, which can use the standard lib <code>re</code> to achieve that.</p>
<pre><code>import re

exp = '&gt;(.*)&lt;/'
print(re.findall(exp, msg))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can iterate over the <code>msg</code> element to get its children. This produces the wanted output:</p>
<pre><code>from xml.etree import ElementTree as ET

XML = """
&lt;msg&gt;
  &lt;timestamp&gt;2019-01-08T03:45:30-08:00&lt;/timestamp&gt;
  &lt;stage&gt;QUEUE&lt;/stage&gt;
  &lt;text&gt;Added to NSG run queue.&lt;/text&gt;
&lt;/msg&gt;"""

msg = ET.fromstring(XML)

print(tuple(c.text for c in msg))  # Tuple
print([c.text for c in msg])       # List
</code></pre>
