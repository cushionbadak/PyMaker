<p>I'm trying to make a calculator in python, so when you type x (root) y it will give you the x root of y, e.g. 4 (root) 625 = 5. </p>
<p>I'm aware of how to do math.sqrt() but is there a way to do other roots?</p>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/math.html#math.pow" rel="nofollow noreferrer">pow</a>:</p>
<pre><code>import math

print(math.pow(8, 1/3))
</code></pre>
<p><strong>Output</strong></p>
<pre><code>2.0
</code></pre>
<p>For your example in particular:</p>
<pre><code>print(math.pow(625, 1/4))
</code></pre>
<p><strong>Output</strong></p>
<pre><code>5.0
</code></pre>
<p>See more about exponentiation with rational exponents, <a href="https://en.wikipedia.org/wiki/Exponentiation#Rational_exponents" rel="nofollow noreferrer">here</a>. Note that in Python 2, as mentioned by @Jean-Fran√ßoisFabre, you  need to specify the exponent to <code>1.0/n</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you want to 625^(1/4){which is the same as 4th root of 625}
then you type 625**(1/4)</p>
<p><code>**</code> is the operator for exponents in python.</p>
<p><code>print(625**(1/4))</code></p>
<p><strong>Output:</strong></p>
<p><code>5.0</code></p>
<p>To generalize:</p>
<p>if you want to find the xth root of y, you do:</p>
<p><code>y**(1/x)</code></p>
</div>
