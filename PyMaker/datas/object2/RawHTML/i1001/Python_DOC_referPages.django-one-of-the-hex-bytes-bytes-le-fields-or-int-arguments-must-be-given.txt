<div class="post-text" itemprop="text">
<p>The error message tells you exactly what the problem is: you need to specify one of those arguments in your call to <code>uuid.UUID</code> (on line 78 of views.py).  <a href="https://docs.python.org/3/library/uuid.html#uuid.UUID" rel="nofollow noreferrer">The doc</a> is pretty clear:</p>
<blockquote>
<p>Exactly one of hex, bytes, bytes_le, fields, or int must be given.</p>
</blockquote>
