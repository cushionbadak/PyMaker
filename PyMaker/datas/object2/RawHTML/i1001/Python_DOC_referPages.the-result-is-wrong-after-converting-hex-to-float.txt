<p>I am trying to convert Hex <code>4991a480</code> to float using Python 3.6</p>
<pre><code>&gt;&gt;&gt; struct.unpack('!f',bytes.fromhex("4991a480"))
(1193104.0,)
</code></pre>
<p>But it doesn't work correctly.</p>
<p>The result I want to get is 1.1931e+06. I used a conversion website, and it gave the result I was expecting. What am I doing wrong? </p>
</div>
<div class="post-text" itemprop="text">
<p>It's the same thing, you just need to format it (<a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="nofollow noreferrer">[Python 3]: Format Specification Mini-Language</a>) to the <em>exponent notation</em> (<a href="https://en.wikipedia.org/wiki/IEEE_754" rel="nofollow noreferrer">[Wikipedia]: IEEE 754</a>):</p>
<blockquote>
<pre><code>&gt;&gt;&gt; import struct
&gt;&gt;&gt; unpacked = struct.unpack("!f", bytes.fromhex("4991a480"))
&gt;&gt;&gt; unpacked
(1193104.0,)
&gt;&gt;&gt; print("{:e}".format(unpacked[0]))
1.193104e+06
</code></pre>
</blockquote>
<p>Note that <a href="https://docs.python.org/3/library/struct.html#struct.unpack" rel="nofollow noreferrer">[Python 3]: struct.<strong>unpack</strong>(<em>format, buffer</em>)</a> returns a <em>tuple</em>.</p>
</div>
<div class="post-text" itemprop="text">
<p>The value which printed is actually the answer you expected, written in long-form notation instead of scientific notation. There is not a problem here.</p>
<p>See <a href="https://en.wikipedia.org/wiki/Scientific_notation" rel="nofollow noreferrer">this</a> for more on scientific notation.</p>
</div>
<span class="comment-copy">The result is not wrong, is just displayed in a different format. What you are looking for is called scientific notation.</span>
