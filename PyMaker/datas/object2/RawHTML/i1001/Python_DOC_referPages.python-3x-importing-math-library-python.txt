<div class="post-text" itemprop="text">
<p><code>math.log2</code> was introduced in <a href="https://docs.python.org/3.3/library/math.html#math.log2" rel="nofollow noreferrer">Python 3.3</a>. You are probably using an earlier version.</p>
<p>In those earlier versions, you can use</p>
<pre><code>math.log(num, 2)
</code></pre>
<p>instead.</p>
</div>
<div class="post-text" itemprop="text">
<p>You have a file called <code>math.py</code> in the same directory as the code you're trying to run. Delete or rename it.</p>
<p>When you <code>import math</code>, Python walks through the directories in <a href="https://docs.python.org/3/library/sys.html#sys.path" rel="nofollow noreferrer"><code>sys.path</code></a> and imports the first file called <code>math.py</code> (or a directory called <code>math</code> with an <code>__init__.py</code> file inside) that it sees. The first entry in <code>sys.path</code> is the current directory, so it sees your <code>math.py</code> first.</p>
<p>See the documentation for <a href="https://docs.python.org/3/tutorial/modules.html" rel="nofollow noreferrer">modules</a> or for <a href="https://docs.python.org/3/reference/import.html" rel="nofollow noreferrer">the <code>import</code> statement</a>. </p>
</div>
