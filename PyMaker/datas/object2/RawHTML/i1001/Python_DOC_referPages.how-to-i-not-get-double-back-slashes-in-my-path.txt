<p>Differentiate between "display in your ide /debugging session" and "what is in the string".
You might want to quit using python2 string formating and get raw-strings vs. normal strings straight:</p>
<pre><code>database       = "E:\\Stock Database\\Historical Data\\Historical Stock List\\"
# you need 1 space at the end, else \" is treated as escaped "
database_raw   = r"E:\Stock Database\Historical Data\Historical Stock List\ ".-rstrip()

with_file      = f"{database}\\stockfactsheet.csv"
with_file_raw  = fr"{database}\stockfactsheet.csv" # no escapes needed

print(with_file)
print(with_file_raw)
</code></pre>
<p>Output:</p>
<pre><code>database       = "E:\\Stock Database\\Historical Data\\Historical Stock List\\"
