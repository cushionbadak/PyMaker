<div class="post-text" itemprop="text">
<p>I'm trying to modify some Python 2 code for Python 3.  The code imports <code>DictMixin</code>.  Is there a way to import this into Python 3?</p>
</div>
<div class="post-text" itemprop="text">
<p>The <code>DictMixin</code> class is no longer available in Python3.</p>
<p>The <a href="https://docs.python.org/2/library/userdict.html?highlight=dictmixin#UserDict.DictMixin" rel="nofollow noreferrer">recommended solution</a> is to subclass <a href="https://docs.python.org/3/library/collections.abc.html#collections.abc.MutableMapping" rel="nofollow noreferrer">collections.abc.MutableMapping</a> instead.</p>
<p>As with the <code>DictMixin</code> class, your class will need to implement some methods itself for the subclassing to work.</p>
<pre><code>&gt;&gt;&gt; class C(MutableMapping, object):pass
... 
&gt;&gt;&gt; c = C()
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: Can't instantiate abstract class C with abstract methods __delitem__, __getitem__, __iter__, __len__, __setitem__
</code></pre>
</div>
