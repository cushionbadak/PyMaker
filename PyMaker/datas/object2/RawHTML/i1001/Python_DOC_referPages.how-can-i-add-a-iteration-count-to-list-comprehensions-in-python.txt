<div class="post-text" itemprop="text">
<p>Comprehensions take an <em>expression</em> as its first element; <code>count += 1</code> is a <em>statement</em>.</p>
<p>Instead, use <code>enumerate</code>, which takes a generator producing <code>x</code> and returns a generator producing <code>(i, x)</code> where <code>i</code> is the index:</p>
<pre><code>'-'.join(char.upper() + char.lower() * i for i, char in enumerate(s))
</code></pre>
</div>
