<p>Assume I have the following list: </p>
<pre><code>list = [["A",0,"C"],["B",1,"C"]]
</code></pre>
<p>I want to do something that accomplishes the following: Set the values in the 1st column equal to "D" where the value in the 2nd column is equal to 0.</p>
<p>So the list after the update should look like this:</p>
<pre><code>list = [["D",0,"C"],["B",1,"C"]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Naming a variable <code>list</code> is not recommended, since it will shadow the builtin function <a href="https://docs.python.org/3/library/functions.html#func-list" rel="nofollow noreferrer"><code>list()</code></a>.  </p>
<p>If you wish to do this in place, use indices:</p>
<pre><code>lst = [["A", 0 ,"C"],["B", 1 ,"C"]]

for i in range(len(lst)):
    if lst[i][1] == 0:
        lst[i][0] = "D"

print(lst)
# [['D', 0, 'C'], ['B', 1, 'C']]
</code></pre>
<p>Otherwise, you can rebuild the list with a list <em>comprehension</em>:</p>
<pre><code>print([["D", y, z] if y == 0 else [x, y ,z] for x, y, z in lst])
# [['D', 0, 'C'], ['B', 1, 'C']]
</code></pre>
<p>You can also do it this way without unpacking:</p>
<pre><code>print([["D"] + sublst[1:] if sublst[1] == 0 else sublst for sublst in lst])
# [['D', 0, 'C'], ['B', 1, 'C']]
</code></pre>
<p>Theres also nothing stopping you from doing it normally:</p>
<pre><code>newlst = []
for sublst in lst:
    if sublst[1] == 0:
        newlst.append(["D"] + sublst[1:])
        continue
    newlst.append(sublst)

print(newlst)
# [['D', 0, 'C'], ['B', 1, 'C']]
</code></pre>
<p>The list of possibilities goes on(no pun intended).</p>
<p>Also in Python <code>bool(0)</code> evaluates to <code>False</code>, so you <em>can</em> use <code>not</code> here instead of explicitly checking <code>== 0</code>. You can decide if this is worth using, since empty types and sequences such as <code>""</code>, <code>[]</code>, <code>()</code>, <code>{}</code> etc. will also evaluate to <code>False</code> when testing for truthiness. </p>
