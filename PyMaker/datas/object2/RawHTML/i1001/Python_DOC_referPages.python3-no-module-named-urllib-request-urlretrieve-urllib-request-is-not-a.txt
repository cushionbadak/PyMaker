<div class="post-text" itemprop="text">
<p>Your import is wrong.</p>
<p>Correct import looks like:</p>
<pre><code>from urllib.request import urlretrieve
def download():
    urlretrieve("url")
download()
</code></pre>
<p>Or:</p>
<pre><code>import urllib.request
def download():
    urllib.request.urlretrieve("url")
download()
</code></pre>
<p>Take a look at the import Reference: <a href="https://docs.python.org/3/reference/import.html" rel="nofollow noreferrer">The import system</a>:</p>
