<p>I have a list <code>m</code>:</p>
<pre><code>m = ['ABC', 'XYZ', 'LMN']
</code></pre>
<p>and I want output as follows:</p>
<pre><code>m = [['a','x','l']
     ['b','y','m']
     ['c','z','n']]
</code></pre>
<p>How can this be done?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>list(zip(*..))</code> to transpose the nested list, and use list comprehension to create nested list:</p>
<pre><code>print(list(zip(*[list(i.lower()) for i in m])))
</code></pre>
<p>Output:</p>
<pre><code>[('a', 'x', 'l'), ('b', 'y', 'm'), ('c', 'z', 'n')]
</code></pre>
<p>If want sub-values to be lists:</p>
<pre><code>print(list(map(list,zip(*[list(i.lower()) for i in m]))))
</code></pre>
<p>Output:</p>
<pre><code>[['a', 'x', 'l'], ['b', 'y', 'm'], ['c', 'z', 'n']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>All you need is a list-comprehension, <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> to transpose and <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map</code></a> to convert to lower.</p>
<pre><code>   &gt; m=['ABC','XYZ','LMN']
   &gt; [list(map(str.lower, sub)) for sub in zip(*m)]
   [['a', 'x', 'l'], ['b', 'y', 'm'], ['c', 'z', 'n']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>m=['ABC','XYZ','LMN']
import numpy as np
new_list = [[0, 0, 0], [0, 0,0],[0,0,0]]

j = 0

for i in m:
    a = list(i.lower())
    print(a)
    new_list[j] = a
    j = j+1
np.transpose(new_list).tolist()
</code></pre>
<p>Output:</p>
<pre><code>[['a', 'x', 'l'], ['b', 'y', 'm'], ['c', 'z', 'n']]
</code></pre>
