<div class="post-text" itemprop="text">
<p>You're using <code>re.match</code>, which sees if the text matches the pattern at the beginning of the string. Use <code>re.search</code> instead, which looks for matches anywhere in the string. See <a href="https://docs.python.org/3/library/re.html#search-vs-match" rel="nofollow noreferrer">here</a> for more info.</p>
<h3>Code:</h3>
<pre><code>import re

text = "Publisher: Broadway Books; Anniversary, Reprint edition (October 8, 2002)"
pattern = re.compile(
    "(Jan(uary)?|Feb(ruary)?|Mar(ch)?|Apr(il)?|May|Jun(e)?|"
    "Jul(y)?|Aug(ust)?|Sep(tember)?|Oct(ober)?|Nov(ember)?|"
    "Dec(ember)?)\s+\d{1,2},\s+\d{4}")

print(pattern.match(text))  # prints None
print(pattern.search(text))
print(pattern.search(text).group())
</code></pre>
<h3>Results:</h3>
<pre><code>None
&lt;_sre.SRE_Match object; span=(57, 72), match='October 8, 2002'&gt;
October 8, 2002
</code></pre>
