<div class="post-text" itemprop="text">
<p>You can use <a href="https://stackoverflow.com/a/312464/9209546">this chunking recipe</a>:</p>
<pre><code>def chunks(L, n):
    """Yield successive n-sized chunks from L."""
    for i in range(0, len(L), n):
        yield L[i:i + n]
</code></pre>
<p>Then iterate with <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>str.join</code></a> or use sequence unpacking with the <code>sep</code> argument of <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>print</code></a>:</p>
<pre><code>for i in chunks(mylist, 2):
    print(' '.join(i))
    # alternatively, unpack and use sep argument:
    # print(*i, sep=' ')

# hello my
# name is
# frank and
# it is
# great to
# be here
# in this
# great country
</code></pre>
