<div class="post-text" itemprop="text">
<p>An example of the python string is <code>'8:30 AM- 10:00 PM Subject: Math'</code>. In general, the string contains a start time, end time, and the subject I want to separate this string into 3 components: the start time, end time, and subject. For example <code>8:30 AM</code>, <code>10:00 PM</code>, and <code>Subject: Math</code>.</p>
<p>How can I do this using regex in python?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow"><code>re.split()</code></a> using a <em>positive lookbehind</em> to <code>AM</code> or <code>PM</code> having an optional <code>-</code> and a space character as a delimiter:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt;
&gt;&gt;&gt; s = "8:30 AM- 10:00 PM Subject: Math"
&gt;&gt;&gt; re.split(r"(?&lt;=AM|PM)-?\s", s)
['8:30 AM', '10:00 PM', 'Subject: Math']
</code></pre>
</div>
