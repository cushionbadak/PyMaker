<div class="post-text" itemprop="text">
<p>Original cat looks like this:</p>
<pre><code>cat = ['a','a,b,c','c,d,e,f']
</code></pre>
<p>I want to convert it to:</p>
<pre><code>cat = [['a'],['a','b','c'],['c','d','e','f']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can achieve this by <code>list comprehension</code> as:</p>
<pre><code>&gt;&gt;&gt; cat = ['a','a,b,c','c,d,e,f']
&gt;&gt;&gt; [c.split(',') for c in cat]
[['a'], ['a', 'b', 'c'], ['c', 'd', 'e', 'f']]
</code></pre>
<p><strong>Alternatively</strong>, you may also use <code>lambda</code> function with <code>map</code> to achieve this:</p>
<pre><code>&gt;&gt;&gt; map(lambda x: x.split(','), cat)
[['a'], ['a', 'b', 'c'], ['c', 'd', 'e', 'f']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You just need to <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow">split</a> each string:</p>
<pre><code>result = [s.split(',') for s in cat]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try This:</p>
<pre><code>cat = ['a','a,b,c','c,d,e,f']
newlist = [a.split(',') for a in cat]
print(newlist)
&gt;&gt;&gt; [['a'], ['a', 'b', 'c'], ['c', 'd', 'e', 'f']]
</code></pre>
</div>
