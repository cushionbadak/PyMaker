<div class="post-text" itemprop="text">
<p>As mentioned, you can use the <a href="https://devdocs.io/python~2.7/library/stdtypes#set.issubset" rel="nofollow"><code>set.issubset</code></a> call:</p>
<pre><code>&gt;&gt;&gt; a = [1,2,3]
&gt;&gt;&gt; b = [1,3]
&gt;&gt;&gt; c = [1,2,3,4]
&gt;&gt;&gt; set(a).issubset(b)
False
&gt;&gt;&gt; set(a).issubset(c)
True
&gt;&gt;&gt; set(b).issubset(c)
</code></pre>
<p>Alternatively, for a strict subset match, you can also use the <em>less than</em> operator:</p>
<pre><code>&gt;&gt;&gt; set(a) &lt; set(c)
True
</code></pre>
