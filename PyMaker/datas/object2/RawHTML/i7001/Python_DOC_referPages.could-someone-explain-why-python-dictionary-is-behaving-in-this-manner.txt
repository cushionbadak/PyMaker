<div class="post-text" itemprop="text">
<p>You are using the wrong operator to do boolean testing. <code>&amp;</code> is the <a href="https://docs.python.org/3/reference/expressions.html#binary-bitwise-operations" rel="nofollow noreferrer">binary bitwise operator</a>, not the <a href="https://docs.python.org/3/reference/expressions.html#boolean-operations" rel="nofollow noreferrer">boolean logic operator</a>. Because it has a <a href="https://docs.python.org/3/reference/expressions.html#operator-precedence" rel="nofollow noreferrer">different operator precedence</a> you are really calculating something else:</p>
<pre><code>x &gt; 1991 &amp; x &lt;= 2000
</code></pre>
<p>is interpreted as</p>
<pre><code>x &gt; (1991 &amp; x) &lt;= 2000
</code></pre>
<p>For 16 of your years that'll be true, including 2001 through to 2007.</p>
<p>Use <code>and</code> instead:</p>
<pre><code>x &gt; 1991 and x &lt;= 2000
</code></pre>
<p>or use comparison chaining:</p>
<pre><code>1991 &lt; x &lt;= 2000
</code></pre>
<p>Put together, simplifying the last test to <code>else</code>:</p>
<pre><code>for x in range(1985, 2008):
    if x &lt;= 1991:
        test_dict[x] = file_0
    elif 1991 &lt; x &lt;= 2000:
        test_dict[x] = file_1
    elif 2000 &lt; x &lt;= 2005:
        test_dict[x] = file_2
    else:
        test_dict[x] = file_3
</code></pre>
