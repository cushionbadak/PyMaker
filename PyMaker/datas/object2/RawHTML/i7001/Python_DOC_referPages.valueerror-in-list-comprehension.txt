<p>I'm calculating distance from center by</p>
<p><code>dist = [sqrt(p*p + q*q) for p, q in (x, y)]</code></p>
<p>But, getting <code>ValueError: too many values to unpack (expected 2)</code>.</p>
<p>x &amp; y are lists with x, y coordinate values.</p>
</div>
<div class="post-text" itemprop="text">
<p>Try using <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a>:</p>
<pre><code>dist = [sqrt(p*p + q*q) for p, q in zip(x, y)]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map()</code></a> is a good fit for this:</p>
<pre><code>&gt;&gt;&gt; list(map(lambda p,q:sqrt(p*p + q*q), x, y))
[4.123105625617661, 5.385164807134504, 6.708203932499369]
</code></pre>
</div>
