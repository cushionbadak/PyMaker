<h2>install plugin</h2>
<p>then install this in VSCode
<a href="https://marketplace.visualstudio.com/items?itemName=donjayamanne.python" rel="noreferrer">https://marketplace.visualstudio.com/items?itemName=donjayamanne.python</a><br/>
and config</p>
<h2>settings</h2>
<blockquote>
<p>./.vscode/settings.json  </p>
</blockquote>
<pre><code>{
    "python.envFile": "${workspaceRoot}/.env",
    "python.pythonPath": "${workspaceRoot}/.env/bin/python",
    "python.linting.flake8Enabled": true,
    "python.linting.pylintEnabled": false
}
</code></pre>
<blockquote>
<p>./.vscode/launch.json</p>
</blockquote>
<pre><code>{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Python",
            "type": "python",
            "request": "launch",
            "stopOnEntry": false,
            "pythonPath": "${config:python.pythonPath}",
            "program": "${file}",
            "cwd": "${workspaceRoot}",
            "env": {},
            "envFile": "${workspaceRoot}/.env",
            "debugOptions": [
                "WaitOnAbnormalExit",
                "WaitOnNormalExit",
                "RedirectOutput"
            ]
        }
    ]
}
</code></pre>
<h2>OMG, this is Python 3 only!</h2>
<blockquote>
<p><a href="https://pypi.python.org/pypi/flake8-mypy/17.3.3" rel="noreferrer">https://pypi.python.org/pypi/flake8-mypy/17.3.3</a> </p>
<p>Yes, so is mypy. Relax, you can run Flake8 with all popular plugins<br/>
  as a tool perfectly fine under Python 3.5+ even if you want to analyze<br/>
  Python 2 code. This way you’ll be able to parse all of the new syntax<br/>
  supported on Python 3 but also effectively all the Python 2 syntax at<br/>
  the same time.  </p>
<p>By making the code exclusively Python 3.5+, I’m able to focus on the<br/>
  quality of the checks and re-use all the nice features of the new<br/>
  releases (check out pathlib) instead of wasting cycles on Unicode<br/>
  compatibility, etc.  </p>
</blockquote>
<h2>IDE &amp; Linter Integrations</h2>
<blockquote>
<p><a href="https://github.com/python/mypy#ide--linter-integrations" rel="noreferrer">https://github.com/python/mypy#ide--linter-integrations</a></p>
<p>IDE &amp; Linter Integrations</p>
<p>Mypy can be integrated into popular IDEs:</p>
<ul>
<li>Vim: vim-mypy</li>
<li>Emacs: using Flycheck and Flycheck-mypy</li>
<li>Sublime Text: SublimeLinter-contrib-mypy</li>
<li>Atom: linter-mypy</li>
<li>PyCharm: PyCharm integrates its own implementation of PEP 484.</li>
</ul>
<p>Mypy can also be integrated into Flake8 using flake8-mypy.  </p>
</blockquote>
