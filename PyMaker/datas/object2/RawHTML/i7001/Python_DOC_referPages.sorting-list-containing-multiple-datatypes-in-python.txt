<div class="post-text" itemprop="text">
<p>use <code>sorted(your_list, key=str)</code> for simple cases when you can sort by the string representation of the item</p>
<pre><code>l = ["Tak","jil","son",54,84]
sorted(l, key=str)
</code></pre>
<p>output: <code>[54, 84, 'Tak', 'jil', 'son']</code></p>
</div>
<div class="post-text" itemprop="text">
<p><code>sorted</code> takes three arguments: an iterable, a key and reverse i.e <code>sorted(iterable, [key=None], [reverse=False])</code>. The <code>key</code> argument allows you define a function which can be called to compare the values of items in the iterable. Some examples are given below:</p>
<pre><code># notice the function name is supplied without brackets
sorted(["Eva", "eva", "string", "Aaron"], key=str.lower)

# the same result using lambda
sorted(["Eva", "eva", "string", "Aaron"], key=lambda x: x.lower())

# comparing by the second element in a tuple: (name, age)
name_ages = [("Eva", 12), ("Tom", 14), ("Gerald", 4)]
# output is [('Gerald', 4), ('Eva', 12), ('Tom', 14)]
sorted(name_ages, key=lambda x: x[1])  
</code></pre>
<p>Using <code>lambdas</code> or by defining functions of your own, the sorted function is very versatile. </p>
<p><strong>NB:</strong> Use can also use the <code>key</code> argument in <code>list.sort()</code></p>
