<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#dict.get" rel="nofollow noreferrer"><code>dict.get()</code></a> to get 'a' if it is given in the keyword args. And if it's not present, the default return value is <code>''</code>:</p>
<pre><code>b = kwargs.get('a', '')
if b:
    b = '//{}'.format(b)
</code></pre>
<p>The first assignment gets <code>'a'</code> from the dict and assigns it to <code>b</code>. Then if <code>b</code> is non-empty (the <code>''</code>), then reformat with the <code>'//'</code>.</p>
<p>Edit: I assume is that if 'a' is not provided in kwargs but is empty (<code>''</code>), then <code>b</code> will also be empty and not just '//'. In your current code, if <code>'a'</code> is provided and is empty, then <code>b</code> becomes <code>'//'</code>. If you want empty or missing <code>a</code>'s to both result in the empty string for <code>b</code>, then use my version. Otherwise, your version is fine - but change to <code>... if 'a' not in kwargs else ...</code> as MartinP put in the comments.</p>
