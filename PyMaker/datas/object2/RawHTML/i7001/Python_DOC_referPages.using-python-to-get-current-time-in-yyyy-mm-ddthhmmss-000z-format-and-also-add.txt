<p>I'm using python and I need to get the time in a certain format that this API uses.  So I need to get current time in the format of yyyy-mm-ddThh:mm:ss.000Z and then also create a new date in the future by adding a number of days to it.  So if I could get the current time of 2017-08-19T07:00:00.000Z and add 30 days to it to get 2017-09-18T07:00:00.000Z.  What would be the easiest way to write this in python?</p>
<p>Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>Current date</p>
<pre><code>import datetime
now = datetime.datetime.now()
format_iso_now = now.isoformat()
</code></pre>
<p>Add 10 days</p>
<pre><code>from datetime import timedelta
later = now + timedelta(days=10)
format_later_iso = later.isoformat()
</code></pre>
<p>Output</p>
<pre><code>print(format_iso_now, format_later_iso)

2017-08-20T02:43:07.177167 2017-08-30T02:43:07.177167
</code></pre>
<p>And to match your needs</p>
<pre><code>print(now.strftime('%Y-%m-%d %H:%M:%S:%fZ'))

2017-08-20 02:48:00:103856Z
</code></pre>
<p>More details on <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer">documentation</a></p>
