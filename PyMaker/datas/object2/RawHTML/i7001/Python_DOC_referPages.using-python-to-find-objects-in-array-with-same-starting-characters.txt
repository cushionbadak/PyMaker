<p>I'm new to python and would like to know if there is an easy way to search for Strings in array that have the same starting characters.</p>
<p>for example I have a list</p>
<pre><code>ex = [exA, exB, teA, exC]
</code></pre>
<p>and want to get result for everything matching the first two characters
something like this:<br/>
 {'ex' : 3, 'te' : 1}</p>
<p>I have tried working with the Counter method from collections but I cant get a result as shown above.</p>
<p>thank you in advanced </p>
</div>
<div class="post-text" itemprop="text">
<p>If you slice off the first two characters of each element you can use <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="noreferrer"><code>collections.Counter</code></a> for this</p>
<pre><code>&gt;&gt;&gt; import collections
&gt;&gt;&gt; ex = ['exA', 'exB', 'teA', 'exC']
&gt;&gt;&gt; collections.Counter(i[:2] for i in ex)
Counter({'ex': 3, 'te': 1})
</code></pre>
</div>
