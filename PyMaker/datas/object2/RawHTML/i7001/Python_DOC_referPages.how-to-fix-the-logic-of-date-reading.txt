<div class="post-text" itemprop="text">
<p>You can access the last accessed time by <code>os.path.getatime()</code>.</p>
<p>From <a href="https://docs.python.org/3/library/os.path.html#module-os.path" rel="nofollow noreferrer">Python docs</a>:</p>
<blockquote>
<p><code>os.path.getatime(path)</code> <br/>
  Return the time of last access of path. The return value is a number giving the number of seconds since the epoch (see the <code>time</code> module). Raise <code>OSError</code> if the file does not exist or is inaccessible.</p>
</blockquote>
<h3>Sample Usage</h3>
<pre><code>os.path.getatime('./test.txt')
</code></pre>
<p>Output</p>
<pre><code>1503332001.0
</code></pre>
