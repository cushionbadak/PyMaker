<p>I am stuck at a problem.</p>
<p>I have a List L = [1,2,3,4,5,6,7,8,9]</p>
<p>I want to find the sum of elements in lists such that, if the sum exceeds's 25. I want to get the previous sum.</p>
<p>for eg. 1+2+3+4+5+6+7=28, 28&gt;25. Therefore i want a final variable sum as 21.</p>
<p>i am doing it using <strong>for-if combination</strong>, but i don't want to use <strong>if</strong> . Is there a better way of doing this? below is my code.</p>
<pre><code>L = [1,2,3,4,5,6,7,8,9]
summ = 0

for val in L:
    summ+= val
    if summ &gt; 25:
        summ-=val
</code></pre>

<div class="post-text" itemprop="text">
<p>I think what bothers you is the <code>summ-=val</code> part, not the <em>if</em>. And there is a one line solution with <code>reduce</code>.</p>
<pre><code>&gt;&gt;&gt; from functools import reduce
&gt;&gt;&gt; L = [1,2,3,4,5,6,7,8,9]
&gt;&gt;&gt; reduce(lambda x, y: x + y if x + y &lt;= 25 else x, L)
21
</code></pre>

<p>You can use a <code>while</code> loop:</p>
<pre><code>L = [1,2,3,4,5,6,7,8,9]
i = 0
s = 0
while i &lt; len(L) and s + L[i] &lt; 25:
    s += L[i]
    i += 1
</code></pre>

<div class="post-text" itemprop="text">
<p>Using pandas, take the cumulative sum on a series consisting of the numbers in the list.  Assign the value <code>NaN</code> to any any cumulative sum over <code>max_value</code>.  Fill forward the result and convert back to a list of integers.</p>
<pre><code>import pandas as pd

max_val = 25
a = range(1, 10)
s = pd.Series(a).cumsum()
s.loc[s &gt; max_val] = np.nan
&gt;&gt;&gt; s.ffill().astype(int).tolist()
[1, 3, 6, 10, 15, 21, 21, 21, 21]
</code></pre>
