<div class="post-text" itemprop="text">
<p>I have a .txt textfile with a list formatted like 
<code>[' ok ', ' test ', ' help ']</code>
How can I read/import this textfile list into a python list?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#eval" rel="nofollow noreferrer"><code>eval()</code></a>:</p>
<pre><code>with open("file.txt", "r") as file:
    result = eval(file.read())
</code></pre>
<p><strong>However</strong>, <code>eval()</code> is usually not recommended, since it will evaluate any string you feed to it. That includes arbitrary code a malicious user might have put in the file. To avoid that, you should use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow noreferrer"><code>literal_eval()</code></a> instead:</p>
<pre><code>from ast import literal_eval

with open("file.txt", "r") as file:
    result = literal_eval(file.read())
</code></pre>
<p>Unlike the regular <code>eval()</code> function, <code>literal_eval()</code> only evaluates strings that convert to basic python types (this includes lists and dictionaries, which  are what this is most commonly used for).</p>
</div>
