<pre><code>name = request.form['stripeBillingName']
</code></pre>
<p>How do I print the value of the variable <code>name</code> in a readable encoding?

<div class="post-text" itemprop="text">
<p>In source header you can declare </p>
<pre><code>#!/usr/bin/env python
# -*- coding: utf-8 -*-
....
</code></pre>
<p>After that you can use utf-8, which should give you your desirable format</p>
<pre><code>name = request.form['stripeBillingName']
nameDec = name.decode('utf8')
print nameDec
</code></pre>
<p>You can also encode it however you want e.g:</p>
<pre><code>nameEnc = nameDec.encode('cp1250')
</code></pre>
