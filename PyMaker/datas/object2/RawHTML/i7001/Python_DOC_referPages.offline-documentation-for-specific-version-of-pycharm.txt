<div class="post-text" itemprop="text">
<p>Often I need to work offline for my python projects. How can I download PyCharm documentation for offline use?</p>
</div>
<div class="post-text" itemprop="text">
<p>There's an option in PyCharm: go "Settings/Tools/Python External Documentation";  <strong>BUT</strong> it seems thishas not been working with local files for a long time due to a  <code>PyCharm</code> bug.</p>
<p>You can start a local server and pretend it to be a <strong>docs.python.org</strong> site.</p>
<ol>
<li><p>Get docs <a href="https://docs.python.org/3/archives/python-3.6.3-docs-html.zip" rel="nofollow noreferrer">archive</a></p></li>
<li><p>Unpack archive to folder named exactly "3.6"</p></li>
<li><p>Edit your <strong>hosts</strong> file:</p></li>
</ol>
<p><em>C:\Windows\System32\drivers\etc\hosts</em> or <em>/etc/hosts</em></p>
<pre><code>127.0.0.1       docs.python.org
::1             docs.python.org
</code></pre>
<ol start="4">
<li><p>Reboot</p></li>
<li><p>Run server. Go to the dir <em>where</em> "3.6" folder is located and run:</p></li>
</ol>
<p><code>python -m http.server 80 --bind 127.0.0.1</code></p>
<p>for linux use
<code>sudo python -m http.server 80 --bind 127.0.0.1</code></p>
</div>
