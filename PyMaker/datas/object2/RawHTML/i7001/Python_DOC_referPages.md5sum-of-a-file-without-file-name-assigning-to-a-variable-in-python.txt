<div class="post-text" itemprop="text">
<p>Rather than shelling out to perform the md5sum, use Python's inbuilt <a href="https://docs.python.org/3/library/hashlib.html#module-hashlib" rel="nofollow noreferrer"><code>hashlib.md5</code></a> implementation:</p>
<pre><code>import hashlib

with open(filename, 'rb') as f:
    hexdigest = hashlib.md5(f.read()).hexdigest()
    print(hexdigest)
</code></pre>
