<div class="post-text" itemprop="text">
<p>Backslash U (<code>\U</code>) has a special meaning in string literals. <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow noreferrer">String and Bytes literals</a> in the documentation says for <code>\Uxxxxxxxx</code> the meaning is <em>„Character with 32-bit hex value xxxxxxxx“</em>.</p>
<p>So you have to escape at least the backslash before the U of Users, or put an <code>r</code> in front of the string so no backslash has a special meaning.</p>
