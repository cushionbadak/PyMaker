<p>I want to know how to format a dictionary so it looks better when it's printed. This is what I get: </p>
<pre><code>{'park': 'Animal Kingdom', 'section': 'DinoLand U.S.A.', 'Attraction': 'The Boneyard', 'Height': 'Any Height', 'Categories': 'Play Areas'}
</code></pre>
<p>I want it to look like this: </p>
<pre><code>park: Animal Kingdom
section: DinoLand U.S.A.
Attraction: The Boneyard
Height: Any Height
Categories: Play Areas
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/pprint.html" rel="nofollow noreferrer"><code>pprint</code></a>(“pretty-print”) is exatly what you are looking for:</p>
<pre><code>&gt;&gt;&gt; from pprint import pprint 
&gt;&gt;&gt; d = {'park': 'Animal Kingdom', 'section': 'DinoLand U.S.A.', 'Attraction': 'The Boneyard', 'Height': 'Any Height', 'Categories': 'Play Areas'}
&gt;&gt;&gt; pprint(d)
{'Attraction': 'The Boneyard',
 'Categories': 'Play Areas',
 'Height': 'Any Height',
 'park': 'Animal Kingdom',
 'section': 'DinoLand U.S.A.'}
</code></pre>
<p>Alternatively, you may use a simple loop:</p>
<pre><code>&gt;&gt;&gt; for key, val in d.items() : print("%s : %s" %( key, val))
...
park : Animal Kingdom
section : DinoLand U.S.A.
Attraction : The Boneyard
Height : Any Height
Categories : Play Areas
</code></pre>
</div>

<p>You just need to loop through all the elements and print them one at a time:</p>
<pre><code>mydict = {'park': 'Animal Kingdom', 'section': 'DinoLand U.S.A.', 'Attraction': 'The Boneyard', 'Height': 'Any Height', 'Categories': 'Play Areas'}

for key in mydict:
    print(key,":",mydict[key])
</code></pre>

<div class="post-text" itemprop="text">
<p>Iterate over all of the items in the dictionary and format the key and value in a string. The for loop below should work for what you are trying to do.</p>
<pre><code>for key, value in my_dict.items():
    print(f"{key}: {value}")
</code></pre>
