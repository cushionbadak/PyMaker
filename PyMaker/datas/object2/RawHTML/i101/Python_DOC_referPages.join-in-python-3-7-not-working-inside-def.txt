<div class="post-text" itemprop="text">
<p>When you call <code>join</code> over <code>str(a)</code>, you are first converting the list to a string, and then joining over every character in the stringified list.</p>
<p>What you want to do instead is convert <em>each element</em> in <code>a</code> to a string. You should use a <a href="https://stackoverflow.com/questions/47789/generator-expressions-vs-list-comprehension">generator expression</a>, like this:</p>
<pre><code>''.join(str(x) for x in a)
</code></pre>
<p>Alternatively, you can use the built-in function <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map(function, iterable)</code></a>, which is useful in the general case when you want to return a new iterable containing the results of <code>function</code> applied to every element in <code>iterable</code>. So you would use:</p>
<pre><code>''.join(map(str, a))
</code></pre>
<p>It's basically a matter of taste as to which you find more readable.</p>
