<pre><code>numbers = [8, 3, 1, 2, 5, 4, 7, 6]
group = [2, 3, 5, 7]

def sort_priority(values, group):
    sorted = 'False'
    def helper(x):
        if x in group:

<div class="post-text" itemprop="text">
<p>If you're using Python 3, you can achieve this with the <a href="https://docs.python.org/3/reference/simple_stmts.html#nonlocal" rel="nofollow noreferrer">nonlocal</a> statement:</p>
<pre><code>def sort_priority(values, group):
    sorted = 'False'
    def helper(x):
        nonlocal sorted
        if x in group:
            sorted = 'True'
            return (0, x)
        else:
            return (1, x)    
    values.sort(key=helper)
    return sorted
</code></pre>
<p>This allows the <code>sorted</code> variable in the outer scope to be modified by the inner scope of the helper function - so <code>sort_priority</code> will then return <code>"True"</code>.</p>
<p>If you're using Python 2, the <code>nonlocal</code> statement isn't available, but something similar can be achieved like this:</p>
<pre><code>def sort_priority(values, group):
    nonlocals = {'sorted': 'False'}
    def helper(x):
        if x in group:
            nonlocals['sorted'] = 'True'
            return (0, x)
        else:
            return (1, x)    
    values.sort(key=helper)
    return nonlocals['sorted']
</code></pre>
