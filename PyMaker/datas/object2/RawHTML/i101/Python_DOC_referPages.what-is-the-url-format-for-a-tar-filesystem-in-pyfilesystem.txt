<p>Found the answer through trial and error:</p>
<pre><code>fs.open_fs('tar://~/tmp/testtar.tar.gz')
</code></pre>
<p>The resource part of the URL (<code>~/tmp/testtar.tar.gz</code>) can be an OS path including file expansion.</p>
