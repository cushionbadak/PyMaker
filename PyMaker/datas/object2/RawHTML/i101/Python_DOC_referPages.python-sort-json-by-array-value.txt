<div class="post-text" itemprop="text">
<p>You can sort a list. So you can sort the list that contains your dictionaries using one value or more that are contained in the dictionary itself as key, but you cannot sort a dictionary. Dictionaries don't work like that.</p>
<p><a href="https://docs.python.org/3/library/stdtypes.html#dict" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html#dict</a></p>
