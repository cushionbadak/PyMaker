<div class="post-text" itemprop="text">
<p>I was wondering if there would be a way to sort a list in order according to the increasing floats as values in a dictionary. I don't really know how to explain it so I'll demonstrate what I mean</p>
<pre><code>[3, 6, 4]
</code></pre>
<p>to</p>
<pre><code>[6, 4, 3]
</code></pre>
<p>with dictionary like this</p>
<pre><code>{6: 15.6, 3: 120.0, 4: 17.3}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use the <code>key</code> argument of <a href="https://docs.python.org/3/library/functions.html#sorted" rel="noreferrer">sorted</a>:</p>
<pre><code>&gt;&gt;&gt; mylist = [3, 6, 4]
&gt;&gt;&gt; mydict = {6:15.6, 3:120.0, 4:17.3}
&gt;&gt;&gt; sorted(mylist, key=mydict.get)
[6, 4, 3]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>For your example, you don't even need that list as the <code>dict</code> keys are sufficient:</p>
<pre><code>&gt;&gt;&gt; my_dict = {6: 15.6, 3: 120.0, 4: 17.3}
&gt;&gt;&gt; sorted(my_dict, key=my_dict.get)
[6, 4, 3]
</code></pre>
</div>
