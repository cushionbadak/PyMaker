<div class="post-text" itemprop="text">
<p>How do you stop the output from subprocess.Popen from being output?  Printing can sometimes be slow if there is a great deal of it.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you want to totally throw it away:</p>
<pre><code>import subprocess
import os
with open(os.devnull, 'w') as fp:
    cmd = subprocess.Popen(("[command]",), stdout=fp)
</code></pre>
<p>If you are using Python 2.5, you will need <code>from __future__ import with_statement</code>, or just don't use <code>with</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>In Python 3.3+ you could use <a href="http://docs.python.org/3/library/subprocess.html#subprocess.DEVNULL"><code>subprocess.DEVNULL</code></a>, to suppress the output:</p>
<pre><code>from subprocess import DEVNULL, STDOUT, check_call

check_call([cmd, arg1, arg2], stdout=DEVNULL, stderr=STDOUT)
</code></pre>
<p>Remove <code>stderr=STDOUT</code> if you don't want to suppress <code>stderr</code> also.</p>
</div>
