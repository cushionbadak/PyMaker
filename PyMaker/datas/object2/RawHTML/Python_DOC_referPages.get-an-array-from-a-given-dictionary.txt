<div class="post-text" itemprop="text">
<p>Iterate over the dictionary values and store the first value (at index 0) in a temporary list.</p>
<pre><code>dict1 = {'car': [10000, 'Income'],
    'project': [456, 'Income'],
    'drone': [345, 'Income'],
    'laptop': [678, 'Income']}

temp_list=[]
for x in dict1.values():
    temp_list.append(x[0])

print(temp_list)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As a one liner:</p>
<pre><code>money = [x[0] for x in dict1.values()]
</code></pre>
