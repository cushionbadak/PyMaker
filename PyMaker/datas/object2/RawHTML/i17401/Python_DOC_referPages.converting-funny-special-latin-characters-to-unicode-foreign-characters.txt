<div class="post-text" itemprop="text">
<p>You need to give BeautifulSoup the right codec to use, because otherwise make an educated guess and get it wrong (some of the time).</p>
<p>If you are using <code>urllib2</code> to load the page, you can pass along any encoding the server set with:</p>
<pre><code>soup = BeautifulSoup(response.read(),
                     from_encoding =response.info().getparam('charset'))
</code></pre>
<p>See the <a href="http://www.crummy.com/software/BeautifulSoup/bs4/doc/#encodings" rel="nofollow">encodings section</a> of the BeautifulSoup documentation.</p>
</div>
<div class="post-text" itemprop="text">
<p>According to the web site <a href="http://docs.python.org/2/library/codecs.html#standard-encodings" rel="nofollow">Standard Encodings</a> </p>
<blockquote>
<p>cp424 EBCDIC-CP-HE, IBM424    Hebrew<br/>
  cp856                     Hebrew<br/>
  cp862         862, IBM862 Hebrew<br/>
  cp1255    windows-1255    Hebrew 
  iso8859_8 iso-8859-8, hebrew  Hebrew</p>
</blockquote>
