<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/2/library/re.html#re.sub" rel="noreferrer">re.sub()</a>:</p>
<pre><code>import re
newName = re.sub('[\\\\/:*?"&lt;&gt;|]', 'special char', name)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/3/library/stdtypes.html#str.translate" rel="noreferrer"><code>str.translate</code></a> and a <a href="http://www.python.org/dev/peps/pep-0274/" rel="noreferrer">dictionary comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; name = ":1?2/3&lt;4|5"
&gt;&gt;&gt; bad = ['\\', '/', ':', '*', '?', '"', '&lt;', '&gt;', '|']
&gt;&gt;&gt; newName = name.translate({ord(c):'special char' for c in bad})
&gt;&gt;&gt; newName
'special char1special char2special char3special char4special char5'
&gt;&gt;&gt;
</code></pre>
<hr/>
<p>If you use <a href="http://docs.python.org/3/library/timeit.html#timeit.timeit" rel="noreferrer"><code>timeit.timeit</code></a>, you will see that this method is generally faster than the others supplied:</p>
<pre><code>&gt;&gt;&gt; from timeit import timeit
&gt;&gt;&gt; name = ":1?2/3&lt;4|5"
&gt;&gt;&gt; bad = ['\\', '/', ':', '*', '?', '"', '&lt;', '&gt;', '|']
&gt;&gt;&gt;
&gt;&gt;&gt; timeit("import re;re.sub('[\\/:*?\"&lt;&gt;|]', 'special char', name)", "from __main__ import name")
11.773986358601462
&gt;&gt;&gt;
&gt;&gt;&gt; timeit("for char in bad: name = name.replace(char, 'special char')", "from __main__ import name, bad")
9.943640323001944
&gt;&gt;&gt;
&gt;&gt;&gt; timeit("name.translate({ord(c):'special char' for c in bad})", "from __main__ import name, bad")
9.48467780122894
&gt;&gt;&gt;
</code></pre>
</div>
