<p>By calling <a href="http://docs.python.org/3/library/stdtypes.html#bytes.decode" rel="nofollow"><code>bytes.decode()</code></a>, you can get <code>str</code> object from <code>bytes</code>:</p>
<pre><code>&gt;&gt;&gt; b = b'abitrary bytes'
&gt;&gt;&gt; type(b)
&lt;class 'bytes'&gt;
&gt;&gt;&gt; s = b.decode('utf-8')
&gt;&gt;&gt; type(s)
&lt;class 'str'&gt;

&gt;&gt;&gt; print(' from server ' + b)
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: Can't convert 'bytes' object to str implicitly
&gt;&gt;&gt; print(' from server ' + s)
 from server abitrary bytes
</code></pre>
