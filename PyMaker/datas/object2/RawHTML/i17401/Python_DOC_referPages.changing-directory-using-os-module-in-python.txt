<div class="post-text" itemprop="text">
<p>You need to expand <code>~</code> to actual home directory using <a href="http://docs.python.org/3/library/os.path.html#os.path.expanduser" rel="nofollow"><code>os.path.expanduser</code></a></p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; os.path.expanduser('~/Desktop')
'/home/falsetru/Desktop'
</code></pre>
<p>Otherwise, <code>~</code> mean the directory <code>~</code> (literally <code>~</code>).</p>
</div>
<div class="post-text" itemprop="text">
<p><code>os.chdir</code> doesn't do tilde expansion. You need</p>
<pre><code>os.chdir(os.path.join(os.getenv("HOME"), "Desktop"))
</code></pre>
<p>to go to <code>~/Desktop</code>.</p>
</div>
