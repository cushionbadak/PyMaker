<p>I am very new to regex and learning by practice. I wrote the following regex for finding a number inside a string of characters, however, it returns nothing. Why is that? </p>
<pre><code>string = "hello world &amp; bello stack 12456";

findObj = re.match(r'[0-9]+',string,re.I);

if findObj:
    print findObj.group();
else:
    print "nothing matched"
</code></pre>
<p>Regards</p>

<div class="post-text" itemprop="text">
<p><code>re.match</code> must match from the beginning of the string.
Use <code>re.search</code> instead.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>re.match</code> matches from the start of the string. Use <code>re.search</code></p>
<pre><code>&gt;&gt;&gt; my_string = "hello world &amp; bello stack 12456"
&gt;&gt;&gt; find_obj = re.search(r'[0-9]+', my_string, re.I)
&gt;&gt;&gt; print find_obj.group()
12456
</code></pre>
<p>P.S semicolons are not necessary.</p>
</div>
