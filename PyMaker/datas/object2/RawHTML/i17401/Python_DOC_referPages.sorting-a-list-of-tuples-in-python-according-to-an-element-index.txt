<div class="post-text" itemprop="text">
<p>How can I sort a list of tuples according to the int in a certain position? (without a for loop)</p>
<p>eg. Sorting <code>l = [(1,5,2),(7,1,4),(1,6,3)]</code> according to the third element in each tuple?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow noreferrer"><code>list.sort</code></a>, its key function, and <a href="http://docs.python.org/3/library/operator.html#operator.itemgetter" rel="nofollow noreferrer"><code>operator.itemgetter</code></a>:</p>
<pre><code>&gt;&gt;&gt; from operator import itemgetter
&gt;&gt;&gt; l = [(1,5,2),(7,1,4),(1,6,3)]
&gt;&gt;&gt; l.sort(key=itemgetter(2))
&gt;&gt;&gt; l
[(1, 5, 2), (1, 6, 3), (7, 1, 4)]
&gt;&gt;&gt;
</code></pre>
<hr/>
<p>You could also use a <a href="http://docs.python.org/3/tutorial/controlflow.html#lambda-expressions" rel="nofollow noreferrer">lambda function</a> instead of <code>operator.itemgetter</code>:</p>
<pre><code>&gt;&gt;&gt; l = [(1,5,2),(7,1,4),(1,6,3)]
&gt;&gt;&gt; l.sort(key=lambda x: x[2])
&gt;&gt;&gt; l
[(1, 5, 2), (1, 6, 3), (7, 1, 4)]
&gt;&gt;&gt;
</code></pre>
<p>However, the latter is <a href="https://stackoverflow.com/a/17243726/2555451">generally faster</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can specify the key by which to sort a list using the <a href="http://docs.python.org/2/library/functions.html#sorted" rel="nofollow"><code>sorted</code></a> builtin:</p>
<pre><code>&gt;&gt;&gt; mylist = sorted([(1,5,2),(7,1,4),(1,6,3)], key = lambda t: t[2])
&gt;&gt;&gt; mylist
[(1, 5, 2), (1, 6, 3), (7, 1, 4)]
&gt;&gt;&gt;
</code></pre>
</div>
