<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/3/library/itertools.html#itertools.islice" rel="nofollow"><code>zip</code></a> and <a href="http://docs.python.org/3/library/itertools.html#itertools.islice" rel="nofollow"><code>itertools.islice</code></a>:</p>
<pre><code>&gt;&gt;&gt; from collections import OrderedDict
&gt;&gt;&gt; import itertools
&gt;&gt;&gt;
&gt;&gt;&gt; o = OrderedDict([('xmllist', 123), ('A', 124), ('B', 125), ('C', 126), ('D', 127)])
&gt;&gt;&gt; OrderedDict((key1, o[key2]) for key1, key2 in zip(itertools.islice(o, 1, None), o))
OrderedDict([('A', 123), ('B', 124), ('C', 125), ('D', 126)])
</code></pre>
<p>mapping: <code>o['A'] = o['xmllist']</code>, <code>o['B'] = o['A']</code>, <code>o['C'] = o['B']</code>, ...</p>
