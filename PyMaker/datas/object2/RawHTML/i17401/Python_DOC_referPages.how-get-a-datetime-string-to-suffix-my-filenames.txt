<pre><code>data_201402181502.txt

<p>My code...</p>
<pre><code>import time

prefix_file = 'data'
tempus = time.localtime(time.time())    
suffix = str(tempus.tm_year)+str(tempus.tm_mon)+str(tempus.tm_mday)+
    str(tempus.tm_hour)+str(tempus.tm_min)
name_file = prefix_file + '_' + suffix + '.txt'

<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/2/library/time.html#time.strftime" rel="nofollow"><code>time.strftime</code></a> for this, which handles padding leading zeros e.g. on the month:</p>
<pre><code>from time import strftime

name_file = "{0}_{1}.txt".format(prefix_file, 
                                 strftime("%Y%m%d%H%M"))
</code></pre>
<p>If you simply turn an integer to a string using <code>str</code>, it will not have the leading zero: <code>str(2) == '2'</code>. You can, however, specify this using the <code>str.format</code> syntax: <code>"{0:02d}".format(2) == '02'</code>.</p>
</div>
