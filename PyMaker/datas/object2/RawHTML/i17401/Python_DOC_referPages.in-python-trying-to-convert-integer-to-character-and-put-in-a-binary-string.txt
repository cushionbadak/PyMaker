<div class="post-text" itemprop="text">
<p>In Python 3.3 I need to convert an integer into the middle of three bytes to send it over a serial connection. </p>
<p>That is, I need to have a value of: <code>b'\x4c\x00\x46'</code>, except that the \x00 byte will need to take the single-byte value of an integer variable that may vary from 0 to 255. I thought chr(value) would work, but that gives a string rather than a byte. </p>
<p>For example, if value is 255, I want to get <code>b'\x4c\xff\x46'</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/3/library/functions.html#bytearray" rel="nofollow"><code>bytearray</code></a>:</p>
<pre><code>&gt;&gt;&gt; b'\x4c\x00\x46'
b'L\x00F'
&gt;&gt;&gt; a = bytearray(b'\x4c\x00\x46')
&gt;&gt;&gt; a[1] = 255
&gt;&gt;&gt; a
bytearray(b'L\xffF')
&gt;&gt;&gt; bytes(a)
b'L\xffF'
</code></pre>
<p>You can also use <code>list</code> in place of <code>bytearray</code>. But using <code>list</code> does not work in Python 2.x.</p>
</div>
