<div class="post-text" itemprop="text">
<p>You need to expand <code>~</code> as actual path using <a href="http://docs.python.org/3/library/os.path.html#os.path.expanduser" rel="nofollow"><code>os.path.expanduser</code></a>:</p>
<pre><code>&gt;&gt;&gt; os.path.expanduser('~/Desktop')
'/home/falsetru/Desktop'
</code></pre>
<hr/>
<pre><code>os.chdir(os.path.expanduser('~/Desktop'))
</code></pre>
