<p>I'm trying to create a define a helper function that sums a list of lists to get a sum list. </p>
<p>For example: </p>
<pre><code>[[1,2,3],[1,2,3],[2,3,4]]
</code></pre>
<p>Would come out as:</p>
<pre><code>[4,7,10]
</code></pre>
<p>I found this on another post to work:</p>
<pre><code>[sum(i) for i in zip(*listoflists)]
</code></pre>
<p>I need to to translate this into a defined function. This is what I have so far but it's not working.</p>
<pre><code>def addhours(listoflists):
    for list in zip(*listoflists):
        newsum = sum (list)
return newsum
</code></pre>

<div class="post-text" itemprop="text">
<p>Just put the list comprehension inside the function. (with <code>return</code>)</p>
<pre><code>&gt;&gt;&gt; def addhours(listoflists):
...     return [sum(lst) for lst in zip(*listoflists)]
...
&gt;&gt;&gt; addhours([[1,2,3],[1,2,3],[2,3,4]])
[4, 7, 10]
</code></pre>
<h2>Why your code does not work?</h2>
<pre><code>def addhours(listoflists):
    for list in zip(*listoflists):
        newsum = sum (list)  # &lt;---- This statement overwrites newsum
    return newsum # returns the last sum.
</code></pre>
<p>You need a list to hold sums as items.</p>
<pre><code>def addhours(listoflists):
    result = []
    for lst in zip(*listoflists):
        result.append(sum(lst)) # Append to the list instead of overwriting.
    return result
</code></pre>
<p>BTW, don't use <code>list</code> as a variable name. It shadows builtin function <a href="http://docs.python.org/3/library/functions.html#func-list" rel="nofollow"><code>list</code></a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Since the function is  quite small, you could also use an anonymous definition:</p>
<pre><code>&gt;&gt;&gt; addhours = lambda l: [sum(i) for i in zip(*l)]
&gt;&gt;&gt; addhours([[1,2,3],[1,2,3],[2,3,4]])
&gt;&gt;&gt; [4,7,10]
</code></pre>
