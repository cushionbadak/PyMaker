<div class="post-text" itemprop="text">
<p>Empty strings will not trigger any code in the for loop, since <code>for ch in ''</code> is essentially a no-op here (there's nothing to iterate), so your <code>is_alpha</code> returns <code>True</code> for empty strings. You should add something like</p>
<pre><code>if not x:
    return False
</code></pre>
<p>to the beginning of your function.</p>
<p>(As a side note, your <code>break</code> statement is unnecessary since <code>return False</code> will exit the function.</p>
</div>
<div class="post-text" itemprop="text">
<p>Your loop</p>
<pre><code>for ch in x: 
</code></pre>
<p>Will <em>never run</em> if <code>len(x) == 0</code>, you go straight to</p>
<pre><code>return True
</code></pre>
<p>Also, if you</p>
<pre><code>import string
</code></pre>
<p>You can use <code>string.ascii_uppercase</code> and <code>string.ascii_lowercase</code>. </p>
