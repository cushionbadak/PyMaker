<p>Here, let me make a NonNegative:</p>
<pre><code> class NonNegative(object):
   def __init__(self, x):
     if x &lt; 0: raise Exception()
     self.x = x

   @property
   def value(self):
     return self.x

   @value.setter
   def set_value(self, that):
     if that &lt; 0: raise Exception()
     self.x = that

   def __add__(self, x):
     if isinstance(x, NonNegative):
       return NonNegative(x.value + self.x)
     else:
       return NotImplemented
</code></pre>
<p>Used like this:</p>
<pre><code>   a = NonNegative(1)
   b = NonNegative(2)
   c = a + b
</code></pre>
<p>so calling <code>+</code> would delegate to calling <code>__add__</code>.</p>
