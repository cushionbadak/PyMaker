<div class="post-text" itemprop="text">
<p>You can create a list of the timestamp values using a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>[element.timestamp for element in mySerializedData.processed_row]
</code></pre>
<p>Here's a generalized example. In <code>example.proto</code>:</p>
<pre><code>package example;

message Element {
  required double timestamp = 1;
  required string data = 2;
}

message Container {
  repeated Element elements = 1;
}
</code></pre>
<p>Generate the corresponding Python output using:</p>
<pre><code>protoc --python_out=. example.proto
</code></pre>
<p>Then, in Python:</p>
<pre><code>&gt;&gt;&gt; from example_pb2 import Container, Element
&gt;&gt;&gt; 
&gt;&gt;&gt; container = Container(
...   elements=[
...     Element(timestamp=.1, data='Some data.'),
...     Element(timestamp=.2, data='Some more data.'),
...   ]
... )
&gt;&gt;&gt; 
&gt;&gt;&gt; [element.timestamp for element in container.elements]
[0.1, 0.2]
</code></pre>
</div>
<span class="comment-copy">But you don't have a double array. <code>mySerializedData.processed_row[:]</code> returns a copy of that row/list.</span>
