<p>I have string that i received through Ethernet port and i have decoded it like this:   </p>
<pre><code>data, address = p1.recvfrom(1040)  
text = data.decode('ascii')
stri = ''  
for i in text:
    stri = + ord(i)  
</code></pre>
<p>Is there a way that doesn't require for loop, that can give me same string right away?</p>
</div>
<div class="post-text" itemprop="text">
<p>You could use a one-liner if you just want to minimize your code :</p>
<pre><code>stri = ''.join(str(ord(c)) for c in text)
</code></pre>
<p>Or use the <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow"><code>map</code></a> function if you really don't want to use a loop :</p>
<pre><code>stri = ''.join(map(lambda c: str(ord(c)),text))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It can be made simple with <code>map</code> and <code>ord</code>
here is the example</p>
<pre><code>&gt;&gt;&gt; reduce(lambda x, y: str(x)+str(y), map(ord,"hello world"))
'10410110810811132119111114108100'
</code></pre>
