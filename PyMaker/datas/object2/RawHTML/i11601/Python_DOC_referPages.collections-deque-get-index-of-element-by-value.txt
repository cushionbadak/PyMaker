<p>For list we can get index of element <code>list_name.index(3)</code></p>
<p>How to get index of item in deque.</p>
<p>ex:
<code>d_list = deque([1, 2, 3, 4])</code> what is the best way to get the index of element 3.</p>
<p>Edit:
I am using Python 2.7.6</p>

<div class="post-text" itemprop="text">
<p>According to the <a href="https://docs.python.org/3/library/collections.html#collections.deque" rel="nofollow">pythons docs</a>, if you have python3.5 or better, you can use <code>index</code> (as per tobspr's answer).  </p>
<p>For older pythons, you can convert the deque to a list and then use <code>index</code>:</p>
<pre><code>In [5]: from collections import deque

In [6]: d_list = deque([1, 2, 3, 4])

In [7]: list(d_list).index(3)
Out[7]: 2
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just use it like regular lists:</p>
<pre><code>&gt;&gt;&gt; d_list = collections.deque([1, 2, 3, 4])
&gt;&gt;&gt; d_list.index(3)
2
</code></pre>
<p>Edit: This is only possible in Python 3.5 right now</p>
</div>
<div class="post-text" itemprop="text">
<p>If you are using Python 2.x, <code>deque</code> is not a good choice if you want to do lookup. You can either use <code>list</code> or do the following using <code>enumerate()</code>:</p>
<pre><code>&gt;&gt;&gt; def index(element, queue):
...     for i, ele in enumerate(queue):
...         if ele == element:
...             return i
...     raise ValueError("{} is not in queue".format(element))

&gt;&gt;&gt; index(2, deque([1, 2, 3, 4]))
1
</code></pre>
<p>Alternatively, you can implement your own queue:</p>
<pre><code>&gt;&gt;&gt; class IndexedDeque(deque):
...     def index(self, element):
...         for i, ele in enumerate(self):
...             if ele == element:
...                 return i
...         raise ValueError("{} is not in queue".format(element))    

&gt;&gt;&gt; d_list = IndexedDeque([1, 2, 3, 4])
&gt;&gt;&gt; d_list.index(2)
1
</code></pre>
