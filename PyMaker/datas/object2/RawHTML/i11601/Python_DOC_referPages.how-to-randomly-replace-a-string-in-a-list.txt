<p>If I have something like this:</p>
<pre><code>L = ['-','-','-','-','-','-','-']
</code></pre>
<p>And let's say that I want to replace certain number of those strings. How do I randomly select a position within the list to replace it for something else? For example:</p>
<pre><code>L = ['-','*','-','-','-','*','*']
</code></pre>

<div class="post-text" itemprop="text">
<p>Use random.randrange</p>
<pre><code>import random

some_list=["-","-","-","-","-","-","-"]

n=2
for i in range(n):
    some_list[random.randrange(0,len(some_list))]="*"
</code></pre>
<p>Non-repeat solution:</p>
<pre><code>import random

some_list=["-","-","-","-","-","-","-"]

n=8
if n&gt;len(some_list):
    some_list=["*" for i in some_list]
else:
    for i in range(n):
           position=random.randrange(0,len(some_list))
           while some_list[position]=="*":
                   position=random.randrange(0,len(some_list))
           some_list[position]="*"

print(some_list)
</code></pre>

<div class="post-text" itemprop="text">
<p>Actually, you can use the module <code>random</code> and its function <code>randint</code>, as it follows:</p>
<pre><code>import random

num_replacements = 3 
L = ['-','-','-','-','-','-','-']

idx = random.sample(range(len(L)), num_replacements)

for i in idx:
    L[i] = '*'
</code></pre>
<p>For more information, you can check the <code>random</code> module documentation at: <a href="https://docs.python.org/3/library/random.html" rel="nofollow">https://docs.python.org/3/library/random.html</a></p>
<p>EDIT: Now sampling random number using <code>random.sample</code>, rather than using <code>random.randint</code>, which may generate the same number during different iterations.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>import random as r

L = ['-','-','-','-','-','-','-']

def replaceit(L,char):
    L[r.randint(0,len(L))] = char
    return L

newL = replaceit(L,'*')
print newL
</code></pre>
<p>Simply call replaceit with newL to replace another random character. </p>

<p>use <code>random.choice</code></p>
<pre><code>import random

L = ['-','-','-','-','-','-','-']

while L.count('*') &lt; 3:
    pos = random.choice(range(len(L)))
    L[pos] = '*'

print(L)
</code></pre>
