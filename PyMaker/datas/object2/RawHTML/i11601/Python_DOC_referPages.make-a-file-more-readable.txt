<div class="post-text" itemprop="text">
<p>Say we have a file with the following inside:</p>
<pre><code>{{{banana},{kinky},{bronze-spaghetti,
definitly-not},{legitimate-style, witness,
drunk}}}
</code></pre>
<p>I wanted to rearrange this file so that it look like this:</p>
<pre><code>{{{banana},
{kinky},
{bronze-spaghetti,definitly-not},
{legitimate-style, witness, drunk}}}
</code></pre>
<p>Notice that each line ends with <code>},</code> except for the last one. Now my question is how would I do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>you can find the regex "}," and insert after it a new line</p>
<p>edit:
as jDo suggested, you can do something like that:</p>
<pre><code>with open('in.txt', 'r') as infile:
    data=infile.read()
with open('out.txt', 'w') as outfile:
    outfile.write(data.replace("\n", "").replace("},", "},\n"))
</code></pre>

<div class="post-text" itemprop="text">
<p>I think you could simply use <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow"><code>replace</code></a> (triple quotes for multi-line strings):</p>
<pre><code>original_text = """{{{banana},{kinky},{bronze-spaghetti, definitly-not},{legitimate-style, witness, drunk}}}"""
reformatted = original_text.replace("\n", "").replace("},", "},\n")
print(reformatted)
</code></pre>
<p>This can easily be done in the shell:</p>
<pre><code>&gt;&gt;&gt; my_str = """{{{banana},{kinky},{bronze-spaghetti,
... definitly-not},{legitimate-style, witness,
... drunk}}}"""
&gt;&gt;&gt; 
&gt;&gt;&gt; my_str = my_str.replace("\n", "").replace("},", "},\n")
&gt;&gt;&gt; 
&gt;&gt;&gt; print(my_str)
{{{banana},
{kinky},
{bronze-spaghetti,definitly-not},
{legitimate-style, witness,drunk}}}
&gt;&gt;&gt; 
</code></pre>
