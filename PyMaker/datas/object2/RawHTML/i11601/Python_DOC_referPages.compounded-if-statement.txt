<pre><code>if A == B &amp; C &gt;= D &amp; C &lt;= E:
</code></pre>
<p>each of the sub statements works on there own but I can't seem to combine them together in one command.</p>

<div class="post-text" itemprop="text">
<p><code>&amp;</code> is a <a href="https://docs.python.org/3/reference/expressions.html#binary-bitwise-operations" rel="nofollow">binary bitwise operator</a>, not the <a href="https://docs.python.org/3/reference/expressions.html#boolean-operations" rel="nofollow">boolean operator <code>and</code></a> you are looking for.</p>
<p>Do this:</p>
<pre><code>if A == B and C &gt;= D and C &lt;= E:
</code></pre>
</div>
<span class="comment-copy"><code>&amp;</code> is a <a href="https://docs.python.org/3/reference/expressions.html#binary-bitwise-operations" rel="nofollow noreferrer">binary bitwise operator</a>, not the logical <code>and</code>.</span>
