<p>I have the following two lists of strings with the same size:</p>
<pre><code>l1 = ['foo', 'foo','bar','cho']
l2 = ['foo', 'qux','bar','cxx']
              *           *
</code></pre>
<p>What I want to do is to find the position where the members differs,
yielding:</p>
<pre><code>1, 3
</code></pre>
<p>How can we do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use list comprehension:</p>
<pre><code>&gt;&gt;&gt; [i for i, v1 in enumerate(l1) if v1 != l2[i]]
[1, 3]
</code></pre>
<p>This will iterate over the first list, and compare the values with the second list, and incase they do not match, adds the index to the result.</p>
</div>
<div class="post-text" itemprop="text">
<p>When iterating over 2 lists of the same size, use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip</code></a>:</p>
<pre><code>diff = [i for i, (x, y) in enumerate(zip(l1, l2)) if x != y]
</code></pre>
