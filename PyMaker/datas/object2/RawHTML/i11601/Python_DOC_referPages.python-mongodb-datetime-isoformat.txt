<div class="post-text" itemprop="text">
<p>You do not need to create any special format for your date. pymongo works with python standard library types and automatically converts them, which means you can use the python standard library <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow">datetime</a> module for your date/time/datetime.</p>
