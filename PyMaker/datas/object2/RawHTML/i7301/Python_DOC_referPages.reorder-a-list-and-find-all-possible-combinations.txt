<p>I have a Python list with a variable number of elements.</p>
<pre><code>[a,b,c,d,e]
</code></pre>
<p>I need to rearrange these elements inside the list and find all possible unique combinations.</p>
<pre><code>[a,b,c,d,e]
[d,b,a,e,c]
[e,d,a,c,b], etc.
</code></pre>
<p>I cannot have same elements repeated, or have any elements removed:</p>
<pre><code>[a,a,a,d,e]  &lt;-- bad
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You should use <code>itertools.permutations</code>, for example:</p>
<pre><code>import itertools
my_list = [a, b, c, d]
for x in itertools.permutations(my_list):
    print(x)
</code></pre>
</div>
