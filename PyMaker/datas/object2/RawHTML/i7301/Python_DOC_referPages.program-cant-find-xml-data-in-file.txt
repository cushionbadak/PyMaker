<p><code>attri = dom.findall('attribute')</code> is returning no results.</p>
<p>The section of the docs titled <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#finding-interesting-elements" rel="nofollow noreferrer">Finding interesting elements</a> notes that</p>
<blockquote>
<p><code>Element.findall()</code> finds only elements with a tag which are direct children of the current element.</p>
</blockquote>
<p>But that</p>
<blockquote>
<p>More sophisticated specification of which elements to look for is possible by using <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#elementtree-xpath" rel="nofollow noreferrer">XPath</a>.</p>
</blockquote>
<p>The simplest fix would be to change your code to</p>
<pre><code>for c in dom.findall('.//attribute'):
    name = c.find('.//name').text
    value = c.find('.//value').text
    print(name + ':' + value, end="│")
</code></pre>
<p>See <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#supported-xpath-syntax" rel="nofollow noreferrer">Supported XPath syntax</a> for more information.</p>
</div>
