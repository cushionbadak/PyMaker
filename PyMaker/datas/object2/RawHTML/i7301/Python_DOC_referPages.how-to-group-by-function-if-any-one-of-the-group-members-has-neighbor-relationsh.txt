<p><a href="https://docs.python.org/3/library/functools.html#functools.partial" rel="nofollow noreferrer">https://docs.python.org/3/library/functools.html#functools.partial</a></p>
<pre><code>from functools import partial 
...
groupda = groupby(testing1, partial(isneighborlocation, centre))
</code></pre>
<p>but, of course, it means that you have to explicitly test every 'centre'.<br/>
Btw, be aware of the sorting requirement on groupby - it seems likely that it may trip you up.</p>
