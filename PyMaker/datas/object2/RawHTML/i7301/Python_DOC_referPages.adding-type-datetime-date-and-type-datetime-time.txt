<div class="post-text" itemprop="text">
<p>I have 2017-07-21 as a'datetime.date'  object and </p>
<p>15:30:00 as a datetime.time' object </p>
<p>How do I combine the two to get</p>
<p>2017-07-21-15:30:00? </p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>datetime.combine</code> from <code>datetime</code>.</p>
<pre><code>import datetime

d = datetime.date(2017, 07, 21)
t = datetime.time(15, 30, 0)
dt = datetime.datetime.combine(d, t)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<blockquote>
<p><em>classmethod</em> <code>datetime.combine(</code><em>date</em>, <em>time</em>, <em>tzinfo</em>=<em>self.tzinfo</em>)</p>
<p>Return a new <code>datetime</code> object whose date components are equal to the given <code>date</code> object’s, and whose time components are equal to the given <code>time</code> object’s.</p>
</blockquote>
<p><a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.combine" rel="nofollow noreferrer">source</a></p>
</div>
<div class="post-text" itemprop="text">
<pre><code>     &gt;&gt;&gt; from datetime import datetime, date, time

        &gt;&gt;&gt; d = date(2017, 7, 21)
        &gt;&gt;&gt; t = time(15, 30, 0)
        &gt;&gt;&gt; newDate = datetime.combine(d, t)
        &gt;&gt;&gt; newDate
        datetime.datetime(2017, 7, 21, 15, 30)
        &gt;&gt;&gt; newDate.strftime(%Y-%m-%d-%H:%M:%S)
        '2017-07-21-15:30:00'
</code></pre>
<p>You can use the <code>datetime.combine</code> method as above. More about info <a href="https://docs.python.org/3/library/datetime.html#datetime.time" rel="nofollow noreferrer">here</a></p>
