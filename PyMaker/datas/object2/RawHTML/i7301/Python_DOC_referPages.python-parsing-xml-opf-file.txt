<p>What you are looking for (<code>&lt;dc:title&gt;</code>) is an element, not an attribute. Here is how you can get its value:</p>
<pre><code>from xml.etree import ElementTree as ET

tree = ET.parse('content.opf')
title = tree.find(".//{http://purl.org/dc/elements/1.1/}title")
print(title.text)
</code></pre>
<p>Output:</p>
<pre><code>Faust_I_
</code></pre>
