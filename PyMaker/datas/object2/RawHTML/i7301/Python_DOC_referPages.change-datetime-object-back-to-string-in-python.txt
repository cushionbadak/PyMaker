<p>If I have a datetime object such as <code>datetime.datetime(2017, 7, 20, 15, 0)</code>, how do I convert it back to a string in the format of <code>"2017-07-20-15-00"</code>? </p>
</div>
<div class="post-text" itemprop="text">
<p>python has date formattting built-in either using <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer"><code>datetime.strftime</code></a>, <a href="https://docs.python.org/3/library/functions.html?highlight=format#format" rel="nofollow noreferrer"><code>str.format</code></a> or the new <a href="https://www.python.org/dev/peps/pep-0498/" rel="nofollow noreferrer">f-strings</a> in python &gt;= 3.6:</p>
<pre><code>from datetime import datetime

dt = datetime(2017, 7, 20, 15, 0)

# str.format
strg = '{:%Y-%m-%d %H-%M-%S}'.format(dt)
print(strg)  # 2017-07-20 15-00-00

# datetime.strftime
strg = dt.strftime('%Y-%m-%d %H-%M-%S')
print(strg)  # 2017-07-20 15-00-00

# f-strings in python &gt;= 3.6
strg = f'{dt:%Y-%m-%d %H-%M-%S}'
print(strg)  # 2017-07-20 15-00-00
</code></pre>
<p>you can tweak the format string according to your needs. <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer"><code>strftime()</code> and <code>strptime()</code> Behavior</a> explains what the format specifiers mean.</p>
