<p>Search about python destructor.</p>
<pre><code>class Package:
    ...
    def __del__(self):
        pass

    def __exit__(self, exc_type, exc_value, traceback):
        pass
</code></pre>
<p>or decorator</p>
<pre><code>@SomeDecorator
def function():
    pass

def SomeDecorator(func):
    def wrapper(self, *args, **kwargs):
        func(self, *args, **kwargs)
        # after func
return wrapper
</code></pre>
