<div class="post-text" itemprop="text">
<p>you can parse your string using <a href="https://docs.python.org/3/library/datetime.html?highlight=strptime#datetime.datetime.strptime" rel="nofollow noreferrer"><code>datetime.strptime</code></a> to a <code>datetime</code> object and then call <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.time" rel="nofollow noreferrer"><code>.time()</code></a> on that to get the time:</p>
<pre><code>from datetime import datetime

strg = "2017-07-20-10-30"

dt = datetime.strptime(strg, '%Y-%m-%d-%H-%M')
tme = dt.time()
print(tme)  # 10:30:00
</code></pre>
<p>the <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer"><code>strftime()</code> and <code>strptime()</code> Behavior</a> is well documented.</p>
<p>of course you can also chain these calls:</p>
<pre><code>tme = datetime.strptime(strg, '%Y-%m-%d-%H-%M').time()
</code></pre>
