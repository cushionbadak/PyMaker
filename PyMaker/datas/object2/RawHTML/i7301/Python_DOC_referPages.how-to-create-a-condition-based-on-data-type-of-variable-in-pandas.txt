<div class="post-text" itemprop="text">
<p>You can use the <code>isinstance(object, classinfo)</code> built-in function from Python defined <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer">here</a>.</p>
<blockquote>
<p>Return true if the object argument is an instance of the classinfo argument</p>
</blockquote>
<p>So you can use it like this:</p>
<pre><code>if isinstance(i, pd.Series)
</code></pre>
<p>and</p>
<pre><code>if isinstance(i, int)
</code></pre>
</div>
