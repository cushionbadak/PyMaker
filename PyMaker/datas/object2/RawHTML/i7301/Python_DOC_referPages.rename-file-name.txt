<p>I guess You're using <code>Python3</code>, since this is valid syntax in <code>Python2</code>. For <code>Python3</code>, use following:</p>
<pre><code>import os
def rename_file():
    file_list = os.listdir(r"C:\Users\Sushant\Desktop\test")
    print(file_list)

    for file_name in file_list:
        os.rename(file_name, file_name.translate(str.maketrans("", "", "0123456789")))

rename_file()
</code></pre>
<p>And here's the docs for <a href="https://docs.python.org/3/library/stdtypes.html#str.maketrans" rel="nofollow noreferrer">maketrans</a> function.</p>

<div class="post-text" itemprop="text">
<p>When using <code>translate()</code> you have to pass in a table made by <code>maketrans()</code> as the first argument. This is the reason for getting a <code>TypeError</code> since you are passing in <code>None</code>.</p>
<p>With <code>maketrans()</code> you pass in the character you want to change in one string. The second argument is a string with the character you want to put in their place. This function returns a table, which you then pass into <code>translate()</code>.</p>
<p>In your case something like this would work.</p>
<pre><code>file_name.translate(str.maketrans("", "", "0123456789"))
</code></pre>
<p>Further reading on <a href="https://www.tutorialspoint.com/python/string_maketrans.htm" rel="nofollow noreferrer">maketrans()</a> and <a href="https://www.tutorialspoint.com/python/string_maketrans.htm" rel="nofollow noreferrer">translate()</a></p>
</div>
