<div class="post-text" itemprop="text">
<p>You don't need regex for this at all. A list comprehension with <code>any</code> will suffice</p>
<pre><code>&gt;&gt;&gt; subs = ['ra', 'dec', 'lat']
&gt;&gt;&gt; strings = ["ra", "dec", "ra-error", "dec-error", "glat", "glon", "flux", "l", "b"]
&gt;&gt;&gt; [s for s in strings if any(i in s for i in subs)]
['ra', 'dec', 'ra-error', 'dec-error', 'glat']
</code></pre>
