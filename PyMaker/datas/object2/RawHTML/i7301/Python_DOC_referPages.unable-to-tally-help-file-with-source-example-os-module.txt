<div class="post-text" itemprop="text">
<p>If you print the representation of <code>os.DirEntry</code>, and as you can see in the help, it is the object <code>nt.DirEntry</code> (Because you are on a windows system):</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; os.DirEntry
&lt;class 'nt.DirEntry'&gt;
&gt;&gt;&gt; os.DirEntry.__module__
'nt'
&gt;&gt;&gt; help(os.DirEntry)
Help on class DirEntry in module nt:

class DirEntry(builtins.object)
...
&gt;&gt;&gt; import nt
&gt;&gt;&gt; nt.DirEntry is os.DirEntry
True
</code></pre>
<p>The place where <code>DirEntry</code> is imported from <code>nt</code> in the <code>os</code> module is here:</p>
<pre><code>elif 'nt' in _names:
    name = 'nt'
    linesep = '\r\n'
    from nt import *
</code></pre>
<p>The <code>from nt import *</code> imports (mostly) everything from the <code>nt</code> module, which is where <code>DirEntry</code> is added to the <code>os</code> module.</p>
