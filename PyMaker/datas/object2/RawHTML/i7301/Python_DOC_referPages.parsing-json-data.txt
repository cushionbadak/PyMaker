<p>You have already parsed your data with <code>data = response.json()</code></p>
<p>What you are showing below that is the string representation of this object, which is a list containing two dictionaries.</p>
<p>To access the data you can do something like this:</p>
<pre><code>for d in data:
    print(d['name'])
    for l in d['links']:
        print("  " + l['href'])
    print("\n")
</code></pre>
