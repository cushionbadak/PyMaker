<p>Is there better way of writing this code: </p>
<pre><code>def add (exe1, exe2):
    try:
        a = float (exe1)
        b = float (exe2)
        total = float (a + b)
    except ValueError:
        return None
    else:
        return total
</code></pre>

<div class="post-text" itemprop="text">
<p>You can have it all inside a <code>try/except</code> block (calculation and <code>return</code>):</p>
<pre><code>def add(exe1, exe2):
    try:
        return float(exe1) + float(exe2)
    except ValueError:
        return None
</code></pre>
<p>Also note, the default return value from function is <code>None</code>, so the second <code>return</code> is not really necessary (you could have <code>pass</code> instead), but it makes code more readable.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can also use contextlib.suppress, if you find it more readable.</p>
<pre><code>from contextlib import suppress
def add(exe1, exe2):
    with suppress(ValueError):
        return float(exe1) + float(exe2)
</code></pre>
<p>See the documentation <a href="https://docs.python.org/3/library/contextlib.html#contextlib.suppress" rel="nofollow noreferrer">here</a>.</p>
