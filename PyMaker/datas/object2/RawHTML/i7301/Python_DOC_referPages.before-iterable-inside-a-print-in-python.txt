<p>I have a list of names and I want to print each element of the list in a different line without a for loop. So, after some research I found this example: <code>print(*names, sep='\n')</code>, witch results in exactly what I want. But what does this <code>*</code> character before the list name means?</p>
</div>
<div class="post-text" itemprop="text">
<p>The * is used to unpack argument lists when calling a function. In this case it unpacks your list of names.</p>
</div>
<span class="comment-copy"><code>print(*[1,2,3])</code> is the same as <code>print(1,2,3)</code>.</span>
<span class="comment-copy">See <a href="https://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists" rel="nofollow noreferrer">Unpacking Argument Lists</a> in the official tutorial.</span>
