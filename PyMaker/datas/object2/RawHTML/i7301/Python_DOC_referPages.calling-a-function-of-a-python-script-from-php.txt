<div class="post-text" itemprop="text">
<p><strong>PHP</strong>:</p>
<p>It could be possible that your <code>myscript.py</code> is not found, use a Full Path:</p>
<pre><code> $output = shell_exec("&lt;your Full Path&gt;myscript.py '$varone' 'vartwo' ");
</code></pre>
<p><strong>myscript.py</strong></p>
<p>You must <strong>Read</strong> the passed Parameters in your Python Script, for instance:  </p>
<pre><code>def InitPython(ID,Name):
  'DO STUFF'

if __name__ == '__main__':
   import sys
   InitPython(sys.argv[1], sys.argv[2])
</code></pre>
<blockquote>
<p>Relevant: Python Documentation: <a href="https://docs.python.org/3/library/argparse.html#module-argparse" rel="nofollow noreferrer">module-argparse</a></p>
<p>The argparse module makes it easy to write user-friendly command-line interfaces. The program defines what arguments it requires, and argparse will figure out how to parse those out of sys.argv</p>
</blockquote>
