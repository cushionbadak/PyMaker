<div class="post-text" itemprop="text">
<p>If you want, that the image slightly moves in the direction of the mouse, then you have to add a vector to the current position of the image, which points in the direction of the mouse. The length of the has to be less than the distance to the mouse:</p>
<pre class="lang-py prettyprint-override"><code>dx, dy = (x - self.x), (y - self.y)
self.x, self.y = self.x + dx * 0.1, self.y + dy * 0.1 
</code></pre>
<p>this causes that the ball makes a small step to the mouse in each frame.</p>
<pre class="lang-py prettyprint-override"><code>class Cat(object):
    def __init__(self):
        self.image = pygame.image.load('ball.png')
        self.x = 1
        self.y = 1
        self.t = 0.1

    def draw(self, surface):
        mosx = 0
        mosy = 0
        x,y = pygame.mouse.get_pos()
        dx, dy = (x - self.x), (y - self.y)
        self.x, self.y = self.x + dx * self.t, self.y + dy * self.t
        w, h = self.image.get_size()
        surface.blit(self.image, (self.x - w/2, self.y - h/2))
        pygame.display.update()
</code></pre>
