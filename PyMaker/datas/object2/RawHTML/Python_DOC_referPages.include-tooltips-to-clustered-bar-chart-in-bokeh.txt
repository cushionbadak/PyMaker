<p>You could show some data with the hover tools by adding these lines:</p>
<pre><code>source = ColumnDataSource(data=dict(
    x=factors,
    east=[ 5, 5, 6, 5, 5, 4, 5, 6, 7, 8, 6, 9 ],
    west=[ 5, 7, 9, 4, 5, 4, 7, 7, 7, 6, 6, 7 ],
))

tooltips = [
    ("x", "@x"),
    ("name", "$name"),
    ("value", "@$name")
]

p = figure(x_range=FactorRange(*factors), plot_height=250,
           toolbar_location="left", tools="hover", tooltips = tooltips)
</code></pre>
<p>The hover variable <code>@$name</code> can be used to look up values from the stack column for each layer. For instance, if a user hovers over a stack glyph with the name "East", then <code>@$name</code> is equivalent to <code>@{East}</code>. Source: <a href="https://bokeh.pydata.org/en/latest/docs/user_guide/categorical.html#hover-tools" rel="nofollow noreferrer">https://bokeh.pydata.org/en/latest/docs/user_guide/categorical.html#hover-tools</a></p>
