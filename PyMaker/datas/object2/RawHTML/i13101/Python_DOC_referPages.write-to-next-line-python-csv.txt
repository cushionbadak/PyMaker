<p>I am working on a project which I figured would need to use something like a CSV document for storing data. But I ran into a problem, I need python to write the variable into the next line every time instead of the first line (this is overwriting the existing data in that column). Could anyone please help me to solve my problem?</p>
<p>Here is my code:</p>
<pre><code>import csv
tag=input('Enter label for the new profile: ')
pwd=input('Type password for the new profile: ')
b=open('database.csv', 'w',)
a=csv.writer(b)
a.writerow([tag,pwd])
b.close()
</code></pre>
<p>Thanks in advance.</p>
</div>
<div class="post-text" itemprop="text">
<p>You want to use 'append' mode for your write. </p>
<p>Change your 'w' mode in your open 'a'. </p>
<p>Change this: </p>
<pre><code>b=open('database.csv', 'w',)
</code></pre>
<p>to this:</p>
<pre><code>b=open('database.csv', 'a',)
</code></pre>
<p>Read the documentation <a href="https://docs.python.org/3/tutorial/inputoutput.html#reading-and-writing-files" rel="nofollow">here</a> on the different modes</p>
<blockquote>
<p>'w' for only writing (an existing file with the same name will be
  erased), and 'a' opens the file for appending; any data written to the
  file is automatically added to the end.</p>
</blockquote>
</div>
