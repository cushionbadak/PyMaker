<div class="post-text" itemprop="text">
<p>I am calling python method from robot keyword. But I do not want robot framework to wait until test is finished because I want to run that method continuously. How to continue running next test case?
e.g. I am calling start logs (python method) from robot keyword </p>
<pre><code>def start_logs(self):
    subprocess.call('adb logcat &gt; test.txt', shell=True)
</code></pre>
<p>After starting logs I want to run other test cases. But robot framework keeps waiting for above.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do:</p>
<pre><code>def start_logs(self):
    subprocess.Popen(['adb', 'logcat', '&gt;', 'test.txt'])
</code></pre>
<p>using <a href="https://docs.python.org/3/library/subprocess.html#subprocess.Popen" rel="nofollow noreferrer">subprocess.Popen</a> as <a href="https://stackoverflow.com/users/646619/colonel-thirty-two">Colonel Thirty Two</a> mentioned in the comments.</p>
</div>
<div class="post-text" itemprop="text">
<p>There is a Keyword for that! <a href="http://robotframework.org/robotframework/latest/libraries/Process.html#Start%20Process" rel="nofollow">Start Process</a> will call any keyword but will not wait for it to return. Don't forget to add the Process library to your project.</p>
