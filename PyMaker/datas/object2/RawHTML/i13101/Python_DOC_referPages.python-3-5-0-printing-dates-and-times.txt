<div class="post-text" itemprop="text">
<p><code>datetime.time()</code> is equivalent to <code>datetime.time(hours=0, minute=0)</code>:</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; datetime.time()
datetime.time(0, 0)
&gt;&gt;&gt; datetime.time(hour=0, minute=0)
datetime.time(0, 0)
</code></pre>
<p><a href="https://docs.python.org/3/library/datetime.html#module-datetime" rel="nofollow"><code>datetime</code></a> is a module. <a href="https://docs.python.org/3/library/datetime.html#datetime.time" rel="nofollow"><code>datetime.time</code></a> is a class. <code>datetime.time()</code> is an instance of that class created using the default hour, minute values (<code>0</code>,<code>0</code>) e.g., to create <a href="https://docs.python.org/3/library/datetime.html#datetime.time" rel="nofollow">time</a> object that corresponds to <code>19:00</code> instead, you could use <code>datetime.time(19, 0)</code>.</p>
<p>To print only time for an existing <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime" rel="nofollow"><code>datettime.datetime</code></a> or <a href="https://docs.python.org/3/library/datetime.html#datetime.time" rel="nofollow"><code>datetime.time</code></a> instance, you could use <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow"><code>%H:%M</code> time format</a>:</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; current_time = datetime.datetime.now()
&gt;&gt;&gt; print("{:%H:%M}".format(current_time))
08:40
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>date_time = datetime.datetime.now()
time = date_time.time()

# date_time is a datetime object tuple of year, month, day, hour, minnute, second, ms 
# to access the time portion, simply call the time() function, which will return a tuple of only the hour, minue, second portion

# printing the time will give timestamp representation, but the object is actually of type datetime.time
</code></pre>
</div>

<p>like this:</p>
<pre><code>from datetime import datetime;

t = datetime.now();

</code></pre>
