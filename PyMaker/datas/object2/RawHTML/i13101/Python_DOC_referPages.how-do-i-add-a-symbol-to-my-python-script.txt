<div class="post-text" itemprop="text">
<p>The probably best way would be to just use the percent format type:</p>
<pre><code>format(average, '.1%')

# or using string formatting:
'{:.1%}'.format(average)
</code></pre>
<p>This already multiplies the <code>average</code> by 100 and will also show a percent sign at the end.</p>
<pre><code>&gt;&gt;&gt; average = .123
&gt;&gt;&gt; print('The average is: {:.1%}'.format(average))
The average is: 12.3%
</code></pre>
