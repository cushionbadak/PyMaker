<div class="post-text" itemprop="text">
<p>You can try this:</p>
<pre><code>def reverse(ip):
        if len(ip) &lt;= 1:
                return ip
        l = ip.split('.')
        return '.'.join(l[::-1])

ip = '10.1.2.3'
print reverse(ip)
</code></pre>
<p>Output:</p>
<pre><code>3.2.1.10
</code></pre>

<p>You are getting tripped up by python's insistence that strings are arrays of characters.  How about converting them to numbers and then reversing them?</p>
<pre><code>&gt;&gt;&gt; numbers = [int(v) for v in ip.split('.')]
&gt;&gt;&gt; numbers.reverse()
&gt;&gt;&gt; ip_reversed = '.'.join([str(v) for v in numbers])
&gt;&gt;&gt; ip_reversed
'3.2.1.10'
</code></pre>
