<pre><code>print ("hello world")
</code></pre>
<p>would appear on the user's screen on the left side, so can i make it centre-aligned instead?</p>

<div class="post-text" itemprop="text">
<p>First, use <a href="https://docs.python.org/3/library/os.html#os.get_terminal_size" rel="noreferrer"><code>os.get_terminal_size()</code></a> function to get the console's width(so you don't need to know your console before):</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; os.get_terminal_size()
os.terminal_size(columns=80, lines=24)
&gt;&gt;&gt; os.get_terminal_size().columns
80
&gt;&gt;&gt; os.get_terminal_size().columns  # after I changed my console's width
97
&gt;&gt;&gt; 
</code></pre>
<p>Then, we can use <a href="https://docs.python.org/3/library/stdtypes.html#str.center" rel="noreferrer"><code>str.center()</code></a>:</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; print("hello world".center(os.get_terminal_size().columns))
                                  hello world                                  
&gt;&gt;&gt; 
</code></pre>
<p>So the clear code looks like:</p>
<pre><code>import os

width = os.get_terminal_size().columns
print("hello world".center(width))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Knowing the console width, you can also print centered using <code>format</code>:</p>
<pre><code> # console width is 50 
 print "{: ^50s}".format("foo")
</code></pre>
<p>will print 'foo' in the middle of a 50-column console.</p>
