<div class="post-text" itemprop="text">
<p>I have three variables, I would like to know how to multiply all of these variables by another variable <code>number</code> simultaneously.</p>
<p>For example</p>
<pre><code>number = 2

var1 = 0
var2 = 1
var3 = 2
</code></pre>
<p>The output should be: </p>
<pre><code>0
2
4
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could just use a simple for loop as follows:</p>
<pre><code>number = 2

var1 = 0
var2 = 1
var3 = 2

for output in (var1, var2, var3):
    print output * number
</code></pre>
<p>This would display:</p>
<pre><code>0
2
4
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="noreferrer">list comprehension</a></p>
<pre><code>&gt;&gt;&gt; number = 2
&gt;&gt;&gt; 
&gt;&gt;&gt; var1 = 0
&gt;&gt;&gt; var2 = 1
&gt;&gt;&gt; var3 = 2
&gt;&gt;&gt; 
&gt;&gt;&gt; [i*number for i in (var1,var2,var3)]
[0, 2, 4]
</code></pre>
<p>And to print it</p>
<pre><code>&gt;&gt;&gt; for i in output:
...     print(i)
... 
0
2
4
</code></pre>
<p>You can use <code>map</code> and <code>lambda</code> also</p>
<pre><code>&gt;&gt;&gt; for i in map(lambda x:x*number,(var1,var2,var3)):
...     print(i)
... 
0
2
4
</code></pre>
</div>
