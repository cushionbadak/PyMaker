<div class="post-text" itemprop="text">
<p>You are making two mistakes:</p>
<ul>
<li><p>You are <em>ignoring</em> the return values of <code>str.strip()</code> and <code>str.split()</code>. Strings are immutable, these methods return <em>new</em> objects.</p></li>
<li><p>You are stripping, then splitting, leaving punctuation <em>between</em> words in place, because stripping only removes characters from the start and end.</p></li>
</ul>
<p>Split first, then strip, and store the results:</p>
<pre><code>result = [word.strip(string.punctuation) for word in str2.split()]
</code></pre>
<p>I used a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> to process each result from the <code>str2.split()</code> call in turn:</p>
<pre><code>&gt;&gt;&gt; import string
&gt;&gt;&gt; str2 = "abc, 'bca'"
&gt;&gt;&gt; [word.strip(string.punctuation) for word in str2.split()]
['abc', 'bca']
</code></pre>
