<div class="post-text" itemprop="text">
<p>You are defining callbacks not binding them to anything in your application.  To get a callback to react it needs to be bound  to an event (e.g. <code>tk.bind ("&lt;Key&gt;", callback)</code> )</p>
<p>Please have a look on the <a href="https://docs.python.org/3/library/tkinter.html?highlight=event%20tkinter#bindings-and-events" rel="nofollow noreferrer">documentation</a> of event / event-binding.  </p>
<p>You can bind the event like shown <a href="https://stackoverflow.com/questions/16153715/python-event-binding-with-tkinter">in this SO question</a>.</p>
<p>What is it you want to do using repr.event.F11 / repr.event.Escape? print something?</p>
<p>Please make sure to fix your indentation.</p>
<p>You can also bind <code>F11</code> and <code>Escape</code> to your callbacks using </p>
<ul>
<li><code>tk.bind("&lt;F11&gt;", fullscreen_on)</code></li>
<li><code>tk.bind("&lt;Escape&gt;", fullscreen_off)</code></li>
</ul>
