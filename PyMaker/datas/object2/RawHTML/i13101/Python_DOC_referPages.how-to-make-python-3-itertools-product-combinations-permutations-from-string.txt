<div class="post-text" itemprop="text">
<p>Add your strings to a list, use <a href="https://docs.python.org/3/library/itertools.html#itertools.permutations" rel="nofollow">itertools permutations</a>, and then join the result.</p>
<pre><code>import itertools

L = [string1, string2, string3]
# Convert perms to list if you want to iterate multiple times
perms = itertools.permutations(L)
for perm in perms:
    print(''.join(perm))
</code></pre>
