<p>Suppose I have a function that cleans a string (ex. make the string all uppercase letters and etc).</p>
<pre><code>def clean_string(s)
</code></pre>
<p>How do I apply this function to strings in a list and return a cleaned list?
I think this uses for loops but I just can't get around it.</p>
</div>
<div class="post-text" itemprop="text">
<p>List comprehensions are the modern way to do this:</p>
<pre><code>new_string = [clean_string(s) for s in your_list]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>That's exactly what <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow"><code>map()</code></a> function is for:</p>
<pre><code>map(clean_string, your_list)
</code></pre>
<p><em>Note</em>:In python 3.X <code>map</code> returns an iterator which is more optimized in terms of memory use, but if you want to get a list you can use <code>list()</code> function to convert the result to list, or better you can use a list comprehension </p>
<pre><code>[clean_string(item) for item in your_list]
</code></pre>
<p>But if you are dealing with large data and you just want to loop over the result you better to use <code>map()</code> in in python 3.X or a generator expression in python 2.X.</p>
</div>
