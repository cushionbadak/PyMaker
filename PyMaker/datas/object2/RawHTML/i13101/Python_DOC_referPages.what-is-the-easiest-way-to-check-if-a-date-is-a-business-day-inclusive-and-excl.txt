<h1>something like</h1>
<p>is_Business_day(date) returns true or false.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/3/library/datetime.html#datetime.date.weekday" rel="nofollow"><code>date.weekday()</code></a> function to get the weekday that a date is on. Monday is 0 and Sunday is 6, so a business day would be <code>mydate.weekday() &lt; 5</code></p>
</div>
<div class="post-text" itemprop="text">
<p>I found this solution:</p>
<pre><code>from bdateutil import isbday

isbday(date, holidays = holidays.US())
</code></pre>
