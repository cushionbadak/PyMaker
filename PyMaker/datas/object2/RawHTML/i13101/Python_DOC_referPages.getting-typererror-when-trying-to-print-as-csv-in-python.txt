<div class="post-text" itemprop="text">
<p>You are using Python 2, and the <a href="https://docs.python.org/2.7/library/functions.html#open" rel="nofollow">open</a> function has no <code>newline</code> argument.</p>
<p>You should use Python 3 in order to use that argument. See the new <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow">open</a> documentation.</p>
</div>
<div class="post-text" itemprop="text">
<p>In Python 2, use <code>'wb'</code> instead of <code>newline=''</code> to achieve the same effect. Python 3 uses the latter. </p>
