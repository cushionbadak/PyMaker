<p>I want to merge 2 lists : </p>
<pre><code>A=['A','B','C']
B=['1','2','3']
</code></pre>
<p>I want to get a result like this : </p>
<pre><code>C = ['A1','B2','C3']
</code></pre>
<p>Any ideas to do this in python 3. 
Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip()</code></a> function  within a list comprehension :</p>
<pre><code>&gt;&gt;&gt; ['{}{}'.format(i,j) for i,j in zip(A,B)]
['A1', 'B2', 'C3']
</code></pre>
<p>Or with <code>str.join()</code> :</p>
<pre><code>&gt;&gt;&gt; [''.join(i) for i in zip(A,B)]
['A1', 'B2', 'C3']
</code></pre>
</div>
