<div class="post-text" itemprop="text">
<p>I would first point out some mistakes you have made in your code. I'm hoping this will help you to understand where you have gone wrong.</p>
<ol>
<li><p>I'm not sure why you think you can pass the argument <code>default</code> to the <code>datetime</code> constructor. It is not available option. You can see the available arguments in the <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime" rel="nofollow noreferrer">documentation</a></p></li>
<li><p><code>strftime</code> is a class function returns a new <code>datetime</code> object. If you call it on an existing <code>datetime</code> object like you do in your example, it does not modify the existing object.</p></li>
<li><p>You are specifying the date format string as a f string. This is not necessary.</p></li>
</ol>
<p>So here is an example of how you could do this:</p>
<pre><code>from datetime import datetime

input_string = '05/13/1982'
d = datetime.strptime(input_string, '%m/%d/%Y')

print(repr(d))
# prints datetime.datetime(1982, 5, 13, 0, 0)

# d is a datetime object. If you want it as a unix timestamp int, call .timestamp()
print(d.timestamp())
# prints 390088800.0
</code></pre>
