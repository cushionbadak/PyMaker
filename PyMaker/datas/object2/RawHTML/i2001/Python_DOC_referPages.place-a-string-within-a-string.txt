<div class="post-text" itemprop="text">
<p>I want to do something along the lines of </p>
<pre><code>print("hello, your name is [name]")
</code></pre>
<p>but I don't want to do it like </p>
<pre><code>print("hello, your name is "+name) 
</code></pre>
<p>because I want to be able to place <code>[name]</code> anywhere in the string. </p>
<p>Is there some way to do this in python?</p>
</div>
<div class="post-text" itemprop="text">
<p>From python3.6 you can use f string:</p>
<pre><code>print(f'your name is {name}')
</code></pre>
<p>Or you can use format:</p>
<pre><code>print('your name is {}'.format(name))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Option 1, old style formatting.</p>
<pre><code>&gt;&gt;&gt; name = 'Jake'
&gt;&gt;&gt; print('hello %s, have a great time!' % name)
hello Jake, have a great time!
</code></pre>
<p>Option 2, using <code>str.format</code>.</p>
<pre><code>&gt;&gt;&gt; print('hello {}, have a great time!'.format(name))
hello Jake, have a great time!
</code></pre>
<p>Option 3, string concatenation.</p>
<pre><code>&gt;&gt;&gt; print('hello ' + name + ', have a great time!')
hello Jake, have a great time!
</code></pre>
<p>Option 4, format strings (as of Python 3.6).</p>
<pre><code>&gt;&gt;&gt; print(f'hello {name}, have a great time!')
hello Jake, have a great time!
</code></pre>
<p>Option 2 and 4 are the <a href="https://stackoverflow.com/questions/5082452/python-string-formatting-vs-format">preferred</a> ones. For a good overview of Python string formatting, check out <a href="https://pyformat.info" rel="nofollow noreferrer">pyformat.info</a>.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>print(f"hello, your name is {name}")
</code></pre>
<p>It is called an f-string: <a href="https://docs.python.org/3/reference/lexical_analysis.html#f-strings" rel="nofollow noreferrer">https://docs.python.org/3/reference/lexical_analysis.html#f-strings</a></p>
<p>There are other methods as well.</p>
</div>
<div class="post-text" itemprop="text">
<p>This is called string formatting: <code>print('Hello, my name is {}'.format(name))</code></p>
<p>You can do something more complex as well:</p>
<pre><code>print('Hello, my name is {0}, and here is my name again {0}'.format(name)'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>print("hello, your name is %s" % ('name'))</code> also works for you.</p>
<p>You can extend it to more variables like:</p>
<pre><code>&gt;&gt;&gt; print("%s, your name is %s" % ('hello', 'name'))
hello, your name is name
</code></pre>
</div>
