<div class="post-text" itemprop="text">
<p>Simply put a <code>printed</code> bool:</p>
<pre><code>printed = False
while True:
    if not printed:
        print('Hello World!')
        printed = True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can set a flag</p>
<pre><code>flag = True

while True:
    print("I have picked a number between 1 and 100 you can try to guess it.")
    if flag:
        print("only")
        flag = False
</code></pre>
</div>
