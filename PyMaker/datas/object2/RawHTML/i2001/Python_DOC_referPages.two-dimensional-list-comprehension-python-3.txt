<div class="post-text" itemprop="text">
<p>You are actually trying to round the numbers in list. For this, use <a href="https://docs.python.org/3/library/functions.html#round" rel="nofollow noreferrer"><code>round()</code></a> with a list-comprehension:</p>
<pre><code>lst = [[0.5 , 0.1, 0.75],
       [0.5 , 0.1, 0.75],
       [0.5 , 0.1, 0.75]]

print([[round(y) for y in x] for x in lst])
# [[0, 0, 1],
#  [0, 0, 1],
#  [0, 0, 1]]
</code></pre>
<p>Or using <code>map</code> + list-comprehension:</p>
<pre><code>print([list(map(round, x)) for x in lst])
</code></pre>
</div>
