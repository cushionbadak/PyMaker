<p>I this include microseconds and milliseconds. So I can divide it by /1000 to lose the accuracy of microseconds, but I want to store milliseconds too.</p>

<div class="post-text" itemprop="text">
<p>A quick workaround:</p>
<pre><code>timestamp = 1539965545566873
time, micro = divmod(timestamp, 1000000)
timestamp = datetime.utcfromtimestamp(time).strftime('%Y-%m-%d %H:%M:%S.')
timestamp += str(micro)

timestamp
&gt;&gt;&gt;'2018-10-19 16:12:25.566873'
</code></pre>
<p>AFAIK <code>utcfromtimestamp</code> does not have a parameter that allows you to pass in microseconds.</p>
