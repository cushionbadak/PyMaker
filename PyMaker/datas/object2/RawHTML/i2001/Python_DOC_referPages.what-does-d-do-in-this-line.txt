<div class="post-text" itemprop="text">
<p>It takes the value that is after your string and places it into the string at the %d. The first %d gets the first value, and the second gets the second and so on...</p>
<p>So it prints like this. 
"and Puppy (whatever j+1 is) (User (whatever I +1 is) ) "</p>
