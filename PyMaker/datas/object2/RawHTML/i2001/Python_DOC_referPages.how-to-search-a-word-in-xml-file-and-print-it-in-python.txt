<p>the error happens because you are passing an elementtree object to the fromstring () methods. Do like this:</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; import xml.etree.ElementTree as etree
&gt;&gt;&gt; a = etree.parse('a.xml')
&gt;&gt;&gt; a
&lt;xml.etree.ElementTree.ElementTree object at 0x10fcabeb8&gt;
&gt;&gt;&gt; b = a.getroot()
&gt;&gt;&gt; b
&lt;Element 'words' at 0x10fb21f48&gt;
&gt;&gt;&gt; b[0][0].text
'John'
</code></pre>
<p>Use find() and findall() methods to search.</p>
<p>for more info, check lib: <a href="https://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow noreferrer">https://docs.python.org/3/library/xml.etree.elementtree.html</a></p>
<p>Simple example:</p>
<p><strong>test.xml</strong></p>
<pre><code>&lt;?xml version="1.0" encoding="UTF-8"?&gt;
&lt;words&gt;
  &lt;word value="John"&gt;&lt;/word&gt;
  &lt;word value="Mike"&gt;&lt;/word&gt;
  &lt;word value="Scott"&gt;&lt;/word&gt;
&lt;/words&gt;
</code></pre>
