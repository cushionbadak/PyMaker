<p>If the input is </p>
<pre><code>d1={'A':5,'B':6,'C':8}
d2={'A':4,'B':9,'E':2}
</code></pre>
<p>I want the output to be</p>
<pre><code>d3={'A':9,'B':15,'C':8,'E':2}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>collections.Counter</code></a> for counting:</p>
<pre><code>from collections import Counter

d3 = Counter(d1) + Counter(d2)

Counter({'A': 9, 'B': 15, 'C': 8, 'E': 2})
</code></pre>
<p>Since <code>Counter</code> is a subclass of <code>dict</code>, you will likely not want to convert this explicitly to a regular <code>dict</code>. There are some limitations, namely <code>Counter</code> works only with positive integers.</p>
</div>
<div class="post-text" itemprop="text">
<p>I would do it like this:</p>
<pre><code>&gt;&gt;&gt; fst = {'A': 5, 'B': 6, 'C': 8}
&gt;&gt;&gt; snd = {'A': 4, 'B': 9, 'E': 2}
&gt;&gt;&gt; out = {k: fst[k] + snd[k] for k in fst.keys() &amp; snd.keys()}
&gt;&gt;&gt; out.update({k: fst[k] if k in fst else snd[k] for k in fst.keys() ^ snd.keys()})
&gt;&gt;&gt; out
{'A': 9, 'B': 15, 'C': 8, 'E': 2}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>d1.update(d2)</code>
This will update d1 with the contents of d2</p>
<p>Or </p>
<pre><code>d1={'A':5,'B':6,'C':8} 
d2={'A':4,'B':9,'E':2}
d3 = {**d1, **d2}
print(d3)
</code></pre>
<p>This will merge the contents of multiple dictionaries and create a new one.</p>
