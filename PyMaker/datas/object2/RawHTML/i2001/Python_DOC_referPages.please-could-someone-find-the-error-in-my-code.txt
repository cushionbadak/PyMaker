<div class="post-text" itemprop="text">
<p>It is easier to build up than descend given you want to efficiently calculate this factorial sum. E.g. <code>f</code> is the increasing factorial, <code>t</code> is the total of those factorials:</p>
<pre><code>def fac_sum(n):
    t, f = 0, 1
    for x in range(1, n+1):
        f *= x
        t += f
    return t

In []:
fac_sum(3)

Out[]:
9
</code></pre>
<p>Note: you can use <a href="https://docs.python.org/3/library/itertools.html#itertools.accumulate" rel="nofollow noreferrer"><code>itertools.accumulate()</code></a> to simplify:</p>
<pre><code>import itertools as it
import operator as op

def fac_sum(n):
    return sum(it.accumulate(range(1, n+1), op.mul))
</code></pre>
