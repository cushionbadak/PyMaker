<p>I believe you need <a href="https://docs.python.org/3/library/glob.html" rel="nofollow noreferrer"><code>glob</code></a> for list of files, then loop and create DataFrame, <code>map</code> and last save back:</p>
<pre><code>import glob

s = replace.set_index('Snack')['Cat']

for file in glob.glob('files/*.csv'):
    #df = pd.read_csv(file)
    df['Snack'] = df['Snack'].map(s)
    df.to_csv(f'{file}', index=False)
</code></pre>
