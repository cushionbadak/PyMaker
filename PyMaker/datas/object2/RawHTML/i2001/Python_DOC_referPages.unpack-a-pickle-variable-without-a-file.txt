<p>To unpickle from a bytes object, use <a href="https://docs.python.org/3/library/pickle.html#pickle.loads" rel="nofollow noreferrer"><code>pickle.loads</code></a>:</p>
<pre><code>In [3]: import pickle
   ...:
   ...: class Foo:
   ...:    attr = 'a class attr'
   ...:
   ...: picklestring = pickle.dumps(Foo)
   ...:
   ...: print(picklestring)
   ...:
b'\x80\x03c__main__\nFoo\nq\x00.'

In [4]: pickle.loads(picklestring)
Out[4]: __main__.Foo
</code></pre>
