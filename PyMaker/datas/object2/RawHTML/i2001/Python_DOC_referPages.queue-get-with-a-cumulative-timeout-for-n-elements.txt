<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/queue.html#queue.Queue.get" rel="nofollow noreferrer" title="queue.Queue.get"><code>queue.Queue.get</code></a> method has a <code>timeout</code> parameter to wait for the presence of one element to retrieve.</p>
<p>I would like to retrieve "at most" 25 elements "at once", waiting "at most" 5 seconds. How can I achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Simply calculate the remaining time(out).</p>
<pre><code>TIMEOUT=5

t = time.monotonic()+TIMEOUT
items = []
for n in range(25):
    try:
        items.append(q.get(timeout=t-time.monotonic()))
    except queue.Empty:
        break
</code></pre>
