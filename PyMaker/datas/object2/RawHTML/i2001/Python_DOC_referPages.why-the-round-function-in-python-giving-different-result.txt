<div class="post-text" itemprop="text">
<p>See <a href="https://docs.python.org/3/library/functions.html#round" rel="nofollow noreferrer">official doc</a>.</p>
<p>Basically it called even round. It will round <code>x.5</code> value into even choice.</p>
<p>And here is <a href="https://en.wikipedia.org/wiki/Rounding#Round_half_to_even" rel="nofollow noreferrer">why</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>From <a href="https://docs.python.org/3/library/functions.html#round" rel="nofollow noreferrer">docs</a>:</p>
<blockquote>
<p>if two multiples are equally close, rounding is done toward the even choice (so, for example, both round(0.5) and round(-0.5) are 0, and round(1.5) is 2).</p>
</blockquote>
<p>I.e. Python 3 does <a href="https://en.wikipedia.org/wiki/Rounding#Round_half_to_even" rel="nofollow noreferrer">"rounding half to even"</a>.</p>
<p>You can do "round up" this way:</p>
<pre><code>import math
def round_up(x):
    return math.floor(x + 0.5)
</code></pre>
