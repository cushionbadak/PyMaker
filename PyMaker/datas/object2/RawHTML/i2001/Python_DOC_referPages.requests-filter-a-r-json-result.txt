<div class="post-text" itemprop="text">
<p>In Python, JSON practically works just like a <a href="https://docs.python.org/3/library/stdtypes.html#typesmapping" rel="nofollow noreferrer">dictionary</a>, so you can use the square brackets, <code>[]</code>, to filter it. See below.</p>
<pre><code>import requests

r = requests.get('https://REDACTED.zportal.nl/api/v3/appointments?user=~me&amp;start=1542006900&amp;end=1542009900&amp;access_token=REDACTED')

locations = r.json()["response"]["data"][0]["locations"]

print(locations)
</code></pre>
