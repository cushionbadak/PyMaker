<div class="post-text" itemprop="text">
<p>For space:</p>
<pre><code>print(str(b), f1)
</code></pre>
<p>For printing with a newline between b and f1:</p>
<pre><code>print(str(b) + '\n' + f1)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The most basic way would be to use string concatenation:</p>
<pre><code>print(str(b) + ' ' + f1)
</code></pre>
<p>But note the default <code>sep</code> parameter for <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>print</code></a> is a single whitespace. So you can just use:</p>
<pre><code>print(str(b), f1)            # single space separator
print(str(b), f1, sep='\n')  # new line separator
</code></pre>
<p>With Python 3.6+ you can use formatted string literals (<a href="https://www.python.org/dev/peps/pep-0498/" rel="nofollow noreferrer">PEP 498</a>):</p>
<pre><code>print(f'{b} {f1}')           # single space separator
print(f'{b}\n{f1}')          # new line separator
</code></pre>
