<div class="post-text" itemprop="text">
<p>I have a list of different polynoms created with np.poly1d: </p>
<pre><code> polys = [p1,p2,...,pn]
</code></pre>
<p>and I want to know the values that take these polynoms for specific parameters:</p>
<pre><code> xs = [8.1, 2.3, ..., 4.2] #nvalues
</code></pre>
<p>The results I want is:</p>
<pre><code> ys = [p1(8.1), p2(2.3), ..., pn(4.2)] #I want the numeric values.
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>[poly(x) for (poly, x) in zip(polys, xs)]
</code></pre>
<p>I think what you want to do is <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer" title="zip">zip</a> the two lists together, then apply the function in the first list to the value in the second.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can simply use <code>zip</code> to obtain an iterable of <code>(function, value)</code> pairs such that you can call <code>function(value)</code>.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; polys = [lambda x: x, lambda x: x**2, lambda x: x**3]
&gt;&gt;&gt; xs = [2, 4, 6]
&gt;&gt;&gt; 
&gt;&gt;&gt; ys = [f(x) for f, x in zip(polys, xs)]
&gt;&gt;&gt; ys
[2, 16, 216]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>ys = []
for p, val in zip(polys, xs):
    ys.append(p(val))
</code></pre>
