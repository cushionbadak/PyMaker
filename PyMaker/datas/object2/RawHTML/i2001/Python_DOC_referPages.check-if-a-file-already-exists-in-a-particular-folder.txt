<p>You can check whether a <code>path</code> exists or not by using <code>os.path.exists</code> method, it returns <code>True</code> if the path passed as a parameter exists and returns <code>False</code> if it doesn't.</p>
<p>For instance: If you want to check if there is any file named <code>hello.c</code> in the current working directory, then you can use the following code:</p>
<pre><code>from os import path
path.exists('hello.c') # This will return True if hello.c exists and will return False if it doesn't
</code></pre>
<p>Or You can use the absolute path too</p>
<p>For instance: If you want to check if there is any folder named <code>Users</code> in your C drive then you can use the following code:</p>
<pre><code>from os import path
path.exists('C:\Users') # This will return True if Users exists in C drive and will return False if it doesn't
</code></pre>
