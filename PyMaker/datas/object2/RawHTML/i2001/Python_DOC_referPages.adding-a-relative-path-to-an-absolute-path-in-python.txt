<p>Assuming I have a absolute path and a relative path</p>
<pre><code>abspath = os.path.abspath(__file__)
relpath = '../../folder/file'
</code></pre>
<p>How do I 'add' these two path together without the <code>../..</code>s, is there a module that does this? I can't find anything so far. I'm thinking of a format like:</p>
<pre><code>mypath = some_module.function(abspath, relpath)
</code></pre>
<p>Instead of doing </p>
<pre><code>mypath = os.path.join(os.path.dirname(os.path.dirname(abspath))), folder, file)
</code></pre>
<p>Which I think is too bothersome.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can <code>join</code> them and use <a href="https://docs.python.org/3/library/os.path.html#os.path.normpath" rel="nofollow noreferrer">normpath</a>:</p>
<pre><code>os.path.normpath(os.path.join(abspath, relpath))
</code></pre>
<p>From <code>normpath</code> doc:</p>
<blockquote>
<p>Normalize a pathname by collapsing redundant separators and up-level
  references so that A//B, A/B/, A/./B and A/foo/../B all become A/B.</p>
</blockquote>
<hr/>
<p>You could also use the <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow noreferrer">pathlib</a> module since Python 3.4:</p>
<pre><code>from pathlib import Path

abspath = Path('/home/xxxx/yyy')
relpath = Path('../../folder/file')
(abspath / relpath).resolve()
# PosixPath('/home/xxxx')
</code></pre>
