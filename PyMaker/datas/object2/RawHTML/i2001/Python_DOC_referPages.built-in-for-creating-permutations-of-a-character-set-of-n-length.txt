<div class="post-text" itemprop="text">
<p>What you're looking for is the cartesian product of 7 <code>["*", "/", "+", "-"]</code> lists. <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="noreferrer"><code>itertools.product</code></a> takes a <code>repeat</code> argument for precisely this purpose:</p>
<pre><code>for row in itertools.product(["*", "/", "+", "-"], repeat=7):
    print(row)
</code></pre>
</div>
