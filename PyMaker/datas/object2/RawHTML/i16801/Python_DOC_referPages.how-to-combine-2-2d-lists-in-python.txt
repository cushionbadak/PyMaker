<p>I am pretty new to python, and I have a little problem.</p>
<p>I need to combine two 2D lists like this:</p>
<pre><code>list1= [[some,1],[thing,5]]
list2= [[some,1],[other,1],[thing,5]]
</code></pre>
<p>result should be like this:</p>
<pre><code>result= [[some,2],[other,1],[thing,10]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow"><code>collections.Counter</code></a>:</p>
<pre><code>&gt;&gt;&gt; from collections import Counter
&gt;&gt;&gt;
&gt;&gt;&gt; list1 = [['some',1],['thing',5]]
&gt;&gt;&gt; list2= [['some',1],['other',1],['thing',5]]
&gt;&gt;&gt;
&gt;&gt;&gt; [[k,v] for k,v in (Counter(dict(list1)) + Counter(dict(list2))).items()]
[['thing', 10], ['other', 1], ['some', 2]]
</code></pre>
<p>Or if a list of tuples is acceptable:</p>
<pre><code>&gt;&gt;&gt; (Counter(dict(list1)) + Counter(dict(list2))).items()
[('thing', 10), ('other', 1), ('some', 2)]
</code></pre>
<p>Using tuples seems to make more sense here.</p>
<p>You should consider if you actually need the final result to be a list. If order is not important (as you say it isn't), then the dictionary <code>Counter(dict(list1)) + Counter(dict(list2))</code> will probably suffice on its own.</p>
<pre><code>&gt;&gt;&gt; Counter(dict(list1)) + Counter(dict(list2))
Counter({'thing': 10, 'some': 2, 'other': 1})
</code></pre>
</div>
