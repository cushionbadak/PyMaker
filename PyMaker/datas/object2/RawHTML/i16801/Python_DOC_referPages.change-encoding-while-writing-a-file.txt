<div class="post-text" itemprop="text">
<p>In <code>Python 3</code>,  according to documentation in :<a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow">https://docs.python.org/3/library/functions.html#open</a><br/>
you can specify the encoding with the encoding parameter.</p>
<pre><code>with open("test_%s.tem" %i, "w", encoding="utf16") as f1:
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>On Python 2 you can use <a href="http://docs.python.org/2/library/codecs.html#codecs.open" rel="nofollow">codecs.open</a> with the <em>encoding</em> parameter.</p>
<p>If you use Python 3 just use the <em>encoding</em> parameter of <a href="http://docs.python.org/3.4/library/functions.html#open" rel="nofollow">open</a>.</p>
<p>UCS-2 is a subset of UTF-16, so try to use <code>'UTF-16'</code> as the parameter value.</p>
