<p>I know that the <code>pprint</code> python standard library is for pretty-printing python data types. However, I'm always retrieving json data, and I'm wondering if there is any easy and fast way to pretty-print json data?</p>
<p>No pretty-printing:</p>
<pre><code>import requests
r = requests.get('http://server.com/api/2/....')
r.json()
</code></pre>
<p>With pretty-printing:</p>
<pre><code>&gt;&gt;&gt; import requests
&gt;&gt;&gt; from pprint import pprint
&gt;&gt;&gt; r = requests.get('http://server.com/api/2/....')
&gt;&gt;&gt; pprint(r.json())
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Python's builtin <a href="https://docs.python.org/3/library/json.html">JSON module</a> can handle that for you:</p>
<pre><code>&gt;&gt;&gt; import json
&gt;&gt;&gt; a = {'hello': 'world', 'a': [1, 2, 3, 4], 'foo': 'bar'}
&gt;&gt;&gt; print(json.dumps(a, indent=2))
{
  "hello": "world",
  "a": [
    1,
    2,
    3,
    4
  ],
  "foo": "bar"
}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>import requests
import json
r = requests.get('http://server.com/api/2/....')
pretty_json = json.loads(r.text)
print (json.dumps(pretty_json, indent=2))
</code></pre>
</div>
