<div class="post-text" itemprop="text">
<p>I want to set predefined window size. Example application:</p>
<pre><code>from tkinter import *

root = Tk()
button = Button(root, text="Example button")
button.pack()

root.mainloop()
</code></pre>
<p>The resulting window is very small. It is as big as button. How to make the window large?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use geometry.</p>
<pre><code>from tkinter import *

root = Tk()
button = Button(root, text="Example button")
button.pack()

root.geometry("200x200")

root.mainloop()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/tkinter.html#tk-option-data-types" rel="nofollow"><code>tkinter.Tk.geometry</code></a> method:</p>
<pre><code>from tkinter import *

root = Tk()
root.geometry("500x500")  # Set the window's size to 500 by 500 pixels

button = Button(root, text="Example button")
button.pack()

root.mainloop()
</code></pre>
</div>
