<div class="post-text" itemprop="text">
<p>I know I can write a CSV file with something like:</p>
<pre><code>with open('some.csv', 'w', newline='') as f:
</code></pre>
<p>How would I instead write that output to STDOUT?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/sys.html#sys.stdout" rel="noreferrer"><code>sys.stdout</code></a> is a file object corresponding to the program's standard output. You can use its <code>write()</code> method. Note that it's probably not necessary to use the <code>with</code> statement, because <code>stdout</code> does not have to be opened or closed.</p>
<p>So, if you need to create a <code>csv.writer</code> object, you can just say:</p>
<pre><code>import sys
spamwriter = csv.writer(sys.stdout)
</code></pre>
</div>
<span class="comment-copy">On Windows, this results in an extra carriage return character after each row.</span>
<span class="comment-copy">You can give the csv.writer constructor a lineterminator option:<code>writer = csv.writer(sys.stdout, lineterminator=os.linesep)</code></span>
