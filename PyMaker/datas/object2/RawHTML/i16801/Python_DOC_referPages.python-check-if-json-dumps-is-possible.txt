<div class="post-text" itemprop="text">
<p>There is no way to check if an object is serializable to a <code>json</code> format. However, you can (even if it is not really <a href="https://docs.python.org/2.7/glossary.html#term-eafp" rel="nofollow noreferrer"><em>EAFP</em> compliant</a>) try and if it fails go for a fallback process:</p>
<pre><code>try:
    json.dumps(my_object)
except TypeError:
    print("Unable to serialize the object")
</code></pre>
<p>A way to be sure would be to <a href="https://docs.python.org/3/library/json.html#encoders-and-decoders" rel="nofollow noreferrer">implement your own <code>JSONEncoder</code></a> if you are entirely sure of the data processed.</p>
</div>
