<p>What is the logic behind</p>
<pre><code>     int("string", integer)
</code></pre>
<p>for example:</p>
<pre><code>     int("220", 3)
</code></pre>
<p>yields </p>
<pre><code>      24
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The optional integer is the numeric base to use in converting the string (defaults to base 10).</p>
<pre><code>220 base 3 = 2 * (3**2) + 2 * (3**1) + 0 * (3**0)
           = 2*9 + 2*3 + 0*1
           = 18 + 6 + 0
           = 24
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>the int() method in python takes two inputs. First is the string you want to evaluate the second is the base.</p>
<p>In your example the base is three therefore:</p>
<pre><code>1*0 = 0
3*2 = 6
9*2 = 18

0 + 6 + 18 = 24
</code></pre>
</div>
