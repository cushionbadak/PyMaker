<p>I am trying to create a simple program that prints the day month and year in mm/dd/yy form but I keep getting a trace-back error:</p>
<pre><code>%s/%s/%s
Traceback (most recent call last):
  File "C:/Python34/timenow.py", line 4, in &lt;module&gt;
    print('%s/%s/%s')%(now.month, now.day, now.year)
TypeError: unsupported operand type(s) for %: 'NoneType' and 'tuple'
</code></pre>
<p>I'm using python 3.4.0</p>
<p>Here's my code:</p>
<pre><code>from datetime import datetime
now = datetime.now()

print('%s/%s/%s')%(now.month, now.day, now.year)
</code></pre>

<div class="post-text" itemprop="text">
<p>You need to place the <code>%</code> operator just after the format string:</p>
<pre><code>print('%s/%s/%s' % (now.month, now.day, now.year))
</code></pre>
<p>Your current code has Python trying to use it on the <code>None</code> object returned by <code>print</code>:</p>
<pre><code>&gt;&gt;&gt; type(print('hi'))
hi
&lt;class 'NoneType'&gt;
&gt;&gt;&gt;
</code></pre>
<hr/>
<p>Also, I would like to mention two things.  The first is that the modern approach for string formatting is to use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format</code></a> rather than the <code>%</code> operator:</p>
<pre><code>print('{}/{}/{}'.format(now.month, now.day, now.year))
</code></pre>
<p>The second is that string formatting is not strictly needed here because we have the <code>print</code> function's <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow"><code>sep</code> parameter</a>, which is used to separate the arguments given to <code>print</code> by a certain string:</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; now = datetime.now()
&gt;&gt;&gt; print(now.month, now.day, now.year, sep='/')
5/20/2014
&gt;&gt;&gt;
</code></pre>
