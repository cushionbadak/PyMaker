<pre><code>Wed, 23 July 2014 19:25:52 GMT

<pre><code>date = 23/07/2014

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/datetime.html?highlight=strptime#datetime.datetime.strptime" rel="nofollow"><code>datetime.strptime</code></a> to parse the date from string and then <a href="https://docs.python.org/3/library/datetime.html?highlight=datetime.datetime#strftime-and-strptime-behavior" rel="nofollow">format</a> it using the <a href="https://docs.python.org/3/library/datetime.html?highlight=strptime#datetime.date.strftime" rel="nofollow"><code>strftime</code></a> method of <a href="https://docs.python.org/3/library/datetime.html?highlight=datetime.datetime#datetime.datetime" rel="nofollow"><code>datetime</code> objects</a>:</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; dt = datetime.strptime("Wed, 23 July 2014 19:25:52 GMT", "%a, %d %B %Y %H:%M:%S %Z")
&gt;&gt;&gt; dt.strftime('%d/%m/%Y')
'23/07/2014'
&gt;&gt;&gt; dt.strftime('%H/%M/%S')
'19/25/52'
</code></pre>
<p>But if you're okay with the ISO format you can call <code>date</code> and <code>time</code> methods:</p>
<pre><code>&gt;&gt;&gt; str(dt.date())
'2014-07-23'
&gt;&gt;&gt; str(dt.time())
'19:25:52'
</code></pre>
