<p>Let me clear two points here :</p>
<ul>
<li>firstly non-default argument should not follow default argument , it means you can't define (a="b",c) in function the order of defining parameter in function are :

<ul>
<li>positional parameter or non-default parameter  i.e (a,b,c)</li>
<li>keyword parameter or default parameter         i.e (a="b",r="j")</li>
<li>keyword-only parameter                         i.e (*args)</li>
<li>var-keyword  parameter                         i.e (**kwargs)</li>
</ul></li>
</ul>
<p><code>def example(a, b, c=None, r="w" , d=[], *ae,  **ab):</code></p>
<p>(a,b) are positional parameter</p>
<p>(c=none) is optional parameter</p>
<p>(r="w") is keyword parameter </p>
<p>(d=[]) is list parameter </p>
<p>(*e) is keyword-only</p>
<p>(*opts) is var-keyword parameter</p>
<p>so first re-arrange your parameters</p>
<ul>
<li>now secondary thing is you have define len1 when you are doing hgt=len1
the len1 argument is not defined when default values are saved,Python computes and saves default values when you define the function
len1 is not defined, does not exist, when this happens (it exists only when the function is executed) </li>
</ul>
<p>so second remove this "len1=hgt" its not allowed in python.</p>
<p>keep in mind difference between argument and parameters , you can read more about here : <a href="http://www.cryptroix.com/?p=67" rel="noreferrer">Arguments and parameters in python</a></p>

<p><strong>UPDATE</strong></p>
<p>Another issue that is hidden by the SyntaxError.</p>
<p><a href="https://docs.python.org/2/library/os.html#os.system"><code>os.system</code></a> accepts only one string parameter.</p>
<pre><code>def a(len1, hgt, til, col=0):
    system('mode con cols=%s lines=%s' % (len1, hgt))
    system('title %s' % til)
    system('color %s' % col)
</code></pre>

<p>You can't have a non-keyword argument after a keyword argument.</p>
<p>Make sure you re-arrange your function arguments like so:</p>
<pre><code>def a(len1,til,hgt=len1,col=0):
    system('mode con cols='+len1,'lines='+hgt)
    system('title',til)
    system('color',col)
