<div class="post-text" itemprop="text">
<p>I have a file that contains one line </p>
<pre><code>car 
</code></pre>
<p>I want to append a new line to the file </p>
<p>when I  used this code: </p>
<pre><code>f = open("motors.txt","a")
f.write("Bicycle")
</code></pre>
<p>I got the following:</p>
<pre><code>carBicycle
</code></pre>
<p>how could I append the new word to a new line</p>
</div>
<div class="post-text" itemprop="text">
<p>Prefix <code>Bicycle</code> with a <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow">newline character</a> (<code>\n</code>) to have it write on a new line:</p>
<pre><code>f.write("\nBicycle")
</code></pre>
</div>
<span class="comment-copy">what if Bicycle the first word I want to append in the file?</span>
<span class="comment-copy">In that case, put the newline character at the end of each string that you want to write: <code>f.write("Bicycle\n")</code>.</span>
