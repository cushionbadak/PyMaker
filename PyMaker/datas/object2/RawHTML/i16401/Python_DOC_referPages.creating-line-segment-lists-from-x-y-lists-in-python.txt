<div class="post-text" itemprop="text">
<p>In Python 3 when using <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow">zip</a> you need to  use <code>list(zip())</code>:</p>
<pre><code>In [10]: vertices = list(zip(x, y))

In [11]: list(zip(vertices, vertices[1:] + [vertices[0]]))
Out[11]: [((1, 2), (8, 3)), ((8, 3), (9, 6)), ((9, 6), (4, 7)), ((4, 7), (1, 2))]
</code></pre>
<p>In <code>python3</code> <code>zip</code> returns an iterator as opposed to a list in <code>python2</code> so you need to use the <code>list(zip())</code> syntax.</p>
