<div class="post-text" itemprop="text">
<p><code>str.replace</code> lets you specify the number of replacements. Unfortunately there is no <code>str.rreplace</code>, so you'd need to reverse the string before and after :) eg.</p>
<pre><code>&gt;&gt;&gt; def f(s):
...     return s[::-1].replace(".", "_", 1)[::-1]
... 
&gt;&gt;&gt; f('I.like.football')
'I.like_football'
&gt;&gt;&gt; f('sky.is.blue')
'sky.is_blue'
</code></pre>
<p>Alternatively you could use one of <code>str.rpartition</code>, <code>str.rsplit</code>, <code>str.rfind</code></p>
</div>
<div class="post-text" itemprop="text">
<p>This doesn't even need to run in a loop:</p>
<pre><code>import re
p = re.compile(ur'\.(?=[^\.]+$)', re.IGNORECASE | re.MULTILINE)
test_str = u"I.like.football\nsky.is.blue"
subst = u"_"

result = re.sub(p, subst, test_str)
</code></pre>
