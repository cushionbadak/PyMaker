<p>You can use the <a href="https://docs.python.org/library/os.html#os.listdir" rel="nofollow"><code>os.listdir()</code></a> function to get a list of the files in a directory.</p>

<div class="post-text" itemprop="text">
<p>So, here's how to put all the files in a directory and in all its sub-directories in a list, print out the list.</p>
<p>Note: How to 'walk' was found here: </p>
<p><a href="https://stackoverflow.com/a/2909998/5082676"> concatenate the directory and file name</a></p>
<pre><code># Task: Get a printout of all the files in a directory.

import os

# The directory that we are interested in
myPath = "/users/george/documents/"

# All the file paths will be stored in this list
filesList= []

for path, subdirs, files in os.walk(myPath):
    for name in files:
        filesList.append(os.path.join(path, name))

for i in filesList:
    print (str(i))
</code></pre>
