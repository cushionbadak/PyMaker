<div class="post-text" itemprop="text">
<p>I have two lists :</p>
<pre><code>name = ['A', 'B', 'C']
sex = ['M', 'F', 'M']
</code></pre>
<p>What is the best way with good performance to create a list like this from these two:</p>
<pre><code>comb = [['A', 'M'], ['B', 'F'], ['C', 'M']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/2/library/functions.html#zip" rel="nofollow"><code>zip</code></a>:</p>
<pre><code>&gt;&gt;&gt; name = ['A', 'B', 'C']
&gt;&gt;&gt; sex = ['M', 'F', 'M']
&gt;&gt;&gt; zip(name, sex)
[('A', 'M'), ('B', 'F'), ('C', 'M')]
</code></pre>
<p>If you want items to be lists, you can use <a href="https://docs.python.org/2/library/functions.html#map" rel="nofollow"><code>map</code></a> or list comprehension to convert them:</p>
<pre><code>&gt;&gt;&gt; map(list, zip(name, sex))
[['A', 'M'], ['B', 'F'], ['C', 'M']]
&gt;&gt;&gt; [list(x) for x in zip(name, sex)]
[['A', 'M'], ['B', 'F'], ['C', 'M']]
</code></pre>
<p>NOTE: If you are using Python 3.x, <code>zip</code>, <code>map</code> return an iterator instead of a list. To get a list pass the returned value to <a href="https://docs.python.org/3/library/functions.html#func-list" rel="nofollow"><code>list</code> function</a>:</p>
<pre><code>list(zip(name, sex))
</code></pre>
</div>
<span class="comment-copy">You just <code>zip()</code> them.</span>
