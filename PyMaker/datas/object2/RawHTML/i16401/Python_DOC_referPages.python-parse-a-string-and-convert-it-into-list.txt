<div class="post-text" itemprop="text">
<p>I've a string</p>
<pre><code>a = "sequence=1,2,3,4&amp;counter=3,4,5,6"
</code></pre>
<p>How do I convert it into a list,ie</p>
<pre><code>sequence = [1,2,3,4]
counter = [3,4,5,6]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/2/library/urlparse.html#urlparse.parse_qs"><code>urlparse.parse_qs</code></a> (<a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.parse_qs"><code>urllib.parse.parse_qs</code></a> in Python 3.x) to parse query string:</p>
<pre><code>&gt;&gt;&gt; import urlparse
&gt;&gt;&gt; a = "sequence=1,2,3,4&amp;counter=3,4,5,6"
&gt;&gt;&gt; {key: [int(x) for x in value[0].split(',')]
...  for key, value in urlparse.parse_qs(a).items()}
{'counter': [3, 4, 5, 6], 'sequence': [1, 2, 3, 4]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>import re

items = "sequence=1,2,3,4&amp;counter=3,4,5,6".split('&amp;')

pattern = re.compile(r'\d+')

for i in items:
    print [int(i) for i in re.findall(pattern, i)]
</code></pre>
