<p>It's possible. You need to do it either by rendering the address on the response body or by requesting it with ajax after the response has already been rendered. </p>
<p>It would be hard to give you a code solution without seeing what web server you are using, but here are a couple of pointers for the first approach. To obtain the address, on the server side (python handler):</p>
<pre><code>import socket
ip = socket.gethostbyname(socket.gethostname())
</code></pre>
<p>or if you are using something like Google App Engine:</p>
<pre><code>ip = self.request.remote_addr
</code></pre>
<p>you should then write the IP to the response. For example, if you are using a templating engine to render your HTML, your HTML can look like something similar to this:</p>
<pre><code>&lt;html&gt;
&lt;script&gt;
var ip = {{ip}}
&lt;/script&gt;
</code></pre>
<p>and on the python code that renders the template you should do something like that:</p>
<pre><code>htmlContent = template.render(ip=ip)
self.response.write(htmlContent)
</code></pre>
