<div class="post-text" itemprop="text">
<p>I have a dictionary.</p>
<pre><code>dict = {'A':['a', 'b'], 'B':['c', 'b', 'a'], 'C':['d', 'c'], }
</code></pre>
<p>what is easy way to find out similar values from keys of dictionary?</p>
<pre><code>output : 

A&amp;B : 'a', 'b'
A&amp;C : None
B&amp;C : 'c'
</code></pre>
<p>How this can be achieved?</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/library/stdtypes.html#set.intersection" rel="nofollow"><code>set &amp; other_set</code> operator or <code>set.intersection</code></a> and <a href="https://docs.python.org/3/library/itertools.html#itertools.combinations" rel="nofollow"><code>itertools.combinations</code></a>:</p>
<pre><code>&gt;&gt;&gt; import itertools
&gt;&gt;&gt;
&gt;&gt;&gt; d = {'A':['a', 'b'], 'B':['c', 'b', 'a'], 'C':['d', 'c'], }
&gt;&gt;&gt; for a, b in itertools.combinations(d, 2):
...     common = set(d[a]) &amp; set(d[b])
...     print('{}&amp;{}: {}'.format(a, b, common))
...
A&amp;C: set()
A&amp;B: {'b', 'a'}
C&amp;B: {'c'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>In [1]: dct = {'A':['a', 'b'], 'B':['c', 'b', 'a'], 'C':['d', 'c'], }

In [2]: set(dct['A']).intersection(dct['B'])
Out[2]: {'a', 'b'}

In [3]: set(dct['A']).intersection(dct['C'])
Out[3]: set()

In [4]: set(dct['B']).intersection(dct['C'])
Out[4]: {'c'}
</code></pre>
</div>
<span class="comment-copy">As your list contents are hashable, try e.g. <code>set(dict['A']).intersection(dict['B'])</code>. You shouldn't call your own object <code>dict</code>, though.</span>
<span class="comment-copy">and what if I want to get differences?</span>
<span class="comment-copy">Use <code>set(..) - set(..)</code> or <code>set(..).difference(set(..))</code>. See <a href="https://docs.python.org/3/library/stdtypes.html#set.difference" rel="nofollow noreferrer"><code>set.difference</code></a></span>
