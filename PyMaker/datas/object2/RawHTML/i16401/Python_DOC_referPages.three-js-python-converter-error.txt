<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/2/library/functions.html#xrange" rel="nofollow"><code>xrange</code> is a Python2 function</a> which has been <a href="https://docs.python.org/3/whatsnew/3.0.html#views-and-iterators-instead-of-lists" rel="nofollow">removed from Python3</a>. The error </p>
<pre><code>NameError: name 'xrange' is not defined
</code></pre>
<p>suggests you have installed Python3 while <code>convert_obj_three.py</code> requires Python2.</p>
