<p>In Ruby I can do this:</p>
<pre><code>"This is a string with the value of #{variable} shown."
</code></pre>
<p>How do I do that same thing in Python?</p>
</div>
<div class="post-text" itemprop="text">
<p>You have a lot of options.</p>
<pre><code>"This is a string with the value of " + str(variable) + " shown."

"This is a string with the value of %s shown." % (str(variable))

"This is a string with the value of {0} shown.".format(variable)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The modern/preferred way is to use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format</code></a>:</p>
<pre><code>"This is a string with the value of {} shown.".format(variable)
</code></pre>
<p>Below is a demonstration:</p>
<pre><code>&gt;&gt;&gt; 'abc{}'.format(123)
'abc123'
&gt;&gt;&gt;
</code></pre>
<p>Note that in Python versions before 2.7, you need to explicitly number the format fields:</p>
<pre><code>"This is a string with the value of {0} shown.".format(variable)
</code></pre>
</div>
