<p>Web app is breaking on the following line;</p>
<pre><code>start_time = int(time.mktime(start)) * 1000
</code></pre>
<p>The error is a TypeError - argument must be 9-item sequence, not datetime.datetime</p>
<p>How to I convert my datetime.datetime to a 9-item-sequence?</p>
</div>
<div class="post-text" itemprop="text">
<p>Instead of passing <code>datetime</code> object directory, use <a href="https://docs.python.org/2/library/datetime.html#datetime.date.timetuple" rel="noreferrer"><code>datetime.timetuple</code></a> method:</p>
<pre><code>&gt;&gt;&gt; int(time.mktime(start)) * 1000
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: argument must be 9-item sequence, not datetime.datetime

&gt;&gt;&gt; int(time.mktime(start.timetuple())) * 1000
1406215043000L
</code></pre>
<p>BTW, if you use Python 3.3+, you don't need to use <code>time.mktime</code>. Use <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.timestamp" rel="noreferrer"><code>datetime.timestamp</code></a>:</p>
<pre><code>&gt;&gt;&gt; start.timestamp()
1406215043.0
&gt;&gt;&gt; int(start.timestamp()) * 1000
1406215043000
</code></pre>
