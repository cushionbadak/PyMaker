<div class="post-text" itemprop="text">
<p>I have a list like this <br/><code>['1000', '6', '1', '5', '14', '34', '43', '3', '18', '28', '16']</code><br/><br/> I want like this <br/><code>[1000, 6, 1, 5, 14, 34, 43, 3, 18, 28, 16]</code> <br/>How to do this in python.</p>
</div>
<div class="post-text" itemprop="text">
<p>If I understand you right you have list of strings and want to get list of integers.
This can be easily done by:</p>
<pre><code>lst = [int(item) for item in lst]
</code></pre>
<p>For example:</p>
<pre><code>&gt;&gt;&gt; lst = ['1000', '6', '1', '5', '14', '34', '43', '3', '18', '28', '16']
&gt;&gt;&gt; lst
['1000', '6', '1', '5', '14', '34', '43', '3', '18', '28', '16']
&gt;&gt;&gt; lst = [int(item) for item in lst]
&gt;&gt;&gt; lst
[1000, 6, 1, 5, 14, 34, 43, 3, 18, 28, 16]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can convert each element into an Integer.</p>
<pre><code>&gt;&gt;&gt; lst_string = ['1000', '6', '1', '5', '14', '34', '43', '3', '18', '28', '16']
&gt;&gt;&gt; lst_int = [int(n) for n in lst_string]
&gt;&gt;&gt; print lst_int
[1000, 6, 1, 5, 14, 34, 43, 3, 18, 28, 16]
</code></pre>
<p>Or you can use the <code>map</code>-function which performs a <strong>function</strong> on a <strong>sequence</strong>. In this case we perform the <code>int</code>-function to your list.</p>
<pre><code>&gt;&gt;&gt; map(int, lst_string)
[1000, 6, 1, 5, 14, 34, 43, 3, 18, 28, 16]
</code></pre>
</div>

<span class="comment-copy">You're using python3. In python3 <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer">map</a> returns iterator-object, not a list as in python2. So try <code>list(map(int, lst_string))</code></span>
