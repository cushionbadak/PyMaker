<p>I want to generate a random number following a lognormal distribution with Scipy. I understand from the <a href="http://docs.scipy.org/doc/scipy-0.7.x/reference/generated/scipy.stats.lognorm.html" rel="nofollow">documentation</a> that you can use:</p>
<pre><code>import scipy.stats
R = scipy.stats.lognorm.rvs()
</code></pre>
<p>But I want to use specific parameters. I cannot find an example. This is what I want to do:</p>
<pre><code>params = (0.6718101713289113, -3.5861418072007032, 44.598580116376482)
R = scipy.stats.lognorm.rvs(params)
</code></pre>
<p>But this does not work. What is the correct syntax? </p>
</div>
<div class="post-text" itemprop="text">
<p>You only pass one parameter, a tuple, to rvs. You have to unpack the tuple:</p>
<pre><code>R = scipy.stats.lognorm.rvs(*params)
</code></pre>
</div>
