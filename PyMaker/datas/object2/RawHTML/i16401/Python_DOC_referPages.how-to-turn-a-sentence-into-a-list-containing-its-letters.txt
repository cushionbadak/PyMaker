<p>Suppose I have the following string:</p>
<pre><code>sentence = 'Python is a programming language'
</code></pre>
<p>How can I turn it into:</p>
<pre><code>sentence_list = [P,y,t,h,o,n,i,s,a,p,r,o,g,r,a,m,m,i,n,g,l,a,n,g,u,a,g,e]
</code></pre>
<p>I wrote the following code but it doesn't give me what I want:</p>
<pre><code>sentence = 'Python is a programming language'
for i in range(len(sentence)):
    if sentence[i] == ' ':
        sentence_new = sentence.replace('sentence[i]', '')
sentence_spaced = ' '.join(sentence_new)
sentence_list = sentence_spaced.split(' ')
print sentence_list
#OUTPUT: ['P', 'y', 't', 'h', 'o', 'n', '', '', 'i', 's', '', '', 'a', '', '', 'p', 'r', 'o', 'g', 'r', 'a', 'm', 'm', 'i', 'n', 'g', '', '', 'l', 'a', 'n', 'g', 'u', 'a', 'g', 'e']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; sentence = 'Python is a programming language'
&gt;&gt;&gt; 
&gt;&gt;&gt; [c for c in sentence if not c.isspace()]
['P', 'y', 't', 'h', 'o', 'n', 'i', 's', 'a', 'p', 'r', 'o', 'g', 'r', 'a', 'm', 'm', 'i', 'n', 'g', 'l', 'a', 'n', 'g', 'u', 'a', 'g', 'e']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>All you need is <a href="https://docs.python.org/2/library/functions.html#list" rel="nofollow"><code>list</code></a> and <a href="https://docs.python.org/2/library/stdtypes.html#str.replace" rel="nofollow"><code>str.replace</code></a> (to remove spaces):</p>
<pre><code>&gt;&gt;&gt; sentence = 'Python is a programming language'
&gt;&gt;&gt; list(sentence.replace(' ', ''))
['P', 'y', 't', 'h', 'o', 'n', 'i', 's', 'a', 'p', 'r', 'o', 'g', 'r', 'a', 'm', 'm', 'i', 'n', 'g', 'l', 'a', 'n', 'g', 'u', 'a', 'g', 'e']
&gt;&gt;&gt;
</code></pre>
<p>If you want to remove <em>everything</em> that is not a letter, you can use <a href="https://docs.python.org/2/library/functions.html#filter" rel="nofollow"><code>filter</code></a> and <a href="https://docs.python.org/2/library/stdtypes.html#str.isalpha" rel="nofollow"><code>str.isalpha</code></a>:</p>
<pre><code>&gt;&gt;&gt; sentence = 'ab1%^ &amp;#$2cd'
&gt;&gt;&gt; filter(str.isalpha, sentence)
['a', 'b', 'c', 'd']
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Or list comprehension with <code>replace</code>:</p>
<pre><code>[s for s in sentence.replace(' ','')]
</code></pre>
</div>
