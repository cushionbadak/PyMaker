<p>I have seen methods for list comparison on stackoverflow before, but they all seem to return the matches for a list - not the non matches. I am interested to find the method that returns the non matching values (and possibly which list they came from).</p>
<pre><code>listA = ['spanish', 'english', 'chinese', 'italian', 'english']
listB = ['spanish', 'english', 'italian', 'japanese']
</code></pre>
<p>I would like to return at least which words were NOT found in bouth;</p>
<pre><code>['chinese','japanese']
</code></pre>
<p>If not this;</p>
<pre><code>[('chinese',listA),('japanese',listB)]
</code></pre>
<p>Please offer any solutions for this comparison method(s).</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/2/library/stdtypes.html#set.symmetric_difference"><code>set.symmetric_difference</code></a> for this:</p>
<pre><code>&gt;&gt;&gt; set(listA).symmetric_difference(listB)
set(['japanese', 'chinese'])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#types-set" rel="nofollow">sets</a> for this:</p>
<pre><code>&gt;&gt;&gt; listA = ['spanish', 'english', 'chinese', 'italian', 'english']
&gt;&gt;&gt; listB = ['spanish', 'english', 'italian', 'japanese']
&gt;&gt;&gt; list(set(listA) ^ set(listB))  # Note that the results will be unordered
['japanese', 'chinese']
&gt;&gt;&gt;
</code></pre>
<p>The <code>^</code> operator finds the symmetric difference between the two sets.</p>
</div>
<div class="post-text" itemprop="text">
<p>In other words this is what you are looking for: <code>set(listA) - set(listB) | set(listB) - set(listA)</code></p>
<pre><code>&gt;&gt;&gt; listA = ['spanish', 'english', 'chinese', 'italian', 'english']
&gt;&gt;&gt; listB = ['spanish', 'english', 'italian', 'japanese']
&gt;&gt;&gt; setA = set(listA)
&gt;&gt;&gt; setB = set(listB)
&gt;&gt;&gt; setA - setB | setB - setA
set(['japanese', 'chinese'])
&gt;&gt;&gt; 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Alternative solution, which does not involve <code>set()</code>:</p>
<pre><code> &gt;&gt;&gt; from collections import Counter
 &gt;&gt;&gt; [v for v, k in Counter(listA + listB).iteritems() if k==1]
 ['japanese', 'chinese']
</code></pre>
</div>
