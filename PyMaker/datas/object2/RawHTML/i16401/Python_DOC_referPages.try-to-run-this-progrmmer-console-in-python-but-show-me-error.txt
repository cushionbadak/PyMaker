<div class="post-text" itemprop="text">
<p>You are trying to run a Python3 code in Python2. The <a href="https://docs.python.org/2/reference/simple_stmts.html#print" rel="nofollow noreferrer">Python2 <code>print</code></a> is a keyword and just prints what's given, whereas the <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer">Python3 <code>print</code></a> is a function with some additional parameters. <a href="https://docs.python.org/2/library/functions.html#print" rel="nofollow noreferrer">Python3 <code>print</code> was backported to Python2 and you can made it available using <code>from __future__ import print_function</code></a>:</p>
<pre><code>&gt;&gt;&gt; from __future__ import print_function
&gt;&gt;&gt; def fib(n):
...     a, b = 0, 1
...     while a &lt; n:
...         print(a, end=' ')
...         a, b = b, a+b
...     print()
... 
&gt;&gt;&gt; fib(5)
0 1 1 2 3 
</code></pre>
<p>In plain Python2 it would look like this:</p>
<pre><code>&gt;&gt;&gt; def fib(n):
...     a, b = 0, 1
...     while a &lt; n:
...         print a, ' '
...         a, b = b, a + b
...     print
... 
&gt;&gt;&gt; fib(5)
0  
1  
1  
2  
3 
</code></pre>
<p>Unfortunately, Python2 <code>print</code> writes a newline at the end if you don't print a dot (<code>print 1, 'something', '.'</code>). See <a href="https://stackoverflow.com/questions/493386/how-to-print-in-python-without-newline-or-space">How to print without newline or space?</a> for ways to go around it.</p>
<p>Or you could just store the results and then concatenate and print them at once, e.g.:</p>
<pre><code>&gt;&gt;&gt; def fib(n):
...     a, b = 0, 1
...     results = []
...     while a &lt; n:
...         result.append(a)
...         a, b = b, a + b
...     print ' '.join([str(r) for r in results])
... 
&gt;&gt;&gt; fib(5)
0 1 1 2 3
</code></pre>
