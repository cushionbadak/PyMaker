<div class="post-text" itemprop="text">
<p>In Python 3.x, you have to write <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow"><code>print()</code></a> as follows, because now it's a function:</p>
<pre><code>print(line)
</code></pre>
<p>In Python 2.x it was possible to omit the <code>()</code>, but that's no longer the case.</p>
