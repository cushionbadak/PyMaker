<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>subprocess.call(["php",  "C:\\Python27\\a.php"])
</code></pre>
<p>From the <a href="https://docs.python.org/3/library/subprocess.html#frequently-used-arguments" rel="nofollow">documentation</a>:</p>
<blockquote>
<p><em>args</em> is required for all calls and should be a string, or a sequence
  of program arguments. Providing a sequence of arguments is generally
  preferred, as it allows the module to take care of any required
  escaping and quoting of arguments (e.g. to permit spaces in file
  names). If passing a single string, either <em>shell</em> must be <code>True</code> (see
  below) or else the string must simply name the program to be executed
  without specifying any arguments.</p>
</blockquote>
<p>Also note that in python, like many other languages, the backslash in a normal string has <a href="https://docs.python.org/3/reference/lexical_analysis.html#strings" rel="nofollow">special meaning</a>. You'll have to use double backslashes or a raw string to get the behavior you want.</p>
