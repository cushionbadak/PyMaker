<div class="post-text" itemprop="text">
<p>I call an exe tool from Python: </p>
<pre><code>args = '-i' + inputFile
cl = ['info.exe', args]
subprocess.call(cl)
</code></pre>
<p>How would I redirect output from the info.exe to <code>out.txt</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can redirect output to file object by specifying it as <code>stdout</code> argument to <a href="https://docs.python.org/3/library/subprocess.html#subprocess.call" rel="nofollow"><code>subprocess.call</code></a>. Put it in a context manager to write to file safely.</p>
<pre><code>with open('out.txt', 'w') as f:
    subprocess.call(cl, stdout=f)
</code></pre>
<p>Or open the file in wb mode and use <a href="https://docs.python.org/3/library/subprocess.html#subprocess.check_output" rel="nofollow"><code>subprocess.check_output</code></a>:</p>
<pre><code>with open('out.txt', 'wb') as f:
    f.write(subprocess.check_output(cl))
</code></pre>
</div>
