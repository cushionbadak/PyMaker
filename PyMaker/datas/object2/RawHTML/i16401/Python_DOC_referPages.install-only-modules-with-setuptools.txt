<div class="post-text" itemprop="text">
<p>If you have a <a href="https://docs.python.org/3/glossary.html#term-package" rel="nofollow">Python package</a>, you can specify it in <a href="https://docs.python.org/3/distutils/setupscript.html#listing-whole-packages" rel="nofollow">the <code>packages</code> parameter</a>.</p>
<p>As you describe it, though, you do not have Python packages; you have only modules.</p>
<p>If you want to install Python modules that are <em>not</em> packages, you instead use <a href="https://docs.python.org/3/distutils/setupscript.html#listing-individual-modules" rel="nofollow">the <code>py_modules</code> parameter</a>.</p>
