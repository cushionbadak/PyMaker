<p>I think this is what you're looking for. It iterates over the file line by line.</p>
<pre class="lang-py prettyprint-override"><code>input_file = open(file_name, 'r')

</code></pre>

<div class="post-text" itemprop="text">
<p><s> You probably meant the <code>size</code> of the file.</s></p>
<p>
  Using <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow noreferrer"><code>pathlib</code></a>:</p>
<pre><code>from pathlib import Path
file = Path() / 'list.txt'  # or Path('./file_name.txt')
size = file.stat().st_size  # size is in bytes
print("The size of the file is {} bytes".format(size))
</code></pre>
<p><strong>OUTPUT</strong> (for one of my files):</p>
<pre><code>The size of the file is 42 bytes 
</code></pre>
<p><strong>OR</strong> </p>
<p>Using <code>os.stat</code>:</p>
<pre><code>import os
print(os.stat('list.txt').st_size)   # 42
</code></pre>
<p><strong>EDIT</strong>:</p>
<p>If you want to get the number of lines in a file:</p>
<pre><code>print(sum (1 for _ in open("list.txt")))    # 3
</code></pre>
