<p>I find the solution with somebodies help:  </p>
<pre><code>cur.execute("select count() from tanulo")
szam = (cur.fetchone()[0])
szam = szam + 1
print (szam)
</code></pre>
<p>The output of this code is<br/>
183<br/>
so the 'szam' variable is now an integer.</p>
