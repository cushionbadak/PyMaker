<div class="post-text" itemprop="text">
<p>Try the following, using <a href="https://docs.python.org/3/library/random.html" rel="nofollow noreferrer"><code>random.choice()</code></a> and <a href="https://docs.python.org/3.6/library/functions.html#format" rel="nofollow noreferrer">string formats</a>:</p>
<pre><code>import random

suits = ['Hearts', 'Diamonds', 'Clubs', 'Spades']
numbers = [
    '2', '3', '4', '5', '6', '7', '8', '9', '10',
    'Jack', 'Queen', 'King', 'Ace']

print('{number} of {suit}'.format(
    number=random.choice(numbers),
    suit=random.choice(suits)))

&gt;&gt; Ace of Hearts
</code></pre>
<p>You could use some <code>range</code> function to generate the numbers, but in this particular case, you can probably get away with hardcoding it. Note that strings are probably the better option as well because you're mixing that with <code>'Jack', 'Queen', 'King', and 'Ace'</code>.</p>
