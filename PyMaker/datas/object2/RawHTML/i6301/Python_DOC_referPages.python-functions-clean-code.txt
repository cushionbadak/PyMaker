<div class="post-text" itemprop="text">
<p>Returning print will return <code>None</code> as print does not return anything, so option 2 will not. Also, functions are not called by putting def in front of them, that is how you define them, so neither Option 1 nor Option 2 will work.</p>
<p>In this case, the base way would probably be:</p>
<pre><code>def is_number_in_string(string, number):
    return str(number) in string

print(is_number_in_string("myString84", 84))
</code></pre>
</div>
