<p>There you go. Just iterate through the islands, finding their area and update your answer if the current area is less than minimum.</p>
<pre><code>def land_rectangle_area(x1, y1, x2, y2):   
    area=abs((int(x1)-int(x2))*(int(y1)-int(y2)))
    return(area)

islands = {
    "Banana island"    : (3,5,7,6),
    "Mango island"     : (10,3,19,4),
    "Pineapple island" : (8,8,9,20),
    "Coconut island"   : (2,13,5,9) 
}
minimum = float('Inf') # equivalent to infinity
ans=''
for i in islands:
    a = land_rectangle_area(*islands[i])
    if a&lt;minimum:
        minimum = a
        ans = i
print (ans,minimum)
</code></pre>
<p>Output :</p>
<pre><code>('Banana island', 4)
</code></pre>

<p>You can use Python's <code>min</code> function with your <code>land_rectangle_area</code> function as a key to do this:</p>
<pre><code>def land_rectangle_area(x1, y1, x2, y2):   
    area=abs((int(x1)-int(x2))*(int(y1)-int(y2)))
    return(area)

&gt;&gt;&gt; min(islands.items(), key=lambda (k,t): land_rectangle_area(*t))
('Banana island', (3, 5, 7, 6))
</code></pre>
<p>You can find the <code>max</code> the same way:</p>
<pre><code>&gt;&gt;&gt; max(islands.items(), key=lambda (k,t): land_rectangle_area(*t))
('Pineapple island', (8, 8, 9, 20))
</code></pre>
<p>Or, use a list comprehension to transform the tuple into the area:</p>
<pre><code>&gt;&gt;&gt; [(k,land_rectangle_area(*t)) for k,t in islands.items()]
[('Pineapple island', 12), ('Coconut island', 12), ('Banana island', 4), ('Mango island', 9)]
</code></pre>
<p>And then take the <code>min</code> of that:</p>
<pre><code>&gt;&gt;&gt; min([(k,land_rectangle_area(*t)) for k,t in islands.items()], key=lambda t: t[1])
</code></pre>
<p>Or sort them smallest to largest:</p>
<pre><code>&gt;&gt;&gt; sorted(islands.items(), key=lambda (k,t): land_rectangle_area(*t))
[('Banana island', (3, 5, 7, 6)), ('Mango island', (10, 3, 19, 4)), ('Pineapple island', (8, 8, 9, 20)), ('Coconut island', (2, 13, 5, 9))]
</code></pre>
<p>(Since Coconut Island and Pineapple Island have the same area, either of them could be considered the <code>max</code> for the function and the sort. You would add another key to be definitive...)</p>
<p>Or you can bypass the named function and just use <code>min</code> with a key function:</p>
<pre><code>&gt;&gt;&gt; min(islands.items(), key=lambda (k,t): abs((t[0]-t[2])*(t[1]-t[3])))
</code></pre>
