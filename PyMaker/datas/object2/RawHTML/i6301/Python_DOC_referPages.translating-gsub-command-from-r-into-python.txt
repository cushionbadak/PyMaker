<p>I want to split this mystring="0G15^GAC0T60T4^AA0C0" and get the following output with python:</p>
<pre><code>['0','G','15','^GAC','T','60','T','4','^AA','C']
</code></pre>
<p>It could be done with this command in R:</p>
<pre><code>mystring &lt;- "0G15^GAC0T60T4^AA0C0"
gsub("([\\^]*[ACGT]+)[0]*", " \\1 ", mystring)
</code></pre>
<p>how can I translate the R script into python?</p>
<p>Thanks</p>
</div>
<div class="post-text" itemprop="text">
<p>You can reuse your existing regular expression using Pythons <a href="https://docs.python.org/3/library/re.html#re.sub" rel="noreferrer"><code>re</code></a> module</p>
<pre><code>import re

mystring = "0G15^GAC0T60T4^AA0C0"
l = re.sub("([\\^]*[ACGT]+)[0]*", " \\1 ", mystring).split()
</code></pre>
<p><code>l</code> is then </p>
<pre><code>['0', 'G', '15', '^GAC', 'T', '60', 'T', '4', '^AA', 'C']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can try this:</p>
<pre><code>mystring="0G15^GAC0T60T4^AA0C0" 
import re
new_data = re.findall('(?&lt;!\^[GAC])\d+|(?&lt;!\^)\w|\^[a-zA-Z]+', mystring)
final_data = [a for i, a in enumerate(new_data) if a != '0' or not new_data[i-1].startswith("^")][:-1]
</code></pre>
<p>Output:</p>
<pre><code>['0', 'G', '15', '^GAC', 'T', '60', 'T', '4', '^AA', 'C']
</code></pre>
