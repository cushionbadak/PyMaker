<div class="post-text" itemprop="text">
<p>because you <a href="https://docs.python.org/3/library/logging.html#logging.Logger.setLevel" rel="noreferrer">need</a> to set <code>logger</code> lever as well.</p>
<pre><code>import logging

logger = logging.getLogger()
# here
logger.setLevel(logging.INFO)
handler = logging.StreamHandler()
handler.setLevel(logging.INFO)
logger.addHandler(handler)
</code></pre>
<p>output:</p>
<pre><code>&gt;&gt;&gt; logging.info("my info")
my info
&gt;&gt;&gt; logging.warning("my warning")
my warning
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Loggers have levels as well as handlers. The <a href="https://docs.python.org/3/library/logging.html#logging.Logger.setLevel" rel="nofollow noreferrer">default level</a> is warn so your info is ignored.</p>
<p>To get your example to work, include <code>logger.setLevel(logging.INFO)</code>.</p>
</div>
<span class="comment-copy">If you are just wanting logging to a single handler, you can use <a href="https://docs.python.org/3/library/logging.html#logging.basicConfig" rel="nofollow noreferrer"><code>logging.basicConfig</code></a></span>
