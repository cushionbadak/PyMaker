<div class="post-text" itemprop="text">
<p>I have following variable:</p>
<pre><code>number = "456367"
</code></pre>
<p>I need to add it to list by one number, like this:</p>
<pre><code>list = [['4', '5', '6', '3', '6', '7']]
</code></pre>
<p>How can I achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Since a string is <a href="https://docs.python.org/3/glossary.html#term-iterable" rel="nofollow noreferrer">iterable</a>, pass it to the <a href="https://docs.python.org/3.6/library/functions.html#func-list" rel="nofollow noreferrer">list</a> constructor:</p>
<pre><code>&gt;&gt;&gt; list("456367")
['4', '5', '6', '3', '6', '7']
</code></pre>
<p>Which is why you would not want to name the result <code>list</code> because you would stomp on the name of that convenient function ;-)</p>
<hr/>
<p>If you really want <code>[['4', '5', '6', '3', '6', '7']]</code> vs just a list of characters:</p>
<pre><code>&gt;&gt;&gt; s="456367"
&gt;&gt;&gt; map(list, zip(*s))
[['4', '5', '6', '3', '6', '7']]
</code></pre>
<p>Or,</p>
<pre><code>&gt;&gt;&gt; [list(s)]
[['4', '5', '6', '3', '6', '7']]
</code></pre>
<p>But I am assuming with the first answer that the extra brackets were a typo.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; number = "456367"
&gt;&gt;&gt; [char for char in number]
['4', '5', '6', '3', '6', '7']
</code></pre>
</div>
