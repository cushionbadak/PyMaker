<p>For example, if I do this:</p>
<pre><code>for i in '12345':
        print("Welcome",i,"times")
</code></pre>
<p>It outputs:</p>
<pre><code>Welcome 1 times
Welcome 2 times
Welcome 3 times
Welcome 4 times
Welcome 5 times
</code></pre>
<p>But I want it like this:</p>
<pre><code>Welcome1times
Welcome2times
Welcome3times
Welcome4times
Welcome5times
</code></pre>
<p>Is this possible?</p>
</div>
<div class="post-text" itemprop="text">
<p>Yes you can, by using the <code>sep</code> parameter of the <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer">print function</a>:</p>
<pre><code>for i in '12345':
    print("Welcome", i, "times", sep="")

&gt;&gt;&gt;
Welcome1times
Welcome2times
Welcome3times
Welcome4times
Welcome5times
</code></pre>
<p>By the way, you should think about generating your string using standard string formating methods.
For example, you could do : </p>
<pre><code>for i in '12345':
    print("Welcome%stimes"%i)
</code></pre>
<p>Or in a more python-3 way : </p>
<pre><code>for i in '12345':
    print("Welcome{i}times".format(i=i))
</code></pre>
<p>Or even shorter (thanks @StefanPochmann)</p>
<pre><code>for i in '12345':
    print(f"Welcome{i}times")
</code></pre>
