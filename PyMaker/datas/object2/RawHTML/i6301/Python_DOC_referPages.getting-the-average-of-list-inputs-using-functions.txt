<div class="post-text" itemprop="text">
<p><code>input()</code> returns a string, not an integer. This means that <code>scores</code> has a bunch of strings in it, not integers, and you can't <code>sum()</code> a list of strings.</p>
<p>Change <code>grade.append(input('Enter Score: '))</code> to <code>grade.append(int(input('Enter Score: ')))</code></p>
<p>edit: to fix the error mentioned in the comment, you need to change your main function:</p>
<pre><code>def main():
    scores = []
    get_scores(scores)
    final_score = calculate_average(scores)
    show_results(final_score)
</code></pre>
