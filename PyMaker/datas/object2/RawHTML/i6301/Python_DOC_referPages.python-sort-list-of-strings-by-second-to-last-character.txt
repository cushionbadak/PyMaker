<p>Is this what you want? If yes, make that penultimate letter the key for the <code>sort()</code> function.</p>
<pre><code>a = ['message', 'essagem', 'ssageme', 'sagemes', 'agemess', 'gemessa', 'emessag']
a.sort(key=lambda x:x[-2])
print(a)
</code></pre>
<p>Output:</p>
<pre><code>['emessag', 'essagem', 'sagemes', 'message', 'ssageme', 'agemess', 'gemessa']
</code></pre>

<div class="post-text" itemprop="text">
<p>Just to add something alternative to Miraj50's answer: You can do it using <code>itemgetter</code> too in a bit <a href="https://docs.python.org/3/howto/sorting.html" rel="nofollow noreferrer">cleaner/faster way</a>. </p>
<pre><code>from operator import itemgetter

in_list = ['message', 'essagem', 'ssageme', 'sagemes', 'agemess', 'gemessa', 'emessag']
result = sorted(in_list, key=itemgetter(-2))
print(result)
</code></pre>
<p>output: </p>
<pre><code>['emessag', 'essagem', 'sagemes', 'message', 'ssageme', 'agemess', 'gemessa']
</code></pre>
</div>
