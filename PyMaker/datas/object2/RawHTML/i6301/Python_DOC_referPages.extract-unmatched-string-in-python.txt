<p>I have two strings s and s2</p>
<pre><code>s = "catwalksonterrace9_ontheweekend_at7am"
s2= "catwalksonterrace$no_ontheweekend_at.*"
</code></pre>
<p>I need to compare two strings and extract the unmatched portions that are
<code>$no = 9</code> and <code>.* = 7am</code> from both the strings in Python. I am new to python how would I achieve this?</p>

<div class="post-text" itemprop="text">
<p>Take a look at difflib, it's awesome, made to do exactly what you want :)
<a href="https://docs.python.org/2/library/difflib.html" rel="nofollow noreferrer">https://docs.python.org/2/library/difflib.html</a></p>
<pre><code>import difflib
d = difflib.Differ()
diffs = []
in_diff = False
for c in d.compare(s, s2):
    if not in_diff and (c.startswith("+") or c.startswith("-")):
        diffs.append(["", ""])
        in_diff = True
    if c.startswith("+"):
        diffs[-1][0] += c.replace("+ ", "")
    elif c.startswith("-"):
        diffs[-1][1] += c.replace("- ", "")
    else:
        in_diff = False
print(diffs)
</code></pre>
<p>This creates a list of lists, where first value of each sublist is diff on line 1, and seocnd value is diff on line 2 </p>
<p>Output will be:</p>
<pre><code>[['$no', '9'], ['.*', '7am']]
</code></pre>
<p>You can then loop through that, printing out as requested:</p>
<pre><code>for diff in diffs:
    print(diff[0], "=", diff[1])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>With <a href="https://docs.python.org/3/library/difflib.html?highlight=difflib#difflib.unified_diff" rel="nofollow noreferrer"><strong><em>difflib.unified_diff()</em></strong></a> function:</p>
<pre><code>import difflib, itertools

s1 = "catwalksonterrace9_ontheweekend_at7am"
s2 = "catwalksonterrace$no_ontheweekend_at.*"

result = []
for i in itertools.islice(difflib.unified_diff(s1, s2, lineterm=''), 2, None):
    if i.startswith('@@'):    # diff control line
        result.append(['',''])
    elif i.startswith('-'):   # line unique to sequence 1
        result[-1][0] += i[1:]
    elif i.startswith('+'):   # line unique to sequence 2
        result[-1][1] += i[1:]

print(result)
</code></pre>
<p>The output:</p>
<pre><code>[['9', '$no'], ['7am', '.*']]
</code></pre>
<hr/>
<p>Each sublist in the resulting <code>result</code> list contains a pair of "<em>old</em>" and "<em>new</em>" value (i.e. <code>[&lt;old&gt;, &lt;new&gt;]</code>)</p>
