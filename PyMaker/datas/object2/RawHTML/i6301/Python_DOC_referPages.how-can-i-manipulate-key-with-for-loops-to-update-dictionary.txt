<p>I am trying to put a list into a dictionary and count the number of occurrences of each word in the list. The only problem I don't understand is when I use the update function, it takes x as a dictionary key, when I want x to be the x value of list_ . I am new to python so any advice is appreciated. Thanks    </p>
<pre><code>list_ = ["hello", "there", "friend", "hello"]
d = {}
for x in list_:
    d.update(x = list_.count(x))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/2/library/collections.html#collections.Counter" rel="nofollow noreferrer">Counter</a> object if you want a simple way of converting a list of items to a dictionary which contains a mapping of <code>list_entry: number_of_occurences</code> .</p>
<pre><code>&gt;&gt;&gt; from collections import Counter
&gt;&gt;&gt; words = ['hello', 'there', 'friend', 'hello']
&gt;&gt;&gt; c = Counter(words)

&gt;&gt;&gt; print(c)
Counter({'hello': 2, 'there': 1, 'friend': 1})

&gt;&gt;&gt; print(dict(c))
{'there': 1, 'hello': 2, 'friend': 1}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>An option would be using dictionary comprehension with <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer"><code>list.count()</code></a> like this:</p>
<pre><code>list_ = ["hello", "there", "friend", "hello"]
d = {item: list_.count(item) for item in list_}
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; d
{'hello': 2, 'there': 1, 'friend': 1}
</code></pre>
<p>But the best option should be <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>collections.Counter()</code></a> used in @AK47's solution.</p>
