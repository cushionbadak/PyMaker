<div class="post-text" itemprop="text">
<p>You need to use <code>split()</code> not <code>strip()</code>.</p>
<p><code>strip()</code> is used to remove the specified characters from a string. </p>
<p><code>split()</code> is used to split the string to list based on the value provided.</p>
<pre><code>date = str(input()) # reading input date in dd.mm.yyyy format
splitted_date = date.split('.') # splitting date 

day = splitted_date[0] # storing day
month = splitted_date[1] # storing month
year = splitted_date[2] # storing year

# Display the values
print('Date : ',date)
print('Month : ',month)
print('Year : ',year)
</code></pre>
<p>You can split date given in <code>DD.MM.YYYY</code> format like this.</p>
</div>
<div class="post-text" itemprop="text">
<p>Instead of splitting the string, you should be using <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strptime" rel="nofollow noreferrer"><strong><code>datetime.strptime(..)</code></strong></a> to convert the string to the <a href="https://docs.python.org/3/library/datetime.html#datetime-objects" rel="nofollow noreferrer">datetime object</a> like:</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; my_date_str = "24.02.1999"
&gt;&gt;&gt; my_date = datetime.strptime(my_date_str, '%d.%m.%Y')
</code></pre>
<p>Then you can access the values you desire as:</p>
<pre><code>&gt;&gt;&gt; my_date.day  # For date
24
&gt;&gt;&gt; my_date.month  # For month
2
&gt;&gt;&gt; my_date.year  # For year
1999
</code></pre>
</div>

</code></pre>
