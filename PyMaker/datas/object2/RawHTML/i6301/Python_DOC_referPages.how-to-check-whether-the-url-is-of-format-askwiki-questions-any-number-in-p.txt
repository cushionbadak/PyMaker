<div class="post-text" itemprop="text">
<p>You're on the right track! The missing component is the <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer" title="re module">re module</a>.</p>
<p>I think what you want is something like this:</p>
<pre><code>import re
matcher = re.compile(r"/askwiki/questions/[0-9]+")
if matcher.search(link.get("href")):
    print(link.text)
</code></pre>
<p>Alternatively, you can just drop the number component, if you're only really looking for links with "/askwiki/questions" in:</p>
<pre><code>if "/askwiki/questions" in link.get("href")
    print(link.text)
</code></pre>
