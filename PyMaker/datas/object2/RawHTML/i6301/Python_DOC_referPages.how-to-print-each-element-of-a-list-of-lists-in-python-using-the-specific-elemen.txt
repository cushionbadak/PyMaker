<div class="post-text" itemprop="text">
<p>Easiest to use for-each loops:</p>
<pre><code>for l in priceChart:
    for c in l:
        print(c, end= " ")
    print()    
</code></pre>
<p>If you have to use indexes:</p>
<pre><code>for r in range(len(priceChart)):  # no '- 1' here, range(x) loops from 0 to x-1
    for c in range(len(priceChart[r])):  # use 'r', not 0
        print(priceChart[r][c], end= " ")
    print()  
</code></pre>
