<p><a href="https://docs.python.org/3/library/glob.html#module-glob" rel="nofollow noreferrer"><code>glob.glob</code></a> could help you:</p>
<pre><code>from glob import glob

<p>Functions you may find handy:<br/>
<code>os.getcwd()</code> get current working directory<br/>
<code>os.chdir(path)</code> Change your working directory to path given<br/>
<code>os.listdir(path=None)</code> get list of contents in current dir, or path given. </p>
