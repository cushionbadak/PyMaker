<p><a href="http://www.fileformat.info/info/unicode/category/index.htm" rel="nofollow noreferrer">FileFormat.info</a> has a list of Unicode character categories.</p>
<p>See if this is what you want to achieve:</p>
<pre><code>import unicodedata

def split_clusters(txt):
    """ Generate grapheme clusters for the Devanagari text."""

    stop = '्'
    cluster = u''
    end = None

    for char in txt:
        category = unicodedata.category(char)
        if (category == 'Lo' and end == stop) or category == 'Mn':
            cluster = cluster + char        
        else:
            if cluster:
                yield cluster
            cluster = char
        end = char

    if cluster:
        yield cluster
</code></pre>
<p>Testing the function:</p>
<pre><code>&gt;&gt;&gt; list(split_clusters('धर्मक्षेत्रे'))
['ध', 'र्म', 'क्षे', 'त्रे']
&gt;&gt;&gt; list(split_clusters('कुरुक्षेत्र'))
['कु', 'रु', 'क्षे', 'त्र']
</code></pre>
