<p>Loop over the list, comparing each element with the previous:</p>
<pre><code>def get_first_index_of_third_part(three_parts_list, part=3):
    current_part = 0
    old = None
    for i, el in enumerate(three_parts_list):
        if el != old:
            # New part found
            current_part += 1
        # Stop at the beginning of the correct part
        if current_part == part:
            return i
        # Keep record of previous element
        old = el
</code></pre>
<p>Here the <code>part</code> argument to the <code>get_first_index_of_third_part</code> function determines the number of parts and defaults to 3. </p>
