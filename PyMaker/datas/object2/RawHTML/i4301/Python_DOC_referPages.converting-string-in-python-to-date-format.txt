<div class="post-text" itemprop="text">
<p>Firstly, the error is because you're using <code>%Y</code>, <code>%m</code>, and <code>%d</code> twice in your <code>time.strptime()</code> call. </p>
<p>Secondly, you're using the wrong format. The format you pass to <code>strptime()</code> has to match the format of the date / time string you pass, which in this case is: <code>%B %d, %Y</code>.</p>
<p><a href="http://strftime.org/" rel="nofollow noreferrer">This</a> is a good reference on the different format types.</p>
</div>
<div class="post-text" itemprop="text">
<p>I normally use <code>datetime</code> for this:</p>
<pre><code>from datetime import datetime
review_date = "April 18, 2018"
review_date = datetime.strptime(review_date, '%B %d, %Y').strftime('%Y-%m-%d')
</code></pre>
<p>This code returns review_date = '2018-04-18'. See <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer">https://docs.python.org/3/library/datetime.html</a>
The date format for <code>April</code> is <code>%B</code>. <code>strptime()</code> converts to a datetime object, <code>.strftime()</code> converts the datetime object to a string. </p>
</div>
<div class="post-text" itemprop="text">
<p><em>time.strptime()</em> is for <strong>parsing</strong> strings into date/time structures. It takes two arguments, the string to be parsed and another string describing the format of the string to be parsed.</p>
<p>Try this:</p>
<pre><code>time.strptime("April 18, 2018", "%B %d, %Y")
</code></pre>
<p>... and notice that "%B %d, %Y" is:</p>
<ol>
<li>Full locale name of the month ("April")</li>
<li>[Space]</li>
<li>Date of the month (18)</li>
<li>[Comma]</li>
<li>[Space]</li>
<li>Four digit year (2018)</li>
</ol>
<p>The format string specification that you provided bears no resemblance to the formatting of your date string.</p>
<p>These "magic" formatting codes are enumerated in the documentation for <a href="https://docs.python.org/2/library/time.html#time.strftime" rel="nofollow noreferrer">time.strftime()</a></p>
</div>
