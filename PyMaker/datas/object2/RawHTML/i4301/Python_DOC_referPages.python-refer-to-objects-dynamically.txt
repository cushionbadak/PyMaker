<div class="post-text" itemprop="text">
<p>use <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer">getattr</a>, For example, <code>getattr(x, 'foobar')</code> is equivalent to <code>x.foobar</code>.</p>
<pre><code>for i in form_fields:
    self.participant.vars[i] =  [Constants.fields_dict[i][0], Constants.fields_dict[i][1], getattr(self.player, i)]
</code></pre>
