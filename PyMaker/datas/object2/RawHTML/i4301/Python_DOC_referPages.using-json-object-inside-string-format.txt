<div class="post-text" itemprop="text">
<p>Your <code>jsonData</code> variable is a python dict. To access values in the format mini language you need to use <code>{my_dict[my_key]}</code>. Note that the key is not enclosed in quotes.
To fix your example, the html input should be as follow:</p>
<pre><code>...
&lt;a href={links[one]}&gt;&lt;/a&gt;
...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>According to <a href="https://docs.python.org/3.4/library/string.html#format-examples" rel="nofollow noreferrer">[Python]: Format examples</a>, you should use (templates like):</p>
<pre><code>&lt;a href={links[one]}&gt;&lt;/a&gt;
</code></pre>
<p>as <em>html</em> format specifier (since <a href="https://docs.python.org/3/library/json.html#json.load" rel="nofollow noreferrer">[Python]: json.<strong>load</strong>(<em>fp, *, cls=None, object_hook=None, parse_float=None, parse_int=None, parse_constant=None, object_pairs_hook=None, **kw</em>)</a> returns a <code>dict</code>).</p>
<p>Example:</p>
<blockquote>
<pre><code>&gt;&gt;&gt; json_dict
{'links': {'one': 'www.one.com', 'two': 'www.two.com'}}
&gt;&gt;&gt; href_text
'&lt;a href={links[one]}&gt;&lt;/a&gt;'
&gt;&gt;&gt; href_text.format(links=json_dict["links"])
'&lt;a href=www.one.com&gt;&lt;/a&gt;'
</code></pre>
</blockquote>
