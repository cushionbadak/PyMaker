<p>Try using the full path with operating-system-robust path separators. Basically build your path with <a href="https://docs.python.org/3/library/os.path.html#os.path.join" rel="nofollow noreferrer"><code>os.path.join</code></a>:</p>
<pre><code>import os
my_path = os.path.join('path', 'to', 'my', 'file.csv')
df = pd.read_csv(my_path)
</code></pre>
