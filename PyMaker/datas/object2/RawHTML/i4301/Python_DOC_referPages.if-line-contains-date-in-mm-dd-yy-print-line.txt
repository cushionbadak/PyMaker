<p>You can use this regex to recognize this model MM/DD/YYYY :</p>
<p><code>date_regex = '(0[1-9]|1[12])\/(0[1-9]|1[0-9]|2[0-9]|3[0-1])\/(\d{4})'</code></p>
<p><a href="https://regexr.com/3o5fs" rel="nofollow noreferrer">regexr example</a></p>
<p>and you can use this for MM/DD/YY :</p>
<pre><code>date_regex = '(0[1-9]|1[12])\/(0[1-9]|1[0-9]|2[0-9]|3[0-1])\/(\d{2})'
</code></pre>
<p><a href="https://regexr.com/3o5fj" rel="nofollow noreferrer">regexr example</a></p>

<p>Using <strong>regex</strong></p>
<pre><code>import re
s = "dssdfsdfsdfsdf 10/01/18 ssdfsfsf dsfsdfsdfsd"
print(re.findall("\d{2}/\d{2}/\d{2}", s))
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>['10/01/18']
</code></pre>

<pre><code>for line in f:
