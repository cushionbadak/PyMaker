<div class="post-text" itemprop="text">
<p>You would have to serialize the Webdriver instance in Java to something readable for both languages (so avoid the default Java serializer in this case). What you need is an XML or a JSON parser/serializer. Serialize the instance in Java to XML/JSON and parse the created serialization in Python. This serialization could be transferred via a file or WebSockets (<a href="http://www.baeldung.com/java-websockets" rel="nofollow noreferrer">Java</a> - <a href="https://pypi.org/project/websockets/" rel="nofollow noreferrer">Python</a>).</p>
<hr/>
<p><strong>XML:</strong></p>
<ul>
<li><a href="https://mvnrepository.com/artifact/javax.xml.parsers/jaxp-api" rel="nofollow noreferrer">Java</a></li>
<li><a href="https://docs.python.org/2/library/xml.etree.elementtree.html" rel="nofollow noreferrer">Python 2</a></li>
<li><a href="https://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow noreferrer">Python 3</a></li>
</ul>
<p><strong>JSON:</strong></p>
<ul>
<li><a href="https://mvnrepository.com/artifact/com.google.code.gson/gson" rel="nofollow noreferrer">Java</a></li>
<li><a href="https://docs.python.org/2/library/json.html" rel="nofollow noreferrer">Python 2</a></li>
<li><a href="https://docs.python.org/3/library/json.html" rel="nofollow noreferrer">Python 3</a></li>
</ul>
