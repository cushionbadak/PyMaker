<div class="post-text" itemprop="text">
<p>You are looking for the built-in function <a href="https://docs.python.org/3/library/functions.html#round" rel="nofollow noreferrer"><code>round()</code></a></p>
<blockquote>
<p>Return number rounded to ndigits precision after the decimal point. If ndigits is omitted or is None, it returns the nearest integer to its input.</p>
</blockquote>
<pre><code>round(1.4)
#&gt; 1
round(1.5)
#&gt; 2
</code></pre>
