<p>Hello everyone I am new to Django and using newsapi to get some news, the API returns a JSON object having a datetime attribute like this <code>2018-04-22T11:38:05Z</code>. I want to serialize it into Django <code>DateTimeField</code>. I have tried <code>.date()</code> but it didn't work. Any help will be appreciated.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strptime" rel="nofollow noreferrer">strptime</a> to convert the string to Python <code>datetime</code> object.</p>
<pre><code>import datetime

datetime_string = '2018-04-22T11:38:05Z'
datetime_obj = datetime.datetime.strptime(datetime_string, '%Y-%m-%dT%H:%M:%SZ')
print(datetime_obj)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>2018-04-22 11:38:05
</code></pre>
