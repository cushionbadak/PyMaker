<div class="post-text" itemprop="text">
<p>Using backslashes (<code>\</code>) in python strings can lead to surprises. In the code above, the backslash is producing a character with from an octal value that's invalid in a filename on Windows. See <a href="https://docs.python.org/3/reference/lexical_analysis.html#literals" rel="nofollow noreferrer">https://docs.python.org/3/reference/lexical_analysis.html#literals</a> for details.</p>
<p>The Python runtime is perfectly happy with forward slashes (<code>/</code>) on Windows. Use those.</p>
