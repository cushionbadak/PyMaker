<p>When I put a class method under a decorator it hides the method description ("""x""") in help(class_name).
How can I fix that please ?</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/library/functools.html" rel="nofollow noreferrer">functools.wraps</a>:</p>
<pre><code>def decorator(f):
    @functools.wrap(f)
    def wrapped(*args):
        # do something
    return wrapped
</code></pre>
