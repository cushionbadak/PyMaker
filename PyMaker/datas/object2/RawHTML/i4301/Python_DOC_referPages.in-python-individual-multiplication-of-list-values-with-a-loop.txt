<p>If I have a list <code>data_x = [1,2,3,4]</code>, how do I multiply each element of <code>data_x</code> with each element of another list <code>data_y = [2,3,4,5]</code> and sum the values?</p>
<p>The answer should be <code>1*2 +2*3 +3*4+ 4*5 = 40</code>. I am looking for some kind of <code>for</code> loop.</p>
</div>
<div class="post-text" itemprop="text">
<p>Try this (using a for loop, as requested by the OP):</p>
<pre><code>answer = 0
for i in range(len(data_x)):
    answer += (data_x[i] * data_y[i])
</code></pre>
<p>Another (quicker) way (using <code>list</code> comprehensions):</p>
<pre><code>answer = sum([x*y for x in zip(data_x, data_y)])
## Use zip to perform index-wise operations between two lists
## zip returns a list of tuples, with each tuple at index i containing the elements of data_x[i] and data_y[i]
</code></pre>
<p>You could also try this (if you have <code>numpy</code> installed):</p>
<pre><code>np.multiply(data_x, data_y).sum()
</code></pre>
</div>

<pre><code>sum([x * y for x, y in zip(data_x, data_y)])
</code></pre>

<pre><code>np.multiply(data_x, data_y).sum()
</code></pre>

<div class="post-text" itemprop="text">
<p>Use this</p>
<pre><code>result = sum([x * y for x, y in zip(data_x, data_y)])
</code></pre>
<p><code>result = zip(data_x, data_y)</code> creates a zip object of tuples where each tuple <code>result[i]</code> corresponds to the values <code>data_x[i]</code> and <code>data_y[i]</code></p>
<p>That is if </p>
<pre><code>data_x = [1, 2, 3, 4]
data_y = [2, 3, 4, 5]
</code></pre>
<p>Then, </p>
<pre><code>result = [(1, 2), (2, 3), (3, 4), (4, 5)]
</code></pre>
</div>
