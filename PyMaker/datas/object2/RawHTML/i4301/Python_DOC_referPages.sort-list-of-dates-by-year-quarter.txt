<div class="post-text" itemprop="text">
<p>Let's say I have a list:</p>
<pre><code>Date = ['2015-Q1, '2013-Q4', '2017-Q2', '2018-Q1']
</code></pre>
<p>How can I sort it into chronological order such that:</p>
<pre><code>Date = ['2013-Q4','2015-Q1','2017-Q2','2018-Q1']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>By using <code>natsort</code><br/>
<a href="https://pypi.org/project/natsort/" rel="nofollow noreferrer"><em>natsort provides a function natsorted that helps sort lists “naturally” (“naturally” is rather ill-defined, but in general it means sorting based on meaning and not computer code point)</em></a></p>
<pre><code>import natsort
natsort.natsorted(Date)
Out[103]: ['2013-Q4', '2015-Q1', '2017-Q2', '2018-Q1']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>These are periods. You'll want to generate a PeriodIndex and then call <code>sort_values</code>.</p>
<pre><code>p = pd.PeriodIndex(['2013-Q4','2015-Q1','2017-Q2','2018-Q1'], freq='Q')    
p.sort_values() # p = p.sort_values() # np.sort(p)
</code></pre>
<p></p>
<pre><code>PeriodIndex(['2013Q4', '2015Q1', '2017Q2', '2018Q1'], dtype='period[Q-DEC]', freq='Q-DEC')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.to_datetime.html" rel="nofollow noreferrer"><code>pandas.to_datetime</code></a> with <a href="https://docs.python.org/3/howto/sorting.html" rel="nofollow noreferrer"><code>sorted</code></a>:</p>
<pre><code>import pandas as pd

date = ['2015-Q1', '2013-Q4', '2017-Q2', '2018-Q1']

res = sorted(date, key=pd.to_datetime)
</code></pre>
<p>Result:</p>
<pre><code>print(res)

['2013-Q4' '2015-Q1' '2017-Q2' '2018-Q1']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>import pandas as pd    
pd.to_datetime(Date).sort_values().to_period('Q')
</code></pre>
<p>Output:</p>
<pre><code>PeriodIndex(['2013Q4', '2015Q1', '2017Q2', '2018Q1'], dtype='period[Q-DEC]', freq='Q-DEC')
</code></pre>
<p>Or like this if you want it back to strings:</p>
<pre><code>pd.to_datetime(Date).sort_values().to_period('Q').strftime('%Y-Q%q')
array(['2013-Q4', '2015-Q1', '2017-Q2', '2018-Q1'], dtype='&lt;U6')
</code></pre>
</div>
