<p>strings have a <code>.replace()</code> method: <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html#str.replace</a></p>
<pre><code>&gt;&gt;&gt; s = """
... #Version number
... Version=20
... 
... #Mode name
... Mode=Slow
... list=True
... type=8475
... """
&gt;&gt;&gt; s = s.replace('Slow', 'Fast')
&gt;&gt;&gt; print(s)
