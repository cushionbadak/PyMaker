<p>So the issue here is that these two data are not equivalent:</p>
<pre><code>datetime.datetime(2018, 4, 5, 7, 01),
datetime.datetime(2018, 4, 5, 7, 21)
</code></pre>
<p>because they differ on the minute value. So == will say that they are different. </p>
<pre><code>j = datetime.datetime(year,month,day,hour,minute)

for i in attendance:
    if i.year == j.year and i.month == j.month and i.day == j.day:
        pick the earlier (pseudocode)
    else
        print the data

    j = i
</code></pre>
