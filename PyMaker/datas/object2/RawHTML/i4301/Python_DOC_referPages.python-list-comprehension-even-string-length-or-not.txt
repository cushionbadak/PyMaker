<p>I am trying to print "even" or not with list comprehension, but I get an error.</p>
<pre><code>myNames = ['A','BB','CCC','DDDD']
myList3 = [ 'even' if x%2==0 else 'nope' for x in myNames]

Error: TypeError: not all arguments converted during string formatting
</code></pre>
<p>What is the reason behind it? </p>
</div>
<div class="post-text" itemprop="text">
<p>You are using the <code>%</code> operator on a string:</p>
<pre><code>&gt;&gt;&gt; x = 'A'
&gt;&gt;&gt; x % 2
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: not all arguments converted during string formatting
</code></pre>
<p>When using <code>%</code> on a string, you are not getting the modulus, but are using <a href="https://docs.python.org/3/library/stdtypes.html#old-string-formatting" rel="nofollow noreferrer"><em><code>printf</code>-style string formatting</em></a> instead. And that requires a <code>%</code>-style placeholder for the value on the right to be formatted into. With no placeholders in the string on the left, you get the error you see.</p>
<p>If you wanted to test if the <em>length</em> of the string is even, you need to use the <code>len()</code> function to get that length:</p>
<pre><code>myList3 = ['even' if len(x) % 2 == 0 else 'nope' for x in myNames]
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; myNames = ['A','BB','CCC','DDDD']
&gt;&gt;&gt; ['even' if len(x) % 2 == 0 else 'nope' for x in myNames]
['nope', 'even', 'nope', 'even']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The other answers explain why your syntax is incorrect.</p>
<p>If you are interested, here is an alternative implementation using a dictionary.</p>
<p>Eliminating <code>if</code> / <code>else</code> constructs in favour of a dictionary is often both efficient and readable.</p>
<pre><code>myNames = ['A','BB','CCC','DDDD']

mapper = {0: 'even', 1: 'nope'}
res = [mapper[len(i) %2] for i in myNames]

# ['nope', 'even', 'nope', 'even']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The <code>%</code> operator, when used with a string as the left-hand argument, is for printf-style formatting. If you want to test the <em>length</em> of your strings, you need to convert from string to the length with <code>len</code>, e.g.</p>
<pre><code>myList3 = [ 'even' if len(x) % 2==0 else 'nope' for x in myNames]
</code></pre>
