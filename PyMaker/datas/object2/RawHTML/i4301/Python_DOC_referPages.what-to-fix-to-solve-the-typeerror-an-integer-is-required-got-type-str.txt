<div class="post-text" itemprop="text">
<p>Your error is probably that you forgot to cast the <code>input(...)</code> to <code>int</code>:</p>
<pre><code>start_year = input('Enter start year')
start_year = int(start_year)
</code></pre>
<p>you should do the same for <code>end_year</code> and <code>output</code>.</p>

<span class="comment-copy">Notice the error says, "an integer is required (got type str)". When you get user input via <code>input()</code> you get a string. <code>dt.date</code> requires integers (see <a href="https://docs.python.org/3/library/datetime.html#datetime.date" rel="nofollow noreferrer">documentation</a> so you must convert the input. This question may be helpful: <a href="https://stackoverflow.com/q/20449427/945456">How can I read inputs as integers</a>.</span>
