print([s.strip() for s in sentences])
</code></pre>

<p>There is an easy way to do this:</p>
<pre><code>document = "Hello World. Goodbye World"

def sentence_separator(document):
    sentences = document.split(". ")
    return "\n".join(sentences)

print(sentence_separator(document))
</code></pre>
<p>NOTE: The reason why you got this error <strong>IndexError: string index out of range</strong> is because with <strong>for in</strong> loop working with string, each incrementation takes one character for that string. So at this line:</p>
<pre><code>if char[pos] == ".":
</code></pre>
<p>Imagine if 1 is the value of <code>pos</code> variable: it will try to find the second character of one character (sounds bad, it'snt?). Consequently, you get <strong>IndexError: string index out of range</strong></p>
</div>

<pre><code>def sentence_separator(document):
    return document.replace('.', '\n')
</code></pre>
