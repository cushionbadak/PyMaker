<p>When you do <code>fr=open("ddddfile.txt",'r').read()</code>, you are getting the contents of the file in the form of a string.  You need to convert that string into a list before trying to index it in the way that you are.  You should look into <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow noreferrer"><code>ast.literal_eval</code></a>.</p>
