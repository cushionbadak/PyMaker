<p>It is basically just one line.</p>
<pre><code>import itertools

s = {0,2,4,7,10}
sorted([abs(a-b) for (a,b) in itertools.combinations(s,2)])

<p>you can use <code>itertools</code></p>
<pre><code>import itertools

s = {0,2,4,7,10}
k = itertools.combinations(s,2)
distance = []
l = list(k)
for p in l:
    distance.append(abs(p[1]-p[0]))
print(sorted(distance))
</code></pre>

<p>A simple way is to convert your set to a list, sort it, and then use a double for loop to compute the differences:</p>
<pre><code>X = {0,2,4,7,10}  # original set
sorted_X = sorted(list(X))
diffs = []
for i, a in enumerate(sorted_X):
    for j, b in enumerate(sorted_X):
        if j &gt; i:
            diffs.append(b-a)           
print(diffs)
#[2, 4, 7, 10, 2, 5, 8, 3, 6, 3]
</code></pre>
<p>And if you want the diffs sorted as well:</p>
<pre><code>print(sorted(diffs))
#[2, 2, 3, 3, 4, 5, 6, 7, 8, 10]
</code></pre>
<hr/>
<p>Another option that would work in this case is to use <code>itertools.product</code>:</p>
<pre><code>from itertools import product
print(sorted([(y-x) for x,y in product(sorted_X, sorted_X) if y&gt;x]))
#[2, 2, 3, 3, 4, 5, 6, 7, 8, 10]
</code></pre>
