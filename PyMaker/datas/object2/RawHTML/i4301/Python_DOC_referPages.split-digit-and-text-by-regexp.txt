<div class="post-text" itemprop="text">
<p>You can simply use <a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow noreferrer"><code>re.split</code></a>, just make sure you use a capturing group so the split characters are not removed.  </p>
<p>Also I added in a check in the list comprehension to avoid empty strings when the first or last characters are matched by the regular expression.</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; s = "this1is2a3string4"
&gt;&gt;&gt; [i for i in re.split(r'([0-9]+)', s) if i]
['this', '1', 'is', '2', 'a', '3', 'string', '4']
</code></pre>
