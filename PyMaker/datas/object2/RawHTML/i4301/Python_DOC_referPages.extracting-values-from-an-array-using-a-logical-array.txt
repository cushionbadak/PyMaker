<p>I have the following MATLAB code which I would like to replicate using Python.</p>
<p>The MATLAB code creates a logical array for when <code>xDiff == 2</code> and then uses that logical array to extract corresponding values from the <code>tDiff</code> array to create the resulting array, <code>tTacho</code>.</p>
<p>MATLAB code:</p>
<pre><code>tTacho = tDiff(xDiff == 2)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do <a href="https://docs.scipy.org/doc/numpy-1.14.0/user/basics.indexing.html#boolean-or-mask-index-arrays" rel="nofollow noreferrer">boolean indexing</a> with NumPy.<br/>
For example: </p>
<pre><code>import numpy as np

x_diff = np.array([0, 2, 2, 0, 0, 2])
t_diff = np.array([0, 1, 2, 3, 4, 5])

print(t_diff[x_diff == 2])
</code></pre>
<p>gives:</p>
<pre><code>array([1, 2, 5])
</code></pre>
<hr/>
<p>If you don't want to use NumPy, then you can use <a href="https://docs.python.org/3.6/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehensions</a> with <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip</a>:</p>
<pre><code>x_diff = [0, 2, 2, 0, 0, 2]
t_diff = [0, 1, 2, 3, 4, 5]

print([t for t, x in zip(t_diff, x_diff) if x == 2])
</code></pre>
<p>gives:</p>
<pre><code>[1, 2, 5]
</code></pre>
</div>
