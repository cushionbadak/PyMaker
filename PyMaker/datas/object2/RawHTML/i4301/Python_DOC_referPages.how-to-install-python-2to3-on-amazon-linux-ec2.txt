<div class="post-text" itemprop="text">
<p>If you know the name of a command, you can use <code>yum provides</code> to see if it's available in any packages in the repo:</p>
<pre><code>yum provides \*bin/2to3
</code></pre>
<p>When I ran this on Amazon Linux, I found the following packages: <code>python27-tools</code> and <code>python36-tools</code>.</p>
<p>If it were not available in any packages, you could install this using <code>pip</code>:</p>
<pre><code>sudo pip install 2to3
</code></pre>
