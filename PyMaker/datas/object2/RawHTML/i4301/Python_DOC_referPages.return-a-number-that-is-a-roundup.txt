<p>Python program that will take two numbers: a and b.Return a number that is
a roundup version of a so that it is multiple of b:
Examples: </p>
<pre><code>a=13, b = 5, return 15.
a=15, b = 3, return 15
a = 16, b = 3, returns 18'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>A solution using <a href="https://docs.python.org/3/library/math.html#math.ceil" rel="nofollow noreferrer"><code>math.ceil</code></a>:</p>
<pre><code>import math

def round_up(a,b):
  return math.ceil(a/b) * b

print(round_up(13, 3))
print(round_up(16, 3))
</code></pre>
<p>Output:</p>
<pre><code>15
18
</code></pre>
