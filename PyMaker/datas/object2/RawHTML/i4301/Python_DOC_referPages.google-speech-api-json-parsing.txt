<div class="post-text" itemprop="text">
<p>The JSON parser in Python expects your blob to use double quotation marks, since that is the <a href="http://www.json.org/" rel="nofollow noreferrer">JSON standard</a>.</p>
<pre><code>{
  "name": "John Doe"
}
</code></pre>
<p>You could replace the single quotes with double quotes, as explained in <a href="https://stackoverflow.com/a/21154138/1878974">this answer</a>. </p>
<p>However, I’m pretty sure the problem can be solved elsewhere, since the Google API most likely uses valid JSON in it’s responses. How do you parse the response from Google’s API?</p>
