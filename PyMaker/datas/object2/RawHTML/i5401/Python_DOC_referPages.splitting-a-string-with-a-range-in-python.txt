<p>I have a file which contains something similar to the following lines:</p>
<pre><code>[&lt;data_type0&gt;,&lt;data_type1&gt;] name(data)
"DATA_VALUE0"|"DATA_VALUE1" name(data)
</code></pre>
<p>I am looking to split each line into two. The first part being between either the '&lt;' and '&gt;' the '[' and ']' or  " and ".</p>
<p>So the output from the desired split would be something like:</p>
<pre><code>valueA[0] = [data_type0,data_type1]
valueA[1] = [name(data)]

valueB[0] = [DATA_VALUE0,DATA_VALUE1]
valueB[1] = [name(data)]
</code></pre>
<p>One snag is that the data values are of an unknown length, so some lines could read:</p>
<pre><code>&lt;date_type0&gt; name(data)
</code></pre>
<p>and others could be:</p>
<pre><code>&lt;data_type0&gt;,&lt;data_type1&gt;,&lt;data_type2&gt;...&lt;data_type8&gt; name(data)
</code></pre>
<p>Any ideas how?</p>
</div>
<div class="post-text" itemprop="text">
<p>What you are looking for is <a href="https://docs.python.org/3/library/stdtypes.html#str.rsplit" rel="nofollow noreferrer"><code>rsplit()</code></a>:</p>
<h3>Code:</h3>
<pre><code>lines = (
    '"[ &lt; data_type0 &gt;, &lt; data_type1 &gt;] name(data)',
    '"DATA_VALUE0" | "DATA_VALUE1" name(data)',
)

for line in lines:
    print(line.rsplit(' ', 1))
</code></pre>
<h3>Results:</h3>
<pre><code>['"[ &lt; data_type0 &gt;, &lt; data_type1 &gt;]', 'name(data)']
['"DATA_VALUE0" | "DATA_VALUE1"', 'name(data)']
</code></pre>
</div>

<div class="post-text" itemprop="text">
<p>There is also a general approach to finding stuff in strings and breaking them apart.</p>
<pre><code>a = '&lt;data_1&gt;,&lt;data_2&gt; name(data)'
division = a.find('name(')
b = a[:division-1]
c = a[division:]
</code></pre>
<p>Results:</p>
<pre><code>&gt;&gt;&gt; b
'&lt;data_1&gt;,&lt;data_2&gt;'
&gt;&gt;&gt; c
'name(data)'
</code></pre>
