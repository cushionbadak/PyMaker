<div class="post-text" itemprop="text">
<p>I cannot compile with Python 3, i.e. the .pyc files are not generated.</p>
<p>For example:</p>
<pre><code>python -m py_compile test.py
</code></pre>
<p>generates test.pyc</p>
<p>Hovewer, </p>
<pre><code>python3 -m py_compile test.py
</code></pre>
<p>does not <strong>generate</strong> test.pyc</p>
<p>test.py</p>
<pre><code>print("hello world")
</code></pre>
<p>Why? </p>
</div>
<div class="post-text" itemprop="text">
<p>In Python 3, you can find <strong>.pyc</strong> file in the <code>__pycache__</code> directory, beside your <code>test.py</code> script.</p>
<h3>From terminal:</h3>
<pre><code>$ python3 -m py_compile test.py
$ ls __pycache__
 test.cpython-36.pyc
</code></pre>
<h3>Inside python3</h3>
<p>Use <code>py_compile</code> module:</p>
<pre><code>&gt;&gt;&gt; import py_compile
&gt;&gt;&gt; py_compile.compile('test.py')
</code></pre>
<p>See:
<a href="https://docs.python.org/3/library/py_compile.html" rel="nofollow noreferrer">https://docs.python.org/3/library/py_compile.html</a></p>
</div>
<span class="comment-copy">Did you check the <code>__pycache__</code> directory? By default, <code>/foo/bar/baz.py</code> will be compilied into <code>/foo/bar/__pycache__/baz.cpython-36.pyc</code>. (The exact name varies depending on python version).</span>
