<div class="post-text" itemprop="text">
<p>you can build your new file contents first and afterward write to it:</p>
<pre><code>import os

i = open("in", "r+")
o = open("out", "a+")

result = ""

for line in i.readlines():
        if ":" not in line:
                result += line.replace(" ", "\n")


o.write(result)

i.close()
o.close()
</code></pre>

<div class="post-text" itemprop="text">
<ul>
<li><p>Replaces the spaces with newlines <em>before</em> iterating over the file's contents. if you don't, then you'll only be iterating over a single line.</p></li>
<li><p>There's no need for the <code>os</code> library in your code.</p></li>
<li><p>Call <code>o.write(line)</code> after testing if the line has no colon(<code>:</code>).</p></li>
<li><p>Use the context manger to do your I/O operations.</p></li>
<li><p>Use more descriptive names than <code>i</code> and <code>o</code> to avoid writing confusing code.</p></li>
</ul>
<p>With the above modifications, your code becomes:</p>
<pre><code>with open("input.txt", "r+") as infile, open("output.txt", "a+") as outfile:
    contents = infile.read().replace(' ', '\n')
    for line in contents.split('\n'):
        if ':' not in line:
            outfile.write(line + '\n')
</code></pre>
