<div class="post-text" itemprop="text">
<p>A Value error is </p>
<blockquote>
<p>Raised when a built-in operation or function receives an argument that has the right type but an inappropriate value</p>
</blockquote>
<p>the <code>float</code> function can take a string, ie <code>float('5')</code>, it's just that the value <code>'string'</code> in <code>float('string')</code> is an inappropriate (non-convertible) string</p>
<p>On the other hand,</p>
<blockquote>
<p>Passing arguments of the wrong type (e.g. passing a list when an int is expected) should result in a TypeError</p>
</blockquote>
<p>so you would get a <code>TypeError</code> if you tried <code>float(['5'])</code> because a list can never be converted into a float.</p>
<p><a href="https://docs.python.org/3/library/exceptions.html" rel="nofollow noreferrer">Cite</a></p>
</div>
<span class="comment-copy">Some strings can be converted to float, for example <code>123</code>.</span>
<span class="comment-copy">And it's the <b>value</b> of the string (<code>'string'</code>) that's problematic because it is not convertible to a float.</span>
<span class="comment-copy">Another valid example is <code>float("infinity")</code>.</span>
<span class="comment-copy">Another one is <code>float("nan")</code></span>
