<p>I have a list of dictionaries:</p>
<pre><code>dicty = [defaultdict(list)] * (2)
dicty

[defaultdict(list, {}), defaultdict(list, {})]
</code></pre>
<p>Now I want to know how to index into these dictionaries?</p>
<p>I have tried:</p>
<pre><code>dicty[0][0].append('abc')
dicty

[defaultdict(list, {0: ['abc']}), defaultdict(list, {0: ['abc']})]
</code></pre>
<p>But as you can see that it appends in both the dictionaries. I want to learn how to append into an individual dictionary.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>from collections import defaultdict

""" Here, you are saying that you wish to add twice "defaultdict(list)".
    These are the same objects. You are just duplicating the same one with *
"""
dicty = [defaultdict(list)] * (2)
dicty[0][0].append('abc')
print dicty
print dicty[0] is dicty[1], '\n'

""" However, here : you are creating two distinct dictionnaries.
"""
dicty = []
for _ in range(2):
    dicty.append(defaultdict(list))
dicty[0][0].append('abc')
print dicty
print dicty[0] is dicty[1]

# outputs :
#[defaultdict(&lt;type 'list'&gt;, {0: ['abc']}), defaultdict(&lt;type 'list'&gt;, {0: ['abc']})]
#True 

#[defaultdict(&lt;type 'list'&gt;, {0: ['abc']}), defaultdict(&lt;type 'list'&gt;, {})]
#False
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use:</p>
<pre><code>dicty = [defaultdict(list) for _ in range(2)]
</code></pre>
<p>When using <code>[defaultdict(list)] * (2)</code>, all your dicts point to the same object in memory.</p>
