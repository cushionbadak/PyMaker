<div class="post-text" itemprop="text">
<p>You can use <code>.split()</code> to split a string.
It seems that we can split from the last occurence of "_", remove the part after "." to get the timestamp. <br/>
So, method to return timestamp from the file_name is:</p>
<pre><code>def get_timestamp(file_name):
    return file_name.split("_")[-1].split('.')[0]
</code></pre>
<p>As all the dates are of same format, python can sort those using the timestamp string itself.<br/>
To get the sorted list of filenames using that timestamp, you can do:</p>
<pre><code>sorted_list = sorted(list_files, key=get_timestamp)
</code></pre>
<p>More about the <a href="https://docs.python.org/3/howto/sorting.html#key-functions" rel="nofollow noreferrer">Key</a> function can be learned from official python documentation.</p>
</div>
