<div class="post-text" itemprop="text">
<p>I have a json dumped string dumped in a file.
Here is the file format - </p>
<pre><code>{
u'key1':u'abc'
}
</code></pre>
<p>It's invalid json because its should have double quotes. How do I convert this into valid json?</p>
</div>
<div class="post-text" itemprop="text">
<p>If you have read this data from your file:</p>
<pre><code>s = """
{
    u'key1':u'abc'
}
"""
</code></pre>
<p>You may be able to convert it to a Python object using <code>ast.literal_eval()</code>:</p>
<pre><code>import ast
data = ast.literal_eval(s)
# data = {'key1': 'abc'}
</code></pre>
<p>Note that it might not work with other contents, because it seems you dumped a string (<code>str()</code> or <code>repr()</code>) representation of a dictionary to a file, instead of JSON. Other, more complex types may not be readable by <code>ast.literal_eval()</code>. 
You should use the <code>json</code> module instead to produce and write proper JSON to the file.</p>
<p>For reference:</p>
<ul>
<li><a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow noreferrer">https://docs.python.org/3/library/ast.html#ast.literal_eval</a></li>
<li><a href="https://docs.python.org/3/library/json.html" rel="nofollow noreferrer">https://docs.python.org/3/library/json.html</a></li>
</ul>
<p>As it seems you are working with Python 2:</p>
<ul>
<li><a href="https://docs.python.org/2/library/ast.html#ast.literal_eval" rel="nofollow noreferrer">https://docs.python.org/2/library/ast.html#ast.literal_eval</a></li>
<li><a href="https://docs.python.org/2/library/json.html" rel="nofollow noreferrer">https://docs.python.org/2/library/json.html</a></li>
</ul>
