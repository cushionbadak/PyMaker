<p>Follow-up to my <a href="https://stackoverflow.com/questions/48352804/decoding-utf-8-string-using-python-3-6#comment-83692457">comment above</a>.</p>
<p>You have to reverse the sequence of operation when you decode a <a href="https://en.wikipedia.org/wiki/Base64" rel="nofollow noreferrer">Base64</a> encoded string:</p>
<pre><code>&gt;&gt;&gt; s = "hasni zinda ha u are my man boy yes u are "
# Encode the Python str into bytes.
&gt;&gt;&gt; b = s.encode("utf-8")
# Base64 encode the bytes.
&gt;&gt;&gt; s_b64 = base64.b64encode(b)
&gt;&gt;&gt; print("Encoding: " + str(s_b64))
Encoding: b'aGFzbmkgemluZGEgaGEgdSBhcmUgbXkgbWFuIGJveSB5ZXMgdSBhcmUg'
</code></pre>
<p>Now that you have the encoded string, decoding works in reverse order:</p>
<pre><code># Base64 decode the encoded string into bytes.
&gt;&gt;&gt; b = base64.b64decode(s_b64)
# Decode the bytes into str.
&gt;&gt;&gt; s = b.decode("utf-8")
print("Decoding: " + s)
Decoding: hasni zinda ha u are my man boy yes u are 
</code></pre>
<p>For more details, please see the documentation for <a href="https://docs.python.org/3/library/base64.html#base64.b64encode" rel="nofollow noreferrer"><code>b64encode()</code></a> and <a href="https://docs.python.org/3/library/base64.html#base64.b64decode" rel="nofollow noreferrer"><code>b64decode()</code></a>, as well as the <a href="https://en.wikipedia.org/wiki/Base64#Output_padding" rel="nofollow noreferrer">Output padding</a> section for Base64 (required to ensure that a Base64 encoded string’s length is divisible by 4).</p>
<p>To use your two-liners:</p>
<pre><code>&gt;&gt;&gt; a = base64.b64encode(bytes("hasni zinda ha u are my man boy yes u are ", "utf-8"))
&gt;&gt;&gt; print("Encoding:", a)
Encoding: b'aGFzbmkgemluZGEgaGEgdSBhcmUgbXkgbWFuIGJveSB5ZXMgdSBhcmUg'
&gt;&gt;&gt; b = base64.b64decode(a).decode("utf-8")
&gt;&gt;&gt; print("Decoding:", b)
Decoding: hasni zinda ha u are my man boy yes u are 
</code></pre>
