<div class="post-text" itemprop="text">
<p><code>snap_date.head()</code> returns <code>pandas.core.frame.DataFrame</code> object:</p>
<pre><code>&gt;&gt;&gt; type(snap_date.head())
pandas.core.frame.DataFrame
</code></pre>
<p>You should convert it to str, because <a href="https://docs.python.org/3/library/logging.html#logging.debug" rel="nofollow noreferrer">logger.debug</a> arguments (msg and *args) must be format string and string arguments. Try <a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.to_string.html#pandas.DataFrame.to_string" rel="nofollow noreferrer">pandas.DataFrame.to_string</a>:</p>
<pre><code>logger.debug("Snap_date:\n %s", snap_date.head().to_string())
</code></pre>
