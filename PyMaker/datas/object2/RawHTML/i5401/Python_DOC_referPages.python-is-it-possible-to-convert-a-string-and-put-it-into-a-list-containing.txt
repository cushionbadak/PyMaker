<p>If it is a list of elements:</p>
<pre><code>a = "2012-01-01 09:00    San Jose    Men's Clothing    214.05    Amex"
print [i for i in a.split("    ")]
</code></pre>
<p><strong>Result:</strong></p>
<pre><code>['2012-01-01 09:00', 'San Jose', "Men's Clothing", '214.05', 'Amex']
</code></pre>
<p><em>or if it is a list of tuple:</em></p>
<pre><code>a = "2012-01-01 09:00    San Jose    Men's Clothing    214.05    Amex"
print [tuple(i for i in a.split("    "))]
</code></pre>
<p><strong>Result:</strong></p>
<pre><code>[('2012-01-01 09:00', 'San Jose', "Men's Clothing", '214.05', 'Amex')]
</code></pre>
<p><em>And if you have multiple line of the string:</em></p>
<pre><code>a = """2012-01-01 09:00    San Jose    Men's Clothing    214.05    Amex
2012-01-01 09:00    San Jose    Men's Clothing    214.05    Amex
2012-01-01 09:00    San Jose    Men's Clothing    214.05    Amex
2012-01-01 09:00    San Jose    Men's Clothing    214.05    Amex
2012-01-01 09:00    San Jose    Men's Clothing    214.05    Amex"""


print [tuple(j.split("    ")) for j in a.split("\n")]
</code></pre>
<p><strong>Result:</strong></p>
<pre><code>[('2012-01-01 09:00', 'San Jose', "Men's Clothing", '214.05', 'Amex'), ('2012-01-01 09:00', 'San Jose', "Men's Clothing", '214.05', 'Amex'), ('2012-01-01 09:00', 'San Jose', "Men's Clothing", '214.05', 'Amex'), ('2012-01-01 09:00', 'San Jose', "Men's Clothing", '214.05', 'Amex'), ('2012-01-01 09:00', 'San Jose', "Men's Clothing", '214.05', 'Amex')]
</code></pre>
