<div class="post-text" itemprop="text">
<p>So you can simply use <a href="https://docs.python.org/3/library/argparse.html" rel="nofollow noreferrer">argparser</a> for it. </p>
<pre><code>import argparse

parser = argparse.ArgumentParser(description='Process input data')
parser.add_argument('--density', type=int, dest='density',
                    help='Density Ratio (%)')
parser.add_argument('--ra', type=int, dest='ra', 
                    help='Surface Roughness- Ra (um)')

args = parser.parse_args()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The easiest solution for such a simple use case would be to use <code>sys.argv</code>.</p>
<p>After <code>from sys import argv</code> you can access the <code>argv</code> object, which is a ordered list of all script arguments starting with the script name as the first argument.</p>
