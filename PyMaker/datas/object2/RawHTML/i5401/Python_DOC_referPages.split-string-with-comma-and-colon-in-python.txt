<p>I'm developing a program that has the following input type:</p>
<pre><code>&lt;anything&gt;:&lt;anything&gt;,&lt;anything&gt;,&lt;anything&gt;
</code></pre>
<p>So it's a string divided by colon and comma. I have to get each part of <code>&lt;&gt;</code> separated so I was thinking about using the <code>split()</code> function from Python. The thing is, how can I split both colons and commas?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>result = [fragment.split (',') for fragment in the_input.split (':')]
</code></pre>
<p>(Assuming that there's a hierarchy and , has a higher priority than : in the syntax of your strings, which is probably the case)</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>string = "&lt;anything&gt;:&lt;anything&gt;,&lt;anything&gt;,&lt;anything&gt;"

wordList = string.split(",")

wordList2 = []
for item in wordList:
    for string in item.split(":"):
        wordList2.append(string)

print(wordList2)
</code></pre>
