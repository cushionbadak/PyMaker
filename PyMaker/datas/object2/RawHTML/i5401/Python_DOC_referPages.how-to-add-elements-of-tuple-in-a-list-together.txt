<div class="post-text" itemprop="text">
<p>You need to initialize the <code>sales</code> variable and add to it the <code>float</code> value of each string as you loop through the list of tuples.</p>
<pre><code>myList = [('2012-01-01', '09:00', 'San Jose', "Men's Clothing", '214.05', 'Amex'), 
          ('2012-01-01', '09:00', 'Fort Worth', "Women's Clothing", '153.57', 'Visa'), 
          ('2012-01-01', '09:00', 'San Diego', 'Music', '66.08', 'Cash')]

sales = 0

for x in range(len(myList)):
    sales += float(myList[x][4])

print(sales)
# 433.7
</code></pre>
<p>You could do it without <code>range()</code> and <code>len()</code> in the loop:</p>
<pre><code>sales = 0

for tup in myList:
    sales += float(tup[4])

print(sales)
# 433.7
</code></pre>
<p>You could also use a generator expression:</p>
<pre><code>print(sum(float(tup[4]) for tup in myList))
# 433.7
</code></pre>
