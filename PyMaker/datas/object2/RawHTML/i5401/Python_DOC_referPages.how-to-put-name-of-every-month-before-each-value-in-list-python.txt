<div class="post-text" itemprop="text">
<p>I would just create a single for-loop, with the help of <a href="https://docs.python.org/3/library/calendar.html#calendar.month_name" rel="nofollow noreferrer">Calendar API</a>:</p>
<pre><code>import calendar 

M = [0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

print(M[0], end=" ")  # For whatever reason you are printing this value

for i in range(1,len(M)):
    print(calendar.month_name[i],"=",(i - (i // 7) - ((i + 1) // 7)),end=" ")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/calendar.html#calendar.month_name" rel="nofollow noreferrer">Calendar API</a> to do this if you don't wish to create list of months manually. It will <code>calendar.month_name[i]</code> will outputs <code>January</code> if <code>i=0</code> and so on...</p>
<pre><code>&gt;&gt;&gt; import calendar
&gt;&gt;&gt; M = [0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
&gt;&gt;&gt; mod_sums = [ (s - (s // 7) - ((s + 1) // 7)) * 8 for s in M]
&gt;&gt;&gt; values = str(mod_sums[0]) + " " + ' '.join("{} = {}".format(calendar.month_name[i], v) for i,v in enumerate(mod_sums[1:],start=1))
</code></pre>
<p><strong>Outputs</strong></p>
<pre><code>&gt;&gt;&gt; values
&gt;&gt;&gt; '0 January = 184 February = 160 March = 184 April = 176 May = 184 June = 176 July = 184 August = 184 September = 176 October = 184 November = 176 December = 184'
</code></pre>
