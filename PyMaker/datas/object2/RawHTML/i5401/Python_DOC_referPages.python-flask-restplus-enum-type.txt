<p>Am using <code>python 3.4.</code> I have created an <code>enum</code> type as shown below:</p>
<pre><code>import enum
class EnumGender(enum.Enum):
    blank = ' '
    female = 'Female'
    male = 'Male'
    other = 'Other'
</code></pre>
<p>my question is how do i assign it to flask-restplus field as the only examples i can see are: </p>
<p><code>fields.String(description='The object type', enum=['A', 'B'])</code></p>

<div class="post-text" itemprop="text">
<p>You can assign the member names to it:</p>
<pre><code>fields.String(description='The object type', enum=EnumGender._member_names_)
</code></pre>
</div>

<span class="comment-copy">Is it possible to use a enum.IntEnum with fields.Integer?</span>
<span class="comment-copy"><a href="https://docs.python.org/3/library/enum.html#iteration" rel="nofollow noreferrer">Docs</a> tell me that there is no <code>Enum._member_names_</code> but there is <code>Enum.__members__</code> which is a view. So you'll have to do <code>fields.String(description='The object type', enum=EnumGender.__members__.items())</code>. A little verbose.</span>
<span class="comment-copy">For me <i>_members__items() throws unserializable error, although _member_names</i> is protected but it works.</span>
