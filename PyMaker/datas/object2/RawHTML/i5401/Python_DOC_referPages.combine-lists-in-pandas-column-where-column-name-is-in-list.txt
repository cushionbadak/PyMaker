<p><strong>Option 1</strong><br/>
Slice on the columns and call <code>sum</code> along the first axis. </p>
<pre><code>df['new'] = df[['allmz','allint','allx']].sum(axis=1)
</code></pre>

<p>If you are having a large set of column's name then an easy way to solve this problem is shown below :</p>
<pre><code>col = df.loc[: , "allint":"allx"]
</code></pre>
<p>where "allint" is the start column name and "allx" is the end column name</p>
<pre><code>df['new'] = col.sum(axis=1)
df
</code></pre>
<p>This will give you the same result you got after writing the name of each columns.</p>
