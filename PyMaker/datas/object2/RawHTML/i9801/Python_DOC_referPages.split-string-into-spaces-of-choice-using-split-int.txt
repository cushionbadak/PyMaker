<div class="post-text" itemprop="text">
<p>You could get that output by using <code>rsplit</code> and performing only one split, i.e:</p>
<pre><code>'Christopher Arthur Hansen Brooks'.rsplit(' ', 1)
</code></pre>
<p>Which returns a list:</p>
<pre><code>['Christopher Arthur Hansen', 'Brooks']
</code></pre>
<p>That you can unpack into <code>firstname</code> and <code>lastname</code>:</p>
<pre><code>firstname, lastname = 'Christopher Arthur Hansen Brooks'.rsplit(' ', 1)
</code></pre>
<p>For input that might be short (i.e user only enters first name) it is better to use <code>rpartition</code> if you also want to unpack; the unpacking simply must handle the 3-element tuple returned:</p>
<pre><code>firstname, _, lastname = 'Christopher Arthur Hansen Brooks'.rpartition(' ')
</code></pre>
