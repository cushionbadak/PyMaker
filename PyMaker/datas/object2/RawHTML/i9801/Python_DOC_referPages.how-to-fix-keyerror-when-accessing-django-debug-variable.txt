<div class="post-text" itemprop="text">
<p>The <code>KeyError</code> means that the <code>DJANGO_DEBUG</code> envariable was not set. To avoid it you can either call <code>get()</code> on the <a href="https://docs.python.org/3/library/os.html#os.environ" rel="nofollow noreferrer"><code>os.environ</code></a> mapping or use the <a href="https://docs.python.org/3/library/os.html#os.getenv" rel="nofollow noreferrer"><code>os.getenv()</code></a> function:</p>
<pre><code>DEBUG = os.environ.get('DJANGO_DEBUG') == 'True'
</code></pre>
<p>or</p>
<pre><code>DEBUG = os.getenv('DJANGO_DEBUG') == 'True'
</code></pre>
