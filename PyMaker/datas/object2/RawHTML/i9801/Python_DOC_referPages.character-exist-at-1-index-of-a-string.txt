<div class="post-text" itemprop="text">
<p>Negative indices in Python means they are relative to the end of the sequence. Which means -1 will give you the last, and -2 the second last, etc.</p>
<p>Or, if you prefer, you can think of the string as circular:</p>
<pre><code>  -3-2-1 0 1 2 3 4 5 6
...l l o h e l l o h e...
</code></pre>
</div>
<span class="comment-copy">Strings, as sequences, can be indexed with negative numbers. <code>-1</code> signaling the last element, <code>-2</code> the second from last on so on. See the section on <a href="https://docs.python.org/3/tutorial/introduction.html#strings" rel="nofollow noreferrer">Strings</a> in the Python Tutorial.</span>
