<p>I can do this in my ipython notebook:</p>
<pre><code>print(u"\u2605")
</code></pre>
<blockquote>
<p>★</p>
</blockquote>
<p>But how do I go backwards? That is, going from the symbol to the unicode string. Encoding it in UTF-8 or UTF-16 is giving binary representations. For example:</p>
<pre><code>print('★'.encode('utf-16'))
</code></pre>
<blockquote>
<p>b'\xff\xfe\x05&amp;'</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/codecs.html#text-encodings" rel="nofollow noreferrer"><code>unicode-escape</code> encoding</a>:</p>
<pre><code>&gt;&gt;&gt; '★'.encode('unicode-escape')
b'\\u2605'
&gt;&gt;&gt; print('★'.encode('unicode-escape').decode())
\u2605
</code></pre>
<p>or <a href="https://docs.python.org/3/library/functions.html#ord" rel="nofollow noreferrer"><code>ord</code></a> if you just want to know the codepoint:</p>
<pre><code>&gt;&gt;&gt; ord('★')
9733
&gt;&gt;&gt; hex(ord('★'))  # as hexa decimal
'0x2605'
&gt;&gt;&gt; print(r'\u%x' % ord('★'))
\u2605
</code></pre>
<p><strong>UPDATE</strong></p>
<p>You can also use <a href="https://docs.python.org/3/library/functions.html#ascii" rel="nofollow noreferrer"><code>ascii</code></a>:</p>
<pre><code>&gt;&gt;&gt; print(ascii('★'))  # NOTE: surrounding quote
'\u2605'
&gt;&gt;&gt; print(ascii('★').strip("'"))
\u2605
</code></pre>
</div>
