<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer"><code>input</code></a> returns a string; string cannot be equal to <code>int</code>:</p>
<pre><code>&gt;&gt;&gt; '5' == 5   # str &lt;-&gt; int
False

&gt;&gt;&gt; int('5') == 5  # int &lt;-&gt; int
True
</code></pre>
<p>You need to convert the string to <code>int</code>:</p>
<pre><code>wordNumber = int(input("Word Number: "))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Python treats input as a string so you have to convert it to an int to get the number:</p>
<pre><code>wordNumber = int(input("Word Number: "))
addedWords = 0  
wordList = []


while addedWords != wordNumber:
    Word = input("Word: ")
    wordList.append(Word)
    addedWords = addedWords + 1
</code></pre>
</div>
