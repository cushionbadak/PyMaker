<div class="post-text" itemprop="text">
<p>Regex should get close. Unfortunately, has to read the whole file content to a string. </p>
<pre><code>import re

test_str = '''Girl: I am girl.

Boy:

I am boy

Dog: Bark

Cat: 

Meow'''

regex = r':\s*\n+'
subst = ": "

for line in re.sub(regex, subst, test_str, 0, re.MULTILINE).split('\n'):
    if line != '':
        print (line.strip())
</code></pre>
<p>Output</p>
<pre><code>Girl: I am girl.
Boy: I am boy
Dog: Bark
Cat: Meow
</code></pre>
