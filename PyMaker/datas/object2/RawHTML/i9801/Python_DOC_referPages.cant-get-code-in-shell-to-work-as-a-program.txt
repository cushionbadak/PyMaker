You need convert the string to list before passing to the function using <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>str.split</code></a></p>
<pre><code>myList = input().split()  # 'apple banana' -&gt; ['apple', 'banana']
</code></pre>
