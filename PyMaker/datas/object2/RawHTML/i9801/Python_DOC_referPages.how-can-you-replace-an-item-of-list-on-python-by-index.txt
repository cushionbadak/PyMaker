<p>you are iterating over the list element, but later you are using the element as an index where you need an integer instead. 
seems like what you want should be as below</p>
<pre><code>msg = raw_input("Msg: ")
dmsg = list(msg)
for i in range(len(dmsg)):
    if msg[i] == "^":
        dmsg[i] = " "
print dmsg
</code></pre>
<p>OR</p>
<pre><code>msg = raw_input("Msg: ")
dmsg = list(msg)
for n,i in enumerate(dmsg):
    if i == "^":
        dmsg[n] = " "
print dmsg
</code></pre>
<p>better way can just use <code>str.replace</code> method</p>
<pre><code>msg = raw_input("Msg: ")
dmsg = msg.replace('^', ' ')
</code></pre>
