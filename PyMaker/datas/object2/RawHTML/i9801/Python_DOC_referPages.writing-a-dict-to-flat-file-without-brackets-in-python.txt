<p>I have this dict:</p>
<pre><code>my_dict = {'1': {'a', 'b'},
          '2': {'b', 'd'},
          '3': {'c'}
          }
</code></pre>
<p>I need to write it to a flat file then I wrote this code:</p>
<pre><code>with open("myfile", 'w') as f:
    for key, value in my_dict.items():
        f.write('{} {}\n'.format(key, value))
</code></pre>
<p>I got this:</p>
<pre><code>2 {'d', 'b'}
1 {'b', 'a'}
3 {'c'}
</code></pre>
<p>It works (order doesn't matter) but I need the output file without any extra character, just the values of the dict keys and values itself, something like this:</p>
<pre><code>1 a
2 b
3 c
</code></pre>
<p>How can I do it?</p>
</div>
<div class="post-text" itemprop="text">
<p>Convert the <code>set</code> items to string using <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>str.join</code></a> before write to the file:</p>
<pre><code>&gt;&gt;&gt; ' '.join({'d', 'b'})
'b d'
</code></pre>
<hr/>
<pre><code>for key, value in my_dict.items():
    f.write('{} {}\n'.format(key, ' '.join(value)))
</code></pre>
</div>
