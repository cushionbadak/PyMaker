<p>Using python 3.5 is it possible to have a list of definitions and have a function choose and call that random definition? I am asking this because all of the docs I have seen for random just says about generating a random pseudo number.</p>

<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; def foo(): print('foo')
&gt;&gt;&gt; def bar(): print('bar')
&gt;&gt;&gt; from random import choice
&gt;&gt;&gt; choice([foo, bar])
&lt;function foo at 0x10499d668&gt;
&gt;&gt;&gt; choice([foo, bar])()
foo
&gt;&gt;&gt; choice([foo, bar])()
bar
&gt;&gt;&gt; choice([foo, bar])()
foo
