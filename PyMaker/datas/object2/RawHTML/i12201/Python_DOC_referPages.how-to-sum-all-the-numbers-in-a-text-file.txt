<p>I have to calculate the sum of any numbers in the file and print the sum. </p>
<p>A number is defined as any string beginning with a digit 0 through 9 followed by any number of digits 0 through 9. </p>
<p>Alphanumeric strings (strings including both numbers and letters) are not to be included in the summation.</p>
<p>This is the content of the file:</p>
<pre><code>a b cddde ff 1
5
hH five lll 0
l 10
99 abcd7
9kk
0
</code></pre>
<p>So the answer would be 115 in this case.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>def function():

    infile = open("test.txt", 'r')
    content = infile.read()       
    infile.close()
    wordList = content.split()

    total = 0

    for i in wordList:
        if i.isnumeric():
            total += int(i)
    return total
</code></pre>
<p>In this solution, I named the file test.txt. The idea is you loop through wordList which is a list containing every item spliced in test.txt (try printing wordList before the loop to see for yourself). We then try to cast each item as a int (this assumes no decimals will be in the file, if so you can include a float cast). We then catch ValueError which gets raised when casting i.e 'a' as an int. </p>

<p>All you need to do is use <code>item.isnumeric()</code>.  If the item made up of only numbers and not letters or other characters it will return true. </p>
<p>So you check the all the items in <code>wordList</code> and if the item <code>isnumeric()</code> you add the item to <code>total</code>.</p>

<p>I'd suggest that use RegEx:</p>
<pre><code>import re

with open('file') as f:
    print(sum(int(i) for i in re.findall(r'\b\d+\b', f.read())))
</code></pre>
<p>In this case:</p>
<ul>
<li><p><code>\b+</code> match all the numbers, and <code>\b</code> checks if there a letter after (or before) the number so we can ignore <code>abcd7</code> or <code>9kk</code>.</p></li>
<li><p><a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>re.findall()</code></a> try to find all the numbers in the file use the RegEx <code>\b\d+\b</code> and returns a list.</p></li>
<li><p>A <em>list compression</em>, <code>int(i) for i in re.findall(r'\b\d+\b')</code>, convert all the elements in the list which returned by <code>re.findall()</code> to <code>int</code> object.</p></li>
<li><p><a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum()</code></a> built-in function sums the elements of a list, and returns the result.</p></li>
</ul>
