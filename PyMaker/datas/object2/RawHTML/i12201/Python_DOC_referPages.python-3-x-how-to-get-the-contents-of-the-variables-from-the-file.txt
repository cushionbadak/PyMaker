<div class="post-text" itemprop="text">
<p>You could read it in as a dictionary:</p>
<pre><code>d = {}
with open("myfile.conf", 'r') as f :
    for line in f:
        if not line.strip():
            continue
        key,value = line.split('=')
        d[key.strip()] = value.strip().replace("'","")
</code></pre>
<p>You could then get the variables by accessing the dictionary:</p>
<pre><code>print(d['var1']) 
'foo'
</code></pre>
