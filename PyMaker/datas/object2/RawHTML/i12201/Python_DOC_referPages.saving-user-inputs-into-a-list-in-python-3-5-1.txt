<p>I want to prompt the user for a list of phrases over and over again until they input <code>END</code> and save all of the user's inputs into a list. How can I do that?</p>

<div class="post-text" itemprop="text">
<p>One way to do is with a <code>while loop</code>:</p>
<pre><code>phraseList = []

phrase = input('Please enter a phrase: ')

while phrase != 'END':
    phraseList.append(phrase)
    phrase = input('Please enter a phrase: ')

print(phraseList)
</code></pre>
<p>Result:</p>
<pre><code>&gt;&gt;&gt; Please enter a phrase: first phrase
&gt;&gt;&gt; Please enter a phrase: another one
&gt;&gt;&gt; Please enter a phrase: one more
&gt;&gt;&gt; Please enter a phrase: END
&gt;&gt;&gt; ['first phrase', 'another one', 'one more']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can simply use <a href="https://docs.python.org/3/library/functions.html#iter" rel="nofollow"><code>iter(input, 'END')</code></a>, which returns a <code>callable_iterator</code>. Then we can use <code>list()</code> to get a <em>real</em> list:</p>
<pre><code>&gt;&gt;&gt; l = list(iter(input, 'END'))
foo
bar
foobar
END
&gt;&gt;&gt; l
['foo', 'bar', 'foobar']
</code></pre>
<p>About how does it work, if you take a look at <code>help(iter)</code>:</p>
<pre><code>iter(...)
    iter(iterable) -&gt; iterator
    iter(callable, sentinel) -&gt; iterator

    Get an iterator from an object.  In the first form, the argument must
    supply its own iterator, or be a sequence.
    In the second form, the callable is called until it returns the sentinel.
</code></pre>
<hr/>
<p>You can also use a <code>while</code> loop if you think it's more simple and clear anyway:</p>
<pre><code>l = []
while True:
    password = input()
    if password != 'END':
        l.append(password)
    else:
        break
</code></pre>
