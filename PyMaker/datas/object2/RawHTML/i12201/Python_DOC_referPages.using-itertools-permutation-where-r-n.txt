<p>I am trying to generate all permuations of a set of items and the my R needs to be larger than the size of the set of items</p>
<p>Here is an example :</p>
<pre><code>itertools.permutations ("ABC", 4)
</code></pre>
<p>this always returns 0 items as R &gt; N. </p>
<p>I want this</p>
<blockquote>
<p>[A, A, A, A]</p>
<p>[A, A, A, B]</p>
<p>[A, A, B, A]</p>
<p>[A, B, A, A]</p>
<p>...</p>
</blockquote>
<p>How can I achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You don't seem to want the permutation, but the <a href="https://en.wikipedia.org/wiki/Cartesian_product" rel="noreferrer">Cartesian product</a>:</p>
<pre><code>itertools.product("ABC", repeat=4)
</code></pre>
<p><a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="noreferrer">https://docs.python.org/3/library/itertools.html#itertools.product</a></p>
</div>
