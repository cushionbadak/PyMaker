<div class="post-text" itemprop="text">
<p>You are receiving a JSON encoded response. You can parse that with the <a href="https://docs.python.org/3/library/json.html#json.loads" rel="nofollow"><code>json.loads()</code></a> function:</p>
<pre><code>import json
import urllib.request

with urllib.request.urlopen('http://xkcd.com/info.0.json') as response:
    data = json.loads(response.read().decode('utf8'))

&gt;&gt;&gt; data
{'link': '', 'transcript': '', 'month': '2', 'year': '2016', 'alt': 'Using diacritics correctly is not my fortÃ©.', 'num': 1647, 'img': 'http://imgs.xkcd.com/comics/diacritics.png', 'day': '24', 'safe_title': 'Diacritics', 'news': '', 'title': 'Diacritics'}
</code></pre>
<p>This is easier with the <a href="http://docs.python-requests.org" rel="nofollow"><code>requests</code></a> module:</p>
<pre><code>import requests
response = requests.get('http://xkcd.com/info.0.json')
data = response.json()

&gt;&gt;&gt; data
{'link': '', 'transcript': '', 'month': '2', 'year': '2016', 'alt': 'Using diacritics correctly is not my fortÃ©.', 'num': 1647, 'img': 'http://imgs.xkcd.com/comics/diacritics.png', 'day': '24', 'safe_title': 'Diacritics', 'news': '', 'title': 'Diacritics'}
</code></pre>
<p><code>requests</code> saves you the hassle of decoding the incoming data and decoding the JSON.</p>

<div class="post-text" itemprop="text">
<p>In Python 2.7 you need to <code>import urllib2</code> and then <code>import json</code> to load the data into a variable as a Python dictionary. Resource <a href="https://docs.python.org/2/howto/urllib2.html" rel="nofollow">here</a></p>
<pre><code>import urllib2
import json
response = urllib2.urlopen('http://xkcd.com/info.0.json')
html = response.read().decode('utf8')

data = json.loads(html)
type(data) is dict # True
</code></pre>
