<p>I want to create a list of datetimes that increment by 10 seconds. The format should be consistent with the following:</p>
<pre><code>start = '2016_02_23_14_32_50'
end = '2016_02_24_17_16_00'
</code></pre>
<p>I'm aware that I could use deltatime but I'm unsure what the correct implementation would be.</p>
</div>
<div class="post-text" itemprop="text">
<p>First, convert your string to a <code>datetime</code> object with <code>strptime</code>. Then you can increment it by 10 seconds with <code>timedelta</code>.</p>
<pre><code>import datetime

start = '2016_02_23_14_32_50'
dt = datetime.datetime.strptime(start, '%Y_%m_%d_%H_%M_%S')
dt += datetime.timedelta(seconds=10)
</code></pre>
