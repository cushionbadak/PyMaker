<p>You should look <a href="https://docs.python.org/2/library/re.html" rel="nofollow">here</a> for python2 and <a href="https://docs.python.org/3/library/re.html" rel="nofollow">here</a> for python3. Now I will write a posible solution, but you should read befor use it.</p>
<pre><code>re.sub('POSTBYNAVN.*?POSTNR_TIL', '', original)
</code></pre>

<div class="post-text" itemprop="text">
<p>This gets rid of every <code>POSTBYNAVN</code> part. I think You can figure out the rest by yourself. Cheers.    </p>
<pre><code>re.sub('\"POSTBYNAVN\":\s\"[a-zA-z]+\"\,\s', '', original)
</code></pre>
<p>EDIT:</p>
<p>Unfortunately this one doesn't match <code>"POSTBYNAVN": "M\xc3\xa5l\xc3\xb8v"</code> because it contains special symbols, so I have changed the regex to:</p>
<pre><code>re.sub('\"POSTBYNAVN\":\s\"([^\s]+)\s', '', original)
</code></pre>
<p><code>([^\s]+)</code> matches any word until first space and now it works as intended.
However the final solution doesn't even use this and looks like this:</p>
<pre><code>re.sub('\"POSTBYNAVN.*?POSTNR_TIL\":\s', '', original)
