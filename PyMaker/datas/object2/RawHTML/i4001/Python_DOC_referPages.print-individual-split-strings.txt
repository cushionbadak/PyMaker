<div class="post-text" itemprop="text">
<p>The following code takes a string such as abcd#1234, removes the '#' and splits it into abcd and 1234</p>
<pre><code>import sys
import re

print("User ID: {0}").format(sys.argv[1])
print("User Type: {0}").format(sys.argv[2])

sub = re.sub('[#]', '', sys.argv[1])
split = re.match(r"([a-z]+)([0-9]+)", sub, re.I)
print(split.groups())
</code></pre>
<p>The output from <code>print(split.groups())</code> is 
&lt;'abcd', '1234'&gt;
How can I take an individual section of the split such as abcd and just output that?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>split.group(1) # --&gt; 'abcd'
split.group(2) # --&gt; '1234'
</code></pre>

<div class="post-text" itemprop="text">
<p>Use index.</p>
<p><strong>Ex:</strong></p>
<pre><code>s = "abcd#1234"
import re
sub = re.sub('[#]', '', s)
split = re.match(r"([a-z]+)([0-9]+)", sub, re.I)
print(split.groups()[0])
print(split.groups()[1])
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>abcd
1234
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can simply get the values of each as shown below:</p>
<pre><code>data = split.groups()
if len(data) == 2:
    name, id = data
    print(name, id)
</code></pre>
<p>Or you can use <a href="https://docs.python.org/2/library/re.html" rel="nofollow noreferrer"><code>group([group1, ...])</code></a>.</p>

<div class="post-text" itemprop="text">
<p>You can also use split directly and get just tell it to take the second element.</p>
<pre><code>import sys
import re

string="abcd#1234"
string1=string.split("#")[1]  # tells it to take the second element of the split

print(string1) # Prints 1234
</code></pre>
