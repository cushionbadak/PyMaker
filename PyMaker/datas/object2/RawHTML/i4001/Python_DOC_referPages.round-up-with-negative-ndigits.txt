<p>I have the following below code, which works as :</p>
<pre><code>num = 191360.789242721
round(num , -3)/1000
&gt;&gt;&gt; 191.0
</code></pre>
<p>But how I can round UP and get <code>192.0</code> ?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/math.html#math.ceil" rel="nofollow noreferrer"><code>math.ceil()</code></a> as a ready-made solution or do it manually with a reverse floor division:</p>
<pre><code>num = 191360.789242721
num_ceil = -(-num // 1000)  # 192.0
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>num = 191360.789242721
math.ceil(num/1000)
&gt;&gt;&gt; 192
</code></pre>
</div>
<span class="comment-copy">Round up would be <code>math.ceil(num / 1000)</code></span>
