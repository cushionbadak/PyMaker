<p>I'm currently tasked with replacing <code>"SUPER MAN"</code> with <code>"S^U^P^E^R M^A^N"</code>.</p>
<p>I've tried <code>print("SUPER MAN".replace("" "", "^"))</code> but end up with <code>^S^U^P^E^R^ ^M^A^N^</code>.</p>
<p>How can I fix this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You want to use the <a href="https://docs.python.org/2/library/string.html#string.join" rel="nofollow noreferrer"><code>join</code></a> method along with <a href="https://docs.python.org/2/library/string.html#string.split" rel="nofollow noreferrer"><code>split</code></a>.</p>
<p>First <code>split</code> <code>"SUPER MAN"</code> by a space to get:  </p>
<pre><code>words = "SUPER MAN".split()
["SUPER", "MAN"]
</code></pre>
<p>Then for each of those entries, separate each character with <code>"^"</code> using <code>join</code>:</p>
<pre><code>new_words = map(lambda x: "^".join(x), words)
["S^U^P^E^R", "M^A^N"]
</code></pre>
<p>Then turn it back into one string separated by a space using <code>join</code>, and your resulting code is:</p>
<pre><code>words = "SUPER MAN".split()
new_words = map(lambda x: "^".join(x), words)
print " ".join(new_words)
</code></pre>
<p>Output: <code>S^U^P^E^R M^A^N</code></p>
<p><strong>What your code is doing:</strong></p>
<p>First, notice that the first parameter that you pass into <code>replace</code> is not a space, which is what you presumably intended for it be, but the expression <code>"" ""</code>, which is an implicit way of doing string concatenation.</p>
<p>Thus, <code>"SUPER MAN".replace("" "", "^")</code> becomes <code>"SUPER MAN".replace("", "^")</code>, which replaces each occurrence of <code>""</code> with <code>"^"</code>.</p>
<p>The result of such an operation may at first seem somewhat ill-defined, but in Python, since you can think of a string <code>s</code> as being equal to <code>"" + "".join(s) + ""</code>, replacing each occurrence of <code>""</code> with <code>"^"</code> would yield what you see: a <code>"^"</code> between each letter and a <code>"^"</code> on the ends.</p>
<p>Passing a <code>" "</code> as the first parameter to <code>replace</code> doesn't work, because it will replace spaces (specifically, the space in the middle) with <code>"^"</code>, so you'd get <code>"SUPER^MAN"</code>.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; re.sub(r"(\w)(?=\w)", r"\1^", "SUPER MAN")
'S^U^P^E^R M^A^N'
</code></pre>
<p>For more on why that works: <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">https://docs.python.org/3/library/re.html</a></p>
