<div class="post-text" itemprop="text">
<p><code>raw_input</code> reads from standard input whereas <code>sys.argv</code> is the list of command line arguments. So if you want to use command line to pass in the country code, just delete the line containing <code>raw_input</code> and let <code>country</code> be <code>sys.argv[1]</code>.</p>
