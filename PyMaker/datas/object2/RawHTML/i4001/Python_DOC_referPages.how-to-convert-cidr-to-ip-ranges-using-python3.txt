<p>How to convert list like:  </p>
<pre><code>94.192.0.0/14  
94.0.0.0/12  
93.96.0.0/16 
</code></pre>
<p>To:  </p>
<pre><code>94.192.0.0-94.195.255.255  
94.0.0.0-94.15.255.255  
93.96.0.0-93.96.255.255  
</code></pre>
<p>Using python3?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/3/library/ipaddress.html" rel="nofollow noreferrer"><code>ipaddress</code></a> builtin module:</p>
<pre><code>&gt;&gt;&gt; import ipaddress

&gt;&gt;&gt; net=ipaddress.ip_network('94.192.0.0/14')
IPv4Network('94.192.0.0/14')

&gt;&gt;&gt; '%s-%s' % (net[0], net[-1])
'94.192.0.0-94.195.255.255'
</code></pre>
