<p>If I were to <strong>read</strong> 10 input files and store them in variable names <code>file1</code>, <code>file2</code>, <code>file3</code> etc that are dynamically created inside a for loop using string concatenation, how would I store the values in them?
For example,</p>
<pre><code>'file'+'1'=pandas.read_csv(...)
</code></pre>
<p>doesn't work.
What are the alternatives to create variables dynamically to store values that are available dynamically?</p>
</div>
<div class="post-text" itemprop="text">
<p>One option is to use a <code>dict</code> structure:</p>
<pre><code>datasets = {}
for i in range(10):
    key = "file{}".format(i)
    datasets[key] = pd.read_csv(...)
</code></pre>
</div>
<span class="comment-copy">Just use a <a href="https://docs.python.org/3/library/stdtypes.html#dict" rel="nofollow noreferrer"><code>dict</code></a> and store your file 'names' as keys and your CSV readers as values.</span>
