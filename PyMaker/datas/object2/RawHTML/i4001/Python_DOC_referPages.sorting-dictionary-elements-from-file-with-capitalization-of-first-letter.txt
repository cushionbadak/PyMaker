<div class="post-text" itemprop="text">
<p>The problem is <code>range</code>, in your case, only needs an <code>int</code> input. You are supplying a <code>list</code> input. Specifically, <code>lst[0:len(lst)][0]</code> outputs the first element of <code>lst</code>, which is a list itself.</p>
<p>To loop over your list of lists by row, you should use <code>range(len(lst))</code>:</p>
<pre><code>for i in range(len(lst)):
</code></pre>
<p>See the <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer">docs</a> for more information on <code>range</code> syntax.</p>
