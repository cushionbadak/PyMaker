<div class="post-text" itemprop="text">
<p>As the <a href="https://docs.python.org/2/library/random.html#random.randint" rel="nofollow noreferrer">documentation</a> says:</p>
<blockquote>
<p>random.randint(a, b)</p>
<p>Return a random integer N such that a &lt;= N &lt;= b.</p>
</blockquote>
<p>It seems your problem is in the statement:</p>
<p><code>index = random.randint(0,len(target))</code></p>
<p>The position <code>len(target)</code> is invalid.</p>
<p>I think it should be </p>
<p><code>index = random.randint(0,len(target)-1)</code></p>
</div>
