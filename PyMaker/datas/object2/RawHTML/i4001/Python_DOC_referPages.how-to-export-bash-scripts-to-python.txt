<p>You can use subprocess to execute bash/shell scripts in your python code. Here is an example:</p>
<pre><code>import subprocess

command = "npm-cli-login -u "+user+" -p \""+password+"\" -e "+email+" -r \""+self.nexusUrl+"/repository/deploy.npm.cloud/\" --config-path npmrc"

process = subprocess.Popen(command, stdout=subprocess.PIPE, shell=True)
output, error = process.communicate()
</code></pre>
<p>You can read more here: <a href="https://docs.python.org/3/library/subprocess.html" rel="nofollow noreferrer">https://docs.python.org/3/library/subprocess.html</a></p>

<div class="post-text" itemprop="text">
<p>The easiest way would be to create a bash script with your desired content and then execute it from within python.
Make sure to give it <code>execution Permissions</code> and possibly change the <code>Path</code> to the script if its not in the same directory.</p>
<pre><code>#!/usr/bin/python

import subprocess
subprocess.call("./mybashscript.sh", shell=True)
</code></pre>
