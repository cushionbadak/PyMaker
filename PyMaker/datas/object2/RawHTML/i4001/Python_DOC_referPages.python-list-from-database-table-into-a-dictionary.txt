<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/stdtypes.html#dict" rel="nofollow noreferrer"><code>dict()</code></a> constructor accepts an iterable with <code>key, value</code> pairs and <code>cursor.fetchall()</code> produces exactly that, so all you need to do is just instantiate a <code>dict</code> with the returned result:</p>
<pre><code>def items():
    cur.execute("SELECT name,price FROM items")
    return dict(cur.fetchall())
</code></pre>
</div>
