<p>I have a list that contains elements and want to delete some elements that has particular keyword.</p>
<p>For example:</p>
<p>List contains:
list = ['red rose','blue rose','red color','yellow paper']</p>
<p>And I'm looking for deleting lists with keyword 'RED' and get output as</p>
<p>Output:    ['blue rose', 'yellow paper']</p>
<p>Thank you.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>list = ['red rose','blue rose','red color','yellow paper']
keyword = 'RED'
for i in list:
    if i.find(keyword.lower()) != -1:
        list.remove(i)
print(list)
</code></pre>

<div class="post-text" itemprop="text">
<p>There are primarily two ways of doing this:</p>
<p>Using list comprehension:</p>
<pre><code>lst = ['red rose', 'blue rose', 'red color', 'yellow paper']

output = [i for i in lst if 'red' not in i.lower()]
</code></pre>
<p>Using a for loop:</p>
<pre><code>lst = ['red rose', 'blue rose', 'red color', 'yellow paper']

output = []
for i in lst:
    if 'red' not in i.lower():
        output.append(i)
</code></pre>
<p>Hope that helps!</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; foo = ['red rose','blue rose','red color','yellow paper']

&gt;&gt;&gt; [x for x in foo if 'red' not in x]
['blue rose', 'yellow paper']
</code></pre>
<p>This of course assumes lower case letters.
And note it doesn't delete entries, but rather creates a new list object.</p>

<div class="post-text" itemprop="text">
<p>All what you need is a loop:</p>
<pre><code>s =  ['red rose','blue rose','red color','yellow paper']
s = [x for x in s if not str(x).lower().__contains__('red')]
print(s)
</code></pre>
