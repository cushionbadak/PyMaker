<p>I have a nested list and I want to find the largest value stored in a specific index of the nested list.</p>
<p>For example: <code>data = [[12345678, 14, 1],[135763365, 12, 0],[1234, 9, 0]]</code></p>
<p>I want to return the largest value stored in index 0 of the list <code>data</code>: <code>135763365</code>. </p>
<p>How can I return this? I don't want to brute force figure out the largest myself and return that value. I want to write a function that will return this value in a nested list.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <code>list comprehension</code> and apply <a href="https://docs.python.org/3/library/functions.html#max" rel="nofollow noreferrer">max()</a> function over result.</p>
<pre><code>max_number = max([item[0] for item in data])
</code></pre>
<p>Output</p>
<pre><code>135763365
</code></pre>
<p>Another approach is to specify key in the <code>max</code> function.</p>
<pre><code>max(data, key=operator.itemgetter(0))[0]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html#max" rel="nofollow noreferrer"><code>max()</code></a> function takes a <code>key</code> parameter like:</p>
<pre><code>data = [[12345678, 14, 1], [135763365, 12, 0], [1234, 9, 0]]
print(max(data, key=lambda x: x[0])[0])
</code></pre>
<h3>Results:</h3>
<pre><code>135763365
</code></pre>
</div>
