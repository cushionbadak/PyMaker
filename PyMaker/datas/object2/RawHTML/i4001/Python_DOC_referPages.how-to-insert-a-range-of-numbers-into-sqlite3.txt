<div class="post-text" itemprop="text">
<p>the second argument of <a href="https://docs.python.org/3/library/sqlite3.html?highlight=sqlite#sqlite3.Cursor.executemany" rel="nofollow noreferrer"><code>executemany</code></a> needs to be iterable (i.e. it will do the looping for you):</p>
<pre><code>def data_entry():
    data = ( (i, ) for i in range(1, 10) )
    cursor.executemany("INSERT INTO Tester(Tester_1) VALUES(?)", data)
    ....
</code></pre>
<p>should do. note that each entry in the second argument (when iterating over) needs to be a <code>tuple</code>. therefore the 1-tuple <code>(i, )</code> in the <code>data</code> generator.</p>
</div>
<div class="post-text" itemprop="text">
<p>The second argument of <code>executemany</code> is an iterable (per row to execute) of iterables (per parameters). So the correct syntax is:</p>
<pre><code>cursor.executemany("INSERT INTO Tester(Tester_1) VALUES(?)", ((x,) for x in range(1, 10)))
cursor.close()
connector.commit()
</code></pre>
