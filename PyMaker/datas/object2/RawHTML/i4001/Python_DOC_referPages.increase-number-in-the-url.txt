<div class="post-text" itemprop="text">
<p>You want to just increase the value of <code>i</code> by <code>10</code> every time. You can use a builtin function like <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer"><code>range</code></a> in python. Do check it out, <code>range</code> is used quite often in conjunction with the <code>for</code> loop in python</p>
<pre><code>for i in range(1 , 100, 10):
    url = "https://www.minsalud.gov.co/sites/rid/Paginas/freesearchresults.aspxk=&amp;k=Salud%20Mental%20Legislacion#k=%2CSalud%20Mental%20Legislacion=+ "+ str(i)+")"
    print(url)
</code></pre>
<p>The above <code>for</code> loop will start with a value of <code>i = 1</code>, with an increment of <code>10</code> every time until <code>i</code> reaches a maximum value that is less than <code>100</code>. The value of <code>i</code> in the above for loop will be</p>
<pre><code>1
11
21
31
41
51
61
71
81
91
</code></pre>
<p>When <code>i</code> becomes <code>101</code>, it ends the for loop.</p>

        f.write(url+'\n')
</code></pre>
