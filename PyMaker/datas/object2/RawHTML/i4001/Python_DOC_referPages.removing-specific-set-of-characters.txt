<div class="post-text" itemprop="text">
<p>This can be done very easily by <strong>regular expression</strong>.
You can read more about python regular expression <a href="https://docs.python.org/3/howto/regex.html" rel="nofollow noreferrer">here</a>.</p>
<p>You can use it like this-</p>
<pre><code>import re
str1 = 'there is three --- and now single - and now two --'
str2 = re.sub('---', '', str1)
print(str2)
</code></pre>
<p>Output-</p>
<pre><code>there is three  and now single - and now two --
</code></pre>

<div class="post-text" itemprop="text">
<p>You can use <code>str.replace(patt, repl)</code>:</p>
<pre><code>s = "---test--test-test----"
print(s.replace("---", ""))
# "test--test-test-"
</code></pre>
<p>You could also do it using <code>re</code> if you want something more extensible.</p>
