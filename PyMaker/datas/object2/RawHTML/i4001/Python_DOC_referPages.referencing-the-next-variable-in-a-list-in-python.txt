<div class="post-text" itemprop="text">
<p>I assume 1) you want to compare consecutive elements by the code segment given. 2) You want to print the actual value rather than the index.</p>
<pre><code>for i in range(len(values)-1):
    if values[i+1] &gt; values[i]:
        print(str(values[i+1]) + " &gt; " + str(values[i]))
    else:
        print(str(values[i+1]) + " &lt; " + str(values[i]))
</code></pre>
<p>You need to add <code>range(len(values)-1)</code> so that you <code>i</code> would be the index rather than the value at the position.</p>
