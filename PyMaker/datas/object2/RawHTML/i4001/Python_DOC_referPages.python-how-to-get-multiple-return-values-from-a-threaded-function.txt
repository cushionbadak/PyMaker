<div class="post-text" itemprop="text">
<p>You should use a <a href="https://docs.python.org/3/library/queue.html" rel="nofollow noreferrer"><code>queue</code></a> for retrieve data from threads, here you have an example using a wrapper to store values from the functions into a queue:</p>
<pre><code>import threading
import queue

my_queue = queue.Queue()

def storeInQueue(f):
  def wrapper(*args):
    my_queue.put(f(*args))
  return wrapper


@storeInQueue
def get_name(full_name):
   return full_name, full_name



t = threading.Thread(target=get_name, args = ("foo", ))
t.start()

my_data = my_queue.get()
print(my_data)
</code></pre>
