<pre><code>data = [[12345678, 14, 1],[135763365, 14, 0],[135763365, 12, 0],[1234, 9, 0]]
</code></pre>
<p>where I want to find the max index 1 of all the sublists. </p>
<p>To do this I've created this function: </p>
<pre><code>def findBusiestPeriod ():
    result = max([item[1] for item in data])
    counter = collections.Counter(itertools.chain(*data))
    if counter[result] &gt; 2:
       # Here's where I want to find the smallest index[0] based on 
       # the max index[1]

    return result
</code></pre>
<p>Once I have the max index <code>1</code> of the list, I want to check to see if there are duplicates (there are in the example.) If there are duplicates I want to return only the sublist with the smallest index <code>0</code>. </p>
<p>How can I do this?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer"><code>sorted()</code></a> can do that with a key like :</p>
<h3>Code:</h3>
<pre><code>sorted(data, key=lambda x: (-x[1], x[0]))[0]
</code></pre>
<h3>Test Code:</h3>
<pre><code>data = [[12345678, 14, 1], [135763365, 14, 0],
        [135763365, 12, 0], [1234, 9, 0]]

print(sorted(data, key=lambda x: (-x[1], x[0]))[0])
</code></pre>
<h3>Results:</h3>
<pre><code>[12345678, 14, 1]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As someone already shown the <code>key</code> parameter of <code>sorted</code> function, I want to mention that the <code>min</code> and <code>max</code> functions also have a <code>key</code> parameter.</p>
<pre><code>min(data, key=lambda x: (-x[1], x[0]))
</code></pre>
</div>
<span class="comment-copy">Sorting is wasteful if you only want the max. <code>max</code> supports <code>key</code> as well.</span>
