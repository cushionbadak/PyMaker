<div class="post-text" itemprop="text">
<p>From Python 3.4 upwards you can use just those two lines for that task:</p>
<pre><code>from pathlib import Path
mp3files = list(Path('.').glob('**/*.txt'))
</code></pre>
<p>More info: <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow noreferrer">https://docs.python.org/3/library/pathlib.html</a></p>

<p>why don't you use the glob module that you have imported ?</p>
<pre><code>mp3files = glob.glob('*.txt')
</code></pre>
<p>this will return a list of all mp3 files in your current working directory.
if your files are in a different directory and not in your cwd:</p>
<pre><code>path_to_files_dir = os.path.join(os.getcwd(), 'your_files_dir_name', '*.txt')

mp3files = glob.glob(path_to_files)
</code></pre>
