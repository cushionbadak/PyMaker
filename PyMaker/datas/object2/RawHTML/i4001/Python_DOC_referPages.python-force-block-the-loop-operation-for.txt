<div class="post-text" itemprop="text">
<p>To exit a <code>for</code> or <code>while</code> loop, use the keyword <code>break</code> (<a href="https://docs.python.org/3.5/reference/simple_stmts.html#the-break-statement" rel="nofollow noreferrer">docs</a>). This works for "infinite" loops too (like the one below).
Example:</p>
<pre><code>n = 0
while True:
    print("Running")
    if n == 3:
        break
    n += 1
</code></pre>
<p>will print:</p>
<pre><code>Running
Running
Running
Running
</code></pre>
<p>(since it'll run when <code>n</code> is <code>0</code>, <code>1</code>, <code>2</code> and <code>3</code> -&gt; 4 times)</p>
