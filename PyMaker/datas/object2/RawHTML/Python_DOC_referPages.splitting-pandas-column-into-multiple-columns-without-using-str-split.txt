<p>Is there a way in Pandas to split a column into multiple columns? I have a columns in a dataframe where the contents are as follows:</p>
<pre><code>Test1   Test2   Salary
1   Akash   100_$
2   Akash   200_@
3   Yogi    300_%
4   Akash   400_$
</code></pre>
<p>I would like to split this into:</p>
<pre><code>Test1   Test2   Salary  Currency
1   Akash   100_$   $
2   Akash   200_@   @
3   Yogi    300_%   %
4   Akash   400_$   $
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>using <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.Series.str.extract.html" rel="nofollow noreferrer"><code>s.str.extract</code></a></p>
<pre><code>df['Currency']=df.Salary.str.extract('(\W)')
#or df['Currency']=df.Salary.str.split("_").str[1]
</code></pre>
<blockquote>
<p>\W --&gt; Any character that is not a letter, numeric digit, or the underscore character.</p>
</blockquote>
<hr/>
<pre><code>print(df)

   Test1  Test2 Salary Currency
0      1  Akash  100_$        $
1      2  Akash  200_@        @
2      3   Yogi  300_%        %
3      4  Akash  400_$        $
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you only want the last character you can just use:</p>
<pre><code>df['Currency'] = df.Salary.str[-1]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Without using <code>str.split</code> try following.</p>
<pre><code>df['Currency']=df.Salary.str.replace(".*_", "")

<p><strong><em>OR(in case someone wants to use <code>str.split</code>:</em></strong></p>
<p>Could you please try following, using <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.Series.str.split.html" rel="nofollow noreferrer">str.split</a> here.</p>
