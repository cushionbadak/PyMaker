<div class="post-text" itemprop="text">
<p>You're looking at the docs for Python 3.3 but running Python 3.2.  In Python 3.2 the <code>Request</code> initializer doesn't have a <code>method</code> argument: <a href="http://docs.python.org/3.2/library/urllib.request.html#urllib.request.Request" rel="nofollow">http://docs.python.org/3.2/library/urllib.request.html#urllib.request.Request</a></p>
<p>FWIW depending on what kind of request you make (for example if the request includes a body) <code>urllib</code> will automatically use the appropriate method (i.e. POST).  If you need to make a more specialized type of request such as <code>HEAD</code> you need to dig a little deeper.  There are other answers on SO that help with that.</p>
