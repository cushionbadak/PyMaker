<div class="post-text" itemprop="text">
<p>I think the problem is that you are on Python 3.x but using code that only works on 2.x.</p>
<p>In Python 3.x, the <code>urlopen</code> function is contained in <a href="http://docs.python.org/3/library/urllib.request.html#module-urllib.request" rel="nofollow"><code>urllib.request</code></a>:</p>
<pre><code>&gt;&gt;&gt; from urllib.request import urlopen
&gt;&gt;&gt; urlopen
&lt;function urlopen at 0x020DA7C8&gt;
&gt;&gt;&gt;
</code></pre>
<hr/>
<p><strong>Edit:</strong></p>
<p>I think this does everything you want:</p>
<pre><code>from urllib.request import urlopen
page = urlopen('http://www.pythonchallenge.com/pc/def/linkedlist.php?nothing=12345').read()
print(page)
</code></pre>
