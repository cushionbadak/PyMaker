<pre><code>filename = input('Please enter a filename: ')
col = int(input('Please pick a column number: '))

</code></pre>
<p>or you could accept a command-line argument; <a href="http://docs.python.org/3/library/sys.html#sys.argv" rel="nofollow"><code>sys.argv</code></a> gives you a list of command-line parameters, where index <code>0</code> is the script name, and anything beyond that are values the user put on the command line:</p>
<pre><code>import sys

if len(sys.argv) &lt; 3:
    print('Please specify a filename and column number: {} [csvfile] [col]'.format(sys.argv[0]))
    sys.exit(1)

filename = sys.argv[1]
col = int(sys.argv[2])
</code></pre>
<p>The Python standard library includes a <a href="http://docs.python.org/3/library/argparse.html" rel="nofollow"><code>argparse</code> module</a> to make handling command line interfaces more flexible and powerful.</p>
