<div class="post-text" itemprop="text">
<p>As aelfric5578 said, <code>python</code> defaults to <code>python3</code></p>
<p>But if you wish to run it with python3 then you can search the Arch repos with <code>pacman -Ss urllib</code> which will a return a list of packages you can <code>pacman -S</code></p>
<p>As a naming scheme, Arch sticks to <code>python-libname</code> for python 3 or <code>python2-libname</code> for python2, so it's usually a safe bet to <code>pacman -S python-missinglib</code></p>
</div>
<span class="comment-copy">What Python version are you using? If 2.x, then <code>urllib2</code> does not need to be installed, it's a standard module. If 3.x, then you need to change the imports <a href="http://docs.python.org/3/library/urllib.html" rel="nofollow noreferrer">accordingly</a>.</span>
