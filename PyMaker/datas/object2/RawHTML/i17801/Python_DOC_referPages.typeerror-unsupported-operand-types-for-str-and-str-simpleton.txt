<div class="post-text" itemprop="text">
<p>You cannot substract <code>str</code> object. (<a href="http://docs.python.org/3/library/functions.html#input"><code>input</code></a> in Python 3.x returns <code>str</code> object unlike <a href="http://docs.python.org/2/library/functions.html#input"><code>input</code></a> in Python 2.x)</p>
<pre><code>&gt;&gt;&gt; '2' - '1'
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: unsupported operand type(s) for -: 'str' and 'str'
</code></pre>
<p>Convert them into <a href="http://docs.python.org/3/library/functions.html#int"><code>int</code></a> or <a href="http://docs.python.org/3/library/functions.html#float"><code>float</code></a> first.</p>
<pre><code>&gt;&gt;&gt; int('2') - int('1')
1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to convert the strings to integers first which can be done using the built-in <a href="http://docs.python.org/2/library/functions.html#int" rel="nofollow"><code>int()</code></a> or <a href="http://docs.python.org/2/library/functions.html#float" rel="nofollow"><code>float()</code></a> functions</p>
<p>So Try this instead:</p>
<pre><code>weight= int(input("How much do you weigh?"))
target= int(input("What would you like to weigh?"))
</code></pre>
<h1>Note:</h1>
<p><em>In python, You cannot perform mathematical functions such as division or subtraction  on a string without converting it to a integer first However string addition and multiplication can be performed</em>, <strong>However for your case neither string addition or multiplication will work</strong></p>
<p>Example:</p>
<pre><code>&gt;&gt;&gt; 'Hi' + ' Bye!'
'Hi Bye!'
&gt;&gt;&gt; 'Hi' * 5
'HiHiHiHiHi'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are apparently using python 3. The result of <code>input</code> is always a string, so you must convert them to integers before you can do arithmetic. (In python 2, <code>input</code> will interpret what you type in as python code, so your code would have worked as intended). </p>
<pre><code>weight=int(input("How much do you weigh? "))
</code></pre>
<p>etc.</p>
<p>You can "add" strings together with <code>+</code>, but there's no corresponding subtraction operation. And if there was, given the way string "addition" works, subtracting <code>"25" - "5"</code> would have given you <code>"2"</code>! So you you must convert to integers to get the intended result (a corresponding program about <em>gaining</em> weight would not have triggered a python error, but would have given very worrisome results).</p>
