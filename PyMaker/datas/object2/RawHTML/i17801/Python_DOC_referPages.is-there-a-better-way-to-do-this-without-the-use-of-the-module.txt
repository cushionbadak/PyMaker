<p>Convert binary number, hexadecimal number representation to int using <a href="http://docs.python.org/3/library/functions.html#int" rel="nofollow"><code>int</code></a>:</p>
<pre><code>&gt;&gt;&gt; int('ff', 16) # 16: hexadecimal
255
</code></pre>
<p>Then, use <a href="http://docs.python.org/3/library/functions.html#format" rel="nofollow"><code>format</code></a> or <a href="http://docs.python.org/3/library/stdtypes#str.format" rel="nofollow"><code>str.format</code></a> to convert in to binary, hexadecimal representation.</p>
<pre><code>&gt;&gt;&gt; format(255, 'b') # binary
'11111111'
&gt;&gt;&gt; format(255, 'x') # hexadecimal
'ff'
</code></pre>
<p>Prepend <code>0n</code> (n is number) if you want the result to be zero-padded.</p>
<pre><code>&gt;&gt;&gt; format(5, '08b')
'00000101'
&gt;&gt;&gt; format(5, '02x')
'05'
</code></pre>
