<p>The datetime.timedelta support divide operate as shown in the <a href="http://docs.python.org/3/library/datetime.html#timedelta-objects" rel="nofollow">manual</a></p>
<p>So, I think you should do is:</p>
<ol>
<li>Minuse the datetime.datetime.min <code>a_interval = a - datetime.datetime.min</code>, <code>b_interval = b - datetime.datetime.min</code></li>
<li>Start calculate: <code>print(a_interval - b_interval)</code> and <code>print(a_interval / b_interval)</code></li>
</ol>

<div class="post-text" itemprop="text">
<p>You  have to make the difference between <a href="http://docs.python.org/2/library/datetime.html" rel="nofollow"><code>datetime.datetime</code></a> (which is an actual date and time, for example "1899-12-30 19:45:00") and <a href="http://docs.python.org/2/library/datetime.html#datetime.timedelta" rel="nofollow"><code>datetime.timedelta</code></a> (which is a period, for example "1 hour"). 
Note that your   <code>a-b</code> substraction of two datetimes will result in a timedelta.</p>
<p>If you're calculating race times, you have to specify the start time of the race (how would you know, anyway, how long the race lasted). Then you can do</p>
<pre><code>import datetime
start = datetime.datetime.strptime("1899-12-30 12:00:00", "%Y-%m-%d %H:%M:%S") # whatever the start time is
a = datetime.datetime.strptime("1899-12-30 19:45:00", "%Y-%m-%d %H:%M:%S")
b = datetime.datetime.strptime("1899-12-30 12:42:00", "%Y-%m-%d %H:%M:%S") 
print (a-b)  # it will give you the timedelta difference between racer "a" and racer "b" 
print (a-start).total_seconds()/(b-start).total_seconds() # it will give you the ratio between "a" and "b" race times
</code></pre>
<p>Division is not defined for datetimes. But timedelta objects have <code>total_seconds()</code> which give you a number (the length of period in seconds), and you can divide numbers.</p>
</div>
