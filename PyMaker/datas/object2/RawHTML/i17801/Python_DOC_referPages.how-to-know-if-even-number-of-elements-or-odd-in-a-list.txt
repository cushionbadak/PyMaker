<div class="post-text" itemprop="text">
<p>Even numbers are divisible by 2. Odd numbers are not.</p>
<p><code>len(X)</code> will get the lenght of <code>X</code></p>
<p>If the length of <code>X</code> is divisible by 2, then it is an Even number
If the lenght of <code>X</code> is not divisible by 2 then it is an Odd Number</p>
<p><code>len(X)%2</code> returns the "remainder" of a division problem</p>
<p>for example <code>5%2</code> will return <code>1</code> which is NOT zero, ( because 5 divided by 2 is 2 with a remainder of 1) therefore it is not even.
Same thing as <code>6%4</code> which would return a <code>2</code>, because 6 divided by 4 is 1 with a remainder of 2.</p>
<p>so <code>len(X)%2</code> where <code>X</code> is your list, will return either a 1, indicating it is Odd, or a 0 indiciating it is Even.</p>
</div>
<div class="post-text" itemprop="text">
<p>Using the modulus operator - <code>%</code> gives the remainder. To get if even, you need to divide by 2, for example:</p>
<pre><code>7 % 2 = 1
8 % 2 = 0
</code></pre>
<p>The most pythonic way to do it is:</p>
<pre><code>def is_even(your_list):
    if len(your_list) % 2:  # 0 in case of even i.e. No remainder
        return True
    else:
        return False
</code></pre>
