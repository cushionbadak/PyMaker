<p>This program generates a user defined amount of random numbers and then writes to a file. The program works fine as written, but i want the text file use \n to concatenate. What am I doing wrong?</p>
<p>#This program writes user defined
    #random numbers to a file</p>
<pre><code>import random

randfile = open("Randomnm.txt", "w" )

for i in range(int(input('How many to generate?: '))):
    line = str(random.randint(1, 100))
    randfile.write(line)
    print(line)

randfile.close()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Add "\n":</p>
<pre><code>import random

randfile = open("Randomnm.txt", "w" )

for i in range(int(input('How many to generate?: '))):
    line = str(random.randint(1, 100)) + "\n"
    randfile.write(line)
    print(line)

randfile.close()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could also make use of Python 3's <code>print</code> function's <code>file</code> keyword argument:</p>
<pre><code>import random

with open("Randomnm.txt", "w") as handle:
    for i in range(int(input('How many to generate?: '))):
        n = random.randint(1, 100)

        print(n, file=handle)
        print(n)

# File is automatically closed when you exit the `with` block
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>file.write()</code> simply writes text to a file. It does not concatenate or append anything, so you need to append a <code>\n</code> yourself.</p>
<p><em>(Note that the type would be called <code>_io.TextIOWrapper</code> in Python 3)</em></p>
<p>To do this, simply replace</p>
<pre><code>line = str(random.randint(1, 100))
</code></pre>
<p>with </p>
<pre><code>line = str(random.randint(1, 100))+"\n"
</code></pre>
<p>This will append a newline to every random number.</p>
</div>
