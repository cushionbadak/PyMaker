<p>Do you want <a href="http://docs.python.org/3/library/functions.html#property" rel="nofollow">property</a> ?</p>
<pre><code>from datetime import date

class Person:
    def __init__(self, fname, lname, byear):
        self.first_name = fname
        self.last_name = lname
        self.birth_year = byear

    @property
    def age(self):
        return date.today().year - self.birth_year

    def __str__(self):
        return '{0.first_name}, {0.last_name} Age:{0.age}'.format(self)

p1 = Person('Olivia', 'Parker', 1985)
print(p1)
</code></pre>
<p>prints</p>
<pre><code>Olivia, Parker Age:28
</code></pre>
<p>BTW, the code is mixing use of <code>lower_case_with_underscores</code>, <code>mixedCase</code> for attributes. Choose one. See <a href="http://www.python.org/dev/peps/pep-0008/#descriptive-naming-styles" rel="nofollow">PEP 8 -- Style Guide for Python Code -- Descriptive: Naming Styles</a>.</p>
