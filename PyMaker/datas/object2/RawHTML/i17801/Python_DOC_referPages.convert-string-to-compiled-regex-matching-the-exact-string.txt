<p>Is there a simple way to transform an exact string matching to a <code>re.compile</code> object ? For example, I would like to mix exact string and regexes.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can probably use <a href="http://docs.python.org/3/library/re.html#re.escape" rel="nofollow"><code>re.escape</code></a> to escape everything in a string, and then use the escaped string as a pattern.</p>
<pre><code>def transform(s):
    return re.compile(re.escape(s))
</code></pre>
