<div class="post-text" itemprop="text">
<p>You can use <code>Counter</code> like this</p>
<pre><code>from collections import Counter
alist=[1,1,1,2,2,3,4,2,2,3,2,2,1]
print Counter(alist)
</code></pre>
<p>If you want to use your solution, you can improve it like this</p>
<pre><code>def icount(alist):
    adic = {}
    for i in alist:
        adic[i] = adic.get(i, 0) + 1
    return adic
</code></pre>
<p>Even better, you can use <code>defaultdict</code> like this</p>
<pre><code>from collections import defaultdict
adic = defaultdict(int)
for i in alist:
    adic[i] += 1
return adic
</code></pre>
<p>Also, You might want to look at the Time Complexity of various operations on different Python objects <a href="http://wiki.python.org/moin/TimeComplexity" rel="noreferrer">here</a> </p>
