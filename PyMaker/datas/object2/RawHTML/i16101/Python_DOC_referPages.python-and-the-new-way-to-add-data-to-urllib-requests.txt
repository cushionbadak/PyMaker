<p>I'm getting the error,</p>
<p><code>AttributeError: 'Request' object has no attribute 'add_data'</code></p>
<p>from a library that uses urllib.request</p>
<p><a href="https://docs.python.org/3.1/library/urllib.request.html#urllib.request.Request.add_data" rel="noreferrer">In Python 2.7-3.3 urllib.request contained a <code>add_data()</code> method.</a></p>
<p>But In Python 3.4 the documentation states that,</p>
<p><a href="https://docs.python.org/3.4/library/urllib.request.html#urllib.request.Request.add_data" rel="noreferrer">Changed in version 3.4: The request methods add_data, has_data, get_data, get_type, get_host, get_selector, get_origin_req_host and is_unverifiable that were deprecated since 3.3 have been removed.</a></p>
<p>How do I add data to urllib requests in Python3.4?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/urllib.request.html#urllib.request.Request.data" rel="noreferrer">Just assign to the <code>data</code> attribute for <code>urllib.request</code>.</a></p>
<pre><code> request.data = "Some data"
</code></pre>
