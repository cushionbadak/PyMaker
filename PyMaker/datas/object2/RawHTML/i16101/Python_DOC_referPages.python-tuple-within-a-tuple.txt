<div class="post-text" itemprop="text">
<p>Use slicing to get sub-sequence, and convert it to <code>tuple</code> using <a href="https://docs.python.org/3/library/functions.html#func-tuple" rel="nofollow"><code>tuple</code></a>.</p>
<pre><code>&gt;&gt;&gt; lst = ['14:00',18.7,383.5,266405,5480,212500,183750,52380,6804,57150,17431,65567]
&gt;&gt;&gt; tuple(lst[:3]) + (tuple(lst[3:]),)
('14:00', 18.7, 383.5, (266405, 5480, 212500, 183750, 52380, 6804, 57150, 17431, 65567))
</code></pre>
