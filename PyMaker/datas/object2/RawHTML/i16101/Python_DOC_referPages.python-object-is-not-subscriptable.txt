<div class="post-text" itemprop="text">
<p>What happens is that <a href="https://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow"><code>sort()</code></a> <em>sorts</em> the list <em>in-place</em> so it doesn't return anything (in fact, it returns <code>None</code>), therefore the variable <code>sortedNumbers</code> will be <code>None</code>.</p>
<p><strong>What can you do?</strong> You can use <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow"><code>sorted()</code></a>, which will return the sorted list:</p>
<pre><code>sortedNumbers = sorted(numbers)
</code></pre>
<p><strong>Note:</strong> Don't forget to print your results:</p>
<pre><code>print(boxCheck(4,16,8))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Here is what the manual says about the sort method:</p>
<pre><code>&gt; Help on method_descriptor:
&gt; 
&gt; sort(...)
&gt;     L.sort(cmp=None, key=None, reverse=False) -- stable sort *IN PLACE*;
&gt;     cmp(x, y) -&gt; -1, 0, 1
</code></pre>
<p>Your statement</p>
<pre><code>sortedNumbers = numbers.sort()
</code></pre>
<p>results in nothing being saved to sortedNumbers.</p>
