<div class="post-text" itemprop="text">
<p>This error:</p>
<pre><code>AttributeError: 'module' object has no attribute 'stats'
</code></pre>
<p>Means what it says. There is no attribute named <code>stats</code> in the <code>scipy</code> module.</p>
<p>Not because no such thing exists on disk, but because no such thing has been importedâ€”because you never even tried to import it.</p>
<p><code>scipy</code> is a package. As <a href="https://docs.python.org/3/tutorial/modules.html#packages">the Python tutorial</a> explains, importing a package doesn't import all of its submodules.</p>
<p><em>Some</em> packages have a <code>__init.py__</code> that automatically imports some or all packages.* But that would be a bad idea for <code>scipy</code>, because there are a ton of them, so it would take some time to import all of them, and usually you only need one or two in a given project.</p>
<p>So, you just need to do this:</p>
<pre><code>import scipy.stats
</code></pre>
<hr/>
<p><sub>* There are also some cases like <code>os</code> which fake being packages but aren't, so you can use <code>os.path</code> without importing it, or cases like <code>pyobjc</code> that create special placeholder objects for their modules that automatically import the real modules when first accessed.</sub></p>
