  a = 'items?from=%s&amp;to=%s' % (start.strftime('%Y%m%d'), end.strftime('%Y%m%d'))

some_method(20000101, 20140902)
some_method(time.strptime("30 Nov 00"), time.strptime("30 Nov 05"))

<div class="post-text" itemprop="text">
<p>You need a <a href="https://docs.python.org/2/library/datetime.html#datetime.datetime" rel="nofollow"><code>datetime.datetime()</code> object</a>; these have a <a href="https://docs.python.org/2/library/datetime.html#datetime.datetime.strftime" rel="nofollow"><code>datetime.datetime.strftime()</code> methods</a>. You could also use a <a href="https://docs.python.org/2/library/datetime.html#datetime.date" rel="nofollow"><code>datetime.date</code> object</a>; these have the <a href="https://docs.python.org/2/library/datetime.html#datetime.date.strftime" rel="nofollow">same method</a>.</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; datetime.datetime.now()
datetime.datetime(2014, 9, 3, 19, 40, 38, 174720)
&gt;&gt;&gt; datetime.datetime.now().strftime('%Y%m%d')
'20140903'
&gt;&gt;&gt; datetime.date.today().strftime('%Y%m%d')
'20140903'
</code></pre>
<p>The <a href="https://docs.python.org/2/library/datetime.html#datetime.time" rel="nofollow"><code>datetime.time()</code> type</a> has one too, but you cannot use the <code>'%Y%m%d'</code> format on those as they don't have the required date information.</p>
<p>And did you know the <a href="https://docs.python.org/2/search.html?q=strftime&amp;check_keywords=yes&amp;area=default" rel="nofollow">Python documentation has a search feature</a>?  The above 3 types-with-<code>strftime</code>-methods are the first 3 hits.</p>
</div>
<div class="post-text" itemprop="text">
<p>You need to pass an object of class <a href="https://docs.python.org/2/library/time.html#time.struct_time" rel="nofollow">struct_time</a>.</p>
<p>You can create one from a string using time.strptime like this: <code>time.strptime("30 Nov 00", "%d %b %y")</code>, passing a second argument with the time format; or using <a href="https://docs.python.org/2/library/time.html#time.gmtime" rel="nofollow">gmtime</a> or <a href="https://docs.python.org/2/library/time.html#time.localtime" rel="nofollow">localtime</a>.</p>
<p>The way to use strftime then is passing that object as the second parameter:</p>
<pre><code>start = time.strptime("30 Nov 00", "%d %b %y")
print time.strftime('%Y%m%d', start)
</code></pre>
