<pre><code>for i in range(len(short_seq))
</code></pre>
<p>You're currently looping over a single number, which is not allowed. range(int) creates a list of values from [0,input).</p>

<p>This line has errors. You need to have something like</p>
<pre><code>for i in range(len(short_seq)):
</code></pre>
<p>i in - This basically checks for i in a iterable. Since len(short_seq) is not a iterable, it throws an error. </p>

<span class="comment-copy"><code>for i in len(short_seq)</code> is wrong. One can't <i>iterate</i> a number. It is just a number. Use <code>for i in range(num)</code> to iterate <code>[0..n)</code> or just iterate the sequence directly. In this case I suspect that using <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> and/or comprehensions would be useful.</span>
