<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#bytes.join" rel="noreferrer"><code>bytes.join</code></a>:</p>
<pre><code>&gt;&gt;&gt; b = [b'S', b'\x00', b't', b'\x00', b'a']
&gt;&gt;&gt; b''.join(b)  # b'': separator
b'S\x00t\x00a'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>bytes</code> and <code>bytearray</code> objects have the <a href="https://docs.python.org/3/library/stdtypes.html#bytes.join" rel="nofollow"><code>join</code></a> method:</p>
<pre><code>In [37]: b''.join([b'S', b'\x00', b't', b'\x00', b'a', b'\x00', b'n', b'\x00', b'd', b'\x00'])
Out[37]: b'S\x00t\x00a\x00n\x00d\x00'
</code></pre>
