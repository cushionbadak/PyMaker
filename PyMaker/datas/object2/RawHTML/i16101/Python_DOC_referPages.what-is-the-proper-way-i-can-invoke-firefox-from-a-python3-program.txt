<div class="post-text" itemprop="text">
<p>It depends what you want to do with this site. If all you want to do is open the page use the <a href="https://docs.python.org/3/library/webbrowser.html#webbrowser.open" rel="nofollow">webbrowser module</a> to open the url.</p>
<pre><code>import webbrowser

webbrowser.open('https://www.schwab.com/')
</code></pre>
<p>If you need to something more complicated, you can use the <a href="http://selenium-python.readthedocs.org/" rel="nofollow">Selenium</a> module to interact with the page in pretty much anyway you need.</p>
<pre><code>from selenium import webdriver
from selenium.webdriver.common.keys import Keys

driver = webdriver.Firefox()
driver.get("http://www.python.org")
assert "Python" in driver.title
elem = driver.find_element_by_name("q")
elem.send_keys("selenium")
elem.send_keys(Keys.RETURN)
driver.close()
</code></pre>
</div>
