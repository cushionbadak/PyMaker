<div class="post-text" itemprop="text">
<p>There is a library, <a href="https://github.com/libgit2" rel="nofollow"><code>libgit2</code></a>, which enables git to be used as a shared library more helpful to your cause is the python binding's <a href="https://github.com/libgit2/pygit2" rel="nofollow"><code>pygit</code></a>.</p>
<p>To answer your question using <code>pygit</code> to clone a repo:</p>
<pre><code>&gt;&gt;&gt; from pygit2 import clone_repository
&gt;&gt;&gt; repo_url = 'git://github.com/libgit2/pygit2.git'
&gt;&gt;&gt; repo_path = '/path/to/create/repository'
&gt;&gt;&gt; repo = clone_repository(repo_url, repo_path) # Clones a non-bare repository
&gt;&gt;&gt; repo = clone_repository(repo_url, repo_path, bare=True) # Clones a bare repository
</code></pre>
<p>You can view the repository based docs <a href="http://www.pygit2.org/repository.html" rel="nofollow">here</a></p>
</div>
<div class="post-text" itemprop="text">
<p>Just run the <code>git</code> command with <a href="https://docs.python.org/3/library/subprocess.html#subprocess.check_call" rel="nofollow">subprocess.check_call</a>:</p>
<pre><code>import subprocess
subprocess.check_call(["git", "clone", ...])
</code></pre>
</div>
