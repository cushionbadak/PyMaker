<p>Thank you, Jason. :D.</p>
<pre><code>import urllib.request
page = urllib.request.urlopen('http://services.runescape.com/m=hiscore/ranking?table=0&amp;category_type=0&amp;time_filter=0&amp;date=1519066080774&amp;user=zezima')

<div class="post-text" itemprop="text">
<p>Because you're using Python 3.1, you need to use the new <a href="https://docs.python.org/3/howto/pyporting.html" rel="noreferrer">Python 3.1 APIs</a>.</p>
<p>Try:</p>
<pre><code>urllib.request.urlopen('http://www.python.org/')
</code></pre>
<p>Alternately, it looks like you're working from Python 2 examples. Write it in Python 2, then use the 2to3 tool to convert it. On Windows, 2to3.py is in \python31\tools\scripts. Can someone else point out where to find 2to3.py on other platforms?</p>
<p><strong>Edit</strong></p>
<p>These days, I write Python 2 and 3 compatible code by using six.</p>
<pre><code>from six.moves import urllib
urllib.request.urlopen('http://www.python.org')
</code></pre>
<p>Assuming you have six installed, that runs on both Python 2 and Python 3.</p>

<p>A solution with works with Python 2.X and Python 3.X:</p>
<pre><code>try:
    # For Python 3.0 and later
    from urllib.request import urlopen
except ImportError:
    # Fall back to Python 2's urllib2
    from urllib2 import urlopen

url = 'http://hiscore.runescape.com/index_lite.ws?player=zezima'
response = urlopen(url)
data = str(response.read())
</code></pre>
