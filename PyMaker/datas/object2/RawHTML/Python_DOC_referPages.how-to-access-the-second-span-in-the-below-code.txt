<div class="post-text" itemprop="text">
<p>Using the <code>class=right date</code> from the <code>span</code> tag would do it:</p>
<pre><code>from bs4 import BeautifulSoup

article = "https://www.japantimes.co.jp/tag/cybersecurity/page/1/"
page = urllib.request.urlopen(article)
soup = BeautifulSoup(page, 'html.parser')
date = soup.find('span', class_ ="right date")
print(date.text)
</code></pre>
<p><strong>OUTPUT</strong>:</p>
<pre><code>Mar 19, 2019
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>next</code> to get the date, see the code below!</p>
<pre><code>html = '''
 &lt;a class="category-link" href="https://www.japantimes.co.jp/news_category/world/"&gt;
  World
 &lt;/a&gt;
 &lt;span&gt;
  /
 &lt;/span&gt;
 &lt;a class="category-link" href="https://www.japantimes.co.jp/news_category/crime-legal-world/"&gt;
  Crime &amp;amp; Legal
 &lt;/a&gt;
 &lt;span class="right date"&gt;
  Mar 19, 2019
 &lt;/span&gt;
&lt;/h3&gt;'''

soup = BeautifulSoup(html,'html.parser')
date = soup.find('span',attrs={'class':'right date'}).next
print(date.strip())
</code></pre>
<p>Output:</p>
<pre><code>Mar 19, 2019
</code></pre>
