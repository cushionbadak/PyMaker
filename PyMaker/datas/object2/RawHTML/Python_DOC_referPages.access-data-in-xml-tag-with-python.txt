<p>Here it is:</p>
<p>The idea is to find all Stat elements where Type is prefered_foot</p>
<pre><code>import xml.etree.ElementTree as ET

xml = '''&lt;?xml version="1.0" encoding="UTF-8"?&gt;
&lt;SoccerFeed&gt;
  &lt;SoccerDocument&gt;
    &lt;Team&gt;
      &lt;Founded&gt;1919&lt;/Founded&gt;
      &lt;Name&gt;Angers&lt;/Name&gt;
      &lt;Player uID="p40511"&gt;
        &lt;Name&gt;Denis Petric&lt;/Name&gt;
        &lt;Position&gt;Goalkeeper&lt;/Position&gt;
        &lt;Stat Type="first_name"&gt;Denis&lt;/Stat&gt;
        &lt;Stat Type="last_name"&gt;Petric&lt;/Stat&gt;
        &lt;Stat Type="preferred_foot"&gt;Left&lt;/Stat&gt;
      &lt;/Player&gt;
      &lt;Player uID="p119744"&gt;
        &lt;Name&gt;Mathieu Michel&lt;/Name&gt;
        &lt;Position&gt;Goalkeeper&lt;/Position&gt;
        &lt;Stat Type="first_name"&gt;Mathieu&lt;/Stat&gt;
        &lt;Stat Type="preferred_foot"&gt;Right&lt;/Stat&gt;
      &lt;/Player&gt;
    &lt;/Team&gt;
  &lt;/SoccerDocument&gt;
&lt;/SoccerFeed&gt;'''

root = ET.fromstring(xml)
prefered_foots = root.findall(".//Stat[@Type='preferred_foot']")
for foot in prefered_foots:
    print(foot.text)
</code></pre>

Right
