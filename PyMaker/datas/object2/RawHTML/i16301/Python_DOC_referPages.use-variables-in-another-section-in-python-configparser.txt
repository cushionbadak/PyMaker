<p>I know I can use %(values)s to use the exist variable in the config file. But it seems that I can only use the variable in the same section. Is there any way to use the variable in other sections?</p>

<div class="post-text" itemprop="text">
<p>Take a look at <a href="https://docs.python.org/3/library/configparser.html#configparser.ExtendedInterpolation" rel="nofollow">extended interpolation</a>. This allows you to use the <code>${section:option}</code> syntax to refer to values from other sections.</p>
