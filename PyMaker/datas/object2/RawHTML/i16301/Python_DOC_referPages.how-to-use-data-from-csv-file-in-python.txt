<p>you can use csv package</p>
<pre><code>import csv
data_file = open('your_file.csv', 'rb')
reader = csv.reader(data_file)
column = {}
headers = reader.next()
for h in headers:
    column[h] = []
for row in reader:
    for h, v in zip(headers, row):
        column[h].append(v)
score = column['Score']
</code></pre>
<p>now score variable contains your score values</p>
