<p>As per <a href="https://docs.python.org/3/library/doctest.html#how-are-docstring-examples-recognized" rel="nofollow">the docs</a>, the expected output of a doctest ends at an all-whitespace line or a line starting with <code>&gt;&gt;&gt;</code>.
