<div class="post-text" itemprop="text">
<p>this is pretty straight forward, just use line.rstrip('\n') before writing to the file. this also gives you an opportunity to format it how you want. for example </p>
<pre><code>if "species,subl,cmp=" in line:
    data = line[-13:].rstrip('\n')
    of1.write(data+" ")
</code></pre>
<p>would give you a file that looks like</p>
<pre><code>-0.50264E-14 0.74778E-07 0.28390E-14
</code></pre>
