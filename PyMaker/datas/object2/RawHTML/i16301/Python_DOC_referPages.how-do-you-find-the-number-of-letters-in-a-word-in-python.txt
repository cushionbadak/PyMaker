<div class="post-text" itemprop="text">
<p>You're looking for the <strong>len</strong> method, which works on any object with a definable length:</p>
<pre><code>&gt;&gt;&gt;test_string = "Hello"
Hello
&gt;&gt;&gt;len(test_string)
5
</code></pre>
<p>For a string, it will treat the string as a list of characters. This means "Hello" has 5, but "Hello " would have 6, and "Hello World!" would have 12.</p>
<p>docs: <a href="https://docs.python.org/3/library/functions.html#len" rel="nofollow">https://docs.python.org/3/library/functions.html#len</a></p>
</div>

<div class="post-text" itemprop="text">
<p>The input command reads the value as python code rather than a string input.  For example, if you have:</p>
<pre><code>def Python():
    print "it enters this method"

word = input("Enter a word: ")
word()
</code></pre>
<p>It will grab the input from the user (in this case, let's say you enter Python), and convert it to code.  Therefore, word() would be Python() rather than "Python"().  </p>
<p>There's 2 ways around this.  Firstly, you could do the inconvenient method and enter the word in with quotes (so when asked for input, you write "Python" rather than Python).  This will read it in as a string.</p>
<p>The more reasonable way of doing this is using raw_input:</p>
<pre><code>word = raw_input("Enter a word: ")
</code></pre>
<p>From there, you simply use the len() method:</p>
<pre><code>length = len(word)
print length
</code></pre>
</div>
