<div class="post-text" itemprop="text">
<p>Enumerate makes a <code>list</code> of <code>tuple</code> which are <code>(index, element)</code>.   </p>
<pre><code>l = ['thing', 'foo', 'bar', 'doodad']
list(enumerate(l))
</code></pre>
<p>This is the output of <code>enumerate</code></p>
<pre><code>[(0, 'thing'), (1, 'foo'), (2, 'bar'), (3, 'doodad')]
</code></pre>
<p>You can <code>unpack</code> these tuples and do whatever you'd like with each component of the tuple.</p>
<pre><code>for index, element in enumerate(l):
    print 'index', index
    print 'element', element

index 0
element thing

index 1
element foo

index 2
element bar

index 3
element doodad
</code></pre>

<div class="post-text" itemprop="text">
<p><code>enumerate</code> returns both the index of the item in the list, and the item itself. Each item in a Python list (and almost any list in any programming language) has a numeric index in addition to the value it contains. In Python, indexes start at 0. So for your <code>choices</code> list, <code>pizza</code> has the index <code>0</code>, <code>pasta</code> has <code>1</code>, <code>salad</code> has <code>2</code>, and <code>nachos</code> has <code>3</code>. This index exists to allow you direct access to the nth value of a list, without having to loop through it (among other uses).</p>
<p>Here's what your code does when executed:</p>
<pre><code>Your choices are:
1 pizza    # Pizza has index 0, and you added 1
2 pasta    # Index 1
3 salad    # Index 2
4 nachos   # Index 3
</code></pre>
<p><a href="https://docs.python.org/2/library/functions.html#enumerate" rel="nofollow">https://docs.python.org/2/library/functions.html#enumerate</a></p>
</div>
<div class="post-text" itemprop="text">
<p><code>enumerate</code> adds an index or position value to each item in the list starting at 0.</p>
<p>so in your example <code>pizza</code> would have an index of 0, <code>pasta</code> has an index of 1 and so on...</p>
<p>Then your <code>for</code> syntax is slightly different to get both the index and value of the current item.</p>
<p>So first time through the loop you will have <code>index = 0</code> and <code>item = 'pizza'</code> </p>
