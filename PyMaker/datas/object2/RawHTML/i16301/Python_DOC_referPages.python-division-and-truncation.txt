<div class="post-text" itemprop="text">
<p>Your calculator is rounding the result to <code>16</code> decimal places.  You can do the same with <a href="https://docs.python.org/3/library/functions.html#round" rel="nofollow"><code>round</code></a>:</p>
<pre><code>&gt;&gt;&gt; round(float(100) / 13000, 16)
0.0076923076923077
&gt;&gt;&gt;
</code></pre>
</div>
