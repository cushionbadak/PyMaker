<div class="post-text" itemprop="text">
<p>Python a allows you to convert from JSON into a native dict (using json or, in versions &lt; 2.6, simplejson)</p>
<p>You can do this with the help of a library <a href="https://github.com/quandyfactory/dict2xml" rel="nofollow noreferrer">https://github.com/quandyfactory/dict2xml</a></p>
<p>Or you can do this by loading it into <code>json.loads</code>.For the more explaiantion of this method please have a look <a href="https://stackoverflow.com/questions/1019895/serialize-python-dictionary-to-xml">here</a></p>
<p>A sample for converting to xml</p>
<pre><code>  &gt;&gt;&gt; import xmltools

&gt;&gt;&gt; d = {'a':1, 'b':2.2, 'c':'three' }
&gt;&gt;&gt; xx = xmltools.WriteToXMLString(d)
&gt;&gt;&gt; print xx
</code></pre>
<p>The result</p>
<pre><code>&lt;?xml version="1.0" encoding="UTF-8"?&gt;
&lt;top&gt;
  &lt;a&gt;1&lt;/a&gt;
  &lt;b&gt;2.2&lt;/b&gt;
  &lt;c&gt;three&lt;/c&gt;
&lt;/top&gt;
</code></pre>
