<div class="post-text" itemprop="text">
<p>You are mixing two styles, <code>matplotlib</code> and the <code>pandas</code> interface to it. Either do it like <a href="https://stackoverflow.com/users/9840637/anky-91">@anky_91</a> suggested in <a href="https://stackoverflow.com/a/55169695/4042267">their answer</a>, or use <code>matplotlib</code> directly:</p>
<pre><code>import matplotlib.pyplot as plt

plt.scatter(df.index, df.value)
plt.xlabel("index")
plt.ylabel("value")
plt.show()
</code></pre>
