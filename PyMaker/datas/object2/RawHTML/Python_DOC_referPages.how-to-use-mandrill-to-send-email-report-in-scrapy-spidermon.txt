<div class="post-text" itemprop="text">
<p>Currently, Scrapy Spidermon extension only shows an example to send email using Amazon Simple Email Service. Is it possible to use Mandrill instead? And how?</p>
</div>
<div class="post-text" itemprop="text">
<p>You will have to write your own "Mandril Send Email" class.</p>
<p>Use <a href="https://github.com/scrapinghub/spidermon/blob/master/spidermon/contrib/actions/email/ses.py" rel="nofollow noreferrer"><code>SendSESEmail</code></a> as a start point, but instead of handling AWS keys, you use your credentials for Mandrill.</p>
<p>You <code>send_message</code> should look something like this:</p>
<pre><code>def send_message(self, message):
    s = smtplib.SMTP('smtp.mandrillapp.com', 587)
    s.login(MANDRILL_USERNAME, MANDRILL_PASSWORD)
    s.send_message(message)
</code></pre>
<p>(based on this snippet: <a href="https://mandrill.com/#script-python" rel="nofollow noreferrer">https://mandrill.com/#script-python</a>)</p>
<p>Hope it helps you.</p>
</div>
