<p><code>head = csv.reader(open("header.csv"))</code> </p>
<p>I have something like this. here is it possible to provide part of the input in the command line like this.</p>
<pre><code>head = csv.reader(open("sys.argv[1]".csv")) 
</code></pre>
<p>So that I can launch the script as</p>
<pre><code>python test.py header
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3.4/library/functions.html#format" rel="nofollow noreferrer">string formatting</a></p>
<pre><code>head = csv.reader(open('{}.csv'.format(sys.argv[1])))
</code></pre>
<p>or in python 3.6 or later</p>
<pre><code>head = csv.reader(open(f'{sys.argv[1]}.csv'))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Since python 3.6 you can use f-strings.
<a href="https://www.python.org/dev/peps/pep-0498/" rel="nofollow noreferrer">https://www.python.org/dev/peps/pep-0498/</a></p>
<pre><code>head = csv.reader(open(f'{sys.argv[1]}.csv'))
</code></pre>
</div>
