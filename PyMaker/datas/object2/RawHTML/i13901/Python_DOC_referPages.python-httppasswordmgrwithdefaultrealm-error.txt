<div class="post-text" itemprop="text">
<p>If we look at the documentation for <code>urllib.request.HTTPPassowrdMgrWithDefaultRealm</code>, the <code>__init__</code> method looks like this:</p>
<pre><code>__init__(self)
</code></pre>
<p>That <code>self</code> is provided implicitly.  You're trying to pass it a second positional parameter (<code>p</code>), hence the error message.</p>
<p>Take a look at the Basic Authentication example in <a href="https://docs.python.org/3/howto/urllib2.html#id5" rel="nofollow">the documentation</a>, which offers the following example usage:</p>
<pre><code># create a password manager
password_mgr = urllib.request.HTTPPasswordMgrWithDefaultRealm()

# Add the username and password.
# If we knew the realm, we could use it instead of None.
top_level_url = "http://example.com/foo/"
password_mgr.add_password(None, top_level_url, username, password)

handler = urllib.request.HTTPBasicAuthHandler(password_mgr)

# create "opener" (OpenerDirector instance)
opener = urllib.request.build_opener(handler)

# use the opener to fetch a URL
opener.open(a_url)

# Install the opener.
# Now all calls to urllib.request.urlopen use our opener.
urllib.request.install_opener(opener)
</code></pre>
