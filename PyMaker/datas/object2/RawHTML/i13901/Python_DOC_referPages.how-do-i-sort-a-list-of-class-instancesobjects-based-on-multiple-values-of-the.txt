<div class="post-text" itemprop="text">
<p>For the sorting, You can use <code>sorted()</code> function with <code>key</code> argument as -</p>
<pre><code>sortedlist = sorted(alist, key=lambda x: (x.firstname , x.lastname))
</code></pre>
<p>Also, if by json format , you mean something like -</p>
<pre><code>{'color':'&lt;colorname&gt;', 'firstname','&lt;firstname&gt;' ....}
</code></pre>
<p>Such a dictionary for each object. Then you can create a function that returns the <code>__dict__</code> attribute of the object and use <a href="https://docs.python.org/2/library/json.html#json.dumps" rel="nofollow"><code>json.dumps</code></a> and <code>map</code> like -</p>
<pre><code>def cdict(a):
    return a.__dict__

s = json.dumps(list(map(cdict,&lt;your list&gt;)))
print(s)
</code></pre>

<p>I like using <a href="https://docs.python.org/3/library/operator.html#operator.attrgetter" rel="nofollow"><code>operator.attrgetter</code></a> to make things a bit more descriptive/readable</p>
<pre><code>import operator

entries.sort(key = last_then_first)
