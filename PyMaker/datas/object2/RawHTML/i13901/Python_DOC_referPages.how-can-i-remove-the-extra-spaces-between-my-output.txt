<div class="post-text" itemprop="text">
<p>Your lines contain a newline character at the end of them, so in this loop:</p>
<pre><code>for line in lines:
    print (line)
</code></pre>
<p>it will print the newline in the <code>line</code> string. Simply strip <code>line</code> to remove the newline:</p>
<pre><code>for line in lines:
    print (line.strip())
</code></pre>

<p>Try this:</p>
<pre><code>import random

with open('filename.txt') as fin:
    lines = list(fin)
    random.shuffle(lines)
    for line in lines:
        print(line, end='') 
</code></pre>
<p>Setting <code>end=''</code> in <code>print()</code> avoids printing an extra new line.</p>
<p>See the <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow">Python docs on <code>print()</code></a> for more information.</p>

<span class="comment-copy">Your first script uses <code>trim()</code>, which removes the newline at the end of the input lines. You're not doing this in the second program, so it prints that newline, and then <code>print()</code> adds its own newline.</span>
