<div class="post-text" itemprop="text">
<p>The way the python <code>int()</code>function works rounds a number to the smallest value.</p>
<p>Such that <code>4.99 = 4</code> </p>
<p>What can I do if I want 4.99 be equal to 5?</p>
</div>
<div class="post-text" itemprop="text">
<p><code>int()</code> <em>truncates</em> float values, removing the non-integral portion of the number. You are looking for <em>rounding</em> instead.</p>
<p>You can use the <a href="https://docs.python.org/3/library/functions.html#round"><code>round()</code> function</a> for that:</p>
<pre><code>&gt;&gt;&gt; round(4.99)
5
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>There is a <a href="https://docs.python.org/3/library/functions.html#round" rel="nofollow"><code>round()</code></a> builtin:</p>
<pre><code>&gt;&gt;&gt; round(4.99)
5
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/2/library/math.html#math.ceil" rel="nofollow"><code>math.ceil</code></a> and <a href="https://docs.python.org/2/library/math.html#math.floor" rel="nofollow"><code>math.floor</code></a> return floats which are the nearest whole numbers above or below the parameter.</p>
</div>
