<div class="post-text" itemprop="text">
<p>[The question was solved in the comments, hence adding it as an answer now.]</p>
<p>The below code works. (Source : <a href="https://stackoverflow.com/a/7244263/952112" title="Download file from web in Python 3 - Stack Overflow">this answer</a>)</p>
<pre><code>import urllib.request
# Download the file from `url` and save it locally under `file_name`:
urllib.request.urlretrieve(url, file_name)
</code></pre>
<p>Note the import statement.</p>
<p>You need to do <code>import urllib.request</code> instead of <code>import urllib</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>As the error says, <code>urllib</code> does not have a <code>retrieve</code> function.</p>
<p>In Python 2, the module did have a <strong><code>urlretrieve</code></strong> function. In Python 3, that function has been moved to <code>urllib.request.urlretrieve</code>.</p>
<p>You can find all this in the documentation: <a href="https://docs.python.org/3/library/urllib.html" rel="nofollow">https://docs.python.org/3/library/urllib.html</a></p>
