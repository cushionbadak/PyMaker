<div class="post-text" itemprop="text">
<p>You have defined your <code>___init___()</code> function with an extra underscore on both side, so it is not the <code>__init__()</code> (initialization method, but just another instance method) .</p>
<p>Define the <code>__init__()</code> function with just two underscore , not three -</p>
<pre><code># Television Remote

class Television(object):

    def __init__(self, channel = 15, volume = 20):
        self.channel = channel
        self.volume = volume
</code></pre>
