<div class="post-text" itemprop="text">
<p>Is it possible to pass in a custom list of arguments?</p>
<p>More out of curiosity vs. a real life use case, but for example, I might want to convert all parameters to lowercase before having argparse do its job.</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/argparse.html#parsing-arguments" rel="nofollow">Yes</a>. Just pass in the list you want to <code>parse_args</code>.</p>
<p>From the example in the documentation:</p>
<pre><code>&gt;&gt;&gt; parser.parse_args(['--sum', '7', '-1', '42'])
Namespace(accumulate=&lt;built-in function sum&gt;, integers=[7, -1, 42])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>parse_args</code> uses <code>sys.argv[1:]</code> if you don't give it a list argument. You can load <code>sys.argv</code> yourself, and modify it.  If modified in-place it can still be passed implicitily.</p>
<pre><code>import sys
import argparse
parser = ...

# modify sys.argv inplace
sys.argv[1:] = [x.lower() for x in sys.argv[1:]]
args = parser.parse_args()  # pass it implicitily
# or pass it explicitly
args = parser.parse_args(sys.argv[1:])
</code></pre>
</div>
