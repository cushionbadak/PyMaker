<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip</code></a> function and <code>dict.setdefault</code> method to group your values in a dictionary then write the items to file :</p>
<pre><code>&gt;&gt;&gt; groups = ['a','b','c','a','b','b']
&gt;&gt;&gt; values = [1, 2, 3, 4, 5, 6]
&gt;&gt;&gt; 
&gt;&gt;&gt; d={}
&gt;&gt;&gt; 
&gt;&gt;&gt; for i,j in zip(groups,values):
...   d.setdefault(i,[]).append(j)
... 
&gt;&gt;&gt; d
{'a': [1, 4], 'c': [3], 'b': [2, 5, 6]}



with open ('out.txt','w') as fo:
   for k,v in d.items():
        fo.write(k +'  '+ ' '.join(map(str,values)) + '\n')
</code></pre>
