<p>Is there a way to do the following in one line?</p>
<pre><code>[del item for item in new_json if item['Country'] in countries_to_remove]
</code></pre>
<p>The above gives me a <code>SyntaxError</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/reference/simple_stmts.html#the-del-statement" rel="nofollow noreferrer"><code>del</code> is a statement</a> and you cannot use that <a href="https://docs.python.org/3/reference/expressions.html#grammar-token-comprehension" rel="nofollow noreferrer">as an expression in list comprehenstion</a>. That is why you are getting a <code>SyntaxError</code>.</p>
<p>You can use list comprehension to create a new list, without the elements you don't want, like this</p>
<pre><code>[item for item in new_json if item['Country'] not in countries_to_remove]
</code></pre>
<p>This is actually equivalent to,</p>
<pre><code>result = []
for item in new_json:
    if item['Country'] not in countries_to_remove:
        result.append(item)
</code></pre>
<hr/>
<p>This kind of operation is called filtering a list and you can use the builtin <code>filter</code> function, like this</p>
<pre><code>list(filter(lambda x: x['Country'] not in countries_to_remove, new_json))
</code></pre>
<hr/>
<p>As suggested by <a href="https://stackoverflow.com/questions/31552570/how-to-do-a-one-line-dict-delete-operation/31552586#comment51063286_31552586">mgilson in the comments section</a>, if you just want to mutate the original list, then you can use slicing assignment, like this</p>
<pre><code>new_json[:] = [x for x in new_json if x['Country'] not in countries_to_remove]
</code></pre>
</div>

<p><a href="https://docs.python.org/2/reference/simple_stmts.html#the-del-statement" rel="nofollow"><code>del</code></a> is a statement in python, and you cannot have statements inside list comprehension (You can only have expressions there). Why not just create <code>new_json</code> as a new list or dictionary that does not include the items you want to delete. Example =</p>
<pre><code>new_json = [item for item in new_json if item['Country'] not in countries_to_remove]
</code></pre>
