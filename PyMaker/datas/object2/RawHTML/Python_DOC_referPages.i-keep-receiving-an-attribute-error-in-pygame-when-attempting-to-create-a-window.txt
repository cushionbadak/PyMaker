<div class="post-text" itemprop="text">
<p>The event queue is accessed via the function <code>pygame.event.get()</code>.  You are trying to use the pygame <code>display</code>.  Also <code>QUIT</code> is not a function.</p>
<p>Please try:</p>
<pre><code>import pygame

pygame.init()
screen = pygame.display.set_mode((400, 300))
done = False

while not done:
    for event in pygame.event.get():   # &lt;-- HERE
        if event.type == pygame.QUIT:  # &lt;-- AND HERE
            done = True
    pygame.display.flip()
</code></pre>
</div>
