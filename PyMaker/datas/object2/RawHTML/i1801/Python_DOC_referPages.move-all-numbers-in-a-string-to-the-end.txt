<div class="post-text" itemprop="text">
<p>I need help figuring out how to move numbers to the end of a string.</p>
<p>For example, <code>'y0b3'</code> should become <code>'yb03'</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could do something like this:</p>
<pre><code>def group(string):

    digits = []
    non_digits = []
    for c in string:
        if c.isdigit():
            digits.append(c)
        else:
            non_digits.append(c)

    return ''.join(non_digits) + ''.join(digits)

result = group('y0b3')

print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>yb03
</code></pre>
<p>The idea is to group each character into digit and <code>non_digits</code> then put the non digits first and the <code>digits</code> last. For joining both <code>digits</code> and <code>non_digits</code> use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer">join</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Since you tagged generator... you could write the following generator for this task:</p>
<pre><code>def digits_last(string):
    hold = []
    for c in string:
        if c.isdigit():
            hold.append(c)
        else:
            yield c
    yield from hold
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; ''.join(digits_last('y0b3y0b3u0b2'))
&gt;&gt;&gt; 'ybybub030302'
</code></pre>
