<p><a href="https://docs.djangoproject.com/en/2.1/ref/request-response/#django.http.HttpRequest.body" rel="nofollow noreferrer">request.body</a> returns byte string. So you can use <a href="https://docs.python.org/3/library/json.html#json.loads" rel="nofollow noreferrer">json.loads</a> function, for example:</p>
<pre><code>import json

def initiate_sms(request):
    user_data = request.body
    user_data = json.loads(user_data)
    print(user_data)
</code></pre>
