<p>My aim is to check a text file for a size (&gt; 0 bytes) and if True, copy the file to another directory. What i would like to add is a date to the file by generating a date variable. The part i am unable to figure out is how to add the date variable when I copy the file. The below code works fine except adding the variable called <code>yesterday</code> to the filename.
Any ideas on how do perform this last part of the code ?</p>
<pre><code>import os
import shutil
import datetime

# a variable called yesterday will be generated (yesterdays date)
today = datetime.date.today()
one_day = datetime.timedelta(days=1)
yesterday = today - one_day

# Check if file is True

file_path = r"C:\temp\Missing_File.txt"
if os.stat(file_path).st_size &gt; 0:
    shutil.copy("C:/temp/Missing_File.txt", "C:/temp2/Missing_File.txt")
</code></pre>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>str.format()</code></a>:</p>
<pre><code>shutil.copy("C:/temp/Missing_File.txt", "C:/temp2/Missing_File_{}.txt".format(yesterday))
</code></pre>
<p>If you have Python 3.6+, another option is to use an <a href="https://docs.python.org/3/reference/lexical_analysis.html#formatted-string-literals" rel="nofollow noreferrer">f-string</a>:</p>
<pre><code>shutil.copy("C:/temp/Missing_File.txt", f"C:/temp2/Missing_File_{yesterday}.txt")
</code></pre>
<p>You can also customize <a href="https://docs.python.org/3.7/library/datetime.html?highlight=strftime#datetime.date.__format__" rel="nofollow noreferrer">stringification</a> of a <code>date</code> object by specifying a format string in the placeholder, e.g.:</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; yesterday = datetime.date.today() - datetime.timedelta(days=1)
&gt;&gt;&gt; "C:/temp2/Missing_File_{:%A}.txt".format(yesterday)
'C:/temp2/Missing_File_Sunday.txt'
</code></pre>
