<p>Trying to tokenize sentences in python using nltk except I want to tokenize the \n and \t characters as well.</p>
<p>Example:</p>
<p>In: "This is a\n test"</p>
<p>Out: ['This', 'is', 'a', '\n', 'test']</p>
<p>Is there a way that's directly supported to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You could use a <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">regex</a>:</p>
<pre><code>import re

text = "This is a\n test with\talso"
pattern = re.compile('[^\t\n]+|[\t\n]+')

output = [val for values in map(pattern.findall, text.split(' ')) for val in values]
print(output)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>['This', 'is', 'a', '\n', 'test', 'with', '\t', 'also']
</code></pre>
<p>The idea is to first split on a single whitespace then, apply findall for each element in the list resulting from the split. The pattern <code>[^\t\n]+|[\t\n]+</code> matches everything that is not a tab or a newline and multiple times and also everything that is a new line or tab multiple times. If you want to consider each tab and newline as a single token change the pattern to:</p>
<pre><code>import re

text = "This is a\n test\n\nwith\t\talso"
pattern = re.compile('[^\t\n]+|[\t\n]')
output = [val for values in map(pattern.findall, text.split(' ')) for val in values]
print(output)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>['This', 'is', 'a', '\n', 'test', '\n', '\n', 'with', '\t', '\t', 'also']
</code></pre>
