<div class="post-text" itemprop="text">
<p>What you've get is a DNS query error indicating that domain <code>myserver</code> does not exist.</p>
<p>You have to replace the argument <code>myserver</code> in <code>server = smtplib.SMTP('myserver')</code> with the actual address of SMTP server, such as <code>smtp.mail.yahoo.com</code>.</p>
</div>

<p>This is how I do it.</p>
<pre><code>import smtplib
server=smtplib.SMTP('smtp.gmail.com',587)
server.starttls()
server.login('your_email@gmail.com','your_password')
server.sendmail('your_email@gmail.com','your_email@gmail.com','test email')
</code></pre>
