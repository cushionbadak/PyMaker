<p>I have this below code written to remove numbers greater than 5 from the list.
But it does not work though the code looks fine to me. I'm new to python and trying to get my basics right. I decided to use remove since i'm checking based on the value</p>
<pre><code>a = [1,5,4,6,3,8,9,5,9] 
for i in a:
    print (i)
    if i&lt;=5:
        continue
    a.remove(i)
print (a)
</code></pre>
<p>Result - <code>[1, 5, 4, 3, 5, 9]</code></p>

<div class="post-text" itemprop="text">
<p>it will not work because you are removing item from list while iterating the same one....so when you remove an element, next iteration you will jump one one them.</p>
<p>Solution:
1) use 2 array;
2) use a while loop and when you remove an element you decrease the len of the array and you don't increment the iteration</p>
<pre><code>a = [1,5,4,6,3,8,9,5,9]
i=0
l=len(a)
while i&lt;l:
    if a[i]&gt;5:
        a.pop(i)
        l-=1
    else:
        i+=1
print (a)
</code></pre>
<p>Result:</p>
<pre><code>[1, 5, 4, 3, 5]
</code></pre>
