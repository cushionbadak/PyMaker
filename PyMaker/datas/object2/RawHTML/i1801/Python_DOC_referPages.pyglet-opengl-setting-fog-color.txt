<div class="post-text" itemprop="text">
<p>I'd like to know how to set fog color using pyglet.gl.</p>
<p>I couldn't find information about fog color in the <a href="https://pyglet.readthedocs.io/en/pyglet-1.3-maintenance/" rel="nofollow noreferrer">documentation</a>, but I found <a href="https://www.khronos.org/registry/OpenGL-Refpages/gl2.1/xhtml/glFog.xml" rel="nofollow noreferrer">this</a> useful. However, the code I wrote </p>
<pre><code>pyglet.gl.glFogfv(gl.GL_FOG_COLOR, (0.5, 0.7, 1.0, 1.0))
</code></pre>
<p>produces the error: <code>ctypes.ArgumentError: argument 2: &lt;class 'TypeError'&gt;: expected LP_c_float instance instead of tuple</code></p>
<p>I'm not sure how to give it a LP_c_float from my color tuple, as it is not found in pyglet.gl or in the above documentation.</p>
</div>
<div class="post-text" itemprop="text">
<p>The 2nd paramter of <code>pyglet.gl.glFogfv</code> is an array of 4 floats, sp you have to create an ctypes array by <code>(GLfloat * 4)</code>, see <a href="https://programtalk.com/python-examples-amp/pyglet.gl.glFogfv/" rel="nofollow noreferrer">pyglet.gl.glFogfv</a>: </p>
<pre><code>pyglet.gl.glFogfv(gl.GL_FOG_COLOR, (GLfloat * 4)(0.5, 0.7, 1.0, 1.0))
</code></pre>
<p>See <a href="https://docs.python.org/3/library/ctypes.html" rel="nofollow noreferrer">ctypes â€” A foreign function library for Python - Arrays</a>:</p>
<blockquote>
<p>Arrays are sequences, containing a fixed number of instances of the same type.</p>
<p>The recommended way to create array types is by multiplying a data type with a positive integer:</p>
<pre><code>TenPointsArrayType = POINT * 10
</code></pre>
</blockquote>
</div>
<span class="comment-copy"><code>pyglet.gl.glFogfv(gl.GL_FOG_COLOR, (GLfloat * 4)(0.5, 0.7, 1.0, 1.0))</code></span>
