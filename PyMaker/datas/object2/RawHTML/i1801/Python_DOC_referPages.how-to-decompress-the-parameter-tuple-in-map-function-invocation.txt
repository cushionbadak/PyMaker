<p>suppose we have a function sum takes two num and return its sum:</p>
<pre><code>def sum(a, b):
    return a + b
</code></pre>
<p>and we want to use map to calculate the sum on some iterable container:</p>
<pre><code>map(sum, [(1, 2), (3, 4)])
</code></pre>
<p>it will raise an exception:</p>
<pre><code>TypeError: sum() takes exactly 2 arguments (1 given)
</code></pre>
<p>What happens here is that sum get invoked on parameter (1, 2) and (3, 4), that gives us an error. We can do that:</p>
<p>sum(*(1, 2)) would correctly return. here is my problem:
how could we use the map to pass this tuple into sum gracefully?
here is a solution, but not quite as gracefully as what I want:</p>
<pre><code>map(lambda x: sum(*x), [(1, 2), (3, 4)])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>That is exactly what <a href="https://docs.python.org/3/library/itertools.html#itertools.starmap" rel="noreferrer"><code>starmap</code></a> is for:</p>
<pre><code>from itertools import starmap

starmap(sum, [(1, 2), (3, 4)])
</code></pre>
</div>
