<p>You forgot to add <code>()</code>: <code>mystr[x].isalpha</code>. To call function you should do <code>mystr[x].isalpha()</code>. <code>mystr[x].isalpha</code> is always evaluated to <code>True</code>, that's why your code doesn't print anything</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer">enumerate()</a> wich returns the pos and the character you iterate:</p>
<pre><code>mystr = "saddas das"
for pos,c in enumerate(mystr):
    # change your conditions to make it easier to understand, isalpha() helps
    if c.isdigit() or c.isalpha() or c in "@_.":
        continue # do nothing
    else:
        print (pos)
</code></pre>
<p>Output:</p>
<pre><code>6 
</code></pre>

<div class="post-text" itemprop="text">
<p>Using a regex:</p>
<pre><code>import re

pattern = re.compile('[^\d\w\.@]')
s = "saddas das"

for match in pattern.finditer(s):
    print(match.start())
</code></pre>
<p><strong>Output</strong></p>
<pre><code>6
</code></pre>
<p>The pattern <code>'[^\d\w\.@]'</code> matches everything that is not a digit, not a letter, nor <code>_</code>, <code>.</code> or <code>@</code>.</p>
