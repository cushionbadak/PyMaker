<p>What about collecting your items in a <a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow noreferrer"><code>collections.defaultdict()</code></a>, and then applying <a href="https://docs.python.org/3/library/statistics.html#statistics.median" rel="nofollow noreferrer"><code>statistics.median()</code></a> on each list of medians:</p>
<pre><code>from csv import reader
from statistics import median
from collections import defaultdict

data = defaultdict(list)
with open('data.csv') as file:
    csv_reader = reader(file)
    headers = list(map(str.strip, next(csv_reader)))

    for line in csv_reader:
        for col, value in enumerate(map(str.strip, line)):
            data[headers[col]].append(value)

medians = {k: median(map(float, v)) for k, v in data.items() if k.startswith('A')}

print(medians)
</code></pre>
<p>Which outputs a dictionary of medians:</p>
<pre><code>{'A1': 78.0, 'A2': 58.0, 'A3': 68.0, 'A4': 66.0}
</code></pre>
