<div class="post-text" itemprop="text">
<p>The problem is that the <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer"><code>csv.reader</code></a> returns text, and therefore the plot does not order the values.
You should convert the values with <code>int</code> or <code>float</code>:</p>
<pre><code>for row in reader:
        data.append(np.array([int(x) for x in row[5:]]))
</code></pre>
