<p>I like having these utility functions around:</p>
<pre><code>def string_to_file(string, path):
    with open(path, 'w') as f:
        f.write(string)


def file_to_string(path):
    with open(path, 'r') as f:
        return f.read()
</code></pre>
<p>With those, the answer becomes:</p>
<pre><code>string_to_file(file_to_string(path).replace("'", ""), path)
</code></pre>
