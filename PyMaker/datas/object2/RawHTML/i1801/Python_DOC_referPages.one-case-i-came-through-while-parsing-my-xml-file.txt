<div class="post-text" itemprop="text">
<p>You have to take the tail of that element.</p>
<p>Please check the code from ipython console below, </p>
<pre><code>In [1]: import xml.etree.ElementTree as ET

In [2]: cat myxml.xml
&lt;Root&gt;
  &lt;S_CellBody Id = "16393"&gt;
   &lt;?OldID 16393?&gt;
   &lt;ph Id = "19393"&gt;SELx&lt;/ph&gt;
    (x=0)
  &lt;/S_CellBody&gt;
&lt;/Root&gt;

In [3]: tree = ET.parse('myxml.xml')

In [4]: root = tree.getroot()

In [5]: elem = root.find('S_CellBody')

In [6]: if elem:
   ...:     print(elem[0].tail)
   ...:     
/usr/local/bin/ipython:1: FutureWarning: The behavior of this method will change in future versions.  Use specific 'len(elem)' or 'elem is not None' test instead.
  #!/usr/bin/python

    (x=0)


In [7]: if elem is not None:
   ...:     print(elem[0].tail)
   ...:     

    (x=0)
</code></pre>
