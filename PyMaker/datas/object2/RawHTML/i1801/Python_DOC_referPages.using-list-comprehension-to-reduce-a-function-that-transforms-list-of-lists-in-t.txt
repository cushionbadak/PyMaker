<p>If you want it in a functional form, this is one way. You do not need an index here because <code>lst in lista</code> iterates over the elements directly.</p>
<pre><code>lista = [[1],[2],[3]]
def lol(lista):
    novotuplo = tuple(tuple(lst) for lst in lista)
    return novotuplo

print (lol(lista))
# ((1,), (2,), (3,))
</code></pre>
