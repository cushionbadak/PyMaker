<pre><code>a = "Hello, World!"
print(a.len())
</code></pre>

<div class="post-text" itemprop="text">
<p>You are doing it wrong. <code>a.len()</code> doesn't exist. You should do</p>
<pre><code>a = "Hello, World!"
print(len(a))
</code></pre>
<p>Read more about it <a href="https://docs.python.org/3/tutorial/introduction.html#strings" rel="nofollow noreferrer">here</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>dot notations such us <code>.upper(), .lower()</code> are methods and <code>len(), abs()</code> etc are functions
Use <code>len(a)</code> instead</p>
</div>
