<p>This is the code I have and I'm getting errors:</p>
<pre><code>def digitize(n):
  return n.split()
</code></pre>

<div class="post-text" itemprop="text">
<p>You could do something like this:</p>
<pre><code>def digitize(n):
    return [int(d) for d in str(n)]


print(digitize(123))
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[1, 2, 3]
</code></pre>
<p>First convert it to a string, using <a href="https://docs.python.org/3/library/functions.html#func-str" rel="nofollow noreferrer">str</a> then iterate over the characters of the string (the digits) and transform each back to an integer. As an alternative you could also use <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer">map</a>, for example:</p>
<pre><code>return [e for e in map(int, str(n))]
</code></pre>
<p>Or as pointed by @Ev.Kounis you could simply do:</p>
<pre><code>return list(map(int, str(n)))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>In python split is a method of strings, not integers.</p>
<p>Convert the integer to string first and then transform it to <a href="https://stackoverflow.com/questions/4978787/how-to-split-a-string-into-array-of-characters">list</a> of characters:</p>
<pre><code>list(str(123)) # ['1', '2', '3']
</code></pre>
<p>You can then map each element to int again:</p>
<pre><code>list(map(int,list(str(123))))  # [1, 2, 3]
</code></pre>
