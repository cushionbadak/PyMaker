<div class="post-text" itemprop="text">
<p>Put it into a class like:</p>
<pre><code>class fruit:
    a = 0
    def apple(self):
            self.a = 2
    def orange(self):
            if(self.a == 2):
                print('your fruit is apple')
a = fruit()
</code></pre>
