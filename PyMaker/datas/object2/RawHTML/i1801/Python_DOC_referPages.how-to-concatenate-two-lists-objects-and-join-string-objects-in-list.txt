<p>How do I concatenate two lists in Python?
example:
input:</p>
<pre><code>    x=['saeed']
    y=['gharif']
</code></pre>
<p>output expected:</p>
<pre><code>    z=['saeedgharif']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have two lists, each with a single element, so you can make a new list by concatenating the <code>[0]</code> element of each list</p>
<pre><code>&gt;&gt;&gt; z = [x[0] + y[0]]
&gt;&gt;&gt; z
['saeedgharif']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <code>zip</code> and a list comprehension.</p>
<p><strong>Ex:</strong></p>
<pre><code>x=['saeed']
y=['gharif']

print( [i+v for i,v in zip(x, y)] )
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>['saeedgharif']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="http://www.tutorialspoint.com/python/string_join.htm" rel="nofollow noreferrer">Join</a> is way more readable, and just use <code>+</code> to combine both lists.</p>
<pre><code>x=['saeed']
y=['gharif']
z=''.join(x+y)
</code></pre>
