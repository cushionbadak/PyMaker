<pre><code>import configparser

cfg = configparser.ConfigParser()

</code></pre>

<div class="post-text" itemprop="text">
<p><code>open('Config.ini', 'w')</code> deletes the file. See <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer">Python docs on <code>open</code></a>, which says:</p>
<blockquote>
<p>open for writing, truncating the file first</p>
</blockquote>
<p>Instead of that, open the file in reading mode: <code>open('Config.ini', 'r')</code>, or better yet, for reading as a textual file: <code>open('Config.ini', 'rt')</code>.</p>
