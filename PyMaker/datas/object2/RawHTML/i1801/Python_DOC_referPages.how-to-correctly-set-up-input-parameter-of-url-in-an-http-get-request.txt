<p>Write a function named "variable_get" that takes a string as a parameter representing part of a path of a url and returns the response of an HTTPS GET request to the url "<a href="http://python.org/(input)" rel="nofollow noreferrer">http://python.org/(input)</a>" as a string where (input) is the input parameter of this function.</p>
<pre><code>import urllib2
def variable_get(input1):
    content = urllib2.urlopen('http://python.org/input1').read()
    return content
</code></pre>
<p>My question is how to correctly set up input parameter of url in an HTTP GET request?</p>
</div>
<div class="post-text" itemprop="text">
<p>In Python &gt; 3.6, you can use <a href="https://www.python.org/dev/peps/pep-0498/#no-use-of-globals-or-locals" rel="nofollow noreferrer">f-strings</a>:</p>
<pre><code>from urllib import request
def variable_get(input1):
    url = f'http://python.org/{input1}'
    print(url)
    content = request.urlopen(url).read()
    return content

variable_get('somePath')
</code></pre>
<p>should print <code>'http://python.org/somePath'</code></p>
