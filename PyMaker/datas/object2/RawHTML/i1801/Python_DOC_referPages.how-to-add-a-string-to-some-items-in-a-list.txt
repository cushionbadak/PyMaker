<div class="post-text" itemprop="text">
<p>There is a way: <a href="https://docs.python.org/3/library/stdtypes.html#str.endswith" rel="nofollow noreferrer"><code>str.endswith</code></a>.</p>
<pre><code>resultat = span.text.strip()
if not resultat.endswith(' (S)'):
    resultat += ' (S)'
</code></pre>
<p>(I removed the surrounding parentheses in the first line; they are not necessary.)</p>
<p>To put a first name at the end with a comma, split the text on spaces and join them again with the first item at the end:</p>
<pre><code>if not ',' in resultat:
    temp_list = resultat.split()
    resultat = ' '.join(temp_list[1:])+', '+temp_list[0]
</code></pre>

<pre class="lang-none prettyprint-override"><code>Löfven, Stefan (S)
Ygeman, Anders (S)
Karlsson, Annelie (S)
Rådström Baastad, Lena (S)
Ahlberg, Ann-Christin (S)
Andersson, Johan (S)
Axelsson, Marie (S)
</code></pre>
