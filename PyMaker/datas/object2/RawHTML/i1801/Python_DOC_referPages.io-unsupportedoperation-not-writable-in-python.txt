<p>Add writing and reading mode to <code>open()</code> function:</p>
<pre><code>f = open("password.txt", "r+")
</code></pre>
<blockquote>
<p>file_object  = open(“filename”, “mode”)</p>
</blockquote>
<p>The modes are: </p>
<ul>
<li><strong>r</strong> – Read mode which is used when the file is only being read </li>
<li><strong>w</strong> – Write mode which is used to edit and write new information to the file (any existing files with the same name will be erased when this mode is activated) </li>
<li><strong>a</strong> – Appending mode, which is used to add new data to the end of the file; that is new information is automatically amended to the end </li>
<li><strong>r+</strong> – Special read and write mode, which is used to handle both actions when working with a file </li>
</ul>
<p>Reference:
<a href="https://docs.python.org/3/tutorial/inputoutput.html#reading-and-writing-files" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/inputoutput.html#reading-and-writing-files</a></p>
</div>
<span class="comment-copy">You are opening the file in reading mode (<code>'r'</code>), hence it is not writable.</span>
