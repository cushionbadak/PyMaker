<div class="post-text" itemprop="text">
<p>Your frame is not getting placed/rendered in the ui. You need to call the <a href="https://docs.python.org/3/library/tkinter.html#the-packer" rel="nofollow noreferrer"><code>pack</code></a> method on it:</p>
<pre><code>from tkinter import *

root = Tk()

def leftClick(event):
    print("left")

def middleClick(event):
    print("middle")

def rightClick(event):
    print("right")

frame = Frame(root, width=300, height=250)
frame.bind("&lt;Button-1&gt;", leftClick)
frame.bind("&lt;Button-2&gt;", middleClick)
frame.bind("&lt;Button-3&gt;", rightClick)
frame.pack() # &lt;---------

root.mainloop()
</code></pre>
