<div class="post-text" itemprop="text">
<p>You can convert values of column to <a href="http://pandas.pydata.org/pandas-docs/stable/categorical.html" rel="nofollow noreferrer">ordered categoricals</a>, so possible use <a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.sort_values.html" rel="nofollow noreferrer"><code>sort_values</code></a>:</p>
<pre><code>cats = ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday', 'Sunday']
df['Date'] = pd.Categorical(df['Date'], categories=cats, ordered=True)
df = df.sort_values('Date')
print (df)
        Date  Transactions
1     Monday    114.000000
5    Tuesday    107.826087
6  Wednesday    105.608696
4   Thursday    118.217391
0     Friday    140.652174
2   Saturday    208.826087
3     Sunday    140.565217
</code></pre>
<p>Or create index by column <code>Date</code> by <a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.set_index.html" rel="nofollow noreferrer"><code>set_index</code></a>, then <a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.reindex.html" rel="nofollow noreferrer"><code>reindex</code></a> and last <a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.reset_index.html" rel="nofollow noreferrer"><code>reset_index</code></a>:</p>
<p><em>Notice:</em><br/>
<em>Solution working only if unique values of column</em></p>
<pre><code>df = df.set_index('Date').reindex(cats).reset_index()
print (df)

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/calendar.html#calendar.day_name" rel="nofollow noreferrer"><code>calendar.day_name</code></a> with categorical data:</p>
<pre><code>from calendar import day_name

df['Date'] = pd.Categorical(df['Date'], categories=day_name, ordered=True)

df = df.sort_values('Date')

print(df)

        Date  Transactions
1     Monday    114.000000
5    Tuesday    107.826087
6  Wednesday    105.608696
4   Thursday    118.217391
0     Friday    140.652174
2   Saturday    208.826087
3     Sunday    140.565217
</code></pre>
<p>If in your culture Monday is <em>not</em> considered the first day of the week, you can rotate your days of the week by <em>n</em> days. For example:</p>
<pre><code>from collections import deque

days = deque(day_name)
days.rotate(1)

print(days)

deque(['Sunday', 'Monday', 'Tuesday', 'Wednesday',
       'Thursday', 'Friday', 'Saturday'])
</code></pre>
<p>Then feed <code>categories=days</code> as an argument to <code>pd.Categorical</code>.</p>
