<p>I have a list like below, I want to find simple permutation with little bit modification,</p>
<p>For Example </p>
<pre><code>l=['a', 'b']
</code></pre>
<p>Output:</p>
<pre><code>[('a', 'a'), ('a', 'b'), ('b', 'b')]
</code></pre>
<p>I followed,</p>
<p>Try-1 </p>
<pre><code>list(itertools.product(L, repeat=2))
</code></pre>
<p>returns,</p>
<pre><code>[('a', 'a'), ('a', 'b'), ('b', 'a'), ('b', 'b')]
</code></pre>
<p>Try -2 </p>
<pre><code>print list(itertools.permutations(['a', 'b']))
</code></pre>
<p>returns,</p>
<pre><code>[('a', 'b'), ('b', 'a')]
</code></pre>
<p>Try-3</p>
<p>i can do like below,</p>
<pre><code>temp= [tuple(sorted((i,j))) for i in ['a', 'b'] for j in ['a', 'b']]
print list(set(temp))
</code></pre>
<p>But it seems inefficient way to solve this.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/itertools.html#itertools.combinations_with_replacement" rel="noreferrer">combinations_with_replacement</a>:</p>
<pre><code>from itertools import combinations_with_replacement

l=['a', 'b']
for c in combinations_with_replacement(l, 2):
    print(c)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>('a', 'a')
('a', 'b')
('b', 'b')
</code></pre>
