<pre><code>import numpy as np

<div class="post-text" itemprop="text">
<p>Pass to tuple:</p>
<pre><code>print("My First number is %3.4f, second %1.2f and third %2.9f" % tuple(array))
</code></pre>
<p>Or use the <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer">new format</a></p>
<pre><code>array = np.array([123.456789, 1.23456, 12.3456])
print("My First number is {:3.4f}, second {:1.2f} and third {:2.9f}".format(*array))
&gt;&gt; My First number is 123.4568, second 1.23 and third 12.345600000
</code></pre>
</div>

</code></pre>
