<p>I want to split a string in two parts like this:</p>
<pre><code>&gt;&gt;&gt; label = ('A1')
['A', '1']
</code></pre>
<p>Is there any method to do this in python?</p>
<p>I tried:</p>
<pre><code>&gt;&gt;&gt; label = label.split(',')
['A1']
</code></pre>
<p>As you can see the comma is not printed.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can simply use <a href="https://docs.python.org/3/library/functions.html#func-list" rel="nofollow"><code>list</code></a>:</p>
<pre><code>&gt;&gt;&gt; label = 'A1'
&gt;&gt;&gt; list(label)
['A', '1']
&gt;&gt;&gt;
</code></pre>
<p><code>list</code> will iterate over the string and collect its characters into a new list.</p>
<p>Also, you cannot use <code>str.split</code> here because the method was designed to split on characters/substrings and remove them from the resulting list.  For example, <code>'a b c'.split()</code> would split on whitespace and remove those characters from the returned list, which is <code>['a', 'b', 'c']</code>.  You however want to break the string up into individual characters while still keeping all of them.</p>
</div>
