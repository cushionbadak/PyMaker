<p>I assume, that you have already extracted the string from excel. You may use <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strptime" rel="nofollow">datetime.strptime</a> with subsequent <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strftime" rel="nofollow">strftime</a>. format descriptions are <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow">here</a></p>

<div class="post-text" itemprop="text">
<p>Use datetime module</p>
<pre><code>import datetime
date_str = "11/12/2014" # Assume date is 2014 Dec 11.
datetime.datetime.strptime(date_str, "%d/%m/%Y").strftime("%m/%d/%Y") # result: '12/11/2014'
</code></pre>
