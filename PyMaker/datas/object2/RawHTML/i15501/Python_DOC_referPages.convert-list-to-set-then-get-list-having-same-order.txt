<div class="post-text" itemprop="text">
<p>I have a List with some duplicate values,</p>
<pre><code>myList = [2,7,6,6,9,2,4,2]
</code></pre>
<p>I want to get unique elements</p>
<pre><code>myList  = set(myList)
myListisBack = list(myList)
</code></pre>
<p>But the problem is that I get </p>
<pre><code>[9, 2, 4, 6, 7]
</code></pre>
<p>How can I get the List of that same order as it was like</p>
<pre><code>[2,7,6,9,4]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>set</code> is an unordered collection. Use <a href="https://docs.python.org/3/library/collections.html#collections.OrderedDict" rel="noreferrer"><code>collections.OrderedDict.fromkeys</code></a> to keep the order:</p>
<pre><code>&gt;&gt;&gt; myList = [2,7,6,6,9,2,4,2]
&gt;&gt;&gt; import collections
&gt;&gt;&gt; list(collections.OrderedDict.fromkeys(myList))
[2, 7, 6, 9, 4]
</code></pre>
</div>
<span class="comment-copy">Cheat and use an OrderedDict... <code>myListisBack = list(collections.OrderedDict.fromkeys(myList))</code> - this has been asked many times before - just trying to find a comprehensive answer</span>
