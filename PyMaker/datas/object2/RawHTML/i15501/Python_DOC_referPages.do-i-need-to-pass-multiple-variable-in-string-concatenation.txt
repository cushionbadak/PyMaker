</code></pre>

<div class="post-text" itemprop="text">
<p>If you use the newer <a href="https://docs.python.org/3/library/stdtypes.html#str.format"><code>str.format</code></a> method, you can do:</p>
<pre><code>String2 = String + "{0}, {0}".format(String1)
</code></pre>
<p>In fact, you should always prefer <code>str.format</code> over <code>%</code> formatting in modern Python.  The latter approach is pseudo-deprecated and will most likely be removed from a future version of the language.</p>
</div>
