<p>I know <a href="http://php.net/manual/en/reserved.variables.globals.php" rel="nofollow"><code>$GLOBALS</code></a> in PHP is a rough equivalent of Python's <a href="https://docs.python.org/3/library/functions.html#globals" rel="nofollow"><code>globals()</code></a>, but is there an equivalent of <a href="https://docs.python.org/3/library/functions.html#locals" rel="nofollow"><code>locals()</code></a>?</p>
<p>My Python:</p>
<pre><code>&gt;&gt;&gt; g = 'global'
&gt;&gt;&gt; def test():
...     l = 'local'
...     print repr(globals());
...     print repr(locals());
... 
&gt;&gt;&gt; 
&gt;&gt;&gt; test()
{'g': 'global', [...other stuff in the global scope...]}
{'l': 'local'}
</code></pre>
<p>My PHP port:</p>
<pre><code>&lt;?php
$g = 'global';
function test(){ 
    $l = 'local';
    print_r($GLOBALS);
    //...please fill in the dots...:-)
}
test();
?&gt;
Array
(
    [g] =&gt; global
    [...other stuff in the global scope...]
)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="http://php.net/manual/en/function.get-defined-vars.php" rel="nofollow"><code>get_defined_vars</code></a> is what you're looking for.</p>
<pre><code>function test(){
    $a = 'local';
    $b = 'another';
    print_r(get_defined_vars());
}

test();

#Array
#(
#    [a] =&gt; local
#    [b] =&gt; another
#)
</code></pre>
</div>
