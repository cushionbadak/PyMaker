<div class="post-text" itemprop="text">
<p>Putting strings together <code>"like " "this"</code> will join them into one, <code>"like this"</code>.  It can be used across lines.</p>
<pre><code>return ("Cargo(location=%s, cargocapacity=%s, cargo=%s, "
        "name=%s, mast_count=%s)") % (
        self.location,
        self.cargocapacity,
        self.cargo,
        self.name,
        self.mast_count)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can split up a string between parentheses and have Python automatically join them at compilation time:</p>
<pre><code>return (
    "Cargo(location=%s, cargocapacity=%s, cargo=%s, "
    "name=%s, mast_count=%s)") % (
            self.location,
            self.cargocapacity,
            self.cargo,
            self.name,
            self.mast_count)
</code></pre>
<p>The parentheses are not really required if you use <code>\</code> before the end of the line, but with parentheses this is just a little cleaner.</p>
