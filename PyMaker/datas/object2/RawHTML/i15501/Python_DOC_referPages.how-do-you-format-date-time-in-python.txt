<div class="post-text" itemprop="text">
<p>I have a variable that has value like this:</p>
<pre><code>val='14/12/15 0000'
</code></pre>
<p>it is in two digit year/month/day hourminute format.</p>
<p>I need to convert this to epoch time.</p>
<p>I tried this</p>
<pre><code>import datetime

datetime.datetime.strptime(val, "%y/%m/%d %HH%MM").strftime('%s')
</code></pre>
<p>I get this error:</p>
<pre><code>ValueError: time data '14/12/15 0000' does not match format '%y/%m/%d %HH%MM'
</code></pre>
<p>what am I doing wrong here?</p>
</div>
<div class="post-text" itemprop="text">
<p>Hours (24 hr) are <code>%H</code>, not <code>%HH</code>, and minutes are <code>%M</code>, not <code>%MM</code>.</p>
<pre><code>datetime.datetime.strptime(val, "%y/%m/%d %H%M").strftime('%s')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://github.com/ralphavalon/easy_date" rel="nofollow">easy_date</a> to make it easy:</p>
<pre><code>import date_converter
my_datetime = date_converter.string_to_string('14/12/15 0000', '%y/%m/%d %H%M', '%s')
</code></pre>
<p>Or even convert <strong>directly</strong> to a timestamp:</p>
<pre><code>import date_converter
timestamp = date_converter.string_to_timestamp('14/12/15 0000', '%y/%m/%d %H%M')
</code></pre>
