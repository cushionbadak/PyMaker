<p><strong>Edit:</strong></p>
<p>Actually, it would probably be better to do this instead:</p>
<pre><code>income = int(input("May I ask {}{} monthly income:  "
                   .format(name, "'" if name.endswith('s') else "'s")))
</code></pre>
<p>By using a <a href="https://docs.python.org/3/reference/expressions.html#conditional-expressions" rel="nofollow">conditional expression</a> and <a href="https://docs.python.org/3/library/stdtypes.html#str.endswith" rel="nofollow"><code>str.endswith</code></a>, we ensure that we do not accidentally add a <code>'s</code> on the end of a name that already ends in <code>s</code>.</p>
<hr/>
<p>You can just insert the name into the prompt string with <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format</code></a>:</p>
<pre><code>income = int(input("May I ask {}'s monthly income:  ".format(name)))
</code></pre>
<p>Also, there is no reason to do <code>str(input(...))</code> since <code>input</code> always returns a string object.</p>
<p>Finally, you should use a <code>,</code> with <code>print</code> instead of <code>+</code>:</p>
<pre><code>print("Hello!", name)
</code></pre>
<p>Otherwise, the output will be <code>Hello!hui</code>, which isn't very readable.</p>
</div>
