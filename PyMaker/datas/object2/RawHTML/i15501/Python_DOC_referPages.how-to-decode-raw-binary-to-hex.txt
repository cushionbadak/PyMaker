<p>I'm required to decode a raw binary value that looks like <code>b'\xa3\x13\xa4;\xcb\xda\x1b\x1b,Ut\xde\xeb2\xb5\x84\xe5&amp;\x85;'</code> to a hex value which then would look like <code>0x90D152B5ED57E00336FD8E106A7BCE28FC3EA588</code>.</p>
<p>I've tried to use raw_bin.decode("hex"), but that tells me to use <code>codecs.decode()</code>, but I'm lost..</p>
<p>How can I decode <code>b'\xa3\x13\xa4;\xcb\xda\x1b\x1b,Ut\xde\xeb2\xb5\x84\xe5&amp;\x85;'</code> to <code>0x90D152B5ED57E00336FD8E106A7BCE28FC3EA588</code> in python?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/binascii.html#binascii.hexlify" rel="nofollow"><code>binascii.hexlify</code></a></p>
<pre><code>&gt;&gt;&gt; x = b'\xa3\x13\xa4;\xcb\xda\x1b\x1b,Ut\xde\xeb2\xb5\x84\xe5&amp;\x85;'
&gt;&gt;&gt; binascii.hexlify(x).decode()
'a313a43bcbda1b1b2c5574deeb32b584e526853b'
</code></pre>
<p>Convert it to number using <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow"><code>int</code></a> with <code>base</code> parameter as 16:</p>
<pre><code>&gt;&gt;&gt; int(binascii.hexlify(x), 16)
931003516565576134942949873523045876335469036859
&gt;&gt;&gt; int(binascii.hexlify(x), 16) == 0xa313a43bcbda1b1b2c5574deeb32b584e526853b
True
</code></pre>
</div>
