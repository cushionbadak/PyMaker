<div class="post-text" itemprop="text">
<p>As described in <a href="https://docs.python.org/3/reference/datamodel.html#object.__getitem__" rel="nofollow">special method names</a>, the method name <code>__getitem__()</code> allows a class to override the <code>foo[bar]</code> syntax.  Lists do this to provide both subscripting (e.g. <code>foo[5]</code>) and slicing (<code>foo[1:5]</code>).  Dictionaries do this to provide key lookup.</p>
</div>
<span class="comment-copy"><code>list[index]</code> retrieves the item at position <code>index</code> from <code>list</code>. This functionality is implemented as a <code>__getitem__(self, index)</code> function. Basically, <code>rawcontent.split('')[0]</code> splits <code>rawcontent</code> into a list of strings, and then retrieves the first value (<code>[0]</code>). Does that answer your question?</span>
