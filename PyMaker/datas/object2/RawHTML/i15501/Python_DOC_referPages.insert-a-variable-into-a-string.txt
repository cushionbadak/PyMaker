<div class="post-text" itemprop="text">
<p>Concatenating large strings with <code>+</code> is kind of ugly in my opinion.  You should only use <code>+</code> when you have a few small strings to concatenate.</p>
<p>A cleaner approach would be to use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format</code></a> to insert the value into the string:</p>
<pre><code>html_str = """
&lt;div class=c-box&gt;
         &lt;img display="none" class=cimg src="http://example.com"&gt;
             &lt;h3&gt;Example Information&lt;/h3&gt;
              &lt;div class=c-boxcontent&gt;
                &lt;p&gt;

{}

                &lt;/p&gt;
              &lt;/div&gt;
&lt;/div&gt;
""".format(wikipedia.summary("Julius Caesar", sentences=2))
</code></pre>
<p>Note too that your posted code is actually invalid since <a href="https://docs.python.org/3/reference/simple_stmts.html#grammar-token-assignment_stmt" rel="nofollow">assignment is a statement</a> in Python.  Meaning, you cannot perform an assignment in the middle of a line.</p>
