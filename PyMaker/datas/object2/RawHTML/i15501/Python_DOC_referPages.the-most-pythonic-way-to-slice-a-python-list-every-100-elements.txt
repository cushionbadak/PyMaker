<p>I have a list contains many elements. I want to slice it every 100 elements to a list of several lists.</p>
<p>For example:</p>
<pre><code>&gt;&gt;&gt; a = range(256)
&gt;&gt;&gt; b = slice(a, 100)
</code></pre>
<p><code>b</code> should then be <code>[[0,1,2,...99],[100,101,102,...199],[200,201,...,255]]</code>.</p>
<p>What's the most pythonic and elegent way to do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>This should do the trick:</p>
<pre><code>[a[i:i+100] for i in range(0, len(a), 100)]
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html#func-range"><code>range</code> takes an optional third <code>step</code> argument</a>, so <code>range(0, n, 100)</code> will be <code>0, 100, 200, ...</code>.</p>
