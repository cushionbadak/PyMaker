<p>You can use <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow"><code>collections.Counter</code></a> and <a href="https://docs.python.org/3/library/string.html#string.ascii_letters" rel="nofollow"><code>string.ascii_letters</code></a> to get your letter count. The use of <code>string.ascii_letters</code> will allow you to only count letters, and not punctuation, etc.</p>
<pre><code>from collections import Counter
from string import ascii_letters

def get_word_frequencies(filename):
    with open(filename) as f:
        c = Counter(f.read())
    return {k:v for k,v in c.items() if k in ascii_letters}

c = get_word_frequencies('derp.py')

print(c)
