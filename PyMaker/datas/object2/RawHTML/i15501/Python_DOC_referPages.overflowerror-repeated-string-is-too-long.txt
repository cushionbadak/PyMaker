    print (wl)

<div class="post-text" itemprop="text">
<p>Your code</p>
<pre><code>if cmd == 'wavelength':
    freq = input("Enter frequency (mHz): ") * 10**6 
    wl = freq * 299792458
    print (wl)
</code></pre>
<p>has not just one problem, it has two problems</p>
<ol>
<li><p>The <code>input</code> function returns a string of characters, that you concatenate to itself a million times, say "107.6107.6107.6 ... 107.6", just one million times.<br/>
You want to convert your response to a floating point nuber, using <code>float</code>, <em>before</em> multiplying it by 10**6</p>
<pre><code>freq = float(input("Enter frequency (mHz): ")) * 10**6
</code></pre></li>
<li><p>The wavelength calculation is wrong<br/>
Example dimensional analysis: <code>[L] != (1/[T]) * [L]/[T] = [L]/[T]^2</code></p>
<pre><code>wl = c/freq
</code></pre></li>
</ol>
</div>
<div class="post-text" itemprop="text">
<p>If you're using Python 3.x, <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow"><code>input</code></a> returns a string object.</p>
<p>If you want to number multiplication, convert the return value of the string to numerical value.</p>
<pre><code>freq = int(input("Enter frequency (mHz): ")) * 10**6 
</code></pre>
<p>or</p>
<pre><code>freq = float(input("Enter frequency (mHz): ")) * 10**6 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You're multiplying a string:</p>
<pre><code>input("Enter frequency (mHz): ") * 10**6
wl = freq * 299792458
</code></pre>
<p>You probably meant:</p>
<pre><code>freq = int(input("Enter frequency (mHz): ")) * 10**6
wl = freq * 299792458
</code></pre>
</div>
