<div class="post-text" itemprop="text">
<p>In Python 2, <a href="https://docs.python.org/2/library/functions.html#zip" rel="noreferrer"><code>zip</code></a> returned a list. In Python 3, <a href="https://docs.python.org/3/library/functions.html#zip" rel="noreferrer"><code>zip</code></a> returns an iterable object. But you can make it into a list just by calling <a href="https://docs.python.org/3/library/functions.html#func-list" rel="noreferrer"><code>list</code></a>. </p>
<p>That would be:</p>
<pre><code>list(zip(*ngram))
</code></pre>
<p>That makes a list, like in Python 2, so you can use:</p>
<pre><code>list(zip(*ngram))[0]
</code></pre>
<p>etc. </p>
<p>But if you only need the first element, then you don't strictly need a list. You could just use <a href="https://docs.python.org/3/library/functions.html#next" rel="noreferrer"><code>next</code></a>.</p>
<p>That would be:</p>
<pre><code>next(zip(*ngram))
</code></pre>
