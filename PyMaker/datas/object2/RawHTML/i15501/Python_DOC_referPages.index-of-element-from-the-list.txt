<p>If I understand correctly, you want this:</p>
<pre><code>MailList = ["email1@gmail.com", "email2@gmail.com", "email3@gmail.com"]

for index, value in enumerate(MailList):
    print('user {} is {}'.format(index, value.split("@")[0]))
</code></pre>
<p>See <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow">the docs</a> for details on <code>enumerate</code>.</p>

<div class="post-text" itemprop="text">
<p>You need to split the emails with <code>@</code> :</p>
<pre><code>&gt;&gt;&gt; MailList = ['email1@gmail.com', 'email2@gmail.com', 'email3@gmail.com']
&gt;&gt;&gt; for i in MailList :
...   print ('user'+ ' is {}'.format(i.split('@')[0]) )
... 
user is email1
user is email2
user is email3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>using <code>itertools.takewhile</code>:</p>
<pre><code>&gt;&gt;&gt; import itertools
&gt;&gt;&gt; MailList = ['email1@gmail.com', 'email2@gmail.com', 'email3@gmail.com']
&gt;&gt;&gt; for x in MailList:
...     print("user is {}".format("".join(itertools.takewhile(lambda x:x!='@',x))))
... 
user is email1
user is email2
user is email3
</code></pre>
<p>using <code>index</code>:</p>
<pre><code>&gt;&gt;&gt; for x in MailList:
...     print("user is {}".format(x[:x.index('@')]))
... 
user is email1
user is email2
user is email3
</code></pre>
<p>using <code>find</code>:</p>
<pre><code>&gt;&gt;&gt; for x in MailList:
...     print("user is {}".format(x[:x.find('@')]))
... 
user is email1
user is email2
user is email3
</code></pre>
