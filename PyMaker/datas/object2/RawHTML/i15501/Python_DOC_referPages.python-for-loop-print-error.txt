<pre><code>for i in range(10):
    x = 0.1*i
    print x
    print x/(1-x*x)
</code></pre>
<p>I'm trying to print the results out using for loop, but it says <code>Syntax Error: Missing parentheses in call to 'print'</code>.</p>
<p>I'm using Python 3.4 and I'm new to Python.</p>

<div class="post-text" itemprop="text">
<p>The error message is crystal clear, isn't it? Your <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow"><code>print</code> function</a> is missing the parentheses needed for a function call:</p>
<pre><code>print(x)
</code></pre>
<p>Python 2 had a <code>print</code> statement where the syntax <code>print x</code> was correct; Python 3 has changed this. You should be learning Python from a Python 3 specific resource, for example the <a href="http://docs.python.org/3/tutorial/index.html" rel="nofollow">Python tutorial</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You syntax for print statement is valid in 2.X but it has changed in 3.X version. You need <a href="https://docs.python.org/3/whatsnew/3.0.html" rel="nofollow">parentheses</a> around your print statement like:</p>
<pre><code>print (x/(1-x*x))
</code></pre>
</div>
