<div class="post-text" itemprop="text">
<p>I want to loop through a list and terminate when I reach a certain value.
Something like:</p>
<pre><code>ls = ['yes','yes','stop','yes']

while a in ls &lt;&gt; 'stop':
    print a
</code></pre>
<p>Would print:</p>
<pre><code>yes
yes
</code></pre>
<p>I know I can do:</p>
<pre><code>for a in ls:
    if a == 'stop':
        break
    print a
</code></pre>
<p>but it seems messy.  </p>
</div>
<div class="post-text" itemprop="text">
<p>You could use list indexing and slicing:</p>
<pre><code>print ls[:ls.index('stop')]
# =&gt; ['yes', 'yes']
</code></pre>
<p>In a loop:</p>
<pre><code>for a in ls[:ls.index('stop')]:
    print a
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/itertools.html#itertools.takewhile" rel="nofollow"><code>itertools.takewhile</code></a>:</p>
<pre><code>&gt;&gt;&gt; from itertools import takewhile
&gt;&gt;&gt; ls = ['yes','yes','stop','yes']
&gt;&gt;&gt; for i in takewhile(lambda x: x != 'stop', ls):
...     i
...
'yes'
'yes'
&gt;&gt;&gt;
</code></pre>
</div>
<span class="comment-copy">It might be "messy", but it is perfectly clear &amp; does the job</span>
<span class="comment-copy">Also, you should not be using <code>&lt;&gt;</code>.  That operator was deprecated in Python 2.5 and removed entirely from Python 3.x.  You should be using <code>!=</code> instead.</span>
