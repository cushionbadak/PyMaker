<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.join"><code>str.join</code></a> to more easily join a sequence of letters with a given delimiter, in this case a space <code>' '</code></p>
<pre><code>def stars(n, delim=' '):
    for x in reversed(range(n+1)):
        print(delim.join('*'*x))

stars(5)
</code></pre>
<p>This gives</p>
<pre><code>* * * * *
* * * *
* * *
* *
*
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>When you multiply a string, it doesn't have to be a single character, you can use <code>print x * '* '</code></p>
<pre><code>&gt;&gt;&gt; def stars(n):
...         for x in reversed(range(n+1)):
...                 print x * '* '
...
&gt;&gt;&gt; stars(4)
* * * * 
* * * 
* * 
* 
</code></pre>
<p>Note that this will also show trailing space after each line; if this is problematic for you, you can use:</p>
<pre><code>print (x * '* ')[:-1]
</code></pre>
<p>You could also import the print function that use the separator argument <code>sep</code>:</p>
<pre><code>&gt;&gt;&gt; from __future__ import print_function
&gt;&gt;&gt; print(*('*'*4),sep=' ')
