<div class="post-text" itemprop="text">
<p>Let's look at finding the first step for <code>range (4, -2, -2)</code></p>
<p>Range function's documentation: <code>range(start, stop[, step]) -&gt; list of integers</code></p>
<p>So you have <code>start = 4</code>, <code>stop=-2</code>, and <code>step=-2</code>.</p>
<p>In the formula <code>r[i] = start + step*i</code>, <em><code>i</code> is always counting from 0</em>. </p>
<p>So the first element, <code>r[0] = 4 + (-2) * 0 = 4</code>.</p>
<p>The constraints hold. <code>i &gt;= 0</code> because <code>i</code> is 0, and <code>r[i] == 4</code>, which is greater than <code>stop</code> (<code>-2</code>).</p>
