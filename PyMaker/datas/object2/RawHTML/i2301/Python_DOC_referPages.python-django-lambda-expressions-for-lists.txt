<div class="post-text" itemprop="text">
<p>You can not put this inside a lambda expression, but you can use a generator object like this:</p>
<pre><code># please note that i have used tuples instead of sets, 
# because sets are unordered
data = [
    (' Customers ','null,blank '),
    (' CustomersName ','max=50,null,blank '), 
    (' CustomersAddress ','max=150,blank '),
    (' CustomersActive ','Active ')
]

# Indexing is not allowed for set objects
values = [item[0].strip() for item in data]
</code></pre>
<p>see: </p>
<p><a href="https://wiki.python.org/moin/Generators" rel="nofollow noreferrer">https://wiki.python.org/moin/Generators</a></p>
<p><a href="https://docs.python.org/3/tutorial/datastructures.html#sets" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/datastructures.html#sets</a></p>
<p>EDIT:</p>
<p>If you wan't to use dictionaries you could use something like this:</p>
<pre><code>data = [
    {' Customers ': 'null,blank '},
    {' CustomersName ': 'max=50,null,blank '}, 
    {' CustomersAddress ': 'max=150,blank '},
    {' CustomersActive ': 'Active '}
]
# expecting a single value in the dicts
values = [item.values()[0].strip() for item in data]
</code></pre>
