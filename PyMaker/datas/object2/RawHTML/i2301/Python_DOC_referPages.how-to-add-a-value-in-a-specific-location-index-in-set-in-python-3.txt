<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/tutorial/datastructures.html#sets" rel="nofollow noreferrer">Sets are unordered collections.</a> What you want is impossible. Consider these alternatives:</p>
<ul>
<li><code>list</code>, ordered but permits duplicates.</li>
<li><code>dict.keys</code> in Python 3.6+, where dictionaries are insertion ordered.</li>
<li><code>OrderedDict.keys</code> in Python versions prior to 3.6.</li>
<li>Create a custom class, e.g. <a href="https://github.com/ActiveState/code/blob/3b27230f418b714bc9a0f897cb8ea189c3515e99/recipes/Python/576696_OrderedSet_with_Weakrefs/README.md" rel="nofollow noreferrer">this recipe</a>.</li>
<li>Use a 3rd party library, e.g. <a href="https://pypi.org/project/ordered-set/" rel="nofollow noreferrer"><code>ordered-set</code></a>.</li>
</ul>
<p>See also: <a href="https://stackoverflow.com/questions/1653970/does-python-have-an-ordered-set">Does Python have an ordered set?</a></p>
