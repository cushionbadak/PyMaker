<p>It is required in the finished lines to replace the characters with the corresponding html-code.</p>
<p>for example:</p>
<pre><code>rules = {
    '&lt;':'&amp;lt;',
    '&gt;':'&amp;gt;'
}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Since python 3.2 there is the <a href="https://docs.python.org/3/library/html.html" rel="nofollow noreferrer">html</a> module with is <a href="https://docs.python.org/3/library/html.html#html.escape" rel="nofollow noreferrer">escape function</a>.</p>
<p>For example</p>
<pre><code>&gt;&gt;&gt; import html
&gt;&gt;&gt; html.escape("5 &gt; 3")
'5 &amp;gt; 3'
</code></pre>
</div>
<span class="comment-copy">Rather than doing this, consider using something that manipulates a DOM and letting it encode the output.</span>
