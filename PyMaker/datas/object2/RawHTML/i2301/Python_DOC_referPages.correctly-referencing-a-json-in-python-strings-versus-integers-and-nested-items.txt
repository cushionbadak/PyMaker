<p>If you're using the regular JSON module:</p>
<pre><code>import json
</code></pre>
<p>And you're opening your JSON like this:</p>
<pre><code>json_data = open("my_json.json").read()
data      = json.loads(json_data)

# Equivalent to:
data      = json.load(open("my_json.json"))

# Notice json.load vs. json.loads
</code></pre>
<p>Then this should do what you want:</p>
<pre><code>distance_text, duration_text = [data['rows'][0]['elements'][0][key]['text'] for key in ['distance', 'duration']]
</code></pre>
<p>Hope this is what you wanted!</p>
