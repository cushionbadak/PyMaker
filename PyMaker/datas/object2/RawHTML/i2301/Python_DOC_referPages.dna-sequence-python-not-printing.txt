<div class="post-text" itemprop="text">
<p>You need to assign a value to <code>base</code>. In your case it would look something like that:</p>
<pre><code>def sequence():
    base = pair()
    n = random.randint(1, 3)
    print(base * n)
</code></pre>
<p>And you should try to avoid using a variable name multiple times, so you can read your code more easily. :)</p>
