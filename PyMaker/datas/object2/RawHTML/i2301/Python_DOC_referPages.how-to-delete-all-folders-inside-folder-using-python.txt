<div class="post-text" itemprop="text">
<p>Yes it is, Use shutil's <code>rmtree</code> method.</p>
<pre><code>import shutil 
shutil.rmtree('directory') # the directory you want to remove
os.listdir()
</code></pre>
<p>You can also use <code>os.rmdir</code> but that won't work if there is any content in it.</p>
<p>If you want to check if that specific path is a directory then you can use <code>os.path.isdir</code> then run <code>rmtree</code> if that returns <code>TRUE</code></p>
<p>And incase you want to keep the <em>root</em> folder intact then you could <code>walk</code> that directory and calling <code>rmtree</code> on each item.</p>
</div>
<div class="post-text" itemprop="text">
<p>As suggested by <a href="https://stackoverflow.com/questions/52491352/how-to-delete-all-folders-inside-folder-using-python/52491393#52491393">@Vineeth Sai</a> earlier, If you want to delete all sub directories in a directory, just loop through each file with <a href="https://docs.python.org/3/library/os.html#os.listdir" rel="nofollow noreferrer"><code>os.listdir()</code></a> and if the file is a directory, apply <a href="https://docs.python.org/3/library/shutil.html#shutil.rmtree" rel="nofollow noreferrer"><code>shutil.rmtree()</code></a>:</p>
<pre><code>from os import listdir

from os.path import abspath
from os.path import isdir
from os.path import join

from shutil import rmtree

path = 'YOUR PATH HERE'

for file in listdir(path):
    full_path = join(abspath(path), file)

    if isdir(full_path):
        rmtree(full_path)
</code></pre>
<p>The above also uses <a href="https://docs.python.org/3/library/shutil.html#shutil.rmtree" rel="nofollow noreferrer"><code>os.isdir()</code></a> to check if a file is a directory. </p>
</div>
<div class="post-text" itemprop="text">
<p>If <code>Vineeth's</code> answer is not suitable for your case, you could you <code>subprocess</code> module to run <code>os</code> specific commands like below</p>
<pre><code>import subprocess
subprocess.call('rm -rf /path/of/the/dirctory/*', shell=True)
</code></pre>
<p>The above command is <code>linux</code> specific, you could use windows counterpart of the same command above.</p>
<p>Note - Here <code>shell=True</code> will expand <code>*</code> into files/folders.  </p>
<p>Also, note <code>Vineeth's</code> answer is <code>os</code> independent, and above will be <code>os</code> specific. <strong>Be cautious</strong>.   </p>
<p>P.S. - You can also run <code>powershell</code> commands using <code>subprocess</code> module.</p>
