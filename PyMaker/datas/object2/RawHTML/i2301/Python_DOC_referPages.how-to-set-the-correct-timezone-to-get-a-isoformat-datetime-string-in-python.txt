<div class="post-text" itemprop="text">
<p><code>utcnow()</code> already gives you the the time at <code>+00:00</code>, if you'd like to recieve the time at a specific timezone, you should provide the timezone as an argument to <code>now([tz])</code>.</p>
<p><a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer">https://docs.python.org/3/library/datetime.html</a></p>
<pre><code>&gt;&gt;&gt; import datetime as dt
&gt;&gt;&gt; dt.datetime.now(tz = dt.timezone(offset = dt.timedelta(hours = 2))).replace(microsecond = 0).isoformat()
'2018-09-28T09:20:19+02:00'
</code></pre>
