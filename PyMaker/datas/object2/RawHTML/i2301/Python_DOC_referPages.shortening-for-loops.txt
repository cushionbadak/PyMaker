<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer"><code>sum</code></a> function is the Pythonic way to sum.  <code>sum()</code> takes an iterable.  In this case, a very helpful iterable is the <a href="https://docs.python.org/3/reference/expressions.html#generator-expressions" rel="nofollow noreferrer">generator expression</a>. Generator expressions use (basically) the same syntax as <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehensions</a>.  Using <code>sum()</code> and a generator expression, what you are trying to do, can be expressed, quite simply, as:</p>
<pre><code>sum(human.limbs for human in humans)
</code></pre>
