<div class="post-text" itemprop="text">
<p>Currently, you're trying to add <code>1</code> to the string <code>s</code>, which will throw a <code>TypeError</code>. Even if <code>s</code> <em>was</em> an <code>int</code>, incrementing it in this fashion would not move to the next iteration of the loop in Python (or any language I know).</p>
<p>You can use the keyword <code>continue</code> to move to the next iteration of a loop immediately. Docs here: <a href="https://docs.python.org/3/tutorial/controlflow.html#break-and-continue-statements-and-else-clauses-on-loops" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/controlflow.html#break-and-continue-statements-and-else-clauses-on-loops</a></p>
<p>However, I don't think that is exactly what you want to do here, since it looks like you are trying to count the occurrences of the substring <code>'bob'</code> in your main string <code>s</code>.</p>
<p>Instead you should iterate over the indices of the characters of <code>s</code> and at each point check if the current character and next two together form the substring <code>'bob'</code>. If so increment <code>counter_bob</code>.</p>
<p>An example refactor of your code with this in mind:</p>
<pre><code>s = 'mmmmbobob'
counter_bob = 0

for i in range(len(s)):
    if s[i:i+3] == 'bob':
        counter_bob += 1

print('Number of times bob occurs is: %d' % (counter_bob))
</code></pre>
<p>Which prints:</p>
<pre><code>Number of times bob occurs is: 2
</code></pre>
