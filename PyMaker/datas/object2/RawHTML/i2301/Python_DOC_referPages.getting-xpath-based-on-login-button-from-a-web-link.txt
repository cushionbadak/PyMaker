<p>The <code>KeyError</code> in python generally occurs when your key is not found. You sure 'loginButton' is in set of existing keys?</p>
<p>From official python libraries, </p>
<blockquote>
<p>exception KeyError</p>
<p>Raised when a mapping (dictionary) key is not found in the set of
  existing keys.</p>
</blockquote>
<p>Please see <a href="https://docs.python.org/3/library/exceptions.html#KeyError" rel="nofollow noreferrer">this</a> link.</p>
