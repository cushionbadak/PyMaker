<p>I have the following list:</p>
<pre><code>my_list = ["shanya", 1.2, 1, False, "test", 2, 3.3, True, "3", "4.0"]
</code></pre>
<p>But how do I access multiple elements in this list?</p>
<p>I know how to access one specific element (e.g., <code>my_list[1]</code> gives me <code>[1.2]</code>) and a range (e.g., <code>my_list[1:3]</code> gives me <code>[1.2, 1]</code>) but how do I get the first and third to last element?</p>
<pre><code>["shanya", 1, False, "test", 2, 3.3, True, "3", "4.0"]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Simply contact the sub-lists, like:</p>
<pre><code>result = l[:1] + l[2:]
</code></pre>
<p>output:</p>
<pre><code>['shanya', 1, False, 'test', 2, 3.3, True, '3', '4.0']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/operator.html#operator.itemgetter" rel="nofollow noreferrer"><strong><code>itemgetter</code></strong></a>:</p>
<pre><code>&gt;&gt;&gt; from operator import itemgetter

&gt;&gt;&gt; values = ["shanya", 1.2, 1, False, "test", 2, 3.3, True, "3", "4.0"]

&gt;&gt;&gt; itemgetter(0, -3)(values)
('shanya', True)
</code></pre>
</div>
<span class="comment-copy">Index <code>1</code> will give you the second item. If you want the first item you need to use <code>0</code>.</span>
