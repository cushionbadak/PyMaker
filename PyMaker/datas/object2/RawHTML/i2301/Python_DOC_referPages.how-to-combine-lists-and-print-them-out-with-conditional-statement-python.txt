<p>If you know you have lists of equal lengths:</p>
<pre><code>for i in range(len(stations)):
    print(stations[i], lats[i], lons[i])
</code></pre>

<h1>Create a dictionary with the stations and their coordinates</h1>
<p>You can use the built-in function <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip()</a> to combine these lists and then iterate over them:</p>
<pre><code>locations = {}

for station, latitude, longitude in zip(stations, lats, lons):
   locations[station] = latitude, longitude
</code></pre>
<p>This will result in a dictionary with the following format:</p>
<pre><code>locations = {
  "Hanko Russarö" : (59.77, 22.95),
  "Heinola Asemantaus" : (61.2, 26.05),
  ...
}
</code></pre>
<p>Of course this can also be written on a single line</p>
<pre><code>locations = { station : (lat, lon) for station, lat, lon in zip(statios, lats, lons)}
</code></pre>
<h1>Creating the dictionary with additional conditions</h1>
<p>This can be combined with some conditions, for example:</p>
<pre><code>locations = { station : (lat, lon) for station, lat, lon in zip(statios, lats, lons) if lat &gt; 50}
</code></pre>
<p>or</p>
<pre><code>locations = {}

for station, latitude, longitude in zip(stations, lats, lons):
    if latitude &gt; 50:
        locations[station] = latitude, longitude
</code></pre>
