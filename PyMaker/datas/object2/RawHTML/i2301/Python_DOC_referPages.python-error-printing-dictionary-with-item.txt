<div class="post-text" itemprop="text">
<p>I asume you are using <code>python 3.x</code> (because of the <code>print()</code> function you use).</p>
<p>The <a href="https://docs.python.org/3/library/stdtypes.html?highlight=dict#dict.items" rel="nofollow noreferrer"><code>dict.items()</code></a> method returns a view, not a list directly in this version of <code>python</code>.</p>
<p>To get a list you need to iterate over the view, maybe converting it to a list (using <code>list()</code>) or tuple (using <code>tuple()</code>). Here is an example:</p>
<pre><code>print(list(vital_info.items()))
</code></pre>
</div>
