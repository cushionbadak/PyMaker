<div class="post-text" itemprop="text">
<p>I am building up <code>slice</code>s programatically for reasons, and would like to create a slice that grabs "everything".  My immediate intuition was that </p>
<pre><code>my_slice = slice() 
</code></pre>
<p>would produce an object such that</p>
<pre><code>assert (my_list[my_slice] == my_list) #True
</code></pre>
<p>However, a quick look at <a href="https://docs.python.org/3/library/functions.html#slice" rel="nofollow noreferrer">the docs</a> reveals that <code>slice</code> <em>requires</em> a <code>stop</code> parameter.</p>
<p>Can I obtain an object <code>all_slice</code> such that <code>any_list[all_slice] == any_list</code> for all <code>any_list</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can pass <code>None</code>:</p>
<pre><code>my_list = [1, 2, 3, 4]
my_slice = slice(None)
print(my_list[my_slice])
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[1, 2, 3, 4]
</code></pre>
</div>
