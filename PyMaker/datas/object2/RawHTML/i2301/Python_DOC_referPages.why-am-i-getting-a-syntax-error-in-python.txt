<div class="post-text" itemprop="text">
<pre><code># works in python 2
return_age = 4
if return_age != 0:
    print "Your age is %s years" %(return_age)

# works in python 3 
return_age = 4
if return_age != 0:
    print("Your age is %s years" %(return_age))
</code></pre>

<p>Print is a function in Python 3, so you need parentheses.</p>
