<div class="post-text" itemprop="text">
<p>I have two nested lists like this:</p>
<pre><code>list_1 = [[5, 3, 4], [1, 5, 8], [6, 4, 2]]
list_2 = [[8, 3, 4], [1, 5, 9], [6, 7, 2]]
</code></pre>
<p>I want to list the different values from both of the nested lists, into a list like this:</p>
<pre><code>[5,8,8,9,4,7]
</code></pre>
<p>Is there any way in Python 3 to do this?</p>
<p>This what I have tried:</p>
<pre><code>list_1 = [[5, 3, 4], [1, 5, 8], [6, 4, 2]]
list_2 = [[8, 3, 4], [1, 5, 9], [6, 7, 2]]
storer = []
for x in list_2:
    if x not in list_1:
       storer.append(x)
print(storer)
</code></pre>
<p>but it returns:</p>
<pre><code>[[8, 3, 4], [1, 5, 9], [6, 7, 2]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>#!/usr/bin/env python

list_1 = [[5, 3, 4], [1, 5, 8], [6, 4, 2]]
list_2 = [[8, 3, 4], [1, 5, 9], [6, 7, 2]]

difflist = [a[i][j] for a in zip(list_1, list_2) for i in range(len(a)) for j in range(len(a[i])) if a[i][j] != a[i-1][j]]
print(difflist)
</code></pre>
<p>Which prints:</p>
<pre><code>[5,8,8,9,4,7]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Another approach using <code>set</code> difference:</p>
<pre><code>final = sum((list(set(k) - set(v)) + list(set(v) - set(k)) for k, v in zip(list_1, list_2)), [])
# [5, 8, 8, 9, 4, 7]
</code></pre>
<p>And the same result using <code>reduce</code>:</p>
<pre><code>from functools import reduce

final = reduce(list.__iadd__, (list(set(k) - set(v)) + list(set(v) - set(k)) for k, v in zip(list_1, list_2)))
# [5, 8, 8, 9, 4, 7]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Later answer, but you can use <code>set(list_1[i]) ^ set(list_2[i])</code>, i.e.:</p>
<pre><code>final = []
for i in range(len(list_1)):
    [final.append(x) for x in reversed([y for y in set(list_1[i]) ^ set(list_2[i])])]
# [5, 8, 9, 8, 7, 4]
</code></pre>
