<p>As <a href="https://stackoverflow.com/questions/50094765/how-to-convert-bytearray-to-string-in-python/50094913#comment87205717_50094765">t.m.adam</a> said in the comments, use <a href="https://docs.python.org/3/library/stdtypes.html#bytearray.decode" rel="nofollow noreferrer"><code>bytearray.decode</code></a></p>
<pre><code>b = bytearray("test", encoding="utf-8")
b.decode()
#&gt; 'test'
</code></pre>
