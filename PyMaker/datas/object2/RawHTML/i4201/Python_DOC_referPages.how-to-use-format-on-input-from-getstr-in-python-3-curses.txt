<p><a href="https://docs.python.org/3/library/curses.html#curses.window.getstr" rel="nofollow noreferrer">From <code>getstr</code> docs</a>
:</p>
<blockquote>
<p>Read a bytes object from the user, with primitive line editing capacity.</p>
</blockquote>
<p>So this API returns a <code>bytes</code> object that represents, well, raw data. You can convert it to text using <code>a.decode()</code> (it will assume UTF-8 encoding by default).</p>
