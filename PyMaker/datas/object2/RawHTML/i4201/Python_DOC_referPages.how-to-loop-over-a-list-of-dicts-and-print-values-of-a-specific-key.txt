<p>Running Python 3.4.</p>

<pre><code>[{'state': 'DEAD',
  'id': 'phwl',
  'type_name': 'GAME',
  'unit_structure': 'lattice',
  'vendor': 'Downward',
  'type_id': 'shiftable'

 {'state': 'ALIVE',

  'type_name': 'BOARD',
  'unit_structure': 'frame',
  'vendor': 'Sniggles',
  'weight': 'light'}]
</code></pre>
<p>I want to do this using a for loop where I pull just the value after the <code>'id'</code> key and print it to my console. A simple for loop looks like this:</p>
<pre><code>for i in parts:
    print(i)

<pre><code>id
id
id
id
....etc
</code></pre>
<p>So I need to do something where I tell it "every time you see <code>'id'</code>, print the value after it, but I'm unsure how to structure that loop.</p>

<div class="post-text" itemprop="text">
<p>If I understood you correctly. You are looking for fetching a value using key from a dict.</p>
<p><strong>Ex:</strong></p>
<pre><code>for i in parts:
    print(i["id"])   #or print(i.get("id"))
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>phwl
a06c5
</code></pre>
