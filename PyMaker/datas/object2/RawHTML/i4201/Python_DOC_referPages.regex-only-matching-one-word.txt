<p>In python, using <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><code>re.findall</code></a> exhibits the following behavior:</p>
<blockquote>
<p>If one or more groups are present in the pattern, return a list of groups;</p>
</blockquote>
<p>Since <code>.*</code> (anything found after your numbers) is not contained in a group, it will not be in the final result.  Either remove the groups altogether, or add a group for the text following your number:</p>
