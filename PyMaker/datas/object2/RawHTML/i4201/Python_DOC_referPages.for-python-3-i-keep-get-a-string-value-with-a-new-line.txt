<div class="post-text" itemprop="text">
<p>You need to <a href="https://docs.python.org/3/library/stdtypes.html#str.rstrip" rel="nofollow noreferrer">strip</a> the newlines off each of the dictionary lines. In other words:</p>
<pre><code>for word in a:
    c,b = word.rstrip().split(",")
    dictionary[c] = b
</code></pre>
<p>When you iterate a file like <code>for word in a:</code>, you get a string for each line in the file, <em>including the newline at the end</em>. So, your dictionary ends up full of entries like <code>'y': 'why\n'</code> instead of <code>'y': 'why'</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can strip the trailing newline from <code>word.split(",")</code> by calling <a href="https://docs.python.org/3/library/stdtypes.html" rel="nofollow noreferrer">str.strip</a>.</p>
<pre><code>word.strip().split(",")
</code></pre>
<p>You can also use <code>read()</code> to load the contents of the file which doesn't include newlines.</p>
