<p>I have a list of sets like this:</p>
<pre><code>A=[{1,2},{3,4},{1,5}]
</code></pre>
<p>I want to map <code>A</code> to a list of lists instead of list of sets like this:</p>
<pre><code>A=[[1,2],[3,4],[1,5]]
</code></pre>
<p>can any body help me How can I do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map</code></a>.</p>
<p><strong>Ex:</strong></p>
<pre><code>&gt;&gt;&gt; A=[{1,2},{3,4},{1,5}]
&gt;&gt;&gt; print(list(map(list, A)))
[[1, 2], [3, 4], [1, 5]]
</code></pre>
