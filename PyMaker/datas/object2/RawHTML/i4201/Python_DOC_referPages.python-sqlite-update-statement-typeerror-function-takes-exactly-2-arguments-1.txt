<div class="post-text" itemprop="text">
<p>I'm not sure you need to use <code>executemany</code> here when you're only executing a single statement. You could use simply:</p>
<pre><code>cur.execute("UPDATE SEN_Table SET SenNumber = " + senNumStr + " WHERE FormName='" + nameGroup + "'")
</code></pre>
<p>It would be better to use a parameterised query rather than a formatted string though, to avoid SQL injection vulnerabilities:</p>
<pre><code>cur.execute("UPDATE SEN_Table SET SenNumber = ? WHERE FormName = ?", (senNumStr, nameGroup))
</code></pre>
</div>
<span class="comment-copy"><a href="https://xkcd.com/327/" rel="nofollow noreferrer">Don't build SQL using strings and <code>+</code>.</a> Put <code>?</code> as a placeholder wherever you want to use a value. See the <a href="https://docs.python.org/3/library/sqlite3.html#sqlite3.Cursor.executemany" rel="nofollow noreferrer"><code>executemany</code> documentation</a>.</span>
