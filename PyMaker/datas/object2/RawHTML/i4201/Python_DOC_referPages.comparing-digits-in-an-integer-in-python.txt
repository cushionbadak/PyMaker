<p>You would need a loop.</p>
<p>for example:</p>
<pre><code>a = int(input("Enter a 4-digit number: "))

b = [int(i) for i in str(a)]


def isAscending(b):
  #loop for as many digits in the array
  for x in range(0, len(b) - 1):
    # if the next number is less than the previous return false
    if b[x] &gt; b[x+1]:
      return False
  #  did not fail so return true
  return True

if isAscending(b):
  print ("the number is in ascending order")
else:
  print ("the number is not in ascending order")
</code></pre>

<div class="post-text" itemprop="text">
<p>If you're working with such a small number of digits, just create a new list w/ the elements in ascending order. Fortunately, Python has a function that does exactly that, the built-in function <a href="https://docs.python.org/3/howto/sorting.html" rel="nofollow noreferrer"><code>sorted</code></a>:</p>
<pre><code>def is_ascending(lst):
    return sorted(lst) == lst
</code></pre>
