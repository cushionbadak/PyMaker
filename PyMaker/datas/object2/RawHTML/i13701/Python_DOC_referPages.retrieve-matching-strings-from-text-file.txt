<p>I have the following text file and I want to retrieve the numbers in brackets</p>
<pre><code>ID&amp;number:Track_number(12930)_
ID&amp;number:Track_number(394839)_
ID&amp;number:Track_number(958236)_
</code></pre>
<p>So I've tried this </p>
<pre><code> import re

 file = open("text.txt", "r")
 text = file.read()
 file.close()
 pattern = re.compile(ur'Track_number(.*)_', re.UNICODE)
 string = pattern.search(text).group(1)
 print string
</code></pre>
<p>But it only displays the first result : <code>(12930)</code>.
I was wondering if it was possible to have a list of all the matching results.
Thanks</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow"><code>re.findall</code></a> for example</p>
<pre><code>&gt;&gt;&gt; re.findall('\((\d+)\)', text)
['12930', '394839', '958236']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>All you have to do is replace that <code>search</code> with <code>findall</code>. This will produce a <code>list</code> of all the matches.</p>
