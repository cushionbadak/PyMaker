<p>This feels like something that can be done with <a href="https://docs.python.org/3/library/datetime.html#datetime-objects">datetime</a> and <a href="https://docs.python.org/3/library/datetime.html#timedelta-objects">timedelta</a> objects.</p>
<p>For the tuple creation tuple I've used a generator expression.</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime, timedelta
&gt;&gt;&gt; dt = datetime(1970 ,1 ,1)
&gt;&gt;&gt; t = timedelta(minutes=30)
&gt;&gt;&gt; tuple((str(i), (dt + (i * t)).strftime("%X")) for i in range(48))
(('0', '00:00:00'), ..., ('47', '23:30:00'))

<div class="post-text" itemprop="text">
<p>You can utilize a <a href="http://anandology.com/python-practice-book/iterators.html#generators" rel="nofollow">generator</a> to build a list of times at 30 minute intervals. </p>
<pre><code>def yield_times():  
    start = datetime.combine(date.today(), time(0, 0))
    yield start.strftime("%H:%M:%S")
    while True:
        start += timedelta(minutes=30)
        yield start.strftime("%H:%M:%S")
</code></pre>
<hr/>
<p>Using this generator, we can create a list of tuples. <code>48</code> in this case, is the number of iterations through the generator. This <code>range</code> statement will start at <code>0</code>, giving us values 0 - 47.</p>
<pre><code>gen = yield_times()
tuple_list = []
for ii in range(48):
    tuple_list.append((ii, gen.next()))
</code></pre>
<hr/>
<p>Finally, we need to convert the list of tuples to a tuple of tuples:</p>
<pre><code>tuple_times = tuple(tuple_list)
</code></pre>
<hr/>
<p>Full script:</p>
<pre><code>from datetime import date, time, datetime, timedelta
def yield_times():  
    start = datetime.combine(date.today(), time(0, 0))
    yield start.strftime("%H:%M:%S")
    while True:
        start += timedelta(minutes=30)
        yield start.strftime("%H:%M:%S")

gen = yield_times()
tuple_list = []
for ii in range(48):
    tuple_list.append((ii, gen.next()))

tuple_times = tuple(tuple_list)
</code></pre>
