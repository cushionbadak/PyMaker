<p>You can use <a href="https://docs.python.org/3/library/io.html#io.IOBase.writelines" rel="nofollow"><code>writelines()</code></a> to write an iterable to your file :</p>
<pre><code>with open (filename, "a") as f :
    f.writelines(map(lambda x:"==&gt; {}\n".format(x),diff))
</code></pre>

<div class="post-text" itemprop="text">
<p>You could use <code>file.writelines()</code> and a generator expression:</p>
<pre><code>first = dict(a=1, b=2)
second = dict(b=0, c=3)

with open("disk_output.txt","w") as f:
    f.writelines("==&gt; {}\n".format(k) for k in set(first) ^ set(second))
</code></pre>
<p>Contents of <code>disk_output.txt</code> file afterwards:</p>
<pre class="lang-none prettyprint-override"><code>==&gt; a
==&gt; c
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You don't need a set, you can use <code>dict.viewkeys</code> for python2 then just format the output however you want and write, also open the file once using <code>with</code> which will automatically close your file:</p>
<pre><code>first = dict(a=1, b=2)
second = dict(b=0, c=3)
with open("disk_output.txt","w") as f:
    f.writelines("==&gt;{}\n".format(k) for k in first.viewkeys() ^ second)
</code></pre>
<p>Just use <code>.keys</code> for python3.</p>
</div>
