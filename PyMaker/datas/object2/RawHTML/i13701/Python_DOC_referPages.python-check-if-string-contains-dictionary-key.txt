<div class="post-text" itemprop="text">
<p>I have these strings:</p>
<pre><code>&gt;&gt;&gt; a="foo"
&gt;&gt;&gt; b="foo_KEY"
</code></pre>
<p>And a dictionary like:</p>
<pre><code>&gt;&gt;&gt; DICT
{
'KEY': ['...'],
'KEY2': ['...', '...'],
...
}
</code></pre>
<p>I'd like to build a function that check if the test value is in any key of the input dict:</p>
<pre><code>&gt;&gt;&gt; has_key(a, DICT)
False
&gt;&gt;&gt; has_key(b, DICT)
True
</code></pre>
<p>What is the most elegant way to do this task in Python 3?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>has_key = lambda a, d: any(k in a for k in d)
</code></pre>
