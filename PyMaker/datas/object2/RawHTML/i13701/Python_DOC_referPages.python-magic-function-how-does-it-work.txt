<div class="post-text" itemprop="text">
<p><code>%x</code> is a <a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="nofollow">format code</a> for hex. Your function is representing a decimal value in hex.</p>
<pre><code>&gt;&gt;&gt; hex(265631021230191344138857284998518456)
'0x33289ed88646a64b9fc63c808fd6b8'
</code></pre>
<p>The full format code</p>
<pre><code>'%032x'
</code></pre>
<p>Means represent the value in <code>hex</code> and pad it with zeros on the left until it is 32 characters wide.</p>
</div>
