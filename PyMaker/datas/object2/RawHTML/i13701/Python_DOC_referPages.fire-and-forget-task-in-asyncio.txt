<p>When using Python's <code>asyncio</code> library, how can I fire off a task and then not care about its completion?</p>
<h2>Example</h2>
<pre><code>@asyncio.coroutine
def f():
    yield From(asyncio.sleep(1))
    print("world!")

@asyncio.coroutine
def g():
    desired_operation(f())
    print("Hello, ")
    yield From(asyncio.sleep(2))

&gt;&gt;&gt; loop.run_until_complete(g())
'Hello, world!'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You're looking for <a href="https://docs.python.org/3/library/asyncio-task.html#asyncio.ensure_future" rel="nofollow"><code>asyncio.ensure_future</code></a> (or <code>asyncio.async</code> if your version of <code>trollius</code>/<code>asyncio</code> is too old to have <code>ensure_future</code>):</p>
<pre><code>from __future__ import print_function

import trollius as asyncio
from trollius import From

@asyncio.coroutine
def f():
    yield From(asyncio.sleep(1))
    print("world!")

@asyncio.coroutine
def g():
    asyncio.ensure_future(f())
    print("Hello, ", end='')
    yield From(asyncio.sleep(2))

loop = asyncio.get_event_loop()
loop.run_until_complete(g())
</code></pre>
<p>Output:</p>
<pre><code>Hello, world!
</code></pre>
</div>
