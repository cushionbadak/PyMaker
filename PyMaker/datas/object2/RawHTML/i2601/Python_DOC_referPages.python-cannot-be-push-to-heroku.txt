<p>Any chance this could be a python3 import error on the import for the module <code>ConfigParser</code>?</p>
<p>I hope this helps.</p>
<p>This is <a href="https://stackoverflow.com/users/1709587/mark-amery">Abe Karplus's</a> <a href="https://stackoverflow.com/a/14087705/7070685">answer</a>: </p>
<blockquote>
<p>In Python 3, <a href="https://docs.python.org/3/library/configparser.html" rel="nofollow noreferrer"><code>ConfigParser</code></a> has been renamed to <a href="https://docs.python.org/3/library/configparser.html" rel="nofollow noreferrer"><code>configparser</code></a> for PEP 8 compliance. It looks like the package you are installing does not support Python 3.</p>
</blockquote>
