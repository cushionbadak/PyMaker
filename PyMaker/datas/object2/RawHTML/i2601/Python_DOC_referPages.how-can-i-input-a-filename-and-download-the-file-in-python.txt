<p>Well, do you have the database online?
If so I would suggest you the <a href="http://docs.python-requests.org/en/master/" rel="nofollow noreferrer">requests</a> module, very pythonic and fast.
Another great module based on requests is <a href="https://robobrowser.readthedocs.io/en/latest/readme.html" rel="nofollow noreferrer">robobrowser</a>.</p>
<p>Eventually, you may need <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/" rel="nofollow noreferrer">beautiful soup</a> to parse the HTML or XML data.</p>
<p>I would avoid using selenium because it's designed for web-testing, it needs a browser and its webdriver and it's pretty slow. It doesn't fit your needs at all.</p>
<p>Finally, to interact with the database I'd use <a href="https://docs.python.org/3/library/sqlite3.html" rel="nofollow noreferrer">sqlite3</a></p>
<p>Here a sample:</p>
