<div class="post-text" itemprop="text">
<p>How can we use <strong>python3 -m</strong> option to run on a package/module that resides in different directory rather than current working directory?</p>
<p>Suppose, my current working directory is "/tmp/dir_one"
And my python package resides in "/tmp/dir_two/package".</p>
<p>How to execute the following command from <code>/tmp/dir_one</code>?</p>
<blockquote>
<p>python3 -m dir_two.package.module</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>For a module in <code>/tmp/foo</code> like:</p>
<p><code>/tmp/foo/__main__.py</code>:</p>
<pre><code>if __name__ == '__main__':
    print('/tmp/foo/__main__.py: main')
</code></pre>
<p><code>/tmp/foo/__init__.py</code>:</p>
<pre><code>print('/tmp/foo/__init__.py: init')
</code></pre>
<p>The command</p>
<pre><code>PYTHONPATH=$PYTHONPATH:/tmp/ python -m foo
</code></pre>
<p>Gives</p>
<pre><code>/tmp/foo/__init_.py: init
/tmp/foo/__main__.py: main
</code></pre>
<p><a href="https://docs.python.org/3/using/cmdline.html#envvar-PYTHONPATH" rel="nofollow noreferrer">More about PYTHONPATH</a></p>
</div>
