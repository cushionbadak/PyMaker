<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/operator.html#operator.itemgetter" rel="nofollow noreferrer"><code>operator.itemgetter</code></a> and supply column indices via a list. This works with negatives indices too:</p>
<pre><code>from io import StringIO
import csv
from operator import itemgetter

x = StringIO("""*\t10\t6cb2.aeee.0db7\tdynamic\t300\tF\tF\tPo11""")
cols = [1, 2, -1]

# replace x with open('file.txt', 'r', newline='')
with x as fin:
    reader = csv.reader(fin, delimiter='\t', quoting=csv.QUOTE_NONE)
    for row in reader:
        print(itemgetter(*cols)(row))

('10', '6cb2.aeee.0db7', 'Po11')
</code></pre>
</div>
