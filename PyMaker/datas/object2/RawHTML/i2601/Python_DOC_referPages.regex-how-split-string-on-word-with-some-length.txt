<div class="post-text" itemprop="text">
<p>Why quantify yourself? Let <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer">Counter()</a> do it:</p>
<pre><code>from collections import Counter
text = "tata, ohhhhh, tata, oh, tata, ohhhh"
c = Counter ( (len(w.strip()) for w in text.split(",") ))

print(c.most_common())
</code></pre>
<p>Output:</p>
<pre><code>[(4, 3), (2, 1), (5, 1), (6, 1)] # (word-length, count)
</code></pre>
<hr/>
<p>Using a <a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow noreferrer">defaultdict</a> will give you the words as well:</p>
<pre><code>d = defaultdict(list)
for w in (w.strip() for w in text.split(",")):
    d[len(w)].append(w)

print(d)
</code></pre>
<p>Output:</p>
<pre><code>defaultdict(&lt;type 'list'&gt;, 
            {2: ['oh'], 4: ['tata', 'tata', 'tata'], 5: ['ohhhh'], 6: ['ohhhhh']})
</code></pre>
<p>but you would have to get the <code>len()</code> the list afterwards.</p>
