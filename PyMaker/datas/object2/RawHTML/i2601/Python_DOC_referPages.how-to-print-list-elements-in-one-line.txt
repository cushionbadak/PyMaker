<div class="post-text" itemprop="text">
<p>You can unpack the list to <code>print</code> using <code>*</code> which will automatically split by a space</p>
<pre><code>print(*l)
</code></pre>
<p>if you want a comma, use the <code>sep=</code> argument</p>
<pre><code>print(*l, sep=', ')
</code></pre>
</div>

    print(l[i], end=' ')

<pre><code>print(*l, end=' ')
</code></pre>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/2/library/stdtypes.html#str.join" rel="nofollow noreferrer">join</a> and <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehensions</a> for that.
The only thing, items in a list should be strings, not integers</p>
<pre><code>import random

n = int(input(""))
l = []

for i in range(n):
    x = int(input())
    l.append(x)

not_sorted = True

while not_sorted:    
    x = random.randint(0, n-1)
    y = random.randint(0, n-1)
    while x == y:
        y = random.randint(0, n-1)
    if x &gt; y:
        if l[x] &lt; l[y]:
            (l[x], l[y]) = (l[y], l[x])
    if x &lt; y:
        if l[x] &gt; l[y]:
            (l[x], l[y]) = (l[y], l[x])

    for i in range(0, n-1):
        if l[i] &gt; l[i+1]:
            break
    else:
       not_sorted = False

print(', '.join([str(l[i]) for i in range(n)]))
</code></pre>
