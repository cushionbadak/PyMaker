<div class="post-text" itemprop="text">
<p>You can check if a number is negative by comparing it to zero. </p>
<pre><code>if field1 &lt; 0:
    field2 = -field1
else:
    field2 = 0.0
</code></pre>
<p>Or more succinctly:</p>
<pre><code>field2 = (-field1 if field1 &lt; 0 else 0.0)
</code></pre>
