<div class="post-text" itemprop="text">
<p>Easiest is to use <a href="https://docs.python.org/3/library/itertools.html#itertools.combinations" rel="nofollow noreferrer"><code>itertools.combinations</code></a> which takes care of avoiding repetitions for you:</p>
<pre><code>from itertools import combinations
a = [5, 7, 3, 1, 2]
for x, y in combinations(a, 2):
    print(x, y)

5 7
5 3
5 1
5 2
7 3
7 1
7 2
3 1
3 2
1 2
</code></pre>
<p>If you want to do it without library help, you can do the following, using <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a> and slicing:</p>
<pre><code>for i, x in enumerate(a):
    for y in a[i+1:]:  # combine only with elements after x (index i)
        print(x, y)
</code></pre>

<p>Yes you can.
Try following:</p>
<pre><code>a = [5,7,3,1,2]
for i in range(len(a)):
    for j in a[i:]:
        if(a[i]==j):
            continue
        else:
            print(a[i],j)
    print("")
</code></pre>
<p>This will give output:</p>
<pre><code>5 7
5 3
5 1
