<div class="post-text" itemprop="text">
<p>I have a pandas series with boolean entries.  I would like to get a list of indices where the values are <code>True</code>.</p>
<p>For example the input <code>pd.Series([True, False, True, True, False, False, False, True])</code></p>
<p>should yield the output <code>[0,2,3,7]</code>.</p>
<p>I can do it with a list comprehension, but is there something cleaner or faster?</p>
</div>
<div class="post-text" itemprop="text">
<h3>Using <a href="https://pandas.pydata.org/pandas-docs/stable/indexing.html#boolean-indexing" rel="nofollow noreferrer"><code>Boolean Indexing</code></a></h3>
<pre><code>&gt;&gt;&gt; s = pd.Series([True, False, True, True, False, False, False, True])
&gt;&gt;&gt; s[s].index
Int64Index([0, 2, 3, 7], dtype='int64')
</code></pre>
<p>If need a <code>np.array</code> object, get the <code>.values</code></p>
<pre><code>&gt;&gt;&gt; s[s].index.values
array([0, 2, 3, 7])
</code></pre>
<hr/>
<h3>Using <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.nonzero.html" rel="nofollow noreferrer"><code>np.nonzero</code></a></h3>
<pre><code>&gt;&gt;&gt; np.nonzero(s)
(array([0, 2, 3, 7]),)
</code></pre>
<hr/>
<h3>Using <a href="https://docs.scipy.org/doc/numpy-1.13.0/reference/generated/numpy.flatnonzero.html" rel="nofollow noreferrer"><code>np.flatnonzero</code></a></h3>
<pre><code>&gt;&gt;&gt; np.flatnonzero(s)
array([0, 2, 3, 7])
</code></pre>
<hr/>
<h3>Using <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.where.html" rel="nofollow noreferrer"><code>np.where</code></a></h3>
<pre><code>&gt;&gt;&gt; np.where(s)[0]
array([0, 2, 3, 7])
</code></pre>
<hr/>
<h3>Using <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.argwhere.html" rel="nofollow noreferrer"><code>np.argwhere</code></a></h3>
<pre><code>&gt;&gt;&gt; np.argwhere(s).ravel()
array([0, 2, 3, 7])
</code></pre>
<hr/>
<h3>Using <a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.Series.index.html#pandas.Series.index" rel="nofollow noreferrer"><code>pd.Series.index</code></a></h3>
<pre><code>&gt;&gt;&gt; s.index[s]
array([0, 2, 3, 7])
</code></pre>
<hr/>
<h3>Using python's built-in <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter</code></a></h3>
<pre><code>&gt;&gt;&gt; [*filter(s.get, s.index)]
[0, 2, 3, 7]
</code></pre>
<hr/>
<h3>Using <code>list comprehension</code></h3>
<pre><code>&gt;&gt;&gt; [i for i in s.index if s[I]]
[0, 2, 3, 7]
</code></pre>
