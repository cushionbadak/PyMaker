<p>I have a class in which I read a json into a dictionary and I want to assign in a loop self.variable based on the key-value pairs in the dictionary as in:</p>
<pre><code>class TestClass:
    def __init__(self, name):
        tables_metadata = json.loads(jsonfilepath)

        self.name = name
        for key, value in tables_metadata.items:
            self.key = value
</code></pre>
<p>How can I pass the key value of the dictionary as the self.name_of_the_variable?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#setattr" rel="nofollow noreferrer">setattr</a>:</p>
<pre><code>class TestClass:
    def __init__(self, name):
        tables_metadata = json.loads(jsonfilepath)

        self.name = name
        for key, value in tables_metadata.items:
            setattr(self, key, value)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>setattr(self, key, value)</code>.</p>
<p>However, I don't know your use case, but in most cases it's probably better to just store the actual dict as attribute.</p>
