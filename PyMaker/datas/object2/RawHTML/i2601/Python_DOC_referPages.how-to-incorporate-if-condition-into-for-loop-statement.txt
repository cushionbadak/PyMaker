<div class="post-text" itemprop="text">
<p>While your loop is fine, you can use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>new_data = [i for i in data if i[1] == 'hi']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a list comprehension and a filter to write in on one line. There is a good explanation on filters available at <a href="https://stackoverflow.com/questions/13638898/how-to-use-filter-map-and-reduce-in-python-3">this question</a>. Using filters you can make use of <a href="https://softwareengineering.stackexchange.com/questions/163985/why-is-the-concept-of-lazy-evaluation-useful/164002">lazy evaluation</a>, which benefits the execution time.</p>
<pre><code>data = [(1,'hi'),(2,'hello'),(3,'hi'),(4,'hi'),(5,'hello'),(6,'hello'), (7,'hi'),(8,'hello')]
new_data = list(filter(lambda i: i[1] == 'hi', data))
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[(1, 'hi'), (3, 'hi'), (4, 'hi'), (7, 'hi')]
</code></pre>
</div>
