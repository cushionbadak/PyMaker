<p>How to count values for each specific keys in a python dictionary which is inside a list</p>
<pre><code>my_dict = [{"a":1},{"b":4},{"a":7},{"a":30},{"b":78}]
</code></pre>
<p>expected result sholud be:</p>
<pre><code>new_dict=[{"a":3},{"b":2}]
</code></pre>

<div class="post-text" itemprop="text">
<p>Try <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer">Counter</a>-</p>
<pre><code>from collections import Counter
my_dict = [{"a":1},{"b":4},{"a":7},{"a":30},{"b":78}]
new_dict = [{k:v} for k,v in Counter([i.keys()[0] for i in my_dict]).items()]
</code></pre>
<p>For python3, <code>keys()</code> has changed, use <code>list(keys())</code></p>
<pre><code>new_dict = [{k:v} for k,v in Counter([list(i.keys())[0] for i in my_dict]).items()]
</code></pre>

<div class="post-text" itemprop="text">
<p>Using <code>collections.defaultdict</code></p>
<p><strong>Ex:</strong></p>
<pre><code>from collections import defaultdict

d = defaultdict(int)
l = [{"a":1},{"b":4},{"a":7},{"a":30},{"b":78}]
for i in l:
    val = list(i.items())[0]
    d[val[0]] += 1
print(d)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>defaultdict(&lt;type 'int'&gt;, {'a': 3, 'b': 2})
</code></pre>

<div class="post-text" itemprop="text">
<p>This is typically a job for <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>Counter</code></a> from <code>collections</code>. In this case, we have to modify the input a bit though since <code>dict</code>s are unhashable.</p>
<pre><code>from collections import Counter


my_dict = [{"a":1},{"b":4},{"a":7},{"a":30},{"b":78}]
new_dict = [{k[0]: v} for k, v in Counter(map(tuple, my_dict)).items()]
print(new_dict)  # -&gt; [{'a': 3}, {'b': 2}]
</code></pre>
