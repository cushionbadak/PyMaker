<p>You can do 2 ways. one is to use <code>%s</code> string replacement, the other is using <code>{}</code>. </p>
<pre><code>widgetName = input("Please enter the name you would like to use for your widget: ")

with open(os.path.join(widgetname, 'LockBackground.html'), 'w') as f:
    f.write("""
&lt;!DOCTYPE HTML&gt;
&lt;html&gt;
&lt;head&gt;
&lt;meta http-equiv="Content-Type" content="text/html; charset=utf-8"&gt;
&lt;title&gt;{}&lt;/title&gt;
&lt;style type="text/css"&gt;
@font-face {
    font-family: HelveticaNeue-UltraLigCond;
    src: url('fonts/font-file-goes-here');
}

    # More CSS goes here.
&lt;/style&gt;
&lt;script type="text/javascript" src="scripts/jquery-1.6.4.min.js"&gt;&lt;/script&gt;
# More javascript here.
&lt;/head&gt;
&lt;body&gt;

# Code goes here.

&lt;/body&gt;
&lt;/html&gt; """.format(widgetName))
</code></pre>
<p>Another way</p>
<pre><code>widgetName = input("Please enter the name you would like to use for your widget: ")
