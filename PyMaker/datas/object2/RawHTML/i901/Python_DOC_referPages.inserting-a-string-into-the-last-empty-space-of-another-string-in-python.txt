<div class="post-text" itemprop="text">
<p>I've got two separate strings stored in variable <code>mystring1</code> and <code>mystring2</code>, I'm looking for a way to take <code>mystring1</code> and insert it into the last empty space in <code>mystring2</code></p>
<pre><code>mystring1 = 'English French German'
mystring2 = 'Lorem Ipsum (Lorem Dolor Amet Elit)'
</code></pre>
<p><strong>Desired Output :</strong>
Lorem Ipsum (Lorem Dolor Amet <strong>English French German</strong> Elit)</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.rsplit" rel="nofollow noreferrer"><code>str.rsplit</code></a> the second string and <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>str.join</code></a> the appropriate tokens back together:</p>
<pre><code>t1, t2 = mystring2.rsplit(' ', 1)
' '.join((t1, mystring1, t2))
# 'Lorem Ipsum (Lorem Dolor Amet English French German Elit)'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://www.tutorialspoint.com/python/list_insert.htm" rel="nofollow noreferrer">python list insert</a> function. so your code will be something like that:</p>
<pre><code>mystring1 = 'English French German'
mystring2 = 'Lorem Ipsum (Lorem Dolor Amet Elit)'

# we will break mystring2 into a list
# insert mystring1 into it as a new element
# convert the list back into a string 

str2_parts = mystring2.split(' ')
# ['Lorem', 'Ipsum', '(Lorem', 'Dolor', 'Amet', 'Elit)']    

index = len(str2_parts) - 1
# 5

str2_parts.insert(index, mystring1)
# ['Lorem', 'Ipsum', '(Lorem', 'Dolor', 'Amet', 'English French German', 'Elit)']


print " ".join(str2_parts)
# Lorem Ipsum (Lorem Dolor Amet English French German Elit)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can this with this one liner:</p>
<pre><code>mystring2.rsplit(' ', 1)[0] + mystring1 + " " + mystring2.rsplit(' ', 1)[1]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The <code>rindex()</code> function returns the last index of a substring. Since python strings are immutable you can not simply index and replace. So forming a new string with the use of slices is the way to go.</p>
<pre class="lang-py prettyprint-override"><code>mystring2[:mystring2.rindex(' ') + 1] + mystring1 + mystring2[mystring2.rindex(' '):]

</code></pre>
</div>
