<div class="post-text" itemprop="text">
<p>You can express any character using escape codes. The <a href="https://en.wikipedia.org/wiki/Page_break#Form_feed" rel="nofollow noreferrer">0x0C Form Feed ASCII codepoint</a> can be expressed as <code>\f</code> or as <code>\x0c</code>:</p>
<pre><code>re.split('\f', text)
</code></pre>
<p>See the <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow noreferrer">Python string and byte literals syntax</a> for more details on what escape sequences Python supports when defining a string literal value.</p>
<p>Note: you don't need to use the regex module to split on straight-up character sequences, you can just as well use <code>str.split()</code> here:</p>
<pre><code>text.split('\f')
</code></pre>
