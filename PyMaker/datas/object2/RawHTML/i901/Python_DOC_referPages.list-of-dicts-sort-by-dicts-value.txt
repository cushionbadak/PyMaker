<p>I have a list of dicts like below:</p>
<pre><code>[
 {'init_time': datetime.time(14, 0), 'end_time': datetime.time(19, 0), 'value': 5.2}, 
 {'init_time': datetime.time(17, 0), 'end_time': datetime.time(18, 0), 'value': 6.5},
 {'init_time': datetime.time(15, 0), 'end_time': datetime.time(17, 0), 'value': 2.5}
]
</code></pre>
<p>How can I sort this list by the value of each dict's <code>init_time</code>? Usually with <code>sorted</code> one is able to sort a dict based on either key or value, but in my case, I'm not interested in sorting the dict but the list, using the dict's values.</p>
<p>For example, the output of the above would be:</p>
<pre><code>[
 {'init_time': datetime.time(14, 0), 'end_time': datetime.time(19, 0), 'value': 5.2}, 
 {'init_time': datetime.time(15, 0), 'end_time': datetime.time(17, 0), 'value': 2.5}
 {'init_time': datetime.time(17, 0), 'end_time': datetime.time(18, 0), 'value': 6.5},
]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Python's <code>sorted</code> function has a <code>key</code> parameter that will be used in comparisons,</p>
<pre><code>new_list = sorted(old_list, key=lambda item: item["init_time"])
</code></pre>
</div>
