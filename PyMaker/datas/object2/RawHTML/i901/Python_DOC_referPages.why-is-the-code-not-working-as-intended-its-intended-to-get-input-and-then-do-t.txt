<div class="post-text" itemprop="text">
<p>The issue is <a href="https://docs.python.org/3/reference/expressions.html#comparisons" rel="nofollow noreferrer">operator chaining</a>. <code>bird_guess in bird_names==False</code> is evaluated as:</p>
<pre><code>(bird_guess in bird_names) and (bird_names == False)
</code></pre>
<p>Since the second expression always evaluates to <code>False</code>, the condition is never satisfied. You can use parentheses to avoid the problem:</p>
<pre><code>(bird_guess in bird_names) == False
</code></pre>
<p>More idiomatic is to use the purpose-built <code>not in</code> operator:</p>
<pre><code>bird_guess not in bird_names
</code></pre>
