<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/glob.html" rel="nofollow noreferrer">glob</a>, <code>os.path</code> don't know how to process masks, but <code>glob.glob</code> works:</p>
<pre><code>splitnames = [os.path.basename(fpath) for fpath in glob.glob("./**/*.txt")]
splitnames 
Out:
['A_test1.pdf', 'B_test2.pdf']
</code></pre>
<p>Output of the glob:</p>
<pre><code>glob.glob("./**/*.txt")
Out:
['./some_folder/A_test1.pdf', './another_folder/B_test2.pdf']
</code></pre>
<p>Apply <code>os.path.basename</code> to this list and extract basenames, as it shown above.</p>
<p><strong>Edit</strong><br/>
If <code>xx</code> in the path <code>volume1/xx/</code> is just a folder name, not a mask, you should use following expression:</p>
<pre><code>splitnames = [os.path.basename(fpath) for fpath in glob.glob("./xx/*.txt")]
</code></pre>
<p>because <code>./**/</code> is expression which masks a folder name and it's unnecessary that case.</p>
