<p>What is the best way for me to create a dictionary from a string that key is each character and its upper case and value is its opposite case? I can use two line dictionary comprehensive but any better way?
ex:</p>
<blockquote>
<p>string: abc =&gt; {'a': 'A', 'b': 'B', 'c': 'C', 'C': 'c', 'B': 'b', 'A': 'a'}</p>
</blockquote>
<pre><code>string = 'abc'
d = { i:i.upper() for i in string}
d.update({ i.upper():i for i in string})
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#str.swapcase" rel="noreferrer">swapcase</a>:</p>
<pre><code>s = 'abc'

result = dict(zip(s + s.swapcase(), s.swapcase() + s))
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>{'C': 'c', 'b': 'B', 'B': 'b', 'a': 'A', 'A': 'a', 'c': 'C'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Another way to do the same</p>
<pre><code>import string
d=dict(zip(string.ascii_lowercase[:3],string.ascii_uppercase[:3]))
d.update(dict(zip(string.ascii_uppercase[:3],string.ascii_lowercase[:3])))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>IIUC: You can do this as follows,</p>
<pre><code>string_dict = {
    char: char.upper() if char == char.lower() else char.lower()
    for char in (string.lower() + string.upper())
}
</code></pre>
