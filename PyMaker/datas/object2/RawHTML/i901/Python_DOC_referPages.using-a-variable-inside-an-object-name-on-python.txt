<div class="post-text" itemprop="text">
<p>Probably the best way of doing this is to have a list of Qlabels then access them like this:</p>
<pre><code>self.labels[x].setText('label 1')
</code></pre>
<p>Can also use getattr like this:</p>
<pre><code>getattr(self, "label" + x).setText('label 1')
</code></pre>
<p>Otherwise you could use eval however that isn't really recommended for security issues and makes the code more confusing:</p>
<pre><code>eval("self.label(" + x + ").setText('label 2')")
</code></pre>
