<p>You can also try like this.</p>
<blockquote>
<p><code>re.I</code> is for case insensitive match. You can check <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">https://docs.python.org/3/library/re.html</a> for more details.</p>
</blockquote>
<pre><code>import re

s = "valoirfinalieMONT:23maning"
s2 = "montdj34meaing"
s3 = "thisisthelastmontitwillwork98help"

m = re.match(r".*(?P&lt;name&gt;mont)\D+(?P&lt;number&gt;\d+).*", s, re.I)
print(m.group(1)) # MONT
print(m.group(2)) # 23

# Same as above (2nd way)
print(m.group('name'));
print(m.group('number'))

m2 = re.match(r".*(?P&lt;name&gt;mont)\D+(?P&lt;number&gt;\d+).*", s2, re.I)
print(m2.group(1)) # mont
print(m2.group(2)) # 34

m3 = re.match(r".*(?P&lt;name&gt;mont)\D+(?P&lt;number&gt;\d+).*", s3, re.I)
print(m3.group(1)) # mont
print(m3.group(2)) # 98
</code></pre>

<p>try this:</p>
<pre><code>import re

s='valoirfinalieMONT:23maning '
print(re.findall('(mont)\D*(\d*)', s, re.IGNORECASE))
</code></pre>
<p>the regex will capture 'mont' any number of non digit(\D) characters and then any number of digits(\d)</p>
<p>the ignore case is added so mont and MONT and MoNt and such will also be captured</p>
