<div class="post-text" itemprop="text">
<p>I have three list, I want to convert into a specific dict</p>
<pre><code>A = ["a", "b"]
B = ["10", "20"]
C = ["key1", "key2"]
</code></pre>
<p>I want a dict like that</p>
<pre><code>"key1": {
  "name": "a",
  "age": 10
},
"key2": {
    "name": "b",
    "age": 20
  }
</code></pre>
<p>I try different way by different step, but I don't get this dict</p>
<pre><code>for key in A:
  dict_A["name"].append(key)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using a dictionary comprehension:</p>
<pre><code>res = {key: {'name': name, 'age': age} for key, name, age in zip(C, A, B)}
</code></pre>
<p>This gives:</p>
<pre><code>{'key1': {'age': '10', 'name': 'a'}, 'key2': {'age': '20', 'name': 'b'}}
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html#zip" rel="noreferrer"><code>zip</code></a> allows you to aggregate elements from each iterable by index.</p>
</div>
<div class="post-text" itemprop="text">
<h2>If you don't want to use <code>zip</code> then you can use <code>enumerate</code> as given below:</h2>
<pre><code>print ({v:{"name": A[k],"age": B[k]} for k, v in enumerate(C)})
</code></pre>
</div>
