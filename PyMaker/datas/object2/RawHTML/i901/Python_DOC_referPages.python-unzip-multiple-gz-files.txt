<p>I have compressed a file into several chunks using 7zip:</p>
<p>HAVE:</p>
<pre><code>foo.txt.gz.001
foo.txt.gz.002
foo.txt.gz.003
foo.txt.gz.004
foo.txt.gz.005
</code></pre>
<p>WANT:</p>
<pre><code>foo.txt
</code></pre>
<p>How do I unzip and combine these chunks to get a single file using python?</p>
</div>
<div class="post-text" itemprop="text">
<p>First, get the list of all files.</p>
<pre><code>files = ['/path/to/foo.txt.gz.001', '/path/to/foo.txt.gz.002', '/path/to/foo.txt.gz.003']
</code></pre>
<p>Then iterate over each file and append to a result file.</p>
<pre><code>with open('./result.gz', 'ab') as result:  # append in binary mode
    for f in files:
        with open(f, 'rb') as tmpf:        # open in binary mode also
            result.write(tmpf.read())
</code></pre>
<p>Then extract is using <a href="https://docs.python.org/3/library/zipfile.html" rel="nofollow noreferrer">zipfile</a> lib. You could use <a href="https://docs.python.org/3/library/tempfile.html" rel="nofollow noreferrer">tempfile</a> to avoid handle with temporary zip file.</p>
</div>
<div class="post-text" itemprop="text">
<p>First you must extract all the zip files sequentially:</p>
<pre><code>import zipfile

paths = ["path_to_1", "path_to_2" ]
extract_paths = ["path_to_extract1", "path_to_extrac2"]

for i in range(0, paths):
    zip_ref = zipfile.ZipFile(paths[i], 'r')
    zip_ref.extractall(extract_paths[i])
    zip_ref.close()
</code></pre>
<p>Next you can go to the extracted location and <code>read()</code> individual files with <code>open</code> into a <code>string</code>. Concatenate those strings and save to <code>foo.txt</code>.</p>
</div>
