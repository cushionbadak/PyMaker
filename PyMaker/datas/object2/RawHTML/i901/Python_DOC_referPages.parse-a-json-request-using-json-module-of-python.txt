<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python-requests.org/en/master/user/quickstart/#make-a-request" rel="nofollow noreferrer"><code>requests.get()</code></a> to make a GET request to your URL and get the JSON string:</p>
<pre><code>from requests import get

URL = "https://api.twitter.com/1.1/favorites/list.json?count=2&amp;screen_name=episod"

res = get(URL).text

print(res)
# {"errors":[{"code":215,"message":"Bad Authentication data."}]}

print(type(res))
# &lt;class 'str'&gt;
</code></pre>
<p>Then you can use <a href="https://docs.python.org/3/library/json.html#json.loads" rel="nofollow noreferrer"><code>json.loads()</code></a> to convert the JSON string into a python dictionary:</p>
<pre><code>from json import loads

json_dict = loads(res)

print(json_dict)
# {'errors': [{'code': 215, 'message': 'Bad Authentication data.'}]}

print(type(json_dict))
# &lt;class 'dict'&gt;
</code></pre>
<p>Which you can iterate and parse the information you want. </p>
