<div class="post-text" itemprop="text">
<p>If you don't need the binary mode, you can use this :</p>
<pre><code>with open(File_Name, 'w') as csvfile:
   filewriter = csv.writer(csvfile,delimiter=',')
   filewriter.writerow(['Paul','Mary'])
</code></pre>
<p>Check the table <a href="https://docs.python.org/3.6/library/functions.html?highlight=open#open" rel="nofollow noreferrer">here</a> to see details about the different modes available.</p>

<div class="post-text" itemprop="text">
<p>The <code>csvwriter</code> instance shouldn't be reassigned to. It should be used to write the rows.</p>
<p>In python 3, you should open the file in write only mode with <code>newline=''</code> per <a href="https://docs.python.org/3/library/csv.html#csv.writer" rel="nofollow noreferrer">documentation</a>.</p>
<pre><code>import csv

filename = 'Test.csv'

with open(filename, 'w', newline='') as csvfile:
   file_writer = csv.writer(csvfile,delimiter=',')
   file_writer.writerow(['Paul','Mary'])
</code></pre>
</div>

<p>You have to change from 'wb' to 'w' and file name should be in quotation("Test.csv" or 'Test.csv') see below</p>
<pre><code>import csv
File_Name = "Test.csv"
with open(File_Name, 'w') as csvfile:
    filewriter = csv.writer(csvfile,delimiter=',')
    filewriter.writerow(['Paul','Mary'])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The thing is that you are opened file in a binary mode</p>
<pre><code>with open(File_Name, 'wb') as csvfile:
</code></pre>
<p>This means that all data read from the file is returned as bytes objects, not str.
If you don't need it, try:</p>
<pre><code>with open(File_Name, 'w') as csvfile:
</code></pre>
<p>Also you should use <code>writerow</code> method of the <code>csvwriter</code> instead of reassign it:</p>
<pre><code>file_writer.writerow(['Paul','Mary'])
</code></pre>
</div>
