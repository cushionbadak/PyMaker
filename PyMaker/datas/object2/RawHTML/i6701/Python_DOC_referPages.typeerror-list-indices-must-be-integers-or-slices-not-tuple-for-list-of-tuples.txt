<div class="post-text" itemprop="text">
<p>Python's <code>for</code> loop is a <a href="https://en.wikipedia.org/wiki/Foreach_loop" rel="nofollow noreferrer"><em>Foreach</em> construct</a>; you iterate over the <em>elements of the list</em>, not an index.</p>
<p>So <code>n</code> is one of the tuples from <code>lst</code>, not an index. Use it directly:</p>
<pre><code>for n in lst:
    TimeAxis.append(n[0])
</code></pre>
<p>You could simplify your code by using a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>TimeAxis = [tup[0] for tup in lst]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>for n in lst:
    TimeAxis.append(n[0])
</code></pre>
<p>When you iterate on list, its fetching one by one element, so value of <code>n</code> is your every tuple, just access <code>0th</code> index of it and append</p>

<p>let me help you understand, what has gone wrong in your code:</p>
<pre><code>lst = [(29208, 8, 8, 8), (29209, 8, 8, 8), (29210, 8, 8, 8), (29211, 8, 8, 8)]
for n in lst:  # ns in lst -&gt; (29208, 8, 8, 8), ... , (29211, 8, 8, 8)
    TimeAxis.append(lst[n][0]) # 1st iter: lst[(29208, 8, 8, 8)][0]
</code></pre>
<p>Hence the <code>TypeError</code>. What you probably wanted to do is:</p>
<pre><code>for i in range(len(lst)):  # is in range(len(lst)) -&gt; 0,1,2,3 
    TimeAxis.append(lst[n][0])  # 1st iter: lst[0][0]
</code></pre>
<p>Even though this will work, there is no need for that. There are more pythonic way, which can be found in <a href="https://stackoverflow.com/a/46275767/8617309">@Martijn Pieters answer</a>.</p>
