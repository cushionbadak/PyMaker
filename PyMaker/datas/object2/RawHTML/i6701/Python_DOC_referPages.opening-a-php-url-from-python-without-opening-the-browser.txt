<pre><code>import webbrowser

</code></pre>

<div class="post-text" itemprop="text">
<p>You can use the requests module to perform an http request on that url.  You just just need to install the package with pip.</p>
<p>Install the package with pip</p>
<pre><code>pip install requests
</code></pre>
<p>Sample calling a get request</p>
<pre><code>import requests

response = requests.get("http://testing/test.php")
</code></pre>
