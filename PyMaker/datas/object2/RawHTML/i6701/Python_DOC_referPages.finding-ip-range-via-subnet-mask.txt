<p>Let's say my local ip is <code>10.212.97.23</code> and my Subnet Mask is <code>255.255.254.0</code></p>
<p>According to that, I'm in the same subnet with <code>10.212.96.*</code> and CIDR would be <code>10.212.96.0/23</code></p>
<p>So my question is: Is there any python library which takes local ip and subnet mask and calculates the CIDR value? I checked netaddr library but couldn't find this functionality</p>
</div>
<div class="post-text" itemprop="text">
<p>The basic count in CIDR can be computed manually using the below function without using any libraries:</p>
<pre><code>&gt;&gt;&gt; netmask = "255.255.254.0"
&gt;&gt;&gt; sum([bin(int(x)).count("1") for x in netmask.split(".")])
23
</code></pre>
<p>Let me know if it is helpful.</p>

<div class="post-text" itemprop="text">
<p>If you want to do it with a module, the ipaddress module can help:</p>
<pre><code>‚ùØ python3
&gt;&gt;&gt; import ipaddress
&gt;&gt;&gt; ipaddress.IPv4Network("10.212.97.23/255.255.254.0", strict=False).prefixlen
23
</code></pre>
