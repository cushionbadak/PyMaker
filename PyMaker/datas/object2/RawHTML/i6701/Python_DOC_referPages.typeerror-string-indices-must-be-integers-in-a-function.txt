<div class="post-text" itemprop="text">
<p>String slicing is done with a <code>:</code>, not a <code>,</code>.</p>
<p>Try the following:</p>
<pre><code>def divideEnTam(cadena, num):
  lista=[]
  for x in range (0, len(cadena)//num):
    lista.append(cadena[(x-num):x])  # Fixed slicing
  print (lista)

divideEnTam("Paracaidas", 2)
</code></pre>

<p>You're not paying attention to what python is telling you.</p>
<p>When you run your program, you get a traceback:</p>
<pre><code>Traceback (most recent call last):
  File "so17405138.py", line 7, in &lt;module&gt;

TypeError: string indices must be integers
</code></pre>
<p>The last line in the traceback tells you what the problem is, namely that string indices must be integers. Then as you read upwards you'll see that the problem is on line 4:</p>
<pre><code>lista.append(cadena[(x-num), x])
</code></pre>
<p>If you work your way through the expression, you might try something like this first:</p>
<pre><code>&gt;&gt;&gt; cadena = "Paracaidas"
&gt;&gt;&gt; cadena[(4-3), 4]
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: string indices must be integers
</code></pre>
<p>that would tell you that there's something wrong with what's inside the <code>[]</code>. You know that both the result of 4-3 and 4 are integers, so the next thing that you can try is check what that expression is:</p>
<pre><code>&gt;&gt;&gt; x = (4-3), 4
&gt;&gt;&gt; print(type(x), repr(x))
&lt;class 'tuple'&gt; (1, 4)
</code></pre>
<p>Ah hah! So it turns out that <code>(4-3), 4</code> is a <em>tuple</em>. If you read more <a href="https://docs.python.org/3/tutorial/datastructures.html#tuples-and-sequences" rel="nofollow noreferrer">about tuples</a> you'll see that they're actually not defined by <em>parenthesis</em> but by <em>commas</em>. What you probably wanted was a <em>slice</em>, which you get by replacing <code>,</code> with <code>:</code>.</p>
<p>However, if you try that:</p>
<pre><code>def divideEnTam(cadena, num):
  lista=[]
  for x in range (0, len(cadena)//num):
    lista.append(cadena[(x-num): x])
  print (lista)

divideEnTam("Paracaidas", 2)
</code></pre>
<p>You'll see that you've got something wrong:</p>
<pre><code>['', '', 'Pa', 'ar', 'ra']
</code></pre>
<p>If you think about it for a moment, and adjust your statement to this:</p>
<pre><code>for x in range (0, len(cadena)//num):
    print(x-num, x)
</code></pre>
<p>You'll see this output:</p>
<pre><code>-2 0

<p>If you notice this starts out at <code>-2</code>. It's not the output that you're really looking for. You want something more like this:</p>

<p>Do you know how to do <em>that</em> with a <a href="https://docs.python.org/3.5/library/functions.html#func-range" rel="nofollow noreferrer">range</a>? </p>
