<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>import os
from django.conf import settings

# ... your code ...
# ...
 plt.savefig(os.path.join(settings.BASE_DIR, 'static/img/imagen.png'))
</code></pre>
<hr/>
<p><strong>Note:</strong> From your code, it appears that while you're saving the image you're not giving it a unique name. So, if you create a new image, it will replace the older image. If that is the desired behaviour, ignore this. If not, you can use <a href="https://docs.python.org/3/library/uuid.html#uuid.uuid4" rel="nofollow noreferrer"><code>uuid.uuid4</code></a> to generate unique names for your images. </p>
