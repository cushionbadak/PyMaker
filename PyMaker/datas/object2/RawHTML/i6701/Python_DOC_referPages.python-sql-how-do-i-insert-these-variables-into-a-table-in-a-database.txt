<div class="post-text" itemprop="text">
<p>SQLite does not support the %s syntax as a placeholder (<a href="https://docs.python.org/3/library/sqlite3.html" rel="nofollow noreferrer">https://docs.python.org/3/library/sqlite3.html</a>). Use question marks instead, and don't forget to insert a comma after the SQL string:</p>
<pre><code>c.execute('''INSERT INTO userAcc(name, genre, artist) VALUES(?,?,?)''', (name1, genre1, artist1))
</code></pre>
