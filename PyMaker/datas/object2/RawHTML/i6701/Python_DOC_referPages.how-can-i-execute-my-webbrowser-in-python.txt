<p>Python version 3.6, OS MS Windows 7</p>

<div class="post-text" itemprop="text">
<p>Use <strong>webbrowser</strong> module in python.</p>
<pre><code>import webbrowser as wb
wb.get('chrome %s').open_new_tab('http://www.google.com')
</code></pre>
<p>You can open your <strong>default</strong> web browser in python this way by writing this command in <strong>cmd</strong>:</p>
<pre><code>python -mwebbrowser -n URL
</code></pre>
<p><strong>-n</strong> : new window</p>
<p>Look at the <a href="https://docs.python.org/3/library/webbrowser.html" rel="nofollow noreferrer"><strong><code>webbrowser</code></strong></a> documentation to see other options as well.</p>
</div>
