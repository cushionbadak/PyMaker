<div class="post-text" itemprop="text">
<p>Python has <code>str.isalnum()</code>, <code>str.isdigit()</code>, <code>str.isupper()</code>, <code>str.islower()</code>, <code>str.isalpha()</code>, but does it have any built-in string validation checking for special characters or punctuation? (i.e. !@#$%^&amp;*().,;'": ect) </p>
</div>
<div class="post-text" itemprop="text">
<p>The standard <code>string</code> module provides <a href="https://docs.python.org/3/library/string.html#string.punctuation" rel="nofollow noreferrer"><code>string.punctuation</code></a>, which contains punctuation characters. You can test for membership in that:</p>
<pre><code>def ispunct(ch):
    return ch in string.punctuation
</code></pre>
</div>
<div class="post-text" itemprop="text">

<pre><code>import string
&gt;&gt;&gt; print string.punctuation
!"#$%&amp;'()*+,-./:;&lt;=&gt;?@[\]^_`{|}~
</code></pre>
