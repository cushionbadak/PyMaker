<div class="post-text" itemprop="text">
<p>You can also just use a <a href="https://docs.python.org/3/library/csv.html#csv.reader" rel="nofollow noreferrer"><code>csv.reader</code></a>:</p>
<pre><code>import csv

list_a = []
list_b = []

with open('file.csv', newline='') as csv_file:
    reader = csv.reader(csv_file, delimiter='.')
    next(reader, None)  # Skip the header.
    for a, b in reader:
        list_a.append(float(a))
        list_b.append(float(b))
</code></pre>
<hr/>
<p>Or with the <code>zip</code> trick:</p>
<pre><code>with open('file.csv', newline='') as csv_file:
    reader = csv.reader(csv_file, delimiter='.')
    next(reader, None)  # Skip the header.
    list_a, list_b = zip(*(map(float, row) for row in reader))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>a = [float(item.split('.')[0]) for item in f]</code> reads everything from the file. There is nothing left for the second list comprehension. You must close the file and open it again, e.g.:</p>
<pre><code>with open('file.csv', 'r') as f:
    a = [float(item.split('.')[0]) for item in f] 
with open('file.csv', 'r') as f:
    b = [float(item.split('.')[1]) for item in f]
</code></pre>
<p>But it is even better to read the file only once and process both a's and b's in one loop:</p>
<pre><code>with open('file.csv', 'r') as f:
    a, b = zip(*[map(float, item.split('.')) for item in f])
</code></pre>
</div>
