<div class="post-text" itemprop="text">
<p>You have to import the class using:</p>
<pre><code>from .class1 import G
</code></pre>
<p>Do not forget the dot. When you build a package you have to tell where your class is written using this relative path. This is called intra-package references.</p>
<p>See the corresponding <a href="https://docs.python.org/3/tutorial/modules.html#intra-package-references" rel="nofollow noreferrer">doc</a>.</p>
