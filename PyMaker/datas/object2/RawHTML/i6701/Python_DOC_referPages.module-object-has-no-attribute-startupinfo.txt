<div class="post-text" itemprop="text">
<p>The <code>STARTINFO</code> class is only available on Windows. Clearly stated in the <a href="https://docs.python.org/3/library/subprocess.html#windows-popen-helpers" rel="nofollow noreferrer">docs</a>:</p>
<blockquote>
<p>The <code>STARTUPINFO</code> class and following constants are only available on
  Windows.</p>
</blockquote>
