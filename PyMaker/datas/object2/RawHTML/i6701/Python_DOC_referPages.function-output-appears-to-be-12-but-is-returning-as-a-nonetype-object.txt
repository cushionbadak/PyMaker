<span class="comment-copy">Might also want to check out <a href="https://docs.python.org/3/library/shutil.html#shutil.disk_usage" rel="nofollow noreferrer">shutil.disk_usage</a></span>
<span class="comment-copy">if I add <code>return</code> to it, it still outputs 12.</span>
<span class="comment-copy"><code>os.system(check_disk_space_cmd)</code> returns 12, but <i>your function</i> doesn't.</span>
<span class="comment-copy">Python isn't one of those languages where functions automatically return the value of the last expression they evaluate. You have to <code>return</code> things explicitly.</span>
