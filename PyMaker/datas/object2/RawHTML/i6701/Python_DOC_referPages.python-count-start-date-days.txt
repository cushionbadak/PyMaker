<div class="post-text" itemprop="text">
<p>For adding days into date, you need to user <a href="https://docs.python.org/2/library/datetime.html#datetime.timedelta" rel="nofollow noreferrer">datetime.timedelta</a></p>
<pre><code>start=str(input("type start date (DD.MM.YYYY)"))
end=int(input("how many days from it?"))  

date = datetime.strptime(start, "%d.%m.%Y")
modified_date = date + timedelta(days=end)
print(datetime.strftime(modified_date, "%d.%m.%Y"))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It will be easier to convert to datetime using <code>datetime.datetime.strptime</code> and for the part about adding days just use <code>datetime.timedelta</code>.</p>
<p>Below is a small snippet on how to use it:</p>
<pre><code>import datetime
start = "21.10.2000"
end = 8
dateStart = datetime.datetime.strptime(start, "%d.%m.%Y")
dateEnd = dateStart + datetime.timedelta(days=end)
dateEnd.date()  # to get the date format of the endDate
</code></pre>
<p>If you have any doubts please look at the documentation <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer">python3</a>/<a href="https://docs.python.org/2/library/datetime.html" rel="nofollow noreferrer">python2</a>.</p>

<pre><code>&gt;&gt;&gt; from datetime import datetime
