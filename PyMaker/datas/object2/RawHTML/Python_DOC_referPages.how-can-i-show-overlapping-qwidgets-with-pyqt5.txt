<p>I have a toplevel widget with a layout set and 2 other widgets added to that layout, when I color them all I only see the toplevel widget color but Id like to see the children widgets on top. This is what I have attempted but it just displays a blue QWidget, I am expecting red and green widgets one on top of the other</p>
<pre><code>def set_color(widget, color):
    p = widget.palette()
    p.setColor(widget.backgroundRole(), color)
    widget.setPalette(p)

from PyQt5.QtCore import Qt
from PyQt5.QtWidgets import QApplication, QWidget,QVBoxLayout

win.show()

<div class="post-text" itemprop="text">
<p>The widgets are visible but the background color you use is the same as the parent widget, so that they are applied correctly you must enable the <a href="https://doc.qt.io/qt-5/qwidget.html#autoFillBackground-prop" rel="nofollow noreferrer">autoFillBackground</a> property:</p>
<pre><code>mod_group.setAutoFillBackground(True)
audio_group.setAutoFillBackground(True)
</code></pre>
