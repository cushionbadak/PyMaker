<div class="post-text" itemprop="text">
<p>I would like to use <code>block_diag</code> to make a block diagonal matrix
of multiple matrices.</p>
<p>Let's say I have matrices <code>A</code>, <code>B</code>, and <code>C</code>. I can simply write:</p>
<pre><code>block_diag(A, B, C)
</code></pre>
<p>However, I do wish to have more matrices later on. </p>
<p>How do I combine <code>A</code>, <code>B</code>, and <code>C</code> into a list of matrices <code>X</code> so that 
I can write:</p>
<pre><code>block_diag(X)
</code></pre>
<p>I tried using <code>[A, B, C]</code> but that doesn't work. </p>
</div>
<div class="post-text" itemprop="text">
<p>Use the star operator (<code>*</code>), which spreads out the contents of an iterable as positional function arguments. (I have chosen to use an immutable <a href="https://docs.python.org/3/library/stdtypes.html#tuples" rel="nofollow noreferrer">tuple</a>, but the list <code>[A, B, C]</code> would work equally well.)</p>
<pre><code>X = (A, B, C)

print(block_diag(*X))
</code></pre>
</div>
