<div class="post-text" itemprop="text">
<p>Consider this example:</p>
<pre><code>def g(a,b):
 print(a+b)
</code></pre>
<p>is kind of equivalent to:</p>
<pre><code>def g(a,b):
 print(a+b)
 return None
</code></pre>
<p>So if you call <code>print(g(1,2))</code> This will perform the action of the inner function, but will also output its return value, <code>None</code> in this case</p>
<p>Generally, if you place a function call inside another function call you get <code>None</code>, if the function does not return anything. For example, <code>function(function())</code> outputs <code>None</code>.</p>
<p>So does <code>input(print())</code> or <code>print(print())</code>. Be careful with that.</p>
