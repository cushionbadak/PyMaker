<p>I came across this problem in my project in which I have lists which append from a life to a list. Please see below.</p>
<pre><code>Log_info = [ "['carl@gmail.com', 'carl120', 'Carl']", "['brandi.love@gmail.com', 'brandih0t', 'Brandi']" ]
</code></pre>
<p>Above is the list that have been pull from a file (TXT)</p>
<p>Below is what I want it to look like: I want to remove <code>""</code> so they are sub list in a list, but I can't seem to find how.</p>
<pre><code>Log_info = [ ['carl@gmail.com', 'carl120', 'Carl'], ['brandi.love@gmail.com', 'brandih0t', 'Brandi'] ]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I'd use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="noreferrer"><code>ast.literal_eval</code></a></p>
<pre><code>&gt;&gt;&gt; from ast import literal_eval
&gt;&gt;&gt; [literal_eval(i) for i in Log_info]
[['carl@gmail.com', 'carl120', 'Carl'],
 ['brandi.love@gmail.com', 'brandih0t', 'Brandi']]
</code></pre>
