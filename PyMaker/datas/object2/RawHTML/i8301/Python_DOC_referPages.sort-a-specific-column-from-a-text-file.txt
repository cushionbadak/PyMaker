<p>I am trying to sort a column from a <code>.txt</code> file. I am going do a point-register system where I am going to save the name of the player and his three laps.</p>
<p>I am saving my values in the text-file like so:</p>
<pre><code> 1. name;lap_1;lap_2;lap_3;
 2. name;lap_1;lap_2;lap_3;
 3. name;lap_1;lap_2;lap_3;
</code></pre>
<p>In my code I write them to the file like so:</p>
<pre><code>for result in results:
    my_file.write("{}:{}:{}:{}:{}:{};\n".format(result["name"],
                                                result["lap1"],
                                                result["lap2"],
                                                result["lap3"],
                                                result["total"],
                                                result["average"]))
</code></pre>
<p>How do I sort each column, for example <code>"name"</code>? And how do I print it out?</p>

<div class="post-text" itemprop="text">
<p>first of all, as @sgrg suggested, use CSV-file format, e. g. we can write simply with</p>
<pre><code>import csv


def write_results(results, fields_names):
    # or use mode="a" if you want to append
    with open("my_file.csv", mode="w", newline="") as my_file:
        csv_writer = csv.DictWriter(my_file, fieldnames=fields_names, delimiter=";")
        # remember: you don"t need to add headers in "append" mode
        csv_writer.writeheader()
        for result in results:
            csv_writer.writerow(result)
</code></pre>
<p>then read with    </p>
<pre><code>def read_results(fields_names):
    with open("my_file.csv", mode="r") as my_file:
        # ignoring headers
        next(my_file)
        csv_reader = csv.DictReader(my_file, fieldnames=fields_names, delimiter=";")
        return list(csv_reader)
</code></pre>
<p>sorting of results by name can be done with</p>
<pre><code>sorted_results = sorted(results, key=lambda result: result["name"])
</code></pre>
<p>usage</p>
<pre><code>fields_names = ["name", "lap1", "lap2", "lap3", "total", "avarage"]
results_tuples = [("Luke", "lap1_value", "lap2_value", "lap3_value", 100, 96.3),
                  ("Stephen", "lap1_value", "lap2_value", "lap3_value", 100, 96.3),
                  ("Adrian", "lap1_value", "lap2_value", "lap3_value", 100, 96.3)]
results = [dict(zip(fields_names, result_tuple)) for result_tuple in results_tuples]
write_results(results,
              fields_names=fields_names)
results = read_results(fields_names)
sorted_results = sorted(results, key=lambda result: result["name"])
</code></pre>
