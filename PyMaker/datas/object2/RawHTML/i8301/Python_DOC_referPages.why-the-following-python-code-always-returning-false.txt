<div class="post-text" itemprop="text">
<p>Your lines end in a <em>line separator</em>, which you did not remove. Strip the line to remove whitespace from the start and end:</p>
<pre><code>key, value = line.strip().split("=")
</code></pre>
<p>A line separator (<code>\n</code>) is not a digit:</p>
<pre><code>&gt;&gt;&gt; '100\n'.isdigit()
False
&gt;&gt;&gt; '100'.isdigit()
True
</code></pre>
<p>Rather than build your own parser, consider using the <a href="https://docs.python.org/3/library/configparser.html" rel="noreferrer"><code>configparser</code> module</a> from the standard library. It fully supports the INI format (including using <code>;</code> as a comment).</p>
