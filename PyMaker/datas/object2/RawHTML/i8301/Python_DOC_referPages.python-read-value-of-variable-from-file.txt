<div class="post-text" itemprop="text">
<p>In bash, I have a file that stores my passwords in variable format.</p>
<p>e.g.</p>
<pre><code>cat file.passwd
password1=EncryptedPassword1
password2=EncryptedPassword2
</code></pre>
<p>Now if I want to use the value of <code>password1</code>, this is all that I need to do in bash.</p>
<pre><code>grep password1 file.passwd  | cut -d'=' -f2
</code></pre>
<p>I am looking for an alternative for this in python. Is there any library that gives functionality to simply extract the value or do we have to do it manually 
 like below?</p>
<pre><code>with open(file, 'r') as input:
         for line in input:
             if 'password1' in line:
                 re.findall(r'=(\w+)', line) 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Read the file and add the check statement: </p>
<pre><code>if line.startswith("password1"):
    print re.findall(r'=(\w+)',line)
</code></pre>
<p><strong>Code</strong>:</p>
<pre><code>import re
with open(file,"r") as input:
    lines = input.readlines()
    for line in lines:
        if line.startswith("password1"):
            print re.findall(r'=(\w+)',line)
</code></pre>

<div class="post-text" itemprop="text">
<p>There's nothing wrong with what you've written. If you want to play code golf:</p>
<pre><code>line = next(line for line in open(file, 'r') if 'password1' in line)
</code></pre>
