<pre><code>for num in range(1,13196):

<div class="post-text" itemprop="text">
<p>You could use the <a href="https://docs.python.org/2/reference/expressions.html#binary-arithmetic-operations" rel="nofollow noreferrer">modulo operator</a> to check if a number is divided evenly, and only set x equal to it then. E.g.:</p>
<pre><code>for num in range(1,13196):
    if 13195 % num == 0:
        x = int(13195/num)
        print(x)
</code></pre>

<p>I've tried is.integer and but keep getting syntax errors.</p>
</blockquote>
<p>The method for checking if a float is an integer is called <code>is_integer()</code> not <code>is.integer()</code>, so alternatively, you can do this:</p>
<pre><code>for num in range(1,13196):
    x = 13195/num 
    if x.is_integer():
        print(num)
</code></pre>
<p>Or by adding the brackets:</p>
<pre><code>for num in range(1,13196):
    if (13195/num).is_integer():
        print(num)
</code></pre>
