<div class="post-text" itemprop="text">
<p>if you just want to format a datetime, you can do one of those:</p>
<pre><code>from datetime import datetime

dt = datetime.now()

print("{0.day} - {0.month} - {0.hour}:{0.minute}".format(dt)) # 25 - 4 - 16:49
print('{:%d - %m - %H:%M}'.format(dt))                        # 25 - 04 - 16:49
</code></pre>
<p>the second one uses the <a href="https://docs.python.org/3/library/string.html#formatspec" rel="nofollow noreferrer">format mini-language</a> with the <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">format specifiers for <code>datetime</code></a>.</p>
<p>to get attributes dynamically, use <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer"><code>getattr</code></a>:</p>
<pre><code>for item in ['day', 'month', 'hour', 'minute']:
    print(getattr(dt, item))
</code></pre>
