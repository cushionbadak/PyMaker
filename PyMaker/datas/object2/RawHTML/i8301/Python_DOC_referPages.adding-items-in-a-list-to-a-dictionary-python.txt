<p>Try this:</p>
<pre><code>dct = {}
x_list = (blue, orange, purple, green, yellow, green, blue)
y_list = (1, 2, 5, 2, 4, 3, 8)
for i in range(len(x_list)):
    if x_list[i] in dct.keys():
        dct[x_list[i]] += y_list[i]
    else:
        dct[x_list[i]] = y_list[i]

print dct
</code></pre>

<div class="post-text" itemprop="text">
<p>I would use a <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>Counter</code></a> here:</p>
<pre><code>from collections import Counter

c = Counter()
for k, v in zip(x_list, y_list):
    c[k] += v
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Short solution using <code>enumerate</code> function:</p>
<pre><code>x_list = ['blue', 'orange', 'purple', 'green', 'yellow', 'green', 'blue']
y_list = [1, 2, 5, 2, 4, 3, 8]
result = {}

for i, v in enumerate(x_list):
    result[v] =  y_list[i] if not result.get(v) else result[v] + y_list[i]

print(result)
</code></pre>
<p>The output:</p>
<pre><code>{'yellow': 4, 'orange': 2, 'blue': 9, 'green': 5, 'purple': 5}
</code></pre>
</div>

<p>Try this code:</p>
<pre><code>list_x =["apple", "mango", "orange", "apple","mango"]
list_y = [10,5,25,10,15]
total_dict = {}
for k, v in zip(list_x, list_y):
    total_dict[k] = total_dict.get(k,0)+v 
</code></pre>
<p>The value of <code>total_dict</code> ends up as:</p>
<pre><code>{'orange': 25, 'mango': 20, 'apple': 20}
</code></pre>
