<div class="post-text" itemprop="text">
<p>I have model with TextField that contains html. <br/>Let's say I have a row that contains <code>&lt;a href="http://googke.ru"&gt;google&lt;/a&gt;</code> in <code>TextField</code>. <br/>The API returns <code>"&lt;a href=\"http://googke.ru\"&gt;google&lt;/a&gt;"</code>.</p>
<p>How can I remove <code>"</code> escaping?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/html.html" rel="nofollow noreferrer"><code>html</code></a> module, which has a method named <code>escape</code>: </p>
<blockquote>
<pre><code>html.escape(s, quote=True)
</code></pre>
<p>Convert the characters &amp;, &lt; and &gt; in string s to HTML-safe sequences. Use this if you need to display text that might contain
  such characters in HTML. <strong>If the optional flag quote is true, the
  characters (") and (') are also translated;</strong> this helps for inclusion
  in an HTML attribute value delimited by quotes, as in <code>&lt;a href="..."&gt;</code>.</p>
<p>New in version 3.2.</p>
</blockquote>
<p>Let <code>s</code> be: <code>s = '&lt;a href="http://googke.ru"&gt;google&lt;/a&gt;'</code> then:</p>
<pre><code>from html import escape

html_line = escape(s)
</code></pre>
<p>Now the <code>html_line</code> contains the <code>s</code> string without any 'escaping', looking like this:</p>
<pre><code>&amp;lt;a href=&amp;quot;http://googke.ru&amp;quot;&amp;gt;google&amp;lt;/a&amp;gt;
</code></pre>
<hr/>
<p>If you want to keep the characters <code>&lt; &gt; &amp;</code> etc. but avoid the escaping of <code>"</code>, you can utilize the other method of the <a href="https://docs.python.org/3/library/html.html" rel="nofollow noreferrer"><code>html</code></a> module, called <code>unescape</code>:</p>
<pre><code>from html import unescape

html_line = unescape(s)
</code></pre>
<p>Now the <code>html_line</code> will look like this:</p>
<pre><code>&lt;a href="http://googke.ru"&gt;google&lt;/a&gt;
</code></pre>
</div>
