<p>I know how to do the sum of a row like this: <code>row = [sum(row) for row in Matrix]</code>, but how to find the sum of a column in a python matrix? </p>
<p>If I have this matrix:</p>
<pre><code>Matrix =[[0, 2, 2],
         [0, 2, 2], 
         [0, 0, 2]]
</code></pre>
<p>It should make 3 values, they are: <code>0</code>,<code>4</code>, and <code>6</code>.</p>

<pre><code>def sumColumn(m):
    return [sum(col) for col in zip(*m)]

<div class="post-text" itemprop="text">
<p>A more Pythonic way than some of the other answers:</p>
<pre><code>[sum(row[i] for row in Matrix) for i in range(len(Matrix[0]))]
</code></pre>
<p>Bit this is very inefficient in terms of cache performance. So try your best to avoid it, or transpose the matrix first if you need to perform this operation on the same matrix multiple times.</p>
