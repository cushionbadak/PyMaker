<p>It may just be because your system does not provide the syscall for <code>chflags</code>.<br/>
This is also the case for me running Mint 18.1 and I get the same results than you do, for both python2 and python3.</p>
<p>BSD systems provide this function while others do not necessarily do.</p>

<div class="post-text" itemprop="text">
<p>I am heaving the same issue with my Python 3.6.7. at my Ubuntu 18.04 box.
It is mentioned in <a href="https://bugs.python.org/issue31313" rel="nofollow noreferrer">https://bugs.python.org/issue31313</a>. 
In short, Linux doesn't support chflags.<br/>
Linux uses chattr, sadly Python os module doesn't. </p>
