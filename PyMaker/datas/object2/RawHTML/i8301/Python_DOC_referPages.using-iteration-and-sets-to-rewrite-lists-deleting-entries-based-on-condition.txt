<p>How about using some python builtins <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer">all</a> and <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer">filter</a>:</p>
<pre><code> x = [
    ['car','doug',''], 
    ['dede', 'eli', 'fred'], 
    ['law', '','frank'], 
    ['','roger','w'], 
    ['alex', 'berk' , 'carl']
]

# python 2.7
print filter(all, x)

# python 3.5
print(list(filter(all, x)))

- [['dede', 'eli', 'fred'], ['alex', 'berk', 'carl']]
</code></pre>
<p>Filter evaluates a preedicate and only returns it if it is true</p>
<hr/>
<p>All returns true if all items in an iterator are true</p>
