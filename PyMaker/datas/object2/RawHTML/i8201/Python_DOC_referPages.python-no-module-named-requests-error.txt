<div class="post-text" itemprop="text">
<p>The module <a href="http://docs.python-requests.org/en/master/user/install/" rel="nofollow noreferrer">requests</a> is an external module you need to install first. This may help:</p>
<pre><code>pip3.6 install requests
</code></pre>
<p>It's a shell command to install the latest requests module for Python 3.6. In some machines, you may need to replace <code>pip3.6</code> with <code>pip</code>. You can check the Python external module installation documentation in <a href="https://docs.python.org/3/installing/index.html" rel="nofollow noreferrer">this page</a>.</p>
