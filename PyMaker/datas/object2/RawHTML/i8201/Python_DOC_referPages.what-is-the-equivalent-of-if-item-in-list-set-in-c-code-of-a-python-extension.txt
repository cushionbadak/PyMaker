<div class="post-text" itemprop="text">
<p>Writing a Python extension module which is intended to speed up execution of a function written as Python script I am stuck with the problem of how to check in C-code of a Python extension module if a C-Python object is in a C-Python set/list of C-Python objects. </p>
<p>In other words what I am looking after is the equivalent of Python script expression <code>item in list/set</code> in C code of a Python extension module. </p>
<p>Any hints how to accomplish this are welcome. </p>
</div>
<div class="post-text" itemprop="text">
<blockquote>
<p>int <code>PySequence_Contains(PyObject *o, PyObject *value)</code></p>
<p>Determine if <em>o</em> contains <em>value</em>. If an item in <em>o</em> is equal to <em>value</em>, return <code>1</code>, otherwise return <code>0</code>. On error, return <code>-1</code>. This is equivalent to the Python expression <code>value in o</code>.</p>
</blockquote>
<p><a href="https://docs.python.org/3/c-api/sequence.html#c.PySequence_Contains" rel="nofollow noreferrer">source</a></p>
