<div class="post-text" itemprop="text">
<p>To sort the lines by the score, you will need a <a href="https://docs.python.org/3/glossary.html#term-key-function" rel="nofollow noreferrer">key-function</a> to extract the numerical score from a line:</p>
<pre><code>def get_score(line):
    return int(line.split()[-1])
</code></pre>
<p>Then you can sort effortlessly:</p>
<pre><code>&gt;&gt;&gt; lines = ['Cristiano Ronaldo 7', 'Ander Herrera 21', 'Mario Balotelli 9']
&gt;&gt;&gt; lines.sort(key=get_score)
&gt;&gt;&gt; lines
['Cristiano Ronaldo 7', 'Mario Balotelli 9', 'Ander Herrera 21']
</code></pre>

<span class="comment-copy"><code>while not line[0].isdigit(): i = i + 1</code> will either loop forever or not loop at all.</span>
<span class="comment-copy">Why not just split on the last space? <code>footballName, _, footballNumber = line.strip().rpartition(' ')</code></span>
<span class="comment-copy">The condition expression of a <code>while</code> loop has to depend on something that changes inside the loop, or the loop will run forever (or not at all), unless you have a <code>break</code> inside the loop that depends on something that changes inside the loop.</span>
<span class="comment-copy">I'd have used a lambda for such a simple key function. OTOH, <code>get_score</code> is self-documenting.</span>
