<p>I'm using python on my CentOS7 server.<br/>
By default, there are python, python2, python2.7.<br/></p>
<p>Then, I do <code>yum install tornado</code>. For now everything works fine.<br/></p>
<p>Now I need to use python3.5 with tornado.<br/></p>
<p>I've downloaded and installed python3.5 with its source code. But when I <code>python3.5 server.py</code>, I get an error:<br/></p>
<blockquote>
<p>no module named 'tornado'</p>
</blockquote>
<p>I've also tried <code>python3.5 -m pip install tornado</code> but I get an error:<br/></p>
<blockquote>
<p>no module named pip</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>First install pip with <a href="https://docs.python.org/3/library/ensurepip.html" rel="nofollow noreferrer">ensurepip</a>:</p>
<pre><code>python3.5 -m ensurepip
</code></pre>
<p>Then:</p>
<pre><code>python3.5 -m pip install tornado
</code></pre>
</div>
