<p>I have this code, but I can't figure out how to change directory on the server before uploading files.</p>
<p>Can anyone help me out?</p>
<pre><code>import ftplib
import os

server = 'enter your servername here'
username = 'root'
password = 'passowrd'
myFTP = ftplib.FTP(server, username, password)
myPath = r'C:\path_of_the_folder_goes_here'
def uploadThis(path):
    files = os.listdir(path)
    os.chdir(path)
    for f in files:
        if os.path.isfile(path + r'\{}'.format(f)):
            fh = open(f, 'rb')
            myFTP.storbinary('STOR %s' % f, fh)
            fh.close()
        elif os.path.isdir(path + r'\{}'.format(f)):
            myFTP.mkd(f)
            myFTP.cwd(f)
            uploadThis(path + r'\{}'.format(f))
    myFTP.cwd('..')
    os.chdir('..')
uploadThis(myPath)
</code></pre>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/ftplib.html#ftplib.FTP.cwd" rel="noreferrer"><code>FTP.cwd</code> method</a>:</p>
<pre><code>myFTP.cwd('/remote/path')
</code></pre>
<p>before you call</p>
<pre><code>uploadThis(myPath)
</code></pre>
