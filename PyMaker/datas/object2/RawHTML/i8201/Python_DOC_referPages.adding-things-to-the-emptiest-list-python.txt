<div class="post-text" itemprop="text">
<p>I am looking for a simple way to compare the content of multiple lists and find the one with the fewest variables.</p>
<p>Then I want to set the new variable to be added to that list.</p>
<p>Example:</p>
<pre><code>list1 = [1,5,7,12,4,8]
list2 = [3,2,9,11,14,6,10]
list3 = [13,15,16]
</code></pre>
<p>In this I would want it to find list3 to be the shortest and append the new value to it.</p>
</div>
<div class="post-text" itemprop="text">
<p>Due to pythons <code>min</code>, <code>max</code>, and <code>sort</code>'s <code>key</code> keyword argument this is fairly simple to do in python </p>
<pre><code>min([list1, list2, list3..], key = len).append(item)
</code></pre>
<p>key corresponds to the function applied to each element and whose result is used for comparison in both <a href="https://docs.python.org/3/library/functions.html#min" rel="nofollow noreferrer"><code>min</code></a> and <a href="https://docs.python.org/3/library/functions.html#max" rel="nofollow noreferrer"><code>max</code></a>.  in this case the function <code>len</code> (which retrieves the length of sequence objects like list and tuple and any class that defines <code>__len__</code> ) is used. </p>
<p>from <code>min</code>. </p>
<blockquote>
<p>The key argument specifies a one-argument ordering function like that used for list.sort()</p>
</blockquote>
<p>from <a href="https://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow noreferrer">list.sort()</a></p>
<blockquote>
<p>key specifies a function of one argument that is used to extract a comparison key from each list element (for example, key=str.lower). </p>
</blockquote>
<p>example</p>
<pre><code>&gt;&gt;&gt; x = [11231]
&gt;&gt;&gt; y = [1,2,3,4,5]
&gt;&gt;&gt; z = [1,2,3,4,1,1]
&gt;&gt;&gt; min([x,y,z], key = len)
[11231]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could write a small function that checks the <code>len</code> of each list, then <code>append</code> to that list.</p>
<pre><code>def add_to_shortest(lists, item):
    min(lists, key = lambda i: len(i)).append(item)
</code></pre>
<p>For example</p>
<pre><code>&gt;&gt;&gt; add_to_shortest([list1, list2, list3], 5)
&gt;&gt;&gt; list3
[13, 15, 16, 5]
</code></pre>
