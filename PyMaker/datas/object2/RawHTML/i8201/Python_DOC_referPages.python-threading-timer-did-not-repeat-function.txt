<div class="post-text" itemprop="text">
<p>To make it repeat, you need to setup a new timer after each interval.</p>
<p>One way to do it, although its a bit functional. It may not be the simplest way.</p>
<pre><code>from threading import Timer
from functools import partial

def plus(a,b):
    print (a+b)

def repeat(func,interval,*args):

    # call the provided func
    func(*args)

    Timer(interval, partial(repeat, func, interval), args=args).start()

repeat(plus,1,2,3)
</code></pre>
<p>In <code>repeat</code> call your function, then setup the timer. You need to provide the correct arguments  to <code>interval</code>, so we need to partially apply it to already have the first two arguments.</p>
