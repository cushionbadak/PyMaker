<p>I want to check if a variable's value is one of 'Val' or 'val'.
I can't use <code>var.lower()</code> because the variable can be None.
I know I can use <code>var in ['Val','val']</code> or <code>str(var).lower()</code> but I look for a nicer way...</p>
</div>
<div class="post-text" itemprop="text">
<p>you can just combine those tests:</p>
<pre><code>if var and var.lower() == 'val':
</code></pre>
<p><a href="https://docs.python.org/3/reference/expressions.html#and" rel="nofollow noreferrer"><code>and</code></a> short-circuits: if <code>var</code> is <code>None</code>, <code>var.lower()</code> is not executed.</p>
<p>or if <code>var</code> can also have other types and you want to be a bit more explicit:</p>
<pre><code>if var is not None and var.lower() == 'val':
</code></pre>
