<pre><code>def API(request, selector):
    if request.method == 'POST':
        API2 = callback(f, user, password)
        API2.selector()

<p>In this case <code>callback</code> is a Class which is initiating in API2. Inside the Class <code>callback</code> there is a function called <code>register</code>. I want to call <code>register()</code> via <code>API2.selector()</code>. So <code>API(request, register)</code> should result in <code>API2.register()</code>.</p>
<p>Is something like this possible? </p>
<p>The only why I could think of is if/elif statements.</p>
<p>Finlay I would like it to work when I make a POST to '.../API/register' it will return <code>API2.register()</code>.</p>

<div class="post-text" itemprop="text">
<p>You can write:</p>
<pre><code>getattr(API2, selector)()
</code></pre>
<p>which uses the built-in <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer"><code>getattr</code></a> function. In this case you should provide <code>selector</code> as text, so:</p>
<pre><code>getattr(API2, 'register')()
</code></pre>
<p>If you want to return the result you should use <code>return</code>:</p>
<pre><code>def API(request, selector):
    if request.method == 'POST':
        API2 = callback(f, user, password)
        return getattr(API2, selector)()
</code></pre>
