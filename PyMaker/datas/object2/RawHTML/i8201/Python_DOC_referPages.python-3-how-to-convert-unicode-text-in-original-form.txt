<p>Seems like it encoded with <a href="https://docs.python.org/3/library/codecs.html#python-specific-encodings" rel="nofollow noreferrer"><code>unicode-escape</code> encoding</a> twice; reverse it.</p>
<pre><code>&gt;&gt;&gt; text = 'Al\\\\u0027sPlace'
&gt;&gt;&gt; text.encode().decode('unicode-escape').encode().decode('unicode-escape')
"Al'sPlace"
</code></pre>
