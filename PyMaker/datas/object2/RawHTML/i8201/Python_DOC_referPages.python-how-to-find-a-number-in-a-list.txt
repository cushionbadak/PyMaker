<a href="https://i.stack.imgur.com/ZYIe6.gif" rel="noreferrer"><img alt="enter image description here" src="https://i.stack.imgur.com/ZYIe6.gif"/></a></p>
<pre><code>&gt;&gt;&gt;a = [2, 5, 6, 9]
&gt;&gt;&gt;b = 3
&gt;&gt;&gt;[x for x in a if x&lt;=b]
&gt;&gt;&gt;[2]
</code></pre>
<p>If you want to sorted list,</p>
<pre><code>sortedList = sorted(x for x in a if x &lt;= b)
</code></pre>
<p>or sort after the list creation,</p>
<pre><code>&gt;&gt;&gt;myList = [x for x in a if x &lt;= b]
&gt;&gt;&gt;myList.sort()
</code></pre>
<p>Using <a href="https://docs.python.org/3/library/functions.html#filter" rel="noreferrer">filter</a>,</p>
<pre><code>&gt;&gt;&gt;filter(lambda x: x &lt;= b, a)
</code></pre>
