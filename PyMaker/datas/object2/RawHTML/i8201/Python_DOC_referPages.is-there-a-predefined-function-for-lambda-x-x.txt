<p>I found (lambda **x: x) is very useful for defining a dict in a succinct way, e.g.</p>
<pre><code>xxx = (lambda **x: x)(a=1, b=2, c=3)
</code></pre>
<p>Is there any pre-defined python function does that?</p>
</div>
<div class="post-text" itemprop="text">
<p>The <code>dict</code> function/constructor can be used in the same manner.</p>
<pre><code>&gt;&gt;&gt; (lambda **x: x)(a=1, b=2, c=3) == dict(a=1, b=2, c=3)
True
</code></pre>
<p>See <code>help(dict)</code> for more ways to instantiate <code>dict</code>s. You are not limited to just defining them with <code>{'a': 1, 'b': 2, 'c': 3}</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Try the <code>{}</code> literal dictionary syntax. It is quite succinct. See <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">5.5. <em>Dictionaries</em></a> in the <strong>Data Structures tutorial</strong>. </p>
<pre><code>&gt;&gt;&gt; xxx = {'a': 1, 'b': 2, 'c': 3}
&gt;&gt;&gt; xxx
{'a': 1, 'b': 2, 'c': 3}
</code></pre>
