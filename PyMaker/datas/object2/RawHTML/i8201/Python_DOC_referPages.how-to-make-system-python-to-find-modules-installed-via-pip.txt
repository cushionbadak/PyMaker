<div class="post-text" itemprop="text">
<p>Use the versioned Python commands in combination with the -m switch to run the appropriate copy of pip.</p>
<p><code>python2 -m pip install SomePackage</code>  # default Python 2</p>
<p><code>python2.7.10 -m pip install SomePackage</code>  # specifically Python 2.7.10 </p>
<p><a href="https://docs.python.org/3/installing/index.html?highlight=pip#work-with-multiple-versions-of-python-installed-in-parallel" rel="nofollow noreferrer">check this</a></p>
<p>Best practice is to use <a href="http://python-guide-pt-br.readthedocs.io/en/latest/dev/virtualenvs/#virtualenv" rel="nofollow noreferrer">virtualenv</a>,</p>
<p>If you want to install a package to a particular location using pip,
you can simply do </p>
<p><code>pip install --install-option="--prefix=$PREFIX_PATH" package_name</code></p>
<p>using --target option,</p>
<p><code>pip install --target=&lt;path&gt; package_name</code></p>
<p>you also need to add the <code>&lt;path&gt;</code> location  to <code>PYTHONPATH</code></p>
