<div class="post-text" itemprop="text">
<p><code>track_info['album']['artists']</code> is a list, you need to get item using index (<code>list[0]</code>):</p>
<pre><code>artist_name = track_info['album']['artists'][0]['name']
</code></pre>
<p>It can be multiple artists. In such case use <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>artist_names = [artist['name'] for artist in track_info['album']['artists']]
</code></pre>
