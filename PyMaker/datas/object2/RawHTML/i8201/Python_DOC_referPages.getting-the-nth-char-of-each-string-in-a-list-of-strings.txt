<p>a version using <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow noreferrer"><code>itertools.zip_longest</code></a>:</p>
<pre><code>from itertools import zip_longest

lst = ['abc', 'def', 'gh']
strg = ''.join(''.join(item) for item in zip_longest(*lst, fillvalue=''))
print(strg)
</code></pre>
<p>to get an idea why this works it may help having a look at</p>
<pre><code>for tpl in zip_longest(*lst, fillvalue=''):
    print(tpl)
</code></pre>
