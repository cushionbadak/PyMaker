<div class="post-text" itemprop="text">
<p>I have Python 2.7 as root. I need to install the package "statistics" in Python 3.6, and it is not in the environments of anaconda navigator. How can install "statistics" with <code>conda</code> or <code>pip</code> for a secondary Python environment?</p>
</div>
<div class="post-text" itemprop="text">
<p>Create a new Python 3 environment by running:</p>
<pre><code>conda create --name python3 python=3
</code></pre>
<p>If you want all the standard anaconda packages installed by default, do:</p>
<pre><code>conda create --name python3 python=3 anaconda
</code></pre>
<p>Whenever you need to use <code>python3</code> run:</p>
<pre><code>activate python3
</code></pre>
<p>Then use the command line as normal. So, if you want to install something into your <code>python3</code> environment, make sure you <code>activate python3</code> first.</p>
<p>Note that python 3 <a href="https://docs.python.org/3/library/statistics.html" rel="nofollow noreferrer">has it's own statistics module</a> that you may find useful, and this module <a href="https://pypi.python.org/pypi/statistics" rel="nofollow noreferrer">has been ported to python 2</a> if you would prefer.</p>
</div>
