<p><a href="https://docs.python.org/3/library/cgitb.html" rel="nofollow noreferrer">Usually</a> it is advised to initialize <em>cgitb</em> at the beginning of a script with the function <code>cgitb.enable()</code>.</p>
<p>But how can I do to use cgitb to format a special exception I catched in a try/except block, without registering cgitb as a global exception handler?</p>
<p>I suppose it would look like this:</p>
<pre><code>import cgitb
try:
    1/0
except Exception as exc:
    print(cgitb.something(exc))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>import cgitb
import sys
try:
    1/0
except Exception:
    print(cgitb.text(sys.exc_info()))
</code></pre>
</div>
