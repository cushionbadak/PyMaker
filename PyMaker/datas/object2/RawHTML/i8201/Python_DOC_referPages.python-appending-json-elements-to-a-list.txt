<p>Each value in each dictionary is a list containing one string. Just take the first element:</p>
<pre><code>with open('city.json') as json_data:
    data = json.load(json_data)
    for index in data:
       myList.append(index['city'][0])  # note the indexing
</code></pre>
<p>You may want to use <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>str.strip()</code></a> to remove the <code>\r\n</code> whitespace around each city value:</p>
<pre><code>with open('city.json') as json_data:
    data = json.load(json_data)
    for index in data:
       myList.append(index['city'][0].strip())
</code></pre>
<p>You can put the whole thing into a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>, no need to use <code>list.append()</code>:</p>
<pre><code>with open('city.json') as json_data:
    data = json.load(json_data)
    myList = [d['city'][0].strip() for d in data]
</code></pre>
