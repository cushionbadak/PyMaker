I already tried:</p>
<pre><code>print(response['entries']['playerOrTeamId'])

<div class="post-text" itemprop="text">
<p>You can use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> to retrieve all the ids from the response. </p>
<pre><code>all_ids = [entry["playerOrTeamId"] for entry in response["entries"]]
</code></pre>
<p>This is fancy Python shorthand for looping through the responses and creating a list of the just the ids. </p>
