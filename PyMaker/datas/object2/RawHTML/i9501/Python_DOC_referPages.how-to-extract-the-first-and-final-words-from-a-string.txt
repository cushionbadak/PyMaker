<div class="post-text" itemprop="text">
<p>You have to firstly convert the string to <code>list</code> of words using <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="noreferrer"><code>str.split</code></a> and then you may access it like:</p>
<pre><code>&gt;&gt;&gt; my_str = "Hello SO user, How are you"
&gt;&gt;&gt; word_list = my_str.split()  # list of words

# first word  v              v last word
&gt;&gt;&gt; word_list[0], word_list[-1]
('Hello', 'you')
</code></pre>
<p><strong>From Python 3.x</strong>, you may simply do:</p>
<pre><code>&gt;&gt;&gt; first, *middle, last = my_str.split()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you are using Python 3, you can do this:</p>
<pre><code>text = input()
first, *middle, last = text.split()
print(first, last)
</code></pre>
<p>All the words except the first and last will go into the variable <code>middle</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Let's say <code>x</code> is your input. Then you may do:</p>
<pre><code> x.partition(' ')[0]
 x.partition(' ')[-1]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Some might say, there is never too many answer's using regular expressions (in this case, this looks like the worst solutions..):</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; string = "Hello SO user, How are you"
&gt;&gt;&gt; matches = re.findall(r'^\w+|\w+$', string)
&gt;&gt;&gt; print(matches)
['Hello', 'you']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You would do:</p>
<pre><code>print text.split()[0], text.split()[-1]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Simply pass your string into the following <strong>function</strong>: </p>
<pre><code>def first_and_final(str):
    res = str.split(' ')
    fir = res[0]
    fin = res[len(res)-1]
    return([fir, fin])
</code></pre>
<p><strong>Usage</strong>: </p>
<pre><code>first_and_final('This is a sentence with a first and final word.')
</code></pre>
<p><strong>Result</strong>: </p>
<pre><code>['This', 'word.']
</code></pre>
</div>
