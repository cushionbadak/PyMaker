<p>The question is to create a function that reads a string and print a dictionary listing the positions for each UNIQUE word. The key is the word and the value is a list of its positions in the string.</p>
<p>Here would be an example string: </p>
<p><code>One fish two fish red fish blue fish</code></p>
<p>The correct output would be:</p>
<p><code>{'two': [2], 'one': [0], 'red': [4], 'fish': [1, 3, 5, 7], 'blue': [6]}</code></p>
<p>Here is my output:</p>
<p><code>{'blue': [6], 'two': [2], 'red': [4], 'fish': [1], 'One': [0]}</code></p>
<p>As you notice, the word 'fish' is repeated multiple times in this string. It is not just in position 1. What code do I need to add to print out the multiple positions of any word?</p>
<p>Here is my code:</p>
<pre><code>def wordPositions(s):
    d = {}

    words = s.split()
    for word in words:
        lst = []
        lst.append(words.index(word))
        d[word] = lst
    return d
print(wordPositions('One fish two fish red fish blue fish'))
</code></pre>

<p>Try the following code using <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate()</code></a>:</p>
<pre><code>s = 'One fish two fish red fish blue fish'
res = {}

for i, v in enumerate(s.split(' ')):
    if v in res:
        res[v].append(i)
    else:
        res[v] = [i]
</code></pre>

<pre><code>&gt;&gt;&gt; res
{'blue': [6], 'fish': [1, 3, 5, 7], 'two': [2], 'red': [4], 'One': [0]}
