<p>How would I get Arrow to return the difference in hours between two timestamps?</p>
<p>Here's what I have:</p>
<pre><code>difference = arrow.now() - arrow.get(p.create_time())
print(difference.hour)
</code></pre>
<p><code>p.create_time()</code> being the timestamp of the create time of a currently running process.</p>
<p>Returns:</p>
<pre><code>AttributeError: 'datetime.timedelta' object has no attribute 'hour'
</code></pre>
<p>Edit: I don't want the total time in all three formats, I want it as a remainder eg. "3 days, 4 hours, 36 minutes" not "3 days, 72 hours, 4596 minutes"</p>
</div>
<div class="post-text" itemprop="text">
<p>Given 2 dates that are formatted from a string to <code>arrow</code> type. </p>
<pre><code>&gt;&gt;&gt; date_1 = arrow.get('2015-12-23 18:40:48','YYYY-MM-DD HH:mm:ss')
&gt;&gt;&gt; date_2 = arrow.get('2017-11-15 13:18:20','YYYY-MM-DD HH:mm:ss')
&gt;&gt;&gt; diff = date_2 - date_1
</code></pre>
<p>The difference is a <a href="https://docs.python.org/3/library/datetime.html#timedelta-objects" rel="noreferrer"><code>datetime.timedelta</code></a> data type. </p>
<pre><code>&gt;&gt;&gt; print type(diff)
&lt;type 'datetime.timedelta'&gt;
</code></pre>
<p>And results in: </p>
<pre><code>&gt;&gt;&gt; print diff
692 days, 18:37:32
</code></pre>
<p>To get it formatted such that you would have <code>D days, H hours, M minutes, S seconds</code> you would get the days separately, and then using <code>divmod</code> function get the other information. </p>
<pre><code>&gt;&gt;&gt; days = diff.days # Get Day 
&gt;&gt;&gt; hours,remainder = divmod(diff.seconds,3600) # Get Hour 
&gt;&gt;&gt; minutes,seconds = divmod(remainder,60) # Get Minute &amp; Second 
</code></pre>
<p>The result would be: </p>
<pre><code>&gt;&gt;&gt; print days, " Days, ", hours, " Hours, ", minutes, " Minutes, ", seconds, " Second"
692  Days,  18  Hours,  37  Minutes,  32  Second
</code></pre>
