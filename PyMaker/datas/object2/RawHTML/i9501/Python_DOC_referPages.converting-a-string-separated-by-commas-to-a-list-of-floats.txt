<div class="post-text" itemprop="text">
<p>In Python, I currently have a one element list of elements given like so:</p>
<pre><code>x= ['1.1,1.2,1.6,1.7']
</code></pre>
<p>where each of the values are only separated by commas. I want to make this a list of floats, e.g like</p>
<pre><code>x=[1.1, 1.2, 1.6, 1.7]
</code></pre>
<p>I've tried <code>x=[float(i) for i in x]</code> and <code>x=[float(i) for i in x.split()]</code>, but both return errors.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>x</code> is a list with one string, so to access that string you need <code>x[0]</code>. That string is comma-separated, so you need to specify the delimiter: <code>split(',')</code>. (Otherwise, <code>split()</code> tries to split a string on whitespace, <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer">as described in the docs</a>.)</p>
<p>So you end up with:</p>
<pre><code>[float(i) for i in x[0].split(',')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map()</code></a> like this:</p>
<pre><code>list(map(float, x[0].split(',')))
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map()</code></a> takes a function (<code>float</code> in our case) and an iterable ( in our case this list: <code>x[0].split(',')</code>). <code>float</code> function is called for each item of our list <code>x[0].split(',')</code></p>
<p>It is equivalent to this <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>[float(item) for item in x[0].split(',')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Split the string by commas, and construct a <code>float</code> from each item:</p>
<pre><code>[ float(item) for item in '1.1,2.3,5.1'.split(',') ]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Make sure you are applying your <code>split()</code> function on a string and not the single element list. Use <code>x[0]</code> to ensure that. Also, pass a separator <code>,</code> to the <code>split()</code> function.</p>
<pre><code>x = [float(i) for i in x[0].split(',')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>x=[float(i) for i in x.split()]</code> is almost correct, except for two things. For one, you're not passing in anything to the <code>split()</code> function, so it's not going to split anything in your string (it would split only on whitespace, which your string does not have). You want to split by commas, so you have to pass a comma to it, like <code>x.split(',')</code>.</p>
<p>Second, seeing as <code>x</code> is defined by <code>x= ['1.1,1.2,1.6,1.7']</code>, which is a list containing a single string, you would have to refer to the string in the array with <code>x[0]</code>. The final code would look like this:</p>
<pre><code>x = ['1.1,1.2,1.6,1.7']
floats = [float(i) for i in x[0].split(',')]
print(floats)
</code></pre>
<p>This outputs a list of floats: <code>[1.1, 1.2, 1.6, 1.7]</code></p>
<p>If <code>x</code> were just a string, like <code>x = '1.1,1.2,1.6,1.7'</code>, then you would simply use <code>floats = [float(i) for i in x.split(',')]</code>.</p>
</div>
