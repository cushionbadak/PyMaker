<p>Notice that the names are separated by a colon(<code>:</code>) so add <code>:</code> in <code>split()</code> to split them and store them in multiple variables:</p>
<pre><code>    with open("filename.txt") as f:
        for line in f :
            word1,word2,word3 = line.split(":")

    print(word1)
    print(word2)
    print(word3)
</code></pre>
