<div class="post-text" itemprop="text">
<p>Since <code>diffrence</code> is of <a href="https://docs.python.org/3/library/datetime.html#available-types" rel="nofollow noreferrer"><code>timedelta</code></a> type, you should be using <a href="https://docs.python.org/3/library/datetime.html#datetime.timedelta.total_seconds" rel="nofollow noreferrer"><code>datetime.total_seconds()</code></a> method as:</p>
<pre><code> print(diffrence.total_seconds())
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can simply do it by calling <code>seconds</code>:</p>
<pre><code>print(diffrence.seconds)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This gives you the time difference in seconds:</p>
<pre><code>print(diffrence.total_seconds())
</code></pre>
<p>Example:</p>
<pre><code>t1 = datetime.datetime(2005, 1, 1)
t2 = datetime.datetime(2006, 1, 1)
diff = t2 - t1
print(diff.total_seconds())
</code></pre>
<p>Output:</p>
<pre><code>31536000.0
</code></pre>
<p>Looks alright:</p>
<pre><code>print(365 * 24 * 60 * 60)

31536000
</code></pre>
<p>From the <a href="https://docs.python.org/3.5/library/datetime.html?highlight=datetime#datetime.timedelta.total_seconds" rel="nofollow noreferrer">docs</a>:</p>
<blockquote>
<p>Total seconds in the duration.</p>
