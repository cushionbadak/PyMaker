<div class="post-text" itemprop="text">
<pre><code>a = list of number with length k
b = list of number with length k
</code></pre>
<p>How to get a list c with the lower value of a and b, position by position?</p>
<pre><code>Example:
a = [0.6, 0.8, 0.4]
b = [0.4, 1.0, 0.5]

c = [0.4, 0.8, 0.4]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="noreferrer"><code>zip()</code></a> like this:</p>
<pre><code>c = [min(*item) for item in zip(a, b)]
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; a = [0.6, 0.8, 0.4]
&gt;&gt;&gt; b = [0.4, 1.0, 0.5]
&gt;&gt;&gt;
&gt;&gt;&gt; c = [min(*item) for item in zip(a, b)]
&gt;&gt;&gt; c
[0.4, 0.8, 0.4]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just pass <code>map</code> two list arguments:</p>
<pre><code>c = map(min, a, b)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is a simple application of <code>min</code> and <code>zip</code>:</p>
<pre><code>c = [min(aa, bb) for aa, bb in zip(a, b)]
</code></pre>
<p>If you're going to do computations like this a lot, it might be worth using <code>numpy</code>:</p>
<pre><code>c = numpy.minimum(a, b)
</code></pre>
<p>e.g.:</p>
<pre><code>&gt;&gt;&gt; a = numpy.array([0.6, 0.8, 0.4])
&gt;&gt;&gt; b = numpy.array([0.4, 1.0, 0.5])
&gt;&gt;&gt; numpy.minimum(a, b)
array([ 0.4,  0.8,  0.4])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using numpy's <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.minimum.html" rel="nofollow noreferrer">minimum</a> function which does element-wise minimum of array elements.   </p>
<pre><code>numpy.minimum(a, b)
</code></pre>
</div>
