<div class="post-text" itemprop="text">
<p>Use a list comprehension:</p>
<pre><code>[x[::-1] for x in A]
# [[3, 2, 1], [2, 3, 4], [6, 5, 8]]
</code></pre>
<hr/>
<p>You can also <code>reverse</code> the list in place:</p>
<pre><code>for x in A: 
    x.reverse()

A
# [[3, 2, 1], [2, 3, 4], [6, 5, 8]]
</code></pre>
</div>

<p>Adjust like below:</p>
<pre><code>A = [[1,2,3],[4,3,2],[8,5,6]]
resultado = []

for l in range(len(A)):
    A2 = A[l][::-1]
    resultado.append(A2)

print(resultado)
</code></pre>
<p><code>resultado = []</code> inside your <code>for</code> loop, means for each iteration, you initialize <code>resultado</code> to <code>[]</code></p>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; print(resultado)
[[3, 2, 1], [2, 3, 4], [6, 5, 8]]
</code></pre>
<p><strong>Note:</strong> You can achieve the same result with a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> and <a href="https://docs.python.org/3/library/functions.html#reversed" rel="nofollow noreferrer"><code>reversed()</code></a>:</p>
<pre><code>&gt;&gt;&gt; [list(reversed(item)) for item in A]
[[3, 2, 1], [2, 3, 4], [6, 5, 8]]
</code></pre>
</div>
