<div class="post-text" itemprop="text">
<p>First of all, it checks only first 'win' combination (<code>[7,8,9]</code>) and then just returns false. Second, you can't check it using <code>in</code> operator like so.</p>
<pre><code>&gt;&gt;&gt; [1,2,3] in [4,1,2,3,6]
False
</code></pre>
<p>So if you want to check if the combination is on the board (in the same order):</p>
<pre><code>for w in wins:
    if ''.join(w) in ''.join(board):
        return True
else: # note indentation. else is part of for expression.
    return False 
</code></pre>
<p>if the order does not matter:</p>
<pre><code>for w in wins:
    if all(x in board for x in w):
        return True
else:
    return False
</code></pre>

<div class="post-text" itemprop="text">
<p>There are two problems with your code:</p>
<ul>
<li><code>w in board</code> does not check whether the individual elements of <code>w</code> are in <code>board</code>, but whether <code>w</code>, as a whole, is an element of <code>board</code>, which is not the case for any <code>w</code></li>
<li>you return <code>False</code> as soon as the first <code>w</code> does not work out (assuming that the first check would work), instead of after you tested <em>all</em> the <code>w</code></li>
</ul>
<p>Instead, you can use a nested <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer"><code>any</code></a> and <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer"><code>all</code></a> to check whether for <em>any</em> of the winning combinations, <em>all</em> the elements are in the <code>board</code>.</p>
<pre><code>&gt;&gt;&gt; wins = [[7,8,9],[4,5,6],[1,2,3],[7,4,1],[8,5,2],[9,6,3],[1,5,9],[3,5,7]] 
&gt;&gt;&gt; board = [4,5,6,9]
&gt;&gt;&gt; any(all(x in board for x in w) for w in wins)
True
</code></pre>
<p>Alternatively, since order does not matter and each of the elements should appear only once, you can use <code>sets</code> instead (note the inner <code>{...}</code> in <code>wins</code>) and use <a href="https://docs.python.org/3/library/stdtypes.html#set.issubset" rel="nofollow noreferrer"><code>issubsetof</code></a>:</p>
<pre><code>&gt;&gt;&gt; wins = [{7,8,9},{4,5,6},{1,2,3},{7,4,1},{8,5,2},{9,6,3},{1,5,9},{3,5,7}]
&gt;&gt;&gt; any(w.issubset(board) for w in wins)
True
</code></pre>
