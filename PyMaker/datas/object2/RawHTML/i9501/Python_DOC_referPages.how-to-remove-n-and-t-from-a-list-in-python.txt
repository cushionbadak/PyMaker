<p>Try the following:</p>
<p>Using <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>my_list = ['a', 'b', '\n', 'c', '\t', 'd']

my_list = [item for item in my_list if item not in ['\n', '\t']]
</code></pre>
<p>Using <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter()</code></a>:</p>
<pre><code>my_list = filter(lambda item: item not in ['\n', '\t'], my_list)
</code></pre>
<p><strong>Edit:</strong></p>
<p>For the example of inputs added to the question, you can remove <code>\t</code> and <code>\n</code> like this:</p>
<pre><code>c = [''.join(item.split()) for item in c]
</code></pre>
