<div class="post-text" itemprop="text">
<p>Say I have population data stored in a column of a dataframe using pandas in python with Country names as row indices. How do I convert the whole column of numbers into string thousands separator using commas. 
Basically, I need 12345678,integer, converted into 12,345,678.</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <code>apply</code> to format the numbers.  </p>
<pre><code>In [40]: ps.apply('{:,}'.format)
Out[40]:
CountryA    12,345,678
CountryB     3,242,342
dtype: object

In [41]: ps
Out[41]:
CountryA    12345678
CountryB     3242342
dtype: int64
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use regex as well.</p>
<pre><code>df['Population'].str.replace(r"(?!^)(?=(?:\d{3})+$)", ",")
</code></pre>
<p>See demo.</p>
<p><a href="https://regex101.com/r/cVYAGw/1" rel="nofollow noreferrer">https://regex101.com/r/cVYAGw/1</a></p>
