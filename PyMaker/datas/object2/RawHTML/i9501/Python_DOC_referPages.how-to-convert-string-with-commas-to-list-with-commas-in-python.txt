<p>Given a string with words all ending with commas (some have spaces too):</p>
<pre><code>WordString = 'word one, word-two, word3,'
</code></pre>
<p>How do I convert the string to a list keeping the spaces and commas?</p>
<pre><code>WordList = ['word one,','word-two,','word3,']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do this with the <code>split()</code> method of strings and a list comprehension:</p>
<pre><code>WordList = [s.strip() + ',' for s in WordString.split(',') if s.strip()]
</code></pre>
<p>The assumptions made here, based on your example:</p>
<ul>
<li>You only want to keep <em>interior</em> spaces (spaces between words), not leading/trailing spaces.</li>
<li>You want to keep a comma after each string in the final list.</li>
</ul>
</div>
<div class="post-text" itemprop="text">
<p>Try this with <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>['{},'.format(i).strip() for i in WordString.split(',') if i]
</code></pre>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>split()</code></a> uses comma separator to form a list of strings.</p>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>strip()</code></a> is used to remove trailing spaces from each string of the resulted list.</p>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; WordString = 'word one, word-two, word3,'
&gt;&gt;&gt; 
&gt;&gt;&gt; ['{},'.format(i).strip() for i in WordString.split(',') if i]
['word one,', 'word-two,', 'word3,']
</code></pre>
</div>
