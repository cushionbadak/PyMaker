<div class="post-text" itemprop="text">
<p>You can use the <a href="http://docs.python-requests.org/en/master/user/quickstart/" rel="nofollow noreferrer"><code>requests</code> library</a> for that:</p>
<pre><code>import requests
requests.post("http://robot:&lt;automation secret&gt;@127.0.0.1/monitor/check_mk/webapi.py?action=edit_host", data={"attributes": {"tag_check_puppet": "check_puppet"}, "hostname":"test.test.ch"})
</code></pre>
