<p>I've defined 2 lists, n1 and n2:</p>
<pre><code>In [1]: n1=[1,2,3]

In [2]: n2=[4,5,6]

In [3]: n1+n2
Out[3]: [1, 2, 3, 4, 5, 6]

In [4]: n1+=n2

In [5]: n1
Out[5]: [1, 2, 3, 4, 5, 6]
</code></pre>
<p>Well, what I expected to do is to get a new list:
n3=[5,7,9] as summary of each elements in n1 and n2.</p>
<p>I don't wish to write a for loop to do this routine job. 
Does python operator or library support a one-shot call to do this?</p>

<div class="post-text" itemprop="text">
<blockquote>
<p>I don't wish to write a for loop to do this routine job. Does python operator or library support a one-shot call to do this?</p>
</blockquote>
<p>Python does not support it natively, but you can use the library <a href="http://www.numpy.org/" rel="nofollow noreferrer">NumPy</a>:</p>
<pre><code>import numpy as np

n1 = np.array([1, 2, 3])
n2 = np.array([4, 5, 6])

n3 = n1 + n2
</code></pre>
<p>Alternatively, you can use <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> and <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a>:</p>
<pre><code>n3 = [x + y for x, y in zip(n1, n2)]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>No, there is no one-shot command for that. Adding elements in two lists is not a common operation. You can't avoid a loop here.</p>
<p>Use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a> and a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>[a + b for a, b in zip(n1, n2)]
</code></pre>
<p>Alternatively, use  <code>numpy</code> arrays:</p>
<pre><code>from numpy import array

n3 = array(n1) + array(n2)
</code></pre>
