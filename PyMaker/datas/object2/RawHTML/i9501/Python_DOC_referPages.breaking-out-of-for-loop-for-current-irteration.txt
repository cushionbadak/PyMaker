<p>You meant to use <a href="https://docs.python.org/3/tutorial/controlflow.html#break-and-continue-statements-and-else-clauses-on-loops" rel="nofollow noreferrer"><code>continue</code></a> instead of <code>break</code>:</p>
<pre><code>&gt;&gt;&gt; dummy = ['One','Two','Three','Four','Five']
&gt;&gt;&gt; 
&gt;&gt;&gt; for i in dummy:
...     print('Stage 1')
...     if i == 'Three':
...         continue
...     print('Stage 2')
... 
Stage 1
Stage 2

</code></pre>

<p>C language offers the continue keyword. Here is an example that prints all odd &lt; 100</p>
<pre><code>for(i=0;i&lt;100;i++)
{
    if((i%2) == 0)
    {
         continue;
    }
    printf("%i\n",i);
}
</code></pre>
