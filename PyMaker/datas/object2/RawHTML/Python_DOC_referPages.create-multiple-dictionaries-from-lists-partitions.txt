<p>So I have a list:</p>
<pre><code>[ 1, 2, 3, 4, 5 ]
</code></pre>
<p>And two lists of the form </p>
<pre><code>['A', 'B', 'C']  [ 'D', 'E'] 
</code></pre>
<p>whose total length sum is equal to the original list (partition). How can I obtain the following dictionaries in Python: </p>
<pre><code>{'A': 1, 'B': 2, 'C': 3 } {'D': 4, 'E': 5}
</code></pre>
<p>Thanks</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>next</code> with <code>iter</code>:</p>
<pre><code>values = [ 1, 2, 3, 4, 5 ]
lists = [['A', 'B', 'C'], ['D', 'E']]
itr = iter(values)
result = [{key: next(itr) for key in lst} for lst in lists]
</code></pre>
<p>Output:</p>
<pre><code>[{'A': 1, 'B': 2, 'C': 3}, {'D': 4, 'E': 5}]
</code></pre>
</div>
