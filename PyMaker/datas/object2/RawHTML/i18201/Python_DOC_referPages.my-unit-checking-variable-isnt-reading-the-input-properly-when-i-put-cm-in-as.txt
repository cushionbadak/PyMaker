<div class="post-text" itemprop="text">
<p>Your <code>if</code> statements are separate. Even if the first one is true, you'll still check the second one, and the third one, and the fourth one, and since only the first one was true, the <code>else</code> block will be executed.</p>
<p>Change them to <code>elif</code> and your code should work:</p>
<pre><code>uCheck = input('Unit? ')

if uCheck == 'nm':
    Wave = Wave
elif uCheck == 'cm':
    ...
</code></pre>
<p>Also, a better way to do this would be with a dictionary:</p>
<pre><code>units = {
    'nm': 1,
    'mm': 10**6,
    'cm': 10**7,
    'm': 10**9
}

unit = input('Unit? ')

if unit in units:
    wave *= units[unit]
else:
    print('Invalid Unit! Valid units are: ' + ', '.join(units))
    frequency()
</code></pre>
