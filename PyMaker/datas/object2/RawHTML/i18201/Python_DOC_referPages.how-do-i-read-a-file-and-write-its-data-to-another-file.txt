<p>I have to read from file named as <code>original.txt</code>, in that file I have to read <code>YAL013w</code> and <code>YGL127c</code> which are 1st and 3rd field, which is divided by <code>|</code> in the file.</p>

</code></pre>
<p>Then I have to make another file named as DNA repair which is the 5th field of the line and I have to put  <code>YAL013w</code> and <code>YGL127c</code> in the new file. I have to do it by python, please help me.</p>

<div class="post-text" itemprop="text">
<pre><code>with open('original.txt') as f:
    row = next(f).split('|') # =&gt; ['YAL013w', 'DEP1', 'YGL127c', 'SOH1', 'DNA repair', '\n']
    filename = row[4]
    with open(filename, 'w') as f2:
        f2.write('{}|{}\n'.format(row[0], row[2]))
</code></pre>
