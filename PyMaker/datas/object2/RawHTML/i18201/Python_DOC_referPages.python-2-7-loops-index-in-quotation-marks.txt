<div class="post-text" itemprop="text">
<p>I am using Python 2.7. How does you use the index of a loop inside quotation marks? This appears in a option. I have tried putting index in <code>quote()</code> and after <code>%</code>. </p>
<p>For example: </p>
<pre><code>for i in range(1, 10):  
    parser.add_option('-l', '--location', value="i")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="http://docs.python.org/2/library/stdtypes.html#str.format" rel="nofollow">format option</a> for strings.</p>
<pre><code>for i in range(1,10):
    parser.add_option('-l', '--location', value="{0}".format(i))
</code></pre>
<p>To do something as simple as this, the better method would be</p>
<pre><code>for i in range(1,10):
    parser.add_option('-l', '--location', value=str(i))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Why not simple:</p>
<pre><code>print "Number: %s" % 5
</code></pre>
</div>
