<p>I have a 1000 character long text string and I want to split this text in chunks smaller than 100 characters without splitting a whole word (99 characters are fine but 100 not). The wrapping/splitting should only be made on whitespaces:</p>
<p>Example:</p>
<pre><code>text = "... this is a test , and so on..."
                              ^
                  #position: 100
</code></pre>
<p>should be splitted to:</p>
<pre><code>newlist = ['... this is a test ,', ' and so on...', ...]
</code></pre>
<p>I want to get a list <code>newlist</code> of the text splitted properly into readable (not word-cropped) chunks. How would you do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="http://docs.python.org/2/library/textwrap.html" rel="nofollow">textwrap</a> module's <a href="http://docs.python.org/2/library/textwrap.html#textwrap.wrap" rel="nofollow">wrap</a> function. The below example splits the lines 10 characters wide:</p>
<pre><code>In [1]: import textwrap

In [2]: textwrap.wrap("... this is a test , and so on...", 10)
Out[2]: ['... this', 'is a test', ', and so', 'on...']
</code></pre>
</div>

<p>Wordwrap like the other guys said, however for an alternative option:</p>
<pre><code>def splitter(s, n):
    for start in range(0, len(s), n):
        yield s[start:start+n]

data = "abcdefghijabcdefghijabcdefghijabcdefghijabcdefghij"
for splitee in splitter(data, 10):
    print splitee
</code></pre>
