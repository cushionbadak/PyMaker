<p>What are invalid values for the <code>bool()</code> function? I mean which arguments make it throw exceptions?</p>
<p>As far as I know any custom data structure is <code>True</code> and empty list, empty strings etc. are false. </p>
<p>So are there any values for <code>bool</code> arguments that make it return other than "True" or "False"?</p>
</div>
<div class="post-text" itemprop="text">
<p>No, <code>bool()</code> can only return <code>True</code> or <code>False</code>. <strong>All</strong> values in Python have boolean meaning.</p>
<p>If the value is not an empty container, numeric 0, <code>None</code> or <code>False</code>, <code>bool()</code> returns <code>True</code>.</p>
<p>Note that <code>bool()</code> will use the <a href="http://docs.python.org/2/reference/datamodel.html#object.__len__" rel="nofollow"><code>__len__</code></a>, <a href="http://docs.python.org/2/reference/datamodel.html#object.__nonzero__" rel="nofollow"><code>__nonzero__</code></a> (Python 2) and <a href="http://docs.python.org/3/reference/datamodel.html#object.__bool__" rel="nofollow"><code>__bool__</code></a> (Python 3) special methods if present; if these throw an exception <code>bool()</code> will not catch it. </p>
</div>
