<div class="post-text" itemprop="text">
<p>Adding the prefix "b" to a string converts it to bytes:</p>
<pre><code>b'example'
</code></pre>
<p>But I can't figure out how to do this with a variable. Assuming <code>string = 'example'</code>, none of these seem to work:</p>
<pre><code>b(string)
b string
b'' + string
</code></pre>
<p>Is there a simple way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code># only an example, you can choose a different encoding
bytes('example', encoding='utf-8')
</code></pre>
<p>In Python3:</p>
<blockquote>
<p>Bytes literals are always prefixed with 'b' or 'B'; they produce an
  instance of the bytes type instead of the str type. They may only
  contain ASCII characters; bytes with a numeric value of 128 or greater
  must be expressed with escapes.</p>
</blockquote>
<p>In Python2:</p>
<blockquote>
<p>A prefix of 'b' or 'B' is ignored in Python 2; it indicates that the
  literal should become a bytes literal in Python 3.</p>
</blockquote>
<p>More about bytes():</p>
<blockquote>
<p>bytes([source[, encoding[, errors]]])</p>
<p>Return a new “bytes” object, which is an immutable sequence of
  integers in the range 0 &lt;= x &lt; 256. bytes is an immutable version of
  bytearray – it has the same non-mutating methods and the same indexing
  and slicing behavior.</p>
<p>Accordingly, constructor arguments are interpreted as for bytearray().</p>
<p>Bytes objects can also be created with literals, see String and Bytes
  literals.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/functions.html?highlight=bytes#bytes" rel="noreferrer"><code>bytes()</code></a>:</p>
<pre><code>&gt;&gt;&gt; bytes("hello", encoding="ascii")
b'hello'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>string = bytes(string, encoding= 'utf-8')
</code></pre>
<p>where 'string' is your variable. </p>
</div>
<div class="post-text" itemprop="text">
<p>Or use the <a href="https://docs.python.org/3/library/stdtypes.html#bytes.decode" rel="nofollow noreferrer"><code>bytes.decode()</code></a> method to convert to <code>string</code> (using a given encoding):</p>
<pre class="lang-py prettyprint-override"><code>&gt;&gt;&gt; b'hello'.decode('utf-8')
'hello'
</code></pre>
<p>The opposite conversion is <a href="https://docs.python.org/3/library/stdtypes.html#str.encode" rel="nofollow noreferrer"><code>str.encode()</code></a> to convert a <code>string</code> to <code>bytes</code>:</p>
<pre class="lang-py prettyprint-override"><code>&gt;&gt;&gt; 'hello'.encode('utf-8')
b'hello'
</code></pre>
</div>
