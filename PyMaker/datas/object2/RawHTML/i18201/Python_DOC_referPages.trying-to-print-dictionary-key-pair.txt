<div class="post-text" itemprop="text">
<p>Simplify this so it's not all one line, more readable and newlines are automatically part of the string by using multi-line strings:</p>
<pre><code>mydic = {"printer1": "printserver1", "printer2": "printserver2"}

template = r"""
FOR /F in blah blah blah
    do something with {key}
    with the value of {value}
    blah blah blah
END FOR
"""

for key, value in mydic.iteritems():
    print template.format(key=key, value=value)
</code></pre>

<p>Make the <code>\n</code> newlines part of the string:</p>
<pre><code>print ("For /F \"Tokens=4 delims=\\\" %%I In ('reg query HKCU\\Printers\\Connections       ^|find /I \"" +
       key + "\"') Do If \"%%I\"==\",," + mydic[key] + "," + key +

<p>You really want to use some <a href="http://docs.python.org/3/library/string.html#format-string-syntax" rel="nofollow">string formatting</a> here, together with a triple-quoted raw string to cut down on all those backslashes:</p>
<pre><code>template = r"""For /F "Tokens=4 delims=\" %%I In ('reg query HKCU\Printers\Connections       ^|find /I "{0}"') Do If "%%I"==",,"{1},{0}" goto :REMOVE
goto :SKIP
:REMOVERUNDLL32 printui.dll,PrintUIEntry /n \{1}\{0} /dn
:SKIP"""

for key in mydic:
    print(template.format(key, mydic[key]))
</code></pre>
