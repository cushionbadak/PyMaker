<p>I'm using sublime text 3 and I wanted to find a way to delete a file/folder using just the command line for easier development/ faster development.</p>
<p>I've installed advance_new_file so that I can easily create files and folders using just the shortcut key <code>ctrl</code> + <code>alt</code> + <code>n</code>. It's pretty smooth. But now I cannot find a way to delete files.</p>
<p>There is a command line tool in ST3 and pressing <code>ctrl</code>+ ` will open the command line. Is there a shortcut on deleting files without having me use the mouse?</p>

<div class="post-text" itemprop="text">
<p>It opens the <code>Python Interpreter</code>. To delete the file, you have to import <code>os</code> or <code>subprocess</code> module and invoke <code>rm</code> command.</p>
<p>If you are using <strong>Linux</strong>, you are in luck. Install Guake: <a href="http://guake.org/" rel="nofollow">http://guake.org/</a>.</p>
<p>These two keystrokes: <strong>F12 and Alt+Tab</strong> will allow you to do your tasks without mouse quickly.</p>
</div>
<div class="post-text" itemprop="text">
<p>Switching to an open shell prompt with <kbd>Alt</kbd><kbd>Tab</kbd> or <kbd>âŒ˜</kbd><kbd>Tab</kbd> is probably your fastest bet, but if you have lots of open windows or just want to do everything from within sublime, there are a couple of options.</p>
<p>Hitting <kbd>Ctrl</kbd><kbd>`</kbd> opens the built-in version of Python, in ST3 this is 3.3. From there, you can run <code>import os</code>, then use <a href="http://docs.python.org/3/library/os.html#os.remove" rel="nofollow"><code>os.remove()</code></a> to delete a file and <a href="http://docs.python.org/3/library/os.html#os.rmdir" rel="nofollow"><code>os.rmdir()</code></a> to delete an empty directory. To delete a directory with contents, first <code>import shutil</code> then use <a href="http://docs.python.org/3/library/shutil.html#shutil.rmtree" rel="nofollow"><code>shutil.rmtree()</code></a>. Other <code>shutil</code> methods like <a href="http://docs.python.org/3/library/shutil.html#shutil.copy" rel="nofollow"><code>shutil.copy()</code></a> (or <a href="http://docs.python.org/3/library/shutil.html#shutil.copy2" rel="nofollow"><code>shutil.copy2()</code></a>, for preserving metadata), <a href="http://docs.python.org/3/library/shutil.html#shutil.move" rel="nofollow"><code>shutil.move()</code></a>, and <a href="http://docs.python.org/3/library/shutil.html#shutil.copytree" rel="nofollow"><code>shutil.copytree()</code></a> are also useful.</p>
</div>
