<div class="post-text" itemprop="text">
<p>Use monospace font using <a href="http://docs.python.org/3/library/tkinter.ttk.html#tkinter.ttk.Treeview.tag_configure" rel="nofollow"><code>tkinter.ttk.Treeview.tag_configure</code></a>:</p>
<pre><code>...
for i in range(len(ABC)):
    dates2 = dates[i].split(".")
    date   = "{:&gt;2}.{:&gt;2}.{:&gt;4}".format(dates2[0],dates2[1],dates2[2])
    tree.insert("2",iid="2"+str(i), index="end",text=ABC[i], value=[date],
                tag='monospace') # &lt;----
tree.tag_configure('monospace', font='courier') # &lt;----
...
</code></pre>
<p>See <a href="http://docs.python.org/3/library/tkinter.ttk.html#tag-options" rel="nofollow">Tag Options</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can justify the text in the date column in the same way as you have justified the text in the date heading by using the anchor option.</p>
<pre><code>tree.heading("date", text='Date', anchor=tk.E)
tree.column("date", stretch=0, anchor=tk.E)
</code></pre>
<p>More detailed information on anchor and other options for the <code>heading</code> and <code>column</code> methods can be found in <a href="http://infohost.nmt.edu/tcc/help/pubs/tkinter/web/ttk-Treeview.html" rel="nofollow">Tkinter 8.5 reference: a GUI for Python</a> from New Mexico Tech.</p>
