<div class="post-text" itemprop="text">
<p>If I run the following using Python 3.3.1:</p>
<pre><code>import struct
struct.pack('!Bhh', 1, 1, 10)
</code></pre>
<p>I get this result:</p>
<pre><code>b'\x01\x00\x01\x00\n'
</code></pre>
<p>rather than the result I am expecting:</p>
<pre><code>b'\x01\x00\x01\x00\x0a\n'
</code></pre>
<p>Can anyone tell me where my missing byte has gone?</p>
</div>
<div class="post-text" itemprop="text">
<p>Your lost byte is right there; <code>\n</code> is character 10 in the ASCII table:</p>
<pre><code>&gt;&gt;&gt; chr(10)
'\n'
</code></pre>
<p>Instead of displaying it as <code>\x0a</code> it is displayed as a Python string literal escape code; other known escapes are also shown that way. Printable ASCII characters are shown as characters:</p>
<pre><code>&gt;&gt;&gt; struct.pack('!Bhh', 1, 1, 13)
b'\x01\x00\x01\x00\r'
&gt;&gt;&gt; struct.pack('!Bhh', 1, 1, 9)
b'\x01\x00\x01\x00\t'
&gt;&gt;&gt; struct.pack('!Bhh', 1, 1, 65)
b'\x01\x00\x01\x00A'
</code></pre>
<p>It might help to use <a href="http://docs.python.org/3/library/binascii.html#binascii.b2a_hex" rel="nofollow"><code>binascii.hexlify()</code></a> to convert your bytes to hexadecimal characters:</p>
<pre><code>&gt;&gt;&gt; from binascii import hexlify
&gt;&gt;&gt; hexlify(struct.pack('!Bhh', 1, 1, 10))
b'010001000a'
</code></pre>
</div>
