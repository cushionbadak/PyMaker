<div class="post-text" itemprop="text">
<p>I want to learn packet decoder processing using dpkt. On the site, I saw the following example code:</p>
<pre><code>&gt;&gt;&gt; from dpkt.ip import IP                            
&gt;&gt;&gt; ip = IP(src='\x01\x02\x03\x04', dst='\x05\x06\x07\x08', p=1)
&gt;&gt;&gt; ...
</code></pre>
<p>How do I convert an IP String like <code>'1.2.3.4'</code> to <code>'\x01\x02\x03\x04'</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/2/library/socket.html#socket.inet_aton" rel="nofollow"><code>socket.inet_aton</code></a>:</p>
<pre><code>&gt;&gt;&gt; import socket
&gt;&gt;&gt; socket.inet_aton('1.2.3.4')
'\x01\x02\x03\x04'
</code></pre>
<p>To get the dotted decimal back, use <a href="http://docs.python.org/2/library/socket.html#socket.inet_ntoa" rel="nofollow"><code>socket.inet_ntoa</code></a>:</p>
<pre><code>&gt;&gt;&gt; socket.inet_ntoa('\x01\x02\x03\x04')
'1.2.3.4'
</code></pre>
<p><strong>UPDATE</strong></p>
<p>In Python 3.3+, <a href="http://docs.python.org/3/library/ipaddress.html#ipaddress.IPv4Address" rel="nofollow"><code>ipaddress.IPv4Address</code></a> is another option.</p>
<pre><code>&gt;&gt;&gt; import ipaddress
&gt;&gt;&gt; ipaddress.IPv4Address('1.2.3.4').packed
b'\x01\x02\x03\x04'
&gt;&gt;&gt; ipaddress.IPv4Address(b'\x01\x02\x03\x04')
IPv4Address('1.2.3.4')
&gt;&gt;&gt; str(ipaddress.IPv4Address(b'\x01\x02\x03\x04'))
'1.2.3.4'
</code></pre>
