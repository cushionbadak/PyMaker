<div class="post-text" itemprop="text">
<p>I am using python 3.3</p>
<p>I have a folder <code>model</code> with several modules, each with one class defined in it. The class has the same name as the module.</p>
<p>Now I want it so that when in the main script I do <code>foo=Foo()</code> that automatically the class  module <code>model.Foo</code> is loaded with the effect of <code>from model.Foo import Foo</code></p>
<p>How? </p>
</div>
<div class="post-text" itemprop="text">
<p>It is usually not a good idea to have automatic import. How would the python interpreter know that <code>Foo</code> is <code>model.Foo</code> ?</p>
<p>One way to "ease" importing classes is to load them in the module <code>__init__.py</code></p>
<p>Ex:
<a href="https://github.com/masom/shopify-trois/blob/master/shopify_trois/models/__init__.py" rel="nofollow noreferrer">https://github.com/masom/shopify-trois/blob/master/shopify_trois/models</a></p>
<p>Someone else asked a similar question: <a href="https://stackoverflow.com/questions/1493888/python-auto-importing">Python Auto Importing</a></p>
</div>
<div class="post-text" itemprop="text">
<p>While agreeing with Martin's warning about doing so, if you really want to dynamically import code, have a look at the <a href="http://docs.python.org/3/library/imp.html" rel="nofollow"><code>imp</code> module</a>.</p>
</div>
