<p>I have a function that takes the items in multiple lists and permutates them. So if I have the lists <code>child0 = ['a', 'b']</code> and <code>child1 = ['c', 'd']</code>:</p>
<pre><code>def permutate():
   for i in child0:
      for k in child1:
         print (i, k)

permutate()

#  a c
#  a d
#  b c 
#  b d
</code></pre>
<p>I'm running into problems with saving the output into a text file. I can't assign a var to the print statement because the output will change every time it runs through obviously, and writing the permutate() function to a text file does nothing. Doing a return instead of print won't run the permutation properly.... any tips on how to print all the permutations to a text file properly?</p>

<div class="post-text" itemprop="text">
<p>You need to build a list and return that list object:</p>
<pre><code>def permutate():
    result = []
    for i in child0:
        for k in child1:
            result.append((i, k))
    return result

for pair in permutate():
    print(*pair)
</code></pre>
<p>What you are doing is creating the <em>cartesian product</em>, not the permutations.</p>
<p>The Python standard library has a function to do just this already, in <a href="http://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow"><code>itertools.product()</code></a>:</p>
<pre><code>from itertools import product

list(product(child0, child1))
</code></pre>
<p>would produce the exact same list:</p>
<pre><code>&gt;&gt;&gt; from itertools import product
&gt;&gt;&gt; child0 = ['a', 'b'] 
&gt;&gt;&gt; child1 = ['c', 'd']
&gt;&gt;&gt; for pair in product(child0, child1):
...     print(*pair)
... 
a c
a d
b c
b d
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Pass a file object as an argument, and use <code>file</code> argument of <code>print</code> function.</p>
<pre><code>def permutate(f):
   for i in child0:
      for k in child1:
         print(i, k, file=f)

with open('testfile.txt', 'w') as f:
    permutate(f)
</code></pre>
