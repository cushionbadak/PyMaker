<p>I am using the Python Decimal class for precise floating-point arithmetic. I need to convert the result number consistently into a standard notation number as a string. However, very small decimal numbers are rendered in scientific notation by default.</p>
<pre><code>&gt;&gt;&gt; from decimal import Decimal
&gt;&gt;&gt; 
&gt;&gt;&gt; d = Decimal("0.000001")
&gt;&gt;&gt; d
Decimal('0.000001')
&gt;&gt;&gt; str(d)
'0.000001'
&gt;&gt;&gt; d = Decimal("0.000000001")
&gt;&gt;&gt; d
Decimal('1E-9')
&gt;&gt;&gt; str(d)
'1E-9'
</code></pre>
<p>How would I get <code>str(d)</code> to return <code>'0.000000001'</code>?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>'{:f}'.format(d)
Out[12]: '0.000000001'
</code></pre>
