<div class="post-text" itemprop="text">
<p>I just downloaded Python 3.4 and I'm wondering how you would go about finding the first directory of a relative path?  I.e. given the path <code>a/b/c/d</code> I would like to print <code>a</code>.</p>
<p>The closest I've gotten is:</p>
<pre><code>from pathlib import Path
print(list(Path('a/b/c/d').parents)[-2])
</code></pre>
<p>or </p>
<pre><code>p = Path('a/b/c/d')
print(p.parents[len(p.parents) - 2])
</code></pre>
<p>in both cases the <code>-2</code> part is a bit magical.  I've read the docs and the PEP, and haven't found any better way.. did I miss something obvious?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/pathlib.html#pathlib.PurePath.parts" rel="nofollow"><code>parts</code></a> attribute:</p>
<pre><code>&gt;&gt;&gt; from pathlib import Path
&gt;&gt;&gt; Path('a/b/c/d').parts
('a', 'b', 'c', 'd')
&gt;&gt;&gt; Path('a/b/c/d').parts[0]
'a'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="http://docs.python.org/3.4/library/pathlib.html?highlight=path#pathlib.PurePath.parts" rel="nofollow">Path.parts</a> is what you need.</p>
<pre><code>p = Path("a/b/c/d")
print(p.parts[0])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>From <a href="http://docs.python.org/3.4/library/pathlib.html#methods-and-properties" rel="nofollow">the documentation</a>, <code>parents</code> gives you <em>an immutable sequence providing access to the logical ancestors of the path</em>; the last element of that sequence would be the logical root, and the <em>second to last</em> element would then be the first directory in the path, which is what you're looking for. That is why you need to use the <code>p.parents[len(p.parents)-2]</code> shortcut ('-2' is for the 'second to last' element of a list).</p>
<p>Note that <code>parents</code> always gives you full formed paths. If you only want the directory name you might want to use <code>p.parts</code> instead. In that case the first element is the logical root, so you'd want <code>p.parts[1]</code>.</p>
</div>
<span class="comment-copy"><code>parents</code> doesn't support negative indexing...</span>
