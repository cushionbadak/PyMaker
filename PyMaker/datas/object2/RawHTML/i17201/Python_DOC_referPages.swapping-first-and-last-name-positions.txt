<p>You can do this by splitting the string by the comma—which yields a list—then reversing that list, and joining the list by a comma again to get back a string:</p>
<pre><code>&gt;&gt;&gt; name = 'Albus Percival Wulfric Brian, Dumbledore'
&gt;&gt;&gt; ', '.join(reversed(name.split(', ')))
'Dumbledore, Albus Percival Wulfric Brian'
</code></pre>
<p>If you can’t use the <a href="http://docs.python.org/3/library/functions.html#reversed" rel="nofollow"><code>reversed</code> function</a> (for whatever reason), then you can do the reversal manually by storing the list elements individually first.</p>
<pre><code>&gt;&gt;&gt; name.split(', ')
['Albus Percival Wulfric Brian', 'Dumbledore']
&gt;&gt;&gt; first, last = name.split(', ')
&gt;&gt;&gt; first
'Albus Percival Wulfric Brian'
&gt;&gt;&gt; last
'Dumbledore'
&gt;&gt;&gt; [last, first]
['Dumbledore', 'Albus Percival Wulfric Brian']
&gt;&gt;&gt; ', '.join([last, first])
'Dumbledore, Albus Percival Wulfric Brian'
&gt;&gt;&gt; last + ', ' + first
'Dumbledore, Albus Percival Wulfric Brian'
</code></pre>
<p>As it’s theoretically possible that the string contains multiple commas, you should consider passing <code>1</code> as the second parameter to <a href="http://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>str.split</code></a> to make sure that you only perform a single split, producing two list items:</p>
<pre><code>&gt;&gt;&gt; 'some, name, with, way, too, many, commas'.split(', ', 1)
['some', 'name, with, way, too, many, commas']
</code></pre>
<p>Instead of using <code>str.split</code>, you can also locate the comma in the string and split up the string yourself. You can use <a href="http://docs.python.org/3/library/stdtypes.html#str.find" rel="nofollow"><code>str.find</code></a> to get the index of the comma, and then use <a href="http://docs.python.org/3/library/stdtypes.html#common-sequence-operations" rel="nofollow">sequence splicing</a> to split up the string:</p>
<pre><code>&gt;&gt;&gt; name.index(',')
28
&gt;&gt;&gt; name[:28] # string up to index 28
'Albus Percival Wulfric Brian'
&gt;&gt;&gt; name[30:] # string starting at index 30 (we have to consider the space too)
'Dumbledore'
&gt;&gt;&gt; index = name.index(',')
&gt;&gt;&gt; name[index + 2:] + ', ' + name[:index]
'Dumbledore, Albus Percival Wulfric Brian'
</code></pre>

<p>You don't need many built in functions for this</p>
<pre><code>return name[name.find(',')+2:]+", "+name[:name.find(',')] 
</code></pre>
