<div class="post-text" itemprop="text">
<p>The result is json. Use <a href="http://docs.python.org/3/library/json.html#json.loads" rel="nofollow"><code>json.loads</code></a> to convert it to a dictionary.</p>
<pre><code>response = b'{\n "kind": "drive#file"'

import json
result = json.loads(response.decode()) # Python 3.x
#result = json.loads(response) # Python 2.x
result['kind'] # =&gt; 'drive#file'
</code></pre>
