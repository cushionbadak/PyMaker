<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/stdtypes.html#list.sort" rel="noreferrer"><code>list.sort</code></a> (inplace sorting) or <a href="http://docs.python.org/3/library/functions.html#sorted" rel="noreferrer"><code>sorted</code></a> (returns new sorted list) and pass <code>key</code> function. The return value of the key function is used for comparison:</p>
<pre><code>lst.sort(key=lambda d: d['createdAt'])
</code></pre>
</div>

<pre><code>from operator import itemgetter
