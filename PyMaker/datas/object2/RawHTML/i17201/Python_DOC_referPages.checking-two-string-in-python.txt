<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/3.4/library/stdtypes.html#set" rel="nofollow noreferrer">sets</a>:</p>
<pre><code>&gt;&gt;&gt; set("chayote").issubset("aceihkjouty")
True
</code></pre>
<p>Or, equivalently:</p>
<pre><code>&gt;&gt;&gt; set("chayote") &lt;= set("aceihkjouty")
True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I believe you are looking for <a href="http://docs.python.org/3/library/functions.html#all" rel="nofollow"><code>all</code></a> and a <a href="http://www.python.org/dev/peps/pep-0289/" rel="nofollow">generator expression</a>:</p>
<pre><code>&gt;&gt;&gt; s='chayote'
&gt;&gt;&gt; d='aceihkjouty'
&gt;&gt;&gt; all(x in d for x in s)
True
&gt;&gt;&gt;
</code></pre>
<p>The code will return <code>True</code> if all characters in string <code>s</code> can be found in string <code>d</code>.</p>
<hr/>
<p>Also, if string <code>s</code> contains duplicate characters, it would be more efficient to make it a set using <a href="http://docs.python.org/3/library/functions.html#func-set" rel="nofollow"><code>set</code></a>:</p>
<pre><code>&gt;&gt;&gt; s='chayote'
&gt;&gt;&gt; d='aceihkjouty'
&gt;&gt;&gt; all(x in d for x in set(s))
True
&gt;&gt;&gt;
</code></pre>
