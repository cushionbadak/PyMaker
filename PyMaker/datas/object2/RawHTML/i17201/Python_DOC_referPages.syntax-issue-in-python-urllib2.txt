<p>In Python3 <a href="http://docs.python.org/3.0/whatsnew/3.0.html#print-is-a-function" rel="nofollow"><code>print</code> is a <em>function</em></a>. Therefore it needs parentheses around its argument:</p>
<pre><code>print(response.info())
</code></pre>
<p>In Python2, <code>print</code> is a <em>statement</em>, and hence does not require parentheses.</p>
<hr/>
<p>After correcting the SyntaxError, as alecxe points out, you'll probably encounter an ImportError next. That is because the Python2 module called <code>urllib2</code> was renamed to <a href="http://docs.python.org/3/library/urllib.request.html#module-urllib.request" rel="nofollow"><code>urllib.request</code></a> in Python3. So you'll need to change it to</p>
<pre><code>import urllib.request as request
response = request.urlopen('http://pythonforbeginners.com/')
</code></pre>
<hr/>
<p>As you can see, the tutorial you are reading is meant for Python2. You might want to find a <a href="http://docs.python.org/3/tutorial/" rel="nofollow">Python3 tutorial</a> or <a href="http://docs.python.org/3.1/howto/urllib2.html" rel="nofollow">Python3 urllib HOWTO</a> to avoid running into more of these problems.</p>
