<div class="post-text" itemprop="text">
<pre><code>random.randint(bmin, bmax)
</code></pre>
<p>Using <a href="http://docs.python.org/3/library/random.html#random.randint" rel="nofollow"><code>randint(a, b)</code></a> requires that <code>a &lt;= b</code>.</p>
<p>However, in your case, <code>bmin</code> is <code>-1</code> and <code>bmax</code> is <code>-5</code>, so the relation isnâ€™t true: <code>-1 &lt;= -5</code> is false.</p>
<p>You need to exchange the values, so <code>bmin</code> contains the actual smaller number.</p>
