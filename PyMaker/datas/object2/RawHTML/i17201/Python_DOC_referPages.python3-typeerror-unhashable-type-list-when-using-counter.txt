<div class="post-text" itemprop="text">
<p>The <a href="http://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow">docs</a> say:</p>
<blockquote>
<p>A Counter is a dict subclass for counting hashable objects.</p>
</blockquote>
<p>In your case it looks like <code>results</code> is a dict containing <code>list</code> objects, which are not hashable.</p>
<p>If you are sure that this code worked in Python 2, print <code>results</code> to see its content.</p>
<pre><code>Python 3.3.2+ (default, Oct  9 2013, 14:50:09) 
&gt;&gt;&gt; from collections import Counter
&gt;&gt;&gt; results = {1: [1], 2: [1, 2]}
&gt;&gt;&gt; Counter([r[1] for r in results.items()])
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
  File "/home/vic/projects/venv/trains/lib/python3.3/collections/__init__.py", line 467, in __init__
    self.update(iterable, **kwds)
  File "/home/vic/projects/venv/trains/lib/python3.3/collections/__init__.py", line 547, in update
    _count_elements(self, iterable)
TypeError: unhashable type: 'list'
</code></pre>
<p>By the way, you can simplify your construction:</p>
<pre><code>Counter(results.values())
</code></pre>
