<p>What I am after is like this:</p>
<pre><code>list1 = ["well", "455", "antifederalist", "mooooooo"]
</code></pre>
<p>Something that pulls <code>"455"</code> from the list because of the number of characters.</p>

<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/3/library/functions.html#next" rel="nofollow"><code>next()</code></a> with a generator:</p>
<pre><code>&gt;&gt;&gt; list1 = ["well", "455", "antifederalist", "mooooooo"]
&gt;&gt;&gt; 
&gt;&gt;&gt; next(s for s in list1 if len(s) == 3)
'455'
</code></pre>
<p><code>next()</code> also lets you specify a "default" value to be returned if the list doesn't contain any string of length 3. For instance, to return <code>None</code> in such a case:</p>
<pre><code>&gt;&gt;&gt; list1 = ["well", "antifederalist", "mooooooo"]
&gt;&gt;&gt; 
&gt;&gt;&gt; print next((s for s in list1 if len(s) == 3), None)
None
</code></pre>
<p>(I used an explicit <code>print</code> because <code>None</code>s don't print by default in interactive mode.)</p>
<p>If you want <em>all</em> strings of length 3, you can easily turn the approach above into a list comprehension:</p>
<pre><code>&gt;&gt;&gt; [s for s in list1 if len(s) == 3]
['455']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>filter(lambda s: len(s) == 3, list1)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>And if you're looking to pull all items out of the list greater than some length:</p>
<pre><code> list2 = [string for string in list1 if len(string) &gt;= num_chars]
</code></pre>
