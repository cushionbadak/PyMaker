<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/2/library/string.html#string.ascii_uppercase" rel="nofollow"><code>string.ascii_uppercase</code></a> and simply concatenate the <code>_</code>:</p>
<pre><code>from string import ascii_uppercase

for c in ascii_uppercase + '_':
</code></pre>
</div>

<div class="post-text" itemprop="text">
<p>I would use <a href="http://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow"><code>itertools.chain()</code></a> (along with <a href="http://docs.python.org/3/library/string.html#string.ascii_uppercase" rel="nofollow"><code>string.ascii_uppercase</code></a> as has been suggested by other answers):</p>
<pre><code>from itertools import chain
from string import ascii_uppercase

for c in chain(ascii_uppercase, '_'):
    ...
</code></pre>
