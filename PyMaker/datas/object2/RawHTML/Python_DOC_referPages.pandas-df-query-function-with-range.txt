<p>I want to use range(a,b) in a query call to a pandas dataframe, it's unsuccessful:</p>
<pre><code>df.query('column_a in range(a,b))
</code></pre>
<p>Is there a solution to use range in a query call?</p>
</div>
<div class="post-text" itemprop="text">
<p>Using range is not support. See <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.eval.html#pandas.eval" rel="nofollow noreferrer">https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.eval.html#pandas.eval</a></p>
<p>A possible solution would be:</p>
<pre><code>df.query("column_a &gt;= a and column_a &lt; b")
</code></pre>
</div>
