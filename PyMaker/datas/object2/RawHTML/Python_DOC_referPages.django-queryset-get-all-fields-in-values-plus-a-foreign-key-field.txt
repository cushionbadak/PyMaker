<div class="post-text" itemprop="text">
<p>I have a model with some fields, one of them is a FK. I would like to use</p>
<pre><code>MyModel.objects.all().defer("pk").values()
</code></pre>
<p>and to add <strong>a field</strong> of my FK (myfk__name) without writing all model fields + 1 in the values.</p>
<p>I also would like to avoid doing another query after this one to add it manually.</p>
<p>Is it possible?</p>
</div>
<div class="post-text" itemprop="text">
<p>use <code>annotate</code>:</p>
<pre><code>MyModel.objects.all().defer('pk').values().annotate(name=F('myfk__name'))
</code></pre>
</div>
<span class="comment-copy">I don't think <code>defer("pk")</code> has any effect here; <code>values()</code> returns all fields.</span>
