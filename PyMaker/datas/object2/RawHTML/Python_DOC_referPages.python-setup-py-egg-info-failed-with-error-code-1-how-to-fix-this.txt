<div class="post-text" itemprop="text">
<p>The directory is temporary, as it says in the name. pip will destroy it when it finishes running if it failed or succeeded.</p>
<p>Your error is pretty clear, it's telling you that while reading the readme file it found a unicode character it doesn't know how to decode.</p>
<p>This looks like it's a known issue, <a href="https://github.com/aboSamoor/polyglot/issues/127" rel="nofollow noreferrer">https://github.com/aboSamoor/polyglot/issues/127</a>, <a href="https://github.com/aboSamoor/polyglot/issues/93" rel="nofollow noreferrer">https://github.com/aboSamoor/polyglot/issues/93</a>, <a href="https://github.com/aboSamoor/polyglot/issues/91" rel="nofollow noreferrer">https://github.com/aboSamoor/polyglot/issues/91</a>. A variety of solutions are provided in the linked issues.</p>
