<p><strong>Possible Duplicate:</strong><br/>
<a href="https://stackoverflow.com/questions/5214344/converting-a-string-that-represents-a-list-into-an-actual-list-object">Converting a string that represents a list, into an actual list object</a> </p>
</blockquote>
<p>Having a string like this:</p>
<pre><code>"[81, 102, 114, 132, 194, 210, 228, 317, 357, 358, 379, 396, 407, 417, 418, 420, 470, 471, 506, 526, 533, 538]"
</code></pre>
<p>How can I parse it easily into the corresponding list?</p>
<p>I know I could use the <code>re</code> module or splitting by <code>", "</code>, etc.</p>
<p>Is there any already existing function to do it?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/ast.html#ast.literal_eval" rel="noreferrer"><code>ast.literal_eval()</code></a>:</p>
<pre><code>&gt;&gt;&gt; import ast
&gt;&gt;&gt; s = "[81, 102, 114, 132, 194, 210, 228, 317, 357, 358, 379, 396, 407, 417, 418, 420, 470, 471, 506, 526, 533, 538]"
&gt;&gt;&gt; ast.literal_eval(s)
[81, 102, 114, 132, 194, 210, 228, 317, 357, 358, 379, 396, 407, 417, 418, 420, 470, 471, 506, 526, 533, 538]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; l = "[81, 102, 114, 132]"
&gt;&gt;&gt; map(int, l.strip("[]").split(", "))
[81, 102, 114, 132]
</code></pre>
<p>For Python 3+ you need to make list from <code>map</code> iterator:</p>
<pre><code>&gt;&gt;&gt; l = "[81, 102, 114, 132]"
&gt;&gt;&gt; list(map(int, l.strip("[]").split(", ")))
[81, 102, 114, 132]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The most direct way is to use <a href="http://docs.python.org/2/library/functions.html#eval" rel="nofollow">eval()</a>. That's often considered bad practice, though, because it can lead to security problems.</p>
</div>
<span class="comment-copy"><code>ast.literal_eval</code> is equally direct, and has no security issues.</span>
