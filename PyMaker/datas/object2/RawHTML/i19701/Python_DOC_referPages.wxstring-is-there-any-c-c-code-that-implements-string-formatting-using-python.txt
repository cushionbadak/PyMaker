<p>I am rewriting our application using wxWidgets. One of the goals is to replace our old approach to the localized strings. Another possible goal is to embed the Python interpreter to the application -- but only later. Anyway, it would be nice any C/C++ code or library capable of Python-like string formatting that uses the curly braces.</p>
<p>If you do not know Python, here is the doc for its <a href="http://docs.python.org/3/library/functions.html#format" rel="nofollow">format</a> function. You can find the reference to the <a href="http://docs.python.org/3/library/string.html#format-specification-mini-language" rel="nofollow">Format Specification Mini-Language</a> inside, and the <a href="http://docs.python.org/3/library/string.html#format-examples" rel="nofollow">Format examples</a>. Some extract from the doc... (The <code>&gt;&gt;&gt;</code> is a prompt of the interactive Python mode, the line below shows the result of the call. Python uses single or double quotes as a string delimiter):</p>
<pre><code>&gt;&gt;&gt; '{0}, {1}, {2}'.format('a', 'b', 'c')
'a, b, c'
&gt;&gt;&gt; '{}, {}, {}'.format('a', 'b', 'c')  # 3.1+ only
'a, b, c'
&gt;&gt;&gt; '{2}, {1}, {0}'.format('a', 'b', 'c')
'c, b, a'
&gt;&gt;&gt; '{0}{1}{0}'.format('abra', 'cad')   # arguments' indices can be repeated
'abracadabra'
</code></pre>
<p>I would also like to use the Python formatting with the named placeholders (instead of numeric indices). In Python, the <a href="http://docs.python.org/3/library/stdtypes.html#str.format_map" rel="nofollow">str.format_map(<em>mapping</em>)</a> method of strings is used where the <em>mapping</em> is of the <code>map&lt;string, some_type&gt;</code> like type. Say, <code>my_dictionary</code> contains mapping like:</p>
<pre><code>"name"    --&gt; "Guido"
"country" --&gt; "Netherlands"
</code></pre>
<p>Having a <code>template</code> string like below, I would like to get the <code>result</code>... </p>
<pre><code>wxString template("{name} was born in {country}.");
wxString result(format_map(s, my_dictionary));

// the result should contain...
// "Guido was born in Netherlands."
</code></pre>
<p>Do you know any avaliable C or C++ code capable of that, or do I have to write my own?</p>
<p>Thanks for your time and experience,</p>
<p>Petr</p>

<div class="post-text" itemprop="text">
<p>Yes. <a href="https://github.com/cppformat/cppformat" rel="nofollow">The fmt library</a> uses format string syntax based on Python's <a href="http://docs.python.org/2/library/stdtypes.html#str.format" rel="nofollow">str.format</a>. It supports most of the formatting options of <code>str.format</code> including named arguments. Here's an example:</p>
<pre><code>std::string s = fmt::format("{0}{1}{0}", "abra", "cad");
// s == "abracadabra"
</code></pre>
<p>Disclaimer: I'm the author of this library.</p>
