<div class="post-text" itemprop="text">
<p>You can convert your list of <code>div</code> elements to list of dates using list comprehension like this to get desired output:</p>
<pre><code>comment_list = comment_container.findAll("div", {"class" : "comment-date"})
comment_dates = [comment.text for comment in comment_list]
print(comment_dates)
</code></pre>
