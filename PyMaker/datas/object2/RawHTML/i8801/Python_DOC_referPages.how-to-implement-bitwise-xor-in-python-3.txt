<div class="post-text" itemprop="text">
<p>You need to use <code>0b...</code> to denote binary number.</p>
<p>Also when converting string to a number using <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow noreferrer"><code>int</code></a>, specify base 2 (the second optional parameter). Otherwise it is interpreted as a decimal number:</p>
<pre><code>w2 = int(w1, 2) ^ 0b10000000
#            ^    ^^------------ binary (= 128 in decimal)
#            |
#            optional argument (base 2)
</code></pre>
<p>And, you don't need to call <code>int</code> for <code>0b10000000</code>, because it's already an int object.</p>
<hr/>
<p><strong>UPDATE</strong> If you want to get a string representation back, use <a href="https://docs.python.org/3/library/stdtypes#str.format" rel="nofollow noreferrer"><code>str.format</code></a> or <a href="https://docs.python.org/3/library/functions.html#format" rel="nofollow noreferrer"><code>format</code></a> with format specify <code>08b</code> (0: leading zeros, 8: at least 8 digits, b: binary)</p>
<pre><code>&gt;&gt;&gt; format(w2, '08b')
00100001
</code></pre>
