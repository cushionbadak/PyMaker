<div class="post-text" itemprop="text">
<p>The first <code>split</code> is from the <a href="https://docs.python.org/3/library/re.html#re.split" rel="noreferrer"><code>re</code> module</a></p>
<pre><code>re.split(pattern, string, maxsplit=0, flags=0)
</code></pre>
<p>The second is a <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="noreferrer"><code>str</code> method</a></p>
<pre><code>str.split(sep=None, maxsplit=-1)
</code></pre>
<p>The way you call the <code>str.split</code> method is off a <code>str</code> object like the following</p>
<pre><code>&gt;&gt;&gt; s = 'this is a string'
&gt;&gt;&gt; s.split(' ')
['this', 'is', 'a', 'string']
</code></pre>
