<p>sklearn is a package. It contains the subpackage linear_model, which contains some modules and the class LinearRegression.</p>
<p>You can read more about this structure here:
<a href="https://stackoverflow.com/a/18555203/5952674">https://stackoverflow.com/a/18555203/5952674</a></p>

<span class="comment-copy">It's simply a directory in the sklearn library that Python iterprets as a package. If you look in their <a href="https://github.com/scikit-learn/scikit-learn/tree/master/sklearnl" rel="nofollow noreferrer">GitHub repo</a>, you'll see a folder called <code>linear_model</code> and in that folder is a <code>base.py</code> that defines the <code>LinearRegression</code> class.
