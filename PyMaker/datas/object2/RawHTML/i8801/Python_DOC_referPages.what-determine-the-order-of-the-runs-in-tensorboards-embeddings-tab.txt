<div class="post-text" itemprop="text">
<p>The TensorBoard embedding project backend <a href="https://github.com/tensorflow/tensorflow/blob/100552f943c78cbf90aad521f9981df9b5e3c738/tensorflow/tensorboard/plugins/projector/projector_plugin.py#L277" rel="nofollow noreferrer">serves the runs</a> as a list of keys in a Python <code>dict</code> object:</p>
<pre><code>@wrappers.Request.application
def _serve_runs(self, request):
  """Returns a list of runs that have embeddings."""
  return Respond(request, list(self.configs.keys()), 'application/json')
</code></pre>
<p>Quoting the Python <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">documentation for Dictionaries</a>:</p>
<blockquote>
<p>Performing list(d.keys()) on a dictionary returns a list of all the keys used in the dictionary, in arbitrary order (if you want it sorted, just use sorted(d.keys()) instead).</p>
</blockquote>
