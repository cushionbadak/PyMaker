<p>I am creating a string buffer using ctypes library in python.
Now if I have to print contents of this string buffer later when it has been written, how will i achieve that in python? </p>
<pre><code>import ctypes
init_size = 256
pBuf = ctypes.create_string_buffer(init_size)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>.value</code> and <code>.raw</code> properties to access and manipulate them. This is documented far down in <a href="https://docs.python.org/3/library/ctypes.html?highlight=ctypes.create_string_buffer#fundamental-data-types" rel="nofollow noreferrer">this part of the <code>ctypes</code> docs</a>.</p>
<p>Here is some of the example code from that section:</p>
<pre><code>&gt;&gt;&gt; from ctypes import *
&gt;&gt;&gt; p = create_string_buffer(3)            # create a 3 byte buffer, initialized to NUL bytes
&gt;&gt;&gt; print(sizeof(p), repr(p.raw))
3 b'\x00\x00\x00'
&gt;&gt;&gt; p = create_string_buffer(b"Hello")     # create a buffer containing a NUL terminated string
&gt;&gt;&gt; print(sizeof(p), repr(p.raw))
6 b'Hello\x00'
&gt;&gt;&gt; print(repr(p.value))
b'Hello'
&gt;&gt;&gt; p = create_string_buffer(b"Hello", 10) # create a 10 byte buffer
&gt;&gt;&gt; print(sizeof(p), repr(p.raw))
10 b'Hello\x00\x00\x00\x00\x00'
&gt;&gt;&gt; p.value = b"Hi"
&gt;&gt;&gt; print(sizeof(p), repr(p.raw))
10 b'Hi\x00lo\x00\x00\x00\x00\x00'
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>According to the Python Ctypes docs at:
<a href="https://docs.python.org/2/library/ctypes.html" rel="nofollow noreferrer">https://docs.python.org/2/library/ctypes.html</a></p>
<p>You should be able to print  the string buffer value using the .value object property ie: </p>
<pre><code>print repr(pBuf.value) 
</code></pre>
<p>Or if you want to get a bit fancy with the io you can use something like:</p>
<pre><code>print "pBuff: %s" % pBuf.value
</code></pre>
</div>
