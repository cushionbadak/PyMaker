<p>Let's say I have the following code:</p>
<pre><code>def validator(blacklist: list=['heck', 'muffins']):

    def f(questionable_word: str) -&gt; bool:
        return questionable_word in blacklist

    return f

validator_british = validator(['pish'])
validator_british('pish')  # returns True
validator_british('heck')  # returns False
</code></pre>
<p>My question is how do I type-hint the <code>validator</code> function such that it indicates a function is returned, and specifically a function that takes a <code>str</code> and returns a <code>bool</code>? The <code>f</code> function's signature is:</p>
<pre><code>def f(questionable_word: str) -&gt; bool
</code></pre>
<p>What do I put in the <code>???</code> place for <code>validator</code>?</p>
<pre><code>validator(blacklist: list=['heck', 'muffins']) -&gt; ???:
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/typing.html#typing.Callable" rel="nofollow noreferrer"><code>typing.Callable</code></a> is what you want:</p>
<pre><code>validator(blacklist: list=['heck', 'muffins']) -&gt; Callable[[str], bool]:
</code></pre>
</div>
