<div class="post-text" itemprop="text">
<p>To continue querying the API after an error is raised, you need to <a href="https://docs.python.org/3/library/exceptions.html#index-0" rel="nofollow noreferrer">catch and handle</a> the exception.</p>
<pre><code>def stockplotter(ticker,start,end,numb):
    try:
        stock = pdr.get_data_yahoo(ticker,start,end)
    except RemoteDataError:
        pass

    plt.figure(numb)
    stock["Adj Close"].plot(grid = True)
    plt.title(ticker)
    plt.ylabel('SEK@close')
</code></pre>
