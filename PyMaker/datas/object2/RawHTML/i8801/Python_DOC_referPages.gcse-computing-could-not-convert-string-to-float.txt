<div class="post-text" itemprop="text">
<p><code>float()</code> can convert a string representing a number to a floating point number.</p>
<p>E.g: <code>"3"</code> is converted to <code>3.0</code></p>
<p>When you pass a string of numbers it can't tell what is a number from another.</p>
<p>E.g: <code>"3 4 50 200000 3.0E20"</code> </p>
<p>If you want to convert all numbers in a string, you might want to do something like that:</p>
<pre><code>my_numbers = "3 4 5 6"
my_numbers = my_numbers.split()
my_floats = []
for num in my_numbers:
    my_floats.append(float(num))
</code></pre>
<p>The result is <code>[3.0, 4.0, 5.0, 6.0]</code></p>
<p>More about <code>float</code> here: <a href="https://docs.python.org/3/library/functions.html#float" rel="nofollow noreferrer">https://docs.python.org/3/library/functions.html#float</a></p>
<p>More about <code>split</code> here: <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html#str.split</a></p>
