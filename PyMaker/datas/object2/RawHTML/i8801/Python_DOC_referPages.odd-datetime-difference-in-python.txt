<div class="post-text" itemprop="text">
<p>As you can read <a href="https://docs.python.org/3/library/datetime.html#datetime.timedelta" rel="nofollow noreferrer">here</a> a <code>timedelta</code> object has three fields: <code>days</code>; <code>seconds</code>; and <code>microseconds</code>. Or as is specified in the documentation:</p>
<blockquote>
<p><strong><code>class datetime.timedelta(days=0, seconds=0, microseconds=0, milliseconds=0, minutes=0, hours=0, weeks=0)</code></strong></p>
<p>(...)</p>
<p><strong>Only days, seconds and microseconds are stored internally</strong>. Arguments are converted to those units:</p>
<ul>
<li>A millisecond is converted to 1000 microseconds.</li>
<li>A minute is converted to 60 seconds.</li>
<li>An hour is converted to 3600 seconds.</li>
<li>A week is converted to 7 days.</li>
</ul>
</blockquote>
<p>(formatting added)</p>
<p>Althout the constructor is <code>timedelta(days=0, seconds=0, microseconds=0, milliseconds=0, minutes=0, hours=0, weeks=0)</code> and thus provides ways to enter hours, it thus converts the <code>minutes</code>, <code>hours</code>, etc. all to seconds. The constructor will look like:</p>
<pre><code>def __init__(self, days=0, seconds=0, microseconds=0, milliseconds=0, minutes=0, hours=0, weeks=0):
    self.microseconds = microseconds+1000*milliseconds
    self.seconds = seconds+60*minutes+3600*hours+self.microseconds//1000000
    self.microseconds %= 1000000
    self.days = days+7*weeks+self.seconds//86400
    self.seconds %= 86400
</code></pre>
<p>(but probably a bit more advanced, etc.)</p>
<p>So that means that <code>.seconds</code> actually is modulo day, and without microseconds.</p>
<p>You can however use <a href="https://docs.python.org/3/library/datetime.html#datetime.timedelta.total_seconds" rel="nofollow noreferrer"><code>timediff.total_seconds</code></a> to return the total amount of seconds:</p>
<pre><code>&gt;&gt;&gt; timediff.total_seconds()
87710.0
</code></pre>
<p>So <code>total_seconds()</code> is basically:</p>
<pre><code>def total_seconds(self):
    return 86400.0*self.days+self.seconds+1e-6*self.microseconds
    #      ^ number of seconds in a day   ^
    #                                     | 1 micro is 1e-6
</code></pre>
<p>If you divide your <code>1310</code> by <code>60</code>, you will see that it returns:</p>
<pre><code>&gt;&gt;&gt; 1310/60 # obtain number of minutes
21.833333333333332
&gt;&gt;&gt; 1310%60 # obtain number of seconds  (without minutes)
50
</code></pre>
<p>so 21 minutes and 50 seconds</p>
