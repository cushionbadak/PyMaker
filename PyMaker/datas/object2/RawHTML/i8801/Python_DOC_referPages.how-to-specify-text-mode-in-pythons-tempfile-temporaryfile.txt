<div class="post-text" itemprop="text">
<p>In the <a href="https://docs.python.org/3/library/tempfile.html" rel="nofollow noreferrer">documentation</a>, TemporaryFile() mentions </p>
<blockquote>
<p>The returned object is a file-like object whose _file attribute is either an io.BytesIO or io.StringIO object (depending on whether binary or text mode was specified)...</p>
</blockquote>
<p>However, as of Python 3.6, there is no <code>text=True</code> method of asking for text mode.   How does one do it?</p>
</div>
<div class="post-text" itemprop="text">
<p>TemporaryFile signature:</p>
<pre><code> tempfile.TemporaryFile(mode='w+b', buffering=None, encoding=None, newline=None, suffix=None, prefix=None, dir=None)
</code></pre>
<p>We are interested in a <code>mode</code> parameter. The last symbol is 'b', which means binary mode. If you pass <code>mode='wt'</code>, it will be opened in a text mode.</p>
<p>All modes are described on this page: <a href="https://docs.python.org/3/library/functions.html#open" rel="noreferrer">https://docs.python.org/3/library/functions.html#open</a></p>
</div>
