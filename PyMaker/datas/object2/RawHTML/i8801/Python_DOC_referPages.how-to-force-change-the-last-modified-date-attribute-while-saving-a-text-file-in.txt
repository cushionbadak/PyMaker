<p>I am saving a text file in python after processing it as follows:</p>
<pre><code>outputfile = dirname.split(".")[0] + ".txt"
ProcessedFile.to_csv(outputfile, esp='')
</code></pre>
<p>This file takes by default today's date as the last modified date. Is there a way I can change this and force-give it my own desired last modified date?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>outputfile = dirname.split(".")[0] + ".txt"
ProcessedFile.to_csv(outputfile, esp='')
os.utime(outputfile, (56000000, 56000000))
</code></pre>
<p>In os.utime(outputfile, (atime_ns, mtime_ns)), atime_ns is access time in nanoseconds and mtime_ns is modified time in nanoseconds.</p>
<p>In this example, 56000000 corresponds to the following:</p>
<p>Date in India timezone*: 10/11/1971, 9:03:20 AM</p>
<p>Date in Los Angeles*: 10/10/1971, 8:33:20 PM</p>
<p>Date in Berlin* :10/11/1971, 9:03:20 AM</p>
<p>Date in Beijing*: 10/11/1971, 10:33:20 AM</p>
<p>Date in New York* :10/10/1971, 10:33:20 PM</p>
<p>It will change last modified and access date of the file you are dealing with. You may find the reference for os.utime on <a href="https://docs.python.org/3/library/os.html" rel="nofollow noreferrer">https://docs.python.org/3/library/os.html</a></p>
<p>Hope it helps.</p>
