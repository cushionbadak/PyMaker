<div class="post-text" itemprop="text">
<p>A more concise solution would be:</p>
<pre><code>lengths = [[len(w) for w in s.split()] for s in lucky]
</code></pre>
<p>Output:</p>
<pre><code>[[4, 2, 3, 5, 3, 3, 3], [4, 2, 3, 5, 3, 3, 3], [3, 2, 3, 5, 2, 3, 4], [3, 2, 3, 5, 2, 3, 5]]
</code></pre>
<p>Explanation:</p>
<p><code>for s in lucky</code> will iterate through all strings in lucky. With <code>s.split()</code> we then split each string <code>s</code> into the words it is made up of. Using <code>len(w)</code>, we then obtain the length (number of characters) for each word <code>w</code> in <code>s.split()</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://stackoverflow.com/questions/42608501/python-function-for-list-of-lists/42608844#comment72346651_42608501">The comment</a> in your question gives you the reason why your code is failing. Here is another solution leveraging the use of <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer">map</a>: </p>
<p>Python 3, you get <code>map</code> objects that you will have to call <code>list</code> on if you want to see the output as you expect.</p>
<pre><code>&gt;&gt;&gt; res = [list(map(len, x.split())) for x in lucky]
&gt;&gt;&gt; res
[[4, 2, 3, 5, 3, 3, 3], [4, 2, 3, 5, 3, 3, 3], [3, 2, 3, 5, 2, 3, 4], [3, 2, 3, 5, 2, 3, 5]]
</code></pre>
<p>Python 2 will give you a list from calling <code>map</code>:</p>
<pre><code>&gt;&gt;&gt; res = [map(len, x.split()) for x in lucky]
&gt;&gt;&gt; res
[[4, 2, 3, 5, 3, 3, 3], [4, 2, 3, 5, 3, 3, 3], [3, 2, 3, 5, 2, 3, 4], [3, 2, 3, 5, 2, 3, 5]]
</code></pre>
</div>
