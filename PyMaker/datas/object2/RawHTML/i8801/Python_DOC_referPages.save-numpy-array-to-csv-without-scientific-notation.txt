<p>The default write:</p>
<pre><code>In [154]: np.savetxt('test.txt',(AVG, MN, MX), delimiter=',')
In [155]: cat test.txt
3.333999999999999631e+01
3.332999999999999829e+01
3.336999999999999744e+01
</code></pre>
<p>write with a custom <code>fmt</code>:</p>
<pre><code>In [156]: np.savetxt('test.txt',(AVG, MN, MX), delimiter=',', fmt='%f')
In [157]: cat test.txt
33.340000
33.330000
33.370000
</code></pre>
<p>Or if you want values on one line, make it a 2d array (or equivalent with extra []), so it writes one row per line:</p>
<pre><code>In [160]: np.savetxt('test.txt',[[AVG, MN, MX]], delimiter=',', fmt='%f')
In [161]: cat test.txt
33.340000,33.330000,33.370000
</code></pre>
<p>There are other parameters that you can experiment with.</p>

<p>You can use the CSV module</p>
<pre><code>import csv
f = open('New file.csv',"wb")
writer = csv.writer(f)
for row in [AVG, MN, MX]:
    writer.writerow(row)

f.close()
</code></pre>
