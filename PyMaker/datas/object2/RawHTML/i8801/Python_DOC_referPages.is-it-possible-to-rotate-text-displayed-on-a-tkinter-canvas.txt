<p>Since people are being linked to this answer, I'll add an update. </p>
<p><a href="http://www.tcl.tk/cgi-bin/tct/tip/119" rel="noreferrer">In tcl 8.6, the create_text method got an <code>angle</code> option</a>. You can check your tcl version with <code>Tkinter.TclVersion</code>. If you have 8.6 or greater, you can use:</p>
<pre><code>textID = w1.create_text(5, 5, anchor="nw", angle=90)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you are asking whether <a href="http://effbot.org/tkinterbook/canvas.htm#Tkinter.Canvas.create_text-method" rel="nofollow noreferrer"><code>tkinter.Canvas.create_text</code></a> has something like this:</p>
<pre><code>textID = w1.create_text(5, 5, anchor="nw", orient=tkinter.VERTICAL)
</code></pre>
<p>then the answer is no.  The <code>create_text</code> method can only create horizontal text.</p>
<hr/>
<p>However, you <em>can</em> use <a href="http://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>str.join</code></a> to create vertical text:</p>
<pre><code>from tkinter import Tk, Canvas
root = Tk()
canvas = Canvas()
canvas.grid()
canvas.create_text((10, 5), text="\n".join("This is some text"), anchor="nw")
root.mainloop()
</code></pre>
<p>Example:</p>
<p><img alt="enter image description here" src="https://i.stack.imgur.com/Cmd87.png"/></p>
<p>While this may not be as elegant as simply setting an option on the <code>create_text</code> method, it does work.</p>
</div>

<ol>
<li>Screen shot the text you want and crop it </li>
<li>Screen shot the background colour and crop it </li>
<li>In word or powerpoint or something, put the text over the background </li>
<li>Take another screenshot and crop that </li>
<li>Put that screen shot in paint </li>
<li>Save the paint file in the same folder as your program</li>
<li>convert the paint file to a gif online: 
<a href="http://image.online-convert.com/convert-to-gif" rel="nofollow noreferrer">http://image.online-convert.com/convert-to-gif</a></li>
<li>Use the following code to put the image in your program</li>
</ol>
