<div class="post-text" itemprop="text">
<p>try with</p>
<pre><code>import scipy.integrate as integrate

exact = integrate.ode(eq1)   #notice, no scipy
print(exact)
</code></pre>
<p>the problem is that you <a href="https://docs.python.org/3/reference/simple_stmts.html#the-import-statement" rel="nofollow noreferrer">import</a> the module <code>scipy.integrate</code> and bound it to the variable <code>integrate</code> with the instruction <code>as</code>, that is why you get the name error in <code>scipy.integrate.ode(eq1)</code>, <code>scipy</code> is not in your namespace, just <code>integrate</code>; </p>
<p>if you want to include <code>scipy</code> it then import it as </p>
<pre><code>import scipy
</code></pre>
<p>so you can use its other features</p>
<pre><code>import scipy.integrate as integrate
import scipy
exact = integrate.ode(eq1)
print(exact)
</code></pre>
<p>or just without the <code>as</code></p>
<pre><code>import scipy.integrate 

exact = scipy.integrate.ode(eq1)
print(exact)
</code></pre>
