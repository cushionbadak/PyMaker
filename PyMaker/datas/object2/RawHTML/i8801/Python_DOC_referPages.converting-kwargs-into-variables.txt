<p>How do I convert kwargs objects into local variables?
I am a math teacher and I want to write a helper function where I can use JS-style template strings to write problems.</p>
<p>I want to do something like this:</p>
<pre><code>TemplateString('{{a*b}} apples are shared among {{a}} children. How many apples do each child get?', {'a': 3, 'b': 5})
//return '15 apples are shared among 3 children...'
</code></pre>
<p>I am going to search for '{{}}' in strings and call eval() on what is inside that. But first I need to convert {'a': 3, 'b': 5} into local variables a=3 and b=5. The variables could be int, float or str. What is the best way to do this?</p>

<div class="post-text" itemprop="text">
<p>If you're planning to use <code>eval</code>, you don't need to convert them to locals at all. <a href="https://docs.python.org/3/library/functions.html#eval" rel="nofollow noreferrer"><code>eval</code> takes optional <code>globals</code> and/or <code>locals</code> arguments</a> that defines the active scope variables for that expression. If you provide <code>globals</code> (the second positional argument) and not <code>locals</code> (the third), then <code>globals</code> is reused as <code>locals</code>, so for example:</p>
<pre><code>&gt;&gt;&gt; eval('a * b', {'a': 5, 'b': 3})
15
</code></pre>
</div>
<span class="comment-copy">Is your dictionary always have 2 elements, <code>a</code> and <code>b</code>? If so, just simply do <code>int(dict['a'])*int(dict['b'])</code>. Is that what you want?</span>
