<p>My suggestion is using <code>json.loads()</code></p>
<pre><code>&gt;&gt;&gt; sms_param = '{\"website\":\"hello\"}'
&gt;&gt;&gt; import json
&gt;&gt;&gt; json.loads(sms_param)
{'website': 'hello'}
</code></pre>
<p>What you can do is using <code>json.loads()</code> convert the json string to dictionary and then change the value, finally convert it back to string</p>
