<p>The function names in python are case sensitive, the function is <code>isfile</code> not <code>isFile</code> . </p>
<p>reference link : <a href="https://docs.python.org/2/library/os.path.html#os.path.isfile" rel="nofollow noreferrer">os.path.isfile</a></p>
<pre><code>#!/usr/bin/python
import sys
import os.path
def main(argv):
    if len(sys.argv) &lt; 3:
        print "insuficient arguments. Use like: lichcrypt &lt;d/e&gt; &lt;/path/to/file&gt;"
    else:
        operation = argv[1]
        filepath = argv[2]
        print operation
        print os.path.isfile(filepath)
main(sys.argv)
</code></pre>
