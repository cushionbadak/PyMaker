<p>For example, I have a List with two entries:</p>
<pre><code>list1=[['printer1', '172.1.1.1', '12345678', 'Xerox', 'Xerox Phaser ', 'Team@Departemnt'],['printer2', '172.1.1.2', '12345678', 'Xerox', 'Xerox Ricoh ', 'Team1@Departemnt1']]
</code></pre>
<p>And i want to modify some entries For example i want to split the Department entries and add two more column in the List, with a loop</p>
<pre><code>cache= list1[0][5].split('@')
list1[0][6]= cache[0]

<div class="post-text" itemprop="text">
<p>Your data is in an ideal format for using <code>pandas</code>:<br/>
This method doesn't use any loops, though.</p>
<pre><code>import pandas as pd

df = pd.DataFrame(list1, columns=['name', 'ip', 'number', 'manufacturer', 'device', 'dept'])
print(df)
       name         ip    number manufacturer         device               dept
0  printer1  172.1.1.1  12345678        Xerox  Xerox Phaser     Team@Departemnt
1  printer2  172.1.1.2  12345678        Xerox   Xerox Ricoh   Team1@Departemnt1
</code></pre>
<p>Then you can split on the <code>@</code>:</p>
<pre><code>df['team'], df['dept'] = df['dept'].str.split('@', 1).str
print(df)
       name         ip    number manufacturer         device         dept   team
0  printer1  172.1.1.1  12345678        Xerox  Xerox Phaser    Departemnt   Team
1  printer2  172.1.1.2  12345678        Xerox   Xerox Ricoh   Departemnt1  Team1
</code></pre>
<p>If you need it back as lists you can use:</p>
<pre><code>df.values.tolist()
[['printer1', '172.1.1.1', '12345678', 'Xerox', 'Xerox Phaser ', 'Departemnt', 'Team'], 
 ['printer2', '172.1.1.2', '12345678', 'Xerox', 'Xerox Ricoh ', 'Departemnt1', 'Team1']]
</code></pre>
<p>But you can easily keep manipulating the data as a table with pandas</p>

<div class="post-text" itemprop="text">
<p>If you want to modify the lists in-place, you could use <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer">extend</a>:</p>
<pre><code>list1 = [['printer1', '172.1.1.1', '12345678', 'Xerox', 'Xerox Phaser ', 'Team@Departemnt'],
         ['printer2', '172.1.1.2', '12345678', 'Xerox', 'Xerox Ricoh ', 'Team1@Departemnt1']]


for l in list1:
    l.extend(l[-1].split('@'))


print(list1)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[['printer1', '172.1.1.1', '12345678', 'Xerox', 'Xerox Phaser ', 'Team@Departemnt', 'Team', 'Departemnt'], ['printer2', '172.1.1.2', '12345678', 'Xerox', 'Xerox Ricoh ', 'Team1@Departemnt1', 'Team1', 'Departemnt1']]
</code></pre>
<p>As an alternative if you want a new list, you could add the split to the list:</p>
<pre><code>result = [l + l[-1].split('@') for l in list1]
</code></pre>
