<div class="post-text" itemprop="text">
<p>For a single parameter, you can use <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.urlparse" rel="nofollow noreferrer"><code>urllib.parse</code></a> from the standard library:</p>
<pre><code>from urllib.parse import urlparse

str1 = "https://web2.some.com/hbh/productImages?itemId=5986546"

item1 = urlparse(str1).query.split('=')[-1]  # '5986546'
</code></pre>
<p>For multiple parameters, you can construct a dictionary via <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.parse_qs" rel="nofollow noreferrer"><code>urllib.parse_qs</code></a>:</p>
<pre><code>from urllib.parse import urlparse, parse_qs

str2 = "https://web2.some.com/hbh/productImages?itemId=5986546&amp;somevar=5"

args = parse_qs(urlparse(str2).query)
item2 = args['itemId']  # '5986546'
</code></pre>
</div>

<p>As your URLs do not include more than one <code>=</code>, you can use <code>str.split</code></p>
<pre><code>id = str1.split('=')[-1] # or [1] in this case no diff
</code></pre>

<div class="post-text" itemprop="text">
<p>Use regex:</p>
<pre><code>import re 

str1 = "https://web2.some.com/hbh/productImages?itemId=5986546"
str2 = "https://web2.some.com/hbh/productImages?itemId=5986546231"
str3 = "https://web2.some.com/hbh/productImages?itemId=22432"

regex = re.compile(r'(\d+?)$')
l = regex.findall(str1)
print(l)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>C:\Users\Desktop&gt;py x.py
['5986546']
</code></pre>
<p>Moreover, the below code will return all in once:</p>
<pre><code>all_strings = ''.join( [str1,str2,str3])

regex = re.compile(r'(\d{2,})')
l = regex.findall(all_strings)
print(l)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>C:\Users\Desktop&gt;py x.py
['5986546', '5986546231', '22432']
</code></pre>
