<p>First thing is to remember that python uses zero indexing.</p>
<p>You can iterate throught the list except using the range function to get the indexes of the items you want or slices to get the elements.</p>
<p>What I think is becoming confusing here is that in your example, the values and the indexes are the same so to clarify I'll use this list as example:</p>
<pre><code>I = ['a', 'b', 'c', 'd', 'e']
nI = len(I) # 5
</code></pre>
<p>The range function will allow you to iterate through the indexes:</p>
<pre><code>for i in range(1, nI):
    print(i)
# Prints:
# 1
# 2
# 3
# 4
</code></pre>
<p>If you want to access the values using the range function you should do it like this:</p>
<pre><code>for index in range(1, nI):
    i = I[index]
    print(i)
# Prints:
# b
# c
# d
# e
</code></pre>
<p>You can also use array slicing to do that and you don't even need <code>nI</code>. Array slicing returns a new array with your slice.<br/>
The slice is done with <code>the_list_reference[start:end:steps]</code> where all three parameters are optional and:<br/>
<code>start</code> is the index of the first to be included in the slice<br/>
<code>end</code> is the index of the first element to be excluded from the slice<br/>
<code>steps</code> is how many steps for each next index starting from (as expected) the <code>start</code> (if <code>steps</code> is 2 and start with 1 it gets every odd index).<br/>
Example:</p>
<pre><code>for i in I[1:]:
    print(i)
# Prints:
# b
# c
# d
# e
</code></pre>

<p>If you want to iterate through a list from a second item, just use <code>range(1, nI)</code> (if nI is the length of the list or so). </p>
<pre><code>for i in range(1, nI):
    sum=0
    for v in range(nV):
        for j in range(nJ):
            sum=sum+x1[i][j][v]
</code></pre>
<p>Probaly, a part of your function just lost somewhere, but anyway, in in general range() works like this: </p>
<pre><code>range(start_from, stop_at, step_size)
</code></pre>
<p>i. e. </p>
<pre><code>for i in range(2, 7, 2):
    print(i, end=' ')

Out:
2 4 6
</code></pre>
<p><strong>Edit</strong></p>
<p>Please, remember: python uses zero indexing, i.e. first element has an index 0, the second - 1 etc. </p>
<p>By default, <code>range</code> starts from 0 and stops at the <em>value of the passed parameter minus one.</em> If there's an explicit <strong>start</strong>, iteration starts from it's value. If there's a <strong>step</strong>, it continues while <code>range</code> returns values <em>lesser than</em> <strong>stop</strong> value.</p>
<pre><code>for i in range(1, 7, 2):
    print(i, end=' ')

Out: 
1 3 5  # there's no 7!
</code></pre>
<p>Detailed description of <code>range</code> build-in is <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer">here</a>.</p>
