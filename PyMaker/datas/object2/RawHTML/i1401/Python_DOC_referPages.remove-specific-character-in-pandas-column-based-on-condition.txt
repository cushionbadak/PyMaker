<p>You should use the <a href="https://docs.python.org/3/library/stdtypes.html#str.rstrip" rel="nofollow noreferrer"><code>rstrip</code></a> method, which removes trailing characters:</p>
<pre><code>df['Score'] = df.Score.str.rstrip('.')
