<div class="post-text" itemprop="text">
<p>I have a script. I get this script as a string through the database. I need a specific value in this script. How can I get this data out of string?</p>
<h2>The data(type=str) I extracted from the database</h2>
<pre><code>data = """if(description)
{
  script_name("name");
  script_summary("summary");
  script_version("version");
  script_tag(name:"base_vector", value:"AV:N/AC:L/Au:N/C:C/I:C/A:C");
  exit(0);
}"""
</code></pre>
<h2>The data I want for each script</h2>
<pre><code>base_vector: AV:N/AC:L/Au:N/C:C/I:C/A:C
</code></pre>
<p>This value is changing for each script.</p>
</div>
<div class="post-text" itemprop="text">
<p>Consider using <strong>regular expressions</strong> for that.</p>
<pre><code>import re

rx = r'name\s*\:\s*\"(\w+)\"\s*\,\s*value\s*\:\s*\"(.*)\"'

text = """if(description)
{
  script_name("name");
  script_summary("summary");
  script_version("version");
  script_tag(name:"base_vector", value:"AV:N/AC:L/Au:N/C:C/I:C/A:C");
  exit(0);
}"""

m = re.search(rx, text)
print(m.group(1) + ': ' + m.group(2))
</code></pre>
<p>The output is:</p>
<pre><code>base_vector: AV:N/AC:L/Au:N/C:C/I:C/A:C
</code></pre>
<p>You can enhance regular expression, but provided one will work on your input data.</p>
<p>Note lots of <code>\s*</code> tokens in expression - they allow using spaces in input data.</p>
</div>
