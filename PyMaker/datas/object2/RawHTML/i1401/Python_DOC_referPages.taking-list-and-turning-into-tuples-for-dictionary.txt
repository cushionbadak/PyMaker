<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer">filter</a> with a dictionary comprehension:</p>
<pre><code>data = [['0', '0', '254', '0', 'R'], ['0', '97', '0', '65', 'R'], ['0', '98', '0', '66', 'R'], []]
result = {(a, b): (c, d, e) for a, b, c, d, e in filter(None, data)}
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>{('0', '97'): ('0', '65', 'R'), ('0', '98'): ('0', '66', 'R'), ('0', '0'): ('254', '0', 'R')}
</code></pre>
<p>Or more similar to your code:</p>
<pre><code>rules = [['0', '0', '254', '0', 'R'], ['0', '97', '0', '65', 'R'], ['0', '98', '0', '66', 'R'], []]
result = dict(((a, b), (c, d, e)) for a, b, c, d, e in filter(None, rules))
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>{('0', '0'): ('254', '0', 'R'), ('0', '98'): ('0', '66', 'R'), ('0', '97'): ('0', '65', 'R')}
</code></pre>

<p>How about checking if the size is 5?</p>
<pre><code>l = [
    ['0', '0', '254', '0', 'R'], 
    ['0', '97', '0', '65', 'R'],

    []
]

out = {(x[0],x[1]):(x[2],x[3],x[4]) for x in l if len(x) == 5} # or just if x
print(out)
</code></pre>
