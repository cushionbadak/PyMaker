<p>I have two lists in python</p>
<p>Names:</p>
<pre><code>['Test 1', 'Test 2']
</code></pre>
<p>isTrue:</p>
<pre><code>[False, True]
</code></pre>
<p>I want to merge the lists into one list and the result being:</p>
<pre><code>[{name: 'Test 1', isTrue: False}, {name: 'Test 2', isTrue: True}]
</code></pre>
<p>Here is what I have tried:</p>
<pre><code>list = []
thisdict = {}
for name in names:
    thisdict['name'] = name
    thisdict['isTrue'] = True
    list.append(thisdict)
</code></pre>
<p>The issue is I am not sure how to get the dynamic boolean value or how to do this a lot more effeciently.</p>
<p>Any help will be appreciated.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a list comprehension and <code>zip</code>:</p>
<pre><code>&gt;&gt;&gt; [dict(name=x, isTrue=y) for x, y in zip(names, isTrue)]
</code></pre>
</div>
