<p>As Rup mentioned above, your user agent may have been rejected by the server.</p>
<p>Try augmenting your code with the following:</p>
<pre><code>import urllib.request  # we are going to need to generate a Request object
from bs4 import BeautifulSoup as soup

my_url = "https://www.transfermarkt.com/chelsea-fc/leihspielerhistorie/verein/631/plus/1?saison_id=2018&amp;leihe=ist"

# here we define the headers for the request
headers = {'User-Agent': 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10.13; rv:63.0) Gecko/20100101 Firefox/63.0'}

# this request object will integrate your URL and the headers defined above
req = urllib.request.Request(url=my_url, headers=headers)

# calling urlopen this way will automatically handle closing the request
with urllib.request.urlopen(req) as response:
    page_html = response.read()
</code></pre>
<p>After the code above you can continue your analysis. The Python docs have some useful pages on this topic:</p>
<p><a href="https://docs.python.org/3/library/urllib.request.html#examples" rel="nofollow noreferrer">https://docs.python.org/3/library/urllib.request.html#examples</a></p>
