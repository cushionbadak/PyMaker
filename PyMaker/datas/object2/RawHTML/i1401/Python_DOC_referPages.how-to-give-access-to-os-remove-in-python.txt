<pre><code>os.remove(example)
</code></pre>

<div class="post-text" itemprop="text">
<p>If it is a file in read only mode, change the file permission to 0777 with os.chmod() function and then try to remove it with os.remove().</p>
<pre><code>os.chmod(filePath, 0777)
os.remove(filePath)
</code></pre>
<p>If it is a directory, do the same but with os.removedir() instead of os.remove().</p>
<pre><code>os.chmod(dirPath, 0777)
os.removedir(dirPath)
</code></pre>
