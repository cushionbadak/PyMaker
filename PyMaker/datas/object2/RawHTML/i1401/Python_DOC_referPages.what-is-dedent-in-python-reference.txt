<div class="post-text" itemprop="text">
<p>It's not a character per se - it is a token that represents the fact that the current line is indented by less spaces than the one before.</p>
<p>So for example the code:</p>
<pre><code>foo
bar:
  bay
baz
</code></pre>
<p>would be tokenized as <code>ID(foo), NEWLINE, ID(bar), NEWLINE, INDENT, ID(bay), DEDENT, ID(baz)</code></p>
</div>
