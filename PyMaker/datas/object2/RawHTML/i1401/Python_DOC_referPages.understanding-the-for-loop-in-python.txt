<div class="post-text" itemprop="text">
<p>This is called list comprehension, you can see the link for more details.<a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">Ref</a></p>
<pre><code># the code you listed above translate to this
ordered_list = []
for i in range(0, x+1):
    for j in range(0, y+1):
         for k in range(0, z+1):
            if i + j + k != n:
                ordered_list.append([i, j, k])
print(ordered_list) 
</code></pre>

<div class="post-text" itemprop="text">
<p>That is a list comprehension equivalent to: </p>
<pre><code>ordered_list = [] 
for i in range(0, x+1): 
    for j in range(0, y+1): 
        for k in range(0, z+1):
             if i + j + k != n:
                 ordered_list.append([i, j, k])
</code></pre>
