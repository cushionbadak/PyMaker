<div class="post-text" itemprop="text">
<p>Provided my float number is <code>12.3456</code>, and I need it to be printed as <code>_12.34</code>, which means that the length of the number before <code>.</code> is 3 while the length after the <code>.</code> is still 2. I tried this <code>'{:.2f}'.format(12.3456)</code> which only give me <code>12.34</code> which shorts a space before number <code>1</code>. So how could I make my need possible ?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can specify the total width of the rendered string by putting a number before the period in the format string:</p>
<pre><code>&gt;&gt;&gt; '{:6.2f}'.format(12.3456)
' 12.35'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can apply an additional <code>.rjust(6)</code> (6 because 3 + 2 digits and a decimal point):</p>
<pre><code>&gt;&gt;&gt; '{:.2f}'.format(12.3456).rjust(6)
' 12.35'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>How about first converting the float to 2 decimal places then formatting it?
For example if you want to truncate it use math.floor, or use round() if you want to round or use the decimal module (import decimal) <a href="https://docs.python.org/2/library/decimal.html" rel="nofollow noreferrer">python decimal module</a></p>
<p>Example code using math.floor:</p>
<pre><code>import math
print('{0:_&gt;6}'.format(math.floor(112.345*100)/100)) #112.34
print('{0:_&gt;6}'.format(math.floor(12.345*100)/100))  #_12.34
</code></pre>
</div>
