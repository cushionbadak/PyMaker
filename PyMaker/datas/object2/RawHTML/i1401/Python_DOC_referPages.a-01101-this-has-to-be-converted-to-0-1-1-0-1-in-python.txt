<div class="post-text" itemprop="text">
<p>There is a function called list to do this directly.This can convert any string into list of characters.</p>
<pre><code>list('01101')
</code></pre>
<p>will return </p>
<pre><code>['0', '1', '1', '0', '1']
</code></pre>
<p>One more way is</p>
<pre><code>a = '01101'
a_list=[]
for item in a:
    a_list.append(item)
print(a_list)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#list" rel="nofollow noreferrer">list</a> does exactly that:</p>
<blockquote>
<p>The constructor builds a list whose items are the same and in the same
  order as iterableâ€™s items</p>
</blockquote>
<pre><code>a = '01101'
print(list(a))
</code></pre>
<p>Output:</p>
<pre><code>['0', '1', '1', '0', '1']
</code></pre>
