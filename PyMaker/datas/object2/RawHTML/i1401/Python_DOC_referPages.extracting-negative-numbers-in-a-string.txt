<div class="post-text" itemprop="text">
<p>If the string is a valid Python tuple, you could use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow noreferrer">ast.literal_eval</a>:</p>
<pre><code>import ast

ts = '((-1,-5,-1),(3,1,-3),(4,-1))'
ts = ast.literal_eval(ts)

numbers = [e for t in ts for e in t]
print(numbers)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[-1, -5, -1, 3, 1, -3, 4, -1]
</code></pre>
<p>Or, you could use regular expressions through the built-in <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">re</a> module:</p>
<pre><code>import re

numbers = re.compile('-?\d+')
ts = '((-1,-5,-1),(3,1,-3),(4,-1))'
result = list(map(int, numbers.findall(ts)))
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[-1, -5, -1, 3, 1, -3, 4, -1]
</code></pre>
<p>The pattern <code>-?\d+</code> means match a group of digits (<code>\d+</code>) that can possibly be preceded by a minus sign (<code>-?</code>).</p>
