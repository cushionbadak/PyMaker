<div class="post-text" itemprop="text">
<p>create a file named <code>__init__.py</code> inside directory OrderBook and Rabbit</p>
<p>this will create package, and then you can import </p>
<p><a href="https://docs.python.org/3/tutorial/modules.html#packages" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/modules.html#packages</a></p>
<p>so your directory structure will be looked like :</p>
<pre><code>|-Server/
|-------OrderBook/
|       |--------__init__.py
|       |--------message.py
|       |--------orderBookObject.py
|-------Rabbit/
|       |--------__init__.py
|       |--------emisor.py
|       |--------receptor.py
|-------server.py
|-------processMessage.py
</code></pre>
