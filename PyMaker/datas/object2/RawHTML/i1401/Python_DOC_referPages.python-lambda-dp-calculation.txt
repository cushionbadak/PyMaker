<div class="post-text" itemprop="text">
<p>A lot of things are happening here. 
The <code>min()</code> method is receiving a python list as an argument. </p>
<p>This python list is created through <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>. It helps to start from the back. The list comprehension is saying "For every value <code>c</code> in the iterable <code>coins</code>, append the value <code>dp[i - c]</code> to my list if the boolean <code>i - c &gt;=0</code> is <code>True</code>. Otherwise, append the value <code>MAX</code>. </p>
<p>In other words, the line <code>[dp[i - c] if i - c &gt;= 0 else MAX for c in coins]</code> is equivalent to</p>
<pre><code>def makelist(dp,i,MAX,coins):
    ret = []
    for c in coins:
        if i - c &gt;= 0:
            ret.append(dp[i-c])
        else:
            ret.append(MAX)
    return ret
</code></pre>
<p>The <code>min()</code> method will be performed on this list once it is complete, which simply find the smallest value in the list</p>
