<div class="post-text" itemprop="text">
<p>Assuming you have (note that they're strings):</p>
<pre><code>start_date = '2018-12-01'
end_date = '2018-12-03'
</code></pre>
<p>Then you need to import the <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer"><code>datetime</code></a> module and <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strptime" rel="nofollow noreferrer">parse the strings</a> as datetime objects:</p>
<pre><code>import datetime
st_dates = datetime.datetime.strptime(start_date, '%Y-%m-%d')
en_dates = datetime.datetime.strptime(end_date, '%Y-%m-%d')
</code></pre>
<p>Then do the difference like you've done:</p>
<pre><code>ss = st_dates - en_dates
print(ss.days)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to import the datetime module. For dates you can use <code>datetime.datetime</code>.</p>
<pre><code>from datetime import datetime
start_date = datetime(2018, 12, 1)
end_date = datetime(2018, 12, 3)
ss = start_date - end_date 
print(ss.days)
</code></pre>
<p>or</p>
<pre><code>from datetime import datetime
st_dates = datetime.strptime('2018-12-01', '%Y-%m-%d')
en_dates = datetime.strptime('2018-12-03', '%Y-%m-%d')
ss = st_dates - en_dates
print(ss.days)
</code></pre>
