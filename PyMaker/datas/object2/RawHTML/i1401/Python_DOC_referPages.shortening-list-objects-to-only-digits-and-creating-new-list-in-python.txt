<p>I have a list which has a few hundred dates in it but I need to filter out only the time in the format hhmmss (no : in between so I can turn them into int later on).</p>
<p>The list I have so far looks like this:</p>
<pre><code>list_dates = ['Fri Nov 30 15:56:43 +0000 2018', 'Fri Nov 30 15:56:44 +0000 2018', 'Fri Nov 30 15:56:45 +0000 2018', 'Fri Nov 30 15:56:46 +0000 2018', ... ]
</code></pre>
<p>My approach so far is:</p>
<pre><code>raw_list = []
new_list = []

def ConversionStrings():

    j = 0
    while j &lt; (len(list_dates)+1):

        for j in list_dates:

            raw_list.append((list_dates[j])[11:19])
            j += 1

    i = 0
    while i &lt; len(raw_list):

        for i in raw_list:
            item = (raw_list[i]).replace(":", "")
            new_list.append(item)
            i += 1
</code></pre>
<p>Obviously this doesn't work but I can't figure out what I am doing wrong as I am pretty new to Python. The error I get is this and its referring to the j in the for loop</p>
<pre><code> TypeError: list indices must be integers or slices, not str
</code></pre>
<p>How can I change my code to get this output? Or is there maybe a simpler way to solve this problem?</p>
<pre><code> &gt;&gt;&gt; new_list = [155643, 155644, 155645, ... ]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your issue is with this section of the code:</p>
<pre><code>j = 0
while j &lt; (len(list_dates)+1):

    for j in list_dates:

        raw_list.append((list_dates[j])[11:19])
        j += 1
</code></pre>
<p>Each <code>j</code> in <code>list_dates</code> is an element of the array, not the index of a certain element. So, when you try to get the <code>j</code>th element of <code>list_dates</code> with <code>list_dates[j]</code>, you're passing in a <code>str</code> as an index to the array, which doesn't make sense. You should just use another while loop like you did in the outer loop, or use <code>for k in range(len(list_dates))</code> or something similar.</p>
</div>

<div class="post-text" itemprop="text">
<p>You could use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>list_dates = ['Fri Nov 30 15:56:43 +0000 2018', 'Fri Nov 30 15:56:44 +0000 2018', 'Fri Nov 30 15:56:45 +0000 2018', 'Fri Nov 30 15:56:46 +0000 2018']
result = [''.join(c for c in date.split()[3] if c.isnumeric())  for date in list_dates]
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>['155643', '155644', '155645', '155646']
</code></pre>
<p>If you want the result as integers, use <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow noreferrer">int</a>:</p>
<pre><code>result = [int(''.join(c for c in date.split()[3] if c.isnumeric())) for date in list_dates]
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[155643, 155644, 155645, 155646]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>using <code>datetime</code> module you can do something like this. Link for further reading <a href="https://docs.python.org/2/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">https://docs.python.org/2/library/datetime.html#strftime-strptime-behavior</a></p>
<pre><code>from datetime import datetime
[datetime.strftime(datetime.strptime(i,'%a %b %d %H:%M:%S +0000 %Y'),'%H%M%S') for i in list_dates]
</code></pre>
<p>Output:</p>
<pre><code>['155643', '155644', '155645', '155646']
</code></pre>
<p>In case you need to convert the values in int then cast them in the list comprehension itself to get the list of int values</p>
