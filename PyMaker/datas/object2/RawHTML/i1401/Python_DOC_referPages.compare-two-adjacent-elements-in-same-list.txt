<p>I have already gone through a <a href="https://stackoverflow.com/questions/14012562/how-to-compare-two-adjacent-items-in-the-same-list-python">post</a> but I want to know what I did wrong in my code while using <code>for</code> loop.</p>
<p>List <code>a</code> given as:</p>
<pre><code>a = [2, 4, 7,1,9, 33]
</code></pre>
<p>All I want to compare two adjacent elements as :</p>
<pre><code>2 4
4 7
7 1
1 9
9 33

<p>I did something like:</p>
<pre><code>for x in a:

</code></pre>

<div class="post-text" itemprop="text">
<p>Your outer loop persists for <em>each</em> value in your inner loop. To compare adjacent elements, you can <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> a list with a <em>shifted version</em> of itself. The shifting can be achieved through <a href="https://stackoverflow.com/questions/509211/understanding-pythons-slice-notation">list slicing</a>:</p>
<pre><code>for x, y in zip(a, a[1:]):
    print(x, y)
</code></pre>
<p>In the <em>general</em> case, where your input is any iterable rather than a list (or another iterable which supports indexing), you can use the <a href="https://docs.python.org/3/library/itertools.html#itertools-recipes" rel="nofollow noreferrer"><code>itertools</code> <code>pairwise</code> recipe</a>, also available in the <a href="https://more-itertools.readthedocs.io/en/stable/api.html#more_itertools.pairwise" rel="nofollow noreferrer"><code>more_itertools</code></a> library:</p>
<pre><code>from more_itertools import pairwise

for x, y in pairwise(a):
    print(x, y)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are comparing a stable element with all of the elements in the list but the first.</p>
<p>The correct way would be:</p>
<pre><code>for i in range(len(a)-1):
    x = a[i]
    y = a[i+1]
    print (x,y)
</code></pre>
