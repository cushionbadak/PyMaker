<div class="post-text" itemprop="text">
<p>Get the sum of each list individually, and then sum the both scalar values to get the total_sum:</p>
<pre><code>In [1733]: num_list_1 = [1,2,3,4]

In [1734]: num_list_2 = [5,6,7,8]

In [1737]: sum(num_list_1) + sum(num_list_2)
Out[1737]: 36
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could sum the concatenation of the two lists:</p>
<pre><code>sum(num_list_1+num_list_2)
</code></pre>
<p>This is what I get using the python console:</p>
<pre><code>&gt;&gt;&gt;num_list_1 = [1,2,3,4]
&gt;&gt;&gt;num_list_2 = [5,6,7,8]
&gt;&gt;&gt;sum(num_list_1+num_list_2)
&gt;&gt;&gt;36
</code></pre>
<p>Or you could just sum the sums:</p>
<pre><code>sum(num_list_1) + sum(num_list_2)
</code></pre>
<p>which leads to the same output but in a probably faster way:</p>
<pre><code>&gt;&gt;&gt;num_list_1 = [1,2,3,4]
&gt;&gt;&gt;num_list_2 = [5,6,7,8]
&gt;&gt;&gt;sum(num_list_1) + sum(num_list_2)
&gt;&gt;&gt;36
</code></pre>
</div>

<p>if you have several lists (more than 2) you could <code>sum</code> the sums by applying <code>map</code> to the results:</p>
<pre><code>sum(map(sum,(num_list_1,num_list_2)))
</code></pre>

<div class="post-text" itemprop="text">
<p><code>sum</code> takes an iterable, so you can use <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow noreferrer"><code>itertools.chain</code></a> to chain your lists and feed the resulting iterable to <code>sum</code>:</p>
<pre><code>from itertools import chain

num_list_1 = [1,2,3,4]
num_list_2 = [5,6,7,8]

res = sum(chain(num_list_1, num_list_2))  # 36
</code></pre>
</div>
