<div class="post-text" itemprop="text">
<p>It looks like you're parsing paths, in which case you should really be using <a href="https://docs.python.org/3/library/os.path.html#os.path.basename" rel="nofollow"><code>os.path</code></a> instead of regex:</p>
<pre><code>from os.path import basename
basename('/home//Desktop/3A5F.py')
# 3A5F.py
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It is a simple <code>split</code>, no regex needed:</p>
<pre><code>&gt;&gt;&gt; "/home//Desktop/3A5F.py".split("/")[-1]
'3A5F.py'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As an alternative, you can get same result <strong>without regexps</strong>:</p>
<pre><code>lines = ['/home//Desktop/3A5F.py', 'path/sth/R67G.py', 'a/b/c/d/t/6UY7.py']

result = [l.split('/')[-1] for l in lines]
print result
# ['3A5F.py', 'R67G.py', '6UY7.py']
</code></pre>
</div>
