<div class="post-text" itemprop="text">
<p>Iterate over a <em>copy</em> of the list and use  an <em>elif</em> or else just <em>continue</em> after the remove, you don't want to count empty spaces.</p>
<pre><code>printed = False
sentence = "the quick brown fox jumps over the lazy dog"
chars = list(sentence)

for char in chars[:]:
    if char == ' ':
        chars.remove(char)
        continue
    if not printed:
        count = chars.count(char)
        print "char count: ", char, count
        printed = True
    else:
        printed = False
</code></pre>
<p>You could also just <code>str.join</code> after splitting on whitespace:</p>
<pre><code>for char in "".join(sentence.split()):
    if not printed:
        count = chars.count(char)
        print "char count: ", char, count
        printed = True
    else:
        printed = False
</code></pre>
<p>But neither of your own solutions actually work correctly even making a copy of the list, you are missing letters in the output:</p>
<pre><code>char count:  t 2
char count:  e 3
char count:  u 2
char count:  c 1
char count:  b 1
char count:  o 4
char count:  n 1
char count:  o 4
char count:  j 1
char count:  m 1
char count:  s 1
char count:  v 1
char count:  r 2
char count:  h 2
char count:  l 1
char count:  z 1
char count:  d 1
char count:  g 1
</code></pre>
<p>There are 26 unique letters in the string but you output ~17.</p>
<p>What you need is to keep track of the letters seen and only print the count once, your code keeps no record of what character has been printed it just randomly sets a flag:</p>
<pre><code>sentence = "the quick brown fox jumps over the lazy dog"
chars = list(sentence)

printed = set()
for char in "".join(sentence.split()):
    if char not in printed:
        count = chars.count(char)
        print "char count: ", char, count
        printed.add(char)
</code></pre>
<p>Or if the order first seen does not matter  then just call set on the string:</p>
<pre><code>for char in set("".join(sentence.split())):
    count = chars.count(char)
    print "char count: ", char, count
</code></pre>
<p>Or if you had large amounts of data, you would be better with a <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow">Counter dict</a>:</p>
<pre><code>from collections import Counter
for char, count in Counter("".join(sentence.split())).items():
    print(char, count)
</code></pre>

<div class="post-text" itemprop="text">
<p>The problem arises because you remove the space char while the loop runs, and it disrupts the iteration process.</p>
<p>Simply don't do anything when a space char is encontered.</p>
