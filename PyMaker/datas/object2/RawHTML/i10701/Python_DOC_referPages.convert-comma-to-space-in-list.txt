<p>how can we convert a string <code>[0.0034596999, 0.0034775001, 0.0010091923]</code> to a form <code>[0.0034596999  0.0034775001  0.0010091923]</code> in python. I tried using <code>map</code>, <code>join</code> etc functions but I am unable to do so. Can anyone help?</p>

<div class="post-text" itemprop="text">
<p>Using the string method <code>replace()</code> is an efficient solution; however thought I'd offer an alternate using <code>split()</code> and <code>join()</code>:</p>
<pre><code>print ''.join(i for i in '[0.0034596999, 0.0034775001, 0.0010091923]'.split(','))
&gt;&gt;&gt; [0.0034596999 0.0034775001 0.0010091923]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>"[0.0034596999, 0.0034775001, 0.0010091923]".replace(",", "")</code> returns <code>"[0.0034596999 0.0034775001 0.0010091923]"</code></p>
<p>Have a look at the <a href="https://docs.python.org/3/library/stdtypes.html#string-methods" rel="nofollow">string methods</a> - there are many useful ones.</p>
</div>

<p>Just use the string <code>replace()</code> method:</p>
<pre><code>s = '[0.0034596999, 0.0034775001, 0.0010091923]'
s = s.replace(',', '')
print(s)  # -&gt; [0.0034596999 0.0034775001 0.0010091923]
</code></pre>
