<div class="post-text" itemprop="text">
<p>If you say it works on the terminal and not on apache then apache's <code>php.ini</code> file may be disabling the use of <code>shell_exec()</code>.</p>
<p>See <a href="http://www.php.net/manual/en/ini.core.php#ini.disable-functions" rel="nofollow noreferrer">http://www.php.net/manual/en/ini.core.php#ini.disable-functions</a></p>
<p>Your apache's <code>php.ini</code> file may look something like</p>
<pre><code>disable_functions=exec,passthru,shell_exec,system,proc_open,popen
</code></pre>
<p>Remove <code>shell_exec</code> from this list and restart the web server, although this is a security risk and I don't recommend it.</p>
