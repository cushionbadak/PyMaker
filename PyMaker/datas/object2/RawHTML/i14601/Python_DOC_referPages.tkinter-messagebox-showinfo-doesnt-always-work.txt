<div class="post-text" itemprop="text">
<p><code>messagebox</code>, along with some other modules like <code>filedialog</code>, does not automatically get imported when you <code>import tkinter</code>. Import it explicitly, using <code>as</code> and/or <code>from</code> as desired.</p>
<pre><code>&gt;&gt;&gt; import tkinter
&gt;&gt;&gt; tkinter.messagebox.showinfo(message='hi')
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
AttributeError: 'module' object has no attribute 'messagebox'
&gt;&gt;&gt; import tkinter.messagebox
&gt;&gt;&gt; tkinter.messagebox.showinfo(message='hi')
'ok'
&gt;&gt;&gt; from tkinter import messagebox
&gt;&gt;&gt; messagebox.showinfo(message='hi')
'ok'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is case sensitive - <code>tkinter</code> should be <code>Tkinter</code> wherever it is used. I did this and was able to run your example.</p>
