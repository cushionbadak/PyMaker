<p>I need some help with Python
This is not the classic subtract List B from List A to make List C
Instead I would like to look at the indexes of the items in List A that are not in List B, and store them into a new List C.</p>
<p>e.g.</p>
<pre><code>List A:
0. car
1. plane
2. truck
3. motorcycle
4. jet
5. train

List B:
0. plane
1. jet
</code></pre>
<p>So, the result should be...</p>
<pre><code>List C:
0.
2.
3.
5.
</code></pre>
<p>The result items are not important (car, truck, motorcycle, train), instead I need to keep the original indexes of the items in List A, once the subtraction has been made.
What is the fastest simplest way to do this in Python?
Thanks Advanced</p>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow"><code>enumerate</code></a>:</p>
<pre><code>a = ["car", "plane", "truck", "motorcycle", "jet", "train"]
b = ["plane", "jet"]
c = [i for i, v in enumerate(a, start=1) if v not in b]
print(c)
</code></pre>
<p>Output:</p>
<pre><code>[1, 3, 4, 6]
</code></pre>
</div>
<div class="post-text" itemprop="text">

</code></pre>
