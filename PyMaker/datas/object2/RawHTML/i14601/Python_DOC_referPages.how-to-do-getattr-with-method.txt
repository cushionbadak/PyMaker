<p>How would I do the following:</p>
<pre><code>instance.method()
</code></pre>
<p>I know I can do <code>getattr(instance, method)</code> to get <code>instance.method</code>, but is there a builtin to actually run that method?</p>
</div>
<div class="post-text" itemprop="text">
<p>Just do <code>getattr(instance, method)()</code>.  <code>getattr</code> returns the method object, which you can call with <code>()</code> like any other callable.</p>
</div>
<div class="post-text" itemprop="text">
<p>All you need to do is add a <code>()</code> to the end:</p>
<pre><code>getattr(instance,method)()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/operator.html#operator.methodcaller" rel="nofollow"><code>operator.methodcaller</code></a> to create a function that will run that method on the passed instance when executed. However you still have to actually call it on an instance.</p>
<pre><code>from operator import methodcaller

call_hello = methodcaller('hello', 'Jack')

call_hello(something)   # same as something.hello('Jack')
</code></pre>
<p>This is useful when you want to call the same method, for which you don't know the name, on different instances.</p>
</div>
