<div class="post-text" itemprop="text">
<p>If I have two integer (start, end), </p>
<p>for example, </p>
<pre><code>start, end = 3,8 
</code></pre>
<p>then I want to create a list that includes all integers from 3 to 8 (also includes 3 and 8)</p>
<pre><code>[3,4,5,6,7,8]
</code></pre>
<p>How can I get that?</p>
</div>
<div class="post-text" itemprop="text">
<p>Python 3: use <a href="https://docs.python.org/3.4/library/functions.html#func-range" rel="nofollow noreferrer"><code>range</code></a> as the <a href="https://stackoverflow.com/questions/231767/what-does-the-yield-keyword-do-in-python/231855#231855">generator</a>, and <a href="https://docs.python.org/3.4/library/functions.html#func-list" rel="nofollow noreferrer"><code>list</code></a> to complete:</p>
<pre><code>list(range(start, end+1))
</code></pre>
<p>For Python 2, take note of <a href="https://stackoverflow.com/questions/94935/what-is-the-difference-between-range-and-xrange-functions-in-python-2-x">range vs xrange</a> and that you'd need a call to <code>list</code> if you go the route of <code>xrange</code>, which acts the same as <code>range</code> in Python 3.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use built-in <a href="https://docs.python.org/2/library/functions.html#range" rel="nofollow">range</a> function for it:</p>
<p>In python 2.x</p>
<pre><code>&gt;&gt;&gt; start, end = 3, 8
&gt;&gt;&gt; range(start, end+1)
[3, 4, 5, 6, 7, 8]
</code></pre>
<p>In python 3.x you have to call <code>list</code> function on <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow">range</a> to get list as <code>range</code> by itself returns generator object.</p>
<pre><code>In [1]: start, end = 3, 8
In [2]: range(start, end+1)
Out[2]: range(3, 9)
In [3]: list(range(start, end+1))
Out[3]: [3, 4, 5, 6, 7, 8]
</code></pre>
</div>
