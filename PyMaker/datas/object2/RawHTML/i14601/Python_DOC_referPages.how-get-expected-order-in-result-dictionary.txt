<div class="post-text" itemprop="text">
<p>A python dictionary has an internal order that is randomized and that you shouldn't depend on.</p>
<p>If you would like to retain the order that you inserted objects in, you should use a <a href="https://docs.python.org/3/library/collections.html#collections.OrderedDict"><code>collections.OrderedDict</code></a>.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>dict</code>s are unordered because keys aren't compared, their hashes are.</p>
<p>Use an <a href="https://docs.python.org/3/library/collections.html#ordereddict-objects" rel="nofollow"><code>OrderedDict</code></a> from the <code>collections</code> module to maintain the order you inputted.</p>
<pre><code>&gt;&gt;&gt; import collections
&gt;&gt;&gt;
&gt;&gt;&gt; a = ["a", "b", "c"]
&gt;&gt;&gt; b = [1, 2, 3]
&gt;&gt;&gt;
&gt;&gt;&gt; result = collections.OrderedDict(zip(a, b))
&gt;&gt;&gt; result
OrderedDict([('a', 1), ('b', 2), ('c', 3)])
&gt;&gt;&gt; dict(result) # no longer ordered
{'c': 3, 'a': 1, 'b': 2}
</code></pre>
</div>
