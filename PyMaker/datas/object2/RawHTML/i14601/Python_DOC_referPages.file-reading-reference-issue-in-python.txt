<p>Recently I have referenced a file in my desktop using python 3.4 64bit GUI application. The problem I have got is as follows:</p>
<p><img alt="enter image description here" src="https://i.stack.imgur.com/IIwOq.png"/></p>
<p><strong>The code I tried is :</strong></p>
<pre><code>fo=open("c:\users\Ismail Nuru\Sesktop\myfile\lab.txt","r+")
string1=fo.read()
print(string1)
fo.close()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Python is trying to use the <code>\uXXXX</code> part of your string as a <a href="https://docs.python.org/3/howto/unicode.html" rel="nofollow">unicode escape sequence</a>. </p>
<p>To fix this, you have 3 options here:</p>
<ol>
<li><p>Use a raw string <code>r'C:\users\Ismail Nuru\Sesktop\myfile\lab.txt'</code></p></li>
<li><p>Double the backslashes <code>'C:\\users\\Ismail Nuru\\Sesktop\\myfile\\lab.txt'</code></p></li>
<li><p>Or use forward slashes <code>'C:/users/Ismail Nuru/Sesktop/myfile/lab.txt')</code></p></li>
</ol>
