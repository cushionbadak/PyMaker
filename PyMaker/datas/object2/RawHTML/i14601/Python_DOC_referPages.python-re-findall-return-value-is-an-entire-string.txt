<p>I am writing a crawler to get certain parts of a html file. But I cannot figure out how to use re.findall().</p>
<p>Here is an example, when I want to find all ... part in the file, I may write something like this:</p>
<pre><code>re.findall("&lt;div&gt;.*\&lt;/div&gt;", result_page)
</code></pre>
<p>if result_page is a string <code>"&lt;div&gt; &lt;/div&gt; &lt;div&gt; &lt;/div&gt;"</code>, the result will be </p>
<pre><code>['&lt;div&gt; &lt;/div&gt; &lt;div&gt; &lt;/div&gt;']
</code></pre>
<p>Only the entire string. This is not what I want, I am expecting the two divs separately. What should I do?</p>
</div>
<div class="post-text" itemprop="text">
<p>Quoting <a href="https://docs.python.org/3/library/re.html?highlight=greedy" rel="nofollow">the documentation</a>, </p>
<blockquote>
<p>The <code>'*'</code>, <code>'+'</code>, and <code>'?'</code> qualifiers are all greedy; they match as much
  text as possible. Adding <code>'?'</code> after the qualifier makes it perform the
  match in non-greedy or minimal fashion; as few characters as possible
  will be matched.</p>
</blockquote>
<p>Just add the question mark:</p>
<pre><code>In [6]: re.findall("&lt;div&gt;.*?&lt;/div&gt;", result_page)
Out[6]: ['&lt;div&gt; &lt;/div&gt;', '&lt;div&gt; &lt;/div&gt;']
</code></pre>
<p>Also, you shouldn't use RegEx to parse HTML, since there're HTML parsers made exactly for that. Example using <a href="http://www.crummy.com/software/BeautifulSoup/" rel="nofollow">BeautifulSoup 4</a>:</p>
<pre><code>In [7]: import bs4

In [8]: [str(tag) for tag in bs4.BeautifulSoup(result_page)('div')]
Out[8]: ['&lt;div&gt; &lt;/div&gt;', '&lt;div&gt; &lt;/div&gt;']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>*</code> is a <a href="http://www.rexegg.com/regex-quantifiers.html#greedy" rel="nofollow"><strong>greedy</strong></a> operator, you want to use <code>*?</code> for a <strong>non-greedy</strong> match.</p>
<pre><code>re.findall("&lt;div&gt;.*?&lt;/div&gt;", result_page)
</code></pre>
<p>Or use a parser such as BeautifulSoup instead of regular expression for this task:</p>
<pre><code>from bs4 import BeautifulSoup
soup = BeautifulSoup(html)
soup.find_all('div')
</code></pre>
</div>
