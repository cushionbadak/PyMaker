<p>I try to run command in my python file, which located in dynamic file. I try to use the <code>import</code> statement, which as I understand is the better solution for this case.</p>
<p>Here is the code:</p>
<pre><code>from subprocess import call
import tarfile
from contextlib import closing

def tarfile1(path):
    with closing(tarfile.open(path)) as tar:
      tar.extractall(path)
    import path as runcommand
    runcommand.main()
</code></pre>
<p>The problem is that path is a string, and it gives me the following error:</p>
<pre><code>    import path as runcommand
ImportError: No module named path
</code></pre>
<p>How can I import the file, which I don't know its name, and run the main command from it?</p>
</div>
<div class="post-text" itemprop="text">
<p>You have to use <a href="https://docs.python.org/3/library/importlib.html#importlib.import_module" rel="nofollow"><code>importlib.import_module</code></a>.</p>
<pre><code>import importlib
importlib.import_module(&lt;&lt;mymodule&gt;&gt;)
</code></pre>
<p><strong>Note</strong>: Please make sure your module's parent directory is in PYTHONPATH or added in <code>sys.path</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use</p>
<pre><code>runcommand = __import__(path)
</code></pre>
<p>instead</p>
