<p>Does anyone know why the code below does not have the same outcome in Python?
Why do I need the parenthesis to get the correct outcome?</p>
<pre><code>#example 1
print 1-4 %5
outcome: -3

#example 2
print (1-4)%5
outcome: 2
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is due to <a href="https://docs.python.org/3/reference/expressions.html#operator-precedence" rel="nofollow">operator precedence</a>.  Mod (<code>%</code>) takes precedence over <code>-</code>, so:</p>
<pre><code>1-4 % 5 == 1 - (4 % 5) == 1 - 4 == -3
</code></pre>
<p>but</p>
<pre><code>(1-4) % 5 == -3 % 5 == 2
</code></pre>
</div>
<div class="post-text" itemprop="text">

+, -    Addition, subtraction
</code></pre>
