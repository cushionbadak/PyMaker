<div class="post-text" itemprop="text">
<p>Iterating a dictionary yields keys only. If you want key, value pairs to be yielded, use <code>items()</code> method (<code>iteritems</code> in Python 2.x).</p>
<p>Use <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.urlencode" rel="nofollow"><code>urllib.parse.urlencode</code></a> (<a href="https://docs.python.org/2/library/urllib.html#urllib.urlencode" rel="nofollow"><code>urllib.urlencode</code></a> in Python 2.x):</p>
<pre><code>&gt;&gt;&gt; import urllib.parse
&gt;&gt;&gt; urllib.parse.urlencode([
...     (key, x) for key, xs in params.items()
...     for x in ([xs] if isinstance(xs, str) else xs)
... ])
'key2=value_a&amp;key2=value_b&amp;key1=value1'
</code></pre>
<p>You can use manual string opeator, but it require extra care for special characters that should be <code>%xx</code>-encoded.</p>
</div>
