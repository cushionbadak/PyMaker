<p>I am not able to understand what the <code>re.sub()</code> function does in Python. I have read the documentation and other StackOverflow posts, but none of them clearly explains the <code>re.sub()</code> function. Can someone help me understand? For example:</p>
<pre><code>def _strip(s):
    return re.sub(r'[\W_]+', '', s)
</code></pre>
<p>I know that this function removes non-letter characters from a word. For example </p>
<pre><code>_strip('Hi there!') == 'Hi there'
</code></pre>
<ol>
<li>But why does this function use <code>''</code> after the <code>r</code> tag?</li>
<li>And why is the <code>+</code> symbol not used within the square brackets after the <code>r</code> tag?</li>
</ol>
</div>
<div class="post-text" itemprop="text">
<ul>
<li><code>r</code>: a raw string.</li>
<li><code>''</code>: the replacement</li>
<li><code>+</code>: one or more of the preceeding</li>
<li><code>[\W_]+</code>: one or more of <code>\W</code> or <code>_</code></li>
</ul>
<p>See <a href="https://docs.python.org/3/library/re.html#re.sub" rel="nofollow">https://docs.python.org/3/library/re.html#re.sub</a> where all this is explained.</p>
