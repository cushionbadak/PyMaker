<div class="post-text" itemprop="text">
<p>In java I can use the regex : <a href="http://www.regular-expressions.info/unicode.html"><code>\p{Sc}</code></a> for detecting currency symbol in text. What is the equivalent in Python? </p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the unicode category if you use <a href="https://pypi.python.org/pypi/regex"><code>regex</code></a> package:</p>
<pre><code>&gt;&gt;&gt; import regex
&gt;&gt;&gt; regex.findall(r'\p{Sc}', '$99.99 / €77')  # Python 3.x
['$', '€']
</code></pre>
<hr/>
<pre><code>&gt;&gt;&gt; regex.findall(ur'\p{Sc}', u'$99.99 / €77')  # Python 2.x (NoteL unicode literal)
[u'$', u'\xa2']
&gt;&gt;&gt; print _[1]
¢
</code></pre>
<p><strong>UPDATE</strong></p>
<p>Alterantive way using <a href="https://docs.python.org/3/library/unicodedata.html#unicodedata.category"><code>unicodedata.category</code></a>:</p>
<pre><code>&gt;&gt;&gt; import unicodedata
&gt;&gt;&gt; [ch for ch in '$99.99 / €77' if unicodedata.category(ch) == 'Sc']
['$', '€']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you want to stick with <a href="https://docs.python.org/2/library/re.html?highlight=re#module-re">re</a>, supply the <a href="http://www.fileformat.info/info/unicode/category/Sc/list.htm">characters from Sc manually</a>:</p>
<pre><code>u"[$¢£¤¥֏؋৲৳৻૱௹฿៛\u20a0-\u20bd\ua838\ufdfc\ufe69\uff04\uffe0\uffe1\uffe5\uffe6]"
</code></pre>
<p>will do.</p>
