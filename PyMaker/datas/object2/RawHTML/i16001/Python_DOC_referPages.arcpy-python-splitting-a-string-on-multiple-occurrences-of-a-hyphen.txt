<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>split</code></a> function takes a second argument, that is the number of splits to perform. Pass in a value of <code>1</code>, then you'll get a list back where <code>[0]</code> is the first item and <code>[1]</code> is everything else to the right.</p>
<pre><code>&gt;&gt;&gt; 'Stewart-13-0931-2312'.split('-',1)
['Stewart', '13-0931-2312']

&gt;&gt;&gt; 'Jack-678-987-3-A'.split('-',1)
['Jack', '678-987-3-A']
</code></pre>
