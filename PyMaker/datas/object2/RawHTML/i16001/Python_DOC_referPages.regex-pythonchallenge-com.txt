<p>I need to filter a text like UUUlUUU (i mean Uppercase letter*3 + lowercase letter + Uppercase letter*3). How can i implement that regex?</p>
<p>I know that python alows this:</p>
<pre><code>    re.match(\w)
</code></pre>
<p>i tried like this:</p>
<pre><code>    re.match([[A-Z]+[A-Z]+[A-Z]+[a-z]+[[A-Z]+[A-Z]+[A-Z])
</code></pre>
<p>for finding a single character. (i can add as many \w as i need)</p>
<p>The Question is: is there a way to make a Case Sensitive regex for getting that i what?</p>

<div class="post-text" itemprop="text">
<p>The pattern you want is</p>
<pre><code>'^[A-Z]{3}[a-z][A-Z]{3}$'
</code></pre>
<p>Note that you can use <code>{}</code> to specify repetition.</p>
<p>Example</p>
<pre><code>import re
re.findall('^[A-Z]{3}[a-z][A-Z]{3}$' , 'UUUlUUU')
['UUUlUUU']
</code></pre>
<p>As far as "articles for reading", the <a href="https://docs.python.org/3/howto/regex.html" rel="nofollow">documentation</a> is always a good start.</p>
