<p>I've some problem with "decode" method in python 3.3.4. This is my code:</p>
<pre><code>for lines in open('file','r'):
    decodedLine = lines.decode('ISO-8859-1')
    line = decodedLine.split('\t')
</code></pre>
<p>But I can't decode the line for this problem: </p>
<pre><code>AttributeError: 'str' object has no attribute 'decode'
</code></pre>
<p>Do you have any ideas? Thanks</p>
</div>
<div class="post-text" itemprop="text">
<p>One <em>encodes</em> strings, and one <em>decodes</em> bytes.</p>
<p>You should read bytes from the file and decode them:</p>
<pre><code>for lines in open('file','rb'):
    decodedLine = lines.decode('ISO-8859-1')
    line = decodedLine.split('\t')
</code></pre>
<p>Luckily <code>open</code> has an encoding argument which makes this easy:</p>
<pre><code>for decodedLine in open('file', 'r', encoding='ISO-8859-1'):
    line = decodedLine.split('\t')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>open</code> already decodes to Unicode in Python 3 if you open in text mode. If you want to open it as bytes, so that you can then decode, you need to open with mode 'rb'.</p>
</div>
<div class="post-text" itemprop="text">
<p>This works for me smoothly to read Chinese text in Python 3.6. First, convert str to bytes, and then decode them.</p>
<pre><code>for l in open('chinese2.txt','rb'):
    decodedLine = l.decode('gb2312')
    print(decodedLine)
</code></pre>
</div>
