<div class="post-text" itemprop="text">
<p>Pass the second command line arguments as a single argument in the shell:</p>
<pre><code>python /path/to/script.py file1 "This is a sentence."
</code></pre>
<p>Alternatively, join the arguments after the filename using <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join</code></a>:</p>
<pre><code>with open(input[1], 'a') as fileA:
    fileA.write(' '.join(input[2:]))
</code></pre>
<p>But the alternative solution will not preserve exact count of spaces / tabs.</p>
