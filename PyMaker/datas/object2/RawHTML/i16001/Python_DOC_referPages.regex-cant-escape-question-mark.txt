<p>I can't match the question mark character although I escaped it.<br/>
I tried escaping with multiple backslashes and also using <code>re.escape()</code>. </p>
<p>What am I missing?</p>
<p>Code:</p>
<pre><code>import re

text = 'test?'
result = ''

result = re.match(r'\?',text)

print ("input: "+text)
print ("found: "+str(result))
</code></pre>
<p>Output:</p>
<pre><code>input: test?
found: None
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/re.html#re.match" rel="nofollow"><code>re.match</code></a> only matches a pattern at the begining of string; as in the docs:</p>
<blockquote>
<p>If zero or more characters at the <strong>beginning</strong> of string match the regular expression pattern, return a corresponding match object.</p>
</blockquote>
<p>so, either:</p>
<pre><code>&gt;&gt;&gt; re.match(r'.*\?', text).group(0)
'test?
</code></pre>
<p>or <a href="https://docs.python.org/3/library/re.html#re.search" rel="nofollow"><code>re.search</code></a></p>
<pre><code>&gt;&gt;&gt; re.search(r'\?', text).group(0)
'?'
</code></pre>
</div>
