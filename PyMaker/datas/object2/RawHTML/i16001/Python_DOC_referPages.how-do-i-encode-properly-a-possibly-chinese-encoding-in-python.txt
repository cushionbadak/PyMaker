<p><code>%XX</code> encoding can be done using <a href="https://docs.python.org/2/library/urllib.html#urllib.quote" rel="nofollow"><code>urllib.qutoe</code></a>:</p>
<pre><code>&gt;&gt;&gt; import urllib
&gt;&gt;&gt; urllib.quote('英冠')
'%E8%8B%B1%E5%86%A0'

&gt;&gt;&gt; urllib.quote(u'英冠'.encode('utf-8'))  # with explicit utf-8 encoding.
'%E8%8B%B1%E5%86%A0'
</code></pre>
<p>To get back the original string, use <a href="https://docs.python.org/2/library/urllib.html#urllib.unquote" rel="nofollow"><code>urllib.unquote</code></a>:</p>
<pre><code>&gt;&gt;&gt; urllib.unquote('%E8%8B%B1%E5%86%A0')
'\xe8\x8b\xb1\xe5\x86\xa0'
&gt;&gt;&gt; print(urllib.unquote('%E8%8B%B1%E5%86%A0'))
英冠
</code></pre>
<hr/>
<p>In Python 3.x, use <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.quote" rel="nofollow"><code>urllib.parse.quote</code></a>, <code>urllib.parse.unquote</code>:</p>
<pre><code>&gt;&gt;&gt; import urllib.parse
&gt;&gt;&gt; urllib.parse.quote('英冠', encoding='utf-8')
'%E8%8B%B1%E5%86%A0'
&gt;&gt;&gt; urllib.parse.unquote('%E8%8B%B1%E5%86%A0', encoding='utf-8')
'英冠'
</code></pre>
