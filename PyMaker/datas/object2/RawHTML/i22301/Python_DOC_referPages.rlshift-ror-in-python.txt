<div class="post-text" itemprop="text">
<p>See the documentation for:</p>
<ul>
<li><a href="http://docs.python.org/3/reference/datamodel.html?highlight=rlshift#object.__rlshift__" rel="nofollow noreferrer"><code>object.__rlshift__()</code></a></li>
<li><a href="http://docs.python.org/3/reference/datamodel.html?highlight=ror#object.__ror__" rel="nofollow noreferrer"><code>object.__ror__()</code></a></li>
</ul>
<p><code>__rlshift__</code> is the swapped operands version of <code>__lshift__</code>, used when the right-hand operand supports the operation but the left-hand operand doesn't.</p>
