<div class="post-text" itemprop="text">
<p>You can unpack the first item of each tuple as you iterate over the rows (and you don't need to <code>enumerate</code> as you are not using the row index).</p>
<p>Change:</p>
<pre><code>for row_index, row_data in enumerate(res):
    stainless_getlist.append(row_data)
</code></pre>
<p>to:</p>
<pre><code>for item, in res:
    stainless_getlist.append(item)
</code></pre>
