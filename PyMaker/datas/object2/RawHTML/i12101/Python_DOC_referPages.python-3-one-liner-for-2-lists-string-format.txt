<div class="post-text" itemprop="text">
<p>I have 2 lists in a python3 Class:</p>
<pre><code>self.keys = ["a","b","c","d"]
self.values = [1,2,3,4]
</code></pre>
<p>len(self.keys) == len(self.values), always.</p>
<p>I want to create a string that mimic the printing output of a dictionary: <code>{a:1, b:2, c:3, d:4}</code></p>
<p>I do it in a method that contains:</p>
<pre><code>sr = ""
for i,k in enumerate(self.keys):
    sr += "{}:{},".format(k,self.values[i])
return "{%s}" % sr[:len(sr)-1]
</code></pre>
<p>Is it possible in one line ? if not, is there a better way to do this ?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#zip">zip</a> and then make it a <a href="https://docs.python.org/3/library/functions.html#func-dict">dict</a>:</p>
<pre><code>keys = ["a","b","c","d"]
values = [1,2,3,4]

dict(zip(keys, values))
</code></pre>
<p>Output:</p>
<pre><code>{'b': 2, 'a': 1, 'c': 3, 'd': 4}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Why not just print a dictionary? Use <a href="https://docs.python.org/3/library/functions.html?highlight=zip#zip" rel="nofollow">zip</a> to make tuples of the pairs, then pass them to the <a href="https://docs.python.org/3/library/stdtypes.html#dict" rel="nofollow">dict constructor</a>.</p>
<pre><code>print(dict(zip(self.keys, self.values)))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you want to get the exactly the format your original code produces with a single line of code, you could use</p>
<pre><code>from itertools import starmap

print("{%s}" % ",".join(starmap("{}:{}".format, zip(keys, values))))
</code></pre>
<p>I'm not sure this is more readable than your original code though.</p>
