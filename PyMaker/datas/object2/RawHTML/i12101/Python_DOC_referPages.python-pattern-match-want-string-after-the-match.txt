<div class="post-text" itemprop="text">
<p>You need to capture the text which exists next to =</p>
<pre><code>p = re.match(r'.for[^=]*=\s*(.*)',dict['comment'])
if p:
    print '%s' %p.group(1)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could also use the native <code>string.split()</code> if the entire string is <code>.for $i = 5 .. 13</code>:</p>
<pre><code>s = dict['comment']
if '.for' in s:
    output = s.split('=')[1]
#output is then: " 5 .. 13"
</code></pre>
