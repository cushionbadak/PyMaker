<p>Basically you are counting. Here is an example. A tried to keep it simple so it is easy to follow:</p>
<pre><code>def get_string(idxlist, item_list):
    return ''.join([item_list[i] for i in idxlist])

def increment_vector(idxlist, max_count):
    idxlist[0] += 1
    for i in xrange(len(idxlist)-1):
        if idxlist[i] &lt; max_count:
            break
        idxlist[i]=0
        idxlist[i+1] += 1

def generate(n, item_list):
    max_count = len(item_list)

    idxlist = [0] * n
    while idxlist[-1] &lt; max_count:
        print ( get_string( idxlist, item_list )),
        increment_vector(idxlist, max_count)
        if idxlist[0]==0:
            print 

item_list = map(chr, range(97, 123)) # letters from a-z

generate(3, item_list)
</code></pre>
<p>The only point where you really work with your items is in get_string.</p>
