<p>I'm facing a problem involving lists. I have a nested
list like:</p>
<pre><code>A = [ [0,2,3,4], [0,1,1,2] ]
</code></pre>
<p>where the first list holds the the y and the second list holds the x coordinate for four different points. I'd like to obtain a list:</p>
<pre><code>B = [[0,0], [1,2], [1,3], [2,4]]
</code></pre>
<p>where each list represents a point. Is there a simple pythonic way to do that or must I deal with for loops? Thank you in advance and sorry if a similar question has been already posted.</p>

<div class="post-text" itemprop="text">
<p>That's what the <code>zip</code> function is for.</p>
<pre><code>&gt;&gt;&gt; A = [ [0,2,3,4], [0,1,1,2] ]
&gt;&gt;&gt; B = [list(x) for x in zip(*A[::-1])]
&gt;&gt;&gt; B
[[0, 0], [1, 2], [1, 3], [2, 4]]
</code></pre>
<p>if you don't need your elements to be lists (i.e. tuples are fine, too), this shortens to</p>
<pre><code>&gt;&gt;&gt; zip(*A[::-1])
[(0, 0), (1, 2), (1, 3), (2, 4)]
</code></pre>
<p>in Python 2 and</p>
<pre><code>&gt;&gt;&gt; list(zip(*A[::-1]))
[(0, 0), (1, 2), (1, 3), (2, 4)]
</code></pre>
<p>in Python 3.</p>
<p>Explanation of the syntax:</p>
<p><code>A[::-1]</code> is reversing <code>A</code>:</p>
<pre><code>&gt;&gt;&gt; A[::-1]
[[0, 1, 1, 2], [0, 2, 3, 4]]
</code></pre>
<p>The reversed list is then passed in an unpacked form (notice the <code>*</code>) to the <code>zip</code> function, which is equivalent to</p>
<pre><code>&gt;&gt;&gt; zip([0, 1, 1, 2], [0, 2, 3, 4])
[(0, 0), (1, 2), (1, 3), (2, 4)]
</code></pre>
<p>and <code>zip</code> will build tuples from the first, second, ..., n'th elements from each iterable.</p>
<p>edit:</p>
<p>Minor improvement:</p>
<pre><code>&gt;&gt;&gt; zip(*reversed(A))
[(0, 0), (1, 2), (1, 3), (2, 4)]
</code></pre>
<p>should be more memory efficient because <code>reversed</code> gives you an iterator, <code>A[::-1]</code> gives you a complete list, which we don't really need.</p>

<div class="post-text" itemprop="text">
<p>A couple things. First, those are Python lists which roughly correspond to ArrayLists in other programming languages. Second you asked for tuple elements in the result, but the code you posted has elements of <a href="https://docs.python.org/3/library/functions.html#type" rel="nofollow">type</a> list.</p>
<p>Use the <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow">zip</a> builtin function.</p>
<pre><code>B = list(zip(*A[::-1]))

print(B)
</code></pre>
<p>yields</p>
<pre><code>[(0, 0), (2, 1), (3, 1), (4, 2)]
</code></pre>
<p>If you want elements of type list, you can do the following.</p>
<pre><code>B = list(map(list, zip(*A[::-1])))

print(B)
</code></pre>
<p>yields</p>
<pre><code>[[0, 0], [2, 1], [3, 1], [4, 2]]
</code></pre>
</div>
