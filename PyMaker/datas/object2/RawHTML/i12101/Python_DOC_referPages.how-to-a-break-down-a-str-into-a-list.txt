<div class="post-text" itemprop="text">
<p>You need to change <code>show_entry_fields</code> and use <code>str</code>'s <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>split()</code> method</a>:</p>
<pre><code>def show_entry_fields():
    # assuming the words are separated by one or more spaces
    word_list = e1.get().split()  # break the content of e1 into a list of words
    for word in word_list:
        website = "www.wordreference.com/fren/%s" % word
        webbrowser.open(website)
</code></pre>
<p>If you want the words to be separated by something other than spaces, then you need to pass the delimiter to <code>split</code>. For example, if the words are separated by a comma followed by a space (as in <code>'hello, world, bye'</code>), then the <code>split</code> call should be <code>word_list = e1.get().split(', ')</code>.</p>
<hr/>
<p>As for the errors you get, <code>e1 = [Entry(master)]</code> makes <code>e1</code> a list, which doesn't have a <code>get()</code> method. So you get an <code>AttributeError</code> when you call <code>show_entry_fields</code>. <code>e1 = Entry[(master)]</code> does not make an <code>Entry</code> instance but is trying to index into <code>Entry</code> (the class) which isn't the kind of thing you can index into. So this should immediately raise a <code>TypeError</code>.</p>
