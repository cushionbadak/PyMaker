<p>Simple code gives the below error. It's directly from the docs (<a href="https://docs.python.org/3/library/glob.html" rel="noreferrer">https://docs.python.org/3/library/glob.html</a>)</p>
<p><code>TypeError: iglob() got an unexpected keyword argument 'recursive'</code></p>
<pre><code>import glob
for filename in glob.iglob('C:\\**\\*txt', recursive=True):
    print filename
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It seems you're using Python 2.7 and reading the Python 3.5 documentation.</p>
</div>
<div class="post-text" itemprop="text">
<p>The <code>recursive</code> parameter has been added in python 3.5, which means version <code>3.4.3</code> also has that problem. </p>
<p>If you don't want to upgrade your python version, you can use <a href="https://pypi.python.org/pypi/glob2" rel="noreferrer">glob2</a>, which supports recursive calls (<code>**</code>) by default.</p>
</div>
