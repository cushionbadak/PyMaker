</code></pre>

<div class="post-text" itemprop="text">
<p>Just specify the location in a string variable and then added to the name of file you want to create with doing <code>open</code>:</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; my_dir = 'C:\\Test\\My_Dir'
&gt;&gt;&gt; file_name = 'test.txt'
&gt;&gt;&gt; fname = os.path.join(my_dir, file_name)
&gt;&gt;&gt; file = open(fname,'w')
</code></pre>
