<p>I want to extract in groups values separated by a /.</p>
<p>A couple of examples:</p>
<ul>
<li>/abra/1234fd/kadabra/2314 should return abra, 1234fd, kadabra, 2314</li>
<li>/abra/1234fd/kadabra should return abra, 1234fd, kadabra</li>
<li>/abra/1234fd should return abra, 1234fd</li>
<li>etc.</li>
</ul>
<p>Thanks in advance</p>
<p>Edit: the thing is that I have an arbitrary number of params (alphanumeric values) separated by a / and I'm looking for a way to make it general.</p>

<div class="post-text" itemprop="text">
<p><code>str.split()</code> seems to be what you want:</p>
<pre><code>assert '/abra/1234fd/kadabra/2314'.strip('/').split('/') == ['abra', '1234fd', 'kadabra', '2314']
assert '/abra/1234fd/kadabra'.strip('/').split('/') == ['abra', '1234fd', 'kadabra']
assert '/abra/1234fd'.strip('/').split('/') == ['abra', '1234fd']
</code></pre>
<p>If you must have a regular expression, try this:</p>
<pre><code>import re
assert re.findall('[^/]+', '/abra/1234fd/kadabra/2314') == ['abra', '1234fd', 'kadabra', '2314']
assert re.findall('[^/]+', '/abra/1234fd/kadabra') == ['abra', '1234fd', 'kadabra']
assert re.findall('[^/]+', '/abra/1234fd') == ['abra', '1234fd']
</code></pre>
