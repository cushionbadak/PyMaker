<div class="post-text" itemprop="text">
<p>In Python, this: </p>
<pre><code>$myArray = array
{
    'suchy' =&gt; 'such',
    'anotherarray' =&gt; array { 'something', 'else'}
}
</code></pre>
<p>is this: </p>
<pre><code>my_dict = {
    'suchy': 'such',
    'anotherarray': ['something', 'else']
}
</code></pre>
<p>If you want to add something at the first level, it's simply:</p>
<pre><code>my_dict['stuff'] = [1,2,3,4]
</code></pre>
<p>which will now make it:</p>
<pre><code>my_dict = {
    'suchy': 'such',
    'anotherarray': ['something', 'else']
    'stuff': [1, 2, 3, 4]
}
</code></pre>
<p>If you want to update a list, let's say the list stored in 'anotherarray', you do this: </p>
<pre><code>my_dict['anotherarray'].append('things')
</code></pre>
<p>Output of <code>my_dict['anotherarray']</code></p>
<pre><code>['something', 'else', 'things']
</code></pre>
<p>I suggest reading a tutorial on <a href="http://www.tutorialspoint.com/python/python_dictionary.htm" rel="nofollow">dictionaries</a> in Python: </p>
<p><a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow">Documentation from official docs</a></p>
</div>
