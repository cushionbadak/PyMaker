<div class="post-text" itemprop="text">
<p>You want to use the <a href="https://docs.python.org/3/library/functions.html#all" rel="noreferrer"><code>all</code></a> or <a href="https://docs.python.org/3/library/functions.html#any" rel="noreferrer"><code>any</code></a> built-ins:</p>
<pre><code>if any(testing12(list, n) for n in lists):
    print("at least one matched")
</code></pre>
<p>Or:</p>
<pre><code>if all(testing12(list, n) for n in lists):
    print("All lists matched")
</code></pre>
