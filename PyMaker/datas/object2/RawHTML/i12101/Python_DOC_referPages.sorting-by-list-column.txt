<p>I have this list like this:</p>
<pre><code>list1 = [["A",5,2,8,3],["B",9,5,1,5]]
</code></pre>
<p>And I'm trying to sort it by the second column. I would expect list1 to then be </p>
<pre><code>[["B",9,5,1,5],["A",5,2,8,3]]
</code></pre>
<p>Because 9 is greater than 5 right? I'm using</p>
<pre><code>sorted(lis1, key=lambda a : a[1])
</code></pre>
<p>however it doesn't sort it; it just remains as it was before. Why isn't this working?</p>

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow"><code>sorted</code></a>:</p>
<blockquote>
<p><strong>Return</strong> a new sorted list from the items in iterable</p>
</blockquote>
<p>It does <em>not</em> sort in place.</p>
</div>
<div class="post-text" itemprop="text">
<p>You just need to reverse it:</p>
<pre><code>list1 = [["A",5,2,8,3],["B",9,5,1,5]]
list1 = sorted(list1, key=lambda a : a[1], reverse=True)
print(list1)
# outputs [["B",9,5,1,5],["A",5,2,8,3]]
</code></pre>
<p>9&gt;5, so it comes afterwards, unless you reverse.</p>
</div>
<div class="post-text" itemprop="text">
<p>using operator.itemgetter() function:</p>
<pre><code>from operator import itemgetter

list1 = [["A",5,2,8,3],["B",9,5,1,5],["C",1,2,8,3]]

print(sorted(list1, key=itemgetter(1)))
print(sorted(list1, key=itemgetter(1), reverse=True))
</code></pre>
<p>Output:</p>
<pre><code>[['C', 1, 2, 8, 3], ['A', 5, 2, 8, 3], ['B', 9, 5, 1, 5]]
[['B', 9, 5, 1, 5], ['A', 5, 2, 8, 3], ['C', 1, 2, 8, 3]]
</code></pre>
</div>
