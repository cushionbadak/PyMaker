<p>Put the functions in a list, then use the <a href="https://docs.python.org/3/library/random.html#random.choice" rel="nofollow"><code>random.choice()</code> function</a> to pick one at random for you. Functions are just objects, just like any other value in Python:</p>
<pre><code>import random

def sub_hello():
    print('Well, hello there!')

def sub_nice_weather():
    print("Nice weather, isn't it?")

def sub_sports():
    print('What about them Broncos, eh?')

chat_functions = [sub_hello, sub_nice_weather, sub_sports]
randsub = random.choice(chat_functions)
randsub()
</code></pre>
<p>You got your specific error because you tried to concatenate an integer with a string (<code>"sub"</code> is a string, <code>rand</code> an integer); you'd normally convert the integer to a string first, or use a string template that supports converting other objects to strings for you (like <code>str.format()</code> or <code>str % (values,)</code>). But the string doesn't turn into a function, even if the string <em>value</em> was the same as a function name you happen to have defined.</p>
