<p>I have a string with a formula <i>5 - 3</i>, and I need to get the result in integer. How could I do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>use <a href="https://docs.python.org/3/library/functions.html#eval" rel="nofollow"><code>eval</code></a> function:</p>
<pre><code>eval("5 - 3") # 2
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>test = "5-3"
print(eval(test))
</code></pre>
<p>Gives 2</p>
