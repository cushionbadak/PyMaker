<p>If I am constantly adding in strings from a user's input. How can I check that once a string has been added into a list, the contents has already been inside?</p>
<pre><code>x = 4
y = 9
repeatList = []
abc = str(x)+str(y)
repeatList.append(abc)
x = 3
y = 2
abc = str(x)+str(y)
repeatList.append(abc)
x = 4
y = 9
abc = str(x)+str(y)
repeatList.append(abc)
print(repeatList)
</code></pre>
<p>Gives an output: </p>
<pre><code>['49', '32', '49']
</code></pre>
<p>The desired output is <code>['49','32']</code> and a message "You have inputted '49' already." </p>
<p>Please note in the actual code we do not assign variables to an int, instead, we allow the player to input and if they've already written it before then we let them know they have typed in the same input and that they still lose a 'move'.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>x = 4
y = 9
repeatList = []
if str(x)+str(y) not in repeatList:
    repeatList.append(str(x)+str(y))
else:
     print('You have inputted {0} already.'.format(str(x)+str(y)))

x = 3
y = 2
if str(x)+str(y) not in repeatList:
    repeatList.append(str(x)+str(y))
else:
     print('You have inputted {0} already.'.format(str(x)+str(y)))
x = 4
y = 9
if str(x)+str(y) not in repeatList:
    repeatList.append(str(x)+str(y))
else:
     print('You have inputted {0} already.'.format(str(x)+str(y)))
print(repeatList)
</code></pre>
<p>Output:</p>
<pre><code>You have inputted 49 already.
['49', '32']
</code></pre>

    print ("You have inputted", x, "already.")

<div class="post-text" itemprop="text">
<p>In addition to what @idos already said:<br/>
If you are not interested in the order of insertion and the objects you add are <a href="https://docs.python.org/3/glossary.html#term-hashable" rel="nofollow noreferrer">hashable</a>, you could use a <a href="https://docs.python.org/3/library/stdtypes.html#set" rel="nofollow noreferrer">set</a>, which has <a href="https://stackoverflow.com/a/17945009">faster performance than lists</a> when it comes to check if an element is already inside it.</p>
<pre><code>&gt;&gt;&gt; s = set()
&gt;&gt;&gt; s.add(49)
&gt;&gt;&gt; s
set([49])
&gt;&gt;&gt; s.add(32)
&gt;&gt;&gt; s
set([32, 49])
&gt;&gt;&gt; s.add(49)
&gt;&gt;&gt; s
set([32, 49])
&gt;&gt;&gt; s.add('49')
&gt;&gt;&gt; s
set([32, 49, '49'])
</code></pre>
