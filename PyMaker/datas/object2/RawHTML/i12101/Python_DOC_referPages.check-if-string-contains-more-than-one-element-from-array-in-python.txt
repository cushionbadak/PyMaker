<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum</code></a> here. The trick here is that <code>True</code> is calculated as <code>1</code> while finding the <code>sum</code>. Hence you can utilize the <code>in</code> directly. </p>
<pre><code>&gt;&gt;&gt; sum(x in message for x in myArray)
4
&gt;&gt;&gt; sum(x in message for x in myArray) == 1
False
</code></pre>
<p><code>if</code> clause can look like</p>
<pre><code>&gt;&gt;&gt; if(sum(x in message for x in myArray) == 1):
...     print("Only one match")
... else:
...     print("Many matches")
... 
Many matches
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>any(x in message for x in myArray)
</code></pre>
<p>Evaluates to <code>True</code> if <strong><em>at least one</em></strong> string in <code>myArray</code> is found in <code>message</code>.</p>
<pre><code>sum(x in message for x in myArray) == 1
</code></pre>
<p>Evaluates to <code>True</code> if <strong><em>exactly one</em></strong> string in <code>myArray</code> is found in <code>message</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you are looking for one of the fastest ways to do it use intersections of sets:</p>
<pre><code>mySet  = set(['right', 'left', 'front', 'back'])
message = 'right up down left'

if len(mySet &amp; set(message.split())) &gt; 1:
    print('YES')
</code></pre>
