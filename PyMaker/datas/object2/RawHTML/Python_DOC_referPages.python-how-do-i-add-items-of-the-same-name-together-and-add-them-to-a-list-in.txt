<div class="post-text" itemprop="text">
<p>I have a list in a file of different animals which is opened and read in a function.</p>
<p>For example my list is:</p>
<pre><code>Cats, Dogs, Cows, Cows, Cows, Sheep, Dogs, Sheep, etc.
</code></pre>
<p>How do I put these in a list which will group the same animals together and return the number of each animal in the list, for example for this list I want:</p>
<pre><code>List = (1, 2, 3, 2)
</code></pre>
<p>This will then enable me to work out the length of this new list and give how many different animals there are.</p>
<p>Without using sets if possible.</p>
<p>Also, because it is in a function I cant be specific about each animal.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the values of a <code>collections.Counter</code> object instantiated from the list:</p>
<pre><code>from collections import Counter
l = ['Cats', 'Dogs', 'Cows', 'Cows', 'Cows', 'Sheep', 'Dogs', 'Sheep']
print(list(Counter(l).values()))
</code></pre>
<p>This outputs:</p>
<pre><code>[1, 2, 3, 2]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The below uses pandas to create a DataFrame with your list. </p>
<p>You can then use the pandas value_counts function, which returns a Series containing counts of unique Animals. </p>
<pre><code>import pandas as pd

myAnimals = {'Animals': ['Cats', 'Dogs', 'Cows', 'Cows', 'Cows', 'Sheep', 'Dogs', 'Sheep']}
myList = pd.DataFrame(myAnimals)

pd.value_counts(myList['Animals'].values)
</code></pre>
<p>I find the output of this method more straight-forward:</p>
<pre><code>Cows     3
Sheep    2
Dogs     2
Cats     1
</code></pre>
<p>See <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.Series.value_counts.html" rel="nofollow noreferrer">value_counts</a> for more information.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could use groupby from pandas</p>
<pre><code>    import pandas as pd

    listOfAnimalsDF = pd.DataFrame({"Animal": ["Cats", "Dogs", "Cows", "Cows", "Cows", "Sheep", "Dogs", "Sheep"]})
    grouped = listOfAnimalsDF.groupby(by="Animal")

    print(grouped.groups)
</code></pre>
<p>For more Information about the <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.groupby.html" rel="nofollow noreferrer">pandas.DataFrame.groupby</a></p>
