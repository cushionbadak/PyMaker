<p>I have an numpy array with numbers from 0 to 2:</p>
<pre><code>myArray = np.random.randint(3, size=(255,255,1)).astype('uint8')
</code></pre>
<p>now i want all indexes where it has value 0.</p>
<p>for example <code>(220,15)</code></p>
<p>so that i can do 
<code>myArray[220][15]</code> and retrieve 0.</p>
</div>
<div class="post-text" itemprop="text">
<p>Note that your array has three dimensions, even if the third can only take <code>0</code> as its index, so your indices will have three values.</p>
<p>You can use numpy's n dimensional enumerator to iterate through the array, getting a list of indices where the value is 0.</p>
<pre><code>zero_indices = [index for index, value in np.ndenumerate(myArray) if value == 0]
</code></pre>
<p>Edit: Slam's answer should be a lot faster though.</p>
</div>
<div class="post-text" itemprop="text">
<p>You need <a href="https://docs.scipy.org/doc/numpy-1.15.1/reference/generated/numpy.where.html" rel="nofollow noreferrer"><code>np.where</code></a> without elements, and <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> `em for your preferred format:</p>
<pre><code>&gt;&gt;&gt; indices = np.where(myArray == 0)
&gt;&gt;&gt; element_wise = zip(*indices)
&gt;&gt;&gt; next(element_wise)
(0, 3, 0)
</code></pre>
</div>
