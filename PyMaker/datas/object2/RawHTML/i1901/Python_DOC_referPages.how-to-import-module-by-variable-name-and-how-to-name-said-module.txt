<p>So I want to import a python script depending on a variable name. Say I pass this function the module path, I'd like it to import that module from the path I passed <strong><em>and</em></strong> give it a name. Example:</p>
<pre><code>import some_module as sm
#Would import some_module
</code></pre>
<p>What I want is something like this</p>
<pre><code>module_to_import = some_other_module
import module_to_import as mti
#Would import some_other_module
</code></pre>
<p>Sorry if this doesn't make much sense, was quite difficult to put into words.</p>
<p>Thanks.</p>
</div>
<div class="post-text" itemprop="text">
<p>with <a href="https://docs.python.org/3/library/importlib.html?highlight=importlib#importlib.import_module" rel="nofollow noreferrer"><code>importlib.import_module</code></a> you could do this:</p>
<pre><code>from importlib import import_module
module_name = 'some_module'

mti = import_module(module_name)
</code></pre>
<p>meaning you could import the module from a string. in your case you might get that name from your other module using</p>
<pre><code>module_name = some_other_module.__name__
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>What you probably are looking for is ability to dynamically import modules</p>
<pre><code>module = __import__(module_name)
my_class = getattr(module, class_name)
instance = my_class()
</code></pre>
<p>You can read more about this and alternative solutions at <a href="https://www.blog.pythonlibrary.org/2012/07/31/advanced-python-how-to-dynamically-load-modules-or-classes/" rel="nofollow noreferrer">this link</a></p>
