<div class="post-text" itemprop="text">
<p>As Paul suggests, use <code>isalpha</code> to check if a string contains only letters:</p>
<pre><code>assert name.isalpha()
</code></pre>
<p>Using <code>assert</code> in this way will raise an error if the name contains a non alphabet character.</p>
<p>Your vowel conversion can be simplified: </p>
<pre><code>def replace(c):
    if c in 'aeoui':
        return chr(ord(c) + 1)
    return c # return c if not a vowel

name = "".join([replace(c) for c in name])
</code></pre>
<ul>
<li>Use <code>ord</code> to convert the character to an integer</li>
<li>Increment this by one</li>
<li>Use <code>chr</code> to build a character back from the integer</li>
</ul>
