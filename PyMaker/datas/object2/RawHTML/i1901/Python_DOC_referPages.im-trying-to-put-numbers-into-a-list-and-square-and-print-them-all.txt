<p>In the document of python3, which you can find the link at the bottom, math.sqrt() is described as "Return the square root of x". 
Since ListNum is actually a list, math.sqrt() will have trouble recognizing any types other than numbers. </p>
<p>And the'SyntaxError' comes from a missing ')' at the end of line 4.</p>
<p>The following code could work.</p>
<pre><code>import math
ListNum = [2,4,6,8]
ListNum = list(map(math.sqrt,ListNum))
print (ListNum)
</code></pre>

<div class="post-text" itemprop="text">
<p>You can't take the square root of a list, and <code>ListNum</code> is a list.</p>
<p>If you want to take the <em>square root</em> (not square) of all numbers, you can map <code>math.sqrt</code> on to the list:</p>
<pre><code>import math
ListNum = [2,4,6,8]
roots = list(map(math.sqrt, ListNum))
print(roots)
</code></pre>
<p>Note that there is no loop; the iteration is abstracted into <code>map</code>.</p>
<p>If you want to <em>square</em> the numbers, you need a squaring function:</p>
<pre><code>def square(x) : return x * x
print(list(map(square, ListNum)))
</code></pre>
<p>or </p>
<pre><code>print(list(map(lambda x: x* x, ListNum)))
</code></pre>
