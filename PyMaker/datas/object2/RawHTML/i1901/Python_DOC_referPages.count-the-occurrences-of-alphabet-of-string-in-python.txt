<p>Print will separate the args with a space. Concatenate them with <code>+</code>. <code>count()</code> is an int so it needs to be converted to a string.</p>
<pre><code>String="ABaBCbGc"
counts = {}
String1=String.upper()
for i in String1:
    counts[i] = String1.count(i)

for k in counts.keys():
    print(str(counts[k]) + k)

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer">Counter</a>:</p>
<pre><code>from collections import Counter

String = "ABaBCbGc"

counts = Counter(String.lower())

print(counts)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>Counter({'b': 3, 'c': 2, 'a': 2, 'g': 1})
</code></pre>
<p>If you prefer upper case, just change <code>str.lower</code> to <code>str.upper</code>. Or use a dictionary to keep track of the counts:</p>
<pre><code>string = "ABaBCbGc"
counts = {}
for c in string.upper():
    counts[c] = counts.get(c, 0) + 1

print(counts)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>{'C': 2, 'B': 3, 'A': 2, 'G': 1}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Create the sorted input string as the values and the sorted keys of this string as the keys. Iterate the keys and count their occurrences in the values.</p>
<pre><code>data = "ABaBCbGc".upper()
values = ''.join(sorted(data))
keys = sorted(''.join(set(data)))
for key in keys:
    print(str(values.count(key))+key)
</code></pre>
<p>Output:</p>
<pre><code>2A
3B
2C
1G
</code></pre>
