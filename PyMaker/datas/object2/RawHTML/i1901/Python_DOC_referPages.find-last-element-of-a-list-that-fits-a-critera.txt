<div class="post-text" itemprop="text">
<p>You could do something like this:</p>
<pre><code>matrix = [
    [[0], 'whatever0'],
    [[1], 'whatever1'],
    [[1, 0], 'whatever10'],
    [[1, 1], 'whatever11'],
    [[1, 2], 'whatever12'],
    [[1, 3], 'whatever13'],
    [[1, 4], 'whatever14'],
]

result = next((e for e in reversed(matrix) if len(e[0]) == 1), None)
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[[1], 'whatever1']
</code></pre>
<p><strong>Explanation</strong></p>
<p>First iterate over the list backwards using <a href="https://docs.python.org/3/library/functions.html#reversed" rel="nofollow noreferrer">reversed</a> and return only the elements that have <code>len(e[0]) == 1</code>. Then with <a href="https://docs.python.org/3/library/functions.html#next" rel="nofollow noreferrer">next</a> get only the first result. </p>
</div>
