<p>How can I add an already existing list into a dictionary?</p>
<p>Specifically, if I have:</p>
<pre><code>lst = [["Hi"],["Monty"],["Python"],["snake"],["Anaconda"]]
</code></pre>
<p>And I was looking to get:</p>
<pre><code>d = {0: "Hi", 1: "Monty", 2: "Python", 3: "snake", 4: "Anaconda"}
</code></pre>
<p>How would I go about doing this?</p>
</div>

<div class="post-text" itemprop="text">
<p>The easiest way is with a <a href="https://www.datacamp.com/community/tutorials/python-dictionary-comprehension" rel="nofollow noreferrer">dictionary comprehension</a>. You can get the index of the sublist using the <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer">enumerate</a> function.</p>
<pre><code>lst = [["Hi"],["Monty"],["Python"],["snake"],["Anaconda"]]
d = {i: sublist[0] for i, sublist in enumerate(lst)}
</code></pre>
<p>You can also add each list item using a for loop.</p>
<pre><code>d = {}
for i, sublist in enumerate(lst):
    d[i] = sublist[0]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>enumerate</code>:</p>
<pre><code>newdict = {}
for i, j in enumerate(lst):
    newdict[j] = i[0]
print(newdict)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have a list of lists where each of the inner lists is holding one string.  To turn that into a dictionary holding only the string, you will need to explicitly iterate through each item.  You can do this with a dictionary comprehension.</p>
<pre><code>d = {i: val[0] for i, val in enumerate(lst)}
</code></pre>
<p>or </p>
<pre><code>d = {(i, val[0]) for i, val in enumerate(lst)}
</code></pre>
<p>Here we iterate over each index, inner list pair with val[0] taking the string out of the inner list.</p>
