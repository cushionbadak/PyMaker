<div class="post-text" itemprop="text">
<p>Here's a nested list comprehension that will achieve what you want:</p>
<pre><code>count = len([e for x in list for e in x if e.lower() in 'aeiou'])
</code></pre>
<p>This is equivalent to:</p>
<pre><code>count = 0
for word in list:
    for char in word:
        if char.lower() in 'aeiou':
            count += 1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This could be another way to find the vowels of each string element in your list.</p>
<pre><code>def countvowel(str):
        num_vowel = 0
        for char in str:
            if char in "aeiouAEIOU":
               num_vowel = num_vowel + 1
        return num_vowel

    list = ['lola', 'anna','mary']
    vowel_count = {}
    for str in list:
        vowel_count[str] = countvowel(str)

    print(vowel_count)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>{'lola': 2, 'anna': 2, 'mary': 1}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use Python's <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter()</code></a> function to strip non vowels from the string and then get the length of the remaining string:</p>
<pre><code>for test in ['lola', 'anna', 'mary']:
    print(len(list(filter(lambda x: x in 'AEIOUaeiou', test))))
</code></pre>
<p>This would print:</p>
<pre><code>2
2
1
</code></pre>
