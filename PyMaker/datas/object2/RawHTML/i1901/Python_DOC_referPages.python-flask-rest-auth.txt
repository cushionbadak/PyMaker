<div class="post-text" itemprop="text">
<p>Let's say you want to use the <a href="https://docs.python.org/3/library/configparser.html" rel="nofollow noreferrer">configparser</a>
Your conf file looks like this:</p>
<pre><code>[user.auth]
username = my
password = password
</code></pre>
<p>In your code you should use it like this :</p>
<pre><code>import configparser


config = configparser.ConfigParser()
config.read("C:\myconf\defaults.cfg")
</code></pre>
<p>then you should get your credentials like this:</p>
<pre><code>username = config.get('user.auth', 'username')
password = config.get('user.auth', 'password')
</code></pre>
