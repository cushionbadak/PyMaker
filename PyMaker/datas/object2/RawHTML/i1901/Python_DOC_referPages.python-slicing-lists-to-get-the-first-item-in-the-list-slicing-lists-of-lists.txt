<p>How do I slice a list to get rid of "Hello","World" and "Monty" in each list of lists?</p>
<p>I may have worded that incorrectly, but this is what I mean:</p>
<pre><code> lst1 = [["Hello", 1,2,3], ["World",4,5,6],["Monty",7,8,9]]
</code></pre>
<p>And I want to get this:</p>
<pre><code>lst2 = [[1,2,3],[4,5,6],[7,8,9]]
</code></pre>
<p>Thank you for your help.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can get a slice of a list <code>lst</code> starting with the second element using <code>lst[1:]</code>. To do it for each sublist you can use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; lst1 = [["Hello", 1, 2, 3], ["World", 4, 5, 6], ["Monty", 7, 8, 9]]
&gt;&gt;&gt; lst2 = [lst[1:] for lst in lst1]
&gt;&gt;&gt; lst2
[[1, 2, 3], [4, 5, 6], [7, 8, 9]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can reach it with a list comprehension and get in each iteration the last elements beside the first of the nested list by using the [1:] selector.</p>
<pre><code>lst1 = [["Hello", 1,2,3], ["World",4,5,6],["Monty",7,8,9]]

lst2 = [item[1:] for item in lst1]
print (lst2)
# [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
</code></pre>
