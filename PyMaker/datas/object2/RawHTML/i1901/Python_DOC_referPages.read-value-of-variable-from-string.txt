<div class="post-text" itemprop="text">
<p>To make the code below run, I placed your two lines into a file called <code>file_name.html</code>. Here are two ways of extracting the value of <code>height</code>.</p>
<p><strong>With BeautifulSoup</strong></p>
<pre><code>from bs4 import BeautifulSoup

with open('file_name.html', 'r') as f:
    soup = BeautifulSoup(f, 'html5lib')
    for img_tag in soup.find_all('img'):
        print(img_tag.get('height'))
</code></pre>
<p><strong>With regular expressions</strong></p>
<pre><code>import re

with open('file_name.html', 'r') as f:
    lines = f.readlines()
    regex = '(height=")(\d*)(")'  # 2nd regex group captures the value of height
    heights = [re.search(regex, l).group(2) for l in lines]
    print(heights)
</code></pre>
<p>Note that this particular Regex example only catches the first <code>height</code> value of each line. </p>
