<pre><code>ftp.sendcmd()

<div class="post-text" itemprop="text">
<p>To send your own command use <a href="https://docs.python.org/3/library/ftplib.html#ftplib.FTP.sendcmd" rel="nofollow noreferrer"><code>FTP.sendcmd</code></a>. This function takes a single argument which is the whole command string. Arguments are separated by spaces in FTP commands, i.e.</p>
<pre><code>ftp.sendcmd(" ".join([cmd, args1, arg2, arg3, arg4, arg5]))
</code></pre>
