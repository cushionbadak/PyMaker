<hr/>
<p>Analizing your code, there are a few issues:</p>
<ol>
<li>you don't need to call <code>f.close()</code> if you use a <code>with</code> statement, the file will be closed automatically when exiting the <code>with</code> block</li>
<li>your print statement should go inside the <code>with</code> block, as the variable <code>read_data</code> is defined inside it</li>
<li>you would need to iterate <code>read_data</code> (<code>for line in read_data:</code>), because you already used <code>f.read()</code> so that iterating over <code>f</code> will no yield anything</li>
<li>you are overwriting <code>record</code> each time, so I don't know why you want to it that way</li>
</ol>
