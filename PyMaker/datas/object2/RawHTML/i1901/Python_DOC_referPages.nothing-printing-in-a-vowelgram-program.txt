<p>You could use <code>all</code> to check if all of vowels are in the string and are exactly once:</p>
<pre><code>n = input('Enter string: ')
n = n.lower()

if all(x in n and n.count(x) == 1 for x in {'a', 'e', 'i', 'o', 'u'}):
    print('It is a vowelgram.')
else:
    print('It is not a vowelgram.')
</code></pre>

<div class="post-text" itemprop="text">
<p>If it is not printing anything, for some input, you must be ending up in the <em>missing</em> <code>else</code> branch of the inner <code>if</code>:</p>
<pre><code>if(c==5):
    if(d==0):
        print("The number is a vowelgram")
    # c==5, but d!=0
    # here , you will get no output!
else:  # this else applies only to the outer if: c!=5
    print("The number isnt a vowelgram")
</code></pre>
<p>You can remedy that, e.g. by using the following construct covering all (<em>other</em>) cases:</p>
<pre><code>if c==5 and d==0:
    print("The number is a vowelgram")
else:
    print("The number isnt a vowelgram")
</code></pre>
<p>Generally however, your nested loops are not optimal performance-wise. You could use a <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>collections.Counter</code></a> to obtain a result in linear time:</p>
<pre><code>from collections import Counter

c = Counter(x for x in n if x in 'aeiou')
if len(c) == 5 and all(v==1 for v in c.values()):
    print('vowelgram')
else:
    print('not vowelgram')
</code></pre>
