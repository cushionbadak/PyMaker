<div class="post-text" itemprop="text">
<p>I'd like to split a <code>string = ['O little forests,']</code> so that the output includes both words and spaces, with the last comma appended to the last words. Here's what the desired output should look like:</p>
<p><code>output = ['O', ' ', 'little', ' ', 'forests,']</code></p>
<p>I was able to split the given string into a list of words with spaces excluded by using <code>line.split()</code>. I welcome your suggestions! </p>
</div>
<div class="post-text" itemprop="text">
<p>You may do this work using <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer"><code>re</code></a> like following code.</p>
<pre><code>import re
string = ['O little forests'] 
for a  in string:
  print(re.split(r'(\s+)', a))
</code></pre>
<p>Output:</p>
<pre><code>['O', ' ', 'little', ' ', 'forests']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer">groupby</a>:</p>
<pre><code>from itertools import groupby

sentence = 'O little forests'

result = [''.join(v) for k, v in groupby(sentence, key=str.isspace)]
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>['O', ' ', 'little', ' ', 'forests']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use re (regex):</p>
<pre><code>import re

sentence = 'O little forests'

re.split("( )", sentence) # ['O', ' ', 'little', ' ', 'forests']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can add a space element between each word like this in a list comprehension:</p>
<pre><code>s = 'O little forest'
# puts ' ' element after each word.
st = [k for j in [[e, ' '] for e in s.split()] for k in j]
st.pop() # drop last.
st
&gt;&gt;&gt; ['O', ' ', 'little', ' ', 'forest']
</code></pre>
</div>
