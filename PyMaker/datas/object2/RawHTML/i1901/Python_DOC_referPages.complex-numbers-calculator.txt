<p>So, if I type (4*9j**2) into the console, it returns (-324+0j) Is there any way to re-create this with code, basically, take input and simplify it as if you just typed it into the console (j acts as the imaginary number i)</p>
<p>Thanks in advance!</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the built in function <a href="https://docs.python.org/3/library/functions.html#eval" rel="nofollow noreferrer">eval()</a>.</p>
<pre><code>&gt;&gt;&gt; 4*9j**2
(-324+0j)
&gt;&gt;&gt; equation = "4*9j**2"
&gt;&gt;&gt; eval(equation)
(-324+0j)
</code></pre>
</div>
