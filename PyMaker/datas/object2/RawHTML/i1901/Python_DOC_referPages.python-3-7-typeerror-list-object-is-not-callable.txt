<p>Im hoping somebody can help me out in why the function I'm writing to convert a string into a list of list of binary numbers isn't working.</p>
<pre><code>def string2bin(str):
    bin = []
    for n in range(len(str)):
        char = ord(str[n])
        char = bin(char)
        char[2:].zfill(8)
        bin.append(list(map(int, char)))

    return bin
</code></pre>
<p>Here's the error I'm getting when running the function with these arguments.</p>
<pre><code>string2bin('abc')

    char = bin(char)
TypeError: 'list' object is not callable
</code></pre>
<p>Thanks for the help.</p>

<div class="post-text" itemprop="text">
<p><code>str</code> is a reserved keyword in python. Use another name for your variable.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>bin()</code> is a <a href="https://docs.python.org/3/library/functions.html#bin" rel="nofollow noreferrer">Python built-in method</a> but you named your final list <code>bin = []</code> so when you get to <code>char = bin(char)</code> you've overwritten it. </p>
<p>The error you get is because <code>bin</code> now refers to a list which you then try to call like a function.</p>
<p>Just rename <code>bin</code> to something else.</p>
<p>Also, <code>char</code> is not a great name for <code>ord(str[n])</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You have defined first bin as a list and then you are using it as a function.
I think you want to do <code>char = bin[char]</code> instead</p>
