<p>You could do it like the example below:</p>
<pre><code>dlist = [23.0, 28.6, 32.9, 34.9, 34.2, 31.0, 25.9, 20.1, 15.0, 11.8, 11.1, 13.1, 17.4, 23.0, 28.6, 32.9, 34.9, 34.2, 31.0, 25.9, 20.1, 15.0, 11.8, 11.1, 13.1, 17.4]

checklist = []
for idx, val in enumerate(dlist):
    if val &gt;= 25:
        checklist.append(idx);

print(checklist);
</code></pre>
<p>The reason your code wasn't working correctly is because <code>index</code> returns the index of <em>first occurance</em> in that list, causing values later in the list to show up with a wrong index because they get the <code>index</code> of the item with the same value that's also appearing earlier in the list.</p>

<div class="post-text" itemprop="text">
<p>Using <code>index</code> returns the index of the first occurrence of that item in your list (<a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/datastructures.html</a>). This is why you only get the earlier appearing indexes, to gather all the indexes you would want to use <code>enumerate</code></p>
<p>Use <code>enumerate</code> and only take the indexes of items with a value greater than <code>25</code></p>
<pre><code>res = [i for i, v in enumerate(dlist) if v &gt; 25]
# [1, 2, 3, 4, 5, 6, 14, 15, 16, 17, 18, 19]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As an alternative, you could use numpy's <a href="https://docs.scipy.org/doc/numpy-1.13.0/reference/generated/numpy.where.html" rel="nofollow noreferrer">where</a> function:</p>
<pre><code>import numpy as np

np.where(np.asarray(dlist) &gt; 25)[0]

# array([ 1,  2,  3,  4,  5,  6, 14, 15, 16, 17, 18, 19])
</code></pre>
