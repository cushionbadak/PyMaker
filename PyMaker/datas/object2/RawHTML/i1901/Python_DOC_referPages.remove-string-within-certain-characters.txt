<div class="post-text" itemprop="text">
<p>If i have a string such as </p>
<blockquote>
<p>string = "This text is &amp;This text is also unknown &amp;unknown to the program"</p>
</blockquote>
<p>How do i remove everything within the &amp; character?</p>
</div>
<div class="post-text" itemprop="text">
<p>A simple approach using <a href="https://docs.python.org/3/library/stdtypes.html#str.find" rel="nofollow noreferrer">find</a>:</p>
<pre><code>s = "This text is &amp;This text is also unknown &amp;unknown to the program"

start = s.find('&amp;')
end = s.find('&amp;', start+1)
result = s[:start] + s[end+1:]
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>This text is unknown to the program
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use regedit to remove it (I assume you also want <code>&amp;</code> gone):</p>
<pre><code>import re
string = "This text is &amp;This text is also unknown &amp;unknown to the program"
sEdit = re.sub('&amp;.*&amp;', '', string)
print(sEdit)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Or <code>replace</code>+<code>find</code>:</p>
<pre><code>print(string.replace(string[string.find('&amp;'):string.find('&amp;',string.find('&amp;')+1)+1],''))
</code></pre>
<p>Output:</p>
<pre><code>This text is unknown to the program
</code></pre>
