<div class="post-text" itemprop="text">
<p>Python <code>file.readlines()</code> gives you a list of strings, bu by default it doesn't strip newlines.</p>
<p>You either should do it yourself (i.e. with <code>line.strip('\n').split(',')</code>) or use <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer">csv module</a> to read file, as it looks like csv format.</p>
</div>
<div class="post-text" itemprop="text">
<p>use <code>re</code> module to replace <code>\n</code>:</p>
<pre><code>import re
re.sub('\n','',i)
</code></pre>
