<pre><code> {1: [33.00, 5.30, 50.00],
  2: [17.00, 0.50, 10.00]}

 2: [Decimal('17.00'), Decimal('40.50'), Decimal('10000.00')]}

<div class="post-text" itemprop="text">
<p>Apply <a href="https://docs.python.org/3/library/functions.html#float" rel="nofollow noreferrer">float</a> to values:</p>
<pre><code>from decimal import Decimal

data = [(1, Decimal('33.00'), Decimal('5.30'), Decimal('50.00')),
  (2, Decimal('17.00'), Decimal('0.50'), Decimal('10.00'))]

load_dict = {key: list(map(float, values))  for key, *values in data}

print(load_dict)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>{1: [33.0, 5.3, 50.0], 2: [17.0, 0.5, 10.0]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>you could use a dict-comprehension with a cast to <code>float</code> like this:</p>
<pre><code>from decimal import Decimal

lst = [(1, Decimal('33.00'), Decimal('5.30'), Decimal('50.00')),
  (2, Decimal('17.00'), Decimal('0.50'), Decimal('10.00'))]

ret = {key: [float(f) for f in values] for key, *values in lst}
print(ret)
# {1: [33.0, 5.3, 50.0], 2: [17.0, 0.5, 10.0]}
</code></pre>
