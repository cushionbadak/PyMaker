<p>How to Find the dictionary from List which has key-pair 'isGeo':True</p>
<pre><code>dimensions = [{'key': 2600330, 'id': 'location', 'name': 'Location', 'isGeo': True, 'geoType': 'region'}, {'key': 2600340, 'id': 'subject', 'name': 'Subject', 'isGeo': False, 'geoType': None}, {'key': 2600350, 'id': 'measure', 'name': 'Measure', 'isGeo': False, 'geoType': None}]
</code></pre>
<p>I want to below result: </p>
<pre><code>{'key': 2600330, 'id': 'location', 'name': 'Location', 'isGeo': True, 'geoType': 'region'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#next" rel="nofollow noreferrer"><code>next</code></a> with a <a href="https://www.python.org/dev/peps/pep-0289/" rel="nofollow noreferrer">generator expression</a>:</p>
<pre><code>res = next((d for d in dimensions if d['isGeo']), None)

{'key': 2600330, 'id': 'location', 'name': 'Location', 'isGeo': True, 'geoType': 'region'}
</code></pre>
<p>Since you tagged <a class="post-tag" href="/questions/tagged/pandas" rel="tag" title="show questions tagged 'pandas'">pandas</a>, you can also use Pandas:</p>
<pre><code>import pandas as pd

df = pd.DataFrame(dimensions)
res = df.loc[df['isGeo']].iloc[0].to_dict()
</code></pre>
<p>The above solutions assume you want only the <em>first</em> dictionary satisfying your condition. If you want a list of dictionaries use:</p>
<pre><code>res = [d for d in dimensions if d['isGeo']]
res = df.loc[df['isGeo']].to_dict('records')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Another way is <a href="https://www.programiz.com/python-programming/list-comprehension" rel="nofollow noreferrer"><code>list comprehension</code></a></p>
<pre><code>dimensions = [{'key': 2600330, 'id': 'location', 'name': 'Location', 'isGeo': True, 'geoType': 'region'}, {'key': 2600340, 'id': 'subject', 'name': 'Subject', 'isGeo': False, 'geoType': None}, {'key': 2600350, 'id': 'measure', 'name': 'Measure', 'isGeo': False, 'geoType': None}]

geoDimension = [dimension for dimension in dimensions if dimension['isGeo']][0]
</code></pre>
<p>Output : <code>{'key': 2600330, 'id': 'location', 'name': 'Location', 'isGeo': True, 'geoType': 'region'}</code></p>
