<div class="post-text" itemprop="text">
<p>You likely need to adjust your <code>sys.path</code>.</p>
<p>Here's a complete example showing how to create a module and import it into Colab:
<a href="https://colab.research.google.com/drive/1PtYW0hZit-B9y4PL978kV2ppJJPhjQua" rel="nofollow noreferrer">https://colab.research.google.com/drive/1PtYW0hZit-B9y4PL978kV2ppJJPhjQua</a></p>
<p>The key line in this example for you to add, I believe, is:</p>
<pre><code># Add the local_modules directory to the set of paths
# Python uses to look for imports.
import sys
sys.path.append('local_modules')
</code></pre>
<p>You would need to adjust the path from <code>'local_modules'</code> to something like <code>/content/gdrive/My\ Drive/Colab</code>.</p>
