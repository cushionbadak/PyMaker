<pre><code>def union(x,y):

    for e in y:

a = [1,2,3]

print a

<div class="post-text" itemprop="text">
<p>Just modify your <code>union</code> function to <code>return</code> the list <code>x</code>:</p>
<pre><code>def union(x,y):
    for e in y:
        if e not in x:
            x = x + [e]
    return x
</code></pre>
<p>Then you can call:</p>
<pre><code>&gt;&gt;&gt; a = union(a,b)
&gt;&gt;&gt; a
[1, 2, 3, 4, 6]
</code></pre>
<p>A shorter approach (with your same logic) would be using list comprehension:</p>
<pre><code>&gt;&gt;&gt; a += [i for i in b if i not in a]
&gt;&gt;&gt; a
[1, 2, 3, 4, 6]
</code></pre>
<p>And the <em>super-pythonic</em>:</p>
<pre><code>&gt;&gt;&gt; list(set(a) | set(b))
[1, 2, 3, 4, 6]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It's worth pointing out that Python has the <a href="https://docs.python.org/3/library/stdtypes.html?highlight=set#set" rel="nofollow"><code>set</code></a> data type for exactly this purpose. And it supports a <code>.union(otherset)</code> method!</p>
