<div class="post-text" itemprop="text">
<p>If you want to convert the byte string <code>b'\xbb\xf9\xb2\xee'</code> to the proper characters, you can call <code>decode</code> on the byte string and specify one of <a href="https://docs.python.org/3/library/codecs.html#standard-encodings" rel="nofollow">standard codecs</a>. For example:</p>
<pre><code>x = b'\xbb\xf9\xb2\xee'
x.decode('gbk')  # gbk is listed as the "unified Chinese" codec

'基差'
</code></pre>
<p>I'm just guessing on the codec and you may want to experiment with that.</p>
<p>It's not clear which Python 3.x version you're using, so I tested this with Python 3.1, 3.2, 3.3, and 3.4 and the results are the same for all versions.</p>
