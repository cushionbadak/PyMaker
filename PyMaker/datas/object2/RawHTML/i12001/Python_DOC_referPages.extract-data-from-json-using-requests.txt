<p><a href="http://docs.python-requests.org/en/master/" rel="nofollow">Reading the documentation</a>, surprisingly the <em>first</em> example from the first google match for <code>requests</code> on the first screen tells how to get the json body:</p>
<pre><code>&gt;&gt;&gt; r3 = requests.get('www.websiteexample.com/api', params=header )
&gt;&gt;&gt; struct = r3.json()
</code></pre>
<p>Then you could for example print all top-level objects like this</p>
<pre><code>&gt;&gt;&gt; for k, v in struct.items():
...     print(k, v)
</code></pre>
<p>Or <code>FoundCategories</code>:</p>
<pre><code>&gt;&gt;&gt; struct['FoundCategories']
[]
</code></pre>

<p>Not very well structured question but you could use the json_normalize from pandas <code>from pandas.io.json import json_normalize</code> to extract data from json. This is not from requests though. </p>
<pre><code>import urllib, json
from pandas.io.json import json_normalize

url = "www.websiteexample.com/api"
response = urllib.urlopen(url)
data = json.loads(response.read())
data2 = json_normalize(data)
</code></pre>
