<p>I have a dictionary as follows:</p>
<pre><code>dic=[{'a':1,'b':2,'c':3},{'a':9,'b':2,'c':2},{'a':5,'b':1,'c':2}]
</code></pre>
<p>I would like to filter out those dictionaries with recurring values for certain keys, such as in this case the key 'b' which has duplicate values in the first and second dictionaries in the list. I would like to remove the second entry</p>
<p>Quite simply, I would like my filtered list to look as follows:</p>
<pre><code>filt_dic=[{'a':1,'b':2,'c':3},{'a':5,'b':1,'c':2}]
</code></pre>
<p>Is there a pythonic way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use another dictionary (or <a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow"><code>defaultdict</code></a>) to keep track of what values you have already seen for what keys. This dictionary will hold one <code>set</code> (for fast lookup) for each key of the original dict.</p>
<pre><code>dic=[{'a':1,'b':2,'c':3},{'a':9,'b':2,'c':2},{'a':5,'b':1,'c':2}]

filt_dic = []
for d in dic:
    if not any(d[k] in seen[k] for k in d):
        filt_dic.append(d)
        for k in d:
            seen[k].add(d[k])

print(filt_dic)
</code></pre>
<p>Afterwards, <code>filt_dic</code> is <code>[{'a': 1, 'c': 3, 'b': 2}, {'a': 5, 'c': 2, 'b': 1}]</code> and <code>seen</code> is <code>{'a': set([1, 5]), 'c': set([2, 3]), 'b': set([1, 2])})</code>.</p>
