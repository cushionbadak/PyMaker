<p>I have a simple code line in Python, use too many <code>+</code> is not nice</p>
<pre><code>filename = path + '/static/' + folder + '/' + name + 'html'
</code></pre>
<p>Can be like that in <code>php</code></p>
<pre><code>$filename = "$path/static/$folder/{$name}html"
</code></pre>
<p>So how we write shorten in <code>Python</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do this in python which would be much more readable,</p>
<pre><code>filename = "{}/static/{}/{}html".format(path, folder, name)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>What you want is join the path components. To do this you can use the <a href="https://docs.python.org/3/library/os.path.html#os.path.join" rel="nofollow"><code>os.path.join()</code></a> function which uses <code>os.sep</code> as separator.</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; path = 'workspace'
&gt;&gt;&gt; folder = 'stackoverflow'
&gt;&gt;&gt; name = 'layout'
&gt;&gt;&gt; os.path.join(path, 'static', folder, name, 'html')
'workspace/static/stackoverflow/layout/html'
</code></pre>
<p>You can also use <a href="https://docs.python.org/3/library/pathlib.html#pure-paths" rel="nofollow"><code>PurePath</code></a> if you are using Python 3.4 or newer.</p>
<pre><code>from pathlib import PurePath

p = PurePath(path)
filename = str(p / static / folder / name / 'html')
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; from pathlib import PurePath
&gt;&gt;&gt; p = PurePath(path)
&gt;&gt;&gt; filename = str(p / 'static' / folder / name / 'html')
&gt;&gt;&gt; filename
'workspace/static/stackoverflow/layout/html'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>os.path.join</code> because it will work on all operating systems</p>
