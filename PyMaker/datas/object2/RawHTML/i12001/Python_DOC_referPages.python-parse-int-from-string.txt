<p>Let's say I have bunch of strings which start with 'name' and are followed by a number of any length.</p>
<p>How can I parse out the number in the string?  </p>
<p>Is <code>regex</code> the only way or is there a built-in module that can accomplish this task?</p>

<div class="post-text" itemprop="text">
<p>In Python 3, you could do the following:</p>
<pre><code>import string

for test in ['name1', 'name2', 'name45', 'name1231231', '123test']:
    print(int(test.strip(string.ascii_letters)))
</code></pre>
<p>Giving you:</p>
<pre><code>1
2
45
1231231
123
</code></pre>
<p><code>string.ascii_letters</code> gives you a string containing all upper and lowercase letters. Python's <a href="https://docs.python.org/3/library/stdtypes.html?highlight=strip#str.strip" rel="nofollow noreferrer"><code>strip()</code></a> function takes a string specifying the set of characters to be removed, which is this case is all alpha characters, thus leaving just the numbers behind.</p>
<p>Note: This would not be suitable for a string such as <code>123name456</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you know that the prefix is <code>name</code>, then you can either remove just that string, or you can skip the first four letters, like so:</p>
<pre><code>s = 'name123'
print int(s.replace('name',''))

s = 'name123'
print int(s[4:])
</code></pre>
