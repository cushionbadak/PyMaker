<p><a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strptime" rel="nofollow"><code>strptime</code></a> returns a <code>datetime</code> instance <em>including year, month, and day</em>. If you don't provide these in your input, they will be <code>1900-01-01</code>.</p>
<p>Comparing two <code>datetime</code> instances works, printing them with <code>strftime</code>, too.</p>
<pre><code>import datetime

format = "%H:%M:%S:%f"
d1 = "14:08:33:806443"
d2 = "14:08:02:318642"

p1 = datetime.datetime.strptime(d1, format)
p2 = datetime.datetime.strptime(d2, format)

print(p1 &gt; p2)

print(p1.strftime(format))
print(p2.strftime(format))

print(p1 - p2)
</code></pre>
<p>Output:</p>
<pre><code>True
14:08:33:806443
14:08:02:318642
0:00:31.487801
</code></pre>
