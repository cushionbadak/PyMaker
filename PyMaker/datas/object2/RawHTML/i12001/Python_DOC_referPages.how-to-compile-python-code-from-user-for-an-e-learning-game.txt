<div class="post-text" itemprop="text">
<p>You can use the <code>py_compile</code> module for that. First, dump your code in a text file, for example:</p>
<pre><code>with open('test.py', 'w') as f:
  f.writelines('print("Hello World")')
</code></pre>
<p>Then, you can do the following to compile 'test.py':</p>
<pre><code>import py_compile
py_compile.compile('test.py')
</code></pre>
<p>If <code>test.py</code> has no error, <code>py_compile.compile</code> will succeed and generate a <code>.pyc</code> file called <code>test.pyc</code>. However, if <code>test.py</code> has some error, <code>py_compile.compile</code> will raise an exception, which you could handle and return to your user. For example, if you <code>test.py</code> has a missing <code>(</code> braket, <code>py_compile.compile</code> generated the following error for me:</p>
<pre><code>&gt;&gt;&gt; py_compile.compile('test.py')
  File "hello_world.py", line 1
    print "Hello World")
                   ^
SyntaxError: invalid syntax
</code></pre>
</div>
