<div class="post-text" itemprop="text">
<p>I have a list with <code>int</code> values that I would like to add to each other and log the end value. So far I could create a working solution, but it's not so elegant and I would be happy if somebody could show me a smarter solution to achieve the same result. </p>
<pre><code>numberList  = (list(string_dict.values()))
numz = []
placeholder = 0

for x in numberList:
    numz.append(int(x))

for y in numz:
    placeholder = placeholder + y
print (placeholder)

# [1,2,3]   
# result: 6
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can take out both loops by using the <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow">map()</a> and <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow">sum()</a> functions:</p>
<pre><code>numberList  = list(string_dict.values())
numz = []
placeholder = 0

numz = list(map(int, numberList))

placeholder = sum(numz)
print (placeholder)
</code></pre>
<p>You don't really need to have <code>numberList</code> and <code>numz</code> in there, though.  Just do this:</p>
<pre><code>placeholder = sum(map(int, string_dict.values()))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>sum</code> function:</p>
<pre><code>print(sum(int(x) for x in string_dict.values()))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use reduce function also:</p>
<pre><code>from functools import reduce

print reduce( (lambda x, y: x + y), string_dict.values() )
</code></pre>
