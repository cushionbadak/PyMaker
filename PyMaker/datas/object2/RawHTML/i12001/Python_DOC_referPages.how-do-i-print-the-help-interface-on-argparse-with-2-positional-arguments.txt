<div class="post-text" itemprop="text">
<p>Perhaps what you're after is a <a href="https://docs.python.org/3/library/argparse.html#argparse.ArgumentParser.add_mutually_exclusive_group" rel="nofollow">mutually exclusive group</a>.</p>
<pre><code>parser = argparse.ArgumentParser(description="About the Solar System") # initialises argparse

parser.add_argument("--orderby", help="displays the planets ordered by mass, largest to smallest", action='store_true')
group = parser.add_mutually_exclusive_group(required=True)
group.add_argument("--list", help="displays the planets in alphabetical order", action='store_true')
group.add_argument("planet", help="displays information on the chosen &lt;planet&gt; and opens a wiki page", nargs="?", action="store")

args = parser.parse_args()
</code></pre>
<p>which would result in</p>
<pre><code> % python3 args.py 
usage: args.py [-h] [--orderby] (--list | planet)
args.py: error: one of the arguments --list planet is required
</code></pre>
<p>and</p>
<pre><code> % python3 args.py --list
Namespace(list=True, orderby=False, planet=None)

 % python3 args.py asdf  
Namespace(list=False, orderby=False, planet='asdf')

 % python3 args.py --list asdf
usage: args.py [-h] [--orderby] (--list | planet)
args.py: error: argument planet: not allowed with argument --list
</code></pre>

<span class="comment-copy">Your current argument set is rather odd. Do you really need the program to both be able to list (a kind of subcommand) and display information on chosen planet at the same time? Either make subcommands of list and display, or make list a flag, even if you say it is out of the question.</span>
<span class="comment-copy">Normally we expect <code>argparse</code> questions to include the <code>parser</code> as defined so far, and some sample commandlines and the desired parse. It's hard to tell from text like this what you want and what you have tried.</span>
<span class="comment-copy">An argument, positional or flagged, can take a <code>choices</code> parameter, e.g. <code>choices=['mercury', 'venus', 'earth', ...]</code>.</span>
