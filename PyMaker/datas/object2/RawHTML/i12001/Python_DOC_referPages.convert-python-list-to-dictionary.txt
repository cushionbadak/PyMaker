<div class="post-text" itemprop="text">
<p>Keep it a bit simpler than that, you want to loop over your list, and then assign your iterator <code>i</code> (which will be each value in your list) as the key to each dictionary entry. </p>
<pre><code>l = ['a', 'b', 'c', 'd']

d = {}
for i in l:
    d[i] = []


print(d) # {'a': [], 'c': [], 'b': [], 'd': []}
</code></pre>
<p>With the above understood, you can now actually simplify this in to one line as: </p>
<pre><code>{k: [] for k in l}
</code></pre>
<p>The above is called a dictionary comprehension. You can read about it <a href="https://www.python.org/dev/peps/pep-0274/" rel="nofollow">here</a></p>
