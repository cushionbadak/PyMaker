    list_len = 2 ** bit_len
    return ['{0:0{1}b}'.format(i, bit_len) for i in range(list_len)]

print(bin_list(3))

<div class="post-text" itemprop="text">
<p>See the python <a href="https://docs.python.org/3/library/string.html#formatstrings" rel="nofollow">format string documentation</a></p>
<p>The value before the <code>:</code> specifies the field to format (in this case, 0, so the first param)</p>
<pre><code>'{0}'.format(2) == '2'
</code></pre>
<p>The value after the colon specifies the format. And it gets fancy here. First the <code>b</code> specifies binary rendering</p>
<pre><code>'{0:b}'.format(2) == '10'
</code></pre>
<p>Then you can specify a minimum width, which here is taken from the second parameter:</p>
<pre><code>'{0:5b}'.format(2) == '   10'
'{0:{1}b}'.format(2, 5) == '   10'
</code></pre>
<p>And lastly you can add a 0 before the width to say the padding should be <code>0</code>s and not spaces:</p>
<pre><code>'{0:05b}'.format(2) == '00010'
'{0:0{1}b}'.format(2, 5) == '00010'
</code></pre>
</div>
