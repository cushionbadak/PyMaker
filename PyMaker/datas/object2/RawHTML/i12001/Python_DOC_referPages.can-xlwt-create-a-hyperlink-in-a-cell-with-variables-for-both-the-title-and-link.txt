<p>For instance, </p>
<p>How would the following line be changed such that "test" is variable T and "<a href="http://google.com" rel="nofollow">http://google.com</a>" is variable L?</p>
<pre><code>ws.write(0, 0, xlwt.Formula('"test " &amp; HYPERLINK("http://google.com")'))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>T = 'test'
L = 'http://google.com'
formula = '"{} " &amp; HYPERLINK("{}")'.format(T, L)
ws.write(0, 0, xlwt.Formula(formula))
</code></pre>
<p>This uses <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format()</code></a> to insert the values of variables <code>T</code> and <code>L</code> in to the string <code>formula</code>. Following the above assignments <code>formula</code> will contain:</p>
<pre><code>"test " &amp; HYPERLINK("http://google.com")
</code></pre>
<p>It's also possible to do it without using the temporary variable:</p>
<pre><code>ws.write(0, 0, xlwt.Formula('"{} " &amp; HYPERLINK("{}")'.format(T, L)))
</code></pre>
<p><strong>Update</strong></p>
<p>The code above answers the question as it was stated, however, the OP actually requires that "test" be displayed as the link and the URL be the target, so:</p>
<pre><code>T = 'test'
L = 'http://google.com'
formula = 'HYPERLINK("{}", "{}")'.format(L, T)
ws.write(0, 0, xlwt.Formula(formula))

&gt;&gt;&gt; formula
'HYPERLINK("http://google.com", "test")'
</code></pre>
</div>
