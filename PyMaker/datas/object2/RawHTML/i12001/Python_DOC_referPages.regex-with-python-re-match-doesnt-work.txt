<div class="post-text" itemprop="text">
<p>Seems like it's because you're using <a href="https://docs.python.org/3/library/re.html#re.match" rel="nofollow"><code>re.match()</code></a> instead of <a href="https://docs.python.org/3/library/re.html#re.search" rel="nofollow"><code>re.search()</code></a>. <code>re.match()</code> Searches from the start of the string while <code>re.search()</code> searches anywhere. This works:</p>
<pre><code>sText ="""&lt;firstName name="hello morning" id="2342"/&gt;
&lt;mainDescription description="cooking food blog 5 years"/&gt;
&lt;special description="G10X, U16X, U17X, G26X, C32X, G34X, G37X, U39X, C40X, G46X,C49X, U54X, U55X, A58X"/&gt; 
"""
p = re.compile('&lt;mainDescription description=\"([^\"]+)\"\/&gt;')
print re.search(p, sText).group(1)
</code></pre>
<p>By the way, you do not need to escape the quotation marks (<code>"</code>) if you're using <code>'</code> meaning this is enough:</p>
<pre><code>re.search('&lt;mainDescription description="([^"]+)"/&gt;', sText)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>re.match</code> returns a <code>match</code> object, from which you need to retrieve the desired group.</p>
<pre><code>sText ="""&lt;firstName name="hello morning" id="2342"/&gt;
&lt;mainDescription description="cooking food blog 5 years"/&gt;
&lt;special description="G10X, U16X, U17X, G26X, C32X, G34X, G37X, U39X, C40X, G46X,C49X, U54X, U55X, A58X"/&gt; 
"""
r = re.compile("""&lt;mainDescription description="(?P&lt;description&gt;[^"]+)"\/&gt;""")
m = r.match(sText)
print m.group('description')
</code></pre>
<p>Note that it's also possible to access the group using the index (0 in this case) but I prefer to specify a keyword.</p>
