<div class="post-text" itemprop="text">
<p>Another option is <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join()</code></a>:</p>
<pre><code>project = '1,2'
n = 3
project = ','.join((project, str(n)))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/2/library/stdtypes.html#str.format" rel="nofollow"><code>.format</code></a> should help you.</p>
<pre><code>project = "1,2"
project += ',{}'.format(3)
</code></pre>
<p>And if you have a position, you could use slicing :</p>
<pre><code>project = "1,2,4,5"

position = 3

project = project[:position] + ',3' + project[position:]
</code></pre>

<div class="post-text" itemprop="text">
<p>Just convert the number into string using str() and append division symbol and the number you want to insert</p>
<pre><code>project = "1,2"
project += ',' + str(3)
print (project)
</code></pre>

<div class="post-text" itemprop="text">
<p>You can convert it to a list with <code>numbers = project.split(",")</code>.  You can then add another number with <code>numbers.append('3')</code>.  Then make a new string with <code>newproject = ",".join(numbers)</code>.  Full code:</p>
<pre><code>project = "1,2"
numbers = project.split(",")
numbers.append('3')
newproject = ",".join(numbers)
</code></pre>
