<pre><code>{'Female' :[18,36,35,49,19],'Male' :[23,22,6,36,46]}

<div class="post-text" itemprop="text">
<pre><code>{key:(max(d[key]),min(d[key])) for key in d}
</code></pre>
<p>This will return </p>
<pre><code>{'Female': (49, 18), 'Male': (46, 6)}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; d = {'Female' :[18,36,35,49,19],'Male' :[23,22,6,36,46]}
&gt;&gt;&gt; dd = {k:(min(d[k]), max(d[k])) for k in d}

{'Male': (6, 46), 'Female': (18, 49)}
</code></pre>

<div class="post-text" itemprop="text">
<p>Just an example:</p>
<pre><code>dct = {'Female' :[18,36,35,49,19],'Male' :[23,22,6,36,46]}
for key in dct:
    print(key, (min(dct[key]), max(dct[key])))
</code></pre>
<p>will print:</p>
<pre><code># Female (18, 49)
# Male (6, 46)
</code></pre>
<p>I'll leave you the task of finding out how the <code>:</code> is going to be inserted.</p>
<p>The <a href="https://docs.python.org/3/library/functions.html" rel="nofollow noreferrer">Python documentation on built-in functions</a> is very helpful on how <code>min</code>, <code>max</code> and <code>print</code> can be used.</p>
<hr/>
<p>An alternative would be to simply create a <code>minmax</code> function:</p>
<pre><code>&gt;&gt;&gt; def minmax(some_list):
...     return min(some_list), max(some_list)

&gt;&gt;&gt; {key: minmax(values) for key, values in dct.items()}
{'Female': (18, 49), 'Male': (6, 46)}
</code></pre>
<p>Or if you want it faster I recently created a package containing such a function (<a href="https://iteration-utilities.readthedocs.io/en/latest/generated/minmax.html" rel="nofollow noreferrer"><code>iteration_utilities.minmax</code></a>) that requires only one-pass (and thus less comparisons):</p>
<pre><code>&gt;&gt;&gt; from iteration_utilities import minmax
&gt;&gt;&gt; {key: minmax(values) for key, values in dct.items()}
{'Female': (18, 49), 'Male': (6, 46)}
</code></pre>
