<pre><code>s = 'ABCD'

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/2/library/functions.html#unichr" rel="nofollow"><code>unichr</code></a> (<a href="https://docs.python.org/3/library/functions.html#chr" rel="nofollow"><code>chr</code></a> in Python 3.x) to convert <code>int</code> to unicode string:</p>
<pre><code>s = 'ABCD'

for i in range(10):
    s += unichr(0x2713 + i)

print s
</code></pre>
<p>prints <code>ABCD✓✔✕✖✗✘✙✚✛✜
</code></p>
<hr/>
<p>Instead of appending characters, you can use <a href="https://docs.python.org/2/library/stdtypes.html#str.join" rel="nofollow"><code>str.join</code></a> (or <code>unicode.join</code>):</p>
<pre><code>s = 'ABCD' + u''.join(unichr(0x2713 + i) for i in range(10))
</code></pre>
<p>OR</p>
<pre><code>s = 'ABCD' + u''.join(unichr(ch) for ch in range(0x2713, 0x271d))
</code></pre>
<p>OR</p>
<pre><code>s = 'ABCD' + u''.join(map(unichr, range(0x2713, 0x271d)))
</code></pre>
</div>
