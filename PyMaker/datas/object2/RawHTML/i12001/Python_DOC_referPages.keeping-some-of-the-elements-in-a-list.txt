<p>I have been trying to figure out how to find specific words in a list then move these to a new list. Say i have  :</p>
<pre><code>a = ['a','b','c','d','e']
</code></pre>
<p>and I only want to have a, c and e. Is it possible to do this without using .remove() ?</p>
</div>
<div class="post-text" itemprop="text">
<p>Well, I hope I understood your question right.  You want to only select the contents of one list if they are in another list.  You might use <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehensions</a>.  </p>
<p>Here's an example playing at the interactive prompt.  </p>
<pre><code>&gt;&gt;&gt; a = ['a','b','c','d','e']
&gt;&gt;&gt; [x for x in a if x in ['a', 'c', 'e' ]]
['a', 'c', 'e']
</code></pre>
<p>or with clearer naming</p>
<pre><code>&gt;&gt;&gt; list = ['a','b','c','d','e', 'a', 'd', 'e', 'c']
&gt;&gt;&gt; wanted = ['a', 'c', 'e' ]
&gt;&gt;&gt; [x for x in list if x in wanted]
['a', 'c', 'e', 'a', 'e', 'c']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/2/library/functions.html#filter" rel="nofollow"><code>filter()</code></a> builtin function:</p>
<pre><code>&gt;&gt;&gt; lst = ['a','b','c','d','e']
&gt;&gt;&gt;
&gt;&gt;&gt; def keep(item):
...     # Implement whatever criterion you need here
...     return item in ('a', 'c', 'e')
...
&gt;&gt;&gt; filter(keep, lst)
['a', 'c', 'e']
</code></pre>
<p><code>filter</code> will simply return a filtered sequence (a list in your case) without modifying the input sequence.</p>
<p><code>keep</code> in this example just needs to be a function that, given an item, decides whether to keep it or not by returning <code>True</code> (keep) or <code>False</code> (discard).</p>
<hr/>
<p>Note: As pointed out by @tobias_k, in Python 3 <code>filter()</code> will return an iterator. So if you actually do need a list (not just an iterable), you'd need to pass it to the <code>list()</code> constructor like this:</p>
<pre><code>newlist = list(filter(keep, lst))
</code></pre>
