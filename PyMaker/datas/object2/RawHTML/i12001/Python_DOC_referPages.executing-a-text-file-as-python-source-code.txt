<div class="post-text" itemprop="text">
<p>How can I run a text file use Python?</p>
<p>For example; if the text file is called <code>textfile.txt</code>, and it contains the line:</p>
<pre><code>print("Hello")
</code></pre>
<p>How can I use Python to run this as lines of code?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use exec: <a href="https://docs.python.org/3/library/functions.html#exec" rel="nofollow">https://docs.python.org/3/library/functions.html#exec</a>
so your code would look somehow like this:</p>
<pre><code>filehandle = open("filename.txt","r")
code_to_execute = filehandle.read()
exec(code_to_execute)
</code></pre>
<p>remeber that your file has to have correct syntax so no syntax error occours (mostly its about indentation)</p>
</div>
<div class="post-text" itemprop="text">
<p>Simply call the python interpreter and pass it the file as such:</p>
<pre><code>python yourfile.txt
</code></pre>
</div>
