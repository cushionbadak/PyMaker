<div class="post-text" itemprop="text">
<p>You need to specify newline <code>\n</code> as <a href="https://docs.python.org/3.5/library/csv.html#csv.Dialect.delimiter" rel="nofollow"><code>delimiter</code></a> </p>
<pre><code>formatted_lines = ['BR', 'BRQSB', 'BRQS', 'BRQSBWL9', 'CN', 'CNSHAWLT9001', 'CNS', 'DEA']

import csv

with open((pathsave, 'w') as f:
    writer = csv.writer(f, delimiter='\n')
    writer.writerow(formatted_lines)
</code></pre>
<p>Because the newline character is platform dependent you can use <a href="https://docs.python.org/3.5/library/os.html#os.linesep" rel="nofollow"><code>os.linesep</code></a> instead.</p>
<pre><code>import os
with open((pathsave, 'w') as f:
    writer = csv.writer(f, delimiter=os.linesep)
    writer.writerow(formatted_lines)
</code></pre>
