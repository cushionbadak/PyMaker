<p>I am trying to divide a list of elements which are comma separated into chunks of unequal length. How can I divide it?</p>
<pre><code>list1 = [1, 2, 1]
list2 = ["1.1.1.1", "1.1.1.2", "1.1.1.3", "1.1.1.4"]
</code></pre>
<p>list1 contains the elements which are the sizes of chunks which I wish to divide the list2 in.</p>

<p>Yet another solution</p>
<pre class="lang-py prettyprint-override"><code>list1 = [1,2,1]
list2 = ["1.1.1.1","1.1.1.2","1.1.1.3","1.1.1.4"]

chunks = []
count = 0
for size in list1:
    chunks.append([list2[i+count] for i in range(size)])
    count += size
print(chunks)

# [['1.1.1.1'], ['1.1.1.2', '1.1.1.3'], ['1.1.1.4']]
</code></pre>

<p>You could combine the power of <a href="https://docs.python.org/3/library/itertools.html#itertools.accumulate" rel="nofollow"><code>itertools.accumulate</code></a> and list comprehensions:</p>
<pre><code>In [4]: from itertools import accumulate

In [5]: data = ["1.1.1.1", "1.1.1.2", "1.1.1.3", "1.1.1.4"]

In [6]: lengths = [1, 2, 1]

In [7]: [data[end - length:end] for length, end in zip(lengths, accumulate(lengths))]
Out[7]: [['1.1.1.1'], ['1.1.1.2', '1.1.1.3'], ['1.1.1.4']]
</code></pre>
<p><code>itertools.accumulate</code> returns an iterator to a sequence of accumulated sums. This way you could easily calculate the end of each chunk in the source array:</p>
<pre><code>In [8]: list(accumulate(lengths))
Out[8]: [1, 3, 4]
</code></pre>

<p>Something like:</p>
<pre><code>def divideUnequal(list1, list2):
    counter=0
    step=0
    divided=[]
    for count in list1:
            step= counter+ count
            sublist= list2[counter: step]
            counter= step
            divided.append(sublist)
    return divided
</code></pre>
