<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip()</code></a> function within a list comprehension:</p>
<pre><code>&gt;&gt;&gt; lst = [1,4,5,6,3,2,4,0]
&gt;&gt;&gt; [i*10+j for i,j in zip(lst[0::2],lst[1::2])]
[14, 56, 32, 40]
</code></pre>
<p>As a more general approach for covering the list with odd number of items you can use <code>itertools.izip_longest</code> (in python 3.X <code>itertools.zip_longest</code>) :
by passing the 0 as <code>fillvalue</code> argument:</p>
<pre><code>&gt;&gt;&gt; lst=[1,4,5,6,3,2,4]
&gt;&gt;&gt; 
&gt;&gt;&gt; from itertools import izip_longest
&gt;&gt;&gt; [i*10+j for i,j in izip_longest(lst[0::2],lst[1::2], fillvalue=0)]
[14, 56, 32, 40]
</code></pre>
