<p>How can I print my strings so that there are no spaces between each output.</p>
<pre><code>name_input = str(input("Please enter your name"))
name_input = name_input.strip()
name_input = name_input.lower()
first_letter = name_input[0]
first_space = name_input.find(" ")
last_name = name_input[first_space:]
last_name_3 = last_name[0:4]
random_number = random.randrange(0,999)
print("*********************************************")

print("*********************************************")
print(first_letter + last_name_3, random_number)`
</code></pre>
<p>Incorrect output: b fir 723</p>
<p>what I require: bfir723</p>

<div class="post-text" itemprop="text">
<p>You need to use <code>strip()</code> function for this:</p>
<pre><code>print(first_letter.strip() + last_name_3.strip() + str(random_number).strip())
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>use the separator parameter to the print function, to remove the space by passing in an argument with no space.</p>
<p>Like this:</p>
<pre><code>print(first_letter, last_name_3, random_number, sep='')
</code></pre>
<p>The default separator is a space. Specification is here:</p>
<p><a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow">https://docs.python.org/3/library/functions.html#print</a></p>
</div>
<div class="post-text" itemprop="text">
<p>You don't need <code>strip()</code> and <code>sep=''</code></p>
<p>You only need one or the other, but <code>sep=''</code> is cleaner and more beautiful</p>
</div>
<div class="post-text" itemprop="text">
<p>You could also just do</p>
<pre><code>print(first_letter + last_name + str(random_number))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It turns out a combination of .strip() and ,sep='' was required to print correctly: </p>
<pre><code>print(first_letter.strip() + last_name_3.strip(), random_number, sep="")
</code></pre>
</div>
