<p>Using the same idea of @jezrael but with <a href="https://docs.python.org/3/library/stdtypes.html#float.is_integer" rel="nofollow noreferrer">is_integer</a>:</p>
<pre><code>import numpy as np
import pandas as pd

df3 = pd.DataFrame({
    'T': [11.0, 22.0, 11.23, 20.03],
    'v2': [11.0, 13.0, 55.1, 33.0],
    'v3': [112.1, 2.0, 2.1, 366.0],
    'v4': [np.nan, "blue", 1.0, 2.0]
})

df3['T'] = np.array([int(x) if float(x).is_integer() else x for x in df3['T']], dtype=object)

print(df3)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>T    v2     v3    v4
0     11  11.0  112.1   NaN
1     22  13.0    2.0  blue
2  11.23  55.1    2.1     1
3  20.03  33.0  366.0     2
</code></pre>
<p>Or using <a href="https://docs.scipy.org/doc/numpy-1.10.4/reference/generated/numpy.where.html" rel="nofollow noreferrer">numpy.where</a> with <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.fmod.html#numpy.fmod" rel="nofollow noreferrer">numpy.fmod</a>:</p>
<pre><code>mask = np.fmod(df3['T'].values, 1) == 0
df3['T'] = np.where(mask, df3['T'].values.astype(np.int), df3['T']).astype(dtype=object)
print(df3)
</code></pre>
