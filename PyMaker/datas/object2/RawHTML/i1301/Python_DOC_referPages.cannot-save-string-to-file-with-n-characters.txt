<p>The following code produces a file with content <code>test\\nstring</code>, but I need the file to contain <code>test\nstring</code>. I can't figure out a way to replace the <code>\\</code>symbol either.</p>
<pre><code>s = "test\nstring"
with open('test.txt', 'w') as f:
    f.write(s)
</code></pre>
<p>How can I make sure that the file contains only <code>\n</code> instead of <code>\\n</code>?</p>

<div class="post-text" itemprop="text">
<p>Besides of escaping and raw string, you can encode it (<a href="https://docs.python.org/2/howto/unicode.html#python-2-x-s-unicode-support" rel="nofollow noreferrer">2</a> or <a href="https://docs.python.org/3/library/codecs.html#standard-encodings" rel="nofollow noreferrer">3</a>) with <code>'string_escape'</code>:</p>
<pre><code>s = "test\nstring".encode('string_escape')
with open('test.txt', 'w') as f:
    f.write(s)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The <code>raw strings</code> may help</p>
<pre><code>s = r"test\nstring"
with open('test.txt', 'w') as f:
    f.write(s)
</code></pre>
</div>
