<p>can i use filter() to include multiple conditions? for example, if I have a code like this.</p>
<pre><code>nums = [3, 7, 10, 29, 39, 55, 67, 77, 107]
res = list(filter(lambda x: x &gt; 60, nums))
print(res)
</code></pre>
<p>how can I use filter to include numbers greater than 60 and numbers that are less than 20?</p>
</div>
<div class="post-text" itemprop="text">
<p>You should not be using filter but a list comprehension</p>
<pre><code>[x for x in nums if x &lt; 20 or x &gt; 60]
</code></pre>
<p>You can do the same thing with the filter but it will just be cumbersome </p>
<pre><code>list(filter(lambda x: x &lt; 20 or x &gt; 60, nums))
</code></pre>
</div>
