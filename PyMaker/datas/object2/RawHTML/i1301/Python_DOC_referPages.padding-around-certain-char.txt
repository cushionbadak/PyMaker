<div class="post-text" itemprop="text">
<p>If you want to avoid double-spaces in case padding already exists (as in your example), you could use a regular expression instead, using lookaheads and lookbehind to find places where a character that needs padding is preceeded <code>(?&lt;=...)</code> or followed by <code>(?=...)</code> a non-space character <code>\S</code>:</p>
<pre><code>&gt;&gt;&gt; padding = "()"
&gt;&gt;&gt; p = r"(?&lt;=\S)(?=[{0}])|(?&lt;=[{0}])(?=\S)".format(padding)
&gt;&gt;&gt; re.findall(p, text)
['', '']
&gt;&gt;&gt; re.sub(p, " ", text)
'if ( x &lt; 0 ) {'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try using <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow noreferrer"><code>str.replace</code></a>:</p>
<pre><code>def pad(s):
    for k, v in dictionary.items():
        s = s.replace(k, v)
    return s

&gt;&gt;&gt; dictionary = { "(": " ( ", ")": " ) "}
&gt;&gt;&gt; pad("if (x &lt; 0) {")
'if  ( x &lt; 0 )  {'
</code></pre>
