<p>You can use the <code>urlparse</code> library to split the URLs into pieces, and then extract the parts you need. For example:</p>
<pre><code>&gt;&gt;&gt; from urllib.parse import urlparse
&gt;&gt;&gt; urlparse('http://somesite.com/page.php?id=sas231')
ParseResult(scheme='http', netloc='somesite.com', path='/page.php', params='', query='id=sas231', fragment='')
</code></pre>
<p>Documentation for the python3 version of the library is at <a href="https://docs.python.org/3/library/urllib.parse.html" rel="nofollow noreferrer">urlparse</a></p>
