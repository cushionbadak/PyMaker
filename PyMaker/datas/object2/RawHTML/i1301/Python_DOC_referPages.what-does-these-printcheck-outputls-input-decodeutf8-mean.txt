<div class="post-text" itemprop="text">
<p>Pretty clear,</p>
<p>Calling <code>check_ouput</code> does the command specified <code>ls ../input</code> which list the folder <code>input</code> up a directory.</p>
<p>Then it decodes the command result for it to be in utf-8.</p>
<p>And then it prints it for you to see it.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>check_output</code> runs a shell command and returns its standard output. <code>decode</code> decodes a byte string into a proper Python Unicode string.</p>
<p>You <a href="https://mywiki.wooledge.org/ParsingLs" rel="nofollow noreferrer">should generally not use <code>ls</code> programmatically</a> and you should generally avoid using an external subprocess for a task which Python can easily perform itself. A better, more portable and more efficient way to achieve the same result is</p>
<pre><code> import os

 print('\n'.join(os.listdir('../input')))
</code></pre>
<p>... though this doesn't hard-code the assumption that the file names are returned as UTF-8 (which is probably a good thing for portability, too).</p>
</div>
