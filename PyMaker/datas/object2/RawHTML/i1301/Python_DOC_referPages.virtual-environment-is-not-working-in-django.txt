<div class="post-text" itemprop="text">
<p>Your version of Django requires Python 3, but it looks like <code>python</code> is <code>python2</code>, even after activating the virtual environment. You can use <code>which python</code> to check that <code>python</code> is pointing to the executable in your virtual environment.</p>
<p>If your virtual environment was created with python 2, then you'll have to <a href="https://docs.python.org/3/library/venv.html" rel="nofollow noreferrer">create a new virtual environment</a> that uses python 3.</p>
<pre><code>deactivate
python3 -m venv /path/to/urban.new
source /path/to/urban.new/bin/activate
# install requirements
</code></pre>
