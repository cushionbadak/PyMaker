<p>Having a dict with key-values how can we <strong>map</strong> its values to populate different columns in a dataframe.
Have tried to fill multiple dataframe columns using an <strong>apply</strong> function where the function contains an iteration over range of list of values without success. Any help?</p>
<p>Illustration:</p>
<pre><code>{'k1':['v1', 'v2'], 'k2':['v3', 'v4', 'v5']}
</code></pre>
<p>Desired result (dataframe):</p>
<pre><code>    col1  col2  col3
k1  v1    v2
k2  v3    v4    v5
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Here's one way using <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow noreferrer"><code>itertools.zip_longest</code></a> and the <a href="https://pandas.pydata.org/pandas-docs/version/0.23.4/generated/pandas.DataFrame.html" rel="nofollow noreferrer"><code>pd.DataFrame</code></a> constructor:</p>
<pre><code>from itertools import zip_longest

d = {'k1':['v1', 'v2'], 'k2':['v3', 'v4', 'v5']}

enum = enumerate(zip_longest(*d.values(), fillvalue=np.nan), 1)

df = pd.DataFrame(dict(enum), index=d).add_prefix('col')
</code></pre>
<p>An alternative formulation using formatted string literals (Python 3.6+; <a href="https://www.python.org/dev/peps/pep-0498/" rel="nofollow noreferrer">PEP498</a>):</p>
<pre><code>df = pd.DataFrame({f'col{i}': tup for i, tup in enum}, index=d)
</code></pre>
<p>Result:</p>
<pre><code>print(df)

   col1 col2 col3
k1   v1   v2  NaN
k2   v3   v4   v5
</code></pre>
</div>
