<div class="post-text" itemprop="text">
<p>I think Python 3.6 introduced a new way of creating virtual environments that don't depend on routing through usr/local. </p>
<pre><code>python3 -m venv /path/to/new/virtual/environment
</code></pre>
<p><a href="https://docs.python.org/3/library/venv.html" rel="nofollow noreferrer">Here</a> is documentation.</p>
