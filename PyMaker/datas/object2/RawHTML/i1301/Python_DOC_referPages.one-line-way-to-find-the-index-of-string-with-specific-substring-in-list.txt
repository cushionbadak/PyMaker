<p>Is there a <em>one-line</em> way to find the <code>index</code> of a string with a specific <code>substring</code> in a <code>list</code>?</p>
<p>I have a <code>list</code> like <code>['abc1', 'abc2', 'abc3', 'abc4']</code> and I wanna know what is the <code>index</code> where I have the <code>substring</code> <code>4</code>. </p>
</div>

<p>Does this help?</p>
<pre><code>items = ['abc1', 'abc2', 'abc3', 'abc4']
substring = '4'
index = [i for i, item in enumerate(items) if substring in item][0]
</code></pre>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer">enumerate</a> for that, i.e.:</p>
<pre><code>i = [i for i, s in enumerate(items) if '4' in s]
</code></pre>

<div class="post-text" itemprop="text">
<p>Since, there is only one item containing <code>'4'</code> (like you say), go for a generator:</p>
<pre><code>items = ['abc1', 'abc2', 'abc3', 'abc4']
substring = '4'

index = next((i for i, x in enumerate(items) if substring in x), None)
# 3
</code></pre>
<p>Additionally a <code>None</code> to handle case when there is no string with <code>'4'</code>.</p>
