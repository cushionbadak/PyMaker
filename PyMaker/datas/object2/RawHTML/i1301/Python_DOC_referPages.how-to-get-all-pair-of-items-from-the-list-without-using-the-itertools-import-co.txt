<p>I have a list and I want all possible combinations of items of a list.</p>
<pre><code>from itertools import combinations
l = [1,2,3]
for i in combinations(l,2):
    print(i) // (1, 2) (1,3) (2,3)
</code></pre>
<p>I want the same output but without using the itertools.
I tried:</p>
<pre><code>l = [1,2,3]
for i in l:
    for j in l:
        print((i,j)) // (1, 1), (1, 2), (1, 3), (2, 1), (2, 2), (2, 3), (3, 1), (3, 2), (3, 3)
</code></pre>
<p>How can I get the same output without using itertools?</p>

<div class="post-text" itemprop="text">
<p>You can find sample implementation of <a href="https://docs.python.org/3/library/itertools.html#itertools.combinations" rel="nofollow noreferrer"><code>itertools.combinations</code></a> in docs:</p>
<blockquote>
<p>Roughly equivalent to:</p>
<pre><code>def combinations(iterable, r):
    # combinations('ABCD', 2) --&gt; AB AC AD BC BD CD
    # combinations(range(4), 3) --&gt; 012 013 023 123
    pool = tuple(iterable)
    n = len(pool)
    if r &gt; n:
        return
    indices = list(range(r))
    yield tuple(pool[i] for i in indices)
    while True:
        for i in reversed(range(r)):
            if indices[i] != i + n - r:
                break
        else:
            return
        indices[i] += 1
        for j in range(i+1, r):
            indices[j] = indices[j-1] + 1
        yield tuple(pool[i] for i in indices)
</code></pre>
</blockquote>
</div>

<p>You can do this in a one liner just for efficiency: </p>
<pre><code>l = [1, 2, 3]

([print(i, j) for i in l for j in l])
</code></pre>
