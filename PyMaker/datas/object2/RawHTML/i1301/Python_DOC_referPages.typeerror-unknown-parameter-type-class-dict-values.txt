<div class="post-text" itemprop="text">
<p>It seems you are trying to run some python 2 code in python 3, 
where <code>dict.values</code> returns a <a href="https://docs.python.org/3/library/stdtypes.html#dict-views" rel="nofollow noreferrer">dictionary view object</a></p>
<p>The solution is quite simple - just wrap your <code>dict.values</code> in a <code>list</code>:</p>
<pre><code>train = theano.function(list(inps.values())+[target_values], cost, updates=updates)
</code></pre>
