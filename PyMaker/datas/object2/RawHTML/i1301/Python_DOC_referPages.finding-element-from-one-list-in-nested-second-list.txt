<p>I have two lists with string values, <code>list1 = ['a','g','f','e']</code> and <code>list2 = [['c','v','d'], ['a','d','e'], ['g','h']]</code>. I want to write a code, that will append <code>list1</code> into <code>list2</code> only if first (0th) element in nested list of list2 is  not the same as first (0th) element from <code>list1</code>.</p>
<p>This is the code that I wrote, it does not have errors, but It does not do what I want</p>
<pre><code>list1 = ['a','g','f','e']
list2 = [['c','v','d'], ['a','d','e'], ['g','h']]

print('List 1: ', list1)
print('List 2: ', list2)

for nest in list2:
    if list1[0] != nest[0]:
        list2.append(list1)
        print(list2)        
    else:
        print("Not added")
</code></pre>

<p>Maybe this helps:</p>
<pre><code>list1 = ['a','g','f','e']
list2 = [['c','v','d'], ['a','d','e'], ['g','h']]

print("List 1: ", list1)
print("List 2: ", list2)

for nested_list in list2:
    if nested_list[0] == list1[0]:
        print("Not added")
        break
else:
    list2.append(list1)

print(list2)
</code></pre>
<p>The <code>else</code> block after the <code>for</code> loop is executed, if the loop finished without <code>break</code>. You can read more about <code>for</code>-<code>else</code> at <a href="https://stackoverflow.com/questions/9979970/why-does-python-use-else-after-for-and-while-loops">Why does python use 'else' after for and while loops?</a> for example.</p>

<div class="post-text" itemprop="text">
<p>You can use the built-in <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer"><code>all</code></a> to check if all first elements of the sublists are not the same as the first element of <code>list1</code>:</p>
<pre><code>if all(l[0] != list1[0] for l in list2):
    list2.append(list1)
</code></pre>
