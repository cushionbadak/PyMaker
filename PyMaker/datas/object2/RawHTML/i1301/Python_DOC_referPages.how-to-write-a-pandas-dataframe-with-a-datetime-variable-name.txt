<p>I am scheduling a python script to run everyday,which outputs a dataframe. I wish to save the dataframe everyday with that day's date. For example for today it would be '12/11/18.csv'</p>
<p>How can i write such a csv file? </p>
<p>I'm doing <code>df.to_csv('datetime.datetime.today().strftime.csv')</code> but as expected it is saving it with the filename 'datetime.datetime.today().strftime.csv' not today's date as such</p>
</div>
<div class="post-text" itemprop="text">
<p>This should work:</p>
<pre><code>today = datetime.datetime.today().strftime('%d-%m-%Y')
df.to_csv(f'{today}.csv')
</code></pre>
<p>This works too:</p>
<pre><code>df.to_csv(f'{datetime.datetime.today().strftime("%d-%m-%Y")}.csv')
</code></pre>
<p>In the <code>strftime</code> function you can mention the format you like.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>datetime.datetime.today().strftime</code> is a function.</p>
<pre><code>&gt;&gt;&gt; datetime.datetime.today().strftime                                                                                 
&lt;function datetime.strftime&gt;
</code></pre>
<p>You need to call it with a format specifier.</p>
<p>Since having <code>/</code> in file names is not a good idea for portability, I suggest</p>
<pre><code>&gt;&gt;&gt; datetime.datetime.today().strftime('%F')                                                                           
'2018-12-11'
</code></pre>
<p>which you can then format with</p>
<pre><code>&gt;&gt;&gt; '{}.csv'.format(datetime.datetime.today().strftime('%F'))                                                          
'2018-12-11.csv'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>df.to_csv('%s.csv' % datetime.datetime.today().strftime('%Y-%m-%d'))
</code></pre>
<p>You can change <strong>-</strong> with <strong>/</strong>, but it's quite dirty to have those in file name.</p>
</div>
