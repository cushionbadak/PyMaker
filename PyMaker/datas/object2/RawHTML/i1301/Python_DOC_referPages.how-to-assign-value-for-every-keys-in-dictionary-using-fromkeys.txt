<p>How can I assign a different value for every key?</p>
<p>To be specific: I want <code>{1:"One",2:"Two", 3:"three", 4:"four", 5:"five"}</code> .</p>
<p>Is this possible to do so if I use <code>fromkeys()</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>No, <a href="https://docs.python.org/3/library/stdtypes.html#dict.fromkeys" rel="nofollow noreferrer"><code>.fromkeys()</code></a> is designed to add keys with a single default value.</p>
<blockquote>
<p>Create a new dictionary with keys from seq and values set to value.</p>
<p><code>fromkeys()</code> is a class method that returns a new dictionary. value
  defaults to <code>None</code>.</p>
</blockquote>
<p>For your use case, you may just need to "zip" and make a dictionary out of pairs:</p>
<pre><code>In [1]: dict(zip(range(1, 6), ["One", "Two", "three", "four", "five"]))
Out[1]: {1: 'One', 2: 'Two', 3: 'three', 4: 'four', 5: 'five'}
</code></pre>
</div>
