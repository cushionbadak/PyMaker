<p>I am working with python3 and pandas, and I would like to pass <code>:</code> as a function parameter to state all rows in a slice passed to <code>df.loc</code>.</p>
<p>For example, say I have a function that fills na values like so:</p>
<pre><code>def fill_na_w_value(df, rows, columns, fill):

    for col in columns:
        df.loc[rows, columns].fillna(
            fill,
            inplace=True
        )
</code></pre>
<p>Sometimes I may not want to apply it to some rows but to apply it to <strong>all rows</strong>, in pandas this is accessed with <code>df.loc[:, col]</code></p>
<p>If I am calling this from a function it would like </p>
<pre><code>fill_na_w_value(df, :, ['col1'], 0)
</code></pre>
<p>But the above will give me a syntax error because of the <code>:</code>; how can I pass this as a function parameter?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>slice(None)</code> to represent <code>:</code>. Note you can use <a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.pipe.html" rel="nofollow noreferrer"><code>pipe</code></a> to pass your dataframe through a function, and <a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.loc.html" rel="nofollow noreferrer"><code>loc</code></a> accepts a list for row and index filtering:</p>
<pre><code>df = pd.DataFrame({'col1': [1, 2, np.nan, 4, 5, np.nan, 7, 8, np.nan]})

def fill_na_w_value(df, row_slicer, columns, value):
    df.loc[row_slicer, columns] = df.loc[row_slicer, columns].fillna(value)
    return df

df1 = df.pipe(fill_na_w_value, slice(None), ['col1'], 0)

print(df1)

   col1
0   1.0
1   2.0
2   0.0
3   4.0
4   5.0
5   0.0
6   7.0
7   8.0
8   0.0
</code></pre>
<p>Here's an example using a <code>list</code> instead of a <a href="https://docs.python.org/3/library/functions.html#slice" rel="nofollow noreferrer"><code>slice</code></a> object:</p>
<pre><code>df2 = df.pipe(fill_na_w_value, [2, 5], ['col1'], 0)

print(df2)

   col1
0   1.0
1   2.0
2   0.0
3   4.0
4   5.0
5   0.0
6   7.0
7   8.0
8   NaN
</code></pre>
