<p>I am new to Python, I have seen many examples but not a good summary of examples. Here are the 4 things that I want to get done.  Thanks for your help.</p>
<pre><code>mydict = {'carl':40,
          'alan':2,
          'bob':1,
          'danny':3}
</code></pre>
<p>I wanted to use the following to access the dict elements:</p>
<pre><code>for key in sorted(mydict.keys()):
    print(key, mydict[key])
</code></pre>
<p>How do I get the following output:</p>
<p>Case 1: (by key ascending)</p>
<pre><code>alan   2

</code></pre>
<p>Case 2: (by key descending) </p>
<pre><code>danny  3

<div class="post-text" itemprop="text">
<p>See <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow">the documentation for <code>sorted</code></a>.</p>
<p>Ascending by key:</p>
<pre><code>for key, value in sorted(mydict.items()):
    pass
</code></pre>
<p>Descending by key:</p>
<pre><code>for key, value in sorted(mydict.items(), reverse=True):
    pass
</code></pre>
<p>Ascending by value:</p>
<pre><code>def get_value(item):
    return item[1]

for key, value in sorted(mydict.items(), key=get_value):
    pass
</code></pre>
<p>Descending by value:</p>
<pre><code>for key, value in sorted(mydict.items(), key=get_value, reverse=True):
    pass
</code></pre>
<p>You could also use <code>key=lambda x: x[1]</code> instead of defining a <code>get_key</code> function if you wanted.</p>

<div class="post-text" itemprop="text">
<pre><code>d = {'carl':40,
      'alan':2,
      'bob':1,
      'danny':3}

ks = sorted(d.keys()) #  sort keys
for key in ks:# key ascending
print(key, d[key])


for key in ks[::-1]: #key descending
    print(key, d[key])


vs= sorted(d, key=d.get) # sort keys by values
for key in vs: # values ascending
    print(key, d[key])


for key in vs[::-1]: # values descending
    print(key, d[key])
</code></pre>
