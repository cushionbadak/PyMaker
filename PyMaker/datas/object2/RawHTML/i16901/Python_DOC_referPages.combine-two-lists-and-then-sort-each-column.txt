<p><a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow"><code>sorted</code></a>, <a href="https://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow"><code>list.sort</code></a> accept optional <code>key</code> function. Return values of the function are used for comparison.</p>
<pre><code>&gt;&gt;&gt; list3 = [('Steve',32),('David',65),('Ram',43),('Mary',87)]
&gt;&gt;&gt; sorted(list3, key=lambda item: item[1])
[('Steve', 32), ('Ram', 43), ('David', 65), ('Mary', 87)]

&gt;&gt;&gt; sorted(list3, key=lambda item: -item[1]) # negate the return value.
[('Mary', 87), ('David', 65), ('Ram', 43), ('Steve', 32)]

&gt;&gt;&gt; sorted(list3, key=lambda item: item[1], reverse=True) # using `reverse`
[('Mary', 87), ('David', 65), ('Ram', 43), ('Steve', 32)]
</code></pre>
