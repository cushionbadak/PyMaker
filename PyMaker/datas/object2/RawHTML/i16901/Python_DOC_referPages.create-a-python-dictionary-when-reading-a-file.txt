<p>I have a list derived from a text file (filename) with an header</p>
<pre><code>mylist = [l.split() for l in open(filename, "r")]

mylist = [['A','B','C','D'],['1','2','3','4'],['10','20','30','40'],['100','200','300','400']]
</code></pre>
<p>i wish to create a dictionary directly reading the file in order to save lines of code as:</p>
<pre><code>mtlist_dist = {A: ['1','10','100'], B: [''2,'20','200'], C: ['3','30','300'], D: ['4','40','400']}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do this easily with <a href="https://docs.python.org/2/library/functions.html#zip" rel="nofollow"><code>zip</code></a> and a <a href="https://www.python.org/dev/peps/pep-0274/" rel="nofollow">dictionary comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; mylist = [['A','B','C','D'],['1','2','3','4'],['10','20','30','40'],['100','200','300','400']]
&gt;&gt;&gt; {x[0]:x[1:] for x in zip(*mylist)}
{'A': ('1', '10', '100'), 'C': ('3', '30', '300'), 'B': ('2', '20', '200'), 'D': ('4', '40', '400')}
&gt;&gt;&gt; {x[0]:list(x[1:]) for x in zip(*mylist)}
{'A': ['1', '10', '100'], 'C': ['3', '30', '300'], 'B': ['2', '20', '200'], 'D': ['4', '40', '400']}
&gt;&gt;&gt;
</code></pre>
<p>In Python 3.x, the solution becomes even more concise with <a href="https://docs.python.org/3/whatsnew/3.0.html#new-syntax" rel="nofollow">extended iterable unpacking</a>:</p>
<pre><code>&gt;&gt;&gt; mylist = [['A','B','C','D'],['1','2','3','4'],['10','20','30','40'],['100','200','300','400']]
&gt;&gt;&gt; {x:y for x,*y in zip(*mylist)}
{'D': ['4', '40', '400'], 'A': ['1', '10', '100'], 'C': ['3', '30', '300'], 'B': ['2', '20', '200']}
&gt;&gt;&gt;
</code></pre>
</div>

<p>You can do it as:</p>
<pre><code>my_dict = dict(zip(mylist[0], zip(*mylist[1:])))

&gt;&gt;&gt; print my_dict
{'A': ('1', '10', '100'), 'C': ('3', '30', '300'), 'B': ('2', '20', '200'), 'D': ('4', '40', '400')
</code></pre>
