<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html#hex" rel="nofollow"><code>hex</code></a> built-in returns a string value:</p>
<pre><code>&gt;&gt;&gt; hex(123)
'0x7b'
&gt;&gt;&gt; type(hex(123))
&lt;class 'str'&gt;
&gt;&gt;&gt;
</code></pre>
<p>but <a href="https://docs.python.org/3/library/functions.html#format" rel="nofollow"><code>format</code></a> is expecting a hexadecimal value as its first argument:</p>
<pre><code>&gt;&gt;&gt; format('0x7b', 'x')
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
ValueError: Unknown format code 'x' for object of type 'str'
&gt;&gt;&gt;
&gt;&gt;&gt; format(0x7b, 'x')
'7b'
&gt;&gt;&gt;
</code></pre>
<p>Thus, it cannot be used here.  Instead, you can just strip off the <code>0x</code> with slicing:</p>
<pre><code>with open(r'\\Network\files\build_ver.txt','r+') as f:
    value = int(f.read(), 16)
    f.seek(0)
    write_value = hex(value + 1)[2:]
    f.write(write_value)
</code></pre>
<p><code>[2:]</code> will get every character in the string except for the first two.  See a demonstration below:</p>
<pre><code>&gt;&gt;&gt; hex(123)
'0x7b'
&gt;&gt;&gt; hex(123)[2:]
'7b'
&gt;&gt;&gt;
</code></pre>
