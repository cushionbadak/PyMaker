<div class="post-text" itemprop="text">
<p>One easy way to display image on a Jupyter notebook is using HTML chunk of code in a <code>Markdown cell</code>, here is an example:</p>
<pre><code>&lt;img src='./image/filename.png'&gt;Image description&lt;/img&gt;
</code></pre>
<p>Once you <code>run</code> the cell, the image and its description text should appear in place of the HTML code.</p>
