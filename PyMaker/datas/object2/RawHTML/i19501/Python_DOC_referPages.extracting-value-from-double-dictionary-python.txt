<p>By 'double dictionary' I suppose you mean <em>nested</em> dictionary, right?</p>
<p>Well, when you access a value of the outter dict through a key (let's say, 1), it will return the corresponding inner dict, which values you can access... well, like a dict.</p>
<p>Here's an example:
</p>
<pre><code>data = {1: {'RFC822.SIZE': 2515, 'SEQ': 1}, 2: {'RFC822.SIZE': 2942, 'SEQ': 2}}

# get the inner dictionary with key 1
key_1 = data[1] # {'RFC822.SIZE': 2515, 'SEQ': 1}

# get RFC822.SIZE
size = key_1['RFC822.SIZE']
</code></pre>
<p>Knowing this, you can condense all that code by accesing the value directly:
</p>
<pre><code>size = data[1]['RFC822.SIZE']
</code></pre>
<p>This is some basic python (you use dicts constantly when coding on python) so I recommend (if you are planning to do serious coding on python) that you read some tuturial, at least the introductory one you can find on the <a href="http://docs.python.org/3/" rel="nofollow">python's doc page</a></p>
</div>
