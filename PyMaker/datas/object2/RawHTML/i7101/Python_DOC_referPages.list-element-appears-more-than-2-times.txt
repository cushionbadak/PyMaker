<div class="post-text" itemprop="text">
<p>To get the list of element appear more than 2 times in the dataset.</p>
<pre><code>[x for x,y in Counter(a).items() if y &gt; 2]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to test if the highest count in the counter is greater than 2; you can use <a href="https://docs.python.org/3/library/collections.html#collections.Counter.most_common" rel="nofollow noreferrer"><code>Counter.most_common()</code></a> to extract the highest count:</p>
<pre><code>if Counter(a).most_common(1)[0][1] &gt; 2:
</code></pre>
<p><code>Counter.most_common()</code> returns a list of <code>(value, count)</code> pairs, even when you ask for just a single pair; <code>[0]</code> gets that one <code>(value, count)</code> pair from the list, and <code>[1]</code> extracts the count.</p>
</div>
