<div class="post-text" itemprop="text">
<p><code>.ipynb</code> files are not valid python modules.  They use a custom format to store the code and output.  To use your code, you would need to move it to a valid <code>.py</code> file that is <a href="https://docs.python.org/3/glossary.html#term-importing" rel="nofollow noreferrer">importable</a></p>
