<div class="post-text" itemprop="text">
<p>This an example how to do that;</p>
<h3>1.  Using module <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer"><code>datetime</code></a></h3>
<pre><code>import datetime

today = datetime.datetime.today()
yesterday = today - datetime.timedelta(days=1)

orders = News.objects.filter(date__range=[yesterday, today])
</code></pre>
<h3>2. Using django <a href="https://docs.djangoproject.com/en/dev/topics/i18n/timezones/" rel="nofollow noreferrer"><code>timezone</code></a></h3>
<pre><code>from django.utils import timezone

today = timezone.now()
yesterday = today - timezone.timedelta(days=1)

orders = News.objects.filter(date__range=[yesterday, today])
</code></pre>
