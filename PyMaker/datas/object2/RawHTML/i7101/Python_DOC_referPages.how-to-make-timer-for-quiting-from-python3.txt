<div class="post-text" itemprop="text">
<p>You could use the <code>os._exit</code> function instead of <code>exit()</code></p>
<p>Getting the code as follows:</p>
<pre><code>#!/usr/bin/env python3
import threading
import os
from time import sleep

def qu():
    print("bye")
    os._exit(0)

t=threading.Timer(5.0,qu)
t.start()

while(True):
    sleep(1)
    print("hi")
</code></pre>
<p>Anyways I would suggest you to checkout <a href="https://stackoverflow.com/questions/1489669/how-to-exit-the-entire-application-from-a-python-thread">this question</a> as it is similar to yours.</p>
