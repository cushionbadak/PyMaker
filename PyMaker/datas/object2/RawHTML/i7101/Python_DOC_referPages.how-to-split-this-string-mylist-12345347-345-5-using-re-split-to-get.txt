<div class="post-text" itemprop="text">
<p>Your first solution fails because <code>+</code> and <code>*</code> are regex metacharacters - if you want to use them as literals, you need to escape them. The following would have worked:</p>
<pre><code>print(re.split(r'\+|-|\*|/',mylist)
</code></pre>
<p>Note that I used a raw string - always a good idea with regexes to reduce the number of backslashes you'd need otherwise.</p>
<p>Your second solution fails because <code>-</code> is a metacharacter in a character class (and there is no "range of characters between <code>+</code> and <code>*</code>). By placing it at the start or end of the character class, <code>-</code> loses its special meaning, so the following would have worked:</p>
<pre><code>print(re.split(r'[-+*/]',mylist)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>re.findall('\d+', string)</code>:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; string = '123+45+34*7-3*45/5'
&gt;&gt;&gt; print(re.findall('\d+', string))

['123', '45', '34', '7', '3', '45', '5']
</code></pre>
<p><em>Explanation:</em> <code>\d</code> is a digit (a character in the range 0-9), and <code>+</code> means 1 or more times. So, <code>\d+</code> is 1 or more digits.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>import re
p = re.compile('\W+')
p.split('123+45+34*7-3*45/5')
</code></pre>
<p>output is <code>['123', '45', '34', '7', '3', '45', '5']</code></p>
<p><code>\W</code> Matches any non-alphanumeric character; this is equivalent to the class <code>[^a-zA-Z0-9_]</code>.</p>
<p>For more information refer to <a href="https://docs.python.org/3/howto/regex.html" rel="nofollow noreferrer">Python Docs for regular expressions usage</a></p>
</div>
