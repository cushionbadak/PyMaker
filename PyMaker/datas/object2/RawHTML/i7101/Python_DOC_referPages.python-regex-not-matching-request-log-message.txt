<div class="post-text" itemprop="text">
<p>You missed a forward slash at the beginning of the pattern; Keep in mind <a href="https://docs.python.org/3/library/re.html#re.match" rel="nofollow noreferrer"><code>re.match</code></a> always matches the pattern from the beginning of the string, so everything should match from the start:</p>
<pre><code>r = re.match('/([0-9a-z\-]*.gif) (HTTP/1.0") (200)', '/asdf.gif HTTP/1.0" 200 0')
#             ^

r.group()
# '/asdf.gif HTTP/1.0" 200'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>maybe you should use <code>re.search()</code>, <code>re.match()</code> need match the start of the string, <code>re.search()</code> will scan through the string.</p>
