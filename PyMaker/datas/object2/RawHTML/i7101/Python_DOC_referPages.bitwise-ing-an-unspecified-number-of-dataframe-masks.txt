<div class="post-text" itemprop="text">
<p>So I'm used to combining dataframe masks like so:
<code>final_mask = mask1 &amp; mask2</code></p>
<p>But what if I want to combine many masks? For example, the list:
<code>[mask1, mask2, mask3, mask4, ..., mask20]</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="http://pandas.pydata.org/pandas-docs/stable/cookbook.html#building-criteria" rel="nofollow noreferrer">pandas cookbook solution</a>, last paragraph with <a href="https://docs.python.org/3/library/functools.html#functools.reduce" rel="nofollow noreferrer"><code>reduce</code></a>:</p>
<pre><code>df = pd.DataFrame({'AAA' : [4,5,6,7], 'BBB' : [10,20,30,40],'CCC' : [100,50,-30,-50]})
print (df)
   AAA  BBB  CCC
0    4   10  100
1    5   20   50
2    6   30  -30
3    7   40  -50

mask1 = df.AAA &lt;= 5.5
mask2 = df.BBB == 10.0
mask3 = df.CCC &gt; -40.0

masks = [mask1, mask2, mask3]
mask = functools.reduce(lambda x,y: x &amp; y, masks)

print (df[mask])
   AAA  BBB  CCC
0    4   10  100
</code></pre>
<p>Another solution from <a href="https://stackoverflow.com/questions/45577120/bitwise-ing-an-unspecified-number-of-dataframe-masks#comment78113625_45577120"><code>ayhan</code></a> comment working on <code>1d</code> mask (masks are <code>Series</code>):</p>
<pre><code>mask = np.logical_and.reduce(masks)

print (df[mask])
   AAA  BBB  CCC
0    4   10  100
</code></pre>
<p>As <a href="https://stackoverflow.com/questions/45577120/bitwise-ing-an-unspecified-number-of-dataframe-masks#comment78113860_45577139">ayhan</a> pointed, first solution  also works with <code>2D</code> masks:</p>
<pre><code>mask1 = df &lt;= 5.5
mask2 = df &lt; 1.0
mask3 = df &gt; -40.0

masks = [mask1, mask2, mask3]
mask = functools.reduce(lambda x,y: x &amp; y, masks)
print (mask)
     AAA    BBB    CCC
0  False  False  False
1  False  False  False
2  False  False   True
3  False  False  False
</code></pre>
<hr/>
<pre><code>mask = np.logical_and.reduce(masks)
print (mask)
</code></pre>

<p>ValueError: cannot copy sequence with size 4 to array axis with dimension 3</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>Probably, the easiest way is using a <code>for</code> loop:</p>
<pre><code>final_mask = mask1
for mask in [mask2, mask3, mask4]:
    final_mask = final_mask &amp; mask
</code></pre>
<p>Notice though that, although it is easy to understand, this way may not be considered the most "pythonic" way. Using <code>reduce</code>, as pointed by others, makes your code shorter but may not be easier to read by novices.</p>
