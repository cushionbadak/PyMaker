<div class="post-text" itemprop="text">
<p>I convert  [int, bool ,float] to ['int', 'bool','float'] using many lines of command.</p>
<pre><code>Numbers = [int, bool, float]
&gt;&gt;&gt; [ i for i in Numbers]
[&lt;class 'int'&gt;, &lt;class 'bool'&gt;, &lt;class 'float'&gt;]
&gt;&gt;&gt;foo = [ str(i) for i in Numbers]
&gt;&gt;&gt;foo
["&lt;class 'int'&gt;", "&lt;class 'bool'&gt;", "&lt;class 'float'&gt;"]
&gt;&gt;&gt; bar = [ i.replace('&lt;class ','') for i in foo]
&gt;&gt;&gt; bar
["'int'&gt;", "'bool'&gt;", "'float'&gt;"]
&gt;&gt;&gt; baz = [i.replace('&gt;','') for i in bar]
&gt;&gt;&gt; baz
["'int'", "'bool'", "'float'"]
&gt;&gt;&gt; [ eval(i) for i in baz]
['int', 'bool', 'float']
</code></pre>
<p>How to accomplish such a task in an elegant manner?</p>
</div>
<div class="post-text" itemprop="text">
<p>You want the <a href="https://docs.python.org/3/library/stdtypes.html#definition.__name__" rel="noreferrer"><code>__name__</code></a> attribute.</p>
<pre><code>[i.__name__ for i in Numbers]
</code></pre>
<p>As an aside, if you are interested in performing introspection on Python data structures, use <a href="https://docs.python.org/3/library/functions.html#dir" rel="noreferrer"><code>dir()</code></a>. For example, <code>dir(int)</code> will return a list of all attributes and callable methods you can use on the <code>int</code> type.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>__name__ will help you to show type of it as a string,
</code></pre>
<p>you can find available modules for variables and function using <code>dir(var)</code></p>
<pre><code>&gt;&gt;&gt; num = [1,bool(1),1.10]
&gt;&gt;&gt; [type(i).__name__ for i in num] 
['int', 'bool', 'float']
&gt;&gt;&gt; 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>There may be a more pythonic way to do this but a 1-liner that works for these types is:</p>
<pre><code>&gt;&gt;&gt; [ x.__name__ for x in Numbers ]
['int', 'bool', 'float']
</code></pre>
<p>Btw you can find out attributes an object has using <a href="https://docs.python.org/2/library/functions.html#dir" rel="nofollow noreferrer"><code>dir()</code></a></p>
<p>For example:</p>
<pre><code>&gt;&gt;&gt; dir(Numbers[0])
['__abs__', '__add__', '__and__', '__class__', '__cmp__', '__coerce__', '__delattr__', '__div__', '__divmod__', '__doc__', '__float__', '__floordiv__', '__format__', '__getattribute__', '__getnewargs__', '__hash__', '__hex__', '__index__', '__init__', '__int__', '__invert__', '__long__', '__lshift__', '__mod__', '__mul__', '__neg__', '__new__', '__nonzero__', '__oct__', '__or__', '__pos__', '__pow__', '__radd__', '__rand__', '__rdiv__', '__rdivmod__', '__reduce__', '__reduce_ex__', '__repr__', '__rfloordiv__', '__rlshift__', '__rmod__', '__rmul__', '__ror__', '__rpow__', '__rrshift__', '__rshift__', '__rsub__', '__rtruediv__', '__rxor__', '__setattr__', '__sizeof__', '__str__', '__sub__', '__subclasshook__', '__truediv__', '__trunc__', '__xor__', 'bit_length', 'conjugate', 'denominator', 'imag', 'numerator', 'real']
</code></pre>
</div>
