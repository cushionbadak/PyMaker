<p>The specific question is like this:</p>
<p>I have <code>two lists</code>, namely: </p>
<pre><code>lst1 = [1,2,5,6,7];
lst2 = [11,12,13,14,15];
</code></pre>
<p>I would like to build a relation of those lists to match with the corresponding number of the other list, like <code>1-11,5-13,7-15...</code>and if I input<code>[1,5,7]</code>, then it will automatically generate<code>[11,13,15]</code>. I'm new to python so I don't know if it is doable or could someone give me some advise or ideas, thanks in advance!</p>

<div class="post-text" itemprop="text">
<p>you could create a dictionary with the elements of the first list as keys and the elements of the second list as values:</p>
<pre><code>lst1 = [1,2,5,6,7]
lst2 = [11,12,13,14,15]

dct = dict((a, b) for a, b in zip(lst1, lst2))
# or even:
# dct = {a: b for a, b in zip(lst1, lst2)}
print(dct)  # {1: 11, 2: 12, 5: 13, 6: 14, 7: 15}
print(dct[1], dct[5], dct[7])  # 11 13 15
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If by "relation" you mean a 1-to-1 mapping, then perhaps you should take a look at <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">dictionaries</a>. And if your lists (<code>lst1</code>, <code>lst2</code>) always have same length and order of mapping, you probably don't need a dictionary mapping, just use indexing correctly, along with the <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow noreferrer"><code>index</code> method for lists</a>.</p>

<div class="post-text" itemprop="text">
<p>first <strong>zip</strong> both list then use <strong>list compression</strong> over the output of list with if the condition</p>
<pre><code>&gt;&gt;&gt; lst1 = [1,2,5,6,7]

&gt;&gt;&gt; lst2 = [11,12,13,14,15]

&gt;&gt;&gt; input_lst = [1,2,5]

&gt;&gt;&gt; [d for c,d in zip(lst1,lst2) if c in input_lst]      # first method Zip

&gt;&gt;&gt; [lst2[lst1.index(v)] for v in (lst1) if v in input_lst]  # second mehod using index
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[11, 12, 13]
</code></pre>
