<div class="post-text" itemprop="text">
<p>You could declare a function like this one</p>
<pre><code>def save_csv_data(nm_csv, path='.'):
    with open(path + '/output.csv', 'w') as output:
        output.write(nm_csv)
</code></pre>
<p>then by passing an argument you could specify another path to save the file, or otherwise, use the same directory as the script:</p>
<pre><code>if (len(sys.argv) &gt; 1 and sys.argv[1]):
    save_csv_data(nm.csv(), path=sys.argv[1])
else:
    save_csv_data(nm.csv())
</code></pre>
<p><strong>Edit:</strong> also remember to <code>import sys</code></p>
<p>I also would suggest you, if you decide to use arguments, use a module like <a href="https://docs.python.org/3/library/argparse.html" rel="nofollow noreferrer">argparse</a>.</p>
