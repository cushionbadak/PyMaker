<p>Use <code>calendar</code> and <code>apply</code>:</p>
<pre><code>import calendar
tmaxmin.Month_Day = tmaxmin.Month_Day.apply(lambda x:calendar.month_abbr[int(x[:2])]+x[2:])
</code></pre>

<p>Since your <code>Month_Day</code> key appears to be linking to a <code>str</code>, you don't need to use slice notation, but it would be better to use the <code>str</code> type method <code>startswith</code> instead.</p>
<p>So:</p>
<pre><code>if tmaxmin['Month_Day'][i].startswith('01'):
    # do your stuff
</code></pre>
<p>That's a start to cleaning up your code.</p>
<p>Also, tmaxmin is an iterable, so you can just use <code>for ... in ...:</code> to iterate through it:</p>
<pre><code>for date in tmaxmin:
    if date['Month_Day'].startswith('01'):
        date['Month_Day'] = "something else"
</code></pre>
<p>Since lists and dicts are mutable, this will edit the list in place.</p>
<p>Additionally, you may want to create a tuple to pair the replacements together so you can have cleaner code:</p>
<pre><code>replacements = (
    ('01', 'Jan'),
    ('02', 'Feb'),
    ('03', 'March'),
    ...
    )

for date in tmaxmin:
    for old, new in replacements:
        if date.startswith(old):
            date['Month_Day'] = new + date['Month_Day'][2:]
</code></pre>
