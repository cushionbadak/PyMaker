<p>How to create a dictionary of dictionaries from existed key value lists?  </p>
<pre><code>celebr = ['Tony','Harry','Katty','Sam']
perc = [69,31,0,0]


d = dict(zip(celebr, perc))


dlist = []

for i in d.items():
    dlist.append(i)

print(dlist)
</code></pre>
<p>Output:</p>
<pre><code>[('Tony': 69), ('Harry': 31), ('Katty': 0), ('Sam': 0)]
</code></pre>
<p>When I use d.items it automatically gives me tuples, not dictionaries. Is there elegant way of creating list of dictionaries not tuples?</p>
<p>Need to get following structure :</p>
<pre><code>[{'Tony': 69}, {'Harry': 31}, {'Katty': 0}, {'Sam': 0}]
</code></pre>

<div class="post-text" itemprop="text">
<p>Good use case for <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer"><code>list comprehension</code></a>:</p>
<pre><code>dlist = [{k: v} for k, v in zip(celebr, perc)]
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; celebr = ['Tony', 'Harry', 'Katty', 'Sam']
&gt;&gt;&gt; perc = [69, 31, 0, 0]
&gt;&gt;&gt;
&gt;&gt;&gt; [{k: v} for k, v in zip(celebr, perc)]
[{'Tony': 69}, {'Harry': 31}, {'Katty': 0}, {'Sam': 0}]
</code></pre>
</div>

<div class="post-text" itemprop="text">
<p>You can implement with <code>zip</code> and <code>dict</code>.</p>
<pre><code>dict(zip(celebr,perc))
</code></pre>
<p>Results:</p>
<pre><code>In [14]: celebr = ['Tony', 'Harry', 'Katty', 'Sam'] 
In [15]: perc = [69, 31, 0, 0]

In [16]: dict(zip(celebr,perc))
Out[16]: {'Harry': 31, 'Katty': 0, 'Sam': 0, 'Tony': 69}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your example merely creates a list containing the dicts <code>items</code>. Your question asks for a list of sets. If you really want individual dicts then use a colon to separate the key and the value:</p>
<pre><code>&gt;&gt;&gt; [{k, v} for (k, v) in d.items()]
[{'Tony', 69}, {'Harry', 31}, {'Katty', 0}, {'Sam', 0}]
&gt;&gt;&gt; [{k: v} for (k, v) in d.items()]
[{'Tony': 69}, {'Harry': 31}, {'Katty': 0}, {'Sam': 0}]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try :</p>
<pre><code>celebr = ['Tony','Harry','Katty','Sam']
perc = [69,31,0,0]

b = [{celebr[i]: perc [i]} for i in range(0, len(celebr))]

print(b)
</code></pre>
<p>Output :</p>
<pre><code>[{'Tony': 69}, {'Harry': 31}, {'Katty': 0}, {'Sam': 0}]
</code></pre>
