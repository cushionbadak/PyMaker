<p>Note that a more Pythonic way to do that would be this:</p>
<pre><code>a = [1, 2, 3]
b = [3, 4, 1]
c = [max(i) for i in zip(a, b)]
print(c)  # [3, 4, 3]
</code></pre>
<p>If it can be the case that the two <code>list</code>s do not have the same number of items you can use <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow noreferrer"><code>zip_longest</code></a> from <code>itertools</code>:</p>
<pre><code>from itertools import zip_longest

a = [1, 2, 3, 7]
b = [3, 4, 1]
c = [max(i) for i in zip_longest(a, b, fillvalue=0)]
print(c)  # [3, 4, 3, 7]
</code></pre>
