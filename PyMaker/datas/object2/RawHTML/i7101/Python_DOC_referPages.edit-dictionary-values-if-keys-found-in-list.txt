<div class="post-text" itemprop="text">
<p>You can take the intersection of the dictionary keys with the list, and update values at those keys:    </p>
<pre><code>for key in set(data).intersection(some_list):
    data[key] = 'foobar'

print(data)
# {'twofoo': 'twobar', 'threefoo': 'foobar', 'foo': 'foobar'}
</code></pre>
<p>You could also define your list as a set and do a quick membership check (with <em>O</em>(1) complexity) for each key:</p>
<pre><code>some_set = {'foo', 'threefoo'}

for key in data:
    if key in some_set:
        data[key] = 'randomfoo'
</code></pre>

<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer"><code>dict comprehension</code></a>:</p>
<pre><code>res = {key: 'randomfoo' if key in some_list else value for key, value in data.items()}
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; res
{'twofoo': 'twobar', 'threefoo': 'randomfoo', 'foo': 'randomfoo'}
</code></pre>
