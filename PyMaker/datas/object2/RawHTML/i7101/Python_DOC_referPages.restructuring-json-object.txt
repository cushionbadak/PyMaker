<p>There's a lot of similar questions out there.  I would just use list comprehension:</p>
<pre><code>[{'id': value, 'name': key} for key, value in original.items()]
</code></pre>
<p>If it is JSON, not a python object, it's easy to load/dump it:</p>
<pre><code>import json
...
original = json.loads(json_string)
modified = [{'id': value, 'name': key} for key, value in original.items()]
output = json.dumps(modified) # New JSON string
</code></pre>
