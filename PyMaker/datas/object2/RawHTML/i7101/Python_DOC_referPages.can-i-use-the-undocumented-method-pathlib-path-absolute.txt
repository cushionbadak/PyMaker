<div class="post-text" itemprop="text">
<p>The first method found by auto-complete on a <code>pathlib.Path</code> is <code>absolute()</code>.</p>
<p>It seems to just prepend <a href="https://docs.python.org/3/library/pathlib.html#pathlib.Path.cwd" rel="nofollow noreferrer">Path.cwd()</a> at the start:</p>
<pre class="lang-py prettyprint-override"><code>&gt;&gt;&gt; from pathlib import Path
&gt;&gt;&gt; path = Path('./relative/path') / '../with/some/../relative/parts'
Path('relative/path/../with/some/../relative/parts')
# calling absolute...
&gt;&gt;&gt; absolute_path = path.absolute()
Path('/[current_dir]/relative/path/../with/some/../relative/parts')
# does the same as prepending cwd at the start
&gt;&gt;&gt; Path.cwd() / path
Path('/[current_dir]/relative/path/../with/some/../relative/parts')
</code></pre>
<p>However, <code>Path.absolute()</code> is not listed in the <a href="https://docs.python.org/3/library/pathlib.html#pathlib.Path.cwd" rel="nofollow noreferrer">pathlib documentation</a>.</p>
<p>Compare this to <a href="https://docs.python.org/3/library/pathlib.html#pathlib.Path.resolve" rel="nofollow noreferrer">Path.resolve()</a>, which does the opposite (replace relative parts but don't prepend <code>cwd</code>) and  <em>is</em> documented.</p>
<p>Can I use <code>absolute()</code> or should I avoid it?</p>
</div>
<div class="post-text" itemprop="text">
<p>At least up until Python version 3.6, you should avoid using <code>Path.absolute()</code>.</p>
<p>According to the discussion in <a href="https://bugs.python.org/issue29688" rel="nofollow noreferrer">the bug report about the missing documentation</a>,
<code>absolute()</code> is not tested, and therefore not officially advertised.  In fact, it <em>might</em> even be removed in a future release of Python.</p>
<p>So it's safer to just use <code>Path.cwd()</code> instead.</p>
<p>If you're not sure it's actually necessary to do that, you can use <a href="https://docs.python.org/3/library/pathlib.html#pathlib.PurePath.is_absolute" rel="nofollow noreferrer">Path.is_absolute()</a> to check.</p>
<pre class="lang-py prettyprint-override"><code>&gt;&gt;&gt; # make path absolute if it isn't already
&gt;&gt;&gt; path = path if path.is_absolute() else Path.cwd() / path
Path('/[current_dir]/relative/path/../with/some/../relative/parts')
</code></pre>
</div>
