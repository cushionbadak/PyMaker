<p>Using <a href="https://docs.python.org/3/library/re.html#re.sub" rel="nofollow noreferrer"><code>re.sub</code></a>. I don't know if your file uses tabs or spaces, so I handle both cases to be on the safe side.</p>
<pre><code>import re

s =\
"""
6.85675852     5.7928113, -99.990, -99.990,   8.083,
6.81641565     5.5877682,  10.560,   8.960,   5.465,
6.84986385     5.8423371,   7.390,   7.920,   6.026,
6.86023411     5.7104751,  16.600,  13.800,   7.311,
"""

s_out = re.sub('(\t|\s{5})', r',\1', s)
print(s_out)
</code></pre>
