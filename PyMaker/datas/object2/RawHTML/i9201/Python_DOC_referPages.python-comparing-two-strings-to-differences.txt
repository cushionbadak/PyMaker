<p>The solution using <strong><em>Set</em></strong> difference operator:</p>
<pre><code>str1 = ['test1.jpg', 'test2.jpg', 'test3.jpg']
str2 = ['test1.jpg', 'test2.jpg']

diff = list(set(str1) - set(str2))
# the same can be achieved with set.difference(*others) method:
# diff = list(set(str1).difference(set(str2))) 
print(diff)
</code></pre>
<p>The output:</p>
<pre><code>['test3.jpg']
</code></pre>
<p><a href="https://docs.python.org/3/library/stdtypes.html?highlight=set#set.difference" rel="nofollow noreferrer">https://docs.python.org/3/library/stdtypes.html?highlight=set#set.difference</a></p>

<div class="post-text" itemprop="text">
<p>You can use built-in type <code>set</code></p>
<pre><code>str1 = ['test1.jpg', 'test2.jpg', 'test3.jpg']
str2 = ['test1.jpg', 'test2.jpg']
s1 = set(str1)
s2 = set(str2)
print(s1-s2)
</code></pre>
<p>From there you can easily create function:</p>
<pre><code>def difference(list1,list2):
    return set(list1)-set(list2)
</code></pre>
<p>If you want to create difference between <strong>both</strong> of them you can do the following:
For data : </p>
<pre><code>str1 = ['test1.jpg', 'test2.jpg', 'test3.jpg']
str2 = ['test1.jpg', 'test2.jpg', 'test4.jpg']
</code></pre>
<p>Funcion would be:</p>
<pre><code>def difference(list1,list2):
    return (set(list1)-set(list2)) | (set(list2)-set(list1))
</code></pre>
<p>Or : </p>
<pre><code>def difference(list1,list2):
    return set(list1).symmetric_difference(set(list2))
</code></pre>
<p>Outputs:</p>
<blockquote>
<p>{'test3.jpg', 'test4.jpg'}</p>
</blockquote>
<p>All is documented here: <a href="https://docs.python.org/3/tutorial/datastructures.html#sets" rel="nofollow noreferrer">Python3</a> <a href="https://docs.python.org/2/library/stdtypes.html#set" rel="nofollow noreferrer">Python2</a></p>
