<p>From the <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer">docs</a>:</p>
<p><code>any</code> returns True if any element of the iterable is true. If the iterable is empty, return False. Equivalent to:</p>
<pre><code>def any(iterable):
    for element in iterable:
        if element:
            return True
    return False
</code></pre>
