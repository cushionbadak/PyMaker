<p>If you want to use python 2:</p>
<pre><code>a = list("abcd")
b = list("uvwxyz")
q = list(map(None, a, b))
output = ""
for i in q:
    if i[0] is not None:
        output+=i[0]
    if i[1] is not None:
        output+=i[1]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow noreferrer"><code>itertools.zip_longest</code></a>:</p>
<pre><code>from itertools import zip_longest

s1 = "abcd"
s2 = "uvwxyz"
s3 = ''.join(a + b for a, b in zip_longest(s1, s2, fillvalue=''))
print(s3)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>aubvcwdxyz
</code></pre>
<hr/>
<p>Here's a version that works with Python 2. Spot the difference!</p>
<pre><code>from itertools import izip_longest

s1 = "abcd"
s2 = "uvwxyz"
s3 = ''.join(a + b for a, b in izip_longest(s1, s2, fillvalue=''))
print(s3)
</code></pre>
</div>
