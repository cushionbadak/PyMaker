<p>So I started learning to program last week, and now I can't figure it out by myself. I need final answer to be "Size between 24 to 192." But if I round it, it gives me 192.0. What do I need to change?</p>
<pre><code>minp = 23.75
maxp = 192.4
minp1 = round(minp, 0)
maxp1 = round(maxp, 0)
print("Size between " + str(minp1) + " to " + str(maxp1) + ".")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>That's because the type of <code>minp1</code> and <code>maxp1</code> are still <strong><code>float</code>s</strong>, you can change them to <strong><code>int</code>s</strong> by passing them to the <code>int(..)</code> constructor:</p>
<pre><code>minp = 23.75
maxp = 192.4
minp1 = <b>int(</b>round(minp, 0)<b>)</b>
maxp1 = <b>int(</b>round(maxp, 0)<b>)</b>
print("Size between " + str(minp1) + " to " + str(maxp1) + ".")</code></pre>
<p>Or in a terminal:</p>
<pre><code>$ python3
Python 3.5.2 (default, Nov 17 2016, 17:05:23) 
[GCC 5.4.0 20160609] on linux
Type "help", "copyright", "credits" or "license" for more information.
&gt;&gt;&gt; minp = 23.75
&gt;&gt;&gt; maxp = 192.4
&gt;&gt;&gt; minp1 = int(round(minp, 0))
&gt;&gt;&gt; maxp1 = int(round(maxp, 0))
&gt;&gt;&gt; print("Size between " + str(minp1) + " to " + str(maxp1) + ".")
Size between <b>24</b> to <b>192</b>.</code></pre>
<p>Nevertheless, it is not a good idea to do string processing/concatenations yourself. You better format it with a <a href="https://docs.python.org/3/library/stdtypes.html#printf-style-bytes-formatting" rel="nofollow noreferrer"><em>formatting string</em></a>, like:</p>
<pre><code>minp = 23.75
maxp = 192.4
minp1 = round(minp, 0)
maxp1 = round(maxp, 0)
print("Size between <b>%d</b> to <b>%d</b>."<b>%(minp1,maxp1)</b>)</code></pre>
<p>Here <code>%d</code> stands for "<em>Signed integer decimal</em>". So by using format, Python will fill in the variables at the <code>%d</code> places itself which is elegant and requires less thinking from your side.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you wanted to leave the numbers as <code>float</code> and specifically output in integer format:</p>
<pre><code>def r(val):
    return "{0:.0f}".format(round(val))

minp = 23.75
maxp = 192.4
print("Size between {0} to {1}.".format(r(minp), r(maxp)))
</code></pre>
<p><strong>Result:</strong></p>
<pre><code>Size between 24 to 192.
</code></pre>
