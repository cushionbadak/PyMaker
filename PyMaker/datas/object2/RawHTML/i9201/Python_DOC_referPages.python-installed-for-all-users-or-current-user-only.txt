<div class="post-text" itemprop="text">
<p>You could check if the Python executable is located in the user's home directory. The location of the home directory is retrieved by using the <a href="https://docs.python.org/3/library/os.path.html#os.path.expanduser" rel="nofollow noreferrer"><code>os.path.expanduser()</code></a> method. The location of the Python interpreter is retrieved by using the <a href="https://docs.python.org/3/library/sys.html#sys.executable" rel="nofollow noreferrer"><code>sys.executable()</code></a> method.</p>
<p>The following function returns <code>True</code> if the Python interpreter was installed within the user's home directory, and <code>False</code> otherwise. It works under Linux, and should work under macOS and Windows (but I didn't test those).</p>
<pre class="lang-py prettyprint-override"><code>import sys
import os

def user_python():
    try:
        return sys.executable.startswith(os.path.expanduser("~"))
    except AttributeError:
        return False
</code></pre>
<p>The exception is needed because according to the documentation of <code>sys.executable()</code>, it may return <code>None</code> under some circumstances.</p>
<p>EDIT 2018-12-08: it works on Windows 10.</p>
