<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.cycle" rel="nofollow noreferrer">itertools.cycle(iterable)</a> method for this.</p>
<p>It takes an iterable as parameter and cycles through its items.</p>
<pre><code>&gt;&gt;&gt;cycle('ABCD') --&gt; A B C D A B C D A B C D 
</code></pre>
