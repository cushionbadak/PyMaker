<p>Look at the source, they're merged together:</p>
<pre><code>attrib.update(extra)
</code></pre>
<p><a href="https://hg.python.org/cpython/file/tip/Lib/xml/etree/ElementTree.py#l172" rel="nofollow noreferrer">https://hg.python.org/cpython/file/tip/Lib/xml/etree/ElementTree.py#l172</a></p>
<p>It's been there since ElementTree was a third party library supporting Python 1.5.2.</p>
<p>One would imagine that it's there for convenience, since it may be quite common to have a dictionary of attributes from somewhere, but it should also be ergonomic to write a literal function call with some attribute names, so neither case requires extra punctuation.</p>
