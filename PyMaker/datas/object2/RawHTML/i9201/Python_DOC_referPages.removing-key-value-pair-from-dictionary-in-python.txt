<p>I'm quite a beginner with python.</p>
<p>I'm trying to remove all the 'noone: 0' from this dictionary, so it will look the same as below but without any of the 'noone: 0':</p>
<pre><code>G = {'a': {'b': 10, 'c': 8, 'd': 3, 'noone': 0, 'e': 3}, 'f': {'g': 7, 'c': 5, 'h': 5, 'i': 2, 'j': 4, 'noone': 0, 'l': 2}}
</code></pre>
<p>I searched and found all the ways I should be implementing it, but cannot find a method that works. I tried this to no avail:</p>
<pre><code>for i in G:
    if str(G[i]) == 'noone':
        G.pop('noone', None)
</code></pre>

<div class="post-text" itemprop="text">
<p>I believe this would do what you want.</p>
<pre><code>for i in G:
    if 'noone' in G[i]:
        G[i].pop('noone')
</code></pre>
<p>What you have here (G) is indeed a dictionary, but more specifically, it's a dictionary whose values are also dictionaries.  thus, when we iterate through all the keys in G (<code>for i in G</code>), each key's corresponding value (<code>G[i]</code>) is a dictionary.  You can see this for yourself if you try running:</p>
<pre><code>for i in G:
    print(G[i])
</code></pre>
<p>So what you really want to do is pop from each dictionary in G.  That is, for each G[i], you want to remove 'noone' from <em>those</em> "sub"-dictionaries, not the main top one.</p>
<hr/>
<p>P.S.: if you really want to take advantage of python's convenience, you can even write simply</p>
<pre><code>for i in G:
    G[i].pop('noone', None)
</code></pre>
<p>By using the <a href="https://docs.python.org/3/library/stdtypes.html#dict.pop" rel="nofollow noreferrer">second argument into pop</a>, you don't even have to check to see if 'noone' is a key in G[i] first because the pop method will not raise an exception. (if you tried this two-liner without the second argument, you'd get an error for all sub-dicts that don't have 'noone' in them).</p>

<p>Iterate over the values and pop the key from each value:</p>
<pre><code>for v in G.values():
    _ = v.pop('noone', None)
</code></pre>

<p>I would go with dictionary comprehension:</p>
<pre><code>&gt;&gt;&gt; G = {'a': {'b': 10, 'c': 8, 'd': 3, 'noone': 0, 'e': 3}, 'f': {'g': 7, 'c': 5, 'h': 5, 'i': 2, 'j': 4, 'noone': 0, 'l': 2}}
&gt;&gt;&gt; {k1: {k2: v2 for (k2, v2) in v1.items() if k2 != 'noone'} for k1, v1 in G.items()}
{'f': {'g': 7, 'c': 5, 'j': 4, 'h': 5, 'i': 2, 'l': 2}, 'a': {'b': 10, 'e': 3, 'd': 3, 'c': 8}}
