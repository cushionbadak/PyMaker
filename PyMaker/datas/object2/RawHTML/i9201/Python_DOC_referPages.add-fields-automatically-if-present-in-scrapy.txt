<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate()</code></a> over the <code>response.css()</code> or <code>response.xpath()</code> to index the matching href attribute values:</p>
<pre><code>In [1]: {'Field%d' % index: link 
         for index, link in enumerate(response.css("ul.l-itemcassette.l-space_medium li a::attr('href')").extract(), start=1)}
Out[1]: 
{'Field1': u'#link1',
 'Field2': u'#link2',
 'Field3': u'#link3',
 'Field4': u'#link4'}
</code></pre>
<p>Note that I've assigned different <code>href</code> values for demonstration purposes.</p>
