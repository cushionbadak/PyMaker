<p>I'm having trouble trying to separate string and int/float from a single array. So far I could only find methods to remove/extract from list with specific string or number but not datatypes :(</p>
<p>Input:</p>
<pre><code>a = [1,2,3,'a',4,11,212,'a','b','c',3,32,4,'d','e']
</code></pre>
<p>Desired output:</p>
<pre><code>['a','a','b','c','d','e']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Python has the <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer">isinstance()</a> function to check if objects are of given type.<br/>
Btw your given list is not correct !</p>
<pre><code>&gt;&gt;&gt; l = [1, 2, 3, 'a', 4, 11, 212, 'a', 'b', 'c', 3, 32, 4, 'd', 'e']
&gt;&gt;&gt; [item for item in l if isinstance(item, str)]
['a', 'a', 'b', 'c', 'd', 'e']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/functions.html#type" rel="nofollow noreferrer"><code>type()</code></a> function to distinguish between the different data types.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use:</p>
<pre><code>a = [1,2,3,'a',4,11,212,'a','b','c',3,32,4,'d','e']
[x for x in a if type(x)==str]
</code></pre>
<p>Output is:</p>
<pre><code>['a', 'a', 'b', 'c', 'd', 'e']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>There are many ways to achieve this scenario, You can use idiomatic way too,</p>
<pre><code>a = [1,2,3,'a',4,11,212,'a','b','c',3,32,4,'d','e']

print [item for item in a if type(item) is str]

Output list: ['a', 'a', 'b', 'c', 'd', 'e']
</code></pre>
</div>
