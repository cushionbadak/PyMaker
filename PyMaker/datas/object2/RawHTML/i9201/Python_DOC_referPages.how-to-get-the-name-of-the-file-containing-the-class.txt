<div class="post-text" itemprop="text">
<p>Call <a href="https://docs.python.org/3/library/inspect.html#inspect.getfile" rel="nofollow noreferrer"><code>inspect.getfile</code></a> on the class/function/method/module. It's not guaranteed to work, but it's more reliable than just checking <a href="https://docs.python.org/3/reference/import.html#__file__" rel="nofollow noreferrer">the <code>__file__</code> attribute</a> of the object (for example, on Python 3.5, <code>inspect.getfile(collections.defaultdict)</code> works, while <code>collections.defaultdict.__file__</code> raises <code>AttributeError</code>).</p>
<p>If you have an instance, this won't work without converting to the class first, e.g. <code>inspect.getfile(type(instance))</code> (on Python 2, <code>instance.__class__</code> can be more reliable with old-style classes); it can only tell you where the class was defined, not where the instance was defined.</p>
