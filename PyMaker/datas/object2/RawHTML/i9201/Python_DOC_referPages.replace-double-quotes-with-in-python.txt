<div class="post-text" itemprop="text">
<p>I have a string - </p>
<pre><code>l = '{"a": "1", "b": "2"}'
</code></pre>
<p>I want to convert this string to - </p>
<pre><code>'{\"a\": \"1\", \"b\": \"2\"}'
</code></pre>
<p>For this I am trying to replace " with \"</p>
<p>Here's what I have tried -</p>
<pre><code>l.replace('\"', '\"')
'{"a": "1", "b": "2"}' 

l.replace('\"', '\\"')
'{\\"a\\": \\"1\\", \\"b\\": \\"2\\"}'
</code></pre>
<p>How do I convert <code>{\"a\": \"1\", \"b\": \"2\"}</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>print l.replace('"','\\"')
</code></pre>
<p><code>'\"'</code> doesn't mean anything special to Python, so you needn't to add <code>\</code> before <code>"</code>,if you run </p>
<p><code>print l.replace('\"', '\\"')</code>,you will get a single backslash too.</p>
<p>Actually what you are seeing is the representation of the string, it's added by <a href="https://docs.python.org/2/reference/datamodel.html#object.__repr__" rel="noreferrer"><strong>repr</strong>()</a> method.Python represents backslashes in strings as <code>\\</code> because the backslash is an <a href="https://docs.python.org/2.0/ref/strings.html" rel="noreferrer">Escape Character</a> .</p>
<p>If you print it, you will  get single backslash.</p>
<p>You can see more information from <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="noreferrer">String and Bytes literals</a>.</p>
</div>
