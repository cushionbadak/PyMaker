<div class="post-text" itemprop="text">
<p>You have a problem here:</p>
<pre><code>with open('objs.pickle', "latin1", "r") as f
</code></pre>
<p>Here are the arguments <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer"><code>open()</code></a> takes:</p>
<pre><code>open(file, mode='r', buffering=-1, encoding=None,
         errors=None, newline=None, closefd=True, opener=None)
</code></pre>
<p>In your case:</p>
<pre><code>open('objs.pickle', "latin1", "r")

file = 'objs.pickle'
mode = "latin1"
buffering = "r"
</code></pre>
<p>You should do it like so:</p>
<pre><code>open('objs.pickle', 'r', encoding="latin1")
</code></pre>
