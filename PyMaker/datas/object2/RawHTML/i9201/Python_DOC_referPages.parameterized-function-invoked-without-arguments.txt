<div class="post-text" itemprop="text">
<p>Many functions in Python have an optional <code>key</code> argument that takes a callable (a function) that is applied on each of the elements (here, each tuple pair).</p>
<p>You don't <em>need</em> to supply the argument yourself, <code>min</code> does that for you. From the <a href="https://docs.python.org/3/library/functions.html#min" rel="nofollow noreferrer">docs on <code>min</code></a>:</p>
<blockquote>
<p>There are two optional keyword-only arguments. The key argument specifies a one-argument ordering function like that used for <code>list.sort()</code>. </p>
</blockquote>
<p>Take a look <a href="https://docs.python.org/3/howto/sorting.html#key-functions" rel="nofollow noreferrer">here</a> for additional information on key functions.</p>
</div>
