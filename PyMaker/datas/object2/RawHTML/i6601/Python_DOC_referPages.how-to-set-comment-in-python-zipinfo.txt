<div class="post-text" itemprop="text">
<p>You can write it when you create the <code>ZipFile</code> object:</p>
<pre><code>with zipfile.ZipFile('myzip.zip', 'w') as zip:
    zip.write('file.py')
    zip.comment = b'This is my comment'
</code></pre>
<p>The text has to be entered as binary with prefix <code>b</code></p>
<p><a href="https://docs.python.org/3/library/zipfile.html#zipfile.ZipFile.comment" rel="nofollow noreferrer">https://docs.python.org/3/library/zipfile.html#zipfile.ZipFile.comment</a></p>
<p>If your archive already exists, you can also use the mode <code>a</code> to append the comment alone:</p>
<pre><code>with zipfile.ZipFile('myzip.zip', 'a') as zip:
    zip.comment = b'This is a new comment'
</code></pre>
<hr/>
<p>To set the comment on file zipped, you have to access the <code>ZipInfo</code> object like below, or create it with method <a href="https://docs.python.org/3/library/zipfile.html#zipfile.ZipInfo.from_file" rel="nofollow noreferrer"><code>from_file</code></a>:</p>
<pre><code>with zipfile.ZipFile('myzip.zip', 'w') as zip:
    zip.write('file.py')
    info = zip.getinfo('file.py')
    info.comment = b'zipped file comment'
</code></pre>
