<p>I have two strings, let's say:</p>
<pre><code>a = "Hello, I'm Daniel and 15 years old."
b = "Hello, I'm (name) and (age) years old."
</code></pre>
<p>And now I want python to find the replaced words. It can be something like this:</p>
<pre><code>{"name": "Daniel", "age": "15"}
</code></pre>
<p>I never found a solution! Thank you very much for your help!</p>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a> with <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>str.split()</code></a> and string slicing (to remove <code>(</code> and <code>)</code> from keys):</p>
<pre><code>res = {v[1:-1]: k for k, v in zip(a.split(), b.split()) if k !=v}
</code></pre>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>str.split()</code></a> is used to split each string on whitespace.</p>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; res
{'name': 'Daniel', 'age': '15'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <code>python3</code> you can format the string with values from a dictionary.</p>
<p><code>"Hello, I'm {name} and {age} years old.".format(**{"name": "Daniel", "age": "15"})</code></p>
<p>or</p>
<p><code>c = {"name": "Daniel", "age": "15"}
"Hello, I'm {name} and {age} years old.".format(**c)</code></p>
<p>Alternatively, if you are asking how to extract these values you could find the values using a regular expression :</p>
<pre><code>import re
regularExpression = "^Hello, I'm (.*) and ([0-9]*).*"
input = "Hello, I'm Daniel and 15 years old."
match = re.search(regularExpression,input)
result = {"name": match.group(1), "age": match.group(2)}
</code></pre>
