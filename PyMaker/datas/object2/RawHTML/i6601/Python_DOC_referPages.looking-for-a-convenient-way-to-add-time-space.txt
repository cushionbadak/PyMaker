<p>I want to add a month datetime to my current datetime, if I do this, I only can convert the datetime to timestamp, then add the timestamp space:</p>
<p>If is the February, I will add <code>28 * 24 * 3600</code>.<br/>
If is a solar month of 31 days I will add <code>31 * 24 * 3600</code>.<br/>
If is a solar month of 30 days I will add <code>30 * 24 * 3600</code>.  </p>
<p>Then convert to datetime again.</p>
<p>So, there is a lot of things to do, python is comfortable for developer, if there is a simpler and more convenient way to do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/datetime.html#timedelta-objects" rel="nofollow noreferrer"><code>datetime.timedelta</code></a>:</p>
<pre><code>from calendar import monthrange
from datetime import timedelta, datetime


def add_month(date):
    days_in_month = monthrange(date.year, date.month)[1]
    new_date = date + timedelta(days=days_in_month)
    return new_date


print(add_month(datetime(2017, 1, 1)))
&gt; 2017-02-01 00:00:00

print(add_month(datetime(2017, 2, 5)))
&gt; 2017-03-05 00:00:00
</code></pre>
<hr/>
<p>Or with a third party library <a href="https://dateutil.readthedocs.io" rel="nofollow noreferrer"><code>dateutil</code></a>:</p>
<pre><code>from datetime import datetime

from dateutil.relativedelta import relativedelta


def add_month(date):
    new_date = date + relativedelta(months=1)
    return new_date


print(add_month(datetime(2017, 1, 1)))
&gt; 2017-02-01 00:00:00

print(add_month(datetime(2017, 2, 5)))
&gt; 2017-03-05 00:00:00
</code></pre>
</div>

<p>You can use <a href="http://dateutil.readthedocs.io/en/stable/_modules/dateutil/relativedelta.html" rel="nofollow noreferrer">relativedelta</a> :</p>
<pre><code>from datetime import datetime
from dateutil.relativedelta import relativedelta

def addTimeSpace(ori_datetime, relative_date):

    day = relativedelta(days=+1)

    added_datetime = ori_datetime + relative_date
    if ori_datetime.day &gt; added_datetime.day:
        added_datetime = added_datetime + day

    return added_datetime

print(addTimeSpace(datetime.now(), relativedelta(years=+4, months=+1)))  #there will output: 2021-10-26 20:00:53.222794   
</code></pre>
