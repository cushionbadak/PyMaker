<p>This question has been asked quite a bit, and I've tried each solution I come across, but haven't had any success. I'm trying to print out every variable and its value using the following two lines.</p>
<pre><code>for name, value in globals():
    print(name, value)
</code></pre>
<p>This gives an error: <code>Too many values to unpack
</code>When I run: </p>
<pre><code>for name in globals():
    print(name)
</code></pre>
<p>I only get the names of the variables. Any help would be appreciated. </p>
</div>
<div class="post-text" itemprop="text">
<p><code>globals()</code> returns a <code>dict</code> - so, to iterate through a <code>dict</code>'s key/value pairs, you call it's <code>items()</code> method:</p>
<pre><code>dog = 'cat'
&gt;&gt;&gt; for name, value in globals().items():
...     print(name, value)
... 
('__builtins__', &lt;module '__builtin__' (built-in)&gt;)
('__name__', '__main__')
('dog', 'cat')
('__doc__', None)
('__package__', None)
&gt;&gt;&gt; 
</code></pre>
<p>This is explained within the docs for Data Structures, available <a href="https://docs.python.org/3/tutorial/datastructures.html#looping-techniques" rel="nofollow noreferrer">here!</a></p>
<p>Python 3 version:</p>
<pre><code>for name, value in globals().copy().items():
    print(name, value)
</code></pre>
<p>if we do not copy the output globals functions then there will be RuntimeError.</p>
<p>Also if there are list and dictionaries in the output of some variable then use <a href="https://docs.python.org/3/library/copy.html" rel="nofollow noreferrer">deepcopy</a>() instead of copy()</p>
<pre><code>for name, value in globals().deepcopy().items():
    print(name, value)
</code></pre>
