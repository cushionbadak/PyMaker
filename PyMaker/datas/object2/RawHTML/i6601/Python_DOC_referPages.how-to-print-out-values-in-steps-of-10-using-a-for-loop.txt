<div class="post-text" itemprop="text">
<p>this code prints out the values from -30 up to 40, but I need it to print out the values from -30 to 40 in steps of 10. </p>
<pre><code>for x in range(-30, 40):
        print(x)
</code></pre>
<p>Can someone help me? I wasn't able to find anything when googeling this problem.</p>
</div>
<div class="post-text" itemprop="text">
<p>The third argument for <code>range</code> is the <code>step</code>, so just use:</p>
<pre><code>for x in range(-30, 40, 10):
    print(x)
</code></pre>
<p>However note that the <code>stop</code> is exclusive, so if you wanted to include <code>40</code> it should be:</p>
<pre><code>for x in range(-30, 41, 10):
    print(x)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<h1>Following the <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer"><code>documentation</code></a>:</h1>
<p>The documentation <em>defines</em> <code>range</code> as taking three arguments:</p>
<blockquote>
<p>range(start, stop[, step])</p>
</blockquote>
<p>The last argument, <code>step</code>, is the interval for each <code>loop</code>. So, to <code>loop</code> <code>x</code> in intervals of <code>10</code>, we can set the <code>step</code> argument to <code>10</code>:</p>
<pre><code>for x in range(-30, 40, 10):
   print(x)
</code></pre>
<p>which <code>prints</code> out the following values for <code>x</code>:</p>
<pre><code>-30
-20
-10
0
10
20
30
</code></pre>
<p><em>N.B. that as <code>range</code> stops when it reaches the <code>stop</code> argument (exclusive), <code>40</code> is not included.</em></p>
</div>
<div class="post-text" itemprop="text">
<p>Since print in python 3+ is a function, you can achieve the same result using:</p>
<p><strong>Code:</strong></p>
<pre><code>print(*range(-40, 41, 10), sep="\n")
</code></pre>
<p><strong>output</strong></p>
<pre><code>-40
-30
-20
-10
0
10
20
30
40
</code></pre>
<p>Basically print will unpack the range and use separator <code>\n</code>.</p>
</div>
