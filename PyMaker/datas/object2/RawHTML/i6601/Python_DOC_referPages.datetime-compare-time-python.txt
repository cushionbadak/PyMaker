<div class="post-text" itemprop="text">
<p>You can get <em>just</em> the time object by calling the <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.time" rel="nofollow noreferrer"><code>datetime.time()</code> method</a>. Compare that to another <code>time</code> object:</p>
<pre><code># scheduled after 12 noon.
get_schedule(a).time() &gt; datetime.time(12, 0)
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; datetime.datetime(2016, 1, 1, 2, 25).time()
datetime.time(2, 25)
&gt;&gt;&gt; datetime.datetime(2016, 1, 1, 2, 25).time() &gt; datetime.time(12, 0)
False
</code></pre>
</div>
