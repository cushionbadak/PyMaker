<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer"><code>getattr</code></a> to get attribute from an object:</p>
<pre><code># method_name = 'describe'
method = getattr(uloader, method_name)
method(porsche)
</code></pre>
