<p>Simple question but cannot find a simple solution. If I have this code:</p>
<pre><code>print("I don't like",FavColour,", I prefer Red")
</code></pre>
<p>It prints:</p>
<pre><code>I don't like FavColour , I prefer Red
</code></pre>
<p>How do I get rid of the space between <code>FavColour</code> and the comma <code>,</code> as I want my work to be grammatically correct. If I remove the commas, it becomes a syntax error.</p>
</div>
<div class="post-text" itemprop="text">
<p>Either construct the entire string instead of passing pieces to <code>print()</code>:</p>
<pre><code>print("I don't like" + FavColour + ", I prefer Red")
</code></pre>
<p>or use <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>print()</code>'s <code>sep</code> argument</a> to change the spaces to nothing:</p>
<pre><code>print("I don't like",FavColour,", I prefer Red", sep='')
</code></pre>
</div>
