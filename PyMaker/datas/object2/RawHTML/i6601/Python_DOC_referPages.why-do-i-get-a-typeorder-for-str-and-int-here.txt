<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer"><code>input()</code></a> returns type <code>str</code> in Python 3.x.</p>
<p>So you are doing <code>grosssalary &gt; 100000</code> which is <code>str &gt; int</code>.</p>
<p>To solve, use: </p>
<pre><code>gross_salary = int(input("Enter the employee's gross salary: $"))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>In Python 3.x, return value of <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer"><code>input()</code></a> is always of type <code>str</code>, so you are comparing a <code>str</code> object to an <code>int</code> object, which yields a TypeError. You cannot do that, you have to convert it to <code>int</code> or <code>float</code> before comparison. </p>
<p>Try this:</p>
<pre><code>grosssalary = float(input("Enter the employee's gross salary: $"))
</code></pre>
</div>
