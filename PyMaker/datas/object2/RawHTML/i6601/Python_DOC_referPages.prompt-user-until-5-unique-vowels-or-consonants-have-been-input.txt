<pre><code>import itertools as it


words =  'hello', 'luck', 'liar'

seen_vowels = "".join(c for w in words for c in w if c in "aeiou")
seen_vowels

<p>Trailing consonants could be found via <code>takewhile</code> of each reversed word.</p>
<pre><code>trailing_cons = "".join("".join(it.takewhile(lambda x: x not in "aeiou", w[::-1]))[::-1] for w in words)
trailing_cons
# 'ckr'
</code></pre>
<hr/>
<p><strong>Details</strong></p>
<p>Note, <code>takewhile</code> grabs all consonants before the first observed vowel.  By reversing each word, we get the trailing consonants. </p>
<pre><code>"".join(it.takewhile(lambda x: x not in "aeiou", "luck"[::-1]))
# 'kc'
</code></pre>
