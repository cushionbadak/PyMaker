<p>I want to change the float to hex and hex to float.</p>
<p>So I want to get equal of the first float and end of the float.</p>
<p>This is what I tried, on Python 2.7; <code>f1</code> = first float, <code>h</code> = hex string, <code>f2</code> = new float value from the hex string.</p>
<pre><code>f1 = 15.3
h=f1.hex() #0x1.e99999999999ap+3
</code></pre>
<p>How can I change the <code>h</code> hex string to a float again?</p>
<p>I've tried  </p>
<pre><code>f2 = int(h,16)
f2 = float(h)
f2 = h.float()
</code></pre>
<p>but none of those work.</p>
<p>I can also use Python 3.5 if that helps.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/3/library/stdtypes.html#float.fromhex" rel="noreferrer"><code>float.fromhex()</code> classmethod</a>:</p>
<pre><code>&gt;&gt;&gt; float.fromhex(h)
15.3
</code></pre>
<p>This is the direct inverse of the <code>float.hex()</code> instance method.</p>
