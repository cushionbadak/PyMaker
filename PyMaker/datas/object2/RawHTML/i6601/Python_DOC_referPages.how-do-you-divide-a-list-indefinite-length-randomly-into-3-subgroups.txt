<p>Lets say that my program asks a user to input a number indefinitely and I appended it in a list.</p>
<pre><code>#Given Example:
List = [105, 167, 262, 173, 123, 718, 219, 272, 132]
</code></pre>
<p>what I wanted to do is to randomly pick any of the numbers so my sample output would be:</p>
<pre><code>first = [105, 262, 173]
second = [167, 123, 132]

</code></pre>
<p>Having said, since the user input would be indefinite there will be chances that it would be an odd number.</p>
<p>I am currently using random.sample to choose a number from that list and append it in a list, after that I'll append the sub-groups of list to create a list within a list.</p>

<div class="post-text" itemprop="text">
<p>You could shuffle your list with <a href="https://docs.python.org/3/library/random.html#random.shuffle" rel="nofollow noreferrer"><code>random.shuffle</code></a>:</p>
<pre><code>&gt;&gt;&gt; import random
&gt;&gt;&gt; l = [105, 167, 262, 173, 123, 718, 219, 272, 132]
&gt;&gt;&gt; random.shuffle(l)
&gt;&gt;&gt; l
[167, 123, 173, 718, 262, 272, 105, 219, 132]
</code></pre>
<p>And extract 3 sub-lists with <a href="https://stackoverflow.com/a/509295/6419007">slicing</a>. The first list will have indices 0, 3 and 6 from the shuffled <code>l</code>, the second list will have indices 1, 4 and 7 and the third will have indices 2, 6 and 8:</p>
<pre><code>&gt;&gt;&gt; l[::3]
[167, 718, 105]
&gt;&gt;&gt; l[1::3]
[123, 262, 219]
&gt;&gt;&gt; l[2::3]
[173, 272, 132]
</code></pre>
<p>As a bonus, it will work with lists of any length:</p>
<pre><code>&gt;&gt;&gt; l = [1, 2, 3, 4, 5]
&gt;&gt;&gt; random.shuffle(l)
&gt;&gt;&gt; [l[x::3] for x in range(3)]
[[2, 4], [1, 5], [3]]
</code></pre>
<p>You can be sure that every element will appear exactly once in the nested lists.</p>
<p>You could also define more sublists:</p>
<pre><code>&gt;&gt;&gt; l = list(range(20))
&gt;&gt;&gt; n = 5
&gt;&gt;&gt; random.shuffle(l)
&gt;&gt;&gt; [l[x::n] for x in range(n)]
[[5, 18, 9, 4], [11, 12, 1, 15], [13, 3, 6, 8], [17, 0, 14, 2], [10, 16, 7, 19]]
</code></pre>

<p>If List is going to remain short such that you can read it into the memory, then you can use <code>[random.shuffle][1]</code></p>
<pre><code>import random
List = [105, 167, 262, 173, 123, 718, 219, 272, 132]

random.shuffle(List)
first = L[0:3]
</code></pre>
<p>Now use List Comprehension to subtract the original <code>List</code> with <code>first</code></p>
<pre><code>newList = [item for item in List if item not in first]
</code></pre>
<p>Now use <code>newList</code> for <code>second</code> and <code>third</code> and you'd be good to go! :)</p>
