<p>You probably want to use <a href="http://pandas.pydata.org/" rel="nofollow noreferrer"><code>pandas</code></a> when it comes to CSV files or table-like data:</p>
<pre><code>import pandas as pd

df_data = pd.DataFrame.from_csv('Data.csv')

# Analyze
for index, row in df_data.iterrows():
    pass

df_data.to_csv('new_Data.csv')
</code></pre>
<p>For reading you have several options like</p>
<ul>
<li><a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.from_csv.html" rel="nofollow noreferrer"><code>pandas.DataFrame.from_csv</code></a></li>
<li><a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.read_csv.html" rel="nofollow noreferrer"><code>pandas.read_csv</code></a></li>
<li><a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.read_table.html" rel="nofollow noreferrer"><code>pandas.read_table</code></a></li>
</ul>
<p>and, as you see, use</p>
<ul>
<li><a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.to_csv.html" rel="nofollow noreferrer"><code>pandas.DataFrame.to_csv</code></a> </li>
</ul>
<p>to save your transformed or newly created <code>DataFrame</code>.</p>
