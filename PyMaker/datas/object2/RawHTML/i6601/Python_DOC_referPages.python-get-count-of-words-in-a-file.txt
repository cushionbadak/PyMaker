<div class="post-text" itemprop="text">
<p>use <code>.readlines()</code> </p>
<p><code>.read()</code> return char continuously. so Counter counts the char.
but <code>.readlines()</code> return word(The fact is a line, but in your case, a word in a line)</p>
</div>
<div class="post-text" itemprop="text">
<p>Just change your argument for Counter.
From</p>
<pre><code>wordcount=Counter(f1.read())
</code></pre>
<p>To</p>
<pre><code>wordcount=Counter(f1.readlines().split())
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have to use <code>readlines()</code> instead of <code>read()</code>. Also you need to get rid of <code>\n</code> characters as well since using <code>readlines()</code> will read them as well.    </p>
<pre><code>from collections import Counter

with open('chk.txt') as f:
    mylist = f.read().splitlines()   #get rid of newline character

wordcount=Counter(mylist)

for k,v in wordcount.items():
    print (k ,v)

#Output:
('INSERT', 2)
('UPDATE', 2)
('DELETE', 2)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<blockquote>
<p>Use <a href="https://docs.python.org/3/tutorial/inputoutput.html#methods-of-file-objects" rel="nofollow noreferrer"><strong>readlines()</strong></a> instead of read,</p>
</blockquote>
<pre><code>from collections import Counter

f1 = open("test.txt", "r")
wordcount=Counter(f1.readlines())

#print(wordcount)

for k,v in wordcount.items():
    print (k ,v)
</code></pre>
<p>To get better result use split() or splitlines() to remove <code>\n</code> </p>
<pre><code>wordcount=Counter(f1.readlines().splitlines())
# or
wordcount=Counter(f1.read().split())
</code></pre>
<p>Output:</p>
<pre><code>DELETE 2
INSERT 2
UPDATE 2
</code></pre>
