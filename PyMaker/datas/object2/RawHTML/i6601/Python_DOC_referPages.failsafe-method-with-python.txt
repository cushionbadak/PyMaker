<p>Lets say I have a method <code>method1()</code> which has a blocking HTTP request using <code>urllib</code>.</p>
<p>Say if this HTTP request doesn't return within <code>3 seconds</code> I want to stop this and return a different method's value, say <code>method2()</code>. How can this be done? </p>
</div>
<div class="post-text" itemprop="text">
<p>You can mention the <code>timeout</code> for your <a href="https://docs.python.org/3/library/urllib.request.html#module-urllib.request" rel="nofollow noreferrer">urllib request</a> in the optional parameter timeout.</p>
<pre><code>urllib.request.urlopen(url, data=None, [timeout, ]*, cafile=None,
    capath=None, cadefault=False, context=None)
</code></pre>
<p>Where you can pass the time value in the timeout parameter, you can do other functions according to the data you get.</p>
</div>
<span class="comment-copy">Threads and event loops often have a timeout option. That would be where I'd look.</span>
<span class="comment-copy">Since you've tagged your question <a href="https://stackoverflow.com/tags/python-requests/info">python-requests</a> see the <code>requests</code> documentation for passing a <a href="http://docs.python-requests.org/en/master/user/quickstart/#timeouts" rel="nofollow noreferrer"><code>timeout</code></a> argument.</span>
<span class="comment-copy">Note that a <code>URLError</code> isn't by itself proof of a timeout.</span>
<span class="comment-copy">I think there can others causes for URLError as well, but may not be a good way to check for timeout here.</span>
<span class="comment-copy">The OP would need to check the <code>URLError</code>'s <code>reason</code> attribute which should be something like <code>'&lt;urlopen error timed out&gt;'</code>.</span>
