<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strftime" rel="nofollow noreferrer"><code>strftime()</code></a> is used to format a <code>datetime.datetime</code> object as a string.</p>
<p>Instead use <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strptime" rel="nofollow noreferrer"><code>datetime.strptime()</code></a> to parse the incoming string into a Python <code>datetime</code> object:</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; dt = datetime.strptime('2017-10-28', '%Y-%m-%d')
&gt;&gt;&gt; dt
datetime.datetime(2017, 10, 28, 0, 0)
</code></pre>
<p>You can then convert the <code>datetime</code> to a <code>date</code>:</p>
<pre><code>&gt;&gt;&gt; d = dt.date()
&gt;&gt;&gt; d
datetime.date(2017, 10, 28)
</code></pre>
