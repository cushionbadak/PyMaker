<p>I am trying to open files by getting the path from a dictionary. Some of the file names have commas (,) and other such characters which when used give a "no such file found error"</p>
<p>For instance the following file path will not open: <code>foo,%20bar.mp3</code></p>
<p>If characters like commas exist then it should be encoded as : <code>foo%2C%20bar.mp3</code></p>
<p>Can anyone tell me how to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You may need <code>pathname2url</code></p>
<p>Python 2.x (<a href="https://docs.python.org/2.7/library/urllib.html#urllib.pathname2url" rel="noreferrer">docs</a>)</p>
<pre><code>&gt;&gt;&gt; from urllib import pathname2url 
&gt;&gt;&gt; pathname2url('foo, bar.mp3')
'foo%2C%20bar.mp3'
</code></pre>
<p>Python 3.x (<a href="https://docs.python.org/3/library/urllib.request.html#urllib.request.pathname2url" rel="noreferrer">docs</a>)</p>
<pre><code>&gt;&gt;&gt; from urllib.request import pathname2url
&gt;&gt;&gt; pathname2url('foo, bar.mp3')
'foo%2C%20bar.mp3'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>from urllib import pathname2url
pathname2url('foo,bar.mp3')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use urllib. The following example might need to be changed if you use Python 3.x, but the general idea is the same:</p>
<pre><code>import urllib

encoded_filename = urllib.quote(filename)
f = open(encoded_filename)
</code></pre>
