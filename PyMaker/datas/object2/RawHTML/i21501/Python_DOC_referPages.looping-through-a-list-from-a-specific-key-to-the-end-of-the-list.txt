<p>in Python how do i loop through list starting at a key and not the beginning.
e.g.</p>
<pre><code>l = ['a','b','c','d']
</code></pre>
<p>loop through <code>l</code> but starting at b e.g. <code>l[1]</code></p>
</div>
<div class="post-text" itemprop="text">
<h2>The straightforward answer</h2>
<p>Just use <a href="http://docs.python.org/tutorial/introduction.html#lists" rel="nofollow noreferrer">slicing</a>:</p>
<pre><code>&gt;&gt;&gt; l = ['a','b','c','d']
&gt;&gt;&gt; for i in l[1:]:
...     print(i)
... 
b
c
d
</code></pre>
<p>It will generate a new list with the items before <code>1</code> removed:</p>
<pre><code>&gt;&gt;&gt; l[1:]
['b', 'c', 'd']
</code></pre>
<h2>A more efficient alternative</h2>
<p>Alternatively, if your list is huge, or you are going to slice the list a lot of times, you can use <a href="https://docs.python.org/3/library/itertools.html#itertools.islice" rel="nofollow noreferrer"><code>itertools.islice()</code></a>. It returns an iterator, avoiding copying the entire rest of the list, saving memory:</p>
<pre><code>&gt;&gt;&gt; import itertools
&gt;&gt;&gt; s = itertools.islice(l, 1, None)
&gt;&gt;&gt; for i in s:
...     print(i)
... 
b
c
d
</code></pre>
<p>Also note that, since it returns an interator, you can iterate over it only once:</p>
<pre><code>&gt;&gt;&gt; import itertools
&gt;&gt;&gt; s = itertools.islice(l, 1, None)
&gt;&gt;&gt; for i in s:
...     print(i)
... 
b
c
d
&gt;&gt;&gt; for i in s:
...     print(i)
&gt;&gt;&gt;
</code></pre>
<h2>How to choose</h2>
<p>I find slicing clearer/more pleasant to read but <code>itertools.islice()</code> can be more efficient. I would use slicing most of the time, relying on <code>itertools.islice()</code> when my list has thousands of items, or when I iterate over hundreds of different slices.</p>
