<div class="post-text" itemprop="text">
<p>How to convert GMT time to string time? it's quite strange, when I print the time in back-end, the time is in string time format, but when I transfer it to the front-end via JSON file, the time has changed to GMT format. Why? Could someone help me?</p>
<pre><code>Input: Wed, 25 Jul 2018 19:19:42 GMT

Output: 2018-07-25 19:19:42
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>When you convert the original time to JSON Format, it will be changed to GMT Time format. To solve this problem, you can change the time to string before converting it to JSON.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could do something like -</p>
<pre><code>from datetime import datetime
ip = 'Wed, 25 Jul 2018 19:19:42 GMT'
op = datetime.strftime(datetime.strptime(ip,'%a, %d %b %Y %H:%M:%S %Z'), '%Y-%m-%d %H:%M:%S')
# op 2018-07-25 19:19:42
</code></pre>
<p>Refer to this beautiful table <a href="https://docs.python.org/2/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer">here</a> to get the formats right and then look into <code>strftime</code> and <code>strptime</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You can find resources here on <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">strptime</a></p>
<pre><code>from datetime import datetime
t = 'Wed, 25 Jul 2018 19:19:42 GMT'
datetime.strptime(t,'%a, %d %b %Y %H:%M:%S %Z')
datetime.datetime(2018, 7, 25, 19, 19, 42)
</code></pre>
</div>
<span class="comment-copy">Tip: Look at strptime and strftime.</span>
<span class="comment-copy">You may want to change the third line to <code>datetime.strptime(t,'%a, %d %b %Y %H:%M:%S %Z')</code> to parse the time zone name.</span>
