<p>my question is simple.</p>
<p>I got my string :</p>
<pre><code>a = '0,0127'
</code></pre>
<p>I want to convert it to a number but when i compile </p>
<pre><code>float(a)
</code></pre>
<p>i got the following message error :</p>
<blockquote>
<p>ValueError: could not convert string to float: '0,0127'</p>
</blockquote>
<p>Is there another way to convert it to a number ?</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <code>str.replace</code></p>
<p><strong>Ex:</strong></p>
<pre><code>a = '0,0127'
print(float(a.replace(",", ".")))
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>0.0127
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The reason this isn't working is because the decimal type only recognizes periods (<code>.</code>) for the decimal delimiter as this is what is common in, e.g., english. You could manually change the string or do</p>
<pre><code>a = a.replace(",", ".")
float(a)
</code></pre>
<p>Which should work.</p>
