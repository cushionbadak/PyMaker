<div class="post-text" itemprop="text">
<blockquote>
<p><code>return</code> may only occur syntactically nested in a function definition</p>
</blockquote>
<p>Source: <a href="https://docs.python.org/3/reference/simple_stmts.html#the-return-statement" rel="nofollow noreferrer">https://docs.python.org/3/reference/simple_stmts.html#the-return-statement</a></p>
<p>Your return is not nested inside a function. Therefore, the error.</p>
<p>You can use <code>print("not found")</code> if you want to display the text.</p>
</div>
