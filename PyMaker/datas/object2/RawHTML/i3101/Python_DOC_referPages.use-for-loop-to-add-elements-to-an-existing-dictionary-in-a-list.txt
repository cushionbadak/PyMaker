<p>I have the following list of dictionaries:</p>
<pre><code>d = [{'Sport': 'MLB', 'Position': 'SP', 'Age': '25'},
     {'Sport': 'NBA', 'Position': 'PG', 'Age': '28'}]
</code></pre>
<p>I have another list that i would like to append to the above by using a for loop:</p>
<pre><code>c = [{'2015 Salary' : '25'}, {'2015 Salary': '35'},
    {'2016 Salary' : '30'}, {'2016 Salary' : '40'}]
</code></pre>
<p>I would like the final output to be</p>
<pre><code>e = [{'Sport': 'MLB', 'Position': 'SP', 'Age': '25', '2015 Salary': '25',  
     '2016 Salary': '30'},
     {'Sport': 'NBA', 'Position': 'PG', 'Age': '28', '2015 Salary': '35',  
     '2016 Salary': '40'}}]
</code></pre>
<p>The positions would always be the same thats why i was going to use a for loop. </p>

<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/itertools.html#itertools.cycle" rel="nofollow noreferrer"><code>itertools.cycle</code></a> to repeat over <code>d</code> while updating from <code>c</code>.</p>
<pre><code>import itertools

for existing, new_dict in zip(cycle(d), c):
    existing.update(new_dict)
</code></pre>
<p><code>cycle</code> turns a list into an infinite list by restarting from the beginning whenever it is exhausted. When you zip them together, <code>cycle(d)</code> and <code>c</code> pair up to give:</p>
<pre><code>({'Sport': 'MLB', 'Position': 'SP', 'Age': '25'}, {' 2015 Salary' : '25'})
({'Sport': 'NBA', 'Position': 'PG', 'Age': '28'}, {'2015 Salary': '35'})
({'Sport': 'MLB', 'Position': 'SP', 'Age': '25'}, {'2016 Salary ': '30'})
({'Sport': 'NBA', 'Position': 'PG', 'Age': '28'}, {'2016 Salary' : '40'})
</code></pre>
<p>Which then update <code>d</code> to your expected output.</p>

<p>You can achieve it by:</p>
<pre><code>for i, current_d in enumerate(d):
    current_l = c[i::2]
    for e in current_l:
        current_d.update(e)
</code></pre>
<p><code>d</code> will contain the information that you want.</p>

<div class="post-text" itemprop="text">
<p>Here is my one-line style solution using <strong>comprehensions</strong>, without any needed import, and which works also whatever the length of <code>d</code> input:  </p>
<pre><code>d = [{'Sport': 'MLB', 'Position': 'SP', 'Age': '25'},
     {'Sport': 'NBA', 'Position': 'PG', 'Age': '28'}]

c = [{'2015 Salary': '25'}, {'2015 Salary': '35'},
     {'2016 Salary': '30'}, {'2016 Salary': '40'}]

result = [{k:v for i in ([d[j]] + c[j::len(d)]) for k,v in i.items()} for j in range(len(d))]

print(result)
# [{'Sport': 'MLB', 'Position': 'SP', 'Age': '25', '2015 Salary': '25', '2016 Salary': '30'},
#  {'Sport': 'NBA', 'Position': 'PG', 'Age': '28', '2015 Salary': '35', '2016 Salary': '40'}]
</code></pre>
