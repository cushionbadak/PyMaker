<div class="post-text" itemprop="text">
<p>What you are trying to do is turn a list of digits into the according number (and add <code>1</code>). You can <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a> the reversed list in order to pair a digit with its appropriate power of <code>10</code>:</p>
<pre><code>digits = [1, 2, 3]
sum(10**i * y for i, y in enumerate(digits[::-1])) + 1
# 124
</code></pre>
<p>You can apply that to your other example as follows, using <code>zip</code>:</p>
<pre><code>A = [1,2,3]
B = [6,5,4]
sum(10**i * (x+y) for i, (x, y) in enumerate(zip(B, A[::-1])))
# 579
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do this without a loop:</p>
<pre><code>A = [1,2,3]
B = [6,5,4]

C = list(map(sum,zip(A,B[::-1]) ))

print(C)
</code></pre>
<p><code>zip()</code> - creates pairs of all elements of iterables, you feed it <code>A</code> and <code>B</code> reversed (via slicing). Then you sum up each pair and create a list from those sums.</p>
<p><code>map( function, iterable)</code> - applies the function to each element of the iterable</p>
<p><code>zip()</code> works when both list have the same length, else you would need to leverage <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow noreferrer">itertools.zip_longest()</a> with a defaultvalue of <code>0</code>.</p>
<pre><code>K = [1,2,3,4,5,6]
P = list(map(sum, zip_longest(K,C,fillvalue=0)))

print(P)
</code></pre>
<p>Output:</p>
<pre><code>[5, 7, 9]   # zip of 2 same length lists A and B reversed
[6, 9, 12, 4, 5, 6]   # ziplongest for no matter what length lists
</code></pre>
