<p>So, i have a result and i can limit his floating precision using this:</p>
<pre><code>result = 0.12345
print ("Result = %.2f" % (result) ) 
#will print ~ 0.12
</code></pre>
<p>What can i do to substitute the <strong>'2' on %.2f to a variable number</strong>?</p>
<blockquote>
<p>Something like this:</p>
</blockquote>
<pre><code>from random import randint
LIMIT = randint(1,5)
#pick some random 'limit' between 1 and 5, inclusive, just to demonstrate

print ("Result = %.~LIMIT~f" % (result) ) 
</code></pre>

<div class="post-text" itemprop="text">
<p>You can pass your limit alongside the format call:</p>
<pre><code>LIMIT = 3
result = 0.12345
print("Result = %.*f" % (LIMIT, result))  # 0.123
</code></pre>
<p>Or better yet, use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>str.format()</code></a> as <code>%</code> formatting has been deprecated: </p>
<pre><code>LIMIT = 3
result = 0.12345
print("Result = {:.{}f}".format(result, LIMIT))  # 0.123
</code></pre>
</div>
<span class="comment-copy">You can put an asterisk in place of a format width specification, to take the width from a parameter instead.  So: <code>"Result = %.*f" % (LIMIT, result)</code></span>
