<p>I have a list with sublist's in it and want to group them based on a common value.</p>
<p>For example:</p>
<pre><code>my_list = [["John", 1], ["Mark", 3], ["Peter", 5], ["Mark", 6], ["Mark", 33], ["John", 2]]
</code></pre>
<p>I would like to group these based on the first value (the name). The desired result would be:</p>
<pre><code>grouped_list = [[["John", 1], ["John", 2]], [["Mark", 3], ["Mark", 6], ["Mark", 33]], [["Peter", 5]]] 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer"><strong><code>itertools.groupby</code></strong></a>.</p>
<p>Since you want to only group by the first element of each sublist, you must supply a custom key.</p>
<pre><code>import itertools

[list(g) for _, g in itertools.groupby(sorted(my_list), lambda x: x[0])]
</code></pre>
<p>Output:</p>
<pre><code>[[['John', 1], ['John', 2]],
 [['Mark', 3], ['Mark', 6], ['Mark', 33]],
 [['Peter', 5]]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Nice question. I prefer the answer by @user3483203 using <code>itertools.groupby</code> but here is a solution with no imports;</p>
<pre><code>result = []
name_to_index = {}

for l in my_list:
    if l[0] in name_to_index:
        result[name_to_index.get(l[0])].append(l)
    else:
        result.append([l])
        name_to_index[l[0]] = len(result) - 1
</code></pre>
<p>Output:</p>
<pre><code>[[['John', 1], ['John', 2]], [['Mark', 3], ['Mark', 6], ['Mark', 33]], [['Peter', 5]]]
</code></pre>
</div>
