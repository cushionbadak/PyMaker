<div class="post-text" itemprop="text">
<p>The error message <code>invalid literal for int() with base 2: '3'</code> tells you what is wrong: In <a href="https://en.wikipedia.org/wiki/Binary_number" rel="nofollow noreferrer">base 2</a>, the only valid digits are <code>0</code> and <code>1</code>. If you're trying to parse a string in base 2, <code>3</code> is an invalid digit.</p>
<p>If you want a binary string from an integer string, you can use one of the the following:</p>
<pre><code>&gt;&gt;&gt; bin(int("3"))
'0b11'
&gt;&gt;&gt; bin(int("3"))[2:]
'11'
&gt;&gt;&gt; "{0:b}".format(int("3"))
'11'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>From the <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow noreferrer">documentation</a>:</p>
<blockquote>
<p>if base is given, then x must be a string, bytes, or bytearray
  instance representing an integer literal in radix base</p>
</blockquote>
<p>So you need to supply a string representing an integer in base 2 with this call, i.e. a string with only <code>"0"</code> or <code>"1"</code>.</p>
