<pre><code>List1 = [{"Value": "Value1", "Start": 7.11, "End": 8},
         {"Value": "Value2", "Start": 16.45, "End": 20}]

List2 = [{"From":7.11, "To": 8, "Result": 0},
         {"From":16.45, "To": 20 "Result": 1}
        ]
</code></pre>
<p>I need to produce a list by correlating these lists. So result will be</p>
<pre><code>Result = [{"Value": "Value1", "Start": 7.11, "End": 8, Result: 0},

<div class="post-text" itemprop="text">
<p>You can use a nested dictionary comprehension:</p>
<pre><code>List1 = [{"Value": "Value1", "Start": 7.11, "End": 8},
 {"Value": "Value2", "Start": 16.45, "End": 20}]

List2 = [{"From":7.11, "To": 8, "Result": 0},
 {"From":16.45, "To": 20, "Result": 1}
]

new_list = [{**a, **{'Result':b['Result']}} for a, b in zip(List1, List2)]
</code></pre>
<p>Output:</p>
<pre><code>[{'Value': 'Value1', 'Start': 7.11, 'End': 8, 'Result': 0}, {'Value': 'Value2', 'Start': 16.45, 'End': 20, 'Result': 1}]
</code></pre>
<p>Since, dictionary unpacking (<code>**</code>) is a feature in Python3 only, you can use <code>dict.items</code> in Python2:</p>
<pre><code>new_list = [dict(a.items()+[('Result', b['Result'])]) for a, b in zip(List1, List2)]
</code></pre>
<p>Output:</p>
<pre><code>[{'Start': 7.11, 'End': 8, 'Result': 0, 'Value': 'Value1'}, {'Start': 16.45, 'End': 20, 'Result': 1, 'Value': 'Value2'}]
</code></pre>
