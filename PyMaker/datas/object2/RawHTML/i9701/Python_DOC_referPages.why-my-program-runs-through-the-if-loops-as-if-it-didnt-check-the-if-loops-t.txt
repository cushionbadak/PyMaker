<p>The reason why this is failing is that you compared the modulo of <code>number</code> instead of <code>i</code>. <code>number=10</code> so <code>number%5==0</code> is always <code>True</code>. See also the comment of @JohnColeman.</p>
<p>Change it to <code>i % ... == 0</code>:</p>
<pre><code>def solution(number=10):
    i = 0
    c = 0
    while i &lt; number:
        if i % 3 == 0:
            c += i
        elif i % 5 == 0:
            c += i
        print('i={}, c={}, number={}, i%3={}, i%5={}'.format(i, c, number, i%3, i%5))
        i += 1

    return c

solution()
</code></pre>
<p>which prints:</p>
