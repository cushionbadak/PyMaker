<div class="post-text" itemprop="text">
<p>In fact <code>list</code> does not have keys, <code>list</code> has indices, and <code>dictionary</code> has keys. In your case, you have a list of dictionaries, and what you need is to remove some keys (2 exactly: description and tournament_url) form each item (dictionary) of your list:</p>
<pre><code>for item in my_list:  # my_list if the list that you have in your question
    del item['description']
    del item['tournament_url']
</code></pre>
<p>To retrieve an item from your above list with some criteria, you can do:</p>
<pre><code>[item for item in my_list if your_condition_here]
</code></pre>
<p><strong>Example:</strong></p>
<pre><code>&gt;&gt;&gt; [item for item in my_list if item['itemdef'] == 10541]
[{'leagueid': 1212, 'itemdef': 10541, 'name': 'Dota 2 Just For Fun'}]
</code></pre>
<p><strong>Edit:</strong></p>
<p>To filter <code>my_list</code> items to retrieve only some keys, you can do:</p>
<pre><code>keys_to_keep = ['itemdef', 'name']

res = [{ key: item[key] for key in keys_to_keep } for item in my_list]
print(res)
# Output: [{'itemdef': 10541, 'name': 'Dota 2 Just For Fun'}, {'itemdef': 10742, 'name': 'joinDOTA League Season 3'}]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>For the first question:</p>
<pre><code>for item in table:
    item.pop(tournament_url)
</code></pre>
<p>For the second question:</p>
<pre><code>[item for item in table if item[leagueid] == 1212]
</code></pre>

<div class="post-text" itemprop="text">
<p>So what you have there is a list of dictionaries.  To remove the <code>tournament_url</code> key from the dictionaries, we'll use a dictionary comprehension</p>
<pre><code>my_list = [{k:v for k, v in d.items() if k != 'tournament_url'} for d in my_list]
</code></pre>
<p>Read more about comprehensions in the <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">official python documentation</a></p>
