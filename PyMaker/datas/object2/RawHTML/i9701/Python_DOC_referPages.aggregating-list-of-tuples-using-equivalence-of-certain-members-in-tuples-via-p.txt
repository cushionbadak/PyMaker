<p>I have a list of tuples such as:</p>
<pre><code>[(a, 0), (a, 2), (a, 1), (a, 1), (a, 1), (a, 1), (a, 0), (b, 0), (a, 0), (b, 0), (b, 0), (b, 0), (b, 0), (b, 0), (b, 4)]
</code></pre>
<p>I want to add together all the second elements of tuples where <em>the first elements are the same</em>. I.e., the above list would become:</p>
<pre><code>[(a,0+2+1+1+1+1+0+0),(b,0+0+0+0+0+0+4)] = [(a,6),(b,4)]
</code></pre>
<p>What is the fastest way to accomplish this in python?</p>
</div>
<div class="post-text" itemprop="text">
<p>The solution using <code>itertools.groupby(iterable, key=None)</code> and <code>sum(iterable[, start])</code> functions:</p>
<pre><code>l = [('a', 0), ('a', 2), ('a', 1), ('a', 1), ('a', 1), ('a', 1), ('a', 0), ('b', 0), ('a', 0), ('b', 0), ('b', 0), ('b', 0), ('b', 0), ('b', 0), ('b', 4)]
result = [(k,sum(i[1] for i in list(v))) for k,v in itertools.groupby(sorted(l), key=lambda x: x[0])]

print(result)
</code></pre>
<p>The output:</p>
<pre><code>[('a', 6), ('b', 4)]
</code></pre>
<p><a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer">https://docs.python.org/3/library/itertools.html#itertools.groupby</a></p>
