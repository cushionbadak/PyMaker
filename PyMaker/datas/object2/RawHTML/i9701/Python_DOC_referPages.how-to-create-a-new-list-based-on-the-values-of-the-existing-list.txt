<p>I am new to python.</p>
<p>Let's say I have a list like this : </p>
<pre><code>randomList = [2, 3, 5, 7, 11]
</code></pre>
<p>How to filter the values of above list such that the new resultant list has the values according to the condition mentioned.</p>
<p>For example : </p>
<p>Lets say I want to create a new list of all the numbers <code>n * 2</code> in randomList such that <code>n%2 != 0</code>. Therefore, the new list should look something like this : </p>
<pre><code>newList = [6, 10, 14, 22]
</code></pre>
<p>I can do this by using a <code>for</code> loop and using <code>append</code> command, but is there a better way to this in python?</p>
<p>p.s - I am a beginner.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could do that in one simple line.</p>
<pre><code>randomList = [2, 3, 5, 7, 11]

newList = [n*2 for n in randomList if n%2 != 10]
</code></pre>
<p>That's it.</p>
<p>Now the values in <code>newList</code> would be :</p>
<pre><code>newList = [6, 10, 14, 22]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>What you are looking for is <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">List Comprehensions</a></p>
<pre><code>&gt;&gt;&gt; [ n*2 for n in randomList if n % 2]
[6, 10, 14, 22]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>list comprehension</code> :</p>
<pre><code>newlist = [2*i for i in randomList if i % 2 != 0]
</code></pre>
