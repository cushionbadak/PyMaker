<div class="post-text" itemprop="text">
<p>In examples of using pickle to save data, I often encounter a second parameter where many people use <code>'wb'</code>, for instance: </p>
<p><code>pickle.dump(obj, open('save.p', 'wb'))</code>. </p>
<p>What does the <code>'wb'</code> parameter do? </p>
</div>
<div class="post-text" itemprop="text">
<p>It is not a pickle parameter, but a parameter for the open function. It means: open for writing and open in binary mode.</p>
<p>You can read more in <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer">the documentation.</a></p>
</div>
<div class="post-text" itemprop="text">
<p>'wb' means 'write binary' and is used for the file handle: <code>open('save.p', 'wb'</code>) which writes the pickeled data into a file.</p>
<p>The code you got is a short version of:</p>
<pre><code>handle = open('save.p', 'wb')
pickle.dump(obj, handle)
</code></pre>
</div>
