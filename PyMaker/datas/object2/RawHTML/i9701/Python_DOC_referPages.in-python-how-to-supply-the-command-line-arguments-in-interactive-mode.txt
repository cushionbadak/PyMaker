<div class="post-text" itemprop="text">
<p>I am new to Python. I want to run a software in interactive mode. In the manual it says the usage</p>
<p>python experiment.py --config config.yaml --out result/</p>
<p>The question is, how can I supply the command line arguments to experiment.py in interactive mode?</p>
</div>
<div class="post-text" itemprop="text">
<p>The commandline arguments that e.g. <code>optparse</code> and <code>argparse</code> uses are taken by default from <code>sys.argv</code> element 1 and up. You can always do:</p>
<pre><code>import sys
sys.argv[1:] = ['--config', 'config.yaml', '--out', 'result/']
</code></pre>
<p>Although e.g. in <code>argparse</code> you can provide the arguments explicitly to <code>.parse_args()</code> as well and then <a href="https://docs.python.org/3/library/argparse.html#parsing-arguments" rel="nofollow noreferrer">that method will not inspect <code>sys.argv</code></a></p>
</div>
<div class="post-text" itemprop="text">
<p>If I understood you correctly you need something like this:</p>
<pre><code>while True:
    query = raw_input("&gt; ")
    if query == "exit":
        break
    # do something useful
</code></pre>
</div>
<span class="comment-copy">Take a look at the <code>experiment.py</code> file and see how its <code>main</code> function calls the other functions defined in <code>experiment.py</code>, and that will show you how you can call them yourself.</span>
