<div class="post-text" itemprop="text">
<p>I have a URL as string : </p>
<pre><code>urlString = https://play.com/details?id=blamore
</code></pre>
<p>How to get the <code>id</code> parameter from the given string ?</p>
</div>
<div class="post-text" itemprop="text">
<p>The proper way to to do this in Python 3 is to use the <a href="https://docs.python.org/3/library/urllib.parse.html" rel="nofollow noreferrer"><code>urllib.parse</code></a> module.</p>
<pre><code>from urllib.parse import urlparse, parse_qs 

url = 'https://play.com/details?id=blamore'

t = urlparse(url)
print(t)

d = parse_qs(t.query)
print(d)
</code></pre>
<p><strong>output</strong></p>
<pre><code>ParseResult(scheme='https', netloc='play.com', path='/details', params='', query='id=blamore', fragment='')
{'id': ['blamore']}
</code></pre>
<p>The field names of the <code>ParseResult</code> should be self-explanatory, but please refer to <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.urlparse" rel="nofollow noreferrer">the docs</a> for details.</p>
<p>From the <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.parse_qs" rel="nofollow noreferrer"><code>urllib.parse.parse_qs</code> docs</a></p>
<blockquote>
<p>Parse a query string given as a string argument (data of type
  application/x-www-form-urlencoded). Data are returned as a dictionary.
  The dictionary keys are the unique query variable names and the values
  are lists of values for each name.</p>
</blockquote>
