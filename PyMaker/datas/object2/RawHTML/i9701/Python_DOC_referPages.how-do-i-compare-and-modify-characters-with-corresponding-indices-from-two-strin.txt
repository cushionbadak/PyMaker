<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a> with <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate()</code></a> like this:</p>
<pre><code>s1 = 'abcdef'
s2 = 'axcyez'

l1 = list(s1)
l2 = list(s2)

for i, item in enumerate(zip(s1, s2)):
    if item[0] == item[1]:
        l1[i] = l2[i] = item[0].upper()

s1 = ''.join(l1)
s2 = ''.join(l2)
</code></pre>
<p><code>zip()</code> is used to form tuples of elements from <code>s1</code> and <code>s2</code></p>
<p><code>enumerate()</code> is used to track indexes of each string characters</p>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; s1
'AbCdEf'
&gt;&gt;&gt; s2
'AxCyEz'
</code></pre>
