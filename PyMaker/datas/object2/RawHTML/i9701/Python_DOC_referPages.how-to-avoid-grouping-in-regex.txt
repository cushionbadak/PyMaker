<p>Hello I am trying to parse a string like '2.2.2.2-22'
while using python re.findall function I am getting two groups:
I don't understand what is wrong with the RE:</p>
<pre><code>re.findall(r"([\d.]+(-\d+)?)",  "2.2.2.2-22")
</code></pre>
<p>This gives me result as: [('2.2.2.2-22', '-22')]
I do not want the second group. How can I fix this? </p>
</div>
<div class="post-text" itemprop="text">
<p>To avoid the grouping you can use <code>?:</code> </p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; re.findall(r"([\d.]+(?:-\d+)?)",  "2.2.2.2-22")
['2.2.2.2-22']
</code></pre>
<p>It will force that group to exists, but it will not catch and return the value.</p>
