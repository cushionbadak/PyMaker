<div class="post-text" itemprop="text">
<p>This error message suggests that the key <code>primer1</code> is a list (or contains a list).  </p>
<p>What you're attempting will not be possible, because dict keys must be <a href="https://docs.python.org/3/glossary.html" rel="nofollow noreferrer">hashable</a> in Python.  Instead, you may use a hashable sequence type for the key <code>primer1</code>, such as a <a href="https://docs.python.org/3/library/stdtypes.html#tuples" rel="nofollow noreferrer">tuple</a>.  </p>
