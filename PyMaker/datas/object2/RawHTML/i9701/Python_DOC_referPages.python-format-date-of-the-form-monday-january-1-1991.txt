<div class="post-text" itemprop="text">
<p>I'm creating a web scraper and I'm running into an issue where the date the website gives me is of the form "Monday, January 1, 1991"</p>
<p>What's the best way to format this into a "MM-DD-YYYY" format?  Should I split on the comma, pull out the month and convert it to a number, and then put the numbers together? Or is there some quicker way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <code>datetime</code> module, using <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer"><code>strptime</code> to parse to a <code>datetime</code> object, then <code>strftime</code> to format as you need it</a>:</p>
<pre><code>from datetime import datetime

date = datetime.strptime("Monday, January 1, 1991", "%A, %B %d, %Y")
print(date.strftime("%m-%d-%Y"))
</code></pre>
<p>which outputs:</p>
<pre><code>01-01-1991
</code></pre>
<p>For the record, any time you're considering rolling your own parser, the answer is almost always "Don't". Rolling your own parser is error-prone; if at all possible, look for an existing parser.</p>
</div>
