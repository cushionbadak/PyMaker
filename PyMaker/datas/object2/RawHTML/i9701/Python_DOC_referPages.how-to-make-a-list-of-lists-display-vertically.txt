<p>Solutions are not so elegant, too much code...</p>
<pre><code>L = [[1],[2,3],[3,4,5]]
max_length = max(map(len, L))  # finding max length
output = zip(*map(lambda x: x + [' '] * (max_length - len(x)), L))  # filling every sublist with ' ' to max_length and then zipping it
for i in output: print(*i)  # printing whole result
</code></pre>
<p>Output:</p>
<pre><code>1 2 3
  3 4
    5
</code></pre>
<p>So 3rd line is not that obvious, i will break it down</p>
<pre><code>&gt;&gt;&gt; list(map(lambda x: x + [' '] * (max_length - len(x)), L))
[[1, ' ', ' '], [2, 3, ' '], [3, 4, 5]]
&gt;&gt;&gt; list(zip(*map(lambda x: x + [' '] * (max_length - len(x)), L)))
[(1, 2, 3), (' ', 3, 4), (' ', ' ', 5)]
</code></pre>
<p><strong>UPDATE</strong></p>
<p>To lengthen the spaces you need to provide keyword argument <code>sep</code> to <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>print</code></a> function: <code>for i in output: print(*i, sep= '  ')</code></p>

      print (lst[0], end='')
    if (i == 1 and len(lst)&gt;1):
      print (lst[1], end='')
    if (i == 2 and len(lst)&gt;2):
      print (lst[2], end='')
  print ('\n')
</code></pre>
