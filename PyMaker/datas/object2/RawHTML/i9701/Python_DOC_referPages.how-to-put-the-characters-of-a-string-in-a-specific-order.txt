<div class="post-text" itemprop="text">
<p>I have a <em>list of positions</em>, like this:</p>
<pre><code>a=[[1,0],[0, 2],[0, 4],[1,1],[0, 1],[0, 0],[1,3],[1,4],[1,2]]
</code></pre>
<p>I also have a <em>string</em>, for example this one:</p>
<pre><code>string = "HELLOWORL"
</code></pre>
<p>And I want to put <em>each character of the string in the order 
of the list of positions</em>, in a matrix (where position[0] is the line and position[1] is the column),like this:</p>
<pre><code>"string= WOELE
         HLLOR"
</code></pre>
<p>How do i do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use these functions to achieve that:</p>
<p><a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip()</code></a> to form tuples of elements in <code>s</code> and <code>a</code></p>
<p><a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter()</code></a> to separate characters of line 1 and line 2</p>
<p><a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer"><code>sorted()</code></a> to sort the characters of each line</p>
<pre><code>a = [[1, 0], [0, 2], [0, 4], [1, 1], [0, 1], [0, 0], [1, 3], [1, 4], [1, 2]]
s = "HELLOWORL"

first_line = sorted(filter(lambda i: i[0][0] == 0, zip(a, s)), key=lambda i: i[0][1])
second_line = sorted(filter(lambda i: i[0][0] == 1, zip(a, s)), key=lambda i: i[0][1])

word1 = ''.join(item[1] for item in first_line)
word2 = ''.join(item[1] for item in second_line)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; word1
'WOEL'
&gt;&gt;&gt; word2
'HLLOR'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>There is also the option of using a couple of list comprenhension and a zip function.</p>
<pre><code>a=[[1,0],[0, 2],[0, 4],[1,1],[0, 1],[0, 0],[1,3],[1,4],[1,2]]
string = "HELLOWORL"
# zip(*a) extracts the first and the last element of each elemnt of a
# in a different list
rows, cols = zip(*a)
# Find the maximum possible value and sum 1in order to use then xrange
maxrow=max(rows) + 1
maxcol=max(cols) + 1
# Create an empty list with placeholders fot he characteres
b=[["" for _ in xrange(maxcol)] for _ in xrange(maxrow)]
for i in xrange(len(string)):
    letter=string[i]
    row,col = a[i]
    # Change the placeholder for the correct character
    b[row][col]=letter
# Paste everything pith join
result = "\n".join(["".join(i) for i in b])
</code></pre>
<p>Wich results in result ="WOEL\nHLLOR"</p>
</div>
<div class="post-text" itemprop="text">
<p>It can also be done like this:</p>
<pre><code>string = "HELLOWORL"

a = [[1,0],[0, 2],[0, 4],[1,1],[0, 1],[0, 0],[1,3],[1,4],[1,2]]

def value(v): return v[1]

import itertools
s = sorted(zip(string, a), key=value)
s = ((l, r) for (l, (r, c)) in s)
s = itertools.groupby(s, key=value)
print '\n'.join(''.join(c for (c, r) in g[1]) for g in s)
</code></pre>
<p>This <code>zip</code>s the string with the list which is then <code>sorted</code> by the list element then the column is removed and the items are <code>groupby</code>ed their row and the each of the letters in each groups is <code>join</code>ed after removing their row and then the groups are <code>join</code>ed as lines.</p>
<p>which prints:</p>
<pre><code>WOEL
HLLOR
</code></pre>
