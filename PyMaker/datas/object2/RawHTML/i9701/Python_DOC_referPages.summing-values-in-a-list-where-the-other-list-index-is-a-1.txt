<p>I have two lists:</p>
<pre><code>list_1 = [0, 1, 0, 0, 0, 0, 1]
list_2 = [34, 54, 23, 54, 656 34, 100]
</code></pre>
<p>I want to sum only the values in <code>list_2</code> where the index in <code>list_2</code> matches the value of 1 for the same index in <code>list_1</code>.</p>
<p>Example here would be <code>54 + 100 = 154</code></p>
<p>How can I do this in Python in a Pythonic way?</p>
</div>
<div class="post-text" itemprop="text">
<p>Just use <a href="https://docs.python.org/3/library/itertools.html#itertools.compress" rel="nofollow noreferrer"><code>compress</code></a> from <code>itertools</code> and feed it to <code>sum</code>; it's built for this purpose:</p>
<pre><code>from itertools import compress 
r = sum(compress(list_2, list_1))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Zip the lists, then sum only values where the first element is <code>1</code>:</p>
<pre><code>s = sum(b for a, b in zip(list_1, list_2) if a)
</code></pre>
<p>Another zip approach would be an element-wise multiplication between the lists (thus eliminating elements whose parallel is <code>0</code>):</p>
<pre><code>s = sum(a * b for a, b in zip(list_1, list_2))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Probably less pythonic, but also works for lists of similar size:</p>
<pre><code>&gt;&gt;&gt; l1 = [0,1,0,0,1]
&gt;&gt;&gt; l2 = [10,23,45,1,23]
&gt;&gt;&gt; map(lambda x,y: x*y , l1,l2)
[0, 23, 0, 0, 23]
&gt;&gt;&gt; sum(map(lambda x,y: x*y , l1,l2))
46
&gt;&gt;&gt; 
</code></pre>
