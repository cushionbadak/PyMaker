<p>I'd find all table rows and use <a href="https://docs.python.org/3/library/functions.html#max" rel="nofollow noreferrer"><code>max()</code> function</a> with a custom <code>key</code> function to determine the "latest" record. The custom function would find the inner cell with the date inside and convert it to the <code>datetime.date()</code> in order to compare dates with one another:</p>
<pre><code>import re
from datetime import date

from bs4 import BeautifulSoup


data = """your HTML here"""    

soup = BeautifulSoup(data, "html.parser")

# get the table header label
label = soup.find("span", text=re.compile(r"^AVAILABLE QUOTA FY"))

# extract the year
year = int(re.search(r"\d{4}", label.get_text()).group(0))

table = label.find_next("table")
rows = table.find_all("tr")[1:]  # skipping header row


def get_date(row):
    last_column_cell = row.find_all("td")[-1]

    # extract month and day
    month, day = last_column_cell.get_text().split("/")
    return date(year=year, month=int(month), day=int(day))


# filter the row with the latest date
row = max(rows, key=get_date)

# print the value of every cell in the row
print([td.get_text(strip=True) for td in row.find_all("td")])
</code></pre>
<p>The code is probably missing error-handling and should be further improved, but it works on the provided sample data, prints:</p>
<pre><code>['1724', 'GOM CODGOM HADDGOM BBGREYSOLEDABSGOM YT', '2328445300785031011995', 'Package$9,000', '5/20']
</code></pre>
<hr/>
<p>Note that this would only find the row with the maximum date. If you want to take it further - find all rows with this date:</p>
<pre><code># get the value of the latest cell
latest_date_string = row.find_all("td")[-1].get_text().strip()

# print the value of every cell in the row
print([[td.get_text(strip=True) for td in date_string.find_parent("tr").find_all("td")]
       for date_string in table.find_all(text=latest_date_string)])
