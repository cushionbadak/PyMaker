<p>I have a list that looks like this </p>
<pre><code>list = [blabla\nblabla\nblabla, wowwow\nwowow\n, etc]
</code></pre>
<p>I want to separate the list values based on the delimiter "\n" and create new lists. Like this:</p>
<pre><code>list1 = [blabla, wowwow, etc] list2 = [blabla, wowow, etc]
</code></pre>
<p>Does anyone know how?</p>
<p>Thanks in advance!</p>

<div class="post-text" itemprop="text">
<p>You can accomplish this task with using izip_longest (when the number of elements after the split is different) and list comprehension, like:</p>
<pre><code>from itertools import izip_longest
n = ['blabla\nblabla\nblabla', 'wowwow\nwowow\n']
res = [v for v in izip_longest(*[k.split('\n') for k in n])]
</code></pre>
<p>For more info: <a href="https://docs.python.org/2/library/itertools.html#itertools.izip_longest" rel="nofollow noreferrer">https://docs.python.org/2/library/itertools.html#itertools.izip_longest</a></p>
