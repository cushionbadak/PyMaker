<div class="post-text" itemprop="text">
<p>For each <code>item</code> in <code>companies</code> what you want is <code>item[0][0]</code>. using a <a href="https://docs.python.org/3/reference/expressions.html#generator-expressions" rel="nofollow noreferrer"><code>generator expression</code></a> you can do:</p>
<pre><code>res = (item[0][0] for item in companies)
</code></pre>
<p>To get the final result, apply <code>tuple()</code> like this:</p>
<pre><code>tuple(res)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; companies = [(('Paramount Pictures Corporation (Canada)',),), (('20th Century Fox Home Entertainment',),), (('Shanghai Epic Music Entertainment',),)]
&gt;&gt;&gt;

&gt;&gt;&gt; res

&gt;&gt;&gt; tuple(res)

<p>Once you defined the nested get <code>[0]</code> as such:</p>
<pre><code>first = lambda x: first(x[0]) if type(x)==tuple else x
</code></pre>
<p><em>It will work no matter what is the depth of the tuple nesting.</em></p>
<p>All we have left is to apply it to the <code>companies</code> list</p>
