<div class="post-text" itemprop="text">
<p>whats the best way to find total number of ip addresses in a cidr block using python</p>
<p>like if  input is  10.10.0.0/24 </p>
<p>then the output should be 256 </p>
<p>is there a way to do that with any modules in python like ipaddress?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>ipaddress.ip_network()</code> function from <code>ipaddress</code> module:</p>
<pre><code>import ipaddress

net = ipaddress.ip_network('10.10.0.0/24')
print(net.num_addresses)
</code></pre>
<p>The output:</p>
<pre><code>256
</code></pre>
<p><code>&lt;'ipaddress.IPv4Network'&gt;.num_addresses</code> property points to a total number of addresses in the network</p>
<p><a href="https://docs.python.org/3/library/ipaddress.html#ipaddress.ip_network" rel="nofollow noreferrer">https://docs.python.org/3/library/ipaddress.html#ipaddress.ip_network</a></p>
