<div class="post-text" itemprop="text">
<p>In Python, is it possible to add custom property/metadata to a file?
For example, I need to add "FileInfo" as a new property of the file.
I need a method that works on various file formats</p>
</div>
<div class="post-text" itemprop="text">
<p>The easy way to do this is to simply add your new attribute to the file object instance. Eg,</p>
<pre><code>with open('qdata') as f:
    f.fileinfo = {'description': 'this file contains stuff...'}
    print(f.fileinfo)
</code></pre>
<p><strong>output</strong></p>
<pre><code>{'description': 'this file contains stuff...'}
</code></pre>
<p>Alternatively, create your own file object by deriving from one of the classes defined in the <a href="https://docs.python.org/3/library/io.html" rel="nofollow noreferrer"><code>io</code></a> module.</p>
</div>
