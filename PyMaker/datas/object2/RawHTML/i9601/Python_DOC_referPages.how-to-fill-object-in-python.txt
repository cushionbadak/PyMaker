<div class="post-text" itemprop="text">
<p><code>'tuple' object does not support item assignment.</code>
and just like what Moses Koledoye says, tuple object is immutable.</p>
<p>If you want to add or remove to <code>FILES</code>,
try:</p>
<pre><code>FILES = [('hello3.txt', True)]
FILES.append(('new.txt', False))
</code></pre>
<p>If you want to change the name of the file or the value of T/F, use <code>list</code> instead of <code>tuple</code>.</p>
<pre><code>FILES = [['hello3.txt', True]]
FILES.append(['new.txt', False])
FILES[0][1] = False
</code></pre>
<p>But I think these codes are so ugly. I prefer to create a class instead of using the list.</p>
<pre><code>class File:
    __slots__ = 'name', 'flag'
    def __init__(self, name, flag):
        self.name = name
        self.flag = flag
FILES = [File('hello3.txt', True)]
FILES.append(File('new.txt', False))
FILES[0].flag = False
</code></pre>
<p>There's also a simple way if your file only have 2 attributes (name and flag).</p>
<pre><code>FILES = {'hello3.txt': True}
FILES['new.txt'] = False
</code></pre>
