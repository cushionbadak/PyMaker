<p>I want to insert into a path the content of a txt file.</p>
<p>Example:</p>
<p>I have a txt file  in <code>./path/date.txt</code> with the content</p>
<pre><code>08122016
</code></pre>
<p>How do I put the content (<code>08122016</code>) on the path of a second file?</p>
<p>Something like this:</p>
<pre><code>s = open('/erp/date/**date.txt content**').read()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>#open the date file
f = open("./path/date.txt", 'r')
#read the content
content=f.read()
#close file
f.close()

#insert date in path
s=open("/erp/date/"+str(content)).read()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/os.path.html#os.path.join" rel="nofollow noreferrer"><code>os.path.join</code></a>: </p>
<pre><code>import os

with open(r'./path/date.txt', 'rt') as input_file:
    data = input_file.read()
with open(os.path.join('/erp/date', data), 'rt') as input_file2:
    data2 = input_file2.read()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can insert strings into other strings like that (python 2.7.12):</p>
<p><code>path = 'home/user/path/%s' % content</code></p>
<p>The %s in the the string will be replaced by the content variable.</p>
