<p><a href="https://pypi.python.org/pypi/dropbox" rel="nofollow noreferrer"><code>dropbox</code> requires Python 3.5</a>, as it requires the <a href="https://docs.python.org/3/library/typing.html" rel="nofollow noreferrer">typing</a> module, which wasn't introduced until 3.5. You're trying to use it with Python 3.4.</p>
