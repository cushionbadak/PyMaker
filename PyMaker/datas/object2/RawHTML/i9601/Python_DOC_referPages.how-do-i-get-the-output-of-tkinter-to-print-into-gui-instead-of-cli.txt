<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/subprocess.html" rel="nofollow noreferrer">subprocess</a> instead of <a href="https://docs.python.org/3/library/os.html#os.system" rel="nofollow noreferrer">os.system</a>. There are many functions to work with external command.</p>
<p>I use <code>subprocess.check_output()</code> to get result of executed command. Command has to be as list <code>["ping", entry.get(), "-c", "2"]</code>. Command can be single string if you use <code>shell=True</code>.</p>
<pre><code>import tkinter as tk
import subprocess

def ping():
    cmd = ["ping", entry.get(), "-c", "2"]
    output = subprocess.check_output(cmd)
    #output = subprocess.check_output("ping {} -c 2".format(entry.get()), shell=True)

    print('&gt;', output)
    # put result in label
    result['text'] = output.decode('utf-8')

my_gui = tk.Tk()
entry = tk.StringVar()

my_gui.geometry('300x300')
my_gui.title("Get output inside GUI") 

tk.Label(my_gui, text="Enter target IP or host as required.").pack() 
tk.Entry(my_gui, textvariable=entry).pack()
tk.Button(my_gui,text="Ping Test", command=ping).pack() 

# label for ping result
result = tk.Label(my_gui)
result.pack()

my_gui.mainloop()
</code></pre>
<p>BTW: because <code>ping</code> takes some time so <code>tkinter</code> will freeze for this time. If you need non-freezing version you will need other functions in <code>subprocess</code> or <code>threading</code> module.</p>
