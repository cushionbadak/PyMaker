<p>I do subtraction of two redis sets like as:</p>
<pre><code>items = r.sdiff("apps", "apps_2")
</code></pre>
<p>And after try to create new set:</p>
<pre><code>r.sadd("apps_diff", items)
</code></pre>
<p>But get error.
How can I create new set?</p>
</div>
<div class="post-text" itemprop="text">
<p>You need to use the <code>*</code> operator to unpack the list of arguments for the function call.</p>
<pre><code> items = r.sdiff("apps", "apps_2")

 r.sadd("apps_diff", *items)
</code></pre>
<p>Reference: <a href="https://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists</a></p>
