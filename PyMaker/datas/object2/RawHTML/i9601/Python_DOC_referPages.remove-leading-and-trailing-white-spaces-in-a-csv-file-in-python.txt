<p>Define: </p>
<p><code>str = " a  , b,c, hello there !   ,   my name is +++ ,  g "</code></p>
<p>How do I remove the leading and trailing white spaces so that I output: </p>
<p><code>"a,b,c,hello there !,my name is +++,g"</code></p>
<p>i.e. the output is such that there is no leading or trailing white space between values in a comma separated value string. </p>
<p>I started reading about Regular Expressions? Would this be an appropriate situation to use that? How would I go about completing the task?</p>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>split()</code></a>, <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>strip()</code></a> and <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>join()</code></a> like this:</p>
<pre><code>','.join([w.strip() for w in my_string.split(',')])
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; my_string = '[ a  , b,c, hello there !   ,   my name is +++ ,  g ]'
&gt;&gt;&gt; ','.join([w.strip() for w in my_string.split(',')])
'[ a,b,c,hello there !,my name is +++,g ]'
</code></pre>
<p><strong>Explanation:</strong></p>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>split()</code></a> is used to split <code>my_string</code> by the separator <code>,</code> and the result is the following list:</p>
<pre><code>&gt;&gt;&gt; my_string.split(',')
['[ a  ', ' b', 'c', ' hello there !   ', '   my name is +++ ', '  g ]']
</code></pre>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>strip()</code></a> is used to remove spaces around each word of the previous list:</p>
<pre><code>&gt;&gt;&gt; [w.strip() for w in my_string.split(',')]
['[ a', 'b', 'c', 'hello there !', 'my name is +++', 'g ]']
</code></pre>
<p>The above line is called a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a></p>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>join()</code></a> is used to form the last result by joining the words of the above list.</p>
</div>

<div class="post-text" itemprop="text">
<p>You may use regex expression <code>'\s*,\s*'</code> to replace the matching content with <code>,</code>. For example:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; my_str = "[ a  , b,c, hello there !   ,   my name is +++ ,  g ]"

&gt;&gt;&gt; re.sub('\s*,\s*', ',', my_str)
'[ a,b,c,hello there !,my name is +++,g ]'
</code></pre>
