<p>I want to print multiple functions on one line of code in Python. I am currently using the method in which you put a comma after the print statement  </p>
<pre><code>print("Hello, my name is " + introName + " and I have a " +outroName+.",) 
</code></pre>
<p>but it prints like this </p>
<pre><code>('Hello, my name is Arthur\n and I have a secret.',)
</code></pre>
<p>How do I get rid of the <code>\n</code> <code>()</code> , and <code>'</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>Seems that the value <code>introName</code> contains <code>\n</code> at the end, you can use <code>strip()</code> to remove that.</p>
<pre><code>introName.strip('\n')
</code></pre>
<p>Your line should be:</p>
<pre><code>print("Hello, my name is " + introName.strip('\n') + " and I have a " + outroName.strip('\n')) 
</code></pre>
<p>The better is using <a href="https://docs.python.org/3/library/functions.html#format" rel="nofollow noreferrer"><code>format()</code></a> like this:</p>
<pre><code>s = "Hello, my name is {} and I have a {}".format(introName.strip('\n'), outroName.strip('\n'))
print(s)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; introName = 'Arthur\n'
&gt;&gt;&gt; outroName = 'secret'
&gt;&gt;&gt;
&gt;&gt;&gt; s = "Hello, my name is {} and I have a {}".format(introName.strip('\n'), outroName.strip('\n'))
&gt;&gt;&gt; s
'Hello, my name is Arthur and I have a secret'
</code></pre>
</div>
<span class="comment-copy">It's printing like that because you've put your string into a tuple with a single item, which can be defined like <code>(item1,)</code>. You simply have to remove the comma at the end, so it looks like <code>print("Hello, my name is " + introName + " and I have a " +outroName+.")</code>.</span>
