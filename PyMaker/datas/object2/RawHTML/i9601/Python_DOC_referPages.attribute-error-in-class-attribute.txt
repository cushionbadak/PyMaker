<div class="post-text" itemprop="text">
<p>You're misspelling <code>__init__</code> with <code>__int__</code>:</p>
<pre><code>class cl :
    def __int__(self):
        self.a = 0
</code></pre>
<p>Should be</p>
<pre><code>class cl :
    def __init__(self):
        self.a = 0
</code></pre>
<p>Note that encompassing a main method inside of the <code>test</code> class is really an anti-pattern in Python; the form of</p>
<pre><code>def main():
    ...

if __name__ == "__main__":
    main()
</code></pre>
<p>is preferred and is more Pythonic.</p>
<p>For some reference on the <code>if __name__ == "__main__"</code> pattern, see:</p>
<ul>
<li><a href="https://stackoverflow.com/a/419185/1730261">https://stackoverflow.com/a/419185/1730261</a></li>
<li><a href="https://docs.python.org/3/library/__main__.html" rel="nofollow noreferrer">https://docs.python.org/3/library/<strong>main</strong>.html</a></li>
</ul>
