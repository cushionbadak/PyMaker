<div class="post-text" itemprop="text">
<p>Your misunderstanding how Python <code>import</code> statements work. When importing a module in Python, only the module directly imported is include in the local symbol table.</p>
<p>Thus, if you have not directly imported a name, it cannot be used as a standalone identifier in the current namespace. If you want this behavior, directly import the <code>QApplication</code> name from the <code>QtWidgets</code> namespace:</p>
<pre><code>from PyQt5.QtWidgets import QApplication
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Import modules like this:</p>
<pre><code>import sys
from PyQt5.QtWidgets import *

app = QApplication(sys.argv)
</code></pre>
</div>
