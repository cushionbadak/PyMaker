<p>I'll interpret this as "select the sublists that have any string in them that starts with <code>da_</code>":</p>
<pre><code>new_list = [x for x in list if any(s.startswith('da_') for s in x)]
</code></pre>
<p>Of course if there is always just a single element in the sublists, it's easier:</p>
<pre><code>new_list = [x for x in list if x[0].startswith('da_')]
</code></pre>
