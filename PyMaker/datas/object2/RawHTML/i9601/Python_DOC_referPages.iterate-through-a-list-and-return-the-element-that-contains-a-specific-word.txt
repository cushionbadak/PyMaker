<p>i have this code below:</p>
<pre><code>wordSearch = "ghi"
lst = ['c:/test/abc.bat','c:/test/test1/def.bat','c:/123/ghi.exe','c:/test/testing/aa/jkl.bat']
</code></pre>
<p>by searching "ghi" I would like it to return "c:/123/ghi.exe". So, ultimately i want it to iterate through the list and return an element that contains specific text.</p>
<p>Many thanks.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> like this:</p>
<pre><code>[item for item in lst if wordSearch in item]
</code></pre>
<p>The above line will return any item that contains <code>wordSearch</code>.</p>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; wordSearch = "ghi"
&gt;&gt;&gt; lst = ['c:/test/abc.bat','c:/test/test1/def.bat','c:/123/ghi.exe','c:/test/testing/aa/jkl.bat']
&gt;&gt;&gt; 
&gt;&gt;&gt; [item for item in lst if wordSearch in item]
['c:/123/ghi.exe']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>filter</code>:</p>
<pre><code> print filter(lambda x: 'ghi' in x, lst)
</code></pre>
<p>For Python3, above line will return a <code>filter</code> object. So, you need to use the factory function <code>list()</code> over it.</p>
<pre><code>print(list(filter(lambda x: 'ghi' in x, lst)))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you want a function that only return the first occurrence:</p>
<pre><code>wordSearch = "ghi"
lst = ['c:/test/abc.bat','c:/test/test1/def.bat','c:/123/ghi.exe','c:/test/testing/aa/jkl.bat']

def find_word(wordSearch, lst):
    for path in lst:
        if wordSearch in path:
            return path

print(find_word(wordSearch, lst))
</code></pre>
