<p>You cannot leave something below a ':' blank.</p>

<pre><code>elif choice == "2":

<div class="post-text" itemprop="text">
<p>You can not write something like:</p>
<pre><code>elif choice == "2":

elif choice == "3":
</code></pre>
<p>This raises <code>IndentationError</code> exception, so if you want to implement that later you can use <a href="https://docs.python.org/2/tutorial/controlflow.html#pass-statements" rel="nofollow noreferrer"><code>pass statement</code></a> like below:</p>
<pre><code>elif choice == "2":
    pass

elif choice == "3":
    pass
</code></pre>
<p>From <a href="https://docs.python.org/2/tutorial/controlflow.html#pass-statements" rel="nofollow noreferrer"><code>pass</code></a> documentation:</p>
<blockquote>
<p>The pass statement does nothing. It can be used when a statement is
  required syntactically but the program requires no action.</p>
</blockquote>
</div>
