<div class="post-text" itemprop="text">
<p>You are importing <code>webdriver</code> from the <code>selenium</code> library so the name <code>selenium</code> is not bound to anything.</p>
<pre><code>import selenium.webdriver

driver = selenium.webdriver.Chrome('localhost', 4444, '*firefox', '...')
</code></pre>
<p>or</p>
<pre><code>from selenium import webdriver

driver = webdriver.Chrome('localhost', 4444, '*firefox', '...')
</code></pre>
<p>An <code>import</code> directive introduces a name binding for whatever is to the right of the <code>import</code> keyword.  In the first example, two names are bound -- <code>selenium</code> and <code>selenium.webdriver</code>.  In the second example only a single name is bound.  See the <a href="https://docs.python.org/3/reference/simple_stmts.html#import" rel="nofollow noreferrer">python docs on import</a> for more details.</p>
