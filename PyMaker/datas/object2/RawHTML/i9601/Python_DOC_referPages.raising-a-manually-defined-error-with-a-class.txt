<div class="post-text" itemprop="text">
<p>You just <code>raise</code> it, supplying an instance to <code>raise</code> with the appropriate required argument (<code>num</code> as defined in your <code>__init__</code>):</p>
<pre><code>raise MissingnumError(1)
</code></pre>
<p>If no handler is specified Python will simply <code>__repr__</code> it and return the message as specified:</p>
<pre><code>MissingnumError: Animal #1 not found
</code></pre>
<p>Two additional notes:</p>
<ul>
<li>Don't use <code>type</code> as an argument name. <code>type</code> is already used by Python for another purpose and you mask it inside the method body.</li>
<li><code>CheckError</code> should inherit from <code>Exception</code>, this is required by any object you supply to <code>raise</code>. </li>
</ul>
