<p>I need to turn this file content into a dictionary, so that every key in the dict is a name of a movie and every value is the name of the actors that plays in it inside a set.</p>
<p>Example of file content:</p>
<pre class="lang-none prettyprint-override"><code>Brad Pitt, Sleepers, Troy, Meet Joe Black, Oceans Eleven, Seven, Mr &amp; Mrs Smith
Tom Hanks, You have got mail, Apollo 13, Sleepless in Seattle, Catch Me If You Can
Meg Ryan, You have got mail, Sleepless in Seattle
Diane Kruger, Troy, National Treasure
Dustin Hoffman, Sleepers, The Lost City
Anthony Hopkins, Hannibal, The Edge, Meet Joe Black, Proof
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try the following:</p>
<pre><code>res_dict = {}

with open('my_file.txt', 'r') as f:
    for line in f:
        my_list = [item.strip() for item in line.split(',')]
        res_dict[my_list[0]] = my_list[1:]  # To make it a set, use: set(my_list[1:])
</code></pre>
<p><strong>Explanation:</strong></p>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>split()</code></a> is used to split each line to form a list using <code>,</code> separator</p>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>strip()</code></a> is used to remove spaces around each element of the previous list</p>
<p>When you use <a href="https://docs.python.org/3/reference/compound_stmts.html#with" rel="nofollow noreferrer"><code>with statement</code></a>, you do not need to close your file explicitly.</p>
<p><code>[item.strip() for item in line.split(',')]</code> is called a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer"><code>list comprehension</code></a>.</p>
