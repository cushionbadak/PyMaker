<p>Try this</p>
<pre><code>import datetime
now = datetime.datetime.now()
print now.strftime("%d-%B")
</code></pre>

<div class="post-text" itemprop="text">
<pre><code>from datetime import datetime
dt = datetime.strptime("09/12/16", "%d/%m/%y")
dt.strftime("%d-%B")
</code></pre>
