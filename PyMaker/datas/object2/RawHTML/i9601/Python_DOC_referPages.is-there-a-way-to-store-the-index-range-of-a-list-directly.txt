<div class="post-text" itemprop="text">
<p>If we do </p>
<pre><code>&gt;&gt;&gt; x = range(10)
</code></pre>
<p>then</p>
<pre><code>&gt;&gt;&gt; x[2:6]
[2, 3, 4, 5]
</code></pre>
<p>I wonder if there a way can do something like </p>
<pre><code>b=2:6 
</code></pre>
<p>so that </p>
<pre><code>x[b] 
</code></pre>
<p>prints </p>
<pre><code>[2, 3, 4, 5]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#slice" rel="nofollow noreferrer"><code>slice()</code></a>:</p>
<pre><code>b = slice(2,6)

x[b]
# [2, 3, 4, 5]
</code></pre>
<p>This is equivalent to <code>x[2:6]</code>.</p>
<p>You can also pass a step parameter to slice, for instance:</p>
<pre><code>b = slice(2,6,2)

x[b]
# [2, 4]
</code></pre>
<p>Which is the same as <code>x[2:6:2]</code>.</p>
</div>
