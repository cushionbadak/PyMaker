<div class="post-text" itemprop="text">
<p>Add a variable <code>output_path</code> pointing to the folder to which you want to export the files, then use this variable in the seconde argument of <code>os.rename()</code>, like this :</p>
<pre><code>import os
from tqdm import tqdm

path = './training_data/car/'
output_path = './training_data/output_folder/'

def image_rename():
    cnt = 1
    for img in tqdm(os.listdir(path)):
        if os.path.isfile(path+img):
            filename, file_extention = os.path.splitext(path+img)
            os.rename(os.path.join(path, img), os.path.join(output_path, 
                       str('car.') + str(cnt) + file_extention))
       cnt +=1

image_rename()
</code></pre>
<p>Make sure to create the output folder in your system (by using <code>mkdir</code>, for example).</p>
