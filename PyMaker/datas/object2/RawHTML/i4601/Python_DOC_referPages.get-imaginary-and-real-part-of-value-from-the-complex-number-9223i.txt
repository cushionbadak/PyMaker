<div class="post-text" itemprop="text">
<p>I want to extract numbers from a string and store them in different variables.</p>
<p>for eg:- <code>"92+23i"</code> is a complex number. I want to store 92 in variable <code>num_real</code> and 23 in <code>num_imagin</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Python has predefined type as <a href="https://docs.python.org/3/library/functions.html#complex" rel="nofollow noreferrer"><strong><code>complex</code></strong></a> however it expects the string to be of type <code>"a+bj"</code> <em>(note the <code>j</code> instead of <code>i</code>, and it should be without space)</em>. In your case you may replace <code>"i"</code> with <code>"j"</code> in your string and get your desired values as:</p>
<pre><code>&gt;&gt;&gt; my_str = "92+23i"
&gt;&gt;&gt; my_num = complex(my_str.replace('i', 'j'))
</code></pre>
<p>From this complex number, you need to extract the desired values using <code>my_num.imag</code> for imaginary part and <code>my_num.real</code> for the real part of the complex number. For example:</p>
<pre><code>&gt;&gt;&gt; my_num.imag
23.0
&gt;&gt;&gt; my_num.real
92.0
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your string is very close to parsing as a Python literal:</p>
<pre><code>&gt;&gt;&gt; s = '92+23i'
&gt;&gt;&gt; import ast
&gt;&gt;&gt; ast.literal_eval(s.replace('i', 'j'))
(92+23j)
</code></pre>
<p>Access the real and imaginary parts with <code>num.real</code> and <code>num.imag</code>.</p>
