<div class="post-text" itemprop="text">
<p>You should set your class (in this case <code>Ui_Error</code>) up to accept parameters when initialized. that is accomplished through the class' <code>__init__</code> function. So your class should be declared like this:</p>
<pre><code>class Ui_Error:
    def __init__(self, object, error_message):
        # Whatever you want to do with object and error_message
</code></pre>
<p>You can get more info on classes <a href="https://docs.python.org/3/tutorial/classes.html" rel="nofollow noreferrer">here</a>.</p>
