<p>You could try using <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><code>findall</code></a> which returns all non-overlapping matches of pattern in string, as a list of strings. </p>
<pre><code>import re
m = "Example text 'citedby':34567"  # just an example.
n = re.findall(r"'citedby':\s?(\d+)", m, re.IGNORECASE)
print(' '.join(n))  # 34567
</code></pre>
<p>For your specific question:</p>
<pre><code>import scholarly
import re

m = next(scholarly.search_pubs_query('Perception of physical stability and center of mass of 3D objects'))
n = re.findall(r"'citedby':\s?(\d+)", str(m), re.IGNORECASE)
print(''.join(n))  # 13
</code></pre>
<p><strong>Note</strong>: Here <code>m</code> is <code>&lt;class 'scholarly.Publication'&gt;</code> object. <code>str(m)</code> makes it <code>&lt;class 'str'&gt;</code>. <code>findall</code> works only with strings.</p>
