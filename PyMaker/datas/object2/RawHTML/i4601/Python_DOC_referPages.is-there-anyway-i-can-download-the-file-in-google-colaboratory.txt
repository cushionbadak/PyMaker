<div class="post-text" itemprop="text">
<p>Read the manual, they have good examples and explanations: <a href="https://docs.python.org/3/library/urllib.request.html#module-urllib.request" rel="nofollow noreferrer">urllib.request</a> </p>
<p>To download:</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; import urllib.request
&gt;&gt;&gt; urllib.request.urlretrieve('https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png', 'google.png')
('google.png', &lt;http.client.HTTPMessage object at 0x7fba3c4cb908&gt;)
&gt;&gt;&gt; os.listdir()
['google.png']
</code></pre>
<p>To just check the content:</p>
<pre><code>&gt;&gt;&gt; with urllib.request.urlopen('http://www.python.org/') as f:
...     print(f.read(300))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Possibly simpler: use <code>!wget</code>, e.g., executing a cell with the command:</p>
<pre><code>!wget https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png
</code></pre>
<p>will save the file to the local filesystem of the VM. (Note the leading <code>!</code>. This is a signal to execute the line as a shell command.)</p>
