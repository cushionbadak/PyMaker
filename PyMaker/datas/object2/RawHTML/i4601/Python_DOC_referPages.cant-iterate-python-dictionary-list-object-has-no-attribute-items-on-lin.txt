<div class="post-text" itemprop="text">
<p>The problem lies in the sorted(...) method. From python documentation:</p>
<p><a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer">https://docs.python.org/3/library/functions.html#sorted</a></p>
<p>You'll see that the method returns a <strong>list</strong>. Therefore, when you call</p>
<pre><code>map = sorted(map)
</code></pre>
<p>You are actually converting your dictionary to a sorted list. </p>
<p>Off the top of my head, a better way of doing this would be getting a sorted list of the keys, and then referencing that. Something like </p>
<pre><code>keys = sorted(map.keys()) 
print(map[keys[2])
</code></pre>
<p>Obviously this is not including validation on your initial array. </p>
</div>
<div class="post-text" itemprop="text">
<p><code>sorted</code> returns a list; you are replacing your <code>dict</code> with the list of sorted keys.</p>
<p>Instead, you just want</p>
<pre><code>for key, value in sorted(map.items()):
</code></pre>
<p>which iterates over a list of key/value pairs sorted by key.</p>
</div>
<span class="comment-copy"><code>sorted</code> returns a list. dicts are unordered mappings, they can't be sorted. Use a <code>collections.OrderedDict</code> instead. Also, <code>for key, value in map.items:</code> should be <code>for key, value in map.items():</code>.</span>
