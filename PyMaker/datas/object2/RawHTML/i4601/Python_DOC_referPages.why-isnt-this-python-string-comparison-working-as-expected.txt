<div class="post-text" itemprop="text">
<p>You need to use regex. This can be done with the <code>re</code> module. In Python strings are equal only if every character is the same, not with pattern matching. I believe what you are trying to achieve is:</p>
<pre><code>import re

print(re.match("LC08[^B]*_B[1-7]", "LC08_L1TP_215068_20151114_20170402_01_T1_B1") != None)
</code></pre>

<p>Please have a look at <code>re</code> <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">module</a>.</p>
<pre><code>import re

def main():
    mystring = 'LC08_L1TP_215068_20151114_20170402_01_T1_B1'
    m = re.search('^LC08_.*B\d{1,7}', mystring)
    print(m != None)

if __name__ == '__main__':
    main()
</code></pre>
