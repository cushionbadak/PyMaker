<p>I'm in the middle of writing a python program and I have to add two ints (the players points) together if the "teams" are the same. There are two players per team.</p>
<p>My list looks something like this:</p>
<pre><code>list = [["team1", 10],["team2", 20],["team1", 30],["team3",0],["team2",5]]
</code></pre>
<p>How do I add the values from <code>team1</code> together and then <code>team2</code> and so?</p>
<p>I want the final list (doesn't matter if it's a seperate list) to look like this:</p>
<pre><code>list2 = [["team1", 40],["team2", 25],["team3",0]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>Counter</code></a> or a <a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow noreferrer"><code>defaultdict</code></a>:</p>
<pre><code>list1 = [["team1", 10],["team2", 20],["team1", 30],["team3",0],["team2",5]]

from collections import Counter, defaultdict

counter = Counter()  # alternatively: counter = defaultdict(int)
for team, score in list1:
    counter[team] += score
list2 = list(counter.items())

print(list2)
# output: [('team2', 25), ('team1', 40), ('team3', 0)]
</code></pre>
</div>
