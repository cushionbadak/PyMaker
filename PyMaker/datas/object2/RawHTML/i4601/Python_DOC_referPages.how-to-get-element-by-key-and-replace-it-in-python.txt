<p>I have some list</p>
<pre><code>tests = [
{'module': 'module1', 'test': 'test1', 'v8': 'pass'},
{'module': 'module2', 'test': 'test2', 'v8': 'fail'},
{'module': 'module3', 'test': 'test3', 'v8': 'fail'}
]
</code></pre>
<p>How I can get element 'v8' where 'test' equals test1, and replace it with other value?</p>
</div>
<div class="post-text" itemprop="text">
<p><em>(you shouldn't use <code>list</code> as a name as it overrides the <code>list()</code> function, so I replaced it with <code>l</code>)</em></p>
<hr/>
<p>Just iterate over the list and replace where appropriate:</p>
<pre><code>for d in l:
    if d['test'] == 'test1':
        d['v8'] = 'another'
</code></pre>
<p>giving:</p>
<pre><code>[{'test': 'test1', 'v8': 'another', 'module': 'module1'},
 {'test': 'test2', 'v8': 'fail', 'module': 'module2'},
 {'test': 'test3', 'v8': 'fail', 'module': 'module3'}]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; tests = [
... {'module': 'module1', 'test': 'test1', 'v8': 'pass'},
... {'module': 'module2', 'test': 'test2', 'v8': 'fail'},
... {'module': 'module3', 'test': 'test3', 'v8': 'fail'}
... ]
</code></pre>
<p>First, extract the item you need.</p>
<pre><code>&gt;&gt;&gt; item = [d for d in tests if d['test'] == 'test1']
&gt;&gt;&gt; item
[{'v8': 'pass', 'module': 'module1', 'test': 'test1'}]
</code></pre>
<p>You have a list of one element; check access to field <code>v8</code></p>
<pre><code>&gt;&gt;&gt; item[0]['v8']
'pass'
</code></pre>
<p>Now, simply reassign that field.</p>
<pre><code>&gt;&gt;&gt; item[0]['v8'] = "NEW VALUE"
&gt;&gt;&gt; item
[{'v8': 'NEW VALUE', 'module': 'module1', 'test': 'test1'}]
&gt;&gt;&gt; tests
[{'v8': 'NEW VALUE', 'module': 'module1', 'test': 'test1'}, {'v8': 'fail', 'module': 'module2', 'test': 'test2'}, {'v8': 'fail', 'module': 'module3', 'test': 'test3'}]
</code></pre>
<p>Note that the value of the original dict changes: this happens with references to mutable items.  Go through enough tutorials that you <em>understand</em> this before you try to make further use of it.</p>
