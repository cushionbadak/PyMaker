<pre><code>list1 = ['John, 35.60', 'Alf, 200.13', 'Greg, 62.76', 'Paul, 555', 'George, 123.6', 'Colin, 35.60', 'Phil, 23', 'Bret, 200.13']
</code></pre>
<p>Into this list</p>
<pre><code>list2 = ['Phil, 23', 'John, 35.60', 'Colin, 35.60', 'Greg, 62.76', 'George, 123.6', 'Alf, 200.13', 'Bret, 200.13', 'Paul, 555']
</code></pre>
<p>Basically, I have a list where each element is like <code>[customerName, orderPrice]</code>.
I want to make a function where it takes can take ANY list like <code>list1</code> and output a list (<code>list2</code>) with those same elements except it is ordered by <code>orderPrice</code></p>

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html?highlight=sort#list.sort" rel="noreferrer"><code>sort</code></a> accepts a keyword <code>key</code> where you can enter a function; the result of this function is used to sort:</p>
<pre><code>list1 = ['John, 35.60', 'Alf, 200.13', 'Greg, 62.76', 'Paul, 555', 
         'George, 123.6', 'Colin, 35.60', 'Phil, 23', 'Bret, 200.13']

srt = sorted(list1, key=lambda x: float(x.split(',')[1]))
print(srt)
# ['Phil, 23', 'John, 35.60', 'Colin, 35.60', 'Greg, 62.76', 
#  'George, 123.6', 'Alf, 200.13', 'Bret, 200.13', 'Paul, 555']
</code></pre>
<p>the (nameless) function i used splits your string at <code>,</code>, takes the right part of the string and returns the <code>float</code> of it.</p>
