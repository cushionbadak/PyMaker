<p>As per <a href="https://docs.python.org/3.6/library/datetime.html#datetime.datetime.strptime" rel="nofollow noreferrer">docs</a>, The <code>datetime.strptime()</code> method accepts a <code>date_string</code> as the first argument. So, I tried this:</p>
<pre><code>import datetime
test = datetime.datetime.strptime('2018-01-01', '%Y-%m-%d').date()
print(test)
#datetime.date(2018, 1, 1)
</code></pre>
<p>If I do <code>from_date = 2018-01-01</code>, I get an <code>invalid token</code> error for the <code>type</code> of <code>from_date</code>:</p>
<pre><code>   from_date = 2018-01-01
   print(type(from_date))

   File "&lt;ipython-input-7-9e5449277912&gt;", line 2
   from_date = 2018-01-01
                      ^
   SyntaxError: invalid token
</code></pre>
<p>This means that <code>from_date</code> must be a string. You could use <code>from_date = '2018-01-01'</code> and that would work. If I check the type of <code>from_date</code> now, I will get a type of class <code>string</code>: </p>
<pre><code>from_date = '2018-01-01'
print(type(from_date))
