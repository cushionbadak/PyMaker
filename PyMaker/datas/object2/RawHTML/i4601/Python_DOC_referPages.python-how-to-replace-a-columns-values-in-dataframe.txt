<div class="post-text" itemprop="text">
<p>If there is a column called <code>date</code> in a <code>pandas.DataFrame</code>, For which the values are:</p>
<pre><code>'2018-02-01', 
'2018-02-02',
 ...
</code></pre>
<p>How do I change all the values to integers? For example:</p>
<pre><code>'20180201', 
'20180202',
 ...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>.str.replace()</code> like:</p>
<h3>Code:</h3>
<pre><code>df['newdate'] = df['date'].str.replace('-', '')
</code></pre>
<p>or if not using a regex, faster as a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> like:</p>
<pre><code>df['newdate'] = [x.replace('-', '') for x in df['date']]
</code></pre>
<h3>Test Code:</h3>
<pre><code>df = pd.DataFrame(['2018-02-01', '2018-02-02'], columns=['date'])
print(df)

df['newdate'] = df['date'].str.replace('-', '')
print(df)

df['newdate2'] = [x.replace('-', '') for x in df['date']]
print(df)
</code></pre>
<h3>Results:</h3>
<pre><code>         date
0  2018-02-01
1  2018-02-02

         date   newdate
0  2018-02-01  20180201
1  2018-02-02  20180202

         date   newdate  newdate2
0  2018-02-01  20180201  20180201
1  2018-02-02  20180202  20180202
</code></pre>
</div>
<span class="comment-copy"><code>.replace</code> the dashes with empty strings.</span>
