<p>Lets say I have a multidimensional array [A][B][C], and I want to order the array with respect to <code>A</code>, of course the relationship of the values of <code>B</code> and <code>C</code> which corresponded to their neighboring <code>A</code> needs to be maintained.
Also, assuming we have multiple <code>A</code>s with the same value, how do we order the <code>B</code>s so that we have <code>A</code> ordered primarily then <code>B</code> ordered if its possible.</p>
<p>Something like:</p>
<pre><code>(1,2,3)
(1,3,7)
(1,4,5)
(1,5,2)
(2,3,5)
(2,4,9)
(2,5,0)
...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Let's assume your data was not already sorted:</p>
<pre><code>data = [(1,2,3),
        (1,3,7),
        (1,4,5),
        (1,5,2),
        (2,3,5),
        (2,4,9),
        (2,5,0)]
</code></pre>
<p>Then <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer"><code>sorted</code></a> will return a sorted <code>list</code>:</p>
<pre><code>sorted_data = sorted(data)
</code></pre>
</div>
