<div class="post-text" itemprop="text">
<p>According to <a href="https://docs.python.org/3/library/threading.html?highlight=thread#thread-objects" rel="nofollow noreferrer">threading document</a>, after creating Thread object, its start method should be call so that the new thread can run. You also need to provide None as the place holder for unimplemented group feature. So the code will be as below.</p>
<pre class="lang-py prettyprint-override"><code>Camthread1 = Thread(None, ReVid.recordvideo)
Camthread1.start()
</code></pre>
<p>or</p>
<pre class="lang-py prettyprint-override"><code>Camthread1 = Thread(target=ReVid.recordvideo)
Camthread1.start()
</code></pre>
</div>
