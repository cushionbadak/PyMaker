<p>I have such a code in order to fetch some data from a url:</p>
<pre><code>import urllib2
response = urllib2.urlopen('https://api.genderize.io/?name=joseph')
html = response.read()

&gt;&gt;&gt; print html
{"name":"joseph","gender":"male","probability":"0.99","count":923}
</code></pre>
<p>Everything up to here is fine. But when I want to fetch "gender" data from this dictionary;</p>
<pre><code>&gt;&gt;&gt; print html["gender"]
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: string indices must be integers, not str
</code></pre>
<p>I get the above error.</p>
<p>How can I fix the problem?</p>
</div>
<div class="post-text" itemprop="text">
<p>That's because <code>html</code> is of type string. To validate that, try:</p>
<pre><code>&gt;&gt;&gt; type(html)
&lt;type 'str'&gt;
</code></pre>
<p>Hence the error message saying that <em>string indices must be integers, not str</em>.</p>
<p>You should convert it to JSON using <a href="https://docs.python.org/3/library/json.html" rel="nofollow"><code>json</code></a> module:</p>
<pre><code>json.loads(html)['gender']
</code></pre>
</div>
