<div class="post-text" itemprop="text">
<p>You need to escape <code>\</code> or to use raw string literal:</p>
<pre><code>&gt;&gt;&gt; re.sub("'", "\\'","Newton's method")
"Newton\\'s method"
&gt;&gt;&gt; re.sub("'", r"\'","Newton's method")
"Newton\\'s method"
</code></pre>
<p>BTW, for this case, you don't need to use regular expression. <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow"><code>str.replace</code></a> is enough:</p>
<pre><code>&gt;&gt;&gt; "Newton's method".replace(r"'", r"\'")
"Newton\\'s method"
</code></pre>
<hr/>
<p><strong>UPDATE</strong></p>
<p><code>\\</code> is a way python <code>repr</code> represents backslash chracter in the string. If you print the string, you will see that it's a <code>\</code>.</p>
<pre><code>&gt;&gt;&gt; "Newton\\'s method"
"Newton\\'s method"
&gt;&gt;&gt; print("Newton\\'s method")
Newton\'s method
</code></pre>
