<p>One option could be to enclose the <code>coordinates</code> into square brackets and use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow"><code>literal_eval()</code></a>:</p>
<pre><code>&gt;&gt;&gt; from ast import literal_eval
&gt;&gt;&gt; coordinates = "(-89.38477, 26.6671), (-89.38477, 27.13737), (-88.81348, 27.13737), (-88.81348, 26.6671)"
&gt;&gt;&gt;
&gt;&gt;&gt; data = list(literal_eval(coordinates))
&gt;&gt;&gt; data
[(-89.38477, 26.6671), (-89.38477, 27.13737), (-88.81348, 27.13737), (-88.81348, 26.6671)]
</code></pre>
