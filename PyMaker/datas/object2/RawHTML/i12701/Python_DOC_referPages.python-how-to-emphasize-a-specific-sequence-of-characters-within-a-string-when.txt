<p>I have a program that prints strings consisting of a random sequence of lowercase letters and spaces. </p>
<p>Is there a way emphasize a certain target word within that string when I print the string to make the target easier to spot in the output? </p>
<p>For example: </p>
<h3>Current code:</h3>
<pre><code>&gt;&gt;&gt; print(mystring)
</code></pre>
<blockquote>
<p>mxxzjvgjaspammttunthcrurny dvszqwkurxcxyfepftwyrxqh</p>
</blockquote>
<h3>Desired behaviour:</h3>
<pre><code>&gt;&gt;&gt; some_function_that_prints_with_emphasis(mystring,'spam')
</code></pre>
<blockquote>
<p>mxxzjvgja<strong>spam</strong>mttunthcrurny dvszqwkurxcxyfepftwyrxqh</p>
</blockquote>
<p>Other acceptable forms of emphasis would be:</p>
<ul>
<li>boldening the characters</li>
<li>changing colour</li>
<li>capitalizing</li>
<li>separating with extra characters</li>
<li>any other idea that is easily implemented in Python 3</li>
</ul>
<p>I'm leaving the requirements deliberately vague because as a beginner I'm not aware of all that Python can do so there might be a simpler way to do this that I've overlooked.</p>

<div class="post-text" itemprop="text">
<p>Basically everything you're looking for would be done with the <a href="https://docs.python.org/3/library/curses.html" rel="nofollow"><code>curses</code></a> module; it's designed to perform advanced control over the terminal to do stuff like change colors, bold text, etc. You need to use the various <code>has_*</code> commands to determine terminal capabilities and choose your preferred emphasis style, but after that the <a href="https://docs.python.org/3/library/curses.html" rel="nofollow">docs page</a> and <a href="https://docs.python.org/3/howto/curses.html#curses-howto" rel="nofollow">the linked tutorial</a> should give you all the info you need.</p>
<p>For simpler usage, you can just print out the raw terminal escape codes to add and remove color (you just have to split the line up yourself or use <code>re</code> to perform replacements to add the codes). For example, to highlight 'spam' in a line as blue:</p>
<pre><code>myline = "abc123spamscsdfwerf"
print(myline.replace('spam', '\033[94mspam\033[0m'))
</code></pre>
<p>For ease of use, you can use <a href="https://pypi.python.org/pypi/ansicolors/1.0.2" rel="nofollow"><code>ansicolors</code></a> to avoid having to manually deal with color escapes and the like.</p>

<div class="post-text" itemprop="text">
<p>You can replace the target with its version capitalized:</p>
<pre><code>def emphasis(string, target):
  return string.replace(target, target.upper())
</code></pre>

<div class="post-text" itemprop="text">
<p>you could just find the substring you're looking for and put a marker under them, like:</p>
<pre><code>needle = "spam"
haystack = "mxxzjvgjaspammttunthcrurny dvszqwkurxcxyfepftwyrxqh"
pos = haystack.find(needle)
print(haystack)
print(" "*(pos-1) + "^" * len(needle))
</code></pre>
