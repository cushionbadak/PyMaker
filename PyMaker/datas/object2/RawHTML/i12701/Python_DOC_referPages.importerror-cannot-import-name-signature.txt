<div class="post-text" itemprop="text">
<ul>
<li>In python 2, the <a href="https://docs.python.org/2/library/inspect.html" rel="nofollow"><code>inspect</code></a> module does not have a <code>signature</code> method.</li>
<li>In python 3, the <a href="https://docs.python.org/3/library/inspect.html" rel="nofollow"><code>inspect</code></a> module does have a <code>signature</code> method.</li>
</ul>
<p>This code is just trying to work with both python 2 and 3.</p>
<p>You may want to use the <a href="https://pypi.python.org/pypi/funcsigs" rel="nofollow"><code>funcsigs</code></a> module if you are using python 2, or use <code>sklearn.externals.funcsigs</code> directly (for version <code>sklearn &gt;= 0.17</code>).</p>
