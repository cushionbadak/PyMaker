<p>@aftab You need add <code>log-level</code> please see below.</p>
<pre><code>./odoo.py -d &lt;dbname&gt; --test-enable --log-level=test
</code></pre>
<p>and regarding you question, If you are making changes to installed modules and need to re test all test cases then you need to simple restart you server with <code>-u &lt;module_name&gt;</code> or <code>-u all</code>(for all modules) with the above command.</p>
</div>
<div class="post-text" itemprop="text">
<p>This useful For Run odoo test case:</p>
<pre><code>./odoo.py -i/-u module_being_tested -d being_used_to_test --test-enable
</code></pre>
<p>Common options:</p>
<pre><code> -i INIT, --init=INIT
       install one or more modules (comma-separated list, use "all" for all modules), requires -d

-u UPDATE, --update=UPDATE
       update one or more modules (comma-separated list, use "all" for all modules). Requires -d.
</code></pre>
<p>Database related options:</p>
<pre><code>-d DB_NAME, --database=DB_NAME
       specify the database name
</code></pre>
<p>Testing Configuration:</p>
<pre><code> --test-enable:  Enable YAML and unit tests.
</code></pre>
</div>
