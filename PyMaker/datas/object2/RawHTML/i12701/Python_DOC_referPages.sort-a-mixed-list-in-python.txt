<p>So if I'm right, you're trying to get <code>[1, 4, 5]</code>, <code>[2, 4, 5]</code>, <code>[3, 4, 5]</code>, and then <code>[1, 5, 6]</code>, <code>[2, 5, 6]</code>, <code>[3, 5, 6]</code>. </p>
<p>Then sort your list use <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow"><code>sorted()</code></a>,  and <code>(x[1][1:], x[1][0])</code> gives you something like <code>4, 5, 1</code>, <code>4, 5, 2</code>. And then <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow"><code>sorted()</code></a> function will sort the list based on them.</p>
<hr/>
<p>Note that <code>sorted()</code> will create another list, however it doesn't change the current list. If you want to change it you need use:</p>
<pre><code>list1 = sorted(list1, key=lambda x: (x[1][1:], x[1][0]))
</code></pre>
<p>However, here's another way use <a href="https://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow"><code>list.sort()</code></a>:</p>
<pre><code>list1.sort(lambda x: (x[1][1:], x[1][0]))
</code></pre>
