<p>Doing python exercises already I've a problem with string:</p>
<pre><code>#!/usr/bin/python

str = 'mandarino'
indice = len(str)-1
#print ("indice is:",indice)

while indice &gt; 0:
        lett = str[indice]
        print (lett)
        indice = indice -1
</code></pre>
<p>Putting off "-1" the results is:</p>
<pre><code>IndexError: string index out of range
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can add:</p>
<pre><code>indice = indice - 1
</code></pre>
<p>and change:</p>
<pre><code>while indice &gt; 0:
</code></pre>
<p>to</p>
<pre><code>while indice &gt;= 0:
</code></pre>
<p>so the script will be:</p>
<pre><code>#!/usr/bin/python

str = 'mandarino'
indice = len(str)
indice = indice - 1
#print ("indice is:",indice)

while indice &gt;= 0:

        lett = str[indice]
        print (lett)
        indice = indice -1
</code></pre>
<p>Better and handy:</p>
<pre><code>indice = len(str)-1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>while indice &gt; 0:
</code></pre>
<p>should be</p>
<pre><code>while indice &gt;= 0:
</code></pre>
<p>to print the first character (index <code>0</code>) at last.</p>
<hr/>
<p>BTW, if you use <a href="https://docs.python.org/3/library/functions.html#reversed" rel="nofollow"><code>reversed</code></a>, you don't need to calculate index yourself:</p>
<pre><code>s = 'mandarino'
for ch in reversed(s):
    print(ch)
</code></pre>
<p>Side note: Don't use <code>str</code> as a varaible name. It will shadow a built in function/type <a href="https://docs.python.org/3/library/functions.html#str" rel="nofollow"><code>str</code></a>.</p>
</div>
