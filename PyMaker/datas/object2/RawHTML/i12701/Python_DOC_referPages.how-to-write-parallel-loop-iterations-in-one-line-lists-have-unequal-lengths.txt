<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="noreferrer"><code>zip</code></a>:</p>
<pre><code>&gt;&gt;&gt; zip(li1, li2)
&lt;zip object at 0x0000000000723248&gt;
&gt;&gt;&gt; list(zip(li1, li2))
[(['a', 'b', 'c'], ['c', 'a', 'b']), (['c', 'd', 'e'], ['c', 'e', 'd'])]
</code></pre>
<p>and <a href="https://docs.python.org/3/library/functions.html#all" rel="noreferrer"><code>all</code></a>:</p>
<pre><code>&gt;&gt;&gt; all([True, True, True])
True
&gt;&gt;&gt; all([True, False, True])
False
</code></pre>
<hr/>
<pre><code>k = all(sorted(x) == sorted(y) for x, y in zip(li1, li2))
