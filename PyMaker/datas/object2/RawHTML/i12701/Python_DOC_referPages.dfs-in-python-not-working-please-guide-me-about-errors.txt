<p><code>graph[start]</code> will raise <a href="https://docs.python.org/3/library/exceptions.html#KeyError" rel="nofollow"><code>KeyError</code></a> if <code>start</code> is not in the dictionary. You can replace it with <a href="https://docs.python.org/3/library/stdtypes.html#dict.get" rel="nofollow"><code>dict.get</code></a> to return default value (empty sequence):</p>
<pre><code>def dfs(graph, start, visited=None):
    if visited is None:
        visited = set()
    if start in visited:
        return
    visited.add(start)
    for a in graph.get(start, []):  # &lt;---
        if a not in visited:
            dfs(graph, a, visited)
    return visited
