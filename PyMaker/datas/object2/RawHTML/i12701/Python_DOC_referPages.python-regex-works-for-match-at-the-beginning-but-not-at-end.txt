<div class="post-text" itemprop="text">
<p>You should use <a href="https://docs.python.org/3/library/re.html#search-vs-match" rel="nofollow"><code>search</code></a> instead of <a href="https://docs.python.org/3/library/re.html#search-vs-match" rel="nofollow"><code>match</code></a>. According to the docs:</p>
<blockquote>
<p>Python offers two different primitive operations based on regular expressions: <code>re.match()</code> checks for a match <strong>only at the beginning of the string</strong>, while <code>re.search()</code> checks for a match <strong>anywhere in the string</strong> (this is what Perl does by default).</p>
</blockquote>
<p>So when you use <code>match</code> it's the same as using <a href="http://www.regular-expressions.info/anchors.html" rel="nofollow"><code>^</code></a> in your regex (matches the position before the first character in the string).</p>
</div>
