<div class="post-text" itemprop="text">
<p>I'd like to take a python dict object and transform it into its equivalent string if it were to be submitted as html form data.</p>
<p>The dict looks like this:</p>
<pre><code>{
   'v_1_name':'v_1_value'
  ,'v_2_name':'v_2_value'
}
</code></pre>
<p>I believe the form string should look something like this:</p>
<pre><code>v_1_name=v_1_value&amp;v_2_name=v_2_value
</code></pre>
<p>What is a good way to do this?</p>
<p>Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>Try <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.urlencode" rel="nofollow">urllib.parse.urlencode</a>:</p>
<pre class="lang-py prettyprint-override"><code>&gt;&gt;&gt; from urllib.parse import urlencode
&gt;&gt;&gt; urlencode({'foo': 1, 'bar': 2})
'foo=1&amp;bar=2'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Simply iterte over the items, join the key and value and create a key/value pair separated by '=' and finally join the pairs by '&amp;'</p>
<p>For Ex...</p>
<pre><code>If d={'v_1_name':'v_1_value','v_2_name':'v_2_value','v_3_name':'v_3_value'}
</code></pre>
<p>Then</p>
<pre><code>'&amp;'.join('='.join([k,v]) for k,v in d.iteritems())
</code></pre>
<p>is</p>
<pre><code>'v_2_name=v_2_value&amp;v_1_name=v_1_value&amp;v_3_name=v_3_value'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>For Python 2.7, you will encounter this error:</p>
<pre><code>&gt;&gt;&gt; from urllib.parse import urlencode
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
ImportError: No module named parse
</code></pre>
<p>Use <a href="https://docs.python.org/2/library/urllib.html#urllib.urlencode" rel="nofollow noreferrer">urllib.urlencode</a> instead.</p>
<pre><code>from urllib import urlencode
d = {'v_1_name': 'v_1_value', 'v_2_name': 'v_2_value'}
print urlencode(d)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>'v_2_name=v_2_value&amp;v_1_name=v_1_value'
</code></pre>
</div>
<span class="comment-copy">this won't work with special characters in strings</span>
