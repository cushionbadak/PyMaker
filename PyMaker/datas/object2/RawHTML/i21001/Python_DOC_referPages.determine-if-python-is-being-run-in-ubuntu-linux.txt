<div class="post-text" itemprop="text">
<p>The currently accepted answer uses a deprecated function. The proper way to do this as of Python 2.6 and later is:</p>
<pre><code>import platform
print(platform.linux_distribution())
</code></pre>
<p>The documentation doesn't say if this function is available on non-Linux platforms, but on my local Windows desktop I get:</p>
<pre><code>&gt;&gt;&gt; import platform
&gt;&gt;&gt; print(platform.linux_distribution())
('', '', '')
</code></pre>
<p>There's also this, to do something similar on Win32 machines:</p>
<pre><code>&gt;&gt;&gt; print(platform.win32_ver())
('post2008Server', '6.1.7601', 'SP1', 'Multiprocessor Free')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Looks like <code>platform.dist()</code> and <code>platform.linux_distribution()</code> are <a href="https://docs.python.org/3/library/platform.html#platform.dist" rel="nofollow noreferrer">deprecated in Python 3.5 and will be removed in Python 3.7</a>. The following works in python 2/3</p>
<pre><code>import platform
'ubuntu' in platform.platform().lower()
</code></pre>
<p>Example return value</p>
<pre><code>&gt;&gt;&gt; platform.platform()
'Linux-4.10.0-40-generic-x86_64-with-Ubuntu-16.04-xenial'
</code></pre>
</div>
