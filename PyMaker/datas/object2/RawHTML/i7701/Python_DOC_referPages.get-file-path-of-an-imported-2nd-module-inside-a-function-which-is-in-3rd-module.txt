<p>I am importing module <code>Y.py</code> in <code>X.py</code> and there is a function <code>my_func()</code> in module <code>Z.py</code> which I am importing in <code>Y.py</code> How can I print the path of <code>Y.py</code> in <code>my_func()</code> while executing <code>X.py</code> without passing any parameter to <code>my_func()</code></p>
<p>Module X</p>
<pre><code># this is module X

import Y

#some code
</code></pre>
<p>Module Y</p>
<pre><code># this is module Y

from Z import *
my_func()

#some code
</code></pre>
<p>Module Z</p>
<pre><code># this is module Z

def my_func()
    print("the path of Y module is: ")  # here I want to print the path of Y.py

# do something with the path of Y.py
</code></pre>
<p>When I execute X.py it should print the path of Y.py, I tried using <code>os.getcwd()</code> but its giving me the path of X.py not the path of Y.py</p>
</div>
<div class="post-text" itemprop="text">
<p>You use the <a href="https://docs.python.org/3/reference/import.html#__file__" rel="nofollow noreferrer"><code>__file__</code></a> attribute of the module object:</p>
<pre><code>import Y

def my_func():
    path = Y.__file__
</code></pre>
