<div class="post-text" itemprop="text">
<p><code>all()</code> returns true if <em>all values</em> are true, see the <a href="https://docs.python.org/3/library/functions.html#all" rel="noreferrer">function documentation</a>:</p>
<blockquote>
<p>Return <code>True</code> if all elements of the iterable are true (or if the iterable is empty).</p>
</blockquote>
<p><code>0</code> is not a true value:</p>
<pre><code>&gt;&gt;&gt; bool(0)
False
</code></pre>
<p>See the <a href="https://docs.python.org/3/library/stdtypes.html#truth-value-testing" rel="noreferrer"><em>Truth Value Testing</em> section</a> of the Python standard types documentation:</p>
<blockquote>
<p>The following values are considered false:</p>
<p>[...]</p>
<ul>
<li>zero of any numeric type, for example, 0, 0.0, 0j.</li>
</ul>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>Your iterable contains a <code>0</code>, and 0 evaluates to <code>False</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>In Python the integer 0 is equal to False. </p>
</div>
<div class="post-text" itemprop="text">
<p>To check if the array contains boolean value "False" anywhere (as I guess 
you had intended to do), simply do:</p>
<pre><code>if all(map(lambda v: v is not False, [0, 4, "Sample string"])):
    print "Yes"
else:
    print "No"
</code></pre>
</div>
