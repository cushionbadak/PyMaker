<p>I have a list of floats like following:</p>
<pre><code>numbers = [23.23, 0.123334987, 1, 4.223, 9887.2]
</code></pre>
<p>I want to format it so that I could print up to 4 decimal places. So, I wrote the following code:</p>
<pre><code>for x in numbers:
    print("{:10.04f}".format(x))
</code></pre>
<p>It prints following, i.e., each number in a separate line. </p>
<pre><code>   23.2300
    0.1233
    1.0000
    4.2230
 9887.2000
</code></pre>
<p>However, I want to print all of them on the same line. Using following code, I could achieve this.</p>
<pre><code>print('{:10.4f}{:10.4f}{:10.4f}{:10.4f}{:10.4f}'.format(numbers[0],numbers[1],numbers[2],numbers[3],numbers[4]))
</code></pre>
<p>And I got this:</p>
<pre><code>   23.2300    0.1233    1.0000    4.2230 9887.2000
</code></pre>
<p>My question is: Is there any better way to achieve this without explicitly writing each element of the list using a loop? I have a list of 110 entries and writing each explicitly each element so many times is painful.</p>
</div>
<div class="post-text" itemprop="text">
<p>How about:</p>
<pre><code>print(' '.join("{:10.04f}".format(x) for x in numbers))
</code></pre>
<p>See <a href="https://www.tutorialspoint.com/python/string_join.htm" rel="nofollow noreferrer">String join()</a></p>
<p>Within the join(), we apply the format to each element in the list of numbers.</p>
</div>
<div class="post-text" itemprop="text">
<p>The print built-in methon has an "end" keyword argument which defaults to a linebreak. You just need to replace it with a space or tab, like so:</p>
<pre><code>print("{:10.04f}".format(x), end="\t")
</code></pre>
<p>Check <a href="https://docs.python.org/3/library/functions.html?highlight=print#print" rel="nofollow noreferrer">https://docs.python.org/3/library/functions.html?highlight=print#print</a> for more information on it.</p>
</div>
