<div class="post-text" itemprop="text">
<p>Simply use <a href="https://docs.python.org/3/library/stdtypes.html#str.startswith" rel="nofollow noreferrer"><code>str.startwith()</code></a> :</p>
<pre><code>for line in inputfile:
    if line.startwith('PuckTaken'):
        continue
    # do something with line
</code></pre>
</div>
