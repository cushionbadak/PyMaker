<p>Lets say i have one list <em>a</em> and a list of lists <em>b</em>:</p>
<pre><code>a = ['1','2','3']
b = [['Hello'],['I'],['am']]
</code></pre>
<p>How could i go about getting the following output?</p>
<pre><code>b = [['Hello','1'],['I','2'],['am','3']]
</code></pre>
<p>I have tried various things unsuccesfully like the following. </p>
<pre><code>for i in a:
    for j in range(b):
        j.append(i[j])
print(b)
</code></pre>
<p>EDIT:
I was looking for a way to do this, in case the two lists is not of the same length, which the duplicate does not answer. If you look in the comments on the accepted answer, you will fin the solution though. </p>

<div class="post-text" itemprop="text">
<p>you could use a list comprehension with <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a>:</p>
<pre><code>r = [[s[0], n] for (n, s) in zip(a, b)]
</code></pre>
<hr/>
<p>update after comments:</p>
<p>if your lists are not of the same length you can use <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow noreferrer"><code>itertools.zip_longest</code></a> and something like</p>
<pre><code>from itertools import zip_longest

r = [s if n is None else
     [n] if s is None else
     [s[0], n] for (n, s) in zip_longest(a, b)]
</code></pre>

&gt;&gt;&gt; b = [['Hello'],['I'],['am']]

</code></pre>
