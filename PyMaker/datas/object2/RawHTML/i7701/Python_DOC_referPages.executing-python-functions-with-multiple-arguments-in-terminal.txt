add this code, the sys.argv first parameter is this file name</p>
<pre><code>import sys
if __name__ == "__main__":
    a = sys.argv[1]
    b = sys.argv[2]
    count(a,b)
</code></pre>

<div class="post-text" itemprop="text">
<p><code>a</code> and <code>b</code> are the arguments of <code>count</code>. You cannot use them outside that scope. You could instead use <code>sys.argv</code> to access the commandline arguments:</p>
<pre><code>from sys import argv
if __name__ == "__main__":
    print(count (argv[1], argv[2]))
</code></pre>
