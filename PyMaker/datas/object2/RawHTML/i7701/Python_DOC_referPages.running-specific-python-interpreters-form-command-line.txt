<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/using/windows.html#python-launcher-for-windows" rel="nofollow noreferrer">Python launcher for Windows</a>:</p>
<ul>
<li><p>Python 2.7 interptreter:</p>
<blockquote>
<p>C:\&gt; py -2.7</p>
</blockquote></li>
<li><p>Python 3.6 interpreter:</p>
<blockquote>
<p>C:\&gt; py -3.6</p>
</blockquote></li>
</ul>
<p>You also can run a script with a specific interpreter:</p>
<blockquote>
<p>C:\&gt; py -2.7 myScript.py</p>
</blockquote>
</div>

<p>If its windows machine, to open a specific version of python you can use py from the command line.</p>
<p>example:
py2.7 #for version 2.7</p>
<p>and </p>
<p>py3.6 #for version 3.6</p>
<p>if its a Linux machine you can use the following commands in the command line python </p>
<p>python2.7 #opens version 2.7 </p>
<p>and</p>
<p>python3.6 #opens version 3.6</p>

<span class="comment-copy">You can copy or symlink python.exe to python2.7.exe, and then add the Python 2.7 installation directory to the <code>PATH</code> environment variable. Or just use the py.exe launcher, e.g. <code>py -2</code> or <code>py -2.7</code>.</span>
<span class="comment-copy">It's <code>py -2.7</code> and <code>py -3.6</code>. <code>py</code> on its own used to default to the latest version of Python 2 installed. Starting with 3.6, it uses the latest version of Python 3. An active virtual environment overrides this default. You can also override it via the <code>PY_PYTHON</code> environment variable.</span>
