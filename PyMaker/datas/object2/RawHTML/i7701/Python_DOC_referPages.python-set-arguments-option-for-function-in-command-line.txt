<div class="post-text" itemprop="text">
<p>I have written simple function to cut lines in txt file, takes 3 arguments :</p>
<blockquote>
<p>cut(inicial_line, final_line, file)</p>
</blockquote>
<p>now how do i put inicial_line and final_line to -c option to execute it for example:</p>
<blockquote>
<p>$ python cut.py -c 5 8 f.txt</p>
</blockquote>
<p>and it print file text from 5th to 8th line</p>
</div>
<div class="post-text" itemprop="text">
<p>i found solution</p>
<blockquote>
<pre><code>__name__=='__main__':
        import argparse
        import sys
        pars = argparse.ArgumentParser()
        pars.add_argument('-n', nargs=2, type=int)
        args=pars.parse_args(sys.argv[1:4])
        print cut(args.n[0], args.n[1], sys.argv[4])
</code></pre>
</blockquote>
<p>main problem was <strong>args=pars.parse_args(sys.argv[1:4])</strong></p>
<p>i didnt know that <strong>args=pars.parse_args()</strong> is the same as <strong>args=pars.parse_args(sys.argv[1:])</strong></p>
</div>
