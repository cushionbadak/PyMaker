<div class="post-text" itemprop="text">
<p>You are not producing JSON, so <strong>don't use the JSON module</strong>. You are producing CSV data, with a space as delimiter. Use the <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer"><code>csv</code> module</a>, or use simple string formatting.</p>
<p>Using the <code>csv</code> module:</p>
<pre><code>import csv

with open(foutput, 'a', newline='') as f1:
    writer = csv.writer(f1, delimiter=' ')
    writer.writerows(sorted(newdict.items()))
</code></pre>
<p>or simply using string formatting:</p>
<pre><code>with open(foutput, 'a') as f1:
    for key, value in sorted(newdict.items()):
        f1.write('{} {}\n'.format(key, value)
</code></pre>
