<div class="post-text" itemprop="text">
<p><code>%m</code> stands for months; the second time you should've used <code>%M</code> for minutes. Also, <code>%S</code> for seconds.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the following format string:</p>
<pre><code>datetime.strptime(mytime, '%Y-%m-%dT%H:%M:%S.%fZ')
</code></pre>
<p>In yours, you have <code>%m</code> instead of <code>%M</code> for the minute, <code>%s</code> instead of <code>%S</code> for the second, and you were missing the <code>%f</code> for the microsecond.</p>
<p>Check <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">the docs</a> for a full explanation of datetime string formatting.</p>
</div>
