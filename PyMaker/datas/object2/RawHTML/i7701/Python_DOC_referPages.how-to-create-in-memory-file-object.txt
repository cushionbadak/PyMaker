<div class="post-text" itemprop="text">
<p>You are probably looking for <code>BytesIO</code> or <code>StringIO</code> classes from Python <code>io</code> package, both available in <a href="https://docs.python.org/2.7/library/io.html" rel="noreferrer">python 2</a> and <a href="https://docs.python.org/3/library/io.html" rel="noreferrer">python 3</a>.</p>
<p><code>StringIO</code> instance will represent a file-like object in memory, with text data:</p>
<pre><code>import io

f = io.StringIO("some initial text data")
</code></pre>
<p><code>BytesIO</code> instance will represent a file-like object in memory, with binary data:</p>
<pre><code>import io

f = io.BytesIO(b"\x00\x00\x00\x00\x00\x00\x00\x00\x01\x01\x01\x01\x01\x01")
</code></pre>
<p>To store MP3 file data, you will probably need the <code>BytesIO</code> class. To initialize it from a GET request to a server, proceed like this:</p>
<pre><code>import requests
from pygame import mixer
import io

r = requests.get("http://example.com/somesmallmp3file.mp3")
inmemoryfile = io.BytesIO(r.content)

mixer.music.init()
mixer.music.load(inmemoryfile)
mixer.music.play()
</code></pre>
