<div class="post-text" itemprop="text">
<p>Pass the value to the <code>milliseconds</code> argument of <a href="https://docs.python.org/3/library/datetime.html#datetime.timedelta" rel="nofollow noreferrer">the <code>timedelta</code> constructor</a>.</p>
<pre><code>&gt;&gt;&gt; datetime.datetime(1904, 1, 1) + datetime.timedelta(milliseconds=3577382125627)
datetime.datetime(2017, 5, 11, 21, 15, 25, 627000)
</code></pre>
