<div class="post-text" itemprop="text">
<p>Python indicates blocks via indentation <em>alone</em>.  Your "true" and "else" branches have to be indented.  You <em>cannot</em> indent the leading <code>print</code> statement.  Does this match your intended logic?</p>
<pre><code>print("1 Exact | 2  π(r*r)")
choice=input()
if int(choice)&gt;2:
    print("Restart and hit 1 or 2")

if choice=="1":

    pi=22/7
    print("radius=...")
    radius=int(input)
    area=pi(radius**2)
    print(area)

else: 

    pi2='π'
    print("radius=...")
    radius2=int(input)
    squareradius=int(radius2**2)
    str(squareradius)
    area2=pi2+squareradius
    print("area2")
</code></pre>
<p>You do have other problems, but this will get you closer to execution.  For one thing, you never print the computed value in your <em>else</em> branch -- just a literal string.</p>
<p>Among other things, I <em>strongly</em> recommend that you practice incremental programming: write just a line or two at a time, and don't continue until those are syntactically legal and do what you want.  Use <code>print</code> statements to trace values as you go -- and delete them later.  There is no need to get to 20 lines of code when you haven't yet managed to read in a value in the first 4 lines.</p>

<div class="post-text" itemprop="text">
<p>Python uses indentation to define blocks of code.</p>
<p>This is important, as without it, there is no way for the interpreter to know what is expected to be part of that block of code.</p>
<pre><code>if x == 5:
print ('foo')
print ('bar')
</code></pre>
<p>(Ignoring that the lack of indentation causes an error)
If <code>x==5</code> what should happen?</p>
<pre><code>foo
bar
</code></pre>
<p>If it isn't? then what? does it print anything? does it print <code>bar</code>?</p>
<p>So, how does indentation fix this confusion?</p>
<p>Doing this:</p>
<pre><code>if x == 5:
    print ('foo')
    print ('bar')
</code></pre>
<p>will print 
    foo
    bar
when <code>x==5</code> and nothing otherwise</p>
<pre><code>if x == 5:
    print ('foo')
print ('bar')
</code></pre>
<p>will print </p>
<pre><code>foo
bar
</code></pre>
<p>when <code>x==5</code> and </p>
<pre><code>bar
</code></pre>
<p>otherwise</p>
