<p>I am creating a Python3 dictionary:</p>
<pre><code>parents = {
    'parent_0': {
        'f_name': 'john',
        'l_name': 'doe',
        'location': 'New York, NY',
        'birth_month': 'september',
        'nationality': 'france',
        },
    'parent_1': {
        'f_name': 'jane',
        'l_name': '',
        'location': '',
        'birth_month': 'may',
        'nationality': ''
    },

}
</code></pre>
<p>How can parent_1 inherit the key values for l_name, location and nationality from parent_0? </p>
</div>
<div class="post-text" itemprop="text">
<pre><code>for key, value in parents['parent_1'].items():
    if not value:
        parents['parent_1'][key] = parents['parent_0'][key]
</code></pre>

<pre><code>for key in parents['parent_1'].keys:
    if parents['parent_1'] == '':
        parents['parent_1'][key] = parents['parent_0'][key]
</code></pre>
