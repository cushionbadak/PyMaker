<p>My goal is to remove all the occurrences of the characters in one string from another string via some sort of function.</p>
<p>I can show you how I would do this in Python :     </p>
<pre><code>def removechars (s, chars)
    return s.translate(None, chars)
</code></pre>
<p>And I can use it like this :  </p>
<pre><code>print(removechars("The quick brown fox jumped over the sleazy dog!", "qot"))
</code></pre>
<p>It would give this output :   </p>
<pre><code>The uick brwn fx jumped ver the sleazy dg!     
</code></pre>
<p>How can I something similar in Kotlin?</p>

<div class="post-text" itemprop="text">
<p>I suggest using <a href="https://kotlinlang.org/api/latest/jvm/stdlib/kotlin.text/filter-not.html" rel="nofollow noreferrer"><code>filterNot()</code></a> in Kotlin:</p>
<pre><code>"Mississippi".filterNot { c -&gt; "is".contains(c)}
</code></pre>
<p>This should output <code>"Mpp"</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://kotlinlang.org/api/latest/jvm/stdlib/kotlin.text/-regex/" rel="nofollow noreferrer"><strong>Regex</strong></a> (the equivlant module in Python would be <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer"><strong>re</strong></a>):</p>
<pre><code>fun removeChars(s: String, c: String) = s.replace(Regex("[$c]"), "")

println(removeChars("The quick brown fox jumped over the sleazy dog!", "qot"))
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>The uick brwn fx jumped ver he sleazy dg!
</code></pre>
</div>
