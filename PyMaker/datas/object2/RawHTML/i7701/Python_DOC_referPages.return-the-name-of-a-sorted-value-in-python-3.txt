<div class="post-text" itemprop="text">
<p>Define a <strong>mapping</strong> between the names and the numbers:</p>
<pre><code>numbers = dict(dauntless=42, amity=0, abnegation=1, candor=4, erudite=2)
</code></pre>
<p>Then <strong>sort</strong>:</p>
<pre><code>d = sorted(numbers.items(), key=lambda x: x[1])
print(d)
# [('amity', 0), ('abnegation', 1), ('erudite', 2), ('candor', 4), ('dauntless', 42)]
</code></pre>
<p>To keep the result as a mapping/dictionary, call <a href="https://docs.python.org/3/library/collections.html#collections.OrderedDict" rel="nofollow noreferrer"><code>collections.OrderedDict</code></a> on the sorted list:</p>
<pre><code>from collections import OrderedDict

print(OrderedDict(d))
# OrderedDict([('amity', 0), ('abnegation', 1), ('erudite', 2), ('candor', 4), ('dauntless', 42)])
</code></pre>
