<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.chain.from_iterable" rel="nofollow noreferrer"><code>chain</code></a> and <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer"><code>groupby</code></a> from <a href="https://docs.python.org/3/library/itertools.html#module-itertools" rel="nofollow noreferrer"><code>itertools</code></a> module along with a <code>comprehension</code> like this example:</p>
<pre><code>from itertools import chain, groupby

a = [{'bus': 100, 'car': 10, 'lorry': 12, 'truck': 89},
 {'bus': 19, 'car': 12, 'lorry': 1, 'truck': 90},
 {'bus': 22, 'car': 14, 'lorry': 44, 'truck': 0}]

b = chain.from_iterable(k.items() for k in a)
final = dict(max(list(v), key=lambda x: x[1]) for _, v in groupby(sorted(b, key=lambda x:x[0]), lambda x: x[0]))

print(final)
</code></pre>
<p>Output:</p>
<pre><code>{'bus': 100, 'car': 14, 'lorry': 44, 'truck': 90}
</code></pre>
</div>
