<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/email.html" rel="noreferrer"><code>email</code></a> is a standard library package that <code>requests</code> indirectly depends on.  By having a file named <code>email.py</code> in the same directory as your script, you're preventing <code>requests</code> from using the real <code>email</code> package.  Rename, move, or delete <code>email.py</code>.</p>
