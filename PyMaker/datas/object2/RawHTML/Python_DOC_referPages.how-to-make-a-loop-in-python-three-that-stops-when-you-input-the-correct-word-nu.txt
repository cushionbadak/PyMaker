<p>The word you are looking for in place of <code>==</code> is <code>in</code> since <code>required_number</code> is a tuple you're looking to see if the input is <code>in</code> <code>required_number</code>. Also the correct syntax for the tuple would be using a comma not <code>or</code>.</p>
<p>I also would make <code>required_number</code> plural to be a more accurate description of what it holds, and you probably want to use integers and not strings.</p>
<pre><code>required_numbers = (8, 9)

while True:
    number = int(input("whats the number?"))
    if number in required_numbers:
        print("GOT IT")
        break #Stop asking
    else:
        print ("Wrong number try again")
</code></pre>

<div class="post-text" itemprop="text">
<ol>
<li><p><code>input</code> treats it as a <code>str</code> in <strong>Python 3.x</strong></p></li>
<li><p>Preferably use a <code>list</code> for the required numbers</p></li>
<li><p>Using <code>in</code> to check for the number in the <code>required_numbers</code></p></li>
<li><p>Put it in a try block to catch value error exceptions.</p></li>
</ol>
<p>Hence:</p>
<pre><code>required_number = [8,9]  # a list of integer types

while True:
    try:
        number = int(input("whats the number? "))   # Using `int` to convert the `str`
        if number in required_number:
            print ("GOT IT")
            break   # break out when the number is found
        else:
            print ("Wrong number try again")
    except ValueError:
        print("Invalid Input, Please enter an integer only.")
</code></pre>
<blockquote>
<p>Note: <code>==</code> determines if the values are <code>equal</code>, while <code>in</code> operator iterates over the list of elements and returns  <code>True</code> or <code>False</code>.</p>
</blockquote>
<p><strong>OUTPUT</strong>:</p>
<pre><code>whats the number? g
Invalid Input, Please enter an integer only.
whats the number? abc
Invalid Input, Please enter an integer only.
whats the number? 3
Wrong number try again
whats the number? 9
GOT IT
</code></pre>
