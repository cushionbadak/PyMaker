<p>The solution using <a href="https://docs.python.org/3/library/fileinput.html" rel="nofollow noreferrer"><strong><em>fileinput</em></strong></a> module (to modify the file in place):</p>
<pre><code>import fileinput

with fileinput.input(files="file.txt", inplace=True, backup="file.bak") as f:
    for l in f:
        if l.strip().startswith("@"):
            c = 2
            next_line = f.readline().strip()  # reading next line (line afer `@` line)
            if not next_line:   # if next line is empty
                while c:        # skip rest 2 lines
                    c -= 1
                    try:
                        next(f)
                    except StopIteration:
                        break
            else:
                # printing valid 4 lines
                print(l.strip())
                print(next_line.strip())
                while c:
                    c -= 1
                    try:
                        print(next(f).strip())
                    except StopIteration:
                        break
</code></pre>
