<div class="post-text" itemprop="text">
<p>Well, python refers the global library directory or local directory when trying to import a file. But for files in directories other than local you can either add file path to <code>sys.path</code> or create a <code>__init__.py</code> file in all directories including the parent directory <code>PyProjects</code>. Try this -</p>
<p><code>from modules.Pyrime import is_prime</code></p>
<p>So your directory will look something like this -</p>
<pre><code>PyProjects
    -- __init__.py
    -- PE010
        -- __init__.py
        -- PE010_Summation_of_Primes.py
    -- modules
        -- __init__.py
        -- Pyrime.py
</code></pre>
<p>Hope it helps.</p>
