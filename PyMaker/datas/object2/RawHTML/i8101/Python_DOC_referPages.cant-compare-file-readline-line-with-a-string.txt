<div class="post-text" itemprop="text">
<p>Instead of parsing the html by yourself, take a look at <a href="https://docs.python.org/3/library/html.parser.html" rel="nofollow noreferrer">this</a> built-in python html parser (or <a href="https://docs.python.org/2/library/htmlparser.html" rel="nofollow noreferrer">this</a> for python 2).</p>
<p>It will probably be easier and more robust than any code you will write by your own.</p>
<p>The example from the python documentation:</p>
<pre><code>from html.parser import HTMLParser

class MyHTMLParser(HTMLParser):
    def handle_starttag(self, tag, attrs):
        print("Encountered a start tag:", tag)

    def handle_endtag(self, tag):
        print("Encountered an end tag :", tag)

    def handle_data(self, data):
        print("Encountered some data  :", data)

parser = MyHTMLParser()
parser.feed('&lt;html&gt;&lt;head&gt;&lt;title&gt;Test&lt;/title&gt;&lt;/head&gt;'
        '&lt;body&gt;&lt;h1&gt;Parse me!&lt;/h1&gt;&lt;/body&gt;&lt;/html&gt;')
</code></pre>
<p>To use this example just add a member to the class which keeps track of the <code>content</code> you have.</p>
</div>
<div class="post-text" itemprop="text">
<p>Make sure 'line' object does not have spaces at the beginning and at the end. You can strip it and then compare.</p>
<p>Because, </p>
<p><code>x='&lt;TEXT&gt;'</code> is not equal to <code>y='&lt;TEXT&gt;    '</code> </p>
<p>Use <code>line = line.strip()</code> and then compare.</p>
