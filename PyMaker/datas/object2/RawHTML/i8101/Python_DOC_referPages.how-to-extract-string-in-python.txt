<div class="post-text" itemprop="text">
<p><code>s = string[0, p_index]</code> isn't a valid syntax in python, you should rather do:</p>
<pre><code>s = string[0:p_index]
</code></pre>
<p>Since an omitted first index defaults to zero, this returns the same result:</p>
<pre><code>s = string[:p_index]
</code></pre>
<p>I'd recommend reading <a href="https://docs.python.org/3/tutorial/introduction.html#strings" rel="nofollow noreferrer">this page</a> for reference on Python string's slicing and it's syntax in general.</p>
</div>
