<div class="post-text" itemprop="text">
<p>I am looking for a data structure which allows fast search and also preserves the order of the data it received?</p>
</div>
<div class="post-text" itemprop="text">
<p>Lists and tuples preserve order, dictionaries and set don't. An available ordered version of a dictionary is <a href="https://docs.python.org/3/library/collections.html#collections.OrderedDict" rel="noreferrer"><code>collections.OrderedDict</code></a>, which provides a faster search than a list.</p>
</div>
<div class="post-text" itemprop="text">
<p>Take a look at <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer">this</a></p>
<p>You have dictionaries and set:(<strong>unordered</strong>)</p>
<ul>
<li>Dictionary is <code>an unordered set of key: value pairs</code></li>
<li>set is <code>an unordered collection with no duplicate elements</code></li>
</ul>
<p>You can either use for <strong>preserving the order</strong>:</p>
<ul>
<li>list</li>
<li>tuple</li>
<li>collections.OrderedDict</li>
</ul>
<p>Choosing the data structure totally depends on your requirement.</p>
<ul>
<li><p>List and tuple hold a basic difference. <code>List is mutable and tuples is immutable</code></p></li>
<li><p>ordered dictionary is nothing but a key value pair.</p></li>
</ul>
<p>So you can choose accordingly!</p>
