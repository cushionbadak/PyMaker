<div class="post-text" itemprop="text">
<p>I have <strong>model Product</strong> where as primary key use code field which is <strong>UUID</strong>. I want take list of ids of all products.</p>
<p>I want list like this:</p>
<p><code>['14431ec4-3224-480f-85e2-b541891919b1', '6e54636b-7283-4504-a16f-f54d29db26f2']</code></p>
<p>I tried next code but it return me this:</p>
<pre><code>[UUID('14431ec4-3224-480f-85e2-b541891919b1'), UUID('6e54636b-7283-4504-a16f-f54d29db26f2')]
</code></pre>
<p><strong>models.py:</strong></p>
<pre><code>class Product(models.Model):
    code = models.UUIDField(_('Code'), primary_key=True, default=uuid.uuid4, editable=False)
</code></pre>
<p><strong>views.py:</strong></p>
<pre><code>products = Product.objects.all()
product_ids = []
for product in products:
    product_ids.append(product.code)
print(product_ids)
</code></pre>
<p><em>How to fix this problem?</em></p>
</div>
<div class="post-text" itemprop="text">
<p>You may use <a href="https://docs.djangoproject.com/en/1.11/ref/models/querysets/#django.db.models.query.QuerySet.values_list" rel="nofollow noreferrer"><code>.values_list()</code></a> with <code>flat=True</code> on your <a href="https://docs.djangoproject.com/en/1.11/ref/models/querysets/#queryset-api-reference" rel="nofollow noreferrer">Queryset</a> object to flatten the list of model's <code>code</code> and <a href="https://docs.python.org/3/library/functions.html#func-str" rel="nofollow noreferrer"><code>str</code></a> with <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map</code></a> to type-case each UUID object to string as:</p>
<pre><code>map(str, Product.objects.all().values_list('code', flat=True))
</code></pre>
<p>Please also see <a href="https://docs.python.org/3/library/uuid.html#example" rel="nofollow noreferrer">UUID examples</a> for the typical usage of the <a href="https://docs.python.org/3/library/uuid.html" rel="nofollow noreferrer">uuid</a> module.</p>
</div>
<div class="post-text" itemprop="text">
<p>This is just because it is a <code>UUID</code> field. If you definitely need them as strings, just call <code>str()</code>:</p>
<pre><code>product_ids = [str(product.code) for product in products]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can get string value using .hex from <code>uuid</code> as follows:</p>
<pre><code> ids = [product.uuid.hex for product in products]
</code></pre>
</div>
