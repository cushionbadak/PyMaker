<p>I have two lists, <code>t</code> and <code>e</code>, of the same size: to each element <code>t[i]</code> corresponds an element <code>e[i]</code>. I want to print each pair <code>t[i] e[i]</code>, but summing together elements <code>e[i]</code> and <code>e[i-1]</code> whenever the corresponding <code>t[i]</code> and <code>t[i-1]</code> are closer than 1. In this case, elements <code>e[5]</code> and <code>e[4]</code> should be summed up, since the corresponding <code>t[5]-t[4]&lt;1</code>. The point is, I want to REPLACE the 4th elements with the 4th+5th elements, without printing the 5th elements. My code is:</p>
<pre><code>t = [1, 5, 10, 16, 20, 21, 28, 33, 39, 42, 45, 50, 56]
e = ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J', 'K', 'L', 'M']

for j in range(1,12):
    if t[j]-t[j-1] &lt;= 1:
        print t[j-1], e[j-1]+e[j]
    else:
        if t[j] &gt; t[j-1]:
            print t[j-1], e[j-1]
</code></pre>
<p>But this gives me:</p>
<pre><code>1 A
5 B
10 C
16 D
20 EF
21 F
28 G
33 H
39 I
42 J
45 K
</code></pre>
<p>I don't want to print <code>21 F</code>, as it is already summed in the <code>20 EF</code>. How can I do that? Is there a way to increase the counter <code>i</code> at the end of the <code>if</code> condition? Thank you in advance.</p>

<p>Another solution is a while loop : </p>
<pre><code>t = [1, 5, 10, 16, 20, 21, 28, 33, 39, 42, 45, 50, 56]
e = ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J', 'K', 'L', 'M']

j = 1
while j &lt; len(t):
    if t[j]-t[j-1] &lt;= 1:
        print t[j-1], e[j-1]+e[j]
        j += 1
    else:
        if t[j] &gt; t[j-1]:
            print t[j-1], e[j-1]
    j += 1
</code></pre>

<pre><code>t = [1, 5, 10, 16, 20, 21, 28, 33, 39, 42, 45, 50, 56]
e = ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J', 'K', 'L', 'M']

skip = False
for j in range(len(t)):
    if not skip:
        if t[j]-t[j-1] &lt;= 1:
            print t[j-1], e[j-1]+e[j]
            skip = True
        else:
            if t[j] &gt; t[j-1]:
                print t[j-1], e[j-1]
    else:
        skip = False
