<div class="post-text" itemprop="text">
<p>'geetansh' is a string in python.
So</p>
<pre><code>if 'geeta' in 'geetansh':
</code></pre>
<p>that check whether the string 'geetansh' contain a string 'geeta'.</p>
<p>On the other hand, 'geetansh'.split() is a list in python. split() return a list, so 'geetanch'.split() return ['geetansh']. 
So</p>
<pre><code>if 'geeta' in 'geetansh'.split():
</code></pre>
<p>that check whether the list ['geetansh'] contain a item 'geeta'. The list ['geetansh'] only contain item 'geetansh'.</p>
</div>
<div class="post-text" itemprop="text">
<p>The basic difference is, str.split() returns a list!</p>
<p>When you compare <code>a in b</code>, the comparison is like,</p>
<ul>
<li><code>b</code> is a list, then a must be an item in b.</li>
<li><code>b</code> is a string, then a part of a is in b</li>
</ul>
<p>That is,</p>
<pre><code>&gt;&gt;&gt; 'geeta' in 'geetansh'
True
&gt;&gt;&gt; 'geeta' in 'geetansh'.split()
False
&gt;&gt;&gt; print 'geetansh'.split()
['geetansh']
&gt;&gt;&gt; print type('geetansh'.split())
&lt;type 'list'&gt;
&gt;&gt;&gt; print type('geetansh')
&lt;type 'str'&gt;
</code></pre>
</div>
