<div class="post-text" itemprop="text">
<p><code>str.strip</code> strips by character (keeps removing until a character not in the parameter is reached), and the problem is that you're including a whitespaces before <code>is</code> in your input:</p>
<pre><code>&gt;&gt;&gt; 'is it'.strip('is')
' it'
&gt;&gt;&gt; 'is it'.strip('- is')
't'
</code></pre>
<p>If what you actually want to do is <em>slice</em> off the substring from the start or end of the larger string, then you can use the following:</p>
<pre><code>def rcut(a, b):
    return a[:-len(b)] if a.endswith(b) else a

def cut(a, b):
    a = rcut(a, b)
    return a[len(b):] if a.startswith(b) else a

print cut('- is it available?', '- is')
# it available? 
</code></pre>

<div class="post-text" itemprop="text">
<p>Judging by this prompt in your program</p>
<pre><code>b=raw_input("enter the string to be stripped off")
</code></pre>
<p>you expect that <code>strip()</code> strips off substring prefixes and suffixes. It doesn't. <code>strip()</code> removes unwanted <em>characters</em>. </p>
<p>If you want to remove one instance of substring <code>b</code> from anywhere in string <code>a</code>:</p>
<pre><code>pieces = a.partition(b)
i = pieces[0] + pieces[2]
</code></pre>
<p>If, on the other hand, you want to remove only prefixes and sufffixes, like <code>strip()</code> does:</p>
<pre><code>i = a
if i.startswith(b):
    i = i[len(b):]
if i.endswith(b):
    i = i[:len(b)]
</code></pre>
<p>And if you want to remove <em>multiple occurrences</em> of the same prefix or suffix substring, again like <code>strip()</code> does, then substitute <code>while</code> for <code>if</code>.</p>
