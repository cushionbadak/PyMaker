<p>The block of <code>if str(varAble1_size).split(".")[1] == "0":</code> is empty, so you will need to add a <code>pass</code> statement after it. Keef Baker is also correct in spotting that the block for <code>else:</code> on line 37 is not properly indented.</p>

<pre><code>else:

<p>And as Andrew Kulpa noted, the block after </p>
<pre><code>if str(varAble1_size).split(".")[1] == "0":
</code></pre>
<p>is empty, so you either need to do something in that block, or add a <code>pass</code> statement.</p>
<p>Python does not use brackets but indentation for control flow. 
In your code, the Python interpreter sees an <strong>else</strong> but no statement for it, so it throws an error. 
See more about control flow here : <a href="https://docs.python.org/3/tutorial/controlflow.html" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/controlflow.html</a></p>
