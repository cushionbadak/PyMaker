<p>You can use <a href="https://docs.python.org/3/reference/datamodel.html#object.__str__" rel="nofollow noreferrer"><strong>str</strong></a> method to print the object value as you wish.</p>
<pre><code>class persondata:
    def __init__(self,Name,Age):
        self.Name=Name
        self.Age=Age
    def __str__(self):
        return(",".join([self.Name,str(self.Age)]))
try:
    person=[[],[]]
    person[0]=persondata('pav',26)
    person[1]=persondata('sunny',24)
    print(*person)
except Exception as e:
    print(str(e))
</code></pre>
<p>Output: </p>
<pre><code>pav,26 sunny,24
</code></pre>

<p>Your code is OK but persondata is an object and when you print it you see the memory location.</p>
<p>Here is a more pythonic version of your code</p>
<pre><code>class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __repr__(self):
        return repr([self.name, self.age])

people = []
people.append(Person('pav', 26))
people.append(Person('sunny', 24))
print(people)
</code></pre>
<p>Output:</p>
<pre><code>[['pav', 26], ['sunny', 24]]
</code></pre>
