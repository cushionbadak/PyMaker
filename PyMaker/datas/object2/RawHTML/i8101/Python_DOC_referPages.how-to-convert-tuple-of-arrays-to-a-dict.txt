<p>I have the following tuple:</p>
<pre><code>t = (array([0, 1, 2, 3], dtype=uint8), array([1568726,  346469,  589708,   91961]))
</code></pre>
<p>that I'd need to convert to a dict as follows:</p>
<pre><code>dict = {0: 1568726, 1: 346469, 2: 589708, 3: 91961}
</code></pre>
<p>I'm trying with </p>
<pre><code>d = dict((x, y) for x, y in t)
</code></pre>
<p>but it's not resolving the nesting of the tuple I have. Any suggestions?</p>
<p><a href="https://stackoverflow.com/questions/3783530">Another SO question</a> appears to be similar, but is not: its main issue is re transposing the dict elements, while this question focuses on how to join 2 arrays within a tuple into a dict.</p>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions#zip" rel="noreferrer"><code>zip</code></a> (to create key-value pairs) and <a href="https://docs.python.org/3/library/functions.html#dict" rel="noreferrer"><code>dict</code></a> (to convert the pairs to a dictionary):</p>
<pre><code>&gt;&gt;&gt; from numpy import array, uint8
&gt;&gt;&gt; t = (array([0, 1, 2, 3], dtype=uint8),
         array([1568726,  346469,  589708,   91961]))
&gt;&gt;&gt; dict(zip(*t))
{0: 1568726, 1: 346469, 2: 589708, 3: 91961}
</code></pre>
</div>
<span class="comment-copy"><code>d = dict(zip(*t))</code></span>
