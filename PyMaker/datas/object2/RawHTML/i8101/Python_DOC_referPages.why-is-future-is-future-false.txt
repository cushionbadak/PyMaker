<div class="post-text" itemprop="text">
<p>If you take a look at the <a href="https://docs.python.org/3.6/reference/expressions.html#is" rel="nofollow noreferrer">Python documentation</a>:</p>
<blockquote>
<h2>6.10.3. Identity comparisons</h2>
<p>The operators <code>is</code> and <code>is not</code> test for object identity: <code>x is y</code> is true if and only if <em>x</em> and <em>y</em> are the same object. Object identity is determined using the <a href="https://docs.python.org/3.6/library/functions.html#id" rel="nofollow noreferrer"><code>id()</code></a> function. <code>x is not y</code> yields the inverse truth value.</p>
</blockquote>
<p>Thus, <code>is</code> does not check if an object is an instance of a class but if two objects have the same identity. For that you use the builtin function <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer"><code>isinstance(instance, class)</code></a>:</p>
<pre><code>if isinstance(f, Future):
</code></pre>
</div>
<span class="comment-copy"><code>is</code> is used to check if two objects have the same identity, not if one is an instance of the other. Use <code>isinstance</code>.</span>
