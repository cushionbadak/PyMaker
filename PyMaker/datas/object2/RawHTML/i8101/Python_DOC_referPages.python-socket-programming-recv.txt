<p>You set a timeout in <code>socket.setdefaulttimeout(2)</code>, therefore it raises an exception.</p>
<p>Try this instead:</p>
<pre><code>import socket
s = socket.socket()
socket.setdefaulttimeout(10) #set timeout for ten seconds (can be whatever you want)
try:
    s.connect(("192.168.95.148",21))
except Exception as e:
    print("An error occured:\n{}".format(e))
</code></pre>
</div>
<span class="comment-copy">You are unable to connect on port 21 according to what you show here. You never get to the part where you read the socket. Make sure the server has something running on that port.</span>
