File parser.py contains following code and nothing more:</p>
<pre><code>class A:
    def __init__(self):
        self.x = 0
</code></pre>
<p>for file parser_tests.py I tried following combinations:
1)</p>
<pre><code>import unittest


class ParserTests(unittest.TestCase):
    def one(self):
        from parser import A
        x = A()
        self.assertTrue(True)
</code></pre>
<p>2)</p>
<pre><code>import unittest
from parser import A


class ParserTests(unittest.TestCase):

        self.assertTrue(True)
</code></pre>
<p>3)</p>
<pre><code>import unittest

<div class="post-text" itemprop="text">
<p><code>parser</code> is a python standard library module: <a href="https://docs.python.org/3/library/parser.html" rel="nofollow noreferrer">https://docs.python.org/3/library/parser.html</a></p>
<p>You'll have to rename your module to something else</p>
</div>
<span class="comment-copy">Where is your file located, and where are you running it from?  There is a builtin Python module called <code>parser</code>, which is likely what you're importing instead of your own file.</span>
