<div class="post-text" itemprop="text">
<p><s>Two additions to your code may give the expected result.</s></p>
<p><a href="http://docs.python.org/3/library/os.html#os.system" rel="nofollow noreferrer">os.system</a> expects a "command" for the operating system:</p>
<blockquote>
<p>Execute the command (a string) in a subshell.</p>
</blockquote>
<p>like <code>python3 myscript.py</code>. </p>
<p>However, since you are using a custom python programming environment, pass the path of the <a href="http://edupython.tuxfamily.org/" rel="nofollow noreferrer">EduPython</a>'s <code>python.exe</code> to <code>os.system</code>:</p>
<pre><code>from tkinter import *
import tkinter.filedialog
import os
import sys

root = Tk()
pyexec = sys.executable

def open():
    PathPy = tkinter.filedialog.askopenfilename(title="Open a file",filetypes=[('PYTHON file','.py')])
    os.system('%s %s' % (pyexec, PathPy))

B = Button(root, text="Open a file", command=open).pack()
root.mainloop()
</code></pre>
<p><s>Second, add the last line <code>root2.mainloop()</code> to your second script to make the second Tk window appear.</s></p>
<p>Hope this helps. </p>
</div>
