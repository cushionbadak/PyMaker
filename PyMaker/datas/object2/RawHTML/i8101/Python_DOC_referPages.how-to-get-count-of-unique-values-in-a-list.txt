<div class="post-text" itemprop="text">
<p>Given a list:
a = ['ed', 'ed', 'ed', 'ash', 'ash, 'daph']</p>
<p>I want to iterate through the list and get the top 2 most used names. So I should expect a result of ['ed', 'ash']</p>
<p>[Update]</p>
<p>how to go about this without using a library</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/collections.html#counter-objects" rel="nofollow noreferrer"><code>collections.Counter</code></a> has a <code>most_common</code> method:</p>
<pre><code>from collections import Counter

a = ['ed', 'ed', 'ed', 'ash', 'ash', 'daph']

res = [item[0] for item in Counter(a).most_common(2)]

print(res)  # ['ed', 'ash']
</code></pre>
<p>with <code>most_common(2)</code> i get the 2 most common elements (and their multiplicity); the list-comprehension then removes the multiplicity and just removes the item in your original list.</p>
</div>
<div class="post-text" itemprop="text">
<p>try:</p>
<pre><code>&gt;&gt;&gt; from collections import Counter

&gt;&gt;&gt; c = Counter(a)

&gt;&gt;&gt; c
Counter({'ed': 3, 'ash': 2, 'daph': 1})

# Sort items based on occurrence using most_common()
&gt;&gt;&gt; c.most_common()
[('ed', 3), ('ash', 2), ('daph', 1)]

# Get top 2 using most_common(2)
&gt;&gt;&gt; [item[0] for item in c.most_common(2)]
['ed', 'ash']

# Get top 2 using sorted
&gt;&gt;&gt; sorted(c, key=c.get, reverse=True)[:2]
['ed', 'ash']
</code></pre>
