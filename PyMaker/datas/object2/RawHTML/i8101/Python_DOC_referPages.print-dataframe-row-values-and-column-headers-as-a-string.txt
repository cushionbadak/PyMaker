<p>I am new to both python and pandas so please forgive me as I believe this should be easy. So if I have a pandas dataframe like this:</p>
<pre><code>    col1 col2 col3
0    13   23   93
1    11   99   15
2    88   25   55
</code></pre>
<p>and a user specifies they want row 1. How do I print a string like this:</p>
<p>col1 - 11, col2 - 99, col3 - 15</p>
<p>I know I can get at the row by using df.iloc[[1]] but from there I get lost. Thanks in advance</p>
</div>
<div class="post-text" itemprop="text">
<p>A possible solution:</p>
<pre><code>", ".join(["{} - {}".format(k, v) for k, v in df.iloc[1].to_dict().items()])
</code></pre>
<ol>
<li><a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.to_dict.html" rel="nofollow noreferrer"><code>.to_dict()</code></a> generates the dictionary <code>{'col1': 11, 'col2': 99, 'col3': 15}</code> from the row.</li>
<li>the <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> generates a list of "col_name - value" strings</li>
<li>the strings in the list are <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer">joined</a> together, glued by ", "</li>
</ol>
<p>Final result:</p>
<pre><code>col1 - 11, col2 - 99, col3 - 15
</code></pre>
</div>
