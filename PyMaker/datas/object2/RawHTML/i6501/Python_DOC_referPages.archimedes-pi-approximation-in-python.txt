<p><a href="https://docs.python.org/3/library/math.html#math.sin" rel="nofollow noreferrer"><code>math.sin</code></a> expects angles to be specified in radians.  </p>
<pre><code>&gt;&gt;&gt; print(math.sin.__doc__)
sin(x)

Return the sine of x (measured in radians).
</code></pre>
<p>You are specifying it in degrees.  Specify the angle in radians instead:</p>
<pre><code>value = math.pi / sides
</code></pre>
<p>Or, if you don't want to use <code>math.pi</code> so explicitly, use the helper function to convert units:</p>
<pre><code>value = math.radians(360/(2*sides))
</code></pre>
