<p>I'm trying to match numbers that have 5 digits using regex in python. This is my code:</p>
<pre><code> import re
 matchstring = "These are 10101 random numbers 34345."
 regex_string = r'(?&lt;!\d)\d{5}(?!\d)'
 all_matches = re.findall(matchstring, regex_string) 
</code></pre>
<p>I have no idea why I get 0 matches when I should get 2. I've tried using this regex on 2 python validator sites and on both sites these work. Any idea why in my code I get an empty list?</p>
</div>
<div class="post-text" itemprop="text">
<p>Function <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer">re.findall</a> accepts arguments in order: pattern, then string:</p>
<pre><code>re.findall(regex_string, matchstring)
['10101', '34345']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can try this:</p>
<pre><code>import re
matchstring = "These are 10101 random numbers 34345."
final_numbers = [i for i in re.findall("\d+", matchstring) if len(i) == 5]
</code></pre>
<p>Output:</p>
<pre><code>['10101', '34345']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>import re
matchstring = "These are 10101 random numbers 34345."
regex_string = r"\b\d{5}\b"
re.findall(regex_string , matchstring) 
</code></pre>
<p>output:</p>
<pre><code>['10101', '34345']
</code></pre>
