<p>You should use <code>deep copy</code> to copy objects and create a new memory value and a new pointer :</p>
<pre><code>import copy
theta_old = copy.deepcopy(theta_new)
theta_new[j] = theta_new[j] - alpha*deriv
...
</code></pre>
