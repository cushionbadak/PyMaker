<div class="post-text" itemprop="text">
<p>In python, you can use <a href="https://docs.python.org/2/library/stdtypes.html#str.split" rel="nofollow noreferrer">split() function</a> which returns array of elements splitted by the character you specify. Then you get the first element: </p>
<pre><code>yourString = "TEST/0001"

yourString.split("/")[0]

&gt;&gt;&gt; 'TEST'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I would go for the <code>split</code> solution, but in case you are looking for a more complete and at the same time simple solution (assuming you know regexes, which anyway should belong to any programmer's knowledge) then you can use a few shortcut methods from the standard-library <a href="https://docs.python.org/3/library/re.html#module-re" rel="nofollow noreferrer">re module</a>.</p>
<p>An example with your same data would be:</p>
<pre><code>import re

lines = ["TEST/1000", "TEST2/1000", "TEST3/1000"]
pattern = "TEST\d*(?=/)"    # Take any string beginning with TEST, followed by 0 or more digits and a / character

for line in lines:
    match = re.match(pattern, line)

    if match is not None:
        print(match.group(0))    # match.group(0) returns the whole matched string, and not a part of it
    else:
        print("No match for %s" % line)
</code></pre>
<p>With my setup, running this script inside a <code>test.py</code> file yields:</p>
<pre><code>None@vacuum:~$ python3.6 ./test.py 
TEST
TEST2
TEST3
</code></pre>
