<div class="post-text" itemprop="text">
<p>How to find number of permutations when items are repeated?
For example, for the string <code>aab</code> I want the following output: <code>aab</code>, <code>aba</code>, <code>baa</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the standard-library <a href="https://docs.python.org/3/library/itertools.html" rel="nofollow noreferrer">itertools</a> module:</p>
<pre><code>from itertools import permutations


iterable = "aab"
print(set(permutations(iterable, len(iterable))))
</code></pre>
<p>By executing that code, I get the following output:</p>
<pre><code>None@vacuum:~$ python3.6 ./test.py 
{('b', 'a', 'a'), ('a', 'b', 'a'), ('a', 'a', 'b')}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The following should work:</p>
<pre><code>import itertools
set(itertools.permutations('aab'))
</code></pre>
<p>The <a href="https://docs.python.org/3.6/library/itertools.html#itertools.permutations" rel="nofollow noreferrer"><code>permutations</code></a> will produce all permutations, but it will repeat some of it's output because it can't differentiate between <code>a1 a2 b</code>  and <code>a2 a1 b</code>. Here <code>a1</code> is the first occurrence of <code>a</code>, and <code>a2</code> the second. So you need to just keep the unique elements, and a <code>set</code> is a good way to do that, as the constructor transforms the input iterable into a unique collection.</p>
</div>
