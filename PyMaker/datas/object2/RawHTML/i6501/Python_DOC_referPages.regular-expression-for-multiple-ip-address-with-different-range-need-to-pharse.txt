<p>Simple regex to match exactly that number of digits in ip is following:</p>
<p><code>\d{2}(\.\d{1}){3}</code></p>
<p><a href="https://pythex.org/?regex=%5Cd%7B2%7D(%5C.%5Cd%7B1%7D)%7B3%7D&amp;test_string=10.1.1.1%0A192.168.1.1&amp;ignorecase=0&amp;multiline=0&amp;dotall=0&amp;verbose=0" rel="nofollow noreferrer">Pythex</a> for that regular expression</p>

<p>You should provide little more information on which pattern you are looking in ip address?</p>
<p>Basis on your current need you can use this :</p>
<pre><code>import re
pattern=r'\d{2}(?=[.]).\d{1}([.]\d{1})(\1)'

string="IP = [10.0.1.1 , 192.168.1.1, 200.1.1.1]"

match=re.search(pattern,string)

print(match.group())
</code></pre>
<p>output:</p>
<pre><code>10.0.1.1
</code></pre>
