<p>You can use this to sum up all elements of a list except the first one:</p>
<pre><code>sum(a_list[1:])
</code></pre>
<p>If the elements of the list are strings you can do:</p>
<pre><code>sum(map(int, a_list[1:]))
</code></pre>
<hr/>
<ul>
<li><p><code>a_list[1:]</code> this is called slicing, have a look <a href="https://stackoverflow.com/a/509295/6391362">here for an explanation of the syntax</a>.</p></li>
<li><p><code>map(int, a_list)</code> applies the function <code>int</code> (cast to integer) to all elements of <code>a_list</code>. <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer">Link to the docs</a>.</p></li>
</ul>
