<div class="post-text" itemprop="text">
<p>This is from <a href="https://docs.python.org/3/library/stdtypes.html#typesseq-range" rel="nofollow noreferrer">Python 3 docs</a>:</p>
<blockquote>
<p>The arguments to the <strong>range</strong> constructor must be <strong>integers</strong> (either
  built-in int or any object that implements the <code>__index__</code> special
  method).</p>
</blockquote>
<p>So, you need to use <strong>integers</strong> for the range arguments.</p>
<p>I don't know what you exactly need in your application, but changing these lines will fix the error:</p>
<pre><code>...
cell_width = int(sheet_size[0] / horiz_cells)
cell_height = int(sheet_size[1] / vert_cells)
...
</code></pre>
