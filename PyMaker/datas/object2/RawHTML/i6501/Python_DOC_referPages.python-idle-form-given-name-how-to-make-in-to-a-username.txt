<p>What you are trying to accomplish is fairly simple, and you get pretty close to it in the code in your question, even though your for loop syntax is off. Here is what you need:</p>
<pre><code>names = ['kyrie andrew irving', 'saint christopher']

for item in names:
    name_sep = item.split(' ')
    username = ""
    for i in name_sep[0:-1]:
        username += i[0]
    username += name_sep[-1]
    print(username)
</code></pre>
<p>Let me walk you through it, Line by Line:</p>
<ol>
<li>Start out with a list of names</li>
<li>Iterate over the list</li>
<li>Split the name into a list, using space to determine where to make a new element</li>
<li>Assign an empty string to be our user name</li>
<li>Iterate over all the separated names except the last one</li>
<li>Add their first letter to the username</li>
<li>Add the last item in the separated names to the username</li>
<li>Print the username</li>
</ol>

<pre><code>def make_username(name):
    return ''.join(x[0].lower() if i != len(name.split()) - 1 else x.lower() for i, x in enumerate(name.split()))

&gt;&gt;&gt; make_username('saint christopher')
'schristopher'
&gt;&gt;&gt; make_username('kyrie andrew irving')
'kairving'
&gt;&gt;&gt; make_username('holly mother of god')
'hmogod'
</code></pre>
<p>...this will make a username for a name consisting of arbitrary amount of subnames. It takes first letter of every subname, except the last one where it takes the whole subname.</p>
