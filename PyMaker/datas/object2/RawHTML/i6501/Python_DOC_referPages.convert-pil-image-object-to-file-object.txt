<div class="post-text" itemprop="text">
<p>Let's look at what a file object is.</p>
<pre><code>with open('test.txt', 'r') as fp:
    print(fp)
    # &lt;_io.TextIOWrapper name='test.txt' mode='r' encoding='UTF-8'&gt;
</code></pre>
<p><a href="https://docs.python.org/3/library/io.html" rel="nofollow noreferrer">https://docs.python.org/3/library/io.html</a> has more on the subject as well.</p>
<p>I suspect that for your purposes, it would be enough to have a BytesIO object.</p>
<pre><code>import io
from PIL import Image
im = Image.new("RGB", (100, 100))
b = io.BytesIO()
im.save(b, "JPEG")
b.seek(0)
</code></pre>
<p>But if you really want the same object, then -</p>
<pre><code>fp = io.TextIOWrapper(b)
</code></pre>
