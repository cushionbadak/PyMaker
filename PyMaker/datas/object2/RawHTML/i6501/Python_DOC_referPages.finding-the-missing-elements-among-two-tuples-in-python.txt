<div class="post-text" itemprop="text">
<p>I have two tuples <code>a = (('1',), ('2',), ('3',),)</code> and <code>b = (('1',), ('3',),)</code>. I need to get the result as <code>(('2',),)</code> since, 2 is the element that is present in <code>a</code> and not in <code>b</code>. </p>
<p>I referred this <a href="https://stackoverflow.com/questions/642763/find-intersection-of-two-lists">Find intersection of two lists?</a> and <a href="https://stackoverflow.com/questions/20179519/is-there-a-way-to-get-the-difference-and-intersection-of-tuples-or-lists-in-pyth">Is there a way to get the difference and intersection of tuples or lists in Python?</a> for getting an idea, but these are for lists and not for tuples. I am not able to use intersection() for tuples.</p>
<p>Is there a way I can get <code>a-b</code> in python tuples?</p>
</div>
<div class="post-text" itemprop="text">
<p>Convert to <code>set</code> then you can get the difference, then convert it back to <code>tuple</code> using <code>tuple()</code> function:</p>
<pre><code>a = (('1',), ('2',), ('3',),)
b = (('1',), ('3',),)

result = tuple(set(a) - set(b))
print(result)
</code></pre>
<p>Running example: <a href="https://repl.it/M1FD/1" rel="nofollow noreferrer">https://repl.it/M1FD/1</a></p>
<hr/>
<p>If you want the <a href="http://en.wikipedia.org/wiki/Symmetric_difference" rel="nofollow noreferrer">Symmetric Difference</a>, elements in either set but not in the intersection:</p>
<pre><code>set(a) ^ set(b)
</code></pre>
<p>or:</p>
<pre><code>set(a).symmetric_difference(set(b))
</code></pre>
<p>Running example: <a href="https://repl.it/M1FD/2" rel="nofollow noreferrer">https://repl.it/M1FD/2</a></p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/tutorial/datastructures.html#sets" rel="nofollow noreferrer">Sets</a> are very useful for this.</p>
<p>If you're only looking for elements that are in <code>a</code> but not in <code>b</code>:</p>
<pre><code>set(a) - set(b)
</code></pre>
<p>If you're looking for elements that are in either one of the tuples, but not the other:</p>
<pre><code>set(a) ^ set(b)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can still use sets as described in the linked answer:</p>
<pre><code>In [1]: a = (('1',), ('2',), ('3',),)
In [2]: b = (('1',), ('3',),)
In [3]: set(a).intersection(set(b))
Out[3]: {('1',), ('3',)}
In [4]: set(a).difference(set(b))
Out[4]: {('2',)}
</code></pre>
</div>
