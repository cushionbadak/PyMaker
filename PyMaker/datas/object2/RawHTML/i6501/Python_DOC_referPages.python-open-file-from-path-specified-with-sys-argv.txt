<p>I want to open a csv-file with python from a path that is specified in sys.argv. The name of the file is 'file.out' and I want to open it from the scriptlocation as specified in sys.argv[2]. 
However, I do not know how to specify the scriptlocation in the pd.read_csv command. I tried it as follows, but that does not work. What is the problem? </p>
<p>My code is as follows</p>
<pre><code>outputfolder = sys.argv[1]
scriptlocation = sys.argv[2]

df = pd.read_csv(open(scriptlocation('file.out', 'r')), header=None, delim_whitespace=True)
</code></pre>

<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>import os

fn = os.path.join(os.path.dirname(sys.argv[2]), 'file.out')

df = pd.read_csv(fn, header=None, delim_whitespace=True)
</code></pre>
<p>If you use Python v3.4+ and Pandas v0.18.1+ you can use <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow noreferrer">pathlib</a>:</p>
<p>Demo:</p>

<div class="post-text" itemprop="text">
<p>This is not a <code>pandas</code> issue. What you need is to make a filepath based on a root folder (<code>scriptlocation</code> if I understand correctly) and a filename. You will then pass that constructed filepath to <code>pd.read_csv()</code>.
So you are looking for <a href="https://docs.python.org/2/library/os.path.html#os.path.join" rel="nofollow noreferrer"><code>os.path.join()</code></a>:</p>
<pre><code>output_fn = os.path.join(scriptlocation , 'file.out')
df = pd.read_csv(output_fn, header=None, delim_whitespace=True)
</code></pre>
</div>
