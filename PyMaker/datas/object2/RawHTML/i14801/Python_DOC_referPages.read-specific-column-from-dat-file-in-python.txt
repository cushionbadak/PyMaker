<pre><code>7522126 0   0   0   0   0   0   -419.795    -186.24 1852.86 0.134695    -0.995462   -2.53153
7825452 0   0   0   0   0   0   -419.795    -186.24 1852.86 0.134695    -0.995462   -2.53153
8073799 0   0   0   0   0   0   -345.551    -140.711    1819.04 -0.0220266  -0.85992    -2.29598
</code></pre>
<p>The values are each separated by a tab.</p>
<p>I want to extract the value in e.g the 8th column for every single line, and save it to an array. So the output should be this:</p>
<pre><code>-419.795
-419.795
-345.551
</code></pre>
<p>What's the easiest way to accomplish this?</p>

<div class="post-text" itemprop="text">
<pre><code>with open('results.dat') as f:
    [line.split()[7] for line in f]  
</code></pre>
<p>or define a function,  </p>
<pre><code>get_col = lambda col: (line.split('\t')[col-1] for line in open('results.dat'))  
</code></pre>
<p>Now call the function with desired column number. <code>get_col(8)</code> gives 8th column data. To store it in array,  </p>
<pre><code>array.array('d',map(float,get_col(8)))
</code></pre>

<pre><code>import csv
with open('file') as f:
    reader = csv.reader(f, delimiter="\t")
    for line in reader:
        print(line[7])
</code></pre>

<div class="post-text" itemprop="text">
<p>first of all read the file (result.dat) file in a file object</p>
<pre><code>file = open('result.dat')
</code></pre>
<p>now create an empty list</p>
<pre><code>lst = []
</code></pre>
<p>loop through each line of the file</p>
<pre><code>for line in file:
    lst += [line.split()]
</code></pre>
<p>now lst is a list of list , where each inner list is a instance (row ) of the result.dat</p>
<p>now you can extract any column (in your case it is 8th)
apply list comprehension for this</p>
<pre><code>column8 = [x[7] for x in lst]
</code></pre>
