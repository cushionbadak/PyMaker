<div class="post-text" itemprop="text">
<p>You probably want to use the <code>search</code> method. <code>search</code> <a href="https://docs.python.org/3/howto/regex.html#performing-matches" rel="nofollow">"scans through a string, looking for any location where this RE matches."</a></p>
<pre><code>import re

hasSpace = re.compile(' ')

string = 'hello world'

if(hasSpace.search(string)):
    print("Found a space") # gets printed
else:
    print("No space")

string = 'helloworld'

if(hasSpace.search(string)):
    print("Found a space")
else:
    print("No space") # gets printed
</code></pre>
<p>What you were trying to use is <code>match</code>, which <a href="https://docs.python.org/3/howto/regex.html#performing-matches" rel="nofollow">"determines if the RE matches at the beginning of the string."</a></p>
</div>
