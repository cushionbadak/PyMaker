<p>To convert a literal backslash followed by a literal 'n' to a newline, you can decode the string using the <code>string_escape</code> encoding:</p>
<pre><code>&gt;&gt;&gt; s = r'what\nwhat\nwhat'
&gt;&gt;&gt; print(s)
what\nwhat\nwhat
&gt;&gt;&gt; print(s.decode('string_escape'))
what
what
what
&gt;&gt;&gt;
</code></pre>
<p>So your function would be:</p>
<pre><code>def printToFile(data):
    rtl.write(data.decode('string_escape'))
</code></pre>
