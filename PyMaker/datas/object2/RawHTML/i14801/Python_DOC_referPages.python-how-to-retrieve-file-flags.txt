<div class="post-text" itemprop="text">
<p>File flags are made available via <code>os.stat()</code>.</p>
<p>The POSIX implementation of python's os module returns a <code>stat_result</code> object that has an <code>st_flags</code> field:</p>
<pre><code>import os
st = os.stat(filename)
print(st.st_flags)
</code></pre>
<p>The possible flag bits are available in the <code>stat</code> module as</p>
<pre><code>SF_*
UF_*
</code></pre>
<p>constants.</p>
