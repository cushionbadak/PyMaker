<div class="post-text" itemprop="text">
<p>You'd use <code>initialString.index(element)</code> (so reversed) instead; the list can tell you what index it has <code>element</code> stored in; <code>element</code> has no knowledge of the list.</p>
<p>You should really use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow"><code>enumerate()</code></a> here however, to add indices to the loop:</p>
<pre><code> for i, element in enumerate(initialString):
     if element == word:
         print(i)
</code></pre>
<p>If all you wanted to know was the index of <code>word</code>, however, it is simpler still to just use:</p>
<pre><code>print(initialString.index(word))
</code></pre>
</div>
