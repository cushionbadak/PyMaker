<div class="post-text" itemprop="text">
<p>from <a href="https://stackoverflow.com/questions/899103/python-write-a-list-to-a-file">this</a> post, I've been using the following code to write a list/array to a file:</p>
<pre><code>with open ("newhosts.txt",'w') as thefile:
    for item in hostnameIP:
      thefile.write("%s\n" % item)
</code></pre>
<p>where <code>hostnameIP</code> is: </p>
<p><code>[['localhost', '::1'], ['localhost', '::1'], ['localhost', '::1']]</code></p>
<p>In the file, I get the output:</p>
<pre><code>['localhost', '::1']
['localhost', '::1']
['localhost', '::1']
</code></pre>
<p>when I need to to say </p>
<pre><code>localhost, ::1
localhost, ::1
localhost, ::1
</code></pre>
<p>What would be the best way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use:</p>
<pre><code>with open ("newhosts.txt", "w") as thefile:
    for item in hostnameIP:
        thefile.write("%s\n" % ", ".join(item))
</code></pre>
<p>This way each part of item will be printed with an ", " as seperator.</p>
<p>But if you want to make the code even shorter, you can also join each item with a newline:</p>
<pre><code>with open ("newhosts.txt", "w") as thefile:
    thefile.write("\n".join(map(", ".join, hostnameIP)))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>with open ("newhosts.txt",'w') as thefile:
    for item in hostnameIP:
      thefile.write("%s, %s\n" % (item[0], item[1]))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I would use the csv module simply calling writerows on your list of lists:</p>
<pre><code>import csv
lines = [['localhost', '::1'], ['localhost', '::1'], ['localhost', '::1']]
with open ("newhosts.txt",'w') as f:
    wr = csv.writer(f)
    wr.writerows(lines)
</code></pre>
<p>Output:</p>
<pre><code>localhost,::1
localhost,::1
localhost,::1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>From what I can see you have a list with lists as elements. This is why you get the result that you get. Try the following code (see the minor change on the third line) and you will get the wanted result.</p>
<pre><code>with open ("newhosts.txt",'w') as thefile:
    for item in hostnameIP:
      thefile.write("%s\n" % ', '.join(item))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are currently printing string representations of the lists to the file.  Since you are only interested in the lists' items, you can use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format</code></a> and argument unpacking to extract them:</p>
<pre><code>thefile.write("{}, {}\n".format(*item))
</code></pre>
