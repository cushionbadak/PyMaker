<p>I am writing a raingauge precipitation calculator based in the radius of the raingauge. When I run my script, I have this error message:</p>
<pre><code>Type de raingauge radius [cm]: 5.0
Traceback (most recent call last):
  File "pluviometro.py", line 27, in &lt;module&gt;
    area_bocal = (pi * (raio_bocal * raio_bocal)) # cm.cm
TypeError: can't multiply sequence by non-int of type 'str'
</code></pre>
<p>I am using 
<code>raio_bocal = input("Type de raingauge radius [cm]:")</code>
for data input. When using <code>Python2</code> the typecasting was automatic.</p>
<p>How can I have a <code>float</code> entered value using <code>python3</code>? </p>
</div>
<div class="post-text" itemprop="text">
<p>As mentioned in the <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow">docs</a></p>
<blockquote>
<p>The function then reads a line from input, <strong>converts it to a string</strong> (stripping a trailing newline), and returns that</p>
</blockquote>
<p>So you need to type cast it to <code>float</code> explicitly </p>
<pre><code>raio_bocal = float(input("Type de raingauge radius [cm]:"))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to cast to float, input returns a string in python3:  </p>
<pre><code>float(input("Type de raingauge radius [cm]:"))
</code></pre>
<p>Probably safer use a while loop with a try/except when casting input.</p>
<pre><code>while True:
   inp = input("Type de raingauge radius [cm]:")
   try:
       raio_bocal  = float(inp)
       break
   except ValueError:
       print("Invalid input")
</code></pre>
