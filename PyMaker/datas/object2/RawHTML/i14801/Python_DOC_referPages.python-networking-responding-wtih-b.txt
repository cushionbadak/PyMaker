<div class="post-text" itemprop="text">
<p>When you call <a href="https://docs.python.org/3/library/socket.html#socket.socket.recv" rel="nofollow">socket.recv() in Python 3</a> it returns a bytes object, not a normal string. You can convert it to a normal string as follows:</p>
<p>data.decode('utf-8')</p>
