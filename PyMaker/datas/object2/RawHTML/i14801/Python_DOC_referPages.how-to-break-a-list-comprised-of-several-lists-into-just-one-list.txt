<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow"><code>chain</code></a> function to chain your lists. It returns iterator, so to convert it to list just pass it as constructor argument.</p>
<pre><code>from itertools import chain

lists = [[1, 2], [3, 4, 5], [6]]
elements = list(chain(*lists))

print(elements)
</code></pre>
<p>Above code prints <code>[1, 2, 3, 4, 5, 6]</code>.</p>
