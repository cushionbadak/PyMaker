<p>Just found it in <a href="https://flask-admin.readthedocs.io/en/latest/introduction/#working-with-the-built-in-templates" rel="nofollow noreferrer">flask-admin docs</a>.  I had to  create <code>templates/edit_user.html</code> and <code>templates/create_user.html</code>. For <code>list_users</code> is also the same, theres is an example in the docs.</p>
<p>In <code>edit_user.html</code></p>
<pre><code>{% extends 'admin/model/edit.html' %}

{% block body %}
    &lt;h1&gt;User Edit View&lt;/h1&gt;
    {{ super() }}
{% endblock %}
</code></pre>
<p>In <code>create_user.html</code></p>
<pre><code>{% extends 'admin/model/create.html' %}

{% block body %}
    &lt;h1&gt;Create View&lt;/h1&gt;
    {{ super() }}
{% endblock %}
</code></pre>
<p>and then add this to the User model View:</p>
<pre><code>class UserView(ModelView):
    edit_template = 'edit_user.html'
    create_template = 'create_user.html'


admin.add_view(UserView(User, db.session))
</code></pre>
