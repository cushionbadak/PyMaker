<p>I want to know how can I access a class attribute using a string. E.g</p>
<pre><code>class Test:
    def __init__(self):
        # defined self.name here
        self.name.person = 1
</code></pre>
<p>If I have the program:</p>
<pre><code>a = "person" 
b = Test()
</code></pre>
<p>How can I print self.name.person using the variable a?</p>
<pre><code>print(b.name.a)
</code></pre>
<p>Many thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>Python has <code>getattr()</code> as built-in function:</p>
<pre><code>print(getattr(b.name, a))
</code></pre>
</div>
<span class="comment-copy">of course self.name is already defined</span>
