<div class="post-text" itemprop="text">
<p>convert_epub_to_lines returns an iterator to lines, which you need to iterate one by one to get.</p>
<p>Instead, you can get all lines with "convert", see in the documentation of the library:</p>
<p><a href="https://pypi.org/project/epub-conversion/" rel="nofollow noreferrer">https://pypi.org/project/epub-conversion/</a></p>

<p>What about <a href="https://github.com/aerkalov/ebooklib" rel="nofollow noreferrer">https://github.com/aerkalov/ebooklib</a></p>
<blockquote>
<p>EbookLib is a Python library for managing EPUB2/EPUB3 and Kindle
  files. It's capable of reading and writing EPUB files programmatically
  (Kindle support is under development).</p>
<p>The API is designed to be as simple as possible, while at the same
  time making complex things possible too. It has support for covers,
  table of contents, spine, guide, metadata and etc.</p>
</blockquote>
<pre><code>import ebooklib
from ebooklib import epub

book = epub.read_epub('test.epub')

for doc in book.get_items_of_type(ebooklib.ITEM_DOCUMENT):
    print doc
</code></pre>
