<div class="post-text" itemprop="text">

'March'
</code></pre>
<p>for more info, and a full list of formatting codes, see <a href="http://docs.python.org/library/datetime.html#strftime-strptime-behavior" rel="noreferrer">the python <code>datetime</code> docs</a></p>

<div class="post-text" itemprop="text">
<p>For English only, you can use the datetime string formatting method of Python, e.g.</p>
<pre><code>&gt;&gt;&gt; today.strftime('%B')
'March'
</code></pre>
<p>You can also use Django methods, which will return the name <strong>in the current activated language</strong>.</p>
<p>In a Django template you can also use:</p>
<pre><code>{{ a_date|date:'F' }}
</code></pre>
<p>In a Django view function:</p>
<pre><code>from django.template.defaultfilters import date
date(a_date, 'F')
</code></pre>
<p>You can test the later in the Django shell (<code>python manage.py shell</code>) for e.g. Spanish with:</p>
<pre><code>In [1]: from django.utils.translation import get_language, activate

In [2]: activate('es')

In [3]: get_language()
Out[3]: 'es'

In [4]: import datetime

In [5]: today = datetime.date.today()

In [6]: from django.template.defaultfilters import date

In [7]: date(today, 'F')
Out[7]: u'Septiembre'
</code></pre>

<p>I solved the same by registering a custom template tag to help me apply it on any template page that requires the month name.  </p>
<p>templatetags/etras.py</p>
<pre><code>from django import template
import calendar

register = template.Library()

@register.filter
def month_name(value):
    return calendar.month_name[value]
</code></pre>
<p>Usage in template:</p>
<pre><code>{% load extras %}
{{ month_figure|month_name }}
</code></pre>
