<div class="post-text" itemprop="text">
<p>Is it possible to create a UNIX timestamp in Python (number of seconds) with only day, month and year from a date object? I'm essentially looking for what the timestamp would be at midnight (hour, minute and second would be 0).</p>
<p>Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; import time
&gt;&gt;&gt; import datetime
&gt;&gt;&gt; dt = datetime.datetime.strptime('2012-02-09', '%Y-%m-%d')
&gt;&gt;&gt; time.mktime(dt.timetuple())
1328774400.0
</code></pre>
<p><strong>--OR--</strong></p>
<pre><code>&gt;&gt;&gt; dt = datetime.datetime(year=2012, month=2, day=9)
&gt;&gt;&gt; time.mktime(dt.timetuple())
1328774400.0
</code></pre>
<p>For other Time types like Hour and Second go here:
<a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="noreferrer">https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior</a></p>
</div>
<div class="post-text" itemprop="text">
<p>You can also use <a href="http://docs.python.org/library/datetime.html#datetime.datetime.combine" rel="nofollow">datetime.combine()</a></p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; import time
&gt;&gt;&gt; date1 = datetime.date(year=2012,day=02,month=02)
&gt;&gt;&gt; date2 = datetime.datetime.combine(date1,datetime.time())
&gt;&gt;&gt; date2
datetime.datetime(2012, 2, 2, 0, 0)
&gt;&gt;&gt; tstamp = time.mktime(date2.timetuple())
&gt;&gt;&gt; tstamp
1328121000.0
</code></pre>
<p>The result depends on local timezone (in this case IST). I hope somebody can point me on how to get GMT result</p>

<div class="post-text" itemprop="text">
<p>You could do it by getting the number of seconds between the date and the epoch (1 January 1970):</p>
<pre><code>from datetime import datetime

epoch = datetime(1970, 1, 1)
d = datetime(2012, 2, 10)

print (d - epoch).total_seconds()
</code></pre>
</div>
<span class="comment-copy">NB, this will give you the localtime, not the UTC timestamp</span>
