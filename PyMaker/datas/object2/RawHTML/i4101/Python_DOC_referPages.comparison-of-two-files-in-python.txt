<p>As <a href="https://stackoverflow.com/users/6120211/seer-the">Seer.The</a> mentioned above, you can use <code>difflib</code>.</p>
<pre><code>import difflib

# Read the files
f = open('textfile1.txt', 'r').readlines()
list1 = []
for n in f:
    text = n.rstrip().split(" ")
    list1.append(text)


f = open('textfile2.txt', 'r').readlines()
list2 = []
for n in f:
    text = n.rstrip().split(" ")
    list2.append(text)

# Get the output
for ii in range(len(list1)):
    for jj in range(len(list1[0])):
        output_list = [li[-1] 
                       for li in list(difflib.ndiff(list1[ii][jj], list2[ii][jj]))
                       if "-" in li]
        if output_list == []:
            output_list = ["no difference"]
        print "{} {} {}".format(list1[ii][jj], list2[ii][jj], output_list[0])
</code></pre>
<p>The output should look like:</p>
<pre><code>goerge goergy e
hyder hydel r

</code></pre>
