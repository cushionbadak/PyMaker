<div class="post-text" itemprop="text">
<p>You can use the enumerate object. Enumerate will pair each item in a collection with its index in that collection. For example:</p>
<pre><code>enumerate(['a', 'b', 7])
</code></pre>
<p>will return an enumerate object that when casted to a list like this:</p>
<pre><code>list(enumerate(['a', 'b', 7]))
</code></pre>
<p>will return the list</p>
<pre><code>[(0, 'a'), (1, 'b'), (2, 7)]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Another possible solution is using a <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> and <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer"><code>range</code></a> function:</p>
<pre><code>&gt;&gt;&gt; f = ['a', 'b', 7]
&gt;&gt;&gt; list(zip(range(1, len(f)+1), f))
[(1, 'a'), (2, 'b'), (3, 7)]
</code></pre>
