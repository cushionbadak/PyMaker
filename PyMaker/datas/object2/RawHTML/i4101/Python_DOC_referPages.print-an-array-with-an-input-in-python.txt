<p>I want to print an array using an input. This is an example of what I'm trying to do with a lot less arrays:</p>
<pre><code># Input
x = input()

# Array
b = [1, 2, 3, 4, 5]
c = [2, 3, 4]
d = [1, 2, 3]
#Print Array
print(x)
</code></pre>
<p>If someone is to input b it would print:</p>
<pre><code>'b'
</code></pre>
<p>Instead of:</p>
<pre><code>[1, 2, 3, 4, 5]
</code></pre>
<p>Is there anyway I can fix this without using an if statement for every array?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use a dictionary data type:</p>
<pre><code># Input
x = input()

# Array
arraysByLetter = {}
arraysByLetter['b'] = [1, 2, 3, 4, 5]
arraysByLetter['c'] = [2, 3, 4]
arraysByLetter['d'] = [1, 2, 3]

# Print Array
print(arraysByLetter[x])
</code></pre>
