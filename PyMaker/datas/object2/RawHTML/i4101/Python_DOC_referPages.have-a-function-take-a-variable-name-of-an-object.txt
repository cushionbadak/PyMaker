<p>If I have the following code:</p>
<pre><code>class foo:
    def __init__(self):
        self.w = 5
        self.z = 10

def sum(obj,x,y):
    return obj.x+obj.y

f = foo()
print sum(foo,'x','y')
</code></pre>
<p>How would I create a function that takes in two unkown variable names and returns the sum of those variables variables?</p>
<p>In this case the example should print 15.
EDIT:
Typo for the last line it should say print sum(foo,'w','z')</p>

<div class="post-text" itemprop="text">
<p>All (I think?) python objects have a built-in <code>__getattribute__</code> method. Use it like this:</p>
<pre><code>def sum(obj,x,y):
    return obj.__getattribute__(x)+obj.__getattribute__(y)
</code></pre>
<p>The method takes a string, and "unpacks" it to being a variable name.</p>
