<div class="post-text" itemprop="text">
<p>I have following string of date <code>"2018-05-08"</code> and I would like to convert it into the datetime format of python like: <code>"2018-05-08T00:00:00.0000000"</code>. I understand I can combine the string like:</p>
<pre><code>&gt; date = "2018-05-08"
&gt; time = "T00:00:00.0000000"
&gt; date+time
'2018-05-08T00:00:00.0000000'
</code></pre>
<p>But is there pythonic way of doing it where I can use libraries like datetime?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>datetime</code> module like this example: </p>
<pre><code>import datetime 

date = "2018-05-08"
final = datetime.datetime.strptime(date, '%Y-%m-%d').strftime('%Y-%m-%dT%H:%M:%S.%f')
print(final)
</code></pre>
<p>Output:</p>
<pre><code>2018-05-08T00:00:00.000000
</code></pre>
<p>For more details visit <code>datetime</code>'s <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer">documentation</a></p>
</div>
<div class="post-text" itemprop="text">
<pre><code>from datetime import datetime 
datetime.strftime(datetime.strptime('2018-05-08','%Y-%m-%d'),'%Y-%m-%dT%H:%M:%S.%f')
</code></pre>
<p>OUTPUT:</p>
<pre><code>'2018-05-08T00:00:00.000000'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre class="lang-py prettyprint-override"><code>from datetime import datetime
date_as_datetime = datetime.strptime("2018-05-08", '%Y-%m-%d')
date_as_string = date_as_datetime.strftime('%Y-%m-%dT%H:%M:%S')

print(date_as_string)
</code></pre>
<p>Out:
</p>
<pre><code>'2018-05-08T00:00:00'
</code></pre>
<p>See strptime/strftime options here: <a href="https://docs.python.org/2/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer">https://docs.python.org/2/library/datetime.html#strftime-and-strptime-behavior</a></p>
</div>
<div class="post-text" itemprop="text">
<p>Yes, you can use the datetime module and the strftime method from that module. Here is an example below if we want to format the current time. </p>
<pre><code>import datetime

current_time = datetime.datetime.now()

formatted_time = datetime.datetime.strftime(current_time, "%Y-%h-%d %H:%m:%S")
print(formatted_time)
</code></pre>
<p>Here is the output: </p>
<pre><code>2018-May-08 13:05:16
</code></pre>
