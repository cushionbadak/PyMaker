<pre><code>import re 
pattern = '(TAR)|(POSIX)'
string = 'POSIX TAR ARCHIVE (GNU)'
match = re.search(pattern, string , re.IGNORECASE)
match.groups()
</code></pre>

<div class="post-text" itemprop="text">
<p>try this: </p>
<pre><code>re.findall(pattern, string, re.IGNORECASE)
</code></pre>
<p><a href="https://docs.python.org/3/library/re.html#finding-all-adverbs" rel="nofollow noreferrer">re.findall</a> matches all the patterns you are looking for, <br/>
while <a href="https://docs.python.org/3/library/re.html#search-vs-match" rel="nofollow noreferrer">re.search</a> finds and returns the first match from the string. </p>
