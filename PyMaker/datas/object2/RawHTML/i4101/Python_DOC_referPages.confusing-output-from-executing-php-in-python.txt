<p>Why I get output with <em>b</em> and <em>''</em>?</p>
<pre><code>import subprocess

result = subprocess.run(['php', 'index.php'], stdout=subprocess.PIPE, check=True)
print(result.stdout) 
</code></pre>
<p><strong>Output =&gt; b'Hello'</strong></p>
<p>In Php I have just echo().</p>

<div class="post-text" itemprop="text">
<p><code>result.stdout</code> is a <code>bytes</code> object, and the <code>bytes.__str__()</code> methods returns its representation (see <a href="https://docs.python.org/3/library/functions.html#repr" rel="nofollow noreferrer"><code>repr()</code></a>).</p>
<p>Try with <code>print(result.stdout.decode('utf-8'))</code> to decode the bytes as UTF-8.</p>
<p>Suggested reading: <a href="https://docs.python.org/3/howto/unicode.html" rel="nofollow noreferrer">Unicode HOWTO</a></p>
