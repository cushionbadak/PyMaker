<p>Yes. Try sending a HTTP GET request, e.g. bare minimum request to a HTTP 1.1 server:</p>
<pre><code>import socket

s = socket.socket()
s.connect(('httpbin.org', 80))

request = '\r\n'.join(('GET /get HTTP/1.1', 'Host: httpbin.org', '', ''))
s.send(request)
response = s.recv(1024)

&gt;&gt;&gt; print(response)

<p>That's a lot more work than you need to do however. Consider using a library such as <code>requests</code>:</p>
<pre><code>import requests
r = requests.get('http://httpbin.org/get')
&gt;&gt;&gt; print(r.text)

<p>Or the standard library <a href="https://docs.python.org/3/library/urllib.request.html#urllib.request.urlopen" rel="nofollow noreferrer"><code>urlopen()</code></a> function.</p>
