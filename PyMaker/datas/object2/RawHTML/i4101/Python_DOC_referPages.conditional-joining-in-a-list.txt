<p>I have a Python list:</p>
<pre><code>data = ['Cost', '$', 4244, '$', 4090, '$', 3967]
</code></pre>
<p>What I want to do is merge the <code>$</code> with preceding element and obtain a new list:</p>
<pre><code>data = ['Cost', '$4244', '$4090', '$3967']
</code></pre>
<p>What is the best way to achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You could make use of <a href="https://docs.python.org/3/glossary.html#term-iterator" rel="noreferrer"><em>iterators</em></a>:</p>
<pre><code>data_iter = iter(data)
[e + str(next(data_iter)) if e == '$' else e for e in data_iter]
</code></pre>
<p>This makes use of the fact we can obtain the next element of an iterable, even while looping over the iterable. Each time a <code>'$'</code> string is found, the next element is pulled in and concatenated as a string.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; data = ['Cost', '$', 4244, '$', 4090, '$', 3967]
&gt;&gt;&gt; data_iter = iter(data)
&gt;&gt;&gt; [e + str(next(data_iter)) if e == '$' else e for e in data_iter]
['Cost', '$4244', '$4090', '$3967']
</code></pre>
