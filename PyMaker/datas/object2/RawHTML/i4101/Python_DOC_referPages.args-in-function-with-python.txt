<div class="post-text" itemprop="text">
<p>I was trying to define a simple sum function in python. </p>
<p>My code is:</p>
<pre><code>def sum_all_num(*args):
    return sum(*args)
</code></pre>
<p>But I got error.  I understand the *args will gather all arguments in a tuple, but why I can not use sum function to sum it?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer"><code>sum()</code></a> takes an iterable.  Just remove the <code>*</code> and pass the <code>args</code> tuple directly.</p>
<h3>Example:</h3>
<pre><code>x = 1, 2, 3
print(sum(x))
</code></pre>
<h3>Results:</h3>
<pre><code>6
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>sum()</code> takes only one argument (an iterable).</p>
<p>So, instead of unpacking <code>args</code> with <code>*</code>, just pass the <code>args</code> variable to it:</p>
<pre><code>def sum_all_num(*args):
    return sum(args)
</code></pre>
<p>Test:</p>
<pre><code>&gt;&gt;&gt; sum_all_num(1,2,3)
6
</code></pre>
</div>
<span class="comment-copy"><code>sum()</code> can take two arguments.</span>
