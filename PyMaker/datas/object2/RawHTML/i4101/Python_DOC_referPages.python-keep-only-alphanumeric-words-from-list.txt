<p>I have a list of words resembling the following</p>
<pre><code>    mylist=["hi", "h_ello", "how're", "you", "@list"]
</code></pre>
<p>I would like to pull out all of the non-alpha numeric characters to give a results such as:</p>
<pre><code>                  "h_ello", "how're", "@list"
</code></pre>
<p>Please note I have a much longer list in real life, and it contains some non-alpha numeric instances such as ~, ?, &gt;, =, + etc.</p>
<p>Does anyone know how to do this ,please?
Thank you</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>str.isalpha()</code></p>
<p><strong>Ex:</strong></p>
<pre><code>mylist=["hi", "h_ello", "how're", "you", "@list"]
print([i for i in mylist if not i.isalpha()])
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>['h_ello', "how're", '@list']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <code>list comprehension</code> in combination with <code>isalnum()</code> function.</p>
<pre><code>mylist=["hi", "h_ello", "how're", "you", "@list"]
print([i for i in mylist if not i.isalnum()])
</code></pre>
<p>Output</p>
<pre><code>['h_ello', "how're", '@list']
</code></pre>
<p>From <a href="https://docs.python.org/3/library/stdtypes.html?highlight=isalnum#str.isalnum" rel="nofollow noreferrer">python documentation</a>:</p>
<blockquote>
<p><code>str.isalnum()</code>
  Return true if all characters in the string are
  alphanumeric and there is at least one character, false otherwise. A
  character c is alphanumeric if one the following returns True:
  <code>c.isalpha()</code>, <code>c.isdecimal()</code>, <code>c.isdigit()</code>, or <code>c.isnumeric()</code>.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>You can also use <code>filter</code> with <code>re</code>:</p>
<pre><code>import re
mylist=["hi", "h_ello", "how're", "you", "@list"]
new_list = list(filter(lambda x:re.findall('[\W_]', x), mylist))
</code></pre>
<p>Output:</p>
<pre><code>['h_ello', "how're", '@list']
</code></pre>
</div>

<p>Better you go for isalnum or regex , Here i tried a little different approach just for fun, This is not for production code it will take time, I just tried to show you a different way :</p>
<pre><code>import unicodedata
import sys

mylist = ["hi", "h_ello", "how're", "you", "@list"]


def translate(text_):
    pun=[i for i in range(sys.maxunicode) if unicodedata.category(chr(i)).startswith('P')]

    if True in [True if ord(i) in pun else False for i in text_ ]:
        return text_


print(list(filter(lambda x:x,[translate(i) for i in mylist])))
</code></pre>
<p>output:</p>
<pre><code>['h_ello', "how're", '@list']
</code></pre>
