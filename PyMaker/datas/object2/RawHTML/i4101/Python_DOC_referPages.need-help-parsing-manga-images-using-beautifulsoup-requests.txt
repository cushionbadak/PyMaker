<blockquote>
<p>The HTML <code>&lt;script&gt;</code> element is used to embed or reference executable code; this is typically used to embed or refer to JavaScript code.</p>
</blockquote>
<p>The page you scrape is rendered by JavaScript, the initial HTML contains a <code>&lt;script&gt;</code> tag with <code>images</code>  variable that contains URLs of all the pictures. You can either follow <a href="https://stackoverflow.com/users/532312/rakesh">Rakesh's</a> approach of extracting the JavaScript object containing URLs with <a href="https://docs.python.org/3/howto/regex.html" rel="nofollow noreferrer">regular expressions</a> or use <a href="https://html.python-requests.org/" rel="nofollow noreferrer"><code>Requests-HTML</code></a> that uses Chromium under the hood to render the website as it appears in the browser.</p>
<p>You can get all the URLs with the code below:</p>
<pre><code>from requests_html import HTMLSession

session = HTMLSession()
url = 'https://bato.to/chapter/1204541'
r = session.get(url)
r.html.render()

urls = [img.attrs['src'] for img in r.html.find('.item &gt; img')]
</code></pre>

 'https://file-comic-99.anyacg.co/uploads/5ae318893532c3405bd86baa/a0f75d10919430e6ef25a86c3dcc65f343d8e319_518773_800_1138.png',

<p>The issue of this solution is that it takes a long time to execute.</p>
<p>To clarify, <code>.item &gt; img</code> targets all the <code>img</code> elements which are direct children of an element with <code>item</code> class.
You can read more about CSS selectors <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Selectors" rel="nofollow noreferrer">here</a>.</p>
<p><strong>You can also use <a href="https://selenium-python.readthedocs.io/" rel="nofollow noreferrer"><code>Selenium</code></a>:</strong></p>

browser = webdriver.Chrome()

browser.quit()
