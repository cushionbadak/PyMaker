<div class="post-text" itemprop="text">
<p>it looks like you have a typo in <code>def sale_price(self):</code></p>
<p><code>return 5000.0 * self.wheels</code> not <code>return 5000.0 * self_wheels</code></p>
<p>in <code>def spec(self):</code> you are not calling the class methods, you want.</p>
<pre><code>def spec(self):
    x = self.sale_price()
    y = self.purchase_price()
    print "This car should sell at: ", x 
    print "This car should be bought at: ", y
</code></pre>
</div>
<span class="comment-copy">If you want the value, you need to actually call the method self.sale_price(), your are just assigning the method object to x.</span>
