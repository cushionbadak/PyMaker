<div class="post-text" itemprop="text">
<p>I think what you want is to assign the line:</p>
<pre><code>sheet.cell_value(row, 1)
</code></pre>
<p>to a variable. For example:</p>
<pre><code>my_row_value = sheet.cell_value(row, 1)
</code></pre>
<p>Then you can use <strong><em>my_row_value</em></strong> with nltk tokenize as you wish. Better yet, if you want this program to grow and use it after you have exited the python interpreter, the best thing will be to have your code in a script, for example:</p>
<pre><code>myscript.py
</code></pre>
<p>And then you can simply run it using the python interpreter or if you are on linux simply:</p>
<pre><code>python myscript.py
</code></pre>
