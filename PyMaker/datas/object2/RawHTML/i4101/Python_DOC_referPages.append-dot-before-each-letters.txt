<div class="post-text" itemprop="text">
<p>A shorter and faster solution relying on the power of <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">regular expressions</a>:</p>
<pre><code>import re
my_text = "sample"
re.sub(r"[aeiou]*([^aeiou])[aeiou]*", r".\1", my_text.lower())
</code></pre>
<p>This reads: “delete vowels, and prefix each remaining letter with a dot”.</p>
</div>

<p>Removing all vowels and append "." before each letter.</p>
<pre><code>myText = "sample"
vowels = ['a', 'e', 'i', 'o', 'u', 'A', 'E', 'I', 'O', 'U']
result = ''
for letter in myText:
    if letter not in vowels:
        result = result + '.' + letter

print(result.lower())
# .s.m.p.l
</code></pre>

<div class="post-text" itemprop="text">
<p>Yet another way:</p>
<pre><code>import re
myText = "sample"
result = '.' + '.'.join(list(re.sub('[aeiou]', "", myText, flags=re.I)))
print(result)
</code></pre>
<p>Explanation:</p>
<ul>
<li><code>re.sub</code> removes the not required letters (case insensitive = <code>re.I</code>)</li>
<li><code>list</code> makes the string an array of characters</li>
<li><code>join</code> places a dot in between</li>
<li><code>'.' +</code> adds the missing dot in front of the first character</li>
</ul>
