<div class="post-text" itemprop="text">
<p>I guess you use Python 3.</p>
<p>Use <code>zip()</code> builtin instead.</p>
<p>In Python 3 there is no <code>itertools.izip()</code> as the <code>zip()</code> builtin behaves similarly.</p>
