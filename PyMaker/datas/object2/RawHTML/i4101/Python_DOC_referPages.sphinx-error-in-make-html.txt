<div class="post-text" itemprop="text">
<p>The directory above the <code>backend</code> directory must be added to <code>sys.path</code>.</p>
<p>If <code>bb_lite</code> is the parent directory of both <code>backend</code> and <code>docs</code>, the following is needed in <code>docs/conf.py</code>:</p>
<pre><code>sys.path.insert(0, os.path.abspath('..'))
</code></pre>
</div>
<span class="comment-copy">To enable Sphinx to import your modules, the directory <i>above</i> <code>backend</code> should be added to <code>sys.path</code>. You need to adjust the <code>sys.path.insert</code> line.</span>
