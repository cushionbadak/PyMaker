<p>I am fairly new to python and was looking for solutions to generate random MAC addresses when I found this:</p>
<pre><code>':'.join('%02x'%random.randint(0,255) for x in range(6))
</code></pre>
<p>what I am trying to understand is the use of <code>'%02x'%</code> in the code. Is it related to how mac addresses are represented which is a 48 bit hexadecimal value separated by <code>:</code>? </p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/2/library/stdtypes.html#string-formatting" rel="nofollow noreferrer">https://docs.python.org/2/library/stdtypes.html#string-formatting</a></p>
<p><code>'%02x'</code> is a format string for specifying a zero-filled two-digits hexadecimal number.</p>
<p><code>'%02x' % number</code> will create the actual string with the number in such format.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you don't understand a list comprehension then the best tactic is to unwind it.</p>
<pre><code>myhexdigits = []
for x in range(6):
    # x will be set to the values 0 to 5
    a = random.randint(0,255)
    # a will be some 8-bit quantity
    hex = '%02x' % a
    # hex will be 2 hexadecimal digits with a leading 0 if necessary
    # you need 2 hexadecimal digits to represent 8 bits
    myhexdigits.append(hex)
    # save for after the loop ends
print (':'.join(myhexdigits))
# using : as the delimiter, join the 2-digit hex strings together into
# a single string
</code></pre>
