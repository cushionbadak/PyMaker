<div class="post-text" itemprop="text">
<p>Python keeps track of the types of variables without having to explicitly state the type.</p>
<p><a href="https://wiki.python.org/moin/Why%20is%20Python%20a%20dynamic%20language%20and%20also%20a%20strongly%20typed%20language" rel="nofollow">https://wiki.python.org/moin/Why%20is%20Python%20a%20dynamic%20language%20and%20also%20a%20strongly%20typed%20language</a></p>

<div class="post-text" itemprop="text">
<p>That being said you can use type hints in Python 3:</p>
<pre><code>def f(x: int):
    return x ** 2
</code></pre>
<p>In my opinion they are fantastic.</p>
