<div class="post-text" itemprop="text">
<p>What is the super-class of <code>Exception</code> in Python? Please provide me the Python exception hierarchy.</p>
</div>
<div class="post-text" itemprop="text">
<p>You are looking for <a href="https://docs.python.org/3/library/exceptions.html#BaseException" rel="nofollow"><code>BaseException</code></a>.</p>
<p>User defined exception types should subclass <code>Exception</code>.</p>
<p>See the <a href="https://docs.python.org/3/library/exceptions.html" rel="nofollow">docs</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>Exception</code>'s base class:</p>
<pre><code>&gt;&gt;&gt; Exception.__bases__
(BaseException,)
</code></pre>
<p><a href="https://docs.python.org/3/library/exceptions.html#exception-hierarchy" rel="nofollow">Exception hierarchy</a> from the docs confirms that it is a base class for all exceptions:</p>
<pre><code>BaseException
 +-- SystemExit
 +-- KeyboardInterrupt
 +-- GeneratorExit
 +-- Exception
      +-- StopIteration
      +-- ArithmeticError
      |    +-- FloatingPointError
...
</code></pre>
<p>The syntax to catch all exceptions is:</p>
<pre><code>try:
    raise anything
except: 
    pass
</code></pre>
<p>NOTE: use it very very sparingly e.g., you could use it in a <code>__del__</code> method during cleanup when the world might be half-destroyed and there is no other alternative.</p>
<p>Python 2 allows to raise exceptions that are not derived from <code>BaseException</code>:</p>
<pre><code>&gt;&gt;&gt; raise 1
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: exceptions must be old-style classes or derived from BaseException, not int
&gt;&gt;&gt; class A: pass
... 
&gt;&gt;&gt; raise A
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
__main__.A: &lt;__main__.A instance at 0x7f66756faa28&gt;
</code></pre>
<p>It is fixed in Python 3 that enforces the rule:</p>
<pre><code>&gt;&gt;&gt; raise 1
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: exceptions must derive from BaseException
</code></pre>
</div>
