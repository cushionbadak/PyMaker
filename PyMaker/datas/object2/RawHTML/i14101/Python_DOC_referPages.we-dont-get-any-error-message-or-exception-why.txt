<div class="post-text" itemprop="text">
<pre><code>print(10 .__add__("1"))
</code></pre>
<p>the <code>("1")</code> specifies the string,since it is enclosed in a double quotes..</p>
<p>10 indicates integer....</p>
<p>the function to add string and integer is probably may not be implemented earlier..so this is why python shows <code>NotImplemented</code> </p>
<pre><code>print(10 .__add__(1)) 
</code></pre>
<p>and the space also doesn't matter..!!!!</p>
<pre><code>print((10).__add__(1))
</code></pre>
<p>the above two codes also will work</p>
