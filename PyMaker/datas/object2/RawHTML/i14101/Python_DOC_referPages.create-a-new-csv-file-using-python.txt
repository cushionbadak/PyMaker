<div class="post-text" itemprop="text">
<p>You can use <code>",".join</code> to do this. Also, you can directly access the values using <code>d.values()</code>, so you don't really need to do iterate over the dict and then extract the values.</p>
<p>So essentially, your code can be simplified to:</p>
<pre><code>with open(path3 + 'extend.csv', 'w') as fw:
    fw.write(",".join(d.values()))
    fw.write('\n')
fw.close()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use the <code>csv</code> module: <a href="https://docs.python.org/2/library/csv.html" rel="nofollow">https://docs.python.org/2/library/csv.html</a></p>
<pre><code>import csv
with open(path3 + 'extend.csv', 'w') as fw:
    writer = csv.writer(fw, delimiter=',')
    writer.writerow(d.values())
</code></pre>
</div>
<span class="comment-copy">As a side note, <code>dict</code> item order is not maintained. This approach is fine if you don't care what order you're putting those values into the csv file, but I think for most people they do care. Either iterate over specific dict keys or switch to a list for <code>d</code> (which maintains order).</span>
