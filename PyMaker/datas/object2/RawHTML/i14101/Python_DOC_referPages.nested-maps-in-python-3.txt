<div class="post-text" itemprop="text">
<p>I want to transform my list <code>list = [" a , 1 ", " b , 2 "]</code> into a nested list <code>[["a","1"],["b","2"]]</code>.</p>
<p>The following works:</p>
<pre><code>f1_a = map(lambda x :[t.strip() for t in x.split(',',1)],list)
</code></pre>
<p>but this does not work with Python 3 (it does work with Python 2.7!):</p>
<pre><code>f1_b = map(lambda x :map(lambda t:t.strip(),x.split(',',1)),list)
</code></pre>
<p>Why is that?</p>
<p>Is there a more concise way then <code>f1_4</code> to achieve what I want?</p>
</div>
<div class="post-text" itemprop="text">
<p>Python 3's <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow"><code>map</code></a> returns a <code>map</code> object, which you need to convert to list explicitly:</p>
<pre><code>f1_b = list(map(lambda x: list(map(lambda t: t.strip(), x.split(',', 1))), lst))
</code></pre>
<p>Though in most cases you should prefer list comprehensions to <code>map</code> calls:</p>
<pre><code>f1_a = [[t.strip() for t in x.split(',', 1)] for x in lst]
</code></pre>
</div>
