<p>I have:</p>
<pre><code>n = 257
a = n.to_bytes(2, 'little')
a = b'\x01\x01'
</code></pre>
<p>How can i convert this back into <code>257</code></p>
<p>Also, is there any way to show <code>to_bytes</code> without specifying how many bytes?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the complementary <a href="https://docs.python.org/3/library/stdtypes.html#int.from_bytes" rel="nofollow"><code>int.from_bytes</code></a> and specify the byteorder again.</p>
<pre><code>&gt;&gt;&gt; n = 257
&gt;&gt;&gt; n_bytes = n.to_bytes(2, "little")
&gt;&gt;&gt; n_again = int.from_bytes(n_bytes, "little")
&gt;&gt;&gt; n_again == n
True
</code></pre>
</div>
