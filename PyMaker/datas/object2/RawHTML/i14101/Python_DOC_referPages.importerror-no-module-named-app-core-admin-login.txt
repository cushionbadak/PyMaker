<div class="post-text" itemprop="text">
<p>You don't have <code>__init__.py</code> files in those package folders. You application structure should look like this:</p>
<p>Each folder should have it's own <code>__init__.py</code>.</p>
<pre><code>MyApplication/ 
   |-- app/ 
          |-- __init__.py
          |-- core/
                 |-- __init__.py
                 |-- admin/
                        |-- __init__.py
                        |-- login.py
</code></pre>
</div>
