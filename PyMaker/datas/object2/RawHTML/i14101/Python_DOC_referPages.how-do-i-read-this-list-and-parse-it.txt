<p>Your list contains one dictionary

<pre><code>&gt;&gt;&gt; yourlist[0]["popularity"]
2354
</code></pre>
<p><code>[0]</code> for the first item in the list (the dictionary).</p>
<p><code>["popularity"]</code> to get the value associated to the key 'popularity' in the dictionary.</p>
