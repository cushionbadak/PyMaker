<p>I have two columns and I am trying to use pandas to calculate it. I am a spreadsheet user lately learning python for fast calculation. I want to create a new column 'C' and I know only to use excel formulas but in python, I am aware of calculating the difference between two-row values. Not sure how to do in pandas for the result i expect. </p>
<pre><code>A B
a r
b m
c f
</code></pre>
<p>Tried: </p>
<pre><code>df['C']=df['A'] - df['A'].shift(-1)
</code></pre>
<p>Output Expected:</p>
<pre><code>C
(b-a)
(c-b)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use:</p>
<pre><code>df['C'] = df['A'] - df['A'].shift()
</code></pre>
<p>Or:</p>
<pre><code>df['C'] = df['A'].diff()
</code></pre>
<hr/>
<p>Then if need remove first row with missing value:</p>
<pre><code>df = df.iloc[1:]
</code></pre>
<p>Or if only one <code>NaN</code>s in first row:</p>
<pre><code>df = df.dropna(subset=['A'])
</code></pre>
<hr/>
<p>Like <code>MStaino</code> commented, if need shif rows up after difference (get last NaN value in <code>C</code> column):</p>
<pre><code>df['C'] = df['A'].diff().shift(-1)
</code></pre>
</div>
