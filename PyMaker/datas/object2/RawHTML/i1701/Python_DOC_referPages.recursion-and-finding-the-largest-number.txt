<p>There are a few issues with your code, mainly you are missing <code>num_list</code> as a parameter:</p>
<pre><code>def find_largest(num_list):
    if len(num_list) == 1:
        return num_list[0]
    else:
        m = find_largest(num_list[1:])
        return m if m &gt; num_list[0] else num_list[0]


result = find_largest([1, 2, 3, 4])
print(result)
</code></pre>

<pre><code>4
</code></pre>
