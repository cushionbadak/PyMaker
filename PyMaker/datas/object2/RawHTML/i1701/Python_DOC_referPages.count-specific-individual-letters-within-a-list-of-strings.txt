<div class="post-text" itemprop="text">
<p>Sure! Use <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>collections.Counter</code></a>:</p>
<pre><code>from collections import Counter


lst = ['a a b b c c', 'a c b c', 'b c c a b']

counter = Counter()
for word in lst:
    counter.update(word)

print(counter)
# Counter({' ': 12, 'c': 6, 'b': 5, 'a': 4})
</code></pre>
