<div class="post-text" itemprop="text">
<p>You can use <a href="https://pyformat.info/" rel="nofollow noreferrer">string formatting</a> to structure the output positions for every line. I made some assumptions about your values for <code>cat</code>. Do see if the code snippet below makes sense to you.</p>
<pre><code>cat1 = 2
cat2 = 3
cat3 = 1
cat4 = 4

header = ['0-29', '30-39', '40-69', '70-100']
print(' '.join(header))
for x in range(max(cat1, cat2, cat3, cat4)):
    print(" {0}     {1}     {2}     {3}".format(
        '*' if x &lt; cat1 else ' ',
        '*' if x &lt; cat2 else ' ',
        '*' if x &lt; cat3 else ' ',
        '*' if x &lt; cat4 else ' '
    ))
</code></pre>
<p>The resultant output will be like this.</p>
<pre><code>0-29 30-39 40-69 70-100
 *     *     *     *
 *     *           *
       *           *
                   *
</code></pre>

<div class="post-text" itemprop="text">
<p>you can't "print vertically". print is always in horizontal lines. You need to build your output line by line, or use a dedicated library like ncurses:
<a href="https://docs.python.org/3/howto/curses.html" rel="nofollow noreferrer">https://docs.python.org/3/howto/curses.html</a></p>
