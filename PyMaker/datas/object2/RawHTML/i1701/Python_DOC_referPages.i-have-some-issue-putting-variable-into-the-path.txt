<div class="post-text" itemprop="text">
<p>You could use an <a href="https://www.python.org/dev/peps/pep-0498/" rel="nofollow noreferrer">f-string</a> if youâ€™re working in python 3.6+</p>
<p>This is the most efficient method.</p>
<pre><code>counter = 0
filepath = f"/home/user/image{counter}.jpeg"
image = ClImage(file_obj=open(filepath, 'rb'))
</code></pre>
<p>Otherwise the second best would be using the <a href="https://docs.python.org/3.4/library/functions.html#format" rel="nofollow noreferrer">.format()</a> function:</p>
<pre><code>counter = 0
filepath = "/home/user/image{0}.jpeg".format(counter)
image = ClImage(file_obj=open(filepath, 'rb'))
</code></pre>

<div class="post-text" itemprop="text">
<p>You can use Python's <a href="https://realpython.com/python-string-formatting/" rel="nofollow noreferrer">.format()</a> method:</p>
<pre><code>counter = 0
filepath = '/home/user/image{0}.jpeg'.format(counter)
image = ClImage(file_obj=open(filepath, 'rb'))
</code></pre>

<p>You need <a href="https://www.pythonforbeginners.com/concatenation/string-concatenation-and-formatting-in-python" rel="nofollow noreferrer">string concatenation</a>.</p>
<pre><code>&gt;&gt;&gt;counter = 0;
