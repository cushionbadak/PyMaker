<p>So I have been able to create the following program which compares two strings to see if they are anagrams of each other. </p>
<pre><code>def anagrams( string1, string2 ):
    if sorted(string1.lower()) == sorted(string2.lower()):
        return True
    else:
        return False
</code></pre>
<p>However, my issue is that I wish to not return a True value should both input strings be the exact same. For example:</p>
<pre><code>anagrams('silent','silent')
</code></pre>
<p>This outputs True but I do not want it to do that, what changes should I make to implement this?</p>

<p>Just check if the strings are different:</p>
<pre><code>def anagrams(string1, string2):
    if sorted(string1.lower()) == sorted(string2.lower()) and string1.lower() != string2.lower():
        return True
    else:
        return False

result = anagrams('silent', 'silent')
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>False
</code></pre>
<p>You could use <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer">Counter</a> instead of sorted:</p>
<pre><code>from collections import Counter


def anagrams(string1, string2):
    if string1.lower() != string2.lower() and Counter(string1.lower()) == Counter(string2.lower()):
        return True
    else:
        return False

print(anagrams('silent', 'silent'))
print(anagrams('silent', 'sitlen'))
</code></pre>
<p><strong>Output</strong></p>
<pre><code>False
True
</code></pre>
<p><strong>UPDATE</strong></p>
<p>As suggested by @RoadRunner, you could do:</p>
<pre><code>from collections import Counter


def anagrams(string1, string2):
    ls1, ls2 = string1.lower(), string2.lower()
    return ls1 != ls2 and Counter(ls1) == Counter(ls2)

print(anagrams('silent', 'silent'))
print(anagrams('silent', 'sitlen'))
</code></pre>
