<div class="post-text" itemprop="text">
<p>With <code>open()</code>, you have to use the <code>os.path.expanduser()</code> function to expand the tilde <code>~</code> into the actual home directory of the user:</p>
<pre><code>import os
with open(os.path.expanduser('~/Documents/data.csv'), 'r') as f:
    print(f.read())
</code></pre>
<p>Pandas' <code>read_csv()</code> does that for you.</p>
<p>(A word of warning: since <code>~</code> is a valid character in Linux filenames, just replacing <code>~</code> with <code>os.getenv("HOME")</code> is a very bad idea...)</p>
</div>
