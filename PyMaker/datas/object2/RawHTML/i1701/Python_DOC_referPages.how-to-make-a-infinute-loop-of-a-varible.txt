<div class="post-text" itemprop="text">
<p>If you want to create an infinite loop with <code>for</code>, you have some options:</p>
<p>Create your own infinite generator for example:</p>
<pre><code>def infiniteI():
    i = 0
    while True:
        yield i
        i++
for i in infiniteI():
    ...
</code></pre>
<p>Or, you can use some <a href="https://docs.python.org/3/library/itertools.html#module-itertools" rel="nofollow noreferrer">itertools</a>, <a href="https://docs.python.org/3/library/itertools.html#itertools.cycle" rel="nofollow noreferrer">cycle</a> or <a href="https://docs.python.org/3/library/itertools.html#itertools.repeat" rel="nofollow noreferrer">repeat</a>:</p>
<pre><code>for l in itertools.cycle("AEIOU"):
    ...

for _ in itertools.repeat(True):
    ...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This loop iterates for each element in <code>a</code>. If <code>a</code> is finite then the loop will also be finite.</p>
