<div class="post-text" itemprop="text">
<p>Say I have the following string:</p>
<pre><code>    string1 = 'Twenty 20 Twelve 12'
</code></pre>
<p>I would like to convert it into a list that would keep words as strings in separate elements, and numbers in another (as integers):</p>
<pre><code>    list1 = ['Twenty', 20, 'Twelve', 12]
</code></pre>
<p>My current code looks is:</p>
<pre><code>    list1 = [y for y in string1.replace(' ','')]
</code></pre>
<p>and the result prints out as:</p>
<pre><code>    ['T','w','e','n','t','y','2','0','T','w','e','l','v','e','1','2']
</code></pre>
<p>How would I be able to write a code to keep words in separate entries, and turn numbers inside the string into integers in the list?  I am a beginner to programming who is currently learning Python in parallel with C.</p>
</div>
<div class="post-text" itemprop="text">
<p>Look into the <a href="https://docs.python.org/3/library/stdtypes.html" rel="nofollow noreferrer"><code>.split()</code></a> function.</p>
<p>It takes the form of</p>
<pre><code>str.split(sep=None, maxsplit=-1)
</code></pre>
<p>so you want your code to look like this to break it apart.</p>
<pre><code>string1 = 'Twenty 20 Twelve 12'
string1.split()
#['Twenty', '20', 'Twelve', '12']
</code></pre>
<p>To convert the numbers to integers, just check for <code>.isdigit()</code></p>
<pre><code>[int(i) if i.isdigit() else i for i in string1.split()]
#['Twenty', 20, 'Twelve', 12]
</code></pre>
<p>if you're not familiar with <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer">list comprehensions</a>, this is analogous to</p>
<pre><code>values = []
for i in string1.split():
    if i.isdigit():
        values.append(int(i))
    else:
        values.append(i)

values
#['Twenty', 20, 'Twelve', 12]
</code></pre>
