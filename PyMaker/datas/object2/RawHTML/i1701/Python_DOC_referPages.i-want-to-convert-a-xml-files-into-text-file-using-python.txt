<p>Assuming you have a file called <code>file.xml</code>, containing:</p>
<pre><code>&lt;annotation&gt;
    &lt;folder&gt;all_images&lt;/folder&gt;
    &lt;filename&gt;0.jpg&lt;/filename&gt;
    &lt;path&gt;/home/vishnu/Documents/all_images/0.jpg&lt;/path&gt;
    &lt;source&gt;
        &lt;database&gt;Unknown&lt;/database&gt;
    &lt;/source&gt;
    &lt;size&gt;
        &lt;width&gt;4250&lt;/width&gt;
        &lt;height&gt;5500&lt;/height&gt;
        &lt;depth&gt;1&lt;/depth&gt;
    &lt;/size&gt;
    &lt;segmented&gt;0&lt;/segmented&gt;
    &lt;object&gt;
        &lt;name&gt;word&lt;/name&gt;
        &lt;pose&gt;Unspecified&lt;/pose&gt;
        &lt;truncated&gt;0&lt;/truncated&gt;
        &lt;difficult&gt;0&lt;/difficult&gt;
        &lt;bndbox&gt;
            &lt;xmin&gt;308&lt;/xmin&gt;
            &lt;ymin&gt;45&lt;/ymin&gt;
            &lt;xmax&gt;502&lt;/xmax&gt;
            &lt;ymax&gt;162&lt;/ymax&gt;
        &lt;/bndbox&gt;
    &lt;/object&gt;
&lt;/annotation&gt;
</code></pre>
<p>Then the following Python script in the same folder gives you an idea how to use the Standard Library <a href="https://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow noreferrer">ElementTree API</a> to parse the file:</p>
<pre><code>import xml.etree.ElementTree as ET

tree = ET.parse("file.xml")
root = tree.getroot()

print(root.find("./folder").text)
print(root.find("./object/name").text)
print(root.find("./object/bndbox/xmin").text)
</code></pre>
<p>You will need to work out how to write the values to your own text files, but that should be straightforward. There are lots of resources such as <a href="https://www.pythonforbeginners.com/files/reading-and-writing-files-in-python" rel="nofollow noreferrer">this one</a>.</p>
