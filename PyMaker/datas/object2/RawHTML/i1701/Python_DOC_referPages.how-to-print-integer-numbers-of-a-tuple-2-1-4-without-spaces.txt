<p>How to print the <strong>tuple</strong> <code>(2, 1, 4)</code> without spaces in Python.
I am getting a extra space before <code>1</code> and <code>4</code> in the output.
Below is the report:</p>
<pre><code>Expected    Obtained    Difference
(2,1,4)    (2, 1, 4)    (2,_1,_4)
</code></pre>

<div class="post-text" itemprop="text">
<p>Stringify your value, and then replace the spaces with blanks:</p>
<pre><code>val = (2, 1, 4)

print(val)

print(str(val).replace(' ', ''))
</code></pre>
<p>Output:</p>
<pre><code>(2, 1, 4)
(2,1,4)
</code></pre>
<p>This will work for any length tuple or list.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>str.format()</code></a>:</p>
<pre><code>&gt;&gt;&gt; val = (2, 1, 4)
&gt;&gt;&gt; print("({},{},{})".format(*val))
(2,1,4)
</code></pre>
<p>For "coordinates" in higher dimensions, building the string yourself using <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>str.join()</code></a> and <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map()</code></a> could be more generalizable:</p>
<pre><code>&gt;&gt;&gt; val = (2, 1, 4, 7, 8, 9)
&gt;&gt;&gt; print("("+",".join(map(str, val))+")")
(2,1,4,7,8,9)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Here's one way using f-strings (Python 3.6+):</p>
<pre><code>val = (2, 1, 4)

print(f'({",".join(map(str, val))})')

(2,1,4)
</code></pre>
<p>If you do this often, you can create a <code>MyTuple</code> object, which always prints in this fashion:</p>
<pre><code>class MyTuple(tuple):
    def __repr__(self):
        return f'({",".join(map(str, val))})'

val = (2, 1, 4)
val = MyTuple(val)

print(val)

(2,1,4)
</code></pre>
