<p>I will be given a string, and I need to split it every time that it has an "|", "/",  "." or "_"</p>
<p>How can I do this fast? I know how to use the command split, but is there any way to give more than 1 split condition to it? For example, if the input given was</p>
<pre><code>Hello test|multiple|36.strings/just36/testing
</code></pre>
<p>I want the output to give:</p>
<pre><code>"['Hello test', 'multiple', '36', 'strings', 'just36', 'testing']"
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a regex and the regex module:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; s='You/can_split|multiple'
&gt;&gt;&gt; re.split(r'[/_|.]', s)
['You', 'can', 'split', 'multiple']
</code></pre>
<p>In this case, <code>[/_|.]</code> will split on any of those characters.</p>
<p>Or, you can use a list comprehension to insert a single (perhaps multiple character) delimiter and then split on that:</p>
<pre><code>&gt;&gt;&gt; ''.join(['-&gt;&lt;-' if c in '/_|.' else c for c in s]).split('-&gt;&lt;-')
['You', 'can', 'split', 'multiple']
</code></pre>
<hr/>
<p>With the added example:</p>
<pre><code>&gt;&gt;&gt; s2="Hello test|multiple|36.strings/just36/testing"
</code></pre>
<p>Method 1:</p>
<pre><code>&gt;&gt;&gt; re.split(r'[/_|.]', s2)
['Hello test', 'multiple', '36', 'strings', 'just36', 'testing']
</code></pre>
<p>Method 2:</p>
<pre><code>&gt;&gt;&gt; ''.join(['-&gt;&lt;-' if c in '/_|.' else c for c in s2]).split('-&gt;&lt;-')
['Hello test', 'multiple', '36', 'strings', 'just36', 'testing']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer">groupby</a>:</p>
<pre><code>from itertools import groupby

s = 'You/can_split|multiple'

separators = set('/_|.')
result = [''.join(group) for k, group in groupby(s, key=lambda x: x not in separators) if k]
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>['You', 'can', 'split', 'multiple']
</code></pre>
