<p>My goal is to convert a string into a tuple</p>
<p>Currently, I have a string that is surround by parentheses with its contents separated by commas.</p>
<pre><code>&gt;&gt;&gt;'1, 4, 1994' (variable name birthday)
(1, 4, 1994) # desired output
</code></pre>
<p>I have been trying to use split() to convert the string into tuple, but </p>
<pre><code>tuple(birthday.split())
('1,', '4,', '1994')
</code></pre>
<p>but the contents are surrounded by subsequent parentheses.</p>
<p>What pythonic methods can I use to make the conversion?</p>

<p>You can do like this,</p>
<pre><code>In [35]: tuple(map(int,birthday.split(',')))
Out[35]: (1, 4, 1994)
</code></pre>

<div class="post-text" itemprop="text">
<p>You need to convert <code>str</code> to <code>int</code> and specify a <code>sep</code> argument for <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>str.split</code></a>:</p>
<pre><code>res = tuple(map(int, '1, 4, 1994'.split(', ')))  # (1, 4, 1994)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow noreferrer"><code>ast.literal_eval</code></a></p>
<pre><code>&gt;&gt;&gt; import ast
&gt;&gt;&gt; s = '1, 4, 1994'
&gt;&gt;&gt; ast.literal_eval(s)
&gt;&gt;&gt; (1,4,1994)
</code></pre>
</div>
