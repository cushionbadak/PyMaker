<div class="post-text" itemprop="text">
<p>How to make a condition that accepts only in a loop strings that have '_' only two times.</p>
<pre><code>a = ['q_s_e','s_e_v_e','s_v']# many more like these
</code></pre>
<p>in the above list only the fist item should used in the loop:</p>
<pre><code>for i in a:
    if '_' in i:
         ...
</code></pre>
<p>this takes everything.</p>
<p>How to do it?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://www.programiz.com/python-programming/methods/string/count" rel="nofollow noreferrer">count</a> function from string:</p>
<pre><code>a = ['q_s_e','s_e_v_e','s_v']
for i in a:
    if i.count('_') == 2:
      print ('hooray')
</code></pre>
<blockquote>
<p>In simple words, count() method searches the substring in the given
  string and returns how many times the substring is present in it.</p>
<p>It also takes optional parameters start and end to specify the
  starting and ending positions in the string respectively.</p>
<p>The syntax of count() method is:</p>
<p>string.count(substring, start=..., end=...)</p>
</blockquote>
<p>Another way is to filter the array of string at the beginning as well:</p>
<pre><code>a = ['q_s_e','s_e_v_e','s_v']
for i in filter(lambda k: k.count('_') == 2, a):
    print ('hooray')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.count" rel="nofollow noreferrer"><code>str.count</code></a> and iterate over a generator expression:</p>
<pre><code>for i in (el for el in a if el.count('_') == 2):
    # do something
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to specify that with a conditional using the <code>count()</code> method:</p>
<pre><code>for item in list:
    if item.count('_') == 2:
        ...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>regex</code>, this will return the list you want:</p>
<pre><code>filtered_list = [e for e in old_list if re.match('^[^_]*(_[^_]*){2}$', e)]
</code></pre>
<p>Regex <code>^[^_]*(_[^_]*){2}$</code> will make sure each of these element contains exactly 2 <code>_</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You can also solve this with <code>numpy</code> and remove your loop:</p>
<pre><code>arr = np.array(['q_s_e','s_e_v_e','s_v'])
new_arr = arr[np.where(np.char.count(arr, "_") == 2)[0]]
</code></pre>
