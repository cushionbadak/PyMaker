<p>Why do I get the error <code>NameError: name 'l1' is not defined</code> when I run the code below, will placing the l1 and l2 empty lists outside the functions allow me to append/use the return statements?</p>
<pre><code>def function():
    l1 = []
    l2 = []
    for x in range(1):
        if 3 &gt; 2:
            l1.append(1)
            l2.append(2)
    return l1, l2

call = function()
print(l1)
print(l2)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can't call it like that, <code>l1</code> and <code>l2</code> are undefined then, so need to do:</p>
<pre><code>def function():
    l1 = []
    l2 = []
    for x in range(1):
        if 3 &gt; 2:
            l1.append(1)
            l2.append(2)
    return l1, l2

l1,l2 = function()
print(l1)
print(l2)
</code></pre>
<p>Output:</p>
<pre><code>[1]
[2]
</code></pre>
</div>
<span class="comment-copy"><code>l1</code> and <code>l2</code> are not defined outside of the function (something like <code>l1=something</code>). <code>call</code> on the other hand, is a tuple containing the two results from your functions. Perhaps you want to change it to <code>l1, l2 =  function()</code>, or leaving as is, and instead  <code>print(call[0])</code> and <code>print(call[1])</code>.</span>
