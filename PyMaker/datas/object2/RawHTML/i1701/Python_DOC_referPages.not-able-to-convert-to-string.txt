<div class="post-text" itemprop="text">
<p>Simply use:</p>
<pre><code>print(time.strftime("%m/%d/%y",dexpires))
</code></pre>
<p>See also <a href="https://docs.python.org/3/library/time.html" rel="nofollow noreferrer">https://docs.python.org/3/library/time.html</a></p>
<p>In general  python modules usually contain all kinds of reformatting functions you don't have to reinvent them.</p>
<p>Example:</p>
<pre><code>&gt;&gt;&gt; dexpires=time.strptime('20180823131455z','%Y%m%d%H%M%Sz')
&gt;&gt;&gt; dexpires
time.struct_time(tm_year=2018, tm_mon=8, tm_mday=23, tm_hour=13, tm_min=14, tm_sec=55, tm_wday=3, tm_yday=235, tm_isdst=-1)
&gt;&gt;&gt; time.strftime('%m/%d/%y',dexpires)
'08/23/18'
&gt;&gt;&gt;
</code></pre>

<div class="post-text" itemprop="text">
<p>You can use Python's <code>format()</code> method to handle it (much cleaner also):</p>
<pre><code>print("{0}/{1}/{2}".format(bes[0],bes[1],bes[2]))
</code></pre>
<p><sub>...or further simplified (thanks Anton)</sub></p>
<pre><code>print("{0}/{1}/{2}".format(*bes))
</code></pre>
