<p>I have a dataframe df1 with column name Acc Number as the first column and the data looks like: </p>
<pre><code>Acc Number
ASC100.1
MJT122
ASC120.4
XTY111
</code></pre>
<p>I need to make a new dataframe df2 that will have two columns first having the text part and the second having the numbers so the desired output is: </p>
<pre><code>Text    Number 
ASC     100.1
MJT     122
ASC     100.4
XTY     111
</code></pre>
<p>How would I go about doing this?</p>
<p>Thanks! </p>
</div>
<div class="post-text" itemprop="text">
<p>You could do something like this:</p>
<pre><code>import pandas as pd

data = ['ASC100.1',
'MJT122',
'ASC120.4',
'XTY111']

df = pd.DataFrame(data=data, columns=['col'])

result = df.col.str.extract('([a-zA-Z]+)([^a-zA-Z]+)', expand=True)
result.columns = ['Text', 'Number']
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>  Text Number
0  ASC  100.1
1  MJT    122
2  ASC  120.4
3  XTY    111
</code></pre>
<p>The pattern <code>([a-zA-Z]+)([^a-zA-Z]+)</code>  means match a group of letters: <code>([a-zA-Z]+)</code> followed by a group of non letters: <code>([^a-zA-Z]+)</code>. A safer alternative will be to use the following regex: <code>([a-zA-Z]+)(\d+\.?\d+)</code> assuming the numbers can only have at most one point.</p>
