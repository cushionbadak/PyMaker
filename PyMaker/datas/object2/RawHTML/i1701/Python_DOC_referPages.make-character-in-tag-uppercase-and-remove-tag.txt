<p>How to make a character between a tag uppercase?</p>
<p>For example I have this string: </p>
<pre><code>string = "&lt;big&gt;t&lt;big&gt;his should be &lt;big&gt;u&lt;big&gt;ppercase"
</code></pre>
<p>Should result in:</p>
<pre><code>This should be Uppercase
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Constantly as longest as there's <strong>&lt; big&gt;</strong> in txt remove tags and uppercase the text between them</p>
<pre><code>def up(x):
    idx1 = x.index('&lt;big&gt;')
    idx2 = x.index('&lt;big&gt;', idx1+1)

    new_x = x[:idx1]
    new_x += x[idx1+5: idx2].upper()
    new_x += x[idx2+5:]

    return new_x

txt = "&lt;big&gt;t&lt;big&gt;his should be &lt;big&gt;u&lt;big&gt;ppercase"
while '&lt;big&gt;' in txt:
    txt = up(txt)

print(txt) # This should be Uppercase
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>import re

s="&lt;big&gt;t&lt;big&gt;his should be &lt;big&gt;u&lt;big&gt;ppercase"

result = re.finditer('&lt;big&gt;(.)&lt;big&gt;', s)

for i in result:
     s = s.replace(i.group(0), i.group(1).upper())
</code></pre>
<p>Note:</p>
<blockquote>
<p>Do not use <code>str</code> as a name of a variable. <code>str</code> is predefined in python.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>Those don't look like real xml tags. But assuming the number of <code>&lt;big&gt;</code> tags are used correctly, you can split it by tag and apply <code>upper()</code> to strings between. </p>
<pre><code>text = "&lt;big&gt;t&lt;big&gt;his should be &lt;big&gt;u&lt;big&gt;ppercase"

split_str = text.split('&lt;big&gt;')
result = ''.join([x.upper() if i % 2 else x for i, x in enumerate(split_str)])
</code></pre>
