<p>For </p>
<p><code>A=[1,2,3]</code></p>
<p>I would like to get</p>
<p><code>B=['r1','t1','r2','t2','r3','t3']</code></p>
<p>I know it is easy to get <code>['r1','r2','r3']</code> by </p>
<p><code>['r'+str(k) for k in A]</code></p>
<p>How could I get B by one line loop as I showed above?</p>
<p>Many thanks.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a nested list comprehension.</p>
<pre><code>&gt;&gt;&gt; A=[1,2,3]                                                                                                                            
&gt;&gt;&gt; [fmt.format(n) for n in A for fmt in ('r{}', 't{}')]                                                                                    
['r1', 't1', 'r2', 't2', 'r3', 't3']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3/tutorial/datastructures.html#nested-list-comprehensions" rel="noreferrer">nested list comprehension</a>:</p>
<pre><code>A=[1,2,3]

B = [prefix + str(a) for a in A for prefix in 'rt']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <code>itertools.product</code></p>
<pre><code>import itertools
list(itertools.product(*[[1,2,3],['r','t']]))
Out[20]: [(1, 'r'), (1, 't'), (2, 'r'), (2, 't'), (3, 'r'), (3, 't')]
[y +str(x) for x, y in list(itertools.product(*[[1, 2, 3], ['r', 't']]))]
Out[22]: ['r1', 't1', 'r2', 't2', 'r3', 't3']
</code></pre>
