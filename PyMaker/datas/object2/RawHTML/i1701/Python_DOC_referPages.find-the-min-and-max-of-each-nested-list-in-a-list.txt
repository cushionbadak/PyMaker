<div class="post-text" itemprop="text">
<p>You can use the method <code>index</code> to get the index of a value of a list:
<a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/datastructures.html</a></p>
<pre><code>maxl = []
indexl = []
for inner_list in l:
    max_value = max(inner_list)
    maxl.append(max(max_value))
    indexl.append(inner_list.index(max_value)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Other option:</p>
<pre><code>l=[[5,6,7],[6,10,9,6],[2,3,1]]

maxs = [ max(s) for s in l ] #=&gt; [7, 10, 3]
max_idxs = [ s.index(max(s)) for s in l ] #=&gt; [2, 1, 1]
mins = [ min(s) for s in l ] #=&gt; [5, 6, 1]
mins_idxs = [ s.index(min(s)) for s in l ] #=&gt; [0, 0, 2]
</code></pre>
<p><hr/>
Or you can store the results into an array of dict, one liner:</p>
<pre><code>mapp = map(lambda x: {'max': max(x), 'max_idxs': x.index(max(x)), 'min': min(x), 'min_idxs': x.index(min(x)) }, l)

for k in mapp:
  print(k)

#=&gt; {'max': 7, 'max_idxs': 2, 'min': 5, 'min_idxs': 0}
#=&gt; {'max': 10, 'max_idxs': 1, 'min': 6, 'min_idxs': 0}
#=&gt; {'max': 3, 'max_idxs': 1, 'min': 1, 'min_idxs': 2}
</code></pre>
</div>

<p>You can use the following one-liner:</p>
<pre><code>l=[[5,6,7],[6,10,9,6],[2,3,1]]
maxL, index = zip(*[(max(subList), subList.index(max(subList))) for subList in l])
print(maxL) # will output (7, 10, 3)
print(index) # will output (2, 1, 1)
</code></pre>
