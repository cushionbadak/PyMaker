<p><strong>CSV in Editor looks:</strong></p>
<pre><code>Review Text;Review Rating
['much', 'write', 'exactly', 'supposed', 'filters', 'pop', 'sounds', 'recordings', 'much', 'crisp', 'one', 'lowest', 'prices', 'pop', 'filters', 'amazon', 'might', 'well', 'buy', 'honestly', 'work', 'despite', 'pricing'];5.0
</code></pre>
<p><strong>Read CSV:</strong></p>
<pre><code>import csv

with open("../Data_filtered/reviews_Musical_Instruments.csv", "r") as csv_file:
    reader = csv.reader(csv_file, delimiter=';', quotechar='|')
    liste = []
    for row in reader:
        liste.extend(row)
</code></pre>
<p><strong>Then I get:</strong> </p>
<pre><code>liste = 
['Review Text',
 'Review Rating',
 "['much', 'write', 'exactly', 'supposed', 'filters', 'pop', 'sounds', 'recordings', 'much', 'crisp', 'one', 'lowest', 'prices', 'pop', 'filters', 'amazon', 'might', 'well', 'buy', 'honestly', 'work', 'despite', 'pricing']",
 '5.0']
</code></pre>
<p><strong>But I need (without string):</strong></p>
<pre><code>liste = 
['Review Text',
 'Review Rating', 'much', 'write', 'exactly', 'supposed', 'filters', 'pop', 'sounds', 'recordings', 'much', 'crisp', 'one', 'lowest', 'prices', 'pop', 'filters', 'amazon', 'might', 'well', 'buy', 'honestly', 'work', 'despite', 'pricing',
 '5.0']
</code></pre>

<div class="post-text" itemprop="text">
<p>You have to parse it, using <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow noreferrer"><code>ast.literal_eval</code></a> is a good choice:</p>
<pre><code>import csv
import ast

with open("../Data_filtered/reviews_Musical_Instruments.csv", "r") as csv_file:
    reader = csv.reader(csv_file, delimiter=';', quotechar='|')
    liste = []
    for row in reader:
        row[2] = ast.literal_eval(row[2])
        liste.extend(row)
</code></pre>
