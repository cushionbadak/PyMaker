<div class="post-text" itemprop="text">
<p>Create a <code>dict</code> or a <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer">collections.Counter</a> and then inside <code>tabulateANYTHING</code> increment the value that has the key that matches <code>TYPE</code></p>
<p>For example, if your dict is named <code>quantities</code>:
<code>quantities[TYPE]+=1</code></p>
<p>For the files, create a <code>dict</code> of files and write to the one that has the key that matches <code>TYPE</code>.
Same thing for the file headers.</p>
