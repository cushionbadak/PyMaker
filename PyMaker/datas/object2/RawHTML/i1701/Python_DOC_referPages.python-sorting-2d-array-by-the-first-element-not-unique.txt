<p>You can use a custom <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer">sorting function</a> that would return the index at which the first element of a sublist is first found, e.g.:</p>
<pre><code>lst = [['text43','value43'],
       ['text23','value23'],
       ['text12','value12'],
       ['text43','different_val_43'],
       ['text12','another_value12'],
       ['text04','value04'],
       ['text43','anohter_value43']]

d = {}
for i, item in enumerate(lst):
    if item[0] not in d:
        d[item[0]] = i

lst.sort(key=lambda item: d[item[0]])
print(lst)
</code></pre>
<p>Output:</p>
<pre><code>[['text43', 'value43'],
 ['text43', 'different_val_43'],
 ['text43', 'anohter_value43'],
 ['text23', 'value23'],
 ['text12', 'value12'],
 ['text12', 'another_value12'],
 ['text04', 'value04']]
</code></pre>
