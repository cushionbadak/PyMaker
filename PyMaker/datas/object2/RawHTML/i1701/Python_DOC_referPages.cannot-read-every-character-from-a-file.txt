<p>You are calling <code>f.read(1)</code> <em>twice</em> each iteration, once in <code>while f.read(1)</code> and again for <code>print(f.read(1))</code>. You only print the second character, so you will only see <em>half</em> of the data being read.</p>
<p>If you want to see <em>each character</em> printed, call <code>f.read(1)</code> just <strong>once</strong> each iteration. Store the result in a variable and test against the variable, then print the variable:</p>
<pre><code>char = f.read(1)
while char:
    print(char)
    char = f.read(1)
</code></pre>
<p>You can avoid having to assign to <code>char</code> before the loop if you used <code>while True:</code> and break out when <code>f.read(1)</code> doesn't return data:</p>
<pre><code>while True:
    char = f.read(1)
    if not char:
        break
    print(char)
</code></pre>
<p>or you can use the <a href="https://docs.python.org/3/library/functions.html#iter" rel="nofollow noreferrer"><code>iter()</code> function with a sentinel</a>:</p>
<pre><code>for char in iter(lambda: f.read(1), ''):
    print(char)
</code></pre>
<p>If you wanted to read the <em>numbers</em> from the file, don't use <code>f.read(1)</code>. Just loop over the file and split each line on whitespace:</p>
<pre><code>with open("18.txt") as f:
    for line in f:
        print(line.split())
</code></pre>
<p>You probably want to convert the text to <em>integers</em> too, and build up your <code>grid</code> list of lists:</p>
<pre><code>grid = []
with open("18.txt") as f:
    for line in f:
        values = [int(v) for v in line.split()]
        grid.append(values)
</code></pre>
