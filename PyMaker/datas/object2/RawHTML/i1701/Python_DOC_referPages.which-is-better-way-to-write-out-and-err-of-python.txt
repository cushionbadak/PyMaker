<div class="post-text" itemprop="text">
<p>Leave the script the simple, original way:</p>
<pre><code># example.py
print('console output')
print(1/0)
</code></pre>
<p>Use easy trick in shell:</p>
<pre><code>$ python example.py &amp;&gt; log.txt
</code></pre>
<p>If you wanted some other custom behaviour on unhandled exceptions, set a <a href="https://docs.python.org/3/library/sys.html#sys.excepthook" rel="nofollow noreferrer"><code>sys.excepthook</code></a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>To combine both stdout and stderr into one file, which is also helpful when using more (or less) command</p>
<pre><code>python example.py 2&gt;&amp;1 &gt; output
python example.py 2&gt;&amp;1 | more
</code></pre>
<p>To make each go into separate files</p>
<pre><code>python example.py 2&gt; error &gt; output
</code></pre>
<p>To ignore stederr</p>
<pre><code>python example 2&gt;/dev/null
</code></pre>
<p>Hope this helps.</p>
