<div class="post-text" itemprop="text">
<p>The error <code>weka.core.UnsupportedAttributeTypeException: weka.classifiers.trees.J48: Cannot handle numeric class!</code> states that J48 algorithm cannot be used on numeric classes. Here class means the output that you want to learn, not an attribute used when learning. J48 can use numeric attributes but cannot predict numeric classes.</p>
<p>You cannot predict a numeric value in J48 (for example predicting temperature as an integer), you can only predict nominal types with J48 (eg one of cold/neutral/hot).</p>
<p>In order to use J48 you need to change the class to nominal.</p>
