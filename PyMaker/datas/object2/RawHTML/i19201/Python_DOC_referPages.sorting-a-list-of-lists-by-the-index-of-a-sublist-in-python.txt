<div class="post-text" itemprop="text">
<p>I'm in the middle of a larger script and need to order a list by the index of the sub lists. My root list contains sub-lists of numbers corresponding to [Latitude, Longitude, Elevation, Distance]. I need to sort the root list by Distance in the sub lists. Any ideas?</p>
</div>
<div class="post-text" itemprop="text">
<p>you can use an <a href="http://docs.python.org/3/library/operator.html?highlight=operator.itemgetter#operator.itemgetter" rel="nofollow"><code>operator.itemgetter</code></a> to sort the list based on an element of your list:</p>
<pre><code>import operator
lst = [...]
lst.sort(key=operator.itemgetter(3))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to say explicitly what is the value you want the list to be sorted by; in your case this 4th element of each element of your list. One way you can do it is to use <code>key</code> keyword parameter of the <code>sort</code> method and pass to it a function (lambda) which extracts that element: </p>
<pre><code>&gt;&gt;&gt; l = [[1,2,3,3],[1,2,3,2],[1,2,3,1]]
&gt;&gt;&gt; l.sort(key=lambda s: s[3])
&gt;&gt;&gt; l
[[1, 2, 3, 1], [1, 2, 3, 2], [1, 2, 3, 3]]
&gt;&gt;&gt; 
</code></pre>
</div>
