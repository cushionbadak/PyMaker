<div class="post-text" itemprop="text">
<p>In Python 3, <a href="http://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip()</code></a> returns an iterator, not a list. You can either cast it to a list:</p>
<pre><code>columns = list(zip(*rows))
float_columns = [map(float, col) for col in columns[1:]]
</code></pre>
<p>use <code>itertools.islice</code> to select everything but the first element, </p>
<pre><code>from itertools import islice

columns = zip(*rows)
float_columns = [map(float, col) for col in islice(columns, 1)]
</code></pre>
<p>or skip the first element with <code>next()</code>:</p>
<pre><code>columns = zip(*rows)
next(columns, None)  # skip the first column
float_columns = [map(float, col) for col in columns]
</code></pre>
<p>Note that <code>map()</code> returns an iterator as well, not a list. That may be fine for your purposes.</p>
<p>If you wanted to find the max of each column, for example, you can combine that:</p>
<pre><code>columns = zip(*rows)
next(columns, None)  # skip the first column
max_columns = [max(map(float, col)) for col in columns]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are using Python 3, in which zip returns an iterable, not a list.</p>
<p>So, by converting it into a list, you can solve your problem.</p>
<pre><code>columns = list(zip(*rows))
</code></pre>
<p>Also, You can skip the first column in the below manner, in which case, you can remove the slice.</p>
<pre><code>columns = zip(*rows)
next(columns, 0)
</code></pre>
