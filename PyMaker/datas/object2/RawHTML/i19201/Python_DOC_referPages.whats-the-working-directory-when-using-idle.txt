<div class="post-text" itemprop="text">
<p>So, I'm learning Python and would like to create a simple script to download a file from the internet and then write it to a file. However, I am using IDLE and have no idea what the working directory is in IDLE or how to change it. How can I do file system stuff in IDLE if I don't know the working directory or how to change it?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can easily check that yourself using <a href="http://docs.python.org/3/library/os.html#os.getcwd"><code>os.getcwd</code></a>:</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; os.getcwd()
'C:\\Program Files\\Python33'
</code></pre>
<p>That’s on my Windows machine, so it’s probably the installation directory of Python itself.</p>
<p>You can change that directory at runtime using <a href="http://docs.python.org/3/library/os.html#os.chdir"><code>os.chdir</code></a>:</p>
<pre><code>&gt;&gt;&gt; os.chdir('C:\\Users\\poke\\Desktop\\')
&gt;&gt;&gt; os.getcwd()
'C:\\Users\\poke\\Desktop'
&gt;&gt;&gt; with open('someFile.txt', 'w+') as f:
        f.write('This should be at C:\\Users\\poke\\Desktop\\someFile.txt now.')
</code></pre>
<p>This will—not surprisingly—create the file on my desktop.</p>
</div>

<pre><code>In [1]: import os

</code></pre>

<div class="post-text" itemprop="text">
<p>This will depend on OS and how IDLE is executed.</p>
<p>To change the (default) CWD in Windows, right click on the Short-cut Icon, go to "Properties" and change "Start In".</p>
