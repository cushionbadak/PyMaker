<div class="post-text" itemprop="text">
<p>Call <a href="http://docs.python.org/2/library/stdtypes.html#dict.values" rel="nofollow"><code>.values()</code></a> on the dictionary.</p>
<p>In Python 3, that'll return a <a href="http://docs.python.org/3/library/stdtypes.html#dict-views" rel="nofollow">dict view</a>, so call <code>list()</code> on that.</p>
<p>Python 2:</p>
<pre><code>&gt;&gt;&gt; d = {1:'Apple', 2:'Red', 3:'Purple', 4:'Green'}
&gt;&gt;&gt; d.values()
['Apple', 'Red', 'Purple', 'Green']
</code></pre>
<p>Python 3:</p>
<pre><code>&gt;&gt;&gt; d = {1:'Apple', 2:'Red', 3:'Purple', 4:'Green'}
&gt;&gt;&gt; d.values()
dict_values(['Apple', 'Red', 'Purple', 'Green'])
&gt;&gt;&gt; list(d.values())
['Apple', 'Red', 'Purple', 'Green']
</code></pre>
</div>
