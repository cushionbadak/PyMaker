<div class="post-text" itemprop="text">
<p>It's bad practice to create variables dynamically. You should use a <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow">dict</a> or a list instead, for example:</p>
<pre><code>paragraphs= {1: ['line1','line2'],
            2: ['line3','line4']}
</code></pre>
<hr/>
<p>If you insist on using variables, you can use <a href="https://docs.python.org/3.4/library/functions.html#globals" rel="nofollow"><code>globals()</code></a>:</p>
<pre><code>&gt;&gt;&gt;globals()['Paragraph1Line1']= 'some text'
&gt;&gt;&gt;Paragraph1Line1
'some text'
</code></pre>
