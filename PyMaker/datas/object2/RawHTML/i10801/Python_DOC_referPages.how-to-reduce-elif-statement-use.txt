<p>You can just use a <code>dict</code> object. </p>
<p>Example:</p>
<pre><code>words = {'ant': 'Ulf', 'back': 'Zuwp', 'ban' : 'Zul'} # etc
try:
    print(words[search])
except KeyError as e:
    print("No results were found for '" + search + "'")
</code></pre>

<p>Use a <code>dict</code> to map each input to the appropriate output.</p>
<pre><code>print("English to Exrian Dictionary")
d = {"ant": "Ulf",
     "back": "Zuwp",
     # etc
    }

search = input("Enter the word you would like to translate: ").lower()

if search in d:
    print(d[search])
else:
    print("No results were found for '" + search + "'")
</code></pre>
