<p>What about python's  <a href="https://docs.python.org/3/library/stdtypes.html#str.translate" rel="nofollow noreferrer"> string translate</a></p>
<pre><code>import string
with open("crypto.txt","r") as arquivo:
    data =  arquivo.read()
    out = data.translate(string.maketrans("abcdefghijklmnopqrstuvw","defghijklmnopqrstuvwxyz"))
    print out
</code></pre>
