<p>I have 2 python lists:</p>
<pre><code>&gt;&gt;&gt; a_list = [1, 2, 3]
&gt;&gt;&gt; b_list = ["a", "b"]
</code></pre>
<p>I want to interleave these two lists together into <code>c_list</code> such that <code>c_list</code> looks likes this:</p>
<pre><code>[1, "a", 2, "b", 3]
</code></pre>
<p>How best to do it??</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>result = []
while a_list and b_list:
  result.append(a_list.pop(0))
  result.append(b_list.pop(0))
result.extend(a_list)
result.extend(b_list)
</code></pre>
</div>
