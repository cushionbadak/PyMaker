<p>You are using mathematical operands on strings. Here is your fix:</p>
<pre><code>import math
x1 = int(input('Enter first number: '))
x2 = int(input('Enter second number: '))
x3 = int(input('Enter third number: '))
x4 = int(input('Enter fourth number: '))

if ( x4 == (x1 + x2 + x3) / 3):
    print('equal')
</code></pre>
<p>You simply have to cast the strings to int.</p>
