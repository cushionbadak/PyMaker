<div class="post-text" itemprop="text">
<p>In python 3 the module is <code>urllib.request.urlopen</code>. Use it as shown below-</p>
<pre><code>import urllib.request
# and then 
response = urllib.request.urlopen(myurl) 
</code></pre>
