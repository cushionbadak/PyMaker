<p>How can I print a list, say <code>[‘White’,’Yellow’,’Blue’,’Red’,’Orange’]</code> like a “list of words?” That means that I want my list to be printed like so:</p>
<pre><code>White
Yellow
Blue
Red
Orange
</code></pre>

<p>What about using a loop?</p>
<pre><code>&gt;&gt;&gt; colors = ['White', 'Yellow', 'Blue', 'Red', 'Orange']
&gt;&gt;&gt; for color in colors:

...

</code></pre>

<div class="post-text" itemprop="text">
<p>At least three obvious ways I can think of off-hand:</p>
<p>Straightforward print loop:</p>
<pre><code>for x in iterable:
    print(x)
</code></pre>
<p><code>print</code> with varargs unpacking and a newline separator:</p>
<pre><code>print(*iterable, sep="\n")
</code></pre>
<p>Explict join with newline:</p>
<pre><code>print("\n".join(iterable))
</code></pre>
<p>From here on out it gets more esoteric, but one particular case worth mentioning (because it's basically a faster version of the explicit loop that avoids trying to pull a complete iterator into memory at once, and avoids per-item bytecode execution on the CPython reference interpreter by pushing all the work to C):</p>
<pre><code>import sys

sys.stdout.writelines(map('{}\n'.format, iterable))
</code></pre>
</div>
