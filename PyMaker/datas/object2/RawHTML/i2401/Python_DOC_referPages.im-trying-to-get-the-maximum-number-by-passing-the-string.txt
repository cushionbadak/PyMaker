<div class="post-text" itemprop="text">
<p>There are couple of mistakes in your code : </p>
<ul>
<li>You need to remove the enclosing parentheses from the string first. </li>
<li>You need to split on <code>', '</code> and then convert to int also.</li>
</ul>
<p>Modified code : </p>
<pre><code>def find_color(input):
   input = input.replace("rgb","")[1:-1] # input = "125, 50, 175"
   return (max(map(int, input.split(', '))))
</code></pre>
<p>Also, since input is already a tuple(string of a tuple), you can also do <code>return (max(eval(input)))</code></p>
</div>
<div class="post-text" itemprop="text">
<p>split will create a list of strings - therefore you compare in alphabetical order. In alphabetical order '50' &gt; '175'.</p>
<p>You should convert the list to list of integers. </p>
<p>you can use <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer">map</a> for this task :</p>
<pre><code>def find_color(input):
   input = input.replace("rgb(","").replace(")","").replace(' ' , '' )
   return max(map(int,input.split(',')))
print(find_color("rgb(125, 50, 175)"))
</code></pre>
