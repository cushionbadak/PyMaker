<p>How can I dynamically define the cell location. For example, suppose I have a worksheet consisting of six rows. </p>
<p>I am able to get the rows count using <code>Count_Row = df.shape[0]</code> but not sure how do I reference <code>Count_Row</code> parameter in the below statement.</p>
<pre><code>worksheet.conditional_format('A1:A[Count_Row]',
                            {'type':'cell','criteria': '&gt;=','value':0,'format':format2})
</code></pre>
<p>Thanks</p>
</div>
<div class="post-text" itemprop="text">
<p>Simply use <code>.format</code> (or any other string interpolation method of your liking):</p>
<pre><code>worksheet.conditional_format('A1:A{}'.format(Count_Row),
                            {'type':'cell','criteria': '&gt;=','value':0,'format':format2})
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Same as DeepSpace's answer, but using <a href="https://docs.python.org/3/reference/lexical_analysis.html#formatted-string-literals" rel="nofollow noreferrer"><code>f-string</code></a> instead:</p>
<pre><code>worksheet.conditional_format(f'A1:A{Count_Row}',
                            {'type':'cell','criteria': '&gt;=','value':0,'format':format2})
</code></pre>
</div>
