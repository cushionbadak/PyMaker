<div class="post-text" itemprop="text">
<p>Say I want to download files from websites and files are listed by this style</p>
<pre><code>www.website.com/file001.mp3
www.website.com/file002.mp3
www.website.com/file003.mp3
.
.
.
www.website.com/file451.mp3
www.website.com/file452.mp3
www.website.com/file453.mp3
</code></pre>
<p>And I want to make some code that'll download these using a loop. </p>
<pre><code>for i in range(1, 454):
    downloadFunction('www.website.com/file00'+i+'.mp3')
</code></pre>
<p>This would not work when i is bigger than 9 because then the concatenation would look like </p>
<pre><code>'www.website.com/file0010.mp3' 
</code></pre>
<p>instead of </p>
<pre><code>'www.website.com/file010.mp3' 
</code></pre>
<p>I could code the case for each hundreth's value but I have a feeling there is a more elegant way to code this. </p>
</div>
<div class="post-text" itemprop="text">
<p>Using the a <a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="noreferrer">format</a> string, you could do:</p>
<pre><code>url_template = 'www.website.com/file{:03}.mp3'

for i in range(9, 12):
    print(url_template.format(i))
</code></pre>
<p>Output:</p>
<pre><code>www.website.com/file009.mp3
www.website.com/file010.mp3
www.website.com/file011.mp3
</code></pre>
<p>So, for your application:</p>
<pre><code>url_template = 'www.website.com/file{:03}.mp3'
for i in range(1, 454):
    downloadFunction(url_template.format(i))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3.7/library/stdtypes.html?highlight=zfill#str.zfill" rel="nofollow noreferrer"><code>zfill</code></a>, which is a string function that pads a string with leading zeroes. In your case:</p>
<pre><code>for i in range(1, 454):
    downloadFunction('www.website.com/file{}.mp3'.format(str(i).zfill(3)))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do <code>downloadFunction('www.website.com/file%03d.mp3' % i)</code>. The <code>%</code> is a placeholder and the <code>d</code> is a numeric format-specifier. The <code>03</code> tells Python to zero-pad up to three characters.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use:</p>
<pre><code>for i in range(1, 454):
    downloadFunction('www.website.com/file'+str(i).rjust(3,'0')+'.mp3')
</code></pre>
<p>instead.</p>
</div>
<span class="comment-copy">Hint: You can use the <code>.format</code> method to build your file names. Also, your existing code only works if <code>i</code> is a string, it will fail if it's an int.</span>
