<p>You can use <a href="https://docs.python.org/3/library/argparse.html" rel="nofollow noreferrer"><code>argparse</code></a> to parse command-line arguments in a structured way:</p>
<pre><code>from os import system
import sys
import argparse
import os

base_dir = '/root/Music/loop'

parser = argparse.ArgumentParser()
parser.add_argument('--all', action='store_true')
parser.add_argument('files', nargs='*')
args = parser.parse_args()
if args.all:
    files = os.listdir(base_dir)
else:
    files = args.files

def print_file(path):
    system('cat ' + path)

if __name__ == "__main__":
    for file in files:
        print_file(os.path.join(base_dir, file))
</code></pre>
