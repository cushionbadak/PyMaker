<p>There are a few things you can do to improve this:</p>
<ol>
<li><p>The string formatting methods don't require the formatting string to be separated from the surrounding text. The "right" way to use them is to embed the formatting codes into a longer string, then attach the values at the end.</p></li>
<li><p>You are using the older "printf" style formatting, but it is considered <a href="https://docs.python.org/3/library/stdtypes.html#printf-style-string-formatting" rel="nofollow noreferrer">preferable</a> to use the newer <code>str.format()</code> method for new code. </p></li>
<li><p>If you want to place the currency symbol close to your number, but also pad to the left of that up to a certain width, you will need to do it in two steps (first format the number with the currency symbol, then pad to the left), as described in <a href="https://stackoverflow.com/a/42658877/3830997">this answer</a>.</p></li>
</ol>
<p>With that in mind, here's some code that should do the job:</p>
<pre><code>money = 12323.45
random = 2278

# format with decimals and dollar sign (variable width)
money_formatted = '${:.2f}'.format(money)
monthly_money_formatted = '${:.2f}'.format(money/12.0)

print()
print("Random number:    {:9.0f}".format(random))
# pad to 12 characters if needed
print("Money each month: {:&gt;12}".format(monthly_money_formatted))
print("Total money:      {:&gt;12}".format(money_formatted))

# output:
# Random number:         2278
# Money each month:     $1026.95
# Total money:         $12323.45
</code></pre>
