<p>I'm trying to split a string by specific letters(in this case:'r','g' and'b') so that I can then later append them to a list. The catch here is that I want the letters to be copied to over to the list as well.</p>
<pre><code>string = '1b24g55r44r'
</code></pre>
<p>What I want:</p>
<pre><code>[[1b], [24g], [55r], [44r]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer">findall</a>:</p>
<pre><code>import re

print([match for match in re.findall('[^rgb]+?[rgb]', '1b24g55r44r')])
</code></pre>
<p><strong>Output</strong></p>
<pre><code>['1b', '24g', '55r', '44r']
</code></pre>
<p>The regex match:</p>
<ul>
<li><code>[^rgb]+?</code> everything that is not rgb one or more times</li>
<li>followed by one of <code>[rgb]</code>.</li>
</ul>
<p>If you need the result to be singleton lists you can do it like this:</p>
<pre><code>print([[match] for match in re.findall('[^rgb]+?[rgb]', '1b24g55r44r')])
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[['1b'], ['24g'], ['55r'], ['44r']]
</code></pre>
<p>Also if the string is only composed of <strong>digits</strong> and <code>rgb</code> you can do it like this:</p>
<pre><code>import re

print([[match] for match in re.findall('\d+?[rgb]', '1b24g55r44r')])
</code></pre>
<p>The only change in the above regex is <code>\d+?</code>, that means match one or more digits.</p>
<p><strong>Output</strong></p>
<pre><code>[['1b'], ['24g'], ['55r'], ['44r']]
</code></pre>
