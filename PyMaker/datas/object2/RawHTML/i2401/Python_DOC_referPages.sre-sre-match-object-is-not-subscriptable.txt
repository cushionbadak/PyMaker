<div class="post-text" itemprop="text">
<p>Your line here:</p>
<pre><code>if not {parsed_host[0]}.issubset(parsed_saved_host.group(1).split(',')):
</code></pre>
<p>attempts to access item <code>0</code> of <code>parsed_host</code>, a <code>Match</code> object.</p>
<p>Since the <code>Match.__getitem__()</code> method is implemented since Python 3.6 according to <a href="https://docs.python.org/3/library/re.html#re.Match.__getitem__" rel="nofollow noreferrer">the documentation</a>, your code should work in Python 3.6, and if you are getting a <code>'_sre.SRE_Match' object is not subscriptable</code> error, it means that you are not actually using Python 3.6, but an earlier version.</p>
<p>Change the line to:</p>
<pre><code>if not {parsed_host.group(0)}.issubset(parsed_saved_host.group(1).split(',')):
</code></pre>
<p>and the code would work for earlier versions of Python.</p>
