<p>I want convert input from QLineEdit and send to serial as Hex bytes.</p>
<p>Example:</p>
<p>QlineEdit <code>input</code> is <code>03040506</code></p>
<p>Write to serial as <code>0x03 0x04 0x05 0x06</code></p>
<p>Thanks,</p>
</div>
<div class="post-text" itemprop="text">
<p>You can easily do this using <a href="https://docs.python.org/3/library/stdtypes.html#bytes.fromhex" rel="nofollow noreferrer"><code>bytes.fromhex</code></a></p>
<pre><code>data = "03040506"
out = bytes.fromhex(data)
print(out)
</code></pre>
<p><strong>output</strong></p>
<pre><code>b'\x03\x04\x05\x06'
</code></pre>
<p>To send those bytes to the serial port, just do something like <code>ser.write(out)</code>, where <code>ser</code> is an open serial port.</p>
</div>
