<p>I want to combine a generator of generators so that when I call list() on said generator I get a list of the elements of the generators all toghether, not a list of generators. For example, I am currently doing:</p>
<pre><code>list(itertools.chain((i for i in range(j)) for j in range(3)))
</code></pre>
<p>which is returning:</p>
<pre><code>[&lt;generator object &lt;genexpr&gt;.&lt;genexpr&gt; at 0x7fc48921e200&gt;,
&lt;generator object &lt;genexpr&gt;.&lt;genexpr&gt; at 0x7fc48921eeb8&gt;,
&lt;generator object &lt;genexpr&gt;.&lt;genexpr&gt; at 0x7fc48921e7d8&gt;]
</code></pre>
<p>But i want it to return something like:</p>
<pre><code>[0, 0, 1, 0, 1, 2]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do like this,</p>
<pre><code>list(itertools.chain.from_iterable((i for i in range(j+1)) for j in range(3)))
# [0, 0, 1, 0, 1, 2]
</code></pre>
<p><code>range(j)</code> is not won't give your desired output. You have to use <code>range(j+1)</code></p>
<p>You could be done like this without using the iterator.</p>
<pre><code>In [58]: list(i for j in range(3) for i in range(j+1))
Out[58]: [0, 0, 1, 0, 1, 2]
</code></pre>
</div>
