<div class="post-text" itemprop="text">
<p>You can use above line of code in "__init__.py" file of a package demopkg1.
"__init__.py" file contains:</p>
<pre><code>import pkgutil
__path__ = pkgutil.extend_path(__path__, __name__)
</code></pre>
<p>Now use another file test.py with below line of code:</p>
<pre><code>import demopkg1
print ('demopkg1 file          :', demopkg1.__file__)
print ('demopkg1 path          :', demopkg1.__path__)
</code></pre>
<p>When you will execute the file test.py which include package "demopkg1" give below output:</p>
<pre><code>demopkg1 file          : E:\Work\Python\demopkg1\__init__.py
demopkg1 path          : ['E:\\Work\\Python\\demopkg1']
</code></pre>
