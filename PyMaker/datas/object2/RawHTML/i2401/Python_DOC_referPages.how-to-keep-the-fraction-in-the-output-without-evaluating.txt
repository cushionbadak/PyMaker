<p>I try to find out the <strong>Characteristic Polynomial</strong> and <strong>Eigen values</strong> using <code>sympy</code>. But when I print the results in Jupyter notebook all the fractions in the coefficients of the polynomials have been evaluated as <code>float</code>(e.g-<code>0.5x**2+1)</code> but I want <code>$(1/2)x^2+1$)</code>. Is there anything to solve this problem. Thank you.</p>

<div class="post-text" itemprop="text">
<p>these are the relevant chages</p>
<pre><code>from fractions import Fraction
M = sp.Matrix([[0, -Fraction(1, 2)], [Fraction(1, 3), 0]])
</code></pre>
<p>which yield</p>
<pre><code>PurePoly(x**2 + 1/6, x, domain='QQ')
</code></pre>
<p>the point is that <code>1/3</code> is immediately converted to the <code>float</code> 0.3333... the way to avoid that is to use <a href="https://docs.python.org/3/library/fractions.html?module-fractions" rel="nofollow noreferrer"><code>fractions.Fraction</code></a> or (probably better) the sympy version thereof: <a href="https://docs.sympy.org/latest/modules/core.html?rational#rational" rel="nofollow noreferrer"><code>sympy.Rational</code></a>.</p>
