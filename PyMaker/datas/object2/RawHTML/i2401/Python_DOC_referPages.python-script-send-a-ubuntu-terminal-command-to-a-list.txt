<div class="post-text" itemprop="text">
<p>I think, you could use subprocess module. This module replaces some older modules and functions. Official document is here: <a href="https://docs.python.org/3/library/subprocess.html#module-subprocess" rel="nofollow noreferrer">https://docs.python.org/3/library/subprocess.html#module-subprocess</a></p>
<pre><code>import subprocess
res = subprocess.Popen('bgpq3 -4 AS-YAHOO-JP-2 -m 24 -l Google', shell=True,
                       universal_newlines=True,
                       stdout=subprocess.PIPE).communicate()[0]

for line in res.split('\n'):
    print(line)
</code></pre>
