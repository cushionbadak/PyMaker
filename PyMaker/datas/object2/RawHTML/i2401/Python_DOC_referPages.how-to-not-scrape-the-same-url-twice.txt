<p>If I understand the problem well, you are trying to see if each URL found has been done already or not, using:</p>
<pre><code>if value != done:
</code></pre>
<p>Except the problem is that the above allows the check for only one <code>done</code> url instead of the potentially multiple urls already done. So, if <code>done</code> becomes a list, you can use the operator <a href="https://docs.python.org/3/library/operator.html#mapping-operators-to-functions" rel="nofollow noreferrer"><code>in</code></a> (here you will need <code>not in</code> because we want to check if it's not there):</p>
<pre><code>if value not in done:
</code></pre>
<hr/>
<p>As a side note, lists in Python are created using square brackets, so, <code>done</code> will be should similar to something like this:</p>
<pre><code>done = [
    "http://localhost/2.php",
    "http://localhost/next1.php",
    "http://localhost/next2.php"
]
</code></pre>
<p>Braces are for <a href="https://docs.python.org/3/tutorial/datastructures.html#sets" rel="nofollow noreferrer">set</a>s and <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">dictionaries</a>, although making <code>done</code> a set doesn't matter too much here.</p>
