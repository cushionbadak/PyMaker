<div class="post-text" itemprop="text">
<p><code>output.stdout</code> is a file object. You can use the <code>read</code> method of the file object to get the content of the output:</p>
<pre><code>print(output.stdout.read())
</code></pre>
<p>or you can use the <a href="https://docs.python.org/3/library/subprocess.html#subprocess.Popen.communicate" rel="nofollow noreferrer"><code>Popen.communicate</code></a> method instead:</p>
<pre><code>stdout, stderr = output.communicate()
print(stdout)
</code></pre>
