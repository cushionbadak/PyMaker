<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/2/library/codecs.html#python-specific-encodings" rel="nofollow"><code>string-escape</code></a> encoding instead of <code>unicode-escape</code>:</p>
<pre><code>&gt;&gt;&gt; print string1.decode('string-escape')
&lt;value&gt;UK LONDON 8CS Flr01 Rm1ğ&lt;/value&gt;
</code></pre>
<p>or use decode it using <code>unicode-escape</code>, then encode it using <code>latin1</code> encoding:</p>
<pre><code>&gt;&gt;&gt; print string1.decode('unicode-escape').encode('latin1')
&lt;value&gt;UK LONDON 8CS Flr01 Rm1ğ&lt;/value&gt;
</code></pre>
<p><strong>NOTE</strong> <code>string-escape</code> is only available in Python 2.x.</p>
