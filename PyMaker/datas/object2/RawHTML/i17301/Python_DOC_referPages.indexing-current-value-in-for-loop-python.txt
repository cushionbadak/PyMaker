<div class="post-text" itemprop="text">
<p>So I have a for loop.</p>
<pre><code>&gt;&gt;&gt; row = [5, 2, 5, 4, 2, 2, 5, 5, 5, 2]
&gt;&gt;&gt; for i in row:
    if i == 5:
        print(row.index(i))
    if i == 2:
        print(row.index(i))
</code></pre>
<p><strong>OUTPUT</strong> </p>
<pre><code>0
1
0
1
1
0
0
0
1
</code></pre>
<p>I want to get: <code>0 1 2 4 5 6 7 8 9</code>.
In other words, I want to get the index of the <code>i</code> I'm currently looking at in the <code>for</code> loop, if it is = 5 or 2... any easy way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="http://docs.python.org/3/library/functions.html#enumerate"><code>enumerate()</code> function</a> to produce a running index:</p>
<pre><code>for index, i in enumerate(row):
    if i == 5:
        print(index)
    if i == 2:
        print(index)
</code></pre>
<p>or simpler still:</p>
<pre><code>for index, i in enumerate(row):
    if i in (2, 5):
        print(index)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>For completeness sake, an alternative if one didn't want to use <code>enumerate</code> for some reason:</p>
<pre><code>row = [5, 2, 5, 4, 2, 2, 5, 5, 5, 2]
indices = filter(lambda idx: row[idx] in (5,2), xrange(len(row)))
# [0, 1, 2, 4, 5, 6, 7, 8, 9]
</code></pre>
<p>Or as a list-comp:</p>
<pre><code>indices = [idx for idx in xrange(len(row)) if row[idx] in (5, 2)]
</code></pre>
</div>
