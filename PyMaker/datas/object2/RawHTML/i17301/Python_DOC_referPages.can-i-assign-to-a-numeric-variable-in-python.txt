<p>I started working on my simple program and when I open it I get an error that says this can't assign to literal. Here's my code:</p>
<pre><code>print('press 2 to play')
2 = input('you won')
print(2)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="http://docs.python.org/2/reference/expressions.html" rel="nofollow">The rule for Python expressions is that names have to have to be of the form:</a></p>
<pre><code>identifier ::=  (letter|"_") (letter | digit | "_")*
</code></pre>
<p>Where:</p>
<pre><code>letter     ::=  lowercase | uppercase
lowercase  ::=  "a"..."z"
uppercase  ::=  "A"..."Z"
digit      ::=  "0"..."9"
</code></pre>
<p>So you cannot assign a to a digit, because those are reserved for integers and other numeric types.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can not create a variable with numbers. </p>
<pre><code>print ('press 2 to play') 
var = input('you won') 
print (var)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<ol>
<li>a variable name <a href="https://stackoverflow.com/questions/342152/why-cant-variable-names-start-with-numbers">can't start with numbers</a>;</li>
<li><code>=</code> is an assignment operator, you should use <code>==</code> to do comparison.</li>
</ol>
<p>e.g.:</p>
<pre><code>n = input('press 2 to play') #accepts user input, store to a variable
if n == '2': #compare equality with "=="
  print('you won')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You cannot assign to a literal basically means that you cannot assign it to something that can't change. For example, open your python shell and type in <code>2 = 3</code>. It will raise the same error. Edit your code as such:</p>
<pre><code>two = int(input('press 2 to play'))
if two == 2:
    print('you won')
    print(2)
</code></pre>
<p><code>2</code> cannot be stored as a variable because it is a number. You can change the <code>2</code> to a <code>two</code>, because <code>two</code> does not contain a value already.</p>
