<div class="post-text" itemprop="text">
<p>In Python how do I convert:</p>
<p><code>list01 = [ ['a', 'b', 'c'], ['i', 'j', 'k'], ['x', 'y', 'z'] ]</code></p>
<p>into </p>
<p><code>list02 = [ 'abc', 'ijk', 'xyz']</code> </p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/2/library/functions.html#map"><code>map</code></a>:</p>
<pre><code>map(''.join, list01)
</code></pre>
<p>Or with a <a href="http://docs.python.org/2/tutorial/datastructures.html#list-comprehensions">list comprehension</a>:</p>
<pre><code>[''.join(x) for x in list01]
</code></pre>
<p>Both output:</p>
<pre><code>['abc', 'ijk', 'xyz']
</code></pre>
<hr/>
<p>Note that in Python 3, <code>map</code> returns a map object instead of a list. If you do need a list, you can wrap it in <code>list(map(...))</code>, but at that point a list comprehension is more clear.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join</code></a> and a <a href="http://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; list01 = [ ['a', 'b', 'c'], ['i', 'j', 'k'], ['x', 'y', 'z'] ]
&gt;&gt;&gt; [''.join(x) for x in list01]
['abc', 'ijk', 'xyz']
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; map(''.join, list01)
['abc', 'ijk', 'xyz']
</code></pre>
</div>

<p>Hi you can use join to implode the elements in a string and then append. If you dont want to use map.</p>
<pre><code>    # Your list
    someList = [ ['a', 'b', 'c'], ['i', 'j', 'k'], ['x', 'y', 'z'] ]
    implodeList = []

    # make an iteration with for in
    for item in someList :
        implodeList.append(''.join(item))

    # printing your new list
    print (implodeList)
</code></pre>
