<div class="post-text" itemprop="text">
<p>On a project I am currently getting returned a list as follows:</p>
<pre><code>[u'40620', u'00700', u'24150', u'11700']
</code></pre>
<p>How can I edit the list so it is returned just integer values:</p>
<pre><code>[40620, 00700, 24150, 11700]
</code></pre>
<p>Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="http://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> and <a href="http://docs.python.org/3/library/functions.html#int" rel="nofollow"><code>int</code></a>:</p>
<pre><code>&gt;&gt;&gt; lst = [u'40620', u'00700', u'24150', u'11700']
&gt;&gt;&gt; [int(x) for x in lst]
[40620, 700, 24150, 11700]
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Simple one liner:</p>
<pre><code>results = map(int, results)
</code></pre>
</div>
