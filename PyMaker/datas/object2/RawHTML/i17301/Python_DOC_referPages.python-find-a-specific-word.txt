<div class="post-text" itemprop="text">
<p>Use <code>\b</code> to mark the border of some word:</p>
<pre><code>In [782]: s = 'foreign, for fore'

In [783]: re.findall(r'\bfor\b', s)
Out[783]: ['for']
</code></pre>
<p>From the <a href="http://docs.python.org/3/howto/regex.html#more-metacharacters" rel="nofollow">docs</a>:</p>
<blockquote>
<p>\b is an assertion that the current position is located at a word boundary</p>
</blockquote>

<p>Do you just want to check if the string contains the word or part of it?
the pythonic way to do it is:</p>
<pre><code>In [15]: s = 'foreign, for fore'

In [16]: 'for' in s
Out[16]: True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You want to get a boolean? How about <code>bool(re.findall(r'for', str))</code>? (By the way, an <code>if</code> statement would do that implicitely, so <code>if re.findall(r'for', str):</code> is ok).</p>
<p>If you don't need to use a regexp, you can also use: <code>'for' in str</code> (which “returns” a boolean).</p>
</div>
<div class="post-text" itemprop="text">
<p>If you insist on using <code>regex</code> to check if a substring is in a string, try:</p>
<pre><code>&gt;&gt;&gt; import re
# If input is a sentence
&gt;&gt;&gt; s = "foreign"
&gt;&gt;&gt; substr = r"for"
&gt;&gt;&gt; [i for i in s.split() if re.findall(substr,i)]
['foreign']

&gt;&gt;&gt; s = "forex is a foreign concept ."
&gt;&gt;&gt; [i for i in s.split() if re.findall(substr,i)]
['forex', 'foreign']

# If input is a list of words
&gt;&gt;&gt; s = ['forex', 'is', 'a', 'foreign', 'concept', '.']
&gt;&gt;&gt; [i for i in s if re.findall(substr,i)]
['forex', 'foreign']
</code></pre>
<p>If you want to check the existence of a substring in a string and return the string:</p>
<pre><code>&gt;&gt;&gt; s = "foreign"
&gt;&gt;&gt; substr = "for"
&gt;&gt;&gt; [i for i in s.split() if substr in s]
['foreign']
</code></pre>
<p>If you want to check the existence of a substring in a string and return a boolean:</p>
<pre><code>&gt;&gt;&gt; s = "foreign"
&gt;&gt;&gt; substr = "for"
&gt;&gt;&gt; substr in s
True
&gt;&gt;&gt; "for" in "foreign"
True
</code></pre>

</code></pre>
