<div class="post-text" itemprop="text">
<p>The <a href="http://docs.python.org/3/library/functions.html#print"><code>print()</code> function</a> <strong>adds</strong> whitespace between the arguments, there is nothing to strip there.</p>
<p>Use <code>sep=''</code> to stop the function from doing that:</p>
<pre><code>print(name, ', you won!', sep='')
</code></pre>
<p>or you can use <a href="http://docs.python.org/3/library/string.html#formatstrings">string formatting</a> to create one string to pass to <code>print()</code>:</p>
<pre><code>print('{}, you won!'.format(name))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The white space is being added by <code>print</code> because you're passing it two parameters and this is how it works.</p>
<p>Try this:</p>
<pre><code>print(name, ', you won!', sep='')
</code></pre>
<p>Another way would be doing some <em>string formatting</em> like:</p>
<pre><code>print('%s, you won!' % (name))    # another way of formatting.
</code></pre>
