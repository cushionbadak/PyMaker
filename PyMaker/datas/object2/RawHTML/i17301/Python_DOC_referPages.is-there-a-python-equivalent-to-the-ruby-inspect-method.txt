<p>In ruby you can inspect any object with the inspect method:</p>
<p>For example:</p>
<pre><code>print [1,"string",:symbol,[?l, ?i, ?s, ?t]].inspect
</code></pre>
<p>will print</p>
<pre><code>[1, "string", :symbol, ["l", "i", "s", "t"]]
</code></pre>
<p>Is there any similar facility in python which allows me to print the content of some arbitrary variable?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/functions.html#repr" rel="nofollow"><code>repr</code></a>. It returns a string containing a printable representation of an object. (similar to <a href="http://ruby-doc.org/core-2.1.1/Object.html#method-i-inspect" rel="nofollow"><code>Object#inspect</code></a> in Ruby)</p>
<pre><code>&gt;&gt;&gt; repr([1,"string", ':symbol', ['l', 'i', 's', 't']])
"[1, 'string', ':symbol', ['l', 'i', 's', 't']]"
</code></pre>
<p>BTW, there's no symbol literal (<code>:symbol</code>) or single character string literal (<code>?x</code>) in Python; replaced them with string literals in the above example.</p>
</div>
