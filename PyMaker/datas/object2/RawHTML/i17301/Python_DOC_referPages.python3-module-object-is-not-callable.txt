<div class="post-text" itemprop="text">
<p><code>html.parser</code> is the module; you want the <a href="http://docs.python.org/3/library/html.parser.html#html.parser.HTMLParser"><code>HTMLParser</code> class</a> <em>in</em> that module:</p>
<pre><code>parser = html.parser.HTMLParser(AbstractFormatter(DumbWriter(StringIO())))
</code></pre>
<p>or more succinctly:</p>
<pre><code>from html.parser import HTMLParser

parser = HTMLParser(AbstractFormatter(DumbWriter(StringIO())))
</code></pre>
