<h3>Attempt</h3>
<p>I tried to plot some histograms in <strong>python3.7</strong> with <strong>matplotlib 3.0.3</strong> and got a problem:</p>
<h3>Code:</h3>
<pre><code>ind = np.arange(2)
width = 0.35
data = [(0.5, 0.1), (0.8, 0.3)]
for i in data:
    plt.bar(ind, i[0], width, yerr=i[1])
plt.ylabel('scratchwidth /cm')
plt.show
</code></pre>
<h3>Outputs:</h3>
<p>I expect a plot with two bars, <code>(0|0.5)</code> and <code>(1|0.8)</code> with the uncertainties <code>0.1</code> and <code>0.3</code>. What I get is two bars, both of them with <strong>y=0.8</strong> and an uncertainty of <strong>0.3</strong>. Does <code>plt.bar()</code> not work in a for-loop? 
How do I solve this problem? </p>
</div>
<div class="post-text" itemprop="text">
<p>You must pass an array of Y's and an array of Err's to the function <code>bar</code>. Transform your data from an array of points to an array of arrays of Y's and Err's:</p>
<pre><code>ind = np.arange(2)
width = 0.35
data = [(0.5, 0.1), (0.8, 0.3)]
y, err = list(zip(*data)) # Transpose the data array
plt.bar(ind, y, width, yerr=err)
plt.ylabel('scratchwidth /cm')
plt.show()
</code></pre>
<p>Or, since you already use numpy, use a numpy array:</p>
<pre><code>....
data = np.array([(0.5, 0.1), (0.8, 0.3)])
plt.bar(ind, data[:,0], width, yerr=data[:,1])
....
</code></pre>
