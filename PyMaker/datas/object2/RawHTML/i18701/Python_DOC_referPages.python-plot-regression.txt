<div class="post-text" itemprop="text">
<p>Give a list of tuples, creating lists of the first and second elements in the tuple can be done with a list comprehension, as follows:</p>
<pre><code>&gt;&gt;&gt; z = [(1,3),(5,8),(2,4)]
&gt;&gt;&gt; [a for a,b in z]
[1, 5, 2]
&gt;&gt;&gt; [b for a,b in z]
[3, 8, 4]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/functions.html#zip" rel="nofollow">zip</a>:</p>
<pre><code>&gt;&gt;&gt; zip(*[(1,3),(5,8),(2,4)])
&lt;zip object at 0x7f377bebe050&gt;
&gt;&gt;&gt; list(_)
[(1, 5, 2), (3, 8, 4)]
&gt;&gt;&gt; xpoints, ypoints = zip(*[(1,3),(5,8),(2,4)])
&gt;&gt;&gt; xpoints
(1, 5, 2)
&gt;&gt;&gt; ypoints
(3, 8, 4)
</code></pre>
