<p>Just use :</p>
<pre><code>check_output(['./MyFile'])
</code></pre>
<p>You've defined your own function, it's not an attribute of <code>subprocess</code> module(for Python 2.6 and earlier).</p>
<p>You can also assign the function to the imported module object(but that's not necessary):</p>
<pre><code>subprocess.check_output = check_output
location = "%s/folder" % (os.environ["Home"])
subprocess.check_output(['./MyFile'])
</code></pre>

<div class="post-text" itemprop="text">
<p>You <em>probably</em> just want to use <code>check_output</code>, but, just so you know, there is a method <code>subprocess.check_output</code>, but it's not defined until Python 2.7 (<a href="http://docs.python.org/3/library/subprocess.html#subprocess.check_output" rel="nofollow noreferrer">http://docs.python.org/3/library/subprocess.html#subprocess.check_output</a>)</p>
<p>You might even want this, which defines the function in the module if it's not there (i.e. running before v2.7).</p>
<pre><code>try: subprocess.check_output
except: subprocess.check_output = check_output
subprocess.check_output()
</code></pre>
