<div class="post-text" itemprop="text">
<p><code>'UID: {}\n'.format(stat(searchFolder).st_uid)</code>. If there is no <code>{}</code>, it just returns <code>{}: \n</code>.</p>
<p>This is string format. The <code>{}</code> represents a replacement fields. You may read the <a href="http://docs.python.org/3/library/string.html#format-string-syntax" rel="nofollow">doc</a>.</p>
</div>

<p>change</p>
<pre><code>f.write('UID: \n'.format(stat(searchFolder).st_uid))
f.write('{}\n'.format(pathName))
f.write('Size in Bytes: {}\n\n'.format(os.path.getsize(pathName)))
</code></pre>
<p>into</p>
<pre><code>f.write('UID: {0}\n'.format(stat(searchFolder).st_uid))
f.write('{0}\n'.format(pathName))
f.write('Size in Bytes: {0}\n\n'.format(os.path.getsize(pathName)))
</code></pre>
