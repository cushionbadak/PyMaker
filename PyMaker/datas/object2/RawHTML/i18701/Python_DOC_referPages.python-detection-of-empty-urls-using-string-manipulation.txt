<p>You can use <code>re</code>. You'd better read the <a href="http://docs.python.org/3/library/re.html" rel="nofollow">documentation</a>.</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; s='href=""adjfweofhref="   "'
&gt;&gt;&gt; pattern = re.compile(r'href=[\"\']\s*[\"\']')
&gt;&gt;&gt; pattern.findall(s)
['href=""', 'href="   "']
