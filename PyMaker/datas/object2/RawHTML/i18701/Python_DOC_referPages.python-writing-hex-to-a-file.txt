<div class="post-text" itemprop="text">
<p>In Python 3, you must explicitly specify an encoding if you are trying to <code>write</code> a <code>str</code>. What you are trying to write are byte literals (see the <a href="http://docs.python.org/3/howto/pyporting.html" rel="nofollow">python 2 to 3 guide</a>), so change the code to actually use byte literals</p>
<pre><code>junk = b'\x90' * 5
junk += b'\xcc' * 5

fo.write(junk)
</code></pre>
