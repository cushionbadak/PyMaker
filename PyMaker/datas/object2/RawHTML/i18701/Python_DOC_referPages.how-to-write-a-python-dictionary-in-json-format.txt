<div class="post-text" itemprop="text">
<p>There is an inbuilt json module which can be used to convert dictionaries to json format.</p>
<p>so:</p>
<pre><code>import json

json.dumps({'a':1,'b':2}) #this will return a string with your dict in json format.

#'{"a": 1, "b": 2}'
</code></pre>
<p>Here's <a href="http://docs.python.org/3/library/json.html" rel="nofollow">a link</a> incase you want to learn more about the module and explore other functions.</p>

<div class="post-text" itemprop="text">
<p><strong>simplejson</strong> (or json), is a built in python library,quite useful for this purpose.</p>
<pre><code>&gt;&gt;&gt; d = {'a':1,'b':2,'c':3}
&gt;&gt;&gt; import simplejson

&gt;&gt;&gt; # To get a JSON string representation
&gt;&gt;&gt; simplejson.dumps(d)

&gt;&gt;&gt; # To directly add the JSON data to a file
&gt;&gt;&gt; simplejson.dump(d,open("json_data.txt",'w'))
</code></pre>
<p>There are more interesting possibilities, like sorting the keys before dumping to JSON:</p>
<pre><code>&gt;&gt;&gt; print(simplejson.dumps({"c": 3, "b": 2, "a": 1}, sort_keys=True))
</code></pre>
<p>A link to the latest docs - <a href="http://simplejson.readthedocs.org/en/latest/" rel="nofollow">here</a>.</p>
