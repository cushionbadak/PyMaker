<p>This works with Python2:</p>
<pre><code>import Tkinter as tk
import ImageTk

def show_image():
    x = canvas.create_image(125, 125, image=tk_img)
    while True:
        print('show')
        canvas.itemconfigure(x, state=tk.NORMAL)
        button.configure(text = 'Hide')
        yield
        print('hide')        
        canvas.itemconfigure(x, state=tk.HIDDEN)
        button.configure(text = 'Show')        
        yield

root = tk.Tk()
canvas = tk.Canvas(root, width=250, height=250)
canvas.grid(row=0, column=0)
tk_img = ImageTk.PhotoImage(file='image.png')

button = tk.Button(
    root, text="Show", command=show_image().next, anchor='w',
    width=10, activebackground="#33B5E5")
button.grid(row=1, column=0)
root.mainloop()
</code></pre>
<p>In Python3, <code>PhotoImage</code> <a href="http://docs.python.org/3/library/tkinter.html#images" rel="nofollow">can open GIF, PPM/PGM images</a>. To open other formats, you may <a href="https://pypi.python.org/pypi/Pillow/2.1.0" rel="nofollow">need to install Pillow</a> (a fork of the PIL project for Python3).</p>
