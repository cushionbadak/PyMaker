<p><a href="http://www.crummy.com/software/BeautifulSoup/bs4/doc/#contents-and-children" rel="nofollow"><code>.contents</code></a> returns a list of items from a tag. For example:</p>
<pre><code>&gt;&gt;&gt; from bs4 import BeautifulSoup as BS
&gt;&gt;&gt; soup = BS('&lt;span class="foo"&gt; bar baz &lt;a href="http://foo.com"&gt;link&lt;/a&gt;&lt;/span&gt;')
&gt;&gt;&gt; print soup.find('span').contents
[u' bar baz ', &lt;a href="http://foo.com"&gt;link&lt;/a&gt;]
</code></pre>
<p><code>[0]</code> is used to access the first element of the list <code>.contents</code> returns. In the example above, it will return <code>bar baz</code></p>
<p><a href="http://docs.python.org/2.7/library/re.html#re.MatchObject.group" rel="nofollow"><code>.group(1)</code></a> returns the second (indexing starts at 0, remember) matched value from a regular expression. Looking at your regular expression, it returns the second digit from something that looks like <code>n1,n2</code>.</p>
