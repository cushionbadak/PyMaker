<p>You should read <a href="http://docs.python.org/3/library/configparser.html" rel="nofollow">the documentation for the <code>configparser</code> module</a>, it does a decent job explaining things. Be sure to read the docs for the version of Python 3 you are using though, as the interface seems to have been extended quite a bit between versions.</p>
<p>Starting in Python 3.2, it is possible to treat a <code>ConfigParser</code> instance as if it was a Python dictionary (the term used in the docs is the "mapping protocol"). That means, you can probably just create a class instance with <code>FopsTest(**config)</code>, or perhaps <code>FopsTest(**config["TheSectionICareAbout"])</code> if you only want the items from a specific section of the file. See <a href="http://docs.python.org/3.3/library/configparser.html#mapping-protocol-access" rel="nofollow">this section of the docs</a> for details.</p>
