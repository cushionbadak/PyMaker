<p>I'm looking for the syntax I would use to print both text and a saved int in the input function. I'm making a program that asks the user math questions using dynamic terms, and I want to be able to print those and some text on the same line.</p>
<p>I've tried this code, hoping that the parameters would be represented like it were a print statement:</p>
<pre><code>answer = int(input(oneTerm, '+', twoTerm, '='))
</code></pre>
<p>I also tried using two lines like this:</p>
<pre><code>print(oneTerm, '+', twoTerm, '=')
answer = int(input(''))
</code></pre>
<p>But the problem with this is the input is accepted on the line below the print statement, but I need it to be on the same line.</p>
<p>Any help is greatly appreciated!</p>

<div class="post-text" itemprop="text">
<p>The <code>input()</code> function takes only <em>one</em> string argument (and not multiple, like <code>print()</code>). Use string formatting:</p>
<pre><code>answer = int(input('{} + {} = '.format(oneTerm, twoTerm)))
</code></pre>
<p>The <code>{}</code> placeholders are replaced by the two arguments to <code>.format()</code>, creating essentially the same string to prompt the user for input.</p>
<p>See the <a href="http://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format()</code> method</a> documentation for more details.</p>
</div>
