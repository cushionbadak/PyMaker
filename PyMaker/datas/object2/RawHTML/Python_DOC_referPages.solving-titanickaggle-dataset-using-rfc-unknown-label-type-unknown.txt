<div class="post-text" itemprop="text">
<p>The result of <a href="https://scikit-learn.org/stable/modules/generated/sklearn.model_selection.train_test_split.html" rel="nofollow noreferrer">train_test_split()</a> is X_train, X_test, y_train, y_test and you are assigning values in a different order in your code.</p>
<p>Change the line:</p>
<pre><code>trainX,trainY,valX,valY = train_test_split(X,y,random_state = 1)  
</code></pre>
<p>By this one:</p>
<pre><code>trainX,valX,trainY,valY = train_test_split(X,y,random_state = 1)  
</code></pre>
<p>and you will be able to fit the model with the correct parameters.</p>
