<div class="post-text" itemprop="text">
<p><code>enum</code> is not a class you can inherit from <code>Enum</code> in the <code>enum</code> module is (see <a href="https://docs.python.org/3/library/enum.html" rel="nofollow noreferrer">the documentation</a>).</p>
<p>The <code>enum()</code> in the code you are describing is a callable object that has previously been put into the list <code>chosenEnums</code>. We can't tell you what it is, or does, without the rest of the code.</p>
