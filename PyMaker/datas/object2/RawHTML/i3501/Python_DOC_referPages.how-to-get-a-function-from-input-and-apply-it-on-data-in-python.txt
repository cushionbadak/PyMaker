<div class="post-text" itemprop="text">
<p>You can use a dictionary to link to logic already defined:</p>
<pre><code>a = [12,34,2,3,4,23,16,84,3]

def myfun(X):
    return([max(X),min(X),sum(X)])

dispatcher = {'max_min_sum': myfun}

b = input('Enter a dispatcher key:\n')

print(dispatcher[b](a))
</code></pre>
<p>Demo input / output:</p>
<pre><code>Enter a dispatcher key:
max_min_sum
[84, 2, 181]
</code></pre>
</div>

<p>How are you getting the input exactly?</p>
<p>If it helps, just want to point out that functions, like numbers and strings, are objects in python, and can be treated as parameters or return values.</p>
<pre><code>a = [12,34,2,3,4,23,16,84,3]

def dostuff(func):
    return func(a)

def myfun(X):
    return([max(X),min(X),sum(X)])

print(dostuff(myfun))
</code></pre>
<p>outputs</p>
<pre><code>[84, 2, 181]
</code></pre>
