<div class="post-text" itemprop="text">
<p>In Python 3.5, using keyword arguments in <code>str.format</code> has been deprecated:</p>
<pre><code>"Hi {s}".format(s="world")
</code></pre>
<p>From the <a href="https://docs.python.org/3/library/string.html#string.Formatter.format" rel="nofollow noreferrer"><code>string</code> docs</a>:</p>
<blockquote>
<p><em>Deprecated since version 3.5</em>: Passing a format string as keyword argument <code>format_string</code> has been deprecated.</p>
</blockquote>
<p>What are the best alternatives in Python 3.5+?</p>
</div>
<div class="post-text" itemprop="text">
<p>Deprecation is about <code>string.Formatter</code> and not about <code>str.Formatter</code>:</p>
<p><a href="https://python.readthedocs.io/en/stable/whatsnew/3.5.html#deprecated-python-modules-functions-and-methods" rel="nofollow noreferrer">Source:</a></p>
<blockquote>
<p>Passing a format string as keyword argument <code>format_string</code> to the <code>format()</code> method of the <code>string.Formatter</code> class has been deprecated.</p>
</blockquote>
<p>You can use in <code>str.format</code>, but not in <code>string.Formatter</code></p>
</div>
<div class="post-text" itemprop="text">
<p>Or use <code>fstrings</code>:</p>
<pre><code>name = "Bob"
hello = f"Hello {name}"
print (hello)
</code></pre>
<p>Output:</p>
<pre><code>Hello Bob
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try this </p>
<pre><code>name = "john"
hello = "GoodMorning %s" %(name,)
print (hello)
</code></pre>
</div>
