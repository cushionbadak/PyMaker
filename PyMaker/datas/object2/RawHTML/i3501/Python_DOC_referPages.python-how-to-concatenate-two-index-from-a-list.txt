arr_len= len(listvalue)

<div class="post-text" itemprop="text">
<p>You can do this using <a href="https://docs.python.org/3/library/itertools.html#itertools.permutations" rel="nofollow noreferrer"><code>itertools.permutations</code></a>.</p>
<pre><code>&gt;&gt;&gt; places = ['charity','hospital','carrefour']
&gt;&gt;&gt; list(itertools.permutations(places, 2))
[('charity', 'hospital'), ('charity', 'carrefour'), ('hospital', 'charity'), 
('hospital', 'carrefour'), ('carrefour', 'charity'), ('carrefour', 'hospital')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>l = ['a', 'b', 'c']
[[x, y] for x in l for y in l if y != x]
</code></pre>
<p>Output :</p>
<pre><code>[['a', 'b'], ['a', 'c'], ['b', 'a'], ['b', 'c'], ['c', 'a'], ['c', 'b']]
</code></pre>

<p>@Alasdair is right...use itertools.</p>
<p>Code:</p>
<pre><code> from itertools import permutations

 places = ['charity','hospital','carrefour']
 result = list(permutations(places, 2)
</code></pre>
<p>Output:</p>
<pre><code>[('charity', 'hospital'), ('charity', 'carrefour'), ('hospital', 'charity'), ('hospital', 'carrefour'), ('carrefour', 'charity'), ('carrefour', 'hospital')]
</code></pre>
<p>Code:</p>
<pre><code>from itertools import permutations
places = ['charity','hospital','carrefour']
result = [list(place) for place in list(permutations(places, 2))]
</code></pre>
<p>Output:</p>
<pre><code>[['charity','hospital'],['charity','carrefour'],['hospital','charity'],['hospital','carrefour'],['carrefour','charity'],['carrefour','hospital']]
