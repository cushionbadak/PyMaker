<div class="post-text" itemprop="text">
<p>You just need to loop through <code>d1</code> and assign the correct values from <code>d2</code>.</p>
<p><strong>Code:</strong></p>
<pre><code>d1 = {123456:'xyz', 892019:'kjl', 102930491:['plm','kop']}
d2 = {'xyz':987, 'kjl': 902, 'plm': 19240, 'kop':9829}

result = {}
for k, v in d1.items():
    if isinstance(v, list):
        result[k] = {k: d2.get(k) for k in v}
    else:
        result[k] = {v: d2.get(v)}

print(result)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>{123456: {'xyz': 987}, 892019: {'kjl': 902}, 102930491: {'plm': 19240, 'kop': 9829}}
</code></pre>
<p>The above uses <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer"><code>isintance()</code></a> to check if the value is a list, so you can loop over it and assign the correct values from <code>d2</code>. It also uses <a href="https://docs.python.org/3/library/stdtypes.html#dict.get" rel="nofollow noreferrer"><code>dict.get()</code></a> to ensure <code>None</code> is added if a <a href="https://docs.python.org/3/library/exceptions.html#KeyError" rel="nofollow noreferrer">KeyError</a> was encountered when searching for a key  in <code>d2</code>. </p>
<p><strong>One-liner:</strong></p>
<pre><code>{k: {k: d2.get(k) for k in v} if isinstance(v, list) else {v: d2.get(v)} for k, v in d1.items()}
</code></pre>
<p>But this becomes messy and less maintainable.</p>

<div class="post-text" itemprop="text">
<p>First, you hate to put brackets around your keys and values.</p>
<pre><code>d1 = {'123456':'xyz', '892019':'kjl', '102930491':'plm'}
d2= {'xyz':'987', 'kjl':'0902', 'plm':'019240', 'kop':'09829'}
</code></pre>
<p>Then I suggest you do a for loop</p>
<pre><code>for key, value in d1.iter():
    d1[key] = d2[value]
</code></pre>
<p>I have ignored your <code>{[plm,kop]}</code> thing. But it wont be difficult to implement it if you understand this first example.</p>
