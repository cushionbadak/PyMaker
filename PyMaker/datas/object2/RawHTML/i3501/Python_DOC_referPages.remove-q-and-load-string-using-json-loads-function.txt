<div class="post-text" itemprop="text">
<p>try simple to <code>replace</code> the <code>&amp;q;</code> with <code>"</code> double quotes</p>
<pre><code>import json
data = '[{&amp;q;Id&amp;q;:1,&amp;q;Name&amp;q;:&amp;q;Name&amp;q;}]'
data = data.replace('&amp;q;', '"')
print(json.loads(data))
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[{'Id': 1, 'Name': 'Name'}]
</code></pre>
