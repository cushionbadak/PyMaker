<p>You could use the <a href="https://docs.python.org/3.1/library/string.html" rel="nofollow noreferrer">string</a> library to search for the presence of uppercase characters by using the <a href="https://docs.python.org/3/library/string.html#string.ascii_uppercase" rel="nofollow noreferrer">string.ascii_uppercase</a> module</p>
<pre><code>&gt;&gt;&gt; import string
&gt;&gt;&gt; string.ascii_uppercase
'ABCDEFGHIJKLMNOPQRSTUVWXYZ'
</code></pre>
<p>You can then use the <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer">any</a> function to test for the presence of uppercase characters for a given string input.</p>
<pre><code>&gt;&gt;&gt; Pass_Check_1 = 'ThisOneWorks'
&gt;&gt;&gt; any(i in Pass_Check_1 for i in string.ascii_uppercase)
True

&gt;&gt;&gt; Pass_Check_2 = 'thisonedoesntwork'
&gt;&gt;&gt; any(i in Pass_Check_2 for i in string.ascii_uppercase)
False
</code></pre>
