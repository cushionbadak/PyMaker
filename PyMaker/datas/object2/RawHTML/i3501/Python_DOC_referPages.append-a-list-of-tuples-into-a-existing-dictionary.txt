<div class="post-text" itemprop="text">
<p>I have an existing dictionary and I want to add a list of tuples into this dictionary. </p>
<p>Existing dictionary structure:</p>
<pre><code>myD = {'key1': 123 , 'key2': 456}
</code></pre>
<p>List of tuples structure:</p>
<pre><code>myL = [('fkey1',321),('fkey2',432),('fkey3',543)]
</code></pre>
<p>Expected dictionary after adding list of tuples</p>
<pre><code>myD = {'key1': 123 ,'key2': 456 ,'fkey': 321 ,'fkey2': 432 ,'fkey3': 543}
</code></pre>
<p>How can I implement this in python?</p>
</div>
<div class="post-text" itemprop="text">
<p>Just use <a href="https://docs.python.org/3/library/stdtypes.html#dict.update" rel="noreferrer"><code>dict.update</code></a>.</p>
<pre><code>&gt;&gt;&gt; myD = {'key1': 123 , 'key2': 456}
&gt;&gt;&gt; myL = [('fkey1',321),('fkey2',432),('fkey3',543)]
&gt;&gt;&gt; 
&gt;&gt;&gt; myD.update(myL)
&gt;&gt;&gt; 
&gt;&gt;&gt; myD
{'key2': 456, 'key1': 123, 'fkey1': 321, 'fkey2': 432, 'fkey3': 543}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>use simple <code>for loop statment</code></p>
<pre><code>myD = {'key1': 123 , 'key2': 456}

myL = [('fkey1',321),('fkey2',432),('fkey3',543)]

for k, v in myL:
    myD[k] = v

print(myD)
</code></pre>
<p>or use  <code>update</code></p>
<pre><code>myD.update(myL)                                                                                                                                                                                              

print(myD)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>{'key1': 123, 'key2': 456, 'fkey1': 321, 'fkey2': 432, 'fkey3': 543}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>dict.update</code></p>
<p><strong>Ex:</strong></p>
<pre><code>myD = {'key1': 123 , 'key2': 456}
myL = [('fkey1',321),('fkey2',432),('fkey3',543)]
myD.update(myL)
print(myD)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>{'key2': 456, 'key1': 123, 'fkey1': 321, 'fkey2': 432, 'fkey3': 543}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Dictionary unpacking:</p>
<pre><code>&gt;&gt;&gt; &gt;&gt;&gt; myD = {'key1': 123 , 'key2': 456}
&gt;&gt;&gt; myL = [('fkey1',321),('fkey2',432),('fkey3',543)]
&gt;&gt;&gt; {**myD, **dict(myL)}
{'key1': 123, 'key2': 456, 'fkey1': 321, 'fkey2': 432, 'fkey3': 543}
</code></pre>
