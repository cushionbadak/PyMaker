<p>Given a map of simple integers:</p>
<pre><code>this_map = map(int, input().split())
2 4 1 3 5
</code></pre>
<p>How do I print <code>this_map</code> in reverse order? I can print each item with a simple for loop but printing the thing in reverse format requires indexing and this is not possible through standard procedure.</p>
<pre><code>for item in this_map[::-1]:
    print(item)

TypeError: 'map' object is not subscriptable
</code></pre>
<p>A <a href="https://stackoverflow.com/questions/35481061/can-i-use-index-information-inside-the-map-function">similar post</a> on using <code>enumerate()</code> exists but how do I use it here to add indexes?</p>
<p><strong>Note</strong>: I'm using Python 3.6</p>
</div>
<div class="post-text" itemprop="text">
<p>Your example didn't work because <code>map()</code> behaves differently in Python 2 vs Python 3.  In Python 2 it returns a list, but in Python 3 it returns a special <code>map</code> object.</p>
<p>You can convert it to a list by calling <code>list()</code> on it:</p>
<pre><code>this_map = list(this_map)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just use <a href="https://docs.python.org/3/library/functions.html#reversed" rel="nofollow noreferrer"><code>reversed</code></a>:</p>
<pre><code>this_map = map(int, reversed("2 4 1 3 5".split()))
for item in this_map:
    print(item)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Have you tried this:</p>
<pre><code>for i in reversed(this_map):

<div class="post-text" itemprop="text">
<p>Just convert the <code>map</code> into a <code>list</code> first</p>
<pre><code>this_map = list(map(int, input().split()))
for item in this_map[::-1]:
    print(item)
</code></pre>
