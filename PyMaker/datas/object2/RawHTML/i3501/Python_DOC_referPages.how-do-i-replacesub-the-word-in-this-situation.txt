<div class="post-text" itemprop="text">
<p>You can use <code>re.sub</code></p>
<pre><code>re.sub(r'input_path:\s+"[^"]*"', 'input_path: "{}"'.format(new_path), s)
</code></pre>
<p>For example, </p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; new_path = "new_path"
&gt;&gt;&gt; s="""train_input_reader {
...   label_map_path: test
...   tf_record_input_reader {
...     input_path: "PATH_TO_BE_CONFIGURED/mscoco_train.record"
...   }
... }"""
&gt;&gt;&gt; s = re.sub(r'input_path:\s+"[^"]*"', 'input_path: "{}"'.format(new_path), s)
&gt;&gt;&gt;
&gt;&gt;&gt; print(s)
train_input_reader {
  label_map_path: test
  tf_record_input_reader {
    input_path: "new"
  }
}
&gt;&gt;&gt; 
</code></pre>
