<div class="post-text" itemprop="text">
<p>According to <a href="https://docs.python.org/3/library/random.html#random.randint" rel="nofollow noreferrer">docs.python.org</a> , randint returns a random integer N such that a &lt;= N &lt;= b. Alias for randrange(a, b+1). There's nothing about limits.</p>
<p>Also, in <em>Python 3</em> the plain int type is <em>unbounded</em>.</p>
<p>So, <strong><em>I think</em></strong> it should be fine.</p>
</div>
