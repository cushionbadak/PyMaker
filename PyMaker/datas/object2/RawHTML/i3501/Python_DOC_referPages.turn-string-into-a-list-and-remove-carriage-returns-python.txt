<div class="post-text" itemprop="text">
<p>I have a string like this:</p>
<pre><code>['过\r\n啤酒\r\n小心\r\n照顾\r\n锻炼\r\n过去\r\n忘记\r\n哭\r\n包\r\n个子\r\n瘦\r\n选择\r\n奶奶\r\n突然\r\n节目\r\n']
</code></pre>
<p>How do I remove all of the "\r\n", and then turn the string into a list like so:</p>
<pre><code>[过, 啤酒, 小心, 照顾, 过去, etc...]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>str.split</code> removes all whitespace; this includes <code>\r</code> and <code>\n</code>:</p>
<pre><code>A = ['过\r\n啤酒\r\n小心\r\n照顾\r\n锻炼\r\n过去\r\n忘记\r\n哭\r\n包\r\n个子\r\n瘦\r\n选择\r\n奶奶\r\n突然\r\n节目\r\n']

res = A[0].split()

print(res)

['过', '啤酒', '小心', '照顾', '锻炼', '过去', '忘记', '哭', '包', '个子', '瘦', '选择', '奶奶', '突然', '节目']
</code></pre>
<p>As described in the <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer">str.split docs</a>:</p>
<blockquote>
<p>If <em>sep</em> is not specified or is <code>None</code>, a different splitting
  algorithm is applied: runs of consecutive whitespace are regarded as a
  single separator, and the result will contain no empty strings at the
  start or end if the string has leading or trailing whitespace.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>To limit the split to <code>\r\n</code> you can use <code>.splitlines()</code>:</p>
<pre><code>&gt;&gt;&gt; li=['过\r\n啤酒\r\n小心\r\n照顾\r\n锻炼\r\n过去\r\n忘记\r\n哭\r\n包\r\n个子\r\n瘦\r\n选择\r\n奶奶\r\n突然\r\n节目\r\n']
&gt;&gt;&gt; li[0].splitlines()
['过', '啤酒', '小心', '照顾', '锻炼', '过去', '忘记', '哭', '包', '个子', '瘦', '选择', '奶奶', '突然', '节目']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>s = "['过\r\n啤酒\r\n小心\r\n照顾\r\n锻炼\r\n过去\r\n忘记\r\n哭\r\n包\r\n个子\r\n瘦\r\n选择\r\n奶奶\r\n突然\r\n节目\r\n']"
s = s.replace('\r\n', ',').replace("'", '')
print(s)
</code></pre>
<p>Output:</p>
<pre><code>[过,啤酒,小心,照顾,锻炼,过去,忘记,哭,包,个子,瘦,选择,奶奶,突然,节目,]
</code></pre>
<p>This first <code>replace</code> replaces the <code>\r\n</code> and the second one replaces the single quote from the string as you expected as the output.</p>
</div>
