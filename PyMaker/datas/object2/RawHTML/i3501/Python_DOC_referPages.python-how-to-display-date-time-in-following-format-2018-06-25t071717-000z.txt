<p>This format is called ISO format, after standard <a href="https://en.wikipedia.org/wiki/ISO_8601" rel="nofollow noreferrer">ISO 8601</a>. The datetime object has a <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.isoformat" rel="nofollow noreferrer">isoformat</a> method to output this form.</p>
<blockquote>
<p>strftime("%Y-%m-%dTH:M:SZ")</p>
</blockquote>
<p>You seem to have forgotten some <code>%</code> before the H, M, and S. Try <code>strftime("%Y-%m-%dT%H:%M:%SZ")</code>. </p>
<blockquote>
<p>but it seems that doesn't work.</p>
</blockquote>
<p>Generally it works better if you specify exactly what doesn't work, or what you expect and how the reality differs from your expectation.</p>
