<div class="post-text" itemprop="text">
<p><code>a = [0, 18, 27, 43, 52, 65]</code> </p>
<p>is the list of indices from which I want to access bigger list slicing with</p>
<p><code>[0:18],[18:27],[27:43],[43:52], [52:65], [65:]</code></p>
<p>How to do it?</p>
<p>I have tried but i not get correct format</p>
<p><code>slicing = [index for index in (a[:-1])]</code> </p>
<p>but that gives me <code>[0, 18, 27, 43, 52]</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a>:</p>
<pre><code>a = [0, 18, 27, 43, 52, 65]    
res = [bigger_list[x:y] for x, y in zip(a[:-1], a[1:])]
</code></pre>
<p>and finally, extend it:</p>
<pre><code>res.extend(bigger_list[a[-1]:])
</code></pre>
<hr/>
<p>Or to do it in a single line:</p>
<pre><code>res = [bigger_list[x:y] for x, y in zip(a[:-1], a[1:])] + bigger_list[a[-1]:]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>After some tweaking and searching I found answer:</p>
<pre><code>res = [bigger_list[i: a[ind+1]] for ind,i in enumerate(a[:-1])]
</code></pre>
<p>that also gives me <code>slicing of bigger_list</code>.</p>
</div>
