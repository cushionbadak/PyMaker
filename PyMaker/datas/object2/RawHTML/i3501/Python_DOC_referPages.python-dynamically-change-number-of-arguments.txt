<p>I want to calculate a cartesian product from a changing number of sets.</p>
<p>The cartesian product can be computed using <code>itertools.product(*args, repeat=1)</code></p>
<pre><code>itertools.product(set1, set2, repeat=1)
</code></pre>
<p>but the number of sets is actually unknown, it depends on the data. Sometimes it might be 2 groups, sometimes 3 or more. </p>
<p>Is there a way to submit *args such that it can dynamically change the number of arguments?</p>
<pre><code>myargs = [set1, set2, set3]
itertools.product(myargs,repeat=1)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Yes, there is such a way. Use the <code>*</code> argument expansion operator:</p>
<pre><code>myargs = [set1, set2, set3]
itertools.product(*myargs,repeat=1)
</code></pre>
