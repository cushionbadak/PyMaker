<div class="post-text" itemprop="text">
<p>Just use <code>getstatusoutput</code> insted of <code>getoutput</code> (<a href="https://docs.python.org/3/library/subprocess.html#subprocess.getstatusoutput" rel="nofollow noreferrer">doc</a>):</p>
<pre><code>&gt;&gt;&gt; subprocess.getstatusoutput('ls /bin/ls')
(0, '/bin/ls')
&gt;&gt;&gt; subprocess.getstatusoutput('cat /bin/junk')
(1, 'cat: /bin/junk: No such file or directory')
&gt;&gt;&gt; subprocess.getstatusoutput('/bin/junk')
(127, 'sh: /bin/junk: not found')
&gt;&gt;&gt; subprocess.getstatusoutput('/bin/kill $$')
(-15, '')
</code></pre>
