<div class="post-text" itemprop="text">
<p>Use a dictionary comprehension:</p>
<pre><code>d = { 'a': [1, None],
      'b': [2, 2],
      'c': [1, 0],
      'd': [1, 1] }

print({k:v for k, v in d.items() if None not in v})
# {'b': [2, 2], 'c': [1, 0], 'd': [1, 1]}
</code></pre>
<p>If you need a loop like yours:</p>
<pre><code>for key, value in d.items():
    if None not in value:
        print(key, value)
        # do your calculations here
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can try</p>
<pre><code>mydict = { 'a': [1, None],
      'b': [2, 2],
      'c': [1, 0],
      'd': [1, 1]
    }

from copy import copy

newdict = copy(mydict)

for key, values in mydict.items():
    if None in values:
      newdict.pop(key)

print newdict
</code></pre>
