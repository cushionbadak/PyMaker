<p>I want to patch multiples method of same object. I am trying
<code>@patch.multiple('pdb.Pdb', do_continue=DEFAULT, do_step=DEFAULT, do_exit=DEFAULT)</code> it throws error during running Test
<code>NameError: name 'DEFAULT' is not defined</code></p>

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/unittest.mock.html#unittest.mock.DEFAULT" rel="nofollow noreferrer"><code>unittest.patch.DEFAULT</code></a> needs to be imported first. This will run without error:</p>
<pre><code>from unittest.mock import patch
from unittest.mock import DEFAULT

patch.multiple('pdb.Pdb', do_continue=DEFAULT, do_step=DEFAULT, do_exit=DEFAULT)
</code></pre>
