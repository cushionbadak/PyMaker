<p>I know there is a way to specify the type of argument like this:</p>
<pre><code>def f(x: int)
    return x
</code></pre>
<p>but what if I want to pass a function as an argument?
For example:</p>
<pre><code>class Sth:
    def __init__(self, x: int, f: XXX):
        self.__x = x
        self.__f = f

def a():
    return True


obj = Sth(2, a)
</code></pre>
<p>What do I type in the place of XXX for it to work?</p>
<p>When I do <code>type(a)</code> it shows <code>&lt;class 'function'&gt;</code>, but I can not put 'function' in place of XXX</p>
</div>
<div class="post-text" itemprop="text">
<p>You type 'Callable'. See <a href="https://docs.python.org/3/library/typing.html#typing.Callable" rel="nofollow noreferrer">reference.</a></p>
<pre><code>from typing import Callable

def __init__(self, x: int, f: Callable):
    pass
</code></pre>
