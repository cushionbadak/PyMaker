<div class="post-text" itemprop="text">
<p>Because <a href="https://docs.python.org/3/library/random.html#random.choices" rel="nofollow noreferrer"><code>random.choices(population, weights=None, *, cum_weights=None, k=1)</code></a> returns a list of k elements. In your case, since you didn't pass any value for <code>k</code>, the list has only an item.</p>
<p>If you want to get a value, and not a list, you could use <a href="https://docs.python.org/3/library/random.html#random.choice" rel="nofollow noreferrer"><code>random.choice(seq)</code></a>, but in that case you could not pass a list of weights as with <code>random.choices()</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/random.html#random.choices" rel="nofollow noreferrer">random.choices</a> draws a list of values that you can specify with <code>k=..</code> when calling it:</p>
<pre><code>fourty = random.choices(range(10),k=40)
</code></pre>
<p>You can print the single value you get by indexing into it using <code>choices</code> method of specifying probabilities</p>
<pre><code>print(random.choices([a,b,c], [50,3,1])) [0]  # get the one single value from the result list
</code></pre>
<p>or you can use <a href="https://docs.python.org/3/library/random.html#random.choice" rel="nofollow noreferrer">random.choice()</a> providing an iterable that reflects your propabilites:</p>
<pre><code>import random
a=1
b=2
c=3

# create an iterable that reflects your propabilies and draw 1 element from it
print(random.choice(  [a]*50+[b]*3+[c] ) )  
</code></pre>
