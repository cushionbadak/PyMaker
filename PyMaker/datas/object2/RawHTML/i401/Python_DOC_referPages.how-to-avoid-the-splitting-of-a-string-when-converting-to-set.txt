<p>I am trying to convert a string to a set containing that string. How can I do this without splitting?</p>
<p>when I write:</p>
<pre><code>set("abc")
</code></pre>
<p>the result is:</p>
<blockquote>
<p>{'a','b','c'}</p>
</blockquote>
<p>but I want it to be:</p>
<blockquote>
<p>{"abc"}</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#set" rel="nofollow noreferrer">Doku <code>set(iterable)</code></a> will create a set of each element of an iterable - strings <strong>are</strong> iterable - hence you create a set of the characters of the string.</p>
<p>If you want whole strings, use </p>
<pre><code>k = {"mystring",}
</code></pre>
<p>or</p>
<pre><code>k = set( ["mystring"] ) # wrap your string into another iterable
</code></pre>
</div>
