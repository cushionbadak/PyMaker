<div class="post-text" itemprop="text">
<p>I want to print each values of a class, but I don't know how to do it. I understand why my method of doing it doesn't work though.</p>
<pre class="lang-py prettyprint-override"><code>class test:
    def __init__(self, a = 1, b = 2, c = 3):
        self.a = a
        self.b = b
        self.c = c

classobject = test()

for attr in dir(classobject) :
    if not attr.startswith('__'):
        print(str(attr) + ' value= ' + classobject.attr)
</code></pre>
<p>I get the error <code>AttributeError: 'test' object has no attribute 'attr'</code></p>
</div>
<div class="post-text" itemprop="text">
<p><code>classobject.attr</code> is specifically looking for an attribute named <code>attr</code> in <code>classobject</code>.</p>
<p><code>attr</code> does not get magically replaced at runtime by the appropriate value.</p>
<p>What you need is the <a href="https://docs.python.org/2/library/functions.html#getattr" rel="nofollow noreferrer">getattr</a> function:</p>
<pre><code>print(str(attr) + ' value= ' + getattr(classobject, attr))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could do</p>
<pre><code>print(test.__dict__)
</code></pre>
<p>Which will basically scrape out <em>all</em> the attributes of an object. This may or may not be what you want but it is at least a superset of what you want. I suppose you could filter by keys if you have a list of attributes you explicitly care about.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>classobject = test()
attrs = vars(classobject)
for key,value in attrs.items():
    print (str(key) + ' value= ' +str(value))
</code></pre>
<p>result:</p>
<p>a value= 1</p>
<p>b value= 2</p>
<p>c value= 3</p>
</div>
