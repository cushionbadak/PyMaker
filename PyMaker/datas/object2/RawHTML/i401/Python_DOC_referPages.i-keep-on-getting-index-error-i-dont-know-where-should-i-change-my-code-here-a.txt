<div class="post-text" itemprop="text">
<p>You're iterating an index over the list <code>EmpNo</code>, but you're trying to use the same index on two other lists, <code>SSSLoan</code> and <code>MonthlyDeduction</code>, so if any of the two lists has fewer items, it would cause an <code>IndexError</code> when the iteration gets to a point where the index exists for <code>EmpNo</code> but not for <code>SSSLoan</code> or <code>MonthlyDeduction</code>.</p>
<p>You should make sure that the 3 lists are of equal length. Alternatively, you can use <code>zip</code> the 3 lists so that the iteration would end when the shortest list is exhausted:</p>
<pre><code>def disp_sss():
    print("\n\nList of SSS loans\n")
    print("EmployeeNo        SSS        Deduction")
    for row in zip(EmpNo, SSSLoan, MonthlyDeduction):
        print(" {0:15} {1:10s} {2:10.2f}".format(row))
</code></pre>
