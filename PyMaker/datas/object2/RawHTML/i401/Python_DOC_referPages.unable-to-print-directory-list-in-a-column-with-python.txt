<div class="post-text" itemprop="text">
<p>The <code>os.scandir</code> method returns <a href="https://docs.python.org/3/library/os.html#os.DirEntry" rel="nofollow noreferrer"><code>os.DirEntry</code></a> objects rather than file names. You should either use the <code>name</code> attribute of the object, or use the <code>os.listdir</code> method instead since you don't need the other attributes of the files. Also, there is no need for a nested <code>for</code> loop since you can simply iterate through the file names and print each file name on a separate line:</p>
<pre><code>for file in os.scandir(path):
    print(file.name)
</code></pre>
<p>or:</p>
<pre><code>for filename in os.listdir(path):
    print(filename)
</code></pre>
