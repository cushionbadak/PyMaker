<p>I have a list like</p>
<pre><code>key_list=[(2, 'just', 'sd'), (2, 'great', 'sd'), (1, 'just great', 'sd')]
</code></pre>
<p>I want to choose maximum value at 0 index of list for every element like here maximum value is 2 but when there are two maximum values for same value in different sets, it randomly return the the one element like <code>(2, 'just', 'sd')</code> and <code>(2, 'great', 'sd')</code> have maximum value of 2 and return any one of them. </p>
<p>I have written </p>
<pre><code>max([i[0] for i in list_keys])
</code></pre>
<p>to get maximum value but don't know how to choose random value based on this '2'</p>

<div class="post-text" itemprop="text">
<p>There are two ways you could do this:</p>
<p>(1) Randomly choose among the entries that match the maximum value:</p>
<pre><code>import random
key_list=[(2, 'just', 'sd'), (2, 'great', 'sd'), (1, 'just great', 'sd')]
m = max(i[0] for i in key_list)
print(random.choice([i for i in key_list if i[0] == m]))
</code></pre>
<p>(2) Use a special key function that applies a random tie-breaker:</p>
<pre><code>print(max(key_list, key=lambda i: i[0] + random.random()))
# or
print(max(key_list, key=lambda i: (i[0], random.random())))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/random.html#random.choice" rel="nofollow noreferrer"><code>random.choice</code></a>:</p>
<pre><code>import random

key_list = [
    (2, 'just', 'sd'),
    (2, 'great', 'sd'),
    (1, 'just great', 'sd')
]

max_value = max([key[0] for key in key_list])
max_keys = [key for key in key_list if key[0] == max_value]

print(random.choice(max_keys))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can fist find the maximum value, then randomly select among the tuples containing this maximum:</p>
<pre><code>import random

key_list=[(2, 'just', 'sd'), (2, 'great', 'sd'), (1, 'just great', 'sd')]

maxi = max(t[0] for t in key_list)
out = random.choice([t for t in key_list if t[0]==maxi])

print(out)
# (2, 'just', 'sd')
</code></pre>
