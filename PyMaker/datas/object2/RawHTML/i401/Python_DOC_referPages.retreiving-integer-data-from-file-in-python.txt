<p>the data is in the text file is like</p>
<pre><code>2104,3,399900
1600,3,329900
2400,3,369000
1416,2,232000
3000,4,539900
1985,4,299900
</code></pre>
<p>and I want a resultant matrix as
<code>[2104 3], [1600,3],[2400,3], ...</code></p>
<p>and second matrix as 
<code>[399900], [329900], ....</code></p>
</div>

<pre><code>filename = "path-to-filename"
list_a = []
list_b = []

with open(filename, 'r') as f:
        for line in f:
                line = line.rstrip('\n')
                line_list = line.split(',')

                list_a.append(line_list[:2])
                list_b.append(line_list[2])

print(list_a)
print(list_b)
