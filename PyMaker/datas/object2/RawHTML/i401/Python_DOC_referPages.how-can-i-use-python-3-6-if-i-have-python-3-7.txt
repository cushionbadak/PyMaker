<div class="post-text" itemprop="text">
<p>Just install it in different folder (e.g. if current one is in C:\Users\noob\AppData\Local\Programs\Python\Python37, install 3.6. to C:\Users\noob\AppData\Local\Programs\Python\Python36).</p>
<p>Now, when you'll want to run a script, right click the file and under "edit with IDLE" will be multiple versions to choose. Works on my machine :)</p>
