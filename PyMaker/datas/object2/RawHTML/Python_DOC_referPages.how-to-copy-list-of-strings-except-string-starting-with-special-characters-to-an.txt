<p>I'm new to python can you please help me with below question</p>
<p>I have a list like this:</p>
<pre><code>lst = ["qwerty", "asdf", "   Bird", "#Bird", "Bird", "%qwerty"]
</code></pre>
<p>I need to copy this into new_lst, where I need to copy only strings starting with alphabets</p>
<p>Ex: </p>
<pre><code>new_lst = ["qwerty", "asdf", "Bird"]
</code></pre>
<p>I dont want to copy any string starting with special character like space,#,&amp; etc</p>
<p>How can I do this</p>

<div class="post-text" itemprop="text">
<p>Try using list comprehension :</p>
<pre><code>list1 = ["qwerty", "asdf", " Bird", "#Bird", "Bird", "%qwerty"]
list2 = [a for a in list1 if a[0].isalpha()]
</code></pre>
<p>Try not to assign any variable name as <code>list</code>.</p>
<p><strong>OUTPUT</strong> : </p>
<pre><code>['qwerty', 'asdf', 'Bird']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Another approach is using <a href="https://www.w3schools.com/python/ref_func_filter.asp" rel="nofollow noreferrer">filter</a> and <a href="https://www.w3schools.com/python/python_lambda.asp" rel="nofollow noreferrer">lambda</a></p>
<pre><code>lst = ["qwerty", "asdf", "   Bird", "#Bird", "Bird", "%qwerty"]
new_lst = filter(lambda x: x[0].isalpha() , lst)
print(new_lst)
</code></pre>
<p>Output</p>
<pre><code>['qwerty', 'asdf', 'Bird']
</code></pre>
</div>
