<p>As is clearly stated in <a href="https://docs.ansible.com/ansible/2.7/installation_guide/intro_installation.html#control-machine-requirements" rel="nofollow noreferrer">the fine manual</a>, Windows is not supported for use as a control machine. Even if you got it to install, the next problem you'd have is trying to convince it to use a sane ssh client, of which PuTTY is not one.</p>
<p>I am sure you can use the <a href="https://docs.microsoft.com/en-us/windows/wsl/about" rel="nofollow noreferrer">Windows Subsystem for Linux</a> to gain access to a reasonable operating system inside the unreasonable one</p>
