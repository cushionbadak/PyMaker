<p>It's my 2nd day with Selenium 2 library and the pain with Unicode never seem to subside.</p>
<p>I'm just doing the most basic operation, want to print the page source:</p>
<pre><code>from selenium import webdriver


driver = webdriver.Firefox()
driver.get("http://google.com")

print driver.page_source
</code></pre>
<p>Sure enough, I get an error:</p>
<pre><code>UnicodeEncodeError: 'ascii' codec can't encode character u'\u0119' in position 62045:  
ordinal not in range(128)
</code></pre>
<p>How can I please encode this to <code>utf-8</code> ?</p>
</div>
<div class="post-text" itemprop="text">
<p>You have options, based on this similar <a href="https://stackoverflow.com/questions/3224268/python-unicode-encode-error">question</a>.</p>
<p>You can either convert the source to all ascii losing the Unicode characters in the process.</p>
<p><code>(driver.page_source).encode('ascii', 'ignore')</code></p>
<p>Or, and I think you'll prefer this, you can encode it to utf-8 like this:
<code>(driver.page_source).encode('utf-8')</code>.</p>
</div>
