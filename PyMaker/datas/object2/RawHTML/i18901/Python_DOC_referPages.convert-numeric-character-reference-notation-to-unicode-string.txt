<p>Is there a standard, preferably Pythonic, way to convert the <code>&amp;#xxxx;</code> notation to a proper unicode string?</p>
<p>For example,</p>
<pre><code>&amp;#1502;&amp;#1508;&amp;#1490;&amp;#1513;&amp;#1497;
</code></pre>
<p>Should be converted to:</p>
<pre><code>מפגשי
</code></pre>
<p>It can be done - quite easily - using string manipulations, but I wonder if there's a standard library for this.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/2/library/htmlparser.html" rel="noreferrer"><code>HTMLParser.HTMLParser()</code></a>:</p>
<pre><code>&gt;&gt;&gt; from HTMLParser import HTMLParser
&gt;&gt;&gt; h = HTMLParser()
&gt;&gt;&gt; s = "&amp;#1502;&amp;#1508;&amp;#1490;&amp;#1513;&amp;#1497;"
&gt;&gt;&gt; print h.unescape(s)
מפגשי
</code></pre>
<p>It's part of the <a href="http://docs.python.org/2/library/" rel="noreferrer">standard library</a>, too.</p>
<hr/>
<p>However, if you're using Python 3, you have to import from <code>html.parser</code>:</p>
<pre><code>&gt;&gt;&gt; from html.parser import HTMLParser
&gt;&gt;&gt; h = HTMLParser()
&gt;&gt;&gt; s = '&amp;#1502;&amp;#1508;&amp;#1490;&amp;#1513;&amp;#1497;'
&gt;&gt;&gt; print(h.unescape(s))
מפגשי
</code></pre>
</div>
