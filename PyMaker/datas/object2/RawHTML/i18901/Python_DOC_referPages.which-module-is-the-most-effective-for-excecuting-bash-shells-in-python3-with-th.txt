<p>Given the script...</p>
<pre><code>$cat sample.py

total 8

<div class="post-text" itemprop="text">
<p>The <code>commands</code> module <a href="http://docs.python.org/2/library/commands.html" rel="nofollow">has been deprecated</a>:</p>
<blockquote>
<p>Deprecated since version 2.6: The <code>commands</code> module has been removed in Python 3. Use the <code>subprocess</code> module instead.</p>
</blockquote>
<p>As stated, use the <a href="http://docs.python.org/3/library/subprocess.html" rel="nofollow"><code>subprocess</code> module</a> instead, both in Python 2 <em>and</em> 3:</p>
<pre><code>import subprocess

print(subprocess.check_output(['ls', '-l']))
</code></pre>
