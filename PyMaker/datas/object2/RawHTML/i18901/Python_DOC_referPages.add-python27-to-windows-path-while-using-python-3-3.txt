<div class="post-text" itemprop="text">
<p>I use python 3.3.2 on windows everyday, so i added C:\Python33 to my PATH in order to be able to call "python foo.py" from console and get python 3.3.2 to execute it.</p>
<p>But sometimes, i also need to use python 2.7. How could i add a "python27" entry to my path, in order to call "python27 bar.py" and get python 2.7 to execute it ?</p>
</div>
<div class="post-text" itemprop="text">
<p>Run it using <a href="http://docs.python.org/3/using/windows.html#launcher" rel="nofollow">pylauncher</a>:</p>
<pre><code>C:\&gt; py -2.7 bar.py
</code></pre>
<p>Or add at the top of bar.py:</p>
<pre><code>#! python2.7
</code></pre>
<p>then:</p>
<pre><code>C:\&gt; py bar.py
</code></pre>
<p>will use Python 2.7 version. </p>
<p>If pylauncher is configured to handle Python scripts; you could just call:</p>
<pre><code>C:\&gt; bar.py
</code></pre>
<p>Or</p>
<pre><code>C:\&gt; bar
</code></pre>
<p>if .py is in PATHEXT</p>
</div>
<div class="post-text" itemprop="text">
<ol>
<li>Move <code>C:\Python27\python.exe</code> to <code>C:\Python27\python27.exe</code></li>
<li>Add <code>C:\Python27</code> to your PATH.</li>
</ol>
</div>
