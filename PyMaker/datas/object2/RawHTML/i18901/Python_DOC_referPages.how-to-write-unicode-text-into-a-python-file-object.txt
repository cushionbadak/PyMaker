<p>Simply put, the following code:</p>
<pre><code>f.write(u'Río Negro')
</code></pre>
<p>raises the following error: </p>
<pre><code>UnicodeEncodeError: 'ascii' codec can't encode character u'\xed' in position 1: ordinal not in range(128)
</code></pre>
<p>What can I do?<br/>
I'm using Python 2.7.3.</p>
</div>
<div class="post-text" itemprop="text">
<p>You need to encode your string. Try this:</p>
<pre><code>f.write(u'Río Negro'.encode('utf-8'))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/2/library/codecs.html#codecs.open" rel="nofollow"><code>open</code></a> from the <a href="http://docs.python.org/2/library/codecs.html" rel="nofollow"><code>codecs</code></a> module will eliminate the need for you to manually encode:</p>
<pre><code>import codecs

with codecs.open('file.txt', 'w', encoding='utf-8') as f:
    f.write(u'Río Negro')
</code></pre>
<p>In Python 3, this functionality is built in to <a href="http://docs.python.org/3/library/functions.html#open" rel="nofollow">the standard <code>open</code> function</a>:</p>
<pre><code>with open('file.txt', 'w', encoding='utf-8') as f:
    f.write(u'Río Negro')
</code></pre>
</div>
