<div class="post-text" itemprop="text">
<p>there is standard module configparser. documentation can be found <a href="http://docs.python.org/3/library/configparser.html" rel="nofollow">configparser documentation</a></p>
<p>simple example : </p>
<pre><code>

    #/usr/bin/env python
    import configparser
    import sys

    def config_reader(filename):
        try:
            config = configparser.ConfigParser()
            config.read(filename )
            section_list = config.sections()
            for section_name in section_list:
                for key in config[section_name]:
                    print("Key : " +  config[section_name ][key])
        except configparser.Error as e:
            print(e)
            return 

    def main():
        print("config file " + sys.argv[1])
        config_reader(sys.argv[1])


    if __name__ == "__main__":
        main()


</code></pre>
