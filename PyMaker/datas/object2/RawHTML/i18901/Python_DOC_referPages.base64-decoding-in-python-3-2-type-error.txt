<div class="post-text" itemprop="text">
<p>Psychic debugging: You probably passed a <code>str</code> object to the function, but <code>base64.b64decode</code> requires a <code>bytes</code> object.  So, you'll need to encode the string.</p>
<pre><code>key = base64.b64decode(key.encode('ASCII'))
</code></pre>
<p>Note that <a href="http://docs.python.org/3/whatsnew/3.3.html#base64" rel="nofollow">Python 3.3</a> is more lenient and lets you pass either <code>str</code> or <code>bytes</code>.</p>
