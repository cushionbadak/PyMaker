<p><a href="http://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow">http://docs.python.org/3/library/xml.etree.elementtree.html</a></p>
<pre><code>&gt;&gt;&gt; from xml.etree import ElementTree as ET
&gt;&gt;&gt; elem = ET.fromstring('''&lt;Atom3d ID="18" Mapping="43" Parent="2" Name="C7"
... XYZ="0.0148299997672439,0.283699989318848,1.0291999578476" Connections="33,39"
... TemperatureType="Isotropic" IsotropicTemperature="0.0677"
... AnisotropicTemperature="0,0,0,0,0,0,0,0,0" Occupancy="0.708" Components="C"/&gt;
... ''')
</code></pre>
<p>get attribute using get('attribute-name'):</p>
<pre><code>&gt;&gt;&gt; elem.get('XYZ')
'0.0148299997672439,0.283699989318848,1.0291999578476'
</code></pre>
<p>split string by ',':</p>
<pre><code>&gt;&gt;&gt; elem.get('XYZ').split(',')
['0.0148299997672439', '0.283699989318848', '1.0291999578476']
</code></pre>
