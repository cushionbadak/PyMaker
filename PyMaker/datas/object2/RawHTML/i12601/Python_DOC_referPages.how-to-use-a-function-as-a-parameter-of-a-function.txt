<p>Say I have a function <code>sum(x,y)</code> which returns <code>x+y</code>. Another function <code>adder(s,...)</code> which takes <code>s</code> as a parameter.</p>
<p>I want to find the number of parameters of <code>s</code> in adder:</p>
<pre><code>import inspect
adder(s):
     x = inspect.getargspecs(s)  # no of parameter is `s`
     print x
</code></pre>
<p>I get the error:</p>
<pre><code>'module' object has no attribute 'getargspecs'
</code></pre>
<p>What's wrong? How do I get the number of parameters of <code>s</code> in <code>adder()</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>The method is called <a href="https://docs.python.org/2/library/inspect.html#inspect.getargspec" rel="nofollow"><code>.getargspec()</code></a>, not <code>.getargspecs()</code>:</p>
<pre><code>import inspect

def adder(s):
    x = inspect.getargspec(s) #no of parameter is s
    print(x)

adder(sum)
</code></pre>
<p>Note that the method was <em>deprecated in Python 3.0</em> in favor of <a href="https://docs.python.org/3/library/inspect.html#inspect.signature" rel="nofollow"><code>.signature()</code></a> method.</p>
</div>
