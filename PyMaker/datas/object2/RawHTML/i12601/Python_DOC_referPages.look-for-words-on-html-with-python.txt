<div class="post-text" itemprop="text">
<p>If you are looking for a text string in the html page, you can use Python <a href="http://docs.python-requests.org/en/latest/" rel="nofollow">requests</a> module. This is probably the easiest way to do it.     </p>
<pre><code>import requests

r = requests.get('http://192.168.0.200:9090/')
if "ArduinoOn" in r.text:
    print ('True')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Following the <a href="https://docs.python.org/3/library/urllib.request.html" rel="nofollow">docs</a> the <code>urllib.request.urlopen</code> returns a <code>http.client.HTTPResponse</code> object. So if you want to get the content of the page you should read it:</p>
<pre><code>x = urllib.request.urlopen('http://192.168.0.200:9090/')
if 'Arduino On' in x.read():
    print 'True'
</code></pre>
<p>Hope this helps.</p>
