<p>I have a string of binary. I need to convert this to a wav file. Is there any existing library to achieve that goal? If so how would I write the bytes to an audio file in preferably Java or Python, but any language works.    </p>

<div class="post-text" itemprop="text">
<p>See the documentation for <a href="https://docs.python.org/3/library/wave.html#wave-write-objects" rel="nofollow">Wave_write Objects</a> included with Python's standard library. You will need to <a href="https://docs.python.org/3/library/wave.html#wave.open" rel="nofollow">open</a> a WAV file first in <code>'wb'</code> (write only) mode, and then use the appropriate methods:</p>
<ul>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.close" rel="nofollow">Wave_write.close()</a></li>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.setnchannels" rel="nofollow">Wave_write.setnchannels(n)</a></li>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.setsampwidth" rel="nofollow">Wave_write.setsampwidth(n)</a></li>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.setframerate" rel="nofollow">Wave_write.setframerate(n)</a></li>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.setnframes" rel="nofollow">Wave_write.setnframes(n)</a></li>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.setcomptype" rel="nofollow">Wave_write.setcomptype(type, name)</a></li>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.setparams" rel="nofollow">Wave_write.setparams(tuple)</a></li>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.tell" rel="nofollow">Wave_write.tell()</a></li>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.writeframesraw" rel="nofollow">Wave_write.writeframesraw(data)</a></li>
<li><a href="https://docs.python.org/3/library/wave.html#wave.Wave_write.writeframes" rel="nofollow">Wave_write.writeframes(data)</a></li>
</ul>
