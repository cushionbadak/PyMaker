<p>I have been struggling to troubleshoot a program, and I believe it comes down to this statement:</p>
<pre><code>while line != '\n' or '':
</code></pre>
<p>I am reading a file using readline() and I'm struggling to fix the code in my function. 
For this while loop, I want it continue with these conditions:
- line is not equal to '\n'
OR
- line is not equal to ''</p>
<p>Why doesn't this statement represent the above situation?</p>
</div>
<div class="post-text" itemprop="text">
<p>You have to read</p>
<pre><code>line != '\n' or ''
</code></pre>
<p>as </p>
<pre><code>(line != '\n') or ''
</code></pre>
<p>The empty string evaluates to <code>False</code> in a boolean context, so your expression is equivalent to</p>
<pre><code>line != '\n' or False
</code></pre>
<p>which is equivalent to</p>
<pre><code>line != '\n'
</code></pre>
<p>You either need to write the expression like @houcros suggested, as</p>
<pre><code>line != '\n' or line != ''
</code></pre>
<p>or as I would suggest, as</p>
<pre><code>line not in ('\n', '')
</code></pre>
<p>which I think is easier to read. In your particular case, for this particular program, a simple</p>
<pre><code>while line.strip()
</code></pre>
<p>would suffice, too - if <code>line</code> consists only of whitespace (check if that is what you <em>actually</em> want), <code>line.strip()</code> will be <code>''</code>, which evaluates to <code>False</code> as already mentioned. </p>
</div>
<div class="post-text" itemprop="text">
<p>While you say you want "<code>line</code> is not equal to <code>'\n'</code> OR <code>line</code> is not equal to <code>''</code>", I think you really want "<code>line</code> is not equal to <code>'\n'</code> AND line is not equal to <code>''</code>".  Otherwise, your condition is trivially true (because <code>line</code> cannot both be equal to <code>'\n'</code> and be empty).</p>
<p>Therefore, I believe the condition you are looking for is:</p>
<pre><code>while line != '\n' and line != '':
</code></pre>
<p>or equivalently,</p>
<pre><code>while not (line == '\n' or line == ''):
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I believe it should be</p>
<pre><code>while line != '\n' or line != '':
</code></pre>
