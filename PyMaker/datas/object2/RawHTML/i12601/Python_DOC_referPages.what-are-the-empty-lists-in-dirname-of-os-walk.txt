<div class="post-text" itemprop="text">
<p>Quoting from <a href="https://docs.python.org/3/library/os.html?highlight=os.walk#os.walk" rel="nofollow">Python Documentation</a>:</p>
<blockquote>
<p><code>os.walk(top, topdown=True, onerror=None, followlinks=False)</code> <br/>
  Generate the file names in a directory tree by walking the tree either top-down
  or bottom-up. For each directory in the tree rooted at directory top
  (including top itself), it yields a <strong>3-tuple (dirpath, dirnames,
  filenames)</strong>.</p>
<p><strong>dirpath is a string, the path to the directory. dirnames is a list of
  the names of the subdirectories in dirpath (excluding '.' and '..').
  filenames is a list of the names of the non-directory files in
  dirpath</strong>. Note that the names in the lists contain no path components.
  To get a full path (which begins with top) to a file or directory in
  dirpath, do os.path.join(dirpath, name).</p>
</blockquote>
<p>Since <code>testdir</code> has only two empty directories <code>t1</code> and <code>t2</code>, so you will get :</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; t = '/home/pontiac/testdir'
&gt;&gt;&gt; for root, dirs, files in os.walk(t):
    print root, dirs, files


/home/testdir ['t1', 't2'] []
# ^rootdir     ^dirs list  ^files in rootdir 
/home/testdir/t1 [] []
/home/testdir/t2 [] []
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The call <code>os.walk</code> returns a tuple <code>(dirpath, dirnames, filenames)</code>. You print the <code>[1]</code> element of this tuple, <code>dirnames</code>, which is the list of subdirectories in a given directory. Your directories <code>t1</code> and <code>t2</code> apparently do not contain any subdirectories, hence the empty lists.</p>
</div>
<div class="post-text" itemprop="text">
<p>According to <code>os</code> <a href="https://docs.python.org/3/library/os.html" rel="nofollow">documentation</a> what <code>os.walk</code> returns is a 3-tuple <code>(dirpath, dirnames, filenames)</code>.</p>
<p>Let's say you have a directory <code>testdir</code>:</p>
<pre><code>testdir
      |- dir1
      |    |- file3.txt
      |- dir2
      |- file1.txt
      |- file2.txt
</code></pre>
<p>Then looping over generator returned by <code>os.walk('path/to/testdir')</code> will produce the following tuples:</p>
<pre><code>('path/to/testdir', ['dir2', 'dir1'], ['file3.txt', 'file1.txt', 'file2.txt'])
('path/to/testdir/dir2', [], [])
('path/to/testdir/dir1', [], ['file3.txt'])
</code></pre>
<p>Since you're extracting first elements from the 3-tuples and getting empty lists, that means these are directories that contain no subdirectories in them.</p>
