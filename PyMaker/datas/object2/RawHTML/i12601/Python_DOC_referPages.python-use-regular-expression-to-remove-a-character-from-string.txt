<p>You could use something like so: <code>(\\"title\\" : \\".+?);(.+?\\")</code> (example <a href="https://regex101.com/r/iW8oY5/1" rel="nofollow">here</a>) and replace the string with regex groups number 1 and 2. This expression will look for strings containing <code>\"title\"</code> and a <code>;</code> character within it and use this information to create two regular expression groups, this given <code>\"title\" : \"244442424268391(:)7)$(.:$?3.&amp;!&amp;3$83;767:2\",</code>, the output would be:;</p>
<pre><code>Group 1: \"title\" : \"244442424268391(:)7)$(.:$?3.&amp;!&amp;3$83
Group 2: 767:2\"
</code></pre>
<p>When you combine these 2 strings, you will get the result which you are after.</p>
<pre><code> str0 = re.sub(r'(\\"title\\" : \\".+?);(.+?\\")',r"\1\2", str0) 
</code></pre>
