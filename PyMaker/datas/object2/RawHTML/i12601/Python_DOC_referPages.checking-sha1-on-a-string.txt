<p>The word <code>Fox</code> produces the following sha1 hash:</p>
<p><code>dfcd3454bbea788a751a696c24d97009ca992d17</code></p>
<p>In python I'm simply trying to get this same output by doing the following:</p>
<pre><code>import hashlib

myhash = hashlib.sha1("Fox".encode('utf-8'))
</code></pre>
<p><code>myhash</code> just produces the following byte object:</p>
<p><code>b'\xdf\xcd4T\xbb\xeax\x8au\x1ail$\xd9p\t\xca\x99-\x17'</code></p>
<p>I've tried <code>binascii</code> and none of the methods there seem to be able to produce the above output.</p>
<p>How can I produce the resulting ascii hash from here?</p>
</div>
<div class="post-text" itemprop="text">
<p>You have a <em>hexadecimal representation</em> of a digest. You can use the <a href="https://docs.python.org/3/library/hashlib.html#hashlib.hash.hexdigest" rel="noreferrer"><code>hash.hexdigest()</code> method</a> to produce the same in Python:</p>
<pre><code>&gt;&gt;&gt; import hashlib
&gt;&gt;&gt; myhash = hashlib.sha1("Fox".encode('utf-8'))
&gt;&gt;&gt; myhash.digest()
b'\xdf\xcd4T\xbb\xeax\x8au\x1ail$\xd9p\t\xca\x99-\x17'
&gt;&gt;&gt; myhash.hexdigest()
'dfcd3454bbea788a751a696c24d97009ca992d17'
</code></pre>
<p>You could also convert the binary digest to hexadecimal with the <a href="https://docs.python.org/3/library/binascii.html#binascii.hexlify" rel="noreferrer"><code>binascii.hexlify()</code> function</a>:</p>
<pre><code>&gt;&gt;&gt; import binascii
&gt;&gt;&gt; binascii.hexlify(myhash.digest())
b'dfcd3454bbea788a751a696c24d97009ca992d17'
&gt;&gt;&gt; binascii.hexlify(myhash.digest()).decode('ascii')
'dfcd3454bbea788a751a696c24d97009ca992d17'
</code></pre>
<p>However, that's just a more verbose way of achieving the same thing.</p>
</div>
