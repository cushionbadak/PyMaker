I ran into an error though with the formatter and the syntax error read <code>not all arguments converted during string formatting</code>. My inches to cm and pounds to kg is fine but it didn't show the variable in the string. It looked like this:</p>
<pre><code>print "%s is", cm, "tall in centimeters." % name
</code></pre>
<p>I was having a hard time posting the variables because it wants to be posted a certain way. The variables aren't the issue I'm confused why my variable didn't print using the formatter. <div class="post-text" itemprop="text">
<p>use <a href="https://docs.python.org/2/library/string.html#string-formatting" rel="nofollow"><code>format</code></a>:</p>
<pre><code>print "{0} is {1} tall in centimeters.".format(name, cm)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your string formatting is acting on this :</p>
<pre><code>"tall in centimeters." % name
</code></pre>
<p>As <code>%s</code> is not found in the above string, the interpreter fails.</p>
<p>A way to do what you want :</p>
<pre><code>print "%s is %s tall in centimeters." % (name, cm)
</code></pre>
<p>Or </p>
<pre><code>print "%s is %.2f tall in centimeters." % (name, cm)
</code></pre>
<p>if you'd like to show <code>cm</code> on 2 digits.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do in two common ways - </p>
<p>Using the old way (<code>%</code>):</p>
<pre><code>print "%s is %s tall in centimeters." % (name, cm)
</code></pre>
<p>References: <a href="https://docs.python.org/2/library/stdtypes.html#string-formatting" rel="nofollow">https://docs.python.org/2/library/stdtypes.html#string-formatting</a></p>
<p>Using the new way (<code>format</code>) </p>
<pre><code>print "{} is {} tall in centimeters.".format(name, cm)
</code></pre>
<p>References: <a href="https://docs.python.org/3/library/string.html#string-formatting" rel="nofollow">https://docs.python.org/3/library/string.html#string-formatting</a></p>
<p>Please note using <code>format</code> is a good practice since it's the new way. You can read more about these on this website: <a href="https://pyformat.info/" rel="nofollow">https://pyformat.info/</a> </p>
</div>
