<p>To find count of unique elements of list use the combination of <code>len()</code> and <code>set()</code>.</p>
<pre><code>&gt;&gt;&gt; ls = [1, 2, 3, 4, 1, 1, 2]
&gt;&gt;&gt; len(ls)
7
&gt;&gt;&gt; len(set(ls))
4
</code></pre>
