<div class="post-text" itemprop="text">
<p>You can use the <code>py_modules</code> argument instead of the <code>packages</code> argument to list single file modules.</p>
<p>See <a href="https://docs.python.org/3/distutils/setupscript.html#listing-individual-modules" rel="noreferrer">https://docs.python.org/3/distutils/setupscript.html#listing-individual-modules</a></p>
</div>
<div class="post-text" itemprop="text">
<p>For <code>distutils</code>, from <a href="https://docs.python.org/3/distutils/introduction.html#a-simple-example" rel="nofollow noreferrer">https://docs.python.org/3/distutils/introduction.html#a-simple-example</a> :</p>
<pre><code>from distutils.core import setup
setup(name='foo',
      version='1.0',
      py_modules=['foo'],
      )
</code></pre>
<p>Then you only need a file:</p>
<pre><code>foo.py
</code></pre>
<p>And in Ubuntu 14.04:</p>
<pre><code>sudo python setup.py
</code></pre>
<p>puts it under:</p>
<pre><code>/usr/local/lib/python2.7/dist-packages/foo.py
</code></pre>
<p>without any directories.</p>
</div>
