<div class="post-text" itemprop="text">
<p>Use id selector with <code>a</code> tag selector passed to <code>select_one</code>. <code>select_one</code> returns the first match. It should be faster than using find.</p>
<pre><code>soup.select_one("#downloads a")['href']
</code></pre>

<div class="post-text" itemprop="text">
<p>I think what you are looking for is the <em><a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/#find" rel="nofollow noreferrer">find()</a></em> method, which returns only the first match.</p>
<pre><code>crisporDL=soup.find('a', href=True, text='Guides')['href']
</code></pre>
<p>In this case you don't need to use a for loop.</p>
<p>One other possibility without making much change to you code is to add the <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/#the-limit-argument" rel="nofollow noreferrer">limit argument</a> to your <em>find_all()</em>.</p>
<pre><code>soup.find_all('a', href=True, text='Guides',limit=1)
</code></pre>
<p>Although it would be totally unnecessary here, there can be cases in which you may want to get the <em>first n matches</em>.</p>
