<p>I'm going to assume that you are trying to <code>print</code> the return of <code>easy_donor_rank</code>.  </p>
<pre><code>$ cat test.py
def easy_donor_rank(percent_donated):    
    if percent_donated &lt;= 0:
        print("Error")
    if percent_donated &lt; 2:
        print("Bronze")
    elif percent_donated &gt;= 2 and percent_donated &lt;= 15:
        print("Silver")
    else:
        print("Gold")

print(easy_donor_rank(1.2))
</code></pre>
<p>But because you don't <code>return</code> anything, it will return <code>None</code>, so <code>None</code> gets printed.</p>
<pre><code>$ python3 test.py
Bronze
None
</code></pre>
<p>You just need to <code>return</code> the result instead of <code>print</code>ing it inside the function.<br/>
See <a href="https://stackoverflow.com/q/7129285/2745495">What is the purpose of the return statement?</a></p>
<pre><code>$ cat test.py
def easy_donor_rank(percent_donated):
    if percent_donated &lt;= 0:
        return "Error"
    if percent_donated &lt; 2:
        return "Bronze"
    elif percent_donated &gt;= 2 and percent_donated &lt;= 15:
        return ("Silver")
    else:
        return "Gold"

print(easy_donor_rank(1.2))

$ python3 test.py
Bronze
</code></pre>
