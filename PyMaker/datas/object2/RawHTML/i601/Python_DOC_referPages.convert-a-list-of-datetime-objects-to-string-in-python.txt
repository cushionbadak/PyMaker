<p>I have a list of Dates returning Mondays between two dates</p>
<p>mondays =<code>[datetime.date(2019, 2, 14), datetime.date(2019, 2, 21)]</code></p>
<p>How do I format this list to %Y%m%d format?</p>
<p>Expecting output list to be in the below format:</p>
<p><code>['20190214','20190221']</code></p>
<p>is there a way?</p>
</div>
<div class="post-text" itemprop="text">
<p>the following will work</p>
<pre><code>[date_obj.strftime('%Y%m%d') for date_obj in mondays]
</code></pre>
<p>Output</p>
<pre><code>['20190214', '20190221']
</code></pre>
</div>
<span class="comment-copy">Look at <a href="https://docs.python.org/3/library/datetime.html?highlight=strftime#datetime.datetime.strftime" rel="nofollow noreferrer"><code>datetime.strftime()</code></a>, e.g. <code>[d.strftime('%Y%m%d') for d in mondays]</code></span>
