<div class="post-text" itemprop="text">
<p>It is pretty simple with Python. </p>
<p>Just use <a href="https://docs.python.org/3/library/ast.html" rel="nofollow noreferrer">ast</a> or <a href="https://docs.python.org/2/library/json.html" rel="nofollow noreferrer">json</a> and you're fine.</p>
<pre><code>import ast

string_to_be_converted = '[ "I", "am", "a", "string"]'

mylist = ast.literal_eval(string_to_be_converted)
</code></pre>
<p>Here, <code>type(mylist)</code> shows us that <code>mylist</code> is a <code>list</code>, not a <code>string</code>.</p>
<p>However, <strong>I recommend using <a href="https://docs.python.org/2/library/json.html" rel="nofollow noreferrer">json</a></strong> for such an operation. Not only it is more secure, but also <strong>way faster</strong>.</p>
<pre><code>import json

string_to_be_converted = '[ "I", "am", "a", "string"]'

mylist = json.loads(string_to_be_converted)
</code></pre>
<p><strong>Speed comparison:</strong></p>
<p><strong>Ast:</strong></p>
<pre><code>In [1]: %timeit ast.literal_eval(string_to_be_converted)
13.6 µs per loop
</code></pre>
<p><strong>Json:</strong></p>
<pre><code>In [2]: %timeit json.loads(string_to_be_converted)
2.99 µs per loop
</code></pre>
