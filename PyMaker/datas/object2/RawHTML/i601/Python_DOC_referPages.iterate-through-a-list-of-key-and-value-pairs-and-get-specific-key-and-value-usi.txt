<div class="post-text" itemprop="text">
<p>with this you get a new list with only the keys you are interested on:</p>
<pre><code>new_list = [{
    'category': stanza['category'],
    'component_type': stanza['component_type'],
    'event': stanza['event'],
 } for stanza in data]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you know that every dict inside your current list of dicts should have at least the keys you're trying to extract their data, then you can use <code>dict[key]</code>, however for safety, i prefer using <code>dict.get(key, default value)</code> like this example:</p>
<pre><code>out = [
    {
        'category': elm.get('category'),
        'component_type': elm.get('component_type'),
        'event': elm.get('event'),
        'current_severity': elm.get('current_severity')
    } for elm in data
]
print(out)
</code></pre>
<p>Output:</p>
<pre><code>[{'category': 'software',
  'component_type': 'update',
  'current_severity': 'info',
  'event': 'new update available'},
 {'category': 'software',
  'component_type': 'host',
  'current_severity': 'critical',
  'event': 'misconfiguration'},
 {'category': 'array',
  'component_type': 'hardware',
  'current_severity': 'warning',
  'event': 'failure'}]
</code></pre>
<p>For more informations about when we should use <code>dict.get()</code> instead of <code>dict[key]</code>, see this <a href="https://stackoverflow.com/a/11041421/3926995">answer</a></p>
</div>
<div class="post-text" itemprop="text">
<p>You have a list of dictionaries, simple way to iterate over this is </p>
<pre><code>category = [x['category'] for x in data]
</code></pre>
<p>Which prints the values of <strong>category</strong> key</p>
<pre><code>['software', 'software', 'array']
</code></pre>
<p>Do the same for <code>component_type</code>, <code>event</code> and <code>current_severity</code> and you're good to go</p>
