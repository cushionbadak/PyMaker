<p>you might want to format your strings like the following: </p>
<p><a href="https://stackoverflow.com/questions/8234445/python-format-output-string-right-alignment">Python: Format output string, right alignment</a> 
the with in the output like </p>
<pre><code>for align, text in zip('&lt;^&gt;', ['left', 'center', 'right']): 
    '{0:{fill}{align}16}'.format(text, fill=align, align=align)
</code></pre>
<p>in the output, you can specify the with (or get that based on the length of your inputs). </p>
<p>I have not worked with gtk2.0 myself but perhaps it will help <a href="https://docs.python.org/3/library/string.html#formatstrings" rel="nofollow noreferrer">https://docs.python.org/3/library/string.html#formatstrings</a></p>
