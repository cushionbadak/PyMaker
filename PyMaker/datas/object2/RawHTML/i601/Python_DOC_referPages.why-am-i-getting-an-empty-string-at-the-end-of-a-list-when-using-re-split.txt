<div class="post-text" itemprop="text">
<p>According to <a href="http://docs.python.org/3/library/re.html#re.split" rel="nofollow">re.split documentation</a>:</p>
<blockquote>
<p>If there are capturing groups in the separator and it matches <strong>at the
  start of the string</strong>, the result will start with an empty string. The
  same holds for <strong>the end of the string</strong>:</p>
</blockquote>
<p>If you want to get <code>People</code> and <code>Robots</code>, use <a href="http://docs.python.org/3/library/re.html#re.findall" rel="nofollow">re.findall</a>:</p>
<pre><code>&gt;&gt;&gt; re.findall('([A-Z][a-z]+)', 'PeopleRobots')
['People', 'Robots']
</code></pre>
<p>You can omit grouping:</p>
<pre><code>&gt;&gt;&gt; re.findall('[A-Z][a-z]+', 'PeopleRobots')
['People', 'Robots']
</code></pre>
</div>
