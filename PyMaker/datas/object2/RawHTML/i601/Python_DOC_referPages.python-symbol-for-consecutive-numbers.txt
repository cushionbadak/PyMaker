<p>This is a mathematical function called a <a href="https://en.wikipedia.org/wiki/Factorial" rel="nofollow noreferrer">factorial</a>, and is covered in depth in <a href="https://stackoverflow.com/questions/5136447/function-for-factorial-in-python">another question</a>.</p>
<p>The simplest approach is:</p>
<pre><code>import math
math.factorial(5)
</code></pre>
<p>A functional approach:</p>
<pre><code>from functools import reduce
import operator
answer = reduce(operator.mul, range(1, 6))
</code></pre>
<p>Loop approach:</p>
<pre><code>answer = 1
for i in range(5):
    answer *= i+1
</code></pre>
<p><a href="https://stackoverflow.com/questions/41062600/create-list-of-factorials-using-list-comprehension">Another answer uses a list comprehension</a>.</p>
