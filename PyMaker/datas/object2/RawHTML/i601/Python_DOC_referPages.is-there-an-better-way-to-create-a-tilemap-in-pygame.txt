<div class="post-text" itemprop="text">
<p><code>tilemap</code> is a list. The list  has 4 elements and each element of the list is a list with 4 elements, too. </p>
<blockquote>
<pre><code> tilemap = [
       [GRASS, GRASS, GRASS, WATER],
       [GRASS, GRASS, WATER, GRASS],
       [GRASS, WATER, GRASS, GRASS],
       [WATER, GRASS, GRASS, GRASS]
    ]
</code></pre>
</blockquote>
<p>So the "width" and the "height" of the tilemap is 4:</p>
<pre class="lang-py prettyprint-override"><code>mapwidth = 4
mapheight = 4
</code></pre>
<p>But you don't need that variables at all, because you can get the length of the list by <a href="https://docs.python.org/3/library/functions.html#len" rel="nofollow noreferrer"><code>len()</code></a>: </p>
<pre class="lang-py prettyprint-override"><code>for row in range( len(tilemap.tilemap) ):

    for column in range( len(tilemap.tilemap[row]) ):

        window.blit(tilemap.textures[tilemap.tilemap[row][column]],
                    (column*tilemap.tilesize, row*tilemap.tilesize))
</code></pre>
<hr/>
<p>If you want to put more tiles on the map, then it is sufficient to add more tiles to <code>tilemap:</code></p>
<p>e.g.</p>
<pre><code>tilemap = [
       [WATER, WATER, WATER, WATER, WATER, WATER],
       [WATER, GRASS, GRASS, GRASS, WATER, WATER],
       [WATER, GRASS, GRASS, WATER, GRASS, WATER],
       [WATER, GRASS, WATER, GRASS, GRASS, WATER],
       [WATER, WATER, GRASS, GRASS, GRASS, WATER],
       [WATER, WATER, WATER, WATER, WATER, WATER]
    ]
mapwidth = len(tilemap.tilemap)
mapheight = len(tilemap.tilemap[0])
</code></pre>
