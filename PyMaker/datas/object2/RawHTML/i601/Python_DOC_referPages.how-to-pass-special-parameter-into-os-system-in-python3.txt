<div class="post-text" itemprop="text">
<p>The <code>shlex</code> module knows how to escape strings you want to pass to the shell.</p>
<p>However, the shell isn't doing anyting useful here, so it's better to simply avoid it.</p>
<pre><code>from subprocess import run
run(["scrapy", "crawl", "googlebook", "-a", "arg=" + scrapy_url],
    # This will raise an exception if scrapy fails
    # which is _probably_ a good idea in most scenarios
    check=True)
</code></pre>
<p>Of course, given that <code>scrapy</code> is a Python library, perhaps the best solution is to simply <code>import scrapy</code> and take it from there.</p>
</div>
