<div class="post-text" itemprop="text">
<p>I have a list and I want to add an integer to every value in the list, past the fifth value.</p>
<pre><code>PageNumbers = [30, 50, 80, 120, 160, 200, 240, 280]
PageNumbers = [x+100 for x in PageNumbers]
</code></pre>
<p>How do I make it <code>PageNumbers = if PageNumber[index] &gt; 5, then [x+100 for x in PageNumbers] else PageNumber[index]</code>.</p>
<p>My desired output is <code>[30, 50, 80, 120, 160, 300, 340, 380]</code></p>
</div>
<div class="post-text" itemprop="text">
<p>Divide the list into two parts:</p>
<pre><code>PageNumbers = PageNumbers[:5] + [x+100 for x in PageNumbers[5:]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Check out <code>enumerate</code> to grab both the index and the value while iterating a list.</p>
<p>This will allow you to do something like the below:</p>
<pre><code>PageNumbers = [30, 50, 80, 120, 160, 200, 240, 280]
PageNumbers = [val + 100 if idx &gt; 4 else val for idx, val in enumerate(PageNumbers)]
</code></pre>
<p>*Note that I actually used <code>idx &gt; 4</code> which means anything past the 5th value (0-index based) as you noted and used in your example. You can, of course, change to use whichever index you'd like.</p>
<p>See a short post and discussion on <code>enumerate</code> here:
<a href="https://stackoverflow.com/questions/10777271/python-using-enumerate-inside-list-comprehension">Python using enumerate inside list comprehension</a></p>
</div>
<div class="post-text" itemprop="text">
<pre><code>for i in range(5, len(PageNumbers)):
    PageNumbers[i] += 100
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just assign it back </p>
<pre><code>PageNumbers[5:]=[x +100 for x in PageNumbers[5:]]
PageNumbers
[30, 50, 80, 120, 160, 300, 340, 380]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3/reference/expressions.html#conditional-expressions" rel="nofollow noreferrer">conditional expression</a> and the <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer">enumerate</a> builtin:</p>
<pre><code>&gt;&gt;&gt; PageNumbers = [30, 50, 80, 120, 160, 200, 240, 280]
&gt;&gt;&gt; [x + 100 if i &gt; 4 else x for i, x in enumerate(PageNumbers)]
[30, 50, 80, 120, 160, 300, 340, 380]

</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just an example using NumPy:</p>
<pre><code>import numpy as np

PageNumbers = np.array([30, 50, 80, 120, 160, 200, 240, 280])

PageNumbers[5:] += 100

print(PageNumbers)
#=&gt; [ 30  50  80 120 160 300 340 380]
</code></pre>
