<div class="post-text" itemprop="text">
<p>The <a href="https://docs.scrapy.org/en/latest/topics/media-pipeline.html#std:setting-IMAGES_STORE" rel="nofollow noreferrer"><code>IMAGE_STORE</code></a> settings must be a single path.</p>
<p>Replace:</p>
<pre><code>IMAGES_STORE = {
    '/home/jaki/Dev/WebScrapingScratch/images'
}
</code></pre>
<p>with:</p>
<pre><code>IMAGES_STORE = '/home/jaki/Dev/WebScrapingScratch/images'
</code></pre>
<p><code>{'asdf'}</code> is a <a href="https://docs.python.org/3/library/stdtypes.html#set-types-set-frozenset" rel="nofollow noreferrer"><code>set</code></a> with the string <code>asdf</code>, hence the error message.</p>
</div>
