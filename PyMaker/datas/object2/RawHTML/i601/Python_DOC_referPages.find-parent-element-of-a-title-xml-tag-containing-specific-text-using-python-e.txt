<div class="post-text" itemprop="text">
<p>Do a (nested) iteration in 2 steps: on <em>sec</em> and then on <em>title</em>. Something like:</p>
<pre class="lang-py prettyprint-override"><code>for sec in parent.iter("sec"):
    for title in sec.iter("title"):
        text = title.text
        if text and "methods" in text.lower():
            print("**title: " + text + " **** sec id: " + sec.get("id", ""))
</code></pre>
<p>For more details, check <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#module-xml.etree.ElementTree" rel="nofollow noreferrer">[Python 3]: xml.etree.ElementTree - The ElementTree XML API</a>.</p>
