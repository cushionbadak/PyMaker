<p>Check indents in your class or refactor file reading to:</p>
<pre><code>import scrapy

class TermSpider(scrapy.Spider):
    name = 'TermCheck'
    allowed_domains = ['google.com']

    def start_requests(self):
        with open('urls.csv') as file:
            for line in file.readlines():
                yield scrapy.Request(line.strip())

    def parse(self, response):
        item = {
            'search_title': response.css('input#sbhost::attr(value)').get(),
            'results': response.css('#resultStats::text').get(),
            'url': response.url,
        }
        yield item
</code></pre>
<p>Also please check your csv file and remove quotes and commas.</p>
