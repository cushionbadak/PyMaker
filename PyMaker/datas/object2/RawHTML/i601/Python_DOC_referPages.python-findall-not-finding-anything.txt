<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><code>findall()</code></a> returns a simple list of strings matching the pattern.</p>
<p>It has no <code>group()</code> method, just omit that:</p>

<div class="post-text" itemprop="text">
<p>It seems that you're trying to match phone numbers in <code>resume</code>, for that you can use:</p>
<pre><code>resume = '''
    (738) 383-5729
    (373) 577-0492
    (403) 443-2759
    (375) 880-8576
    (641) 576-2342
    (951) 268-8744
    '''

mo = re.findall(r'\(\d{3}\) \d{3}-\d{4}', resume)

for x in mo:
    print(x)
</code></pre>

<div class="post-text" itemprop="text">
<p>Since (it looks like) you're just out for the numbers, you could do something like</p>
<pre><code>&gt;&gt;&gt; [''.join(c for c in l if c in '0123456789')  for l in resume.strip().splitlines()]
['7383835729', '3735770492', '4034432759', '3758808576', '6415762342', '9512688744']
</code></pre>
<p>That might save you some trouble from internationally formed numbers (such as <code>+46-(0)7-08/123 456</code> and the like).</p>
