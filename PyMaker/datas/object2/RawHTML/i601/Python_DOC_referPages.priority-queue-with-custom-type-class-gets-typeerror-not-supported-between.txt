trace=[(0, 0)]

priori_queue

<div class="post-text" itemprop="text">
<p>You're passing <code>temp</code> in <code>PriorityQueue(temp)</code>, but the only argument to <a href="https://docs.python.org/3/library/queue.html#queue.PriorityQueue" rel="nofollow noreferrer"><code>PriorityQueue</code></a> is <code>maxsize</code>, which should be an integer. The default is <code>0</code>, which means the queue grows with no size limit.  </p>
<pre><code>priori_queue = queue.PriorityQueue()
priori_queue.put(temp)
</code></pre>
