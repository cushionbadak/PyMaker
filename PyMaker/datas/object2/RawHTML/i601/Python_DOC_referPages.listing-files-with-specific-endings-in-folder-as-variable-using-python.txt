<p>I am trying to list files with specific ending ('.txt') in folder which was set as a variable, using python.</p>
<p>I tried to use glob.glob('userFolder/*.txt') in order to do it.</p>
<pre><code>import os
import glob
userFolder='/homes/myFolder'
glob.glob('userFolder/*.txt')
</code></pre>
<p>I got an empty list.</p>
</div>
<div class="post-text" itemprop="text">
<p>The text <code>userFolder</code> in your <code>glob()</code> call is just part of the string value, it's not related to the variable with the same name. If it was, you could never use something like <code>print</code> or <code>os</code> or any other variable name directly in a string.</p>
<p>You could just use <code>+</code> to concatenate the variable value with the glob pattern:</p>
<pre><code>text_files = glob.glob(userFolder + '/*.txt')
</code></pre>
<p>but the better method is to use <a href="https://docs.python.org/3/library/os.path.html#os.path.join" rel="nofollow noreferrer"><code>os.path.join()</code></a> to handle path construction:</p>
<pre><code>files = glob.glob(os.path.join(userFolder, '*.txt'))
</code></pre>
<p>Another option is to use the [<code>pathlib</code> module], which has its own glob support:</p>
<pre><code>import pathlib

userFolder = pathlib.Path('/homes/myFolder')
text_files = userFolder.glob('*.txt')
</code></pre>
</div>
