<div class="post-text" itemprop="text">
<p>The df looks like</p>
<pre><code>  type
0 a
1 b
2 c
</code></pre>
<p>The output should looks like</p>
<pre><code>string="'a','b','c'"
</code></pre>
<p>I have used <code>df.type.str.cat(sep=',')</code> to add , in between but how to add quotation mark for each element?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>join</code></a> or <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.Series.str.cat.html" rel="nofollow noreferrer"><code>Series.str.cat</code></a>:</p>
<pre><code>string = ','.join("'"+df['type']+"'")
</code></pre>
<p>Or:</p>
<pre><code>string = ("'"+df['type']+"'").str.cat(sep=',')

string
"'a','b','c'"
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>One other alternative may be to use <code>format</code> with <code>apply</code> and <code>sum</code>:</p>
<pre><code>result = df.type.apply("'{}',".format).sum()[:-1]
</code></pre>
</div>
