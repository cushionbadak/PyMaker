<div class="post-text" itemprop="text">
<p>The program is waiting for your input, since you're using <code>stdin.read()</code>. This waits until the input is terminated by (for instance) pressing ctrl-D.
Normally you'd use <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer"><code>input()</code></a> for this, which reads one line from stdin.</p>
<pre><code>def main():
    line = input('&gt; ')
    n = int(line)
    print(get_fib_last_digit_fast(n))
</code></pre>
</div>
<span class="comment-copy">You don't want to use <code>sys.stdin.read()</code> here... it will read until it hits the end of file (EOF).  The enter key does not signify the end of the file, only the start of a new line.  You can either try <code>sys.stdin.readline()</code> or use the built-in function <code>input()</code> instead (assuming you're using Python 3).</span>
