<div class="post-text" itemprop="text">
<p>Consider the following file path:</p>
<pre><code>path = r'C:\this\is\a\test_test\file.asc'
</code></pre>
<p>How can I extract the nth component of the path? </p>
<pre><code>"test_test"
</code></pre>
<p><em>without</em> slicing </p>
<pre><code>path[13:22]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>split()</code> as follows:</p>
<pre><code>&gt;&gt;&gt; path = r'C:\this\is\a\test_test\file.asc'
&gt;&gt;&gt; path = path.split('\\')
&gt;&gt;&gt; print path[4]
test_test
</code></pre>
<p>You need to use <code>'\\'</code> instead of <code>'\'</code> since you need to escape the backslash. Otherwise, you will end up escaping the second <code>'</code> and get an EOL parsing error.</p>
</div>
<div class="post-text" itemprop="text">
<p>Split the path with <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>str.split</code></a>:</p>
<pre><code>&gt;&gt;&gt; path = r'C:\this\is\a\test_test\file.asc'
&gt;&gt;&gt; path.split("\\") # Make sure you double the \
['C:', 'this', 'is', 'a', 'test_test', 'file.asc']
&gt;&gt;&gt; path.split("\\")[4]
'test_test'
&gt;&gt;&gt;
</code></pre>
</div>
