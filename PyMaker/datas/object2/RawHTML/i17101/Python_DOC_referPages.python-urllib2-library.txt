<div class="post-text" itemprop="text">
<p>It's most probable that you are using Python 3.</p>
<p>With Python versions &gt; 3, those relevant modules are structured like <a href="http://legacy.python.org/dev/peps/pep-3108/#urllib-package" rel="nofollow">this</a>:</p>
<pre><code>Current Name      Replacement Name
urllib2           urllib.request, urllib.error
urlparse          urllib.parse
urllib            urllib.parse, urllib.request, urllib.error [6]
robotparser       urllib.robotparser
</code></pre>
</div>
