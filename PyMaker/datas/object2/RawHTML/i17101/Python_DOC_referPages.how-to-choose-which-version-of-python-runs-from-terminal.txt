<div class="post-text" itemprop="text">
<p>I have a few different versions of python on my computer. How do I choose which one is run from my terminal when I type "python" into the prompt?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>which</code> to see where your python command resides. Then use <code>ls -l</code> to find out where it <em>really</em> is. Then link the one you want instead. Note that the other installed versions are usually all available by their respective names.</p>
<pre><code>$ which python
/usr/bin/python
$ ls -l /usr/bin/python
lrwxrwxrwx 1 root root 9 Jun 18  2013 /usr/bin/python -&gt; python2.7
$ ls /usr/bin/python*
/usr/bin/python   /usr/bin/python2.7         /usr/bin/python2-config
/usr/bin/python2  /usr/bin/python2.7-config  /usr/bin/python-config
$ sudo ln -sf /usr/bin/python2 /usr/bin/python
</code></pre>
<p>Note that this changes which Python version <strong>all programs for all users on your computer will probably use</strong>! If you only want to change it for yourself. You can alias it by adding a <code>alias python='/usr/bin/python2'</code> line (with <code>python2</code> replaced by the version you want) to <code>~/.bashrc</code> in linux or <code>~/.bash_profile</code> in Mac. (You'll need to restart your terminal session in this case.)</p>
</div>
<div class="post-text" itemprop="text">
<p>You should have multiple executables for every python version you have. For example, if I type <code>python</code> and hit tab, I see:</p>
<pre><code>$ python
python             python2.5-config   python2.7-config   python3.3          python3.3m-config  pythonw2.7         pythonw3.3-32      
python-config      python2.6          python3            python3.3-32       pythonw            pythonw3           
python2            python2.6-config   python3-32         python3.3-config   pythonw2.5         pythonw3-32        
python2.5          python2.7          python3-config     python3.3m         pythonw2.6         pythonw3.3 
</code></pre>
<p>So, if, for example, I want python 2.5 version - I run <code>python2.5</code>.</p>

<div class="post-text" itemprop="text">
<p>To choose which version of python is run when you type 'python' into a terminal, you may want to try using an alias.</p>
<p>For example:</p>
<pre><code>alias python='python2.7'
</code></pre>
<p>Would make python2.7 execute when you type 'python' into your terminal.</p>
