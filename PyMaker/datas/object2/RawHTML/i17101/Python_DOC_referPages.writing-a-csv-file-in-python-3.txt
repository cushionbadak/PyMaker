<div class="post-text" itemprop="text">
<p>There is no need to create variable <code>data</code>. Use built-in function zip. It makes an iterator that aggregates elements from each of the iterables.</p>
<pre><code>for item in zip(a, b, c):
    mywriter.writerow(item)
</code></pre>
<p>Documentation for reference <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow">https://docs.python.org/3/library/functions.html#zip</a></p>
</div>

<p>You have to break the problem down first:</p>
<pre><code>a = [1,3,5,7]
b = [2,4,6,8]
c = [0,0,0,0]
data = [a,b,c]

for i in range(len(data[0])):

        print data[j][i],
    print ''
</code></pre>

<pre><code>1 2 0 
3 4 0

</code></pre>
<h2>Putting it all together:</h2>
<pre><code>import csv

my_output = open('myfile.csv', 'wb')
mywriter = csv.writer(my_output)

a = [1,3,5,7]
b = [2,4,6,8]
c = [0,0,0,0]
data = [a,b,c]

for i in range(len(data[0])):
    row = [data[j][i] for j in range(len(data))]
    mywriter.writerow(row)

my_output.close()
</code></pre>
<h3>Shorter (safer) version:</h3>
<pre><code>import csv

my_output = open('myfile.csv', 'wb')
mywriter = csv.writer(my_output)

a = [1,3,5,7]
b = [2,4,6,8]
c = [0,0,0,0]
data = [a,b,c]

for row in zip(*data):
    mywriter.writerow(row)

my_output.close()
</code></pre>
<p>See <a href="https://docs.python.org/2.7/library/functions.html#zip" rel="nofollow"><code>zip([iterable, ...])</code></a> for information on this built-in function.</p>
