<div class="post-text" itemprop="text">
<p>The easiest way to pass a variable between functions is to define the necessary function to accept an argument. <a href="https://docs.python.org/2/tutorial/controlflow.html#defining-functions" rel="nofollow">Check out functions -part in the official tutorial</a></p>
<p>In your case you want to pass <code>variable1</code> to <code>b</code>. Define <code>b</code> as such:</p>
<pre><code>def b(var):
     print var
     # Whatever else you want to do..

def a():
    variable1 = 0
    answer = raw_input()
    if answer == "a":
       print "Correct"
       b(variable1)
</code></pre>
<p>Your original code does not work because variables inside functions are local to those functions. Using function arguments is the correct way of doing what you want.</p>
