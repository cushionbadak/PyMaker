<p>Input:</p>
<pre><code>[['Tryndamere', 'Barbarian', 'Fighter'],['Caitlyn', 'Sheriff', 'Marksmen'],...['Veigar', 'Midget', 'Mage']]
</code></pre>
<p>Expected output:</p>
<pre><code>['Barbarian']['Caitlyn']['Fighter']['Mage']['Marksmen']['Midget']['Sheriff']['Tryndamere']['Veigar']...
</code></pre>
<p>Question:
How to separate the lists within a list till they become individual elements? Thanks guys</p>

<div class="post-text" itemprop="text">
<p>Let's say the data looks like this</p>
<pre><code>&gt;&gt;&gt; lists = [
...     ['Tryndamere', 'Barbarian', 'Fighter'],
...     ['Caitlyn', 'Sheriff', 'Marksmen'],
...     ['Veigar', 'Midget', 'Mage']
... ]
</code></pre>
<p>then, you can use list comprehension like this</p>
<pre><code>&gt;&gt;&gt; from pprint import pprint
&gt;&gt;&gt; pprint([[item] for c_list in lists for item in c_list])
[['Tryndamere'],
 ['Barbarian'],
 ['Fighter'],
 ['Caitlyn'],
 ['Sheriff'],
 ['Marksmen'],
 ['Veigar'],
 ['Midget'],
 ['Mage']]
</code></pre>
<p>Or using <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow noreferrer"><code>itertools.chain</code></a></p>
<pre><code>&gt;&gt;&gt; from itertools import chain
&gt;&gt;&gt; pprint([[item] for item in chain.from_iterable(lists)])
[['Tryndamere'],
 ['Barbarian'],
 ['Fighter'],
 ['Caitlyn'],
 ['Sheriff'],
 ['Marksmen'],
 ['Veigar'],
 ['Midget'],
 ['Mage']]
</code></pre>
<p>But if you are thinking about flattening those lists into a single list, then please check <a href="https://stackoverflow.com/questions/952914/making-a-flat-list-out-of-list-of-lists-in-python">this</a>. You can easily extend the <code>chain</code> version to do that, like this</p>
<pre><code>&gt;&gt;&gt; pprint(list(chain.from_iterable(lists)))
['Tryndamere',
 'Barbarian',
 'Fighter',
 'Caitlyn',
 'Sheriff',
 'Marksmen',
 'Veigar',
 'Midget',
 'Mage']
</code></pre>
<p>Or the list comprehension version,</p>
<pre><code>&gt;&gt;&gt; pprint([item for c_list in lists for item in c_list])
['Tryndamere',
 'Barbarian',
 'Fighter',
 'Caitlyn',
 'Sheriff',
 'Marksmen',
 'Veigar',
 'Midget',
 'Mage']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just use a list comprehension that iterates over the sublist also. Then you can convert that string to a list of length 1 just by wrapping it in square brackets.</p>
<pre><code>&gt;&gt;&gt; lst = [['Tryndamere', 'Barbarian', 'Fighter'], 
           ['Caitlyn', 'Sheriff', 'Marksmen'], 
           ['Veigar', 'Midget', 'Mage']]
&gt;&gt;&gt; [[item] for sub in lst for item in sub]
[['Tryndamere'], ['Barbarian'], ['Fighter'], ['Caitlyn'], ['Sheriff'], ['Marksmen'], ['Veigar'], ['Midget'], ['Mage']]
</code></pre>
