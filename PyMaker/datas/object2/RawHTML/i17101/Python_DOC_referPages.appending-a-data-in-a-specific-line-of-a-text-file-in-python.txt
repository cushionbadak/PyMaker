<div class="post-text" itemprop="text">
<p>Let's say I have this textfile: date.txt. month|day|year</p>
<pre><code>January|20|2014
February|10|
March|5|2013
</code></pre>
<p>I want to put 2012 after February|10|. how can i do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You need to read the file into memory, modify the desired line and write back the file.</p>
<pre><code>temp = open('temp', 'wb')
with open('date.txt', 'r') as f:
    for line in f:
        if line.startswith('February'):
            line = line.strip() + '2012\n'
        temp.write(line)
temp.close()
shutils.move('temp', 'data.txt')
</code></pre>
<p>If you don't want to use a temporary file:</p>
<pre><code>with open('date.txt', 'rw') as f:
    lines = f.readlines()
    for i, line in enumerate(lines):
        if line.startswith('February'):
            line[i] = line[i].strip() + '2012\n'
    f.seek(0)
    for line in lines:
        f.write(line)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer"><code>csv</code></a> module, for example:</p>
<pre><code>import csv

data = [
    "January|20|2014",
    "February|10|",
    "March|5|2013"
]

reader = csv.reader(data, delimiter="|")
for line in reader:
    line = [i if i != "" else "2012" for i in line]
    print(line)
</code></pre>
<p>Please note: <code>csv.reader()</code> take as argument any iterable object. So, you can easily pass it a file object</p>
