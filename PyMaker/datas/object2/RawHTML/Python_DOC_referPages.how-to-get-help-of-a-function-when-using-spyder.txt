<div class="post-text" itemprop="text">
<p>You need to install a library called <code>rope</code>:</p>
<p><strong>Python 2.x</strong>:</p>
<pre><code>pip install rope
</code></pre>
<p><strong>Python 3.x</strong>:</p>
<pre><code>pip install rope_py3k
</code></pre>
<blockquote>
<p>Code completion on the Editor should work now with both <code>Ctrl+Space</code>
  and with <code>Tab</code>.</p>
