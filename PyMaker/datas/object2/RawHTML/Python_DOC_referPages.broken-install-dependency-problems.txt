<div class="post-text" itemprop="text">
<p>I wanted to upgrade python 3.5 to python 3.6 and because I was dumb and didn't run <code>apt-get remove python3</code> command I deleted the /usr/bin/python3 folder and now I have troubles installing another package.
The output of the error is : <a href="https://pastebin.com/ytrKUZRc" rel="nofollow noreferrer">https://pastebin.com/ytrKUZRc</a></p>
<p>What I tried:</p>
<pre><code>apt-get install python3
</code></pre>
<p>then when the error showed:</p>
<pre><code>apt-get install -f
</code></pre>
<p>then </p>
<pre><code>apt-get update
</code></pre>
<p>then</p>
<p><code>dpkg --configure -a</code> then</p>
<p><code>apt-get clean</code> but it didn't work.</p>
</div>
<div class="post-text" itemprop="text">
<p>What I did to solve the problem:</p>
<p>I deleted my source.list in <code>/etc/apt/sources.list</code> and generated a new one.</p>
<p>Then I ran the command <code>apt-get install python3</code>, </p>
<p>after it finished ran the command 
<code>apt-get -f install</code> </p>
<p>after everything was finished I ran the command 
<code>dpkg --configure -a</code> </p>
<p>and then the dependencies that were still broken I reinstalled them using the command</p>
<p><code>apt-get install --reinstall dependency name</code> </p>
</div>
