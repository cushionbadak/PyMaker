<p>To quote the <a href="https://docs.python.org/3/library/os.path.html" rel="noreferrer"><code>os.path</code> module documentation</a>:</p>
<blockquote>
<p>Unlike a unix shell, Python does not do any automatic path expansions. Functions such as <a href="https://docs.python.org/3/library/os.path.html#os.path.expanduser" rel="noreferrer"><code>expanduser()</code></a> and <a href="https://docs.python.org/3/library/os.path.html#os.path.expandvars" rel="noreferrer"><code>expandvars()</code></a> can be invoked explicitly when an application desires shell-like path expansion. (See also the <code>glob</code> module.)</p>
</blockquote>
<p>So you can use those functions to perform that expansion:</p>
<pre><code>&gt;&gt;&gt; os.path.expanduser('~/ApenasMeu')
'/home/macabeus/ApenasMeu'
&gt;&gt;&gt; os.path.isdir(os.path.expanduser('~/ApenasMeu'))
True
</code></pre>
