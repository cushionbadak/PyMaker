<div class="post-text" itemprop="text">
<p>Here is a short piece of code which is meant to check what data type a certain variable is, the return says <code>True</code>, followed by <code>False</code>, <code>False</code>, <code>False</code>. Could somebody tell me what is wrong with this code and how I could do a process such as this more efficiently?</p>
<pre><code>examples = [100, 1.45, "Bob", True]
types = [int, float, str, bool]

for x in range(0, 3):
    if type(examples[x]) == type(types[x]):
        print("True")
    else:
        print("False")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have to compare the type with the word in the list and not it's type. 
Also note that <a href="https://docs.python.org/2/library/functions.html#range" rel="nofollow"><code>range</code></a> excludes the second argument, and hence you need to do <code>range(0,4)</code> or <code>range(4)</code> which is better.</p>
<pre><code>for x in range(0, 4):
    if type(examples[x]) == (types[x]):
        print("True")
    else:
        print("False")
</code></pre>
<p>A better way is to use <a href="https://docs.python.org/2/library/functions.html#isinstance" rel="nofollow"><code>isinstance</code></a></p>
<blockquote>
<p>Return true if the object argument is an instance of the classinfo argument, or of a (direct, indirect or virtual) subclass thereof.</p>
</blockquote>
<p>You can change your code as,</p>
<pre><code>for x in range(0, 4):
    if isinstance(examples[x],types[x]):
        print("True")
    else:
        print("False")
</code></pre>
<p>As <code>isinstance</code> returns a boolean value, you can directly do</p>
<pre><code>for x in range(0, 4):
    print(isinstance(examples[x],types[x]))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You don't want to do <code>type(types[x])</code>.  <code>types</code> already contains types.  If you take the type of a type, you get <code>type</code>.</p>
<p>Just do <code>if type(examples[x]) == types[x]</code>.</p>
<p>Even better is to do it this way:</p>
<pre><code>for example, typ in zip(examples, types):
    if type(example) == typ:
        print("True")
    else:
        print("False")
</code></pre>
<p>This has the added advantage of comparing all the types in the list, not just the first 3.</p>
<p>Why you want to do this is another question.</p>
</div>
<div class="post-text" itemprop="text">
<p>Elements of <code>types</code> are classes (types) and <code>type</code> returns <code>type</code> for each of those. There're no types in <code>examples</code>, thus <code>type(examples[x]) == type</code> will always evaluate to <code>False</code>.</p>
<p>This should work:</p>
<pre><code>for x in range(4):
    if type(examples[x]) == types[x]: # &lt;- remove type(...)
        print("True")
    else:
        print("False")
</code></pre>
<p>You could also do this using <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow"><code>map</code></a> and <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow"><code>isinstance</code></a>:</p>
<pre><code>In [3]: for x in map(isinstance, examples, types):
   ...:     print(x)
   ...:     
True
True
True
True
</code></pre>
