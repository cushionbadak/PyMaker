<p>How do I set the arguments when I create an exception? Where do I find the list of arguments available for each <code>Exception</code> subclass? What are the best practices?</p>
<p>For example, if I know that a file does not exist, how do I raise the <code>FileNotFoundError(missing_file)</code> exception?</p>
<p>This shows the list of members of the <code>FileNotFoundError</code> exception:</p>
<pre><code>&gt;&gt;&gt; [a for a in dir(FileNotFoundError) if a&gt;'a']
['args', 'characters_written', 'errno', 'filename', 'filename2', 'strerror', 'winerror', 'with_traceback']
</code></pre>
<p>This shows that it is possible to set some of the arguments when creating an exception: </p>
<pre><code>&gt;&gt;&gt; FileNotFoundError(1,2,3,4,5).filename
3
&gt;&gt;&gt; FileNotFoundError(1,2,3,4,5).filename2
5
</code></pre>
<p>And this shows that those arguments mean something:</p>
<pre><code>&gt;&gt;&gt; raise FileNotFoundError(1,2,3,4,5)
Traceback (most recent call last):
  File "&lt;pyshell#8&gt;", line 1, in &lt;module&gt;
    raise FileNotFoundError(1,2,3,4,5)
FileNotFoundError: [WinError 4] 2: 3 -&gt; 5
</code></pre>
<p>So I know the arguments are there, can be set and can be used. But I couldn't find any documentation about it.</p>
<p>Tthe <a href="https://docs.python.org/3.4/reference/simple_stmts.html#the-raise-statement" rel="nofollow noreferrer">raise</a> documentation, the <a href="https://docs.python.org/3/library/exceptions.html#FileNotFoundError" rel="nofollow noreferrer"><code>FileNotFoundError</code></a> documentation or <a href="https://stackoverflow.com/questions/2052390/manually-raising-throwing-an-exception-in-python">this post</a> don't talk about the exception arguments.</p>

<p>Looking at the page you linked for <code>FileNotFoundError</code>, it does say that it's a subclass of <code>OSError</code>, which has signature</p>
<pre><code>OSError(errno, strerror[, filename[, winerror[, filename2]]])
</code></pre>
<p>One could reasonably infer that the subclass constructor has the same signature.</p>
