<p>First of all for checking the format of your file you can not use <code>in</code> operand you can use <code>endswith</code> or <code>fnmatch</code> and then sort the file list with <a href="https://docs.python.org/3/library/os.html#os.stat_result.st_ctime" rel="nofollow"><code>os.stat().st_ctime</code></a>:</p>
<pre><code>import  os, datetime
from fnmatch import fnmatch

def get_path_list(arg):
app_list = []
for file in os.listdir(arg):
    if fnmatch(file,'*.jar'):
        app_list.append(file)
print sorted(app_list,key=lambda f:os.stat(f).st_ctime)
</code></pre>

<div class="post-text" itemprop="text">
<p>You can use <em>os.path.getmtime()</em> to get the latest modified time of a file. If you combine this with the list function <em>sort</em>, you can specify the key to sort on. By sending a lambda function as key, we specify the key to be the last modified time. The <em>os.path.join(arg, x)</em> is to get the full path to the file, where <em>arg</em> is your path argument, and <em>x</em> is the current file.</p>
<pre><code>import  os, datetime

def get_path_list(arg):
    app_list = []
    for file in os.listdir(arg):
        if '.jar' in file:
            app_list.append(file)

    app_list.sort(key=lambda x: os.path.getmtime(os.path.join(arg, x)))

    print app_list

if __name__ == '__main__':
    arg = '/home/xyz/test
</code></pre>
