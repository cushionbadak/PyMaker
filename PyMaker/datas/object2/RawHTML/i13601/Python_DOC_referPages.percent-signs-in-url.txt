<div class="post-text" itemprop="text">
<p>Is it possible to change an URL which contains percent signs like</p>
<pre><code> http%3A%2F%2Funiversities.ac%2Fshow_article.php%3Fid%3D61&amp;amp
</code></pre>
<p>to normal URL readable by human like</p>
<pre><code>http://universities.ac/show_article.php?id=61
</code></pre>
<p>using Selenium?</p>
</div>
<div class="post-text" itemprop="text">
<p>This doesn't really have anything to do with Selenium. You can use the <code>urlparse</code> module in the standard library.</p>
<pre><code>from urlparse import unquote      # python2
from urllib.parse import unquote  # python3
unquote('http%3A%2F%2Funiversities.ac%2Fshow_article.php%3Fid%3D61&amp;amp')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use urllib unquote as follows</p>
<pre><code>url='http%3A%2F%2Funiversities.ac%2Fshow_article.php%3Fid%3D61&amp;amp'
url=urllib.unquote(url).decode('utf8') 
</code></pre>
</div>
<span class="comment-copy">possible duplicate of <a href="http://stackoverflow.com/questions/8136788/decode-escaped-characters-in-url">Decode escaped characters in URL</a></span>
<span class="comment-copy">The ampersand <code>&amp;</code> (HTML-entity: <code>&amp;amp</code>) at the and of the URL does not seem to be unquoted. As a quick workaround you could pass the unquoted URL string to <a href="https://docs.python.org/3/library/html.html#html.unescape" rel="nofollow noreferrer"><code>html.unencode()</code></a> in oder to convert HTML-entities</span>
