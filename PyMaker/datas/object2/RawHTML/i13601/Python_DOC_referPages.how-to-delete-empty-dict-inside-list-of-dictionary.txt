<p>How can I remove empty dict from list of dict as,</p>
<pre><code>{
 "ages":[{"age":25,"job":"teacher"},
         {},{},
         {"age":35,"job":"clerk"}
        ]
}
</code></pre>
<p>I am beginner to python. 
Thanks in advance.</p>
</div>
<div class="post-text" itemprop="text">
<p>Try this</p>
<pre><code>In [50]: mydict = {
   ....:  "ages":[{"age":25,"job":"teacher"},
   ....:          {},{},
   ....:          {"age":35,"job":"clerk"}
   ....:         ]
   ....: }

In [51]: mydict = {"ages":[i for i in mydict["ages"] if i]}

In [52]: mydict
Out[52]: {'ages': [{'age': 25, 'job': 'teacher'}, {'age': 35, 'job': 'clerk'}]}
</code></pre>
<p><strong>OR</strong> simply use <code>filter</code></p>
<pre><code>&gt;&gt;&gt;mylist = [{'age': 25, 'job': 'teacher'}, {}, {}, {'age': 35, 'job': 'clerk'}]
&gt;&gt;&gt;filter(None, mylist)
[{'age': 25, 'job': 'teacher'}, {'age': 35, 'job': 'clerk'}]
</code></pre>
<p>So in your dict, apply it as</p>
<pre><code>{
 "ages":filter(None, [{"age":25,"job":"teacher"},
         {},{},
         {"age":35,"job":"clerk"}
        ])
}
</code></pre>

<div class="post-text" itemprop="text">
<p>This <code>while</code> loop will keep looping while there's a <code>{}</code> in the list and remove each one until there's none left.</p>
<pre><code>while {} in dictList:
    dictList.remove({})
</code></pre>

<div class="post-text" itemprop="text">
<p>There's a even simpler and more intuitive way than <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><em>filter</em></a>, and it works in Python 2 and Python 3:</p>
<p>You can do a <a href="https://docs.python.org/2/library/stdtypes.html#truth-value-testing" rel="nofollow noreferrer">"truth value testing"</a> on a dict to test if it's empty or not:</p>
<pre class="lang-py prettyprint-override"><code>&gt;&gt;&gt; foo = {}
&gt;&gt;&gt; if foo:
...   print(foo)
...
&gt;&gt;&gt;
&gt;&gt;&gt; bar = {'key':'value'}
&gt;&gt;&gt; if bar:
...    print(bar)
...
{'key':'value'} 
</code></pre>
<p>Therefore you can iterate over <em>mylist</em> and test for empty dicts with an if-statement:</p>
<pre class="lang-py prettyprint-override"><code>&gt;&gt;&gt; mylist = [{'age': 25, 'job': 'teacher'}, {}, {}, {'age': 35, 'job': 'clerk'}]
&gt;&gt;&gt; [i for i in mylist if i]
[{'age': 25, 'job': 'teacher'}, {'age': 35, 'job': 'clerk'}]
</code></pre>
