    def Print(self):
        print('my sub classes are : ')
        #print all childs

class Sub1(SuperClass,...):

        print('My parents are :')

<p>SuperClass has Print that print all classes that inherit from , and Sub  has Print method that print all his SuperClasses . how do that ?</p>

<div class="post-text" itemprop="text">
<p>Python classes have three attributes that help here:</p>
<ul>
<li><p><a href="https://docs.python.org/3/library/stdtypes.html#class.__subclasses__" rel="nofollow"><code>class.__subclasses__()</code></a>; a method that returns all subclasses of the class.</p></li>
<li><p><a href="https://docs.python.org/3/library/stdtypes.html#class.__bases__" rel="nofollow"><code>class.__bases__</code></a>, a tuple of the <em>direct</em> parent classes of the current class.</p></li>
<li><p><a href="https://docs.python.org/3/library/stdtypes.html#class.__mro__" rel="nofollow"><code>class.__mro__</code></a>, a tuple with <em>all</em> classes in the current class hierarchy. Find the current class object in that tuple and everything following is a parent class, directly or indirectly.</p></li>
</ul>
<p>Using these that gives:</p>
<pre><code>class SuperClass(object):
    def print(self):
        print('my sub classes are:', ', '.join([
            cls.__name__ for cls in type(self).__subclasses__()]))

class Sub1(SuperClass):
    def print(self):
        print('My direct parents are:', ', '.join([
            cls.__name__ for cls in type(self).__bases__]))
        all_parents = type(self).__mro__
        all_parents = all_parents[all_parents.index(Sub1) + 1:]
        print('All my parents are:', ', '.join([
            cls.__name__ for cls in all_parents]))
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; SuperClass().print()
my sub classes are: Sub1
&gt;&gt;&gt; Sub1().print()
My direct parents are: SuperClass
All my parents are: SuperClass, object
</code></pre>
