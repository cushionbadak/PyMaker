<div class="post-text" itemprop="text">
<p>In Python 2.x, where <a href="http://docs.python.org/2/reference/simple_stmts.html#the-print-statement"><code>print</code> is a statement</a>, add a trailing comma:</p>
<pre><code>print 'Hi!',  # No newline will be printed
</code></pre>
<p>In Python 3.x, where <a href="http://docs.python.org/3/library/functions.html#print"><code>print()</code> is an ordinary function</a> instead of a statement, pass an empty string for the <code>end</code> keyword parameter:</p>
<pre><code>print('Hi!', end='')
</code></pre>
<p>If you use the <code>from __future__ import print_function</code> in Python 2.x code for forwards compatibility, then you'll need to use the function version instead of the statement version.</p>
</div>

<pre><code>print 'Hi!',
print 'How are you?'
</code></pre>

<pre><code>print 'Hi!',
print 'How are you?'

<p>Output:</p>
<pre><code>Hi! How are you?
</code></pre>

<blockquote>
<p>A <code>'\n'</code> character is written at the end, <strong>unless the <code>print</code> statement ends with a comma</strong>.    </p>
</blockquote>

<div class="post-text" itemprop="text">
<p>In python 2.x you can use a comma after the <code>print</code> statement</p>
<pre><code>print 'Hi!',
print 'How are you?'
</code></pre>
<p>which will output:</p>
<pre><code>Hi! How are you?     
</code></pre>
<p>You can also import the <code>print</code> function from python 3.x and do </p>
<pre><code>from __future__ import print_function
print("Hi!", end='')
</code></pre>
<p>This puts an empty string as the ending character <strong>instead</strong> of a <code>\n</code></p>
</div>

<p>In Python3:</p>
<pre><code>print('text', end='')
print('text', end='')

<pre><code>print('text', 'text', 'text', sep = '')
