<div class="post-text" itemprop="text">
<p>you could use this function to encode email:</p>
<pre><code>&gt;&gt;&gt; def encodeemail(email):
       e = email.split("@")
       return "@".join(["x" * len(e[0]), e[1]])

&gt;&gt;&gt; encodeemail("harry@hogwarts.com")
xxxxx@hogwarts.com
</code></pre>
<p>or even</p>
<pre><code>&gt;&gt;&gt; def encodeemail(email):
        d = email.split(" ")
        for i, f in enumerate(d):
            e = f.split("@")
            if len(e) &gt; 1: d[i] = "@".join(["x" * len(e[0]), e[1]])
    return " ".join(d)

&gt;&gt;&gt; encodeemail("this is harry@hogwarts.com")
this is xxxxx@hogwarts.com
</code></pre>
<p>without enumerate:</p>
<pre><code>&gt;&gt;&gt; def encodeemail(email):
        d = email.split(" ")
        for i in range(len(d)):
            e = d[i].split("@")
            if len(e) &gt; 1: d[i] = "@".join(["x" * len(e[0]), e[1]])
    return " ".join(d)
</code></pre>

<p>You can use <code>re</code> module if the string does not contain email only.</p>
<pre><code>&gt;&gt;&gt; s
'my email id is harry@hogwards.com'
&gt;&gt;&gt; re.sub('(?&lt;=\s)\w+(?=@)', lambda y: 'x'*len(y.group()), s)
'my email id is xxxxx@hogwards.com'
</code></pre>

<div class="post-text" itemprop="text">
<p>I assume you want to check if a part of the string passes or fails the isalpha() test.  That isn't what your code is doing.</p>
<p>While there may be better ways to solve your problem, you can also make your code work.  Hopefully that is useful.</p>
<pre><code>while((indeces-1).isalpha()):
</code></pre>
<p>indeces is an integer, as is -1, so the result of what you are applying isalpha to is an int, as the error says.  </p>
<pre><code>(str(indeces-1)).isalpha()
</code></pre>
<p>This doesn't work either.  That is making a string out of the int, so str(2) results in "2", which isn't the test you want either.</p>
<p>If you want to check the character, just index into the string, like so:</p>
<pre><code>&gt;&gt;&gt; for i in range(len(s)):
...    if s[i].isalpha():
...         print 'x'
...    else:
...         print s[i]
</code></pre>
