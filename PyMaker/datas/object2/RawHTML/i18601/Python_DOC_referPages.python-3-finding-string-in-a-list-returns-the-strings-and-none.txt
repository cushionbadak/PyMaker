<div class="post-text" itemprop="text">
<p>You're not <code>return</code>ing anything, so the function returns <code>None</code> by default. Also, you can do this in a more Pythonic way:</p>
<pre><code>def find(string_list, search):
    return [i for i in string_list if search in i]
</code></pre>
<p>This is called a <em>list comprehension</em>, and you can read more about them <a href="http://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">here</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>This is the solution</p>
<pre><code>def find(string_list, search):
    new_list = []
    for i in string_list:
        if search in i:
            new_list.append(i)
    return new_list

print(find(['she', 'sells', 'sea', 'shells', 'on', 'the', 'sea-shore'], 'he'))
</code></pre>
