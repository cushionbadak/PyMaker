<div class="post-text" itemprop="text">
<p>The module defines an abstract class interface for use with the <a href="http://docs.python.org/2/library/htmllib.html" rel="nofollow"><code>htmllib.HTMLParser</code></a>. That module was removed from Python 3; the fact that the <code>formatter</code> module is still there is likely an oversight.</p>
<p>Use it to turn HTML tree events into formatted documents. For example, if during parsing a HTML document a <code>&lt;H1&gt;</code> header is encountered, the <a href="http://docs.python.org/2/library/formatter.html#formatter.formatter.end_paragraph" rel="nofollow"><code>formatter.end_paragraph()</code> method</a> will be called, followed by a <a href="http://docs.python.org/2/library/formatter.html#formatter.formatter.push_font" rel="nofollow"><code>formatter.push_font()</code> method</a> call.</p>
<p>Use this module if you are transforming HTML documents; like building a HTML-to-plaintext converter, or a web browser.</p>
