<div class="post-text" itemprop="text">
<p>It should be <code>elif</code> not <code>else</code>:</p>
<pre><code>elif clothes_total1 &gt; 200:
</code></pre>
<p>From <a href="http://docs.python.org/3/reference/compound_stmts.html#" rel="nofollow">docs</a>:</p>
<pre><code>if_stmt ::=  "if" expression ":" suite
             ( "elif" expression ":" suite )*
             ["else" ":" suite]
</code></pre>
<p>And instead of using <code>tot1 + tot2 + tot3 + tot4+ tot5</code>, I think you should use a list.</p>
<p><strong>Example:</strong></p>
<pre><code>tot = [12,56,....] #some values
</code></pre>
<p>and then use <code>sum</code>:</p>
<pre><code>clothes_total = sum(tot)
</code></pre>
<p>If the list has more than 5 items and you want sum of just first 5 items then use slicing:</p>
<pre><code>clothes_total = sum(tot[:5])
</code></pre>

<pre><code>elif clothes_total1 &gt; 200:
