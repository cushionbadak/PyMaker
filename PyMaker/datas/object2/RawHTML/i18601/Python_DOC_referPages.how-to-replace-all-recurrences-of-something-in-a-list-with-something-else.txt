<p>Sorry if the title is a bit vague, but I would like to go through a list and replace all occurrences of <code>'\\'</code> with <code>'/'</code>. Here's an example:</p>
<pre><code>list = ['C:/dir\\file.txt', 'C:/dir\\example.zip', 'C:/dir\\example2.zip']
</code></pre>
<p>I'd like to filter out the <code>'\\'</code> in all of the individual paths with <code>'/'</code> for readability reasons.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/stdtypes#str.replace" rel="nofollow">str.replace</a>:</p>
<pre><code>&gt;&gt;&gt; 'C:/dir\\file.txt'.replace('\\', '/')
'C:/dir/file.txt'
</code></pre>
<p>Applying <code>str.replace</code> to all paths, you get replaced paths.</p>
<pre><code>&gt;&gt;&gt; paths = list = ['C:/dir\\file.txt', 'C:/dir\\example.zip', 'C:/dir\\example2.zip']
&gt;&gt;&gt; paths = [path.replace('\\', '/') for path in paths]
&gt;&gt;&gt; paths
['C:/dir/file.txt', 'C:/dir/example.zip', 'C:/dir/example2.zip']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You should be handling paths using <code>os.path</code> rather than by hand, but since you must, here you go:</p>
<pre><code>[re.sub("\\\\",'/',x) for x in list]
</code></pre>
