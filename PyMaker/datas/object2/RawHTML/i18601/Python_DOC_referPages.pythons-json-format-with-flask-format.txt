<div class="post-text" itemprop="text">
<p>If you are using flask, why not use jsonify</p>
<pre><code>from flask import jsonify

@app.route('/do')
def do_whatever():
    return jsonify(name=test)
</code></pre>
<p>This will send a JSON response like this to the browser:</p>
<pre><code>{
    "name": "test",
}
</code></pre>
