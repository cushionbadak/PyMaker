</code></pre>
<p>If I have a list of 10 variables including strings and variables of that kind.. how can I construct an if statement to check whether an object in the list "mylist" is of <code>&lt;type 'my.object.kind'&gt;</code>?</p>

<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/functions.html#isinstance" rel="noreferrer"><code>isinstance</code></a>, this will return true even if it is an instance of the subclass:</p>
<pre><code>if isinstance(x, my.object.kind)
</code></pre>
<p>Or:</p>
<pre><code>type(x) == my.object.kind #3.x
</code></pre>
<p>If you want to test all in the list:</p>
<pre><code>if any(isinstance(x, my.object.kind) for x in alist)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>if any(map(lambda x: isinstance(x, my.object.kind), my_list_of_objects)):
    print "Found one!"
</code></pre>
