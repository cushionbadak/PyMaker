<div class="post-text" itemprop="text">
<p>I've got a list which I want to print in random order. Here is the code I've written so far:</p>
<pre><code>import random
words=["python","java","constant","immutable"]
for i in words:
    print(i, end=" ")
input("") #stops window closing
</code></pre>
<p>I've tried a variety of things to print them out randomly, such as making a variable which selects only one of them randomly and then deleting the randomly. I would then repeat this step until they are all deleted and within another variable. Then I would put the variables in a list then print them out. This kept on generating errors though. Is there another way this can be done?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/random.html#random.shuffle" rel="nofollow"><code>random.shuffle()</code></a> to shuffle a list, in-place:</p>
<pre><code>import random

words = ["python", "java", "constant", "immutable"]
random.shuffle(words)
print(*words)

input('')
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; import random
&gt;&gt;&gt; words = ["python", "java", "constant", "immutable"]
&gt;&gt;&gt; random.shuffle(words)
&gt;&gt;&gt; words
['python', 'java', 'constant', 'immutable']
</code></pre>
<p>If you wanted to preserve <code>words</code> (maintain the order), you can use <code>sorted()</code> with a random key to return a new randomized list:</p>
<pre><code>words = ["python", "java", "constant", "immutable"]
print(*sorted(words, key=lambda k: random.random()))
</code></pre>
<p>This leaves <code>words</code> unaltered:</p>
<pre><code>&gt;&gt;&gt; words = ["python", "java", "constant", "immutable"]
&gt;&gt;&gt; sorted(words, key=lambda k: random.random())
['immutable', 'java', 'constant', 'python']
&gt;&gt;&gt; words
['python', 'java', 'constant', 'immutable']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>import random
words2 = words[::]
random.shuffle(words2)
for w in words2:
    print(w, end=" ")
</code></pre>
<p>Notice that I copied the original list first, in case you want to preserve it. If you don't mind shuffling it, this should do the trick:</p>
<pre><code>import random
random.shuffle(words)
for w in words:
    print(w, end=" ")
</code></pre>
