<div class="post-text" itemprop="text">
<p>You can use a dictionary directly:</p>
<pre><code>dictionary = {}
file_data.readline()   # skip the first line         
for key in file_data:
    key = key.replace('"', '').strip()
    k = key.split(",")
    dictionary[k[0]] = k[1]
</code></pre>

<div class="post-text" itemprop="text">
<p>Use the <a href="http://docs.python.org/3/library/csv.html" rel="nofollow"><code>csv</code> module</a> to handle your data; it'll remove the quoting and handle the splitting:</p>
<pre><code>results = {}

with open('hk.txt', 'r', newline='') as txt:
    reader = csv.reader(txt)
    next(reader, None)  # skip the header line
    for row in reader:
        results[row[0]] = row[1]
</code></pre>
<p>For your sample input, this produces:</p>
<pre><code>{'4': 'korilakkuma', '1': 'hello kitty', '3': 'keroppi', '2': 'rilakkuma'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/2/library/csv.html#csv.DictReader" rel="nofollow">csv DictReader</a>:</p>
<pre><code>import csv

result={}
with open('/tmp/test.csv', 'r', newline='') as f:
    for d in csv.DictReader(f):
        result[d['ID#']]=d['name']

print(result)
# {'1': 'hello kitty', '3': 'keroppi', '2': 'rilakkuma', '4': 'korilakkuma'}
</code></pre>
