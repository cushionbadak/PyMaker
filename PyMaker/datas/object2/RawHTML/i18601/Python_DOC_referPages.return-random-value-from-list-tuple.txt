<div class="post-text" itemprop="text">
<p>How can I return a random key value from this list tuple? I'm only concerned with returning 'r', 'p', or 's' from moves.</p>
<pre><code># Snippet

moves = [('r', "rock"), ('p', "paper"), ('s', "scissors")]

view_all(moves):
    print "Player moves:"
    for move in moves:
        print " =&gt; ".join((move[0], move[1]))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/3/library/random.html#random.choice" rel="nofollow"><code>random.choice</code></a>.</p>
<pre><code>&gt;&gt;&gt; import random
&gt;&gt;&gt; moves = [('r', "rock"), ('p', "paper"), ('s', "scissors")]
&gt;&gt;&gt; random.choice(moves)
('s', 'scissors')
</code></pre>
<p>If only the first value of the tuple is wanted:</p>
<pre><code>random.choice(moves)[0]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/2/library/random.html#random.choice" rel="nofollow"><code>random.choice</code></a>.</p>
<pre><code>&gt;&gt;&gt; import random
&gt;&gt;&gt; moves = [('r', "rock"), ('p', "paper"), ('s', "scissors")]
&gt;&gt;&gt; print random.choice(moves)[0] 
's'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>import random
moves = [('r', "rock"), ('p', "paper"), ('s', "scissors")]
move_keys = [ x[0] for x in moves ]

print random.choice(move_keys)
</code></pre>
</div>
