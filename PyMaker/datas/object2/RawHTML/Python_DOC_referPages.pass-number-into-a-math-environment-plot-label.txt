<div class="post-text" itemprop="text">
<p>You can pass the parameter into the math environment of the label like this:</p>
<pre><code>def function(param):
    x = np.linspace(0, param, 10)
    plt.plot(x, x, label="some label that includes the parameter like " + r"$A_{%d}$"%param)
    plt.legend()
    plt.show()
</code></pre>
<p>For instance, calling your function as <code>function(100)</code> would produce the following result:
<a href="https://i.stack.imgur.com/Nh7wD.png" rel="nofollow noreferrer"><img alt="enter image description here" src="https://i.stack.imgur.com/Nh7wD.png"/></a></p>
<p>BTW, <code>function</code> is a built-in name. It is better not to use python built-in names as variables' or functions' names.   </p>
