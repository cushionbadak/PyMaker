<p>You need to call your <code>__main__</code> function for the program to get executed. So below everything put:</p>
<pre><code>__main__()
</code></pre>
<p>Your function probably shouldn't be named <code>__main__</code> though since the double underscore functions/variables have special meanings, better name would be <code>main</code>.</p>
