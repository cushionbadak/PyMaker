<p><code>os.system</code> will return the value of the process, <code>0</code> if it had no error. You need to use <a href="https://docs.python.org/3/library/subprocess.html#subprocess.check_output" rel="nofollow noreferrer"><code>subprocess.check_output</code></a> to get the <code>stdout</code> from your command.</p>
<pre><code>import subprocess

output = subprocess.check_output(["ping -c 10 " + "www.google.com" + " | tail -1| awk '{print $4}' | cut -d '/' -f 2"])
</code></pre>
