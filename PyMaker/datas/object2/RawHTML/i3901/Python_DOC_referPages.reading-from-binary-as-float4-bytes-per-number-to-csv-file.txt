<div class="post-text" itemprop="text">
<p>Its just a string form of a <code>tuple</code>.</p>
<pre><code>unpack(...)
</code></pre>
<p>Gives back a <code>tuple</code>.This is stated in the <a href="https://docs.python.org/3/library/struct.html#struct.unpack" rel="nofollow noreferrer">docs</a>:</p>
<blockquote>
<p>... The result is a tuple even if it contains exactly one item. </p>
</blockquote>
<p>If you want the first element:</p>
<pre><code>first = unpack('f', byte)[0]
</code></pre>
<p>In your code, use:</p>
<pre><code>output_file.write(str(unpack('f', byte)[0]))
</code></pre>
<p>Tip: Use the <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer">csv module</a>.</p>
