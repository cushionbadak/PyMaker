<p>A 2D array in python is just a list of lists. List can be defined with some pretty simple syntax -- <code>a = [1,2,3,4]</code> defines <code>a</code> as the list <code>[1,2,3,4]</code>. </p>
<p>Your check function currently just prints the lists you make. If you want to keep the lists you make to use later, you need to <code>return</code> them, so that <code>a = check(temp)</code> will assign the value of <code>check(temp)</code> to the variable <code>a</code>.</p>
<p>If your check function returns the lists, you can create a list of your results with <code>results = [check(temp), check(rainfall), check(humidity), check(population)]</code>.</p>
