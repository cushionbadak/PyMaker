<p>Try using <code>str.format</code>.</p>
<p><strong>Ex:</strong></p>
<pre><code>f=open("Output.txt", "a")
f.write("Number Plate: {0}".format(np))
f.write("Valid: {0}".format(valid ))
f.write("Speed: {0}".format(speed ))
f.write("Ticket: {0}".format(ticket ))
f.write("Ticket Price: {0}".format(ticketprice ))
f.write("\n")
f.close()
</code></pre>
<p><strong>Note</strong>: <code>f.write</code> takes only a single argument, You are trying to pass 3<code>("Number Plate:", np ,"\n")</code></p>

<div class="post-text" itemprop="text">
<p>You can try like this:</p>
<pre><code>with open("Output.txt", "a") as f:
    f.write("Number Plate:" + str(np) + "\n")
    f.write("Valid:" + str(valid) + "\n")
    f.write("Speed:" + str(speed) + "\n")
    f.write("Ticket:" + str(ticket) + "\n")
    f.write("Ticket Price:" + str(ticketprice) + "\n")
    f.write("\n")
</code></pre>
<p>Explanation:</p>
<p>If you use <code>with open....</code> there is no need to explicitly specify the <code>f.close()</code>. And also in <code>f.write()</code> using string concatenation <code>+</code> you can get the required one.</p>
</div>
