<div class="post-text" itemprop="text">
<p>How can I multiply this list with python:</p>
<pre><code>A = [ [0.45, 0.89, 0.91],
      [0.5, 0.78, 0.55],
      [0.134, 0.571, 0.142] ]
</code></pre>
<p>How can I multiply every column, for the example 0.45*0.5*0.134 = 0.03015; 0.89*0.78*0.571 = 0.3961; 0,91*0.55*0.142 = 0.071071</p>
<pre><code> [0.03015,0.3961,0.071071]
</code></pre>
<p>How can i do that with python?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do this in pure Python:</p>
<pre><code>from operator import mul
from functools import reduce  # no need for this in Python 2.x

res = [reduce(mul, i) for i in zip(*A)]
</code></pre>
<p>Or you can use <code>numpy</code>:</p>
<pre><code>import numpy as np

res = np.prod(A, axis=0)

array([ 0.03015  ,  0.3963882,  0.071071 ])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <strong><em><a href="https://docs.python.org/3.3/library/functions.html#zip" rel="nofollow noreferrer">zip</a></em></strong>,</p>
<pre><code>In [5]: A = [[0.45, 0.89, 0.91], [0.5, 0.78, 0.55], [0.134, 0.571, 0.142]]

In [6]: [a*b*c for a,b,c in zip(*A)]
Out[6]: [0.030150000000000003, 0.39638819999999997, 0.071071]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use numpy:</p>
<pre><code>import numpy
A = numpy.array([[0.45, 0.89, 0.91], [0.5, 0.78, 0.55], [0.134, 0.571, 0.142]])
result = numpy.product(A, axis=0)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> and list unpacking:</p>
<pre><code>&gt;&gt;&gt; [x*y*z for x, y, z in zip(*A)]
</code></pre>
