<p>I'm trying to write a register program using python 3.6.2 and i'm getting the following error: <code>write() argument must be str, not list</code></p>
<p>Here is the code: </p>
<pre><code>database = open('database.csv', 'a')
linetowrite = ['username', 'password', 'forename', 'surname', 'dob', 'artist', 'genre']
database.write(linetowrite)
database.close()
</code></pre>

<div class="post-text" itemprop="text">
<p>You're calling <code>write</code> with a list, the error message is pretty straightforward. To make this work simply (but not necessarily correctly), you could convert the list to a string like so:</p>
<pre><code>database = open('database.csv', 'a')
linetowrite = ['username', 'password', 'forename', 'surname', 'dob', 'artist', 'genre']
database.write(",".join(linetowrite))
database.close()
</code></pre>
<p>Better would be to use a proper CSV library, e.g.: <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer">https://docs.python.org/3/library/csv.html</a> . Otherwise you're likely to end up with escaping errors (what if one of your fields contains a comma?).</p>
