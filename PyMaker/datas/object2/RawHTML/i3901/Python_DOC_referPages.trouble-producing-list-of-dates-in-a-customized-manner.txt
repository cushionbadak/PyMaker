<p>I've written a script to produce list of dates. The script I've tried with can do it. However, the way I need the list of dates is very different from what I'm getting at this moment.</p>
<p>The script I've tried with:</p>
<pre><code>import datetime

start = datetime.datetime(2017,1,1)
end = datetime.datetime(2017,6,20)
daterange = [start + datetime.timedelta(days=x) for x in range(0, (end-start).days)]
for idate in daterange:
    print(idate)
</code></pre>
<p>Current results are like:</p>
<pre><code>2017-01-01 00:00:00
2017-01-02 00:00:00
2017-01-03 00:00:00
2017-01-04 00:00:00
2017-01-05 00:00:00
2017-01-06 00:00:00
</code></pre>
<p>The way I expect to get the dates like:</p>
<pre><code>01-01-2017 
01-02-2017  
01-03-2017  
01-04-2017  
01-05-2017  
01-06-2017
</code></pre>
<p>How can I modify my script to achieve the same? Thanks in advance.</p>

<div class="post-text" itemprop="text">
<p>You can try using <code>strftime</code> in <code>print</code>, for more details you can look into <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">documentation</a>:</p>
<pre><code>print(idate.strftime("%d-%m-%Y"))
</code></pre>
<p>From <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">documentation</a>:</p>
<blockquote>
<p>%d : Day of the month as a zero-padded decimal number.</p>
<p>%m : Month as a zero-padded decimal number.   </p>
<p>%Y: Year with century as a decimal number.</p>
</blockquote>
<p>For testing the code above:</p>
<pre><code>import datetime

start = datetime.datetime(2017,1,1)
end = datetime.datetime(2017,6,20)
daterange = [start + datetime.timedelta(days=x) for x in range(0, (end-start).days)]
for idate in daterange:
    print(idate.strftime("%d-%m-%Y"))
</code></pre>
<p>Result:</p>
<pre><code>01-01-2017
02-01-2017
03-01-2017
04-01-2017
05-01-2017
06-01-2017
07-01-2017
08-01-2017
09-01-2017
.
.
.
</code></pre>
