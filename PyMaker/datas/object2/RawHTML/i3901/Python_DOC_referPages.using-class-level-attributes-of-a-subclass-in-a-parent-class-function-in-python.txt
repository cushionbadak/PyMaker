<p>How can I change the respective class level attributes using the function of the base class without overloading the function?</p>
<pre><code>class A:
    a = 0

    def addOne(self):
        print(A.a)      # prints 0
        A.a = A.a + 1
        print(A.a)      # it stores the value of a over all the child classes

class B(A):
    a = 0
    print(A.a)      # prints 0
    print(B.a)      # prints 0
    self.addOne()
    print(A.a)      # prints 1
    print(B.a)      # prints 0

Class C(A):
    a = 0
    print(A.a)      # prints 1
    print(C.a)      # prints 0
    self.addOne()
    print(A.a)      # prints 2
    print(C.a)      # prints 0
</code></pre>
<p>I want <code>B.a = 1</code> and <code>C.a = 1</code> after <code>self.addOne()</code>. </p>
<p>I don't want to use instance attributes as I will have to overload the addOne function for all the child classes.</p>
</div>
<div class="post-text" itemprop="text">
<p><strong>Option 1:</strong> Use the <a href="https://docs.python.org/3/library/functions.html#type" rel="nofollow noreferrer"><code>type</code></a> function to get the class of <code>self</code>.</p>
<pre><code>def addOne(self):
    cls = type(self)
    cls.a += 1
</code></pre>
<p><strong>Option 2:</strong> Turn <code>addOne</code> into a <a href="https://docs.python.org/3/library/functions.html#classmethod" rel="nofollow noreferrer"><code>classmethod</code></a>:</p>
<pre><code>@classmethod
def addOne(cls):
    cls.a += 1
</code></pre>
