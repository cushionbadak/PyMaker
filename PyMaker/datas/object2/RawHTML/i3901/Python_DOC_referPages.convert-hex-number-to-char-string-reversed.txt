<div class="post-text" itemprop="text">
<p>I have this variable</p>
<pre><code>x = 0x61626364
</code></pre>
<p>I want the string <code>"dcba"</code>, converting the hex number in char, and then reverse the string.</p>
<p>How can I do that in python?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can try this:</p>
<pre><code>x = 0x61626364
print(x.to_bytes(4, 'little').decode('utf-8'))
</code></pre>
<p>Explanation:</p>
<p>Using <code>to_bytes()</code> we will get the bytecode and to get the string <code>dcba</code> use the decode function.</p>
<p>Output:</p>
<pre><code>dcba
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Interpret the integer as bytes in little-endian order, using the <a href="https://docs.python.org/3/library/stdtypes.html#int.to_bytes" rel="nofollow noreferrer"><code>int.to_bytes()</code> method</a>:</p>
<pre><code>&gt;&gt;&gt; x = 0x61626364
&gt;&gt;&gt; x.to_bytes(4, 'little')
b'dcba'
</code></pre>
<p>You do need to know the output length for this.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>import math

a = [chr(0xFF&amp;(x&gt;&gt;(8*i))) for i in range(math.ceil(math.log(x, 2)/8))]

b = ""

for i in range(len(a)): b += a[i]

print(b)
</code></pre>
</div>

<p>Enjoy!</p>
<pre><code>def convert(h):
    result = ''
    while h&gt;0:
        result+=chr(h%256)
        h//=256
    return result


&gt;&gt;&gt; convert(0x61626364)
'dcba'
&gt;&gt;&gt; convert(0x21646c726f57206f6c6c6548)
'Hello World!'
</code></pre>
