<div class="post-text" itemprop="text">
<p>Suppose I have a named list as follows:</p>
<pre><code>myListOfPeople = [{'ID': 0, 'Name': 'Mary', 'Age': 25}, {'ID': 1, 'Name': 'John', 'Age': 28}]
</code></pre>
<p>I want to select the element (not only the field) where an specific field meets certain criteria, e.g., the element with the minimum 'Age'. Something like:</p>
<pre><code>youngerPerson = [person for person in myListOfPeople if person = ***person with minimum age***]
</code></pre>
<p>And will get as answer:</p>
<pre><code>&gt;&gt;youngerPerson: {'ID': 0, 'Name': Mary, 'Age': 25}
</code></pre>
<p>How can I do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>key</code> parameter of <code>min</code>:</p>
<pre><code>&gt;&gt;&gt; myListOfPeople = [{'ID': 0, 'Name': 'Mary', 'Age': 25}, {'ID': 1, 'Name': 'John', 'Age': 28}]
&gt;&gt;&gt; 
&gt;&gt;&gt; min(myListOfPeople, key=lambda x: x["Age"])
{'ID': 0, 'Name': 'Mary', 'Age': 25}
&gt;&gt;&gt; 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/operator.html" rel="nofollow noreferrer"><code>itemgetter</code></a> :</p>
<pre><code>from operator import itemgetter
myListOfPeople = [{'ID': 0, 'Name': 'Mary', 'Age': 25}, {'ID': 1, 'Name': 'John', 'Age': 28}]
sorted(myListOfPeople, key=itemgetter('Age'))[0]
# {'ID': 0, 'Name': 'Mary', 'Age': 25}
</code></pre>
