<div class="post-text" itemprop="text">
<p>Your code had two mistakes:</p>
<ul>
<li><p>First in the <code>except</code> part</p></li>
<li><p>Second is that you declared <code>is_json</code> at the bottom</p></li>
</ul>
<p>If you declare at bottom you may get <code>NameError: name 'is_json' is not defined</code> error.</p>
<pre><code>    import json

    def is_json(str):
        try:
            json.loads(str)
        except ValueError as e:
            return False
        return True

    if __name__ == '__main__':
        is_json('test')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Have a look at the below 2 points.</p>
<p>(1) Python3 uses different syntax for <strong>try-except</strong> statement for exception handling.</p>
<blockquote>
<p><strong>Replace</strong> <code>except ValueError, e:</code> with <code>except ValueError as e:</code>.</p>
<p><a href="https://docs.python.org/3/tutorial/errors.html" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/errors.html</a></p>
</blockquote>
<p>(2) <strong>Define</strong> your function before if statement. Function definition should be available before the point of usage (call).</p>
<pre><code>import json

def is_json(str):
    try:
        json.loads(str)
    except ValueError as e:
        return False
    return True 

if __name__ == '__main__':
    print(is_json('test')); # False
</code></pre>
