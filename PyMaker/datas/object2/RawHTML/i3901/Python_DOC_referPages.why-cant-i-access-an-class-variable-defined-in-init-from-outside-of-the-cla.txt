<div class="post-text" itemprop="text">
<p>You had done a mistake while creating objects, please find below differences:</p>
<pre><code>class test:
def __init__(self):
    self.testy=0
def __str__(self):
    return self.testy

a = test()
b = test()
a.testy
       output: 0
</code></pre>
<p>What you have done:</p>
<pre><code>c = test
d = test
c.testy
Traceback (most recent call last):
  File "&lt;input&gt;", line 1, in &lt;module&gt;
AttributeError: type object 'test' has no attribute 'testy'
</code></pre>
<p>Explanation:</p>
<p>when you are creating <code>objects</code> for a class use <code>object = class_name()</code></p>
<p>**<a href="https://docs.python.org/3/tutorial/classes.html#class-objects" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/classes.html#class-objects</a></p>
