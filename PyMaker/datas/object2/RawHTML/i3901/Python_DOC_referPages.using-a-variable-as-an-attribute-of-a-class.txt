<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer"><code>getattr</code></a> function:</p>
<pre><code>if event.key == getattr(pygame, left_key):
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could assign <code>K_LEFT</code> to <code>left_key</code>:</p>
<pre><code>left_key = K_LEFT
</code></pre>
<p>instead of <code>left_key = 'K_LEFT'</code></p>
</div>
