<div class="post-text" itemprop="text">
<p>I am trying to create a helper function that converts the values of an Enum to list. </p>
<p>It works if I do this:</p>
<pre><code>from enum import Enum

class TestEnum(Enum):
  item1 = 'Def1',
  item2 = 'Def2',
  item3 = 'Def3',

@staticmethod
def createListFromEnum(enum):
  return [elem.value[0] for elem in enum]

print(createListFromEnum(TestEnum))
</code></pre>
<p>Which outputs:</p>
<pre><code>['Def1','Def2','Def3']
</code></pre>
<p>But if I don't put the trailing comma after the last item, I get:</p>
<pre><code>['Def1','Def2','D']
</code></pre>
<p>Because the final entry is not registered as a tuple for some reason. I have looked at the other responses for converting an enum to value list, but none of them seem to address this as a problem. </p>
</div>
<div class="post-text" itemprop="text">
<p>You don't need the commas in the enum.</p>
<p>I think this is what you want:</p>
<pre><code>from enum import Enum

class TestEnum(Enum):
    item1 = 'Def1'
    item2 = 'Def2'
    item3 = 'Def3'

def createListFromEnum(enum):
    return [elem.value for elem in enum]

print(createListFromEnum(TestEnum))
</code></pre>
<p>Output:</p>
<pre><code>['Def1', 'Def2', 'Def3']
</code></pre>
<p>See the documentation: <a href="https://docs.python.org/3/library/enum.html#creating-an-enum" rel="nofollow noreferrer">https://docs.python.org/3/library/enum.html#creating-an-enum</a></p>
</div>
<div class="post-text" itemprop="text">
<p>In the first case you address the first element in a <code>tuple</code>. In the second case you don't have a <code>tuple</code> and address the first character of a <code>string</code>.</p>
<p>Look at this simple example:</p>
<pre><code>&gt;&gt;&gt; print(('python',)[0])
python
&gt;&gt;&gt; print('python'[0])
p
</code></pre>
