<p>If have a list of different types of strings and from that I want to combine all the alphabetic strings in the list into one single value. </p>
<p>For example: </p>
<pre><code>    ['000000001', 'Aaron', 'Appindangoye', '26', '183', '84.8']
</code></pre>
<p>Here, I want to get <code>Aaron Appindangoye</code> together. </p>
</div>
<div class="post-text" itemprop="text">
<p>You can access to the 2 names by index:</p>
<pre><code>items = ['000000001', 'Aaron', 'Appindangoye', '26', '183', '84.8']

name = ' '.join(items[1:3])
print(name)
</code></pre>
