<div class="post-text" itemprop="text">
<p>You can use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>    def code(self):
        return [self.get_map(each) for each in self.event_log_row]

[print(x) for x in obj.code()]
</code></pre>

<div class="post-text" itemprop="text">
<p>If you get a result of </p>
<pre><code>result  = [["One","Two"],["Three"], ["Four","Five","Six"]]        # obj.code() result
</code></pre>
<p>from your <code>setMap.code()</code> you can print this using </p>
<pre><code>result  = [["One","Two"],["Three"], ["Four","Five","Six"]]

print(*result, sep ="\n") # unpack outer list for printing 

print( *(b for a in result for b in a), sep ="\n") # flatten inner lists, unpack
</code></pre>
<p>resulting in </p>
<pre><code>['One', 'Two']
['Three']
['Four', 'Five', 'Six']

One
Two
Three
Four
Five
Six
</code></pre>
<p>You specify a print-seperator of newline to get them into different lines, no string-join neeeded. </p>
<p>Details on sep: <a href="https://stackoverflow.com/questions/22116482/what-does-print-sep-t-mean">What does print(... sep='', '\t' ) mean?</a> or <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer">Doku print()</a></p>
