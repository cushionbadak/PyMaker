<p>You can try this :</p>
<pre><code>from collections import deque
deq = deque([1, 2, 3, 4])

del deq[1]
print(deq)
</code></pre>
<p>Output:</p>
<pre><code>deque([1, 3, 4])

<span class="comment-copy">@Denly: Not necessary. <code>deque</code>s are documented to be <a href="https://docs.python.org/3/library/collections.abc.html#collections.abc.MutableSequence" rel="nofollow noreferrer">mutable sequences</a>, and <i>all</i> mutable sequences support this behavior (via <code>__delitem__</code>). <code>deque</code>s are also explicitly documented to support subscript operations (of which <code>del</code> of a subscript is just one more use case). Same thing applies to mutable mappings (e.g. <code>dict</code> and <code>dict</code>-like things).</span>
