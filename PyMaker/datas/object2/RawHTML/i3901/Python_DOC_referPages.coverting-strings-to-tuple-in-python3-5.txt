<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow noreferrer">itertools.zip_longest</a>
 wich also works for zipping unevenly lengthy list by supplying a default value (of None if not otherwise specified) to the shorter lists:</p>
<p>You split data at spaces and feed a sublists to zip : once starting at 0 and once starting at 1, both using every other (2nd) element only:</p>
<pre><code>data = ['I went to work but got delayed at other work and got stuck in a traffic and I went to drink some coffee but got no money and asked for money']

import itertools
d1 = data[0].split() 

# use 2 partial lists, using every 2nd word, once staring at 0, once at 1
# you can change   fillvalue=None   to some other value or remove it - None is the default.
output = list(itertools.zip_longest(d1[::2],d1[1::2], fillvalue = None)) 

print(output)
</code></pre>
<p>Output:</p>
<pre><code>[('I', 'went'), ('to', 'work'), ('but', 'got'), ('delayed', 'at'), ('other', 'work'),

 ('asked', 'for'), ('money', None)]
</code></pre>
