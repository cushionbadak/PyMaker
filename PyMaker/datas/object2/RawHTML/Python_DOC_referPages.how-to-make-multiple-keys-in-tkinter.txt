<div class="post-text" itemprop="text">
<p>You have to bind to the widget that the Buttons are in. For the most effectiveness, bind to root. </p>
<pre><code>btnUp = tkinter.Button(master=None, text="Up", command=up)
root.bind("w", up)
btnUp.pack(side=tkinter.TOP, anchor=tkinter.W)
</code></pre>
<p>If you don't have access to root in that scope you can add this to the top: </p>
<pre><code>root = tkinter._default_root
</code></pre>
<hr/>
<p>Remember that the Button command callback and the bind callback have different signatures, so you need to define the functions with an optional event argument like this: </p>
<pre><code>def up(event=None):
    # code
</code></pre>
