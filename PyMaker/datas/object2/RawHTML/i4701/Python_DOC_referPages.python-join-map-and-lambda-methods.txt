<div class="post-text" itemprop="text">
<p>This is a simplified version of your code, which you should be able to follow:</p>
<pre><code>from math import ceil, sqrt

s = 'hello'

y = ceil(sqrt(len(s)))
# 3

res = ' '.join(map(lambda x: s[x::y], range(y)))
# 'hl eo l'
</code></pre>
<p><strong>Essential points</strong></p>
<ul>
<li><code>y</code> is the rounded-up square root of the length of <code>s</code>, in this case sqrt(5) is rounded up to 3.</li>
<li><code>lambda</code> is an anonymous function which maps each value in <code>range(y)</code>, i.e. <code>0, 1, 2</code> is mapped to <code>s[x::y]</code>, i.e. return every <code>y</code>th element of sequence beginning from index <code>x</code>. See also <a href="https://stackoverflow.com/questions/509211/understanding-pythons-slice-notation">Understanding Python's slice notation</a>. <code>x</code> is arbitrary notation for a member of <code>range(y)</code>.</li>
<li>Join all resulting values with a space to form a single string.</li>
<li>In your original code <code>{0}</code> and <code>str.format</code> are used to incorporate <code>y</code> in your string in one line. In this case, I consider it convoluted and bad practice.</li>
</ul>
</div>
<div class="post-text" itemprop="text">
<p>This line is using the <a href="https://docs.python.org/3.4/library/functions.html#format" rel="nofollow noreferrer"><code>format()</code></a> function on a string.
Thus, the <code>{0}</code> will be replaced by the first element in the format function which is <code>ceil(sqrt(len(s))))</code></p>
</div>
<div class="post-text" itemprop="text">
<p>When using the str.format() method (<a href="https://docs.python.org/3/tutorial/inputoutput.html" rel="nofollow noreferrer">see here</a>), the brackets and characters in them will be replaced with the objects passed in the method. The "0" indicates replacement with the first character passed in the method.</p>
</div>
<div class="post-text" itemprop="text">
<p>From the <a href="https://docs.python.org/3.4/library/string.html#format-string-syntax" rel="nofollow noreferrer">documetnation</a> you can see what <code>format()</code> does.</p>
<p>In your example:</p>
<pre><code>exec("print(' '.join(map(lambda x: s[x::{0}], range({0}))))".format(ceil(sqrt(len(s)))))
</code></pre>
<p>You are inserting the result of <code>ceil(sqrt(len(s)))</code> in two places, namely:</p>
<pre><code>lambda x: s[x::ceil(sqrt(len(s)))]
</code></pre>
<p>and</p>
<pre><code>range(ceil(sqrt(len(s))))
</code></pre>
<p>Now, <code>exec</code> could be avoided here with simple variable:</p>
<pre><code>def encryption(s):
    var = ceil(sqrt(len(s)))
    print(' '.join(map(lambda x: s[x::var], range(var))))
</code></pre>
