<div class="post-text" itemprop="text">
<p>Regular expressions alone cannot solve the problem, use a <a href="https://docs.python.org/3/library/stdtypes.html#set-types-set-frozenset" rel="nofollow noreferrer"><code>set</code></a> to keep track of what you have already seen:</p>
<pre><code>pattern=re.compile(r'(1[6][2]\.[0-9][0]+)')
seen = set()
if pattern.search(line):
    if line not in seen:
        seen.add(line)
        print line
</code></pre>
