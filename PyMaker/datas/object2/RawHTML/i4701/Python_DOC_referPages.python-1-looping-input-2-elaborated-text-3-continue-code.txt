<div class="post-text" itemprop="text">
<p>The below solution resolves several problems with your code.</p>
<pre><code>positive = {'yes','Yes','y','Y'}
negative = {'no','No', 'n','N'}
month = 'March'

while True:
    a = input('The month that you want to archive is {0}: (Y or N)?'.format(month))

    if a in negative:
        input("You will now leave, press Enter......")
        break

    elif a in positive:
        print("Let's do it")
        input("The archive will start now, press Enter......")
        # perform your tasks here
        break

    else:
        print("Enter either 'Y' to continue or 'N' to leave")
</code></pre>
<p><strong>Explanation</strong></p>
<ul>
<li>Test <code>a</code> versus the sets you have defined.</li>
<li>Use <code>str.format</code> to incorporate a variable in your <code>input</code> string.</li>
<li>Use <code>break</code> instead of <code>continue</code> to exit a loop after a positive value is entered; if you wish, you can include your "success" tasks inside the loop before <code>break</code>.</li>
</ul>
