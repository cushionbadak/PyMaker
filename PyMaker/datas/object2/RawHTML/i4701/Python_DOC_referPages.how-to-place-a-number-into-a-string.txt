<p>Assuming you want to put the number next to another string, you can also use <a href="https://pyformat.info/#string_pad_align" rel="nofollow noreferrer"><code>%</code></a> formatting to achieve the same result:</p>
<pre><code>&gt;&gt;&gt; w1 = "your number is:"
&gt;&gt;&gt; num = 20
&gt;&gt;&gt; line = '%s%10s' % (w1, num)
&gt;&gt;&gt; print(line)
'your number is:        20'
</code></pre>
