<p><code>display</code> is a function in the <code>IPython.display</code> module that runs the appropriate dunder method to get the appropriate data to ... display.  If you really want to run it</p>
<pre><code>from IPython.display import display
import pandas as pd

data = pd.DataFrame(data=[tweet.text for tweet in tweets], columns=['Tweets'])

display(data.head(10))
</code></pre>
