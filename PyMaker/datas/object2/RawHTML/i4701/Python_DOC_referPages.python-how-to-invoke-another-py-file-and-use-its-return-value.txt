<p>How can I use return value in another file?</p>
<pre><code>File1:
    def system(self):
        # do something
        return: value_1, value_2

File2:
    def to_do(self):
       # do something
       result = file_location.file1(value_1, value2)
</code></pre>
<p>Is this correct way?</p>

<div class="post-text" itemprop="text">
<p>You should import the specific method of <code>File1.py</code> in <code>File2.py</code> that you want to use.
Also note that python can return multiple values and how I have stored them in File2.py.</p>
<p><strong>File1.py</strong></p>
<pre><code>    def system(self):
        # do something
        return value_1, value_2
</code></pre>
<p><strong>File2.py</strong></p>
<pre><code>    from File1 import system
    def to_do(self):
       # do something
       result1, result2 = system(value_1, value2)
</code></pre>
