<div class="post-text" itemprop="text">
<p>You're trying to change directories into the <code>testProject/css/assets</code> folder instead of the <code>testProject/assets</code> folder.</p>
<p>The quickest way to get there from the <code>css</code> folder is with something like <code>os.chdir('..\assets')</code>, though there are lots of other ways to maneuver using <a href="https://docs.python.org/3/library/os.html#os.pardir" rel="nofollow noreferrer">os.pardir</a> and other <a href="https://docs.python.org/3/library/os.path.html" rel="nofollow noreferrer">os.path</a> manipulations.</p>
