<p>I have a list of lists:</p>
<pre><code>L = [['Bob'],['5', '0', '0', '0', '0'],['Jack'],['3', '0', '0', '1', '-3'],['Larry'],['1', '-3', '1', '-5', '0'], ...]
</code></pre>
<p>I want to convert this list of lists into a dictionary with the names as keys and the numbers as values so that it returns:</p>
<pre><code>{'Bob':[5, 0, 0, 0, 0], 'Jack':[3, 0, 0, 1, -3], 'Larry':[1, -3, 1, -5, 0], ...}
</code></pre>
<p>So I want a dictionary with strings as keys and an array of integers as values for each string, however, I can't figure out how to go about doing this.</p>
<p>The major thing here that troubles me is how to make it so the dictionary takes every other entry of the list as the values, while at the same time convert the strings into integers.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a dictionary comprehension:</p>
<pre><code>L = [['Bob'],['5', '0', '0', '0', '0'],['Jack'],['3', '0', '0', '1', '-3'],['Larry'],['1', '-3', '1', '-5', '0']]
new_l = {L[i][0]:list(map(int, L[i+1])) for i in range(0, len(L), 2)}
</code></pre>
<p>Output:</p>
<pre><code>{'Bob': [5, 0, 0, 0, 0], 'Larry': [1, -3, 1, -5, 0], 'Jack': [3, 0, 0, 1, -3]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Here is an alternative</p>
<pre><code>L = [['Bob'],['5', '0', '0', '0', '0'],['Jack'],['3', '0', '0', '1', '-3'],['Larry'],['1', '-3', '1', '-5', '0']]
{k[0]: list(map(int, v)) for k, v in zip(L[::2], L[1::2])}
</code></pre>
<p>Output:</p>
<pre><code>{'Bob': [5, 0, 0, 0, 0], 'Larry': [1, -3, 1, -5, 0], 'Jack': [3, 0, 0, 1, -3]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><strong>Update:</strong> </p>
<pre><code>dict_from_list = {key[0]: value for key, [int(elem) for elem in value] in zip(L[::2], L[1::2])}
</code></pre>
<p>I didn't notice the individual elements of the value lists are str. To convert them to int type, <a href="https://stackoverflow.com/a/49390453/5397214">what bphi has done</a> is the best, cleanest and pythonic solution among all posted here IMO. I would recommend studying it in detail and incorporating in your python programming as a matter of course.<br/>
If you do not wish to use <code>map</code> then using a list comprehension within the dict comprehension is the way to go, as demonstrated in the code above.  </p>
<p><strong>Original Answer:</strong><br/>
<code>dict_from_list = {key[0]: value for key, value in zip(L[::2], L[1::2])}</code></p>
