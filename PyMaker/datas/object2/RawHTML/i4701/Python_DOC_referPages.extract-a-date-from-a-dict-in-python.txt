<p>If the format does not change, you can use this (no need of RegEx):</p>
<pre><code>SHOPS_AND_ORDERNUM = {
    'Shop - Produce - 20180212.xlsx': 1334,
    'Shop - Organic - 20180223.xlsx': 8893,
    'Shop - Fresh - 20180226.xlsx': 5557,
    'Shop - Dairy - 20180227.xlsx': 3870
}

for item in SHOPS_AND_ORDERNUM:
    date = item.split('.xlsx')[0][-8:]
    print(date)
</code></pre>
<p>Output:</p>
<pre><code>20180212
20180223
20180226
20180227
</code></pre>
<p>Now, to get the date in the format you want, you can use the <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer"><code>datetime</code></a> module, like this:</p>
<pre><code>for item in SHOPS_AND_ORDERNUM:
    date = datetime.datetime.strptime(item.split('.xlsx')[0][-8:], '%Y%m%d').strftime('%d-%m-%Y')
    print(date)
</code></pre>
<p>Output:</p>
<pre><code>12-02-2018
23-02-2018
26-02-2018
27-02-2018
</code></pre>

<p>You can use the <code>datetime</code> module to get your required date format</p>
<p><strong>Ex:</strong></p>
<pre><code># -*- coding: utf-8 -*-
import datetime
SHOPS_AND_ORDERNUM = {
'Shop - Produce - 20180212.xlsx': 1334,
'Shop - Organic - 20180223.xlsx': 8893,
'Shop - Fresh - 20180226.xlsx': 5557,
'Shop - Dairy - 20180227.xlsx': 3870
}

for k,v in SHOPS_AND_ORDERNUM.items():
    print datetime.datetime.strptime(k.split("-")[-1].rstrip(".xlsx").strip(), "%Y%m%d"  ).strftime("%d-%m-%Y")
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>27-02-2018
26-02-2018
23-02-2018
12-02-2018
</code></pre>
<p><strong>MoreInfo</strong></p>
<ol>
<li><code>k.split("-")[-1].rstrip(".xlsx").strip()</code>  #to get the date string from key. ex: <code>20180212</code></li>
<li><code>datetime.datetime.strptime</code> #to convert datetime to your required format. <code>"%d-%m-%Y"</code></li>
</ol>

<p>You can use regex:</p>
<pre><code>import re
SHOPS_AND_ORDERNUM = {
'Shop - Produce - 20180212.xlsx': 1334, 
'Shop - Organic - 20180223.xlsx': 8893, 
'Shop - Fresh - 20180226.xlsx': 5557,
'Shop - Dairy - 20180227.xlsx': 3870 
}
new_data = {(lambda x:(x[7:], x[4:6], x[:4]))(re.findall('\d+', a)[0]):b for a, b in SHOPS_AND_ORDERNUM.items()}
</code></pre>
<p>Output:</p>
<pre><code>{('2', '02', '2018'): 1334, ('3', '02', '2018'): 8893, ('6', '02', '2018'): 5557, ('7', '02', '2018'): 3870}
</code></pre>
<p>Or, instead of tuples:</p>
<pre><code>new_data = {'{}-{}-{}'.format(*a[::-1]):b for a, b in new_data.items()}
</code></pre>
<p>Output:</p>
<pre><code>{'2018-02-2': 1334, '2018-02-7': 3870, '2018-02-3': 8893, '2018-02-6': 5557}
</code></pre>

<div class="post-text" itemprop="text">
<p>The basic regex you're looking for is ([0-9]+)(?=.)\g.  You can play around with it on <a href="https://regex101.com/" rel="nofollow noreferrer">https://regex101.com/</a>.</p>
