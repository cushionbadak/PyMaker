<p>I'd do something like this to use the format strings to create the widths for each number being converted to a printable format:</p>
<pre><code>print("{0:0&gt;{width}.{prec}}".format(d, width=5, prec=4))
</code></pre>
<p>The <code>0&gt;</code> part above will zero fill the padding on the left hand side.</p>
<p>Also I'd recommend having a look at the <a href="https://docs.python.org/3/library/decimal.html" rel="nofollow noreferrer">decimal</a> module in the standard library, this will let you represent fixed point numbers without having to write the code yourself.</p>
