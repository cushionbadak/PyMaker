<div class="post-text" itemprop="text">
<p>just split and <em>store</em> the result.</p>
<p>Then change the second word, and join again:</p>
<pre><code>text = "i eat salad"
words = text.split()
words[1] = 'ate'
print(" ".join(words))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use replace() to change it. </p>
<pre><code>text = "i eat salad"
text = text.replace('eat', 'ate')
print(text)
</code></pre>
