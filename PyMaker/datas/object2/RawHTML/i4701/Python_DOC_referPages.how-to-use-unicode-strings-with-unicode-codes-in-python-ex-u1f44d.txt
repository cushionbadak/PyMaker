<p>How do I use Unicode strings in Python? Say I want to use the Unicode <code>U+1F44D</code> (which is the thumbs up emoji). Do I use <code>u'U+1F44D'</code>, <code>u'1F44D'</code>, or <code>u'0x1F44D'</code>? Or should I use <code>chr(0x1F44D)</code>? I'm just not sure.</p>
<p>Thanks for your help, and sorry if this wasted your time.</p>
</div>
<div class="post-text" itemprop="text">
<p>Unicode characters at codepoints above 0xffff can be entered as literals like this: <code>"\U0001f44D"</code>.</p>
<p>For characters bellow 0xffff, the four digit version uses the  <code>u</code> character, like in <code>\u2654</code>.</p>
<p>If you have your emoji code as numeric data during program  runtime, not at coding time, you can use the <code>chr</code> build in - actually, you can use the chr builtin all along. The inconveninence is that it yields a single character string, always:  <code>chr(0x1f44d)</code>.</p>
<p>Back to literals, it is also possible to use the official unicode name for the character inside a Python 3 literal string - for that, use the <code>\N</code> prefix and enclose the character name between <code>{}</code>: <code>"\N{THUMBS UP SIGN}"</code>.
If you want to fetch characters by name at runtime, use the unicodedata.lookup: <code>import unicodedata; unicodedata.lookup("THUMBS UP SIGN")</code></p>
