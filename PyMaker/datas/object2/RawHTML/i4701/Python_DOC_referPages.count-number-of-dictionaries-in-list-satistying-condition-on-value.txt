<p>How do I get the number of dictionaries where value satisfies a given condition?</p>
<pre><code>list_of_dict = [
                   {'seq_no': 10, 'file_path': 'file/path', 'rel_item': None},
                   {'seq_no': 22, 'file_path': 'file/path', 'rel_item': 0},
                   {'seq_no': 32, 'file_path': 'file/path', 'rel_item': 0},
                   {'seq_no': 90, 'file_path': 'file/path', 'rel_item': 0},
                   {'seq_no': 10, 'file_path': 'file/path', 'rel_item': None},
 ]
</code></pre>
<p>I would like to count the number of dictionaries where key <code>rel_item=None</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Here is one way.</p>
<pre><code>c = sum(i['rel_item'] is None for i in list_of_dict)
# 2
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter</code></a> the <code>list_of_dict</code> based on your criteria</p>
<pre><code>&gt;&gt;&gt; len(list(filter(lambda x: x['rel_item'] is None, list_of_dict)))
&gt;&gt;&gt; 2
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This will do the job:</p>
<pre><code>len([x for x in list_of_dict if x["rel_item"] == None])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can using <code>map</code> then </p>
<pre><code>sum((map(lambda x : x['rel_item'] is None,list_of_dict)))
Out[721]: 2
</code></pre>
