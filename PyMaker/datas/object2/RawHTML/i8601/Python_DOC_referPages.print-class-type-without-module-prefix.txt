<p>I have a bunch of class types (derived from <code>type</code>). When I print them, I get stuff like:</p>
<pre><code>...
&lt;class 'Default.new_templates.NewSnippetCommand'&gt;
&lt;class 'Default.new_templates.NewSyntaxCommand'&gt;
&lt;class 'Default.pane.ClosePaneCommand'&gt;
&lt;class 'Default.pane.FocusNeighboringGroup'&gt;
&lt;class 'Default.pane.MoveToNeighboringGroup'&gt;
&lt;class 'Default.pane.NewPaneCommand'&gt;
&lt;class 'Default.pane.SetMaxColumns'&gt;
...
</code></pre>
<p>I would like to print:</p>
<pre><code>...
NewSnippetCommand
NewSyntaxCommand
ClosePaneCommand
FocusNeighboringGroup
MoveToNeighboringGroup
NewPaneCommand
SetMaxColumns
...
</code></pre>
<p>How can I get to the class name part and leave out the module part?</p>
</div>
<div class="post-text" itemprop="text">
<p>You could use the <a href="https://docs.python.org/3/library/stdtypes.html#definition.__name__" rel="nofollow noreferrer"><code>__name__</code> attribute</a>:</p>
<pre><code>&gt;&gt;&gt; type(OrderedDict())
&lt;class 'collections.OrderedDict'&gt;
&gt;&gt;&gt; type(OrderedDict()).__name__
'OrderedDict'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The name of a class is the <code>__name__</code> attribute:</p>
<pre><code>print(whateverclass.__name__)
</code></pre>
</div>
