<p>If you want the code to run multiple times, you need to use a loop:</p>
<pre><code>while beep == 1:
     if list1[letter] == "a":
         print abinary

     ...
     letter = letter + 1
     if letter &gt;= length(list):
          beep = 2
</code></pre>
<p>You can even do this more beautiful using a <a href="https://docs.python.org/3/tutorial/controlflow.html#for-statements" rel="nofollow noreferrer">for-loop</a> to iterate over all letters in your list:</p>
<pre><code>for letter in list1:
    if letter == "a":
        print abinary
    ...
</code></pre>
<p>instead of chaining alot of elifs you should store the binary representation of the letters in a <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">dictionary</a>:</p>
<pre><code>binary_letters = {
    "a": "01100001",
    "b": "01100010",
    ...
    "z": "01111010"
    }
</code></pre>
<p>this will simplify the loop to:</p>
<pre><code>for letter in list1:
    print binary_letters[letter]
</code></pre>
