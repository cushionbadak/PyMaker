<p>I have the following string</p>
<pre><code>myString = "cat(50),dog(60),pig(70)"
</code></pre>
<p>I try to convert the above string to 2D array.
The result I want to get is</p>
<pre><code>myResult = [['cat', 50], ['dog', 60], ['pig', 70]]
</code></pre>
<p>I already know the way to solve by using the legacy string method but it is quite complicated. So I don't want to use this approach.</p>
<pre><code># Legacy approach
# 1. Split string by ","
# 2. Run loop and split string by "(" =&gt; got the &lt;name of animal&gt;
# 3. Got the number by exclude ")".
</code></pre>
<p>Any suggestion would appreciate.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the re.findall method:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; re.findall(r'(\w+)\((\d+)\)', myString)
[('cat', '50'), ('dog', '60'), ('pig', '70')]
</code></pre>
<p>If you want a list of lists as noticed by RomanPerekhrest convert it with a list comprehension:</p>
<pre><code>&gt;&gt;&gt; [list(t) for t in re.findall(r'(\w+)\((\d+)\)', myString)]
[['cat', '50'], ['dog', '60'], ['pig', '70']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Alternative solution using <a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow noreferrer"><strong><em>re.split()</em></strong></a> function:</p>
<pre><code>import re
myString = "cat(50),dog(60),pig(70)"
result = [re.split(r'[)(]', i)[:-1] for i in myString.split(',')]

print(result)
</code></pre>
<p>The output:</p>
<pre><code>[['cat', '50'], ['dog', '60'], ['pig', '70']]
</code></pre>
<hr/>
<p><code>r'[)(]'</code> - pattern, treats parentheses as delimiters for splitting</p>
<p><code>[:-1]</code> - slice containing all items except the last one(which is empty space <code>' '</code>)</p>
</div>
