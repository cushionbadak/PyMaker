<p>How do I insert one list fully inside another python list?</p>
<pre><code>&gt;&gt;&gt; list_one = [1,2,3]
&gt;&gt;&gt; list_two = [4,5,7]

&gt;&gt;&gt; list_one.insert(2, [2, 3])
&gt;&gt;&gt; list_one
[1, 2, [2, 3], 3, 4, 5, 7]
</code></pre>
<p>But I want the result to be:</p>
<pre><code>[1, 2, 2, 3, 3, 4, 5, 7]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using slice assignment (if you meant <code>[1,2,4,5,6,3]</code>):</p>
<pre><code>&gt;&gt;&gt; list_one = [1,2,3]
&gt;&gt;&gt; list_two = [4,5,6]
&gt;&gt;&gt; list_one[2:2] = list_two
&gt;&gt;&gt; list_one
[1, 2, 4, 5, 6, 3]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Are you sure you don't want this result <code>[1,2,4,5,6,3]</code> instead? If so, try this:</p>
<pre><code>list_one[:2]+list_two+list_one[2:]
</code></pre>
</div>
