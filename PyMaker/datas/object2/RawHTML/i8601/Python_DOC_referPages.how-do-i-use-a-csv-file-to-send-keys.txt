<p>I can use the <code>csv</code> module to read that. <code>csv.DictReader</code> reads each row into its own <code>dict</code> that lets me reference cells by the name given in the header. Since I'll be looking up records by email name later, I'll read it into another <code>dict</code> that will act as an index into the records. If the same user is in there multiple times, only the last one will be remembered.</p>
<p>With the index in place, I can grab the row by email name.</p>
