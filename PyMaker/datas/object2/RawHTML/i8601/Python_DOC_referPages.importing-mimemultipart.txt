<p><a href="https://docs.python.org/3/library/email.mime.html#email.mime.multipart.MIMEMultipart" rel="nofollow noreferrer"><code>MIMEMultipart</code></a> is not a module; it is a class in the <code>email.mime.multipart</code> module.  To import it, you have to do:</p>
<pre><code>from email.mime.multipart import MIMEMultipart
</code></pre>
