<div class="post-text" itemprop="text">
<p>In your for loop you are replacing the variable <code>generate_random_names</code> everytime the loops iterate. Instead maybe put them into a list. Note <code>choice()</code> will give you duplicate values if that's what you wish for. </p>
<pre><code>people_count = int(input("Enter how many names you wish to have, min 0, max 20: "))
generate_random_names = []
for l in range(people_count):
     generate_random_names.append(random.choice(names))

print (generate_random_names)
</code></pre>
<p>Now you will have a list of names</p>
<p>Also you don't need a for loop to do what you want, you could use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>people_count = int(input("Enter how many names you wish to have, min 0, max 20: "))
generate_random_names = [random.choice(names) for _ in range(people_count)]
</code></pre>
<p>Heck if you don't want to have duplicate values in new list, you can use <a href="https://docs.python.org/3/library/random.html#random.sample" rel="nofollow noreferrer">sample</a> which takes a collection to choose from and how many you want in the new list.</p>
<pre><code>people_count = int(input("Enter how many names you wish to have, min 0, max 20: "))
generate_random_names = random.sample(names,people_count)
print(generate_random_names)
</code></pre>

<p>you are looking for <a href="https://docs.python.org/3/library/random.html#random.sample" rel="nofollow noreferrer"><code>random.sample</code></a>:</p>
<pre><code>from random import sample

names = ('Benjamin', 'Damien', 'Dexter', 'Jack', 'lucas', 'Norman', 'Thorn', 'Bella', 'Blair',
         'Ivy', 'Lilth', 'Megan', 'Rue', 'Sabrina', 'Samara',
         'Anthea', 'Jessica', 'Igor', 'Luther', 'Boris', 'Abel')

print(sample(names, k=4))
# ['lucas', 'Ivy', 'Dexter', 'Abel']
</code></pre>
<p>where the <code>k</code> parameter is the number of elements you want so select.</p>
<p><a href="https://docs.python.org/3/library/random.html#random.choice" rel="nofollow noreferrer"><code>random.choice</code></a> will always select one single element only.</p>

<div class="post-text" itemprop="text">
<p>You are assigning the value to variable in loop <strong>"n"</strong> times and so when you print, the last assigned value is returned.</p>
<p>What you need to do is store those random names something like this : </p>
<pre><code>import random

names = ('Benjamin', 'Damien', 'Dexter', 'Jack', 'lucas', 'Norman', 'Thorn', 'Bella', 'Blair',
         'Ivy', 'Lilth', 'Megan', 'Rue', 'Sabrina', 'Samara', 
         'Anthea', 'Jessica', 'Igor', 'Luther', 'Boris', 'Abel', )

people_count = int(input("Enter how many names you wish to have, min 0, max 20: "))

generate_random_names_list = []
for l in range(people_count):
     generate_random_names_list.append(random.choice(names))

print(generate_random_names_list)
</code></pre>
<p>For input as <strong>3</strong>, this resulted in something like this :</p>
<pre><code>['Norman', 'Rue', 'Rue']
</code></pre>
<p>To get only the unique names inside the list, you can do something like this :</p>
<pre><code>import random

names = ('Benjamin', 'Damien', 'Dexter', 'Jack', 'lucas', 'Norman', 'Thorn', 'Bella', 'Blair',
         'Ivy', 'Lilth', 'Megan', 'Rue', 'Sabrina', 'Samara', 
         'Anthea', 'Jessica', 'Igor', 'Luther', 'Boris', 'Abel', )

people_count = int(input("Enter how many names you wish to have, min 0, max 20: "))

generate_random_names_list = []
while len(generate_random_names_list) &lt; people_count :
    choice = random.choice(names)
    if choice not in generate_random_names_list :
        generate_random_names_list.append(choice)
print(generate_random_names_list)
</code></pre>
<p>This will result in something like this for <strong>10</strong> as input : </p>
<pre><code>['Boris', 'Lilth', 'Jack', 'Igor', 'Luther', 'lucas', 'Damien', 'Rue', 'Abel', 'Blair']
</code></pre>
