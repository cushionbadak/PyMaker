<div class="post-text" itemprop="text">
<p>You're using the <code>ipdb</code> <em>command</em> <code>list</code> from what I can understand. If <code>ipdb</code> follows the interface defined by <code>pdb</code> <a href="https://docs.python.org/3/library/pdb.html#pdbcommand-list" rel="noreferrer">this doesn't invoke the <code>list()</code> function as you'd expect</a>.</p>
<p>Exit the <code>ipdb</code> debugger to get this to work correctly or, again if <code>ipdb</code> uses the same commands as <code>ipdb</code>, use <code>p list(patterns_and_values.values())</code> in order to get an expression evaluated inside the debugger.</p>
</div>
