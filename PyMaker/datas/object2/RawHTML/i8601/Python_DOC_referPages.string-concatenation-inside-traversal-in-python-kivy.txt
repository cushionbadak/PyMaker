<div class="post-text" itemprop="text">
<p>To get attributes dynamically, you have to use <code>getattr(obj, 'attr_name')</code>.</p>
<p>In your case it could be like this:</p>
<pre><code>getattr(self.root.ids, 'button_%d' % i).text
</code></pre>
<p>For additional details see the <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer">docs</a>.</p>
</div>
