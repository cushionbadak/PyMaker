<p>Say I have a python array and a numpy array</p>
<pre><code>import numpy as np
python_array = [range(20), range(20), range(20)] 
numpy_array = np.array(python_array)
</code></pre>
<p>You can do:</p>
<pre><code>numpy_array + python_array
</code></pre>

<pre><code>python_array + numpy_array

<pre><code>python_array.\__add__(numpy_array)
</code></pre>

<pre><code> can only concatenate list (not "numpy.ndarray") to list

<div class="post-text" itemprop="text">
<p>There's also an <a href="https://docs.python.org/3/reference/datamodel.html#object.__radd__" rel="nofollow noreferrer"><code>__radd__</code></a> method for <code>b</code> to implement <code>a + b</code> if <code>a</code> doesn't understand the operation. You're seeing <code>numpy.ndarray.__radd__</code>.</p>
