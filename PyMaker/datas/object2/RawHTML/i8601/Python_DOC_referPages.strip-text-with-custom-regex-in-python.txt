<p>Above is a tfidf matrix which has been written to a file.
I want to read only the tf-idf values like 0.74132892839 and append them to a list.</p>
<p>Is there a way to do f.read() and then strip the indices off?</p>

<div class="post-text" itemprop="text">
<p>Simple solution using <a href="https://docs.python.org/3/library/re.html#re.sub" rel="nofollow noreferrer"><strong><em>re.sub()</em></strong></a> function:</p>
<pre><code>import re

# specify your actual file name
with open('lines.txt', 'r') as fh:
    result = re.sub(r'\([^)]+\)\s*', '', fh.read()).split('\n')

print(result)
</code></pre>
<p>The output:</p>
<pre><code>['0.74670222994', '0.74132892839', '0.671141877647', '0.699490245832', '0.422715095257', '0.433278265941', '0.379862196713', '0.653731227092', '0.756726821729']
</code></pre>
<hr/>
<p><code>\([^)]+\)</code> - matches a sequence between parentheses </p>
