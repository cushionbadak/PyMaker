<pre><code>file1.txt
abc

hij

file2.txt
def
abc
qlm

file3.txt
def
lop
tmn
</code></pre>
<p>desired output:</p>
<p>mergedfile.csv</p>
<pre><code>file1      file2     file2
abc        def       def
def        abc       lop
hij        qlm       tmn
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Python Documentation <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer">CSV File Reading and Writing</a><br/>
For instance, <strong>lowest memory</strong> footprint:  </p>
<pre><code>import io, csv
fieldnames = [file1, file2, file3]
with io.open(csv_file, 'w', newline='') as fh_csv, \
    open(file1) as fh1, \
    open(file2) as fh2, \
    open(file3) as fh3:

    writer = csv.writer(fh_csv, delimiter='\t')
    writer.writerow(fieldnames)

    while True:
        out = []
        for fh in [fh1, fh2, fh3]:
            out.append( fh.readline().strip('\n') )

        if all(out):
            writer.writerow(out)
        else:
            break
</code></pre>
<p><strong>Output</strong>:  </p>
<pre><code>file1.txt   file2.txt   file3.txt  
abc         def         def  
def         abc         lop  
hij         qlm         tmn  
</code></pre>
<p><strong><em>Tested with Python:3.4.2</em></strong> </p>

<p>My way:</p>
<pre><code>file_list=['a','b','c']

open("o","w").write("\n".join([" ".join(i).replace("\n","") for i in list(zip(*[list(open(i)) for i in file_list]))]))
</code></pre>
<p>Result:</p>
<pre><code>file1.txt file2.txt file3.txt
abc def def
def abc lop
hij qlm tmn
</code></pre>
<hr/>
<p>I read the three files and append the data into three lists and then use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip</a> to aggregates elements.If you want to be compatible with <code>Python2.x</code> and <code>Python3.x</code> you can add this to the top of your code:</p>
<pre><code>try:
    from itertools import izip as zip
except ImportError:
    pass
</code></pre>
<p>Then I remove the <code>\n</code> character,and combine the list and write them into a file.</p>

df1=pd.read_csv('file1.txt',names=['file1'])
df2=pd.read_csv('file2.txt',names=['file2'])

<p>or more python style:</p>
<pre><code>import os
import pandas as pd

def merge_files(file_list, export_file):
    df_list = [pd.read_csv(
        one_file, names = [os.path.splitext(os.path.basename(one_file))[0]]
    ) for one_file in file_list if os.path.isfile(one_file)]
    all_df=pd.concat(df_list,axis=1)
    all_df.to_csv(export_file, index=False)


if __name__=='__main__':
    sample_files=['./file1.txt', '../temp/file2.txt','file3.txt']
    merge_files(sample_files,"mergedfile.txt")
</code></pre>
