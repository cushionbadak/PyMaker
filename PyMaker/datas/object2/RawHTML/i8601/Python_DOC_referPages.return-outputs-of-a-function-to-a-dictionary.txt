<p>I have a function which can return a variable number of outputs. I would like to save this to a dictionary like so:</p>
<pre><code>val_dict={}
idx = 0

for key in some_test_function(1,2,3):
    val_dict[idx] = key
    idx +=1
</code></pre>
<p>Now while this thing does work, I was wondering if there is a more elegant way of doing it. Any ideas?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate()</code> function</a> to add the indices in the dictionary:</p>
<pre><code>val_dict = dict(enumerate(some_test_function(1, 2, 3)))
</code></pre>
<p>This gives you the exact same integer index to value mapping. However, just a list would give you the exact same index-to-value mapping:</p>
<pre><code>val_list = list(some_test_function(1, 2, 3))
</code></pre>
<p>(if <code>some_test_function</code> already returns a <code>list</code> object, just remove the <code>list()</code> call).</p>
</div>
