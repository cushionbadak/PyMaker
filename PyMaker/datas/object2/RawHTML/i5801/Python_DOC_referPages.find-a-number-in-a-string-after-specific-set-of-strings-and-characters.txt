<p>this looks like a <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer"><code>dictionary</code></a> - so you can simply access it by <code>set["id"]</code></p>
<pre><code>di = {'first_name': 'Abigail', 'last_name': 'Smith', 'updated': '2017-12-04 22:12:24.699-08', 'account_id': '338502', 'created': '2017-12-04 22:12:24.699-08', 'student_id': '1730', 'notes': '', 'school_id': '16', 'major_id': '20', 'status': 'OPEN', 'id': '37'}

print(di["id"])
</code></pre>
<p>Output:</p>
<pre><code>37
</code></pre>
<p>Done.</p>
<p>If it is a string, you can split it into a dict like this:</p>
<pre><code>di = "{'first_name': 'Abigail', 'last_name': 'Smith', 'updated': '2017-12-04 22:12:24.699-08', 'account_id': '338502', 'created': '2017-12-04 22:12:24.699-08', 'student_id': '1730', 'notes': '', 'school_id': '16', 'major_id': '20', 'status': 'OPEN', 'id': '37'}"

pairs = [x.replace("'","").strip("{}") for x in di.split(",")]
dic = dict( [ x.replace("'","").strip().split(":",1) for x in pairs] )
print(dic)
print(dic ["id"])
</code></pre>
<p>Output:</p>
<pre><code>{'first_name': ' Abigail', 'last_name': ' Smith', 'updated': ' 2017-12-04 22:12:24.699-08', 'account_id': ' 338502', 'created': ' 2017-12-04 22:12:24.699-08', 'student_id': ' 1730', 'notes': '', 'school_id': ' 16', 'major_id': ' 20', 'status': ' OPEN', 'id': ' 37'}
 37
</code></pre>
