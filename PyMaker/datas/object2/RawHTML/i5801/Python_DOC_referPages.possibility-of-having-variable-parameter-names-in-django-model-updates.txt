<div class="post-text" itemprop="text">
<p>You can just <a href="https://docs.python.org/3/tutorial/controlflow.html#keyword-arguments" rel="nofollow noreferrer"><em>unpack into keyword arguments</em></a>:</p>
<pre><code>model_instance.update(**{field: value})
</code></pre>
<hr/>
<p>There is also an option to set the model field value and then save (there are though <a href="https://stackoverflow.com/questions/30449960/django-save-vs-update-to-update-the-database">differences</a>):</p>
<pre><code>setattr(model_instance, field, value)
model_instance.save()
</code></pre>
</div>
