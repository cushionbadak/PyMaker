<div class="post-text" itemprop="text">
<p>so I'm writing a program in PyQt5 and making use of the QObject class. Here's the basic program.</p>
<pre><code>class Object(QObject):

    def __init__(self, parent=None):
        super(Object, self).__init__(parent)
        self.field = []


class Object2(Object):

    def __init__(self):
        super(Object, self).__init__()
        self.field.append(1)

if __name__ == '__main__':
    o = Object2()
</code></pre>
<p>But I'm getting this error:</p>
<pre><code>AttributeError: 'Object2' object has no attribute 'field'
</code></pre>
<p>I can't seem to find the cause of problem. Is it that a python child class cannot access it's parents attributes?</p>
</div>
<div class="post-text" itemprop="text">
<p>The error you are getting is because of the arguments you are passing into <code>super</code>. In Python 2 it takes 2 arguments: The first argument is the current class (<code>Object2</code>), and the second argument is the current instance (<code>self</code>).</p>
<p><em>The issue is you've passed the <strong>parent</strong> class instead of the <strong>current</strong> class.</em></p>
<p>So you want:</p>
<pre><code>class Object2(Object):
    def __init__(self):
        super(Object2, self).__init__() # Current class: Object2
        self.field.append(1)
</code></pre>
<p>In Python 3, it is no longer required to pass these arguments into <code>super</code>. So you would just do:</p>
<pre><code>class Object2(Object):
    def __init__(self):
        super().__init__()
        self.field.append(1)
</code></pre>
