<div class="post-text" itemprop="text">
<p>This should give you what you're looking for:</p>
<pre><code>cmd = 'espeak -s110 "{0} is {1} and the distance is {2} cm" --stdout | aplay -D sysdefault:CARD=2'.format(label, spos, str(distance))

os.system(cmd)
</code></pre>
