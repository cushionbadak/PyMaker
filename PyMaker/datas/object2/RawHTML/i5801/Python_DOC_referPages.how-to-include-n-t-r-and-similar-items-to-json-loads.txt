<p>I want to include \n not as newline but as a string literal in json.loads</p>
<p>for example, json.loads('"hello\n"')</p>
<p>how do i handle the \?</p>
</div>
<div class="post-text" itemprop="text">
<p><code>\n</code> is the <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow noreferrer">escape sequence</a> for line feed. Use the <code>\\</code> escape sequence: <code>json.loads('"hello\\n"')</code></p>
<p>Alternatively, use a <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow noreferrer">raw string</a>, which treats a backslash as a literal character: <code>json.loads(r'"hello\n"')</code></p>
</div>
