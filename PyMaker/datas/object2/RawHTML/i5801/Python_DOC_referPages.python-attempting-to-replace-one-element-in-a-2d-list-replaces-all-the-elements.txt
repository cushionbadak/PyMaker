<div class="post-text" itemprop="text">
<p>List multiplication creates several references to the same list. So when you change one, all recieve the changes.</p>
<p>To fix this, you can use list comprehension.</p>
<p>Change this:</p>
<pre><code>matrix = [[None]*length]*length
</code></pre>
<p>To this:</p>
<pre><code>matrix = [[None]*length for _ in range(length)]
</code></pre>
<p>More about list comprehensions <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">here</a>.</p>
