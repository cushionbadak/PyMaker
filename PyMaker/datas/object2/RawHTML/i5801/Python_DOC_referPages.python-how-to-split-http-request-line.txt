<div class="post-text" itemprop="text">
<p>Two step approach using <code>str.partition</code> and <code>str.rpartition</code> (could also be accomplished by a <code>str.split</code> and <code>str.rsplit</code> with a <code>maxsplit=1</code> parameter each time):</p>
<pre><code>line = 'GET https://example.com/hello?ageSelections=0&amp;category=6000&amp;productPeriodType=3&amp;resourcePlace=resourcePlace AND 5089=5089-- PStU&amp;twoCategory=6200 HTTP/1.1'
method, _, rest = line.partition(' ')
url, _, protocol = rest.rpartition(' ')
parsed_line = [method, url, protocol]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to be a bit more precise with specifications like whether it's a string. But given that it is a string it should look like:</p>
<pre><code>string = GET https://example.com/hello?ageSelections=0&amp;category=6000&amp;productPeriodType=3&amp;resourcePlace=resourcePlace AND 5089=5089-- PStU&amp;twoCategory=6200 HTTP/1.1

data = string.split();
</code></pre>
<p>Of course if it is not a string you can't do that. </p>
