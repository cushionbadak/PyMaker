<p>You can use list comprehension:</p>
<pre><code>a = [3, 1, 2, 1]
new_a = [c for i, c in enumerate(a) if c != 1 and i+1 &lt; len(a) and a[i+1] != 2]+[a[-1]]
</code></pre>
<p>Output:</p>
<pre><code>[3, 2, 1]
</code></pre>

<div class="post-text" itemprop="text">
<p>I would use a <em>good-ol'</em> explicit loop for this one:</p>
<pre><code>a = [3, 1, 2, 1]
b = [a[0]]

for i, item in enumerate(a[1:], 1):
  if item == 1 and a[i-1] == 2:
    continue
  else:
    b.append(item)
print(b)  # -&gt; [3, 1, 2]
</code></pre>
<hr/>
<p>Note that <code>a</code> does not get modified but rather a new list <code>b</code> is created.</p>

<p>The <code>del</code> keyword might be your best bet here, for example:</p>
<pre><code>a = [3, 1, 2, 1]
index = 1
while index &lt; len(a):
    if a[index - 1] == 2 and a[index] == 1:
        del a[index]
    else:
        index += 1
print(a)
</code></pre>

<p>I don't know about the more pythonic way to do this, but you can keep a counter to check for the second appearance of a value you want to delete from a list.</p>
<pre><code>a = [3, 1, 2, 1]
counter = 0

for i, v in enumerate(a):
    if v==1:
        if counter == 1:
            del a[i]
        else:
            counter += 1

print a

#output: [3, 1, 2]
</code></pre>
