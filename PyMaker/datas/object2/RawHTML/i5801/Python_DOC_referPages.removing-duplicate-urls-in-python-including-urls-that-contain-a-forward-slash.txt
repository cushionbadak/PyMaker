<p>You can always <a href="https://docs.python.org/3/library/stdtypes.html?highlight=rstrip#str.rstrip" rel="nofollow noreferrer">right-strip the slash</a> - it would be removed if exists and nothing will be done if not:</p>
<pre><code>result_set = myJoin(my_url, found_link).rstrip("/")
</code></pre>
