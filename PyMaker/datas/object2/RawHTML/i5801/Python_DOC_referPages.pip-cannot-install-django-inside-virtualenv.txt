<div class="post-text" itemprop="text">
<p>Make sure your Python version in the virtualenv is Python 3.4+.  </p>
<p>Django 2.0 only supports Python 3.4+, and <a href="https://docs.python.org/3/library/functools.html#functools.lru_cache" rel="nofollow noreferrer"><code>functools.lru_cache</code></a> is only available in 3.2+.</p>
<p>To check the Python version in your current virtualenv:</p>
<pre><code>python --version
</code></pre>
<p>To create a new virtualenv with python 3:</p>
<pre><code>python3 -m venv venv --prompt=myenv
</code></pre>
</div>

<p>This error is because of wrong picking of pip and python version . You can solve it by adding below lines in the ~/.bash_profile</p>
<pre><code>  alias python='python3'
  alias pip='pip3.6' 
</code></pre>
<p>After that close the terminal and run the following command to install Django</p>
<pre><code>  pip install Django==2.0
</code></pre>
