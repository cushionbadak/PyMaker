<div class="post-text" itemprop="text">
<p>Use parentheses:</p>
<pre><code>species_plots_types.groupby("plot_type").count() 
</code></pre>
<p>Instead of square brackets</p>
<pre><code>species_plots_types.groupby["plot_type"].count()
</code></pre>
<p>Refer the given <a href="https://docs.python.org/3/library/itertools.html?highlight=groupby#itertools.groupby" rel="nofollow noreferrer">link</a> for further details.</p>
</div>
<div class="post-text" itemprop="text">
<p>You need <code>()</code>:</p>
<pre><code>species_plots_types.groupby("plot_type").size().to_frame()
</code></pre>
<p>Or:</p>
<pre><code>species_plots_types.groupby("plot_type").count()
</code></pre>
<p>Explanation:</p>
<p><code>groupby["plot_type"]</code> means you call <code>__getitem__</code> on <code>groupby</code>, but since this method isn't defined, you get a <code>TypeError: 'type' object is not subscriptable</code>. Check python documentation for <a href="https://docs.python.org/3/reference/datamodel.html#object.__getitem__" rel="nofollow noreferrer">getitem</a> for more details.</p>
