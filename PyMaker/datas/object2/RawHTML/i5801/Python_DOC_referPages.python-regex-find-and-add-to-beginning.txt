<pre><code>str = '''
Here are some order bullets:

</code></pre>
<p>How do I get to this:</p>
<pre><code>str = '''
Here are some order bullets:
- i. Foo
- ii. Bar

- iv. YOLO
...
- x. LOL
'''
</code></pre>

<div class="post-text" itemprop="text">
<p>You can use this regex in <code>MULTILINE</code> mode:</p>
<pre><code>&gt;&gt;&gt; print re.sub(r'(?m)^([a-zA-Z0-9]+\.)', r'- \1', str)

Here are some order bullets:
- i. Foo
- ii. Bar
- iii. Baz
- iv. YOLO
...
- x. LOL
</code></pre>
<p><strong>RegEx Breakup:</strong></p>
<p><strong>Search:</strong></p>
<ul>
<li><code>(?m)</code>: Enable MULTILINE mode</li>
<li><code>^</code>: Start</li>
<li><code>([a-zA-Z0-9]+\.)</code>: Match and group 1+ alphanumeric characters</li>
</ul>
<p><strong>Replacement:</strong></p>
<ul>
<li><code>-</code>: Place literal <code>"- "</code></li>
<li><code>\1</code>: Back-reference to captured group #1</li>
</ul>
</div>
