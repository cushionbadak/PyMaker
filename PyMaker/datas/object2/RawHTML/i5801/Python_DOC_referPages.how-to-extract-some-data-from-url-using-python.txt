<p>Desired output using <code>re</code>.</p>
<pre><code>import re

url = "https://some_url/vivi/v2/ZUxOZmVrdzJqTURxV20wQ0RvRld6SytEQWNocThwMGVnbFJ4RDQrZzJMeGRBcnhPYnUzV1pRPT0=/BE?category=PASSENGER&amp;make=30&amp;model=124&amp;regmonth=3&amp;regdate=2015-03&amp;body=443,4781&amp;facelift=252&amp;seats=4&amp;bodyHeight=443&amp;bodyLength=443&amp;weight=-1&amp;engine=1394&amp;wheeldrive=196&amp;transmission=400"

</code></pre>
<p>Resulting with <code>['ZUxOZmVrdzJqTURxV20wQ0RvRld6SytEQWNocThwMGVnbFJ4RDQrZzJMeGRBcnhPYnUzV1pRPT0=']</code>.</p>

<p>The string that you are after is not a query parameter, it is part of the URL path.</p>
<p>In the general case you can use the <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.urlparse" rel="nofollow noreferrer"><code>urllib.parse</code></a> module to parse the URL into its components, then access the path. Then extract the required part of the path:</p>
<pre><code>import base64
from urllib.parse import urlparse, parse_qs

parsed_url = urlparse(url)
s = parsed_url.path.split('/')[-2]    # second last component of path
&gt;&gt;&gt; s
'ZUxOZmVrdzJqTURxV20wQ0RvRld6SytEQWNocThwMGVnbFJ4RDQrZzJMeGRBcnhPYnUzV1pRPT0='
&gt;&gt;&gt; base64.b64decode(s)
b'eLNfekw2jMDqWm0CDoFWzK+DAchq8p0eglRxD4+g2LxdArxObu3WZQ=='
</code></pre>
<hr/>
<p>The keys and values of the query string can also be processed into a dictionary and accessed by key:</p>
<pre><code>params = parse_qs(parsed_url.query)
&gt;&gt;&gt; params
{'category': ['PASSENGER'], 'make': ['30'], 'model': ['124'], 'regmonth': ['3'], 'regdate': ['2015-03'], 'body': ['443,4781'], 'facelift': ['252'], 'seats': ['4'], 'bodyHeight': ['443'], 'bodyLength': ['443'], 'weight': ['-1'], 'engine': ['1394'], 'wheeldrive': ['196'], 'transmission': ['400']}
&gt;&gt;&gt; params['category']
['PASSENGER']
</code></pre>
