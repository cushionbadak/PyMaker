<div class="post-text" itemprop="text">
<p>i assume your "in order" means the character match in order. In your example text "Ce" name Arsenic should return False and should return True if text is "eC" instead.</p>
<p>first we check if all the character are inside the matching text, if yes, we further check if the matching index are in order, else we just return False</p>
<pre><code>def check_text(text, name):
    name_lower = name.lower()
    if all(x in name_lower for x in text.lower()):
        char_index = [name_lower.index(x) for x in text.lower()]
        return char_index == sorted(char_index)
    else:
        return False


&gt;&gt;&gt; check_text("Ce", "Arsenic")
False
&gt;&gt;&gt; check_text("eC", "Arsenic")
True
</code></pre>

<div class="post-text" itemprop="text">
<p>Essentially, you want to check if <code>text</code> is contained in <code>name</code>. So instead of doing it at the character by character level (i.e. if all characters in text appears in name, the way that you are doing now), use the <code>in</code> operator on the two strings to check if <code>text</code> is <code>in</code> <code>name</code>.</p>
<p>i.e.
<code>return text.lower() in name.lower()</code></p>
<p>see <a href="https://www.pythoncentral.io/how-to-see-if-a-string-contains-another-string-in-python/" rel="nofollow noreferrer">https://www.pythoncentral.io/how-to-see-if-a-string-contains-another-string-in-python/</a></p>
</div>
