<blockquote>
<p>for example I can use <code>&lt;h2&gt;</code> tag and use keyword <code>Recipe</code> to find the data of second <code>&lt;h2&gt;</code> tag</p>
</blockquote>
<p>Yes, you can do exactly that. You can use the Python <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer"><code>re</code></a> (Regular Expression) module to match partial text inside a tag.</p>
<p>From the <strong><a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/#a-regular-expression" rel="nofollow noreferrer">documentation</a></strong>:</p>
<blockquote>
<p>If you pass in a regular expression object, Beautiful Soup will filter against that regular expression using its <code>search()</code> method.</p>
</blockquote>
<p><strong>Demo:</strong></p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; from bs4 import BeautifulSoup
&gt;&gt;&gt; 
&gt;&gt;&gt; html = '''&lt;h2&gt;Video Instructions For Making Soft Idlis&lt;/h2&gt;
    &lt;div class="embed-responsive embed-responsive-16by9"&gt;
    &lt;iframe class="embed-responsive-item" src="https://www.youtube.com/embed/p3uF3LK5734?rel=0" allowfullscreen="allowfullscreen"&gt;&lt;/iframe&gt;
    &lt;/div&gt;

    &lt;h2&gt;Recipe For Making Soft Idlis&lt;/h2&gt;'''
&gt;&gt;&gt;
&gt;&gt;&gt; soup = BeautifulSoup(html, 'html.parser')
&gt;&gt;&gt; tag = soup.find('h2', text=re.compile('Recipe'))
&gt;&gt;&gt; tag
&lt;h2&gt;Recipe For Making Soft Idlis&lt;/h2&gt;
&gt;&gt;&gt; tag.text
'Recipe For Making Soft Idlis'
</code></pre>
