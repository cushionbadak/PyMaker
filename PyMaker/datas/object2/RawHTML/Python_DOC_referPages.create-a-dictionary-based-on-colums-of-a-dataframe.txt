<div class="post-text" itemprop="text">
<p>I have a dataframe of the form:</p>
<pre><code>pandas as pd
df = pd.DataFrame({'col1': [1,2,3,4,5,6], 
                   'col2': ['A','A','B','C','D','D']}, 
                   index=['row1', 'row2','row3', 'row4','row5', 'row6'])
</code></pre>
<p>I would like to create the following dictionary based on the values of the dataframe df:</p>
<pre><code>dict = {'A':[1,2], 'B':[3], 'C':[4], 'D':[5,6]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Convert values of <code>col1</code> per groups by <a href="http://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.core.groupby.GroupBy.apply.html" rel="nofollow noreferrer"><code>GroupBy.apply</code></a> to lists and then call <a href="http://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.Series.to_dict.html" rel="nofollow noreferrer"><code>Series.to_dict</code></a>:</p>
<pre><code>d = df.groupby('col2')['col1'].apply(list).to_dict()
print (d)
{'A': [1, 2], 'B': [3], 'C': [4], 'D': [5, 6]}
</code></pre>
</div>
