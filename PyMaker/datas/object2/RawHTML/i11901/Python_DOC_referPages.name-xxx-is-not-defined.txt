<div class="post-text" itemprop="text">
<p>In python, it is necessary to use <code>self</code> to access other methods and attributes of the same object. When you simply call <code>view_splash</code>, python looks for the function definition but <em>will not</em> look at the methods of <code>Window</code>. By explicitly prefixing <code>view_splash</code> with <code>self.</code>, python then knows that you want the method <code>Window.view_splash</code> and it should work as you expect.</p>
<p>So for your specific code, this would require you to update your <code>run</code> method to be the following:</p>
<pre><code>def run(self):
    for i in range(len(lines)):
        n = random.choice(words)
        self.view_splash(0)
        self.view_splash(1)
        time.sleep(600)
</code></pre>
<p>I'm assuming that there is additional code <em>outside</em> of the class which defines <code>lines</code> and <code>words</code> as global variables which <code>Window.run</code> can then access.</p>
