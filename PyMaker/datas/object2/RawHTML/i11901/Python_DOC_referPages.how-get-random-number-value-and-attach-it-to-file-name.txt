<div class="post-text" itemprop="text">
<p>You can use string concatenation to create a path to the file of that number. Like this:</p>
<pre><code>path = 'D:/my path/' + str(ran) + '.wav'
wave_obj = sa.WaveObject.from_wave_file(path)
</code></pre>
<p>Here's more information about <a href="http://www.pythonforbeginners.com/concatenation/string-concatenation-and-formatting-in-python" rel="nofollow">string formatting</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>What I might recommend is leveraging a <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow">format string</a>. You can do something like this:</p>
<pre><code>path = "/path/to/my/files/{0}.wav"
random_file = path.format(ran)
</code></pre>
<p>This will be the albsolute path to the randomly selected file, which can then be passed to <code>from_wave_file</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>A possible solution is to create a dictionary of your files:</p>
<pre><code>dic_files = {1 : 'file1.wav', 2: 'file2.wav' ...}
</code></pre>
<p>then you can simply call the file using:</p>
<pre><code>wave_obj = sa.WaveObject.from_wave_file('D:/my path/'+dic_files[ran])
</code></pre>
</div>
