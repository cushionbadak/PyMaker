<div class="post-text" itemprop="text">
<p>You are attempting to match the string <code>a.b.\\[c\\]</code> instead of <code>a.b.[c]</code>.<br/>
Try this:</p>
<pre><code>import re
li= r"a\.b\.\[c\]"
s = "a.b.[c]"
if re.search(li, s, re.IGNORECASE):
    print("Matched")
</code></pre>
<p><code>re.IGNORECASE</code> is not needed in here by the way.</p>
</div>
