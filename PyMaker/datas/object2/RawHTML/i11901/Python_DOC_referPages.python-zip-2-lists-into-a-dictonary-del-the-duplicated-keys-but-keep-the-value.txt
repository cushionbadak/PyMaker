<div class="post-text" itemprop="text">
<p>I'm new to Python. I'm trying to zip 2 lists into a dictionary without losing the values of the duplicated keys and keep values as a list in the dictionary.</p>
<p>Example:</p>
<pre><code>list1 = [0.43, -1.2, 50, -60.5, 50]

list2 = ['tree', 'cat', 'cat', 'tree', 'hat']
</code></pre>
<p>I'm trying to get the following outcome:</p>
<pre><code>{'tree': [0.43, -60.5],'cat': [-1.2, 50],'hat': [50]}
</code></pre>
<p>Thanks for your help.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/stdtypes.html?highlight=setdefault#dict.setdefault" rel="nofollow">setdefault</a> method of the dictionary: </p>
<pre><code>list1 = [0.43, -1.2, 50, -60.5, 50]
list2 = ['tree', 'cat', 'cat', 'tree', 'hat']
d = {}
for k, v in zip(list2, list1):
    d.setdefault(k, []).append(v)
</code></pre>
<p>Results:</p>
<pre><code>&gt;&gt;&gt; d
{'cat': [-1.2, 50], 'hat': [50], 'tree': [0.43, -60.5]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The answer provided by Mike Muller works perfectly well. An alternative, perhaps slightly more pythonic way would be to use <a href="https://docs.python.org/2/library/collections.html#collections.defaultdict" rel="nofollow">defaultdict</a> from the collections library:</p>
<pre><code>from collections import defaultdict

list1 = [0.43, -1.2, 50, -60.5, 50]
list2 = ['tree', 'cat', 'cat', 'tree', 'hat']
d = defaultdict(list)
for k, v in zip(list2, list1):
    d[k].append(v)
</code></pre>
</div>
