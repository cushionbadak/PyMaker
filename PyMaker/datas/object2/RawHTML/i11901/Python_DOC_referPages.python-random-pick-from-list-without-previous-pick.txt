<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/random.html#random.sample" rel="noreferrer">random.sample</a> to select unique elements from a given sequence, like this</p>
<pre><code>import random

Names = ['Andrew', 'John', 'Jacob', 'Bob']

choice = random.sample(Names, 2) # choose 2 unique names from Names

print(choice[0])
print(choice[1])
</code></pre>
