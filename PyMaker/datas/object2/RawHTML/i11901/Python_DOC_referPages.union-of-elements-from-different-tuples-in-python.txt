<div class="post-text" itemprop="text">
<p>I have 3 tuples: </p>
<pre><code>a = ('mary', 'peter', 'james')
b = ('brown_eyes', 'blue_eyes', 'green-eyes')
c = ('blond_hair', 'short_hair', 'dark_hair')
</code></pre>
<p>I need to join the first element of each tuple, the second element of each tuple, the third one... In order to print something like this:</p>
<pre><code>mary brown_eyes blond_hair
peter blue_eyes short_hair
james green_eyes dark_hair 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip</code></a> to do this combination for you. It accepts multiple iterables and will return an iterable where each result contains an element from each of the inputs just as you have described.</p>
<pre><code>d,e,f = zip(a,b,c)
</code></pre>
<p>Here is a complete example using your input data:</p>
<pre><code>a = ('mary', 'peter', 'james')
b = ('brown_eyes', 'blue_eyes', 'green-eyes')
c = ('blond_hair', 'short_hair', 'dark_hair')

d,e,f = zip(a,b,c)
</code></pre>
<p>Then you can print each of these using <code>print</code> and expanding the tuple:</p>
<pre><code>print(*d) # mary brown_eyes blond_hair
</code></pre>
<p>If you want, you can also do this in one line if you don't care about the intermediate values and just want to print the combinations.</p>
<pre><code>_ = [print(*item) for item in zip(a,b,c)]
# mary brown_eyes blond_hair
# peter blue_eyes short_hair
# james green-eyes dark_hair
</code></pre>
</div>

<p>Solution is:</p>
<pre><code>a = ['mary', 'peter', 'james']
b = ['brown_eyes', 'blue_eyes', 'green-eyes']
c = ['blond_hair', 'short_hair', 'dark_hair']

for element in zip(a, b, c):
    print(' '.join(element))
</code></pre>
