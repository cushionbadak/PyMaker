<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/2/library/functions.html#reduce" rel="nofollow"><code>reduce</code></a>, which accepts a function and an iterable, and apply the function cumulatively to the iterable.</p>
<pre><code>&gt;&gt;&gt; from operator import add, sub
&gt;&gt;&gt; from random import choice, randint
&gt;&gt;&gt; ops = {'+':add, '-':sub}                                                       
&gt;&gt;&gt; op = choice('+-')    
&gt;&gt;&gt; op
'+'
&gt;&gt;&gt; nums = [randint(1,10) for i in range(10)]
&gt;&gt;&gt; nums
[7, 8, 1, 4, 10, 7, 10, 7, 10, 1]
&gt;&gt;&gt; reduce(ops[op], nums)
65
</code></pre>
<p><code>reduce(add, [1, 2, 3])</code> =&gt; <code>add(add(1, 2), 3)</code></p>
<hr/>
<p>If you want to use <code>reduce</code> in Python 3.x, import it from <a href="https://docs.python.org/3/library/functools.html#functools.reduce" rel="nofollow"><code>functools</code></a>:</p>
<pre><code>from functools import reduce
</code></pre>
