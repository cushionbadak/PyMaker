<blockquote>
<p>how to get the first day and last day of current month in python</p>
</blockquote>
<p>There's a function in the standard lib <a href="https://docs.python.org/3/library/calendar.html#calendar.monthrange" rel="noreferrer"><code>calendar.monthrange(year, month)</code></a></p>
<pre><code>&gt;&gt;&gt; import calendar
&gt;&gt;&gt; calendar.monthrange(2016, 3)
(1, 31)
</code></pre>
<p><strong>Careful</strong>, <code>monthrange</code> <strong>doesn't</strong> return the dates of first and last days, but returns weekday of first day of the month and number of days in month, for the specified year and month.</p>
<p>So to get first and last date objects out of it:</p>
<pre><code>&gt;&gt;&gt; _, num_days = calendar.monthrange(2016, 3)
&gt;&gt;&gt; first_day = datetime.date(2016, 3, 1)
&gt;&gt;&gt; last_day = datetime.date(2016, 3, num_days)
&gt;&gt;&gt; first_day
datetime.date(2016, 3, 1)
&gt;&gt;&gt; last_day
datetime.date(2016, 3, 31)
</code></pre>
<h2>String format</h2>
<pre><code>&gt;&gt;&gt; first_day.strftime('%d-%m-%Y')
'01-03-2016'
&gt;&gt;&gt; last_day.strftime('%d-%m-%Y')
'31-03-2016'
</code></pre>
