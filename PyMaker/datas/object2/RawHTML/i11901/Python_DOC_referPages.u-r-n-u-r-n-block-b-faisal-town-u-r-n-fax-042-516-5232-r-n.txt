<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> (to map, filter list items) and <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow"><code>str.strip</code></a> (to remove spaces around the strings, including <code>\r</code>, <code>\n</code>):</p>
<pre><code>&gt;&gt;&gt; result = [u'\r\n ', u'\r\n Block-B, Faisal Town', u'\r\n Fax: (042) 516 5232\r\n ']
&gt;&gt;&gt; result = [x.strip() for x in result]
&gt;&gt;&gt; result = [x for x in result if x]
&gt;&gt;&gt; result
[u'Block-B, Faisal Town', u'Fax: (042) 516 5232']
</code></pre>
