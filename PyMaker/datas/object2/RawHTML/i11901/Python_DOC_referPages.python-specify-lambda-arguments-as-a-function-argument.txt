<p>Is there any possibility to specify how many arguments a lambda as a function argument can take?</p>
<p>For example:</p>
<pre><code>def func(k=lambda x:return x**2):
   return k
</code></pre>
<p>Could i specify, if k is not the standard lambda, that k is supposed to take exactly one argument?</p>
<p>Max</p>
</div>
<div class="post-text" itemprop="text">
<p>You could do something like this using <a href="https://docs.python.org/3/library/inspect.html#inspect.getargspec" rel="nofollow"><code>inspect.getargspec</code></a>:</p>
<pre><code>import inspect
def func (k = lambda x: x ** 2):
    if not callable(k) or len(inspect.getargspec(k).args) != 1:
        raise TypeError('k must take exactly one argument.')
    # Do whatever you want
</code></pre>
<p>Note that the above will miserably fail with something like (while it shouldn't):</p>
<pre><code>func (lambda x, y = 8: x + y)
</code></pre>
<p>...so you will need something a bit more complicated if you want to handle this case:</p>
<pre><code>import inspect
def func (k = lambda x: x ** 2):
    if not callable(k):
        raise TypeError('k must be callable.')
    argspec = inspect.getfullargspec(k)
    nargs = len(argspec.args)
    ndeft = 0 if argspec.defaults is None else len(argspec.defaults)
    if nargs != ndeft + 1:
        raise TypeError('k must be callable with one argument.')
    # Do whatever you want
</code></pre>
