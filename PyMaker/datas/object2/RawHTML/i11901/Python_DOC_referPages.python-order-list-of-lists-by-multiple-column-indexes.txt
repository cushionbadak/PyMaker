<p>Let's say I've the following list x in Python</p>
<pre><code>[['a',6,'aa']
['d',7,'bb']]
['c',1,'cc']
['a',4,'dd']
['d',2,'ee']]
</code></pre>
<p>and I want to sort its elements in order to obtain the following result</p>
<pre><code>[['a',4,'dd']
['a',6,'aa']
['c',1,'cc']
['d',2,'ee']
['d',7,'bb']]
</code></pre>
<p>that is I want to sort it by two columns, the first one (the most important) and the second one (the less important). This is probably a duplicate question but I haven't been able to find the solution...</p>

<div class="post-text" itemprop="text">
<p>Since it does not matter whether you sort by the third column as well, or not, you can use plain <code>sort</code> here, and get the same result:</p>
<pre><code>&gt;&gt;&gt; sorted(a)
[['a', 4, 'dd'], ['a', 6, 'aa'], ['c', 1, 'cc'], ['d', 2, 'ee'], ['d', 7, 'bb']]
</code></pre>
<p>This is because lists are compared left to right and sorted in lexicographical order.</p>
<p>If you did want to order by arbitrary column order, you should use <a href="https://docs.python.org/3/library/operator.html#operator.itemgetter" rel="nofollow"><code>operator.itemgetter</code></a>, which is faster than using a <code>lambda</code> function for the <code>key</code>.</p>
<pre><code>&gt;&gt;&gt; import operator
&gt;&gt;&gt; sorted(a, key=operator.itemgetter(1, 0))  # order by column 1 first, then 0.
[['c', 1, 'cc'], ['d', 2, 'ee'], ['a', 4, 'dd'], ['a', 6, 'aa'], ['d', 7, 'bb']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The following sorts list by the first element, then by the second element:</p>
<pre><code>&gt;&gt;&gt; sorted(a, key=lambda x:(x[0], x[1]))
[['a', 4, 'dd'], ['a', 6, 'aa'], ['c', 1, 'cc'], ['d', 2, 'ee'], ['d', 7, 'bb']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can simply use <code>sorted</code> as <code>sorted</code> automatically handles if the sorting is applied from first element to last.</p>
<pre><code>a = [['a',6,'aa'],['d',7,'bb'],['c',1,'cc'],['a',4,'dd'],['d',2,'ee']]
sorted(a)

[['a', 4, 'dd'],
['a', 6, 'aa'],
['c', 1, 'cc'],
['d', 2, 'ee'],
['d', 7, 'bb']]
</code></pre>
