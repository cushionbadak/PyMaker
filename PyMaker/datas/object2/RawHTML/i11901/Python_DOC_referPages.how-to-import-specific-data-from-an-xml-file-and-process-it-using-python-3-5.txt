<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.findall" rel="nofollow"><code>.findall()</code></a> to locate the <code>ROW</code> elements inside <code>RECORD</code> elements and <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.attrib" rel="nofollow"><code>.attrib</code></a> to access the <code>AMOUNT</code> attribute. Complete example:</p>
<pre><code>import xml.etree.ElementTree as ET

data = """&lt;?xml version="1.0" encoding="windows-1252" standalone="yes"?&gt;&lt;!-- Generated by SMExport 4.88--&gt;&lt;ROOT&gt;  &lt;RECORDS&gt;
&lt;METADATA&gt;&lt;FIELDS&gt;&lt;FIELD attrname="SALENUM" fieldtype="i4"/&gt;&lt;FIELD attrname="TIME" fieldtype="time"/&gt;&lt;FIELD attrname="DATE" fieldtype="date"/&gt;&lt;FIELD attrname="AMOUNT" fieldtype="r8" SUBTYPE="Money"/&gt;&lt;/FIELDS&gt;&lt;PARAMS DEFAULT_ORDER="1" PRIMARY_KEY="1" LCID="1033"/&gt;&lt;/METADATA&gt;
  &lt;RECORD&gt;
    &lt;ROW
      SALENUM="1"
      TIME="125108"
      DATE="20160122"
      AMOUNT="22.9"
    /&gt;
  &lt;/RECORD&gt;
  &lt;RECORD&gt;
    &lt;ROW
      SALENUM="2"
      TIME="125243"
      DATE="20160122"
      AMOUNT="22.9"
    /&gt;
  &lt;/RECORD&gt;
&lt;/RECORDS&gt;
&lt;/ROOT&gt;"""

root = ET.fromstring(data)
amounts = [float(row.attrib["AMOUNT"]) for row in root.findall(".//RECORD/ROW")]
print(amounts)
</code></pre>
<p>Prints:</p>
<pre><code>[22.9, 22.9]
</code></pre>
<p>Then, you can use the built-in <code>sum()</code> to add up the amounts: <code>sum(amounts)</code>.</p>
