<div class="post-text" itemprop="text">
<p>You're only asking the number of rows but not the content of rows themselves. Following code shows how to generate the full matrix:</p>
<pre><code>def read_matrix(count):
    list_of_lists = [list(map(int, input('Enter a row: ').split())) for _ in range(count)]

    return list_of_lists

x = input("enter the list of lists of numbers?")
print(read_matrix(int(x)))
</code></pre>
<p>It will first query the user the number of rows and pass that number to <code>read_matrix</code> which will ask the user to input every line.</p>
</div>
<div class="post-text" itemprop="text">
<p>According to the <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow">official documentation</a>,</p>
<blockquote>
<p>The function then <strong>reads a line</strong> from input,
  converts it to a string (stripping a trailing newline)</p>
</blockquote>
<p>Please, note <strong>reads a line</strong>. But here you are trying to read 4 lines with one call to <code>input</code>. You might want to run a loop to take this input.</p>
