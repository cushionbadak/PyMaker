<p>I am using an API, and the API needs this data format:</p>
<pre><code>Wed Jan 07 2015 18:58:40
</code></pre>
<p>How I can convert the time now to this data format, using the <code>datetime</code> and <code>time</code> modules?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>print(datetime.now().strftime('%a %b %d %Y %H:%M:%S'))
</code></pre>
<p>Would display something like:</p>
<pre class="lang-none prettyprint-override"><code>Thu Mar 24 2016 10:09:18
</code></pre>
<p>The <a href="https://docs.python.org/2/library/datetime.html?highlight=datetime.strftime#strftime-and-strptime-behavior" rel="nofollow">formatting options</a> used are as follows:</p>
<ul>
<li><code>%a</code> Weekday as locale’s abbreviated name.</li>
<li><code>%b</code> Month as locale’s abbreviated name.</li>
<li><code>%d</code> Day of the month as a zero-padded decimal number.</li>
<li><code>%Y</code> Year with century as a decimal number.</li>
<li><code>%H</code> Hour (24-hour clock) as a zero-padded decimal number.</li>
<li><code>%M</code> Minute as a zero-padded decimal number.</li>
<li><code>%S</code> Second as a zero-padded decimal number.</li>
</ul>
<hr/>
<p>To then convert this to a format for sending, you probably want to investigate <a href="https://docs.python.org/3/library/urllib.parse.html?highlight=quote_plus#urllib.parse.quote_plus" rel="nofollow"><code>quote_plus()</code></a>, for example:</p>
<pre><code>from datetime import datetime
import urllib

now = datetime.now().strftime('%a %b %d %Y %H:%M:%S')
print(urllib.parse.quote_plus(now))
</code></pre>
<p>This would give you:</p>
<pre class="lang-none prettyprint-override"><code>Thu+Mar+24+2016+10%3A32%3A51
</code></pre>
