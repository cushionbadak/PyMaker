<h1>Answer to the question</h1>
<p>Don't use the name <code>str</code> for your string variable. It will mask the built-in <code>str</code>:</p>
<pre><code>intab = "0123456789"
outtab = intab[::-1]
trantab = str.maketrans(intab, outtab)
mystring = "p1y2t3h4o5n6"
print(mystring.translate(trantab))
</code></pre>
<p>Output:</p>
<pre><code>p8y7t6h5o4n3
</code></pre>
<p>Your <code>intab</code> and <code>outtab</code> must have the same length. Most functions from <code>string</code> in Python 2 became methods of <code>str</code> for a while in Python 2 and were dropped as functions from <code>string</code> in Python 3. So use <code>str.maketrans()</code>.</p>
<h1>Solving the problem</h1>
<p>If want to remove characters from a string you can do:</p>
<pre><code>remove = set("0123456789")
mystring = "p1y2t3h4o5n6"
print(''.join(x for x in mystring if x not in remove))
</code></pre>
<p>Output:</p>
<pre><code>python
</code></pre>
<p>If you want to remove numbers you can also do:</p>
<pre><code>print(''.join(x for x in mystring if not x.isdigit()))
</code></pre>
<p>Output:</p>
<pre><code>python
</code></pre>

<div class="post-text" itemprop="text">
<p>As already mentioned in from Python 3.x you need to use the static method <a href="https://docs.python.org/3/library/stdtypes.html#str.maketrans" rel="nofollow"><code>str.maketrans</code></a>. That being said the <code>intab</code> and <code>outtab</code> must have equal length.</p>
<pre><code>&gt;&gt;&gt; str = "p1y2t3h4o5n6"
&gt;&gt;&gt; intab = "0123456789"
&gt;&gt;&gt; outtab = " "
&gt;&gt;&gt; trantab = str.maketrans(intab, outtab * len(intab))
&gt;&gt;&gt; print(mystring.translate(trantab).replace(" ", ""))
python
</code></pre>
<p>But you can also do this using regular expression </p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; re.sub(r'[0-9]', '', mystring)
'python'
</code></pre>
