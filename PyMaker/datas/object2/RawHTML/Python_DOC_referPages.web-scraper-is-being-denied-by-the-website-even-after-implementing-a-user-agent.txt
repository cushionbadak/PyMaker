<div class="post-text" itemprop="text">
<p>I believe the first parameter on the 'findAll' method won't help you, so the issue might have nothing to do with the HTTP request-response cycle.</p>
<p>I queried the url you're using and all possible attributes of all 'td' elements on the document are:</p>
<pre><code>{'class': ['xml-file-size', 'file-20']}
{'class': ['xml-file-type', 'file-20']}
{'colspan': '2', 'class': ['xml-file-type', 'file-20']}
{'rowspan': '3'}
{'colspan': '2'}
{}
</code></pre>
<p>Which makes querying for 'rowspan' of 1 and 'class' 'x-hidden-focus' return the empty list.</p>
<p>Try on the second to last line:</p>
<pre><code>containers = page_soup.findAll('td', {'colspan'='1', 'class':'file-20'})
</code></pre>
<p>or:</p>
<pre><code>containers = page_soup.findAll('td', {'rowspan': '3'})
</code></pre>
<p>or even just:</p>
<pre><code>containers = page_soup.findAll('td')
</code></pre>
<p>Is up to you which specific 'td' elements you're looking for.</p>
<p>Check out the <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/#a-function" rel="nofollow noreferrer">documentation</a> also to learn about more ways to use BeautifulSoup, including passing functions as arguments, etc.</p>
