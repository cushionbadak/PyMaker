<div class="post-text" itemprop="text">
<p>I have a dockerfile and I want to run every python script in a folder without naming each one, just run every python script in that folder</p>
<p>how can I do so?</p>
</div>
<div class="post-text" itemprop="text">
<p><strong>Dockerfile</strong></p>
<pre><code>FROM python:3.7.2-alpine3.8

WORKDIR /opt

COPY . /opt

RUN find . -name "*.py" | xargs -n 1 python3
</code></pre>
<p><strong>Python files</strong></p>
<pre><code>#!/usr/local/bin/python3

print("1")
</code></pre>
<p><strong>Command</strong></p>
<pre><code>docker build -t temp .
</code></pre>
<p><strong>Output</strong></p>
<pre><code>....
Step 4/5 : RUN find . -name "*.py" | xargs -n 1 python3
---&gt; Running in 68326b65cdda
1
2
</code></pre>
<p>1 and 2 are the result of 1.py and 2.py</p>
