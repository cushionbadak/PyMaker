<div class="post-text" itemprop="text">
<p>You're replacing the ampersand with an ampersand plus a space. </p>
<p>Use this:</p>
<pre><code>Python&gt; "some &amp; text".replace("&amp;", ",")
</code></pre>
<p>See <a href="https://docs.python.org/3/library/stdtypes.html?highlight=replace#str.replace" rel="nofollow">the docs</a>. </p>
<p><em>Return a copy of the string with all occurrences of substring old replaced by new. If the optional argument count is given, only the first count occurrences are replaced.</em></p>
</div>

<p>Your <code>.replace()</code> parameters are wrong. Try this to replace "&amp;" and also "&amp; ".</p>
<pre><code>commafile.write(my_file_contents.replace('&amp;', ','))
</code></pre>
