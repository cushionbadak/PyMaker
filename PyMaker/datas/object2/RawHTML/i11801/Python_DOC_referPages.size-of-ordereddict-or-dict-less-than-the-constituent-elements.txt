<div class="post-text" itemprop="text">
<p>By the <a href="https://docs.python.org/3/library/sys.html#sys.getsizeof" rel="nofollow"><code>sys.getsizeof()</code></a> documentation you can see it specifically says:</p>
<blockquote>
<p>Only the memory consumption directly attributed to the object is accounted for, not the memory consumption of objects it refers to.</p>
</blockquote>
<p>Meaning, the items inside the dictionary are not counted as part of the size, only the dictionary object itself.<br/>
It also links to a recursive sizeof <a href="http://code.activestate.com/recipes/577504/" rel="nofollow">recipe</a> which will include the items in the dictionary.</p>
</div>
<span class="comment-copy">A dictionary, like most Python data structures, just holds <i>references</i> to the elements it contains.</span>
