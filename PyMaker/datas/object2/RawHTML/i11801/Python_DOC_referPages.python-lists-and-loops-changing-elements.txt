<p>I have created a while-loop (below) that accesses each element of the list and prints its square. Now, how would I change this program so that it replaces each element by it's square.  For example: If x = [2,4,2,6,8,10], then x would be changed to x = [4,16,4,36,4,64,100]</p>
<pre><code>    print("Enter any into the list: ")
    x = eval(input())
    n=0
    while n &lt; len(x):
        print("The square of", x[n], "is", x[n]**2)
        n += 1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can set it in the <code>while</code> loop:</p>
<pre><code>print("Enter any into the list: ")
x = eval(input())
n=0
while n &lt; len(x):
    print("The square of", x[n], "is", x[n]**2)
    x[n] = x[n] ** 2
    n += 1
</code></pre>
<p>It isn't a good idea to use <code>eval()</code>, though.  You should use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow"><code>ast.literal_eval()</code></a>:</p>
<pre><code>import ast

print("Enter any into the list: ")
x = ast.literal_eval(input())
...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You would almost do the same thing, except with a <code>for</code> loop:</p>
<pre><code>for i in range(0, len(x)):   # x must be a list
    x[i] **= 2   
</code></pre>
<p>You could also set it in the <code>while</code> loop:</p>
<pre><code>print("Enter any into the list: ")
x = eval(input())
n=0
while n &lt; len(x):
    print("The square of", x[n], "is", x[n]**2)
    x[n] **= 2
    n += 1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>x = [n**2 for n in x]
</code></pre>
<p>List comprehensions are your friends.</p>
