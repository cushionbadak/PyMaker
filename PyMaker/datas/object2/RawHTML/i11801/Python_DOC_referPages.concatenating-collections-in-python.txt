<p>You misread the <a href="https://stackoverflow.com/questions/952914/making-a-flat-list-out-of-list-of-lists-in-python/953097#953097">answer you referenced</a></p>
<p>In that answer was this line:</p>
<pre><code>merged = list(itertools.chain(*list2d))
</code></pre>
<p>Note the use of the asterisk (or 'splat') operator.</p>
<p>In your question, there is no such operator:</p>
<pre><code>&gt;&gt;&gt; list(itertools.chain([[1,2,3],[4,5,6]]))  
[[1, 2, 3], [4, 5, 6]]
</code></pre>
<p>To make sense of the answer you reference, try this:</p>
<pre><code>&gt;&gt;&gt; list(itertools.chain(*[[1,2,3],[4,5,6]]))
[1, 2, 3, 4, 5, 6]
</code></pre>
<p>Which is equivalent to removing the outer-most brackets:</p>
<pre><code>&gt;&gt;&gt; list(itertools.chain([1,2,3],[4,5,6]))
[1, 2, 3, 4, 5, 6]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The fist argument is a list of iterables (tuples) not <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow">one iterable</a>. You can use <code>chain.from_iterable()</code> to flatten your list:</p>
<pre><code>&gt;&gt;&gt; from itertools import chain
&gt;&gt;&gt; list(chain.from_iterable([(1,2,3),(4,5,6)]))
[1, 2, 3, 4, 5, 6]
</code></pre>
