<div class="post-text" itemprop="text">
<p><code>str</code> has a <a href="https://docs.python.org/3/library/stdtypes.html#str.join"><code>join</code></a> built-in to handle this:</p>
<pre><code>', '.join(str(_) for _ in x)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can try this:</p>
<pre><code>res = ', '.join(map(str, res)) 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You should use <code>.join</code>:</p>
<pre><code>x = [1,2,3,4,5]
s = ', '.join(str(i) for i in x)
</code></pre>
<p><code>str</code> is used because <code>join</code> can only connect elements that are strings.</p>
<p>Output:</p>
<pre><code>&gt;&gt;&gt; s
'1, 2, 3, 4, 5'
</code></pre>
