<div class="post-text" itemprop="text">
<p>I have several files in my folder <code>xyz_driver</code>, <code>xyz_tool</code>, <code>xyz_mac</code>, <code>aaa_driver</code>, <code>aaa_tool</code>, <code>aaa_mac</code></p>
<p>If I were to do </p>
<pre><code>for filename in glob.glob('xyz_driver'):
    os.remove(filename)
</code></pre>
<p>it works.</p>
<p>But how do I delete all files <strong>only</strong> if the name begins with <code>xyz</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/glob.html#module-glob"><code>glob()</code></a> supports wildcard pattern matching:</p>
<pre><code>for filename in glob.glob('xyz*'):
    os.remove(filename)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>for filename in glob.glob('xyz_driver'):
  if filename.startswith('xyz'):
    os.remove(filename)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can try this</p>
<pre><code>for filename in os.listdir():
    if filename[:3] == 'xyz':
        os.remove(filename)
</code></pre>
<p>EDIT:</p>
<p>As zondo implied, this might give you an error if you use python2.x, the correct syntax is then <code>os.listdir(path)</code> where path is the folder in which you wish to list content.</p>
</div>
