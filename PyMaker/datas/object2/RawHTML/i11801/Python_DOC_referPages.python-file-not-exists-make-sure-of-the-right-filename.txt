<p><code>python trailrun.py datan 21</code> has only 2 arguments but the script has:</p>
<pre><code>delimeter = sys.argv[3]
</code></pre>
<p>If you check the error I believe that you'll see an <code>IndexError</code> because you're trying to access an index that doesn't exist in <code>sys.argv</code>. You should supply the correct amount of arguments or try to handle cases when a third argument isn't provided to the script.</p>
<p>Also, check out <a href="https://docs.python.org/3/library/argparse.html" rel="nofollow">ArgParse</a>. It's the Python provided API for creating command line arguments in scripts.</p>
