<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/2/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> to build a new list with the changes applied to all values over 20:</p>
<pre><code>list_1 = [v - 5 if v &gt; 20 else v for v in list_1]
</code></pre>
<p>The expression <code>v - 5 if v &gt; 20 else v</code> uses a <a href="https://docs.python.org/2/reference/expressions.html#conditional-expressions" rel="nofollow"><em>conditional expression</em></a> to produce either <code>v</code> itself, or <code>v</code> minus 5 subtracted if it is greater than 20.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; list_1 = [20, 19, 24, -10, 100]
&gt;&gt;&gt; [v - 5 if v &gt; 20 else v for v in list_1]
[20, 19, 19, -10, 95]
</code></pre>
<p>You don't really need to test beforehand; a test to see if there are any values over 20 in the list would require a full traverse of the list <em>anyway</em>; the list comprehension would produce a copy of the list with no changes made if there are no values over 20.</p>
<p>If you <em>must</em> test, use the <a href="https://docs.python.org/2/library/functions.html#any" rel="nofollow"><code>any()</code> function</a> with a <a href="https://docs.python.org/2/tutorial/classes.html#generator-expressions" rel="nofollow">generator expression</a>; this at least then aborts iteration when one such value is found:</p>
<pre><code>if any(v &gt; 20 for v in list_1):
    # there is at least one value in list_1 that is greater than 20
    list_1 = [v - 5 if v &gt; 20 else v for v in list_1]
</code></pre>
</div>
