<div class="post-text" itemprop="text">
<p>You need to replace each <code>\</code> with <code>\\</code>. Also, you should use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>str.format()</code></a> instead of <code>%s%</code>:</p>
<pre><code>import os
tmppath = "C:\\$SB52EF.tmpUsers\\{}\\AppData\\Local\\Temp".format(os.getlogin())
</code></pre>
<p>You might be interested in this <a href="https://stackoverflow.com/questions/2953834/windows-path-in-python">question</a>.</p>
