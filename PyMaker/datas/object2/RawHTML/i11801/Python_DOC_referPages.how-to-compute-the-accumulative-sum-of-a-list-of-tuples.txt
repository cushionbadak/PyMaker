<p>I have this list of tuples and want to make a new list, with the accumulative sum of the previous list indexes:</p>
<pre><code>List = [(1.0, 1.0), (3.0, 3.0), (5.0, 5.0)]

newList = [(1.0, 1.0), (4.0, 4.0), (9.0, 9.0)]
</code></pre>
<p>I'm using:</p>
<pre><code>l1 = []
for j in l: #already a given list
    result = tuple(map(sum, zip(j, j+1)))
    #or
    result = (map(operator.add, j, j+1,))
    l1.append(result)
</code></pre>
<p>both cases (<code>zip</code> and <code>operator</code>) returns </p>
<blockquote>
<p>"TypeError: can only concatenate tuple (not "int") to tuple "</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.accumulate" rel="nofollow"><code>itertools.accumulate</code></a>:</p>
<pre><code>&gt;&gt;&gt; import itertools
&gt;&gt;&gt; itertools.accumulate([1, 3, 5])
&lt;itertools.accumulate object at 0x7f90cf33b188&gt;
&gt;&gt;&gt; list(_)
[1, 4, 9]
</code></pre>
<p>It accepts an optional <code>func</code> which is used to do addition:</p>
<pre><code>&gt;&gt;&gt; lst = [(1.0, 1.0), (3.0, 3.0), (5.0, 5.0)]
&gt;&gt;&gt; import itertools
&gt;&gt;&gt; list(itertools.accumulate(lst, lambda a, b: tuple(map(sum, zip(a, b)))))
[(1.0, 1.0), (4.0, 4.0), (9.0, 9.0)]
</code></pre>
<hr/>
<p><code>itertools.accumulate</code> is introduced in Python 3.2. If you use lower version, use the following <code>accumulate</code> (came from the function documentation):</p>
<pre><code>import operator
def accumulate(iterable, func=operator.add):
    it = iter(iterable)
    try:
        total = next(it)
    except StopIteration:
        return
    yield total
    for element in it:
        total = func(total, element)
        yield total
</code></pre>
