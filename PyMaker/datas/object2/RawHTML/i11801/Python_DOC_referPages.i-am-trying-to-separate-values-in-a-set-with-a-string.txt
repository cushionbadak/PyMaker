<div class="post-text" itemprop="text">
<p>You could just use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join</code></a> method in order to create a string of an iterable of strings with specified separator between items:</p>
<pre><code>In [58]: s = {10, 12, 14, 15, 18, 20, 25, 28, 29, 30}

In [59]: '**'.join(map(str, s))
Out[59]: '10**12**14**15**18**20**25**28**29**30'
</code></pre>
<p>Note, that <code>str.join</code> accepts an iterable of strings, so you have to convert each number to string using <code>map(str, s)</code>.</p>
<p>In order to add <code>**</code> at the end of the string, you could just use <code>+</code> as string concatenation operator:</p>
<pre><code>In [60]: '**'.join(map(str, s)) + '**'
Out[60]: '10**12**14**15**18**20**25**28**29**30**'
</code></pre>
<p>You should also note, that the order of <code>set</code> elements is <strong>not specified</strong>. In order to print the elements in the ascending order, you should manually sort them:</p>
<pre><code>In [61]: '**'.join(map(str, sorted(s))) + '**'
Out[61]: '10**12**14**15**18**20**25**28**29**30**'
</code></pre>
</div>

<p>What I get is that you're trying to print a delimited sequence. You can use the str join method and the map function to solve this. </p>
<pre><code>&gt;&gt;&gt; print("**".join(map(str, random_set)) + "**")
12**16**17**18**20**21**22**23**24**28**
</code></pre>
<p>Map is converting each element inside of your set to a string equivalent value. The join method returns a string where the elements of a sequence have been joined together with some str separator. </p>
<p>You can use <code>+</code> to add the final <code>**</code> to the end of your string.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could use a for loop that iterates through random_set and then appends all of the items into a list, then print the list joined with <code>'**'</code>:</p>
<pre><code>list = []
for i in random_set:
    list.append(str(i))
print('**'.join(list)
</code></pre>
<p>This would print it in the format you want.</p>
