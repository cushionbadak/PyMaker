<p>I have different variables of an unknown type (in ABAQUS, it says "Sequence") and want to combine them through a loop:</p>
<pre><code>a = [[unknown type], [unknown type], ...]
x = []
for i in a:
    x.append(i)
</code></pre>
<p>The problem now is that when I initialize x with = [] I get the error message</p>
<blockquote>
<p>TypeError: Can only concatenate list (not "Sequence") to list.</p>
</blockquote>
<p>Is there another (simple/efficient) way, e.g. to automatically create x in the first loop?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>x = [v for v in a]
</code></pre>
