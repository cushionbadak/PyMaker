<p>If i try to give a function an optional argument, this way it won't work:</p>
<pre><code>def speed(self, max, now):
    self.min = 0
    self.max = max
    if now != None:
        self.now = now
    else:
        self.now = 0
    return self.max, self.now
</code></pre>
<p>Can someone help me to understand this and how to do it better?
Do i really need the if statement? Maybe there is an easier and faster way with some kind of special argument?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/2/tutorial/controlflow.html#default-argument-values" rel="nofollow">default argument value</a> to set <code>now</code> to <code>0</code> when omitted:</p>
<pre><code>def speed(self, max, now=0):
    self.min = 0
    self.max = max
    self.now = now
    return self.max, self.now
</code></pre>
<p>Now the caller can simply omit specifying a value for the <code>now</code> argument:</p>
<pre><code>some_instance.speed(42)
</code></pre>
<p>would leave <code>now</code> set to <code>0</code>.</p>
</div>
<span class="comment-copy">It's in a very early step of the official tutorial: <a href="https://docs.python.org/3/tutorial/controlflow.html#default-argument-values" rel="nofollow noreferrer">Default Argument Values</a>.</span>
