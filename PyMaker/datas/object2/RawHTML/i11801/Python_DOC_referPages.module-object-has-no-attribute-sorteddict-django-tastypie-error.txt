<div class="post-text" itemprop="text">
<p>Well, from Django 1.9 onwards, <code>SortedDict</code> has been removed. Check this <a href="https://code.djangoproject.com/wiki/SortedDict" rel="nofollow">link</a> for reference.</p>
<blockquote>
<p><a href="https://code.djangoproject.com/wiki/SortedDict" rel="nofollow"><code>SortedDict</code></a> is deprecated as of Django 1.7 and will be removed in Django 1.9. Use ​<a href="http://docs.python.org/3/library/collections.html#collections.OrderedDict" rel="nofollow"><code>collections.OrderedDict</code></a> instead. Available in Python 2.7 and 3.1+</p>
</blockquote>
<p>You can replace <code>SortedDict</code> with <code>​collections.OrderedDict</code> as mentioned in the link by making changes to the library code and some other changes as given in this Pull Request submitted <a href="https://github.com/wlanslovenija/django-tastypie-mongoengine/pull/94/files" rel="nofollow">here</a>. But fair <strong>warning</strong>, this might not work as the Pull Request was not accepted so far and has failed the build test.</p>
<p>Another option is to downgrade Django to either version 1.8 or 1.7 till django-tastypie-mongoengine can release a stable version which will work for Django 1.9.</p>
