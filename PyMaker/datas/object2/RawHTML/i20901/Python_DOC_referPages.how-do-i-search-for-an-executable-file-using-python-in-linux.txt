<p>How do I search for an executable file using python in linux? The executable files have no extensions and are in a folder together with files that have different extensions. Thanks</p>
<p>EDIT: What I mean by search is to get the filenames of all the executable files and store them in a list or tuple. Thanks</p>

<div class="post-text" itemprop="text">
<p>To do it in Python:</p>
<pre><code>import os
import stat

executable = stat.S_IEXEC | stat.S_IXGRP | stat.S_IXOTH
for filename in os.listdir('.'):
    if os.path.isfile(filename):
        st = os.stat(filename)
        mode = st.st_mode
        if mode &amp; executable:
            print(filename,oct(mode))
</code></pre>

<p>If by search you mean to list all the executable files in a directory than use the command from this <a href="https://superuser.com/q/38981/96916">SuperUser Link</a>.You can use <a href="http://docs.python.org/library/subprocess.html" rel="nofollow noreferrer">Subprocess module</a> to execute the commands from python code.</p>
<pre><code>import shlex   
executables = shlex.split(r'find /dir/mydir -executable -type f')
output_process = subprocess.Popen(executables,shell=True,stdout=subprocess.PIPE)
</code></pre>

<div class="post-text" itemprop="text">
<p>Function <a href="https://docs.python.org/3/library/os.html#os.access" rel="nofollow">os.access()</a> is in some cases better than <a href="https://docs.python.org/3/library/os.html#os.stat" rel="nofollow">os.stat()</a>, because it check whether the file can be executed by <em>you</em>, according to file owner, group and permissions.</p>
<pre><code>import os

for filename in os.listdir('.'):
    if os.path.isfile(filename) and os.access(filename, os.X_OK):
        print(filename)
</code></pre>
