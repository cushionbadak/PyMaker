<div class="post-text" itemprop="text">
<p>For starters you don't need the first and last slash python automatically uses them, secondly you need to make your expression a re object</p>
<pre><code>import re
expr = re.compile(r'((([A-Za-z]{3,9}:(?:\/\/)?)(?:[-;:&amp;=\+\$,\w]+@)?[A-Za-z0-9.-]+(:[0-9]+)?|(?:www.|[-;:&amp;=\+\$,\w]+@)[A-Za-z0-9.-]+)((?:\/[\+~%\/.\w-]*)?\??(?:[-\+=&amp;;%@.\w]*)#?(?:[\w]*))?)')
</code></pre>
<p>after that you can use the function match to see if the expression returns a valid instance</p>
<pre><code>url="www.google.com"

if expr.match(url):
    print("It is valid")
</code></pre>
</div>
