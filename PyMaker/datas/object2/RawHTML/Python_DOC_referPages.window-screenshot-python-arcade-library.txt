<div class="post-text" itemprop="text">
<p>How does one take a screenshot of the current window using the Python Arcade library?</p>
<p>I have built a simulator using the Python Arcade library and would like to take a screenshot of the window programmatically when interesting events occur.  Is this possible?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>arcade.draw_commands.get_image(x=0, y=0, width=None, height=None)
</code></pre>
<p>Get an image from the screen. You can save the image like:</p>
<pre><code>image = get_image()
image.save('screenshot.png', 'PNG')
</code></pre>
<p><a href="http://arcade.academy/arcade.html?highlight=screenshot#arcade.draw_commands.get_image" rel="nofollow noreferrer">Source</a></p>
