<div class="post-text" itemprop="text">
<p>You need to pick a top-level name for your module, then arrange your code like this:</p>
<pre><code>Home
  script_name.py
  somename
    Common.py
    Proj1
      Scripts1.py
    Proj2
      Scripts2.py
</code></pre>
<p>Then use <code>import somename</code> and use <code>somename.Common</code> within the module itself and <code>somename.Proj1.Scripts1</code> etc. Also, you probably want to move <code>script_name.py</code> outside of the module directory itself as shown.</p>
