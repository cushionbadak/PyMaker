<p>Your function contains wrong expression:<br/> <code>if sle in re.match('[3-4][0-9]{4}',sle):</code> - it tries to find a column value <code>sle</code> IN <a href="https://docs.python.org/3/library/re.html#match-objects" rel="nofollow noreferrer">match object</a> which "always have a boolean value of <code>True</code>" (<code>re.match</code> returns <code>None</code> when there's no match)</p>
<hr/>
<p>I would suggest to proceed with <a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.Series.str.isnumeric.html" rel="nofollow noreferrer"><code>pd.Series.str.isnumeric</code></a> function:</p>
<pre><code>In [544]: df
Out[544]: 
  Selection No.
0         37001
1         37002
2         37003
3         asnsh
4         37004
5         singh
6         37005

In [545]: df['Status'] = df['Selection No.'].str.isnumeric().astype(int)

In [546]: df
Out[546]: 
  Selection No.  Status
0         37001       1
1         37002       1
2         37003       1
3         asnsh       0
4         37004       1
5         singh       0
6         37005       1
</code></pre>
<hr/>
<p>If a strict regex pattern is required - use <a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.Series.str.contains.html#pandas.Series.str.contains" rel="nofollow noreferrer"><code>pd.Series.str.contains</code></a> function:</p>
<pre><code>df['Status'] = df['Selection No.'].str.contains('^[3-4][0-9]{4}$', regex=True).astype(int)
</code></pre>
