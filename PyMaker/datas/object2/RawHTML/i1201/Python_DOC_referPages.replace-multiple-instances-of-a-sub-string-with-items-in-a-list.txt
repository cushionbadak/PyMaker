<pre><code>e = "how are you how do you how are they how"
</code></pre>
<p>My expected output is:</p>
<pre><code>out = "how1 are you how2 do you how3 are they how4"
</code></pre>
<p>I'm trying in the following way:</p>
<pre><code>def givs(y,x):
    tt = []
    va = [i+1 for i in list(range(y.count(x)))]
    for i in va:
        tt.append(x+str(i))
    return tt

ls = givs(e, 'how')

ls = ['how1', 'how2', 'how3', 'how4']

fg = []
for i in e.split(' '):
    fg.append(i)

fg = ['how', 'are', 'you', 'how', 'do', 'you', 'how', 'are', 'they', 'how']
</code></pre>
<p>For every instance of 'how' in 'fg' I want to replace with items in 'ls' and finally use join function to get the required output. </p>
<pre><code>expected_output = ['how1', 'are', 'you', 'how2', 'do', 'you', 'how3', 'are', 
                  'they', 'how4']
</code></pre>
<p>so that I can join the items by:</p>
<pre><code>' '.join(expected_output)
</code></pre>
<p>to get:</p>
<pre><code>out = "how1 are you how2 do you how3 are they how4"
</code></pre>

<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/itertools.html#itertools.count" rel="nofollow noreferrer">itertools.count</a>:</p>
<pre><code>from itertools import count

counter = count(1)

e = "how are you how do you how are they how"

result = ' '.join([w if w != "how" else w + str(next(counter)) for w in e.split()])

print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>how1 are you how2 do you how3 are they how4
</code></pre>

<p>There is no need to make your code complex, just add a counter and add it to every "how". At the end make the new string.</p>
<pre><code>e = "how are you how do you how are they how"
e_ok = ""
count = 1
for i in e.split():
    if i == "how":
        i = i+str(count)
        count += 1
    e_ok += i + " "
print(e_ok)
</code></pre>
