<div class="post-text" itemprop="text">
<p>You probably need to use <code>elif</code>. The <code>else</code> in your case is executed because <code>args.url</code> is empty. </p>
<p><strong>Ex:</strong></p>
<pre><code>if args.ip:
    print("~ IP Address is " + args.ip)
elif args.url:
    print("~ URL is " + args.url)
else:
    print("No Argument Provided")
</code></pre>
</div>
<span class="comment-copy">'else' refers to the last <code>if</code>, so if you provide an IP but no URL the else-part is executed.</span>
<span class="comment-copy">Rakesh 's answer is right but besides that, there is a more elegant solution for the underneath problem, see: <a href="https://docs.python.org/3/library/argparse.html#mutual-exclusion" rel="nofollow noreferrer">docs.python.org/3/library/argparse.html#mutual-exclusion</a></span>
