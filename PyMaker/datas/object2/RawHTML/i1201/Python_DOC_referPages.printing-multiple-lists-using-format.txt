<div class="post-text" itemprop="text">
<p>I am having 2 lists with variable sizes that need to be printed alongside each each other. For instance, if </p>
<pre><code>A = [30, 40, 50]
B = [1,2,3]
</code></pre>
<p>Then, I want to print an output that looks like:</p>
<pre><code>A 30 B 1 A 40 B 2 A 50 B 3
</code></pre>
<p>I have tried something similar to </p>
<pre><code>print (len(A)* ('A {} B {}').format(*A,*B) 
</code></pre>
<p>but this does not give me what I am looking for.</p>
<p>Any help would be appreciated.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip</a> both lists:</p>
<pre><code>A = [30, 40, 50]
B = [1,2,3]

result = ' '.join('A {} B {}'.format(a, b) for a, b in zip(A, B))
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>A 30 B 1 A 40 B 2 A 50 B 3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using a simple iteration with <code>enumerate</code></p>
<p><strong>Ex:</strong></p>
<pre><code>A = [30, 40, 50]
B = [1,2,3]

print(" ".join("A {} B {}".format(v, B[i]) for i,v in enumerate(A)))
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>A 30 B 1 A 40 B 2 A 50 B 3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>One of the possible solution:</p>
<pre><code>print(*map(lambda x, y: "A {} B {}".format(x, y), A, B))
</code></pre>
<p><code>lambda x, y</code> - lambda function takes 2 arguments (each element from <code>A</code> and <code>B</code>) and create a data structure called <code>map</code> (similar to list). After this, we treat <code>map</code>-like data structure as a regular <code>list</code> and print each specific element.</p>
<p><strong>Input</strong></p>
<pre><code>A = [1, 2, 3]
B = [4, 5, 6]
</code></pre>
<p><strong>Intermediate  result</strong></p>
<p><em>For explanation purpose, exclusively.</em></p>
<pre><code>print(*map(["A 1 B 4", "A 2 B 5", "A 3 B 6"]))
</code></pre>
<p><strong>Output</strong></p>
<pre><code>A 1 B 4 A 2 B 5 A 3 B 6
</code></pre>
<p>In case when one of the arrays will be shorter, it will return the following:</p>
<p><strong>Input</strong></p>
<pre><code>A = [1, 2]
B = [4, 5, 6]
</code></pre>
<p><strong>Output</strong></p>
<pre><code>A 1 B 4 A 2 B 5
</code></pre>
<p>For more details, take a look on <a href="https://docs.python.org/3.5/library/functions.html#map" rel="nofollow noreferrer">map</a>.</p>
