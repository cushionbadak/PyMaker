<p>How to use Python to handle scientific notation and perform calculations?
e.g:</p>
<pre><code>a='1e+03'
b='1e+02'
</code></pre>
<p>How to sum a and b and get result of 1.1+03</p>
</div>
<div class="post-text" itemprop="text">
<p>Python's built-in <a href="https://docs.python.org/3/library/functions.html#float" rel="nofollow noreferrer"><code>float</code></a> supports scientific notation:</p>
<pre><code>a = '1e+03'
b = '1e+02'

res = float(a) + float(b)  # 1100.0
print int(res)             # 1100
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>c = '{0:0.2e}'.format(float(a)+float(b))

In[10]: c
Out[10]: '1.10e+03'
</code></pre>
</div>
