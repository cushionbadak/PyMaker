<div class="post-text" itemprop="text">
<p>I am trying to write the code to output the index of each location that a word occurs in a string that has no spaces. </p>
<p>Specifically the gene: 'act' within the sequence: 'avdgagtactsdfactactfgactlkact'. </p>
<p>The output should be:</p>
<p>Locations of gene: [7, 13, 16, 21, 26]'</p>
<p>Please help</p>
</div>
<div class="post-text" itemprop="text">
<p>Regular expression match object has the handy <a href="https://docs.python.org/3/library/re.html#re.Match.start" rel="nofollow noreferrer"><code>.start()</code></a> method:</p>
<pre><code>In [1]: import re

In [2]: s = "avdgagtactsdfactactfgactlkact"

In [3]: [m.start() for m in re.finditer("act", s)]
Out[3]: [7, 13, 16, 21, 26]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>One alternative could be to use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>sequence = 'avdgagtactsdfactactfgactlkact'
gene = 'act'

result = [i for i in range(len(sequence) - len(gene) + 1) if sequence[i:].startswith(gene)]
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[7, 13, 16, 21, 26]
</code></pre>
</div>

<p>An answer with regular expressions:</p>
<pre><code>import re

gene = 'act'
sequence = 'avdgagtactsdfactactfgactlkact'

[m.start() for m in re.finditer(gene, sequence)]
</code></pre>
