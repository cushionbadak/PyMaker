<p>I need to find a string in a file and reformat it.</p>
<p>String to find format:</p>
<pre><code>[title](link)
</code></pre>
<p>example:</p>
<pre><code>[template application](https://stackoverflow.com/sample/base-app)
</code></pre>
<p>I would like to change that to HTML link:</p>
<pre><code>&lt;a href="link"&gt;title&lt;/a&gt; 
</code></pre>
<p>example:</p>
<pre><code>&lt;a href="https://stackoverflow.com/sample/base-app"&gt;template application&lt;/a&gt; 
</code></pre>
<p>What is the best way to do it?
I am thinking regular expressions but I have no clue to how achieve that. Is there a simple way?</p>

<div class="post-text" itemprop="text">
<p>You  could take advantage of the fact that the <a href="https://docs.python.org/3/library/re.html#re.sub" rel="nofollow noreferrer">sub</a> function receives another function as a parameter for replacement:</p>
<pre><code>import re

line = '[template application](https://stackoverflow.com/sample/base-app)'


def repl(match):
    return '&lt;a href="{}"&gt;{}&lt;/a&gt;'.format(match.group(2), match.group(1))


result = re.sub('\[(.+?)\]\((https?.+?)\)', repl, line)

print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>&lt;a href="https://stackoverflow.com/sample/base-app"&gt;template application&lt;/a&gt;
</code></pre>
<p>The pattern <code>'\[(.+?)\]\((https?.+?)\)'</code> captures everything between brackets followed by something link like (starts with http), notice that you must escape brackets and parenthesis because they have a special meaning inside a regex.</p>
<p>Or as suggested by @JonClements, you could use:</p>
<pre><code>re.sub('\[(.+?)\]\((https?.+?)\)', r'&lt;a href="\2"&gt;\1&lt;/a&gt;', line)
</code></pre>
<p>instead of the <code>repl</code> function.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can reach your desired result using <code>re</code> as Daniel said, But if you dont want to use regex, you can do this with <code>str.split</code>:</p>
<pre><code>line = '[template application](https://stackoverflow.com/sample/base-app)'
link = line.split('(')[1][:-1]
title = line.split(']')[0][1:]

result = '&lt;a href="{}"&gt;{}&lt;/a&gt;'.format(link, title)
</code></pre>
<p>if you are using Python 3.6 or higher:</p>
<pre><code>result = f'&lt;a href="{link}"&gt;{title}&lt;/a&gt;'
</code></pre>
