<p>I have a list of instances (of <code>Process</code> class) named <code>process_list</code>. I'm strugling to find min value of attribute <code>l_time</code> greater than 0.
This is how my class looks: </p>
<pre><code>class Process:

    def __init__(self, id, a_time, p_time):
        self.id = id
        self.a_time = a_time
        self.p_time = p_time
        self.l_time = p_time
        self.w_time = 0
        self.ta_time = 0
        self.is_available = False
</code></pre>
<p>I tried something like:</p>
<pre><code>current_process = min(i for i in process_list if i.l_time &gt; 0)
</code></pre>
<p>it gives an error : </p>
<blockquote>
<p>TypeError: '&lt;' not supported between instances of 'Process' and 'Process'</p>
</blockquote>
<p>Do I have to write a <code>__gt__</code>function somehow or is there a way to do it using <code>lambda</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the key parameter of min:</p>
<pre><code>class Process:

    def __init__(self, id, a_time, p_time):
        self.id = id
        self.a_time = a_time
        self.p_time = p_time
        self.l_time = p_time
        self.w_time = 0
        self.ta_time = 0
        self.is_available = False

process_list = [Process(1, 10, 10), Process(1, 5, 5), Process(1, -5, -5)]
result = min((p for p in process_list if p.l_time &gt; 0), key= lambda x: x.l_time)

print(result.l_time)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>5
</code></pre>
</div>
<span class="comment-copy">You can pass a <code>key</code> function to <code>min</code> (often a lambda) that will function as though you had defined a <code>__gt__</code>. See <a href="https://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow noreferrer">docs.python.org/3/library/stdtypes.html#list.sort</a> for details on <code>key</code> (sort, min and max all use it the same way).</span>
