<div class="post-text" itemprop="text">
<p>You have to pass it as a single string, instead of individual arguments as you have in your example.</p>
<p>You could do this:</p>
<pre><code>file.write("You have" + str(len(gen1)) + "customers in 'Generation Z'.")
</code></pre>
<p>or this:</p>
<pre><code>file.write ("You have {} customers in 'Generation Z'.".format(len(gen1)))
</code></pre>
<p>or even:</p>
<pre><code>file.write ("You have %s customers in 'Generation Z'." % len(gen1))
</code></pre>
<p>Not that for concatenation (first option), you'll have to explicitly convert <code>len(gen1)</code> to a string using <code>str()</code> while it's a bit cleaner using the <code>.format</code> or <code>%</code> notation.</p>
