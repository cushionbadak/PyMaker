<div class="post-text" itemprop="text">
<p>You can do this with creative string formatting and <a href="https://docs.python.org/3/library/operator.html#mapping-operators-to-functions" rel="nofollow noreferrer">modulo</a> / <a href="https://docs.python.org/3/library/operator.html#operator.floordiv" rel="nofollow noreferrer">floordiv</a> operation:</p>
<pre><code>for k in range(0,90):
    print("layer[{}] = level[{}].{}".format(k+1, (k//3)+1, ["x1","x2","T"][k%3]) )
</code></pre>
<p>The floordiv is used for the <em>level- number</em>, the modulo to index into a "suffix-list" providing one of <code>["x1","x2","T"]</code>.</p>
<p>Output:</p>
<pre><code>layer[1] = level[1].x1
layer[2] = level[1].x2
layer[3] = level[1].T
layer[4] = level[2].x1
layer[5] = level[2].x2
layer[6] = level[2].T
...snipp...
layer[88] = level[30].x1
layer[89] = level[30].x2
layer[90] = level[30].T
</code></pre>
<hr/>
<p>Python 3.6+ string interpolation:</p>
<pre><code>print(f"layer[{k+1}] = level[{k//3}].{['x1','x2','T'][k%3]}")                           
</code></pre>
