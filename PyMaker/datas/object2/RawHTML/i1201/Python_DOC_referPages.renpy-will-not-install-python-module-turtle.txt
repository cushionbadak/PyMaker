<div class="post-text" itemprop="text">
<p>The syntax <code>except ValueError, ve:</code> is Python2-syntax and you're installing the module with Python3. The module was not updated since 2009, it's certainly Python2-only.</p>
<p>The actual module <code>turtle</code> is a part of the standard library, you don't need to install it, it must always be available.</p>
