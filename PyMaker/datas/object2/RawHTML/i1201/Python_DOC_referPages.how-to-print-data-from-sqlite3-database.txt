<div class="post-text" itemprop="text">
<p>Once you've executed a query, you need to fetch the result. Either with <a href="https://docs.python.org/2/library/sqlite3.html#sqlite3.Cursor.fetchone" rel="nofollow noreferrer"><code>fetchone</code></a> (fetches one result row), <a href="https://docs.python.org/2/library/sqlite3.html#sqlite3.Cursor.fetchmany" rel="nofollow noreferrer"><code>fetchmany</code></a> (fetches multiple rows) or <a href="https://docs.python.org/2/library/sqlite3.html#sqlite3.Cursor.fetchall" rel="nofollow noreferrer"><code>fetchall</code></a> fetches all of them. E.g.:</p>
<pre><code>print(cursor.fetchall())
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can treat the cursor as an iterator, or get results from the cursor using <code>fetchone()</code> or <code>fetchall()</code> </p>
<p>ie: </p>
<p><code>print(cursor.fetchone())</code></p>
<p>If you are only executing a SELECT statement, you don't need to commit.</p>
