<div class="post-text" itemprop="text">
<p>You can use python built-in <a href="https://docs.python.org/3/library/base64.html" rel="nofollow noreferrer">base64</a> library.</p>
<pre><code>import base64

with open('/path/to/file','rb') as imageFile:
    str = base64.b64encode(imageFile.read())

imageBytes = base64.decodebytes(str)
imageBinary = "".join(["{:08b}".format(x) for x in imageBytes])

print(imageBinary)
</code></pre>
<p>will result in something like:</p>
<pre><code>0011111110111011011110111011100000
</code></pre>
<p>Details can be seen on this <a href="https://stackoverflow.com/questions/43207978/python-converting-from-base64-to-binary">answer</a>.</p>
