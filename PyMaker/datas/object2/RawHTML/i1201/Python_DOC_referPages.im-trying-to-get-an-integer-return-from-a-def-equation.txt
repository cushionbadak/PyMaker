<div class="post-text" itemprop="text">
<p>There are a couple of ways to do this. If you want it to become an integer, you can use <code>int</code>:</p>
<pre><code>def number_of_cookies(amount, height, radius):
    return int((amount / 10) / (radius * radius * 3.14158 * height))
</code></pre>
<p>If there is a specified number of decimal places you want to round it to, you can use <code>round</code>:</p>
<pre><code>def number_of_cookies(amount, height, radius):
    return round((amount / 10) / (radius * radius * 3.14158 * height), 0)
</code></pre>
<p>Finally, if you want to round it down, you can do this:</p>
<pre><code>import math
def number_of_cookies(amount, height, radius):
    return math.floor((amount / 10) / (radius * radius * 3.14158 * height))
</code></pre>
