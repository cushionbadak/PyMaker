<p>How to compare string in list</p>
<pre><code>a = [['abc','Hello World'],['bcd','Hello Python']]
b = [['abc','Hello World'],['bcd','Hello World'],['abc','Python World']]
</code></pre>
<p>I want to compare every value between two lists.
For one, I want result to be (values in b but not in a): </p>
<pre><code>[['bcd','Hello World'],['abc','Python World']]
</code></pre>
<p>Other one to be (values in a but not in b): </p>
<pre><code>['bcd','Hello Python']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is a job for <a href="https://docs.python.org/3/library/stdtypes.html#set" rel="nofollow noreferrer">sets</a></p>
<p>Convert your lists of lists to sets of tuples (you can't have sets of lists, as sets can only contain <a href="https://docs.python.org/3/glossary.html#term-hashable" rel="nofollow noreferrer">hashable</a> objects, which lists, as all built-in mutable objects, are not)</p>
<pre><code>a = set(map(tuple, [['abc','Hello World'],['bcd','Hello Python']]))
b = set(map(tuple, [['abc','Hello World'],['bcd','Hello World'],['abc','Python World']]))
</code></pre>
<p>or create them directly as sets of tuples:</p>
<pre><code>a = {('abc','Hello World'),('bcd','Hello Python')}
b = {('abc','Hello World'),('bcd','Hello World'),('abc','Python World')}
</code></pre>
<p>You can then easily and efficiently get your differences:     </p>
<pre><code>print(b - a)
# {('abc', 'Python World'), ('bcd', 'Hello World')}

print(a - b)
# {('bcd', 'Hello Python')}
</code></pre>
<p>or even the intersection</p>
<pre><code>print(a &amp; b)
# {('abc', 'Hello World')}
</code></pre>
<p>or the union:</p>
<pre><code>print(a | b)
# {('abc', 'Python World'), ('bcd', 'Hello World'), ('abc', 'Hello World'), ('bcd', 'Hello Python')}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you want to have a quick solution without much concerns about performance you can use </p>
<pre><code>b_not_in_a = [i for i in b if i not in a]
a_not_in_b = [i for i in a if i not in b]
</code></pre>
