<div class="post-text" itemprop="text">
<p>I am going to assume that when you add the dictionaries, you may have duplicate keys in which you can just add the values together.</p>
<pre><code>def addScore(scorebord, scores):
    for key, value in scores.items():
        if key in scorebord:
            scorebord[key] += value
        else:
            scorebord[key] = value

def main():
    scorebord = {}
    score = {'a':1,
             'b':2,
             'c':3}

    addScore(scorebord, score)

if __name__ == "__main__":
    main()
</code></pre>

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>collections.Counter</code></a> is designed specifically to count positive integers:</p>
<pre><code>from collections import Counter

def addScore(scorebord, scores):
    scorebord += scores
    print(scorebord)

def main():
    scorebord = Counter()
    score = Counter({'a': 1, 'b': 2, 'c': 3})

    addScore(scorebord, score)

main()

# Counter({'c': 3, 'b': 2, 'a': 1})
</code></pre>
