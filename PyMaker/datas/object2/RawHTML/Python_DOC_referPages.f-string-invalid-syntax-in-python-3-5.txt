<p>For older versions of Python (before 3.6):</p>
<p>Using <a href="https://github.com/asottile/future-fstrings" rel="nofollow noreferrer"><code>future-fstrings</code></a>:</p>
<pre><code>pip install future-fstrings 
</code></pre>
<p>you have to place a special line at the top of your code:</p>
<pre><code>coding: future_fstrings
</code></pre>
<p>Hence in your case:</p>
<pre><code>-*- coding: future_fstrings -*-     # should work even without -*-
# rest of the code
location = fromstr(f'POINT({longitude} {latitude})', srid=4326)
</code></pre>
