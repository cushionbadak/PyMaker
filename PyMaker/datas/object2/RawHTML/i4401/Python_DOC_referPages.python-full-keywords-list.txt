<pre><code>import keyword
print(keyword.kwlist)
</code></pre>
<p>lists all the keywords.</p>
<p>In order to get all the built-ins,</p>
<pre><code>import builtins
print(dir(builtins))
</code></pre>
<p>gives you all the built-ins:</p>
