<p>I want to convert this list </p>
<pre><code>li = [3, 2 , 1 , 4]
</code></pre>
<p>to the following NumPy ndarray</p>
<pre><code>[[3], [2], [1], [4]]
</code></pre>
<p>I tried with <code>np.asarray()</code> but it is not converting it into an <code>ndarray</code>. Is there any way to mention the axis? </p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.expand_dims.html" rel="nofollow noreferrer"><code>numpy.expand_dims</code></a> to create an array with an extra axis,</p>
<pre><code>&gt;&gt;&gt; np.expand_dims(li, -1)
array([[3],
       [2],
       [1],
       [4]])
</code></pre>
<p>or if you prefer, add the axis after array creation.</p>
<pre><code>np.array(li)[..., np.newaxis]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">List Comprehension</a> to create a nested list, then convert the nested list to a Numpy ndarray using <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.array.html" rel="nofollow noreferrer"><code>numpy.array()</code></a>.</p>
<pre><code>&gt;&gt;&gt; import numpy as np
&gt;&gt;&gt; li = [3, 2, 1, 4]

&gt;&gt;&gt; np.array([[x] for x in li])    
[[3]
 [2]
 [1]
 [4]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>asarray</code> does produce a <code>ndarray</code>:</p>
<pre><code>In [41]: li = [3, 2 , 1 , 4]
In [42]: x = np.asarray(li)
In [43]: x
Out[43]: array([3, 2, 1, 4])
In [44]: type(x)
Out[44]: numpy.ndarray
</code></pre>
<p>To produce the extra nesting, you need to add a dimension (in one way or other):</p>
<pre><code>In [45]: np.asarray(li)[:,None]
Out[45]: 
array([[3],
       [2],
       [1],
       [4]])
</code></pre>

<div class="post-text" itemprop="text">
<p>The shortest is I believe</p>
<pre><code>np.c_[li]
# array([[3],
#        [2],
#        [1],
#        [4]])
</code></pre>
<p>For general dimension and axis:</p>
<pre><code>dummy, ndim, axis = 0, 4, 2
np.r_[f'{dummy},{ndim},{axis}', li]
# array([[[[3],
#          [2],
#          [1],
#          [4]]]])
_.shape
# (1, 1, 4, 1)
</code></pre>
<p>Another example:</p>
<pre><code>dummy, ndim, axis = 0, 3, 0
np.r_[f'{dummy},{ndim},{axis}', li]
# array([[[3]],
#        [[2]],  
#        [[1]],
#        [[4]]])
_.shape
# (4, 1, 1)
</code></pre>
