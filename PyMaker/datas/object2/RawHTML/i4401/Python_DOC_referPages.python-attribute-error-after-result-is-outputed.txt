<p>According to <a href="https://docs.python.org/3/library/re.html#re.search" rel="nofollow noreferrer">document</a>, <code>search</code>  returns None if no position in the string matches the pattern.
So, may be you need to check if pattern search returned something before calling <code>group</code>. May be you can try your 
<code>else</code> as following:</p>
<pre><code>else:
    pattern_match = pattern.search(line)
    # check for search first before calling group
    if pattern_match:
        output = pattern_match.group()
        print(output)
</code></pre>
<p>May be something like below for above code:</p>
<pre><code>import re

pattern = re.compile(r'(?&lt;=\&gt;)(.*?)(?=\|)')

# open input file
input_file = open('input.fas', 'r')

while True:
    line = input_file.readline()
    if line == '' or line is None:
        print('EOF')
        break
    else:
        pattern_match = pattern.search(line)
        # check for search first before calling group
        if pattern_match:
            output = pattern_match.group()
            print(output)
</code></pre>
