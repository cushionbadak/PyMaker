<p>I'd just advise to declare multiple values for a key like this:</p>
<pre><code>data = {
    'key1': 'value',
    'key2': ['value1', 'value2'],
}
</code></pre>
<p>Then you can simply iterate over the corresponding keys' values and filter them appropriately.</p>
