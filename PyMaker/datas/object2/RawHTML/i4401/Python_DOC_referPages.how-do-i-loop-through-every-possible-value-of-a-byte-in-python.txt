<p>I am conducting a Padding Oracle Attack for my Information Security course. Without getting into the details of the attack, I need to have a for loop that loops through all possible 1 byte hex values.</p>
<p>Pseudo-code of what I need to do:</p>
<pre><code>for x in range('\x00', '\xFF'):
    replace last byte of ciphertext with byte
    perform padding check
</code></pre>
<p>I cannot figure out how to accomplish this. Any ideas?</p>
</div>
<div class="post-text" itemprop="text">
<p>Bytes are really just integers in the range 0-255 (inclusive), or in hex, 0 through to FF. Generate the integer, then create the byte value from that. The <code>bytes()</code> type takes a list of integers, so create a list of length 1:</p>
<pre><code>for i in range(0xff):
    b = bytes([i])
</code></pre>
<p>If you store the ciphertext in a <a href="https://docs.python.org/3/library/stdtypes.html#bytearray-objects" rel="nofollow noreferrer"><code>bytearray()</code> object</a>, you could even trivially replace that last byte by using the integer directly:</p>
<pre><code>ciphertext_mutable = bytearray(ciphertext)
for i in range(0xff):
    ciphertext_mutable[-1] = i  # replace last byte
</code></pre>
