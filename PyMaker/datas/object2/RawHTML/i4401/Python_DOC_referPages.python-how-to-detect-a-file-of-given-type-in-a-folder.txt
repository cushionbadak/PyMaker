<div class="post-text" itemprop="text">
<p>Thank you for your help everyone. This turned out to be the easiest way of doing this</p>
<pre><code>import os

folder = os.listdir(r'C:\Users\name\Desktop\somefolder')
for item in folder:
    if item.endswith('.xlsx') == True:
        print ('excel folder is in this folder')   
</code></pre>
<p>I can then add a shutil.move function with ""if item.endswith('xlsx') == True""
to move the file elsewhere.
