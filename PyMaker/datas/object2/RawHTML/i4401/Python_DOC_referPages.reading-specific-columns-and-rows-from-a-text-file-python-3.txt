<p>For your test data, you can simply ignore lines without digits, and use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>format()</code></a>:</p>
<pre><code>with open('test.txt') as f:
    data = f.read().splitlines()
    data = [i.split() for i in data if any(j.isdigit() for j in i)]
    for i in data:
        print('Day: {}\nYear: {}\nLowest Temperature: {}\n'.format(i[0], i[4], i[1]))
</code></pre>
