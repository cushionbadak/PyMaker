<p>Channels 2 <a href="https://channels.readthedocs.io/en/latest/releases/2.0.0.html#release-notes" rel="nofollow noreferrer">dropped support</a> for Python 2.7 and 3.4. It now requires Python 3.5 or 3.6.</p>
<p>The <a href="https://docs.python.org/3/library/asyncio-task.html#coroutines" rel="nofollow noreferrer"><code>async def</code></a> statement was added in Python 3.5. You get a syntax error because you're using Python 3.4.</p>
