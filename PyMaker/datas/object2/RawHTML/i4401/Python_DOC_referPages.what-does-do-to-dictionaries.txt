<div class="post-text" itemprop="text">
<p>They are not dictionaries, they are <a href="https://docs.python.org/3/tutorial/datastructures.html#sets" rel="nofollow noreferrer">sets</a>:</p>
<pre><code>&gt;&gt;&gt; a = {1, 2, 3}
&gt;&gt;&gt; b = {3, 2, 1}
&gt;&gt;&gt; type(a)
&lt;type 'set'&gt;
</code></pre>
<p>The <code>&amp;</code> operator when applied to sets means set <a href="https://docs.python.org/3/library/stdtypes.html#frozenset.intersection" rel="nofollow noreferrer">intersection</a>.</p>
<p>As for your example, the result would actually be <code>set([1, 2, 3])</code>. Set <code>a</code> and set <code>b</code> are actually identical because sets are unordered collections:</p>
<pre><code>&gt;&gt;&gt; a == b
True
&gt;&gt;&gt; a &amp; b
set([1, 2, 3])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It gives the intersection of two sets.</p>
<p>In Python, below quick operands can be used for different set operations.</p>
<ul>
<li>| for union.</li>
<li>&amp; for intersection.</li>
<li>â€“ for difference</li>
<li>^ for symmetric difference</li>
</ul>
<p>Here is the code:</p>
<pre><code># Program to perform different set operations
# as we do in  mathematics

# sets are define
A = {0, 2, 4, 6, 8};
B = {1, 2, 3, 4, 5};

# union
print("Union :", A | B)

# intersection
print("Intersection :", A &amp; B)

# difference
print("Difference :", A - B)

# symmetric difference
print("Symmetric difference :", A ^ B)
</code></pre>
