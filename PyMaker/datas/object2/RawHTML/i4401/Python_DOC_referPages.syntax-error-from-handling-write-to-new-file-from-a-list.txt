<div class="post-text" itemprop="text">
<p>The problem is that % is an operator that is used between strings, see the <a href="https://docs.python.org/3/library/stdtypes.html#old-string-formatting" rel="nofollow noreferrer">documentation</a>. This should work:</p>
<pre><code>import random
SIZE = 10

for i in range (0, SIZE, 1):
    y = random.randrange(0,100)

x = open("data.txt","w+")
x.write("\n")
for i in range (0, SIZE, 1):
    x.write("Results: %d" %  y)

x.close()
</code></pre>
<p>Also, <code>range(0, SIZE, 1)</code> can be replaced by <code>range(SIZE)</code>, see the <a href="https://docs.python.org/3/library/stdtypes.html#ranges" rel="nofollow noreferrer">documentation</a>.</p>
<p>You can use the syntax <code>with open("data.txt", "w+") as f</code> that is preferred according to the <a href="https://docs.python.org/3/tutorial/inputoutput.html#reading-and-writing-files" rel="nofollow noreferrer">documentation</a>.</p>
<pre><code>import random
SIZE = 10

for i in range(SIZE):
    y = random.randrange(0, 100)

with open("data.txt", "w+") as f:
    f.write("\n")
    for i in range(SIZE):
        f.write("Results: %d" % y)
</code></pre>
<p>Also, I'm not sure about what you are doing, isn't the correct code:</p>
<pre><code>import random
SIZE = 10

with open("data.txt", "w+") as f:
    f.write("\n")
    for i in range(SIZE):
        y = random.randrange(0, 100)
        f.write("Results: %d" % y)
</code></pre>
<p>Also, be aware that <code>write</code> doesn't put any newline, maybe you want to write <code>"Results: %d\n"</code> instead.</p>

<p>This should solve your problem. Main problem here is <code>%</code> sign used improperly, I also suggest some code edits that is opening file using <code>with</code> statement ensures it closes every time so this is a best practice.</p>
<pre><code>import random
    SIZE = 10

    with open("data.txt", 'w+') as x:
        x.write("\n")
        for i in range (0, SIZE, 1):
            y = random.randrange(0,100)
            x.write("Results: %d" %  y)
</code></pre>
