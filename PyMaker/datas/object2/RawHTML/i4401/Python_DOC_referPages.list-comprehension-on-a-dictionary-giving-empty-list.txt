<p>Either you can change <code>'str'</code> to <code>str</code> i.e. </p>
<pre><code>result = [x for x in mydict.items() if type(x[1][0]) == str]
</code></pre>
<p>or may be you can try with <code>isinstance</code> method to check if it is the instance of <code>string</code> (<a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer">details</a>):</p>
<pre><code>result = [x for x,value in mydict.items() if isinstance(x[1][0],str)]
print(result)
</code></pre>

<pre><code>['increasing', 'maintaining', 'fierce', 'permanent', 'fixtures', 'underscore', 'news', 'weeklies', 'discounts', 'become', 'give', 'competition', 'spending']
