<p>I've this string variable:
date_string = "17 Apr"</p>
<p>How can I get this: 17-04-2018?</p>
<p>I'm trying with this</p>
<pre><code>datetime_object = datetime.strptime(date_string, "%d %B")
</code></pre>
<p>But I'm getting:</p>
<pre><code>builtins.ValueError: time data '17 Apr' does not match format '%d %B
</code></pre>
<p>Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <code>datetime</code> module.</p>
<p><strong>Ex:</strong></p>
<pre><code>import datetime
s = "17 Apr"
print(datetime.datetime.strptime("{0} 2018".format(s), "%d %b %Y").strftime("%d-%m-%Y"))
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>17-04-2018
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As you can see in <a href="https://docs.python.org/3.5/library/time.html#time.strftime" rel="nofollow noreferrer">the documentation</a>, you should use <code>%b</code> (lowercase) for "Locale’s abbreviated month name". <code>%B</code> is for "Locale’s full month name".</p>
<pre><code>import time
date_string = "17 Apr"
datetime_object = time.strptime(date_string, "%d %b")
print(datetime_object)
</code></pre>

<span class="comment-copy"><code>%B</code> is for the full month name (April). Use <code>%b</code> for the abbreviation (Apr). Refer to <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">strftime() and strptime() Behavior</a> for a list of <code>datetime</code> format specifiers.</span>
