<p>In python, if I use a ternary operator:</p>
<pre><code>x = a if &lt;condition&gt; else b
</code></pre>
<p>Is <code>a</code> executed even if <code>condition</code> is false? Or does <code>condition</code> evaluate first and then goes to either <code>a</code> or <code>b</code> depending on the result?</p>
</div>
<div class="post-text" itemprop="text">
<p>The condition is evaluated first, if it is False, <code>a</code> is not evaluated: <a href="https://docs.python.org/3/reference/expressions.html#conditional-expressions" rel="noreferrer">documentation</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>It gets evaluated depending if meets the condition. For example:</p>
<pre><code>condition = True
print(2 if condition else 1/0)
#Output is 2

print((1/0, 2)[condition])
#ZeroDivisionError is raised
</code></pre>
<p>No matter if <code>1/0</code> raise an error, is never evaluated as the condition was True on the evaluation.</p>
<p>Sames happen in the other way:</p>
<pre><code>condition = False
print(1/0 if condition else 2)
#Output is 2
</code></pre>
