<div class="post-text" itemprop="text">
<p>since <code>thisdir[i]</code> is not converted to its value during execution, so the error <code>No such file or directory</code></p>
<p>Replace <code>os.system('py pdf2txt.py -o text_path thisdir[i]')</code>
with</p>
<p><code>os.system("python ./pdf2txt.py -o {0} {1}".format(text_path,thisdir[i]))</code> </p>
