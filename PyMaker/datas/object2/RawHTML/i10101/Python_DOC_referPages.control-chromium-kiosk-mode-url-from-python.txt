<p>You can add options to selenium's chromedriver, similar to how you would using <code>os.system</code></p>
<pre><code>from selenium import webdriver
from selenium.webdriver.chrome.options import Options
chrome_options = Options()
chrome_options.add_argument("--kiosk")
driver = webdriver.Chrome(chrome_options=chrome_options)
driver.get("http://example.com")
</code></pre>
<p>To refresh or reload the page repeatedly, you could do <code>driver.get(url)</code> or <code>driver.refresh()</code> in a <code>while True</code> loop.</p>
