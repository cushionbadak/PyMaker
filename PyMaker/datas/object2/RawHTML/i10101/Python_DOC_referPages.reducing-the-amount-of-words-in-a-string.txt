<p>Im using a simple function that should filter all words with a length &lt; 3. Therefore I have written the following function.</p>
<pre><code>def my_tokenizer(tokens):
 tokens = [t for t in tokens if len(t) &gt; 3]
 return tokens

x = my_tokenizer("sdfgds hj")
print x
</code></pre>
<p>X however gives my an empty array. Any thoughts on where I go wrong?</p>

<div class="post-text" itemprop="text">
<p>You treat "tokens", a string, as a sequence, so it is going to loop through each character.  The string method <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow">split()</a> will separate the string on whitespace characters, (words).  So try:</p>
<pre><code>tokens = [t for t in tokens.split() if len(t) &gt; 3]
</code></pre>
</div>
<span class="comment-copy"><code>[t for t in tokens.split(' ') if len(t) &gt; 3]</code>? Otherwise you are looping through the string character by character.</span>
<span class="comment-copy">An alternative method is to use a <code>filter</code> function (which I only suggest because you specifically use the word "filter" in your question), e.g. <code>return list(filter(lambda x: len(x) &gt; 3, tokens.split()))</code>.</span>
