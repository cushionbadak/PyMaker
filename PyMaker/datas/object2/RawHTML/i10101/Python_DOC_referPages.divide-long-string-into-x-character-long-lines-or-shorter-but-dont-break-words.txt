<p>I found this function, but it breaks words. I also found <a href="https://stackoverflow.com/questions/1678051/divide-long-string-into-60-character-long-lines-but-dont-break-words">Divide long string into 60 character long lines but don't break words</a>, but it uses C# not Python.</p>

<div class="post-text" itemprop="text">
<p>Textwrap will do that if you ask it to:</p>
<pre><code>import textwrap
print(textwrap.fill(text, 50, break_long_words=False)
</code></pre>
<p>See the <a href="https://docs.python.org/3/library/textwrap.html" rel="nofollow">textwrap module documentation</a> for details on <code>break_long_words</code> and other optional parameters.</p>
