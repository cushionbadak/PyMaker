<p>so I have a python dictionary in the form of {'key1':[value1, value2]},
I hope to write it to a json file in the format of:</p>
<pre><code>{
   "key1": [value1, value2],
   "key2": [value, value],
   ....
}
</code></pre>
<p>I am currently using json.dump(dictionary, file_to_write_to), and the output looks like:</p>
<pre><code>{"key1": [value1, value2], "key2": [value, value]}
</code></pre>

<div class="post-text" itemprop="text">
<p>Use json pretty print as defined here:</p>
<p><a href="https://docs.python.org/3/library/json.html" rel="nofollow">https://docs.python.org/3/library/json.html</a></p>
</div>
<div class="post-text" itemprop="text">
<p>From simplejson documentation (see pretty print) <a href="https://simplejson.readthedocs.io/en/latest/" rel="nofollow">https://simplejson.readthedocs.io/en/latest/</a></p>
<pre><code>&gt;&gt;&gt; import simplejson as json
&gt;&gt;&gt; print(json.dumps({'4': 5, '6': 7}, sort_keys=True, indent=4 * ' '))
{
    "4": 5,
    "6": 7
}
</code></pre>
