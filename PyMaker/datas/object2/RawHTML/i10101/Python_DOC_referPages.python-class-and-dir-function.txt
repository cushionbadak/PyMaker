<p>I created an empty class in python and i put that class name to <code>dir()</code> (<code>dir(A)</code>) function then it returned output like this:</p>
<pre><code>['__class__', '__delattr__', '__dict__', '__doc__', '__eq__',
 '__format__', '__ge__', '__getattribute__', '__gt__', '__hash__',
 '__init__', '__le__', '__lt__', '__module__', '__ne__', '__new__',
 '__reduce__', '__reduce_ex__', '__repr__', '__setattr__', 
 '__sizeof__', '__str__', '__subclasshook__', '__weakref__']
</code></pre>
<p>So I want to know from where those things came from?? </p>

<div class="post-text" itemprop="text">
<p>Most of them comes from <code>object</code>, which all classes inherit from.
But some are set when defining the new class.</p>
<pre><code>set(dir(A)) - set(dir(object))

{'__dict__', '__module__', '__weakref__'}
</code></pre>
<p>So these three does not come from <code>object</code>.</p>
