<div class="post-text" itemprop="text">
<p>Did you name your file <code>os.py</code>? If you did, it is shadowing the stdlib os module. Change the name of your file and delete <code>os.pyc</code> or <code>__pycache__</code>.</p>
<p>Do <code>print os</code> or similar inside the script to see the file path Python is using to get the <code>os</code> module.</p>
