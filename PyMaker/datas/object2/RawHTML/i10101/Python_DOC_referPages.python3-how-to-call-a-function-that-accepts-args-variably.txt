<h3>The General Question</h3>
<p>What is the python3.5+ syntax for calling a function that accepts *args, where *args references a variable with multiple items.</p>
<blockquote>
<p>I am self taught python; I can't help but think I am missing something rather elementary about how *args works. Feel free to point me to some necessary reading.</p>
</blockquote>
<h3>A Specific Question</h3>
<p>When sorting a list of lists by columns using operator.itemgetter(*items), how can I provide *items variably?</p>
<h3>Sample List</h3>
<pre><code>import operator

records = [
    ['0055613893 ', 'AP', 'BU', 'AA', '00005000012404R'],
    ['0048303110 ', 'MM', 'BU', 'AB', '00028230031385R'],
    ['0044285346 ', 'AP', 'BU', 'AA', '00062240050078R'],
    ['0048303108 ', 'MO', 'BU', 'AF', '00047780036604R'],
    ['0048751228 ', 'AP', 'ME', 'AC', '00000750013177R'],
    ['0045059521 ', 'AR', 'YT', 'AB', '00005430014570R']
]
</code></pre>
<p>If I wanted to sort this particular recordset by the 3rd column and then by the 1st column I would:</p>
<p><code>records.sort(key=operator.itemgetter(3, 1))</code></p>
<p>This, of course, works as one would expect.</p>
<p>What I'm getting at is how can I provide the "3, 1" to itemgetter() variably?</p>
<p>i.e.:</p>
<p><code>indices = {insert some code here to provide itemgetter() what it needs}</code>
<code>records.sort(key=operator.itemgetter(indices))</code></p>
<p><strong>I've Tried:</strong></p>
<p>tuple: </p>
<p><code>indices = (3, 1)</code> 
or 
<code>indices = 3, 1</code></p>
<p><code>TypeError: list indices must be integers or slices, not tuple</code></p>
<p>list: </p>
<p><code>indices = [3, 1]</code></p>
<p><code>TypeError: list indices must be integers or slices, not list</code></p>

<p>It's never too late to be humbled by the basics. Thanks in advance for assisting in filling this gap in my python foundation.</p>

<div class="post-text" itemprop="text">
<p>Let's say you have </p>
<pre><code>indices = [3, 1]
</code></pre>
<p>The you can use Python unpacking to give 3 and 1 to function calls:</p>
<pre><code>itemgetter(*indices)
</code></pre>
<p>This is equivalent to</p>
<pre><code>itemgetter(3, 1)
</code></pre>
<p>I'm not sure what versions of Python this works on.  This is one of the rapidly changing parts, so you might have to do a little reading if you're using an old version.</p>
