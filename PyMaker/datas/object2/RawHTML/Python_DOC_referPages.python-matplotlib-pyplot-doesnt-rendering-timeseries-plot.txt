<div class="post-text" itemprop="text">
<p><code>plot</code> does not have named <code>x</code> and <code>y</code> arguments. So suppose you have <code>xdata</code> and <code>ydata</code> you want to plot, you cannot use <strike><code>plt.plot(x=xdata, y=ydata)</code></strike>, but instead need</p>
<pre><code>plt.plot(xdata, ydata)
</code></pre>
<p>For scatter this is different, here you can use both</p>
<pre><code>plt.scatter(xdata, ydata)
plt.scatter(x=xdata, y=ydata)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I found out the real reason: <code>plt.plot</code> can only plot numerical data, so if we want to plot the time series data with <code>np.datetime64</code> or <code>pd.datetime</code> format, we need to use the following command:</p>
<pre><code>plt.plot_date(date, y)
</code></pre>
