<p>Is there any easier way to convert non-html letters to html letters? for example if I do <code>function("a")</code> it would return <code>"&amp;#97;"</code> the Only way I know how to do this is by:</p>
<pre><code> def function(text):
      return text.replace('a','&amp;#97;')
</code></pre>
<p>So is there a better way to do this or is using replace the only way to achieve this?</p>

<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/3/library/html.entities.html#html.entities.codepoint2name" rel="nofollow"><code>html.entities.codepoint2name</code></a> and <a href="http://docs.python.org/3/library/re.html#re.sub" rel="nofollow"><code>re.sub</code></a>:</p>
<pre><code>import html.entities
import re

def to_entitydef(match):
    n = ord(match.group())
    name = html.entities.codepoint2name.get(n)
    if name is None:
        return '&amp;#{};'.format(n)
    return '&amp;{};'.format(name)

def escape(text):
    return re.sub('.', to_entitydef, text)
</code></pre>
<p>Example:</p>
<pre><code>&gt;&gt;&gt; escape('&lt;a&gt;')
'&amp;lt;&amp;#97;&amp;gt;'
</code></pre>
