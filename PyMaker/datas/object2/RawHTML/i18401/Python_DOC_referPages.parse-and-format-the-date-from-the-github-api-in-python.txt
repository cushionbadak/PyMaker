<p>The current value that is returned from a <a href="http://developer.github.com/v3/" rel="nofollow">GitHub API</a> request looks like this:</p>
<p><code>
2013-09-12T22:42:02Z
</code></p>
<p>How do I parse this value and make it look nicer?</p>
</div>
<div class="post-text" itemprop="text">
<p>The date format returned from the GitHub API is in the ISO 8601 format: <code>YYYY-MM-DDTHH:MM:SSZ</code></p>
<p>To convert that string into a Python date object, use the module <code>datetime</code>:</p>
<pre><code>import datetime
date = datetime.datetime.strptime(&lt;date_string&gt;, "%Y-%m-%dT%H:%M:%SZ")
</code></pre>
<p>You can then parse this string to the format of your choosing using <code>date.strftime()</code>:</p>
<pre><code># Result: Thursday Sep 12, 2013 at 22:42 GMT
date.strftime('%A %b %d, %Y at %H:%M GMT')
</code></pre>
<p>Or if you want it to be more "automatic", the directive <code>%c</code> will automatically choose a date/time string based on your system's locale and language settings.</p>
<pre><code># On my system, I get the following output:
#  Thu Sep 12 22:42:02 2013
date.strftime('%c')
</code></pre>
<p>If you want to customize it, a full list of directives can be found here: <a href="http://docs.python.org/3/library/datetime.html#strftime-strptime-behavior">http://docs.python.org/3/library/datetime.html#strftime-strptime-behavior</a></p>
