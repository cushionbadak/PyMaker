<div class="post-text" itemprop="text">
<p><a href="http://docs.python.org/3/library/datetime.html#date-objects" rel="nofollow"><code>datetime.date</code> objects</a> have a <a href="http://docs.python.org/3/library/datetime.html#datetime.date.year" rel="nofollow"><code>year</code> attribute</a>:</p>
<pre><code>from datetime import date

print(date.today().year)
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; from datetime import date
&gt;&gt;&gt; print(date.today().year)
2013
</code></pre>
<p>You can get the <a href="http://docs.python.org/3/library/datetime.html#datetime.datetime.year" rel="nofollow">same attribute</a> from <a href="http://docs.python.org/3/library/datetime.html#datetime-objects" rel="nofollow"><code>datetime.datetime</code> objects</a>:</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; print(datetime.now().year)
2013
</code></pre>
<p>You can also get the year from the <a href="http://docs.python.org/2/library/time.html" rel="nofollow"><code>time</code> module</a>; the <a href="http://docs.python.org/2/library/time.html#time.struct_time" rel="nofollow"><code>time.struct_time</code> structure</a> has a <code>.tm_year</code> attribute; the structure is a return value for several functions:</p>
<pre><code>&gt;&gt;&gt; import time
&gt;&gt;&gt; print(time.localtime().tm_year)
2013
</code></pre>
