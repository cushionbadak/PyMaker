<p>In Python 3 these statements are True:</p>
<pre><code>ord('A') == 65
chr(65) == 'A'
bin(65) == '0b1000001'
hex(65) == '0x41'
int(b'1000001',2) == 65
</code></pre>
<p>Given the integer <code>i = 65</code>, how do I in Python 3 open a file in binary mode and write the byte <code>01000001</code> to file?</p>
<p>Despite being simple, I haven't been able to find a duplicate question. </p>

<div class="post-text" itemprop="text">
<p>You may use any of the forms below, according to the needs of your program. They all do precisely the same thing -- write one byte to the output file.</p>
<pre><code>i = 65
with open('outputfile.bin', 'wb') as f:
    f.write(bytes([i]))
    f.write('A'.encode('ascii'))
    f.write(bytes('A', encoding='ascii'))
</code></pre>
