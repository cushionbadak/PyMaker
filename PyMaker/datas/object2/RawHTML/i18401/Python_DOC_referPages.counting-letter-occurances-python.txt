<div class="post-text" itemprop="text">
<p><code>d</code> is a <em>dictionary</em>, not a list. Loop over the keys:</p>
<pre><code>for key in d:
    print('{} | {}'.format(key, d[key]))
</code></pre>
<p>or you'll get <code>KeyError</code> exceptions.</p>
<p>You may be interested in the <a href="http://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow"><code>collections.Counter()</code> class</a>; it's a counting dictionary:</p>
<pre><code>from collections import Counter

def histogram(L):
    d = Counter(letter for line in L for letter in x)
    for letter in d:
        print('{} | {}'.format(letter, d[letter]))
    return d
</code></pre>

<p>Just for fun, lets simplify your code. You can use a set() on the initial string to get a list of the unique characters, then just use the count method of the list.</p>
<pre><code>def histogram(L):
    d = {letter:L.count(letter) for letter in set(L)}
    for key in d:
        print "{} | {}".format(key, d[key]}
</code></pre>
