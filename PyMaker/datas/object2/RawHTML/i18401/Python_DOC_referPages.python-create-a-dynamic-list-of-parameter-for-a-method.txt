<p>I want to create a XLSX file in python. For this, I use <a href="https://xlsxwriter.readthedocs.org" rel="nofollow">xlsxwriter</a>.</p>
<p>In my XLSX I want to highlight some part of text with the <a href="https://xlsxwriter.readthedocs.org/en/latest/example_rich_strings.html" rel="nofollow">write_rich_string</a> method.</p>
<p>But, my string is not a fixed value. The part I want to highlight can be randomly placed into it. Or I can have multiples part of the text to highlight.</p>
<p>So, can I create a "list of parameters" for the call of write_rich_string? And what is the method to made this?</p>
<p>Example:</p>
<pre><code>mystring = "I want this in [tag_1]yellow[/tag_1] and this in [tag_2]green[/tag_2]."

worksheet.write_rich_string('A1',
                        'I want this in',
                        yellow, 'yellow',
                        ' and this is ',
                        green, 'green')

worksheet.write_rich_string('A1', my_list_of_parm)
</code></pre>

<div class="post-text" itemprop="text">
<p>If I understand what you're askingâ€¦ you've got a list like this:</p>
<pre><code>my_list_of_parm = ['I want this in',
                   yellow, 'yellow',
                   ' and this is ',
                   green, 'green']
</code></pre>
<p>And you want to pass that to <code>write_rich_string</code> as a bunch of separate arguments (along with one normal argument).</p>
<p>The way to do that is explained in the tutorial under <a href="http://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists" rel="nofollow">Unpacking Argument Lists</a>: Just put <code>*</code> before <code>my_list_of_parm</code>, and it will be unpacked into separate arguments:</p>
<pre><code>worksheet.write_rich_string('A1', *my_list_of_parm)
</code></pre>
<hr/>
<p>In general, if you want to convert a list (or other iterable) into separate values or vice-versa, <code>*</code> is the answer:</p>
<pre><code>&gt;&gt;&gt; def func(a, b, *args):
...     print(args)
&gt;&gt;&gt; func(1, 2, 3, 4, 5)
[3, 4, 5]
&gt;&gt;&gt; a, b, *rest = [1, 2, 3, 4, 5]
&gt;&gt;&gt; rest
[3, 4, 5]
&gt;&gt;&gt; func(1, *rest)
[4, 5]
</code></pre>
<p>But for the exact rules on what you can and can't do, you'll need to read the docs.</p>
