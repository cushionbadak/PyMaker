<p>Using <a href="http://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow"><code>xml.etree.ElementTree</code></a>:</p>
<pre><code>import xml.etree.ElementTree as ET

root = ET.fromstring('''&lt;?xml version="1.0"?&gt;
&lt;BCPFORMAT 
...
&lt;/BCPFORMAT&gt;''')
# Accessing parent node: http://effbot.org/zone/element.htm#accessing-parents
parent_map = {c: p for p in root.getiterator() for c in p}    child = root.find('.//*[@ID="1"]')
print(list(parent_map[child]).index(child)) # =&gt; 0
</code></pre>
<p>Using <a href="http://lxml.de/" rel="nofollow"><code>lxml</code></a>:</p>
<pre><code>import lxml.etree as ET

root = ET.fromstring('''&lt;?xml version="1.0"?&gt;
&lt;BCPFORMAT 
...
&lt;/BCPFORMAT&gt;''')
child = root.find('.//*[@ID="1"]')
print(child.getparent().index(child)) # =&gt; 0
</code></pre>
