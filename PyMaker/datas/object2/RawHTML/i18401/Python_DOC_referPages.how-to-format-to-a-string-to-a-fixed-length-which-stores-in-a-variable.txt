<p>I have a number and I want to format to a string with zero padding and a specific length . Suppose if the fixed length is 22 , I know I can do in this way</p>
<pre><code> format(variable,"%022d")
</code></pre>
<p>but does there exist any way can do by this </p>
<pre><code>length = 22
format(variable,"%0lengthd")
</code></pre>
<p>It means I need to use variable in <code>% variable d</code> expression</p>
<p>thanks</p>
</div>
<div class="post-text" itemprop="text">
<p>You should use <a href="http://docs.python.org/3/library/stdtypes.html#str.zfill" rel="nofollow"><code>str(variable).zfill(length)</code></a> for this:</p>
<pre><code>&gt;&gt;&gt; str(12345).zfill(8)
'00012345'
&gt;&gt;&gt; str(12345).zfill(10)
'0000012345'
</code></pre>
<p>It might seem somewhat unintuitive to have to convert the number to a string before zero-padding it, but (at least in my opinion) this is much cleaner than using multiple formats. After all, why hack together a solution when it's already builtin?</p>
</div>
<div class="post-text" itemprop="text">
<p>Yes, by using string formatting inside:</p>
<pre><code>length = 22
format(variable,"%0{}d".format(length))
</code></pre>
<p>Infact, you could probably do this whole thing with one call of <a href="http://docs.python.org/2/library/stdtypes.html#str.format" rel="nofollow"><code>str.format()</code></a>:</p>
<pre><code>length = 22
"{0:{1}}".format(variable, length)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Even classic string interpolation supports this:</p>
<pre><code>&gt;&gt;&gt; variable = 100
&gt;&gt;&gt; length = 22
&gt;&gt;&gt; print "%0*d" % (length,variable)
0000000000000000000100
</code></pre>
