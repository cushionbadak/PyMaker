<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/stdtypes.html#str.translate" rel="noreferrer"><code>str.translate()</code></a> to remove codepoints; any codepoint mapping to <code>None</code> is removed:</p>
<pre><code>remove = dict.fromkeys(map(ord, '\n ' + string.punctuation))
f.translate(remove)
</code></pre>
<p>The <code>dict.fromkeys()</code> class method makes it easy to create a dictionary mapping all keys to <code>None</code>.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; import string
&gt;&gt;&gt; remove = dict.fromkeys(map(ord, '\n ' + string.punctuation))
&gt;&gt;&gt; sample = 'The quick brown fox, like, totally jumped, man!'
&gt;&gt;&gt; sample.translate(remove)
'Thequickbrownfoxliketotallyjumpedman'
</code></pre>
<p>Adjusted to your code:</p>
<pre><code>remove = dict.fromkeys(map(ord, '\n ' + string.punctuation))

with open('cipher.txt') as inputfile:
    f = inputfile.read().translate(remove)
</code></pre>
