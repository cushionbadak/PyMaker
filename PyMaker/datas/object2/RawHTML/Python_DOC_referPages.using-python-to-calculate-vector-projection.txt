<div class="post-text" itemprop="text">
<p>Maybe, just maybe, this is what you really want:</p>
<pre><code>np.dot(x, y) / np.linalg.norm(y)
</code></pre>
<p>This should give the projection of vector <code>x</code> onto vector <code>y</code> - see <a href="https://en.wikipedia.org/wiki/Vector_projection" rel="nofollow noreferrer">https://en.wikipedia.org/wiki/Vector_projection</a>. Alternatively, if you want to compute the projection of <code>y</code> onto <code>x</code>, then replace <code>y</code> with <code>x</code> in the denominator (<code>norm</code>) of the above equation.</p>
