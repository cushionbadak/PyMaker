<div class="post-text" itemprop="text">
<p>Using the standard hashlib module:</p>
<pre><code>import hashlib

hash = hashlib.md5()
hash.update('555555555')
print hash.hexdigest()
</code></pre>
<p><strong>output</strong></p>
<pre><code>3665a76e271ada5a75368b99f774e404
</code></pre>
<p>As mentioned in timkofu's comment, you can also do this more simply, using </p>
<pre><code>print hashlib.md5('555555555').hexdigest()
</code></pre>
<p>The <code>.update()</code> method is useful when you want to generate a checksum in stages. Please see the <a href="https://docs.python.org/2/library/hashlib.html" rel="nofollow">hashlib documentation</a> (or the <a href="https://docs.python.org/3/library/hashlib.html" rel="nofollow">Python 3 version</a>) for further details.</p>
