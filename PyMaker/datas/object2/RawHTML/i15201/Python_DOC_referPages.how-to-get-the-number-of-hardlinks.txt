<p>Is there some library function, or some other way, to get the hard link count of some file in Python?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/os.html#os.stat_result.st_nlink" rel="nofollow"><code>st_nlink</code></a>:</p>
<pre><code>os.stat("foo.txt").st_nlink
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try <a href="https://docs.python.org/2/library/os.html#os.stat" rel="nofollow"><code>os.stat</code></a>:</p>
<pre><code>os.stat(whatever).st_nlink
</code></pre>
<p>Not sure how this works on Windows.</p>
</div>
