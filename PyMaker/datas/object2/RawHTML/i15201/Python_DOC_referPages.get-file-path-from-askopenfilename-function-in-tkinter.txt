<div class="post-text" itemprop="text">
<p>I am writing a script to automate changing a particular set of text in one file into a particular set in another with a different name.</p>
<p>I want to get the name of the file using the <code>askopenfilename</code> function, but when I try to print the file name, it returns:</p>
<p><code>&lt;_io.TextIOWrapper name='/home/rest/of/file/path/that/I/actually/need.txt' mode='w' encoding='ANSI_X3.4-1968'&gt;</code></p>
<p>I need just the file name because the <code>&lt;_io.TextIOWrapper ...&gt;</code> is not sub scriptable.</p>
<p>Any suggestions to remove the extraneous bits?</p>
</div>
<div class="post-text" itemprop="text">
<p><code>askopenfilename()</code> returns the path of the selected file or empty string if no file is selected:</p>
<pre><code>from tkinter import filedialog as fd

filename = fd.askopenfilename()
print(len(filename))
</code></pre>
<p>To open the file selected with <code>askopenfilename</code>, you can simply use normal Python constructs and functions, such as the <a href="https://docs.python.org/3/library/functions.html#open" rel="noreferrer"><em><code>open</code></em></a> function:</p>
<pre><code>if filename:
    with open(filename) as file:
        return file.read()
</code></pre>
<p>I think you are using <code>askopenfile</code>, which opens the file selected and returns a <code>_io.TextIOWrapper</code> object or <code>None</code> if you press the <em>cancel</em> button. </p>
<p>If you want to stick with <code>askopenfile</code> to get the file path of the file just opened, you can simply access the property called <code>name</code> of the <code>_io.TextIOWrapper</code> object returned:</p>
<pre><code>file = fd.askopenfile()
if file: 
    print(file.name)
</code></pre>
<p>If you want to know more about all the functions defined under the <code>filedialog</code> (or <code>tkFileDialog</code> for Python 2) module, you can read <a href="http://tkinter.unpythonic.net/wiki/tkFileDialog" rel="noreferrer"><em>this article</em></a>.</p>
