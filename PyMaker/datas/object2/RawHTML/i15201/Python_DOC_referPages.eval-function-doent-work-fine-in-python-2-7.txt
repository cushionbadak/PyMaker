<div class="post-text" itemprop="text">
<p>In Python 2 <a href="https://docs.python.org/2/reference/simple_stmts.html#print" rel="nofollow"><code>print xxx</code></a> is a statement, not an expression. <a href="https://docs.python.org/2/library/functions.html#eval" rel="nofollow"><code>eval</code></a> is for evaluating expressions; you can use <a href="https://docs.python.org/2/reference/simple_stmts.html#exec" rel="nofollow"><code>exec</code></a> for executing statements.</p>
<p>In Python 3, <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow"><code>print(xxx)</code></a> is a function call, so it can be evaluated.</p>
