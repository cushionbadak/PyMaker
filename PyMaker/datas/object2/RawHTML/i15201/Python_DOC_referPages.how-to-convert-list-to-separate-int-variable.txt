<div class="post-text" itemprop="text">
<p>So, I have a list named </p>
<pre><code>list1 = ['1', '2', '3']
</code></pre>
<p>Then I converted it into int - </p>
<pre><code>list1 = [1, 2, 3]
</code></pre>
<p>My question is, how do I separate list1 to individual int variable? Like this:</p>
<pre><code>a = 1
b = 2
c = 3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This will unpack the list and assign it's content to the corresponding variables.</p>
<pre><code>&gt;&gt;&gt; list1 = [1, 2, 3]
&gt;&gt;&gt; a,b,c = list1
&gt;&gt;&gt; a
1
&gt;&gt;&gt; b
2
&gt;&gt;&gt; c
3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists" rel="nofollow">unpacking</a></p>
<pre><code>&gt;&gt;&gt; list1 = ['1', '2', '3']
&gt;&gt;&gt; a, b, c = map(int, list1)
&gt;&gt;&gt; a
1
&gt;&gt;&gt; b
2
&gt;&gt;&gt; c
3
</code></pre>
</div>
