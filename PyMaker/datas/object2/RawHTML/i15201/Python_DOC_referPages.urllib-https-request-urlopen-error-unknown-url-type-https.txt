<p>Most likely your Python installation or operating system is broken.</p>
<p>Python has only support for HTTPS if it was compiled with HTTPS support. However, this should be the default for all sane installations.</p>
<pre><code>HTTPS support is only available if the socket module was compiled with SSL support.
</code></pre>
<p><a href="https://docs.python.org/3/library/http.client.html" rel="noreferrer">https://docs.python.org/3/library/http.client.html</a></p>
<p>Please clarify how you installed Python. Official Python distributions are available at python.org</p>
