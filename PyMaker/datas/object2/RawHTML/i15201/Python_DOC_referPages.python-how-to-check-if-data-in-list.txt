<p>I have a code:</p>
<pre><code>dates = []

dates.append(['imie', 'nazwisko'])
dates.append(['test', 'test2'])
</code></pre>
<p>How can check if "imie" and "nazwisko" is in dates?</p>
<p>if 'imie' and 'nazwisko' in dates - not work</p>
<p>Thanks for help</p>

<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow"><code>any</code></a> and <a href="https://docs.python.org/3/tutorial/classes.html#generator-expressions" rel="nofollow">generator expression</a>:</p>
<pre><code>&gt;&gt;&gt; dates = []
&gt;&gt;&gt; dates.append(['imie', 'nazwisko'])
&gt;&gt;&gt; dates.append(['test', 'test2'])
&gt;&gt;&gt; any(('imie' in d and 'nazwisko' in d) for d in dates)
True
</code></pre>
<p><strong>UPDATE</strong></p>
<p>You can also use <a href="https://docs.python.org/3/library/stdtypes.html#set.issubset" rel="nofollow"><code>set.issubset</code></a> as suggested by Jon Clements:</p>
<pre><code>&gt;&gt;&gt; any({'imie', 'nazwisko'}.issubset(d) for d in dates)
True
</code></pre>
</div>
