<div class="post-text" itemprop="text">
<p>You are seeing the <a href="https://docs.python.org/3/library/functions.html#repr" rel="nofollow">repr</a> representation of the strings in your list:</p>
<pre><code>[{'name':u'M\xd3NICA MEN\xc9NDEZ GALLEGOS', 'address': u'GENERAL YAG\xdcE 32}
</code></pre>
<p>When you print you are seeing the <code>str</code> output.</p>
<p>They are both equal unicode strings:</p>
<pre><code>In [1]: (l[0]["address"]) 
Out[1]: u'GENERAL YAG\xdcE 32'
In [2]: (l[0]["address"]) == u"GENERAL YAGÜE 32"
Out[2]: True
</code></pre>
