<p>I have following code:</p>
<pre><code>import datetime
x = '2015-01-01'
date = datetime.datetime.strptime(x, "%Y-%m-%d")
print(date)
</code></pre>
<p>The problem is that I want to print date like <code>"2015-01-01"</code> and not <code>"2015-01-01 00:00:00"</code>. How to print it without hours, minutes and seconds?</p>
</div>
<div class="post-text" itemprop="text">
<p><code>datetime</code> objects have a <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.date" rel="nofollow"><code>date</code> method</a> which does this for you:</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; x = '2015-01-01'
&gt;&gt;&gt; date = datetime.datetime.strptime(x, "%Y-%m-%d")
&gt;&gt;&gt; print(date.date())
2015-01-01
&gt;&gt;&gt;
</code></pre>
