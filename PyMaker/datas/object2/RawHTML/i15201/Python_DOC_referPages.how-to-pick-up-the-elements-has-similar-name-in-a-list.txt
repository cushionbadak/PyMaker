<p>For the second question, if you mean to extract digits part out of the string:</p>
<p>You can use the string slice because the pattern is fixed:</p>
<pre><code>&gt;&gt;&gt; 'TIMESTEPEMITTANCE185.TBL'[17:20]
'185'
</code></pre>
<p>or, you can use <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow"><code>filter</code></a> with <a href="https://docs.python.org/3/library/stdtypes.html#str.isdigit" rel="nofollow"><code>str.isdigit</code></a> (<code>''.join</code> is not required if you use Python 2.x)</p>
<pre><code>&gt;&gt;&gt; ''.join(filter(str.isdigit, 'TIMESTEPEMITTANCE185.TBL'))
'185'
</code></pre>
<p>or, using regular expression:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; re.search(r'\d+', 'TIMESTEPEMITTANCE185.TBL').group()
'185'
</code></pre>
