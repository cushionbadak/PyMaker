<p>You can easily modify your <code>xml</code> file using the <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#modifying-an-xml-file" rel="nofollow">ElementTree API</a></p>
<pre><code>from xml.etree.ElementTree import parse
doc = parse('data.xml')
root = doc.getroot()
for t in root.iterfind('pubgroup/member'):
    t.attrib['objlocation'] = "spam"

doc.write('output.xml', xml_declaration=True)
</code></pre>
<p>The <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.iterfind" rel="nofollow"><code>iterfind</code></a> method returns a <code>generator</code> function instead of <code>list</code> which is very convenient if your have <code>xml</code> file  is very large</p>
<p>Output</p>
<pre><code>&lt;?xml version='1.0' encoding='us-ascii'?&gt;
&lt;publish show="STATE"&gt;

    &lt;pubgroup location="/user_data/STATE/ITEM/character/ANM/ANM_rig_WALK_sg_v001/ANM_rig_WALK_sg_v001.xml" objtype="ELE"&gt;

        &lt;member objidx="15283942" objlabel="anm" objlocation="spam" /&gt;

        &lt;member objidx="15283952" objlabel="fbx" objlocation="spam" /&gt;

        &lt;member objidx="15283962" objlabel="mov" objlocation="spam" /&gt;

        &lt;member objidx="15283972" objlabel="libraryinfo" objlocation="spam" /&gt;

    &lt;member objidx="15283982" objlabel="thumbnail" objlocation="spam" /&gt;

&lt;/pubgroup&gt;
</code></pre>
<p></p>
<p>Here <code>spam</code> is <code>objlocation</code> new value.</p>

<p>The shortest code I can suggest:</p>
<pre><code>from xml.etree.ElementTree import ElementTree
tree = ElementTree()
root = tree.parse('test.txt') # root represents &lt;publish&gt; tag

for member in root.findall('pubgroup/member'):
    print member.attrib['objlocation']
</code></pre>
<p>Output:</p>
<pre><code>/user_data/STATE/ITEM/character/ANM/ANM_rig_WALK_sg_v001/ANM_rig_WALK_sg_v001.anm
/user_data/STATE/ITEM/character/ANM/ANM_rig_WALK_sg_v001/ANM_rig_WALK_sg_v001_M_WALK_None.fbx
/user_data/STATE/ITEM/character/ANM/ANM_rig_WALK_sg_v001/ANM_rig_WALK_sg_v001.mov

/user_data/STATE/ITEM/character/ANM/ANM_rig_WALK_sg_v001/ANM_rig_WALK_sg_v001.mng
</code></pre>
<p>To make changes:</p>
<pre><code>for member in root.findall('pubgroup/member'):
    member.attrib['objlocation'] = 'changed'
tree.write('output.txt')
</code></pre>
