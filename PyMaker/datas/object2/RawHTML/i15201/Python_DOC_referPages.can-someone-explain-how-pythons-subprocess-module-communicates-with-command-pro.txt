<div class="post-text" itemprop="text">
<p>Backslashes have a special meaning inside strings in Python (and almost every other programming language) as <a href="https://docs.python.org/3/reference/lexical_analysis.html#literals" rel="nofollow">documented here</a>, and so you need to write either</p>
<pre><code>call(['Z:\\input\\convertToRinex.exe', 'Z:\\input\\74390010.T01'])
</code></pre>
<p>or</p>
<pre><code>call([r'Z:\input\convertToRinex.exe', r'Z:\input\74390010.T01'])
</code></pre>
</div>
