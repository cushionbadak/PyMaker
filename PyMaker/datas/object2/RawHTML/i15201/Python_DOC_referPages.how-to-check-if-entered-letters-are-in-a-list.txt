<p>For example like this?</p>
<pre><code>letters = set(["a","b","c","d"])

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#set-types-set-frozenset" rel="nofollow">sets</a>; either test for an intersection or for a subset:</p>
<pre><code>if set(inpt) &amp; set(letters):
    # *some* letters are in inpt
</code></pre>
<p>or</p>
<pre><code>if set(letters) &lt;= set(inpt):
    # *all* letters are in inpt
</code></pre>
<p>or if you wanted all characters in <code>inpt</code> to be in <code>letters</code> (so only <code>a</code>, <code>b</code>, <code>c</code> or <code>d</code> can be used, use the inverse:</p>
<pre><code>if set(inpt) &lt;= set(letters):
    # *all* of inpt is in letters
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; letters = ['a', 'b', 'c', 'd']
&gt;&gt;&gt; inpt = 'the quick brown fox jumped over the lazy dog'
&gt;&gt;&gt; set(letters) &amp; set(inpt)
{'d', 'c', 'a', 'b'}
&gt;&gt;&gt; set(letters) &lt;= set(inpt)
True
&gt;&gt;&gt; inpt = 'no letters in use!'
&gt;&gt;&gt; set(letters) &amp; set(inpt)
set()
&gt;&gt;&gt; set(letters) &lt;= set(inpt)
False
&gt;&gt;&gt; inpt = 'abbacac'
&gt;&gt;&gt; set(inpt) &lt;= set(letters)
True
</code></pre>
<p>where an empty <code>set()</code> is considered false when used in an <code>if</code> statement, a non-empty <code>set()</code> is considered true.</p>
