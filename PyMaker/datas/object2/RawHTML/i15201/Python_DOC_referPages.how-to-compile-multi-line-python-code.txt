<div class="post-text" itemprop="text">
<p>I'd not use the interactive interpreter code for this. You can use the <a href="https://docs.python.org/3/library/functions.html#exec" rel="nofollow"><code>exec()</code> function</a> to interpret Python code.</p>
<p>Under the hood, that is exactly what <code>code.runcode()</code> <em>does</em>.</p>
