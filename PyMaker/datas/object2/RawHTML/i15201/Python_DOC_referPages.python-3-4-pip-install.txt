<div class="post-text" itemprop="text">
<blockquote>
<p>To avoid conflicts between parallel Python 2 and Python 3
  installations, only the versioned pip3 and pip3.4 commands are
  bootstrapped by default</p>
</blockquote>
<p>You can try <code>sudo pip3 install xlrd</code> or <code>sudo pip3.4 install xlrd</code> to use the Python 3.4, see docs here <a href="https://docs.python.org/3/whatsnew/3.4.html#bootstrapping-pip-by-default" rel="nofollow">https://docs.python.org/3/whatsnew/3.4.html#bootstrapping-pip-by-default</a></p>
