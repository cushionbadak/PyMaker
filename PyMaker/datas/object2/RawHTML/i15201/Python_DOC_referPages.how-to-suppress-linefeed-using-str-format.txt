<div class="post-text" itemprop="text">
<p>I know how to use <code>end=""</code> with the <code>print</code> function in Python 3.0 to suppress a linefeed, but how do I suppress a linefeed while using the <code>str.format</code> method?</p>
<pre><code>print("Hello", end=",") # Works
print(" Hola")
print("{}".format(""))
print("{}".format("Hello", end = ",")) # Doesn't work
print("{}".format(" Hola"))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The newline is being added by the <code>print</code> function, not <code>str.format</code>.  From the <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow">docs</a>:</p>
<blockquote>
<p><code>print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)</code></p>
<p>Print <code>objects</code> to the text stream <code>file</code>, separated by <code>sep</code> and followed by <code>end</code>.</p>
</blockquote>
<p>Notice how the <code>end</code> parameter defaults to <code>'\n'</code> (a newline). There is no way for <code>str.format</code> to remove this newline because it isn't there until <em>after</em> the formatted string is given to <code>print</code>.</p>
<p>You will need to use <code>end=""</code>:</p>
<pre><code>print("{}".format("Hello"), end="")
</code></pre>
</div>
