<div class="post-text" itemprop="text">
<p>Python supports modulo operations on floating point numbers, so you can do this with</p>
<pre><code>&gt;&gt;&gt; 9.6 % 1
0.5999999999999996
</code></pre>
<p>Note that this won't give you what you want if the number is negative:</p>
<pre><code>&gt;&gt;&gt; -9.6 % 1
0.40000000000000036
</code></pre>
<p>So in general you'll want</p>
<pre><code>frac = abs(value) % 1
</code></pre>
<p>This is the fastest of the ones I looked at:</p>
<pre><code>%timeit frac = abs(value) % 1
77.1 ns ± 0.568 ns per loop (mean ± std. dev. of 7 runs, 10000000 loops each)

%timeit frac = abs(value) % int(value)
196 ns ± 5.77 ns per loop (mean ± std. dev. of 7 runs, 1000000 loops each)

%timeit frac = value - int(value)
156 ns ± 2.42 ns per loop (mean ± std. dev. of 7 runs, 10000000 loops each)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>given an integer <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow noreferrer">int()</a> truncates the fractional part, so with</p>
<pre><code>n = 4.5
fractional = n - int(n) #0.5
</code></pre>
<p>you can get it.</p>
</div>
<div class="post-text" itemprop="text">
<p>Even easier approach using strings would be :</p>
<pre><code>number = 123.4    
number_dec = str(number).split(".")[1]
print(number_dec)
</code></pre>
<p>Output:</p>
<pre><code>4
</code></pre>
