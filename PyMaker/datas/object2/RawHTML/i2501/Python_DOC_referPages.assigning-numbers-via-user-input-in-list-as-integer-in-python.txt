<div class="post-text" itemprop="text">
<p>You cannot cast the entire <code>list</code> (or the input string that contains all the commas) to <code>int</code> as a whole, you would have to do it for the split tokens individually, e.g. with a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>assignments = [int(x) for x in input(“assignments:”).split(",")]
</code></pre>
<p>or using <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map</code></a>:</p>
<pre><code>assignments = list(map(int, input(“assignments:”).split(",")))
</code></pre>
