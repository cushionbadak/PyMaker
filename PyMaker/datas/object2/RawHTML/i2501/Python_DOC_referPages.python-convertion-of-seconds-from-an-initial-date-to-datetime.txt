<div class="post-text" itemprop="text">
<p>You will first have to create a datetime object for Jan 1st 1993 and then add the number of seconds to that date. The code below should help you get started.</p>
<pre><code>from datetime import datetime, timedelta

original_date = datetime.strptime('01-01-1993', '%d-%m-%Y')
original_date + timedelta(seconds= 10000)
</code></pre>
<p>output: datetime.datetime(1993, 1, 1, 2, 46, 40)</p>

<p>It is like UNIX time, but with a different start.</p>
<p>You can compute the offset once:</p>
<pre><code>&gt;&gt;&gt; import datetime as dt
&gt;&gt;&gt; dt.datetime(1993,1,1).timestamp()
725842800.0
</code></pre>
<p>and use it in your program:</p>
<pre><code>OFFSET = 725842800.0
mydate = dt.datetime.fromtimestamp(OFFSET + seconds_from_1993)
</code></pre>
