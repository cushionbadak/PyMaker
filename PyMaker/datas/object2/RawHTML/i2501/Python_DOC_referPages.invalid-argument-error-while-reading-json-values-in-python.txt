<div class="post-text" itemprop="text">
<p>As <code>open</code> is for opening local files, not URLs as commented by <a href="https://stackoverflow.com/users/3001761/jonrsharpe">jonrsharpe</a> so, go with <a href="https://docs.python.org/3/library/urllib.html" rel="nofollow noreferrer">urllib</a> as commented by <a href="https://stackoverflow.com/users/298624/fl00r">fl00r</a>.</p>
<p>Though the link provided by him was for <code>python-2</code></p>
<p>Try this (<code>python-3</code>):</p>
<pre><code>import json
from urllib.request import urlopen

with urlopen('https://www.w3schools.com/js/json_demo.txt') as json_file:
    data = json.load(json_file)
    #for p in data['people']:
    print('Name: ' + data['name'])
</code></pre>
