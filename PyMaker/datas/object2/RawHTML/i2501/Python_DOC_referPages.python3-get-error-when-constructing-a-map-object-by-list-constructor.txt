print(list(list_of_input)[0])</code></p>

<div class="post-text" itemprop="text">
<p>The conversion of a map to list is done only once, meaning map is a generator object and once you transform it to list, it gets exhausted : <a href="https://stackoverflow.com/questions/37644176/python-3-generator-for-map">python 3: generator for map</a>. So the error comes from second print statement and not the first. </p>
</div>
<div class="post-text" itemprop="text">
<p>in Python 3 <code>map()</code> returns an iterator not a <code>list</code>. When you pass this iterator to <code>list</code> the first time it consumes the iterator so the second time you get an empty <code>list</code>, hence the IndexError.</p>
