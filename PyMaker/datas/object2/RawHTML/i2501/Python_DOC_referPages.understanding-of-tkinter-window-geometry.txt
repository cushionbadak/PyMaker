<div class="post-text" itemprop="text">
<p><code>w, h = window.winfo_screenwidth(), window.winfo_screenheight()</code></p>
<p><code>w</code>, and <code>h</code> are now holding the value of the screen height and width, say <code>1280</code>, and <code>920</code></p>
<p><code>window.geometry("%dx%d+0+0" % (w, h))</code></p>
<p>Then it is a straightforward python string formatting (the old fashion way):</p>
<p><code>window.geometry("1280x920+0+0")</code></p>
<p>which is a string that tkinter can parse. (welcome to the 1990'es)</p>
