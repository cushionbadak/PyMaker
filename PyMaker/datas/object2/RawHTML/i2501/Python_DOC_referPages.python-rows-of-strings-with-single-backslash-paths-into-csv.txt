<p>Try <code>myColumn = [s.replace('\\\\', '\\') for s in myColumn]</code>. This should replace double backslashes (4 backshlash literals) with single backsladhes (2 backslash literals).</p>

<span class="comment-copy">Assuming <code>myColumn = [...</code> is a quote from your code, you don't have strings with double backslashes. You have string <i>literals</i> with double backslashes, which are understood by Python as strings with single backslashes. <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow noreferrer">Relevant documentation</a></span>
<span class="comment-copy">The double backslashes you see in a string literal actually denotes to one literal backslash, so there is no need for conversion.</span>
