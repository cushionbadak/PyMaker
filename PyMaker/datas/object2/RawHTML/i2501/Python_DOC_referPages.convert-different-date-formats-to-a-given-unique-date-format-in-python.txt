<p>I have different date formats present as strings those are:</p>
<pre><code>[ "2018-06-22",
"June 22, 2018",
"12/15/2017",
"12-24-2015" ]
</code></pre>
<p>I want to convert them into a single yyyy-mm-dd ("2018-06-22") format. I can split and then do the reformatting but I am looking for a more better way. Thanks in advance!</p>
</div>
<div class="post-text" itemprop="text">
<p>The module <code>dateutil</code> does exactly what you need:</p>
<pre><code>import dateutil.parser

dates = [ "2018-06-22",
"June 22, 2018",
"12/15/2017",
"12-24-2015" ]

for date in dates:
    print(dateutil.parser.parse(date).strftime("%Y-%m-%d"))
</code></pre>
<p>OUTPUT:</p>
<blockquote>
<p>2018-06-22<br/>
  2018-06-22<br/>
  2017-12-15<br/>
  2015-12-24  </p>
</blockquote>
<p>You can have a look at the <a href="https://dateutil.readthedocs.io/en/stable/#" rel="nofollow noreferrer">module</a>, or directly install it:</p>
<pre><code>pip install python-dateutil
</code></pre>
<p>formatters should be the same as <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer">datetime</a>.</p>
