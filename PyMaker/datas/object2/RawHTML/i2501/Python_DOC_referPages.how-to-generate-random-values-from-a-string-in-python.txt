<div class="post-text" itemprop="text">
<p>I have a string <code>'ABCDEF'</code>, I would like 4 <em>unique</em> characters to be generated randomly every time I run the program.</p>
<p>The result should give something like:</p>
<pre><code>BDAF

EDCB

CAFE

...
</code></pre>
<p>I could use <code>random.sample</code> but it would require a <code>list</code> to be used instead, are there ways of achieving the same result using a <code>string</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can indeed use <code>random.sample</code>; it'll give you a list, but it's easy enough to join the resulting list into a string:</p>
<pre><code>result = ''.join(random.sample("ABCDEF", 4))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the functions provided by Python's <a href="https://docs.python.org/3/library/random.html" rel="nofollow noreferrer"><code>random</code></a> module.
Use <a href="https://docs.python.org/3/library/random.html#random.sample" rel="nofollow noreferrer"><code>random.sample</code></a> to extract a random set of characters from the string and then <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>join</code></a> them:</p>
<pre><code>import random
source = "ABCDEFG"
result = "".join(random.sample(source, 4))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do this using <a href="https://docs.python.org/2/library/random.html#random.sample" rel="nofollow noreferrer">random</a> then join your resulting list with '' to create a string:</p>
<pre><code>import random
c = "ABCDEF"
''.join(random.sample(c, 4))
</code></pre>
<p>For input, you can use a string as in Python a string is also a list (list of characters) </p>
</div>
<div class="post-text" itemprop="text">
<p>There are 3 solutions (and probably 100 more):</p>
<ol>
<li><p>The"classic" <code>for</code> method appending everyone of the <code>N=4</code> letters to the same string and the <code>choice</code> method of the <code>random</code> module that picks one element among its arguments,</p></li>
<li><p>Using the <code>join</code> <code>method</code> to add in one time the elements generated by the <code>choices</code> method of the <code>random</code> module. The <code>choices</code> method is very versatile and can be used with several arguments (<a href="https://docs.python.org/3/library/random.html" rel="nofollow noreferrer">official documentation</a>),</p></li>
<li><p>Using the <code>join</code> <code>method</code> to add in one time the elements generated by the <code>sample</code> method of the <code>random</code> module. The <code>sample(set,num)</code>method picks randomly <code>num</code> elements in <code>set</code>.</p></li>
</ol>
<hr/>
<pre><code>import random
N=4
source_string = 'ABCDEF
</code></pre>
<p>Method1</p>
<pre><code>random_string_1=""
for it in range(0,N):
    random_string_1 = random_string_1 + random.choice(source_string)
</code></pre>
<p>Method2</p>
<pre><code>random_string_2 = "".join(random.choices(source_string, weights=None, 
cum_weights=None, k=N))
</code></pre>
<p>Method3</p>
<pre><code>random_string_3 = "".join(random.sample(source_string, N))
</code></pre>
