<div class="post-text" itemprop="text">
<p>I'm applying a number of regex patterns to text entries, and I wish for them to be automatically entered into a list.</p>
<p>e.g.</p>
<pre><code>import re
raw='NIH05484225_1_binders_list ID_054345 Proteincomplex /D1/ERBH1_ERV_ACWX'

regex= '.*Proteincomplex /[\S]+\/([\S]+)_([\S]+)_([\S]+)'
regex=re.compile(regex)


result = re.match(regex, raw)
answer=[]
answer.append (result.group(1))
answer.append (result.group(2))
answer.append (result.group(3))

print (answer)
</code></pre>
<p>Is there a way to have <code>answer</code> collect all the captured results?
As in </p>
<p><code>answer.extend (result.allgroups()) # does not work, of course</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You may use <a href="https://docs.python.org/3/library/re.html#re.Match.groups" rel="nofollow noreferrer"><code>Match.groups()</code></a>:</p>
<blockquote>
<p>Return a tuple containing all the subgroups of the match, from 1 up to however many groups are in the pattern. The <em>default</em> argument is used for groups that did not participate in the match; it defaults to <code>None</code>.</p>
</blockquote>
<p>So, you may modify the code this way:</p>
<pre><code>import re
raw='NIH05484225_1_binders_list ID_054345 Proteincomplex /D1/ERBH1_ERV_ACWX'
regex=re.compile(r'.*Proteincomplex\s+/\S+\/(\S+)_(\S+)_(\S+)')
result = re.match(regex, raw)
if result:
    print (answer.groups())
</code></pre>
<p>Note you do not need to wrap each single <code>\S</code> with a character class and a <code>\s+</code> instead of a space will let you match any one or more whitespace (it is considered more readable and fail-safe, especially if you plan to use <code>re.VERBOSE</code> / <code>re.X</code> later to insert comments or formatting whitespace into the pattern).</p>
</div>
