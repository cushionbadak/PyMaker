<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/stdtypes.html#bytes.decode" rel="nofollow noreferrer"><code>bytes.decode()</code></a> method to convert bytes into a string:</p>
<p>Change:</p>
<pre><code>filename=(repr(data))
</code></pre>
<p>to:</p>
<pre><code>filename=repr(data).decode()
</code></pre>
</div>
