<p>How can I create Dynamic size Multidimentional array in python?</p>
<p>Aim is to create a dynamic size arrays within arrays, e.g.:</p>
<pre><code>ExampleArray{
   book1 : { key:val }
   book2 : { key:val }
}
</code></pre>
<p>This returns an error:</p>
<pre><code>ExampleArray = {}
ExampleArray['book1']['key'] = 'val';
</code></pre>
<p>Why?</p>
</div>
<div class="post-text" itemprop="text">
<p>replace by</p>
<pre><code>ExampleArray = {}
ExampleArray['book1'] = {}
ExampleArray['book1']['key'] = 'val'
</code></pre>
<p>when you do  <code>ExampleArray['book1']</code> you are trying to access it but no to affect it, therefore since the key does not exists it throws an exception</p>
<p>you have to affect a value to <code>ExampleArray['book1']</code> (in this case <code>dict()</code>)</p>
<p>PS. loose the ; at the end of the lines. you are not doing C or C++</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow noreferrer">https://docs.python.org/3/library/collections.html#collections.defaultdict</a></p>
<pre><code>from collections import defaultdict

ExampleArray = defaultdict(dict)

ExampleArray['book1']['key'] = 'val'

print(ExampleArray) # defaultdict(&lt;class 'dict'&gt;, {'book1': {'key': 'val'}})
</code></pre>
