<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/3/library/glob.html" rel="nofollow noreferrer"><code>glob</code></a> and the <code>os</code> module to iterate over every file in a given <code>path</code> with a given file extension:</p>
<pre><code>import glob
import os

path = "F:/Process/Process_files/xmls/"

for filename in glob.glob(os.path.join(path, "*.xml")):
    with open(filename) as open_file:
        content = open_file.read()

    soup = BeautifulSoup(content, "html.parser")
</code></pre>
<p><strong>Tip:</strong> Use the <a href="https://docs.python.org/3/reference/compound_stmts.html#the-with-statement" rel="nofollow noreferrer"><code>with</code></a> statement so the file gets automatically closed at the end.</p>
<p><strong>Source:</strong> <a href="https://stackoverflow.com/questions/18262293/how-to-open-every-file-in-a-folder">Open Every File In A Folder</a></p>

<p>So you need to iterate over files in the folder? You can try something like this:</p>
<pre><code>for file in os.listdir(path):
    filepath = os.path.join(path, file)
    with open(filepath) as fp:
        contents = fp.read()
        soup = BeautifulSoup(contents, 'html.parser')
</code></pre>
