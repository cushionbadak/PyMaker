<p>I would like to iterate through a list of strings and replace each instance of a character ('1', for example) with a word. I am confused why this would not work.</p>
<pre><code>for x in list_of_strings:
    x.replace('1', 'Ace')
</code></pre>
<p>Side note, the strings within the lists are multiple characters long. ('1 of Spades)</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a list comprehension:</p>
<pre><code>list_of_strings = [x.replace('1', 'Ace') for x in list_of_strings]
</code></pre>
<p>This is natural in Python. There is no significant benefit in changing your original list directly; both methods will have O(<em>n</em>) time complexity.</p>
<p>The reason your code does not work is <code>str.replace</code> does not work in place. It returns a copy, as <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow noreferrer">mentioned in the docs</a>. You <em>can</em> iterate over a <code>range</code> object to modify your list:</p>
<pre><code>for i in range(len(list_of_strings)):
    list_of_strings[i] = list_of_strings[i].replace('1', 'Ace')
</code></pre>
<p>Or use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a>:</p>
<pre><code>for idx, value in enumerate(list_of_strings):
    list_of_strings[idx] = value.replace('1', 'Ace')
</code></pre>
</div>
