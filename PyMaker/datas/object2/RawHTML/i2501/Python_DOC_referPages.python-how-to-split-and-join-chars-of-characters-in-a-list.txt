<div class="post-text" itemprop="text">
<p>You should concatenate new characters to a string instead of appending them to a list. Only append a new string to the output list at the start of a tag:</p>
<pre><code>with open('doc.txt', 'r') as f:
    empty_tag_list = []
    for line in f:
        isTag = False
        for chars in line:
            if chars == "&lt;":
                isTag = True
                empty_tag_list.append('')
            elif chars == "&gt;":
                isTag = False
            elif isTag:
                empty_tag_list[-1] += chars
    print(empty_tag_list)
</code></pre>

<div class="post-text" itemprop="text">
<p>Instead of manually iterating and checking each character, I'd suggest using a <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">regular expression</a>:</p>
<pre><code>import re
with open("doc.txt") as f:
    text = "".join(f)
    tags = re.findall(r'&lt;(/?\w+).*?&gt;', text)
</code></pre>
<p>Here, <code>&lt;(/?\w+).*?&gt;</code> means <code>&lt;</code> followed by optional <code>/</code> and word-characters (to be kept), then optionally more stuff, and a <code>&gt;</code>. Afterwards, <code>tags</code> is <code>['html', 'head', 'title', '/title', '/head', 'body', '/body', '/html']</code>.</p>
<p>For more complex tasks you could also consider using a proper HTML parser like Beautiful Soup, but for this, regex might be good enough.</p>
