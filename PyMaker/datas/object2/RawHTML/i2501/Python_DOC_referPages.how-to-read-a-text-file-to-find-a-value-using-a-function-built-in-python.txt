</code></pre>

<div class="post-text" itemprop="text">
<p>Simple way to do it:</p>
<pre><code>def todict(vals,file):
    d={}
    for line in file:
        if line.rstrip() in vals:
            d[line.rstrip()]=None

    return d

file = open("Text.txt",'r')
print(todict(['LC1','LC2'],file))
</code></pre>
<p>But still the easiest way is:</p>
<pre><code>def todict(vals,file):
    return {}.fromkeys([i.rstrip() for i in file if i.rstrip() in vals])

file = open("Text.txt",'r')
print(todict(['LC1','LC2'],file))
</code></pre>
<p>Both reproduce:</p>
<pre><code>{'LC1': None, 'LC2': None}
</code></pre>
