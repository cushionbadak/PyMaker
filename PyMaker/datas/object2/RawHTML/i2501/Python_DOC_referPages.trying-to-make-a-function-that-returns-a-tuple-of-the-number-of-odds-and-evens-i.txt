<p>Use a list comprehension and exploit the neat function <code>list.count(value)</code>. No need for for-loops:</p>
<pre><code>def odds_evens(lista):
    remainder = [x % 2 for x in lista]
    return (remainder.count(0), remainder.count(1))

&gt;&gt;&gt; lista = [1,2,3,5,7,8,9,10,12,13,17]
&gt;&gt;&gt; odds_evens(lista)
(4, 7)
</code></pre>
