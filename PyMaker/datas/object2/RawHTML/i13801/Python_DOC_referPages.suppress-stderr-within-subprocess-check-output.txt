<p>I'm trying to find a way to ignore the stderr stream (something similar to <code>2&gt; /dev/null</code>):</p>
<pre><code>output = subprocess.check_output("netstat -nptl".split())
</code></pre>
<p>What should I add to the above command to achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Just tell <code>subprocess</code> to redirect it for you:</p>
<pre><code>import os
import subprocess

with open(os.devnull, 'w') as devnull:
    output = subprocess.check_output("netstat -nptl".split(), stderr=devnull)
</code></pre>
<p>Note that Python 3 adds a <a href="https://docs.python.org/3/library/subprocess.html#subprocess.DEVNULL"><code>subprocess.DEVNULL</code> object</a>, saving you having to open this yourself.</p>
</div>
