<div class="post-text" itemprop="text">
<p>It looks like you're trying to write text (strings) to this file. If so, you'll need to specify a non-binary mode when creating the <code>TemporaryFile</code> by changing</p>
<pre><code>temp = tempfile.TemporaryFile()
</code></pre>
<p>to</p>
<pre><code>temp = tempfile.TemporaryFile(mode='w')
</code></pre>
<p>See <a href="https://stackoverflow.com/questions/26945613/str-does-not-support-the-buffer-interface-python3-from-python2">this answer</a> for more details and the <a href="https://docs.python.org/3/library/tempfile.html#tempfile.TemporaryFile" rel="nofollow noreferrer">tempfile docs</a> for the fact that it defaults to expecting bytes, not strings.</p>
