<p>I have a block of text with the format 'abc,def,ghi,jkl, etc'
I need to turn each 3-digit bit into something else, in this case a letter. However, using a for loop, I can't find a way of looking beyond the current character. I'm looking for something a little like this:</p>
<pre><code>for char in message[::4]:
   if char == 'a':
      if char(+1) == 'b':
         newmessage += 1
</code></pre>
<p>I'd prefer to use a for loop, but if there's no way of doing it, a while loop would be fine too.</p>

<div class="post-text" itemprop="text">
<pre><code>for word in message.split(","):
    for ind, char in enumerate(word[:-1], 1):
        if char == 'a' and word[ind] == "b":
            newmessage += 1
</code></pre>
<p>If you want the sum use a generator expression with sum:</p>
<pre><code>new_message = sum(char == 'a' and word[ind] == "b" for word in message.split(",")
                  for ind, char in enumerate(word[:-1], 1))
</code></pre>
