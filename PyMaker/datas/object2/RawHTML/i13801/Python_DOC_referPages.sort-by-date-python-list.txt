<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/time.html#time.strptime" rel="nofollow"><code>time.strptime</code></a></p>
<pre><code>&gt;&gt;&gt; from time import strptime
&gt;&gt;&gt; sorted(d, key = lambda i: strptime(i.split('@')[0], '%d-%m-%Y'))
['7-8-2015@Fri, Aug 07, 2015',
 '8-8-2015@Sat, Aug 08, 2015',
 '9-8-2015@Sun, Aug 09, 2015',
 '10-8-2015@Mon, Aug 10, 2015',
 '11-8-2015@Tue, Aug 11, 2015',
 '12-8-2015@Wed, Aug 12, 2015',
 '13-8-2015@Thu, Aug 13, 2015',
 '14-8-2015@Fri, Aug 14, 2015']
</code></pre>
<p>Basically, you can split at the <code>'@'</code> character, then all of your strings can be used to create <a href="https://docs.python.org/3/library/time.html#time.struct_time" rel="nofollow"><code>struct_time</code></a> objects. You can use a lambda in the <code>key</code> argument of <code>sorted</code> to sort by these <code>struct_time</code> objects.</p>
<p>Also note that you can reverse the sort order using the <code>reverse</code> argument</p>
<pre><code>&gt;&gt;&gt; sorted(d, key = lambda i: strptime(i.split('@')[0], '%d-%m-%Y'), reverse=True)
['14-8-2015@Fri, Aug 14, 2015',
 '13-8-2015@Thu, Aug 13, 2015',
 '12-8-2015@Wed, Aug 12, 2015',
 '11-8-2015@Tue, Aug 11, 2015',
 '10-8-2015@Mon, Aug 10, 2015',
 '9-8-2015@Sun, Aug 09, 2015',
 '8-8-2015@Sat, Aug 08, 2015',
 '7-8-2015@Fri, Aug 07, 2015']
</code></pre>
