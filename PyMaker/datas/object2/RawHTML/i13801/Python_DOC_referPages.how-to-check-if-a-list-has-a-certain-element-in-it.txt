<p>Say I don't know what is in <code>my_list</code>, how can I check if <code>"bye"</code> is an element in my list.  And if <code>"bye"</code> is an element in <code>my_list</code>, then how can I know which index it has (for this case <code>"bye"</code> is in <code>my_list[1]</code>)?</p>

<pre><code>"bye" in my_list
</code></pre>

<p>then how can I know which index it is</p>

<pre><code>my_list.index("bye")

<p>You can skip the first step because if you try to find the index of a value that isn't in the list, you will get a value error which you can handle like this:</p>
<pre><code>try:
    my_list.index("not here")
except ValueError:
    print("'not here' isn't in the list!!")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>try:
    idx = ['hi', 'bye', 'hi'].index('bye')
except ValueError:
    print('Not in list.')
else:
    print('In list at index', idx)
</code></pre>
</div>

<p>you can try <code>list.index</code>.</p>
<pre><code>try:
    i = my_list.index('hi')
except ValueError:
    i = False
</code></pre>
<p>i will hold the index if <code>hi</code> is in the list, otherwise it will be False, because <code>list.index</code> raises ValueErrror if it can't return the index.</p>

<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/reference/expressions.html#in" rel="nofollow">in</a> operator to check if a specific item exists in a list. To find the index of any item in a list just use the <code>.index()</code> function. If you are a beginner in Python you can use the below method else you can check out <a href="http://www.tutorialspoint.com/python/python_exceptions.htm" rel="nofollow">exception handling</a>.</p>
<pre><code>my_list = ["hi", "bye", "hi"]
if 'bye' in my_list:
   print "Word found at index %i",my_list.index('bye')
else
   print 'Word not Found !'
</code></pre>
</div>
