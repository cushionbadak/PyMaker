<p>Exactly right :-)</p>
<pre><code>people = {
    "Paul": [1,2],
    "Joe": [3,4]
}

def myFunc(name):
    return people[name]

print myFunc("Paul")
print myFunc("Joe")
</code></pre>
<p>Of course, you can also cut out the <code>myFunc</code> middleman and directly do <code>print people["Paul"]</code>.</p>

<p>You can use <a href="https://docs.python.org/3/library/functions.html#globals" rel="nofollow"><code>globals()</code></a> function to access to global variables inside the function and then use a <a href="https://www.python.org/dev/peps/pep-0289/" rel="nofollow"><em>generator expression</em></a> within <a href="https://docs.python.org/3/library/functions.html#next" rel="nofollow"><code>next</code></a> function to loop over its items then check if your name is in a global name (the key) then print its corresponding value :</p>
<pre><code>&gt;&gt;&gt; personPaul = [1,2]
&gt;&gt;&gt; personJoe = [3,4]
&gt;&gt;&gt; def myFunc(name):
...     return next((v for k,v in globals().items() if name in k),None)
... 
&gt;&gt;&gt; myFunc('Paul')
[1, 2]
&gt;&gt;&gt; myFunc('jack')
&gt;&gt;&gt; 
</code></pre>
