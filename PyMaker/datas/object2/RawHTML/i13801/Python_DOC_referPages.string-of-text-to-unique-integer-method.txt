<p>Is there a method that converts a string of text such as <code>'you'</code> to a number other than </p>
<pre><code>y = tuple('you')
for k in y:
  k = ord(k)
</code></pre>
<p>which only converts one character at a time?</p>

<div class="post-text" itemprop="text">
<p>In order to convert a string to a number (and the reverse), you should first always work with <code>bytes</code>. Since you are using Python 3, strings are actually Unicode strings and as such may contain characters that have a <code>ord()</code> value higher than 255. <code>bytes</code> however just have a single byte per character; so you should always convert between those two types first.</p>
<p>So basically, you are looking for a way to convert a <code>bytes</code> string (which is basically a list of bytes, a list of numbers 0â€“255) into a single number, and the inverse. You can use <a href="https://docs.python.org/3/library/stdtypes.html#int.to_bytes" rel="noreferrer"><code>int.to_bytes</code></a> and <a href="https://docs.python.org/3/library/stdtypes.html#int.from_bytes" rel="noreferrer"><code>int.from_bytes</code></a> for that:</p>
<pre><code>import math
def convertToNumber (s):
    return int.from_bytes(s.encode(), 'little')

def convertFromNumber (n):
    return n.to_bytes(math.ceil(n.bit_length() / 8), 'little').decode()
</code></pre>
<pre><code>&gt;&gt;&gt; convertToNumber('foo bar baz')
147948829660780569073512294
&gt;&gt;&gt; x = _
&gt;&gt;&gt; convertFromNumber(x)
'foo bar baz'
</code></pre>

<div class="post-text" itemprop="text">
<ol>
<li>You don't need to convert the string into tuple</li>
<li><code>k</code> is overwritten. Collect items using something like list comprehension:</li>
</ol>
<hr/>
<pre><code>&gt;&gt;&gt; text = 'you'
&gt;&gt;&gt; [ord(ch) for ch in text]
[121, 111, 117]
</code></pre>
<p>To get the text back, use <a href="https://docs.python.org/3/library/functions.html#chr" rel="nofollow"><code>chr</code></a>, and join the characters using <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join</code></a>:</p>
<pre><code>&gt;&gt;&gt; numbers = [ord(ch) for ch in text]
&gt;&gt;&gt; ''.join(chr(n) for n in numbers)
'you'
</code></pre>
</div>

<p>Treat the string as a base-255 number.</p>
<pre><code># Reverse the digits to make reconstructing the string more efficient
digits = reversed(ord(b) for b in y.encode())
n = reduce(lambda x, y: x*255 + y, digits)

new_y = ""
while n &gt; 0:
    n, b = divmod(n, 255)
    new_y += chr(b)
assert y == new_y.decode()
</code></pre>
<p>(Note this is essentially the same as poke's answer, but written explicitly rather than using available methods for converting between a byte string and an integer.)</p>
