<p>When x1 is set you are using quotes making it a string.  Remove the quotes and it will be a numeric.  This is important later when you compare it to a numeric value.</p>
<pre><code>x1 = random.choice([10,172,192])
</code></pre>
<p>Then the comparison elif (x1 == 172) will work as expected.</p>

<p>Can be done with this code?</p>
<pre><code>from random import randint
def randomPrivateIPaddress():
    network_type = randint(0, 2)
    if network_type == 0:
        return "10.%d.%d.%d" % (randint(0, 255), randint(0, 255), randint(0, 255))
    elif network_type == 1:
        return "172.%d.%d.%d" % (randint(16, 31), randint(0, 255), randint(0, 255))
    else:
        return "192.168.%d.%d" % (randint(0,255), randint(0,255))
</code></pre>
<p>Then, you can use those string with ipaddress module.</p>
