<p>I'm getting the error <code>list assignment index out of range</code> when trying to assign values to an array.  Why am I getting this error and how do I fix it?</p>
<pre><code>array_questions = input("How many words would you like to search for? (maximum of 5): ")
to_search_words = []

x = int(array_questions)
y = 0
z = y + 1

while x &gt; 0:
    y + 1
    str = ("%d. " % z)
    to_search_words[y] = input(str)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Becuase you're using list, you may want to use <a href="https://docs.python.org/3/tutorial/datastructures.html#more-on-lists" rel="nofollow"><code>append</code></a>, which is part of every list's API:</p>
<blockquote>
<p>Add an item to the end of the list. Equivalent to a[len(a):] = [x].</p>
</blockquote>
<p>Example:</p>
<pre><code>while x &gt; 0:
    # not needed # y + 1
    str = ("%d. " % z)
    to_search_words.append(input(str))
    x -= 1 # prevent infinite loop
</code></pre>
</div>
