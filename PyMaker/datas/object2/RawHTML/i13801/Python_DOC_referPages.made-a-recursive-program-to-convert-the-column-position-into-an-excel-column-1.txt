<p>The problem here is that:</p>
<pre><code>if to_print is 0:
</code></pre>
<p>happens <em>before you've converted <code>to_print</code> to an integer</em>. Also, you should really be using equality (<code>==</code>) not identity (<code>is</code>); small integers are interned in CPython, but this is an implementation detail you shouldn't rely on.</p>
<p>The simplest fix is:</p>
<pre><code>if to_print == '0':  # compare to string, and by equality
</code></pre>
<p>but a better way is to convert the number first, and use the fact that zero numerical values <a href="https://docs.python.org/3/library/stdtypes.html#truth-value-testing" rel="nofollow">evaluate false-y</a>:</p>
<pre><code>def alpha_convert(to_print):  # note typo in function name
    """A docstring would be nice, too!"""
    to_print = int(to_print)
    return chr(to_print + 64) if to_print else 'Z'
</code></pre>
