How can I increment the count value (initially 0)?</p>
<pre><code> if serialIndex &lt; 200: 
     serialBuffer[serialIndex += 1] = serialData
  # i get this error         ^ ']' expected

<div class="post-text" itemprop="text">
<p>You can't do this on just one line. You'll have to split it up.</p>
<pre><code>if serialIndex &lt; 200: 
    serialBuffer[serialIndex] = serialData
    serialIndex += 1
</code></pre>
