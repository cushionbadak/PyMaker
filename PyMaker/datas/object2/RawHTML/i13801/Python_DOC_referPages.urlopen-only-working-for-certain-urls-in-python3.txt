<div class="post-text" itemprop="text">
<p>If you look closer at the error message you'll see that it is a HTTP error and a special one:</p>
<pre><code>HTTP Error 403: Forbidden
</code></pre>
<p>So you talked to the server and got your response back but you don't know why you were denied.</p>
<p>You can get a more detailed message in an HTML returned by the server with something like this:</p>
<pre><code>from urllib.request import urlopen
from urllib.error import HTTPError

try:
    html = urlopen("http://www.stackoverflow.com/")
except HTTPError as e:
    print(e.read().decode('utf-8'))

html.read()
</code></pre>
<p>For me it says:</p>
<pre><code>&lt;h2 data-translate="what_happened"&gt;What happened?&lt;/h2&gt;
&lt;p&gt;The owner of this website (www.stackoverflow.com) has banned your access based on your browser's signature (213702c58d2116a6-ua48).&lt;/p&gt;
</code></pre>
<p>You can treat <code>HTTPError</code> as a file object (<a href="https://docs.python.org/3/library/urllib.error.html#urllib.error.HTTPError" rel="nofollow">https://docs.python.org/3/library/urllib.error.html#urllib.error.HTTPError</a>):</p>
<blockquote>
<p>Though being an exception (a subclass of URLError), an HTTPError can
  also function as a non-exceptional file-like return value (the same
  thing that urlopen() returns). This is useful when handling exotic
  HTTP errors, such as requests for authentication.</p>
</blockquote>
