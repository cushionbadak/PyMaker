<div class="post-text" itemprop="text">
<p>I have a text file and a list of integers in python. I want to read the contents of the file line by line whereas parse the elements of the list simultaneous.</p>
<p>Here is an example of contents of the text file (myfile.txt):</p>
<pre><code>line1
line2
line3
</code></pre>
<p>and the list is:</p>
<pre><code>mylist = (1, 2, 3)
</code></pre>
<p>for example, I want to have a loop like below:</p>
<pre><code>for line, item in open(myfile.txt), mylist:
  print line
  print item
</code></pre>
<p>and I expect to see this output:</p>
<pre><code>line1
1
line2
2
line3
3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow">zip</a>:</p>
<pre><code>for line, item in zip (open ('myfile.txt'), mylist):
  print (line)
  print (item)
</code></pre>
<p>the loop will stop when the shortest of the iterables is exhausted. </p>
</div>
<div class="post-text" itemprop="text">
<p>If you don't want to read the whole file in one go you could use <a href="https://docs.python.org/2/library/itertools.html#itertools.izip" rel="nofollow"><code>itertools.izip</code></a> if you using python2:</p>
<pre><code>for line, item in itertools.izip(open("myfile.txt", mylist)):
    print(line)
    print(item)
</code></pre>
<p>If you're using python3 you should use <a href="https://docs.python.org/3.4/library/functions.html#zip" rel="nofollow"><code>zip</code></a> instead:</p>
<pre><code>for line, item in zip(open("myfile.txt", mylist)):
    print(line)
    print(item)
</code></pre>
<p>If you don't mind reading the file in one go you could use <code>zip</code> anyway (both for python2 and 3).</p>
</div>
<span class="comment-copy">by the way, mylist is a tuple here (), use [] to create lists.</span>
