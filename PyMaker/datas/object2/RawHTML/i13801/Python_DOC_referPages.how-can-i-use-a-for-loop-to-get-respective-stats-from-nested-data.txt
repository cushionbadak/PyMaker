<div class="post-text" itemprop="text">
<p>To get the dates:</p>
<pre><code>data["stats"]
</code></pre>
<p>To get the amounts:</p>
<pre><code>data["tickers_data"][0]["amount"]
</code></pre>
<p>Once you have those two lists, you can use the <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><code>zip</code> function</a> to combine the two lists into pairs of elements you can iterate over:</p>
<pre><code>data = {
    u'stats': [
        u'Jul 23, 2015',
        u'Jul 24, 2015',
        u'Jul 25, 2015',
        u'Jul 26, 2015'
    ],
    u'tickers_data': [
        {u'amount': [936, 804, 977, 954], u'name': u'adv'}
    ]
}                                                                    
days = data['stats']
ticker = data['tickers_data'][0]['amount']

for day, amount in zip(days, ticker):
    print(day, "-&gt;", amount)
</code></pre>
<p>Output:</p>
<pre><code>$ python3 x.py
Jul 23, 2015 -&gt; 936
Jul 24, 2015 -&gt; 804
Jul 25, 2015 -&gt; 977
Jul 26, 2015 -&gt; 954
</code></pre>

<p>You could use something like the following to display it:</p>
<pre><code>data_dict = {
    u'stats': [
        u'Jul 23, 2015',
        u'Jul 24, 2015',
        u'Jul 25, 2015',
        u'Jul 26, 2015'
    ],
    u'tickers_data': [
        {u'amount': [936, 804, 977, 954], u'name': u'adv'}
    ]
}

print "Ticker name '%s':" % data_dict[u'tickers_data'][0][u'name']

dates = data_dict[u'stats']
amounts = data_dict['tickers_data'][0][u'amount']

for date, amount in zip(dates,amounts):
    print "  Date: %s, Amount: %s" % (date, amount)
</code></pre>
<p>This gives the following output:</p>
<pre><code>Ticker name 'adv':
  Date: Jul 23, 2015, Amount: 936
  Date: Jul 24, 2015, Amount: 804
  Date: Jul 25, 2015, Amount: 977
  Date: Jul 26, 2015, Amount: 954
</code></pre>
<p>If you wanted to store the data in a file, you could add the following:</p>
<pre><code>with open("ticker_data.txt", "w") as f_output:
    f_output.write("Ticker name '%s':\n" % data_dict[u'tickers_data'][0][u'name'])

    for date, amount in zip(dates,amounts):
        f_output.write("  Date: %s, Amount: %s\n" % (date, amount))
</code></pre>
