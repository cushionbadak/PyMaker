<p>Use <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow"><code>range</code></a> to make an iterable (list if you're using Python 2.x) which will be used for iteration.</p>
<p>and you need to pass two integer to <a href="https://docs.python.org/3/library/random.html#random.randint" rel="nofollow"><code>random.randint</code></a>, not an interable (or list):</p>
<pre><code>from random import randint

number_of_trials = 100
range_of_random_numbers = 1000
for each in range(number_of_trials):

<div class="post-text" itemprop="text">
<p>One issue and one suggestion in your code -</p>
<ol>
<li><p>You are doing - <code>for each in number_of_trails:</code> - this is wrong, you are trying to iterate over an integer, you should do - <code>for each in range(number_of_trails):</code> .</p></li>
<li><p>The best to use here is <a href="https://docs.python.org/2/library/random.html#random.choice" rel="nofollow"><code>random.choice()</code></a> (Instead of <code>randint()</code>). Especially if the <code>range_of_random_numbers</code> is not continuous.</p></li>
</ol>
<p>Example -</p>
<pre><code>from random import choice
number_of_trials = 100
range_of_random_numbers = 1000
for each in range(number_of_trials):
    print choice(range(range_of_random_numbers))
</code></pre>
<hr/>
<p>From <a href="https://docs.python.org/2/library/random.html#random.choice" rel="nofollow">documentation of <code>random.choice()</code></a> -</p>
<blockquote>
<p>Return a random element from the non-empty sequence seq.</p>
</blockquote>
