<div class="post-text" itemprop="text">
<p>You need to use <code>or</code>:</p>
<pre><code>if end == "end" or end == "End":
    ....
</code></pre>
<p>if you want to check the string case-insensitive, use <a href="https://docs.python.org/3/library/stdtypes.html#str.casefold" rel="nofollow"><code>str.casefold</code></a>:</p>
<pre><code>if end.casefold() == "end":
    ...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The condition is wrong -</p>
<pre><code>if end == ("end") and ("End"):
</code></pre>
<p>This would only evaluate to true (when end is <code>'end'</code> ), because this would get translated as -</p>
<pre><code>if ((end == "end") and "End")
</code></pre>
<p>And any non-empty string in python is True-like (has true value in boolean context.)</p>
<p>To check for all cases, best thing to do would be to check your variable's <code>.lower()</code> with <code>'end'</code> , example -</p>
<pre><code>if end is not None and end.lower() == "end":
</code></pre>
<p>And since you are taking input using <code>input()</code> function, you actually do not need the <code>is not None</code> part, so you can do -</p>
<pre><code>if end.lower() == "end":
</code></pre>
