<div class="post-text" itemprop="text">
<p><a href="https://github.com/python/cpython/blob/v3.6.3/Lib/site.py#L372-L373" rel="nofollow noreferrer">bulit-in <code>help()</code></a> is <a href="https://github.com/python/cpython/blob/v3.6.3/Lib/_sitebuiltins.py#L88-L103" rel="nofollow noreferrer">a wrapper around <code>pydoc.Helper</code></a>, it writes to <code>stdout</code> by <a href="https://github.com/python/cpython/blob/v3.6.3/Lib/pydoc.py#L1841-L1843" rel="nofollow noreferrer">default</a>, you could temporarily redirecting <code>sys.stdout</code> to a file:</p>
<pre><code>&gt;&gt;&gt; import contextlib
&gt;&gt;&gt; with contextlib.redirect_stdout(open('browser_help.txt', 'w')):
...     help(browser)
</code></pre>
<p>or you could call <code>pydoc.Helper</code> directly, :</p>
<pre><code>&gt;&gt;&gt; import pydoc
&gt;&gt;&gt; with open('browser_help.txt', 'w') as f:
...     h = pydoc.Helper(output=f)
...     h(browser)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can try to change the current stdout:</p>
<pre><code>import sys
sys.stdout = open('webdriver.md', 'w')
help(browser)
</code></pre>
<p>Starting from <strong>Python 3.4</strong>, you can also use <a href="https://docs.python.org/3/library/contextlib.html#contextlib.redirect_stdout" rel="nofollow noreferrer"><code>contextlib.redirect_stdout</code></a>:</p>
<pre><code>from contextlib import redirect_stdout
with redirect_stdout(open('webdriver.md', 'w')):
    help(browser)
</code></pre>
