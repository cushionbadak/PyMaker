<p>I have a variable x and whose type is :</p>
<pre><code>type(x)
&gt;&gt; &lt;class '__main__.XmlListConfig'&gt;
</code></pre>
<p>in the next part, I want to see if the type of x is <code>&lt;class'__main__.XmlListConfig'&gt;</code>
I am unable to compare and see it 
So far, I have tried:</p>
<pre><code>if type(x) == "__main__.XmlListConfig":
</code></pre>
<p>This does not work because ofcourse I am comparing it to a string. 
Any Suggesting will help and please feel free to ask me for more clarification. </p>

<div class="post-text" itemprop="text">
<p>To get around the string comparison problem you could try </p>
<pre><code>if str(type(x)) == "&lt;class '__main__.XmlListConfig'&gt;":
    do_stuff()
</code></pre>
<p>although that is inconsistent and hard to read, so better to use <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow noreferrer"><code>isinstance</code></a>:</p>
<pre><code>if isinstance(x, XmlListConfig):
    do_stuff()
</code></pre>
</div>
<span class="comment-copy">Look at the type's <code>__name__</code> attribute. This is a terrible code smell, though, because it removes what little type safety Python provides.</span>
<span class="comment-copy"><code>isinstance(x, XmlListConfig)</code> - or, in the unlikely event you want an exact type check rather than accepting subclasses: <code>type(x) is XmlListConfig</code>.</span>
