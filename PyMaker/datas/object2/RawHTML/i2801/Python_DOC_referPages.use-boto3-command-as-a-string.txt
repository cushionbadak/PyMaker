<p>We can use <code>boto3</code> package to get the data from mTurk account (or do anything else in AWS). For example:</p>
<pre><code>client = boto3.client('mturk')
balance = client.get_account_balance()
</code></pre>
<p>Is there a way to use boto3 command as a string? Something like:</p>
<pre><code>balance = client.get_command('get_account_balance')
</code></pre>
<p><code>get_command</code> is totally just for illustrative purposes here of course.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could use the built-in <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer"><code>getattr</code></a>:</p>
<pre><code>def get_command(client, command):
    return getattr(client, command)()
</code></pre>
</div>
