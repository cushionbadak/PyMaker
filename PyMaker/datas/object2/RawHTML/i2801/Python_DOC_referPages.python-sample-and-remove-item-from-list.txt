<pre><code>l = [1,2,4,6]

<div class="post-text" itemprop="text">
<p>Use <code>random.randint</code> to get a random index and use <code>pop</code> to get the element with that index from the list</p>
<pre><code>&gt;&gt;&gt; import random
&gt;&gt;&gt; l = [1,2,4,6]
&gt;&gt;&gt; idx = random.randint(0, len(l)-1)
&gt;&gt;&gt; r = l.pop(idx)
&gt;&gt;&gt; r
4
&gt;&gt;&gt; l
[1, 2, 6]
</code></pre>

<div class="post-text" itemprop="text">
<p>Shuffle using <code>random.shuffle</code> and then <code>pop</code> from list:</p>
<pre><code>import random

lst = [1, 2, 4, 6]
random.shuffle(lst)
r = lst.pop()

print(r)    # 4
print(lst)  # [1, 2, 6]
</code></pre>

<div class="post-text" itemprop="text">
<p>The easiest way I can think of is to use <a href="https://docs.python.org/3/library/random.html#random.shuffle" rel="nofollow noreferrer"><code>shuffle()</code></a> to randomise the elements position in the list and then use <code>pop()</code> as and when required.</p>
<pre><code>&gt;&gt;&gt; from random import shuffle
&gt;&gt;&gt; shuffle(l)
&gt;&gt;&gt; l.pop()
</code></pre>
<p>#driver values :</p>
<pre><code>IN : l = [1,2,4,6]
OUT : 4
</code></pre>
<hr/>
<p>From PyDocs :</p>
<blockquote>
<p>random.shuffle(x[, random])</p>
<pre><code>Shuffle the sequence x in place.
</code></pre>
</blockquote>

<div class="post-text" itemprop="text">
<p>Use the function <code>choice</code> from the module <code>random</code> and use <code>remove</code> to remove the item from list.</p>
<pre><code>&gt;&gt;&gt; from random import choice as get
&gt;&gt;&gt; l = [1,2,3,4,6]
&gt;&gt;&gt; r = get(l)
&gt;&gt;&gt; r
3
&gt;&gt;&gt; l.remove(r)
&gt;&gt;&gt; l
[1, 2, 4, 6]
</code></pre>
<p>In short:</p>
<pre><code>from random import choice as get
l = [1,2,3,4,6]
r = get(l)
l.remove(r)
</code></pre>
