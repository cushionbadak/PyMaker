<div class="post-text" itemprop="text">
<p>I'm trying to format the datetime in python, and this is what I was trying: </p>
<pre><code>import time

datestr = "8-DEC-17"
v=time.strptime(datestr,"%d-%b-%y")
l = time.mktime(v) 
print(time.strftime("%d/%m/%y ", time.gmtime(l)))
</code></pre>
<p>The output of this code is :
<strong>07/12/17</strong> which is not the one I want</p>
<p>I am expecting : <strong>08/12/17</strong></p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>datetime</code> which is a bit shorter and gives the result you want:</p>
<pre><code>from datetime import datetime 
datetime.strptime(datestr, "%d-%b-%y").strftime("%d/%m/%y")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>According to <a href="https://docs.python.org/3/library/time.html#time.mktime" rel="nofollow noreferrer">https://docs.python.org/3/library/time.html#time.mktime</a>, <code>mktime()</code> is the inverse function of <code>localtime()</code>. So, you need to use <code>localtime()</code> instead of <code>gmtime()</code> to print the result:</p>
<pre class="lang-python prettyprint-override"><code>print(time.strftime("%d/%m/%y ", time.localtime(l)))
</code></pre>
<p>outputs <code>08/12/17</code></p>
</div>
<div class="post-text" itemprop="text">
<pre><code>from dateutil.parser import parse
datestr = "8-DEC-17"
dt = parse(datestr)
print(dt.strftime("%d/%m/%y"))
</code></pre>
