<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/glob.html" rel="nofollow noreferrer">https://docs.python.org/3/library/glob.html</a></p>
<blockquote>
<p>If <em>recursive</em> is true, the pattern “<code>**</code>” will match any files and <strong>zero or more directories and subdirectories</strong>. If the pattern is followed by an <code>os.sep</code>, only directories and subdirectories match.</p>
</blockquote>
<p>So, the following files will match your pattern when <code>recursive</code> is true:</p>
<ul>
<li><code>27-Jul/a/foo.csv</code></li>
<li><code>27-Jul/c/d/baz.csv</code></li>
<li><code>27-Jul/boo.csv</code></li>
</ul>
<p>If you use a single <code>*</code> instead of <code>**</code>, only <code>27-Jul/a/foo.csv</code> will match. <code>*</code> means “exactly one file or subdirectory,” and <code>**</code> means “zero or more directories or subdirectories.”</p>
