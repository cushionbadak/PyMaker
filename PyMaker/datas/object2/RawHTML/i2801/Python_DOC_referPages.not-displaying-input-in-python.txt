<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/2/library/getpass.html" rel="nofollow noreferrer">getpass</a> module in python.</p>
<pre><code>a = getpass.getpass("Write your password here:")
</code></pre>
<p>When you do the above you'll get the following:</p>
<pre><code>Write your password here: #No Characters Shown
</code></pre>
<p>You can however write a function to store the entered text and report only a string of *'s when called. Kinda like <a href="http://pythonfiddle.com/censor-replace-with-asterisks/" rel="nofollow noreferrer">this</a>, which I did not write</p>
