<pre><code>app.send(
    functions.channels.InviteToChannel(
        channel=app.resolve_peer("xyz123"), 
        users=[
            for account in accounts:
                app.resolve_peer(accounts)
        ]
    )
)
</code></pre>

<div class="post-text" itemprop="text">
<p>You want to use <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>app.send(
    functions.channels.InviteToChannel(
        channel=app.resolve_peer("xyz123"), 
        users=[
            app.resolve_peer(account) for account in accounts     
        ]
    )
)
</code></pre>
</div>
<span class="comment-copy"><code>[app.resolve_peer(accounts) for account in accounts]</code> is a <b>list comprehension</b>, not a for-loop. You're confusing the two. A list comprehension is an expression which returns a list of some sort. A for-loop is a command, not an expression.</span>
<span class="comment-copy">You have an extra colon at the end of your comprehension</span>
