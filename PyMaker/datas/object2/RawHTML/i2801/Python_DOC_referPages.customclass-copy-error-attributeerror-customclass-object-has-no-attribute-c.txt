<div class="post-text" itemprop="text">
<p>The problem is that the class <code>CashFlowSchedule</code> does not have a <code>copy()</code> method. </p>
<p>Assuming you want to create a copy of <code>CashFlowSchedule</code>, you should use Python's <code>copy</code> <a href="https://docs.python.org/3/library/copy.html" rel="nofollow noreferrer">library</a>.</p>
<p>To create a shallow copy:</p>
<pre><code>import copy
import cf_schedule as cfs
x=cfs.CashFlowSchedule(pd.date_range('20180101','20190101'))
x_copy = copy.copy(x)
</code></pre>
<p>To create a deep copy, just replace the last line with this:</p>
<pre><code>x_copy = copy.deepcopy(x)
</code></pre>
</div>
