<p>I think this solves your problem</p>
<pre><code>def aesthetic_print(dic):
    print()
    for i in dic:
        print(i)
        if type(dic[i]) == list:
            for j in dic[i]:
                print('\t', j)
            print()
        else:
            print('\t', dic[i])


D = {'a': [1.1, 1.2, 1.3], 'b': [2.1, 2.2], 'c': 3}


aesthetic_print(D)
</code></pre>
