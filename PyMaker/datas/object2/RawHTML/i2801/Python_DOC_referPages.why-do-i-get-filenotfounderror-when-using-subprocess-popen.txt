<div class="post-text" itemprop="text">
<p>if you want use the <code>Popen</code> method, then the program name and arguments need to be separate elements of the list, like this:</p>
<pre><code>for i in lines:
    p1 = subprocess.Popen(['ping', '-c 2', i], stdout=subprocess.PIPE)

    # Run the command
    output = p1.communicate()[0]
    print(output)
</code></pre>
<p>Read from the official document, may be the <code>run</code> method more used.</p>
