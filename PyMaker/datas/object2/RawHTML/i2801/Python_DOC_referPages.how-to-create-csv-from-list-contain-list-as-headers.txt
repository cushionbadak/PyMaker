<p>I want to create csv file with list of lists. Below is the example.</p>
<pre><code>l = [['test1', 'test2', 'test3', 'test4'], ['name1','name2','name3', 'name4' ]]

headers = ['profile', 'name']
</code></pre>
<p>in above example, 1st list should update all rows for profile header and second list should update all rows for name column.</p>
</div>
<div class="post-text" itemprop="text">
<p>There's probably a cleaner way to iterate over a list of lists, but this is a start</p>
<pre><code>import csv 

with open('file.csv', 'w') as f:
  w = csv.writer(f)
  w.writerow(headers)
  for p,n in zip(l[0], l[1]):
    w.writerow([p,n])
</code></pre>
<h3>Output</h3>
<pre><code>profile,name
test1,name1
test2,name2
test3,name3
test4,name4
</code></pre>
</div>
<span class="comment-copy">You need a CSV writer. It is <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer">well-documented</a>.</span>
<span class="comment-copy">its better practice to do it like <code>{header:[data], header2:[data']}</code></span>
<span class="comment-copy">with the way CSV files are structured in the background, each row is a list of things separated by CSVs (comma separated values) and then at the end of that first row, you can fill in the next row by starting a new line and then preceding to add more comma-separated values for that. to leave a column blank, simply put nothing between the preceding command and the closing comma for that column space.</span>
