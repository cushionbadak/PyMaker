<p>The answer above returns a byte, not a character, so it needs to be decoded.</p>
<p>Here is a solution that returns a single character without having to hit Enter. It works in python 3.4, on Windows</p>
<pre><code>def getChar():
    import msvcrt 
    return msvcrt.getch().decode('utf-8')
</code></pre>
