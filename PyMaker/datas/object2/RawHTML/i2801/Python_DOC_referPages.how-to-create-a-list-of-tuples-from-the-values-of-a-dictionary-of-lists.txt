<p>Use <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow noreferrer"><code>chain</code></a> or <a href="https://docs.python.org/3/library/itertools.html#itertools.chain.from_iterable" rel="nofollow noreferrer"><code>chain.from_iterable</code></a> from <code>itertools</code> module:</p>
<pre><code>from itertools import chain

d = {'Alec': [(1508, 1512),
          (2882, 2886),
          (3011, 3015),
          (3192, 3196),
          (3564, 3568),
          (6453, 6457)],
 'Downworlders': [(55, 67)],
 'Izzy': [(1499, 1503), (1823, 1827), (7455, 7459)],
 'Jace': [(1493, 1497),
          (1566, 1570),
          (3937, 3941),
          (5246, 5250)]}

print(list(chain(*d.values())))

# [(1508, 1512), (2882, 2886), (3011, 3015), (3192, 3196), (3564, 3568),
#  (6453, 6457), (55, 67), (1499, 1503), (1823, 1827), (7455, 7459),
#  (1493, 1497), (1566, 1570), (3937, 3941), (5246, 5250)]
</code></pre>
<p>Or:</p>
<pre><code>print(list(chain.from_iterable(d.values())))
</code></pre>
</div>
<span class="comment-copy">It's a simple flattening operation over <code>my_entities.values()</code>. So [t for v in my_entities.values() for t in v]` or <code>itertools.chain.from_iterable(my_entities.values())</code> if you need an iterator only.</span>
