<p>I am new to Python. I am trying to adjust the format of a list which looks like below:</p>
<pre><code>data=[1,10,313,4000,51234,123456]
</code></pre>
<p>and I would like to convert them to a list of strings with leading zeros:</p>
<pre><code>result=['000001','000010','000313','004000','051234','123456']
</code></pre>
<p>each of the element has 6 digits.</p>
<p>I know for a single number X, I can do:</p>
<pre><code>str(X).zfill(6)
</code></pre>
<p>but I am not sure how to apply this to a list. I would like to solve this problem without using a for loop.</p>
<p>Anyone could help? Thanks.</p>
</div>
<div class="post-text" itemprop="text">
<p>Apply the same <a href="https://docs.python.org/3/library/stdtypes.html#str.zfill"><code>zfill</code></a> function in a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions">list comprehension</a>, like this</p>
<pre><code>&gt;&gt;&gt; [str(item).zfill(6) for item in data]
['000001', '000010', '000313', '004000', '051234', '123456']
</code></pre>
<p>Alternatively, you can use the string's <a href="https://docs.python.org/3/library/stdtypes.html#str.format"><code>format</code></a> method, with <a href="https://docs.python.org/3/library/string.html#formatspec">format specifiers</a>, like this</p>
<pre><code>&gt;&gt;&gt; ["{:06d}".format(item) for item in data]
['000001', '000010', '000313', '004000', '051234', '123456']
</code></pre>
<p>If you are going to do the formatting more often, then you can store that in a variable, like this</p>
<pre><code>&gt;&gt;&gt; formatter = "{:06d}".format
&gt;&gt;&gt; [formatter(item) for item in data]
['000001', '000010', '000313', '004000', '051234', '123456']
</code></pre>
<p>If you are using Python 2.x, then you can use <a href="https://docs.python.org/2/library/functions.html#map"><code>map</code></a> and the <code>formatter</code> function, like this</p>
<pre><code>&gt;&gt;&gt; map(formatter, data)
['000001', '000010', '000313', '004000', '051234', '123456']
</code></pre>
<p>If you are using Python 3.x, <a href="https://docs.python.org/3/library/functions.html#map"><code>map</code></a> returns an iterable <code>map</code> object. So, you need to explicitly create a list, like this</p>
<pre><code>&gt;&gt;&gt; list(map(formatter, data))
['000001', '000010', '000313', '004000', '051234', '123456']
</code></pre>
