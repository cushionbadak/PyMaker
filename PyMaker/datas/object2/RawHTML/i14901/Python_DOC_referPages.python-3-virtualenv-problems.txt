<div class="post-text" itemprop="text">
<p>The <code>pip</code> program what you are using corresponds to the Python 2.7 version. You need to use the <code>pip</code> which corresponds to Python 3.x. So, you should be using</p>
<pre><code>pip3 install virtualenv
</code></pre>
<p>Alternatively, you can create virtual environments in Python 3.3+ with the <a href="https://docs.python.org/3/library/venv.html#module-venv" rel="nofollow"><code>venv</code> module</a>, like this</p>
<pre><code>python3 -m venv test
</code></pre>
