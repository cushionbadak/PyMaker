<div class="post-text" itemprop="text">
<p>You need to assign the result of <code>isnumeric()</code> to a variable and check it's value:</p>
<pre><code>value = sys.argv[1].isnumeric()
if value:
    print('argument is number')
else:
    print("Usage: python3 WnnnnnnnnAssg3.py number file1 file2 . . . filen")
    quit()
</code></pre>
<p>Besides, using <a href="https://docs.python.org/3/library/argparse.html" rel="nofollow"><code>argparse</code></a> built-in package would make the command-line parameter parsing and validation a lot cleaner and readable (<a href="https://docs.python.org/3/howto/argparse.html" rel="nofollow">tutorial</a> link).</p>
