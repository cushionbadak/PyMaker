<div class="post-text" itemprop="text">
<p>There is a list of tuple like </p>
<pre><code>l = [(1, 2, 'a', 'b'), (3, 4, 'c', 'd'), (5, 6, 'e', 'f')]
</code></pre>
<p>I can use </p>
<pre><code>[(i[0], i[2], i[3]) for i in l]
</code></pre>
<p>to get the result </p>
<pre><code>[(1, 'a', 'b'), (3, 'c', 'd'), (5, 'e', 'f')]
</code></pre>
<p>But if given a variable list such as <code>[0, 2, 3]</code>,  how to get the similar result?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/operator.html#operator.itemgetter" rel="nofollow"><code>operator.itemgetter</code></a>, like this</p>
<pre><code>&gt;&gt;&gt; from operator import itemgetter
&gt;&gt;&gt; getter = itemgetter(0, 2, 3)
&gt;&gt;&gt; [getter(item) for item in l]
[(1, 'a', 'b'), (3, 'c', 'd'), (5, 'e', 'f')]
</code></pre>
<p>If you have a list of indices, then you can <a href="https://docs.python.org/3/tutorial/controlflow.html#tut-unpacking-arguments" rel="nofollow">unpack them</a> to the <code>itemgetter</code>, like this</p>
<pre><code>&gt;&gt;&gt; getter = itemgetter(*[0, 2, 3])
&gt;&gt;&gt; [getter(item) for item in l]
[(1, 'a', 'b'), (3, 'c', 'd'), (5, 'e', 'f')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use a generator expression and <code>tuple()</code> to pull out specific indices:</p>
<pre><code>[tuple(t[i] for i in indices) for t in l]
</code></pre>
<p>or you can use a <a href="https://docs.python.org/2/library/operator.html#operator.itemgetter"><code>operator.itemgetter()</code> object</a> to create a callable that does the same:</p>
<pre><code>from operator import itemgetter

getindices = itemgetter(*indices)
[getindices(t) for t in l]
</code></pre>
<p>where <code>indices</code> is your list of indexes. This works because <code>operator.itemgetter()</code> just happens to return a <code>tuple</code> object when retrieving multiple indexes.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; l = [(1, 2, 'a', 'b'), (3, 4, 'c','d'), (5, 6, 'e','f')]
&gt;&gt;&gt; indices = [0, 1, 2]
&gt;&gt;&gt; [tuple(t[i] for i in indices) for t in l]
[(1, 2, 'a'), (3, 4, 'c'), (5, 6, 'e')]
&gt;&gt;&gt; getindices = itemgetter(*indices)
&gt;&gt;&gt; from operator import itemgetter
&gt;&gt;&gt; [getindices(t) for t in l]
[(1, 2, 'a'), (3, 4, 'c'), (5, 6, 'e')]
</code></pre>
</div>
