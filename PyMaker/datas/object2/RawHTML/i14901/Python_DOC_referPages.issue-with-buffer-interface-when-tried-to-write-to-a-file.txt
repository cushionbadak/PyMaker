<p>TypeError: 'str' does not support the buffer interface</p>
<pre><code>import os,stat

fd = os.open("foo.txt",os.O_RDWR|os.O_CREAT)
os.write(fd,"This is Test")
os.closerange(fd,fd)
print("Closed all the files successfully")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to write <em>bytes</em>, not Unicode strings:</p>
<pre><code>os.write(fd, b"This is Test")
</code></pre>
<p>The <code>b'...'</code> is a <code>bytes</code> type literal. You could also encode a <code>str</code> type by using the <code>str.encode()</code> method:</p>
<pre><code>os.write(fd, "This is Test".encode('ascii'))
</code></pre>
<p>Generally speaking, you want to use the <a href="https://docs.python.org/3/library/io.html" rel="nofollow"><code>io</code> library</a> rather than use <code>os.open()</code> and low-level file access. This is easily done by using the <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow"><code>open()</code> function</a>.</p>
