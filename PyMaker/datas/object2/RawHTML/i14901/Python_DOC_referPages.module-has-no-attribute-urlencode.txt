<div class="post-text" itemprop="text">
<p><code>urllib</code> has been split up in Python 3.  The <code>urllib.urlencode()</code> function is now <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.urlencode"><code>urllib.parse.urlencode()</code></a>, and the <code>urllib.urlopen()</code> function is now <a href="https://docs.python.org/3/library/urllib.request.html#urllib.request.urlopen"><code>urllib.request.urlopen()</code></a>.</p>
