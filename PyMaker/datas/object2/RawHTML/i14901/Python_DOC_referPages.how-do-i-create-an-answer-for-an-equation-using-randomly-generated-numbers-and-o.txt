<pre><code>ops = ['+','-','*']

      answer = sum(var1*var2)

<div class="post-text" itemprop="text">
<p><code>sum</code> is <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow">for summing up an iterable</a> - meaning a list or something behaving as a list.</p>
<p>Just use <code>answer = var1 + var2</code>.</p>
<p>There is also a fun module named <code>operators</code> that <a href="https://docs.python.org/3/library/operator.html" rel="nofollow">allows you to supply an operator</a> that mirrors the built-in operators, which would allow you to generalize this even further.</p>
<p>Something like this would probably be more pythonic :</p>
<pre><code>import operators
ops = [operators.iadd, operators.imul, operators.isub]
# pick your op = ..
# pick your vars = ..

answer = op(var1, var2)
</code></pre>
</div>
