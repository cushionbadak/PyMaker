<p>Trying to do something like:</p>
<pre><code>facility = 'LOG_LOCAL7'
syslog.openlog( 'Blah', 0, syslog.facility)
</code></pre>
<p>But I get:</p>
<pre><code> AttributeError: 'module' object has no attribute 'facility'
</code></pre>
<p>I want to be able to set the facility in a variable...</p>
<p>On a side note; in Python, what are the different parts of this statement actually called?</p>
<pre><code>syslog.openlog( 'Blah', 0, syslog.LOG_LOCAL7)
</code></pre>
<ul>
<li>syslog</li>
<li>openlog</li>
<li>'Blah'</li>
</ul>
<p>Is it class, method, attribute?</p>
</div>
<div class="post-text" itemprop="text">
<p>To answer the first part, you need <code>getattr</code>:</p>
<blockquote>
<p>Help on built-in function <code>getattr</code> in module <code>__builtin__</code>:</p>
<p><code>getattr(...)</code>
<code>getattr(object, name[, default]) -&gt; value</code></p>
<pre><code>Get a named attribute from an object; getattr(x, 'y') is equivalent to x.y.
When a default argument is given, it is returned when the attribute doesn't
exist; without it, an exception is raised in that case.
</code></pre>
</blockquote>
<p>So you, need:</p>
<pre><code>syslog.openlog('Blah', 0, getattr(syslog, facility))
</code></pre>
<p>For the second part of your question, as to the breakdown of the statement itself:</p>
<ul>
<li><code>syslog</code> (module)</li>
<li><code>.</code> (<a href="http://en.wikipedia.org/wiki/Scope_resolution_operator" rel="nofollow">scope resolution operator</a>)</li>
<li><code>openlog</code> (an attribute of the module, in this case, its a function)</li>
<li><code>'Blah'</code> (an argument to the function)</li>
</ul>
</div>
<div class="post-text" itemprop="text">
<p>You want to use getattr (<a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow">https://docs.python.org/3/library/functions.html#getattr</a>)</p>
<pre><code>facility = 'LOG_LOCAL7'
syslog.openlog( 'Blah', 0, getattr(syslog, facility))
</code></pre>
</div>
