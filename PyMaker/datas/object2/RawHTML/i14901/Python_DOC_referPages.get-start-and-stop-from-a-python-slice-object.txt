<div class="post-text" itemprop="text">
<p>The members of a <a href="https://docs.python.org/3/library/functions.html#slice" rel="nofollow"><code>slice</code></a> are <code>start</code>, <code>step</code> and <code>stop</code>:</p>
<blockquote>
<p>Slice objects have read-only data attributes <code>start</code>, <code>stop</code> and <code>step</code> which merely return the argument values (or their default).</p>
</blockquote>
<hr/>
<p>Thus as simple as: </p>
<pre><code>&gt;&gt;&gt; sl = slice(0L, 832L)
&gt;&gt;&gt; sl
slice(0L, 832L, None)
&gt;&gt;&gt; start = sl.start
&gt;&gt;&gt; stop = sl.stop
&gt;&gt;&gt; start, stop
(0L, 832L)
</code></pre>
</div>
