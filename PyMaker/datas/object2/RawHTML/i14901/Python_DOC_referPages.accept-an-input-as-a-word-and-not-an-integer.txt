<p>How to make word inputs in Python
I want to be able to have the computer to ask a question to the user like</p>
<pre><code>test = int(input('This only takes a number as an answer'))
</code></pre>
<p>I want to be able to have 'test' not be a number, rather a word, or letter.</p>
</div>
<div class="post-text" itemprop="text">
<p>Just remove the <code>int</code> call!  <strong>That</strong> is what makes the statement accept integer numbers only.</p>
<p>I.e, use:</p>
<pre><code>test = input('This takes any string as an answer')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Remove the type cast to <code>int</code></p>
<pre><code>test = input('This only takes a word as an answer :')
</code></pre>
<p>A demo</p>
<pre><code>&gt;&gt;&gt; test = input('This only takes a word as an answer :')
This only takes a word as an answer :word
&gt;&gt;&gt; test
'word'
</code></pre>
<p>Note - From the <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow">docs</a></p>
<blockquote>
<p>The function then reads a line from input, converts it to a string (stripping a trailing newline), and returns that</p>
</blockquote>
<p>Therefore <code>input</code> automatically converts it to a <code>str</code> and there is no need of any explicit cast.</p>
</div>
<div class="post-text" itemprop="text">
<p>This should work:</p>
<pre><code>test = str(input('This only takes a string as an answer:  '))
</code></pre>
<p>BUT</p>
<p>Because Python works with String by default, actually you don't need any casting like <code>int</code> or <code>str</code></p>
<p>Also, if you were using version prior to 3.x, it would be <code>raw_input</code> instead of <code>input</code>. Since your solution seem to have been accepting <code>input</code>, I can be safe assuming that your Python is OK.
    test = input('This only takes a string as an answer')</p>
</div>
<div class="post-text" itemprop="text">
<pre><code> test = input("This only takes a number as an answer")
 test = raw_input("This only takes a number as an answer")
</code></pre>
<p>Either one should work</p>
</div>
<div class="post-text" itemprop="text">
<p>If you are using python 2.7, just use raw_input function.</p>
<pre><code>test = raw_input('This only takes a string as an answer:  ')
</code></pre>
<p>I.e.:</p>
<pre><code>&gt;&gt;&gt; test = raw_input('This only takes a string as an answer:  ')
This only takes a string as an answer:  2.5
&gt;&gt;&gt; type (test)
&lt;type 'str'&gt;
</code></pre>
<p>If you use input, you can have only a number:
Right input:</p>
<pre><code>&gt;&gt;&gt; test = input('This only takes a number as an answer:  ')
This only takes a string as an answer:  2.5
&gt;&gt;&gt; type (test)
&lt;type 'float'&gt;
</code></pre>
<p>Wrong input:</p>
<pre><code>&gt;&gt;&gt; test = input('This only takes a number as an answer:  ')

</code></pre>
