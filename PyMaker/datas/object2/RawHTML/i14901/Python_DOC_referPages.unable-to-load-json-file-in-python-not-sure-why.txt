<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/webbrowser.html#webbrowser.open" rel="nofollow">webbrowser.open</a> does not load a file from URL; instead it opens a web browser with that page, and returns <code>True</code> (a <code>bool</code> value), which passed to <code>json.load</code> causes an exception being thrown.</p>
<p>If you have a file that you want to open, just remove the <code>from webbrowser import open</code> line.</p>
