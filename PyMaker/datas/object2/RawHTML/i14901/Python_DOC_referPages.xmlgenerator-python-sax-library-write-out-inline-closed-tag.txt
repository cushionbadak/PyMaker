<p>I write out xml by using the python XMLGenerator class from <code>xml.sax.saxutils</code>.</p>
<pre><code>    xmlwriter = XMLGenerator(output, encoding)
    xmlwriter.startDocument()
    xmlwriter.startElement("foo", {})
    ...
    xmlwriter.endElement("foo", {})
</code></pre>
<p>Writing out tags is always done by the <code>startElement</code> and <code>endElement</code> functions. For inline tags (something like XHTML's <code>&lt;br /&gt;</code> or <code>&lt;img /&gt;</code>) I would want to write out the tag with end directly. Is there a way to do that with SAX?</p>
</div>
<div class="post-text" itemprop="text">
<p>If you're using Python 3.2 or later, there's a <code>short_empty_elements</code> parameter you can pass to <code>XMLGenerator</code> (<a href="https://docs.python.org/3/library/xml.sax.utils.html#xml.sax.saxutils.XMLGenerator" rel="nofollow">see the Python xml.sax.utils docs</a>).</p>
<p>For example:</p>
<pre><code>from xml.sax.saxutils import *
import sys
output = sys.stdout
encoding = 'utf-8'
xmlwriter = XMLGenerator(output, encoding, short_empty_elements=True)
xmlwriter.startDocument()
xmlwriter.startElement(name="foo", attrs={})
xmlwriter.endElement(name="foo")
</code></pre>
<p>will produce the output:</p>
<pre><code>&lt;?xml version="1.0" encoding="utf-8"?&gt;
&lt;foo/&gt;
</code></pre>
