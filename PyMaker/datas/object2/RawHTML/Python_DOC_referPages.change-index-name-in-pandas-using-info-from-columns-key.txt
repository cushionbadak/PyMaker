<p>Good afternoon all,</p>
<p>I am new to both Python and Pandas and wanted to know how to rename the index name with that of concatenated columns info.</p>
<p>For example rename the below:</p>
<pre><code>      Bus1 Bus2 Bus3 ...

0     12   45   50   ...
0     12   45   51   ...
0     13   22   44   ...
</code></pre>
<p>To:</p>
<pre><code>          Bus1 Bus2 Bus3 ...
12_45_50  12   45   50   ...
12_45_51  12   45   51   ...
13_22_44  13   22   44   ...
</code></pre>
<p>I basically would like the Index to be a key using column info.  Thank you very much in advance for the help</p>
</div>
<div class="post-text" itemprop="text">
<p>Use:</p>
<pre><code>df.set_index(df.astype(str).apply('_'.join, axis=1), inplace=True)
</code></pre>
<p>[out]</p>
<pre><code>          Bus1  Bus2  Bus3
12_45_50    12    45    50
12_45_51    12    45    51
13_22_44    13    22    44
</code></pre>
