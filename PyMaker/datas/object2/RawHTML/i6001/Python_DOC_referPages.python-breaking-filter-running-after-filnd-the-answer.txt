<p>Here's an example of stopping an iteration early using <code>filter</code> by setting a global flag that causes the filter function to raise <code>StopIteration</code>:</p>
<pre><code>def my_filter(n):
    """ filter for odd numbers """

    if my_flag:
        raise StopIteration

    return n % 2 == 1

my_flag = False

for my_number in filter(my_filter, range(1, 22)):

    print(my_number)

    if my_number % 13 == 0:
        my_flag = True
</code></pre>
<p><strong>OUTPUT</strong></p>
<pre><code>&gt; python3 test.py
1
3
5
7
9
11
13
&gt; 
</code></pre>
<p>Is this what you have in mind?  But, of course, most people would simply do:</p>
<pre><code>if my_number % 13 == 0:
        break
</code></pre>
<p>So you must have some specific scenario in mind.  (Hopefully something more than, "Because I want it to be a one line comprehension.")</p>
