<div class="post-text" itemprop="text">
<p>Assuming your file is called <code>file.txt</code>, you can add all those variables to the <code>global</code> <code>scope</code> with:</p>
<pre><code>with open("/Users/discoveryone/Documents/file.txt") as f:
    for l in f:
        var, val = l.split("=")
        globals()[var] = val.strip().replace("'", "")
</code></pre>
<p>and then you could access <code>email</code> for instance and it would have the value:</p>
<pre><code>'noone1@nowhere.com'
</code></pre>
<hr/>
<p>The only problem with this is that <code>domains</code> is only a <code>string</code> of the <code>domains</code>:</p>
<pre><code>'apple.com, google.com, yahoo.com, aol.com, disney.com, mydomain.com, hotmail.com'
</code></pre>
<p>to convert this to a list of <code>domains</code>, simply add the <code>line</code>:</p>
<pre><code>domains = [d.strip() for d in domains.split(",")]
</code></pre>
<p>which will modify <code>domains</code> to:</p>
<pre><code>['apple.com', 'google.com', 'yahoo.com', 'aol.com', 'disney.com', 'mydomain.com', 'hotmail.com']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you can, use configparser (<a href="http://docs.python.org/3/library/configparser.html" rel="nofollow noreferrer">http://docs.python.org/3/library/configparser.html</a>). But that will force you to change the structure of your config file.</p>
<p>If you only need the domains values:</p>
<pre><code>with open('/Users/discoveryone/Documents/file') as my_file:
    d_line = my_file[0]
    d_values = [v.replace("'", "") for v in d_line.split('=')[1].split(',')]
</code></pre>
<p>You could also create a dictionary:</p>
<pre><code>config = {}
with open('/Users/discoveryone/Documents/file') as my_file:
    for line in my_file:
        k, v = line.split('=')
        v = v.replace('"', "")  # Remove uggly single quotes.
        if k == 'domains':   # If its the domain values, create a list out of them
            v = v.split(',')
        config[k] = v
</code></pre>
