<p>You cannot run your present code with Python 3 interpreter, so you have two options: translate this code to Python 3 or run it with Python 2.</p>
<p><strong>Porting 2 to 3</strong></p>
<p>There is some materials about porting from python 2 to 3:</p>
<p><a href="https://docs.python.org/3/howto/pyporting.html" rel="nofollow noreferrer">Official docs</a></p>
<p><a href="https://www.digitalocean.com/community/tutorials/how-to-port-python-2-code-to-python-3" rel="nofollow noreferrer">DigitalOcean tutorial</a></p>
<p>Also there is a tool for automated translation: <a href="https://docs.python.org/2/library/2to3.html" rel="nofollow noreferrer">2to3</a>.
I haven't use it by myself so I cannot recommend it, but you can have a look and decide if it suites your needs.</p>
<p><strong>Running with python 2</strong></p>
<p>I think the best way here is to create a <a href="https://virtualenv.pypa.io/en/stable/" rel="nofollow noreferrer">virtual environment</a> with python 2. Something like this</p>
<pre><code>$ virtualenv -p python2 venv
$ source venv/bin/activate
$ python &lt;your-script&gt;.py
</code></pre>
<p>For the further reading, there are a techniques to write python 2/3 compatible code. <a href="http://python-future.org/compatible_idioms.html" rel="nofollow noreferrer">See this</a>.</p>
