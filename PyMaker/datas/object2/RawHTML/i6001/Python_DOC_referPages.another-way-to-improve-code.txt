<p><a href="https://docs.python.org/3/library/stdtypes.html#str.maketrans" rel="nofollow noreferrer"><code>str.maketrans</code></a> and <a href="https://docs.python.org/3/library/stdtypes.html#str.translate" rel="nofollow noreferrer"><code>str.translate</code></a> are designed around problems like these: mapping one set of character onto another.</p>
<pre><code>from string import ascii_lowercase, ascii_uppercase
from itertools import chain

alphabets = [ascii_lowercase, ascii_uppercase]

def shift(msg, rotate, alphabets):
    paired_letters = (zip(alphabet, alphabet[rotate:]+alphabet[:rotate]) for alphabet in alphabets)
    trans = str.maketrans(dict(chain.from_iterable(paired_letters)))
    return msg.translate(trans)

print(shift('helL@', 2, alphabets))
