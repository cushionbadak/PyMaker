<p>You can use  <a href="https://docs.python.org/3/library/heapq.html#heapq.nsmallest" rel="nofollow noreferrer"><strong><code>heapq.nsmallest</code></strong></a>:</p>
<pre><code>import heapq

manhattan = {'path1_a': {'path2_a': 1,
                         'path2_b': 5},
             'path1_b': {'path2_c': 3,
                         'path2_d': 7},
             'path1_c': {'path2_e': 4,
                         'path2_f': 9}}

print heapq.nsmallest(3,
                      ((path1, path2, value)
                       for path1, path2s in manhattan.items()
                       for path2, value in path2s.items()),
                      key=lambda (path1, path2, value): value)
</code></pre>
<p>Output:</p>
<pre><code>[('path1_a', 'path2_a', 1),
 ('path1_b', 'path2_c', 3),
 ('path1_c', 'path2_e', 4)]
</code></pre>
