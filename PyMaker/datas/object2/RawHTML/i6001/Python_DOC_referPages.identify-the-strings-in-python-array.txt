<p>You can try this   </p>
<pre><code>import numpy as np

def is_number(s):
   try:
    int(s)
    return True
   except ValueError:
    return False

A = np.array( [['1','2','3'], ['a','3','5']] )

for i in range(0,2):
    for j in range(0,3):
        if not is_number(A[i][j]):
            print i , j
</code></pre>
