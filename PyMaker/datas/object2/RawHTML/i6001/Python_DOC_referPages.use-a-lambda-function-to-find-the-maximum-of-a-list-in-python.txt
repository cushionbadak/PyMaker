<div class="post-text" itemprop="text">
<p>You can wrap your lambda using <strong><a href="https://docs.python.org/3/library/functools.html#functools.reduce" rel="nofollow noreferrer">reduce</a></strong>:</p>
<pre><code>&gt;&gt;&gt; from functools import reduce
&gt;&gt;&gt; 
&gt;&gt;&gt; lst = [22, 36, 47, 2, 13]
&gt;&gt;&gt; reduce(lambda x, y: x if (x &gt; y) else y, lst)
47
</code></pre>
