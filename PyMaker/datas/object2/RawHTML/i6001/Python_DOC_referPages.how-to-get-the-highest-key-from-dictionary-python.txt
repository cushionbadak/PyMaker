<p>I'm new to Python and I am having a small problem I want to solve</p>
<pre><code>a={"1" : 3, "2" : 3, "6":3}
</code></pre>
<p>I have this dic and I would like to get the maximum key out of him
for example in the dictionary above I would like to get the key 6 as a value</p>
</div>
<div class="post-text" itemprop="text">
<p>There are many ways, one way of doing it is:</p>
<pre><code>greatest = max(int(k) for k in a)
</code></pre>
<p>Or a human readable format, so you can understand the logic:</p>
<pre><code>greatest = None
for k in a:
    k = int(k)
    if greatest == None:
        greatest = k
    if k &gt; greatest:
        greatest = k
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>For a simple, functional approach:</p>
<pre><code>max(map(int,a))
</code></pre>
<p>This applies the <code>int</code> function to each key of <code>a</code> using <code>map</code> to convert the keys as strings to integers, then finds the maximum of these using <code>max</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you want the original key (as as string, not an integer), you could simply use <a href="https://docs.python.org/3/library/functions.html#max" rel="nofollow noreferrer"><code>max</code></a> with <code>int</code> as key argument:</p>
<pre><code>a = {"1" : 3, "2" : 3, "6":3}
max(a, key=int)
# '6'
</code></pre>
</div>
