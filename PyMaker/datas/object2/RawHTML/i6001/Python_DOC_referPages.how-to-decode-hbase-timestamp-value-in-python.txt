<div class="post-text" itemprop="text">
<p>The timestamp value is the number of milliseconds since the epoch (January 1, 1970 UTC).  You can use the python <code>datetime</code> module to manipulate it.  Example:</p>
<pre><code>from datetime import datetime as dt
print (dt.fromtimestamp(1511356398000 / 1000))

Output: 
2017-11-22 07:13:18
</code></pre>
<p>The result is a <code>datetime</code> object in my local time zone. (Central USA)  The <code>datetime.fromtimestamp</code> method wants a floating point value that is the time in seconds since the epoch, so divide the time in milliseconds by 1000.</p>
