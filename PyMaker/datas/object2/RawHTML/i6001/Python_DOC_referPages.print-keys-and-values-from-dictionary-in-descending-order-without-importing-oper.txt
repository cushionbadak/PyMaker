<div class="post-text" itemprop="text">
<p>Something like this?</p>
<pre><code>sortedKeysAndValues = sorted(myDict.items(), key=lambda kv: -kv[1])
</code></pre>
<p>And then to print,</p>
<pre><code>for k, v in sortedKeysAndValues:
    print(k, v)
</code></pre>

<p><strong>Python 2</strong></p>
<pre><code>sorted_dict = sorted([(value,key) for (key,value) in myDict.iteritems()])
</code></pre>
<p><strong>Python 3</strong></p>
<pre><code>sorted_dict = sorted([(value,key) for (key,value) in myDict.items()])
</code></pre>
<p>Printing for both python <strong>2</strong> and <strong>3</strong></p>
<pre><code>print([(key, value) for value, key in sorted_dict]
</code></pre>

<span class="comment-copy">Yeah sure, you can look at the documentation of <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer">sorted()</a> for more information but essentially, <code>myDict.items()</code> will split the dictionary up into key-value pairs. <code>key=lambda kv: -kv[1]</code> specifies the comparison key to sort the list, since you needed it in descending order, we use <code>-kv[1]</code> as <code>sorted()</code> sorts in ascending order by default and <code>-kv[1]</code> is just taking the negative of the occurrences. And yes, lambda is just an anonymous function.</span>
