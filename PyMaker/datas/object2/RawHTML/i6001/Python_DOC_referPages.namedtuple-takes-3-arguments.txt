<div class="post-text" itemprop="text">
<p>The first argument of named tuple constructor, is the <code>typename</code>: the name of the named tuple, not the arguments. <a href="https://docs.python.org/2/library/collections.html#collections.namedtuple" rel="nofollow noreferrer">[documentation]</a></p>
<p>So you should construct <code>t</code> as:</p>
<pre><code>#                   v parameters
t = namedtuple(<b>'t', ('a', 'b')</b>)
#              ^ type name</code></pre>
<p>To make it more convenient, you can also provide a <em>space separated</em> string of parameters. So an equivalent one is:</p>
<pre><code>#t = namedtuple('t', <b>'a b'</b>)
#                    ^ space separated list of parameters</code></pre>
