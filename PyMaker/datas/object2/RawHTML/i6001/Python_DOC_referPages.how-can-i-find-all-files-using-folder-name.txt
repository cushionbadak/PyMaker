<div class="post-text" itemprop="text">
<p>Here's a quick example of using a f-string</p>
<pre><code>outdir = 'max/test/'
path = f'{outdir}/*{{}}*.sh'
</code></pre>
<p><code>path</code> is then equal to <code>'max/test/*{}*.sh'</code>.  Note that we have to escape the curly braces by doubling up on them.  <code>{{</code> escapes to <code>{</code> and <code>}}</code> to <code>}</code></p>
