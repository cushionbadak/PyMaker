<p><code>range</code> solves half of the problem itself. Rest of the thing is just formatting and inputting the number.</p>
<p><code>range(n,1,-1)</code> means the range starts from <code>n</code> and ends in <code>1</code> and there is a step of <code>-1</code> in between them.</p>
<p><code>n, n+(-1), n+(-2),...,1</code>.</p>
<hr/>
<p>Your C++ program generates wrong output:-</p>
<p><code>/n 1+4=4/n 2+3=4/n 3+2=4</code></p>

<p>You need a for loop that decrements.</p>
<pre><code># range(4,1,-1) would give a list of [4,3,2]
# for loop iterates through each number.
k = 0
n = 4
# This iterates from n = 4 to 1
# decrementing one step at a time, needn't specifically handle i in your loop body. 
# So i would get values of 4,3,2
for i in range(n,1,-1):
    print(k,i)
    # no ++ operator in python.
    k+=1
</code></pre>
