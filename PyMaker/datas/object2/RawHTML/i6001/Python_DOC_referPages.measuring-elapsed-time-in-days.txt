<p>I would like to measure elapsed time in days with the Python time module. My attempt is:</p>
<pre><code>import time, datetime

def elapsed_time(start)
    return datetime.datetime.fromtimestamp(time.time()-start).strftime('%Y-%m-%d-%H-%M-%S')

start = time.time()
print(elapsed_time(start))
# 1970-01-01-00-00-06
</code></pre>
<p>I would like the string to start from year 0, month 0, and day 0. How do I do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>Just use the difference between <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.now" rel="nofollow noreferrer"><code>datetime.now()</code></a> and your start timestamp. The resulting <a href="https://docs.python.org/3/library/datetime.html#datetime.timedelta" rel="nofollow noreferrer"><code>datetime.timedelta</code></a> object has a <code>days</code> attribute:</p>
<pre><code>from datetime import datetime

def elapsed_days(start):
    return (datetime.now() - start).days

start = datetime(2017, 11, 1)
&gt;&gt;&gt; elapsed_days(start)
19

start = datetime(2017, 1, 1)
&gt;&gt;&gt; elapsed_days(start)
323

&gt;&gt;&gt; elapsed_days(datetime(1, 1, 1))
736652
</code></pre>
