<p>Try this:</p>
<pre><code>import ast
lst = ['[[2.1, 3.7], [5.5, -1.3]]']
ast.literal_eval(lst[0])
=&gt; [[2.1, 3.7], [5.5, -1.3]]
</code></pre>

<div class="post-text" itemprop="text">
<p>It looks like you have a list with a single element, which is a string, and you want to evaluate the string and return it, discarding the outer list.  If so, then you can use <code>literal_eval</code> from <code>ast</code> to safely evaluate the expression:</p>
<pre><code>import ast
s = ['[[2.1, 3.7], [5.5, -1.3]]']
t = s[0]
ast.literal_eval(t)
</code></pre>
<p>This evaluates to:</p>
<pre><code>[[2.1, 3.7], [5.5, -1.3]]
</code></pre>
</div>
