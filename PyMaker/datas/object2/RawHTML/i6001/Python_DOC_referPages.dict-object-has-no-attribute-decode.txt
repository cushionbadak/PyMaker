<div class="post-text" itemprop="text">
<p>On iteration, a <code>csv.DictReader</code> yields <code>dict</code>s - hence its name.<br/>
Each dict represents a CSV record in the style <code>{&lt;column_header&gt;: &lt;column_value&gt;}</code>.
To decode the file's contens as UTF-32, you can specify the encoding as argument of <code>open</code>.</p>
<pre><code>header_rdr = DictReader(open(â€¦, 'r', encoding='utf-32'))
</code></pre>
</div>
<span class="comment-copy"><code>DictReader</code> yields an OrderedDict for each row of the CSV. But you should be opening the file with the correct encoding mode. You aren't using Python 2, are you?</span>
