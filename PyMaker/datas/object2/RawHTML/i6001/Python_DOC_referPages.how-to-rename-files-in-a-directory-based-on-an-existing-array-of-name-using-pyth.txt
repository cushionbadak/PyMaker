<div class="post-text" itemprop="text">
<p>If you need a 1-1 correspondence between 2 or more lists, <code>zip</code> would be the way to do it.</p>
<pre><code>a = [1, 2, 3]
b = ['a', 'b', 'c']

for i, j in zip(a, b):
 print(i, j)

1 a
2 b
3 c
</code></pre>
<p>For your example, you'd need to <code>zip</code> <code>files</code> and <code>names</code> together, so that<code>files[i]</code> is renamed to <code>names[i]</code> for any <code>i</code>.</p>
<pre><code>for i, j in zip(files, names):
    os.rename(os.path.join(path, i), os.path.join(path, '{}.mp4'.format(j)))
</code></pre>
<hr/>
<p>You could also use <code>enumerate</code> to track the indices of the filenames from <code>os.listdir</code> and provide these indices to the <code>names</code> list while you are iterating:</p>
<pre><code>for i, name in enumerate(files):
    os.rename(os.path.join(path, name), os.path.join(path, '{}.mp4'.format(names[i])))
</code></pre>
