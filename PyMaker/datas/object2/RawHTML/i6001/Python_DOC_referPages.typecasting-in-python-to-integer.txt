<div class="post-text" itemprop="text">
<p>You are trying to cast a list of 2 numbers to int. <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow noreferrer">int only takes a number or a string as its argument</a>.</p>
<p>What you want is to <a href="http://book.pythontips.com/en/latest/map_filter.html#map" rel="nofollow noreferrer">map</a> the int function to each item in the list.</p>
<pre><code>&gt;&gt;&gt; w, h = map(int, input().split(" "))
5 10
&gt;&gt;&gt; w
5
&gt;&gt;&gt; h
10
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>int(...)</code> constructs an integer which cannot be unpacked to a tuple <code>W, H</code>. What you probably want is</p>
<pre><code>W, H = (int(x) for x in input().split(" "))
</code></pre>
</div>
