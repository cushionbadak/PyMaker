<p>I am able to save the cmd data onto a text file using the following command:</p>
<pre><code>python code_3.py &gt; output.txt
</code></pre>
<p>However I am calling code_3.py from primary_script.py by writing:</p>
<pre><code>import code_3
os.system('loop3.py')
</code></pre>
<p>But I want it to perform the functionality of the what the previous line does. This doesn't work:</p>
<pre><code>os.system('loop3.py &gt; opt.txt ')
</code></pre>
<p>Can someone please tell me what to do?</p>
</div>
<div class="post-text" itemprop="text">
<p>Here's how to do it with the <a href="https://docs.python.org/3/library/subprocess.html#module-subprocess" rel="nofollow noreferrer"><code>subprocess</code></a> module:</p>
<pre><code>import subprocess
import sys

p1 = subprocess.Popen([sys.executable, "loop3.py"], stdout=subprocess.PIPE)
output, err = p1.communicate()

with open('opt.txt', 'w') as file:
    file.write(output.decode())
</code></pre>
</div>
