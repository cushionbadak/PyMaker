<p>I am trying to write a function that returns a hangman style output given an input of guessed characters and a secret to be guessed. I start by making a new list <code>a</code>set to <code>-</code> as many times as the length of the secret. I then iterate through this variable and test if <code>a[stepper] == letter</code> where <code>letter</code> is the walker as we loop through. </p>
<p>Currently to keep track of this stepper I am using this code</p>
<pre><code>a = ["-"] * len(secret) 
for curr in guessed:
    letter_count = 0  # Stepper
    for letter in secret:
        if curr == letter:
            a[letter_count] = curr
        letter_count += 1
</code></pre>
<p>Is there a built in functionality for <code>letter_count</code> as I am using here?</p>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate()</code></a> builtin function like this:</p>
<pre><code>a = ['-'] * len(secret)
for curr in guessed:
    for i, letter in enumerate(secret):
        if curr == letter:
            a[i] = curr
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could create the displayed word dynamically with <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>join</code></a> and a comprehension. The guesses are stored in a <a href="https://docs.python.org/2/library/sets.html" rel="nofollow noreferrer"><code>set</code></a> and a ternary operator checks if the letter or <code>-</code> should be displayed:</p>
<pre><code>&gt;&gt;&gt; word = "hangman"
&gt;&gt;&gt; guessed = set()
&gt;&gt;&gt; ''.join(letter if letter in guessed else '-' for letter in word)
'-------'
&gt;&gt;&gt; guessed.add('a')
&gt;&gt;&gt; ''.join(letter if letter in guessed else '-' for letter in word)
'-a---a-'
&gt;&gt;&gt; guessed.add('m')
&gt;&gt;&gt; ''.join(letter if letter in guessed else '-' for letter in word)
'-a--ma-'
&gt;&gt;&gt; guessed.add('e')
&gt;&gt;&gt; ''.join(letter if letter in guessed else '-' for letter in word)
'-a--ma-'
&gt;&gt;&gt; guessed.add('n')
&gt;&gt;&gt; ''.join(letter if letter in guessed else '-' for letter in word)
'-an-man'
</code></pre>
