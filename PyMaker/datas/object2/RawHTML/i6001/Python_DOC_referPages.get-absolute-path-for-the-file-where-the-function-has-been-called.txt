<p>I wanted to make a util function that returns the absolute path for the file, where the function has been called, however I did not found a way to do so.</p>
<p>For a better picture, this is what I'm trying to achieve.</p>
<p>Suppose I have the following directory structure:</p>
<pre><code>core/
    utils/
        __init__.py
        files.py
    __init__.py
    service.py
</code></pre>
<p>where <code>files.py</code> contains</p>
<pre><code>import os

def get_current_dir():
    return os.path.abspath(os.path.dirname(__file__))
</code></pre>
<p>and <code>service.py</code> contains</p>
<pre><code>from utils.files import get_current_dir


print(get_current_dir())
</code></pre>
<p>The expected results are not the desired ones. <br/></p>
<p>What I wanted to achieve: <code>/usr/src/app/core</code> <br/>
The actual output: <code>/usr/src/app/core/utils</code></p>
<p>I've been looking for on many answers here, but none of them has an example where a function is used to determine the path.</p>
<p>Are there any ways or is it at least possible to achieve what I'm looking for?</p>
<p>Thanks!</p>

<div class="post-text" itemprop="text">
<p>You could simply pass <code>__file__</code> as a parameter
in files.py</p>
<pre><code>import os

def get_current_dir(file):
    return os.path.abspath(os.path.dirname(file))
</code></pre>
<p>in service.py</p>
<pre><code>from utils.files import get_current_dir


print(get_current_dir(__file__))
</code></pre>
</div>
