<p>The problem is </p>
<pre><code>data = mycursor.fetchall()
</code></pre>
<p>returns a list of <em>tuples</em>, even if the query is returning a single column.</p>
<p>So rather than returning something like </p>
<pre><code>['a', 'b', 'c', 'd', 'e', 'f']
</code></pre>
<p>it returns</p>
<pre><code>[('a',), ('b',), ('c',), ('d',), ('e',), ('f',)]
</code></pre>
<p>The solution is to pass the first element of each tuple to the <code>sent_tokenize</code> function.</p>
<pre><code>for row in data:
    text_sents = sent_tokenize(row[0])
</code></pre>
