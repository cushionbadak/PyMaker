<div class="post-text" itemprop="text">
<p>You're looking for <a href="https://docs.python.org/3/library/shutil.html#shutil.copytree" rel="nofollow noreferrer"><code>shutil.copytree()</code></a>, as <code>shutil.copy()</code> only copies single files.</p>
<p>Excerpt from the manual:</p>
<blockquote>
<p><code>shutil.copytree(src, dst, symlinks=False, ignore=None, copy_function=copy2, ignore_dangling_symlinks=False)</code></p>
<p>Recursively copy an entire directory tree rooted at <code>src</code>, returning the destination directory. The destination directory, named by <code>dst</code>, must not already exist; it will be created as well as missing parent directories. Permissions and times of directories are copied with <code>copystat()</code>, individual files are copied using <code>shutil.copy2()</code>.</p>
</blockquote>
