<p>I have a list of timestamps in Python above. How do I find the median time stamp? </p>
</div>
<div class="post-text" itemprop="text">
<p>Assuming you work with <code>pandas</code>, this would work:</p>
<pre><code>import pandas as pd
from pandas import Timestamp

ser = pd.Series([Timestamp('2008-01-25 00:00:00'), 
                 Timestamp('2008-01-31 00:00:00'), 
                 Timestamp('2008-01-31 00:00:00'), 
                 Timestamp('2008-01-09 00:00:00'),
                 Timestamp('2008-01-17 00:00:00')])

pd.Timestamp.fromordinal(int(ser.apply(lambda x: x.toordinal()).median()))
</code></pre>
<p>Result:</p>
<pre><code>Timestamp('2008-01-25 00:00:00')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you are using <strong>Python 3.4 or newer</strong>, there is no need to implement your own <code>median</code> function.</p>
<p>The <a href="https://docs.python.org/3/library/statistics.html" rel="nofollow noreferrer">statistics</a> package in the standard library already includes a <a href="https://docs.python.org/3/library/statistics.html#statistics.median" rel="nofollow noreferrer">median</a> function.  Batteries included!</p>
<hr/>
<p>so, assuming you have a sortable sequence of timestamps, you can just do:</p>
<pre><code>statistics.median(timestamps)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Usually timestamps are numerical values, therefore it is not too hard to calculate the (mathematical) median.</p>
<pre><code>&gt;&gt;&gt; t = [Timestamp('2008-01-25 00:00:00'), Timestamp('2012-01-25 00:00:00')]
&gt;&gt;&gt; t
[1201215600.0, 1327446000.0]
&gt;&gt;&gt; mid = sum(t)/len(t)
&gt;&gt;&gt; datetime.datetime.fromtimestamp(mid)
datetime.datetime(2010, 1, 24, 12, 0)
</code></pre>
