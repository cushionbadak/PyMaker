<div class="post-text" itemprop="text">
<p>I read a csv file, using <code>DictReader</code>.</p>
<p>I have a list of dictionaries:</p>
<p>eg:</p>
<pre><code>a = [{'Name':'A','Class':'1'},{'Name':'B','Class':'1'},{'Name':'C','Class':'2'}]
</code></pre>
<p>I want to count the number of entries in the list that have 'Class' == 1.</p>
<p>Is it possible to do it without a loop?</p>
<p>EDIT:</p>
<p>I have tried the following:</p>
<pre><code>count = 0
for k in a:
    if k['Class'] == '1':
        count += 1
print(count)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/library/functions#sum" rel="nofollow noreferrer"><code>sum</code></a> with <a href="https://docs.python.org/3/tutorial/classes.html#generator-expressions" rel="nofollow noreferrer">generator expression</a>:</p>
<pre><code>&gt;&gt;&gt; xs = [{'Name':'A','Class':'1'},
          {'Name':'B','Class':'1'},
          {'Name':'C','Class':'2'}]
&gt;&gt;&gt; sum(x.get('Class') == '1' for x in xs)
2
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using list comprehension as well, to retrieve dictionaries matching your criteria then calculating its <code>len</code>:</p>
<pre><code>&gt;&gt;&gt; len([d for d in a if d.get('Class')=='1'])
2
</code></pre>
<p>EDIT: Timing Profile under Python3.5.2</p>
<pre><code>&gt;&gt;&gt;import timeit
&gt;&gt;&gt;
&gt;&gt;&gt; timeit.timeit(stmt="len([d for d in a if d.get('Class')=='1'])", globals={'a':a})
0.8056499021768104
&gt;&gt;&gt;
&gt;&gt;&gt; timeit.timeit(stmt="sum(x.get('Class') == '1' for x in a)", globals={'a':a})
0.9977147589670494
&gt;&gt;&gt;
&gt;&gt;&gt; timeit.timeit(stmt="len(list(filter(lambda x: x.get('Class')=='1' , a)))", globals={'a':a})
1.3259506113099633
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Sure:</p>
<pre><code>a = [{'Name':'A','Class':'1'},{'Name':'B','Class':'1'},{'Name':'C','Class':'2'}]
print(len(filter(lambda x: x.get('Class')=='1' , a)))
</code></pre>
<p>Output:</p>
<pre><code>2
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is pretty the old fashion way to solve the problem, but if you want to do it in this way using the for loop, you can try this.</p>
<pre><code>a = [{'Name':'A','Class':'1'},{'Name':'B','Class':'1'},{'Name':'C','Class':'2'}]

cnt = 0
for k in a:
    if k['Class'] == '1':
        cnt += 1
print(cnt)
</code></pre>
<p>output</p>
<pre><code>2
</code></pre>
</div>
