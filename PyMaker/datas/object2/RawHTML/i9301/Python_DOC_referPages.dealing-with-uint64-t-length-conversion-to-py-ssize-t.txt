<p>I am reading a binary data format which contains a <code>length</code> field of type <code>unsigned long long</code> using cython.</p>
<p>After reading this value, I cast it to a <code>Py_ssize_t</code> variable to return to python space to be used in indexing.</p>
<p>Since <code>Py_ssize_t</code> is signed, it's maximum positive range must be lower than the maximum value <code>unsigned long long</code>.</p>
<p>Is there a general way of checking if the value I read can be stored in <code>Py_ssize_t</code>? Or put differently: Is there a way to obtain the maximum positive value of <code>Py_ssize_t</code>?</p>
