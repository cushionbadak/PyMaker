<div class="post-text" itemprop="text">
<p><code>...</code> is the <a href="https://docs.python.org/3/library/stdtypes.html?highlight=ellipsis#the-ellipsis-object" rel="nofollow noreferrer"><em>ellipsis object</em></a>. Here it is issued because else it would print forever! (due to infinite recursion)</p>
<p>and: <code>print(a[1][0])</code>:</p>
<p><code>a[1]</code> is <code>a</code> so <code>a[1][0]</code> is 2, like <code>a[1][1][1][1][0]</code> is 2 too.</p>
</div>
<div class="post-text" itemprop="text">
<p>Ellipsis <code>...</code> is used for slicing multidimensional numpy arrays.</p>
<p>The ellipsis syntax may be used to indicate selecting in full any remaining unspecified dimensions.</p>
<p><code>a[1]</code> is <code>[2, [...]]</code></p>
<p>so, <code>a[1][0]</code> is <code>2</code>.</p>
