<div class="post-text" itemprop="text">
<p>You can remove the dirty items from your list using a list comprehension:</p>
<pre><code>clean_list = [i for i in given_list if i not in dirty_set]
</code></pre>
<p>Or using the difference of sets if you want duplicates removed:</p>
<pre><code>clean_set = set(given_list).difference(dirty_set)
</code></pre>
<p>You can use the <code>-</code> operator if <code>dirty_set</code> is already a set but <code>set.difference</code> is more preferable since it takes any iterable.</p>
