<div class="post-text" itemprop="text">
<p>How can I run a batch file with Python (in the same directory as the Python scripts)?</p>
<p>Note that the directory cannot be constant as it can be changed from one user to another.</p>
</div>
<div class="post-text" itemprop="text">
<p>You need to find out where your script is and assemble an absolute path:</p>
<pre><code>import os
import subprocess

dirname = os.path.dirname(os.path.abspath(__file__))
cmd = os.path.join(dirname, 'mybatch_file')

subprocess.call(cmd)
</code></pre>
<h3>In Steps</h3>
<p>You can find out the name of script with:</p>
<pre><code>__file__
</code></pre>
<p>Now make it an absolute path:</p>
<pre><code>os.path.abspath
</code></pre>
<p>and get the directory it is in:</p>
<pre><code>os.path.dirname
</code></pre>
<p>Finally, join this path with your batch file name:</p>
<pre><code>os.path.join
</code></pre>
<p>before you feed it to:</p>
<pre><code>subprocess.call
</code></pre>
</div>
