<div class="post-text" itemprop="text">
<p>After a good while of research, I've been unable to find why this code counts the capital letters in a sentence when they're all capitalized, but will count <code>"0"</code> capital letters if I were to enter a sentence that contains any lowercase letters, such as: <code>"Hello World"</code>. </p>
<pre><code>message = input("Enter your sentence: ")

print("Capital Letters: ", sum(1 for c in message if message.isupper()))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Because the condition in your <code>if</code> clause is flawed; the <code>isupper</code> method returns <code>True</code> if <em>all</em> characters in the given string are uppercase: </p>
<pre><code>&gt;&gt;&gt; "UPPER".isupper()
True
&gt;&gt;&gt; "UPPEr".isupper()
False
</code></pre>
<p>See the <a href="https://docs.python.org/3/library/stdtypes.html#str.isupper" rel="nofollow noreferrer">documentation for <code>str.isupper</code></a>:</p>
<blockquote>
<p>Return <code>True</code> <em>if all cased characters in <code>str</code> are uppercase</em> and there is at least one cased character in <code>str</code>, <code>False</code> otherwise.</p>
</blockquote>
<p><sub>(Emphasis mine)</sub></p>
<p>Since you're checking on <code>message</code> with <code>message.isupper(()</code> this will return <code>False</code> all the time resulting in a sum of <code>0</code>.</p>
<p>Instead of checking on the full message with <code>message.isupper()</code>, use your <code>if</code> on a per-character basis with <code>if c.isupper()</code> for every <code>c</code> in <code>message</code>:</p>
<pre><code>print("Capital Letters: ", sum(1 for c in message if c.isupper()))
</code></pre>
<p>You could also take advantage of the fact that <code>True</code> acts like <code>1</code> and <code>False</code> like <code>0</code> to slightly trim that down if you'd want:</p>
<pre><code>print("Capital Letters: ", sum(c.isupper() for c in message))
</code></pre>
<p>or, if you like functional approaches, <code>map</code> the function to your input:</p>
<pre><code>print("Capital Letters: ", sum(map(str.isupper, message)))
</code></pre>
<p>Even though these might, subjectively, look nicer; they are generally more mystifying than the original approach (and slightly less efficient too).</p>
</div>
