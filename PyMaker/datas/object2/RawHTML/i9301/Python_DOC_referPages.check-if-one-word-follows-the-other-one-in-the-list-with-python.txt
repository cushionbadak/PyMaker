<p>I want to count the occurrence of words which are followed by each other in a given list using python.
E.g.</p>
<pre><code>my_list = ['ABC', 'DEF', 'PQR', 'ASD', 'ABC', 'DEF', 'ZXC', 'ABC', 'PQR'] 
</code></pre>
<p>In the above list, the count of times when <code>'ABC'</code> is followed by <code>'DEF'</code> is <code>2</code>.</p>
<p>Kindly help me out.
Thanks.  </p>
</div>
<div class="post-text" itemprop="text">
<p>A not very efficient way to do this might be:</p>
<pre><code>count = 0
for first,second in zip(my_list,my_list[1:]):
    if first == 'ABC' and second == 'DEF':
        count += 1
</code></pre>
<p>Or in a <em>one-liner</em> using <a href="https://docs.python.org/3/library/functions.html?highlight=sum#sum" rel="nofollow noreferrer"><code>sum(..)</code></a>:</p>
<pre><code>count = sum(1 for first,second in zip(my_list,my_list[1:]) if first == 'ABC' and second == 'DEF')
</code></pre>
<p>Or as @khelwood says, you can exploit the fact that <a href="https://docs.python.org/3/library/functions.html?highlight=sum#int" rel="nofollow noreferrer"><code>int(..)</code></a> of a boolean returns <code>0</code> and <code>1</code>:</p>
<pre><code>count = sum(first == 'ABC' and second == 'DEF' for first,second in zip(my_list,my_list[1:]))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>A <em>straightforward</em> way that does not involve creating an extra slice for pairwise iteration would be to simply use indexes:</p>
<pre><code>sum(1 for index in range(len(my_list) - 1) 
    if my_list[index] == 'ABC' and my_list[index + 1] == 'DEF')
</code></pre>
<p>Or, a bit shorter (thanks to @khelwood):</p>
<pre><code>sum(my_list[index] == 'ABC' and my_list[index + 1] == 'DEF' 
    for index in range(len(my_list) - 1))
</code></pre>
<hr/>
<p>Or, without <code>sum()</code> and expanded:</p>
<pre><code>c = 0
for index in range(len(my_list) - 1):
    if my_list[index] == 'ABC' and my_list[index + 1] == 'DEF':
        c += 1
</code></pre>
<p>Or:</p>
<pre><code>c = 0
for index in range(len(my_list) - 1):
    c += my_list[index] == 'ABC' and my_list[index + 1] == 'DEF'
</code></pre>
