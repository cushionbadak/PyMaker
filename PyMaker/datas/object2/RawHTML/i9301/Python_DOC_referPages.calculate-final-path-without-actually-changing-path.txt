<div class="post-text" itemprop="text">
<p>Is there an easy way in Python to resolve path operators like <code>..</code>?  For instance is there a function call that will convert: <code>/../../test/../path</code> to <code>/path</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.python.org/library/os.path.html#os.path.realpath"><code>os.path.realpath()</code></a> to get the canonical path.  To get a normalised relative path, use <a href="http://docs.python.org/library/os.path.html#os.path.normpath"><code>os.path.normpath()</code></a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>abspath</code> in the <code>os.path</code> module.</p>
<blockquote>
<p>os.path.abspath(path)</p>
<p>Return a normalized absolutized
  version of the pathname path. On most
  platforms, this is equivalent to
  normpath(join(os.getcwd(), path)).</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>If you are using the <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow">pathlib</a> library, the method to use is <a href="https://docs.python.org/3/library/pathlib.html#pathlib.Path.resolve" rel="nofollow"><code>resolve</code></a>:</p>

</div>
