<p>Here is an example. See <a href="https://stackoverflow.com/questions/2087370/decode-html-entities-in-python-string">this question</a> on how to decode the escaped HTML strings according to your version of python.</p>
<pre><code>table = """
&lt;table&gt;
&lt;tr&gt;
    &lt;th&gt;Montag&lt;/th&gt;
    &lt;td&gt;
     09:00 &amp;ndash; 00:30
    &lt;/td&gt;
&lt;/tr&gt;
&lt;tr&gt;
  &lt;th&gt;Dienstag&lt;/th&gt;
  &lt;td&gt;
   geschlossen
  &lt;/td&gt;
&lt;/tr&gt;
&lt;tr&gt;
  &lt;th&gt;Mittwoch&lt;/th&gt;
  &lt;td&gt;
  12:00 &amp;ndash; 00:30
  &lt;/td&gt;
&lt;/tr&gt;
&lt;tr&gt;
  &lt;th&gt;Donnerstag &amp;ndash; Sonntag&lt;/th&gt;
  &lt;td&gt;
    09:00 &amp;ndash; 00:30
  &lt;/td&gt;
&lt;/tr&gt;
&lt;/table&gt;"""

import json
from bs4 import BeautifulSoup

soup = BeautifulSoup(table, 'html5lib')

data = {}

for row in soup.findAll('tr'):
    th = row.find('th')
    td = row.find('td')
    data[th.text.strip()] = td.text.strip()

print(json.dumps(data))
</code></pre>

<div class="post-text" itemprop="text">
<p>Taking into account your actual problem statement of converting an HTML to JSON, you can use <a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.read_html.html" rel="nofollow noreferrer"><code>pandas.read_html()</code></a> to read HTML into a <code>DataFrame</code> and then convert it to a dictionary:</p>
<pre><code>import pandas as pd

data = """
your HTML abbreviated to save space
"""

df = pd.read_html(data)[0]
df.columns = ["label", "value"]
print(dict(zip(df.label, df.value)))
</code></pre>
<p>Prints:</p>
<pre><code>{'Montag': '09:00 – 00:30', 
 'Dienstag': 'geschlossen', 
 'Mittwoch': '12:00 – 00:30', 
 'Donnerstag – Sonntag': '09:00 – 00:30'}
</code></pre>
<p>You can then use <a href="https://docs.python.org/3/library/json.html#json.dumps" rel="nofollow noreferrer"><code>json.dumps()</code></a> to further dump the dictionary into a JSON string. </p>
<p>There is also <a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.to_json.html" rel="nofollow noreferrer"><code>.to_json()</code> method</a> that can directly dump a <code>DataFrame</code> to <code>JSON</code>, but I have not figured out how to use it in this particular case.</p>
