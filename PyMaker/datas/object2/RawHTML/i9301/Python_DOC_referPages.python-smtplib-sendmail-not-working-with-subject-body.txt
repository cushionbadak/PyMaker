<p>Whenever I try to use <code>sendmail(sender,reciever,message)</code>, it sends the email successfully, and the subject line is fine, but the 'body' of the e-mail is always missing.</p>
<p>Here is my full code:</p>
<pre><code>s = smtplib.SMTP("smtp.gmail.com",587)
s.ehlo()
s.starttls()
s.login(sender,password)
message = """From %s
Subject: %s
This is the body part""" % (sender,subject)
s.sendmail(sender,reciever,message)
s.quit()
</code></pre>
<p>Why is the body not being received?</p>
</div>
<div class="post-text" itemprop="text">
<p>You're missing an extra blank line between the Subject and the body, as outlined in <a href="https://docs.python.org/3/library/smtplib.html#smtp-example" rel="nofollow noreferrer">the official documentation's example</a> (paraphrased):</p>
<pre><code>msg = "From: %s\r\nTo: %s\r\n\r\nBody text"
</code></pre>
<p>In any case, I would follow the recommendations in the documentation and use the Message object from the email package instead. That way you don't have to worry about formatting the string correctly, and can just do this:</p>
<pre><code>msg = EmailMessage()
msg['Subject'] = "Subject"
msg['From'] = Address("Name", "mailbox", "domain")
msg['To'] = (Address("Name Too", "test", "example.com"),
             Address("And so on", "so", "forth"))
msg.set_content("The body of the e-mail")

with smtplib.SMTP("smtp.gmail.com", 587) as s:
    s.send_message(msg)
</code></pre>
<p>See the <a href="https://docs.python.org/3/library/email.examples.html#email-examples" rel="nofollow noreferrer">official documentation's examples</a> for more.</p>
