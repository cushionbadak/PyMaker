<p>Given a set of edges of a graph as follows:</p>
<pre><code>/* Edges */
("C", "E", 2)
("C", "D", 3)
("E", "D", 1)
("E", "G", 3)
("E", "F", 2)
("D", "F", 4)
("G", "H", 2)
("F", "G", 2)
("F", "H", 1)
</code></pre>
<p>How can I write the code to save them in a dictionary as follows (C to H, in python):</p>
<pre><code>{"C": {"E": 2, "D": 3}, "E": {"D": 1, "G": 3, "F": 2}, "D": {"F": 4}, "G": {"H": 2}, "F": {"G": 2, "H": 1}, "H": {}}
</code></pre>
<p>Thank you for your help.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can parse the keys and value with <code>re</code> and use <code>collections.defaultdict</code>:</p>
<pre><code>import re, collections
d = collections.defaultdict(dict)
_, *data = [re.findall('(?&lt;=")\w+(?=")', i) for i in content.split('\n')]
for a, b, c in data:
  d[a][b] = int(c)

d = {**d, **{i[1]:{} for i in data if not any(c[0] == i[1] for c in data)}}
print(dict(d))
</code></pre>
<p>Output:</p>
<pre><code>{'C': {'E': 2, 'D': 3}, 'E': {'D': 1, 'G': 3, 'F': 2}, 'D': {'F': 4}, 'G': {'H': 2}, 'F': {'H': 1, 'G': 2}, 'H': {}}   
</code></pre>
