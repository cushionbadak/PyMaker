<div class="post-text" itemprop="text">
<p>You are probably using Python 2.x, where division of two integers gets rounded result.
Try to specify at least one number as float, e.g. <code>print("{:.4f}".format(333.0/106))</code>
or <code>print("{:.4f}".format(float(333)/106))</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Try the following code:</p>
<pre><code>pi = 3.1415926
precision = 4
