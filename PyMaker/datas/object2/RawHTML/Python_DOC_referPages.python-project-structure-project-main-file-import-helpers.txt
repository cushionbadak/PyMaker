<div class="post-text" itemprop="text">
<p>When you import from a package, you are importing from the <code>__init__.py</code> of that package....</p>
<p>so in your utils package, your <code>__init__.py</code> is empty. </p>
<p>trying adding this to your <code>utils/__init__.py</code></p>
<pre><code>print("utils/__init__.py")
from . import eggs
from . import spam
</code></pre>
<p>Now, when you say <code>from utils import  eggs, spam</code> you are saying, from the <strong>init</strong>.py of the utils package, import the things that I imported in there.</p>
<p>Also, in <code>pystructure.py</code></p>
<p>change this </p>
<pre><code>from .utils import  eggs, spam 
</code></pre>
<p>into this</p>
<pre><code>from utils import  eggs, spam 
</code></pre>
