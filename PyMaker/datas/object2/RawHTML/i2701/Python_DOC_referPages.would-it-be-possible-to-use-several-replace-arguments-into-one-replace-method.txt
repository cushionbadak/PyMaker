<p>I would like to delete some characters into a string (text with several words).</p>
<p>For this purpose I use the code below:</p>
<pre><code>text2 = text.replace(',', '').replace('\n', '').replace('.', '').replace(':', '')
</code></pre>
<p>I have to add one replace method for each character to replace.</p>
<p>Is there any form to code in a smarter way? Something like this:</p>
<pre><code>text2 = text.replace(',' '.' '\n' ':', '') # of course this is not working, just an example about how it could be.
</code></pre>
<p>Thank you  </p>

<div class="post-text" itemprop="text">
<p>use <code>re.sub(r'[,.:\n]', '', text)</code></p>
</div>
<span class="comment-copy">You can use <a href="https://docs.python.org/3/library/re.html#re.sub" rel="nofollow noreferrer"><code>re.sub</code></a> with a regular expression.</span>
<span class="comment-copy">Or list comprehension and a join: <code>text2 = ''.join(t for t in text if t not in ',\n.:')</code></span>
<span class="comment-copy"><code>str.translate</code> would be likely fastest for this case (single character replacements)</span>
