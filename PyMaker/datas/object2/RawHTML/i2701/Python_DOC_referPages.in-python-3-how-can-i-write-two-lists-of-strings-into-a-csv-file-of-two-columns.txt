<p>I currently have two lists in Python that look like:</p>
<pre><code>list1 = ['Steve', 'Mark', 'Jeff', ... ]
list2 = ['steve@mac.com', 'mark223@mac.com', 'jeffrey@mac.com', ...]
</code></pre>
<p>From this, I am trying to create a csv file that looks like:</p>
<pre><code>Steve   steve@mac.com
Mark    mark223@mac.com
Jeff    jeffrey@mac.com
....
</code></pre>
<p>The code I am using is:</p>
<pre><code>import csv

with open('output.csv','w') as f:
    writer = csv.writer(f)
    writer.writerows(list1,list2)
</code></pre>
<p>This results in a file that treats each character as a new cell. What am I doing wrong? thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>You can <a href="https://docs.python.org/3.7/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> those lists together so each pair (list1, list2) is treated  as single row:</p>
<pre><code>import csv

list1 = ['Steve', 'Mark', 'Jeff']
list2 = ['steve@mac.com', 'mark223@mac.com', 'jeffrey@mac.com']

with open('output.csv','w') as f:
    writer = csv.writer(f)
    writer.writerows(zip(list1,list2))
</code></pre>
<p>Sample output:</p>
<pre><code>$ cat output.csv 
Steve,steve@mac.com
Mark,mark223@mac.com
Jeff,jeffrey@mac.com
</code></pre>
<p>If you need your output to be space or tab separated (instead of comma), you can specify it as a parameter to <a href="https://docs.python.org/3/library/csv.html#csv.writer" rel="nofollow noreferrer"><code>csv.writer</code></a>. For example:</p>
<pre><code>import csv

list1 = ['Steve', 'Mark', 'Jeff']
list2 = ['steve@mac.com', 'mark223@mac.com', 'jeffrey@mac.com']

with open('output.csv','w') as f:
    writer = csv.writer(f, delimiter='\t')
    writer.writerows(zip(list1,list2))
</code></pre>
<p>Produces:</p>
<pre><code>$ cat output.csv 
Steve   steve@mac.com
Mark    mark223@mac.com
Jeff    jeffrey@mac.com
</code></pre>
