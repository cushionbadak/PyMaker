<p>If you have the two dictionaries as <code>dict1</code>(<code>None</code> values) and <code>dict2</code>(meaningful values) you can <code>update</code> <code>dict1</code> as:</p>
<pre><code>dict1.update(dict2)
</code></pre>
<p>As other answers have pointed out, please note that you do not have a <code>dict</code>, you have a <code>list</code> of <code>dict</code>s, and you will need to make a dict first, which you can do as:</p>
<pre><code>a_var = [{1 : None}, {2 : 'b'}]
dict1 =  {x.keys()[0]: x.values()[0] for x in a_var}
</code></pre>
<p>Once you have that:</p>
<pre><code>dict2 = {1 : 'a', 2 : 'bb', 3 : 'c'}
dict1.update(dict2)

print(dict1)
</code></pre>
<p>Output:</p>
<pre><code>{1: 'a', 2: 'bb', 3: 'c'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>With the original dictionaries as <code>dict1</code> and <code>dict2</code> the new one can be created in one line using <a href="https://docs.python.org/3/library/stdtypes.html#dict.get" rel="nofollow noreferrer"><code>dict.get</code></a> as follows:</p>
<pre><code>new_dict = {k: dict2.get(k, v) for k, v in dict1.items()}
</code></pre>
<p>Or, because you know the default value in <code>dict1</code> is <code>None</code>, you could just use the keys:</p>
<pre><code>new_dict = {k: dict2.get(k, None) for k in dict1}
</code></pre>
<p>Or if you don't care about mutating <code>dict1</code>:</p>
<pre><code>dict1.update(dict2)
</code></pre>
