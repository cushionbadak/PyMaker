<p>I would suggest you create a class with static methods:</p>
<pre><code>class Fibonacii(object):
   def __init__(self):
      # this is the constructor you can initialize stuff here if required
      # if not just pass it like in this example
      pass

   @staticmethod
   def fib(n):
    if n&lt;=1:
        return n
    else:
        return (Fibonacii.fib(n-1) + Fibonacii.fib(n-2))

   @staticmethod
   def facts(n):
    if n == 0 or n == 1:
        return 1
    if n&gt;=1:
       return n * Fibonacii.facts(n-1)
</code></pre>
<p>static method have the advantage that they can be used without creating an object of that class. So in your script you can do:</p>
<pre><code>if __name__ == "__main__":
   n = int(input("Enter number of terms:"))
   print("Fibonacii series are: ")
   for i in range (n):
       print(Fibonacii.fib(i))

   #for factorial
   n=int(input("Enter the no to find factorial: "))
   print("The facatorial of a given no",str(n),"is:",Fibonacii.facts(n))
</code></pre>
<p>The part after main is the part of the script that is being executed if it is the main script. Notice that the class is never used to create a object. But it is used to save your two functions and can be used to call them.</p>
