<div class="post-text" itemprop="text">
<p><code>x</code> inside the for loop is already the relevant item from <code>products</code>. It is not a counter and there is no need to say <code>products[x]</code>.</p>
<pre><code>for x in products:
   if x[0] == item:
       ...
</code></pre>
<p>(Also, you probably want to use a dictionary rather than a list here; then you can find <code>products[item]</code> directly rather than iterating through the list each time.)</p>
</div>
<div class="post-text" itemprop="text">
<p>The variable <code>x</code> is assigned with each item in the <code>products</code> list with the <code>for</code> statement, and is not an index, so you should access it as an item in the <code>products</code> list:</p>
<pre><code>def instore(item):
    for x in products:
        if x[0] == item:
            print(x[0] + ': \n id: ' + x[1] + ' In Stock: ' + x[2])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As the other's have already told you about <code>x</code> being an element of the list, instead of being a counter variable. If you still want to use a counter, you can do something like,</p>
<pre><code>for x in range(len(products)):
   if products[x][0] == item:
       ...
</code></pre>
<p>The <code>range(n)</code> function creates a range (list) from <code>0 to n-1</code>. You can read more about range, from <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer">here</a>.</p>
</div>
