<div class="post-text" itemprop="text">
<p>Simply yes, see also the <a href="https://docs.python.org/3/howto/sorting.html" rel="nofollow noreferrer">documentation</a>:</p>
<blockquote>
<p>A simple ascending sort is very easy: just call the sorted() function. It returns a new sorted list</p>
</blockquote>
<pre><code>&gt;&gt;&gt; sorted([5, 2, 3, 1, 4])
[1, 2, 3, 4, 5]
</code></pre>
<p>You can also try it yourself:</p>
<pre><code>&gt;&gt;&gt; a = [1, 2, 3]
&gt;&gt;&gt; sorted(a) is a
False
</code></pre>
