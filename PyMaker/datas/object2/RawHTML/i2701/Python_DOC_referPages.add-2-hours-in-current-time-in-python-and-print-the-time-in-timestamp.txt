<p>I would recommend the following </p>
<pre><code>from datetime import datetime, timedelta
two_hours_from_now = datetime.now() + timedelta(hours=2)
print(two_hours_from_now.timestamp())
</code></pre>

<div class="post-text" itemprop="text">
<p>You can do as follow:</p>
<pre><code>import datetime


current_time = datetime.datetime.now()

later = current_time + datetime.timedelta(hours=2)

print(later.timestamp())
</code></pre>
<p>You get:</p>
<pre><code> 1535031690.031316
</code></pre>
<p>Quoting the <a href="https://docs.python.org/3/library/datetime.html#datetime-objects" rel="nofollow noreferrer">documentation</a>:</p>
<blockquote>
<p>Return POSIX timestamp corresponding to the datetime instance. The return value is a float similar to that returned by <code>time.time()</code>.</p>
</blockquote>
</div>
