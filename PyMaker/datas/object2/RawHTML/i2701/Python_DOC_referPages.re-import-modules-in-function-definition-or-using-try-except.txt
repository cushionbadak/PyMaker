<p>Which is more elegant of these two:</p>
<p>Importing the module anyway:</p>
<pre><code>def shuffle_sequence(seq):
    import random
    seq = list(seq)
    random.shuffle(seq)
    return ''.join(seq)
</code></pre>
<p>Or use Try block to do it:</p>
<pre><code>def shuffle_sequence(seq):
    seq = list(seq)
    try:
        random.shuffle(seq)
    except NameError:
        import random
        random.shuffle(seq)
    return ''.join(seq)
</code></pre>

<p>Neither. </p>
<p>Import at the top of the module always as desribed in the <a href="https://www.python.org/dev/peps/pep-0008/#imports" rel="nofollow noreferrer"><strong>python style guide</strong></a>!
