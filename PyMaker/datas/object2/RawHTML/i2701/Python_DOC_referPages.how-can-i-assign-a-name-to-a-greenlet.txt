<div class="post-text" itemprop="text">
<p><code>gevent.Greenlet.name</code> is not a common <code>property</code>, but a <code>gevent.util.readproperty</code> obj, which is <a href="https://github.com/gevent/gevent/blob/master/src/gevent/_util.py#L137-L152" rel="nofollow noreferrer">a special non-data descriptor</a> works like <code>@property</code>, and for a non-data descriptor:</p>
<blockquote>
<p>... In contrast, non-data descriptors can be overridden by instances.</p>
</blockquote>
<p>you could override it simply:</p>
<pre><code>&gt;&gt;&gt; gr = gevent.spawn(gevent.sleep, (1,))
&gt;&gt;&gt; gr.name = "a_cool_name"
&gt;&gt;&gt; print(gr)
&lt;Greenlet "a_cool_name" at 0x1082a47b8: sleep((1,))&gt;
</code></pre>
<p>read more on the gevent source and <a href="https://docs.python.org/3/reference/datamodel.html#invoking-descriptors" rel="nofollow noreferrer">the descriptor doc</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Instantiate the class first and then overwrite the method as such;</p>
<pre><code>my_obj = Class() #Instantiate the class
my_obj.name = 'this is the string'
</code></pre>
<p>This will replace the constructed name with your own string.</p>
