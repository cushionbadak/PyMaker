<div class="post-text" itemprop="text">
<p>If you want to convert a string to a hex representation of its bits, that's not what <code>int(s, 16)</code> does. <code>int(s, 16)</code> tries to treat the characters of <code>s</code> as the base-16 representation of an integer. <code>'deadbeef'</code> contains only characters that are valid digits in base 16:</p>
<pre><code>d = 13
e = 14
a = 10
d = 13
b = 11
e = 14
e = 14
f = 15

deadbeef = 13*16**7 + 14*16**6 + 10*16**5 + 13*16**4 + 11*16**3 + 14*16**2 + 14*16**1 + 15
         = 3735928559
</code></pre>
<p>None of your other strings have that property.</p>
<p>If you want hex digits corresponding to the bits representing a string, first, choose an encoding (UTF8, probably), then use something like <a href="https://docs.python.org/3/library/binascii.html#binascii.hexlify" rel="nofollow noreferrer"><code>binascii.hexlify</code></a>:</p>
<pre><code>&gt;&gt;&gt; import binascii
&gt;&gt;&gt; binascii.hexlify('hello world'.encode('utf8'))
b'68656c6c6f20776f726c64'
</code></pre>

<p>int( hex_value,16) is used to convert hex string to int. You are facing an error if you pass a invalid hex character.
For your use case of converting string to bits you can use <strong>bitarray</strong> module</p>
<pre><code>import bitarray
ba = bitarray.bitarray()
ba.frombytes('Hi'.encode('utf-8'))
print ba
</code></pre>
