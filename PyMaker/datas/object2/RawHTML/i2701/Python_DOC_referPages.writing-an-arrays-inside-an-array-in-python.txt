<div class="post-text" itemprop="text">
<p>Everything is an object in Python. And Lists can accept any type of data. So List can accept Lists too. Using this you can just place lists in a list. </p>
<pre><code>array1 = [] # main array

array2 = [] 
array3 = []
array4 = []

array1 = [array2, array3, array4] # Or you can just give [] instead of variable names
</code></pre>
<h1>This will create the following</h1>
<pre><code>array1
[[], [], []]
</code></pre>
