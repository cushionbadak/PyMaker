<div class="post-text" itemprop="text">
<p>You can use <code>iter(input, '')</code> to accept input until a blank line:</p>
<pre><code>recipe_name = input ("Recipe name: ");
print('Please add the directions AND ingredients')
with open (recipe_name + ".txt", "x") as f:
    for line in iter(input, ''):
        f.write(line + '\n');
</code></pre>

<div class="post-text" itemprop="text">
<p>OR using a while loop, and keep appending to list:</p>
<pre><code>l=[]
a=input()
while a:
   l.append(a)
   a=input()
print(l)
</code></pre>
<p>Example output:</p>
<pre><code>First line
Second line

['First line', 'Second line']
</code></pre>
<p>Or for full code:</p>
<pre><code>reciept_name=input('Reciept: ')
a=input('Your input: ')
with open(reciept_name+'.txt','w') as f:
   while a:
      f.write(a+'\n')
      a=input('Your input: ')
</code></pre>
