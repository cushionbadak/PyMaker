<p>You should not do in this way. First you need to call the external command with <a href="https://stackoverflow.com/questions/2502833/store-output-of-subprocess-popen-call-in-a-string">output captured</a>, and then write the output you captured to a file:</p>
<pre><code>import subprocess

ip2 = raw_input('Enter ip of target: ')
p = subprocess.Popen(["nmap", "-sP", ip2], stdout=subprocess.PIPE)
out, err = p.communicate()
with open('123.txt', 'w') as f:
    f.write(out)
</code></pre>
