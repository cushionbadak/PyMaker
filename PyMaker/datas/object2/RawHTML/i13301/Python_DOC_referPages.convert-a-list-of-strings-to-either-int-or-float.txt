<div class="post-text" itemprop="text">
<p>I have a list which looks something like this:</p>
<pre><code>['1', '2', '3.4', '5.6', '7.8']
</code></pre>
<p>How do I change the first two to <code>int</code> and the three last to <code>float</code>?</p>
<p>I want my list to look like this:</p>
<pre><code>[1, 2, 3.4, 5.6, 7.8]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://stackoverflow.com/q/4406389/4099593">conditional inside a list comprehension</a></p>
<pre><code>&gt;&gt;&gt; s = ['1', '2', '3.4', '5.6', '7.8']
&gt;&gt;&gt; [float(i) if '.' in i else int(i) for i in s]
[1, 2, 3.4, 5.6, 7.8]
</code></pre>
<p>Interesting edge case of exponentials. You can add onto the conditional.</p>
<pre><code>&gt;&gt;&gt; s = ['1', '2', '3.4', '5.6', '7.8' , '1e2']
&gt;&gt;&gt; [float(i) if '.' in i or 'e' in i else int(i) for i in s]
[1, 2, 3.4, 5.6, 7.8, 100.0]
</code></pre>
<p>Using <a href="https://docs.python.org/2/library/stdtypes.html#str.isdigit" rel="nofollow noreferrer"><code>isdigit</code></a> is the best as it takes care of all the edge cases (mentioned by <a href="https://stackoverflow.com/users/1322401/steven-rumbalski">Steven</a> in a <a href="https://stackoverflow.com/questions/33130279/convert-a-list-of-strings-to-int-and-float/33130297?noredirect=1#comment54072755_33130297">comment</a>)</p>
<pre><code>&gt;&gt;&gt; s = ['1', '2', '3.4', '5.6', '7.8']
&gt;&gt;&gt; [int(i) if i.isdigit() else float(i) for i in s]
[1, 2, 3.4, 5.6, 7.8, 100.0]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a helper function:</p>
<pre><code>def int_or_float(s):
    try:
        return int(s)
    except ValueError:
        return float(s)
</code></pre>
<p>Then use a list comprehension to apply the function:</p>
<pre><code>[int_or_float(el) for el in lst] 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Why not use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow"><code>ast.literal_eval</code></a>?</p>
<pre><code>import ast

[ast.literal_eval(el) for el in lst]
</code></pre>
<p>Should handle all corner cases. It's a little heavyweight for this use case, but if you expect to handle any Number-like string in the list, this'll do it.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <em>isdigit</em> method of string:</p>
<pre><code>numbers = [int(s) if s.isdigit() else float(s) for s in numbers]
</code></pre>
<p>or with map:</p>
<pre><code>numbers = map(lambda x: int(x) if x.isdigit() else float(x), numbers)
</code></pre>
</div>
