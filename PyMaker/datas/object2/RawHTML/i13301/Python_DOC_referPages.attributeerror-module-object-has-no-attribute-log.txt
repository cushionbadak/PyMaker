<p>Use Python's <code>logging</code> module for your logging needs.
Initialize a logger first, then log to it using <code>logger.info</code>/<code>debug</code>/<code>warning</code>/<code>critical</code>. 
like so:</p>
<pre><code>import logging
logging.basicConfig(filename=filename)
logging.info('Session Start')
</code></pre>
<p>I strongly suggest you read up on the logger module documentation.<br/>
