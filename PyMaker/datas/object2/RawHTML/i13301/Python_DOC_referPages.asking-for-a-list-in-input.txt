<div class="post-text" itemprop="text">
<p>Python 2.7 will just work:</p>
<pre><code>&gt;&gt;&gt; input() # [1, 2, 3]
[1, 2, 3]

&gt;&gt;&gt; type(_)
list
</code></pre>
<p>Python 3:</p>
<pre><code>&gt;&gt;&gt; import ast
&gt;&gt;&gt; ast.literal_eval(input()) # [1, 2, 3]

</code></pre>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow"><code>ast.literal_eval</code></a> to parse a list of numbers entered by the user:</p>
<pre><code>import ast

numbers = input('Type in a list of numbers, separated by comma:\n')

lst = list(ast.literal_eval(numbers)))

print('You entered the following list of numbers:')
print(lst)
</code></pre>
<pre><code>Type in a list of numbers, separated by comma:
1, 523, 235235, 34645, 56756, 21124, 346346, 658568, 123123, 345, 2
You entered the following list of numbers:
[1, 523, 235235, 34645, 56756, 21124, 346346, 658568, 123123, 345, 2]
</code></pre>
<p>Note that with Python 2, you need to use <code>raw_input()</code> instead of just <code>input()</code>.</p>
