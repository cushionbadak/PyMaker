<p>I have float value as follow:</p>
<pre><code>6.07345232904e+23
</code></pre>
<p>How can I convert this to two decimal places?  </p>
<p>I have tried with <code>round()</code>.<br/>

<div class="post-text" itemprop="text">
<p>You can't. That value is approximately 607345232904000000000000, which already has more than two decimal places. If you want the <em>representation</em> to have two decimal places then you must specify that.</p>
<pre><code>&gt;&gt;&gt; '{:.3g}'.format(6.07345232904e+23)
'6.07e+23'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>To answer your clarification of what you actually want to do, namely comparing two numbers with a given precision:</p>
<p>There are different ways to do this, depending on what your criteria are. The simplest solution is</p>
<pre><code>abs(a-b) &lt;= tolerance
</code></pre>
<p>If you are using Python 3.5, you can use <a href="https://docs.python.org/3/library/math.html#math.isclose" rel="nofollow">math.isclose</a>. For earlier version, this should be functionally equivalent</p>
<pre><code>def isclose(a, b, *, rel_tol=1e-09, abs_tol=0.0):
  return abs(a-b) &lt;= max(rel_tol * max(abs(a), abs(b)), abs_tol)
</code></pre>
<p>If you really need to handle all corner cases, you will have to to <a href="https://github.com/PythonCHB/close_pep/blob/master/is_close.py" rel="nofollow">some additional work</a>.</p>
</div>
