<p>I have a function that takes a list of items i.e. [item_1, item_2] etc. </p>
<p>Each item itself is a dictionary on another page. i.e. item_1 {key: value, etc}
I want to display the value of the key (called "name") for each item in the list.</p>
<p>For example if the input to the function, list_of_items() is [item_1, item_2]</p>
<p>Should return 'name of item 1, name of item 2' as a string separated by commas.</p>
<pre><code>new_list = []
for i in items:
    new_list.append(i["name"])

return str(new_list).strip('[]')
</code></pre>
<p>This code returns "'name of item 1', 'name of item 2'" and it is the closest I have gotten.</p>
<p>How can I return the string I want?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join()</code></a> to join the list of strings into a single string. Example -</p>
<pre><code>new_list = []
for i in items:
    new_list.append(i["name"])

return ','.join(new_list)
</code></pre>
<p>This can be simplified using the below list comprehension -</p>
<pre><code>return ','.join([i['name'] for i in items])
</code></pre>
<hr/>
<p>Demo -</p>
<pre><code>&gt;&gt;&gt; items = [{'name':'a'},{'name':'b'},{'name':'c'}]
&gt;&gt;&gt; ','.join([i['name'] for i in items])
'a,b,c'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The answer from @AnandSKumar is perfect. In addition, note that you don't actually have a function in the code that you provided. A function in python begins with the <code>def</code> statement. So copying some code from @AnandSKumar, you have the following function:</p>
<pre><code>def dictsName2str ( listOfDicts ) :
    return ','.join([i['name'] for i in listOfDicts])
</code></pre>
<p>Then you can call your function with the following code:</p>
<pre><code>&gt;&gt;&gt; items = [{'name':'a'},{'name':'b'},{'name':'c'}]
&gt;&gt;&gt; myStr = dictsName2str( items )
</code></pre>
