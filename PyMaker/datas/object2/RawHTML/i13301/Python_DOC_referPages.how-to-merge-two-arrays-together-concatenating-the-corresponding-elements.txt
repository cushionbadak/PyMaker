<p>I'm not asking about the easy merging of array like <code>c = a+b</code> (which would yield <code>[1,2,3,4,5,6]</code>.</p>
<p>What I am looking for is to join the contents of two arrays, so that the end-result would be as the following (given <code>a</code> and <code>b</code> written previously).</p>
<pre><code>c = <em>what-do-I-write-here?</em> # [14,25,36]</code></pre>
<p>How do I solve this?</p>

<div class="post-text" itemprop="text">
<h3>Introduction</h3>
<p>You can <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow"><em><code>zip</code></em></a> together your two lists, forming a new lists that will contain <em>two-element tuples</em> (pairs), where each pair consists of the elements at the corresponding index, from the two lists.</p>
<pre class="lang-none prettyprint-override"><code>Python 3.5.0 (default, Sep 20 2015, 11:28:25)
[GCC 5.2.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
&gt;&gt;&gt; <b>a = [1, 2, 3]</b>
&gt;&gt;&gt; <b>b = [4, 5, 6]</b>
&gt;&gt;&gt; <b>list (zip (a,b))</b>
[(1, 4), (2, 5), (3, 6)]
</code></pre>
<p>This in turn will make it very easy to iterate over each pair, and create a new list with the desired contents.</p>
<hr/>
<h3>Sample Implementation</h3>
<p>In your question you have written that the result should be <code>[14,25,36]</code> â€” implying that you are looking to concatenate the elements lexiographically, but then have the result still be <em>ints</em>.</p>
<p>You could solve it in a easy manner using code such as;</p>
<pre><code>a = [1, 2, 3]
b = [4, 5, 6]
c = [ int(''.join (map (str, xs))) for xs in zip (a,b) ]
</code></pre>
<pre class="lang-none prettyprint-override"><code>c <em>is now</em> [14, 25, 36]</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/2/library/functions.html#zip" rel="nofollow"><code>zip</code></a> to match the appropriate elements, and from there it's just a matter of converting them to strings and joining them:</p>
<pre><code>a = [1,2,3]
b = [4,5,6]
c = [''.join(str(s) for s in x) for x in zip(a, b)]
</code></pre>
</div>
