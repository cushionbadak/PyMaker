<div class="post-text" itemprop="text">
<p>Without a regex:</p>
<pre><code>lambda x: int(x.partition('/')[0][2:])
</code></pre>
<p>This takes the part of the string before the <code>/</code>, then everything but the starting <code>co</code>, then converts that to an integer.</p>
<p>I used <a href="https://docs.python.org/3/library/stdtypes.html#str.partition" rel="nofollow"><code>str.partition()</code></a> as it is faster than <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>str.split()</code></a> for the <em>split only once</em> case.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; wrong_order = [u'co1/co2', u'co10/co11', u'co11/co12', u'co12/co13', u'co2/co3',
...        u'co3/co4', u'co4/co5', u'co5/co6', u'co6/co7', u'co7/co8', u'co8/co9',
...        u'co9/co10']
&gt;&gt;&gt; sorted(wrong_order, key=lambda x: int(x.partition('/')[0][2:]))
[u'co1/co2', u'co2/co3', u'co3/co4', u'co4/co5', u'co5/co6', u'co6/co7', u'co7/co8', u'co8/co9', u'co9/co10', u'co10/co11', u'co11/co12', u'co12/co13']
</code></pre>
</div>
