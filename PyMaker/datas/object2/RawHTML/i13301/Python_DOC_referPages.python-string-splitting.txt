<p>Writing
<code>'1000011'.split('1')</code>
gives</p>
<pre><code>['', '0000', '', '']
</code></pre>
<p>What I want is:</p>
<pre><code>['1', '0000', '11']
</code></pre>
<p>How do I achieve this?</p>

<div class="post-text" itemprop="text">
<p>The <code>str.split(sep)</code> method does not add the <code>sep</code> delimiter to the output list.</p>
<p>You want to <em>group</em> string characters e.g. using <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow"><code>itertools.groupby</code></a>:</p>
<pre><code>In:  import itertools

In:  [''.join(g) for _, g in itertools.groupby('1000011')]
Out: ['1', '0000', '11']
</code></pre>
<p>We didn't specify the <code>key</code> argument and the default <code>key</code> function just returns the element unchanged. <code>g</code> is then the group of <code>key</code> characters.</p>
</div>
