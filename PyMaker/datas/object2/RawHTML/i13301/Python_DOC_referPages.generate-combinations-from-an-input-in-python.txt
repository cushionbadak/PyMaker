<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow"><code>itertools.product</code></a> to get the combinations and <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>string.format</code></a> to merge those into the template string. (First, replace the <code>?</code> with <code>{}</code> to get <a href="https://docs.python.org/3/library/string.html#formatstrings" rel="nofollow">format string syntax</a>.)</p>
<pre><code>def combine(template, options):
    template = template.replace('?', '{}')
    for opts in itertools.product(*options):
        yield template.format(*opts)
</code></pre>
<p>Example:</p>
<pre><code>&gt;&gt;&gt; list(combine('AB?D?', ['ABC', 'DEF']))
['ABADD', 'ABADE', 'ABADF', 'ABBDD', 'ABBDE', 'ABBDF', 'ABCDD', 'ABCDE', 'ABCDF']
</code></pre>
