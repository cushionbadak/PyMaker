<p>You should check out the <a href="http://www.lexicon.net/sjmachin/xlrd.html#xlrd.Sheet-class" rel="nofollow noreferrer">functions in the <code>Sheet</code> class of xlrd</a>.</p>
<p>If you are using python 3.4+ there is a <a href="https://docs.python.org/3/library/statistics.html" rel="nofollow noreferrer">standard library module for statistics</a> which will do mean and standard deviation. Check out the documentation here and decide whether you need <code>stdev</code> or <code>pstdev</code>. Otherwise <a href="https://stackoverflow.com/a/27758326/798684">here is a previous answer</a> that provides some example code for the mean and standard deviation operations.</p>
<pre><code>from statistics import mean, stdev
my_stats = []
for i in range(sheet.ncols):
    m = mean(c.value for c in sheet.col(i))
    s = stdev(c.value for c in sheet.col(i))
    my_stats.append((m,s))
</code></pre>

<p>You can use <strong>numpy</strong> library for calculate the std of a huge number of values in a excel sheet.
See this code below:</p>
<p>import csv</p>
<pre><code>import numpy as np
import math

#read csv file and convert it in to an array
csv= np.genfromtxt
csv = np.genfromtxt('your excel sheet name.csv',delimiter=",")
N=len(csv[:])
#print N(N is number of raws in the sheet)
#X is x bar which is the mean of each column
X = np.mean(csv,axis=0)
sum = 0

for j in range(len (X[:])):
    for i in range(N):
        val= csv[i,j]-X[j]
        squre = val**2
        sum= sum+squre
    final = math.sqrt(sum/(N-1))
    print final
</code></pre>
<p>The output of this code will show an array of stds of each column.</p>
