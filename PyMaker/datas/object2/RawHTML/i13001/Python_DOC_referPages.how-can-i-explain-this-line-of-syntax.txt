<p>Basically  what exactly does this code below mean in english</p>
<pre><code>for i in values : 
   for x in othervalues :
</code></pre>

<div class="post-text" itemprop="text">
<p>As @khelwood said, it simply iterates on two loops.</p>
<p>It iterates through the values in <code>values</code>, assigning each value to the variable <code>i</code></p>
<p>Inside such loop it does the same thing, iterating on the values of <code>othervalues</code> and assigning each value to the variable <code>x</code></p>
<p>You can verify it simply adding a print statement inside the loop, that shows the values of <code>i</code> and <code>x</code></p>
<pre><code>for i in values : 
    for x in othervalues :
        print('i={}, x={}'.format(i,x))
</code></pre>
<p>e.g. with an input of</p>
<pre><code>values = 'abc'
othervalues = [1, 2, 3, 4, 5]
</code></pre>
<p>it produces</p>
<pre><code>i=a, x=1
i=a, x=2
i=a, x=3
i=a, x=4
i=a, x=5
i=b, x=1
i=b, x=2
i=b, x=3
i=b, x=4
i=b, x=5
i=c, x=1
i=c, x=2
i=c, x=3
i=c, x=4
i=c, x=5
</code></pre>
<p>Please make sure you understand how iteration works in python, reading the <a href="https://docs.python.org/3/library/stdtypes.html#iterator-types" rel="nofollow noreferrer">official docs</a>
and <a href="https://stackoverflow.com/questions/9884132/what-exactly-are-pythons-iterator-iterable-and-iteration-protocols">this SO Q&amp;A</a>
and more tutorials you can find on the internet.</p>
