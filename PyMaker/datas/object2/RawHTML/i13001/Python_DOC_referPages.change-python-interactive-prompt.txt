<div class="post-text" itemprop="text">
<p>You remember correctly.  </p>
<p>It's in the <a href="https://docs.python.org/3/library/sys.html#sys.ps1" rel="noreferrer">sys</a> module (sys.ps1 &amp; sys.ps2):</p>
<blockquote>
<p>Strings specifying the primary and secondary prompt of the interpreter. These are only defined if the interpreter is in interactive mode. Their initial values in this case are '&gt;&gt;&gt; ' and '... '. If a non-string object is assigned to either variable, its str() is re-evaluated each time the interpreter prepares to read a new interactive command; this can be used to implement a dynamic prompt.</p>
</blockquote>
<p>For example:</p>
<pre>
   &gt;&gt;&gt; import sys
   &gt;&gt;&gt; sys.ps1 = "3.5&gt;&gt;&gt; "
   3.5&gt;&gt;&gt; sys.ps2 = "3.5... "
   3.5&gt;&gt;&gt;
</pre>
</div>
<div class="post-text" itemprop="text">
<p>It's great to set it to either:</p>
<ol>
<li>a <em>color</em> for better visual aspect</li>
<li>a <em>blank</em> or <em>space</em> for easier copy/paste operations</li>
</ol>
<p>Paste this into your bash shell: </p>
<pre><code>tee ~/.pyrc &lt;&lt;EOF
#!/usr/bin/env python3
import sys
sys.ps1='\x1b[1;49;33m&gt;&gt;&gt;\x1b[0m '  # bright yellow
sys.ps2='\x1b[1;49;31m...\x1b[0m '  # bright red
#sys.ps1='\x1b[33m&gt;&gt;&gt;\x1b[0m '      # dark yellow
#sys.ps2='\x1b[31m...\x1b[0m '      # dark red

# For easy copy/paste of proper code, use a blank or space:
#sys.ps1=' '
#sys.ps2=' '
EOF

# Then do this:
chmod 755 ~/.pyrc
</code></pre>
<p>Finally add this line to your <strong><code>~/.bash_profile</code></strong>:</p>
<p><code>export PYTHONSTARTUP=~/.pyrc</code></p>
<p>Enjoy!</p>
