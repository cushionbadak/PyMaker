<p>Does python support dynamic argument passing when I <strong>call a function</strong>?</p>
<pre><code>  import itertools.product

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow"><code>itertools.product</code></a> accepts an optional keyword argument <code>repeat</code>:</p>
<p>So, you can do:</p>
<pre><code>for n in itertools.product(l, repeat=i+1):
    ...
</code></pre>
<hr/>
<p>Alternatively, to dynamically pass argument, you can use <code>*args</code> (See <a href="https://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists" rel="nofollow">Unpacking argument lists</a>):</p>
<pre><code>for n in itertools.product(*([l] * (i+1))):
    ...
</code></pre>
