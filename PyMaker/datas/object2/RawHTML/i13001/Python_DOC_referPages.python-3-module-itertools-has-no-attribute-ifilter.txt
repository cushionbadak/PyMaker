<p><code>itertools.ifilter()</code> was removed in Python 3 because the built-in <a href="https://docs.python.org/3/library/functions.html#filter" rel="noreferrer"><code>filter()</code> function</a> provides the same functionality now.</p>
<p>If you need to write code that can run in both Python 2 and Python 3, use imports from the <a href="https://docs.python.org/2/library/future_builtins.html" rel="noreferrer"><code>future_builtins</code> module</a> (only in Python 2, so use a <code>try...except ImportError:</code> guard):</p>
<pre><code>try:
    # Python 2
    from future_builtins import filter
except ImportError:
    # Python 3
    pass

return filter(lambda i: i.state == "IS", self.storage)
</code></pre>
