<div class="post-text" itemprop="text">
<p>I have come across the following code.</p>
<pre><code>print ('%04.3f' % random.uniform(1, 100))
</code></pre>
<p>What does <code>'%04.3f'</code> represent in the code?</p>
</div>
<div class="post-text" itemprop="text">
<p>The format specifiers for most languages are derived from the <a class="post-tag" href="/questions/tagged/c" rel="tag" title="show questions tagged 'c'">c</a> language.</p>
<p>You can reference the full format <a href="http://www.cplusplus.com/reference/cstdio/printf/" rel="nofollow">list here</a>.</p>
<hr/>
<p>The format <code>%04.3f</code> specifies:</p>
<ul>
<li><code>%f</code> =&gt; float value precision</li>
<li><code>%.3f</code> =&gt; float value with 3 decimal precision</li>
<li><code>%04.3f</code> =&gt; float value, with 3 decimal precision and 4 digits to the left of decimal. The <code>04</code> implies that the number will be padded with <code>0</code>s if the value is less than <code>9999</code>.</li>
</ul>
