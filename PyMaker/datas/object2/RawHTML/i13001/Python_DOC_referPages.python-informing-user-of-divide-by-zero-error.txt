<div class="post-text" itemprop="text">
<p>You should use a <a href="https://docs.python.org/3/reference/compound_stmts.html#try" rel="nofollow">try/except</a>:</p>
<p>So, for your method here: </p>
<pre><code>def DivideTwoNumbers(num1,num2):
    totalDivide= num1 / num2
    return totalDivide
</code></pre>
<p>Simply, use a try/except around your <code>totalDivide = num1 / num2</code> to catch the <code>ZeroDivisionError</code> and then perform whatever operation you want in there. </p>
<p>This is an example: </p>
<pre><code>try:
    5/0
except ZeroDivisionError:
    print("You tried dividing by zero")
</code></pre>
