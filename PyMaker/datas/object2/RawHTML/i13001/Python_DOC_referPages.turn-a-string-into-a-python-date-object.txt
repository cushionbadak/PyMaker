<div class="post-text" itemprop="text">
<p>I have a String, <code>Sent: Fri Sep 18 00:30:12 2009</code> that I want to turn into a Python date object.</p>
<p>I know there's a <code>strptime()</code> function that can be used like so:</p>
<pre><code>&gt;&gt;&gt; dt_str = '9/24/2010 5:03:29 PM'
&gt;&gt;&gt; dt_obj = datetime.strptime(dt_str, '%m/%d/%Y %I:%M:%S %p')
&gt;&gt;&gt; dt_obj
datetime.datetime(2010, 9, 24, 17, 3, 29)
</code></pre>
<p>Can anybody think of an easier way to accomplish this than going through a bunch of conditionals to parse out if <code>Sep, month = 9</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>To parse <a href="https://tools.ietf.org/html/rfc5322#section-3.3" rel="nofollow noreferrer">rfc 822-like date-time string</a>, you could use <code>email</code> stdlib package:</p>
<pre><code>&gt;&gt;&gt; from email.utils import parsedate_to_datetime
&gt;&gt;&gt; parsedate_to_datetime('Fri Sep 18 00:30:12 2009')
datetime.datetime(2009, 9, 18, 0, 30, 12)
</code></pre>
<p>This is Python 3 code, see <a href="https://stackoverflow.com/a/23117071/4279">Python 2.6+ compatible code</a>.</p>
<p>You could also provide the explicit format string:</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; datetime.strptime('Fri Sep 18 00:30:12 2009', '%a %b %d %H:%M:%S %Y')
datetime.datetime(2009, 9, 18, 0, 30, 12)
</code></pre>
<p>See <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">the table with the format codes</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <code>python-dateutil</code> library!</p>
<p>First: <code>pip install python-dateutil</code> into your virtual-env if you have one then you can run the following code:</p>
<pre><code>from dateutil import parser

s = u'Sent: Fri Sep 18 00:30:12 2009'
date = parser.parse(s.split(':', 1)[-1])
</code></pre>
