<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow"><code>ast.literal_eval</code></a> to turn strings into other built-in data types:</p>
<pre><code>&gt;&gt;&gt; import ast
&gt;&gt;&gt; line = "[3, [9, [11, [4,[],[]], []], [] ], [7 , [], [6,[],[]]] ]"
&gt;&gt;&gt; myList = ast.literal_eval(line)
&gt;&gt;&gt; myList
[3, [9, [11, [4, [], []], []], []], [7, [], [6, [], []]]]

&gt;&gt;&gt; myList[0]
3
</code></pre>
