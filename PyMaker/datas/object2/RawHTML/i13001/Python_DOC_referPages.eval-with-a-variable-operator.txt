<div class="post-text" itemprop="text">
<p>Using <code>eval</code> is really bad practice, and should be avoided. For what you are trying to do, you should be making use of <a href="https://docs.python.org/3/library/operator.html" rel="nofollow">operator</a>. </p>
<p>Change your data structure to use a dictionary to make it easier on you to perform your operations. Something like this: </p>
<pre><code>import operator

operators = {
    "+": operator.add
} 

num1 = 4
num2 = 5

res = operators.get("+")(num1, num2)
</code></pre>
<p>Output of res:</p>
<pre><code>9
</code></pre>
<p>To apply your random implementation in to this, you make use of the dictionaries <code>keys()</code> to do a <code>random.choice</code> on that:</p>
<pre><code>random.choice(list(operators.keys()))
</code></pre>
<hr/>
<p>Simple example applying random: </p>
<pre><code>import operator
import random

operators = {
    "+": operator.add,
    "-": operator.sub,
    "*": operator.mul
}

num1 = 4
num2 = 5

res = operators.get(random.choice(list(operators.keys())))(num1, num2)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are mixing <code>int</code>, <code>num1</code> and <code>num2</code> and <code>str</code>, <code>currentop</code>. Cast them to <code>str</code> and it would work:</p>
<pre><code>if answer == eval(str(num1)+currentop+str(num2)):
</code></pre>
<p>PS: You should <a href="https://stackoverflow.com/a/1832957/742269">avoid</a> using <code>eval()</code>.</p>

<p>You need to convert it to a string, also "Incorrect" needs to be quoted:</p>
<pre><code>import random
operator=["+","-","*"]
num1=random.randint(0,10)
num2=random.randint(0,10)
currentop=random.choice(operator)

answer = input("What is " + str(num1) + str(currentop) + str(num2) + "?\n")
if answer== eval(str(num1)+str(currentop)+str(num2)):
    print("correct")
else:
    print("incorrect")
</code></pre>
<p>And as others have pointed out, unless for testing purposes, don't use eval.</p>
