<div class="post-text" itemprop="text">
<p>I would recommend <a href="https://docs.python.org/3/reference/import.html" rel="nofollow">import</a>. This is designed to use objects defined in one Python source file (module) in another.  </p>
<p>Create two files in the same directory:</p>
<p>First file <code>my_lib.py</code>:</p>
<pre><code>def my_func():
     return 42
</code></pre>
<p>Second file <code>my_script.py</code>:</p>
<pre><code>import my_lib

print(my_lib.my_func())
</code></pre>
<p>Now, <code>python my_script.py</code> in the same directory shows <code>42</code>.</p>
