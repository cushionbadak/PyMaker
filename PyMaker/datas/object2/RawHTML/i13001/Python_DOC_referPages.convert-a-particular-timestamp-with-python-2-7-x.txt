<div class="post-text" itemprop="text">
<p>I have a timestamp like:</p>
<p><code>2014-01-01T05:00:00.000Z</code></p>
<p>How do I convert this so that I can easily get the month like "January"? And in general convert it to a nice format like:</p>
<p><code>January 1st, 2014</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>datetime</code> module. <code>datetime.datetime</code> expects a time string and its formatting and returns a <code>datetime.datetime</code> object, on which you can call <code>strftime()</code> to <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow">format it according to your needs</a>.</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; my_date = datetime.datetime.strptime("2014-01-01T05:00:00.000Z", "%Y-%m-%dT%H:%M:%S.%fZ")
&gt;&gt;&gt; my_date.strftime('%d.%m.%Y')
01.01.2014
&gt;&gt;&gt; date.strftime('%H:%M:%S %d.%m.%Y')
'05:00:00 01.01.2014'
</code></pre>
<p>There is also a <a href="https://pypi.python.org/pypi/python-dateutil" rel="nofollow">python-dateutils</a> module, <a href="https://dateutil.readthedocs.org/en/latest/examples.html" rel="nofollow">which can do the same.</a></p>
</div>
<div class="post-text" itemprop="text">
<p>The strftime() method in datetime modulecan achieve this. It expects a string pattern explaining how you want to format your date.</p>
<pre><code>import datetime

today = datetime.date.today()
print today.strftime('It is %d %b %Y')
</code></pre>
<p>The above code prints something like "It is 12 Nov 2015"</p>
<p>You can find more format codes at <a href="https://docs.python.org/2/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow">https://docs.python.org/2/library/datetime.html#strftime-and-strptime-behavior</a></p>
</div>
