<pre><code>import glob

mydir = "C:\Data"

file_list = glob(mydir + "*.csv")
print('file_list {}'.format(file_list))
</code></pre>

<p>Looks like you just need to include your slash to search in the correct directory.</p>
<pre><code>import glob

mydir = "C:\Data"

file_list = glob.glob(mydir + "/*.csv") # Include slash or it will search in the wrong directory!!
print('file_list {}'.format(file_list))
</code></pre>

<div class="post-text" itemprop="text">
<p>Try <a href="https://docs.python.org/3/library/fnmatch.html#fnmatch.fnmatch" rel="nofollow"><code>fnmatch</code></a>:</p>
<pre><code>import os
from fnmatch import fnmatch

mydir = "C:/Data"
file_list = [file for file in os.listdir(mydir) if fnmatch(file, '*.csv')]

print('file_list {}'.format(file_list))
</code></pre>
<p>Also, use RegEx:</p>
<pre><code>import os
import re

mydir = "C:/Data"
file_list = [file for file in os.listdir(mydir) if re.search('.*\.png', file)]  

print('file_list {}'.format(file_list))
</code></pre>

<p>By the way, <a href="https://docs.python.org/3/library/glob.html" rel="nofollow"><code>glob</code></a> is a module, you should use <a href="https://docs.python.org/3/library/glob.html#glob.glob" rel="nofollow"><code>glob.glob()</code></a> like this:</p>
<pre><code>from glob import glob

mydir = "C:/Data"

file_list = glob(mydir + "/*.csv")
print('file_list {}'.format(file_list))
</code></pre>

<div class="post-text" itemprop="text">
<p>You are missing a backslash between the filename and the directory. BUT you can't end a string with a single backslash, because it will think you are trying to escape the endquote. You can put it in the call to glob instead. (Note, that you'll want to make both strings raw, it's good practice in case your directory starts with an "n" or a "t", which in this case is interpreted as spacing characters.):</p>
<pre><code>import glob
mydir = r"C:\Data"
file_list = glob.glob(mydir + r"\*.csv")
print('file_list {}'.format(file_list))
</code></pre>
<p>Also you might want to try using <a href="http://www.pylint.org/" rel="nofollow">pylint</a> to check for common errors or warnings. It would have thrown a warning for the unescaped directory name.</p>
<p>Update:</p>
<p>In fact I would just simplify it to this:</p>
<pre><code>import glob
file_list = glob.glob(r"C:\Data\*.csv")
print('file_list {}'.format(file_list))
</code></pre>
