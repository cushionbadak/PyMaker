<p>I would like to create an enum type at runtime by reading the values in a YAML file. So I have this:</p>
<pre><code># Fetch the values
v = {'foo':42, 'bar':24}

# Create the enum
e = type('Enum', (), v)
</code></pre>
<p>Is there a proper way to do it? I feel calling <code>type</code> is not a very neat solution. </p>

<div class="post-text" itemprop="text">
<p>You can create new enum type using <a href="https://docs.python.org/3/library/enum.html#functional-api" rel="nofollow noreferrer"><code>Enum</code> functional API</a>:</p>
<pre><code>In [1]: import enum

In [2]: DynamicEnum = enum.Enum('DynamicEnum', {'foo':42, 'bar':24})

In [3]: type(DynamicEnum)
Out[3]: enum.EnumMeta

In [4]: DynamicEnum.foo
Out[4]: &lt;DynamicEnum.foo: 42&gt;

In [5]: DynamicEnum.bar
Out[5]: &lt;DynamicEnum.bar: 24&gt;

In [6]: list(DynamicEnum)
Out[6]: [&lt;DynamicEnum.foo: 42&gt;, &lt;DynamicEnum.bar: 24&gt;]
</code></pre>
</div>
