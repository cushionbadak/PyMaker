<p>you can do this with Counter pretty easily:</p>
<pre><code>import string
from collections import Counter

# s = s.lower() # if you don't care about case
chars = set(string.ascii_letters + string.whitespace)
c = Counter(''.join(c for c in s if c in chars).split())
</code></pre>

<p>There are several things that may help. First, <a href="https://docs.python.org/2/library/collections.html#collections.Counter" rel="nofollow"><code>Counter</code></a>:</p>
<pre><code>s = '''Shank spare ribs ball tip, frankfurter alcatra rump ...'''
c = Counter(s.split())
</code></pre>
<p>Note not passing a parameter to <a href="https://docs.python.org/2/library/string.html#string.split" rel="nofollow"><code>str.split</code></a>:</p>
<blockquote>
<p>If the optional second argument sep is absent or None, the words are separated by arbitrary strings of whitespace characters (space, tab, newline, return, formfeed).</p>
</blockquote>
<p>Next, instead of <code>Counter</code>, you could use <a href="https://docs.python.org/2/library/string.html#string.count" rel="nofollow"><code>str.count</code></a> to count each word's occurrences:</p>
<pre><code>for word in s.split():
    print word, s.count(word)
</code></pre>
<p>Another option over <code>str.split</code> is a simple <a href="https://docs.python.org/2/library/re.html" rel="nofollow">regex</a>:</p>
<pre><code>for word in re.findall(r'[a-zA-Z-_]+', s):
    print word, s.count(word)
</code></pre>
<p>Finally, as part of your question exactly, you can use <a href="https://docs.python.org/2/library/string.html#string.rstrip" rel="nofollow"><code>str.rstrip</code></a> to strip characters from the end of strings:</p>
<pre><code>s = 'foobar.,'
t = s.rstrip('.,')
</code></pre>

<p>You can just split your string by the occurrences of non word character except those between expression like <code>-</code> in "T-bone"  using <a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow"><code>re.split</code></a></p>
<pre><code>import re
from collections import Counter

s = '''Shank spare ribs ball tip, frankfurter alcatra rump pancetta 
       picanha beef ribs biltong pig venison chicken ham hock. 
       T-bone beef ribs chicken pork jerky...
    '''

Counter(re.split(r'(?!&lt;\w)\W(?!\w)|\s+', s))
</code></pre>
<p><kbd><a href="https://regex101.com/r/yO5dL6/2" rel="nofollow">DEMO</a></kbd></p>
<p>But if what you want is remove only commas and period the following will work fine:</p>
<pre><code>Counter(re.split(r'[,\s.]', s))
</code></pre>
