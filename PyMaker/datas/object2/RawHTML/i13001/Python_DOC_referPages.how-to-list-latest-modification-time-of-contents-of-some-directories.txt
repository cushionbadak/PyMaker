<pre><code>People/
      Jimmy/
           my_Files/
                    work.txt
           report.txt
      Mohammad/
           the_work/
                    something2.sth
                    something2.sth
</code></pre>
<p>As you see, I have one directory for each clerk in the <em>People</em> directory and each clerk has a lot of files or directory in his/her directory. </p>
<p>I want to write a program to receive a date as input and search these directories and return which clerk hasn't modified any file after that date!</p>
<p>I'm using the following program to list all files:</p>
<pre><code>files_path= list()
for root,dirs,files in os.walk(r"G:\People"):
    for filename in files:
        files_path.append(root+filename)
</code></pre>
<p>and the following program to receive the last modification time of them:</p>
<pre><code>for single_file in files_path:
    print "last modified: %s" % time.ctime(os.path.getmtime(single_file))

</code></pre>

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/time.html?highlight=strptime#time.strftime" rel="nofollow">time.strftime</a> gives you control over the formatting:</p>
<pre><code>time.strftime('%H:%M:%S', time.gmtime(os.path.getmtime(single_file)))
</code></pre>
<p>A small helper function:</p>
<pre><code>def format_time(seconds):
    return time.strftime('%H:%M:%S', time.gmtime(seconds))
</code></pre>
<p>Makes it 
more approachable:</p>
<pre><code>print "last modified: %s, created: %s" %(format_time(os.path.getmtime(single_file)),
                                         format_time(os.path.getctime(single_file)))
</code></pre>
<p>Output:</p>
<pre><code>last modified: 13:05:56, created: 13:05:56
</code></pre>
