<div class="post-text" itemprop="text">
<p>There are multiple errors in your code. The invalid syntax error is quite easy to fix, just add the colon in the end of the if statement and the else does not take any conditions (elif, however would). This is very basic Python syntax. You can always check the official Python tutorials and documentations as a first step to troubleshoot syntax errors. e.g.:</p>
<p><a href="http://www.tutorialspoint.com/python/python_if_else.htm" rel="nofollow">http://www.tutorialspoint.com/python/python_if_else.htm</a></p>
<p>after you fix the SyntaxError you will run into other issues when comparing strings with integers, as in <code>hrs &lt;= 40</code>. Instead you want to compare the your converted input <code>h</code>. The same goes for the calculations: use <code>h</code> instead of <code>hrs</code> and <code>r</code> instead of rate.</p>
<p>See if you can solve all the issues. If you don't manage, here is a working example of the code: <a href="https://gist.github.com/fabianegli/bae9864e5166fac4dd2baeccd5ed3f8d" rel="nofollow">https://gist.github.com/fabianegli/bae9864e5166fac4dd2baeccd5ed3f8d</a></p>

<div class="post-text" itemprop="text">
<p>You are missing colons (<code>:</code>) after the conditions. Also, note that <code>else</code> doesn't take a condition, you need to use <code>elif</code>:</p>
<pre><code>if hrs &lt;= 40:
    # Here -^
    pay = hrs * rate
    print pay

elif hrs &gt; 40: # Note the elif
    # Here --^
    pay = hrs * 15.75
    print pay
</code></pre>
