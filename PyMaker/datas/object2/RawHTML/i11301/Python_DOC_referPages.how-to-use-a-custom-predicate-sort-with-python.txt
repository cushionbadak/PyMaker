<div class="post-text" itemprop="text">
<p>I have a list of lists like <code>[[1,2,"s"],[1,5,"e"],...]</code> where the 3rd value is always either <code>s</code> or <code>e</code>. How can I call <code>sort()</code> such that the list is sorted based on:</p>
<ol>
<li>the first index</li>
<li>if the first index is same, <code>s</code> comes first.</li>
</ol>
<p>Thanks</p>
</div>
<div class="post-text" itemprop="text">
<p>Write a custom <code>key</code> function.</p>
<pre><code>def sort_key(list_):
    if list_[2] == 's':
        return (list_[0], 0)
    return (list_[0], 1)

test_list = [[1, 2, 's'], [1, 5, 'e'], [2, 4, 'e'], [2, 3, 's']]
test_list.sort(key=sort_key)
# [[1, 2, 's'], [1, 5, 'e'], [2, 3, 's'], [2, 4, 'e']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can pass custom key function to <a href="https://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow"><code>list.sort</code></a> that is used to generate the comparison key for the items:</p>
<pre><code>&gt;&gt;&gt; l = [[1, 5, 'e'], [1, 2, 's'], [0, 4, 'e']]
&gt;&gt;&gt; l.sort(key=lambda x: (x[0], -ord(x[2])))
&gt;&gt;&gt; l
[[0, 4, 'e'], [1, 2, 's'], [1, 5, 'e']]
</code></pre>
