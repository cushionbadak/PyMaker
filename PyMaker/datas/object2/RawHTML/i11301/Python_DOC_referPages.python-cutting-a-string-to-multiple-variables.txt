<div class="post-text" itemprop="text">
<p>You can slice strings like this:</p>
<pre><code>&gt;&gt;&gt; 'very long string, much longer than 20 characters'[0:20]
'very long string, mu'
</code></pre>
<p>you don't need to check if the string is longer than 20 characters, you can just always add the slicing to your subprocess statement: <code>TextA[0:20]</code> will not touch the string if it's shorter than 20 characters. </p>

<div class="post-text" itemprop="text">
<p>If you don't need proper word wrapping, try a comprehension like this:</p>
<pre><code>from math import ceil
str = 'very long string, much longer than maxLength characters.'\
      'very long string, much longer than maxLength characters'

maxLength = 20
numStringsOut = int(ceil(len(str)/float(maxLength)))

subStringList = [str[p * maxLength: p * maxLength + maxLength]
                 for p in range(numStringsOut)]
print(subStringList)
</code></pre>
<p>output:</p>
<pre><code>['very long string, mu', 'ch longer than maxLe', 'ngth characters.very', ' long string, much l', 'onger than maxLength', ' characters']
</code></pre>
