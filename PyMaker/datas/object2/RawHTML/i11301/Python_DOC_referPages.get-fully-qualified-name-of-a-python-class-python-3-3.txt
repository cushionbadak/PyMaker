<p>How can I get name of class including full path from its module root? For Python 3.3 and up?</p>
<p>Here is example of Python code:</p>
<pre><code>class A:
    class B:
        class C:
            def me(self):
                print(self.__module__)
                print(type(self).__name__)
                print(repr(self))

x = A.B.C()
x.me()
</code></pre>
<p>This code outputs me on Python 3.3:</p>
<pre><code>__main__
C
&lt;__main__.A.B.C object at 0x0000000002A47278&gt;
</code></pre>
<p>So, Python internally knows that my object is <code>__main__.A.B.C</code>, but how can I get this programmatically? I can parse <code>repr(self)</code>, but it sounds like a hack for me.</p>
</div>
<div class="post-text" itemprop="text">
<p>You are looking for <a href="https://docs.python.org/3/library/stdtypes.html#class.__qualname__" rel="noreferrer"><code>__qualname__</code></a> (introduced in Python 3.3):</p>
<pre><code>class A:
    class B:
        class C:
            def me(self):
                print(self.__module__)
                print(type(self).__name__)
                print(type(self).__qualname__)
                print(repr(self))
</code></pre>
</div>
