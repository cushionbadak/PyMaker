<div class="post-text" itemprop="text">
<p>You can <code>assert list</code> to confirm list is not empty, or <code>assert not list</code> to confirm list is empty:</p>
<pre><code>&gt;&gt;&gt; assert not []
&gt;&gt;&gt; assert []
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
AssertionError
&gt;&gt;&gt; assert [1, 2, 3]
</code></pre>
<p>So in your case, you can just write down:</p>
<pre><code>assert not function_returns_list()
</code></pre>
<p>You can read more about <a href="https://docs.python.org/3/library/stdtypes.html#truth-value-testing" rel="noreferrer">Truth Value Testing</a> on python.org.</p>
</div>
