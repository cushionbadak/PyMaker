<div class="post-text" itemprop="text">
<p>I've got the following signal handler:</p>
<pre><code>def signal_handler(signal, frame):
    print('You pressed Ctrl+C!')
    sys.exit(0)
</code></pre>
<p>The signal handler registration is the following:</p>
<pre><code>signal.signal(signal.SIGINT, signal_handler)
</code></pre>
<p>How can I find out the affected process ID in the signal handler when the SIGINT happens?</p>
</div>
<div class="post-text" itemprop="text">
<p>I suggest using <code>os.getpid()</code>:</p>
<pre><code>import os, sys, signal

def signal_handler(signal, frame):
    pid = os.getpid()
    print('You pressed Ctrl+C (pid = {0})'.format(pid))
    sys.exit(0)

signal.signal(signal.SIGINT, signal_handler)
</code></pre>
<p>Reference: <a href="https://docs.python.org/3/library/os.html#os.getpid" rel="nofollow">https://docs.python.org/3/library/os.html#os.getpid</a></p>
</div>
