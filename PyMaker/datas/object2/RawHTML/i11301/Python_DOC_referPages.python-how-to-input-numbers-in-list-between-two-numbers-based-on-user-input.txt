<p>That is not the correct way to generate a list. If you want a list of numbers based on input you can use this:</p>
<pre><code>start = int(input("Enter starting number: "))
stop = int(input("Enter number to stop viewing at: "))
numbers = [i for i in range(start,stop+1)]
</code></pre>
<p>You might want to read up on <a href="http://www.tutorialspoint.com/python/python_lists.htm" rel="nofollow"><code>lists</code></a> and <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow"><code>range</code></a>.</p>
