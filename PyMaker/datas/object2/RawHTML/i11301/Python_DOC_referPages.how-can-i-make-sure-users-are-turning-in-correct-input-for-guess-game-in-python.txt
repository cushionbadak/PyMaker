<div class="post-text" itemprop="text">
<p>Attempt to parse entered value with <code>guess=int(guess)</code> can lead to different exceptions. Quick and dirty way to fix it - is to wrap it in <code>try ... except</code> block. Like:</p>
<pre><code>try:
    guess=int(guess)
except ValueError:
    print("Cannot understand your input. Pls, try again")
    continue
</code></pre>
<p>This code will catch all ValueError exceptions occured during parsing. You may find here more info on exceptions: <a href="https://docs.python.org/3/tutorial/errors.html#errors-and-exceptions" rel="nofollow">https://docs.python.org/3/tutorial/errors.html#errors-and-exceptions</a></p>
