<p>Given this list:</p>
<pre><code>&gt;&gt;&gt; a = "123DJY65TY"
&gt;&gt;&gt; list(a)
['1','2','3','D','J','Y','6','5','T','Y']
</code></pre>
<p>How can I produce a list where integers are not treated as strings? Like this:</p>
<pre><code>[1,2,3,'D','J','Y',6,5,'T','Y']   
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> and <a href="https://docs.python.org/3/library/stdtypes.html#str.isdigit" rel="nofollow"><code>str.isdigit</code></a> to convert each character that is a digit:</p>
<pre><code>&gt;&gt;&gt; a = "123DJY65TY"
&gt;&gt;&gt; [int(x) if x.isdigit() else x for x in a]
[1, 2, 3, 'D', 'J', 'Y', 6, 5, 'T', 'Y']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can convert all strings in a list containing only digits this way, using <a href="https://docs.python.org/2/library/functions.html#map" rel="nofollow"><code>map()</code></a> and <a href="https://docs.python.org/2/library/stdtypes.html#str.isdigit" rel="nofollow"><code>str.isdigit()</code></a>:</p>
<pre><code>a = map(lambda char: int(char) if char.isdigit() else char, list(a))
</code></pre>
<p>For example:</p>
<pre><code>In [3]: a = map(lambda char: int(char) if char.isdigit() else char, list(a))

In [4]: a
Out[4]: [1, 2, 3, 'D', 'J', 'Y', 6, 5, 'T', 'Y']
</code></pre>
<p><a href="https://stackoverflow.com/users/5043793/niemmi">@niemmi</a>'s solution using list comprehensions is probably a better approach given that we start with a string, not a list.</p>
