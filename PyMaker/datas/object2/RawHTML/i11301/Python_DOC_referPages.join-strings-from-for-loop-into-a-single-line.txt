<div class="post-text" itemprop="text">
<p>If you want not just print, but store the result in a string, you can use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join()</code></a> and a single <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>tags = [tag 
        for sentence in sent_text 
        for _, tag in nltk.pos_tag(nltk.word_tokenize(sentence))]
result = " ".join(tags)
print(result)
</code></pre>
<p>Note that the <code>_</code> is a common variable name for throwaway variables.</p>

<div class="post-text" itemprop="text">
<p>If you are using python 2.x
use <code>print(tag,)</code>, the <code>,</code> puts output in the same line. so you can use <code>print (tag + ' ',)</code></p>
<p>If you are using python 3
use <code>print(tag, end="")</code> or <code>print(tag, end=" ")</code> depending on whether you want whitespace or not.</p>
