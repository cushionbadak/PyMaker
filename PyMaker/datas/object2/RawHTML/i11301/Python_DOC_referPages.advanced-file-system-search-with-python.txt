<div class="post-text" itemprop="text">
<p>I'd like to create a Python script that will make some file searches recursively based on extension and file date. The most simple way (to me) to specify the requirements is this SQL like statement:</p>
<pre><code>select filedate, count(*) from 'c:\folder\*' 
where filename like '%.txt'
group by filedate
</code></pre>
<p>Any ideas on what is the simpliest solution?</p>
</div>
<div class="post-text" itemprop="text">
<p>You will have to piece a solution together from Python's various facilities.</p>
<p>I like to use <a href="https://docs.python.org/3/library/os.html#os.walk" rel="nofollow noreferrer"><code>os.walk</code></a> for traversing a file system recursively.</p>
<p><a href="https://docs.python.org/3/library/os.path.html#os.path.getctime" rel="nofollow noreferrer"><code>os.path.getctime()</code></a> can get the creation time (as epoch time) of a given file. You can use the time module to get the date from there. </p>
<p>I would use a <a href="https://docs.python.org/2/library/stdtypes.html#typesmapping" rel="nofollow noreferrer">dictionary</a> to link dates to lists of files.</p>
</div>
