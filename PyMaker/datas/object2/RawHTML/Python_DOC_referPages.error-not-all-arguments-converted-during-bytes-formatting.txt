<div class="post-text" itemprop="text">
<p>You are using the "?" ("qmark") style for parameter substitution characters, but MySQLdb expects the "%s" ("format") style.</p>
<p>There are a number of different ways that Python database-driver modules may use to format variables for inclusion in SQL statements.  The style a particular module accepts can be determined by inspecting the module's <code>paramstyle</code> attribute:</p>
<pre><code>&gt;&gt;&gt; import MySQLdb
&gt;&gt;&gt; MySQLdb.paramstyle
'format'
</code></pre>
<p>The full list of possible formats is <a href="https://www.python.org/dev/peps/pep-0249/#paramstyle" rel="nofollow noreferrer">here</a>.</p>
<p>You need to replace each question mark in your INSERT statement's VALUE clause with "%s":</p>
<pre><code>cursor.execute(
    """INSERT INTO studentdetails (First_Name,Last_Name,Roll_No,Phone_No,Email_ID) """
    """VALUES(%s,%s,%s,%s,%s)""",
    (fname,lname,rollno,phonenumber,emailid))
</code></pre>
<p>(I've broken the statement into two strings to make it more readable here on Stack Overflow - you don't need to do this in your own code).</p>
