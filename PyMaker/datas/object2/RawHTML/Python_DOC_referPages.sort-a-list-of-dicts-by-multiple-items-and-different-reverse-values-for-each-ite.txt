<p>I have the following list of objects and I want to sort them such that first all dicts with <code>k2 == True</code> (alphabetically ordered A-Z) and afterwards all dicts with <code>k2 == False</code> (alphabetically ordered A-Z). 
I tried something like this <code>sorted(test, key=lambda k: (k['k2'], k['k1'].lower()), reverse=(True,False))</code> but it doesn't work.</p>
<pre><code>&gt;&gt;&gt; test = [
...     {"k1": "qsd", "k2": True},
...     {"k1": "JKd", "k2": False},
...     {"k1": "Ukz", "k2": False},
...     {"k1": "aqd", "k2": True},
...     {"k1": "Asd", "k2": True},
...     {"k1": "wef", "k2": False},
...     {"k1": "Wgr", "k2": True},
...     {"k1": "weg", "k2": False},
...     {"k1": "lfe", "k2": True},
... ]
&gt;&gt;&gt;
&gt;&gt;&gt; test = sorted(test, key=lambda k: (k['k2'], k['k1'].lower()), reverse=True)
&gt;&gt;&gt; for t in test:
...     print(t)
...
{'k1': 'Wgr', 'k2': True}
{'k1': 'qsd', 'k2': True}
{'k1': 'lfe', 'k2': True}
{'k1': 'Asd', 'k2': True}
{'k1': 'aqd', 'k2': True}
{'k1': 'weg', 'k2': False}
{'k1': 'wef', 'k2': False}
{'k1': 'Ukz', 'k2': False}
{'k1': 'JKd', 'k2': False}
</code></pre>
<p>I'm looking for:</p>
<pre><code>{'k1': 'aqd', 'k2': True}
{'k1': 'Asd', 'k2': True}
{'k1': 'lfe', 'k2': True}
{'k1': 'qsd', 'k2': True}
{'k1': 'Wgr', 'k2': True}
{'k1': 'JKd', 'k2': False}
{'k1': 'Ukz', 'k2': False}
{'k1': 'wef', 'k2': False}
{'k1': 'weg', 'k2': False}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You're close, use <code>not k['k2']</code> for the first sorter.</p>
<pre><code>&gt;&gt;&gt; test.sort(key=lambda k: (not k['k2'], k['k1'].lower()))
&gt;&gt;&gt; test
[{'k1': 'aqd', 'k2': True},
 {'k1': 'Asd', 'k2': True},
 {'k1': 'lfe', 'k2': True},
 {'k1': 'qsd', 'k2': True},
 {'k1': 'Wgr', 'k2': True},
 {'k1': 'JKd', 'k2': False},
 {'k1': 'Ukz', 'k2': False},
 {'k1': 'wef', 'k2': False},
 {'k1': 'weg', 'k2': False}]
</code></pre>
<p><code>not k['k2']</code> is <code>False</code> when <code>k['k2']</code> is True, and False (=0) &lt; True (=1).</p>
