<div class="post-text" itemprop="text">
<p>Is there a way to make <a href="https://docs.python.org/3/library/unittest.html#unittest.TextTestRunner" rel="nofollow noreferrer">unittest.TextTestRunner</a> completely quiet, meaning it never prints to output on its own? Even at <code>verbosity=0</code> it prints results when done.</p>
<p>I want to process the TestResult object returned by the runner before anything is printed.</p>
</div>
<div class="post-text" itemprop="text">
<p>TextTestRunner has a <code>stream=sys.stderr</code> in its constructor:</p>
<pre><code>def __init__(self, stream=sys.stderr, descriptions=1, verbosity=1)
</code></pre>
<p>Change it to a null stream. </p>
<pre><code>result = unittest.TextTestRunner(stream = open(os.devnull, 'w')).run(alltests)
if len(result.failures) or len(result.errors):
    print "Sorry."
</code></pre>
</div>
