<p>You can override get_method with something like this:</p>
<pre><code>def _make_request(url, data, method):
    request.urllib2.Request(url, data=data)
    request.get_method = lambda: method
</code></pre>
<p>Then you pass "DELETE" as method.</p>
<p><a href="https://stackoverflow.com/questions/4511598/how-to-make-http-delete-method-using-urllib2">This</a> answer covers the details.</p>

<div class="post-text" itemprop="text">
<p>As far as I know, urllib and urllib2 only support <code>GET</code> and <code>POST</code> requests. You should probably take a look at <a href="http://docs.python.org/library/httplib.html" rel="nofollow noreferrer">httplib</a> or <a href="http://code.google.com/p/httplib2/" rel="nofollow noreferrer">httplib2</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>The method is set implicitly in the <a href="http://docs.python.org/py3k/library/urllib.request.html" rel="nofollow noreferrer">urlopen</a> call</p>
<p>When you provide the data parameter a POST will be used.</p>
<pre><code>urllib.request.urlopen(url, data=None[, timeout])
</code></pre>
<p>I don't think it's possible to use a DELETE HTTP method with urlib because of this line:</p>
<blockquote>
<p><strong>Request.get_method()</strong><br/>
  Return a string
  indicating the HTTP request method.
  This is only meaningful for HTTP
  requests, and currently always returns
  'GET' or 'POST'. </p>
</blockquote>
<p>Consider using <a href="http://docs.python.org/library/httplib.html" rel="nofollow noreferrer">httplib</a>, <a href="http://code.google.com/p/httplib2/" rel="nofollow noreferrer">httplib2</a>, or <a href="http://twistedmatrix.com/trac/" rel="nofollow noreferrer">Twisted</a> instead .for better support of HTTP methods.</p>
</div>
<div class="post-text" itemprop="text">
<p>The default HTTP methods in urllib library are POST and GET:</p>
<pre><code>def get_method(self):
    """Return a string indicating the HTTP request method."""
    default_method = "POST" if self.data is not None else "GET"
    return getattr(self, 'method', default_method)
</code></pre>
<p>But we can override this get_method() function to get DELETE request:</p>
<pre><code>req = urllib.request.Request(new_url)
req.get_method = lambda: "DELETE"
</code></pre>
