<div class="post-text" itemprop="text">
<p>You should not manipulate paths directly, there is os.path module for that.</p>
<pre><code>&gt;&gt;&gt; import os.path
&gt;&gt;&gt; print os.path.dirname("C:\Users\myFile.txt")
C:\Users
&gt;&gt;&gt; print os.path.dirname(os.path.dirname("C:\Users\myFile.txt"))
C:\
</code></pre>
<p>Like this.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can also use <a href="https://docs.python.org/3/library/os.path.html#os.path.split" rel="noreferrer"><code>os.path.split</code></a>, like this</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; os.path.split('product/bin/client')
('product/bin', 'client')
</code></pre>
<p>It splits the path into two parts and returns them in a tuple. You can assign the values in variables and then use them, like this</p>
<pre><code>&gt;&gt;&gt; head, tail = os.path.split('product/bin/client')
&gt;&gt;&gt; head
'product/bin'
&gt;&gt;&gt; tail
'client'
</code></pre>
