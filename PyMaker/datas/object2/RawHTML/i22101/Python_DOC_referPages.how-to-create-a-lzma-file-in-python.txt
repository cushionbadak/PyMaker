<div class="post-text" itemprop="text">
<p>Use <a href="http://www.joachim-bauch.de/projects/pylzma/" rel="noreferrer">PyLZMA</a>.</p>
<p>1st result for Google: <code>python lzma</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Well, as of Python 3.3, there is an <a href="http://docs.python.org/3/library/lzma">lzma</a> module in the standard library.</p>
<p>For earlier Pythons, <a href="https://pypi.python.org/pypi/pylzma">PyLZMA</a> (as mentioned by @mcandre) is probably still your best bet.</p>
</div>
<span class="comment-copy">LZMA is a compression method, not a file archive format, so one cannot add files to a .lzma file. A typical example is that one creates a .tar archive of the input files and then compresses the archive using the LZMA method.</span>
