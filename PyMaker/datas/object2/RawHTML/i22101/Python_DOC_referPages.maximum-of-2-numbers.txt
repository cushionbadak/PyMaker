<p>How to find the maximum of 2 numbers?</p>
<pre><code>value = -9999
run = problem.getscore()
</code></pre>
<p>I need to compare the 2 values i.e <code>value</code> and <code>run</code> and find the maximum of 2. I need some python function to operate it?</p>

<div class="post-text" itemprop="text">
<p>Use the builtin function <a href="https://docs.python.org/3/library/functions.html#max" rel="noreferrer"><code>max</code></a>.</p>
<p>Example: 
<code>max(2, 4)</code> returns 4.</p>
<p>Just for giggles, there's a <code>min</code> as well...should you need it. :P</p>
</div>
<div class="post-text" itemprop="text">

</div>

<div class="post-text" itemprop="text">
<p>You can use <code>max(value, run)</code></p>
<p>The function <a href="http://docs.python.org/library/functions.html#max" rel="noreferrer"><code>max</code></a> takes any number of arguments, or (alternatively) an iterable, and returns the maximum value.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>max(value,run)
</code></pre>
<p>should do it.</p>
</div>
<div class="post-text" itemprop="text">
<p>Just for the fun of it, after the party has finished and the horse bolted.</p>
<p>The answer is: <code>max()</code> !</p>
</div>
<div class="post-text" itemprop="text">
<p>You could also achieve the same result by using a <a href="https://docs.python.org/3/reference/expressions.html#conditional-expressions" rel="nofollow noreferrer">Conditional Expression</a>:</p>
<pre><code>maxnum = run if run &gt; value else value
</code></pre>
<p>a bit more flexible than <code>max</code> but admittedly longer to type.</p>
</div>
