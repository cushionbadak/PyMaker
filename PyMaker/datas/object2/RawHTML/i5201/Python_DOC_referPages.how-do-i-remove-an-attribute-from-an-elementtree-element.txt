<div class="post-text" itemprop="text">
<p>I have a simple XML object <code>&lt;person&gt;</code> created with <a href="https://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow noreferrer"><code>xml.etree.ElementTree</code></a>.</p>
<pre><code>&lt;person name='John' age='21' /&gt;
</code></pre>
<p>I want to modify the XML element to remove the <code>age</code> attribute.</p>
<pre><code>&lt;person name='John' /&gt;
</code></pre>
<p>I can access the attribute using <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.get" rel="nofollow noreferrer"><code>.get("age")</code></a>, but using <a href="https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.remove" rel="nofollow noreferrer"><code>.remove("age")</code></a> results in a TypeError.</p>
<pre><code>import xml.etree.ElementTree as ElementTree

xml = ElementTree.Element('person', name="john", age="21")
xml.remove("age") 
# TypeError: remove() argument must be xml.etree.ElementTree.Element, not str
</code></pre>
<p><a href="https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.remove" rel="nofollow noreferrer">According to the documentation</a>, <code>.remove()</code> can only be used to remove subelements. There doesn't seem to be any alternative option for removing attributes.</p>
<p><strong>How can I remove an XML attribute from an <code>xml.etree.ElementTree</code> object?</strong></p>
</div>
<div class="post-text" itemprop="text">
<p>You can try <code>del xml.attrib["age"]</code></p>
<pre><code>import xml.etree.ElementTree as ElementTree

xml = ElementTree.Element('person', Name="john", age="21")
print(xml.items())
del xml.attrib["age"]
print(xml.items())
</code></pre>
<p>Will produce:</p>
<pre><code>[('Name', 'john'), ('age', '21')]
[('Name', 'john')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>xml.attrib.pop("age")</code>.</p>
<pre><code>from xml.etree import ElementTree

xml = ElementTree.Element('person', name="john", age="21")
print(ElementTree.tostring(xml).decode())

xml.attrib.pop("age")
print(ElementTree.tostring(xml).decode())
</code></pre>
<p>Output:</p>
<pre><code>&lt;person name='John' age='21' /&gt;
&lt;person name='John' /&gt;
</code></pre>
<p>Explanation:</p>
<p>The <code>attrib</code> member of <code>ElementTree</code> is implemented as a <a href="https://docs.python.org/3/library/stdtypes.html#mapping-types-dict" rel="nofollow noreferrer"><code>dict</code></a> object, meaning you can use any standard dictionary methods on it.</p>
</div>
