<p>I am opening a file in my python code. For a particular user, the path is /home/user_name/IB/MAIN
where user_name is the personal user name. For testing and stuff I have used my own user name but if I give this code to someone else, it won't run as the path is different for that user, so how do I get a  generic path for this file?</p>

<div class="post-text" itemprop="text">
<p>Use </p>
<pre><code>'~/IB/MAIN'
</code></pre>
<p>to denote relative path to user home.</p>
<p>or </p>
<pre><code>import os
homepath = os.getenv("HOME")
path = os.path.join(homepath, "IB/MAIN")
</code></pre>
<p>via environment variable </p>
