<code>python test.py --layer 3 --sizes 100,100,100 </code></p>

<div class="post-text" itemprop="text">
<p>Use argument parser from python : <a href="https://docs.python.org/3/library/argparse.html" rel="nofollow noreferrer">https://docs.python.org/3/library/argparse.html</a></p>
<p>You have examples in the doc.</p>
<p>Hint for your sizes:
You can store it as a string and split it with ',' as a separator
If sizes_string is "100,100,100", you do:</p>
<pre><code>sizes = [int(s) for s in sizes_string.split(',')]
</code></pre>

<span class="comment-copy">You can use an assert that will stop your program if the condition is not satisfied. <code>assert(layer == len(sizes))</code>.</span>
<span class="comment-copy">Test the <code>len(sizes)</code> and complain to your user.  <code>parse.errror</code> is convenient for issuing custom messages.</span>
