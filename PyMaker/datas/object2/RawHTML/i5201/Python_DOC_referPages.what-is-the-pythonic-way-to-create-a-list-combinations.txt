<p>Let's assume I have two lists:</p>
<pre><code>list_one = [1A, 1B]
list_two = [2A]
</code></pre>
<p>How can I get a list containing pairs of words from both these arrays? To visualize:</p>
<pre><code>answer = [1A-2A, 1B-2A]
</code></pre>
<p>I know I can do it using two <code>for</code> loops but is there a 'pythonic' way?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can simply use <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="noreferrer"><code>product</code> from <code>itertools</code></a>.</p>
<pre><code>from itertools import product

list_one = ['1A', '1B']
list_two = ['2A']

result = ['-'.join(sub) for sub in product(list_one, list_two)]
print(result)  # -&gt; ['1A-2A', '1B-2A']
</code></pre>
<p>I do not think it gets any more <em>pythonic</em> than this.</p>
<hr/>
<p>Under the hood, <code>product(list_one, list_two)</code> generates tuples of the form <code>('1A', '2A')</code> which we simply <code>'-'.join()</code> to produce the desired result.</p>
