<pre><code>def imput(filename):
    with open(filename, 'r') as f: #open the file
        for line in f:
            input('sample.txt')`
</code></pre>
<p>The file 'sample.txt' consists of the numbers <code>1,2,4,6,8</code> and when I use the print function I get: </p>
<pre><code>print(line)

1

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer"><code>sum()</code></a> to total number like:</p>
<h3>Code:</h3>
<pre><code>total = sum(int(line) for line in open('file1', 'rU'))
print(total)
</code></pre>
<p>And if you like the terseness of <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map()</code></a> that can be even shorter as:</p>
<pre><code>total = sum(map(int, open('file1', 'rU')))
</code></pre>
<h3>Results:</h3>
<pre><code>21
</code></pre>
