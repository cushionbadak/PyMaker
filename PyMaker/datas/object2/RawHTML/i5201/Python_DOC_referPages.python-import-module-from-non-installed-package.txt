<p>For my project i want to use external libraries, connected to git repository as submodules. They are located in <code>shared-lib</code> folder, that is added to <code>sys.path</code>.
All libraries contains <code>setup.py</code> file and subfolder with python module. How can i import that module without installing whole package and without changing it structure?</p>
<p><code>import module_name.module_name</code> does not work, i get <code>No module named module_name.module_name</code> because there is no <code>__init__.py</code> file inside module_name directory.</p>
<p>My project does not require installation, because it is plugin for another system. So i can use only python import.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/imp.html" rel="nofollow noreferrer"><code>imp</code></a>.</p>
<pre><code>import imp

foo = imp.load_source('module1.foo', 'module1/module1/foo.py')
bar = imp.load_source('module2.bar', 'module2/module2/bar.py')

print(foo)
print(bar)
</code></pre>
<p>Output:</p>
<pre><code>&lt;module 'module1.foo' from 'module1/module1/foo.py'&gt;
&lt;module 'module2.bar' from 'module2/module2/bar.py'&gt;
</code></pre>
</div>
