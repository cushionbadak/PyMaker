<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#dict.fromkeys" rel="nofollow noreferrer"><code>dict.fromkeys</code></a> if you don't mind your dictionary pointing to the same <code>list</code>.</p>
<pre><code>temp_dic = dict.fromkeys(other_concords, leamanyi_concords)

# {'a': ['fruit', 'drink', 'snack'],
#  'b': ['fruit', 'drink', 'snack'],
#  'c': ['fruit', 'drink', 'snack']}
</code></pre>
</div>
