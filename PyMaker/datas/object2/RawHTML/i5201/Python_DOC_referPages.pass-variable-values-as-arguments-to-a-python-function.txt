<div class="post-text" itemprop="text">
<p>You are not extracting the arguments passed to your method properly. In Python, you simply just use the arguments that you defined in your method like this: </p>
<pre><code>def createCluster(CName, No01, No02):
    print(CName)
    print(No01)
    print(No02)

createCluster('a', 'b', 'c')
</code></pre>
<p>The above will output </p>
<pre><code>a
b
c
</code></pre>
<p>For your issue on your usage of <code>sys.argv</code> that you are incorrectly using:</p>
<p>Snippet from the documentation on <a href="https://docs.python.org/3/library/sys.html#sys.argv" rel="nofollow noreferrer">sys.argv</a>:</p>
<blockquote>
<p>The list of command line arguments passed to a Python script.</p>
</blockquote>
<p>In example of this would be when calling your code from a shell prompt:</p>
<blockquote>
<p>python your_script.py a b c</p>
</blockquote>
<p>Inside <code>your_script.py</code> you will ahve</p>
<pre><code>import sys

script_name = sys.argve[0]
arg1 = sys.argv[1]
arg2 = sys.argv[2]
arg3 = sys.argv[3]
</code></pre>
<p>Which will output: </p>
<pre><code>/absolute/path/to/your_script.py
a
b
c
</code></pre>
<p>So, you see here that it ends up giving you the name of the script and all arguments. </p>
<p>This is maybe where your confusion came from between calling a script and calling the method in the script. </p>
