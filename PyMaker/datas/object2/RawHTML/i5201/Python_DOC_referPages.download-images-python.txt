<div class="post-text" itemprop="text">
<p>You need the folder "Images" created before executing the code.
Your code won't create the folder if it doesn't exists, that's the problem.</p>
<p>With the <a href="https://docs.python.org/3/library/os.html" rel="nofollow noreferrer"><code>os</code> module</a> you can check and create/remove folders (if you want to do it at run-time)</p>
