<p>What would be the equivalent of <code>time.ctime()</code> for UTC time?</p>
<p>Currently, if I enter <code>time.ctime()</code> in the interpreter, I get this:</p>
<pre><code>'Mon Feb  5 17:24:48 2018'
</code></pre>
<p>This returns the local time and I would like to know how to get time in the same format except in UTC. </p>
<p>Edit: It's <code>time.asctime(time.gmtime())</code></p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/time.html#time.gmtime" rel="nofollow noreferrer"><code>time.gmtime()</code></a>
returns an object representing the utctime.</p>
<p>To get it into the same format as <code>time.ctime</code>, you can use <a href="https://docs.python.org/3/library/time.html#time.asctime" rel="nofollow noreferrer"><code>time.asctime()</code></a>.</p>
<pre><code>&gt;&gt;&gt; time.asctime(time.gmtime())
'Mon Feb  5 12:03:39 2018'
</code></pre>
</div>

<pre><code>import datetime

utc = datetime.datetime.utcnow()
# datetime.datetime(2018, 2, 5, 12, 2, 56, 678723)

str(utc)
# '2018-02-05 12:05:10.617973'
</code></pre>
