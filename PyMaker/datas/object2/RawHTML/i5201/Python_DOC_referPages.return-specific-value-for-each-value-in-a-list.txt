<p>I want to return a specific value for each value in a list. </p>
<p><code>list = ['sathya','varna','summi']</code></p>
<p>I want to return values such that if my key is 'sathya' return -5, if key is 'varna' return -4.</p>
<p>As of now my code is:</p>
<pre><code>def return_value():
   if key == 'sathya':
         return -5
   if key == 'varna':
         return -4
</code></pre>
<p>But I don't want to use so many if conditions. can anyone help me in this</p>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a> if the return value is their index.</p>
<pre><code>your_list = ['sathya','varna','summi']
</code></pre>
<p><em>Method 1</em></p>
<pre><code>def return_value(key):
    for index, key_ in enumerate(your_list, start=-5):
        if key == key_:
            return index
    return "Not found"

print(return_value('sathya')) # return -5
print(return_value('invalid item')) # return not found
</code></pre>
<p><em>Method 2</em>(Using Dict Comprehension)</p>
<pre><code>def return_value(key):
    return {
            value: index 
            for index, value in enumerate(your_list ,start=-5)
           }.get(key, "Not found")

print(return_value('sathya')) # return -5
print(return_value('invalid item')) # return not found
</code></pre>
</div>

<p>Use a dictionary to store the values and access by key.</p>
<pre><code>def return_value(key):
    return {'sathya': -5,'varna': -4,'summi': 'any value..'}[key]
</code></pre>

<div class="post-text" itemprop="text">
<p>The basic python data structure you are looking for is a dictionary, not a list.
Dictionaries allow key-value pairing in a native way.
Example.</p>
<pre><code>d={akey:aval}
d['mykey']=myval
</code></pre>
<p>then you can retrieve you values</p>
<pre><code>print(d['mykey'])
</code></pre>
