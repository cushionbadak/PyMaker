<div class="post-text" itemprop="text">
<p>You can try a simple workaround of storing microseconds separately then <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.replace" rel="nofollow noreferrer"><code>replace</code></a> it into <code>datetime</code> object</p>
<pre><code>&gt;&gt;&gt; timestamp = 15179052980000000
&gt;&gt;&gt; m_seconds = 38002
&gt;&gt;&gt; dt = datetime.fromtimestamp(timestamp/10000000)
&gt;&gt;&gt; dt
&gt;&gt;&gt; datetime.datetime(2018, 2, 6, 16, 21, 38)
</code></pre>
<p>Then finally replace the <code>dt</code> with desired microseconds</p>
<pre><code>&gt;&gt;&gt; dt.replace(micorsecond=m_seconds)
&gt;&gt;&gt; dt
&gt;&gt;&gt; datetime.datetime(2018, 2, 6, 16, 21, 38, 38002)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You would need nanosecond precision to encode that value - note that you don't have .038002, you have 0.0380002 (extra digit).</p>
<p>This is 38000us (rounded), or 38000200ns.</p>
