<div class="post-text" itemprop="text">
<p>In Python 2.6 or above, you can use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>string.format</code></a>:</p>
<pre><code>print('"{}" does not even sound like a word anymore.'.format(favoriteword))
</code></pre>
<p>In lower versions, Ketzak's method will work.</p>
<hr/>
<p>To print multiple times on a single line, you want to prevent <code>print</code> from appending a newline.</p>
<p>In Python 3, use the <code>end</code> argument:</p>
<pre><code>for i in range(12):
    print(favoriteword, end='')
print('')  # for newline
</code></pre>
<p>or in lower versions:</p>
<pre><code>import sys

for i in range(12):
    sys.stdout.write(favoriteword)
print('')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use the Python interpolation operator if you can guarantee <code>favoriteword</code> will always be a string:</p>
<p><code>print('"%s" does not even sound like a word anymore.' % favoriteword)</code></p>
</div>
<div class="post-text" itemprop="text">
<p>In python &gt;= 3.6, you can use "f-string":</p>
<p><code>print(f'"{favoriteword}" does not even sound like a word anymore.')</code></p>
<p>See <a href="https://docs.python.org/3/whatsnew/3.6.html#pep-498-formatted-string-literals" rel="nofollow noreferrer">this link</a> for more information about it.</p>
