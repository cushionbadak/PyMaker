<p>Although there are many cases where it is a good idea to use <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer"><code>set()</code></a> it is for use in lists.  </p>
<p>In the example you gave above it seems you are not actually using lists so you don't need the <code>set()</code>.  Here it is rewritten without it:</p>
<pre><code>def functionX():
    fce = int(input('Enter number: '))
    if fce&lt;=5:
        print('equal or lower than 5')
    else:
        print('higher than 5')
functionX()
</code></pre>
