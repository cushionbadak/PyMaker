<div class="post-text" itemprop="text">
<p>In Python 3, <code>print(i, j)</code> and <code>print(i)</code> does <strong>not</strong> return the same result. </p>
<p><code>print(i, j)</code> prints the key <code>i</code> followed by it's value <code>j</code></p>
<p><code>print(i)</code> prints a tuple containing the dictionary key followed by its value</p>
</div>
<div class="post-text" itemprop="text">
<p><code>items()</code> returns a view object which allows you to iterate over the <code>(key, value)</code> tuples. So basically you can just manipulate them as what you do with tuples. The <a href="https://docs.python.org/3/library/stdtypes.html?highlight=items#dictionary-view-objects" rel="nofollow noreferrer">document</a> may help:</p>
<blockquote>
<p>iter(dictview)
  Return an iterator over the keys, values or items (represented as tuples &gt; of (key, value)) in the dictionary.</p>
</blockquote>
<p>Also I think <code>print(i, j)[1]</code> would result in error in Python 3 since <code>print(i, j)</code> returns <code>None</code>.</p>
</div>
