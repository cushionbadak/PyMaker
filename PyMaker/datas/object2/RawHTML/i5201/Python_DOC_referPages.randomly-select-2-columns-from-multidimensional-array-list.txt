<p>We can use <a href="https://docs.python.org/3/library/itertools.html#itertools.combinations" rel="nofollow noreferrer"><code>itertools.combinations</code></a> to get all the combinations of a certain size.</p>
<pre><code>from itertools import combinations

def transpose(mat):
    return list(zip(*mat))

test_list = [[1, 2, 3, 4, 5],
             [6, 7, 8, 9, 0]]

pairs = (transpose(x) for x in combinations(transpose(test_list), 2))

for pair in pairs:
    print(pair)
</code></pre>
<p>gives us</p>
<pre><code>[(1, 2), (6, 7)]
[(1, 3), (6, 8)]
[(1, 4), (6, 9)]
[(1, 5), (6, 0)]
[(2, 3), (7, 8)]
[(2, 4), (7, 9)]
[(2, 5), (7, 0)]
[(3, 4), (8, 9)]
[(3, 5), (8, 0)]
[(4, 5), (9, 0)]
</code></pre>
