<p>I wrote a tkinter code, with a label. There was a variable in the label text.</p>
<pre><code>lab = Label(root, text = 'randomstrings', x, 'randomstrings')
    lab.pack()
</code></pre>
<p>When I ran the code, it gave an error message here: , x,</p>
<p>It said: positional argument follows keyword argument</p>
</div>
<div class="post-text" itemprop="text">
<p>You must put all label pieces together before passing them to the <code>Label</code>:</p>
<pre><code>Label(root, text = 'randomstrings' + str(x) + 'randomstrings', ...)
</code></pre>
<p>or:</p>
<pre><code>Label(root, text = 'randomstrings{}randomstring'.format(x), ...)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using commas to stitch strings together only works in the <code>print</code> function. Everywhere else you need to do the string formatting yourself: </p>
<pre><code>lab = Label(root, text = 'randomstrings {} randomstrings'.format(x))
lab.pack()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<h2>Your problem</h2>
<p>Commas are used to separate <code>Label()</code>'s arguments, so <code>x</code> and <code>'randomstrings'</code> are interpreted as <code>Label()</code> arguments. This is not what you want.</p>
<h2>Solution</h2>
<p>You need to concatenate your strings.</p>
<p>You can use the <code>+</code> operator to do so:</p>
<pre><code>lab = Label(root, text = 'randomstrings ' + x + ' randomstrings')
lab.pack()
</code></pre>
<p>If <code>x</code> is not a string, you can convert it with <a href="https://docs.python.org/3/library/functions.html#func-str" rel="nofollow noreferrer"><code>str()</code></a>, like this:</p>
<pre><code>lab = Label(root, text = 'randomstrings ' + str(x) + ' randomstrings')
lab.pack()
</code></pre>
</div>
