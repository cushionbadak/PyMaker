<div class="post-text" itemprop="text">
<p>What does the following mean in python? </p>
<pre><code>var1 = [var2]
</code></pre>
<p>example: </p>
<pre><code>self.locations = [sys.resources_location]
</code></pre>
<p>is var1 being declared as an array? </p>
</div>
<div class="post-text" itemprop="text">
<p><strong>var1</strong> is initialized as <em>list</em> with one element which is <strong>var2</strong>.</p>
<pre><code>&gt;&gt;&gt; var2 = 1
&gt;&gt;&gt; var1 = [var2]
&gt;&gt;&gt; type(var1)
&lt;type 'list'&gt;
</code></pre>
<p>You could learn more about list from this tutorial.</p>
<p><a href="https://www.digitalocean.com/community/tutorials/understanding-lists-in-python-3" rel="nofollow noreferrer">Understanding Lists in Python 3</a></p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/dev/reference/expressions.html#list-displays" rel="nofollow noreferrer">The syntax <code>[x]</code></a> creates a new list object with one element <em>x</em>.</p>
<p>You can try with the interactive shell, and use <a href="https://docs.python.org/dev/library/functions.html#type" rel="nofollow noreferrer"><code>type</code></a> to find out more about the object:</p>
<pre><code>&gt;&gt;&gt; var2 = 42
&gt;&gt;&gt; var1 = [var2]
&gt;&gt;&gt; var1
[42]
&gt;&gt;&gt; type(var1)
&lt;type 'list'&gt;
</code></pre>
<p><a href="https://docs.python.org/dev/library/stdtypes.html#list" rel="nofollow noreferrer">Lists</a> are not <a href="https://docs.python.org/dev/library/array.html" rel="nofollow noreferrer">arrays</a>; lists can be resized and contain elements of different types.</p>
</div>
