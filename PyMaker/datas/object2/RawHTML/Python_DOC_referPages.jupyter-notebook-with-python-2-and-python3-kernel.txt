<div class="post-text" itemprop="text">
<ol>
<li>Make sure you have pip version greater than 9.0</li>
</ol>
<pre><code>$ python2 -m pip --version
</code></pre>
<ol start="2">
<li>Then do this</li>
</ol>
<pre><code>$ python2 -m pip install ipykernel OR python2 -m pip install ipykernel --user
$ python2 -m ipykernel install --user
</code></pre>
<ol start="3">
<li>Start or restart Jupyter and you should be done.</li>
</ol>
<p>This solution is from the <a href="https://ipython.readthedocs.io/en/latest/install/kernel_install.html" rel="nofollow noreferrer">ipython docs</a> by the way.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do:</p>
<pre><code>conda create —name py2 python=2.7 anaconda 
conda activate py2
(py2) conda install ipykernel -y
(py2) python -m ipykernel install --user --name py2 --display-name "Python 2.7"
</code></pre>
<p>This create an environment called py2 with Python 2.7 and add it to your kernel with name Python 2.7</p>
<p>If we want to have other versions e.g. Python 3.7 also,  we can do the same steps:</p>
<pre><code>conda update conda
conda create —name py3 python=3.7 anaconda 
conda activate py3
(py3) conda install ipykernel -y
(py3) python -m ipykernel install --user --name py3 --display-name "Python 3.7"
</code></pre>
<p>Note: you do not have to add 'anaconda' packages. Hope this helps you understand how to add environments to your jupyter kernel.</p>
</div>
