<div class="post-text" itemprop="text">
<p>This what solved the problem...</p>
<p>Local Security Policy -&gt; Local Policies -&gt; Security Options -&gt; User Account Control: Run all administrators in Admin Approval Mode -&gt; ENABLED</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>browser = webdriver.Edge(r'C:\ALTIUM_WORK\Altium_Python\MicrosoftWebDriver.exe')
</code></pre>
<p>This is not working because in your office laptop <code>User Account Control settings</code> is <code>Turned Off</code>.</p>
<p><a href="https://i.stack.imgur.com/nJkN8.png" rel="nofollow noreferrer"><img alt="enter image description here" src="https://i.stack.imgur.com/nJkN8.png"/></a></p>
<p>You need turned On <code>User Account Control settings</code> and restart your machine and then run your code.It will work as expected.</p>
<p><a href="https://i.stack.imgur.com/fArdE.png" rel="nofollow noreferrer"><img alt="enter image description here" src="https://i.stack.imgur.com/fArdE.png"/></a></p>
<p>To go to path on OS :</p>
<pre><code>Control Panel--&gt;All Control Panel Items--&gt;User Accounts--&gt; Change User Account Control settings
</code></pre>
<p><a href="https://i.stack.imgur.com/sdVrA.png" rel="nofollow noreferrer"><img alt="enter image description here" src="https://i.stack.imgur.com/sdVrA.png"/></a></p>
