<p>I have a list in python </p>
<pre><code>a=['one_foo','two_foo','bar_foo']
</code></pre>
<p>Expected output:</p>
<pre><code>['one','two','bar']
</code></pre>
<p>I tried doing by first converting list into string and then using regex</p>
<pre><code>import re
re.sub("_.*$","", str(a))
</code></pre>
<p>I know converting list to string is incorrect way to achieve this task,may be is there any elegant way without converting it to string?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a list comprehension with <a href="https://docs.python.org/3/library/stdtypes.html#str.rsplit" rel="nofollow noreferrer"><code>str.rsplit</code></a>:</p>
<pre><code>a = ['one_foo', 'two_foo', 'bar_foo']

res = [x.rsplit('_', 1)[0] for x in a]

['one', 'two', 'bar']
</code></pre>
<p>In general, regex solutions tend to underperform <code>str</code> methods for similar operations.</p>
