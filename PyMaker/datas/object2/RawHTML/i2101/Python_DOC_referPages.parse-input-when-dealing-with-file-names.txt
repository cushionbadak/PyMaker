<p>How can I parse the input when it is a list of paths?</p>
<pre><code>file_in = input("Insert paths: ")  # foo.jpg "C:\Program Files\bar.jpg"
print(file_in)  # foo.jpg "C:\Program Files\bar.jpg"
</code></pre>
<p>I'm looking for a clean way to get the input <code>foo.jpg "C:\Program Files\bar.jpg"</code> in a list <code>['foo.jpg', 'C:\Program Files\bar.jpg']</code> (note the quotes in the second path because of the space in <code>Program Files</code>).</p>
<p>Is there something like <a href="https://docs.python.org/3/library/argparse.html" rel="noreferrer">argparse</a> but for <code>input()</code>s?<br/>
What is the best way to handle it?</p>
</div>
<div class="post-text" itemprop="text">
<p>Here is what you want:</p>
<pre><code>import shlex

file_in = input("Insert paths: ")  # foo.jpg "C:\Program Files\bar.jpg"
print(shlex.split(file_in))  # foo.jpg "C:\Program Files\bar.jpg"
</code></pre>
<p>Output:</p>
<pre><code>['foo.jpg', 'C:\\Program Files\\bar.jpg']
</code></pre>
