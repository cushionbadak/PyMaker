<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/time.html#time.strftime" rel="nofollow noreferrer"><code>time.strftime</code></a> function is what you're after:</p>
<blockquote>
<p>Convert a tuple or <code>struct_time</code> representing a time as returned by <code>gmtime()</code> or <code>localtime()</code> to a string as specified by the <em>format</em> argument.</p>
</blockquote>
<p>To get the required output, use the following format string:</p>
<pre><code>&gt;&gt;&gt; import time
&gt;&gt;&gt; time.strftime("%a %b %d %H:%M %Y", time.localtime())
'Thu Oct 11 17:23:28 2018'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the built in split function, which splits strings into array elements by checking for patterns. Default is ' ', e.g.</p>
<pre><code>'abc def'.split()
</code></pre>
<p>will return</p>
<pre><code>['abc', 'def']
</code></pre>
<p>So, applying this:</p>
<pre><code>localtime = localtime.split()
</code></pre>
<p>will return:</p>
<pre><code>['Thu','Oct','11','17:23:28','2018']
</code></pre>
<p>we want to extract the 4th element of the array, and arrays start at 0, so:</p>
<pre><code>localtime = localtime[3]
</code></pre>
<p>this returns:</p>
<pre><code>'17:23:28'
</code></pre>
<p>now, we need to split again, this time the ':' character:</p>
<pre><code>localtime = localtime.split(':')
</code></pre>
<p>this returns:</p>
<pre><code>['17','23','28']
</code></pre>
<p>now, for the last part:</p>
<p>extract the individual values:</p>
<pre><code>hour = localtime[0]
minute = localtime[1]
second = localtime[2]
</code></pre>
</div>
