<p>If I have a url of <a href="http://www.example.com/myarea" rel="nofollow noreferrer">http://www.example.com/myarea</a>, what python code can I use to obtain the string "myarea"?</p>
<p>Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>If you need the full path after the domain, you can use the <a href="https://docs.python.org/3/library/urllib.parse.html#module-urllib.parse" rel="nofollow noreferrer"><code>urllib.parse</code></a> module:</p>
<pre><code>from urllib.parse import urlparse

x = 'http://www.example.com/myarea'
res = urlparse(x).path  # '/myarea'
</code></pre>
<p>Or you can use <a href="https://docs.python.org/3/library/stdtypes.html#str.rsplit" rel="nofollow noreferrer"><code>str.rsplit</code></a> to extract only the last component:</p>
<pre><code>res = x.rsplit('/', maxsplit=1)[-1]  # 'myarea'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Split on <code>/</code> and take the last element.    </p>
<pre><code>url = "http://www.example.com/myarea"
lastElement = url.split('/')[-1]
</code></pre>
