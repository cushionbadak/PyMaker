<div class="post-text" itemprop="text">
<p>I have a really long list in Python named <code>my_list</code>. I would like to print the contents of this list following these two conditions:</p>
<ul>
<li>One item per line.</li>
<li>Each item needs to be numbered using three-digit numbers (e.g., 001, 002...)</li>
</ul>
<p>This is my desired output:</p>
<pre><code>001  csfiohfewiehns
002  hdifhdigbdgib
003  fdifhihiwhgighigh
004  jfjodopfhaeohgheg
</code></pre>
<p>So far, I have tried the following code:</p>
<pre><code>print("\n".join.(my_list))
</code></pre>
<p>This worked and now each item appears in a new line. However, I still don't know how to number the lines using the format I want. I have heard of the function <code>enumerate</code>, though I don't know how it could be combined with my current line of code.</p>
<p>Any ideas?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>enumerate</code> to generate indices:</p>
<pre><code>print("\n".join('%03d %s' % (i, s) for i, s in enumerate(my_list, 1)))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Agree with blhsing, but since this case list comprehension's are faster, do:</p>
<pre><code>print("\n".join(['%03d %s' % (i, s) for i, s in enumerate(my_list, 1)]))
</code></pre>
<p>Also got Mateen's this way:</p>
<pre><code>print('\n'.join([f'{i + 1:03}  {x}' for i, x in enumerate(xs)]))
</code></pre>
<p>Or <code>zfill</code> as bhlsing's way:</p>
<pre><code>print('\n'.join(['  '.join((str(i).zfill(3),v)) for i,v in enumerate(a,1)]))
</code></pre>
<p>Or as Mateen's way:</p>
<pre><code>print('\n'.join(['  '.join((str(i+1).zfill(3),v)) for i,v in enumerate(a)]))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is probably the simplest way:</p>
<pre><code>'\n'.join(f'{i + 1:03}  {x}' for i, x in enumerate(xs))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could always just use a for loop rather than making a new string:</p>
<pre><code>my_list = ["abc", "def"]

for i, item in enumerate(my_list):
    print(f'{i:03} {item}')
</code></pre>
<p>This will output:</p>
<pre><code>000 abc
001 def
</code></pre>
