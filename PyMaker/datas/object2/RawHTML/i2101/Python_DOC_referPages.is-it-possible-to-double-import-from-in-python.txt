<div class="post-text" itemprop="text">
<p>Let's say i have a <code>src</code> folder with <code>Player.py</code> file, that has class <code>PlayVideo</code> in it. Can i directly import that class from <code>main.py</code> file that is in same folder as <code>src</code>?</p>
<p>Normally, to use that class i would do</p>
<pre><code>from src import Player

Player.PlayVideo("videofile.mp4")
</code></pre>
<p>Is it possible to directly import the <code>PlayVideo</code> class?</p>
</div>
<div class="post-text" itemprop="text">
<p>Yes, as long as <code>Player</code> is a Python file (<code>.py</code>) or Python module (directory containing an <code>__init__.py</code> file), you can do:</p>
<pre><code>from src.Player import PlayVideo
</code></pre>
<p>See the docs on importing submodules here: <a href="https://docs.python.org/3/tutorial/modules.html#packages" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/modules.html#packages</a></p>
</div>
<div class="post-text" itemprop="text">
<pre><code>import matplotlib.pyplot as plt 
</code></pre>
<p>The previous is an extremely common import. I'd guess your answer would be:</p>
<pre><code>import src.Player.PlayVideo as PlayVideo
</code></pre>
<p>It works for something like:</p>
<pre><code>import statsmodels.regression.linear_model as line
</code></pre>
<p>Hope it helps!</p>
</div>
