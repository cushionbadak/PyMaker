<p>Try changing the format of the <code>DateTimeLocalField</code> from</p>
<pre><code>format='%Y-%m-%d %H:%M:%S' 
</code></pre>
<p>to:</p>
<pre><code>format='%Y-%m-%dT%H:%M'
</code></pre>
<p><em>Tip: you can print the actual content of the input field prior to the validation to confirm the correct formatting of the <code>DateTimeLocalField</code> field.</em></p>

<p>Use <code>wtforms.fields.html5.DateTimeLocalField</code> instead of  <code>wtforms.DateTimeLocalField</code>. Set the format with date and time separated by a 'T'. If you would want the current time as the default value, set default parameter.</p>
<pre><code>from wtforms.fields.html5 import DateTimeLocalField

class InterviewForm(Form):
    posted = DateTimeLocalField('Posted:', default=datetime.today, format='%Y-%m-%dT%H:%M')
</code></pre>
