<p>My question is very simple:</p>
<p>I have <code>A = ['AA','BB']</code>, <code>B = ['CC','DD']</code></p>
<p>How do I get <code>AB = ['AACC','AADD','BBCC',BBDD']</code>?</p>
<p>Thank you!</p>

<div class="post-text" itemprop="text">
<p>with <em>list comprehension</em>:</p>
<pre><code>AB = [x + y for x in A for y in B]
</code></pre>
<p>we thus iterate over the elements in <code>A</code> and for each element <code>x</code> in <code>A</code>, we iterate over <code>B</code>, and then add <code>x + y</code> to the list.</p>
<p>Or for a variadic number of lists, and with a generator:</p>
<pre><code>from itertools import product

map(''.join, product(A, B))
</code></pre>
<p>This can be easily extended to a variable number of elements, like:</p>
<pre><code>&gt;&gt;&gt; A = ['AA','BB']; B = ['CC','DD']; C = ['EE', 'FF']
&gt;&gt;&gt; list(map(''.join, product(A, B, C)))
['AACCEE', 'AACCFF', 'AADDEE', 'AADDFF', 'BBCCEE', 'BBCCFF', 'BBDDEE', 'BBDDFF']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow noreferrer"><code>itertools.product</code></a>:</p>
<pre><code>&gt;&gt;&gt; from itertools import product
&gt;&gt;&gt; A = ['AA','BB']
&gt;&gt;&gt; B = ['CC','DD']
&gt;&gt;&gt; AB = [''.join(p) for p in product(A, B)]
&gt;&gt;&gt; AB
['AACC', 'AADD', 'BBCC', 'BBDD']
</code></pre>
<p>This has the benefit of working with any number of iterables.</p>
</div>
<div class="post-text" itemprop="text">
<p>Its easier to see whats happening in a complete loop, here we are going to take <code>i in a</code> which will be <code>AA and BB</code> and <code>j in b</code> which will be <code>CC and DD</code>. On our first iteration we combine the first two <code>AA + CC</code> then <code>append</code> them to our new list , next comes  <code>AA + DD</code> then onto <code>BB</code> and the process repeats.</p>
<pre><code>a = ['AA','BB'] 
b = ['CC','DD']

res = []
for i in a:
    for j in b:
        x = i + j
        res.append(x)
print(res)
# ['AACC', 'AADD', 'BBCC', 'BBDD']
</code></pre>
<p>After you understand this you can skip that process and do it with list comprehension, this is identical.</p>
<pre><code>res = [i + j for i in a for j in b]
</code></pre>
</div>
