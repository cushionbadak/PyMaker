<p>How to print the stack trace of an exception object in Python?</p>
<p>Note that the question is <strong>NOT about printing stack trace of LAST</strong> exception. Exception object may be saved at some distant point of time in the past.</p>
</div>
<div class="post-text" itemprop="text">
<p>It's a bit inconvenient, but you can use <a href="https://docs.python.org/3/library/traceback.html#traceback.print_exception" rel="nofollow noreferrer"><code>traceback.print_exception</code></a>. Given an exception <code>ex</code>:</p>
<pre><code>traceback.print_exception(type(ex), ex, ex.__traceback__)
</code></pre>
<p>Example:</p>
<pre><code>import traceback

try:
    1/0
except Exception as ex:
    traceback.print_exception(type(ex), ex, ex.__traceback__)

# output:
# Traceback (most recent call last):
#   File "untitled.py", line 4, in &lt;module&gt;
#     1/0
# ZeroDivisionError: division by zero
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>you can manually iterate through the <code>__traceback__</code> attribute to print lines &amp; files:</p>
<pre><code>def function():
    raise ValueError("flfkl")

try:
    function()
except Exception as e:
    traceback = e.__traceback__
    while traceback:
        print("{}: {}".format(traceback.tb_frame.f_code.co_filename,traceback.tb_lineno))
        traceback = traceback.tb_next
</code></pre>
</div>
