<div class="post-text" itemprop="text">
<p><code>.reverse()</code> reverses your list in-place and returns <code>None</code>:</p>
<pre><code>sorted_averages = list(range(3))
insorted_averages = sorted_averages.reverse()
print(insorted_averages)
</code></pre>
<p><code>insorted_averages</code> is now <code>None</code>. <code>sorted_averages</code> is reversed though...</p>
<p>read (4) in the python doc under <a href="https://docs.python.org/3/library/stdtypes.html?highlight=list#mutable-sequence-types" rel="nofollow noreferrer">mutable sequence types</a>.</p>
