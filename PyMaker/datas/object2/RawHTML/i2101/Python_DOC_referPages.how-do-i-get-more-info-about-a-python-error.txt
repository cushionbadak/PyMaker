<div class="post-text" itemprop="text">
<p>You linked the documentation for Python 3 however you seem to be using Python 2. <a href="https://docs.python.org/2/library/urllib.html" rel="nofollow noreferrer"><code>urllib</code> on Python 2</a> doesn't have that <code>error</code> module, just as the error states.</p>
<p><a href="https://docs.python.org/2/library/urllib2.html#urllib2.URLError" rel="nofollow noreferrer"><code>urllib2</code></a> on the other hand has this class, so you can use <code>urllib2.URLError</code> instead. </p>
</div>

<div class="post-text" itemprop="text">
<p>Note that Ubuntu 16.04 <code>/usr/bin/python</code> is a symlink to <code>python2</code>. If you want to use python 3.x, you should  change the first line of your script from </p>
<pre><code>#!/usr/bin/python
</code></pre>
<p>to</p>
<pre><code>#!/usr/bin/python3
</code></pre>
<p>(Note: the "#!" at the beginning of this line is called "shebang".)</p>
<p>Alternatively you can create a virtual environment so that when you activate it, it will run python 3.</p>
