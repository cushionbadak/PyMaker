<div class="post-text" itemprop="text">
<p>You can use <a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.Series.str.endswith.html" rel="nofollow noreferrer"><code>pd.Index.str.endswith</code></a> with a <code>tuple</code>, followed by Boolean indexing:</p>
<pre><code>L = ['IDENTIF','RIVER', 'LOCATION', 'ERECTED', 'PURPOSE', 'LENGTH',
     'LANES', 'CLEAR-G', 'T-OR-D', 'MATERIAL', 'SPAN', 'REL-L', 'TYPE']

df = pd.DataFrame(columns=L)

cols = df.columns[df.columns.str.endswith(tuple('HNS'))]

Index(['LOCATION', 'LENGTH', 'LANES', 'SPAN'], dtype='object')
</code></pre>
<p>The functionality mimics Python's built-in <a href="https://docs.python.org/3/library/stdtypes.html#str.endswith" rel="nofollow noreferrer"><code>str.endswith</code></a>, which allows you to supply a <code>tuple</code> to match against multiple items as alternative conditions.</p>
