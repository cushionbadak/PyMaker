<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3/tutorial/controlflow.html#lambda-expressions" rel="nofollow noreferrer"><code>lambda</code></a> to specify a custom sorting criterion:</p>
<pre><code>d = sorted(payload, key=lambda x: len(x['title']))
print(d)
</code></pre>
<p>The official documentation actually has an example of sorting with a <code>lambda</code> here: <a href="https://docs.python.org/3/howto/sorting.html#key-functions" rel="nofollow noreferrer">https://docs.python.org/3/howto/sorting.html#key-functions</a></p>
<p>Then, to print titles and their lengths, you could do something like:</p>
<pre><code>titles = [(p['title'], len(p['title'])) for p in sorted(payload, key=lambda x: len(x['title']))]
print(titles)
&gt;&gt;&gt; [('qui est esse', 12), ('sunt aut facere repellat provident occaecati excepturi optio reprehenderit', 74)]
</code></pre>

<div class="post-text" itemprop="text">
<p>One could do it in three stages:</p>
<p>1) extract title</p>
<p>2) sort by len</p>
<p>3) build pairs</p>
<p>This still comfortably fits one line and is actually quite readable:</p>
<pre><code>[(x, len(x)) for x in sorted((x['title'] for x in mock), key=len)]
</code></pre>
<p>Sample run</p>
<pre><code>[('jfzq', 4), ('uqixc', 5), ('xparg', 5), ('uvuuk', 5), ('gsibnde', 7), ('pophwash', 8), ('cudisvgf', 8), ('swptewjg', 8), ('rthjtjylh', 9), ('ezvwpqhfn', 9)]
</code></pre>
<p>Code to create <code>mock</code> data:</p>
<pre><code>&gt;&gt;&gt; from random import choices, randint
&gt;&gt;&gt; from string import ascii_lowercase
&gt;&gt;&gt; 
&gt;&gt;&gt; mock = [{k: ''.join(choices(ascii_lowercase, k=randint(4, 10))) for k in ('userId', 'id', 'title', 'body')} for _ in range(10)]
</code></pre>
