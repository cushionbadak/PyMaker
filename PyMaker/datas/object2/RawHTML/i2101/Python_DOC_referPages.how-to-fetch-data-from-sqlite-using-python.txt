<p>I am using this sample <a href="http://www.sqlitetutorial.net/sqlite-sample-database/" rel="nofollow noreferrer">sqlite database</a> and my code is </p>
<pre><code>import sqlite3

conn=sqlite3.connect('chinook.db')
conn.execute("SELECT * FROM tracks")
rows= conn.cursor().fetchall()

for row in rows:
    print row
</code></pre>
<p>It should have worked but there is no output? What am I doing wrong here?</p>

<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/sqlite3.html#sqlite3.Connection.execute" rel="nofollow noreferrer"><code>Connection.execute</code> shortcut</a> returns a <code>cursor</code> instance, which you need to use with <code>fetchall</code>. In your code, you're creating a new, independent cursor.</p>
<p>Thus:</p>
<pre><code>import sqlite3

conn = sqlite3.connect('chinook.db')
cursor = conn.execute("SELECT * FROM tracks")
rows = cursor.fetchall()

for row in rows:
    print row
</code></pre>
<p>Or don't use <code>Connection.execute</code> shortcut, to avoid confusion:</p>
<pre><code>import sqlite3

conn = sqlite3.connect('chinook.db')
cursor = conn.cursor()
cursor.execute("SELECT * FROM tracks")
rows = cursor.fetchall()

for row in rows:
    print row
</code></pre>
