<div class="post-text" itemprop="text">
<p>I want to create a new list (V) from other lists (a, b, c) and using a function, but I would like to take advantage of python and apply the function to the three lists and not element by element. </p>
<p>For example, I have the lists a, b and c; and the result after apply the function should be V. Thanks.</p>
<pre><code>def mag(a, b, c):
    # something sophisticated
    return (a+b)*c

a = [1, 5, 7]
b = [4, 8, 3] 
c = [2, 6, 3]
V = [10, 78, 30]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can easily do this with the <code>map</code> function:</p>
<pre><code>V = list(map(mag, a, b, c))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You want to first zip the arguments, then map the function on the unpacked tuples:</p>
<pre><code>from itertools import starmap

starmap(mag, zip(a,b,c))
</code></pre>
<p>See <a href="https://ideone.com/uTNa5L" rel="nofollow noreferrer">here</a> for an example.</p>
</div>
<div class="post-text" itemprop="text">
<p>An alternate solution is to use <code>map</code> and <code>lambda</code></p>
<pre><code>In [16]: list(map(lambda p: mag(*p), zip(a, b, c)))
Out[16]: [10, 78, 30]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>What about using only built-in functions? Like <a href="https://docs.python.org/3.3/library/functions.html#zip" rel="nofollow noreferrer">zip</a></p>
<pre><code>&gt;&gt;&gt; [mag(a_, b_, c_) for a_,b_,c_ in zip(a, b, c)]
[10, 78, 30]
</code></pre>
<p>Plus another python buit-in function, <a href="https://docs.python.org/3.3/library/functions.html#map" rel="nofollow noreferrer">map</a> which returns an
iterator and thus makes things go faster and ends up saving memory:</p>
<pre><code>&gt;&gt;&gt; gen = map(lambda uple:mag(*uple), zip(a, b, c))
&gt;&gt;&gt; list(gen)
[10, 78, 30]
</code></pre>
</div>
