<p>I am trying to get subsequence of items between 2 lists in a list of lists. For example, if I have</p>
<pre><code>list=[[1,2,3,4],[],[6,9],[2],[3],[4]]
</code></pre>
<p>I want to extract the items ranging from <code>list[0][1]</code> until <code>list[2][1]</code> into another list. This resulting list would then be <code>[2,3,4,6]</code> (ignoring the empty list in between).</p>
<p>How can I do that? I have been trying to use a <code>for i in range(...)</code> loop but it is not working for me.</p>
</div>
<div class="post-text" itemprop="text">
<p>Assuming your list is called <code>list_of_lists</code><sup><strong>1</strong></sup>, this <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> using <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate()</code></a> works:</p>
<pre><code>[n for i, sub_list in enumerate(list_of_lists)
   for j, n in enumerate(sub_list)
   if (0, 1) &lt;= (i, j) &lt; (2, 1)]
</code></pre>
<p><sup><strong>1</strong></sup> It's a bad idea to call a list <code>list</code> as in your example, because that masks the name of the type <code>list</code>, which you then don't have access to.</p>
</div>
<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>def complex_slice(my_list, i, j, k):
    result = my_list[i][k:]
    for sub_list in my_list[ (i + 1) : (j - 1) ]:
        result.extend(sub_list)
    result.extend(my_list[j][:k])
    return result
</code></pre>
