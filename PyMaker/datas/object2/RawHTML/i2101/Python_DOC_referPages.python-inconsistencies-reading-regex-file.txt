<div class="post-text" itemprop="text">
<p>Most likely it does not match when you use raw string and double backslash.    </p>
<pre><code>s = "ABC 23"

re.findall('\d+',s)
['23']

re.findall(r'\d+',s)

['23']

re.findall('\\d+',s)
['23']

re.findall(r'\\d+',s)     

[]
</code></pre>
</div>
