<div class="post-text" itemprop="text">
<p>You need a blank <code>__init__.py</code> file in your utility directory in order for Python to see it as a package. See <a href="https://docs.python.org/3/tutorial/modules.html#packages" rel="nofollow noreferrer">the tutorial</a>.</p>
