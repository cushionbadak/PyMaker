<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">re</a> module for this. For the first keyword match:</p>
<pre><code>pattern = "(.+) (.+) (.+) (.+) (.+) grand master (.+) (.+) (.+) (.+) (.+)"
match = re.search(pattern, text)
if match:
    firstword_before = match.group(1) # first pair of parentheses
    lastword_before = match.group(5)

    firstword_after = match.group(6)
    lastword_after = match.group(10)
</code></pre>
<p>Parentheses in the pattern indicates the group number. First pair of parentheses corresponds to match.group(1), second pair of parentheses corresponds to match.group(2) and so on. If you want all the groups you can use:</p>
<pre><code>match.groups() # returns tuple of groups
</code></pre>
<p>or</p>
<pre><code>match.group(0) # returns string of groups
</code></pre>
<p>For all the keyword match in the text, use re.findall. Read <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">re</a>
 for details.</p>
