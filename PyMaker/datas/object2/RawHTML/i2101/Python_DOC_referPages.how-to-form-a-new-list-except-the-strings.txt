<p>Try this:</p>
<pre><code>data_new = [int(x) for x in data if x.isdigit()]
</code></pre>

<div class="post-text" itemprop="text">
<p>Here is the full loop solution:</p>
<pre><code>data_new = []
for i in data:
    if i.isdigit():
        i = int(i)
        data_new.append(i)
</code></pre>
<p>What is happening is you are looping through all the items in data, you check each item to see if it <a href="https://docs.python.org/3/library/stdtypes.html#str.isdigit" rel="nofollow noreferrer"><code>isdigit()</code></a>. From there you convert that item now to a <code>int</code> from a <code>str</code>, that new item gets appended to <code>data_new</code></p>
<p>Shorter methods are available with the use of <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map</code></a> and <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter</code></a> or <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a></p>
<pre><code>data_new = list(map(int, filter(lambda x: x.isdigit(), data)))

data_new = [int(i) for i in data if i.isdigit()]
</code></pre>
