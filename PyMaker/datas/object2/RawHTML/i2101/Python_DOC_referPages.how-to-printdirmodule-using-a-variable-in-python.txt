<pre><code>import PyQt5
from PyQt5.QtWidgets import *

print(dir(PyQt5.QtWidgets))
</code></pre>
<p>That above code runs perfectly. But when I use a variable it gives an error. </p>
<pre><code>qw="QtWidgets"
print(dir(PyQt5.qw))
</code></pre>
<p><strong>AttributeError: module 'PyQt5' has no attribute 'qw'</strong>.
Is there anyway I can do that? The reason I use variable is because I want a user input to choose what modules to print(dir()). </p>
</div>
<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer"><code>getattr</code></a> built-in function will let you look up an attribute by name:</p>
<pre><code>qw = "QtWidgets"
print(dir(getattr(PyQt5, qw)))
</code></pre>
</div>
