<div class="post-text" itemprop="text">
<p>The error occurs only in python 3.7, because of an incompatible change, <a href="https://docs.python.org/3/whatsnew/3.7.html" rel="nofollow noreferrer">async is a reserved keyword</a> since this version.</p>
<p>The solution is to keep using python 3.6 until the library is adapted to the new version, there is an already closed <a href="https://github.com/dpkp/kafka-python/issues/1566" rel="nofollow noreferrer">issue</a>:</p>
<pre><code>FROM python:3.6
MAINTAINER kamal.nandan@&lt;myemailservice&gt;

RUN pip install requests requests_oauthlib kafka

ADD twitter_app.py /
CMD python3 twitter_app.py
</code></pre>
<p>(I took the liberty of reducing the Dockerfile)</p>
