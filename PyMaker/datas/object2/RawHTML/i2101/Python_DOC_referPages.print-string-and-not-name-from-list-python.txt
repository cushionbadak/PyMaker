<span class="comment-copy">You realize a string like <code>"hello"</code> is already a sequence that supports iteration, right? You don't need to boostrap that on your own.</span>
<span class="comment-copy">You are mixing up variable <i>names</i> (<code>a</code>, <code>b</code>, and so on) and the <i>contents</i> of a string (<code>"hello"</code>). That does not mix, in Python. Best start with <a href="https://docs.python.org/3/tutorial/" rel="nofollow noreferrer">the official tutorial</a>; you may want to read all the way up to dictionaries and lists. (And there is a function to write out a number in binary.)</span>
