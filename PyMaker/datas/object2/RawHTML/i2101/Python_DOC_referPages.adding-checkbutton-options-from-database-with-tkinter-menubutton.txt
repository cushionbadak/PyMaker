<p>If you want the items in the <code>geo_list</code> for menu labels, why not just set them; you are already looping over them: </p>
<pre><code>amb.menu.add_checkbutton(label=category)
</code></pre>
<p>Also, don't pack amb at the very end; you have already gridded it earlier. </p>
<p>I'm adding an example of how to trace changes associated with each menu item. I have changed the code slightly but I think you will have no problem to get the general idea. </p>
<p>I'm using a <code>BooleanVar()</code> instead of an <code>IntVar()</code>, then each var is saved in <code>location_vars</code> with the key <code>"Item{0}".format(category)</code>. Finally I'm setting up a trace for changes in each menu item to to a callback function which will inspect the selections. </p>
<p>Is this what you are after?</p>
<pre><code>from tkinter import *

root = Tk()

amb = Menubutton(root, text="Geo Category", relief=RAISED)
amb.pack(padx=50, pady=25)
amb.menu = Menu(amb, tearoff=0)
amb['menu'] = amb.menu

def callback(*args):
    # Callvack function is called when menu items are changed
    for key, value in location_vars.items():
        print(key, value.get())
    print() # To make the printout more readable

geo_list= ["None","geo1","geo2","geo3","geo4"]
location_vars = {}
for category in geo_list:
    location_vars["Item{0}".format(category)] = BooleanVar()
    # Set "variable" for every menu item
    amb.menu.add_checkbutton(label=category,
        variable=location_vars["Item{0}".format(category)])
    # Trace changes in the variables
    location_vars["Item{0}".format(category)].trace("w", callback)

root.mainloop()
</code></pre>
