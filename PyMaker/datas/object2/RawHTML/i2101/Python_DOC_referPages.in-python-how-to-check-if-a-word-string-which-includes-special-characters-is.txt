<p>I have a list of strings like my_list = {'(.,0)', '[6,.)', '(b)' , '(,2]'}. I need to write a code which checks if the strings in the list are compatible with any of formats bellow:</p>
<p>format1: '/(/./,/a float number/)'</p>
<p>format2: '/[/float number/,/./)' </p>
<p>For instance '(.,0)' is compatible with format1 and '[6,.)' is compatible with format2. </p>
<p>I also checked <a href="https://stackoverflow.com/questions/14966647/check-python-string-format">here</a> but could not find how to check for float?</p>

<div class="post-text" itemprop="text">
<p>'\d*\.?\d*' may help you check float.</p>
<p>Try these two regular expressions:</p>
<pre><code>pattern1: ^\(\.,\d*\.?\d*\)$
pattern2: ^\[\d*\.?\d*\,\.\)$
</code></pre>
<p>( I assume your example is correct and your description to pattern2 is actually closedbracket float comma dot parenthesis)</p>
<pre><code>import re
r = '(.,0)'
if(re.match('^\(\.,\d*\.?\d*\)$',r) is not None):
    print('matches')
elif(re.match('^\[\d*\.?\d*\,\.\)$',r) is not None):
    print('matches')
</code></pre>
