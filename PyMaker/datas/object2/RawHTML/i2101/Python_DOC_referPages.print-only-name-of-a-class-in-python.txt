<p>I’m learning Python 3.</p>
<p>My code </p>
<pre><code>numbers = [1, 2]
print( type( numbers[0] ))
</code></pre>
<p>outputs</p>
<pre><code>&lt;class ‘int’&gt;
</code></pre>
<p>How do I print only </p>
<pre><code>‘int’
</code></pre>
<p>or </p>
<pre><code>int
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/stdtypes.html#definition.__name__" rel="nofollow noreferrer"><code>__name__</code></a> attribute from the type.</p>
<pre><code>&gt;&gt;&gt; type(5).__name__
'int'
</code></pre>
<p>Or without using the <a href="https://docs.python.org/3/library/functions.html#type" rel="nofollow noreferrer">type built-in</a>:</p>
<pre><code>&gt;&gt;&gt; (5).__class__.__name__
'int'
</code></pre>
</div>
