<p>So, I made the auth this way. It works!</p>
<pre><code>def get_movies(self):
    """Get api data

    """
    url  = '{}/Get_Cinema_Movies'.format(self.url)
    auth = '{"AUTH":{"USER":"user","PASS":"password"}}'

    return requests.post(url, data=auth).json()
</code></pre>
