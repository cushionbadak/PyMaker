<p>Your first approach is OK, for the second way use this:</p>
<pre><code>n=int(input("how many number you want to enter:"))
l=map(int, input().split())[:n] # l is now a list of at most n integers
</code></pre>
<p>This will <a href="https://docs.python.org/2/library/functions.html#map" rel="nofollow"><code>map</code></a> the function <a href="https://docs.python.org/2/library/functions.html#int" rel="nofollow"><code>int</code></a> on the split parts (what <a href="https://docs.python.org/2/library/stdtypes.html#str.split" rel="nofollow"><code>split</code></a> gives) of the user input (which are <code>1</code>, <code>2</code>, <code>3</code> and <code>4</code> in your example.</p>
<p>It also uses slicing (the <code>[:n]</code> after <code>map</code>) to slice in case the user put more integers in.</p>

<div class="post-text" itemprop="text">
<p>The <code>input()</code> function returns a string that the user enters. The <code>int()</code> function expects to convert a number as a string to the corresponding number value. So <code>int('3')</code> will return 3. But when you type in a string like <code>1 2 3 4</code> the function <code>int()</code> does not know how to convert that.</p>
<p>You can either follow your first example:</p>
<pre><code>    n = int(input('How many do you want to read?'))
    alist = [] 

    for i in range(n):
        x = int(input('--&gt;'))
        alist.append(x)
</code></pre>
<p>The above requires you to only enter a number at a time.</p>
<p>Another way of doing it is just to split the string.</p>
<pre><code>    x = input('Enter a bunch of numbers separated by a space:')
    alist = [int(i) for i in x.split()]
</code></pre>
<p>The <code>split()</code> method returns a list of numbers as strings excluding the spaces</p>
</div>
