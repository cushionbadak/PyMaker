<div class="post-text" itemprop="text">
<p>Haven't tested but a theory: you are using python3! </p>
<p>From <a href="https://docs.python.org/3/whatsnew/3.0.html">https://docs.python.org/3/whatsnew/3.0.html</a></p>
<blockquote>
<p>dict methods dict.keys(), dict.items() and dict.values() return “views”     instead of lists. For example, this no longer works: k = d.keys(); k.sort(). Use k = sorted(d) instead (this works in Python 2.5 too and is just as efficient).</p>
</blockquote>
<p>as I understand it a "view" is an iterator, and an iterator does not have the sort function. Change it to </p>
<pre><code>sorted(all_colors)
</code></pre>
<p>according to the documentation</p>
</div>
<div class="post-text" itemprop="text">
<p>So the total solution based on Johan's answer is:</p>
<pre><code>all_colors = sorted(reportlab.lib.colors.getAllNamedColors().items())
</code></pre>
