<div class="post-text" itemprop="text">
<p>You cannot import <code>spiders</code> from the <code>tests</code> directory like that, unless you have it in your <code>PYTHON_PATH</code> environment variable. Please read about Python's <a href="https://docs.python.org/3/reference/import.html" rel="nofollow noreferrer">import system</a>.</p>
<p>For a quick and dirty (!) fix add this at the top of your file:</p>
<pre><code>import sys
sys.path.append("..")
</code></pre>
<p>Or read this post about relative imports: <a href="https://stackoverflow.com/questions/279237/import-a-module-from-a-relative-path">Import a module from a relative path</a></p>
</div>
