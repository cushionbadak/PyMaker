<div class="post-text" itemprop="text">
<p>I want to pass any number of strings to a a python program and have them parsed as a list or array:</p>
<pre><code>foo.py hello world
foo.py a b e f g ddd qqq vvv
</code></pre>
<p>etc, up to 200 strings.</p>
<p>I'm using <a href="https://docs.python.org/3/library/argparse.html#module-argparse" rel="nofollow">argparse</a> to parse command-line arguments in python. How can I do this? The documentation doesn't give any examples of parsing arbitrary-number of arguments.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/2/library/argparse.html#nargs" rel="nofollow"><code>nargs</code></a> keyword argument.</p>
<pre><code>import argparse

p = argparse.ArgumentParser()
p.add_argument('foo', nargs='+')
args = p.parse_args()

print args.foo
</code></pre>
<p><code>nargs='+'</code> requires one argument.  Usage:</p>
<pre><code>$ python test.py a b c
['a', 'b', 'c']

$ python test.py
usage: p.py [-h] foo [foo ...]
p.py: error: too few arguments
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just do, e.g:</p>
<pre><code>import sys
print(sys.argv[1:])
</code></pre>
<p>You'll see that <code>sys.argv</code> is exactly the list you want (except <code>sys.argv[0]</code> is the name of your program, which you probably don't care about -- which is why I'm slicing it out:-).</p>
</div>
