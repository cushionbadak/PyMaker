<div class="post-text" itemprop="text">
<p>For example I have something like this:</p>
<pre><code>{'key1': 'value', 'key2': 'value', 'key3': 'value','key4': 'value','key4': 'value'}
</code></pre>
<p>And have an array:</p>
<pre><code>['key1', 'key3']
</code></pre>
<p>How to check if all keys from second array exists in first? </p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#all"><code>all</code></a> and a <a href="https://wiki.python.org/moin/Generators">generator expression</a>. This will iterate over your array and check that each key is in the keys of <code>d</code>. If at least one of the keys is missing then it'll return False.</p>
<pre><code>d = {'key1': 'value', 'key2': 'value', 'key3': 'value','key4': 'value','key4': 'value'}
a = ['key1', 'key3']

all(key in d for key in a) # True

a2 = ['key1', 'key5']

all(key in d for key in a2) # False
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#set.issubset" rel="nofollow">set.issubset()</a> method to check</p>
<pre><code>a = {'key1': 'value', 'key2': 'value', 'key3': 'value','key4': 'value','key4': 'value'}
b = ['key1', 'key3']

set(b).issubset(a)
#True

c = ['key1', 'key5']
set(c).issubset(a)
#False
</code></pre>
<p>Edit:  apply <code>issubset</code> directly on <code>a</code> according to @Maroun Maroun comment</p>
</div>
<div class="post-text" itemprop="text">
<p>Another way (using <a href="https://docs.python.org/3/library/stdtypes.html#set" rel="nofollow"><code>set</code></a> and <a href="https://docs.python.org/3/library/stdtypes.html#set.issubset" rel="nofollow"><code>issubset</code></a>):</p>
<pre><code>set(['key1', 'key3']).issubset(your_dict)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a loop and in operator with dict object:</p>
<pre><code>adict = {'key1': 'value', 'key2': 'value', 'key3': 'value','key4': 'value','key4': 'value'}

keys_list = ['key1', 'key3']
for key in keys_list:
  if key in adict:
    print key
</code></pre>
<p>"in" can check if the key in a dict </p>
</div>
