<div class="post-text" itemprop="text">
<p>Without giving too much away... the "numbers" in your list are actually <code>string</code>s, and python allows you to add strings... by concatenating them together. So <code>'a' + 'b' + '9'</code> gives you <code>'ab9'</code>. You need to convert these <code>string</code>s into <code>int</code>s as <code>int('19')</code>, so <code>int('9') + int('7')</code> gives <code>16</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Convert everything to an integer, sum what you need, print it:</p>
<pre><code>print("1.",sum(map(int, grades[:7])))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to convert strings to <code>int</code>s to do numeric operations (instead of string concatenations):</p>
<pre><code>grades = ['9', '7', '7', '10', '3', '9', '6', '6', '2']
grades = [int(x) for x in grades]
print("1.", grades[0] + grades[1] + grades[2] + grades[3] +
            grades[4] + grades[5] + grades[6])
</code></pre>
<p>Or, using <code>sum</code> with list slice (useful if the items to add are continuous items)</p>
<pre><code>grades = ['9', '7', '7', '10', '3', '9', '6', '6', '2']
grades = [int(x) for x in grades]
print("1.", sum(grades[:7]))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>These are individual strings. You cannot do math on strings.</p>
<p>What happens when you do <code>+</code> on strings is called string concatenation.</p>
<p>ie. <code>'1'+'2'</code> will produce <code>'12'</code> and not <code>'3'</code></p>
<p>You need to convert these to integers using <code>int()</code> in order to perform arithmetic.</p>
<pre><code>var result = int('1') + int('2')
&gt;&gt; 3
</code></pre>
<p>So either</p>
<pre><code>print("1.", int(grades[0]) + int(grades [1]) + int(grades [2]) + int(grades [3]) + int(grades [4]) + int(grades[5]) + int(grades[6]) )
</code></pre>
<p>or a snazzy list comprehension with <a href="https://stackoverflow.com/questions/509211/explain-pythons-slice-notation">list slicing</a> and <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer">sum</a></p>
<pre><code>grades = [int(x) for x in grades]
print("1.", sum(grades[:7]))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The list <code>grades[]</code> contains string variables. Integers are whole numbers. You need to change the values in the <code>grades[]</code> list to integers. To do that, you need to remove the quotes around the values in <code>grades[]</code>. Then it will output the sum of all the numbers in that list. <strong><em>Hope this helps.</em></strong></p>
</div>
<div class="post-text" itemprop="text">
<pre><code>sum = 0
for i in range(0,7):
    sum += int(grades[i])
print(sum)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>reduce</code> with <code>lambda</code> for this ,</p>
<pre><code>&gt;&gt;&gt; print( "1.",reduce(lambda x,y:int(x)+int(y),grades[:7]))
</code></pre>
<p>As you tagged this question with python 3.x the <code>reduce</code> is now part of <code>functools</code> module.</p>
<pre><code>import functools
print( "1.",functools.reduce(lambda x,y:int(x)+int(y),grades[:7]))
</code></pre>
