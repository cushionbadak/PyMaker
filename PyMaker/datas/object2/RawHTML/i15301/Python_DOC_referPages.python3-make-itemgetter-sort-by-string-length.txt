<div class="post-text" itemprop="text">
<p>I am using the line of code below to sort a list of dictionaries by the value of a specific key. The code does perform the desired sorting. However, it it sorting by Alphanumeric order rather than by string-length. How do I change itemgetter's behavior and make it sort by length?</p>
<pre><code>sorted(DATABASE, key=itemgetter('SPECIAL_KEY'), reverse=True)
</code></pre>
<p>NOTE: In th script, itemgetter is imported via "from operator import itemgetter".</p>
<p>EDIT: Below is some sample data that I am trying to to sort.</p>
<p>Unsorted:</p>
<pre><code>DATABASE = [
    {'SPECIAL_KEY': 'LONG STRING', 'VARIOUS':'DATA'},
    {'SPECIAL_KEY': 'SHORT STRING', 'VARIOUS':'DATA'},
    {'SPECIAL_KEY': 'VERY LONG STRING', 'VARIOUS':'DATA'},
    {'SPECIAL_KEY': 'MEDIUM STRING', 'VARIOUS':'DATA'},
]
</code></pre>
<p>Desired:</p>
<pre><code>DATABASE = [
    {'SPECIAL_KEY': 'VERY LONG STRING', 'VARIOUS':'DATA'},
    {'SPECIAL_KEY': 'LONG STRING', 'VARIOUS':'DATA'},
    {'SPECIAL_KEY': 'MEDIUM STRING', 'VARIOUS':'DATA'},
    {'SPECIAL_KEY': 'SHORT STRING', 'VARIOUS':'DATA'},
]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to pass a custom function to do it:</p>
<pre><code>sorted(DATABASE, key=lambda x: len(x['SPECIAL_KEY']), reverse=True)
</code></pre>
<p>To sort the list in-place, you need to use <a href="https://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow"><code>list.sort</code></a> instead of <code>sorted</code> (<code>sorted</code> returns a new list):</p>
<pre><code>DATABASE.sort(key=lambda x: len(x['SPECIAL_KEY']), reverse=True)
</code></pre>
</div>
