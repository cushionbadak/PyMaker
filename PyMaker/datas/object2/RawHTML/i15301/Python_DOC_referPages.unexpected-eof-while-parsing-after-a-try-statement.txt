<div class="post-text" itemprop="text">
<p>You need an except after the try. It's just a small syntax error; the eof comes from the python parser before it executes the code. </p>
<pre><code>def start ():   
    print ("This is where text would be")   
    prompt_sta ()

def prompt_sta ():  
    prompt_0=raw_input("Input a Command: ")  
    try:  
        if prompt_0 == 'Okay':                             
           next_screen ()  
        else:   
            print ('Type Okay.')   
            prompt_sta ()  
    except Exception as ex:
        print (ex)
</code></pre>
