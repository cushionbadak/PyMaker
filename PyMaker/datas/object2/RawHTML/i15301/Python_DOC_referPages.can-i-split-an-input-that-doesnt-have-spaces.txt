<div class="post-text" itemprop="text">
<p>I'm attempting to create a program that relates to poker using python.
When someone inputs <code>"9H"</code>, is there a way to separate the <code>9</code> from the <code>H</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>The easiest way to break a string into individual characters is to call <a href="https://docs.python.org/3/library/functions.html#func-list"><code>list()</code></a> on it:</p>
<pre><code>&gt;&gt;&gt; list("9H")
['9', 'H']
&gt;&gt;&gt;
</code></pre>
<p>This will iterate over the string and collect its characters into a new list object.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can access each individual character in a string using its index, there's no need to convert the input to a list. If it's just two characters, you can always do this:</p>
<pre><code>card = '9H'
number, suit = card[0], card[1]
</code></pre>
<p>Or even simpler, we can unpack the elements in the string:</p>
<pre><code>number, suit = card
</code></pre>
<p>Now <code>number</code> will contain the string value <code>'9'</code> and <code>suit</code> will be equal to <code>'H'</code>. Be careful with the edge case though - how do we process a card such as <code>'10S'</code>? a bit clunkier, but we can use regular expressions, and this will work for all valid inputs of cards:</p>
<pre><code>import re
card = '10S'
number, suit = re.findall(r'(\d+)(\D)', card)[0]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Convert it to a list:</p>
<pre><code>$ python
Python 2.7.8 (default, Oct 30 2014, 18:30:15)
&gt;&gt;&gt; "example"[0]
'e'
&gt;&gt;&gt; "example"[1]
'x'
&gt;&gt;&gt; list("example")
['e', 'x', 'a', 'm', 'p', 'l', 'e']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Note that all these answers are scary if you'll be expecting input like <code>10H</code> rather than <code>TH</code>. In which case I'd consider stripping the last token (which should always be a single character representing the suit) from the rest and doing that instead.</p>
<pre><code>your_input = "10H"
rank, suit = your_input[:-1], your_input[-1]
</code></pre>
<p>This plays nicely for more "normal" inputs as well:</p>
<pre><code>rank, suit = "9H"[:-1], "9H"[-1]
</code></pre>
</div>
