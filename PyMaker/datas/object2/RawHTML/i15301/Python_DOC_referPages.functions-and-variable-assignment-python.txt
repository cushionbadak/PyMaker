<div class="post-text" itemprop="text">
<p><code>sort()</code> and <code>sorted()</code> are different:</p>
<ul>
<li><code>sort()</code> sorts the list in-place and returns <code>None</code>.</li>
<li><code>sorted()</code> creates a new list and returns it.</li>
</ul>
<p>See <a href="https://docs.python.org/3/howto/sorting.html">here</a> for further details.</p>
