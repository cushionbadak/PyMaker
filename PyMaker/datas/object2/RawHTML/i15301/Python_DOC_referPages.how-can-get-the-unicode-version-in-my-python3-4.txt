<div class="post-text" itemprop="text">
<p>The latest unicode version is Unicode 7, according to <a href="http://www.unicode.org/versions/" rel="nofollow">http://www.unicode.org/versions/</a>.</p>
<p>Which Unicode version was python3.4 built on? How to get it?</p>
</div>
<div class="post-text" itemprop="text">
<p>From the <a href="https://docs.python.org/3.4/library/unicodedata.html" rel="nofollow"><code>unicodedata</code> module documentation for Python 3.4</a>:</p>
<blockquote>
<p>The data contained in this database is compiled from the UCD version 6.3.0.</p>
</blockquote>
<p>The <a href="https://docs.python.org/3/library/unicodedata.html#unicodedata.unidata_version" rel="nofollow"><code>unicodedata.unidata_version</code> value</a> tells you the version used your current Python installation; demo on Python 3.4:</p>
<pre><code>&gt;&gt;&gt; import unicodedata
&gt;&gt;&gt; unicodedata.unidata_version
'6.3.0'
</code></pre>
<p>Python 3.5 (first alpha <a href="https://www.python.org/dev/peps/pep-0478/" rel="nofollow">due out in February 2015</a>) will use Unicode 7.0.0.</p>
</div>
