<div class="post-text" itemprop="text">
<p><code>itertools.imap()</code> is in Python 2, but not in Python 3.</p>
<p>Actually, that function was moved to just the <code>map</code> function in Python 3 and if you want to use the old Python 2 map, you must use <code>list(map())</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you want something that works in both Python 3 and Python 2, you can do something like:</p>
<pre><code>try:
    from itertools import imap
except ImportError:
    # Python 3...
    imap=map
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are using Python 3, therefore there is no <code>imap</code> function in <code>itertools</code> module. It was removed, because global function <a href="https://docs.python.org/3/library/functions.html#map" rel="noreferrer"><code>map</code></a> now returns iterators.</p>
</div>
<div class="post-text" itemprop="text">
<p>How about this?</p>
<pre><code>imap = lambda *args, **kwargs: list(map(*args, **kwargs))
</code></pre>
<p>In fact!! :)</p>
<pre><code>import itertools
itertools.imap = lambda *args, **kwargs: list(map(*args, **kwargs))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I like the <a href="http://python-future.org/compatible_idioms.html" rel="nofollow noreferrer"><code>python-future</code> idoms</a> for universal Python 2/3 code, like this:</p>
<pre><code># Works in both Python 2 and 3:
from builtins import map
</code></pre>
<p>You then have to refactor your code to use <code>map</code> everywhere you were using <code>imap</code> before:</p>
<pre><code>myiter = map(func, myoldlist)

# `myiter` now has the correct type and is interchangeable with `imap`
assert isinstance(myiter, iter)
</code></pre>
<p>You do need to install future for this to work on both 2 and 3:</p>
<pre><code>pip install future
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the 2to3 script (<a href="https://docs.python.org/2/library/2to3.html" rel="nofollow noreferrer">https://docs.python.org/2/library/2to3.html</a>) which is part of every Python installation to translate your program or whole projects from Python 2 to Python 3.</p>
<pre><code>python &lt;path_to_python_installation&gt;\Tools\scripts\2to3.py -w &lt;your_file&gt;.py
</code></pre>
<p>(-w option writes modifications to file, a backup is stored)</p>
