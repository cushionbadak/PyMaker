<div class="post-text" itemprop="text">
<p>Firstly, you can't be only alpha and only numeric, so your expression will always be false.</p>
<p>Secondly the methods <code>isalpha()</code> and <code>isdigit()</code> equate to <code>True</code> or <code>False</code>, so no need to use <code>== False</code>.</p>
<p>I would suggest using <code>.isalnum()</code>.</p>
<p>If that doesn't satisfy you're requirements you should use a regex.</p>
<p><code>alnum</code> looks for both: <a href="https://docs.python.org/2/library/stdtypes.html#str.isalnum">https://docs.python.org/2/library/stdtypes.html#str.isalnum</a> and works in Python 3.x and 2.</p>
<p>Example:</p>
<pre><code>&gt;&gt;&gt; 'sometest'.isalnum()
True
&gt;&gt;&gt; 'some test'.isalnum()
False
&gt;&gt;&gt; 'sometest231'.isalnum()
True
&gt;&gt;&gt; 'sometest%231'.isalnum()
False
&gt;&gt;&gt; '231'.isalnum()
True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have three problems:</p>
<ul>
<li><code>if a and b == False</code> is <strong>not</strong> the same is <code>if a == False and b == False</code>;</li>
<li><code>if b == False</code> should be written <code>if not b</code>; and</li>
<li>You aren't using <a href="https://docs.python.org/3/library/stdtypes.html#str.isalnum" rel="noreferrer"><code>str.isalnum</code></a>, which saves you from the problem anyway.</li>
</ul>
<p>So it should be:</p>
<pre><code>if not userInput.isalnum():
</code></pre>
