<div class="post-text" itemprop="text">
<p>Using a generator expression:</p>
<pre><code>&gt;&gt;&gt; myList = [1, 3, 5, 6, 8, 10, 34, 2, 0, 3]
&gt;&gt;&gt; sum(num for num in myList if not num%2)
60
</code></pre>
<p>Using <code>filter()</code>:</p>
<pre><code>&gt;&gt;&gt; myList = [1, 3, 5, 6, 8, 10, 34, 2, 0, 3]
&gt;&gt;&gt; sum(filter(lambda x: not x%2, myList))
60
</code></pre>
<p>Using a manual loop:</p>
<pre><code>&gt;&gt;&gt; myList = [1, 3, 5, 6, 8, 10, 34, 2, 0, 3]
&gt;&gt;&gt; result = 0
&gt;&gt;&gt; for item in myList:
...     if not item%2:
...             result += item
...
&gt;&gt;&gt; result
60
</code></pre>

<p>Sorry, I just had to golf this. Maybe it'll teach someone <a href="https://docs.python.org/3/reference/expressions.html#unary-arithmetic-and-bitwise-operations" rel="nofollow">the ~ operator</a>.</p>
<pre><code>&gt;&gt;&gt; myList = [1, 3, 5, 6, 8, 10, 34, 2, 0, 3]
&gt;&gt;&gt; sum(~i%2*i for i in myList)
60
</code></pre>
<p>Found another one with the same length:</p>
<pre><code>&gt;&gt;&gt; sum(i&amp;~i%-2for i in myList)
60
</code></pre>

<div class="post-text" itemprop="text">
<p>You need to store the result in a variable and add the even numbers to the variable, like so:</p>
<pre><code>&gt;&gt;&gt; myList = [1, 3, 5, 6, 8, 10, 34, 2, 0, 3]
&gt;&gt;&gt; result = 0  # Initialize your results variable.
&gt;&gt;&gt; for i in myList:  # Loop through each element of the list.
...   if not i % 2:  # Test for even numbers.
...     result += i
... 
&gt;&gt;&gt; print(result)
60
&gt;&gt;&gt; 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can filter out all non-even elements like so</p>
<pre><code>my_list = [1, 3, 5, 6, 8, 10, 34, 2, 0, 3]
even_list = filter(lambda x: x%2 == 0, my_list)
</code></pre>
<p>and then sum the output like so:</p>
<pre><code>sum(even_list)
</code></pre>
