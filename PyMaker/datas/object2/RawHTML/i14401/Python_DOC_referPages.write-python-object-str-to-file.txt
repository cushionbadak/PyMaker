<p>You can use the <code>str()</code> function to convert an object to a string the same way <code>print</code> does:</p>
<pre><code>for j in range(10):
    c = C()
    print c
    file.write(str(c))
</code></pre>
<p>This will not include a newline, however. If you need a newline as well, you can manually add one:</p>
<pre><code>file.write(str(c) + '\n')
</code></pre>
<p>or use string formatting:</p>
<pre><code>file.write('{}\n'.format(c))
</code></pre>
<p>or use the <code>print</code> statement with redirection (<code>&gt;&gt; fileobject</code>):</p>
<pre><code>print &gt;&gt; file, c
</code></pre>

<span class="comment-copy">In Python 2, you can use <code>print &gt;&gt;file, c</code> to send the output of a <a href="https://docs.python.org/2/reference/simple_stmts.html#the-print-statement" rel="nofollow noreferrer"><code>print</code> statement</a> to the file. In Python 3 the <a href="https://docs.python.org/3/library/functions.html?highlight=print#print" rel="nofollow noreferrer"><code>print()</code> function</a> uses a keyword argument to do this, i.e. <code>print(c, file=file)</code>. A <code>file</code>'s <code>write()</code> method does not automatically call an object's <code>__str__()</code> method like <code>print</code> does, so you'd need do it explicitly via <code>file.write(str(c)+'\n')</code>.</span>
