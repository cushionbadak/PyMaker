<div class="post-text" itemprop="text">
<p>Python 2.6 doesn't support the <code>str.casefold()</code> method.</p>
<p>From the <a href="https://docs.python.org/3/library/stdtypes.html#str.casefold" rel="nofollow noreferrer"><code>str.casefold()</code> documentation</a>:</p>
<blockquote>
<p>New in version 3.3.</p>
</blockquote>
<p>You'll need to switch to Python 3.3 or up to be able to use it.</p>
<p>There are no good alternatives, short of implementing the Unicode casefolding algorithm yourself. See <a href="https://stackoverflow.com/questions/18271077/how-do-i-case-fold-a-string-in-python-2">How do I case fold a string in Python 2?</a></p>
<p>However, since you are handling a <em>bytestring</em> here (and not Unicode), you could just use <code>str.lower()</code> and be done with it.</p>
</div>
<div class="post-text" itemprop="text">
<p>In python 2.x you will get an error when you use <code>casefold()</code>.</p>
<p>You may just use <code>lower()</code>, these are not the same but comparable.</p>
<p>Read: <a href="https://docs.python.org/3/library/stdtypes.html#str.casefold" rel="nofollow noreferrer">str.casefold()</a></p>
<blockquote>
<p>Casefolding is similar to lowercasing but more aggressive because it
  is intended to remove all case distinctions in a string. For example,
  the German lowercase letter 'ß' is equivalent to "ss". Since it is
  already lowercase, lower() would do nothing to 'ß'; casefold()
  converts it to "ss".</p>
</blockquote>
