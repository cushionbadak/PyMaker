<p>Considering you've imported <code>Counter</code>, the easiest way to do what you want is:</p>
<pre><code>from collections import Counter

word = input('What do you want to type? ')
bag = Counter(word)
pickedLetter = input('What letter do you want to pick? ')
print(bag.get(pickedLetter, 0))
</code></pre>

<div class="post-text" itemprop="text">
<p>To count the <em>exact</em> character, you can use the builtin <a href="https://docs.python.org/2/library/stdtypes.html#str.count" rel="nofollow"><code>count</code> method</a>.</p>
<pre><code>&gt;&gt;&gt; mystring = "This is a test."
&gt;&gt;&gt; mystring.count("t")
2
</code></pre>
<p>If you want to count <em>regardless of the case</em>, you can use the <a href="https://docs.python.org/2/library/stdtypes.html#str.lower" rel="nofollow"><code>str.lower</code> method</a>.</p>
<pre><code>&gt;&gt;&gt; mystring = "This is a test."
&gt;&gt;&gt; mystring.lower().count("t")
3
</code></pre>
<p>You have to <code>lower</code> the string, because the character codes for <code>T</code> and <code>t</code> are not the same. They are 84 and 116, respectively.</p>
<p>In your function, it would look like this:</p>
<pre><code>def count_letters(word):
    return word.count(pickedletter)
</code></pre>
<p>Or</p>
<pre><code>def count_letters(word):
    return word.lower().count(pickedletter)
</code></pre>
<p><strong>EDIT</strong> (thanks to abarnert)</p>
<p>If you intend to use this in non-English languages, you should use the <a href="https://docs.python.org/3/library/stdtypes.html#str.casefold" rel="nofollow"><code>casefold</code> method</a> instead. However, this is only available in Python 3.</p>
<pre><code>&gt;&gt;&gt; mystring = "This is a test."
&gt;&gt;&gt; mystring.casefold().count("t")
3
</code></pre>
