<div class="post-text" itemprop="text">
<p>Probably not, but it would be trivial to write something to do it. Import <a href="https://docs.python.org/3/library/csv.html" rel="nofollow noreferrer">Python's <code>csv</code> module</a>, then use the data you read to update an <a href="http://www.blender.org/api/blender_python_api_2_74_5/bpy.types.Object.html#bpy.types.Object.location" rel="nofollow noreferrer">object's <code>location</code></a> when the <a href="http://www.blender.org/api/blender_python_api_2_74_5/bpy.app.handlers.html#bpy.app.handlers.frame_change_post" rel="nofollow noreferrer">frame changes</a>. Or you could transfer the data to an animation curve.</p>
<p>This question might be better asked on the <a href="https://blender.stackexchange.com/">Blender Stack Exchange site</a>.</p>
