<div class="post-text" itemprop="text">
<p>I have 3 Python scripts, script1, script2 and script3 in a folder. I want to run script2 and script3 using script1. How can I do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>In script1 you need to import script2 and script3:</p>
<p>At the top of script1:</p>
<pre><code>import script2
import script3
</code></pre>
<p>To run a function from script2 for example:</p>
<pre><code>script2.function()
</code></pre>
<p>You may also need to add a blank file called <code>__init__.py</code> in the same directory as the scripts, so that python can see that the directory is a library.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use </p>
<pre><code>    execfile("script2.py")
    execfile("script3.py")
</code></pre>
<p>or</p>
<pre><code>    subprocess.call("script2.py")
    subprocess.call("script3.py")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can also use os.system:</p>
<pre><code>os.system("script2.py")
os.system("script3.py")
</code></pre>
</div>
