<p><a href="http://effbot.org/tkinterbook/button.htm" rel="nofollow">This</a> is the first result I get in google:</p>
<pre><code>from Tkinter import *

master = Tk()

def callback():
    print "click!"

b = Button(master, text="OK", command=callback)
b.pack()

mainloop()
</code></pre>
<p>As you can see, <code>command</code> is used to assign a callback function to run when the user clicks the button.</p>
