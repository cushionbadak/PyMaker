<p>Given a variable like:</p>
<pre><code>stories = """adress:address
alot:a lot
athiest:atheist
noone:no one
beleive:believe
fivety:fifty
wierd:weird
writen:written"""
</code></pre>
<p>How would I go about converting it to a nested list of pairs? i.e:</p>
<pre><code>new_stories = [['adress', 'address'], ['alot', 'a lot'], ['athiest', 'atheist'], ['noone', 'no one']] # etc.
</code></pre>
<p>I am currently doing this:</p>
<pre><code>s = stories.split("\n")
</code></pre>
<p>which gives:</p>
<pre><code>['adress:address', 'alot:a lot', 'athiest:atheist', 'noone:no one']  # etc.
</code></pre>
<p>Then I'm not sure what to do or if that's the right step at all.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> to split each line; I'd use the <a href="https://docs.python.org/3/library/stdtypes.html#str.splitlines" rel="nofollow"><code>str.splitlines()</code> method</a> to produce the lines (as that covers more corner cases), and then <code>str.split()</code> to produce the per-line pair:</p>
<pre><code>[s.split(':') for s in stories.splitlines()]
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; stories = """adress:address
... alot:a lot
... athiest:atheist
... noone:no one
... beleive:believe
... fivety:fifty
... wierd:weird
... writen:written"""
&gt;&gt;&gt; [s.split(':') for s in stories.splitlines()]
[['adress', 'address'], ['alot', 'a lot'], ['athiest', 'atheist'], ['noone', 'no one'], ['beleive', 'believe'], ['fivety', 'fifty'], ['wierd', 'weird'], ['writen', 'written']]
</code></pre>
</div>
<span class="comment-copy">Split each item in the list again using <code>:</code> as the delimeter and then add them to <code>new_stories</code> using <code>append()</code>.</span>
