<p>If I use Pythons sum function, I get an "unsupported operand type(s)" error, though adding <code>c1 = Counter(..), c2 = Counter(..)</code> like this: <code>c1+c2</code> works.</p>
</div>
<div class="post-text" itemprop="text">
<p>Always read the full error:</p>
<pre><code>TypeError: unsupported operand type(s) for +: 'int' and 'Counter'
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum</code></a> takes a starting value to sum from, which by default is the integer <code>0</code>. You need to specify a starting value of type <code>Counter</code>, such as the empty counter:</p>
<pre><code>sum([c1, c2, c3], Counter())
</code></pre>
<p>Or alternatively, spell it:</p>
<pre><code>reduce(operator.add, [c1, c2, c3])
</code></pre>
</div>
