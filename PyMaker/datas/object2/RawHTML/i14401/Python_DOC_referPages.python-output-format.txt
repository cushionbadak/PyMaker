<p><a href="http://pythonhosted.org/psutil/#psutil.disk_partitions" rel="nofollow"><code>psutil.disk_partitinos()</code></a> gives you a list of partitions on your systme.</p>
<p>Each element in that list is instance of <a href="https://github.com/giampaolo/psutil/blob/08964b08700fcfc5a4917a01107009a3ee913341/psutil/_common.py#L204" rel="nofollow"><code>sdiskpart</code></a> which is a <a href="https://docs.python.org/2/library/collections.html#collections.namedtuple" rel="nofollow"><code>namedtuple</code></a> with the following properties:</p>
<pre><code>['count', 'device', 'fstype', 'index', 'mountpoint', 'opts']
</code></pre>
<p>You <em>will have to</em> process this list and format and display it the way you <em>want</em> using <a href="https://docs.python.org/2/library/string.html#string.Formatter" rel="nofollow"><code>str.format()</code></a> and <a href="https://docs.python.org/2/library/functions.html#print" rel="nofollow"><code>print()</code></a>.</p>
<p>Please refer to the <a href="http://pythonhosted.org/psutil/" rel="nofollow">psutil</a> documentation.</p>
<p>A simple function that displays "disk information" in a "better way" could be something as simple as:</p>
<p><strong>Example:</strong></p>
<pre><code>from psutil import disk_partitions


def diskinfo():
    for i, disk in enumerate(disk_partitions()):
        print "Disk #{0:d} {1:s}".format(i, disk.device)
        print " Mount Point: {0:s}".format(disk.mountpoint)
        print " File System: {0:s}".format(disk.fstype)
        print " Options: {0:s}".format(disk.opts)


diskinfo()
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>bash-4.3# python /app/foo.py
Disk #0 /dev/mapper/docker-8:1-2762733-bdb0f27645efd726d69c77d0cd856d6218da5783b2879d9a83a797f8b896b4be
 Mount Point: /
 File System: ext4
 Options: rw,relatime,discard,stripe=16,data=ordered
</code></pre>
