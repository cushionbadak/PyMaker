<div class="post-text" itemprop="text">
<p>I want to check if a question mark or a dot are in the last position of a word. So doing something like:</p>
<pre><code>if ".","?" in word[-1]:
</code></pre>
<p>Or:</p>
<pre><code>if [".","?"] in word[-1]:
</code></pre>
<p>Result in an invalid syntax. What is the right way to do this? (preferably without regex)</p>
</div>
<div class="post-text" itemprop="text">
<p>You got the order backwards.</p>
<pre><code>if word[-1] in ".?":
</code></pre>
<p>Refer to the <code>in</code> operator in the <a href="http://docs.python.org/3/library/stdtypes.html#sequence-types-list-tuple-range" rel="nofollow">documentation</a></p>
</div>
<div class="post-text" itemprop="text">
<p>For this <em>particular</em> case, where you want to check the end:</p>
<pre><code>if word.endswith(('.','?')):
</code></pre>
</div>
