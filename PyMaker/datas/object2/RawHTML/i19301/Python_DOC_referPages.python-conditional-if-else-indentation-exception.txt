<div class="post-text" itemprop="text">
<p><code>if</code>, <code>else</code>, <code>for</code>, <code>def</code> etc. <strong>must</strong> be followed by a code block. A comment doesn't count. This is what the <a href="http://docs.python.org/3/tutorial/controlflow.html#pass-statements" rel="nofollow"><code>pass</code> statement</a> is for:</p>
<pre><code>if not item["price"]:
    pass
else:
    items.append(item)
</code></pre>
<p>That said, why not just invert the condition?</p>
<pre><code>if item["price"]:
    items.append(item)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Here's the problem:</p>
<pre><code>    if not items ["price"]:
        #do nothing
    else:
        items.append(item)
</code></pre>
<p>...you can't have just a comment where indented statements are expected. Python has a keyword that's for exactly this case -- <code>pass</code>:</p>
<pre><code>    if not items ["price"]:
        #do nothing
        pass
    else:
        items.append(item)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If the if does nothing and you're only using the else you probably want to use its negation. If you are just testing, you  need to add a <code>pass</code> into it:</p>
<pre><code>if True:
  pass  # just a comment here does not work
else:
  dostuff()
</code></pre>
