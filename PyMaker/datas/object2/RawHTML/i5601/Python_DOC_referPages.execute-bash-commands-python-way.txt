<p>Possible duplicate of <a href="https://stackoverflow.com/questions/4256107/running-bash-commands-in-python">this question</a>.</p>
<p>It is recommended to use <code>subprocess</code> module instead. <code>os.system</code> has been depreciated in favor of <code>subprocess</code>.</p>
<pre><code>import subprocess

command = 'sudo add-apt-repository ppa:ondrej/php')
process = subprocess.Popen(command.split(), stdout=subprocess.PIPE)
output, error = process.communicate()
</code></pre>
