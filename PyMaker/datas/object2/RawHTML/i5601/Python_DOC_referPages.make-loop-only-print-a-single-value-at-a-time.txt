<div class="post-text" itemprop="text">
<p>In <code>python 3.3</code>, you can do like below</p>
<pre><code>print(c , end="\r", flush=True)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use carriage return (<code>"\r"</code>) to do the trick:</p>
<pre><code>import time

c = 5
for i in range(10):
    print(c, end='\r')
    c += 1
    time.sleep(1)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>To print text with a space instead of a newline at the end you can use the <code>end</code> parameter.</p>
<pre><code>print("text", end=" ")
</code></pre>
<p>in python2 this can be achieved with a trailing comma:</p>
<pre><code>print "text",
</code></pre>
<p>for Example,</p>
<pre><code>c = 5
for i in range(10):
    print c,
    c += 1
</code></pre>
</div>
