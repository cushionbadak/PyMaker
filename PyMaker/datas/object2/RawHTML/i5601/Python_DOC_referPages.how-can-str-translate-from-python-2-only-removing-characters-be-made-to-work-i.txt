    for file_name in file_list:
        os.rename(file_name, file_name.translate(None, "0123456789"))

<div class="post-text" itemprop="text">
<p>You are confusing the Python 2 <a href="https://docs.python.org/2/library/stdtypes.html#str.translate" rel="nofollow noreferrer"><code>str.translate()</code> method</a> with the <a href="https://docs.python.org/3/library/stdtypes.html#str.translate" rel="nofollow noreferrer">Python 3 version</a> (which is really the same as <code>unicode.tranlate()</code> in Python 2).</p>
<p>You can use the <a href="https://docs.python.org/3/library/stdtypes.html#str.maketrans" rel="nofollow noreferrer"><code>str.maketrans()</code> static method</a> to create a translation map; pass your string of digits to be removed in as the <em>third</em> argument to that function:</p>
<pre><code>file_name.translate(str.maketrans('', '', '0123456789'))
</code></pre>
<p>Better still, store the result of <code>str.maketrans()</code> outside of the loop, and re-use it:</p>
<pre><code>no_digits = str.maketrans('', '', '0123456789')

for file_name in file_list:
    os.rename(file_name, file_name.translate(no_digits))
</code></pre>
