<p>Let's say I have the following code</p>
<pre><code>arr = ["a", "b", "c", "d"...]
</code></pre>
<p>I don't know how many items there may be, but I know it'll be a multiple of two. How would I go about appending every pair of items together, leaving me with</p>
<p><code>arr = ["ab", "cd"...]</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can try this:</p>
<pre><code>arr = ["a", "b", "c", "d"]
new_data = [''.join([arr[i], arr[i+1]]) for i in range(0, len(arr), 2)]
</code></pre>
<p>Output:</p>
<pre><code>['ab', 'cd']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Another way would be using the <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> clustering idiom</p>
<pre><code>def pairs(it):
    return [''.join(x) for x in zip(*[iter(it)]*2)]
</code></pre>
</div>
