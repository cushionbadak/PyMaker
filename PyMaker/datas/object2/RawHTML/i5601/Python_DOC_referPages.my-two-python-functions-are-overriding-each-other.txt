<p><code>re.match</code> looks for the characters at the BEGINNING of the given string. Use <code>re.search</code> instead. </p>
<p>See also <a href="https://docs.python.org/3.5/library/re.html#search-vs-match" rel="nofollow noreferrer">search() vs. match()</a>.</p>

<h2>Brief</h2>
<p>You don't even need regex for this. Python has <a href="https://docs.python.org/3/library/stdtypes.html#str.isupper" rel="nofollow noreferrer"><code>isupper()</code></a> and <a href="https://docs.python.org/3/library/stdtypes.html#str.islower" rel="nofollow noreferrer"><code>islower()</code></a> functions available.</p>
<hr/>
<h2>Code</h2>
<p><a href="https://ideone.com/tsia7M" rel="nofollow noreferrer">See code in use here</a></p>
<pre><code>passwords = [
    "all lowercase letters",
    "ALL UPPERCASE LETTERS",
    "Mixed lowercase and UPPERCASE letters"
]

def lower_score(string, weight):
    if any(s.islower() for s in string):
        print "Contains: LOWERCASE"
        return weight
    else:
        print "Message: The password needs lowercase letters for a higher score"
    return 0

def upper_score(string, weight):
    if any(s.isupper() for s in string):
        print "Contains: UPPERCASE"
        return weight
    else:
        print "Message: The password needs capital letters for a higher score"
    return 0

for password in passwords:
    print "Password: " + password
    score = 0
    score += lower_score(password, 5)
    score += upper_score(password, 5)
    print "Score: " + str(score) + "\n"
