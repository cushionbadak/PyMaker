<div class="post-text" itemprop="text">
<p>I have a list of lists:</p>
<pre><code>a = [['0.06'], ['0.54'], ['0.61']] 
</code></pre>
<p>I want to convert this list into normal list like this:</p>
<pre><code>a = [0.06, 0.54, 0.61]
</code></pre>
<p>How can I do it it</p>
</div>
<div class="post-text" itemprop="text">
<p>Using a list comprehension:</p>
<pre><code>&gt;&gt;&gt; a = [['0.06'], ['0.54'], ['0.61']]
&gt;&gt;&gt; [float(x) for [x] in a]
[0.06, 0.54, 0.61]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Alternatively, using a functional approach:</p>
<pre><code>list(map(lambda x: float(x[0]), a))
# [0.06, 0.54, 0.61]
</code></pre>
<p>Using <code>itertools</code> to first merge the sublists:</p>
<pre><code>import itertools as it


list(map(float, it.chain.from_iterable(a)))
# [0.06, 0.54, 0.61]
</code></pre>
<p>A list comprehension is arguably more elegant and clever.</p>
</div>
