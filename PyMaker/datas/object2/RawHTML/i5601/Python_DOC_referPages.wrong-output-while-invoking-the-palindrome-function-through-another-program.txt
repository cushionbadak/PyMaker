<div class="post-text" itemprop="text">
<p>You are passing the literal <em>string</em> <code>"product"</code>, which is not a palindrome, to your palindrome function:</p>
<pre><code>check_palindrome('product')
</code></pre>
<p>Remove the single quotations and convert to a string:</p>
<pre><code>check_palindrome(str(product))
</code></pre>
<p>to pass the string representing the integer stored in <code>product</code>.</p>
<hr/>
<p>As an aside, here is an easy way to test if a string is a palindrome:</p>
<pre><code>def check_palindrome(s):
    return s == s[::-1]
</code></pre>
<p>which compares the reverse of the string <code>s</code> with itself. It's a palindrome if both are equal.</p>
