<div class="post-text" itemprop="text">
<p>You don't create directly an instance of this class, this class will be an attribute on the Network client. </p>
<p>Example on how to create a Network client:</p>
<ul>
<li><a href="https://docs.microsoft.com/python/api/overview/azure/network" rel="nofollow noreferrer">https://docs.microsoft.com/python/api/overview/azure/network</a></li>
</ul>
<p>Some info about authentication:</p>
<ul>
<li><a href="https://docs.microsoft.com/python/azure/python-sdk-azure-authenticate" rel="nofollow noreferrer">https://docs.microsoft.com/python/azure/python-sdk-azure-authenticate</a></li>
</ul>
<p>Once you have a client, you can access VMSS NICs:</p>
<pre><code>nics = client.network_interfaces.list_virtual_machine_scale_set_network_interfaces(
    resource_group_name,
    virtual_machine_scale_set_name
)
</code></pre>
<p>List operations returns an <a href="https://docs.python.org/3/library/stdtypes.html#iterator-types" rel="nofollow noreferrer">Iterator</a> like subclass. In this case, content are <a href="https://docs.microsoft.com/en-us/python/api/azure.mgmt.network.v2017_09_01.models.networkinterface?view=azure-python" rel="nofollow noreferrer">NetworkInterface</a> instance.</p>
<pre><code>for nic in nics:
    do_something(nic)
</code></pre>
<p>All Iterators syntax are allowed (<code>next()</code>, list-comprehension, <code>list()</code>, etc.)</p>
