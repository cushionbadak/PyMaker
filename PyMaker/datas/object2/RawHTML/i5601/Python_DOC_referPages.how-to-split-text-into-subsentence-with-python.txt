<div class="post-text" itemprop="text">
<p>I want to split a text into subsentence. How can I do that?</p>
<p>For example:</p>
<pre><code>text = "Hi, this is an apple. Hi, that is pineapple."
</code></pre>
<p>The result should be:</p>
<pre><code>['Hi,',
 'this is an apple.',
 'Hi,',
 'that is pineapple.']
</code></pre>
<p>(P.S. I tried with the <code>string.split(r'[,.]')</code>, but it will remove the separators.) </p>
</div>
<div class="post-text" itemprop="text">
<p>You could split on whitespace <code>\s+</code> with <a href="http://www.regular-expressions.info/lookaround.html" rel="nofollow noreferrer">a zero-length look-behind assertion</a> <code>(?&lt;=[,.])</code> for the punctuation.</p>
<pre><code>import re

text = "Hi, this is an apple. Hi, that is pineapple."
subsentence = re.compile(r'(?&lt;=[,.])\s+')

print(subsentence.split(text))
</code></pre>
<p><code>['Hi,', 'this is an apple.', 'Hi,', 'that is pineapple.']</code></p>
</div>

<div class="post-text" itemprop="text">
<p><a href="https://stackoverflow.com/questions/4576077/python-split-text-on-sentences">Related question</a></p>
<p>The <a href="http://www.nltk.org/" rel="nofollow noreferrer">Natural Language Toolkit</a> provides a tokenizer that you can use to split sentences. For example:</p>
<pre><code>&gt;&gt;&gt; import nltk
&gt;&gt;&gt; nltk.download()   # enter "punkt"

&gt;&gt;&gt; import nltk.data
&gt;&gt;&gt; tokenizer = nltk.data.load('tokenizers/punkt/english.pickle')
&gt;&gt;&gt; data = "Hi, this is an apple. Hi, that is pineapple."
&gt;&gt;&gt; data = data.replace(',', '.')
&gt;&gt;&gt; tokenizer.tokenize(data)
['Hi.', 'this is an apple.', 'Hi.', 'that is pineapple.']
</code></pre>
<p>Details of the tokenizer are <a href="http://www.nltk.org/api/nltk.tokenize.html" rel="nofollow noreferrer">documented here</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Here is another possible solution using <a href="https://docs.python.org/3/library/re.html#re.finditer" rel="nofollow noreferrer"><code>re.finditer()</code></a>:</p>
<pre><code>import re

text = "Hi, this is an apple. Hi, that is pineapple."

punct_locs = [0] + [i.start() + 1 for i in re.finditer(r'[,.]', text)]

sentences = [text[start:end].strip() for start, end in zip(punct_locs[:-1], punct_locs[1:])]

print(sentences)
</code></pre>
<p>Which outputs:</p>
<pre><code>['Hi,', 'this is an apple.', 'Hi,', 'that is pineapple.']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Why you are making it too much complex by importing heavy modules , just go with simple and clean method without importing any module :</p>
<pre><code>text = "Hi, this is an apple. Hi, that is pineapple."
for i in text.split('.'):
    if i:
        print(i.strip().split(','))
</code></pre>
<p>output:</p>
<pre><code>['Hi', ' this is an apple']
['Hi', ' that is pineapple']
</code></pre>
<blockquote>
<p>You can do in one line :</p>
</blockquote>
<pre><code>text = "Hi, this is an apple. Hi, that is pineapple."
print([i.strip().split(',') for i in text.split('.') if i])
</code></pre>
<p>output:</p>
<pre><code>[['Hi', ' this is an apple'], ['Hi', ' that is pineapple']]
</code></pre>
