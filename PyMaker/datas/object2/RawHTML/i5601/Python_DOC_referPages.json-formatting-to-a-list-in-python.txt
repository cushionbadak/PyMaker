<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/json.html#json.loads" rel="nofollow noreferrer"><code>json.loads()</code></a> and then iterate on the result</p>
</div>
<div class="post-text" itemprop="text">
<p>If you want data out of a .json file you can load it from a string using<code>json.loads(&lt;string&gt;)</code>. You can load from a file directly via <code>json.load(&lt;filename&gt;)</code>. </p>
<pre><code># Create a list
result = []

while True:
    r = json.loads(requests.get("yourAPI).text)
    result.append(r)
</code></pre>
<p>Now its just a matter of displaying the data to the user.</p>
