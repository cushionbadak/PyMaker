<div class="post-text" itemprop="text">
<p>You want 76 digits right?</p>
<pre><code>from decimal import *
getcontext().prec = 76
Decimal(1)/Decimal(3)
</code></pre>
<p>Output</p>
<pre><code>Decimal('0.3333333333333333333333333333333333333333333333333333333333333333333333333333')
</code></pre>
