<p>If you have a pandas dataframe you could set the display options.</p>
<pre><code>import pandas as pd
import numpy as np

pd.options.display.float_format = '{:.2f}'.format

pd.DataFrame(dict(randomvalues=np.random.random_sample((5,))))
</code></pre>
<p>Returns:</p>
<pre><code>   randomvalues
0          0.02
1          0.66
2          0.24
3          0.87
4          0.63
</code></pre>

<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/string.html#custom-string-formatting" rel="nofollow noreferrer">str.format</a>:</p>
<pre><code>&gt;&gt;&gt; '{:.2g}'.format(3.658940e-11)

'3.7e-11'
</code></pre>
<p>String slicing will not work here, because it does not round the values:</p>
<pre><code>&gt;&gt;&gt; s = '3.658940e-11'
&gt;&gt;&gt; s[:3] + 'e' + s.split('e')[1]

'3.6e-11'
</code></pre>
