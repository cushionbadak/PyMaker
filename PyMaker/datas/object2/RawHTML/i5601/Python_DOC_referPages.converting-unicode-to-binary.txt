<p>I am trying to convert Arabic text to <code>utf8</code> encoded bytes then to binary by using <a href="https://stackoverflow.com/questions/19255832/converting-binary-to-utf-8-in-python">this answer here</a>.</p>
<p>First, I used the code as it is in the example:</p>
<pre><code>'{:b}'.format(int(u'سلام'.encode('utf-8').encode('hex'), 16))

<p>But I got this error:</p>
<pre><code>AttributeError: 'bytes' object has no attribute 'encode'

<p>Also I removed <code>.encode('hex')</code> but still gives the same error.</p>

<p>How about this:</p>
<pre><code>&gt;&gt;&gt; ''.join('{:08b}'.format(b) for b in 'سلام'.encode('utf8'))
'1101100010110011110110011000010011011000101001111101100110000101'
</code></pre>
<p>This iterates over the encoded <code>bytes</code> object, where you get an integer in the range 0..255 for each iteration.
Then the integer is formatted in binary notation with zero padding up to 8 digits.
Then glue everything together with <code>str.join()</code>.</p>
<p>For the inverse, the approach given in an answer from the question you linked to can be adapted to Python 3 as follows (<code>s</code> is the output of the above example, ie. a <code>str</code> of 0s and 1s):</p>
<pre><code>&gt;&gt;&gt; bytes(int(b, 2) for b in re.split('(........)', s) if b).decode('utf8')
'سلام'
</code></pre>
