<p><a href="https://docs.python.org/3/glossary.html#term-eafp" rel="nofollow noreferrer">Easier to ask for forgiveness</a> - try loading with <code>datetime.strptime()</code> and handle a possible <code>ValueError</code> - <code>%Y%m%d%f</code> format should be what you are looking for:</p>
<pre><code>In [1]: from datetime import datetime

In [2]: def validate(date_string):
            try:
                datetime.strptime(date_string, '%Y%m%d%f')
                print('Valid')
            except ValueError:
                print('Invalid')


In [3]: validate('2017122512345')
Valid

In [4]: validate('20171')
Invalid

In [5]: validate('illegal')
Invalid
</code></pre>
