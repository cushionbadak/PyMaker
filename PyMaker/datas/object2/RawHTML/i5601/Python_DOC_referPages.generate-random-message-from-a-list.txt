<p>I'm using this Python code to pick 3 words to form a message from the 5 words randomly.</p>
<pre><code>def randomMessage():    
    word = ['Animal', 'Dog', 'Cat', 'Queen', 'Bird']

    return (random.sample(word, 3))
</code></pre>
<p>and then, in a later part of the code, I want to put this in a message using</p>
<pre><code>" + randomMessage() + "
</code></pre>
<p>but it gives me an error saying "must be str, not a list".</p>
<p>How can I fix this?</p>

<div class="post-text" itemprop="text">
<p>You need to use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><strong><code>str.join</code></strong></a> on the <code>list</code>.</p>
<p>Calling <code>random.sample(word, 3)</code> returns a <code>list</code>, so to convert it to a <code>string</code>, with spaces between every element, you need to do something like:</p>
<pre><code>' '.join(random.sample(word, 3))
</code></pre>
<p>which would give:</p>
<pre><code>&gt;&gt;&gt; ' '.join(random.sample(word, 3))
'Bird Dog Cat'
</code></pre>
<hr/>
<p>So, to complete the answer, you would need to modify the line at the end of your <code>randomMessage()</code> function, making the whole thing:</p>
<pre><code>def randomMessage():
    word = ['Animal', 'Dog', 'Cat', 'Queen', 'Bird']
    return ' '.join(random.sample(word, 3))
</code></pre>
<p>and some tests show that it now returns a <code>string</code>:</p>
<pre><code>&gt;&gt;&gt; randomMessage()
'Animal Queen Bird'
&gt;&gt;&gt; randomMessage()
'Cat Dog Animal'
&gt;&gt;&gt; randomMessage()
'Animal Cat Bird'
&gt;&gt;&gt; randomMessage()
'Bird Dog Animal'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your method returns a list, not a string. The quickest way to fix it</p>
<pre><code>return " ".join(random.sample(word, 3))
</code></pre>
<p>Or you can make it more explicit</p>
<pre><code>def random_message(): 
  words = ['Animal', 'Dog', 'Cat', 'Queen', 'Bird']
  three_words = random.sample(words, 3)
  return " ".join(three_words)
</code></pre>
<p>Note: use this style in Python: <code>random_message</code>, not camelcase <code>randomMessage</code></p>
