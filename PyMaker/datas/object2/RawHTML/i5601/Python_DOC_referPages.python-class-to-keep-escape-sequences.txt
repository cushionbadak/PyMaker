<div class="post-text" itemprop="text">
<p><code>__init__()</code> method is used while creating an object from class. If you do not want to create any objects you do not have to define it. However, if you only want to it use to separate namespaces you could also simply move it to separate module.</p>

<div class="post-text" itemprop="text">
<p>This would be the pythonic way to assign variables inside the class and access them outside.</p>
<pre><code>class Color:
  def __init__(self):
    self.HEAD = '\033[95m'
    self.OKBL = '\033[94m'
    self.OKGR = '\033[92m'
    self.FAIL = '\033[93m'
    self.WARN = '\033[91m'
    self.ENDC = '\033[0m'
    self.BOLD = '\033[1m'
    self.UNDR = '\033[4m'

color = Color()
print(color.WARN + "Output file will be overwritten!" + color.ENDC)
</code></pre>
