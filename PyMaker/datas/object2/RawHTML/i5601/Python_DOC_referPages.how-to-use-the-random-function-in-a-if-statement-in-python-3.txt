<div class="post-text" itemprop="text">
<p>Like the name <code>randint</code> already suggests, <code>random.randint(1,2)</code> <strong>returns an integer</strong> between 1 and 2 (both inclusive).</p>
<p>If you perform <code>1 == '1'</code> you <strong>compare an <code>int</code> with a <code>str</code></strong>, which is always <code>False</code>. You should compare with <code>1</code> instead of <code>'1</code>'.</p>
<p>You can also drop the <code>elif</code>, and use <code>else</code>, since we know that it will be <code>2</code> if it is not <code>1</code>, so:</p>
<pre><code>input("Heads or Tails:")
rand = random.randint(1,2)  # no brackets necessary

if rand == <b>1</b>:  # use an integer, instead of a string:
    print("Heads Wins!")
<b>else</b>:  # else instead of elif
    print("Tails Wins!")</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>random.randint(1,2)</code> returns the value of integer type. <a href="https://docs.python.org/3/library/random.html#random.randint" rel="nofollow noreferrer"><code>random.randint(a,b)</code></a> document clearly says that it:</p>
<blockquote>
<p><strong>Returns a random integer N</strong> such that a &lt;= N &lt;= b.</p>
</blockquote>
<p>But in your <code>if</code> you are comparing it against the value of string type, and hence your conditional statements are failing.</p>
<p>You need to modify your conditional statements as:</p>
<pre><code>#          v no quotes here
if rand == 1:
    print("Heads Wins!")
#            v no quotes here
elif rand == 2 :   # Better to simply use `else` 
    print("Tails Wins!")
</code></pre>
