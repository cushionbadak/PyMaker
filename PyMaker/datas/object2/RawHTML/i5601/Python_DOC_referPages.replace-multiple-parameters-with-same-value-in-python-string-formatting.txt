<p>I have a huge string that looks like this:</p>
<pre><code>100hello100string100formatting ...
</code></pre>
<p>Furthermore, <code>100</code> is a dynamic value. In the original string, there are 60 occurrences of <code>100</code>. How do I format the string so that I can pass the parameter only once?</p>
<p>I am currently doing the following:</p>
<pre><code>"%dhello%dstring%dformatting%d ..." % (100, 100, 100, ...)
</code></pre>
<p>Is there any way to pass the value <code>100</code> only once so that it will take the same value for all other parameters?</p>

<p>Try string <a href="https://www.programiz.com/python-programming/methods/string/format" rel="nofollow noreferrer">formatting</a> (<a href="https://www.python.org/dev/peps/pep-3101/" rel="nofollow noreferrer">see also PEP</a>):</p>
<pre><code>"{0}hello{0}string{0}formatting".format(100)
# '100hello100string100formatting'
</code></pre>

<div class="post-text" itemprop="text">
<p>I suggest using <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>str.join</code></a>.</p>
<p>If you have 60 occurrences of <code>100</code>, that also means that you have 60 words in your string. Most likely you got those words from somewhere, for instance from a file, or from a database or from user input.</p>
<p>Assuming that you can get all of those words into a list, why not just join the elements of the list, instead of going to the trouble of typing all the words out?</p>
<pre><code>&gt;&gt;&gt; words = ['hello', 'string', 'formatting']
&gt;&gt;&gt; '100'.join(words)
hello100string100formatting
</code></pre>
<p>For this to work, your dynamic value <code>100</code> must be a string, so if it's stored as a number, you'll need to convert it first:</p>
<pre><code>&gt;&gt;&gt; num = 100
&gt;&gt;&gt; str(num).join(words)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>"%dhello%dstring%dformatting" % (*[100]*3,)</code></p>
<p>Basically, you create a temp list containing the values you need (<code>[100]</code>), multiply that by how many times you need it (<code>*3</code>) and prepend that with <code>*</code> so that you unpack the list</p>
</div>
