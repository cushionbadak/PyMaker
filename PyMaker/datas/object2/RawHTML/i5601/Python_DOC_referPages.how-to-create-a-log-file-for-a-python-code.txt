<p>How to create a log file for a python code ?
I have created an .exe file of a python code I need to save the exceptions into a separate log file.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the logging module in python for this. This helps you to log the timestamp along with the log message.</p>
<pre><code>import logging
LOG_FILENAME = 'exception.log'
logging.basicConfig(filename=LOG_FILENAME,level=logging.ERROR)

logging.error('This message should go to the log file')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Open a file in your code</p>
<pre><code>file = open('log.dat', 'w+')
</code></pre>
<p>When ever you need to write log, write to that file using file.write()</p>
