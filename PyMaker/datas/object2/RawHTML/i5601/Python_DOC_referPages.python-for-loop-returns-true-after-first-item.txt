        else:

I planned to use it as a 'checker' to see if all input values of another function contained only digits.</p>
<p>Problem: If the first item in a_list is not a number, checker returns False as it should. However, if the first item is a number while any others in the list aren't, checker returns True anyways. This causes the next function to proceed with the non-number variables and results in an error.</p>
<p>How do I make it so my function checks the entire list before returning True?

<p>There are usefull built-in fuctions <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer">all</a> (and <code>any</code>) for checking multiple conditions:</p>
<pre><code>def checker(a_list):
    return all(str(item).isdigit() for item in a_list)
</code></pre>

<div class="post-text" itemprop="text">
<p>Don't return True in the loop. In the loop return False if an item is NOT <em>a digit</em>. Move <code>return True</code> after the loop has completed.</p>
<pre><code>def checker(a_list):
    for item in a_list:
        if not str(item).isdigit():
            return False
    return True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I'm assuming you want to check that all elements of <code>a_list</code> return <code>True</code> as a return value from <code>isdigit()</code>.</p>
<p>In this case, use the built-in function <code>all</code> </p>
<p><code>all(str(s).isdigit() for s in a_list)</code></p>
<p>for more information on <code>any</code> and <code>all</code>, check out this link on SO: <a href="https://stackoverflow.com/questions/19389490/how-do-pythons-any-and-all-functions-work" title="any and all explained">any and all explained</a></p>
