<span class="comment-copy"><a href="https://docs.python.org/3/library/stdtypes.html#string-methods" rel="nofollow noreferrer">From the docs</a> on <code>.split()</code> <i>"Return a list of the words in the string"</i>. You, I hope, know what a list is. You're trying to convert <code>['1','2']</code> to a int. <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow noreferrer">From the docs on <code>int()</code></a> <i>"Return an integer object constructed from a number or string x"</i>. You've passed in a list which is invalid, hence the error. A list is an iterable so you could iterate through the list converting each element of the list to an integer.</span>
<span class="comment-copy">You can use this:<code>D = list(map(int, input.split(',')))</code> to take multiple integers as input</span>
<span class="comment-copy">you need <code>list(map(int, input().split(",")))</code></span>
