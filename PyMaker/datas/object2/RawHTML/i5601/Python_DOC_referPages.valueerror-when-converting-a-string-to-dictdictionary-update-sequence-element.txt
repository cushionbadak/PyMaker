<p>I'm trying to convert a string to a dictionary but I keep getting this error:<br/>
<code>ValueError: dictionary update sequence element #0 has length 1; 2 is required</code><br/>
Here's my code:      </p>
<pre><code>string = r'{"extra":[{"text":"Hello World"}]}'

</code></pre>

<div class="post-text" itemprop="text">
<p>I believe you are looking for:</p>
<pre><code>import json
string = r'{"extra":[{"text":"Hello World"}]}'
json_data = json.loads(string)
</code></pre>
<p>There are a few JSON parsing libraries. Default one should suffice, simplejson is another option (just to give you an alternative).</p>
</div>
<span class="comment-copy">You can't just load a JSON text just like that. You need the json parser: <a href="https://docs.python.org/3/library/json.html" rel="nofollow noreferrer">docs.python.org/3/library/json.html</a> (particularly the <code>loads</code> function to decode the string into a Python <code>dict</code>)</span>
