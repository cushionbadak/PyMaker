<div class="post-text" itemprop="text">
<p>You cannot put tuple in the function definition as a parameter. Check <a href="https://www.learnpython.org/en/Multiple_Function_Arguments" rel="nofollow noreferrer">Multiple Function Arguments</a> or <a href="https://docs.python.org/3/reference/compound_stmts.html#function-definitions" rel="nofollow noreferrer">8.6. Function definitions</a> in the Python language reference.</p>
<p>Try something like this this:</p>
<pre><code>def twovectors(vector1, vector2):
    velocity1, length1 = vector1
    velocity2, length2 = vector2
    # Other code...
</code></pre>
<p>I used <a href="https://docs.python.org/3/tutorial/datastructures.html#tuples-and-sequences" rel="nofollow noreferrer">tuple unpacking</a> to expand provided tuple arguments.</p>
</div>
