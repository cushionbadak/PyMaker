<div class="post-text" itemprop="text">
<p>In Python 3 the line</p>
<pre><code>except Exception, e:
</code></pre>
<p><a href="https://stackoverflow.com/q/2535760/35070">must be written as</a></p>
<pre><code>except Exception as e:
</code></pre>
<p>Likely, if you want your module to be published, you should also give it a more descriptive name than <code>config</code>.</p>
