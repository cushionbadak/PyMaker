<div class="post-text" itemprop="text">
<p>To answer your question:</p>
<pre><code>import urllib

link = "http://www.somesite.com/details.pl?urn=2344"
f = urllib.urlopen(link)
myfile = f.read()
print(myfile)
</code></pre>
<p>You need to <code>read()</code>, not <code>readline()</code></p>
<p>EDIT (2018-06-25): Since Python 3, the legacy <code>urllib.urlopen()</code> was replaced by <code>urllib.request.urlopen()</code> (see notes from <a href="https://docs.python.org/3/library/urllib.request.html#urllib.request.urlopen" rel="noreferrer">https://docs.python.org/3/library/urllib.request.html#urllib.request.urlopen</a> for details).</p>
<p>If you're using Python 3, see answers by Martin Thoma or i.n.n.m within this question:
<a href="https://stackoverflow.com/a/28040508/158111">https://stackoverflow.com/a/28040508/158111</a> (Python 2/3 compat)
<a href="https://stackoverflow.com/a/45886824/158111">https://stackoverflow.com/a/45886824/158111</a> (Python 3)</p>
<p>Or, just get this library here: <a href="http://docs.python-requests.org/en/latest/" rel="noreferrer">http://docs.python-requests.org/en/latest/</a> and seriously use it :)</p>
<pre><code>import requests

link = "http://www.somesite.com/details.pl?urn=2344"
f = requests.get(link)
print(f.text)
</code></pre>

<div class="post-text" itemprop="text">
<p>A solution with works with Python 2.X and Python 3.X makes use of the Python 2 and 3 compatibility library <a href="https://pythonhosted.org/six/" rel="noreferrer"><code>six</code></a>:</p>
<pre><code>from six.moves.urllib.request import urlopen
link = "http://www.somesite.com/details.pl?urn=2344"
response = urlopen(link)
content = response.read()
print(content)
</code></pre>
</div>
