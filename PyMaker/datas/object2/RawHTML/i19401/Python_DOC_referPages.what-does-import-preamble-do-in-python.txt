<p><code>_preamble</code> is a module like any other. In twisted's case, this is simply <a href="http://twistedmatrix.com/trac/browser/trunk/bin/_preamble.py" rel="noreferrer">the module</a> that sets up <a href="http://docs.python.org/3/library/sys.html#sys.path" rel="noreferrer"><code>sys.path</code></a> so that you can run twisted in development setups.</p>
