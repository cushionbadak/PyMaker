<div class="post-text" itemprop="text">
<p>On python 3, <code>struct.unpack()</code> expects an object that implements the buffer protocol, such as a <a href="http://docs.python.org/3/library/stdtypes.html#bytes" rel="nofollow"><code>bytes</code></a> value, not a <a href="http://docs.python.org/3/library/stdtypes.html#text-sequence-type-str" rel="nofollow">unicode <code>str</code></a>:</p>
<pre><code>&gt;&gt;&gt; import struct
&gt;&gt;&gt; s = b'\x00\x00\x00\x01\x00\x00\x00\xff\xff\x00\x00'
&gt;&gt;&gt; struct.unpack('11B', s)
(0, 0, 0, 1, 0, 0, 0, 255, 255, 0, 0)
</code></pre>
<p>If you are reading this data from a file, open the file in binary mode instead of text mode to get bytes.</p>
