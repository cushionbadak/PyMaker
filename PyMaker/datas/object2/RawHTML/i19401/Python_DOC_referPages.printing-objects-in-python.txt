<div class="post-text" itemprop="text">
<p>The <code>&lt;__main__.Cars object at 0x027FB970&gt;</code> is the standard string for custom objects that do not implement their own <a href="http://docs.python.org/3/reference/datamodel.html#object.__str__" rel="nofollow"><code>.__str__()</code> hook</a>. You can customize it by implementing that method:</p>
<pre><code>class Cars:
    # ....

    def __str__(self):
        return 'Car instance with variables: {!r}'.format(self.variables)
</code></pre>
