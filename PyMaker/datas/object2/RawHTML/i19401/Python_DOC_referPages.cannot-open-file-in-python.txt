<div class="post-text" itemprop="text">
<p><a href="http://docs.python.org/3/library/os.html?highlight=chdir#os.chdir" rel="nofollow"><code>os.chdir</code></a> does not tell you what is in the directory, it changes the current working path. All relative file path you give are relative to the working dir.</p>
<p>You should either</p>
<ul>
<li>remove the <code>os.chdir("../result/files/")</code> and call os.listdir("../result/files/") or</li>
<li>not add <code>"../result/files/"</code> to the filename you want to open.</li>
</ul>
</div>
<div class="post-text" itemprop="text">
<p>Do</p>
<pre><code>with open(fileName, 'rb') as csvFile:
</code></pre>
<p>instead of looking relative to the directory you've changed to.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could try using <code>os.path.abspath()</code> to find out what actual path Python is trying to use.</p>
