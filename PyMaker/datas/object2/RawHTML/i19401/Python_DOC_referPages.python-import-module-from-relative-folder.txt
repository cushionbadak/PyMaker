<div class="post-text" itemprop="text">
<p>Importing the module with a simple <code>import</code> statement does <em>not</em> copy the names from that module into your own global namespace.</p>
<p>Either refer to the <code>main</code> name through attribute access:</p>
<pre><code>print(lib.config.main)
</code></pre>
<p>or use the <code>from ... import ...</code> syntax:</p>
<pre><code>from lib.config import main
</code></pre>
<p>instead.</p>
<p>You can learn more about how importing works in the <a href="http://docs.python.org/3/tutorial/modules.html" rel="nofollow">Modules section of the Python tutorial</a>.</p>
