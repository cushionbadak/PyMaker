<p>So say I have a zip file named "files.zip"
it contains
"text1.txt":</p>
<pre><code>words
</code></pre>
<p>and 
"text2.txt":</p>
<pre><code>other words
</code></pre>
<p>How do I tell python to open and read the text1.txt file? I know that usually to open a text file outside of a zip file I would just do this:</p>
<pre><code>file = open('text1.txt','r')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/zipfile.html" rel="noreferrer">zipfile</a> module like so:</p>
<pre><code>zip = zipfile.ZipFile('test.zip')
file = zip.read('text1.txt')
</code></pre>
<p>Don't forget to import zipfile module: <code>import zipfile</code></p>
