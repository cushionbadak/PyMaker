<div class="post-text" itemprop="text">
<p>In python 3 (2.6+) you should use <code>str.format()</code> and not the "%-systax".</p>
<pre><code>format_str = '{name:30} {address:08x} {size:8d}'
print(format_str.format(name='Name', address='Address', size='Size')
print(format_str.format(name=names[i], address=addresses[i], size=sizes[i]))
</code></pre>
<p>For more infos about the <code>format()</code> method look <a href="http://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow">here</a></p>
</div>
