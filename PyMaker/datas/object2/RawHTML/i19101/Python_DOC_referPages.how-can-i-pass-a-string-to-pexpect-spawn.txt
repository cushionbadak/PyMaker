<div class="post-text" itemprop="text">
<p>I want to ssh to another node on my network as part of a larger python script, I am using pexpect which works when I do something like this:</p>
<pre><code>session=spawn('ssh root@172.16.210.254')
</code></pre>
<p>I want to replace the address with a variable so I can cycle through addresses in a list however when I try:</p>
<pre><code>address = "172.16.210.253"

session=spawn('ssh root@'address)
</code></pre>
<p>It doesn't work as using address in this way is invalid syntax. What is the correct syntax for this?</p>
</div>
<div class="post-text" itemprop="text">
<p><code>session=spawn('ssh root@' + address)</code>  to concatenate the strings</p>
</div>
<span class="comment-copy">Or, more generally, <code>'ssh root@{}'.format(address)</code>, which allows you to insert a variable anywhere in a string, instead of just at the end. See <a href="http://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow noreferrer"><code>str.format</code></a> and <a href="http://docs.python.org/3/library/string.html#formatstrings" rel="nofollow noreferrer">Format String Syntax</a> for more details.</span>
