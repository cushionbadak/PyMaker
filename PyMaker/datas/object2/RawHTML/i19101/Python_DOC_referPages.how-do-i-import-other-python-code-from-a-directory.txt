<p>In <code>example.py</code> I just tried:</p>

</code></pre>

<div class="post-text" itemprop="text">
<p>Do you have a <code>__init__.py</code> file in <code>root/child/</code> directory? After creating this file you should be able to do this:</p>
<pre><code>import child.config
</code></pre>
<p>or </p>
<pre><code>from child import config
</code></pre>
<p>You can also import multiple modules from child directory like this:</p>
<pre><code>from child import first, second, third
</code></pre>
<p>Read about modules and packages <a href="http://docs.python.org/3/tutorial/modules.html#packages" rel="nofollow">here</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you want to import <code>config.py</code> with importing <code>child</code> you need to define <code>child</code> as a package.</p>
<p>To do so you need to create an <code>__init__.py</code> file in your <code>child</code> directory.</p>
<p>Check this about <a href="http://docs.python.org/2/tutorial/modules.html#packages" rel="nofollow">packages</a></p>
</div>
<div class="post-text" itemprop="text">
<p>Your directory should have <strong>init</strong>.py so that python will understand it is a package. So the directory structure would be like </p>
<pre><code>root
     __init__.py
     child
         __init__.py
         config.py

example.py
import root.child
</code></pre>
<p>note, you should import <code>root.child</code> not <code>child</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Create an empty <code>__init__.py</code> file in same directory with <code>config.py</code>. This is required for importing files like a package.</p>
<p>Then you can import it.</p>
</div>
