<div class="post-text" itemprop="text">
<p>What you need is <a href="http://docs.python.org/3/library/functions.html#max" rel="nofollow"><code>max()</code></a> function.</p>
<p>So, having that, you'll have the following program:</p>
<pre><code>def input_max3():
    return max(int(input()) for i in range(3))
</code></pre>
<p>Here <code>input()</code> reads a string from the user and <code>int(...)</code> converts it to integer value. To have 3 values we have an generator expression <code>(... for i in range(3))</code> and <code>max(...)</code> takes maximum out of the values generated.</p>
<p>Or, if you don't need any command-line input from the user, you might just use it as it is: <code>max(1, 3, 5)</code>.</p>
