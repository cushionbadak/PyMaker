<div class="post-text" itemprop="text">
<p>First of all, <code>re.match()</code> returns a <em><code>MatchObject</code></em> instance, not an integer. The expression</p>
<pre><code>re.match('^-a$|^-b$|^-c$',argv) != 1
</code></pre>
<p>will <em>always</em> be True. Just use <code>not</code>:</p>
<pre><code>if not re.match('^-a$|^-b$|^-c$', argv):
</code></pre>
<p>and:</p>
<pre><code>elif re.match('-help', arg):
</code></pre>
<p>You need to remove the <code>,</code> from the print statement; the <code>%</code> operator acts on the string on the left hand side, but that does mean there should not be a comma in between:</p>
<pre><code>print ("file %s doesnot exists" % arg)
</code></pre>
<p>It looks as if you are trying to parse command line arguments. You could use the <a href="http://docs.python.org/3/library/argparse.html" rel="nofollow"><code>argparse</code> module</a> instead for that.</p>
