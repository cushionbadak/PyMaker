<p>How can I make variable using non English character.
for example if I use Japanese character like this :</p>
<pre><code>あ = 10
</code></pre>
<p>I get this error :</p>
<pre><code>Unsupported characters in input
</code></pre>
<p>or if I use Arabic character like this:</p>
<pre><code>أ = 'Something'
</code></pre>
<p>I get another error :</p>
<pre><code>SyntaxError: invalid syntax
</code></pre>
<p>Note : When I make variable using non English character in <code>php</code> it worked but in Python it does not work.
So How can I make variable using non English character in python ?</p>
</div>
<div class="post-text" itemprop="text">
<p>You cannot, at least not in Python 2. You can <em>only</em> use ASCII letters, digits and underscores in Python names (with digits only allowed as long as they are not the first character).</p>
<p>See the <a href="http://docs.python.org/2/reference/lexical_analysis.html#identifiers" rel="nofollow">Identifiers and keywords</a> section of the documentation:</p>
<blockquote>
<pre><code>identifier ::=  (letter|"_") (letter | digit | "_")*
letter     ::=  lowercase | uppercase
lowercase  ::=  "a"..."z"
uppercase  ::=  "A"..."Z"
digit      ::=  "0"..."9"
</code></pre>
</blockquote>
<p>Python 3 <a href="http://docs.python.org/3/reference/lexical_analysis.html#identifiers" rel="nofollow">has expanded what is permissible</a>. Anything that is considered a Unicode letter or digit is allowed, plus underscores (and numbers not at the start).</p>
</div>
<span class="comment-copy">Different programming languages have different rules for variable names.</span>
