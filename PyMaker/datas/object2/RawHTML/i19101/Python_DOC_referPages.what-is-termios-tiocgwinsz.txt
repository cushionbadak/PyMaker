<div class="post-text" itemprop="text">
<p>I want to get the size of the terminal.  I am using this functionality:</p>
<pre><code>import sys, struct, fcntl, termios

s = struct.pack('HHHH', 0, 0, 0, 0)
t = fcntl.ioctl(sys.stdout.fileno(), termios.TIOCGWINSZ, s)
print(struct.unpack('HHHH', t))
</code></pre>
<p>But what on earth is <code>termios.TIOCGWINSZ</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p>It is a magic constant determined by the system you are running on resp. by the terminal driver.</p>
<p>In combination with <code>ioctl()</code>, it serves to tell exectly what you want, in your case call IOCtl to Get the Window Size. Thus the name <code>TIOCGWINSZ</code>, <code>IOC</code>tl to <code>G</code>et the <code>WIN</code>dow <code>S</code>i<code>Z</code>e.</p>
<p><a href="http://www.delorie.com/djgpp/doc/libc/libc_495.html">This bit of documentation</a> might help you clear things up.</p>
</div>
