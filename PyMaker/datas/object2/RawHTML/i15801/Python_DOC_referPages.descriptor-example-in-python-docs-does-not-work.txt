<p>I think you've found a bug in the documentation. You can't pass (to my knowledge) arguments to getter-type properties. I think the example should be:</p>
<pre><code>class Cell(object):
    def __init__(self, cell):
        self.cell = cell

    def recalc(self):
        self._value = 100 # this wouldn't really return a static value

    def getvalue(self):
        "Recalculate cell before returning value"
        self.recalc()
        return self._value
    value = property(getvalue)

&gt;&gt;&gt; Cell('A1').value
100
</code></pre>
