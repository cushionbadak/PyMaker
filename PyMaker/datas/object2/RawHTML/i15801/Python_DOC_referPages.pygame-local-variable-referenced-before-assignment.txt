<div class="post-text" itemprop="text">
<p>The name <code>x</code> is in the global scope.  Therefore, in order to modify its value inside the function <code>main</code>, you need to declare it as being global with <a href="https://docs.python.org/3/reference/simple_stmts.html#the-global-statement" rel="nofollow"><code>global</code></a>:</p>
<pre><code>def main():
    global x
    ...
        x += movex
</code></pre>
<p>Note that you only need to do this if you <em>modify</em> a global.  Accessing their values works just fine.</p>
