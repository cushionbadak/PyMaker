<div class="post-text" itemprop="text">
<p>I know I can do:</p>
<pre><code>if 'hello' in 'hello world':
</code></pre>
<p>What if I have a several needles like ('.css', '.js', '.jpg', '.gif', '.png', '.com') and I want to check if either of these is in a string.</p>
<p>(note: <code>endswith</code> won't do it in this case, they may not be suffixes)</p>
</div>
<div class="post-text" itemprop="text">
<p>You may find <a href="https://docs.python.org/3/library/functions.html#any"><code>any</code></a> useful:</p>
<pre><code>haystack = 'hello world'
needles = ['.css', '.js', '.jpg', '.gif', '.png', '.com']
if any(needle in haystack for needle in needles):
    pass  # ...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>for needle in ['.css', '.js', '.jpg', '.gif', '.png', '.com']:
  if needle in haystack:
    return 'found'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use regex to "multi-match":</p>
<pre><code>import re
pat = r'(\.css|\.js|\.jpg|\.gif|\.png|\.com)'
files = ['file.css', 'file.exe', 'file.js', 'file.bat']
for f in files:
    if re.findall(pat, f):
        print f
</code></pre>
<p><strong>OUTPUT</strong></p>
<pre><code>file.css
file.js
</code></pre>
<p>Notice that this solution enables you to run on arbitrary different number of file-names and compare it against multiple different extensions! </p>
</div>
