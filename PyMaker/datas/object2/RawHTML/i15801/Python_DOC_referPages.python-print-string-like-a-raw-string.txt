<p>Say I got a unicode string from json which is <code>u'a\tb'</code>, if I <code>print</code> it , I will get <code>a    b</code>.But what I want now is print a string exactly as <code>a\tb</code>. How can I do it?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>print(repr('a\tb'))
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html#repr" rel="noreferrer"><code>repr()</code></a> gives you the "representation" of the string rather than the printing the string directly.</p>
