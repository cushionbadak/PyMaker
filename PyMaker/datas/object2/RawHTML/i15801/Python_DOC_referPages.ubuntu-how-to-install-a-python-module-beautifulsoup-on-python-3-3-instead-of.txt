<p>I have this code (as written in BS4 documentaion):</p>
<pre><code>  from bs4 import BeautifulSoup
</code></pre>
<p>When I run the script (using python3) I get the error:</p>
<pre><code>  ImportError: No module named 'bs4'
</code></pre>
<p>So installed BeatifulSoup by:</p>
<pre><code>  sudo pip install BeatifulSoup4
</code></pre>
<p>But when I try to run the script again I get the same error.
Indeed BS4 is installed in:</p>
<pre><code>  BeautifulSoup4 in /usr/local/lib/python2.7/dist-packages
</code></pre>
<p>But I want to install and use it with python3.3 (as there are other module which are not working with python2.7).</p>
<p>I tried with:</p>
<pre><code>  virtualenv --python=/usr/bin/python2.7 /usr/bin/python3.3
</code></pre>
<p>and then install BS4 again, but nothing solved.</p>
<p>Any clue?
Thanks in advance</p>

<div class="post-text" itemprop="text">
<p>Ubuntu has beautifulsoup packaged. I found it by running apt-cache search</p>
<pre><code>$ apt-cache search beautifulsoup
</code></pre>
<p>I see it has both a 2.7 and 3.3 version in the results. You can get the 3.3 version by installing python3-bs4</p>
<pre><code>$ sudo apt-get install python3-bs4
</code></pre>
</div>
