<p>In the below random array:</p>
<pre><code>a = [[1,2,3,4],
     [6,7,8,9]] 
</code></pre>
<p>Could you please tell me how to remove element at a specific position. For example, how would I remove <code>a[1][3]</code>?</p>
<p>I understand <code>list.pop</code> is used for only list type DS here.</p>
</div>
<div class="post-text" itemprop="text">
<p>Simple, just pop on the list item.</p>
<pre><code>&gt;&gt;&gt; a = [[1,2,3,4], [6,7,8,9]]
&gt;&gt;&gt; a[1].pop(3)
&gt;&gt;&gt; a
[[1, 2, 3, 4], [6, 7, 8]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You should use <a href="https://docs.python.org/3/reference/simple_stmts.html#grammar-token-del_stmt" rel="nofollow"><code>del</code></a> to <em>remove</em> an item at a specific index:</p>
<pre><code>&gt;&gt;&gt; a = [[1,2,3,4], [6,7,8,9]]
&gt;&gt;&gt; del a[1][3]
&gt;&gt;&gt; a
[[1, 2, 3, 4], [6, 7, 8]]
&gt;&gt;&gt;
</code></pre>
<p><code>list.pop</code> should only be used when you need to save the value you just removed.</p>
</div>
<div class="post-text" itemprop="text">

<code>a[1].remove(9)</code>
