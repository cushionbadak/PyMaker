<p>From <a href="https://docs.python.org/3.4/installing/index.html" rel="nofollow">https://docs.python.org/3.4/installing/index.html</a></p>
<blockquote>
<p>pip is the preferred installer program. Starting with Python 3.4, it
  is included by default with the Python binary installers.</p>
</blockquote>
<p>Before python 3.4, pip was a 'seperate package' and was just called 'pip'.<br/>
Because they have now included it with the python installer, and other versions of pip might have been previously installed on the system, the new pip that comes with python 3.4 was named pip3.4 to be able to easily distinguish between different pip versions.</p>
<p>Also see <a href="https://docs.python.org/3/whatsnew/3.4.html" rel="nofollow">https://docs.python.org/3/whatsnew/3.4.html</a> and <a href="http://bugs.python.org/issue20139" rel="nofollow">http://bugs.python.org/issue20139</a></p>
