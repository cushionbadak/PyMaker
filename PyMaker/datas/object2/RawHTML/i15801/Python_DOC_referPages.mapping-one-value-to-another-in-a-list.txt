<div class="post-text" itemprop="text">
<p>In a Python list, how can I map all instances of one value to another value?</p>
<p>For example, suppose I have this list:</p>
<pre><code>x = [1, 3, 3, 2, 3, 1, 2]
</code></pre>
<p>Now, perhaps I want to change all <code>1</code>'s to <code>'a'</code>, all <code>2</code>'s to <code>'b'</code>, and all <code>3</code>'s to <code>'c'</code>, to create another list:</p>
<pre><code>y = ['a', 'c', 'c', 'b', 'c', 'a', 'b']
</code></pre>
<p>How can I do this mapping elegantly?</p>
</div>
<div class="post-text" itemprop="text">
<p>You should use a dictionary and a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="noreferrer">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; x = [1, 3, 3, 2, 3, 1, 2]
&gt;&gt;&gt; d = {1: 'a', 2: 'b', 3: 'c'}
&gt;&gt;&gt; [d[i] for i in x]
['a', 'c', 'c', 'b', 'c', 'a', 'b']
&gt;&gt;&gt;
&gt;&gt;&gt; x = [True, False, True, True, False]
&gt;&gt;&gt; d = {True: 'a', False: 'b'}
&gt;&gt;&gt; [d[i] for i in x]
['a', 'b', 'a', 'a', 'b']
&gt;&gt;&gt;
</code></pre>
<p>The dictionary serves as a translation table of what gets converted into what.</p>
</div>
<div class="post-text" itemprop="text">
<p>An alternative solution is to use the built-in function <a href="https://docs.python.org/2/library/functions.html#map" rel="nofollow noreferrer"><code>map</code></a> which applies a function to a list:</p>
<pre><code>&gt;&gt;&gt; x = [1, 3, 3, 2, 3, 1, 2]
&gt;&gt;&gt; subs = {1: 'a', 2: 'b', 3: 'c'}
&gt;&gt;&gt; list(map(subs.get, x)) # list() not needed in Python 2
['a', 'c', 'c', 'b', 'c', 'a', 'b']
</code></pre>
<p>Here the <code>dict.get</code> method was applied to the list <code>x</code> and each number was exchanged for its corresponding letter in <code>subs</code>.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>In [255]: x = [1, 3, 3, 2, 3, 1, 2]

In [256]: y = ['a', 'c', 'c', 'b', 'c', 'a', 'b']

In [257]: [dict(zip(x,y))[i] for i in x]
Out[257]: ['a', 'c', 'c', 'b', 'c', 'a', 'b']
</code></pre>
</div>
