<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.translate" rel="nofollow"><code>str.translate</code></a> and a <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow">dict comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; 'aeiouAEIOU'.translate({ord(x):None for x in 'aeiouAEIOU'})
''
&gt;&gt;&gt;
</code></pre>
<p>The dict comprehension is used to create a mapping for <code>str.translate</code> of what characters should be translated into what.  Mapping characters to <code>None</code> causes the method to remove them.</p>
<p>Note that you could also use <a href="https://docs.python.org/3/library/stdtypes.html#str.maketrans" rel="nofollow"><code>str.maketrans</code></a> instead of the dict comprehension:</p>
<pre><code>&gt;&gt;&gt; 'aeiouAEIOU'.translate(str.maketrans('', '', 'aeiouAEIOU'))
''
&gt;&gt;&gt;
</code></pre>
