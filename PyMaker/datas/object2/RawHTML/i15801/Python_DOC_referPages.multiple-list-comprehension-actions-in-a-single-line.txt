<p>I am currently performing the following actions to make a list lowercase then remove the dots.</p>
<pre><code>lowercase_list = [x.lower() for x in my_list]
lowercase_stripped_list = [x.replace('.', '') for x in lowercase_list]
</code></pre>
<p>is there a way to do this in a single line?</p>
<p>Thanks</p>

<div class="post-text" itemprop="text">
<p>Chain the method calls (<a href="https://docs.python.org/3/library/stdtypes.html#str.lower" rel="nofollow"><code>str.lower</code></a> returns a string, you can call another string method <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow"><code>str.replace</code></a> on the return value):</p>
<pre><code>&gt;&gt;&gt; my_list = ['Hello. World', 'Big.Big.World']
&gt;&gt;&gt; [x.lower().replace('.', '') for x in my_list]
['hello world', 'bigbigworld']
</code></pre>
