<p>I'm trying to do something really simple:</p>
<pre><code>vowels = {'a':10, 'e'11, 'i':15, 'o':17, 'u':3}
</code></pre>
<p>All I want to do, is access the key and value inside of a loop.</p>
<p>But when using a for loop like so:</p>
<pre><code>for vowel, count in vowels:

    print(count)
</code></pre>
<p>I get the error:</p>
<pre><code>need more than 1 value to unpack
</code></pre>

<div class="post-text" itemprop="text">
<p>You need to call the <a href="https://docs.python.org/3/library/stdtypes.html#dict.items" rel="nofollow"><code>items</code></a><sup>1</sup> method of the dictionary:</p>
<pre><code>for vowel, count in vowels.items():

    print(count)
</code></pre>
<p>Otherwise, you will only iterate over the dictionary's keys.</p>
<hr/>
<p><sup>1</sup>In Python 2.x, you should call <a href="https://docs.python.org/2/library/stdtypes.html#dict.iteritems" rel="nofollow"><code>dict.iteritems</code></a> instead to avoid creating an unnecessary list.</p>
</div>
