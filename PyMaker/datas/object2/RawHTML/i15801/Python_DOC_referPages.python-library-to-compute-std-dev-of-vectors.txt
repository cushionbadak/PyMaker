<div class="post-text" itemprop="text">
<p>You can use <a href="http://docs.scipy.org/doc/numpy/reference/generated/numpy.std.html" rel="nofollow"><code>numpy.std</code></a> for population standard deviation:</p>
<pre><code>&gt;&gt;&gt; [numpy.std(arr) for arr in (A, B, C)]
[0.47140452079103168, 0.5, 0.47140452079103168]
</code></pre>
<hr/>
<p>However, if you want to stick to built-in modules (and have Python v3.4+), you could check out the functions <code>pstdev</code> and <code>stdev</code> in the <a href="https://docs.python.org/3/library/statistics.html" rel="nofollow"><code>statistics</code></a> module to calcultae population standard deviation and sample standard deviation respectively.</p>
</div>
