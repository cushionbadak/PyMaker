<p>Perhaps you are looking for <a href="https://docs.python.org/3/library/string.html#string.ascii_letters" rel="nofollow"><code>string.ascii_letters</code></a>:</p>
<pre><code>from string import ascii_letters
stripped_str = whatever_str.rstrip(ascii_letters)
</code></pre>
<p>It allows you to do the same as your current code, but without typing the entire alphabet.</p>
<p>Below is a demonstration:</p>
<pre><code>&gt;&gt;&gt; from string import ascii_letters
&gt;&gt;&gt; ascii_letters
'abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ'
&gt;&gt;&gt;
&gt;&gt;&gt; '123abdjihdkffyifbgh'.rstrip(ascii_letters)
'123'
&gt;&gt;&gt;
</code></pre>
