<p>I have a list value, <code>my_list</code>, that I would want to get its minimum value:</p>
<pre><code>min_value = min(my_list)
</code></pre>
<p>This works fine if <code>my_list</code> is not empty but raises <strong>ValueError</strong> if the list is empty.</p>
<p>Is there any way I can return something else in a very <em>Pythonic</em> way if <code>my_list</code> is empty and also have the <strong>ValueError</strong> arrested without having to check with <code>if</code>(like shown below):</p>
<pre><code>if my_list:
    min_value = min(my_list)
</code></pre>
<p>I want a one line thing. Something like:</p>
<pre><code>min_value = min(my_list) or another_value
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>In Python 3.4, you can use the <a href="https://docs.python.org/3/whatsnew/3.4.html#other-language-changes" rel="nofollow">new <code>default</code> keyword</a>:</p>
<pre><code>min_value = min(my_list, default=0)
</code></pre>
<p>In older versions, a simple <a href="https://docs.python.org/3/reference/expressions.html#grammar-token-conditional_expression" rel="nofollow">conditional expression</a> can be used:</p>
<pre><code>min_value = min(my_list) if my_list else 0  # Empty lists evaluate to False
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>One way to do it is <code>min(my_list or [0])</code></p>
</div>
<div class="post-text" itemprop="text">
<p>Simply do this:</p>
<pre><code>my_list = []
another_value = 555
print min(my_list) if my_list else another_value
</code></pre>
</div>
