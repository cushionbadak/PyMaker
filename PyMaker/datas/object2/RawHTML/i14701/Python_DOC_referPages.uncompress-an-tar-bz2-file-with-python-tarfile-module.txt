<p>I have many files with the extension "tar.bz2" and I want to uncompress them.
So I use the "tarfile" module as explained here : <i><a href="https://docs.python.org/3/library/tarfile.html" rel="nofollow noreferrer">https://docs.python.org/3/library/tarfile.html</a>.</i> </p>
<p>I try the following code :</p>
<pre><code>import tarfile
tar = tarfile.open("path_to/test/sample.tar.bz2", "r:bz2")  
for i in tar:
  tar.extractall(i)
tar.close()
</code></pre>
<p>But nothing happens : the tar.bz2 file has not been uncompressed into the folder "path_to/test/".</p>
<p>Would you have any ideas ?<br/>
Thanks !</p>

<div class="post-text" itemprop="text">
<p>You use tar.extractall with wrong argument. I think, you need something like this</p>
<pre><code>import tarfile
tar = tarfile.open("path_to/test/sample.tar.bz2", "r:bz2")  
tar.extractall()
tar.close()
</code></pre>
<p>or</p>
<pre><code>import tarfile
tar = tarfile.open("path_to/test/sample.tar.bz2", "r:bz2")  
for i in tar:
  tar.extractfile(i)
tar.close()
</code></pre>
<p>If you need to extract files to some specific folder </p>
<pre><code>import tarfile
tar = tarfile.open("path_to/test/sample.tar.bz2", "r:bz2")  
tar.extractall(some_path)
tar.close()
</code></pre>

<div class="post-text" itemprop="text">
<p>I like the context manager:</p>
<pre><code>import tarfile

def extract_bz2(filename, path="."):
    with tarfile.open(filename, "r:bz2") as tar:
        tar.extractall(path)
</code></pre>
