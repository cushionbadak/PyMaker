<p>Okay so I'm trying to make a function that will take a string and a sequence of items (in the form of either a list, a tuple or a string) and remove all items from that list from the string.</p>
<p>So far my attempt looks like this:</p>
<pre><code>def eliminate(s, bad_characters):
    for item in bad_characters:
        s = s.strip(item)
    return s
</code></pre>
<p>However, for some reason when I try this or variations of this, it only returns either the original string or a version with only the first item in bad_characters removed. </p>
<pre><code>&gt;&gt;&gt; eliminate("foobar",["o","b"])
'foobar'
</code></pre>
<p>Is there a way to remove all items in bad_characters from the given string? </p>

<div class="post-text" itemprop="text">
<p>The reason your solution doesn't work is because <code>str.strip()</code> only removes characters from the outsides of the string, i.e. characters on the leftmost or rightmost end of the string. So, in the case of <code>'foobar'</code>, <code>str.strip()</code> with a single character argument would only work if you wanted to remove the characters <code>'f'</code> and <code>'r'</code>.</p>
<p>You <em>could</em> eliminate more of the inner characters with strip, but you would need to include one of the outer characters as well.</p>
<pre><code>&gt;&gt;&gt; 'foobar'.strip('of')
'bar'
&gt;&gt;&gt; 'foobar'.strip('o')
'foobar'
</code></pre>
<p>Here's how to do it by string-joining a generator expression:</p>
<pre><code>def eliminate(s, bad_characters):
    bc = set(bad_characters)
    return ''.join(c for c in s if c not in bc)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>strip</code> is not a correct choice for this task as it remove the characters from leading and trailing of the string, instead you can use <a href="https://docs.python.org/3/library/stdtypes.html#str.translate" rel="nofollow"><em>str.translate</em></a> method :</p>
<pre><code>&gt;&gt;&gt; s,l="foobar",["o","b"]
&gt;&gt;&gt; s.translate(None,''.join(l))
'far'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try this, may be time consuming <code>using recursion</code></p>
<pre><code>def eliminate(s, seq):
    while seq:
        return eliminate(s.replace(seq.pop(),""), seq)
    return s
&gt;&gt;&gt;eliminate("foobar",["o","b"])
'far'
</code></pre>
