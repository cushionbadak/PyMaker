<p>I have a list of dicts like below:</p>
<pre><code>lod = [
{'name': 'Tom', 'score': 60},
{'name': 'Tim', 'score': 70},
{'name': 'Tam', 'score': 80},
{'name': 'Tem', 'score': 90}
]
</code></pre>
<p>I want to get <code>{'name': 'Tem', 'score':90}</code> but I only can do below:</p>
<pre><code>max(x['score'] for x in lod)
</code></pre>
<p>This only return the value <code>90</code>.</p>
<p>How can I get the whole dict?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/glossary.html#term-key-function" rel="nofollow">key function</a> of <code>max</code>:</p>
<pre><code>&gt;&gt;&gt; lod = [
... {'name': 'Tom', 'score': 60},
... {'name': 'Tim', 'score': 70},
... {'name': 'Tam', 'score': 80},
... {'name': 'Tem', 'score': 90}
... ]
... 
&gt;&gt;&gt; max(lod, key=lambda x: x['score'])
{'name': 'Tem', 'score': 90}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just pass your list to <code>max</code>, like this:</p>
<pre><code>&gt;&gt;&gt; from operator import itemgetter
&gt;&gt;&gt; lod = [
... {'name': 'Tom', 'score': 60},
... {'name': 'Tim', 'score': 70},
... {'name': 'Tam', 'score': 80},
... {'name': 'Tem', 'score': 90}
... ]
&gt;&gt;&gt; max(lod, key=itemgetter('score'))
{'score': 90, 'name': 'Tem'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I dont know whether <code>sorting</code> is time consuming, </p>
<pre><code>&gt;&gt;&gt;sorted(lod, key=lambda x:x['score'])[-1]
{'name': 'Tem', 'score': 90}
</code></pre>
