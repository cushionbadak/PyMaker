<p>I have 2 lists like this: </p>
<pre><code>prefix = ['b', 'bo', 'br'] 
word = ['bring', 'boring', 'bold', 'bells']
</code></pre>
<p>How do I count how many times is each prefixes within the list word? </p>
<p>It should return</p>
<pre><code>[4, 2, 1]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using list comprehension, <a href="https://docs.python.org/3/library/stdtypes.html#str.startswith" rel="nofollow"><code>str.startswith</code></a> and <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum</code></a></p>
<pre><code>[sum(w.startswith(p) for w in word) for p in prefix] # [4, 2, 1]
</code></pre>
</div>
