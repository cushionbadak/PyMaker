<div class="post-text" itemprop="text">
<p>If we are looking for the intersection of two lists, we usually do </p>
<pre><code>b1 = set(aList).intersection(aList2)
print(b1)
</code></pre>
<p>however what if we want to get the intersection of the two lists of words of length 4 only. </p>
<p>for example.</p>
<p>So if </p>
<pre><code>aList = ["james", "kobe", "ball"]
</code></pre>
<p>and </p>
<pre><code>aList2 = ["jimmy","james","kobe"]
</code></pre>
<p>I expect b1 to be = ["kobe"]  because kobe is of length 4 while james is not</p>
<p>Do we need to take out all the words of length 4 from the list first ? or is there a way to check while doing the intersection ?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <a href="https://docs.python.org/3/reference/expressions.html#displays-for-lists-sets-and-dictionaries" rel="nofollow">comprehension</a> for this:</p>
<pre><code>four_letter_intersection = {
    word for word in set(list_a).intersection(list_b) if len(word) == 4
    }
</code></pre>
</div>
<div class="post-text" itemprop="text">
<ol>
<li>Get common element by <strong>Intersection</strong> method of SET.</li>
<li>Use <strong>List Comprehension</strong>  to get only words which length is 4</li>
</ol>
<p><strong>Demo</strong>:</p>
<pre><code>&gt;&gt;&gt; aList = ["james", "kobe", "ball"]
&gt;&gt;&gt; aList2 = ["jimmy","james","kobe"]
&gt;&gt;&gt; b1 = set(aList).intersection(aList2)
&gt;&gt;&gt; b1
set(['james', 'kobe'])
&gt;&gt;&gt; result = [i for i in b1 if len(i)==4]
&gt;&gt;&gt; result
['kobe']
&gt;&gt;&gt; 
</code></pre>
<hr/>
<p><strong>Pass only words which length is 4 for Intersection operation.</strong></p>
<p>Create <code>tmp</code> list from the <code>aList</code> which have only words with length 4.</p>

<pre><code>&gt;&gt;&gt; aList = ["james", "kobe", "ball"]
&gt;&gt;&gt; aList2 = ["jimmy","james","kobe"]

&gt;&gt;&gt; tmp
['kobe', 'ball']
&gt;&gt;&gt; b1 = set(tmp).intersection(aList2)
