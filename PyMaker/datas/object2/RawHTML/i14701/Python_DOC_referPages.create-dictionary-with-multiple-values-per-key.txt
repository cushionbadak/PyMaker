<div class="post-text" itemprop="text">
<p>How can I create a dictionary with multiple values per key from 2 lists?</p>
<p>For example, I have:</p>
<pre><code>&gt;&gt;&gt; list1 = ['fruit', 'fruit', 'vegetable']

&gt;&gt;&gt; list2 = ['apple', 'banana', 'carrot']
</code></pre>
<p>And, I want something to the effect of:</p>
<pre><code>&gt;&gt;&gt; dictionary = {'fruit': ['apple', 'banana'], 'vegetable': ['carrot']}
</code></pre>
<p>I have tried the following so far:</p>
<pre><code>&gt;&gt;&gt; keys = list1
&gt;&gt;&gt; values = list2
&gt;&gt;&gt; dictionary = dict(zip(keys, values))
&gt;&gt;&gt; dictionary
    {'fruit': 'banana', 'vegetable': 'carrot'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>dict.setdefault</code> and a simple for-loop:</p>
<pre><code>&gt;&gt;&gt; list1 = ["fruit", "fruit", "vegetable"]
&gt;&gt;&gt; list2 = ["apple", "banana", "carrot"]
&gt;&gt;&gt; dct = {}
&gt;&gt;&gt; for i, j in zip(list1, list2):
...     dct.setdefault(i, []).append(j)
... 
&gt;&gt;&gt; dct
{'fruit': ['apple', 'banana'], 'vegetable': ['carrot']}
</code></pre>
<p>From the <a href="https://docs.python.org/3/library/stdtypes.html#dict.setdefault">docs</a>:</p>
<blockquote>
<p><code>setdefault(key[, default])</code></p>
<p>If <code>key</code> is in the dictionary, return its value. If not, insert <code>key</code> with a value of <code>default</code> and return <code>default</code>. <code>default</code> defaults to <code>None</code>.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/2/library/collections.html#collections.defaultdict" rel="noreferrer"><em>collections.defaultdict</em></a> for such tasks :</p>
<pre><code>&gt;&gt;&gt; from collections import defaultdict
&gt;&gt;&gt; d=defaultdict(list)
&gt;&gt;&gt; for i,j in zip(list1,list2):
...    d[i].append(j)
... 
&gt;&gt;&gt; d
defaultdict(&lt;type 'list'&gt;, {'vegetable': ['carrot'], 'fruit': ['apple', 'banana']})
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is a bit different from the other answers. It is a bit simpler for beginners.</p>
<pre><code>list1 = ['fruit', 'fruit', 'vegetable']
list2 = ['apple', 'banana', 'carrot']
dictionary = {}

for i in list1:
    dictionary[i] = []

for i in range(0,len(list1)):
    dictionary[list1[i]].append(list2[i])
</code></pre>
<p>It will return</p>
<pre><code>{'vegetable': ['carrot'], 'fruit': ['apple', 'banana']}
</code></pre>
<p>This code runs through <code>list1</code> and makes each item in it a key for an empty list in <code>dictionary</code>. It then goes from 0-2 and appends each item in <code>list2</code> to its appropriate category, so that index 0 in each match up, index 1 in each match up, and index 2 in each match up.</p>
