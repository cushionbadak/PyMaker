<p>I have a path in a python script as:</p>
<pre><code>    D://mycom/hjkl/Recent
</code></pre>
<p>Now i want to print only Recent in the output.
How will i achieve that in my python script?</p>
</div>
<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/os.html" rel="nofollow"><code>os</code></a> module (more specifically <a href="https://docs.python.org/3/library/os.path.html#os.path.basename" rel="nofollow"><code>os.path.basename</code></a>) is what you're looking for:</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; path = "D://mycom/hjkl/Recent"
&gt;&gt;&gt; print os.path.basename(path)
'Recent'
</code></pre>
</div>
<span class="comment-copy">Show some effort before you ask. This is quite a trivial task</span>
