<p>In python I want to cut the drive name from given path,</p>
<pre><code>Example:
if path given is D:\John\myfolder\newfolder
then I should get John\myfolder\newfolder
</code></pre>
<p>please tell how can I do this? I am trying to use os.path.abspath(path) but this is not working...</p>
</div>
<div class="post-text" itemprop="text">
<p>With <a href="https://docs.python.org/3/library/os.path.html#os.path.splitdrive" rel="nofollow"><code>os.path.splitdrive</code></a>:</p>
<pre><code>In [1]: import os

In [2]: drive, tail = os.path.splitdrive(r'D:\John\myfolder\newfolder')

In [3]: tail
Out[3]: '\\John\\myfolder\\newfolder'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use slicing.</p>
<pre><code>&gt;&gt;&gt; path = r"D:\John\myfolder\newfolder"
&gt;&gt;&gt; print(path[3:])
John\myfolder\newfolder
</code></pre>
