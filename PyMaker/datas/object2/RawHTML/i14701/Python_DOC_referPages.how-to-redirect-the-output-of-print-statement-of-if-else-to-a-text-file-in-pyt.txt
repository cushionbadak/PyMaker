<div class="post-text" itemprop="text">
<p>In Python3 the output redirection is as simple as</p>
<pre><code>print(....., file = open("filename",'w'))
</code></pre>
<p>Refer the <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow">docs</a></p>
<p>In your particular case you can even use the <code>with open</code> syntax as in</p>
<pre><code>if(cp&lt;0):

    print("No Common Predecessor")
elif (posa &lt; posb):

    if(posa&lt;0):
        posa=0
    with open('out.txt','w')as f:
        print("Common Predecessor: %s" %n[posa])
        f.write("Common Predecessor: %s" %n[posa])
else:

    if(posb &lt; 0):
        posb=0
    with open('anotherout.txt','w')as f:
        print("Common Predecessor: %s" %m[posb])
        f.write("Common Predecessor: %s" %m[posb])
</code></pre>
<p>Note - It is always better to use <code>'a'</code> (append) instead of <code>'w'</code> incase you are re-executing the program. </p>
