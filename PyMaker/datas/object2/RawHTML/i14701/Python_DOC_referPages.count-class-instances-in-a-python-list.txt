<div class="post-text" itemprop="text">
<p>I wonder what´s the best and fastest way to <strong>count the number of occurrences of a particular object</strong> (user-defined class instance) <strong>in a Python list</strong>?</p>
<p>For instance if we have a <strong>class called <code>MyClass</code></strong></p>
<pre><code>class MyClass:
    """A simple example class"""
    i = 42
    def foo(self):
        return “Hello world”
</code></pre>
<p>and a <strong>list <code>myList</code></strong> consisting of six elements:</p>
<pre><code>myList = [MyClass(), MyClass(), 2, 3, 'egg', MyClass()]
</code></pre>
<p>we could <strong>count the number of occurrences of an object</strong> which is a <code>MyClass</code> instance <strong>by iterating <code>myList</code> and using the <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow"><code>isinstance()</code></a> function</strong>:</p>
<pre><code>count = 0

for item in myList:
    if isinstance(item, MyClass):
        count += 1
</code></pre>
<p>But is there <strong>any better</strong> or <strong>more pythonic</strong> way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do that kind of count using <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum</code></a>, since <code>True</code> equals one and <code>False</code> equals zero:</p>
<pre><code>count = sum(isinstance(x, MyClass) for x in myList)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It sounds like you're asking for a <code>filter</code>:</p>
<pre><code>count = len( filter(lambda i: isinstance(i, myClass), myList))
</code></pre>
</div>
<span class="comment-copy">or <code>sum(1 for x in myList if isinstance(x, MyClass))</code>.  As an 18 year Python user, I can't decide which is better or more 'pythonic'.</span>
