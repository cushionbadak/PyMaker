<p>Is there a built in function in Python that will return a single result given a list and a validation function? </p>
<p>For example I know I can do the following:</p>
<pre><code>    resource = list(filter(lambda x: x.uri == uri, subject.resources))[0]
</code></pre>
<p>The above will extract a resource from a list of resources, based on ther resource.uri field. Although this field value is uinique, so I know that I will either have 1 or 0 results. <code>filter</code> function will iterate the whole list. In my case its 20 elements, but I want to know if there is some other built-in way to stop the iteration on first match.</p>
</div>
<div class="post-text" itemprop="text">
<p>See <a href="https://docs.python.org/3/library/functions.html#next" rel="noreferrer">https://docs.python.org/3/library/functions.html#next</a></p>
<blockquote>
<p><strong>next(iterator[, default])</strong></p>
<p>Retrieve the next item from the iterator by calling its <strong>next</strong>() method. If default is given, it is returned if the iterator is exhausted, otherwise StopIteration is raised.</p>
</blockquote>
<p>e.g. in your case:</p>
<pre><code>resource = next(filter(lambda x: x.uri == uri, subject.resources), None)
</code></pre>
</div>
