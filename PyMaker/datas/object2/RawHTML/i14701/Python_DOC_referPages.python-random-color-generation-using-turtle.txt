<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/2/library/turtle.html#turtle.colormode" rel="nofollow">turtle.colormode</a> needs to be set to 255 to give color strings in Hex Code or R G B. </p>
<p>adding </p>
<pre><code>screen.colormode(255)
</code></pre>
<p>no longer returned an error.</p>

<div class="post-text" itemprop="text">
<p>Your variables r g and b are not global. You either have to add a global declaration at the top of your function or add them as parameters. </p>
<pre><code>def my_function(r, g, b):
    # some stuff
</code></pre>
<p>Or...</p>
<pre><code>def myfunction():
    global r, g, b
    # some stuff
</code></pre>
