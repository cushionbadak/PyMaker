<div class="post-text" itemprop="text">
<p>To count the number of zeros in the calculated values you have many option.</p>
<p>Here are two:</p>
<p>Use variable to store the count:</p>
<pre><code>zeros_count = 0
for i in myNum:
    divideNum = int(myNum) % int(i)
    print(divideNum)
    if divideNum == 0:
        # value is 0, add 1 to the count
        zeros_count = zeros_count + 1
print("number of zeros: %d" % zeros_count)
</code></pre>
<p>Or create a list of zeros from the values using <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> and then use the <a href="https://docs.python.org/3/library/functions.html#len" rel="nofollow"><code>len</code></a> function, which embeds the loop in the list construction:</p>
<pre><code>zeros_count = len([i for i in myNum if i == '0'])
print("number of zeros: %d" % zeros_count)
</code></pre>
