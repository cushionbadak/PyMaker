<div class="post-text" itemprop="text">
<p>Use the <a href="http://docs.python.org/3/library/functions.html#any" rel="nofollow"><code>any()</code> function</a> with a list comprehension to <em>keep</em> entries with numbers instead:</p>
<pre><code>transposedlist = [entry for entry in transposedlist if any(e.isdigit() for e in entry)]
</code></pre>
<p>The <code>any()</code> function here returns <code>True</code> if there is any value in the iterable (a generator expression here) that is <code>True</code>, <code>False</code> otherwise. If the expression <code>(e.isdigit() for e in entry)</code> only yields <code>False</code> then that <code>entry</code> will not be included in the new list.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; transposedlist = [('-', '*', '*', '1'), ('7', '6', '6', '1'), ('-', '*', '1', '*'), ('-', '*', '*', '*'), ('1', '3', '3', '*'), ('-', ' ', ' ', '*')]
&gt;&gt;&gt; any(e.isdigit() for e in transposedlist[0])
True
&gt;&gt;&gt; any(e.isdigit() for e in transposedlist[3])
False
&gt;&gt;&gt; [entry for entry in transposedlist if any(e.isdigit() for e in entry)]
[('-', '*', '*', '1'), ('7', '6', '6', '1'), ('-', '*', '1', '*'), ('1', '3', '3', '*')]
</code></pre>
</div>
