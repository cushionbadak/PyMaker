<pre><code>string = """
Hello World

<p>I want to clear all the line-breaks from the string in python.</p>
<p>I tried</p>
<pre><code>string.strip()
</code></pre>
<p>But it's not working as desired.</p>
<p>What I should do?</p>

<div class="post-text" itemprop="text">
<p><a href="http://docs.python.org/3/library/stdtypes.html#str.strip" rel="noreferrer"><code>str.strip</code></a> removes whitespace from the start and the end of the string.</p>
<pre><code>&gt;&gt;&gt; string
'\nHello World\n123\nHelloWorld\n'
&gt;&gt;&gt; string.strip()
'Hello World\n123\nHelloWorld'
</code></pre>
<p>If you want to remove the new line characters <em>inside</em> of the string, you can replace them by something else using <a href="http://docs.python.org/3/library/stdtypes.html#str.replace" rel="noreferrer"><code>str.replace</code></a>:</p>
<pre><code>&gt;&gt;&gt; string.replace('\n', ' ')
' Hello World 123 HelloWorld '
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I would like to point out that, depending on how that triple-quoted string is being used, you could avoid the issue entirely.</p>
<p>In Python triple-quoted strings, you can put a backslash ("<code>\</code>") at the end of a line to ignore the line break. In other words, you can use it to put a line break at that spot in your code without making a line break in your string.</p>
<p>For example:</p>
<pre><code>"""line 1 \
line 2 \
line 3"""
</code></pre>
<p>will actually become</p>
<pre><code>line 1 line 2 line 3
</code></pre>
<p>if printed or written to some file or other output.</p>
<p>Using it this way can eliminate any need for a function to replace the line breaks, making your code clearer and cleaner.</p>
<p>EDIT:</p>
<p>If you're using backslash line continuations like this, you can also use simple single-quoted strings the same way;</p>
<pre><code>"line 1 \
line 2 \
line 3"
</code></pre>
<p>is also equivalent to</p>
<pre><code>line 1 line 2 line 3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>translate</code> instead of <code>replace</code> for this kind of operation. </p>
<pre><code>&gt;&gt; string.translate(None, "\n")
Hello World123HelloWorld
</code></pre>
<p>You will realize how <a href="https://stackoverflow.com/questions/1276764/stripping-everything-but-alphanumeric-chars-from-a-string-in-python">fast translate</a> is when you work with big files. </p>
</div>
<div class="post-text" itemprop="text">
<p>Simple <code>str.replace</code>:</p>
<pre><code>string.replace('\n', ' ')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The main question is already answered, but I would like to add that if you plan on <strong>using</strong> the individual lines you might be better off doing this -</p>
<pre><code>string = """
Hello World
123
HelloWorld
"""
for line in string.strip().split('\n'):
    print(line)
    # do something with the line
</code></pre>
<p><strong>Output</strong> from above code -</p>
<pre><code>Hello World
123
HelloWorld
</code></pre>
