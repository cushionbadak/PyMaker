<p>Your first line is missing the format placeholders:</p>
<pre><code>f.write('20/10/2013;SP;83428407;10:00;-;10:30;{};{};:00:15;:03:12;;;0;2;:00:58;50,0;2;0;0;0;0;0;0;0;0;0\n'.format(
    random.randrange(0,6), random.randrange(0,6)))
</code></pre>
<p>It is the <code>{}</code> placeholders that <code>str.format()</code> fills. Note that you only need <strong>one</strong> string, with placeholders where the arguments to <code>str.format()</code> are to be filled in.</p>
<p>Demo output:</p>
<pre><code>&gt;&gt;&gt; import random
&gt;&gt;&gt; '20/10/2013;SP;83428407;10:00;-;10:30;{};{};:00:15;:03:12;;;0;2;:00:58;50,0;2;0;0;0;0;0;0;0;0;0\n'.format(
...     random.randrange(0,6), random.randrange(0,6))
'20/10/2013;SP;83428407;10:00;-;10:30;0;5;:00:15;:03:12;;;0;2;:00:58;50,0;2;0;0;0;0;0;0;0;0;0\n'
&gt;&gt;&gt; '20/10/2013;SP;83428407;10:00;-;10:30;{};{};:00:15;:03:12;;;0;2;:00:58;50,0;2;0;0;0;0;0;0;0;0;0\n'.format(
...     random.randrange(0,6), random.randrange(0,6))
'20/10/2013;SP;83428407;10:00;-;10:30;3;1;:00:15;:03:12;;;0;2;:00:58;50,0;2;0;0;0;0;0;0;0;0;0\n'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your string looks malformed in some fashion -- If you're looking to append the randrange values between the strings, you need to add '+' to both sides of the format.</p>
<pre><code>f.write('20/10/2013;SP;83428407;10:00;-;10:30;'+str(random.randrange(0,6))+';'+str(random.randrange(0,6))+';:00:15;:03:12;;;0;2;:00:58;50,0;2;0;0;0;0;0;0;0;0;0\n')
</code></pre>
<p>It's usually clearer to just create a string and explicitly insert the values though.</p>
<pre><code>f.write('20/10/2013;SP;83428407;10:00;-;10:30;%s;%s;:00:15;:03:12;;;0;2;:00:58;50,0;2;0;0;0;0;0;0;0;0;0\n' % (str(random.randrange(0,6)), str(random.randrange(0,6))))
</code></pre>
<p>This also gives you the ability to control the output if you wanted to do any type of string formatting or aligning of the values.</p>
