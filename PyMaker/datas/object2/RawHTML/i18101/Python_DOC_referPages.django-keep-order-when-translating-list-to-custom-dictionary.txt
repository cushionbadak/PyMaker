<span class="comment-copy">Dictionaries are unordered. Use <a href="http://docs.python.org/3/library/collections.html#collections.OrderedDict" rel="nofollow noreferrer"><code>collections.OrderedDict</code></a> if you want to maintain order.</span>
<span class="comment-copy">Please people learn to use dicts... <code>response.update({test.slug: whatever})</code> is an inefficient and overcomplexificated way to write <code>reponse[test.slug] = whatever</code>.</span>
