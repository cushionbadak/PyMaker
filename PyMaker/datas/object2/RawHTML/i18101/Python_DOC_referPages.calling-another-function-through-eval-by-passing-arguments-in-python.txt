<div class="post-text" itemprop="text">
<p>Your <code>Login_CSA.py</code> is broken per se. As soon as you import it,</p>
<pre><code>def Main(a,b,c,d):

   z=a+b+c+d
   print z

Main()
</code></pre>
<p>you get an error because the <code>Main()</code> call won't work as <code>Main()</code> expects exactly 4 arguments.</p>
<p>If you <strong>have removed this <code>Main()</code> call</strong>, you will be able to import the module as wanted, e. g. with Martijn's solution which he, alas, deleted:</p>
<pre><code>import importlib

mod = importlib.import_module('Login_CSA')
mod.Main(10, 20, 30, 40)
</code></pre>
<p>Now the import process should succeed, and you should have access to the module's <code>Main()</code> function. (But please rename it the right way - titlecase names are reserved for classes.)</p>
