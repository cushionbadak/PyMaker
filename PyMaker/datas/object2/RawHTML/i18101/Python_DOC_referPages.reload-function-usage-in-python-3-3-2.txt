<div class="post-text" itemprop="text">
<p>You need to pass actual module objects to <code>imp.reload()</code>.</p>
<p>If you only have the module <em>name</em>, look up the module object in the <a href="http://docs.python.org/3/library/sys.html#sys.modules" rel="nofollow"><code>sys.modules</code> mapping</a>:</p>
<pre><code>import sys
import imp

imp.reload(sys.modules['fileread'])
</code></pre>
<p>This only works on modules that have already been imported; if some of your entries are not imported yet, at the very least catch the <code>KeyError</code> skip these:</p>
<pre><code>try:
    imp.reload(sys.modules[modulename])
except KeyError:
    # not loaded, no point in reloading
    pass
</code></pre>
<p>Optionally, you could use <a href="http://docs.python.org/3/library/importlib.html#importlib.import_module" rel="nofollow"><code>importlib.import_module()</code></a> to load such modules instead.</p>
