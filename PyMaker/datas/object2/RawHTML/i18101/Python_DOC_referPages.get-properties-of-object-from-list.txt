<p>Consider an object <code>customer</code> and a list of properties <code>attrs</code>. <strong>How might one iterate over the list to get the properties from the list?</strong></p>
<pre><code>class Human():     
    name = 'Jenny'         
    phone = '8675309'

customer = Human()
attrs = ['name', 'phone']

print(customer.name)    # Jenny
print(customer.phone)    # 8675309

for a in attrs:
    print(customer.a)    # This doesn't work!
    print(customer[a])    # Neither does this!
</code></pre>
<p>I am specifically targeting Python3 (Debian Linux) but a Python2 answer would be welcome as well.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/functions.html#getattr" rel="nofollow"><code>getattr</code></a>:</p>
<pre><code>getattr(customer, a)
</code></pre>
<hr/>
<pre><code>&gt;&gt;&gt; class Human:
...     name = 'Jenny'
...     phone = '8675309'
...
&gt;&gt;&gt; customer = Human()
&gt;&gt;&gt; for a in ['name', 'phone']:
...     print(getattr(customer, a))
...
Jenny
8675309
</code></pre>
