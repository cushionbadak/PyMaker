<div class="post-text" itemprop="text">
<p>In Python 2.x, Use <a href="https://docs.python.org/2/library/urllib.html#urllib.urlopen" rel="nofollow"><code>urllib.urlopen</code></a>:</p>
<pre><code>import urllib
urlilb.urlopen('http://10.16.16.96/set?params=default').read()
</code></pre>
<p>In Python 3.x, use <a href="https://docs.python.org/3/library/urllib.request.html#urllib.request.urlopen" rel="nofollow"><code>urllib.request.urlopen</code></a>:</p>
<pre><code>import urllib.request
urlilb.request.urlopen('http://10.16.16.96/set?params=default').read()
</code></pre>
<hr/>
<p><strong>UPDATE</strong></p>
<p>You can also use <a href="http://python-requests.org/" rel="nofollow"><code>requests</code></a> if you don't mind installing third party library:</p>
<pre><code>import requests
requests.get('http://10.16.16.96/set?params=default').content
</code></pre>
