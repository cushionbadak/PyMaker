<div class="post-text" itemprop="text">
<p><code>random.randint(a, b)</code> returns random integers between <em>a</em> and <em>b</em> <strong>inclusive</strong>. Here, <code>randoms[len(randoms)]</code> is giving an error. Try <code>random.randrange</code> or <code>random.choice</code> instead.</p>
</div>

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/random.html#random.randint" rel="nofollow"><code>randint</code></a> takes as arguments the <em>inclusive</em> bounds for the range. So if your lower bound is 0 you probably want to set the upper bound to <code>len(randoms)-1</code>.</p>
</div>
