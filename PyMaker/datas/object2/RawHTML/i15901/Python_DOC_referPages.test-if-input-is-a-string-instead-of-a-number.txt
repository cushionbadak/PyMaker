<div class="post-text" itemprop="text">
<p>It sounds like you are looking for <a href="https://docs.python.org/3/library/stdtypes.html#str.isdigit" rel="nofollow noreferrer"><code>str.isdigit</code></a>:</p>
<pre><code>&gt;&gt;&gt; x = input(':')
:abcd
&gt;&gt;&gt; x.isdigit()  # Returns False because x contains non-digit characters
False
&gt;&gt;&gt; x= input(':')
:123
&gt;&gt;&gt; x.isdigit()  # Returns True because x contains only digits
True
&gt;&gt;&gt;
</code></pre>
<p>In your code, it would be:</p>
<pre><code>if not x.isdigit():
    print('Please enter a number')  
else:
    print('blah blah blah')
</code></pre>
<hr/>
<p>On a side note, you should avoid using <code>eval(input(...))</code> because it can be used to execute arbitrary expressions.  In other words, it is very often a security hole and is therefore considered a bad practice by most Python programmers. Reference:</p>
<p><a href="https://stackoverflow.com/questions/1832940/is-using-eval-in-python-a-bad-practice">Is using eval in Python a bad practice?</a></p>
