<div class="post-text" itemprop="text">
<p>I'm looking for a Python function which returns the element of an array at the index i.</p>
<p>For example, if the function is search(i), and A=[1, 4, 5, 2, 6], A.search(3) will return 2. </p>
<p>After googling I couldn't find what I was looking for.</p>
</div>
<div class="post-text" itemprop="text">
<p>The function you are looking for is called <a href="https://docs.python.org/3/reference/datamodel.html#object.__getitem__" rel="nofollow"><code>__getitem__</code></a>:</p>
<pre><code>&gt;&gt;&gt; A=[1, 4, 5, 2, 6]
&gt;&gt;&gt; A.__getitem__(3)
2
&gt;&gt;&gt;
</code></pre>
<p>It is the <a href="https://docs.python.org/3/reference/datamodel.html" rel="nofollow">special method</a> that is called when you do:</p>
<pre><code>A[3]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>So it's obviously A[3].</p>
<p>For people getting stuck on such a topic, the python documentation for <a href="https://docs.python.org/2/" rel="nofollow">Python 2.7</a> and <a href="https://docs.python.org/3.3/" rel="nofollow">Python 3.3</a> are available, I suggest you read them thoroughly.</p>
