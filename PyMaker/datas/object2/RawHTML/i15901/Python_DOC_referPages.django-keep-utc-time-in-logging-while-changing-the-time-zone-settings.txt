<div class="post-text" itemprop="text">
<p>I set the time zone in the <code>settings.py</code> file of my django project:</p>
<p><code>TIME_ZONE = 'US/Eastern'</code></p>
<p>and now my logs contain US/Eastern times.</p>
<p>I would like to keep an UTC time in my logs. Is that possible?</p>
</div>
<div class="post-text" itemprop="text">
<p>Django uses Python's logging facilities, so there shouldn't be anything Django-specific here.</p>
<p>According to <a href="https://docs.python.org/3.4/library/logging.html#logging.Formatter.formatTime" rel="noreferrer">the logging documentation</a>, setting <code>logging.Formatter.converter = time.gmtime</code> should make all logs output in UTC.</p>
<p>Alternatively, you can make your own <code>Formatter</code> class to use UTC:</p>
<pre><code>class UtcFormatter(logging.Formatter): 
    converter = time.gmtime
</code></pre>
<p>And then configure it using the <code>()</code> key (documented <a href="https://docs.python.org/3/library/logging.config.html#user-defined-objects" rel="noreferrer">here</a>) in the dictconfig:</p>
<pre><code>LOGGING = {
    'formatters': {
        'utc': { 
            '()': 'my.package.UtcFormatter',
            'format': '...',
            'datefmt': '...',
        }
    }
}
</code></pre>
</div>
