<p>As @poke stated, <code>list.remove(x)</code> only removes the first appearance of <code>x</code> in <code>list</code>. I would simply use a new <code>my_list</code> list in every iteration and get rid of the entire <code>if</code> inside the loop.</p>
<pre><code>while my_list.count(1) != 25: # check if there is more or less 1 (heads) in the list
    print "Throwing heads and tails:"
    my_list = []
    for i in range(50):
        toss = random.randint(int(1),int(2)) #tried this also without int() = (1,2)
        my_list.append(toss)
</code></pre>
<p>You don't need to check again if you have 25 heads inside the loop because you are just checking it in the loop condition <code>while my_list.count(1) != 25</code></p>
<p>BTW: </p>
<pre><code>my_list.count(1) &lt; 25 or my_list.count(1) &gt; 25
</code></pre>
<p>is the same as your <code>while</code> condition but less readable.</p>

<p><a href="https://docs.python.org/3/library/stdtypes.html#mutable-sequence-types" rel="nofollow"><code>list.remove(x)</code></a> only removes the first item which equals to <code>x</code>, so you are only removing a single item each time with this:</p>
<pre><code>my_list.remove(1)
my_list.remove(2)
</code></pre>
<p>As such, you don’t clear your list at all. Instead, you can just clear the list completely, by setting it to a new empty list:</p>
<pre><code>my_list = []
</code></pre>
<p>Since you are only interested in the number of head/tails throws, you can also just <em>count</em> them instead of remembering all individual tosses. So you’d just have a counter:</p>
<pre><code>headCount = 0
while headCount != 25:
    print "Throwing heads and tails:"
    headCount = 0
    for i in range(50):
        toss = random.randint(1, 2)
        if toss == 1:
            headCount += 1
</code></pre>
