<div class="post-text" itemprop="text">
<p>You wouldn't use <code>random.randint()</code> <strong>at all</strong>. You'd use <a href="https://docs.python.org/2/library/random.html#random.choice"><code>random.choice()</code></a> instead:</p>
<pre><code>import random

questions = [question1, question2, question3]
random_question = random.choice(questions)
</code></pre>
<p>The function picks one element at random from a sequence.</p>
<p>If you need to produce questions at random <em>without repetition</em>, you want to do something different; you would use <a href="https://docs.python.org/2/library/random.html#random.shuffle"><code>random.shuffle()</code></a> to randomize the whole list of questions, then just pick one from that list (perhaps removing it from the list) every time you need a new question. That produces a random <em>sequence</em> of questions.</p>
<pre><code>import random

questions = [question1, question2, question3]
random.shuffle(questions)

for question in questions:
    # questions are iterated over in random order
</code></pre>
<p>or</p>
<pre><code>questions = [question1, question2, question3]
random.shuffle(questions)

while questions:
    next_question = questions.pop()
</code></pre>

<div class="post-text" itemprop="text">
<p>Don't use <code>randint</code>, use <a href="https://docs.python.org/2/library/random.html#random.choice" rel="nofollow"><code>random.choice</code></a>. This function will select a random item from a list.</p>
<pre><code>import random

l = [1,2,3]

&gt;&gt;&gt; random.choice(l)
2
&gt;&gt;&gt; random.choice(l)
1
&gt;&gt;&gt; random.choice(l)
1
&gt;&gt;&gt; random.choice(l)
3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you <em>need</em> to use <a href="https://docs.python.org/3/library/random.html#random.randint" rel="nofollow noreferrer"><code>randint</code></a> because of an assignment, you can look at how <a href="https://hg.python.org/cpython/file/3.4/Lib/random.py#l250" rel="nofollow noreferrer"><code>choice</code></a> works. Slightly simplified, it's this:</p>
<pre><code>def choice(seq):
    i = randrange(len(seq))
    return seq[i]
</code></pre>
<p>And <code>randint(a, b)</code> is just "an alias for <code>randrange(a, b+1)</code>".</p>
<p>So, you know how to use <code>choice</code> from <a href="https://stackoverflow.com/a/26370109/908494">Martijn Pieter's answer</a>, you know what <code>choice</code> does, you should be able to figure out from there how to use <code>randint</code> from there.</p>
</div>
<div class="post-text" itemprop="text">
<p>If <a href="https://stackoverflow.com/questions/26370089/how-to-use-random-randint-for-list#comment41397449_26370089"><em>"you want to pick with replacement (that is, each time there's a 1/3 chance of each)"</em></a>:</p>
<pre><code>#!/usr/bin/env python
import random

questions = ["question1", "question2", "question3"]
while True: # infinite loop, press Ctrl + C to break
    print(random.choice(questions)) 
</code></pre>
<p><em>"without replacement (that is, each one only shows up once, so after 3 choices there's nothing left)"</em>:</p>
<pre><code>#!/usr/bin/env python
import random

questions = ["question1", "question2", "question3"]
random.shuffle(questions)
while questions: # only `len(questions)` iterations
    print(questions.pop()) 
</code></pre>
<p><em>"or some hybrid (e.g., pick all 3 in random order, then repeat all 3 in random order again, etc."</em>:</p>
<pre><code>#!/usr/bin/env python
import random

questions = ["question1", "question2", "question3"]
while True: # infinite loop, press Ctrl + C to break
    random.shuffle(questions)
    for q in questions: # only `len(questions)` iterations
        print(q) 
</code></pre>
