<div class="post-text" itemprop="text">
<p>I have an array of strings: </p>
<pre><code>["aaa 1", "aaa 2", "bbb 2", "ccc 3", "ddd 4"]
</code></pre>
<p>I need to search for some string in this vector, but only the first part. For example, I need the position of the strings starting with <code>"aaa"</code> (in this example, that would be 0 and 1).</p>
<p>How do I do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> with <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow"><code>enumerate</code></a> and <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>str.split</code></a>:</p>
<pre><code>&gt;&gt;&gt; lst = ["aaa 1", "aaa 2", "bbb 2", "ccc 3", "ddd 4"]
&gt;&gt;&gt; [x for x,y in enumerate(lst) if y.split()[0] == "aaa"]
[0, 1]
&gt;&gt;&gt;
</code></pre>
<p><code>y.split()[0]</code> will split the string on spaces and return the first element.  Thus:</p>
<pre><code>if y.split()[0] == "aaa"
</code></pre>
<p>only checks the first part of each string.  If however the strings are always like those given in your example, a simple <a href="https://docs.python.org/3/reference/expressions.html#in" rel="nofollow"><code>in</code> membership test</a> will suffice:</p>
<pre><code>[x for x,y in enumerate(lst) if "aaa" in y]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Us <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow">enumerate</a> and <a href="https://docs.python.org/2/library/stdtypes.html#str.startswith" rel="nofollow">str.startswith</a> :</p>
<pre><code>&gt;&gt;&gt; l =  ["aaa 1", "aaa 2", "bbb 2", "ccc 3", "ddd 4"]
&gt;&gt;&gt; print [ind for ind, ele in enumerate(l) if ele.startswith("aaa")]
[0, 1]
</code></pre>
<p><code>ind</code> is the index of each element in your list, <code>ele</code> is each element, so if the string starts with "aaa" we add the index to the list</p>
<p>If there is no chance you will have <code>"aaa"</code> in the second half of the string or a substring of the first half just using <a href="http://www.tutorialspoint.com/python/membership_operators_example.htm" rel="nofollow">in</a> will be the most efficient</p>
<pre><code>[ind for ind, ele in enumerate(l) if "aaa" in ele]
</code></pre>
</div>
