Such that:    </p>
<pre><code>for key_word in ["John","Jane"]:
    print(key_word, special_hash(key_word, 13)) 
&gt;&gt;&gt; John 8 
    Jane 4

<p>The function needs to convert each letter of the string to number form using alphabet position(eg. a=1, b= 2, c=3, etc). So the hash will be: John = 10+15+8+14= 47 -----&gt; = 47 % tablesize(13) </p>

<div class="post-text" itemprop="text">
<p>You can convert the string to lowercase with <code>lower</code> function and iterate through the characters in the word, with a <code>for</code> loop, like this</p>
<pre><code>def special_hash(word, tablesize):
    for char in word.lower():
        ...
</code></pre>
<p>Then, you can get the character code corresponding to the character with <a href="https://docs.python.org/3.5/library/functions.html#ord" rel="nofollow"><code>ord</code></a> function.</p>
<pre><code>def special_hash(word, tablesize):
    total_value = 0
    for char in word.lower():
        total_value += ord(char) - ord('a') + 1
</code></pre>
<p>Since we need to get the offset of the character in the alphabets, you can subtract the first value from the current value. Then finally, you can use modulo operator <code>%</code> to get the remainder of the division by <code>tablesize</code></p>
<pre><code>def special_hash(word, tablesize):
    total_value = 0
    for char in word.lower():
        total_value += ord(char) - ord('a') + 1
    return total_value % tablesize
</code></pre>
<p>The same can be written succintly with a <a href="https://docs.python.org/3/reference/expressions.html#generator-expressions" rel="nofollow">generator expression</a> and built-in <a href="https://docs.python.org/3.5/library/functions.html#sum" rel="nofollow"><code>sum</code></a> function, like this</p>
<pre><code>def special_hash(word, tablesize):
    return sum(ord(char) - ord('a') + 1 for char in word.lower()) % tablesize
</code></pre>
