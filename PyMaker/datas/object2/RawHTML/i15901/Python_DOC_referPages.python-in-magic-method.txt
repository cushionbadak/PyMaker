<p>Is there a magic method in python for the "in" method?
For example,</p>
<pre><code>y = [5, 4, 3, 2,1]
5 in y #returns true
</code></pre>
<p>if so what is it, and if not, how would one implement it.</p>
</div>
<div class="post-text" itemprop="text">
<p>It's <a href="https://docs.python.org/3/reference/datamodel.html#object.__contains__" rel="noreferrer"><code>__contains__()</code></a>:</p>
<pre><code>&gt;&gt;&gt; y = [5, 4, 3, 2, 1]
&gt;&gt;&gt; y.__contains__(3)
True
&gt;&gt;&gt; y.__contains__(6)
False
</code></pre>
</div>
<span class="comment-copy">Try it and you'll know.</span>
<span class="comment-copy">Special methods are documented in section 3.3 of the reference manual, which has subsections like 3.3.6 'Emulating Container Types'.</span>
