<div class="post-text" itemprop="text">
<p>The class statement creates a new class definition. The name of the class immediately follows the keyword class followed by a colon as follows:</p>
<pre><code>class ClassName:
  statement-1
  .
  .
  .
  statement-N
</code></pre>
<p>look here:
<a href="https://docs.python.org/3.4/tutorial/classes.html" rel="nofollow">https://docs.python.org/3.4/tutorial/classes.html</a></p>
