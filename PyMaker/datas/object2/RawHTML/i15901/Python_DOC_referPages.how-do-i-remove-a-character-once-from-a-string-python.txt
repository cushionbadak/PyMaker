<div class="post-text" itemprop="text">
<p>You can use the string replace function with a maximum count:</p>
<pre><code>s = s.replace('a','',1);
</code></pre>
<p>as in the following transcript:</p>
<pre><code>&gt;&gt;&gt; s = "baab"
&gt;&gt;&gt; s = s.replace('a','',1)
&gt;&gt;&gt; s
'bab'
</code></pre>
<p>From the <a href="https://docs.python.org/3/library/stdtypes.html" rel="nofollow">documentation</a>:</p>
<blockquote>
<p><strong>str.replace(old, new[, count])</strong></p>
<p>Return a copy of the string with all occurrences of substring <code>old</code> replaced by <code>new</code>. If the optional argument <code>count</code> is given, only the first <code>count</code> occurrences are replaced.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<pre><code>In[3]: "baab".replace('a', '', 1)
Out[3]: 'bab'
</code></pre>
<p>Will replace <code>a</code> only once with nothing, hence removing it.</p>
</div>
