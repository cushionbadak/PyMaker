<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow"><code>itertools.product</code></a></p>
<pre><code>&gt;&gt;&gt; import itertools
&gt;&gt;&gt; list(itertools.product(a,b,c))
[(1, 4, 8), (1, 4, 9), (1, 5, 8), (1, 5, 9), (1, 6, 8), (1, 6, 9), (1, 7, 8), (1, 7, 9),
 (2, 4, 8), (2, 4, 9), (2, 5, 8), (2, 5, 9), (2, 6, 8), (2, 6, 9), (2, 7, 8), (2, 7, 9),
 (3, 4, 8), (3, 4, 9), (3, 5, 8), (3, 5, 9), (3, 6, 8), (3, 6, 9), (3, 7, 8), (3, 7, 9)]
</code></pre>
<p>So to iterate and do stuff with the numbers, you could say</p>
<pre><code>for i, j, k in itertools.product(a,b,c):
    # do stuff with i,j,k
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your nested for loops create the products of lists so you can use <a href="https://docs.python.org/3.1/library/itertools.html#itertools.product" rel="nofollow"><code>itertools.product</code></a> instead !</p>
<pre><code>&gt;&gt;&gt; import itertools

&gt;&gt;&gt; list(itertools.product([1, 2, 3],[4, 5, 6, 7],[8, 9]))
[(1, 4, 8), (1, 4, 9), (1, 5, 8), (1, 5, 9), (1, 6, 8), (1, 6, 9), (1, 7, 8), (1, 7, 9), (2, 4, 8), (2, 4, 9), (2, 5, 8), (2, 5, 9), (2, 6, 8), (2, 6, 9), (2, 7, 8), (2, 7, 9), (3, 4, 8), (3, 4, 9), (3, 5, 8), (3, 5, 9), (3, 6, 8), (3, 6, 9), (3, 7, 8), (3, 7, 9)]
</code></pre>
<p>and if you want the result as list , use <code>map</code> :</p>
<pre><code>&gt;&gt;&gt; map(list,itertools.product([1, 2, 3],[4, 5, 6, 7],[8, 9]))
[[1, 4, 8], [1, 4, 9], [1, 5, 8], [1, 5, 9], [1, 6, 8], [1, 6, 9], [1, 7, 8], [1, 7, 9], [2, 4, 8], [2, 4, 9], [2, 5, 8], [2, 5, 9], [2, 6, 8], [2, 6, 9], [2, 7, 8], [2, 7, 9], [3, 4, 8], [3, 4, 9], [3, 5, 8], [3, 5, 9], [3, 6, 8], [3, 6, 9], [3, 7, 8], [3, 7, 9]]
</code></pre>
