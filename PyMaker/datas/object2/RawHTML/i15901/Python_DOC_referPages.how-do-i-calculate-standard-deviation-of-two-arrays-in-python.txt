<p>You want to compare two signals, e.g. <code>A</code> and <code>B</code> in the following example:</p>
<pre class="lang-py prettyprint-override"><code>import numpy as np

A = np.random.rand(5)
B = np.random.rand(5)

print "A:", A
print "B:", B
</code></pre>
<p>Output:</p>
<pre><code>A: [ 0.66926369  0.63547359  0.5294013   0.65333154  0.63912645]
B: [ 0.17207719  0.26638423  0.55176735  0.05251388  0.90012135]
</code></pre>
<h2>Analyzing individual signals</h2>
<p>The standard deviation of each single signal is <em>not</em> what you need:</p>
<pre><code>print "standard deviation of A:", np.std(A)
print "standard deviation of B:", np.std(B)
</code></pre>
<p>Output:</p>
<pre><code>standard deviation of A: 0.0494162021651
standard deviation of B: 0.304319034639
</code></pre>
<h2>Analyzing the difference</h2>
<p>Instead you might compute the difference and apply some common measure like the sum of absolute differences (SAD), the sum of squared differences (SSD) or the correlation coefficient:</p>
<pre><code>print "difference:", A - B
print "SAD:", np.sum(np.abs(A - B))
print "SSD:", np.sum(np.square(A - B))
print "correlation:", np.corrcoef(np.array((A, B)))[0, 1]
</code></pre>
<p>Output:</p>
<pre><code>difference: [ 0.4971865   0.36908937 -0.02236605  0.60081766 -0.2609949 ]
SAD: 1.75045448355
SSD: 0.813021824351
correlation: -0.38247081
</code></pre>

<p>Use numpy.</p>
<pre><code>import numpy as np
data = [1.2, 2.3, 1.3, 1.2, 5.4]
np.std(data)
</code></pre>
<p>Or you could try this:</p>
<pre><code>import numpy as np
obs = np.array([1.2, 2.3, 1.3, 1.2, 5.4])
model = np.array([1.1, 2.4, 1.2, 1.2, 5.3])
np.std(obs-model)
</code></pre>

<p>If you're doing anything more complicated than just finding the standard deviation and/or mean, use <code>numpy</code>/<code>scipy</code>. If that's <em>all</em> you need to do, use <a href="https://docs.python.org/3/library/statistics.html" rel="nofollow">the <code>statistics</code> package from the Python Standard Library</a>.</p>

1.0
