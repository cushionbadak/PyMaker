<div class="post-text" itemprop="text">
<p>STATICFILES_DIRS is a <strong>List</strong> more about lists you can read in official docs (see: <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow">https://docs.python.org/3/tutorial/datastructures.html</a>).</p>
<p>You can get the element of this list like this : <code>STATICFILES_DIRS[0]</code> when 0 is the position of a given element.</p>
<p>So I'm not sure if it's usual manipulate staticfiles_dirs in views. Read more about Django to do on the right way:
<a href="https://docs.djangoproject.com/en/1.9/howto/static-files/" rel="nofollow">https://docs.djangoproject.com/en/1.9/howto/static-files/</a></p>
