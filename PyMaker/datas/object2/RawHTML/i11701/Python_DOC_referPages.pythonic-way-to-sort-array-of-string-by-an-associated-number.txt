<div class="post-text" itemprop="text">
<p>You can use the <code>key</code> parameter of <code>sort</code> or <code>sorted</code> to set a function that is called before sorting. In your case, <code>sorted(hosts, key=getSpeed)</code> should be enough.</p>
</div>
<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html?highlight=sorted#sorted" rel="nofollow"><code>sorted</code></a> builtin function takes a <code>key=</code> argument. You can use that to provide a function that will return the comparison key (in this case, the speed) like so:</p>
<pre><code>def getSpeed(host):

    return speed

hosts = ['h1', 'h2', 'h3']
