<p>For example, I already know a binary data:  </p>
<pre><code>"b'\x80\x03}q\x00(X\x03\x00\x00\x00ageq\x01K\x14X\x05\x00\x00\x00scoreq\x02KXX\x04\x00\x00\x00nameq\x03X\x03\x00\x00\x00Bobq\x04u.'"
</code></pre>
<p>It is a dict object. </p>
<p>How can I translate it into a dict object? I can only find the way to read files into a object, but I can't find the way to translate the existing data into object.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can convert the binary data to a python object using the <a href="https://docs.python.org/3/library/pickle.html" rel="nofollow">pickle module</a>:</p>
<pre><code>&gt;&gt;&gt; bs = b'\x80\x03}q\x00(X\x03\x00\x00\x00ageq\x01K\x14X\x05\x00\x00\x00scoreq\x02KXX\x04\x00\x00\x00nameq\x03X\x03\x00\x00\x00Bobq\x04u.'
&gt;&gt;&gt; import pickle
&gt;&gt;&gt; loaded = pickle.loads(bs)
&gt;&gt;&gt; loaded
{'score': 88, 'age': 20, 'name': 'Bob'}
</code></pre>
<p>Note that this will only work on objects that were serialized using the tools in the pickle module - <code>pickle.dump</code> or <code>pickle.dumps</code>.  The pickle module can't deserialise arbitrary binary data.  </p>
<p>Also, you should only unpickle objects from a trusted source, as unpickling an object could cause code execution.</p>
