<p>I <em>think</em> that you want is a way  to create an executable from a Python script. There are a few tools to do this:</p>
<ul>
<li><a href="http://www.py2exe.org/" rel="nofollow">py2exe</a>  (Windows)</li>
<li><a href="http://www.pyinstaller.org/" rel="nofollow">pyinstaller</a> (multiple platforms)</li>
<li><a href="http://cx-freeze.sourceforge.net/" rel="nofollow">cx_Freeze</a> (multiple platforms)</li>
</ul>
<p>Alternatively you can <a href="https://docs.python.org/3/extending/embedding.html" rel="nofollow">embed the Python interpreter in C</a>.</p>
