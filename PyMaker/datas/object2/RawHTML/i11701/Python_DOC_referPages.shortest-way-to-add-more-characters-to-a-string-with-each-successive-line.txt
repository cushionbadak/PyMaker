<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>join</code></a> and a generator expression (avoiding the loop is not possible)</p>
<pre><code>&gt;&gt;&gt; print"\n".join("Wow{}".format('!'*i) for i in range(1,4))
Wow!
Wow!!
Wow!!!
Wow!!!!
</code></pre>
<hr/>
<p>If you really want to avoid <code>for</code> keyword </p>
<pre><code>&gt;&gt;&gt; n = 4
&gt;&gt;&gt; word = "Wow"
&gt;&gt;&gt; symb = "!"
&gt;&gt;&gt; print("\n".join(map(lambda x:x[0]+x[1],zip([word]*(n-1),map(lambda i:symb*i,range(1,n))))))
Wow!
Wow!!
Wow!!!
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You won't be able to do this without some sort of loop, as you need to change the number of exclamation points from 1 to infinity, and for that you need a loop.</p>
<p>You can do that without writing the loop yourself, though:</p>
<pre><code>["Wow" + '!'*n for n in range(500000000)]
</code></pre>
<p>There's still a loop in there.</p>
</div>
<div class="post-text" itemprop="text">
<p>You need to enclose your format string like this:</p>
<pre><code>print "Wow%s\n" % ("!"*3)
</code></pre>
