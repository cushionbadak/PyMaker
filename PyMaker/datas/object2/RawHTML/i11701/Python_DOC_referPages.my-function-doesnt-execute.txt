<div class="post-text" itemprop="text">
<p>The calls to the function happen before the function is defined. Move it to the start of your program.</p>
<pre><code>def mysum(param):
    var = 0
    while(param != 0):
        var += (param % 10)
        print(var)
        param /=  10
        print(param)
    return var

a = input('\n Enter the first number : ' )
x = mysum([a])
print('\n The sum of the digits number ' +str(x)+ ' este : %d' %x)

b = input('\n Enter the second number : ')
y = mysum([b])
print('\n The sum of the digits number ' +str(y)+ ' este : %d' %y)
</code></pre>
<p>Oh, and don't override the builtin <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum</code></a> (that's why I used <code>mysum</code>).</p>
<p>Also, the sum of digits can be computed by using <code>map</code> and <code>sum</code>:</p>
<pre><code>sum_of_digits = sum(int(x) for x in str(123) if x.isdigit())
</code></pre>
<p>Or as a function:</p>
<pre><code>def sum_of_digits(numstr):
    return sum(int(x) for x in str(numstr) if x.isdigit())
</code></pre>
</div>
