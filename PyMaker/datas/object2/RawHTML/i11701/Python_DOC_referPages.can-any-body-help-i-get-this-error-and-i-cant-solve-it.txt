<div class="post-text" itemprop="text">
<p>You have to provide an <em>absolute path</em> to you file.
Without that, the subprocess module can't find it.</p>
<p>The <code>os</code> module provides <a href="https://docs.python.org/3/library/os.path.html#os.path.abspath" rel="nofollow">routine</a> for that.</p>
