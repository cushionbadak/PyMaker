<p><a href="https://docs.python.org/3/tutorial/inputoutput.html" rel="nofollow">Check the Python 3 docs.</a></p>
<p>Python 3 uses <code>()</code> around the print contents - <code>print ('python')</code></p>
</div>
<span class="comment-copy">As of Python 3, <code>print</code> is now a function. Use <code>print()</code> like this: <code>print('hello world')</code>.</span>
