<p>For example:</p>
<pre><code>a = [1,2,3,4,5,6]
</code></pre>
<p>I want to do:</p>
<pre><code>1/2/3/4/5/6
</code></pre>
<p>I have tried using the <code>operator.div</code> function but it doesn't seem to give the correct result. By the way, I am fairly new to python.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functools.html#functools.reduce" rel="nofollow noreferrer"><code>reduce</code></a>. </p>
<blockquote>
<p>Apply function of two arguments cumulatively to the items of sequence, from left to right, so as to reduce the sequence to a single value.</p>
</blockquote>
<p>The code can be demonstrated as </p>
<pre><code>&gt;&gt;&gt; from functools import reduce 
&gt;&gt;&gt; l = [1,2,3,4,5,6]
&gt;&gt;&gt; reduce(lambda x,y:x/y,l)
0.001388888888888889
</code></pre>
<p>which is equivalent to</p>
<pre><code>&gt;&gt;&gt; 1/2/3/4/5/6
0.001388888888888889
</code></pre>
<hr/>
<p>As <code>truediv</code> has already been demonstrated by the other <a href="https://stackoverflow.com/a/38349427/4099593">answer</a>, this is an alternative (the other way is preferred) for Python2</p>
<pre><code>&gt;&gt;&gt; from __future__ import division
&gt;&gt;&gt; l = [1,2,3,4,5,6]
&gt;&gt;&gt; reduce(lambda x,y:x/y,l)
0.001388888888888889
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/2/library/functions.html#reduce" rel="nofollow"><code>reduce()</code></a> and <code>operator.truediv</code>:</p>
<pre><code>&gt;&gt;&gt; a = [1,2,3,4,5,6]
&gt;&gt;&gt; from operator import truediv
&gt;&gt;&gt; 
&gt;&gt;&gt; reduce(truediv, a)
0.001388888888888889
</code></pre>
<p><em>Note</em>: In python3.x you need to import the <code>reduce()</code> function from from <a href="https://docs.python.org/3/library/functools.html#module-functools" rel="nofollow"><code>functools</code></a> module.</p>
</div>
<div class="post-text" itemprop="text">
<p>Why not just use a loop?</p>
<pre><code>&gt;&gt;&gt; a = [1,2,3,4,5,6]
&gt;&gt;&gt; i = iter(a)
&gt;&gt;&gt; result = next(i)
&gt;&gt;&gt; for num in i:
...     result /= num
...
&gt;&gt;&gt; result
0.001388888888888889
&gt;&gt;&gt; 1/2/3/4/5/6
0.001388888888888889
</code></pre>
