<p>You can use a list comprehension and apply a <code>filter</code> that takes only new items in <code>liste2</code>:</p>
<pre><code>result = [i[0] for i in liste2 if i[0] not in (j[0] for j in liste1)]
#                              ^      filtering is done here       ^
print(result)
# ['nani']
</code></pre>

<p>You can create a set of the first element in each list item, and then take the difference. It will return the item in liste2 which is not in liste1:</p>
<pre><code>l3 = set(map(lambda x:x[0],liste1))
l4 = set(map(lambda x: x[0],liste2))
print list(l4.difference(l3))[0]
</code></pre>

<pre><code>nani
