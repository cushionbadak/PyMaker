<p>I have a class that specifies the column layout of a csv file as:</p>
<pre><code>class RecordLayout(object):
    YEAR = 0
    DATE = 1
    MONTH = 2
    ID = 3
    AGE = 4
    SALARY = 5
</code></pre>
<p>I need to get the list of the class variables in the order they are defined. </p>
<p>So far, I tried:</p>
<pre><code>[attr for attr in dir(RecordLayout()) if not callable(attr) and not attr.startswith("__")]
</code></pre>
<p>but it returns:</p>
<pre><code>['AGE', 'DATE', 'ID', 'MONTH', 'SALARY', 'YEAR']
</code></pre>
<p>which is the class variables ordered alphabetically. I need the variables to be returned in the order they are defined in the class:</p>
<pre><code>['YEAR', 'DATE', 'MONTH', 'ID', 'AGE', 'SALARY']
</code></pre>
<p>Is there a way to achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You should use an <a href="https://docs.python.org/3/library/enum.html" rel="nofollow">Enum</a> for this.</p>
<pre><code>&gt;&gt;&gt; class RecordLayout(Enum):
...     YEAR = 0
...     DATE = 1
...     MONTH = 2
...     ID = 3
...     AGE = 4
...     SALARY = 5
...
&gt;&gt;&gt; for i in RecordLayout:
...   print(i)
...
RecordLayout.YEAR
RecordLayout.DATE
RecordLayout.MONTH
RecordLayout.ID
RecordLayout.AGE
RecordLayout.SALARY
</code></pre>
