<p>I'm using a Python script to read data from our corporate instance of JIRA. There is a value that is returned as a string and I need to figure out how to extract one bit of info from it. What I need is the <strong>'name= ....'</strong> and I just need the numbers from that result.</p>
<pre><code>&lt;class 'list'&gt;: ['com.atlassian.greenhopper.service.sprint.Sprint@6f68eefa[id=30943,rapidViewId=10468,state=CLOSED,name=2016.2.4 - XXXXXXXXXX,startDate=2016-05-26T08:50:57.273-07:00,endDate=2016-06-08T20:59:00.000-07:00,completeDate=2016-06-09T07:34:41.899-07:00,sequence=30943]']
</code></pre>
<p>I just need the <strong><em>2016.2.4</em></strong> portion of it. This number will not always be the same either.</p>
<p>Any thoughts as how to do this with RE? I'm new to regular expressions and would appreciate any help.</p>

<div class="post-text" itemprop="text">
<p>A simple regular expression can do the trick: <code>name=([0-9.]+)</code>.</p>
<p>The primary part of the regex is <code>([0-9.]+)</code> which will search for any digit (<code>0-9</code>) or period (<code>.</code>) in succession (<code>+</code>).</p>
<p>Now, to use this:</p>
<pre><code>import re
pattern = re.compile('name=([0-9.]+)')

string = '''&lt;class 'list'&gt;: ['com.atlassian.greenhopper.service.sprint.Sprint@6f68eefa[id=30943,rapidViewId=10468,state=CLOSED,name=2016.2.4 - XXXXXXXXXX,startDate=2016-05-26T08:50:57.273-07:00,endDate=2016-06-08T20:59:00.000-07:00,completeDate=2016-06-09T07:34:41.899-07:00,sequence=30943]']'''

matches = pattern.search(string)
# Only assign the value if a match is found
name_value = '' if not matches else matches.group(1)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/2/howto/regex.html#grouping" rel="nofollow">capturing group</a> to extract the version name:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; s = 'com.atlassian.greenhopper.service.sprint.Sprint@6f68eefa[id=30943,rapidViewId=10468,state=CLOSED,name=2016.2.4 - XXXXXXXXXX,startDate=2016-05-26T08:50:57.273-07:00,endDate=2016-06-08T20:59:00.000-07:00,completeDate=2016-06-09T07:34:41.899-07:00,sequence=30943]'
&gt;&gt;&gt; re.search(r"name=([0-9.]+)", s).group(1)
'2016.2.4'
</code></pre>
<p>where <code>([0-9.]+)</code> is a capturing group matching one or more digits or dots, parenthesis define a capturing group.</p>
<p>A non-regex option would involve some <a href="https://docs.python.org/2/library/stdtypes.html#str.split" rel="nofollow">splitting</a> by <code>,</code>, <code>=</code> and <code>-</code>:</p>
<pre><code>&gt;&gt;&gt; l = [item.split("=") for item in s.split(",")]
&gt;&gt;&gt; next(value[1] for value in l if value[0] == "name").split(" - ")[0]
'2016.2.4'
</code></pre>
<p>This, of course, needs testing and error handling.</p>
