<div class="post-text" itemprop="text">
<p>Your instance methods are missing the <code>self</code> reference:</p>
<pre><code>def computeCorrelation(self, newCorrel)
#                       ^

def add(self, x, y, newCorrel)
#        ^
</code></pre>
<p>The first parameter passed by an instance method is a reference to that instance of the class. So your <code>add</code> method <em>should</em> take 4 arguments but you will only be required to pass 3 when the method is called from an instance of the class. Python passes the reference to that instance for you.</p>
<hr/>
<p>More so, you are referencing your attributes from the class itself as <code>Blob.some_attribute</code>, you should instead use <code>self</code>:</p>
<pre><code>def __init__(self):
    self.mass = 0
    self.correlationVal = 0
</code></pre>
<p>This modifies those attributes only on that instance of the class which is what you intend with <code>__init__</code>. The same should apply to the other methods.</p>

<div class="post-text" itemprop="text">
<p>The first argument to a Python class method needs to be <code>self</code> i.e. <code>def add(self, x, y, newCorrel):</code> instead of <code>def add(x, y, newCorrel):</code>.</p>
<p>Self is implicitly passed to class instance methods when they are called, hence why 4 arguments are being passed instead of just the 3 you explicitly pass.</p>
<p>N.B. Non-instance methods (static methods) don't require self as their first parameter. See <a href="https://docs.python.org/2/library/functions.html#staticmethod" rel="nofollow">here</a> for more details.</p>
