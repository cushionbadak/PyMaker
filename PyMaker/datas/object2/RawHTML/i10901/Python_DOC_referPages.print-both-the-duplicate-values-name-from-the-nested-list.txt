<p>I would do it like that:</p>
<pre><code>students = [['Ash', 85.25], ['Kai', 85.25], ['Ray', 75], ['Jay', 55.5]]

common_names = []
for i, i_x in enumerate(students):
    for i_y in students[:i] + students[i + 1:]:
        if i_x[1] == i_y[1]:
            common_names.append(i_x[0])

print(common_names)
#['Ash', 'Kai']

# or if you want it to print every entry in a single line:
print('\n'.join(x for x in common_names))
#Ash
#kai
</code></pre>
<p>Explain:</p>
<ol>
<li>I grab an object from the original list <code>students</code>. Object is <code>i_x</code> and its <code>['Ash', 85.25]</code> on the first iteration for example.</li>
<li>Then i slice the list <code>students[:i] + students[i + 1:]</code> to create another one in memory that contains all the elements of the original one apart from <code>i_x</code></li>
<li>I check to see if there is any item in the newly created list that has the same <code>[1]</code> index value as that of <code>i_x</code>. If yes, i append the <code>i_x[0]</code> value to a third list that holds the results.</li>
<li>I do this for as many elements as there are originally in the <code>students</code> list.</li>
</ol>
