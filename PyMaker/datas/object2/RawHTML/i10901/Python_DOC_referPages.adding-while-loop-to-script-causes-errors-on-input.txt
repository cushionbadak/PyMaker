<div class="post-text" itemprop="text">
<p>The problem occurs because you mask the built-in function <strong><a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow"><code>input()</code></a></strong> with the assignment:</p>
<pre><code>input = input('Input: ')
</code></pre>
<p>For one iteration, this will work just fine because you're not calling <code>input()</code> again. Doing it more than once though will use a value of <code>input</code> that is masked, in your case with a value of type <code>str</code>; calling a <code>str</code> value will result in a <code>TypeError</code>.</p>
<p>Change the name to something different, like:</p>
<pre><code>my_input = input('Input: ')
</code></pre>
<p>for the error to leave, make sure you change it throughout your code.</p>
<p>Remember to never mix user defined names with <a href="https://docs.python.org/3/library/functions.html" rel="nofollow">built-in function names</a> because you'll get weird behavior.</p>
