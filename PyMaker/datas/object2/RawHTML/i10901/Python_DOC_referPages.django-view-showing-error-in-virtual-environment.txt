<div class="post-text" itemprop="text">
<p>In your <code>__init__.py</code> try to use local imports instead, this may be a problem if you are using <code>python3</code> in your virtual environment.</p>
<pre><code>from .VehicleView import *
from .DashboardView import *
</code></pre>
<p>Besides file and module names in python should follow snake case convention, only classes should use CamelCase.</p>
<pre><code>from .vehicle_view import *
form .dashboard_view import *
</code></pre>
