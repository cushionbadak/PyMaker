<p>I'm going to take an educated guess and assume you are using python3. In python3, urllib2 has been split into <a href="https://docs.python.org/3/library/urllib.request.html" rel="noreferrer"><code>urllib.request</code></a> and <a href="https://docs.python.org/3/library/urllib.error.html" rel="noreferrer"><code>urllib.error</code></a>. <a href="https://docs.python.org/2/library/urllib2.html#module-urllib2" rel="noreferrer">See note at the top of the urllib2 page</a>. The function you are looking for is contained in <code>urllib.request</code>. Try the following:</p>
<pre><code>from urllib.request import urlopen
page = urlopen("https://docs.python.org/3/howto/urllib2.html")
contents = page.read()
</code></pre>

<p>The answer to this question breaks down into two sections.  The solution differs based on if you are using python 2 or python 3.</p>
<p>In python 3 urllib2 is no longer used.  Try using urllib.request.</p>
<p>In python 2 you may just have a bad install or old version of urllib2.  Try pip install urllib2.</p>
