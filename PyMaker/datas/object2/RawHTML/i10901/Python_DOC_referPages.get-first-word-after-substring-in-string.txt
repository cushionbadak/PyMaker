<p>Hi I'm trying to get the word after a specified sub string, like...</p>
<pre><code>str = Quote from: Bob1 ...
</code></pre>
<p>I'm trying to search for everytime Quote from: appears and get the word after, in this case Bob1.</p>
<p>I've tried:</p>
<pre><code>print((re.findall(r'Quote from:\a\X\\9', str)))
</code></pre>
<p>But it just returns <code>[]</code>.</p>

<div class="post-text" itemprop="text">
<p>This should work for you, using <code>split</code>.</p>
<pre><code>&gt;&gt;&gt; str = "Quote from: Bob1 ..."
&gt;&gt;&gt; str.split("Quote from:")[1].split()[0]
'Bob1'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>import re

s = 'Quote from: Bob1 ...'
re.sub(r'Quote from: (\S+).*', r'\1', s)

'Bob1'
</code></pre>
