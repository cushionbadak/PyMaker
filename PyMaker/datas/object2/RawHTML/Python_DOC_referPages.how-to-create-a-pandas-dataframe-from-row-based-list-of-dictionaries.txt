<div class="post-text" itemprop="text">
<p>The <code>DataFrame</code> constructor takes row-based arrays (amoungst other structures) as data input. Therefore the following works:</p>
<pre><code>data = [{
  "name": "leopard",
  "character": "mean",
  "skills": ["sprinting", "hiding"],
  "pattern": "striped",
 },
 {
  "name": "antilope",
  "character": "good",
  "skills": ["running"],
 }]
df = pd.DataFrame(data)
print(df)
</code></pre>
<p>Output:</p>
<pre><code>  character      name  pattern               skills
0      mean   leopard  striped  [sprinting, hiding]
1      good  antilope      NaN            [running]
</code></pre>
