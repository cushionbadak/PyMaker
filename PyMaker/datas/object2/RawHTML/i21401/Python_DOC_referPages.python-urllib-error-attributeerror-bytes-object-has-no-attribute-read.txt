<div class="post-text" itemprop="text">
<p>Use <code>json.loads</code> not <code>json.load</code>.</p>
<p>(<code>load</code> loads from a file-like object, <code>loads</code> from a string. So you could just as well omit the <code>.read()</code> call instead.)</p>

html = page.read()
soup = BeautifulSoup(html)
print(soup.prettify('latin-1'))
