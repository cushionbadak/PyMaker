<div class="post-text" itemprop="text">
<p>I'm trying to encode non-ASCII characters so I can put them inside an url and use them in <code>urlopen</code>. The problem is that I want an encoding like JavaScript (that for example encodes <code>ó</code> as <code>%C3%B3</code>):</p>
<pre><code>encodeURIComponent(ó)
'%C3%B3'
</code></pre>
<p>But <code>urllib.quote</code> in python returns <code>ó</code> as <code>%F3</code>:</p>
<pre><code>urllib.quote(ó)
'%F3'
</code></pre>
<p>I want to know how to achieve an encoding like javascript's <code>encodeURIComponent</code> in Python, and also if I can encode non <code>ISO 8859-1</code> characters like Chinese. Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>You want to make sure you're using unicode.</p>
<p>Example:</p>
<pre><code>import urllib

s = u"ó"
print urllib.quote(s.encode("utf-8"))
</code></pre>
<p>Outputs:</p>
<p><code>%C3%B3</code></p>
</div>
<div class="post-text" itemprop="text">
<p>in <strong>Python 3</strong> the <code>urllib.quote</code> has been renamed to <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.quote" rel="noreferrer"><code>urllib.parse.quote</code></a>.</p>
<p>Also in Python 3 all strings are unicode strings (the byte strings are called <a href="https://docs.python.org/3/library/stdtypes.html#bytes" rel="noreferrer"><code>bytes</code></a>).</p>
<p>Example:</p>
<pre><code>from urllib.parse import quote

print(quote('ó'))
# output: %C3%B3
</code></pre>
