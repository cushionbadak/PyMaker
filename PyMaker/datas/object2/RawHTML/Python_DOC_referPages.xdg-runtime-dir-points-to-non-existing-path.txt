<div class="post-text" itemprop="text">
<p>Just create a directory using:</p>
<pre><code>$ mkdir -p /run/user/1000/snap.pycharm-community
</code></pre>
<p>And change it's permissions using:</p>
<pre><code>$ chmod 0700 /run/user/1000/snap.pycharm-community
</code></pre>
<p>Alternatively you could set your environment variable <code>XDG_RUNTIME_DIR</code> using </p>
<pre><code>$ export XDG_RUNTIME_DIR=/your/dir
</code></pre>
<p><a href="https://wiki.archlinux.org/index.php/XDG_Base_Directory" rel="nofollow noreferrer">Source and more information on the topic</a></p>
</div>
<span class="comment-copy"><code>/run/user/1000</code> is typically created by systemd, <i>already owned by uid 1000</i>, when the user with that UID logs in. There should not be a need for any further privileged invocations.</span>
