<p>I would like to ask how to exchange dates from loop in to an array in python?
I need an array of irregular, random dates with hours. So, I prepared a solution:</p>
<pre><code>import datetime
import radar
r2 =()
for a in range(1,10):
       r2 = r2+(radar.random_datetime(start='1985-05-01', stop='1985-05-04'),)
r3 = list(r2)
print(r3)
</code></pre>

<p>You can convert the datetime to a string with <a href="https://docs.python.org/3/library/stdtypes.html#str" rel="nofollow noreferrer"><code>str()</code></a> like:</p>
<h3>Code:</h3>
<pre><code>str(radar.random_datetime(start='1985-05-01', stop='1985-05-04'))
</code></pre>
<h3>Test Code:</h3>
<pre><code>import radar

r2 = ()
for a in range(1, 10):
    r2 = r2 + (str(
        radar.random_datetime(start='1985-05-01', stop='1985-05-04')),)
r3 = list(r2)
print(r3)
</code></pre>
<h3>Results:</h3>
<pre><code>['1985-05-01 21:06:29', '1985-05-01 04:43:11', '1985-05-02 13:51:03', 
 '1985-05-03 03:20:44', '1985-05-03 19:59:14', '1985-05-02 21:50:34', 
 '1985-05-01 04:13:50', '1985-05-03 23:28:36', '1985-05-02 15:56:23']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/2/library/time.html#time.strftime" rel="nofollow noreferrer">strftime</a> to convert the date generated by radar before adding it to the list.
e.g.</p>
<pre><code>import datetime
import radar
r2 =()
for a in range(1,10):
 t=datetime.datetime(radar.random_datetime(start='1985-05-01', stop='1985-05-04'))
 r2 = r2+(t.strftime('%Y-%m-%d %H:%M:%S'),)
r3 = list(r2)
print(r3)
</code></pre>
</div>
