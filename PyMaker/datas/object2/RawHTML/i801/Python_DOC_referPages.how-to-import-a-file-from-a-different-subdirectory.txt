<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/functions.html#__import__" rel="nofollow noreferrer"><code>__import__</code></a> function:</p>
<pre><code>models = __import__('models', fromlist=['v1'])
</code></pre>
