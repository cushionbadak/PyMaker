<p>As mentioned in the title - how can the default <code>(Pdb)</code> prompt be changed to something else? There doesn't appear to be a built-in way of setting the prompt per the <a href="https://docs.python.org/3/library/pdb.html" rel="nofollow noreferrer">docs</a> or <a href="https://github.com/python/cpython/blob/658ff844963b68b420d663c188cd5fc78442db3f/Lib/pdb.py" rel="nofollow noreferrer">source</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>One workaround could be to set it dynamically in <code>.pdbrc</code>, like</p>
<pre class="lang-py prettyprint-override"><code>!(lambda: exec('import gc, pdb; next(o for o in gc.get_objects() if isinstance(o, pdb.Pdb)).prompt = "pdb&gt; "', {}))()
</code></pre>
<p>which results in the prompt "<code>pdb&gt;</code>". This also avoids polluting the local scope.</p>
<p>Alternatively, use another debugger like <a href="https://pypi.org/project/pdbpp/" rel="nofollow noreferrer">pdb++</a> for which the prompt and other attributes can be configured (per <a href="https://github.com/antocuni/pdb#configuration-and-customization" rel="nofollow noreferrer">here</a>).</p>
</div>
