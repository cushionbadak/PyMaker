<p>I am using session in my Django Application. </p>
<p><code>request.session['permissions']</code> is structured in the following manner. The key is same for every values. Basically it is a list of dictionaries</p>
<pre><code>[{'mykey': 'value1'}, {'mykey': 'value2'}, {'mykey': 'value3'}, 
 {'mykey': 'value4'}, {'mykey': 'value5'}, {'mykey': 'value6'}]
</code></pre>
<p></p>
<pre><code>permissions = request.session['permissions']
</code></pre>
<p>Now I want to check if <code>value4</code> is present inside <code>permissions</code>. I am not getting how to check the value. </p>
<p>Whenever I am trying to access value like this </p>
<pre><code>permissions.mykey
</code></pre>
<p>It is giving me error </p>
<pre class="lang-none prettyprint-override"><code>'list' object has no attribute 'mykey'
</code></pre>
<p>I want to do similar like this </p>
<pre><code>permissions = request.session['permissions']
if value not in permissions:
   print('Value is inside list')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I think you can do it like this using <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer">filter</a>:</p>
<pre><code>permissions = request.session['permissions']
exists = list(filter(lambda x: x.get('mykey') == 'value4', permissions))  # for python 3

# for python2
#  exists = filter(lambda x: x.get('mykey') == 'value4', permissions)

if len(exists) &gt; 0:
   print('value4 exists')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>use a for loop to get the single dictionary and then check the key</p>
<pre><code>for perm in permissions: 
    if value in perm.values():
        print('Value is inside list')
</code></pre>

<div class="post-text" itemprop="text">
<p>As the other answers point out. The issue is that you are trying to use a dictionary accessor on a list object. Since <code>permissions</code> is a list of dictionaries you need to check all the dictionaries in the list for the <code>value4</code>. This can be done in many ways, in my mind the most pythonic way would be to use list comprehension, something like <code>[value4 == d for d['mykey'] in permissions]</code></p>
