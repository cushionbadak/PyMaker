<div class="post-text" itemprop="text">
<p>You could try to use <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer"><code>getattr()</code></a>:</p>
<pre><code># 'exch1_object' should be a string
func = getattr(ccxt, exch1_object)
exch1_object = func()
</code></pre>
<p>This will get the function from the module <code>ccxt</code> that has the name that is currently inside the variable <code>exch1_object</code> (which should be a string).</p>
<p>This can also be shortened to this:</p>
<pre><code>exch1_object = getattr(ccxt, exch1_object)()
</code></pre>
<p>you will have to decide which is clearer and more readable for your case.</p>
<hr/>
<p>So, for example, these lines:</p>
<pre><code>exch1_object = 'poloniex'
exch1_object = getattr(ccxt, exch1_object)()
</code></pre>
<p>will yield the same results as:</p>
<pre><code>exch1_object = ccxt.poloniex()
</code></pre>
