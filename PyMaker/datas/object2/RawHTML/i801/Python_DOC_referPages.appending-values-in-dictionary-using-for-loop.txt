<div class="post-text" itemprop="text">
<p>You have a couple of options. Either redefine <code>l</code> as an empty list <em>within</em> your outer <code>for</code> loop, so that the same list is not appended to throughout. Or use <a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow noreferrer"><code>collections.defaultdict</code></a> to append to a default list value. Here's an example of the second option:</p>
<pre><code>from collections import defaultdict

dic = defaultdict(list)
for i in range(2):
    c = input("Enter your name").strip()
    for i in range(3):
        dic[c].append(int(input("Enter Your marks: ")))
</code></pre>
<p>Using a <code>defaultdict</code> is more efficient and arguably more readable.</p>
</div>

<pre><code>dic={}
for i in range(0,2):

    c=input("Enter your name").strip()

</code></pre>
<p>That way, a new list object is instantiated at each iteration.</p>
