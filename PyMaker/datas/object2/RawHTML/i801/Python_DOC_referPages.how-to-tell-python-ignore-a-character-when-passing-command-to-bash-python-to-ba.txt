<div class="post-text" itemprop="text">
<p>You cannot nest single quotes. The string which starts with <code>os.system('</code> ends at the next (unescaped) single quote.</p>
<p>Python offers triple quotes which provide for a trivial fix:</p>
<pre><code>os.system(r"""composer transaction submit -c admin@tutorial-network -d '{"$class": "org.acme.frame.auction.SetupDemo1"}'""")
</code></pre>
<p>A better solution altogether is to use <code>subprocess.run</code> without <code>shell=True</code> so you don't have to understand both Python's and the shell's quoting mechanisms.</p>
<pre><code>subprocess.run([
        'composer', 'transaction',
        'submit', '-c', 'admin@tutorial-network',
        '-d', '{"$class": "org.acme.frame.auction.SetupDemo1"}'],
    # probably a good idea
    check=True)
</code></pre>
<p>For (much) more on this topic, see further <a href="https://stackoverflow.com/a/51950538/874188">https://stackoverflow.com/a/51950538/874188</a></p>
