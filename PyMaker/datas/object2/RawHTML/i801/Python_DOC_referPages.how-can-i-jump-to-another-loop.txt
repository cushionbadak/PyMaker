<p>You can break the inner loop to continue the outer loop</p>
<pre><code>while True:
    while True:
        # do some checks if you want to
        break

</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The <code>break</code> key word allows you to exit the presiding loop you are in. So if you have a statement like:</p>
<pre><code>while True:
    while True:
         break
</code></pre>
<p>The break will send you back to the first loop.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use break as suggested it exits the inner loop when a condition is met. Consider this example from [here <a href="https://docs.python.org/3/tutorial/controlflow.html][1]" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/controlflow.html][1]</a></p>
<pre><code>for n in range(2, 10):
 for x in range(2, n):
     if n % x == 0:
         print(n, 'equals', x, '*', n//x)
         break
 else:
     # loop fell through without finding a factor
     print(n, 'is a prime number')
</code></pre>
<p>results
2 is a prime number
3 is a prime number
4 equals 2 * 2
5 is a prime number
6 equals 2 * 3
7 is a prime number
8 equals 2 * 4
9 equals 3 * 3</p>
