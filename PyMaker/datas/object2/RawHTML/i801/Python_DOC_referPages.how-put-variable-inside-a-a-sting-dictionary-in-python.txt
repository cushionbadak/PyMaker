<p>I'm trying to interact with the Trakt.tv api, but i'm struggling in create a variable inside a string dictionary.</p>
<p>The api OAuth make a call to the server, that provides 2 access codes, one of those code need to be forwarded to a following server call. like this:</p>
<pre><code>values = """{
    "code": "code provide by the api",
    "client_id": "code provide by user",
    "client_secret": "code provide by user"
  }
""" 
headers = {
  'Content-Type': 'application/json'
}
request = Request('https://api.trakt.tv/oauth/device/token', data=values, headers=headers)

</code></pre>
<p>I have the needed code in variable api_code and would like to put this variable inside the string, like the following example.</p>
<pre><code>values = """{
    "code": f"api_code",
    "client_id": f"user_code",
    "client_secret": f"user_code_2"
  }
""" 
headers = {
  'Content-Type': 'application/json'
}
request = Request('https://api.trakt.tv/oauth/device/token', data=values, headers=headers)
</code></pre>

<div class="post-text" itemprop="text">
<p>IIUC you can use <code>.format()</code> like this:</p>
<pre><code>values = """{{
    "code": f"{0}",
    "client_id": f"user_code",
    "client_secret": f"user_code_2"
  }}
""" 

api_code = 'abc123'
values = values.format(api_code)
</code></pre>
<p>Note that in order to get actual <code>{}</code> characters in the format string, you need to double them as I have in lines 1 and 4 here</p>
</div>
<span class="comment-copy">Was just writing a longer answer when the question was closed, but don't put the variable in a string, instead use a real python dict and transform it via the <a href="https://docs.python.org/3/library/json.html" rel="nofollow noreferrer">json</a> module to valid json <code>data = {"code": api_code}; values = json.dumps(data)</code></span>
