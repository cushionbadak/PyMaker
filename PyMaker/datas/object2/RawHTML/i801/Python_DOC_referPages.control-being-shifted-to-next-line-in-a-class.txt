<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>print</code></a> function (in Python 3) defaults to printing a newline character:</p>
<blockquote>
<pre><code>print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
</code></pre>
<p>Print <code>objects</code> to the text stream file, separated by <code>sep</code> and followed by <code>end</code>.</p>
</blockquote>
<p>If you don't want this behavior, pass an empty string for <code>end</code>:</p>
<pre><code>print("hello ", end="")
print("world")
</code></pre>
</div>
