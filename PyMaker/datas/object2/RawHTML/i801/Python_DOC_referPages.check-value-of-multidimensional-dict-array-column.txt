<div class="post-text" itemprop="text">
<p>You can use a list comprehension:</p>
<pre><code>multi_dict = [
    {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'},
    {'key1': 'value10', 'key2': 'value20', 'key3': 'value30'}
]

result = [d['key1'] for d in multi_dict]
print(result)
</code></pre>
<p>Output:</p>
<pre><code>['value1', 'value10']
</code></pre>
<hr/>
<p>If you wanted to emulate the third argument of <code>array_column</code>, the index key, you can use dictionary comprehension:</p>
<pre><code>result = {d['key1']: d['key2'] for d in multi_dict}
print(result)
</code></pre>
<p>Output:</p>
<pre><code>{'value1': 'value2', 'value10': 'value20'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can extract all values for a key of all dicts:</p>
<pre><code>multi_dict = [ {'key1': 'value1', 'key2': 'value2', 'key3': 'value3'},
               {'key1': 'value10', 'key2': 'value20', 'key3': 'value30'},
               {                   'key2': 'value30', 'key3': 'value40'},
             ]

val_key1 = [d.get('key1') for d in multi_dict] 

print(val_key1)  
</code></pre>
<p>Output:</p>
<pre><code>['value1', 'value10', None]
</code></pre>
<p>You should use <code>dict.get(key)</code> to get a <code>None</code> in case a key is missing.</p>
