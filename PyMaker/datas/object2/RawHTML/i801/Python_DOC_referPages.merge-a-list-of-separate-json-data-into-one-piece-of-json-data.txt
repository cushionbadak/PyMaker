<p>In Python, I am trying to turn a list of separate JSON data:</p>
<pre><code>[[{'id': 1, 'name': 'pencil', 'description': '2b or not 2b, that is the question'}], [{'id': 2, 'name': 'oil pastel', 'description': None}], [{'id': 3, 'name': 'gouache', 'description': None}], [{'id': 4, 'name': 'paper', 'description': None}]]
</code></pre>
<p>into one piece of JSON data:</p>
<pre><code>{'id': 1, 'name': 'pencil', 'description': '2b or not 2b, that is the question'}, {'id': 2, 'name': 'oil pastel', 'description': None}, {'id': 3, 'name': 'gouache', 'description': None}, {'id': 4, 'name': 'paper', 'description': None}, {'id': 5, 'name': 'coloured pencil', 'description': None}
</code></pre>
<p>Been struggling with it for a few hours. Does anyone have any ideas?</p>

<div class="post-text" itemprop="text">
<p>Use simple list-comprehension</p>
<pre><code>[y for x in list_of_lists for y in x]
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>[{'description': '2b or not 2b, that is the question', 'id': 1, 'name': 'pencil'}, {'description': None, 'id': 2, 'name': 'oil pastel'}, {'description': None, 'id': 3, 'name': 'gouache'}, {'description': None, 'id': 4, 'name': 'paper'}]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow noreferrer"><code>itertools.chain</code></a></p>
<pre><code>&gt;&gt;&gt; list(itertools.chain.from_iterable(j))
</code></pre>
<p>Or a list comprehension </p>
<pre><code>&gt;&gt;&gt; [x[0] for x in j] # Assuming there is only one item in each list
</code></pre>
<p>Both outputs</p>
<pre><code>[{'id': 1,
  'name': 'pencil',
  'description': '2b or not 2b, that is the question'},
 {'id': 2, 'name': 'oil pastel', 'description': None},
 {'id': 3, 'name': 'gouache', 'description': None},
 {'id': 4, 'name': 'paper', 'description': None}]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <code>functools</code> with <code>operator</code></p>
<pre><code> j = [[{'id': 1, 'name': 'pencil', 'description': '2b or not 2b, that is the question'}], [{'id': 2, 'name': 'oil pastel', 'description': None}], [{'id': 3, 'name': 'gouache', 'description': None}], [{'id': 4, 'name': 'paper', 'description': None}]]


import functools
import operator
functools.reduce(operator.iadd,j,[])
</code></pre>
<p>Output:</p>
<pre><code>[{'id': 1,
'name': 'pencil',
  'description': '2b or not 2b, that is the question'},
 {'id': 2, 'name': 'oil pastel', 'description': None},
 {'id': 3, 'name': 'gouache', 'description': None},
 {'id': 4, 'name': 'paper', 'description': None}]
</code></pre>
</div>
