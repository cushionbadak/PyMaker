<p>The <code>return</code> on the last line of function definition is within the for loop, because of this the function will return on the first iteration, and further iterations will never happen. Actually in python you do not need to write return at the end of the function, it will default to returning <code>None</code>.</p>
<p>or change the identation: </p>
