<div class="post-text" itemprop="text">
<p>You don't define <code>value</code> if no conditions execute. So, what value must be printed if <code>x == '1'</code>?</p>
<p>The solution is to define a default value before <code>if</code>:</p>
<pre><code>for x in setluv:
    value = -1

    if x=='a':
        value=int(1)
    # ... etc

    print(str(value))
</code></pre>

<div class="post-text" itemprop="text">
<p>On a side note (too long for comment), your code could be simplified quite a bit. For instance, you could be using a look up table as in:</p>
<pre><code>import string
tmp = list(string.ascii_lowercase)
lookup = dict(zip(tmp, range(1, len(tmp)+1)))

def letterbreak(luv):
    setluv=list(str(luv))
    for x in setluv:
        print(str(lookup.get(x, "Value not defined")))
    return
</code></pre>
