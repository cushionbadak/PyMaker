<p>Manually formatting dates is error prone - you changed the name of your variables - <code>month</code> is never used to create the URI and <code>mesec</code> stays at 10 because it is never reassigned for <code>d.month &gt; 9</code>. Seee @metatoaster's comment.</p>
<p>Instead of manually formatting, use <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">strftime - date formatting</a>:</p>
<pre><code>import datetime
d = datetime.date(2012,1,1)

# avoid off-by-1 leap-year mishaps due to hardcoded days/year
while d.year &lt; 2013:   
    # format as dd.mm.yyyy including leading 0 if need be
    dd = d.strftime("%d.%m.%Y")
    print("http://www.ljse.si/cgi-bin/jve.cgi?doc=2561&amp;subtab=0_2&amp;date1="+dd)

    d = d + datetime.timedelta(days=1)
</code></pre>
