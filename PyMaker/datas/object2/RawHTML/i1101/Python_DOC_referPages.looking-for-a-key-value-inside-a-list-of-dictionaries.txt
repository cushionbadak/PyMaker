def AddItem(id):

<div class="post-text" itemprop="text">
<p>We create a list of all the values of the nested dictionaries and search them.</p>
<pre><code>TodoList=[{'id':5} ,{'id':3}  ,{'id':6}];

def AddItem(id):
     values_TodoList = [i['id'] for i in TodoList]
     if id in values_TodoList:
          print('inside');
     else:
          print('not inside')

AddItem(5)
    inside

AddItem(10)
    not inside
</code></pre>

<div class="post-text" itemprop="text">
<p>A bit more efficient than the other answers since it will stop as soon as it finds a first match (and also supports the case where these dictionaries will have more key-value pairs):</p>
<pre><code>TodoList = [{'id': 5}, {'id': 3}, {'id': 6}]

def AddItem(i):
    if any(d['id'] == i for d in TodoList):
        print('inside')

AddItem(5)
</code></pre>

<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer">any</a>:</p>
<pre><code>TodoList=[{'id':5} ,{'id':3}  ,{'id':6}];
def AddItem(i):
     if any(i in d for d in TodoList):
          print('inside')
</code></pre>
<p>As a side-note do not use id as a variable name it shadows the built-in function <a href="https://docs.python.org/3/library/functions.html#id" rel="nofollow noreferrer">id</a>. If you want to search for a given value inside the dictionaries this will do as suggested by @DeepSpace:</p>
<pre><code>def AddItem(i):
    if any(i in d.values() for d in TodoList):
        print('inside')


AddItem(5)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>inside
</code></pre>
