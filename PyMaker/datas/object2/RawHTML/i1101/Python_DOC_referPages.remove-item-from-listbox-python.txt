<pre><code>s.index(x[, i[, j]])    
index of the first occurrence of x in s (at or after index i and before index j)    
</code></pre>
<blockquote>
<p><strong>Note</strong>: (8)
  index raises ValueError when x is not found in s. Not all implementations support passing the additional arguments i and j. These arguments allow efficient searching of subsections of the sequence. Passing the extra arguments is roughly equivalent to using s[i:j].index(x), only without copying any data and with the returned index being relative to the start of the sequence rather than the start of the slice.</p>
</blockquote></li>

<ol>
<li><p>Get the index of a <code>Listbox</code> Item using it's label string</p>
<pre><code>label = "Shrimp Tempura"
idx = listbox.get(0, tk.END).index(label)
</code></pre></li>
<li><p>Delete the <code>Listbox</code> Item at this index</p>
<pre><code>listbox.delete(idx)
</code></pre></li>
