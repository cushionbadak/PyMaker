<p>Looks like you want to keep the first word (which is not "trimming" though). So you do two things</p>
<ol>
<li>break the string into a list of words (where "word" is something separated by spaces)</li>
<li><p>take the first element of that list</p>
<pre><code>words = mystring.split(' ')
result = words[0]
</code></pre></li>
</ol>

<div class="post-text" itemprop="text">
<p>For a slicing answer:</p>
<pre><code>def sub_string(str, start,end):
    return str[start:end]
</code></pre>
<p>You can also use <code>split</code>, by definition, this splits by spaces, if any other delimiter needed, you can identity it inside the split arguments <code>split(',')</code></p>
<pre><code>def split_string(str):
    return str.split()
</code></pre>
<p>This function will return an array of strings.  Choose whichever you want from this array</p>

</code></pre>

<div class="post-text" itemprop="text">
<p>You can use str's <a href="https://docs.python.org/3/library/stdtypes.html#str.partition" rel="nofollow noreferrer">partition</a> method to avoid creating a list like <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer">str.split</a></p>
<pre><code>&gt;&gt;&gt; first_word, _, _ = s.partition(' ')  # _ is a convention for a throwaway variable
&gt;&gt;&gt; print(first_word)
bangalore
</code></pre>
<p><code>str.partition</code> takes one argument - the separator - and returns the parts of the string before and after the first occurrence of the separator.</p>
