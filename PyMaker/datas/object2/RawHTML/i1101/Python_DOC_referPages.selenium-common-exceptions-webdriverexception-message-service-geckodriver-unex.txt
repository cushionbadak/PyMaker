<div class="post-text" itemprop="text">
<p>This error message...</p>
<pre><code>selenium.common.exceptions.WebDriverException: Message: Service geckodriver unexpectedly exited. Status code was: -11
</code></pre>
<p>...implies that the <a href="https://docs.python.org/3/library/subprocess.html#module-subprocess" rel="nofollow noreferrer"><strong>subprocess</strong></a> <a href="http://pubs.opengroup.org/onlinepubs/9699919799/functions/exit.html" rel="nofollow noreferrer"><strong><code>exited</code></strong></a> and <strong>Status code was: -11</strong></p>
<p>You need to download the latest matching <strong>geckodriver</strong> from <a href="https://github.com/mozilla/geckodriver/releases" rel="nofollow noreferrer"><strong>Releases Â· mozilla/geckodriver</strong></a>. </p>
<p>As you are on <strong>Raspberry Pi Zero W</strong> you need to download <strong><code>geckodriver-v0.23.0-arm7hf.tar.gz</code></strong> and save it within your system. Additionally, you need to mention the <em>absolute path</em> of the <strong>geckodriver</strong> binary passing the <em>argument</em> <strong><code>executable_path</code></strong> as follows:</p>
<pre><code>from selenium import webdriver

driver = webdriver.Firefox(executable_path='/path/to/geckodriver')
driver.get("http://google.com/")
driver.quit()
</code></pre>
