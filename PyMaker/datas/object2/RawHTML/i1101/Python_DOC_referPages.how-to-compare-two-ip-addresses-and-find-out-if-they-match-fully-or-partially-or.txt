<p>Use Python 3 ipaddress library. I guess you need to check if networks of <code>a</code> and <code>b</code> IP addresses overlap. </p>
<p>If we construct <a href="https://docs.python.org/3/library/ipaddress.html#interface-objects" rel="nofollow noreferrer">Interface Objects</a>, each one will have it's own network attribute accessible through <code>.network</code>.</p>
<pre><code>from ipaddress import ip_interface
a = ip_interface('10.10.1.2/25')
b = ip_interface('10.10.1.6/24')

print(b.network.overlaps(a.network))
</code></pre>
<p><strong>Output</strong>:</p>
<pre><code>Python 3.6.1 (default, Dec 2015, 13:05:11)
[GCC 4.8.2] on linux
&gt; True
</code></pre>
