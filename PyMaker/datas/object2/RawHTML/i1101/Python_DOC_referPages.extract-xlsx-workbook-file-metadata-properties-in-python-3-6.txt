<p>How can I read the properties/metadata like Title, Author, Subject, Last modified and Keywords stored in a xlsx file using Python? I've used <strong>xlrd</strong> library however, there are no such properties to extract theses entities.

<div class="post-text" itemprop="text">
<p>You may be interested in <a href="https://openpyxl.readthedocs.io/en/stable/index.html" rel="nofollow noreferrer">openpyxl</a>:</p>
<p>Something to get you started may look like:</p>
<pre><code>from openpyxl import load_workbook
wb = load_workbook('yourfile.xlsx')
wb.properties
</code></pre>
<p>Here's the sample output:</p>
<pre><code>&lt;openpyxl.packaging.core.DocumentProperties object&gt;
Parameters:
creator=u'User', title=None, description=None, subject=None, identifier=None,
language=None, created=datetime.datetime(2018, 12, 11, 9, 55, 2),
modified=datetime.datetime(2018, 12, 11, 10, 30, 38), lastModifiedBy=u'User',
category=None, contentStatus=None, version=None, revision=None, keywords=None,
lastPrinted=None
</code></pre>
<p>Is this something you can work with?</p>

<div class="post-text" itemprop="text">
<p>You should be looking at <code>os</code> library: <a href="https://docs.python.org/3/library/os.html" rel="nofollow noreferrer">https://docs.python.org/3/library/os.html</a> because the metadata are stored above the contents of your file level.<br/>
Example: <code>os.path.getmtime</code> gets you the last modified time.</p>
