<p>Try this? The function test should do what you need.</p>
<pre><code>import re

def test(string):
    m = re.search(r'\d+$', string)
    if m is not None:
        print(m.group())
</code></pre>

<div class="post-text" itemprop="text">
<pre><code>import re
s=r"""1231ololo767980716"""
re.match('.*?([0-9]+)$', s).group(1)

&gt;&gt;&gt; '767980716'
</code></pre>
<p>or </p>
<pre><code>next(re.finditer(r'\d+$', s)).group(0)
&gt;&gt;&gt; '767980716'
</code></pre>
<p>Explanation of regexp components:</p>
<ul>
<li><p><code>.*?</code> is a non-greedy match and consumes only as much as possible (a
greedy match would consume everything except for the last digit).</p></li>
<li><p><code>[0-9]</code> and <code>\d</code> are two different ways of capturing digits. Note that the latter also matches digits in other writing schemes, like <code>рнк</code>
or <code>р╡и</code></p></li>
<li>Parentheses (()) make the content of the expression a group, which can be retrieved with group(1) (or 2 for the second group, 0  for the whole match).</li>
<li><code>+</code> means multiple entries (at least one number at the end).</li>
<li><code>$</code> matches only the end of the input.</li>
</ul>
<p>and all components you can find here: <a href="https://www.rexegg.com/regex-quickstart.html" rel="nofollow noreferrer">https://www.rexegg.com/regex-quickstart.html</a> very helpful cheat sheet      </p>

<div class="post-text" itemprop="text">
<p>This code which uses regex <code>\d+$</code> should get you what you are exactly looking for,</p>
<pre><code>import re

arr = ['2te2xt12',
'text1',
'text143',
'te2xt341']

for s in arr:
 m = re.search(r'\d+$', s)
 if (m):
  print(s + ' &gt;&gt; ' + m.group())
</code></pre>
<p>Prints,</p>
<pre><code>2te2xt12 &gt;&gt; 12
text1 &gt;&gt; 1
text143 &gt;&gt; 143
te2xt341 &gt;&gt; 341
</code></pre>
</div>
