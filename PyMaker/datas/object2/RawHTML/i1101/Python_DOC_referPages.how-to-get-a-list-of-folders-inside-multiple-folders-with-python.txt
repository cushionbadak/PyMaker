<p>As mentioned in the <a href="https://docs.python.org/3/library/os.html#os.walk" rel="nofollow noreferrer">os.walk() documentation</a>, <code>os.walk()</code> returns a 3-tuple of <code>(dirpath, dirnames, filenames)</code> and functions as an iterable. In python, this means that it may be used as the second operand of a for loop i.e. as the iterable in the syntax </p>
<pre><code>for identifier in iterable:
    [some operation with "identifer" for each element of "iterable"]
</code></pre>
<p>In the case of <code>os.walk()</code>, we can easily construct a list of all folders with </p>
<pre><code>import os

def get_all_folders(path):
    folder_list = []
    for dirpath, dnames, fnames in os.walk(path):
        folder_list += list(dnames)
    return folder_list


print(get_all_folders(os.getcwd()))
</code></pre>
<p>Or, using the more succinct list comprehension with <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow noreferrer">itertools.chain()</a>:</p>
<pre><code>import os

def get_all_folders(path):
    return list(chain(*[dir_list for _, dir_list, _ in os.walk(path)]))


print(get_all_folders(os.getcwd()))
</code></pre>
