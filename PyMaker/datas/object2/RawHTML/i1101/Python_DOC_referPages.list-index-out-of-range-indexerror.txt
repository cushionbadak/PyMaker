<p>I'm trying to simulate Monopoly on Python but when I try to have dice roll a random number between 1 and 6.</p>
<pre><code>from random import randint

dice_numbers = [1,2,3,4,5,6]
dice1 = dice_numbers[randint(0, 6)]
dice2 = dice_numbers[randint(0, 6)]
</code></pre>
<p>When rolled, and one of the dice numbers is 6 it comes up with a </p>
<blockquote>
<p>Indexerror: list index out of range</p>
</blockquote>
<p>I tested if it was the number 6 by printing the numbers.</p>
<pre><code>print(dice1, dice2)
</code></pre>
<p>And all of the numbers came up alright but I did not see 6 once.</p>
<p>Any ideas why?</p>

<div class="post-text" itemprop="text">
<p>You get an indexing error because your generated index ranges from 1 to 6 and your list's indexes start at 0 and go up to 6:</p>
<pre><code>[1 , 2 , 3 , 4 , 5 , 6]   # list
 0   1   2   3   4   5    # index
</code></pre>
<p>You create a <em>random index</em> to index into a <em>list of numbers</em> to get a number from it ... that is far too complicated. You do not need the list of numbers - you can create the number you want directly.</p>
<p>To get a random integer use</p>
<pre><code>import random

print( random.randint(1,6) )  # one integer between 1 and 6 (inclusivly)
</code></pre>
<p>To get <em>one</em> element from an itertable use</p>
<pre><code>print( random.choice( [1,2,3,4,5,6] ) )  # chooses one element randomly
</code></pre>
<p>To get a random number between <em>a</em> and <em>b</em> you can also use</p>
<pre><code>a = 1
b = 6  # range(a,b) is exclusively - you need to add 1 to the upper limit to include it
print (random.choice( range(a, b+1) ) )  # return one number from the range [a,..,b]
</code></pre>
<p>To get multiple values from an iterable use</p>
<pre><code>print( random.choices( range( 1, 7), k= 100) ) # return 100 numbers as list between 1 and 6
</code></pre>
<p>You can find the whole random documentation here: <a href="https://docs.python.org/3/library/random.html" rel="nofollow noreferrer">python.org: module random</a></p>

<li><a href="https://docs.python.org/3/library/random.html#random.randint" rel="nofollow noreferrer">random.randint</a></li>

</ul>
