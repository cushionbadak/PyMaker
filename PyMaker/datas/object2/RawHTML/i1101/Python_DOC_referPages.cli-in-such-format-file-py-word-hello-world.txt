<p>Let's say I have such function:</p>
<pre><code>def say(word):
    print(word)
</code></pre>
<p>And I want to run it through CMD in such way:</p>
<pre><code>file.py --word 'Hello world!'
</code></pre>
<p>I know it works somehow that way, but I can't find out how it is.</p>
</div>
<div class="post-text" itemprop="text">
<p>use the argparse module</p>
<p><a href="https://docs.python.org/3/library/argparse.html" rel="nofollow noreferrer">https://docs.python.org/3/library/argparse.html</a></p>
<pre><code>import argparse

parser = argparse.ArgumentParser()
parser.add_argument('word', type=str, required=True)
args = parser.parse_args()
print(args.word)  # say(args.word)
</code></pre>
