<p>You could do that using <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer">itertools.groupby</a>:</p>
<pre><code>from itertools import groupby

string = 'AbcDEfGhIJK'
out = [''.join(group) for key, group in groupby(string, key=lambda c: c.islower())]
print(out)
# ['A', 'bc', 'DE', 'f', 'G', 'h', 'IJK']
</code></pre>
<p>Here, <code>groupby</code> will group the characters that give the same output for <code>islower()</code></p>
</div>
<div class="post-text" itemprop="text">
<p>You could use a regex:</p>
<pre><code>import re

text = 'AbcDEfGhIJK'

result = re.split('([a-z]+)', text)
print(result)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>['A', 'bc', 'DE', 'f', 'G', 'h', 'IJK']
</code></pre>
<p>The idea is to split the string on lower case letters <code>'([a-z]+)'</code>, but keeping the splitting pattern.</p>
