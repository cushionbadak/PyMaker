<p>datetime.date.today() does not give you the current hour and minute.</p>
<pre><code>import logging, datetime
a=datetime.datetime.now()
b='D:\\'+a.strftime("%y_%B_%a_%H%M")+'s_python.log'
print(b)
logging.basicConfig(filename=b,level=logging.DEBUG,format="%(levelname)s %(asctime)s-%(message)s")
logging.debug("Hi Sudhirrrrrrrrr")
</code></pre>
</div>
<span class="comment-copy">You will need a special logging handler for that. See <a href="https://docs.python.org/3/library/logging.handlers.html" rel="nofollow noreferrer">the docs</a> about logging handlers for details.</span>
<span class="comment-copy">you'll need to specify the %b in the format section and not the filename.</span>
<span class="comment-copy">Did you try <code>filename="D:\{0}_python.log".format(b),level=logging.DEBUG,format="{0} {1}-{2}".format(levelname, asctime, message)</code>?</span>
