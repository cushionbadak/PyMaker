<div class="post-text" itemprop="text">
<p>Convert the string to a time object with <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer"><code>strptime()</code></a> first.</p>
<pre><code>&gt;&gt;&gt; import time
&gt;&gt;&gt; temp_time = '2006-11-06T09:49:43.000+08:00'
&gt;&gt;&gt; s_time = time.strptime(temp_time, "%Y-%m-%dT%H:%M:%S.000+08:00")
&gt;&gt;&gt; time.strftime("%Y-%m-%d %H:%M:%S", s_time)
'2006-11-06 09:49:43'
</code></pre>
