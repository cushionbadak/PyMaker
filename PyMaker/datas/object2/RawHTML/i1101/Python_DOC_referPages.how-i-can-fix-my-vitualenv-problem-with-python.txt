<p>I was trying to create my first django project so I installed the virtualenv. after that I wrote typed this into the terminal: </p>
<pre><code>~$ virtualenv --python-python3 firstdjango 
</code></pre>
<p>My terminal gave me this error: </p>
<pre><code>Usage: virtualenv [OPTIONS] DEST_DIR  
virtualenv: error: no such option: --python-python3
</code></pre>
<p>What should I do?</p>
</div>
<div class="post-text" itemprop="text">
<p>Since you're using python 3 you should use:</p>
<pre><code>python3 -m venv /path/to/new/virtual/environment
</code></pre>
<p>as documented here: <a href="https://docs.python.org/3/library/venv.html" rel="nofollow noreferrer">Creation of virtual environments</a></p>
<p>[FYI: the problem in your original is the that flag takes an <code>=</code> like <code>--python=python3</code>]</p>
