<p>The cause of your error is quite simple and obvious: you defined the function as <code>displayUserArray(UserArray, num)</code> but call it with <code>displayUserArray(numOfUsers, userArray)</code> - IOW you pass the arguments in the wrong order.</p>
<p>This being said, almost all your code is wrong, you obviously don't get the difference between a class and instance and how to use a class to create instances. I strongly suggest you read <em>at least</em> <a href="https://docs.python.org/3/tutorial/classes.html#class-objects" rel="nofollow noreferrer">the official tutorial</a>, and check a couple other tutorial and/or example code on the topic of classes and instances.</p>
