<div class="post-text" itemprop="text">
<p>This is five arguments being passed to <code>input</code>:</p>
<pre><code>input("What is", RanNum1, " - ", RanNum2, " = ?")
</code></pre>
<p>Use the <code>str.format</code> method to provide a single string to input.</p>
<pre><code>inputstring = "What is {0} - {1} = ?".format(RanNum1, RanNum2)
userInput= int(input(inputstring))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>input</code> expects a single string, not 5 arguments.</p>
<p>You can use the <code>format</code> function to generate your string using these variables.</p>
<pre><code>userInput= int(input("What is {} - {} = ?".format(RanNum1, RanNum2))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The error pretty clear... You're passing 5 arguments:</p>
<pre><code>userInput= int(input("What is", RanNum1," - ", RanNum2," = ?"))
                         ^^^      ^^^    ^^^     ^^^     ^^^
                          1        2      3       4       5
</code></pre>
<p>Coming from Java, are we? <strong>:-)</strong></p>
<p>If you want to display variables in an string in python, you can take a look to <a href="https://docs.python.org/2/library/string.html#format-string-syntax" rel="nofollow">string formatting</a> (what other answers use) or you could also do:</p>
<pre><code>userInput= int(input("What is %s - %s = ?" % (RanNum1, RanNum2)))
</code></pre>
<p>Also, if you're running this using Python 2.x (any version less than 3) I strongly recommend using <a href="https://docs.python.org/2/library/functions.html#raw_input" rel="nofollow">raw_input</a> instead of <a href="https://docs.python.org/2/library/functions.html#input" rel="nofollow">input</a>. Input will <a href="https://docs.python.org/2/library/functions.html#eval" rel="nofollow">eval</a> the user's input as python code, and <a href="http://nedbatchelder.com/blog/201206/eval_really_is_dangerous.html" rel="nofollow">eval is evil</a>!</p>
</div>
<div class="post-text" itemprop="text">
<p>Although <code>print</code> optionally takes multiple argument, <code>input</code> doesn't.</p>
<p>You either need to use the <code>format</code> function - <code>userInput= int(input("What is {} - {} = ?".format(RanNum1, RanNum2))</code>, or add the values <code>userInput= int(input("What is" + str(RanNum1) + " - " + str(RanNum2) + " = ?"))</code> (the first option there is best).</p>
</div>
