<div class="post-text" itemprop="text">
<p>Your <code>giveresult()</code> function is not <em>returning</em> anything, so the default <code>None</code> is returned. Printing is not the same thing; you are writing text to the terminal, not returning and the caller cannot use text written to a terminal.</p>
<p>Replace <code>print()</code> with <code>return</code>:</p>
<pre><code>def giveResult(tempDice):
    if fiveofakind(tempDice):
        return 10
    else:
        return 0
</code></pre>
<p>I also simplified your function; there is no need to test for <code>is True</code>; <code>if</code> already tests if the result of <code>fiveofakind()</code> is true or not. And because you already tested for <code>fiveofakind()</code>, all you need to do is use <code>else</code> to pick the other case.</p>
<p>Next, avoid calling <code>giveResult</code> more than per dice, and again, <em>return</em> the result from your function:</p>
<pre><code>def compareresult():
    dice_result = giveResult(dice)
    newdice_result = giveResult(newdice)
    if dice_result &gt; newdice_result
        return dice_result

    elif newdice_result &gt; dice_result:
        return newdice_result

    return dice, newdice
</code></pre>
<p>If you must return the greater result, just use the <a href="https://docs.python.org/3/library/functions.html#max" rel="nofollow"><code>max()</code> function</a>:</p>
<pre><code>def compareresult():
    dice_result = giveResult(dice)
    newdice_result = giveResult(newdice)
    return max(dice_result, newdice_result)
</code></pre>
