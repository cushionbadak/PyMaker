<p>I'm having some issues trying to split a string by punctuation. Specifically every character that belongs to string.punctuation. For example for the string</p>
<p>Input: </p>
<pre><code>"Hi. How2 are you doing??    "
</code></pre>
<p>I would like the output:</p>
<pre><code>['Hi', '.', 'How2', 'are', 'you', 'doing', '?', '?')
</code></pre>
<p>For reference string.punctuation is:</p>
<pre><code>'!"#$%&amp;\'()*+,-./:;&lt;=&gt;?@[\\]^_`{|}~'
</code></pre>
<p>I would like any character in the string to be its own token. </p>
<p>I've currently tried using <code>re.split()</code>. However, all I've been able to achieve is splitting based on non word characters. For example: </p>
<pre><code>print([i for i in re.split(r'(\W+)', 'hi. how are you   doing') if i])
</code></pre>
<p>which gives the output:</p>
<pre><code>['hi', '. ', 'how', ' ', 'are', ' ', 'you', '   ', 'doing', '??']
</code></pre>
<p>Furthermore, the two <code>'??'</code> should be split. Any suggestions? </p>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow"><code>re.findall</code></a>:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; mystr = "Hi. How2 are you doing??    "
&gt;&gt;&gt; re.findall(r'[.?]|\w+', mystr)
['Hi', '.', 'How2', 'are', 'you', 'doing', '?', '?']
&gt;&gt;&gt;
</code></pre>
<p>The pattern above finds all occurrences of <code>.</code>, <code>?</code>, or a series of one or more word characters.  I kept the pattern simple just to demonstrate, but any other punctuation characters that you want to capture would be placed inside the character set <code>[...]</code>.</p>
<p>Also, you may want to use <a href="https://docs.python.org/3/library/string.html#string.punctuation" rel="nofollow"><code>string.punctuation</code></a> instead of typing out all of those punctuation characters.  </p>

<pre><code>&gt;&gt;&gt; import nltk

&gt;&gt;&gt; nltk.word_tokenize(a)
['Hi.', 'How2', 'are', 'you', 'doing', '?', '?']
&gt;&gt;&gt; map(nltk.word_tokenize,nltk.sent_tokenize(a))
[['Hi', '.'], ['How2', 'are', 'you', 'doing', '?'], ['?']]
</code></pre>

<p>A little late on this.  </p>
<p>You should be able to do this with split().  </p>
<p>All you're doing is consuming whitespace, so everything else is between characters.  </p>
<p>Example:  <code>\s+|\b|(?=\pP)</code> will split on (and consume) whitespace, or word boundry,<br/>
or punctuation (within a look ahead).  </p>
<p>Don't have Python to test with, but in Perl  </p>
<pre><code>   $data =~ s/^\s+//;      # &lt;- trim leading spaces (a Perl problem)
   my @vals = split /\s+|\b|(?=\pP)/, $data;  
</code></pre>
<p>gives <code>['Hi', '.', 'How2', 'are', 'you', 'doing', '?', '?']</code> </p>
<p>For reference, this raw regex is the same thing   </p>
<pre><code>\s+|\b|(?=[!"#$%&amp;'()*+,\-./:;&lt;=&gt;?@\[\\\]^_`{|}~])  
</code></pre>
<p>but is with the punctuation characters in your list.  </p>
<p><em>Note that the inverse is to find all words and punctuation</em>.  For every split there<br/>
is a negative find all ...</p>
