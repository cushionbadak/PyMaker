<div class="post-text" itemprop="text">
<p>I am trying to remove a key and value from an <code>OrderedDict</code> but when I use:</p>
<pre><code>dictionary.popitem(key)
</code></pre>
<p>it removes the last key and value even when a different key is supplied. Is it possible to remove a key in the middle if the dictionary?</p>
</div>
<div class="post-text" itemprop="text">
<p>Yes, you can use <a href="https://docs.python.org/3/reference/simple_stmts.html#grammar-token-del_stmt"><code>del</code></a>:</p>
<pre><code>del dct[key]
</code></pre>
<p>Below is a demonstration:</p>
<pre><code>&gt;&gt;&gt; from collections import OrderedDict
&gt;&gt;&gt; dct = OrderedDict()
&gt;&gt;&gt; dct['a'] = 1
&gt;&gt;&gt; dct['b'] = 2
&gt;&gt;&gt; dct['c'] = 3
&gt;&gt;&gt; dct
OrderedDict([('a', 1), ('b', 2), ('c', 3)])
&gt;&gt;&gt; del dct['b']
&gt;&gt;&gt; dct
OrderedDict([('a', 1), ('c', 3)])
&gt;&gt;&gt;
</code></pre>
<p>In fact, you should always use <code>del</code> to remove an item from a dictionary.  <code>dict.pop</code> and <code>dict.popitem</code> are used to remove an item <strong>and return the removed item</strong> so that it can be saved for later.  If you do not need to save it however, then using these methods is less efficient.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use pop, popitem removes the last by default:</p>
<pre><code>d = OrderedDict([(1,2),(3,4)])
d.pop(your_key)
</code></pre>
