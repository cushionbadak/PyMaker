<p>Not sure exactly what you mean by the scoring as you don't discuss how you arrive at the score of 40. However, one way of checking if all elements in a list are the same would be:</p>
<pre><code>totals.count(totals[0]) == len(totals)
</code></pre>
<p>This simply returns <code>True</code> if they are all the same or <code>False</code> if not.</p>

<p>Why don't you create your simple function?</p>
<pre><code>def all_equal(totals):
    if len(totals) &gt; 1:
        start = totals[0] # compare with the first element
        for i in range(1, len(totals)):
            if totals[i] != start:
                 return False
    return True
#
totals = [12, 14, 28]
totals_2 = [12, 12, 12]
