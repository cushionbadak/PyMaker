<div class="post-text" itemprop="text">
<p>Is there a build-in function in python 3 to let me get <code>b</code> from <code>a</code>?</p>
<pre><code>a = '\\xe9\\x82\\xa3'
b = b'\xe9\x82\xa3'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/codecs.html#text-encodings" rel="noreferrer"><code>unicode-escape</code></a> encoding:</p>
<pre><code>&gt;&gt;&gt; a = '\\xe9\\x82\\xa3'
&gt;&gt;&gt; a.encode().decode('unicode-escape').encode('latin1')
b'\xe9\x82\xa3'
</code></pre>
<hr/>
<pre><code>&gt;&gt;&gt; import codecs
&gt;&gt;&gt; codecs.decode(a, 'unicode-escape').encode('latin1')
b'\xe9\x82\xa3'
</code></pre>
</div>
