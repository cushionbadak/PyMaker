<div class="post-text" itemprop="text">
<p>If you need the name as a string, you can use the <code>.winfo_class()</code> method:</p>
<pre><code>for d in dataTypes:
    if d.winfo_class() == 'Entry':
        print("Found Entry!")
    elif d.winfo_class() == 'Checkbutton':
        print("Found Checkbutton!")
</code></pre>
<p>Or, you could access the <code>__name__</code> attribute:</p>
<pre><code>for d in dataTypes:
    if d.__name__ == 'Entry':
        print("Found Entry!")
    elif d.__name__ == 'Checkbutton':
        print("Found Checkbutton!")
</code></pre>
<p>That said, using <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow"><code>isinstance</code></a> is a more common/pythonic approach:</p>
<pre><code>for d in dataTypes:
    if isinstance(d, Entry):
        print("Found Entry!")
    elif isinstance(d, Checkbutton):
        print("Found Checkbutton!")
</code></pre>
<p>Also, your current code is failing because <code>type(d).__name__</code> does not return what you think it does:</p>
<pre><code>&gt;&gt;&gt; from tkinter import Checkbutton
&gt;&gt;&gt; type(Checkbutton).__name__
'type'
&gt;&gt;&gt;
</code></pre>
<p>Notice that it returns the name of the type object returned by <code>type</code>, not the name of <code>Checkbutton</code>.</p>
