<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join</code></a>:</p>
<pre><code>&gt;&gt;&gt; ' '.join(['NEW', 'BRITAIN', 'REGION,', 'PAPUA', 'NEW', 'GUINEA'])
'NEW BRITAIN REGION, PAPUA NEW GUINEA'
</code></pre>
