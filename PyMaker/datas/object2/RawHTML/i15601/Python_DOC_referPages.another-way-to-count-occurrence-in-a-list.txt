<p>I have this list:</p>
<pre><code>['Boston Americans', 'New York Giants', 'Chicago White Sox', 'Chicago Cubs', 'Chicago Cubs', 'Pittsburgh Pirates', 'Philadelphia Athletics', 'Philadelphia Athletics', 'Boston Red Sox', 'Philadelphia Athletics', 'Boston Braves', 'Boston Red Sox', 'Boston Red Sox', 'Chicago White Sox', 'Boston Red Sox', 'Cincinnati Reds', 'Cleveland Indians', 'New York Giants', 'New York Giants', 'New York Yankees', 'Washington Senators', 'Pittsburgh Pirates', 'St. Louis Cardinals', 'New York Yankees', 'New York Yankees', 'Philadelphia Athletics', 'Philadelphia Athletics', 'St. Louis Cardinals', 'New York Yankees']
</code></pre>
<p>Say I want to count how many times <code>"Boston Americans"</code> is in list.</p>
<p>How can I do that without using the <code>.count</code> method <code>list.count("Boston Americans")</code> or any import?</p>
</div>
<div class="post-text" itemprop="text">
<p>Just count :)</p>
<pre><code>count = 0
for item in items:
    if item == 'Boston Americans':
        count += 1
print count
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>One more way to use <code>sum</code>:</p>
<pre><code>sum( x==value for x in mylist )
</code></pre>
<p>Here, we use the fact <code>True</code> and <code>False</code> can be treated as the integers 0 and 1.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the Built-in <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum()</code></a> function :</p>
<pre><code>&gt;&gt;&gt; l=['Boston Americans', 'New York Giants', 'Chicago White Sox', 'Chicago Cubs', 'Chicago Cubs', 'Pittsburgh Pirates', 'Philadelphia Athletics', 'Philadelphia Athletics', 'Boston Red Sox', 'Philadelphia Athletics', 'Boston Braves', 'Boston Red Sox', 'Boston Red Sox', 'Chicago White Sox', 'Boston Red Sox', 'Cincinnati Reds', 'Cleveland Indians', 'New York Giants', 'New York Giants', 'New York Yankees', 'Washington Senators', 'Pittsburgh Pirates', 'St. Louis Cardinals', 'New York Yankees', 'New York Yankees', 'Philadelphia Athletics', 'Philadelphia Athletics', 'St. Louis Cardinals', 'New York Yankees']
&gt;&gt;&gt; sum(1 for i in l if i=="Boston Americans")
1
&gt;&gt;&gt; sum(1 for i in l if i=='Boston Red Sox')
4
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>using <code>filter</code> and <code>lambda</code>:</p>
<pre><code>&gt;&gt;&gt; a = ['Boston Americans', 'New York Giants', 'Chicago White Sox', 'Chicago Cubs', 'Chicago Cubs', 'Pittsburgh Pirates', 'Philadelphia Athletics', 'Philadelphia Athletics', 'Boston Red Sox', 'Philadelphia Athletics', 'Boston Braves', 'Boston Red Sox', 'Boston Red Sox', 'Chicago White Sox', 'Boston Red Sox', 'Cincinnati Reds', 'Cleveland Indians', 'New York Giants', 'New York Giants', 'New York Yankees', 'Washington Senators', 'Pittsburgh Pirates', 'St. Louis Cardinals', 'New York Yankees', 'New York Yankees', 'Philadelphia Athletics', 'Philadelphia Athletics', 'St. Louis Cardinals', 'New York Yankees']
&gt;&gt;&gt; len(filter(lambda x : x=='Boston Americans',a))
1
&gt;&gt;&gt; len(filter(lambda x : x=='Boston Red Sox',a))
4
</code></pre>
<p>Others good Method but you need to import modules</p>
<p>using <code>itertools.groupby</code>:</p>
<pre><code>import itertools
&gt;&gt;&gt; my_count = {x:len(list(y)) for x,y in itertools.groupby(sorted(a))}
&gt;&gt;&gt; my_count['Boston Americans']
1
&gt;&gt;&gt; my_count['Boston Red Sox']
4
</code></pre>
<p>using <code>collection.Counter</code>:</p>
<pre><code>&gt;&gt;&gt; from collections import Counter
&gt;&gt;&gt; my_count = Counter(a)
&gt;&gt;&gt; my_count['Boston Americans']
1
&gt;&gt;&gt; my_count['Boston Red Sox']
4
</code></pre>
