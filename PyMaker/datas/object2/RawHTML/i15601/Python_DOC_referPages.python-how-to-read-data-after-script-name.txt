<div class="post-text" itemprop="text">
<p>You have to use <a href="https://docs.python.org/2/library/sys.html#sys.argv" rel="nofollow"><code>sys.argv</code></a> to retrive parameter after scripts</p>
<p><code>sys.argv[0]</code> will give you first argument.</p>
<p>Please use different argument as <code>python script_name.py arg1 arg2 arg3 arg4</code></p>
<p>If you have to use unix type argument syntext like <code>python script arg1=val1 arg2=val2</code> then user <a href="https://docs.python.org/3/howto/argparse.html" rel="nofollow"><code>argparser</code></a></p>
</div>
