<div class="post-text" itemprop="text">
<p>Simply pass the whole string to a <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow"><code>collections.Counter()</code> object</a> and it'll count each character.</p>
<p>It may be more efficient to do so line by line, so as not to require too much memory:</p>
<pre><code>from collections import Counter

counts = Counter()

with open('inputtextfilename') as infh:
    for line in infh:
        counts.update(line.strip())
</code></pre>
<p>The <code>str.strip()</code> call removes any whitespace (such as the newline character).</p>
<p>A quick demo using your sample input:</p>
<pre><code>&gt;&gt;&gt; from collections import Counter
&gt;&gt;&gt; sample = '''\
... TCCATCTACT
... GGGCCTTCCT
... TCCATCTACC
... '''.splitlines(True)
&gt;&gt;&gt; counts = Counter()
&gt;&gt;&gt; for line in sample:
...     counts.update(line.strip())
... 
&gt;&gt;&gt; for letter, count in counts.most_common():
...     print(letter, count)
... 
C 13
T 10
A 4
G 3
</code></pre>
<p>I used the <a href="https://docs.python.org/3/library/collections.html#collections.Counter.most_common" rel="nofollow"><code>Counter.most_common()</code> method</a> to get a sorted list of letter-count pairs (in order from most to least common).</p>
