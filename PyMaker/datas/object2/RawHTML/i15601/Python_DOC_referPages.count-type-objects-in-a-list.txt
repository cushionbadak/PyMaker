<p>Have you a great method to count number of <code>Sugar</code> in my <code>goblet</code>? </p>
<pre><code>class Sugar:pass
class Milk:pass
class Coffee:pass

goblet = [Sugar(), Sugar(), Milk(), Coffee()]
sugar_dosage = goblet.count(Sugar) #Doesn't work
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <a href="https://docs.python.org/3/reference/expressions.html#grammar-token-generator_expression" rel="nofollow">generator expression</a> with <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum</code></a> and <a href="https://docs.python.org/3/library/functions.html#isinstance" rel="nofollow"><code>isinstance</code></a>:</p>
<pre><code>sum(isinstance(x, Sugar) for x in goblet)
# or
sum(1 for x in goblet if isinstance(x, Sugar))
</code></pre>
<p>Also, <code>list.count</code> did not work because the method was testing how many items in the list <em>equaled</em> <code>Sugar</code>.  In other words, it was basically doing <code>item==Sugar</code> for each item in the list.  This is incorrect because you want to be testing the <em>type</em> of each item, which is what <code>isinstance</code> does.</p>
