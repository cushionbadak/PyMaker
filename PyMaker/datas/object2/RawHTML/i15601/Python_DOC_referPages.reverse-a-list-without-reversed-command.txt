<p>I want to reverse a list on Python without use <code>reversed</code> command. Any ideas?
The query was to find another way to do the reverse the list without use any "standar" Python command like reverse. Thank in advance!</p>
<pre><code>list = []
x = 1

#Import 20 random numbers
for i in range (0,20):
    list.append (x)

print (list)

for i in range (20,0,-1): # &lt;-- I know, this is wrong. I used it as an example
    print (list[i])
</code></pre>
<p><strong>SOLUTION:</strong> </p>
<pre><code>list = []
    x = 1

    #Import 20 random numbers
    for i in range (0,20):
        list.append (x)

    print (list)

for i in range(len(list) - 1, -1, -1):
    print(list[i])
</code></pre>

<div class="post-text" itemprop="text">
<p>You can simply use <a href="https://stackoverflow.com/questions/509211/explain-pythons-slice-notation">Python's slice notation</a> to step through the list backwards:</p>
<pre><code>print(list[::-1])
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; lst = [1, 2, 3]
&gt;&gt;&gt; lst[::-1]
[3, 2, 1]
&gt;&gt;&gt;
</code></pre>
<p>In fact, if you test this with <a href="https://docs.python.org/3/library/timeit.html#timeit.timeit" rel="nofollow noreferrer"><code>timeit.timeit</code></a>, you will see that it is a lot faster than using <code>reversed</code>:</p>
<pre><code>&gt;&gt;&gt; from timeit import timeit
&gt;&gt;&gt; lst = list(range(1000))
&gt;&gt;&gt; timeit('list(reversed(lst))', 'from __main__ import lst')
18.91587534169846
&gt;&gt;&gt; timeit('lst[::-1]', 'from __main__ import lst')
9.677450346342901
&gt;&gt;&gt;
</code></pre>
<p>It is also less typing. :)</p>
