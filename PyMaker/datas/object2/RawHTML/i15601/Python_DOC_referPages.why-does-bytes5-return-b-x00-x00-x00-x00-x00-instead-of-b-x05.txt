<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/functions.html#bytes" rel="nofollow"><code>bytes()</code> function documentation</a> points to the <a href="https://docs.python.org/3/library/functions.html#bytearray" rel="nofollow"><code>bytearray()</code> documentation</a>, which states:</p>
<blockquote>
<p>The optional <em>source</em> parameter can be used to initialize the array in a few different ways:</p>
<ul>
<li><em>[....]</em></li>
<li>If it is an integer, the array will have that size and will be initialized with null bytes.</li>
</ul>
</blockquote>
<p>You asked for a <code>bytes()</code> object of size 5, initialised to null bytes.</p>
<p>You probably want to turn <code>a</code> into a string first:</p>
<pre><code>bytes(str(a), 'utf-8')
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; a = 5
&gt;&gt;&gt; bytes(str(a), 'utf-8')
b'5'
</code></pre>
<p>If you wanted to have the byte value <code>5</code> (so the <a href="http://en.wikipedia.org/wiki/ASCII#ASCII_control_code_chart" rel="nofollow"><code>ENQ</code> ASCII control code</a> or whatever else you might want it to mean) you'll need to put it <code>a</code> in a list:</p>
<pre><code>bytes([a])
</code></pre>
<p>(no need to provide an encoding then):</p>
<pre><code>&gt;&gt;&gt; bytes([a])
b'\x05'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are creating a byte array of length 5.</p>
<p>To get the binary represenation of the number 5 you can use bin()</p>
<pre><code>bin(5)
0b101
</code></pre>
