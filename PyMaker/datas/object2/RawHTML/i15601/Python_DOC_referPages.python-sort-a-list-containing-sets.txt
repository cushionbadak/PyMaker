<p>Any idea how to sort a list that contains elements of type set ?
This is the code I'm using:</p>
<pre><code> sorted_by_second = sorted(result_list, key=lambda set: set[1])

<p>Example of how my result_list looks like:</p>

<p>I would like to sort the list by descending order according to the 2nd element in each tupple of the list elements.
The sorted list result should look like</p>

<div class="post-text" itemprop="text">
<p>I suggest to use <a href="https://docs.python.org/3/library/functions.html#float" rel="nofollow"><code>float</code></a> to map the second element from a string to a number. This will allow <code>sorted</code> to sort by numbers. If you don't use <code>float</code>, <code>sorted</code> will sort the second element as strings.</p>
<pre><code>sorted(result_list, key=lambda x: float(x[1]), reverse=True)
</code></pre>
<p>Result:</p>
<pre><code>[['past due', '32.86691794423967'],
 ['why:customer', '27.65754595407057'],
 ['code', '23.24240338748313']]
</code></pre>
<p>Hint: Don't use <code>set</code> as a name since <a href="https://docs.python.org/3/library/functions.html#func-set" rel="nofollow">it is a built-in function</a>.</p>
</div>
