<div class="post-text" itemprop="text">
<p>Because <code>print</code> returns <code>None</code>?</p>
<p>So, the print is done (<code>4, 5, 6, 7,</code>) and then the list comprehension is returned (<code>[None, None, None, None]</code>)</p>
<p>What you want to do is use join:</p>
<pre><code>&gt;&gt;&gt; ', '.join(str(i) for i in map(inc, counters))
'4, 5, 6, 7'
</code></pre>
<p>or use the <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow"><code>sep</code> argument of print</a> (I don't have python3 right now but something like this:)</p>
<pre><code>print(*map(inc, counters), sep=', ')
</code></pre>
</div>
