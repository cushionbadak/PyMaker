<div class="post-text" itemprop="text">
<p>Using preassigned names for your vectors lead to lots of duplicated logic, that gets more and more complicated if you add new vectors to your data description...</p>
<p>It's much better to use <em>dictionaries</em></p>
<pre><code>data=[['a', 12.3], ['a', 12.4], ['b', 0.4], ['c', 1.2]]

vectors = {}  # an empty dictionary
for key, value in data:
     vectors.setdefault(key,[]).append(value)
</code></pre>
<p>The relevant docs, from <a href="https://docs.python.org/3/library/index.html" rel="nofollow">the python official documentation</a></p>
<blockquote>
<p>setdefault(key[, default])</p>
<pre><code>     If key is in the dictionary, return its value.
     If not, insert key with a value of default and return default.
     default defaults to None.
</code></pre>
<p>append(x)</p>
<pre><code>     appends x to the end of the sequence (same as s[len(s):len(s)] = [x])
</code></pre>
</blockquote>
