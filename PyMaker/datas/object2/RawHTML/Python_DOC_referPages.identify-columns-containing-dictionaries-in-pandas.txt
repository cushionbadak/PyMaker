<div class="post-text" itemprop="text">
<p>You can use a list comprehension to loop through the columns and check whether the first element in each is a dict. Note, this works for the case given and assumes that any column that contains dicts contains <em>only</em> dicts</p>
<pre><code>[i for i in df.columns if isinstance(df[i][0],dict)]

['col3']
</code></pre>
