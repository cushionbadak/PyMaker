<div class="post-text" itemprop="text">
<p>Your method call is wrong. You should call it with object reference. And one more thing you have to define your list as <strong>global</strong> then only you will be able to append next element. Otherwise it will give <strong>NameError: global name 'inventory' is not defined</strong> error. 
Try this out:</p>
<pre><code>class Product(object):

    def __init__(self, ind, name, price, quantity):
        self.ind = ind
        self.name = name
        self.price = price
        self.quantity = quantity        

    global inventory
    inventory = []

    def add(self):
        inventory.append(self.ind)
        inventory.append(self.name)
        inventory.append(self.price)
        inventory.append(self.quantity)
        print('product %s added')% self.name

obj = Product(63456, 'Meow', 60.00, 0)
obj.add()
</code></pre>
<p>Or If you want to have the separate copy of inventory for each object then define inventory as <code>self.inventory = []</code>
Hence your code will look something like:</p>
<pre><code>class Product(object):

    def __init__(self, ind, name, price, quantity):
        self.ind = ind
        self.name = name
        self.price = price
        self.quantity = quantity        
        self.inventory = []


    def add(self):
        self.inventory.append(self.ind)
        self.inventory.append(self.name)
        self.inventory.append(self.price)
        self.inventory.append(self.quantity)
        print('product %s added')% self.name

obj = Product(63456, 'Meow', 60.00, 0)
obj.add()
</code></pre>

<p>You're calling the method as if it were a static method. It is an instance method. You need to create an instance of <code>Product</code>, then call the method on that instance.</p>
<pre><code>my_product = Product(63456, 'Meow', 60.00, 0)
