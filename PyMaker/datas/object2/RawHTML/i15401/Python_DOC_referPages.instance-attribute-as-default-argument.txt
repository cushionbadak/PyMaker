<span class="comment-copy">You are missing the <code>self</code> parameter from both method definitions. And no, you can't use an instance attribute as a default argument; they don't exist when the method is created.</span>
<span class="comment-copy">Your methods miss a <code>self</code> as their first argument where will the reference to the object be stored. See <a href="https://docs.python.org/3/tutorial/classes.html#method-objects" rel="nofollow noreferrer">here</a>.</span>
