<p>I am solving a problem in python.</p>
<p>I have 2 strings with numbers and I need to find out whether a number from string B is located in a string A.</p>
<p>for example:</p>
<pre><code>a = "5, 7" 
b = "6,5"
</code></pre>
<p>A and B may contain any number and quantity.</p>
<p>If the number of string B is in A I want result true or false.</p>
<p>Unfortunately, I do not know how to do this.</p>
</div>
<div class="post-text" itemprop="text">
<p>You have strings, not integers, so you'd have to convert them to integers first:</p>
<pre><code>a_nums = [int(n) for n in a.split(',')]
b_nums = [int(n) for n in b.split(',')]
</code></pre>
<p>This uses a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a> to turn each result of a <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>str.split()</code> method</a> call into integers with the <a href="https://docs.python.org/3/library/functions.html#int" rel="nofollow"><code>int()</code> function</a>.</p>
<p>To test if there are numbers in both sequences, you'd use <a href="https://docs.python.org/3/library/stdtypes.html#set-types-set-frozenset" rel="nofollow"><em>sets</em></a>, then test if there is an intersection:</p>
<pre><code>set(a_nums) &amp; set(b_nums)
</code></pre>
<p>If the result is not empty, there were numbers shared between the sequences. Since non-empty sets <a href="https://docs.python.org/3/library/stdtypes.html#truth-value-testing" rel="nofollow">are considered 'true'</a>, in Python, you'd convert this to a boolean with <a href="https://docs.python.org/3/library/functions.html#bool" rel="nofollow"><code>bool()</code></a>:</p>
<pre><code>bool(set(a_nums) &amp; set(b_nums))
</code></pre>
<p>Sets are by far the most efficient method to test for such intersections.</p>
<p>You can do all this in one line, and a little more efficiently, with generator expressions and the <a href="https://docs.python.org/3/library/stdtypes.html#set.intersection" rel="nofollow"><code>set.intersection()</code> method</a>:</p>
<pre><code>bool(set(int(n) for n in a.split(',')).intersection(int(n) for n in b.split(',')))
</code></pre>
<p>or perhaps a little more compact still with the <code>map()</code> function:</p>
<pre><code>bool(set(map(int, a.split(','))).intersection(map(int, b.split(','))))
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; a = "5, 7" 
&gt;&gt;&gt; b = "6,5"
&gt;&gt;&gt; bool(set(map(int, a.split(','))).intersection(map(int, b.split(','))))
True
</code></pre>
<p>or breaking it down a little:</p>
<pre><code>&gt;&gt;&gt; [int(n) for n in a.split(',')]
[5, 7]
&gt;&gt;&gt; [int(n) for n in b.split(',')]
[6, 5]
&gt;&gt;&gt; set(map(int, a.split(',')))
set([5, 7])
&gt;&gt;&gt; set(map(int, a.split(','))).intersection(map(int, b.split(',')))
set([5])
&gt;&gt;&gt; bool(set(map(int, a.split(','))).intersection(map(int, b.split(','))))
True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If the numbers are integers, to obtain a list of <code>True</code> / <code>False</code> entries corresponding to whether each number in <code>b</code> is in <code>a</code>, try:</p>
<pre><code>a = "5, 7" 
b = "6,5"
A = [int(x) for x in a.split(',')]
B = [int(x) for x in b.split(',')]
c = [x in A for x in B]
print(c)
</code></pre>
<p>Output: </p>
<pre><code>[False, True]
</code></pre>
<p>To find out if <em>any</em> of the numbers in <code>b</code> are in <code>a</code>, then:</p>
<pre><code>any(c)
</code></pre>
<p>Output:</p>
<pre><code>True
</code></pre>
</div>
