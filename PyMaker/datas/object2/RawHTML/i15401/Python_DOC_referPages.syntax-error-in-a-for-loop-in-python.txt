<div class="post-text" itemprop="text">
<p>If you want to use a range, then you need to <strong>call</strong> the  function:</p>
<pre><code>for j in range(row):
</code></pre>
<p><code>range</code> is not part of the Python syntax, it is just another <a href="https://docs.python.org/3/library/stdtypes.html#ranges" rel="nofollow">built-in object type</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>The syntax is:</p>
<pre><code>for j in range(row):
</code></pre>
<p><code>range</code> is a function taking an argument. And you have to specify which variable you will use to iterate (<code>j</code>)</p>
</div>
<div class="post-text" itemprop="text">
<p>The range function have following syntax: range(limit 1, limit 2), you pass like argument a size of array you want.</p>
</div>
<span class="comment-copy">Look up what <code>range</code> does, and how you use it.</span>
<span class="comment-copy">Look up what <code>for</code> does, and how you use it.</span>
<span class="comment-copy">Do you expect this 2D list to get filled with random numbers? It won't. But you need to read the docs on the <code>random</code> module to understand why.</span>
