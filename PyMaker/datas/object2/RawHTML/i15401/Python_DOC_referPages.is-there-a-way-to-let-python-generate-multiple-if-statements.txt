<div class="post-text" itemprop="text">
<p>Use a for loop.</p>
<pre><code>for n in range(0, 36 * 30, 30):
    if n &lt;= x &lt;= n + 30 and 0 &lt;= y &lt;= n + 30:
        pass #do something
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>for n in range(0, 36 * 30, 30):
    if n &lt;= x &lt;= (n+30) and n &lt;= y &lt;= (n+30):
        pass  # (do stuff)
</code></pre>
<p><code>range</code> can take an optional third argument for the "step" value. For reference, see <a href="https://docs.python.org/3/library/stdtypes.html#typesseq-range" rel="nofollow">Python's documentation on <code>range</code></a>.</p>
