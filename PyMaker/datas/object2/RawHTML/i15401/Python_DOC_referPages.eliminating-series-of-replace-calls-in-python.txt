<div class="post-text" itemprop="text">
<p>I'm working on a project that involves parsing pages of text. I've written the following function to remove certain punctuation from a word and convert it to lowercase:</p>
<pre><code>def format_word(word):
    return word.replace('.', '').replace(',', '').replace('\"', '').lower()
</code></pre>
<p>Is there any way to combine all of the calls to .replace() into one? This looks rather ugly the way it is! The only way I can think of doing it is as follows:</p>
<pre><code>def format_word(word):
    for punct in '.,\"':
        word.replace(punct, '')
    return word.lower()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.translate" rel="nofollow"><code>str.translate</code></a> if you want to remove characters:</p>
<p>In python 2.x:</p>
<pre><code>&gt;&gt;&gt; 'Hello, "world".'.translate(None, ',."')
'Hello world'
</code></pre>
<p>In python 3.x:</p>
<pre><code>&gt;&gt;&gt; 'Hello, "world".'.translate(dict.fromkeys(map(ord, ',."')))
'Hello world'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/2/library/re.html" rel="nofollow"><strong><code>re</code> module</strong></a> for that</p>
<pre><code>import re
&gt;&gt;&gt; def format_word(word):
...     return re.sub(r'[,."]', "", word)
...
&gt;&gt;&gt; print format_word('asdf.,"asdf')
asdfsdf
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are quite close. If you don't only call <code>.replace()</code>, but as well use its result, you are done:</p>
<pre><code>def format_word(word):
    for punct in '.,\"':
        word = word.replace(punct, '')
    return word.lower()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do this using regular expressions:</p>
<pre><code>re.sub("[.,\"]", "", "\"wo,rd.")
</code></pre>
</div>
