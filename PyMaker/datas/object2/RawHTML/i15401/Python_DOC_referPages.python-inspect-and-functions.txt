<p>I am trying to go through the json module and return just the functions. and then I would like to use inspect module to return inspect.formatargspec(*inspect.getfullargspec(func)) on each function from json.</p>
<p>This is what I was thinking, this obviously fails because <code>func</code> is a string.</p>
<pre><code>import inspect
import json as m 

for func in dir(m):

    if inspect.isfunction(func):
        print(func)
</code></pre>

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/functions.html#dir" rel="nofollow"><code>dir</code></a> returns a list of attribute names of the object, not the attributes. You need to use <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow"><code>getattr</code></a> to get the attribute.</p>
<pre><code>import inspect

for func in dir(m):  # `func`: str
    if inspect.isfunction(getattr(m, func)):  # &lt;----
        print(func)
</code></pre>
