<div class="post-text" itemprop="text">
<p>Create an empty list and then append the results of for loop to that list or you could use <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>.</p>
<pre><code>l = []
for agern in anime.info['Genres']:
    l.append(agern['name'])
print l
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>you could also store that into a generator function, something like this:</p>
<pre><code>for agern in anime.info['Genres']:
    yield agern['name']
</code></pre>
<p>so that you will actually compute this thing only on request, i.e. during a for cycle, so you will not store in memory each agern['name'] two times.</p>
