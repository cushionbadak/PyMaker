<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/2/library/functools.html#functools.partial"><code>functools.partial</code></a>:</p>
<pre><code>&gt;&gt;&gt; import functools
&gt;&gt;&gt;
&gt;&gt;&gt; def a(x, y):
...     return x + y
...
&gt;&gt;&gt; b = functools.partial(a, x=1, y=2)
&gt;&gt;&gt; b()
3
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functools.html#functools.partial" rel="nofollow"><code>functools.partial</code></a> to return a new <code>partial</code> object. <em>Effectively</em> it provides you with the same function but one or more of the arguments have been filled with set values.</p>
<pre><code>from functools import partial

def a(x, y):
    return x + y

b = partial(a, x=1, y=2)

print(b())
# 3
</code></pre>
