<div class="post-text" itemprop="text">
<p>Use <code>zip()</code> to pair up the lists:</p>
<pre><code>{'video': [{'title': title, 'id': id} for title, id in zip(titles, ids)]}
</code></pre>
<p>The <code>video</code> value is formed by a list comprehension; for every <code>title, id</code> pair formed by <code>zip()</code> a dictionary is created:</p>
<pre><code>&gt;&gt;&gt; titles = ['New', 'New']
&gt;&gt;&gt; ids = ['123', '123']
&gt;&gt;&gt; {'video': [{'title': title, 'id': id} for title, id in zip(titles, ids)]}
{'video': [{'title': 'New', 'id': '123'}, {'title': 'New', 'id': '123'}]}
</code></pre>
<p>or with a little more interesting content:</p>
<pre><code>&gt;&gt;&gt; from pprint import pprint
&gt;&gt;&gt; titles = ['Foo de Bar', 'Bring us a Shrubbery!', 'The airspeed of a laden swallow']
&gt;&gt;&gt; ids = ['42', '81', '3.14']
&gt;&gt;&gt; pprint({'video': [{'title': title, 'id': id} for title, id in zip(titles, ids)]})
{'video': [{'id': '42', 'title': 'Foo de Bar'},
           {'id': '81', 'title': 'Bring us a Shrubbery!'},
           {'id': '3.14', 'title': 'The airspeed of a laden swallow'}]}
</code></pre>
<p>In case you also don't know how to then encode the result to JSON with the <a href="https://docs.python.org/3/library/json.html" rel="nofollow"><code>json</code> library</a>, to write to a file use:</p>
<pre><code>import json

with open('output_filename.json', 'w', encoding='utf8') as output:
    json.dump(python_object, output)
</code></pre>
