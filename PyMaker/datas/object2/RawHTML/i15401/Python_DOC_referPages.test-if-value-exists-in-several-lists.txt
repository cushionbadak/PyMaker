<div class="post-text" itemprop="text">
<p>I would like to check if a value exists in every list. </p>
<p>The following returns <code>True</code> as expected, but seems un-pythonic. </p>
<p>What is the correct/more elegant way to do this?</p>
<pre><code>a = [1 ,2]
b = [1, 3]
c = [1, 4]
d = [2, 5]

False in [True if 1 in l else False for l in [a, b, c, d]  ]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#all"><code>all</code></a> and a <a href="https://docs.python.org/3/reference/expressions.html#grammar-token-generator_expression">generator expression</a>:</p>
<pre><code>all(1 in x for x in (a, b, c, d))
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; a = [1 ,2]
&gt;&gt;&gt; b = [1, 3]
&gt;&gt;&gt; c = [1, 4]
&gt;&gt;&gt; d = [2, 5]
&gt;&gt;&gt; all(1 in x for x in (a, b, c, d))
False
&gt;&gt;&gt; all(1 in x for x in (a, b, c))
True
&gt;&gt;&gt;
</code></pre>
<p>In addition to being more readable, this solution is more efficient since it uses lazy-evaluation.  It will only check as many items as is necessary to determine the result.</p>
<p>Also, there is never a good reason to do:</p>
<pre><code>True if 1 in l else False
</code></pre>
<p>or something similar since <code>in</code> already returns a boolean result.  All you need is:</p>
<pre><code>1 in l
</code></pre>
<p>For the negated version, use:</p>
<pre><code>1 not in l
</code></pre>
</div>
