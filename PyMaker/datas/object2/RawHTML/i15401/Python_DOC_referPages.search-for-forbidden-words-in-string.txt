<p>You actually do not need Regex for this.  You can use <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow"><code>all</code></a> and a <a href="https://docs.python.org/3/reference/expressions.html#grammar-token-generator_expression" rel="nofollow">generator expression</a>:</p>
<pre><code>def check(input):
    return all(x not in input for x in bad_words)
</code></pre>
<p>To make your string of words into a list, use <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>str.split</code></a>:</p>
<pre><code>&gt;&gt;&gt; words = "word, something, dog, cat"
&gt;&gt;&gt; words.split(', ')
['word', 'something', 'dog', 'cat']
&gt;&gt;&gt;
</code></pre>
<p>You might also want to call <code>set()</code> on the list to improve lookup times.  Sets have <code>O(1)</code> (constant) complexity with the <code>in</code> and <code>not in</code> operators while lists have <code>O(n)</code> (linear).  So, it is faster to find things in a set than it is with a list.</p>
