<p>Using <a href="https://docs.python.org/3/library/stdtypes.html#str.format"><code>str.format</code></a>:</p>
<pre><code>nameList = ["grade 1", "grade 2222222222", "grade 3"]
gradeList = ["1/1", "2/2", "100000/100000"]

fmt = '{:&lt;8}{:&lt;20}{}'

print(fmt.format('', 'Assignment', 'Grade'))
for i, (name, grade) in enumerate(zip(nameList, gradeList)):
    print(fmt.format(i, name, grade))
</code></pre>
<p>output:</p>
<pre><code>        Assignment          Grade
0       grade 1             1/1
1       grade 2222222222    2/2
2       grade 3             100000/100000
</code></pre>
<p>Alternatively, you can also use <a href="https://docs.python.org/3/library/stdtypes.html#printf-style-string-formatting"><code>printf</code> style formatting using % operator</a>:</p>
<pre><code>fmt = '%-8s%-20s%s'

print(fmt % ('', 'Assignment', 'Grade'))
for i, (name, grade) in enumerate(zip(nameList, gradeList)):
    print(fmt % (i, name, grade))
</code></pre>
