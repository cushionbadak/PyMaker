<div class="post-text" itemprop="text">
<p>You can first define a <code>DataFrame</code> with the column names, and after that change the types with <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.astype.html" rel="nofollow noreferrer"><code>.astype</code></a> like following:</p>
<pre><code>bins = pd.DataFrame(columns=["chrom", "start", "end"])

bins = bins.astype({'chrom':'object',
                    'start':'int64',
                    'end':'int64'})

print(bins.dtypes)
chrom    object
start     int64
end       int64
dtype: object
</code></pre>
<p><strong>note</strong>: I used <code>object</code> as type to define a <em>string</em> column, which is how <code>text</code> column is defined in <code>pandas</code> </p>
</div>
