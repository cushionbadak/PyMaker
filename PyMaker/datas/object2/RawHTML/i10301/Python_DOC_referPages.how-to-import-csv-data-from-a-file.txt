<p>Use <a href="https://docs.python.org/3/library/glob.html" rel="nofollow"><code>glob</code></a>:</p>
<pre><code>import glob
import csv
for f_name in glob.glob("*.csv"):
    with open(f_name) as f:
        reader = csv.reader(f)
        # do stuff here
</code></pre>

<p>That's a basis, what you're going to want to do is put that into a for loop going through all the filenames in the directory, like so: </p>
<pre><code>import os
import csv
dirlist = os.scandir(DIRNAME)
for x in dirlist:
    f = open(str(x)+".csv")     
    r = csv.reader(f)
    f.close()
</code></pre>
