</code></pre>

</code></pre>

<pre><code>S = 'spam'
S.replace('pa', 'xx')
print S
</code></pre>
<p>You will get the same string <code>'spam'</code></p>

<span class="comment-copy">Who says the string object <code>S</code> references itself is altered? You got a <i>new string</i> back, which was echoed.</span>
<span class="comment-copy">Try <code>T = S.replace('pa', 'xx')</code> and you'll see that <code>S</code> is unaltered afterwards. <a href="https://docs.python.org/3/library/stdtypes.html#str.replace" rel="nofollow noreferrer"><i>"Return a copy of the string with all occurrences of substring old replaced by new."</i></a></span>
