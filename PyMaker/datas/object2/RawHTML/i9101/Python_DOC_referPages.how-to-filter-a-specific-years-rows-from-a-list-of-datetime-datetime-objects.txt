<div class="post-text" itemprop="text">
<p>If you have a very long list of dates, skipping the ones undesired could be done iteratively using the built-in <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter()</code></a> function in Python 3, or the <a href="https://docs.python.org/2/library/itertools.html#itertools.ifilter" rel="nofollow noreferrer"><code>itertools.ifilter()</code></a> in Python 2. </p>
<p>In either case, creating a second—potentially also very long—list is avoided.</p>
<pre><code>import datetime
try:
    from itertools import ifilter
except ImportError:  # Python 3
    ifilter = filter

dates = [datetime.datetime(2016,5,1), datetime.datetime(2015,2,5)]

for date in ifilter(lambda d: d.year != 2015, dates):
    # do something with non-2015 date...
    print(date)  # -&gt; 2016-05-01 00:00:00
</code></pre>
