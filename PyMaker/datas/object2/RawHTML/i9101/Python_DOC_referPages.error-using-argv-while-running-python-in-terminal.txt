<div class="post-text" itemprop="text">
<p>You pass arguments to functions in Python by placing them between the parentheses (these are called "parameters," <a href="https://docs.python.org/3.6/tutorial/controlflow.html#defining-functions" rel="nofollow noreferrer">see documentation</a>). So you'll need to modify your functions so that they can take arguments (rather than read from <code>sys.argv</code>), and then run the function like:</p>
<p><strong>my_library.py</strong></p>
<pre><code>def function1(filename):
    print filename
</code></pre>
<p><strong>Interpreter</strong></p>
<pre><code>&gt;&gt;&gt; import my_library
&gt;&gt;&gt; my_library.function1("file1.xml")
&gt;&gt;&gt; file1.xml
</code></pre>
<p>If you want your function be able to process an indefinite number of arguments (as you can with <code>sys.argv</code>), you can use the <code>*</code> syntax at the end of your arguments list to catch the remaining parameters as a list (<a href="https://docs.python.org/3/tutorial/controlflow.html#arbitrary-argument-lists" rel="nofollow noreferrer">see documentation</a>). For example:</p>
<p><strong>my_library.py</strong></p>
<pre><code>def function1(*filenames):
    for filename in filenames:
        print filename
</code></pre>
<p><strong>Interpreter</strong></p>
<pre><code>&gt;&gt;&gt; import my_library
&gt;&gt;&gt; my_library.function1("file1.xml", "file2.csv", "file3.xml")
file1.xml
file2.csv
file3.xml
</code></pre>
</div>
