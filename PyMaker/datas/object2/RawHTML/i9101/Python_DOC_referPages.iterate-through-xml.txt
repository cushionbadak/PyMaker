<div class="post-text" itemprop="text">
<p>As @deceze already pointed out, you can use <a href="https://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow noreferrer"><code>xml.etree.ElementTree</code></a> here.</p>
<pre><code>import xml.etree.ElementTree as ET
tree = ET.parse("path_to_xml_file")
root = tree.getroot()
</code></pre>
<p>You can iterate over all children nodes of root:</p>
<pre><code>for child in root.iter():
    if child.tag == 'clientid':
        print child.tag, child.text.strip()
</code></pre>
<p>Children are nested, and we can access specific child nodes by index, so <code>root[0][1]</code> should work (as long as the indices are correct).</p>
