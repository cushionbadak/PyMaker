<div class="post-text" itemprop="text">
<p>This is the Python code to encrypt the policy document to base64 encryption:</p>
<pre><code>import base64

input = open("policy.txt", "rb")

policy = input.read()

policy_encoded = base64.b64encode(policy)

print("%s" % (policy_encoded))
</code></pre>
<p>create your own policy document as policy.txt</p>
<p>For more information regarding this, please visit: </p>
<p><a href="http://s3upload.blogspot.in/" rel="nofollow noreferrer">http://s3upload.blogspot.in/</a></p>
