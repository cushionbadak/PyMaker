<p>I have a question.
I am trying to grab a specific integer from a text file.</p>
<p>The text file is larger, but part of it contains this:</p>
<pre class="lang-none prettyprint-override"><code>class="checkbox" name="reports[]" id="reports_43440"
</code></pre>
<p>Now, I am interested in the integer next to "reports_", which is different every time.</p>
<p>How do I get that specific integer from the text file?</p>
<p>I have tried this, but I couldn't get it work.</p>
<pre><code>filereader = myFile.read()
reportid = re.findall(r'"([^"]*)"', filereader)
</code></pre>
<p>The problem is, that it returns everything within brackets from the text file. I would like to get only that specific integer though.</p>

<div class="post-text" itemprop="text">
<p>Let you have a text file called <code>test.txt</code> with following contents:</p>
<pre><code>class="checkbox" name="reports[]" id="reports_12343440"
class="checkbox" name="reports[]" id="reports_4342340"
class="checkbox" name="reports[]" id="reports_4344430"
class="checkbox" name="reports[]" id="reports_4344130"
class="checkbox" name="reports[]" id="reports_434410"
class="checkbox" name="reports[]" id="reports_434403"
</code></pre>
<p>Now you can get the id value of reports_ using the regular expression like this:</p>
<pre><code>import re
myFile = open("test.txt","r")
filereader = myFile.read()
regex = r"reports_([\d]+)"
reportid = re.findall(regex, filereader)
for id in reportid:
    print(id)
</code></pre>
<p>The output of the snippet is:</p>
<pre><code>12343440
4342340
4344430
4344130
434410
434403
</code></pre>
<p>Here I have searched for matches with <code>reports_</code>. You can customize as much as you need.</p>

<p>This code should fix your problem:</p>
<pre><code>input = 'class="checkbox" name="reports[]" id="reports_43440"'

results = re.findall(r'"reports\_([^"]+?)"', input)
print results
</code></pre>

<div class="post-text" itemprop="text">
<p>Simple solution using <a href="https://docs.python.org/3/library/re.html#re.search" rel="nofollow noreferrer"><strong>re.search</strong></a> function(if it should get a single integer value):</p>
<pre><code>str = 'class="checkbox" name="reports[]" id="reports_43440"'
report_id = re.search(r'id="reports_(\d+)"', str).group(1)

print(report_id)
</code></pre>
<p>The output:</p>
<pre><code>43440
</code></pre>
