<p>I have a list of strings and I want to sort by the second element of each string.</p>
<p>Here's my code:</p>
<pre><code>import sys

line = ""
for n in [line for line in sys.stdin.readlines()]:
    list = []
    list = n
    print list.split()
</code></pre>
<p>output:</p>
<pre><code>['qlwekjrwer', '0']
['sdlkfsdf', '6245']
['lkdjfs', '3985']
['sdfsdfs', '4085']
</code></pre>
<p>What I want to do is sort by the second element of each string in the list in descending order. I know you would have to use the 'list.split()[1]' somehow, but I don't know where to begin.</p>
<p>what I want:</p>
<pre><code>['sdlkfsdf', '6245']
['sdfsdfs', '4085']
['lkdjfs', '3985']
['qlwekjrwer', '0']
</code></pre>
<p>would i have to implement some bubble sort?</p>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer">sorted()</a> function:</p>
<pre><code>result = sorted([l.split() for l in sys.stdin.readlines()], key=lambda l: int(l[1]), reverse=True)
print(result)
</code></pre>
<p>The output:</p>
<pre><code>[['sdlkfsdf', '6245'], ['sdfsdfs', '4085'], ['lkdjfs', '3985'], ['qlwekjrwer', '0']]
</code></pre>
