<p>I am trying to recreate some of my ruby code in python(can't wait till lambda supports ruby!!), and I'm running in to an issue on returning the indicies of filter results on an array(I think I am saying that correctly).  Here is some example code:</p>
<pre><code>array = [200,100,150,500,25,650,175]
res = filter(lambda x: x &lt; 200, array)
print res
[100, 150, 25, 175]
</code></pre>
<p>But what I really need it to return the index of each element that matches the filter and get:</p>
<pre><code>[1,2,4,6]
</code></pre>
<p>Is there an easy way to do this in python? I was going to try numpy, but easyinstall went nuts when trying to install it.</p>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a> which yields tuples containing indices with original values:</p>
<pre><code>&gt;&gt;&gt; enumerate(array)
&lt;enumerate object at 0x7f4ad46d0190&gt;
&gt;&gt;&gt; list(enumerate(array))
[(0, 200), (1, 100), (2, 150), (3, 500), (4, 25), (5, 650), (6, 175)]
</code></pre>
<hr/>
<p>combining with <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; array = [200,100,150,500,25,650,175]
&gt;&gt;&gt; [i for i, x in enumerate(array) if x &lt; 200]
[1, 2, 4, 6]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Yes</p>
<pre><code>[i for i,x in enumerate(array) if x &lt; 200]
</code></pre>
