<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/reference/expressions.html#calls" rel="nofollow noreferrer">class definition statement</a> follows the known rules from function calls on how the arguments are passed: </p>
<pre><code>classdef    ::=  [decorators] "class" classname [inheritance] ":" suite
inheritance ::=  "(" [argument_list] ")"
classname   ::=  identifier
</code></pre>
<p>where <code>argument_list</code> is defined in <a href="https://docs.python.org/3/reference/expressions.html#grammar-token-argument_list" rel="nofollow noreferrer">the section on Calls</a>.</p>
<p>You'll need to provide any keyword arguments <em>after</em> the positional ones.</p>
<p>In this case, the <code>metaclass</code> kwarg should be provided after the base class arg:</p>
<pre><code>class UserBase(Base, metaclass=ABCMeta)
</code></pre>
