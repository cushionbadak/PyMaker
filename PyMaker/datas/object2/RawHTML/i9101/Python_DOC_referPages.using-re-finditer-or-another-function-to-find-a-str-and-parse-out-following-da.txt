<div class="post-text" itemprop="text">
<p>In order to get the <code>5</code> characters after a pattern, you can use <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><code>re.findall</code></a> with a group:</p>
<pre><code>&gt;&gt;&gt; s = '123VID foo 456VID bar '
&gt;&gt;&gt; re.findall('VID(.{5})', s)
[' foo ', ' bar ']
</code></pre>
<p>From the <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer">docs</a>:</p>
<blockquote>
<p>If one or more groups are present in the pattern, return a list of groups; this will be a list of tuples if the pattern has more than one group. </p>
</blockquote>
<p>If you want to stick with <code>finditer</code>, the syntax will be slightly more complex because you have to extract the group manually:</p>
<pre><code>&gt;&gt;&gt; [m.group(1) for m in re.finditer('VID(.{5})', s)]
[' foo ', ' bar ']
</code></pre>
