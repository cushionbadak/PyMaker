<div class="post-text" itemprop="text">
<p>list.append() requires a string - you're giving an integer.
Change</p>
<pre><code>fileNames.append(os.path.basename(int))
</code></pre>
<p>to</p>
<pre><code>fileNames.append(os.path.basename(str))
</code></pre>
<p>The error was thrown within the Python code (<code>os</code> module, <a href="https://docs.python.org/3/library/os.path.html#os.path.basename" rel="nofollow noreferrer">see the docs over here</a> (<a href="https://docs.python.org/2/library/os.path.html#os.path.basename" rel="nofollow noreferrer">Python 2</a>)) when it tried to get the <code>len(int)</code>, which is impossible (since <code>int</code> doesn't have the method <code>__len__</code>)</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>for i in range(len(array)):
    fileNames.append(os.path.basename(i))
</code></pre>
<p>In your above code, <code>i</code> is an integer and <a href="https://docs.python.org/2/library/os.path.html#os.path.basename" rel="nofollow noreferrer">os.path.basename(path)</a> expects a pathname like string. </p>
<p>When iterating over a list, you can direcly iterate through items. </p>
<pre><code>for file in array:
    fileNames.append(os.path.basename(file))
</code></pre>
<p>or if you want to use indexes, you can do </p>
<pre><code>for i in range(len(array)):
    fileNames.append(os.path.basename(array[i]))
                                      ^^^^^ notice accessing an item using index
</code></pre>
</div>
