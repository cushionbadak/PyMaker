<p>I want to take a list like this</p>
<pre><code>mylist = [['_','X','_'],['X','_','_'],['X','_','_']]
</code></pre>
<p>and display it like this:</p>
<pre><code>_ X _
X _ _
X _ _
</code></pre>
<p>would I use 2 nested loops and build a string?</p>
<pre><code>mystring=''
for line in mylist:
   for char in line:
       mystring += char + ' '
   mystring += '\n'
screen.addstr(mystring,0,0)
</code></pre>

<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>str.join</code></a>:</p>
<pre><code>mylist = [['_','X','_'],['X','_','_'],['X','_','_']]

mystring = '\n'.join(' '.join(sublist) for sublist in mylist)

screen.addstr(mystring,0,0)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do this using <a href="https://www.google.co.in/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=2&amp;cad=rja&amp;uact=8&amp;sqi=2&amp;ved=0ahUKEwiU6pCkzYPSAhUEp48KHcIDBnkQFggeMAE&amp;url=http%3A%2F%2Fstackoverflow.com%2Fquestions%2F1876191%2Fexplain-python-join&amp;usg=AFQjCNEINa8hMth6ucTQUDQbiUCqGxo2Mw&amp;sig2=ZcjVlP2W4XU3zO5lg6AEZg&amp;bvm=bv.146496531,d.c2I" rel="nofollow noreferrer">str.join</a> :</p>
<pre><code>for i in mylist :
    print ''.join(i)
</code></pre>
<p>If you want spaces between the characters, replace <code>''</code> by a whitespace <code>' '</code>.</p>
