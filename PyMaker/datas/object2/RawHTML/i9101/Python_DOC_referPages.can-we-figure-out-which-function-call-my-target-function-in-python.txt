<div class="post-text" itemprop="text">
<p>I was wondering if maybe we could figure out which functions call another specific function?</p>
<p>For example,</p>
<pre><code>def f1(): print 1 

def f2(): f1()
</code></pre>
<p>When we execute this script</p>
<pre><code>&gt;&gt;&gt; f2()
1
</code></pre>
<p>We should know <code>f2</code> call my target <code>f1</code>. Is that possible?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/inspect.html#inspect.getframeinfo" rel="nofollow noreferrer">inspect.getframeinfo</a> and other related functions in inspect can help:  </p>
<pre><code>&gt;&gt;&gt; import inspect
&gt;&gt;&gt; def f1(): f2()
... 
&gt;&gt;&gt; def f2():
...   curframe = inspect.currentframe()
...   calframe = inspect.getouterframes(curframe, 2)
...   print 'caller name:', calframe[1][3]
... 
&gt;&gt;&gt; f1()
caller name: f1
&gt;&gt;&gt; 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>What I think you want to do is to do a stack trace. This is possible by calling </p>
<pre><code>traceback.print_exc()
</code></pre>
<p>Where you of course need to import traceback. </p>
</div>
<div class="post-text" itemprop="text">
<p>You can use module <code>traceback</code></p>
<pre><code>def f():
    pass
    import traceback
    traceback.print_stack()
    print "Still working just fine"
    pass

def caller():
    f()

caller()
</code></pre>
<p>yields </p>
<pre><code>  File "traceback.py", line 12, in &lt;module&gt;
    caller()
