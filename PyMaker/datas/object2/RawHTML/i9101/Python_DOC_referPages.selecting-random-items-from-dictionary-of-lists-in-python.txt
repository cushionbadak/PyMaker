<p>You could use nested list comprehension:</p>
<pre><code>&gt;&gt;&gt; import pprint
&gt;&gt;&gt; import random
&gt;&gt;&gt; num = [["ENST", "10", "5"], ["ENGT", "8", "2"], ["ENHT", "5", "1"]]
&gt;&gt;&gt; a = {"ENST" : ['acac', 'tgtgtgt', 'hahah'], "ENHT": ['aaaa', 'tttt', 'gig', 
... 'cccc'], 'ENGT' : ['ddd', 'eeee', 'ffff', 'hhhh', 'pppp']}
&gt;&gt;&gt; res = [[[k, random.choice(a[k])] for _ in range(int(count))] for k, count, _ in num]
&gt;&gt;&gt; pprint.pprint(res)
[[['ENST', 'acac'],
  ['ENST', 'tgtgtgt'],
  ['ENST', 'hahah'],

<p>In above <code>for k, count, _ in num</code> will iterate over elements in <code>num</code> and unpack them to respective variables. Note that we only need the name and count so the throwaway variable is named as <code>_</code>:</p>
<pre><code>&gt;&gt;&gt; [(k, count) for k, count, _ in num]
[('ENST', '10'), ('ENGT', '8'), ('ENHT', '5')]
</code></pre>
<p>For every element in <code>num</code> a list is constructed by selecting <code>count</code> elements from respective list in <code>a</code> with <a href="https://docs.python.org/3/library/random.html#random.choice" rel="nofollow noreferrer"><code>random.choice</code></a>:</p>
<pre><code>&gt;&gt;&gt; k = 'ENST'
&gt;&gt;&gt; count = '10'
&gt;&gt;&gt; [[k, random.choice(a[k])] for _ in range(int(count))]
[['ENST', 'tgtgtgt'], ['ENST', 'hahah'], ['ENST', 'acac'], ['ENST', 'hahah'], ['ENST', 'tgtgtgt'], ['ENST', 'hahah'], ['ENST', 'hahah'], ['ENST', 'hahah'], ['ENST', 'acac'], ['ENST', 'acac']]
</code></pre>

<div class="post-text" itemprop="text">
<p>The problem is in the line where you make the random choice. You are using <code>random.choice</code> incorrectly. Also, we want to make sure you are iterating over line[1] correctly. The lines:</p>
<pre><code>for i in int(line[1] -1):
    final_List.append([k, random.choice(0, len(v))])
</code></pre>
<p>should be</p>
<pre><code>for i in range(int(line[1])):
    final_List.append([k, random.choice(v)])
</code></pre>
</div>
