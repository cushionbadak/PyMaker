<p>I have this link from youtube:</p>
<p>www.youtube.com/watch?v=wMXHHuEGuO8</p>

<p>how can i do that in Django?</p>

<div class="post-text" itemprop="text">
<p>This answer is specific to Python 3, and doesn't use any Django-specific calls. You can use the <a href="https://docs.python.org/3/library/urllib.parse.html#module-urllib.parse" rel="nofollow noreferrer">urllib.parse</a> module to parse URLs into their parts:</p>
<pre><code>import urllib.parse

url = 'www.youtube.com/watch?v=wMXHHuEGuO8'

query_string =  urllib.parse.urlparse(url).query
print(query_string)
# v=wMXHHuEGuO8

query_dict = urllib.parse.parse_qs(query_string)
print(query_dict)
# {'v': ['wMXHHuEGuO8']}

print(query_dict['v'])
# ['wMXHHuEGuO8']

print(query_dict['v'][0])
# wMXHHuEGuO8
</code></pre>
</div>

<p>If it always is the same format like youtube.com/watch?v= you can split it like this:</p>
<pre><code>your_url = www.youtube.com/watch?v=wMXHHuEGuO8
splited = your_url.split('watch?v=') 
# you will have a list of two elements, first one is
# www.youtube.com/ and the other one is wMXHHuEGuO8
</code></pre>
