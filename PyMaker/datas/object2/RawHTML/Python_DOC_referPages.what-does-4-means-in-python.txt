<pre><code>genre_word = [(genre, word) for genre in ['news', 'romance']
                            for word in brown.words(categories=genre)]

genre_word[:4]
genre_word[-4:]

<div class="post-text" itemprop="text">
<p>For any iterable in python <code>[-4:]</code> denotes the indexing of last four items of that iterable. For example :</p>
<pre><code>list1 = [1,2,3,4,5,6]
list1[-4:]
</code></pre>
<blockquote>
<p>[3, 4, 5, 6]</p>
</blockquote>
<pre><code>str1 = 'Hello world'
str1[-4:]
</code></pre>
<blockquote>
<p>'orld'</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>It gets the last four elements of the sequence:</p>
<pre><code>&gt;&gt;&gt; l = [1,2,3,4,5,6,7,8,9]
&gt;&gt;&gt; l[-4:]
[6, 7, 8, 9]
&gt;&gt;&gt; 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As you said in python the indexing of an array starts with 0 </p>
<p>let us say </p>
<pre><code>my_word = 'hello world'
print(my_word[0])  #prints 'h'
</code></pre>
<p>in python we also have a feature to index from the last </p>
<p>let us say </p>
<pre><code>my_word = 'hello world'
print(my_word[-1]) # prints 'd'
# - stands from last starting with index 1
print(my_word[-4:]) # prints 'orld'
# if you know the length you can directly use
print(my_word[7:])  # prints 'orld'
</code></pre>
<p>This feature helps you to print the values indexing from last if you don't know the length</p>
</div>
<span class="comment-copy">The some_list[-n] syntax gets the nth-to-last element. So some_list[-4] gets the last four elements.</span>
