<p>I have an array </p>
<pre><code>a=[0, 10, 20, 30, 40, 50, 60] 
</code></pre>
<p>I selected the 2nd and 3rd element with </p>
<pre><code>a[[1,2]] 
</code></pre>
<p>obtaining </p>
<pre><code>array([10, 20])
</code></pre>
<p>How do I select the other elements of <code>a</code> except the elements I've already selected? </p>
<p>That is, I want to obtain: </p>
<pre><code>array([0, 30, 40, 50, 60])
</code></pre>
<p>Logically, should be something like </p>
<pre><code>a[![1,2]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Like this:</p>
<pre><code>a=[0, 10, 20, 30, 40, 50, 60]

b = a[1:3]
c =[x for x in a if x not in b]

print(a)
print(b)
print(c)
</code></pre>
<p>Output:</p>
<pre><code>[0, 10, 20, 30, 40, 50, 60]
[10, 20]
[0, 30, 40, 50, 60]
</code></pre>
<p>If order does not matter, you can stuff the list in a set and use <a href="https://docs.python.org/2/library/sets.html#set-objects" rel="nofollow noreferrer">these set operations</a>:
<code>yourSet.union(otherSet)</code> , <code>yourSet.intersect(otherSet)</code> , `yourSet.difference(otherSet) , etc</p>
</div>
<div class="post-text" itemprop="text">
<p>In case on python 2.7, the easiest solution is that:</p>
<pre><code>a=[0, 10, 20, 30, 40, 50, 60]
c=[1,2]
values = [a[i] for i, x in enumerate(a) if i not in c]
print values
[0, 30, 40, 50, 60]
</code></pre>
