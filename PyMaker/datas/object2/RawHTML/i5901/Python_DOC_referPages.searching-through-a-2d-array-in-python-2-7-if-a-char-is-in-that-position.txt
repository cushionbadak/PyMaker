<div class="post-text" itemprop="text">
<p>Adding to other answers, you could try using <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter()</code></a> also:</p>
<pre><code>&gt;&gt;&gt; students = [['Gary','Y'],['Steve','N'],['Tom','Y']]
&gt;&gt;&gt; filtered = list(filter(lambda x: x[1] == 'Y', students))
&gt;&gt;&gt; print(filtered)
[['Gary', 'Y'], ['Tom', 'Y']]
&gt;&gt;&gt; print([x[0] for x in filtered])
['Gary', 'Tom']
</code></pre>
