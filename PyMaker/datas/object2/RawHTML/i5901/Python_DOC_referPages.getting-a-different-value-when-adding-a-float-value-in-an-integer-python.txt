<p>I am adding a float value in an integer using loops but the answer is different.
Here is my code:</p>
<pre><code>import time
test = 0
while True:
    test += 0.0001
    print(test)
    time.sleep(0.3)
</code></pre>
<p>When I ran it I should have got:</p>
<pre><code>0.0001
0.0002
0.0003
0.0004
0.0005
.........
</code></pre>
<p>But I got:</p>
<pre><code>0.0001
0.0002
0.00030000000000000003
0.0004
0.0005
0.0006000000000000001
0.0007000000000000001
0.0008000000000000001
0.0009000000000000002
0.0010000000000000002
0.0011000000000000003
0.0012000000000000003
</code></pre>
<p>Why it is so and how to solve this problem?</p>

<div class="post-text" itemprop="text">
<p>You can set the output precision  using <code>{:.4f}</code> to keep four digits .</p>
<pre><code>&gt;&gt;&gt; pi
3.141592653589793
&gt;&gt;&gt; print("{}".format(pi))
3.141592653589793
&gt;&gt;&gt; print("{:.3f}".format(pi))
3.142
&gt;&gt;&gt; print("{:.4f}".format(pi))
3.1416
&gt;&gt;&gt; print("{:.5f}".format(pi))
3.14159
</code></pre>
</div>
