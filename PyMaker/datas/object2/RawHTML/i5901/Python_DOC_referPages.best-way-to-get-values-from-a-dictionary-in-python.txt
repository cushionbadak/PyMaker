<div class="post-text" itemprop="text">
<p>You can iterate through the keys and values of a dict using <a href="https://docs.python.org/3/library/stdtypes.html#dict.items" rel="nofollow noreferrer"><code>items()</code></a>.</p>
<pre><code>for i, js in info.items():
    for j in js:
        # Make query with j
</code></pre>
<p>(In Python 2 you might prefer to use <a href="https://docs.python.org/2/library/stdtypes.html#dict.iteritems" rel="nofollow noreferrer"><code>iteritems()</code></a> instead of <code>items()</code>.)</p>
<p>If you don't need to make use of the keys at all, you could just use <a href="https://docs.python.org/3/library/stdtypes.html#dict.values" rel="nofollow noreferrer"><code>values</code></a> (or <a href="https://docs.python.org/2/library/stdtypes.html#dict.itervalues" rel="nofollow noreferrer"><code>itervalues</code></a> for Python 2).</p>
<pre><code>for js in info.values():
    for j in js:
        # Make query with j
</code></pre>
<p>Maybe a list comprehension would work for you; it is impossible to tell without knowing what "Make query with j" entails.</p>
</div>

</code></pre>

<p>You can do it this way:</p>
<pre><code>for vals in info.values():
    for j in vals:
        # make queries with j
</code></pre>
<p>You can get the list of all the values like this:</p>
<pre><code>values = [x for vals in info.values() for x in vals]
</code></pre>
<p>Then you can use</p>
<pre><code>for j in values:
     # make queries with j
</code></pre>
