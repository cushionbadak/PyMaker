<div class="post-text" itemprop="text">
<p>you can use "bitwise and" for masks like as follows:</p>
<pre><code>item = mydf["Item Name"]=="Pregabalin"
typ = mydf["type"]=="Pregabalin 300mg"

mydf[item &amp; typ]
</code></pre>
<p>or simply (in the following case we must use parentheses because of <a href="https://docs.python.org/3/reference/expressions.html#operator-precedence" rel="nofollow noreferrer">Operator precedence rules</a>):</p>
<pre><code>mydf[(mydf["Item Name"]=="Pregabalin") &amp; (mydf["type"]=="Pregabalin 300mg")]
</code></pre>
