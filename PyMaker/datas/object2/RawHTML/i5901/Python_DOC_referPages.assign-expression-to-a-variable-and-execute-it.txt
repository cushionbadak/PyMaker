<p>I suspect you are suffering from backslashitis. You need one more slash before the <code>n</code>:</p>
<pre><code>textFormat = 'soup.find("div", {"class" : "article-entry text"}).text.replace(\'\\n\', "")'
</code></pre>
<p>But, instead of <code>exec</code>ing code this way, you might want expose a subroutine instead:</p>
<pre><code>def textFormat(soup):
    return soup.find("div", {"class" : "article-entry text"}).text.replace('\n', "")
</code></pre>
</div>
