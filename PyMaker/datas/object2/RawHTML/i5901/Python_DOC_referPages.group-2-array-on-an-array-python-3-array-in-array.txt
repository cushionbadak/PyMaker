<p>To merge arrays in such way you could use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip</a> built-in function.</p>
<p>example:</p>
<pre><code>&gt;&gt;&gt; arr1 = [1,2,3,4,5,6]
&gt;&gt;&gt; arr2 = [7,8,9,10,11,12]
&gt;&gt;&gt; zipped = zip(arr1,arr2)
&gt;&gt;&gt; list(zipped)
[(1, 7), (2, 8), (3, 9), (4, 10), (5, 11), (6, 12)]
