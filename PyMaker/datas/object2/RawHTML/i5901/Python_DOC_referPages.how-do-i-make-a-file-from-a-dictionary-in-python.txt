<p>How do I make a file from a dictionary in python?</p>
<p>For example this is my dictionary: 
dict = {'a':1,'b':2,'c':3}
How do I make it into the first sentence of a file that shows this?
a,1.b,2.c,3.</p>
<p>Thank you to anyone who answers my question.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can try this:</p>
<pre><code>f = open('file.txt', 'w')
dict = {'a':1,'b':2,'c':3}
f.write('.'.join('{},{}'.format(a, b) for a, b in dict.items())+'.\n')
f.close()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>import json
mydict = {'a':1,'b':2,'c':3} 
with open('dict_file.txt', 'w') as file:
     file.write(json.dumps(mydict))
</code></pre>
<p>Hope this helps.</p>
