<div class="post-text" itemprop="text">
<p>Given a list of tuples:</p>
<pre><code>total = [(4, 0), (2, 1), (1, 2), (3, 3)]
</code></pre>
<p>I want to check if the first number == second number + 1 in each tuple.
In case they are equal I need to return <code>False</code> and else return <code>True</code>.</p>
<p>I tried two different methods but both check only the first tuple (<code>(4, 0)</code>)  and then return <code>True</code>, despite that it should return <code>False</code> (because the second tuple <code>(2, 1)</code> is <code>2 == 1 + 1</code>).</p>
<p><strong>Attempt #1:</strong></p>
<pre><code>for tuples in total:
    if tuples[0]==(int(tuples[1])+1):
        return False
    else:
        return True
</code></pre>
<p><strong>Attempt 2:</strong></p>
<pre><code>for a,b in total:
    if a==b+1:
        return False
    else:
        return True
</code></pre>
<p>What am I doing wrong?</p>
</div>
<div class="post-text" itemprop="text">
<p>You are only testing the <strong>first</strong> tuple in your list. <code>return</code> ends the function.</p>
<p>Test <strong>all</strong> tuples before you decide to return <code>True</code>i:</p>
<pre><code>for a, b in total:
    if a == b + 1:
        return False
return True
</code></pre>
<p>So when the first tuple is tested, the <code>a == b + 1</code> test is still false, so you continue to test the next tuple. At that point <code>(2, 1)</code> is tested and <code>False</code> is returned. If you were to remove the <code>(2, 1)</code> tuple from the input, then the <code>for</code> loop ends and the <code>return True</code> line is reached.</p>
<p>You can use the <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer"><code>all()</code> function</a>, together with the inverse test (not equal) to make the above loop more compact:</p>
<pre><code>return all(a != b + 1 for a, b in total)
</code></pre>
<p>This returns <code>False</code> the moment a tuple that matches your criteria is found, <code>True</code> otherwise.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can try this:</p>
<pre><code>def tuple_listing(s):
  return not any(c == b+1 for c, b in s)


total=[(4, 0), (2, 1), (1, 2), (3, 3)]
print(tuple_listing(total))
</code></pre>
<p>Output:</p>
<pre><code>False
</code></pre>
