<p>Probably you are using Python 3: <code>input()</code> returns a string, and <code>raw_input()</code> is deprecated. In contrast, in Python 2, <code>input()</code> returned a numerical value. For details, refer to <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer">Built-In Functions</a> section of the documentation.</p>
<p>You will need to do something like</p>
<pre><code>...
print(contents2)
choose = int(input("So, what is your answer?")) # Notice the `int()` here
...
</code></pre>
<p>or if you are getting many numbers, use  </p>
<pre><code>choose_map = map(int, input("So, what are your answers?").split())
choose_list = list(choose_map)
</code></pre>
<p>If you are using Python 2, and still having this issue, please, provide your list of <code>import</code>s. Something might be off there.</p>
