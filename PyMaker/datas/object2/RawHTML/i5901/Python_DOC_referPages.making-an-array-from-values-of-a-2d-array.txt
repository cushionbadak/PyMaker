<div class="post-text" itemprop="text">
<p>You can use a <code>list-comprehension</code> that iterates through each list in <code>latitudeInfo</code> and takes the <em>first</em> element from that list with <code>l[0]</code>.</p>
<pre><code>possibleLatitudes = [l[0] for l in latitudeInfo]
</code></pre>
<hr/>
<p><em>Note that you are actually using <code>lists</code> here, and <strong>not</strong> <code>arrays</code>. This is a common misconception for people learning Python - especially coming from other languages. Basically the only time you use <code>arrays</code> in Python is with the <code>numpy</code> module. Everything else with square brackets <code>[]</code> is normally a <code>list</code>.</em></p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>map</code> in python like below. <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer">https://docs.python.org/3/library/functions.html#map</a>. It is similar to map in Javascript, Ruby or any other language. </p>
<p><code>lambda</code> is an anonymous function that returns the specified criteria.
<a href="http://www.secnetix.de/olli/Python/lambda_functions.hawk" rel="nofollow noreferrer">http://www.secnetix.de/olli/Python/lambda_functions.hawk</a></p>
<pre><code>lats = list(map(lambda l: l[0], latitudeInfo))
</code></pre>
<p><strong>EXAMPLE</strong></p>
<pre><code>&gt;&gt;&gt; k = [[1, 2], [3, 4], [5, 6]]
&gt;&gt;&gt; list(map(lambda i: i[0], k))
[1, 3, 5]
</code></pre>
</div>
<span class="comment-copy">You neither need <code>lambda</code> nor <code>map</code> for this task. In general list comprehensions should be preferred rather functional programming` methods.</span>
