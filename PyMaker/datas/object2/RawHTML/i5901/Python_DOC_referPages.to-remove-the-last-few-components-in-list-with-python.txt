<div class="post-text" itemprop="text">
<p><code>pop(i)</code> pops the element at index <code>i</code> in the list. When you do <code>a.pop(a[3])</code>, you're popping the element at index <code>a[3]=3</code>. If you want to pop the last element, just do <code>a.pop()</code> without any argument, it will implicitly pop the last element.</p>
</div>
<div class="post-text" itemprop="text">
<p>The reason you're getting the error is because you're looking up what index to pop by retrieving an integer from the list, so by doing <code>a[2]</code>, you are first fetching the value <code>3</code> and obviously this is outside of the range of the list.</p>
<p>So as said otherwise, just call <code>pop</code> without the parameter - <code>pop()</code></p>
</div>
<div class="post-text" itemprop="text">

a.sort()
a=a[:-2]
</code></pre>

<div class="post-text" itemprop="text">
<p>First of all <code>pop</code> is able to just remove the last element, you don't need to specify the argument.
Removing the sorting as it doesn't change much here you get:</p>
<pre><code>a=[2,2,3,3]
a.pop()
print(a)
a.pop()
print(a)
</code></pre>
<p>It should do what you probably wanted.</p>
<p><strong>More context and explanation</strong></p>
<p>As <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer">explained in the docs</a> if you optionally specify the parameter to <code>pop</code> - it takes in the index of the element to be removed.
So in yet even more simpler code:</p>
<pre><code>a=[3,2,2,3]
a.pop(a[3])
</code></pre>
<p>What it means is:</p>
<ul>
<li><code>a[3]</code> - take third element of the list, it's 3 (remember this is 0-indexed)</li>
<li>so in this situation <code>a.pop(a[3])</code> happens to be the same as <code>a.pop(3)</code></li>
<li>so it removes the third element</li>
</ul>
<p>Now your error happens because you do <code>a.pop(a[2])</code> which is again a 3, and the list is too short. </p>
</div>
<div class="post-text" itemprop="text">
<p>Others have already pointed out your usage error.</p>
<p>Even easier, just take the slice of list that you want:</p>
<pre><code>a = sorted(a)[:-2]
</code></pre>
<p>This sorts the list, strips off the last two entries, and assigns the result back to <code>a</code>.</p>
<pre><code>&gt;&gt;&gt; a = [5, 8, 3, 2]
&gt;&gt;&gt; a = sorted(a)[:-2]
&gt;&gt;&gt; a
[2, 3]
</code></pre>
</div>
