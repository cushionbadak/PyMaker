<p>I have some variable from API.
For example:</p>
<pre><code>a == '\xd0\xbc'
</code></pre>
<p>I need to get variable b by decoding a:</p>
<pre><code>b == 'лю'
</code></pre>
<p>But I can't write in code something like this:</p>
<pre><code>a = b'\xd0\xbc'
b = a.decode()
</code></pre>
<p>because this text (\xd0\xbc) is already in the variable.</p>

<div class="post-text" itemprop="text">
<p>You need to use <code>str.encode()</code> and pass <code>encoding="latin"</code> to get <code>b == 'M'</code>.</p>
<pre><code>a = str.encode('\xd0\xbc', 'latin')
b = a.decode()
print(b)
</code></pre>
<p>If <code>"latin"</code> is not the standard encodings you were looking for, <a href="https://docs.python.org/3/library/codecs.html#standard-encodings" rel="nofollow noreferrer">try others</a>. </p>
