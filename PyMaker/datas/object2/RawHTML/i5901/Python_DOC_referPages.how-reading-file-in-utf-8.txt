<div class="post-text" itemprop="text">
<p>You are storing the <em>representation of your strings</em>. <code>repr()</code> is a debugging tool, and outputs a valid Python expression to reproduce your string. So you get a string that contains a valid Python string literal, with any non-printable non-ASCII characters replaced with escape sequences (which always start with <code>\</code> followed by a single character, or <code>x</code> plus 2 hex characters, <code>u</code> with 4 hex characters or <code>U</code> with 8, depending on the codepoint).</p>
<p>Don't use <code>repr()</code>. All you have is strings with some whitespace (tabs and newlines), so <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>str.strip()</code></a> is all you need:</p>
<pre><code>apellidos_list = []
with io.open('Apellidos200.txt',encoding='utf-8') as fp:
    for line in fp:
        apellidos_list.append(line.strip())
</code></pre>
<p>or use a list comprehension:</p>
<pre><code>with io.open('Apellidos200.txt',encoding='utf-8') as fp:
    apellidos_list = [line.strip() for line in fp]
</code></pre>
