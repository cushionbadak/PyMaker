<p>I have a list of objects and I want to only return objects that have a field that contains a certain value. </p>
<p>Is there a more terse/pythonic way of doing this:</p>
<pre><code>list-o-dicts = get-my-objects()

for dict in list-o-dicts:
    if 'mystring' in dict['myfield']:
        pprint( dict )
</code></pre>
<p>Does python have a cool, sugary, shortcut for doing this?</p>
</div>
<div class="post-text" itemprop="text">
<p>Try <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter</code></a>:</p>
<pre><code>filter(lambda x: 'mystring' in x['myfield'], list-o-dicts)
</code></pre>
</div>
<span class="comment-copy">I suppose you could use a list comprehension if you wanted to sugar it up for no good reason: <code>output = [dict for dict in list-o-dicts if 'mystring' in dict['myfield']]</code></span>
