<p>I have a text file with data.
In each line there is one or two words and one number, so the problem is, that sometimes there are 2 columns and sometimes 3.</p>
<pre><code>aaa bbb 4
ccc 10
ddd eee 3
fff 5
</code></pre>
<p>I only need those numbers. How to copy them to a list in Python?</p>
<p>I tried with</p>
<pre><code>file = open("data.txt","r")
data = file.readlines()
result = []
for line in data:
    result.append(line.split(' ')[-1])
file.close()
</code></pre>
<p>But I don't know how to get rid of '\n' in the end of number? Because when I change <code>result.append(line.split(' ')[-1])</code>
to </p>
<p><code>result.append(int(line.split(' ')[-1]))</code>, it doesn't work.
Is there a better way?</p>

<div class="post-text" itemprop="text">
<p>If the line number is irrelevant:</p>
<pre><code>import re

s = '''aaa bbb 4
ccc 10
ddd eee 3
fff 5'''

matches = re.findall('\d+', s)
for match in matches:
    print(match)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>When python reads text files, it also reads in the newline characters (<code>'\n'</code> at the end of each line). You may want to remove these characters, and since these are numbers cast the value to <code>int</code> before appending to your list:</p>
<pre><code>file = open("data.txt","r")
data = file.readlines()
result = []
for line in data:
    result.append(int(line.strip().split(' ')[-1])) # the only line changed 
file.close()

print(result)
</code></pre>
<p>output:</p>
<pre><code>[4, 10, 3, 5]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you split each line by the space you would have a list of everything in the line. You can then use <code>lst[-1]</code> to get the last element in that list. for example:</p>
<pre><code>for line in file:
    lst=line.split(" ")
    lastCol=lst[-1]
</code></pre>
<p>this should get you just the last column in each line. If you also need to store it as a number you can do <code>lastCol=int(lst[-1])</code> as the last line.</p>
<p>EDIT:</p>
<p>You can use <code>strip()</code> to remove the <code>"\n"</code> and all other white-space at the end of the line:</p>
<pre><code>lst=line.strip().split(" ")
</code></pre>
