<p>I am writing a function and it should be able to take different arguments in one time. How can I define this arguments of the function?</p>
<pre><code>def function('005=5=0002555')
def function('00025:001:002325', '/')
def function(date='05-0004-002294', separator='/')
def function(separator='*', date='0008=006=2001')
def function('21 12 2012', separator='+')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a combination of <code>*args</code> and <code>**kwargs</code>:</p>
<pre><code>def function(*args, **kwargs):
   args1 = args if args else None
   args2 = [kwargs.get(i) for i in ['date', 'separator']]
</code></pre>
