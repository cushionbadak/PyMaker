<p>I have this list of average water flow values in gallons from a set of 5 garden hoses:</p>
<p><code>[6.6, 14, 12.6, 8.8, 11.8]</code></p>
<p>I want to calculate the subcontrary/harmonic mean of this list and I found this <a href="https://stackoverflow.com/a/13855567/9042556">answer</a> that works:</p>
<pre><code>def hmean(args):
    return len(args) / sum(1. / val for val in args)

hose_flows = [6.6, 14, 12.6, 8.8, 11.8]
print(hmean(hose_flows)) # 9.986200495805194
</code></pre>
<p>However I don't find this very elegant... There surely is a more elegant way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>If you're using Python 3.6.0 or above you can use <a href="https://docs.python.org/3/library/statistics.html#statistics.harmonic_mean" rel="nofollow noreferrer">statistics.<strong>harmonic_mean</strong></a>:</p>
<pre><code>&gt;&gt;&gt; import statistics
&gt;&gt;&gt; hose_flows = [6.6, 14, 12.6, 8.8, 11.8]
&gt;&gt;&gt; statistics.harmonic_mean(hose_flows)
9.986200495805194
</code></pre>
