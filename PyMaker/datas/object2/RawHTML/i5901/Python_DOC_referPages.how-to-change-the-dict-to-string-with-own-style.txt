<p>What seems like you're doing is changing a <code>dict</code> into url <code>params</code>, you can do <a href="https://docs.python.org/2/library/urllib.html#urllib.urlencode" rel="noreferrer">urlencode</a>.</p>
<pre><code>&gt;&gt;&gt; import urllib
&gt;&gt;&gt; a = {'a':123,'b':234,'c':345}
&gt;&gt;&gt; urllib.urlencode(a)
'a=123&amp;c=345&amp;b=234'
</code></pre>
