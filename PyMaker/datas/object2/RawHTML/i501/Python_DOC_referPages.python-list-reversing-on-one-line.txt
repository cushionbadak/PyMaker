<pre><code>ls = list(range(10))
ls.reverse()
print(ls)
</code></pre>
<p>Why does this work in producing a list that counts backwards from 9 not but not this:</p>
<pre><code>ls = list(range(10)).reverse()
print(ls)
</code></pre>
<p>These last two lines prints this instead:</p>
<pre><code>None
</code></pre>
<p>Shouldn't they be the same thing?</p>
</div>
<div class="post-text" itemprop="text">
<p>No, because <code>list.reverse()</code> returns <code>None</code> since it reverses the list in place. See the <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="nofollow noreferrer">list documentation</a></p>
<p>You could use <code>reversed()</code> like so:</p>
<pre><code>countdown = list(reversed(range(10)))
print(countdown)
</code></pre>
<p>See the <a href="https://docs.python.org/3/library/functions.html#reversed" rel="nofollow noreferrer">reversed documenation</a></p>
<p>See also <a href="https://stackoverflow.com/questions/4280691/list-reverse-does-not--list">this question</a></p>
</div>
<span class="comment-copy">because <code>.reverse</code> works in-place, and so by convention it returns <code>None</code>. There is no reason to believe they should be the same thing. They can and do work differently. Note, you can just do <code>list(reversed(range(10))</code></span>
