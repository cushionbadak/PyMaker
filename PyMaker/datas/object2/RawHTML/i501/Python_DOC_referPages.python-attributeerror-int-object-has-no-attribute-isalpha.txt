<div class="post-text" itemprop="text">
<p>In python 3, <a href="https://docs.python.org/3/library/socket.html#socket.socket.recv" rel="nofollow noreferrer"><code>socket recv</code></a> returns an immutable sequence of <a href="https://docs.python.org/3/library/functions.html#func-bytes" rel="nofollow noreferrer"><code>bytes</code></a>. Each element has type <code>int</code> and is in the range [0, 255]. <a href="https://docs.python.org/3/library/stdtypes.html#str.isdecimal" rel="nofollow noreferrer"><code>isalpha</code></a> is a method of <code>str</code> and is not a method of <code>int</code>.</p>
<p>If you'd like to treat the server response as a string, you can decode the bytes.</p>
<pre><code>string = response.decode('utf-8')

for c in string:
