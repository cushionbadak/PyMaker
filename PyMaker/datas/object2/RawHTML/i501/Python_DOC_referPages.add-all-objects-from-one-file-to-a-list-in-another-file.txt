<div class="post-text" itemprop="text">
<p>I have the following python file <code>constants.py</code> which stores a bunch of constants:</p>
<pre><code>ONE = 1
TWO = 2
ANOTHER = "Test"
....
</code></pre>
<p>I'm trying to import all of the variables and add them to a list called <code>data</code>. This is what I have so far but if I add a constant to <code>constants.py</code> then I need to remember to add it to the <code>data</code> list:</p>
<pre><code>from constants import *
data = [ONE, TWO, ANOTHER, ...]
</code></pre>
<p>Is there a better approach here?</p>
</div>
<div class="post-text" itemprop="text">
<p>The solution would be to create the list in the <code>constants.py</code> file. It would like as follows:</p>
<pre><code>ONE = 1
TWO = 2
ANOTHER = "Test"
....
data = [ONE, TWO, ANOTHER, ...]
</code></pre>
<p>Then you can simply import the list in your main file.</p>
</div>
<div class="post-text" itemprop="text">
<p>One approach would be to use the <a href="https://docs.python.org/3/library/importlib.html" rel="nofollow noreferrer"><code>importlib</code></a> module. If your <code>constants.py</code> module contains:</p>
<pre><code>ONE = 1
TWO = 2
ANOTHER = "Test"

__all__ = [ "ONE", "TWO", "THREE", "ANOTHER" ]
</code></pre>
<p>You can import those as a module:</p>
<pre><code>&gt;&gt;&gt; import importlib
&gt;&gt;&gt; constants = importlib.import_module('constants')
&gt;&gt;&gt; constants.ONE
1
&gt;&gt;&gt; constants.__dict__
{ ... 'ONE': 1, 'TWO': 2, 'THREE': 3, 'ANOTHER': 'Test'}
</code></pre>
<p>Due to the bloat of the "dunder" attributes (<code>__spec__</code>, <code>__builtins__</code> etc.), it can be useful to include the <code>__all__</code> attribute.</p>
<pre><code>&gt;&gt;&gt; constants_dict = { k: constants.__dict__[k] for k in constants.__all__ }
&gt;&gt;&gt; constants_dict
{'ONE': 1, 'TWO': 2, 'THREE': 3, 'ANOTHER': 'Test'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If the variables start with uppercase, you can do this</p>
<pre><code>ONE = 1
TWO = 2
ANOTHER = "Test"

exec('data = [' + ','.join([e for e in dir() if e[0].isupper()]) + ']')
</code></pre>
<p>and you don't have to remember to add them to <code>data</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Somewhat similar to other responses, but no other modules are required.</p>
<pre><code>data = {name:getattr(constants, name) for name in dir(constants) if name[0] != '_'}
print(data) #Prints {'ANOTHER': 'Test', 'ONE': 1, 'TWO': 2}
</code></pre>
<p>Note that here the import statement should be <code>import constants</code> and not <code>from constants import *</code>.</p>
</div>
