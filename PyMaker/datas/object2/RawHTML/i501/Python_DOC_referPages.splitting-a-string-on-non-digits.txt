<div class="post-text" itemprop="text">
<p>Well ... you are using <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="noreferrer"><code>str.split()</code></a>  - which takes characters to split at - not regex. Your code would split on any <code>'\D+'</code> - string inside your text:</p>
<pre><code>orig = 'Some\\D+text\\D+tosplit'
results = orig.split(r'\D+')  # ['Some', 'text', 'tosplit']
</code></pre>
<p>You can use <a href="https://docs.python.org/3/library/re.html#re.split" rel="noreferrer">re.split()</a> instead: </p>
<pre><code>import re

orig = '0 1,2.3-4:5;6d7'
results = re.split(r'\D+',orig)
print(results)
</code></pre>
<p>to get</p>
<pre><code>['0', '1', '2', '3', '4', '5', '6', '7']
</code></pre>
<p>Use <code>data = list(map(int,results))</code> to convert to int.</p>
</div>
