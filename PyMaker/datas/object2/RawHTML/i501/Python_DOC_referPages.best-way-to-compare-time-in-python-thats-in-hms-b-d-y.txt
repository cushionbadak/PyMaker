<div class="post-text" itemprop="text">
<p>You're right, there's certainly an easier way. You may want to try this approach:</p>
<ul>
<li>Convert the given string into a <code>datetime</code> object.  Click here for info on Python's <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer">datetime module</a>.  See also <a href="https://stackoverflow.com/questions/14295673/convert-string-into-datetime-time-object">this answer</a>.</li>
<li>Generate the current time as another <code>datetime</code> object.</li>
<li>Get the time difference between the two.  This is just a matter of subtracting one from the other. <a href="https://stackoverflow.com/questions/1345827/how-do-i-find-the-time-difference-between-two-datetime-objects-in-python">See this answer</a> for an example.</li>
<li>Convert this to minutes.  This is a little trickier.  If your two times are <code>a</code> and <code>b</code>, then <code>c = a-b</code> gives you a new datetime object representing the difference.  You can take the values <code>c.days</code>, <code>c.seconds</code> and <code>c.microseconds</code>, then mathematically convert each of the values to minutes, and add them all up.</li>
</ul>
</div>

<p>Best way is to parse your string into <code>datetime</code> object with <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.strptime" rel="nofollow noreferrer"><code>strptime</code></a> an operate on that level.</p>
