<p>As @James commented, the magic methods you were trying to define require <em>double</em> underscores. Use this and it should work:</p>
<pre><code>class Set:
    def __init__(self, values=None):
        self.dict = {}
        if values is not None:
            for value in values:
                self.add(value)

    def __repr__(self):
        return "Set:"+str(self.dict.keys())
. .  .
</code></pre>
