<div class="post-text" itemprop="text">
<p><code>.index()</code> is not used for showing its original index but the first index that matches the value you passed into.</p>
<p>You can use <code>enumerate</code> to get both the index and the value in the list.</p>
<pre class="lang-py prettyprint-override"><code>a = [0,3,3,7,5,3,11,1]
for index, value in a:
    print('{} {}'.format(index, value))
</code></pre>
<p>And not sure why you wanted to convert a to a list of strings instead of integers?</p>
</div>

<p>You can use Python's built-in function <code>enumerate()</code>.</p>
<p>In your case you can do:</p>
<pre><code>a = [0,3,3,7,5,3,11,1] 

for index, element in enumerate(a):
    print index
</code></pre>
<p>The output should be:</p>
<pre><code>0
1
2
3
4
5
6
7
</code></pre>
<p>You can check the documentation in this <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer">link</a>.</p>
