<div class="post-text" itemprop="text">
<p>Instead of </p>
<pre class="lang-py prettyprint-override"><code>doc.add_picture('logo.PNG')
</code></pre>
<p>try:</p>
<pre class="lang-py prettyprint-override"><code>doc.add_picture(os.path.join(os.path.dirname(__file__), 'logo.PNG'))
</code></pre>
<p>This will make sure the PNG is next to the python script.</p>
<p>Also, don't use uppercase.</p>
