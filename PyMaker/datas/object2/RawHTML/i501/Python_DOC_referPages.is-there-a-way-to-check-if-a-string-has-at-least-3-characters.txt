<div class="post-text" itemprop="text">
<p>You can use either the <code>len</code> built-in method or regex.</p>
<ol>
<li><code>len</code> built-in:</li>
</ol>
<pre class="lang-py prettyprint-override"><code>s = 'a'
print(len(s) &gt;= 3)
</code></pre>
<p>Doc: <a href="https://docs.python.org/3/library/functions.html#len" rel="nofollow noreferrer"><code>len</code> built in</a></p>
<ol start="2">
<li>Regex</li>
</ol>
<pre class="lang-py prettyprint-override"><code>import re

s = 'apple'
print(re.search('^[a-zA-Z0-9]{3,}$', s))
</code></pre>
<p>Doc: <a href="https://docs.python.org/3/library/re.html#re.Pattern.search" rel="nofollow noreferrer"><code>re.search</code></a></p>

<p>You can use the len built-in method.</p>
<pre><code>myStr = 'foo'
print(len(myStr) &gt;= 3)
</code></pre>
<p>Description
The len() method returns the length of the string.</p>
