<div class="post-text" itemprop="text">
<p>Try the subprocess <a href="https://docs.python.org/3/library/subprocess.html" rel="nofollow noreferrer">manual</a>. You have options with subprocess to work with stdin, stdout, and stderr of the process you're calling.</p>
<pre><code>from subprocess import Popen, PIPE, STDOUT

p = Popen(['ask', 'util', 'generate-lwa-tokens'], stdout=PIPE, stdin=PIPE, stderr=STDOUT, universal_newlines=True)

# Storing the values that should be passed

values = ["client_id", "client_secret"]

# Interacting with the shell 
output, err = p.communicate(input=f'{values[0]}\n{values[1]}\n')
# Displaying outputs
print(output)
</code></pre>
