<p><a href="https://docs.python.org/3/library/stdtypes.html?highlight=popitem#dict.popitem" rel="nofollow noreferrer">popitem()</a> in dictionary guarantee to return last inserted pair for 3.7 and onward. Dictionary is unordered type. So, how dictionary is able to achieve this? </p>

<div class="post-text" itemprop="text">
<p>It is no longer unordered starting 3.6 (implementation detail) and guaranteed as of 3.7. So naturally methods that are based on order have predictable results now.</p>
