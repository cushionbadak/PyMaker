<div class="post-text" itemprop="text">
<p>You can use the <code>sum</code> function with a generator expression:</p>
<pre><code>def getNumWordsWithNVowels(wordList, num):
    return sum(1 for w in wordList if sum(c in 'aeiou' for c in w.lower()) == num)
</code></pre>
<p>so that:</p>
<pre><code>aList = ['hello', 'aloha', 'world', 'foo', 'bar']
print(getNumWordsWithNVowels(aList, 1))
print(getNumWordsWithNVowels(aList, 2))
print(getNumWordsWithNVowels(aList, 3))
</code></pre>
<p>outputs:</p>
<pre><code>2 # world, bar
2 # hello, foo
1 # aloha
</code></pre>
