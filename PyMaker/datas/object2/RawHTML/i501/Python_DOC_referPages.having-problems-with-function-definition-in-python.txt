<p>The <code>input()</code> built-in function in python takes only 1 parameter - <em>prompt</em>. Please refer to the python documentation for <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow noreferrer">input function</a></p>
<p>Edit: As per your comment, you need to update your prompt to include the parameter that you've sent. See the code below. As chris mentioned in the comments, the f-strings will work only in Python versions 3.6</p>
<pre><code>def prompt(n): 
    value=int(input(f"Please enter integer {}".format(n))) 
    return value
</code></pre>
<p>For Python versions &lt; 3.6, you can you use the old formatting strings as shown in the code below</p>
<pre><code>def prompt(n): 
    value=int(input("Please enter integer {}".format(n))) 
    return value
</code></pre>
