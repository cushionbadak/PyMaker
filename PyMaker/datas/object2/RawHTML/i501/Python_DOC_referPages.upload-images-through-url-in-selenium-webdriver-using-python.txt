<p>You first need to download the image to your computer and then upload it...</p>
<p>You can use <code>requests</code>:</p>
<pre><code>import requests

URL = "https://ercess.com//images//events//-Blockchain-2019-36613-banner.png"
picture_req = requests.get(URL)
if picture_req.status_code == 200:
    with open("/path/to/image.jpg", 'wb') as f:
        f.write(picture_req.content)
</code></pre>
<p>Then send the <code>/path/to/image.jpg</code>:</p>
<pre><code>driver.find_element(By.XPATH, "//*[@id='upl-zone']/input").send_keys("/path/to/image.jpg")
</code></pre>
<hr/>
<p>Or you can use the <a href="https://docs.python.org/3/library/urllib.request.html?highlight=request#legacy-interface" rel="nofollow noreferrer"><code>Legacy interface</code></a> of <code>urllib</code> you will use <code>urlretrieve</code>:</p>
<pre><code>import urllib.request

URL = "https://ercess.com//images//events//-Blockchain-2019-36613-banner.png"
urllib.urlretrieve(URL, "file_name.png")
driver.find_element(By.XPATH, "//*[@id='upl-zone']/input").send_keys("file_name.png")
</code></pre>
<p><strong>EDIT:</strong></p>
<p>To send the path to the file with send_keys you can use <a href="https://docs.python.org/3/library/pathlib.html#methods" rel="nofollow noreferrer"><code>pathlib</code></a> </p>
<pre><code>from pathlib import Path

# `cwd`: current directory is straightforward
cwd = Path.cwd()
# using "F"string for format you can use: image_file_name = str(cwd) + "\" + "file_name.png" 
image_file_name = fr"{cwd}\file_name.png"
# this print is just to show the image_file_name   
print(image_file_name)
</code></pre>
<p>Hope this helps you!</p>
