<div class="post-text" itemprop="text">
<p>How can you convert a tuple into a string in the most simplest way possible? for instance. Lets say I have a list of pairs such as:</p>
<pre><code>[(1,"hi"),(True,False),(3,4)] 
</code></pre>
<p>and I want to convert the first pair to '1hi'</p>
</div>
<div class="post-text" itemprop="text">
<p>use this:</p>
<pre><code>&gt;&gt;&gt; l = [(1,"hi"),(True,False),(3,4)]
&gt;&gt;&gt; for i in l:
...     print str(i[0])+str(i[1])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>For the first element only:</p>
<pre><code>l =  [(1,"hi"),(True,False),(3,4)]
s = "".join(str(x) for x in l[0])
print s
</code></pre>
<p>To do this to all elements:</p>
<pre><code>for t in l:
    s = "".join(str(x) for x in t)
    print s
</code></pre>
<p>Gives:</p>
<pre><code>1hi
TrueFalse
34
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>map(lambda e: ''.join(map(str,e)), 
        [(1,"hi"),(True,False),(3,4)])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="http://docs.python.org/3/library/functions.html#map" rel="nofollow"><code>map</code></a>:</p>
<pre><code>&gt;&gt;&gt; l = [(1,"hi"),(True,False),(3,4)]
&gt;&gt;&gt; ''.join(map(str, l[0]))
'1hi'
&gt;&gt;&gt; ''.join(map(str, l[1]))
'TrueFalse'
&gt;&gt;&gt; ''.join(map(str, l[2]))
'34'
</code></pre>
