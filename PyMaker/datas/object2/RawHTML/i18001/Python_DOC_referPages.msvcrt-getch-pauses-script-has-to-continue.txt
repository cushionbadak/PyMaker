<p>Use <a href="http://docs.python.org/3/library/msvcrt.html#msvcrt.kbhit" rel="nofollow"><code>msvcrt.kbhit</code></a> to check whether key was pressed:</p>
<pre><code>import sys, msvcrt
import time

print("Please press a key to see its value")
while 1:
    print("the key is")
    if msvcrt.kbhit(): # &lt;--------
        key = msvcrt.getch()
        print(key)
        if ord(key) == 27:
            sys.exit()
    time.sleep(0.1)
</code></pre>
