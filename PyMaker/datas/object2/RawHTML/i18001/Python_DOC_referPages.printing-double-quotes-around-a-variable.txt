<p>For instance, we have:</p>
<pre><code>word = 'Some Random Word'
print '"' + word + '"'
</code></pre>
<p>is there a better way to print double quotes around a variable?</p>
</div>
<div class="post-text" itemprop="text">
<p><strong>Update :</strong></p>
<p>From <a href="https://docs.python.org/3/whatsnew/3.6.html#whatsnew36-pep498" rel="nofollow noreferrer">Python 3.6</a>, you can use <a href="https://docs.python.org/3/reference/lexical_analysis.html#f-strings" rel="nofollow noreferrer">f-strings</a></p>
<pre><code>&gt;&gt;&gt; print(f'"{word}"')
"Some Random Word"
</code></pre>
<p><strong>Original Answer :</strong></p>
<p>You can try <a href="https://pyformat.info/" rel="nofollow noreferrer">%-formatting</a></p>
<pre><code>&gt;&gt;&gt; print('"%s"' % word)
"Some Random Word"
</code></pre>
<p>OR <a href="https://docs.python.org/3.6/library/string.html#format-examples" rel="nofollow noreferrer"><code>str.format</code></a></p>
<pre><code>&gt;&gt;&gt; print('"{}"'.format(word))
"Some Random Word"
</code></pre>
<p>OR escape the quote character with <a href="https://docs.python.org/3/reference/lexical_analysis.html#string-and-bytes-literals" rel="nofollow noreferrer"><code>\</code></a></p>
<pre><code>&gt;&gt;&gt; print("\"%s\"" % word)
"Some Random Word"
</code></pre>
<p>And, if the double-quotes is not a restriction (i.e. single-quotes would do)</p>
<pre><code>&gt;&gt;&gt; from pprint import pprint, pformat
&gt;&gt;&gt; print(pformat(word))
'Some Random Word'
&gt;&gt;&gt; pprint(word)
'Some Random Word'
</code></pre>
<p>OR like others have already said (include it in your declaration)</p>
<pre><code>&gt;&gt;&gt; word = '"Some Random Word"'
&gt;&gt;&gt; print(word)
"Some Random Word"
</code></pre>
<p>Use whichever <em>you</em> feel to be better or less confusing.</p>
<p>And, if you need to do it for multiple words, you might as well create a function</p>
<pre><code>def double_quote(word):
    return '"%s"' % word

print(double_quote(word), double_quote(word2))
</code></pre>
<p>And (if you know what you're doing &amp;) if you're concerned about performance of these, see <a href="https://stackoverflow.com/a/48465349/2689986">this comparison</a>.</p>
</div>
<div class="post-text" itemprop="text">
<p>How about <code>json.dumps</code>:</p>
<pre><code>&gt;&gt;&gt; import json
&gt;&gt;&gt; print(json.dumps("hello world"))
"hello world"
</code></pre>
<p>The advantage over other approaches mentioned here is that it escapes quotes inside the string as well (take that <code>str.format</code>!), always uses double quotes and is actually intended for reliable serialization (take that <code>repr()</code>!):</p>
<pre><code>&gt;&gt;&gt; print(json.dumps('hello "world"!'))
"hello \"world\"!"
</code></pre>
</div>
