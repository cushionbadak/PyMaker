<div class="post-text" itemprop="text">
<p><code>sections</code> attr of PE class instance is sections list and each item have get_data function:</p>
<pre><code>&gt;&gt;&gt; import pefile
&gt;&gt;&gt; pe = pefile.PE('./psfile.exe')

&gt;&gt;&gt; for section in pe.sections:
...   print (section.Name, hex(section.VirtualAddress),
...     hex(section.Misc_VirtualSize), section.SizeOfRawData )
... 
('.text\x00\x00\x00', '0x1000', '0xd3e4', 57344)
('.rdata\x00\x00', '0xf000', '0x5504', 24576)
('.data\x00\x00\x00', '0x15000', '0x3684', 8192)
('.rsrc\x00\x00\x00', '0x19000', '0x444', 4096)
</code></pre>
<p>10 first bytes from text section data for example:</p>
<pre><code>&gt;&gt;&gt; pe.sections[0].get_data()[:10]
'\x81\xec\x90\x00\x00\x00j&gt;\x8dD'
</code></pre>
