<div class="post-text" itemprop="text">
<p>Judging by your <code>print</code> syntax as well as the fact that you are using <code>input</code> and not <code>raw_input</code>, I believe this is Python 3.x.  If so, then you can just do this:</p>
<pre><code>print(*e2, sep="\n")
</code></pre>
<p>See a demonstration below:</p>
<pre><code>&gt;&gt;&gt; EnterString = input("Enter the first word of your sentence ")
Enter the first word of your sentence Hello world
&gt;&gt;&gt; e = EnterString
&gt;&gt;&gt; e2 = e.split()
&gt;&gt;&gt; print(*e2, sep="\n")
Hello
world
&gt;&gt;&gt;
</code></pre>
<p>Here is a <a href="https://stackoverflow.com/questions/2322355/proper-name-for-python-operator">reference</a> on what the <code>*</code> is.</p>
<p>Also, <code>str.split</code> defaults to split on whitespace.  So, all you need is <code>e.split()</code>.</p>
<hr/>
<p>However, if you are in fact on Python 2.x (namely, Python 2.7), then you will also need to put this line at the top of your script:</p>
<pre><code>from __future__ import print_function
</code></pre>
<p>The above will make <code>print</code> like it is in Python 3.x, allowing you to use it as I demonstrated.</p>
