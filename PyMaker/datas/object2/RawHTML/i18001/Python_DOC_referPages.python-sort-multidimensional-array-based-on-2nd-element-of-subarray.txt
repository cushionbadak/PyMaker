<div class="post-text" itemprop="text">
<p>I have an array like this:</p>
<pre><code>[['G', 10], ['A', 22], ['S', 1], ['P', 14], ['V', 13], ['T', 7], ['C', 0], ['I', 219]]
</code></pre>
<p>I'd like to sort it based on the 2nd element in descending order.
An ideal output would be:</p>
<pre><code>[['I', 219], ['A', 22], ['P', 14], ... ]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>list.sort</code>, <a href="http://docs.python.org/2/library/functions.html#sorted" rel="noreferrer"><code>sorted</code></a> accept optional <code>key</code> parameter. <code>key</code> function is used to  generate comparison key.</p>
<pre><code>&gt;&gt;&gt; sorted(lst, key=lambda x: x[1], reverse=True)
[['I', 219], ['A', 22], ['P', 14], ['V', 13], ['G', 10], ...]

&gt;&gt;&gt; sorted(lst, key=lambda x: -x[1])
[['I', 219], ['A', 22], ['P', 14], ['V', 13], ['G', 10], ...]

&gt;&gt;&gt; import operator
&gt;&gt;&gt; sorted(lst, key=operator.itemgetter(1), reverse=True)
[['I', 219], ['A', 22], ['P', 14], ['V', 13], ['G', 10], ...]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>itemgetter</code></p>
<pre><code>from operator import itemgetter
a = [[1, 3, 5], [2, 511, 7], [17, 233, 1]]
a = sorted(a, key=itemgetter(1))
</code></pre>
<p>Output : [[1, 3, 5], [17, 233, 1], [2, 511, 7]]</p>
<p><code>itemgetter</code> can also be used to sort by multiple subarrays.</p>
</div>
