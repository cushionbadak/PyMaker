<div class="post-text" itemprop="text">
<pre><code>def jinja_render(template, **params):
</code></pre>
<p>means that, it accepts only one non-named parameter and all of them should be named parameters . To fix this error, you can call the function like this</p>
<pre><code>jinja_render('welcome.html', username = 'test')
</code></pre>
<p>when we call like this, <code>welcome.html</code> is a non-named parameter, where is <code>test</code> is a named parameter and the name of the parameter is <code>username</code>. There is another way to call this,</p>
<pre><code>jinja_render('welcome.html', **{'username': 'test'})
</code></pre>
<p>This way we can unpack the values from the dictionary as named parameters to the function.</p>
