<p>Use <code>str.translate()</code> to use a translation table for faster translation.  Here is an example:</p>
<pre><code>with open("sample.txt","r") as fo:
    lines = fo.readlines()

replacements = {'4':'A', '8':'b', '3':'e', '5':'s', '7':'t', '@':'a', '3':'e', '0':'o'}
trans_table = str.maketrans(replacements)
for line in lines:
    print(line, end="")
    new_line = line.translate(trans_table)
    print(new_line)
</code></pre>
