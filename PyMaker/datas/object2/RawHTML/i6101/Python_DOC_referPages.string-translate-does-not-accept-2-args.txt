<p>I have looked at several answers on
<a href="https://stackoverflow.com/questions/265960/best-way-to-strip-punctuation-from-a-string-in-python">Best way to strip punctuation from a string in Python</a>
but none of these seem to address my problem. I am trying to strip punctuation from a string using string.translate().</p>
<p>When I run the code: </p>
<pre><code>import string
s = "This. has? punctuation," 
noPunct = s.translate(s.maketrans("",""), string.punctuation)
</code></pre>
<p>I get:   </p>
<pre><code>TypeError: translate() takes exactly one argument (2 given)
</code></pre>
<p>Is this perhaps an issue with the python version I am using? I am using python 3.5.4 for compatibility with nltk. Otherwise I am stumped. Any help would be appreciated. </p>

<div class="post-text" itemprop="text">
<p>The Python 3 interface for <code>str.translate</code> uses a mapping.  Make one with <code>str.maketrans</code>:</p>
<pre><code>&gt;&gt;&gt; import string
&gt;&gt;&gt; table = str.maketrans({}.fromkeys(string.punctuation))
&gt;&gt;&gt; "This. has? punctuation,".translate(table)
'This has punctuation'
</code></pre>
</div>
