<div class="post-text" itemprop="text">
<p>You can use <code>re.sub</code>:</p>
<pre><code>import re
string = "\n\nHello"
new_string = re.sub("\n+", '', string)
</code></pre>
<p>Output:</p>
<pre><code>'Hello'
</code></pre>
<p>This will work for any number of <code>\n</code> at the beginning of a string.</p>
</div>
<div class="post-text" itemprop="text">
<p>You want <a href="https://docs.python.org/3/library/stdtypes.html#str.lstrip" rel="nofollow noreferrer"><code>str.lstrip()</code></a>. Exact same as <code>strip()</code> except it strips from the left only, not both directions.</p>
<pre><code>&gt;&gt;&gt; s = '\n\nhi\n'
&gt;&gt;&gt; s.lstrip('\n')
'hi\n'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Duplicate of <a href="https://stackoverflow.com/questions/1185524/how-do-i-trim-whitespace-with-python">How do I trim whitespace with Python?</a></p>
<p>But basically: <code>string = string.lstrip('\n')</code></p>
