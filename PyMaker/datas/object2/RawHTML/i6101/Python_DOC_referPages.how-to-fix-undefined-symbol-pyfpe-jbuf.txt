<p>I've just figured it out. I solve it out by uninstalling the cython module and then reinstalling a different version of it. As the official documentation says, there is not any kind of guarantee if the newer versions work fine while compilation. So, first uninstall cython:</p>
<pre><code>pip uninstall cython
</code></pre>
<p>Install the version 0.23:</p>
<pre><code>pip install cython==0.23
</code></pre>
<p>And then again install Kivy, and make sure there's no error while compilation:</p>
<pre><code>pip install kivy
</code></pre>
<p>Hope, this will work fine. </p>
