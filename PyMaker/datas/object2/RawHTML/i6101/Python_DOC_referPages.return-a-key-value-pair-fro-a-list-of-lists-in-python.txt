<div class="post-text" itemprop="text">
<p>I have a list of lists: <code>myList = [['make', 'Ford'], ['model', 'Mustang'], ['year', 1964]]</code>
Would like to return a dictionary with key, value pairs <code>dict={make:'Ford',model:'Mustang',......}</code></p>
<pre><code>d ={}
for row in myList:
  for col in row:
    d[col]=row[1]
</code></pre>
<p>This returns the first key value pair correct <code>{'make':'Ford','Ford':'Ford','model':'Mustang','Mustang':'Mustang'...}</code> but then it repeats the second value.</p>
</div>
<div class="post-text" itemprop="text">
<p>Just use the built-in <a href="https://docs.python.org/3/library/stdtypes.html#dict" rel="nofollow noreferrer"><code>dict</code></a> funtion </p>
<pre><code>myList = [['make', 'Ford'], ['model', 'Mustang'], ['year', 1964]]
d = dict(myList)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>d={}
for row in myList:
    d[row[0]] = row[1]
</code></pre>
<p>This is how youâ€™d do this if you want to use loops, but you can also just <code>d = dict(myList)</code> to cast the list to a dict.</p>

<p>You can try this:</p>
<pre><code>myList = [['make', 'Ford'], ['model', 'Mustang'], ['year', 1964]]
final_dict = {a:b for a, b in myList}

</code></pre>
