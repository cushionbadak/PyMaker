<blockquote>
<p>rpy2.rinterface.RRuntimeError: Error in integrate(PIntegrand, lower =
  0, upper = Inf, lambda, vbar, eta,  :    roundoff error was detected</p>

<p>How do I go about catching this RuntimeError in Python?</p>

<div class="post-text" itemprop="text">
<p>The <code>rpy2.rinterface.RRuntimeError</code> <a href="https://bitbucket.org/rpy2/rpy2/src/6c9ec88e0be3642f3e657d670d55b4f2f51731cf/rpy/rinterface_lib/embedded.py?fileviewer=file-view-default#embedded.py-57" rel="nofollow noreferrer">is derived from <code>Exception</code></a> so you should be able to catch it as you would with any other exception.</p>
<pre><code>try:
    # your code
except rpy2.rinterface.RRuntimeError:
    # handle exception
</code></pre>
<p>More on this: <a href="https://docs.python.org/3/tutorial/errors.html#handling-exceptions" rel="nofollow noreferrer">https://docs.python.org/3/tutorial/errors.html#handling-exceptions</a></p>
</div>
<div class="post-text" itemprop="text">
<p>Python makes catching exceptions relatively easy.</p>
<pre><code>try:
    # some code
except Exception, e:
    # Log the exception.
</code></pre>
