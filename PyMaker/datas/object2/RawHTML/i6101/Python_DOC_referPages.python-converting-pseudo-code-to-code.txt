<p><strong>EDIT</strong>:  Oops!  Forgot that <code>m</code> is a list!</p>
<p>This is what <a href="https://docs.python.org/2/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehensions</a> are for.  Because <code>m</code> is a list, you'll also need to make use of python's <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> function.  The definition of the new array is:</p>
<pre><code>S = [x for x, y in zip(Q, m) if abs(x - y) &lt; d]
</code></pre>
<p>You'll have to supply <code>d</code>.  Also, for completeness:</p>
<pre><code>num = len(S)
