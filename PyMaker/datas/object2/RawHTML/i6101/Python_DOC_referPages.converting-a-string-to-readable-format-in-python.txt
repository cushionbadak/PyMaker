<p>From My understanding <code>\xe1\x984a\x82@Z\xb4\x85\xd0</code> is look like bytes stream. </p>
<p>so you can convert in readable format like this.</p>
<pre><code>&gt;&gt;&gt; b'\xe1\x984a\x82@Z\xb4\x85\xd0'.decode('utf-8','ignore')
'4a@Z'
</code></pre>
<p>you byte stream string also give the output in integer format like this.</p>
<pre><code>&gt;&gt;&gt; int.from_bytes(b'\xe1\x984a\x82@Z\xb4\x85\xd0', byteorder='big', signed=True)
-143585681428579525294640
</code></pre>
<p>for more you can visit <a href="https://docs.python.org/3/library/stdtypes.html" rel="nofollow noreferrer">here</a> and <a href="https://docs.python.org/3/howto/unicode.html#python-s-unicode-support" rel="nofollow noreferrer">here</a>.</p>
