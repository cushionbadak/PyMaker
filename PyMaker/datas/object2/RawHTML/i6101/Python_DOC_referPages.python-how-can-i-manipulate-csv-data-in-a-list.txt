<p>I have created a list which has the totality of all the data in the csv file.<br/>
How do I seperately call upon data in rows and columns?</p>
<p>For instance:</p>
<pre><code>   **a,  b  ,c** 
**1**  a1  b1  c1
**2**  a2  b2  c2
</code></pre>
<p>How can I identify a single cell within the list?</p>

<div class="post-text" itemprop="text">
<p>try below code:</p>
<pre><code>l = ['a', 'b', 'c','1','a1', 'b1', 'c1', '2', 'a2', 'b2','c2']
columns = 3
result = list(zip(*[iter(l[columns:])]*(columns+1)))
result2 = {i[0]:i[1:] for i in result}
item_id = '2'
result2[item_id]
</code></pre>
<p>output:</p>
<pre><code> ('a2', 'b2', 'c2')
</code></pre>
<p>or you could try below code:</p>
<pre><code>l = ['a', 'b', 'c','1','a1', 'b1', 'c1', '2', 'a2', 'b2','c2']
columns = 3
item_id = '2'
index = l.index(item_id)
l[index:index+columns]
</code></pre>
<p>output:</p>
<pre><code>['a2', 'b2', 'c2']
</code></pre>
