<p>I get a string from a socket that contains a float array as bytes. Now I want to convert this string in to a float (4bit) numpy array.</p>
<p>for example I get:</p>
<pre><code>"\x00\x00\x00\x00\x00\x00\x00\x00"
</code></pre>
<p>and I want an output like this:</p>
<pre><code>[0, 0]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>fromstring</code> method and specify the number of bytes to convert to (<code>32 bits = 4 bytes</code> in this particular case):</p>
<pre><code>np.fromstring( "\x00\x00\x00\x00\x00\x00\x00\x00", np.float32 )
</code></pre>
<p>You need to ensure your <code>string</code> has lenght multiple of <code>4</code>. Otherwise <code>numpy</code> will rise:</p>
<pre><code>ValueError: string size must be a multiple of element size
</code></pre>
</div>
