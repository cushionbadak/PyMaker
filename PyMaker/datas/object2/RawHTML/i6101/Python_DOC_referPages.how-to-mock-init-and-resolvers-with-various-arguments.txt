<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/unittest.mock.html" rel="nofollow noreferrer">unittest.mock</a> python package. Here is an example:</p>
<pre><code>from unittest import mock

info = mock.create_autospec(graphql.execution.base.ResolveInfo)
info.context.user = AnonymousUser()
</code></pre>
