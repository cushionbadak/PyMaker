<p>For example a variable is integer 00000550000066000007700000880000099 and so on. There are five zeroes between desired numbers which are in this case 55,66,77,88. How do separate these numbers in a different list?</p>
</div>
<div class="post-text" itemprop="text">
<p>Can be achieved with using <code>split('0')</code> . First convert your int to str</p>
<pre><code>&gt;&gt;&gt; x = '00000550000066000007700000880000099'
&gt;&gt;&gt; [int(i) for i in x.split('0') if i ]    
&gt;&gt;&gt; [55, 66, 77, 88, 99]
</code></pre>
