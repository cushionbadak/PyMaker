<p>This should be similar to image resealing, suppose having this vector,</p>
<pre><code>x = np.array([10,50,40,20])
</code></pre>
<p>I need to scale it up to size 10, and fill the missing values using interpolation. </p>
<p>Any <code>numpy</code> or <code>scipy</code> function that does this?</p>
</div>
<div class="post-text" itemprop="text">
<p>One way is using <a href="https://docs.scipy.org/doc/scipy-0.15.1/reference/generated/scipy.ndimage.interpolation.zoom.html" rel="nofollow noreferrer"><code>scipy.ndimage.interpolation.zoom</code></a>. This will be <code>zooming</code> using <a href="https://en.wikipedia.org/wiki/Spline_interpolation" rel="nofollow noreferrer">spline interpolation</a>. In order to use it you need to provide a zooming factor which in this case, given that you want an array of size <code>10</code>, it should be <code>10/len(x)</code>:</p>
<pre><code>from scipy.ndimage import interpolation

x = np.array([10,50,40,20])
i = 10
z = i / len(x)
# 2.5

x_int = interpolation.zoom(x,z)
</code></pre>
<hr/>
<p><b> Output </b></p>
<pre><code>array([10, 18, 35, 50, 54, 49, 40, 30, 23, 20])
</code></pre>
