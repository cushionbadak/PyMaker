<p>I have a <code>.txt</code> file with 100 50-digit numbers in it. Each number is on a different line.</p>
<p>Example of the file:</p>
<pre><code>37107287533902102798797998220837590246510135740250
46376937677490009712648124896970078050417018260538

91942213363574161572522430563301811072406154908250
23067588207539346171171980310421047513778063246676
#95 more numbers
</code></pre>
<p>I want to be able to append each number into a list, after opening the file. How can I do this?</p>
<p>I know how to open the file: <code>fo = open('file_name', 'r')</code>. And how to close it at the end: <code>fo.close()</code>.</p>
<p>Thanks in advance!</p>

<div class="post-text" itemprop="text">
<p>Here's the simplest way I can think of:</p>
<pre><code>with open('numbers.txt') as file:
    lst = [line.strip() for line in file]
</code></pre>
