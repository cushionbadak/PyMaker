<div class="post-text" itemprop="text">
<p>The <code>PROT_READ</code> and <code>PROT_WRITE</code> are Unix-specific. You're likely looking for:</p>
<pre><code>mmap.mmap(f.fileno(), 0, access=mmap.ACCESS_READ)
</code></pre>
<p>The <a href="http://docs.python.org/3/library/mmap.html"><code>mmap</code> page</a> actually has different entries for Unix/Windows version.</p>
</div>
