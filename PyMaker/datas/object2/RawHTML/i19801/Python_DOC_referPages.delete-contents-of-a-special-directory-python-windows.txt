<p>You could access the userâ€™s documents directory like this:</p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; documents = os.path.join(os.environ['USERPROFILE'], 'Documents')
&gt;&gt;&gt; documents
'C:\\Users\\poke\\Documents'
&gt;&gt;&gt; directories = [os.path.join(documents, 'VideoEditor{0}\\JunkFiles'.format(y)) for y in ('09', '10', '11')]
&gt;&gt;&gt; directories
['C:\\Users\\poke\\Documents\\VideoEditor09\\JunkFiles', 'C:\\Users\\poke\\Documents\\VideoEditor10\\JunkFiles', 'C:\\Users\\poke\\Documents\\VideoEditor11\\JunkFiles']
</code></pre>
<p>You can then just loop through the files of the directories to delete them, or if deleting the folders is okay as well (you could create them afterwards again), you could also just use <a href="http://docs.python.org/3/library/shutil.html#shutil.rmtree" rel="nofollow"><code>shutil.rmtree</code></a>:</p>
<pre><code>&gt;&gt;&gt; import shutil
&gt;&gt;&gt; for directory in directories:
        shutil.rmtree(directory)
</code></pre>

<p>What you could try is the following:</p>
<pre><code>import os
user = 'some_user_name'
videoEditors = ['VideoEditor09','VideoEditor10','VideoEditor11']
for i in videoEditors:
    os.chdir('C:\Users\%s\Documents\%s\JunkFiles'%(user,i))
    files = os.listdir('.')
    for file in files:
        os.remove(file)
</code></pre>
<p>The main thing I used here was the <a href="http://docs.python.org/2/library/os.html" rel="nofollow">Python os package</a> and a list of the desired folders to browse.  Also the '%s' is part of Python <a href="http://docs.python.org/2/library/stdtypes.html#string-formatting" rel="nofollow">string formatting</a>.</p>
<p>Hope this helps!</p>
