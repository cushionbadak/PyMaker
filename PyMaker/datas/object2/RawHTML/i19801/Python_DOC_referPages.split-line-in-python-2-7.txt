<div class="post-text" itemprop="text">
<p><a href="http://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>split</code></a> is a method of the string itself, and you can access elements of a list with <code>[42]</code>, not the method call <code>(42)</code><a href="http://docs.python.org/3/tutorial/introduction.html#lists" rel="nofollow"><sup>doc</sup></a>. Try:</p>
<pre><code>s = 'W03*17*65.68*KG*0.2891*CR*1*1N'
lst = s.split('*')
qty = lst[1]
weight = lst[2]
weight_unit = lst[3]
</code></pre>
<p>You may also be interested in tuple unpacking:</p>
<pre><code>s = 'W03*17*65.68*KG*0.2891*CR*1*1N'
_,qty,weight,weight_unit,_,_,_,_ = s.split('*')
</code></pre>
<p>You can even use a <a href="http://docs.python.org/3/tutorial/introduction.html#strings" rel="nofollow">slice</a>:</p>
<pre><code>s = 'W03*17*65.68*KG*0.2891*CR*1*1N'
qty,weight,weight_unit = s.split('*')[1:4]
</code></pre>

<p>You need to invoke <code>split</code> method on a certain string to split it. Just using <code>Split(my_str, "x")</code> won't work: -</p>
<pre><code>&gt;&gt;&gt; my_str = "Python W03*17*65.68*KG*0.2891*CR*1*1N"
&gt;&gt;&gt; tokens = my_str.split('*')
&gt;&gt;&gt; tokens
['Python W03', '17', '65.68', 'KG', '0.2891', 'CR', '1', '1N']
&gt;&gt;&gt; tokens[1]
'17'
&gt;&gt;&gt; tokens[2]
'65.68'
</code></pre>

<div class="post-text" itemprop="text">
<p>If you'd like to capture Value qty as 17 Value kg as 65.68, 
one way to solve it is using dictionary after splitting strings.</p>
<pre><code>&gt;&gt;&gt; s = 'W03*17*65.68*KG*0.2891*CR*1*1N'
&gt;&gt;&gt; s.split('*')
['W03', '17', '65.68', 'KG', '0.2891', 'CR', '1', '1N']
&gt;&gt;&gt; t = s.split('*')
&gt;&gt;&gt; dict(qty=t[1],kg=t[2])
{'kg': '65.68', 'qty': '17'}
</code></pre>
<p>Hope it helps.</p>
