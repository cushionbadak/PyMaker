<div class="post-text" itemprop="text">
<p>By default, only functions whose name that start with <code>test</code> are run:</p>
<pre><code>class Prueba(unittest.TestCase):

    def setUp(self):
        pass
    def testPrintsTrue(self):
        self.assertTrue(True)
</code></pre>
<p>From the <a href="http://docs.python.org/2/library/unittest.html#basic-example">unittest basic example</a>:</p>
<blockquote>
<p>A testcase is created by subclassing <code>unittest.TestCase</code>. The three individual tests are defined with methods whose names start with the letters <code>test</code>. This naming convention informs the test runner about which methods represent tests.</p>
</blockquote>
