<div class="post-text" itemprop="text">
<p>Your indentations are wrong. - If you're going to put the <code>@app.route</code> decorated function as a virtual function - nested function; then that code will execute once the <code>login</code> function is called. </p>
<p>Hence you don't have any registered routes. - Bring your routes out of the function. </p>
<p>As a side note and suggestion; please, do follow <a href="https://www.python.org/dev/peps/pep-0008/" rel="nofollow noreferrer">PEP8 - Coding Style Guide</a>.</p>
