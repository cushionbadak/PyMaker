<p>As <a href="https://stackoverflow.com/users/8204776/meowgoesthedog">@meowgoesthedog</a> suggested a simple loop is your answer.</p>
<pre><code>import string
buttons = {}

for letter in string.ascii_uppercase:
    buttonb = tkinter.Button(bg = "blue", text = letter, width = 2,command = ex)
    buttonb.pack(side = "left")
    buttons[letter] = buttonb
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a loop to iterate over the letters and since the buttons belong together they can be stored in a data structure like a list.</p>
<pre><code>buttons = []
for letter in 'ABCDEFGHIJKLMNOPQRSTUVWXYZ':
    button = tkinter.Button(bg="blue", text = letter, width=2,command=ex)
    button.pack(side="left")
    buttons.append(button)
</code></pre>
<p>Feel free to use a list comprehension if you like.</p>
</div>
