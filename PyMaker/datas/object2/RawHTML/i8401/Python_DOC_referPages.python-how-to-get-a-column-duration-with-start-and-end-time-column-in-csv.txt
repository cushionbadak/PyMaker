<p>I have a problem in manipulating csv file.
In my csv file, there is a column 'Start Time' and 'Complete Time' and I want to get a column 'Duration' using it.
When I tried <code>duration = row[2] - row[1]</code>, it showed me an error message</p>
<p><code>'unsupported operand type(s) for -: 'str' and 'str'</code>.</p>

<div class="post-text" itemprop="text">
<p>Since the data you obtained from csv file is of str type, you can not do the operation:</p>
<pre><code>duration = row[2] - row[1]
</code></pre>
<p>I would advice you to first convert row[2] and row[1] content, and then do the operation.</p>
<pre><code>import time
complete_time = time.strptime(row[2])
start_time = time.strptime(row[1])
duration = complete_time - start_time
</code></pre>
