<p>Is there a way, using pandas, to construct a <code>Timestamp</code> based on 3 input: </p>
<ul>
<li>the year (ex: 2010)</li>
<li>the week of the year (ex: 3 : the 3rd week)</li>
<li>the day of the week: (for instance Monday)</li>
</ul>
<p>In that particular example, the result would be <code>Timestamp('2010-01-18')</code></p>
<p>Anyone have a hint of a clever way to achieve this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can generate a date using the logic you've provided like this:</p>
<pre><code>&gt;&gt;&gt; datetime.datetime.strptime("2010-3-Monday", "%Y-%W-%A")
datetime.datetime(2010, 1, 18, 0, 0)
</code></pre>
<p>You may need to adjust your format, depending on what, exactly, you are passing to <code>strptime</code>, but you can find the appropriate formatting options <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="noreferrer">here</a></p>
</div>
<span class="comment-copy">Try df['DateTime'] = df[['Year', 'Month', 'Day']].apply(lambda s : datetime.datetime(*s),axis = 1)</span>
