<div class="post-text" itemprop="text">
<p>You can replace elements from your template by looping over the file list, with list wrapping. Here's an example for aaa.html using aaa,bbb,ccc:</p>
<pre><code>#f = ['aaa.html','bbb.html','ccc.html']
f = sorted(['aaa.html','bbb.html','ccc.html'])  # explicit sorting

t = """&lt;nav&gt;
    &lt;ul class="pager"&gt;
        &lt;li class="previous"&gt;&lt;a href="#"&gt;Previous&lt;/a&gt;&lt;/li&gt;
        &lt;li class="next"&gt;&lt;a href="#"&gt;Next&lt;/a&gt;&lt;/li&gt;
    &lt;/ul&gt;
&lt;/nav&gt;"""  # sample aaa.html file

for i in xrange(len(f)-1):
    #print f[i]
    t = t.replace('&lt;li class="previous"&gt;&lt;a href="#"&gt;Previous','&lt;li class="previous"&gt;&lt;a href="'+f[(i % len(f)) -1]+'"&gt;Previous')
    t = t.replace('&lt;li class="next"&gt;&lt;a href="#"&gt;Next','&lt;li class="next"&gt;&lt;a href="'+f[(i % len(f)) +1]+'"&gt;Next')

print t
</code></pre>
<p>To do the list-wrapping I use this <a href="https://stackoverflow.com/a/22122635/2327328">concept</a> (After zzz comes aaa)</p>
<p>Gives as an output for aaa.html:</p>
<pre><code>&lt;nav&gt;
    &lt;ul class="pager"&gt;
        &lt;li class="previous"&gt;&lt;a href="ccc.html"&gt;Previous&lt;/a&gt;&lt;/li&gt;
        &lt;li class="next"&gt;&lt;a href="bbb.html"&gt;Next&lt;/a&gt;&lt;/li&gt;
    &lt;/ul&gt;
&lt;/nav&gt;
</code></pre>
<p>To complete the code, you'd have to loop over *.html files (see <a href="https://stackoverflow.com/a/3964691/2327328">glob.glob</a>)</p>

<p>You can use the <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/#beautifulsoup" rel="nofollow noreferrer">beautifulsoup library</a> to modify html:</p>
<pre><code>from bs4 import BeautifulSoup

file_names = ['bbb.html', 'ccc.html', ... , 'yyy.html']
# we exclude first and last files (not sure what to do with them ?)

for ind, file_name in enumerate(file_names):
    with open(file_name, 'r+') as f:
        soup = BeautifulSoup(f.read(), 'html.parser')
        # we suppose that there is only one link for previous and next
        soup.find_all(class_='previous')[0]['href'] = file_names[ind - 1]
        soup.find_all(class_='next')[0]['href'] = file_names[ind + 1]
        # erase contents and replace with new html
        f.seek(0)
        f.truncate()
        f.write(soup.prettify("utf-8"))  # to get readable HTML
</code></pre>
<p>If the filenames aren't as consistent as in your example, and you want to generate the list from the files in the directory, you can use <a href="https://docs.python.org/3/library/os.html#os.walk" rel="nofollow noreferrer">os.walk</a> or <a href="https://docs.python.org/3/library/glob.html#glob.glob" rel="nofollow noreferrer">glob.glob</a>.</p>
