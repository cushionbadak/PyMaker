<div class="post-text" itemprop="text">
<p>I recommend bs4, but if you want to use regular expressions :   </p>
<pre><code>my_str = '&lt;td class="standing-table__cell standing-table__cell--name"&gt;Chelsea&lt;/td&gt;'  
match = re.search('&gt;(.*?)&lt;/', my_str)  
if match : 
    match = match.group(1)
</code></pre>
<p>This will match any string between html tags  </p>
<pre><code>print(match)
Chelsea
</code></pre>
<p>Keep in mind that <code>search</code> will return the first match ( if any , else none )<br/>
If you want all occurrences , you should use <code>findall</code> instead</p>
