<p>When calling a dictionary that may or may not have a given key you can set a default like this:</p>
<pre><code>&gt;&gt;&gt; my_dict = {'color':'red', 'size':'2'}
&gt;&gt;&gt; my_dict.setdefault('style', 'round')
'round'

&gt;&gt;&gt; my_dict = {'color':'red', 'size':'2', 'style':'square'}
&gt;&gt;&gt; my_dict.setdefault('style', 'round')
'square'
</code></pre>
<p>regarding your edited question:</p>
<blockquote>
<p>Edit2:</p>
</blockquote>
<pre><code>spouse={John:Joan, Bob:Marry}
</code></pre>
<blockquote>
<p>when I ask for spouse[Dan] I should get "not married" same should go
  for any male name that comes to user's mind and it is not a key in the
  dictionary</p>
</blockquote>
<p>You could accomplish that like this:</p>
<pre><code>&gt;&gt;&gt; spouse = {}
&gt;&gt;&gt; spouse['Jim']='Lucy'
&gt;&gt;&gt; spouse['Alex']='Sandra'
&gt;&gt;&gt; users = ['Dan', 'Phil','Jim','Alex']
&gt;&gt;&gt; for i in range(len(users)):
&gt;&gt;&gt;     s = spouse.setdefault(users[i], 0)
&gt;&gt;&gt;     if s == 0:
&gt;&gt;&gt;         print "%s is not married" % users[i]
&gt;&gt;&gt;     else:
&gt;&gt;&gt;         print "%s's spouse is %s" % (users[i],s)

'Dan is not married'
'Phil is not married'
'Jim's spouse is Lucy'
'Alex's spouse is Sandra'
</code></pre>
