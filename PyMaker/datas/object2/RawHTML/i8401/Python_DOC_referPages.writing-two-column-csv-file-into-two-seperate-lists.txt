<div class="post-text" itemprop="text">
<p>I have a csv file that has two columns in it: one for names and the other for emails. I have been looking for a way to take the csv file and read it into two lists, one for each column, to later be processed into a database. Could someone please be of assistance? </p>
<p>If it helps the file is formated like:</p>
<pre><code>email,name
email,name
etc. . . .
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can just read the file and convert it to <code>list</code>, and then split the line with comma:</p>
<p>csv:</p>
<pre><code>email,name
1@gmail.com,joe
2@gmail.com,mike
</code></pre>
<hr/>
<pre><code>name,email=[],[]
for i in list(open("ooo.csv")):
    name.append(i.strip('\n').split(',')[1])
    email.append(i.strip('\n').split(',')[0])
</code></pre>
<p>Also you can use <code>csv.DictReader()</code>:</p>
<p>e.g.</p>
<pre><code>from csv import DictReader

name,email=[],[]
with open("ooo.csv") as f:
    for row in  DictReader(f):
        name.append(row["name"])
        email.append(row["email"])
</code></pre>
<p>Result:</p>
<pre><code>['joe', 'mike']
['1@gmail.com', '2@gmail.com']
</code></pre>
<p>Or <code>pandas.read_csv()</code> method</p>
<pre><code>import pandas
colnames = ['name', 'email']
data = pandas.read_csv('ooo.csv', names=colnames)

print(data.name.tolist())
print(data.email.tolist())
</code></pre>
<p>Output:</p>
<pre><code>['email', '1@gmail.com', '2@gmail.com']
['name', 'joe', 'mike']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>file:</p>
<pre><code>a,b
a,b
a,b
</code></pre>
<p>A <a href="https://docs.python.org/3/library/csv.html#csv.reader" rel="nofollow noreferrer">csv.reader</a> object will produce <code>['a', 'b']</code> for each line/row of the file. You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip</a> to transpose the result:</p>
<pre><code>import csv
with open(file) as f:
    reader = csv.reader(f)
    email, name = zip(*reader)
</code></pre>
<hr/>
<pre><code>&gt;&gt;&gt; email
('a', 'a', 'a')
&gt;&gt;&gt; name
('b', 'b', 'b')
&gt;&gt;&gt;
</code></pre>
<hr/>
<p>You need to ensure that you have as many names on the left hand side of the assignment as there are columns in the csv file - or you could use <a href="https://www.python.org/dev/peps/pep-3132/" rel="nofollow noreferrer">extended unpacking</a> to handle excess columns.</p>
