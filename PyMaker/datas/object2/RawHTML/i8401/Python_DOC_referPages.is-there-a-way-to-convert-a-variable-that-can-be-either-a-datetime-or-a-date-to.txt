<div class="post-text" itemprop="text">
<p>I have a function that can accept a date or a datetime as one of its arguments.  If the value is a datetime, I would like to convert it to its <code>date</code> components.  What is the pythonic way of doing that?</p>
</div>
<div class="post-text" itemprop="text">
<p>Since <code>datetime</code> objects <strong>are</strong> <code>date</code> objects, you should be able to call any <code>date</code> functions on <code>datetime</code> and have it work.</p>
<p>But if you must:</p>
<pre><code>if isinstance(param_date, datetime.datetime): 
    param_date = param_date.date()
</code></pre>
<p>will normallize any <code>datetime</code> and <code>date</code> objects to <code>date</code> objects... now you only have to deal with them, as <code>date</code> objects are not <code>datetime</code> objects:</p>
<pre><code>&gt;&gt;&gt; isinstance(datetime.date.today(), datetime.datetime)
False
</code></pre>
<p><strong>NOTE</strong>: the reverse is not applicable:</p>
<pre><code>&gt;&gt;&gt; isinstance(datetime.datetime.now(), datetime.date)
True
</code></pre>
<p>as <code>datetime</code> inherits from <code>date</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could first check if it is of type <code>datetime.datetime</code> and then go on to convert it to its <code>date</code> component using <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.date" rel="nofollow noreferrer"><code>datetime.datetime.date()</code></a>.</p>
<p>An example of this in code would be as follows.</p>
<p>Importing the <code>datetime</code> module, and setting a variable <code>d</code> to be the current <code>datetime</code>.</p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; d = datetime.datetime.now()
&gt;&gt;&gt; d
datetime.datetime(2017, 4, 7, 22, 10, 51, 89213)
</code></pre>
<p>Then checking if <code>d</code> is a <code>datetime</code> object, and if it is set it to be its <code>date</code> counterpart.</p>
<pre><code>&gt;&gt;&gt; if type(d) is datetime.datetime:
&gt;&gt;&gt;     d = d.date()
&gt;&gt;&gt; d
datetime.date(2017, 4, 7)
</code></pre>
