<div class="post-text" itemprop="text">
<p>number in the format string should starts from 0, not from 1.</p>
<pre><code>&gt;&gt;&gt; '{1}'.format(1)
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
IndexError: tuple index out of range
&gt;&gt;&gt; '{0}'.format(1)
'1'
&gt;&gt;&gt; '{}'.format(1)  # auto numbering
'1'
</code></pre>
<p>See <a href="https://docs.python.org/3/library/string.html#formatstrings" rel="nofollow noreferrer">Format string syntax</a>.</p>
<p>If you're using python 2.7+, you can omit the number completely. (auto numbering). And the second format can be just <code>b</code> (no need to use format at all, because <code>b</code> is already a string object)</p>
<pre><code>sftp.get('{}/{}'.format(pwd1, b), b)
</code></pre>
</div>
