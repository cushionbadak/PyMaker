<p>I am looking for a pythonic way of parsing a datetime object into an RFC 1132 compliant string:</p>
<p>Assuming (GMT)</p>
<p>So: 2008-10-22T10:52:40Z into Wed, 22 Oct 2008 10:52:40 GMT</p>
<p>I am sure there is a very simple way but i haven't been able to find it.</p>
<p>Thanks</p>

<div class="post-text" itemprop="text">
<p>I could advise something like this:</p>
<pre><code>from datetime import datetime
import pytz

date_str = '2008-10-22T10:52:40Z'

mytime = datetime.strptime(date_str, "%Y-%m-%dT%H:%M:%SZ")
mytime = mytime.replace(tzinfo=pytz.timezone('GMT'))
print(mytime.strftime("%a, %d %b %Y %H:%M:%S %Z"))
# Output: Wed, 22 Oct 2008 10:52:40 GMT
</code></pre>
<p>List of formatting directives for <code>strftime</code> you could find <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer">here</a>.</p>
