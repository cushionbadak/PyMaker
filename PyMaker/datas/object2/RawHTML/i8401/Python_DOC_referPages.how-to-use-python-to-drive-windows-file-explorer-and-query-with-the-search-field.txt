<div class="post-text" itemprop="text">
<p>For an automated data review project, a method is required to open a specific local or network directory in Windows File Explorer and search for image file names with a specific string.  In this case, the requirement is only display specific images (out of many) in File Explorer for a quick visual verification.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>import subprocess

query_string = 'file_name.png'
local_path = r'C:\Users\your_name\Pictures' # r is raw for dealing with backslashes
network_path = r'\\your\network\fold\path'

# for a network location
subprocess.Popen(f'explorer /root,"search-ms:query={query}&amp;crumb=location:{network_path}&amp;"')

#for a local folder
subprocess.Popen(f'explorer /root,"search-ms:query={query}&amp;crumb=folder:{local_path}&amp;"')
</code></pre>
<ol>
<li>subprocess.Popen is from the Python standard library <a href="https://docs.python.org/3/library/subprocess.html" rel="nofollow noreferrer">Subprocess management</a>.</li>
<li>search-ms:parameter=value[&amp;parameter=value]&amp; is from MSDN <a href="https://msdn.microsoft.com/en-us/library/windows/desktop/ff684385(v=vs.85).aspx" rel="nofollow noreferrer">Getting started with parameter-value arguments</a>.  Parameter-value arguments can be configured in a variety ways not exclusive to the way shown here.  For example, folder will only located local folders, but location will work for network and local folders.</li>
<li>f'some_string {variable}' is from PEP498: Formatted String Literals.</li>
<li>explorer /root are Windows commands.</li>
</ol>
