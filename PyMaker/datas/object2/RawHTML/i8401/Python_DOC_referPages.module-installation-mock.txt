<div class="post-text" itemprop="text">
<p>You are using Python 3.5; mock is already <a href="https://docs.python.org/3/library/unittest.mock.html" rel="nofollow noreferrer">part of the standard library</a> in that version, and you don't need to install it. You should import it like this:</p>
<pre><code>from unittest.mock import Mock
</code></pre>
