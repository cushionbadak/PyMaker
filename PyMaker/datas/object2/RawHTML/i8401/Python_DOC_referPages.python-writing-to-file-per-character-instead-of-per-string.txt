<p>The problem lies here - <code>f2.write('\n'.join('%s %s' % (k,v)))</code>.<br/>
The join method expects an iterable (like a list) in the parameters. When you give it the string 'String1 String2 Float', it 'joins' every two letters with a new line.<br/>
It seems all that you want to do is write every entry in a new line (not every letter!), so replace the line with something like <code>f2.write('{} {}\n'.format(k, v))</code><br/>
If you haven't come across <code>format</code> yet, read up about it <a href="https://pyformat.info/" rel="nofollow noreferrer">here</a>.</p>
