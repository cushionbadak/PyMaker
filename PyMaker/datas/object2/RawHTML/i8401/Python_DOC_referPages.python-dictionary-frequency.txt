<p><a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>Counter</code></a> from the <code>collections</code> module is the way to go on this:</p>
<p><strong>Code:</strong></p>
<pre><code>from collections import Counter
from string import ascii_lowercase 

def make_letter_pairs(text):
    return Counter([t for t in [text[i:i+2] for i in range(len(text) - 1)]
                    if t[0] in ascii_lowercase and t[1] in ascii_lowercase])

test = 'how now, brown cow, ok?'
print(make_letter_pairs(test))
</code></pre>
