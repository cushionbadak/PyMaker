<p>I have a dataframe and the first column contains id. How do I sort the first column when it contains alphanumeric data, such as:</p>
<pre><code>id = ["6LDFTLL9", "N9RFERBG", "6RHSDD46", "6UVSCF4H", "7SKDEZWE", "5566FT6N","6VPZ4T5P", "EHYXE34N", "6P4EF7BB", "TT56GTN2", "6YYPH399" ]
</code></pre>
<p>Expected result is</p>
<pre><code>id = ["5566FT6N", "6LDFTLL9", "6P4EF7BB", "6RHSDD46", "6UVSCF4H", "6VPZ4T5P", "6YYPH399", "7SKDEZWE", "EHYXE34N", "N9RFERBG", "TT56GTN2" ] 
</code></pre>

<div class="post-text" itemprop="text">
<p>You can utilize the <a href="https://docs.python.org/3/library/stdtypes.html#list.sort" rel="nofollow noreferrer"><code>.sort()</code></a> method:</p>
<pre><code>&gt;&gt;&gt; id.sort()
['5566FT6N', '6LDFTLL9', '6P4EF7BB', '6RHSDD46', '6UVSCF4H', '6VPZ4T5P', '6YYPH399', '7SKDEZWE', 'EHYXE34N', 'N9RFERBG', 'TT56GTN2']
</code></pre>
<p>This will sort the list in place. If you don't want to change the original <code>id</code> list, you can utilize the <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow noreferrer"><code>sorted()</code></a> method</p>
<pre><code>&gt;&gt;&gt; sorted(id)
['5566FT6N', '6LDFTLL9', '6P4EF7BB', '6RHSDD46', '6UVSCF4H', '6VPZ4T5P', '6YYPH399', '7SKDEZWE', 'EHYXE34N', 'N9RFERBG', 'TT56GTN2']
&gt;&gt;&gt; id
['6LDFTLL9', 'N9RFERBG', '6RHSDD46', '6UVSCF4H', '7SKDEZWE', '5566FT6N', '6VPZ4T5P', 'EHYXE34N', '6P4EF7BB', 'TT56GTN2', '6YYPH399']
</code></pre>
<p>Notice, with this one, that <code>id</code> is unchanged.</p>
<hr/>
<p>For a DataFrame, you want to use <a href="http://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.sort_values.html#pandas.DataFrame.sort_values" rel="nofollow noreferrer"><code>sort_values()</code></a>.</p>
<pre><code>df.sort_values(0, inplace=True)
</code></pre>
<p><code>0</code> is either the numerical index of your column or you can pass the column name (eg. <code>id</code>)</p>
<pre><code>           0

</code></pre>
