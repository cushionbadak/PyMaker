<div class="post-text" itemprop="text">
<p>You can also use <code>re.match()</code> plus the beginning-of-string and end-of-string markers:</p>
<pre><code>user_input = input('&gt;&gt;&gt;: ')
if not re.match('^[0-9]+$', user_input):
    print('That wasn\'t an integer.')
else:
    print('That was an integer!')
</code></pre>
<p>Remember that, in regex:</p>
<ul>
<li><code>^</code> Matches the start of a line;</li>
<li><code>$</code> Matches the end of a line;</li>
<li><code>+</code> Indicates that the previous pattern should appear one or more times.</li>
</ul>
<p>I would recommend taking a look at <a href="https://docs.python.org/2/library/re.html" rel="nofollow noreferrer">Python official 're' documentation</a> for a more complete and well explained list.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you aren't tied to the idea of using a regex for this, I would suggest simply using <a href="https://docs.python.org/3/library/stdtypes.html#str.isdigit" rel="nofollow noreferrer"><code>str.isdigit</code></a></p>
<pre><code>user_input = input('&gt;&gt;&gt;: ')
if not user_input.isdigit():
    print('That wasn\'t an integer.')
else:
    print('That was an integer!')
</code></pre>
<p>There's also this approach</p>
<pre><code>try:
    user_input = int(input('&gt;&gt;&gt;: '))
    print('That was an integer!')
except ValueError:
    print('That wasn\'t an integer.')
</code></pre>
