<div class="post-text" itemprop="text">
<p>You do not have to import <code>inf</code> from math module. The <code>math.isinf</code> function exists but for defining an infinite value you can use <code>float('inf')</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>As mentioned, <code>math.inf</code> is new with python 3.5. <code>inf</code> has always been around as <code>float('inf')</code> and <code>math.inf</code> is really just a convenience. If you like using <code>math.inf</code>, you can restrict your code to supported versions</p>
<pre><code>import sys
if sys.version_info &lt; (3,5):
    raise ImportError('{} requires python version 3.5 or later'.format(__name__))
</code></pre>
<p>or you can just add it yourself</p>
<pre><code>import sys
import math
if sys.version_info &lt; (3,5):
    math.inf = float('inf')
    math.nan = float('nan')
    math.tau = math.pi * 2. 
</code></pre>
</div>
