<p>I would put the JSON data into something called <code>json_string</code> or <code>json_data</code> instead of <code>json</code> so you don't obscure/confuse the name <code>json</code>. </p>
<p>I'm assuming that the data that you show is in string format in your <code>json</code> variable. I copied your data into a variable as multi-line text, but I had to change the <code>backslash-double quote</code> to a single quote in the string. I don't know if you'll need to do that to get json.loads() to work on your data, but if so, then when you need to extract the HTML, you can convert the single quotes back to double quotes. Then I fed it into the json package which makes a list of dictionaries. Then you can extract the titles from that using a list comprehension.</p>
<pre><code>import json

json_string = """[
  {
    "title": "scname",
    "html": "&lt;div&gt;&lt;iframe src=/scenario/test3dxblock.0/ width='400' height='500'&gt;&lt;/iframe&gt;&lt;/div&gt;",
    "description": "desc",
    "status": "417"
},
{
    "title": "test3dxblock.0",
    "html": "&lt;div&gt;&lt;iframe src=/scenario/test3dxblock.0/ width='400' height='500'&gt;&lt;/iframe&gt;&lt;/div&gt;",
    "description": "desc",
    "status": "417"
},
{
    "title": "filethumbs.0",
    "html": "&lt;div&gt;&lt;iframe src=/scenario/filethumbs.0/ width='400' height='500'&gt;&lt;/iframe&gt;&lt;/div&gt;",
    "description": "desc",
    "status": "417"
}
]"""

json_data = json.loads(json_string)
titles = [item['title'] for item in json_data]
</code></pre>
<p>When that is pasted into the REPL, titles contains:</p>
<pre><code>['scname', 'test3dxblock.0', 'filethumbs.0']
</code></pre>
<p>If the variable you show a <code>json</code> is actually a list already, then you just need to use the list comprehension to get the titles. If you have a new <code>title</code> that you want to see if it is in the list <code>titles</code>, you can just say:</p>
<pre><code>if title in titles:
    # do whatever with it
else:
    # add it to list - or whatever
</code></pre>
