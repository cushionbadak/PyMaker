<div class="post-text" itemprop="text">
<p>You can use <code>system()</code> call to execute your python scripts, but there is no need of specifying the version of python executable on the command line.</p>
<p>You can use shebang.</p>
<p>Use <code>#!/usr/bin/python3</code> as the very first line for scripts that you want to run with <code>python3</code> and <code>#!/usr/bin/python2</code> in scripts that you want to run with python2.</p>
<p>If you are working on windows then please have a look at <a href="https://docs.python.org/3/using/windows.html#launcher" rel="nofollow noreferrer">python launcher</a></p>
