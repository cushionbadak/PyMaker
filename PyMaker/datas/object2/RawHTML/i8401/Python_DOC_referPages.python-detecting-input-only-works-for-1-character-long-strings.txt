<div class="post-text" itemprop="text">
<p><code>is</code> is an identity operator. It is used to check if two values (or variables) are located on the same part of the memory. Two variables that are equal does not imply that they are identical. To check if <code>inpt</code> contains <code>'&gt;&gt;&gt;'</code> string use either <code>in</code> or <code>==</code> (if you want to be more specific) operator.</p>
<pre><code>'quit' == inpt
</code></pre>

<span class="comment-copy">You need to use <code>==</code> not <code>is</code>. <code>is</code> checks for identity, <code>==</code> checks for equality. It happens to work for <code>'t'</code> because Python interns small string literals.</span>
