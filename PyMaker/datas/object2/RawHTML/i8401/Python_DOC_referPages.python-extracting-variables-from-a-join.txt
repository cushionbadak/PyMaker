<div class="post-text" itemprop="text">
<p>Tuples are immutable. <code>split()</code> is for <code>str</code> types.</p>
<p>This will return the b and c separately:</p>
<pre><code>def extract_lasttwo_cols(three_cols):
    b, c = three_cols[1][0], three_cols[1][1]
    return b, c
</code></pre>
