<p>If some of the columns only have 1 space between them and yet <code>+ 0.</code> must be recognized as a single value, then you could use the regex pattern <code>(?&lt;![+-])\s+</code>. This regex matches 1-or-more spaces <em>not preceeded by</em> a plus or minus sign. Thus a single space between numbers can be recognized as a separator, but the space between <code>+</code> and <code>0.</code> would not be considered a separator because this space is preceeded by a plus sign.</p>
<p>If the columns are separated by tabs, then you could specify the column separator by the regex pattern <code>\t+</code>, which matches 1-or-more tabs. The snippet you posted does not appear to use tabs as separators, but I thought I'd mention this just in case the tabs somehow got converted to spaces during the pasting process.</p>
