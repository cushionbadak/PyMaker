<p>I have this sample json data in a list:</p>
<pre><code>data = [ 
  { "Name": "John_Doe", "Age" : 25 } 
  { "Name": "Jane_Roe", "Age" : 26 }
]
</code></pre>
<p>I need a way to extract the all the key value pairs from an element in the list based on the 'Name' Key. If my variable = 'John_Doe', then the script should only return the values related to John_Doe, i, e only the following values :</p>
<pre><code>{ "Name": "John_Doe", "Age" : 25 }
</code></pre>

<div class="post-text" itemprop="text">
<p>Just extract all the dictionaries with the value <code>"John_Doe"</code> associated with the key <code>"Name"</code>:</p>
<pre><code>print([d for d in data if d['Name'] == "John_Doe"])
# [{ "Name": "John_Doe", "Age" : 25 }]
</code></pre>
<p>Or with <a href="https://docs.python.org/3/library/functions.html#filter" rel="nofollow noreferrer"><code>filter()</code></a>:</p>
<pre><code>print(list(filter(lambda x : x['Name'] == "John_Doe", data)))
# [{ "Name": "John_Doe", "Age" : 25 }]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If all you need is the dict with a <code>Name</code> element matching <code>John_Doe</code>, then:</p>
<pre><code>matches = [m for m in data if "Name" in m and m["Name"] == "John_Doe"]
</code></pre>
<p>You can unroll this list comprehension to see what it does:</p>
<pre><code>matches = []
for m in data:
    if "Name" in m and m["Name"] == "John_Doe":
        matches.append[m]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>def get_details(data, name):
    for i in data:
        if i['Name'] == name:
            return i
    return {}

data = [{"Name": "John_Doe", "Age" : 25 },{"Name": "Jane_Roe", "Age" : 26 }]
name = "John_Doe"
get_details(data, name)
</code></pre>
<p>output:</p>
<pre><code>{'Age': 25, 'Name': 'John_Doe'}
</code></pre>
