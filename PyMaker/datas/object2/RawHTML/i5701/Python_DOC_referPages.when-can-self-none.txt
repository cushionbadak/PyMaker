<p>I am looking at a snippet <code>if not self:</code> in <a href="https://codereview.stackexchange.com/a/146386/66848">an answer to another question</a> which implements <code>__nonzero__()</code>.</p>
<p>This gets me wondering: apart from <code>__nonzero__()</code> returning <code>False</code> or the trivial local assignment <code>self = None</code>, are there other situations, in which the conditional <code>if not self</code> is true?</p>

<div class="post-text" itemprop="text">
<p>According to Python's documentation on <a href="https://docs.python.org/3/library/stdtypes.html#truth-value-testing" rel="nofollow noreferrer">truth value testing</a>:</p>
<blockquote>
<p>Any object can be tested for truth value, for use in an <a href="https://docs.python.org/3/reference/compound_stmts.html#if" rel="nofollow noreferrer"><code>if</code></a> or <a href="https://docs.python.org/3/reference/compound_stmts.html#while" rel="nofollow noreferrer"><code>while</code></a>
  condition or as operand of the Boolean operations below.</p>
<p>By default, an object is considered true unless its class defines
  either a <a href="https://docs.python.org/3/reference/datamodel.html#object.__bool__" rel="nofollow noreferrer"><code>__bool__()</code></a> method that returns <code>False</code> or a <a href="https://docs.python.org/3/reference/datamodel.html#object.__len__" rel="nofollow noreferrer"><code>__len__()</code></a> method
  that returns zero, when called with the object.</p>
</blockquote>
<p>In the code you reference, <a href="https://docs.python.org/2/reference/datamodel.html#object.__nonzero__" rel="nofollow noreferrer"><code>__nonzero__()</code></a> is the Python 2 equivalent of Python 3's <code>__bool__()</code>.</p>
<p>So, an alternative to the <code>__bool__()</code> method in your question could be something like:</p>
<pre><code>class Lenny:

    def __len__(self):
        return 0 if self.value == '#' else len(self.children)
</code></pre>
<p><strong>Note:</strong> None of this has anything much to do with the title of your question: "When can self == None". <em>Equality</em> (whether to <code>None</code> or to anything else) is a different concept from <em>truth value</em>, and is defined by the <a href="https://docs.python.org/3/reference/datamodel.html#object.__eq__" rel="nofollow noreferrer"><code>__eq__()</code></a> method:</p>
<pre><code>class Nonelike:

    def __eq__(self, other):
        return other == None
</code></pre>
