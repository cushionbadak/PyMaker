<p>for i in range(10):<br/>
x(i) = 'abc'</p>
<p>so that I would get x1, x2, x3, x4, .... x10, all equal to 'abc'</p>

<div class="post-text" itemprop="text">
<p>You should not be doing that, store your values in a <code>dict</code> if you want to access them by name, or a <code>list</code> if you want to access them by index, but if you really insist:</p>
<pre><code>for i in range(10):
    locals()["x" + str(i)] = "abc"  # use globals() to store the vars in the global stack

print(x3)  # abc
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is probably the easiest way for beginners. You might want to learn how to use lists:</p>
<pre><code>&gt;&gt;&gt; x = []
&gt;&gt;&gt; for i in range(10):
    x.append('abc')

&gt;&gt;&gt; print(x[0])
abc
&gt;&gt;&gt; print(x[8])
abc
&gt;&gt;&gt; print(x)
['abc', 'abc', 'abc', 'abc', 'abc', 'abc', 'abc', 'abc', 'abc', 'abc']
</code></pre>
<p>You can also use <a href="https://docs.python.org/3/library/functions.html" rel="nofollow noreferrer">globals</a> or locals:</p>
<pre><code>&gt;&gt;&gt; for i in range(10):
    globals()["x" + str(i)] = 'abc'
&gt;&gt;&gt; x1
'abc'
</code></pre>
