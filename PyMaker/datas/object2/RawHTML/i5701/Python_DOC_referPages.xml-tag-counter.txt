<p>As stated in the question: </p>
<blockquote>
<p>you can check if a string is an XML tag if it starts with <code>&lt;</code> and ends with <code>&gt;</code></p>
</blockquote>
<p>You need to iterate over every string in a list and use <a href="https://docs.python.org/3/library/stdtypes.html#str.startswith" rel="nofollow noreferrer"><code>str.startswith()</code></a> and <a href="https://docs.python.org/3/library/stdtypes.html#str.endswith" rel="nofollow noreferrer"><code>str.endswith()</code></a> to check the first and last characters:</p>
<pre><code>In [1]: l = ["&lt;string1&gt;", "somethingelse", "&lt;/string1&gt;"]

In [2]: [item for item in l if item.startswith("&lt;") and item.endswith("&gt;")]
Out[2]: ['&lt;string1&gt;', '&lt;/string1&gt;']
</code></pre>
<p>Here we just filtered the desired strings in a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>, but to count how many matches we've got, we may use <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer"><code>sum()</code></a> adding a <code>1</code> every time there is a match:</p>
<pre><code>In [3]: sum(1 for item in l if item.startswith("&lt;") and item.endswith("&gt;"))
Out[3]: 2
</code></pre>
<hr/>
<p>This was though just one way to do it and I am not sure how far have you got in your course. A more naive and straightforward version of the answer might be:</p>
<pre><code>def tag_count(l):
    count = 0
    for item in l:
        if item.startswith("&lt;") and item.endswith("&gt;"):
            count += 1
    return count
</code></pre>
