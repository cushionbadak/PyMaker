<p><strong>I have a list of tuple:</strong></p>
<pre><code>[('Lebron James', datetime.date(2017, 12, 23), 500), ('Julie Peralta', 
datetime.date(2017, 12, 13), 1500), ('Reynaldo Pahay', datetime.date(2017, 1
2, 11), 2500)]
</code></pre>
<p><strong>I want to convert it into a list of dictionary:</strong></p>
<pre><code>[{'Name':'Lebron James','date' :datetime.date(2017, 12, 23), 'penalty': 
500},{'Name:'Julie Peralta','date': datetime.date(2017, 12, 13),'penalty': 
1500},{'Name:'Reynaldo Pahay','date': datetime.date(2017, 12, 11),'penalty': 
2500}]
</code></pre>
<p>What's the best and efficient way to do it?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can make use of <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> function.</p>
<pre><code>import datetime

def get_list_of_dict(keys, list_of_tuples):
     """
     This function will accept keys and list_of_tuples as args and return list of dicts
     """
     list_of_dict = [dict(zip(keys, values)) for values in list_of_tuples]
     return list_of_dict
</code></pre>
