<p>If you mean <code>Path</code> in standard library, use <a href="https://docs.python.org/3/library/pathlib.html#pathlib.Path" rel="nofollow noreferrer"><code>pathlib.Path</code></a>, not <code>path.Path</code>.</p>
<pre><code>&gt;&gt;&gt; from pathlib import Path
&gt;&gt;&gt; Path
&lt;class 'pathlib.Path'&gt;
</code></pre>
