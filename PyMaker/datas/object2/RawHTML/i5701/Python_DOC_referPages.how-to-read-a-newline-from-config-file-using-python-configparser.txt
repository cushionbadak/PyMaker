<p>I have a config File "Email_Properties.cfg" as below</p>
<pre><code>[email_Properties]
Subject=Security keywords Scan Info
Body=securityDetails\n Note : This is an auto generated mail
</code></pre>
<p>I am using ConfigParser to read file and print the content, Not able to get newLine using \n</p>
<pre><code>config = ConfigParser.ConfigParser()
config.readfp(open(r'Email_Properties.cfg'))
body=config.get('email_Properties', 'Body')
print body
</code></pre>
<p>output: </p>
<pre><code>securityDetails\n Note : This is an auto generated mail
</code></pre>
<p>How i can get NewLine present in the string of Config file ? Expected output as below,</p>
<pre><code>securityDetails
Note : This is an auto generated mail
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>ConfigParser escapes <code>\n</code> in ini values as <code>\\n</code>, so try with something like</p>
<pre><code>body.replace('\\n', '\n')
</code></pre>
<p>FWIW: The question's example is in Python 2, but for users of Python &gt;=3.2: <code>ConfigParser.read_file()</code> <a href="https://docs.python.org/3/library/configparser.html#configparser.ConfigParser.read_file" rel="nofollow noreferrer">replaces</a> the <code>ConfigParser.readfp()</code> which as of then is deprecated. This is <em>not</em> affecting the question/solution though. </p>
