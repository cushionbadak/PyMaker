<p>I'm calling a function <code>A(i)</code> in python . I want it to be terminated if it executes for more than X milliseconds. I've looked at some ways of timing out including <code>signal.alarm()</code>, but they all take the time in integral seconds. I want to do something like: </p>
<p><code>signal.alarm(0.26)  //time out after 0.26 seconds</code></p>
<p>How do I do this? </p>
</div>
<div class="post-text" itemprop="text">
<p>Aside from the <a href="https://stackoverflow.com/questions/3770711/python-signal-alarm-function">solution I linked as a possible duplicate</a>, you could also use <a href="https://docs.python.org/3/library/ctypes.html" rel="nofollow noreferrer"><code>ctypes</code></a> to use <a href="https://linux.die.net/man/3/ualarm" rel="nofollow noreferrer"><code>ualarm</code></a> if your system supports it.</p>
<p>For example, on macOS:</p>
<pre><code>import ctypes

libc = ctypes.CDLL("libc.dylib")

libc.ualarm(10000, 0) # 10,000 microseconds
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/signal.html#signal.setitimer" rel="nofollow noreferrer">singal.settimer</a> requires unix.
Else you can use <a href="https://docs.python.org/2/library/time.html#time.time" rel="nofollow noreferrer">time.time()</a></p>
<pre><code>import time

start = time.time()
now = time.time()
while now - start &lt; 0.26: # 0.26 seconds
   now = time.time()
print (now-start)
</code></pre>
<p>you can use a <a href="https://wiki.python.org/moin/PythonDecorators" rel="nofollow noreferrer">decorator</a>, put your function under the while loop or raise an exception when timer ends (and probably others).</p>
<p>You may have to <a href="https://docs.python.org/2/library/threading.html" rel="nofollow noreferrer">thread</a> the timer.</p>
