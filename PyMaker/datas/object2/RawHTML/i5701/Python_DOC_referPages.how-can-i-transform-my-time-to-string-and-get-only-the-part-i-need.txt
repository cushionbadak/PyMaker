<h1>Method 1: Strip space</h1>
<p>One quick solution, if your input doesn't change. </p>
<pre><code>from datetime import datetime, timedelta
api_time = datetime.strptime("1900-01-01 12:30:22".split(" ")[1], "%H:%M:%S")
print api_time
&gt;1900-01-01 12:30:22
</code></pre>
<p>In your case:</p>
<pre><code>api_time = datetime.strptime(parsed_json["result"]["parameters"]["time"], "%H:%M:%S")
</code></pre>
<h1>Method 2: Construct datetime</h1>
<p>Or you can just construct your <code>datetime</code> object and then access it's <code>second</code>, <code>minute</code>, <code>hour</code> attributes as follows:</p>
<pre><code>from datetime import datetime, timedelta
api_time = datetime.strptime("1900-01-01 12:30:22", "%Y-%m-%d %H:%M:%S")
print "{0}:{1}:{2}".format(api_time.second, api_time.minute, api_time.hour)
&gt;22:30:12
</code></pre>
