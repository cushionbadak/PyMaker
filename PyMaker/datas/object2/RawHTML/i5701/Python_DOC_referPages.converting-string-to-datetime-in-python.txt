<p>I have a simple three line script that converts a string to a datetime in Python. </p>
<pre><code>from datetime import datetime

mydate='Feb-22-1732'

print(datetime.strptime(mydate,'%b-%dd-%Y'))
</code></pre>
<p>But when I run this code, I get an error saying:</p>
<pre><code>ValueError: time data 'Feb-22-1732' does not match format '%b-%dd-%Y'
</code></pre>
<p>Can you please help me understand what am I doing wrong here? </p>
<p>Thanks in advance.</p>
</div>
<div class="post-text" itemprop="text">
<p>You don't need to repeat <code>d</code> two times - <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer"><code>%d</code> handles two-digit days by definition</a>:</p>
<blockquote>
<p><code>%d</code> - Day of the month as a zero-padded decimal number.</p>
</blockquote>
<pre><code>print(datetime.strptime(mydate,'%b-%d-%Y'))
</code></pre>
</div>
