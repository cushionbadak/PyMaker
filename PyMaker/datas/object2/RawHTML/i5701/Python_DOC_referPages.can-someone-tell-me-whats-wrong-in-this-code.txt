<div class="post-text" itemprop="text">
<p>The error is obviously that the file could not be found. However, since your source filename <code>f</code> is taken from <code>os.listdir()</code> that can’t be it: The file <em>has</em> to be there.</p>
<p>The problem is here:</p>
<pre><code>new_name = '{} - {}/{}{}'.format(f_nota, f_giorno, f_mese, f_ext)
</code></pre>
<p>The new name will be a name that contains a slash. As such, the part before the slash will be a directory in which the file will be moved. If that directory does not exist, then <code>os.rename</code> will fail with above error.</p>
<p>In order to move a file into a different directory, you need to make sure that the directory exists first, for example using <a href="https://docs.python.org/3/library/os.html#os.makedirs" rel="nofollow noreferrer"><code>os.makedirs</code></a>:</p>
<pre><code>os.makedirs(os.dirname(new_name))
os.rename(f, new_name)
</code></pre>
</div>
