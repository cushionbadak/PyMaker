<div class="post-text" itemprop="text">
<p>According to <a href="https://docs.python.org/3/library/turtle.html#turtle.bgpic" rel="nofollow noreferrer">https://docs.python.org/3/library/turtle.html#turtle.bgpic</a>
, the background pic must be a GIF. PNG is not supported. But you can convert the PNG to a GIF on the fly using either PIL/Pillow (Python Imaging Library) or NumPy. </p>
</div>
<span class="comment-copy">I tested your image with <code>bgpic()</code> on Python 2.7.12 on Linux Mint 18.2 based on Ubuntu 16 and it works. <code>turtle</code> is built on <code>tkinter</code> which supports only <code>gif</code> as default, so it can be problem in <code>turtle</code> too. There is special class in module <code>PIL/pillow</code> to use <code>png</code> (and other format) with <code>tkinter</code>. I already have <code>pillow</code> installed and I can't check if <code>bgpic</code> has problem without <code>pillow</code>. You can try to install <code>pillow</code>.</span>
