<div class="post-text" itemprop="text">
<p>This code should do the trick:</p>
<pre><code>import os
import shutil
import re

pattern = "^\d{4}2013$"
searchDir = 'C:/Users/USER/Desktop/test/'
innerDirs = os.listdir(searchDir)
for dir in innerDirs:
    if (re.search(pattern, dir)):
        shutil.rmtree(searchDir + dir) 
</code></pre>
<p>The code above will remove any "date" folder that ends with '2013'.</p>

<span class="comment-copy">You wrote a regex and then appended it to a string, creating a string.  There's nowhere I'm setting you apply the regex to the directory names.</span>
