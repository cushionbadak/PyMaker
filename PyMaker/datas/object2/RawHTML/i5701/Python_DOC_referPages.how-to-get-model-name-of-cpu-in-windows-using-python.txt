<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/platform.html#platform.processor" rel="nofollow noreferrer">platform.processor()</a> function returns the processor name as a string.</p>
<pre><code>&gt;&gt;&gt; import platform
&gt;&gt;&gt; platform.processor()
'Intel64 Family 6 Model 23 Stepping 6, GenuineIntel'
</code></pre>
