<div class="post-text" itemprop="text">
<p>Filter them out using <a href="https://docs.python.org/3/library/logging.html#logging.Logger.addFilter" rel="nofollow noreferrer"><code>Logger.addFilter</code></a>:</p>
<pre><code>class KeepalivesFilter (object):
    def filter(self, record):
        return record.msg.find('keepalive@openssh.com') &lt; 0

paramiko.util.get_logger('paramiko.transport').addFilter(KeepalivesFilter())
</code></pre>
<hr/>
<p>(using pysftp won't help as it uses Paramiko internally)</p>
