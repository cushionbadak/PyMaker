<div class="post-text" itemprop="text">
<p>You are calling the wrong class name in your super() call:</p>
<pre><code>class SimpleHelloWorld(IRCReplyModule):

     def __init__(self):
            #super(IRCReplyModule,self).__init__('hello world')
            super(SimpleHelloWorld,self).__init__('hello world')
</code></pre>
<p>Essentially what you are resolving to is the <code>__init__</code> of the object base class which takes no params.</p>
<p>Its a bit redundant, I know, to have to specify the class that you are already inside of, which is why in python3 you can just do:  <code>super().__init__()</code></p>
