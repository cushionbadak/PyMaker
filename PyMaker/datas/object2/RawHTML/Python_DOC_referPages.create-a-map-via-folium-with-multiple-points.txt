<div class="post-text" itemprop="text">
<p>What probably happens is that longitude/latitude is expected as a tuple/list with 2 elements, then follows <code>popup</code>, but you're providing 2 elements <em>then</em> <code>popup</code> again</p>
<p>Example from the <a href="https://python-visualization.github.io/folium/quickstart.html" rel="nofollow noreferrer">documentation</a>:</p>
<pre><code>folium.Marker([45.3288, -121.6625], popup='&lt;i&gt;Mt. Hood Meadows&lt;/i&gt;', 
</code></pre>
<p>Standalone example:</p>
<pre><code>def f(a,b,c=None):
    pass

f(12,34,b=12)
</code></pre>
<p>gives:</p>
<pre><code>TypeError: f() got multiple values for argument 'b'
</code></pre>
<p>because <code>b</code> is passed as positional <em>and</em> keyword.</p>
<p>Grouping lat/long in a list fixes your code:</p>
<pre><code>folium.Marker([lon[i], lat[i]], popup=df.Docking[i]).add_to(m)
</code></pre>
