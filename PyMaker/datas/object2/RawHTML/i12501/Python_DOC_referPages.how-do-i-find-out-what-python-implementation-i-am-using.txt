<p>Is there a simple way, like one command, that will enable me to find what Python implementation I am using (CPython, JPython, etc.)?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/2/library/platform.html#platform.python_implementation" rel="noreferrer"><code>platform.python_implementation()</code></a>:</p>
<pre><code>&gt;&gt;&gt; import platform
&gt;&gt;&gt; platform.python_implementation()
'CPython'
</code></pre>
