<p>I'm posting this question after not finding the answer in the many existing issues related to this topic.<br/>
I have a dictionary similar to this one:</p>
<p><code>foo = {'a': [(1, 0.5), (2, 0.3)], 'b': [(3, 0.4), (4, 0.1)]}
</code></p>
<p>and would like to access the items of the nested tupels, in order to create the following dictionary out of it:</p>
<p><code>foo = {'a': [1, 2], 'b': [3, 4]}
</code></p>
<p>How do I go about doing this, preferably in a pythonic way?</p>
</div>
<div class="post-text" itemprop="text">
<p>I think these two solutions are easiest to read:</p>
<pre><code>&gt;&gt;&gt; from operator import itemgetter
&gt;&gt;&gt; {k:map(itemgetter(0), foo[k]) for k in foo}
{'a': [1, 2], 'b': [3, 4]}
</code></pre>
<p>or import-less:</p>
<pre><code>&gt;&gt;&gt; {k:[x[0] for x in foo[k]] for k in foo}
{'a': [1, 2], 'b': [3, 4]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>what about?</p>
<pre><code>{x: map(lambda z: z[0], y) for x, y in foo.iteritems()}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <code>dict comprehension</code>: </p>
<pre><code>In [214]: {key:list(map(lambda x: x[0], foo[key])) for key in foo}
Out[214]: {'a': [1, 2], 'b': [3, 4]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>{k:list(zip(x,y)[0]) for k,(x,y) in foo.iteritems()}
</code></pre>
<p>This gives:</p>
<pre><code>{'a': [1, 2], 'b': [3, 4]}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do it with <a href="https://docs.python.org/3/library/collections.html?highlight=defaultdict#collections.defaultdict" rel="nofollow"><code>defaultdict</code></a> from <code>collections</code> module:</p>
<pre><code>&gt;&gt;&gt; from collections import defaultdict
&gt;&gt;&gt; d = defaultdict(list)
&gt;&gt;&gt; d = defaultdict(list)
&gt;&gt;&gt; for k in foo:
        for v in foo[k]:
            d[k].append(v[0])
&gt;&gt;&gt; d
defaultdict(&lt;class 'list'&gt;, {'a': [1, 2], 'b': [3, 4]})
&gt;&gt;&gt; d['a']
[1, 2]
</code></pre>
