<p>You probably want to go with</p>
<pre><code>buffer.encode('utf8')
</code></pre>
<p>in order to make the string a byte object and pass it to a socket connection.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you are trying to convert a <a href="https://docs.python.org/3/library/stdtypes.html#str" rel="nofollow"><code>str</code></a> to a <a href="https://docs.python.org/3/library/stdtypes.html#bytes" rel="nofollow"><code>bytes</code></a> object, the proper method is <a href="https://docs.python.org/3/library/stdtypes.html#str.encode" rel="nofollow"><code>str.encode()</code></a>:</p>
<pre><code>&gt;&gt;&gt; s = "foo"
&gt;&gt;&gt; s.encode("utf-8")
b'foo'
</code></pre>
