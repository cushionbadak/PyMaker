<div class="post-text" itemprop="text">
<p>Because you're converting a list to a string, actually you really shouldn't do that. I'd suggest use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>', '.join()</code></a> instead:</p>
<pre><code>List = [list(data.values())[x] for x in [7, 9, 12]]
string = ', '.join(map(str, List))
</code></pre>
<p>Let's see what's wrong:</p>
<pre><code>&gt;&gt;&gt; l = ['123', 456]
&gt;&gt;&gt; str(l)
"['123', 456]"

&gt;&gt;&gt; print(str(l))
['123', 456]

&gt;&gt;&gt; ', '.join(map(str, l))
'123, 456'

&gt;&gt;&gt; print(', '.join(map(str, l)))
123, 456
</code></pre>
<p>Because Python use quotes to says that: <em>This object is a string</em>.</p>
<p>And when you convert an object which has a string object within it, the quotes will still there instead of removed automatically. </p>
</div>
<span class="comment-copy">Briefly: <code>text_file.write(string.replace("'", '') + "\n")</code>.</span>
