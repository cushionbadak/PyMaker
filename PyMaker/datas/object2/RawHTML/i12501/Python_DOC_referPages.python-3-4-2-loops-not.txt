<div class="post-text" itemprop="text">
<p><code>input</code> always returns text - even there are only digits.</p>
<p>Use <code>int(Telephone)</code> to convert to integer or <code>Telephone.isdigit()</code> to check if there are only digits in text.</p>
<hr/>
<p><strong>BTW:</strong> both methods are useless if user use spaces or <code>-</code> in phone number ;)   </p>
<p>Maybe you will have to remove spaces and <code>-</code> using <code>replace()</code> before you use <code>int()</code> or <code>isdigit()</code></p>
