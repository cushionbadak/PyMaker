<pre><code>In [620]: [i if i in list2 else False for i in list1]

<div class="post-text" itemprop="text">
<p>Pretty simple to do using an <a href="https://docs.python.org/3/reference/expressions.html#conditional-expressions" rel="nofollow"><code>if</code> conditional expression</a> in a <a href="https://docs.python.org/3/reference/expressions.html#displays-for-lists-sets-and-dictionaries" rel="nofollow"><code>list</code> comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; list1 = [1, 2, 3, 4]
&gt;&gt;&gt; list2 = [3, 1, 4]
&gt;&gt;&gt; res = [i if i in list2 else False for i in list1]
&gt;&gt;&gt; print(res)
[1, False, 3, 4]
</code></pre>
<p>What this does is it takes every element in the <code>list1</code> list and checks if it is <code>in</code> the second list <code>list2</code>. If so, returns the item <code>i</code> itself, if not returns <code>False</code>.</p>
