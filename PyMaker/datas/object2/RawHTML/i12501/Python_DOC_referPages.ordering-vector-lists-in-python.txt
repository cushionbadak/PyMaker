<div class="post-text" itemprop="text">
<p>I have a list with several registers from other lists. I put an index at the beginning of each element so that I can order it later, for writing purposes.</p>
<pre><code>reg = index + ";" + cell[1] + ";" + hour + ";" + cell[0] + ";" + text[1] + ";"
</code></pre>
<p>At the end, <code>reg</code> should give something like this:</p>
<pre><code>['5;01;11.32;RN_RT;Prep;', '9;02;2.12;ING;Prep;', '1;06;4.23;ING;Rec;']
</code></pre>
<p>Is there a way to order/sort it using the index at the beginning?</p>
</div>
<div class="post-text" itemprop="text">
<p>Sort with <code>key=lambda s: int(s.split(';', 1)[0])</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can write a small helper function:</p>
<pre><code>def by_index(x):
    return int(x.split(';', 1)[0])
</code></pre>
<p>and use it for sorting:</p>
<pre><code>L = ['5;01;11.32;RN_RT;Prep;','9;02;2.12;ING;Prep;','1;06;4.23;ING;Rec;']

sorted(L, key=by_index)

['1;06;4.23;ING;Rec;', '5;01;11.32;RN_RT;Prep;', '9;02;2.12;ING;Prep;']
</code></pre>
<p>The function <a href="https://docs.python.org/3/library/functions.html?highlight=sorted#sorted" rel="nofollow">sorted()</a> takes the keyword argument <code>key</code>:</p>
<blockquote>
<p>key specifies a function of one argument that is used to extract a comparison key from each list element: key=str.lower. The default value is None (compare the elements directly).</p>
</blockquote>
<p>This function is called for each element in the list. So:</p>
<pre><code>int(x.split(';', 1)[0])
</code></pre>
<p>splits off the value before the first <code>;</code> and turns it into an integer. This is the index. This index will be used for sorting (the sorting key) but the sort result is the list elements in the order of the index.</p>
<p><code>lambda</code> is just another way to define a function without using <code>def</code>. It is limited to expressions but can be used inline. </p>
</div>
