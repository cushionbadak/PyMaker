<div class="post-text" itemprop="text">
<p>You can convert text to integer and then you can use <code>struct</code> module</p>
<pre><code>import struct

text = "0000001011001100"
number = int(text, 2) # 716

result = struct.pack("h", number)

b'\xcc\x02'

# or with &gt; to change bytes order

result = struct.pack("&gt;h", number)

b'\x02\xcc'
</code></pre>
<p>see: <a href="https://docs.python.org/3/library/struct.html" rel="nofollow">https://docs.python.org/3/library/struct.html</a></p>
</div>
<div class="post-text" itemprop="text">
<p>You need to convert the binary representation into an int <strong>with base 2</strong></p>
<pre><code>text="0000001011001100"
num =int(text,2) #interprets string in base 2
result = chr(num) #is this what you want?
</code></pre>
<p>if you are using python 3.x and want to convert the text into a <code>bytes</code> object from a string of any length by splitting up each 8 characters:</p>
<pre><code>bytes(int(text[i:i+8],2) for i in range(0,len(text),8))
</code></pre>
<p>but for your example it gives <code>b'\x02\xcc'</code> so it is possible you want something different</p>
