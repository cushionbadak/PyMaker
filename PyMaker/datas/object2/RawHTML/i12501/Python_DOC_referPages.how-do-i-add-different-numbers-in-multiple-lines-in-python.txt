<div class="post-text" itemprop="text">
<p>So I have a file with 9 different phone numbers. There is a line for every call, and the duration. I want to add up the duration of the calls and match them to their own phone number.</p>
<p>Here is part of the file:</p>
<pre><code>    7803214567;94
    7801234567;602
    7801234567;910
    7808765432;925
    7801234567;631
</code></pre>
<p>The first being the phone number and the second being duration in seconds. To be more clear, an example: <code>7801234567</code>, I want to add <code>602</code>, <code>910</code> and <code>631</code> and match it to its own phone number <code>7801234567</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>It seems you just want to count the total number of minutes for each unique phone number.</p>
<p>You could use a <a href="https://docs.python.org/2/library/collections.html#counter-objects" rel="nofollow">counter</a>:</p>
<pre><code>from collections import Counter
c = Counter()

with open("path/to/file.txt", 'r') as f:
    for line in f:
        k, v = line.split(";")
        c += Counter({k: int(v)})

&gt;&gt;&gt; print(c)
Counter({'7801234567': 2143, '7808765432': 925, '7803214567': 94})
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/collections.html?highlight=defaultdict#collections.defaultdict" rel="nofollow">defaultdict</a>:</p>
<pre><code>from collections import defaultdict

call_times = defaultdict(int)

with open('phone.txt') as fobj:
    for line in fobj:
        number, call_time = line.split(';')
        call_times[number.strip()] += int(call_time)

&gt;&gt;&gt; call_times
defaultdict(int, {'7801234567': 2143, '7803214567': 94, '7808765432': 925})
</code></pre>
<p><code>defaultdict</code> allows to specify a callable that creates a new value for a missing key. For example, this <code>defaultdict</code>:</p>
<pre><code>&gt;&gt;&gt; call_times = defaultdict(int)
&gt;&gt;&gt; call_times
defaultdict(int, {})
</code></pre>
<p>returns <code>0</code> for a key it does not have:</p>
<pre><code>&gt;&gt;&gt; call_times['a']
0
</code></pre>
<p>because it uses <code>int()</code> for such keys:</p>
<pre><code>&gt;&gt;&gt; int()
0
</code></pre>
<p>Therefore, you can use <code>+=</code> to add up you numbers:</p>
<pre><code>&gt;&gt;&gt; defaultdict(int, {'a': 0})
&gt;&gt;&gt; call_times['b'] += 1
&gt;&gt;&gt; call_times
defaultdict(int, {'a': 0, 'b': 1})
</code></pre>
