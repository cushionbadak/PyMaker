<p>I have a list containing some tuples as well as normal strings like below :       </p>
<pre><code>a = ['sore', ('PHENOMENA', 'sore'), 'throat', ('ANATOMY', 'throat'), 'and',   'leg', ('ANATOMY', 'leg')]
</code></pre>
<p>I want to write it in a space separated text file in a format like:</p>
<pre><code>sore ('PHENOMENA', 'sore') throat ('ANATOMY', 'throat') and leg ('ANATOMY', 'leg')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Something like this?</p>
<pre><code>&gt;&gt;&gt; ' '.join(map(str, a))
"sore ('PHENOMENA', 'sore') throat ('ANATOMY', 'throat') and leg ('ANATOMY', 'leg')"
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map(str, a)</code></a> convert all elements in <code>a</code> to string object, then <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>' '.join()</code></a> join them by space and returns a string object.</p>
<p>You can also use a <a href="https://stackoverflow.com/questions/34835951/what-does-list-comprehension-in-python-mean-how-does-it-work-and-how-can-i-us"><em>list comprehension</em></a> instead of <code>map()</code>:</p>
<pre><code>&gt;&gt;&gt; ' '.join(str(i) for i in a)
"sore ('PHENOMENA', 'sore') throat ('ANATOMY', 'throat') and leg ('ANATOMY', 'leg')"
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try this</p>
<pre><code>a = ['sore', ('PHENOMENA', 'sore'), 'throat', ('ANATOMY', 'throat'), 'and',   'leg', ('ANATOMY', 'leg')]
s = ' '.join(str(part) for part in a)    
with open('outfile.txt', 'w') as outfile:
    outfile.write(s)
</code></pre>
<p>In the directory where you run this code you will get a file named 'outfile.txt' containing this:</p>
<pre><code>sore ('PHENOMENA', 'sore') throat ('ANATOMY', 'throat') and leg ('ANATOMY', 'leg')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can join each element of the list by a space via ' '.join(). The caveat is you need to convert everything to string, including the tuples, which you can do using <code>map</code>.</p>
<p>One possible way of coding is as simple as:</p>
<pre><code>with open('file.txt', 'w') as f:
    f.write(' '.join(map(str, a)))
</code></pre>
