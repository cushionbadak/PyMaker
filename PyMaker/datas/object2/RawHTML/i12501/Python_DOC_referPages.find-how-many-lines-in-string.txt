<p>I am creating a python movie player/maker, and I want to find the number of lines in a multiple line string. I was wondering if there was any built in function or function I could code to do this:</p>
<pre><code>x = """
line1
line2 """

getLines(x)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If newline is <code>'\n'</code> then <code>nlines = x.count('\n')</code>.</p>
<p>The advantage is that you don't need to create an unnecessary list as <code>.split('\n')</code> does (the result may differ depending on <code>x.endswith('\n')</code>).</p>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.splitlines" rel="noreferrer"><code>str.splitlines()</code></a> accepts more characters as newlines: <code>nlines = len(x.splitlines())</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can <code>split()</code> it and find the length of the resulting <code>list</code>:</p>
<pre><code>length = len(x.split('\n'))
</code></pre>
<p>Or you can <code>count()</code> the number of newline characters:</p>
<pre><code>length = x.count('\n')
</code></pre>
<p>Or you can use <code>splitlines()</code> and find the length of the resulting <code>list</code>:</p>
<pre><code>length = len(x.splitlines())
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do:</p>
<pre><code>len(x.split('\n'))
</code></pre>
