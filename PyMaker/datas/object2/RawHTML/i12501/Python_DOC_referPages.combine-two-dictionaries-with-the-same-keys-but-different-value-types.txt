<pre><code>food = {'fruit' : 5, 'vegetable' : 2, 'dairy' : 1, 'meat' : 3, 'grain' : 1}

fav = { 'fruit' : ['apple', 'orange', 'banana', 'grape', 'plum'],
        'vegetable' : ['carrot', 'corn'],
        'dairy' : ['milk'],
        'meat' : ['chicken', 'egg', 'beef'],
        'grain' : ['bread']
      }
</code></pre>
<p>The result I'm looking to achieve will be something like this,</p>
<pre><code>eats = { 'fruit' : 5, ['apple', 'orange', 'banana', 'grape', 'plum'], 
         'vegetable' : 2, ['carrot', 'corn'],
       ...
</code></pre>

</code></pre>

<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow">dict comprehension</a> and tuples. I think you couldn't get the output you expecting because it's not valid dict (with comma should be separated neihbor keys):</p>
<pre><code>eats = {key : (food[key],fav[key]) for key in food}

print(eats)
{'dairy': (1, ['milk']),
 'fruit': (5, ['apple', 'orange', 'banana', 'grape', 'plum']),
 'grain': (1, ['bread']),
 'meat': (3, ['chicken', 'egg', 'beef']),
 'vegetable': (2, ['carrot', 'corn'])}
</code></pre>
<p><strong>EDIT</strong></p>
<p>For your edit you need to add check key in <code>fav</code> dict:</p>
<pre><code>eats = {key : (food[key],fav[key]) for key in food if key in fav}
</code></pre>
<p>The result for your example is the same as posted.</p>

<pre><code>for key in food:
    eats[key] = [food[key], fav.get(key, [])]
</code></pre>
<p>Or, just one line,</p>
<pre><code>eats = map(lambda x: [food[x], fav.get(x, [])], food.keys())
</code></pre>
