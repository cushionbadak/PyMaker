<div class="post-text" itemprop="text">
<p><code>datetime.strptime(date_string, format)</code> returns a <code>datetime</code> corresponding to <code>date_string</code>, parsed according to <code>format</code>. You can then use <code>datetime.timestamp()</code> to convert a datetime tuple to the epoch time.
For example:</p>
<pre><code>from datetime import datetime

datetime.timestamp(datetime.strptime('Sunday, September 26, 2010 at 8:35 PM', "%A, %B %d, %Y at %I:%M %p"))

&gt;&gt;1285547700.0
</code></pre>
