<p>I am trying to create a turtle in Python so I could increase/ decrease it's size by pressing +/- on keyboard  </p>
<pre><code>import turtle

turtle.setup(700,500)
wn = turtle.Screen()
testing_turtle = turtle.Turtle()


size = 1
def dropsize():
    global size
    if size&gt;1:
        size-=1
        print(size)    # To show the value of size
        testing_turtle.shapesize(size)

def addsize():
    global size
    if size&lt;20:    # To ensure the size of turtle is between 1 to 20
        size+=1
        print(size)
        testing_turtle.shapesize(size)


wn.onkey(addsize,'+')
wn.onkey(dropsize,'-')


wn.listen()
wn.mainloop()
</code></pre>

<div class="post-text" itemprop="text">
<p>You need to use <code>'plus'</code> and <code>'minus'</code> to bind the functions to the key-release event of the <kbd>+</kbd> and <kbd>-</kbd> key:</p>
<pre><code>wn.onkey(addsize,'plus')
wn.onkey(dropsize,'minus')
</code></pre>
<p>To solve the <code>NameError</code> exception, either place your <code>size</code> variable otside your main loop or use the <a href="https://docs.python.org/3/reference/simple_stmts.html#nonlocal" rel="nofollow"><code>nonlocal</code></a> statement instead of <code>global</code>:</p>
<pre><code>def addsize():
    nonlocal size
    if size&lt;20:    # To ensure the size of turtle is between 1 to 20
        size+=1
        print(size)
        testing_turtle.shapesize(size)
</code></pre>
