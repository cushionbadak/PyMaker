<div class="post-text" itemprop="text">
<p>If you want to print "1 &lt; 2 &lt; 3" all you need to do is change the type from an int to a string as such:</p>
<pre><code>print(' &lt; '.join(str(n) for n in (1,2,3)))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have to convert the ints to strings first:</p>
<pre><code>numbers = (1, 2, 3)
print(' &lt; '.join(str(x) for x in numbers))
</code></pre>

<div class="post-text" itemprop="text">
<p>You can convert your array using <code>map</code>:</p>
<pre><code>print(' &lt; '.join(map(str,(1,2,3))))
</code></pre>
