<h2>Basic version</h2>
<p>This could be a solution:</p>
<pre><code>from __future__ import print_function

while True:
    username = raw_input("Please give me a username: ")
    if not any(c in username for c in '0123456789'):
        print("Username needs to contain at least one number.")
        continue
    if len(username) &gt;= 8:
        print("Well done")
    elif len(username) &lt;= 1:
        print("More characters please.")
        print("Please try again, input your username again.")
        continue
    break
</code></pre>
<p>Keep asking the user in a <code>while</code> loop until you get what you want.</p>
<p>This checks if the user name contains at least one digit:</p>
<pre><code>&gt;&gt;&gt; any(c in 'abc' for c in '0123456789')
False
&gt;&gt;&gt; any(c in 'abc1' for c in '0123456789')
True
</code></pre>
<p>This part is a so-called generator expression:</p>
<pre><code>&gt;&gt;&gt; (c in 'abc' for c in '0123456789')
&lt;generator object &lt;genexpr&gt; at 0x10aa339e8&gt;
</code></pre>
<p>The easiest way to visualize what it is doing, is to convert it into a list:</p>
<pre><code>&gt;&gt;&gt; list((c in 'abc' for c in '0123456789'))
[False, False, False, False, False, False, False, False, False, False]
&gt;&gt;&gt; list((c in 'abc1' for c in '0123456789'))
[False, True, False, False, False, False, False, False, False, False]
</code></pre>
<p>It lets <code>c</code> run through all elements of <code>0123456789</code> i.e. it takes on the values <code>0</code>, <code>1</code>, ... <code>9</code> in turn, and checks if this value is contained in <code>abc</code>.</p>
<p>The built-in <a href="https://docs.python.org/3/library/functions.html?highlight=any#any" rel="nofollow">any</a> returns <code>True</code> if any element is true:</p>
<blockquote>
<p>Return <code>True</code> if any element of the iterable is true. If the iterable is empty, return <code>False</code>. </p>
</blockquote>
<p>An alternative way to check for a digit in a string would be to use regular expressions. The module <a href="https://docs.python.org/3/library/re.html?highlight=re#module-re" rel="nofollow">re</a> offers this functionality:</p>
<pre><code>import re

for value in ['abc', 'abc1']:
    if re.search(r'\d', value):
        print(value, 'contains at least one digit')
    else:
        print(value, 'contains no digit')
</code></pre>
<p>Prints:</p>
<pre><code>abc contains no digit
abc1 contains at least one digit
</code></pre>
<h2>In a function</h2>
<p>You can put this functionality in a function (as requested by the OP in a comment):</p>
<pre><code>def ask_user_name():
    while True:
        username = raw_input("Please give me a username: ")
        if not any(c in username for c in '0123456789'):
            print("Username needs to contain at least one number.")
            continue
        if len(username) &gt;= 8:
            print("Well done")
        elif len(username) &lt;= 1:
            print("More characters please.")
            print("Please try again, input your username again.")
            continue
        break
    return username

print(ask_user_name())
</code></pre>
