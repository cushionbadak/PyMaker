<p>this is rough, but it will work: </p>
<pre><code>import matplotlib.pyplot as plt
import random 

x = [1, 2, 4,6, 7, 100, 200, 100, 34]
tenxArray = []
fiveXPlusTwenty = []

for i in range(0, len(x)):
    tenxArray.append(10*x[i])
    fiveXPlusTwenty.append(5*x[i] + 20)

plt.plot(x, tenxArray)
plt.plot(x, fiveXPlusTwenty)

plt.show()
</code></pre>

<div class="post-text" itemprop="text">
<p>You could use the <a href="https://docs.python.org/3/library/stdtypes.html#range" rel="nofollow"><code>range</code></a> generator to generate x values and list comprehensions to calculate y values for your functions:</p>
<pre><code>import matplotlib.pyplot as plt

def fun1(x):
    return 10*x

def fun2(x):
    return 5*x+20

MAX_X = # A maximum value for x goes here
x = range(MAX_X)

y1,y2 = zip(*[ (fun1(val), fun2(val)) for val in x ])

plt.plot(x, y1)
plt.plot(x, y2)
plt.ylabel('some numbers')
plt.show()
</code></pre>
