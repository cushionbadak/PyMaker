<div class="post-text" itemprop="text">
<p><code>index</code> returns the index of the first element only. From the <a href="https://docs.python.org/2/tutorial/datastructures.html" rel="nofollow noreferrer">docs</a></p>
<blockquote>
<p>Return the index in the list of the first item whose value is x. It is an error if there is no such item.</p>
</blockquote>
<p>Use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate</code></a> instead. When you iterate using <em>enumerate</em> you can access the element and its index on the loop:</p>
<pre><code>&gt;&gt;&gt; li = ['a', 'e', 'a', 'd', 'b', 'a', 'e']
&gt;&gt;&gt; for i,element in enumerate(li):
...     if element == 'a':
...        print(i)
... 
0
2
5
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>li = ['a', 'e', 'a', 'd', 'b', 'a', 'e']
</code></pre>
<p>Use List Comprehension:</p>
<pre><code>[i for i, x in enumerate(li) if x=='a']
</code></pre>
<p>Output: <code>[0, 2, 5]</code></p>
</div>
<div class="post-text" itemprop="text">
<p>Use keyword enumerate()  this will generate a counter from 0 to N.</p>
<pre><code>for i, x in enumerate(li):
</code></pre>
<p>So <code>i</code> will contain the indexes for <code>li</code>.</p>
</div>
