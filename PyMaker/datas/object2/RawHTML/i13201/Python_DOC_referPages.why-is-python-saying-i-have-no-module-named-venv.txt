<div class="post-text" itemprop="text">
<p>Since you are on Python 2, you need to execute using the virtualenv module that you installed. </p>
<p>First step, as you originally tried to do, but this time you specify the "virtualenv" module and the name of the virtualenv. In this case flask:</p>
<pre><code>python -m virtualenv flask
</code></pre>
<p>Then you activate your virtualenv like this: </p>
<pre><code>source flask/bin/activate
</code></pre>
<p>Then install flask with pip inside the virtualenv</p>
<pre><code>pip install flask
</code></pre>
<p>If you want to deactivate your virtualenv, simply type:</p>
<pre><code>deactivate
</code></pre>
<p>If running on <em>Python 3</em>, the <code>venv</code> command is built-in and you can simply do: </p>
<pre><code>python3 -m venv flask
</code></pre>
<p>Note, depending on how your Python 3 is installed, your python execution command might differ. You could be running it as <code>python3</code>, <code>python3.5</code>, <code>python3.6</code>.</p>
