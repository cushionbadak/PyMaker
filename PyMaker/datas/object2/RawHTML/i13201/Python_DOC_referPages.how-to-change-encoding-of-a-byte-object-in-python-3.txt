<p>You have JSON there, with the Arabic characters escaped as <a href="https://tools.ietf.org/html/rfc7159.html#page-8" rel="nofollow noreferrer">permitted by RFC 7159</a>.  You need to parse it with <a href="https://docs.python.org/3/library/json.html" rel="nofollow noreferrer"><code>json</code></a> in order to undo the escaping.  Once you've done that, you should be able to extract the "contents" value and print that (to a file, since Windows <a href="https://stackoverflow.com/questions/14630288/unicodeencodeerror-charmap-codec-cant-encode-character-maps-to-undefined?lq=1">can't always display Unicode properly</a> at the console).  Something like this:</p>
<pre><code>&gt;&gt;&gt; import urllib.request as urllib2
&gt;&gt;&gt; result = json.loads(urllib2.urlopen('...').read().decode('utf8'))
&gt;&gt;&gt; with open('example.txt', 'w', encoding='utf8') as f:
...     print(result['captions'][0]['content'], file=f)
</code></pre>
<p>You should then be able to open example.txt with your editor of choice.  If it displays incorrectly, be sure to set the encoding to UTF-8.</p>
