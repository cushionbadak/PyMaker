<p>The result of running my code is something like: </p>
<pre><code>b'{"username":"test","available":false,"status":"unavailable","failed_reason":null,"callback_url":"http://www.twitter.com/"}'
</code></pre>
<p>How can I actually get the value of [username] from this response ?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <code>json</code> module to convert your data to json and then it's a simple access with <code>get</code> after that. Observe the demo below with your data structure:</p>
<h1>Python 2 approach</h1>
<pre><code>In [5]: import json

In [6]: a = json.loads(b'{"username":"test","available":false,"status":"unavailable","failed_reason":null,"callback_url":"http://www.twitter.com/"}')

In [7]: a
Out[7]:
{u'available': False,
 u'callback_url': u'http://www.twitter.com/',
 u'failed_reason': None,
 u'status': u'unavailable',
 u'username': u'test'}

In [8]: a.get('username')
Out[8]: u'test'
</code></pre>
<h1>Python 3 Approach</h1>
<p>You have to be careful in Python 3 as json expects a string, therefore for your particular case you need to decode <code>utf-8</code>. So this example works in Python 3 as such: </p>
<pre><code>&gt;&gt;&gt; a = b'{"username":"test","available":false,"status":"unavailable","failed_reason":null,"callback_url":"http://www.twitter.com/"}'
&gt;&gt;&gt; a = a.decode('utf-8')
&gt;&gt;&gt; import json
&gt;&gt;&gt; a = json.loads(a)
&gt;&gt;&gt; a.get('username')
'test'
</code></pre>

<div class="post-text" itemprop="text">
<p>Since it is a byte string, you just have to decode it first.</p>
<pre><code>import json

data = b'{"username":"test","available":false,"status":"unavailable","failed_reason":null,"callback_url":"http://www.twitter.com/"}'
a = json.loads(data.decode('utf-8'))
print(a['username'])
</code></pre>
