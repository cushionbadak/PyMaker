<div class="post-text" itemprop="text">
<p>I'm trying to cut a string into pieces in Python with the following code:</p>
<p><code>re.split("[A-Z][a-z]?[0-9]*","CO2")</code></p>
<p>I expect some result like this:</p>
<pre><code>["C","O2"]
</code></pre>
<p>however I get:</p>
<pre><code>['','','']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Since you have no capturing groups in your re, the things you split on are not part of the result. What you're getting is the empty string before the 'C', the empty string between 'C' and 'O2' and the empty string after 'O2'.</p>
<p>If you use</p>
<pre><code>re.split("([A-Z][a-z]?[0-9]*)","CO2")
</code></pre>
<p>you'll get</p>
<pre><code>['', 'C', '', 'O2', '']
</code></pre>
<p>One dirty approach would be to use</p>
<pre><code>filter(None, re.split("([A-Z][a-z]?[0-9]*)","CO2"))
</code></pre>
<p>to remove elements that are not truthy (in this case, empty strings).</p>
</div>
<div class="post-text" itemprop="text">
<p><strong>If you want to split</strong></p>
<p>Use look arounds like this:</p>
<pre><code>(?&lt;=[A-Z\d])(?=[A-Z])
</code></pre>
<p><a href="https://regex101.com/r/eV0rY5/1" rel="nofollow">Regex live here.</a></p>
<hr/>
<p><strong>Else</strong></p>
<p>To simply match (without split):</p>
<pre><code>[A-Z][a-z]?\d*
</code></pre>
<p><a href="https://regex101.com/r/eV0rY5/4" rel="nofollow">Regex live here.</a></p>
<hr/>
<p>Hope it helps.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can also use this one:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; x = re.findall(r"^([A-Z][a-z]?)([A-Z][0-9]*)$","CO2")
&gt;&gt;&gt; list(x[0])
['C', 'O2']
</code></pre>
</div>
