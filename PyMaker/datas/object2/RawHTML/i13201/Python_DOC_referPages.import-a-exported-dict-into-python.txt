<div class="post-text" itemprop="text">
<p>If you'd like save some data such as <code>list</code>, <code>dict</code> or <code>tuple</code>, etc. to a file. And you want to edit them or you just want them be <em>readable</em> . Use <a href="https://docs.python.org/3/library/json.html"><code>json</code></a> module like this:</p>
<pre><code>&gt;&gt;&gt; import json
&gt;&gt;&gt; d = {'apple': 1, 'bear': 2}

&gt;&gt;&gt; print(d)
{'bear': 2, 'apple': 1}

&gt;&gt;&gt; print(json.dumps(d))
{"bear": 2, "apple": 1}  # these are json data
&gt;&gt;&gt; 
</code></pre>
<p>Now you could save these data to a file. If you want to load them, use <code>json.loads()</code> like this:</p>
<pre><code>&gt;&gt;&gt; json_data = '{"bear": 2, "apple": 1}'
&gt;&gt;&gt; d = json.loads(json_data)
&gt;&gt;&gt; d['bear']
2
&gt;&gt;&gt;
</code></pre>
</div>
