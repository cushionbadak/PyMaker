<div class="post-text" itemprop="text">
<p>You're on the right track with <code>sum()</code>, but you should put the iteration inside it, so that it actually has something to add up:</p>
<pre><code>def avg_price(CL):
    print(sum(i.price for i in CL) / len(CL))
</code></pre>
<p>To find the average, just divide by the length of the <code>list</code>. You don't need to divide it by all numbers from 0 to the length of the <code>list</code> (exclusive).</p>
<p>And I would use <code>return</code> here:</p>
<pre><code>def avg_price(CL):
    return sum(i.price for i in CL) / len(CL)
</code></pre>
<p>Then put the call in a <code>print()</code> call:</p>
<pre><code>print(avg_price(CL))
</code></pre>
