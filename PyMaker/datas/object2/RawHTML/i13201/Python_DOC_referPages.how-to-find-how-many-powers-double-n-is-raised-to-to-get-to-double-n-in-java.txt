<p>I want to find the equivalent of <code>Math.log</code> in Python in Java. Basically how many powers is a <code>double 'n'</code> raised to to get to the full <code>double 'n'</code>. Example <code>(100, 10)</code> returns 2. Because, 10 is raised to the second power to get to 100.
<code>27, 3</code> returns: 3.
<code>49, 7</code> returns 2. Etc... 
Does anyone know what this would be in Java.</p>
</div>
<div class="post-text" itemprop="text">
<p>In Python you can use <a href="https://docs.python.org/3/library/math.html#math.log" rel="nofollow"><code>math.log</code></a> which takes a second argument which is the base.</p>
<pre><code>&gt;&gt;&gt; math.log(100, 10)
2.0
&gt;&gt;&gt; math.log(27, 3)
3.0
</code></pre>
<p>In <code>Java</code> there is no such function, but you can trivially solve for the same thing</p>
<pre><code>public double logOfBase(int base, int num) {
    return Math.log(num) / Math.log(base);
}
</code></pre>
