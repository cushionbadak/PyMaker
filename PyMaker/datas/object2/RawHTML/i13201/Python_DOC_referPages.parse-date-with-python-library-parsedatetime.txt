<p>To parse <code>"Oct 4"</code>, you could use <a href="https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior" rel="nofollow"><code>datetime.strptime()</code></a>:</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; datetime.strptime("Oct 4", "%b %d")
datetime.datetime(1900, 10, 4, 0, 0)
&gt;&gt;&gt; d = datetime.strptime("Oct 4", "%b %d")
&gt;&gt;&gt; datetime(datetime.now().year, d.month, d.day)
datetime.datetime(2015, 10, 4, 0, 0)
</code></pre>
<p>If you want dates from the past; pass "Jan 1" for the corresponding year as the source time (default is the current time "Oct 25"):</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; import parsedatetime as pdt # $ pip install parsedatetime
&gt;&gt;&gt; cal = pdt.Calendar()
&gt;&gt;&gt; cal.parseDT("Oct 4") # today is Oct 25
(datetime.datetime(2016, 10, 4, 14, 19, 9), 1)
&gt;&gt;&gt; cal.parseDT("Oct 4", datetime(2015, 1, 1))
(datetime.datetime(2015, 10, 4, 0, 0), 1)
</code></pre>
