<p>I have a list of item as follows:</p>
<pre><code>language = ["python", "C", "C++", "Java"]
</code></pre>
<p>I want to print a list item as follows:</p>
<pre><code>w[0] = "Pyhon"
w[1] = "C"
W[2] = "C++"
</code></pre>
<p>I have tried as follow:</p>
<pre><code>for id, elem in enumerate(language):
    if elem is not None:
        print("w['id']=",elem)
</code></pre>
<p>However it is not working as per my requirement.</p>
</div>
<div class="post-text" itemprop="text">
<p>When you print - <code>print("w['id']=",elem)</code> - Python would not automatically substitute the <code>id</code> inside the string, you would need to format the string correctly so that <code>id</code> is printed there.</p>
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format()</code></a> for that. Example -</p>
<pre><code>for id, elem in enumerate(language):
        if elem is not None:
            print("w[{0}] = {1}".format(id,elem))
</code></pre>
<p>If you want the elements in the output within quotes, you can use the following <code>print</code> function call -</p>
<pre><code>print('w[{0}] = "{1}"'.format(id,elem))
</code></pre>
<p>If you want the first letter capitalized, then -</p>
<pre><code>print('w[{0}] = "{1}"'.format(id,elem.capitalize()))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The string in <code>w['id']=</code> is being treated as a literal string, that would work in PHP but not python. Use string concatenation like this:</p>
<pre><code>language =["python","C","C++","Java"]

for id, elem in enumerate(language):
    if elem is not None:
        print('w[%s] = "%s"' % (id, elem))
</code></pre>
</div>

<pre><code>for id, elem in enumerate(language):
    if elem:
        print('w[{}]="{}"'.format(id,elem))
</code></pre>
