<p>I'm trying to copy book names that are &lt;2000 into a new list. But the problem I'm having is that it only copies "1984" and it's copying it as an individual character to the list rather than copying all the names that it's supposed to</p>
<pre><code>from collections import namedtuple
Book = namedtuple('Book','author title genre year price instock')

BSI = [ Book('John Green', 'Paper Towns', 'Young Adult', 2008, 7.00, 200),
        Book('Beverly Clearly', 'Ramona Forever', 'Children', 1924, 9.00, 150),
        Book('Vladmir Nabokov', 'Lolita', 'Tragicomedy', 1958, 15.00 , 80),
        Book('J.D. Salinger', 'Catcher in the Rye', 'Young Adult', 1951, 10.00, 130),
        Book('George Orwell', '1984', 'Dystopia', 1949, 7.00, 300),
        Book('Jeannette Walls','The Glass Castle','Memoir', 2006, 15.00 , 100)]

older_books = []

for books in BSI:
    if (books.year&lt;2000):
        older_books=list(books.title)

print(older_books) #Outputs ['1', '9', '8', '4']
</code></pre>
<p>What it should output</p>
<blockquote>
<p>Ramona Forever<br/>
  Lolita<br/>
  Catcher in the Rye<br/>
  1984<br/></p>
</blockquote>

<div class="post-text" itemprop="text">
<p>With each <code>for</code> loop, the code is overwriting the list <code>older_books</code> with the last book title (converted to list) since you're not appending to it but merely creating a new list each time.</p>
<pre><code>&gt;&gt;&gt; list('Ramona Forever')
['R', 'a', 'm', 'o', 'n', 'a', ' ', 'F', 'o', 'r', 'e', 'v', 'e', 'r']
&gt;&gt;&gt; list('1984')
['1', '9', '8', '4']
</code></pre>
<p>Use <a href="https://docs.python.org/3/library/stdtypes.html#mutable-sequence-types" rel="nofollow"><code>list.append</code></a> so that all book titles that match the condition will be appended to a single list:</p>
<pre><code>&gt;&gt;&gt; older_books = []
&gt;&gt;&gt; older_books.append('Ramona Forever')
&gt;&gt;&gt; older_books.append('1984')
&gt;&gt;&gt; older_books
['Ramona Forever', '1984']
</code></pre>
<hr/>
<pre><code>for books in BSI:
    if books.year &lt; 2000:
        older_books.append(books.title)  # &lt;---
</code></pre>
<p><strong>UPDATE</strong> Alternative using <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>older_books = [b.title for b in BSI if b.year &lt; 2000]
</code></pre>
</div>
<span class="comment-copy">Alternatively, this problem is well-handled by a list comprehension: <code>older_books = [b.title for b in BSI if b.year &lt; 2000]</code></span>
