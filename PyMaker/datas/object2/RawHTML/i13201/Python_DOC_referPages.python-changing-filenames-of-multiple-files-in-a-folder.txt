<p>I ran your code and here are your two main problems: </p>
<p>The first being, you are not closing your bracket on your <code>os.rename</code> line.</p>
<p>You have this: </p>
<pre><code>os.rename(str(lijstmetfiles[i]), str("doggy"+str(i))
</code></pre>
<p>You are missing a parentheses, so should be: </p>
<pre><code>os.rename(str(lijstmetfiles[i]), str("doggy"+str(i)))
</code></pre>
<p>However, this could be an edit issue from copying your code over in to your post.</p>
<p>Secondly, and most importantly, you are not specifying the path you want to do your rename on, you are just giving two filenames, so you are most likely getting a file not found error. </p>
<p>You need to make use of Python's <code>os.path.join</code> method using your directory and the filename, as in the code sample below:</p>
<pre><code>os.rename(
    os.path.join(directoryname, str(lijstmetfiles[i])),
    os.path.join(directoryname, str("doggy"+str(i)))
    ) 
</code></pre>
<p>So now you are explicitly specifying what the full path is for your rename.</p>
<p>Another point to make, you don't need to be casting to a "str" for your filenames. Even if your filename is <code>5</code> for example, getting the list of files from <code>listdir</code> will still come back as a string.</p>
<p>Finally, putting it all together, your code should look something like this:</p>
<pre><code>import os
directoryname="C:\\Users\\Ineke\\Documents\\Python Scripts\\images"
lijstmetfiles = os.listdir(directoryname)
print(lijstmetfiles)
for i in range(len(lijstmetfiles)):
    os.rename(
        os.path.join(directoryname, lijstmetfiles[i]),
        os.path.join(directoryname, "doggy"+str(i))
        )
</code></pre>
<p>I tested this on my end and it should work.</p>

<p>Python 2: <a href="https://docs.python.org/2/library/os.html" rel="nofollow">https://docs.python.org/2/library/os.html</a></p>
<p>Python 3: <a href="https://docs.python.org/3/library/os.html" rel="nofollow">https://docs.python.org/3/library/os.html</a></p>
