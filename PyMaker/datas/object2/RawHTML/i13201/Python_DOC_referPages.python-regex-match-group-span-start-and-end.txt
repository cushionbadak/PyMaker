<p>How can I find the span of a inside group by regex?
I have the following code, but I don't know how to get the span (start, end) of the matched group inside the parentheses:</p>
<pre><code>statement = r'new (car)|old (car)'
text = 'I bought a new car and got rid of the old car'
match = re.search(statement, text)
match.span()
Out: (11, 18)
for match in re.finditer(statement, text):
    print match.span()
Out: (11, 18)
Out: (38, 45)
</code></pre>
<p>In this case for example, I only need to match the span of the 'car' not the whole statement.</p>
</div>
<div class="post-text" itemprop="text">
<p>You need to pass <a href="https://docs.python.org/3/library/re.html" rel="noreferrer"><code>span</code></a> an argument:</p>
<pre><code>for match in re.finditer(statement, text):
    print match.span(1)
</code></pre>
<p>1 is referring to the <em>first</em> group, the default is zero - which means the whole match.</p>
