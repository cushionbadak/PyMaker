<p>Here is my code:</p>
<pre><code>text="""col1 col2 col3
a 1 $
b 2 @
c 3 &amp;
"""
mList = []
for line in text.splitlines():
    for item in line.split(" "):
        mList = mList.append(item)
</code></pre>
<p>This raises an error, <code>'NoneType' object has no attribute 'append'</code>. I've tried other ways of creating the list and doing this, but the best I get is the list turned to <code>None</code>. What's going on here?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/2/tutorial/datastructures.html#more-on-lists" rel="nofollow"><code>list.append()</code></a> is an in-place method, it does not return anything ( and hence by default it returns <code>None</code> , as all function <em>calls</em> must return some value, and if some function does not explicitly return anything, the call returns `None). </p>
<p>Hence, when you assign it back to <code>mList</code>, it becomes <code>None</code> and in the next iteration when you do - <code>mList.append()</code> , it errors out as <code>mList</code> is <code>None</code>.</p>
<p>You should try -</p>
<pre><code>mList.append(item)
</code></pre>
<hr/>
<p>Or you can simply do what you are trying to do in a list comprehension -</p>
<pre><code>mList = [item for line in text.splitlines() for item in line.split(" ")]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#mutable-sequence-types" rel="nofollow"><code>list.append</code></a> does not return (= returns <code>None</code>), but just append the item into the list.</p>
<pre><code>&gt;&gt;&gt; lst = []
&gt;&gt;&gt; return_value_of_append = lst.append('something')
&gt;&gt;&gt; return_value_of_append is None
True
</code></pre>
<p>So the following line make the <code>mList</code> become <code>None</code>:</p>
<pre><code>mList = mList.append(item)
</code></pre>
<p>Replacing the above line with the following solve the issue:</p>
<pre><code>mList.append(item)
</code></pre>
