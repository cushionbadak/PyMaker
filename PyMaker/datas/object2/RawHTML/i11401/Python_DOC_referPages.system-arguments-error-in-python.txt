<p>You can use <code>len(sys.argv)</code> to check and give the message of course. But a more pythonic way to do this is using <code>try</code> block</p>
<pre><code>import sys
try 
    filename=sys.argv[0]
    NumoflinesToread=sys.argv[1]
    tmax=sys.argv[2]
except IndexError:
    print Your_error_message
</code></pre>
<p>Finally, I would suggest you to use a more comprehensive module like <a href="https://docs.python.org/2/library/optparse.html" rel="nofollow">optparse</a> or <a href="http://click.pocoo.org/5/" rel="nofollow">Click</a>.</p>
