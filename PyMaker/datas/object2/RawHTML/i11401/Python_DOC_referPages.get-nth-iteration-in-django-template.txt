<p>Your <code>pModel</code> variable is not a queryset or a list, but a <a href="https://docs.python.org/3/library/functions.html#reversed" rel="nofollow">reverse iterator</a>. You cannot access individual elements of an iterator, you can only iterate over the iterator once, exhausting it in the process.</p>
<p>To support access of individual elements, you need to convert <code>pModel</code> to a <a href="https://docs.python.org/3/library/stdtypes.html#sequence-types-list-tuple-range" rel="nofollow">sequence</a>, such as a list:</p>
<pre><code>pModel = list(reversed(PostModel.objects.filter(author = name[0])))
</code></pre>
<p>You can then access the index in your template:</p>
<pre><code>{{ pModel.0.post }}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.djangoproject.com/en/1.9/ref/templates/builtins/#for" rel="nofollow"><code>forloop.counter0</code> template variable</a>. For example, to access the <code>n'th</code> element:</p>
<pre><code>{% for item in pModel %}
    {% if forloop.counter0 == n %}
        {{ item.post }}
    {% endif %}
{% endfor %}
</code></pre>
<p>You can also use <code>first</code> as a special case:</p>
<pre><code>{{ item.first.post }}
</code></pre>
