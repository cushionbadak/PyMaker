<p>If I have a file that contains these dictionaries:</p>
<pre><code>bank = {
    "88386180860608512" : {
        "balance" : 100,
        "name" : "Aehnon"
    },
    "88469127777243136" : {
        "balance" : 100,
        "name" : "Balanceofevil"
    },
    "89741285950246912" : {
        "balance" : 3310,
        "name" : "Sharky"
    }
}
</code></pre>
<p>if this evaluates to True:</p>
<pre><code>b  = [subdict["Name"] for subdict in bank.values()]
if "Sharky" in b:
    return True
else:
    return False
</code></pre>
<p>How can I print the dictionary it belongs to, 89741285950246912?</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#dict.items" rel="nofollow">Iterate items</a> to get both <code>key</code> (89741285950246912) and <code>value</code>:</p>
<pre><code>for account_no, account in bank.items():
    if account['name'] == 'Sharky':
        print(account_no)
</code></pre>
<p>prints:</p>
<pre><code>89741285950246912
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a list comprehensions which returns a list of keys.</p>
<pre><code>&gt;&gt;&gt; bank = {
...     "88386180860608512" : {
...         "balance" : 100,
...         "name" : "Aehnon"
...     },
...     "88469127777243136" : {
...         "balance" : 100,
...         "name" : "Balanceofevil"
...     },
...     "89741285950246912" : {
...         "balance" : 3310,
...         "name" : "Sharky"
...     }
... }
&gt;&gt;&gt; [key for key, value in bank.items() if value['name'] == 'Sharky']
['89741285950246912']
</code></pre>
