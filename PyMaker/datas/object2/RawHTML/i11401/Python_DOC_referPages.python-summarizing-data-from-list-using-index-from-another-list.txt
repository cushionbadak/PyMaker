<p>I have two lists:</p>
<pre><code>L1 = ['A','B','A','C','A']
L2 = [1,   4,  6,  1,  3]
</code></pre>
<p>I want to create a dictionary which has the following output:</p>
<pre><code>DictOutSum = {'A':10, 'B':4, 'C':1}
DictOutCount = {'A':3, 'B':1, 'C':1}
</code></pre>
<p>i.e. Lists L1 and L2 both have same number of elements and the elements in them corresponds one to one. I want to find sum of all numbers in L2 for each unique element in L1 and make a dictionary out of it(DictOutSum). I also want to create another dictionary which stores the counts of number of unique elements of L1(DictOutCount).</p>
<p>I don't even have an idea where to start for this other than to use a for loop.</p>

<p>Pure python implementation:</p>
<pre><code>&gt;&gt;&gt; dict_sum = dict.fromkeys(L1, 0)
&gt;&gt;&gt; dict_count = dict.fromkeys(L1, 0)
&gt;&gt;&gt; for k,n in zip(L1, L2):
...    dict_sum[k] += n
...    dict_count[k] += 1
...     
&gt;&gt;&gt; dict_sum
{'A': 10, 'B': 4, 'C': 1}
&gt;&gt;&gt; dict_count
{'A': 3, 'B': 1, 'C': 1}
</code></pre>
<p>Fancy one-liner implementations:</p>
<pre><code>&gt;&gt;&gt; from collections import Counter
&gt;&gt;&gt; Counter(L1)  # dict_count
Counter({'A': 3, 'B': 1, 'C': 1})
&gt;&gt;&gt; sum((Counter({k:v}) for k,v in zip(L1, L2)), Counter())  # dict_sum
Counter({'A': 10, 'B': 4, 'C': 1})
</code></pre>

<p>The mega elementary way</p>
<pre><code>L1 = ['A','B','A','C','A']
L2 = [1,   4,  6,  1,  3]

# Carries the information
myDict = {}

# Build the dictionary
for x in range(0,len(L1)):

    # Initialize the dictionary IF the key doesn't exist
    if L1[x] not in myDict:
        myDict[L1[x]] = {}
        myDict[L1[x]]['sum'] = 0
        myDict[L1[x]]['count'] = 0

    # Collect the information you need
    myDict[L1[x]][x] = L2[x]
    myDict[L1[x]]['sum'] += L2[x]
    myDict[L1[x]]['count'] += 1

# Build the other two dictionaries
DictOutSum = {}
DictOutCount = {}

# Literally feed the data
for element in myDict:
    DictOutSum[element] = myDict[element]['sum']
    DictOutCount[element] = myDict[element]['count']

print DictOutSum
# {'A': 10, 'C': 1, 'B': 4}

print DictOutCount
# {'A': 3, 'C': 1, 'B': 1}
</code></pre>

<p>Previous answer, DictOutSum, </p>
<pre><code>import itertools
import operator
import functools

DictOutSum  = dict()
for name, group in itertools.groupby(sorted(itertools.izip(L1, L2)), operator.itemgetter(0)):
    DictOutSum[name] = functools.reduce(operator.add, map(operator.itemgetter(1), group))

print(DictOutSum)
{'A': 10, 'C': 1, 'B': 4}
</code></pre>
<p>The main steps are:</p>
<ul>
<li>use <a href="https://docs.python.org/2/library/itertools.html?highlight=izip#itertools.izip" rel="nofollow"><code>itertools.izip</code></a> to make an iterator that aggregates elements from each of <code>L1</code> and <code>L2</code></li>
<li>use <a href="https://docs.python.org/2/library/itertools.html?highlight=groupby#itertools.groupby" rel="nofollow"><code>itertools.groupby</code></a> to make an iterator that returns consecutive keys and groups from the iterable (sorting before that)</li>
<li>use <a href="https://docs.python.org/2/library/functions.html#reduce" rel="nofollow"><code>functools.reduce</code></a> for cumulatively addition</li>
</ul>
