<pre><code>&gt;&gt;&gt; [i for l in list(zip(segments_init[0::3], list(map(lambda x:-x, segments_init[2::3])), [0]*(len(segments_init)//3))) for i in l]

</code></pre>

<p>Using a one-line list comprehension:</p>
<pre><code>&gt;&gt;&gt; segments_init = [1,2,3,4,5,6,7,8,9]

&gt;&gt;&gt;&gt; [j if i%3 == 0 else -segments_init[i+1] if i%3 == 1 else 0 for i, j in enumerate(segments_init)]
[1, -3, 0, 4, -6, 0, 7, -9, 0]
</code></pre>
<p>Alternatively, using <a href="https://docs.python.org/3/reference/datamodel.html#the-standard-type-hierarchy" rel="nofollow">extended slicing</a>:</p>
<pre><code>&gt;&gt;&gt; segments_init = [1,2,3,4,5,6,7,8,9]

&gt;&gt;&gt; segments_new = list(segments_init)
&gt;&gt;&gt; segments_new[1::3] = [-i for i in segments_init[2::3]]
&gt;&gt;&gt; segments_new[2::3] = len(segments_init)//3 * [0]

&gt;&gt;&gt; segments_new
[1, -3, 0, 4, -6, 0, 7, -9, 0]
</code></pre>
