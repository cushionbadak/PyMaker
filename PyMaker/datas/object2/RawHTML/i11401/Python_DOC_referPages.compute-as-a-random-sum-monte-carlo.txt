<div class="post-text" itemprop="text">
<p>You're calling <code>sum</code> on the result of <code>Pi</code>. But <code>Pi</code> returns a number, not an iterable. Just remove the call to <code>sum</code>. <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow"><code>sum</code></a> takes an iterable (list, dictionary, tuple, etc) and adds all the values together. So when you pass it something it can't iterate over (like a floating point number), it can't sum anything, because there's no values to iterate over.</p>
<pre><code>import random

def Pi(N,x):
     S = (1/(N+1))*(1 / (1-x**2) ** 0.5)
     return S

def function(N,a,b):
     x = random.uniform(a,b)
     total = Pi(N,x)
     return total

print(function(1000,0.0,1.0))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I'm pretty sure you want to be running the <code>Pi</code> function <code>N</code> times, not just once. For this you'll need a loop of some sort. Since you want to use the builtin <code>sum</code>, a generator expression would be very natural:</p>
<pre><code>def function(N, a, b):
    total = sum(Pi(N, random.uniform(a, b)) for _ in range(N))
    return total
</code></pre>
<p>This doesn't return anything too close to <code>pi</code> though. Maybe it's <code>pi/2</code>? I don't know how your formula is supposed to work well enough to spot the issue with the calculation.</p>

random.seed(12345)
