<p>Is there a way to "touch" a existing directory in Linux using python so that its modtime becomes the current system time?</p>
<p>From the command line this is the equivalent of <code>touch $directory</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/os.html#os.utime" rel="nofollow"><code>os.utime()</code></a> will allow you to set the atime and mtime of an existing filesystem object, defaulting to the current date and time.</p>
<pre><code>os.utime(path)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do that with <a href="https://docs.python.org/3.5/library/os.html#os.utime" rel="nofollow noreferrer"><code>os.utime</code></a>:</p>
<pre><code>now = time.time()
os.utime('/tmp/marker', (now, now))
</code></pre>
</div>
<span class="comment-copy">Works for py3 but py2, have to specify the atime mtime tuple</span>
<span class="comment-copy">No need to cast now time.time() to int</span>
