<p>I need to make a numbered list from list elements in python. 
Example list:</p>
<pre><code>destinations = ['Los Angeles ', 'Rhodos ', 'Dubai ', 'Manila ', 'Mallorca ', 'New York ']
</code></pre>
<p>I need to print out elements as numbered list:</p>
<pre><code>1. Los Angeles 
2. Rhodos 
3. Dubai 
4. Manila 
5. Mallorca 
6. New York
</code></pre>
<p>If I do:</p>
<pre><code>print ('\n'.join(destinations)) 
</code></pre>
<p>It prints out elements on separate lines, but I cannot add numbers.</p>
</div>
<div class="post-text" itemprop="text">
<p>You simply use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow"><code>enumerate()</code></a> and count from <code>1</code></p>
<pre><code>&gt;&gt;&gt; destinations = ['Los Angeles ', 'Rhodos ', 'Dubai ', 'Manila ', 'Mallorca ', 'New York ']  
&gt;&gt;&gt; for index, value in enumerate(destinations, 1):
...     print("{}. {}".format(index, value))
... 
1. Los Angeles 
2. Rhodos 
3. Dubai 
4. Manila 
5. Mallorca 
6. New York 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>for i, dest in enumerate(destinations, 1):
    print(" %d. %s" % (i, dest))
</code></pre>
