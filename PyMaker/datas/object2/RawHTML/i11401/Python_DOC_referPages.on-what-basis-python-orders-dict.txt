<p>I have simple list of data which looks like:</p>
<pre><code>_MOBILE_CATEGORIES = {
    100152307: 424,
    100152547: 55,
    100152385: 66,
    100152395: 13,
    100152405: 130,
    100152365: 50,
    100152290: 218,
    100152446: 26
}
</code></pre>
<p>When I print it I get:</p>
<blockquote>
<p>{100152385: 66, 100152290: 218, 100152307: 424, 100152405: 130,
  100152547: 55, 100152395: 13, 100152365: 50, 100152446: 26}</p>
</blockquote>
<p>As you cant see first goes <code>100152385: 66</code> which in list is third.</p>
<pre><code>_MOBILE_CATEGORIES = {
    '100152307': 424,
    '100152547': 55,
    '100152385': 66,
    '100152395': 13,
    '100152405': 130,
    '100152365': 50,
    '100152290': 218,
    '100152446': 26
}
</code></pre>
<p>If I define keys as string I get other order like this:</p>
<blockquote>
<p>{'100152290': 218, '100152307': 424, '100152385': 66, '100152446': 26,
  '100152405': 130, '100152395': 13, '100152365': 50, '100152547': 55}</p>
</blockquote>
<p>So I need to define dict so that it would be in the order I need?</p>

<div class="post-text" itemprop="text">
<p>Python dicts are unordered by their own nature. To get the feature you want use an <a href="https://docs.python.org/3/library/collections.html#collections.OrderedDict" rel="nofollow">OrderedDict</a>.</p>
