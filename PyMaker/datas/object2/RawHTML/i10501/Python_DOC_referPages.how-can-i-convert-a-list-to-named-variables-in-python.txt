<p>Is there a quick way in python to convert a list of values to named variables?</p>
<p>For example:</p>
<pre><code>row = ['files', 'file.pdf', 'D1234', 43]      

folder = row[0]
name = row[1]
doc_id = row[2]
page_num = row[3]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists">unpacking</a></p>
<pre><code>folder, name, doc_id, page_num = row
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use this below patterns to do it more professionally</p>
<pre><code>v = ['files', 'file.pdf', 'D1234', 43]    
keys = ['folder', 'name', 'doc_id', 'page_num']
junk = map(lambda k, v: dict.update({k: v}), keys, values)
</code></pre>
