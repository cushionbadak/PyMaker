<p>Try to use the + operator when concatenating strings: <code>'&lt;h5&gt;&lt;a href=/"' + i.split("\t")[4] + '" target=...'</code></p>

<span class="comment-copy">Are you trying to concatenate strings? Yes, this is a syntax error... <code>txt.write('&lt;h5&gt;&lt;a href=/"'i.split("\t")[4]'</code></span>
<span class="comment-copy">I hope the contents of <code>stats</code> are not coming from an untrusted source (say: a random visitor to your web page). :-]</span>
<span class="comment-copy">you cannot concatenate strings like that. there are many ways to do it but try <code>+</code> between terms like: <code>'&lt;h5&gt;&lt;a href=/"' + i.split("\t")[4] + ...</code>. Enclosing a variable in quotes does not convert it to string. Also look at this (<a href="https://docs.python.org/3/library/string.html" rel="nofollow noreferrer">docs.python.org/3/library/string.html</a>)</span>
