<div class="post-text" itemprop="text">
<p>"If a component is an absolute path, all previous components are thrown away and joining continues from the absolute path component." applies here: <code>STATIC_URL</code> is an absolute path because it starts with <code>/</code>, so <code>BASE_DIR</code> is dropped.</p>
<p>Drop the leading <code>/</code> else dirname thinks that <code>STATIC_URL</code> is absolute and keeps only that.</p>
<pre><code>BASE_DIR = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
STATIC_URL = 'static/'
</code></pre>
