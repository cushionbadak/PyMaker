<p>Here is the list:</p>
<ul>
<li><code>[[('mobile','VB')],[('margin','NN')],[('and','CC')]]</code></li>
</ul>
<p>But I want to remove <code>[]</code> from list. Output should be:</p>
<ul>
<li><code>[('mobile','VB'),('margin','NN'),('and','CC')]</code></li>
</ul>
</div>

<pre><code>L = [[('mobile','VB')],[('margin','NN')],[('and','CC')]]
R = [ x[0] for x in L ]
</code></pre>

<div class="post-text" itemprop="text">
<p>One solution is to iterate over the list and take the first item inside and add it to a new list. </p>
<pre><code>In [1] _list = [[('mobile','VB')],[('margin','NN')],[('and','CC')]]
In [2] cleaned = []
In [3] for item in _list: cleaned.append(item[0])

In [4] print(cleaned)
Out[4] [('mobile', 'VB'), ('margin', 'NN'), ('and', 'CC')]
</code></pre>
</div>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/itertools.html?highlight=itertools#itertools.chain" rel="nofollow">itertools.chain</a> :</p>
<pre><code>import itertools
def unlist_items(list1):
    return list(itertools.chain(*list1))

list1 = [[('mobile','VB')],[('margin','NN')],[('and','CC')] ]
print( unlist_items( list1 ) ) # prints [('mobile', 'VB'), ('margin', 'NN'), ('and', 'CC')]
</code></pre>
