<div class="post-text" itemprop="text">
<p>I want to calculate the difference (in terms of days, hour, min, sec, microsecond) between two time stamps (having microsecond resolution) in python e.g. between (1)<code>2016-05-10 11:32:18.025635</code> and (2) <code>2016-05-13 18:59:52.094672</code></p>
<p>Any pointers, how to do it?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://pypi.python.org/pypi/python-dateutil/2.5.3" rel="nofollow"><code>python-dateutil</code></a> package:</p>
<pre><code>&gt;&gt;&gt; import dateutil.parser
&gt;&gt;&gt; d1 = dateutil.parser.parse('2016-05-10 11:32:18.025635')
&gt;&gt;&gt; d2 = dateutil.parser.parse('2016-05-13 18:59:52.094672')
&gt;&gt;&gt; d2 - d1
datetime.timedelta(3, 26854, 69037)
</code></pre>
<p>This will give you a <a href="https://docs.python.org/3/library/datetime.html#timedelta-objects" rel="nofollow"><code>datetime.timedelta</code></a> object which represents the time difference between the two datetime objects.</p>
<p>If you don't want to install a package then it can also be done using <a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow"><code>datetime.strptime</code></a> but in that case you have to define what the string looks like.</p>
