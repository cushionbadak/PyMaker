<div class="post-text" itemprop="text">
<p>I've installed Python 3.5 and 2.7 side by side on a Windows machine. Rather than messing around with my <code>PATH</code>, I'm using the Python Launcher to call different Python versions, for instance <code>py -2</code> if I want to use Python 2. My question is: how do I call the <code>pip</code> executable for that installation?</p>
</div>
<div class="post-text" itemprop="text">
<p>You have to start <code>pip</code> as a module like</p>
<pre><code>py -2 -m pip install virtualenv
</code></pre>
<p>Actually if you do want to mess around with python-environments (like installing conflicting libraries for the same python-version) you should take a look a <a href="https://virtualenv.pypa.io/en/stable/">virtualenv</a> or <a href="https://docs.python.org/3/library/venv.html">venv</a></p>
</div>
