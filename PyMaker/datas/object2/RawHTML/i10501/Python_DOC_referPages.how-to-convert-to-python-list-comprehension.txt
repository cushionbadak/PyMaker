<div class="post-text" itemprop="text">
<p>For your example where you have either strings or integers or floats, you can use a simple list comprehension:</p>
<pre><code>ls = ['abc', 56, 49, 63, 66, 80]
print([str(l) for l in ls])
</code></pre>
<p>Or alternatively, you can use <a href="https://docs.python.org/3/library/functions.html#map" rel="noreferrer"><code>map</code></a>:</p>
<pre><code>print(list(map(str, ls)))
</code></pre>
<p>If you only want to convert floats and integers to strings (and ignore anything else like booleans, decimal objects, etc.):</p>
<pre><code>print([str(l) if isinstance(l, int) or isinstance(l, float) else l for l in ls])
</code></pre>
