<div class="post-text" itemprop="text">
<p><code>file</code> should be a file object, not a file name. File objects have <code>write</code> method, <code>str</code> objects don't.</p>
<p>From the doc on <a href="http://docs.python.org/3/library/functions.html#print"><code>print</code></a>:</p>
<blockquote>
<p>The <em>file</em> argument must be an object with a <code>write(string)</code> method; if it
  is not present or <code>None</code>, <code>sys.stdout</code> will be used.</p>
</blockquote>
<p>Also note that the file should be open for writing:</p>
<pre><code>with open('redata.txt', 'w') as redata: # note that it will overwrite old content
    for i in range(randrange(5,11)):
        ...
        print('...', file=redata)
</code></pre>
<p>See more about the <code>open</code> function <a href="http://docs.python.org/3/library/functions.html#open">here</a>.</p>
