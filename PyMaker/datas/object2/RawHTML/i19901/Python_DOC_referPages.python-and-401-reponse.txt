<p>You're looking for the Keyword <strong>Basic HTTP Authentication</strong>.</p>
<p>I do not recommend using 3rd party modules if you won't be going further than doing that. If you will, the <code>requests</code> library already suggested is a great choice.</p>
<p>The following example is taken from <a href="http://docs.python.org/2/howto/urllib2.html" rel="nofollow">the urllib2 docs</a>:</p>
<pre><code>import urllib2
# create a password manager
password_mgr = urllib2.HTTPPasswordMgrWithDefaultRealm()

# Add the username and password.
# If we knew the realm, we could use it instead of None.
top_level_url = "http://example.com/foo/"
password_mgr.add_password(None, top_level_url, username, password)

handler = urllib2.HTTPBasicAuthHandler(password_mgr)

# create "opener" (OpenerDirector instance)
opener = urllib2.build_opener(handler)

# use the opener to fetch a URL
opener.open(a_url)

# Install the opener.
# Now all calls to urllib2.urlopen use our opener.
urllib2.install_opener(opener)
</code></pre>

<p>There are three good options here.</p>
<p>First, you can use <a href="http://docs.python.org/2/library/urllib2.html" rel="nofollow"><code>urllib2</code></a> (Python 2) or <a href="http://docs.python.org/3/library/urllib.html" rel="nofollow"><code>urllib</code></a> (Python 3), which are built in, and pretty easy to use.</p>
<p>Second, you can use an even easier third-party library like <a href="http://pypi.python.org/pypi/requests" rel="nofollow"><code>requests</code></a>. (Often, code that takes a dozen lines to write with <code>curl</code> or <code>urllib</code> is a two-liner with <code>requests</code>.)</p>
<p>Finally, since you already know how to use php's low-level libcurl wrappers, there are a few different third-party alternatives for Python that are nearly identical. See <a href="http://pypi.python.org/pypi?%3aaction=search&amp;term=curl&amp;submit=search" rel="nofollow">this search</a>, and look through <code>pycurl</code>, <code>pycurl2</code>, and <code>pyclibcurl</code> to see which one feels most familiar.</p>
