<p>Everything is working fine, <code>b'{'</code> is just another way of writing <code>b'\x7b'</code>:</p>
<pre><code>&gt;&gt;&gt; b'{' == b'\x7b'
True
</code></pre>
<p>You can create a character string of only escapes with the following helper method:</p>
<pre><code>import binascii
def to_byte_escapes(b):
   return ''.join('\\x' + binascii.hexlify(byte) for byte in b)
</code></pre>
<p>Then you'll get:</p>
<pre><code>&gt;&gt;&gt; print(to_byte_escapes(b'\xde{\x1d\x86\xa0\x81kw\xbc\x8a'))
\xde\x7b\x1d\x86\xa0\x81\x6b\x77\xbc\x8a
</code></pre>
