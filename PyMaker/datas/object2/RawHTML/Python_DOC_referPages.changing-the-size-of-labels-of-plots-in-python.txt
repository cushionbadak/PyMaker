<div class="post-text" itemprop="text">
<p>you can use the <code>plot.legend(loc=2, prop={'size': 6})</code> to increase the legend size This takes a dictionary of keywords corresponding to <code>matplotlib.font_manager.FontProperties</code> properties. <a href="https://matplotlib.org/api/pyplot_api.html#matplotlib.pyplot.legend" rel="nofollow noreferrer">more about legends</a></p>
<p>1). if you want to increase the size of the plotting data according to x values this would be helpful.</p>
<p><code># yvalues is the y value list
widthscale = len(yvalues)/4 
figsize = (8*widthscale,6) # fig size in inches (width,height)
figure = pylab.figure(figsize = figsize) # set the figsize</code></p>
<p>if you want increase them without dynamically you can use <a href="https://matplotlib.org/api/_as_gen/matplotlib.pyplot.rc.html" rel="nofollow noreferrer">plot.rc</a> function
eg.</p>
<pre><code>import matplotlib.pyplot as plt

SMALL_SIZE = 8
MEDIUM_SIZE = 10
BIGGER_SIZE = 12

plt.rc('font', size=SMALL_SIZE)          # controls default text sizes
plt.rc('axes', titlesize=SMALL_SIZE)     # fontsize of the axes title
plt.rc('axes', labelsize=MEDIUM_SIZE)    # fontsize of the x and y labels
plt.rc('xtick', labelsize=SMALL_SIZE)    # fontsize of the tick labels
plt.rc('ytick', labelsize=SMALL_SIZE)    # fontsize of the tick labels
plt.rc('legend', fontsize=SMALL_SIZE)    # legend fontsize
plt.rc('figure', titlesize=BIGGER_SIZE)  # fontsize of the figure title
</code></pre>
<p>2).second option would be </p>
<pre><code>plt.rcParams["axes.labelsize"] = 22
</code></pre>
<p> or directly control the size of the label </p>
<pre><code>ax.set_xlabel("some label", fontsize=22)
</code></pre>
<p>To control the legend's fontsize you can use rcParams </p>
<pre><code>plt.rcParams["legend.fontsize"] = 22
</code></pre>
<p> or directly specify the size in the legend </p>
<pre><code>ax.legend(fontsize=22)
</code></pre>
