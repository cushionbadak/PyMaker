<p>It is pretty simple. You can check if the first char of the string is <code></code>. See below example</p>
<pre><code>sentences = ['  this wont print', 'this will print', "   this wont print either", " this will print"]
for sentence in sentences:
    if sentence.startswith('  '):
        print(sentence)
</code></pre>
