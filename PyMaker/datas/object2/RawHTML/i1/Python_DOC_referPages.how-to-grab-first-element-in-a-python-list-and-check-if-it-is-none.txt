<h1>TL;DR</h1>
<ul>
<li><p><code>a[0] is None</code>
returns <code>True</code> if the first element in <code>a</code> is <code>None</code>.</p></li>
<li><p><code>a.pop(0) is None</code>
returns <code>True</code> if the first element in <code>a</code> is <code>None</code> <strong>and removes that element</strong> from <code>a</code>.</p></li>
</ul>
<h1>Grabbing the first element in a <code>list</code></h1>
<p>You can access the first element in a <code>list</code> by specifying <code>0</code> as an index:</p>
<pre><code>&gt;&gt;&gt; x = [1,2,3]
&gt;&gt;&gt; x[0]
1
</code></pre>
<p>Specifying higher index numbers gives you the other elements, or an error if that index doesn't exist:</p>
<pre><code>&gt;&gt;&gt; x = [1,2,3]
&gt;&gt;&gt; x[0]
1
&gt;&gt;&gt; x[1]
2
&gt;&gt;&gt; x[2]
3
&gt;&gt;&gt; x[3]
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
IndexError: list index out of range
</code></pre>
<p>Alternatively, you can simultaneously retrieve and remove an element from the <code>list</code> by using <code>pop()</code>:</p>
<pre><code>&gt;&gt;&gt; x = [1,2,3]
&gt;&gt;&gt; x.pop()
3
&gt;&gt;&gt; x
[1, 2]
</code></pre>
<p>Note that, without an argument, <code>pop()</code> returns the last element of the <code>list</code>, and removes it from the <code>list</code>.</p>
<p>With an argument (which is an index number), it will return (and remove) the specified element:</p>
<pre><code>&gt;&gt;&gt; x = [1,2,3]
&gt;&gt;&gt; x.pop(0)
1
&gt;&gt;&gt; x
[2, 3]
</code></pre>
<h1>Checking if a value is <code>None</code></h1>
<p>Simply use the <code>is None</code> comparison:</p>
<pre><code>&gt;&gt;&gt; x = [1,2,3]
&gt;&gt;&gt; x[0] is None
False
&gt;&gt;&gt; a = [None, None, 1234, 'Apple', 1]
&gt;&gt;&gt; b = [None, 4321, 'Cheese', 1]
&gt;&gt;&gt; a[0] is None
True
&gt;&gt;&gt; b[0] is None
True
&gt;&gt;&gt; a[0] is None and b[0] is None
True
</code></pre>
<p>Using <code>pop()</code>, which <strong>will remove the specified element</strong> from the <code>list</code>:</p>
<pre><code>&gt;&gt;&gt; a = [None, None, 1234, 'Apple', 1]
&gt;&gt;&gt; b = [None, 4321, 'Cheese', 1]
&gt;&gt;&gt; a.pop(0) is None and b.pop(0) is None
True
&gt;&gt;&gt; a
[None, 1234, 'Apple', 1]
&gt;&gt;&gt; b
[4321, 'Cheese', 1]
</code></pre>
<h1>Bonus: negative indices</h1>
<pre><code>&gt;&gt;&gt; x = [1,2,3]
&gt;&gt;&gt; x[-1]
3
&gt;&gt;&gt; x[-2]
2
&gt;&gt;&gt; x[-3]
1
&gt;&gt;&gt; x[-4]
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
IndexError: list index out of range
</code></pre>
<p>A roundabout way of getting the first element. Not practical, just for fun:</p>
<pre><code>&gt;&gt;&gt; x = [1,2,3]
&gt;&gt;&gt; x[len(x) * -1]
1
</code></pre>
