<p>Remove will only eliminate the first instance of the element, if you wanna keep the line as a string try this:</p>
<pre><code>import os
num= ''

<p>You can use the following code:</p>
<pre><code>with open('3data.txt','r') as fp:
  for line in fp:
    line = line.translate(None,'][') #no translation matrix, just deletion
    elems = line.split() #separator is space
    elems = [x if x !='0' else '' for x in elems] #replace the '0' by empty string ''
    print(elems) #print the newly generated lines
</code></pre>
<p>input:</p>
<pre><code>[10 20 30 40 0 5 3 21 0]
</code></pre>
<p>output:</p>
<pre><code>['10', '20', '30', '40', '', '5', '3', '21', '']
</code></pre>
<p>explanations:</p>
<ul>
<li>You open the file </li>
<li>You read each line of the file, remove the brackets via the translate methode</li>
<li>Split each line into the different numbers</li>
<li>repack the elems and replace all '0' by ''</li>
<li>print the new line</li>
</ul>
