<p>You can simply decode the byte string to <code>utf</code> (or the encoding you're using) using the below:</p>
<pre><code>example_byte_string = b'this is a byte string'
example_utf8_string = example_byte_string.decode("utf-8") 
</code></pre>
<p>or you could simply use <code>str(b'bytes')</code> - The default encoding is <code>utf-8</code> as shown in the definition below:</p>
<pre><code>class str(object=b'', encoding='utf-8', errors='strict')
</code></pre>
