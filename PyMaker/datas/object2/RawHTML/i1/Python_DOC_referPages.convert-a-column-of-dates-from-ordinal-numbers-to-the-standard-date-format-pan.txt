<p>I have to convert a column of dates from the integer/date format to the date format d-m-Y. Example:</p>
<pre><code>import pandas as pd
col1 = [737346, 737346, 737346, 737346, 737059, 737346]
col2 = ['cod1', 'cod2', 'cod3', 'cod4', 'cod1', 'cod2']
dict = {'V1' : col1, 'V2' : col2}   
df = pd.DataFrame.from_dict(dict)

df

       V1    V2
0  737346  cod1
1  737346  cod2
2  737346  cod3
3  737346  cod4
4  737059  cod1
5  737346  cod2
</code></pre>
<p>expected:</p>
<pre><code>df
           V1    V2
0  14-10-2019  cod1
1  14-10-2019  cod2
2  14-10-2019  cod3
3  14-10-2019  cod4
4  31-12-2018  cod1
5  14-10-2019  cod2
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>datetime <code>fromordinal</code> should help.</p>
<pre><code>import datetime as dt

col1 = [737346, 737346, 737346, 737346, 737059, 737346]
col2 = ['cod1', 'cod2', 'cod3', 'cod4', 'cod1', 'cod2']
dd = {'V1' : col1, 'V2' : col2}   
df = pd.DataFrame.from_dict(dd)

df['V1'] = df['V1'].apply(dt.datetime.fromordinal)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Just <code>pandas</code> <code>Timestamp.fromordinal</code></p>
<pre><code>df.V1.map(pd.Timestamp.fromordinal)

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer"><code>date.fromordinal</code></a> for this.</p>
<pre><code>from datetime import datetime as dt

df['V1'] = df.V1.apply(lambda x: dt.fromordinal(x)).dt.strftime('%d-%m-%Y')

print(df)
           V1    V2
0  14-10-2019  cod1
1  14-10-2019  cod2
2  14-10-2019  cod3
3  14-10-2019  cod4
4  31-12-2018  cod1
5  14-10-2019  cod2
</code></pre>
</div>
