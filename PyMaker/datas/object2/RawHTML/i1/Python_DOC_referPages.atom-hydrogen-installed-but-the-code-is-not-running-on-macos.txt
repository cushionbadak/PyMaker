<div class="post-text" itemprop="text">
<p>Your kernelspec argv argument is pointing to <code>../python.3x/bin</code> instead of directly to the <code>python3</code> executable. Pointing to the executable directly with <code>../python.3x/bin/python3</code> is more typical and might be the issue.</p>
<p>How did you install your kernel? I'd recommend taking a look at <a href="https://nteract.io/kernels/python" rel="nofollow noreferrer">this kernel installation guide</a> and probably reinstalling your kernel with, in your case, <code>python3 -m ipykernel install</code>. I think this will work, but, manually editing the kernel.json file would be another (not recommended) way to fix this.</p>
