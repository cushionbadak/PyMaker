<div class="post-text" itemprop="text">
<p>Remove sudo. ie</p>
<pre><code>python3 Documents/Replicator/dfp/dfp-rep/manage.py runserver &amp;
</code></pre>
<p>create a bash script with this. Say it is /root/run.sh</p>
<p>chmod +x run.sh</p>
<p>edit your /etc/rc.local
and add:
/root/run.sh</p>
<p>before the "exit 0" entry in the file.</p>
<p>Reboot should start your server.</p>
<p>However, it is not best practice. Use uwsgi to run the service. </p>
