<div class="post-text" itemprop="text">
<p>I want to iterate through a numpy ndarray and, if any values are less than X, replace one of them with X.</p>
<p>I have tried doing <code>array_name[ array_name &lt; X] = X</code> but this replaces all of the values that are less than X.</p>
<p>I can use a for loop, but I feel like there's probably a more concise way already bundled with numpy.</p>
<pre><code>        for i in array_name:
            if i &lt; X:
                i = X
                break
</code></pre>
<p>Is there a way to do this more elegantly?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>array_name &lt; X
</code></pre>
<p>Returns same array but with True or False. Then you can just pick an index where the cell is True</p>
<pre><code>idx = np.argwhere(array_name &lt; X)[i]
array_name[idx] = value
</code></pre>
<p>Here, you can choose <code>i</code> arbitrarily</p>
</div>
<span class="comment-copy">What are you doing with <code>i</code> there?</span>
<span class="comment-copy">Any index in <code>np.argwhere(array_name &lt; X)</code> satisfies your constraint. So <code>i</code> is one of them</span>
