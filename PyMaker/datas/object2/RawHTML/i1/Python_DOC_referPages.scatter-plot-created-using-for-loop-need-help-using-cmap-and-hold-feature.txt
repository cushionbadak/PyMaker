<p>From the <a href="https://matplotlib.org/api/_as_gen/matplotlib.pyplot.scatter.html" rel="nofollow noreferrer">docs</a>:</p>
<blockquote>
<p>cmap : Colormap, optional, default: None<br/>
  A Colormap instance or registered colormap name. <strong>cmap is only used if c is an array of floats.</strong> If None, defaults to rc image.cmap.</p>
</blockquote>
<p>If I understood correctly what you want, try to edit the line:</p>
<pre><code>ax1.scatter(xdata, ydata, c=i, cmap='bwr')
</code></pre>
<p>in</p>
<pre><code>ax1.scatter(xdata, ydata, c=xdata, cmap='bwr')
</code></pre>
<p>and should work.</p>
<p>Here a very simple sample code to show it:</p>
<pre><code>import matplotlib.pyplot as plt
import numpy as np

fig1, ax1 = plt.subplots()
xdata = np.linspace(1, 10, 100)
ydata = xdata**2

plt.scatter(xdata, ydata, c=xdata, cmap='bwr')
ax1.set_xlabel('x-axis')
ax1.set_ylabel('y-axis')
fig1.savefig('plot.jpg')
</code></pre>
