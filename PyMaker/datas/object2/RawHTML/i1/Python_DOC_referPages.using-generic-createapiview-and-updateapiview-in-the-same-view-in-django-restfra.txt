<div class="post-text" itemprop="text">
<p>DRF not providing a <strong>CreateUpdateAPIView</strong> class out of the box. But we can make it with less effort</p>
<pre><code>from rest_framework import generics


class CreateUpdateAPIView(generics.CreateAPIView,
                          generics.UpdateAPIView,
                          generics.GenericAPIView):
    pass</code></pre>
<p><br/>Here the <strong><code>CreateUpdateAPIView</code></strong> class is similar to DRF's <code>ListCreateAPIView</code>,<code>RetrieveUpdateAPIView</code> etc</p>
<p>Now, you can use this <strong><code>CreateUpdateAPIView</code></strong> in your views, as </p>
<pre><code>class QuestionList(<b>CreateUpdateAPIView</b>):
    queryset = Question.objects.all()
    serializer_class = QuestionSerializer</code></pre>
