<div class="post-text" itemprop="text">
<p>You have to implement <code>__str__</code> in your model if you want to render a model instance like this</p>
<pre><code>class Tld(models.Model):
    def __str__(self):
        # return something meaningful here like self.some_attribute
</code></pre>
<p>Or you have to explicitly use an attribute in template like this</p>
<pre><code> {{ authdomain.some_attribute }}
</code></pre>
<p>Otherwise python does not know how to get something meaningful out of an object to show. </p>
</div>
<div class="post-text" itemprop="text">
<p>It's because you are trying to display the whole object, not the value.</p>
<p>You need to replace</p>
<pre><code>{{ notauthdomain }}
</code></pre>
<p>with </p>
<pre><code>{{ notauthdomain.name }} 
</code></pre>
<p>replace 'name' with the field name of your value - websiteabc.com</p>
