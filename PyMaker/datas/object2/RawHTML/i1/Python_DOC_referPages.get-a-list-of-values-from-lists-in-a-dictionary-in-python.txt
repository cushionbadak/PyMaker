<div class="post-text" itemprop="text">
<p>I created a dictionary in the following format with keys and a list of attributes associated with each key:</p>
<pre><code>inputDatasets = {
    "data1": ["Path to data 1", "Attribute1", "Attribute2", "Attribute3"],
    "data2": ["Path to data 2", "Attribute1", "Attribute2", "Attribute3"],
}
</code></pre>
<p>I would like to get a list of all the "Attribute2" contained in the dictionary. For now I use:</p>
<pre><code>attr2 = []
for a, b in inputDatasets.items():
    attr2.append(b[2])
</code></pre>
<p>But is there a more elegant way to do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You could use a list comprehension, looping only through the values instead of key-value pairs:</p>
<pre><code>attr2 = [b[2] for b in inputDatasets.values()]
</code></pre>
<p>Alternatively, but not as elegantly, a <code>map</code> with a <code>lambda</code> could be useful:</p>
<pre><code>attr2 = list(map(lambda b:b[2], inputDatasets.values()))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can also do it as follows</p>
<pre><code>attr2 = [v[2] for v in inputDatasets.values()]
</code></pre>
</div>
