<p>Try the below code wherein you set content, fruit and its number. This is an alternative way rather than creating multiple classes and initializing them in the SuperMarket class.</p>
<pre><code>class SuperMarket:

    def __init__(self):
        print 'Initializing fruit data --- '

    def setContent(self, content):
        self.content = content
        return self

    def setFruit(self, fruit):
        self.fruit = fruit
        return self

    def setNumber(self, number):
        self.number = number
        return self

a = SuperMarket().setContent(1).setFruit('banana').setNumber(10)

print a.content #1
print a.fruit #banana
print a.number #10
</code></pre>
