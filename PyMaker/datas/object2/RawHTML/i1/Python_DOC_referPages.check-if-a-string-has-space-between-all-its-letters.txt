<p>You can split the string into a list on spaces, then find out the max length of each space in a single command. If every character is spaced out, the maximum length of each element in the list will be one. If there are multiple letters together, the max will be &gt; 1.</p>
<p><code>len(max(teststring.split(' '), key=len))</code></p>
<p>So simply this becomes:</p>
<pre><code>if len(max(teststring.split(' '), key=len)) == 1:
    print("is separated")
else:
    print("is not separated")
</code></pre>

<div class="post-text" itemprop="text">
<p>Split the string on spaces, then check the length of each component.</p>
<pre><code>demo='a b c d ef'
if all([len(item)&lt;2 for item in demo.split(' ')]):
    print('is separated')
else:
    print('not separated')
</code></pre>
<p><code>demo.split(' ')</code> splits the list on a space. giving <code>['a','b','c','d','ef']</code></p>
<p><code>[len(item)&lt;2 for item in demo.split(' ')]</code> runs through the above list and checks the length of each component. It returns True if the length is less than 2 and False otherwise. For this example, this gives <code>[True,True,True,True,False]</code></p>
<p><code>all([len(item)&lt;2 for item in demo.split(' ')])</code> uses all() to check if all the booleans in the list are True. It returns true if everything is True, and False otherwise. </p>
