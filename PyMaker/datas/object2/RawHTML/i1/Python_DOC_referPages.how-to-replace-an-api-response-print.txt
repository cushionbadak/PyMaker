<div class="post-text" itemprop="text">
<p>Once you've done <code>r.json</code>, rather than having a json string like</p>
<pre><code>"""{"success": true}"""
</code></pre>
<p>You have a Python dictionary like:</p>
<pre><code>{"success": True}
</code></pre>
<p>That means that <code>data['success']</code> will be a boolean, not a string. You can replace it conditionally if you like:</p>
<pre><code>sadboi = "on_true" if data['success'] else 'on_false'
</code></pre>
</div>
