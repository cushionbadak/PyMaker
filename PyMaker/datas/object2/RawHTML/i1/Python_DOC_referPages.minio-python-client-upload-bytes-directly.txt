<p>I read the minio docs and I see two methods to upload data:</p>
<ul>
<li><a href="https://docs.minio.io/docs/python-client-api-reference.html#put_object" rel="nofollow noreferrer">put_object()</a> this needs a io-stream</li>
<li><a href="https://docs.minio.io/docs/python-client-api-reference.html#fput_object" rel="nofollow noreferrer">fput_object()</a> this reads a file on disk</li>
</ul>
<p>I want to test minio and upload some data I just created with <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.random.bytes.html" rel="nofollow noreferrer">numpy.random.bytes()</a>.</p>
<p>How to upload data which is stored in a variable in the python interpreter?</p>
</div>
<div class="post-text" itemprop="text">
<p>Take a look at <a href="https://docs.python.org/3/library/io.html#binary-i-o" rel="nofollow noreferrer"><code>io.BytesIO</code></a>.  These allow you to wrap byte arrays up in a stream which you can give to minio.</p>
<p>For example:</p>
<pre class="lang-py prettyprint-override"><code>import io
from minio import Minio

value = "Some text I want to upload"
value_as_bytes = value.encode('utf-8')
value_as_a_stream = io.BytesIO(value_as_bytes)

client = Minio("my-url-here", ...) # Edit this bit to connect to your Minio server
client.put_object("my_bucket", "my_key", value_as_a_stream , file_size=len(value_as_bytes))
</code></pre>
