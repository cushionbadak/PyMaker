<p>The <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/" rel="nofollow noreferrer">BeautifulSoup</a> module in Python can easily traverse though any sort of XML-looking code like this.</p>
<p>After putting your code inside of a string I've named <code>pagecode</code>, I ran this to extract what was in the fourth row, third column:</p>
<pre><code>from bs4 import BeautifulSoup
soup = BeautifulSoup(pagecode, 'lxml')
rows = soup.find_all("wf:row")
cell = rows[3].find_all("wf:c")[2]  # Indexing starts at 0, not 1!
print(cell)  # Displays &lt;wf:c tfmt="1"&gt;&lt;/wf:c&gt;
</code></pre>
