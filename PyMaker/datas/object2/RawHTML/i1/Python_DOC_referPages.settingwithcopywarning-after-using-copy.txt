<div class="post-text" itemprop="text">
<p>I think you need <code>copy</code> because <a href="http://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.select_dtypes.html" rel="nofollow noreferrer"><code>DataFrame.select_dtypes</code></a> is slicing operation, filtering by types of column, check <a href="https://stackoverflow.com/a/53954986/2901002">Question 3</a>:</p>
<pre><code>df_numeric=df.select_dtypes(include='number').copy()
df_non_numeric=df.select_dtypes(exclude='number').copy()
</code></pre>
<p>If you modify values in <code>df_non_numeric</code> later you will find that the modifications do not propagate back to the original data (<code>df</code>), and that Pandas does warning.</p>
