<div class="post-text" itemprop="text">
<p>Try adding a comma as per the docs<br/>
<a href="https://matplotlib.org/tutorials/intermediate/legend_guide.html#legend-handlers" rel="nofollow noreferrer">https://matplotlib.org/tutorials/intermediate/legend_guide.html#legend-handlers</a></p>
<pre><code>d1, = plt.plot(xdata, ydata1, "sb", markersize = 5, ls = "solid", label = 'name1', linestyle = 'dashed')
d2, = plt.plot(xdata, ydata2, "vr", markersize = 5, ls = "solid", label = 'name2', linestyle = 'dashed')

plt.legend(handles=[d1, d2])
</code></pre>
</div>
<span class="comment-copy"><code>d1</code> and <code>d2</code> are lists (of one element each). You could directly add those, <code>plt.legend(handles=d1+d2)</code>.</span>
