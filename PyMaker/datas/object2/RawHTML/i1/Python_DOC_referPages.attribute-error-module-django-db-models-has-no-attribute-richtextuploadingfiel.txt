<div class="post-text" itemprop="text">
<p>You are using:  <code>models.RichTextUploadingField()</code> that is not actually an model module Class, you are importing it from another library, so you should use just <code>body = RichTextUploadingField()</code>.</p>
</div>
<span class="comment-copy">Here's the error: <code>body = models.RichTextUploadingField()</code>. Change this to <code>body = RichTextUploadingField()</code>.</span>
