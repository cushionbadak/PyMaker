<div class="post-text" itemprop="text">
<p>Given what little code you've supplied, I'm guessing you're already in trouble because of this line:</p>
<pre><code>t=turtle.pen()
</code></pre>
<p>What you most likely want is:</p>
<pre><code>t = turtle.Pen()
</code></pre>
<p>Letter case matters as the lowercase <code>pen()</code> function returns a list of attributes about the (default) pen whereas the uppercase <code>Pen()</code> function creates a new pen (turtle) that you can manipulate.</p>
<p>Python is a case sensitive language, so this is why that matters. </p>
