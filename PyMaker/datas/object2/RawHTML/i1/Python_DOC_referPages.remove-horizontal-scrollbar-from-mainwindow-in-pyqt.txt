<div class="post-text" itemprop="text">
<p>If you just want to delete the Scrollbar you must use:</p>
<pre><code>{QtableWidget}.setVerticalScrollBarPolicy(QtCore.Qt.ScrollBarAlwaysOff)
{QtableWidget}.setHorizontalScrollBarPolicy(QtCore.Qt.ScrollBarAlwaysOff)
</code></pre>
<p>If you want to show the expanded <code>QTableWidget</code>, add this to the end of the <code>setData()</code> method: </p>
<pre><code>self.setMaximumSize(self.getQTableWidgetSize())
self.setMinimumSize(self.getQTableWidgetSize())
</code></pre>
<p>and define <code>getQTableWidgetSize(self)</code> like this: </p>
<pre><code>def getQTableWidgetSize(self):
    w = self.verticalHeader().width() + 4  # +4 seems to be needed
    for i in range(self.columnCount()):
        w += self.columnWidth(i)  # seems to include gridline (on my machine)
    h = self.horizontalHeader().height() + 4
    for i in range(self.rowCount()):
        h += self.rowHeight(i)
    return QtCore.QSize(w, h)
</code></pre>

<p><strong>Note:</strong>
The function <code>getQTableWidgetSize</code> is a conversion of the code in C ++ to python of the following post: <a href="https://stackoverflow.com/questions/8766633/how-to-determine-the-correct-size-of-a-qtablewidget">How to determine the correct size of a QTableWidget?</a></p>
