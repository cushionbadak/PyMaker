<p>I created this solution:</p>
<pre><code>names = df.Status.tolist()
i=0
for colname in enumerate(names):
    print(colname)
    df.set_value(i, 'Click here', '=HYPERLINK("[pandas_multiple.xlsx]'+colname[1]+'!A1","Link")')
    i=i+1
</code></pre>
<p>In this case, I am reading the column Status and creating another column called 'Click here'. the HYPERLINK, is directing from the sheets with the same name of the colname[1], for example Error and Running</p>
