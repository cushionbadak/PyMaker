<div class="post-text" itemprop="text">
<p>say I have a matrix : <code>a = [[1,2,3],[4,5,6],[7,8,9]]</code>. How can I combine it to <code>b = [1,2,3,4,5,6,7,8,9]</code>?</p>
<p>Many thanks</p>
</div>
<div class="post-text" itemprop="text">
<p>Using numpy:</p>
<pre><code>import numpy
a = [[1,2,3],[4,5,6],[7,8,9]]
b = numpy.hstack(a)
list(b)
[1, 2, 3, 4, 5, 6, 7, 8, 9]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Maybe this is not the most beautiful, but it works:</p>
<pre><code>a = [[1,2,3],[4,5,6],[7,8,9]]
b = [sub_thing for thing in a for sub_thing in thing]
print(b)
</code></pre>
<p>Prints:</p>
<blockquote>
<p>[1, 2, 3, 4, 5, 6, 7, 8, 9]</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>Without using numpy:</p>
<pre><code>#make the empty list b
b=[]
for row in a:#go trough the matrix a
    for value in row: #for every value
        b.append(value) #python is fun and easy
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Another way to combine integer matrix could be using the <code>itertools</code> <code>chain</code>.</p>
<pre><code>a = [[1,2,3],[4,5,6],[7,8,9]]
list(itertools.chain.from_iterable(a)
</code></pre>
<p>prints:</p>
<blockquote>
<p>[1, 2, 3, 4, 5, 6, 7, 8, 9]</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>using numpy: </p>
<p><code>list(np.array(a).flatten())</code></p>
