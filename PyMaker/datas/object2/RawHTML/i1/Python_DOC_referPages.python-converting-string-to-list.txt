<p>I am sending data between python socket connection. The data is an array and I am sending it in blocks of each line. When the data comes in it looks like this:</p>
<pre><code>data = '[1,0,1]'
</code></pre>
<p>I want to convert this back into array form so I can then add it back into my master array that is being sent over. Any idea how I can convert this string back into an array.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>import json

data = '[1,0,1]'

data = json.loads(data)
</code></pre>
<p>Out:</p>
<pre><code>[1, 0, 1]
</code></pre>

<span class="comment-copy"><code>data = '[1,0,1]'</code> is not an array, it is a list. You should be sending data in some sort of serialized format, not just calling <code>str</code> on your object and pretending it is serialization.</span>
