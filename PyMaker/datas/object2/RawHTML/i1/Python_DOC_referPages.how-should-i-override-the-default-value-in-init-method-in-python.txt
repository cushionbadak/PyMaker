<div class="post-text" itemprop="text">
<p>So, one of your classes is inherited from another one. And when we're talking about default value in <code>__init__</code> - that means the value, which will be used when no specific value was provided. For instance:</p>
<pre class="lang-py prettyprint-override"><code>google_auth = GoogleAuth()  # default value for `settings_file` will be used
google_auth = GoogleAuth(http_timeout=23)  # default value for `settings_file` will be used
google_auth = GoogleAuth(settings_file='specific_one.yaml')  # specified one will be used instead of default
...
</code></pre>
<p>And most likely your base class <code>ApiAttributeMixin</code> has its own default value for <code>settings_file</code> parameter. So you want to override that default value with another one for your subclass <code>GoogleAuth</code>.</p>
<p>Let's say you want <code>google_settings.yaml</code> as the default for <code>settings_file</code> parameter in your <code>GoogleAuth</code> class. The code will look like the following:</p>
<pre class="lang-py prettyprint-override"><code>class ApiAttributeMixin(object):

    def __init__(self, settings_file='settings.yaml', http_timeout=None):
        pass


class GoogleAuth(ApiAttributeMixin, object):

    def __init__(self, settings_file='google_settings.yaml', http_timeout=None):
        super(GoogleAuth, self).__init__(settings_file, http_timeout)
</code></pre>
