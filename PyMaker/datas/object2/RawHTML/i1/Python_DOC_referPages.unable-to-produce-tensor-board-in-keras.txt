<div class="post-text" itemprop="text">
<p>It looks like the problem is the definition of the log_dir. Basically, TensorBoard is expecting a directory name not a file name. </p>
<p>You can make sure this works by using os library:</p>
<pre><code>dir = "logs/{}".format(time())
if not os.path.isdir(dir):
    os.mkdir(dir)
</code></pre>
