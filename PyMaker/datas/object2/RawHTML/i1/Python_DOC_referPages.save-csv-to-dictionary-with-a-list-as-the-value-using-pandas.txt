<p>My CSV file looks something like this:</p>
<pre><code>Version,1,2,3,4,5
Letter,A,B,C,D,E
</code></pre>
<p>Version and letter are both the keys I want in the dictionary which I read in using indexcol = 0. However, I want to save these values into a dictionary like this: </p>
<pre><code>{Version: ["1,2,3,4,5"], Letter: ["A,B,C,D,E"]}
</code></pre>
<p>How would I do this using pandas?
Thanks so much for your help.</p>

<div class="post-text" itemprop="text">
<p>Since you have the keys in rows, one simple way is to transpose the csv file and then converting columns to <code>list</code> followed by <code>zip</code> and to dictionary</p>
<pre><code>df = pandas.read_csv("filename.csv").transpose()
dictionary = dict(zip(list(df.column_name1), list(df.column_name2)))
</code></pre>
<p>Other efficient way and if you have too many values in rows to enter each column name then you can try:</p>
<pre><code>df = pandas.read_csv("filename.csv").transpose().to_dict()
</code></pre>
