<div class="post-text" itemprop="text">
<p>You need to parse the file as json (using <code>json.load</code>/<code>json.loads</code>), not just read it. If you just read the file you get a <code>str</code> with the entire content, and if you iterate on that it just iterates on individual codepoints.</p>
<p>Incidentally your iteration won't necessarily work as Python iterates dicts on keys only (where other languages iterate them by (key, value) pairs).</p>
