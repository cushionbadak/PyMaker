<p>I expected it to print just the number, but it looks like its returning an empty string for each item in the string that doesn't have a match. </p>

<div class="post-text" itemprop="text">
<p><code>*</code> means "0 or more". So it matches against all positions, whether or not there is a digit.</p>
<p>To only match 1 or more digits, use <code>+</code>:</p>
<pre><code>&gt;&gt;&gt; re.findall(r"\d+", string)
['10427645']
</code></pre>
