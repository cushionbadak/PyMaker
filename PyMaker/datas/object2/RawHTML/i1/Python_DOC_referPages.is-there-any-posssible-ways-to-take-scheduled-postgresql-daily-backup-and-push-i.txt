<p>I need to take backup of postgresql db in every day and push it into S3 bucket. is there any solution for that?</p>
<p>i am using ubuntu 16.04 and postgresql version is 9.5</p>

<div class="post-text" itemprop="text">
<p>Using the <a href="https://github.com/django-dbbackup/django-dbbackup" rel="nofollow noreferrer">django-dbbackup</a> package is probably your easiest option.
<a href="https://django-dbbackup.readthedocs.io/en/stable/storage.html#amazon-s3" rel="nofollow noreferrer">It supports backups to S3</a>.</p>
<p>You can then run a daily cron job using <a href="https://github.com/Tivix/django-cron" rel="nofollow noreferrer">django-cron</a> as instructed <a href="https://django-dbbackup.readthedocs.io/en/stable/integration.html?highlight=cron#django-cron" rel="nofollow noreferrer">here</a>.</p>
