<p>Is there a way to format certain cells within a column using pandas? The only formatting I have been able to find is align the entire column but I would like to have certain rows indented and others fully left aligned (example below). Any help would be greatly appreciated.</p>
<pre><code>Category         Value
Total Revenue    2.00
  Widget 1       1.00
  Widget 2       1.00
Total Expenses   2.00
  Widget 1       1.00
  Widget 2       1.00
</code></pre>

<div class="post-text" itemprop="text">
<p>Assuming you want to add a tab in the strings named Widget, I would go for something like:</p>
<p>-'<em>df</em>' is your dataframe</p>
<p>-'<em>df['Category']</em>' is the column you want to replace for a new one with the tabulated strings</p>
<p>-Creating a new column</p>
<p><code>new_col = ["".join(["\t",i]) if ("Widget" in i) else i for i in df['Category']]</code></p>
<p>-Now replacing old column with new column</p>
<p><code>df['Category']= new_col</code></p>
<p>If you don't want to see the tabulation, you can use spaces("  ") instead of tabulation ("\t"). So:</p>
<p><code>new_col = ["".join(["  ",i]) if ("Widget" in i) else i for i in df['Category']]</code></p>
<p><code>df['Category']= new_col</code></p>
<p>Hope it helps.</p>
