<p>You just access the list you want and then append to that</p>
<pre><code>Dict1['CorrectionHistory'][0]['CorrectionsAll'].append({'CorrChngDesc': 'Discount Line Changed'})
</code></pre>
<p>Outputs:</p>
<blockquote>
<p>{'OrigSic': 'LJB', 'PickupDate': '2016/01/01 00:00:00', 'DestSic': 'XCF', 'ShpmtID': 123, 'CorrectionHistory': [{'key': 405013, 'CorrectionsAll': [{'CorrChngDesc': 'Commodity Line Changed'}, {'CorrChngDesc': 'Discount Line Changed'}]}], 'EstimatedDeliveryDate': '2016/01/10 00:00:00'}</p>
</blockquote>
