<div class="post-text" itemprop="text">
<p>You've defined <code>loop_regions(output_file)</code> but you're calling it like <code>loop_regions()</code>. Doing something like <code>loop_regions(initialize())</code> will make it work or assigning the result of <code>initialize</code> to a variable and passing that into <code>loop_regions</code>.</p>
<p>Just note that the fact you named two different things <code>output_file</code> doesn't bind them together.</p>
