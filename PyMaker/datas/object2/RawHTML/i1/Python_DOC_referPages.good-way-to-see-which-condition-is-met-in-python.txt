<div class="post-text" itemprop="text">
<p>Simply this:</p>
<pre><code>check_list = [s &lt; fixed_number for s in sample]
</code></pre>
<p>if you really care having 0s and 1s, use <code>int(s &lt; fixed_number)</code>.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>check_list = []
sample = [1, 4, 7]
fixed_number = 5

for i in sample:
    if i &lt; fixed_number:
        check_list.append(1)
    else:
        check_list.append(0)

print(check_list)
</code></pre>
<p>You can use a loop to check the condition at once instead of calling every time.</p>
