<p>is the 'input ()' function used to input an integer? And the raw_input () function is used to input a string? </p>
</div>
<div class="post-text" itemprop="text">
<p><code>input</code> evaluates the input, while <code>raw_input</code> does not.</p>
<p>For example, capturing the <code>input</code> <code>5</code>, returns the <code>int</code> <code>5</code>. However, capturing the <code>raw_input</code> <code>5</code>, returns the <code>str</code> <code>'5'</code></p>
