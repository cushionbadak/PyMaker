<div class="post-text" itemprop="text">
<p>You can use, df.T as below:</p>
<pre><code>df = pd.DataFrame({'name': ['countryCode', 'vatNumber','requestDate', 'valid', 'name', 'address']})

df = df.T
df

             0        1          2            3          4        5
name    countryCode vatNumber   requestDate valid      name    address

df.columns = df.values[0]
df = df.reset_index(drop = True)
df


    countryCode vatNumber   requestDate valid   name    address
0   countryCode vatNumber   requestDate valid   name    address
</code></pre>
<p>Then replace all non null values with np.nan:</p>
<pre><code>df[df.notna()] = np.nan
df
countryCode vatNumber   requestDate valid   name    address
0   NaN        NaN          NaN       NaN   NaN      NaN
</code></pre>
