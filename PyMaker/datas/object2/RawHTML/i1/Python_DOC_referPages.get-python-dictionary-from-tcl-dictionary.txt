<p>I want to get a python dictionary from a Tcl dictionary in a single tk.call. Is this somehow possible?</p>
<p>Example:</p>
<pre><code>import tkinter

desiredDict =  {"first": "Foo", "second": "Bar", "third": "Baz"}

tk = tkinter.Tcl()
tk.call("set", "data(first)", "Foo" )
tk.call("set", "data(second)", "Bar" )
tk.call("set", "data(third)", "Baz" )
foo = tk.call("array", "get", "data" )
tclKeys =  tk.call("dict", "keys", foo)
fromTcl  = tk.call("dict", "get", foo, "first")

print(foo)
print(tclKeys)
print(fromTcl)
print(type(foo))
# print(dir(foo))
</code></pre>
<p>I  know I can get the keys with <code>tk.call("dict", "keys", foo)</code> and then every single value with <code>tk.call("dict", "get", foo, "...")</code> but I want to get the Python dictionary (see <code>desiredDict</code>) in one single <code>tk.call</code>. This is no gui problem, I'm not working wit a gui here.</p>
</div>
<div class="post-text" itemprop="text">
<p>There's no public function in Tkinter for retrieving a Tcl dict, but there is a private one:</p>
<pre><code>&gt;&gt;&gt; tkinter._splitdict(tk, foo)
{'second': 'Bar', 'third': 'Baz', 'first': 'Foo'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I don't know where the relevant documentation is but if you wanted to go with Python and  the methods you presented, this does work:</p>
<pre><code>keys = desiredDict.keys()
d = dict(zip(keys, (tk.call("dict", "get", foo, key) for key in keys)))
assert d == desiredDict
</code></pre>
