<div class="post-text" itemprop="text">
<p>You need to set <code>highlightthickness</code> to zero as well.</p>
<pre><code>self.c1 = tk.Canvas(..., highlightthickness=0)
</code></pre>
<p>From the <a href="http://effbot.org/tkinterbook/canvas.htm" rel="nofollow noreferrer">canvas page of effbot</a> <code>highlightthickness</code> explained as:</p>
<blockquote>
<p>The width of the highlight border. The default is system specific
  (usually one or two pixels). (highlightThickness/HighlightThickness)</p>
</blockquote>
</div>
