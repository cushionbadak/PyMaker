<p>I have an array of matrices all of different sizes/dimensions and I need to pad them all to the same size (805, 16866). Bellow is the code I'm currently using to do this but I believe it could be vectorised, however I am not sure how to do this. </p>
<pre><code>x1 = np.zeros((805, 16866))

for i in range(x[0].shape[0]):
    for j in range(x[0].shape[1]):
        x1[i, j] = x[0][i, j]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>In the above example, <code>x1[:x[0].shape[0], :x[0].shape[1]] = x</code> should work</p>
</div>
<div class="post-text" itemprop="text">
<p>There's the <code>numpy.pad</code> function <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.pad.html" rel="nofollow noreferrer">(docs)</a> which does what you need.</p>
<p>The code you want is</p>
<pre><code>x1 = np.zeros((805,16866))

for i in range(x.shape[0])
  x1[i,:]= np.pad(x[i,:],x1.shape[0],'constant',constant_values=0)
</code></pre>
</div>
<span class="comment-copy">You can use <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.pad.html" rel="nofollow noreferrer"><code>np.pad</code></a>.</span>
