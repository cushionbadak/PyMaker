<p>You could achieve this with <a href="https://docs.scipy.org/doc/numpy/reference/arrays.indexing.html" rel="nofollow noreferrer">Basic slicing and indexing</a>.</p>
<pre><code>import numpy as np

u=1.2
d=0.8
n=3

array=np.zeros(n)

for i in range(n):
    if i + 1 &lt;= n:
        array[i:n]=u**(n-i-1)
        array[i:n] =d**(i)

</code></pre>
