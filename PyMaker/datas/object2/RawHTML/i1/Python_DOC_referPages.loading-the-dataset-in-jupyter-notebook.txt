<p><code>def fetch_housing_data()</code> is not called, so there is no directory or downloaded files. 
you need to call <code>fetch_housing_data()</code> in the body of <code>def load_housing_data</code></p>
<p>like this:</p>
<pre><code>def load_housing_data(housing_path=HOUSING_PATH):
    # missing function call to fetch the data
    fetch_housing_data()
    csv_path = os.path.join(housing_path, "housing.csv")
    return pd.read_csv(csv_path)
