<div class="post-text" itemprop="text">
<p>What you are looking for is <code>plt.xticks()</code>:</p>
<pre><code>import numpy as np
import matplotlib.pyplot as plt

data = np.random.normal(60, 13, 500)

plt.hist(data, orientation = 'horizontal')
plt.grid()
plt.axis([0, 150, 0, 120])
plt.xticks(np.arange(0,151,25))

plt.savefig('HISTOGRAM.png')
</code></pre>
<p>There you can specify where to put the ticks. Same for y-axis.</p>

<div class="post-text" itemprop="text">
<p>A cure for your OCD</p>
<pre><code>import numpy as np
import matplotlib.pyplot as plt

data = np.random.normal(60, 13, 500)

plt.hist(data, orientation = 'horizontal')
plt.grid()
plt.xticks(np.linspace(0,150,16))
plt.axis([0, 150, 0, 120])
plt.savefig('HISTOGRAM.png')
</code></pre>
<p>basically <code>plt.xticks</code> and <code>plt.yticks</code> accept lists as input and use them as markers on the x axis and y axis respectively, <code>np.linspace</code> generates an array with start,stop and number of points.</p>
