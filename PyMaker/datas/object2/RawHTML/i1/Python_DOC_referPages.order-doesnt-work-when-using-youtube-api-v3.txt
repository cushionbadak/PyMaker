<p>You can use a workaround. Instead using the <code>search.list</code> request, retrieve the uploaded videos.</p>
<p>By using the <code>channel_id</code> change the letter as is explained here:</p>
<blockquote>
<p><code>channel_id</code>: U<strong>C</strong>q-Fj5jknLsUf-MWSy4_brA</p>
<p><code>upload_playlist_id</code>: U<strong>U</strong>q-Fj5jknLsUf-MWSy4_brA</p>
</blockquote>
<p>Use the <a href="https://developers.google.com/youtube/v3/docs/playlistItems/list" rel="nofollow noreferrer"><code>PlaylistItems.list</code></a> request for retrieve the uploaded videos from a given channel.</p>
<p>This is the URL request:</p>
<pre><code>https://www.googleapis.com/youtube/v3/playlistItems?part=snippet%2CcontentDetails&amp;playlistId=&lt;UPLOAD_PLAYLIST_ID&gt;&amp;fields=items(contentDetails(videoId%2CvideoPublishedAt)%2Csnippet(position%2Ctitle)%2Cstatus)&amp;key={YOUR_API_KEY}
</code></pre>
