<div class="post-text" itemprop="text">
<p>I <code>'/Users/aleksandrpavlenko/Documents/Development/prepandas'</code> is your working directory from where you are executing <code>python -m site</code> command. Python by default appends current working directory in sys.path that's why you are seeing this.</p>
<p>You do not need to delete it or uninstall anything for making your project run smoothly. Python use this path to let you import packages from your current (project) directory.</p>
