<div class="post-text" itemprop="text">
<p>You can't pass in your js variable like that. Try changing the way you pass your data. Considering you have a trailing forward slash (i.e. <code>/</code>) in your url.</p>
<p>Change this:</p>
<pre><code>function Myfunction(li){
    src1="&lt;iframe src='{% url 'threadcount1' li %}' width='2000' height='1000';&gt;&lt;/iframe&gt;";
    document.getElementById('frame').innerHTML = src1 ;
}
</code></pre>
<p>To this:</p>
<pre><code>function Myfunction(li){
    src1 = "&lt;iframe src='{% url 'threadcount1' %}" + li + "' width='2000' height='1000';&gt;&lt;/iframe&gt;";
    document.getElementById('frame').innerHTML = src1 ;
}
</code></pre>
