<p>Thank You, </p>
<p>That did the trick, </p>
<p>I installed azure module which came with graphrbac of version 0.40</p>
<pre><code>$pip freeze | grep rbac
  azure-graphrbac==0.40.0
</code></pre>
<p>Upon checking <a href="https://pypi.org/project/azure-graphrbac/" rel="nofollow noreferrer">azure-graphrbac</a> current version is 0.60    </p>
<p>I had to uninstall azure-graphrbac alone and reinstall the current version using pip. That solved the issue.</p>
<p>Although it throwed an error that its not compatible with azure module, I don't see that as an issue for now.</p>
<pre><code>azure 4.0.0 has requirement azure-graphrbac~=0.40.0, but you'll have azure-graphrbac 0.60.0 which is incompatible.
</code></pre>
