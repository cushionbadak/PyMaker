<p>Your element's <code>id</code> is not <code>"post-18448"</code>, it's <code>post-18448</code>. In your first example, you're arbitrarily adding double quotes to the string you're matching on. When you're defining it manually, you've removed those.</p>
<p>Drop the double quotes from your setting of <code>z</code>:</p>
<pre><code>z = 'post-'+me
</code></pre>
