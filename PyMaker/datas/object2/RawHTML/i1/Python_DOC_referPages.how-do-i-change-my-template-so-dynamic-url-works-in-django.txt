<p>If this mean url for <strong>details</strong> page of the product :</p>
<pre><code>path('products/&lt;int:my_id&gt;/', views.product_detail, name='product_detail'),
</code></pre>
<p>You should add url for <strong>list</strong> of product : </p>
<pre><code>path('products/', views.product_list, name='product_list'),
</code></pre>
<p>Then fix your products view for better understanding and name it <strong>product_detail</strong> view :</p>
<pre><code>def product_detail(request, my_id, *args):
    obj = Product.objects.get(id=my_id)
    custom_context = {
        'title': 'product_detail',
        'object': obj
    }
    return render(request, 'product_detail.html', custom_context)
</code></pre>
<p>After you have to create your <strong>product_list</strong> view like :</p>
<pre><code>def product_list(request, *args):
    object_list = Product.objects.all()
    custom_context = {
        'title': 'product_list',
        'object_list': object_list
    }
    return render(request, 'product_list.html', custom_context)
</code></pre>
<p>And for urls in template you have to call them like that: </p>
<pre><code>&lt;a href="{% url 'product_list' %}"&gt;Product List&lt;/a&gt;
&lt;a href="{% url 'product_detail' my_id=object.id %}"&gt;Product Details for product {{ object.id }} &lt;/a&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to have two separate urls and views, one for the general products listing page (for all products), and the other for the specific product page.</p>
<pre><code>path('products/', views.products, name='products'),
path('product/(?P&lt;id&gt;\d+)/', views.product, name='product'),
</code></pre>
