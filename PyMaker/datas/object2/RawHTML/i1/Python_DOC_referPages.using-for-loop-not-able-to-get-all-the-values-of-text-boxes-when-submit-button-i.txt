<p>Actually you are creating 3 submit buttons in each row but 2 of them are covered by the text boxes, so you can only see the last submit button.  That is why you get only the content of last text box.  In order to print all the content of text boxes in a row, you only need to create one submit button in each row and pass the list of text boxes to the command function as below:</p>
<pre><code>from tkinter import *

root = Tk()

def display_text(wlist):
    for w in wlist:
        print(w.get(1.0, END))
    print(x,y)

for x in range(4):
    wlist = []  # hold the text boxes in current row
    for y in range(3):
        w = Text(root, width=15, height=2)
        w.grid(row=x,column=y)
        w.insert(END,'')
        wlist.append(w)
    b=Button(root,text="Submit",command=lambda wl=wlist: display_text(wl))
    b.grid(row=x, column=y+1)

root.minsize(400, 400)
root.mainloop()
</code></pre>
