<div class="post-text" itemprop="text">
<p>If you are starting the program from a folder that is not 'master', Python is most likely not finding rpSubmit because it searches for the 'scripts' folder in this folder (where you started the program), instead of in the 'master' folder.</p>
<p>To remedy this, change the <code>from scripts.rpSubmit import rpSubmit</code> line in menu.py to:</p>
<pre><code>import imp
rpSubmit = imp.load_source('rpSubmit', r'Z:\ZMM\00_pipeline\pipeline\nuke\master\scripts\rpSubmit\rpSubmit.py')
</code></pre>
<p>This way, you set up an absolute path to the right script.</p>
