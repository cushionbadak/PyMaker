<p>This is what I use. Hope it helps.</p>
<pre><code>import firebase_admin
from firebase_admin import credentials
from firebase_admin import storage

# Fetch the service account key JSON file contents
cred = credentials.Certificate("credentials.json")

# Initialize the app with a service account, granting admin privileges
app = firebase_admin.initialize_app(cred, {
    'storageBucket': '&lt;BUCKET_NAME&gt;.appspot.com',
}, name='storage')

bucket = storage.bucket(app=app)
blob = bucket.blob("&lt;your_blob_path&gt;")

print(blob.generate_signed_url(datetime.timedelta(seconds=300), method='GET'))
</code></pre>
<p>It generates a <strong>public URL</strong> (for 300 secs) for you to download your files. </p>
<p>For example, in my case, I use that URL to display stored pictures in my django website with <code>&lt;img&gt;</code> tag.</p>
<p><a href="https://googleapis.github.io/google-cloud-python/latest/storage/blobs.html#google.cloud.storage.blob.Blob" rel="nofollow noreferrer">Here</a> is the doc for more usefull functions.</p>
