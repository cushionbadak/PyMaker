<p>The error is already pointing you to the right direction.</p>
<pre><code># Your case
array([1,2], dtype=float32)

# Output
NameError: name 'array' is not defined


#Correct case
np.array([1,2], dtype='float32')

# Output
array([1., 2.], dtype=float32)
</code></pre>
<p>So two things are wrong in your case <code>array</code> should be <code>np.array</code> and <code>float32</code> should be <code>'float32'</code></p>
<p><strong>Plus</strong><br/>
You cannot pass <code>pd.DataFrame</code> a 1d array. So in this case you can use <code>pd.Series</code>. Heres your code fixed:</p>
