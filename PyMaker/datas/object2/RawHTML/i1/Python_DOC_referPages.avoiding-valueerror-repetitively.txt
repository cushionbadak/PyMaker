<p>Try this :</p>
<pre><code>def meh():
  try:
    n=int(input("\nEnter any number : "))
  except ValueError:
    print('ERROR!! Kindly enter a number only!')
    meh()
</code></pre>
<p>Previously, when the function got the non-integer type string second time, it would throw the same <code>ValueError</code> again as there is no other <code>try</code> block in <code>except</code> part. So, just call the function again in the <code>except</code> block with an dummy error printing before.</p>

<p>I agree to other answers but enclosing your code in try/except is not a good idea.</p>
<pre><code>def meh():
    n = input("Enter any number:")
    if not n.isdigit():
        raise ValueError('ERROR!! Kindly enter a number only!')
    return int(n)
</code></pre>
<p>Then you can call it like</p>
<pre><code>if __name__ == "__main__":
    while True:
        try:
            j=meh()
            print ('You entered ', j)
            break
        except ValueError as e:
            print (e)
            continue
</code></pre>
