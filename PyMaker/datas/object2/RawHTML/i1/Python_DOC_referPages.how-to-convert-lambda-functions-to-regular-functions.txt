<div class="post-text" itemprop="text">
<pre><code>gpdf = lambda B0, mu, sigma2: 1/np.sqrt(2*pi*sigma2)*np.exp(-1/2*((B0-mu)**2)/sigma2)
</code></pre>
<p>is equivalent to:</p>
<pre><code>def gpdf(B0, mu, sigma2):
    return 1/np.sqrt(2*pi*sigma2)*np.exp(-1/2*((B0-mu)**2)/sigma2)
</code></pre>
<p>In both cases you call it in the same way with <code>gpdf(B0, mu, sigma2)</code> so it shouldn't make a difference. </p>
