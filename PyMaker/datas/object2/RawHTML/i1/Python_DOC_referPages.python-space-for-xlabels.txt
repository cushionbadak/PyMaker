<div class="post-text" itemprop="text">
<p>You could use <code>plt.tight_layout()</code> it is used for exactly such problems. Additionally if its automatic algorithm to distribute everything nicely on the canvas, it as a kwarg <code>rect</code>, which needs a list of <code>[xmin, ymin, xmax, ymax]</code> values defining the space of the figure to be used for the (sub)plot(s), e.g.</p>
<pre><code>plt.tight_layout(rect=[0, .1, 1, .95)
</code></pre>
<p>It should be called at the end of the plot creation, right before <code>plt.show()</code> if your script needs the latter.</p>
