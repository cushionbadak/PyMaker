<p>There are some other ways to install pip</p>
<p><strong>Pip install</strong></p>
<p>For pip install in the safe side, we can run some command to prepare our ubuntu
In Ubuntu open terminal (<kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>T</kbd>)</p>
<pre><code>sudo apt-get update
sudo apt install build-essential python-dev python3-dev python-setuptools
</code></pre>
<p>Once the update is complete the pip install is very easy for the latest Ubuntu. Only you can run the following code in a terminal</p>
<pre><code>sudo apt-get install python-pip
</code></pre>
<p>if need pip install specific python3</p>
<pre><code>sudo apt-get install python3-pip
</code></pre>
<p>Pip installed successfully.</p>
<p><strong>Pip install with curl</strong></p>
<p>An alternative way to install pip: 
It is another easy way to install pip by using Curl and python.</p>
<pre><code>sudo apt-get update
curl "https://bootstrap.pypa.io/get-pip.py" -o "get-pip.py"
python get-pip.py
</code></pre>
<p>Now verify the pip installation:
Write the pip on the terminal. You can see some codes.</p>
<p>If you want to check the version of</p>
<p><code>pip -V</code></p>
