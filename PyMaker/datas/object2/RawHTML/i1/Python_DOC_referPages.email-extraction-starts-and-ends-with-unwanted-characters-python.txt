<p>So because I don't have your original text, I'm just going to use a string from your example.</p>
<p>See if the following two regexes might work for you. I also include a third which is a bit more precise.</p>
<p><code>'(?&lt;=\dU)[\w]+@[\w\.]+?(?=U|\s|$)'</code></p>
<p>.</p>
<p><code>'(?&lt;=\dU)[\w]+@[\w]+\.[\w]+?(?=U|\s|$)'</code></p>
<p>.</p>
<p><strong>Example Tests</strong></p>
<pre><code>&gt;&gt;&gt; import re


&gt;&gt;&gt; string = '''3465Usjohnson@astate.eduUProvost instead of sjohnson@astate.edu The surround text that it is being extracted from: 870-972-3465Usjohnson@astate.eduUProvost and Vice ChancellorDr. Lynita Cooksey870-972-2 030 870-972-2036Ulcooksey@astate.edu'''


&gt;&gt;&gt; re.findall('(?&lt;=\dU)[\w]+@[\w\.]+?(?=U|\s|$)', string)

#Output
['sjohnson@astate.edu', 'sjohnson@astate.edu', 'lcooksey@astate.edu']




&gt;&gt;&gt; re.findall('(?&lt;=\dU)[\w]+@[\w]+\.[\w]+?(?=U|\s|$)', string)

#Output
['sjohnson@astate.edu', 'sjohnson@astate.edu', 'lcooksey@astate.edu']
</code></pre>
<p>.</p>
<p><strong>To be a bit more precise, because the emails all end in <code>.edu</code></strong></p>
<p><code>'(?&lt;=\dU)[\w]+@[\w]*\.edu(?=U|\s|$)'</code></p>
<p>.</p>
<p><strong>Example Test</strong></p>
<pre><code>&gt;&gt;&gt; string = '''3465Usjohnson@astate.eduUProvost instead of sjohnson@astate.edu The surround text that it is being extracted from: 870-972-3465Usjohnson@astate.eduUProvost and Vice ChancellorDr. Lynita Cooksey870-972-2 030 870-972-2036Ulcooksey@astate.edu'''


&gt;&gt;&gt; re.findall('(?&lt;=\dU)[\w]+@[\w]*\.edu(?=U|\s|$)', string)

#Output
['sjohnson@astate.edu', 'sjohnson@astate.edu', 'lcooksey@astate.edu']
</code></pre>
