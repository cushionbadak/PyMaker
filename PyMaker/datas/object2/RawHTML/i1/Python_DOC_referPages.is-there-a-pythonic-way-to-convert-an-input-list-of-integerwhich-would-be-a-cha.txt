<p>I'm trying to take a single line of intergers and using a for loop to convert the elements into integers, Is there a more pyhtonic way of doing so??</p>
<pre><code>a = input().strip().split()
l = []
for i in a:
    l.append(int(i))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can do <code>map</code> directly on <code>split</code>:</p>
<pre><code>l = list(map(int, input().strip().split()))
</code></pre>
</div>
<span class="comment-copy">if you don't mind using NumPy, you can use <code>l = np.array(input().strip().split(), dtype=int)</code></span>
