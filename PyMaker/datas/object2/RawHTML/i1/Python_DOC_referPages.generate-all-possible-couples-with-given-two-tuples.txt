<p>I need to build a function in python that inputs two tuples and returns all the possible pairs for example. </p>
<p><code>def mult_tuple(tuple1,tuple2):</code> </p>
<p>let the tuples be: </p>
<p><code>first_tuple = (1,2) 
 second_tuple = (4,5)</code></p>
<p>and the output will be:</p>
<p><code>((1,4),(4,1),(1,5),(5,1),(2,4),(4,2),(2,5),(5,2))</code> </p>
<p><strong>Note</strong>: The input tuples can be of any size. For eg: <code>(1,4) and (8,2,9,6)</code></p>

<div class="post-text" itemprop="text">
<p>you can use <code>product</code> from <code>itertools</code></p>
<pre><code>import itertools

for element in itertools.product((1,2),(3,4,5,6)):
    print(element, (element[1],element[0]))
</code></pre>
<p>and output will be:</p>
<pre><code>(1, 3) (3, 1)
(1, 4) (4, 1)
(1, 5) (5, 1)
(1, 6) (6, 1)
(2, 3) (3, 2)
(2, 4) (4, 2)
(2, 5) (5, 2)
(2, 6) (6, 2)
</code></pre>
<p>so your func will be for example</p>
<pre><code>def mult_tuple(tuple1,tuple2):
  for element in itertools.product(tuple1, tuple2):
    yield element
    yield (element[1],element[0])
</code></pre>
