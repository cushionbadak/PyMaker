<div class="post-text" itemprop="text">
<p>You can access the <code>&lt;attribute id="8"&gt;act1&lt;/attribute&gt;</code> with </p>
<pre><code>box.find('attribute')
</code></pre>
<p>To get the act you use:</p>
<pre><code>&gt;&gt;&gt;box.find('attribute').text
act1 # or act2
</code></pre>
<p>The python docs are a very good resource: <a href="https://docs.python.org/2/library/xml.etree.elementtree.html" rel="nofollow noreferrer">https://docs.python.org/2/library/xml.etree.elementtree.html</a></p>
</div>
<div class="post-text" itemprop="text">
<p>You can also use array notation which could be useful if you are scraping or putting it in a massive loop</p>
<p><code>root[0][0][0].text</code></p>
<p><code>root[0][0][1].text</code>
and so on</p>
