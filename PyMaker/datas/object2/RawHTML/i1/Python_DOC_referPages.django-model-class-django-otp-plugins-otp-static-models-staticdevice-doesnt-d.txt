<div class="post-text" itemprop="text">
<p>I think some process you miss. You did not add the <code>allauth_2fa</code> in INSTALLED_APPS and not migrate it. I try to write the full process its configuration.</p>
<p><strong>Step 1 :</strong> </p>
<p><code>pip install django-allauth-2fa</code></p>
<p><strong>Step 2 :</strong> </p>
<p>apply those in settings.py file</p>
<pre><code>INSTALLED_APPS = (
    # Required by allauth.
    'django.contrib.sites',

    # Configure Django auth package.
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',

    # Enable allauth.
    'allauth',
    'allauth.account',

    # Configure the django-otp package.
    'django_otp',
    'django_otp.plugins.otp_totp',
    'django_otp.plugins.otp_static',

    # Enable two-factor auth.
    'allauth_2fa',
)
</code></pre>
<p><strong>Step 3:</strong></p>
<pre><code>MIDDLEWARE_CLASSES = (
    # Configure Django auth package.
    'django.contrib.auth.middleware.AuthenticationMiddleware',

    # Configure the django-otp package. Note this must be after the
    # AuthenticationMiddleware.
    'django_otp.middleware.OTPMiddleware',

    # Reset login flow middleware. If this middleware is included, the login
    # flow is reset if another page is loaded between login and successfully
    # entering two-factor credentials.
    'allauth_2fa.middleware.AllauthTwoFactorMiddleware',
)
# Set the allauth adapter to be the 2FA adapter.
ACCOUNT_ADAPTER = 'allauth_2fa.adapter.OTPAdapter'
</code></pre>
<p><strong>Step 4:</strong></p>
<pre><code>python manage.py migrate
</code></pre>
<p><strong>Step 5 :</strong></p>
<p>Now you can import it in your views file
also must configure in urls.py file</p>
<p>If face any problem comment below</p>
