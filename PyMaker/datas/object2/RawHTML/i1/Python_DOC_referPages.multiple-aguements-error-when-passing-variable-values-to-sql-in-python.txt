<div class="post-text" itemprop="text">
<p>If you look at the documentation for <code>executescript</code> you'll see it takes a single argument: the script to run. You're passing it two arguments: a script to run and a tuple. <code>executescript</code> does not take parameter substitution the way some other database methods do.</p>
<p>The solution is to use <code>execute</code> rather than <code>executescript</code>.</p>
</div>
