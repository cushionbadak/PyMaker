<div class="post-text" itemprop="text">
<p>What is the conventional way to convert python list of lists to PyTorch tensors?</p>
<pre><code>a = [0,0]
b = [1,1]
c = [2]
c = [a, b, c]
</code></pre>
<p>I want c to be converted to a flattened Torch tensor as below:</p>
<pre><code>torch([ 0, 0, 1, 1, 2])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can flatten your list first in Python:</p>
<pre><code>flat_list = [item for sublist in c for item in sublist]
</code></pre>
<p>And create your Tensor:</p>
<pre><code>flattened_tensor = torch.FloatTensor(flat_list)
</code></pre>
</div>
