<p>Just set the color of the polygons to be the same:</p>
<pre><code>import matplotlib.pyplot as plt

x = [4, 1, 2]
y = [1, 2, 1]
z = [0, 2, 1]

plt.fill(x, y, z, c='C0')
plt.show()
</code></pre>
<p><a href="https://i.stack.imgur.com/5TnlH.png" rel="nofollow noreferrer"><img alt="Filled polygon" src="https://i.stack.imgur.com/5TnlH.png"/></a></p>
<p>I'm not completely certain why the preceding code works like it does. <code>plt.fill()</code> is used for plotting 2D polygons, and the third argument should be the color, so what you should really write is this:</p>
<pre><code>x = [4, 1, 0, 2]
y = [1, 2, 0, 1]

plt.fill(x, y, c='C0')
plt.show()
</code></pre>
<p>(which gives the same plot)</p>
<p><a href="https://i.stack.imgur.com/z2elJ.png" rel="nofollow noreferrer"><img alt="Filled polygon" src="https://i.stack.imgur.com/z2elJ.png"/></a></p>

<span class="comment-copy"><code>plt.fill(x,y,z)</code> is the same as calling <code>plt.fill(x,y); plt.fill(z)</code>. You can in fact draw as many filled shapes as you like, though for better readability I would propose to always define x and y, like <code>plt.fill(x1,y1, x2,y2, x3, y3)</code>.</span>
