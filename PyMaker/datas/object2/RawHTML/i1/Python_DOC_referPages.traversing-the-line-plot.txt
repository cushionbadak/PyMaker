<p>Since the <code>y</code> values depend on the <code>a</code> values you need to specify an <code>a</code> for the <code>y</code> lookup. Consider the following:</p>
<pre><code>def costfunction(b, a):
    # b Value
    x = np.linspace(b*(-b), b*(b), 100)
    y = (x - a)**2
    return x, y
a = 5
c = costfunction(20, a)
plt.plot(c[0], c[1], linestyle='-', linewidth=1)
plt.ylabel("Cost Value");

b = 100
yb = (b - a)**2    # Find the corresponding y-value
plt.plot(b, yb, marker='o', color='b')
plt.show()
</code></pre>
<p>This will give you </p>
<p><a href="https://i.stack.imgur.com/GMOqd.png" rel="nofollow noreferrer"><img alt="Cost function with tracer" src="https://i.stack.imgur.com/GMOqd.png"/></a></p>
<p>You might also note that I modified the <code>costfunction</code> definition to return the <code>x</code> values, otherwise matplotlib will just use whatever values it pleases. </p>

<pre><code>def costfuntion(b, a):
    # b Value
    x = np.linspace(b*(-b), b*(b), 100)
    y = (x - a)**2
    return x, y

x, y = costfuntion(20,5)
plt.plot(x, y)
for i in range(0, len(x), 2):
    plt.plot(x[i], y[i], marker='o', color='b')
</code></pre>
