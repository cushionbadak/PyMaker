<div class="post-text" itemprop="text">
<p>You can use one of these queries:</p>
<pre><code>a_models = ModelA.objects.filter(model_b__b=known_b_value)
for a in a_models:
    print(a.f)

b_models = ModelB.objects.filter(b=known_b_value).select_related('a')
for b in b_models:
    print(b.a.f)
</code></pre>
<p><code>select_related</code> in the second query is not strictly necessary; it is only used to get the <code>f</code> values in the first query rather than in one additional query per loop item.</p>
