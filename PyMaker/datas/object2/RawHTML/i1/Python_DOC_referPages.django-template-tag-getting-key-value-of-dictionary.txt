<div class="post-text" itemprop="text">
<p>Dirty solution (because your original dict has nested list and dicts):</p>
<pre><code>{% for key, values in options.items %}
    {% for value in values %}
        {% for k, v in value.items %}
            key = {{ k }}
            val = {{ v }}
        {% endfor %}
    {% endfor %}
{% endfor %}
</code></pre>
<hr/>
<p><strong>Better Solution:</strong></p>
<p>You should consider flattening your dictionary so that it becomes easier to loop over. You don't need nested lists and dicts. Example:</p>
<pre><code>options = {
    235: &lt;Option ...&gt;,
    236: &lt;Option ...&gt;,
    237: &lt;Option ...&gt;,
    238: &lt;Option ...&gt;,
}
</code></pre>
<p>Now, you'll be able to loop over it in a single loop.</p>
