<div class="post-text" itemprop="text">
<p>I think you are trying to find the <strong>index of rows</strong> with null values sum greater 5. Use <code>np.where</code> instead of <code>df.loc</code> to find the index and then drop them.</p>
<p>Try:</p>
<pre><code>import pandas as pd
import numpy as np
df = pd.read_csv('https://query.data.world/s/Hfu_PsEuD1Z_yJHmGaxWTxvkz7W_b0')
d = np.where(df.isnull().sum(axis=1)&gt;5)
df= df.drop(df.index[d])
print(round(100*(1-df.count()/len(df)),2))
</code></pre>
<p>output:</p>
<pre><code>Ord_id                 0.00
Prod_id                0.00
Ship_id                0.00
Cust_id                0.00
Sales                  0.00
Discount               0.42
Order_Quantity         0.42
Profit                 0.42
Shipping_Cost          0.42
Product_Base_Margin    1.06
dtype: float64
</code></pre>
