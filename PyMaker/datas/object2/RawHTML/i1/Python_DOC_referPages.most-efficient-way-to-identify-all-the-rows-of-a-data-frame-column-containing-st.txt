<p>I want to identify all the rows of a data frame column containing strings that are in a list. For only one string, it can be done via </p>
<pre><code>df['Column Name'].str.contains('string_to_find')
</code></pre>
<p>However, <strong>.str.contains</strong> doesn't accept a list as an input which forces me to iterate through the list. The solution I found is the following:</p>
<pre><code>import pandas as pd

df = pd.DataFrame(['Mexico', 'Province of Ontario', 'NY', 'Alaska', 'ONT', 'Quebec', 'QC', 'Quebec (Province)'],
                  columns=['Province'])

province_filter_list = ['ONT', 'QC', 'Quebec']

df['Indicator'] = False
for x in province_filter_list:
    df['Indicator'] = df['Indicator'] | df['Province'].str.contains(x, case=False)
</code></pre>
<p>What I would like to do is</p>
<pre><code>df['Indicator'] = df['Province'].str.contains(province_filter_list, case=False)
</code></pre>
<p>The output should be the following:</p>
<pre><code>index Province             Indicator
0     Mexico               False
1     Province of Ontario  True
2     NY                   False
3     Alaska               False
4     ONT                  True
5     Quebec               True
6     QC                   True
7     Quebec (Province)    True
</code></pre>
<p>My question is: Is there a better and easier way to do this than a for loop using <strong>.str.contains</strong> ?</p>
