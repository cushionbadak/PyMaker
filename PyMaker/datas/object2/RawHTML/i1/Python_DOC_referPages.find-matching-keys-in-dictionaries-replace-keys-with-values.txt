<p>I have two dictionaries and I need to compare the keys and if the keys match, I need to replace the second dictionary keys with the values from the first.</p>
<pre><code>d1 = {'22': 'Jane', '33': 'Tom', '44': 'John', '55': 'Laura'}    
d2 = {'22': {'Lilly', 'Jake'}, '33' : {'Janet'}, '44': {'Tim'}}
</code></pre>
<p>I started, but something is already wrong...</p>
<pre><code>for k, v in d2.items():
    d2[k] = d1(k, v)
    print(d2)
</code></pre>
<p>It should look like this:</p>
<pre><code>{'Jane': {'Lilly', 'Jake'}, 'Tom': {'Janet'}, 'John': {'Tim'}}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a dict comprehension that iterates over the set intersection of the keys of the two dicts and output the corresponding values as the keys and values of the new dict:</p>
<pre><code>{d1[k]: d2[k] for k in d1.keys() &amp; d2.keys()}
</code></pre>
<p>This returns:</p>
<pre><code>{'Tom': {'Janet'}, 'Jane': {'Jake', 'Lilly'}, 'John': {'Tim'}}
</code></pre>
<p>Sets are unordered, however. If you would like to preserve the key order of <code>d2</code> (as keys are ordered since Python 3.7), you can iterate over the items of <code>d2</code> instead:</p>
<pre><code>{d1[k]: v for k, v in d2.items() if k in d1}
</code></pre>
<p>This returns:</p>
<pre><code>{'Jane': {'Lilly', 'Jake'}, 'Tom': {'Janet'}, 'John': {'Tim'}}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>What you can do is the following:<br/>
1) Iterate through the key values in the first dictionary<br/>
2) If the key value exists as a key in the second dictionary, then create a new entry for the value of the <em>first</em> dictionary's key, and delete the entry of the old key.</p>
<p>This worked for me:</p>
<pre><code>d1 = {'22': 'Jane', '33': 'Tom', '44': 'John', '55': 'Laura'}

d2 = {'22': {'Lilly', 'Jake'}, '33' : {'Janet'}, '44': {'Tim'}}

for key in d1:
    if key in d2:
        d2[d1[key]] = d2[key]
        del d2[key]
</code></pre>
