<p>I have a set of strings like this:</p>
<pre><code>span-aud-result-nhd34-124
span-aud-result-jsh43-125
span-aud-result-i843-127
span-aud-result-mj43-126
</code></pre>
<p>I want to extract the ending substring after <strong>-</strong></p>
<p>For Example like this:</p>
<pre><code>124
125
127
126
</code></pre>
<p>The problem is that the substrings like <code>nhd34, jsh43, i843</code> are dynamic. So how can I extract the ending substring just after <strong>-</strong></p>
<p>Thanks.</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>split()</code> to split the string on <code>-</code> and access last element of list:</p>
<pre><code>x = "span-aud-result-nhd34-124"

print(x.split("-")[-1])
</code></pre>
<p>Output:</p>
<pre><code>124
</code></pre>
<p><strong>Explanation:</strong></p>
<p>the <code>split</code> will return:</p>
<pre><code>["span","aud","result","nhd34","124"]
</code></pre>
<p><code>-1</code> refers to the last index of an array</p>
</div>
<div class="post-text" itemprop="text">
<p>Another way is to find the last occurrence of the '-' character in your string and then subset your initial string based on that index:</p>
<pre><code>&gt;&gt;&gt; s = 'span-aud-result-nhd34-124span-aud-result-nhd34-124'

&gt;&gt;&gt; s[(s.rfind('-') + 1):]
'124'
</code></pre>
</div>
<span class="comment-copy">The length of the ending numbers may vary.</span>
