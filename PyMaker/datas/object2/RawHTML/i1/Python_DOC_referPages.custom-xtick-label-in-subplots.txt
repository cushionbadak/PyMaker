<div class="post-text" itemprop="text">
<p>What you are looking for is the <a href="https://matplotlib.org/api/_as_gen/matplotlib.pyplot.subplots.html" rel="nofollow noreferrer"><code>sharex</code> keyword argument</a>. If set to <code>True</code>, each subplot of a figure will have the same properties. Additionally, tick labels only go to the bottom subplot. </p>
<p>The <code>tight_layout</code> commands is just an easy way to arange the subplots nicely in the figure.</p>
<pre><code>import matplotlib.pyplot as plt
import numpy as np


x = np.random.randint(0, 100, 10)
y = np.random.randint(0, 100, (4, 10))

fig, axs = plt.subplots(4, 1, sharex=True, sharey=True, figsize=(10, 6))

for i, (ax, data) in enumerate(zip(axs, y)):
    ax.plot(x, data)
    ax.set_ylabel('y{}(nT)'.format(i+1))

ax.set_xlabel('x(h)')
fig.tight_layout() 
</code></pre>
