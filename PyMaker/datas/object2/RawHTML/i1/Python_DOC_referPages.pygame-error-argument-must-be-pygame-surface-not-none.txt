<p>The loop </p>
<blockquote>
<pre><code>while UserSprite != (SPR1...)
    print ("Nope")
    break
</code></pre>
</blockquote>
<p>is useless, because you <code>break</code> the loop in any case. </p>
<p>Repeat the input until a valid name was read:</p>
<pre class="lang-py prettyprint-override"><code>UserSprite = None
while UserSprite == None:
    SpriteName = input("""Choose a sprite [...]""")
    UserSprite = vars()[SpriteName] if SpriteName in vars() else None
</code></pre>
<p>Note, you have to input capital letters, because all the variable name are in capital letters. The input has to be case sensitive.</p>
