<p>I want to create a dataframe of float features without my target, so I can further manipulate with them.</p>
<p>I've tried:</p>
<pre><code>float_col = data.select_dtypes('float').drop(['TARGET'], axis=1, inplace=True)

# Remove sparse numerical features
for f in float_col:
   if data[f].isnull().sum() / data.shape[0] &gt; 0.1667: del data[f] #Remove above 1/6 of NANs
</code></pre>
<p>...which returns:</p>
<blockquote>
<p>TypeError: 'NoneType' object is not iterable</p>
</blockquote>
<p>I've also tried to do:</p>
<pre><code>float_col = data.select_dtypes('float').drop(['TARGET'], axis=1, inplace=True).update()
</code></pre>
<p>...which returned:</p>
<blockquote>
<p>AttributeError: 'NoneType' object has no attribute 'update'</p>
</blockquote>
<p>I believe the issue is caused by the target drop.</p>
</div>
<div class="post-text" itemprop="text">
<p>When you call methods that apply changes <em>in place</em>, <code>None</code> is returned. You used <code>.drop(..., inplace=True)</code>:</p>
<pre><code>data.select_dtypes('float').drop(['TARGET'], axis=1, inplace=True)
</code></pre>
<p>so the result of <code>data.select_dtypes(...)</code> is directly changed and <code>None</code> is assigned to <code>float_col</code>. You can't use iteration (like a <code>for</code> loop) on <code>None</code>.</p>
<p>From the <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.drop.html" rel="nofollow noreferrer"><code>DataFrame.drop()</code> method documentation</a>:</p>
<blockquote>
<p><strong>inplace</strong> : <em>bool, default False</em></p>
<p>If <code>True</code>, do operation inplace and return <code>None</code>.</p>
</blockquote>
<p>If you wanted to get all <code>float</code> columns except <code>TARGET</code>, then just remove <code>inplace=True</code>:</p>
<pre><code>float_col = data.select_dtypes('float').drop(['TARGET'], axis=1)
</code></pre>
</div>
<span class="comment-copy">when you use <code>inplace=True</code> the operation happens inplace and there is no return value.To assign the result to a variable, just remove the <code>inplace=True</code></span>
