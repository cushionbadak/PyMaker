<div class="post-text" itemprop="text">
<ol>
<li><p>If the browser was triggered from Selenium you can <a href="https://stackoverflow.com/questions/19944745/how-to-get-session-id-on-selenium-webdriver-using-python">determine its sessionid</a> like:</p>
<pre><code>self.driver.session_id
</code></pre>
<p>and then start another WebDriver instance providing the aforementioned session_id as the parameter:</p>
<pre><code>driver = webdriver.Remote(command_executor=url,desired_capabilities={})
driver.session_id = session_id
</code></pre>
<p>if the browser wasn't kicked off via Selenium - it's not possible.</p></li>
<li><p>You can use <a href="https://www.blazemeter.com/blog/using-jmeters-transaction-controller" rel="nofollow noreferrer">Transaction Controller</a> to measure the cumulative execution time of its children </p>
<p><a href="https://i.stack.imgur.com/8NLdQ.png" rel="nofollow noreferrer"><img alt="enter image description here" src="https://i.stack.imgur.com/8NLdQ.png"/></a></p></li>
<li><p>You can put your shared logic into a separate .py file and use <a href="https://stackoverflow.com/questions/32239372/how-to-import-files-in-python-using-sys-path-append">sys.path</a> to load it where required like:</p>
<pre><code>from sys import path
path.append(path_to_your_shared_module)

import YourSharedModule

//call functions from the shared module
</code></pre></li>
</ol>
