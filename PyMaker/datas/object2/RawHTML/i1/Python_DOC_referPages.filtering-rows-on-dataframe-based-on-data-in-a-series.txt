<p>There's a DataFrame <code>df</code> containing following data:</p>
<pre><code>+------+----------+-------+
| YEAR | CATEGORY | GRADE |
+------+----------+-------+
| 1999 |     A    |  3.5  |
| 1999 |     A    |  7.2  |
| 1999 |     B    |  0.2  |
| 1999 |     B    |  6.4  |
| 2000 |     A    |  1.4  |
| 2000 |     A    |  2.5  |
| 2000 |     B    |  3.3  |
| 2000 |     B    |  8.4  |
+------+----------+-------+
</code></pre>
<p>And there's a multi-indexed Series <code>s</code> containing passing scores:</p>
<pre><code>YEAR  CATEGORY 
1999  A         5.2
      B         5.1
2000  A         3.7
      B         6.1
dtype: float64
</code></pre>
<p>So I would like to filter rows on <code>df</code> that <code>GRADE</code> is greater than value on <code>s</code> in a fashion that the result is like the following:</p>
<pre><code>+------+----------+-------+
| YEAR | CATEGORY | GRADE |
+------+----------+-------+
| 1999 |     A    |  7.2  |
| 1999 |     B    |  6.4  |
| 2000 |     B    |  8.4  |
+------+----------+-------+
</code></pre>
<p>How can I proceed to obtain this result?</p>
<p>Thanks in advance!</p>
</div>
<div class="post-text" itemprop="text">
<p><code>set_index</code> and using <code>gt</code> with boolean to filter the df </p>
<pre><code>yourdf=df[df.set_index(['YEAR','CATEGORY']).GRADE.gt(s).values]
yourdf
   YEAR CATEGORY  GRADE
1  1999        A    7.2
3  1999        B    6.4
7  2000        B    8.4
</code></pre>
