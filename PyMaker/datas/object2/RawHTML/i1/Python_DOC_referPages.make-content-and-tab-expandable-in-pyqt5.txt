<p>The size policy only works if the widget geometry is handled by a layout, so a solution is to set the QTabWidget as centralwidget to do it right click on an area that is outside the QTabWidget but inside the QMainWindow, then click on the icon <a href="https://i.stack.imgur.com/C3ER4.png" rel="nofollow noreferrer"><img alt="enter image description here" src="https://i.stack.imgur.com/C3ER4.png"/></a> or <a href="https://i.stack.imgur.com/HVcey.png" rel="nofollow noreferrer"><img alt="enter image description here" src="https://i.stack.imgur.com/HVcey.png"/></a> icon at the top of Qt Designer generating the following:</p>
