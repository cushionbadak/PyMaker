<div class="post-text" itemprop="text">
<p>You have some spelling errors in the code.
The first is about the <code>__init__</code> function that you spelled <code>__int__</code>
You also use a different variable to assign a widget and a similar one (but <em>different</em>) for adding it (e.g. <code>self.lastName = TextInput(multiline=False)</code> and <code>self.add_widget(self.lastname)</code>)<br/>
This code should work:</p>
<pre><code>from kivy.app import App
from kivy.uix.label import Label
from kivy.uix.gridlayout import GridLayout
from kivy.uix.textinput import TextInput


class MyGrid(GridLayout):
    def __init__(self, **kwargs):
        super(MyGrid, self).__init__(**kwargs)
        self.cols = 2
        self.add_widget(Label(text="Name: "))
        self.name = TextInput(multiline=False)
        self.add_widget(self.name)

        self.add_widget(Label(text="last name : "))
        self.lastName = TextInput(multiline=False)
        self.add_widget(self.lastName)

        self.add_widget(Label(text="email: "))
        self.emailName = TextInput(multiline=False)
        self.add_widget(self.emailName)


class MyApp(App):
    def build(self):
        return MyGrid()


if __name__ == '__main__':
    MyApp().run()
</code></pre>
