<div class="post-text" itemprop="text">
<p>The variable <code>cotacao</code>, is a list, which has only one item. So we access it with index [0]. That object, is a dictionary, which we can access its fields using their key:</p>
<pre><code>import json
import requests

r = requests.get("https://olinda.bcb.gov.br/olinda/servico/PTAX/versao/v1/odata/CotacaoDolarDia(dataCotacao=@dataCotacao)?@dataCotacao='03-15-2019'&amp;$top=1&amp;$format=json&amp;$select=cotacaoVenda")

if r.status_code == 200:
    cotacao = json.loads(r.content)
    print(cotacao['value'][0]['cotacaoVenda'])
</code></pre>
</div>
