<p>You can do this by adding an inner loop like this:</p>
<pre><code>r = requests.get("jsonURL", params="jsonparams")
data = r.json()

for items in data['orders']:
    for item in items['line_items']:
       print(item['title'])
</code></pre>
<p>This will print out the titles of the items in each of the orders.</p>
<p>If you only want to print out a subset of the items you can use a slice on the <code>items['line_items']</code>. If you want to print only the first <code>n</code> items, then you would write <code>items['line_items'][:n]</code>. If you want to print only the last <code>n</code> items, you would write <code>items['line_items'][-n]</code>. You can even combine the two, for instance <code>items['line_items'][1:-2]</code> will skip the first and last two items in the list.</p>

<p>The JSON from your code will look somewhat like this:</p>
<pre><code>    data =  {
        "orders": {
        "line_items": [
            {
                "title": "abc"
            },
            {
                "title": "def"
            },
            {
                "title": "ghi"
            },
            {
                "title": "jkl"
            },
            {
                "title": "mno"
            }
        ]
    }
}
</code></pre>
<p>So, in order to get titles for all items, loop through each item in the <code>line_items</code> list, look for the 'title' key and append its values to an empty list called <code>purchased</code>:</p>
<pre><code>purchased = []
for item in data['orders']['line_items']:
    if 'title' in item.keys():
        purchased.append(item['title'])
print(purchased)
#Output:
['abc', 'def', 'ghi', 'jkl', 'mno']
</code></pre>
<p>If you also want the quantity of each item purchased, you can use <code>Counter</code>:</p>
<pre><code>from collections import Counter
print(Counter(purchased))
#Output:
Counter({'abc': 1, 'def': 1, 'ghi': 1, 'jkl': 1, 'mno': 1})
</code></pre>
