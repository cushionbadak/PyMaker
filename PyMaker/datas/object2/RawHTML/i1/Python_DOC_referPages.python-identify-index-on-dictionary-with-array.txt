<p>I will assume that you have only the <code>CorrectionHistory</code> key as this is your example. In my example the function i wrote is very simple, just to prove is gets the job you mentioned done, but it can be easily generalized:</p>
<pre><code>a = {
        'CorrectionHistory': [
            {'key': 123, 'CorrectionsAll': [{'CorrChngDesc': 'Discount Line Changed'}, {'CorrChngDesc': 'Commodity Line Changed'}]},
            {'key': 456, 'CorrectionsAll': [{'CorrChngDesc': 'AC Added/Modified'}, {'CorrChngDesc': 'CMDY Added/Modified'}]},
            {'key': 789, 'CorrectionsAll': [{'CorrChngDesc': 'AC Added/Modified'}]}
        ]
    }


def get_index_for_key(key):
    for index, item in enumerate(a['CorrectionHistory']):
        if item['key'] == key:
            return index
    return None


print(get_index_for_key(456))
</code></pre>
<p>Result: <code>1</code> the index you are looking for</p>
