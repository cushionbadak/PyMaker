<p>If the website is unreachable, try fixing the broken packages with:</p>
<pre><code>$ sudo apt -f install
</code></pre>
<p>Afterwards, finish the fix with:</p>
<pre><code>$ sudo apt update &amp;&amp; sudo apt dist-upgrade
</code></pre>
<p>Now, you should be able to run:</p>
<pre><code>$ sudo apt install python3-pip
</code></pre>
<p>Best of luck.</p>
