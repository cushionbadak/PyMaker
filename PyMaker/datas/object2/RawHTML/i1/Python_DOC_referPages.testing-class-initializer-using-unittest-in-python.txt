<p>I am using unittest module for writing tests.
I need to test initialization of the object inside a testcase using different inputs.
For this purpose I am importing the class inside setUp(). But when I try to use the class inside test_*() functions, I get this error - <code>NameError: name 'Example' is not defined</code></p>
<p>Here is my code sample-</p>
<pre><code>import unittest
class TestExample(unittest.TestCase):
    def setUp(self):
        import Example

    def test_sample_function(self):
        e = Example(1,2)
</code></pre>
<p>I know that I can simply import the class at top of the script. But I do not want to do that. I need to import it only during setup of the testscript.
Looking for some help here.</p>

<div class="post-text" itemprop="text">
<p>There's no reason to import the module in <code>setUp</code>. The module is still available globally in <code>sys.modules</code>, but you've only bound it to a <em>local</em> name that goes away after <code>setUp</code> returns. Just import it globally.</p>
<pre><code>import unittest
import Example


class TestExample(unittest.TestCase):
    def test_sample_function(self):
        e = Example(1,2)
</code></pre>
