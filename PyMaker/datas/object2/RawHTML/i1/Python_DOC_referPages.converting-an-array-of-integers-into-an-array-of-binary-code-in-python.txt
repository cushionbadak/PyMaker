<p>I'm new in Python and need to convert an array of integers into one in binary code.</p>
<pre><code>s_str = "test"    
s_acii = [ord(c) for c in s_str]
print(s_acii)
&gt;&gt; [116, 101, 115, 116]
</code></pre>
<p>What I need:</p>
<pre><code>&gt;&gt; [1110100, 1100101, 1110011, 1110100]
</code></pre>
<p>Thanks!</p>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#bin" rel="nofollow noreferrer"><code>bin</code> built-in function</a> to convert the integers into their binary representations:</p>
<pre><code>&gt;&gt;&gt; l = [116, 101, 115, 116]
&gt;&gt;&gt; [bin(i) for i in l]
['0b1110100', '0b1100101', '0b1110011', '0b1110100']
</code></pre>
<p>If you do not want the <code>0b</code> prefix, use string formatting with <a href="https://docs.python.org/3/library/string.html#format-specification-mini-language" rel="nofollow noreferrer">binary integer representation</a>:</p>
<pre><code>&gt;&gt;&gt; l = [116, 101, 115, 116]
&gt;&gt;&gt; ["{0:b}".format(i) for i in l]
['1110100', '1100101', '1110011', '1110100']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <code>str.format</code> method with <code>b</code> as the presentation type:</p>
<pre><code>print('[{}]'.format(', '.join(map('{:b}'.format, s_acii))))
</code></pre>
<p>This outputs:</p>
<pre><code>[1110100, 1100101, 1110011, 1110100]
</code></pre>
