<pre><code>d=[1,1,1,2,2,3,4,5,5,5]

<div class="post-text" itemprop="text">
<p>More fancy; <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer"><code>groupby()</code></a>:</p>
<pre><code>from itertools import groupby

d = [1,1,1,2,2,3,4,5,5,5]

for _, y in groupby(d):
    print('\n'.join(map(str, y)), end='\n\n')
</code></pre>

<div class="post-text" itemprop="text">
<p>With your code, you are using <code>i</code> to iterate through the array <code>d</code> but then using <code>i</code> to access the element of the array <code>d[i]</code> thus <code>i == d[i]</code> will be true for the first three iterations then after that it'll be false (e.g. <code>i = 2</code>, <code>d[2] = 1</code>, therefore <code>(i == d[i]) == False</code>. Also, no need for an <code>elif</code> clause since you are only worried about one case. Use a variable to keep track of the previous element in the array and that should do the trick.</p>
<pre><code>d=[1,1,1,2,2,3,4,5,5,5]
temp = d[0]
for i in d:
  if temp != i:
     print('\n')
  print(i)
  temp = i
</code></pre>
