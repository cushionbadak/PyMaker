<div class="post-text" itemprop="text">
<p>You need to put <code>&lt;tbody&gt;</code> tags out of the <code>for</code> loop. Same for <code>&lt;/tbody&gt;</code>, <code>&lt;/table&gt;</code> and <code>&lt;/div&gt;</code>.</p>
<p>You want to have only <code>&lt;tr&gt;...&lt;/tr&gt;</code> part inside the loop:</p>
<pre><code>                        &lt;tbody&gt;

                          {% for member in primary_users %}
                            &lt;tr&gt;
                                &lt;td&gt;&lt;a href="{% url 'users:individual_member' member.id %}"&gt;{{ member.first_name }} {{ member.last_name }}&lt;/a&gt;&lt;/td&gt;
                                &lt;td&gt;{{ member.rating }}&lt;/td&gt;
                                &lt;td&gt;{{ member.email }}&lt;/td&gt;
                                &lt;td&gt;{{ member.area }}&lt;/td&gt;
                            &lt;/tr&gt;
                          {% endfor %}

                        &lt;/tbody&gt;
                    &lt;/table&gt;
                    &lt;/div&gt;
</code></pre>
<p>In general, when doing loops and conditions, pay attention to html tags, so that you don't end up with unmatched tags.</p>
