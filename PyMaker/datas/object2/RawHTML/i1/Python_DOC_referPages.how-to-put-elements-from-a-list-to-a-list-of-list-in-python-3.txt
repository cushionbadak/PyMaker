<p>I am trying to put the elements in <code>list2</code> inside each of the nested lists in <code>list1</code>. This is what I have tried so far:</p>
<pre><code>list_1 = [[0, 1], [1, 4], [2, 3]]
list_2 = [100, 100, 100]
store_1 = []
for x in list_1:
    for y in list_2:
        x.append(y)
        store_1.append(x)
print(store_1)
</code></pre>

<pre><code>[[0, 1, 100, 100, 100], [0, 1, 100, 100, 100], [0, 1, 100, 100, 100], [1, 4, 100, 100, 100], [1, 4, 100, 100, 100], [1, 4, 100, 100, 100], [2, 3, 100, 100, 100], [2, 3, 100, 100, 100], [2, 3, 100, 100, 100]]
</code></pre>
<p>The output should be like this:</p>
<pre><code>[[0,1,100],[1,4,100], [2,3,100]]
</code></pre>
<p>How can i fix my code to get the desired output?</p>

<div class="post-text" itemprop="text">
<p>Using <code>zip</code></p>
<p>Ex:</p>
<pre><code>list_1 = [[0, 1], [1, 4], [2, 3]]
list_2 = [100, 100, 100]
store_1 = [x + [y] for x, y in zip(list_1, list_2)]
print(store_1)
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>[[0, 1, 100], [1, 4, 100], [2, 3, 100]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Without using <code>zip</code></p>
<pre><code>list_1 = [[0, 1], [1, 4], [2, 3]]
list_2 = [100, 100, 100]
[list_1[idx] + [x] for idx, x in enumerate(list_2)]

&gt; [[0, 1, 100], [1, 4, 100], [2, 3, 100]]
</code></pre>
