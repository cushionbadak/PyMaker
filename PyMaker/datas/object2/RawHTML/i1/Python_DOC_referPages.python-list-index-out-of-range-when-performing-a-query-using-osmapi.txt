<div class="post-text" itemprop="text">
<p>First - you should pass url and credentials in constructor:</p>
<p><code>api = osm.OsmApi(api="https://api.openstreetmap.org", username="username", password="secret")</code></p>
<p>Next -<code>api/0.6/way/{id}</code> - maybe you are looking for <code>WayGet</code> method.</p>
<p>Code:</p>
<p><code>import osmapi as osm
api = osm.OsmApi(api="https://api.openstreetmap.org", username="username", password="secret")
way = api.WayGet(77517260)
</code></p>
