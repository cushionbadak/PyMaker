<div class="post-text" itemprop="text">
<p>according to the package <a href="https://pypi.org/project/python-rtkit/#read-a-ticket" rel="nofollow noreferrer">documentation</a> it seems like the proper way to read the response is to use <code>response.parsed</code>:</p>
<pre class="lang-py prettyprint-override"><code>try:
    response = resource.get(path='ticket/1')
    for r in response.parsed:
        for t in r:
            logger.info(t)
except RTResourceError as e:
    logger.error(e.response.status_int)
    logger.error(e.response.status)
    logger.error(e.response.parsed)
</code></pre>
