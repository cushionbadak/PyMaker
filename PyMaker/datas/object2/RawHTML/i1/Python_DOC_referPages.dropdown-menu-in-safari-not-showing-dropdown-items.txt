<div class="post-text" itemprop="text">
<p>It looks like your <code>&lt;option&gt;</code> tags should be <code>&lt;a&gt;</code> tags instead since they are not wrapped within a <code>&lt;select&gt;</code>.</p>
<p><a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/option" rel="nofollow noreferrer">https://developer.mozilla.org/en-US/docs/Web/HTML/Element/option</a></p>
<p>Try this:</p>
<pre><code>&lt;div class="dropdown-menu" id="plat-form-options"&gt;
  {% for platform in game.platform.all %}
      &lt;a href="#" class="dropdown-item" value="{{ platform.id }}"
              onclick="platFormSelect('{% url 'title-updates-ajax' slug=game.slug platform_id=platform.id %}', '{{ platform }}')"&gt;{{ platform }}&lt;/a&gt;
  {% endfor %}
&lt;/div&gt;
</code></pre>
