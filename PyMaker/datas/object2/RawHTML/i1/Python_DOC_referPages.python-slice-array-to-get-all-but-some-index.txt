<p>Is it possible to use python's slicing to get all but some item of a given array?</p>
<p>meaning, for example, </p>
<pre><code>arr = range(10)
print(slicing_magic(arr, 4))
</code></pre>
<p>would output</p>
<pre><code>[0 1 2 3 5 6 7 8 9]
</code></pre>
<p>I realize this could be done with</p>
<pre><code>def slicing_magic(arr, ind):
    return arr[0:ind] + arr[ind+1:]
</code></pre>
<p>I wonder if there is a way without list addition, or</p>
<p><strong>what is the most pythonic way?</strong></p>

<div class="post-text" itemprop="text">
<p>You could do this? </p>
<pre class="lang-py prettyprint-override"><code>def magic_slicer(arr, idx):
    b = arr.copy()
    del b[idx]
    return b
</code></pre>
