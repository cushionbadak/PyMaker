<p>Just add a pdf here: <code>content/images/resume.pdf</code></p>
<p>Then add a link to it in your header (in <code>pelican-themes/elegant/templates/base.html</code>). Underneath the following block:</p>
<pre><code>&lt;div class="nav-collapse collapse"&gt;
  &lt;ul class="nav pull-right top-menu"&gt;
</code></pre>
<p>You'll see a bunch of <code>&lt;li&gt;</code> tags, e.g.:</p>
<pre><code>&lt;li {% if page_name == 'archives' %} class="active"{% endif %}&gt;&lt;a href="{{ SITEURL }}/{{ ARCHIVES_URL|default('archives') }}"&gt;Archives&lt;/a&gt;&lt;/li&gt;
</code></pre>
<p>Add <code>&lt;li&gt;&lt;a href='images/my.pdf'&gt;&lt;/a&gt;&lt;/li&gt;</code> below the Archives link referenced above and rebuild your site.</p>
