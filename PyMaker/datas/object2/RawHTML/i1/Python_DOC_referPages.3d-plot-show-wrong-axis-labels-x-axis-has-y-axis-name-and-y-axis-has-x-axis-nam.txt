<p>Note what the <a href="https://docs.scipy.org/doc/numpy-1.14.1/reference/generated/numpy.histogram2d.html" rel="nofollow noreferrer"><code>numpy.histogram2d</code></a> documentation says concerning the output array: </p>
<blockquote>
<p>"The bi-dimensional histogram of samples x and y. Values in x are histogrammed along the first dimension and values in y are histogrammed along the second dimension."</p>
</blockquote>
<p>This means you would want to transpose the resulting array, probably like </p>
<pre><code>dz = hist.T.flatten()
</code></pre>
