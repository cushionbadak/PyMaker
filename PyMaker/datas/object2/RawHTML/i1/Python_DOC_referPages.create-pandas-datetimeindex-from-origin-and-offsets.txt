<div class="post-text" itemprop="text">
<p>I want to convert a set of seconds into a Pandas index suitable for a dataframe.</p>
<p>I think I'm looking for the inverse of the <code>dt.total_seconds()</code> operation as in the <a href="https://stackoverflow.com/a/36156620/1653571">https://stackoverflow.com/a/36156620/1653571</a> answer to  <a href="https://stackoverflow.com/questions/36156582/accessing-total-seconds-in-pandas-data-column">Accessing total_seconds() in pandas data column</a></p>
<p>Given:</p>
<pre><code>t0="2010-10-01 00:00:00 UTC"
seconds = [0,1,2,3,5,7,11,13]
</code></pre>
<p>I want something like:</p>
<pre><code>ii = pd.DatetimeIndex( ???(t0,seconds))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Convert the <code>seconds</code> list to <code>Timedelta</code> and do a little <code>datetime</code> arithmetic.</p>
<pre><code># origin           + offset
pd.to_datetime(t0) + pd.to_timedelta(seconds, unit='s')
</code></pre>
<p></p>
<pre><code>DatetimeIndex(['2010-10-01 00:00:00+00:00', '2010-10-01 00:00:01+00:00',
               '2010-10-01 00:00:02+00:00', '2010-10-01 00:00:03+00:00',
               '2010-10-01 00:00:05+00:00', '2010-10-01 00:00:07+00:00',
               '2010-10-01 00:00:11+00:00', '2010-10-01 00:00:13+00:00'],
              dtype='datetime64[ns, UTC]', freq=None)
</code></pre>
</div>
