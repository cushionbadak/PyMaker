<p>I'm using PySide2 and I want to have multiple shortcuts that carry out the same function but also would depend on which key was pressed.</p>
<p>I tried to link the functions as such inside a QMainWindow:</p>
<pre><code>QtWidgets.QShortcut(QtGui.QKeySequence("1"),self).activated.connect(self.test_func)
QtWidgets.QShortcut(QtGui.QKeySequence("2"),self).activated.connect(self.test_func)
</code></pre>
<p>Such that they both execute this function.</p>
<pre><code>def test_func(self, signal):
    print(signal)
</code></pre>
<p>Hoping that print("1") happens when the key "1" is pressed and print("2") happens when the second key is pressed.
When I tried running this and press the keys 1 and 2, I get this error:</p>
<pre><code>TypeError: test_func() missing 1 required positional argument: 'signal'
</code></pre>
<p>How can I accomplish this?</p>

<div class="post-text" itemprop="text">
<p>The activated signal does not send any information so the one option is to obtain the object that emits the signal (ie the QShortcut) to obtain the QKeySequence, and from the latter the string:</p>
<pre><code>from PySide2 import QtCore, QtGui, QtWidgets

class MainWindow(QtWidgets.QMainWindow):
    def __init__(self, parent=None):
        super(MainWindow, self).__init__(parent)
        QtWidgets.QShortcut(QtGui.QKeySequence("1"), self, activated=self.test_func)
        QtWidgets.QShortcut(QtGui.QKeySequence("2"), self, activated=self.test_func)

    @QtCore.Slot()
    def test_func(self):
        shorcut = self.sender()
        sequence = shorcut.key()
        print(sequence.toString())

if __name__ == '__main__':
    import sys
    app = QtWidgets.QApplication(sys.argv)
    w = MainWindow()
    w.show()
    sys.exit(app.exec_())
</code></pre>
