<div class="post-text" itemprop="text">
<p>I am getting started on using Zappa. However, I already had installed python 3.7 on my computer while Zappa uses 3.6. I installed python 3.6.8, but when I try to use zappa in the cmd (zappa init) it uses python 3.7 by default. How can I direct zappa to use 3.6 instead?</p>
</div>
<div class="post-text" itemprop="text">
<p>As mentioned in Zappa <a href="https://github.com/Miserlou/Zappa#installation-and-configuration" rel="nofollow noreferrer">README</a>:</p>
<blockquote>
<p>Please note that Zappa must be installed into your project's virtual environment. </p>
</blockquote>
<p>You should use something like <code>virtualenv</code> to create a virtual environment, which makes it easy to switch Python version.</p>
<p>If you use virtualenv, you can try create an environment by:</p>
<pre><code>$ virtualenv -p /usr/bin/python3.6 venv
$ source activate venv
</code></pre>
<p>Then <code>pip install zappa</code> in this virtual environment.</p>
</div>
<div class="post-text" itemprop="text">
<p>I don't know about Zappa, but if you want use a specific version of python can do:</p>
<pre><code>python3.6 my_program.py
</code></pre>
<p>and if whant use the command <em>python</em> with a specific version permanently, in <strong>linux</strong> modify the file <em>/home/[user_name]/.bashrc</em> and add the next line:</p>
<pre><code>alias python=python3.6
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>virtualenv</code> to setup an environment using a specific Python version by:</p>
<pre><code>% pip install virtualenv
% virtualenv -p python3.6 .venv
</code></pre>
<p>You can also use an absolute path to the Python executable if it's named the same but located in different folders.</p>
<p>Then switch to use the environment:</p>
<pre><code>% source .venv/bin/activate
</code></pre>
<p>This environment uses Python 3.6, so install Zappa with pip like normal and you're good to go.</p>
<p>You can read more about usage of virtualenv <a href="https://help.dreamhost.com/hc/en-us/articles/115000695551-Installing-and-using-virtualenv-with-Python-3" rel="nofollow noreferrer">here</a>.</p>
</div>
