<p>I have a Checkbutton and an <code>IntVar</code> object associated with it, but when I try to get the value of the <code>var</code>, I am receiving <code>PY_VAR0</code>.</p>
<p>Here's my code:</p>
<pre><code>from tkinter import *

root = Tk()

def show_state():
    print(var)

var = IntVar()

cbtn = Checkbutton(root, text='Check', variable=var, command=show_state)
cbtn.pack()

root.mainloop()
</code></pre>
<p>Why am I getting <code>PY_VAR0</code>?</p>
</div>
<div class="post-text" itemprop="text">
<p><code>var</code> is a reference to a <code>Tkinter.IntVar</code> object. You need to call its <a href="http://effbot.org/tkinterbook/variable.htm" rel="noreferrer"><code>get</code></a> method to access the value that it represents:</p>
<pre><code>print(var.get())
</code></pre>
