<p>I have a <code>df</code>,</p>
<pre><code>delta1    delta2
  0         -1
  2          0
 -1          0
  0          0
</code></pre>
<p>I am wondering how to assign values of <code>delta2</code> to <code>delta1</code> only if <code>delta1 &gt; 0</code> and <code>delta2 &lt;= 0</code>; the result look like,</p>
<pre><code>delta1    delta2
  0         -1
  0          0
 -1          0
  0          0 
</code></pre>

<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.scipy.org/doc/numpy/reference/generated/numpy.where.html" rel="noreferrer"><code>numpy.where</code></a>:</p>
<pre><code>df['delta1'] = np.where(df.delta1.gt(0) &amp; df.delta2.le(0), df.delta2, df.delta1)
</code></pre>
