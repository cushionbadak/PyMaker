10

<div class="post-text" itemprop="text">
<p>You can just do it like this:</p>
<pre><code>print("Enter the minimum number of feet (not less than 0):")
minN = input()
print("Enter the maximum number of feet (not more than 99):")
maxN = input()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Simply add a new line to the input prompt with <code>\n</code> (Python newline character) like this:</p>
<pre class="lang-py prettyprint-override"><code>minFeet = input("Enter the minimum number of feet (not less than 0): \n")
maxFeet = input("Enter the maximum number of feet (not more than 99): \n")
</code></pre>
