<div class="post-text" itemprop="text">
<p><code>ax</code> is a 2D array with shape (2, 2). Therefore, <code>each_ax</code> is a 1D array with shape (2,). You have to access every element separately. For example:</p>
<pre><code>for row in ax:
    for each_ax in row:
        each_ax.legend(['Male', 'Female'])
</code></pre>
