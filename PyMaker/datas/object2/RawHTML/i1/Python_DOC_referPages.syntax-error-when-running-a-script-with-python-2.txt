<p>There are some ways to use any version and run your project in your desire version.</p>
<p><strong>Step 1:</strong></p>
<p>If you are using Ubuntu. run this command in your command line</p>
<p><code>pip3 install virtualenv</code></p>
<p><strong>Step 2 :</strong> 
Enter your avs project folder</p>
<p><code>cd avs</code></p>
<p>Then run this</p>
<p><code>virtualenv -p python3 venv</code>   [python3 = python version; venv=virtualenv name]</p>
<p><code>source venv/bin/activate</code></p>
<p><strong>Finally</strong></p>
<p><code>python setup.py install</code></p>
<p>above the example your project will run on python3</p>
<p><strong>Another way that is on your gitrepo example</strong>
<code>https://github.com/respeaker/avs</code></p>
<pre><code>sudo apt-get install mpg123 mpv
sudo apt-get install gstreamer1.0-plugins-good gstreamer1.0-plugins-bad gstreamer1.0-plugins-ugly \
gir1.2-gstreamer-1.0 python3-gi python3-gst-1.0
sudo apt-get install python3-pyaudio
</code></pre>
<p>I change python-pyaudio to python3-pyaudio</p>
