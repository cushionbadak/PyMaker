<p>Try with <a href="https://pandas.pydata.org/pandas-docs/version/0.23.4/generated/pandas.cut.html" rel="nofollow noreferrer"><code>pd.cut()</code></a> and <a href="https://pandas.pydata.org/pandas-docs/stable/generated/pandas.DataFrame.groupby.html" rel="nofollow noreferrer"><code>df.groupby()</code></a></p>
<pre><code>bins=[1999,2004,2009,2014]
labels=['2000-2004','2005-2009','2010-2014']
df['Year']=pd.cut(df.year,bins,labels=labels)
print(df.groupby('Year',as_index=False).agg({'Product':'first','Total sale':'mean'}))
</code></pre>
