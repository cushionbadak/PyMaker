<p>I'm trying to get the integral of a Gaussian distrubution between two bounds given the mean and standard deviation of the Gaussian.</p>
<pre><code>import numpy as np
import scipy

mu = 5
sigma = 30
lowerbound = 0.5
upperbound = np.inf

# generate Gaussian function
x = np.linspace(mu - 3*sigma, mu + 3*sigma, 100)
gauss = scipy.stats.norm.pdf(x, mu, sigma)

# integrate between bounds
integral = scipy.integrate.quad(gauss, lowerbound, upperbound)
</code></pre>
<p>This raises a <code>ValueError: invalid callable given</code>, so I'm doing something wrong but I don't know what it is.</p>
</div>
<div class="post-text" itemprop="text">
<p>The <code>quad</code> function takes a function as its first input, but you were providing data from the gaussian evaluated at x:</p>
<pre><code>import numpy as np
import scipy

mu = 5
sigma = 30
lowerbound = 0.5
upperbound = np.inf

# generate Gaussian function
def gauss(x):
    return scipy.stats.norm.pdf(x, mu, sigma)

# integrate between bounds
integral = scipy.integrate.quad(gauss, lowerbound, upperbound)
print(integral)
</code></pre>
<pre><code>(0.5596176923702426, 5.087725389583706e-10)
</code></pre>
<p>If you want to integrate discrete data, <code>scipy.integrate.quad</code> is not the tool for the job. Use <code>scipy.integrate.simps</code> instead.</p>
</div>
<span class="comment-copy">FYI: You could skip <code>quad</code> and instead use <code>integral = scipy.stats.norm.cdf(upperbound, mu, sigma) - scipy.stats.norm.cdf(lowerbound, mu, sigma)</code>.</span>
<span class="comment-copy">...and since your upper bound is <code>inf</code>, you could use the <a href="https://en.wikipedia.org/wiki/Survival_function" rel="nofollow noreferrer">survival function</a> method, <code>sf</code>: <code>integral = scipy.stats.norm.sf(lowerbound, mu, sigma)</code>.</span>
