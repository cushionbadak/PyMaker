<p>How can I get a Series on special conditions?
I want to find outliers and put them in the series <code>df["has_outliers"]</code>, like if a row has a value more than 550 in any column, then True, otherwise False. </p>
<p>The output for this dataframe should be </p>

<div class="post-text" itemprop="text">
<p>Compare by <a href="http://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.gt.html" rel="nofollow noreferrer"><code>DataFrame.gt</code></a> with <a href="http://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.any.html" rel="nofollow noreferrer"><code>DataFrame.any</code></a> for check at least one True per row:</p>
<pre><code>df["has_outliers"] = df.gt(500).any(axis=1)
</code></pre>
<p>Or count <code>True</code>s and cast to integers:</p>
<pre><code>df["has_outliers"] = df.gt(500).sum(axis=1).astype(bool)
</code></pre>
