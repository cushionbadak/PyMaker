<p>this function is suppose to take each word in the string list and print them as lowercase, (I am doing this by modifying the original list) but I am getting the error:</p>
<blockquote>
<p>builtins.TypeError: list indices must be integers or slices, not str</p>
</blockquote>
<pre><code>def set_lowercase(strings):
    """takes a list of strings and modifies by replacing each string with lowercase"""
    for word in strings:
        strings[word] = strings[word].lowercase()
    print(strings)

set_lowercase(['Right', 'SAID', 'Fred']) 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>When you write a <code>for</code> loop with:</p>
<pre><code>for word in strings:
</code></pre>
<p>you iterate on the elements of <code>strings</code>, which are your words. On each loop, <code>word</code> becomes one of your words, not its index.</p>
<p>If you need both the word and its index (and you need it here, as you want to update the list in place), you can use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer">enumerate</a> which will give you on each iteration both the word and the index:</p>
<pre><code>def set_lowercase(strings):
        """takes a list of strings and modifies by replacing each string with lowercase"""
        for index, word in enumerate(strings):
            strings[index] = word.lower()


words = ['Right', 'SAID', 'Fred'] 
set_lowercase(words)
print(words)
# ['right', 'said', 'fred']
# your original list has been modified in place.
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><strong>The culprit</strong>:</p>
<pre><code>strings[word] = strings[word].lowercase()
</code></pre>
<p><strong>The reason</strong>:</p>
<pre><code>strings[word]
</code></pre>
<p>where word is an element in the list, You cannot iterate over a list by its element. Furthermore, there is no such method as <code>lowercase()</code> but <code>lower()</code></p>
<p><strong>The fix</strong>:</p>
<pre><code> for i in range(len(strings)):
        strings[i] = strings[i].lower()
    print(strings)
</code></pre>
<p><strong>Alternative</strong>:</p>
<p>You can achieve the same using <code>list-comprehension</code>:</p>
<pre><code>[x.lower() for x in ['Right', 'SAID', 'Fred']]
</code></pre>
<p><strong>OUTPUT</strong>:</p>
<pre><code>['right', 'said', 'fred']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>When you write <code>strings[word]</code>, it expects <code>word</code> to be the index rather than the actual value you are looking for. 
A better way to achieve this result would be using a list comprehension as follows:</p>
<pre><code>upper_case = ['Right', 'SAID', 'Fred']
lower_case = [x.lower() for x in upper_case]
</code></pre>
