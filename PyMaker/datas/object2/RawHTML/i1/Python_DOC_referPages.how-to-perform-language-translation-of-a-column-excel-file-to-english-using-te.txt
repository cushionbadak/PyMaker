<div class="post-text" itemprop="text">
<p>It looks like you accidentally inserted a space before <code>apply</code>.</p>
<p>Change this:</p>
<pre><code>df_file2= df_file['Repair Details']. apply.translate(from_lang='zh-CN',to ='en')
</code></pre>
<p>To this:</p>
<pre><code>df_file2= df_file['Repair Details'].apply.translate(from_lang='zh-CN',to ='en')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You probably want to use <code>TextBlob</code>'s <code>translate</code> method.</p>
<p>Assuming <code>df_file['Repair Details']</code> have TextBlob objects:</p>
<p><code>df_file2= df_file['Repair Details'].apply(lambda x: x.translate(from_lang='zh-CN',to ='en'))</code></p>
