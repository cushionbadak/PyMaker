<p>Add a init fun to the form:</p>
<pre><code>    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        for myField in self.fields:
            self.fields[myField].widget.attrs['class'] = 'form-control'
</code></pre>
<p>Also, remember to add bootstrap CSS and js to template</p>

<div class="post-text" itemprop="text">
<p>You can use django-crispy-forms <a href="https://django-crispy-forms.readthedocs.io/en/latest/" rel="nofollow noreferrer">https://django-crispy-forms.readthedocs.io/en/latest/</a></p>
<p>in your settings.py</p>
<pre><code>CRISPY_TEMPLATE_PACK = 'bootstrap3' #or bootstrap4
</code></pre>
<p>With that your template will look like this</p>
<pre><code>{% extends 'applicant/base.html' %}
{% load crispy_forms_tags %}

{% block content %}
&lt;center&gt;
    &lt;!-- Default unchecked --&gt;
    &lt;form method = "post"&gt;
        {% csrf_token %}
        {{ form|crispy }}
        &lt;button type="submit" class="btn btn-default"&gt;Save&lt;/button&gt;
    &lt;/form&gt;
&lt;/center&gt;
{% endblock %}
</code></pre>
</div>
