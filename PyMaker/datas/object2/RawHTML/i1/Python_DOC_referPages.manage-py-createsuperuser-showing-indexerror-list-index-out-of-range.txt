<div class="post-text" itemprop="text">
<p>It seems that you do not set PASSWORD_HASHERS (by default it is set).
Try to set this code in your settings:</p>
<pre><code>PASSWORD_HASHERS = [
    'django.contrib.auth.hashers.PBKDF2PasswordHasher',
    'django.contrib.auth.hashers.PBKDF2SHA1PasswordHasher',
    'django.contrib.auth.hashers.Argon2PasswordHasher',
    'django.contrib.auth.hashers.BCryptSHA256PasswordHasher',
]
</code></pre>
<p>See: <a href="https://docs.djangoproject.com/en/2.1/topics/auth/passwords/#how-django-stores-passwords" rel="nofollow noreferrer">https://docs.djangoproject.com/en/2.1/topics/auth/passwords/#how-django-stores-passwords</a></p>
