<div class="post-text" itemprop="text">
<p>You can read in the CSV file into a pandas DataFrame using <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.read_csv.html" rel="nofollow noreferrer">pandas.read_csv</a> and then convert to html using <a href="https://pandas.pydata.org/pandas-docs/version/0.23/generated/pandas.DataFrame.to_html.html" rel="nofollow noreferrer">pandas.to_html</a></p>
<p>For a CSV File "input.csv"</p>
<pre><code>employee_name, 03/15/2019,03/16/2019
employee1, task1,task2 
employee2, task3, task4
</code></pre>
<p>We can read the CSV file as a DataFrame</p>
<pre><code>import pandas as pd
df = pd.read_csv("input.csv", index_col="employee_name")
</code></pre>
<p>where df is </p>
<pre><code>               03/15/2019 03/16/2019
employee_name
employee1           task1     task2
employee2           task3      task4
</code></pre>
<p>Then we can convert the DataFrame to an HTML table with</p>
<pre><code>df.to_html("input.html")
</code></pre>
<p>And the output of the HTML file "intput.html" will be</p>
