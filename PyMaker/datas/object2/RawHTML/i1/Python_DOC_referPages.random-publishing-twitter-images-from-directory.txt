<p>You appear to be missing indentation on your <code>tweet_photos</code> method. Without this indentation, the interpreter won't be able to tell where the method starts and ends. </p>
<p>Additionally, you are attempting to iterate over a <code>str</code>. In this case, the value of <code>x</code> will be each individual character in this string. You can verify this by running the following code in the Python interpreter:</p>
<pre><code>imagePaths = "E:\Fotosprueba"
for x in imagePaths:
  print(x)
</code></pre>

<p>By the looks of things, you may want to pass the value of this str to the <code>getPathsFromDir</code> method instead. Try this:</p>
<pre><code>def getPathsFromDir(dir, EXTS="extensions=,png,jpg,jpeg,gif,tif,tiff,tga,bmp"):
    return this.listPaths(dir, EXTS) # This now uses the value of dir


def tweet_photos(api):

    # This now passes imagepaths to the getPathsFromDir method,

            api.update_with_media(filename=x,status=status)

<p>In theory this should work as intended, as long as your class also includes a <code>listPaths</code> method. If it doesn't, you'll need to modify to include this method, or change the call to <code>this.listpaths</code> to point to somewhere else.</p>
