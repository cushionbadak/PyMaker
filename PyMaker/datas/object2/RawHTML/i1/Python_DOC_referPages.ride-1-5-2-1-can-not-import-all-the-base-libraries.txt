<div class="post-text" itemprop="text">
<p>There were changes in the database for keywords documentation storage in 1.7.3.
To clean up that database you have to delete the <code>librarykeywords.db</code> file in the <code>%APPDATA%\RobotFramework\ride</code> directory. Then start RIDE and it will be recreated whenever you open a test suite (it may take some time until the red color changes to black on the Library imports).</p>
</div>
<span class="comment-copy">1- Full uninstallation of python and related libraries 2- Delete Remaining Files and Folders of the Program, to delete these leftovers, you'll need check following folders:  %programfiles% %programfiles(x86)% %appdata%  3- Remove Software Keys from Windows Registry. Type regedit in RUN or Start Menu search box  KEY_CURRENT_USER\Software HKEY_LOCAL_MACHINE\SOFTWARE HKEY_USERS.DEFAULT\Software  4- Empty Temp Folder: %temp%</span>
