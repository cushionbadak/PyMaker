<p>Use the below:</p>
<pre><code>specific_address=df[df.P_o_box.eq(22)&amp;df.House_num.eq(100)]
print(specific_address)

   P_o_box  House_num
0       22        100
1       22        100
</code></pre>
<p>EDIT, for a new df:</p>
<pre><code>new_df=pd.DataFrame({'P_o_box': df.loc[df['P_o_box'] == 22,'P_o_box'],
'House_num': df.loc[df['House_num'] == 100,'House_num']})
print(new_df)

   P_o_box  House_num
0       22      100.0
1       22      100.0
2       22        NaN
</code></pre>
<p>you can use: <code>new_df=new_df.dropna()</code> for dropping the <code>NaN</code> values</p>
