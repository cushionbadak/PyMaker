<pre><code>Preferences -&gt; Languages &amp; Frameworks -&gt; Django
</code></pre>
<p>Set your project root, settings file and manage.py script. You probably will also need to configure the template language at:</p>
<pre><code>Preferences -&gt; Languages &amp; Frameworks -&gt; Python Template Language
</code></pre>
<p>Selecte Django as Template language</p>
