<p>Is there a way to detect when an external monitor is plugged to the VGA port in Ubuntu Linux, using a Python script? Should I poll using xrandr, or <a href="https://unix.stackexchange.com/a/122301/9108">monitor a UDEV message</a>?</p>
</div>
<div class="post-text" itemprop="text">
<p>Take a look at <a href="https://github.com/jceb/srandrd" rel="nofollow noreferrer">srandrd</a>. It's a <a href="https://www.x.org/wiki/Projects/XRandR/" rel="nofollow noreferrer">randr</a> daemon for which you can install handlers to be run for certain events, like a monitor connecting, for instance.</p>
<p>If you have the following script in "handler.sh":</p>
<pre class="lang-sh prettyprint-override"><code>#!/bin/sh
case "$S{RANDRD_OUTPUT} ${SRANDRD_EVENT}" in
  "VGA-0 connected")
  # React to event.
  ;;
esac
</code></pre>
<p>Then you can install the handler via:</p>
<pre class="lang-sh prettyprint-override"><code>% srandrd handler.sh
</code></pre>
</div>
