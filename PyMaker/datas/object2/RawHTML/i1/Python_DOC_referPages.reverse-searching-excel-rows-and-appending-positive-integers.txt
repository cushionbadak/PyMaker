<div class="post-text" itemprop="text">
<p>The <code>isdigit</code> method is supposed to be <a href="https://docs.python.org/3/library/stdtypes.html" rel="nofollow noreferrer">used with strings</a>, not floats. Try using either</p>
<pre><code>if str(item).isdigit() == True and item &gt; 0:
    ...
</code></pre>
<p>or</p>
<pre><code>if isinstance(item, float) and item &gt; 0:
    ...
</code></pre>
