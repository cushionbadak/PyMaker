<p>As described in my comment I'd advise you to research, <a href="https://www.w3schools.com/html/html_forms.asp" rel="nofollow noreferrer">how HTML forms</a> and inputs work.</p>
<p>The URL you need to send your data to is either determined by the <code>action</code> parameter or encoded in some Javascript function.
The HTTP method you need to use is set in the <code>method</code> parameter of the form or <code>get</code> by default.</p>
<p>Here's how you could use that in python:</p>
<p>Adapted from the <a href="http://docs.python-requests.org/en/master/user/quickstart/" rel="nofollow noreferrer">documentation</a> of the <code>requests</code> module.</p>
<pre><code>import requests

# You might need to choose the get method here depending on the value of the forms method parameter
response = requests.post("your-url.here", data={'limit': 0})

print(response.text)
</code></pre>
<hr/>
<p><strong>Edit</strong></p>
<p>In response to the Comments I'll add the example for <code>get</code> Requests as well</p>
<pre><code>import requests

response = requests.get("your-url.here", params={'limit': 0})

print(response.text)
</code></pre>
<p>The version has the advantage compared to the comment, that it will take care of URL-Encoding special characters for you.</p>
