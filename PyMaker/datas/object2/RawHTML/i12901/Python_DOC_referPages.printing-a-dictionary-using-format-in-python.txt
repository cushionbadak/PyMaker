<p>I have a dictionary in the following format :- </p>
<pre><code>d = { 'x' : 1, 'y' : 2, 'z' : 1, 'a' : 3 }
</code></pre>
<p>How can I print this to shell in the following format :- </p>
<pre><code> alphabet      number
----------------------
    a            3
    y            2
    x            1
    z            1
</code></pre>
<p>My code to print the dictionary to shell :- (ignoring printing 'alphabet,'number' and '-----')</p>
<pre><code>for key,value in sorted(d.items()):
    print("{:&gt;10}{:&gt;20}".format(key,value))
</code></pre>
<p>So I know how to use the .format method to print the dictionary, but looking at the desired output I cannot understand how to print the dictionary by largest value to the smallest value.</p>
<p>Any suggestions?!</p>
</div>
<div class="post-text" itemprop="text">
<p>Iterate like this:</p>
<pre><code>for k in sorted(d, key=d.get, reverse=True):
    print("{:&gt;10}{:&gt;20}".format(k, d[k]))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>It would be,</p>
<pre><code>&gt;&gt;&gt; d = { 'x' : 1, 'y' : 2, 'z' : 1, 'a' : 3 }
&gt;&gt;&gt; for i,j in sorted(d.items(), key=lambda x: x[1], reverse=True):
    print("{:&gt;10}{:&gt;20}".format(i,j))

</code></pre>
