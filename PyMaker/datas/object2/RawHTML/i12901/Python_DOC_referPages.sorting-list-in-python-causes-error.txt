for x in lista.sort():

<div class="post-text" itemprop="text">
<p>The <code>sort</code> method sorts the list in-place; it always returns <code>None</code>.
What you can do:</p>
<pre><code>lista = [ 1, .89, .65, .90]
lista.sort()

for x in lista:
    print (x)
</code></pre>
<p>Or, as @Delgan pointed out, you can use the <code>sorted</code> function, which returns the sorted list:</p>
<pre><code>lista = [ 1, .89, .65, .90]

for x in sorted(lista):
    print (x)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><code>.sort()</code> doesn't return anything.  If you want to modify the list to be sorted you can do</p>
<pre><code>lista = [ 1, .89, .65, .90]
lista.sort()
for x in lista:
    print (x)
</code></pre>
<p>or if you want to keep the list in the original order you can use <code>sorted</code> which returns a new list of the sorted elements:</p>
<pre><code>lista = [ 1, .89, .65, .90]

for x in sorted(lista):
    print (x)
</code></pre>
</div>

<pre><code>lista = [1, .89, .65, .90]
lista.sort()

for x in lista:
    print(x)
</code></pre>
<p>Or simply use <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow"><code>sorted()</code></a>:</p>
<pre><code>lista = [1, .89, .65, .90]

for x in sorted(lista):
    print (x)
</code></pre>
<p><code>sorted()</code> return a new list but sorted, note that it doesn't change the current list.</p>
