<p>In command line I am able to pass arguments to a python file as:</p>
<pre><code>python script.py arg1 arg2
</code></pre>
<p>I can than retrieve <code>arg1</code> and <code>arg2</code> within <code>script.py</code> as:</p>
<pre><code>import sys
arg1 = sys.argv[1]
arg2 = sys.argv[2]
</code></pre>
<p>However, I would like to send keyword arguments to a python script, and retrieve them as a dictionary:</p>
<pre><code>python script.py key1=value1 key2=value2
</code></pre>
<p>Then I would like to access the keyword arguments as a dictionary within python:</p>
<pre><code>{'key1' : 'value1', 'key2' : 'value2'}
</code></pre>
<p>Is this possible?</p>

<div class="post-text" itemprop="text">
<p>I think what you're looking for is the argparse module <a href="https://docs.python.org/dev/library/argparse.html" rel="noreferrer">https://docs.python.org/dev/library/argparse.html</a>.</p>
<p>It will allows you to use command line option and argument parsing.</p>
<p>e.g. Assume the following for script.py</p>
<pre><code>import argparse

if __name__ == '__main__':
   parser = argparse.ArgumentParser()
   parser.add_argument('--arg1')
   parser.add_argument('--arg2')
   args = parser.parse_args()

   print args.arg1
   print args.arg2

   my_dict = {'arg1': args.arg1, 'arg2': args.arg2}
   print my_dict
</code></pre>
<p>Now, if you try:</p>
<pre><code>  $ python script.py --arg1 3 --arg2 4
</code></pre>
<p>you will see:</p>
<pre><code>3
4
{'arg1': '3', 'arg2': '4'}
</code></pre>
<p>as output. I think this is what you were after.</p>
<p>But read the documentation, since this is a <strong><em>very</em></strong> watered down example of how to use argparse. For instance the '3' and '4' I passed in are viewed as str's not as integers</p>
