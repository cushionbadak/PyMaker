<div class="post-text" itemprop="text">
<p>Try this simple regex:</p>
<pre><code>({{(\w+):(\w+)}})
</code></pre>
<p>It returns the full match and elements:</p>
<blockquote>
<p>MATCH 1<br/>
  1.  [4-23]  <code>{{dsfdss:dssssasa}}</code><br/>
  2.  [6-12]  <code>dsfdss</code><br/>
  3.  [13-21] <code>dssssasa</code> </p>
</blockquote>
<p>Try it live <a href="https://regex101.com/r/rG2tT9/1" rel="nofollow">here</a></p>
<p><strong>Python code:</strong></p>
<pre><code>import re
string = u'fdfd{{dsfdss:dssssasa}} fdsfdsfds'
pattern = re.compile(ur'({{(\w+):(\w+)}})')
print(re.findall(pattern, string))
</code></pre>
<p>Edit:
For only the inner part, the regex is <code>{{(\w+:\w+)}}</code></p>
</div>

<p>You can use the following regex:</p>
<pre><code>(\{\{([^\:]+)\:([^\}]+)\}\}
</code></pre>
