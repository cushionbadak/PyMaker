<p>so i want to remove an item from a list and store it into the variable at the same time in python. I tried a sample code like this:</p>
<pre><code>rvals = []
rvals.append("row")
r = rvals.remove("row")
print(r)
</code></pre>
<p>but it turns out this doesnt really work and it gives r a NoneType value instead of removing what i wanted and storing it. Is there a way to do this?</p>

<div class="post-text" itemprop="text">
<p><strong>list.remove(x)</strong></p>
<blockquote>
<p>Remove the first item from the list whose value is x. It is an error
  if there is no such item.</p>
</blockquote>
<p>So, as stated in the docs, you will not get the value returned.</p>
<p><strong>This can help:</strong></p>
<pre><code>value = "row"

rvals = []
rvals.append(value)
print rvals.pop(rvals.index(value))
</code></pre>
<p>Or just use <strong>pop()</strong>, if you only want to remove and get the last inserted item:</p>
<pre><code>value = "row"

rvals = []
rvals.append(value)
print rvals.pop()
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>row
</code></pre>
