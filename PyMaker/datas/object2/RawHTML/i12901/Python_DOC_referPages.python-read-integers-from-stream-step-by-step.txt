<p>Python doesn't special-case such a specific form of input for you. By default, the <a href="https://docs.python.org/3/library/functions.html#input" rel="nofollow"><code>input()</code></a> function then reads a line from input (delimited by newline character) and converts it to a string.</p>
<p>You'll have to use the <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>split()</code></a> to separate the values.</p>
