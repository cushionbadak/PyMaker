<div class="post-text" itemprop="text">
<p>Do you mean do the covert in Python?</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; t = datetime.strptime('07 Aug 2008, 16:25', '%d %b %Y, %H:%M')
&gt;&gt;&gt; t.strftime('%Y-%m-%d %H:%M:%S')
'2008-08-07 16:25:00'
&gt;&gt;&gt; 
</code></pre>
<p>Check <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow">the document</a> for more details.</p>
