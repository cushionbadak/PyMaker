<div class="post-text" itemprop="text">
<p>Iterate the <code>data</code> list items, and build new dictionary with <code>row[1]</code> as a key and <code>row[:1] + row[2:]</code> for values (list items):</p>
<pre><code>&gt;&gt;&gt; data = [
...     ['1','A','D','X'],
...     ['2','A','D','X'],
...     ['3','A','C','X'],
...     ['1','B','F','X'],
...     ['2','B','F','X'],
... ]

&gt;&gt;&gt; d = {}
&gt;&gt;&gt; for row in data:
...     d.setdefault(row[1], []).append(row[:1] + row[2:])
... 
&gt;&gt;&gt; d
{'B': [['1', 'F', 'X'], ['2', 'F', 'X']],
 'A': [['1', 'D', 'X'], ['2', 'D', 'X'], ['3', 'C', 'X']]}
</code></pre>
<p><a href="https://docs.python.org/3/library/stdtypes.html#dict.setdefault" rel="nofollow"><code>dict.setdefault</code></a> is used to populate the dictionary with a empty list when there's no key in the dictionary, so the <code>list.append</code> call can be done safely.</p>
<hr/>
<p>If you use <a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow"><code>collections.defaultdict</code></a>, it could be simpler:</p>
<pre><code>&gt;&gt;&gt; from collections import defaultdict
&gt;&gt;&gt; d = defaultdict(list)
&gt;&gt;&gt; for row in data:
...     d[row[1]].append(row[:1] + row[2:])
... 
&gt;&gt;&gt; d
defaultdict(&lt;type 'list'&gt;, {
    'A': [['1', 'D', 'X'], ['2', 'D', 'X'], ['3', 'C', 'X']],
    'B': [['1', 'F', 'X'], ['2', 'F', 'X']]})
&gt;&gt;&gt; dict(d)  # to convert it back to dict
{'A': [['1', 'D', 'X'], ['2', 'D', 'X'], ['3', 'C', 'X']],
 'B': [['1', 'F', 'X'], ['2', 'F', 'X']]}
</code></pre>
