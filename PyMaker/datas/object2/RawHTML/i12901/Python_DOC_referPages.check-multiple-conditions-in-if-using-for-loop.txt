<div class="post-text" itemprop="text">
<p>Simply use <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow"><code>all()</code></a> here:</p>
<pre><code>if all(a[i]==b[i] for i in range(1, 5)):
</code></pre>
<p>From the document:</p>
<blockquote>
<p>Return <code>True</code> if all elements of the iterable are true (or if the iterable is empty).</p>
</blockquote>
<p>And <code>(a[i]==b[i] for i in range(1, 5))</code> return a <a href="https://wiki.python.org/moin/Generators" rel="nofollow"><strong>generator</strong></a>(it's an iterable), then <code>all()</code> return <code>True</code> if all elements in that generator is <code>True</code>, else <code>False</code>. Like your code does.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do </p>
<pre><code>if all(a[i]==b[i] for i in range(1,5)):
    # logic here
</code></pre>
<p>Usage of <code>all()</code>:</p>
<ol>
<li>if all the elements in iterable  are <code>True</code> returns <code>True</code> </li>
<li>if one lement is <code>False</code> it return <code>False</code></li>
</ol>
<p>eg: <code>all([True,True])</code> --&gt; <code>True</code>
<code>all([False,True])</code>--&gt; <code>False</code></p>
</div>
