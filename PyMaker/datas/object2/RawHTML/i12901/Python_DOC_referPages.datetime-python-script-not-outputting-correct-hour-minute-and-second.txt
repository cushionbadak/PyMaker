<p>I have the following code:</p>
<pre><code>import datetime
Today = datetime.date.today()
print (Today.strftime("%m/%d/%Y-%H:%M:%S")) 
</code></pre>
<p>And get the output <code>11/28/2015-00:00:00</code>. How do I make it display the correct hour, minute, and second instead of just zeros?</p>
</div>
<div class="post-text" itemprop="text">
<p>Just use <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.now" rel="nofollow"><code>datetime.datetime.now()</code></a> instead of <a href="https://docs.python.org/3/library/datetime.html#datetime.date.today" rel="nofollow"><code>datetime.date.today()</code></a>:</p>
<pre><code>In [8]: today = datetime.datetime.now()

In [9]: print (today.strftime("%m/%d/%Y-%H:%M:%S"))
11/29/2015-10:27:32
</code></pre>
