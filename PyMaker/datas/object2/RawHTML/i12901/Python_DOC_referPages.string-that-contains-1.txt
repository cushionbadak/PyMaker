<div class="post-text" itemprop="text">
<p>I have a script that scrolls through all the folders of unzipped files and get information of the contents.</p>
<p>The problem is in the name of files such as:</p>
<pre><code>filename="SearchView$10.smali"
</code></pre>
<p>Because if I want to get the stats the $1 in the string disappears,and I get the following error:</p>
<pre><code>    subprocess.check_output("stat "+ filename,shell=True)
cannot open `SearchView0.smali' (No such file or directory)
</code></pre>
<p>How can I fix it?</p>
</div>
<div class="post-text" itemprop="text">
<p>Avoid <code>shell=True</code>, <a href="https://docs.python.org/3/library/subprocess.html#security-considerations" rel="nofollow">it leads to security issues</a>. And it is also at the root of your problem, as the <code>$1</code> is interpreted.</p>
<p>Do this instead:</p>
<pre><code>subprocess.check_output(["stat", filename])
</code></pre>
</div>
