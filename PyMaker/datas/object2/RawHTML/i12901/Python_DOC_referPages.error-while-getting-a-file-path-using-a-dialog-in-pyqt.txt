<p>I am using a dialog to get a filename path. I have managed to get it running and saving it to a local variable but Qt does not return the file path written in the pythonic way.</p>
<p>My cood is simple as that:</p>
<pre><code>    u = str( QtGui.QFileDialog.getOpenFileName())
</code></pre>
<p>but it returns me something like this:</p>
<pre><code>C:/Artur/PC/Trabalho Darlan/
</code></pre>
<p>instead</p>
<pre><code>C:\Artur\PC\Trabalho Darlan\
</code></pre>
<p>I have tried to replace / for \ with no success. What should I do?</p>
</div>
<div class="post-text" itemprop="text">
<p>Qtreturns all paths using / as the separator. </p>
<p>You can use the <code>abspath()</code> method:</p>
<pre><code>os.path.abspath(u)
</code></pre>
<p>Or the toNativeSeparators method provided by Qt's QDir:</p>
<pre><code>QDir.toNativeSeparators(u)
</code></pre>
</div>
