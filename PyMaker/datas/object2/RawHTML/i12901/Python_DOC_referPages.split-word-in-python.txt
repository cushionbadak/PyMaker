<p>I am trying to use <code>re.split</code> in python. I want to remove all these characters like <code>" , ; &lt; &gt; { } [ ] / \ ? !</code> .I am trying to do something like this-</p>
<pre><code>re.split("[, \_!?,;:-]+", word)
</code></pre>
<p>How can I add characters like <code>"</code> <code>(</code> <code>)</code> or <code>&lt;</code> <code>&gt;</code> <code>'</code> so that they can also be removed?</p>
<p><strong>Edit</strong></p>
<pre><code>re.split('\W+',word)
</code></pre>
<p>This works fine but it is not removing underscore symbol. How can I also remove underscore?</p>

<div class="post-text" itemprop="text">
<p>Try:</p>
<pre><code>re.split('\W+|\_', word)
</code></pre>
<p>Also just remove them:</p>
<pre><code>re.sub('\W+|\_', '', word)
</code></pre>
<p>Take a look at <a href="https://docs.python.org/3/library/re.html#regular-expression-syntax" rel="nofollow">the document</a> for more details.</p>
