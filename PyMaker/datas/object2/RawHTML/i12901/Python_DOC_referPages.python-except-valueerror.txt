<p><code>except</code> does not go with <code>if/elif/else</code>. Instead, it goes with <code>try</code>, like so:</p>
<pre><code>user_input = input('How old are you?: ')
try:
    age = int(user_input) #this can raise ValueError
    if age &gt;= 18:
        print('You are an adult')
    elif age &lt; 18:
        print('You are quite young')
except ValueError:
    print('Please type in an integer') 
</code></pre>
