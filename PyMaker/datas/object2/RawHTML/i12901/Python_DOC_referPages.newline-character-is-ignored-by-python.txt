<p>When I'm trying to write a string <code>b'&lt;?xml version="1.0" encoding="utf-8"?&gt; \n&lt;response list="true"&gt;\n &lt;audio&gt;\n  &lt;aid&gt;412317542&lt;/aid&gt;\n</code>... to a new .xml file, it gets written as a <em>single line</em>, ignoring all <code>\n</code> characters. </p>
<p>Why is it happening, and how do I get <em>several lines</em>? </p>
<p>My code:</p>
<pre><code>page = urlopen(url)
html = page.read()    # Here I get the xml data
with open('output.xml', 'w', encoding='utf-8') as f:
    f.write(str(html))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Your string looks like is <a href="https://docs.python.org/3/library/functions.html#bytes" rel="nofollow">bytes object</a>, so you should use <a href="https://docs.python.org/3/library/stdtypes.html#bytes.decode" rel="nofollow"><code>bytes.decode()</code></a>, not just <a href="https://docs.python.org/3/library/stdtypes.html#str" rel="nofollow"><code>str()</code></a>:</p>
<pre><code>page = urlopen(url)
html = page.read()    # Here I get the xml data

with open('output.xml', 'w', encoding='utf-8') as f:
    f.write(html.decode(page.headers.get_content_charset()))
</code></pre>
</div>
