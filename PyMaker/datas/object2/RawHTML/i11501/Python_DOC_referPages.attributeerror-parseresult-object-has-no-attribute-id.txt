<div class="post-text" itemprop="text">
<p>The <code>urlparse()</code> function doesn't parse out the query string into key-value pairs; it only returns the constituent parts of the URL (which includes the query string part, but also the scheme, the hostname, the fragment identifier, etc).</p>
<p>You'll need to parse out the query string after extracting it from the URL; use the <a href="https://docs.python.org/3/library/urllib.parse.html#urllib.parse.parse_qs" rel="nofollow"><code>urllib.parse.parse_qs()</code> function</a>:</p>
<pre><code>from urllib.parse import parse_qs, urlparse

def getID(url):
    query_string = urlparse(url).query
    return parse_qs(query_string)['id'][0]
</code></pre>
