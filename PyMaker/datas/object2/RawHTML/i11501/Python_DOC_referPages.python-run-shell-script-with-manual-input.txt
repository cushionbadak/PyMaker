<div class="post-text" itemprop="text">
<p>As the earlier answer explained <a href="https://docs.python.org/2/library/subprocess.html#popen-constructor" rel="nofollow"><code>subprocess.Popen</code></a> can be used to create process that can be interacted with <a href="https://docs.python.org/2/library/subprocess.html#subprocess.Popen.communicate" rel="nofollow"><code>communicate</code></a>. <code>communicate</code> takes string as a parameter that will be passed to the created process and returns tuple <code>(stdout, stderr)</code>. Below is a short example of two Python scripts communicating with it:</p>
<p><strong>Child</strong></p>
<pre><code>nums = raw_input()
print sum((int(n) for n in nums.split()))
</code></pre>
<p><strong>Parent</strong></p>
<pre><code>import subprocess

p = subprocess.Popen(['python', 'test.py'], stdin=subprocess.PIPE, stdout=subprocess.PIPE)
out, err = p.communicate('3 4 5')
print 'From other process: ' + out
</code></pre>
<p><strong>Output</strong></p>
<pre><code>From other process: 12
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You probably want to use the <a href="https://docs.python.org/3/library/subprocess.html#subprocess.Popen.communicate" rel="nofollow"><code>subprocess.Popen.communicate()</code></a> function. The docs are quite expressive.</p>
