<div class="post-text" itemprop="text">
<p>I assume from your <code>input()</code> that you use Python3.</p>
<p>Then to write in a file you can do so:</p>
<pre><code>with open("filename.txt", "w") as myfile:
    myfile.write("firstname={}\n".format(firstname))
    myfile.write("surname={}\n".format(surname))
    myfile.write("password={}\n".format(password))
</code></pre>
<p>But I am not sure exactly what you try to achieve... You could also simply write all on a single line without descriptions:</p>
<pre><code>    myfile.write("{} {} {}\n".format(firstname, surname, password))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The easiest way to do it is:</p>
<pre><code>with open('file\path\and\name.txt', 'w') as f:
    f.write("{firstname};{surname};{password}".format(firstname=firstname, surname=surname, password=password)
</code></pre>
<p>Personally I would recommend you to look at the YAML (PyYaml) module for storing data. Also, it's a bad idea to store a password unencrypted.</p>
