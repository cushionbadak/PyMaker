<div class="post-text" itemprop="text">
<p>Use <code>split()[0]</code> to get only the date for each entry, <a href="https://docs.python.org/3/library/functions.html#func-set" rel="nofollow"><code>set</code></a> to remove dupliactes, <a href="https://docs.python.org/3/library/functions.html#func-list" rel="nofollow"><code>list</code></a> to make a list from the set, finally <a href="https://docs.python.org/3/library/functions.html#sorted" rel="nofollow"><code>sorted</code></a> to sort the list.:</p>
<pre><code>sorted(list(set(x.split()[0] for x in l)))
</code></pre>
<p>Result:</p>
<pre><code>['2010-01-29',
 '2014-02-05',
 '2014-05-23',
 '2014-05-24',
 '2014-05-27',
 '2014-05-31',
 '2014-06-10',
 '2014-06-11']
</code></pre>
</div>

<p>You can use this:  <code>sorted(list(set(i.split()[0] for i in l)))</code></p>
<p>Sample run:  </p>
<pre><code>&gt;&gt;&gt; l=['2010-01-29 07:30:02', '2010-01-29 07:55:04', '2014-02-05 08:51:39', '2014-05-23 01:29:06', 
... '2014-05-23 01:29:46', '2014-05-23 01:39:25', '2014-05-24 01:56:42', '2014-05-27 06:55:58', 
... '2014-05-27 07:10:48', '2014-05-31 02:04:58', '2014-06-10 07:41:12', '2014-06-11 09:56:48', 
... '2014-06-11 10:28:52']
&gt;&gt;&gt; l
['2010-01-29 07:30:02', '2010-01-29 07:55:04', '2014-02-05 08:51:39', '2014-05-23 01:29:06', '2014-05-23 01:29:46', '2014-05-23 01:39:25', '2014-05-24 01:56:42', '2014-05-27 06:55:58', '2014-05-27 07:10:48', '2014-05-31 02:04:58', '2014-06-10 07:41:12', '2014-06-11 09:56:48', '2014-06-11 10:28:52']
&gt;&gt;&gt; sorted(list(set(i.split()[0] for i in l)))

<p>Using collections.Counter</p>
<pre><code>&gt;&gt;&gt; my_list = ['2010-01-29 07:30:02', '2010-01-29 07:55:04', '2014-02-05 08:51:39', '2014-05-23 01:29:06',
... '2014-05-23 01:29:46', '2014-05-23 01:39:25', '2014-05-24 01:56:42', '2014-05-27 06:55:58',
... '2014-05-27 07:10:48', '2014-05-31 02:04:58', '2014-06-10 07:41:12', '2014-06-11 09:56:48',
... '2014-06-11 10:28:52',]
&gt;&gt;&gt; my_counter = collections.Counter(x.split()[0] for x in my_list)
&gt;&gt;&gt; my_counter
Counter({'2014-05-23': 3, '2014-06-11': 2, '2014-05-27': 2, '2010-01-29': 2, '2014-05-31': 1, '2014-06-10': 1, '2014-05-24': 1, '2014-02-05': 1})
&gt;&gt;&gt; my_counter.keys()
['2014-05-31', '2014-06-10', '2014-06-11', '2014-05-24', '2014-05-27', '2010-01-29', '2014-05-23', '2014-02-05']
</code></pre>
<p>Using <code>itertools.groupby</code>:</p>
<pre><code>&gt;&gt;&gt; result = [x for x,y in itertools.groupby(sorted(x.split()[0] for x in my_list))]
&gt;&gt;&gt; result
['2010-01-29', '2014-02-05', '2014-05-23', '2014-05-24', '2014-05-27', '2014-05-31', '2014-06-10', '2014-06-11']
</code></pre>
