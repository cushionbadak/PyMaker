<p>I'm trying to add an is_empty(self) method. Which will return true if the hash table maps no keys to values, else false. 
This is what I have at the moment I'm just unsure how to approach the is_empty function using self. </p>
<pre><code>class MyHashTable:

    def __init__(self, capacity): 
        self.capacity = capacity 
        self.slots = [None] * self.capacity

    def __str__(self): 
        return str(self.slots )

    def is_empty(self) 
        pass
</code></pre>

<div class="post-text" itemprop="text">
<p>As <code>self.slots</code> is a <code>list</code>, and the goal is to test that all the elements are <code>None</code>. I would suggest:</p>
<pre><code>def is_empty(self) 
   return self.slots.count(None) == len(self.slots)
</code></pre>
<p>See <a href="https://stackoverflow.com/questions/6518394/how-to-check-if-all-items-in-the-list-are-none">How to check if all items in the list are None?</a> for my answer there and others.</p>
</div>
<div class="post-text" itemprop="text">
<p>Another approach can be using a built-in function called <strong>all()</strong>. You may check the <a href="https://docs.python.org/2/library/functions.html#all" rel="nofollow">reference</a> for further detail.</p>
<pre><em>Return True if all elements of the iterable are true (or if the iterable is empty).</em></pre>
<p>Sample Code:</p>
<pre><code>def is_empty(self) 
    return all(item is None for item in self.slots)
</code></pre>
</div>
<span class="comment-copy">It would probably be worth to implement <a href="https://docs.python.org/3/reference/datamodel.html#object.__len__" rel="nofollow noreferrer"><code>__len__()</code></a> first, then you can use <code>len(table) == 0</code> (or just <code>if table: ...</code>) for empty checks.</span>
<span class="comment-copy">One stylistic issue: Comparing with <code>None</code> should be done by <code>item is None</code>.</span>
