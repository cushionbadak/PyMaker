<div class="post-text" itemprop="text">
<p>I have a URL string which is <code>https://example.com/about/hello/</code></p>
<p>I want to split string as <code>'https://example.com', 'about' ,'hello'</code></p>
<p>How to do this ??</p>
</div>
<div class="post-text" itemprop="text">
<p>There are lots of ways to do this. You could use <code>re.split()</code> to split on a regular expression, for instance. </p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; re.split(r'\b/\b', 'https://example.com/about/hello/')
['https://example.com', 'about', 'hello']
</code></pre>
<p><code>re</code> is part of the standard library, documented here.
<a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow">https://docs.python.org/3/library/re.html#re.split</a>
The regex itself uses <code>\b</code> which means a boundy between a "word" character and a "non-word" character. You can use regex101 to explore how it works. <a href="https://regex101.com/r/mY8fV8/1" rel="nofollow">https://regex101.com/r/mY8fV8/1</a></p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <code>urlparse</code> to correctly parse a URL:</p>
<pre><code>import urlparse

url = 'https://example.com/about/hello/'
parts = urlparse.urlparse(url)
paths = [p for p in parts.path.split('/') if p]

print 'Scheme:', parts.scheme       # https
print 'Host:', parts.netloc         # example.com
print 'Path:', parts.path           # /about/hello/
print 'Paths:', paths               # ['about', 'hello']
</code></pre>
<p>At the end of the day, the information you want are in the <code>parts.scheme</code>, <code>parts.netloc</code> and <code>paths</code> variables.</p>
</div>
<div class="post-text" itemprop="text">
<p>You may do this :</p>
<ol>
<li>First split by '/'</li>
<li>Then join by '/' only before the 3rd occurance</li>
</ol>
<p>Code:</p>
<pre><code>text="https://example.com/about/hello/"
groups = text.split('/')
print( "/".join(groups[:3]),groups[3],groups[4])
</code></pre>
<p>Output:</p>
<pre><code>https://example.com about hello
</code></pre>

<p>Inspired in <a href="https://stackoverflow.com/users/459745/hai-vu">Hai Vu's</a> <a href="https://stackoverflow.com/a/37076947/841339">answer</a>. This solution is for Python 3</p>
<pre><code>from urllib.parse import urlparse

url = 'https://example.com/about/hello/'
parts = [p for p in urlparse(url).path.split('/') if p]
parts.insert(0, ''.join(url.split('/')[:3]))
</code></pre>
