<p>You could use ljust and rjust</p>
<p><a href="http://www.tutorialspoint.com/python/string_ljust.htm" rel="nofollow">http://www.tutorialspoint.com/python/string_ljust.htm</a>
<a href="http://www.tutorialspoint.com/python/string_rjust.htm" rel="nofollow">http://www.tutorialspoint.com/python/string_rjust.htm</a></p>
<pre><code>print("A String".ljust(30, " ") + "Another String")
</code></pre>
<p>results in</p>
<pre><code>A String                      Another String
</code></pre>
<p>This is a possible way to do the trick:</p>
<pre><code>#!/usr/bin/python

import sys
import socket
print("Server Name".ljust(30, " ") + "IP ADRESS")
print("="*39)
with open("mylabList.txt", 'r') as f:
    for host in f:
        print("{0[0]}\t{0[2][0]}".format(socket.gethostbyname_ex(host.rstrip())))
</code></pre>
