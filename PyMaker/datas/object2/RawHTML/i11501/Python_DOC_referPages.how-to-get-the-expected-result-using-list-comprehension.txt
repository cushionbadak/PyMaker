<div class="post-text" itemprop="text">
<p>I have the following lists</p>
<pre><code>a=[1,2,3]
b=[4,5,6]
c=[a,b]
</code></pre>
<p>i need to combine both list a and b.</p>
<p>result should be like <code>[1,2,3,4,5,6]</code></p>
<p>i tried with list comprehension</p>
<pre><code>[x for x in i for i in c]
</code></pre>
<p>output</p>
<pre><code>[3, 3, 4, 4, 5, 5]
</code></pre>
<p>How can i get the result as <code>[1,2,3,4,5,6]</code> using list comprehension.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can just do:</p>
<pre><code>a + b
</code></pre>
<p>If you must use list comprehension:</p>
<pre><code>In [10]: a = [1, 2, 3]

In [11]: b = [4, 5, 6]

In [12]: c = [a, b]

In [13]: [j for i in c for j in i]
Out[13]: [1, 2, 3, 4, 5, 6]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/itertools.html#itertools.chain" rel="nofollow"><code>itertools.chain</code></a>.</p>
<pre><code>import itertools

a=[1,2,3]
b=[4,5,6]

c = list(itertools.chain(a, b))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You are concatenating, use <code>+</code> to do so:</p>
<pre><code>c = a + b
</code></pre>
<p>If you are concatenating an arbitrary number of lists, use <a href="https://docs.python.org/2/library/itertools.html#itertools.chain.from_iterable" rel="nofollow"><code>itertools.chain.from_iterable()</code></a>:</p>
<pre><code>from itertools import chain

list_of_lists = [a, b]
c = list(chain.from_iterable(list_of_lists))
</code></pre>
<p>Note that if all you need to do is iterate over the concatenation result, you can leave of the <code>list()</code> call altogether.</p>
<p>Do <strong>not</strong> use <code>sum()</code> for this; that leads to quadratic behaviour as intermediate results are built for every element summed, which takes a full loop.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do it with + operation</p>
<pre><code>a = [1, 2, 3]
b = [3, 4, 5]
c = a + b # Equal [1, 2, 3, 3, 4, 5]
</code></pre>
</div>
