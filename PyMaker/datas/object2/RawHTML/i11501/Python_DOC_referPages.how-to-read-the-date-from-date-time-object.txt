<p>If you have a string of this format you can just split on whitespace</p>
<p><code>dateString.split(" ")[0]</code></p>
<p>where <code>dateString</code> is your input string</p>

<div class="post-text" itemprop="text">
<p>If you have that date set to a variable, such as <code>time_and_date</code>, you would use <code>time_and_date[0, 9] = date</code> to select the first 9 characters which are the date. the first '2' is the 0, and the '8' is the 9. You can also change this to select the time.</p>
