<p>Seems to me, you should use another data structure than <code>list</code> as <code>table</code> variable. I suppose that <code>dict</code> should be nice for the purpose.</p>
<p>I've just made a quick try to mimic your PHP code in Python:</p>
<pre><code>table = {} # use dictionary instead of list here
for char in text:
    if char not in table:
        table[char] = []
    # do your stuff with table[char]
    pass 
</code></pre>
<p>Also, I suggest you look into <a href="https://docs.python.org/3/library/collections.html#collections.defaultdict" rel="nofollow">https://docs.python.org/3/library/collections.html#collections.defaultdict</a></p>
<p>With the class the code could be rewritten in the following way:</p>
<pre><code>import collections

table = collections.defaultdict(list)
for char in text:
    # do your stuff with table[char], empty list is created by default
    pass
</code></pre>
