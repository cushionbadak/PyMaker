<p>String examples --&gt; Expected output</p>
<pre><code>"    1     1    1-1 SOME MESSAGE HERE" --&gt; ['1','1','1-1','SOME MESSAGE HERE']

"   71    91    1-1 SOME MESSAGE HERE" --&gt; ['71','91','1-1','SOME MESSAGE HERE']

"  171   191    1-1 SOME MESSAGE HERE" --&gt; ['171','191','1-1','SOME MESSAGE HERE']

" 1171  1191    1-1 SOME MESSAGE HERE" --&gt; ['1171','1191','1-1','SOME MESSAGE HERE']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<h3>In Python 2.7</h3>
<pre><code>x = "    1     1    1-1 SOME MESSAGE HERE"
y = x.split(None, 3)
print (y)  # ['1', '1', '1-1', 'SOME MESSAGE HERE']
</code></pre>
<h3>In Python 3</h3>
<pre><code>x = "    1     1    1-1 SOME MESSAGE HERE"
y = x.split(maxsplit=3)
print (y)  # ['1', '1', '1-1', 'SOME MESSAGE HERE']
</code></pre>
</div>
<span class="comment-copy">Use .split with a maxsplit arg <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer">docs.python.org/3/library/stdtypes.html#str.split</a></span>
