<p>Why do you need a loop? Use <code>in</code> to check if the input value exists in the list and follow accordingly:</p>
<pre><code>option=input("option: ")
iplist=['192.168.1.1', '192.168.1.2', '192.168.1.254']

if option in iplist:
   # do the rest
    pass
</code></pre>
<p><strong>OR</strong>:</p>
<p>If you want to get the <code>Index</code> of the element in the list:</p>
<pre><code>for index, elem in enumerate(iplist):
    if option == elem:
        print("Element found at Index: {}".format(index))
</code></pre>
<p><strong>OUTPUT</strong>:</p>
<pre><code>option: 192.168.1.2
Element found at Index: 1
</code></pre>
<p><strong>EDIT 2</strong>:</p>
<p>There are a couple of things to note first:</p>
<ol>
<li><p>Take the input from the user and convert it to an <code>int</code> since you cannot access the list with str indexing:</p></li>
<li><p>I still do not see a point of loop</p></li>
</ol>
<p>So:</p>
<pre><code>import subprocess
option= int(input("option: "))    # 1
iplist=['192.168.1.1', '192.168.1.2', '192.168.1.254']

nub = iplist[option]
subprocess.call(["ping", nub])
</code></pre>
