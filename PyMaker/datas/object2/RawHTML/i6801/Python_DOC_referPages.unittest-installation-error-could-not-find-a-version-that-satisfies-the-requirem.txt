<div class="post-text" itemprop="text">
<p>If you tried this:</p>
<pre><code>$ pip install unittest
Collecting unittest

  Could not find a version that satisfies the requirement unittest (from versions: )
No matching distribution found for unittest
</code></pre>
<p>It's normal to have an error because <a href="https://pypi.python.org/pypi/unittest/0.0" rel="noreferrer"><code>unittest</code></a> is not a valid package on Pypi. The only distribuable is <code>unittest2-0.0.0.tar.gz</code> but the name is not correct, it should be <code>unittest-0.0.0.tar.gz</code> (without "2"). In fact I think it's an error and if you want to use this one you need to install <a href="https://pypi.python.org/pypi/unittest2" rel="noreferrer"><code>unittest2</code></a> (which is a backport of <code>unittest</code> for Python 2.6).</p>
<p>But <a href="https://docs.python.org/3/library/unittest.html" rel="noreferrer"><code>unittest</code></a> is a standard module, which you don't need to install:</p>
<pre><code>$ python -m unittest

----------------------------------------------------------------------
Ran 0 tests in 0.000s

OK
</code></pre>
