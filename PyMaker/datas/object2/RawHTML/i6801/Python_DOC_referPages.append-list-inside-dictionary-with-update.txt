<p>if i have below dictionary with one of the element is list, as follow:</p>
<pre><code>myDict = dict(a=1, b='2', c=[])
</code></pre>
<p>how do I update myDict and at the same time append c </p>
<p>e.g.</p>
<pre><code>myDict .update(a='one', b=2, c=append('newValue'))
myDict .update(a='1', b='two', c=append('anotherValue'))
</code></pre>
<p>and the final result should be:</p>
<pre><code>myDict = a='1', b='two', c=['newValue', 'anotherValue']
</code></pre>
<p>in one statement....</p>
</div>
<div class="post-text" itemprop="text">
<p>You can't use <code>append</code> within <code>update</code> because <code>append</code> is trying to perform an inplace operation on the dict value. Try list concatenation instead:</p>
<pre><code>d = dict(a=1, b='2', c=[])
d.update(a='one', b=2, c=d['c'] + ['newValue'])
print(d)
{'a': 'one', 'b': 2, 'c': ['newValue']}
</code></pre>
<p>Or:</p>
<pre><code>d.update(a='one', b=2, c=d['c'] + ['newValue'] + ['anotherValue'])
</code></pre>
</div>
