<div class="post-text" itemprop="text">
<p>You have to specify the command in the <a href="https://docs.python.org/3/library/ftplib.html#ftplib.FTP.storbinary" rel="nofollow noreferrer"><code>FTP.storbinary</code> call</a>.</p>
<blockquote>
<p>Store a file in binary transfer mode. <strong><em>command</em> should be an appropriate <code>STOR</code> command: <code>"STOR filename"</code>.</strong> </p>
</blockquote>
<pre><code>session.storbinary('STOR a csv1.csv', file)  
</code></pre>
