<div class="post-text" itemprop="text">
<p>If my comprehension is correct, you want to timestamp your file name in order to not overwrite it ? 
to generate a timestamp, you shall use datetime module</p>
<pre><code>from datetime import datetime
</code></pre>
<p>then generate a timestamp when you create your log file</p>
<pre><code>my_date = datetime.now()
timestamp = my_date.strftime('%Y-%m-%D_%H-%M-%S') # formatting YYYY-mm-DD_HH-MM-SS - you may format it a different way
# build your log file name
my_file_name = "DailyLogFile" + timestamp + ".txt"
filename = open(my_file_name,'w') ... etc.
</code></pre>
<p>For the different possible options, you may have a look to <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer">https://docs.python.org/3/library/datetime.html</a></p>
</div>
