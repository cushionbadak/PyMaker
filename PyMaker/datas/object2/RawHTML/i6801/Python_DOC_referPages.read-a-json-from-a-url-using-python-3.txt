<p>You could use <a href="http://docs.python-requests.org/en/master/user/quickstart/#json-response-content" rel="nofollow noreferrer">Requests</a> library which is more simple than urllib:</p>
<p>For instance:</p>
<pre><code>import requests

r = requests.get('https://dummy_link')
obj = r.json()
</code></pre>
<p><strong>EDIT</strong></p>
<p>If you want to use urllib, you can do as below:</p>
<pre><code>import urllib.request
import json

with urllib.request.urlopen("https://dummy_link") as f:
    content = f.read()

obj = json.loads(content)
</code></pre>
<p>There is no need to convert the binary content to unicode string.</p>
<p>There is an <a href="https://docs.python.org/3/howto/urllib2.html#urllib-howto" rel="nofollow noreferrer">urllib howto</a> in the official documentation.</p>
