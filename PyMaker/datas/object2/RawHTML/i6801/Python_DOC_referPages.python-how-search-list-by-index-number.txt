<div class="post-text" itemprop="text">
<p>Just iterate over it:</p>
<pre><code>for element in my_list:
    print(element)
</code></pre>
<p>If you want to print the indices as well, use <a href="https://docs.python.org/3/library/functions.html#enumerate" rel="nofollow noreferrer"><code>enumerate()</code></a>:</p>
<pre><code>for index, element in enumerate(my_list):
    print(index, element)
</code></pre>
</div>
