<p>I am writing a <code>config.ini</code> file for my python project. I want to mention the standard date format of a column. </p>
<p>Something like this</p>
<pre><code>prod_db_end_date   = 2017-01-01
prod_db_end_date_format = %Y-%m-%d
</code></pre>
<p>But it interpolates something else and errors out. Could someone please let me know how to use this?</p>
<pre><code>Error traceback - File "C:\ProgramData\Anaconda3\lib\configparser.py", line 443, in _interpolate_some
      "found: %r" % (rest,))
configparser.InterpolationSyntaxError: '%' must be followed by '%' or '(', found: '%Y-%m-%d'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You have three options:</p>
<ul>
<li><p>Disable interpolation, or pick a different interpolation handler. You can set the <code>interpolation</code> option to <code>None</code> to disable interpolation for the whole file:</p>
<pre><code>ConfigParse(interpolation=None)
</code></pre>
<p>See the <a href="https://docs.python.org/3/library/configparser.html#interpolation-of-values" rel="nofollow noreferrer"><em>Interpolation of values</em> section</a>.</p></li>
<li><p>Access that specific value with interpolation disabled, setting <code>raw=True</code> for the <a href="https://docs.python.org/3/library/configparser.html#configparser.ConfigParser.get" rel="nofollow noreferrer"><code>ConfigParser.get()</code> method</a>:</p>
<pre><code>config.get('section', 'prod_db_end_date_format', raw=True)
</code></pre></li>
<li><p>Double the <code>%</code> characters to <code>%%</code>:</p>
<pre><code>prod_db_end_date_format = %%Y-%%m-%%d
</code></pre></li>
</ul>
</div>
