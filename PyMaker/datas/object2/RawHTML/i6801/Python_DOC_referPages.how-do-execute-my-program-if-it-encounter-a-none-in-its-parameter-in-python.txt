<p>None, True and False tests are done with words:</p>
<pre><code>if dice_set_aside_str is not None:
    return 0
</code></pre>
<p>Anything <a href="https://docs.python.org/3/glossary.html?highlight=iterable" rel="nofollow noreferrer">iterable</a> without any items resolves to <code>False</code> in boolean tests. So do empty strings and so does None. This we can combine:</p>
<pre><code>if not dice_set_aside_str:
    return 0
else:
    return len(dice_set_aside_str)
</code></pre>
