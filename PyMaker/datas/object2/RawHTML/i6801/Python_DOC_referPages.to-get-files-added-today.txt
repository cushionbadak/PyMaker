<p>You can use <a href="https://docs.python.org/3/library/os.html#os.stat" rel="nofollow noreferrer">os.stat</a> and/or <a href="https://docs.python.org/3/library/os.html#os.stat" rel="nofollow noreferrer">os.lstat</a> to retrieve file attributes:</p>
<pre><code>import os
for file in os.listdir(SourceFolder):
    f_attr = os.stat(file)
    if datetime.fromtimestamp(f_attr.st_mtime) == date.today():
        print('Good file! %s' % file)
    else:
        print('Bad file! %s' % file)
</code></pre>
