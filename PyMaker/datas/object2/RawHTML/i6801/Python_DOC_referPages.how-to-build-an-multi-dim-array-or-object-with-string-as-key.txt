<div class="post-text" itemprop="text">
<p>This is not how lists or arrays work or are meant to be working. Instead, you should consider using a <code>dictionary</code> like this:</p>
<pre><code>self.arr = {}
self.arr["*.html"] = settings.BASE_DIR + "/html/"
self.arr["*.json"] = settings.BASE_DIR + "/jsons/"
</code></pre>
