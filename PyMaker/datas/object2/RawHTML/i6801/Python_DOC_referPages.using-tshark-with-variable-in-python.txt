<div class="post-text" itemprop="text">
<p>You can use <code>os.system</code> to execute the particular command, in which case you can join the two strings either by using the <code>+</code> operator, string formatting (<code>.format()</code>), string substitution or some other method.</p>
<p>However, consider the case when the user enters the command <code>5; rm -rf /</code> or some other <a href="https://en.wikipedia.org/wiki/Code_injection#Shell_injection" rel="nofollow">malicious</a> command.  Instead of using <code>os.system</code> you might want to take a look at <a href="https://docs.python.org/2/library/subprocess.html#module-subprocess" rel="nofollow">subprocess</a></p>
<p>If you use subprocess you might find the following example handy:</p>
<pre><code>import subprocess
s = raw_input('test&gt;')
subprocess.call(["shutdown", "-s", "-t", "10", "-c", s])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/2/library/os.html#os.system" rel="nofollow"><code>os.system</code></a> takes a <em>string</em> as the argument, so you can use anything that modifies a string. For example, <a href="https://docs.python.org/2/library/stdtypes.html#str.format" rel="nofollow">string formatting</a>:</p>
<pre><code>os.system('shutdown -s -t 10 -c {0}'.format(s))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/2/library/subprocess.html#subprocess.check_call" rel="nofollow">subprocess.check_call</a>, pass a list of args and you can add the variable wherever you like:</p>
<pre><code>from subprocess import check_call

check_call(["shutdown", some_var ,"-s", "-t" "10", "-c"])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Passing place-holder <strong>"%s string alongside os.system"</strong> should work.</p>
<p>os.system() will execute the command in shell.</p>
<pre><code>import os
var = raw_input('test&gt;') # changed "s" variable to "var" to keep clean
os.system("shutdown -%s -t 10 -c", %(var)) # "var" will be passed to %s place holder
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>os.system("shutdown -s -t 10 -c" + s)
</code></pre>
<ol>
<li>The plus sign joins two strings</li>
<li>No quotes are used around <code>s</code>.  That way you get the value the user entered and not just a literal "s".</li>
</ol>
