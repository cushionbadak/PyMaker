<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/functions.html#any" rel="nofollow noreferrer"><code>any</code></a> built-in:</p>
<blockquote>
<p>Return True if any element of the iterable is true. If the iterable is empty, return False.</p>
</blockquote>
<pre><code>if any(elt != 0 for elt in array):
    print("Non zero detected")
else:
    print("All elements are zero")
</code></pre>
<p>As a bonus, if the only elements that evaluates to <code>False</code> Boolean-wise is <code>0</code>, you can simply do:</p>
<pre><code>if any(array):
    print("Non zero detected")
else:
    print("All elements are zero")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can create a mask using conditionals:</p>
<pre><code>mask = array == 0
print(mask) 
array([ True, False, False,  True], dtype=bool)
</code></pre>
<p>You can chain this with a <code>.all</code> or <code>.any</code> call, depending on your use case. If you want to check whether all elements are zero, you'd do:</p>
<pre><code>if (array == 0).all():
    print('All elements are 0!')
</code></pre>
<p>Or, </p>
<pre><code>if (array != 0).any():
    print('Non-zero elements detected!')
</code></pre>
<p>And so on.</p>
<hr/>
<p>You can do the same thing using <code>np.mask</code> as long as you provide two additional arguments:</p>
<pre><code>mask = np.where(array == 0, True, False)
print(mask)
array([ True, False, False,  True], dtype=bool)
</code></pre>
</div>

<span class="comment-copy"><code>where</code> gives you a tuple of arrays, one array per dimension.  If there aren't any non-zeros those arrays will have shape (0,), i.e. no elements.  So if you use <code>where</code> you need to check the length of one of those arrays.  I'd suggest experimenting with <code>where</code> in an interactive session. Looking at its result should make this clearer.</span>
