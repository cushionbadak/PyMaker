<pre><code>while True:
  ask = input("Enter a word: ")

  if not ask.isalpha():
    print("Must be letters")
    continue

  if ask == ask[::-1]:

<span class="comment-copy"><code>input()</code> will return a string (in python3), so maybe you need to check if this string contains only letters ?</span>

<span class="comment-copy"><code>input</code> <i>always</i> returns a string. You want to check if that string contains only letters. A <code>ValueError</code> is the appropriate exception to raise in that case.</span>
