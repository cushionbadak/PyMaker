<div class="post-text" itemprop="text">
<p>You could use <strong><a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer">itertools.groupby()</a></strong>, observe:</p>
<pre><code>import itertools

user_input = input("Please enter a string:")
groups = []

for _, group in itertools.groupby(user_input):
  groups.append(''.join(group))

print('Here is that string split when a character changes: %s' % ', '.join(groups))
</code></pre>
<p><strong>Example usage with given example:</strong></p>
<pre><code>Please enter a string: b444Mrr--&lt;&lt;&lt;]0
Here is that string split when a character changes: b, 444, M, rr, --, &lt;&lt;&lt;, ], 0
</code></pre>
<p><strong><em>N.B.</em></strong> In Python use <a href="https://en.wikipedia.org/wiki/Snake_case" rel="nofollow noreferrer"><strong>snake_case</strong></a> rather than <a href="https://en.wikipedia.org/wiki/Letter_case#Title_Case" rel="nofollow noreferrer"><strong>TitleCase</strong></a> (what you are doing in your attempt) or  <a href="https://en.wikipedia.org/wiki/Camel_case" rel="nofollow noreferrer"><strong>camelCase</strong></a></p>
