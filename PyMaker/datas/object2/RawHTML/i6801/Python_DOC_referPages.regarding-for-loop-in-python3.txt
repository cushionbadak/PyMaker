<div class="post-text" itemprop="text">
<p>Your loop will run twice for i = 3, and i = 4.</p>
<p>The first run:</p>
<pre><code>ans = 0 + 1 * 1 #1
a = 1
b = 1
print(1)
</code></pre>
<p>The second run:</p>
<pre><code>ans = 1 + 1 * 1 # 2
a = 1
b = 2
print(2)
</code></pre>
<p>Now <code>ans</code> is 2. It's not clear why you think it should be 5.</p>

<p>I think this is what you are trying to do:</p>
<pre><code>a, b, c = [int(x) for x in input().split()]
ans = None
for i in range(2,c):
    ans=a + (b*b)
    a=b
    b=ans
print (ans)
</code></pre>
<p>Note that <code>for i in range(2,c)</code> i will be 2 then 3 and then 4. It will not be 5 (=c) it stops at c-1.</p>
<p>Edit: if you just want the loop to run c times you can say <code>for i in range(c)</code>. This way it will run for i=0,1,2,...,c-1.</p>
