<div class="post-text" itemprop="text">
<p>1.<code>python2.7 -m pip install virtualenv</code><br/>
<a href="https://docs.python.org/3/installing/index.html?highlight=pip#work-with-multiple-versions-of-python-installed-in-parallel" rel="nofollow noreferrer">Reference  Python official document</a> </p>
<p>2.You can Install virtualenv for any python interpreter and use for a different python interpreter like:-   </p>
<pre><code>sudo pip3 install virtualenv 
</code></pre>
<p>and then if you want to use python2.7 python interpreter then run following command to make virtualenv:-  </p>
<pre><code>virtualenv --python=/usr/bin/python2.7 virtualenv_name
</code></pre>
<p><strong>NOTE</strong> :- python2.7 interpreter has to be present in <code>/usr/bin/*</code> folder<br/>
 For more on above command see <a href="https://stackoverflow.com/a/45903517/6761181">this</a> answer</p>
