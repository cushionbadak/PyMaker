<div class="post-text" itemprop="text">
<p>If the data is a valid python literal (list, dict etc..) you can use the <code>literal_eval</code> function from pythons built-in <code>ast</code> package. This is better than a solution using <code>eval</code> as it will only evaluate a valid data structure, and does not allow arbitrary code execution. There are almost zero cases where using plain <code>eval</code> is a good idea.</p>
<pre><code>from ast import literal_eval

with open("/Users/jj/Desktop/Courtney_fan.txt","r") as f:
    my_list = literal_eval(f.read())
</code></pre>

<p>As your  <code>.txt</code> file appears to be correct formatted and ist read in as a <code>string</code> the build in function <code>eval()</code> does this for you.</p>
<p>Here an example which is similar to your string in the txt-file: </p>
<pre><code>test = "['a', ['b','c']]"
&gt;&gt;&gt; trial = eval(test)
&gt;&gt;&gt; type(trial)
&lt;class 'list'&gt;
&gt;&gt;&gt; trial
['a', ['b', 'c']]
</code></pre>
<p>For you it would then be: </p>
<pre><code>with open("/Users/jj/Desktop/Courtney_fan.txt") as f:
       sd = f.read()

sd_list = eval(sd)
</code></pre>
