<p>How to Find the first occurrence of a character in a sentence using re and split based on it.</p>
<p>If the sentence is  'a,bc,de'</p>
<p>I want to split based on the first occurrence of ','</p>
<p>so the op will be ['a','bc,de']</p>
<p>I don't want to split based on ',' and concat the rest to from the last element.</p>
<p>I want an regular expression that will only split the first occurrence.</p>
</div>
<div class="post-text" itemprop="text">
<p>in split method, there's a parameter 'maxsplit', if maxsplit is given, at most maxsplit splits are done (thus, the list will have at most maxsplit+1 elements). If maxsplit is not specified or -1, then there is no limit on the number of splits (all possible splits are made)</p>
<pre><code>'a,bc,de'.split(',', 1)
</code></pre>
<p>for your case, just use</p>
<pre><code>'a,bc,de'.split(',', 1)[0]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Don't forget <a href="https://docs.python.org/3.6/library/stdtypes.html#str.partition" rel="nofollow noreferrer">.partition</a>:</p>
<pre><code>&gt;&gt;&gt; 'a,bc,de'.partition(',')
('a', ',', 'bc,de')
</code></pre>
<p>Or, using a maxsep argument to <a href="https://docs.python.org/3.6/library/stdtypes.html#str.split" rel="nofollow noreferrer">.split</a></p>
<pre><code>&gt;&gt;&gt; 'a,bc,de'.split(',',1)
['a', 'bc,de']
</code></pre>
<p>Or, to a <a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow noreferrer">regex split</a>:</p>
<pre><code>&gt;&gt;&gt; re.split(r',','a,bc,de',maxsplit=1)
['a', 'bc,de']
</code></pre>
<p>Or, since you are using Python 3, you can do:</p>
<pre><code>&gt;&gt;&gt; first,*the_rest='a,bc,de'.split(',')
&gt;&gt;&gt; first
'a'
&gt;&gt;&gt; the_rest
['bc', 'de']
</code></pre>
</div>
