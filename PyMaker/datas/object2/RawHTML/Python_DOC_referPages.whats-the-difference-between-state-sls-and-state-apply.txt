<p>What's the difference between state.apply and state.sls? When should I use state.apply vs state.sls?</p>
</div>
<div class="post-text" itemprop="text">
<p><code>state.sls &lt;state_file&gt;</code> will execute the states in a specific file.</p>
<p><code>state.highstate</code> will execute the highstate configured by <code>top.sls</code></p>
<p><code>state.apply</code> will do a <code>state.highstate</code> if no arguments are given and a <code>state.sls &lt;state_file&gt;</code> when a  argument is given.</p>
<p>You can look at <code>state.apply</code> as an easy way of using states command.</p>
</div>
