<pre><code>l = [['GRRSCLOE1', "&lt;DstTzInfo 'America/Santiago' LMT-1 day, 19:17:00 STD&gt;", -3.0, 'RR'], ['GRTFRAIX1', "&lt;DstTzInfo 'Europe/Berlin' LMT+0:53:00 STD&gt;", 1.0, 'P'], ['GRAMDZGP1', "&lt;DstTzInfo 'America/Argentina/Mendoza' LMT-1 day, 19:25:00 STD&gt;", -3.0, 'P+PE'], ['GRASAOTB2', "&lt;DstTzInfo 'America/Sao_Paulo' LMT-1 day, 20:54:00 STD&gt;", -3.0, 'PE'], ['GRTVAPEM4',"&lt;DstTzInfo 'America/Santiago' LMT-1 day, 19:17:00 STD&gt;", -3.0, 'P']]

</code></pre>
<p><strong>output</strong>:</p>
<pre><code>['GRRSCLOE1', 'GRAMDZGP1', 'GRASAOTB2', 'GRTVAPEM4']

<p><strong>Explanations:</strong></p>
<p>First of all your initial list strings should be surrounded by double or simple quotes, if you have single quotes in the string and you do not want to escape them, use double quotes.</p>
<p>Then what you are looking for is list comprehensions:</p>
<ul>
<li>The construction <code>[ &lt;operation on a&gt; for a in list if &lt;condition on a&gt;]</code> will allow you to return a list containing all the elements x of your initial list that fulfill the condition</li>
<li>This would return all the nested lists of your example, as you are interested only by the first element replace <code>&lt;operation on a&gt;</code> by <code>a[0]</code></li>
</ul>
