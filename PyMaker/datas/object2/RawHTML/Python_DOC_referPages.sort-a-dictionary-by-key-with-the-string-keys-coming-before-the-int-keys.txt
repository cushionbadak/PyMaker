<p>I have a dictionary <code>d</code> which looks like:</p>
<pre><code>d = {345: 23, 'c': 56, 'd': 23, 448: 78, 'a': 43, 'b': 43}
</code></pre>
<p>the dictionary keys are of mixed type.</p>
<p>Is it possible to sort the dictionary by key but having the string values first in the dictionary so the desired output in this example would be:</p>
<pre><code>d = {'a': 43,'b': 43,'c': 56, 'd': 23, 345: 23, 448: 78}
</code></pre>
<p>I have tried using:</p>
<pre><code>sortedlist = [(k, d[k]) for k in sorted(a)]
</code></pre>
<p>without success. I have had a look and there are answers for sorting single type dictionaries and how to sort a list but not for mixed type dictionaries.</p>
</div>
<div class="post-text" itemprop="text">
<p>Try this :</p>
<pre><code>d = {345: 23, 'c': 56, 'd': 23, 448: 78, 'a': 43, 'b': 43}
sorteddict = {m : d[m] for m in sorted([k for k in d if type(k)==str])+sorted([k for k in d if type(k)!=str])}
</code></pre>
<p><strong>OUTPUT</strong> :</p>
<pre><code>{'a': 43, 'b': 43, 'c': 56, 'd': 23, 345: 23, 448: 78}
</code></pre>
