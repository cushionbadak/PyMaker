<p>Python itself does not have many predefined functions, but you can use scipy.stats:
@Terminal:</p>
<pre><code>pip install scipy
</code></pre>
<p>@Python:</p>
<pre><code>from scipy.stats import expon

print(expon.pdf(1))
</code></pre>
