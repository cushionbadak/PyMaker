<div class="post-text" itemprop="text">
<p>If you use <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>split</code></a> and don't add a separator then leading and trailing whitespaces will be discarded. If you add the <code>maxsplit</code> parameter only leading whitespaces will be discarded. No need to use <code>strip</code> here.</p>
<pre><code>&gt;&gt;&gt; print(' aa bb cc dd  '.split())
['aa', 'bb', 'cc', 'dd']
&gt;&gt;&gt; print(' aa bb cc dd  '.split(maxsplit=1))
['aa', 'bb cc dd  ']
</code></pre>
