<div class="post-text" itemprop="text">
<p><code>requests-oauthlib</code> is installed at <code>/usr/local/lib/python2.7/site-packages</code>.</p>
<p>But the latter is not in your PYTHONPATH/sys.path</p>
<p>So add it.  This is one way.</p>
<pre><code>sys.path.append('/usr/local/lib/python2.7/site-packages/')
</code></pre>
<p><a href="https://docs.python.org/3/using/cmdline.html?highlight=pythonpath#envvar-PYTHONPATH" rel="nofollow">Learn about pythonpath</a></p>
</div>

<p>I had this issue and in addition to making sure the correct path was set, I found that running 
<code>$ pip install requests requests_oauthlib</code> 
solved the issue, as oauthlib is not included in the core <code>requests</code> library. </p>
<p><a href="https://github.com/requests/requests-oauthlib" rel="noreferrer">https://github.com/requests/requests-oauthlib</a></p>
