<div class="post-text" itemprop="text">
<p>The syntax has changed in that <a href="https://docs.python.org/3/whatsnew/3.0.html#print-is-a-function"><code>print</code> is now a function</a>.  This means that the <code>%</code> formatting needs to be done inside the parenthesis:<sup>1</sup></p>
<pre><code>print("%d. %s appears %d times." % (i, key, wordBank[key]))
</code></pre>
<p>However, since you are using Python 3.x., you should actually be using the newer <a href="https://docs.python.org/3/library/stdtypes.html#str.format"><code>str.format</code></a> method:</p>
<pre><code>print("{}. {} appears {} times.".format(i, key, wordBank[key]))
</code></pre>
<p>Though <code>%</code> formatting is not <em>officially</em> deprecated (yet), it is discouraged in favor of <code>str.format</code> and will most likely be removed from the language in a coming version (Python 4 maybe?).</p>
<hr/>
<p><sub><sup>1</sup>Just a minor note: <code>%d</code> is the format specifier for integers, not <code>%s</code>.</sub></p>
</div>
<div class="post-text" itemprop="text">
<p>Version 3.6+: Use a <a href="https://docs.python.org/3/reference/lexical_analysis.html#formatted-string-literals" rel="noreferrer">formatted string literal</a>, <em>f-string</em> for short</p>
<pre><code>print(f"{i}. {key} appears {wordBank[key]} times.")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try the format syntax:</p>
<pre><code>print ("{0}. {1} appears {2} times.".format(1, 'b', 3.1415))
</code></pre>
<p>Outputs:</p>
<pre><code>1. b appears 3.1415 times.
</code></pre>
<p>The print function is called just like any other function, with parenthesis around all its arguments.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can also format the string like so:</p>
<pre><code>&gt;&gt;&gt; print ("{index}. {word} appears {count} times".format(index=1, word='Hello', count=42))
</code></pre>
<p>Which outputs</p>
<pre><code>1. Hello appears 42 times.
</code></pre>
<p>Because the values are named, their order does not matter. Making the example below output the same as the above example.</p>
<pre><code>&gt;&gt;&gt; print ("{index}. {word} appears {count} times".format(count=42, index=1, word='Hello'))
</code></pre>
<p>Formatting string this way allows you to do this.</p>
<pre><code>&gt;&gt;&gt; data = {'count':42, 'index':1, 'word':'Hello'}
&gt;&gt;&gt; print ("{index}. {word} appears {count} times.".format(**data))
1. Hello appears 42 times.
</code></pre>
</div>
