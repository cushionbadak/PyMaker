<div class="post-text" itemprop="text">
<p>For convert to string with space you need to convert all the entries to <code>str</code> with <code>map(str,l)</code> and join them with <code>' '.join</code> , the for reverse to list first you need to split the string with <code>str.split()</code> and then convert to int with <code>map</code> :</p>
<pre><code>&gt;&gt;&gt; l=[1,2,3]
&gt;&gt;&gt; ' '.join(map(str,l))
'1 2 3'
&gt;&gt;&gt; string=' '.join(map(string,l))
&gt;&gt;&gt; map(int,string.split())
[1, 2, 3]
</code></pre>

<div class="post-text" itemprop="text">
<p>Your question doesn't really make sense. the <code>[</code>, <code>,</code>, and <code>]</code> are what DEFINE it as a list. Those markings are simply the way that Python shows it is a list. If you want to DISPLAY it as something other than a list, you probably want <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow"><code>str.join</code></a>. However that requires that each element in the list be a string, so you're left with either:</p>
<pre><code>&gt;&gt;&gt; some_list = [1,2,3]
&gt;&gt;&gt; print(some_list)
[1, 2, 3]
&gt;&gt;&gt; print(" ".join(map(str, some_list)))
1 2 3
</code></pre>
<p>or:</p>
<pre><code>&gt;&gt;&gt; print(" ".join([str(el) for el in some_list]))
1 2 3
</code></pre>
</div>
