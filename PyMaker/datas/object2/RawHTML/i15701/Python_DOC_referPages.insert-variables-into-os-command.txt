<div class="post-text" itemprop="text">
<p>Python does not support string interpolation directly.  Instead, you need to use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format</code></a> to insert values into strings:</p>
<pre><code>watermark = "gm composite /home/pi/photobooth/watermark.png {0} {0}".format(filename)
</code></pre>
