<p>I'm trying to print two strings on the same line using a timer in between. Here's the code:</p>
<pre><code>import time

print "hello",
time.sleep(2)
print "world"
</code></pre>
<p>But it seems the program waits for two seconds then prints both strings.</p>

<div class="post-text" itemprop="text">
<p>in python 2.7 you could use the print_function from the <strong>future</strong> package</p>
<pre><code>from __future__ import print_function
from time import sleep

print("hello, ", end="")
sleep(2)
print("world!")
</code></pre>
<p>But like you said, this waits 2 seconds then prints both strings. In light of Gui Rava's answer you can flush stdout, here is a sample which should get you in the right direction:</p>
<pre><code>import time
import sys

def _print(string):
    sys.stdout.write(string)
    sys.stdout.flush()

_print('hello ')
time.sleep(2)
_print('world')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The issue is that , by default, the console output is buffered.
Since Python 3.3 <code>print()</code> supports the keyword argument <code>flush</code> (<a href="http://docs.python.org/3/library/functions.html?highlight=print#print" rel="noreferrer">see documentation</a>):</p>
<pre><code>print('hello', flush=True)
</code></pre>
<p>If you use an prior versionf of python, you can force a flush like this:</p>
<pre><code>import sys
sys.stdout.flush()
</code></pre>
