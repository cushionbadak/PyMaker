<pre><code>for i in dict1:
    dict1[i] = [int(dict1[i])]

print dict1
</code></pre>

<div class="post-text" itemprop="text">
<p>Your current code is not working because your loop is just repeatedly reassigning the name <code>v</code> to <code>list(v)</code>.  It is not actually changing the values inside <code>dict1</code>.</p>
<p>The easiest way to do what you want is to use a <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow">dict comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; dict1 = {'usa' : '10', 'japan' : '20', 'canada' :'30'}
&gt;&gt;&gt; {k:[int(v)] for k, v in dict1.items()}
{'japan': [20], 'canada': [30], 'usa': [10]}
&gt;&gt;&gt;
</code></pre>
<p>Note that in Python 2.x, you should do <code>dict1.iteritems()</code> to avoid creating an unnecessary list.</p>
</div>
