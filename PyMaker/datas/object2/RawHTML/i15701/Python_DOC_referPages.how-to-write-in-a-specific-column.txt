<div class="post-text" itemprop="text">
<p>You should be doing:</p>
<pre><code>' '*(70-len(s))
</code></pre>
<p>or, with a little spacing:</p>
<pre><code>' ' * (70 - len(s))
</code></pre>
<p>This will create a string of spaces whose length is equal to <code>70</code> minus the length of the string <code>s</code>.  So, if <code>s</code> is <code>20</code> characters long, the code will make a string of <code>70-20</code> or <code>50</code> spaces.</p>
<p>That said, a better solution would be to use <a href="https://docs.python.org/3/library/stdtypes.html#str.rjust" rel="nofollow"><code>str.rjust</code></a>:</p>
<pre><code>s.rjust(70)
</code></pre>
<p>Below is a demonstration:</p>
<pre><code>&gt;&gt;&gt; s = 'abcde'
&gt;&gt;&gt; s.rjust(70)
'                                                                 abcde'
&gt;&gt;&gt; len(s.rjust(70))
70
&gt;&gt;&gt;
</code></pre>
<p>The <code>str.rjust</code>, <code>str.ljust</code>, and <code>str.center</code> methods (all explained in the link above) were designed specifically for tasks like this.</p>
</div>
