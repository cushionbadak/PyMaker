<div class="post-text" itemprop="text">
<p>You can sort <code>myDict.items</code> like so:</p>
<pre><code>&gt;&gt;&gt; myDict = {'ID1': 50, 'ID2': 40, 'ID3': 60}
&gt;&gt;&gt; for item in sorted(myDict.items(), key=lambda x: x[1]):
...     item
...
('ID2', 40)
('ID1', 50)
('ID3', 60)
&gt;&gt;&gt; for key, _ in sorted(myDict.items(), key=lambda x: x[1]):
...     key
...
'ID2'
'ID1'
'ID3'
&gt;&gt;&gt;
</code></pre>
<p>The important part here is the <a href="https://docs.python.org/3/glossary.html#term-key-function" rel="nofollow">key function</a> of <code>sorted</code>, which allows you to specify how you want each item sorted.  In this case, we sort on each key's value.</p>
<p>You may also want to read <a href="https://docs.python.org/3/howto/sorting.html" rel="nofollow">Sorting HOW TO</a> in the documentation to get a better understanding of how to use <code>sorted</code> and <code>key</code>.</p>
