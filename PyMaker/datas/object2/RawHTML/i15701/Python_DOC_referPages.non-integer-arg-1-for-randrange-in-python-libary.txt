<div class="post-text" itemprop="text">
<p>You are trying to pass <em>floating point</em> values to <code>random.randint()</code>. That function only takes <em>integers</em>.</p>
<p>You need to use the <a href="https://docs.python.org/3/library/random.html#random.uniform" rel="noreferrer"><code>random.uniform()</code> function</a> instead; it'll produce a uniformly random value between the lower and upper bound (inclusive):</p>
<blockquote>
<p>Return a random floating point number N such that a &lt;= N &lt;= b for a &lt;= b and b &lt;= N &lt;= a for b &lt; a.</p>
</blockquote>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; import random
&gt;&gt;&gt; random.uniform(0.1, 0.9)
0.6793304134926453
</code></pre>
</div>

<p><code>random.randint(0.1,0.9)</code></p>
<p>You have to pass integers to that function. 0.1 and 0.9 are not integers</p>
