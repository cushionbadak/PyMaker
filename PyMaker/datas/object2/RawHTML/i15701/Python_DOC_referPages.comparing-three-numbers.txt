<div class="post-text" itemprop="text">
<p>Python allows you to <a href="https://docs.python.org/3/reference/expressions.html" rel="noreferrer">chain comparison operators</a>:</p>
<pre><code>if a &lt; b &lt; c:
</code></pre>
<p>This will test if <code>b</code> is between <code>a</code> and <code>c</code> exclusive.  If you want inclusive, try:</p>
<pre><code>if a &lt;= b &lt;= c:
</code></pre>
<p>So, in your code, it would be something like:</p>
<pre><code>if a &lt; read &lt; b:
    return print("Yes")
elif b &lt; read &lt; a:
    return print("Yes")
else:
    return print("No")
</code></pre>
<p>or, more concisely:</p>
<pre><code>if (a &lt; read &lt; b) or (b &lt; read &lt; a):
    return print("Yes")
else:
    return print("No")
</code></pre>
<p>Note too that <code>print</code> always returns <code>None</code> in Python.  So, <code>return print("Yes")</code> is equivalent to <code>return None</code>.  Perhaps you should remove the return statements:</p>
<pre><code>if (a &lt; read &lt; b) or (b &lt; read &lt; a):
    print("Yes")
else:
    print("No")
</code></pre>
