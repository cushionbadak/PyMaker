<div class="post-text" itemprop="text">
<p>I don't know if I would generally store information this way, but this is how it's been presented to me.</p>
<p>Say I have a list of dictionaries that records the details of let's say UFO sightings, and looks like this:</p>
<pre><code>aList = [{'country': 'japan', 'city': 'tokyo', 'year': 1995}, {'country': 'japan', 'city': 'hiroshima', 'year': 2005}, {'country': 'norway', 'city': 'oslo', 'year': 2005} ... etc]
</code></pre>
<p>I know how to count the number of occurrences in a list, but with dictionaries involved, I'm not sure how I'd go about it.</p>
<p>If I wanted to know which country had the most sightings of UFOs for example, how could I do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow"><code>collections.Counter</code></a> and a <a href="https://docs.python.org/3/reference/expressions.html#generator-expressions" rel="nofollow">generator expression</a> to count how many times each country appears in the list.  After that, you can use the <code>most_common</code> method to get the one that appears the most.  The code will look like this:</p>
<pre><code>from collections import Counter
aList = [{'country': 'japan', 'city': 'tokyo', 'year': 1995}, {'country': 'japan', 'city': 'hiroshima', 'year': 2005}, {'country': 'norway', 'city': 'oslo', 'year': 2005}]

[(country, _)] = Counter(x['country'] for x in aList).most_common(1)

print(country)
# Output: japan
</code></pre>
<p>Below is a demonstration of what each part does:</p>
<pre><code>&gt;&gt;&gt; from collections import Counter
&gt;&gt;&gt; aList = [{'country': 'japan', 'city': 'tokyo', 'year': '1995'}, {'country': 'japan', 'city': 'hiroshima', 'year': '2005'}, {'country': 'norway', 'city': 'oslo', 'year': '2005'}]
&gt;&gt;&gt; # Get all of the country names
&gt;&gt;&gt; [x['country'] for x in aList]
['japan', 'japan', 'norway']
&gt;&gt;&gt; # Total the names
&gt;&gt;&gt; Counter(x['country'] for x in aList)
Counter({'japan': 2, 'norway': 1})
&gt;&gt;&gt; # Get the most common country
&gt;&gt;&gt; Counter(x['country'] for x in aList).most_common(1)
[('japan', 2)]
&gt;&gt;&gt; # Use iterable unpacking to extract the country name
&gt;&gt;&gt; [(country, _)] = Counter(x['country'] for x in aList).most_common(1)
&gt;&gt;&gt; print(country)
japan
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Here's a concise version:</p>
<pre><code>from collections import Counter
from operator import itemgetter

aList = [{'country': 'japan', 'city': 'tokyo', 'year': 1995}, {'country': 'japan', 'city': 'hiroshima', 'year': 2005}, {'country': 'norway', 'city': 'oslo', 'year': 2005}]

countries = Counter(map(itemgetter('country'), aList))
print countries.most_common()

cities = Counter(map(itemgetter('city'), aList))
print cities.most_common()
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[('japan', 2), ('norway', 1)]
[('oslo', 1), ('hiroshima', 1), ('tokyo', 1)]
</code></pre>
