<div class="post-text" itemprop="text">
<p>Better don't do this via subprocess. Believe me it's a bad idea.</p>
<p>For simple filesystem copy operations take a look on <a href="https://docs.python.org/3/library/shutil.html" rel="nofollow noreferrer">python shutil module</a>.</p>
<p>For creating new directories use <a href="https://docs.python.org/3/library/os.html#os.makedirs" rel="nofollow noreferrer">os.makedirs</a>.</p>
<p>For creating files of certain size see <a href="https://stackoverflow.com/questions/8816059/create-file-of-particular-size-in-python">this answers</a>.</p>
