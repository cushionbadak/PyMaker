<div class="post-text" itemprop="text">
<p>try:</p>
<pre><code>for i in xrange(1,6,2):
    print i
</code></pre>
<p>This print:</p>
<pre><code>1
3
5
</code></pre>
<blockquote>
<p>Use xrange instead of range if you are using python 2.x because it is more efficient as it generates an iterable object, and not the whole list.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow"><code>range</code></a> function:</p>
<pre><code>for i in range(1, 6, 2):
    print(i)
</code></pre>
</div>
