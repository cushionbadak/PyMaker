<p>I want to delete the keys in my dictionary with the value of <code>1</code>, but the only way I know of is to manually input keys to delete which is inefficient. Any ideas?</p>
<p>I have a feeling I should use:</p>
<pre><code>for key,value in mydict.items():
</code></pre>
<p>For example, if I had the dictionary</p>
<pre><code>mydict = {'a':1, 'b':2, 'c':3, 'd':1, 'e':1}
</code></pre>
<p>how would I delete the keys with a value of <code>1</code> so that I can print a dictionary purely of keys with values <code>&gt; 1</code>?</p>

<div class="post-text" itemprop="text">
<p>You can use a <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow">dict comprehension</a> to filter the dictionary:</p>
<pre><code>&gt;&gt;&gt; mydict = {'a':1, 'b':2, 'c':3, 'd':1, 'e':1}
&gt;&gt;&gt; {k:v for k,v in mydict.items() if v != 1}
{'c': 3, 'b': 2}
&gt;&gt;&gt;
</code></pre>
<p>Note that you should use <code>mydict.iteritems()</code> if you are on Python 2.x to avoid creating an unnecessary list.</p>
</div>
<div class="post-text" itemprop="text">
<p>You could go though each of the items (the key value pair) in the dictionary to check the conflict and add value in new Dict.(here.. result) that not contain duplicate.</p>
<pre><code>mydict = {'a':1, 'b':2, 'c':3, 'd':1, 'e':1}
result = {}

for key,value in mydict.items():
    if value not in result.values():
        result[key] = value

print result
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>To create a new dictionary with the items which has the value greater than 1 use <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow">dict comprehension</a> like below.</p>
<pre><code>&gt;&gt;&gt; mydict = {'a':1, 'b':2, 'c':3, 'd':1, 'e':1}
&gt;&gt;&gt; {key:value for key,value in mydict.items() if value &gt; 1}
{'c': 3, 'b': 2}
</code></pre>
