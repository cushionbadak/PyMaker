<p>The question is as it is in the 'Title', but for the sake of completeness -</p>
<p>Suppose :</p>
<pre><code>&gt;&gt;&gt; a= 23
&gt;&gt;&gt; h ="Liam Neeson"
&gt;&gt;&gt; k ={'a','b','c'}
</code></pre>
<p>Are three variables. Then I do a whole bunch of other stuff. And I want to see what variables I have initialised so far, is there a way to do it? Something that will return </p>
<blockquote>
<blockquote>
<blockquote>
<p>[a,h,k].</p>
</blockquote>
</blockquote>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>If you only want user defined variables, you could use <a href="https://docs.python.org/3/library/functions.html#dir" rel="nofollow"><code>dir</code></a> and strip out the builtins:</p>
<pre><code>set(dir()) - set(dir(__builtins__)) - {'__builtins__'}
</code></pre>
<hr/>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; a = 23
&gt;&gt;&gt; h = "Liam Neeson"
&gt;&gt;&gt; k = {'a', 'b', 'c'}
&gt;&gt;&gt; set(dir()) - set(dir(__builtins__)) - {'__builtins__'}
{'a', 'h', 'k'}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>yes, using the <a href="https://docs.python.org/3/library/functions.html#dir" rel="nofollow"><code>dir()</code></a> function:</p>
<blockquote>
<p>dir([object])
  Without arguments, return the list of names in the current local scope. With an argument, attempt to return a list of valid attributes for that object.</p>
</blockquote>
<p>example:</p>
<pre><code>&gt;&gt;&gt; dir()
['__builtins__', '__doc__', '__name__', '__package__']
&gt;&gt;&gt; a = 1
&gt;&gt;&gt; dir()
['__builtins__', '__doc__', '__name__', '__package__', 'a']
</code></pre>
</div>
