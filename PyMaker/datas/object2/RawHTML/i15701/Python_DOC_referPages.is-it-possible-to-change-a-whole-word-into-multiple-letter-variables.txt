<div class="post-text" itemprop="text">
<p>I'm not entirely sure what you're asking, but you can access the individual characters of a string using indexing:</p>
<pre><code>word = "because"
print(word[0])  # Prints "b"
print(word[1])  # Prints "e"
print(word[2])  # Prints "c"
print(word[3])  # Prints "a"
print(word[4])  # Prints "u"
print(word[5])  # Prints "s"
print(word[6])  # Prints "e"
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Dynamic variables are a bad practice and should be avoided.  It is very easy to lose track of them, accidentally overshadow them, etc.</p>
<p>Why not use a <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow">dictionary</a> instead?</p>
<pre><code>&gt;&gt;&gt; word = 'because'
&gt;&gt;&gt; dct = dict(enumerate(word, 1))
&gt;&gt;&gt; dct
{1: 'b', 2: 'e', 3: 'c', 4: 'a', 5: 'u', 6: 's', 7: 'e'}
&gt;&gt;&gt; dct[1]  # Would be the same as 'var_1'
'b'
&gt;&gt;&gt; dct[5]  # Would be the same as 'var_5'
'u'
&gt;&gt;&gt;
</code></pre>
<p>As you can see, it is about the same as dynamic variable names except that the data is stored cleanly in a dictionary object.</p>
