<p>Let's say for example one wants to iterate through 3 list containing the same number of items. How would I be able to compare the items of same index and put the median of those in a new list called median list?</p>
<pre><code>example output:

list_1 = [1,2,3]
list_2 = [1,3,5]
list_3 = [2,4,6]

median_list = [1, 3, 5]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/2/library/functions.html#zip" rel="nofollow"><code>zip</code></a> and <a href="https://docs.python.org/2/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>&gt;&gt;&gt; list_1 = [1,2,3]
&gt;&gt;&gt; list_2 = [1,3,5]
&gt;&gt;&gt; list_3 = [2,4,6]

&gt;&gt;&gt; zip(list_1, list_2, list_3)  # To make 1st items pair, 2nd items pairs, ...
[(1, 1, 2), (2, 3, 4), (3, 5, 6)]

&gt;&gt;&gt; [sorted(xs)[1] for xs in zip(list_1, list_2, list_3)]
[1, 3, 5]
</code></pre>
<p>If you are using Python 3.4+, you can use <a href="https://docs.python.org/3/library/statistics.html#statistics.median" rel="nofollow"><code>statistics.median</code></a>:</p>
<pre><code>&gt;&gt;&gt; [statistics.median(xs) for xs in zip(list_1, list_2, list_3)]
[1, 3, 5]
</code></pre>
