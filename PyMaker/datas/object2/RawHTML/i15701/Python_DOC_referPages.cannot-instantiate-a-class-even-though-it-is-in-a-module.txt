<div class="post-text" itemprop="text">
<p>The <a href="https://github.com/attwad/pycvss/blob/master/pycvss/__init__.py" rel="nofollow"><code>__init__</code> file</a> in that project is <em>empty</em>. You need to import the <a href="https://github.com/attwad/pycvss/blob/master/pycvss/pycvss.py" rel="nofollow">nested <code>pycvss</code> module</a>:</p>
<pre><code>from pycvss import pycvss

c = pycvss.Cvss()
</code></pre>
<p>The documentation is rather unclear about this; I'd file a <a href="https://github.com/attwad/pycvss/issues?q=is%3Aopen+is%3Aissue" rel="nofollow">documentation issue</a> with the project.</p>
<p>Personally, I'd add one line to the <code>__init__.py</code> file here:</p>
<pre><code>from pycvss import Cvss
</code></pre>
<p>and update the documentation to illustrate how to import that class.</p>
<p>Note that the project expects the <a href="https://docs.python.org/3/library/enum.html" rel="nofollow"><code>enum</code> library</a> to be available. If you are not using Python 3.4 or newer, you'll have to install the <a href="https://pypi.python.org/pypi/enum34" rel="nofollow"><code>enum34</code> backport package</a> too.</p>
