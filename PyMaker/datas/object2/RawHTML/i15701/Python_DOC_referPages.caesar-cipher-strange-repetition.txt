<p>As @Kevin, mentions, you are overwriting <code>phrase</code> so some characters will get encoded multiple times.</p>
<p>The solution is to build up the answer in a new variable:</p>
<pre><code>answer = ""
for each in range(len(phrase)):
    letternum = alphabet.index(phrase[each])
    if letternum &gt; 21:
        letternum = letternum+5-26    
    else:
        letternum = letternum + 5
    answer += alphabet[letternum]

print "Your encrypted phrase is:", answer
</code></pre>
<p>This could be written more succinctly using the <code>%</code> operator:</p>
<pre><code>for each in range (len(phrase)):
  letternum = alphabet.index(phrase[each])
  answer += alphabet[ (letternum + 5) % 26 ]
</code></pre>

<p>Let me show you a simpler way of writing your Caesar cipher program:</p>
<pre><code>import string

def main():
    key = 5
    table = str.maketrans(string.ascii_letters,
                          string.ascii_lowercase[key:] +
                          string.ascii_lowercase[:key] +
                          string.ascii_uppercase[key:] +
                          string.ascii_uppercase[:key])
    plaintext = input('Please enter a phrase: ')
    ciphertext = plaintext.translate(table)
    print('Your encrypted phrase is:', ciphertext)

if __name__ == '__main__':
    main()
</code></pre>

<span class="comment-copy">I think you have a logic problem when you do <code>replace</code>. For example, If you're scanning "AF", you do "replace A with F" and get "FF". Then you do "replace F with K" and get "KK".</span>
<span class="comment-copy">The easiest way to do this is use a <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer">dictionary</a> <code>{start_char: end_char}</code>. Also, note that <code>phrase = str.upper(phrase)</code> can be <code>phrase = phrase.upper()</code> (in general, you can always write <code>Class.method(instance)</code> as <code>instance.method()</code>).</span>
