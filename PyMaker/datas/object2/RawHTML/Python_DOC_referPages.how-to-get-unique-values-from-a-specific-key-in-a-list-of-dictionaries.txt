<p>Give I have a list with dictionaries and I want to get all unique values from the key first_name from the dictionaries, how do I do that in python?</p>
<pre><code>data = [
{
    "id": 1,
    "first_name": "John"
},
{   "id": 2,
    "first_name": "Mary"
},
{   "id": 3,
    "first_name": "John"
}
]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use a <a href="https://python-reference.readthedocs.io/en/latest/docs/comprehensions/set_comprehension.html" rel="nofollow noreferrer"><strong>set comprehension</strong></a> :</p>
<pre><code>first_names = {d["first_name"] for d in data}
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>&gt;&gt;&gt; set(i["first_name"] for i in data)
{'John', 'Mary'}
</code></pre>
<p>if you want a <code>list</code> instead of <code>set</code> you can convert it to list:</p>
<pre><code>&gt;&gt;&gt; list(set(i["first_name"] for i in data))
['John', 'Mary']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can map the operator <code>itemgetter()</code> to dictionaries in the list:</p>
<pre><code>from operator import itemgetter

iget = itemgetter('first_name')

set(map(iget, data))
# {'Mary', 'John'}
</code></pre>
