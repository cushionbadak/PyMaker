<div class="post-text" itemprop="text">
<p>Try <code>collections.Counter</code>.</p>
<pre><code>import collections

x = [1, 2, 1, 1]
counter = collections.Counter(x)
if 3 in counter.values():
    print('3 are the same')
</code></pre>
<p>Output:</p>
<pre><code>3 are the same
</code></pre>
<p><strong>UPDATE</strong></p>
<p>If you are interested in checking for 3 or more occurrences, you can check the maximum value in the <code>Counter</code> like this:</p>
<pre><code>if max(counter.values()) &gt;= 3:
    print('3 or more are the same')
</code></pre>
<p>This method has the added advantage that it works for larger lists as well without modification.</p>

<p>I would suggest using <code>collections.Counter</code>.</p>
<p>Convert the list to a counter. The counter should have two keys, and one of its values should be 3:</p>
<pre><code>In [1]: from collections import Counter

In [2]: c = Counter([0, 1, 1, 1])

In [3]: len(c) == 2
Out[3]: True

In [4]: 3 in c.values()
Out[4]: True
</code></pre>
<p>In short:</p>
<pre><code>In [5]: len(c) == 2 and 3 in c.values()
Out[5]: True
</code></pre>
<p>Let's try a example that doesn't meet the criteria:</p>
<pre><code>In [8]: d = Counter([0, 0, 1, 1])

In [9]: len(d) == 2 and 3 in d.values()
Out[9]: False
</code></pre>

<p>This solution uses a <a href="https://docs.python.org/3/library/collections.html#collections.Counter" rel="nofollow noreferrer"><code>collections.Counter</code></a></p>
<pre><code>from collections import Counter

mylist1 = [1, 2, 4, 4]
mylist2 = [1, 3, 3, 3]

c1 = Counter(mylist1)
c2 = Counter(mylist2)

c1.most_common(1)
&gt;&gt;&gt; [(4, 2)]

c1.most_common(1)[0][1] == 3
&gt;&gt;&gt; False

c2.most_common(1)[0][1] == 3
&gt;&gt;&gt; True
</code></pre>
