<div class="post-text" itemprop="text">
<p>Python <a href="https://docs.python.org/3/tutorial/datastructures.html#dictionaries" rel="nofollow noreferrer"><code>dict</code></a>s use immutable values (like strings) as keys.</p>
<p>Try something like this instead:</p>
<pre><code>import pyrebase

config = {
    'apiKey': '...',
    'authDomain': '...',
    'databaseURL': '...',
    'projectId': '...',
    'storageBucket': '...',
    'messagingSenderId': '...',
}

firebase = firebase.initializeApp(config)
</code></pre>
<p>Note also that semicolons usually aren't used with Python, though they are legal.</p>
</div>
<span class="comment-copy">If those are your real credentials you should invalidate them <b><i>immediately</i></b>. They are forever compromised, and you need to generate new ones.</span>
