<p><strong>Creating a virtual environment in Windows:</strong></p>
<p>Create a folder and open a command prompt inside this.</p>
<p>Then, create a virtual environment by using: <code>virtualenv venv</code></p>
<p>To use this virtual environment, you need to activate it first. Activate it using: <code>venv\Source\activate.bat</code></p>
<p>Now you will see a <code>(venv)</code> prefix in the prompt. That indicates the virtual environment is ready to use. Finally, install packages in this virtual environment as required like this: <code>pip install bottle</code></p>
<p>List the installed packages in a <code>.txt</code> file for later: <code>pip freeze&gt;requirements.txt</code></p>
