<p>I have two dictionaries dic_1 and dic_2.</p>
<pre><code>dic_1={"HI":"HELLO" , "NAME":"XYZ"}
dic_2={"HI" : "HELLO" , "NAME" : "XYZ" , "AGE" : "YY"}
</code></pre>
<p>I want to compare the key ,value pairs of these two dictionaries.If the values are same then output should print "YES" else "NO". It should check the values of the common keys in both dictionaries.</p>

<div class="post-text" itemprop="text">
<p>If you only want a single output for the entire test, you could make use of the <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer"><code>all()</code> built-in function</a>:</p>
<pre><code>dic_1={"HI":"HELLO" , "NAME":"XZ"}
dic_2={"HI" : "HELLO" , "NAME" : "XYZ" , "AGE" : "YY"}
if all(dic_1[k] == dic_2[k] for k in dic_1 if k in dic_2):
    print("YES")
else:
    print("NO")
</code></pre>
