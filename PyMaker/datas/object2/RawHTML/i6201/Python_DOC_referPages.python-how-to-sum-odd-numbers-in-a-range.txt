<p>I must use While for this calc.</p>
<pre><code>num = int(input("Insert number: "))

<div class="post-text" itemprop="text">
<p>You can use the built-in <a href="https://docs.python.org/3/library/functions.html#sum" rel="nofollow noreferrer">sum()</a> function like this:</p>
<pre><code>num = int(input("Insert number: "))
s = sum(range(1, num+1, 2))
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer"><code>range()</code></a> takes start (inclusive), end (exclusive), and step (In our case: <code>start=1</code>, <code>end=num+1</code> and <code>step=2</code>)</p>
<p><strong>Output:</strong></p>
<pre><code>&gt;&gt;&gt; num = 9
&gt;&gt;&gt; s = sum(range(1, num+1, 2))
&gt;&gt;&gt; s
25
</code></pre>
<p>If using <code>while</code> is a requirement, then you can achieve the same result with:</p>
<pre><code>&gt;&gt;&gt; s = 0
&gt;&gt;&gt; i = 1
&gt;&gt;&gt; while i &lt; num + 1:
...     if i % 2:  # Or: i % 2 != 0, which means we only consider odd nums
...         s += i
...     i += 1
...
&gt;&gt;&gt; s
25
</code></pre>
