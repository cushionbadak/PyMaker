<p>I am having a hard time to find a solution to merge two different type of element within the list.</p>
<pre><code>list_i = ['teacher', '10', 'student', '100', 'principle', '2']
</code></pre>
<p>Result:</p>
<pre><code>list_1 = ['teacher:10', 'student:100', 'principle:2']
</code></pre>
<p>Any help is greatly appreciated!!</p>

<div class="post-text" itemprop="text">
<p>This will work:</p>
<pre><code>[list_i[i] + ":" + list_i[i+1] for i in range(0, len(list_i), 2)]
</code></pre>
<p>This produces:</p>
<pre><code>['teacher:10', 'student:100', 'principle:2']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use following code    </p>
<pre><code>[':'.join(item) for item in zip(list_i[::2],list_i[1::2])]
</code></pre>
<p>This will just slice the list in 2 parts and joins them with zip</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://github.com/erikrose/more-itertools" rel="nofollow noreferrer"><code>more_itertools</code></a>, a third-party library, you can apply a <a href="https://stackoverflow.com/questions/8269916/what-is-sliding-window-algorithm-examples">sliding window</a> technique: </p>
<pre><code>&gt; pip install more_itertools
</code></pre>
<p><strong>Code</strong></p>
<pre><code>import more_itertools as mit


iterable = ['teacher', '10', 'student', '100', 'principle', '2']

[":".join(i) for i in mit.windowed(iterable, 2, step=2)]
# ['teacher:10', 'student:100', 'principle:2']
</code></pre>
<p>Alternatively apply the <code>grouper</code> <a href="https://docs.python.org/3/library/itertools.html#itertools-recipes" rel="nofollow noreferrer">itertools recipe</a>, which is also implemented in <a href="https://more-itertools.readthedocs.io/en/latest/api.html" rel="nofollow noreferrer"><code>more_itertools</code></a>.</p>
<pre><code>[":".join(i) for i in mit.grouper(2, iterable)]
# ['teacher:10', 'student:100', 'principle:2']
</code></pre>
