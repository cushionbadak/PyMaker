<div class="post-text" itemprop="text">
<p>Try using <code>getattr</code>:</p>
<pre><code>class character(object):
        def __init__(self, name, name1, name2):
            self.name = name
            self.name1 = name1
            self.name2= name2

bob = character('bob', 'bob1', 'bob2')


def print_name(var1, var2):
    print(getattr(var1,var2))

print_name(bob, 'name1')
</code></pre>
<p>This is from the <a href="https://docs.python.org/3/library/functions.html#getattr" rel="nofollow noreferrer">documentation</a>:</p>
<blockquote>
<p>Return the value of the named attribute of object. name must be a
  string. If the string is the name of one of the objectâ€™s attributes,
  the result is the value of that attribute. For example, getattr(x,
  'foobar') is equivalent to x.foobar. If the named attribute does not
  exist, default is returned if provided, otherwise AttributeError is
  raised.</p>
</blockquote>
