<div class="post-text" itemprop="text">
<p>I have a following directory structure: </p>
<pre><code>source
       source_1.py
       __init__.py
</code></pre>
<p>source1.py has  class Source defined</p>
<pre><code>source1.py

class Source(object):
    pass
</code></pre>
<p>I am able to import using this</p>
<pre><code>&gt;&gt;&gt; from source.source1 import Source
&gt;&gt;&gt; Source
&lt;class 'source.source1.Source'&gt;
</code></pre>
<p>However when trying to import using the below method it fails.</p>
<pre><code>&gt;&gt;&gt; from source import *
&gt;&gt;&gt; source1.Source
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
NameError: name 'source1' is not defined
</code></pre>
<p>Please let me know how can we use the 2nd import ?</p>
</div>
<div class="post-text" itemprop="text">
<p>For importing from a package (unlike importing from a module) you need to specify what <code>*</code> means. To do that, in <code>__init__.py</code> add a line like this:</p>
<pre><code>__all__ = ["source1"]
</code></pre>
<p>See the Python documentation for <a href="https://docs.python.org/3/tutorial/modules.html#importing-from-a-package" rel="nofollow noreferrer">Importing * From a Package</a>.</p>
</div>
