<p>One solution could be:</p>
<pre><code>s = "abcd"

def slicer(string):

    return string[int(input("Lower bound (inclusive): ")): \
                  int(input("Upper bound (inclusive): ")) + 1]

print("Output: {}".format(slicer(s)))
print("Output: {}".format(slicer(s)))
</code></pre>
<p>You could also use the <code>slice()</code> function. </p>
<pre><code>s = "abcd"

def slicer(string):

    return string[slice(int(input("Lower bound (inclusive): ")), \
                        int(input("Upper bound (inclusive): ")) + 1)]

print("Output: {}".format(slicer(s)))
print("Output: {}".format(slicer(s)))
</code></pre>
<p>Both cases will return a <code>list</code> sliced from the user entered lower bound up to and including the user entered upper bound.</p>
<p>Output sample:</p>
<pre><code>Lower bound (inclusive): 0
Upper bound (inclusive): 2
Output: abc
Lower bound (inclusive): 1
Upper bound (inclusive): 3
Output: bcd
</code></pre>
