<div class="post-text" itemprop="text">
<p>Take a look at <a href="https://docs.python.org/3/library/os.html#os.walk" rel="nofollow noreferrer">os.walk</a></p>
<p>This produces output like in your example. Current directory <code>.</code> is used as an example:</p>
<pre><code>for path, dirs, files in os.walk('.'):
    for file in files:
        print(os.path.join(path,file))
</code></pre>
<p>If you want absolute paths (starting with / on Linux or \ on Windows), start with an absolute dirname.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can get the list for all your files within your <code>MainDirectory</code> with below code:</p>
<pre><code>import sys,os

MainDirectory = "/tmp/"

filesList = list()
for path, subdirs, files in os.walk(MainDirectory):
    for name in files:
        filesList.append(os.path.join(path, name))
</code></pre>
<p>Then the <code>filesList</code> list contains all of your files within that directory and its subdirectories.</p>
</div>
