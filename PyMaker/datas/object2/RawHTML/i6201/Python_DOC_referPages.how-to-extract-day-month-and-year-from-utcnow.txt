<div class="post-text" itemprop="text">
<p>Use <code>datetime.utcnow().date()</code></p>
<pre><code>datetime.utcnow().date()
datetime.date(2017, 11, 7)
</code></pre>
<p><br/></p>
<p>Adding to answer</p>
<p>The datetime object always contains year, month, day as well as hours, minutes, seconds, and microseconds. It is a combination of  what the date and time objects contain, see <a href="https://docs.python.org/3/library/datetime.html#datetime-objects" rel="nofollow noreferrer" title="datetime Objects">datetime Objects</a></p>
<pre><code>from datetime import datetime

# this is your datetime object
time_created = datetime.utcnow()

# when you want to see it formatted as Y,M,D call the date method
date_created = time_created.date()

time_created
date_created
</code></pre>
<p>Output:</p>
<pre><code>datetime.datetime(2017, 11, 7, 23, 43, 43, 761750)
datetime.date(2017, 11, 7)`
</code></pre>
