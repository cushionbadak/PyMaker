<div class="post-text" itemprop="text">
<p><code>str.split()</code> doesn't take a regular expression, you need to use <a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow noreferrer"><code>re.split()</code></a>.</p>
<pre><code>address = re.split(r'Manufacturer\/Distributor name:?', arg, 1)[1]
</code></pre>
<p>You should also get in the habit of using raw strings for regular expressions, otherwise you need to escape the <code>\</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>I'm assuming <code>arg</code> is a string. <code>string.split()</code> does not accept regex as delimiter. You can read about it <a href="https://docs.python.org/3.6/library/stdtypes.html#str.split" rel="nofollow noreferrer">here</a>.</p>
<p>Instead, you should use <code>arg.split("Manufacturer/Distributor name", 1)[1]</code>.</p>
</div>
