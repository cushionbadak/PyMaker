<div class="post-text" itemprop="text">
<p>Try using <a href="https://docs.python.org/2/library/webbrowser.html#webbrowser.get" rel="nofollow noreferrer"><code>get()</code></a>:</p>
<pre><code>webbrowser.get('chrome').open('https://www.youtube.com')
</code></pre>
<p><strong>EDIT</strong></p>
<p>Make sure to use the full path to the site</p>
</div>
<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>&gt;&gt;&gt; import webbrowser
&gt;&gt;&gt; browser_path = "C:/Program Files (x86)/Google/Chrome/Application/chrome.exe %s"
&gt;&gt;&gt; url = "https://www.youtube.com"
&gt;&gt;&gt; webbrowser.get(browser_path).open(url)
True
</code></pre>
<p>Note the Unix-style path. <a href="https://stackoverflow.com/a/24873636/8709791">This is because webbrowser internally does a shlex.split on the path, which will just erase Windows-style path separators</a></p>
<p><a href="https://docs.python.org/3/library/webbrowser.html#webbrowser.register" rel="nofollow noreferrer">Registering</a> the browser also works on Windows:</p>
<pre><code>&gt;&gt;&gt; import webbrowser as wb
&gt;&gt;&gt; wb.register('chrome', None)
&gt;&gt;&gt; wb.open('https://www.youtube.com')
True
&gt;&gt;&gt; wb.open('https://www.google.com')
True
&gt;&gt;&gt; wb.open('https://stackoverflow.com')
True
</code></pre>
</div>
