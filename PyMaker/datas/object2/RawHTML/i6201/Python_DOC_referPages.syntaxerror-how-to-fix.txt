<div class="post-text" itemprop="text">
<p><code>find_contact(tel)</code> is an instance method hence it should be declared as:</p>
<pre><code>def find_contact(self, tel):
    ...
</code></pre>
<p>when you call it with:</p>
<pre><code>C1.find_contact('1111')
</code></pre>
<p>since it's being called on an instance of the class, it's translated to:</p>
<pre><code>find_contact(c1, '1111')
</code></pre>
<p>and that's why you see the error that says this method should be called with only one argument but two were given.</p>
