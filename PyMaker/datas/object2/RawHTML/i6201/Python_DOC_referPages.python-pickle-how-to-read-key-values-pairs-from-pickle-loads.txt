<div class="post-text" itemprop="text">
<p>I have a <strong>.pkl</strong> file that I'm writing with pickle, with key-value pairs and that works.</p>
<p>However I was wondering how can I access the key-value pairs once I read from this file using <a href="https://docs.python.org/3/library/pickle.html#pickle.loads" rel="nofollow noreferrer">pickle.load</a>.</p>
<pre><code>content = pickle.load(open(COOKIE_FILENAME, "rb"))
</code></pre>
<p><code>content</code> is a dictionary (<code>&lt;type 'dict'&gt;</code>). </p>
<p>If I do:</p>
<pre><code>for key in content:
    # this works, but I don't have the values
</code></pre>
<p>What I want to do is instead something like:</p>
<pre><code>for key, value in content:
    # do something with keys and values
</code></pre>
<p>but if I do that, I get:</p>
<blockquote>
<p>E       ValueError: too many values to unpack</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>The solution is:</p>
<pre><code>for key, value in content.items():
    # do something with keys and values
</code></pre>
<p>thanks @Jose, @roganjosh and @timgeb for your help via comments.</p>
</div>
