<p>Yo can instead use the built-in function <code>exec</code> after you compose the module string</p>
<pre><code>    &gt;&gt;&gt; modOs = 'os'
    &gt;&gt;&gt; exec('import ' + modOs)
    &gt;&gt;&gt; os.getcwd()
    'C:\\Desktop'
</code></pre>
<p>Note: Python 3.6 <a href="https://docs.python.org/3/library/functions.html#exec" rel="nofollow noreferrer">exec</a></p>

<div class="post-text" itemprop="text">
<p>I think you have to understand the difference between module name in Python and file name. importlib.import_module() gets a "module name", not a "file name", so "fe_fi_fo_fam.py" is not a valid argument, instead you have to use "fe_fi_fo_fam" with no .py extension.</p>
