<div class="post-text" itemprop="text">
<p>I'm using git bash to run a python script that calls some Linux tools (scp, ssh) on Windows. </p>
<p>My python script takes a path as a parameter. However, when calling os.path.join to do some operation on that path, the returned value is a windows-formatted path (back slashes). Passing that path to the scp or ssh tools fails or does not behave properly.</p>
<p>Is there any easy way to 'force' Linux path style in python 3 ?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/pathlib.html" rel="nofollow noreferrer">pathlib</a>:</p>
<pre><code>import os
import sys

&gt;&gt;&gt; from pathlib import Path
&gt;&gt;&gt; import os
&gt;&gt;&gt; import sys
&gt;&gt;&gt; p = Path(os.path.dirname(sys.executable))
&gt;&gt;&gt; p
WindowsPath('C:/Python36')
&gt;&gt;&gt; x = p.joinpath('Scripts/pip3.6.exe')
&gt;&gt;&gt; y = p.joinpath('Scripts\\pip3.6.exe')
&gt;&gt;&gt; print(x.as_posix())
C:/Python36/Scripts/pip3.6.exe
&gt;&gt;&gt; print(y.as_posix())
C:/Python36/Scripts/pip3.6.exe
</code></pre>
</div>
