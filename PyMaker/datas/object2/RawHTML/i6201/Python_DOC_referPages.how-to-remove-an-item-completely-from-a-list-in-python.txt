<p>I have a list,</p>
<pre><code>numbers=["one","two","three","one","one"]
</code></pre>
<p>when I am doing,</p>
<pre><code>numbers.remove("one")
</code></pre>
<p>it returns</p>
<pre><code>["two","three","one","one"]
</code></pre>
<p>but I want to remove it completely,</p>
<p>My expected output is,</p>
<pre><code>output=["two","three"]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try this:</p>
<pre><code>numbers = ["one", "two", "three", "one", "one"]
[element for element in numbers if element != "one"]
=&gt; ["two", "three"]
</code></pre>
<p>We used a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> for doing the job, check the link for details. But basically - we created a new list, filtering out the undesired element via a condition.</p>
</div>
<div class="post-text" itemprop="text">
<p>How about a <code>list comprehension</code></p>
<pre><code>[i for i in numbers if i!="one"]
</code></pre>
<p>You can also use a <code>filter</code> (<strong>Python 2.x</strong>)</p>
<pre><code>filter(lambda x: x!="one",numbers)
</code></pre>
<p>Output:</p>
<pre><code>['two', 'three']
</code></pre>
