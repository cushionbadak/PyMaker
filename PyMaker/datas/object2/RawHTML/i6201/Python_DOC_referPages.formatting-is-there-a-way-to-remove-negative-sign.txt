<p>What's happened is that <code>sin(radians(360))</code> is returning a very small, negative float value (<code>-2.4492935982947064e-16</code>). To format a small negative float as positive 0, you can first round it to exactly negative 0, then add 0 so it becomes positive 0:</p>
<pre><code>from math import sin, radians

for x in range(0, 361, 10):
    print("{:3d}  {:6.3f}".format(x, round(sin(radians(x)), 3) + 0))
</code></pre>
