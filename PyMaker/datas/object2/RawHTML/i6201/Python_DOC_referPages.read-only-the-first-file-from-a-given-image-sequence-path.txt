<div class="post-text" itemprop="text">
<p>Try using <a href="https://docs.python.org/3/library/glob.html" rel="nofollow noreferrer"><code>glob</code></a>. Something like:</p>
<pre><code>import glob
import os
path = '/host_server/master/images/set01a/env_basecolor_default_v001'
pattern = 'basecolor_default.*.jpg'
filenames = glob.glob(os.path.join(path, pattern))
# read filenames[0]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The simplest solution seems to be to use several functions.</p>
<p>1) To get ALL of the full filepaths, use</p>
<pre><code> main_path = "/host_server/master/images/set01a/env_basecolor_default_v001/"
 all_files = [os.path.join(main_path, filename) for filename in os.listdir(main_path)]
</code></pre>
<p>2) To choose only those of a certain kind, use a filter.</p>
<pre><code>beta_files = list(filter(lambda x: "beta" in x, all_files))
beta_files.sort()
</code></pre>
</div>

<blockquote>
<p>read the <strong>first</strong> file based on the above file path given?</p>
</blockquote>
<p>With effective <a href="https://docs.python.org/3/library/glob.html#glob.iglob" rel="nofollow noreferrer"><strong><code>glob.iglob(pathname, recursive=False)</code></strong></a> (if you need the name/path of the 1st found file):</p>
<pre><code>import glob

path = '/host_server/master/images/set01a/env_basecolor_default_v001/basecolor_default.*.jpg'
it = glob.iglob(path)
first = next(it)
</code></pre>
<hr/>
<blockquote>
<p><code>glob.iglob()</code> - Return an iterator which yields the same values as
  <code>glob()</code> without actually storing them all simultaneously.</p>
</blockquote>
