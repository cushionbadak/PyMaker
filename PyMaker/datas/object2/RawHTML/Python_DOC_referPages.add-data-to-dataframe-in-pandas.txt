<p>I'm trying to add data of a specific datatype to a new dataframe, however, the code returns an empty dataframe. I tried doing it with just one entry to see if that's the problem</p>
<pre><code>date = pd.DataFrame(columns=['Date'])
date.append(pd.Series(report2.loc[1,'Serv']), ignore_index=True)

print(date)
</code></pre>
<p>This returns an empty dataframe as well.</p>

<div class="post-text" itemprop="text">
<p>You have to reassign <code>date</code>: </p>
<pre><code>date = pd.DataFrame(columns=['Date'])
date = date.append(pd.Series(report2.loc[1,'Serv']), ignore_index=True)

print(date)
</code></pre>
<p>Indeed, according to the documentation, <a href="https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.DataFrame.append.html" rel="nofollow noreferrer">append()</a> returns the appended DataFrame</p>
<blockquote>
<p>Returns: appended : DataFrame</p>
</blockquote>
