<div class="post-text" itemprop="text">
<p>Use <code>sys.platform()</code> to get the platform.
Or for more detailed information use <a href="http://docs.python.org/library/platform.html#platform.platform" rel="nofollow noreferrer">platform.platform()</a></p>
<pre><code>&gt;&gt;&gt; import platform
&gt;&gt;&gt; print platform.linux_distribution()
('Ubuntu', '12.04', 'precise')
</code></pre>
<p><strong>Note</strong>: <a href="https://docs.python.org/3/library/platform.html#unix-platforms" rel="nofollow noreferrer"><code>platform.linux_distribution()</code></a> has been deprecated since Python 3.5, and will be removed in Python 3.7.</p>
