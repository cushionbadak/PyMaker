<div class="post-text" itemprop="text">
<pre><code>if myval == 0:
   nyval=1
if myval == 1:
   nyval=0
</code></pre>
<p>Is there a better way to do a toggle in python, like a  nyvalue = not myval ?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="http://docs.python.org/2/reference/expressions.html#boolean-operations"><code>not</code> boolean operator</a>:</p>
<pre><code>nyval = not myval
</code></pre>
<p><code>not</code> returns a <em>boolean</em> value (<code>True</code> or <code>False</code>):</p>
<pre><code>&gt;&gt;&gt; not 1
False
&gt;&gt;&gt; not 0
True
</code></pre>
<p>If you must have an integer, cast it back:</p>
<pre><code>nyval = int(not myval)
</code></pre>
<p>However, the python <code>bool</code> type is a subclass of <code>int</code>, so this may not be needed:</p>
<pre><code>&gt;&gt;&gt; int(not 0)
1
&gt;&gt;&gt; int(not 1)
0
&gt;&gt;&gt; not 0 == 1
True
&gt;&gt;&gt; not 1 == 0
True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>In python, <a href="http://docs.python.org/3/reference/expressions.html#boolean-operations" rel="nofollow"><code>not</code></a> is a boolean operator which gets the opposite of a value:</p>
<pre><code>&gt;&gt;&gt; myval = 0
&gt;&gt;&gt; nyvalue = not myval
&gt;&gt;&gt; nyvalue
True
&gt;&gt;&gt; myval = 1
&gt;&gt;&gt; nyvalue = not myval
&gt;&gt;&gt; nyvalue
False
</code></pre>
<p>And <code>True == 1</code> and <code>False == 0</code> (if you need to convert it to an integer, you can use <code>int()</code>)</p>
</div>
