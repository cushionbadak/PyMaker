<p>I am processing a file line by line</p>
<pre><code>for line in mix_files[i]:
    if my_string in line:
       use the next line # How to get the next line
</code></pre>
<p>After a line containing the text is found, I need to work with the next line. Is there any easy way how to get the next line (e.g., increment an iterator), instead of the less confortable method: </p>
<pre><code>get line number -&gt; increment line number -&gt; use the next line
</code></pre>
<p>Thanks for your help.</p>

<div class="post-text" itemprop="text">
<p>Yes, use the <a href="http://docs.python.org/3/library/functions.html#next" rel="nofollow"><code>next()</code> function</a> on the iterator:</p>
<pre><code>next_line = next(mix_files[i])
</code></pre>
<p>Note that this can raise <code>StopIteration</code> if the file has no more lines. You can tell <code>next()</code> to return a default value instead:</p>
<pre><code>next_line = next(mix_files[i], None)
</code></pre>
<p>which will be returned instead of raising the exception.</p>
</div>
<div class="post-text" itemprop="text">
<p>Martijn's response is good, but you could also use enumerate:</p>
<pre><code>for j, line in enumerate(mix_files[i]):
    if my_string in line:
        try:
            next_line = mix_files[i][j+1]
        except IndexError:
            handle_end_of_file()
        else:
            do_whatever_with_next_line()
</code></pre>
</div>
<span class="comment-copy">Only if <code>mix_files[i]</code> is a sequence; if it is a file object this won't work.</span>
