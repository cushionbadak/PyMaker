<div class="post-text" itemprop="text">
<p>You need to use something like:</p>
<pre><code>print('[{0}]'.format(','.join(map(str, l))))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can modify the result if it isn't too big:</p>
<pre><code>print(repr(l).replace(' ', ''))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could convert it to a string and then replace the spaces. E.g:</p>
<pre><code>print ("{}".format(l)).replace(' ', '')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Join the list elements.</p>
<pre><code>print("[" + ",".join([str(i) for i in l]) + "]")
</code></pre>
