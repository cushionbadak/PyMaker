<p>Given a String is there any short way in Python 3 to find the different combinations of the space seperated words in that string ?</p>
<p>For eg:</p>
<p>If the input string is 'Peaches Apples Bananas', I want output as:</p>
<p>'Peaches Apples Bananas'</p>
<p>'Peaches Bananas Apples'</p>
<p>'Apples Bananas Peaches'</p>
<p>'Apples Peaches Bananas'</p>
<p>'Bananas Peaches Apples'</p>
<p>'Bananas Apples Peaches'</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>import itertools

string = 'Peaches Apples Bananas'
word_list = string.split(' ')

output = [' '.join(permutation) for permutation in itertools.permutations(word_list)]
</code></pre>

<div class="post-text" itemprop="text">
<p>I think you are looking for <a href="http://docs.python.org/3/library/itertools.html#itertools.permutations" rel="nofollow"><code>itertools.permutations</code></a>:</p>
<pre><code>import itertools

for perm in itertools.permutations('Peaches Apples Bananas'.split(' ')):
    print(' '.join(perm))
</code></pre>
