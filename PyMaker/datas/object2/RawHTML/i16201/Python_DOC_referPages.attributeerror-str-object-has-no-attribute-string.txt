<div class="post-text" itemprop="text">
<p>If the user is inputting numbers in that way then you probably want to use raw_input.  After that you can use .split() to get a list of the numbers and then use .sort().</p>
<p>Example:</p>
<pre><code>myRawInput = raw_input("Enter some numbers separated by spaces")
myList = myRawInput.split()
myList.sort()
</code></pre>
