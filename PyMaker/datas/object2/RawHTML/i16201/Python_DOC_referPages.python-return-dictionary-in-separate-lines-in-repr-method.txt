<div class="post-text" itemprop="text">
<p>First, if the only reason you think you can't use <code>pprint</code> is because you haven't looked at <a href="https://docs.python.org/3/library/pprint.html" rel="nofollow">the docs</a> and assume <code>pprint.pprint</code> is the only thing in the moduleâ€”well, it's not, and even if it were, you could always create a <code>StringIO</code> and pass it in as the <code>stream</code> argument.</p>
<p>But if you want to do it manually, of course you can. You just need to think through the rules, and translate them to Python.</p>
<p>Let's write it the long way:</p>
<pre><code>def __repr__(self):
    lines = ['{']
    for key, value in self.maze.items():
        lines.append('{}:{}'.format(key, value))
    lines.append(['}'])
    return '\n'.join(lines)
</code></pre>
<p>If you know how to write list comprehensions, you can probably turn that into a one-liner; if not, either go learn, or keep it as 5 lines.</p>
