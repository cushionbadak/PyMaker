<p>The easiest way to do this is with a <code>while True</code> loop and <code>break</code>:</p>
<pre><code>while True:
    line = input("Line: ")
    if not line:
        break
    ...
</code></pre>
<p>This works because empty strings <code>""</code> evaluate <code>False</code>-y. You can also make your code more efficient with lists:</p>
<pre><code>words = []
while True:
    line = input("Line: ")
    if not line:
        break
    for word in line.split():
        words.append(word[::-1])
print(" ".join(words))
</code></pre>
<p>This avoids the constant string concatenation with <code>+</code>. </p>
