<div class="post-text" itemprop="text">
<p>You have declared seleted_columns_final as a string ("'a', 'c'") and again converting it to string? I guess you want to make list of selected columns as a list and create a dataframe using that list.</p>
<p>Try: </p>
<pre><code>seleted_columns_final= ['a','c']
df_final = df[seleted_columns_final]
print(df_final)
</code></pre>
<p>Output:</p>
<pre><code>    a   c
0   5   7
1   8  10
2  11  13
</code></pre>
