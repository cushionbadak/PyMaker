<div class="post-text" itemprop="text">
<p>You need to use <code>get_page</code> method to get the paginated values:</p>
<pre><code>def BlogsList(request):  # Please use snake_case for method name(as per pep8 style guide
    blogs=Blog.objects.all()
    paginator = Paginator(blogs, 5)
    page = request.GET.get('page')
    context={
        'blogs':paginator.get_page(page)
    }
    return render(request, 'blog/home.html', context)
</code></pre>
<p>For more information, please check the <a href="https://docs.djangoproject.com/en/2.1/topics/pagination/#using-paginator-in-a-view" rel="nofollow noreferrer">documentation</a>.</p>
