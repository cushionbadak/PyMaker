<p>the example with <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/" rel="nofollow noreferrer">bs4</a></p>
<pre><code>import bs4
import json

html = """
&lt;div class="swatch-data"&gt;
{"thumbnailImageUrl":"https://www.jbl.com.ph/dw/image/v2/AAUJ_PRD/on/demandware.static/-/Sites-masterCatalog_Harman/default/dw367304ef/JBL_Endurance-SPRINT_Product-Image_Black_Front-1605x1605px.jpg?sw=270&amp;amp;sh=330&amp;amp;sm=fit&amp;amp;sfrm=png","productUrl":"https://www.jbl.com.ph/JBL+Endurance+SPRINT.html?cgid=in-ear-headphones&amp;amp;dwvar_JBL%20Endurance%20SPRINT_color=Black-GLOBAL-","productSupportUrl":"","productID":"JBLENDURSPRINTBLK","orderable":false,"availability":{"message":"","status":"NOT_AVAILABLE"},"price":{"unitLabel":"each","priceType":"standard","salesPrice":"N/A"},"realprice":{"salesPrice":"N/A"},"badges":["new"],"buttonText":"Sold Out","showProdLimit":{"status":""},"CTAEnable":true,"commerceSiteFlag":false,"showPromoTimerFlag":false,"isProProd":false}
&lt;/div&gt;
"""

soup=bs4.BeautifulSoup(html,'lxml')
js_data = json.loads(soup.find('div').text)

# if you want productID just get it
print(js_data['productID'])
</code></pre>
<p><strong>Output</strong></p>
<pre><code>JBLENDURSPRINTBLK

<p>What you see in there is actually a <a href="https://docs.python.org/3/library/json.html" rel="nofollow noreferrer">JSON</a>.</p>
<p>You first of all need to take out the div. Using <a href="https://www.crummy.com/software/BeautifulSoup/bs4/doc/" rel="nofollow noreferrer">BeautifulSoup</a> is one of the recommended methods.</p>
<p>Then, you can load the string using <code>json.loads(str)</code>.</p>
