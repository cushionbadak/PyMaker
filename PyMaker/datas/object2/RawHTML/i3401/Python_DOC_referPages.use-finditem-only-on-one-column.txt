<p>I have a QTableWidget populated with QtableWidgetItems.</p>
<p>I want a searchbar, where I can type in and as Response the Table should be refreshing and only showing the items that match partially with the string in the search field.</p>
<p>Im using finditem for that, but i want that only one column is used for the search. How can I do that?</p>
</div>
<div class="post-text" itemprop="text">
<p>Iterate the table manually.    </p>
<pre><code>columnOfInterest = 1 # or whatever
valueOfInterest = "foo" 
for rowIndex in range(self.myTable.rowCount()):
    twItem = self.myTable.item(rowIndex, columnOfInterest)
    if twItem.text() == valueOfInterest:
        self.myTable.setRowHidden(rowIndex, False)
    else:
        self.myTable.setRowHidden(rowIndex, True)
</code></pre>
<p>You will have to implement better matching criteria. You can use string functions like <a href="https://docs.python.org/3/library/stdtypes.html#string-methods" rel="nofollow noreferrer"><code>str.find</code> and <code>str.startswith</code> and others</a> if you want to do it yourself.</p>
</div>
