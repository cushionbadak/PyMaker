<p>I am looking to extract numbers in the format:</p>
<blockquote>
<p>[number]['/' or ' ' or '\' possible, ignore]:['/' or ' ' or '\'
  possible, ignore][number]['/' or ' ' or '\' possible, ignore]:...</p>
</blockquote>
<p>For example:</p>
<blockquote>
<p>"4852/: 5934: 439028/:\23"</p>
</blockquote>
<p>Would extract: ['4852', '5934', '439028', '23']</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>re.findall</code> to extract all occurrences of a pattern. Note that you should use double backslash to represent a literal backslash in quotes.</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; re.findall(r'\d+', '4852/: 5934: 439028/:\\23')
['4852', '5934', '439028', '23']
&gt;&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Python does have a regex package <a href="https://docs.python.org/2/library/re.html" rel="nofollow noreferrer">2.7</a>, <a href="https://docs.python.org/3/library/re.html" rel="nofollow noreferrer">3.*</a></p>
<p>The function that you would probably want to use is the <code>.split()</code> function</p>
<p>A code snippet would be</p>
<pre><code>import re
numbers = re.split('[/:\]', your_string) 
</code></pre>
<p>The code above would work if thats you only split it based on those non-alphanumeric characters. But you could split it based on all non numeric characters too. like this</p>
<pre><code>numbers = re.split('\D+', your_string)
</code></pre>
<p>or you could do</p>
<pre><code>numbers = re.findall('\d+',your_string)
</code></pre>
<p>Kudos!</p>
