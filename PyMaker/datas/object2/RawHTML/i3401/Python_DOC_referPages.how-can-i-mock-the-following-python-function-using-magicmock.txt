<div class="post-text" itemprop="text">
<pre><code>class C:
  def f():
    calls g

  def g():
    # Do something
</code></pre>
<p>How can I mock g to test f in a testing module which imports class C?</p>
</div>
<div class="post-text" itemprop="text">
<p>u can Mock that function with using path object like that;</p>
<pre><code>with patch.object(C, 'g', MagicMock(return_value='something')):
    c.f()
</code></pre>
<p>with that way when your code calls g function, it returned mocked response</p>
<p>Note: use should import patch before use it</p>
<p>You can find detailed information from link below:</p>
<p><a href="https://docs.python.org/3/library/unittest.mock.html" rel="nofollow noreferrer">https://docs.python.org/3/library/unittest.mock.html</a></p>
</div>
