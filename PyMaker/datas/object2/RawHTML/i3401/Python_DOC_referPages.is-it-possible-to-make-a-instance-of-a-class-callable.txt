<p>I want to call a instance of a class directly, </p>
<p>for expample- </p>
<pre><code>class A(Object):
    def __init(self,temp):
        super(A, self).__init__()
        self.val = temp

    def get_val(self):
        return self.val
</code></pre>
<p>Now, creating an instance - </p>
<pre><code>my_var = A('something')
print(my_var.get_val())
</code></pre>
<p>but i want to print the value only by using -</p>
<pre><code>print(my_var) or print(my_var())
</code></pre>
<p>Is this possible?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can override <code>__str__()</code> method. When you are printing an object reference directly, it internally calls <code>obj.__str__()</code>. Hence, you can achieve what you want by overriding the said method. </p>
<pre><code>class A():
    def __init__(self,temp):
        self.val = temp

    def get_val(self):
        return self.val


    def __str__(self):
        return self.get_val()


my_var = A('something')
print my_var
</code></pre>
<p>OUTPUT:</p>
<pre><code>something
</code></pre>
</div>
