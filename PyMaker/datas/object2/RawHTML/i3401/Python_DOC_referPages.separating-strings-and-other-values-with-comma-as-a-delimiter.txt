<p>I'm struggling to understand your question - it seems you have a string with data inside the string, separated by commas:</p>
<pre><code>data='"Hello, Hey",123,10.04'
</code></pre>
<p>You can use the <a href="https://docs.python.org/3/library/shlex.html" rel="nofollow noreferrer"><code>shlex</code> module</a> to split it respecting the quotes</p>
<pre><code>&gt;&gt;&gt; import shlex
&gt;&gt;&gt; s = shlex.shlex(data)
&gt;&gt;&gt; s.whitespace = ','
&gt;&gt;&gt; s.wordchars += '.'
&gt;&gt;&gt; print(list(s))
['"Hello, Hey"', '123', '10.04']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You may use the <code>re</code> module like so:</p>
<pre><code>[m.group(1) or m.group(2) for m in re.finditer(r'"([^"]*)",?|([^,]*),?', '"Hello, Hey",123,10.04')]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>re.findall</code> with regex pattern <code>"[^"]+"|[^,]+</code>:</p>
<pre><code>import re
print(re.findall(r'"[^"]+"|[^,]+', '"Hello, Hey",123,10.04'))
</code></pre>
<p>This outputs:</p>
<pre><code>['"Hello, Hey"', '123', '10.04']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use <code>re.split</code> to split on a combination of either a double quote before a comma or a comma followed by a digit</p>
<pre><code>import re

data='"Hello, Hey",123,10.04'
re.split(r'(?&lt;="),|,(?=\d)', data)
['"Hello, Hey"', '123', '10.04']
</code></pre>
