<div class="post-text" itemprop="text">
<p>You need to make setUpClass - and tearDownClass - classmethods:</p>
<pre><code>@classmethod
def setUpClass(cls):
    ...

@classmethod
def tearDownClass(cls):
    ...
</code></pre>
<p>See <a href="https://docs.python.org/3/library/unittest.html#unittest.TestCase.setUpClass" rel="nofollow noreferrer">the unittest docs</a>.</p>
