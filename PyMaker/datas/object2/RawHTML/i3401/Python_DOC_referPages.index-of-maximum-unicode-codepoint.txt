<blockquote>
<p>An integer giving the value of the largest Unicode code point, i.e. <code>1114111</code> (<code>0x10FFFF</code> in hexadecimal).</p>
<p><em>Changed in version 3.3:</em> Before PEP 393, <code>sys.maxunicode</code> used to be either <code>0xFFFF</code> or <code>0x10FFFF</code>, depending on the configuration option that specified whether Unicode characters were stored as UCS-2 or UCS-4.</p>
