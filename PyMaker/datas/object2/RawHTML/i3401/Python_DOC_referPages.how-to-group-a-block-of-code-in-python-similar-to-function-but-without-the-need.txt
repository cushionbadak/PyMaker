<div class="post-text" itemprop="text">
<p>How to group a block of code in python, similar to function but without the need to define the parameters?
for example, the block is like this:</p>
<pre><code>code_block(){
    c = a + b
}
</code></pre>
<p>I want to use it in code like this:</p>
<pre><code>a = 2
b = 3
code_block()
print(c)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You need to use global variables inside your function and initialise <code>c</code></p>
<pre><code>a = 2
b = 3
c = 0

def code_block():
    global a, b, c
    c = b + a

code_block()
print(c)
</code></pre>
<p>From the documentation: <a href="https://docs.python.org/3/faq/programming.html#what-are-the-rules-for-local-and-global-variables-in-python" rel="nofollow noreferrer">What are the rules for local and global variables in Python?</a></p>
<blockquote>
<p>In Python, variables that are only referenced inside a function are
  implicitly global. If a variable is assigned a value anywhere within
  the function’s body, it’s assumed to be a local unless explicitly
  declared as global.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>As long as the variables are visible from the scope of the function declaration, you can define yourself a function without parameters that can operate on these values:</p>
<pre><code>x = 0
y = 1
def add():
    return x+y
print(add())
</code></pre>
<p>Note that the scope creates an implicit copy of these values, i.e. it is not possible to manipulate the values outside of the function, i.e.</p>
<pre><code>x = 0
def manipulate():
    x = 1
manipulate()
print(x)
</code></pre>
<p>will still return 0. A nice read is <a href="http://python-textbok.readthedocs.io/en/1.0/Variables_and_Scope.html" rel="nofollow noreferrer">this article</a>.</p>
</div>
<span class="comment-copy">As ScottMcC in his answer noted, you can also manipulate the variables outside, if you define them as <code>global</code>.</span>
