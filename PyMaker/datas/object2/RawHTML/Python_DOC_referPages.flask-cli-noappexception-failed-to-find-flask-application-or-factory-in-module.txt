<div class="post-text" itemprop="text">
<p>Depending on the OS you are using you need to export flaskr in your environment </p>
<p>For Linux and Mac:</p>
<pre><code>export FLASK_APP=flaskr
export FLASK_ENV=development
flask run
</code></pre>
<p>For Windows cmd, use set instead of export:</p>
<pre><code>set FLASK_APP=flaskr
set FLASK_ENV=development
flask run
</code></pre>
<p>For Windows PowerShell, use $env: instead of export:</p>
<pre><code>$env:FLASK_APP = "flaskr"
$env:FLASK_ENV = "development"
flask run```

</code></pre>
