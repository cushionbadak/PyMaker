<div class="post-text" itemprop="text">
<p>You are looking to <strong>apply a function to a list of objects</strong> -- specifically one that is generated as a slice of a larger list.</p>
<p>For this, we use the <a href="http://book.pythontips.com/en/latest/map_filter.html" rel="nofollow noreferrer"><code>map()</code></a> functionality. This takes two parameters: a <code>function</code>, and a <code>list</code>, and applies the function to each element of the list.</p>
<pre><code>map(ABC.add_for_elmt, A[1:3])
</code></pre>
