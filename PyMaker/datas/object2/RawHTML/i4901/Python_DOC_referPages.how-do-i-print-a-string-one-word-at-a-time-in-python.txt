I'\n'
need'\n'
practice</p>
<p>My code so far looks like this:</p>

  for char in phrase:
     print (char, end ='')
  if char == '':
    print('\n')
</code></pre>
<p>However, my output looks like this:</p>
<p>I need practice</p>

<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>.split()</code></a> function to split a string by spaces and that will give you back a list of words that you can then loop and print them out. </p>
<p>The function accepts a string that will be used as the delimiter (e.g ",") <strong>if this argument is not specified or is None</strong> it will run an algorithm that will consider a sequence of whitespaces as a single separator and hence, as a result, no matter how many spaces there are between the words you will get a list of words with no whitespaces at the start or the end of each substring.</p>
<pre><code>phrase = input("Enter a phrase: ")
words = phrase.split()  # ['I', 'need', 'practice']
for word in words:
  print(word)
</code></pre>
</div>
