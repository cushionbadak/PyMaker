<div class="post-text" itemprop="text">
<p>You can convert to int after the split using a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> like:</p>
<pre><code>userin = [int(x) for x in input("Enter first 2x2 matrix: ").split(' ')]
</code></pre>
