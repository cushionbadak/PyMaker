<p>I am struggling to combine the values of two arrays in Python. I want to get the values of two arrays as pairs.<br/>
EXAMPLE: Suppose we have two arrays <code>a</code> and <code>b</code> as below:</p>
<pre><code>a = [[1, 2, 3], [4, 5, 6], [0, 3, 1]]
for i in range(len(a)):
    for j in range(len(a[i])):
        print(a[i][j], end=' ')
    print() 

b = [[4, 0, 3], [6, 3, 6], [1, 4, 1]]

for i in range(len(b)):
    for j in range(len(b[i])):
        print(b[i][j], end=' ')
    print()
</code></pre>
<p>How can I combine the values of two arrays as pairs similar to:      </p>
<pre><code>array([[(1,4), (2,0), (3,3)],
       [(4,6), (5,3), (6,6)],
       [(0,1), (3,4), (1,1)]])
</code></pre>
<p>I am not sure if it can be done as an array or a list.</p>

<div class="post-text" itemprop="text">
<p>You can combine list comprehension and <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip()</a> to do that:</p>
<pre><code>a = [[1, 2, 3], [4, 5, 6], [0, 3, 1]]
b = [[4, 0, 3], [6, 3, 6], [1, 4, 1]]

c = [ list(zip(a[x],b[x])) for x in range(len(a))] # works b/c len(a) = len(b)

print(c)
</code></pre>
<p>Output</p>
<pre><code>[[(1, 4), (2, 0), (3, 3)], [(4, 6), (5, 3), (6, 6)], [(0, 1), (3, 4), (1, 1)]]
</code></pre>
<p>This works correctly as <code>a</code> and <code>b</code> have the same amount of inner lists <em>and</em> the inner lists have the same length. <code>zip()</code> does only create tuples for 2 lists up to the shorter length of both. For inequeal length lists you can use <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow noreferrer"><code>itertools.zip_longest(...)</code></a> which uses <code>None</code> or a specified default to fill up the shorter list.</p>
</div>
