<p>I have a column in a sqlite3 db. This column contains one or more value for single entry.</p>
<p><a href="https://i.stack.imgur.com/wMOCy.png" rel="nofollow noreferrer"><img alt="src" src="https://i.stack.imgur.com/wMOCy.png"/></a></p>
<p>I am talking about <code>tags</code> column.
What I want to do is to extract them as single element.
So I developed this code:</p>
<pre><code>import sqlite3

db = sqlite3.connect('db\\archivio')
db.row_factory = lambda cursor, row: row[0]
c = db.cursor()

tags = c.execute("SELECT tags from documenti").fetchall()

lista_tags = []

for i in tags:
    lista_tags.append(i)

#user_tag = input('Insert tag to search for: ')

for val in lista_tags:
    print(val)
</code></pre>
<p>Result is this:</p>
<pre><code>hansel, gretel
cappuccetto, rosso, lupo
signore, anelli, gollum
blade,runner
pinocchio
incredibili
</code></pre>
<p>The problem is that I can't add them to a list one by one.
If you print <code>lista_tags</code> this is the output:</p>
<pre><code>['hansel, gretel', 'cappuccetto, rosso, lupo', 'signore, anelli, gollum', 'blade,runner', 'pinocchio', 'incredibili']
</code></pre>
<p>How can I add this group of values in single ones inside a list ?</p>
<pre><code>['hansel', 'gretel', 'cappuccetto', 'rosso', 'lupo', 'signore', 'anelli', 'gollum', 'blade','runner', 'pinocchio', 'incredibili']
</code></pre>

<div class="post-text" itemprop="text">
<p>Assuming that your <code>tags = ['hansel, gretel', 'cappuccetto, rosso, lupo', 'signore, anelli, gollum', 'blade,runner', 'pinocchio', 'incredibili']</code> after your <code>fetchall()</code> your can achieve your result with this:</p>
<pre><code>lista_tags = []

for entry in tags:
    lista_tags += entry.split(",")
</code></pre>
<p>The output will be:</p>
<pre><code>['hansel', ' gretel', 'cappuccetto', ' rosso', ' lupo', 'signore', ' anelli', ' gollum', 'blade', 'runner', 'pinocchio', 'incredibili']
</code></pre>
</div>
