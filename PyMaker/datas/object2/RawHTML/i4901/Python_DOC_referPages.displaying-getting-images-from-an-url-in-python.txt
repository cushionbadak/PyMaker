<p>This worked for me</p>
<pre><code>from PIL import Image
import requests
from io import BytesIO

url = "https://baobab-poseannotation-appfile.s3.amazonaws.com/media/project_5/images/images01/01418849d54b3005.o.1.jpg"
response = requests.get(url)
img = Image.open(BytesIO(response.content))
img.show()
</code></pre>
<p>You are getting an error because you used the line <code>response.read("https://baobab-poseannotation-appfile.s3.amazonaws.com/media/project_5/images/images01/01418849d54b3005.o.1.jpg").decode('utf-8')</code> instead. I'd switch back to using <code>response = requests.get(url)</code></p>
<p>Additionally, for your error: <code>ImportError: No module named 'cStringIO'</code>, you are using python3. StringIO and cStringIO from python 2 were removed in python 3. Use <code>from io import StringIO</code> instead. See <a href="https://stackoverflow.com/questions/11914472/stringio-in-python3">StringIO in Python3</a> for more details.</p>

<p>For your third option with <code>PIL</code> you can try this:</p>
<pre><code>from PIL import Image
import requests
import matplotlib.pyplot as plt

response = requests.get(url, stream=True)
img = Image.open(response.raw)

plt.imshow(img)
plt.show()
</code></pre>
