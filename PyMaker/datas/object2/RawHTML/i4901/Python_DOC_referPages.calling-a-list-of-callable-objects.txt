<div class="post-text" itemprop="text">
<p>Say I have a list of callable objects:</p>
<pre><code>[&lt;Python.Proto object at 0x1046ff940&gt;,
&lt;Database.Proto object at 0x10540c518&gt;]
</code></pre>
<p>How would I loop through the list executing every callable, leaving the result in a list?</p>
</div>
<div class="post-text" itemprop="text">
<p>That is fairly straight forward with a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> like:</p>
<pre><code>results = [f() for f in callables]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>def foo():
   pass

def bar():
   pass

l = [str, map, foo, bar]
result = [call() for call in l]
</code></pre>
<p>You have to make sure what parameters you want to pass to each function you can have it as a <code>dict</code></p>
