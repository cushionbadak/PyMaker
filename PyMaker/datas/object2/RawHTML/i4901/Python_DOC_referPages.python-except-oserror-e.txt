<div class="post-text" itemprop="text">
<p>Python 3 has changed how this expression should be written.  <a href="https://docs.python.org/3/reference/compound_stmts.html#the-try-statement" rel="nofollow noreferrer">From the documentation</a>, it is now <code>except OSError as e</code>.</p>
<p>Note that in Python 2, <a href="https://docs.python.org/2.7/reference/compound_stmts.html#the-try-statement" rel="nofollow noreferrer">both comma and <code>as</code> were permitted</a>.</p>
</div>
<span class="comment-copy">That is deprecated Python 2 syntax. Replace it with <code>except OSError as e</code>.</span>
