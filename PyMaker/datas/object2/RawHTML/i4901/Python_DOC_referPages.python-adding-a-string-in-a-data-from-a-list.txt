<div class="post-text" itemprop="text">
<p>Using a list comprehension:</p>
<pre><code>&gt;&gt;&gt; words = ["Hello","how","are","you"]
&gt;&gt;&gt; words = [x + ".mp4" for x in words]
&gt;&gt;&gt; words
['Hello.mp4', 'how.mp4', 'are.mp4', 'you.mp4']
</code></pre>

<pre><code>words = ["Hello","how","are","you"]

<div class="post-text" itemprop="text">
<p>You can use the built-in <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map()</code></a> function.</p>
<p>Directly from the docs:</p>
<blockquote>
<p>Returns an iterator that applies function to every item of iterable, yielding the results.</p>
</blockquote>
<p>Here the function is a <a href="https://docs.python.org/3/tutorial/controlflow.html#lambda-expressions" rel="nofollow noreferrer">lambda expression</a> instead and <em>maps</em> each element from <code>words</code> with <code>+ '.mp4'</code>, which is exactly what you want:</p>
<pre><code>words = ['Hello', 'how', 'are', 'you']
words = list(map(lambda s: s + '.mp4', words))
print(words)
</code></pre>
</div>
