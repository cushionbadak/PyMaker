<div class="post-text" itemprop="text">
<p>Just apply <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>strip()</code></a> first:</p>
<pre><code>c_aux = texto.strip().split(" ")
</code></pre>
<p><code>str.strip()</code> returns a copy of the string with the leading and trailing whitespace removed.</p>
<p>If you need to just remove from the left or right, you can use <code>lstrip()</code> or <code>rstrip()</code> respectively.</p>
