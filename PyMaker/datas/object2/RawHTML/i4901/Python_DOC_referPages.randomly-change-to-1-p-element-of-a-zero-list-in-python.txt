<p>I have a zero list with size of <code>m</code>. I wanna randomly select <code>p</code> indexes and change them to 1.</p>
<p><code>Y = [0 for x in range(M)]</code></p>
<p>What is the best way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use Python's <a href="https://docs.python.org/3/library/random.html#random.shuffle" rel="nofollow noreferrer">random module</a>:</p>
<pre><code>import random

m, p = 5, 2
l = [0] * m

indices = list(range(len(l)))
random.shuffle(indices)

for i in indices[:p]:
    l[i] = 1
</code></pre>
<p>As mentioned by @MrT in the comments if you don't already have your "zero list" you can do:</p>
<pre><code>l = [0] * (m - p) + [1] * p
random.shuffle(l)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can use random.sample()    </p>
<pre><code>import random

m,p = 10, 5
my_list = [0]*m
index = random.sample(range(len(my_list)), p)

for i in index:
    my_list[i] = 1
</code></pre>
