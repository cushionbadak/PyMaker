<div class="post-text" itemprop="text">
<p>Using Python's CSV library with a <code>\t</code> delimiter as follows:</p>
<pre><code>import csv        

data = ['text1', 'text2', 'text3', 'text4']    

with open('output.tsv', 'w', newline='') as f_output:
    tsv_output = csv.writer(f_output, delimiter='\t')
    tsv_output.writerow(data)
</code></pre>
<p>Giving you <code>output.tsv</code> containing one row as:</p>
<pre class="lang-none prettyprint-override"><code>text1   text2   text3   text4
</code></pre>
<p>If you are using Python 2.x, use:</p>
<pre><code>with open('output.tsv', 'wb') as f_output:
</code></pre>
