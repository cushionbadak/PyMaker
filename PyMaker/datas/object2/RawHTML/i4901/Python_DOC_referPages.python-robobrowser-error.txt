<div class="post-text" itemprop="text">
<p>It seems like you are trying to access the <a href="https://robobrowser.readthedocs.io/en/latest/api.html?highlight=open#robobrowser.browser.RoboBrowser.open" rel="nofollow noreferrer"><code>open</code> method of the <code>bra</code> variable (RoboBrowser)</a>. The problem is that you are using an equals sign and are actually calling the standard library <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer">built-in function <code>open</code></a>. Changing the equals sign to a period should fix this.</p>
<pre><code>bra=open("http://webpage.com")
   ^
   v
bra.open("http://webpage.com")
</code></pre>
