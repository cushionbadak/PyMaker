<div class="post-text" itemprop="text">
<p>You can not assign to a <code>list</code> element that does not already exist. Instead simply append the result to <code>y</code> like:</p>
<pre><code>y = []
for i in range(0, 20):
    y.append(m * i + b)
</code></pre>
<p>or as a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>:</p>
<pre><code>y = [m * i + b for i in range(0, 20)]
</code></pre>
