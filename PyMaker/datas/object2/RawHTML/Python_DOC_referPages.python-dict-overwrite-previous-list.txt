<div class="post-text" itemprop="text">
<p>use <code>.copy()</code> </p>
<pre><code>Temp = {}
Final = {}

OK =['OK','OK']
NOTOK =['NOTOK','NOTOK']

Temp[1] = OK.copy()
Final['u1']=Temp.copy()

print(Final)

Temp.clear()
Temp[2] = OK.copy()
Final['u2']=Temp.copy()

print(Final)

Temp.clear()
Temp[3] = NOTOK.copy()
Final['u3']=Temp.copy()

print(Final)
</code></pre>
