<p>You are sending the data in the path like this: </p>
<pre><code>$.post('/insert/', '&amp;register_new_user=register_new_user' + 
'&amp;csrftoken={{ csrf_token }}' , function(response) {
    $('#results_container').html(response);
});
</code></pre>
<p>Try sending the data in the body:</p>
<pre><code>$.post( '/insert/', { register_new_user: "register_new_user", csrftoken: "your_token_here" } );
</code></pre>
