<p>Suppose I have two lists of different length.</p>
<pre><code>a = [8,9,4,7,5,6,1,4,8]
b = [6,4,7,1,5,8,3,6,4,4]
</code></pre>
<p>I want a list like this:</p>
<pre><code>c= a-b

#output = [2, 5, -3, 6, 0, -2, -2, -2, 4]
</code></pre>
<p>How can i achieve this?</p>
<p>I tried <code>operator.sub</code> with map function. But I am getting an error because of different lengths of list.</p>
<pre><code>c = map(operator.sub, a, b)
</code></pre>
<blockquote>
<p>TypeError: unsupported operand type(s) for -: 'NoneType' and 'int'</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<pre><code>from itertools import starmap
from operator import sub

a = [8,9,4,7,5,6,1,4,8]
b = [6,4,7,1,5,8,3,6,4,4]

output = list(starmap(sub, zip(a, b)))
</code></pre>
<p>If you do not want to use a <em>list comprehension</em>, this can be done with <a href="https://docs.python.org/2/library/itertools.html#itertools.starmap" rel="nofollow noreferrer"><code>itertools.starmap</code></a>.</p>
<p>You could also use map, though I think starmap is the better option. With map you could use a nested <code>zip</code> to shorten the longer argument.</p>
<pre><code>output = map(sub, *zip(*zip(a, b)))
print(list(output))
# [2, 5, -3, 6, 0, -2, -2, -2, 4]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You may use <a href="https://docs.python.org/2/library/functions.html#zip" rel="noreferrer"><code>zip</code></a> along with <em>list comprehension</em> expression as:</p>
<pre><code>&gt;&gt;&gt; a = [8,9,4,7,5,6,1,4,8]
&gt;&gt;&gt; b = [6,4,7,1,5,8,3,6,4,4]

&gt;&gt;&gt; [x - y for x, y in zip(a, b)]
[2, 5, -3, 6, 0, -2, -2, -2, 4]
</code></pre>
