<div class="post-text" itemprop="text">
<p>In python3.x,you should use this:</p>
<pre><code>from hashlib import md5
</code></pre>
<blockquote>
<p>You can now feed this object with <strong>bytes-like</strong> objects (normally
  bytes) using the update() method.</p>
</blockquote>
<p>e.g.</p>
<pre><code>from hashlib import md5

m = md5()
m.update(b"Nobody")
print(m.hexdigest())
</code></pre>
<blockquote>
<p>Module name:   md5 .<br/>
  Rationale:     Replaced by the 'hashlib' module.<br/>
  Date:          15-May-2007 .<br/>
  Documentation: Documented as deprecated as of Python 2.5, but listing
  in this PEP was neglected.DeprecationWarning raised as of Python 2.6.</p>
</blockquote>
<p>See more details from <a href="https://docs.python.org/3/library/hashlib.html" rel="nofollow noreferrer">hashlib</a>.</p>
