<p>You'll have to make the count a class variable.</p>
<pre><code>def __init__(self, ball, paddle):
    ...
    self.count = 0

def points(self, paddle):
    point = 0
    if self.ball.x &lt; paddle.getX():
        self.count += 1
        print(count)
</code></pre>
<p>If you set <code>count</code> to 0 in the <code>points()</code> method, count is set to 0 again every time the method is called.</p>
