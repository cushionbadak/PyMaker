<div class="post-text" itemprop="text">
<p>Given a string as input, I want to find and convert the integer elements of the line with the string <code>num</code>. Here is an example:</p>
<p>input: </p>
<pre><code>"abc 123 def 1 in 1s and 100MB"
</code></pre>
<p>output:</p>
<pre><code>"abc &gt;num&lt; def &gt;num&lt; in 1s and 100MB"
</code></pre>
<p>How can I do it?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/3.6/library/re.html" rel="nofollow noreferrer">regular expression</a>:</p>
<pre class="lang-py prettyprint-override"><code>import re

s = "abc 123 def 1 in 1s and 100MB"

print(re.sub(r"\b\d+\b", "&gt;num&lt;", s))
# output: "abc &gt;num&lt; def &gt;num&lt; in 1s and 100MB"
</code></pre>
<p>The regular expression string <code>r"\b\d+\b"</code> matches a character string that consists of at least one numeral character (<code>\d+</code>) enclosed by word boundaries (<code>\b</code>). The <code>\b</code> part also matches the beginning and the end of a string.</p>
<p>The call to <code>re.sub()</code> replaces all occurrences of a matching character string in the third argument <code>s</code>  by the second argument (<code>"&gt;num&lt;"</code>).</p>
</div>
<div class="post-text" itemprop="text">
<p>That is how.</p>
<pre><code>text_in = "abc 123 def 1 in 1s and 100MB"
text_out = ' '.join(['&gt;num&lt;' if x.isnumeric() else x for x in text_in.split()])
print(text_out)  # prints: abc &gt;num&lt; def &gt;num&lt; in 1s and 100MB
</code></pre>
<p>I took the liberty to assume that you can read the text from a file and can also write the new text to a file so I am skipping those parts.</p>
<p>It all boils down to the use of <a href="https://docs.python.org/3/library/stdtypes.html#str.isnumeric" rel="nofollow noreferrer"><code>isnumeric()</code></a>.</p>
