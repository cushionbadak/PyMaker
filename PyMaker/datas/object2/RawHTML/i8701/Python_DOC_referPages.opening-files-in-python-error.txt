<p>When opening files in python, I get the error:</p>
<pre><code>UnicodeDecodeError: 'ascii' codec can't decode byte 0xe2 in position 0: ordinal not in range(128)
</code></pre>
<p>My code is</p>
<pre><code>p=open("afile.txt","r")
file=p.read()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This is a UnicodeError. You are trying to read a Unicode character in with the ASCII codec.</p>
<p>Try:</p>
<pre><code>import codecs

p = codecs.open("afile.txt", "r", "utf-8")
f = p.read()
</code></pre>
<p>or</p>
<pre><code>import codecs

p = codecs.open("afile.txt", "r", "utf-16")
f = p.read()
</code></pre>
<p>You should also probably consider using:</p>
<pre><code>with codecs.open("afile.txt", "r", "utf-8") as f:
    # Do whatever you want with f
</code></pre>
<p>This makes it able to auto-close the file when you exit the <code>with</code> statement.</p>
<p>You could also try <code>iso-8859-15</code> or <code>cp437</code> Take a look at <a href="https://pypi.python.org/pypi/chardet" rel="nofollow noreferrer">https://pypi.python.org/pypi/chardet</a> and <a href="https://docs.python.org/3/library/codecs.html#codecs.open" rel="nofollow noreferrer">https://docs.python.org/3/library/codecs.html#codecs.open</a>.</p>
