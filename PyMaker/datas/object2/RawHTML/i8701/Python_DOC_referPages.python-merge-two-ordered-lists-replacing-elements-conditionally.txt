<p>Beginner's question.</p>
<p>I have two ordered lists with the same order, </p>
<pre><code>list_1=['de', 'de', 'de', '3']
list_2=['6', '5', '3', 'not']
</code></pre>
<p>so the n-th element of list_1 corresponds to the n-th element of list_2. </p>
<p>I want to merge the two lists while perserving their order. Moreover, the list resulting from the merge should consist entirely out of numbers, i.e. </p>
<pre><code>merged_list=['6', '5', '3', '3']
</code></pre>
<p>Preferably, I merge by position in the list conditionally on string or numeric values. I have the same issue with other ordered lists, but with those lists I want to replace numeric values with string values. I.a. to solve my troubles with all the ordered lists I have I'd like to do something like:</p>
<pre><code>replace element_n of list_i with element_n of list_j if element_n of list_i equals 'z'
</code></pre>
<p>where z is either a numeric value or a string value, dependent on the nature of the list.</p>

<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> to aggregates two list items:</p>
<pre><code>&gt;&gt;&gt; list_1 = ['de', 'de', 'de', '3']
&gt;&gt;&gt; list_2 = ['6', '5', '3', 'not']
&gt;&gt;&gt; zip(list_1, list_2)
&lt;zip object at 0x1029e53c8&gt;
&gt;&gt;&gt; list(zip(list_1, list_2))
[('de', '6'), ('de', '5'), ('de', '3'), ('3', 'not')]
</code></pre>
<p>, and <a href="https://docs.python.org/3/library/stdtypes.html#str.isdigit" rel="nofollow noreferrer"><code>str.isdigit</code></a> to check if the given string is a number:</p>
<pre><code>&gt;&gt;&gt; '123'.isdigit()
True
&gt;&gt;&gt; 'de'.isdigit()
False
</code></pre>
<p>By combining with <a href="https://docs.python.org/3/reference/expressions.html#conditional-expressions" rel="nofollow noreferrer">conditional expression</a> and <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a>, you will get what you want:</p>
<pre><code>&gt;&gt;&gt; [a if a.isdigit() else b for a, b in zip(list_1, list_2)]
['6', '5', '3', '3']
</code></pre>
