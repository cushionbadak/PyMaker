<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/ast.html#ast.literal_eval" rel="nofollow noreferrer"><code>ast.literal_eval</code></a> to evaluate a string that contains a valid Python literal. To handle your data we need to decode the <code>bytes</code> object to a string. We can use the default UTF-8 encoding.</p>
<pre><code>from ast import literal_eval

s = b"['cat', 'dog']"
a = literal_eval(s.decode())
print(a, type(a))
</code></pre>
<p><strong>output</strong></p>
<pre><code>['cat', 'dog'] &lt;class 'list'&gt;
</code></pre>
</div>
<span class="comment-copy">You can use <code>ast.literal_eval</code>, but it would probably be better to fix how you're creating or getting that cached data so it's actually a proper list in the first place.</span>
