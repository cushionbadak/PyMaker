<div class="post-text" itemprop="text">
<p>I'm not sure where you got your call syntax, but if you <a href="https://docs.python.org/3/library/http.client.html" rel="nofollow noreferrer">read the documentation</a>, you'll see that you need to write you code as</p>
<pre><code>import http.client
def request(word):
    conn = http.client.HTTPSConnection('inputtools.google.com')
    conn.request('GET', '/request?text=' + word + '&amp;itc=fr-t-i0-und&amp;num=5&amp;cp=0&amp;cs=1&amp;ie=utf-8&amp;oe=utf-8&amp;app=test')
    res = conn.getresponse()
    print(res.status, res.reason)
    print(res.read())

request(word='deep')
</code></pre>
