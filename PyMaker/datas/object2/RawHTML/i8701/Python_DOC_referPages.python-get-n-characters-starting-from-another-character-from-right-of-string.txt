<div class="post-text" itemprop="text">
<p>I have the following string:</p>
<pre><code>st = "../dir1/dir2/dirN/thisiswhatiwantonlyfirstsevencharacters"
</code></pre>
<p>I am trying to get the first seven characters starting from the first slash from the right.
Currently I do it manually:</p>
<pre><code>st[18:-32]
</code></pre>
<p>how can i do it by looking up the first slash from right and then get the first seven characters?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use <code>str.rsplit()</code> and a simple indexing:</p>
<pre><code>In [19]: st.rsplit('/', 1)[-1][:7]
Out[19]: 'thisisw'
</code></pre>
</div>
<div class="post-text" itemprop="text">

<pre><code>st = "../dir1/dir2/dirN/thisiswhatiwantonlyfirstsevencharacters"

print st[last_slash_index:last_slash_index+8]
</code></pre>
</div>

<div class="post-text" itemprop="text">
<p>Use <code>rfind</code>, r stands for right, meaning it will start looking at the string right to left.</p>
<pre><code>i = st.rfind('/')
st[i + 1: i + 8]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<pre><code>st = "../dir1/dir2/dirN/thisiswhatiwantonlyfirstsevencharacters"
slashes = st.split('/')
print(slashes)
print(slashes[-1:][0][:7])
</code></pre>
