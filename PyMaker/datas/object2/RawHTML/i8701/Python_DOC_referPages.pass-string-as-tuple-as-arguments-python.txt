<div class="post-text" itemprop="text">
<p>If I have a string like:</p>
<pre><code>"spam, foo, moo"
</code></pre>
<p>How would I pass that to a function so it would turn into:</p>
<pre><code>myFunction("spam", "moo", "foo")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can split your string to list of things:</p>
<pre><code>your_string = "spam, foo, moo"
your_function(*your_string.split(', '))
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can split your string and <a href="https://docs.python.org/3/tutorial/controlflow.html#unpacking-argument-lists" rel="nofollow noreferrer">unpack the elements</a>, in order to get 3 arguments. </p>
<p>With just split, you'd get one argument, a list of 3 strings.</p>
<pre><code>myFunction(*"spam, foo, moo".split(', '))
</code></pre>
</div>
