<div class="post-text" itemprop="text">
<p>There is no way to change the behaviour of <a href="https://pypi.python.org/pypi/faulthandler" rel="nofollow">faulthandler</a> to do something different other than log faults to a file-like object according to the <a href="https://docs.python.org/3/library/faulthandler.html" rel="nofollow">documetnation</a></p>
<p>However you can change <a href="https://docs.python.org/2/library/sys.html#sys.excepthook" rel="nofollow"><code>sys.excepthook</code></a> and use a <a href="http://pyqt.sourceforge.net/Docs/PyQt4/qmessagebox.html" rel="nofollow"><code>PyQt4.QtGui.QMessageBox</code></a></p>
<p><strong>Example:</strong></p>
<pre><code>#!/usr/bin/env python

import sys


from PyQt4.QtGui import QMainWindow, QMessageBox


class App(QMainWindow):

    def __init__(self, parent=None):
        QMainWindow.__init__(self, parent)

        sys.excepthook = self._displayError

    def _error(self, etype, evalue, etraceback):
        QMessageBox.critical(
            self,
            "ERROR",
            "An unexpected error occurred: {0:s}".format(evalue)
        )
</code></pre>
