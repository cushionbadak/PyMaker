<p>It is the same tag instances in both cases, but with different <a href="https://docs.python.org/2/reference/datamodel.html#object.__repr__" rel="nofollow">string representations</a>.</p>
<pre><code>print tag        # using tag.__str__
print [tag, tag] # using tag.__repl__ for each element
</code></pre>
<p>You can either add <code>str(tag)</code> to your list, or use <code>str(element)</code> when reading them out again.</p>
<p>Example:</p>
<pre><code>from exifread import process_file

with open('dfround2.jpg', 'rb') as f:
    tags = process_file(f)
    time = tags['Image DateTime']
    orig = tags['EXIF DateTimeOriginal']
    print [str(time), str(orig)]
    # ['2014:12:05 18:12:09', '2013:07:20 15:53:30']
</code></pre>

<p>I'm only about 80% sure I understand what you're asking, but…</p>
<p>This prints out the string you want:</p>
<pre><code>print("DateTimeOriginal : %s" % (tags[tag]))
</code></pre>
<p>This stores some complex <code>tag</code> object that you don't want:</p>
<pre><code>time.append(tags[tag])
</code></pre>
<p>What you want is just the same string you printed out, not the <code>tag</code> object.</p>
<p>If you look at the docs for <a href="https://docs.python.org/3/library/stdtypes.html#printf-style-string-formatting" rel="nofollow"><code>printf</code>-style String Formatting</a>, the conversion type <code>%s</code> gives you:</p>
<blockquote>
<p>String (converts any Python object using <code>str()</code>).</p>
</blockquote>
<p>So, if you want to store the same thing that gets printed with <code>%s</code>, just call <code>str()</code>, the same way it does:</p>
<pre><code>time.append(str(tags[tag]))
</code></pre>
<hr/>
<p>I'm not sure why you want to build a list containing the string representations of all the <code>DateTime</code> and <code>DateTimeOriginal</code> tags interleaved like this, but if that's what you want, that's how to do it.</p>
</div>

<p>Try to use <code>.printable</code> to get just string representations. Actually if you'll check sources of <code>exifread</code> you'll see that <code>str(tag)</code> returns <code>printable</code>.</p>
<pre><code>if 'Image DateTime' in tag:
    print("DateTime : %s" % (tags[tag]))
    time.append(tags[tag].printable)
if 'EXIF DateTimeOriginal' in tag:
    print("DateTimeOriginal : %s" % (tags[tag]))
    time.append(tags[tag].printable)
</code></pre>
