<p>Is there a way to convert the following forward into a time object?</p>
<pre><code>PT1H37M21S
</code></pre>
<p>Or is it required that a do <code>splits</code> and <code>strips</code>, etc. in order to extract the time object?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use format strings to parse date/times. You can either use Python's datetime <a href="https://docs.python.org/2/library/datetime.html#datetime.datetime.strptime" rel="nofollow">strptime</a> directly or more human-friend Arrow library <a href="http://crsmithdev.com/arrow/#creation" rel="nofollow">arrow.get()</a> (see parse string).</p>
<p>The format does not look like any common datetime format so you need to write your own parser formatter.</p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/re.html" rel="nofollow">Regular expressions</a> are often better than splits and strips.</p>
<pre><code>from datetime import time
import re

m = re.match(r'^PT([0-9]+)H([0-9]+)M([0-9]+)S$', 'PT1H37M21S')
t = time(int(m.group(1)), int(m.group(2)), int(m.group(3)))
# t is now datetime.time(1, 37, 21)
</code></pre>
<p>(I don't know what <code>PT</code> means so I hope it's just a prefix.)</p>
</div>
