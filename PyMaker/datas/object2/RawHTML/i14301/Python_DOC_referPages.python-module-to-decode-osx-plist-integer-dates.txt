<p>Is there a python module/snippet which can decode the integer datestamps contained in OSX - specifically mail - plists?</p>
<p>EG, this bit: </p>
<pre><code>&lt;key&gt;date-sent&lt;/key&gt;    
&lt;integer&gt;1264001747&lt;/integer&gt;
</code></pre>
<p>is likely in Jan 2010.</p>
<p>How to deconstruct? I'm aware of the very good plistlib - but this only gets me to that integer.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/datetime.html#datetime.datetime.fromtimestamp" rel="nofollow"><code>datetime.datetime.fromtimestamp</code></a></p>
<pre><code>&gt;&gt;&gt; import datetime
&gt;&gt;&gt; datetime.datetime.fromtimestamp(1264001747)
datetime.datetime(2010, 1, 20, 10, 35, 47)
</code></pre>
<p>The value <code>1264001747</code> is the timestamp given in seconds from epoch. The returned <code>datetime</code> object is shown in the order <code>(year, month, day, hour, minute, second)</code></p>
