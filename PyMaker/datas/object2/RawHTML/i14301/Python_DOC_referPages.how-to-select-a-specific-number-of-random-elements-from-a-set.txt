<p>I have a set of 9 elements and I want to write a program which prompts the user for an integer <code>n</code>, then displays <code>n</code> number of elements selected randomly from my set. </p>
<p>This is what I tried:</p>
<pre><code>import random

def main():    
    best_of = eval(input("How many maps do you want to choose? : "))

    choices = ["Arkansas", "Manchuria", "Bengal", "Baja California", "Tibet", "Indonesia", "Cascade Range", "Hudson Bay", "High Plains"]

    random_choice = random.choice(choices)

    for i in range(bes_of):    
        print(random_choice)

main()
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use the <a href="https://docs.python.org/3/library/random.html#random.sample" rel="nofollow"><code>random.sample()</code> function</a> to pick <code>n</code> random elements without repetition:</p>
<pre><code>sampled = random.sample(choices, best_of)
for choice in sampled:
    print(choice)
</code></pre>
<p>If all you need is an integer from the user, don't use <code>eval()</code>; stick to using <code>int()</code> instead:</p>
<pre><code>best_of = int(input("How many maps do you want to choose? : "))
</code></pre>
<p><code>eval()</code> gives you more than you bargained for; it executes <em>any valid Python expression</em>, letting the user do anything they want with your program.</p>
</div>
<div class="post-text" itemprop="text">
<p>You neeed to call the <code>random.choice()</code> method inside the for loop so that <code>n</code> random elements are printed.</p>
<pre><code>import random

def main():    
    best_of = input("How many maps do you want to choose? : ")

    choices = ["Arkansas", "Manchuria", "Bengal", "Baja California", "Tibet", "Indonesia", "Cascade Range", "Hudson Bay", "High Plains"]

    for i in range(int(best_of)):    
        random_choice = random.choice(choices)
        print(random_choice)

main()
</code></pre>
