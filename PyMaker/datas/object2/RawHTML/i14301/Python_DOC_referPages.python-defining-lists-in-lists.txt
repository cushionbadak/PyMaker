<div class="post-text" itemprop="text">
<p>Lists don't take name-value pairs. You probably want dictionaries here instead:</p>
<pre><code>definitions = {
    'title_label': {
        'text': (236, 218, 51),
        'background': (125, 142, 246)
    },
    'start_button': {
        'text': (32, 40, 145),
        'background': (236, 235, 136),
        'pressed': (44, 51, 112)
    },
    'quit_button': {
        'text': (166, 21, 13),
        'background': (48, 61, 188),
        'pressed': (31, 40, 129)
    }
}
</code></pre>
<p>I'm not sure where you found your syntax, but it is not valid Python. Python lists, using <code>[...]</code> square brackets, can only take a sequence of individual Python expressions:</p>
<pre><code>some_list = ['one object', {'dictionary': 'value'}, ['another', 'list']]
</code></pre>
<p>See the <a href="https://docs.python.org/3/tutorial/datastructures.html" rel="noreferrer"><em>Data structures</em> section</a> of the Python tutorial.</p>
