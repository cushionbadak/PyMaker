<div class="post-text" itemprop="text">
<p>I am working on creating a list from a long paragraph concatenated based on character count. </p>
<p>As an example: if the character count is set to 100 the string containing a long paragraph should be split in to several lines with a max limit of 100. Lines should not contain incomplete words (if the line contains part of the word it should move to the next line).</p>
<p>I can split the string but I can't think of handling incomplete words (words are set of characters separated from space). </p>
<p>Finally, those lines should be returned as a list. </p>
</div>
<div class="post-text" itemprop="text">
<p>First, split all text into words:</p>
<pre><code>words = [w for w in text.split(' ') if w]
</code></pre>
<p>Then loop through the words and add them one by one to the new string, until it's length not breaks the limit. In the case add the string to the list of results and start to create the next string.</p>
<pre><code>MAX_LENGTH = 100

results = []
r = ''

for w in words:
    if len(r) + len(w) + 1 &gt; MAX_LENGTH:
        results.append(r)
        r = ''
    r += '{}{}'.format(' ' if r else '', w)

print results
</code></pre>
</div>
