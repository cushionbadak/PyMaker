<p>You'll need to provide a <code>key</code> function to sort case insensitively.</p>
<p>On Python 3 you'd use the <a href="https://docs.python.org/3/library/stdtypes.html#str.casefold" rel="nofollow"><code>str.casefold()</code> function</a>, on Python 2 sticking to <code>str.lower()</code> is fine:</p>
<pre><code>OrderedDict(sorted(named_sets.items(), key=lambda i: i[0].lower()))
</code></pre>
<p>Note the <code>lambda</code>; you are sorting <em>key-value</em> pairs, but <code>set</code> objects are not orderable so you want to return just the key, folded to compare without case.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; from collections import OrderedDict
&gt;&gt;&gt; named_sets  = {'DMXAA':set(['1441326_at', '1460062_at']), 'cdiGMP':set(['1441326_at', '1460062_at']),'cGAMP': set(['1441326_at', '1460062_at'])  }
&gt;&gt;&gt; OrderedDict(sorted(named_sets.items(), key=lambda i: i[0].lower()))
OrderedDict([('cdiGMP', set(['1441326_at', '1460062_at'])), ('cGAMP', set(['1441326_at', '1460062_at'])), ('DMXAA', set(['1441326_at', '1460062_at']))])
&gt;&gt;&gt; _.keys()
['cdiGMP', 'cGAMP', 'DMXAA']
</code></pre>
