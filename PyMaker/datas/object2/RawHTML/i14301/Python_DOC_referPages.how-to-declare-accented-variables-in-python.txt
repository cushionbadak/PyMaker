<div class="post-text" itemprop="text">
<p>Python 1 and 2 only support ASCII alphanumeric characters and <code>_</code> in <a href="https://docs.python.org/2/reference/lexical_analysis.html#identifiers">identifiers</a>.</p>
<p>Python 3 supports all Unicode letters and certain marks in <a href="https://docs.python.org/3/reference/lexical_analysis.html#identifiers">identifiers</a>.</p>
<pre><code>#!/usr/bin/python3
# -*- coding: utf-8 -*-
mia_età = 31
print(mia_età)
</code></pre>
<p>Identifiers are <a href="https://en.wikipedia.org/wiki/Unicode_equivalence#Normal_forms">normalized</a> according to NFKC, so you can indifferently write U+0061 LATIN SMALL LETTER A followed by U+0301 COMBINING ACUTE ACCENT, or U+00E1 LATIN SMALL LETTER A WITH ACUTE.</p>
