<span class="comment-copy">@joey you need to either <code>import urllib.request</code>, in which case the code in your comment will work as-is, or do something like <code>from urllib.request import pathname2url</code>, in which case you should change your code to <code>url = pathname2url(file)</code>. Also, while using <code>file</code> as a variable name is technically OK, <a href="https://docs.python.org/2/library/functions.html#file" rel="nofollow noreferrer"><code>file</code></a> is a built-in function in Python 2, so if you'll ever be working in that version of the language you'll have to be careful about masking it.</span>
