<div class="post-text" itemprop="text">
<p>This seems like a simple question but I've looked around and cannot find a simple answer.</p>
<p>I've set up a virtual environment in my project folder using <code>canopy_cli venv &lt;myvenv&gt;</code>, and now I can run plain python in that environment. How can I run the Canopy GUI using this virtual environment instead? </p>
</div>
<div class="post-text" itemprop="text">
<p>Your launch script will have to activate that virtual environment:
    $ source myenv/bin/activate
    $ python my_app.py</p>
<p>or the equivalent in Windows:
    C:&gt; /Scripts/activate.bat
    C:&gt; python my_app.py</p>
<p>See <a href="https://docs.python.org/3/library/venv.html" rel="nofollow" title="here">here</a> for details.</p>
</div>
