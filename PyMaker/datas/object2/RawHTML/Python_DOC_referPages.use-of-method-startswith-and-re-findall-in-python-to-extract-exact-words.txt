&amp;attri 'cde' real
&amp;attri 'efg' {'0', '1'}
&amp;attri 'ghi_jkl' real
&amp;attri 'lmn' real
&amp;attri 'nop' real
&amp;attri 'pqr_stu_uvw' real
&amp;attri 'xy_z' {'0', '1'}
</code></pre>
<p>I want to extract all the words <code>'abc', 'cde', 'efg', 'ghi_jkl', 'lmn', 'nop', 'pqr_stu_uvw', 'xy_z'</code> from that file.</p>
<p>I wrote the following code, but it prints just <code>[]</code>.</p>
<pre><code>import re

col = []

with open('text1.txt', 'r') as f:
    lines=f.readlines()
    for line in lines:
        l = line.strip()
        if l.startswith('&amp;attri'):
            col.append(re.findall(r"'(.{1})'", l))
print(col)  
</code></pre>

<div class="post-text" itemprop="text">
<p>You don't need regex here considering that you always need to extract second word of each line out of file. Use <code>split()</code> and extract second split:</p>
<pre><code>col = []

with open('text1.txt', 'r') as f:
    for line in f:
        if line.strip().startswith('&amp;attri'):
            col.append(<b>line.split()[1]</b>.strip("'"))

print(col)</code></pre>

<p>Try this:</p>
<pre><code>import re

s = '''&amp;attri 'abc' real
&amp;attri 'cde' real
&amp;attri 'efg' {'0', '1'}
&amp;attri 'ghi_jkl' real
&amp;attri 'lmn' real
&amp;attri 'nop' real
&amp;attri 'pqr_stu_uvw' real
&amp;attri 'xy_z' {'0', '1'}'''


print(re.findall("'[a-zA-Z_]*'",s)
</code></pre>
<h1>Result:</h1>
<pre><code>["'abc'", "'cde'", "'efg'", "'ghi_jkl'", "'lmn'", "'nop'", "'pqr_stu_uvw'", "'xy_z'"]
</code></pre>
