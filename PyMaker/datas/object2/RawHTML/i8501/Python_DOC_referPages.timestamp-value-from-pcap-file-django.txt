<p><a href="https://docs.python.org/3/library/struct.html#struct.unpack" rel="nofollow noreferrer"><code>struct.unpack()</code></a> returns a tuple.</p>
<p><a href="https://docs.python.org/3/library/time.html#time.localtime" rel="nofollow noreferrer"><code>time.localtime()</code></a> takes an integer argument.</p>
<p>Hence:</p>
<blockquote>
<p>Error: TypeError: an integer is required (got type tuple)</p>
</blockquote>
