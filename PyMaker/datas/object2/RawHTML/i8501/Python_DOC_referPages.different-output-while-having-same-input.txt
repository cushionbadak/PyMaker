<div class="post-text" itemprop="text">
<p>You defined a set, which is unordered. You might use tuple or lists instead:</p>
<pre><code>groceries = ['fruits', 'milk', 'sweet', 'egg'] # list
groceries = ('fruits', 'milk', 'sweet', 'egg') # tuple
</code></pre>
<p>Also there is an <a href="https://docs.python.org/2/library/collections.html#collections.OrderedDict" rel="nofollow noreferrer">ordered version</a> of the dictionary.</p>
