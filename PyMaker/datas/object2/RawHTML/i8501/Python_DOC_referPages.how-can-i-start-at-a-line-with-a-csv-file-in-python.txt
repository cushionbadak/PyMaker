<div class="post-text" itemprop="text">
<p>Use <code>enumerate()</code> you can have the line number as below, by default the count <code>start</code> is starting from 0, you can force it starting from 1: </p>
<pre><code>for line_number, line in enumerate(reader,start=1):
    if line_number &gt; 3:
        print(line)
</code></pre>
</div>
