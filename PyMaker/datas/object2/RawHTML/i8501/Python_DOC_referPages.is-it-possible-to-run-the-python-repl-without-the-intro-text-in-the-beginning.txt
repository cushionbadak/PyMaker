<div class="post-text" itemprop="text">
<p>You can omit the version and copyright messages by starting the interpreter with the <code>-q</code> flag (Python <a href="https://docs.python.org/3/whatsnew/3.2.html#other-language-changes" rel="nofollow noreferrer">3.2+</a>):</p>
<pre><code>$ python -q  
&gt;&gt;&gt;
</code></pre>
<p>The primary prompt (<code>"&gt;&gt;&gt; "</code> by default) can  be  changed  by assignment to <a href="https://docs.python.org/3.6/library/sys.html#sys.ps1" rel="nofollow noreferrer"><code>sys.ps1</code></a>:</p>
<pre><code>&gt;&gt;&gt; import sys
&gt;&gt;&gt; sys.ps1 = "&gt;&gt; "
&gt;&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Eugene's answer is perfect when you're using Python 3. In Python 2, however, that option does not exist. You'll be forced to do something like <code>python -ic ''</code>. That works because <code>-c ''</code> means to execute nothing and <code>-i</code> means to do it and then enter interactive mode. A side effect of this is that it will not read your startup scripts, if you have any.</p>
