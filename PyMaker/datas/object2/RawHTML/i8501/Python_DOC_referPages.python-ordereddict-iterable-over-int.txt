<p>I'm learning about dictionaries and I wrote a small bit of code and built it.</p>
<pre><code>from collections import OrderedDict

d1 = OrderedDict([('a',0),('b',9),('c',8),('d',7),('e',6)])
d2 = OrderedDict([(1),(2),(3),(4),(5)])
</code></pre>
<p>I get the following error:</p>
<pre><code>d2 = OrderedDict([(1),(2),(3),(4),(5)])
TypeError: 'int' object is not iterable
</code></pre>
<p>I don't understand why it isn't iterable? And what has <code>'int'</code> got to do with the problem?</p>
<p>Thanks</p>

<div class="post-text" itemprop="text">
<p>As documented <a href="https://docs.python.org/3/library/collections.html#ordereddict-examples-and-recipes" rel="nofollow noreferrer">here</a> you need to provide <code>key-value</code> pairs to <code>OrderedDict</code>.</p>
<p>At its core, <code>OrderedDict</code> is a <code>dictionary</code> and thus needs a <code>value</code> to store at a <code>key</code>.
Passing it (1) is wrong because:</p>
<pre><code>&gt;&gt;a = (1)
&gt;&gt;type(a)
&lt;class 'int'&gt;
</code></pre>
<p>So, (1) is not a tuple but an <code>integer</code> object.</p>
</div>
<div class="post-text" itemprop="text">
<p><code>OrderedDict</code>s (or any sub-class of <code>dict</code> for that matter) are a set of key-value pairs.</p>
<pre><code>&gt;&gt;&gt; d1 = OrderedDict([('a',0),('b',9),('c',8),('d',7),('e',6)])
&gt;&gt;&gt; d1['b']
9
</code></pre>
<p>The following statement does not provide any such pairs.</p>
<pre><code>d2 = OrderedDict([(1),(2),(3),(4),(5)])
</code></pre>
<p>Also note that<br/><code>(1)</code> is an <code>int</code>.<br/>
<code>(1,)</code> is a <code>tuple</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>The list elements should be a tuple of 2 items (one key and another value).</p>
<p><code>(1)</code> becomes <code>1</code> an <code>int</code>. So what python is doing is it is trying to split it into key-value pair. So you get that error.</p>
</div>
<span class="comment-copy"><code>(1)</code> is the integer 1. OrderedDict expects two-element iterables (with a key and a value).</span>
