<p>example txt file :</p>
<pre><code>vrf X.x.X
hello
!
how are you
!
vrf y.y.y.
hi
!
</code></pre>
<p>I want to run through above provided text and print the output starting from matching string "<code>vrf</code>" till matching string "<code>!</code>".
As per above text , I should get output as</p>
<pre><code>vrf X.x.X
hello
vrf y.y.y.
hi
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The solution using <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><strong><em>re.findall()</em></strong></a> function:</p>
<pre><code>import re

with open('lines.txt', 'r') as fh:
    result = re.findall(r'vrf[^!]+(?=!)', fh.read(), re.M)

print(''.join(result))
</code></pre>
<p>The output:</p>
<pre><code>vrf X.x.X
hello
vrf y.y.y.
hi
</code></pre>
</div>
