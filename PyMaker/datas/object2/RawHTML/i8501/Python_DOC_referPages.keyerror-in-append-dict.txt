<blockquote>
<p>How i can set default value for any key in case if this key not exist?</p>
</blockquote>
<p>The <a href="https://docs.python.org/3/library/stdtypes.html#dict.setdefault" rel="nofollow noreferrer">dict.setdefault()</a> method is likely what you want here:</p>
<pre><code>d = {'hits': 10, 'gold': 5}
print( d.setdefault('weapons', 0) )
print( d.setdefault('hits', 10) )
</code></pre>
