<p>You <em>are</em> using <code>elif</code> and <code>else</code> already. It looks like you do understand, mathematically, how to process the withdrawal, as well.</p>
<p>The Python 3 docs explain the syntax of how to declare and call a function
in the section <a href="https://docs.python.org/3/tutorial/controlflow.html#defining-functions" rel="nofollow noreferrer"><strong>4.6. Defining Functions</strong></a>.</p>
<pre><code>def fib(n):    # write Fibonacci series up to n
    """Print a Fibonacci series up to n."""
    a, b = 0, 1
    while a &lt; n:
        print(a, end=' ')
        a, b = b, a+b
    print()

# Now call the function we just defined:
fib(2000)
