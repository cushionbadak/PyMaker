<div class="post-text" itemprop="text">
<p>You want <a href="https://docs.python.org/3/library/itertools.html#itertools.product" rel="nofollow noreferrer">itertools.product</a>.</p>
<pre><code>import itertools

map(lambda t: "-".join(t), 
    itertools.product(
        map(str, range(0,39)), 
        map(str, range(0,10)), 
        map(str, range(0,10)), 
        ['A', 'B', 'C'])
    )
</code></pre>
