<p>You can use the built-in function <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer">all()</a>.</p>
<pre><code>X1 = [i for i in arr if all([j.startswith('Activity') for j in i])]
</code></pre>
<p>I just used a list comprehension that checks whether each of the elements in each sub-array matched the condition you specified.</p>
