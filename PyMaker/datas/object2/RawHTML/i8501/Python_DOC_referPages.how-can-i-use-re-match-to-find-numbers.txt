<p>I'm trying to use python <code>re</code> module:</p>
<pre><code>import re

res = re.match(r"\d+", 'editUserProfile!input.jspa?userId=2089')
print(res)
</code></pre>
<p>I got <code>None</code> type for res, but if I replace the <code>match</code> by <code>findall</code>, I can find the <code>2089</code>.</p>

<div class="post-text" itemprop="text">
<p>The problem is that you're using <a href="https://docs.python.org/3/library/re.html#re.regex.match" rel="nofollow noreferrer"><code>match()</code></a> to search for a substring in a string.</p>
<p>The method <code>match()</code> only works for the whole string. If you want to search for a substring inside a string, you should use <a href="https://docs.python.org/3/library/re.html#re.regex.search" rel="nofollow noreferrer"><code>search()</code></a>.</p>
<p>As stated by <a href="https://stackoverflow.com/users/3890632/khelwood">khelwood</a> in the comments, you should take a look at: <a href="https://docs.python.org/3/library/re.html#search-vs-match" rel="nofollow noreferrer">Search vs Match</a>.</p>
<hr/>
<p><strong>Code:</strong></p>
<pre><code>import re
res = re.search(r"\d+", 'editUserProfile!input.jspa?userId=2089')
print(res.group(0))
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>2089
</code></pre>
<hr/>
<p>Alternatively you can use <code>.split()</code> to isolate the user id.</p>
<p><strong>Code:</strong> </p>
<pre><code>s = 'editUserProfile!input.jspa?userId=2089'
print(s.split('=')[1])
</code></pre>
<p><strong>Output:</strong></p>
<pre><code>2089
</code></pre>
