<p>I am trying to get the index of the first occurrence of a character that occurs in a string <em>after</em> a specified index. For example:</p>
<pre><code>string = 'This + is + a + string'

# The 'i' in 'is' is at the 7th index, find the next occurrence of '+'
string.find_after_index(7, '+')

# Return 10, the index of the next '+' character
&gt;&gt;&gt; 10
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Python is so predicable:</p>
<pre><code>&gt;&gt;&gt; string = 'This + is + a + string'
&gt;&gt;&gt; string.find('+',7)
10
</code></pre>
<p>Checkout <code>help(str.find)</code>:</p>
<pre><code>find(...)
    S.find(sub[, start[, end]]) -&gt; int

    Return the lowest index in S where substring sub is found,
    such that sub is contained within S[start:end].  Optional
    arguments start and end are interpreted as in slice notation.

    Return -1 on failure.
</code></pre>
<p>Also works with <code>str.index</code> except that this will <code>raise ValueError</code> instead of <code>-1</code> when the substring is not found.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>string.find('+', 7)
</code></pre>
<p>Read <a href="https://docs.python.org/3/library/stdtypes.html#string-methods" rel="nofollow noreferrer">the documentation</a>.</p>

<div class="post-text" itemprop="text">
<pre><code>for i in range(index, len(string)):
    if string[i] == char:
         print(i)
</code></pre>
<p>The above code will loop through from the index you provide <code>index</code> to the length of the string <code>len(string)</code>. Then if the index of the string is equal to the character, <code>char</code>, that you are looking for then it will print the index.</p>
<p>You could put this in a function and pass in the, string, index and character and then return i.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use:</p>
<pre><code>start_index = 7
next_index = string.index('+', start_index)
</code></pre>
