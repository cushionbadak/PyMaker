<div class="post-text" itemprop="text">
<p>If</p>
<pre><code>monthStr = months[month-1]     
</code></pre>
<p>had an index error, it means <code>month</code> was either smaller than 1 or bigger than 12. You could check it's in the correct range before this line.</p>
<h1>A few notes</h1>
<ul>
<li><code>eval</code> is dangerous, and not needed in this case. You could use <code>split</code>, <code>map</code> and <code>int</code> to extract the integers.</li>
<li>you could use <code>format</code> with <code>%02d</code> to show the dates, if you want to write it yourself</li>
<li><a href="https://docs.python.org/2/library/datetime.html#strftime-strptime-behavior" rel="nofollow noreferrer"><code>strftime</code> and <code>strptime</code></a> do exactly what you want. They parse a string to datetime and display a <code>datetime</code> in any given format.</li>
<li>calendar already has <a href="https://docs.python.org/3/library/calendar.html#calendar.month_name" rel="nofollow noreferrer"><code>month_name</code></a>. The list has 13 elements though, and the first one is empty.</li>
</ul>
<h1>Modified code :</h1>
<pre><code>def main():
    # get the day month and year
    day, month, year = map(int, input("Please enter day, month, year numbers: ").split(','))

    date1 = '%02d/%02d/%d' % (day, month, year)

    months = ["January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"]
    if month &gt; 0 and month &lt; 13:
        month_str = months[month-1]
        ## alternative :
        # import calendar
        # month_str = calendar.month_name[month]
        date2 = '%s %d, %d' % (month_str, day, year)
        print("The date is", date1, "or", date2+ ".")
    else:
        print("Invalid month")
main()
</code></pre>
<p>As an example :</p>
<pre><code>Please enter day, month, year numbers: 24,12,2017
The date is 24/12/2017 or December 24, 2017.
</code></pre>
<p>but :</p>
<pre><code>Please enter day, month, year numbers: 24,0,2017
Invalid month
</code></pre>
