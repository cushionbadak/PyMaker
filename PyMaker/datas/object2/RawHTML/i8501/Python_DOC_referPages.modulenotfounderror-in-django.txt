<div class="post-text" itemprop="text">
<p>Points to note</p>
<ul>
<li>First of all you are thinking it wrong, <code>from media import *</code> expecting python files or modules to import not files.  </li>
<li>If you want a directory to act as a python module add <code>__init__.py</code> inside the directory  </li>
<li><p>If you want to serve media files you don't need any extra view use this in your <code>urls.py</code> file.    </p>
<p>urlpatterns += patterns('', (r'^media/(?P.*)$',
                        'django.views.static.serve',
                        {'document_root': settings.MEDIA_ROOT}))</p></li>
</ul>
