<p>I'd start by adding the prespacing, then I add in the amount of characters:</p>
<pre><code>def pyramide(size, char):
    resultat = ''
    for i in range(0, size):
        resultat  += (' ' * (size-i)) + (char * int(i*2+1)) + '\n'
    return resultat
</code></pre>

<div class="post-text" itemprop="text">
<p>You can also use <code>format</code> to center:</p>
<pre><code>def pyramide(nbr, char):
    resultat = ''
    for i in range(0, nbr+1):
        chars = char * int(i*2+1)
        resultat += '{:^30}'.format(chars) + '\n'
    return resultat
</code></pre>
