<p>I would create a function to do it.<br/> 
Note that line and column number are counted from 1, not zero.</p>
<pre><code>def find_line_and_column(filename, line_num, column_num):
    with open(filename, 'r') as file:
        for i in range(line_num-1):  # skip over lines before the desired one
            next(file)
        line = next(file)  # read the desired line
        return line.split()[column_num-1]

# sample usage
print(find_line_and_column('zz_ssmv11034tS__T0001TTNATS2012021505HP001.Hdr', 4, 3))
</code></pre>
<p>Alternatively, using the <a href="https://docs.python.org/3/library/itertools.html#itertools.islice" rel="nofollow noreferrer"><code>islice()</code></a> function in <a href="https://docs.python.org/3/library/itertools.html#module-itertools" rel="nofollow noreferrer"><code>itertools</code></a> module would allow it to be done a little bit more succinctly:</p>
<pre><code>import itertools

def find_line_and_column(filename, line_num, column_num):
    with open(filename, 'r') as file:
        line = next(itertools.islice(file, line_num-1, None), None) # get line_num-th line
        return line.split()[column_num-1]
</code></pre>
