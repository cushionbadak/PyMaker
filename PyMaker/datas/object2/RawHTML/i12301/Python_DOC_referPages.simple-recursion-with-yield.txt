<p>If I understand you correctly, this should work:</p>
<pre><code>def get_val(n, last):
    if n &lt; 5:
        for k in range(-1, 1+1, 1):
            yield from get_val(n+1, last+[k])
    else:
        yield last

get_val(0, [])
</code></pre>
<p>Works in Python 3.3+, below that you can replace <code>yield from get_val(n+1, last+[k])</code> with:</p>
<pre><code>for thing in get_val(n+1, last+[k]):
    yield thing
</code></pre>
