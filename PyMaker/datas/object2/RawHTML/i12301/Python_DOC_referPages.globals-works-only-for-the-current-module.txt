<p>By <code>globals()</code> I can see symbols in the current module. When I import another module, I can't see them. So how can I access a global class in another module by its name via <code>globals()</code> or however it's possible?</p>
</div>
<div class="post-text" itemprop="text">
<p>Yes, <code>globals()</code> only ever shows the current module namespace.</p>
<p>Modules are <a href="https://docs.python.org/3/library/stdtypes.html#modules" rel="nofollow">objects with attributes</a>; you can access their <a href="https://docs.python.org/3/library/stdtypes.html#object.__dict__" rel="nofollow"><code>__dict__</code> attribute</a>, or better still, use the <a href="https://docs.python.org/3/library/functions.html#vars" rel="nofollow"><code>vars()</code> function</a>, to treat their namespace as a dictionary:</p>
<pre><code>&gt;&gt;&gt; import string
&gt;&gt;&gt; list(vars(string))
['whitespace', '_string', 'ChainMap', 'ascii_uppercase', 'digits', '__name__', 'punctuation', 'Formatter', 'Template', '__doc__', '__spec__', '_re', 'hexdigits', '__package__', 'ascii_lowercase', '__cached__', 'octdigits', 'ascii_letters', '__builtins__', '_TemplateMetaclass', '__file__', 'capwords', '__loader__', 'printable']
</code></pre>
</div>
