<div class="post-text" itemprop="text">
<p>Since the empty string is a <em>falsy</em> value, you may use <a href="http://www.tutorialspoint.com/python/string_strip.htm" rel="nofollow"><code>.strip()</code></a>:</p>
<pre><code>for line in f:
    if not line.strip():
        ....
</code></pre>
<p>The above ignores lines with only whitespaces.</p>
<p>If you want <em>completely empty</em> lines you may want to use this instead:</p>
<pre><code>if line in ['\r\n', '\n']:
    ...
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Please use a context manager (<code>with</code> statement) to open files:</p>
<pre><code>with open('file.txt') as f:
    print(sum(line.isspace() for line in f)) 
</code></pre>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.isspace" rel="nofollow"><code>line.isspace()</code></a> returns <code>True</code> (== <em>1</em>) if <code>line</code> doesn't have any non-whitespace characters, and <code>False</code> (== <em>0</em>) otherwise. Therefore, <code>sum(line.isspace() for line in f)</code> returns the number of lines that are considered empty.</p>
<hr/>
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow"><code>line.split()</code></a> always returns a list. Both</p>
<pre><code>if line.split() == []:
</code></pre>
<p>and</p>
<pre><code>if not line.split():
</code></pre>
<p>would work.</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>FILE_NAME = 'file.txt'

empty_line_count = 0 

with open(FILE_NAME,'r') as fh:
    for line in fh:
      # The split method will split the word into list. if the line is
      # empty the split will return an empty list. ' == [] ' this will
      # check the list is empty or not.
       if line.split() == []:
            empty_line_count += 1

print('Empty Line Count : ' , empty_line_count)
</code></pre>
