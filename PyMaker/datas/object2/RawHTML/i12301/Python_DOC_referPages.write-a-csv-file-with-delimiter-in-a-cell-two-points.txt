<p>You can <a href="https://docs.python.org/3/library/stdtypes.html?highlight=join#str.join" rel="nofollow">join</a> the elements of <code>ltarg</code> together with a colon:</p>
<pre><code>&gt;&gt;&gt; ltarg2 = list()
&gt;&gt;&gt; for elem in ltarg:
...     ltarg2.append(':'.join(elem))
...     
&gt;&gt;&gt; ltarg2
['1974:3655:1862:279:1936', '238:1833:228:234:1821:225:211:245:1941:315:2035:222:3371:231:224:216:1942']
</code></pre>
<p>Then continue as you were with the new list:</p>
<pre><code>for row in zip(lche,ltarg2):
</code></pre>

<p>You can use <code>join</code></p>
<p>Try:</p>
<pre><code>import csv
lche=['1171837', '1172']
ltarg=[['1974', '3655', '1862', '279', '1936'], ['238', '1833', '228', '234', '1821', '225', '211', '245', '1941', '315', '2035', '222', '3371', '231', '224', '216', '1942']]

data="list.csv"
csv_out = open(data, 'wb')
mywriter = csv.writer(csv_out)
l = [':'.join(x) for x in ltarg]
for row in zip(lche,l):
    mywriter.writerow(row)
csv_out.close()
</code></pre>

<div class="post-text" itemprop="text">
<p>You need to combine (<code>join</code>) the sub-lists in <code>ltarg</code> into single strings while <code>zip</code>ping its contents with the <code>lche</code> list. </p>
<pre><code>import csv

lche = ['1171837', '1172']
ltarg = [['1974', '3655', '1862', '279', '1936'],
         ['238', '1833', '228', '234', '1821', '225', '211', '245', '1941', 
          '315', '2035', '222', '3371', '231', '224', '216', '1942']]

data = "list.csv"
with open(data, 'wb') as csv_out:
    mywriter = csv.writer(csv_out)
    for row in zip(lche, (':'.join(v for v in lt) for lt in ltarg)):
        mywriter.writerow(row)
</code></pre>
<p>Contents of the <code>list.csv</code> file afterwards:</p>
<pre class="lang-none prettyprint-override"><code>1171837,1974:3655:1862:279:1936
1172,238:1833:228:234:1821:225:211:245:1941:315:2035:222:3371:231:224:216:1942
</code></pre>
