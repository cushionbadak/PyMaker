<div class="post-text" itemprop="text">
<p>Python has a builtin package named <code>signal</code></p>
<ul>
<li><a href="https://docs.python.org/2/library/signal.html" rel="nofollow">Python2</a></li>
<li><a href="https://docs.python.org/3/library/signal.html" rel="nofollow">Python3</a></li>
</ul>
<p>So, when you do <code>import signal</code> that is being imported. </p>
<p>If you want to import your <code>signal_class</code> - either rename <code>signal.py</code> or do</p>
<pre><code>from .signal import signal_class
</code></pre>
<p>and inherit <code>SMA</code> from there</p>
