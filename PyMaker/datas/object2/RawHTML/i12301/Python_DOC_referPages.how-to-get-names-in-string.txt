<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/os.path.html#os.path.dirname" rel="nofollow">There is a module for this.</a></p>
<pre><code>&gt;&gt;&gt; import os
&gt;&gt;&gt; s = "/folderA/folderB/folderC/folderD"
&gt;&gt;&gt; os.path.basename(s)
'folderD'
&gt;&gt;&gt; os.path.dirname(s)
'/folderA/folderB/folderC'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You may use <a href="https://docs.python.org/2/library/stdtypes.html#str.rsplit" rel="nofollow"><code>str.rsplit()</code></a>:</p>
<pre><code>&gt;&gt;&gt; editstr = str.rsplit('/folderD', 1)[0]
&gt;&gt;&gt; print(editstr)
/folderA/folderB/folderC
</code></pre>
</div>
