<div class="post-text" itemprop="text">
<p>I have the following list:</p>
<pre><code>a = [[['trial1', 'trial2'], 4], [[], 2]]
</code></pre>
<p>and I want to remove the list that have inside an empty list.So the following result:</p>
<pre><code>c = [[['trial1', 'trial2'], 4]]
</code></pre>
<p>I am using the following code:</p>
<pre><code>c = []
for b in a:
    temp =[x for x in b if x]
    if len(temp)&gt;1:
        c.append(temp)
</code></pre>
<p>It works ok, but it seems not to be a 'good way' of doing this task. Is there a more elegant way?</p>
</div>
<div class="post-text" itemprop="text">
<pre><code>c = [l for l in a if [] not in l]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use a <a href="https://docs.python.org/2/tutorial/datastructures.html#list-comprehensions" rel="nofollow">list comprehension</a>:</p>
<pre><code>c = [x for x in a if [] not in x]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use <code>list comprehension</code> and <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow"><code>all</code></a> function to check whether everything in the list evaluted to the <code>True</code>:</p>
<pre><code>c = [i for i in a if all(i)]

print(c)
[[['trial1', 'trial2'], 4]]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You may use <a href="https://docs.python.org/2/library/functions.html#filter" rel="nofollow"><code>filter</code></a> built-in function.</p>
<pre><code>a = [[['trial1', 'trial2'], 4], [[], 2]]
assert filter(lambda o: [] not in o, a) == [[['trial1', 'trial2'], 4]]
</code></pre>
<p>Since in Python 3 filter is lazy, you may need explicit conversion to list.</p>
<pre><code>assert list(filter(lambda o: [] not in o, a)) == [[['trial1', 'trial2'], 4]]
</code></pre>
</div>
