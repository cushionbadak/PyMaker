<p>I want to extract digits from a short string, base on a condition that the digits is in front of a character (<code>S</code> flag).</p>
<p>example and result:</p>
<pre><code>&gt; string = '10M26S'
&gt; 26

&gt; string = '18S8M10S'
&gt; [18,10] OR 28

&gt; string = '7S29M'
&gt; 7
</code></pre>
<p>I can split the string to a list to get the individual element, </p>
<pre><code>result = [''.join(g) for _, g in groupby('18S8M10S', str.isalpha)]
&gt; ['18', 'S', '8', 'M', '10', 'S'] 
</code></pre>
<p>but how could I just get the <code>18</code> and <code>10</code>? </p>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><code>re.findall</code></a> with the regex <code>r'(\d+)S'</code>. This matches all digits before a capital <code>S</code>.</p>
<pre><code>&gt;&gt;&gt; string = '10M26S'
&gt;&gt;&gt; re.findall(r'(\d+)S',string)
['26']
&gt;&gt;&gt; string = '18S8M10S'
&gt;&gt;&gt; re.findall(r'(\d+)S',string)
['18', '10']
&gt;&gt;&gt; string = '7S29M'
&gt;&gt;&gt; re.findall(r'(\d+)S',string)
['7']
</code></pre>
<p>To get integer output, you can convert them in a list comp or use <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map</code></a></p>
<pre><code>&gt;&gt;&gt; list(map(int,['18', '10']))
[18, 10]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use a <a href="https://docs.python.org/3.5/library/re.html" rel="nofollow">regular expression</a>.</p>
<pre><code>import re
regex = r"(\d+)S"
match = re.search(regex, '10M26S')
print(match.group(1))  # '26'
</code></pre>
