<p>What is the best way to split the following string into 6 float values. The number of decimal points will always be six.</p>
<pre><code>x='  2C   6s         0.043315-143.954801 17.872676 31.277358-18.149649114.553363'
</code></pre>
<p>The output should read:</p>
<pre><code>y=[0.043315, -143.954801, 17.872676, 31.277358, 18.149649, 114.553363]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Assuming that you want to get <code>-18.149649</code> instead of <code>18.149649</code> since that would be consistent I suggest using a <a href="https://docs.python.org/3/library/re.html#module-re" rel="nofollow">regex</a> in combination with the <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow"><code>.findall()</code> function</a> as follows:</p>
<pre><code>import re

regex = '(-?[0-9]{1,}\.[0-9]{6})'

x = '  2C   6s         0.043315-143.954801 17.872676 31.277358-18.149649114.553363'

out = re.findall(regex, x)

print(out)
</code></pre>
<p>Giving:</p>
<pre><code>['0.043315', '-143.954801', '17.872676', '31.277358', '-18.149649', '114.553363']
</code></pre>
<p><em>Update due to comment:</em></p>
<p>You could replace <code>[0-9]</code> with <code>\d</code> which is equivalent since <code>\d</code> matches a digit (number) as shown <a href="https://regex101.com/r/cP3wO0/1" rel="nofollow">here</a>.</p>
</div>
