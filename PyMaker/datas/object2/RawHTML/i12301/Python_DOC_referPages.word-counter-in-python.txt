<div class="post-text" itemprop="text">
<p>You could try the below code:</p>
<pre><code>word = input("Enter word or sentence")
input("Your word/ sentence has"+ str(len(word)) + " letters")
</code></pre>
<p>Here, I'm using <code>str(len(word))</code> instead of just <code>len(word)</code>. Because <code>len(word)</code> returns a <em>number</em> and it's an <code>int</code> object. </p>
<p>And you're doing <code>str_object + int_object</code>, and Python doesn't understand what do you really want to do. </p>
<p>Let's see:</p>
<pre><code>&gt;&gt;&gt; len('foobar')
6
&gt;&gt;&gt; type(len('foobar'))
&lt;class 'int'&gt;
&gt;&gt;&gt; len('foobar') + 'foobar'
Traceback (most recent call last):
  File "&lt;input&gt;", line 1, in &lt;module&gt;
TypeError: unsupported operand type(s) for +: 'int' and 'str'
&gt;&gt;&gt; 
</code></pre>
<p>So you have to convert the <code>int_object</code> (which returned by <code>len(word)</code>) to a <code>str</code> object use <a href="https://docs.python.org/3/library/stdtypes.html#str" rel="nofollow"><code>str()</code></a> function.</p>
<p>For example:</p>
<pre><code>&gt;&gt;&gt; str(len('foobar'))
'6'
&gt;&gt;&gt; type(str(len('foobar')))
&lt;class 'str'&gt;
&gt;&gt;&gt; str(len('foobar')) + 'foobar'
'6foobar'
</code></pre>
<p>You can also use <a href="https://docs.python.org/3/library/stdtypes.html#str.format" rel="nofollow"><code>str.format()</code></a> instead of two <code>+</code>. Which can auto convert all objects to <code>str</code>, also it's more readable than your code.</p>
<p>So just use:</p>
<pre><code>word = input("Enter word or sentence")
input("Your word/ sentence has {} letters".format(len(word)))
</code></pre>
</div>
