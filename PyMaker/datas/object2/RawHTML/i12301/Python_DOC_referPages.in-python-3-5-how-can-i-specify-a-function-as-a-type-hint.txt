<p>What is the appropriate type hint to specify that a variable should be a function (the equivalent of a delegate, <code>Func&lt;T&gt;</code> or <code>Action</code> in C#)?</p>

<p>According to the <a href="https://docs.python.org/3/library/typing.html" rel="noreferrer">docs</a>, you can use Callable, imported from typing.</p>
<p>You can view more details in <a href="https://www.python.org/dev/peps/pep-0484/#callable" rel="noreferrer">PEP 0484</a>.</p>
