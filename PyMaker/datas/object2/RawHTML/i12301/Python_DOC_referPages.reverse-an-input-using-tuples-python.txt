<div class="post-text" itemprop="text">
<p>It would be stored in a <code>list</code>, not a <code>tuple</code>. Here's how you do it in 2 lines:</p>
<pre><code>my_input = input('Your phrase: ')
print(*reversed(my_input.split()))  # Python 3
</code></pre>
<p>or</p>
<pre><code>print ' '.join(reversed(a.split()))  # Python 2
</code></pre>
<p>So the function would be:</p>
<pre><code>func = lambda x: reversed(x.split())
</code></pre>
<p>which will return a list of reversed words from the phrase if called as follows: </p>
<pre><code>arg = input('Enter a phrase :')
splitted = func(arg)

print(*splitted)  # or in Python 2: print ' '.join(splitted)
</code></pre>

<p>you can use the <a href="https://docs.python.org/3/library/functions.html#reversed" rel="nofollow">reversed</a> built-in</p>
<pre><code>def reverser():
    return ' '.join(reversed(input('Please input a phrase:').split()))
</code></pre>
<p>the input string is split on whitespace, then passed to reversed, then joined again</p>
<p>For example use it as</p>
<pre><code>&gt;&gt;&gt; print(reverser())
Please input a phrase:hello world how are you
you are how world hello
</code></pre>

<blockquote>
<p>Python programming language provides a neat syntax to reverse a
  tuple.The syntax is <code>var[::-1]</code> which returns a reverse copy of the var.</p>
<p>Example:</p>
</blockquote>
<pre><code>def reverser():
    inputphrase = tuple(x.strip() for x in raw_input("Please input a phrase: ").split(' '))
    return inputphrase[::-1]

print reverser()
</code></pre>
<blockquote>
<p>Output: </p>
</blockquote>
<pre><code>Please input a phrase: Hello World 
('World', 'Hello')
</code></pre>
<blockquote>
<p>Or you can do it this way.</p>
</blockquote>
<pre><code>def reverser():
            inputphrase = tuple(x.strip() for x in raw_input("Please input a phrase: ").split(' '))
            return tuple(reversed(inputphrase))

print reverser()
</code></pre>
<blockquote>
<p>Output: </p>
</blockquote>
<pre><code>Please input a phrase: Hello World. 
('World.', 'Hello')
</code></pre>

<span class="comment-copy">Tuples are immutable. You cannot change them. Any manipulations must be stored in a separate variable (either as a tuple or a list) and used later. You can change it from a list to a tuple though: <code>my_tuple = tuple(reversed(x.split()))</code>. By the way, I implemented the function for you too. You can add <code>tuple()</code> to it, so that it returns a tuple.</span>
<span class="comment-copy">If you mean how you can used <code>reversed</code> on tuples, it's the same. However, if you mean how you can use <code>split()</code> on a tuple, that you cannot. Split can only be implemented on a string. You can do this to split everything in your tuple though: <code>var = [item.split() for item in my_tuple]</code></span>
