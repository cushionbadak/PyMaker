<div class="post-text" itemprop="text">
<p>You want to use <a href="https://docs.python.org/3/library/configparser.html" rel="nofollow"><code>configparser</code></a> for this.</p>
<pre><code>&gt;&gt;&gt; import configparser
&gt;&gt;&gt; config = configparser.ConfigParser()
&gt;&gt;&gt; config.read(r'somesettings.ini')
&gt;&gt;&gt; config['Settings']['Minimum Free Space']
32000
</code></pre>
<p>You can access any of your settings in the <code>[Settings]</code> section this way. </p>
<p>At this point <code>32000</code> is a string. You will need to convert it to an <code>int</code>, if you expect an <code>int</code> later in the application:</p>
<pre><code>&gt;&gt;&gt; int(config['Settings']['Minimum Free Space'])
</code></pre>
