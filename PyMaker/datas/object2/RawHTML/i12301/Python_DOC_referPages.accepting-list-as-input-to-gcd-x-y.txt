<p>I am trying to write a function that accepts list as an argument to my <code>gcd</code> function. Would something like this work?</p>
<pre><code>def find_my_gcd(a_list):
    def gcd (x,y):
        while y!=0:
             x,y=y,x%y
        return x
</code></pre>
<p>Now,how do I make my <code>gcd</code> function step through a list,i.e <code>a_list=[12,24,18,6]</code>, iterate through <code>gcd(12,24)</code>, <code>gcd(12,18)</code>,<code>gcd(12,6)</code> and returns a single value.</p>

<div class="post-text" itemprop="text">
<p>you have to keep the current value in a variable, and use that with the next value in the list, something like this</p>
<pre><code>def find_my_gcd(a_list):
    def gcd (x,y):
        while y!=0:
            x,y=y,x%y
        return x
    acum = a_list[0]
    for e in a_list[1:]:
        acum = gcd(acum,e)
    return acum
</code></pre>
<p>you can also use <a href="https://docs.python.org/3/library/functools.html#functools.reduce" rel="nofollow">reduce</a> to keep the acumulated for you</p>
<pre><code>def find_my_gcd(a_list):
    def gcd (x,y):
        while y!=0:
            x,y=y,x%y
        return x
    return reduce(gcd,a_list)
</code></pre>
<p>also you can find gcd in the <a href="https://docs.python.org/3/library/math.html#math.gcd" rel="nofollow">math.gcd</a> library in python 3.5+ and in previous versions is in <a href="https://docs.python.org/3/library/fractions.html#fractions.gcd" rel="nofollow">fractions.gcd</a> library so the your function can be reduced to</p>
<pre><code>from functools import reduce
from math import gcd # from fractions import gcd in 3.4 or previous

def find_my_gcd(a_list):
    return reduce(gcd,a_list)
</code></pre>
