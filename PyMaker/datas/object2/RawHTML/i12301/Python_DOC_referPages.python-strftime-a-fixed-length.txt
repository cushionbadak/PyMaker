<p>I am using the following date format:</p>
<pre><code>d.strftime("%A, %d %B %Y %H:%m")
</code></pre>
<p>and since the length of the weekday (<code>%A</code>) changes,
I would like to always print the weekday with 
10 characters, pad spaces to the left, and align it right.</p>
<p>Something like </p>
<pre><code>d.strftime("10%A, %d %B %Y %H:%m")
</code></pre>
<p>What is the simplest way?</p>
</div>

<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#str.rjust" rel="nofollow noreferrer"><code>str.rjust(10)</code></a> does exactly that:</p>
<pre><code>s = d.strftime('%A').rjust(10) + d.strftime(', %d %B %Y %H:%M')
</code></pre>
<p>It is likely that you want <a href="http://strftime.org/" rel="nofollow noreferrer">%M (minutes), not %m (months)</a> in your format string as <a href="https://stackoverflow.com/questions/35201605/python-strftime-a-fixed-length/35226085#comment67236284_35226085">@Ahmad pointed out</a>.</p>
<p>In Python 3.6+, to get a more concise version, one can abuse nested f-strings:</p>
<pre><code>&gt;&gt;&gt; f"{f'{d:%A}':&gt;10}, {d:%d %B %Y %H:%M}"
'    Friday, 15 December 2017 21:31'
</code></pre>
<hr/>
<p>Prefer standard time formats such as rfc 3339:</p>
<pre><code>&gt;&gt;&gt; from datetime import datetime
&gt;&gt;&gt; datetime.utcnow().isoformat() + 'Z'
'2016-02-05T14:00:43.089828Z'
</code></pre>
<p>Or rfc 2822:</p>
<pre><code>&gt;&gt;&gt; from email.utils import formatdate
&gt;&gt;&gt; formatdate(usegmt=True)                                          
'Fri, 05 Feb 2016 14:00:51 GMT'
</code></pre>
<p>instead.</p>
</div>
<div class="post-text" itemprop="text">
<p>This is roughly equivalent to <a href="https://stackoverflow.com/a/35226085/241211">jfs's answer,</a> but you can <a href="https://pyformat.info/#string_pad_align" rel="nofollow noreferrer">use <code>.format()</code></a> to make it slightly shorter:</p>
<pre><code>s = '{:&gt;10}, {:%d %B %Y %H:%m}'.format(d.strftime('%A'), d)
</code></pre>
<p>or if you're putting more than just the date into the string:</p>
<pre><code>args = {
    'weekday' = d.strftime('%A'),
    'date' = d,
    'foo' = some_other_stuff(),
    'bar' = 17.5422,
}
s = '{weekday:&gt;10}, {date:%d %B %Y %H:%m} {foo} {bar:&gt;3.2f}'.format(**args)
</code></pre>
