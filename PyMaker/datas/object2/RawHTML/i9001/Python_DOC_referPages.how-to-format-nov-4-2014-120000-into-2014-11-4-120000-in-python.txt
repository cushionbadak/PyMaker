<div class="post-text" itemprop="text">
<p>First of all you are using <code>strptime</code> wrong <a href="https://docs.python.org/3.6/library/datetime.html#datetime.datetime.strptime" rel="nofollow noreferrer">https://docs.python.org/3.6/library/datetime.html#datetime.datetime.strptime</a></p>
<p>After you converted <code>datetext</code> to datetime with <code>strptime</code> you need to convert it back to string using <code>strftime</code> <a href="https://docs.python.org/3.6/library/datetime.html#datetime.datetime.strftime" rel="nofollow noreferrer">https://docs.python.org/3.6/library/datetime.html#datetime.datetime.strftime</a></p>
<p>Using @FeebleOldMan partial answer</p>
<pre><code>from datetime import datetime
datetext="Nov 1 2004 12:00:00"
datev = datetime.strptime(datetext,"%b %d %Y %H:%M:%S")
new_datetext = datev.strftime('%Y-%m-%d %H:%M:%S')
print(new_datetext)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Put the correct formatting into strptime to mirror your input:</p>
<pre><code>datev = datetime.strptime(datetext,"%b %d %Y %H:%M:%S")
</code></pre>
<p>See <a href="https://docs.python.org/2/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer">here</a> for the formatting directives. (<a href="https://docs.python.org/3/library/datetime.html#strftime-and-strptime-behavior" rel="nofollow noreferrer">Here</a> for Python 3.)</p>
<hr/>
<p>If you <code>print</code> the subsequent <code>datev</code> <code>datetime</code> object you created, the default <code>str</code> formatting of the <code>datetime</code> object will be as you requested (2014-11-4 12:00:00). If you want to modify how the printed output looks, use <code>strftime</code> as shown in Sardorbek's answer and again, use the formatting directives you desire.</p>
