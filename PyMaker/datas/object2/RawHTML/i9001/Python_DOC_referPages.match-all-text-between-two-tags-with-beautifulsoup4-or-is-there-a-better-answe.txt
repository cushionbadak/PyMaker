<p>The html I'm trying to match is something like this...</p>
<pre><code>...
&lt;!-- Start Body --&gt;
Paragraph 1&lt;br&gt;&lt;br&gt;
Paragraph 2&lt;br&gt;&lt;br&gt;
&lt;!-- End Body --&gt;
...
</code></pre>
<p>The site I'm scraping is old, so from page to page it can change from br's to paragraph tags, etc. It's a mess, the only sure thing is that all the content I need is between the start body and end body comment tags.</p>
<p>What's the best way to do this? Should I even be using beautifulsoup or is that overkill?</p>

<div class="post-text" itemprop="text">
<p>I'd just grab the text directly, without BeautifulSoup:</p>
<pre><code>start = "&lt;!-- Start Body --&gt;"
end = "&lt;!-- End Body --&gt;"
html = "..." # Page HTML

_s = html.find(start)
_e = html.find(end)
if _s != -1 and _e != -1:
  found = html[_s + len(start):_e]
</code></pre>
<p>There might be an off-by one in the last line, but that should get you started. You'll also need to decide what to do if the tags appear multiple times.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/stdtypes.html#str.split" rel="nofollow noreferrer"><code>str.split</code></a>:</p>
<pre><code>&gt;&gt;&gt; html = '''
... ...
... &lt;!-- Start Body --&gt;
... Paragraph 1&lt;br&gt;&lt;br&gt;
... Paragraph 2&lt;br&gt;&lt;br&gt;
... &lt;!-- End Body --&gt;
... ...
... '''
&gt;&gt;&gt; chunk = html.split('&lt;!-- Start Body --&gt;')[1].split('&lt;!-- End Body --&gt;')[0]
&gt;&gt;&gt; print(chunk)

Paragraph 1&lt;br&gt;&lt;br&gt;
Paragraph 2&lt;br&gt;&lt;br&gt;
</code></pre>
<ul>
<li><code>html.split('&lt;!-- Start Body --&gt;')[1]</code> - get part after the <code>Start Body</code> comment</li>
<li><code>.split('&lt;!-- End Body --&gt;')[0]</code> - get part before the <code>End Body</code> comment</li>
</ul>
<hr/>
<p>Alternative using <a href="https://docs.python.org/3/library/re.html#re.findall" rel="nofollow noreferrer"><code>re.findall</code></a>:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; re.findall('&lt;!-- Start Body --&gt;(.+?)&lt;!-- End Body --&gt;', html, flags=re.S)
['\nParagraph 1&lt;br&gt;&lt;br&gt;\nParagraph 2&lt;br&gt;&lt;br&gt;\n']
</code></pre>
</div>
