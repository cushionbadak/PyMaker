<p>What's the most efficient way to reverse the string and swap the case? For example, if my input string is:</p>
<pre><code>input_string = "Hello Python World"
</code></pre>
<p>I want the content of the string to be:</p>
<pre><code>output_string = "DLROw NOHTYp OLLEh"
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You may swap the case using <a href="https://docs.python.org/3/library/stdtypes.html#str.swapcase" rel="nofollow noreferrer"><strong><code>str.swapcase()</code></strong></a> on the reversed string (or vice-versa) as:</p>
<pre><code>input_string = "Hello Python World"

output_string = input_string[::-1].swapcase()
#        to reverse the string ^                 

# OR, input_string.swapcase()[::-1]
</code></pre>
<p>where <code>output_string</code> will hold:</p>
<pre><code>&gt;&gt;&gt; output_string
'DLROw NOHTYp OLLEh'
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Something like</p>
<pre><code>''.join([c.lower() if c.isupper() else c.upper() for c in my_string][::-1])
</code></pre>
