<div class="post-text" itemprop="text">
<p>I have a a list of ['A', 'B', 'C', 'D'] and [1, 2, 3, 4], both list will always have the same number of items. I need to multiply each string by its number, so the final product I am looking for is:</p>
<p>['A', 'B', 'B', 'C', 'C', 'C', 'D', 'D', 'D', 'D']. </p>
<p>Any help would be appreciated, thanks. </p>
</div>
<div class="post-text" itemprop="text">
<p>Nested list comprehension works too:</p>
<pre><code>&gt;&gt;&gt; l1 = ['A', 'B', 'C', 'D'] 
&gt;&gt;&gt; l2 = [1, 2, 3, 4]
&gt;&gt;&gt; [c for c, i in zip(l1, l2) for _ in range(i)]
['A', 'B', 'B', 'C', 'C', 'C', 'D', 'D', 'D', 'D']
</code></pre>
<p>In above <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> returns <code>(char, count)</code> tuples:</p>
<pre><code>&gt;&gt;&gt; t = list(zip(l1, l2))
&gt;&gt;&gt; t
[('A', 1), ('B', 2), ('C', 3), ('D', 4)]
</code></pre>
<p>Then for every tuple the second <code>for</code> loop is executed <code>count</code> times to add the character to the result:</p>
<pre><code>&gt;&gt;&gt; [char for char, count in t for _ in range(count)]
['A', 'B', 'B', 'C', 'C', 'C', 'D', 'D', 'D', 'D']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>I would use <code>itertools.repeat</code> for a nice, efficient implementation:</p>
<pre><code>&gt;&gt;&gt; letters = ['A', 'B', 'C', 'D']
&gt;&gt;&gt; numbers = [1, 2, 3, 4]
&gt;&gt;&gt; import itertools
&gt;&gt;&gt; result = []
&gt;&gt;&gt; for letter, number in zip(letters, numbers):
...     result.extend(itertools.repeat(letter, number))
...
&gt;&gt;&gt; result
['A', 'B', 'B', 'C', 'C', 'C', 'D', 'D', 'D', 'D']
&gt;&gt;&gt;
</code></pre>
<p>I also think it is quite readable.</p>
</div>
<div class="post-text" itemprop="text">
<p>The code is pretty straight forward, see inline comments</p>
<pre><code>l1 = ['A', 'B', 'C', 'D'] 
l2 = [1, 2, 3, 4]
res = []
for i, x in enumerate(l1): # by enumerating you get both the item and its index
    res += x * l2[i] # add the next item to the result list
print res
</code></pre>
<p><strong>OUTPUT</strong> </p>
<pre><code>['A', 'B', 'B', 'C', 'C', 'C', 'D', 'D', 'D', 'D']
</code></pre>

<div class="post-text" itemprop="text">
<p>You use <code>zip()</code> to do it like this way:</p>
<pre><code>a = ['A', 'B', 'C', 'D']
b = [1, 2, 3, 4]

final = []
for k,v in zip(a,b):
    final += [k for _ in range(v)]

print(final)
</code></pre>
<p>Output:</p>
<pre><code>&gt;&gt;&gt; ['A', 'B', 'B', 'C', 'C', 'C', 'D', 'D', 'D', 'D']
</code></pre>
<p>Or you can do it, too, using <code>zip()</code> and <code>list comprehension</code>:</p>
<pre><code>a = ['A', 'B', 'C', 'D']
b = [1, 2, 3, 4]
final = [k for k,v in zip(a,b) for _ in range(v)]
print(final)
</code></pre>
<p>Output:</p>
<pre><code>&gt;&gt;&gt; ['A', 'B', 'B', 'C', 'C', 'C', 'D', 'D', 'D', 'D']
</code></pre>
