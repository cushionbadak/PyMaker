<div class="post-text" itemprop="text">
<p><code>Creation_date</code> is just a regex. </p>
<p>You need to match your input string before you can call <code>group(0)</code></p>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/re.html#regular-expression-objects" rel="nofollow noreferrer">Regular expression object</a> (the result of <a href="https://docs.python.org/3/library/re.html#re.compile" rel="nofollow noreferrer"><code>re.compile</code></a>) does not have <code>group</code> method:</p>
<pre><code>&gt;&gt;&gt; pattern = re.compile('CreationDate="([0-9]+)"')
&gt;&gt;&gt; pattern.group
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
AttributeError: '_sre.SRE_Pattern' object has no attribute 'group'
</code></pre>
<p>To get a <a href="https://docs.python.org/3/library/re.html#match-objects" rel="nofollow noreferrer">match object</a> (which has a <a href="https://docs.python.org/3/library/re.html#re.match.group" rel="nofollow noreferrer"><code>group</code></a> method), you need to match the pattern against the string (<code>line</code>) using <a href="https://docs.python.org/3/library/re.html#re.regex.search" rel="nofollow noreferrer"><code>regex.search</code> method</a> (or <a href="https://docs.python.org/3/library/re.html#re.regex.match" rel="nofollow noreferrer"><code>regex.match</code> method</a> depending on your need):</p>
<pre><code>&gt;&gt;&gt; pattern.search('CreationDate="2013"')
&lt;_sre.SRE_Match object at 0x7fac5c64e8a0&gt;
&gt;&gt;&gt; pattern.search('CreationDate="2013"').group(1)  # returns a string
'2013'
</code></pre>
<hr/>
<pre><code>Creation_date = re.compile('CreationDate="([0-9]+)"')

def mapper(self, _, line):
    date_only = Creation_date.search(line), group(1)
    if int(date_only) &gt; 2013:
       yield None, 1
</code></pre>
<p>NOTE: modifed the regular express to capture the numeric part as a group. and convert the matched string to int (comparing string with the number 2013 has no meaning, or raise exception depending on Python version)</p>
