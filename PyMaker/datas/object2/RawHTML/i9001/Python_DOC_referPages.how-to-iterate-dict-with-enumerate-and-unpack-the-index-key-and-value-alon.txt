<p>How to iterate <code>dict</code> with <code>enumerate</code> such that I could unpack the index, key and value at the time of iteration?</p>
<p>Something like:</p>
<pre><code>for i, (k, v) in enumerate(mydict):
    # some stuff
</code></pre>
<p>I want to iterate through the keys and values in a dictionary called <code>mydict</code> and count them, so I know when I'm on the last one.</p>
</div>
<div class="post-text" itemprop="text">
<p>Instead of using <code>mydict</code>, you should be using <a href="https://docs.python.org/3/library/stdtypes.html#dict.items" rel="noreferrer"><code>mydict.items()</code></a> with <a href="https://docs.python.org/3.3/library/functions.html#enumerate" rel="noreferrer"><code>enumerate</code></a> as:</p>
<pre><code>for i, (k, v) in enumerate(mydict.items()):
    # your stuff
</code></pre>
<p>Sample example:</p>
<pre><code>mydict = {1: 'a', 2: 'b'}
for i, (k, v) in enumerate(mydict.items()):
    print("index: {}, key: {}, value: {}".format(i, k, v))

# which will print:
# -----------------
# index: 0, key: 1, value: a
# index: 1, key: 2, value: b
</code></pre>
<p><strong>Explanations:</strong></p>
<ul>
<li><code>enumerate</code> returns an iterator object which contains tuples in the format: <code>[(index, list_element), ...]</code></li>
<li><code>dict.items()</code> returns an iterator object <em>(in Python 3.x. It returns <code>list</code> in Python 2.7)</em> in the format: <code>[(key, value), ...]</code></li>
<li>On combining together, <code>enumerate(dict.items())</code> will return an iterator object containing tuples in the format: <code>[(index, (key, value)), ...]</code></li>
</ul>
