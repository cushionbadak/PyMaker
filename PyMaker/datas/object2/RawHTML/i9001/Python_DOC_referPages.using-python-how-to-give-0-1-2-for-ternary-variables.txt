<div class="post-text" itemprop="text">
<p>Python provides the <a href="https://docs.python.org/3/library/enum.html" rel="nofollow noreferrer">Enum</a> class.  Why you want to use Enums is covered <a href="https://stackoverflow.com/a/1970200/7311767">here</a>.  In your case they would look something like:</p>
<p><strong>Code:</strong></p>
<pre><code>from enum import Enum
PH = Enum('PH', 'Acidic Neutral Alkaline')
Concentration = Enum('Concentration', 'Low Medium High')
</code></pre>
<p><strong>Demo Code:</strong></p>
<pre><code>print(PH['Acidic'].value)
print(Concentration['Medium'].value)
</code></pre>
<p><strong>Produces:</strong></p>
<pre><code>1
2
</code></pre>
<p><strong>Demo Code2:</strong></p>
<pre><code>for i in range(1, 4):
    ph = PH(i)
    concentration = Concentration(i)
    print(ph, ph.name, ph.value)
    print(concentration, concentration.name, concentration.value)
</code></pre>
<p><strong>Produces:</strong></p>
<pre><code>PH.Acidic Acidic 1
Concentration.Low Low 1
PH.Neutral Neutral 2
Concentration.Medium Medium 2
PH.Alkaline Alkaline 3
Concentration.High High 3
</code></pre>
