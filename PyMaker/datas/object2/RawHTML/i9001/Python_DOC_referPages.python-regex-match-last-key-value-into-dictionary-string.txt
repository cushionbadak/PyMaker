<p>I read this dictionary from a file as a string:</p>
<pre><code>string = '{"key1":"value1", "key2":"value2"}'
</code></pre>
<p>I don't know which values key1 and key2 can assume, I need to access the value of the last pair written on the string (in this case 'value2'). So converting it into a python dictionary is out of the question since it would probably change the order.</p>
<p>I've tried something like:</p>
<pre><code>value = re.search(r'"(.*?)":"(.*?)"', string)
</code></pre>
<p>But I don't know how to access the .group(2) of the last match.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can do this :</p>
<pre><code>import re
output = re.findall(".*\"[a-zA-Z0-9]+\"[\w]*:[\w]*\"(.+)\"[\w]*}", string)
</code></pre>
<p>Output :</p>
<pre><code>&gt;&gt;&gt; re.findall(".*\"[a-zA-Z0-9]+\"[\w]*:[\w]*\"(.+)\"[\w]*}", string)
['value2']
</code></pre>
