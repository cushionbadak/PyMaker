<div class="post-text" itemprop="text">
<p>You can iterate through the leaf elements like this:</p>
<p><strong>Python3 Code</strong></p>
<pre><code>d = {'edge': {'edge1': {'interface': {'interface1': {'ip': 10, 'name': 'uplink-to-x'}}, 'name': 'x'}, 'edge2': {'interface': {'interface1': {'ip': 20, 'name': 'uplink-to-y'}}, 'name': 'y'}}}
def nested_print(d):
    for key in d:
        if isinstance(d[key],dict):
            nested_print(d[key])
        else:
            print(key,d[key])
nested_print(d)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>ip 20
name uplink-to-y
name y
ip 10
name uplink-to-x
name x
</code></pre>
<p><strong>N.B.:</strong> I did not find any element as <code>edge["edge1"]["name"]</code> in your given dictionary</p>
