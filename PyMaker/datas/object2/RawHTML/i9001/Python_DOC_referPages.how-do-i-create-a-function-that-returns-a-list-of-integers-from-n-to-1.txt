<p>How do I write a function called <code>countdown</code> that counts down starting from <code>n</code> and goes until 1 ? The function should return a <code>list</code>, the contents of which should be integers going from <code>n</code> down to 1.</p>
<pre><code>def countdown(n):
    if n &gt;= 1:
        countdown(n-1)
    print(n)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Since you want to return a <code>list</code>, you need to create that list in the function.</p>
<pre><code>def countdown(n):
    return list(range(n, 0, -1))
</code></pre>
<p><a href="https://docs.python.org/3/library/functions.html#func-range" rel="nofollow noreferrer"><code>range</code></a> creates your sequence from n to 0 (non-inclusive, which means it'll stop at 1), with a step of -1 each time.</p>
<p><a href="https://docs.python.org/3/library/functions.html#func-list" rel="nofollow noreferrer"><code>list</code></a> then converts the sequence into the list that you want returned.</p>
<p>This also means that you don't actually have to create a specific function for a countdown list. You can just directly call <code>list(range(n, 0, -1))</code>. </p>
</div>
<div class="post-text" itemprop="text">
<p>Using recursion:</p>
<pre><code>def countdown(n):
    if n &lt; 1:
        return []
    return [n] + countdown(n-1)
</code></pre>
<p>This approach provides the "base case" and creation of a list of integers once the base is reached.</p>
