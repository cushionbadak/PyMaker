<p>I have the following string I want to convert into a datetime object in Python 3:</p>
<pre><code>"05/Feb/2016"
</code></pre>
<p>Iv'e used the following approach before: </p>
<pre><code>datetime.date(datetime.strptime("05/02/2016", '%d/%m/%Y'))
</code></pre>
<p>Now the format has changed, and I need to know what to change <code>'%d/%m/%Y'</code> into (if possible). Thanks!</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="http://strftime.org/" rel="nofollow noreferrer">http://strftime.org/</a> for reference on Python's strftime directives</p>
<p>In your case, this should do:</p>
<pre><code>datetime.date(datetime.strptime("05/Feb/2016", '%d/%b/%Y'))
</code></pre>
<p>Notice <code>%b</code> is used to denote month in the 3 letter style (eg Feb)</p>
</div>
<div class="post-text" itemprop="text">
<p>To decode "Feb" if that is the correct local abbreviated name for the month, use <code>%b</code> in place of <code>%d</code>. See the <code>datetime</code> documentation <a href="https://docs.python.org/3/library/datetime.html" rel="nofollow noreferrer">https://docs.python.org/3/library/datetime.html</a></p>
<p>Alternatively investigate <code>dateparser</code> <a href="https://dateparser.readthedocs.io/en/latest/" rel="nofollow noreferrer">https://dateparser.readthedocs.io/en/latest/</a>, or even roll your own decoder.</p>
