<div class="post-text" itemprop="text">
<p>Try:</p>
<pre><code>c = conn.cursor()
c.execute("INSERT INTO friendList VALUES (?)", (nombre,))
conn.commit()  &lt;-- important, needed to save transaction
</code></pre>
<p>Since <code>nombre</code> is a variable, you can bind it this way. You <em>could</em> insert it directly by concatenating, but this opens you up to SQL Injection.</p>
