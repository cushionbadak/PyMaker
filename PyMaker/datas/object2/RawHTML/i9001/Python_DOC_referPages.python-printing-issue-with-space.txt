<p>PS: The <code>print</code> statement has been replaced with a <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>print()</code></a> function since Python 3.0. </p>
<pre><code>Old: print x,           # Trailing comma suppresses newline
New: print(x, end=" ")  # Appends a space instead of a newline
</code></pre>
<p>Refer to <a href="https://docs.python.org/3/whatsnew/3.0.html#print-is-a-function" rel="nofollow noreferrer">Print Is A Function</a> for the detailed descriptions.</p>
</div>
<div class="post-text" itemprop="text">
<p>If you use <code>print</code> with multiple arguments, separated by <code>,</code>, a single space <code>' '</code> is inserted as a separator in between each of those.</p>
<p>When using Python 3's <a href="https://docs.python.org/3/library/functions.html#print" rel="nofollow noreferrer"><code>print</code> function</a>, you can specify the <code>sep</code> parameter; default is <code>' '</code>.</p>
<pre><code>&gt;&gt;&gt; from __future__ import print_function  # when in Python 2
&gt;&gt;&gt; print("hello=", 10)
hello= 10
&gt;&gt;&gt; print("hello=", 10, sep="")
hello=10
&gt;&gt;&gt; print("hello=", 10, sep="###")
hello=###10
</code></pre>
<p>For Python 2's <a href="https://docs.python.org/2/reference/simple_stmts.html#print" rel="nofollow noreferrer"><code>print</code> statement</a>, there is to the best of my knowledge no such option.</p>
</div>
<div class="post-text" itemprop="text">
<p>You may also consider using the Python 3 compatible <code>print()</code> function:</p>
<p>This function can be used after a <code>__future__</code> directive:</p>
<pre><code>from __future__ import print_function

print("hello=", 10, sep='')
</code></pre>
<p>Output:</p>
<pre><code>hello=10
</code></pre>
<p>The <code>print()</code> function as a <em>sep</em> keyword argument which allows you to replace the space separator by an empty string.</p>
<p>Here is the online help:</p>
<blockquote>
<p>Help on built-in function print in module builtins:</p>
<p>print(...)
      print(value, ..., sep=' ', end='\n', file=sys.stdout, flush=False)</p>
<pre><code>Prints the values to a stream, or to sys.stdout by default.
Optional keyword arguments:
file:  a file-like object (stream); defaults to the current sys.stdout.
sep:   string inserted between values, default a space.
end:   string appended after the last value, default a newline.
flush: whether to forcibly flush the stream.
</code></pre>
</blockquote>
