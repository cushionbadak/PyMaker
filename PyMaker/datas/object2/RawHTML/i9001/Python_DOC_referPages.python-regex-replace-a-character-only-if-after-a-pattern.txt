<div class="post-text" itemprop="text">
<p>I have a lot of strings with this pattern <code>[INTEGER]-[URL-SLUG]</code>. Examples:</p>
<pre><code>234234-url-slug-1
2-urlslug2
301-url-slug4
</code></pre>
<p>I need to replace the '-' after the integer with a '/' but leave the other '-' intact. So final strings would be:</p>
<pre><code>234234/url-slug-1
2/urlslug2
301/url-slug4
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/re.html#re.sub" rel="nofollow noreferrer"><strong><em>re.sub</em></strong></a> function has the optional argument <em>count</em> which is the maximum number of pattern occurrences to be replaced:</p>
<pre><code>s = '234234-url-slug-1'
result = re.sub(r'^(\d+)-', r'\1/', s, 1)

print(result)
</code></pre>
<p>The output:</p>
<pre><code>234234/url-slug-1
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>This should work using <a href="https://docs.python.org/2/library/re.html#re.sub" rel="nofollow noreferrer"><code>re.sub</code></a>:</p>
<pre><code>&gt; import re

&gt; s = '234234-url-slug-1'
&gt; re.sub(r'(\d)\-', r'\1/', s)
'234234/url-slug-1'
</code></pre>
<p><code>\1</code> in the replacement string refers to the first captured group in the matching pattern: <code>'(\d)'</code>.</p>
</div>
