<div class="post-text" itemprop="text">
<p>By adding a <code>+</code> to <code>'[' + delimiters + ']'</code>, you should be able to get what you want.</p>
<pre><code>def SplitStringIntoWords(inputString, delimiters=' \t,'):
    words = re.split('[' + delimiters + ']+', inputString.strip())
    return len(words), words
</code></pre>

<span class="comment-copy">Append a <code>+</code> sign to <code>'[' + delimiters + ']'</code> =&gt; <code>'[' + delimiters + ']+'</code></span>
<span class="comment-copy">You could use <code>[delimiters]+</code> to match 1 or more along with <a href="https://docs.python.org/3/library/re.html#re.split" rel="nofollow noreferrer"><code>maxsplit</code> for <code>re.split()</code></a>.</span>
