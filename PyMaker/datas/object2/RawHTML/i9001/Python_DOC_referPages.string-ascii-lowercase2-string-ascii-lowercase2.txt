<p>Maybe it's best explained when the steps are done seperatly:</p>
<pre><code>&gt;&gt;&gt; import string
&gt;&gt;&gt; string.ascii_lowercase
'abcdefghijklmnopqrstuvwxyz'

&gt;&gt;&gt; string.ascii_lowercase[:2]  # Take the first two items from the string
'ab'

&gt;&gt;&gt; string.ascii_lowercase[2:]  # Take everything starting by the third item
'cdefghijklmnopqrstuvwxyz'

&gt;&gt;&gt; string.ascii_lowercase[2:] + string.ascii_lowercase[:2]  # concatenate them
'cdefghijklmnopqrstuvwxyzab'
</code></pre>
<p>This is explained in the <a href="https://docs.python.org/3/tutorial/introduction.html#strings" rel="nofollow noreferrer">official Python tutorial (in the strings section)</a>.</p>
