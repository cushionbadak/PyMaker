<p>In Python tests I know how to check the expected value of a variable:</p>
<pre><code>def test_smth(self):
    var = 4
    self.assertEqual(var, 4)
</code></pre>
<p>But is it possible to check with one or another variable without flooding the code with <code>if</code>?</p>
<p>Something like:</p>
<pre><code>def test_smth(self):
    var = 5
    self.assertEqual(var, 4 or 5)  # this doesn't work
</code></pre>
</div>

<p>Thanks to <a href="https://stackoverflow.com/users/3001761/jonrsharpe">jonrsharpe</a>.</p>
<pre><code>self.assertIn(var, (4, 5))
</code></pre>
<p><a href="https://docs.python.org/3/library/unittest.html#assert-methods" rel="nofollow noreferrer">Here</a> is link to docs.</p>
