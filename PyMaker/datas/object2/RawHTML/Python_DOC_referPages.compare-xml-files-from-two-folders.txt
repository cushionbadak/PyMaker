<p>In Python:</p>
<p><a href="https://docs.python.org/2/library/filecmp.html" rel="nofollow noreferrer"> filecmp: File and Directory Comparisons</a>:</p>
<pre><code>import filecmp
cmp = filecmp.cmp('file_1.xml', 'file_2.xml')

# Files are equal
if cmp:
    continue
else:
    out_file.write('file_1.xml') 
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>groovy with usage of xmlunit:</p>
<pre class="lang-groovy prettyprint-override"><code>@Grab(group='xmlunit', module='xmlunit', version='1.6')
import org.custommonkey.xmlunit.XMLUnit 

XMLUnit.setIgnoreWhitespace(true)
def r1 = new File('/11/1.xml').newReader("UTF-8")
def r2 = new File('/11/2.xml').newReader("UTF-8")
def diff = XMLUnit.compareXML(r1, r2)
assert diff.similar()
</code></pre>
</div>

<p>You can  compare two XML files using TreeOps visual tool  <a href="https://github.com/treeops/treeops" rel="nofollow noreferrer">https://github.com/treeops/treeops</a>. It transforms and compares tree data in XML/JSON/CSV.
User can select paths to be ignored by comparison.</p>
<p>TreeOps defines set of data tree transformations to facilitate typical data manipulations:</p>
<ul>

<li>renaming and moving nodes
along the tree </li>
<li>flattening and de-flattening to convert between tree and tabular formats </li>
<li>change nodes using regular expressions under selected nodes</li>
</ul>
