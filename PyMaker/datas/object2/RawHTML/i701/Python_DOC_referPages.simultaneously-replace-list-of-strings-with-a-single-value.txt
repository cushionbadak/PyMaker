<p>I want to replace a list of strings with a single string in a pythonic way. To clarify what I mean, this code does what I want:</p>
<pre><code>my_string = 'abc'
list_strings = ['a', 'b']
for replace_string in list_strings:
    my_string = my_string.replace(replace_string, 'c')
</code></pre>
<p>I would have imagined though, that </p>
<pre><code>my_string = my_string.replace(list_strings, 'c')
</code></pre>
<p>would have done the trick. But the <code>replace</code> method only accepts a string as input. </p>
<p>Is there a more pythonic way (I imagine, a way without a <code>for</code> cycle) to perform this operation?</p>
<p>Thanks in advance!</p>
</div>
<div class="post-text" itemprop="text">
<p>On solution could be to use regular expression (see <a href="https://docs.python.org/3/library/re.html#re.sub" rel="nofollow noreferrer"><code>re.sub</code></a>):</p>
<pre><code>import re

my_string = 'abc'

list_strings = ['a', 'b']

pattern = '|'.join(list_strings)
output = re.sub(pattern, 'bc', my_string)
print(output)
#  bcbcc
</code></pre>
<p>note: I modified the replacement string to <code>bc</code> to show a case leading to a different output </p>
