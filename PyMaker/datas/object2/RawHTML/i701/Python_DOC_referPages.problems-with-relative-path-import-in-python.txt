<div class="post-text" itemprop="text">
<p>I'll be honest: relative import could be hell. That's why Python 3 does absolute imports by default (<a href="https://docs.python.org/2.5/whatsnew/pep-328.html" rel="nofollow noreferrer">https://docs.python.org/2.5/whatsnew/pep-328.html</a>). </p>
<p>If you use Python 3 - make all your packages absolute.</p>
<p>If you still use Python 2 you can switch default behavior by adding the following line at the top:</p>
<pre><code>from __future__ import absolute_import
</code></pre>
