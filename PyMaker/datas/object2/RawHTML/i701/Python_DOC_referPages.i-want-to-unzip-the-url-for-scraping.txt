<p>Take a look at <a href="https://docs.python.org/3/library/gzip.html" rel="nofollow noreferrer">gzip</a></p>
<pre><code>import requests
from io import BytesIO
import gzip
r = requests.get('https://www.tokopedia.com/sitemap/product/1.xml.gz',stream=True)
g=gzip.GzipFile(fileobj=BytesIO(r.content))
content=g.read()
print(content)
</code></pre>
