<div class="post-text" itemprop="text">
<p>Make your subfolders python <a href="https://docs.python.org/3/tutorial/modules.html#packages" rel="nofollow noreferrer">packages</a></p>
<p>Make sure that only the root is on the python path. </p>
<p>Even better use a standard layout and use pip -e installs</p>
<p>See <a href="https://blog.ionelmc.ro/2014/05/25/python-packaging/" rel="nofollow noreferrer">this</a> for a great layout.</p>
<p>Then use absolute imports like import package.subpackage.module or explicit relative ones like import .module</p>
</div>
<div class="post-text" itemprop="text">
<p>Add <code>__init__.py</code> to each directory to turn them into packages.  Then you can use relative imports, such as in fileb.py</p>
<pre><code>from ..FolderA import FileA
</code></pre>
<p>See <a href="https://stackoverflow.com/q/14132789/8357969">Relative imports for the billionth time</a> for detailed explanation of relative imports and how they work in python. </p>
