<p>It took me a while to understand but it is quite easy the reason, the character table is read in many ways, in your case the problem is that you want to print in utf8 but the table utf-8 is related to the system language, you have to print in utf-16</p>
<p>I'll give you some examples:</p>
<p>in javascript:</p>
<pre><code>console.log("pom\u{00f4}\u{017E}e");
</code></pre>
<p>in python 3</p>
<pre><code>print("pom"+u"\u00F4"+u"\u017E"+"e")
</code></pre>
<p>in python 2</p>
<pre><code>print("pom"+u"\u00F4".encode('utf-8')+u"\u017E".encode('utf-8')+"e")
</code></pre>
