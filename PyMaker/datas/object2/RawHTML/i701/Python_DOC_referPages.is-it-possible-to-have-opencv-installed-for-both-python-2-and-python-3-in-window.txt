<p>According to the PyPi repository, <a href="https://pypi.org/project/opencv-python/#files" rel="nofollow noreferrer">opencv</a> is available for many versions of Python from 2.7 to 3.7. So, yes you can install it twice for Python 2.7 and 3.7 (if it is available for your OS).</p>
<p>But you need to have two Python environments, a.k.a. Virtualenvs! Of course Conda can solve this problem but you can also use the <a href="https://virtualenv.pypa.io/en/latest/" rel="nofollow noreferrer">virtualenv</a> tools. Follow the instructions to install it. It is a requirement for Python 2.</p>
<p>Choose a directory to store your virtualenvs, for Instance <code>~/virtualenv</code> in your HOME. Create it if necessary.</p>
<p>For Python 2.7, you can run:</p>
<pre><code>cd ~/virtualenv
virtualenv -p /path/to/python2 py2-demo
source py2-demo/bin/activate
pip install opencv-python
</code></pre>
<p>For Python &gt; 3.3, you can use the <a href="https://docs.python.org/3/library/venv.html" rel="nofollow noreferrer">venv</a> module:</p>
<pre><code>cd ~/virtualenv
python3 -m venv py3-demo
source py3-demo/bin/activate
pip install opencv-python
</code></pre>
