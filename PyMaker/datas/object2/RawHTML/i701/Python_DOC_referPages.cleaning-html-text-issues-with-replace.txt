<div class="post-text" itemprop="text">
<p>Your error is in how you use the <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer">strip</a> method, which removes any leading or trailing  sequence of the <code>'&lt;br&gt;'</code> characters. So <code>&lt;b&gt;hello&lt;/b&gt;</code> would be stripped to <code>hello&lt;/</code>, for example.</p>
<p>If you want to remove any <code>&lt;br&gt;</code> in the beginning and in the end of the <code>value</code> string, you can do it like so:</p>
<pre><code>if value.startswith('&lt;br&gt;'):
    value = value[4:]
if value.endswith('&lt;br&gt;'):
    value = value[:-4]
</code></pre>
