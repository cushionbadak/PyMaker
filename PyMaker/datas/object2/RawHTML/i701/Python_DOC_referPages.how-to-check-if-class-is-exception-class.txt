<div class="post-text" itemprop="text">
<p>I define a custom exception:</p>
<pre><code>&gt;&gt;&gt; class MyException(Exception):
&gt;&gt;&gt;     pass
</code></pre>
<p>I create an exception instance:</p>
<pre><code>&gt;&gt;&gt; a = MyException()
</code></pre>
<p>I check if this is an exception. As expected, it is:</p>
<pre><code>&gt;&gt;&gt; isinstance(a, Exception)
True 
</code></pre>
<p>But how do I check if the class is an exception class?</p>
<pre><code>&gt;&gt;&gt; myclass = MyException
&gt;&gt;&gt; isinstance(myclass, Exception)
False
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>you can check if one class is inherited from other class by using <a href="https://docs.python.org/3/library/functions.html#issubclass" rel="nofollow noreferrer">issubclass</a> function</p>
<pre><code>print issubclass(MyException, Exception)
</code></pre>
<p>result:</p>
<pre><code>True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You can try checking if <code>Exception</code> is under <code>__bases__</code>:</p>
<pre><code>&gt;&gt;&gt; my_class = MyException
&gt;&gt;&gt; Exception in my_class.__bases__
True
</code></pre>
</div>
