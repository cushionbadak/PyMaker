<div class="post-text" itemprop="text">
<p>I have a list of lists: </p>
<pre><code>data = [['SiteUrl','Url','Title'],['SiteUrl','Url','Title']]
</code></pre>
<p>How can I iterate through the list and lowercase all the content using python? </p>
</div>
<div class="post-text" itemprop="text">
<p>Using a list comprehension:</p>
<pre><code>data = [[x.casefold() for x in sublst] for sublst in data]
</code></pre>
<p>Or functionally:</p>
<pre><code>data = [list(map(str.casefold, x)) for x in data]
</code></pre>
<p>From <a href="https://docs.python.org/3/library/stdtypes.html#str.casefold" rel="nofollow noreferrer">the docs</a>:</p>
<blockquote>
<p>Casefolding is similar to lowercasing but more aggressive because it
  is intended to remove all case distinctions in a string.</p>
</blockquote>
</div>
<div class="post-text" itemprop="text">
<p>Iterate over each of the inner lists:</p>
<pre><code>data = [[string.lower() for string in sublist] for sublist in data]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could <code>map()</code> all sub lists of <code>data</code> to <code>map()</code> that will make all of its elements <code>lower()</code>.</p>
<pre><code>data= [['SiteUrl','Url','Title'],['SiteUrl','Url','Title']]

data = list(map(lambda x: list(map(lambda y: y.lower(), x)), data))
print(data)
</code></pre>
<p><strong>Output</strong></p>
<pre><code>[['siteurl', 'url', 'title'], ['siteurl', 'url', 'title']]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>For an arbitrary number of levels, you can try this recursion </p>
<pre><code>data= [['SiteUrl','Url','Title'],['SiteUrl','Url','Title']]
def recursive_apply(x, f=lambda v: v.lower()):
    if type(x) is list: 
        return [recursive_apply(el) for el in x]
    return f(x)
recursive_apply(data)   
</code></pre>
</div>
