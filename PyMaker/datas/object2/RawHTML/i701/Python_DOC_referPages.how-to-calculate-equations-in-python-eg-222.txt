<div class="post-text" itemprop="text">
<p>First of all, check your Python version, by running <code>python --version</code>.</p>
<p>In Python 2.x, <code>print</code> can be called without parentheses, but one of the main changes in Python 3.x is that <code>print(...)</code> requires them.</p>
<p>Python is a dynamic language, where types are inferred from their use. "2*2*2" is a <em>string type</em>, that depending on your version you can be printed by using either <code>print "2*2*2"</code>, or <code>print("2*2*2")</code>.</p>
<p>The returned data types can sometimes be confusing</p>
<pre><code>Python 2.7.12 (default, Dec  4 2017, 14:50:18)
Type "help", "copyright", "credits" or "license" for more information.
&gt;&gt;&gt;
&gt;&gt;&gt; 2*2*2      # This will return an integer
8
&gt;&gt;&gt; 2.*2.*2.   # This will return a floating-point number
8.0
&gt;&gt;&gt; "2*2*2"    # This will return the literal string
'2*2*2'
&gt;&gt;&gt; 2*"hello"  # This will again, return a string
'hellohello'
</code></pre>

<div class="post-text" itemprop="text">
<p>In the python shell (type <code>python</code> in terminal or cmd), type </p>
<pre><code>print(2*2*2)
</code></pre>
<p>to print the result.
You can also type </p>
<pre><code>2*2*2
</code></pre>
<p>and it will print <code>8</code> again.
This should work. Write the same line in a script and it will work too. However, in a script, you can't just type 2*2*2 to print the results, you need to use <code>print()</code>.</p>
<p>If instead of writnig 2*2*2, you need to do something like 1+2+3+4+5, which is a sum on a list like <code>list=[1,2,3,4,5]</code>, simply write</p>
<pre><code>sum(list)
</code></pre>
<p>If you want to do the sum of some elements in a dictionary like <code>d={1:2,2:4,3:8,4:16,5:32,6:64}</code> on the element from 3 to 5 inclusively, do</p>
<pre><code>sum(d[i] for i in range(3, 5 + 1))
</code></pre>
