<div class="post-text" itemprop="text">
<p>If <code>togglePlayaPause</code> is 1, then  <code>btn_play</code>  has to be evaluated else if <code>togglePlayaPause</code> is 0, then <code>btn_pause</code> has to be evaluated.  </p>
<p>Use  a <a href="https://docs.python.org/3/tutorial/controlflow.html" rel="nofollow noreferrer"><code>if</code> <code>elif</code> <code>else</code></a> statement to do so:</p>
<pre class="lang-py prettyprint-override"><code>if togglePlayaPause == 0:
    if btn_play.click():

        # [...]

elif togglePlayaPause == 1:
