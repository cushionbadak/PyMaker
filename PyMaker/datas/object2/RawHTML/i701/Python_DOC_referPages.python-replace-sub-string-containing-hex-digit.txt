<p>How can I replace the substring value (the hex value 0x20E00) with a new hex value in python?</p>
<pre><code> &lt;d:var name="Length" value="0x20E00"/&gt;
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use a regular expression and <a href="https://docs.python.org/3/library/re.html#re.sub" rel="nofollow noreferrer"><code>re.sub()</code></a>:</p>
<pre><code>&gt;&gt;&gt; import re
&gt;&gt;&gt; s = '&lt;d:var name="Length" value="0x20E00"/&gt;'
&gt;&gt;&gt; re.sub(r'0[xX][0-9a-fA-F]+', '0xdeadbeef', s)
'&lt;d:var name="Length" value="0xdeadbeef"/&gt;'
</code></pre>
<p>That said, if you're working with XML, you're much better off using an <a href="https://docs.python.org/3/library/xml.etree.elementtree.html" rel="nofollow noreferrer">XML parser</a> to manipulate it.</p>
</div>
<div class="post-text" itemprop="text">
<p>Here's a way using <a href="https://docs.python.org/2/library/re.html" rel="nofollow noreferrer"><code>re.sub</code></a> with <a href="https://docs.python.org/2/library/re.html" rel="nofollow noreferrer"><code>look behind</code></a>:</p>
<pre><code>re.sub(r'(?&lt;=value=\").*(?=\")', 'new hex value',s)
'&lt;d:var name="Length" value="new hex value"/&gt;'
</code></pre>
</div>
