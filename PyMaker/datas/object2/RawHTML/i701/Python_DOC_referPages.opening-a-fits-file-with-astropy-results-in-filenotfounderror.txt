<span class="comment-copy">As a general answer to "What should I do?" so that you can have an easier time with problems like this: Although the <code>astropy.io.fits</code> code is complicated, you can see in the end of the traceback that the code which raised the <code>FileNotFoundError</code> was <code>open(filename, mode, buffering=0)</code>.  Forget the <code>mode</code> and <code>buffering</code> parts for a sec: Most importantly it is calling <code>open(filename)</code>.  <a href="https://docs.python.org/3/library/functions.html#open" rel="nofollow noreferrer"><code>open()</code></a> is the Python built-in for opening a file, by default for reading.</span>
<span class="comment-copy">You could try calling <code>open()</code> directly on the path you gave and you'll get the same error.  Or even <code>import os</code> then check <code>os.path.exists(filename)</code> to see if it's an existing path on your system.</span>
