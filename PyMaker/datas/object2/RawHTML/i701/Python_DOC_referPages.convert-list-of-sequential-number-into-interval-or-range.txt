<p>I have a list like this:</p>
<pre><code>a = [1, 2, 3, 4, 5, 8, 9, 10, 11, 15, 16, 17]
</code></pre>
<p>I would like to get the result as:</p>
<pre><code>b = [1-5, 8-11, 15-17]
</code></pre>
<p>since in my list <code>a</code> I have sequential number from <code>1 to 5, 8 to 11</code> and then <code>15 to 17</code>.</p>
<p>How should I do this in Python?</p>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/library/itertools.html#itertools.groupby" rel="nofollow noreferrer">itertools.groupby</a> as <a href="https://stackoverflow.com/questions/4628333/converting-a-list-of-integers-into-range-in-python/4629241#4629241">described here</a> but with a little change  by <a href="https://stackoverflow.com/questions/4628333/converting-a-list-of-integers-into-range-in-python">@luca</a>:</p>
<pre><code>import itertools

def to_ranges(iterable):
    iterable = sorted(set(iterable))
    for key, group in itertools.groupby(enumerate(iterable),
                                        lambda t: t[1] - t[0]):
        group = list(group)
        yield group[0][1], group[-1][1]


x = [1, 2, 3, 4, 5, 8, 9, 10, 11, 15, 16, 17]
print( list(to_ranges(x)))
</code></pre>
<p>OUTPUT:</p>
<pre><code>[(1, 5), (8, 11), (15, 17)]
</code></pre>
