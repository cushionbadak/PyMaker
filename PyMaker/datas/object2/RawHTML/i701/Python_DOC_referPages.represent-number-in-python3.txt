<div class="post-text" itemprop="text">
<p>class <a href="https://docs.python.org/3/library/stdtypes.html#str" rel="nofollow noreferrer">str</a>(object=''). In particular you can use <a href="https://docs.python.org/3/library/string.html#format-examples" rel="nofollow noreferrer">str.format()</a></p>
<p>Just simply call .format() function and pass in the value. For example:</p>
<pre><code>number = 5435
print('bin: {0:b}   hex: {0:x}   int: {0:d}'.format(number))
</code></pre>
<p>output:
<code>bin: 1010100111011   hex: 153b   int: 5435</code></p>
<p>or</p>
<pre><code>number = 5435
print('bin: {0:#b}   hex: {0:#x}   int: {0:d}'.format(number))
</code></pre>
<p>output:
<code>bin: 0b1010100111011   hex: 0x153b   int: 5435</code></p>
</div>
<div class="post-text" itemprop="text">
<p>This can be achieved with the built-in integers and functions:</p>
<pre class="lang-py prettyprint-override"><code>Python 3.7.2 (default, Jan 10 2019, 23:51:51) 
[GCC 8.2.1 20181127] on linux
Type "help", "copyright", "credits" or "license" for more information.
&gt;&gt;&gt; i = 5
&gt;&gt;&gt; bin(i)
'0b101'
&gt;&gt;&gt; hex(i)
'0x5'
&gt;&gt;&gt; i
5
</code></pre>
<p>If you want to obtain a string representation of <code>i</code>, you can use <code>str(i)</code>.</p>
<p>To obtain the number itself from other bases: <code>i = int("ff", 16)</code>.</p>
