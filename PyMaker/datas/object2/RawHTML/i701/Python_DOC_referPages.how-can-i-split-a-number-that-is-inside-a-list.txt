<div class="post-text" itemprop="text">
<p>According to the comments in the question, each list has a single number on it. Here's a simple way to do the conversion, using <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehensions</a>:</p>
<pre><code>lst = [198]
[int(x) for x in str(lst[0])]
=&gt; [1, 9, 8]
</code></pre>
<p>To extend my solution for lists with more elements (not a requirement in the question, but what the heck):</p>
<pre><code>lst = [198, 199, 200]
[[int(x) for x in str(y)] for y in lst]
[[1, 9, 8], [1, 9, 9], [2, 0, 0]]
</code></pre>
</div>
