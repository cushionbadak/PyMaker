<div class="post-text" itemprop="text">
<p>I think what you are trying to do is to import <code>FirstFolder</code> and <code>SecondFolder</code> modules into main. </p>
<p>Now, trying to do <code>from TopFolder import *</code> will result in a circular import, because main will be importing itself (main.py is part of *), which will do the <code>import *</code> again, ... and so on. Furthermore, doing <code>import *</code> is discouraged (see <a href="https://stackoverflow.com/questions/2386714/why-is-import-bad">Why is “import *” bad?
</a>).</p>
<p>A better solution is to define FirstFolder and SecondFolder as <a href="https://docs.python.org/3/tutorial/modules.html#packages" rel="nofollow noreferrer">packages</a>.<br/>
Then import specific modules from those packages in main.</p>
<pre><code>TopFolder
 |- FirstFolder
    |- __init__.py
    |- foo1.py
 |- SecondFolder
    |- __init__.py
    |- foo2.py
 |- main.py
</code></pre>
<p>In foo1.py:</p>
<pre><code>def bar1():
    return "bar 1"
</code></pre>
<p>In foo2.py:</p>
<pre><code>def bar2():
    return "bar 2"
</code></pre>
<p>In main.py:</p>
<pre><code># from TopFolder import *  # Don't do this
from FirstFolder import foo1
from SecondFolder import foo2

print(foo1.bar1())
print(foo2.bar2())
</code></pre>
<p>Result:</p>
<pre><code>$ python3 main.py
bar 1
bar 2
</code></pre>
