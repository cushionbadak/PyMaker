<div class="post-text" itemprop="text">
<p>I created a simple game using Pygame. Is it possible to run this game on other computer, where Pygame is not installed? 
I tried to write a BATCHT file, which installs Pygame using pip. But it isn't working too. </p>
</div>
<div class="post-text" itemprop="text">
<p>To make your game run in other computer where there is no pygame installed you need to make it executable. Following package can make your script executable:</p>
<ol>
<li><a href="https://pyinstaller.readthedocs.io/en/stable/" rel="nofollow noreferrer">pyinstaller</a></li>
<li><a href="http://www.py2exe.org/" rel="nofollow noreferrer">py2exe</a></li>
</ol>
<p><strong>Tutorial</strong></p>
<ol>
<li><a href="https://www.youtube.com/watch?v=lOIJIk_maO4" rel="nofollow noreferrer">Converting to executable using pyinstaller</a></li>
<li><a href="https://www.youtube.com/watch?v=VKQ1Ph81Gps" rel="nofollow noreferrer">Converting to executable using py2exe</a></li>
</ol>
<p>Pyinstaller is way more easier than py2exe. But both of them works like a charm!</p>
</div>
<div class="post-text" itemprop="text">
<p>Maybe you could try to build a dist, in case your target system is linux based (.rpm or .deb). You find a detailed description here:</p>
<p><a href="https://docs.python.org/3/distutils/builtdist.html" rel="nofollow noreferrer">https://docs.python.org/3/distutils/builtdist.html</a></p>
</div>
<div class="post-text" itemprop="text">
<p>Try auto-py-to-exe (<a href="https://pypi.org/project/auto-py-to-exe/" rel="nofollow noreferrer">https://pypi.org/project/auto-py-to-exe/</a>)
With this you can pack the whole thing into an exe.
It supports everything above python 2.7</p>
</div>
<span class="comment-copy">If your target machine runs Windows, you can try <a href="http://www.py2exe.org" rel="nofollow noreferrer">py2exe</a>.</span>
