<p>I'm trying to make a function so that it can check to see if the first letter in each string in a list is uppercase.</p>
<pre><code>def capital(str_list):
    for n in str_list:
        if n[0].isupper():
            return True
        else:
            return False
</code></pre>
<p>If I have a list, it's only checking the first word. How do I check if the first letter of each word in the list is uppercase or not?</p>

<div class="post-text" itemprop="text">
<p>Return <code>False</code> immediately when you stumble upon a non-uppercase letter and return <code>True</code> only after you've checked each element:</p>
<pre><code>def capital(str_list):
    for n in str_list:
        if not n[0].isupper():
            return False 
    return True
</code></pre>
<p>You may also consider a more compact solution using <code>all</code>:</p>
<pre><code>def capital(str_list): return all(i[0].isupper() for i in str_list)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Instead of returning <code>True</code> if you find a word that starts with upper case, you need to return <code>False</code> when one word doesn't start with upper case. After completing the loop just return <code>True</code> because now all your word starts with upper case.</p>
<pre><code>def capital(str_list):
    for n in str_list:
        if not n[0].isupper():
            return False
    return True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>More pythonic way:</p>
<pre><code>def capital(str_list):
    return all(n[0].isupper() for n in str_list)
</code></pre>
</div>

<p>You might find python's builtin <code>any()</code> and <code>all()</code> functions useful: <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer">https://docs.python.org/3/library/functions.html#all</a></p>

<div class="post-text" itemprop="text">
<p>As i mentioned in my comment on your question, the reason for your function to only check for the first word in the list is due to the <code>return</code> statement, ie. as soon as a function gets its <code>return</code> statement executed, the execution of the function finishes.</p>
<p>Another way to do what you want to accomplish is via <code>map</code> and a <code>lambda</code> function, which can easily be customised in case you want to do something else later on.</p>
<pre><code>str_list = ['Apple', 'berry', 'Watermelon', 'Bananas']

def capital(str_list):
    return list(map(lambda x: x[0].isupper(), str_list))

capital(str_list)
</code></pre>
<p>Output</p>
<pre><code>[True, False, True, True]
</code></pre>
</div>
<span class="comment-copy">the reason why your function only checks the first word in the list is because you have a <code>return</code> statement within your <code>for loop</code> in the <code>if else</code> statement. as soon as a python function finds a <code>return</code> statement in it and it gets executed, in your case, it finishes checking the first word, it exits the function.</span>
