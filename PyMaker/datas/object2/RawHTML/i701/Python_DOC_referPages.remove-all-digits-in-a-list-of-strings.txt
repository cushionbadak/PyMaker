<div class="post-text" itemprop="text">
<p>You could use a list comprehension and remove all digits using <code>re.sub</code>:</p>
<pre><code>import re
words = ['Duration12', 'Noun1', 'Adjective7']
[re.sub(r'[0-9]', '', w) for w in words]
</code></pre>
<p><b> Result </b></p>
<pre><code>['Duration', 'Noun', 'Adjective']
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Try This One:</p>
<pre><code>In [8]: from string import digits

In [9]: remove_digits = str.maketrans('', '', digits)

In [10]: [word.translate(remove_digits) for word in words]
Out[10]: ['Duration', 'Noun', 'Adjective']
</code></pre>
<p>For more info refer to the link: <a href="https://docs.python.org/2/library/string.html" rel="nofollow noreferrer">Link</a></p>
</div>
<div class="post-text" itemprop="text">
<p>You can filter out the numbers, using <a href="https://docs.python.org/3/library/stdtypes.html#str.isalpha" rel="nofollow noreferrer">isalpha</a>:</p>
<pre><code>words = ['Duration12', 'Noun1', 'Adjective7']

result = [''.join(c for c in w if c.isalpha()) for w in words]
print(result)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>If you are sure all digits are at end of words, you might use <code>.strip</code> following way:</p>
<pre><code>words = ['Duration12', 'Noun1', 'Adjective7']
results = [i.strip('0123456789') for i in words]
print(results) #give ['Duration', 'Noun', 'Adjective']
</code></pre>
<p>but keep in mind that this method would not work if digits are inside rather than at end</p>
