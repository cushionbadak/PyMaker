<div class="post-text" itemprop="text">
<p>Returning as soon as you find one is better. You can leverage <a href="https://docs.python.org/3/library/functions.html" rel="nofollow noreferrer">pythons built in functions</a> to do that:</p>
<pre><code>def isIncreasing(seq):
   return all(a&lt;b for a,b in zip(seq,seq[1:]))
</code></pre>
<p>This zippes your sequence into pairs:</p>
<pre><code>[1,2,3,4] =&gt; [(1,2),(2,3),(3,4)]
</code></pre>
<p>and checks each pair. <code>all()</code> terminates as soon as it finds a <code>False</code>.</p>
<p><strong>Documentation</strong>:</p>
<ul>
<li><a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow noreferrer">all()</a> </li>
<li><a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer">zip()</a> </li>
</ul>
</div>
<div class="post-text" itemprop="text">

<pre><code>In [33]: all(i&gt;=1 for i in np.diff(b))
