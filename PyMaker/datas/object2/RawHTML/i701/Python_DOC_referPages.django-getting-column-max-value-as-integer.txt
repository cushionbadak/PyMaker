<p><a href="https://docs.djangoproject.com/en/2.1/topics/db/aggregation/" rel="nofollow noreferrer">Aggregation</a> actually returns a dictionary, and you can get the value same way as getting a value from dictionary using a key. For example (by using <a href="https://docs.python.org/3/library/stdtypes.html#dict.get" rel="nofollow noreferrer"><code>dict.get()</code></a>):</p>
<pre><code>maxAux = Auxi.objects.all().aggregate(max_val = Max('nroAux'))  # providing a key max_val

maxAux.get('max_val')
</code></pre>
