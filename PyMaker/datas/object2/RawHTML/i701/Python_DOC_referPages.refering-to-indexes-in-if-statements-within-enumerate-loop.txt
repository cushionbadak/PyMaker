<pre><code>stringy = ['j', 'k', 'l']

for i, v in enumerate(stringy):
    if (i == 0, 1):
        print('hi')
    elif(i == 2):
        print('bye')
</code></pre>

<div class="post-text" itemprop="text">
<pre><code>stringy = ['j', 'k', 'l']

for i, v in enumerate(stringy):
    if i &lt;2 :
        print('hi')
    else:
        print('bye')
</code></pre>

<span class="comment-copy">(i==0,1) is a tuple - non empty tuples are Truthy - hence your 1st condition is always used ... use <code>if i in {0,1}:</code></span>
<span class="comment-copy">see <a href="https://docs.python.org/3/library/stdtypes.html#truth-value-testing" rel="nofollow noreferrer">truth-value-testing</a></span>
<span class="comment-copy">Yes...this approach works.</span>
