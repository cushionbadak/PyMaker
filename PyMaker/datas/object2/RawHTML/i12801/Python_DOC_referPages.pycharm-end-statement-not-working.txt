<p>As already answered in comments, only python3 have keywords arguments for <a href="https://docs.python.org/3/library/functions.html?highlight=print#print" rel="nofollow"><code>print()</code></a>:</p>
<pre><code>print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
</code></pre>
<p>For python2, <a href="https://docs.python.org/2/reference/simple_stmts.html#print" rel="nofollow"><code>print</code></a> is a statement defined as this and have no keywords arguments:</p>
<pre><code>print_stmt ::=  "print" ([expression ("," expression)* [","]]
                | "&gt;&gt;" expression [("," expression)+ [","]])
</code></pre>
<p>You can either switch to python3 or disable the statement and use the <a href="https://docs.python.org/2/library/functions.html?highlight=print#print" rel="nofollow"><code>print()</code></a> function with this future line at the top of your module:<br/>
<code>from __future__ import print_function</code></p>
<pre><code>$ python2
&gt;&gt;&gt; help(print) # fail because print is a statement in python2
         ^
SyntaxError: invalid syntax
&gt;&gt;&gt; from __future__ import print_function
&gt;&gt;&gt; help(print)
&gt;&gt;&gt; Help on built-in function print in module __builtin__:

print(...)
    print(value, ..., sep=' ', end='\n', file=sys.stdout)
.....
</code></pre>
