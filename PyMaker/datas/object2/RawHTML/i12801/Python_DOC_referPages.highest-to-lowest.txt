<ol>
<li><p><code>sorted(f)</code> gives you something like:</p>
<pre><code>['abdur rauf ,2, 0, 0 \n',
 'adil ,8, 9, 7 \n',
 'danniel ,7, 7, 2\n',
 'masuda ,7, 4, 6 \n',
 'nathan ,2, 0, 10 \n',
 'zain ,0, 0, 3 \n']
</code></pre>
<p>Note that there's some <code>\n</code>, which means new lines.</p>
<p><strong>Solution:</strong> </p>
<p>Simpely use <a href="https://docs.python.org/3/library/stdtypes.html#str.splitlines" rel="nofollow noreferrer"><code>str.splitlines()</code></a> like:</p>
<pre><code>for f in sorted(f.read().splitlines())
</code></pre></li>
<li><p><code>f.split(',')</code> gives you (after fix that <code>\n</code> issue):</p>
<pre><code>['abdur rauf ,2, 0, 0 ',
'adil ,8, 9, 7 ',
'danniel ,7, 7, 2',
'masuda ,7, 4, 6',
'nathan ,2, 0, 10 ',
'zain ,0, 0, 3 ']
</code></pre>
<p>This doesn't work because there's some spaces behind <code>,</code>. Also there's another space at the end of the line.</p>
<p><strong>Solution:</strong></p>
<p>Use <a href="http://docs.python.org/3/library/re.html#re.split" rel="nofollow noreferrer"><code>re.split('\s*,\s*', f)</code></a>, to remove the another space at the end of the line, you can use <a href="https://docs.python.org/3/library/stdtypes.html#str.strip" rel="nofollow noreferrer"><code>str.strip()</code></a> like this:</p>
<pre><code>&gt;&gt;&gt; for f in sorted(f.read().splitlines()):
...     re.split('\s*,\s*', f.strip())
...     
... 
['abdur rauf', '2', '0', '0']
['adil', '8', '9', '7']
['danniel', '7', '7', '2']
['masuda', '7', '4', '6']
['nathan', '2', '0', '10']
['zain', '0', '0', '3']
</code></pre></li>
<li><p><code>list(reversed(scores))</code> can only reverse the <code>scores</code> list. you should use <code>sorted()</code> here again. </p>
<p>But since use <code>sorted</code> on str object(like <code>'2'</code>, <code>'3'</code>) will gives you incorrect output. For example <code>'10' &lt; '2'</code> is <code>True</code>. Covert them to int object use <a href="https://docs.python.org/3/library/functions.html#map" rel="nofollow noreferrer"><code>map(int, scores)</code></a>.</p>
<p>Also you need <em>Highest to lowest</em>. Then you could enable <code>reverse</code> option of <code>sorted</code> via:</p>
<pre><code>hl = sorted(map(int, scores), reverse=True)
</code></pre>
<p><a href="https://stackoverflow.com/questions/9969698/python-difference-between-reversedlist-and-list-sortreverse-true">Here's another question on SO</a> about what's the difference between <code>reversed</code> and <code>sorted(list, reverse=True)</code></p></li>
<li><p><code>print(hl)</code> can only print that list out. In this case I think you need join them use <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>' ,'.join([name]+scores)</code></a>:</p>
<pre><code>print(' ,'.join([name]+list(map(str, hl))))  # covert `hl` elements to str objects again since we need use `str.join`
</code></pre></li>
</ol>
<hr/>
<p>So the full code looks like:</p>
<pre><code>import re

with open('class1.txt', 'r') as f:
    for f in sorted(f.read().splitlines()):
        name, *scores = re.split('\s*,\s*', f.strip())
        hl = sorted(map(int, scores), reverse=True)
        print(' ,'.join([name]+list(map(str, hl))))
</code></pre>
<p>The output would be:</p>
<pre><code>abdur rauf ,2 ,0 ,0
adil ,9 ,8 ,7
danniel ,7 ,7 ,2
masuda ,7 ,6 ,4
nathan ,10 ,2 ,0
zain ,3 ,0 ,0
</code></pre>
