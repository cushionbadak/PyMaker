<div class="post-text" itemprop="text">
<p>I have a text file that is laid out like this:</p>
<pre><code>Text Text
Text Text Text Text Text
Text Num Num Num Num
Text Num Num Num Num
Text Num Num Num Num
</code></pre>
<p>My code is:</p>
<pre><code>def readData():
    myList =[]
    myFile = open("football.txt", "r")
    for lines in myFile:
        league = lines.split()
        myList.append(league)
    return myList
</code></pre>
<p>How can I skip the first two row so I can store everything else into my list?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can also use the <code>readlines()</code> for this purpose</p>
<pre><code>myFile = open("football.txt", "r")
lines = myFile.readlines()[2:] #To skip two lines.
#rest of the code
</code></pre>
<p>You could also specify the number of lines you want to skip.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#next" rel="nofollow"><code>next()</code></a> </p>
<pre><code>def readData():
    with open("football.txt", "r") as MyFile:
        for i in range(2):
            next(myFile) # skip line
        myList = [lines.split() for lines in myFile]
    return myList
</code></pre>
