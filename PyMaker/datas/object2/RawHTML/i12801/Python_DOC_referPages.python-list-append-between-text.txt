<p>In Python 3, how would you go about taking the string between header tags, for example, printing <code>Hello, world!</code>, out of <code>&lt;h1&gt;Hello, world!&lt;/h1&gt;</code>:</p>
<pre><code>import urllib
from urllib.request import urlopen

#example URL that includes an &lt;h&gt; tag: http://www.hobo-web.co.uk/headers/
userAddress = input("Enter a website URL: ")

webPage = urllib.request.urlopen(userAddress)

list = []

while webPage != "":
    webPage.read()
    list.append() 
</code></pre>

<div class="post-text" itemprop="text">
<p>You need an <em>HTML Parser</em>. For example, <a href="http://www.crummy.com/software/BeautifulSoup/bs4/doc/" rel="nofollow noreferrer"><code>BeautifulSoup</code></a>:</p>
<pre><code>from bs4 import BeautifulSoup

soup = BeautifulSoup(webPage)
print(soup.find("h1").get_text(strip=True))
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; from urllib.request import urlopen
&gt;&gt;&gt; from bs4 import BeautifulSoup
&gt;&gt;&gt;
&gt;&gt;&gt; url = "http://www.hobo-web.co.uk/headers/"
&gt;&gt;&gt; webPage = urlopen(url)
&gt;&gt;&gt;
&gt;&gt;&gt; soup = BeautifulSoup(webPage, "html.parser")
&gt;&gt;&gt; print(soup.find("h1").get_text(strip=True))
