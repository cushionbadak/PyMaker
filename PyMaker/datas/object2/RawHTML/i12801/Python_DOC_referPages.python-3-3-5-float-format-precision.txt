<div class="post-text" itemprop="text">
<p>You need to include the type <code>f</code> (for float) to control the number of digits after the decimal place:</p>
<pre><code>&gt;&gt;&gt; '{:.10f}'.format(100/3)
'33.3333333333'
</code></pre>
<p>Without specifying the type, Python falls back to the general number type and the number is rounded to the specified number of significant digits. From the <a href="https://docs.python.org/3.1/library/string.html#format-specification-mini-language" rel="nofollow">documentation</a>:</p>
<blockquote>
<p>The precision is a decimal number indicating how many digits should be displayed after the decimal point for a floating point value formatted with 'f' and 'F', or before and after the decimal point for a floating point value formatted with 'g' or 'G'.</p>
</blockquote>
