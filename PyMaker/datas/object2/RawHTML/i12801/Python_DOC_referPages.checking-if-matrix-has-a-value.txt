<div class="post-text" itemprop="text">
<p>If I have a matrix where it prints all zeros so:</p>
<pre><code>m=[[0,0,0],[0,0,0],[0,0,0]]
</code></pre>
<p>I want to check if after I filled in with different numbers whether or not there any zeros left. Just has to return a boolean <code>True</code> or <code>False</code>.</p>
<pre><code>m=[[1,1,1],[1,1,1],[1,1,1]]
&gt;&gt; True
m=[[1,1,1],[1,0,1],[1,1,1]]
&gt;&gt; False
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Use <a href="https://docs.python.org/2/library/functions.html#any" rel="nofollow"><code>any</code></a> and <code>not</code></p>
<pre><code>&gt;&gt;&gt; m=[[1,1,1],[1,0,1],[1,1,1]]
&gt;&gt;&gt; not any(j==0 for i in m for j in i)
False
&gt;&gt;&gt; m=[[1,1,1],[1,1,1],[1,1,1]]
&gt;&gt;&gt; not any(j==0 for i in m for j in i)
True
</code></pre>
<p><code>any</code> returns <code>True</code> if any element of the iterable is true. If the iterable is empty, it returns <code>False</code>.</p>
</div>
<div class="post-text" itemprop="text">
<p>Alternative solution that makes use of Numpy's <a href="http://docs.scipy.org/doc/numpy/reference/generated/numpy.count_nonzero.html" rel="nofollow">count_nonzero</a> instead of list comprehensions</p>
<pre><code>&gt;&gt;&gt; import numpy as np
&gt;&gt;&gt; m = [[1,1,1],[1,1,1],[1,1,1]]
&gt;&gt;&gt; m = np.asarray(m)
&gt;&gt;&gt; np.count_nonzero(m) != m.size
False

&gt;&gt;&gt; m=[[1,1,1],[1,0,1],[1,1,1]]
&gt;&gt;&gt; m = np.asarray(m)
&gt;&gt;&gt; np.count_nonzero(m) != m.size
True
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using <a href="https://docs.python.org/3/library/functions.html#all" rel="nofollow"><code>all</code></a>, <a href="https://docs.python.org/3/tutorial/classes.html#generator-expressions" rel="nofollow">generator expression</a> with <a href="https://docs.python.org/3/reference/expressions.html#not-in" rel="nofollow"><code>not in</code> operator</a>:</p>
<pre><code>&gt;&gt;&gt; m = [[1,1,1], [1,1,1], [1,1,1]]
&gt;&gt;&gt; all(0 not in x for x in m)
True

&gt;&gt;&gt; m = [[1,1,1], [1,0,1], [1,1,1]]
&gt;&gt;&gt; all(0 not in x for x in m)
False
</code></pre>
</div>
