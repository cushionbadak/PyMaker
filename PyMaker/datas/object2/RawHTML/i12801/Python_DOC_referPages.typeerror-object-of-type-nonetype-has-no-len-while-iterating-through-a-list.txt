<div class="post-text" itemprop="text">
<p><a href="https://docs.python.org/3/library/stdtypes.html#mutable-sequence-types" rel="nofollow"><code>list.append</code></a> returns <code>None</code>.</p>
<p>The following line:</p>
<pre><code>word_list = word_list.append(word)
</code></pre>
<p>should be replaced with:</p>
<pre><code>word_list.append(word)
</code></pre>
<p>Otherwise, <code>word_list</code> becomes <code>None</code> causing the <code>TypeError</code> later.</p>
</div>
<span class="comment-copy">Briefly: <code>append()</code> mutates the object in place and returns <code>None</code>. Don't save the result of an <code>append()</code> operation back to the same reference. Just do the operation, <code>word_list.append(word)</code>.</span>
