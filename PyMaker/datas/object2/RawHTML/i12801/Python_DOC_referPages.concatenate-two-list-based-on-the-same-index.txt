<div class="post-text" itemprop="text">
<p>I have two lists.</p>
<pre><code>A = [1,5,10]
B = [72,36,58]
</code></pre>
<p>I'd like to concatenate two lists based on the same index.</p>
<pre><code>Output = [1,72,5,36,10,58]
</code></pre>
<p>I know I can use <code>zip(A,B)</code> to do so but in this way I need to remove tuple from the list. </p>
<p>Any hint or elegant way to do this?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use a nested list comprehension :</p>
<pre><code>&gt;&gt;&gt; [i for tup in zip(A,B) for i in tup]
[1, 72, 5, 36, 10, 58]
</code></pre>
<p>If you are dealing with huge datasets, using <code>Numpy</code> extension is a good choice for you, which in that case you would be able to use a lot of cool features. And in this case you can use <code>numpy.hstack()</code> to flatten the <code>zip()</code> result :</p>
<pre><code>&gt;&gt;&gt; import numpy as np
&gt;&gt;&gt; 
&gt;&gt;&gt; np.hstack(zip(A,B))
array([ 1, 72,  5, 36, 10, 58])
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Here's an <a href="https://docs.python.org/3/library/itertools.html#itertools.chain.from_iterable" rel="nofollow"><code>itertools</code></a> approach:</p>
<pre><code>&gt;&gt;&gt; from itertools import chain
&gt;&gt;&gt; list(chain.from_iterable(zip(A, B)))
[1, 72, 5, 36, 10, 58]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Something likes this should do the job:</p>
<pre><code>A = [1,5,10]
B = [72,36,58]

Out = []
for i in range(0, 3):
    Out.append(A[i])
    Out.append(B[i])
</code></pre>
<p>And you get:</p>
<pre><code>Out = [1, 72, 5, 36, 10, 58]
</code></pre>
