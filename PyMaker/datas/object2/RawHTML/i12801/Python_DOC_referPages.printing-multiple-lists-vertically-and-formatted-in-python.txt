<p>Let's say I have four lists:</p>
<pre><code>l1=[3,5,6]
l2=[0,2]
l3=[3,4,3,1,2]
l4=[2,3,2]
</code></pre>
<p>And I want to print them like this:</p>
<pre><code>      2
      1  
6     3  2
5  2  4  3
3  0  3  2
</code></pre>
<p>Can anyone help me, please?</p>
</div>
<div class="post-text" itemprop="text">
<p>If you are using Python 2.x you can use <a href="https://docs.python.org/2/library/itertools.html#itertools.izip_longest" rel="nofollow"><code>izip_longest()</code></a> from <strong>itertools</strong>:</p>
<pre><code>for i in reversed([' '.join(map(str,i)) for i in izip_longest(l1, l2, l3, l4, fillvalue=' ')]):
    print i
</code></pre>
<p>In Python 3.x you can use <a href="https://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="nofollow"><code>zip_longest()</code></a> from <strong>itertools</strong>.</p>
<p><strong>Output:</strong></p>
<pre><code>    2
    1
6   3 2
5 2 4 3
3 0 3 2
</code></pre>
</div>

I believe that if you structure it like that you may have some success </p>
