<p>I have a Dictionary in Python carrying an <code>id</code> and a <code>date</code>, something like </p>
<pre><code>{'snap-c8': '2015/12/02',
 'snap-23': '2015/12/05',
 'snap-60': '2015/12/06',
 'snap-1a': '2015/12/07',
 'snap-d1': '2015/12/04',
 'snap-67': '2015/12/03',
 'snap-b9': '2015/12/08'}
</code></pre>
<p>I don't know Python and I need help to create a function that would get me the <code>snap-</code> value from the oldest date.</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use <a href="https://docs.python.org/3/library/functions.html#min" rel="nofollow"><code>min</code></a> with <a href="https://docs.python.org/3/library/stdtypes.html#dict.get" rel="nofollow"><code>dict.get</code></a> as a key function:</p>
<pre><code>&gt;&gt;&gt; d = {'snap-c8': '2015/12/02', ..., 'snap-b9': '2015/12/08'}
&gt;&gt;&gt; min(d, key=d.get)
'snap-c8'
</code></pre>
</div>
