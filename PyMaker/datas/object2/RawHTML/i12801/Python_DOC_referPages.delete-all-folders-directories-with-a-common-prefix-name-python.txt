<div class="post-text" itemprop="text">
<p>I would like to delete all folders/directories with a common prefix name.
How can this be achieved in python ?</p>
<p>example :
all folders with names with prefix - 'reports_'</p>
</div>
<div class="post-text" itemprop="text">
<p>this is the simplest solution that I could think of to solve your problem.</p>
<pre><code>import shutil
import glob
import os
base_path = ""
dir_list = glob.iglob(os.path.join(base_path, "reports_*"))
for path in dir_list:
    if os.path.isdir(path):
        shutil.rmtree(path)
</code></pre>
