<p>I have a text file which is Tab delimited.  I read this line by line into an "array" called lstLine.</p>
<p>Some lines have more elements than others.  And by elements I think you know that means "columns of data".  Some lines have 5 columns of data, others have 7.  </p>
<p>How do I count the number of elements that the line I just read has?
Here's my code for getting the line into the lstLine array.</p>
<pre><code>for oLine in inputfile:  
...
    lstLine = oLine.replace("\n", "").split("\t")
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>The <code>split</code> function returns a <code>list</code>. Hence the number of elements in <code>list</code> can be extracted from <code>len()</code> function.
Use it as follows.</p>
<pre><code>lstLine = oLine.replace("\n", "").split("\t")
numElements = len(lstLine)
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>As @Fejs mentioned in the comment, just do <code>len(lstLine)</code>.</p>
<p>By <code>split</code>, you get <code>lstLine</code> as a Python <code>List</code>, <code>len(lstLine)</code> counts number of elements in the list.</p>
