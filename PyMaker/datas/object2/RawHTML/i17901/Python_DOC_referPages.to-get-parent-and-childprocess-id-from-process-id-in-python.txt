<div class="post-text" itemprop="text">
<p>The answer depends on your system's <code>ps</code> command. On Linux, <code>ps</code> will include the PPID for each process with the <code>-l</code> flag (among others), so <code>ps -ale | grep ruby</code> will include the ruby process id in <code>str[3]</code> and ruby's PPID in <code>str[4]</code>.</p>
