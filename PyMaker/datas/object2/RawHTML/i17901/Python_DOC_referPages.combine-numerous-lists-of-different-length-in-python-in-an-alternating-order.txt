<div class="post-text" itemprop="text">
<p>Use <a href="http://docs.python.org/3/library/itertools.html#itertools.zip_longest" rel="noreferrer"><code>itertools.zip_longest()</code></a> here:</p>
<pre><code>try:
    from itertools import zip_longest
except ImportError:
    # Python 2
    from itertools import izip_longest as zip_longest

def alternate(list1, list2):
    return [v for v in sum(zip_longest(list1, list2), ()) if v is not None]
</code></pre>
<p>The <code>zip_longest()</code> call adds <code>None</code> placeholders (similar to your own attempt to add <code>-</code> characters), which we need to remove again from the <code>sum()</code> output after zipping.</p>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; alternate(list1, list2)
['1', '5', '2', '6', '3', '7', '8']
&gt;&gt;&gt; alternate(alternate(list1, list2), list3)
['1', '9', '5', '2', '6', '3', '7', '8']
</code></pre>
