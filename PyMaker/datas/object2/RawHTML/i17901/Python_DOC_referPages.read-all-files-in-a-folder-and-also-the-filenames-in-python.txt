<p>Maybe something like this?  Or os.listdir() is simpler if you don't need recursion.</p>
<p>Even on Windows, Python abstracts away the differences between operating systems if you use it well.</p>
<pre><code>#!/usr/local/cpython-3.3/bin/python

import os

def main():
    for root, dirs, files in os.walk('/home/dstromberg/src/outside-questions'):
        for directory in dirs:
            print('directory', os.path.join(root, directory))
        for file_ in files:
            print('file', os.path.join(root, file_))

main()
</code></pre>
<p>See <a href="http://docs.python.org/3/library/os.html" rel="nofollow">http://docs.python.org/3/library/os.html</a> for more info.</p>
