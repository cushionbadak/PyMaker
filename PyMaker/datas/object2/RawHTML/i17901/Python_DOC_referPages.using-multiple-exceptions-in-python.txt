<p>Is there a way to use multiple exceptions in python? Like code below:</p>
<pre><code>try:
   #mycode
except AttributeError TypeError ValueError:
   #my exception
</code></pre>
<p>What I mean is how to use  <code>AttributeError</code> <code>TypeError</code> <code>ValueError</code> with each other?</p>
</div>
<div class="post-text" itemprop="text">
<p>Use a tuple:</p>
<pre><code>try:
   # mycode
except (AttributeError, TypeError, ValueError):
   # catches any of the three exception types above
</code></pre>
<p>Quoting the <a href="http://docs.python.org/3/reference/compound_stmts.html#the-try-statement" rel="nofollow">reference <code>try</code> statement documentation</a>:</p>
<blockquote>
<p>When an exception occurs in the <code>try</code> suite, a search for an exception handler is started. This search inspects the except clauses in turn until one is found that matches the exception.<br/>
<em>[...]</em><br/>
  For an except clause with an expression, that expression is evaluated, and the clause matches the exception if the resulting object is “compatible” with the exception. An object is compatible with an exception if it is the class or a base class of the exception object <strong>or a tuple containing an item compatible with the exception</strong>.</p>
</blockquote>
<p>Emphasis mine.</p>
