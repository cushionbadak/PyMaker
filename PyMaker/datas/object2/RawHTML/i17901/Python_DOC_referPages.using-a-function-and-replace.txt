<div class="post-text" itemprop="text">
<p>Python 3 dictionaries only have a <code>dict.items()</code> method, <code>dict.iteritems()</code> has been removed.</p>
<pre><code>def replace_all(text, dic):
    for i, j in dic.items():
        text = text.replace(i, j)
    return text

my_text = "Hello everybody."

reps = {"H": "|-|", "e": "3", "o": "0"}

txt = replace_all(my_text, reps)
</code></pre>
<p>works just fine.</p>
<p>However, it would be much more efficient to use <a href="http://docs.python.org/3/library/stdtypes.html#str.translate" rel="nofollow"><code>str.translate()</code></a> here:</p>
<pre><code>trans_map = {ord(k): v for k, v in reps.items()}
txt = my_text.translate(trans_map)
</code></pre>
<p>Demo:</p>
<pre><code>&gt;&gt;&gt; replace_all(my_text, reps)
'|-|3ll0 3v3ryb0dy.'
&gt;&gt;&gt; trans_map = {ord(k): v for k, v in reps.items()}
&gt;&gt;&gt; my_text.translate(trans_map)
'|-|3ll0 3v3ryb0dy.'
</code></pre>
