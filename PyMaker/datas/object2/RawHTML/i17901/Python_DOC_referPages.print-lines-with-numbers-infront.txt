<div class="post-text" itemprop="text">
<p>The parentheses are unbalanced in the following line:</p>
<pre><code>   print("\n".join(str(n,x))
</code></pre>
<p>Replace it with:</p>
<pre><code>   print("\n".join(str(n,x)))
   #                        ^
</code></pre>
<p>Another issue: <code>str</code> accepts only one argument.</p>
<pre><code>&gt;&gt;&gt; str(1, 2)
Traceback (most recent call last):
  File "&lt;stdin&gt;", line 1, in &lt;module&gt;
TypeError: str() takes at most 1 argument (2 given)
&gt;&gt;&gt; str(1)
'1'
</code></pre>
<hr/>
<p>Complete example code:</p>
<pre><code>n=0
for x in List:
    print(n, x)
    n += 1
</code></pre>
<p>using <a href="http://docs.python.org/3/library/functions#enumerate" rel="nofollow"><code>enumerate</code></a>:</p>
<pre><code>for n, x in enumerate(List, 1):
    print(n, x)
</code></pre>

<div class="post-text" itemprop="text">
<p>You missed a closing parenthesis:</p>
<pre><code>print("\n".join(str(n,x))
#                --------^
</code></pre>
<p>but that's not a correct statement still as <code>str()</code> takes one string argument with optional encoding.</p>
<p>It'll be easier to use <code>enumerate()</code> to add a count, and pass in two arguments to <code>print()</code>:</p>
<pre><code>for n, x in enumerate(Line):
    print(n, x)
</code></pre>
