<p>I printed my result from predictions.csv file, and I'm trying to save my result, that showed as 0 and 1, to CSV file in 2 columns. So how can I save my result to a CSV file of 2 columns as it showed?</p>

<div class="post-text" itemprop="text">
<p>You don't have to save the printed output - you can directly save the data to a CSV when you're printing it in the loop, like this:</p>
<pre><code>with open('new_file.csv', mode='a', newline='') as file:
    csv_writer = csv.writer(file)
    if a &gt; b :
        print(1, 0)
        csv_writer.writerow(['1', '0'])
    else:
        print(0, 1)
        csv_writer.writerow(['0', '1'])
</code></pre>

<pre><code>%%capture cap --no-stderr
with open('predictions.csv', 'r') as f:
    csvreader = csv.reader(f)
    next(csvreader)
    for r in csvreader:
        df = r[:-1]
        a = float(str(df[:][0]))
        b = float(str(df[:][1]))
        with open('output.csv', 'w') as writer:
            if a &gt; b :
                print(1,',',0)
            else:
                print(0,',',1)
            writer.write(cap.stdout)
</code></pre>
