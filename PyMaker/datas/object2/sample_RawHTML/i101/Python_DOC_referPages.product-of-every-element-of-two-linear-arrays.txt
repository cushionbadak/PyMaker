<p>How can I multiply every element of two linear arrays with each other, i.e. if I got these two arrays:</p>
<pre><code>x=[1, 4, 0 ,3]
y=[2, 1, 9 ,4]
</code></pre>
<p>I would like to get the following one as the output :</p>
<pre><code>z=[2, 4, 0, 12]
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>using a <a href="https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions" rel="nofollow noreferrer">list comprehension</a> is one way; using <a href="https://docs.python.org/3/library/functions.html#zip" rel="nofollow noreferrer"><code>zip</code></a> to iterate over both lists simultaneously:</p>
<pre><code>z = [a * b for a, b in zip(x, y)]
</code></pre>
<p>a different way is to use <a href="https://www.numpy.org/" rel="nofollow noreferrer">numpy</a>:</p>
<pre><code>import numpy as np

x = np.array([1, 4, 0 ,3])
y = np.array([2, 1, 9 ,4])

z = x * y
print(z)  # [ 2  4  0 12]
</code></pre>
</div>

<p>You can use <code>.zip()</code> to accomplish just that</p>
<p>Using list comprehension:</p>
<pre><code>[x*y for x,y in zip(list_x,list_y)]
</code></pre>
<p>Without list comprehension:</p>
<pre><code>for x,y in zip(list_x,list_y):
    print(x*y)
</code></pre>

<div class="post-text" itemprop="text">
<p>The multiplication of two array is only possible when size of both arrays are equal in length.</p>
<p>Try this code !</p>
<p><strong>Code :</strong></p>
<pre><code>x=[1, 4, 0 ,3]
y=[2, 1, 9 ,4]
z= []
if (len(x)==len(y)):
    for i in range(0,len(x)):
        z.append(x[i]*y[i])
    print(z)
else:
    print("Array are not equal in size.... Multiplication is not possible")
</code></pre>
<p><strong>Output :</strong></p>
<pre><code>[2, 4, 0, 12]
</code></pre>
