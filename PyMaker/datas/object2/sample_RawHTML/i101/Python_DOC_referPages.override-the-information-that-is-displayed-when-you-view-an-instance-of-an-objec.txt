<p>I'd like to override the information shown when I attempt to display the instance of an object on the command line. </p>
<p>For instance, I create the instance of a mock object like so:</p>
<pre><code>x = Test()
</code></pre>
<p>When I now type x and hit enter I see: </p>
<pre><code>&lt;class '__main__.Test'&gt;
</code></pre>
<p>What method do I have to override to change what is displayed by default? </p>
</div>
<div class="post-text" itemprop="text">
<p>Python has 2 methods to display an object: <code>__str__</code> and <code>__repr__</code></p>
<p><code>__str__</code> will return a human-readable string.</p>
<p><code>__repr__</code> function returns the object representation.**</p>
<pre><code>class Person:
    name = ""
    age = 0

    def __init__(self, personName, personAge):
        self.name = personName
        self.age = personAge

    def __repr__(self):
        return '{name:'+self.name+', age:'+str(self.age)+ '}'

    def __str__(self):
        return 'Person(name='+self.name+', age='+str(self.age)+ ')'
</code></pre>
<p>Now:</p>
<pre><code>p = Person('Mahrez', 22)
print(p)
</code></pre>
<p>==&gt;</p>
<pre><code>Person(name=Mahrez, age=22)
</code></pre>
<p>and :</p>
<pre><code>print(repr(p))
</code></pre>
<p>==&gt;</p>
<pre><code>{name:Mahrez, age:22}
</code></pre>
<p><strong>Read <a href="https://docs.python.org/3/library/functions.html" rel="nofollow noreferrer">this</a> from python documentation it's so useful</strong> </p>
