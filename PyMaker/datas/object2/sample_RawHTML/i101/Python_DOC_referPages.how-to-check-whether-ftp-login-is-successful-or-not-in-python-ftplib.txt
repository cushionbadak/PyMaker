<p>I am developing a FTP client using Python ftplib. Here is my code:</p>
<pre><code>def connectftp():
ftp=ftplib.FTP('hostname')
print(ftp.getwelcome())
ftp.login(username, password)
</code></pre>
<p>I want to display a message saying <code>"Login successful"</code> when the FTP login is successful. How to do that? How to check whether the login is successful or not?</p>
</div>
<div class="post-text" itemprop="text">
<p>The <a href="https://docs.python.org/3/library/ftplib.html#ftplib.FTP.login" rel="nofollow noreferrer"><code>FTP.login</code> method</a> throws an exception, when an authentication fails, so you can simply do:</p>
<pre><code>ftp.login(username, password)
print("Login successful")
</code></pre>
<p>If you want to react to authentication failures as well, handle the exception:</p>
<pre><code>try:
    ftp.login(username, password)
    print("Login successful")
    # do your FTP stuff here
except:
    print("Login failed")
</code></pre>
</div>

<p>What you want is </p>
<pre class="lang-py prettyprint-override"><code>def connectftp():
    ftp=ftplib.FTP('hostname')
    print(ftp.getwelcome())
    ftp.login(username, password)
    login_response = ftp.login(username, password)
    print(login_response)
</code></pre>
<p>login_response should be '230 Login successful.'</p>
