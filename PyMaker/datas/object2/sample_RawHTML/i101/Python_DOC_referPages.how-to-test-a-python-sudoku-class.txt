<p>You should change the names of methods in TestSum class</p>
<pre class="lang-py prettyprint-override"><code>    def willWork(self):
</code></pre>
<p>To</p>
<pre class="lang-py prettyprint-override"><code>    def testWillWork(self):
</code></pre>
