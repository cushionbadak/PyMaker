<div class="post-text" itemprop="text">
<p>I have a string that contains several <code>html tags</code> with this form:</p>
<pre><code>string= '&lt;input type="hidden" name="csrf" value="7629b234d1cc2f2a5383f5e6d7dc6bd2"&gt;'
</code></pre>
<p>I would like to extract the <code>"value"</code>, I would do it with a <code>.split</code>, but I am not sure if it is the best way.</p>
<p>output desired: </p>
<pre><code>7629b234d1cc2f2a5383f5e6d7dc6bd2
</code></pre>
<p>any idea or a better way?</p>
</div>
<div class="post-text" itemprop="text">
<p>You can use the <a href="https://docs.python.org/3/library/html.parser.html" rel="nofollow noreferrer"><code>html.parser</code></a></p>
<pre><code>from html.parser import HTMLParser

class ValueFinder(HTMLParser):
    def handle_starttag(self, tag, attrs):
        for attr in attrs:
            if attr[0] == "value":
                print(attr[1])

parser = ValueFinder()
parser.feed('&lt;input type="hidden" name="csrf" value="7629b234d1cc2f2a5383f5e6d7dc6bd2"&gt;')
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>You could use <a href="https://docs.python.org/3/library/re.html#re.search" rel="nofollow noreferrer"><code>re.search</code></a>:</p>
<pre><code>import re
string= '&lt;input type="hidden" name="csrf" value="7629b234d1cc2f2a5383f5e6d7dc6bd2"&gt;'
m = re.search('value="(.+)?"',string)[1]
print(m)
</code></pre>
<p>Output:</p>
<pre><code>629b234d1cc2f2a5383f5e6d7dc6bd2
</code></pre>
</div>
<div class="post-text" itemprop="text">
<p>Using Regular expression</p>
<pre><code>import re

match = re.compile(r'(value=\"(.*)\"\&gt;)$').finditer(string)

for i in  match:
    print(i.group(2))
</code></pre>
<p>Output:</p>
<pre><code>7629b234d1cc2f2a5383f5e6d7dc6bd2
</code></pre>
</div>
