<span class="comment-copy">This is not a run-time error, it's a syntax error. You have a missing quote somewhere.</span>
<span class="comment-copy"><code>.replace('\','')</code> is escaped incorrectly. It should be <code>'\''</code>.</span>
<span class="comment-copy">If you want a comma-separated list of set values, do so explicitly; don't abuse <code>list.__repr__</code> to do it. <code>', '.join(private_ips)</code>.</span>
<span class="comment-copy">I have a feeling that you're looking for <a href="https://docs.python.org/3/library/stdtypes.html#str.join" rel="nofollow noreferrer"><code>str.join</code></a>... but as it stands, the question is somewhat unclear.</span>
