Why Python's set object has no sort () method
Short answer from python doc.https://docs.python.org/3.4/tutorial/datastructures.html#setsA set is an unordered collection with no duplicate elements. https://docs.python.org/2/library/sets.htmlSince sets only define partial ordering (subset relationships), the output of the list.sort() method is undefined for lists of sets.
