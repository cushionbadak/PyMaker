There are many ways to do tests, often it depends on what framework you use. But, just assuming that you want some form of a formal test framework, you can use pytest. Step 1, install pytestStep 2, Format your code. You will test your output according to the return of the function. Step 3, Now write the testsStep 4, run the testsThis is not a unittest, but a boilerplate test framework. If you want to do unittests, there is a great guide to it here: http://www.onlamp.com/pub/a/python/2004/12/02/tdd_pyunit.html
