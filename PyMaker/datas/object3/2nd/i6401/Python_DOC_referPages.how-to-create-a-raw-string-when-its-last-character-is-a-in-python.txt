duplicated Why can't Python's raw string literals end with a single backslash?https://docs.python.org/3/reference/lexical_analysis.htmlSpecifically, a raw literal cannot end in a single backslash (since the backslash would escape the following quote character). Note also that a single backslash followed by a newline is interpreted as those two characters as part of the literal, not as a line continuation.
