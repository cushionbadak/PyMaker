When you pass in a list only the first item is treated as a command. The rest is treated as arguments to the shell used to run the command. From the  Python docs :On POSIX with shell=True, the shell defaults to /bin/sh. If args is a  string, the string specifies the command to execute through the shell.  This means that the string must be formatted exactly as it would be  when typed at the shell prompt. This includes, for example, quoting or  backslash escaping filenames with spaces in them. If args is a  sequence, the first item specifies the command string, and any  additional items will be treated as additional arguments to the shell  itself.
