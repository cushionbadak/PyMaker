The problem comes from the way you write and read the message and the key to and from the files. Indeed, the encrypted text and the keys are not , but  and need to be treated in a different way.   In your case, when reading and writing it is enough to specify that you do it in binary mode and you can do it by adding the b flag. See the documentation for more detailed information.Moreover, in your decrypt you also need to read the key from the file.
