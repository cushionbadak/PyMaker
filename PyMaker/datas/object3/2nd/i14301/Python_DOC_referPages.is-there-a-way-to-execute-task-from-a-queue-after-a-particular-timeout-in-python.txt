Let us assume that I have  different number of tasks and a stream of tasks are coming with different given . Is there a way to ensure that each task gets executed after their given ? I'm using python to create and execute these tasks.
Keep the required execution times in a sorted list.Wait till the time present at the beginning of the list.Remove the first item from the list and execute the task.Repeat.
