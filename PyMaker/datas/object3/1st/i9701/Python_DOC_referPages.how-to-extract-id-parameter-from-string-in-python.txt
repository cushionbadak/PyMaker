I have a URL as string : How to get the id parameter from the given string ?
The proper way to to do this in Python 3 is to use the urllib.parse module.outputThe field names of the ParseResult should be self-explanatory, but please refer to the docs for details.From the urllib.parse.parse_qs docsParse a query string given as a string argument (data of type  application/x-www-form-urlencoded). Data are returned as a dictionary.  The dictionary keys are the unique query variable names and the values  are lists of values for each name.
