Use max with a key:
Using max() + operator.itemgetter():How the above works:itemgetter(-1) fetches the last item, and applies it the key argument of max(), which indicates the max must be found by comparing the last element of every sublist.  A side note, max() is always O(N), since you need to check every element of the list to determine the maximal element.
