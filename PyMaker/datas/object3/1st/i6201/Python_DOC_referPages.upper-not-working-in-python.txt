The .upper() and .lower() functions do not modify the original str. Per the documentation,For .upper():str.upper()Return a copy of the string with all the cased characters converted to uppercase.For .lower():str.lower()Return a copy of the string with all the cased characters converted to lowercase.So if you want the uppercase and lowercase characters respectively, you need to print lines[o][p][u].upper() and lines[o][p][u].lower() as opposed to lines[o][p][u].
