Is it UDP or TCP-IP? How do I write this as TCP-IP if it's UDP and vice  versa?
This will create a TCP socket because the type argument of socket.socket(...) defaults to SOCK_STREAM. For UDP you would need to use SOCK_DGRAM as type instead. See the extensive documentation for more details.
