When I use:I get this:<socket.socket fd=376, family=AddressFamily.AF_INET, type=SocketKind.SOCK_STREAM, proto=0, laddr=('192.168.31.244', 4160), raddr=('192.168.31.244', 7061)>I can also successfully print:But if I try to print the address:I get and error:How can I print the laddr and raddr attributes?
Try using the .getsockname() and .getpeername() methods instead. As noted in the Socket object docs only the family, proto, and type fields are available as attributes.
For laddr use mySocket.getsockname() and for raddr use mySocket.getpeername()
