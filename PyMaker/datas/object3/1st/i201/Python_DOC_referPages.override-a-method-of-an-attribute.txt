I have a class that has an attribute as an object from another class. A third class inherits from the first class, and in there, I want to override the attribute's method ...What is the best way to override in such a situation?
This technique is called "monkey patching". In general it should be avoided, because it makes the code very hard to understand and reason about. Avoid it unless you have a very good cause. One of the good causes might be to mock a method in unit tests.If you still want to monkey patch just one method on one object self.door - check out this answer.
