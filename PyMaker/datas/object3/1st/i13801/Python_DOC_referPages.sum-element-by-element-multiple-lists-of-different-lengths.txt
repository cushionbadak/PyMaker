Is there any way to sum over multiple lists, index by index, to get one final list? Knowing that these lists might not have the same length? For example, with theseI would like to haveas a result.I haven't found any result so far.
You can use itertools.zip_longest with the fillvalue argument set to 0. If you use this in a list comprehension, you can unpack and zip the inner lists and add them in an element-wise fashion.
