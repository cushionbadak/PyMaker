i would like to know how to convert datetime Wed Apr 24 19:25:06 2013 GMT into Linux timestamp 1366831506000 (13 digits)] and reverse using python.for example:toand fromto
From string to timestamp, use time.strptime(), passing the resulting struct_time tuple to time.mktime(); your timestamp uses milliseconds, not the UNIX seconds-as-floating-point value, so you need to multiply by 1000:In the other direction, use time.strptime(), passing in a struct_time tuple created by time.gmtime(), dividing the timestamp by 1000 first:Demo:
