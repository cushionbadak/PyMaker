Use operator.itemgetter, like thisIf you have a list of indices, then you can unpack them to the itemgetter, like this
You could use a generator expression and tuple() to pull out specific indices:or you can use a operator.itemgetter() object to create a callable that does the same:where indices is your list of indexes. This works because operator.itemgetter() just happens to return a tuple object when retrieving multiple indexes.Demo:
