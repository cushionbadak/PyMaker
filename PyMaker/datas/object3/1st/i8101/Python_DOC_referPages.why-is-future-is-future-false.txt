If you take a look at the Python documentation:6.10.3. Identity comparisonsThe operators is and is not test for object identity: x is y is true if and only if x and y are the same object. Object identity is determined using the id() function. x is not y yields the inverse truth value.Thus, is does not check if an object is an instance of a class but if two objects have the same identity. For that you use the builtin function isinstance(instance, class):
