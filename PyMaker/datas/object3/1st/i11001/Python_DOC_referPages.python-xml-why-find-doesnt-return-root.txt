Why doesn't element.find("..") return root when root.findall("*") returns element?First 5 lines of XML file
Python's xml.etree.ElementTree implementation doesn't record the parent of an Element. Consequently, the documentation for XPath includes this:..  Selects the parent element. Returns None if the path attempts to reach the ancestors of the start element (the element find was called on).
