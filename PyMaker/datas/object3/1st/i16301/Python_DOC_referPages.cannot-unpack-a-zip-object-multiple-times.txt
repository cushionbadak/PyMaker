Why is this not working?:We cannot unpack a second time in Python, why is that?
zip returns an iterator in Python 3.x instead of a list like it did in Python 2.x.  This means that, after you unpack it once, it will become exhausted and no longer usable:You need to explicitly convert the iterator into a sequence (a list, tuple, etc). if you want to unpack it multiple times:
