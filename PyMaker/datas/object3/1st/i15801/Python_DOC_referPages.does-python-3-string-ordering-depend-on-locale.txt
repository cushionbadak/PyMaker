Do Python's str.__lt__ or sorted order characters based on their unicode index or by some locale-dependent collation rules? 
No, string ordering does not take locale into account. It is based entirely on the Unicode codepoint sort order.The locale module does provide you with a locale.strxform() function that can be used for locale-specific sorting:This tool is quite limited; for any serious collation task you probably want to use the PyICU library:
