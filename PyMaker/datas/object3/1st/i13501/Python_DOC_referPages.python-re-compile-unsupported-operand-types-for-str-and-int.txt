'i' is not a valid flag value, because all compilation flags used by re functions must be integers (re uses bitwise operations to manipulate flags).Use re.I (or re.IGNORECASE) insteadTechnically you can specify flags as strings but in this case they'll have to be included in the pattern:From the docs:(?aiLmsux)One or more letters from the set 'a', 'i', 'L', 'm', 's',  'u', 'x'. The group matches the empty string; the letters set the  corresponding flags.So (?i) has the same effect as passing re.I (or re.IGNORECASE) to compile.
