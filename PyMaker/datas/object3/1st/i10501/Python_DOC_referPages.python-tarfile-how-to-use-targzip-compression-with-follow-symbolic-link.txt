How could I use tar+gzip compression with "follow symbolic link" feature in Python 3.4? The problem is:tarfile.open() supports "w:gz" mode but does not support "dereference" optiontarfile.tarfile() supports "dereference" but does not support "w:gz" modeCode:
tarfile.open is a shortcut for the TarFile.open classmethod, that in turn calls the TarFile constructor. The documentation is a bit vague, but from the code it is apparent that the first two will pass the dereference keyword argument and all other unconsumed kwargs to the TarFile constructor. Thus you can use dereference with any of them, if you just pass it as a keyword argument:
