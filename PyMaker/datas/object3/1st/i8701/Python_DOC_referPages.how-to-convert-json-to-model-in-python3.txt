In order to do that you should provide your custom callback as an object_hook argument to the json.loads function.object_hook is an optional function that will be called with the  result of any object literal decode (a dict). The return value of  object_hook will be used instead of the dict. This feature  can be used to implement custom decoders (e.g. JSON-RPC class hinting).
