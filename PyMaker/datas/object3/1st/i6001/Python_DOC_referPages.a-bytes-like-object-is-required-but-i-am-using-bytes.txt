print() always writes str values. It'll convert any arguments to strings first, including bytes objects.From the print() documentation:All non-keyword arguments are converted to strings like str() does and written to the stream, separated by sep and followed by end.You can't use print() on a binary stream, period. Either write directly to the stream (using it's .write() method), or wrap the stream in a TextIOWrapper() object to handle encoding.These both work:and
