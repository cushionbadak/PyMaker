All your lines have the newline; you need to remove it first before processing the line:Python doesn't remove it for you. The str.rstrip() method used here will remove any number of \n newline characters from the end of the line; there will never be more than one. You could also extend this to any whitespace, on both ends of the string, by using str.strip() with no arguments.You already start with strings, so there is no need to use str() calls here.
