you could use groupby to group elements. To use a groupby make sure a sequence is sorted. groupby takes a sorted data and then breaks it to groups where the number changes. for example [1,1,1,3,5,5,5,8] will be broken into groups [1,1,1], [3], [5,5,5],[8], these groups will have a key 1, 3, 5, 8 respectively. groupby is a iterator and the keys and groups in the iterator can be exposed using for loops or next() operator.
