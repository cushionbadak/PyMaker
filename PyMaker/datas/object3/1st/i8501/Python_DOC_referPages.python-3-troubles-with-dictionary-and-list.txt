str.find() returns 0 when the text is found at the start:That is considered a false value in a boolean context:If the text is not there, str.find() returns -1 instead. From the str.find() documentation:Return the lowest index in the string where substring sub is found [...]. Return -1 if sub is not found.This means that only if the text is at the start will your code not print anything. In all other cases (including not finding the title), the tracks will be printed.Don't use str.find(). Use in to get True if the text is there, False if it is not:Demo using your json data:
