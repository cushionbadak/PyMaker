I am getting output 15.36 but I want only 15. I am using round function e.g round(15.36,2) but still getting the same result.
The documentation explains the arguments for the round function pretty well:round(number[, ndigits])Return number rounded to ndigits precision after the decimal point. If ndigits is omitted or is None, it returns the nearest integer to its input.So if you want to round to the nearest integer just use it without the second argument:
Use the  round function in python round(number, ndigits)So, for your example use round(15.36,0)
