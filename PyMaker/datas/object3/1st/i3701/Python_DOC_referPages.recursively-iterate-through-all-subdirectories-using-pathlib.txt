How can I use pathlib to recursively iterate over all subdirectories of a given directory?only seems to iterate over the immediate children of a given directory.I know this is possible with os.walk() or glob, but I want to use pathlib because I like working with the path objects.
You can use the glob method of a Path object:
pathlib has glob method where we can provide pattern as an argument. For example : Path('abc').glob('**/*.txt') - It will look for current folder abc and all other subdirectories recursively to locate all txt files.
Alternatively, use Path.rglob (substitutes the leading **):
