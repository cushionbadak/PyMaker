I don't think a simple reference to dict.items() should be counted a full fledged answer but since you insist:The issue was due to kwargs being a dict, and if you want to do a parameter expansion into key and value, you have to use kwargs.items(). This will return a list of key-value tuples which can then be readily expanded into, well, key and value.Beware that in Python 2.x dict.items() creates a whole list of key-value tuples instead of just giving you an iterator. To avoid needless resource usage, use dict.iteritems() instead.
Kwargs is a dict, so just use items()  in order to iterate over it.state_values = [value for key, value in kwargs.items() if key.startswith("state_")]
